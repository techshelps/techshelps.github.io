Output Pane

The Output Pane allows you to view any errors related to JavaScript or document processes.

 

This pane is a context-aware element, and its tab is in the side-bar if there are JavaScript or other errors when opening or using the PDF file.

 

It opens automatically to alert you of any important output messages.

To view the Output pane:

Click on the Output tab in the side-bar.

To clear messages in the Output pane:

  1. Click on the Output tab on the side-bar to open the Output pane

  2. Click the Clear to reset the Output Pane
