JavaScript

To provide a complete description of how you can use JavaScript with Nitro Pro 7 is well beyond the scope of this user manual. If you wish to gain programming knowledge in JavaScript, we recommend you review sources you can find on the Internet, as well as in books on JavaScript that can be obtained from your local library or book seller.

You can also find many examples of PDF documents containing JavaScript on the Internet. The most common use of JavaScript with PDF is to be found in PDF forms. You can locate these types of documents on the Internet, then open the JavaScript in the JavaScript Console, and copy/paste the routines in your own files. By examination, and trial and error, you can quickly learn the programming code necessary for the tasks you wish to perform.
