View and navigate a PDF file

Nitro Pro 7 is designed to make opening, viewing, and navigating PDF documents as easy as possible. The document and page controls allow you to manage your workspace, especially when working with more than one document at the same time.

 

You can learn more about viewing and navigating a PDF file by reading any of the following topics:

  • Open a PDF file

  • The Navigation Panes

  • Search and find text

  • Change the page view

  • Adjust the zoom level

  • Page through the document

  • Multiple documents in parallel view

 

For information on how to read a PDF file in your web-browser, see Nitro PDF web browser plugin.
