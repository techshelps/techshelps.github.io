Partner Plugins

The Partner Plugin framework enables Nitro Pro 7 to process and exchange files with third-party productivity tools and services. These plugins are streamlined into an easy to use interface to enhance the PDF workflow by reducing the need to manually save, copy, re-open, and upload files into your favorite utilities.

To manage partner plugins:

  1. In the File tab, select Preferences. The Nitro Pro 7 Preferences dialog opens.

  2. Select Plugin Manager

  3. Check the checkboxes for plugins you wish to enable, and un-check other checkboxes

  4. Click OK, and then restart Nitro Pro 7 for your changes to take effect
