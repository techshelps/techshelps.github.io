Delete an image

You can use the Edit tool to remove images from your PDF files.

To delete images:

  1. On the Edit ribbon tab, in the Text/Images panel, click the Edit button
  2. Click Edit Text & Images
  3. Click the image to select it
  4. Do one of the following:
    • Press the Delete key
    • Right-click and then click Delete

To select multiple images:

  1. On the Edit ribbon tab, in the Text/Images panel, click the Edit button
  2. Do one of the following:
    • Hold down the Shift key and then click on each image to select
    • Click an drag the cursor to select a group of images
  3. Do one of the following:
    • Press the Delete key
    • Right-click and then click Delete
