Check for software updates

Nitro Pro 7 can be setup to automatically check for product updates or you can check for updates manually. By default, the Check for Updates preference utility will check for software updates every week.

To check for updates manually

  1. On the Help tab, in the Product panel, click Check for Updates

To change frequency of Check for Updates

  1. In the File menu, click Preferences
  2. In the Preferences dialog, click Check for Updates
  3. Click the setting to use
  4. Click OK
