Combine files to PDF

Nitro Pro 7 supports combining PDF files and any other file type you provide, as long as you have an application installed on your computer that can view that file type. Nitro Pro 7 will convert non-PDF files to PDF files in the background, and then combine together every PDF file in the order specified.

To combine files to PDF from a folder or the desktop:

  1. Open the folder that contains the files you want to combine
  2. Hold down CTRL and click the files you want to select
  3. Right-click, and then choose Combine supported files in Nitro PDF
  4. Click Move Up and Move Down to reorder the files as desired
  5. In the Make PDF files list do one of the following, depending on whether file size or print quality is more important to you:
    • If the document requires high print quality, click Print-ready
    • If the print quality is less important than file size, click Office-ready
    • If a small file size is the most important quality, click Web-ready
  6. In the Save PDF to list, select the location you want to save the file to
  7. Click Create

To combine files to PDF in Nitro PDF Professional:

  1. On the Home tab, in the Create group, click Combine Files
  2. On the Combine Files dialog, click the Add Files button, and then select and order the files to convert
  3. In the Make PDF files list do one of the following, depending on whether file size or print quality is more important to you:
    • If the document requires high print quality, click Print-ready
    • If the print quality is less important than file size, click Office-ready
    • If a small file size is the most important quality, click Web-ready
  4. In the Save PDF to list, select the location you want to save the file to
  5. Click Create
