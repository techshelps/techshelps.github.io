Extract an image

The Extract Image feature lets you copy images out of PDF files for reuse in Nitro Pro 7 or other applications.

To extract an image:

  1. Use the Select tool to click an image. The image to highlight with a blue selection box.

  2. Right-click the selected image, and select Extract Image. The Save As dialog appears.

  3. Browse to a location, and specify a filename for the image

  4. Save as type: Specify an image file format: JPG, TIF or PNG

  5. Click Save
