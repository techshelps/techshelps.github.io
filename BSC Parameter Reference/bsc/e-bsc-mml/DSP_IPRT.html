Display IP Route(DSP IPRT)

Function

An OM command that queries the IP routes.

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning: number of the subrack for running the command

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: number of the slot for running the command

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None
DESTIP Dest IP address

Explanation: destination IP address to be routed

Value range: 0.0.0.0-255.255.255.255

Explanation: The value 255.255.255.255 indicates that all the routing information is queried.

Default value: 255.255.255.255

Example

Query the device IP route of a board (Subrack No. = 0; Slot No. = 24):
DSP IPRT: SRN=0, SN=24;
The result is as follows:
