Display MTP3 Status Between Signaling Points(DSP MTP3STATUSBTSP)

Function

An OM command that queries the operating status between specified MTP signaling points.

Note

None.

Parameters

ID Name Description
DPCINDEX MTP3 DSP index

Meaning: MTPS destination signaling point index

Value range: 0-255

Explanation: This command queries the status of a specified MTP3 signaling point.

Default value: None

Example

Query the information between two MTP3 signaling points (MTP DSP index = 3):
DSP MTP3STATUSBTSP: DPCINDEX=3;
The result is as follows:
+++    HUAWEI        2008-06-05 19:22:43
O&M    #6739
%%DSP MTP3STATUSBTSP: DPCINDEX=3;%%
RETCODE = 0  Operation succeeded

Display MTP3 Status Between SP Result
-------------------------------------
DSPINDEX  DPC     OPC     NI               ACCESSFLAG    CONGFLAG    

3         H'2001  H'2000  national backup  STATUS TRUE  STATUS FALSE
(Number of results = 1)


---    END

Output Description

Domain Name Description
DSPINDEX

Index of a destination signaling point entered for executing the command
DPC

Value of a destination signaling point in the record corresponding to the destination signaling point index
OPC

Value of OPC in the record corresponding to the OPC index
NI

Network where the destination signaling point is located. Value range: International, International backup, national, national backup
ACCESSFLAG

Indicates whether the destination signaling point is available. Value range: Accessible, Inaccessible
CONFLAG

Indicates whether the destination signaling point is congested. Value range: Uncongestion, Congestion
