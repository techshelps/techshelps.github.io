Cancel BATCHFILE Session Operation(CNL BATCHFILESN)

Function

An OM command that cancels the batch configuration command session.

Note

1. Ensure that the specified session ID exists. Query the session ID by running the command LST BATCHFILESN.

2. The execution of this command interrupts the current operations such as downloading and activation, but it does not affect the uploading of output files of batch configuration commands.

Parameters

ID Name Description
SID SessionID

Meaning: session ID

Value range: 0-268435455

Explanation: Session ID uniquely identifies a session.

Default value: None

Example

Cancel the current operation of the batch configuration command session 111111:
CNL BATCHFILESN:SID=111111;
Note:
This command is used to cancel the batch configuration command operations existing in the current system.
If a batch configuration command session does not exist in the system, then a message is displayed informing that the batch configuration session does not exist.
