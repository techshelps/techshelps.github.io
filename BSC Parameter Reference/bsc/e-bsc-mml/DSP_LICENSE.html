Display License of Board(DSP LICENSE)

Function

An OM command that queries the license of the board, and returns the license file information on the specified board.

Note

If the slot number is not specified, the license information of all normal boards (enable process) in the specified subrack is displayed.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: The subrack number is used to uniquely identify a subrack.

Default value: 0
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number are used to uniquely identify a board in a subrack.

Default value: number of any slot in which a board is positioned

Example

Query the license configuration of the board (Subrack No. = 0; Slot No. = 6):
DSP LICENSE:SRN=0,SN=6;
