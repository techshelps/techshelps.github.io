Display PVC Statistics(DSP PVC)

Function

An OM command that queries the information about PVC statistics.

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation:

1. This parameter specifies the subrack number of the PVC.

2. The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation:

1. This parameter specifies the slot number of the PVC.

2. The subrack number and slot number uniquely identify a board.

Default value: None
PN Port No.

Meaning: port number

Value range: 0-31

Explanation: This parameter specifies the port number of the PVC.

Default value: None
BCID BC Id

Meaning: BC ID

Value range: 0-255

Explanation: This parameter identifies the BC ID of the PVC.

Default value: None
DLCI DLCI

Meaning: data link connection ID

Value range: 16-1007

Explanation: data link connection ID of the PVC

Default value: None

Example

Query the information about PVC statistics (Subrack No. = 0; Slot No. = 22; Port No. = 0; BC Id = 0; DLCI = 66):
DSP PVC: SRN=0, SN=22, PN=0, BCID=0, DLCI=66;
