Clear PVC Statistics(CLR PVC)

Function

An OM command that clears the PVC statistics information.

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation:

1. This parameter specifies the subrack where the PVC statistics are to be cleared.

2. The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation:

1. This parameter specifies the slot where the PVC statistics are to be cleared.

2. The subrack number and slot number uniquely identify a board.

Default value: None
PN Port No.

Meaning: port number

Value range: 0-31

Explanation: This parameter specifies the port where the PVC statistics are to be cleared.

Default value: None
BCID BC Id

Meaning: BC ID

Value range: 0-255

Explanation: This parameter specifies the BC where the PVC statistics are to be cleared.

Default value: None
DLCI DLCI

Meaning: data link connection ID

Value range: 16-1007

Explanation: This parameter specifies the ID of the data link connection.

Default value: None

Example

Clear the information of PVC statistics (Subrack No. = 0; Slot No. = 22; Port No. = 0; BC Id = 0; DLCI = 66):
CLR PVC: SRN=0, SN=22, PN=0, BCID=0, DLCI=66;
