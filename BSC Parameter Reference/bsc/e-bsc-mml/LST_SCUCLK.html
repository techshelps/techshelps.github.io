List SCU Reference Clock(LST SCUCLK)

Function

An OM command that queries the configuration of the SCU reference clock.

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:None.

Default value:None

Example

Query the configuration of the SCU clock reference (Subrack No. = 0):
LST SCUCLK: SRN=0;
The result is as follows:
+++    HUAWEI        2008-07-24 08:34:45
O&M    #1153
%%LST SCUCLK: SRN=0;%%
RETCODE = 0  Operation succeeded

LST SCUCLK Result
-----------------
Subrack No.  Slot No.  Top Reference Clock

0            6         GGCU Back          
0            7         GGCU Back          
(Number of results = 2)


---    END

Output Description

For details, see the parameter description of the MOD SCUCLK command.
