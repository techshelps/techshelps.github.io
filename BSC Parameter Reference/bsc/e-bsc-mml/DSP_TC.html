Display TC Resources(DSP TC)

Function

An OM command that queries the channel status of a specified GDPUC or GDPUX.

Note

1. This command consumes certain system resources.

2. The MML client should communicate with the BSC properly.

3. The GDPUC or GDPUX should run properly.

4. This command does not apply to the BSC that adopts IP transmission on the A interface.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: This parameter specifies the subrack number that uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-3, 8-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None

Example

Query the DSP usage:
DSP TC: SRN=0, SN=10;
The result is as follows:
%%DSP TC: SRN=0, SN=10;%%
RETCODE = 0  Operation succeeded
