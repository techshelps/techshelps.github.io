List Transfer Server(LST TRANFTP)

Function

An OM command that queries the transfer server.

Note

None

Parameters

None.

Example

Query the transfer server:
LST TRANFTP:;
The result is as follows:
+++    HUAWEI        2008-12-29 18:46:20
O&M    #7585
%%LST TRANFTP:;%%
RETCODE = 0  Operation succeeded

LST TRANFTP Result
-------------- 
Is At OMU          IP            Port No         User Name

NO                 10.161.10.10  21              admin           
(Number of results = 1)


---    END

Output Description

For details, see the parameter description of the SET TRANFTP command.
