List Current Software Version(LST VER)

Function

An OM command that queries the current software information of the NE.

Note

The current software version state can be Normal or Fractional. Normal indicates that the version running on all the boards of the NE is consistent with the version of main area on the GOMU/GBAM. Fractional indicates that the version running on some boards of the NE is inconsistent with the version of main area on the GOMU/GBAM.

Parameters

None.

Example

List the current software version:
LST VER:;
The returned result is as follows: 
MML% LST VER:;
+++    HUAWEI        2008-12-10 00:23:04
O&M    #153
%%LST VER:;%%
RETCODE = 0  Operation succeeded

Result of current software query
--------------------------------
Current Software Version  =  BSC6000V900R008C12
 Current Software Status  =  Fractional
           Total Modules  =  12
         Succeed Modules  =  0
          Failed Modules  =  12
(Number of results = 1)
---    END

Output Description

Domain Name Description
Current Software Version

Current software version(the version of main area on the GOMU/GBAM)
Current Software Version State

Current software version state (Normal/Fractional)
Total Modules

Total number of modules
Succeeded Modules

Number of succeeded modules
Failed Modules

Number of failed modules
