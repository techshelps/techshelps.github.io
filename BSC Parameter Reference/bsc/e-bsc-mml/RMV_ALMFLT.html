Manually Clear Alarm(RMV ALMFLT)

Function

Manually clear the fault alarms that are not cleared in the system according to the alarm serial number.

Note

If the active alarm of a specified serial number is cleared, the alarm is removed from the active alarm records. You can run the LST ALMLOG instead of LST ALMAF command to query the alarm.

Parameters

ID Name Description
CSN Serial No

Meaning: serial number of the alarm records

Value range: 1-2147483646

Explanation: Each reported alarm is assigned a serial number in succession. The clear alarm and corresponding fault alarm have the same serial number.

Default value: None

Example

Clear the fault alarm with Serial No. as 33.:
RMV ALMFLT: CSN=33;
