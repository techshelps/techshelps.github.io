Display OIU Sub-board Attributes(DSP OPTSUBBRD)

Function

An OM command that queries the OIU sub-board properties.

Note

None.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board in a subrack.

Default value: None
SUBBRDN Sub-board No.

Meaning: sub-board number

Value range: 0

Explanation: The sub-board number uniquely identifies a sub-board on a board.

Default value: 0
J0_TYPE J0 show type

Meaning: J0 show type

Value range: SHOWTYPE_HEX(Hex), SHOWTYPE_STRING(String)

Explanation: SHOWTYPE_HEX indicates that byte J0 is displayed in hexadecimal mode, and SHOWTYPE_STRING indicates that byte J0 is displayed in string mode.

Default value: None
J1_TYPE J1 show type

Meaning: J1 show type

Value range: SHOWTYPE_HEX(Hex), SHOWTYPE_STRING(String)

Explanation: SHOWTYPE_HEX indicates that byte J1 is displayed in hexadecimal mode, and SHOWTYPE_STRING indicates that byte J1 is displayed in string mode.

Default value: None

Example

Query the attributes of sub-board 0 in slot 18, subrack 0, and display the J0 byte and the J1 byte as a hexadecimal numeral respectively:
DSP OPTSUBBRD: SRN=0, SN=18, J0_TYPE=SHOWTYPE_HEX, J1_TYPE=SHOWTYPE_HEX;
