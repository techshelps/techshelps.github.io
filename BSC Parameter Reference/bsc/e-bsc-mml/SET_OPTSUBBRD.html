Modify STM1 Sub-board Attributes(SET OPTSUBBRD)

Function

A configuration command that sets the parameters of the sub-board of an optical interface board. In this command, the following parameters uniquely identify an optical interface board:

Subrack No.

Slot No.

To modify sub-board attributes of all the optical interface boards in a specified subrack, set Subrack No..

To modify sub-board attributes of a specified optical interface board, set Subrack No. and Slot No..

Note

2. When the optical interface boards work in active/standby mode, if the sub-board attributes of one board are modified, those of the other board are modified automatically.

2. The value of B2 Signal Disabled Threshold must be larger than or equal to that of B2 Signal Degraded Threshold. The value of B3 Signal Disabled Threshold must be larger than or equal to that of B3 Signal Degraded Threshold.
