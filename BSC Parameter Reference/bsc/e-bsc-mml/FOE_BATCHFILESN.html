Force End BATCHFILE Session(FOE BATCHFILESN)

Function

An OM command that terminates the batch configuration command session forcibly. This command functions basically the same as the END BATCHFILESN command. The difference between them is that the former is used to end batch configuration command session initiated by the same terminal while the latter is used to end the batch configuration command session initiated by the different terminal.

Note

1. This command cannot forcibly end the batch configuration command session in the pre-activated state.

2. This command is usually used to deal with the exceptions, for example, when an MML terminal cannot end its own batch configuration command session normally or the configuration right is occupied for a long time, then end the batch configuration command session forcibly by using the terminal. After the command takes effect, the batch configuration command cannot resume on the operation terminal, and the file of the batch configuration session result may lose. In addition, the activated configuration data cannot be rolled back.

Parameters

ID Name Description
SID SessionID

Meaning: session ID

Value range: 0-268435455

Explanation: Session ID identifies a session.

Default value: None

Example

Forcibly end the batch configuration command session 111111:
FOE BATCHFILESN:SID=111111;
