Add Physical Port IP Address(ADD PHYPORTIP)

Function

A configuration command that adds the IP address of a physical port.

Note

1. This command applies to only the GFGUA/GOGUA/GFGUB/GOGUB/GFGUG.

2. The number of a GE port on the GFGUA/GFGUB/GFGUG can only be 0 or 4; the number on the GOGUA/GOGUB can only be 0 or 1.

3. Only one IP address can be added to a physical port.

4. The IP addresses of physical ports must be different from the IP addresses that have been configured.

5. The IP addresses of different physical ports and device IP addresses cannot be on the same network segment or network segments with inclusion relationship. This rule does not apply to the scenario in which a pair of active and standby boards are configured.

6. The port IP address and the destination IP address in the BSC route cannot be on the same network segment. If they are on the same network segment, the destination IP address is invalid.

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation: This parameter specifies the subrack number of an IP-based interface board.

Default value:None
SN Slot No.

Meaning:Slot No.

Value range:0~27

Explanation: This parameter specifies the slot number of an IP-based interface board.

Default value:None
PORT Port No.

Meaning:Port No.

Value range:0~7

Explanation: This parameter specifies the port number of an IP-based interface board.

Default value:None
IPINDEX IP address index

Meaning:IP address index

Value range:0~5

Explanation: This parameter specifies the IP address index. A maximum of six IP addresses can be configured for one port.

Default value:None
IPADDR IP address

Meaning:IP address

Value range:0.0.0.0-255.255.255.255

Explanation: This parameter specifies the port IP address of an IP-based interface board, which is also called as physical IP address.

Default value:None
MASK Subnet mask

Meaning:Subnet mask

Value range:0.0.0.0-255.255.255.255

Explanation: This parameter specifies the network mask of port IP address on the IP-based interface board.

Default value:None
IPADDR2 Standby IP address

Meaning:Standby IP address

Value range:0.0.0.0-255.255.255.255

Explanation: This parameter specifies the port IP address of the standby IP interface board when the IP interface boards are configured in active/standby mode.

Default value:None

Example

Add the IP address of a physical port (Subrack No. = 0; Slot No. = 14; Port No. = 0; IP address index = 0; IP address = 5.5.5.5; Subnet mask = 255.255.255.0): 
ADD PHYPORTIP: SRN=0, SN=14, PORT=0, IPINDEX=0, IPADDR="5.5.5.5", MASK="255.255.255.0";
