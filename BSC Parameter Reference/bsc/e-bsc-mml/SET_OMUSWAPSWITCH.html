Set Swap Switch(SET OMUSWAPSWITCH)

Function

An OM command that sets the status of the swap switch.

Note

1. The local Daemon must be running.

2. The settings of the swap switch are synchronized from the active OMU to the standby OMU, and are saved to the configuration file of the Daemon permanently.

3. If the swap switch is set to OFF(Disable), swap is disabled and execution of the SWP OMU command fails.

4. If the swap switch is set to ON(Enable), the manual swap cannot be performed if batch configuration is in process.

Parameters

ID Name Description
SW Swap switch

Meaning: This parameter specifies whether the swap is enabled or disabled.

Value range: OFF(Disable), ON(Enable)

Explanation: The value OFF(Disable) indicates that the swap is disabled; the value ON(Enable) indicates that the swap is enabled.

Default value: None

Example

1. Set the swap switch to OFF(Disable):
SET OMUSWAPSWITCH: SW=OFF;
2. Set the swap switch to ON(Enable):
SET OMUSWAPSWITCH: SW=ON;
