Start BTS Test(STR BTSTST)

Function

An OM command that starts the TRX loopback test, channel loopback test, and speech codec test.

Note

1. The BSC must communicate properly with the BTS.

2. The target BTS must be configured, and the TRX must be in active state.

3. TRX loopback is performed to test the TRX performance and the channel transmission condition. The TRX loopback test includes the BIU loopback test and the TRX self loopback test.

4. Channel loopback is performed to test the transmission capability of the traffic channel and the speech channel. The channel loopback test includes the BIU loopback test and the TRX RF self loopback test on the traffic channel and the BTS speech loopback test and TRX speech loopback test on the speech channel.

5. Channel codec test is performed to test the speech codec mode of the speech channel. The test result is reported.

Parameters

ID Name Description
TSTMODE Test mode

Meaning: test mode

Value range: TRXLOOPBACK, CHLLOOPBACK, CHLCODEC

Explanation:

1. In TRXLOOPBACK mode, TRX loopback is performed to test the performance of the TRX and the channel transmission condition. The loopback test includes the BIU loopback test and the TRX self loopback test.

2. In CHLLOOPBACK mode, channel loopback is performed to test the transmission capability of the traffic channel and the speech channel. The channel loopback test includes the BIU loopback test and the TRX RF self loopback test on the traffic channel and the BTS speech loopback test and TRX speech loopback test on the speech channel.

3. In CHLCODEC mode, channel codec test is performed to test the speech codec mode of the speech channel.

Default value: None
IDXTYPE Index Type

Meaning: how to specify a BTS

Value range: BYNAME(By Name), BYIDX(By Index)

Explanation: This parameter determines whether to specify a BTS by its name or index.

Default value: None
BTSIDX BTS Index

Meaning: BTS index

Value range: 0-2047

Explanation:

1. This parameter specifies the number of the BTS. Each BTS is numbered uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYIDX(By Index).

Default value: None
BTSNAME BTS Name

Meaning: BTS name

Value range: 1-32 characters

Explanation:

1. This parameter specifies the name of the BTS. Each BTS is named uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYNAME(By Name).

Default value: None
BTTSTTYPE BT test type

Meaning: TRX loopback test type

Value range: BIULOOP, TRXSELFTEST

Explanation:

1. In BIULOOP type, the transmission condition of the timeslots of the TRX signaling channel on the DBUS is tested.

2. In TRXSELFTEST type, the running status of the TRX is tested.

Default value: None
CHNTSTTYPE Channel test type

Meaning: channel loopback test type

Value range: TRXRFSELFLOOP, BIULOOP, BTSSPEECHLOOP, TRXSPEECHLOOP

Explanation:

1. In TRXRFSELFLOOP type, the quality of the receiving channel and transmitting channel of the TRX is tested.

2. In BIULOOP type, the transmission condition of the timeslots of the TRX traffic channel on the DBUS is tested.

3. In BTSSPEECHLOOP type, the connection condition of the TRX speech channel between the Um interface and the BTS DBUS is tested.

4. In TRXSPEECHLOOP type, the connection condition of the TRX speech channel between the Um interface and the DSP is tested.

5. For BTS3001C, BTS3002C, or BTS3001CP, this parameter cannot be set to BTSSPEECHLOOP or TRXSPEECHLOOP.

Default value: None

Default value: None

Example

Start the TRX loopback test on a BTS (Test mode = TRXLOOPBACK; Index Type = BYIDX(By Index); BTS Index = 0; BT test type = TRXSELFTEST; TRX No. = 0):
STR BTSTST: TSTMODE=TRXLOOPBACK, IDXTYPE=BYIDX, BTSIDX=0, BTTSTTYPE=TRXSELFTEST, TRXNO=0;
