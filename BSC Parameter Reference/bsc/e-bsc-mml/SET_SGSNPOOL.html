Set SGSN Pool Status(SET SGSNPOOL)

Function

A configuration command that set the status of the SGSN pool.

Note

When the SGSN Pool feature is enabled, 32 PTPBVCs can be configured in a cell; when the SGSN Pool feature is disabled, only one PTPBVC can be configured in a cell.

Parameters

ID Name Description
CNOPNAME Service Provider Name

Meaning:Service Provider Name

Value range:1-32 characters

Explanation: This parameter uniquely identifies an Service Provider entity in the system.

Default value:None
POOLALLOW Allow SGSN Pool

Meaning:Allow SGSN Pool

Value range:DENY(deny),ALLOW(allow)

Explanation: This parameter specifies whether to enable the SGSN Pool function.

Yes: Multiple SGSNs serve one cell.

No: One SGSN serves one cell.

Default value:None
NRILEN NRI length

Meaning:NRI length

Value range:1~10

Explanation: If the SGSN pool function is enabled, the NRI length cannot be 0.

Default value:None
NULLNRI NullNRI Value

Meaning:NullNRI Value

Value range:0~1023

Explanation: NullNRI Value.

Default value:None

Example

Set the SGSN pool status ( Operator Name = 46000; SGSN Pool Allow = ALLOW(allow); SGSN NRI Length = 8; NullNRI Value = 11):
SET SGSNPOOL: CnOpName="46000", POOLALLOW=ALLOW, NRILEN=8, NULLNRI=11;
