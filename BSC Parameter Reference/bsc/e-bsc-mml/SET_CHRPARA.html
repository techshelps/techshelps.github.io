Set The Configuration of CHR(SET CHRPARA)

Function

A configuration command that sets the CS-domain CHR configuration parameters or the PS-domain CHR configuration parameters.

Note

The MML client must communicate properly with the BSC.

Parameters

ID Name Description
CHRTYPE CHR Type

Meaning: CHR type

Value range: CS(CHR of CS), PS(CHR of PS)

Explanation: This parameter specifies the type of the CHR.

Default value: None
CSRCDSW CHR Record Switch

Meaning: CHR record switch

Value range: OFF(CHR Not Allowd), ON(CHR Allowed)

Explanation: This parameter specifies whether to enable the CHR record switch.

Default value: None

Meaning: CHR output control mode
