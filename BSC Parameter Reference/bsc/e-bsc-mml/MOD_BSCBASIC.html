Modify Basic Attributes of BSC(MOD BSCBASIC)

Function

A configuration command that modifies some parameters related to BTS basic attributes.

Note

1. The local mobile country code (MCC) and local mobile network code (MNC) can only be numeric.

2. If you have configured cells that operated over 1800 MHz, you cannot modify the attributes of supporting high frequency band as PSC1900. If you have configured cells that operated on the 1900 MHz, you cannot modify the attributes of supporting high frequency band as DCS1800.

3. If the CBCH channel is CBCH+SDCCH8 and the BS_AG_BLKS_RES is 0, you cannot set the attribute of whether to support the cell broadcast to Yes.

4. To disable the shielding of single pass check number, set the number to 32 Fs or 32 fs; to set the prefix of the operator number to invalid, set the number of 10 Fs or 10 fs.

5. After the BSC service mode is modified, the service modes of all subracks are updated at the same time. You must reset all subracks.

STREAMSWH Occupy Switch of Streaming Resource

Meaning:Occupy Switch of Streaming Resource

Value range:NO(NO),YES(YES)

Explanation:The general principle of priority-based streaming resource preemption is as follows: When the PDCHs are insufficient, the streaming service with high priority can preempt the PDCHs that are seized by the streaming service with low priority. After the PDCH resource preemption, the streaming service with low priority initiates the QoS negotiation with the SGSN. If the QoS negotiation fails, or if all the PDCHs are preempted, then the streaming service with low priority is disrupted.

The LMT provides the parameter that specifies whether to enable the priority-based streaming resource preemption. The function is enabled by default.

Default value:None
CROCALTMTHRD Cross Call Detect Time Threshold

Meaning:Cross Call Detect Time Threshold

Value range:1~255

Explanation:The crosstalk detection is implemented through the cooperation of the TC and BTS. The call flag is written into idle bits in the FR and EFR speech frame, then the BTS and TC check whether the call flag written in the speech frame is the same as the call flag of the current call. If the call flag written in the speech frame are different from the call flag of the current call for many times within the check threshold of call confusion, then the BTS or TC infers that a crosstalk exists. They send a message to the BSC, the BSC then record the information about the current call, including the ABIS, ATER, BTS, cell, carrier frequency, channel, and TC, to the log.

Default value:None

After the actual symbol offset information is collected, the BTS is adjusted on the basis of the offset calculated with the following formula: (Actual symbol offset - Reference BTS offset) - Symbol Offset (the parameter value).The symbol offset of the BSC plus the symbol offset of the BTS is less than 625 bit. Example

1. Set location code to 021, country code to 86, MCC to 460, and MNC to 01:
MOD BSCBASIC: AreaCode=021, CC=86, MCC="460", MNC="01";
2. Set service mode to A over IP mode:
MOD BSCBASIC: ServiceMode=AIP;
