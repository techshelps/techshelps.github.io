Add SGSN(ADD SGSN)

Function

A configuration command that adds an SGSN.

Note

This command can be run only when the built-in PCU is configured in the BSC.

Parameters

ID Name Description
SGSNNAME SGSN Name

Meaning:SGSN Name

Value range:1-20 characters

Explanation: This parameter specifies the SGSN name.

Default value:None
IPADDR1 IP Address1

Meaning:IP Address1

Value range:0.0.0.0-255.255.255.255

Explanation: This parameter specifies the local address 1.

Default value:None
IPADDR2 IP Address2

Meaning:IP Address2

Value range:0.0.0.0-255.255.255.255

Explanation: This parameter specifies the local address 2.

Default value:None
IPADDR3 IP Address3

Meaning:IP Address3

Value range:0.0.0.0-255.255.255.255

Explanation: This parameter specifies the peer address 1.

Default value:None
IPADDR4 IP Address4

Meaning:IP Address4

Value range:0.0.0.0-255.255.255.255

Explanation: This parameter specifies the peer address 2.

Default value:None

Example

1. Add an SGSN (SGSN Name = sgsn; IP Address1 = 1.1.1.1):
ADD SGSN: SGSNNAME="sgsn", IPADDR1="1.1.1.1";
2. Add an SGSN (SGSN Name = sgsn2; IP Address1 = 1.1.1.1; IP Address2 = 2.2.2.2):
ADD SGSN: SGSNNAME="sgsn2", IPADDR1="1.1.1.1", IPADDR3="2.2.2.2";
