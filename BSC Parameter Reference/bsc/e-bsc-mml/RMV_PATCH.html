Remove Patch(RMV PATCH)

Function

An OM command that removes the patch at the level of the entire NE.

Note

1. After the hot patch is removed, the status of the patch changes to Idle, and you cannot query the information on the boards.

2. The command removes the hot patch files on both the GOMU/GBAM and the boards.

Parameters

None.

Example

Remove patch:
RMV PATCH:;
