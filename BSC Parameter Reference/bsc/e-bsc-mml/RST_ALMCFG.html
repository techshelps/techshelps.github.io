Reset Alarm Configuration(RST ALMCFG)

Function

A configuration command that is used to reset the alarm configuration to the default value based on the alarm ID. The reset configuration includes Alarm name,Alarm shield flag and To alarm box flag.

Alarm Name: You can change the name of the user-defined alarm (environment alarm). This alarm can be used to restore the user-defined alarm to the default name.

Shield Flag: whether to shield the alarm.

To Alarm Box Flag: whether to report the alarm to the alarm box.

Note

If you query the configuration corresponding to the specified alarm ID after running this command, the default configuration is obtained, which is not modified.

Parameters

ID Name Description
AID Alarm ID

Meaning: ID of the alarm to be cleared

Value range: 1-65533

Explanation: This parameter is mandatory. You can specify only one alarm ID at a time.

Default value: None

Example

To restore alarm configuration record with Alarm ID of 801, execute the following command: 
RST ALMCFG: AID=801;
