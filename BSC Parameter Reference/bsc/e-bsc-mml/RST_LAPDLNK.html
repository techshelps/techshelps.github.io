Reset LAPD Link(RST LAPDLNK)

Function

An OM command that resets a specified LAPD link.

Note

When resetting a specified LAPD link, the system clears the message buffer of the link and reestablishes the link with the peer end. This may cause call drops on the link, so perform this operation with caution.

Parameters

ID Name Description
CESNO CES No.

Meaning: logical link number

Value range: 0-12287

Explanation: A key field of an LAPD link that indicates the LAPD link to be reset. You can obtain the field through the LST LAPDLNK command.

Default value: None

Example

Reset an LAPD link (CES No. = 0):
RST LAPDLNK: CESNO=0;
