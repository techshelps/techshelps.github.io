Modify BTS Alarm Attributes(MOD BTSALM)

Function

A configuration command that modifies the alarm parameters of the BTS. In this command, the following parameters uniquely identify the BTS:

BTS Index or BTS Name

Note

1. The parameters of the BTS3006C and BTS3002E that can be changed include Alarm Shield Flag 1 through Alarm Shield Flag 5, Extended Switch of Alarm, and Alarm Voltage Definition 1 through Alarm Voltage Definition 5.

2. The parameters of the BTS30, BTS312, BTS3002C, BTS3012, BTS3012AE, BTS3012II, DBS3900 GSM, BTS3900 GSM, BTS3900A GSM, DBS3036, BTS3036, BTS3036A or BTS3900E GSM that can be changed include Alarm Shield Flag 1 through Alarm Shield Flag 3, Extended Switch of Alarm, and Alarm Voltage Definition 1 through Alarm Voltage Definition 3.

3. The parameters of the other BTSs that can be changed include Alarm Shield Flag 1 through Alarm Shield Flag 3.

Parameters

Meaning:Index Type

Value range:BYNAME(By Name),BYIDX(By Index)

Explanation:Index type.

Default value:None

Meaning:BTS Index

Value range:0~2047

Explanation:This parameter specifies the index number of a BTS. Each BTS is numbered uniquely in a BSC.

Meaning:BTS Name

Value range:1-32 characters

Explanation:This parameter specifies the name of a BTS. The name of each BTS is unique in a BSC. Excluding the invalid characters ';',',','"','''','=','\','%%','+++','&','#'.

Example

Suppose the serving BTS is the BTS30, modify the alarm parameters of the BTS (Index Type = BYIDX(By Index); BTS Index = 0; BTS Type = BTS30; Alarm Shield Flag 1 = 1; Alarm Shield Flag 2 = 2; Alarm Shield Flag 3 = 3; Extended Switch of Alarm = YES(YES); Alarm Voltage Definition 1 = 4; Alarm Voltage Definition 2 = 5; Alarm Voltage Definition 3 = 6):
MOD BTSALM: IDXTYPE=BYIDX, BTSIDX=0, BTSTYPE=BTS30, AlmFlag1=1, AlmFlag2=2, AlmFlag3=3, EXTFlag1=YES, AVD6=4, AVD7=5, AVD8=6;
