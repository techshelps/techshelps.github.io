List Commands(LST CMDS)

Function

An OM command that queries the authorized commands that the operator can execute.

Note

None.

Parameters

None.

Example

1. Query the commands that can be executed by the operator(admin):
LST CMDS:;

The result is as follows:

%%LST CMDS:;%%
RETCODE = 0  Execution succeeded

All commands can be executed          
         
--- END

2. Query the commands that can be executed by the operator(other op):
LST CMDS:;
LST CMDS:;
The result is as follows:
+++    HUAWEI        2008-01-01 20:59:09
O&M    #1659
%%lst cmds:;%%
RETCODE = 0 Execution succeeded

Operator can execute the following command
