ADD TRX(ADD TRX)

Function

A configuration command that adds a TRX. The following key parameters must be determined after negotiation with the peer BTS:

TRX index

TRX Board Type

Note

1. If the BTS is in BM/TC separated configuration mode, the parameter TRX Board Type is mandatory. If the specified TRX board is installed, the setting of the parameter TRX Board Type must be consistent with the type of the installed TRX board. For a BTS that is not in BM/TC separated configuration mode, you do not need to specify the type of the TRX board.

2. The BTS3012 and the BTS3012II support combined cabinets and cabinet groups.

3. For the TRX boards of the DBS3900 GSM, BTS3900 GSM, BTS3900A GSM, DBS3036, BTS3036, BTS3036A, BTS3900E GSM, you need to set attributes of the RXU chain, or you can use ADD RXUBRD command to add a RXU board and use ADD TRX command to bind the logical TRX to the RXU board.

4. To add a TRX in different cabinet groups, you need add an internal connection between BTSs.

5. If a great number of TRXs are configured, you need to add slave chains connected to the BSC or the BTS.

Example

1. Add a TRX to the BTS that support separate mode (Index Type = BYIDX(By Index); Cell Index = 7; TRX Board No. = 0; TRX Board Type = QTRU(QTRU); TRX Board Pass No. = 1; Freq. = 56):
ADD TRX: IDXTYPE=BYIDX, CELLIDX=7, TRXBN=0, TRXTP=QTRU, TRXPN2=1, FREQ=56;
2. Add a TRX to the BTS that does not support separate mode (Index Type = BYIDX(By Index); Cell Index = 7; TRX Board No. = 0; Freq. = 58):
ADD TRX: IDXTYPE=BYIDX, CELLIDX=7, TRXBN=1, FREQ=58;
