Set Alarm Box Screen Severity(SET ALMSCRN)

Function

Use this command to set the severity levels of the alarms to be reported to the alarm box.

Note

The alarms satisfying the following conditions simultaneously can be reported to the alarm box:

- The alarm severity levels allowed for reporting are set by the SET ALMSCRN command

- The alarm parameter [Report Flag] in the SET BOXRPT command is set to RPT

Parameters

ID Name Description
ALVL Alarm Severity

Meaning: alarm severity

Value range: Critical(Critical Alarm), Major(Major Alarm), Minor(Minor Alarm), Warning(Warning)

Explanation: This parameter specifies the lowest alarm severity, at which alarms are reported to the alarm box.

Default value: None

Example

To set the severity levels of the alarms to MAJOR, execute the following command:
SET ALMSCRN: ALVL=MAJOR;
