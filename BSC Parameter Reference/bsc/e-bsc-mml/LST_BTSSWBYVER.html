List BTS Software by Version(LST BTSSWBYVER)

Function

An OM command that queries the BTS software downloaded to the BSC.

To query all the BTS software under the current BSC, do not set any parameter.

To query the BTS software of a certain version, set "Version".

Note

1. Only the BTS software positioned in the directory /BSC6000/btsswm/ can be queried.

2. The BTS software must be positioned in the folder named by version that is in the directory /BSC6000/btsswm/.

3. The BTS software directly positioned in the directory /BSC6000/btsswm/ is not displayed.

Parameters

ID Name Description
VER Version

Meaning: BTS software version

Value range: 1-32 characters

Explanation: This parameter specifies the BTS software version to be queried.

Default value: None
