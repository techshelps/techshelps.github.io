Set OIU Sub-board Loopback Mode(SET OPTSUBBRDLOP)

Function

An OM command that sets the loopback mode of the OIU sub-board.

Note

This command may disrupt the services on the sub-board to be looped back. Do not use this command during routine maintenance.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board in a subrack.

Default value: None
SUBBRDN Sub-board No.

Meaning: sub-board number

Value range: 0

Explanation: The sub-board number uniquely identifies a sub-board on a board.

Default value: 0
MODE Loopback mode

Meaning: sub-board loopback mode

Value range: NON(No loopback), LINE(RS line loopback), SYSTEM(RS system loopback)

Explanation:

1. None indicates that no loopback is performed on the sub-board.

2. LINE indicates that the loopback mode of the sub-board is RS line loopback.

3. SYSTEM indicates that the loopback mode of the sub-board is RS system loopback.

Default value: None

Example

Set the loopback mode to line loopback (Subrack No. = 0; Slot No. = 16; Sub-board No. = 0):
SET OPTSUBBRDLOP: SRN=0, SN=16,SUBBRDN=0, MODE=LINE;
