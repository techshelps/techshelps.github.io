Reset Board(RST BRD)

Function

An OM command that resets a specified BSC board.

Note

1. This command leads to service disruption. Therefore, run this command with caution.

2. The specified board must be configured and in operation, and the Operate State of the board must be Enabled. You can query the status of the current board by running the DSP BRDSTAT command.

3. This command is invalid for the OMU.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None

Example

Reset the GGCU in slot 12 of subrack 0:
RST BRD:SRN=0,SN=12;
