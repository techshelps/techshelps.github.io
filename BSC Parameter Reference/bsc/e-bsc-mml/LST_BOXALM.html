List Active Alarms Triggering Alarm Box(LST BOXALM)

Function

List the driving alarm of the current alarm box After the command is executed, all levels information of the driving alarm is displayed.

Note

If no audible alarm is triggered on the alarm box, the number of returned active alarms is 0.

Parameters

ID Name Description
BOXLOC Alarm Box Position

Meaning: position of the alarm box

Value range: loc(Local), omu(OMU)

Explanation:

1. The value loc(Local) indicates that the alarm box is connected to the serial port on the local PC; the value omu(OMU) indicates that the alarm box is connected to the OMU.

2. This parameter specifies the position of the alarm box, and is used to query the record of the active alarms that trigger an audible alarm on the alarm box.

Default value: loc(Local)

Example

1.List the driving alarm of the current alarm box  
LST BOXALM:BOXLOC=loc;
