Modify HW III Power Control Parameters of Cell(MOD CELLPWRHW3)

Function

A configuration command that modifies Huawei third generation power control parameters of the cell.

Note

None

HW3SAICTHREDAPDTVALUE HW3 threshold Adjust for SAIC

Meaning:HW3 threshold Adjust for SAIC

Value range:0~4

Explanation:This parameter specifies the quality threshold adjustment of downlink signals in Huawei II power control algorithm when the MS supports SAIC. On the network side, if power control is performed on the SAIC-supportive MS, the threshold must be lowered to reduce the transmit power of the BTS and thus reduce the interference of the entire network. At this time, for both power control downlink good-quality and poor-quality thresholds, the adjustment value must be set higher than the configured parameters.

Default value:None

HW3PWRCTRLOPTIMIZEDEN HwIII Power Control Optimized Enable

Meaning:HwIII Power Control Optimized Enable

Value range:NO(No),YES(Yes)

Explanation: This parameter specifies whether to enable the Huawei III power control optimization algorithm. If the parameter is set to YES(YES), the Huawei III power control optimization algorithm is enabled, thus improving the power control performance. If the parameter is set to NO(NO), the original Huawei power control algorithm is used. It is recommended that the parameter be set to YES(YES).

Default value:None

Default value:None

Example

MOD CELLPWRHW3: IDXTYPE=BYIDX, CELLIDX=0, MRMISSNUM=5, ULADJPRD=3, DLADJPRD=3, SDMRCUTNUM=1, TCHMRCUTNUM=4, DLREXLEVEXPFLTLEN=4, DLREXQUALEXPFLTLEN=4, DLREXLEVSLDWINDOW=1, DLREXQUALSLDWINDOW=1, DLREXLEVADJFCTR=6, DLREXQUALADJFCTR=6, DLREXLEVHIGHTHRED=35, DLREXLEVLOWTHRED=25, DLFSREXQUALHIGHTHRED=22, DLFSREXQUALLOWTHRED=16, DLHSREXQUALHIGHTHRED=22, DLHSREXQUALLOWTHRED=16, DLAFSREXQUALHIGHTHRED=16, DLAFSREXQUALLOWTHRED=12, DLAHSREXQUALHIGHTHRED=8, DLAHSREXQUALLOWTHRED=8, DLMAXDOWNSTEP=4, DLMAXUPSTEP=4, ULREXLEVEXPFLTLEN=1, ULREXQUALEXPFLTLEN=1, ULREXLEVSLDWINDOW=4, ULREXQUALSLDWINDOW=6, ULREXLEVADJFCTR=3, ULREXQUALADJFCTR=2, ULREXLEVHIGHTHRED=22, ULREXLEVLOWTHRED=16, ULFSREXQUALHIGHTHRED=22, ULFSREXQUALLOWTHRED=16, ULHSREXQUALHIGHTHRED=16, ULHSREXQUALLOWTHRED=12, ULAFSREXQUALHIGHTHRED=16, ULAFSREXQUALLOWTHRED=12, ULAHSREXQUALHIGHTHRED=8, ULAHSREXQUALLOWTHRED=8, ULMAXDOWNSTEP=8, ULMAXUPSTEP=8, HOPGAIN1=0, HOPGAIN2=20, HOPGAIN3=30, HOPGAIN4=40, HOPGAIN5=43, HOPGAIN6=47, HOPGAIN7=50, HOPGAIN8=53;
