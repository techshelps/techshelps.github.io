Start Test Battery of BTS(STR BTSBATTTST)

Function

An OM command that starts the test on the battery of a BTS.

Note

1. The BSC must communicate properly with the BTS.

2. The BTS must be configured with the battery.

3. To protect the battery, conduct the test at an appropriate interval. The default interval is 14 days.

4. The duration of the simple test depends on the factory settings of the battery. Generally, the duration is 30 minutes or 1 hour.

Parameters

ID Name Description
IDXTYPE Index Type

Meaning: how to specify a BTS

Value range: BYNAME(By Name), BYIDX(By Index)

Explanation: This parameter determines whether to specify a BTS by name or by index.

Default value: None
STIDX Site Index

Meaning: BTS index

Value range: 0-2047

Explanation:

1. This parameter specifies the number of the target BTS. Each BTS is numbered uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYIDX(By Index).

Default value: None
STNAME Site Name

Meaning: BTS name

Value range: 1-32 characters

Explanation:

1. This parameter specifies the name of the target BTS. Each BTS is named uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYNAME(By Name).

Default value: None
SRN Subrack No.

Meaning: subrack number

Value range: 0-53

Explanation: This parameter specifies the number of the subrack in which the BTS power module is located. The subrack number is unique in a BTS.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-23

Explanation: This parameter specifies the number of the slot that holds the BTS power module.

Default value: None
TSTTYPE Test Type

Meaning: test mode

Value range: SIMPLETEST(Simple Test), MANUALTEST(Manual Test)

Explanation: This parameter specifies the mode of testing the battery. This parameter is mandatory.

Default value: None

Example

Start the battery test on a BTS (Index Type = BYIDX(By Index); Site Index = 0; Subrack No. = 0; Slot No. = 15; Test Type = SIMPLETEST(Simple Test)):
STR BTSBATTTST: IDXTYPE=BYIDX, STIDX=0, SRN=0, SN=15, TSTTYPE=SIMPLETEST;
