Add BSC Board(ADD BRD)

Function

A configuration command that adds a BSC board.

Note

1. Before adding a board, check whether the subrack in which the board is located exists. If it does not exist, use the MML command ADD BSCSUBRACK first to add the subrack.

2. This command cannot be used to add the following boards: GTNU, GSCU, GGCU, and GOMU. These boards are automatically added with the subrack.

3. The GDPUC/GDPUP/GDPUX/GXPUI does not support the active/standby mode.

4. Currently, the BSC supports the SONET and SDH when the optical interface board works in T1 mode; the BSC supports the SDH only when the optical interface board works in E1 mode.

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:This parameter specifies the number of the subrack where a board is located.

Default value:None
SN Slot No.

Meaning:Slot No.

Value range:0~27

Explanation:This parameter specifies the number of the slot where a board is located.

Default value:None
BT Board Type

Meaning:Board Type

Value range:GEIUB,GEIUP,GEIUT,GEIUA,GOIUB,GOIUP,GOIUT,GOIUA,GFGUA,GOGUA,GEHUB,GFGUB,GOGUB,GFGUG,GEPUG,GDPUC,GDPUP,GDPUX,GXPUT,GXPUM,GXPUI

Explanation:Logical Type.

Default value:None
WORKMODE Work Mode

Meaning:Work Mode

Value range:E1,T1

Explanation:This parameter specifies the system that the transmission port uses. The work modes are classified into E1 and T1, which are the short forms of PCM30 and PCM24 respectively. The transmission rate of an E1 link is 2,048 kbit/s and that of a T1 link is 1,544 kbit/s.

Default value:None
PORTTYPE Port Type

Meaning:Port Type

Value range:FE,GE

Explanation:This parameter specifies the FE/GE port is Megabit or Gigabit.

Default value:None
BACKTYPE Back Type

Meaning:Back Type

Value range:OUTSHARE(LoadShare Mode),OUTPEER(Active/Standby Mode),NONE(Independent Mode)

Explanation:This parameter specifies whether a board is in Independent Mode, Active/StandBy Mode, or LoadShare Mode.

Default value:None
OPTSTANDARD Optical Interface Standard

Meaning:Optical Interface Standard

Value range:SDH(SDH),SONET(SONET)

Explanation:Optical Interface Standard.

Default value:None

Example

1. Add a BSC board (Subrack No. = 0; Slot No. = 0; Board Type = GXPUM; Back Type = OUTPEER(Main/Standby Mode):
ADD BRD: SRN=0, SN=0, BT=GXPUM, BackType=OUTPEER;
2. Add a BSC board (Subrack No. = 1; Slot No. = 0; Board Type = GDPUC; Back Type = NONE(Independent Mode):
ADD BRD: SRN=1, SN=0, BT=GDPUC, BackType=NONE;
