Reset Pb Interface CIC(RST PBCIC)

Function

An OM command that resets a specified Pb-interface circuit.

Note

1. Resetting a Pb-interface circuit may affect the services on the interface.

2. If the PCU is built in the BSC, this command is not supported.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation:

1. This parameter specifies the number of a subrack.

2. A key field that identifies the state reset for a Pb-interface CIC and indicates the subrack number of the Pb-interface CIC to be reset.

Default value: None
PCU PCU No.

Meaning: PCU number corresponding to the CIC

Value range: 0-31

Explanation: A key field that identifies the state reset for a Pb-interface CIC and indicates the PCU number of the Pb-interface CIC to be reset.

Default value: None (indicating slot 0 and slot 15)
OPMODE Operation Mode

Meaning: operation mode

Value range: BYCICRANGE(BY CIC RANGE), BYCICVALUE(BY CIC VALUE)

Explanation:

1. 0: searching by CIC range; 1: searching by CIC value

2. A key field that identifies the state reset for a Pb-interface CIC and indicates the searching mode for the Pb-interface CIC to be reset.

Default value: None
STARTCIC Start CIC

Meaning: start CIC

Value range: 0-65535

Explanation:

1. The start CIC of the Pb-interface CIC range to be reset.

2. This parameter is mandatory if the searching is based on CIC range.

Default value: None
ENDCIC End CIC

Meaning: end CIC

Value range: 0-65535

Explanation:

1. The end CIC of the Pb-interface CIC range to be reset.

2. This parameter is mandatory if the searching is based on CIC range.

Default value: None
CIC CIC

Meaning: CIC number of the board

Value range: 0-65535

Explanation:

1. The number of the Pb-interface CIC to be reset.

2. This parameter is mandatory if the searching is based on a specified CIC number.

3. This parameter supports multiple values.

Default value: None

Example

1. Reset a Pb-interface circuit (Subrack No. = 0; PCU No. = 2; Operation Mode = BYCICVALUE(BY CIC VALUE); CIC = 4):
RST PBCIC: SRN=0, PCU=2, OPMODE=BYCICVALUE, CIC=4;
1. Reset the Pb-interface circuits (Subrack No. = 0; PCU No. = 2; Operation Mode = BYCICVALUE(BY CIC VALUE); CIC = 2&3):
RST PBCIC: SRN=0, PCU=2, OPMODE=BYCICVALUE, CIC=2&3;
Note:
If the subrack or PCU is not configured, or the specified CIC is not in the PCU, the MML client prompts a parameter error.
