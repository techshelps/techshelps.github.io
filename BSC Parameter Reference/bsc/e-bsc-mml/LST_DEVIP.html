List Device IP Address(LST DEVIP)

Function

An OM command that queries the data of the device IP address of a board.

To query the data of the device IP address of a board in a specific subrack, set only Subrack No..

To query the data of a specific device IP address, set Subrack No., Slot No., and Subsystem No..

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation: This parameter specifies the subrack number of an IP-based interface board.

Default value:None
SN Slot No.

Meaning:Slot No.

Value range:0~27

Explanation: This parameter specifies the slot number of an IP-based interface board.

Default value:None
SSN SubSystem No.

Meaning:SubSystem No.

Value range:0~4

Explanation: The GFGUG board value is 0~4, the other IP-based interface board value is 0.

Default value:None

Example

Query the device IP address (Subrack No. = 1; Slot No. = 26):
LST DEVIP: SRN=0, SN=26;
The result is as follows:
+++    HUAWEI        2007-12-21 21:21:16
O&M    #167
%%LST DEVIP: SRN=0, SN=26;%%
RETCODE = 0  Operation succeeded

Query the IP address of a board
----------
    Subrack No. = 0
  Slot No. = 26
Subsystem No.  =  0
  IP Address =  3.3.3.3
Subnet Mask  =  255.255.255.0
(Number of results = 1)


---    END
