Display BTS Board Extension Information(DSP BTSBRDEX)

Function

An OM command that queries the extended information of the BTS board. The available extended information is board-specific.

1. Currently you can query the extended information about the following boards: main TMU, TRX, PTRX, ETRX, RTRX, DTRX, RETR, DETR, EETR, PETR, PBU, RPBU, DPBU, PPBU, IASU and OSSU of the common BTS3x; the active/standby DTMU, DPMU, DHEU, TRU, PTU and PMU of the double-transceiver BTS.

2. The available extended information for the main TMU of the BTS3002C includes site No., board No., board type, burned chip version, logic version, chip type and DIP type.

3. The available extended information about the main TMU of the common BTS3x includes site No., board No., board type and DIP type.

4. The available extended information about the TRX in the BTS3002C includes site No., board No., board type, burned chip version, logic version, chip type, high/low end frequency, power type and modulated chip type.

5. The available extended information about the TRX in the common BTS3X includes site No., board No., board type, EDGE feature support, actual TRX type and frequency band type.

6. The available extended information about the PTRX, ETRX, RTRX, DTRX, RETR, DETR, EETR and PTTR of the common BTS3X includes site No., board No., board type, EDGE feature support, actual TRX type and frequency band type.

7. The available extended information about the common PBU, PPBU, RPBU and DPBU includes site No., board No., board type, actual PBU type and frequency band type.

8. The available extended information about the IASU includes site No., board No., board type, burned chip version, logical version and chip type.

9. The available extended information about the OSSU includes site No., board No., board type, OSSU type 1, OSSU type 2. If the BTS is the BTS3006A, additional extended information can be queried for the OSSU: equipment type, manufacturer, hardware version and software version.

10. The available extended information about the DTMU includes site No., board No., board type, DIP name, and E1-interface capability. In the case of the BTS3006C and BTS3002E, additional extended information can be queried for the DTMU: optical transmission supported or not.

11. The available extended information about the TRU includes site No., board No., board type, EDGE feature support, actual TRU type, frequency band, RF RX/TX mode and CBUS2 link number.

12. The available extended information about the DPMU includes site No., board No., board type, power supply type, manufacturer and production date.

13. The available extended information about the PMU includes site No., board No., board type, logical version, manufacturer and production date.

14. The available extended information about the PTU includes site No., board No., board type, module type, manufacturer and production date.

15. The available extended information about the DHEU includes site No., board No., board type, equipment type, manufacturer and production date.

16. The available extended information about the DRRU/DRFU of the DBS3900 GSM/BTS3900 GSM/BTS3900A GSM includes site name, board type, board No., support EDGE capability, board detail type, frequency band, receive mode, send mode and current CBUS2 link.

17. The available extended information about the BTS3900E of the BTS3900E GSM includes site name, board type, board No., support EDGE capability, board detail type, frequency band, receive mode, send mode, RF band capability and E1-interface capability.

Note

1. The communication between the MML Client and the BSC is normal.

2. The communication between the BSC and the BTS is normal.

3. The specified board in the command must support the extended information querying.

Parameters

ID Name Description
IDXTYPE Index Type

Meaning: how to specify a BTS

Value range: BYNAME(By Name), BYIDX(By Index)

Explanation: This parameter determines whether to specify a BTS by its name or index.

Default value: None
BTSIDX BTS Index

Meaning: BTS index

Value range: 0-2047

Explanation:

1. This parameter specifies the number of a BTS. Each BTS is numbered uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYIDX(By Index).

Default value: None
BTSNAME BTS Name

Meaning: BTS name

Value range: 1-32 characters

Explanation:

1. This parameter specifies the name of the target BTS. Each BTS is named uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYNAME(By Name).

Default value: None
BRDTYPE Board Type

Meaning: board type

Value range: NORMAL(Normal Board), RXU(RXU Board)

Explanation: This parameter specifies the type of a BTS board.

Default value: None
SRN Subrack No.

Meaning: subrack number

Value range: 0-53

Explanation: This parameter specifies the number of the subrack where a BTS board is located. The subrack number is unique in a site.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-23

Explanation: This parameter specifies the number of the slot in which a BTS board is located.

Default value: None
RXUIDXTYPE Index Type for RXU

Meaning: index type of the RXU

Value range: RXUN(RXU No.), RXUNAME(RXU Name), RXUSRNSN(RXU Subrack No. and Slot No.)

Explanation: RXUN(RXU No.) indicates the query based on the RXU No., the RXUNAME(RXU Name) determins the query based on the RXU name, and
RXUN RXU No.

Meaning: number of an RXU board

Value range: 0-71

Explanation: This parameter specifies the number of an RXU board.

Default value: None
RXUNAME RXU Name

Meaning: name of an RXU board

Value range: 1-32 characters

Explanation: This parameter specifies the name of an RXU board.

Default value: None
RXUSRN Subrack No.

Meaning: subrack number

Value range: 0-53

Explanation: This parameter specifies the number of the subrack where a BTS board is located. The subrack number is unique in a site.

Default value: None
RXUSN Slot No.

Meaning: slot number

Value range: 0-23

Explanation: This parameter specifies the number of the slot in which a BTS board is located.

Default value: None

Example

Query the board extended information (Index Type = BYIDX; BTS Index = 0; Board Type = NORMAL; Subrack No. = 0; Slot No. = 0):
DSP BTSBRDEX: IDXTYPE=BYIDX, BTSIDX=0, BRDTYPE=NORMAL, SRN=0, SN=0;
