List Time Zone and Daylight Saving Time(LST TZ)

Function

An OM command that queries the time zone and daylight saving time.

Note

None

Parameters

None.

Example

1. Query time zone and daylight saving time information in a non-DST situation:
LST TZ:;
The result is as follows:
+++    HUAWEI        14.06.07 03:19:04 PM
O&M    #928
%%LST TZ:;%%
RETCODE = 0  Operation succeeded

Time zone and daylight saving time information
--------------
        Time Zone  =  GMT-12:00
              DST  =  No
---    END                                                                                                                             2. Query time zone and daylight saving time information in a DST situation:
LST TZ:;
The result is as follows:
+++    HUAWEI        2007-06-14 15:20:18 DST
O&M    #931
%%LST TZ:;%%
RETCODE = 0    Operation succeeded

Time zone and daylight saving time information
--------------
                 Time Zone  =  GMT-12:00
            DST  Start Date =  the last Wednesday of March
             DST Start Time =  01:01:01
              DST End Date  =  Jan 1
              DST End Time  =  01:01:01
          DST Offset (min)  =  0

---    END

Output Description

Domain Name Description
Time Zone

GMT time zone (a positive value indicates the east time zone and a negative value indicates the west time zone)
DST Start Date

Start date of the daylight saving time (date mode only, day mode only, or hybrid mode)
DST Start Time

Start time of the daylight saving time (hh:mm:ss)
DST End Date

End date of the daylight saving time (date mode only, week mode only, or hybrid mode)
DST End Time

End time of the daylight saving time (hh:mm:ss)
DST Offset (min)

Offset of the daylight saving time in minutes (integral multiple of 30 minutes)
