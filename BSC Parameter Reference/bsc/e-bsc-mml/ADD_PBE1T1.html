Add Pb Interface E1/T1(ADD PBE1T1)

Function

A configuration command that adds an E1/T1 on the Pb interface.

Note

1. The BSC must be configured with at least one PCU.

2. The PCU must be configured on the GEIUP or GOIUP.

3. The value of Start PCIC must be the multiple of 128.

4. In E1 transmission mode, timeslots 0, 1, 2, and 3 are used for synchronization and cannot be used to carry traffic.

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:None.

Default value:None
SN Slot No.

Meaning:Slot No.

Value range:0~27

Explanation:None.

Default value:None
PN Port No.

Meaning:Port No.

Value range:0~83

Explanation:None.

Default value:None
PCUNO PCU No.

Meaning:PCU No.

Value range:0~31

Explanation:This parameter specifies the number of PCU that is connected to the E1 link on the Pb interface.

Default value:None
SPCIC Start PCIC

Meaning:Start PCIC

Value range:0~65535

Explanation:A packet circuit identity code (PCIC) uniquely identifies a timeslot that carries traffic (a timeslot used for PS services).

The transmission rate of each E1 timeslot on the Pb interface is 16 kbit/s. Thus, an E1 link on the Pb interface can be divided into 128 timeslots.

The Start PCIC corresponds to E1 timeslot 0. If the Start PCIC increases by one, E1 timeslot 1 is indicated. If the Start PCIC increases by two, E1 timeslot 2 is indicated.

This parameter can be configured and modified by users.

The 128 E1 timeslots on the Pb interface have three states: occupied, PCIC, and idle. If a timeslot is in idle state, the timeslot cannot be used. Logically, the idle state is the same as the occupied state.

Default value:None
ALLTSTYPE Timeslot Type

Meaning:Timeslot Type

Value range:ALLCIC(Set all Timeslots CIC),ALLNULL(Set all Timeslots Null)

Explanation:This parameter indicates the function and status of a timeslot. The CIC type indicates a traffic timeslot.

The NULL type indicates that this timeslot is not available.

The Semicon type indicates that this timeslot is a semi-permanent link timeslot.

The SimeTs type indicates that this timeslot is a monitoring timeslot of a BTS.

The ASL type of the A interface indicates that the timeslot is an ASL timeslot, that is, A interface signaling link timeslot (SS7 link timeslot).

The AterOML type of the Ater interface indicates that the timeslot is an Ater OML timeslot.

The AterSL type of the Ater interface indicates that the timeslot is an Ater signaling link timeslot.

The ASL type of the Ater interface indicates that the timeslot is an ASL timeslot. It is used to transmit the signaling (SS7) on the A interface.

The PbSL type of the Pb interface indicates that the timeslot is a Pb signaling timeslot.

The OML type of the Abis interface indicates that the timeslot is an OML timeslot, that is, OML timeslot of the BTS.

The RSL type of the Abis interface indicates that the timeslot is an RSL timeslot, that is, signaling link timeslot of TRXs.

The Sime type of the Abis interface indicates that the timeslot is a Sime timeslot, that is, a monitoring timeslot.

The Idle typeTS of the Abis interface indicates that the timeslot is an idle timeslot,used for testing the interference of the idle channels.

The Chan type of the Abis interface indicates that the timeslot is a traffic channel timeslot of TRXs.

Default value:ALLCIC
TNMODE Transmission Mode

Meaning:Transmission Mode

Value range:TRRS(Terrestrial Transmission),SATEL(Satellite Transmission)

Explanation:This parameter specifies the transmission mode of the E1 link on the Pb interface. The E1 link can either work in terrestrial transmission mode or in satellite transmission mode.

Satellite transmission is generally used in the areas that are inconvenient for terrestrial transmission, such as deserts or lakes.

Default value:None

Example

1. Add an E1/T1 link on the Pb interface (Subrack No. = 0; Slot No. =24; Port No. = 0; PCU No. = 0; Start PCIC = 256; Timeslot Type = ALLCIC(Set all Timeslots CIC)):
ADD PBE1T1: SRN=0, SN=24, PN=0, PCUNO=0, SPCIC=256, ALLTSTYPE=ALLCIC;
