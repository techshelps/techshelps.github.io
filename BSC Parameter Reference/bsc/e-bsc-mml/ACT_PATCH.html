Activate Patch(ACT PATCH)

Function

An OM command that activates the hot patch files loaded to the patch area in the board memory.

Note

1. After a hot patch is activated, the status of the hot patch changes to Activated. The hot patch status of the corresponding boards also changes. Run the DSP PATCH command to query the hot patch status.

2. The prerequisite for the successful activation of the hot patch file on each board is that the hot patch file is already loaded to the hot patch area in the board memory.

Parameters

ID Name Description
PV Patch version

Meaning: patch version number

Value range: 1-63 characters

Explanation: This parameter is a character string, for example, BSC6000V900R003C01SP01.

Default value: None

Example

Activate patch BSC6000V900R008C01SP01:
ACT PATCH:PV="BSC6000V900R008C01SP01";
