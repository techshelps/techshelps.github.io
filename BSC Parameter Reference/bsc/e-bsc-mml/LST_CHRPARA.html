List The Configuration of CHR(LST CHRPARA)

Function

An OM command that queries the CS-domain CHR configuration parameters or the PS-domain CHR configuration parameters.

Note

The MML client must communicate properly with the BSC.

Parameters

ID Name Description
CHRTYPE CHR Type

Meaning: CHR type

Value range: CS(CS CHR), PS(PS CHR)

Explanation: This parameter specifies the type of the CHR object. CS(CS CHR) indicates that the CHR objects of the CS domain are queried; PS(PS CHR) indicates that the CHR objects of the PS domain are queried.

Default value: None

Example

1. Query the CS-domain CHR configuration parameters:
LST CHRPARA: CHRTYPE=CS;
The result is as follows:
