Add Subrack-OPC Mapping(ADD SROPCMAP)

Function

A configuration command that adds a mapping between the subrack and the originating signaling point.

Note

1. Only the BM subrack can be bound with the originating signaling point.

2. One BM subrack can be bound with only one originating signaling point.

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:This parameter specifies the number of the subrack that is configured with the OSP.

Default value:None
OPC OPC Code

Meaning:OPC Code

Value range:1~16777215

Explanation:1.In the SS7 network, each SP has a unique signaling point code called the local signaling point. The OPC corresponds to the local signaling point specified in the signaling network.

2.This parameter can be displayed in decimal, hexadecimal, and binary mode. By default, this parameter is displayed in decimal mode. The hexadecimal format is H' and the binary format is B'.

Default value:None

Example

Add a mapping between the subrack and the originating signaling point (Subrack No. = 0; OPC Code = H'B7):
ADD SROPCMAP: SRN=0, OPC=H'B7;
