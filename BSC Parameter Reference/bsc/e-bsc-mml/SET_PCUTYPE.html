Set PCU Type(SET PCUTYPE)

Function

A configuration command that modifies the type of the PCU.

Note

1. Before switching to built-in PCU mode from external PCU mode, ensure that no Pb interface boards, external PCUs, or dynamic PDCHs exist.

2. Before switching to external PCU mode from built-in PCU mode, ensure that no Gb interface boards, NSE, GDPUP boards, SGSN, or routes associated with the Gb interface exist.

Parameters

ID Name Description
TYPE PCU Type

Meaning:PCU Type

Value range:OUTER(Outer PCU),INNER(Inner PCU)

Explanation: This parameter specifies the type of the PCU. Two types of PCU are available: the built-in PCU and external PCU.

The built-in PCU provides PS services through the BSC6000.

The external PCU provides PS services through an independent PCU NE outside the BSC.

Default value:None

Example

1. Set the PCU type (PCU Type = OUTER(Outer PCU)):
SET PCUTYPE: TYPE=OUTER;
2. Set the PCU type (PCU Type = INNER(Inner PCU)):
SET PCUTYPE: TYPE=INNER;
