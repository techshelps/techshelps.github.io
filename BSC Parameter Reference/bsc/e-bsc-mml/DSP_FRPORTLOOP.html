Display FR Port Loopback Statistics(DSP FRPORTLOOP)

Function

An OM command that queries the statistics of the FR port loopback.

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation:

1. This parameter specifies the subrack number of the FR port.

2. The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation:

1. This parameter specifies the slot number of the FR port.

2. The subrack number and slot number uniquely identify a board.

Default value: None
PN Port No.

Meaning: port number

Value range: 0-31

Explanation: This parameter specifies the number of the FR port.

Default value: None
BCID BC Id

Meaning: BC ID

Value range: 0-255

Explanation: This parameter specifies the BC ID of the FR.

Default value: None

Example

Query the statistics of the FR port loopback (Subrack No. = 0; Slot No. = 22; Port No. = 0; BC Id = 0):
DSP FRPORTLOOP: SRN=0, SN=22, PN=0, BCID=0;
The result is as follows:
+++    HUAWEI        2008-07-18 14:14:17
O&M    #7276
%%DSP FRPORTLOOP: SRN=0, SN=22, PN=0, BCID=0;%%
RETCODE = 0  Operation succeeded

Query FR port loopback statistics
------------------
                 Number of Sent Packets  =  0
Number of Packets with Correct Loopback  =  0
              FR Port Loopback Duration  =  0
(Number of results = 1)

---    END

Output Description

Domain Name Description
Number of Sent Packets

Number of packets sent on the FR port
Number of Packets with Correct Loopback

Number of packets with correct loopback on the FR port. If the number of packets with correct loopback equals that of sent packets, it indicates that no packet is lost; otherwise, packet loss occurs.
FR Port Loopback Duration

Duration of the loopback on the FR port
