Modify Basic Parameters of Cell Power Control(MOD CELLPWRBASIC)

Function

A configuration command that modifies the power control basic parameters of the cell.

Note

When UL RX_LEV Upper Threshold is modified, the BSC simultaneously modifies AMR UL RX_LEV Upper Threshold and UL Expected Level at HO Access accordingly.

Parameters

ID Name Description
IDXTYPE Index Type

Meaning:Index Type

Value range:BYNAME(By Name),BYIDX(By Index)

Explanation: Index Type.

Default value:None
CELLIDX Cell Index

Meaning:Cell Index

Value range:0~2047

Explanation: This parameter indicates the source cell index. The source cell index must be unique.

Default value:None
CELLNAME Cell Name

Meaning:Cell Name

Value range:1-32 characters

Explanation: This parameter indicates the name of the source cell. The name of each cell under one BSC must be unique.

Default value:None
PCADJPERIOD PC Interval

Meaning:PC Interval

Value range:1~15

Explanation:This parameter specifies the minimum time interval between two continuous power control commands.

Default value:None
ULSSHIGHTHRED UL RX_LEV Upper Threshold

Meaning:UL RX_LEV Upper Threshold

Value range:0~63

Explanation:The power control step is calculated based on the signal level. The signal level has an upper threshold and a lower threshold. Power control is not performed if the signal level is between the upper threshold and the lower threshold. Power control is performed only when the signal level exceeds the upper threshold or is below the lower threshold.

The level values 0 through 63 map to -110 dBm to -47 dBm.

Default value:None
ULSSLOWTHRED UL RX_LEV Lower Threshold

Meaning:UL RX_LEV Lower Threshold

Value range:0~63

Explanation:The power control step is calculated based on the signal level. The signal level has an upper threshold and a lower threshold. Power control is not performed if the signal level is between the upper threshold and the lower threshold. Power control is performed only when the signal level exceeds the upper threshold or is below the lower threshold.

The level values 0 through 63 map to -110 dBm to -47 dBm.

Default value:None
ULQHIGHTHRED UL Qual. Upper Threshold

Meaning:UL Qual. Upper Threshold

Value range:0~7

Explanation:When the power control step is calculated based on the signal quality, the upper threshold and the lower threshold of the quality zone are set. When the signal quality exceeds the upper threshold or is below the lower threshold, power control is performed. This parameter specifies the upper threshold of the uplink quality for power control.

The mapping between the BER and the quality level is as follows:

Level 0: BER smaller than 0.2%

Level 1: BER ranges from 0.2% to 0.4%

Level 2: BER ranges from 0.4% to 0.8%

Level 3: BER ranges from 0.8% to 1.6%

Level 4: BER ranges from 1.6% to 3.2%

Level 5: BER ranges from 3.2% to 6.4%

Level 6: BER ranges from 6.4% to 12.8%

Level 7: BER greater than 12.8%.

Default value:None

Meaning:AMR SACCH Downlink Power Upgrade

Value range:0~15

Explanation:This parameter is configured for SACCH power control. When the DSP sends SACCH frames, the power control level is raised by N according to this configuration.

Default value:None

Meaning:Power Forecast Allowed

Explanation:This parameter is used to evaluate the downlink power in assignment procedure. The Combiner Loss is proportional to the downlink power.

Value range:NO(NO),YES(YES)

Explanation:None.

Example

Modify the power control basic parameters of the cell (Index Type = BYIDX(By Index); Cell Index = 0):
MOD CELLPWRBASIC: IDXTYPE=BYIDX, CELLIDX=0, PCADJPERIOD=3, ULSSHIGHTHRED=35, ULSSLOWTHRED=25, ULQHIGHTHRED=0, ULQLOWTHRED=2, DLSSHIGHTHRED=35, DLSSLOWTHRED=25, DLQHIGHTHRED=0, DLQLOWTHRED=2, PWRCTRLSW=HW2, AMRSADLUPGRADE=0;
