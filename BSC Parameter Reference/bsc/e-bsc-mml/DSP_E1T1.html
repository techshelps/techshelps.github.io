Display E1/T1 Port Attributes(DSP E1T1)

Function

An OM command that queries the properties of the E1/T1 port.

Note

None.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board in a subrack.

Default value: None
PN Port No.

Meaning: port number

Value range: 0-31

Explanation: The port number uniquely identifies a port on a board.

Default value: None

Example

1. Query properties of the port (Subrack No. = 0; Slot No. = 14; Port No. = 0):
DSP E1T1: SRN=0,SN=14,PN=0;
The result is as follows:

O&M    #50314

Output Description

Domain Name Description
Work Mode

Work mode (value range: E1, T1)
TX Frame Format

Format of transmitted frames (value range in E1 transmission mode: DOUBLE_FRAME and CRC4_MULTIFRAME; value range in T1 transmission mode: SUPER_FRAME and EXTENDED_SUPER_FRAME)
RX Frame Format

Format of received frames (value range in E1 transmission mode: DOUBLE_FRAME and CRC4_MULTIFRAME; value range in T1 transmission mode: SUPER_FRAME and EXTENDED_SUPER_FRAME)
TX Line Code

Code of TX line (value range in E1 transmission mode: HDB3, AMI; value range in T1 transmission mode: B8ZS, AMI)
RX Line Code

Code of RX line (value range in E1 transmission mode: HDB3, AMI; value range in T1 transmission mode: B8ZS, AMI)
Loopback Status

Loopback status (value range: NONE, REMOTE, LOCAL, PAYLOAD, SINGLE CHANNEL LOCAL, TDM REMOTE)
System Data Rate[kbps]

System data rate (value range: 2048, 8192, INVALID)
System Clock[kHz]

System clock frequency (value range: 2048, 8192, INVALID)
System Multiplex Mode

System multiplex mode (value range: MULTIPLEXED, NONE MULTIPLEXED)
Interleave Mode

Interleave mode (valid only in multiplexing mode; value range: BIT INTERLEAVE, BYTE INTERLEAVE, INSIGNIFICANT)
Line Impedance[ohm]

Line impedance (value range: 75, 120, 100, IMPEDANCE ERROR)
Communication Distance

Communication distance (value range: SHORT HAUL, LONG HAUL)
Clock Mode

Clock mode (value range: MASTER, SLAVE)
Available Status

Available status of the port (value range: Normal, Fault, Intest, Degraded)
Alarm Report Flag

Alarm report flag of the port. (value range: DISABLED, ENABLED)
Sabit4 Value

Sa4 bit value transmitted (valid only in E1 transmission mode; value range: 0, 1)
Sabit5 Value

Sa5 bit value transmitted (valid only in E1 transmission mode; value range: 0, 1)
Sabit6 Value

Sa6 bit value transmitted (valid only in E1 transmission mode; value range: 0, 1)
Sabit7 Value

Sa7 bit value transmitted (valid only in E1 transmission mode; value range: 0, 1)
Sabit8 Value

Sa8 bit value transmitted (valid only in E1 transmission mode; value range: 0, 1)
