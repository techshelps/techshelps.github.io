Add BSC Subrack(ADD BSCSUBRACK)

Function

A configuration command that adds a BSC subrack.

Note

1. Before adding a BSC subrack, check whether the cabinet where this subrack is located exists. If the cabinet does not exist, run the ADD BSCCAB command to add a cabinet.

2. Only subracks 1 to 11 can be added to the BSC; subrack 0 is configured by default.

3. When adding a BSC subrack, the GTNU and GSCU are added by default.

4. When a cabinet is configured remotely, you cannot add a local GTCS.

5. If a remote GTCS is added, this command can be used to set whether the subrack is a central subrack.

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:1~11

Explanation:This parameter specifies the number of a subrack.

Default value:None
SRT Subrack Type

Meaning:Subrack Type

Value range:GEPS,GTCS

Explanation:This parameter specifies the logical type of a subrack. Subracks are classified into three logical types: GMPS, GEPS, and GTCS.

Default value:None
ISCENTRAL Is Remote Central

Meaning:Is Remote Central

Value range:NO(No),YES(Yes)

Explanation:This parameter specifies whether a subrack is the main one.

Default value:NO

Example

1. Add a BSC subrack (Subrack No. = 1; Subrack Type = GEPS):
ADD BSCSUBRACK: SRN=1, SRT=GEPS;
2. Add a BSC subrack (Subrack No. = 3; Subrack Type = GTCS; Is Remote Central = No(No)):
ADD BSCSUBRACK: SRN=3, SRT=GTCS, IsCentral=NO;
