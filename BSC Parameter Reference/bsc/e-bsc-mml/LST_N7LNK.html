List SS7 Signaling Link(LST N7LNK)

Function

An OM command that queries the configuration of the SS7 signaling link.

To query all the SS7 signaling links in a specific BM, set BM Subrack No. and Service Mode.

To query a specific SS7 signaling link, set BM Subrack No., Service Mode, and MTP3 Link No.. When the Service Type is set to TOGETHER(BM/TC Together), set A Interface Slot No..

Note

None

Parameters

ID Name Description
BMSRN BM Subrack No.

Meaning:BM Subrack No.

Value range:0~11

Explanation: Mtp2FramNo.

Default value:None
SERVICEMODE Service Mode

Meaning:Service Mode

Value range:SEPARATE(BM/TC Separate),TOGETHER(BM/TC Together)

Explanation: Index Type.

Default value:None
ASN A Interface Slot No.

Meaning:A Interface Slot No.

Value range:0~27

Explanation: This parameter specifies the number of the slot where the A interface board (configured with MTP2LINK) is located.

Default value:None
LINKNO MTP3 Link No.

Meaning:MTP3 Link No.

Value range:0~31

Explanation: LinkNo.

Default value:None
