Inhibit MTP3 Link(INH MTP3LNK)

Function

An OM command that inhibits a specified MTP signaling link.

Note

1. Inhibiting an MTP link enables the link to carry signaling services, so perform the operation with caution.

2. If the MTP link to be inhibited is the only available link that leads to a certain DSP, this operation fails.

Parameters

ID Name Description
SIGLKSX Signal linkset index

Meaning: signaling link set identifier

Value range: 0-255

Explanation: One of the key fields of an MTP link that indicates the link set of the MTP link to be inhibited. You can obtain the field through the LST N7LNK command.

Default value: None
SIGSLC Signal link code

Meaning: signaling link identifier

Value range: 0-15

Explanation: One of the key fields of an MTP3 link that indicates the link set of the MTP link to be inhibited. You can obtain the field through the LST N7LNK command.

Default value: None

Example

Inhibit an MTP3 link (Signal linkset index = 0; Signal link code = 0):
INH MTP3LNK: SIGLKSX=0,SIGSLC=0;
