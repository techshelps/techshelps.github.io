Display Ethernet Port Dynamic ARP(DSP ARP)

Function

An OM command that queries the dynamic ARP of an Ethernet port.

Note

This command applies to only the GFGUA/GFGUB/GFGUG/GOGUA/GOGUB.

Parameters

ID Name Description
SRN Subrack No.

Meaning: number of the subrack for running the command

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: number of the slot for running the command

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None
IPADDR IP address

Explanation: destination IP address

Value range: 0.0.0.0-255.255.255.255

Explanation: 0.0.0.0 indicates that all the ARP information is queried.

Default value: 0.0.0.0

Example

Query the ARP list of a board (Subrack No. = 0; Slot No. = 24):
DSP ARP: SRN=0, SN=24;
The result is as follows:
+++    HUAWEI        2007-12-17 15:44:00
O&M    #329
%%DSP ARP: SRN=0, SN=24;%%
RETCODE = 0  Operation succeeded

Query the ARP of the Ethernet ports:
---------------
IP address    MAC address        

80.1.102.65  00.e0.fc.80.a5.ae
80.0.1.129   00.e0.fc.fc.00.c0
80.0.102.65  00.e0.fc.80.a5.ae
(Number of results = 3)


---    END

Output Description

Domain Name Description
IP address

Peer device IP address after the ARP negotiation. There might be several IP addresses.
MAC address

Peer device MAC address after the ARP negotiation. There might be several MAC addresses.
