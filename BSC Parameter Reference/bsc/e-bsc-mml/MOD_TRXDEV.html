Modify TRX Device Attributes(MOD TRXDEV)

Function

A configuration command that modifies the attributes of a TRX device. In this command, the following parameters uniquely identify the TRX board:

Cell Index or Cell Name

TRX Board No.

Note

1. The maximum power level of the BTS3001C, BTS3001C+, and BTS22C is 13. The maximum power level of the BTS20 is 9, and the maximum power level of other BTSs is 10.

2. For BTS3002C, the parameter Power Type can only be set to 30W or 20W. For BTS3006C, the parameter Power Type can be set to 20W, 40W, 63W (in DCS1800 or PCS1900), or 55W (in non-DCS1800 and non-PCS1900). For BTS3001C, BTS3001C+, BTS20, and BTS22C, only the default value of the parameter Power Type is supported. Other types of BTSs support only the power type of 40 W, 60 W, or the default value.

3. For BTS3001C, BTS3001C+, BTS20, and BTS22C, the value of the parameter Shut Down Enabled cannot be changed. By default, the parameter is set to ENABLE(Enable).

4. The transmit and receive mode of non-double-transceiver BTSs cannot be changed. The transmit mode of the BTS3006C cannot be set to the default value, PBT or WBC. The receive mode of the BTS3006C cannot be set to the default value. The transmit and receive modes of the BTS3012 and the BTS3012AE cannot be set to the default value.

5. The power finetune can be supported only in the EDGE TRX (there are four at present) of the BTS3X, in the DTRU, QTRU and DDRM of the BTS3012/BTS3006C/BTS002E, in the EDGE TRX of the BTS3012 combined group, and in the RXU of the DBS3900 GSM/BTS3900 GSM/BTS3900A GSM/DBS3036/BTS3036/BTS3036A. In addition, the sum of the power finetune and the power level cannot be greater than 20 dB.

6. For BTS3002C, BTS22C, BTS20, BTS3001C, and BTS3001C+, the parameter Shut Down Enabled cannot be set to DISABLE(Disable).

7. If the cell does not support EDGE, the parameter TRX 8PSK level must be set to 0.

8. The parameter Dynamic Close TRX Amplifier Power switch can be configured only for the TRX of double-transceiver BTS. This parameter cannot be configured for the main BCCH TRX, baseband FH TRX, antenna hopping TRX, or cells including the PDCH.

9. If the DRRU is bounded with two TRXs and the power type of one TRX is 30 W, 25 W, or 12 W, this DRRU must be configured to Double Feeder_2TX + 2RX or Double Feeder_2TX + 4RX.

10. If a board is configured to Single Feeder_1TX + 2RX and bounded with two TRXs, the Power Type of the TRX cannot be set to a high power (45 W for GSM900/GSM850, and 40 W for DCS1800/PCS1900).

Parameters

ID Name Description
IDXTYPE Index Type

Meaning:Index Type

Value range:BYNAME(By Name),BYIDX(By Index)

Explanation: Index type.

Default value:None
CELLIDX Cell Index

Meaning:Cell Index

Value range:0~2047

Explanation: This parameter indicates the source cell index. The source cell index must be unique.

Default value:None
CELLNAME Cell Name

Meaning:Cell Name

Value range:1-32 characters

Explanation: This parameter indicates the name of the source cell. The name of each cell under one BSC must be unique.

Default value:None
TRXBN TRX Board No.

Meaning:TRX Board No.

Value range:0~71

Explanation: This parameter specifies the unique number of each board type in a site.

Default value:None
TRXPN TRX Board Pass No

Meaning:TRX Board Pass No

Value range:0~7

Explanation: Number of a TRX on uplink tributary 0

Value 255 indicates that no TRX is connected.

Default value:None
POWL Power Level

Meaning:Power Level

Value range:0~13

Explanation: This parameter specifies the transmit power level of the TRX. The greater this parameter is, the smaller the transmit power is.

When this parameter is set to 0, the transmit power level of the TRX is the greatest. Each time this parameter increases by one level, the transmit power reduces by 2 dB. For different types of BTSs, the value range of this parameter is different.

BTS3X: 0-10

BTS3001C: 0-13

BTS3002C: 0-10

Double-transceiver BTSs (BTS3012,BTS3012AE,BTS3006C): 0-10

DBS3900 GSM, BTS3900 GSM, BTS3900A GSM:0-10.

Default value:None
POWT Power Type

Meaning:Power Type

Value range:80W(80W),63W(63W),60W(60W),55W(55W),50W(50W),45W(45W),40W(40W),31W(31W),30W(30W),27W(27W),26W(26W),25W(25W),21W(21W),20W(20W),19W(19W),18W(18W),17W(17W),16W(16W),15W(15W),14W(14W),13_3W(13.3W),13W(13W),12_5W(12.5W),12W(12W),11_5W(11.5W),11W(11W),10W(10W),9_5W(9_5W),9W(9W),8_5W(8_5W),8W(8W),7_5W(7.5W),7W(7W),6_5W(6.5W),6W(6W),5_5W(5.5W),5W(5W),4_5W(4.5W),4W(4W),3_7W(3.7W),3_1W(3.1W),3W(3W),2_5W(2.5W),2W(2W),200mW(200mW)

Explanation: This parameter indicates the maximum power supported by a TRX. The macro BTS and the micro BTS3002C support different types of power. The default value indicates the practical power adopted by the TRX.

Default value:None
OPTL TRX Priority

Meaning:TRX Priority

Value range:L0(Level0),L1(Level1),L2(Level2),L3(Level3),L4(Level4),L5(Level5),L6(Level6),L7(Level7)

Explanation: This parameter specifies the TRX priority. It is used for Huawei II channel assignment algorithm.

Default value:None
SDFLAG Shut Down Enabled

Meaning:Shut Down Enabled

Value range:DISABLE(Disable),ENABLE(Enable)

Explanation: This parameter specifies whether to turn off the power amplifier of the TRX automatically for saving power when the BTS is powered by batteries after the external power supply is cut off.

Default value:None
TCHAJFLAG TCH Rate Adjust Allow

Meaning:TCH Rate Adjust Allow

Value range:NO(NO),YES(YES)

Explanation: This parameter specifies whether a cell can convert full rate channels to half rate channels, or convert the half rate channels to full rate channels.

If this parameter is set to Yes, the conversion is allowed; if the parameter is set to No, the conversion is not allowed. the TCHF that has been converted to the TCHH will be forcedly restored; the TCHH that has been converted to the TCHF will be forcedly restored.

This parameter also specifies whether the channel supports the dynamic adjustment priority in the channel assignment algorithm. In the channel assignment, the channels on the TRX not supporting the dynamic adjustment are assigned first, to ensure the channels on the TRX supporting the dynamic adjustment are used for dynamic adjustment. Thus, the access requests of users are satisfied.

Default value:None
PL8PSK TRX 8PSK level

Meaning:TRX 8PSK level

Value range:0~50

Explanation: This parameter specifies the power attenuation levels of the EDGE TRX. There are 50 levels, and the attenuation between levels is 0.2 dB.

The EDGE-enabled TRX transmits 8PSK signals with less power than transmits GMSK signals. Thus, this parameter needs to be set to meet the frequency requirements.

Default value:None
PWRSPNR Power Finetune

Meaning:Power Finetune

Value range:DB0(0.0dB),DB2(-0.2dB),DB4(-0.4dB),DB6(-0.6dB),DB8(-0.8dB),DB10(-1.0dB),DB12(-1.2dB),DB14(-1.4dB),DB16(-1.6dB),DB18(-1.8dB),Default(Default)

Explanation: This parameter specifies the following: Currently, when the BSC performs the static power control on the TRX, the step of increasing or reducing the power of the TRX is 2 dB. In some scenarios, the TRX has different losses if it is combined on different tributaries, and the output power difference before and after the combination is not an integral multiple of 2 dB. Thus, the cabinet top output power of the BTS cannot be adjusted in the step of 2 dB, so the TRX output power may be different from the cabinet top output power of the BTS.

Therefore, through the setting of this parameter, a finer step can be provided for adjusting the cabinet top output power of the BTS.

Default value:None
RCVMD Receive Mode

Meaning:Receive Mode

Value range:INDEPENDENT(Independent Receiver),DIVIDING(Dividing Receiver),FOURDIVERSITY(Four Diversity Receiver),MAINDIVERSITY(Main Diversity),NONE(none)

Explanation: This parameter specifies the RF receive mode of the DTRU.

The RF receive mode can be Not Support, Independent Receiver, Dividing Receiver, Four Diversity Receiver, or Main Diversity.

The BTS3012, BTS3012AE, BTS3012II, BTS3006C, and BTS3002E do not support Main Diversity.

The DBS3900 GSM and BTS3900 GSM support Four Diversity Receiver and Main Diversity.

Default value:None
SNDMD Send Mode

Meaning:Send Mode

Value range:NOCOMB(NoCombin),PBT(PBT),WBANDCOMB(WideBandCombin),DIVERSITY(DiversityTransmit),DDIVERSITY(DDiversityTransmit),DPBT(DPBT),DTIC(TransIndependOrCombin),NONE(none)

Explanation: This parameter specifies the RF transmit mode of the TRX.

The RF transmit mode can be Not Support, No Combining, Power Booster Technology, Wide Band Combining, Diversity Transmitter, DDIVERSITY, DPBT, or Transmitter Independent or Combining.

The BTS3006C and BTS3002E do not support Wide Band Combining, Power Booster Technology, DPBT, or Transmitter Independent or Combining.

The DBS3900 GSM GRRU does not support Wide Band Combining, Transmitter Independent or Combining, Power Booster Technology, or DPBT.

The BTS3900 GSM and BTS3900A GSM do not support Transmitter Independent or Wide Band Combining.

Default value:None
CPS Allow Shutdown of TRX Power Amplifier

Meaning:Allow Shutdown of TRX Power Amplifier

Value range:NO(NO),YES(YES)

Explanation: This parameter specifies whether the BSC determines to enable or disable the power amplifier of a TRX based on the traffic volume.

Default value:None
TSPWRRESERVE Time Slot Power Reserve

Meaning:Time Slot Power Reserve

Value range:0~50

Explanation:This parameter specifies the power difference allowed between the maximum output power of the multi-transceiver unit and the maximum nominal output power.

Default value:None
FREQREUSEMODE Frequency Reuse Mode

Meaning:Frequency Reuse Mode

Value range:LOOSE(LOOSE),TIGHT(TIGHT)

Explanation: This parameter specifies the mode of reusing the frequencies configured on the TRX. The frequency reuse is loose if it is in 4x3 mode; the frequency reuse is tight if it is in 1x3 or 1x1 mode.

Default value:None
PAOPTILEVEL Priority of Power Amplifier Turning Off

Meaning:Priority of Power Amplifier Turning Off

Value range:L0(Level0),L1(Level1),L2(Level2),L3(Level3),L4(Level4),L5(Level5),L6(Level6),L7(Level7)

Explanation: None.

Default value:None
OUTHOPWROVERLOADTHRESHOLD Power Overload Threshold In Ho. Out

Meaning:Power Overload Threshold In Ho. Out

Value range:0~50

Explanation: This parameter specifies the power overload threshold that is used to initiate an outgoing handover caused inadequate power. The power overload threshold is based on the maximum output power of the power amplifier.

Default value:None
INHOPWROVERLOADTHRESHOLD Power Overload Threshold In Ho. In

Meaning:Power Overload Threshold In Ho. In

Value range:0~50

Explanation: This parameter specifies the power overload threshold for an incoming handover caused by inadequate power. The power overload threshold is based on the maximum output power of the power amplifier.

Default value:None

Example

Modify the attributes of a TRX (Index Type = BYIDX(By Index); Cell Index = 0; TRX Board No. = 0; Power Level = 4; TRX Priority = L1(Level1); TCH Rate Adjust Allow = NO(NO)):
MOD TRXDEV: IDXTYPE=BYIDX, CELLIDX=0, TRXBN=0, POWL=4, OPTL=L1, TCHAJFLAG=NO;
