Set SCCP SSN Admin Status(SET SCCPSSN)

Function

A configuration command that modifies the administrative state of the SCCP destination signaling point.

Note

If the parameter SCCP SSN adminstatus is set to INHIBIT(INHIBIT), the services of the SCCP DPC become unavailable. Perform this operation with caution.

Parameters

ID Name Description
SSNINDEX SCCP SSN index

Meaning:SCCP SSN index

Value range:0~511

Explanation: SsnIndex.

Default value:None
ADMINSTATUS SCCP SSN adminstatus

Meaning:SCCP SSN adminstatus

Value range:INHIBIT(INHIBIT),UNINHIBIT(UNINHIBIT)

Explanation:Used to change the management status of SCCPSSN, to inhibit or to uninhibit.

Default value:None

Example

Set the administation status of the SCCP SSN(SCCP SSN index = 3; SCCP SSN adminstatus = INHBIT(INHBIT);):
SET SCCPSSN: SSNINDEX=3, ADMINSTATUS=INHIBIT;
