Export Inventory Management File(EXP DEVFILE)

Function

An OM command that queries the generated inventory management file. The generated inventory management file is stored in the directory \bsc6000\data\mtndata\inventory\ivt. For a BTS, the file name is W_BSC6000%N.XML with N indicating the BTS ID. For a BSC, the file name is W_BSC6000.XML.

Note

1. This command consumes certain system resources.

2. The inventory management file cannot be generated if the btsinventory.xml and bscinventory.xml files are not in the directory \bsc6000\data\mtndata\inventory\template.

3. The generation and upload of an inventory management file cannot be performed simultaneously.

Parameters

None.

Example

This command exports the inventory management file.
EXP DEVFILE:;
Note:
If btsinventory.xml or bscinventory.xml does not exist in the directory /data/mtndata/inventory/templat/, the system prompts export failure. If the inventory file is being generated, the system prompts that the file is in the progress of generation. If the inventory file is being uploaded, the system prompts that the file is in the progress of upload.
