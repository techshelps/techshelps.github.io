Display OIU Port Attributes(DSP OPT)

Function

An OM command that queries the properties of the OIU port.

Note

None.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board in a subrack.

Default value: None
SUBBRDN Sub-board No.

Meaning: sub-board number

Value range: 0

Explanation: The sub-board number uniquely identifies a sub-board on a board.

Default value: 0
PN Port No.

Meaning: port number

Value range: E1: 0-62; T1: 0-83

Explanation: The port number uniquely identifies a port on a board.

Default value: None
J2_TYPE J2 Show Type

Meaning: J2 show type

Value range: SHOWTYPE_HEX(Hex), SHOWTYPE_STRING(String)

Explanation: SHOWTYPE_HEX indicates that byte J2 is displayed in hexadecimal mode, and SHOWTYPE_STRING indicates that byte J2 is displayed in string mode.

Default value: None

Example

1. Query the properties of the port, and byte J2 byte is displayed in string mode (Subrack No. = 0; Slot No. = 18; Sub-board No. = 0; Port No. = 0):
DSP OPT: SRN=0, SN=18, PN=0, J2_TYPE=SHOWTYPE_STRING;

Domain Name Description
Port No.

Port number (value range: E1: 0-62; T1: 0-83)
Work Mode

Work mode (value range: E1, T1)
TX Frame Format

Format of transmitted frames (value range in E1 transmission mode: DOUBLE_FRAME and CRC4_MULTIFRAME; value range in T1 transmission mode: SUPER_FRAME and EXTENDED_SUPER_FRAME)
RX Frame Format

Format of received frames (value range in E1 transmission mode: DOUBLE_FRAME and CRC4_MULTIFRAME; value range in T1 transmission mode: SUPER_FRAME and EXTENDED_SUPER_FRAME)
Loopback Status

Loopback status (value range: NONE, LOW CHANNEL LINE, TRIBUTARY LINE, TRIBUTARY SYSTEM, SINGLE CHANNEL LOCAL, TDM REMOTE)
Available Status

Available status of the port (value range: Normal, Fault, Intest, Degraded)
Operation Status

Operating status of the port (value range: Enabled, Disabled)
Alarm Report Flag

Alarm report flag of the port. (value range: DISABLED, ENABLED)
Sabit4 Value

Sa4 bit value transmitted (valid only in E1 transmission mode; value range: 0, 1)
Sabit5 Value

Sa5 bit value transmitted (valid only in E1 transmission mode; value range: 0, 1)
Sabit6 Value

Sa6 bit value transmitted (valid only in E1 transmission mode; value range: 0, 1)
Sabit7 Value

Sa7 bit value transmitted (valid only in E1 transmission mode; value range: 0, 1)
Sabit8 Value

Sa8 bit value transmitted (valid only in E1 transmission mode; value range: 0, 1)
TX J2 Byte

J2 byte transmitted
Expect RX J2 Byte

Received J2 byte expected
RX J2 Byte

J2 byte received
J2 Mode

J2 byte mode
V5 Byte

V5 byte information (value range: 0-255)
