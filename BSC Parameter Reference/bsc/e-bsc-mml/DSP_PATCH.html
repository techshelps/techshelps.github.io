Display Patch of Board(DSP PATCH)

Function

An OM command that queries the board patch.

Note

The board patch cannot be queried based on the board type but the subrack number (mandatory) and slot number (optional).

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: The subrack number is used to uniquely identify a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number are used to uniquely identify a board in a subrack.

Default value: None

Example

Query the patch of the board (Subrack No. = 0; Slot No. = 6):
DSP PATCH:SRN=0,SN=6;
The returned result is as follows: 
MML% DSP PATCH:SRN=0,SN=6;
