Test Internal Speech Channel(CHK INNSPCH)

Function

An OM command that tests the internal speech channel.

Note

1. The BSC must communicate properly with the BTS.

2. The BTS must be configured with TRXs.

3. At least one MS is in conversation.

Parameters

ID Name Description
IDTYPE User Query Type

Meaning: mode of testing the internal speech channel

Value range: BYTMSI, BYIMSI, BYMSISDN, BYIMEI

Explanation:

1. This parameter specifies the mode of testing the internal speech channel.

2. When this parameter is set to BYMSISDN, it is recommended that you trace the calling party by using the called number.

3. When this parameter is set to BYTMSI or BYIMSI, you need to check the reassignment strategy on the MSC side. If the TMSI is carried, you can trace the MS by the TMSI; if the IMSI is carried, you can trace the MS by the IMSI.

4. If this parameter is set to BYIMSI, you need to check and ensure that the IMEI can be obtained on the MSC side.

Default value: None
TMSI TMSI

Meaning: TMSI

Value range: 0-4294967295

Explanation:

1. This parameter specifies the TMSI. You need to set this parameter when User Query Type is set to BYTMSI.

2. The length of the parameter must be eight digits.

Default value: None
IMSI IMSI

Meaning: IMSI

Value range: 5-15 characters

Explanation:

1. This parameter specifies the IMSI. You need to set this parameter when User Query Type is set to BYIMSI.

2. The value of this parameter ranges from 00000 to 999999999999999.

Default value: None
MSISDN MSISDN

Meaning: MSISDN

Value range: 3-16 characters

Explanation:

1. This parameter specifies the MSISDN. You need to set this parameter when User Query Type is set to BYMSISDN.

2. The value of this parameter ranges from 000 to 9999999999999999.

3. If the calling party is to be traced, the MSISDN is the called number, that is, the number that the MS calls. For example, if the called number is 12345, you need to set the MSISDN to 12345. If the called party is traced, the MSISDN is the calling number. In this case, the caller ID display function must be enabled. For example, if the caller ID displayed on the MS is 54321, you need to set the MSISDN to 54321; if the called ID displayed on the MS is 8654321, you need to set the MSISDN to 8654321.

Default value: None
IMEI IMEI

Meaning: IMEI

Value range: 8-15 characters

Explanation:

1. This parameter specifies the IMEI. You need to set this parameter when User Query Type is set to BYIMEI.

2. The value of this parameter ranges from 00000000 to 999999999999999.

Default value: None

Example

Test the internal speech channel (User Query Type = BYMSISDN; MSISDN = 13988880048):
CHK INNSPCH: IDTYPE=BYMSISDN, MSISDN="13988880048";
