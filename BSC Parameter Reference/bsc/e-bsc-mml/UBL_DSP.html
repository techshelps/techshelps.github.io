Unblock DSP(UBL DSP)

Function

An OM command that unblocks a specified DSP.

Note

None.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: This parameter specifies the subrack number that uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None
DSPN Dsp No.

Meaning: DSP number

Value range: any 22-digit number

Explanation: The subrack number, slot number, and DSP number uniquely identify a destination signaling point.

Default value: None

Example

Unblock a DSP (Subrack No. = 0; Slot No. = 10; Dsp No. = DSP0(DSP0)):
UBL DSP: SRN=0, SN=10, DSPN=DSP0-1;

Output Description

None.
