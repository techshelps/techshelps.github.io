Download BTS Software(DLD BTSSW)

Function

An OM command that queries the system to download BTS software to the operation and maintenance unit.

Note

1. The communication between the MML and the BSC is normal.

2. The BTS software exists on the FTP server.

3. The FTP runs properly.

4. By default, Dir is the root directory of the FTP server.

5. Stop the downloading of BTS software through STP BTSSWDLD.

6. Download software packaget format only support the format that HuaWei Company release,Other format not support.

Parameters

ID Name Description
IP FTPServer IP

Meaning: IP address of the FTP server

Value range: 0.0.0.0-255.255.255.255

Explanation: This parameter specifies the IP address of the FTP server.

Default value: None
USR FTPServer user

Meaning: user name of the FTP server

Value range: 1-32 characters

Explanation: This parameter specifies the user name of the FTP server.

Default value: None
PWD User password

Meaning: password of the FTP server

Value range: 1-32 characters

Explanation: This parameter specifies the password of the FTP server.

Default value: None
DIR File directory

Meaning: the directory of the software to be downloaded

Value range: 1-256 characters

Explanation: This parameter specifies the destination directory.

Default value: root directory
FN File name

Meaning: the software to be downloaded

Value range: 1-256 characters

Explanation: This parameter specifies the name of the file of the downloaded software. This file must be saved in the .zip format.

Default value: None

Example

1.  With name SWMTEST and password ******, a user logs in to the FTP server whose IP address is 10.161.25.13 and downloads the BTS software of BTS3000V100R001C07B031.zip.
DLD BTSSW: IP="10.161.25.13", USR="SWMTEST", PWD=******, FN ="BTS3000V100R001C07B031.zip";
