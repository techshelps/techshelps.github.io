Remove SGSN(RMV SGSN)

Function

A configuration command that removes an SGSN.

Note

This command can be run only when the built-in PCU is configured in the BSC.

Parameters

ID Name Description
SGSNNAME SGSN Name

Meaning:SGSN Name

Value range:1-20 characters

Explanation: This parameter specifies the SGSN name.

Default value:None

Example

Remove an SGSN (SGSN Name = sgsn):
RMV SGSN: SGSNNAME=sgsn;
