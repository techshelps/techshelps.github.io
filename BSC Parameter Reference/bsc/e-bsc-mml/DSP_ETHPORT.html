Display Ethernet Port Information(DSP ETHPORT)

Function

An OM command that queries the status and flow statistics of an Ethernet port.

Note

This command applies to the GFGUA/GFGUB/GFGUG/GOGUA/GOGUB.

Parameters

ID Name Description
SRN Subrack No.

Meaning: This parameter specifies the number of the subrack for running the command.

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: This parameter specifies the number of the slot for running the command.

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None
PN Port No.

Meaning: port number

Value range: 0-7

Explanation:

1. For the GFGUA/GFGUB/GFGUG, the value range is 0 to 7.

2. For the GOGUA/GOGUB, the value range is 0 to 1.

Default value: None

Example

Query the information about a port (Subrack No. = 0; Slot No. = 24; Port No. = 0):
DSP ETHPORT: SRN=0, SN=24, PN=0;
The result is as follows:

Output Description

Domain Name Description
Port Type

Type of the Ethernet port. Value range: FE Electrical Port, GE Electrical Port, GE Optical Port
Port Active/Inactive State

Activation status of the port. Value range: Active, Inactive
Link Status

Availability of the link. Value range: Available, Unavailable
Port Rate

Data rate at the port. Value range: 10 Mbit/s, 100 Mbit/s, 1000 Mbit/s, Auto
Duplex Mode

Duplex mode of the transmission over the Ethernet port. Value range: FULL, HALF
Auto-Negotiation Mode

Specifies whether to enable the auto-negotiation mode. Value range: ENABLE; DISABLE
Flow Control Mode

Loopback mode
