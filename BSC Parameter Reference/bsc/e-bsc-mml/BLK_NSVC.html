Block NSVC(BLK NSVC)

Function

An OM command that blocks an NSVC.

Note

Blocking or unblocking the NSVC affects the PS services. If the NSVC to be blocked is the only NSVC on the Gb interface, blocking the NSVC disrupts the PS services. If the NSVC to be blocked is one of the multiple NSVCs on the Gb interface, blocking the NSVC reduces the traffic flow on the Gb interface. Thus, run this command with caution.

Parameters

ID Name Description
NSEI NSE Id

Meaning: NSE ID

Value range: 0-65534

Explanation: This parameter specifies the ID of a network service entity.

Default value: None
NSVCI NSVC Id

Meaning: NSVC ID

Value range: 0-65534

Explanation: This parameter specifies the ID of the network service virtual connection.

Default value: None

Example

Block the NSVC (NSE Id = 0; NSVC Id = 159):
BLK NSVC: NSEI=0, NSVCI=159;
