Display SCCP Status Between Signaling Points(DSP SCCPSTATUSBTSP)

Function

An OM command that queries the reachability status between two SCCP signaling points.

Note

None.

Parameters

ID Name Description
DPCINDEX SCCP DPC index

Meaning: SCCP destination signaling point index

Value range: 0~288

Explanation: This parameter specifies the mapping between an SCCP source signaling point and an SCCP destination signaling point

Default value: None

Example

Query the status between SCCP signaling points (SCCP DPC index = 3):
DSP SCCPSTATUSBTSP: DPCINDEX=3;
The result is as follows:
+++    HUAWEI        2008-06-05 19:32:44
O&M    #7001
%%DSP SCCPSTATUSBTSP: DPCINDEX=3;%%
RETCODE = 0  Operation succeeded

Display SCCP Status Between SP Result
--------------------
DpcIndex  Dpc Status  RepDpcStatus  SccpStatus  CongestStatus

3               ALLOW    PROHIBIT      ALLOW    Normal          
(Number of results = 1)


---    END

Output Description

Domain Name Description
DpcIndex

Keyword for identifying the DPC record
Dpc Status

Indicates whether the destination signaling point is available in the signaling network. "PROHIBIT" indicates that the destination signaling point is unavailable, "ALLOW" indicates that the destination signaling point is available, and "OPERATION PROHIBIT" indicates that the destination signaling point is unavailable due to manual operations.
RepDpcStatus

Indicates whether the standby destination signaling point is available in the signaling network. "PROHIBIT" indicates that the standby destination signaling point is unavailable, "ALLOW" indicates that the destination signaling point is available, and "OPERATION PROHIBIT" indicates that the destination signaling point is unavailable due to manual operations.
SccpStatus

Indicates whether the SCCP is available in the signaling network. "PROHIBIT" indicates that the SCCP is unavailable, "ALLOW" indicates that the SCCP is available, and "OPERATION PROHIBIT" indicates that the SCCP is unavailable due to manual operations.
CongestStatus

Indicates whether the signaling point is congested. "Normal" indicates that the signaling point is not congested, and "Congestioned" indicates that the signaling point is congested.
