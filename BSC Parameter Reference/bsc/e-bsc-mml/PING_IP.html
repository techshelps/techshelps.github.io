Ping Peer IP Address(PING IP)

Function

An OM command that pings the peer IP address.

Note

1. You cannot run this command again before the previous result is returned.

2. You cannot specify the device IP address of a standby board as the source IP address.

Parameters

ID Name Description
SRN Subrack No.

Meaning: number of the subrack for running the command

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: number of the slot for running the command.

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None
DESTIP Destination IP address

Explanation: destination IP address

Value range: 0.0.0.0-255.255.255.255

Explanation: This parameter specifies the destination IP address.

Default value: None
SRCIP Source IP address

Explanation: source IP address

Value range: 0.0.0.0-255.255.255.255

Explanation: If you do not set this parameter, it indicates the local IP address.

Default value: None
TIMES Ping times

Explanation: number of sent PING packets

Value range: 1-60

Explanation: None

Default value: 5
PKGSIZE Package sizes

Meaning: PING packet size

Value range: 20-2700

Explanation:

1. Number of the PING packets sent, where the IP header and ICMP header are excluded

2 Unit: byte

Default value: 20
TIMEOUT Time out

Meaning: time expired in response to the PING packets

Value range: 1-10000

Explanation:

1. Time expired in response to the PING packets

2 Unit: millisecond

Default value: 2000
TTL TTL

Meaning: TTL (Time to Live) value of a PING packet

Value range: 1-255

Explanation:

1. Time to Live (TTL) value of a PING packet

Default value: 255
TOS TOS

Meaning: Type of Service (TOS) value in the PING packet

Value range: 0-255

Explanation:

1. Type of Service (TOS) value in the PING packet

Default value: 0

Example

Ping the peer IP address of a board (Subrack No. = 0; Slot No. = 24; Source IP address = 10.0.0.1; Destination IP address = 20.0.0.3):
PING IP: SRN=0, SN=24, DESTIP="20.0.0.3", SRCIP="10.0.0.1";
