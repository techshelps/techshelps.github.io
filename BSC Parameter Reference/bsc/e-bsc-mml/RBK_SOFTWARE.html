Roll Back Host Software(RBK SOFTWARE)

Function

An OM command that rolls back the loaded host software version.

Note

If the board version is the initial one, or if a newly downloaded version is not yet loaded to the board, the rollback fails.If software is rolled back,the whole system is reset and the services are interrupted temporarily.

Parameters

ID Name Description
RBT Rollback type

Meaning: rollback type

Value range: SOFTWARE(Rollback software only)

Explanation: This parameter indicates that only the software is rolled back.

Default value: None

Example

Roll back host software:
RBK SOFTWARE: RBT=SOFTWARE;
