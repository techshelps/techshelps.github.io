Download License File(DLD LICENSE)

Function

An OM command that downloads the specified license files on the FTP server to the GOMU/GBAM.The FTP server is the PC that is installed with LMT or M2000 server software.

Note

1. This command only downloads the license files to the GOMU/GBAM. The GOMU/GBAM still uses the original license configuration, even if the system restarts. The new license files are not activated.

2. The deciphering of the license files and ESN check are performed when you run the ACT LICENSE command. When the OMUESN changes, the license files must be simultaneously updated.

3. When the updated license file is downloaded, the NE must store two license files before the license is activated. The updated license file must not overwrite the current license file.

4. The file name entered is case sensitive and must be the same as the name of the file to be downloaded.

Parameters

ID Name Description
IP FTPServer IP

Meaning: IP address of the FTP server

Value range: 0.0.0.0-255.255.255.255

Explanations:

1. This parameter specifies the IPv4 type as xxx.xxx.xxx.xxx.

2. Generally, the FTP server refers to the PC on which the LMT is installed. The parameter usually indicates the IP address of the PC on which the LMT is installed. Sometimes, the FTP server in use is on another PC. In this case, the parameter indicates the IP address of the PC where the FTP is running.

Default value: None
USR FTPServer user

Meaning: user name used to log in to the FTP server

Value range: 0-24 characters

Explanation: 1.This parameter is a character string *******.

2.The default user name for logging in to the FTP server to which the LMT is connected is swmtest.

Default value: None
PWD User password

Meaning: user password used to log in to the FTP server

Value range: 0-24 characters

Explanation: 1.This parameter is a character string *******.

2.The default password for logging in to the FTP server to which the LMT is connected is swmtest.

Default value: None
DIR Source directory

Meaning: the FTP server directory where the files to be downloaded are stored

Value range: 1-127 characters

Explanation: This parameter is a character string *:\***\***.

Default value: None
FN File name

Meaning: license file name (including the filename extension)

Value range: 1-31 characters

Explanation: This parameter is a character string *******.

Default value: None

Example

Download license file OR8290.DAT:
DLD LICENSE:IP="80.1.102.111",USR="swmtest",PWD="swmtest",DIR="Common\SWM\license",FN="OR8290.DAT";
