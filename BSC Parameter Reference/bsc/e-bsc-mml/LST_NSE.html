List NSE(LST NSE)

Function

An OM command that queries the configuration of the network service entity (NSE).

To query the configuration of all NSEs, do not set any parameter.

To query the configuration of a specific NSE, set NSE Identifier.

Note

None

Parameters

ID Name Description
NSEI NSE Identifier

Meaning:NSE Identifier

Value range:0~65534

Explanation: This parameter specifies the identifier of a network service entity (NSE).

An NSE manages a group of NSVCs.

Default value:None

Example

1. Query all NSEs:
LST NSE:;
The result is as follows:
