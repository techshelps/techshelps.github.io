List Software Parameters of PS Domain(LST PSSOFTPARA)

Function

An OM command that queries the configuration of the PS-domain software parameters.

Note

None

Parameters

None.

Example

Query PS-domain software parameters:
LST PSSOFTPARA:;

The result is as follows:
