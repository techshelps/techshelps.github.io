Remove IP Route(RMV IPRT)

Function

A configuration command that removes an IP route.

Note

1. The route to be removed must exist.

2. If you remove the IP route of a board in active/standby mode, the IP route of the other board is also removed.

3. Removing the IP address may interrupt the communication.

Parameters

ID Name Description
RTIDX Route index

Meaning:Route index

Value range:0~65535

Explanation: This parameter specifies the index number of a route.

Default value:None

Example

Remove the IP route (Route index = 0):
RMV IPRT: RTIDX=0;
