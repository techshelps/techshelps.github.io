Add BSC EAC(ADD EAC)

Function

A configuration command that adds an EAC.

Note

1. Before adding an EAC to a subrack, check whether the subrack exists.

2. If the subrack does not exist, you can add it by using the ADD BSCSUBRACK command.

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:This parameter specifies the number of the subrack where the environment monitoring instrument is located.

Default value:None

Example

Add a BSC EAC in subrack 0:
ADD EAC: SRN=0;
