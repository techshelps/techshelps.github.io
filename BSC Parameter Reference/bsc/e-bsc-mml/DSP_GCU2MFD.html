Display the 2M Clock Frequency Deviation of GGCU(DSP GCU2MFD)

Function

An OM command that queries the frequency deviation data of the 2 MHz clock on the GGCU.

Note

The specified GGCU should be in operation.

Parameters

None.

Example

1. Query the frequency deviation data of the 2 MHz clock of the GGCU in subrack 0:
DSP GCU2MFD:;
The result is as follows:
+++    HUAWEI        2008-08-06 13:55:16
O&M    #3329
%%DSP GCU2MFD:;%%
RETCODE = 0  Operation succeeded

Display the 2M Clock Frequency Deviation of GGCU
------------------------------------------------
2M Clock Frequency Excursion[Hz]  2M Clock Frequency STATUS

0                                 Signal UnSynchronized
0                                 Signal UnSynchronized
(Number of results = 2)


---    END

Output Description

Domain Name Description
2M Clock Frequency Excursion[Hz]

2M Clock Frequency Excursion.
2M Clock Frequency STATUS

The status can be Signal Synchronized, Signal UnSynchronized, Test Unfinished, or Test Not Started.
