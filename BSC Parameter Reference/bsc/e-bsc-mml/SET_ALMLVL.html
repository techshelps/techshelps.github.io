Set Severity of Alarm(SET ALMLVL)

Function

A configuration command that sets the alarm severity by alarm ID.

Note

After the alarm severity of an alarm ID is set, the new alarm severity immediately applies to the active alarms and history alarms corresponding to the specified alarm ID.

Parameters

ID Name Description
AID Alarm ID

Meaning: alarm ID

Value range: 1-65533

Explanation: This parameter specifies the ID of the alarm to be modified.

Default value: None
ALVL Alarm Severity

Meaning: alarm severity

Value range: Critical(Critical Alarm), Major(Major Alarm), Minor(Minor Alarm), Warning(Warning)

Explanation: This parameter specifies the alarm severity. Each alarm ID corresponds to only one alarm severity.

Default value: None

Example

To set the alarm severity of the alarm configuration record with alarm ID of 802 to CRITICAL, execute the following command: 
SET ALMLVL: AID=802, ALVL=CRITICAL;
