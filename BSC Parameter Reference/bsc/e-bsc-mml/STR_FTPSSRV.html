Start FTP Server Relay(STR FTPSSRV)

Function

A configuration command that displays the status of the FTP server.

Note

1. If this runs successfully, the data in the configuration database is modified. The data can be restored after the resetting.

2. The BSC start the FTP server automatically if the GBAM or GOMU are reset.

Parameters

None.

Example

Start the FTP server relay:
STR FTPSSRV:;
