Remove Remote NSVL(RMV NSVLREMOTE)

Function

A configuration command that removes a remote NSVL.

Note

1. The remote NSVL to be removed must exist.

2. If you remove the last remote NSVL of an NSE, all cells under the NSE cannot provide data services.

Parameters

ID Name Description
NSVLI Remote NSVL Identifier

Meaning:Remote NSVL Identifier

Value range:0~1023

Explanation: This parameter specifies the NSVL identifier on the SGSN side. This parameter uniquely identifies an NSVL.

Default value:None

Example

Remove a remote NSVL (Remote NSVL Identifier = 1):
RMV NSVLREMOTE: NSVLI=1;
