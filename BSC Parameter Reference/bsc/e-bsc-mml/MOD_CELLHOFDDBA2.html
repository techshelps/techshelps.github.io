Modify Cell Handover Data FDD BA2(MOD CELLHOFDDBA2)

Function

A configuration command that modifies the FDD BA2 of the handover parameters of the cell. In this command, Cell Index or Cell Name uniquely identifies a cell.

Note

If 3GBA2 Input Tag is MAMU(Manu), each record of the DL ARFCN, scrambling code, and diversity indication must be complete to ensure the successful modification.

Parameters

ID Name Description
IDXTYPE Index type

Meaning:Index type

Value range:BYNAME(By Name),BYIDX(By Index)

Explanation: Index Type.

Default value:None
CELLIDX Cell index

Meaning:Cell index

Value range:0~2047

Explanation: This parameter indicates the number of the cell. Each cell is numbered uniquely in a BSC.

Default value:None
CELLNAME Cell name

Meaning:Cell name

Value range:1-32 characters

Explanation: This parameter indicates the name of the cell. Each cell is named uniquely in a BSC.

Default value:None
FDDBA2TAG 3GBA2 Input Tag

Meaning:3GBA2 Input Tag

Value range:AUTO(Auto),MANU(Manu)

Explanation: This parameter specifies whether manually entering information in the BA table is allowed.

Default value:None
FDDBA2ITEM NCELL NO.

Meaning:NCELL NO.

Value range:1~64

Explanation:The record number of FDD BA2.

Default value:None
FDDDLUARFAN DL UARFAN

Meaning:DL UARFAN

Value range:0~16383

Explanation: This parameter indicates a table recording the 3G neighboring cells. The table is for neighboring cell search when an MS is in the call state, and sent through system information 2,QUATER.

Default value:None
FDDSCRAMBLE Scrambling code

Meaning:Scrambling code

Value range:0~511

Explanation: This parameter indicates the scrambling code of a 3G cell. The scrambling code distinguishes MSs or cells.The code is used after spreading, so the bandwidth of a signal is not changed and signals from different sources are differentiated. Scrambling prevents several transmitters from using the same code word for spreading.

On the uplink, the scrambling differentiates MSs. On the downlink, the function differentiates cells.

Default value:None
FDDDIVERSITY Diversity

Meaning:Diversity

Value range:0~1

Explanation: This parameter is the diversity indication of a 3G cell. This parameter indicates whether the transmit diversity mode on the common channel in a cell is activated.

Default value:None

Example

Modify the FDD BA2 of the cell (Index Type = BYIDX(By Index); Cell Index = 0; 3GBA2 Input Tag = AUTO(Auto)):
MOD CELLHOFDDBA2: IDXTYPE=BYIDX, CELLIDX=0, 3GBA1TAG=AUTO;
