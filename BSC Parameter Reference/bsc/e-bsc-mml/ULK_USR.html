Unlock User Account(ULK USR)

Function

An OM command that unlocks a user account.

Note

1. This command can be applied to all the existing users (locked or unlocked).

2. For the locked user, this command unlocks the user and resets the number of password retries to zero. For the unlocked user, this command resets the number of password retries to zero.

Parameters

ID Name Description
OP Account name

Meaning: account name

Value range: 1-32 characters

Explanation: the account name to be unlocked

Default value: None

Example

Unlock the user "zhangsan" to enable this user to log in:
ULK USR:OP="zhangsan";
