Function

An OM command that queries the license file information.

To query all the license file information on the GOMU/GBAM, do not set any parameter.

To query the information of a certain type of license files or a specified file on the GOMU/GBAM, set the File Name parameter.

Note

None

Parameters

ID Name Description
FN File name

Meaning: license file name

Value range: 1-31 characters

Explanation: This parameter is a character string, for example, OR9455.DAT.

Default value: None

Example

1. Query the usage of all the license files on the GOMU/GBAM:
LST LICENSE:;
The returned result is as follows: 

+++    HUAWEI        2007-07-10 14:54:54
O&M    #91573

(Number of results = 2)


---    END

LST LICENSE:FN="License_170F_C3.dat";
