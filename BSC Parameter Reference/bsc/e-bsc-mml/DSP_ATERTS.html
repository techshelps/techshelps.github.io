Display Ater Interface Timeslot(DSP ATERTS)

Function

An OM command that queries the state of timeslots of a specified port on the Ater interface. The timeslot state on the Ater interface is of five types: occupied, failure, blocked, uninstalled and idle. All the five states are included by default if the state is not specified in the command.

Note

1. The number of timeslots that are queried at a time decides the time consumption.

2. In BM/TC combined configuration mode or if the A interface is in IP mode, the BSC does not support this operation.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation:

1. This parameter specifies the number of a subrack.

2. A key field that identifies the status query of Ater timeslots and indicates the subrack number of the Ater timeslots to be queried.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation:

1. This parameter specifies the number of a slot.

2. A key field that identifies the status query of Ater timeslots and indicates the slot number of the Ater timeslots to be queried.

Default value: None
PN Port No.

Meaning: port number

Value range: 0-83

Explanation: A key field that identifies the state query of Ater timeslots and indicates the port number of the Ater timeslots to be queried.

Default value: None
STAT State

Meaning: state of the timeslots on the Ater interface

Value range: OCCUPIED(OCCUPIED), FAILURE(FAILURE), BLOCKED(BLOCKED), UNINSTALLED(UNINSTALLED), IDLE(IDLE), ALL(ALL)

Explanation:

1. A state filtering parameter for the Ater timeslots to be queried.

2. OCCUPIED(OCCUPIED): occupied state; FAILURE(FAILURE): failure state; BLOCKED: blocked state; UNINSTALLED(UNINSTALLED): not installed; IDLE(IDLE): idle state; ALL(ALL): all status

3. If this parameter is not specified, all the Pb-interface CICs are queried.

Default value: ALL(ALL)

Example

Display all the Ater-interface timeslots (Subrack No. = 5; Slot No. = 15; Port No. = 0;):
DSP ATERTS: SRN=5, SN=15, PN=0;
The result is as follows:
+++    HUAWEI        13.08.07 13:41:08
O&M    #424209
%%DSP ATERTS: SRN=5, SN=15, PN=0;%%
RETCODE = 0  Operation succeeded
