List Configuration Rights(LST CFGRIGHT)

Function

An OM command that queries the current configuration right.

Note

You can query only the current information of the configuration right. If the configuration right cannot be obtained, run this command to check the related information of configuration right application.

Parameters

None.

Example

List the information of the current configuration right:
LST CFGRIGHT:;
The result is as follows:
a. When the configuration right is available:
+++    HUAWEI        2008-07-23 14:29:56
O&M    #834
%%LST CFGRIGHT:;%%
RETCODE = 0  Operation succeeded

List Config Right Info
----------------------
Right Status  =  Unlocked
(Number of results = 1)


---    END
b. When the configuration right is unavailable:
+++    HUAWEI        2008-07-23 14:30:44
O&M    #857
%%LST CFGRIGHT:;%%
RETCODE = 0  Operation succeeded

List Config Right Info
----------------------
   IP Address  =  10.161.24.210
    Tool Type  =  MML
Locked by OMT  =  True
    User Name  =  admin
      Op Time  =  2008-07-23 14:30:39
 Right Status  =  User config right
(Number of results = 1)


---    END

Output Description

Domain Name Description
IP Address

IP address of the applicant
Tool Type

Type of internal tool
Locked by OMT

Yes: locked by OMT; No: locked by NE
User Name

User name of the operator
Op Time

Operation time
Right Status

User (global) configuration right
