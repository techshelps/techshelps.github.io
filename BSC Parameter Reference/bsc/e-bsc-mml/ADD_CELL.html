Add Cell(ADD CELL)

Function

A configuration command that adds a 2G internal cell.

Note

1. A BSC supports up to 2,048 cells.

2. A BTS supports up to 12 cells.

3. The name of a cell must be unique in a BSC and the name cannot be empty.

4. Any two cells in a BSC cannot have the same CGI.

5. When Freq. Band is set to PCS1900, you cannot configure a DCS1800 cell or a Co-BCCH cell. When Freq. Band is set to DCS1800, you cannot configure a PCS1900 cell.

6. When the TRX is configured in cabinet group 1 or 2, you need to add a slave chain or internal connection to the BTS.

7. Only the BTS3012 and the BTS3012II support combined cabinets and cabinet groups.

ISRFUCFGBYSLOT Cfg RFU by Slot

Meaning:Cfg RFU by Slot

range:YES(YES),NO(NO)

Explanation:

1. This parameter specifies whether to configure the DRFU, MRFU, GRFU, or XRFU by slot or link.

2. When setting the BTS to support the configuration of the RFU by slot, set this parameter to "YES". When setting the BTS not to support the configuration of the RFU by slot, set this parameter to "NO".

3. If this parameter is set to "YES", you must enter the numbers of the subrack and slot where the new board is located.

If you need to specify the CPRI connection relations (SFP port number and cascading level) based on a rule that is different from the cabinet mapping rule specified by the system, the parameters "RXU Chain Head Port No." and "Insert RXU Position" must be specified.

If this parameter is set to "NO", you must set "TRX Board No.", "RFU Link No." and "RXU Chain Head Port No.".

Example

Add a 2G internal cell (BTS Index = 0; Cell Name = cell3; MCC = 460; MNC = 10; LAC = 12; CI = 14; TRX Board No. = 53; TRX Board Type = QTRU(QTRU); TRX Board Pass No. = 1; TRX Freq = 64; Need Add Slave Chain OR Inn Connect = INN_CONNECT(NEED INN CONNECTION); BTS Out Port No. = 3; BTS In Port No. = 16):
ADD CELL: IDXTYPE=BYIDX, BTSIDX=0, CELLNAME="cell3", CTYPE=GSM900, AST=ACTIVATED, MCC="460", MNC="10", LAC=12, CI=14, TRXBN=53, TRXTP=QTRU, TRXPN2=1, TRXFREQ=64, NEEDCHAIN=INN_CONNECT, OUTPN=3, INPN=16;
