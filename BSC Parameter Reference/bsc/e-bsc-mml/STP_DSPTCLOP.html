Stop DSP TC Loop(STP DSPTCLOP)

Function

An OM command that stops the loopback test on the DSP channel.

Note

1. The DPU must work normally.

2. The subrack number and slot number to be entered are the numbers of the subrack and slot where the DPU is located.

3. The end channel number cannot be smaller than the start channel number.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: This parameter specifies the number of the subrack in which the GDPUC or GDPUX is located.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: This parameter specifies the number of the slot that holds the GDPUC or GDPUX.

Default value: None
DSPN DSP No.

Meaning: number of DSP

Value range: 0-21

Explanation: This parameter specifies the number of the DSP.

Default value: None
STRCHN Start channel

Meaning: number of the start channel

Value range: 0-69

Explanation: This parameter specifies the number of start channel.

Default value: None
ENDCHN End channel

Meaning: number of end channel

Value range: 0-69

Explanation: The number of end channel must be greater than that of start channel.

Default value: None

Example

Stop the loopback test on a DSP channel (Subrack No. = 1; Slot No. = 9; DSP No. = 0; Start channel = 0; End channel = 0):
STP DSPTCLOP: SRN=1, SN=9, DSPN=0, STRCHN=0, ENDCHN=0;

Output Description

None.
