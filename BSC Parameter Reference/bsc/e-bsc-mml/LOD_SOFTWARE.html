Load Software Online(LOD SOFTWARE)

Function

An OM command that loads software forcibly. To be specific, this command loads software from the GOMU/GBAM to the corresponding board.

Note

This command, which is optional, usually runs before the ACT SOFTWARE command. If you run the ACT SOFTWARE command without running the LOD SOFTWARE command, the NE automatically compares the software version of the GOMU/GBAM with that of the board. If the versions are inconsistent, the NE runs the LOD SOFTWARE command automatically.

Parameters

ID Name Description
SWT Software type

Meaning: software type

Value range: BIN(software), BIOS(BOOTROM)

Explanation: BIN indicates the host software, and BIOS indicates the BOOTROM.

Default value: BIN(software)
SV Software version

Meaning: software version

Value range: 1-63 characters

Explanation: This parameter is a character string, for example, BSC6000V900R003C01.

Default value: None

Example

1. Load software online:
LOD SOFTWARE:SWT=BIN,SV="BSC6000V900R008C01";
2. Load level 2 BIOS online:
LOD SOFTWARE:SWT=BIOS,SV="BSC6000V900R008C01";
