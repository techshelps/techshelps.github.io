Save BTS Clock Para(SAV BTSCLKPARA)

Function

An OM command that saves the settings of the clock parameters of a BTS.

Note

1. The BSC must communicate properly with the BTS.

2. The BTS must be configured with a clock board.

3. This command should be ran after the running of the SET BTSCLKPARA command to make the setting take effect.

Parameters

ID Name Description
SITEIDXTYPE Site Index Type

Meaning: how to specify a BTS

Value range: BYNAME(By Name), BYIDX(By Index)

Explanation: This parameter determines whether to specify a BTS by its name or index.

Default value: None
SITEIDX Site Index

Meaning: BTS index

Value range: 0-2047

Explanation:

1. This parameter specifies the number of a BTS. Each BTS is numbered uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYIDX(By Index).

Default value: None
SITENAME Site Name

Meaning: BTS name

Value range: 1-32 characters

Explanation:

1. This parameter specifies the name of the target BTS. Each BTS is named uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYNAME(By Name).

Default value: None

Example

Save the settings of the clock parameters of a BTS (Site Index Type = BYIDX(By Index); Site Index = 0):
SAV BTSCLKPARA: SITEIDXTYPE=BYIDX, SITEIDX=0;
