Confirm Patch(CON PATCH)

Function

An OM command that changes the status of the activated hot patch unit to Running state.

Note

1. The precondition for the successful confirmation of the hot patch is that the patch is already activated.

2. After a hot patch is confirmed, the status of the hot patch is changed to Running. The hot patch status of the corresponding boards is also changed. Run the DSP PATCH command to query the hot patch status.

Parameters

ID Name Description
PV Patch version

Meaning: patch version number

Value range: 1-63 characters

Explanation: This parameter is a character string, for example, BSC6000V900R008C11SP01.

Default value: None

Example

Confirm patch BSC6000V900R008C01SP01:
CON PATCH:PV="BSC6000V900R008C01SP01";
