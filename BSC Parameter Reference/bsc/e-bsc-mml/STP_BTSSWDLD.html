Stop Downloading BTS Software(STP BTSSWDLD)

Function

An OM command that queries the system to download BTS software to the operation and maintenance unit.

Note

1. The communication between the MML Client and the BSC is normal.

2. The OMU is downloading the BTS software.

Parameters

ID Name Description
SE Session ID

Meaning: session ID

Value range: 0-4294967295

Explanation: This parameter specifies the session ID for downloading the BTS software. Ensure that the session ID and the returned session ID when you run the command for downloading the BTS software are the same.

Default value: None

Example

Stop the download of the BTS software Session number12345:
STP BTSSW:SE=12345;
