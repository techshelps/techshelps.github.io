Upload License File(ULD LICENSE)

Function

An OM command that uploads the license files from the GOMU/GBAM to the FTP server.The FTP server is the PC that is installed with LMT or M2000 server software.

Note

After the license file is uploaded from the GOMU/GBAM to the LMT computer, the file still exists on the GOMU/GBAM.

Parameters

ID Name Description
IP FTP Server's IP Address

Meaning: IP address of the FTP server

Value range: 0.0.0.0-255.255.255.255

Explanation: This parameter specifies the IPv4 type as xxx.xxx.xxx.xxx.

Default value: None
USR FTP Server's User Name

Meaning: user name of the FTP server

Value range: 0-24 characters

Explanation: 1.This parameter is a character string *******.

2.The default user name for logging in to the FTP server to which the LMT is connected is swmtest.

Default value: None
PWD FTP Server's User Password

Meaning: user password of the FTP server

Value range: 0-24 characters

Explanation: 1.This parameter is a character string *******.

2.The default password for logging in to the FTP server to which the LMT is connected is swmtest.

Default value: None
DIR Directory

Meaning: directory name

Value range: 1-127 characters

Explanation:

1. This parameter is a character string *:\***\***.

2. The directory must be at the same level with or at a lower level than the FtpServer directory.

Default value: None
FN File Name

Meaning: file name

Value range: 1-31 characters

Explanation: 1.This parameter is a character string *******.

2.The file name must be the same as the name of the license file to be uploaded.

You must not name it by yourself.

Default value: None

Example

ULD LICENSE: IP="80.0.102.50", USR="swmtest", PWD="swmtest", DIR="C:\HW LMT\BSC6000\Common", FN="License_B9.dat";
