Start Transport Performance Test(STR TRANSPERFTST)

Function

An OM command that starts the test on the BTS transmission performance.

Note

1. The BSC must communicate properly with the BTS.

2. The running of this command leads to the resetting of the BTS, which disrupts the services. Use this command with caution.

Parameters

ID Name Description
IDXTYPE Index Type

Meaning: how to specify a BTS

Value range: BYNAME(By Name), BYIDX(By Index)

Explanation: This parameter determines whether to specify a BTS by its name or index.

Default value: None
BTSIDX BTS Index

Meaning: BTS index

Value range: 0-2047

Explanation:

1. This parameter specifies the number of a BTS. Each BTS is numbered uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYIDX(By Index).

Default value: None
BTSNAME BTS Name

Meaning: BTS name

Value range: 1-32 characters

Explanation:

1. This parameter specifies the name of the target BTS. Each BTS is named uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYNAME(By Name).

Default value: None
SLPTYPE Self loop type

Meaning: mode of self loop

Value range: E1SELFLOOP, IASUSELFLOOP

Explanation: Only the BTS3002C, BTS3006C, and BTS3002E support the IASUSELFLOOP mode.

Default value: None
E1SLCTTMU TMU type

Meaning: TMU type in E1SELFLOOP mode

Value range: ACTIVE(ACTIVE TMU/MMU/IOMU/DOMU), ACTIVESTANDBY(ACTIVE/STANDBY TMU), SLAVE1(SLAVE TMU1/DOMU1), SLAVE2(SLAVE TMU2/DOMU2)

Explanation:

1. For the BTS3002C, BTS3001C, and BTS3001CP, set this parameter to ACTIVE(ACTIVE TMU/MMU/IOMU/DOMU).

2. For the BTS3006C and BTS3002E, do not set this parameter to ACTIVESTANDBY(ACTIVE/STANDBY TMU).

Default value: None
LPBKMODE Loop back mode

Meaning: loopback mode

Value range: E1(E1 LOOP BACK), TIMESLOT(TIMESLOT LOOP BACK)

Explanation: This parameter specifies the loopback mode.

Default value: None
E1PORT Port

Meaning: E1 port number

Value range: 0-7

Explanation: For the BTS3012, BTS3012AE, and BTS3012 II, the value range is from 0 to 7; For other BTSs, the value range is from 0 to 3.

Default value: None
E1DURAT Duration

Meaning: duration in E1 loop back mode

Value range: 1-60

Explanation: This parameter specifies the duration in E1 loopback mode. The parameter is expressed in minutes.

Default value: None
TSDURAT Duration * 10 minutes

Meaning: duration in timeslot loopback mode (*10 minutes)

Value range: 1-144

Explanation: This parameter specifies the duration in timeslot loopback mode.

Default value: None
TSTTIMESLOT Test timeslot

Meaning: number of the timeslot to be tested

Value range: 0-31

Explanation: This parameter specifies the number of timeslots to be tested.

Default value: None
IASUSLCTTMU TMU type

Meaning: TMU type in IASUSELFLOOP mode

Value range: MAINDOMU, SLAVEDOMU1, SLAVEDOMU2

Explanation: The BTS3002C supports only the MAINDOMU type.

Default value: None
IASUPORT Port

Meaning: type of ports in IASUSELFLOOP mode

Value range: E1PORT0, E1PORT1, E1PORT2, E1PORT3, OPTICALPORT0, OPTICALPORT1

Explanation: The BTS3002C does not support the E1PORT3.

Default value: None
IASUDURAT Duration

Meaning: duration in IASUSELFLOOP mode

Value range: 1-60

Explanation: This parameter specifies the duration in IASUSELFLOOP mode. The parameter is expressed in minutes.

Default value: None

Example

Start the BTS transmission performance test (Index Type = BYIDX(By Index); BTS Index = 0; Self loop type = E1SELFLOOP; TMU type = ACTIVE(ACTIVE TMU/MMU/IOMU/DOMU); Loop back mode = E1(E1 LOOP BACK); Port = 0; Duration = 1):
STR TRANSPERFTST: IDXTYPE=BYIDX, BTSIDX=0, SLPTYPE=E1SELFLOOP, E1SLCTTMU=ACTIVE, LPBKMODE=E1, E1PORT=0, E1DURAT=1;
