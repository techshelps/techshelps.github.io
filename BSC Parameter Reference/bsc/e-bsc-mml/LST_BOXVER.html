List Alarm Box Version(LST BOXVER)

Function

List the version information of the current alarm box.

After the command is executed, the version information of the current alarm box is displayed.

Note

None.

Parameters

ID Name Description
BOXLOC Alarm Box Position

Meanings:Alarm Box Position

Value range:loc~0,omu~1

Content:none

Default value:Local

Example

1.To list the version information of the current alarm box, execute the following command:
LST BOXVER:;
The result is displayed as follows:
+++    HUAWEI        2008-07-16 08:49:25
O&M    #705
%%LST BOXVER:;%%
RETCODE = 0  Operation succeeded

Alarm box version  =  GM11ALMM REV0 SV02200(20010928)LV01100
(Number of results = 1)

---    END

Output Description

Domain Name Description
Alarm box version

Alarm box version
