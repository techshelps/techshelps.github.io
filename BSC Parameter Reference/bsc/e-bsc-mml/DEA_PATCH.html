Deactivate Patch(DEA PATCH)

Function

An OM command that deactivates the activated hot patch files in the patch area in the board memory.

Note

1. The precondition for the successful deactivation of hot patch files on each board is that the hot patch files in the hot patch area in the board memory are already activated.

2. After a hot patch is deactivated successfully, the status of the hot patch changes to Deactive. The hot patch status of the corresponding boards also changes. You can run DSP PATCH command to query the hot patch status.

Parameters

ID Name Description
PV Patch version

Meaning: patch version number

Value range: 1-63 characters

Explanation: This parameter is a character string, for example, BSC6000V900R003C01SP01.

Default value: None

Example

Deactivate patch BSC6000V900R008C01SP01:
DEA PATCH:PV="BSC6000V900R008C01SP01";
