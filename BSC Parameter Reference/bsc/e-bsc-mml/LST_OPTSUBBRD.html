List STM1 Sub-board Attributes(LST OPTSUBBRD)

Function

An OM command that queries the attributes of the sub-boards of optical port boards.

To query attributes of the sub-boards of all the optical interface boards in a specified subrack, set the parameter Subrack No.

To query sub-board attributes of a specific optical interface board, set the parameters Subrack No. and Slot No..

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:None.

Default value:None
SN Slot No.

Meaning:Slot No.

Value range:0~27

Explanation:None.

Default value:None
