Set Communication IP type(SET ITFIPTYPE)

Function

A configuration command that modifies the IP communication type on the A interface, Abis interface, and Gb interface.

Note

This command can be run only when the IP communication is used on the interface.

Parameters

ID Name Description
AIPTYPE A-interface Communication IP type

Meaning:A-interface Communication IP type

Value range:PORTIP,DEVIP

Explanation:This parameter specifies whether the device IP or the port IP is used for communication over the A interface.

Default value:None
ABISIPTYPE Abis-interface Communication IP type

Meaning:Abis-interface Communication IP type

Value range:PORTIP,DEVIP

Explanation:This parameter specifies whether the device IP or the port IP is used for communication over the Abis interface.

Default value:None
GBIPTYPE Gb-interface Communication IP type

Meaning:Gb-interface Communication IP type

Value range:PORTIP,DEVIP

Explanation:This parameter specifies whether the device IP or the port IP is used for communication over the Gb interface.

Default value:None

Example

1. Set the IP communication schemes over A interface(A-interface Communication IP type = DEVIP):
SET ITFIPTYPE: AIPTYPE=DEVIP;
2. Set the IP communication schemes over Gb interface(Gb-interface Communication IP type = PORTIP):
SET ITFIPTYPE: AIPTYPE=DEVIP,GBIPTYPE=PORTIP;
