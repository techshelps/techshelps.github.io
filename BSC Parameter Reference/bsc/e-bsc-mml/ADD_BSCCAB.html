Add BSC Cabinet(ADD BSCCAB)

Function

A configuration command that adds a BSC cabinet.

Note

1. Only cabinets 1 to 3 can be added to the BSC; cabinet 0 is configured by default.

2. When a GTCS is configured locally, you cannot add a remote cabinet.

Parameters

ID Name Description
CABNO Cabinet No.

Meaning:Cabinet No.

Value range:1~3

Explanation:This parameter specifies the number of a cabinet.

Default value:None
CABTYPE Is Remote

Meaning:Is Remote

Value range:NO(No),YES(Yes)

Explanation:This parameter specifies whether a cabinet is a remote one.

Default value:None

Example

Add a BSC cabinet (Cabinet No. = 1; Is Remote = YES(Yes)):
ADD BSCCAB: CABNO=1, CABType=YES;
