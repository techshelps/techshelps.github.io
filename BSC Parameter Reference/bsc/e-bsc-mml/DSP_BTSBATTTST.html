Query Test Battery of BTS(DSP BTSBATTTST)

Function

An OM command that queries the result of the BTS battery test.

Note

1. The BSC must communicate properly with the BTS.

2. The BTS is configured with the battery equipment.

Parameters

ID Name Description
IDXTYPE Index Type

Meaning: how to specify a BTS

Value range: BYNAME(By Name), BYIDX(By Index)

Explanation: This parameter determines whether to specify a BTS by its name or index.

Default value: None
STIDX Site Index

Meaning: BTS index

Value range: 0-2047

Explanation:

1. This parameter specifies the number of a BTS. Each BTS is numbered uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYIDX(By Index).

Default value: None
STNAME Site Name

Meaning: BTS name

Value range: 1-32 characters

Explanation:

1. This parameter specifies the name of the target BTS. Each BTS is named uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYNAME(By Name).

Default value: None
SRN Subrack No.

Meaning: subrack number

Value range: 0-53

Explanation: This parameter specifies the number of the subrack in which the BTS power module is located. The subrack number is unique in a BTS.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-23

Explanation: This parameter specifies the number of the slot that holds the BTS power module.

Default value: None

Example

Query the battery test result (Index Type = BYIDX(By Index); Site Index = 0; Subrack No. = 0; Slot No. = 15):
DSP BTSBATTTST: IDXTYPE=BYIDX, BTSIDX=0, SRN=0, SN=15;
