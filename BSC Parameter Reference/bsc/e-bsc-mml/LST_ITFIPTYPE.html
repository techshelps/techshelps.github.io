List communication IP type(LST ITFIPTYPE)

Function

An OM command that queries the IP communication type on the A interface, Abis interface, and Gb interface.

Note

None

Parameters

None.

Example

Query the communication type of the IP transmission:
LST ITFIPTYPE:;
The result is as follows:
+++    HUAWEI        2008-07-07 14:20:56
O&M    #3389
%%LST ITFIPTYPE:;%%
RETCODE = 0  Operation succeeded

List communication IP type Result
---------------------------------
A-interface IP Type  Abis-interface IP Type  Gb-interface IP Type

DEVIP                DEVIP                   DEVIP               
(Number of results = 1)


---    END

Output Description

For details, see the parameter description of the SET ITFIPTYPE command.
