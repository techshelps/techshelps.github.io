List Status of Local User Management(LST OPSW)

Function

List the status of local user management

Note

None.

Parameters

None.

Example

Query the status of local user management:
LST OPSW:;
The result is as follows:
+++    HUAWEI        2007-10-28 15:11:09
O&M    #2
%%lst opsw:;%%
RETCODE = 0  Operation succeeded

List local operator switch of NM
--------------------------------
status  =  on
(Number of results = 1)


---    END

Output Description

Domain Name Description
Status

Whether the local user management function is enabled. Value range: off(disable), on(enable)
