Display E1/T1 Port Timeslot Loopback Attributes(DSP E1T1TS)

Function

An OM command that queries the loopback properties of E1/T1 port timeslots.

Note

None.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board in a subrack.

Default value: None
PN Port No.

Meaning: port number

Value range: 0-31

Explanation: The port number uniquely identifies a port on a board.

Default value: None

Example

Query the loopback information of all the timeslots (Subrack No. = 0; Slot No. = 14; Port No. = 0):
DSP E1T1TS: SRN=0,SN=14,PN=0;
The result is as follows:
