Remove Pb Interface E1/T1(RMV PBE1T1)

Function

A configuration command that removes an E1/T1 on the Pb interface.

Note

You can remove an E1/T1 on the Pb interface only when the type of its all timeslots is NULL or CIC.

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:None.

Default value:None
SN Slot No.

Meaning:Slot No.

Value range:0~27

Explanation:None.

Default value:None
PN Port No.

Meaning:Port No.

Value range:0~83

Explanation:None.

Default value:None

Example

1. Remove the E1/T1 on the Pb interface (Subrack No. = 0; Slot No. = 24; Port No. = 0):
RMV PBE1T1: SRN=0, SN=24, PN=0;
