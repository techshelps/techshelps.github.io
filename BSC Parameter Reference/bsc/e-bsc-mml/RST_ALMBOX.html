Reset Alarm Box(RST ALMBOX)

Function

Reset an alarm box.

After the alarm box is reset, the current audible and visual alarm information is resent to the alarm box.

Note

After the alarm box is reset, the settings for the indicator and sound are cleared. After the alarm box is successfully reset, it automatically checks for active alarms that drive the alarm box. Therefore, the settings for the indicator and sound do not take effect during the reset of the alarm box.

Parameters

ID Name Description
BOXLOC Alarm Box Position

Meaning: position of the alarm box

Value range: loc(Local), omu(OMU)

Explanation:

1. The value loc(Local) indicates that the alarm box is connected to the serial port on the local PC; the value omu(OMU) indicates that the alarm box is connected to the OMU.

2. This parameter specifies the position of the alarm box to be reset.

Default value: loc(Local)

Example

Reset the alarm box.
RST ALMBOX:;
