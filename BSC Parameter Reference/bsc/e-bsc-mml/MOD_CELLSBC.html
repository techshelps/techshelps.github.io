Modify Simplified Broadcast Parameters of Cell(MOD CELLSBC)

Function

A configuration command that modifies the simplified broadcast parameters of the cell. In this command, Cell Index or Cell Name uniquely identifies a cell.

Note

None

Parameters

ID Name Description
IDXTYPE Index type

Meaning:Index type

Value range:BYNAME(By Name),BYIDX(By Index)

Explanation:Index Type.

Default value:None
CELLIDX Cell index

Meaning:Cell index

Value range:0~2047

Explanation: This parameter indicates the source cell index. The source cell index must be unique.

Default value:None
CELLNAME Cell name

Meaning:Cell name

Value range:1-32 characters

Explanation: This parameter indicates the name of the source cell. The name of each cell under one BSC must be unique.

Default value:None
SUPPORTCELLBROADCAST Support cell broadcast name

Meaning:Support cell broadcast name

Value range:NO(NO),YES(YES)

Explanation:This parameter specifies whether the Support Cell Broadcast Name function is enabled in a cell.

Default value:None
GEOGRAHPYRANGE Geography range

Meaning:Geography range

Value range:0(Cell Range immediately Send),1(PLMN Range Normally Send),2(Position Range Normally Send),3(Cell Range Normally Send)

Explanation:This parameter specifies the geographical range in which a cell broadcast message is sent.

Default value:None
CHANNELNUMBER Channel number

Meaning:Channel number

Value range:0~65535

Explanation:This parameter specifies the frequency number of a cell broadcast message.

Default value:None
CODESCHEME Code Scheme

Meaning:Code Scheme

Value range:GERMAN(German),ENGLISH(English),ITALIA(Italia),FRENCH(French),SPANISH(Spanish),DUTCH(Dutch),SWEDISH(Swedish),DANISH(Danish),PORTUGUESE(Portuguese),FINNISH(Finnish),NORWEGIAN(Norwegian),GREEK(Greek),TURKISH(Turkish),HUNGARIAN(Hungarian),POLISH(Polish),CHINESE(ChineseGB2312),CYRILLIC(Cyrillic Windows),ARABICASMO(Arabic ASMO),ARABICWINDOWS(Arabic Windows),THAI(Thai Windows),GREEKWINDOWS(Greek Windows),TURKISHWINDOWS(Turkish Windows),HEBREW(Hebrew Windows),BALTIC(Baltic Windows),EASTERNEUROPEAN(Eastern European Windows),WESTERNEUROPEAN(Western European Windows),UCS2(UCS-2),RESERVED1(Reserved Code Scheme 1),RESERVED2(Reserved Code Scheme 2),RESERVED3(Reserved Code Scheme 3),RESERVED4(Reserved Code Scheme 4),RESERVED5(Reserved Code Scheme 5)

Explanation:This parameter specifies the coding scheme of a cell broadcast message.

Default value:None
BROADCASTCONTENT Broadcast content

Meaning:Broadcast content

Value range:1-80 characters

Explanation:This parameter specifies the content of a cell broadcast message.

Default value:None
BROADCASTINTERVAL Broadcast interval[s]

Meaning:Broadcast interval[s]

Value range:2~3600

Explanation:This parameter specifies the interval at which a cell broadcast message is sent.

Default value:None

Example

Modify the simplified broadcast parameters of the cell (Index type = BYIDX(By Index); Cell index = 0; Support cell broadcast name = YES(YES)):
MOD CELLSBC: IDXTYPE=BYIDX, CELLIDX=0, SUPPORTCELLBROADCAST=YES;
