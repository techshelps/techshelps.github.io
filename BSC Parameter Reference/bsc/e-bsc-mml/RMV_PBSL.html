Remove Pb Signaling Link(RMV PBSL)

Function

A configuration command that removes a signaling link on the Pb interface.

Note

When the last signaling link to the PCU on the Pb interface is removed, the connection to the PCU is broken.

Parameters

ID Name Description
LGIDX PbSL Logical Index

Meaning:PbSL Logical Index

Value range:0~65535

Explanation:Logic No.

Default value:None

Example

Remove a signaling link on the Pb interface (PbSL Logical Index = 4096):
RMV PBSL: LGIDX=4096;
