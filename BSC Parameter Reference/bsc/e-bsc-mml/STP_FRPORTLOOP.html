Stop FR Port Loopback(STP FRPORTLOOP)

Function

An OM command that stops the FR port loopback.

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation:

1. This parameter specifies the subrack where the FR loopback is to be stopped.

2. The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation:

1. This parameter specifies the slot where the FR loopback is to be stopped.

2. The subrack number and slot number uniquely identify a board.

Default value: None
PN Port No.

Meaning: port number

Value range: 0-31

Explanation: This parameter specifies the port where the FR loopback is to be stopped.

Default value: None
BCID BC Id

Meaning: BC ID

Value range: 0-255

Explanation: This parameter specifies the BC ID of the FR loopback to be stopped.

Default value: None

Example

Stop loopback for the FR port (Subrack No. = 0; Slot No. = 22; Port No. = 0; BC Id = 0):
STP FRPORTLOOP: SRN=0, SN=22, PN=0, BCID=0;
