Start BATCHFILE Session(STR BATCHFILESN)

Function

An OM command that starts the batch configuration command session.

Normally, the procedures for batch configuration are as follows: 1. Start the batch configuration session; 2. Download the batch configuration files; 3. Pre-activate the batch configuration session; 4. Activate the batch configuration session. 5. Upload the results of the batch configuration; 6. Stop the batch configuration session.

Note

1.After this command takes effect, the current MML terminal occupies the system configuration right and configuration operations from other terminals are prohibited. Therefore, stop the batch configuration command in time for operation configuration of other terminals when the execution of the batch configuration command completes.

2.This command is the first step of the batch configuration command operations. It applies for configuration right for batch configuration command session and prepares for the activation of the batch configuration command sessions.

3. After the batch configuration is complete, use the END BATCHFILESN command to release the configuration rights.

4. If the configuration rights are occupied by others for a long time, contact the administrator to use the END BATCHFILESN or FOE BATCHFILESN command to release the configuration rights.

5. If the batch configuration is started by the same type of clients, use the END BATCHFILESN command to release the configuration rights. If the batch configuration is started by different types of clients such as LMT and M2000, use the FOE BATCHFILESN command to release the configuration rights.

Parameters

ID Name Description
SID SessionID

Meaning: session ID

Value range: 0-268435455

Explanation: Session ID identifies a session.

Default value: None

Example

Start the batch configuration command session (SID=1111111):
STR BATCHFILESN:SID=111111;
Note:
If another batch configuration command session exists in the system, the message "Not Supported by the NE" is displayed. Otherwise, the specified batch configuration command session is successfully started.
