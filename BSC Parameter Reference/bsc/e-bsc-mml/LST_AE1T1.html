List A Interface E1/T1(LST AE1T1)

Function

An OM command that queries the configuration information of the E1/T1 on the A interface.

To query the information of the E1/T1 on the A interface in specified subracks, set SubRack No..

To query the information of the specified E1/T1 on the A interface, set Subrack No., Slot No., and Port No..

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:None.

Default value:None
SN Slot No.

Meaning:Slot No.

Value range:0~27

Explanation:None.

Default value:None
PN Port No.

Meaning:Port No.

Value range:0~83

Explanation:None.

Default value:None
LSTTYPE List Type

Meaning:List Type

Value range:SUMMARY(Summary Information),DETAIL(All Information)

Explanation:List Type.

Default value:SUMMARY

Example

1. Query the data information of the E1/T1 on the A interface (Subrack No. = 1; List Type = SUMMARY(Summary Information)):
LST AE1T1: SRN=1, LSTTYPE=SUMMARY;
The result is as follows:
