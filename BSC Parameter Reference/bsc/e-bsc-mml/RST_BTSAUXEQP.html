Reset Bts Auxiliary Equipment(RST BTSAUXEQP)

Function

An OM command that resets the auxiliary equipment of the BTS.

Note

1. This command applies to the BTS3006A only.

2. The BTS3006A must be configured with the OSSU, which is connected to the auxiliary equipment. Currently, the auxiliary equipment refers to only the heat exchanger.

3. Running this command resets the auxiliary equipment. Use this command with caution.

Parameters

ID Name Description
SITEIDXTYPE Site Index Type

Meaning: how to specify a BTS

Value range: BYNAME(By Name), BYIDX(By Index)

Explanation: This parameter determines whether to specify a BTS by name or by index.

Default value: None
SITEIDX Site Index

Meaning: BTS index

Value range: 0-2047

Explanation:

1. This parameter specifies the index number of the BTS. Each BTS is numbered uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYIDX(By Index).

Default value: None
SITENAME Site Name

Meaning: BTS name

Value range: 1-32 characters

Explanation:

1. This parameter specifies the name of the BTS. Each BTS is named uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYNAME(By Name).

Default value: None
SRN Subrack No

Meaning: subrack number

Value range: 0-53

Explanation: This parameter specifies the number of the subrack where the board connected to the auxiliary equipment is located. The subrack number is unique in a BTS.

Default value: None
SN Slot No

Meaning: slot number

Value range: 0-23

Explanation: This parameter specifies the number of the slot that holds the board connected to the auxiliary equipment of the BTS.

Default value: None
EQPTYPE Equipment Type

Meaning: type of the auxiliary equipment

Value range: HEATEXCHANGER

Explanation: This parameter specifies the type of the auxiliary equipment. Currently, only the heat exchanger is supported.

Default value: None

Example

Reset a heat exchanger (Site Index Type = BYIDX(By Index); Site Index = 0; Subrack No = 0; Slot No = 17; Equipment Type = HEATEXCHANGER):
RST BTSAUXEQP: SITEIDXTYPE=BYIDX, SITEIDX=0, SRN=0, SN=17, EQPTYPE=HEATEXCHANGER;
