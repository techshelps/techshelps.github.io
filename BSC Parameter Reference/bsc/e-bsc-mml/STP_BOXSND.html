Stop Alarm Sound(STP BOXSND)

Function

Use this command to stop the alarm sound.

Note

1. After the running of this command, the audible alarms for all alarm severities are stopped. Neither existing alarms nor new alarms trigger an audible alarm on the alarm box.

2. After the running of this command, the existing alarms and new alarms that trigger the alarm box indicator to blink.

Parameters

ID Name Description
BOXLOC Alarm Box Position

Meaning: position of the alarm box

Value range: loc(Local), omu(OMU)

Explanation: The value loc(Local) indicates that the alarm box is connected to the serial port on the local PC; the value omu(OMU) indicates that the alarm box is connected to the OMU.

Default value: loc(Local)

Example

To stop the alarm sound, execute the following command:
STP BOXSND:;
