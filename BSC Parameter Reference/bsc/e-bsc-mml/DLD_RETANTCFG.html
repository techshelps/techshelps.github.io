Download RET Antenna Cfg File(DLD RETANTCFG)

Function

An OM command that downloads the RET antenna configuration files to the GOMU.

Note

1. The MML client must communicate properly with the BSC.

2. The FTP server runs properly.

3. The antenna configuration files must be placed in the root directory where the FTP server is installed.

Parameters

ID Name Description
IP FTPServer IP

Meaning: IP address of the FTP server

Value range: 0.0.0.0-255.255.255.255

Explanation: This parameter specifies the IP address of the FTP server.

Default value: None
USR FTPServer user

Meaning: user name of the FTP server

Value range: 1-32 characters

Explanation: This parameter specifies the user name of the FTP server.

Default value: None
PWD User password

Meaning: user password for logging in to the FTP server

Value range: 1-32 characters

Explanation: This parameter specifies the user password for the FTP server.

Default value: None
DIR File directory

Meaning: directory to which the file is saved

Value range: 1-256 characters

Explanation: This parameter specifies the directory to which the file to be downloaded is saved.

Default value: None
FN File name

Meaning: file name

Value range: 1-256 characters

Explanation: This parameter specifies the name of the file to be downloaded.

Default value: None

Example

Download the RET antenna configuration files to the GOMU (FTPServer IP = 10.161.24.71; FTPServer user = swmtest; User password = ******; File directory = D:/HW LMT/BSC6000/Common/FtpServer/; File name = KA_742212_1710-2170.bin):
DLD RETANTCFG: IP="10.161.24.71", USR="swmtest", DIR=" D:/HW LMT/BSC6000/Common/FtpServer/", FN="KA_742212_1710-2170.bin";
