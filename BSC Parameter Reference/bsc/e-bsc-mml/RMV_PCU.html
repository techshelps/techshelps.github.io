Remove PCU(RMV PCU)

Function

A configuration command that removes a PCU.

Note

1. The PCU to be removed must exist.

2. The PCU to be removed is not used by the Pb E1.

3. The PCU to be removed is not used by the PCU.

Parameters

ID Name Description
NAME PCU Name

Meaning:PCU Name

Value range:1-50 characters

Explanation:This parameter specifies the name of the PCU that is connected to the E1 link on the Pb interface. The PCU name is unique in one BSC.

Default value:None

Example

1. Remove the PCU (PCU Name = PCU01):
RMV PCU: NAME="PCU01";
