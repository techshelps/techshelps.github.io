Modify UTRAN TDD Handover Data of Cell(MOD CELLHOUTRANTDD)

Function

A configuration command that modifies the UTRAN TDD handover parameters of the cell. In this command, Cell Index or Cell Name uniquely identifies a cell.

Note

None

Parameters

ID Name Description
IDXTYPE Index type

Meaning:Index type

Value range:BYNAME(By Name),BYIDX(By Index)

Explanation:Index Type.

Default value:None
CELLIDX Cell index

Meaning:Cell index

Value range:0~2047

Explanation: This parameter indicates the source cell index. The source cell index must be unique.

Default value:None
CELLNAME Cell name

Meaning:Cell name

Value range:1-32 characters

Explanation: This parameter indicates the name of the source cell. The name of each cell under one BSC must be unique.

Default value:None
HOOPTSEL Tdd inter-rat handover preference

Meaning:Tdd inter-rat handover preference

Value range:Pre_2G_Cell(Preference for 2G Cell),Pre_3G_Cell(Preference for 3G Cell),Pre_2G_CellThres(Preference 2G Cell By Thresh)

Explanation: This parameter specifies whether an MS is preferentially handed over to a 2G cell or to a 3G cell.

Default value:None
2GHOPRETH Tdd handover preference threshold for 2g cell

Meaning:Tdd handover preference threshold for 2g cell

Value range:0~63

Explanation: If the Inter-RAT HO Preference parameter is set to Preference for 2G Cell By Threshold, and if the receive level of the first candidate cell among 2G candidate cells is lower than or equal to the HO Preference Threshold for 2G Cell, the 3G cell handover is preferred. Otherwise, the 2G cell handover is preferred.

The level values 0 through 63 map to -110 dBm to -47 dBm.

Default value:None
3GHORSCPTH Tdd rscp threshold for better 3g cell handover

Meaning:Tdd rscp threshold for better 3g cell handover

Value range:0~63

Explanation: If both the Inter-RAT Out BSC Handover Enable and the Better 3G Cell HO Allowed parameters are set to Yes, a 3G better cell handover is triggered when the RSCP of an adjacent 3G cell is greater than the TDD RSCP Threshold for Better 3G Cell HO during a period of time.

The level values 0 through 63 map to -110 dBm to -47 dBm.

Default value:None
3GHOSTAT Tdd 3g better cell handover watch time<s>

Meaning:Tdd 3g better cell handover watch time<s>

Value range:1~16

Explanation: According to the P/N criterion, if the triggering conditions of TDD 3G better cell handover are met for N consecutive seconds within P seconds, a TDD 3G better cell handover is triggered.

This parameter corresponds to P of the P/N criterion.

Default value:None
3GHODURT Tdd 3g better cell handover valid time<s>

Meaning:Tdd 3g better cell handover valid time<s>

Value range:1~16

Explanation: According to the P/N criterion, if the triggering conditions of TDD 3G better cell handover are met for N consecutive seconds within P seconds, a TDD 3G better cell handover is triggered.

This parameter corresponds to N of the P/N criterion.

Default value:None
BET3GHOEN Tdd better 3g cell handover allowed

Meaning:Tdd better 3g cell handover allowed

Value range:NO(NO),YES(YES)

Explanation: This parameter specifies whether the 3G better cell handover algorithm is allowed.

Yes: The 3G better cell handover algorithm is allowed.

No: The 3G better cell handover algorithm is forbidden.

Default value:None

Example

Modify the UTRAN TDD handover parameters of the cell (Index Type = BYIDX(By Index); Cell Index = 0):
MOD CELLHOUTRANTDD: IDXTYPE=BYIDX, CELLIDX=0, HOOPTSEL=Pre_2G_Cell, 2GHOPRETH=2, 3GHORSCPTH=3, 3GHOSTAT=10, 3GHODURT=1, BET3GHOEN=YES;
