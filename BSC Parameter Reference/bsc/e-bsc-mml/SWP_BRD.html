Swap Board(SWP BRD)

Function

An OM command that switches over the boards in active/standby mode. That is, after the running of this command, the standby board upgrades to be the active one while the active board degrades to be the standby one.

Note

1. The specified board must be an active board that is configured and functional. You can query the status of the board by running the DSP BRDSTAT command.

2. The original active board is reset and restarted after the switchover. Run this command with caution.

3. The switchover may fail if a hardware fault occurs on the standby board, the time after the startup of the standby board is less than five minutes, or the standby CPU is faulty.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None

Example

Switch over the GSCU in slot 6 of subrack 0:
SWP BRD:SRN=0,SN=6;
