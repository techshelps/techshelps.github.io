Set Transfer Server(SET TRANFTP)

Function

A configuration command that configures the transfer server.

Note

1.Only one configuration record is allowed. Therefore, every time you configure a new transfer server, the previous configuration is overwritten.

2.If the transfer server is configured on the OMU, you must use the STR FTPSSRV command to start the FTP server on the OMU. Otherwise, the LMT may fail to be connected to the OMU.

3.If the IP Address is configured 0.0.0.0,the transfer server is disabled.

Parameters

ID Name Description
ISATOMU OMU Transfer Server

Meaning:OMU Transfer Server

Value range:NO(NO),YES(YES)

Explanation: If the transfer server is configured on the OMU, you need not specify "IP Address", "Port No.", "User Name", and "User Password".

Default value:None
IP IP Address

Meaning:IP Address

Value range:0.0.0.0-255.255.255.255

Explanation: This parameter specifies the IP address of the transfer server. If the IP Address is configured 0.0.0.0,the transfer server is disabled.

Default value:None
USERNAME User Name

Meaning:User Name

Value range:1-32 characters

Explanation: This parameter specifies the user name for logging in to the FTP server. Excluding the invalid characters ';',',','"','''','=','\','%%','+++','&','#'.

Default value:None
PWD User password

Meaning:User password

Value range:Password type parameter

Explanation: This parameter specifies the user password for logging in to the FTP server. The password cannot contain the following characters: comma (,), semicolon (;), equal mark (=), double quotation marks ("), single quotation marks ('), and backward slash (\).

Default value:None
TRANALIAS Transfer Alias

Meaning:alias of the transfer server

Value range: 1-32 characters

Explanation:This parameter specifies the alias of the transfer server.Excluding the invalid characters ';',',','"','''','=','\','%%','+++','&','#'.

Default value: None
PORTNO Port No.

Meaning:Port No.

Value range:0~65534

Explanation: This parameter specifies the port number of the FTP server.

Default value:None

Example

Configure the transfer server that is not on the OMU (FTPServer IP = 10.161.10.10; FTPServer user = admin; User password = *****; Port No. = 21):
 SET TRANFTP: ISATOMU=NO, IP="10.161.10.10", USERNAME="admin", PWD="*****", PORTNO=21;
