Load Patch(LOD PATCH)

Function

An OM command that loads the hot patch from the GOMU/GBAM Flash memory to the hot patch area in the memory of the destination board. The hot patch is also saved in the Flash memory of the board.

Note

After the hot patch is loaded successfully, the status of the hot patch changes to Deactive. The hot patch status of the corresponding boards also changes. Run the DSP PATCH command to query the hot patch status.

Parameters

ID Name Description
PV Patch version

Meaning: patch version number

Value range: 1-63 characters

Explanation: This parameter is a character string, for example, BSC6000V900R003C01SP01.

Default value: None

Example

Load patch BSC6000V900R008C01SP01:
LOD PATCH:PV="BSC6000V900R008C01SP01";
