List EMS Information(LST EMS)

Function

An OM command that queries the login information about the EMS.

Note

None.

Parameters

None.

Example

1. Query the login information about the online EMS:
LST EMS:;
The result is as follows:
+++    HUAWEI        2007-10-28 16:18:08
O&M    #602
%%LST EMS:;%%
RETCODE = 0  Execution succeeded

The information about the EMS is as follows:
-----------
    Operator = emscomm
  Password = ******
Date limit = NOLIMIT
Time limit = NOLIMIT
    Week limit= NOLIMIT
    Status = Online
      Bound IP address = 10.161.74.213 
(Number of results = 1)


--- END
2. Query the login information about the offline EMS:
LST EMS:;
The result is as follows:
+++    HUAWEI        2007-10-28 16:18:08
O&M    #602
RETCODE = 0  Execution succeeded

The information about the EMS is as follows:
-----------
    Operator = emscomm
  Password = ******
Date limit = NOLIMIT
Time limit = NOLIMIT
    Week limit= NOLIMIT
    Status = Offline
      IP Address = NULL
(Number of results = 1)


---    END

Output Description

Domain Name Description
Operator

User name of the operator
Password

Password of the operator (displayed as "*")
Date Limit

Limit on the operation date
Time Limit

Limit on the operation time
Week Limit

Limit on the weekday of operation
Status

Online/Offline
IP Address

Login IP address of the client
