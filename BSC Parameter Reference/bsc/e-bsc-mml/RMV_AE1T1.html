Remove A Interface E1/T1(RMV AE1T1)

Function

A configuration command that removes an E1/T1 on the A interface.

Note

1. The E1/T1 to be removed must exist on the A interface.

2. When the specified port is configured with the SS7 signaling link, semipermanent link, monitoring timeslot, or inter-board E1 connection, the E1/T1 on the A interface cannot be removed.

3. If the specified port is configured with the link clock, the E1/T1 on the A interface cannot be removed.

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:No. of the TC.

Default value:None
SN Slot No.

Meaning:Slot No.

Value range:0~27

Explanation:No. of the slot of the board.

Default value:None
PN Port No.

Meaning:Port No.

Value range:0~83

Explanation:No. of the Port, 0~31(GEIUA), 0~83(GOIUA).

Default value:None

Example

Remove the E1/T1 on the A interface (Subrack No. = 0; Slot No. = 16; Port No. = 0):
RMV AE1T1: SRN=0, SN=16, PN=0;
