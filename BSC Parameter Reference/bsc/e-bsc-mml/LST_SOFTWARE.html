List Software Version(LST SOFTWARE)

Function

An OM command that queries the software version information on the GOMU/GBAM.

Note

1. The value of software version status can be Available or Unavailable. Value Available indicates that the software version saved on the GOMU/GBAM is complete. Value Unavailable indicates that the files required for the version are incomplete or damaged.

2. When the standby area is empty, the message of only the active area is returned. In other words, there is only one record.

Parameters

None.

Example

Query the software version saved on the OMU/GBAM:
LST SOFTWARE:;
The returned result is as follows: 
MML% LST SOFTWARE:;
+++    HUAWEI        2007-07-11 11:33:23
O&M    #97480
%%LST SOFTWARE:;%%
RETCODE = 0  Operation succeeded

Results of the operation:
--------------------
Storage Area              Software version        Software Status

Main    Area              BSC6000V900R008C01      Available
Standby Area              BSC6000V900R008C11      Available
(Number of results = 2)


---    END

Status of software version (Available/Unavailable)
