Modify Idle Advanced Parameters of Cell(MOD CELLIDLEAD)

Function

A configuration command that modifies the advanced idle parameters of the cell. In this command, Cell Index or Cell Name uniquely identifies a cell.

Note

None

Parameters

ID Name Description
IDXTYPE Index Type

Meaning:Index Type

Value range:BYNAME(By Name),BYIDX(By Index)

Explanation:Index Type.

Default value:None
CELLIDX Cell Index

Meaning:Cell Index

Value range:0~2047

Explanation: This parameter indicates the source cell index. The source cell index must be unique.

Default value:None
CELLNAME Cell Name

Meaning:Cell Name

Value range:1-32 characters

Explanation: This parameter indicates the name of the source cell. The name of each cell under one BSC must be unique.

Default value:None
ACS Additional reselect parameter indication

Meaning:Additional reselect parameter indication

Value range:COMPUL0(Compulsory 0),COMPUL1(Compulsory 1),FLEX(Flexible)

Explanation: Additional Reselect Param Indication (ACS) is used to inform an MS where cell reselection parameters can be obtained.

When the value of ACS is "0", the MS obtains the value of PI and C2 parameters from the remaining characters in system information type 4.

When the value of ACS is "1", the MS obtains the value of PI and C2 parameters from the remaining characters in system information type 7 or 8.

"Compulsory 0" indicates that the value of ACS in the "Cell Selection Parameters" information element should always be "0" in the system information type 4.

"Compulsory 1" indicates that the value of ACS in the "Cell Selection Parameters" information element should always be "1" in the system information type 4.

"Flexible" indicates that the value of ACS can be either "0" or "1". If the system information type 4 cannot carry all the cell reselection parameters, the value of ACS is "1". In this case, parts of the cell reselection parameters are carried in the system information type 7 or 8. Otherwise, the value of ACS is "0". In this case, all the cell reselection parameters are carried in the system information type 4.

Default value:None
TO Cell reselect temporary offset

Meaning:Cell reselect temporary offset

Value range:0~7

Explanation:This parameter specifies the temporary correction of C2. This parameter is valid only before the penalty time of cell reselection expires. For details, see GSM Rec. 0508 and 0408.

This parameter applies only to GSM Phase II MSs.

Default value:None
PT Cell reselect penalty time[s]

Meaning:Cell reselect penalty time[s]

Value range:0~31

Explanation:The Cell Reselect Penalty Time (PT for short) is used to ensure the safety and validity of cell reselection because it helps to avoid frequent cell reselection. For details, see GSM Rec. 05.08 and 04.08.

This parameter applies to only GSM Phase II MSs.

Default value:None

Example

Modify the advanced idle parameters of the cell (Index Type = BYIDX(By Index); Cell Index = 0; Additional reselect parameter indication = YESYES; Call reselect temporary offset = 3; Call reselect penalty time[s] = 16): 
MOD CELLIDLEAD: IDXTYPE=BYIDX, CELLIDX=0, ACS=YES, TO=3, PT=16;
