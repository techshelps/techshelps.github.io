Set Status of Local User Management(SET OPSW)

Function

A configuration command that sets the status of the local user management (enabled or disabled).

Note

None.

Parameters

ID Name Description
SWOP state of local operator management

Meaning: status of the local user management

Value range: off(disable), on(enable)

Explanation:

1. If the local user management is disabled, other local users except for admin cannot add, remove, or modify users.

2. If the local user management is enabled, the authorized user can add, remove, or modify users.

Default value: on(enable)

Example

1. Enable the NM function of local operators:
SET OPSW: SWOP=ON;
2. Disable the NM function of local operators:
SET OPSW: SWOP=OFF;
