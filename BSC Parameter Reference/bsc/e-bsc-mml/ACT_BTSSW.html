Activate BTS Software(ACT BTSSW)

Function

An OM command that used to activate BTS software.

Note

1. The BTS software loaded to boards runs after being activated.Activation refers to the process where boards obtain the required data or software information from the flash.You can activate more than one BTS in one operation with & character.

2. You must activate the software of the main control board first and then activate the software of other boards.

3. The BTS software is successfully loaded to the flash of BTS boards.

Parameters

ID Name Description
IDXTYPE Index type

Meaning: index type

Value range: BYNAME(By Name), BYIDX(By Index)

Explanation: This parameter specifies whether to activate a BTS by BTS name or BTS index.

Default value: None
BTSIDXLST BTS Index list

Meaning: BTS index list

Value range: 0-2047

Explanation: This parameter specifies the number of the BTS. Each BTS has a unique number in a BSC. Multiple BTS indexes can be quoted with the symbol &.

Default value: None
BTSNAMELST BTS Name list

Meaning: BTS name list

Value range: 1-2048 characters

Explanation: This parameter specifies the name of the BTS. Each BTS has a unique name in a BSC. Multiple BTS names can be quoted with the symbol &.

Default value: None
VERTYPE Version Index type

Meaning: index type

Value range: BYBTSSWVER, BYBTSBRDSWVER

Explanation: 0: Value BYBTSSWVER specifies that the BTS is indexed by the BTS software version; value BYBTSBRDSWVER specifies that and the BTS is indexed by the board software version.

Default value: None
BTSSWVER BTS Software Version

Meaning: BTS software version

Value range: 1-32 characters

Explanation: This parameter specifies the BTS software version, which is the name of the folder for storing the decompressed BTS software.

Default value: None
BRDSWVER BTS Board Software Version

Meaning: board software version

Value range: 1-32 characters

Explanation: The board software version is written in the vercfg.xml file stored under the folder for the decompressed BTS software.

Default value: None
BRDSWTYPE BTS Board Software type

Meaning: board software type

Value range: DATM_MAIN, DATU_MAIN, DDPM_MAIN, DDPU_MAIN, DDRM_MAIN, DEMU_MAIN, DFCU_MAIN, DGPS_MAIN, DHEU_MAIN, DOMU_MAIN, DPMU_MAIN, DPTU_MAIN, DRFU_MAIN, DRFUBOOT_MAIN, DRRU_MAIN, DRRUBOOT_MAIN, DRU_MAIN, DTCU_MAIN, DTMU_MAIN, DTRU_MAIN, ETR_MAIN, FMU_MAIN, FMUA_MAIN, GATM_MAIN, GTMU_MAIN, IASU_MAIN, IOMU_MAIN, MFU_3001C, MMU_3001C, MRRU_MAIN, MTR_MAIN, NFCB_MAIN, PBU_MAIN, QTRU_MAIN, TMU_MAIN, TRX_MAIN, GRRU_MAIN, BTS3900B_MAIN, BTS3900B_ROM, MRFU_MAIN, GRFU_MAIN, MRRUBOOT_MAIN, MRFUBOOT_MAIN, USCU_MAIN, BTS3900E_MAIN, BTS3900E_ROM
BRDLST BTS Board List

Meaning: BTS board list

Value range: 0-71

Explanation: This parameter specifies the number of the BTS board list.

Default value: None
REPORT Report Progress Flag

Meaning: whether to report the progress

Value range: REPORT, NOTREPORT

Explanation: This parameter specifies whether to report the progress.

Default value: REPORT

Example

1. BTSIDXLST is 5,BRDSWVER is "V100R001C04B099SP02",Activate BRDSWTYPE is  "NFCB_MAIN" by BYBTSBRDSWVER:
ACT BTSSW:IDXTYPE=BYIDX, BTSIDXLST=5, VERTYPE=BYBTSBRDSWVER, BRDSWVER="V100R001C04B099SP02", BRDSWTYPE=NFCB_MAIN;
