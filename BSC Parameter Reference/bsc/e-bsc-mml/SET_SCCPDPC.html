Set SCCP DPC Admin Status(SET SCCPDPC)

Function

A configuration command that modifies the administrative state of the SCCP subsystem.

Note

If the parameter SCCP DPC adminstatus is set to INHIBIT(INHIBIT), the services of the SCCP DPC become unavailable. Perform this operation with caution.

Parameters

ID Name Description
DPCINDEX SCCP DPC index

Meaning:SCCP DPC index

Value range:0~288

Explanation:DpcIndex.

Default value:None
ADMINSTATUS SCCP DPC adminstatus

Meaning:SCCP DPC adminstatus

Value range:INHIBIT(INHIBIT),UNINHIBIT(UNINHIBIT)

Explanation:Used to change the management status of SCCPDPC, to inhibit or to uninhibit.

Default value:None

Example

Set the administation status of the SCCP DPC(SCCP DPC index = 0; SCCP DPC adminstatus = INHBIT(INHBIT);):
SET SCCPDPC: DPCINDEX=0, ADMINSTATUS=INHIBIT;
