Add Ater OML(ADD ATEROML)

Function

A configuration command that adds an Ater OML.

Note

1. The Ater OML can be configured only when the GTCS is configured remotely.

2. Before configuring the Ater OML, configure the Ater connection path.

3. The BM and TC subracks where the Ater OML is configured must be main subracks.

4. Except for timeslot 1, the Ater OML must have at least four successive timeslots.

5. At most two Ater OMLs can be configured in the entire system.

Parameters

ID Name Description
ATERPIDX Ater Connection Path Index

Meaning:Ater Connection Path Index

Value range:0~254

Explanation:This parameter specifies the index of an Ater connection path.

Default value:None
TSMASK Timeslot Mask

Meaning:Timeslot Mask

Value range:TS0(0),TS1(1),TS2(2),TS3(3),TS4(4),TS5(5),TS6(6),TS7(7),TS8(8),TS9(9),TS10(10),TS11(11),TS12(12),TS13(13),TS14(14),TS15(15),TS16(16),TS17(17),TS18(18),TS19(19),TS20(20),TS21(21),TS22(22),TS23(23),TS24(24),TS25(25),TS26(26),TS27(27),TS28(28),TS29(29),TS30(30),TS31(31)

Explanation:Config TS.

Default value:None

Example

1. Add an Ater OML (Ater Connection Path Index = 0; Timeslot Mask = TS2(Timeslot 2), TS3(Timeslot 3), TS4(Timeslot 4), TS5(Timeslot 5), TS6(Timeslot 6), TS7(Timeslot 7), TS8(Timeslot 8)):
ADD ATEROML: ATERPIDX=0, TSMASK=TS2-1&TS3-1&TS4-1&TS5-1&TS6-1&TS7-1&TS8-1;
