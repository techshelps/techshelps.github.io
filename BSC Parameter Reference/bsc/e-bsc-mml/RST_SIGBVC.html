Reset SIGBVC(RST SIGBVC)

Function

An OM command that resets a specified SIG BVC.

Note

1. Resetting the SIG PVC disrupts all the PTP BVC services. Thus, run this command with caution.

2. The result contains two reports. The first report indicates that the command is run successfully. The second report is displayed upon the response from the SGSN or when the time expires.

Parameters

ID Name Description
NSEI NSE Id

Meaning: NSE ID

Value range: 0-65534

Explanation:

1. This parameter specifies the ID of the network service identity.

2. An NSEI uniquely identifies an NSE.

Default value: None

Example

Reset a SIG BVC (NSE Id = 2):
RST SIGBVC: NSEI=2;
The result is as follows:
+++    HUAWEI        2008-04-26 13:26:45
O&M    #2976
%%RST SIGBVC: NSEI=2;%%
RETCODE = 0  Operation succeeded


Subsequent reports to be output

---    END

%%RST SIGBVC: NSEI=2;%%
RETCODE = 0  Operation succeeded


2 reports in total
---    END
