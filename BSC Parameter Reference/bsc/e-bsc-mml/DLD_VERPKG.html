Download and Decompress Installation Package(DLD VERPKG)

Function

An OM command that downloads the self-decompressing software version package from the FTP server to the GOMU/GBAM. After the downloading is complete, the GOMU/GBAM triggers the self-decompressing action.The FTP server is the PC that is installed with LMT or M2000 server software.

Note

The file name entered is case sensitive and must be the same as the name of the file to be downloaded.

Parameters

ID Name Description
IP FTPServer IP

Meaning: IP address of the FTP server

Value range: 0.0.0.0-255.255.255.255

Explanation: This parameter specifies the IPv4 type as xxx.xxx.xxx.xxx.

Default value: None
USR FTPServer user

Meaning: user name used to log in to the FTP server

Value range: 0-24 characters

Explanation: 1.This parameter is a character string *******.

2.The default user name for logging in to the FTP server to which the LMT is connected is swmtest.

Default value: None
PWD User password

Meaning: user password used to log in to the FTP server

Value range: 0-24 characters

Explanation: 1.This parameter is a character string *******.

2.The default password for logging in to the FTP server to which the LMT is connected is swmtest.

Default value: None
DIR Source directory

Meaning: the FTP server directory where the files to be downloaded are stored

Value range: 1-127 characters

Explanation: This parameter is a character string *:\***\***.

Default value: None
SV Software version

Meaning: software version

Value range: 0-63 characters

Explanation: This parameter is a character string *******.

Default value: None

Example

Download and extract an installation package: 
DLD VERPKG:IP="80.1.102.111",USR="swmtest",PWD="swmtest",DIR="Common\SWM\software";
