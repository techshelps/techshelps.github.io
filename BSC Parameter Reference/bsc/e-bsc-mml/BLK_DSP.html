Block DSP(BLK DSP)

Function

An OM command that blocks a specified DSP.

Note

1. The call on this DSP may have a call drop after the resource release time specified by the user. Therefore, use this command with caution.

2. The GDPUC or GDPUX must be in position and run properly.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: This parameter specifies the subrack number that uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None
DSPN Dsp No.

Meaning: DSP number

Value range: any 22-digit number

Explanation: The subrack number, slot number, and DSP number uniquely identify a destination signaling point.

Default value: None
TIME Time[s]

Meaning: resource release time

Value range: 0-255

Description: resource release time that the DSP is required to wait for, unit: second

Default value: None

Example

Block the DSPs (Subrack No. = 1; Slot No. = 8; Dsp No. = 0&1; Time[s] = 20):
BLK DSP: SRN=1, SN=8, DSPN=DSP0-1&DSP1-1, Time=20;

Output Description

None.
