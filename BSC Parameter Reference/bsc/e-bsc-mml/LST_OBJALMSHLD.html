List Conditions for Shielding Alarms(LST OBJALMSHLD)

Function

An OM command that queries the alarms shielded by alarm ID or by object.

Note

The result may be displayed in multiple reports.

Parameters

ID Name Description
QRMD Query mode

Meaning: query mode for the shielded alarms

Value range: OBJ(OBJ)

Explanation: This parameter whether to query shielded alarms by object.All shielded alarms related to the specified object are listed.

Default value: None

Example

1. Query the shielding conditions by object:
LST OBJALMSHLD: QRMD=OBJ;
The result is as follows:
+++    HUAWEI        2008-07-21 15:01:42
O&M    #74598
%%LST OBJALMSHLD: QRMD=OBJ;%%
RETCODE = 0  Operation Result:Operation succeeded

Corresponding results not found

---    END

Output Description

Domain Name Description
Object Type

Object type including BTS, cell and BSC board
Cell Index

Global index to a cell
BTS Index

Global index to a BTS
Subrack No.

Number of the subrack where the BSC board is located
Slot No.

Number of the slot that holds the BSC board
Port No.

Number of the port
Alarm IDn

Alarm ID (n is a sequence number beginning with 1)
