Release Configuration Rights(REL CFGRIGHT)

Function

An OM command that releases the configuration right after the configuration is completed. REL CFGRIGHT is opposite to LCK CFGRIGHT.

Note

1. This command can release only the configuration right of the same operation terminal. It cannot release the configuration right of another operation terminal. If you want to release configuration right forcibly, run the FOE CFGRIGHT command.

2. Release the configuration right after the configuration operation is complete. Otherwise, other operation terminals will not be able to perform any configuration operations.

3. If the configuration right is not previously applied by this terminal, the failure message containing the information of the configuration right application is displayed when you run the command.

Parameters

None.

Example

Release the configuration rights:
REL CFGRIGHT:;
