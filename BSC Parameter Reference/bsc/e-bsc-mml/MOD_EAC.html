Modify BSC EAC(MOD EAC)

Function

A configuration command that modifies the parameters of the environment monitoring device. In this command, Subrack No. uniquely identifies the EAC.

Note

1. If the attributes of the environment monitoring device support floating-point input, the input precision must be two significant digits.

2. If the attributes of the environment monitoring device support floating-point input, the value range is -32768.00 to 32767.00.

Explanation:This parameter specifies the number of the subrack where the environment monitoring instrument is located.

If the parameter is set to an overgreat or oversmall value, some alarms may not be reported. It is recommended that you retain the default value.

Default value:None

Example

1. Modify the parameters of the BSC environment monitoring device (Subrack No. = 0; Switch for Relay 1 = OPEN(Open)):
MOD EAC: SRN=0, POWER_RELAY1=OPEN;
2. Modify the parameters of the BSC environment monitoring device (Enable Voltage Alarm Reporting for 24 V Power Supply = OPEN(Open)):
MOD EAC: VOL24_MASK=OPEN;
