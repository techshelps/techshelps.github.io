Display Time(DSP TIME)

Function

An OM command that queries the current NE time.

Note

None

Parameters

None.

Example

Query the current NE time:
DSP TIME:;
The result is as follows:
+++    HUAWEI                2004-02-03 18:21:31
O&M    #24
%%DSP TIME:;%%
RETCODE = 0  Operation succeeded

Time information
----------------
Time  =  2004-02-03 18:21:31 +08:00 DST

---    END

Output Description

Domain Name Description
Time

The time is expressed in the Year-month-day and hh:mm:ss format. For the GMT east and west time zone, a positive value (+) indicates east time zones and a negative value (-) indicates west time zones. Daylight saving time flag DST appears if it is daylight saving time
