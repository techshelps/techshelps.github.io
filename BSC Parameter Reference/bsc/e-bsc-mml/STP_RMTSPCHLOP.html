Stop Remote Audio Loop(STP RMTSPCHLOP)

Function

An OM command that stops the remote speech channel loopback.

Note

1. The BSC must communicate properly with the BTS.

2. The BTS must be configured with TRXs.

3. At least one MS is in conversation.

4. When stopping the loopback on the remote speech channel, set the parameter User Stop Loop Type to the same as that for starting the loopback on the remote speech channel. For example, if the parameter User Start Loop Type is set to BYMSISDN when the loopback is started, User Stop Loop Type must be set to BYMSISDN when the loopback is stopped. You can run the DSP RMTSPCHLOP command to query the result of the loopback on the remote speech channel.

Parameters

ID Name Description
IDTYPE User Stop Loop Type

Meaning: mode of stopping loopback

Value range: BYTMSI, BYIMSI, BYMSISDN, BYIMEI

Explanation: This parameter specifies the mode of stopping the loopback of the remote speech channel.

Default value: None
TMSI TMSI

Meaning: TMSI

Value range: 0-4294967295

Explanation:

1. This parameter specifies the TMSI. You need to set this parameter when User Stop Loop Type is set to BYTMSI.

2. The length of the parameter must be eight digits.

Default value: None
IMSI IMSI

Meaning: IMSI

Value range: 5-15 characters

Explanation:

1. This parameter specifies the IMSI. You need to set this parameter when User Stop Loop Type is set to BYIMSI.

2. The value of this parameter ranges from 00000 to 999999999999999.

Default value: None
MSISDN MSISDN

Meaning: MSISDN

Value range: 3-16 characters

Explanation:

1. This parameter specifies the MSISDN. You need to set this parameter when User Stop Loop Type is set to BYMSISDN.

2. The value of this parameter ranges from 000 to 9999999999999999.

Default value: None
IMEI IMEI

Meaning: IMEI

Value range: 8-15 characters

Explanation:

1. This parameter specifies the IMEI. You need to set this parameter when User Stop Loop Type is set to BYIMEI.

2. The value of this parameter ranges from 00000000 to 999999999999999.

Default value: None

Example

Stop the loopback on the remote speech channel (User Stop Loop Type = BYMSISDN; MSISDN = 13988880048):
STP RMTSPCHLOP: IDTYPE=BYMSISDN, MSISDN="13988880048";
