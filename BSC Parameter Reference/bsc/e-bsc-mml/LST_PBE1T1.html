List Pb Interface E1/T1(LST PBE1T1)

Function

An OM command that queries the configuration of the E1/T1 link on the Pb interface.

To query the configuration of all the E1/T1 links on the Pb interfaces in a specific subrack, set the parameter Subrack No..

To query the configuration of the E1/T1 link on the Pb interface according to specified conditions, set the parameters Subrack No., Slot No., and Port No..

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:None.

Default value:None
SN Slot No.

Meaning:Slot No.

Value range:0~27

Explanation:None.

Default value:None
PN Port No.

Meaning:Port No.

Value range:0~83

Explanation:None.

Default value:None
LSTTYPE List Type

Meaning:List Type

Value range:SUMMARY(Summary Information),DETAIL(All Information)

Explanation:List Type.

Default value:SUMMARY

Example

1. Query the E1/T1 links on the Pb interface (Subrack No. = 0):
LST PBE1T1: SRN=0;
