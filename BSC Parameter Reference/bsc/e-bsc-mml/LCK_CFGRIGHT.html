Lock Configuration Rights(LCK CFGRIGHT)

Function

An OM command that requests the configuration rights from the system. You can only configure the terminal that obtains the configuration right.

Note

1. The configuration right is exclusive in the system. That is, only one terminal can obtain the configuration right, and other application requests will be rejected. Therefore, the configuration user needs to run the REL CFGRIGHT command to release the configuration right immediately after the use.

2. If the configuration right is already obtained by another terminal, then a message is displayed, indicating that the configuration right is obtained by another terminal.

Parameters

None.

Example

Lock the configuration rights:
LCK CFGRIGHT:;
