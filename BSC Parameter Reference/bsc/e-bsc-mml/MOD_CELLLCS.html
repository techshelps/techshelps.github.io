Modify Cell LCS Parameters(MOD CELLLCS)

Function

A configuration command that modifies the LCS parameters of the cell. In this command, Cell Index or Cell Name uniquely identifies a cell.

Note

The int part and decimal part of the latitude and longitude parameters are set separately.

Parameters

ID Name Description
IDXTYPE Index Type

Meaning:Index Type

Value range:BYNAME(By Name),BYIDX(By Index)

Explanation: Index Type.

Default value:None
CELLIDX Cell Index

Meaning:Cell Index

Value range:0~2047

Explanation: This parameter indicates the source cell index. The source cell index must be unique.

Default value:None
CELLNAME Cell Name

Meaning:Cell Name

Value range:1-32 characters

Explanation: This parameter indicates the name of the source cell. The name of each cell under one BSC must be unique.

Default value:None
INPUTMD Latitude and Longitude input mode

Meaning:Latitude and Longitude input mode

Value range:Degree(degree),D_min_sec(degree minute second)

Explanation:This parameter specifies the input type of the longitude and latitude where a cell is located. The input type can be either degree or multi-degree.

Default value:None
NSLATI NS Latitude

Meaning:NS Latitude

Value range:North_latitude(North latitude),South_latitude(South latitude)

Explanation:This parameter specifies whether the cell location is latitude north or latitude south.

Default value:None
LATIINT Integer of Latitude

Meaning:Integer of Latitude

Value range:0~90

Explanation:This parameter specifies the integer part of the latitude where a cell is located. The input type of latitude is degree.

Default value:None
LATIDECI Fraction of Latitude

Meaning:Fraction of Latitude

Value range:1-5 characters

Explanation:This parameter specifies the fraction part of the latitude where a cell is located. The input type of latitude is degree.

Default value:None
WELONGI WE Longitude

Meaning:WE Longitude

Value range:East_Longitude(East Longitude),West_Longitude(West Longitude)

Explanation:This parameter specifies whether the cell location is longitude east or longitude west.

Default value:None
LONGIINT Integer of Longitude

Meaning:Integer of Longitude

Value range:0~180

Explanation:This parameter specifies the integer part of the longitude where a cell is located. The input type of longitude is degree.

Default value:None
LONGIDECI Fraction of Longitude

Meaning:Fraction of Longitude

Value range:1-5 characters

Explanation:This parameter specifies the fraction part of the longitude where a cell is located. The input type of longitude is degree.

Default value:None
LATIDEGREE Degree of Latitude

Meaning:Degree of Latitude

Value range:0~90

Explanation:This parameter specifies the degree of the latitude where a cell is located. The input type of latitude is multi-degree.

Default value:None
LATIMINUTE Minute of Latitude

Meaning:Minute of Latitude

Value range:0~59

Explanation:This parameter specifies the minute of the latitude where a cell is located. The input type of latitude is multi-degree.

Default value:None
LATISECONDINT Integer of Latitude Seconds

Meaning:Integer of Latitude Seconds

Value range:0~59

Explanation:This parameter specifies the integer part of the second of the latitude where a cell is located. The input type of latitude is multi-degree.

Default value:None
LATISECONDDECI Fraction of Latitude Seconds

Meaning:Fraction of Latitude Seconds

Value range:0~9

Explanation:This parameter specifies the fraction part of the second of the latitude where a cell is located. The input type of latitude is multi-degree.

Default value:None
LONGIDEGREE Degree of Longitude

Meaning:Degree of Longitude

Value range:0~180

Explanation:This parameter specifies the degree of the longitude where a cell is located. The input type of longitude is multi-degree.

Default value:None
LONGIMINUTE Minute of Longitude

Meaning:Minute of Longitude

Value range:0~59

Explanation:This parameter specifies the minute of the longitude where a cell is located. The input type of longitude is multi-degree.

Default value:None
LONGISECONDINT Integer of Longitude Seconds

Meaning:Integer of Longitude Seconds

Value range:0~59

Explanation:This parameter specifies the integer part of the second of the longitude where a cell is located. The input type of longitude is multi-degree.

Default value:None
LONGISECONDDECI Fraction of Longitude Seconds

Meaning:Fraction of Longitude Seconds

Value range:0~9

Explanation:This parameter specifies the fraction part of the second of the longitude where a cell is located. The input type of longitude is multi-degree.

Default value:None
ANTAANGLE Antenna Azimuth Angle[degree]

Meaning:Antenna Azimuth Angle[degree]

Value range:0~360

Explanation:This parameter is calculated according to the Included Angle and the actual Antenna Azimuth Angle. The Included Angle refers to the coverage area of the cell.

Antenna Azimuth Angle = actual Antenna Azimuth Angle - Included Angle/2.

Default value:None
INCLUDEANG Included Angle[degree]

Meaning:Included Angle[degree]

Value range:0~360

Explanation:This parameter specifies the included angle formed by the major lobe azimuth of the antennas in two cells under one BTS. The major lobe azimuth is measured from the north to the direction of the cell antenna in a clockwise direction.

Default value:None
ALTITUDE Integer of Antenna Altitude[m]

Meaning:Integer of Antenna Altitude[m]

Value range:0~6553

Explanation:This parameter specifies the integer part of the altitude of the cell antenna.

Default value:None
ALTIDECI Fraction of Antenna Altitude

Meaning:Fraction of Antenna Altitude

Value range:0~9

Explanation:Fraction of Antenna Altitude.

Default value:None

Example

Modify the LCS parameters of the cell (Index Type = BYIDX(By Index); Cell Index = 0, NS Latitude = North_latitude(North latitude)):
MOD CELLLCS: IDXTYPE=BYIDX, CELLIDX=0, NSLATI=North_latitude;
