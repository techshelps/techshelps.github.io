Display DSP Link Status(DSP DSPLINK)

Function

An OM command that queries the status of a specified DSP link or multiple DSP links.

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation:

1. This parameter specifies the subrack number of the DSP link.

2. The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation:

1. This parameter specifies the slot number of the DSP link.

2. The subrack number and slot number uniquely identify a board.

Default value: None
DSPNO DSP No.

Meaning: DSP number

Value range: 0-21

Explanation: This parameter specifies the number of the DSP.

Default value: None
LINKNO Link No.

Meaning: link number

Value range: 0-191

Explanation: This parameter specifies the link number of the DSP.

Default value: None

Example

1. Query the status of a DSP link (Subrack No. = 0; Slot No. = 0; DSP No. = 0), and do not set the Link No.:
DSP DSPLINK: SRN=0, SN=10, DSPNO=0;
The result is as follows:

Output Description

Domain Name Description
Subrack No.

Subrack number of the DSP link
Slot No.

Slot number of the DSP link
DSP No.

DSP number
DSP Link No.

DSP link number
DSP Link State

DSP link status, value range: No Loopback , Local Loopback, Auxiliary Loopback. No Loopback indicates that the DSP link is normal; Local Loopback indicates that the signals that should be in the transmitting direction are in the receiving direction as the received data; Auxiliary Loopback indicates that the DSP link facilitates other loopback.
Link Assignment State

Link assignment status, value range: Unavailable, Normal and Not Assigned, Assigned. Unavailable indicates that the DSP link is not available; Normal and Not Assigned indicates that the DSP link is normal and idle; Assigned indicates that the DSP link is occupied.
Link Synchronization State

Link synchronization status, value range: Unsynchronized, Synchronized, Out-of-Sync., Test, Inactive. Unsynchronized indicates that the link is in the initial status; Synchronized indicates that the link on the Abis interface is synchronized with the BTS; Out-of-Sync. indicates that the synchronization fails and the link on the Abis interface is in the out-of-synchronization status; Test indicates that the link is used for test; Inactive indicates that the link is not used.
