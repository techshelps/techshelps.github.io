Add PTPBVC(ADD PTPBVC)

Function

A configuration command that adds a PTP BVC.

Note

1. The specified cell and the NSE ID must exist.

2. When the SGSN Pool function is enabled, a maximum of 32 PTP BVCs can be configured in a cell.

2. When the SGSN Pool function is disabled, only one PTP BVC can be configured in a cell.

3 One NSE ID can correspond to 2048 PTP BVCs.

4. This command can be run only when the built-in PCU is configured in the BSC.

Parameters

ID Name Description
NSEI NSE Identifier

Meaning:NSE Identifier

Value range:0~65534

Explanation: This parameter specifies the identifier of a network service entity (NSE).

Default value:None
BVCI PTPBVC Identifier

Meaning:PTPBVC Identifier

Value range:2~65534

Explanation: This parameter specifies the BSSGP virtual connection identifier. It is used to identify multiple BVCs that multiplexes the NS-VC.

Default value:None
IDXTYPE Index Type

Meaning:Index Type

Value range:BYCELLIDX(By Cell Index),BYCELLNAME(By Cell Name)

Explanation:Index Type.

Default value:None
CELLIDX Cell Index

Meaning:Cell Index

Value range:0~2047

Explanation: This parameter indicates the source cell index. The source cell index must be unique.

Default value:None
CELLNAME Cell Name

Meaning:Cell Name

Value range:1-32 characters

Explanation: This parameter indicates the name of the source cell. The name of each cell under one BSC must be unique.

Default value:None

Example

1. Add a PTP BVC (NSE Identifier = 1; PTPBVC Identifier = 2; Index Type = BYCELLNAME(By Cell Name); Cell Name = cell):
ADD PTPBVC: NSEI=1, BVCI=2, IDXTYPE=BYCELLNAME, CELLNAME="cell";
