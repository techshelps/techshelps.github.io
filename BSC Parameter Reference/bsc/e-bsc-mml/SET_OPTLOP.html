Set OIU Port Tributary Loopback Mode(SET OPTLOP)

Function

An configuration command that sets the loopback mode of the OIU port.

Note

Setting the loopback mode may affect the services carried on the port, so use this command with caution.

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation:

1. This parameter specifies the number of a subrack.

2. This is a key parameter that identifies the port loopback setting of an optical-port board and specifies the subrack number of the optical-port board for port loopback.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation:

1. This parameter specifies the number of a slot.

2. This is a key parameter that identifies the port loopback setting of an optical-port board and specifies the slot number of the optical-port board for port loopback.

Default value: None
SUBBRDN Sub-board No.

Meaning: sub-board number

Value range: 0

Explanation: This parameter specifies the sub-board number of the optical-port board for port loopback.

Default value: 0
PN Port No.

Meaning: port number

Value range: 0-83

Explanation:

1. This parameter specifies the number of a port.

Description: This is a key parameter that identifies the port loopback operation on the optical-port board and specifies the optical-port number of the optical-port board for port loopback.

Default value: None
MODE Loopback mode

Meaning: port loopback mode

Value range: NOLOOP(no loopback); HWLOOP(lowordpathline loopback); LINELOOP(triline loopback); SYSLOOP(trisystem loopback), SIN_CHN_LOC(singlechn local loopback); SIN_CHN_REM(tdm timeslot remote loopback)

Explanation: This is a key parameter that identifies the port loopback setting of the optical-port board and specifies the port loopback mode of the optical-port board for port loopback.

Default value: None
LCN Loopback channel

Meaning: loopback channel number when the port loopback mode is TDM remote timeslot loopback

Value range: time slots 1-31

Explanation:

1. This parameter specifies the port loopback channel number of the optical-port board for port loopback.

2. This parameter supports multiple options.

Default value: None
LCN Loopback channel

Meaning: loopback channel number when the port loopback mode is single channel local loopback

Value range: time slots 1-31

Explanation:

1. This parameter specifies the port loopback channel number of the optical-port board for port loopback.

2. This parameter does not support multiple options.

Default value: None
AUSTOP Loopback time

Meaning: loopback period (minute)

Value range: 1-65535

Description: This is a key parameter that identifies the port loopback of an optical-port board and specifies the duration of the port loopback of the optical-port board.

Default value: 60

Example

1. Set the loopback mode for an OIU port (Subrack No. = 0; Slot No. = 14; Sub-board No. = 0; Port No. = 0; Loopback mode = SYSLOOP(trisystem loopback); Loopback time = 30):
SET OPTLOP: SRN=0, SN=14, SUBBRDN=0, PN=0, MODE=SYSLOOP, AUSTOP=30;
2. Set the loopback mode for an OIU port (Subrack No. = 0; Slot No. = 14; Port No. = 0; Sub-board No. = 0; Loopback mode = SIN_CHN_REM(tdm timeslot remote loopback); Loopback channel = TS2(Time Slot 2); Loopback time = 45):
SET OPTLOP: SRN=0, SN=14, PN=0, MODE=SIN_CHN_REM, LCN=TS2-1, AUSTOP=45;
