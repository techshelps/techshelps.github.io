Add Line Clock(ADD LINECLK)

Function

A configuration command that adds a line clock.

Note

1. If the line clock is configured on the GEIUT, the line clock must be configured on the Ater connection path; if the line clock is configured on the GOIUT, the line clock must be configured on port 0 of the GOIUT.

2. If the line clock is configured on the GEIUA, the line clock must be configured on the E1 port on the A interface; if the line clock is configured on the GOIUA, the line clock must be configured on port 0 of the GOIUA.

2. The line clock cannot be configured in the GEPS.

Parameters

ID Name Description
SRN Subrack No.

Meaning:Subrack No.

Value range:0~11

Explanation:None.

Default value:None
SN Slot No.

Meaning:Slot No.

Value range:0~27

Explanation:None.

Default value:None
LINENO Line Clock No.

Meaning:Line Clock No.

Value range:LINE0(Line 0),LINE1(Line 1)

Explanation:Line No.

Default value:None
PN Port No. of line clock

Meaning:Port No. of line clock

Value range:0~31

Explanation:This parameter specifies the number of the port through which the line clock is transmitted.

Default value:None
BAKPN Backup port No. of line clock

Meaning:Backup port No. of line clock

Value range:0~31

Explanation:This parameter specifies the number of the backup port through which the line clock is transmitted.

Default value:None

Example

1. Add a line clock for the GOIUT (Subrack No. = 0; Slot No. = 18; Line Clock No. = 0; Port No. of line clock = 0):
ADD LINECLK: SRN=0, SN=18, LINENO=LINE0, PN=0;
2. Add a line clock for the GEIUA (Subrack No. = 1; Slot No. = 16; Line Clock No. = 1; Port No. of line clock = 1; Backup port No. of line clock):
ADD LINECLK: SRN=1, SN=16, LINENO=LINE1, PN=0, BakPN=1;
