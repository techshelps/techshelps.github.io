Remove Device IP Address(RMV DEVIP)

Function

A configuration command that removes a device IP address.

Note

1. If a board is configured with a route, then the device IP address cannot be removed.

2. The IP address to be removed cannot be used by the M3UALNK, NSVLLOCAL, and PTUPARA.

Parameters

ID Name Description
IPADDR IP address

Meaning:IP address

Value range:0.0.0.0-255.255.255.255

Explanation: This parameter specifies the equipment IP address of an IP-based interface board.

Default value:None

Example

Remove the device IP address (IP address = 3.3.3.3): 
RMV DEVIP: IPADDR="3.3.3.3";
