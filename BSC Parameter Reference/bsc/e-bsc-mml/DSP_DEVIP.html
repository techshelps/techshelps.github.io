Display Device IP Address(DSP DEVIP)

Function

An OM command that queries the device IP addresses over the A, Abis, and Gb interfaces.

Note

Each GFGUA/GFGUB/GFGUG/GOGUA/GOGUB has only one IP address, so active and standby boards share the same IP address.

Parameters

ID Name Description
SRN Subrack No.

Meaning: number of the subrack for running the command

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: number of the slot for running the command

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None
SSN SubSystem No.

Meaning: subsystem number

Value range: 0-4

Explanation: Number of the subsystem where the device IP address is located. You can query this parameter through the LST DEVIP MML command.

Default value: None

Example

Query the device IP address of a board (Subrack No. = 0; Slot No. = 24):
DSP DEVIP: SRN=0, SN=24;
The result is as follows:
+++    HUAWEI        2007-12-17 15:51:18
O&M    #459
%%DSP DEVIP: SRN=0, SN=24;%%
RETCODE = 0  Operation succeeded
