Download TC File(DLD TCFILE)

Function

An OM command that downloads TC resource files in the specified directory to the DSP.

Note

1. This command consumes certain system resources.

2. Place the files to be downloaded in the root directory of the FTP server. The root directory of the FTP server is the directory of the executable file on the FTP server. If you use the built-in FTP server of the LMT, the FTP server is positioned at HW LMT\BSC6000 .The specified TC filename must be complete without any wildcard character.

3. The FTP server is started.

4. The specified DSP must be in position and work normally.

Parameters

ID Name Description
IP FTPServer IP

Meaning: IP address of the FTP server

Value range: 0.0.0.0-255.255.255.255

Explanation: This parameter specifies the IP address of the FTP server.

Default value: None
USR FTPServer user

Meaning: user name of the FTP server

Value range: 1-16 characters

Explanation: This parameter specifies the user name of the FTP server.

Default value: None
PWD User password

Meaning: password of the FTP server

Value range: 1-16 characters

Explanation: This parameter specifies the password of the FTP server.

Default value: None
NM File name

Meaning: file name

Value range: 4-62 characters

Explanation:

Meaning: This parameter specifies the name of the file to be downloaded.

2. The input filename must be complete without wildcards.

Default value: None
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: This parameter specifies the subrack number that uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-3/8-13

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None
DSPN Dsp No.

Meaning: destination signaling point number

Value range: 0-21

Explanation: The subrack number, slot number, and DSP number uniquely identify a destination signaling point.

Default value: None

Example

Download the TC file (FTPServer IP = 10.161.69.65; FTPServer User = swmtest; User Password = *******; Download File Name = 011011002-20070630005340-FFF7_AR.dat; Subrack No. = 1; Slot No. = 10; Dsp No. = 0):
DLD TCFILE: IP="10.161.69.65", USR="swmtest", PWD="*******", NM="011011002-20070630005340-FFF7_AR.dat", SRN=1, SN=10, DSPN=0;
Explanation:
If the DSP runs improperly, MML Client prompts DSP-not-in-position error.
Only a TC file can be downloaded at a time.
