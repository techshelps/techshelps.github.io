Query Remote Audio Loop(DSP RMTSPCHLOP)

Function

An OM command that displays the result of a remote speech channel loopback.

Note

1. The communication between the BSC and the BTS is normal.

2. The BTS is configured with the TRX and supports speech service.

3. One or more MSs are in conversation state.

Parameters

None.

Example

Result of the remote speech channel loopback:
DSP RMTSPCHLOP:;
The result is as follows:
