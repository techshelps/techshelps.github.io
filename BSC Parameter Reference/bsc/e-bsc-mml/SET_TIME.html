Set Time(SET TIME)

Function

A configuration command that sets the system time of the NE.

Note

Set the date in the YYYY&MM&DD format, and set the time in the HH&MM&SS format.

Parameters

ID Name Description
DATE Date

Meaning: date

Value range: YYYY&MM&DD

Explanation: This parameter specifies the date and time in the YYYY&MM&DD format.

Default value: None
TIME Time

Meaning: time

Value range: HH&MM&SS

Explanation: This parameter sets the time in the HH&MM&SS format.

Default value: None

Example

Set time for the NE. Take Dec.9,2007 as an example, and the time is 11:52:01.
SET TIME: DATE=2007&11&09, TIME=11&52&01;
