Display Ethernet Port Loopback Mode(DSP ETHLOP)

Function

An OM command that queries the loopback mode of an Ethernet port.

Note

In the active/standby mode, the active and standby boards use different physical slot numbers but the same logical slot number. The slot number specified for querying the loopback mode refers to the physical slot number.

Parameters

ID Name Description
SRN Subrack No.

Meaning: number of the subrack for running the command

Value range: 0-11

Explanation: The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: number of the slot for running the command

Value range: 0-27

Explanation: The subrack number and slot number uniquely identify a board.

Default value: None
PN Port No.

Meaning: port number

Value range: 0-7

Explanation:

1. For the GFGUA/GFGUB/GFGUG, the value range is 0 to 7.

2. For the GOGUA/GOGUB, the value range is 0 to 1.

Default value: None

Example

Query the loopback modes of all the ports (Subrack No. = 0; Slot No. = 24):
DSP ETHLOP: SRN=0, SN=24;
The result is as follows:
+++    HUAWEI        2007-12-17 08:42:40
O&M    #1751
%%DSP ETHLOP: SRN=0, SN=24;%%
RETCODE = 0  Operation succeeded

Loop mode of the Ethernet port Value range: No_Loop, Line_Loop, Sys_Loop
