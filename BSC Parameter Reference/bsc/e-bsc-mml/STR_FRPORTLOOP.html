Start FR Port Loopback(STR FRPORTLOOP)

Function

An OM command that starts the FR port loopback. After the FR port loopback starts, the BSC periodically sends loopback packets.

Note

None

Parameters

ID Name Description
SRN Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation:

1. This parameter specifies the subrack where the loopback is initiated.

2. The subrack number uniquely identifies a subrack.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-27

Explanation:

1. This parameter specifies the slot number where the loopback is initiated.

2. The subrack number and slot number uniquely identify a board.

Default value: None
PN Port No.

Meaning: port number

Value range: 0-31

Explanation: This parameter specifies the port where the FR loopback is initiated.

Default value: None
BCID BC Id

Meaning: BC ID

Value range: 0-255

Explanation: This parameter specifies the BC where the FR loopback is initiated.

Default value: None
LOOPMODE Loop Type

Meaning: loopback mode

Value range: BYLOCAL(By Local Loop), BYREMOTE(By Remote Loop)

Explanation: This parameter specifies the loopback mode. BYLOCAL indicates local loopback and BYREMOTE indicates remote loopback.

Default value: None
TMLEN Time Length[m]

Meaning: loopback duration

Value range: 1-120

Explanation: This parameter specifies the loopback duration (unit: minute).

Default value: None
PKTLEN Packet Length[byte]

Meaning: Size of the loopback data packet

Value range: 2-1500

Explanation:

1. This parameter specifies the size of the loopback data packet.

2. The unit of this parameter is byte.

Default value: None

Example

Start the FR port loopback (Subrack No. = 0; Slot No. = 22; Port No. = 0; BC Id = 0; Loop Type = BYLOCAL(By Local Loop); Time Length[m] = 5; Packet Length[byte] = 6):
STR FRPORTLOOP: SRN=0, SN=22, PN=0, BCID=0, LOOPMODE=BYLOCAL, TMLEN=5, PKTLEN=6;
