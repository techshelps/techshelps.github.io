DSP Elabel(DSP ELABEL)

Function

An OM command that queries the electronic label of a BSC board, a BSC fan, BSC power, or a BTS board.

Note

1. The MML client must communicate properly with the BSC.

2. The BSC must communicate properly with the BTS.

3. If you set Subrack No. to the number of the subrack where GTMU locates and set Slot No. to 23, the electronic label of the BBU backboard is listed.

Parameters

ID Name Description
DEVTYPE Device Type

Meaning: type of the device

Value range: BSCBRD(BSC BRD), BSCFAN(BSC FAN), BSCPOWER(BSC POWER), BTSBRD(BTS BRD)

Explanation: The value BSCBRD indicates a BSC board; the value BSCFAN indicates a BSC fan; the value BSCPOWER indicates a BSC power module; the value BTSBRD indicates a BTS board.

Default value: None
CN Cabinet No.

Meaning: number of a BSC cabinet

Value range: 0-3

Explanation: This parameter specifies the number of a BSC cabinet.

Default value: None
IDXTYPE Index Type

Meaning: how to specify a BTS

Value range: BYNAME(By Name), BYIDX(By Index)

Explanation: This parameter determines whether to specify a BTS by name or by index.

Default value: None
BTSIDX BTS Index

Meaning: BTS index

Value range: 0-2047

Explanation:

1. This parameter specifies the number of the target BTS. Each BTS is numbered uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYIDX(By Index).

Default value: None
BTSNAME BTS Name

Meaning: BTS name

Value range: 1-32 characters

Explanation:

1. This parameter specifies the name of the target BTS. Each BTS is named uniquely in a BSC.

2. This parameter is mandatory if Index Type is set to BYNAME(By Name).

Default value: None
SRN Subrack No.

Meaning: subrack number

Value range: 0-53

Explanation: This parameter specifies the number of the subrack where a BTS board is located.

Default value: None
SN Slot No.

Meaning: slot number

Value range: 0-23

Explanation: This parameter specifies the number of the slot that holds a BTS board.

Default value: None
SUBRACKNO Subrack No.

Meaning: subrack number

Value range: 0-11

Explanation: This parameter specifies the number of the subrack where a BSC board is located.

Default value: None
SLOTNO Slot No.

Meaning: number of the slot that holds a BSC board

Value range: 0-27

Explanation: This parameter specifies the number of the slot that holds a BSC board.

Default value: None

Example

Query the electronic label (Device Type = BSCBRD(BSC BRD); Subrack No. = 0; Slot No. = 19):
DSP ELABEL: DEVTYPE=BSCBRD, SUBRACKNO=0, SLOTNO=19;
