Activate License File(ACT LICENSE)

Function

An OM command that activates the specified license file.

Note

1. When the NE receives the ACT LICENSE command, it sets the specified license file to the current license file and reads the license file again to complete the setting. If the specified license file is the current license file, the NE still returns the Operation Succeeded information and reads the license file again to complete the setting.

2. This command usually works together with the DLD LICENSE command to decipher and check the newly downloaded license file and send it to the boards.

Parameters

ID Name Description
FN File name

Meaning: license file name

Value range: 1-31 characters

Explanation: This parameter specifies the name of the license file loaded on the board.

Default value: name of the loaded license file

Example

Activate license file OR8290.DAT:
ACT LICENSE:FN="OR8290.DAT";
