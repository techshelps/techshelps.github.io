Display OMU Ethernet Adapter Information(DSP OMUNIC)

Function

An OM command that queries the Ethernet adapter information of the GOMU.

Note

1. Only the Ethernet adapter information of the GOMU can be queried.

2. The LOCAL process on the GOMU should be operational.

Parameters

ID Name Description
TG Target

Meaning: identifier of the active or standby GOMU

Value range: ACTIVE(Active), STANDBY(Standby)

Explanation: ACTIVE(Active) indicates the active GOMU, and STANDBY(Standby) indicates the standby GOMU.

Default value: None

Example

Query the Ethernet adapter information of the active GOMU:
DSP OMUNIC: TG=ACTIVE;
The result is as follows:
MML% DSP OMUNIC:TG=ACTIVE;

Output Description

Domain Name Description
Subrack No.

Number of the subrack where the board is located.
Slot No.

Number of the slot where the board is located.
Ethernet Name

Ethernet name.
Operation Name

Operation name.
Ethernet IP Address

IP address of the Ethernet port.
Ethernet Link Status

Connection status of the Ethernet port. The status can be Connect, Disconnect, or Down.
Ethernet MAC Address

MAC address of the Ethernet port.
