List Patch Information(LST PATCH)

Function

An OM command that queries the patch information.

Note

The query result includes the hot patches for the primary software in both active and standby areas.

Parameters

None.

Example

Query the patch information on the OMU/GBAM:
LST PATCH:;
The returned result is as follows: 
MML% LST PATCH:;
+++    HUAWEI        2007-07-11 11:15:19
O&M    #97417
%%LST PATCH:;%%
RETCODE = 0  Operation succeeded

BSC6000V900R008C01SP01      BSC6000V900R008C01SP01          Idle           
(Number of results = 2)
