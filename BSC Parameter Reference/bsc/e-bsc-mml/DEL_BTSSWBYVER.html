Delete BTS Software by Version(DEL BTSSWBYVER)

Function

An OM command that requests the system to delete the BTS software by version

Note

1. Only the BTS software positioned in the directory /BSC6000/btsswm/ can be deleted.

2. The BTS software must be positioned in the folder named by version that is in the directory /BSC6000/btsswm/.

3. The folder corresponding to the specified version and the files in the folder are deleted.

4. The BTS software directly positioned in the directory /BSC6000/btsswm/ is not deleted.

Parameters

ID Name Description
VER Version

Meaning: BTS software version

Value range: 1-32 characters

Explanation: This specifies the BTS software version to be removed.

Default value: None

Example

Delete the BTS software with the version BTS3000V100R008C01B242:
DEL BTSSWBYVER:VER=BTS3000V100R008C01B242;

If the specified file is not in the directory, MML Client prompts no file.
