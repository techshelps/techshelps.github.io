Upload BATCHFILE Result(ULD BATCHFILERST)

Function

An OM command that uploads output files of batch configuration commands from NEs to servers. During the uploading, a progress bar is displayed to indicate the upload progress.

Note

1. Ensure that the specified session ID exists.

2. Ensure that a batch configuration command session is activated before running the command.

3. Do not run this command during the activation of the batch configuration command session.

4. After a batch configuration command session is activated, only the Output file of this batch configuration command session activation can be uploaded. The Output file of the previous batch configuration command session activations cannot be uploaded.

Parameters

ID Name Description
SID SessionID

Meaning: session ID

Value range: 0-268435455

Explanation: Session ID identifies a session.

Default value: None
FT File type

Meaning: file type

Value range: PREACTIVATE_RESULT, ACTIVATE_RESULT, ROLLBACK_RESULT, other

Explanation: PREACTIVATE_RESULT indicates the pre-active file type; ACTIVATE_RESUL indicates the activated file type; TROLLBACK_RESULT indicates the rollback file type; OTHER indicates other file types.

Default value: None
DSTF Destination file name

Meaning: destination file name

Value range: 1-100 characters

Explanation:

1. This parameter specifies the name and path of the destination file on the FTP server.

2. The file name contains a maximum number of 31 characters (excluding the path).

Default value: None
IP FTPServer IP

Meaning: IP address of the FTP server

Parameter format: 0.0.0.0-255.255.255.255

Explanation: the IP address of the FTP Server on the MML Client.

Default value: None
USR FTPServer user

Meaning: user name of the FTP server

Maximum length: 1-16 characters

Explanation: This parameter specifies the user name of the FTP server.

Default value: None
PWD User password

Meaning: user password

Maximum length: 1-64 characters

Explanation: This parameter specifies the user password of the FTP server.

Default value: None

Example

Upload the output file of batch configuration command from the NE to the server (10.0.0.10): 
ULD BATCHFILE:SID=111111,USR="xxx",PWD="xxx",IP="xxx.xxx.xxx.xxx",
FT=ACTIVATE_RESULT,DSTF="batchfil_001_act.txt";
