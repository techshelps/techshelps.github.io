Download Backup Configuration Data File(DLD CFGFILE)

Function

An OM command that downloads backup configuration data files to the OMU.

Note

1.The file name (case sensitive) set must be the same as the name of the file to be downloaded.

2.The BSC6000 does not have the Area flag.

Parameters

ID Name Description
IP FTPServer IP

Meaning: IP address of the FTP server

Value range: 0.0.0.0-255.255.255.255

Explanation: the IP address of the FTP Server on the MML Client.

Default value: None
USR FTPServer user

Meaning: user name of the FTP server

Value range: 1-32 characters

Explanation: This parameter specifies the user name of the FTP server.

Default value: None
PWD User password

Meaning: user password

Value range: 1-64 characters

Explanation: This parameter specifies the user password of the FTP server.

Default value: None
DIR Directory name

Meaning: directory name

Value range: 1-256 characters

Explanation: This parameter specifies the directory of the configuration data file on the FTP server (excluding the file name).

Default value: FTP root directory for downloading
FN File name

Meaning: file name

Value range: 1-32 characters

Explanation:

1. For a network management terminal, this parameter is optional.

2. For other terminals, this parameter is mandatory and cannot be named M2000.dat.

Default value: None
FT File type

Meaning: backup file type

Value range: BIN(BIN format)

Explanation: This parameter specifies the backup file type.

Default value: BIN(BIN format)
AF Area flag

Meaning: active/standby area indicator

Value range: STANDBY(Standby),ACTIVE(Active)

Explanation: This parameter is not listed in the value range for the BSC6000 NEs.

Default value: STANDBY(Standby)
GA Progress flag

Meaning: whether to report the progress

Value range: Y(YES), N(NO)

Explanation: Y(YES) means to report the progress; N(NO) means not to report the progress.

Default value: Y(YES)

Example

1. Download the configuration data on the M2000 to the OMU:
DLD CFGFILE:IP="XXX.XXX.XXX.XXX",USR="swmtest",PWD="swmtest";
2. Download the configuration data on the LMT to the OMU:
DLD CFGFILE:IP=10.161.69.65,USR=swmtest,PWD=swmtest,FN="011000043-20070630011926";
