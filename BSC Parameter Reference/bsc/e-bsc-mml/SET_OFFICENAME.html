Set System Name(SET OFFICENAME)

Function

A configuration command that sets the office name.

1. The entire NE has only one office name, which is also called source identifier.

2. This parameter generally specifies the location of an equipment.

3. The office names in all MML messages are displayed as the office name set by this command.

4. The initial office name is HUAWEI.

5. The office name is the string behind the +++ identifier in a message, which is shown as follows:

SET OFFICENAME:SID="BSC6000";

+++ BSC6000 2008-05-21 17:31:00

O&M #61815

%%SET OFFICENAME:SID="BSC6000";%%

RETCODE = 0 Operation succeeded

--- END

Note

None.

Parameters

ID Name Description
SID System name

Meaning: office name

Value range: 2-64 characters

Explanation: the office name to be specified

Default value: None

Example

Set the current office name to BSC6000:
SET OFFICENAME:SID="BSC6000";
