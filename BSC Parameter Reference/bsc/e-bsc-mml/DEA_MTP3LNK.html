Deactivate MTP3 Link(DEA MTP3LNK)

Function

An OM command that deactivates a specified MTP signaling link.

Note

Deactivating an MTP link disables the link to carry signaling services, so use this command with caution.

Parameters

ID Name Description
SIGLKSX Signal linkset index

Meaning: signaling link set identifier

Value range: 0-255

Explanation: One of the key fields of an MTP link that indicates the link set of the MTP link to be inhibited. You can obtain the field through the LST N7LNK command.

Default value: None
SIGSLC Signal link code

Meaning: signaling link identifier

Value range: 0-15

Explanation: One of the key fields of an MTP3 link that indicates the MTP link set to be inhibited. You can obtain the field through the LST N7LNK command.

Default value: None

Example

Deactivate an MTP3 link (Signal linkset index = 0; signal link code = 1):
DEA MTP3LNK: SIGLKSX=0,SIGSLC=1;
