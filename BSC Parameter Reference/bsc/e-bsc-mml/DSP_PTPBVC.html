Display PTPBVC Status(DSP PTPBVC)

Function

An OM command that queries the status of a specified PTP BVC.

Note

None

Parameters

ID Name Description
NSEI NSE Id

Meaning: NSE ID

Value range: 0-65534

Explanation:

1. This parameter specifies the ID of the network service identity.

2. An NSEI uniquely identifies an NSE.

Default value: None
BVCI BVC Id

Meaning: BVC ID

Value range: 2-65534

Explanation:

1. This parameter specifies the ID of the BSSGP virtual connection.2. A BVCI uniquely identifies a BVC.

Default value: None

Example

1. Query the status of a PTP BVC (NSE Id = 178; BVC Id = 100):
DSP PTPBVC: NSEI=178, BVCI=100;
The result is as follows:
+++    HUAWEI        2007-12-19 13:31:54
O&M    #10552
%%DSP PTPBVC: NSEI=178, BVCI=100;%%
RETCODE = 0  Operation succeeded

PTPBVC Administration State  =  Unblocked
       PTPBVC Service State  =  Abnormal 
(Number of results = 1)

---    END

2. Query the status of a PTP BVC (NSE Id = 178; BVC Id = 101):
DSP PTPBVC: NSEI=178, BVCI=101;
The result is as follows:
+++    HUAWEI        2007-12-19 13:33:39
O&M    #10616
%%DSP PTPBVC: NSEI=178, BVCI=101;%%
RETCODE = 0  Operation succeeded

PTPBVC Administration State  =  Unblocked
       PTPBVC Service State  =  Abnormal 
(Number of results = 1)

---    END

Output Description

Domain Name Description
PTPBVC Administration State

Administration status of the PTP BVC, value range: Blocked, Unblocked
PTPBVC Service State

Service status of the PTP BVC, value range: Normal, Abnormal, Reset. Normal indicates that the PTP BVC can carry PS services; Abnormal indicates that the PTP BVC cannot carry PS services; Reset indicates that the PTP BVC is reset.
