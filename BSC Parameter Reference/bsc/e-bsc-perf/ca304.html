Call Setup Indications Timed Out

Measurement Counter

CA304: CELL_ESTB_IND_EXP

Description

This measurement provides the number of Call Setup Indications Timed Out.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, each time the BSC does not receive an EST IND but the counter expires after the BSC sends an IMM ASS CMD message to the measured cell (see measurement point A), the counter is incremented by one.

Figure 1 Immediate assignment procedure

Formula

Call Setup Indications Timed Out = [Call Setup Indications Timed Out (SDCCH) + [Call Setup Indications Timed Out (TCHF)] + [Call Setup Indications Timed Out (TCHH)]
