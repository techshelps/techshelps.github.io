Number of Reclaimed Dynamic PDCHs

Measurement Counter

R9343: RECLAIM_DYN_PDCH

Description

This measurement provides the number of times the BSC converts a dynamic PDCH to a TCH in a cell in a granularity period. This counter indicates the number of times that the PDCH is converted to the TCH when the CS service pre-empts the PS service in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, every time the BSC converts a dynamic PDCH to a TCH, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
