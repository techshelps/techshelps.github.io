Total Number of Uplink EGPRS MCS4 RLC Data Blocks

Measurement Counter

L9205:UP_EGPRS_MCS4_RLC_DATA_BLK

Description

This measurement provides the total number of uplink EGPRS RLC data blocks using MCS-4 coding scheme in a cell in a granularity period.

Unit

Integer number or integer.

Measurement Point

The total number of uplink EGPRS RLC data blocks using MCS-4 coding scheme that the BSC receives from the MS is measured as follows:
  1. If the BSC receives an uplink EGPRS RLC MCS-4 data block for the first time, and the data block in the V(N) on the BSC side is in NACKED state, you can infer that the data block has been sent more than once. In this case, the counter is incremented by two.
  2. If the BSC receives an uplink EGPRS RLC MCS-4 data block for the first time, and the data block in the V(N) on the BSC side is in UNRECEIVED state, you can infer that the data block has been sent once only. In this case, the counter is incremented by one.
  3. If the BSC receives an uplink EGPRS RLC MCS-4 data block for the first time, and the data block in the V(N) on the BSC side is in RECEIVED state, you can infer that the data block is received. In this case, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
