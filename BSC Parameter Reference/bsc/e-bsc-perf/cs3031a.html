Rate of MRs (Weak Coverage)

Measurement Counter

CS3031A: CELL_MR_WEAK_COVERAGE_RATIO

Description

This measurement provides the ratio of the number of MRs in a weakly-covered cell to the total number of MRs in a cell.
NOTE:

A weakly-covered cell specifies that the downlink level of the cell is smaller than the Cell Weak Coverage Downlink RxLev Threshold and the TA value is smaller than or equal to the Cell Weak Coverage TA Threshold.

Unit

%.

Measurement Point

After receiving valid MRs from the BTS in TCHs, the BSC calculates the ratio of the number of MRs in a weakly-covered cell to the total number of MRs in a cell. Then, the value of the counter is obtained.

Formula

This is an original counter without involving any formula.
