Total Number of Uplink EGPRS MCS3 RLC Data Blocks

Measurement Counter

L9204: UP_EGPRS_MCS3_RLC_DATA_BLK

Description

This measurement provides the total number of uplink EGPRS RLC data blocks using MCS-3 coding scheme in a cell in a granularity period.

Unit

Integer number or integer.

Measurement Point

The uplink EGPRS RLC data blocks using MCS-3 coding scheme that the BSC receives from the MS are categorized into three types: MCS-3 data blocks, retransmitted MCS-6 data blocks with MCS-3 coding scheme , and retransmitted MCS-9 data blocks with MCS-3 coding scheme. The total number of uplink EGPRS RLC data blocks using MCS-3 coding scheme is measured as follows:
  • When the MCS-3 data blocks that the BSC receives from the MS is not used to retransmit MCS-6 or MCS-9 data blocks:
    1. If the BSC receives an uplink EGPRS RLC MCS-3 data block for the first time, and the data block in the V(N) on the BSC side is in NACKED state, you can infer that the data block has been sent several times. In this case, the counter is incremented by two.
    2. If the BSC receives an uplink EGPRS RLC MCS-3 data block for the first time, and the data block in the V(N) on the BSC side is in UNRECEIVED state, you can infer that the data block has been sent once only. In this case, the counter is incremented by one.
    3. If the BSC receives an uplink EGPRS RLC MCS-3 data block for the first time, and the data block in the V(N) on the BSC side is in RECEIVED state, you can infer that the data block is received. In this case, the counter is incremented by one.
  • When the MCS-3 data blocks that the BSC receives from the MS is used to retransmit MCS-6 data blocks:
    1. If part of a data block using MCS-6 coding scheme is received, and the rest part of the data block using MCS-3 coding scheme is also received, then the counter is incremented by one.
    2. If all the data blocks using MCS-6 coding scheme (including all the radio data blocks using MCS-3 coding scheme) are received, the counter is incremented by one.
    3. If part of a data block using MCS-6 coding scheme is received, and the rest part of the data block using MCS-3 coding scheme is also received, then the counter is incremented by one.
  • When the MCS-3 data blocks that the BSC receives from the MS is used to retransmit MCS-9 data blocks:
    1. If part of a data block using MCS-9 coding scheme is received, and the rest part of the data block using MCS-3 coding scheme is also received, then the counter is incremented by one.
    2. If all the data blocks using MCS-9 coding scheme (including all the radio data blocks using MCS-3 coding scheme) are received, the counter is incremented by one.
    3. If part of a data block using MCS-9 coding scheme is received, and the rest part of the data block using MCS-3 coding scheme is also received, then the counter is incremented by one.

Formula

This is an original counter without involving any formula.
