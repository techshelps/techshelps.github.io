Successful TCH Seizures for MTC

Measurement Counter

CA3039C: CELL_TCH_SUCC_MTC

Description

This measurement counter provides the number of Successful TCH Seizures for MTC, including the Successful TCH Seizures for MTC (Signaling Channel) and the Successful TCH Seizures for MTC (Traffic Channel). The Successful TCH Seizures for MTC (Signaling Channel) is the difference between the Call Setup Indications (MTC) (TCH) and the Mode Modify Commands (MTC) (TCH). The Successful TCH Seizures for MTC (Traffic Channel) is the difference between the Assignment Requests (MTC) and the Failed Assignments during MTC on the A Interface (Including Directed Retry).

Unit

Integer number or integer.

Measurement Point

None.

Formula

Successful TCH Seizures for MTC =

([Call Setup Indications (MTC) (TCHF)] + [Call Setup Indications (MTC) (TCHH)]) -

([Mode Modify Commands (MTC) (TCHF)] + [Mode Modify Commands (MTC) (TCHH)) +

([Assignment Requests (MTC)] - [Failed Assignments during MTC on the A Interface (Including Directed Retry)])
