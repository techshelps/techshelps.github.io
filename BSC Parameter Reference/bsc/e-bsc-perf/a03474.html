Unsuccessful BSC Internal VGCS TCH Handovers with Successful Reversion

Measurement Counter

A03474: CELL_GCC_INTRABSC_HO_FAIL_RECONN_SUCC

Description

The intra-BSC VGCS TCH handover is classified into the internal intra-cell VGCS TCH to the dedicated channel handover and the internal inter-cell VGCS TCH handover. During the handover procedure, when the BSC waits for a HANDOVER SUCCEEDED message (in the case of the intra-cell handover), or when the BSC sends a HANDOVER COMMAND message to the MS, we can infer that the handover fails but the MS successfully returns to the original channel if the BSC receives a HANDOVER FAIL message from the MS. This measurement provides the number of intra-BSC VGCS TCH handover failures with successful reversions. Along with Unsuccessful BSC Internal VGCS TCH Handovers with Unsuccessful Reversion, this measurement reflects the ratio of successful reversions in case of handover failures.

Unit

Integer number or integer.

Measurement Point

(1) This counter is incremented by one if the BSC receives a HANDOVER FAIL message from the MS when the BSC waits for a HANDOVER SUCCEEDED message in the case of the intra-cell VGCS TCH to the dedicated channel handover.

(2) This counter is incremented by one if the BSC receives a HANDOVER FAIL message from the MS when the BSC waits for the inter-cell handover finish in the case of the inter-cell VGCS TCH handover.

Formula

This is an original counter without involving any formula.
