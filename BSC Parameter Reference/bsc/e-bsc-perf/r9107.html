Number of Release Requests of IP PATH or HDLC Bandwidth (16K)

Measurement Counter

R9107: IP_PATH_BANDWIDTH_REL_REQ_16K

Description

This measurement provides the number of times that the 16 kbit/s Abis bandwidth is released on the PDCH in a cell.

Unit

Integer number or integer.

Measurement Point

In Abis over IP or HDLC mode, the bandwidth application or release may be initiated for the adjustment of the PDCH coding scheme. Each time a 16 kbit/s bandwidth is released, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
