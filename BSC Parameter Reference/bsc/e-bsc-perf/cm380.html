Call Drops in Stable BTS Loopback State

Measurement Counter

CM380:CELL_TRAF_CH_STATIC_STATE_CALL_DROPS_BTS_LOOP

Description

After seizing a channel, the MS is in the stable BTS loopback state. This measurement provides the number of call drops due to various reasons.

Unit

Integer number or integer.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

Call Drops in Stable BTS Loopback State = Call Drops due to Connection Failures in Stable BTS Loopback State + Call Drops due to Error Indications in Stable BTS Loopback State + Call Drops due to Release Indications in Stable BTS Loopback State +Call Drops due to No MRs from MS in Stable BTS Loopback State + Call Drops due to Forced Handovers in Stable BTS Loopback State + Call Drops due to Abis Terrestrial Link Failure in Stable BTS Loopback State + Call Drops due to Equipment Failures in Stable BTS Loopback State
