Call Drop Ratio on TCH per cell (Excluding Handover)

Measurement Counter

ZTR304A: CELL_RATE_TCH_CALL_DROP_EXCLUDE_HO

Description

This measurement provides the ratio of TCH call drops (excluding handover) to successful TCH seizures (excluding handover).

Unit

%.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

Call Drop Ratio on TCH per cell(Excluding Handover) = [Call Drops on the Traffic Channels] x {100}/([Successful TCH Seizures (Traffic Channel)] + [Successful Incoming Internal Inter-Cell Handovers] + [Successful Incoming External Inter-Cell Handovers] - [Successful Outgoing Internal Inter-Cell Handover] - [Successful Outgoing External Inter-Cell Handovers])
