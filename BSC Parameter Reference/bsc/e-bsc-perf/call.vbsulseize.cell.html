VBS Uplink Seizure

Description

VBS Uplink Seizure is used to measure the VBS uplink call drops, the number of busy VBS uplink channels, and the traffic volume on the VBS uplink channels.

Measurement Object

Cells.

Measurement Counter

The following table lists the measurement counters related to VBS Uplink Seizure.

VBS TCH Uplink Call Drop

A03459: CELL_KPI_VBS_UL_TRAF_ERL
