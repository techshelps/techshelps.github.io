Immediate Assignment Commands per BSC (PS Service)

Measurement Counter

ZTA301H:BSCRPT_IMM_ASS_CMD_PACKET_SERV_TOTAL

Description

The measurement of this counter provides the total number of immediate assignment messages (PS services) in all the cells controlled by a BSC.

Unit

Integer number or integer.

Measurement Point

BSC counters are calculation counters. This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Immediate Assignment Commands (Packet Service)".

Formula

Immediate Assignment Commands per BSC (PS Service) = Immediate Assignment Commands (Packet Service)
