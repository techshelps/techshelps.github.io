Unsuccessful BSC Internal VGCS TCH Handovers with Unsuccessful Reversion

Measurement Counter

A03475: CELL_GCC_INTRABSC_HO_FAIL_RECONN_FAIL

Description

During the intra-BSC VGCS TCH handover procedure, if the radio link quality is bad or if the equipment failure occurs, the MS is disconnected from the originating cell before the MS establishes a connection with the target cell. This measurement provides the number of internal VGCS TCH handover failures with reversion failures. Related counter: Unsuccessful BSC Internal VGCS TCH Handovers with Successful Reversion

Unit

Integer number or integer.

Measurement Point

(1) For the inter-cell VGCS TCH handover: This counter is incremented by one if the BSC does not receive the message indicating the handover finish before the timer expires.

(2) For the inter-cell VGCS TCH handover: This counter is incremented by one if the BSC receives a CONNECTION FAILURE INDICATION message from the BTS during the BSC waits for the message indicating the handover finish.

(3) For the inter-cell VGCS TCH handover: This counter is incremented by one if the BSC receives an ERROR INDICATION message from the BTS during the BSC waits for the message indicating the handover finish.

(4) For the inter-cell VGCS TCH handover: If the target cell receives a HANDOVER DETECT message and does not receive a HANDOVER COMPLETE message before the timer expires, then the target cell requests the originating cell to release the resources. This counter is incremented by one when the originating cell receives the request from the target cell.

(5) For the intra-cell VGCS TCH to the dedicated channel handover: The BSC initiates the external handover procedure. A timer is triggered when the BSC sends a HANDOVER REQUIRED message to the MSC. If the timer expires, this counter is incremented by one.

(6) For the intra-cell VGCS TCH to the dedicated channel handover: The BSC starts a timer set to wait for a message indicating the handover finish when the BSC receives a HANDOVER COMMAND message from the MSC. If the timer expires, this counter is incremented by one.

(7) For the intra-cell VGCS TCH to the dedicated channel handover: After the BSC initiates the external handover procedure, this counter is incremented by one when the BSC sends a HANDOVER REQUIRED message to the MSC and the MSC responds with a HANDOVER REQUIRED REJECT message.

Formula

This is an original counter without involving any formula.
