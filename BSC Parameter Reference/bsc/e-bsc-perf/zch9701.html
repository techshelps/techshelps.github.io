Attempted Handovers per BSC (Directed Retry)

Measurement Counter

ZCH9701:BSCRPT_HO_REQ_DR

Description

This measurement provides the number of outgoing inter-cell handover requests of all the cells controlled by a BSC due to directed retry, including outgoing internal inter-cell handover requests and outgoing external inter-cell handover requests.

The handover can be performed between GSM900/850 and GSM900/850, between GSM1800/1900 and GSM1800/1900, between GSM900/850 and GSM1800/1900, or between GSM1800/1900 and GSM900/850.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.

Formula

Attempted Handovers per BSC (Directed Retry) =

Outgoing Internal Inter-Cell Handover Requests (Directed Retry) (900/850-900/850) +

Outgoing External Inter-Cell Handover Requests (Directed Retry) (1800/1900-1800/1900) +

Outgoing Internal Inter-Cell Handover Requests (Directed Retry) (900/850-1800/1900) +

Outgoing Internal Inter-Cell Handover Requests (Directed Retry) (1800/1900-900/850) +

Outgoing External Inter-Cell Handover Requests (Directed Retry) (900/850-900/850) +

Outgoing External Inter-Cell Handover Requests (Directed Retry) (1800/1900-1800/1900) +

Outgoing External Inter-Cell Handover Requests (Directed Retry) (900/850-1800/1900) +

Outgoing External Inter-Cell Handover Requests (Directed Retry) (1800/1900-900/850)
