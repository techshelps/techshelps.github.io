Call Setup Indications (Non-Call Type)

Measurement Counter

CA303B: CELL_ESTB_IND_NO_CA

Description

This measurement provides the number of EST IND messages received by the BSC from the BTS when the MS access reason indicated in the EST IND message is not MOC Non SMS, MOC SMS, MTC, Emergency Call, or Call Re-establishment.

Unit

%.

Measurement Point

As shown in Figure 1, each time the BSC receives an EST IND message in which the L3 message type is not MOC Non SMS, MOC SMS, MTC, Emergency Call, or Call Re-establishment from the BTS (see measurement point C), this counter is incremented by one.

Figure 1 Immediate assignment procedure

Formula

Call Setup Indications (Non-Call Type) = [Call Setup Indications (Location Updating) (SDCCH)] + [Call Setup Indications (IMSI Detach) (SDCCH)] + [Call Setup Indications (Packet Service) (SDCCH)] + [Call Setup Indications (Location Updating) (TCHF)] + [Call Setup Indications (IMSI Detach) (TCHF)] + [Call Setup Indications (Packet Service) (TCHF)] + [Call Setup Indications (Location Updating) (TCHH)] + [Call Setup Indications (IMSI Detach) (TCHH)] + [Call Setup Indications (Packet Service) (TCHH)] + [Call Setup Indications (SS) (SDCCH)] + [Call Setup Indications (LCS) (SDCCH)] + [Call Setup Indications (Others) (SDCCH)] + [Call Setup Indications (SS) (TCHF)] + [Call Setup Indications (LCS) (TCHF)] + [Call Setup Indications (Others) (TCHF)] + [Call Setup Indications (SS) (TCHH)] +| [Call Setup Indications (LCS) (TCHH)] + [Call Setup Indications (Others) (TCHH)]
