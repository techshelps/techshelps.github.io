Total Blocks Received over Abis Port

Measurement Counter

L01041: ABISE1_RBBTN

Description

This measurement provides the total number of error blocks received over the Abis port in a granularity period. Along with Error Blocks Received over Abis Port, this measurement can be used to calculate Rate of Error Blocks Received over Abis Port. Given a specified Total Blocks received over Abis Port, the transmission quality deteriorates when the value of Error Blocks Received over Abis Port increases.

Unit

Integer number or integer.

Measurement Point

In a granularity period, this counter is measured when an error block is received over the Abis port.

Formula

This is an original counter without involving any formula.
