Number of Packet Paging Requests Received from the Gb Interface

Measurement Counter

A9601: RX_GB_PS_PAGING_REQ

Description

This measurement provides the number of packet paging requests received from the Gb interface in a granularity period.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure of the packet paging request initiated by the SGSN. Each time the BSC receives the PS PAGING REQUEST message from the Gb interface (see measurement point A), the counter is incremented by one. In addition, the SGSN can determine which cell to initiate the paging according to the NSEI or BVCI information element carried in the PS PAGING REQUEST message.

Figure 1 PS paging request received from the Gb interface

Formula

This is an original counter without involving any formula.
