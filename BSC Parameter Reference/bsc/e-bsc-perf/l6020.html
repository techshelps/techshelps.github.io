64 kbit/s Timeslots Seized by Signaling Link

Measurement Counter

L6020:MTP3_LNK_CIRCUITS

Description

This measurement provides the number of 64 kbit/s timeslots seized by a signaling link.

Unit

Integer number or integer.

Measurement Point

The measurement point is on the BSC side, as shown in the figure.

When the signaling links are established (the number of 64 kbit/s timeslots used for each link is not modified dynamically), the counter is measured.



Formula

This is an original counter without involving any formula.
