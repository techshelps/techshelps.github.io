Successful VGCS Assignment

Measurement Counter

A03406: CELL_VGCS_ASS_SUCC

Description

During the establishment of the VGCS, the MSC sends a VGCS ASSIGNMENT REQUEST message to the BSC for setting up a VGCS TCH. If the assignment is successful, the BSC sends a VGCS ASSIGNMENT RESULT message to the MSC. Otherwise, the BSC sends a VGCS ASSIGNMENT FAILURE message to the MSC. This measurement provides the number of VGCS ASSIGNMENT RESULT messages sent by the BSC. This counter is related to VGCS Assignment Request.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC sends a VGCS ASSIGNMENT RESULT message to the MSC. In the signaling procedure, this counter is measured at B in Figure 1, B in Figure 2, and point D in Figure 3.
