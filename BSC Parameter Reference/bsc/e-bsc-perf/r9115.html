Number of Unsuccessful Application Attempts of Abis Timeslot for the limit of BTS DSP

Measurement Counter

R9115: CIC_APPLY_FAIL_TIMES_DSP_LIMTED

Description

This measurement provides the number of unsuccessful applications for the Abis timeslot because of the insufficient DSP capability of the BTS. The DSP capability restriction of the BTS indicates that the BTS limits the maximum number of PDCHs on each link.

Unit

Integer number or integer.

Measurement Point

When the PDCH coding scheme is adjusted, the bandwidth application may be initiated to support the adjustment of the PDCH coding scheme. If the application of the Abis timeslot fails because of the insufficient DSP capability of the BTS, then the counter is incremented by one.

Formula

This is an original counter without involving any formula.
