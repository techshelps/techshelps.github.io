Attempted Handovers

Measurement Counter

CR443A: TRX_HO_CMD

Description

In the handover procedure, the BSC sends an HO CMD message through the source cell to the MS after receiving a CHANNEL ACTIVATION ACKNOWLEDGE message from the target cell. In the HO CMD message, the handover type can be SDCCH handover or TCH handover. At this time, the counter CR443A "Attempted Handovers" is incremented by one. This counter covers the number of HO CMD messages sent by the BSC when any channel of the measured TRX is used as the target channel in the handover procedure.

Unit

Integer number or integer.

Measurement Point

This counter is involved in the Intra-Cell Handover Measurement, Incoming Internal Inter-Cell Handover Measurement, Incoming External Inter-Cell Handover Measurement, and Incoming Inter-RAT Inter-Cell Handover Measurement. See the corresponding sections for the flow charts. Each time the BSC sends an HO CMD message to an MS, the counter is incremented by one.

Formula

Attempted Handovers = Attempted Handovers (SDCCH) + Attempted Handovers (TCH)
