Incoming External Inter-Cell Handover Responses

Measurement Counter

CH341:CELL_INTERBSC_INCELL_HO_REQ_ACK

Description

After receiving an HO REQ message from the MSC, the BSC requests a channel from the target cell and activates the channel. The BSC requests for speech service resources. If the request is successful, the BSC returns an HO REQ ACK message to the MSC.

This measurement provides the number of incoming external inter-cell handover responses. The measurement object is the target cell.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, the counter is measured at B when the target BSC (BSC2) receives the HO REQ ACK message from the MSC. The measurement object is the target cell.
