Call Drops in TCH Handovers (Traffic Channel)

Measurement Counter

K3012B:CELL_KPI_TCH_HO_DROPS_TRAF

Description

This measurement provides the number of call drops due to the Um interface failure during the handovers on the TCH carrying traffic.

Unit

Integer number or integer.

Measurement Point

Refer to Call Drops on Radio Interface in Handover State (TCH).

Formula

Call Drops in TCH Handovers (Traffic Channel) =[Call Drops on Radio Interface in Handover State (Traffic Channel)]
