Number of Uplink Frames Among the FR Frames in the Customized MR

Measurement Counter

S4550: TRX_EXT_FR_TCH_UP_NUM

Description

When decoding the MR, the BSC checks whether EI_FRAME cell is contained in the MS. If the EI_FRAME is found and it contains the number of FR frames, this measurement provides the number of uplink frames among the FR frames in the customized MR.

Unit

Integer number or integer.

Measurement Point

After receiving a customized extended MR or pre-processed customized extended MR from the BTS, the BSC decodes the MR. This counter is measured when the EI_FRAME cell is contained in the MS and the number of FR uplink frames is contained in the EI_FRAME cell.

Formula

This is an original counter without involving any formula.
