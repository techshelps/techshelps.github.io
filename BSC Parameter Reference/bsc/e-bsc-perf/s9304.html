Utilization Rate of GPRS Uplink Timeslots

Measurement Counter

S9304: CELL_USER_GPRS_TBF_UPLINK_TS_UTILI_RATE

Description

This measurement provides the utilization rate of timeslots allocated to the GPRS uplink TBFs in transmission state in a granularity period. This counter is used to calculate Average Utilization Rate of Uplink TBF Timeslots.

Unit

%.

Measurement Point

In a granularity period, the BSC periodically samples all the GPRS uplink TBFs in a cell. If the sampled GPRS uplink TBFs are in transmission state, the BSC calculates the maximum number of uplink timeslots based on the multislot capability and service type of the MS. In addition, the BSC calculates the number of timeslots that are actually allocated to uplink TBFs. Then, the number of actually allocated uplink timeslots is multiplied by 100 and then divided by the maximum number of uplink timeslots. The calculated value is accumulated to Utilization Rate of GPRS Uplink Timeslots.

Formula

This is an original counter without involving any formula.
