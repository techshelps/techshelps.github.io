Number of TCH to PDTCH Conversion Attempts

Measurement Counter

R9341: TCH_TO_PDTCH_ATMPT

Description

This measurement provides the number of conversion attempts from the TCH to the PDTCH in a cell in a granularity period. Based on the Number of TCH to PDTCH Conversion Attempts and Number of Successful TCH to PDTCH Conversions, the rate of successful dynamic PDTCH requests can be obtained. If the request for dynamic PDTCH fails, you can infer that the traffic of the BSC is too high or the channel state of the BSC is abnormal.

Unit

Integer number or integer.

Measurement Point

In a granularity period, every time a TCH attempts to be converted to a PDTCH, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
