Failed Assignments (TCH Assigned Repeatedly over A Interface)

Measurement Counter

CA312A: CELL_ASS_FAIL_TIMES_A_REP_TCH

Description

On receiving the assignment request message from the BSC, the BSC increases the counter value if the assignment of the TCH fails thought the assignment is completed before.

Unit

Integer number or integer.

Measurement Point

On receiving the repeated assignment request message from the MSC, the BSC triggers the measurement if the assignment fails this time. The counter is measured at D, as shown in the following figure.



Formula

This is an original counter without involving any formula.
