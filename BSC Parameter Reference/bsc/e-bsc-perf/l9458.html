Total Bytes of Received Uplink GPRS Information Messages

Measurement Counter

L9458: UPLINK_GPRS_INFO_TOTAL_BYTE

Description

This measurement provides the total bytes of uplink GPRS information messages in a granularity period.

Unit

byte.

Measurement Point

In a granularity period, every time the BSS receives a GPRS information message, packs this message into an LLC PDU, and then sends the LLC PDU to the SGSN, the bytes of the LLC PDU are accumulated to the counter.

Figure 1 Uplink GPRS information message sent to the SGSN

Formula

This is an original counter without involving any formula.
