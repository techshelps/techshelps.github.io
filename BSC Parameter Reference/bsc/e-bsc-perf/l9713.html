Number of Sent BLOCK ACK Messages

Measurement Counter

L9713: NSVC_TRAN_TX_BLOCK_ACK_MSG

Description

This measurement provides the number of BLOCK ACK messages sent from the BSC to the SGSN in a granularity period.

When the SGSN initiates the procedure for blocking the NSVC, the BSC sends an NSVC BLOCK ACK message to the SGSN after receiving an NSVC BLOCK message.

The value of the counter should be consistent with the number of received BLOCK messages. If not, check the link status and related data configuration of the Gb interface.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the counter is incremented by one every time the BSC sends an NSVC BLOCK ACK message on an NSVC to the SGSN (see measurement point B in Figure 1).
Figure 1 NSVC BLOCK ACK message sent by the BSC

Formula

This is an original counter without involving any formula.
