Description

During the outgoing inter-RAT inter-cell handover procedure, if the handover fails because of various causes before the BSC receives the CLEAR CMD message with the cause value of Ho Success from the MSC, the specific counter is measured based on the source channel types, handover failure causes, handover causes. Together with Outgoing Inter-RAT Inter-Cell Handover Requests and Outgoing Inter-RAT Inter-Cell Handover Commands, the measurement indicates the performance of outgoing inter-RAT inter-cell handovers.

The following counter provides the number of failed outgoing inter-RAT inter-cell handovers (excluding directed retry) due to various causes. The counter is measured upon the termination of the outgoing inter-RAT inter-cell handover.
  • H3520:CELL_INTERRAN_OUTCELL_HO_FAIL_NOT_INCLUDE_DR_SD
The following two counters provide respectively the number of failed outgoing inter-RAT inter-cell handovers when the MS seizes a TCHF or a TCHH before the handover. In this measurement, both the handovers initiated by directed retry and those initiated according to the handover algorithms based on the measurement reports are included. The counters are measured upon the termination of the outgoing inter-RAT inter-cell handover.
  • H3527:CELL_INTERRAN_OUTCELL_HO_FAIL_TCHF
  • H3528:CELL_INTERRAN_OUTCELL_HO_FAIL_TCHH

If the BSC enables the local switching function, then the calls that meet the local switching condition of the BSC use the local switching function of the BSC for the conversation and those that meet the local switching condition of the BTS use the local switching function of the BTS for the conversation.

The following two counters provide the number of failed outgoing inter-RAT inter-cell handovers initiated by the calls using the BSC local switch and those using the BTS local switch. The counters are measured upon the termination of the outgoing inter-RAT inter-cell handover.
  • H352Aa:CELL_INTERRAN_OUTCELL_HO_FAIL_BSC_LOOP
  • H352Ab:CELL_INTERRAN_OUTCELL_HO_FAIL_BTS_LOOP
NOTE:
The main causes of failed outgoing inter-RAT inter-cell handovers include:
  • (1) The timer T7 for the BSC to wait for an HO CMD message expires.
  • (2) The BSC receives an HO RQD REJ message from the MSC.
  • (3) The MS fails to access the new channel and then reconnects to the old channel.
  • (4) The timer T8 for the BSC to wait for an HO CMP message expires.
  • (5) The BSC receives a CLEAR CMD message from the MSC.
  • (6) SS7 link failure on the A interface
The following counters provide the number of failed outgoing inter-RAT inter-cell handovers due to the specific cause, and are measured upon the termination of the outgoing inter-RAT inter-cell handover.
  • (1) During the outgoing external inter-cell handover procedure, timer T7 is started after the BSC sends the HO RQD to the MSC and waits for the HO CMD command from the MSC. If no HO CMD is received by the BSC before T7 expires, the BSC re-sends the HO RQD message. The following counter provides the number of failed outgoing inter-RAT inter-cell handovers when the number of resending times has exceeded the maximum configuration.
    • H352L:CELL_INTERRAN_OUTCELL_HO_FAIL_T7_EXP
  • (2) The following counters provide the number of times that the BSC terminates the outgoing inter-RAT inter-cell handover because it receives an HO RQD REJ message from the MSC after sending the HO RQD message to the MSC.
    • The counter H352Ka:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_OM_INTERVENTION is measured when the cause value carried in the HO RQD REJ message is OM Intervention.
    • The counter H352Kb:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_EQUIP_FAIL is measured when the cause value carried in the HO RQD REJ message is Equipment Failure.
    • The counter H352Kc:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_NO_RADIO_RES is measured when the cause value carried in the HO RQD REJ message is No Radio Resource Available.
    • The counter H352Kd:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_REQ_NO_TER_RES is measured when the cause value carried in the HO RQD REJ message is Requested Terrestrial Resource Unavailable.
    • The counter H352Ke:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_BSS_NOT_EQUIP is measured when the cause value carried in the HO RQD REJ message is BSS not Equipped.
    • The counter H352Kf:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_INVAILD_CELL is measured when the cause value carried in the HO RQD REJ message is Invalid Cell.
    • The counter H352Kg:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_REQ_TRANSC_UNAIVL is measured when the cause value carried in the HO RQD REJ message is Requested Transcoding/Rate Adaption Unavailable.
    • The counter H352Kh:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_CIR_POOL_MISMATCH is measured when the cause value carried in the HO RQD REJ message is Circuit Pool Mismatch.
    • The counter H352Ki:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_REQ_NO_SV is measured when the cause value carried in the HO RQD REJ message is Requested Speech Version Unavailable.
    • The counter H352Kj:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_CIPH_ALG_NOT_SUPT is measured when the cause value carried in the HO RQD REJ message is Ciphering Algorithm not Supported.
    • The counter H352Kk:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_TER_CIR_ALRDY_ALLOC is measured when the cause value carried in the HO RQD REJ message is Terrestrial Circuit Already Allocated.
    • The counter H352Kl:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_INVAILD_MSG is measured when the cause value carried in the HO RQD REJ message is Invalid Message.
    • The counter H352Km:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_PROTOCOL_ERR is measured when the cause value carried in the HO RQD REJ message is Protocol Error between BSS and MSC.
    • The counter H352Kn:CELL_INTERRAN_OUTCELL_HO_FAIL_HO_REJ_OTHER is measured when the cause value carried in the HO RQD REJ message is Other Causes.
  • (3) In the outgoing inter-RAT inter-cell handover procedure, the BSC sends an HO CMD message to the MS through the originating cell and initiates timer T8 to wait for an HO CMP message. If the MS reconnects to the old channel and sends an HO FAIL message on the old channel before T8 expires, the specific one of the following counters is measured in the target cell based on the cause value.
    • The counter H352Da:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_ABNORM_REL_UNS is measured when the cause value carried in the HO FAIL message is Abnormal Release, Unspecified.
    • The counter H352Db:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_ABNORM_REL_CHN is measured when the cause value carried in the HO FAIL message is Abnormal Release, Channel Unacceptable.
    • The counter H352Dc:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_ABNORM_REL_T_E is measured when the cause value carried in the HO FAIL message is Abnormal Release, Timer Expired.
    • The counter H352Dd:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_ABNORM_REL_NO_ACT is measured when the cause value carried in the HO FAIL message is Abnormal Release, No Activity on the Radio Path.
    • The counter H352De:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_PREEMPT_RLS is measured when the cause value carried in the HO FAIL message is Preemptive Release.
    • The counter H352Df:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_HO_TA is measured when the cause value carried in the HO FAIL message is Handover Failed, Timing Advance out of Range.
    • The counter H352Dg:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_NO_CH_MODE is measured when the cause value carried in the HO FAIL message is Channel Mode Unavailable.
    • The counter H352Dh:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_FREQ is measured when the cause value carried in the HO FAIL message is Frequency Unavailable.
    • The counter H352Di:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_CALL_CLR is measured when the cause value carried in the HO FAIL message is Call Already Cleared.
    • The counter H352Dj:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_SEMANT is measured when the cause value carried in the HO FAIL message is Semantically Incorrect Message.
    • The counter H352Dk:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_INV_MAN is measured when the cause value carried in the HO FAIL message is Invalid Mandatory Information.
    • The counter H352Dl:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_MSG_TYPE_NEXISTENT is measured when the cause value carried in the HO FAIL message is Message Type Non-existent or Not Implemented.
    • The counter H352Dm:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_MSG_TYPE_NOT_COMPATIBLE is measured when the cause value carried in the HO FAIL message is Message Type Not Compatible with Protocol State.
    • The counter H352Dn:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_CONDITION_IE_ERR is measured when the cause value carried in the HO FAIL message is Conditional IE Error.
    • The counter H352Do:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_NO_CA is measured when the cause value carried in the HO FAIL message is No Cell Allocation Available.
    • The counter H352Dp:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_PROTOCL is measured when the cause value carried in the HO FAIL message is Protocol Error Unspecified.
    • The counter H352Dq:CELL_INTERRAN_OUTCELL_HO_FAIL_MS_RPT_OTHER is measured when the cause value carried in the HO FAIL message is Other Causes.
  • (4) In the outgoing inter-RAT inter-cell handover procedure, after sending the HO CMD message to the MS, the BSC starts the timer T8 to wait for the CLEAR CMD message from the MSC. The following counter provides the number of times that the BSC terminates the handover due to the expiration of T8.
    • H352C:CELL_INTERRAN_OUTCELL_HO_FAIL_T8_EXP
  • (5) The following counters provide the number of times that the BSC terminates the outgoing inter-RAT inter-cell handover because the BSC receives a CLEAR CMD message from the MSC and the cause value carried in the message is not Handover successful.
    • The counter H352Ha:CELL_INTERRAN_OUTCELL_HO_FAIL_CLR_CMD_RADIO_INTF_MSG_FAIL is measured when the cause value carried in the CLEAR CMD message is Radio Interface Message Failure.
    • The counter H352Hb:CELL_INTERRAN_OUTCELL_HO_FAIL_CLR_CMD_RADIO_INTF_FAIL is measured when the cause value carried in the CLEAR CMD message is Radio Interface Failure.
    • The counter H352Hc:CELL_INTERRAN_OUTCELL_HO_FAIL_CLR_CMD_OM_INTERVENTION is measured when the cause value carried in the CLEAR CMD message is OM Intervention.
    • The counter H352Hd:CELL_INTERRAN_OUTCELL_HO_FAIL_CLR_CMD_EQUIP_FAIL is measured when the cause value carried in the CLEAR CMD message is Equipment Failure.
    • The counter H352He:CELL_INTERRAN_OUTCELL_HO_FAIL_CLR_CMD_PREEMPTION is measured when the cause value carried in the CLEAR CMD message is Preemption.
    • The counter H352Hf:CELL_INTERRAN_OUTCELL_HO_FAIL_CLR_CMD_INVALID_MSG is measured when the cause value carried in the CLEAR CMD message is Invalid Message.

    • The counter H352Hg:CELL_INTERRAN_OUTCELL_HO_FAIL_CLR_CMD_PROTOCOL_ERR is measured when the cause value carried in the CLEAR CMD message is Protocol Error between BSS and MSC.

    • The counter H352Hh:CELL_INTERRAN_OUTCELL_HO_FAIL_CLR_CMD_OTHER is measured when the cause value carried in the CLEAR CMD message is Other Causes.
  • (6) The following counter provides the number of times that the outgoing inter-RAT inter-cell handover fails because the BSC detects an SS7 link failure on the A interface and releases the call.
    • H352G:CELL_INTERRAN_OUTCELL_HO_FAIL_A_INTERF_FAIL
