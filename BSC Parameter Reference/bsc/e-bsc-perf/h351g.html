Outgoing Inter-RAT Inter-Cell Handover Commands (Load Based Handover)

Measurement Counter

H351G: CELL_INTERRAN_OUTCELL_HO_CMD_LOAD_BASED_HO

Description

The load-based handover decision algorithm is used to select the candidate cells for handover based on the cell load.

This counter is measured when the BSC sends the Handover Command message to the MS with the cause value being Traffic, in the GSM-to-UMTS handover procedure.

This measurement is performed in the originating cell. Together with Outgoing Inter-RAT Inter-Cell Handover Requests and Failed Outgoing Inter-RAT Inter-Cell Handovers, this counter indicates the performance of the Outgoing Inter-RAT Inter-Cell Handover. Together with the measurement counters of Incoming Inter-RAT Inter-Cell Handover Measurement per Cell, this counter indicates the overall conditions of inter-RAT inter-cell handovers.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, this counter is incremented by one at point B when the BSC sends a Handover Command message to the MS with the cause value being Traffic during the outgoing inter-RAT inter-cell handover (service-based) procedure.

Figure 1 GSM-to-UMTS handover procedure

Formula

This is an original counter without involving any formula.
