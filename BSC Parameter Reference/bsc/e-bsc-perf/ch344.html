Incoming External Inter-Cell Handover Detection Messages Received by BSC

Measurement Counter

CH344:CELL_INTERBSC_INCELL_HO_DETECT

Description

After receiving the HO REQ ACK message from the target BSC, the MSC sends the HO CMD message to the MS through the originating BSC. The MS then sends the HO ACCESS message on the channel specified in the HO CMD message. After receiving the HO ACCESS message, the BTS sends the HO DETECT message to the BSC.

This measurement provides the number of HO DETECT messages received by the BSC.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, the counter is measured at C when the target BSC (BSC2) receives the HO DETECT message from the BTS. The measurement object is the target cell.

Figure 1 Measurement of the counters related to incoming external inter-cell handover

A

Measurement point of Incoming External Inter-Cell Handover Requests

Measurement point of Incoming External Inter-Cell Handover Responses

Measurement point of Incoming External Inter-Cell Handover Detection Messages Received by BSC

Measurement point of Successful Incoming External Inter-Cell Handovers

Formula

Incoming External Inter-Cell Handover Detection Messages Received by BSC =

[Incoming External Inter-Cell Handover Detection Messages Received by BSC (SDCCH)] +

[Incoming External Inter-Cell Handover Detection Messages Received by BSC (TCHF)] +

[Incoming External Inter-Cell Handover Detection Messages Received by BSC (TCHH)]
