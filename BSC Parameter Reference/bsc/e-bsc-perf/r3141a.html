DTM Channel Assignments Successful Times (TCHF)

Measurement Counter

R3141A: CELL_DTM_CHAN_ALLOC_SUCC_TIMES_TCHF

Description

This measurement provides number of times that the TCHF is successfully assigned upon the DTM assignment request in a cell.

Unit

Integer number or integer.

Measurement Point

On receiving the DTM RES REQ message, the BSC attempts the reassignment of CS resources based on the status of the TCH and PDCH. If the TCHF is successfully assigned, the counter is incremented.

Formula

This is an original counter without involving any formula.
