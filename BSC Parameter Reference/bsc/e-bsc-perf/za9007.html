Number of Uplink GPRS TBF Abnormal Releases due to N3103 Overflow (MS No Response) per BSC

Measurement Counter

ZA9007: BSCRPT_BSC_TBF_UP_GPRS_TBF_ABNORM_REL_N3103_OVERFLOW

Description

This measurement provides the total number of abnormal uplink GPRS TBF releases due to N3103 overflow in all the cells controlled by the BSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see Number of Uplink GPRS TBF Abnormal Releases due to N3103 Overflow (MS No Response).

Formula

Number of Uplink GPRS TBF Abnormal Releases due to N3103 Overflow (MS No Response) per BSC = Number of Uplink GPRS TBF Abnormal Releases due to N3103 Overflow (MS No Response)

(That is, the sum of Number of Uplink GPRS TBF Abnormal Releases due to N3103 Overflow in all the cells controlled by the BSC.)
