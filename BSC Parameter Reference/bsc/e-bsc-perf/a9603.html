Number of Circuit Paging Requests Sent on PACCH

Measurement Counter

A9603: PACCH_CS_PAGING_REQ

Description

If Network Operation Mode I is configured on the BSC side, and the Gs interface is configured, then the CS PAGING REQUEST message can be transmitted on the PACCH, provided that the MS to be paged is in transmission state. That is, the network side confirms that the MS is processing PS services and is being paged for CS services, and thus the BSC does not need to send CS paging requests in other cells.

If Network Operation Mode II is configured on the BSC side, the Gs interface is not configured, and A Interface Collaboration Paging Switch is set to Open, then the CS PAGING REQUEST message can be transmitted on the PACCH, provided that the MS to be paged is in transmission state.

This measurement provides the number of circuit paging requests sent by the BSC on the PACCH in a granularity period.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the CS paging request sent by the BSC on the PACCH. Each time the BSC sends a CS PAGING REQUEST message on the PACCH (see measurement point A), the counter is incremented by one.

Formula

This is an original counter without involving any formula.
