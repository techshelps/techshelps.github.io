Mean Number of Idle TC Resources

Measurement Counter

AR0754:BS_RES_IDLE_TC_AVG

Description

This measurement provides the average number of idle TC Resources.

Unit

Integer number or integer.

Measurement Point

The BSC measures the number of idle TC Resources every 10 seconds.

This counter is measured by using the following formula.

Formula

Mean Number of Idle TC Resources = [Sampling Number of Idle TC Resources]/({GP} x {6})

GP: Granularity Period. (Unit: minute)
