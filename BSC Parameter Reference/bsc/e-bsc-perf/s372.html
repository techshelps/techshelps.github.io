Measurement Reports with Signal Strength of Neighbor Cell Greater Than Absolute Level Threshold

Measurement Counter

S372:CELL_NCELL_LEV_GT_ABS_THRES_MR_NUM

Description

This measurement provides the number of MRs in which the downlink receive level of a specified neighbor cell is greater than the absolute level threshold.

Unit

Integer number or integer.

Measurement Point

After receiving the valid MR from the MS in the serving cell, the BSC compares the receive level of each neighbor cell with the receive level threshold. If the receive level of a specified neighbor cell is greater than the receive level threshold, the corresponding counter S372:CELL_NCELL_LEV_GT_ABS_THRES_MR_NUM is incremented by one.

Formula

This is an original counter without involving any formula.
