Number of MCS Degrades on Uplink EGPRS TBF

Measurement Counter

L9230: UP_EGPRS_TBF_MCS_DEGRADE_NUM

Description

This measurement provides the number of MCS degrades on the uplink EGPRS TBF in a cell in a granularity period. If the Number of MCS Upgrades on Uplink EGPRS TBF is greater than the Number of MCS Degrades on Uplink EGPRS TBF, it indicates that the network quality is good. If both the Number of MCS Upgrades on Uplink EGPRS TBF and the Number of MCS Degrades on Uplink EGPRS TBF are great, it indicates that the network quality is poor, and the coding scheme of the MS may be frequently adjusted.

Unit

Integer number or integer.

Measurement Point

Every time the BSC changes the MCS coding scheme, it determines whether to upgrade or degrade the MCS coding scheme by comparing the current MCS coding scheme with the previous MCS coding scheme. Figure 1 shows the procedure in which the MS sends a data block using less robust MCS coding scheme to the BSC. Every time the BSC receives an UPLINK RLC DATA BLOCK message for the degrade of the MCS coding scheme (see measurement point A), the counter is incremented by one.

Formula

This is an original counter without involving any formula.
