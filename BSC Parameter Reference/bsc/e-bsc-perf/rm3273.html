Call Drops Due to Loopback Start Failure (TCHH)

Measurement Counter

RM3273: CELL_LOOP_START_FAIL_DROPS_TCHH

Description

After seizing a channel, the MS starts the local switching. This measurement is triggered each time a call drops on the TCHH due to the failure in starting the local switching caused by different reasons. The cause value can be Terrestrial Resource Request Failure, Failures on the BTS Side, or Timer Expired.

Unit

Integer number or integer.

Measurement Point

At the end of the granularity period, the value of this counter is calculated according to the following formula.

Formula

Call Drops Due to Loopback Start Failure (TCHH)=

[Call Drops on TCHH Due to BSC Loopback Start Failure (Territorial Resource Request Failure)] +

[Call Drops on TCHH Due to BSC Loopback Start Failure (Others)] +

[Call Drops on TCHH Due to BTS Loopback Start Failure (Territorial Resource Request Failure)] +

[Call Drops on TCHH Due to BTS Loopback Start Failure (Failures on the BTS Side)] +

[Call Drops on TCHH Due to BTS Loopback Start Failure (Other Causes)]
