Number of Failed Uplink DTM Assignments Received on DCCH

Measurement Counter

A9804: DTM_FAIL_DCCH_UPLINK_DTM_ASS

Description

This measurement provides the number of failed uplink DTM assignments due to various causes after the BSC sends the uplink DTM assignment command on the DCCH. The possible causes are as follows:
  • The MS sends the DTM ASS FAIL message to the BSS.
  • The start of the INTRAHO CMP timer fails after the BSS sends the uplink DTM ASS CMD message to the MS.
  • The INTRAHO CMP timer starts and then expires after the BSS sends the downlink DTM ASS CMD message to the MS.

Unit

Integer number or integer.

Measurement Point

If the MS does not respond with the DTM assignment message after the BSS sends an uplink DTM ASS CMD message due to various causes, the counter is incremented by one respectively according to the specific cause.
  • After the BSC receives the DTM ASS FAIL message from the MS (see measurement point C), the counter is incremented by one.
  • If the start of the INTRAHO CMP timer fails after the BSC sends the uplink DTM ASS CMD message to the MS, the counter is incremented by one.
  • If the INTRAHO CMP timer starts and then expires after the BSC sends the uplink DTM ASS CMD message to the MS, the counter is incremented by one.
