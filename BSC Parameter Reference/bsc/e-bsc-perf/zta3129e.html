Failed Assignments per BSC (Requested Terrestrial Resource Unavailable)

Measurement Counter

ZTA3129E:BSCRPT_TOTALCELL_CHAN_ACT_NACK_TIMES_TCH

Description

In the assignment procedure, the BSC sends a CHAN ACTIV message to the BTS. If the BTS returns a CHAN ACTIV NACK message, it indicates TCH activation failure. According to the channel type, the BSC collects the counters CHAN ACTIV NACK Messages Sent by BTS in Assignment Procedure (TCHF), CHAN ACTIV NACK Messages Sent by BTS in Assignment Procedure (TCHH), CHAN ACTIV NACK Messages Sent by BTS in Mode Modify Procedure (TCHF), and CHAN ACTIV NACK Messages Sent by BTS in Mode Modify Procedure (TCHH) in the cell where the TCH resides. This measurement provides the sum of the preceding counters in all cells.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Failed Assignments per Cell" and "Failed Mode Modify Attempts per Cell".

Formula

Failed Assignments per BSC (Requested Terrestrial Resource Unavailable) = [ CHAN ACTIV NACK Messages Sent by BTS in Assignment Procedure (TCHF)] + [ CHAN ACTIV NACK Messages Sent by BTS in Assignment Procedure (TCHH)] + [ CHAN ACTIV NACK Messages Sent by BTS in Mode Modify Procedure (TCHF)] + [ CHAN ACTIV NACK Messages Sent by BTS in Mode Modify Procedure (TCHH)]
