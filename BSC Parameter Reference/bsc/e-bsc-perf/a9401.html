Number of N-Bit Packet Channel Requests Received on CCCH

Measurement Counter

A9401: CCCH_8BIT_PS_CHAN_REQ

A9404: CCCH_11BIT_PS_CHAN_REQ

Description

This measurement provides the number of N-bit packet channel request messages received on the CCCH in a granularity period. Where, N can be 8 or 11. This measurement indicates the packet services performed on the MS. Greater value of the measurement indicates that more MSs are performing the packet services.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure for the MS initiating a packet channel request on the CCCH. Each time the BSC receives an N-bit CHANNEL REQUEST message (N can be 8 or 11) from the MS (see measurement point A), the counter is incremented by one.

Figure 1 Channel request received on the CCCH

Formula

This is an original counter without involving any formula.
