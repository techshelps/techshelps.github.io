Number of Downlink Packet Assignments Sent on DCCH

Measurement Counter

A9811: DTM_DCCH_DOWNLINK_PACKET_ASS

Description

This measurement provides the number of downlink packet assignment messages sent on the DCCH.

Unit

Integer number or integer.

Measurement Point

After the CS domain receives the PS ASSIGNMENT message from the PS domain, the BSC sends the downlink PACKET ASSIGNMENT message from to the MS if the current call is in stable state (stable signaling or stable service), or the MS is waiting for the PS ASS message from the PS domain and the CS channels are not adjusted. In this case, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
