Number of Expiries of the Paging for PS Services on the PCH

Measurement Counter

A340: CELL_PACUIT_PCH_EXP

Description

This measurement provides the number of times when the paging for PS services on the PCH expires during PCH scheduling.

Each paging has a lifetime, and the paging message is deleted after the lifetime expires. The lifetime is five seconds by default and can be changed through the software parameter of the BTS.

The BTS measures the value of CELL_PACUIT_PCH_EXP and reports it in the BTS_STATIS_REPORT message to the BSC.

Unit

Integer number or integer

Measurement Point

This counter is incremented each time the information element CELL_PACUIT_PCH_EXP in the BTS_STATIS_REPORT message received from the BTS is successfully decoded by the BSC, at C1 as shown in Figure 1.

Figure 1 Procedure for the BTS sending the BTS_STATIS_REPORT message

Formula

This is an original counter without involving any formula.
