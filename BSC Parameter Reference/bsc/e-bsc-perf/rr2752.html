Congestion Ratio of Dynamic Assign Abis Resource (16K)

Measurement Counter

RR2752: SITE_16K_ASS_FAIL_RATE

Description

This counter is used to measure the congestion ratio of the dynamic assignment of 16 kbit/s Abis resources in the site.

Unit

%.

Measurement Point

None.

Formula

[Congestion Ratio of Dynamic Assign Abis Resource (16K)] =[Failed Dynamic Assign Abis Resource (16K_CS)] + [Failed Dynamic Assign Abis Resource (PS_MainLink)] + [Failed Dynamic Assign Abis Resource (PS_SubLink)] x {100}/[Dynamic Assign Abis Resource (16K_CS)] + [Dynamic Assign Abis Resource (PS_MainLink)] + [Dynamic Assign Abis Resource (PS_SubLink)]
