Average Number of Occupied Dynamic EGPRS PDCHs

Measurement Counter

AR9323: AVG_OCCU_DYN_EPDCH_NUM

Description

This measurement provides the average number of occupied dynamic EGPRS PDCHs in a cell in a granularity period. The occupied EGPRS PDCH is the PDCH that carries the TBFs. If the measurement value is high, you can infer that the PS services are busy.

Unit

Integer number or integer.

Measurement Point

None.

Formula

Average Number of Occupied Dynamic EGPRS PDCHs = Total Number of Sampled Dynamic EGPRS PDCHs Occupied/Sampling Times of PDCH Measurement
