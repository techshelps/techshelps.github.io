Channel Assignment Success Measurement per TRX

Description

The Channel Assignment Success Measurement per TRX measurement unit is used to measure the successful channel assignments in the following procedures based on different channel types:

  • Immediate assignment
  • Assignment
  • Internal intra-cell handover
  • Incoming internal inter-cell handover
  • Incoming external inter-cell handover

Measurement Object

TRX.
