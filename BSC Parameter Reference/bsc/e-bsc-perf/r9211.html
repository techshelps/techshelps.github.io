Total Sampled PPCH Queue Lengths

Measurement Counter

R9211: CELL_RADIO_PPCH_TOTAL_QUEUE_LEN

Description

This measurement provides the total length of the paging message queues on the PPCH in the BSC in a granularity period.

Unit

byte.

Measurement Point

In the granularity period, the PPCH queue length is sampled every five seconds to obtain the total sampled PPCH queue lengths.

Formula

This is an original counter without involving any formula.
