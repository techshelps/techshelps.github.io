Processing Times of PS Services by MSs Supporting Multislot Class X in DTM GPRS

Measurement Counter

A9709: CELL_DTM GPRS MULTISLOT CLASS5

A9710: CELL_DTM GPRS MULTISLOT CLASS6

A9711: CELL_DTM GPRS MULTISLOT CLASS9

A9712: CELL_DTM GPRS MULTISLOT CLASS10

A9713: CELL_DTM GPRS MULTISLOT CLASS11

A9721: CELL_DTM_GPRS_HIGH_MULTISLOT_CLASS31_36

A9722: CELL_DTM_GPRS_HIGH_MULTISLOT_CLASS32_37

A9723: CELL_DTM_GPRS_HIGH_MULTISLOT_CLASS33_38

A9724: CELL_DTM_GPRS_HIGH_MULTISLOT_CLASS41

A9725: CELL_DTM_GPRS_HIGH_MULTISLOT_CLASS42

A9726: CELL_DTM_GPRS_HIGH_MULTISLOT_CLASS43

A9727: CELL_DTM_GPRS_HIGH_MULTISLOT_CLASS44

Description

This measurement provides the number of times MSs supporting multislot class x in DTM GPRS process PS services in a cell.

Unit

Integer number or integer.

Measurement Point

After the MS processes PS services and the MS context is released, this measurement is triggered according to the MS RA capability information element saved in the MS context. Note: In "Multislot class x", "x" can be 5, 6, 9, 10, 11, 31 and 36, 32 and 37, 33 and 38, 41, 42, or 43 and 44.

Formula

This is an original counter without involving any formula.
