Number of Downlink GMSK_MEAN_BEP Different Value Measurement

Description

The PSCALL.GmskMBepDifNum.Cell measurement unit indicates the quality of the downlink air interface, which serves as a reference for establishing the service model. The following table lists the measurement counters related to the PSCALL.GmskMBepDifNum.Cell measurement unit. Except the GMSK_MEAN_BEP with different values, the introduction, measurement points, and formulas of these measurement counters are the same. Therefore, only GMSK_MEAN_BEP = 0 is taken as an example and described in details.

Cell.
