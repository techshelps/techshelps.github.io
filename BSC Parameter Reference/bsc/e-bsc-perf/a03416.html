VGCS Assignment Queue Request

Measurement Counter

A03416: CELL_VGCS_ASS_QUEUE_REQ

Description

On receiving the VGCS ASSIGNMENT REQUEST message from the MSC, the BSC puts the request onto a queue if no radio resource is available for allocation and triggers the measurement.

Unit

Integer number or integer.

Measurement Point

If no radio resource is available and VGCS assignment procedure queues for idle resources, the BSC sends a VGCS QUEUING INDICATION message to the MSC. This counter is incremented by one when a VGCS QUEUING INDICATION message is sent. In the signaling procedure, this counter is measured at B in Figure 1.

Formula

This is an original counter without involving any formula.
