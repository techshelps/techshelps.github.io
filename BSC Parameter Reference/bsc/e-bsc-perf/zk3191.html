External Incoming Cell Handover Commands per BSC

Measurement Counter

ZK3191:BSCRPT_INTERBSC_INCELL_HO_CMD

Description

This measurement provides the number of external incoming cell handover commands of all the cells controlled by a BSC. The channel involved in the measurement can be the SDCCH, TCHF, or TCHH.

After receiving the handover request, the BSC requests and activates the channel from the target cell and requests voice service resources. After the request is successful, the BSC returns a handover request ack message. This counter is measured based on the channel type and frequency band type.

The handover can be performed between GSM900/850 and GSM900/850, between GSM1800/1900 and GSM1800/1900, between GSM900/850 and GSM1800/1900, or between GSM1800/1900 and GSM900/850.

According to the counters External Incoming Cell Handover Commands per BSC and Successful External Incoming Cell Handovers per BSC, the external incoming cell handover per BSC can be well understood. According to the counters related to external outgoing cell handovers, the external inter-cell handover per BSC can be well understood.

Unit

Integer number or integer.

Measurement Point

BSC counters are calculation counters. This counter is measured according to the formula below when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.

Formula

External Incoming Cell Handover Commands per BSC = Incoming External Inter-Cell Handover Responses

(That is, the total of the original counter Incoming External Inter-Cell Handover Responses in all the cells of the BSC.)
