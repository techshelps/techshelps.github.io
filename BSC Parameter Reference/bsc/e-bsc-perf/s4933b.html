Average Number of Decoded Blocks in the Downlink MR

Measurement Counter

S4933B:CELL_DL_SIG_NBRRCVDBLOCKS_AVR

Description

During a granularity period, every time the BSC receives an EMR or pre-processed EMR from the BTS, the BSC accumulates the NbrRcvBlocks in the EMR. At the end of the granularity period, the value of this counter is the result of dividing the accumulative value of NbrRcvBlocks by the number of EMRs.

Unit

Integer number or integer.

Measurement Point

In the granularity period, the counter is calculated as follows.

Formula

Average Number of Decoded Blocks in the Downlink MR = [Average Number of Decoded Blocks in the Downlink MR]/([Number of EMRs on the TCHF] + [Number of EMRs on the TCHH])
