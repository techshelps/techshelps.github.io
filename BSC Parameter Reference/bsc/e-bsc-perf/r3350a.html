Channel Activation Attempts in Incoming External Inter-Cell Handover Procedure

Measurement Counter

R3350A: CELL_INTERBSC_INCELL_HO_CH_ACT_SD

R3350B: CELL_INTERBSC_INCELL_HO_CH_ACT_NACK_SD

R3350C: CELL_INTERBSC_INCELL_HO_CH_ACT_EXP_SD

R3357A: CELL_INTERBSC_INCELL_HO_CH_ACT_TCHF

R3358A: CELL_INTERBSC_INCELL_HO_CH_ACT_TCHH

R3357B: CELL_INTERBSC_INCELL_HO_CH_ACT_NACK_TCHF

R3358B: CELL_INTERBSC_INCELL_HO_CH_ACT_NACK_TCHH

R3357C: CELL_INTERBSC_INCELL_HO_CH_ACT_EXP_TCHF

R3358C: CELL_INTERBSC_INCELL_HO_CH_ACT_EXP_TCHH

Description

Upon receipt of an HO REQ message from the MSC, the BSC assigns a channel and sends a CHANNEL ACTIVATION message to the BTS. At this time, one of the following three counters is incremented by one for the cell where the target TCH resides based on the type of the channel assigned: R3350A "Channel Activation Attempts in Incoming External Inter-Cell Handover Procedure (SDCCH)", R3357A "Channel Activation Attempts in Incoming External Inter-Cell Handover Procedure (TCHF)", or R3358A "Channel Activation Attempts in Incoming External Inter-Cell Handover Procedure (TCHH)".

In the incoming external inter-cell TCH handover procedure, the BSC sends a CHANNEL ACTIVATION message to the BTS. If the BTS returns a CHANNEL ACTIVATION NEGATIVE ACKNOWLEDGE message indicating the channel activation failure, one of the following three counters is incremented by one for the cell where the target TCH resides based on the channel type: R3350B "CHAN ACTIV NACK Messages Sent by BTS in Incoming External Inter-Cell Handover Procedure (SDCCH)", R3357B "CHAN ACTIV NACK Messages Sent by BTS in Incoming External Inter-Cell Handover Procedure (TCHF)", or R3358B "CHAN ACTIV NACK Messages Sent by BTS in Incoming External Inter-Cell Handover Procedure (TCHH)".

In the incoming external inter-cell TCH handover procedure, the BSC sends a CHANNEL ACTIVATION message to the BTS and starts an internal timer (5 s by default) to wait for a CHANNEL ACTIVATION ACKNOWLEDGE message. If the BSC does not receive the CHANNEL ACTIVATION ACKNOWLEDGE message after the timer expires, one of the following three counters is incremented by one for the cell where the target TCH resides based on the channel type: R3350C "Channel Activation Timeouts in Incoming External Inter-Cell Handover Procedure (SDCCH)", R3357C "Channel Activation Timeouts in Incoming External Inter-Cell Handover Procedure (TCHF)", or R3358C "Channel Activation Timeouts in Incoming External Inter-Cell Handover Procedure (TCHH)".

Unit

Integer number or integer.

Measurement Point

Upon receipt of an HO REQ message from the MSC, the BSC assigns a channel and sends a CHANNEL ACTIVATION message to the BTS. At this time, the counter R3350A, R3357A, or R3358A is incremented by one. If the BTS returns a CHANNEL ACTIVATION NEGATIVE ACKNOWLEDGE message, the counter R3350B, R3357B, or R3358B is incremented by one. After sending a CHANNEL ACTIVATION message, the BSC starts the timer and waits for a response from the BTS. If the BSC does not receive any response after the timer expires, the counter R3350C, R3357C, or R3358C is incremented by one.

Formula

This is an original counter without involving any formula.
