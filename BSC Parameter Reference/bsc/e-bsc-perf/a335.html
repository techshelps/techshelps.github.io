Minimum decodable Level in the Code Streams on the RACH

Measurement Counter

A335: CELL_MIN_SOLVE_LEV

Description

This measurement provides the minimum decodable level in the code streams on the RACH.

The BTS measures the value of the CELL_MIN_SOLVE_LEV and reports it in the BTS_STATIS_REPORT message to the BSC every minute.

Unit

dB

Measurement Point

If the information element CELL_MIN_SOLVE_LEV in the BTS_STATIS_REPORT message is successfully decoded in the granularity period and the value is not 0xFF, then the minimum value is measured, at C1 as shown in Figure 1. That is, this counter is measured only when the current value is smaller than the minimum history value.

Figure 1 Procedure for the BTS sending the BTS_STATIS_REPORT message

Formula

This is an original counter without involving any formula.
