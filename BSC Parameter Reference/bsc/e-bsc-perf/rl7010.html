Ratio of CREF by Local SCCP

Measurement Counter

RL7010: SCCP_REJ_LOCAL

Description

This counter is predefined. It provides the ratio of the Number of Sent CREF Messages to Number of Received CR Messages.

Unit

%.

Measurement Point

The measurement point is on the BSC side, as shown in the figure.

This counter is measured when the measurement results are reported in a GP.



Formula

Ratio of CREF by Local SCCP (%) =

Number of Sent CREF Messages x 100/Number of Received CR Messages
