Number of Sent Valid TRAU Frames

Measurement Counter

L9A04: CELL_GABIS_TRAU_TX_VALID_FR

Description

This measurement provides the number of valid TRAU frames sent by a cell in a granularity period. The proportion of sent valid TRAU frames to all the sent frames indicates the traffic on the downlink.

Unit

Integer number or integer.

Measurement Point

The following figure shows the process where the BSC sends a downlink TRAU frame. In a granularity period, the counter is incremented by one every time the BSC sends a TRAU frame with valid RLC/MAC block on the G-Abis interface (see measurement point A in Figure 1).

Figure 1 Downlink frame with RLC/MAC block received on the G-Abis interface

Formula

This is an original counter without involving any formula.
