Uplink Level during Radio Link Failure (SDCCH)

Measurement Counter

S4300D:TRX_SD_DROP_UL_LEV

Description

After receiving a valid MR or pre-processed MR, the BSC takes statistics of some parameter values based on the requirements. The statistics can help users better understand the network performance. This measurement provides the accumulated uplink level reported by the MRs during the radio link failure on the SDCCH. This counter is used to calculate the counter AS4300D: TRX_SD_DROP_UL_LEV_AVR.

Unit

dBm.

Measurement Point

In the granularity period, this counter is incremented by the uplink level in the last valid MR or pre-processed MR before the radio link failure occurs on the SDCCH.

Formula

This is an original counter without involving any formula.
