BSC Loopback to Normal Call Success Rate

Measurement Counter

TR3860:CELL_BSC_LOOP_RESTORE_RATE

Description

This measurement provides the ratio of the number of successful attempts to return to normal call from BSC loopback to the number of attempts to return to normal call from BSC loopback. This measurement is used to calculate the BSC loopback to normal call success rate.

Unit

%.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

BSC Loopback to Normal Call Success Rate =

[Number of Successful Attempts to Return to Normal Call from BSC Loopback] x {100}/

[Number of Attempts to Return to Normal Call from BSC Loopback]
