Analyzed Measurement of Outgoing Cell Handovers

Measurement Counter

RH373:CELLCELL_OUTCELL_HO_SUCC_RATE

H371:CELLCELL_OUTCELL_HO_CMD

H372:CELLCELL_OUTCELL_HO_FAIL

H373:CELLCELL_OUTCELL_HO_SUCC

Description

The counters indicate the performance of the outgoing inter-cell handover in a specified cell, for example, the number of requests for the handover from the specified cell to another cell, the number of successful handovers, and the success rate of handovers.

It is recommended that the counters related to Analyzed Measurement of Outgoing Cell Handovers are registered at the initial stage of BSC running or during the analysis of handover performance in a specified cell for network optimization. When the BSC system is running stably, these counters are deleted. Along with the counters related to Analyzed Measurement of Incoming Cell Handovers, these counters indicate the performance of the handover between two specified cells. The recommended granularity period is 60 minutes.

Unit

RH373: %.

H371/H372/H373: Integer number or integer.

Measurement Point

The counters are calculated by using the following formula.

Formula

Success Rate of Cell Outgoing Handovers = [Successful Cell Outgoing Handovers] x {100}/[Number of Outgoing Cell Handover Commands]

Number of Outgoing Cell Handover Commands =

[Cell Outgoing Handover Commands (Uplink Quality)] +

[Cell Outgoing Handover Commands (Downlink Quality)] +

[Cell Outgoing Handover Commands (Uplink Strength)] +

[Cell Outgoing Handover Commands (Downlink Strength)] +

[Cell Outgoing Handover Commands (Timing Advance)] +

[Cell Outgoing Handover Commands (Better Cell)] +

[Cell Outgoing Handover Commands (Load)] +

[Cell Outgoing Handover Commands (Rapid Level Drop)] +

[Cell Outgoing Handover Commands (MSC Intervention)] +

[Cell Outgoing Handover Commands (OM Intervention)] +

[Cell Outgoing Handover Commands (Other Causes)]+

[Number of Outgoing Cell Handover Commands (Underlaid to Overlaid Subcell Handover in Enhanced Dualband Network)]+

[Number of Outgoing Cell Handover Commands (Overlaid to Underlaid Subcell Handover in Enhanced Dualband Network)]+

[Number of Outgoing Cell Handover Commands (MS Handover in the Overlaid Subcell Covered by the Enhanced Dualband Network)]

Failed Cell Outgoing Handovers =

[Failed Cell Outgoing Handovers (Uplink Quality)] +

[Failed Cell Outgoing Handovers (Downlink Quality)] +

[Failed Cell Outgoing Handovers (Uplink Strength)] +

[Failed Cell Outgoing Handovers (Downlink Strength)] +

[Cell Outgoing Handover Commands (Timing Advance)] +

[Failed Cell Outgoing Handovers (Better Cell)] +

[Failed Cell Outgoing Handovers (Load)] +

[Failed Cell Outgoing Handovers (Rapid Level Drop)] +

[Failed Cell Outgoing Handovers (MSC Intervention)] +

[Failed Cell Outgoing Handovers (OM Intervention)] +

[Failed Cell Outgoing Handovers (Other Causes)]+

[Number of Outgoing Cell Handover Failures (Underlaid to Overlaid Subcell Handover in Enhanced Dualband Network)]+

[Number of Outgoing Cell Handover Failures (Overlaid to Underlaid Subcell Handover in Enhanced Dualband Network)]+

[Number of Outgoing Cell Handover Failures (MS Handover in the Overlaid Subcell Covered by the Enhanced Dualband Network)]

Successful Cell Outgoing Handovers = [Outgoing Inter-Cell Handover Requests] - [Failed Cell Outgoing Handovers]
