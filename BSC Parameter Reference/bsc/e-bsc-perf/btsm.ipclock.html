IPClock Measurement of PTU

Description

The BTS using IP transmission obtains the IP clock reference from the IP Clock Server. To get knowledge of the transmission performance of the IP clock reference, the communication-related counters should be measured.

The BTS reports the measurement information to the BSC for processing only when IP transmission is implemented in the BTS.

Measurement Object

BTS
