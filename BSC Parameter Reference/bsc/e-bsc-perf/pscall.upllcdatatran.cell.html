Uplink LLC Data Transfer Capability Measurement

Description

The PSCALL.UpLlcDataTran.Cell measurement unit indicates the transmission performance of the uplink LLC data in a cell and is used as a reference for establishing the service model.

The method of measuring this counter is as follows: In a certain granularity period, measure the uplink counters related to the transmission performance, such as, number of TBFs, number of PDUs, and PDU bytes.

Measurement Object

Cell.
