Average Number of PDCHs Occupied by Uplink TBF

Measurement Counter

AR9313: AVG_UPLINK_OCCU_PDCH

Description

This measurement provides the average number of PDCHs occupied by the uplink TBFs in a cell in a granularity period. If the measurement value is high, you can infer that the PS services are busy.

Unit

Integer number or integer.

Measurement Point

None.

Formula

Average Number of PDCHs Occupied by Uplink TBF = Total Number of Sampled PDCHs Occupied by Uplink TBF/Sampling Times of PDCH Measurement
