Number of Uplink EGPRS TBF Abnormal Releases due to SUSPEND

Measurement Counter

A9208: UP_EGPRS_TBF_ABNORM_REL_SUSPEND

Description

This measurement provides the number of uplink EGPRS TBF abnormal releases due to cell suspension in a granularity period. A high measurement value is yielded because the MS often initiates CS services during the process of PS services. Another possible cause is that the cell is on the edge of the network location area and thus the MS frequently performs the location update.

Unit

Integer number or integer.

Measurement Point

When a class B MS initiates a CS service request, the BSC terminates all the PS services of the MS irrespective of the MS status.

When the MS is in transmission state and the uplink EGPRS TBF exists, the MS initiates the suspending procedure by sending a SUSPEND message. Accordingly, the BSC releases the uplink EGPRS TBF.
