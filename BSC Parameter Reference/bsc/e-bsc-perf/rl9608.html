Uplink Utilization Rate on BC

Measurement Counter

RL9608: BC_TRAN_UP_UTILIZATION_RATE

Description

This measurement provides the ratio of the Uplink Bandwidth Actually Used on BC to the Configured Link Bandwidth on BC in a granularity period. The counter indicates the uplink usage of the BC. If the usage ratio exceeds 80%, it indicates the uplink traffic is heavy and the BC link needs to be expanded; otherwise, it indicates the uplink traffic is light and the link bandwidth is wasted. After the BC is configured, this counter can be measured.

Unit

%.

Measurement Point

The ratio of the Uplink Bandwidth Actually Used on BC to the Configured Link Bandwidth on BC is measured in a granularity period.

Formula

Uplink Utilization Rate on BC = [Uplink Bandwidth Actually Used on BC] x 100/[Configured Link Bandwidth on BC]
