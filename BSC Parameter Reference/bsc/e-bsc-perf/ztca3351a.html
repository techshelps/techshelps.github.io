Number of Calls Originated or Terminated by MSs Supporting A5/7 per BSC

Measurement Counter

ZTCA3351A: BSCRPT_SPT_A5_7_MS_NUM

Description

When the "A5/7" field in classmark 3 is set to "1", it is considered that the MS supports A5/7 if any of the following situations occurs: the MS sends the classmark change message; the MSC sends the classmark update message; the incoming BSC handover is complete.

This measurement provides the number of calls originated or terminated by MSs that support A5/7 in all cells under the BSC.

Unit

Integer number or integer.

Measurement Point

None.

Formula

Number of Calls Originated or Terminated by MSs Supporting A5/7 per BSC = Number of Calls Originated or Terminated by MSs Supporting A5/7

(the sum of the number of calls originated or terminated by MSs supporting A5/7 in all cells under the BSC)
