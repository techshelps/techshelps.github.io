64K Abis Timeslots Occupied to EGPRS PDCH

Measurement Counter

S9209: PSALL_PCUCAP_EQUAL_EGPRS_PDCH_ALLOC_PEAK

Description

This measurement provides the maximum number of equivalent EGPRS PDCHs of 64 kbit/s in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the number of equivalent EGPRS PDCHs of 64 kbit/s is sampled periodically and the maximum sampled value is used as the value of the counter.

According to the actual occupied bandwidth, the number of seized PDCHs can be converted to the number of equivalent EGPRS PDCHs of 64 kbit/s.

For example, the three PDCHs can be seized by the current MS. The bandwidth of two PDCHs is 16 kbit/s, and the bandwidth of another MS is 32 kbit/s. The total bandwidth of three PDCHs is 64 kbit/s. Thus, the number of equivalent EGPRS PDCHs of 64 kbit/s is one.

Formula

This is an original counter without involving any formula.
