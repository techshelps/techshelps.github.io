Normal Speech Disconnections

Measurement Counter

A03701: CELL_SPEECH_NORMAL_DISC_TIMES

Description

The measurement provides the number of normal speech disconnections, including Bad Speech Quality Disconnections.

Unit

Integer number or integer.

Measurement Point

After the BSC receives the CLEAR CMD message (excluding the situation that the BSC initiates the call release by self) from the MSC, if the cause value is Call Control (the speech service is released), the counter is incremented by one.

Formula

This is an original counter without involving any formula.
