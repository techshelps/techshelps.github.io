Number of Uplink Frames with FR AMR of N kbit/s in the Customized MR of Cell

Measurement Counter

S3101: CELL_EXT_AMR_FR_475_UP_NUM

S3102: CELL_EXT_AMR_FR_515_UP_NUM

S3103: CELL_EXT_AMR_FR_590_UP_NUM

S3104: CELL_EXT_AMR_FR_670_UP_NUM

S3105: CELL_EXT_AMR_FR_740_UP_NUM

S3106: CELL_EXT_AMR_FR_795_UP_NUM

S3107: CELL_EXT_AMR_FR_1020_UP_NUM

S3108: CELL_EXT_AMR_FR_1220_UP_NUM

Description

When decoding the uplink MR, the BSC checks whether the EI_FRAME is contained in the TYPE field of the MR. If the EI_FRAME is found and it contains the AMR Nkbit/s frames, this measurement provides the number of uplink frames with the FR AMR of N kbit/s in the customized MR.

N represents the coding rate. The coding rate of the full rate speech ranges from 4.75 kbit/s to 12.2 kbit/s.

Unit

Integer number or integer.

Measurement Point

After receiving the MR or pre-processed MR sent over the AMR channel from the BTS in the serving cell, the BSC decodes the uplink MR. If the TYPE field in the MR contains the EI_FRAME, and the EI_FRAME contains the AMR N kbit/s frames, this counter is accumulated.

Formula

This is an original counter without involving any formula.
