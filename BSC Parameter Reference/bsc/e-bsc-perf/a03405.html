VGCS Assignment Request

Measurement Counter

A03405: CELL_VGCS_ASS_REQ

Description

During the establishment of the VGCS, the MSC sends a VGCS ASSIGNMENT REQUEST message to the BSC for setting up a VGCS TCH. If the assignment is successful, the BSC sends a VGCS ASSIGNMENT RESULT message to the MSC. Otherwise, the BSC sends a VGCS ASSIGNMENT FAILURE message to the MSC. This measurement provides the number of VGCS ASSIGNMENT REQUEST messsages received by a cell. Along with the Successful VGCS Assignment and Unsuccessful VGCS Assignment, this counter indicates the success ratio of VGCS assignments in the cell.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC receives a VGCS ASSIGNMENT REQUEST message from the MSC. In the signaling procedure, this counter is measured at A in Figure 1, Figure 2, and Figure 3.
