Uplink Bandwidth Actually Used on BC

Measurement Counter

L9607: BC_TRAN_UP_USED_BC_BANDWIDTH

Description

This measurement provides the number of kilobits per second sent on the BC in a granularity period. After the BC is configured, this counter can be measured.

Unit

Kbit/s.

Measurement Point

In a granularity period, the number of kilobits of NS PDUs sent on the BC is sampled on the Gb interface board every ten seconds. The counter is incremented by the number of kilobits divided by ten seconds.

Formula

This is an original counter without involving any formula.
