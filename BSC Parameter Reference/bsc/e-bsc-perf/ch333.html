Successful Outgoing External Inter-Cell Handovers

Measurement Counter

CH333:CELL_INTERBSC_OUTCELL_HO_SUCC

Description

This measurement provides the total number of successful outgoing external inter-cell handovers. The counter is calculated by subtracting the Failed Outgoing External Inter-Cell Handovers from the Outgoing External Inter-Cell Handover Requests.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1 and Figure 2, the counter is measured at C when the BSC receives the CLEAR CMD message from the MSC during the outgoing external inter-cell handover and the cause carried in the CLEAR CMD message is Handover successful (0x0B).

Figure 2 Measurement of the counters related to outgoing external inter-cell handover (directed retry)

A

Measurement point of Outgoing External Inter-Cell Handover Requests (Directed Retry)

Measurement point of Outgoing External Inter-Cell Handover Responses (Directed Retry)

C

Measurement point of Successful Outgoing External Inter-Cell Handovers (Directed Retry)

Formula

Successful Outgoing External Inter-Cell Handover =

[Outgoing External Inter-Cell Handover Requests] -

[Failed Outgoing External Inter-Cell Handovers]
