Failed Outgoing Inter-RAT Inter-Cell Handovers (Service Based Handover)

Measurement Counter

H352M:CELL_INTERRAN_OUTCELL_HO_FAIL_SERVICE_BASED_HO

Description

This measurement provides the number of failed handovers during the outgoing inter-RAT inter-cell handover (service-based) procedure. Together with Outgoing Inter-RAT Inter-Cell Handover Requests(service based handover) and Outgoing Inter-RAT Inter-Cell Handover Commands(service based handover), this counter indicates the outgoing inter-RAT inter-cell handover (service-based) performance. This measurement is performed in the originating cell.

Unit

Integer number or integer.

Measurement Point

During the outgoing inter-RAT inter-cell handover (service-based) procedure, this counter is incremented by one when any of the following events occurs: the BSC receives a Handover Required Reject message from the MSC, the timer expires before the BSC receives the HO CDM message from the MSCâ€¨, or a call drop occurs.

Formula

This is an original counter without involving any formula.
