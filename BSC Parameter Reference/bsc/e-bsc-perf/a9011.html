Total Number of Sampled Concurrent Uplink GPRS TBF

Measurement Counter

A9011:UP_GPRS_TBF_TOTAL_SAMPLE_NUM

Description

This measurement provides the total number of sampled concurrent uplink GPRS TBFs in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the BSC samples the number of concurrent uplink GPRS TBFs for several times. Each time the sampling is performed, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
