Intra-BSC Radio Handover Success Rate

Measurement Counter

RH303C: CELL_INTRABSC_RD_HO_SUCC_RATE

Description

The measurement provides the success rate of intra-cell and inter-cell handovers on the Um interface.

Unit

%.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

Intra-BSC Radio Handover Success Rate = ([Successful Internal Intra-Cell Handovers] + [Successful Incoming Internal Inter-Cell Handovers]) x {100}/([Internal Intra-Cell Handover Commands] + [Incoming Internal Inter-Cell Handover Responses])
