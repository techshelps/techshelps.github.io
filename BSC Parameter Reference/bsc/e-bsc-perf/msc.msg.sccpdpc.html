MSC Measurement per DPC

Description

MSC Measurement per DPC is used to measure the number of Number of MSC Resets and the number of messages that the BSC receives from the MSC.

Measurement Object

SCCPDPC

Measurement Counter

The following table lists the measurement counters related to MSC Measurement per DPC.

Counter

Refer to Section...

L0056A:BS_A_MSC_RST

Number of MSC Resets

L0056B:BS_MSG_RX_FROM_A_INTF

Number of Messages Received from MSC

L0057A: BS_USR_NUM_BY_IMSI_NOT_PAGRSP

Number of Users Assigned to the MSC According to IMSI Load Sharing (Non-Paging Response Access)

L0057B: BS_USR_NUM_BY_IMEI

Number of Users Assigned to the MSC According to IMEI Load Sharing

L0057C: BS_USR_NUM_BY_NULLNRI

Number of Users Assigned to the MSC According to NULL NRI Load Sharing in TMSI

L0057D: BS_USR_NUM_BY_IMSI_PAGRSP

Number of Users Assigned to the MSC According to IMSI Load Sharing (Paging Response Access)

L0057E: BS_USR_NUM_BY_IMSI_NOT_EMRG

Number of Users Assigned to the MSC According to IMSI Load Sharing (Non-Emergency Call Access)
