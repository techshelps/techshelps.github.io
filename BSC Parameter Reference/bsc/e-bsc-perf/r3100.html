DTM Channel Assignments Times

Measurement Counter

R3100: CELL_DTM_CHAN_ALLOC_REQ_TIMES

Description

This measurement provides the number of DTM assignment requests received in a cell.

Unit

Integer number or integer.

Measurement Point

On receiving the DTM RES REQ message, the BSC triggers the measurement if it decides to reassign CS resources.

Formula

This is an original counter without involving any formula.
