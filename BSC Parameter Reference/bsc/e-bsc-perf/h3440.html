Measurement of Incoming External Inter-Cell Handover Detection Messages Received by BSC

Measurement Counter

H3440:CELL_INTERBSC_INCELL_HO_DETECT_SD

H3447:CELL_INTERBSC_INCELL_HO_DETECT_TCHF

H3448:CELL_INTERBSC_INCELL_HO_DETECT_TCHH

Description

After receiving the HO REQ ACK message from the target BSC, the MSC sends the HO CMD message to the MS through the originating BSC. The MS then sends the HO ACCESS message on the channel specified in the HO CMD message. After receiving the HO ACCESS message, the BTS sends the HO DETECT message to the BSC.

The specific counter provides the number of HO DETECT messages received by BSC from the BTS based on the type of the target channel (SDCCH/TCHF/TCHH), and is measured when the BSC receives the HO DETECT message from the BSC.
  • H3440:CELL_INTERBSC_INCELL_HO_DETECT_SD
  • H3447:CELL_INTERBSC_INCELL_HO_DETECT_TCHF
  • H3448:CELL_INTERBSC_INCELL_HO_DETECT_TCHH

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, the counter is measured at C when the target BSC (BSC2) receives the HO DETECT message from the BTS. The measurement object is the target cell.

This is an original counter without involving any formula.
