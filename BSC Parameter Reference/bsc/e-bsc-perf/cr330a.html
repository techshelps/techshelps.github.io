Channel Activation Attempts in Immediate Assignment Procedure

Measurement Counter

CR330A: CELL_IMM_ASS_CH_ACT

Description

This measurement counter provides the total number of CHANNEL ACTIVATION messages sent by the BSC to the BTS in the immediate assignment procedure. The channel type carried in the CHANNEL ACTIVATION message can be SDCCH, TCHF, or TCHH.

Unit

Integer number or integer.

Measurement Point

Each time the BSC sends a CHANNEL ACTIVATION message to the BTS in the immediate assignment procedure, the counter is incremented by one.

Formula

Channel Activation Attempts in Immediate Assignment Procedure = [Channel Activation Attempts in Immediate Assignment Procedure (SDCCH)] + [Channel Activation Attempts in Immediate Assignment Procedure (TCHF)] + [Channel Activation Attempts in Immediate Assignment Procedure (TCHH)]
