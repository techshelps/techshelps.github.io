Description

After receiving an HO REQ message from the MSC, BSC requests a channel from the target cell and activates the channel. BSC2 requests for speech service resources. If the request is successful, BSC2 returns an HO REQ ACK message to the MSC.

This measurement provides the number of HO REQ ACK messages that the BSC responses to the MSC after receiving the handover request from the MSC, based on the type of the target channel (SDCCH/TCHF/TCHH) when the handover occurs between the frequency bands as listed in Table 1. This measurement is performed when the BSC sends an HO REQ ACK message to the MSC.
