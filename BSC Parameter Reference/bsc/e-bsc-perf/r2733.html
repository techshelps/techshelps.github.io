Failed Dynamic Assign Abis Resource (PS_SubLink)

Measurement Counter

R2733: SITE_ASS_FAIL_PS_SUB_LNK

Description

This counter is used to measure the number of failed dynamic assignments of Abis resources for the packet service sublink (idle timeslots) when the BSC receives the assignment request from the MSC.

Note:

The Abis resources of the packet service sublink are of 16 kbit/s.

Unit

Integer number or integer.

Measurement Point

This counter is measured when allocating Abis resources for the packet service sublink (time slots) fails.

Formula

This is an original counter without involving any formula.
