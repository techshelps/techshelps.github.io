Number of Calls Originated or Terminated by MSs Supporting TDD per BSC

Measurement Counter

ZTCA3350A: BSCRPT_SPT_TDD_CALL_NUM

Description

After the MS sends the classmark change message or the MSC sends the classmark update message, the MS is regarded to support TDD if the "Support TDD" bit in classmark 3 is set to "Yes".

This counter provides the number of MSs that support TDD within the cells controlled by the BSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details about the measurement point, see Number of Calls Originated or Terminated by MSs Supporting TDD (A03623).

Formula

Number of Calls Originated or Terminated by MSs Supporting TDD per BSC = Number of Calls Originated or Terminated by MSs Supporting TDD

(That is, the total of the Number of Calls Originated or Terminated by MSs Supporting TDD in each cell of the BSC.)
