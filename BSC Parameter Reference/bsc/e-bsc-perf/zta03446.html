Attempted Incoming Inter-BSC Inter-Cell VGCS TCH Handovers per BSC

Measurement Counter

ZTA03446:BSCRPT_GCC_INTERBSC_INTOCELL_HO_REQ

Description

In the inter-BSC VGCS TCH handover procedure, upon receiving a HANDOVER REQUEST message from the MSC, the target BSC encapsulates the message and then forwards the message to the target cell. This measurement provides the number of incoming-BSC handover request messages received by the current cell. This counter indicates the frequency of the handover from a cell under control of another BSC to the current cell.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see Attempted Incoming Inter-BSC Inter-Cell VGCS TCH Handovers.

Formula

Attempted Incoming Inter-BSC Inter-Cell VGCS TCH Handovers per BSC = Attempted Incoming Inter-BSC Inter-Cell VGCS TCH Handovers

(That is, the total of the counter Attempted Incoming Inter-BSC Inter-Cell VGCS TCH Handovers in each cell of the BSC.)
