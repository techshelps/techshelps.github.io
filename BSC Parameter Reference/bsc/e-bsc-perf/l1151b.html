Maximum Number of Busy Timeslots per Abis Port

Measurement Counter

L1151B: BS_ABIS_USED_TS_MAX

Description

This measurement provides the maximum number of busy timeslots per Abis port within the measurement period.

Unit

Integer number or integer.

Measurement Point

The BSC counts the number of busy timeslots per Abis port every 10 seconds. Maximum Number of Busy Timeslots per Abis port compares the measurement results within a specific measurement period, and takes the maximum value as the maximum number of busy timeslots per Abis port during this period.

Formula

This is an original counter without involving any formula.
