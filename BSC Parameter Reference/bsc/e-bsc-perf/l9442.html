Number of Transmissions of Uplink GPRS Interactive Service THPx

Measurement Counter

L9442: CELL_LLC_TX_UP_GPRS_INTACT_THP1_LLC_PDU_TRANS_TIMES

L9443: CELL_LLC_TX_UP_GPRS_INTACT_THP2_LLC_PDU_TRANS_TIMES

L9444: CELL_LLC_TX_UP_GPRS_INTACT_THP3_LLC_PDU_TRANS_TIMES

Description

This measurement provides the number of transmissions of uplink GPRS interactive service THPx in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, every time the transmission of interactive THPx service is complete on the uplink GPRS TBF, the counter is incremented by one. Note: THPx indicates THP1-THP3.

Formula

This is an original counter without involving any formula.
