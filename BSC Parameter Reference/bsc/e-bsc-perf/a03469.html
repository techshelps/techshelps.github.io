Unsuccessful Internal Inter-Cell VGCS TCH Handovers (Channel Mode Unacceptable)

Measurement Counter

A03469: CELL_GCC_INTRABSC_INTERCELL_HO_FAIL_CH_ILLEGAL

Description

After the internal inter-cell VGCS TCH handover procedure is started, the originating cell sends a HANDOVER COMMAND message to the MS. When the MS successfully accesses the target cell, the target cell requests the originating cell to clear the original resources. Thus, the internal inter-cell handover procedure is complete. If the MS fails to access the target cell because the TCH assigned to the MS is invalid, for example, the channel mode or channel type is invalid, the outgoing internal inter-cell handover fails. This measurement provides the number of unsuccessful outgoing internal inter-cell VGCS TCH handovers due to invalid TCH.

Unit

Integer number or integer.

Measurement Point

In the internal inter-cell VGCS TCH handover procedure, the MS stays in the originating cell when the attempt to accessing the target cell fails. This measurement provides the number of HANDOVER FAILURE messages due to 0x02 (Abnormal release, channel unacceptable) or 0x09 (Channel mode unacceptable) that the originating cell receives from the MS.
