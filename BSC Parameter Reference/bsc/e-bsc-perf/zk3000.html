SDCCH Seizure Requests per BSC

Measurement Counter

ZK3000:BSCRPT_KPI_SD_REQ

Description

In the immediate assignment procedure, the MS requests a channel from the BSS. In the incoming BSC handover procedure, the MSC requests the BSS to assign a channel to the MS; in the internal BSC handover procedure, the BSC requests the BSS to assign a channel to the MS. In the preceding procedures, this measurement provides the number of requests for an SDCCH after the BSC receives the request.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "SDCCH Seizure Requests".

Formula

SDCCH Seizure Requests per BSC = SDCCH Seizure Requests

(That is, the total of the original counter SDCCH Seizure Requests in all the cells of the BSC.)
