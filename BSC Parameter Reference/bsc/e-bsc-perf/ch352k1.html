Failed Outgoing Inter-RAT Inter-Cell Handovers (Handover Request Rejected) (TDD)

Measurement Counter

CH352K1:CELL_INTERRAN_TDD_OUTCELL_HO_FAIL_REQ_REJ

Description

This counter provides the number of failed outgoing inter-RAT inter-cell handovers (TDD) because the handover is terminated when the BSC receives the HO RQD REJ message from the MSC. The cause carried in the HO RQD REJ message can be:
  • OM intervention
  • Equipment failure
  • No radio resource available
  • Requested terrestrial resource unavailable
  • BSS not equipped
  • Invalid cell
  • Requested transcoding/rate adaption unavailable
  • Circuit pool mismatch
  • Requested speech version unavailable
  • Ciphering algorithm not supported
  • Terrestrial circuit already allocated
  • Invalid message
  • Protocol error between BSS and MSC
  • Other causes

Unit

Integer number or integer.

Measurement Point

At point A, as shown in Figure 1 and Figure 2, the counter is measured when the BSC receives the HO RQD REJ message from the MSC.
