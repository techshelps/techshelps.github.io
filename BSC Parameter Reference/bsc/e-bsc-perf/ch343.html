Successful Incoming External Inter-Cell Handovers

Measurement Counter

CH343:CELL_INTERBSC_INCELL_HO_SUCC

Description

This measurement provides the total number of successful incoming external inter-cell handovers. The counter is calculated by subtracting the Failed Incoming External Inter-Cell Handovers from the Incoming External Inter-Cell Handover Requests.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, this counter is measured at D when the BSC receives the HO CMP message from the MS during the incoming external inter-cell handover.

Figure 1 Measurement of the counters related to incoming external inter-cell handover

A

Measurement point of Incoming External Inter-Cell Handover Requests

B

Measurement point of Incoming External Inter-Cell Handover Responses

C

Measurement point of Incoming External Inter-Cell Handover Detection Messages Received by BSC

D

Measurement point of Successful Incoming External Inter-Cell Handovers

Formula

Successful Incoming External Inter-Cell Handovers = [Incoming External Inter-Cell Handover Requests] - [Failed Incoming External Inter-Cell Handovers]
