Channel Activation Measurement of the Mode Modify Procedure

Measurement Counter

R3327A: CELL_CHMD_CH_ACT_TCHF

R3328A: CELL_CHMD_CH_ACT_TCHH

R3327B: CELL_CHMD_CH_ACT_NACK_TCHF

R3328B: CELL_CHMD_CH_ACT_NACK_TCHH

R3327C: CELL_CHMD_CH_ACT_EXP_TCHF

R3328C: CELL_CHMD_CH_ACT_EXP_TCHH

Description

In the stable signaling or service status, the BSC receives an ASS REQ message from the MSC. If the request type is voice or data and the BSC determines that the Mode Modify procedure should follow, the BSC sends a MODE MODIFY message to the BTS and at the same time the counter R3327A "Channel Activation Attempts in Mode Modify Procedure (TCHF)" or R3328A "Channel Activation Attempts in Mode Modify Procedure (TCHH)" is incremented by one for the cell where the TCH resides based on the channel type.

In the Mode Modify procedure, the BSC sends a MODE MODIFY message to the BTS. If the BTS returns a MODE MODIFY NACK message indicating the BTS Mode Modify failure, the BSC sends a MODE MODIFY message to the BTS and at the same time the counter R3327B "CHAN ACTIV NACK Messages Sent by BTS in Mode Modify Procedure (TCHF)" or R3328B "CHAN ACTIV NACK Messages Sent by BTS in Mode Modify Procedure (TCHH)" is incremented by one for the cell where the TCH resides based on the channel type.

In the Mode Modify procedure, the BSC sends a MODE MODIFY message to the BTS and starts an internal timer (5 s by default) to wait for a MODE MODIFY ACK message. If the BSC does not receive the MODE MODIFY ACK message after the timer expires, the counter R3327C "Channel Activation Timeouts in Mode Modify Procedure (TCHF)" or R3328C "Channel Activation Timeouts in Mode Modify Procedure (TCHH)" is incremented by one for the cell where the TCH resides based on the channel type.

Unit

Integer number or integer.

Measurement Point

Upon receipt of an ASS REQ message from the MSC, the BSC determines that the Mode Modify procedure should follow and so sends a MODE MODIFY message to the BTS. At this time, the counter R3327A or R332A is incremented by one. If the BTS returns a MODE MODIFY NACK message, the counter R3327B or R3328B is incremented by one. After sending a MODE MODIFY message, the BSC starts the timer and waits for a response from the BTS. If the BSC does not receive any response after the timer expires, the counter R3327C or R3328C is incremented by one.

Formula

This is an original counter without involving any formula.
