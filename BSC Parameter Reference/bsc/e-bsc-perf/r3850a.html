Number of Failures to Return to Normal Call from BSC Loopback (Territorial Resource Request Failure)

Measurement Counter

R3850a:CELL_LOCALSWITCH_LOOP_RESTORE_APPLY_TRSL_FAIL_BSC_LOOP

Description

During the process where the BSC loopback is restored to a normal call, the transmission channel between the Ater interface and the Abis interface should be reconnected and TC resources should be requested. The speech signals reach the MSC through the BSC, and are switched to the call processing module at the peer end. If the reconnection or request fails, the cause value in the loopback acknowledgement message is Territorial Resource Request Failure. Then, the BSC regards it as a call drop and then releases the call.

This measurement provides the number of failures to return to normal call from BSC loopback due to territorial resource request failure.

Unit

Integer number or integer.

Measurement Point

During the process where the BSC loopback is restored to a normal call, the transmission channel between the Ater interface and the Abis interface should be reconnected and TC resources should be requested. The speech signals reach the MSC through the BSC, and are switched to the call processing module at the peer end. If the reconnection or request fails, the cause value in the loopback acknowledgement message is Territorial Resource Request Failure. Then, this counter is measured.

Formula

This is an original counter without involving any formula.
