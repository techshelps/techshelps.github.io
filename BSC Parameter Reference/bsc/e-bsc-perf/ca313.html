Successful Assignments

Measurement Counter

CA313: CELL_ASS_SUCC

Description

This counter provides the number of successful assignments of the BSC. The number of successful assignments is equal to the number of ASS REQ messages received by the BSC minus the number of ASS FAIL messages reported by the BSC to the MSC in various scenarios, such as air interface assignment, Mode Modify, and directed retry.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, Figure 2, and Figure 3, each time the BSC sends an ASS CMP message to the MSC (see measurement point C), the counter is incremented by one.

Figure 2 Mode Modify procedure on the radio interface

Figure 3 Directed retry procedure on the radio interface

NOTE:

The directed retry inside the BSC is taken as an example.

Formula

Successful Assignments =

[Assignment Requests]-

([Failed Assignments (Signaling Channel)] +

[Failed Assignments during MOC on the A Interface (Including Directed Retry)] +

[Failed Assignments during MTC on the A Interface (Including Directed Retry)] +

[Failed Assignments during Emergency Call on the A Interface (Including Directed Retry)] +

[Failed Assignments during Call Re-establishment on the A Interface (Including Directed Retry)] +

[Failed Mode Modify Attempts (MOC) (TCHF)] +

[Failed Mode Modify Attempts (MTC) (TCHF)] +

[Failed Mode Modify Attempts (Emergency Call) (TCHF)] +

[Failed Mode Modify Attempts (Call Re-establishment) (TCHF)] +

[Failed Mode Modify Attempts (MOC) (TCHH)] +

[Failed Mode Modify Attempts (MTC) (TCHH)] +

[Failed Mode Modify Attempts (Call Re-establishment) (TCHH))
