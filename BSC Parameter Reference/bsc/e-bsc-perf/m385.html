Call Drops Due to Forced Handovers in Stable BTS Loopback State

Measurement Counter

M385:CELL_CALL_DROPS_FORCE_HO_BTS_LOOP

Description

After an MS seizes a channel, the call is in the BTS local switch state. The system initiates a forced handover and the handover fails. If the BSC needs to initiate a release procedure, the BSC measures this counter.

Unit

Integer number or integer.

Measurement Point

If the forced handover initiated by the BSC fails, the BSC regards it as a call drop. The release procedure is started. If the MS is detected in the BTS local switch state, this counter is measured.

Formula

This is an original counter without involving any formula.
