Number of Uplink EGPRS RLC Control Blocks

Measurement Counter

L9231: UP_EGPRS_RLC_SIG_BLK_NUM

Description

This measurement provides the number of uplink EGPRS RLC control blocks in a cell in a granularity period.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure in which the BSC sends a PACKET UPLINK ACK/NACK message to the MS to acknowledge the receipt of the uplink EGPRS RLC data block. Every time the BSC sends a PACKET UPLINK ACK/NACK message (see measurement point A), the counter is incremented by one.

Figure 1 Uplink EGPRS TBF control block carrying the PACKET UPLINK ACK/NACK message

In addition, every time the BSC receives the PACKET CONTROL ACKNOWLEDGEMENT message on the uplink EGPRS TBF, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
