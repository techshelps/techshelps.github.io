Channel Assignment Overflows in Concentric Cell (TCHF)

Measurement Counter

R3225H:CELL_CH_OVERFLOW_TIMES_ULCELL_FR

R3224H:CELL_CH_OVERFLOW_TIMES_OLCELL_FR

Description

If a concentric cell requests for a TCHF in the underlaid subcell or the overlaid subcell, but the assigned channel does not meet the requirement or the assignment fails, the BSC measures the number of channel assignment overflows under the related conditions.

Unit

Integer number or integer.

Measurement Point

If a concentric cell requests for a TCHF in the underlaid subcell or the overlaid subcell, the measurements are triggered when the assigned channel does not meet the requirement or when the assignment fails.

Formula

This is an original counter without involving any formula.
