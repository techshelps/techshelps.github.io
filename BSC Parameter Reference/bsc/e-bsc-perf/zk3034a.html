TCHH Traffic Volume per BSC (Excluding Very Early Assignment)

Measurement Counter

ZK3034A: BSCRPT_TCHH_TRAF_ERL_NO_EARLIEST_ASSIGN

Description

The value of this counter is equal to the average number of busy TCHHs in each cell of the BSC.

Unit

Erl.

Measurement Point

After the channels are activated, the BSC collects the number of busy TCHHs per second. At the end of the granularity period, the value of this counter is the result of dividing the total number of busy TCHHs by the number of sampling times.

Formula

This is an original counter without involving any formula.
