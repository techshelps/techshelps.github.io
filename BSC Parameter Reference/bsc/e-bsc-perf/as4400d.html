Description

The counters provide the mean RQI carried in the MRs based on TA.

Unit

dB.

Measurement Point

In the granularity period, this counter is calculated according to the following formula.

Formula

Mean RQI (TA = X) = [RQI (TA = X)]/[MRs with RQI (TA = X)]

X indicates a value or a value range.
