Call Drops in TCH Handovers per BSC (Traffic Channel)

Measurement Counter

ZCM339B:BSCRPT_A_CLR_REQ_TCH_TRAF_HO

Description

This measurement provides the number of call drops due to the Um interface failure during TCH (traffic channel) handovers.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Call Drops on Radio Interface in Handover State (TCH)".

Formula

Call Drops in TCH Handovers per BSC (Traffic Channel) =

[Failed Internal Intra-Cell Handovers (Timer Expired) (TCHF) (Signaling Channel) per BSC ] +

[Failed Internal Intra-Cell Handovers (Timer Expired) (TCHH) (Signaling Channel) per BSC ] +

[Failed Outgoing Internal Intra-Cell Handovers (Timer Expired) (TCHF) (Signaling Channel) Per BSC] +

[Failed Outgoing Internal Intra-Cell Handovers (Timer Expired) (TCHH) (Signaling Channel) Per BSC] +

[Failed Outgoing External Intra-Cell Handovers (T8 Expiry) (TCHF) (Signaling Channel) Per BSC]+

[Failed Outgoing External Intra-Cell Handovers (T8 Expiry) (TCHH) (Signaling Channel) Per BSC]
