Paging Messages Discarded from the PCH Queue

Measurement Counter

L3188L:CELL_FCTRL_PAGING_MSG_DEL_PCH_QUE

Description

When the downlink CCCH (PCH) in the cell is overloaded, the BTS measures the number of paging messages discarded from the PCH queue, including the discarded CS domain paging messages and the discarded PS domain paging messages, and sends a STATUS REPORT message to the BSC.

The reason for discarding the paging message may be queue overflow, queue overflow due to preferential insertion, or queue counter overflow.

This measurement provides the total number of discarded paging messages reported in the BTS STATUS REPORT message in a measurement period. When the cell is configured with the PCCCH, this measurement provides the number of discarded paging messages in only the CS domain.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC receives a BTS STATUS REPORT message from the BTS (including the discarded CS domain paging messages and the discarded PS domain paging messages).

Formula

This is an original counter without involving any formula.
