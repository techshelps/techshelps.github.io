MTP3 Link Measurement

Description

MTP3 Link Measurement is used to measure the MTP3 link in normal, faulty, congestion, and changing status. The measurement includes the number of messages sent and received on the link, the duration of various states, and the number of status changes.

Measurement Object

MTP3 Link
