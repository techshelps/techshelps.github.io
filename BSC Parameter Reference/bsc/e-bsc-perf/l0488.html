Maximum CBCH Message Queue Usage

Measurement Counter

L0488: CB_CBCH_PERC_LOAD_MAX

Description

This counter is used to measure the maximum CBCH message queue usage.

Unit

%.

Measurement Point

This counter is measured every other minute when the cell broadcast module is working.

Formula

This is an original counter without involving any formula.
