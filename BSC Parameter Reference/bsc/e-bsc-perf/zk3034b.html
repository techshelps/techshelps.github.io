TCHF Traffic Volume per BSC (Excluding Very Early Assignment)

Measurement Counter

ZK3034B: BSCRPT_TCHF_TRAF_ERL_NO_EARLIEST_ASSIGN

Description

The value of this counter is equal to the average number of busy TCHFs in each cell of the BSC.

Unit

Erl.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

TCHF Traffic Volume per BSC (Excluding Very Early Assignment) = [Traffic Volume on TCH per BSC (Traffic Channel)] - [TCHH Traffic Volume per BSC (Excluding Very Early Assignment)]
