Successful Reconnections to Old Channels upon Inter-RAT Outgoing Cell Handover Failure per BSC (TDD)

Measurement Counter

ZH954C:BSCRPT_INTERRAN_OUTCELL_HO_FAIL_RECONN_SUCC_TDD

Description

This measurement provides the number of failed outgoing inter-RAT inter-cell handovers per BSC in which the reconnection to the old channel succeeds.

In the inter-RAT outgoing cell handover (TDD) procedure, the BSC collects the counter Failed Outgoing Inter-RAT Inter-Cell Handovers (Reconnection to Old Channels) (TDD) in the source cell if the old channels are reconnected after the handover failure.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Outgoing Inter-RAT Inter-Cell Handover Measurement per Cell (TDD)".

Formula

Successful reconnections to old channels upon inter-RAT outgoing cell handover failure per BSC (TDD) = failed outgoing inter-RAT inter-cell handovers per BSC (reconnection to old channel succeeds) (abnormal release, unspecified) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Abnormal Release, Channel Unacceptable) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Abnormal Release, Timer Expired) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Abnormal Release, No Activity on the Radio Interface) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Preemptive Release) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Handover Failed, Timing Advance out of Range) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Channel Mode Unavailable) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Frequency Unavailable) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channels) (Call Already Cleared) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Semantically Incorrect Message) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Invalid Mandatory Information Element) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Message Type Not Existed) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Message Type Not Compatible with Protocol State) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Conditional IE Error) (TDD) +

Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (No Cell Allocation Available) (TDD) +

\Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (Reconnection to Old Channel Succeeds) (Protocol Error) (TDD) +
