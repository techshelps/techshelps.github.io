RACH Failures (Too High TA)

Measurement Counter

A349A: CELL_RACH_FAIL_TOO_HIGH_TA

Description

The BTS measures the RACH failures (too high TA) once the RACH is not successfully decoded and sends the BTS STATIS REPORT message to the BSC every minute. This measurement provides the number of RACH failures (too high TA) once the RACH is not successfully decoded.

Unit

Integer number or integer

Measurement Point

If the BTS decodes the CH REQ message with a too high TA, it sends the measurement result to the BSC every minute through the BTS STATIS REPORT message. If the BSC successfully decodes RACH FAIL INFO carried in the BTS STATIS REPORT message from the BTS in the granularity period, this counter is incremented by one.
