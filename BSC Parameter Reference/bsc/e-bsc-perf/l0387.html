Total Number of Messages Received from PCU

Measurement Counter

L0387: PCU_MSG_RX_FROM_PB_INTF_TOTAL

Description

This measurement provides the total number of messages received by the BSC on the Pb interface. The messages are sent from all the PCUs to the BSC. This counter specifies the number of all the uplink signaling messages sent from all the PCUs to the BSC. The uplink signaling messages specifies the messages of cell reset, channel request, and paging request related to PS services.

Unit

Integer number or integer.

Measurement Point

The counter is incremented by one every time the BSC receives a message from the PCU.

Formula

This is an original counter without involving any formula.
