Immediate Assignment Commands

Measurement Counter

K3101: CELL_KPI_IMM_ASS_CMD

Description

In the immediate assignment procedure, the MS sends a CHAN REQ message to the BTS. After receiving the message, the BTS sends a CHAN RQD message to the BSC. After the BSC receives the CHAN RQD message, the BSC assigns an activated signaling channel to the MS. If the BSC receives a CHAN ACT ACK message from the BTS, the signaling channel is ready for assignment. Then, the BSC sends an IMM ASS CMD message, requesting the MS to access the specified signaling channel. This signaling channel can be an SDCCH or a TCH. This counter includes the number of channel requests except for the PS service.

Unit

Integer number or integer.

Measurement Point

This counter is measured at C when the BSC sends an IMM ASS CMD message, as shown in Figure 1.
