Call Setup Indication Success Rate (Call Type)

Measurement Counter

RA303H: CELL_ESTB_IND_SUCC_RATE_CALL

Description

This counter provides the ratio of the EST IND messages received by the BSC from the BTS to the CHAN RQD messages received by the BSC when the access reason of the MS in the measured cell is MOC Non SMS, MOC SMS, MTC, Emergency Call, or Call Re-establishment.

Unit

%.

Measurement Point

None.

Formula

Call Setup Indication Success Rate (Call Type) = [Call Setup Indications (Call Type)] x {100%}/[Channel Requests (Call Type)]
