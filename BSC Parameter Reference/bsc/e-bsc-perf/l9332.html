Number of Downlink EGPRS RLC Dummy Blocks

Measurement Counter

L9332: DOWN_EGPRS_RLC_NULL_BLK_NUM

Description

This measurement provides the number of downlink EGPRS RLC dummy blocks in a cell in a granularity period.

Unit

Integer number or integer.

Measurement Point

When the BSC is not required to send downlink EGPRS RLC blocks, it sends a DOWNLINK RLC DUMMY BLOCK message to maintain the activated state of the downlink EGPRS TBF. In Abis over IP or Abis over HDLC, the number of DOWNLINK RLC DUMMY BLOCK messages sent by the BSC is reduced to save the bandwidth. Figure 1 shows the procedure in which the BSC sends an EGPRS RLC dummy block. Every time the BSC sends a DOWNLINK RLC DUMMY BLOCK message (see measurement point A), the counter is incremented by one.
