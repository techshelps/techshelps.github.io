Number of EMRs on the TCHH

Measurement Counter

CS3028: CELL_HR_EMR_NUM

Description

This measurement provides the number of EMRs or pre-processing MRs from the TCHH of the measured cell.

EMR is the acronym for Enhanced Measurement Report. EMR is an optional function provided in the MS and BSS. To use the EMR function, the MS and BSS must support the EMR processing.

Unit

Integer number or integer.

Measurement Point

After receiving a valid EMR from the BTS in the measured cell, the BSC judges whether the EMR is from TCHH. If the MR is from TCHH, this counter is incremented by one.

Formula

This is an original counter without involving any formula.
