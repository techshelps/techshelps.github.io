Number of Failed Downlink EGPRS TBF Establishments due to Other Causes (BSC Fails to Send Resource Assignment Command)

Measurement Counter

A9333: DOWN_EGPRS_TBF_ESTB_FAIL_SEND_ASSMSG_FAIL

Description

This measurement provides the number of failed downlink EGPRS TBF establishments due to failure in sending the resource assignment command in a granularity period.

Unit

Integer number or integer.

Measurement Point

In dedicated mode, the BSS sends the resource assignment command to the MS if the network side and the MS support the DTM. If the downlink EGPRS TBF establishment fails due to failure in sending the resource assignment command, the counter is incremented by one.

Figure 1 Failed downlink EGPRS TBF establishments due to other causes (BSC fails to send resource assignment command)

Formula

This is an original counter without involving any formula.
