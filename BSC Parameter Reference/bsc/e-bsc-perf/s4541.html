Number of Downlink Frames with HR AMR of N kbit/s in the Customized MR

Measurement Counter

S4541: TRX_EXT_AMR_HR_475_DN_NUM

S4543: TRX_EXT_AMR_HR_515_DN_NUM

S4545: TRX_EXT_AMR_HR_590_DN_NUM

S4547: TRX_EXT_AMR_HR_670_DN_NUM

S4549: TRX_EXT_AMR_HR_740_DN_NUM

S4535A: TRX_EXT_AMR_HR_795_DN_NUM

Description

This counter is used to measure the TRX level. The BTS reports the number of downlink frames with AMR HR of N kbit/s in the customized MR, and the coding rate should be 4.75 kbit/s, 5.15 kbit/s, 5.90 kbit/s, 6.70 kbit/s, 7.40 kbit/s, or 7.95 kbit/s.

When decoding the customized extended MR or pre-processed customized extended MR from the BTS, the BSC decodes the number of downlink frames and the specified counter Number of Downlink Frames with HR AMR of N kbit/s in the Customized MR is incremented by the number of downlink frames, if the EI_FRAME is contained in the customized extended MR or pre-processed customized extended MR, and the frame type is AMR HR of N kbit/s.

NOTE:

EI_FRAME: If the EI_FRAME is contained in the MR from the BTS, it indicates that the number of uplink/downlink frames of a certain coding rate, and uplink/downlink power information are carried in the MR.

Unit

Integer number or integer.

Measurement Point

After receiving a customized extended MR or pre-processed customized extended MR from the BTS, the BSC decodes the MR. This counter is measured when the EI_FRAME is found and it contains the number of downlink frames with AMR HR of N kbit/s.

N specifies the coding rate, and the values of N are as follows: 4.75 kbit/s, 5.15 kbit/s, 5.90 kbit/s, 6.70 kbit/s, 7.40 kbit/s, and 7.95 kbit/s.

Measurement object: TRX

Formula

This is an original counter without involving any formula.
