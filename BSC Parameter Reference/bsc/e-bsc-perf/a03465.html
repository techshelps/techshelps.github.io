Successful Incoming Internal Inter-Cell VGCS TCH Handovers

Measurement Counter

A03465: CELL_GCC_INTRABSC_INTOCELL_HO_SUCC

Description

In the internal inter-cell VGCS TCH handover, after the MS successfully accesses the target cell, the target cell becomes the serving cell. Then, the target cell sends a HANDOVER PERFORMED message to the MSC. This measurement provides the number of HANDOVER PERFORMED messages that the target cell sends to the MSC.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the target cell sends a HANDOVER PERFORMED message to the MSC. In the signaling procedure, this counter is measured at point D in Figure 1.

Figure 1 Procedure of internal inter-cell VGCS TCH handover

Formula

This is an original counter without involving any formula.
