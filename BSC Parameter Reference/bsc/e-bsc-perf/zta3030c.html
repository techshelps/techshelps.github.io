Successful Pagings on the Abis Interface per BSC (CS Service)

Measurement Counter

ZTA3030C:BSCRPT_ESTB_IND_MTC_ABIS

Description

This measurement provides the count of successful paging on the Abis interface for the CS service per BSC, that is, the total of successful paging on the Abis interface for the CS service in all cells.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.

Formula

Successful Pagings on the Abis Interface per BSC (CS Service) =Call Setup Indications (MTC) (SDCCH) +Call Setup Indications (MTC) (TCHF) +Call Setup Indications (MTC) (TCHH)
