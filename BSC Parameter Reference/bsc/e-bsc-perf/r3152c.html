Number of Failed Additions of Sub-Timeslots (Dynamic Abis Timeslots) (Territorial Resource Request Failure)

Measurement Counter

R3152C:CELL_SUB_TS_REQ_FAIL_TIMES_FLEX_ABIS_TER_CIR_REQ_FAIL

Description

The prerequisite for this counter is that the PCU requests the BSC to assign additional sub-timeslots. The BSC fails to assign dynamic Abis timeslots because the request for territorial resources fails. This counter is used to measure the number of failed additions of dynamic Abis timeslots.

Unit

Integer number or integer.

Measurement Point

The PCU requests the BSC to assign additional sub-timeslots. If the BSC fails to assign dynamic Abis timeslots because the request for territorial resources fails, this counter is measured.

Formula

This is an original counter without involving any formula.
