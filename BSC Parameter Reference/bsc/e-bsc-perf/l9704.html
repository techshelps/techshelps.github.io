Total Number of Bytes of Sent NS PDUs (NSVC)

Measurement Counter

L9704: NSVC_TRAN_TX_NS_PDU_TOTAL_BYTE

Description

This measurement provides the total number of bytes of NS PDUs sent on an NSVC in Gb over FR in a granularity period.This counter indicates the average throughput of uplink NS PDUs. It is used to measure the throughput of uplink NS PDUs on an NSVC of the NS layer in a granularity period.

Unit

byte.

Measurement Point

In a granularity period, every time the BSC sends an uplink NS PDU message on an NSVC, the counter is incremented by the number of bytes of the NS PDU message.

Formula

This is an original counter without involving any formula.
