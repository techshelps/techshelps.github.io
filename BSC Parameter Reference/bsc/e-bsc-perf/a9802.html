Number of Uplink DTM Assignments Sent on DCCH

Measurement Counter

A9802: DTM_DCCH_UPLINK_DTM_ASS

Description

This measurement provides the number of uplink DTM assignment messages sent on the DCCH.

Unit

None.

Measurement Point

After receiving the DTM REQUEST message from the MS, the BSC sends the uplink DTM ASSIGNMENT COMMAND message to the MS if the channel reassignment is required. At measurement point B in Figure 1, the counter is incremented by one.

Figure 1 DTM access procedure in the case of channel reassignment

Formula

This is an original counter without involving any formula.
