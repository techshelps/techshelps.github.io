Average Number of GPRS PDCHs Occupied by GPRS TBF

Measurement Counter

AR9329: AVG_ALL_GPRS_TBF_HO_GPRS_PDCH_NUM

Description

This measurement provides the average number of GPRS PDCHs occupied by all GPRS TBFs in a cell in a granularity period. If the measurement value is high, you can infer that the GPRS services are busy.

Unit

Integer number or integer.

Measurement Point

None.

Formula

Average Number of GPRS PDCHs Occupied by GPRS TBF = Total Number of Sampled GPRS PDCHs Occupied by GPRS TBF/Sampling Times of PDCH Measurement
