Failed Dual-Band Handovers per BSC

Measurement Counter

ZK3173F: BSCRPT_KPI_DUBAND_HO_FAIL

Description

This measurement provides the total number of failed handovers between cells on different frequency bands, including the failed internal inter-cell handovers and failed external inter-cell handovers.

Unit

Integer number or integer.

Measurement Point

The value of this counter is calculated according to the following formula.

Formula

Failed Dual-Band Handovers per BSC =

[Failed Incoming Internal Inter-Cell Handovers (SDCCH) (900/850-1800/1900) per BSC] +

[Failed Incoming Internal Inter-Cell Handovers (SDCCH) (900/850-1800/1900) per BSC] +

[Failed Incoming External Inter-Cell Handovers (SDCCH) (900/850-1800/1900) per BSC] +

[Failed Incoming External Inter-Cell Handovers (TCH) (900/850-1800/1900) per BSC] +

[Failed Outgoing External Inter-Cell Handovers (SDCCH) (Excluding Directed Retry) (900/850-1800/1900) per BSC] +

[Failed Outgoing External Inter-Cell Handovers (TCHF) (Excluding Directed Retry) (900/850-1800/1900) per BSC] +

[Failed Outgoing External Inter-Cell Handovers (TCHH) (Excluding Directed Retry) (900/850-1800/1900) per BSC] +

[Failed Outgoing External Inter-Cell Handovers (Directed Retry) (900/850-1800/1900)] +

[Failed Incoming Internal Inter-Cell Handovers (SDCCH) (1800/1900-900/850) per BSC] +

[Failed Incoming Internal Inter-Cell Handovers (TCH) (1800/1900-900/850) per BSC] +

[Failed Incoming External Inter-Cell Handovers (SDCCH) (1800/1900-900/850) per BSC] +

[Failed Incoming External Inter-Cell Handovers (TCH) (1800/1900-900/850) per BSC] +

[Failed Outgoing External Inter-Cell Handovers (SDCCH) (Excluding Directed Retry) (1800/1900-900/850) per BSC] +

[Failed Outgoing External Inter-Cell Handovers (TCHF) (Excluding Directed Retry) (1800/1900-900/850 per BSC] +

[Failed Outgoing External Inter-Cell Handovers (TCHH) (Excluding Directed Retry) (1800/1900-900/850) per BSC] +

[Failed Outgoing External Inter-Cell Handovers (Directed Retry) (1800/1900-900/850)]
