Sent PS Bytes

Measurement Counter

S4011B: TRX_TX_PS_TRAFFIC_NUM

Description

This measurement provides the number of the bytes in the PS packets that the TRX sends to the BSC over the HDLC path when transmission optimization is implemented in the BTS.

Unit

byte.

Measurement Point

The TRX performs the measurement when sending the PS packets. The BTS periodically reports the results, and the BTSM disassembles the packets and processes the measurement results.

Formula

This is an original counter without involving any formula.
