LOAD IND Messages Sent on the A Interface

Measurement Counter

L3188G:CELL_CELL_LOAD_IND_MSC

Description

When the BSC detects that there is no available traffic channel in a cell but the cell is configured with external neighbor cells, the BSC sends a LOAD IND message to the MSC to avoid the handover of the calls in the external neighbor cells to the cell. The MSC then forwards the LOAD IND message to the BSS where the external neighbor cells are located. When selecting candidate cells for handovers, the BSS where the external neighbor cells are located does not select the cell where congestion occurs. The BSC checks the number of idle TCHs every five seconds in the cells under the control of the BSC.

This measurement provides the number of LOAD IND messages sent from the BSC to the MSC.

Unit

Integer number or integer.

Measurement Point

When the BSC detects that there is no available traffic channel in a cell but the cell is configured with external neighbor cells, the BSC sends a LOAD IND message to the MSC and this counter is incremented by one.

Formula

This is an original counter without involving any formula.
