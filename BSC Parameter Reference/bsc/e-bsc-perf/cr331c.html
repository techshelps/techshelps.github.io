Channel Activation Timeouts in Assignment Procedure

Measurement Counter

CR331C: CELL_ASS_CH_ACT_EXP

Description

This measurement counter provides the total number of channel activation timeouts (the timer for the BSC to wait for a channel activation response from the BTS expires) in the assignment procedure. The channel type carried in the CHANNEL ACTIVATION message can be TCHF or TCHH.

Unit

Integer number or integer.

Measurement Point

Each time the timer for the BSC to wait for a channel activation response from the BTS expires in the assignment procedure, the counter is incremented by one.

Formula

Channel Activation Timeouts in Assignment Procedure = [Channel Activation Timeouts in Assignment Procedure (TCHF)] + [Channel Activation Timeouts in Assignment Procedure (TCHH)]
