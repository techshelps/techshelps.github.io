MSC Rejects in Response to the CM Service Requests from the MS per BSC (Congestion)

Measurement Counter

ZA3050:BSCRPT_CM_SERV_REJ_CONG

Description

In the immediate assignment procedure where an MS requests for connection setup, the MS establishes an RR connection with the BSS first, and then sends a CM SERV REQ message on the MM layer. The BSS transparently transmits this message to the NSS. If the NSS does not support or cannot respond to the service request because of congestion, the MSC sends a CM SERV REJ message to the MS and the BSC transparently transmits this message.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Rejects to Service Requests (Congestion)".

Formula

MSC Rejects in Response to the CM Service Requests from the MS per BSC (Congestion) = Rejects to Service Requests (Congestion)

(That is, the total of the original counter Rejects to Service Requests (Congestion) in all the cells of the BSC.)
