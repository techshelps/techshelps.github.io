Number of Failures to Decode the Code Streams on the RACH

Measurement Counter

A334: CELL_UNSOLVES

Description

This measurement provides the number of failures in decoding the code streams on the RACH.

The BTS measures the value of the CELL_UNSOLVES and reports it in the BTS_STATIS_REPORT message to the BSC every minute.

Unit

Integer number or integer

Measurement Point

This counter is incremented each time the information element CELL_UNSOLVES in the BTS_STATIS_REPORT message is successfully decoded at C1 as shown in Figure 1.

Figure 1 Procedure of the BTS sending the BTS_STATIS_REPORT message

Formula

This is an original counter without involving any formula.
