Description

If the incoming inter-RAT inter-cell handover (TDD) fails during the handover procedure, the specific counter is measured in the target cell under control of the target BSC, based on the type of the target channel and the cause of the failure. Together with Incoming Inter-RAT Inter-Cell Handover Requests (TDD) and Incoming Inter-RAT Inter-Cell Handover Responses (TDD), this counter indicates the performance of the incoming inter-RAT inter-cell handover (TDD).

The following counters provide the numbers of failed incoming inter-RAT inter-cell handovers (TDD), based on the specific type of the target channel: SDCCH, TCHF, or TCHH. They are measured upon the termination of the incoming inter-RAT inter-cell handover (TDD).
  • H36201:CELL_INTERRAN_TDD_INCELL_HO_FAIL_SD
  • H36271:CELL_INTERRAN_TDD_INCELL_HO_FAIL_TCHF
  • H36281:CELL_INTERRAN_TDD_INCELL_HO_FAIL_TCHH
NOTE:
The main causes of failed incoming inter-RAT inter-cell handovers (TDD) include:
  • (1) Causes related to message decoding
    • The BSC fails to decode the HO REQ message.
  • (2) Causes related to resource allocation
    • No traffic channel is available.
    • No speech version is available.
    • The BSC fails to obtain the terrestrial resources.
    • No CIC is available.
    • The CIC is already allocated.
  • 3) Causes related to the Um interface
    • The timer for the BSC to wait for an HO CMP message expires.
  • (4) Other causes
    • The BSC releases the call after receiving the CLEAR CMD from the MSC.
The following counters provide the number of failed incoming inter-RAT inter-cell handovers (TDD) due to the specific cause, and are measured upon the termination of the incoming inter-RAT inter-cell handover (TDD).
  • (1) The following counter provides the number of a failed incoming inter-RAT inter-cell handover (TDD) because the BSC fails to decode an HO REQ message with invalid contents from the MSC:
    • H362J1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_INVALID_CONTENTS
  • (2) The following counters provide the numbers of failed incoming inter-RAT inter-cell handovers (TDD) due to failures in allocating relevant resources.

    • H3620A1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_CONG_SD

      H3629A1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_CONG_TCH

      Provides respectively the number of failed incoming inter-RAT inter-cell handovers (TDD) because the SDCCH or TCH is unavailable in the target cell. The specific counter is measured when the channel allocation fails.

    • H362L1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_NO_SPEECH_VER

      Provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because no proper speech version is available based on the speech versions carried in the HO REQ message from the MSC, the speech versions supported by the target cell, and the speech versions permitted by the A interface phase tag.

    • H362B1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_APPLY_TRSL_FAIL

      Provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because the BSC fails to establish the terrestrial link.

    • H362E1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_NO_CIC

      Provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because the specified CIC on the A interface is unavailable when the BSC establishes the terrestrial link after receiving the HO REQ from the MSC.

    • H362F1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_CIC_ALLOC

      Provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because the specified CIC on the A interface has already been allocated to another MS, after the BSC receives the HO REQ from the MSC.

  • (3) The following counters provide the numbers of failed incoming inter-RAT inter-cell handovers (TDD) because the MS fails to access the new channel on the Um interface.

    • H3620C1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_EXP_SD

      H3629Cb1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_EXP_TCH_SIG

      H3629Ca1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_EXP_TCH_TRAF_CH

      Provides the number of failed incoming inter-RAT inter-cell handovers (TDD), based on the channel type and service type of the target channel (respectively SDCCH, TCH as signaling channel, and TCH as traffic channel), because timer T3121 expires after the BSC allocates and activates the channel in the target cell and sends an HO REQ ACK message to the MSC.

  • (4) The following counter provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because the BSC releases the call after receiving the CLEAR CMD message from the MSC.

    • H362H1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_MSC_CLR_CMD

      Provides the number of failed incoming inter-RAT inter-cell handovers (TDD) because the BSC releases the call after receiving the CLEAR CMD message from the MSC.

Unit

Integer number or integer.

Figure 1 Measurement of counters related to the failed incoming inter-RAT inter-cell handover (TDD)

  • As shown in Figure 1, after the BSC receives an HO REQ message from the MSC, the following counters are measured at A if the incoming inter-RAT inter-cell handover (TDD) fails respectively because the message has an invalid content, no suitable channel is available in the target cell, and no proper speech version is available:
    • H362J1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_INVALID_CONTENTS
    • H3620A1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_CONG_SD
    • H3629A1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_CONG_TCH
    • H362L1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_NO_SPEECH_VER
  • As shown in Figure 1, the following counters are measured at B if the incoming inter-RAT inter-cell handover (TDD) fails because the specified CIC is unavailable or has already been allocated to another MS, or the BSC fails to apply for the terrestrial resources:
    • H362B1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_APPLY_TRSL_FAIL
    • H362E1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_NO_CIC
    • H362F1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_CIC_ALLOC
  • As shown in Figure 1, the following counters are measured at C if the incoming inter-RAT inter-cell handover (TDD) fails because T3121 expires after the BSC sends the HO REQ ACK message to the MSC.
    • H3620C1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_EXP_SD
    • H3629Cb1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_EXP_TCH_SIG
    • H3629Ca1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_EXP_TCH_TRAF_CH
  • As shown in Figure 1, the following counter is measured at D if the incoming inter-RAT inter-cell handover (TDD) fails because the BSC releases the call after receiving the CLEAR CMD from the MSC.
    • H362H1:CELL_INTERRAN_TDD_INCELL_HO_FAIL_MSC_CLR_CMD
  • As shown in Figure 1, the following counters are measured at A, B, C, and D when an incoming inter-RAT inter-cell handover (TDD) fails.
    • H36201:CELL_INTERRAN_TDD_INCELL_HO_FAIL_SD
    • H36271:CELL_INTERRAN_TDD_INCELL_HO_FAIL_TCHF
    • H36281:CELL_INTERRAN_TDD_INCELL_HO_FAIL_TCHH
