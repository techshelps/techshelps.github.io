Sampling Times of On-Line MS Number per BSC

Measurement Counter

S9401:PSALL_USERCAP_BSC_SAMPLE_MS_NUM_TIMES

Description

This measurement provides the total number of sampling times of online MSs in the BSC in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the number of on-line MSs in the BSC is sampled periodically. The counter is incremented by one every time the number of online MSs in the BSC is sampled. The sampling period is 5s.

Formula

This is an original counter without involving any formula.
