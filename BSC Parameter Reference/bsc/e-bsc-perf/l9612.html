Times of Downlink Overload on BC

Measurement Counter

L9612: BC_TRAN_DOWN_OVERLOAD_TIME

Description

This measurement provides the ratio of the downlink bandwidth actually used on the BC to the link bandwidth configured on the BC. This counter is incremented by one if the ratio exceeds 80%. The counter indicates the downlink congestion of the BC. After the BC is configured, this counter can be measured.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the ratio of the downlink bandwidth actually used on the BC to the link bandwidth configured on the BC is measured every ten seconds. If the ratio exceeds 80%, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
