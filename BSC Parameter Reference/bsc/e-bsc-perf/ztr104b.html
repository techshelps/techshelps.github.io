Call Drop Rate on SDCCH (Call Type)

Measurement Counter

ZTR104B:CELL_SD_CALL_DROP_CALL

Description

This measurement provides the ratio of the number of call drops on the SDCCH to the number of successful SDCCH seizures in a BSC cell.

Only the call drop rate of MOC, MTC, call re-establishment, and emergency call on the SDCCH are provided.

Unit

%.

Measurement Point

None.

Formula

Call Drop Rate on SDCCH (Call Type) = ([Call Drops on SDCCH (Call Type)]/[Successful SDCCH Seizures (Call Type)]) x {100}
