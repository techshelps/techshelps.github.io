Sampled Measurement of Initially Configured Channels (SDCCH)

Measurement Counter

R3000A:CELL_CH_CFG_NUM_SD_900

R3000B:CELL_CH_CFG_NUM_SD_1800

Description

R3000A:CELL_CH_CFG_NUM_SD_900 provides the number of configured SDCCHs in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, and GSM850/PCS1900 cells controlled by the BSC at each sampling point. This number does not include the number of SDCCHs that are dynamically converted from TCHFs.

R3000B:CELL_CH_CFG_NUM_SD_1800 provides the number of configured SDCCHs in the DCS1800 and PCS1900 cells controlled by the BSC at each sampling point. This number does not include the number of SDCCHs that are dynamically converted from TCHFs.

Unit

Integer number or integer.

Measurement Point

The BSC takes the samples of the channel allocation status every five seconds. When the sampling timer expires, the specific counter R3000A:CELL_CH_CFG_NUM_SD_900 and R3000B:CELL_CH_CFG_NUM_SD_1800 is measured according to the frequency band of the cell and the number of configured SDCCHs in the cell.

Formula

This is an original counter without involving any formula.
