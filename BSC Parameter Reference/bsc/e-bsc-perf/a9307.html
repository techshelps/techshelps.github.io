Number of Downlink EGPRS TBF Abnormal Releases due to SUSPEND

Measurement Counter

A9307: DOWN_EGPRS_TBF_ABNORM_REL_SUSPEND

Description

This measurement provides the number of downlink EGPRS TBF abnormal releases due to SUSPEND in a granularity period. Check this measurement if the ratio of downlink EGPRS TBF normal release is low. A high measurement value is yielded because the MS often initiates CS services during the process of PS services. Another possible cause is that the cell is on the edge of the network location area, and thus the MS frequently performs the location update.

Unit

Integer number or integer.

Measurement Point

When a class B MS initiates a CS service request, the BSC terminates all the PS services of the MS irrespective of the MS status. When the MS is in transmission state and the downlink EGPRS TBF exists, the MS initiates the suspending procedure by sending the SUSPEND message and disrupts PS services. Accordingly, the BSC releases the downlink EGPRS TBF. Figure 1 shows the procedure of abnormal release of the downlink EGPRS TBF due to SUSPEND. Each time the BSC receives the SUSPEND message (see measurement point A), the counter is incremented by one.

Formula
