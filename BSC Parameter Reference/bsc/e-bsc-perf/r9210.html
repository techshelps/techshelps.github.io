Sampling Times of PDCH Queue Length

Measurement Counter

R9210: CELL_RADIO_PDCH_TIMES_QUEUE

Description

This measurement provides the number of times the PDCH queue length is sampled in the BSC in a granularity period. This measurement is used to calculate the average length of the queues.

Unit

Integer number or integer.

Measurement Point

In the granularity period, the PDCH queue length is sampled every five seconds. The counter is incremented by one every time the PDCH queue length is sampled.

Formula

This is an original counter without involving any formula.
