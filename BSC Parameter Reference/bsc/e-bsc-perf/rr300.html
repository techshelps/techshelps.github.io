SDCCH Availability

Measurement Counter

RR300: CELL_CH_AVAIL_RATE_SD

Description

This measurement provides the ratio of available SDCCHs to configured SDCCHs.

Unit

%.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

SDCCH Availability =

([Mean Number of Available Channels (SDCCH) (900/850 Cell)] +

[Mean Number of Available Channels (SDCCH) (1800/1900 Cell)] x {100}

/

([Mean Number of Dynamically Configured Channels (SDCCH) (900/850 Cell)] +

[Mean Number of Dynamically Configured Channels (SDCCH) (1800/1900 Cell)])
