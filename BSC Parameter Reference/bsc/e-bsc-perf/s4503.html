Number of Measurements of the Diversity Level in the Customized MR

Measurement Counter

S4503: TRX_EXT_SUB_RX_LEV

Description

When decoding the customized extended MR or pre-processed customized extended MR from the BTS, the BSC checks whether the RXLEVEL_SUB_UP cell is contained in the MS. If the RXLEVEL_SUB_UP cell is contained, the BSC decodes the RXLEVEL_SUB and this counter is incremented by the value of the RXLEVEL_SUB.

This counter reflects the diversity receive level strength of the BTS. Each TRX of the BTS has two receiving interfaces: main interface and diversity interface. Main level refer to the uplink level on the main interface. Diversity level refer to the uplink level on the diversity interface. The main level and diversity level are unstable.

Unit

mW.

Measurement Point

After receiving a customized extended MR or pre-processed customized extended MR from the BTS, the BSC decodes the MR. This counter is incremented by the value of the RXLEVEL_SUB when the RXLEVEL_SUB_UP cell is contained in the MS and the BSC decodes the value of the RXLEVEL_SUB.

Formula

This is an original counter without involving any formula.
