Attempted VGCS TCH Handovers for Other Causes per BSC

Measurement Counter

ZTA03464:BSCRPT_GCC_HO_REQ_OTHER

Description

After the originating BSC makes the handover decision, the originating BSC analyzes the reason for the handover. This measurement provides the number of handovers due to other causes. Along with the number of successful VGCS TCH handovers for other causes, this measurement indicates the performance of the handover for other causes.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see Attempted VGCS TCH Handovers for Other Causes.

Formula

Attempted VGCS TCH Handovers for Other Causes per BSC = Attempted VGCS TCH Handovers for Other Causes

(That is, the total of the counter Attempted VGCS TCH Handovers for Other Causes in each cell of the BSC.)
