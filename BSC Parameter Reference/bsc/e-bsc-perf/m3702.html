Call Drops Due to Release Indications in Stable BSC Loopback State

Measurement Counter

M3702:CELL_CALL_DROPS_REL_IND_BSC_LOOP

Description

In the multi-frame link set-up state, when receiving the DISC frame from the MS, the BTS sends the BSC a REL IND message, indicating that call drops occur on the Um interface. If the MS is detected in the BSC local switch state, the BSC measures this counter.

Unit

Integer number or integer.

Measurement Point

If the BSC receives a REL IND message, the BSC regards it as a call drop. The release procedure is started. If the MS is detected in the BSC local switch state, this counter is measured.

Formula

This is an original counter without involving any formula.
