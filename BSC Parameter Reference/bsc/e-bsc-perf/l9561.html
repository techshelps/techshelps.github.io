Number of Sent Downlink GPRS Information Messages

Measurement Counter

L9561: DOWNLINK_GPRS_INFO

Description

This measurement provides the number of sent downlink GPRS information messages in a granularity period.

Unit

Integer number or integer.

Measurement Point

In dedicated mode, the network side can send downlink GPRS information messages on the DCCH. When receiving the LLC PDU from the SGSN, the BSS sends the downlink GRPS information messages to the MS on the DCCH if the LLC PDU can be sent as GPRS information messages. Every time a downlink GPRS information message is successfully sent, the counter is incremented by one.

DTM application scenario: When an MS is in dedicated mode and the SGSN has downlink GPRS information messages to send to the MS, the BSC determines whether the MS and BSC support the DTM function and whether the GPRS information messages can be sent on the DCCH. If the MS and BSC support the DTM function and the GPRS information messages can be sent on the DCCH, the GPRS information messages are sent on the DCCH and no downlink TBF needs to be established.

Formula

This is an original counter without involving any formula.
