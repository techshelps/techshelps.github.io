Description

TA is the acronym for Timing Advance, it indirectly reflects the distance between the MS and BTS. A larger value of TA indicates longer distance between the MS and BTS. RQI reflects the C/I ratio of a call. A measurement report contains RQI only when the call is on the AMR TRX.

During an AMR call, the MS periodically sends an MR with the RQI. After receiving the MR, the BSC decodes the TA in the RQI of the MR. A specified counter is measured based on the value of the TA. This measurement provides the distribution of the RQI based on the TA.

Unit

dB.

Measurement Point

During an AMR call, the MS periodically sends an MR with the RQI to the BTS. The BTS then sends the information to the BSC in an MR or a pre-processed MR. After receiving the a valid MR or pre-processed MR, the BSC decodes the TA in the RQI of the MR. The value of the specified counter TRX_RQI_NUM_BY_TA_N is accumulated based on the value of the TA.

N indicates a value or a value range.

Formula

This is an original counter without involving any formula.
