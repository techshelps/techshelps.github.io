CBCH Loading Indications Received by BSC (Underflow)

Measurement Counter

L0477:CB_BSC_CBCH_UNDERLOAD

Description

This counter is used to measure the number of CBCH Loading Indication (underflow) messages received by the BSC from the BTS.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC receives a CBCH Loading Indication (underflow) message from the BTS.

Formula

This is an original counter without involving any formula.
