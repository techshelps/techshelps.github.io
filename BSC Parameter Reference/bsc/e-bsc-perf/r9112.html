Number of Unsuccessful Application Attempts of Abis Timeslot for Other Cause

Measurement Counter

R9112: CIC_APPLY_FAIL_OTHER_REASON

Description

This measurement provides the number of unsuccessful applications for the Abis timeslots because of other causes.

Unit

Integer number or integer.

Measurement Point

During the coding scheme adjustment of the PDCH or the dynamic channel conversion, the bandwidth request is initiated. If the request for Abis timeslots fails because the internal logical links of the BSC are not established successfully, then the number of failed requests for Abis timeslots is accumulated to the value of the counter.

Formula

This is an original counter without involving any formula.
