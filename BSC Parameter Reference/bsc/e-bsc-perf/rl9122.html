Retransmission Rate of Downlink GPRS RLC Data Blocks

Measurement Counter

RL9122: CELL_RLC_DOWN_RLC_GPRS_BLK_RETRAN_RATE

Description

This measurement provides the retransmission rate of downlink GPRS RLC data blocks in a cell in a granularity period. This measurement indicates the quality of the downlink GPRS RLC data transmission in a granularity period. If the quality of the air interface or the data transmission is poor, the downlink data packets may be lost or faulty, which causes the BSC to retransmit the packets. Therefore, this measurement indicates the quality of the air interface or data transmission to some extent.

Unit

%.

Measurement Point

In a granularity period, the number of retransmission data blocks are equal to the total number of downlink GPRS RLC data blocks minus the number of valid downlink RLC data blocks using CSx. Then, calculate the ratio of the number of retransmission data blocks to the total number of downlink GPRS RLC data blocks to obtain the Retransmission Rate of Downlink GPRS RLC Data Blocks.

Formula

Retransmission Rate of Downlink GPRS RLC Data Blocks = (Total Number of Downlink GPRS RLC Data Blocks - Number of Valid Downlink RLC Data Blocks Using CS1 - Number of Valid Downlink RLC Data Blocks Using CS2 - Number of Valid Downlink RLC Data Blocks Using CS3 - Number of Valid Downlink RLC Data Blocks Using CS4) x 100/Total Number of Downlink GPRS RLC Data Blocks
