Rate of MRs (Excessive Coverage)

Measurement Counter

CS3031B: CELL_MR_OVER_COVERAGE_RATIO

Description

This measurement provides the ratio of the number of MRs in an excessively-covered cell to the total number of MRs in a cell.
NOTE:

An excessively-covered cell specifies that the downlink level of the cell is greater than the Cell Over Coverage Downlink RxLev Threshold and the TA value is greater than the Cell Over Coverage TA Threshold.

Unit

%.

Measurement Point

After receiving valid MRs from the BTS in TCHs, the BSC calculates the ratio of the number of MRs in an excessively-covered cell to the total number of MRs in a cell. Then, the value of the counter is obtained.

Formula

This is an original counter without involving any formula.
