VBS Assignment Requests per BSC

Measurement Counter

ZTA03416:BSCRPT_VBS_ASS_REQ

Description

During the establishment of the VBS, the MSC sends a VGCS/VBS ASSIGNMENT REQUEST message to the BSC for setting up a VBS channel. If the assignment is successful, the BSC sends a VGCS/VBS ASSIGNMENT RESULT message to the MSC; otherwise, the BSC sends a VGCS/VBS ASSIGNMENT FAILURE message to the MSC. This measurement provides the number of VGCS/VBS ASSIGNMENT REQUEST messages that the MSC sends to the BSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see VBS Assignment Request.

Formula

VBS Assignment Requests per BSC = VBS Assignment Request

(That is, the total of the counter VBS Assignment Request in each cell of the BSC.)
