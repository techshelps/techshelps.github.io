Logical Port Measurement per BSC

Description

The measurement unit Logical Port Measurement per BSC involves the measurement of Number of Packets Sent on a Logical Port, Number of Bytes Sent on a Logical Port, Number of Packets Discarded on a Logical Port, Number of Bytes Discarded on a Logical Port, Maximum Number of Bytes Sent on a Logical Port, Maximum Rate of Sending Packets on a Logical Port, and Average Rate of Sending Packets on a Logical Port.

Measurement Object

Logical port.
