Downlink Point-to-Point Short Messages on TCH

Measurement Counter

A3349B:CELL_DL_Pt_to_Pt_SMS_TCH

Description

The downlink point-to-point short message refers to the short message from an MSC to an MS.

This measurement provides the number of downlink point-to-point short messages on the TCHs in a cell.

Unit

Integer number or integer.

Measurement Point

This counter is measured when the BSC sends a downlink point-to-point short message on the TCH.

Formula

This is an original counter without involving any formula.
