Number of MRs on Uplink TCHF (Mean Receive Quality Rank N) per BSC

Measurement Counter

ZS4180SA:BSCRPT_FR_UP_RX_QLTY_0_AVR

ZS4181SA:BSCRPT_FR_UP_RX_QLTY_1_AVR

ZS4182SA: BSCRPT_FR_UP_RX_QLTY_2_AVR

ZS4183SA: BSCRPT_FR_UP_RX_QLTY_3_AVR

ZS4184SA:BSCRPT_FR_UP_RX_QLTY_4_AVR

ZS4185SA:BSCRPT_FR_UP_RX_QLTY_5_AVR

ZS4186SA:BSCRPT_FR_UP_RX_QLTY_6_AVR

ZS4187SA:BSCRPT_FR_UP_RX_QLTY_7_AVR

Description

The receive quality ranges from class 0 to class 7. Each class corresponds to a bit error rate range. This measurement provides the number of MRs with the receive quality on uplink TCHF belonging to certain class in all cells controlled by a BSC. Higher class suggests higher bit error rate and worse receive quality.

Unit

Integer number or integer.

Measurement Point

The value of this counter is calculated according to the following formula.

Formula

Number of MRs on Uplink TCHF (Mean Receive Quality Rank N) per BSC =

[Number of MRs on Uplink TCHF (Receive Level Rank 0 and Receive Quality Rank N) per BSC] +

[Number of MRs on Uplink TCHF (Receive Level Rank 1 and Receive Quality Rank N) per BSC] +

[Number of MRs on Uplink TCHF (Receive Level Rank 2 and Receive Quality Rank N) per BSC] +

[Number of MRs on Uplink TCHF (Receive Level Rank 3 and Receive Quality Rank N) per BSC] +

[Number of MRs on Uplink TCHF (Receive Level Rank 4 and Receive Quality Rank N) per BSC] +

[Number of MRs on Uplink TCHF (Receive Level Rank 5 and Receive Quality Rank N) per BSC] +

[Number of MRs on Uplink TCHF (Receive Level Rank 6 and Receive Quality Rank N) per BSC] +

[Number of MRs on Uplink TCHF (Receive Level Rank 7 and Receive Quality Rank N) per BSC]
