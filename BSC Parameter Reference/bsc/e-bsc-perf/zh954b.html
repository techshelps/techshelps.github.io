Failed Reconnections to Old Channels upon Inter-RAT Outgoing Cell Handover Failure per BSC (TDD)

Measurement Counter

ZH954B:BSCRPT_INTERRAN_OUTCELL_HO_FAIL_T8_EXP_TDD

Description

This measurement provides the number of failed reconnections to old channels upon inter-RAT outgoing cell handover failure per BSC (T8 expired).

In the inter-RAT outgoing cell handover (TDD) procedure, the BSC collects the counter Failed Outgoing Inter-RAT Inter-Cell Handovers (T8 Expired) (TDD) in the source cell if the handover failure is caused by T8 expiry.

T8 is started to wait for a CLR CMD message from the source MSC after the source BSC sends an HO CMD message to the MS through the source cell.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the formula below when the measurement period ends. For details on the measurement point, see "Outgoing Inter-RAT Inter-Cell Handover Measurement per Cell (TDD)".

Formula

(Failed reconnections to old channels upon Inter-RAT outgoing cell handover failure per BSC (TDD) = [Failed Outgoing Inter-RAT Inter-Cell Handovers per BSC (T8 Expired) (TDD)])
