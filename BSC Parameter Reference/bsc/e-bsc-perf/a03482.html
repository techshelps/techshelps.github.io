Unsuccessful Intra-Cell VGCS TCH to Dedicated TCH Handover (Frequency not Implemented)

Measurement Counter

A03482: CELL_GCC_INTRACELL_GCC_TO_TCH_HO_FAIL_FREQ_ILLEGAL

Description

The procedure for the intra-cell VGCS TCH to the dedicated TCH handover is equivalent to the inter-BSC handover procedure. The difference is that the originating BSC and the target BSC are the same. The originating BSC issues a HANDOVER COMMAND message to the MS and the MS accesses the target cell successfully. Then, the originating BSC receives a CLEAR COMMAND (reason: handover success) message from the MSC and starts to release the original resources. Thus, the handover is complete. If the MS cannot access the target cell due to the invalid frequency, then the intra-cell handover fails. This measurement provides the number of unsuccessful intra-cell VGCS TCH to dedicated TCH handovers due to the invalid frequency.

Unit

Integer number or integer.

Measurement Point

The MS cannot access the target cell and returns to the originating cell (the serving cell) due to "Frequency not implemented". This counter is incremented by one when the BSC receives a HANDOVER FAILURE message from the MS. The measurement point is C, as shown in Figure 1.
