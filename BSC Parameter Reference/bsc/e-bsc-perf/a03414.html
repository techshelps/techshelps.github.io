Unsuccessful Channel Activation During VGCS Assignment (NACK)

Measurement Counter

A03414: CELL_VGCS_ASS_FAIL_NACK

Description

After receiving the VGCS ASSIGNMENT REQUEST message from the MSC, the BSC delivers a CHANNEL ACTIVATION message for the newly assigned TCH to the BTS. This measurement provides the number of CHANNEL ACTIVATION NEGATIVE ACKNOWLEDGE messages that the BSC receives from the BTS.

Unit

Integer number or integer.

Measurement Point

In the VGCS assignment procedure, this counter is incremented by one when the BSC receives a CHANNEL ACTIVATION NEGATIVE ACKNOWLEDGE message from the BTS after delivering a CHANNEL ACTIVATION message to the BTS for the newly assigned TCH. In the signaling procedure, this counter is measured at B in Figure 1 and D in Figure 2.
