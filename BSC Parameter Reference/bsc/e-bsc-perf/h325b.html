Incoming Internal Inter-Cell Handover Clear Cmd Times (After Handover Response)

Measurement Counter

H325B:CELL_INTRABSC_INCELL_CLEAR_CMD_AFTER_HO_CMD

Description

During the incoming internal inter-cell handover procedure, after the BSC sends the HO CMD message to the MS, if the BSC receives the CLEAR CMD message from the MSC before the T3103 expires, it releases the call to complete the handover. This measurement provides the number of CLEAR CMD messages received after the HO CMD message is sent.

Unit

%.

Measurement Point

As shown in Figure 1, this counter is measured at G2 when the BSC receives the CLEAR CMD from the MSC before T3103 expires, after the BSC sends the HO CMD message to the MS.

Figure 1 Procedure for clearing the call during the incoming internal inter-cell handover

Formula

This is an original counter without involving any formula.
