Description

If the incoming inter-RAT inter-cell handover fails during the handover procedure, the specific counter is measured in the target cell under control of the target BSC, based on the type of the target channel and the cause of the failure. Together with Incoming Inter-RAT Inter-Cell Handover Requests and Incoming Inter-RAT Inter-Cell Handover Responses, this counter indicates the performance of the incoming inter-RAT inter-cell handover.

The following counters provide the numbers of failed incoming inter-RAT inter-cell handovers, based on the specific type of the target channel: SDCCH, TCHF, or TCHH. They are measured upon the termination of the incoming inter-RAT inter-cell handover.
  • H3620:CELL_INTERRAN_INCELL_HO_FAIL_SD
  • H3627:CELL_INTERRAN_INCELL_HO_FAIL_TCHF
  • H3628:CELL_INTERRAN_INCELL_HO_FAIL_TCHH
NOTE:
The main causes of failed incoming inter-RAT inter-cell handovers include:
  • (1) Causes related to message decoding
    • The BSC fails to decode the HO REQ message.
  • (2) Causes related to resource allocation
    • No traffic channel is available.
    • No speech version is available.
    • The BSC fails to obtain the terrestrial resources.
    • No CIC is available.
    • The CIC is already allocated.
  • (3) Causes related to the Um interface
    • Timer expires before the BSC receives the HO CMP message
  • (4) Other causes
    • The BSC releases the call after receiving the CLEAR CMD from the MSC
The following counters provide the number of failed incoming inter-RAT inter-cell handovers due to the specific cause, and are measured upon the termination of the incoming inter-RAT inter-cell handover.
  • (1) The following counter provides the number of failed incoming inter-RAT inter-cell handovers because the BSC fails to decode an HO REQ message with invalid contents from the MSC:
    • H362J:CELL_INTERRAN_INCELL_HO_FAIL_INVALID_CONTENTS
  • (2) The following counters provide the numbers of failed incoming inter-RAT inter-cell handovers due to failures in allocating relevant resources.

    • H3620A:CELL_INTERRAN_INCELL_HO_FAIL_CONG_SD

      H3629A:CELL_INTERRAN_INCELL_HO_FAIL_CONG_TCH

      Provides respectively the number of failed incoming inter-RAT inter-cell handovers because the SDCCH or TCH is unavailable in the target cell. The specific counter is measured when the channel allocation fails.

    • H362L: CELL_INTERRAN_INCELL_HO_FAIL_NO_SPEECH_VER

      Provides the number of failed incoming inter-RAT inter-cell handovers because no proper speech version is available based on the speech versions carried in the HO REQ message from the MSC, the speech versions supported by the target cell, and the speech versions permitted by the A interface phase tag.

    • H362B:CELL_INTERRAN_INCELL_HO_FAIL_APPLY_TRSL_FAIL

      Provides the number of failed incoming inter-RAT inter-cell handovers because the BSC fails to apply for the terrestrial resources.

    • H362E:CELL_INTERRAN_INCELL_HO_FAIL_NO_CIC

      Provides the number of failed incoming inter-RAT inter-cell handovers because the specified CIC on the A interface is unavailable, after the BSC receives the HO REQ from the MSC.

    • H362F:CELL_INTERRAN_INCELL_HO_FAIL_CIC_ALLOC

      Provides the number of failed incoming inter-RAT inter-cell handovers because the specified CIC on the A interface has already been allocated to another MS, after the BSC receives the HO REQ from the MSC.

  • (3) The following counters provide the numbers of failed incoming inter-RAT inter-cell handovers because the MS fails to access the new channel on the Um interface.

    • H3620C:CELL_INTERRAN_INCELL_HO_FAIL_EXP_SD

      H3629Cb:CELL_INTERRAN_INCELL_HO_FAIL_EXP_TCH_SIG

      H3629Ca:CELL_INTERRAN_INCELL_HO_FAIL_EXP_TCH_TRAF_CH

      Provides the number of failed incoming inter-RAT inter-cell handovers, based on the channel type and service type of the target channel (respectively SDCCH, TCH as signaling channel, and TCH as traffic channel), because timer T3121 expires after the BSC allocates and activates the channel in the target cell and sends an HO REQ ACK message to the MSC.

  • (4) The following counter provides the number of failed incoming inter-RAT inter-cell handovers because the BSC releases the call after receiving the CLEAR CMD message from the MSC.

    • H362H:CELL_INTERRAN_INCELL_HO_FAIL_MSC_CLR_CMD

      Provides the number of failed incoming inter-RAT inter-cell handovers because the BSC releases the call after receiving the CLEAR CMD message from the MSC.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the measurement of counters relates to the failed incoming inter-RAT inter-cell handover procedure.

Figure 1 Measurement of counters related to the failed incoming inter-RAT inter-cell handover

  • As shown in Figure 1, after the BSC receives an HO REQ message from the MSC, the following counters are measured at A if the incoming inter-RAT inter-cell handover fails, respectively because the message has an invalid content, no suitable channel is available in the target cell, and no proper speech version is available:
    • H362J:CELL_INTERRAN_INCELL_HO_FAIL_INVALID_CONTENTS
    • H3620A:CELL_INTERRAN_INCELL_HO_FAIL_CONG_SD
    • H3629A:CELL_INTERRAN_INCELL_HO_FAIL_CONG_TCH
    • H362L: CELL_INTERRAN_INCELL_HO_FAIL_NO_SPEECH_VER
  • As shown in Figure 1, the following counters are measured at B if the incoming inter-RAT inter-cell handover fails because the specified CIC is unavailable or has already been allocated to another MS, or the BSC fails to establish the terrestrial link:
    • H362E:CELL_INTERRAN_INCELL_HO_FAIL_NO_CIC
    • H362F:CELL_INTERRAN_INCELL_HO_FAIL_CIC_ALLOC
    • H362B:CELL_INTERRAN_INCELL_HO_FAIL_APPLY_TRSL_FAIL
  • As shown in Figure 1, if the incoming inter-RAT inter-cell handover fails because the T3121 expires after the BSC sends the HO REQ ACK message to the MSC, the specific one of the following counters is measured at C:
    • H3620C:CELL_INTERRAN_INCELL_HO_FAIL_EXP_SD
    • H3629Cb:CELL_INTERRAN_INCELL_HO_FAIL_EXP_TCH_SIG
    • H3629Ca:CELL_INTERRAN_INCELL_HO_FAIL_EXP_TCH_TRAF_CH
  • As shown in Figure 1, if the incoming inter-RAT inter-cell handover fails because the BSC releases the call after receiving the CLEAR CMD from the MSC, the following counter is measured at D: H362H:CELL_INTERRAN_INCELL_HO_FAIL_MSC_CLR_CMD
  • As shown in Figure 1, when an incoming inter-RAT inter-cell handover fails, the specific one of the following counters is measured at A, B, C, and D irrespective of the cause:
    • H3620:CELL_INTERRAN_INCELL_HO_FAIL_SD
    • H3627:CELL_INTERRAN_INCELL_HO_FAIL_TCHF
    • H3628:CELL_INTERRAN_INCELL_HO_FAIL_TCHH

Formula

This is an original counter without involving any formula.
