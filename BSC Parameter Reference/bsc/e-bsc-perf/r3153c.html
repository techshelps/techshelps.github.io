Number of Pre-emptions of the Dynamic Abis Sub-Timeslots of the Lower-Level Site

Measurement Counter

R3153C:CELL_NEXT_LINK_SITE_FLEX_ABIS_TS_PREEMPT_TIMES

Description

This counter is used to measure the pre-emptions of dynamic Abis sub-timeslots of the lower-level site when the pre-emption of packet sub-timeslots by the CS service is enabled.

Unit

Integer number or integer.

Measurement Point

The BSC supports the dynamic assignment of Abis resources. If the BSC allows pre-empting packet resources when Abis resources are unavailable, the CS service pre-empts packet Abis resources. If dynamic Abis sub-timeslots are successfully pre-empted and the dynamic Abis sub-timeslots are in the lower-level site, this counter is measured.

Formula

This is an original counter without involving any formula.
