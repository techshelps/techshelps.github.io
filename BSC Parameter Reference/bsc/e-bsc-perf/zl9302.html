Total Number of Downlink EGPRS MCS1 RLC Data Blocks per BSC

Measurement Counter

ZL9302: BSCRPT_BSC_PSTRAN_DOWN_EGPRS_MCS1_RLC_DATA_BLK

Description

This measurement provides the total number of downlink EGPRS RLC data blocks using MCS-1 coding scheme in the BSC. This counter indicates the usage of downlink EGPRS RLC data blocks using MCS-1 coding scheme in all the cells controlled by the BSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see the cell-level counter described in Total Number of Downlink EGPRS MCS1-MCS6 Data Blocks.

Formula

Total Number of Downlink EGPRS MCS1 RLC Data Blocks per BSC = Total Number of Downlink EGPRS MCS1 RLC Data Blocks

(That is, the total of the counter Total Number of Downlink EGPRS MCS1 RLC Data Blocks in each cell of the BSC.)
