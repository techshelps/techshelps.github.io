Duration of MTP3 Link Local Inhibition

Measurement Counter

L6004:MTP3_LNK_LOCAL_INH_DUR

Description

This measurement provides the duration of the MTP3 link local inhibition.

Unit

Integer number or integer.

Measurement Point

The measurement point is on the BSC side, as shown in the figure.

This counter is incremented by 1 every second when the MTP3 link is at the local end inhibition.

Formula

This is an original counter without involving any formula.
