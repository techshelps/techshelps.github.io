External Outgoing Cell Handover Requests per BSC

Measurement Counter

ZCH330:BSCRPT_INTERBSC_OUTCELL_HO_REQ

Description

This measurement provides the number of external outgoing cell handover requests of all the cells controlled by a BSC. The channel involved in the measurement can be the SDCCH, TCHF, or TCHH.

This counter is measured based on the channel type, signaling points, and handover initiation causes.

The handover between different signaling points in a BSC is also called outgoing external inter-cell handover.

The handover causes consist of the following: upper quality, downlink quality, uplink strength, downlink strength, timing advance, better cell, load, rapid level drop, MSC intervention, OM intervention, EDGE coverage, and other causes.

According to the counters External Outgoing Cell Handover Commands per BSC and Successful External Outgoing Cell Handovers per BSC, the external outgoing cell handover per BSC can be well understood. According to the counters related to external incoming cell handovers, the external inter-cell handover per BSC can be well understood.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.

Formula

External Outgoing Cell Handover Requests per BSC = Outgoing External Inter-Cell Handover Requests

(That is, the total of the original counter Outgoing External Inter-Cell Handover Requests in all the cells of the BSC.)
