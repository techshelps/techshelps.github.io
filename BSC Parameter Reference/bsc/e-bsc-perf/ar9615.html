Average Uplink Bandwidth per EGPRS User

Measurement Counter

AR9615: CELL_PDCH_UP_EGPRS_TBF_AVG_THRPUT_RATE

Description

This measurement provides the average bandwidth per user of the uplink EGPRS services in a granularity period.

Unit

Kbit/s.

Measurement Point

None.

Formula

Average uplink bandwidth per EGPRS user = Average uplink bandwidth per EGPRS PDCH x Number of actually used uplink PDCHs per EGPRS user (user)
