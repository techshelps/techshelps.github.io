Retransmission Rate of Downlink EGPRS RLC Data Blocks

Measurement Counter

RL9337: CELL_RLC_DOWN_RLC_EGPRS_BLK_RETRAN_RATE

Description

This measurement provides the retransmission rate of downlink EGPRS RLC data blocks in a cell in a granularity period. This measurement indicates the quality of the downlink EGPRS RLC data transmission in a granularity period. If the quality of the air interface or the data transmission is poor, the downlink data packets may be lost or faulty, which causes the BSC to retransmit the packets. Therefore, this measurement indicates the quality of the air interface or data transmission to some extent.

Unit

%.

Measurement Point

In a granularity period, the number of retransmission data blocks are equal to the total number of downlink EGPRS RLC data blocks minus the total number of valid downlink EGPRS MCSx RLC data blocks. Then, calculate the ratio of the number of retransmission data blocks to the total number of downlink EGPRS RLC data blocks to obtain the Retransmission Rate of Downlink EGPRS RLC Data Blocks.

Formula

Retransmission Rate of Downlink EGPRS RLC Data Blocks = (Total Number of Downlink EGPRS RLC Data Blocks - Total Number of Valid Downlink EGPRS MCS1 RLC Data Blocks - Total Number of Valid Downlink EGPRS MCS2 RLC Data Blocks - Total Number of Valid Downlink EGPRS MCS3 RLC Data Blocks - Total Number of Valid Downlink EGPRS MCS4 RLC Data Blocks - Total Number of Valid Downlink EGPRS MCS5 RLC Data Blocks - Total Number of Valid Downlink EGPRS MCS6 RLC Data Blocks - Total Number of Valid Downlink EGPRS MCS7 RLC Data Blocks - Total Number of Valid Downlink EGPRS MCS8 RLC Data Blocks - Total Number of Valid Downlink EGPRS MCS9 RLC Data Blocks) x 100/Total Number of Downlink EGPRS RLC Data Blocks
