Number of Successful Downlink EGPRS TBF Establishments

Measurement Counter

A9302: DOWN_EGPRS_TBF_ESTB_SUCC

Description

This measurement provides the number of successful downlink EGPRS TBF establishments in a granularity period. A low measurement value indicates that the radio environment, network parameter configuration, network resource configuration, or network equipment is not satisfactory, and thus requires optimization. Further analysis should be made on the basis of the causes of EGPRS TBF abnormal release. Together with Number of Successful Downlink EGPRS TBF Establishments, this measurement indicates the downlink assignment success rate in a granularity period. Together with Number of Downlink EGPRS TBF Abnormal Releases, this measurement indicates the call drop rate of the downlink EGPRS TBF in a granularity period.

Unit

Integer number or integer.

Measurement Point

The successful downlink EGPRS TBF establishments involve the following aspects:

  • Successful downlink EGPRS TBF establishment on the CCCH

    The BSC initiates the request for downlink EGPRS TBF establishment by sending the IMMEDIATE ASSIGNMENT message with Starting Time on the CCCH. When Starting Time expires, the BSC sends the POLLING message to the MS to obtain the TA value. The BSC also reserves the block resource for the MS to respond with the assignment acknowledgement message. If the BSC receives the PACKET CONTROL ACKNOWLEDGEMENT message from the MS in the reserved block, then the downlink EGPRS TBF establishment is successful. On the basis of received PACKET CONTROL ACKNOWLEDGEMENT message, the BSC can calculate the TA value. Figure 1 shows the procedure of downlink EGPRS TBF establishment on the CCCH. Each time the BSC receives a PACKET CONTROL ACKNOWLEDGEMENT message (see measurement point B), the counter is incremented by one.

    Figure 1 Downlink packet immediate assignment

  • Successful downlink EGPRS TBF establishment on the PACCH

    The BSC can initiate the downlink EGPRS TBF establishment procedure by sending the PACKET DOWNLINK ASSIGNMENT message on the uplink or downlink PACCH to the MS. The BSC also reserves the blocks for the MS to respond with the assignment acknowledgement message. If the BSC receives the PACKET CONTROL ACKNOWLEDGEMENT message from the MS on the reserved blocks, then the downlink EGPRS TBF establishment is successful. Figure 2 shows the procedure of downlink EGPRS TBF establishment on the PACCH. Each time the BSC receives a PACKET CONTROL ACKNOWLEDGEMENT message (see measurement point B), the counter is incremented by one.

    Figure 2 Downlink assignment on the PACCH

Formula

This is an original counter without involving any formula.
