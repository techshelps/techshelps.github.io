Successful TCH Seizures

Measurement Counter

K3040: CELL_KPI_TCH_SUCC

Description

In the very early assignment, assignment, directed retry, or handover procedure, this measurement provides the number of times that the MS successfully seizes the TCHs assigned by the BSS.

Unit

Integer number or integer.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

Successful TCH Seizures =

[Successful TCH Seizures (Signaling Channel)] +

[Successful TCH Seizures (Traffic Channel)] +

[Successful TCH Seizures in TCH handovers (Traffic Channel)] +
