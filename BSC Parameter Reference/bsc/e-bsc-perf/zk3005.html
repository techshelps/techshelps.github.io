Available SDCCHs per BSC

Measurement Counter

ZK3005:BSCRPT_KPI_SD_AVAIL_NUM

Description

This measurement provides the number of available SDCCHs in a cell in the BSC. The SDCCHs include those that are converted from dynamic TCHs. The value of this counter is the average number of available SDCCHs at each sampling point in the granularity period.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Available SDCCHs".

Formula

Available SDCCHs per BSC = Available SDCCHs

(That is, the total of the original counter Available SDCCHs in all the cells of the BSC.)
