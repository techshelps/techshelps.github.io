Successful Cell Outgoing Handovers (Rapid Level Drop)

Measurement Counter

H373J: CELLCELL_OUT_CELL_HO_SUCC_TIMES_RAPID_LEVEL_DROP

Description

This measurement provides the number of successful intra-BSC and inter-BSC cell outgoing handovers caused by rapid level drop. The counter value is equal to the Failed Cell Outgoing Handovers (Rapid Level Drop) subtracted from the Outgoing Inter-Cell Handover Requests (Rapid Level Drop).

Unit

Integer number or integer.

Measurement Point

See the formula for the measurement points.

Formula

Successful Cell Outgoing Handovers (Rapid Level Drop) = [Outgoing Inter-Cell Handover Requests (Rapid Level Drop)] -

[Failed Cell Outgoing Handovers (Rapid Level Drop)]
