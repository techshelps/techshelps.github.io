Channel Activation Attempts

Measurement Counter

CR33A: CELL_CH_ACT

Description

This measurement provides the total number of CHANNEL ACTIVATION messages and MODE MODIFY messages sent by the BSC to the BTS. The CHANNEL ACTIVATION message is sent in the immediate assignment procedure, the assignment procedure, the internal intra-cell handover procedure, the incoming internal inter-cell handover procedure, and the incoming external inter-cell handover procedure. The MODE MODIFY message is sent only in the Mode Modify procedure.

Unit

Integer number or integer.

Measurement Point

Each time the BSC sends a CHANNEL ACTIVATION or MODE MODIFY message to the BTS, the counter is incremented by one.

Formula

Channel Activation Attempts = [Channel Activation Attempts in Immediate Assignment Procedure] + [Channel Activation Attempts in Assignment procedure] + [Channel Activation Attempts in Mode Modify Procedure] + [Channel Activation Attempts in Internal Intra-Cell Handover Procedure] + [Channel Activation Attempts in Internal Incoming Inter-Cell Handover Procedure] + [Channel Activation Attempts in External Incoming Inter-Cell Handover Procedure]
