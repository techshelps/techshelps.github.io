Number of N-Bit One Phase Access Requests Received on RACH

Measurement Counter

A9402: RACH_8BIT_1PHASE_ACCESS_CHAN_REQ

A9405: RACH_11BIT_1PHASE_ACCESS_REQ

Description

This measurement provides the number of N-bit one phase access request messages received on the RACH in a granularity period. This measurement indicates the type of the N-bit one phase service performed on the MS. Where, N can be 8 or 11.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure for the MS initiating an N-bit one phase access request message on the RACH. The CHANNEL REQUEST message carries a parameter indicating that the access request type is one phase packet access. Each time the BSC receives an N-bit CHANNEL REQUEST message indicating one phase packet access from the MS on the RACH (see measurement point A in Figure 1), the counter is incremented by one. Where, N can be 8 or 11.

Figure 1 One phase packet access request received on the RACH

Formula

This is an original counter without involving any formula.
