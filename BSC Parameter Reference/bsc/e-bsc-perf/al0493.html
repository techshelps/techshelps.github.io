Coverage of Cell Broadcast Short Msg

Measurement Counter

AL0493: CB_BSC_COVER_RATE

Description

This measurement provides the proportion of the Number of Cells Receiving Broadcast Messages to the Number of Configured Cells in the BSC in a granularity.

Unit

%.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

Coverage of Cell Broadcast Short Msg = [Number of Cells Receiving Broadcast Messages] x {100}/[Number of Configured Cells]
