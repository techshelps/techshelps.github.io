Configured Link Bandwidth on BC

Measurement Counter

L9606: BC_TRAN_CONFIG_BC_BANDWIDTH

Description

This measurement provides the total of sampled link bandwidths of N x 64 Kbit/s (N specifies the number of timeslots configured on the BC) configured on the BC in a granularity period. After the BC is configured, this counter can be measured.

Unit

Kbit/s.

Measurement Point

In a granularity period, the link bandwidth is measured every ten seconds according to the number of timeslots configured on the BC. The counter is incremented by the value of the link bandwidth.

Formula

This is an original counter without involving any formula.
