Failed Dynamic Assign Abis Resource (8K_CS)

Measurement Counter

R2730: SITE_ASS_FAIL_CS_8K

Description

This measurement provides the number of failed dynamic assignments of 8 kbit/s Abis resources for the voice channel when the BSC receives the assignment request from the MSC.

Unit

Integer number or integer.

Measurement Point

This counter is measured when the 8 kbit/s Abis resources are dynamically assigned to the voice channel for the CS service.

Formula

This is an original counter without involving any formula.
