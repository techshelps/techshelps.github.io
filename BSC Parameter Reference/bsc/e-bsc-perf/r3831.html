Number of Successful BTS Loopback Attempts

Measurement Counter

R3831:CELL_LOCALSWITCH_BTSLOOP_SUCCESS

Description

After the call is put through or the incoming BSC handover for the speech service is complete, the loopback start conditions are determined, the BTS local switch call detection is performed, and the speech versions of the calling and called MSs are adjusted during the BTS local loopback. Within the BSC, the call processing module initiating loopback start at one end sends a loopback request message to the call processing module at the peer end. Then, the call processing module at the peer end sends a BTS loopback request message to the BTS. After that, if the BTS loopback is successful, the BTS informs the BSC of the success.

This measurement provides the number of successful BTS loopback attempts.

Unit

Integer number or integer.

Measurement Point

After the BSC receives the CONNECT ACK message from the MSC in the MS terminated call procedure, as shown in Figure 1(A1), or after the BSC receives the HANDOVER CMP message from the MS in the incoming BSC handover procedure, as shown in Figure 2 (B1), the BSC selects the BSS local switch policy, determines whether the conditions for initiating the local switch are met, and performs local switch call detection. If all the local switch conditions are met, the BTS local loopback start procedure is started. Within the BSC, the call processing module initiating loopback start at one end sends a loopback request message to the call processing module at the peer end. The call processing module at the peer end sends a BTS loopback request message to the BTS. If the BTS loopback is successful, the BTS responds to the call processing module with a BTS loopback acknowledgement message (loopback success). Then, the call processing module at the peer end returns the call processing module at the TX end a loopback acknowledgement message (loopback success). When the call processing module at the TX end receives a loopback acknowledgement message (handover success) from the call processing module at the peer end, this counter is measured in the cell where the loopback start is initiated.
