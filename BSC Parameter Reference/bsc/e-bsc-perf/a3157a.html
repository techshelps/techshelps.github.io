Measurement of Failed Mode Modify Attempts

Measurement Counter

A3157A: CELL_CHMD_FAIL_MOC_TCHF

A3157C: CELL_CHMD_FAIL_MTC_TCHF

A3157D: CELL_CHMD_FAIL_ECALL_TCHF

A3157E: CELL_CHMD_FAIL_CALL_REEST_TCHF

A3158A: CELL_CHMD_FAIL_MOC_TCHH

A3158C: CELL_CHMD_FAIL_MTC_TCHH

A3158E: CELL_CHMD_FAIL_CALL_REEST_TCHH

A3159E: CELL_CHMD_FAIL_NO_CIC

A3159F: CELL_CHMD_FAIL_CIC_ALLOC

A3159B: CELL_CHMD_FAIL_APPLY_TRSL_FAIL

A3159G: CELL_CHMD_FAIL_A_INTF_FAIL

A3159H: CELL_CHMD_FAIL_MSC_CLR_CMD

A3159C: CELL_CHMD_FAIL_EXP

A3159I: CELL_CHMD_FAIL_NO_SPEECH_VER

Description

The counters indicate the numbers of failed Mode Modify attempts that ultimately caused failed assignments due to various reasons after the BSC sends a CHAN MODE MODIFY message to the MS.

  • The following counters provide the numbers of failed Mode Modify attempts based on the MS access reason and the type of the channel currently seized by the MS. The access reason can be MOC, MTC, Emergency Call, or Call Re-establishment. The channel type can be TCHF or TCHH.
    • A3157A: CELL_CHMD_FAIL_MOC_TCHF
    • A3157C: CELL_CHMD_FAIL_MTC_TCHF
    • A3157D: CELL_CHMD_FAIL_ECALL_TCHF
    • A3157E: CELL_CHMD_FAIL_CALL_REEST_TCHF
    • A3158A: CELL_CHMD_FAIL_MOC_TCHH
    • A3158C: CELL_CHMD_FAIL_MTC_TCHH
    • A3158E: CELL_CHMD_FAIL_CALL_REEST_TCHH
  • The following counters indicate the numbers of failed Mode Modify attempts due to the abnormality of terrestrial resources.

    • A3159E: CELL_CHMD_FAIL_NO_CIC

      The counter A3159E provides the number of ASS FAIL messages sent by the BSC to the MSC after the BSC finds that the received ASS REQ message carries an unavailable A interface CIC when the BSC uses the Mode Modify procedure to complete the assignment procedure.

    • A3159F: CELL_CHMD_FAIL_CIC_ALLOC

      The counter A3159F provides the number of ASS FAIL messages sent by the BSC to the MSC after the BSC finds that the received ASS REQ message carries an A interface CIC that is already occupied by another call when the BSC uses the Mode Modify procedure to complete the assignment procedure.

    • A3159B: CELL_CHMD_FAIL_APPLY_TRSL_FAIL

      The counter A3159B provides the number of ASS FAIL messages sent by the BSC to the MSC after the MS drops from the connection to the air interface, or a circuit fails to be obtained for the call, or the obtained circuit is faulty during the circuit connection of the BSC when the BSC uses the Mode Modify procedure to complete the assignment procedure.

    • A3159G: CELL_CHMD_FAIL_A_INTF_FAIL

      The counter A3159G provides the number of times that the BSC locally releases the call after receiving an SS7 link abnormality indication when the BSC uses the Mode Modify procedure to complete the assignment procedure.

  • The following counter indicates the number of failed Mode Modify attempts due to the call clearing performed by the MSC.

    A3159H: CELL_CHMD_FAIL_MSC_CLR_CMD

    The counter A3159H provides the number of times that the BSC releases the call after receiving a CLEAR CMD message from the MSC when the BSC uses the Mode Modify procedure to complete the assignment procedure.

  • The following counter indicates the number of failed Mode Modify attempts due to the failure of air interface access.

    A3159C: CELL_CHMD_FAIL_EXP

    The counter A3159C provides the number of ASS FAIL messages sent by the BSC to the MSC after the timer for the BSC to wait for a CHAN MODE MODIFY ACK message expires when the BSC uses the Mode Modify procedure to complete the assignment procedure.

  • The following counter indicates the number of failed Mode Modify attempts due to abnormal allocation of air interface resources.

    A3159I: CELL_CHMD_FAIL_NO_SPEECH_VER

    The counter A3159I provides the number of ASS FAIL messages sent by the BSC to the MSC when the Mode Modify fails because the intersection between the speech version set carried in the ASS REQ message from the MSC and the speech version set supported by the current cell of the MS does not have available speech versions when the BSC uses the Mode Modify procedure to complete the assignment procedure.

Unit

Integer number or integer.

Measurement Point

  • As shown in Figure 1, each time the Mode Modify procedure fails after the BSC receives an ASS REQ message from the MSC and enters the Mode Modify procedure (see measurement points F, G, and H), one of the following counters is incremented by one based on the specific failure cause:
    • A3157A: CELL_CHMD_FAIL_MOC_TCHF
    • A3157C: CELL_CHMD_FAIL_MTC_TCHF
    • A3157D: CELL_CHMD_FAIL_ECALL_TCHF
    • A3157E: CELL_CHMD_FAIL_CALL_REEST_TCHF
    • A3158A: CELL_CHMD_FAIL_MOC_TCHH
    • A3158C: CELL_CHMD_FAIL_MTC_TCHH
    • A3158E: CELL_CHMD_FAIL_CALL_REEST_TCHH
    Figure 1 Mode Modify procedure

  • As shown in Figure 1, each time the BSC sends an ASS FAIL message to the MSC after receiving an ASS REQ message from the MSC and entering the Mode Modify procedure (see measurement point F), one of the following counters is incremented by one based on the specific failure reason:
    • A3159B: CELL_CHMD_FAIL_APPLY_TRSL_FAIL
    • A3159E: CELL_CHMD_FAIL_NO_CIC
    • A3159F: CELL_CHMD_FAIL_CIC_ALLOC
    • A3159I: CELL_CHMD_FAIL_NO_SPEECH_VER
  • As shown in Figure 1, each time the BSC locally releases the call due to the failure of the SS7 link on the A interface after receiving an ASS REQ message from the MSC and entering the Mode Modify procedure (see measurement point G), the counter A3159G is incremented by one.
    • A3159G: CELL_CHMD_FAIL_A_INTF_FAIL
  • As shown in the figure, each time the BSC sends a CLEAR CMD message to the MSC after receiving an ASS REQ message from the MSC and entering the Mode Modify procedure (see measurement point H), the counter A3159H is incremented by one:
    • A3159H: CELL_CHMD_FAIL_MSC_CLR_CMD
  • As shown in Figure 1, the BSC sends a CHAN MODE MODIFY message and then starts the timer for waiting for a CHAN MODE MODIFY ACK message from the MS after receiving an ASS REQ message from the MS and entering the Mode Modify procedure. Each time the timer expires but the BSC does not receive a CHAN MODE MODIFY ACK message (see measurement point I), the counter A3159C is incremented by one.
    • A3159C: CELL_CHMD_FAIL_EXP

Formula

This is an original counter without involving any formula.
