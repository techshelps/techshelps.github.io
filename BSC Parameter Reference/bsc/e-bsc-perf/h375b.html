Cell Outgoing Handover Fail Reconn Fail

Measurement Counter

H375B: CELLCELL_OUT_CELL_HO_FAIL_RECONN_FAIL_TIMES

Description

This measurement provides the number of times that the call drops after the intra-BSC outgoing cell handover or inter-BSC outgoing cell handover fails (timer expired).

In intra-BSC outgoing cell handover (including directed retry), the BSC sends the HO CMD message to the MS in the original cell and starts the timer for the reception of the HO CMP message. If the BSC does not receive the HO CMP message when the timer expires, handover fails and call drops.

In inter-BSC outgoing cell handover (including directed retry), the BSC sends the HO CMD message to the MS in the original cell and starts the timer for the reception of the CLEAR CMD message from the peer BSC. If the BSC does not receive the CLEAR CMD message when the timer expires, handover fails and call drops.

Unit

Integer number or integer.

Measurement Point

(1) After sending the HO CMD message in the intra-BSC outgoing cell handover procedure, the BSC triggers the measurement once the timer for the HO CMD message expires.

(2) After sending the HO CMD message in the inter-BSC outgoing cell handover procedure, the BSC triggers the measurement once the timer (T8) set for the reception of the CLEAR CMD message from the peer BSC expires.

Formula

This is an original counter without involving any formula.
