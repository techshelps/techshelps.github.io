Number of Dynamic PDCH Requests Without Application Attempts Because the Number of Activated PDCHs Reaches Board Specification

Measurement Counter

R9346: NEED_DYN_PDCH_FAIL_REACH_PDCH_NUM_LIMIT

Description

This measurement provides the number of failed dynamic PDCH requests in a cell in a granularity period because of the limitation on the number of channels activated on the board.

Unit

Integer number or integer.

Measurement Point

In dynamic channel request process, the BSC determines whether the number of activated channels on the current BSC has reached the upper threshold of the channels activated on the board. If the request for dynamic channel fails, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
