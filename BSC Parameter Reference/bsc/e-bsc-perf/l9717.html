Number of Sent UNBLOCK ACK Messages

Measurement Counter

L9717: NSVC_TRAN_TX_UNBLOCK_ACK_MSG

Description

This measurement provides the number of UNBLOCK ACK messages sent from the BSC to the SGSN in a granularity period.

When the SGSN initiates the procedure for unblocking the NSVC, the BSC sends an NSVC UNBLOCK ACK message to the SGSN after receiving an UNBLOCK message.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the counter is incremented by one every time the BSC sends an NSVC UNBLOCK ACK message on an NSVC to the SGSN (see measurement point B in Figure 1).
Figure 1 NSVC UNBLOCK ACK message sent by the BSC

Formula

This is an original counter without involving any formula.
