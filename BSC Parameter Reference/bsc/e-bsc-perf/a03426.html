Unsuccessful VBS Assignment (Equipment Failure)

Measurement Counter

A03426: CELL_VBS_ASS_FAIL_EQUIP

Description

This measurement provides the number of unsuccessful VGCS assignments due to equipment failure after the BSC receives the VBS ASSIGNMENT REQUEST message from the MSC.

Unit

Integer number or integer.

Measurement Point

(1) In the VBS assignment procedure, this counter is incremented by one when the BSC fails to assign the terrestrial circuit with the CIC specified by the MSC. Meanwhile, the BSC requests the MSC to clear the connection.

(2) In the VBS assignment procedure, this counter is incremented by one when a CHAN ACTIV NACK message is received in activating the newly assigned channel or the channel activation expires.

(3) In the VBS assignment procedure, this counter is incremented by one when the assignment fails because all the internal timers expire.

Formula

This is an original counter without involving any formula.
