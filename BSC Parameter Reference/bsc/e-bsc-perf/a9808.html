Number of Downlink DTM Assignments Sent on DCCH

Measurement Counter

A9808: DTM_DCCH_DOWNLINK_DTM_ASS

Description

This measurement provides the number of downlink DTM assignment messages sent on the DCCH.

Unit

Integer number or integer.

Measurement Point

When the current call is waiting for the PS ASS message from the PS domain, the BSC sends the DTM ASSIGNMENT COMMAND message to the MS if the PS domain sends the PS ASSIGNMENT message carrying the valid downlink data. In this case, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
