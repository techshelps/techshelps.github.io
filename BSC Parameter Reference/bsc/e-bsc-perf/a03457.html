VBS TCH Uplink Call Drop

Measurement Counter

A03457: CELL_VBS_UL_CALL_DROP

Description

During a VBS call, if the link between the MS and the BTS is disconnected, the BTS sends an ERROR INDICATION or CONNECTION FAILURE INDICATION message to the BSC. This measurement provides the number of received ERROR INDICATION and CONNECTION FAILURE INDICATION messages that cause the call drop.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC receives an ERROR INDICATION or CONNECTION FAILURE INDICATION message that causes the call drop. In the signaling procedure, this counter is measured at point A in Figure 1.

Figure 1 VBS TCH uplink call drop procedure

Formula

This is an original counter without involving any formula.
