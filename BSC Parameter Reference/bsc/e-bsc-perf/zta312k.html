Failed Assignments per BSC (Terrestrial Resource Already Allocated)

Measurement Counter

ZTA312K:BSCRPT_TOTALCELL_ASS_FAIL_CIC_ALLOC

Description

In the assignment procedure, the BSS selects different traffic channels or signaling channels after the MSC sends an ASS REQ message to the BSS. Because of various causes, the assignment cannot be successfully completed.

This measurement provides the number of assignment failures due to terrestrial resources already allocated in all the cells controlled by a BSC.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Failed Assignments per Cell" and "Failed Mode Modify Attempts per Cell".

Formula

Failed Assignments per BSC (Terrestrial Resource Already Allocated) = [ Failed Assignments (CIC Allocated)] + [ Failed Mode Modify Attempts (CIC Allocated)]
