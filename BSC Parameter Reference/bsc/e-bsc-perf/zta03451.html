Inter-BSC Inter-Cell VGCS TCH Handovers per BSC

Measurement Counter

ZTA03451:BSCRPT_GCC_INTERBSC_OUTCELL_HO_CMD

Description

In the inter-BSC inter-cell VGCS TCH handover procedure, the originating cell sends a handover request to the target cell through the MSC. After the VGCS TCH activation in the target cell, the target cell sends a HANDOVER COMMAND message to the originating cell through the MSC. This measurement provides the number of times that the originating BSC receives a HANDOVER COMMAND message from the MSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see Inter-BSC Inter-Cell VGCS TCH Handovers.

Formula

Inter-BSC Inter-Cell VGCS TCH Handovers per BSC = Inter-BSC Inter-Cell VGCS TCH Handovers

(That is, the total of the counter Inter-BSC Inter-Cell VGCS TCH Handovers in each cell of the BSC.)
