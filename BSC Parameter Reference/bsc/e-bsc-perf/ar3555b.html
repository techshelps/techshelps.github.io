Rate of TCHs on BCCH TRX in Busy State

Measurement Counter

AR3555B: CELL_CHAN_BUSY_NUM_BCCH_TCH_RATE

Description

This measurement provides the rate of seized TCHs on the main BCCH TRX to all the TCHs on the main BCCH TRX in a granularity period. This measurement indicates the TCH seizure rate on the main BCCH TRX.

Unit

%.

Measurement Point

The BSC samples the channel states every five seconds, and the number of seized TCHs and available TCHs on the main BCCH TRX is measured when the timer expires. The measured value is the rate of TCHs on main BCCH TRX in busy state.

Formula

This is an original counter without involving any formula.
