Discarded PS Pagings

Measurement Counter

A350: CELL_DISC_PS_PAGING

Description

This measurement provides the number of discarded PS paging messages in the measured cell.

Unit

Integer number or integer.

Measurement Point

When the BSC decides to discard the PS paging message because of an overload on the GEIUB/GOIUB or on the link, this measurement is triggered.

Formula

This is an original counter without involving any formula.
