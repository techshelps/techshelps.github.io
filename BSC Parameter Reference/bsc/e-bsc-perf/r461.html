Available TRXs

Measurement Counter

R461:TRX_CFG_AVAIL_NUM

Description

This measurement provides the number of TRXs whose operation state is "ENABLE".

Unit

Integer number or integer.

Measurement Point

This counter is measured when the BSC queries the operation state of TRXs at regular intervals and measures the number of available TRXs.

Formula

This is an original counter without involving any formula.
