Cell Outgoing Handover Fail Reconn Success

Measurement Counter

H375A: CELLCELL_OUT_CELL_HO_FAIL_RECONN_SUCC_TIMES

Description

This measurement provides the number of times that the handover fails but the MS successfully returns to the old channel during the intra-BSC and inter-BSC outgoing cell handover.

In the intra-BSC outgoing cell handover procedure or the inter-BSC outgoing cell handover procedure, the BSC starts the timer for the HO CMP message after sending the HO CMD message to the MS in the original cell. Before the timer expires, the BSC receives the HO FAIL message from the original cell and the MS returns to the old channel.

Unit

Integer number or integer.

Measurement Point

  1. When the intra-BSC outgoing cell handover fails, the BSC triggers the measurement once the MS returns to the old channel.
  2. When the inter-BSC outgoing cell handover fails, the BSC triggers the measurement once the MS returns to the old channel.

Formula

This is an original counter without involving any formula.
