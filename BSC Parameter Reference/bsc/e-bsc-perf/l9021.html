Number of Uplink GPRS RLC Dummy Blocks

Measurement Counter

L9021: CELL_RLC_UP_RLC_NULL_BLK

Description

This measurement provides the number of uplink GPRS RLC dummy blocks in a cell in a granularity period. The network instructs the MS to send the uplink GPRS RLC block through setting the USF on the downlink GPRS RLC block. If there is no uplink GPRS RLC block available for the MS and all the sent uplink GPRS RLC blocks have been acknowledged, then the MS sends an uplink GPRS RLC dummy block.

Unit

Integer number or integer.

Measurement Point

In a granularity period, every time the BSC receives an uplink GPRS RLC dummy block, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
