Number of Received Error Packets

Measurement Counter

L01005:BSC_PIU_FEGE_RX_FCSERROR

Description

This measurement provides the number of error packets received on the Ethernet port on an IP-based interface board in a granularity period.

Unit

Integer number or integer.

Measurement Point

This measurement is performed after messages are received on the FE port from the circuit side.



Formula

This is an original counter without involving any formula.
