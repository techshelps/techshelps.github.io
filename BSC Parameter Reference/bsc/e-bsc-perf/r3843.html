Number of Attempts to Return to Normal Call from BTS Loopback (Service)

Measurement Counter

R3843: CELL_LOCALSWITCH_RESTORE_TRY_BTS_LOOP_SERVICE

Description

After the preprocessing request is handled, the BTS loopback is started. During the process of starting up the BTS, the BTS loopback recovery procedure is started because the BSC detects a supplementary service or another service that needs to be carried and processed on the core network.

Unit

Integer number or integer.

Measurement Point

1. After the BSS receives a service that needs to be carried and processed on the core network in the stable BTS local loopback state, the BTS loopback recovery procedure is started. This counter is measured on both the calling MS and the called MS.

2. After the BSS receives a service that needs to be carried and processed on the core network in the BTS local switching handover process, the local switching is not restarted and the BTS loopback recovery procedure is started if the handover fails. This counter is measured on both the calling MS and the called MS.

Formula

This is an original counter without involving any formula.
