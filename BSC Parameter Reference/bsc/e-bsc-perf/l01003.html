Number of Sent Bytes

Measurement Counter

L01003:BSC_PIU_FEGE_TX_OCTETS

Description

This measurement provides the number of bytes sent on the Ethernet port on an IP-based interface board in a granularity period.

Unit

byte.

Measurement Point

The measurement is performed after service messages from the upper layer are received on the FE port. Though the buffer of the FE port is used out, the measurement is still performed.



Formula

This is an original counter without involving any formula.
