Number of Initially Configured Channels (TCH)

Measurement Counter

CR300B: CELL_CH_CFG_NUM_TCH

Description

CR300B:CELL_CH_CFG_NUM_TCH provides the number of configured TCHs in the cells on all frequency bands controlled by the BSC at each sampling point. This number does not include the number of TCHs that are converted from dynamic PDCHs.

Unit

Integer number or integer.

Measurement Point

In the granularity period, the counters are calculated as follows.

Formula

Number of Initially Configured Channels (TCH) = [Number of Initially Configured Channels (TCHF)] + [Number of Initially Configured Channels (TCHH)]
