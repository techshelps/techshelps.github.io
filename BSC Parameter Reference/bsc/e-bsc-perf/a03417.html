Successful VGCS Assignment Queue Operation

Measurement Counter

A03417: CELL_VGCS_ASS_QUEUE_SUCC

Description

If no radio resource is available and the VGCS assignment procedure queues for idle resources after the BSC receives the VGCS ASSIGNMENT REQUEST message from the MSC, the BSC sends a VGCS QUEUING INDICATION message to the MSC. Idle radio resources can be seized. This measurement provides the number of times that the channel resource is successfully obtained in queuing.

Unit

Integer number or integer.

Measurement Point

If no radio resource is available and VGCS assignment procedure queues for idle resources, the BSC sends a VGCS QUEUING INDICATION message to the MSC. This counter is incremented by one when a channel is obtained in queuing. In the signaling procedure, this counter is measured at C in Figure 1.

Formula

This is an original counter without involving any formula.
