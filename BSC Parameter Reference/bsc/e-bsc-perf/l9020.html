Total Duration of Uplink GPRS TBFs (s)

Measurement Counter

L9020: UP_TOTAL_GPRS_TBF_EXIST_PERIOD

Description

The uplink GPRS TBFs of a cell are scanned per second. If any uplink GPRS TBF is operating in the data transmission state, it is regarded that the cell is in the data transmission state. If there is no established TBF, or the established TBFs are operating in an extended uplink inactive period, or in a non-extended uplink delayed release period, it is regarded that the cell is in the idle state. In a granularity period, the duration of scanned uplink GPRS TBFs in the data transmission state in a cell can be accumulated. That is, Total Duration of Uplink GPRS TBFs (s).

Unit

s.

Measurement Point

In a granularity period, the BSC scans uplink GPRS TBFs per second. If there are uplink GPRS TBFs in the data transmission state, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
