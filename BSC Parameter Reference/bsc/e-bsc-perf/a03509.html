Successful VGCS TCH Handovers for Frequency Shift

Measurement Counter

A03509: CELL_GCC_HO_SUCC_FREQ_SHIFT

Description

This measurement provides the number of successful VGCS TCH handovers for frequency shift.

Unit

Integer number or integer.

Measurement Point

(1) In the case of the intra-BSC intra-cell handover, this counter is incremented by one when the originating BSC receives a HANDOVER SUCCEEDED message from the MSC. The measurement point is B, as shown in Figure 1.

(2) In the case of the intra-BSC inter-cell handover, this counter is incremented by one when the target cell receives a HANDOVER COMPLETE message from the BTS. The measurement point is B, as shown in Figure 2.

(3) In the case of the inter-BSC handover, this counter is incremented by one when the originating BSC receives a HANDOVER SUCCEEDED message from the MSC. The measurement point is B, as shown in Figure 3.

Figure 1 Group call intra-cell handover procedure

Figure 2 Group call inter-cell handover procedure

Figure 3 Group call outgoing inter-BSC handover procedure

Formula

This is an original counter without involving any formula.
