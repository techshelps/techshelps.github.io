REJECT Responses to WRITE-REPLACE Requests

Measurement Counter

L0464:CB_BSC_REJ_TX_MSG

Description

This counter is used to measure the number of REJECT responses to WRITE-REPLACE requests from the BSC to the CBC.

After receiving a WRITE-REPLACE request on the BSC-CBC interface, the BSC sends a REPORT response message or a REJECT response message to the CBC.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC sends a REJECT response message to the CBC.

Formula

This is an original counter without involving any formula.
