PS MS Radio Access Capability

Description

This measurement unit provides the number of times that the MSs with radio access capability process the PS services in a cell. The MS capability can be used to calculate the MS radio access features, guide the network planning, determine whether some optimization functions need to be enabled or the network capacity needs to be expanded. The MS radio access features are as follows:

  • Supporting GPRS extended dynamic allocation
  • Supporting EGPRS extended dynamic allocation
  • Supporting modulation based multislot class
  • Supporting multiple TBF procedures in A/Gb mode
  • Supporting Downlink Advanced Receiver Performance
  • Supporting extended RLC/MAC control message segmentation
  • Supporting PS handover
  • Supporting Single Slot DTM
  • Supporting multislot class 5 in DTM GPRS
  • Supporting multislot class 6 in DTM GPRS
  • Supporting multislot class 9 in DTM GPRS
  • Supporting multislot class 10 in DTM GPRS
  • Supporting multislot class 11 in DTM GPRS
  • Supporting multislot class 5 in DTM EGPRS
  • Supporting multislot class 6 in DTM EGPRS
  • Supporting multislot class 9 in DTM EGPRS
  • Supporting multislot class 10 in DTM EGPRS
  • Supporting multislot class 11 in DTM EGPRS
  • Supporting multislot class 30
  • Supporting multislot class 45
  • Supporting multislot class 31 or 36 in DTM GPRS
  • Supporting multislot class 32 or 37 in DTM GPRS
  • Supporting multislot class 33 or 38 in DTM GPRS
  • Supporting multislot class 41 in DTM GPRS
  • Supporting multislot class 42 in DTM GPRS
  • Supporting multislot class 43 in DTM GPRS
  • Supporting multislot class 44 in DTM GPRS
  • Supporting multislot class 31 or 36 in DTM EGPRS
  • Supporting multislot class 32 or 37 in DTM EGPRS
  • Supporting multislot class 33 or 38 in DTM EGPRS
  • Supporting multislot class 41 in DTM EGPRS
  • Supporting multislot class 42 in DTM EGPRS
  • Supporting multislot class 43 in DTM EGPRS
  • Supporting multislot class 44 in DTM EGPRS
