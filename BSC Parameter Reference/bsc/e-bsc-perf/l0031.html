Number of Data Blocks Retransmitted over SCTP Link

Measurement Counter

L0031:SCTP_LNK_RETRAN_PACKET_NUM

Description

This counter is measured when the BSC retransmits a data block to the MSC over the SCTP link. This counter is measured at A, as shown in the following figure. The BSC and the MSC send messages to each other over the SCTP link. When the BSC receives a SACK message from the MSC, indicating that a data packet is lost. In this case, the BSC retransmits the previous data block and measures this counter.

This measurement provides the number of data blocks to be retransmitted.

Unit

Integer number or integer.
