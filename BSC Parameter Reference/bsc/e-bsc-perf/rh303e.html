External Outgoing Radio Handover Success Ratio per cell

Measurement Counter

RH303E: CELL_INTERBSC_OUT_CELL_RD_HO_SUCC_RATE

Description

The measurement provides the success rate of external outgoing cell handovers on the Um interface.

Unit

%.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

External Outgoing Radio Handover Success Ratio per cell =

[Successful External Outgoing Cell Handovers] x {100}/

[Outgoing External Inter-Cell Handover Commands]
