Channel Requests (Call Type)

Measurement Counter

CA300A: CELL_IMM_ASS_REQ_CA

Description

When an MS requires network services for reasons such as MOC, MTC, Call Re-Establishment, and Emergency Call, it sends the BTS a CHAN REQ message on the RACH. After receiving this message, the BTS reports a CHAN RQD message to the BSC for a dedicated channel. This measurement provides the number of CHAN RQD messages initiated for calls.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, each time the BSC receives a CHAN RQD message from the BTS and the access reason is MOC, MTC, Call Re-Establishment, or Emergency Call (see measurement point A), the counter is incremented by one.

Figure 1 Immediate assignment procedure

Formula

Channel Requests (Call Type) = Channel Requests (MOC) + Channel Requests (MTC) + Channel Requests (Emergency Call) + Channel Requests (Call Re-Establishment)
