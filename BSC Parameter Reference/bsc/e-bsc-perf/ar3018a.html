Analyzed Measurement of Dynamically Configured Channels (TCHH)

Measurement Counter

AR3018A:CELL_CH_NUM_HR_900_AVR

AR3018B:CELL_CH_NUM_HR_1800_AVR

CR3018:CELL_CH_NUM_HR_AVR

Description

AR3018A:CELL_CH_NUM_HR_900_AVR provides the mean number of the configured SDCCHs in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, and GSM850/PCS1900 cells controlled by the BSC in the granularity period. This number includes the number of the TCHHs that are dynamically converted from TCHFs.

AR3018B:CELL_CH_NUM_HR_1800_AVR provides the mean number of the configured TCHHs in the DCS1800 and PCS1900 cells controlled by the BSC in the granularity period. This number includes the number of the TCHHs that are dynamically converted from TCHFs.

CR3018:CELL_CH_NUM_HR_AVR provides the mean number of configured TCHHs in the cells on all frequency bands controlled by the BSC in real-time measurement. This number includes the number of TCHHs that are converted from dynamic PDCHs.

Unit

Integer number or integer.

Measurement Point

In the granularity period, the counters are calculated as follows.

Formula

Mean Number of Dynamically Configured Channels (TCHH) (900/850 Cell) = Total Number of Dynamically Configured Channels (TCHH) (900/850 Cell)/Number of Sampling Times

Mean Number of Dynamically Configured Channels (TCHH) (1800/1900 Cell) = Total Number of Dynamically Configured Channels (TCHH) (1800/1900 Cell)/Number of Sampling Times

Mean Number of Available Channels (TCHH) = Mean Number of Available Channels (TCHH) (900/850 Cell) + Mean Number of Available Channels (TCHH) (1800/1900 Cell)
