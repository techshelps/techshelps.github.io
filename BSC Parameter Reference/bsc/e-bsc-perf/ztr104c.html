Call Drop Rate on SDCCH

Measurement Counter

ZTR104C: CELL_SD_CALL_DROP

Description

This measurement provides the call drop rate on the SDCCH, that is, the percentage of call drops on the SDCCH to successful SDCCH seizures.

Unit

%

Measurement Point

None.

Formula

Call Drop Rate on SDCCH = [Call Drops on SDCCH] x {100}/[Successful SDCCH Seizures]
