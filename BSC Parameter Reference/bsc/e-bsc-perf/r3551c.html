Busy Number of AMR Service Channel (TCHF)

Measurement Counter

R3551C: CELL_CHAN_BUSY_NUM_TCHF_AMR_TRAF

Description

This measurement provides the average number of busy AMR service channels (TCHF) in the granularity period.

Unit

Integer number or integer.

Measurement Point

The BSC takes the samples of the channel states every five seconds, and the number of the busy AMR service channels (TCHF) of the cell is incremented when the timer expires.

Formula

This is an original counter without involving any formula.
