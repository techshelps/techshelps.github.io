Number of Calls Originated or Terminated by MSs Supporting Full-Rate AMR per BSC

Measurement Counter

ZTCA3352G: BSCRPT_SPT_FAMR_MS_NUM

Description

This measurement provides the number of calls originated or terminated by MSs that support full-rate AMR in all cells under the BSC.

Unit

Integer number or integer.

Measurement Point

When the MS sends a SETUP or CALL CONFIRM message, this measurement is triggered if the Bearer Capability information element contains the full-rate AMR.

Formula

Number of Calls Originated or Terminated by MSs Supporting Full-Rate AMR per BSC = Number of Calls Originated or Terminated by MSs Supporting Full-Rate AMR

(the sum of the number of calls originated or terminated by MSs supporting Full-Rate AMR in all cells under the BSC)
