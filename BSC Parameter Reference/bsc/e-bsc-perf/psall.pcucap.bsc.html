PS Capability Measurement

Description

PS Capability Measurement is used to analyze the PS processing capability of the BSC.

This measurement unit provides the measurements on the number of seized PDCHs and number of available PDCHs of the BSC. The seized PDCH indicates that the PDCH is seized and carried with TBF. The available PDCH indicates that the PDCH is requested, the channel status is normal and can be used.

Measurement Object

BSC.
