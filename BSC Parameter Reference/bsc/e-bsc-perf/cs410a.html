Description

The counters provide the number of MRs with the average uplink receive quality rank and downlink receive quality rank from 0 to 7 per granularity period.

Unit

Integer number or integer.

Measurement Point

In the granularity period, the counters are calculated as follows.

Formula

The N in the following formulas ranges from 1 to 7.

Number of MRs on Uplink TCHF (Mean Receive Quality Rank N) =

[Number of MRs on Uplink TCHF (Receive Level Rank 0 and Receive Quality Rank N)] +

[Number of MRs on Uplink TCHF (Receive Level Rank 1 and Receive Quality Rank N)] +
