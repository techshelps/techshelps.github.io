Paging Requests on the Abis Interface per BSC (CS Service)

Measurement Counter

ZTA330:BSCRPT_PAGING_CS_ABIS

Description

This measurement provides the number of paging requests on the Abis interface per BSC (CS service), that is, the total of delivered paging requests on the Abis interface for the CS service in all cells.

Unit

Integer number or integer.

Measurement Point

BSC counters are calculation counters. This counter is measured according to the formula below when the measurement period ends. For details on the measurement point, see "Delivered Paging Messages for CS Service".

Formula

Paging Requests on the Abis Interface per BSC (CS Service) =Delivered Paging Messages for CS Service
