TCH Seizure Requests in TCH Handovers (Traffic Channel)

Measurement Counter

K3010B:CELL_KPI_TCH_HO_REQ_TRAF

Description

In an intra-BSC incoming handover procedure, the MS requests the BSS to reallocate a TCH (traffic channel). In an inter-BSC incoming handover procedure, the MSC requests the BSS to reallocate a TCH (traffic channel).

This measurement provides the total number of TCH seizure requests in the procedures mentioned previously.

Unit

Integer number or integer.

Measurement Point

(1) Incoming internal inter-cell handover procedure: According to the measurement report received from the MS, the BSC decides to initiate the incoming internal inter-cell handover. When receiving the incoming internal inter-cell handover request, the BSC measures the counter. The measurement object is the target cell. See measurement point D in Figure 1and Figure 2.

Formula

TCH Seizure Requests in TCH Handovers (Traffic Channel) =

[Internal Intra-Cell Handover Requests (TCH)] +

[Incoming Internal Inter-Cell Handover Requests (TCH) (900/850-900/850)] +
