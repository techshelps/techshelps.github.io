Number of Circuit Paging Requests Sent on PPCH

Measurement Counter

A9605: PPCH_CS_PAGING_REQ

Description

If Network Operation Mode I is configured on the BSC side, and the Gs interface and PCCCH are configured, then the CS PAGING REQUEST message can be transmitted on the PPCH, provided that the MS to be paged is not in transmission state. This measurement provides the number of circuit paging requests sent by the BSC on the PPCH in a granularity period.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure for the BSC sending a CS paging request on the PPCH. Each time the BSC sends a CS PAGING REQUEST message on the PPCH (see measurement point A), the counter is incremented by one.

Figure 1 CS paging request sent on the PPCH

Formula

This is an original counter without involving any formula.
