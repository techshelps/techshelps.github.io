VBS Assignment Queue Request

Measurement Counter

A03434: CELL_VBS_ASS_QUEUE_REQ

Description

After receiving the VBS ASSIGNMENT REQUEST message from the MSC, the BSC sends a VBS QUEUING INDICATION message to the MSC if no radio resource is available and the VBS assignment procedure queues for idle resources. This measurement provides the number of queuing requests.

Unit

Integer number or integer.

Measurement Point

If no radio resource is available and VGCS assignment procedure queues for idle resources, the BSC sends a VBS QUEUING INDICATION message to the MSC. This counter is incremented by one when a VBS QUEUING INDICATION message is sent. In the signaling procedure, this counter is measured at point B in Figure 1.

Formula

This is an original counter without involving any formula.
