CRC Serious Error Measurement per BTS

Description

CRC Serious Error Measurement per BTS is used to measure the duration of CRC serious error on E1 ports on the Abis interface. This measurement unit indicates the communication status of E1 cables between the BSC and the BTS.

Measurement Object

SITE
