Successful VBS Assignment

Measurement Counter

A03424: CELL_VBS_ASS_SUCC

Description

If the assignment is successful after the BSC receives a VBS ASSIGNMENT REQUEST message from the MSC, the BSC sends a VBS ASSIGNMENT RESULT message to the MSC. This measurement provides the number of VBS ASSIGNMENT RESULT messages that the BSC sends to the MSC.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC sends a VBS ASSIGNMENT RESULT message to the MSC. In the signaling procedure, this counter is measured at Point B in Figure 1, point B in Figure 2, and point D in Figure 3.

Formula

This is an original counter without involving any formula.
