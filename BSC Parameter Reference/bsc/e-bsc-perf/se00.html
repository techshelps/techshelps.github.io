Normal IP Clock Packets

Measurement Counter

SE00: IPCLOCK_NORMAL_PACK

Description

This measurement provides the number of normal clock packets received during the handshake process between the BTS and the IP clock server.

Unit

Integer number or integer.

Measurement Point

The IP clock client performs the measurement after receives the normal packets. The BTS periodically reports the result, and the BTSM disassembles the packets and processes the measurement results.

Formula

This is an original counter without involving any formula.
