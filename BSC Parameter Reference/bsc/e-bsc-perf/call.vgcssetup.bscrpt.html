VGCS Setup Measurement per BSC

Description

VGCS Setup Measurement per BSC is used to measure the number of VGCS setup requests received by the BSC and the number of times that the VGCS call is set up successfully.

Measurement Object

BSCRPT.

Measurement Counter

The following table lists the measurement counters related to VGCS Setup Measurement per BSC.

Counter

Refer to Section...

A03401:BSC_VGCS_SETUP_REQ

VGCS Setup Requests per BSC

A03402:BSC_VGCS_SETUP_SUCC

Successful VGCS Setups per BSC
