HDLC Measurement of Transmission Interface Board

Description

For the 3012 series base station, the PTU communicates with the TMU through the backplane. The PTU is also connected to the BSC. The IP-based base station provides the HDLC paths between the TMU and the PTU, whereas the Hub-BTS provides not only the HDLC paths between the TMU and the PTU but also the HDLC paths between the PTU and the PEU. For the NGBTS, the HDLC base station provides the HDLC paths between the PTU and the PEU, and the Hub-BTS provides the not only the HDLC paths between the PTU and the PEU but also the HDLC paths between the PTUs. The PTU works as an aggregating node, and it takes statistics on the packets over the paths by service type. This helps get an overview of the transmission model and transmission throughput, paving the way for the optimization of transmission model and throughput.

For the IP or Hub-based 3012 series base station, the PTU reports these statistics information to the BSC for processing. For the HDLC or Hub-based NGBTS series base station, the transmission interface board reports these statistics information to the BSC for processing.

Measurement Object

For the 3012 series base stations, the measurement objects are the HDLC paths between the TMU and the PTU, and the HDLC paths between PTU and the PEU.

For the NG series base stations, the measurement objects are the HDLC paths between the PTU and the PEU, and the HDLC paths between PTUs.
