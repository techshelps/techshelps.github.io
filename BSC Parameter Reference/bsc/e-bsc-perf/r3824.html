Number of Loopback Start Failures (Territorial Resource Request Failure)

Measurement Counter

R3824:CELL_LOCALSWITCH_LOOP_FAIL_APPLY_TRSL_FAIL

Description

After the call is put through or the incoming BSC handover for the speech service is complete, the call processing module initiating loopback start at one end sends a loopback request message to the call processing module at the peer end during the BSC local loopback. The call processing module at the peer end sends a network arrangement request message to the BSC physical layer and requests for Abis resources. If the TC resource request fails, the connection of the transmission channel between the Ater interface and the Abis interface fails, or the Abis resource request fails, the BSC physical layer responds to the call processing module with a network arrangement acknowledgement message. Then, the call processing module at the peer end sends an acknowledgement message to the call processing module at the TX end with the cause value of territorial resource request failure.

This measurement provides the number of loopback start failures due to territorial resource request failure.

Unit

Integer number or integer.

Measurement Point

After the BSC receives the CONNECT ACK message from the MSC in the MS terminated call procedure, as shown in Figure 1(A1), or after the BSC receives the HANDOVER CMP message from the MS in the incoming BSC handover procedure, as shown in Figure 2 (B1), the BSC selects the BSS local switch policy, determines whether the conditions for initiating the local switch are met, and performs local switch call detection. If all the local switch conditions are met, the BSC local loopback start procedure is started. Within the BSC, the call processing module initiating loopback start at one end sends a loopback request message to the call processing module at the peer end. The call processing module at the peer end sends a network arrangement request message to the BSC physical layer and requests for Abis resources. If the request fails due to various reasons such as TC resource request failure and failed connection of the transmission channel between Ater interface and Abis interface, the BSC physical layer responds to the call processing module with a network arrangement acknowledgement (territorial resource request failure) message. Then, the call processing module at the peer end sends an acknowledgement message to the call processing module at the TX end with the cause value of territorial resource request failure. In this case, this counter is measured in the cell where the loopback start is initiated.
