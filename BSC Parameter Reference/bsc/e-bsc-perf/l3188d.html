PACKET CCCH LOAD IND Messages Sent on Abis Interface

Measurement Counter

L3188D:CELL_OVERLOAD_PS

Description

The BTS stores the paging messages from the BSC and the PCU on the downlink CCCH (PCH) to different receiving buffer queues. When the length of the receiving buffer queue exceeds the threshold, the downlink CCCH is considered to be overloaded. When the downlink CCCH in the cell is overloaded, the BTS reports the overload information to the BSC and identifies the cause for the overload. If the overload is due to heavy CS services, the BTS reports a CCCH LOAD IND message to the BSC. If the overload is due to heavy PS services, the BTS reports a PACKET CCCH LOAD IND message to the BSC and then the BSC forwards the message to the PCU.

This measurement provides the number of PACKET CCCH LOAD IND messages that the BSC receives from the BTS controlling the cell.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC receives a PACKET CCCH LOAD IND message from the BTS.

Formula

This is an original counter without involving any formula.
