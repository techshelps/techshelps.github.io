DSP CPU Measurement

Description

This measurement unit provides the performance of the DSP CPU for analysis.

The counter indicates the DSP CPU usage. If the average DSP CPU usage exceeds the normal range, the stability of the BSC system may be affected.

Measurement Object

DSP.

Measurement Counter

The following table lists the measurement counters related to DSP CPU Measurement.
