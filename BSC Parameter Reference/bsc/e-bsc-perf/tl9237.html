Throughput of Uplink EGPRS RLC Data Blocks

Measurement Counter

TL9237: CELL_RLC_UP_RLC_EGPRS_BLK_THROUGHPUT

Description

This measurement provides the average throughput at the RLC of the uplink EGPRS TBF in a cell in a granularity period. This measurement indicates the data flow during the granularity period. If the measurement value is high, you can indicate that this granularity period is a peak hour of the PS services.

Unit

kbyte.

Measurement Point

In a granularity period, calculate the numbers of uplink EGPRS RLC data blocks with different coding schemes and convert these numbers to corresponding bytes. The total number of bytes is taken as Throughput of Uplink EGPRS RLC Data Blocks.

Formula

Throughput of Uplink EGPRS RLC Data Blocks = (Total Number of Uplink EGPRS MCS1 RLC Data Blocks x 22 + Total Number of Uplink EGPRS MCS2 RLC Data Blocks x 28 + Total Number of Uplink EGPRS MCS3 RLC Data Blocks x 37 + Total Number of Uplink EGPRS MCS4 RLC Data Blocks x 44 + Total Number of Uplink EGPRS MCS5 RLC Data Blocks x 56 + Total Number of Uplink EGPRS MCS6 RLC Data Blocks x 74 + Total Number of Uplink EGPRS MCS7 RLC Data Blocks x 56 + Total Number of Uplink EGPRS MCS8 RLC Data Blocks x 68 + Total Number of Uplink EGPRS MCS9 RLC Data Blocks x 74)/1024
