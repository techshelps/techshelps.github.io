Retransmission Rate of RLC Uplink Data Blocks Using CS1-CS4 (%)

Measurement Counter

RL9010: UP_CS1_RLC_BLK_RESEND_RATE

RL9011: UP_CS2_RLC_BLK_RESEND_RATE

RL9012: UP_CS3_RLC_BLK_RESEND_RATE

RL9013: UP_CS4_RLC_BLK_RESEND_RATE

Description

This measurement provides the retransmission rates of uplink RLC data blocks in CS1-CS4 coding schemes in a cell in a granularity period. If the retransmission rate is high, you can infer that the signal quality is poor, thus the network needs to be optimized.

Unit

%.

Measurement Point

None.

Formula

Retransmission Rate of RLC Uplink Data Blocks Using CSx (%) = (Total Number of Uplink RLC Data Blocks Using CSx - Total Number of Valid Uplink RLC Data Blocks Using CSx)/Total Number of Uplink RLC Data Blocks Using CS1) x 100 Note: CSx indicated CS1-CS4.
