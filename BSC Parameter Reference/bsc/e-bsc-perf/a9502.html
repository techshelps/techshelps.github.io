Number of Successful Uplink Assignments

Measurement Counter

A9502: SUCC_UPLINK_ASS

A9504: SUCC_UPLINK_IMM_ASS

A9506: SUCC_PACCH_UPLINK_ASS

Description

These three measurements provide the number of successful uplink assignments in a granularity period. Successful assignment means that the BSC receives the uplink data block or single block. This measurement indicates the GPRS QoS of a cell. A low measurement value indicates that the radio environment, network parameter configuration, network resource configuration, or network equipment is not satisfactory, and thus requires optimization.

Unit

Integer number or integer.

Measurement Point

The successful uplink assignment involves the following aspects:

  • Uplink data block received

    If the BSC receives the uplink data block from the MS in response to the packet immediate assignment message or uplink packet assignment message, it indicates that the MS has successfully received the uplink assignment message and the uplink assignment is completed successfully. Figure 1 shows the uplink assignment procedure initiated by the BSC. Each time the BSC receives the first uplink data block from the MS after sending an uplink assignment message (see measurement point A), the counter is incremented by one.

    Figure 1 Uplink data block received

  • Packet resource request or measurement report received

    If the BSC receives the PACKET RESOURCE REQUEST message or measurement report form the MS after sending the immediate assignment message, it indicates that the MS has successfully received the single-block assignment message and the uplink assignment is completed successfully. Figure 2 shows the single-block immediate assignment procedure initiated by the BSC. Each time the BSC receives a PACKET RESOURCE REQUEST message (see measurement point A), the counter is incremented by one.

    Figure 2 Packet resource request or measurement report received

Formula

This is an original counter without involving any formula.
