Mode Modify Commands

Measurement Counter

CA314: CELL_CHMDI_CMD

Description

This counter provides the number of CHAN MODE MODIFY messages sent by the BSC to a cell based on the Measurement of Mode Modify Commands.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, each time the BSC sends a CHAN MODE MODIFY message to the MS (see measurement point E1), the counter is incremented by one.

Figure 1 Mode Modify procedure

Formula

Mode Modify Commands =

[Mode Modify Commands (MOC) (TCHF)] +

[Mode Modify Commands (MTC) (TCHF)] +

Mode Modify Commands (Emergency Call) (TCHF)] +

[Mode Modify Commands (Call Re-establishment) (TCHF)] +

[Mode Modify Commands (MOC) (TCHH)] +

[Mode Modify Commands (MTC) (TCHH)] +

[Mode Modify Commands (Emergency Call) (TCHH)] +

[Mode Modify Commands (Call Re-establishment) (TCHH)]
