Channel Activation Timeout During VBS Assignment (TIMEOUT)

Measurement Counter

A03432: CELL_VBS_ASS_FAIL_NACK

Description

After receiving the VBS ASSIGNMENT REQUEST message from the MSC, the BSC sends a CHANNEL ACTIVATION message to the BTS for the newly assigned TCH. This measurement provides the number of times that the BSC does not receive the CHANNEL ACTIVATION ACKNOWLEDGE message from the BTS before the timer expires.

Unit

Integer number or integer.

Measurement Point

In the VBS assignment procedure, the BSC sends a CHANNEL ACTIVATION message for the newly assigned TCH to the BTS. This counter is incremented by one when the BSC does not receive a CHANNEL ACTIVATION ACKNOWLEDGE message before the timer expires.
