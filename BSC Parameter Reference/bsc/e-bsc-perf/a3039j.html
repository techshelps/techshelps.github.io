SDCCH Seizures for Speech Service

Measurement Counter

A3039J: CELL_SD_SUCC_SPEECH

Description

This measurement is the sum in two cases.

Case 1: The MS access cause carried in the EST IND message is MOC Non SMS, Emergency Call, or Call Re-establishment, and the channel type carried in the IMM ASS CMD message corresponding to the EST IND message that the BSC receives from the BTS is SDCCH.

Case 2: The MS access cause carried in the EST IND message is MTC or SS; the BSC assigns the TCH upon the assignment request from the MSC and the channel type carried in the IMM ASS CMD message is SDCCH.

Unit

Integer number or integer.

Measurement Point

After receiving the setup indication message from the BTS, the BSC measures the counter of different access causes (MOC Non SMS, Emergency Call, Call Re-establishment) according to the type of the Layer 3 information in the message. This counter is measured at point C in Figure 1. On receiving the assignment request from the MSC, the BSC measures the SDCCH Seizures for Speech Service associated with MTC or SS. See measurement point A in Figure 2.
