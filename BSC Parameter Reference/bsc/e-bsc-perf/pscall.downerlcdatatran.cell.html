Downlink EGPRS RLC Data Transfer Capability Measurement

Description

The PSCALL.DownERlcDataTran.Cell measurement unit provides the counters related to the transfer of the downlink EGPRS RLC data in a granularity period. It indicates the traffic load of the PS services in a cell and is used as a reference for establishing the PS service model.

Measurement Object

Cell.
