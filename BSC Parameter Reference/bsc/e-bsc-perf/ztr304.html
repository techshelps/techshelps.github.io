Call Drop Ratio on TCH per cell (Including Handover)

Measurement Counter

ZTR304: CELL_RATE_TCH_CALL_DROP_INCLUDE_HO

Description

This measurement provides the call drop rate on the TCH, that is, the percentage of call drops on the TCH to successful TCH seizures.

NOTE:

Call Drop Ratio on TCH per cell (Including Handover) focuses on channel, whereas Call Drop Ratio on TCH per cell (Excluding Handover) focuses on user experience.

  • This measurement is triggered on the basis of channel seizure. During a call, the number of handovers is equal to the number of channel seizures. If call drops occur after the last handover, it indicates that the channel seizures involved in the previous handovers are successful.
  • Call Drop Ratio on TCH per cell (Excluding Handover) is the call drop rate during a call. Regardless of how many handovers are performed during a call, handovers are considered to be failed if call drops occur.

Unit

%

Measurement Point

None.

Formula

Call Drop Ratio on TCH per cell (Including Handover) = [Call Drops on Traffic Channel] x {100}/([Successful TCH Seizures (Signaling Channel)] + [Successful TCH Seizures (Traffic Channel)] + [Successful TCH Seizures in TCH Handovers (Traffic Channel)])
