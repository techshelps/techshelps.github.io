Channel Assignment Failure Measurement per Cell

Description

The Channel Assignment Failure Measurement per Cell measurement unit is used to measure the failures when the BSC assigns SDCCHs, TCHFs, or TCHHs in the following procedures:

  • Immediate assignment

  • Assignment

  • Internal intra-cell handover

  • Incoming internal inter-cell handover

  • Incoming external inter-cell handover

Measurement Object

CELL.
