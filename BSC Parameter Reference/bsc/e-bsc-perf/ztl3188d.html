PCH Overloads due to PS Service Counted through the Indications from the Abis Interface per BSC

Measurement Counter

ZTL3188D:BSCRPT_OVERLOAD_PACKET_SERV_ABIS

Description

The measurement of this counter provides the total number of PCH overloads (PS services) on the Abis interface in all the cells controlled by a BSC.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "PACKET CCCH LOAD IND Messages Sent on Abis Interface".

Formula

Immediate Assignment Requests per BSC (PS Service)= PACKET CCCH LOAD IND Messages Sent on Abis Interface
