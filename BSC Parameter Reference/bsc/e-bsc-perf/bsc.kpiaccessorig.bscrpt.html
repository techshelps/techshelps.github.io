Channel Measurement per BSC

Description

This measurement unit is used to measure the number of successful assignments of the TCHF, TCHH, and SDCCH and the number of failed assignments of the TCHF, TCHH, and SDCCH in a BSC.

Measurement Object

BSCRPT.
