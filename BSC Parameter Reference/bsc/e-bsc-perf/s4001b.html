Received PS Bytes

Measurement Counter

S4001B: TRX_TX_PS_TRAFFIC

Description

This measurement provides the number of bytes in the PS packets that the TRX receives from the BSC over the HDLC path when transmission optimization is implemented in the BTS.

Unit

byte.

Measurement Point

The TRX performs the measurement after receives the PS packets. The BTS periodically reports the results, and the BTSM disassembles the packets and processes the measurement results.

Formula

This is an original counter without involving any formula.
