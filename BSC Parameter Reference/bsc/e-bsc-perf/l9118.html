Number of Downlink GPRS RLC Control Blocks

Measurement Counter

L9118: DOWN_RLC_SIG_BLK

Description

This measurement provides the number of downlink GPRS RLC control blocks in a cell in a granularity period.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure in which the MS sends a PACKET DOWNLINK ACK/NACK message to the BSC. Every time the BSC receives a PACKET DOWNLINK ACK/NACK message (see measurement point A), the counter is incremented by one.

Figure 1 RLC data blocks carrying the PACKET DOWNLINK ACK/NACK message on the downlink GPRS TBF

Formula

This is an original counter without involving any formula.
