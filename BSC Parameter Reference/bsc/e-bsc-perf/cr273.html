BTS In-Service Duration

Measurement Counter

CR273:SITE_SERV_NORM

Description

This measurement provides the duration of the BTS in service in a GP.

Unit

s.

Measurement Point

None.

Formula

[BTS In-Service Duration] = {GP} x {60} - [BTS Out-of-Service Duration]

GP unit: minute
