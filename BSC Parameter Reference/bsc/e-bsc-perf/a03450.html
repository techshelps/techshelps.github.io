VGCS TCH Measurements

Measurement Counter

A03450: CELL_VGCS_UL_MR

Description

After the VGCS uplink TCH is activated, the MS that seizes the uplink TCH sends a MEASURE REPORT message to the BSC every 480 ms. This measurement provides the number of MEASURE REPORT messages sent from the MS to the BSC for the VGCS uplink TCH.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC receives a MEASURE REPORT message from the MS. In the signaling procedure, this counter is measured at point A in Figure 1.

Figure 1 VGCS TCH measurement procedure

Formula

This is an original counter without involving any formula.
