Description

BSC determines whether to initiate the handover from the originating cell to the target cell through the handover algorithm. If the originating cell and the target cell are controlled by different BSCs, the external inter-cell handover is initiated. The specific counter provides the number of incoming external inter-cell handover requests based on the target cell under control of the target BSC. Together with the Incoming External Inter-Cell Handover Responses and the Failed Incoming External Inter-Cell Handovers, this counter determines the incoming external inter-cell handover performance.

The following counters provide the numbers of incoming external inter-cell handover requests initiated by other BSCs based on the handover algorithm. The MS may be seizing a the channel type (SDCCH/TCH) before the handover. The handover can be performed between the frequency bands as listed in Table 1. They are measured when the BSC receives the HO REQ message from the MSC.

The specific one of the following counters provides the number of incoming external inter-cell handover requests initiated by other BSCs based on the handover algorithm. When an HO REQ message is received from the MSC, the specific counter is measured based on the Channel Type IE contained in the HO REQ message. The following lists the eight types of Channel Type IE contained in the HO REQ message from the MSC and the corresponding counter to be measured:
  • TCHF only: H3401B:CELL_INTERBSC_INCELL_HO_REQ_TCHF_ONLY
  • TCHH only: H3401C:CELL_INTERBSC_INCELL_HO_REQ_TCHH_ONLY
  • TCHF preferred and channel type unchangeable: H3401D:CELL_INTERBSC_INCELL_HO_REQ_TCHF_PREF_NOCHG
  • TCHH preferred and channel type unchangeable: H3401E:CELL_INTERBSC_INCELL_HO_REQ_TCHH_PREF_NOCHG
  • TCHF preferred and channel type changeable: H3401G:CELL_INTERBSC_INCELL_HO_REQ_TCHF_PREF_CHG
  • TCHH preferred and channel type changeable: H3401H:CELL_INTERBSC_INCELL_HO_REQ_TCHH_PREF_CHG
  • TCHF or TCHH and channel type unchangeable: H3401F:CELL_INTERBSC_INCELL_HO_REQ_TCH_NOCHG
  • TCHF or TCHH and channel type changeable: H3401I:CELL_INTERBSC_INCELL_HO_REQ_TCH_CHG
The following counter provides the number of Incoming External Inter-Cell Handover Requests received from the MSC, which is initiated by other BSCs based on the Better Cell algorithm.
