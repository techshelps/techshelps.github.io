Total Frames Received by the BTS Physical Layer on LAPD Link

Measurement Counter

L5118:LAPD_RX_FRMPHY

Description

The BTS receives the physical frames through layer 1 (L1) and then puts the physical frames in the queue from L1 to L2.

This counter is incremented by 1 when L1 receives an LAPD Link frame.

If the physical uplink is disrupted, the measurement cannot be sent to the BSC and the value of this counter does not increase.

Unit

Integer number or integer.

Measurement Point

This counter is measured when BTS L1 receives an LAPD Link frame.

Formula

This is an original counter without involving any formula.
