Immediate Assignment Requests per BSC

Measurement Counter

ZCA300J:BSCRPT_KPI_IMM_ASS_REQ

Description

To obtain the service from the network, an MS sends a CHAN REQ message on the RACH. After receiving this message, the BTS sends a CHAN RQD message to the BSC for a dedicated channel. This measurement provides the access over channels due to other reasons except packet services in all cells controlled by a BSC.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the formula below when the measurement period ends. For details on the measurement point, see "Channel Requests per Cell".

Formula

Immediate Assignment Requests per BSC =

[Channel Requests (MOC) per BSC] +

[Channel Requests (MTC) per BSC] +

[Channel Requests (Emergency Call) per BSC] +

[Channel Requests (Call Re-establishment) per BSC] +

[Channel Requests (Location Updating) per BSC]
