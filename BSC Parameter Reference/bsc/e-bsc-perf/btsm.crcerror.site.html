CRC Error Measurement per BTSCRC Error Measurement per BTS

Description

CRC Error Measurement per BTS is used to measure the duration of CRC error on E1 ports on the Abis interface. This measurement unit reflects the communication status of E1 cables between the BSC and the BTS.

Measurement Object

SITE
