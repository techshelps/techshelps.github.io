Total Number of Downlink EGPRS MCS1-MCS6 Data Blocks

Measurement Counter

L9302: OWN_EGPRS_MCS1_RLC_DATA_BLK

L9303: OWN_EGPRS_MCS2_RLC_DATA_BLK

L9304: OWN_EGPRS_MCS3_RLC_DATA_BLK

L9305: OWN_EGPRS_MCS4_RLC_DATA_BLK

L9306: OWN_EGPRS_MCS5_RLC_DATA_BLK

L9307: OWN_EGPRS_MCS6_RLC_DATA_BLK

Description

This measurement provides the total number of downlink EGPRS RLC data blocks using MCS-1 to MCS-6 coding schemes in a cell in a granularity period.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure in which the BSC sends a downlink EGPRS RLC data block to the MS. Every time the BSC sends a downlink EGPRS RLC data block using MCS-1 coding scheme (see measurement point A), the counter is incremented by one.

Figure 1 Transmission of downlink EGPRS RLC data blocks using MCS-1 coding scheme

The measurement point of the number of downlink EGPRS RLC data blocks using MCS-2 to MCS-6 coding schemes is the same as that of the number of downlink EGPRS RLC data blocks using MCS-1 coding scheme. The measurement is triggered after the data block using the corresponding coding scheme is sent.

Formula

This is an original counter without involving any formula.
