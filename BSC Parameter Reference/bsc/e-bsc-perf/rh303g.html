Dual-Band Handover Success Ratio per Cell (1800/1900-900/850)

Measurement Counter

RH303G: CELL_DOUBLE_FH_HO_SUCC_RATE_1800_900

Description

This measurement provides the success ratio of the handover from the 1800M/1900M cell to the 900M/850M cell, including the intra-BSC handovers, the incoming BSC handovers, and the outgoing BSC handovers.

Unit

%.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

Dual-Band Handover Success Ratio per cell(1800/1900-900/850)=

([Incoming Internal Inter-Cell Handover Requests (1800/1900-900/850)]+

[Incoming External Inter-Cell Handover Requests (1800/1900-900/850)]+

[Outgoing External Inter-Cell Handover Requests (1800/1900-900/850)]-

([Failed Incoming Internal Inter-Cell Handovers (1800/1900-900/850)]+

[Failed Incoming External Inter-Cell Handovers (1800/1900-900/850)]+

[Failed Outgoing External Inter-Cell Handovers (1800/1900-900/850)]))*{100}

/

([Incoming Internal Inter-Cell Handover Requests (1800/1900-900/850)]+

[Incoming External Inter-Cell Handover Requests (1800/1900-900/850)]+

[Outgoing External Inter-Cell Handover Requests (1800/1900-900/850)])
