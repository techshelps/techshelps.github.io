Failed Outgoing External Inter-Cell Handovers (Handover Request Rejected)

Measurement Counter

CH332K:CELL_INTERBSC_OUTCELL_HO_FAIL_REQ_REJ

Description

This measurement provides the number of failed outgoing external inter-cell handovers because the BSC receives the HO RQD REJ message from the MSC. The cause carried in the HO RQD REJ message can be:
  • OM intervention
  • Equipment failure
  • No radio resource available
  • Requested terrestrial resource unavailable
  • BSS not equipped
  • Invalid cell
  • Requested transcoding/rate adaption unavailable
  • Circuit pool mismatch
  • Requested speech version unavailable
  • Ciphering algorithm not supported
  • Terrestrial circuit already allocated
  • Invalid message
  • Protocol error between BSS and MSC
  • Other causes

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1 and Figure 2, the counter is measured at A when the BSC receives the HO RQD REJ message from the MSC.
