MEAN_BEP in EMR Measurement per TRX

Description

The measurement unit provides the MEAN_BEP contained in the EMR. Higher MEAN_BEP suggests lower BER.

MEAN_BEP refers to the mean bit error probability of the received signals. MEAN_BEP is obtained by the following way:
  1. The MS calculates the MEAN_BEPblock of the correctly received and decoded radio block every four bursts.
  2. Obtain the MEAN_BEP by averaging the MEAN_BEPblocks in the whole granularity period.
Then the logarithm of the MEAN_BEP to the base 10 is calculated and mapped onto a certain rank (0-32). The rank is reported. Table 1 lists the mapping between the MEAN BEP value and the BER range.

The EMR can be reported only when the EMR function is supported by all of the MS, BTS, and BSC. In the BSC, you can set whether to report the EMR at the following path: Cell Attributes -> Call Control -> UTRAN System Message -> Report Type.
