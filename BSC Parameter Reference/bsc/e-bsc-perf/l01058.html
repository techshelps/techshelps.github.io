Number of Bytes Discarded on a Logical Port

Measurement Counter

L01058:BSC_LGCPORT_TXDROPBYTES

Description

This measurement provides the total number of bytes discarded on a logical port on an IP-based interface board.

Unit

byte.

Measurement Point

The counter is measured after the packets that join the queue are discarded on a logical port because of congestion.

Formula

This is an original counter without involving any formula.
