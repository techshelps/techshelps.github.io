I Frames Resent on LAPD Link

Measurement Counter

L5007:Q921_RESEND_TX

Description

This measurement provides the number of I frames resent on LAPD links.

Unit

Integer number or integer.

Measurement Point

The measurement point is on the BSC side, as shown in the figure.

This counter is measured when the BSC resends I frames to the BTS or the PCU.

Formula

This is an original counter without involving any formula.
