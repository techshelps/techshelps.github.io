Throughput of Uplink GPRS RLC Data Blocks

Measurement Counter

TL9023: CELL_RLC_UP_RLC_GPRS_BLK_THROUGHPUT

Description

This measurement provides the average throughput at the RLC of the uplink GPRS TBF in a cell in a granularity period. This measurement indicates the data flow during the granularity period. If the measurement value is high, you can indicate that this granularity period is a peak hour of the PS services.

Unit

kbyte.

Measurement Point

In a granularity period, calculate the numbers of uplink GPRS RLC data blocks with different coding schemes and convert these numbers to corresponding bytes. The total number of bytes is taken as Throughput of Uplink GPRS RLC Data Blocks.

Formula

Throughput of Uplink GPRS RLC Data Blocks = (Total Number of Uplink RLC Data Blocks Using CS1 x 23 + Total Number of Uplink RLC Data Blocks Using CS2 x 34 + Total Number of Uplink RLC Data Blocks Using CS3 x 40 + Total Number of Uplink RLC Data Blocks Using CS4 x 54)/1024
