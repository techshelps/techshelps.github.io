Measurement Reports with Signal Strength Difference Between Neighbor Cell and Serving Cell Greater Than Relative Level Threshold

Measurement Counter

S371:CELL_NCELL_SCELL_LEV_GT_RTHRES_MR_NUM

Description

The MR sent from the MS contains the receive level difference between the serving cell and each neighbor cell. This measurement provides the number of MRs in which the downlink receive level difference between the serving cell and a specified neighbor cell is greater than the relative level threshold.

Unit

Integer number or integer.

Measurement Point

After receiving the valid MR from the MS, the BSC compares the receive level of the serving cell with the receive level of each neighbor cell. If the receive level of a specified neighbor cell is greater than or equal to the sum of the receive level of the serving cell and the relative receive threshold, then the corresponding counter S371:CELL_NCELL_SCELL_LEV_GT_RTHRES_MR_NUM is incremented by one.

Formula

This is an original counter without involving any formula.
