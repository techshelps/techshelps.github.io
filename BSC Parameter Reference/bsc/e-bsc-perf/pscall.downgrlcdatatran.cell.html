Downlink GPRS RLC Data Transfer Capability Measurement

Description

The PSCALL.DownGRlcDataTran.Cell measurement unit provides the counters related to the transfer of the downlink GPRS RLC data in a granularity period. This measurement unit indicates the traffic load of the PS services in a cell and is used as a reference for establishing the PS service model.

Measurement Object

Cell.
