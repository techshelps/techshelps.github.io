Maximum Number of EGPRS Uplink Timeslots

Measurement Counter

S9308: CELL_USER_EGPRS_TBF_UPLINK_MAX_TS_NUM

Description

This measurement provides the maximum number of timeslots that can be allocated to the EGPRS uplink TBFs in transmission state in a granularity period, based on the service type and multislot capability of the MS. This counter is used to calculate Average Maximum Number of Uplink TBF Timeslots.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the BSC periodically samples all the uplink EGPRS TBFs in transmission state. Based on the multislot capability and service type of the MS, the BSC calculates the maximum number of timeslots that can be allocated to the TBFs during each sampling. The calculated value is accumulated to Maximum Number of EGPRS Uplink Timeslots.

Formula

This is an original counter without involving any formula.
