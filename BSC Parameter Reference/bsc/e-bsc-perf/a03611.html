Number of Calls Originated or Terminated by MSs Supporting A5/7

Measurement Counter

A03611: CELL_SPT_A5_7_MS_NUM

Description

This measurement provides the number of calls originated or terminated by MSs that process services in this cell and support A5/7.

Unit

Integer number or integer.

Measurement Point

When the "A5/7" field in classmark 3 is set to "1", this measurement is triggered if any of the following situations occurs: the MS sends the classmark change message; the MSC sends the classmark update message; the incoming BSC handover is complete.

Formula

This is an original counter without involving any formula.
