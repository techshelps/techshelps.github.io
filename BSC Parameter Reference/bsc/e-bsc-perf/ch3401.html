Incoming External Inter-Cell Handover TCH Requests

Measurement Counter

CH3401:CELL_INTERBSC_INCELL_HO_REQ_TCH

Description

BSC determines whether to initiate the handover from the originating cell to the target cell through the handover algorithm. If the originating cell and the target cell are controlled by different BSCs, the external inter-cell handover is initiated.

This measurement provides the number of incoming external inter-cell handover requests in the target cell when the target channel is TCH.

Unit

Integer number or integer.

Measurement Point

At point A, as shown in Figure 1, the counter is measured when the target BSC (BSC2) receives the HO REQ message from the MSC. The measurement object is the target cell.

Figure 1 Measurement of the counters related to incoming external inter-cell handover

A

Measurement point of Incoming External Inter-Cell Handover Requests

B

Measurement point of Incoming External Inter-Cell Handover Responses

C

Measurement point of Incoming External Inter-Cell Handover Detection Messages Received by BSC

D

Measurement point of Successful Incoming External Inter-Cell Handovers

Formula

Incoming External Inter-Cell Handover TCH Requests =

[Incoming External Inter-Cell Handover TCH Requests (TCHF Only)] +

[Incoming External Inter-Cell Handover TCH Requests (TCHH Only)] +

[Incoming External Inter-Cell Handover TCH Requests (TCHF Preferred, Channel Type Unchangeable)] +

[Incoming External Inter-Cell Handover TCH Requests (TCHH Preferred, Channel Type Unchangeable)] +

[Incoming External Inter-Cell Handover TCH Requests (TCHH Preferred, Channel Type Unchangeable)] +

[Incoming External Inter-Cell Handover TCH Requests (TCHH Preferred, Channel Type Changeable)] +

[Incoming External Inter-Cell Handover TCH Requests (TCHF or TCHH, Channel Type Unchangeable)] +

[Incoming External Inter-Cell Handover TCH Requests (TCHF or TCHH, Channel Type Changeable)]
