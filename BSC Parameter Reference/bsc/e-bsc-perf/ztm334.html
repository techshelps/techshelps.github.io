Number of Call Drops Due to BSC Loopback Start Failure per BSC (Territorial Resource Request Failure)

Measurement Counter

ZTM334: BSCRPT_CALL_DROP_BSCLOOP_FAIL_APPLY_TRSL_FAIL

Description

This measurement provides the number of call drops due to the failure in starting the BSC local switching. The cause value is Territorial Resource Request Failure.

Unit

Integer number or integer.

Measurement Point

None.

Formula

Number of Call Drops Due to BSC Loopback Start Failure per BSC (Territorial Resource Request Failure) = Call Drops Due to BSC Loopback Start Failure (Territorial Resource Request Failure)

In other words, the Number of Call Drops Due to BSC Loopback Start Failure per BSC (Territorial Resource Request Failure) is the sum of the Number of Call Drops Due to BSC Loopback Start Failure (Territorial Resource Request Failure) of each cell in the BSC.
