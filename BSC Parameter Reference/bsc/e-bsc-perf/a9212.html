Sampling Times of Concurrent Uplink EGPRS TBFs

Measurement Counter

A9212: UP_EGPRS_TBF_SAMPLE_TIMES

Description

This measurement provides the number of sampled concurrent uplink EGPRS TBFs in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the BSC samples the number of concurrent uplink EGPRS TBFs for several times. Each time the sampling is performed, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
