VBS Setup Requests per BSC

Measurement Counter

A03403:BSC_VBS_SETUP_REQ

Description

During the initiation of the VBS, the MSC sends a VBS SETUP message to the BSC for establishing the VBS in the BSC. This measurement provides the number of VBS requests sent from the MSC to the BSC.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC receives a VBS SETUP message from the MSC. ‏‏ In the signaling procedure, this counter is measured at A in Figure 1.

Figure 1 VBS setup procedure

Formula

This is an original counter without involving any formula.
