Total Number of Measurement Reports for AMR with MS Power Level

Measurement Counter

S3290A:CELL_MR_AMR_MS_POWER_NUM

Description

During an AMR call, this measurement provides the number of measurement reports containing the MS power level in the granularity period.

Unit

Integer number or integer.

Measurement Point

During an AMR call, when the BSC receives a valid MR or pre-processed MR that contains the MS power level from the BTS in the serving cell, this counter is incremented by one.

Formula

This is an original counter without involving any formula.
