Number of Point-to-Point Short Messages on the TCH per BSC

Measurement Counter

ZTCA3349:BSCRPT_PT_TO_PT_SMS_TCH

Description

This measurement provides the number of uplink and downlink short messages on the TCHs per BSC.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Uplink Point-to-Point Short Messages on TCH" and "Downlink Point-to-Point Short Messages on TCH".

Formula

Number of Point-to-Point Short Messages on the TCH per BSC = [Number of Uplink Point-to-Point Short Messages per BSC (TCH)] + [Number of Downlink Point-to-Point Short Messages per BSC (TCH)]
