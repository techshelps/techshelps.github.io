Number of Failed Downlink GPRS TBF Establishments due to No Channel per BSC

Measurement Counter

ZA9103: BSCRPT_BSC_TBF_DOWN_GPRS_TBF_ESTB_FAIL_NO_CHAN_RES

Description

This measurement provides the total number of failed downlink GPRS TBF establishments due to no channel resource in all the cells controlled by the BSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see Number of Failed Downlink GPRS TBF Establishments due to No Channel.

Formula

Number of Failed Downlink GPRS TBF Establishments due to No Channel per BSC = Number of Failed Downlink GPRS TBF Establishments due to No Channel

(That is, the total of the counter Number of Failed Downlink GPRS TBF Establishments due to No Channel in each cell of the BSC.)
