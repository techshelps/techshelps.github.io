Successful Internal Intra-Cell Handovers (AMR) (TCHH-TCHF)

Measurement Counter

CH3035B: CELL_INTRACELL_HO_SUCC_AMR_TCHH_TCHF

Description

This measurement counter indicates the number of successful internal intra-cell handovers for a TCHH-TCHF conversion for calls using half-rate speech version 3. It is equal to the Internal Intra-Cell Handover Requests (AMR) (TCHH-TCHF) for calls using half-rate speech version 3 minus the Failed Internal Intra-Cell Handovers (AMR) (TCHH-TCHF) for calls using half-rate speech version 3.

Unit

Integer number or integer.

Measurement Point

None.

Formula

Successful Internal Intra-Cell Handovers (AMR) (TCHF-TCHH) =

[Internal Intra-Cell Handover Requests (AMR) (TCHH-TCHF)]

[Failed Internal Intra-Cell Handovers (AMR) (TCHH-TCHF)]
