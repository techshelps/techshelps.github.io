Total Allocated Seizure Duration of All Available PDCHs (ms)

Measurement Counter

R9324: SAMPLE_NUM_ALL_AVAIL_PDCH_ALLOC_TIME

Description

This measurement provides the total time of all the available PDCHs being allocated in a cell in a granularity period.

Unit

ms.

Measurement Point

In a granularity period, all the PDCHs available in the cell are sampled every five seconds and the BSC checks whether all the PDCHs are seized. The counter is incremented by 5000 ms if all the PDCHs are seized. The counter is not incremented if there are idle PDCHs or no PDCH is available in the cell.

Formula

This is an original counter without involving any formula.
