IPPM Link Performance Measurement

Description

The measurement unit IPPM Link Performance Measurement is used to measure the packet loss, delay, jitter, number of received/sent packets, and rate of packets received/sent on the transport network between the BSC and the BTS.

NOTE:

The delay variation specifies the difference in end-to-end delay between selected packets in a stream.

Measurement Object

IPPM link.
