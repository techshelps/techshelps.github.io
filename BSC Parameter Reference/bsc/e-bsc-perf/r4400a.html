Measurement of Attempted Immediate Assignments

Measurement Counter

R4400A: TRX_IMM_ASS_CMD_SD

R4409A: TRX_IMM_ASS_CMD_TCH

Description

In the immediate assignment (SDCCH) procedure, the BSC receives a CHANNEL ACTIVATION ACKNOWLEDGE message indicating that the SDCCH is successfully activated from the BTS. The BSC sends an IMM ASS CMD message to the MS to instruct the MS to access the assigned SDCCH and at the same time the counter R4400A "Attempted Immediate Assignments (SDCCH)" is incremented by one for the TRX where the SDCCH resides.

In the immediate assignment (TCH) procedure, the BSC receives a CHANNEL ACTIVATION ACKNOWLEDGE message indicating that the TCH is successfully activated from the BTS. The BSC sends an IMM ASS CMD message to the MS to instruct the MS to access the assigned TCH and at the same time the counter R4409A "Attempted Immediate Assignments (TCH)" is incremented by one for the TRX where the TCH resides.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, each time the BSC sends an IMM ASS CMD message to an MS in the measured cell (see measurement point B), the counter R4400A or R4409A is incremented by one based on the channel type.
