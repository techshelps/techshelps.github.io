Average Number of PDCHs Occupied on DSP

Measurement Counter

AR9509: DSP_PDCH_AVG_OCCU_PDCH_NUM

Description

This measurement provides the average number of PDCHs occupied on a DSP in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the value of the counter equals Total Number of Sampled PDCHs Occupied on DSP divided by Sampling Times of PDCH Measurement on DSP.

Formula

Average Number of PDCHs Occupied on DSP = [Total Number of Sampled PDCHs Occupied on DSP]/[Sampling Times of PDCH Measurement on DSP]
