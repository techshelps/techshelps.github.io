Number of Call Drops Due to Loopback Start Failure per BSC (Timer Expired)

Measurement Counter

ZTM339: BSCRPT_CALL_DROPS_LOOP_FAIL_EXP

Description

Introduction

This measurement provides the number of call drops due to timer expiry during starting the BSS local switching. No loopback is performed.

Unit

Integer number or integer.

Measurement Point

None

Formula

Number of Call Drops Due to Loopback Start Failure per BSC (Timer Expired) = Call Drops Due to Loopback Start Failure (Timer Expired)

In other words, the Number of Call Drops Due to Loopback Start Failure per BSC (Timer Expired) is the sum of the Number of Call Drops Due to Loopback Start Failure (Timer Expired) of each cell in the BSC.
