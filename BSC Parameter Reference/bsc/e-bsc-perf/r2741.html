FlexAbis TSs

Measurement Counter

R2741: SITE_FLEX_TS_NUM

Description

This counter is used to measure the number of available FlexAbis timeslots of the site.

Unit

Integer number or integer.

Measurement Point

The BSC regularly checks the number of Flex timeslots and measures the total number of available Flex timeslots.

Formula

This is an original counter without involving any formula.
