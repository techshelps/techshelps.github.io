Incoming Internal Inter-Cell VGCS TCH Handovers

Measurement Counter

A03464: CELL_GCC_INTRABSC_INTOCELL_HO_REQ_ACK

Description

In the nternal inter-cell VGCS TCH handover, after the TCH is activated, the target cell sends an HO REQ ACK message to notify the originating cell that the TCH is ready for handover. The HO REQ ACK message contains the HANDOVER COMMAND message to be sent to the MS. This measurement provides the number of HO REQ ACK messages sent by the target cell.

Unit

Integer number or integer.

Measurement Point

In the incoming internal inter-cell VGCS TCH handover procedure, this counter is incremented by one when the target cell (serving cell) receives an HO REQ ACK message from the originating cell. In the signaling procedure, this counter is measured at Point B in Figure 1.

Figure 1 Procedure of internal inter-cell VGCS TCH handover

Formula

This is an original counter without involving any formula.
