Average Payload of Single Downlink EGPRS TBF (KB)

Measurement Counter

TL9334: DOWN_EGPRS_RLC_SGL_TBF_AVG_THRPUT

Description

This measurement provides the average payload of a single EGPRS TBF at the downlink RLC in a cell in a granularity period. This measurement indicates the type of the service processed by the user. The measurements of uploading and downloading services are higher than those of the WAP services.

Unit

kbyte.

Measurement Point

The Average Payload of Single Downlink EGPRS TBF (KB) is calculated according to the total number of downlink EGPRS RLC data blocks using MCS-1, MCS-2, MCS-3, MCS-4, MCS-5, MCS-6, MCS-7, MCS-8, and MCS-9 coding schemes in a granularity period.

Formula

Average Payload of Single Downlink EGPRS TBF (KB) = (Total Number of Downlink EGPRS MCS1 RLC Data Blocks x 22 + Total Number of Downlink EGPRS MCS2 RLC Data Blocks x 28 + Total Number of Downlink EGPRS MCS3 RLC Data Blocks x 37 + Total Number of Downlink EGPRS MCS4 RLC Data Blocks x 44 + Total Number of Downlink EGPRS MCS5 RLC Data Blocks x 56 + Total Number of Downlink EGPRS MCS6 RLC Data Blocks x 74 + Total Number of Downlink EGPRS MCS7 RLC Data Blocks x 56 + Total Number of Downlink EGPRS MCS8 RLC Data Blocks x 68 + Total Number of Downlink EGPRS MCS9 RLC Data Blocks x 74)/(Total Number of Downlink EGPRS TBFs x 1024)
