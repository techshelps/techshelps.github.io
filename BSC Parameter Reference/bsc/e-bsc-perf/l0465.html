REPORT Responses to KILL Requests

Measurement Counter

L0465:CB_BSC_ACK_DEL_MSG

Description

This counter is used to measure the number of REPORT responses to KILL requests from the BSC to the CBC.

After receiving a KILL request on the BSC-CBC interface, the BSC sends a REPORT response message or a REJECT response message to the CBC.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC sends the CBC a REPORT response to the KILL request.

Formula

This is an original counter without involving any formula.
