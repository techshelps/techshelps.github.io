Average Main Level in the Customized MR

Measurement Counter

S4556: TRX_EXT_MAIN_RX_AVR_LEV

Description

This measurement provides the average main level in the customized MR in a granularity period.

Unit

mW.

Measurement Point

In the granularity period, this counter is calculated by using the following formula.

Formula

Average Main Level in the Customized MR = [Number of Measurements of the Main Level in the Customized MR]/[Number of Measurements of the Main and Diversity Levels in the Customized MR]
