VBS Assignment Preempt Request

Measurement Counter

A03436: CELL_VBS_ASS_PREEMPTION_REQ

Description

If no TCH resource is available in the VBS assignment procedure,, the priority in the VBS ASSIGNMENT REQUEST message determines whether preemption is permitted. If preemption is permitted, a request for TCH preemption is made. This measurement provides the number of requests for TCH preemption.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when a channel preemption request is made.

Formula

This is an original counter without involving any formula.
