Number of Uplink EGPRS TBF Abnormal Releases due to other cause

Measurement Counter

A9218: UP_EGPRS_TBF_ABNORM_REL_OTHER_CAUSE

Description

This measurement provides the number of uplink EGPRS TBF abnormal releases due to other causes in a granularity period. A high measurement value is yielded because the parameter is incorrect, the number of uplink EGPRS TBFs on the PDCH reaches the limit, or the cell is manually blocked. Refer to the related information such as device alarms to locate the cause.

Unit

Integer number or integer.

Measurement Point

If the cause is parameter error, flow control, or manual cell blocking, the BSC releases the channel resources irrespective of the MS status. The uplink EGPRS TBF using the channel as the control channel is also released. Figure 1 shows the procedure of uplink EGPRS TBF abnormal release due to other causes. When the BSC releases the uplink EGPRS TBF that uses the channel as the control channel due to other causes (see measurement point A), the counter is incremented by one.

Formula

This is an original counter without involving any formula.
