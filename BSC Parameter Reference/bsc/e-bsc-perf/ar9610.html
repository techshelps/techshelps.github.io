Average Downlink Bandwidth per GPRS User

Measurement Counter

AR9610: CELL_PDCH_DOWN_GPRS_TBF_AVG_THRPUT_RATE

Description

This measurement provides the average bandwidth per user of the downlink GPRS services in a granularity period.

Unit

Kbit/s.

Measurement Point

None.

Formula

Average Downlink Bandwidth per GPRS User = Average Downlink Bandwidth per GPRS PDCH x Number of Actually Used Downlink PDCHs per GPRS User (User)
