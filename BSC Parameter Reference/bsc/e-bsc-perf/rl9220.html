Description

This measurement provides the retransmission rates of uplink EGPRS RLC data blocks in MCS1-MCS9 coding schemes in a cell in a granularity period. If the measurement value is high, you can infer that the signal quality is poor, thus the network needs to be optimized.

Unit

%

Measurement Point

None.
