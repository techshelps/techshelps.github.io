Number of Failures to Return to Normal Call from BTS Loopback (Others)

Measurement Counter

R3851b:CELL_LOCALSWITCH_LOOP_RESTORE_OTHER_FAIL_BTS_LOOP

Description

During the process where the BTS loopback is restored to a normal call, within the BSC, the call processing module initiates a loopback recovery request. If the loopback fails because of other reasons (rather than territorial resource request failure and failures on the BTS side), a loopback recovery acknowledgement message is returned to the call processing module initiating the loopback recovery. Then, the BSC regards it as a call drop and releases the call directly.

This measurement provides the number of failures to return to normal call from BTS loopback due to other causes.

Unit

Integer number or integer.

Measurement Point

In the situation described in the Number of Attempts to Return to Normal Call from BTS Loopback part, within the BSC, the call processing module initiates a loopback recovery request during the process where the BTS loopback is restored to a normal call. If the loopback fails because of other reasons (rather than territorial resource request failure and failures on the BTS side), a loopback recovery acknowledgement message is returned to the call processing module initiating the loopback recovery. Then, this counter is measured.

Formula

This is an original counter without involving any formula.
