Total Sampled DSP CPU Usage

Measurement Counter

R9401: DSP_CPU_TOTAL_SAMPLE_USAGE

Description

This measurement provides the accumulated value of sampled DSP CPU usage in a granularity. This measurement is used to calculate the average DSP CPU usage.

Unit

%.

Measurement Point

In a granularity period, the DSP CPU usage is sampled every five seconds. All the sampled values are accumulated to the value of the counter.

Formula

This is an original counter without involving any formula.
