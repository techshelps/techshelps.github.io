Ratio of Transmit Bandwidth to Total Bandwidth

Measurement Counter

RL6021:MTP3_LNK_OCTETS_TX_RATIO

Description

This measurement provides the ratio of the transmit bandwidth to the total bandwidth of the signaling link in a GP.

Unit

Erl.

Measurement Point

The measurement point is on the BSC side, as shown in the figure.

This counter is measured when the measurement result is reported in a GP.

Formula

Ratio of Transmit Bandwidth to Total Bandwidth = (MSUs Sent on MTP3 Link x Bytes of MTP2 Message Head in MSU Messages on MTP3 Link + Octets in SIFs and SIOs Sent on MTP3 Link) x 100/(8192 x 64 kbit/s Timeslots Seized by Signaling Link x GP x 60)
