Total Bytes of Uplink GPRS Users LLC PDUs

Measurement Counter

L9421: CELL_LLC_TX_UP_GPRS_LLC_PDU_TOTAL_BYTE

Description

This measurement provides the total bytes of uplink LLC PDUs of a GPRS MS in a granularity period.

Unit

byte.

Measurement Point

When the BSC sends an UPLINK LLC PDU to the SGSN and the MS is a GPRS MS (see measurement point A in Figure 1), the bytes of the UPLINK LLC PDU are accumulated to the counter if the UPLINK LLC PDU is not the signaling about attach, detach, PDP context activation, PDP context deactivation, or routing area update.

Figure 1 Uplink LLC PDU sent to the SGSN

Formula

This is an original counter without involving any formula.
