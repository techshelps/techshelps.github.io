Uplink GPRS TBF Establish and Release Capability Measurement

Description

The PSCALL.UpGTbfEstbRel.Cell measurement unit provides the counters related to the establishment and release of the uplink GPRS TBF in a granularity period. This measurement unit indicates the traffic load of the PS services in a cell and is used as a reference for establishing the PS service model.

Measurement Object

Cell.
