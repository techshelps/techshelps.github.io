Number of Successful Handovers from Overlaid Subcell to Underlaid Subcell

Measurement Counter

H9003: OLCELL_INTO_ULCELL_SUCC_NUM

Description

This measurement provides the number of successful handovers from overlaid subcell to underlaid subcell in a granularity period.

Unit

Integer number or integer.

Measurement Point

Based on the measurement report from the MS and cell feature, the BSC determines whether to initiate the handover from overlaid subcell to the underlaid subcell. The BSC then performs channel reassignment or dynamic channel adjustment. If the operation succeeds, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
