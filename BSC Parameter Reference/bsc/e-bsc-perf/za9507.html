Number of Downlink Assignments per BSC

Measurement Counter

ZA9507: BSCRPT_BSC_ASS_DOWN_ASS

Description

This measurement provides the total number of downlink assignments in all the cells controlled by the BSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see Number of Downlink Assignments.

Formula

Number of Downlink Assignments per BSC = Number of Downlink Assignments

(That is, the total of the counter Number of Downlink Assignments in each cell of the BSC.)
