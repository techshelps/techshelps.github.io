Total Number of LLC PDUs Sent

Measurement Counter

L9505: TX_LLC_PDU

Description

This measurement provides the total number of downlink LLC PDUs sent by the BSC in a cell in a granularity period. Each time the BSC receives the PACKET DOWNLINK ACK/NACK message from the MS, the BSC determines which DOWNLINK LLC PDU is successfully received and must be deleted from the PDU buffer queue according to the receiving acknowledgement bitmap in the message.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure in which the BSC checks the received PACKET DOWNLINK ACK/NACK message to determine whether the DOWNLINK LLC PDU is successfully sent to the MS. Each time the DOWNLINK LLC PDU is successfully sent and deleted from the PDU sending queue (see measurement point A), the counter is incremented by one.

Figure 1 Receiving of the PACKET DOWNLINK ACK/NACK message by the BSC

Formula

This is an original counter without involving any formula.
