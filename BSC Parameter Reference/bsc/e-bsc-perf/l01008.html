Maximum Rate for Receiving Messages on an HDLC Channel

Measurement Counter

L01008:BSC_PIU_HDLCCHANNEL_MAXRECEIVE_RATE

Description

This measurement provides the maximum rate for receiving messages on an HDLC channel in a granularity period. After the measurement is started, the BSC measures the rate of the HDLC channel every 15 seconds.

The BSC stores only the maximum value of the average receive rates. For example, if the average receive rate in one time segment is 200 bit/s and that in the next time segment is 150 bit/s, the BSC regards that the maximum receive rate is 200 bit/s.

Unit

bit/s.

Measurement Point

This counter is measured after the data is received on an HDLC channel.



Formula

BSC_PIU_HDLCCHANNEL_MAXRECEIVE_RATE = Max(RecvData/HDLC_STATS_PFM_TIMER)

RecvData: indicates the number of bytes (data) received on an HDLC channel in a period of HDLC_STATS_PFM_TIMER (15s by default).

HDLC_STATS_PFM_TIMER: indicates the period in which the BSC performs the HDLC rate measurement. The default value is 15s.

BSC_PIU_HDLCCHANNEL_MAXRECEIVE_RATE: indicates the maximum receive rate in a granularity period.
