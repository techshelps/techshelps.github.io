Number of Paging Messages Sent on the PCH in the Second Scheduling Period

Measurement Counter

A345: CELL_PCH_MSG_NUM_Sec_CYCLE

Description

When the PCH are queued for new messages, the delay period between the scheduling time and the reception time is calculated. This counter provides the number of messages in the delay period between two paging frames. The BTS measures the value of the CELL_PCH_MSG_NUM_Sec_CYCLE and reports it in the BTS_STATIS_REPORT message to the BSC.

Unit

Integer number or integer

Measurement Point

This counter is incremented each time the information element CELL_PCH_MSG_NUM_Sec_CYCLE in the BTS_STATIS_REPORT message received from the BTS is successfully decoded by the BSC, at C1 as shown in Figure 1.

Figure 1 Procedure of the BTS sending the BTS_STATIS_REPORT message

Formula

This is an original counter without involving any formula.
