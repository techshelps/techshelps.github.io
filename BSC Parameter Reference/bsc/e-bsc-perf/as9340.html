Average Maximum Number of Downlink TBF Timeslots

Measurement Counter

AS9340: CELL_USER_DOWNLINK_AVG_MAX_TIMESLOT_NUM

Description

This measurement provides the average maximum number of timeslots allocated to downlink TBFs in the transmission state in a cell in a granularity period.

Unit

Integer number or integer.

Measurement Point

This counter is equal to the total maximum number of sampled downlink TBF timeslots divided by the total number of sampled downlink TBFs in a granularity period.

Formula

Average Maximum Number of Downlink TBF Timeslots = ([Maximum Number of GPRS Downlink Timeslots] + [Maximum Number of EGPRS Downlink Timeslots])/([Total Number of Sampled Downlink GPRS TBFs] + [Total Number of Sampled Downlink EGPRS TBFs])
