Traffic Call Drop Ratio per cell

Measurement Counter

ZTR308: CELL_RATE_CALL_DROP

Description

This measurement provides the ratio of traffic volume to call drops.

Unit

None.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

Traffic Call Drop Ratio per cell = [Traffic Volume of TCHs] x {60}/[Call Drops on Traffic Channel]
