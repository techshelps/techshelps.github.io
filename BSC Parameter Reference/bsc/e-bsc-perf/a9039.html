Number of Abnormal Uplink GPRS TBF Releases due to CS Handover

Measurement Counter

A9039:UP_GPRS_TBF_ABNORM_REL_CS_HO

Description

This measurement provides the number of uplink GPRS TBF abnormal releases due to the CS handover in a granularity period.

Unit

Integer number or integer.

Measurement Point

In DTM transmission mode, if the uplink GPRS TBF is released abnormally due to the CS handover, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
