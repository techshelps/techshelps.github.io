Number of Call Drops Due to Failures to Return to Normal Call from BTS Loopback per BSC (Terrestrial Resource Request Failure)

Measurement Counter

ZTM343: BSCRPT_CALL_DROPS_LOOP_RESTORE_APPLY_TRSL_FAIL_BTS_LOOP

Description

This measurement provides the number of call drops due to the failure in returning to normal calls from the BTS local switching. The cause value is Terrestrial Resource Request Failure

Unit

Integer number or integer.

Measurement Point

None.

Formula

Call Drops due to Failures to Return to Normal Call from BTS Loopback per BSC (Territorial Resource Request Failure) = Call Drops Due to Failures to Return to Normal Call from BTS Loopback (Territorial Resource Request Failure)

In other words, the Number of Call Drops due to Failures to Return to Normal Call from BTS Loopback per BSC (Territorial Resource Request Failure) is the sum of the Number of Call Drops due to Failures to Return to Normal Call from BTS Loopback per BSC (Territorial Resource Request Failure) of each cell in the BSC.
