Completed Handovers

Measurement Counter

CR443B: TRX_HO_SUCC

Description

In the handover procedure, the MS sends an HO ACC message to the BTS of the target cell after receiving an HO CMD message to attempt to access the new cell. The BTS notifies the HO ACC message in an HO DETECT message to the BSC. In the case of asynchronous handover (the BTS of the source cell is not the BTS of the target cell), the BTS of the target cell sends a PHY INFO message to the MS. After receiving the PHY INFO message, the MS sends a FIRST SABM frame, sends an EST IND message through the target cell to the BSC to notify the BSC that the MS has successfully accessed the target cell, and sends an HO CMP message to the BSC. When the BSC receives the HO CMP message, the counter CR443B "Completed Handovers" is incremented by one for the TRX where the TCH of the target cell resides. This counter covers the number of HO CMP messages received by the BSC from an MS when any channel of the measured TRX is used as the target channel in the handover procedure.

Unit

Integer number or integer.

Measurement Point

This counter is involved in the Intra-Cell Handover Measurement, Incoming Internal Inter-Cell Handover Measurement, Incoming External Inter-Cell Handover Measurement, and Incoming Inter-RAT Inter-Cell Handover Measurement. See the corresponding sections for the flow charts. Each time the BSC receives an HO CMP message from an MS, the counter is incremented by one.

Formula

Completed Handovers = Completed Handovers (SDCCH) + Completed Handovers (TCH)
