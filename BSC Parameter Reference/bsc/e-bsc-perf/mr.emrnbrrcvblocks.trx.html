NbrRcvBlocks in EMR Measurement

Description

The counters of this measurement unit measure the NbrRcvBlocks in EMR.

The NbrRcvBlocks refers to the number of the correct decoded blocks that the measurement is completed within a measurement period.

Measurement Object

Carrier frequency
