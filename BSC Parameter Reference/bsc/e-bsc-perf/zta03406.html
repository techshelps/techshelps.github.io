Failed VGCS Assignments per BSC (Terrestrial Resource Already Allocated)

Measurement Counter

ZTA03406:BSCRPT_VGCS_ASS_FAIL_CIC_ALLOC

Description

During the establishment of the VGCS, the MSC sends a VGCS/VBS ASSIGNMENT REQUEST message to the BSC for setting up a VGCS TCH. This measurement provides the number of failed VGCS assignments due to the terrestrial resource already allocated. Along with the Unsuccessful VGCS Assignment and counters of other failed causes, this counter analyzes the cause of the failed VGCS assignments in the BSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see Unsuccessful VGCS Assignment (Terrestrial Circuit Already Allocated).

Formula

Failed VGCS Assignments per BSC (Terrestrial Resource Already Allocated) = Unsuccessful VGCS Assignment (Terrestrial Resource Already Allocated)

(That is, the total of the counter Unsuccessful VGCS Assignment (Terrestrial Resource Already Allocated) in each cell of the BSC.)
