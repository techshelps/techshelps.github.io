Channel Assignment Overflows in Concentric Cell (SDCCH)

Measurement Counter

R3225G:CELL_CH_OVERFLOW_TIMES_ULCELL_SIG_CH

R3224G:CELL_CH_OVERFLOW_TIMES_OLCELL_SIG_CH

Description

When the BSC receives an SDCCH request, it fills the concentric attribute in the channel request based on the data already configured, and then initiates the channel assignment procedure. The counters are used to measure the number of times the underlaid subcell or the overlaid subcell has no idle SDCCH to be assigned.

Unit

Integer number or integer.

Measurement Point

In the SDCCH assignment, the measurements are triggered when the underlaid subcell or the overlaid subcell has no idle SDCCH to be assigned.

Formula

This is an original counter without involving any formula.
