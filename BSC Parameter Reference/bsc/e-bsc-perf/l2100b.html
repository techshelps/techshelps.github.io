Description

The BSC and the BTS communicate through E1 cables.

The BTS detects whether there are CRC serious errors on E1 ports every second. If a serious error occurs, the counter is incremented by one, and the total is reported to the BSC every five minutes. Then, the BSC accumulates the reported numbers to get the value in a GP.

These counters measures the duration of CRC serious errors on E1 ports. The greater the value is, the worse the transmission quality is.

Unit

s.

Measurement point

The BTS reports the message containing CRC authentication results to the BSC every five minutes. Then, the BSC measures these counters according to the port number in the message.

Formula

This is an original counter without involving any formula.
