Maximum Seizure Ratio of PCH Paging Queue

Measurement Counter

L3188M:CELL_FCTRL_PCH_QUE_MAX_TN

Description

When the downlink CCCH (PCH) in the cell is overloaded, the BTS measures the seizure ratio of PCH paging queue and sends a STATUS REPORT message to the BSC.

This counter indicates the maximum value of the seizure ratio of PCH paging queue, which is reported in the BTS STATUS REPORT message sent from the BTS in a measurement period.

Unit

%.

Measurement Point

The BSC receives the BTS STATUS REPORT message, parses the seizure ratio of PCH paging queue, and takes the maximum value as the value of this counter in a measurement period.

Formula

This is an original counter without involving any formula.
