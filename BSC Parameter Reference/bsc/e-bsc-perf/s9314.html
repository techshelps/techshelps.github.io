Description

This measurement provides the number of times that N timeslots are actually allocated to downlink TBFs when M timeslots can be allocated according to the calculation based on traffic types and MS multi-timeslot capability in a granularity period. The higher the value, the higher the timeslot utilization rate. The values of N and M consist of 1, 2, 3, 4, and 5. N is equal to or smaller than M.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the BSC periodically samples all the downlink TBFs. If the sampled TBFs are in transmission state, the BSC calculates the maximum number of timeslots that can be allocated to downlink TBFs based on the multislot capability and service type of the MS. In addition, the BSC obtains the number of timeslots that are actually allocated to downlink TBFs. If the number of actually allocated downlink timeslots is N and the calculated maximum number of downlink timeslots is M, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
