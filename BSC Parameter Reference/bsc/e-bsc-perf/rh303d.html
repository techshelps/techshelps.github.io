External Incoming Radio Handover Success Ratio per cell

Measurement Counter

RH303D: CELL_INTERBSC_INTO_CELL_RD_HO_SUCC_RATE

Description

The measurement provides the success rate of external incoming cell handovers on the Um interface.

Unit

%.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

External Incoming Radio Handover Success Ratio per cell = [Successful External Incoming Cell Handovers per BSC] x {100}/[Incoming External Inter-Cell Handover Responses per BSC]
