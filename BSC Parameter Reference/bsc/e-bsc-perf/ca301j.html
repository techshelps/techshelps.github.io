Immediate Assignment Commands (Circuit Service)

Measurement Counter

CA301J: CELL_IMM_ASS_CMD_CIRCUIT

Description

This measurement provides the number of IMM ASS CMD messages sent by the BSC to an MS in the measured cell when the access reason is related to circuit services (MOC, MTC, Call Re-Establishment, Emergency Call, or Locating Updating).

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, each time the BSC sends an IMM ASS CMD message to an MS in the measured cell and the access reason is Circuit Service (see measurement point B), the counter is incremented by one.

Figure 1 Immediate assignment procedure

Formula

Immediate Assignment Commands (Circuit Service) = [Immediate Assignment Commands (SDCCH)] + [Immediate Assignment Commands (TCHF)] + [Immediate Assignment Commands (TCHH)]
