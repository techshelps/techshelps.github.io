Call Drops on TCH per BSC (Error Indication)

Measurement Counter

ZTM328:BSCRPT_TOTAL_CELL_CALL_DROPS_ERR_IND_TCH

Description

After the MS enters the normal call state on the TCH, the BTS sends an ERR IND message to the BSC because of the connection exception of the radio link layer. Through the ERR IND message, the BSC is notified that call drop occurs on the air interface because of radio link errors. This measurement provides the number of call drops on the TCHs per BSC (error indication) when the call drops are caused by error indications.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.

Formula

Call Drops on TCH per BSC (Error Indication) =

[ Call Drops due to ERR IND Received on TCHF (Signaling Channel) in Stable State (T200 Expired)]+

[ Call Drops due to ERR IND Received on SDCCH (Signaling Channel) in Stable State (Unsolicited DM Response)]+

[ Call Drops due to ERR IND Received on SDCCH (Signaling Channel) in Stable State (Sequence Error)]+

[ Call Drops due to ERR IND Received on TCHH (Signaling Channel) in Stable State (T200 Expired)]+

[ Call Drops due to ERR IND Received on TCHH (Signaling Channel) in Stable State (Unsolicited DM Response)]+

[ Call Drops due to ERR IND Received on TCHH (Signaling Channel) in Stable State (Sequence Error)]+

[ Call Drops due to ERR IND Received on TCHF (Traffic Channel) in Stable State (T200 Expired)]+

[ Call Drops due to ERR IND Received on TCHF (Traffic Channel) in Stable State (Unsolicited DM Response)]+

[Call Drops due to ERR IND Received on TCHF (Traffic Channel) in Stable State (Sequence Error)]+

[ Call Drops due to ERR IND Received on TCHH (Traffic Channel) in Stable State (T200 Expired)]+

[ Call Drops due to ERR IND Received on TCHH (Traffic Channel) in Stable State (Unsolicited DM Response)]+

[ Call Drops due to ERR IND Received on TCHH (Traffic Channel) in Stable State (Sequence Error)]
