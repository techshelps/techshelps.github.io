Channel State Measurement per Cell

Description

A busy channel is a channel that is occupied. The number of busy channels indicates the random access frequency of the MSs in a sampling cell.

A blocked channel is a channel that is blocked due to OM channel blocking or RSL link disconnection. The number of blocked channels indicates channel blocking condition due to OM channel blocking or RSL link disconnection in a sampling cell.

An idle channel is a channel that is normal and unoccupied. The number of idle channels indicates the idling conditions of the channels in a sampling cell.

This measurement unit provides the number of the SDCCH and TCH channels in the busy, blocked, and idle state in a 900/850 or 1800/1900 cell, including the number of dynamically changed channels.

Measurement Object

Cells.
