Number of Concurrent Uplink EGPRS TBFs on Per EGPRS PDCH

Measurement Counter

R9376: PER_EGPRS_PDCH_HOLD_UPLINK_ETBF_NUM

Description

This measurement provides the total number of uplink EGPRS TBFs on all the EGPRS PDCHs in a cell in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the total number of uplink EGPRS TBFs on all the EGPRS PDCHs is sampled every five seconds and is accumulated to the counter.

Formula

This is an original counter without involving any formula.
