Unsuccessful VGCS Assignment (Other Causes)

Measurement Counter

A03413: CELL_VGCS_ASS_FAIL_OTHER

Description

This measurement provides the number of times that the BSC finds that the channel type, channel rate, or speech coding algorithm in the VGCS ASSIGNMENT REQUEST message received from the MSC is incorrect.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC finds that the channel type, channel rate, or speech coding algorithm in the VGCS ASSIGNMENT REQUEST message received from the MSC is incorrect.

Formula

This is an original counter without involving any formula.
