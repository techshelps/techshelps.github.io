Failed VGCS Assignments per BSC

Measurement Counter

ZTA03403:BSCRPT_VGCS_ASS_FAIL

Description

During the establishment of the VGCS, the MSC sends a VGCS/VBS ASSIGNMENT REQUEST message to the BSC for setting up a VGCS TCH. If the assignment is successful, the BSC sends a VGCS/VBS ASSIGNMENT RESULT message to the MSC; otherwise, the BSC sends a VGCS/VBS ASSIGNMENT FAILURE message to the MSC. This measurement provides the number of VGCS ASSIGNMENT FAILURE messages that the BSC sends to the MSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see Unsuccessful VGCS Assignment.

Formula

Failed VGCS Assignments per BSC = Unsuccessful VGCS Assignment

(That is, the total of the counter Unsuccessful VGCS Assignment in each cell of the BSC.)
