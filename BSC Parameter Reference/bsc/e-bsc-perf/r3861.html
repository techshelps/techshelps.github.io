Number of Successful Attempts to Return to Normal Call from BTS Loopback

Measurement Counter

R3861:CELL_BTS_LOOP_RESTORE_SUCC

Description

During the process where the BTS loopback is restored to a normal call, within the BSC, the call processing module initiates a loopback recovery request. If TC and Abis resources are successfully requested and the transmission channel between the Ater interface and the Abis interfaces is successfully reconnected, a loopback recovery acknowledgement message (loopback success) is returned to the call processing module initiating the loopback recovery. The call is restored to a normal one.

This measurement provides the number of successful attempts to return to normal call from BTS loopback. This measurement is used to calculate the BTS loopback to normal call success rate.

Unit

Integer number or integer.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

Number of Successful Attempts to Return to Normal Call from BTS Loopback =

[Number of Attempts to Return to Normal Call from BTS Loopback]-

[Number of Failures to Return to Normal Call from BTS Loopback]
