Attempted Intra-Cell VGCS TCH to Dedicated TCH Handover

Measurement Counter

A03478: CELL_GCC_INTRACELL_GCC_TO_TCH_HO_REQ

Description

The procedure for the intra-cell VGCS TCH to the dedicated TCH handover is equivalent to the inter-BSC handover procedure. The difference is that the originating BSC and the target BSC are the same in the intra-cell VGCS TCH to the dedicated TCH handover. The originating BSC sends a HANDOVER REQUIRED message to the MSC to initiate a handover procedure. The HANDOVER REQUEST message sent by the MSC to the target BSC cannot contain the CIC corresponding to the current VGCS TCH. Otherwise, the BSC responds with a rejection message. This measurement provides the number of handover requests sent from the originating BSC to the MSC.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the originating BSC sends a HANDOVER REQUIRED message to the MSC after the handover algorithm decision is made. The measurement point is A, as shown in Figure 1.
