Number of Sent RESET Messages

Measurement Counter

L9706: NSVC_TRAN_TX_RST_MSG

Description

This measurement provides the number of sent RESET messages for an NSVC in a granularity period. The message is sent from the BSC to the SGSN. The BSC initiates the reset procedure for an NSVC under one of the following conditions: The system starts to work or the fault of the physical link on the Gb interface is rectified; the settings of the NSVC are dynamically added or modified; the NSVC is reset through the OM system.

The network quality may deteriorate because of the great value of the counter. The possible causes are as follows: faults of the physical link on the Gb interface, incorrect data configuration, or inconsistency of the data configurations on the BSC side and on the SGSN side. Check the link status and related data configuration of the Gb interface.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the counter is incremented by one every time the BSC sends an NSVC RESET REQ message (see measurement point A in Figure 1).
Figure 1 NSVC RESET REQ message sent by the BSC

Formula

This is an original counter without involving any formula.
