MSG ABIS OVERLOAD (CCCH OVERLOAD) Messages Sent on Abis Interface

Measurement Counter

L3188E:CELL_TRX_OVERLOAD_CCCH_OVERLOADS

Description

When the BTS detects an overloaded CCCH, the BTS reports an OVERLOAD message to the BSC and sends a CCCH LOAD INDICATION message to the BSC.

This measurement provides the number of OVERLOAD messages (CCCH OVERLOAD) that the BSC receives from the BTS controlling the cell.

MSG ABIS OVERLOAD (CCCH OVERLOAD) Messages Sent on Abis Interface = MSG CCCH LOAD IND (RACH) Messages Sent on Abis Interface + MSG CCCH LOAD IND (PCH) Messages Sent on Abis Interface

Actually, due to bit errors, the counter on the left of this equation cannot be equal to the counters on the right.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC receives an OVERLOAD message (CCCH OVERLOAD) from the BTS.

Formula

This is an original counter without involving any formula.
