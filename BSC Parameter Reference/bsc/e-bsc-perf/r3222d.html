Channel Assignment Overflows in Concentric Cell in Incoming Internal Inter-Cell Handover Procedure (TCH)

Measurement Counter

R3222D:CELL_INTRABSC_INCELL_HO_CH_OVERFLOW_TIMES_TCH_PREF_ULCELL

R3223D:CELL_INTRABSC_INCELL_HO_CH_OVERFLOW_TIMES_TCH_PREF_OLCELL

Description

In the incoming internal inter-cell handover procedure, the counters are used to measure the number of TCH assignment overflows when the assignment fails or the channel with the concentric attribute inconsistent with the requirement is assigned. The measurement is based on the requirement for the attributes of the concentric cell.

Unit

Integer number or integer.

Measurement Point

In the incoming internal inter-cell handover procedure, the measurements are triggered when the assignment fails or when the channel with the concentric attributes inconsistent with the requirement is assigned. The measurements are based on the requirement for the attributes of the concentric cell.

Formula

This is an original counter without involving any formula.
