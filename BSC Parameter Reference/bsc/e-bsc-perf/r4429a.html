Measurement of Attempted Mode Modify Procedures

Measurement Counter

R4429A: TRX_CHMD_CMD_TCH

Description

In the Mode Modify procedure, the BSC sends a CHAN MODE MODIFY message to the MS after receiving a MODE MODIFY ACK message from the BTS. At this time, the counter R4429A "Attempted Mode Modify Procedures (TCH)" is incremented by one for the TRX where the TCH resides.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, each time the BSC sends a CHAN MODE MODIFY message to the MS (see measurement point E1), the counter is incremented by one.

Figure 1 Mode Modify procedure

Formula

This is an original counter without involving any formula.
