Number of Measurement Reports with RQI

Measurement Counter

S4440E:TRX_RQI_MR_TOTAL_NUM

Description

This measurement provides the number of MRs with the RQI in the granularity period.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula.

Formula
