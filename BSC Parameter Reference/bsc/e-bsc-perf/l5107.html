I Frames Resent by the BTS on LAPD Link

Measurement Counter

L5107:LAPD_RETX

Description

This measurement provides the number of I frames that the BTS L2 resends on LAPD Link.

If the BTS receives a REJ frame from the BSC, it means that an error occurs when the BSC receives I frames. Then the BTS resends this I frame and all the I frames with numbers bigger than this number.

If the BTS receives an RNR frame from the BSC, it means that the L2 receiver of the BSC is busy and cannot receive I frames. Then the BTS resends this I frame and all the I frames with numbers bigger than this number after the L2 receiver of the BSC is free.

This counter is incremented by 1 when the BTS resends an I frame.

If the physical uplink is disrupted, the measurement result cannot be sent to the BSC and the value of this counter does not increase.

Unit

Integer number or integer.

Measurement point

This counter is measured when the BTS L2 resends an I frame on LAPD Link.

Formula

This is an original counter without involving any formula.
