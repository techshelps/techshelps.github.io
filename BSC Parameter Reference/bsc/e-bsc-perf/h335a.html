Outgoing External Inter-Cell Handover Clear Cmd Times (Before Handover Response)

Measurement Counter

H335A:CELL_INTERBSC_OUTCELL_CLEAR_CMD_BEFORE_HO_CMD

Description

During the outgoing external inter-cell handover procedure, if the BSC receives a CLEAR CMD message from the MSC before the HO CMD message, the BSC releases the call to terminate the handover. This counter provides the number of CLEAR CMD messages in this scenario.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1 and Figure 2, this counter is measured at E1, when the BSC receives a CLEAR CMD message from the MSC before the HO CMD message.

Figure 1 Measurement of the counters related to failed outgoing external inter-cell handover (excluding directed retry)

Figure 2 Measurement of the counters related to failed outgoing external inter-cell handover (directed retry)

Formula

This is an original counter without involving any formula.
