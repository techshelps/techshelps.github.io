Number of Downlink EGPRS Intermit Transfers per BSC

Measurement Counter

ZA9318: BSCRPT_BSC_TBF_DOWN_EGPRS_TRANS_INTERRUPT_TIMES

Description

This measurement provides the total number of intermitted downlink EGPRS transfers in all the cells controlled by the BSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see Number of Downlink EGPRS intermit transfers.

Formula

Number of Downlink EGPRS Intermit Transfers per BSC = Number of Downlink EGPRS Intermit Transfers

(That is, the total of the counter Number of Downlink EGPRS Intermit Transfers in each cell of the BSC.)
