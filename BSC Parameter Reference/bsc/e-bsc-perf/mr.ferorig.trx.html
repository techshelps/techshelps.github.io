FER Measurement per TRX

Description

This measurement unit is used to measure the average frame erasion rate of the BTS in the TCHs to reflect the Um interface quality. The higher the average frame erasion rate, the worse the Um interface quality. The lower the average frame erasion rate, the better the Um interface quality.

NOTE:

Frame erasion rate: The BTS erases the error frame on the Um interface. The frame erasion rate is used to specify the proportion of the erased frames to the total number of frames.
