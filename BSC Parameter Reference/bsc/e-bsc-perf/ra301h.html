Success Rate of Random Access (Call Type)

Measurement Counter

RA301H: CELL_RANDOM_ACCESS_SUCC_RATE_CALL

Description

This counter provides the ratio of the IMM ASS CMD messages sent by the BSC to the MS after the BSC receives a CHAN RQD message in which the MS access reason is call-related, such as MOC, MTC, Call Re-establishment, or Emergency Call, from the BTS.

Unit

%.

Measurement Point

None.

Formula

Success Rate of Random Access (Call Type) = [Immediate Assignment Commands (Call Type)] x {100%}/[Channel Requests (Call Type)]
