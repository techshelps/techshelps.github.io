Average Interval of Packet Access Requests on PCCCH(s)

Measurement Counter

AA9409: PCCCH_PS_ACCESS_REQ_AVG_REACH_TIME

Description

This measurement provides the average interval of packet access requests received on the PCCCHs in a cell in a granularity period. The smaller the value of the counter is, the more frequent the PS services are performed. according to the configuration on site, check whether the congestion occurs.

Unit

s.

Measurement Point

None.

Formula

Average Interval of Packet Access Requests on PCCCH(s) = (GP x 60)/[Number of Packet Channel Requests Received on PCCCH]

GP indicates the measurement granularity, which is expressed in minute.
