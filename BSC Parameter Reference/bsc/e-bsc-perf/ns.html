NS Measurement(NS)

Overview

NS Measurement is used to measure the performance of the NS layer. The following lists the measurement objects in different transmission modes:
  • In Gb over FR, the measurement object is NSVC.
  • In Gb over IP, the measurement object is NSVL.
The following lists the functions of measurement results:
  • The measurement result can be used to analyze the traffic load of the GPRS BSS. According to the measurement results of several days or weeks, the traffic load model can be obtained and offers guidance on the engineering design and network planning. For example, the number of NS PDUs sent on an NSVC or NSVL indicates the load sharing of the NS layer on the BSS.
  • In Gb over IP, the measurement object is NSVL.
    • The number of discarded NS PDUs can be used to analyze the available transmission bandwidth of the NS layer. If the number of discarded NS PDUs is great and each NSVC or NSVL configured for an NSE is in normal state, it indicates that the available transmission bandwidth of the NS layer is insufficient, and some NS PDUs are discarded. The number of available timeslots on the bearer channel (BC) needs to be added.
    • The measurement result can analyze the RESET messages of the Gb interface, such as the number of sent RESET messages, number of received RESET ACK messages, number of received RESET messages, and number of sent RESET ACK messages.
    • The measurement result can analyze the BLOCK and UNBLOCK messages of the NS layer on the Gb interface, such as the number of sent BLOCK messages, number of received BLOCK ACK messages, number of received BLOCK messages, number of sent BLOCK ACK messages, number of sent UNBLOCK messages, number of sent UNBLOCK ACK messages, number of received UNBLOCK messages, and number of sent UNBLOCK ACK messages.
