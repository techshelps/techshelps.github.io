BTSFE Measurement

Description

Two FE ports are available for a BTS when IP transmission is implemented. The BTS measures the packets transferred over the FE ports by category. This helps the customer get knowledge of the IP transmission performance.

The BTS reports the measurement information to the BSC for processing only when IP transmission is implemented in the BTS.

Measurement Object

FE port of the BTS.
