Receive Quality Measurement Distribution per TRX

Description

This counter measures the rank distribution of the receive quality in the measurement report on the TCH. The receive quality ranges from rank 0 to rank 7. Each rank corresponds to a bit error rate range. Higher rank suggests higher bit error rate and worse receive quality.

Measurement Object

TRX.
