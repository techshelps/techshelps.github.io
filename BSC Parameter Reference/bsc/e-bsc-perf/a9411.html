Number of Resource Requests Received on Downlink PACCH

Measurement Counter

A9411: DOWN_PACCH_UPLINK_RES_REQ

Description

This measurement provides the number of PACKET DOWNLINK ACK messages sent by the MS on the downlink PACCH to request uplink packet channels. This measurement indicates the uplink resource request from the MS during the downlink TBF transmission.

Unit

Integer number or integer.

Measurement Point

To establish the uplink TBF in the downlink TBF, the MS sends the BSC a PACKET DOWNLINK ACK/NACK message, which contains the CHANNEL REQUEST information element. Figure 1 shows the procedure for establishing the uplink TBF in the downlink TBF. Each time the BSC receives a PACKET DOWNLINK ACK/NACK message that contains the CHANNEL REQUEST information element (see measurement point A), the counter is incremented by one.

Figure 1 Uplink TBF establishment in the downlink TBF

Formula

This is an original counter without involving any formula.
