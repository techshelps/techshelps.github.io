Number of CS Upgrades on Uplink GPRS TBF

Measurement Counter

L9016: UP_GPRS_TBF_CS_UPGRADE

Description

This measurement provides the number of CS upgrades on the uplink GPRS TBF in a cell in a granularity period. This counter is used to analyze the uplink transmission quality. If the Number of CS Upgrades on Uplink GPRS TBF is greater than the Number of CS Degrades on Uplink GPRS TBF, it indicates that the network quality is good. If both the Number of CS Upgrades on Uplink GPRS TBF and the Number of CS Degrades on Uplink GPRS TBF are great, it indicates that the network quality is poor, and the coding scheme of the MS may be frequently adjusted.

Unit

Integer number or integer.

Measurement Point

Every time the BSC changes the CS coding scheme, it determines whether to upgrade or degrade the CS coding scheme by comparing the current CS coding scheme with the previous CS coding scheme. Figure 1 shows the procedure in which the MS sends a data block using more robust CS coding scheme to the BSC. Every time the BSC receives an UPLINK RLC DATA BLOCK message (see measurement point A), the counter is incremented by one.
