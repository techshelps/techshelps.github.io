Random Access Requests Discarded on LAPD

Measurement Counter

A533:LAPDLNK_DISC_RAND_ACC_REQ

Description

In a period (usually 100 ms), if the number of random access requests exceeds the threshold, the random access request is discarded and this counter is incremented by one.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the a random access request is discarded because the threshold for flow control on the LAPD is reached.

Formula

This is an original counter without involving any formula.
