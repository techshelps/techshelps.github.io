Successful Incoming Inter-BSC Inter-Cell VGCS TCH Handovers

Measurement Counter

A03492: CELL_GCC_INTERBSC_INTOCELL_HO_SUCC

Description

In the inter-BSC handover procedure, upon receiving a HANDOVER REQUEST message from the MSC, the target BSC assigns new channels to the MS. After the MS successfully accesses the target cell, the BSC sends a HANDOVER COMPLETE message to the MSC. This measurement provides the number of times that the BSC sends a HANDOVER COMPLETE message to the MSC.

Unit

Integer number or integer.

Measurement Point

In the inter-BSC handover procedure, this counter is incremented by one when the target cell (the serving cell) receives a HANDOVER COMPLETE message and sends the message to the MSC. The measurement point is C in the incoming inter-BSC inter-cell VGCS TCH handover procedure, as shown in Figure 1.

Figure 1 Incoming inter-BSC inter-cell VGCS TCH handover procedure

Formula

This is an original counter without involving any formula.
