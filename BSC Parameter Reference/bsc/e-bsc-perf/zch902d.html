Successful Reconnections to Old Channels upon Intra-Cell Handover Failure per BSC

Measurement Counter

ZCH902D:BSCRPT_INTRACELL_HO_FAIL_RECONN_SUCC

Description

This measurement provides the number of successful reconnections to old channels upon internal inter-cell handover failure in all the cells controlled by a BSC. The types of channel include SDCCH, TCHF, and TCHH.

In the internal inter-cell handover procedure, the BSC sends an HO CMD message to the MS and starts a timer to wait for an HO CMP message. Before the timer expires, the MS is connected to the old channel and reports a handover failure message on the old channel. After receiving the message, the BSC collects the number of successful reconnections to old channel upon intra-cell handover failure according to different cause values carried.

This counter contains the following cause values of successful reconnections to old channels upon intra-cell handover failure:

Abnormal release, unspecified

Abnormal release, channel unacceptable

Abnormal release, timer expired

Abnormal release, no activity on the radio path

Preemptive release

Handover impossible, timing advance out of range

Channel mode unavailable

Frequency not implemented

Call already cleared

Semantically incorrect message

Invalid mandatory information

Message type non-existent or not implemented

Message type not compatible with protocol state

Conditional IE error

No cell allocation available

Protocol error unspecified

Other

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.
