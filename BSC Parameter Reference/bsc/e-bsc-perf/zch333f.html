Failed Outgoing External Inter-Cell Handovers per BSC

Measurement Counter

ZCH333F: BSCRPT_INTERBSC_OUTCELL_HO_FAIL

Description

This measurement provides the total number of failed outgoing external inter-cell handovers per BSC. The channel involved in the measurement can be the SDCCH, TCHF, or TCHH.

The handover causes consist of the following: uplink quality, downlink quality, uplink strength, downlink strength, timing advance, better cell, load, rapid level drop, MSC intervention, OM intervention, EDGE coverage, and other causes.

Unit

Integer number or integer.

Measurement Point

The value of this counter is calculated according to the following formula.

Formula

Failed Outgoing External Inter-Cell Handovers per BSC =

[Failed Outgoing Internal Inter-Cell Handovers (SDCCH) (Excluding Directed Retry) (900/850-900/850) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (SDCCH) (Excluding Directed Retry) (1800/1900-1800/1900) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (SDCCH) (Excluding Directed Retry) (900/850-1800/1900) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (SDCCH) (Excluding Directed Retry) (1800/1900-900/850) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (TCHF) (Excluding Directed Retry) (900/850-900/850) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (TCHF) (Excluding Directed Retry) (1800/1900-1800/1900) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (TCHF) (Excluding Directed Retry) (900/850-1800/1900) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (TCHF) (Excluding Directed Retry) (1800/1900-900/850) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (TCHH) (Excluding Directed Retry) (900/850-900/850) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (TCHH) (Excluding Directed Retry) (1800/1900-1800/1900) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (TCHH) (Excluding Directed Retry) (900/850-1800/1900) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (TCHH) (Excluding Directed Retry) (1800/1900-900/850) per BSC] +

[Failed Outgoing Internal Inter-Cell Handovers (Directed Retry) (900/850-900/850)] +

[Failed Outgoing Internal Inter-Cell Handovers (Directed Retry) (1800/1900-1800/1900)] +

[Failed Outgoing Internal Inter-Cell Handovers (Directed Retry) (900/850-1800/1900)] +

[Failed Outgoing Internal Inter-Cell Handovers (Directed Retry) (1800/1900-900/850)]
