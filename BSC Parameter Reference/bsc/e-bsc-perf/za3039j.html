SDCCH Seizures for Speech Service Per BSC

Measurement Counter

ZA3039J: BSCRPT_SDCCH_SUCC_TIMES_SPEECH

Description

This measurement provides the number of times that the BSC receives an EST IND message in which the MS access cause is related to speech service, such as MOC Non SMS, MTC, Emergency Call, Call Re-establishment, or USSD service from the BTS and the channel type specified in the IMM ASS CMD message is SDCCH in a cell.

Unit

Times.

Measurement Point

The BSC receives a message of call setup indication sent by the BTS. Based on the layer 3 information carried by the message, the BSC triggers the measurement.

Formula

This is an original counter without involving any formula.
