Intra-BSC Incoming Cell Handover PT Change Times (A_IP)

Measurement Counter

H329A: CELL_INTRABSC_INTO_CELL_HO_PT_CHANGE_TIMES_A_IP

Description

This measurement provides the number of PT changes during an intra-BSC incoming cell handover (A_IP). PT is the payload type, which is a new information element on the IP-based A interface.

Unit

Integer number or integer.

Measurement Point

The measurement is triggered when the BSC sends the Channel Modify Prepare message successfully.

Formula

This is an original counter without involving any formula.
