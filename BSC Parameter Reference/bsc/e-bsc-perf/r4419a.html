Measurement of Attempted Assignments

Measurement Counter

R4419A: TRX_ASS_CMD_TCH

Description

In the assignment procedure, the BSC assigns a TCH and sends a CHANNEL ACTIVATION message to the BTS. Upon receipt of a CHANNEL ACTIVATION ACKNOWLEDGE message, the BSC sends an ASS CMD message to the MS and at the same the counter R4419A "Attempted Immediate Assignments (TCH)" is incremented by one for the TRX where the TCH resides.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, each time the BSC sends an ASS CMD message to the MS (see measurement point B), the counter is incremented by one.

Figure 1 Assignment procedure

Formula

This is an original counter without involving any formula.
