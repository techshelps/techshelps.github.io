Attempted Immediate Assignments

Measurement Counter

CR440A: TRX_IMM_ASS

Description

In the immediate assignment procedure, the BSC receives a CHANNEL ACTIVATION ACKNOWLEDGE message from the BTS, indicating the channel is successfully activated. The BSC sends an IMM ASS CMD message to the MS to instruct the MS to access the assigned channel, which can be an SDCCH or TCH. At this time, the counter CR440A "Attempted Immediate Assignments" is incremented by one. This counter covers the number of IMM ASS CMD messages sent by the BSC on any channel of the measured TRX.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, each time the BSC sends an IMM ASS CMD message to an MS in the measured cell (see measurement point B), the counter is incremented by one.

Figure 1 Immediate assignment procedure

Formula

Attempted Immediate Assignments = Attempted Immediate Assignments (SDCCH) + Attempted Immediate Assignments (TCH)
