Error CRC Packets Received over FE Port of the BTS

Measurement Counter

SC003A: BTSFE_RX_CRC_ERR

Description

This measurement provides the number of error packets against CRC, which are received from the BSC over the FE ports of the BTS using IP transmission.

Unit

Integer number or integer.

Measurement Point

The logical processing module at the physical layer performs the measurement after receives the error packets against CRC. The BTS periodically reports the result, and the BTSM disassembles the packets and processes the measurement results.

Formula

This is an original counter without involving any formula.
