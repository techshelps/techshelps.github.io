Number of Uplink Frames with FR AMR of N kbit/s in the Customized MR

Measurement Counter

S4524: TRX_EXT_AMR_FR_475_UP_NUM

S4526: TRX_EXT_AMR_FR_515_UP_NUM

S4528: TRX_EXT_AMR_FR_590_UP_NUM

S4530: TRX_EXT_AMR_FR_670_UP_NUM

S4532: TRX_EXT_AMR_FR_740_UP_NUM

S4534: TRX_EXT_AMR_FR_795_UP_NUM

S4536: TRX_EXT_AMR_FR_1020_UP_NUM

S4538: TRX_EXT_AMR_FR_1220_UP_NUM

Description

When decoding the customized extended MR or pre-processed customized extended MR from the BTS, the BSC checks whether the EI_FRAME cell is contained in the MS. If the cell is contained, and the cell shows that the frame type is FR AMR N kbit/s, then the BSC decodes the number of uplink frames and the specified counter Number of Uplink Frames with FR AMR of N kbit/s in the Customized MR is incremented by the number of uplink frames.

Unit

Integer number or integer.

Measurement Point

After receiving a customized extended MR or pre-processed customized extended MR from the BTS, the BSC decodes the MR. The specified counter is measured when the EI_FRAME cell is contained in the MS and the number of uplink frames with the type of FR AMR N kbit/s is contained in the EI_FRAME cell.

N represents the coding rate. The coding rate of the full rate speech ranges from 4.75 kbit/s to 12.20 kbit/s.

Measurement object: TRXs.

Formula

This is an original counter without involving any formula.
