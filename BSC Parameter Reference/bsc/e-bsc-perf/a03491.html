Attempted Incoming Inter-BSC Inter-Cell VGCS TCH Handovers

Measurement Counter

A03491: CELL_GCC_INTERBSC_INTOCELL_HO_REQ

Description

In the incoming inter-BSC inter-cell VGCS TCH handover procedure, upon receiving a HANDOVER REQUEST message from the MSC, the target BSC encapsulates the message and then forwards the message to the target cell. This measurement provides the number of times that the current cell receives the incoming-BSC handover request message. This counter indicates the frequency of the handover from a cell under control of another BSC to the current cell.

Unit

Integer number or integer.

Measurement Point

In the incoming inter-BSC inter-cell VGCS TCH handover procedure, this counter is incremented by one when the target cell (the current cell) receives an incoming-BSC handover request. The measurement point is A in the incoming inter-BSC inter-cell VGCS TCH handover procedure, as shown in Figure 1.

Figure 1 Incoming inter-BSC inter-cell VGCS TCH handover procedure

Formula

This is an original counter without involving any formula.
