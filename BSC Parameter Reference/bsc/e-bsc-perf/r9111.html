Number of Unsuccessful Application Attempts of Abis Timeslot for Sending Network Configuration to BTS Failure

Measurement Counter

R9111: CIC_APPLY_FAIL_SEND_NET_CONFIG_TO_BTS_FAIL

Description

This measurement provides the number of unsuccessful applications for the Abis timeslot because the network configuration table is not sent to the BTS successfully.

Unit

Integer number or integer.

Measurement Point

In Abis over TDM mode, the bandwidth application may be initiated when the PDCH coding scheme is adjusted.When the coding scheme of the PDCH is adjusted, the request for Abis timeslots fails because the BTS sends an NACK message to the BSC after receiving the configuration information about the network configuration table from the BSC, then the number of failed requests for Abis timeslots is accumulated to the value of the counter.

Formula

This is an original counter without involving any formula.
