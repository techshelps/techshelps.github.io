Call Drops on Traffic Channel (TCHH)

Measurement Counter

RM3275: CELL_TRAF_CH_CALL_DROPS_TCHH

Description

This measurement provides the number of call drops due to various causes after the MS seizes a traffic channel (TCHH).

The MS seizures a traffic channel in any of the following conditions:

The MS sends an ASS CMP message to the BSC.

The MS seizes a traffic channel and sends an HO CMP message to the BSC.

Unit

Integer number or integer.

Measurement Point

The value of this counter is calculated according to the following formula.

Formula

Call Drops on Traffic Channel (TCHH) =

[Call Drops on Radio Interface in Stable State (Traffic Channel) (TCHH)] +

[Call Drops on Radio Interface in Handover State (Traffic Channel) (TCHH)] +

[Call Drops Due to No MRs from MS for a Long Time (Traffic Channel) (TCCH) ] +

[Call Drops Due to Abis Terrestrial Link Failure (TCHH) (Traffic Channel)] +

[Call Drops Due to Equipment Failure (TCHH) (Traffic Channel)] +

[Call Drops Due to Loopback Start Failure (TCHH)] +

[Call Drops Due to Failures to Return to Normal Call from Loopback (TCHH)]
