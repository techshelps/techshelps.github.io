Successful Intra-Cell VGCS TCH to Dedicated TCH Handover

Measurement Counter

A03479: CELL_GCC_INTRACELL_GCC_TO_TCH_HO_SUCC

Description

The procedure for the intra-cell VGCS TCH to the dedicated TCH handover is equivalent to the inter-BSC handover procedure. The difference is that the originating BSC and the target BSC are the same in the intra-cell VGCS TCH to the dedicated TCH handover. The originating BSC issues a HANDOVER COMMAND message to the MS and the MS accesses the target cell successfully. Then, the originating BSC receives a HANDOVER SUCCEEDED message from the MSC and starts to release the original resources. Thus, the handover is complete. This measurement provides the number of the HANDOVER SUCCEEDED messages received by the BSC from the MSC.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the originating BSC receives the HANDOVER SUCCEEDED message from the MSC. The measurement point is B, as shown in Figure 1.

Formula

This is an original counter without involving any formula.
