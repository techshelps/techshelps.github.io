Failed Outgoing Inter-RAT Inter-Cell Handovers (Reconnection to Old Channels) (TDD)

Measurement Counter

CH352D1:CELL_INTERRAN_TDD_OUTCELL_HO_FAIL_RECONN_SUCC

Description

This counter provides the number of failed outgoing inter-RAT inter-cell handovers (TDD) because the handover is terminated when the BSC receives the HO FAIL message from the MS. The Cause contained in the HO FAIL message can be:
  • Abnormal Release, unspecified
  • Abnormal release, channel unacceptable
  • Abnormal release, timer expired
  • Abnormal release, no activity on the radio path
  • Preemptive release
  • Handover failed, timing advance out of range
  • Channel mode unavailable
  • Frequency unavailable
  • Call already cleared
  • Semantically incorrect message
  • Invalid mandatory information
  • Message type non-existent or not implemented
  • Message type not compatible with protocol state
  • Conditional IE error
  • No cell allocation available
  • Protocol error unspecified
  • Other causes

Unit

Integer number or integer.

Measurement Point

During the outgoing inter-RAT inter-cell handover procedure, after the BSC sends the HO CMD message to the MS, if the MS fails to access the new channel, it reports an HO FAIL message to the BSC. This counter is measured at F when the BSC receives the HO FAIL message, as shown in Figure 1 and Figure 2.
