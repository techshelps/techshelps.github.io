Number of Successful BSC Loopback Attempts

Measurement Counter

R3830:CELL_LOCALSWITCH_BSCLOOP_SUCCESS

Description

After the call is put through or the incoming BSC handover for the speech service is complete, the BSC starts the loopback procedure. In other words, the loopback start conditions are determined, the BSC local switch call detection is performed, the call processing module initiating loopback start at one end sends a loopback request message to the call processing module at the peer end, the speech versions of the calling and called MSs are adjusted, and network arrangement is performed. If the call processing module at the TX end receives a loopback acknowledgement message (loopback success) from the call processing module at the peer end, the call is switched from normal state to loopback state.

This measurement provides the number of successful BSC loopback attempts.

Unit

Integer number or integer.

Measurement Point

After the call is put through or the incoming BSC handover for the speech service is complete, the BSC loopback procedure is started. In other words, the loopback start conditions are determined, the BSC local switch call detection is performed, the call processing module initiating loopback start at one end sends a loopback request message to the call processing module at the peer end, the speech versions of the calling and called MSs are adjusted, and network arrangement is performed. When the call processing module at the TX end receives a loopback acknowledgement message (handover success) from the call processing module at the peer end, this counter is measured in the cell where the loopback start is initiated.

Formula

This is an original counter without involving any formula.
