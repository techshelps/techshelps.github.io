RQI Measurement based on TA per TRX

Description

RQI is the acronym for Radio Quality Indication, it is used as a reference for the intra-cell handover between the TCHF and TCHH. A measurement report contains RQI only when the call is on the AMR TRX, so the intra-cell handover between the TCHF and TCHH is available for an AMR call only when the RQI is contained in the measurement report.

The measurement unit provides the distribution of the RQI based on TA to facilitate network maintenance and optimization.

Measurement Object

TRX.
