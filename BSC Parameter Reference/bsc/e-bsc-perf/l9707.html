Number of Received RESET ACK Messages

Measurement Counter

L9707: NSVC_TRAN_RX_RST_ACK_MSG

Description

This measurement provides the number of received RESET ACK messages for an NSVC in a granularity period. The message is received by the BSC from the SGSN. The RESET ACK message is a response message from the SGSN after it receives the RESET message from the BSC.

The value of the counter should be consistent with the number of sent RESET messages. If not, check the link status and related data configuration of the Gb interface.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the counter is incremented by one every time the BSC receives a NSVC RESET ACK message (see measurement point B in Figure 1).
Figure 1 NSVC RESET ACK message received by the BSC

Formula

This is an original counter without involving any formula.
