Number of Discarded NS PDUs (NSVL)

Measurement Counter

L9803: NSVL_TRAN_DISCARD_NS_PDU_NUM

Description

This measurement provides the number of NS PDUs discarded on an NSVL in Gb over IP in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the counter is incremented by one every time the BSC discards an uplink or a downlink NS PDU message on an NSVL.If the value of the counter is great, verify that each NSVL configured for the NSE is in normal state. Then, check whether the traffic volume is excessively high. If the traffic volume is excessively high, increase the transmission bandwidth of the Gb interface.

Formula

This is an original counter without involving any formula.
