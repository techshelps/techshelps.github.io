Number of Downlink GPRS RLC Dummy Blocks

Measurement Counter

L9119: DOWN_RLC_NULL_BLK

Description

This measurement provides the number of downlink GPRS RLC dummy blocks in a cell in a granularity period. The downlink GPRS RLC dummy block can be sent, provided that no downlink GPRS RLC block is available and the downlink TBF is not released. The number of GPRS RLC dummy blocks indicates the network traffic.

Unit

Integer number or integer.

Measurement Point

When the BSC is not required to send downlink GPRS RLC blocks, it sends a DOWNLINK RLC DUMMY BLOCK message to maintain the activated state of the downlink GPRS TBF. Figure 1 shows the procedure in which the BSC sends a GPRS RLC dummy block. Every time the BSC sends a DOWNLINK RLC DUMMY BLOCK message (see measurement point A), the counter is incremented by one.
