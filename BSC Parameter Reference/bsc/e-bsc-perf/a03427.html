Unsuccessful VBS Assignment (Requested Terrestrial Resource Unavailable)

Measurement Counter

A03427: CELL_VBS_ASS_FAIL_TRSL_UNAVAIL

Description

This measurement provides the number of unsuccessful VGCS assignments because the terrestrial resource is unavailable after the BSC receives the VBS ASSIGNMENT REQUEST message from the MSC.

Unit

Integer number or integer.

Measurement Point

In the VBS assignment procedure, this counter is incremented by one when the BSC finds that the requested CIC is unavailable.

Formula

This is an original counter without involving any formula.
