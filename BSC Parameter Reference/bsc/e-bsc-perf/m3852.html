Call Drops Due to Failures to Return to Normal Call from BTS Loopback (Others)

Measurement Counter

M3852:CELL_CALL_DROPS_LOOP_RESTORE_OTHER_FAIL_BTS_LOOP

Description

After a call is in the BTS local switch state, it incurs a handover. The TRX that carries the target channel of the handover may not support local switch or the specified handover fails, so the local switch cannot be continued. Then, the BSC attempts to restore the call to a normal one. The restoration, however, fails because of other reasons rather than Abis resource request failure. The BSC starts the MS call drop procedure. In this case, the BSC measures this counter.

Unit

Integer number or integer.

Measurement Point

This counter is measured when the call that is in the BTS local switch state fails to be restored to a normal one because of other reasons.

Formula

This is an original counter without involving any formula.
