Failed VBS Assignments per BSC (Terrestrial Resource Already Allocated)

Measurement Counter

ZTA03421:BSCRPT_VBS_ASS_FAIL_CIC_ALLOC

Description

During the establishment of the VBS, the MSC sends a VGCS/VBS ASSIGNMENT REQUEST message to the BSC for setting up a VBS channel. This measurement provides the number of failed VBS assignments due to the terrestrial resource already allocated. Along with Unsuccessful VBS Assignment and counters of other causes, this counter analyzes the cause of failed VBS assignments in the BSC.

Unit

Integer number or integer.

Measurement Point

At the end of each granularity period, this counter is measured by using the following formula. For details, see Unsuccessful VBS Assignment (Terrestrial Circuit Already Allocated).

Formula

Failed VBS Assignments per BSC (Terrestrial Resource Already Allocated) = Unsuccessful VBS Assignment (Terrestrial Resource Already Allocated)

(That is, the total of the counter Unsuccessful VBS Assignment (Terrestrial Resource Already Allocated) in each cell of the BSC.)
