Successful Handovers per BSC (Intra-Cell TCHF-to-TCHH Handover)

Measurement Counter

ZH9035:BSCRPT_HO_SUCC_AMR

Description

This measurement provides the number of successful intra-cell TCHF-to-TCHH handovers of all the cells controlled by a BSC.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.

Formula

Successful Handovers per BSC (Intra-Cell TCHF-to-TCHH Handover) =

(Internal Intra-Cell Handover Requests (AMR) (TCHF-TCHH) + Internal Intra-Cell Handover Requests (AMR) (TCHH-TCHF)) -

(Failed Internal Intra-Cell Handovers (AMR) (TCHF-TCHH) + Failed Internal Intra-Cell Handovers (AMR) (TCHH-TCHF))
