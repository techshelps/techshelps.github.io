Number of Successful Uplink Packet Assignments Sent on DCCH

Measurement Counter

A9806: DTM_SUCC_DCCH_UPLINK_PACKET_ASS

Description

This measurement provides the number of successful assignments after the uplink packet assignment message is sent on the DCCH.

Unit

Integer number or integer.

Measurement Point

After receiving the DTM REQUEST message from the MS, the BSC sends the uplink PACKET ASSIGNMENT message to the MS, specifying the PS channels assigned to the MS. If the BSC receives the uplink LLC PDU from the MS (see measurement point C in Figure 1), the counter is incremented by one.

Figure 1 DTM access procedure in the case of no channel reassignment

Formula

This is an original counter without involving any formula.
