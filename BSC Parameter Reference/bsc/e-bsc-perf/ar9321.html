Average Number of Occupied Static EGPRS PDCHs

Measurement Counter

AR9321: AVG_OCCU_STATIC_EPDCH_NUM

Description

This measurement provides the average number of occupied static EGPRS PDCHs in a cell in a granularity period. The occupied EGPRS PDCH is the PDCH that carries the TBFs. If Average Number of Occupied Static EGPRS PDCHs is smaller than the number of configured static EGPRS PDCHs, you can configure less static EGPRS PDCHs and more dynamic EGPRS PDCHs.

Unit

Integer number or integer.

Measurement Point

None.

Formula

Average Number of Occupied Static EGPRS PDCHs = Total Number of Sampled Static EGPRS PDCHs Occupied/Sampling Times of PDCH Measurement
