Successful Cell Outgoing Handovers (Downlink Quality)

Measurement Counter

H373B: CELLCELL_OUT_CELL_HO_SUCC_TIMES_DL_QUALITY

Description

This measurement provides the number of successful intra-BSC and inter-BSC cell outgoing handovers caused by downlink signal quality. The counter value is equal to the Failed Cell Outgoing Handovers (Downlink Quality) subtracted from the Outgoing Inter-Cell Handover Requests (Downlink Quality).

Unit

Integer number or integer.

Measurement Point

See the formula for the measurement points.

Formula

Successful Cell Outgoing Handovers (Downlink Quality) = [Outgoing Inter-Cell Handover Requests (Downlink Quality)] -

[Failed Cell Outgoing Handovers (Downlink Quality)]
