Total Number of Failed Applications for HDLC Resources

Measurement Counter

L1101: BS_ALLOC_HDLC_FAIL_SUM

Description

This measurement counter provides the total number of call failures due to failed applications for HDLC resources in Abis over IP or Abis over HDLC mode. The causes of failed applications for HDLC resources are HDLC congestion, insufficient HDLC resources, and faulty HDLC.

Unit

Integer number or integer.

Measurement Point

Every time a call is originated, the BSC performs admission control on the call during call assignment. When the admission control fails, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
