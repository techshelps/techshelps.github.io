Number of Discarded PTRAU Packets During Uplink Transmission

Measurement Counter

L9B03:TRX_GABIS_PTRAU_UPLINK_DISCARD_PACKETS

Description

This measurement provides the number of PTRAU packets discarded on a TRX during uplink transmission in the IP or HDLC network topology in a granularity period. This counter indicates the link quality in the IP or HDLC network topology.

Unit

Integer number or integer.

Measurement Point

The following figure shows the process where the BSC discards the PTRAU packets on the uplink. In a granularity period, the counter is incremented by one every time the BSC detects discontinuous PTRAU packets, that is, a PTRAU PACKET is discarded (see measurement point A in Figure 1).

Figure 1 PTRAU packets discarded on the G-Abis interface during uplink transmission

Formula

This is an original counter without involving any formula.
