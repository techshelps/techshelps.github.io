Number of Failures to Return to Normal Call from BSC Loopback

Measurement Counter

CR3850:CELL_BSC_LOOP_RESTORE_FAIL

Description

During the process where the BSC loopback is restored to a normal call, within the BSC, the call processing module initiates a loopback recovery request. If the loopback fails because of various reasons (including failure in TC resource request, failure in reconnection of transmission channel between Ater interface and Abis interface, and other reasons), a loopback recovery acknowledgement message is returned to the call processing module initiating the loopback recovery. Then, the BSC regards it as a call drop and releases the call directly.

This measurement provides the number of failures to return to normal call from BSC loopback due to various causes.

Unit

Integer number or integer.

Measurement Point

At the end of the granularity period, this counter is calculated by using the following formula.

Formula

Number of Failures to Return to Normal Call from BSC Loopback = [Number of Failures to Return to Normal Call from BSC Loopback (Territorial Resource Request Failure)] + [Number of Failures to Return to Normal Call from BSC Loopback (Others)]
