BTS FlexAbis Measurement per BTS

Description

After Abis resources of the BTS are dynamically assigned, calls continuously drop because of the Abis resources. The performance counters related to the Abis resource assignment need to be measured for the purpose of viewing the Abis resources.

These counters are measured when the BTS Abis resource assignment mode is set to FlexAbis.

Measurement Object

Site.
