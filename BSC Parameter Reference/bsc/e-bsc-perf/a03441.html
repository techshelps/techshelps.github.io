VGCS TCH Released for Downlink Detection

Measurement Counter

A03441: CELL_VGCS_DL_DETECT_REL_CH

Description

To save TCH resources, if downlink detection is enabled, the BSC periodically sends a downlink detection message to check whether an MS monitors the downlink TCH after the VGCS TCH is set up. The monitoring MS needs to send a reponse message. If no MS reponds after the downlink detection message is retransmitted for a specified number of times, it indicates that no VGCS MS is available in the cell. Then, the established VGCS TCH is released.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when no MS responds to the periodically sent downlink detection message. In the signaling procedure, this counter is measured at point A in Figure 1.
