Incoming External Inter-Cell Handover Requests (1800/1900-900/850)

Measurement Counter

RH3034G: CELL_INTERBSC_IN_CELL_HO_REQ_TIMES_1800_900

Description

BSC determines whether to initiate the handover from the originating cell to the target cell through the handover algorithm. If the frequency band of the originating cell is DCS1800 or PCS1900 and the frequency band of the target cell is GSM900, GSM900&DCS1800, GSM850, GSM850&DCS1800, GSM850&PCS1900, and if the originating cell and the target cell are controlled by different BSCs, the external inter-cell handover is initiated. This measurement provides the number of incoming external inter-cell handover requests. The measurement object is the target cell.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula.

Formula

Incoming External Inter-Cell Handover Requests (1800/1900-900/850)

= [Incoming External Inter-Cell Handover Requests (SDCCH) (1800/1900-900/850)] + [Incoming External Inter-Cell Handover Requests (TCH) (1800/1900-900/850)]
