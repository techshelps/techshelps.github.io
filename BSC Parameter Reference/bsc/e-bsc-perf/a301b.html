Immediate Assignment Commands (Non-Call Type)

Measurement Counter

A301B: CELL_IMM_ASS_CMD_NO_CA

Description

In the immediate assignment procedure, if the BSC receives a CHANNEL ACTIVATION ACKNOWLEDGE message indicating that the signaling channel is ready from the BTS, the BSC sends an IMM ASS CMD message to the MS to instruct the MS to access the specified signaling channel.

After receiving a CHAN RQD message from the BTS, the BSC sends the MS an IMM ASS REJ message if there is no channel available, the channel activation fails, or the channel activation times out. The immediate assignment fails.

This measurement provides the number of IMM ASS CMD messages for non-call purposes.

Unit

Integer number or integer

Measurement Point

As shown in Figure 1, each time the BSC sends an IMM ASS CMD message to an MS in the measured cell (see measurement point B), the counter is incremented by one based on the channel type and the access reason.

Formula

This is an original counter without involving any formula.
