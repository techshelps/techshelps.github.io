Number of Uplink Point-to-Point Short Messages per BSC (SDCCH)

Measurement Counter

ZTA3340A:BSCRPTL_UP_PT_TO_PT_SMS_SD

Description

This measurement provides the number of uplink short messages on the SDCCHs per BSC.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Uplink Point-to-Point Short Messages on SDCCH".

Formula

Number of Uplink Point-to-Point Short Messages per BSC (SDCCH) =Uplink Point-to-Point Short Messages on SDCCH
