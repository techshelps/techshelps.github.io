Number of Uplink Frames Among the EFR Frames in the Customized MR of Cell

Measurement Counter

S3115: CELL_EXT_EFR_TCH_UP_NUM

Description

When decoding the uplink MR, the BSC checks whether the EI_FRAME is contained in the TYPE field of the MR. If the EI_FRAME is found and it contains the EFR frames, this measurement provides the number of uplink frames among the EFR frames in the customized MR.

Unit

Integer number or integer.

Measurement Point

After receiving the MR or pre-processed MR sent from the BTS in the serving cell, the BSC decodes the uplink MR. If the TYPE field in the MR contains the EI_FRAME, and the EI_FRAME contains the EFR frames, this counter is accumulated.

Formula

This is an original counter without involving any formula.
