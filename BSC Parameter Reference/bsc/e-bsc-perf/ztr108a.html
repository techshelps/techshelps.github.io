Traffic Call Drop Ratio on TCH per BSC

Measurement Counter

ZTR108A:BSCRPT_RATE_TCH_TRAF_CALL_DROP

Description

This measurement provides the traffic call drop ratio on the TCHs per BSC, that is, the call duration of average two call drops on TCH in the case of call and handover.

Unit

None.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.

Formula

Traffic Call Drop Ratio on TCH per BSC =

{60} x [Traffic Volume on TCH per BSC (Traffic Channel)]

/

([Call Drops on TCH in Stable State per BSC (Traffic Channel)] +

[Call Drops in TCH Handovers per BSC (Traffic Channel)])
