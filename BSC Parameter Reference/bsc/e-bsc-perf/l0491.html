Number of Configured Cells

Measurement Counter

L0491: CB_BSC_CELL_NO

Description

This measurement provides the number of cells configured in the BSC. The value of the counter is directly obtained from the system.

Unit

Integer number or integer.

Measurement Point

When the number of cells configured in the BSC changes, the value of the counter is updated according to the number of cells actually configured in the BSC.

Formula

This is an original counter without involving any formula.
