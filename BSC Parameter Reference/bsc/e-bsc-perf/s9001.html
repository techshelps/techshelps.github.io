Description

This measurement provides the number of received EGPRS PACKET DOWNLINK ACK messages whose EGPRS BEP Link Quality Measurements contains a field GMSK_MEAN_BEP whose value is N. This counter reflects the air interface quality on the downlink in GMSK modulation mode. The greater the value of N, the better the air interface quality.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure in which the MS sends an EGPRS PACKET DOWNLINK ACK/NACK message to the network. The EGPRS PACKET DOWNLINK ACK/NACK message sent by the MS in GMSK modulation mode carries the GMSK_MEAN_BEP field, which indicates the radio measurement report sent by the MS. Each time the BSC receives the EGPRS PACKET DOWNLINK ACK message and the GMSK_MEAN_BEP field is 0 (see measurement point A), the counter is incremented by one. Each time the BSC receives an EGPRS PACKET DOWNLINK ACK message whose GMSK_MEAN_BEP field is N, the counter is incremented by one. N ranges from 0 to 31.
