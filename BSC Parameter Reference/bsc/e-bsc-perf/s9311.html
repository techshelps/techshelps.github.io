Maximum Number of EGPRS Downlink Timeslots

Measurement Counter

S9311: CELL_USER_EGPRS_TBF_DOWNLINK_MAX_TS_NUM

Description

This measurement provides the maximum number of timeslots that can be allocated to the EGPRS downlink TBFs in transmission state in a granularity period, based on the service type and multislot capability of the MS. This counter is used to calculate Average Maximum Number of Downlink TBF Timeslots.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the BSC periodically samples all the downlink EGPRS TBFs in transmission state. Based on the multislot capability and service type of the MS, the BSC calculates the maximum number of timeslots that can be allocated to the TBFs during each sampling. The calculated value is accumulated to Maximum Number of EGPRS Downlink Timeslots.

Formula

This is an original counter without involving any formula.
