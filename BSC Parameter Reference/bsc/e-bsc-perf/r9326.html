Total Number of Sampled PDCHs Occupied by TBF

Measurement Counter

R9326: TOTAL_ALL_TBF_IN_CELL_OCCU_PDCH_NUM

Description

This measurement provides the total the number of PDCHs occupied by all the TBFs in a cell in a granularity period. For example, a cell has two TBFs; TBF 1 occupies PDCHs 4, 5, 6, and 7, and TBF 2 occupies PDCHs 5, 6, and 7. Thus, the total number of PDCHs occupied by all the TBFs in a cell is seven (If N channels are occupied by a TBF, the value of the counter is N.)

Unit

Integer number or integer.

Measurement Point

In a granularity period, the total number of PDCHs occupied by all the TBFs in a cell is sampled every five seconds and is accumulated to the counter.

Formula

This is an original counter without involving any formula.
