Channel Assignment Overflows in Concentric Cell (TCH)

Measurement Counter

R3225J:CELL_CH_OVERFLOW_TIMES_ULCELL_TRAF_CH

R3224J:CELL_CH_OVERFLOW_TIMES_OLCELL_TRAF_CH

Description

When the BSC needs to assign an MS a TCH in the assignment, immediate assignment, internal handover, or external handover procedure, it fills the concentric attribute in the channel request based on the data already configured, and then initiates the channel assignment procedure. The counters are used to measure the number of times the underlaid subcell or the overlaid subcell has no idle TCH to be assigned.

Unit

Integer number or integer.

Measurement Point

In the TCH assignment, the measurements are triggered when the underlaid subcell or the overlaid subcell has no idle TCH to be assigned.

Formula

This is an original counter without involving any formula.
