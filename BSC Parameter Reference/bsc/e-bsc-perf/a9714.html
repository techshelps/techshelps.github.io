Processing Times of PS Services by MSs Supporting Multislot Class X in DTM EGPRS

Measurement Counter

A9714: CELL_DTM EGPRS MULTISLOT CLASS5

A9715: CELL_DTM EGPRS MULTISLOT CLASS6

A9716: CELL_DTM EGPRS MULTISLOT CLASS9

A9717: CELL_DTM EGPRS MULTISLOT CLASS10

A9718: CELL_DTM EGPRS MULTISLOT CLASS11

A9728: CELL_DTM_EGPRS_HIGH_MULTISLOT_CLASS31_36

A9729: CELL_DTM_EGPRS_HIGH_MULTISLOT_CLASS32_37

A9730: CELL_DTM_EGPRS_HIGH_MULTISLOT_CLASS33_38

A9731: CELL_DTM_EGPRS_HIGH_MULTISLOT_CLASS41

A9732: CELL_DTM_EGPRS_HIGH_MULTISLOT_CLASS42

A9733: CELL_DTM_EGPRS_HIGH_MULTISLOT_CLASS43

A9734: CELL_DTM_EGPRS_HIGH_MULTISLOT_CLASS44

Description

This measurement provides the number of times MSs supporting multislot class x in DTM EGPRS process PS services in a cell.

Unit

Integer number or integer.

Measurement Point

After the MS processes PS services and the MS context is released, this measurement is triggered according to the MS RA capability information element saved in the MS context. Note: In "Multislot class x", "x" can be 5, 6, 9, 10, 11, 31 and 36, 32 and 37, 33 and 38, 41, 42, or 43 and 44.

Formula

This is an original counter without involving any formula.
