Failed Assignments per BSC (Equipment Failure)

Measurement Counter

ZTA312L:BSCRPT_TOTALCELL_ASS_FAIL_Frst_APPLY_TRSL_FAIL

Description

In the assignment procedure, the BSS selects different traffic channels or signaling channels after the MSC sends an ASS REQ message to the BSS. Because of various causes, the assignment cannot be successfully completed.

This measurement provides the number of assignment failures due to equipment failure in all the cells controlled by a BSC.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Failed Assignments per Cell" and "Failed Mode Modify Attempts per Cell".

Formula

Failed Assignments per BSC (Equipment Failure) = [Failed Assignments (A Interface Failure)] +

[Failed Mode Modify Attempts (A Interface Failure)] + [ Failed Assignments (First Assignment, Terrestrial Resource Request Failed)] + [ Failed Assignments (Reconnection to Old Channels, Terrestrial Resource Request Failed)] + [ Failed Mode Modify Attempts (Terrestrial Resource Request Failed)]
