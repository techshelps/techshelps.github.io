MS Power Level of AMR Call

Measurement Counter

S3240A:CELL_AMR_MS_PWR_LEV

Description

This measurement provides the total MS power level of the AMR calls in the cell within the granularity period.

Unit

Integer number or integer.

Measurement Point

During an AMR call, when the BSC receives a valid MR or pre-processed MR that contains the MS power level from the BTS in the serving cell, this counter is accumulated.

Formula

This is an original counter without involving any formula.
