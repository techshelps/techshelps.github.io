Channel Activation Timeouts in Assignment Procedure per BSC

Measurement Counter

ZTA3159E:BSCRPT_TOTALCELL_CHAN_ACT_TIMEOUT_TIMES_TCH

Description

In the assignment procedure, the BSC sends a CHAN ACTIV message to the BTS and starts a timer (interior, five seconds by default) to wait for a CHAN ACTIV ACK message. If the BSC cannot receive the CHAN ACTIV ACK message when the timer expires, it collects the counters Channel Activation Timeouts in Assignment Procedure (TCHF), Channel Activation Timeouts in Assignment Procedure (TCHH), Channel Activation Timeouts in Assignment Procedure (TCHH), and Channel Activation Timeouts in Mode Modify Procedure (TCHH) in the cell where the TCH resides according to channel types. This measurement provides the sum of the preceding counters in all cells.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Channel Activation Timeouts in Assignment Procedure" and "Channel Activation Timeouts in Mode Modify Procedure".

Formula

Channel Activation Timeouts in Assignment Procedure per BSC = [ Channel Activation Timeouts in Assignment Procedure (TCHF)] + [ Channel Activation Timeouts in Assignment Procedure (TCHH)] + [ Channel Activation Timeouts in Mode Modify Procedure (TCHF)] + [ Channel Activation Timeouts in Mode Modify Procedure (TCHH)]
