Uplink LLC Throughput Rate (Kbit/s)

Measurement Counter

TL9406: CELL_LLC_LLC_UP_THRPUT_RATE

Description

This measurement counter provides the average rate of uplink LLC data in a granularity period. In the case of the same traffic model and traffic volume, this measurement counter indicates the transmission performance of a cell.

Unit

Kbit/s.

Measurement Point

None.

Formula

Uplink LLC Throughput Rate (Kbit/s) = [Total Bytes of Uplink LLC PDUs Sent]x{8}/({1024}x{GP}x{60})
