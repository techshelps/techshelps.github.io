Channel Activation Timeout During VGCS Assignment (TIMEOUT)

Measurement Counter

A03415: CELL_VGCS_ASS_FAIL_EXP

Description

After receiving the VGCS ASSIGNMENT REQUEST message from the MSC, the BSC sends a CHANNEL ACTIVATION message to the BTS. This measurement provides the number of times that the BSC does not receive the CHANNEL ACTIVATION ACK message from the BTS before the timer expires.

Unit

Integer number or integer.

Measurement Point

In the VGCS assignment procedure, the BSC sends the CHANNEL ACTIVATION message for the newly assigned TCH to the BTS. This counter is incremented by one when the BSC does not receive a CHANNEL ACTIVATION ACKNOWLEDGE message before the timer expires.
