Attempted Handovers per BSC (Intra-Cell TCHF-to-TCHH Handover)

Measurement Counter

ZCH9005:BSCRPT_HO_REQ_AMR

Description

This measurement provides the number of internal intra-cell handover requests (AMR) (TCHF-TCHH) and the number of internal intra-cell handover requests (AMR) (TCHH-TCHF) of all the cells controlled by a BSC.

This counter shows the frequency of TCHF-to-TCHH handover requests initiated by the current cell. According to the counters Internal Intra-Cell Handover Commands per BSC and Successful Internal Intra-Cell Handovers per BSC, the intra-cell TCHF-to-TCHH handover per BSC can be well understood.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.

Formula

Attempted Handovers per BSC (Intra-Cell TCHF-to-TCHH Handover) =Internal Intra-Cell Handover Requests (AMR) (TCHF-TCHH) +Internal Intra-Cell Handover Requests (AMR) (TCHH-TCHF)
