Total Number of Valid Uplink EGPRS MCS1-MCS9 RLC Data Blocks

Measurement Counter

L9211: UP_EGPRS_MCS1_RLC_DATA_BLK_VALID_TOTAL

L9212: UP_EGPRS_MCS2_RLC_DATA_BLK_VALID_TOTAL

L9213: UP_EGPRS_MCS3_RLC_DATA_BLK_VALID_TOTAL

L9214: UP_EGPRS_MCS4_RLC_DATA_BLK_VALID_TOTAL

L9215: UP_EGPRS_MCS5_RLC_DATA_BLK_VALID_TOTAL

L9216: UP_EGPRS_MCS6_RLC_DATA_BLK_VALID_TOTAL

L9217: UP_EGPRS_MCS7_RLC_DATA_BLK_VALID_TOTAL

L9218: UP_EGPRS_MCS8_RLC_DATA_BLK_VALID_TOTAL

L9219: UP_EGPRS_MCS9_RLC_DATA_BLK_VALID_TOTAL

Description

This measurement provides the total number of valid uplink RLC data blocks using MCS-1 to MCS-9 in a cell in a granularity period. The more robust coding scheme, the more data transmitted on each data block. High proportion of valid downlink RLC data blocks indicates few retransmissions of uplink RLC data blocks.

Unit

Integer number or integer.

Measurement Point

In a granularity period, if the BSC receives an uplink EGPRS RLC data block using MCSx coding scheme from the MS for the first time and the data block in the V(N) on the BSC side is in NACKED, UNRECEIVED, or RECEIVED state, you can infer that there is a valid data block. In this case, the counter is incremented by one. Note: MCSx indicates MCS1-MCS9.

Formula

This is an original counter without involving any formula.
