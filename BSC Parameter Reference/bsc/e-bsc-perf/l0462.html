RESET Requests Received by BSC

Measurement Counter

L0462:CB_BSC_RST_CMD

Description

This counter is used to measure the number of times the BSC receives a RESET request on the BSC-CBC interface.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the BSC receives a RESET request from the CBC.

Formula

This is an original counter without involving any formula.
