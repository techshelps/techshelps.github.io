TA during Radio Link Failure (TCHF)

Measurement Counter

S4347D:TRX_FR_DROP_TA

Description

After receiving a valid MR or pre-processed MR, the BSC takes statistics of some parameter values based on the requirements. The statistics can help users better understand the network performance. The measurement provides the accumulated TA value reported by the MRs during the radio link failure on the TCHF. This counter is used to calculate the counter AS4347D: TRX_FR_DROP_TA_AVR.

Unit

TA.

Measurement Point

Within the granularity period, this counter is incremented by the TA value in the last valid MR or pre-processed MR before the radio link failure occurs on the TCHF.

Formula

This is an original counter without involving any formula.
