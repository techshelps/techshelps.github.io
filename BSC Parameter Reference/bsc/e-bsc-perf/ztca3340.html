Number of Point-to-Point Short Messages on the SDCCH per BSC

Measurement Counter

ZTCA3340:BSCRPT_PT_TO_PT_SMS_SD

Description

This measurement provides the number of uplink and downlink short messages on the SDCCHs per BSC.

Unit

Integer number or integer.

Measurement Point

BSC counters are calculation counters. This counter is measured according to the formula below when the measurement period ends. For details on the measurement point, see "Uplink Point-to-Point Short Messages on SDCCH" and "Downlink Point-to-Point Short Messages on SDCCH".

Formula

Number of Point-to-Point Short Messages on the SDCCH per BSC = [Number of Uplink Point-to-Point Short Messages per BSC (SDCCH)] + [Number of Downlink Point-to-Point Short Messages per BSC (SDCCH)]
