Number of MRs based on TA per TRX

Description

Number of MRs based on TA per TRX provides the number of MRs from the BTS with a certain TA range. This provides the attribution of traffic volume based on the distance between the MS and the BTS and facilitates network maintenance and optimization.

Measurement Object

TRX.
