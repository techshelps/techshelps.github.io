Received PS Packets

Measurement Counter

S4004A: TRX_RX_PS_LOST

Description

This measurement provides the number of PS packets that the BSC sends to the TRX over the HDLC path but get lost due to transmission congestion or abnormal communications.

Unit

Integer number or integer.

Measurement Point

The logical processing module of the TRX compares the sequence number of the received PS packets with that received before. If the sequence number of the received CS packets is discontinuous, it means packets are lost, and the TRX performs the measurement accordingly. The BTS periodically reports the result, and the BTSM disassembles the packets and processes the measurement results.

Formula

This is an original counter without involving any formula.
