Number of Successful Downlink Assignments

Measurement Counter

A9508: SUCC_DOWN_ASS

A9510: SUCC_DOWN_IMM_ASS

A9512: PACCH_SUCC_DOWN_ASS

Description

These three measurements provide the number of successful downlink assignment messages in a granularity period. This measurement indicates the GPRS QoS of a cell. A low measurement value indicates that the radio environment, network parameter configuration, network resource configuration, or network equipment is not satisfactory, and thus requires optimization.

Unit

Integer number or integer.

Measurement Point

The successful downlink assignment involves the following aspects:

  • Packet immediate assignment on the CCCH

    The BSC initiates the packet immediate assignment on the CCCH and sends a PACKET POLLING REQUEST message on the assigned channel. If the BSC receives the PACKET CONTROL ACKNOWLEDGEMENT from the MS, it indicates that the MS has successfully received the downlink packet immediate assignment message and accepted the assignment from the BSC. Figure 1 shows the downlink packet immediate assignment procedure initiated by the BSC. Each time the BSC receives a PACKET CONTROL ACKNOWLEDGEMENT message from the MS (see measurement point B), the counter is incremented by one.

    Figure 1 Downlink packet immediate assignment

  • Downlink packet assignment on the PACCH

    If the BSC receives the PACKET CONTROL ACKNOWLEDGEMENT message from the MS in response to the PACKET DOWNLINK ASSIGNMENT message, it indicates that the MS has successfully received the downlink packet assignment message and accepted the assignment from the BSC. Figure 2 shows the procedure for the BSC initiating the downlink packet assignment on the PACCH. Each time the BSC receives a PACKET CONTROL ACKNOWLEDGEMENT message from the MS (see measurement point B), the counter is incremented by one.

    Figure 2 Downlink assignment on the PACCH

Formula

This is an original counter without involving any formula.
