TCH Seizure Requests per BSC (Traffic Channel)

Measurement Counter

ZK3010A:BSCRPT_KPI_TCH_ASS_REQ_TRAF

Description

This measurement provides the number of TCH seizures of the ASSIGNMENT REQUEST messages received by the BSC from the MSC in the assignment procedure.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Assignment Requests".

Formula

TCH Seizure Requests per BSC (Traffic Channel) =

[Assinment Request Times(Sig Channel)(TCH) Per BSC] +

[Assignment Requests (TCHF Only) Per BSC] +

[Assignment Requests (TCHH Only) Per BSC]

[Assignment Requests (TCHF Preferred, Channel Type Unchangeable) Per BSC] +

[Assignment Requests (TCHH Preferred, Channel Type Unchangeable) Per BSC] +

[Assignment Requests (TCHF or TCHH, Channel Type Unchangeable) Per BSC] +

[Assignment Requests (TCHF Preferred, Channel Type Changeable) Per BSC] +

[Assignment Requests (TCHH Preferred, Channel Type Changeable) Per BSC] +

[Assignment Requests (TCHF or TCHH, Channel Type Changeable) Per BSC]
