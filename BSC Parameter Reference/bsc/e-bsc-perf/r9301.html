Sampling Times of PDCH Measurement

Measurement Counter

R9301: CELL_PDCH_SAMPLE_PDCH

Description

This measurement provides the sampling times on the PDCH in a granularity period. This measurement is used to calculate the average value of the PDCH measurements.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the PDCH is sampled every five seconds. The counter is incremented by one every time the PDCH is sampled.

Formula

This is an original counter without involving any formula.
