Throughput of Uplink EGPRS Interactive Service THPx LLC PDU

Measurement Counter

L9416: CELL_LLC_TX_UP_EGPRS_INTACT_THP1_LLC_PDU_THRPUT_RATE

L9435: CELL_LLC_TX_UP_EGPRS_INTACT_THP2_LLC_PDU_THRPUT_RATE

L9436: CELL_LLC_TX_UP_EGPRS_INTACT_THP3_LLC_PDU_THRPUT_RATE

Description

This measurement provides the throughput of uplink EGPRS interactive service THPx LLC PDUs in a granularity period.

Unit

Kbit x Kbit/s.

Measurement Point

In a granularity period, every time the uplink EGPRS TBF completes the transmission of interactive THPx service, the throughput of LLC PDUs is calculated. The throughput is equal to the number of transmitted bytes divided by the transmission duration. This throughput multiplied by the number of transmitted bytes is equal to the throughput of transmitted LLC PDUs. In the end, the throughput is accumulated to the value of the counter. Note: THPx indicates THP1-THP3.

Formula

This is an original counter without involving any formula.
