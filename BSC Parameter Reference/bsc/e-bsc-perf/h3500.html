Measurement of Outgoing Inter-RAT Inter-Cell Handover Requests

Measurement Counter

H3500:CELL_INTERRAN_OUTCELL_HO_REQ_NOT_INCLUDE_DR_SD

H3507:CELL_INTERRAN_OUTCELL_HO_REQ_TCHF

H3508:CELL_INTERRAN_OUTCELL_HO_REQ_TCHH

H353a:CELL_INTERRAN_OUTCELL_HO_REQ_BSC_LOOP

H353b:CELL_INTERRAN_OUTCELL_HO_REQ_BTS_LOOP

Description

The BSC decides whether to initiate a handover based on the MR and the handover algorithm. If the target cell of the handover is a cell in a different network system, an outgoing inter-RAT inter-cell handover is initiated. The specific counter provides the number of outgoing inter-RAT inter-cell handover requests in the originating cell initiated based on the handover algorithm. Together with the Outgoing Inter-RAT Inter-Cell Handover Commands and the Failed Outgoing Inter-RAT Inter-Cell Handovers, this counter determines the performance of the outgoing inter-RAT inter-cell handover.

The following counter provides the number of outgoing inter-RAT inter-cell handover requests (excluding directed retry) if the MS is seizing an SDCCH before the handover:
  • H3500:CELL_INTERRAN_OUTCELL_HO_REQ_NOT_INCLUDE_DR_SD
The following two counters provide respectively the number of outgoing inter-RAT inter-cell handover requests when the MS seizes a TCHF or a TCHH before the handover. In this measurement, both the handovers initiated by directed retry and those initiated according to the handover algorithms based on the measurement reports are included.
  • H3507:CELL_INTERRAN_OUTCELL_HO_REQ_TCHF
  • H3508:CELL_INTERRAN_OUTCELL_HO_REQ_TCHH

If the BSC enables the local switching function, then the calls that meet the local switching condition of the BSC use the local switching function of the BSC for the conversation and those that meet the local switching condition of the BTS use the local switching function of the BTS for the conversation.

The following counters provide the numbers of outgoing inter-RAT inter-cell handover requests respectively from the calls using the BSC local switch and those using the BTS local switch. They are measured when the outgoing inter-RAT inter-cell handover is initiated.
  • H353a:CELL_INTERRAN_OUTCELL_HO_REQ_BSC_LOOP
  • H353b:CELL_INTERRAN_OUTCELL_HO_REQ_BTS_LOOP

Unit

Integer number or integer.
