Number of Received Out-of-Synchronization TRAU Frames

Measurement Counter

L9A02: CELL_GABIS_TRAU_RX_OUT_OF_SYN_FR

Description

This measurement provides the number of out-of-synchronization frames received by a cell in a granularity period. This counter indicates the transmission quality of the G-Abis interface. Usually, the satellite and microwave transmission may cause out-of-synchronization frames.

Unit

Integer number or integer.

Measurement Point

After the BSC receives an uplink TRAU frame, this frame needs to be checked. If the synchronization head of this frame is not found, the counter is incremented by one (see measurement point A shown in Figure 1).

Figure 1 Measurements of uplink TRAU frames

Formula

This is an original counter without involving any formula.
