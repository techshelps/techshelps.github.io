Number of Users Assigned to the MSC According to IMSI Load Sharing (Non-Paging Response Access)

Measurement Counter

L0057A: BS_USR_NUM_BY_IMSI_NOT_PAGRSP

Description

This measurement provides the number of Users (non-paging response access) that are assigned to the proper MSC according to the IMSI load sharing with the MSC pool enabled. This counter indicates the number of Users (non-paging response access) that are selected to use the proper method to obtain the DPC according to the IMSI load sharing with the MSC pool enabled.

Unit

Integer number or integer.

Measurement Point

With the MSC pool enabled, every time the BSC receives the acknowledgement message of establishing the A interface connection from the MSC (see measurement point A in Figure 1), the counter is incremented by one if the DPC is obtained according to the value V (the DPC algorithm is selected according to the IMSI load sharing), and the call setup indication message received by the BSC contains IMSI.

Figure 1 Measurement point of the acknowledgement message of the A interface connection received by the BSC with the MSC pool enabled

Formula

This is an original counter without involving any formula.
