Average Interval of Packet Access Requests on CCCH (s)

Measurement Counter

AA9407: CCCH_PS_ACCESS_REQ_AVG_REACH_TIME

Description

This measurement provides the average interval of packet access requests received on the CCCHs in a cell in a granularity period. Lower value of the counter indicates more packet access requests received on the CCCHs in a cell.

Unit

s.

Measurement Point

None.

Formula

Average Interval of Packet Access Requests on CCCH (s) = (GP x 60/[Number of 8-Bit Packet Channel Requests Received on CCCH] + [Number of 11-Bit Packet Channel Requests Received on CCCH])

GP indicates the measurement granularity, which is expressed in minute.
