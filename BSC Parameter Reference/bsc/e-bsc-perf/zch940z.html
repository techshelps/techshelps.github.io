Attempted Incoming BSC Handovers per BSC (1800/1900 to 900/850)

Measurement Counter

ZCH940Z:BSCRPT_INBSC_HO_REQ_1800_900

Description

This measurement provides the number of attempted incoming BSC handovers per BSC (1800/1900 to 900/850).

If the source cell and the target cell are controlled by different BSCs, the external BSC handover is initiated. This counter is measured based on the channel type and seized TCH channel type.

The channel involved in the measurement can be the SDCCH or the TCH.

This counter shows the frequency of handovers from other BSCs to the local BSC. According to the counter Successful Incoming BSC Handovers per BSC (1800/1900 to 900/850), the incoming BSC handovers can be well understood.

The handover frequency band range is 1800/1900 to 900/850.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.

Formula

Attempted Incoming BSC Handovers per BSC (1800/1900 to 900/850) =

Incoming External Inter-Cell Handover Requests (SDCCH) (1800/1900-900/850) +

Incoming External Inter-Cell Handover Requests (TCH) (1800/1900-900/850)
