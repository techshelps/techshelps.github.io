Number of Received Disorderly PTRAU Packets

Measurement Counter

L9B04:TRX_GABIS_PTRAU_DISORDER_PACKETS

Description

This measurement provides the number of disordered PTRAU packets received by a TRX in a granularity period on the IP or HDLC network topology. This counter indicates the link quality in the IP or HDLC network topology. For example, in the IP network topology, IP packets may be in disorder when they are sent through different routes.

Unit

Integer number or integer.

Measurement Point

The following figure shows the process where the BSC receives the disordered PTRAU packet. In a granularity period, the counter is incremented by one every time the BSC detects a disordered PTRAU packet (see measurement point A in Figure 1).

Figure 1 Disordered PTRAU packets received on the G-Abis interface

Formula

This is an original counter without involving any formula.
