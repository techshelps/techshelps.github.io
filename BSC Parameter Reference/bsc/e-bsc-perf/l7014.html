Received DT1 Messages

Measurement Counter

L7014:SCCP_DT1_RX

Description

This meausrement provides the number of times the SCCP module receives a DT1 message.

Unit

Integer number or integer.

Measurement Point

The measurement point is on the BSC side, as shown in the figure.

The measurement is triggered when the SCCP module receives a DT1 message.



Formula

This is an original counter without involving any formula.
