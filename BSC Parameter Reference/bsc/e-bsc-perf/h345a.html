Incoming External Inter-Cell Handover Clear Cmd Times (Before Handover Response)

Measurement Counter

H345A:CELL_INTERBSC_INCELL_CLEAR_CMD_BEFORE_HO_CMD

Description

During the incoming external inter-cell handover procedure, if the BSC receives a CLEAR CMD message before sending the HO REQ ACK message to the MSC, the BSC releases the call to terminate the handover. This counter provides the number of CLEAR CMD messages received before the HO REQ ACK message is sent.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, this counter is measured at D1 when the BSC receives the CLEAR CMD message from the MSC before the BSC sends the HO REQ ACK message to the MSC.

Figure 1 Procedure of clearing the call during the incoming external inter-cell handover

Formula

This is an original counter without involving any formula.
