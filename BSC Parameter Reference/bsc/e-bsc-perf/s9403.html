On-Line EGPRS Users per BSC

Measurement Counter

S9403:PSALL_USERCAP_BSC_EGPRS_USERS

Description

This measurement provides the total number of online EGPRS MSs in the BSC in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the number of online EGPRS MSs in the BSC is sampled periodically. The sampled value is accumulated to the counter. The sampling period is 5s.

Formula

This is an original counter without involving any formula.
