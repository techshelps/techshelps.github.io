Uplink Point-to-Point Short Messages on SDCCH

Measurement Counter

A3340A:CELL_UL_Pt_to_Pt_SMS_SD

Description

The uplink point-to-point short message refers to the short message from the MS to the MSC.

This measurement provides the number of uplink short messages on the SDCCH of a cell.

Unit

Integer number or integer.

Measurement Point

The counter is incremented by one when the BSC receives an uplink short message on the SDCCH.

Formula

This is an original counter without involving any formula.
