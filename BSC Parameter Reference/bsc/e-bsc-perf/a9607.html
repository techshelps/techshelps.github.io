Number of Packet Notification Sent on DCCH

Measurement Counter

A9607: DCCH_PACKET_NOTIFICATION_PAGING_MSG

Description

This measurement counter provides the number of packet paging messages sent on the DCCH in a granularity period.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure for the BSC sending a packet paging message on the DCCH. Each time the BSC sends a packet paging message on the DCCH (see measurement point A), the counter is incremented by one. This measurement is triggered when the MS supports the DTM. If the MS is processing CS services, after receiving the PACKET NOTIFICATION PAGING MAG, the BSC sends the packet paging message on the DCCH if the conditions are met.

Figure 1 Packet paging messages sent on the DCCH

Formula

This is an original counter without involving any formula.
