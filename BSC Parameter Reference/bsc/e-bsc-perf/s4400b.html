Description

The measurement of preceding counters provides the number of MRs with a certain TA or TA range when the radio link failure occurs. The TA ranges from 0 to 63.

Unit

Integer number or integer.

Measurement Point

In the granularity period, when the radio link fails, the specific counter is incremented by one based on the TA of the last valid MR or pre-processed MR before the radio link failure occurs.

Formula

This is an original counter without involving any formula.
