Total Bytes of Uplink GPRS Interactive Service THP1 LLC PDUs Sent

Measurement Counter

L9408: CELL_LLC_TX_UP_GPRS_INTACT_THP1_LLC_PDU_TOTAL_BYTE

L9429: CELL_LLC_TX_UP_GPRS_INTACT_THP2_LLC_PDU_TOTAL_BYTE

L9430: CELL_LLC_TX_UP_GPRS_INTACT_THP3_LLC_PDU_TOTAL_BYTE

Description

This measurement provides the total bytes of uplink GPRS interactive service LLC PDUs in a granularity period.

Unit

byte.

Measurement Point

In a granularity period, every time the BSC sends an UPLINK LLC PDU to the SGSN (see measurement point A in Figure 1), the bytes of the UPLINK LLC PDU are accumulated to the value of the counter, provided that the TBF is a GPRS TBF supporting THP and the LLC PDU is an interactive service THPx LLC PDU. Note: THPx indicates THP1-THP3.

Figure 1 Uplink LLC PDU sent to the SGSN

Formula

This is an original counter without involving any formula.
