Number of RADIO STATUS PDUs Sent by BSSGP Sublayer

Measurement Counter

L9904:CELL_BSSGP_TRAN_TX_RADIO_STATUS_PDU

Description

This measurement provides the number of RADIO STATUS PDUs sent by a cell in a granularity period. If any of the following situations occurs, the BSS sends the RADIO STATUS PDU to the SGSN:

  • The network loses the connection with the MS.

  • The GPRS services cannot be processed due to the poor quality of the link.

  • The network-controlled cell reselection occurs.

  • The MS fails to reselect a cell.

Unit

Integer number or integer.

Measurement Point

The following figure shows the process where the BSC sends the RADIO STATUS PDU to the SGSN. In a granularity period, every time the BSC sends a RADIO STATUS PDU (see measurement point A in Figure 1), the counter is incremented by one.

Figure 1 RADIO STATUS PDU sent by the BSSGP sublayer

Formula

This is an original counter without involving any formula.
