Sampled Measurement of Initially Configured Channels (DYN EDGE)

Measurement Counter

R3006A:CELL_CH_CFG_NUM_DYN_EDGE_900

R3006B:CELL_CH_CFG_NUM_DYN_EDGE_1800

Description

R3006A:CELL_CH_CFG_NUM_DYN_EDGE_900 provides the number of configured dynamic EDGE PDCHs in the GSM900, GSM900/DCS1800, GSM850, GSM850/DCS1800, and GSM850/PCS1900 cells controlled by the BSC at each sampling point. This number includes the number of dynamic EDGE PDCHs that are dynamically converted into TCHFs.

R3006B:CELL_CH_CFG_NUM_DYN_EDGE_1800 provides the number of configured dynamic EDGE PDCHs in the DCS1800 and PCS1900 cells controlled by the BSC at each sampling point. This number includes the number of dynamic EDGE PDCHs that are dynamically converted into TCHFs.

Unit

Integer number or integer.

Measurement Point

The BSC takes the samples of the channel allocation status every five seconds. When the sampling timer expires, the specific counter R3006A:CELL_CH_CFG_NUM_DYN_EDGE_900 or R3006B:CELL_CH_CFG_NUM_DYN_EDGE_1800 is measured according to the frequency band of the cell and the number of configured dynamic EDGE PDCHs in the cell.

Formula

This is an original counter without involving any formula.
