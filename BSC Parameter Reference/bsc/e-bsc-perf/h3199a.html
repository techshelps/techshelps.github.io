Intra-Cell Non-AMR Handover

Measurement Counter

H3199A:CELL_INTRACELL_HO_NO_TRIG_NONE_AMR_TCHF_TCHH

H3199B:CELL_INTRACELL_HO_NO_TRIG_NONE_AMR_TCHH_TCHF

H3200A:CELL_INTRACELL_HO_REQ_NONE_AMR_TCHF_TCHH

H3200B:CELL_INTRACELL_HO_REQ_NONE_AMR_TCHH_TCHF

H3201A:CELL_INTRACELL_HO_CMD_NONE_AMR_TCHF_TCHH

H3201B:CELL_INTRACELL_HO_CMD_NONE_AMR_TCHH_TCHF

H3202A:CELL_INTRACELL_HO_FAIL_NONE_AMR_TCHF_TCHH

H3202B:CELL_INTRACELL_HO_FAIL_NONE_AMR_TCHH_TCHF

Description

For a non-AMR call, when the BSC determines that an intra-cell handover from TCHF to TCHH is required based on the handover algorithm, handover indication is ignored if the MSC does not allow any change in channel type during the call establishment. In this situation, the BSC measures the "H3199A:CELL_INTRACELL_HO_NO_TRIG_NONE_AMR_TCHF_TCHH" counter.

For a non-AMR call, when the BSC determines that an intra-cell handover from TCHH to TCHF is required based on the handover algorithm, handover indication is ignored if the MSC does not allow any change in channel type during the call establishment. In this situation, the BSC measures the H3199B:CELL_INTRACELL_HO_NO_TRIG_NONE_AMR_TCHH_TCHF counter.

For a non-AMR call, when the BSC determines that an intra-cell handover from TCHF to TCHH is required based on the handover algorithm, the BSC measures the H3200A:CELL_INTRACELL_HO_REQ_NONE_AMR_TCHF_TCHH counter if the handover is allowed.

For a non-AMR call, when the BSC determines that an intra-cell handover from TCHH to TCHF is required based on the handover algorithm, the BSC measures the H3200B:CELL_INTRACELL_HO_REQ_NONE_AMR_TCHH_TCHF counter if the handover is allowed.

For a non-AMR call, if the BSC determines that an intra-cell handover from TCHF to TCHH is required based on the handover algorithm, the BSC measures the H3201A:CELL_INTRACELL_HO_CMD_NONE_AMR_TCHF_TCHH counter when the handover procedure is initiated and a handover command is issued.

For a non-AMR call, if the BSC determines that an intra-cell handover from TCHH to TCHF is required based on the handover algorithm, the BSC measures the H3201B:CELL_INTRACELL_HO_CMD_NONE_AMR_TCHH_TCHF counter when the handover procedure is initiated and a handover command is issued.

For a non-AMR call, if the BSC determines that an intra-cell handover from TCHF to TCHH is required based on the handover algorithm, the BSC measures the H3202A:CELL_INTRACELL_HO_FAIL_NONE_AMR_TCHF_TCHH counter when the handover is initiated but the handover fails due to a certain cause.

For a non-AMR call, if the BSC determines that an intra-cell handover from TCHH to TCHF is required based on the handover algorithm, the BSC measures the H3202B:CELL_INTRACELL_HO_FAIL_NONE_AMR_TCHH_TCHF counter when the handover is initiated but the handover fails due to a certain cause.

Unit

Integer number or integer.

Measurement Point

For the description of the Measurement Points of the counters, refer to the Measurement Points of the common handovers.

Formula

Times.
