Number of Downlink 8PSK_MEAN_BEP Different Value Measurement (Cell)

Description

The measurement counters listed here indicate the traffic load in a cell and the quality of the downlink air interface, which serve as the reference for establishing the service model. The following table lists the measurement counters related to the PSCALL.8pskMBepDifNum.Cell measurement unit. Except the 8PSK_MEAN_BEP with different values, the introduction, measurement points, and formulas of these measurement counters are the same. Therefore, only 8PSK_MEAN_BEP=0 is taken as an example and described in details.

Measurement Object

Cell.
