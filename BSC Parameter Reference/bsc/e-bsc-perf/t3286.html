Description

This measurement provides the number of the measurements of original MR or pre-processed MR whose downlink VQI ranges from M to N (0.0-0.5, 0.5-1.0, 1.0-1.5, 1.5-2.0, 2.0-2.5, 2.5-3.0, 3.0-3.5, 3.5-4.0, 4.0-4.5, and 4.5-5.0) based on full-rate speech version.

Unit

Integer number or integer.

Measurement Point

On receiving the pre-processed MR or original MR, the BSC checks whether there is the downlink VQI value in the extended information, and the current speech version is full-rate. If so, the BSC triggers the measurement based on the downlink VQI ranging from M to N (0.0–0.5, 0.5–1.0, 1.0–1.5, 1.5–2.0, 2.0–2.5, 2.5–3.0, 3.0–3.5, 3.5–4.0, 4.0–4.5, and 4.5–5.0).

Formula

This is an original counter without involving any formula.
