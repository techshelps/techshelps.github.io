Total Number of Uplink EGPRS TBFs

Measurement Counter

L9234: UP_EGPRS_TOTAL_TBF

Description

This measurement provides the total number of uplink EGPRS TBFs in a cell in a granularity period.

Unit

Integer number or integer.

Measurement Point

In a granularity period, every time the BSC establishes an uplink EGPRS TBF, the counter is incremented by one. The successful uplink EGPRS TBF establishment involves the following conditions:

After sending an IMMEDIATE ASSIGNMENT message, if the BSC receives the uplink blocks on the channel allocated to the MS, it indicates that the uplink EGPRS TBF establishment using one phase access is successful. Figure 1 shows the procedure of uplink EGPRS TBF establishment using one phase access. Every time the BSC receives the first uplink block from the MS after sending an IMMEDIATE ASSIGNMENT message (see measurement point A), the counter is incremented by one.

Figure 1 Successful uplink EGPRS TBF establishment using one-phase access

After sending a PACKET UPLINK ASSIGNMENT message, if the BSC receives the uplink data blocks on the channel allocated to the MS, it indicates that the uplink EGPRS TBF establishment using single-block access is successful. Figure 2 shows the procedure of uplink EGPRS TBF establishment using single-block access. Every time the BSC receives the first uplink block from the MS after sending a PACKET UPLINK ASSIGNMENT message (see measurement point A), the counter is incremented by one.

Figure 2 Successful uplink EGPRS TBF establishment using single-block access

If the MS initiates an uplink EGPRS TBF establishment request on the PACCH, the BSC receives the uplink data blocks on the channel allocated to the MS after sending a PACKET UPLINK ASSIGNMENT message, it indicates that the uplink EGPRS TBF establishment on the PACCH is successful. Figure 3 shows the procedure of uplink EGPRS TBF establishment on the PACCH. Every time the BSC receives an uplink data block from the MS after sending a PACKET UPLINK ASSIGNMENT message (see measurement point A), the counter is incremented by one.

Figure 3 Successful uplink EGPRS TBF establishment on the PACCH

Formula

This is an original counter without involving any formula.
