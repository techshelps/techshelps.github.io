Discarded Channel Requests (Location Updating)

Measurement Counter

L3189E: CELL_DISC_CH_REQ_LOC_UPDATE

Description

This measurement provides the number of discarded channel requests (location updating) during flow control in a cell.

Unit

Times.

Measurement Point

This measurement is triggered each time the BSC discards a channel request (location updating) during flow control.

Formula

This is an original counter without involving any formula.
