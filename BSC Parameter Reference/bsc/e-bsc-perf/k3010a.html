TCH Seizure Requests (Traffic Channel)

Measurement Counter

K3010A:CELL_KPI_TCH_ASS_REQ_TRAF

Description

This measurement provides the number of TCH (traffic channel) seizure requests received by the BSC from the MSC.

Unit

Integer number or integer.

Measurement Point

In the assignment procedure, the BSC receives the ASSIGNMENT REQUEST message from the MSC and measures this counter if the required channel is the TCH. See measurement point A in Figure 1.

Figure 1 Assignment procedure

Formula

TCH Seizure Requests (Traffic Channel) =
