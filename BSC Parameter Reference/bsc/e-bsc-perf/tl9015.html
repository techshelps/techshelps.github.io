Average Payload of Single Uplink GPRS TBF (KB)

Measurement Counter

TL9015: UP_GPRS_RLC_SGL_TBF_AVG_THRPUT

Description

This measurement provides the average payload of a single GPRS TBF at the uplink RLC in a cell in a granularity period. This measurement indicates the type of the service processed by the user. The measurements of uploading and downloading services are higher than those of the WAP services.

Unit

kbyte.

Measurement Point

The value of Average Payload of Single Uplink GPRS TBF (KB) is calculated according to the total number of uplink RLC data blocks with CS-1, CS-2, CS-3, and CS-4 coding schemes and the number of uplink GPRS TBFs in a granularity period.

Formula

Average Payload of Single Uplink GPRS TBF (KB) = (Total Number of Uplink RLC Data Blocks Using CS1 x 23 + Total Number of Uplink RLC Data Blocks Using CS2 x 34 + Total Number of Uplink RLC Data Blocks Using CS3 x 40 + Total Number of Uplink RLC Data Blocks Using CS4 x 54)/(Total Number of Uplink GPRS TBFs x 1024)
