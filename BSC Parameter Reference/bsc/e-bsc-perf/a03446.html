Unsuccessful VGCS Uplink Seizures (Uplink Request Acknowledge Timeout)

Measurement Counter

A03446: CELL_UL_REQ_FAIL_MSC_ACK_EXP

Description

When the MS initiates a request for the uplink TCH, the BSC sends an UPLINK REQUEST message to the MSC and starts a timer. The BSC stops the timer after receiving the UPLINK REQUEST ACKNOWLEDGE message from the MSC. This measurement provides the number of times that the timer expires.

Unit

Integer number or integer.

Measurement Point

This counter is incremented by one when the timer expires before the BSC receives the UPLINK REQUEST ACKNOWLEDGE message from the MSC. In the signaling procedure, this counter is measured at point D in Figure 1.

Formula

This is an original counter without involving any formula.
