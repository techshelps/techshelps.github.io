Number of Uplink GPRS TBF Movements Because of EGPRS User Access

Measurement Counter

A9035: UP_GPRS_TBF_MOVE_EGPRS_TBF_CONNECT

Description

This measurement provides the number of uplink GPRS TBF movements due to the access of the EGPRS service in a granularity period.

Unit

Integer number or integer.

Measurement Point

When the EGPRS user accesses the EGPRS preferred channel, the GPRS user that previously occupies the channel should be handed over to other channels. The bit maps of available channels that can be used this tine and the previous time for the same uplink GPRS TBF are compared. If the change lies in the EGPRS preferred channel with the EGPRS user, this counter is incremented by one.

Formula

This is an original counter without involving any formula.
