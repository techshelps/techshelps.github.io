Successful Channel Assignments (TCH)

Measurement Counter

CR4119:TRX_CH_ALLOC_SUCC_TIMES_TCH

Description

After the BSC successfully assigns TCHs, this counter is measured in the following procedures:
  • Immediate assignment
  • Assignment
  • Internal intra-cell handover
  • Incoming internal inter-cell handover
  • Incoming external inter-cell handover

Unit

Integer number or integer.

Measurement Point

In the very early assignment of the TCH, the BSC receives the CHANNEL REQUIRED message from the BTS and measures this counter if the TCH is successfully assigned. See measurement point A1 in Figure 1.

Figure 1 Very early assignment procedure of the TCH

In the assignment procedure, the BSC receives the ASSIGNMENT REQUEST message from the MSC and measures this counter if the TCH is successfully assigned. See measurement point E in Figure 2.

Figure 2 Assignment procedure

In the internal handover procedure where the originating cell and the target cell controlled by the same BSC, the originating cell sends the Intercell Handover Request message to the target cell. The BSC measures this counter if the TCH is successfully assigned. See measurement point D in Figure 3. The measurement also involves the intra-cell handover. In this case, the originating cell and the target cell are the same cell.

Figure 3 Incoming internal inter-cell handover procedure

After Huawei BSC receives the HANDOVER REQUEST message from another BSC, it measures this counter if the TCH is successfully assigned. See measurement point D in Figure 4.

Figure 4 Incoming external inter-cell handover procedure

Formula

Successful Channel Assignments (TCH) =

[Successful Channel Assignments (TCHF)] + [Successful Channel Assignments (TCHH)]
