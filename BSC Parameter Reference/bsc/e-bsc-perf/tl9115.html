Average Payload of Single Downlink GPRS TBF (KB)

Measurement Counter

TL9115: DOWN_RLC_SGL_TBF_AVG_THRPUT

Description

This measurement provides the average payload of a single GPRS TBF at the downlink RLC in a cell in a granularity period. This measurement indicates the type of the service processed by the user. The measurements of uploading and downloading services are higher than those of the WAP services.

Unit

kbyte.

Measurement Point

The value of Average Throughput of a GPRS TBF on Downlink RLC (KB) is calculated through the total number of downlink RLC data blocks with CS-1, CS-2, CS-3, and CS-4 coding schemes and the number of downlink GPRS TBFs in a granularity period.

Formula

Average Payload of Single Downlink GPRS TBF (KB) = (Total Number of Downlink RLC Data Blocks Using CS1 x 23 + Total Number of Downlink RLC Data Blocks Using CS2 x 34 + Total Number of Downlink RLC Data Blocks Using CS3 x 40 + Total Number of Downlink RLC Data Blocks Using CS4 x 54)/(Total Number of Downlink GPRS TBFs x 1024)
