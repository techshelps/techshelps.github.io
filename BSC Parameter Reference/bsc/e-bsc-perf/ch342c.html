Failed Incoming External Inter-Cell Handovers (Timer Expired)

Measurement Counter

CH342C:CELL_INTERBSC_INCELL_HO_FAIL_EXP

Description

During the incoming external inter-cell handover procedure, a timer starts after the BSC sends the HO CMD to the MS. This measurement provides the number of failed incoming external inter-cell handovers due to the expiration of the timer. The handovers can involve the SDCCH, TCH signaling channel, or TCH traffic channel.

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1, the counter is measured at C when the incoming external inter-cell handover fails because of the timer for the BSC waiting for handover completion expires.

Figure 1 Measurement of the counters related to incoming external inter-cell handover

A

Measurement point of Incoming External Inter-Cell Handover Requests

B

Measurement point of Incoming External Inter-Cell Handover Responses

C

Measurement point of Incoming External Inter-Cell Handover Detection Messages Received by BSC

D

Measurement point of Successful Incoming External Inter-Cell Handovers

Formula

Failed Incoming External Inter-Cell Handovers (Timer Expired) = [Failed Incoming External Inter-Cell Handovers (Timer Expired) (SDCCH)] +

[Failed Incoming External Inter-Cell Handovers (Timer Expired) (TCH) (Signaling Channel)] + [Failed Incoming External Inter-Cell Handovers (Timer Expired) (TCH) (Traffic Channel)]
