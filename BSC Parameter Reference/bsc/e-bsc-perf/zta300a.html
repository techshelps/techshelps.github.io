Mean Interval between Random Access Attempts per BSC (MOC)

Measurement Counter

ZTA300A:BSCRPT_CH_REQ_MOC_AVG

Description

This measurement provides the mean interval in seconds between random access attempts per BSC (MOC). It indicates the mean interval in seconds between two adjacent channel requests (MOC) in the statistics period.

Unit

s.

Measurement Point

This counter is measured according to the following formula when the measurement period ends.

Formula

Mean Interval between Random Access Attempts per BSC (MOC) = {[GP] x 60}/ Channel Requests (MOC)
