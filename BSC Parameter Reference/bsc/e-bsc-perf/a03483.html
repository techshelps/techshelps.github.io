Unsuccessful Intra-Cell VGCS TCH to Dedicated TCH Handover (Timer Expired)

Measurement Counter

A03483: CELL_GCC_INTRACELL_GCC_TO_TCH_HO_FAIL_EXP

Description

The procedure for the intra-cell VGCS TCH to the dedicated TCH handover is equivalent to the inter-BSC handover procedure. The difference is that the originating BSC and the target BSC are the same. The originating BSC issues a HANDOVER COMMAND message to the MS and the MS accesses the target cell successfully. Then, the originating BSC receives a CLEAR COMMAND (reason: handover success) message from the MSC and starts to release the original resources. Thus, the handover is complete. If a timer expires during the handover procedure, the intra-cell handover procedure fails. This measurement provides the number of unsuccessful intra-cell VGCS TCH to dedicated TCH handovers due to an expired timer.

Unit

Integer number or integer.

Measurement Point

The MS cannot access the target cell and returns to the originating cell due to "Abnormal release, timer expired". This counter is incremented by one when the BSC receives a HANDOVER FAILURE message from the MS. The measurement point is C, as shown in Figure 1.
