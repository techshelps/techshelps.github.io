Number of Preferable Selections of TCHH in a CELL (Radio Resource Congestion)

Measurement Counter

R3167A: CELL_RADIORES_OVERFLOW_TIMES

Description

This counter is used to measure the number of preferable selections of the half-rate channel in a cell because of the radio resource congestion.

Unit

Integer number or integer.

Measurement Point

This counter is measured when the half-rate channel is preferably selected in a cell because of the radio resource congestion in the circuit service.

Formula

This is an original counter without involving any formula.
