Sent Signaling-Plane Bytes

Measurement Counter

S4012B: TRX_TX_SIGNALING_TRAFFIC

Description

This measurement provides the number of bytes in the signaling-plane packets that the TRX sends to the BSC over the HDLC path when transmission optimization is implemented in the BTS.

Unit

byte.

Measurement Point

The TRX performs the measurement when sending the signaling-plane packets. The BTS periodically reports the results, and the BTSM disassembles the packets and processes the measurement results.

Formula

This is an original counter without involving any formula.
