Number of Uplink GPRS RLC Control Blocks

Measurement Counter

L9018: UP_RLC_SIG_BLK

Description

This measurement provides the number of uplink GPRS RLC control blocks in a cell in a granularity period. The service type performed by an MS can be obtained by comparing the Number of Uplink GPRS RLC Control Blocks and total number of RLC blocks.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure in which the BSC sends a PACKET UPLINK ACK/NACK message to the MS to acknowledge the receipt of the uplink GPRS RLC data block. Every time the BSC sends a PACKET UPLINK ACK/NACK message (see measurement point A), the counter is incremented by one.

Figure 1 Uplink GPRS TBF control block carrying the PACKET UPLINK ACK/NACK message

Formula

This is an original counter without involving any formula.
