Requests for TCH from the PCU

Measurement Counter

R3140:CELL_TO_PCU_TCH_REQ_TIMES

Description

This measurement provides the number of requests sent from the BSC to the PCU for releasing the dynamic PDCHs.

Unit

Integer number or integer.

Measurement Point

This counter is measured when the BSC sends a request to the PCU for releasing the already assigned dynamic PDCHs in either of the following cases:

The BSC detects that there are no enough idle TCHs for the CS services.

The BSC predicts that there are no enough idle TCHs for CS services.

Formula

This is an original counter without involving any formula.
