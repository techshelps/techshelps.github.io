Description

This measurement provides the number of received EGPRS PACKET DOWNLINK ACK messages whose EGPRS BEP Link Quality Measurements contains a field 8PSK_MEAN_BEP whose value is N. This value reflects the air interface quality on the downlink in 8PSK modulation mode. The greater the value of N, the better the air interface quality.

Unit

Integer number or integer.

Measurement Point

Figure 1 shows the procedure in which the MS sends an EGPRS PACKET DOWNLINK ACK message to the network. The EGPRS PACKET DOWNLINK ACK message sent by the MS in 8PSK modulation mode carries the 8PSK_MEAN_BEP field, which indicates the radio measurement report sent by the MS. Each time the BSC receives the EGPRS PACKET DOWNLINK ACK message and the 8PSK_MEAN_BEP field is N (see measurement point A), the counter is incremented by one.
