Number of Configured TRXs per BSC

Measurement Counter

ZR460:BSCRPT_CONFIG_CONFIGNUM

Description

This measurement provides the number of configured TRXs per BSC, that is, the total of the configured TRXs in all the cells controlled by a BSC.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Activated TRXs".

Formula

Number of Configured TRXs per BSC = Activated TRXs

(That is, the total of the original counter Activated TRXs in all the cells of the BSC.)
