Completed Assignments

Measurement Counter

A3111A: CELL_ASS_CMP_TIMES_SPEECH_VER_3_DIRECT_RETRY_DR_TCHF

A3112A: CELL_ASS_CMP_TIMES_SPEECH_VER_3_DIRECT_RETRY_DR_TCHH

Description

This measurement provides the number of successful channel assignments based on the channel type, speech version, and support capability specified in the assignment request that the BSC receives from the MSC or the number of successful channel assignments based on support capability of the source cell and target cell and the channel assignment principles in the handover procedure.

Unit

Integer number or integer.

Measurement Point

The measurement is triggered after the BSC receives the handover completion command in the handover procedure (directed retry).The counter is measured at A, as shown in the following figure.



Formula

This is an original counter without involving any formula.
