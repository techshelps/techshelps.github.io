Retransmission Rate of RLC Downlink Data Block Using CSx (%)

Measurement Counter

RL9110: DOWN_CS1_RLC_BLK_RESEND_RATE

RL9111: DOWN_CS2_RLC_BLK_RESEND_RATE

RL9112: DOWN_CS3_RLC_BLK_RESEND_RATE

RL9113: DOWN_CS4_RLC_BLK_RESEND_RATE

Description

This measurement provides the retransmission rates of downlink RLC data blocks in CS1-CS4 coding schemes in a granularity period. If the retransmission rate is high, you can infer that the signal quality is poor, and thus the network needs to be optimized.

Unit

%.

Measurement Point

None.

Formula

Retransmission Rate of Downlink RLC Data Blocks Using CSx (%) = ( (Total Number of Downlink RLC Data Blocks Using CSx - Total Number of Valid Downlink RLC Data Blocks Using CSx)/Total Number of Downlink RLC Data Blocks Using CSx) x 100

Note: CSx indicates CS1-CS4.
