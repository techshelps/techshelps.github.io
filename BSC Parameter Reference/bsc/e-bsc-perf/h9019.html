Number of Failed Network Controlled 2G to 3G Cell Reselections

Measurement Counter

H9019: CELL_HO_NC_CELL_2G_TO_3G_RESELECT_FAIL

Description

This measurement provides the number of failed 2G-to-3G network-controlled cell reselections in a granularity period.

Unit

Integer number or integer.

Measurement Point

Based on the measurement report sent by the MS and the condition of the cell that the MS camps on, the BSC determines whether to initiate 2G-to-3G cell reselection. Every time the network-controlled cell reselection is triggered due to deteriorated transmission quality, traffic overload in the cell, or high receive level in neighboring cells, and a failed cell reselection message is received by the BSC, the Number of Failed Network Controlled Cell Reselections and Number of Failed Network Controlled 2G to 3G Cell Reselections are incremented by one respectively, provided that the Number of Failed Network Controlled 2G to 3G Cell Reselections must be less than the Number of Failed Network Controlled Cell Reselections.

Formula

This is an original counter without involving any formula.
