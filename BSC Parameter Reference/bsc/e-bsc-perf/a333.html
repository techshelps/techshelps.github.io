Average Undecodable Level in the Code Streams on the RACH

Measurement Counter

A333: CELL_AVG_UNSOLVE_LEV

Description

This measurement provides the average undecodable level in the code streams on the RACH.

The BTS measures the value of the CELL_AVG_UNSOLVE_LEV and reports it in the BTS_STATIS_REPORT message to the BSC every minute.

Unit

dB

Measurement Point

If the information element CELL_AVG_UNSOLVE_LEV in the BTS_STATIS_REPORT message is successfully decoded in the granularity period and the value is not 0xFF, then the average value is measured based on the history and the current measurement values, at C1 as shown in Figure 1.

Figure 1 Procedure of the BTS sending the BTS_STATIS_REPORT message

Formula

This is an original counter without involving any formula.
