Analyzed Measurement of Incoming Cell Handovers

Measurement Counter

RH383:CELLCELL_INCELL_HO_SUCC_RATE

H382:CELLCELL_INCELL_HO_FAIL

H383:CELLCELL_INCELL_HO_SUCC

Description

The counters indicate the performance of the incoming inter-cell handover in a specified cell, for example, the number of requests for the handover from a cell to the specified cell, the number of successful handovers, and the success rate of handovers.

For the counters related to Analyzed Measurement of Incoming Cell Handovers, it is recommended that the measurement job be registered at the initial stage of BSC running or during the analysis of handover performance in a specified cell for network optimization. When the BSC system is running stably, the measurement job can be deleted. Along with the counters related to Analyzed Measurement of Outgoing Cell Handovers, these counters indicate the performance of the handover between two specified cells. The recommended period of measuring these counters is 60 minutes.

Unit

RH383: %.

H382/H383: Integer number or integer.

Measurement Point

This counter is calculated by using the following formula.

Formula

Success Rate of Cell Incoming Handovers = [Successful Cell Incoming Handovers] x {100}/[Incoming Inter-Cell Handover Requests]

Failed Cell Incoming Handovers = [Failed Cell Incoming Handovers (Congestion)] + [Failed Cell Incoming Handovers (Other Causes)]

Successful Cell Incoming Handovers = [Cell Incoming Handover Requests] - [Failed Cell Incoming Handovers]
