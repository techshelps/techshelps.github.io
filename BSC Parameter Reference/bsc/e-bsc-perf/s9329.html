S9338: CELL_USER_UPLINK_TIMESLOT_OCCUP_1_PER_1_TIMES

Description

This measurement provides the number of times that N timeslots are actually allocated to the uplink TBFs when M timeslots can be allocated according to the calculation based on the service type and multislot capability of the MS in a granularity period. The higher the value, the higher the timeslot utilization rate. The value of N and M ranges from 1 to 4. N should be smaller than or equal to M.

Unit

Integer number or integer.

Measurement Point

In a granularity period, the BSC periodically samples all the uplink TBFs. If the sampled TBFs are in transmission state, the BSC calculates the maximum number of timeslots that can be allocated to the uplink TBFs based on the multislot capability and service type of the MS. In addition, the BSC obtains the number of timeslots that are actually allocated to the uplink TBFs. If the number of actually allocated uplink timeslots is N and the calculated maximum number of uplink timeslots is M, the counter is incremented by one.

Formula

This is an original counter without involving any formula.
