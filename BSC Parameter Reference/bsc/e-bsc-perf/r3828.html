Number of Loopback Start Failures (Service)

Measurement Counter

R3828: CELL_LOCALSWITCH_LOOP_FAIL_SERVICE

Description

When the called subscriber picks up the phone or the incoming BSC handover for speech services is complete, the BSC loopback procedure is triggered. If the BSC detects that the MS initiates a supplementary service or another service that needs to be carried and processed on the core network, the local switching fails because the local loopback cannot be performed.

Unit

Integer number or integer.

Measurement Point

After receiving a CONNECT ACK message on the downlink from the MSC or receiving a HANDOVER CMP message from the MS, the BSC selects the local switching strategy, determines the local switching start conditions, detects the local switching calling, and starts the local switching forced handover preprocessing procedure for the local switching. After the preprocessing procedure, the local switching start procedure is started if the local switching can be performed. During the preprocessing procedure, the local switching is not started if the BSC receives a service that needs to be carried and processed on the core network. The local switching start failure procedure is started. In this case, this counter is measured on the cell that waits for the local switching preprocessing.

Formula

This is an original counter without involving any formula.
