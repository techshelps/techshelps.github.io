Number of Actually Used Downlink PDCHs per EGPRS User (User)

Measurement Counter

AR9618: CELL_PDCH_DOWN_EGPRS_TBF_OCCU_PDCH_NUM_USER

Description

This measurement provides the number of actually used downlink PDCHs per EGPRS user.

Unit

Integer number or integer.

Measurement Point

None.

Formula

Number of Actually Used Downlink PDCHs per EGPRS User (User) = Number of Actually Used Downlink PDCHs per EGPRS User (Original)/(Sampling Times of EGPRS Downlink Priority Service x 100)
