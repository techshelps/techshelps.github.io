Downlink GPRS TBF Establish and Release Capability Measurement

Description

The PSCALL.DownGTbfEstbRel.Cell measurement unit provides the counters related to the establishment and release of the downlink GPRS TBF in a granularity period. It indicates the traffic load of the PS services in a cell and is used as a reference for establishing the PS service model.

Measurement Object

Cell.
