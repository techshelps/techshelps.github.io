Failed Outgoing Internal Inter-Cell Handovers (Reconnection to Old Channels)

Measurement Counter

CH312D:CELL_INTRABSC_OUTCELL_HO_FAIL_RECONN_SUCC

Description

During the outgoing internal inter-cell handover procedure, the timer T3103 starts after the BSC sends the HO CMD to the MS. This measurement provides the number of failed outgoing internal inter-cell handovers because of the BSC receives the HO FAIL message from the MS before T3103 expires. The failure causes carried in the HO FAIL message can be:

  • Abnormal release, channel unacceptable
  • Abnormal release, timer expired
  • Abnormal release, no activity on the radio path
  • Preemptive release
  • Handover failed, timing advance out of range
  • Channel mode unavailable
  • Frequency unavailable
  • Call already cleared
  • Semantically incorrect message
  • Invalid mandatory information
  • Message type non-existent or not implemented
  • Message type not compatible with protocol state
  • Conditional IE error
  • No cell allocation available
  • Protocol error unspecified
  • Other causes

Unit

Integer number or integer.

Measurement Point

As shown in Figure 1 and Figure 2, during the outgoing internal inter-cell handover procedure, this counter is measured at F when the BSC receives the HO FAIL message from the MS after the BSC sends the HO CMD message to the MS and the MS cannot access to the new channel.
