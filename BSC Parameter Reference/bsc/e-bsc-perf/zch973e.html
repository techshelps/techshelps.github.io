Successful Handovers per BSC (Timing Advance)

Measurement Counter

ZCH973E:BSCRPT_HO_SUCC_TA

Description

This measurement provides the number of successful handovers of all the cells controlled by a BSC due to TA causes (handovers caused by service cell signal or quality decline when the user is far from or near the target cell during a call), including outgoing internal inter-cell handovers and outgoing external inter-cell handovers.

Unit

Integer number or integer.

Measurement Point

This counter is measured according to the following formula when the measurement period ends. For details on the measurement point, see "Measurement Point" of each counter in the formula.

Formula

Successful Handovers per BSC (Timing Advance) =

( Outgoing Internal Inter-Cell Handover Requests (Timing Advance) + Outgoing External Inter-Cell Handover Requests (Timing Advance)) -

( Failed Outgoing Internal Inter-Cell Handovers (Timing Advance) + Failed Outgoing External Inter-Cell Handovers (Timing Advance))
