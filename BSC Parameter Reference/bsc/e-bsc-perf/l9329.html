Number of MCS Upgrades on Downlink EGPRS TBF

Measurement Counter

L9329: DOWN_EGPRS_TBF_MCS_UPGRADE_NUM

Description

This measurement provides the number of successful MCS upgrades on the downlink EGPRS TBF in a cell in a granularity period. If the Number of MCS Upgrades on Downlink EGPRS TBF is greater than the Number of MCS Degrades on Downlink EGPRS TBF, it indicates that the network quality is good. If both the Number of MCS Upgrades on Downlink EGPRS TBF and the Number of MCS Degrades on Downlink EGPRS TBF are great, it indicates that the network quality is poor, and the coding scheme of the MS may be frequently adjusted.

Unit

Integer number or integer.

Measurement Point

Every time the BSC changes the MCS coding scheme, it determines whether to upgrade or degrade the MCS coding scheme by comparing the current MCS coding scheme with the previous MCS coding scheme. Figure 1 shows the procedure in which the BSC sends a data block using more robust MCS coding scheme to the MS. Every time the BSC sends a DOWNLINK RLC DATA BLOCK message for the upgrade of the coding scheme (see measurement point A), the counter is incremented by one.

Formula

This is an original counter without involving any formula.
