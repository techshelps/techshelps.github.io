Utilization Rate of EGPRS Downlink Timeslots

Measurement Counter

S9313: CELL_USER_EGPRS_TBF_DOWNLINK_TS_UTILI_RATE

Description

This measurement provides the utilization rate of timeslots allocated to the EGPRS downlink TBFs in transmission state in a granularity period. This counter is used to calculate Average Utilization Rate of Downlink TBF Timeslots.

Unit

%.

Measurement Point

In a granularity period, the BSC periodically samples all the EGPRS downlink TBFs in a cell. If the sampled EGPRS downlink TBFs are in transmission state, the BSC calculates the maximum number of downlink timeslots based on the multislot capability and service type of the MS. In addition, the BSC calculates the number of timeslots that are actually allocated to downlink TBFs. Then, the number of actually allocated downlink timeslots is multiplied by 100 and then divided by the maximum number of downlink timeslots. The calculated value is accumulated to Utilization Rate of EGPRS Downlink Timeslots.

Formula

This is an original counter without involving any formula.
