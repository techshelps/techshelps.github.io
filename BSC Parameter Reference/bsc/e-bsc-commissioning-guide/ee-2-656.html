Connecting the LMT PC to the GBAM/GOMU

This provides an example of connecting the LMT PC to the GBAM/GOMU in a LAN to describe how the LMT PC is connected to the OM network of the BSC. You need to perform this operation only when the LMT is used for the first time.

Prerequisites

  • The communication between the GBAM/GOMU and the BSC host is normal.
  • The communication between the LAN switch and the BSC host is normal.
  • The physical connection between the LMT PC and the GBAM/GOMU is established.

Procedure

  1. Set the IP address of the LMT PC. For details, refer to Setting IP Address of LMT PC.
  2. On the LMT PC, choose Start > Run. In the Run dialog box, type cmd, and then press Enter. The cmd.exe window is displayed.

    Table 1 Level standard of IP network
    Network level Single-way time delay(ms) Packet loss ratio Jitter(ms)
    Good ≤40 ≤0.1％ ≤10
    Normal ≤100 ≤1％ ≤20
    Bad ≤400 ≤5％ ≤60

  3. In the cmd.exe window, type ping external IP address of GBAM/GOMU.Table 1 shows the Level standard of IP network

    If the information similar to the following display is returned, you can infer that the LMT PC and the GBAM/GOMU can communicate normally. In this example, the IP address of the external network is 10.121.71.228.
