DIP Switch on the PFCU

There is a DIP switch labeled SW1 on the PFCU.

Figure 1 shows the DIP switch on the PFCU.

Figure 1 DIP switch on the PFCU

SW1 (with four bits) is used to set the address of the PFCU.

To set the address of the PFCU, pull out the fan box, and then set SW1 by referring to Table 1.

For how to pull out the fan box, refer to Replacing the BSC Fan Box. After the setting, the address of the PFCU is 1.

Table 1 Settings of S1
