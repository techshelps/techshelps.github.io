Jumper Pins on the PFCB

The PFCB provides eight pairs of pins for jumpers. After being connected to jumpers, these pins are used to set the address and working mode of the PFCB .

Figure 1 shows the jumper pins on the PFCB .

Figure 1 Jumper pins on the PFCB

To set the address, pull out the fan box. Then, set the jumper pins as described in Table 1.

For how to pull out the fan box, refer to Replacing the BSC Fan Box. After the setting, the address of the PFCB is 1.
