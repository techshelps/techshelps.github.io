LEDs on the GTNU Panel

There are three LEDs on the GTNU panel: RUN, ALM, and ACT.

Table 1 describes the LEDs on the GTNU panel.

Table 1 LEDs on the board

LED

Color

Status

Meaning

RUN

Green

On for 1s and off for 1s

The board is working.

On for 0.125s and off for 0.125s

The board is loading data.

On

There is power input but the board is faulty.

Off

There is no power input or the board is faulty.

ALM

Red

Off

There is no alarm.

On or blinking

There is an alarm.

ACT

Green

On

The board works in active mode.

Off

The board works in standby mode.
