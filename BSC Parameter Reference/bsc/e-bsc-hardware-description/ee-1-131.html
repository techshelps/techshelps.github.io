Ports on the GOIUA/GOIUB/GOIUP/GOIUT(OIUa) Panel

This describes the panel of the optical interface boards. The GOIUA, GOIUB, GOIUP, and GOIUT use the same physical board but perform different logical functions by loading different software. These boards have the same ports. Of the four ports, one port is labeled RX/TX, one 2MO, one 2M1, and one TESTOUT.

Table 1 describes the ports on the GOIUA/GOIUB/GOIUP/GOIUT panel.
