LEDs on the GSCU Panel

There are four types of LEDs on the GSCU panel: the RUN, ALM, and ACT LEDs, and the LEDs of the Ethernet ports.

Table 1 describes the LEDs on the GSCU panel.

Table 1 LEDs on the board

LED

Color

Status

Meaning

RUN

Green

On for 1s and off for 1s

The board is working.

On for 0.125s and off for 0.125s

The board is loading data.

On

There is power input but the board is faulty.

Off

There is no power input or the board is faulty.

ALM

Red

Off

There is no alarm.

On or blinking

There is an alarm.

ACT

Green

On

The board works in active mode.

Off

The board works in standby mode.

Table 2 describes the LEDs of the Ethernet ports on the GSCU panel.

Table 2 LEDs of the Ethernet ports

LED

Color

 Status Meaning

LINK

Green

On

LEDs of the Ethernet ports, indicating that the link is connected

Off

LEDs of the Ethernet ports, indicating that the link is disconnected

ACT

Green

Blinking

LEDs of the Ethernet ports, indicating that there is a data flow

Off

LEDs of the Ethernet ports, indicating that there is no data flow
