DIP Switch on the MDMC

There is a DIP switch labeled S2 on the MDMC.

Figure 1 shows the DIP switch on the MDMC.

Figure 1 DIP switch on the MDMC

S2 is used to set the address of the MDMC.

To set the address of the MDMC, pull out the MDMC, and then set S2 by referring to Table 1.

Table 1 Setting of S2

Address

DIP Bit

Switch Status

Meaning

0

1 (the most significant bit)

ON

0

2

ON

0

3

ON

0

4 (the least significant bit)

ON

0
NOTE:

In the BSC, the DIP switch on the MDMC must be set according to the specifications listed in Table 1.
