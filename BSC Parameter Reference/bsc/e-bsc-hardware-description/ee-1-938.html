Classification of the BSC Cabinet

Based on functions, the BSC cabinets are classified into the GBCR and the GBSR.

GBCR

The GBCR processes primary services and provides OM for the BSC. One GBCR is configured in a BSC.

Based on the hardware configuration, the GBCR is classified into configuration type A and configuration type B.

  • In configuration type A, the GBCR is configured with the GBAM. For details, refer to GBCR (Configuration Type A).
  • In configuration type B, the GBCR is configured with the GOMU. For details, refer to GBCR (Configuration Type B).

GBSR

The GBSR processes various services for the BSC. The number of GBSRs to be configured depends on the traffic volume. Zero to three GBSRs can be configured. For details, refer to GBSR Cabinet.
