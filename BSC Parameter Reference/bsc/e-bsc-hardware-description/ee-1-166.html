MDMC Panel

The components on the MDMC are the LEDs and mute switch.

Figure 1 shows the MDMC panel.

Figure 1 MDMC panel

(1) Mute switch
NOTE:

The mute switch is used to control the audible alarm of the power distribution box.

  • If you set the switch to ON, the power distribution box generates an audible alarm when it detects a fault.
  • If you set the switch to OFF, the power distribution box does not generate an audible alarm when it detects a fault.
