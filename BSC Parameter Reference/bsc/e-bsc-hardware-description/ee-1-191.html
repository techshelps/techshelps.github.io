BSC PGND cable

The BSC PGND cables consist of the PGND cable for the cabinet, inter-cabinet PGND cable, PGND cable for the power distribution box, PGND cable for the subracks, and PGND cable for the cabinet door. All these cables are mandatory.

Each BSC cabinet must be configured with one PGND cable for the cabinet. For combined cabinets, adjacent cabinets should be configured with three inter-cabinet PGND cables. All other PGND cables are installed before delivery.

NOTE:

The same types of PGND cables are used regardless of whether the BSC cabinet is configured with the common power distribution box or with the high-power distribution box.
