Querying the Status of a Power Module

This describes how to query the status of a BTS power module, including the status of current limiting, charge, and discharge, and the type of the BTS power module. This also describes how to query the number of DPSUs and the position of the DPSU.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.
  • The power module works in normal state.

Context

You can query the DPMU status of the BTS3012, BTS3012AE, BTS3002E, BTS3012â…¡, BTS3036, BTS3036A, DBS3036, DBS3900 GSM, BTS3900 GSM and BTS3900A GSM, and the APMU status of the BTS3006C.

Procedure

  • Through GUI
    1. On the LMT, choose BTS Maintenance > Maintain Board > Query Power Module Status.
    2. Set parameters in the Location area, and then click Query. The query results are displayed, as shown in Figure 1.

      Figure 1 Querying the status of a power module

    3. Click Cancel to end the query.
  • Through MML
    1. Run the DSP BTSPWRSTAT command on the LMT to query the status of the power module.
