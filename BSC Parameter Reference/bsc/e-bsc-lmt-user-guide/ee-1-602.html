Tracing Modes

The tracing modes decide where the signaling tracing results are stored.

The tracing modes are:

  • Report

    Real-time tracing mode. The server sends the traced message to the LMT PC, which saves the output file in *.tmf format.

  • Saving to the GBAM/GOMU

    You can set the time for a tracing task. The traced message is saved to the server, and the LMT does not receive any message. The output file is saved to the GBAM or GOMU in *.tmf format. The default directory for saving the output file of the traced message is bsc6000\BSC6000\data\mtndata\trc.

    • If the BSC is configured with GBAM, the message is saved to the GBAM.
    • If the BSC is configured with GOMU, the message is saved to the GOMU.

You can review the output file of the traced message. For details, refer to Reviewing Trace Message Files.
