Querying the Location Group Attributes

This describes how to query the location group attributes when co-cell is enabled. You can know the related parameters by querying the location group information.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

Only the DBS3036/DBS3900 GSM supports the configuration of the parameters of the co-cell location group.

Procedure

  • Through GUI
    1. On the LMT, choose BTS Maintenance > Maintain Site > Query Location Group Attributes to query the location group attributes.
    2. Select a site in the Location area, and then click Query. A dialog box is displayed as shown in Figure 1.

      Figure 1 Query Location Group Attributes dialog box

      NOTE:

      Click Bind Relationship, and you can query the board number and the channel number.

    3. Click Cancel to end the query.
  • Through MML
    1. Run the DSP LOCGRP command on the LMT to query the location group information of a specified site.
