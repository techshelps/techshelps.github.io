Collecting the BSC Local Information

Collecting the BSC local information involves setting the information collection, collecting the local information, and generating the collection report. The collection of the BSC local information helps the maintenance personnel to analyze and locate problems.

  • Setting the Information Collection
    This describes how to set the save path and storage mode of the collected information.
  • Collecting the Local Information
    This describes how to collect the BSC or BTS local information on the LMT. Such information is contained in the performance statistics files, logs, and alarm files.
  • Generating the Collection Report
    This describes how to generate the collection report in .htm format and record the feedback personnel and fault information on the LMT.
