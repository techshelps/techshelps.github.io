Managing Software

This describes how to manage the BSC software on the LMT.

  • Concepts of BSC Software Management
    This describes the basic concepts of the BSC software management.
  • Querying Board Version Information
    This describes how to query the information about the software, hardware and patches of a board, and the mode of loading the board on the LMT.
  • Managing the Version Software
    This describes how to manage version software and ensure the compatibility of hardware version and software version by downloading, loading, activating, and rolling back version software.
  • Managing Patches
    Management of patches is performed through operations such as downloading, loading, activating, confirming, deleting, and rollback.
  • Managing the License
    This describes how to download, activate, and query the BSC license. This also describes how to query the BSC license file, equipment serial number, and equipment usage.
