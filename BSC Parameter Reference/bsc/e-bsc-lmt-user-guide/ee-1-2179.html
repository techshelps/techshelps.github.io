GOMU Information

This describes the GOMU information.

The GOMU information consists of the following information:

  • GOMU hard disk information

    Hard disk serial number, GOMU hard disk memory, product model, and disk status

  • GOMU RAID information

    RAID identifier, RAID type, number of physical disks in RAID, step information, RAID capacity, RAID reassembly scale, and RAID status

  • GOMU memory information

    GOMU memory capacity and GOMU memory usage

  • GOMU network adapter information

    Network adapter binding information, MAC address, IP address, Ethernet information, and Ethernet connection status.

  • GOMU Daemon status information

    IP address of Daemon, UDP port of Daemon, switchover switch status of Daemon, switchover inhibit remaining time of Daemon, backup switch status of Daemon, backup status type of Daemon, and active/standby state machine status of Daemon

  • Information of the process monitored by GOMU Daemon

    Name of the monitored process, file name of the monitored process, start parameters of the monitored process, virtual PID of the monitored process, system PID of the monitored process, control type of the monitored process, current status of the monitored process, maximum restart times of the monitored process, current restart times of the monitored process

NOTE:
Except the OMU, all the other boards are called SMU.
