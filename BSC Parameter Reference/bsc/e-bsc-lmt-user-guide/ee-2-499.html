Querying the BSC License

This describes how to query the configuration and application information about the effective license files.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The GXPUM board in the GMPS subrack runs normally.

Context

  • Configuration information

    Contains such items as sequence number of the software, usage time limit, maximum number of carriers, number of cells, and various functions

  • Application information

    Indicates the sequence number of the current license, usage time limit, number of carriers, number of cells, and the functions that are used currently

Procedure

  1. Run the DSP LICENSE command on the LMT to query the BSC license.
