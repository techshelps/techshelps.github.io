User Authority

This describes the authorities of the default users and non-default users of the LMT.

Default Users

By default, a super-user ADMIN and a guest user GUEST are configured. The ADMIN user has all the operation authorities. The two users cannot be changed or deleted.

Non-Default Users

The non-default users are divided into five levels. Different levels of users have different authorities, as described in Table 1.
