Managing the BSC Board Clocks

This describes how to maintain the BSC board clocks, BSC clock references, the frequency offset of the 2 MHz clock on the GGCU, and the line clock.

  • Querying the Clock Status of a Board
    This describes how to query the status of the input clock of a board.
  • Querying the 2 MHz Clock Frequency Offset of the GGCU
    This describes how to query the status of the BSC 2 MHz clock reference for the GGCU.
  • Maintaining the BSC Clock Reference Status
    This describes how to query the status of the BSC clock reference for the GGCU. Check whether the PLL status of the BSC clock reference source is trace locked, check from which line clock the clock is obtained, and check the time when the PLL locks the clock.
  • Maintaining the GSCU Reference Clock
    This describes how to query and modify the GSCU reference clock.
  • Maintaining the Line Clock
    This describes how to maintain the line clock by querying, adding, and removing the line clock.
