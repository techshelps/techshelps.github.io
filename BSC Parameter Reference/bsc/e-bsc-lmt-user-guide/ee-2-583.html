Alarm Management

This describes how to browse, query, and maintain the BSS alarms and how to maintain alarm boxes and manage alarm log files.

  • Concepts of Alarm Management
    This describes the concepts of the BSS alarm management, including the alarm types, alarm IDs, alarm severity, alarm flags, and types of network management.
  • Managing Alarm Configuration
    This describes how to query alarm configuration, set alarm severity, set alarm shield flag, set the flag for reporting the alarm to the alarm box, configure environment alarm, and restore alarm configuration.
  • Managing the Active Alarms
    This describes how to manage the fault alarms that are not cleared in the BSC.
  • Managing Alarm Logs
    This describes how to manage the alarm log files, and how to set the maximum days for the alarm log files to be saved.
  • Managing Alarm Shields
    Managing alarm shields consists of querying shielded alarms, adding shielded alarms, and removing alarm shields.
  • Managing the Alarm Box
    This describes how to manage the alarm box by maintaining the alarm box information and setting the alarm severity of the alarm box.
