Locking the BSC6000 Local Maintenance Terminal

This describes how to lock the BSC6000 Local Maintenance Terminal to prevent unauthorized users from performing operations.

Prerequisites

  • The LMT runs normally.
  • You have logged in to the LMT in online mode.

Context

This system locking can prevent operations of unauthorized users.

The system will be locked automatically if no operations are performed during a period of time. To set the auto lock time, refer to Setting Attributes of the BSC6000 Local Maintenance Terminal.

Procedure

  1. Use one of the following three methods to manually lock the BSC6000 Local Maintenance Terminal.

    • Choose System > Lock System.
    • Press F6.
    • Click on the tool bar.

  2. The running status of the selected sites is displayed, as shown in Figure 1.

    Figure 1 Unlocking the system

  3. To unlock the system, enter the password in Password, and then click OK.
