Managing the BSC Time

Managing the BSC time involves managing the time zone, daylight saving time, and BSC system time.

  • Setting the Time Zone and the DST
    This describes how to configure the time zone and the daylight saving time. The ADMIN users, administrators, operators, and custom users with related authorities can perform this operation.
  • Querying the BSC System Time
    This describes how to query the BSC system time.
  • Setting the BSC System Time
    This describes how to set the BSC system time.
