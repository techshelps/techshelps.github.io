Querying BSC License File Information

This describes how to query and verify the attribute information about the license file saved on the local PC.

Prerequisites

  • The LMT runs normally.
  • The license file exists on the local PC.

Context

  • You can perform this task to view and verify the license file in offline mode.
  • If the activated license file is faulty, the system will be faulty.
  • Before activating the license file, you can check that the name, ESN, and feature field of the license file are correct.
  • You can verify the license file by entering the OPC and DPC or by selecting the configuration file.

Procedure

  1. Choose BSC Maintenance > Maintain License > Query License File. A window is displayed, as shown in Figure 1.

    Figure 1 Query License File window

  2. Click to select the license file to be queried. Set parameters in the Verify License file area, and then click Query to query the information about the license.
