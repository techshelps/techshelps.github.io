Querying the Time Zone and the Daylight Saving Time

This describes how to query the time zone and the daylight saving time (DST).

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Procedure

  • Through GUI
    1. Choose System > Query Time Zone and DST Configuration. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Query Time Zone and Daylight Saving Time dialog box

    2. Click Cancel to end the query.
  • Through MML
    1. Run the LST TZ command on the Local Maintenance Terminal to query the time zone and daylight saving time of the BSC.

      NOTE:

      If the daylight saving time (DST) is set, the value of Start Time of DST includes the time zone instead of the DST deviation, and the value of End Time of DST includes the time zone and the DST deviation.
