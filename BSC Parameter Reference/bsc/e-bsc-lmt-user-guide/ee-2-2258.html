Maintaining the BTS Antenna System

This describes how to maintain the BTS antenna system. The operations involve querying and modifying the parameters associated with the BTS antenna system.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

  • The BTS index or BTS name, subrack number, and slot number are mandatory. If the board you query is not the antenna feeder board, an object error message is displayed.
  • When the board type is changed, the CDU boards and the EDU boards cannot be updated in a mixed mode.

Procedure

  • Run the LST ANTFEEDER command on the LMT to query the BTS antenna system.
  • Run the MOD ANTFEEDER command on the LMT to modify the BTS antenna system.
