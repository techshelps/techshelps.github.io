Tracing PTP Messages on the Gb Interface

This describes how to trace the PTP signaling messages on the Gb interface based on cell or NSEI+BVCI on the LMT when the PCU is installed inside the BSC.

Prerequisites

  • The LMT runs normally.
  • The communication between the BSC and the SGSN is normal.
  • The NSE and the association between BVCs and cells are already configured.

Context

For details of Tracing Mode, refer to Tracing Modes.

The following description is based on the configuration using the GOMU.

Procedure

  1. On the Trace & Monitor tab page, choose Trace > Trace PS Message > Trace Gb Interface Message > PTP Message. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Trace PTP Message on the Gb Interface dialog box

  2. Set parameters in the Location and Tracing Mode areas, and select Save File.

    CAUTION:

    Save the files according to the interface names. For the long-time trace, you must select autosave.

  3. Click OK.

Result

  • If the operation is successful:
    • If the tracing mode is Save on GOMU, the LMT does not display the traced messages. You can review the output file of the traced message. For details, refer to Reviewing Trace Message Files.
    • If the tracing mode is Report, a dialog box is displayed, prompting the information of the message tracing task, as shown in Figure 2.
      Figure 2 Tracing PTP messages on the Gb interface

  • If the operation fails, a dialog box is displayed, prompting the possible causes.

Postrequisite

You can double-click a message to view the detailed information.

Right-click the message to be analyzed. The shortcut menu provides the following options:
  • Clear Message: Clear the displayed messages.
  • Stop Scroll: Stop scrolling the displayed messages.
  • Stop Trace: Stop the current tracing task.
  • Restart Trace: Restart the tracing task.
  • View Task: View the established tracing task.
  • Search: Search for the specified character string in the displayed messages.
  • Save Selection: Save the selected messages.
  • Save All: Save all the traced messages.
