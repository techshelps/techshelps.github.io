Downloading the BSC License

This describes how to download the License file to the active area of on the server from the LMT.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The License and corresponding configuration files to be downloaded are already put in LMT installation directory\BSC6000\Common\SWM\License.

Procedure

  1. Run the DLD LICENSE command on the Local Maintenance Terminal to download BSC License.
