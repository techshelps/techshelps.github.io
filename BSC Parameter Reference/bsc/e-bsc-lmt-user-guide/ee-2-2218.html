Uploading the Inventory Management File

This describes how to upload the inventory management file.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication within the BSC is normal.
  • The inventory management file exists in the specified directory.
  • The FTP server is started.

Context

The BTS2X does not support inventory management.

The inventory management files are saved to IP address of the server\BSC6000\data\mtndata\inventory\ivt.

  • The specified directory name and filename together cannot exceed 255 characters, and the filename cannot exceed 90 characters.
  • The generation of the inventory management file and the uploading of the file cannot proceed at the same time.

Procedure

  1. Run the ULD DEVFILE command on the Local Maintenance Terminal to upload the inventory management file.
