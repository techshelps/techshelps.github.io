Collecting Statistics on Interface Board Resources

This describes how to collect statistics on interface board resources on the LMT to provide engineering parameters. The interface board resources are the Abis interface board resources and the A interface board resources.

  • Collecting Resource Statistics by Abis Interface Board
    This describes how to collect statistics on the total number of ports, idle ports, and TRXs that can be added by A interface board.
  • Collecting Resource Statistics by A Interface Board
    This describes how to collect statistics on the total number of ports and idle ports by A interface board.
