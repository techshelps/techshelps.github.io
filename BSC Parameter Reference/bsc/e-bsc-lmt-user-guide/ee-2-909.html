Exporting Measurement Object Formatted File

This describes how to export Measurement Object Formatted File to the local disk.

Prerequisites

  • The LMT runs normally.
  • The server is functional.
  • The communication between the LMT and the BSC is normal.

Context

The name of the exported measurement object formatted file is MOIDN.xml. The default saving path is: LMT installation directory\version number\BackupCfgData.

When browsing the performance test results saved on the LMT by using the performance browser tool, you can parse the MOIDN.xml file to generate the object names. You can then conveniently query the objects according to the names.

Procedure

  1. Choose Configuration > Export Measurement Object Formatted File.
  2. Select a save path and name the file to be saved.
  3. Click Save.
