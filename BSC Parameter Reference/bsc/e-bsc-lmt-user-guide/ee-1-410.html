Convert Management System

This describes the Convert Management System that is one of the software items of the LMT software package. The Convert Management System is used to forward the alarm information, the control information, and the status information about the alarm box between the server and the alarm box. The convert management system connects to the server through the Ethernet port and communicates with the alarm box through the serial port.

Function

The convert management system has the following functions:

  • Forwarding the instructions on how to operate the alarm box to the GBAM so that the local user can operate the alarm box on the BSC6000 LMT
  • Forwarding alarms to the alarm box, which displays audio and visual alarms
  • Monitoring in real time the communication between the LMT and the server, and the communication between the LMT and the alarm box. When a fault occurs, the Convert Management System reports an alarm to the LMT.
