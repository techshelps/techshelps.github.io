Exporting Network Optimization Data

This describes how to export the network optimization parameters of the cell to the local PC.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Procedure

  1. Choose Configuration > Import or Export Data > Export Network Optimization Parameter. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Export Network Optimization Parameter dialog box

  2. Select the cell whose parameters need be exported from the Cell View box and add them to the Selected Cells box, and then click Next. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Selecting parameters for export

  3. Select the parameters to be exported from the All Parameters box and add them to the Selected Parameters box.
  4. Click Export. The Save As dialog box is displayed. Save the parameters in a file to the specified directory.
  5. The information that saving the parameters is complete is displayed.
