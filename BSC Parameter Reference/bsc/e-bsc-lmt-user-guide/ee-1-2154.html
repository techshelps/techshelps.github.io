Real-Time Monitoring

Real-time monitoring refers to monitoring some counters and functions in real time through the LMT when the BSC is in the running state.

Real-time monitoring consists of the following monitoring tasks:

  • Monitoring the CPU/DSP usage
  • Monitoring the BER
  • Monitoring port errored seconds
  • Monitoring GPRS user flow
  • Monitoring the utilization of Abis HDLC transmission resources
  • Monitoring the cell performance
  • Monitoring the DSP service resources
