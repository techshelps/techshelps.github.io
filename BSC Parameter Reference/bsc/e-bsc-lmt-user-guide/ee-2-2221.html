Maintaining the Ater Interface

This describes how to maintain the Ater interface.

  • Querying the Ater Connection Path
    This describes how to query the configuration information of an Ater connection path according to the index ID, subrack number, or port number of the Ater connection path.
  • Querying the OMLs of the Ater Interface
    This describes how to query the OMLs of the Ater interface according to the index ID of the Ater connection path.
  • Querying the Status of the Ater OML
    This describes how to query the status of the Ater OML when the GTCS is placed on the MSC side.
  • Querying the Timeslots on the Ater Interface
    This describes how to query the timeslots on the Ater interface according to its status.
  • Querying the Signaling Links of the Ater Interface
    This describes how to query the signaling links of the Ater interface according to the index ID or the timeslot ID of the Ater connection path.
  • Maintaining Ater Interface Resources
    This describes how to maintain Ater resources, including setting and querying Ater resources.
