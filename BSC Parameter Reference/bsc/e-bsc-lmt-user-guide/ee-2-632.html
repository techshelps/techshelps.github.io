Starting the Performance Browser Tool

This describes how to start the Performance Browser Tool. Through the Performance Browser Tool, you can manually import the performance description files of NEs, and export the performance measurement files.

Prerequisite

The LMT is already installed.

Context

For the description of the main interface and help system of the Performance Browser Tool, refer to Performance Browser Tool.

Procedure

  1. Click Start on the LMT PC, and then choose All Programs > HUAWEI Local Maintenance Terminal > Performance Browser Tool to start the Performance Browser Tool.
