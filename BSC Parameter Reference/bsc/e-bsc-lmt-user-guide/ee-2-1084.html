Querying the 2 MHz Clock Frequency Offset of the GGCU

This describes how to query the status of the BSC 2 MHz clock reference for the GGCU.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The GGCU is running properly.

Procedure

  • Through GUI
    1. On the LMT, choose BSC Maintenance > Maintain Device > Query BSC 2M Clock Frequency Offset.
    2. Set parameters in the Select GGCU Slot No. area and then click Refresh. The query results are displayed, as shown in Figure 1.

      Figure 1 Querying the clock reference status

    3. Click Cancel to end the query.
  • Through MML
    1. Run the DSP GCU2MFD command on the Local Maintenance Terminal to query 2 MHz clock frequency offset of the GGCU.
