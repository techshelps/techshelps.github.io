Querying the Online Domain Users

This describes how to query all the online domain users who obtain authority from the M2000.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

The LMT login users consist of domain users and local users. The domain users can log in to the system only after obtaining authority from the M2000.

Procedure

  1. Run the LST ONLINEOP command on the LMT to query the online domain users.
