CPU and DSP

BSC boards are configured with the CPU and DSP chip for service processing and algorithm processing.

CPU

CPU: Central Processing Unit All the boards of the BSC have the CPU used for implementing the service control function and general service processing.

DSP

DSP: Digital Signal Processing In the BSC, only the GDPUC, GDPUX, and GDPUP have the DSP used for mathematical algorithm and coding/decoding algorithm processing.
