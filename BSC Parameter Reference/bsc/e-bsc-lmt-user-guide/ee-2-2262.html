Querying the Frequency of a BTS Board

This describes how to query the frequency of a specific BTS board according to the BTS index, slot ID, and slot ID.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Procedure

  1. Run the LST BTSBRDFREQ command on the LMT to query the frequency of a BTS board.
