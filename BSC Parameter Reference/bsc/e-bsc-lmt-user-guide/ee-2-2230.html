Maintaining the E1/T1 Links of the Pb Interface

This describes how to add, modify, or delete an E1/T1 link of the Pb interface.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • At least a PCU is configured.

Context

  • A GEIUP or GOIUP can be configured with a maximum of 32 E1/T1 links of the Pb interface.
  • The start PCIC must be a multiple of 128.
  • In E1 mode, timeslots 0, 1, 2, and 3 are synchronous and cannot be used by services.
  • When modifying the E1/T1 timeslots of the Pb interface, you can modify only the timeslots with the type of CIC or NULL.

Procedure

  • Run the LST PBE1T1 command on the LMT to query an E1/T1 link of the Pb interface.
  • Run the ADD PBE1T1 command on the LMT to add an E1/T1 link of the Pb interface.
  • Run the MOD PBE1T1 command on the LMT to modify an E1/T1 link of the Pb interface.
  • Run the RMV PBE1T1 command on the LMT to delete an E1/T1 link of the Pb interface.
  • Run the MOD PBE1T1TS command on the LMT to modify the E1/T1 timeslots of the Pb interface.
