Managing the Information on the Operator

This describes how to manage the account, password, description, operation authority, and operation time limit of the operator. Before operations, ensure that the operator has a valid account.

  • Querying the Operator Information
    If the user name is not provided, you need to query the brief information on all the users. If the user name is provided, query the operator information.
  • Adding the Operator Information
    This describes how to add an account and set the password, description, operation authority, and operation time limit of the operator. Administrators can perform this operation.
  • Removing the Operator Information
    On the LMT, only the admin users, administrators, and custom users with related authorities can perform this operation.
  • Modifying the Operator Information
    This describes how to modify the parameters recorded in the account of the local operator. On the LMT, only the ADMIN users, administrators, and custom users with related authorities can perform this operation.
  • Querying the Online Domain Users
    This describes how to query all the online domain users who obtain authority from the M2000.
  • Querying the Executable Command Groups
    This describes how to query the executable command groups.
  • Querying the EMS Information
    This describes how to query the login information of the network administrator. You can search the IP address of the network administrator only when the network administrator is online.
  • Querying the Local Non-Default Account Lock Status
    This describes how to query the lock and unblock status of the local non-default accounts. The non-default accounts do not include the GUEST and ADMIN accounts. The non-default accounts can be deleted through commands.
  • Setting the Local Non-Default Account Lock Status
    This describes how to set the local non-default accounts to lock or unblock status.
  • Checking Whether the Management Status of the Local Users Is Activated
    This describes how to check whether the management status of the local users is activated.
  • Activating the Management Function of the Local Users
    This describes how to activate the management function of the local users.
