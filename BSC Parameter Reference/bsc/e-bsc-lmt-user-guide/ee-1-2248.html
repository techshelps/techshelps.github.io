BTS attributes

This describes the BSC attributes, including the site attributes, cell attributes, TRX attributes, and channel attributes. The users can learn the parameters when querying the BTS attributes.

Site Attributes

Table 1 describes the site attributes.

Table 1 Site attributes

Attribute Name

Description

Manufacturer ID

Information about the manufacturer of the BTS to be queried.

OML attribute

Port number, timeslot number, and sub-timeslot number of the OML.

Operation state

Whether the current site is available.

Terminal equipment ID

Terminal equipment ID according to the LAPD protocol. The attribute here indicates the equipment ID of the current site, that is, the terminal equipment identifier (TEI) of the OML.

Clock attribute

Work status of the BTS clock and information related to the clock parameters.

Extended alarm configuration

Configuration of the extended alarms of the current site.

Software configuration

Software configuration of the current site.

Power supply system type

Power supply system type of the BTS cabinet.

SFP Mode

Connection attributes of the SFP port.

TOP Switch Parameters

Information about the port and slot of the TOP switching parameters on the backplane.

Hardware configuration

Hardware configuration of the current site.

Cell attributes

Table 2 describes the cell attributes.

Table 2 Cell attributes

Attribute Name

Description

Management state

Management state of the current cell. You can set the state by choosing BTS Maintenance > Modify Administrative State on the LMT.

BCCH ARFCN

Frequency of the main BCCH of the current cell.

BTS Identity Code

Consisting of BSIC, NCC, BCC.

Cell air-interface timer

Duration of timer T3105. This attribute is used in asynchronous cell handover. Unit: ms

CCCH load indication period

Interval for sending the overload messages.

CCCH load threshold

Ratio of times of successful random access to total collision times within the time.

Connection failure criterion

Detailed criterion for determining the connection failure.

Interference average parameter

Average time for obtaining the interference level. The default value is 20 multiframe.

Interference level threshold

The interference can be divided into six ranks according to the interference signal level. The level value for dividing each rank is called interference level threshold.

The BTS determines which rank the current interference level is in and reports this information to the BSC on the basis of the radio resource indication.

Maximum timing advance

Maximum timing advance of the most distant area of a cell. This attribute can identify the actual coverage of a cell.

NY1

Maximum times of sending physical messages.

Operation state

Whether the current site is available.

Overload period

The BTS calculates the channel usage at a fixed interval to check whether the channel is overloaded. The interval is determined by the overloading period. Unit: s

RACH busy threshold

Determining the level threshold of random access. Unit: dBm

RACH load average timeslots

Measuring the average loading timeslot number for random access channel.

Start frame No.

The command can be run in the case of this frame number. If you want to run the command immediately, set the frame number to 65535.

The value range of frame number is 0 to 42432, and 65535.

T200

Consisting of the following parameters:

SDCCH: timeout duration of T200 defined on the SDCCH over the Um interface

FACCH/Full rate: timeout duration of T200 on the FACCH or full-rate channel at the Um port

FACCH/Half rate: timeout duration of T200 on the FACCH or half-rate channel at the Um port

SACCH with TCH SAPI0: timeout duration of T200 on the SACCH when the TCH at the Um port supports the SAPI0 service

SACCH with SDCCH: timeout duration of T2000 on the SACCH with SDCCH

SDCCH SAIP3: value of T200 when the SDCCH supports the SAPI3 service

SACCH with TCH SAPI3: timeout duration of T200 on the SACCH when the TCH at the Um port supports the SAPI3 service

VSWR threshold

VSWR threshold of the RF voltage, including the VSWR threshold when an antenna incurs a fault and the VSWR threshold when an antenna is not correctly adjusted.

Available state

Current state of the cell.

Frequency hopping mode

Determining whether to use the FH mode, and what kind of FH mode is used

Extended type & extended offset

Extended type and extended offset of the current cell.

Frame offset parameter

The default value of this attribute is 0. This indicates the cell is not offset. If the value is not 0, the handover mode need be changed from synchronous to asynchronous. If the value is 255, the BTS does not send this attribute. For BTS3001C and BTS2X BTSs, the value of this attribute is fixed to 255.

The BTS3X, BTS3002C and double-transceiver BTSs can predict the frame offset according to the value of frame offset parameter.

Interference band statistical algorithm

When the frequency point scanning function is enabled, interference band statistic algorithm I or interference band statistic algorithm II is used.

Uplink frequency scan

Uplink frequency scanning result.

TRX attributes

Table 3 describes the TRX attributes.

Table 3 TRX attributes

Attribute Name

Description

Management state

Management state of the current TRX. You can set the state by choosing BTS Maintenance > Modify Administrative State on the LMT.

ARFCN list

List of channel numbers based on which the channel works. In the case of none FH, each channel has one channel number. In FH, all the available channel numbers are listed in the ARFCN list.

Power level

The static power is divided into five levels. The value set here indicates the power level of the current TRX.

Output power threshold

Maximum RC Power Reduction

Maximum RC power reduction

Maximum power reduction of each level.

Operation state

Whether the TRX is available.

Available state

Current state of the TRX.

8PSK modulation power attenuation level

Power attenuation levels of the EDGE TRX. There are 50 levels, and the attenuation between levels is 0.2 dB.

RSL attribute

Port number, timeslot number and sub-timeslot number of the RSL link.

Terminal equipment ID

Information about the terminal equipment.

Channel Attributes

Table 4 describes the channel attributes.

Table 4 Channel attributes

Attribute Name

Description

Management state

Management state of the current channel. You can modify the state by choosing Modify Administrative State on the LMT.

ARFCN list

List of channel numbers based on which the channel works.

In the case of none FH, each channel has one channel number. In FH, all the available channel numbers are listed in the ARFCN list.

Operation state

Whether the channel is available.

Channel combination

Combination type of the current channel.

Hopping sequence No.

Numbers of 64 kinds of FH. The number 0 indicates the FH in sequence. Other values indicate the FH of pseudo-random sequence.

Mobile allocation index offset

Frequency offset when the TRXs are in FH in the same cell.

Training sequence code

Demodulation part for estimating the channel feature to correctly decode the valid information. There are eight kinds of training sequence codes.

Available state

Current state of the channel.
