Checking LMT Local Data

This describes how to check the local data. A prompt is displayed when the data inconsistency is checked.

Prerequisite

You have logged in to the LMT in offline mode.

Procedure

  1. Choose Configuration > Back Up > Check Local Data.
  2. The data is modified according to the error information.
