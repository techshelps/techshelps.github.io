Concepts of BSC System Monitoring

This provides the concepts of BSC system monitoring, including real-time monitoring, bit error rate, and CPU/DSP usage.

  • Real-Time Monitoring
    Real-time monitoring refers to monitoring some counters and functions in real time through the LMT when the BSC is in the running state.
  • Bit Error Rate
    This describes the definitions of the bit error rate.
  • CPU/DSP Usage
    The LMT can monitor the CPU/DSP usage of the BSC.
