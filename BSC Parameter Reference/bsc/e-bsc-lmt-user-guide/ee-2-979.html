Using the Local Maintenance Terminal

This describes how to start, lock, and exit the Local Maintenance Terminal. This also describes how to set system parameters, office information, and output parameters. You can configure the system through Man Machine Language (MML) on the LMT.

  • MML-Related Concepts
    This describes the MML command checking rules and troubleshooting of the faults related to the MML commands.
  • Starting the Local Maintenance Terminal.
    This describes how to start the Local Maintenance Terminal and connect it to the BSC.
  • Locking the Local Maintenance Terminal
    This describes how to manually lock the Local Maintenance Terminal to avoid insecure operations on the Local Maintenance Terminal after operators leave.
  • Setting System Parameters
    This describes how to set system parameters, including setting output window and terminal.
  • Setting the Office Information
    This describes how to add, modify, delete, import, export, and find office on the Local Maintenance Terminal.
  • Setting Output Parameters
    This describes how to set output parameters to save the report in file format.
  • Setting MML Command Timeout Parameters
    This describes how to set MML Command Timeout (s).
  • Saving the Input Commands
    This describes how to automatically save the successful and failed MML command.
  • Running MML Commands in Batches
    This describes how to run MML commands in batches. Running MML commands in batches refers to running multiple MML commands at a time. The pre-edited batch file is run on the Local Maintenance Terminal so that you do not need to manually run every MML command.
  • Exiting the Local Maintenance Terminal
    This describes how to exit the Local Maintenance Terminal.
