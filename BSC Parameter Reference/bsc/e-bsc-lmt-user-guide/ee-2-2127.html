Querying Shielded Alarms

This describes how to query the shielded alarms according to the alarm shield conditions. The alarm shield conditions consist of alarm ID and alarm object.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Procedure

  1. Run the LST OBJALMSHLD command on the Local Maintenance Terminal to query the shielded alarms according to the alarm shield conditions.
