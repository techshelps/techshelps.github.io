Operation Time Limit

This describes the definition of the operation time limit, principle of operation time limit, and authority to set operation time limit.

Definition of Operation Time Limit

The operation time limit refers to the period during which a user can operate the system.

  • The ADMIN users have no operation time limit.
  • The operation time limit for an external user is set when the user name is added. If no operation time limit is set, the user can operate the system at any time.

Principle of Operation Time Limit

The operation time limit consists of the date, day, and time. The actual operation time allowed by the system is the intersection of these settings.

Authority to Set Operation Time Limit

The ADMIN users, administrators, custom-level users with related authorities can set and change the operation time limit for external users.
