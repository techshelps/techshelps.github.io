Browsing the CPU/DSP Usage Offline

This describes how to analyze the system history load and quickly locate faults by reviewing the CPU/DSP usage offline.

Prerequisite

  • The LMT runs normally.

Procedure

  1. On the Trace & Monitor tab page, choose Monitor > Browse CPU/DSP Usage Offline. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Set CPU/DSP Usage Browsing Parameter dialog box

  2. Set parameters in View Setting and Line Setting areas, and then click in the Options area.
  3. Select the files to be browsed, and then click Open to refresh Figure 1.

    NOTE:

    The files to be reviewed cannot be the files being viewed online. Or the files to be reviewed cannot be opened.

  4. Click Start. The browsing results are displayed, as shown in Figure 2.

    Figure 2 Browsing CPU/DSP usage offline
