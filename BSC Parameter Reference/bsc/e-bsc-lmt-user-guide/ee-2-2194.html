Maintaining the Line Clock

This describes how to maintain the line clock by querying, adding, and removing the line clock.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication within the BSC is normal.

Context

The line clock cannot be configured for the GEPS.

The configuration of the line clock for the electrical interface board is as follows:

  • The line clock of the GMPS can be configured only on the Ater connection path of the current GMPS.
  • The line clock of the GTCS can be configured only on the A interface E1 of the current subrack.

The configuration of the line clock for the optical interface board is as follows:

  • The line clock can be configured for the GMPS if the GOIUT is configured for the GMPS.
  • The line clock can be configured for the GTCS if the GOIUA is configured for the GMPS.
  • The line clock for the optical interface board can be configured only at port 0.

Procedure

  • Run the LST LINECLK command on the LMT to query the line clock.
  • Run the RMV LINECLK command on the LMT to remove the line clock.
  • Run the ADD LINECLK command on the LMT to add the line clock.
