Tracing Messages

This describes how to trace the signaling messages on the BSC interfaces.

  • Concepts of BSC Interface Signaling Tracing
    Tracing management is used in the routine maintenance of equipment. You can trace the interfaces and signaling links to verify data and locate faults. The concepts of BSC interface signaling tracing are message types and tracing modes.
  • Tracing CS Messages
    This describes how to trace the CS messages such as the messages on the A interface, Abis interface, Pb interface (when the external PCU is configured), Um interface, and BSC-CBC interface, user messages, group call messages, and IP messages.
  • Tracing PS Messages
    This describes how to trace messages on the Gb interface, Um interface and User messages.
  • Managing Trace Task
    This describes how to query and delete tracing tasks based on interface type, tracing mode, task status, and time setting.
  • Managing Trace Files
    This describes how to query the tracing files on the server and how to load the files to the LMT.
  • Reviewing Trace Message Files
    This describes how to browse the interface tracing files of the same type saved on the local PC. You can view the files by uploading the files from the server to the LMT.
