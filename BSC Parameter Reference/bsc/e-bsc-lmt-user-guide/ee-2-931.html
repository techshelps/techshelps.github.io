Importing Network Planning Data

This describes how to import the network planning data and modify the network planning data.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

If you log in to the LMT in offline mode, you can import the network planning data from the local PC to the LMT PC. If you log in to the LMT in online mode, you can import the network planning data to the server.

CAUTION:
  • The imported network planning data is in *.txt format.
  • The format of the imported network planning data files should be the same as that of the default exported network planning data files. If not, the import fails.
  • The network planning data file is used for adjusting the neighboring relations of originating cells only. If a cell is not included in the list of original cells, the neighboring relation of the cell is not adjusted.
  • If you delete the neighboring relations of a cell, see Changing Reciprocal Relations Between Cells.

Procedure

  1. Choose Configuration > Import or Export Data > Import Network Plan Parameter. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Import Network Plan Parameter dialog box

  2. Click and select the file that needs to be imported.
  3. Click Finish. A message is displayed that the import is successful. Importing the network planning data is complete.

Postrequisite

If the import fails, the failure message is displayed. You can check the error information in ImportErr_Freq.log file in LMT installation directory\BSC6000\version number\Bin, modify the parameters, and import them again.
