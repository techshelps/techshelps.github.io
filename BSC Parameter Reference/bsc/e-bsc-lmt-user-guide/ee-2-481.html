Querying Board Version Information

This describes how to query the information about the software, hardware and patches of a board, and the mode of loading the board on the LMT.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The board to be queried is running normally.

Context

The patch information includes all patch packages related to each software version in both the active and standby areas.

There are three loading modes: Auto, From Server and From Flash. For details of the three loading modes, refer to BSC Software Loading.

Procedure

  • Through GUI
    1. Choose BSC Maintenance > Query Software > Query Board Version Information. The Query Board Version Information window is displayed, as shown in Figure 1.

      Figure 1 Query Board Version Information window (1)

    2. On the Software and Hardware Information tab page, set parameters in the Query Mode and Board Location areas, and then click Query. The results are displayed, as shown in Figure 2.

      Figure 2 Query Board Version Information window (2)

      NOTE:

      Click the Patch Information and Loading Option tabs in Figure 1 to query the patch information and loading options.

  • Through MML
    1. You can run the following commands on the LMT to query detailed information of BSC boards.

      • Run the DSP BRDVER command to query the board versions.
      • Run the DSP PATCH command to query the information about the board patch.
      • Run the LST BRD command to query the attributes of the board loading.
