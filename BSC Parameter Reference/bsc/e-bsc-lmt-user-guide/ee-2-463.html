Maintaining BSC User Resources

This describes how to maintain the BSC user resources. The user resources consist of port timeslot, interface circuit, TC resources, call resources, DSP speech channel, external speech channel, and internal speech channel.

  • Querying the Resources of An MS
    This describes how to query the usage of single-user resources in the CS and PS. The resources include the common information and mobile information about the MS and the information about BM subrack and TC subrack.
  • Collecting the BSC Local Information
    Collecting the BSC local information involves setting the information collection, collecting the local information, and generating the collection report. The collection of the BSC local information helps the maintenance personnel to analyze and locate problems.
  • Looping Back Remote Speech Channel
    This describes how to start or end the remote speech channel loopback and how to query the status of the current remote speech channel loopback.
  • Testing the Internal Speech Channel
    This describes how to test the internal speech channel and display the testing results. To perform this task, you need to enter the MS ID.
  • Looping Back the DSP Speech Channel
    This describes how to perform loopback on DSP speech channel to locate the fault on the DSP speech channel.
  • Downloading TC Files
    This describes how to download TC files to a specific DSP.
  • Uploading TC Files
    This describes how to upload the audio TC files in the data\mtndata\tcFile directory to the local disk.
