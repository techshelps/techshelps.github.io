Maintaining the BTS Devices

Maintaining the BTS devices involves maintaining the BTS battery, power module, auxiliary equipment, and RET antenna. You can perform the operations associated with the BTS equipment to ensure that the BTS operates normally. The operations involve querying, testing, viewing, and resetting.

  • Querying BTS Running Status
    This describes how to query the running status of the BTSs or cells under a BSC.
  • Querying the Status of a Power Module
    This describes how to query the status of a BTS power module, including the status of current limiting, charge, and discharge, and the type of the BTS power module. This also describes how to query the number of DPSUs and the position of the DPSU.
  • Querying the Temperature in Equipment Room
    This describes how to query the temperature in the equipment room.
  • Querying BTS Attributes
    This describes how to query BTS attributes. By querying BTS attributes, you can learn the operating status and attributes of a BTS in real time and verify the information on the manufacturer, software configuration, and hardware configuration of the BTS.
  • Querying the BTS IP Performance Measurements
    This describes how to query the IP performance measurements of an IP BTS working in the FE/GE mode. If the IP BTS working in the FE/GE transmission mode is faulty, you can run the Query the Stat of the BTS IP performance command on the LMT to query the IP performance measurements of the faulty BTS, and thus identify the problem.
  • Querying the Location Group Attributes
    This describes how to query the location group attributes when co-cell is enabled. You can know the related parameters by querying the location group information.
  • Setting the Master Location Group
    This describes how to set the master location group. When the co-cell is enabled, you can perform the switchover between the master location group and slave location group by setting the master location group on the LMT.
  • Testing the BTS Running Status
    This describes how to detect the fault in the BTS by testing the running status of all boards through the LMT.
  • Testing Transport Performance
    This describes how to test the E1 self loop and IASU self loop and view the testing results through a testing instrument.
  • Measuring the BTS VSWR
    This describes how to measure the voltage standing wave ratio (VSWR) of a BTS. You can run the Measure the BTS VSWR command on the LMT to remotely measure the VSWR or query the information of history VSWR. Thus, you can determine whether the antenna system is properly installed and minimize the impacts of system (including the devices and antenna system) breakdowns on the subsequent network optimization.
  • Managing Batteries
    This describes how to obtain the running status of BTS batteries by performing a discharge test. In addition, this task describes how to query the installation time, power-off count, charge count, and discharge count.
  • Setting/Querying Power Module Parameters
    This describes how to query the power module parameters, including the power parameter, battery parameter, battery management parameter, and the production date of the PSU. This also describes how to restore the parameters to default values.
  • Maintaining the RET Antenna
    This describes how to scan the RET antenna information, set the RET antenna tilt, and reset, enable, calibrate, load and reset switch the configuration data of the RET antenna.
  • Maintaining the BTS Ring Network
    This describes how to maintain the ring network parameters. The user can manually rotate the forward link and reverse link.
  • Maintaining the BTS Antenna System
    This describes how to maintain the BTS antenna system. The operations involve querying and modifying the parameters associated with the BTS antenna system.
  • Monitoring BTS Resources
    This describes how to monitor the resource usage of the TRX and of the main control board on the LMT.
  • Monitoring the BTS Environment
    This describes how to monitor the site environment, for example, to reset the alarm box, clear burglar alarms, disable the EAC alarm, obtain temperature and humidity, and clear smoke alarms.
  • Monitoring Channel Status
    This describes how to monitor the usage of the channels and subchannels of a TRX through the LMT and check the availability status of the channel.
  • Monitoring Channel Interference Band
    This describes how to monitor the channel interference through monitoring the interference band of idle channels.
  • Modifying BTS Administrative State
    This describes how to modify the administrative state of the cell, TRX, and channel through the LMT. The BSC reset does not affect the attributes of the administrative state.
  • Handing Over a Cell Forcibly
    This describes how to hand over a call from one cell to another to ensure that no call drops occur during BTS maintenance.
  • Resetting the Auxiliary Equipment
    This describes how to reset the heat exchanger on the BTS3006A. The HEU of the BTS3006A monitors the running of the heat exchanger. When the heat exchanger incurs an error, the HEU reports an alarm through the main control board. You can reset the heat exchanger through the LMT.
  • Resetting the BTS Hardware
    This describes how to reset the hardware of a BTS.
  • Resetting BTS in Levels
    This describes how to reset a BTS or the cells under the BTS to update the configuration data.
  • Browsing BTS Initialization Process Message
    This describes how to browse the initialization process of all sites. By browsing the initialization process, you can see whether the BTS is reset. If the initialization fails, this message shows the failure cause, thus helping to locate the fault.
  • Maintaining the BTS Through Optical Transport Board Commands
    This describes how to send optical transmission board commands to maintain the BTS.
