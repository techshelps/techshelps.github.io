Querying the OMLs of the Ater Interface

This describes how to query the OMLs of the Ater interface according to the index ID of the Ater connection path.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication within the BSC is normal.
  • The Ater connection path is configured.

Context

You do not need to enter any parameters when querying the OMLs of the Ater interface. All the OMLs that are configured in the BSC are displayed.

Procedure

  1. Run the LST ATEROML command on the LMT to query the OMLs of the Ater interface.
