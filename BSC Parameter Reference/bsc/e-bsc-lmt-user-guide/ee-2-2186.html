Swapping the GOMUs

You can swap the GOMUs so that the original standby GOMU is switched to act as the active one, and the original active POMU is switched to act as the standby one.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • Start the DAEMON process on the active and standby GOMUs.

Context

  • The swapping operation disrupts the communication between the LMT and the server, thus disrupting some time-consuming OM operations. Be careful with this operation.
  • If the server is the GBAM or if not both the active and standby GOMUs are operational, the operation fails.
  • Make sure that the active and standby GOMUs backs up the files in each other. If the backups are not ready, the operation fails.

Procedure

  1. Run the DSP OMUSWAPSWITCH command on the Local Maintenance Terminal to query the setting of swapping.
  2. Check the status of the swapping switch.

    Status

    Operation

    		 
    swap switch = disable swap

    Swapping is disabled. Go to 3.

    		 
    swap switch = enable swap

    Swapping is enabled. Go to 4.

  3. On the LMT, run the SET OMUSWAPSWITCH command to enable swapping.
  4. Run the SWP OMU command on the Local Maintenance Terminal to swap the GOMUs.
