CPU/DSP Usage

The LMT can monitor the CPU/DSP usage of the BSC.

The related concepts are:

  • CPU

    Central Processing Unit

  • DSP

    Digital Signal Processing

  • Usage

    Ratio of working time to total time. Total time equals working time plus idle time.
