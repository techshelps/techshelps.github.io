Refreshing LMT Data

This describes how to refresh the LMT data to ensure the synchronization with the server.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Procedure

  1. Choose Configuration > Refresh LMT Data.
