Backing Up LMT Local Data

This describes how to save the configuration data on the LMT to the local PC.

Prerequisite

You have logged in to the LMT in offline mode.

Context

  • The default path for backing up LMT local data is LMT installation directory\version number\BackupCfgData.
  • You can use the backed-up data file to configure data when you log in to the LMT next time. You can also restore the configuration data with the backup data.
  • You can specify the directory and file name for the backup data. The backup data must be saved as a *.dat file.
  • If the name of the file to be saved already exists on the local PC and the existing file is not a read-only file, the file to be saved overwrites the existing file.

Procedure

  1. Choose Configuration > Back Up > Back Up Local Data.
  2. Type the File name, and then click Save.
