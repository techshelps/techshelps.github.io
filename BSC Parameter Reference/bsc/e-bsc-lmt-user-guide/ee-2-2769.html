Setting the Master Location Group

This describes how to set the master location group. When the co-cell is enabled, you can perform the switchover between the master location group and slave location group by setting the master location group on the LMT.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

Only one master location group exists in a co-cell. The board types between the master location group and slave location group need not be consistent. The radio carrier number, however, must be consistent with the number of radio carriers.

Procedure

  • Through GUI
    1. On the LMT, choose BTS Maintenance > Maintain Site > Set Master Location Group. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Set Master Location Group dialog box

    2. Set the parameters in the displayed dialog box, and then click OK. A dialog box is displayed, prompting that setting the master location group succeeds.
  • Through MML
    1. Run the SWP LOCGRP command on the LMT to perform the switchover between the master location group and the slave location group.
