Removing the Operator Information

On the LMT, only the admin users, administrators, and custom users with related authorities can perform this operation.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

  • If you remove the account that is used currently, this account is logged out forcibly.
  • Do not remove the system accounts, such as admin.
  • Do not remove your account.

Procedure

  1. Run the RMV OP command on the LMT to remove the operator information.
