Password

The LMT identifies a user by the user name and password. This provides the definition of the LMT user password and describes the initial setting and modification of the password.

Definition

To ensure the system security, each user account has a password. When starting the LMT, users must enter valid user names and passwords to log in to the server.

A valid password must have six to thirty-two characters, be case sensitive, and contain no double quotation marks.

Initial Setting of the Password

The initial password of the ADMIN account is admin. The password for external users is set by ADMIN users, administrators, or custom users with related authorities during user account addition.

Password Modification

All the users who have logged in to the LMT can modify their own passwords. The ADMIN users and administrators can modify the passwords of all external users. Only ADMIN users can change the passwords of the ADMIN account.
