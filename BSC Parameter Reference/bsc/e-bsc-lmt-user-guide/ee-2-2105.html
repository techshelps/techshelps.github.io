Managing the Command Group

This describes how to manage the commands in a command group. As the minimum authority unit, the command groups are allocated to the accounts of different levels.

Context

There are 32 command groups ranging from G_0 to G_31.

  • The command groups ranging from G_0 to G_19 are default command groups. You are not allowed to configure the commands in the specified command group.
  • You can use the administrator account or the custom account with related configuration authority to configure the command groups ranging from G_20 to G_31.

    This describes how to query all the commands in a specified command group.

  • Removing Commands from a Command Group
    This describes how to remove the commands from a specified command group.
