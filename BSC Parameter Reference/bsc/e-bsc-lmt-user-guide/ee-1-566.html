Concepts of BSC Data Configuration

This describes the concepts of BSC data configuration, which requires the cooperation of the server, LMT, and M2000. The BSC data is configured on the LMT. The LMT supports data configuration both online and offline. The configuration data can be saved in binary files to meet the requirements of different users and different scenarios.

  • BSC Configuration Data Types
    Based on the system requirements and the location for saving the data, the BSC configuration data is classified into different types.
  • BSC Data Check
    The BSC data check consists of the data validity check and data consistency check.
  • BSC Data Synchronization
    The BSC data synchronization is the process in which the data on each board is synchronized with the data in the GBAM/GOMU database.
