Cell Forcible Handover Mode

This describes the cell forcible handover function of the BSC. The BSC selects a target cell according to the measurement report, thus improving the rate of successful handover.

The LMT provides the following forced handover modes:

  • Free

    Hand over a user from one channel to another. The target cell can be specified by the LMT at random.

  • Intra-cell

    Hand over a user from one channel to another. The success rate of this handover is the highest. Before handing over a user from the faulty TRX, close the channel or TRX that needs to be maintained.

  • Outgoing Cell

    Hand over all users of a cell to another cell.

  • Outgoing Site

    Hand over all users of one cell to another. The target cell must be outside the current site.

  • Outgoing BSC

    Hand over all users of one cell to another. The target cell must be the external cell of the BSC.

  • Outgoing System

    Hand over all users of one cell to another. The target cell can be switched between 2G and 3G systems.

  • Specified Cell

    Hand over all users of one cell to another. The target cell is specified previously.
