Measuring the BTS VSWR

This describes how to measure the voltage standing wave ratio (VSWR) of a BTS. You can run the Measure the BTS VSWR command on the LMT to remotely measure the VSWR or query the information of history VSWR. Thus, you can determine whether the antenna system is properly installed and minimize the impacts of system (including the devices and antenna system) breakdowns on the subsequent network optimization.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

  • VSWR measurement is a key item in the accptance test at the early stage of site construction or at the turnkey stage.
  • In routine maintenance, the BSC6000 supports the VSWR alarm function. When the BTS reports a VSWR alarm, you should clear the alarm on site.

Procedure

  1. Run the STR VSWRTEST command on the LMT to start the VSWR measurement.
