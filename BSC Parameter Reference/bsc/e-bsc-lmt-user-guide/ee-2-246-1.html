Deploying the FTP server relay

This describes how to deploy the FTP server relay by running the MML command on the LMT. The network administrator is responsible for this operation in view of the network security. The LMT user need not know the details of the FTP server relay or set the FTP server relay on the LMT.

Context

If the firewall with Net Address Transfer (NAT) exists between the LMT (or M2000) and the OMU, and the NAT converts the IP address of the LMT or the M2000, the FTP server relay must be deployed. The FTP server relay can be deployed on either the OMU or other FTP server. If the NAT function is enabled, the LMT can cross the NAT only through the FTP server relay to download the data from the OMU to the LMT during the startup. Otherwise, users cannot maintain the BSC on the LMT.

Procedure

  1. Run the LST TRANFTP command to check whether the FTP server relay is configured and whether the configuration of the FTP server relay is correct.
  2. If the FTP server relay is not configured or is configured incorrectly, run the SET TRANFTP command to configure the FTP server relay. You can deploy the FTP server relay on either the OMU or other FTP server by doing as follows:

    • Deploy the FTP server relay on the OMU. In this case, the IP address of the FTP server relay is the IP address of the OMU.
    • Deploy the FTP server relay on other FTP server. In this case, you need enter the following information of the FTP server relay such as IP address, user name, password, and port number.
      NOTE:
      If the FTP server relay on other FTP server, the IP address of the OMU cannot be set to 0.0.0.0. Otherwise, the FTP server relay is unavailable.

  3. Run the STR FTPSSRV command to start the FTP server relay.

    NOTE:
    • If the FTP server relay is deployed on the OMU, you must run the STR FTPSSRV command to start the FTP service on the OMU. Otherwise, you cannot log in to the LMT.
    • If the FTP server relay is deployed on other FTP server, you need start the FTP service on the target FTP server. Otherwise, you cannot log in to the LMT.
