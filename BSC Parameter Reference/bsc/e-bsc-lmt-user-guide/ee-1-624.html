Bit Error Rate

This describes the definitions of the bit error rate.

The bit error rate is defined as follows:

  • Bit Error Rate

    Bit Error Rate indicates the number of bit errors that occur in the unit time.

  • Unit Time

    Unit Time indicates the sampling interval at which the interface board tests bit errors.

  • Remain Time

    Remain Time indicates the remaining time of the bit error rate test. When the remaining time is 0, the test ends.

  • Total Error

    Total Error indicates the total number of bit errors monitored during the test.

  • Unit Error

    Unit Error indicates the number of bit errors monitored at a sampling interval.
