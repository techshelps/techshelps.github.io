Managing IP Addresses

This describes how to manage the IP addresses of the devices, IP addresses of the physical ports, IP address routing information, IP addresses of the Ethernet ports, and peer-end IP addresses.

  • Maintaining the IP Addresses of the Devices
    This describes how to maintain the IP addresses of the devices.
  • Maintaining the IP Addresses of the Physical Ports
    This describes how to maintain the IP addresses of the physical ports.
  • Querying the Routing Information of the IP Addresses
    This describes how to query the routing information of the IP addresses.
  • Querying the IP Addresses of the Ethernet Ports
    This describes how to query the IP addresses of the Ethernet ports. The A, Abis, and Gb interfaces are involved.
  • Querying the Dynamic ARP Information of the Ethernet Ports
    This describes how to query the dynamic ARP information of the Ethernet ports by IP address.
  • Pinging the Peer-End IP Address
    This describes how to ping the peer-end IP address to check the connectivity of the network.
  • Tracking the Peer-End IP Address
    This describes how to track the peer-end IP address to check the connection of the boards.
