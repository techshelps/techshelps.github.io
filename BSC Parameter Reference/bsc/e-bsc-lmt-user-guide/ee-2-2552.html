Managing Performance Measurement Results

This describes how to download, convert, query, filter, and export performance measurement results.

  • Downloading Performance Measurement Results
    This describes how to log in to the server through the FTP to download the performance measurement results. The Performance Browser Tool provides two FTP downloading modes: downloading manually and downloading automatically.
  • Converting Performance Measurement Results
    This describes how to convert the files in *.dtd, *.xml, *.mrf, *.bz2, and *.zip formats to files that can be queried and viewed through the Performance Browser Tool.
  • Querying Performance Measurement Results
    This describes how to query and browse the performance result files in *.mrf or *.xmll format.
  • Filtering Performance Measurement Results
    This describes how to filter the performance measurement data. You can set the filter time, filter object, counter information, and filter conditions at the same time to filter the measurement data.
  • Exporting Performance Measurement Results
    This describes how to export the performance measurement data and then save the data to the local PC in *.csv format.
