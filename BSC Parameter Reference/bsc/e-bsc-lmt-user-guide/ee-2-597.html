Browsing Alarms Offline

This describes how to browse the alarm log files that are stored in the local PC.

Prerequisite

The LMT runs properly.

Procedure

  1. Choose Alarm Maintenance > Browse Alarm Offline. A dialog box is displayed.
  2. Select an alarm log file and click Open. The contents of the file are displayed, as shown in Figure 1.

    Figure 1 Contents of an alarm log file

  3. Right-click the alarm to be queried, and then choose Alarm Detail from the shortcut menu.

Postrequisites

As shown in Figure 1, the shortcut menu also provides the following options:

  • Save All Alarms: Save all the alarms to the local PC as a .txt file.
  • Save Selected Alarms: Save the selected log to the local PC as a .txt file.
  • Set Show Columns: Set the alarm information to be displayed.
  • Filter Alarms: Display the alarms that meet specific conditions.
