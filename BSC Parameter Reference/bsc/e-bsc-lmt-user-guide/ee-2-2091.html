Querying the Operator Information

If the user name is not provided, you need to query the brief information on all the users. If the user name is provided, query the operator information.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

The operator information does not include the password. The account of the operator is not case-sensitive.

Procedure

  1. Run the LST OP command on the LMT to query the operator information.
