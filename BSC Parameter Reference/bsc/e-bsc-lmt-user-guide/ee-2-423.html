Setting the Time Zone and the DST

This describes how to configure the time zone and the daylight saving time. The ADMIN users, administrators, operators, and custom users with related authorities can perform this operation.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

CAUTION:
If daylight saving time (DST) is configured on the NE side, you must configure the DST on the M2000 to ensure that the DST of the NEs on the M2000 can be updated. You are advised not to adjust the DST at N o'clock or half past N o'clock (N is an integral number).

Procedure

  • Through GUI
    1. Choose System > Config Time Zone and DST.
    2. Select Time Zone Configuration or DST Configuration and the set the related parameters. If you set Time Zone Configuration and DST Configuration at the same time, the dialog box is as shown in Figure 1.

      Figure 1 Configure Time Zone and Daylight Saving Time dialog box

    3. Click Submit. A dialog box is displayed, requesting you to confirm the configuration.
    4. Click OK.
  • Through MML
    1. Run the SET TZ command on the Local Maintenance Terminal to set the time zone, DST, and DST deviation.
