Managing Data

This describes how to configure, browse, back up and restore the data of the BSC and BTS on the LMT.

  • Concepts of BSC Data Configuration
    This describes the concepts of BSC data configuration, which requires the cooperation of the server, LMT, and M2000. The BSC data is configured on the LMT. The LMT supports data configuration both online and offline. The configuration data can be saved in binary files to meet the requirements of different users and different scenarios.
  • Importing and Exporting Data
    This describes how to import and export the data of network planning, network optimization, and BTSs. Data import and export facilitates the data modification.
  • Modifying Parameters in Batches
    This describes how to modify BTS and cell data in batches using MML commands.
  • Backing Up Data
    This describes how to save configuration data on the server or LMT. You can restore the configuration data with the backup data.
  • Verifying Data
    This describes how to verify the data files in the BSC server and the boards.
  • Collecting the Information about the Carrier Configuration
    This describes how to collect the information about the carrier configuration by subrack, interface board, cell, or site. The collected information contains the details of different types of carriers.
  • Collecting Statistics on Interface Board Resources
    This describes how to collect statistics on interface board resources on the LMT to provide engineering parameters. The interface board resources are the Abis interface board resources and the A interface board resources.
  • Managing the Local Data in the LMT
    This describes how to manage the local data in the LMT by loading and updating.
