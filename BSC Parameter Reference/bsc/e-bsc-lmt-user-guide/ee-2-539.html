Sending Cell System Message

This describes how to send the system messages of a cell. You can update the system messages of a cell through a cell. The BTS then broadcasts these messages to the MSs through the radio interface.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Procedure

  • Through GUI
    1. Choose BTS Maintenance > Maintain Cell > Send Cell System Message. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Send Cell System Message dialog box

    2. Set Operation Object, and then click Send. A dialog box is displayed, indicating that the message is successfully sent.
  • Through MML
    1. Run the SND SYSMSG command on the LMT to send cell system messages.
