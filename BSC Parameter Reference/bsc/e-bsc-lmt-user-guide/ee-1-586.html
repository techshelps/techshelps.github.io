Alarm ID

This describes alarm ID that is the main keyword of the alarm configuration table.

An Alarm ID specifies a kind of alarms uniquely. It is the main keyword of the alarm configuration table.

The BSS alarm IDs are in the range 1-65533.

The alarm ID helps determine the information such as alarm name, alarm handling, alarm causes, and troubleshooting parameters.

NOTE:
The troubleshooting parameter information does not contain the specific values of the parameters.
