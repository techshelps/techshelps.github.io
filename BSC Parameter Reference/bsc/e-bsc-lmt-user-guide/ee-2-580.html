Loading Other Configuration Data

This describes how to open the local data files in offline mode to load the configuration data in the files to the LMT.

Prerequisite

You have logged in to the LMT in offline mode.

Context

  • Before opening a data file, close all the windows for configuration and for browsing objects.
  • When the data in the opened local data file is inconsistent with the data on the LMT, a message dialog box is displayed, asking you whether to upgrade the data file. If you click Yes, the data file will be updated and match the version of the current LMT. Then you can load the updated local data file directly.
  • If the configuration data is large-sized, the CPU usage is high when you upgrade and save the configuration data. The LMT is down for about 10 minutes. You cannot perform any operation during the 10 minutes.

Procedure

  1. Choose Configuration > Open Local Data File > Others. A message is displayed, asking you whether to save the current data.
  2. Click Yes. A dialog box is displayed.
  3. Select the target data file, and then click Open.
