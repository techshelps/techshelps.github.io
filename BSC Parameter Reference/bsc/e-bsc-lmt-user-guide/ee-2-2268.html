Maintaining the NSVC

Maintaining the NSVC of the Gb interface involves querying, resetting, blocking, and unblocking the NSVC.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The NSVC of the GEPUG is configured.

Context

The NSVC indicates the end-to-end virtual connection between the BSC and the SGSN.

Procedure

  • Run the DSP NSVC command on the LMT to query the NSVC status.
  • Run the RST NSVC command on the LMT to reset the NSVC.
  • Run the BLK NSVC command on the LMT to block the NSVC.
  • Run the UBL NSVC command on the LMT to NSVC the NSVC.
