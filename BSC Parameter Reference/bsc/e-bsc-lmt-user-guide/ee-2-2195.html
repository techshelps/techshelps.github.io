Maintaining the EAC

This describes how to maintain the EAC.

Context

You can maintain the attributes of the EAC in batches:

  • If you enter the subrack number, the attributes of the EAC configured for specified subrack are queried or modified.
  • If you enter the subrack number, the attributes of the EACs in all subracks are queried or modified.

Procedure

  1. You can run the following commands on the Local Maintenance Terminal to maintain the EAC.

    • Run the LST EAC command to query the attributes of the EAC.
    • Run the MOD EAC command to modify the attributes of the EAC.
