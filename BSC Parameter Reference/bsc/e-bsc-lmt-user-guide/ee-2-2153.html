Querying the Usage of the BSC License

This describes how to query the actual usage, including the allowed traffic and occupied service resource, of the effective license.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The GXPUM in the GMPS subrack runs normally.

Procedure

  1. Run the DSP LICUSAGE command on the LMT to query the usage of the BSC license.
