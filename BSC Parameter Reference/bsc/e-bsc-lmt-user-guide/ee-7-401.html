Intended Audience

This document is intended for:

  • Network operators
  • Shift operators
  • System engineers
  • Field engineers

Organization

  • Changes in BSC LMT User Guide
    This provides the changes of the BSC LMT User Guide.
  • Introduction to LMT
    This provides the definition and functions of LMT. It also introduces the operation and maintenance system of the BSC and the configuration requirements of the LMT PC.
  • Components of the LMT
    This describes the LMT software items installed on the LMT PC, including the BSC6000 local maintenance terminal, convert management system, performance browser tool, and local maintenance terminal. Knowing the components of the LMT software helps the user to operate and maintain the network elements.
  • Installing the LMT Software
    This describes how to install the LMT software of V900R008 on the LMT PC.
  • Using the BSC6000 Local Maintenance Terminal
    This describes how to start, configure, use, and quit the BSC6000 Local Maintenance Terminal.
  • Managing BSC Access Authorities
    BSC access authorities are classified into the guest level, common user level, operator level, administrator level, and self-defined level. The first four levels of authorities cannot be changed. The self-defined level is defined by the administrator level.
  • Managing Logs
    This describes how to query operation logs, load log files, maintain CHR logs, and set log-loading parameters.
  • Alarm Management
    This describes how to browse, query, and maintain the BSS alarms and how to maintain alarm boxes and manage alarm log files.
  • Managing Software
    This describes how to manage the BSC software on the LMT.
  • Monitoring the BSC in Real Time
    This describes how to monitor the CPU/DSP usage and port BER and how to browse them offline.
  • Tracing Messages
    This describes how to trace the signaling messages on the BSC interfaces.
  • Managing Data
    This describes how to configure, browse, back up and restore the data of the BSC and BTS on the LMT.
  • Maintaining the BSC
    This describes how to maintain the hardware of the BSC6000 on the LMT.
  • Maintaining the BTS
    This describes how to remotely maintain the BTS through the LMT. The operations involve maintaining the BTS version, devices, and boards.
  • Maintaining a Cell
    This describes how to send system messages of a cell, and query and configure the frequency scanning.
  • Using the Performance Browser Tool
    This describes how to start, configure and use the performance browser tool.
  • Using the Convert Management System
    This describes how to start, use, and exit the convert management system. The convert management system runs on the LMT PC. When the server is not directly connected to the alarm box, the convert management system forwards the alarm instructions and alarm information to the alarm box to monitor the alarm box. As a result, the alarm box can provide audio and visual indications.
  • Using the Local Maintenance Terminal
    This describes how to start, lock, and exit the Local Maintenance Terminal. This also describes how to set system parameters, office information, and output parameters. You can configure the system through Man Machine Language (MML) on the LMT.
