Maintaining the BC

Maintaining the BC of the Gb interface involves querying, blocking, and unblocking the BC.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.
  • The BC of the Gb interface is configured.

Context

  • The BC indicates the bearer channel of the frame relay.
  • Query the BC protocol types and the timeslots occupied by the BC.

Procedure

  • Run the DSP BC command on the LMT to query the BC statistics.
  • Run the CLR BC command on the LMT to clear the BC statistics.
  • Run the BLK BC command on the LMT to block the BC statistics.
  • Run the UBL BC command on the LMT to unblock the BC statistics.
