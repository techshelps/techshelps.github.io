Reviewing Trace Message Files

This describes how to browse the interface tracing files of the same type saved on the local PC. You can view the files by uploading the files from the server to the LMT.

Prerequisite

  • The LMT runs normally.

Context

  • When performing this task, you can log in to the LMT in offline mode.
  • Different tracing messages cannot be opened in a window.

Procedure

  1. On the Trace & Monitor tab page, choose Trace > Review Tracing.
  2. Select the files to be reviewed, and then click Open.

    NOTE:
    Double-click one message and then you can see the detailed information.
