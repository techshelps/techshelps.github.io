Resetting BTS in Levels

This describes how to reset a BTS or the cells under the BTS to update the configuration data.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

CAUTION:

Reset the BTS with caution, for this operation may lead to site initialization, data reloading, and service disruption.

There are two levels of BTS resetting: level 3 and level 4. This task supports the level 3 and level 4 resetting of the cells and BTS.

  • Site reset

    • Level 3 reset: Based on the configuration data of the BTS stored in the memory, the BSC configures the BTS again to reset it.
    • Level 4 reset: Based on the data configuration stored in the database (DB), the BSC updates the data stored in the memory, and then configures the BTS again to reset it.

  • Cell reset

    • Level 3 and level 4 reset: Reset all the TRX boards in the cell. After all the TRX boards are reset, re-initialization is performed. Level 3 and level 4 reset functions are the same.

Procedure

  • Through GUI
    1. Choose BTS Maintenance > Reset BTS in Levels. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Reset BTS in Levels dialog box

    2. Set parameters in the Operation Object and Operation Status areas, and then click Reset.
  • Through MML
    1. Run the RST BTS command on the LMT to reset the BTS in level 4.
