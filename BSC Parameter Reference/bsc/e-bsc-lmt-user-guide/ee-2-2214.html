Querying the Dynamic ARP Information of the Ethernet Ports

This describes how to query the dynamic ARP information of the Ethernet ports by IP address.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication within the BSC is normal.

Context

ARP is the abbreviation of the Address Resolution Protocol.

Procedure

  1. Run the DSP ARP command on the LMT to query the dynamic ARP information of the Ethernet ports.
