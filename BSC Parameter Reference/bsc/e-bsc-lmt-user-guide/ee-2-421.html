Setting Office Information on the BSC6000 Local Maintenance Terminal

This describes how to set the office information, including adding an office, modifying an office, and removing an office on the BSC6000 Local Maintenance Terminal.

Prerequisite

  • The LMT runs normally.

Context

  • The LMT can manage multiple BSCs in different time. The information about these BSCs is stored in the LMT installation path \BSC6000\Common\BscInfo.cfg.
  • Adding an office is used to set a new BSC connection for the LMT.
  • Modifying an office is used to modify the BSC name and IP address.
  • Deleting an office is used to delete the existing BSC information on the LMT.

Procedure

  • Add an office.
    1. On the BSC6000 Local Maintenance Terminal, use one of the following two methods to display the BSC Management dialog box.

      • Choose System > BSC Management.
      • Click on the tool bar.

    2. The running status of the selected sites is displayed, as shown in Figure 1.

      Figure 1 BSC Management dialog box

    3. Enter the correct BSC Name, BSC IP, and Remark (optional), and then click Add. The added information is displayed in the lower pane of the dialog box.

      NOTE:

      The name and IP address of the BSC you want to add must be unique. The BSC name cannot contain illegal characters, such as "?", "\", and "|".

  • Modify an office.
    1. On the BSC6000 Local Maintenance Terminal, use one of the following two methods to display the BSC Management dialog box, as shown in Figure 1.
    2. Select the BSC whose information needs to be modified in the dialog box shown in Figure 1. The information about this BSC is displayed in the upper pane of the dialog box. Modify corresponding information in BSC Name, BSC IP, and Remark, and then click Modify.
  • Remove an office.
    1. On the BSC6000 Local Maintenance Terminal, use one of the following two methods to display the BSC Management dialog box, as shown in Figure 1.
    2. Select the BSC to be removed in the dialog box shown in Figure 1, and then click Delete.
