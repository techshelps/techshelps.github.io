Managing the Active Alarms

This describes how to manage the fault alarms that are not cleared in the BSC.

  • Setting Alarm Display Colors
    This describes how to set the display colors for the alarms that are already cleared and the alarms of different levels. If no display color is set, the default color is used.
  • Querying the Active Alarms
    This describes how to query active alarms and receive the reported alarms in real time. In addition, you can delete unimportant alarms by performing the Manually Clear Alarm operation and obtain the ways of clearing the alarms by viewing Alarm Detail.
  • Clearing BSC EAC Alarms
    This describes how to clear the smoke alarms, infrared alarms, and door status alarms of the BSC EAC.
