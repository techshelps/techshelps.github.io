Querying the BSC System Time

This describes how to query the BSC system time.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

If the daylight saving time is used, the DST flag is displayed.

Procedure

  1. Run the DSP TIME command on the LMT to query the BSC system time.
