Controlling TRX Power

This describes how to set the type of the power supply for the TRX board and how to set the status of the switches of the power supply.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

Only the BTS20 support this operation.

Procedure

  • Through GUI
    1. Choose BTS Maintenance > Maintain TRX > Control TRX Power.
    2. Set parameters in the Location and Setting areas, and then click Apply, as shown in Figure 1.

      Figure 1 Controlling TRX power

  • Through MML
    1. Run the MOD TRXDEV command on the Local Maintenance Terminal to modify the TRX equipment attributes.

      NOTE:

      The TRX equipment attributes include the power supply type and the status of the switches of power supply.
