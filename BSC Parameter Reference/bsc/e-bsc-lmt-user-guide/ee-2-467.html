Maintaining the Circuits on the Pb Interface

When the external PCU is used, this describes how to query the circuit state on the Pb interface, and how to block, unblock, and reset the Pb interface circuits based on the query results.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The GEIUP or the GOIUP is functional.

Context

The Pb interface is the interface between the Packet Control Unit (PCU) and the BSC. It is a Huawei internal interface. The BSC6000 can provide 128 Pb interface circuits of 16 kbit/s.

The status of the Pb interface circuits is of five types: Occupied, Failure, Blocked, Uninstalled, and Idle. For details of each type, see Table 1 in Maintaining the Circuits on the A Interface.

Procedure

  • Through GUI
    1. On the LMT, choose BSC Maintenance > Maintain User Resource > Maintain Pb Interface Circuit. The Maintain Pb Interface Circuit dialog box is displayed.
    2. As shown in Figure 1, set parameters in the Pb Interface Location area, and then click Query. The query results are displayed, as shown in Figure 1.

      Figure 1 Querying Pb interface state

      NOTE:

      In the Pb Interface Location area, if State is selected, the result will be displayed based on the filtering condition selected.

      If State is not selected, the status of all the Pb interface circuits within the CIC range will be displayed.

    3. Select one or several results and click Block, Unblock, or Reset.
  • Through MML
    1. You can run the following commands on the Local Maintenance Terminal to maintain Pb interface circuits.

      • Run the DSP PBCIC to query the status of Pb interface circuits.
      • Run the BLK PBCIC command to inhibit Pb interface circuits.
      • Run the UBL PBCIC command to uninhibit Pb interface circuits.
      • Run the RST PBCIC command to reset Pb interface circuits.
