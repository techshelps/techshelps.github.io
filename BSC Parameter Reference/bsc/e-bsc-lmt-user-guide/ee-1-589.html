Notification Type

This top describes the corresponding relations between alarms and notification types of NMS.

Table 1 lists the notification types of the NMS.

Table 1 Description of the NMS notification types

Notification Type

Related Alarms

Power system

Alarms related to the power system

Environment system

Alarms related to the environment of the equipment room, such as temperature, humidity, and door sensor

Signaling system

Alarms related to the channel associated signaling and common channel signaling

Trunk system

Alarms related to trunk circuits and trunk boards

Hardware system

Alarms related to boards, such as clock board and CPU board

Software system

Alarms related to the software

Operating system

Alarms generated during system operation

Communication system

Alarms related to the communication system, such as the communication between the BSC and the BAM

Quality of services

Alarms related to the quality of services

Processing fault

Other fault alarms
