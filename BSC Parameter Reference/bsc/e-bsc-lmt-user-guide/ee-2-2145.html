Loading the Patch

This describes how to load the patch from the patch area of the server to the patch area in the memory of a board. The patch file is also saved in the flash memory of a board.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

When the patch is successfully loaded, the status of the patch on the server becomes Deactivated, and the status of the patch loaded on the service boards also changes. Run the DSP PATCH command to query the patch status on the service boards.

Procedure

  1. Run the LOD PATCH command on the LMT to load the patch.
