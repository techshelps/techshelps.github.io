Monitoring the BSC in Real Time

This describes how to monitor the CPU/DSP usage and port BER and how to browse them offline.

  • Concepts of BSC System Monitoring
    This provides the concepts of BSC system monitoring, including real-time monitoring, bit error rate, and CPU/DSP usage.
  • Monitoring the CPU/DSP Usage
    This describes how to monitor the CPU/DSP usage to get an overview of the system load.
  • Browsing the CPU/DSP Usage Offline
    This describes how to analyze the system history load and quickly locate faults by reviewing the CPU/DSP usage offline.
  • Monitoring the BER
    This describes how to learn of the transmission quality of an E1/T1 port by monitoring the transmission link and port status.
  • Browsing BER Offline
    This describes how to analyze the transmission quality of an E1/T1 link and quickly locate faults by reviewing the BER on the E1/T1 port.
  • Monitoring Port Fault Seconds
    This describes how to monitor E1/T1 port fault seconds to know the transmission quality of the links on the E1/T1 port in real time.
  • Browsing Port Fault Seconds Offline
    This describes how to browse the statistics of the fault seconds on the E1/T1 port in offline mode to analyze the transmission quality of the E1/T1 link and optical cable and to quickly locate faults.
  • Monitoring GPRS User Flow
    This describes how to monitor the throughput of the GPRS users on the DSPs of the built-in PCU. including the uplink and downlink GPRS flow on the Um and Gb interfaces.
  • Monitoring the Abis HDLC Transport Usage
    This describes how to monitor the Abis HDLC transport usage to know the average transmit and receive rates over the HDLC channel.
  • Monitoring the Cell Performance
    This describes how to monitor the cell performance. The information about the cell performance is displayed in a curve or list.
  • Monitoring the DSP Service Resource
    This describes how to monitor the DSP service resources to collect the types of transmission services and to monitor the number of different types of transmission services.
