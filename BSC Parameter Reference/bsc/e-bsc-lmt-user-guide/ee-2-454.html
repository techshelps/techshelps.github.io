Maintaining Transmission and Signaling

This describes how to maintain the signaling links and the ports of interface boards.

  • Maintaining LAPD Links
    This describes how to query the status of the OML, RSL, EML, and ESL, and the status of the PBSL when the PCU is in external mode. This also describes how to perform loopback test on LAPD links and how to reset LAPD links.
  • Maintaining the SCCP
    This describes how to monitor the DSPs and the subsystems to understand the running of the SCCP signaling links. In addition, you can enable or disable the DSPs and the subsystems if required.
  • Maintaining MTP3 Links
    This describes how to query the link state, route state, and DSP state of MTP3 links, and how to activate, deactivate, disable, and enable MTP3 links on the basis of the query result.
  • Querying Status of MTP2 Links
    This describes how to query the status of MTP2 links.
  • Querying MTP2 Link Statistics
    This describes how to query the usage of MTP2 links through querying MTP2 link statistics.
  • Querying the Status of an Interface Board Port
    This describes how to query the configuration status, work mode, and availability of E1/T1 ports, optical ports, and the IP interface board and transmission optimization interface board, and how to measure the transmission performance of optical boards on the basis of the query results.
  • Looping Back the Interface Board Port
    This describes how to query the port number, loopback status, and loopback remaining time and how to start or stop the loopback on the basis of the query result.
  • Looping Back the Interface Board Port Timeslot
    This describes how to query the mode, status, and remaining time of the timeslot loopback, and how to start or stop the timeslot loopback on the basis of the query result.
