Maintaining the Fan Box

This describes how to query the attributes of the fan box and reset the fan box.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the fan box and the GSCU is functional.

Context

The fan box is used to maintain the working temperature of the BSC boards. To maintain the proper running of the fan box, you need to monitor and maintain the fan box in real time.

Procedure

  • Through GUI
    1. On the LMT, choose BSC Maintenance > Maintain Device > Maintain Fan Box.
    2. Set the subrack No. in Subrack No., select Query, and click Apply. The query results are displayed, as shown in Figure 1.

      Figure 1 Querying the fan box

      NOTE:

      To reset the fan box, select Reset and click Apply. To recover the speed of the fan box, select Recover and click Apply.

  • Through MML
    1. On the LMT, run the following commands to query fan box attributes and reset faulty fan boxes.

      • Run the LST BSCFAN command to query fan box attributes.
      • Run the RST FAN command to reset fan boxes.
      • Run the MOD BSCFAN command to modify fan box attributes.
