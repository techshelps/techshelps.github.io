Managing the Local Data in the LMT

This describes how to manage the local data in the LMT by loading and updating.

  • Loading the History Data
    This describes how to open the local data files in offline mode to load the latest used data configuration files to the LMT.
  • Loading Other Configuration Data
    This describes how to open the local data files in offline mode to load the configuration data in the files to the LMT.
  • Refreshing LMT Data
    This describes how to refresh the LMT data to ensure the synchronization with the server.
  • Browsing Configuration Data
    This describes how to browse configuration data on the LMT. You can browse data of different types at the same time, including the configuration data of the following: BSC, BTS, cell, TRX, channel, and other types.
