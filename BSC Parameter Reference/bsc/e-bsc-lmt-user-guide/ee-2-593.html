Controlling the Alarm Box

This describes how to reset the alarm box, stop the alarm sound, and turn off the alarm indicator.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the LMT and the alarm box is normal.
  • The alarm box is correctly connected.
  • If the alarm box is connected to the local PC, the convert management system must be open. For details, refer to Starting the Convert Management System.

Procedure

  • Through GUI
    1. Choose Alarm Maintenance > Control Alarm Box. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Control Alarm Box dialog box

    2. Set Alarm Box Location and Function Operation.

      NOTE:

      In Figure 1, the operations in the Function Operation bring different effects to the alarm box:

      • To reset the faulty alarm box, select Reset Alarm Box.
      • To stop the alarm sound of the alarm box, select Stop Alarm Sound.
      • To turn off the indicators mapping to alarms of different severity levels, select Reset Alarm Indicator.

    3. Click Set. Setting the alarm box is complete.
  • Through MML
    1. You can run the following commands on the Local Maintenance Terminal to maintain the alarm box.

      • Run the RST ALMBOX command to reset the alarm box.
      • Run the STP BOXSND command to stop the alarm sound.
      • Run the CLR BOXLGT to manually turn off the alarm indicators.
