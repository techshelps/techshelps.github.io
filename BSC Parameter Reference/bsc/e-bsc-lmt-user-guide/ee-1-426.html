Concepts of BSC Security Management

This describes the principles of the LMT software authority management from the following aspects: LMT user password, LMT user authority, and LMT operation time limit.

  • Principles of LMT Authority Management
    This describes the principles of the LMT authority management from the following aspects: user identification, operation authority, and operation time limit.
  • Password
    The LMT identifies a user by the user name and password. This provides the definition of the LMT user password and describes the initial setting and modification of the password.
  • User Authority
    This describes the authorities of the default users and non-default users of the LMT.
  • Operation Time Limit
    This describes the definition of the operation time limit, principle of operation time limit, and authority to set operation time limit.
