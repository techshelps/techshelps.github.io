Viewing the GOMU Information

This describes how to view the information about the hard disk, RAID, memory, Ethernet adapter, DAEMON status, and DAEMON monitoring process of the GOMU.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

The following commands can be used to query only the GOMU information. You cannot use them to query the GBAM information.

Procedure

  • Run the DSP OMUHD command on the LMT to query the hard disk information.
  • Run the DSP RAID command on the LMT to query the RAID information.
  • Run the DSP OMUMEM command on the LMT to query the memory information.
  • Run the DSP OMUNIC command on the LMT to query the Ethernet adapter information.
  • Run the DSP OMUDAEMON command on the LMT to query the DAEMON status.
  • Run the DSP OMUPROCESS command on the LMT to query the DAEMON monitoring process.
