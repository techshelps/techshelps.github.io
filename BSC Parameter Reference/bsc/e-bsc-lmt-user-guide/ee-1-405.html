Functions of the LMT

This describes the functions of the LMT. On the LMT, you can maintain and configure data for the BSC and also for the BTSs that are connected to the BSC.

BSC6000 Local Maintenance Terminal

The BSC6000 LMT provides a graphical user interface (GUI) and the following functions for the BSC6000: security management, configuration management, maintenance management, software management, equipment management, message tracing, real-time monitoring, performance result offline browsing, and alarm management.

Convert Management System

The convert management system is used to forward the alarm information, the control information, and the status information about the alarm box between the server and the alarm box.

Performance Browser Tool

The performance browser tool is used to parse performance description files on the server so that users can view and arrange them.

Local Maintenance Terminal

The local maintenance terminal provides Man Machine Language commands for the configuration and maintenance of the system. Moreover, the local maintenance terminal provides detailed online help information.
