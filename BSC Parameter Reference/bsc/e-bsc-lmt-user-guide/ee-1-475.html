Concepts of BSC Software Management

This describes the basic concepts of the BSC software management.

  • BSC Work Areas
    This describes the work area of the BSC. The work area is used to store all the versions of system software. The work area consists of running area, active area, and standby area.
  • Process of Validating BSC Version Software
    Board version software must be loaded when a board is started or upgraded online. This describes how to download, upload, activate (roll back), and validate BSC version software.
  • Process of Validating the Patch
    This describes how to download, upload, activate (roll back), and validate the BSC patch.
  • Process of Validating the License
    This describes how to validate the license. The license is used by a device to provide authorized service functions and capacity service and other extra services. The process of validating the license consists of generating, loading, and activating the license file.
