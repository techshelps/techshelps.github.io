Exiting the BSC6000 Local Maintenance Terminal

This describes how to exit the BSC6000 local maintenance terminal.

Prerequisite

All the maintenance and configuration operations are complete and the related data is saved.

Procedure

  1. Use one of the following methods to exit the BSC6000 Local Maintenance Terminal:

    • Choose System > Exit.
    • Click on the tool bar.

  2. A dialog box is displayed, asking you whether you have saved the current data.
  3. Click Yes.
