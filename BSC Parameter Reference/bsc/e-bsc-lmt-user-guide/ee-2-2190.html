Querying the Properties of a BSC Subrack

This describes how to query the properties of a BSC subrack. The properties are the rack number of the subrack, whether it is a GMPS, whether it is a remote subrack, and the logical type of the subrack.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication within the BSC is normal.

Context

You can query the properties of multiple subracks:

  • If you enter a subrack number, the system displays the properties of the specified subrack.
  • If you do not enter any subrack number, the system displays the properties of all subracks.

Procedure

  1. Run the LST BSCSUBRACK command on the Local Maintenance Terminal to query the properties of all BSC subracks.
