Importing and Exporting Data

This describes how to import and export the data of network planning, network optimization, and BTSs. Data import and export facilitates the data modification.

  • Importing Network Planning Data
    This describes how to import the network planning data and modify the network planning data.
  • Exporting Network Planning Data
    This describes how to export the network plan data of the cell to the local PC in *.txt format.
  • Importing Network Optimization Data
    This describes how to import the network optimization parameters and modify the related configuration data. The network optimization data include the basic parameters, the parameters associated with the idle mode, call control, handover data, power control, channel management, the GPRS attributes, other attributes, and so on.
  • Exporting Network Optimization Data
    This describes how to export the network optimization parameters of the cell to the local PC.
  • Importing Site Data
    This describes how to import the site parameters in batches to improve the configuration efficiency.
  • Exporting Site Data
    This describes how to save the network optimization parameters of a cell to the local PC in the form of *.sxd.
  • Exporting Measurement Object Formatted File
    This describes how to export Measurement Object Formatted File to the local disk.
  • Exporting Equipment Archive Information
    This describes how to export the BSC equipment data and BTS equipment data and save them as .xml files to view equipment archive information.
