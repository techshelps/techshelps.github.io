Creating a Custom Template

You can save the measurement counter in a custom template format through the Performance Browser Tool. The custom template can facilitate the query of the measurement counter.

Prerequisites

  • The Performance Browser Tool runs normally.
  • You have logged in to the FTP server of an NE.

Procedure

  1. In the navigation tree of the Browse Description File window, choose NE Measurement Information > Template Detail > Custom Template.
  2. Right-click Custom Template, and then choose Add. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Creating a custom template

  3. In the Add dialog box, select a measurement counter, and then click Save Template.
  4. The Info dialog box is displayed. Enter the template name.
  5. Click OK to save the template and to return the dialog box shown in Figure 1.
  6. Repeat 3, 4, through 5 to create other custom templates.
  7. Click Close.
