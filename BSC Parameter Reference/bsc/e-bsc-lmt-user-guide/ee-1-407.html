Components of the LMT

This describes the LMT software items installed on the LMT PC, including the BSC6000 local maintenance terminal, convert management system, performance browser tool, and local maintenance terminal. Knowing the components of the LMT software helps the user to operate and maintain the network elements.

  • BSC6000 Local Maintenance Terminal
    This describes the BSC local maintenance terminal, which is a component of the LMT software. It adopts GUI to operate and maintain the BSC6000, configure data for the BSC6000, and provide detailed online help for the system.
  • Convert Management System
    This describes the Convert Management System that is one of the software items of the LMT software package. The Convert Management System is used to forward the alarm information, the control information, and the status information about the alarm box between the server and the alarm box. The convert management system connects to the server through the Ethernet port and communicates with the alarm box through the serial port.
  • Performance Browser Tool
    This describes the Performance Browser Tool, which is a component of the LMT software. The Performance Browser Tool is used to parse performance description files on the server so that users can view and arrange them.
  • Local Maintenance Terminal
    The Local Maintenance Terminal is a component of the LMT software. It provides Man Machine Language commands for the configuration and maintenance of the system. Moreover, the Local Maintenance Terminal provides detailed online help information.
