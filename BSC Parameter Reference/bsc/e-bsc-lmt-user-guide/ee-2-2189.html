Maintaining a Subrack

You can maintain a subrack by querying its properties and resetting it.

  • Querying the Properties of a BSC Subrack
    This describes how to query the properties of a BSC subrack. The properties are the rack number of the subrack, whether it is a GMPS, whether it is a remote subrack, and the logical type of the subrack.
  • Resetting a Subrack
    This describes how to reset all the boards in a subrack. The ADMIN users, administrators, operators, users, and custom users with related authorities can perform this operation.
