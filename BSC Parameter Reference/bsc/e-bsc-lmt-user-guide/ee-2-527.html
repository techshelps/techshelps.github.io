Monitoring Channel Interference Band

This describes how to monitor the channel interference through monitoring the interference band of idle channels.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

The interference band is the uplink interference level reported by the BTS to the BSC through an RF RESOURCE INDICATION message. The interference band is divided into five ranks. The level range of the five ranks can be set through data configuration.

If the rank of an interference band is high, the interference level is also high. If the interference band of an idle channel is in rank 4 or 5, the channel interference exists.

Only the interference band of speech channels and SDCCHs can be monitored.

Procedure

  • Through GUI
    1. Choose BTS Maintenance > Monitor Channel Interference Band.
    2. As shown in Figure 1, set the parameters in the Operation Object area, and then click Start. The results are displayed, as shown in Figure 1.

      Figure 1 Monitoring the channel interference band

    3. Click Stop to end the monitoring, or click Refresh to manually refresh the monitoring information.
  • Through MML
    1. Run the DSP CHNJAM command on the LMT to monitor the channel interference band.
