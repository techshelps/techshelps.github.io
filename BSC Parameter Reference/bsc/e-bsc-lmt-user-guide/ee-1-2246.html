Concepts Related to BTS Maintenance

This describes the basic concepts of BTS maintenance, including the logical objects, BTS attributes, and cell forcible handover mode. You can learn the parameters associated with the BTS attributes and forced cell handover.

  • Logical Objects
    This describes the basic concepts of BTS logical objects. The logical objects of the BTS consist of the site, cell, TRX, and channel.
  • BTS attributes
    This describes the BSC attributes, including the site attributes, cell attributes, TRX attributes, and channel attributes. The users can learn the parameters when querying the BTS attributes.
  • Cell Forcible Handover Mode
    This describes the cell forcible handover function of the BSC. The BSC selects a target cell according to the measurement report, thus improving the rate of successful handover.
