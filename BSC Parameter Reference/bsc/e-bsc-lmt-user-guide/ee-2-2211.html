Maintaining the IP Addresses of the Physical Ports

This describes how to maintain the IP addresses of the physical ports.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication within the BSC is normal.

Context

  • The physical port IP address maintaining function applies to the GFGUA, GOGUA, GFGUB, GOGUB, and GFGUG.
  • For the GFGUA, GFGUB, and GFGUG, the ports are numbered from 0 to 7. If the port type is GE, the port numbers can only be 0 and 4. For the GOGUA and GOGUB, the port numbers are 0 and 1.
  • Only one IP address can be added for each physical port.
  • The IP address of a physical port cannot be the same as an IP address that is already configured.
  • The IP addresses of the physical ports of different devices cannot be in the same network segment. There cannot be containing relations between the network segments of the IP addresses.

Procedure

  • Run the LST PHYPORTIP command on the LMT to query the IP address of the physical port.
  • Run the MOD PHYPORTIP command on the LMT to modify the IP address of the physical port.
  • Run the ADD PHYPORTIP command on the LMT to add the IP address of the physical port.
  • Run the RMV PHYPORTIP command on the LMT to remove the IP address of the physical port.
