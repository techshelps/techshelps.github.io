System Time

System time refers to the time of the whole BSC system.

System time is required for the operation of the whole BSC system and is provided by the GBAM/GOMU.

NOTE:

When the LMT application is running, do not modify the system time. Otherwise, severe errors may occur on the system. If you have to modify the system time, stop all the LMT application first.
