Using a Custom Template

You can save the counters of the same measurement object in a custom template format to facilitate the query.

Prerequisites

  • The Performance Browser Tool runs normally.
  • You have logged in to the FTP server of an NE.
  • The custom template is available. For details, refer to Creating a Custom Template.

Procedure

  1. In the navigation tree of the Browse Description File window, choose NE Measurement Information > Template Detail > Custom Template.
  2. Double-click the custom template to be queried under the Custom Template node. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Query Performance Results dialog box

  3. Set the parameters in the Query Mode and Date/Time areas.

    NOTE:
    For the description of the parameters, refer to Querying Performance Measurement Results.

  4. Click Query.
