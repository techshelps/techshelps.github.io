Exporting the Inventory Management File

This describes how to export the inventory management file to IP address of the server\BSC6000\data\mtndata\inventory\ivt. Exporting the inventory management file is also referred to as generating the inventory management file.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication within the BSC is normal.

Context

The BTS2X does not support inventory management.

  • The filename for the BTS is W_BSC6000%N.XML, where N indicates the BTS ID. The filename for the BSC is W_BSC6000.XML.
  • The generation of the inventory management file and the uploading of the file cannot proceed at the same time.
  • If btsinventory.xml and bscinventory.xml do not exist in \data\mtndata\inventory\templat\, generating the inventory management files fails.

Procedure

  1. Run the EXP DEVFILE command on the Local Maintenance Terminal to export the inventory management file.
