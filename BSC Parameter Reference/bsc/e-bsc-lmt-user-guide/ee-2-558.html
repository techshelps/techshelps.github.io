Managing Batteries

This describes how to obtain the running status of BTS batteries by performing a discharge test. In addition, this task describes how to query the installation time, power-off count, charge count, and discharge count.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.
  • The power module works in normal state.

Context

There are two types of discharge test.

  • Simple test: No battery capacity is calculated. The batteries are discharged within a period of time. If the battery voltage is lower than Final Voltage of Discharging after the test, the batteries are normal. Otherwise, the batteries are faulty.
  • Manual test: The battery capacity is calculated. The batteries are discharged until the voltage reaches Final Voltage of Discharging, that is, the batteries are fully discharged. If the battery voltage equals to the Final Voltage of Discharging after the test, the batteries are normal. Otherwise, the batteries are faulty.
CAUTION:

To protect the batteries, you need to set an interval between two discharge tests. The default interval is 14 days.

The duration of the simple test is defined before delivery. This value is usually 30 minutes or an hour. The value cannot be set on the LMT.

Procedure

  • Through GUI
    1. Test battery performance

      1. Choose BTS Maintenance > Maintain Board > Maintain Battery. A dialog box is displayed, as shown in Figure 1.
        Figure 1 Maintain Battery dialog box

      2. On the Test Battery tab page, set the parameters in the Location area, select a test in the Implement Test area, and click Start.
      3. After the test is complete, click Query. The results are displayed, as shown in Figure 1.
        NOTE:
        • The Latest 10 Test Result area displays the results of the latest 10 tests. You can select a serial number in the Serial No. drop-down list box to check the result.
        • If you set the serial number to All, the results of all tests are displayed if the number of tests is less than 10.
      4. Right-click in the result area and choose Save As from the shortcut menu to save the results.

    2. Query battery parameters

      1. In Figure 1, click the Query Battery Parameters tab. A tab page is displayed, as shown in Figure 2.
        Figure 2 Query Battery Parameters tab page

      2. Set parameters in the Location area, and then click Query. The query result is displayed, as shown in Figure 3.
        Figure 3 Querying battery parameters

  • Through MML
    1. You can run the following commands on the LMT to manage batteries.

      • Run the STR BTSBATTTST command to start testing batteries.
      • Run the STP BTSBATTTST command to stop testing batteries.
      • Run the DSP BTSBATTTST command to query the output of battery test.
