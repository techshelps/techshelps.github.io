Using the Performance Browser Tool

This describes how to start, configure and use the performance browser tool.

  • Starting the Performance Browser Tool
    This describes how to start the Performance Browser Tool. Through the Performance Browser Tool, you can manually import the performance description files of NEs, and export the performance measurement files.
  • Managing NE
    This describes how to configure and select the NE.
  • Managing Template
    You can save the counters of the same measurement object in a custom template format to facilitate the query. Managing template consists of adding, deleting, and modifying the custom template.
  • Managing Performance Measurement Results
    This describes how to download, convert, query, filter, and export performance measurement results.
  • Browsing Description Files
    This describes how to browse the contents of the description files.
  • Exiting the Performance Browser Tool
    This describes how to exit the performance browser tool.
