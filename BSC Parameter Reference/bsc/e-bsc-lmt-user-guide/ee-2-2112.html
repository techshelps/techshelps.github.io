Managing BSC Logs

This describes how to manage the operation logs, log parameters, and log files, and how to maintain the CS CHRs .

  • Setting BSC Log Parameters
    This describes how to set the parameters of BSC logs that are saved on the server. The BSC log parameters consist of the size and the time span of the BSC log file.
  • Uploading BSC Log Files
    This describes how to query the log file information on the server and how to upload the required log file to the LMT.
  • Querying the Log Files
    This describes how to query operation logs of local users, domain users, and M2000 users in online mode. A maximum of 500 operation logs can be queried.
  • Maintaining CS CHR
    This describes how to maintain the CHRs in the circuit switched domain (CS), including setting Call End Reason Mode and Call Resource Custom Mode.
