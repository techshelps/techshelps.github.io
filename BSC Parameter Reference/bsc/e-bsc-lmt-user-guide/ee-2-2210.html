Maintaining the IP Addresses of the Devices

This describes how to maintain the IP addresses of the devices.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication within the BSC is normal.

Context

  • The GFGUG interface board can be configured with a maximum of five device IP addresses. The other IP-based interface boards can be configured with only one device IP address each.
  • A device IP address cannot be the same as an IP address of another type.
  • A device IP address cannot be in the same subnet as a port IP address.
  • A deleted IP addresses cannot be referenced by the M3UALNK, NSVLLOCAL, or BTS.
  •  

Procedure

  • Run the LST DEVIP command on the LMT to query the device IP addresses.
  • Run the ADD DEVIP command on the LMT to add the device IP addresses.
  • Run the MOD DEVIP command on the LMT to modify the device IP addresses.
  • Run the RMV DEVIP command on the LMT to delete the device IP addresses.
