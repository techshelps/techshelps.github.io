Managing the Alarm Box

This describes how to manage the alarm box by maintaining the alarm box information and setting the alarm severity of the alarm box.

  • Querying Alarm Box Information
    This describes how to query the version information of the alarm box and the active alarms reported by the alarm box.
  • Querying the Numbers of Alarms with Different Severity
    This describes how to query the numbers of alarms with different severity.
  • Controlling the Alarm Box
    This describes how to reset the alarm box, stop the alarm sound, and turn off the alarm indicator.
  • Setting the Shield Severity of the Alarm Box
    This describes how to query and set the shield severity of the alarm box. Only the alarms that are set to be Report and whose shield severity is equal to or greater than the Current Alarm Box Shield Severity will be reported to the alarm box.
