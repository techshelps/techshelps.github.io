Maintaining the BTS Ring Network

This describes how to maintain the ring network parameters. The user can manually rotate the forward link and reverse link.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

The BTS3X, except the BTS 3001C and BTS 3001C+, can be configured in ring topology mode. The BTS ring topology is a special chain topology. For details, refer to BTS Topology.

CAUTION:

Manually rotating the ring topology will lead to service disruption and call drops.

When satellite transmission is enabled, rotating the Huawei â…¡ ring topology will lead to call drops.

Procedure

  • Through GUI
    1. Choose BTS Maintenance > Maintain Site > Maintain Ring Network. A window is displayed, as shown in Figure 1.

      Figure 1 Maintaining ring network

    2. In the Setting area, select the BTSs that need to be queried. Click Query. A window is displayed, as shown in Figure 2.

      Figure 2 Maintaining ring network

      NOTE:

      Click Switch to rotate the forward link and reverse link. You can also set the switchover period to automatically rotate the forward link and reverse link. Before rotating the forward link and reverse link, remove the E1 cable that carries the operation and maintenance services of the BTS.

  • Through MML
    1. You can run the following commands on the LMT to maintain the ring network.

      • Run the LST BTSRINGATTR command to query the BTS ring network attributes.
      • Run the MOD BTSRINGATTR command to modify the BTS ring network attributes.
