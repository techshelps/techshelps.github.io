Modifying Site Parameters in Batches

This describes how to modify site parameters in batches.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

  • The Active State attribute of a maximum of 50 BTSs can be modified in batches.
  • When you modify the Active State attribute of cascaded BTSs, you need to modify the Active State of the highest-level BTS only.
  • If you modify a large number of site parameters in batches, you are advised to run MML commands on the Local Maintenance Terminal.

Procedure

  • Through GUI
    1. Choose Configuration > Configure Site Parameters in Batches. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Configure Site Parameters in Batches dialog box

    2. Select the site to be modified in the Site List and add it to the Selected Sites.

      NOTE:
      • If the Site Type is set to All sites, only the attributes in Active State can be modified in the next step.
      • If the he Site Type is set to other types, the corresponding site attributes can be modified in the next step.

      This topic takes the situation that the Site Type is set to BTS3012 as an example.

    3. Click Next. A dialog box is displayed, as shown in Figure 2.

      Figure 2 Selecting parameters for modification

    4. Select parameters in the All parameters area and add them to the Selected Parameters area,
    5. Click Modify Parameters. A dialog box is displayed, as shown in Figure 3.

      Figure 3 Modifying parameters of sites

    6. Select Batch Value of the parameters to be modified in batches. Change Batch Value, and then click Batch Modify.
    7. Click OK to return to Figure 3.
    8. Click Finish.
  • Through MML
    1. You can run the following commands on the Local Maintenance Terminal to configure BTS parameters.

      • Run the MOD BTSCLK command to modify the BTS clock mode.
      • Run the MOD BTSALM command to modify the BTS alarm attributes.
      • Run the SET BTSEXD command to modify the BTS extension attributes.
      • Run the MOD BTSTHEFTALM command to modify the BTS burglar alarm attributes.
      • Run the MOD BTSOUTPUT command to modify the BTS output attributes.
