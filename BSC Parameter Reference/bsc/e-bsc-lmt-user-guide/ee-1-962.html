Local Maintenance Terminal

The Local Maintenance Terminal is a component of the LMT software. It provides Man Machine Language commands for the configuration and maintenance of the system. Moreover, the Local Maintenance Terminal provides detailed online help information.

Interface

The window of the Local Maintenance Terminal consists of the menu bar, toolbar, navigation tree, command window, output window, and status bar, as shown in Figure 1.

Figure 1 Local Maintenance Terminal BSC6000 window

Table 1 describes the components on the window.

Table 1 Local Maintenance Terminal window

Pane

Description

Menu Bar

Providing the menu operations of the system.

Toolbar

Providing the icon operations of the system.

Navigation Tree

Providing MML commands and description in tree structure.

Command Window
  • Providing detailed online help information on MML commands.
  • Displaying the MML commands.
  • Providing an area where users perform operations.

Output Window

Recording the detailed information about the current operation and the system. The output window consists of the Common and Maintenance tabs.

  • The Maintenance tab page displays the MML commands sent during the communication between the LMT and the server, and the results returned from the MML commands.
  • The Command tab page displays information other than the maintenance information.

Status bar

Displaying the name of the login user, connection status, and IP address of the virtual external network of the server.

Help

The Local Maintenance Terminal provides two types of online help:

  • Online help of the Local Maintenance Terminal
  • Online help of MML commands

Table 3 describes how to access the two types of help.

Table 3 Online help of the Local Maintenance Terminal

Name

Accessing Method

Online help of the Local Maintenance Terminal

The online help of the Local Maintenance Terminal consists of operation, configuration, and reference information. When using the Local Maintenance Terminal, you can press F1 or choose Help > Help Topics to show the help information.

Online help of MML commands

Enter a command with related parameters on the Local Maintenance Terminal, and then press Enter. The help information about the command is displayed in the Help Information window.
