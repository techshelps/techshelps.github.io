Modifying a Custom Template

This describes how to modify a custom template.

Prerequisites

  • The Performance Browser Tool runs normally.
  • You have logged in to the FTP server of an NE.
  • The custom template is available. For details, refer to Creating a Custom Template.

Procedure

  1. In the navigation tree of the Browse Description File window, choose NE Measurement Information > Template Detail > Custom Template.
  2. Right-click a custom template, and then choose Modify.
  3. The Modify dialog box is displayed. Select the measurement counters of the same measurement object, and then click Save Template.
  4. The Confirm dialog box is displayed. Click OK.
