Tracing the Messages on the Gb Interface

This describes how to trace the PTP messages and SIG messages on the Gb interface. Checking the information about the interaction between the PCU and each type of SGSN signaling, which can help identifying the PS problem.

  • Tracing SIG Messages on the Gb Interface
    This describes how to trace SIG signaling messages on the Gb interface based on NSEI on the LMT when the PCU is installed inside the BSC.
  • Tracing PTP Messages on the Gb Interface
    This describes how to trace the PTP signaling messages on the Gb interface based on cell or NSEI+BVCI on the LMT when the PCU is installed inside the BSC.
