Maintaining the Server

This describes how to maintain the core processes and the board and data files in the server.

  • Querying and Resetting the OMU Process
    This describes how to monitor the server status, reset the faulty server process, and check the history records of server maintenance.
  • Querying the Space in the Disk of the Server
    This describes how to query the partitions and status of a server hard disk in the BSC system.
  • Viewing the GOMU Information
    This describes how to view the information about the hard disk, RAID, memory, Ethernet adapter, DAEMON status, and DAEMON monitoring process of the GOMU.
  • Swapping the GOMUs
    You can swap the GOMUs so that the original standby GOMU is switched to act as the active one, and the original active POMU is switched to act as the standby one.
  • Backing Up the GOMU Data Files
    This describes how to manually back up the files in the active GOMU to the standby GOMU.
  • Resetting the Server
    You can request the system to reset the server by resetting the GOMU.
