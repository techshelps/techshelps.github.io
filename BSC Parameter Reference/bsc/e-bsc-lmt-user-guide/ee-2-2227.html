Maintaining the E1/T1 Links of the A Interface

This describes how to add, modify, or delete an E1/T1 link of the A interface.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication within the BSC is normal.
  • The GEIUA or the GOIUA is configured.

Context

The A interface can be configured with a maximum of 512 E1/T1 timeslots.

When two E1/T1 timeslots of the A interface have the same source signaling point and DPC group ID, their CICs must be different.

Procedure

  • Run the ADD AE1T1 command on the LMT to add an E1/T1 link of the A interface.
  • Run the RMV AE1T1 command on the LMT to delete an E1/T1 link of the A interface.
  • Run the MOD AE1T1 command on the LMT to modify an E1/T1 link of the A interface.
  • Run the LST AE1T1 command on the LMT to query an E1/T1 link of the A interface.
  • Run the MOD AE1T1CIC command on the LMT to modify the E1/T1 timeslots and CICs of the A interface.
