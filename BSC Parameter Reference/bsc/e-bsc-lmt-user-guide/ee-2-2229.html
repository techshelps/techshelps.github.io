Maintaining the Pb Interface

This describes how to maintain the E1/T1 links, signaling links, and circuits of the Pb interface.

  • Maintaining the E1/T1 Links of the Pb Interface
    This describes how to add, modify, or delete an E1/T1 link of the Pb interface.
  • Maintaining the Circuits on the Pb Interface
    When the external PCU is used, this describes how to query the circuit state on the Pb interface, and how to block, unblock, and reset the Pb interface circuits based on the query results.
  • Maintaining the Signaling Links of the Pb Interface
    This describes how to add, modify, or delete a signaling link of the Pb interface.
