BSC Work Areas

This describes the work area of the BSC. The work area is used to store all the versions of system software. The work area consists of running area, active area, and standby area.

Running Area

The area on the server that is used to store all the software loaded to a board. Usually, it refers to the flash of a board.

Active Area

The area on the server that is used for the server to store the software to be activated after online upgrade of the system. The patches for the BSC software are also stored in the active area.
NOTE:

The area that stores the patches are further divided into the active patch area and the standby patch area. Both of them belong to the active area of the server. The active patch area is used to store the patches to be loaded. The standby patch area is used to store the rolled back patches.

Standby Area

The area on the server that is used to store the software that supports version rollback.
