Maintaining MTP3 Links

This describes how to query the link state, route state, and DSP state of MTP3 links, and how to activate, deactivate, disable, and enable MTP3 links on the basis of the query result.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The GXPUM board in the subrack that controls the MTP3 links is functional.
  • The MTP3 links are configured.

Context

  • You can query the MTP3 link state in two ways: Query By Link No. or Query By Signaling Point.

    • Query By Link No.

      Each MTP3 link has a unique link number. One interface board corresponds to multiple links. You can specify a board and a link number to query the state of an MTP3 link.

    • Query By Signaling Point

      One DSP corresponds to multiple MTP3 links. You can query the state of multiple MTP3 links by querying the DSP.

    Table 1 describes the parameters on the MTP3 Link State tab page.

    Table 1 Parameters on the MTP3 Link State tab page

    Parameter

    Description

    Link No.

    Unique number of an MTP3 link.

    Using State

    Whether the current link is transmitting signaling or not.

    Fault State

    Whether the MTP2 link is faulty.

    Blocked

    Whether the MTP3 link can carry signaling services.

    Activated

    Whether the MTP3 link set up from the MTP2 is activated.

    Local Inhibited

    Whether the MTP3 link is disabled locally.

    Remote Inhibited

    Whether the MTP3 link is disabled remotely.

    Congestion

    Whether the MTP3 link is congested because of overload.

    Changing Over

    Whether the current MTP3 link is faulty and the services on this link have to be switched over to another link.

    Changing Back

    Whether the current MTP3 link is restored and the services are switched back.

    SLS Code

    Signaling Link Selection code.

  • Querying the MTP3 route state: The MTP3 route is a logic set of multiple links.

    Table 2 lists the parameters on the MTP3 Route State tab page.

    Table 2 Parameters on the MTP3 Route State tab page

    Parameter

    Description

    OPC

    Originating signaling Point Code

    DPC

    Destination signaling Point Code

    Link Set Index

    Unique index of a link set

    Fault State

    Whether the MTP3 route is available.

    Inhibit

    Whether the MTP3 links forming the MTP3 route is inhibited.

    Service Transmit

    Whether the MTP3 route is transmitting signaling.

    Rearrange

    Whether the signaling points of the MTP3 route have changed.

    SLS Code

    Signaling Link Selection code

  • Querying the MTP3 state between SPs: You can query the MTP3 OSP state and MTP3 DSP state.

    Table 3 lists the parameters on the MTP3 State Between SP tab page.

    Table 3 Parameters on the MTP3 State Between SP tab page

    Parameter

    Description

    DPC Code

    Destination signaling Point Code

    OPC

    Originating signaling Point Code

    Network Indicator

    The Network Indicator (NI) is of four types: International Network (INT), International Standby Network (INTB), National Network (NAT), and National Standby Network (NATB).

    IS STP

    Whether the STP is the destination node.

    Congestion Flag

    Whether the link to this signaling point is congested.

    Access Flag

    Accessible to the DSP or not

Operating an MTP3 link: You can activate, deactivate, disable, and enable the MTP3 links.

  • Activate: This operation is performed to set up a new MTP3 link.
  • Deactivate: This operation is performed to remove the MTP3 links and the MTP2 links.
  • Inhibit: This operation is performed to disable the MTP3 links, but the MTP2 links are still available.
  • Uninhibit: This operation is performed to enable the MTP3 links.

Procedure

  • Through GUI
    1. Querying the state of the MTP3 links.

      1. On the LMT, choose BSC Maintenance > Maintain Transmission and Signaling > Maintain MTP3 Link to query the state of the MTP3 links.
      2. Set parameters in the Location and Operation area and then click Query. The query results are displayed, as shown in Figure 1.
        Figure 1 Results of querying the MTP3 link state

      3. Right-click in the result area and choose Save As from the shortcut menu to save the results.

    2. Querying the state of an MTP3 route.

      1. In the dialog box shown in Figure 1, select the MTP3 Route State tab page to query the state of the MTP3 route.
      2. Set parameters in the Location area and then click Query. The query results are displayed, as shown in Figure 2.
        Figure 2 Results of querying the MTP3 route state

      3. Right-click in the result area and choose Save As from the shortcut menu to save the results.

    3. Querying the MTP3 state between SP.

      1. In the dialog box shown in Figure 1, select the MTP3 State Between SP tab page to query the MTP3 state between SP.
      2. Select DPC and then click Query. The query results are displayed, as shown in Figure 3.
        Figure 3 Results of querying the MTP3 state between SP

      3. Right-click in the result area and choose Save As from the shortcut menu to save the results.

    4. Operating the MTP3 link.

      1. In the dialog box shown in Figure 1, select the MTP3 Operation tab page to perform operations on the MTP3 link.
      2. Set parameters in the Location And Operation area, click Apply. The results are displayed, as shown in Figure 4.
        Figure 4 Results of operating the MTP3 link

  • Through MML
    1. You can run the following commands on the LMT to maintain the MTP3 links.

      • Run the DSP MTP3LNK command to query the state of MTP3 links.
      • Run the DSP MTP3RT command to query the state of MTP3 route.
      • Run the DSP MTP3STATUSBTSP command to query the state of specified DPCs.
      • Run the DSP SCCPOSPDSPPAIR command to query the state of all DPCs.
      • Run the ACT MTP3LNK command to activate MTP3 links.
      • Run the DEA MTP3LNK command to deactivate MTP3 links.
      • Run the UIN MTP3LNK command to enable MTP3 links.
      • Run the INH MTP3LNK command to disable MTP3 links.
