Saving the Input Commands

This describes how to automatically save the successful and failed MML command.

Prerequisites

  • The Local Maintenance Terminal runs normally.
  • The communication between the LMT and the BSC is normal.

Procedure

  1. Choose System > Save Input Command. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Save Input Command dialog box

  2. Set the parameter in the Save As area, and then click OK. The input command is saved.
