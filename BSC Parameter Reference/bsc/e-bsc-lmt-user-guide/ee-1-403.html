LMT-Related Definitions

This describes the differences among the LMT, the LMT PC, and the LMT software.

LMT

The LMT is a terminal installed with the Huawei Local Maintenance Terminal software package. The LMT is connected to the BSC6000 OM network.

LMT PC

The LMT PC refers to the computer installed with the Huawei Local Maintenance Terminal software package.

LMT Software

The LMT software refers to the Huawei Local Maintenance Terminal software package.
