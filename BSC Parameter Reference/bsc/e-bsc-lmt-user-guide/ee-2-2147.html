Confirming the Status of the Patch

This describes how to confirm the status of a patch that is activated and effective.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication within the BSC is normal.
  • The patch is in the Activated status.

Context

A patch cannot be deactivated again after its status is confirmed.

When a patch is successfully confirmed, the status of the patch on the server becomes Running, and the status of the corresponding patch in service boards also changes. To query the patch status in service boards, run DSP PATCH.

Procedure

  1. Run CON PATCH on the LMT to confirm the patch.
