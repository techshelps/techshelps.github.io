Exiting the Local Maintenance Terminal

This describes how to exit the Local Maintenance Terminal.

Prerequisite

The Local Maintenance Terminal runs normally.

Procedure

  1. You can exit the Local Maintenance Terminal through either of the following methods:

    • Choose System > Exit.
    • Press Alt+X.
    • Click .

  2. The Confirmation dialog box is displayed. Click OK to exit the Local Maintenance Terminal.
