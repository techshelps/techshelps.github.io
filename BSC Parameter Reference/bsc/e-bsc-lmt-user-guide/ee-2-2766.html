Querying the BTS IP Performance Measurements

This describes how to query the IP performance measurements of an IP BTS working in the FE/GE mode. If the IP BTS working in the FE/GE transmission mode is faulty, you can run the Query the Stat of the BTS IP performance command on the LMT to query the IP performance measurements of the faulty BTS, and thus identify the problem.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

You can query the following IP performance measurements on the LMT:

  • Transmit flow on the FE/GE interface
  • BER on the FE/GE interface
  • Service type and flow of a specific user

Procedure

  1. You can run the following commands on the LMT to query the BTS IP performance measurements.

    • Run the DSP FLOWSTAT command to query the transmit flow on the FE/GE interface.
    • Run the DSP BER command to query the BER on the FE/GE interface.
    • Run the DSP SERVICESTAT command to query the service type and the flow of a specific user.
