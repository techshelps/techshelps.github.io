Monitoring the BTS Environment

This describes how to monitor the site environment, for example, to reset the alarm box, clear burglar alarms, disable the EAC alarm, obtain temperature and humidity, and clear smoke alarms.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

Table 1 lists the description of the parameter configuration.

Table 1 Parameters in the Monitor Environment dialog box

Parameter

Meaning

Reset Alarm box

Resets the environment monitoring device when it fails.

Clear Robbery Alarm

Clear burglar alarms when the burglar alarms or false alarms are removed.

Disable the EAC Alarm

This operation shields the EAC alarm for ten minutes. After then, the EAC alarm reports. After the EAC alarm is reported, the alarm can be shielded again.

Get Temperature and Humidity

Gets the temperature, humidity, temperature upper threshold, temperature lower threshold, humidity upper threshold, and humidity lower threshold.

Clear Smoke Alarm

Indicates the option for clearing smoke alarm on the LMT manually. This option button is available for only the BTS3002E, BTS3012AE, BTS3012, BTS3012â…¡, BTS3006C, BTS3036, BTS3036A, DBS3036, BTS3900 GSM, BTS3900A GSM, and DBS3900 GSM.

Procedure

  • Through GUI
    1. Choose BTS Maintenance > Maintain Site > Monitor Environment.
    2. Set Site and Operation Type, and then click Apply. The results are displayed, as shown in Figure 1.

      Figure 1 Monitoring environment

  • Through MML
    1. You can run the following commands on the LMT to monitor the BTS environment.

      • Run the CLR BTSROBALM command to clear burglar alarms.
      • Run the DIS BTSEACALM command to disable the EAC alarm.
      • Run the DSP BTSENVSTAT command to query the temperature and humidity of the BTS.
      • Run the DSP BTSENVTHD command to query the temperature upper threshold, temperature lower threshold, humidity upper threshold, and humidity lower threshold.
      • Run the STR BTSRACKTEMPMON command to start monitoring the BTS rack temperature.
      • Run the STP BTSRACKTEMPMON command to stop monitoring the BTS rack temperature.
      • Run the RST BTSEAC command to reset BTS EAC.
