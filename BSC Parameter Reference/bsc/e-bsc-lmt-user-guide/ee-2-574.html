Backing Up Data

This describes how to save configuration data on the server or LMT. You can restore the configuration data with the backup data.

Context

The backup function of the LMT backs up only the configuration data, for example, BSC data, site data, cell data, TRX data, and network optimization parameters. It cannot back up the log files, alarm information, and performance statistics.
  • Backing Up LMT Local Data
    This describes how to save the configuration data on the LMT to the local PC.
  • Backing Up Sever Data
    This describes how to save configuration data on the server.
  • Checking LMT Local Data
    This describes how to check the local data. A prompt is displayed when the data inconsistency is checked.
