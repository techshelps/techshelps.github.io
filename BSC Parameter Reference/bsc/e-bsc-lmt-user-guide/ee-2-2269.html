Maintaining the SIG BVC

Maintaining the SIG BVC involves querying and resetting the SIG BVC.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The SIG BVC is configured.

Context

The SIG BVC indicates the signaling point virtual connection between BSSGP layers.

Procedure

  • Run the DSP SIGBVC command on the LMT to query the SIG BVC status.
  • Run the RST SIGBVC command on the LMT to reset the SIG BVC.
