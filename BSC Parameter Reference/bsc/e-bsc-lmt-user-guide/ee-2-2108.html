Querying the Commands in a Command Group

This describes how to query all the commands in a specified command group.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Procedure

  1. Run the LST CCG command on the LMT to query all the commands in a specified command group.
