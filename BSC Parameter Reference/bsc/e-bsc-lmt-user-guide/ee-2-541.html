Configuring Frequency Scan

This describes how to configure the start time and duration for scanning the frequencies in a cell.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

You can set the start time and duration to control the frequency scan of the server.

  • If the server time is in the scan duration, the frequency scan starts and reports the scan results periodically.
  • When the server time is not in the scan duration, the frequency scan stops.
  • When the carrier board is the QTRU, the spacing between physical frequencies cannot exceed 10 MHz.

Procedure

  • Through GUI
    1. Choose BTS Maintenance > Maintain Cell > Configure Frequency Scan. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Configure Frequency Scan dialog box

      NOTE:

      The start time of frequency scan must be an future time. Otherwise, it is reported until this moment of the second day.

    2. Set Location, Option, and Frequency, and then click Start.
  • Through MML
    1. You can run the following commands on the LMT to query and configure the frequency scan.

      • Run the SET FREQSCAN command to configure the frequency scan.
      • Run the LST FREQSCAN command to query the frequency scan.
