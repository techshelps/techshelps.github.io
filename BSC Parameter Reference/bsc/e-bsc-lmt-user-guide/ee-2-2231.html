Maintaining the Signaling Links of the Pb Interface

This describes how to add, modify, or delete a signaling link of the Pb interface.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • At least a PCU is configured.

Context

  • The congestion start threshold must be larger than the congestion end threshold.
  • After modifying a signaling link of the Pb interface, ensure that the Pb interface is configured with an E1/T1 link.
  • If the signaling links of the Pb interface use the E1 mode, the timeslot ranges from 1 to 31.
  • If the signaling links of the Pb interface use the T1 mode, the timeslot ranges from 1 to 24.

Procedure

  • Run the LST PBSL command on the LMT to query a signaling link of the Pb interface.
  • Run the ADD PBSL command on the LMT to add a signaling link of the Pb interface.
  • Run the MOD PBSL command on the LMT to modify a signaling link of the Pb interface.
  • Run the RMV PBSL command on the LMT to delete a signaling link of the Pb interface.
