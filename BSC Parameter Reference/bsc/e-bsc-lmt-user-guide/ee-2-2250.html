Maintaining the BTS Versions

This describes how to manage version software and ensure the compatibility of hardware version and software version by downloading, configuring, loading, and activating the version software. The version software is associated with the BTS and the BTS boards.

  • Querying the BTS Software Version
    This describes how to query the software version of a BTS, software type, and software version of a board. When you maintain the BTS through the LMT, you can get the software version of the BTS by querying that of BTS boards.
  • Querying the Software Version of the BTS Board
    This describes how to query the type, software version, and clock status of a board. You can query the board running software versions of more than one BTS in one operation.
  • Downloading BTS Software
    This describes how to download the BTS software to the server from the LMT through the FTP server.
  • Configuring BTS Software
    This describes how to set the file type, file name, version number, and BTS type for the downloaded software on the server before BTS software loading.
  • Loading BTS Software
    This describes how to load the BTS software that are configured with file information to the BTS main control board on the LMT. This also describes how to verify whether the file configuration is consistent with the version of BTS board software.
  • Activating BTS Software
    This describes how to activate the version software loaded to the BTS boards.
  • Deleting the BTS Software
    If you delete the BTS software based on the version, the folder related to the BTS version and the files in the folder will be deleted.
