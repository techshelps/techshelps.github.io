Maintaining a Cell

This describes how to send system messages of a cell, and query and configure the frequency scanning.

  • Sending Cell System Message
    This describes how to send the system messages of a cell. You can update the system messages of a cell through a cell. The BTS then broadcasts these messages to the MSs through the radio interface.
  • Querying the PDCH
    This describes how to query the number of activated PDCHs and the number of occupied PDCHs.
  • Querying Frequency Scan Results
    This describes how to query the maximum and average values of the main level and diversity level in a frequency scan period.
  • Querying the TRX Power Mode
    This describes how to query the current power mode and the maximum power mode of TRXs.
  • Querying Automatic Power Correction Type
    This describes how to query the automatic power correction type of the TRX board through the LMT.
  • Testing Channel Loopback
    This describes how to test the transmission ability of the service channel and speech channel on the LMT. The test includes the BIU Loop Test and TRX RF Self Loop Test of the service channel and the BTS Sound Loop Test and TRX Sound Loop Test of the speech channel.
  • Testing Channel Performance
    This describes how to test the parameters of TRX module, such as channel BER and transmission power to test the transmission quality of the channel.
  • Testing TRX Loopback
    This describes how to test the TRX loopback to learn the TRX performance and channel transmission. The TRX loopback test consists of BIU loop test and TRX self loop test.
  • Testing TRX Performance
    This describes how to test the performance of all frequencies or the selected frequencies. The test items include the interference level and power level.
  • Testing Idle Timeslot
    This describes how to test the idle timeslots in the network.
  • Testing CODEC Mode
    This describes how to test the audio coder and decoder (CODEC) mode and reports the test results.
  • Maintaining the BTS Port Timeslots
    This describes how to maintain the BTS port timeslots. Maintaining the BTS port timeslots involves disabling/enabling the BTS port timeslots or exchanging the BTS port timeslots. When a port timeslot is disabled, it cannot be caused by the OML, RSL, or TCH. The test personnel can disable certain port timeslots and then check whether the enabled port timeslot is normal.
  • Configuring Frequency Scan
    This describes how to configure the start time and duration for scanning the frequencies in a cell.
  • Performing Loopback on the FR Port
    This describes how to maintain the FR port. Maintaining an FR port loopback involves initiating and stopping an FR port loopback.
  • Controlling TRX Power
    This describes how to set the type of the power supply for the TRX board and how to set the status of the switches of the power supply.
  • Resetting a TRX
    This describes how to reset the faulty TRX of a specified BTS.
