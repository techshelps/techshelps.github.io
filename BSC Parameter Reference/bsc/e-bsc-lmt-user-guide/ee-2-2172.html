Collecting the Information about the Carrier Configuration

This describes how to collect the information about the carrier configuration by subrack, interface board, cell, or site. The collected information contains the details of different types of carriers.

  • Collecting Statistics on DTRU Configuration By BM
    This describes how to measure the number of DTRUs using one TRU, maximum number of TRXs that can be added, and number of TRUs that can be added through license application by a BM subrack.
  • Collecting Statistics on DTRU Configuration by Interface Board
    This describes how to collect statistics on DTRU configuration, port status, and site reuse ratio by an interface board.
  • Collecting Statistics on DTRU Configuration by BTS
    This describes how to measure the number of TRXs, number of DTRUs using one TRU, number of main chain TRXs, and main chain port number by a BTS.
  • Collecting Statistics on DTRU Configuration by Cell
    This describes how to measure the number of TRXs, number of DTRUs, number of DTRUs in double-transceiver working mode, and type of BTS cabinet.
