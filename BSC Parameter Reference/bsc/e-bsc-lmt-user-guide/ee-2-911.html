Collecting Statistics on DTRU Configuration by Interface Board

This describes how to collect statistics on DTRU configuration, port status, and site reuse ratio by an interface board.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The server is functional.

Context

The DTRU configuration files exported by interface boards are in *.xls format. The default saving path is: LMT installation directory\BSC6000\version number\BackupCfgData.

Procedure

  1. Choose Configuration > DTRU Configuration Statistics > DTRU Configuration Statistics By Interface Board. A dialog box is displayed, as shown in Figure 1.

    Figure 1 DTRU Configuration Statistics by Interface Board dialog box

  2. Right-click the table list area, and then choose Export. The Save As dialog box is displayed.
  3. Type the file name, and then click Save.
