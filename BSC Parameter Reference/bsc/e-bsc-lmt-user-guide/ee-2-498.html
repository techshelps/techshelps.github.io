Activating the BSC License

This describes how to activate the BSC license to validate the license files downloaded by users.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The license to be activated is already downloaded.

Context

The license to be activated must match with the signaling point (SP) of the subrack not with the version.

The server reads and then sends the license file information to the service boards. The board then updates the relevant data in the flash memory according to the new license files.

Procedure

  1. Run the ACT LICENSE command on the Local Maintenance Terminal to activate BSC License.
