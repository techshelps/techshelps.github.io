Resetting Smoke Alarms

This describes how to clear smoke alarms for a power monitoring module manually.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.
  • The communication between the BSC and the BTS is normal.

Context

This operation is applicable to the BTS3002E, BTS3012, BTS3012â…¡, BTS3012AE, BTS3006C, BTS3036, BTS3036A, DBS3036, DBS3900 GSM, BTS3900 GSM, and BTS3900A GSM.

Procedure

  • Through GUI
    1. Choose BTS Maintenance > Maintain Board > Reset Smoke Alarm. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Reset Smoke Alarm dialog box

    2. Set Site Name ,Subrack No.and Slot No., and then click Reset.

      NOTE:

      Resetting smoke alarms can remove other than clear the alarms displayed on the BSC6000 Local Maintenance Terminal. You must handle the alarms manually on site.

      If you reset other alarms, you must manually clear the alarms. For details, refer to Querying the Active Alarms.

  • Through MML
    1. Run the RST BTSSMOKEALM command on the Local Maintenance Terminal to reset smoke alarms.
