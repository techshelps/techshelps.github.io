Core Process and Local Process

This describes the core process and the local process.

The BSC processes are classified into the core process and the local process:

  • Core process

    The core process is the management center of the whole BSC system. When the core process is faulty, the communication between the LMT and the server is interrupted. The status of the server core process displayed on the LMT is always Normal unless the communication between the LMT and the server is interrupted.

  • Local process

    The local process is the management process of the server and the OM management center of the BTS. If the local process is faulty, you can infer that the server is faulty or abnormal state alarms are reported.
