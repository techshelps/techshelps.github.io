Managing the License

This describes how to download, activate, and query the BSC license. This also describes how to query the BSC license file, equipment serial number, and equipment usage.

  • Downloading the BSC License
    This describes how to download the License file to the active area of on the server from the LMT.
  • Activating the BSC License
    This describes how to activate the BSC license to validate the license files downloaded by users.
  • Querying the BSC License
    This describes how to query the configuration and application information about the effective license files.
  • Querying BSC License File Information
    This describes how to query and verify the attribute information about the license file saved on the local PC.
  • Querying BSC ESNs
    This describes how to query the BSC equipment serial number (ESN). The expansion of the BSC system requires new BSC ESNs that are provided by Huawei.
  • Querying the Usage of the BSC License
    This describes how to query the actual usage, including the allowed traffic and occupied service resource, of the effective license.
