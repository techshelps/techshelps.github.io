Managing BSC Access Authorities

BSC access authorities are classified into the guest level, common user level, operator level, administrator level, and self-defined level. The first four levels of authorities cannot be changed. The self-defined level is defined by the administrator level.

  • Concepts of BSC Security Management
    This describes the principles of the LMT software authority management from the following aspects: LMT user password, LMT user authority, and LMT operation time limit.
  • Managing the Information on the Operator
    This describes how to manage the account, password, description, operation authority, and operation time limit of the operator. Before operations, ensure that the operator has a valid account.
  • Managing the Workstation
    This describes how to manage the BSC information and how to check that the LMT and MML Server are connected.
  • Managing the Command Group
    This describes how to manage the commands in a command group. As the minimum authority unit, the command groups are allocated to the accounts of different levels.
