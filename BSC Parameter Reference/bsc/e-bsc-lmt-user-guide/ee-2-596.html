Managing Alarm Log Files

This describes how to manage alarm log files, such as creating alarm log files and querying alarm log files stored on the server, and loading the selected alarm log files to the LMT.

Prerequisites

  • The LMT runs normally.
  • The communication between the LMT and the BSC is normal.

Context

  • If the names of alarm log files to be queried are known, you can query the alarm log files based on the file names. The query results contain only the alarm log files that meet the query conditions.
  • The default path for loading alarm log files is LMT installation directory\BscObj\office name\Alm.

Procedure

  • Through GUI
    1. Creating alarm log files

      1. Choose Alarm Maintenance > Manage Alarm Log File. A dialog box is displayed, as shown in Figure 1.
        Figure 1 Managing Alarm Log Files

      2. Click Create. A dialog box is displayed, as shown in Figure 2.
        Figure 2 Create Alarm Log File dialog box

      3. In the Create Options area, set Start Time, End Time, and File Name and Return Count of the file to be created.
      4. Click OK. A dialog box is displayed; prompting that creating an alarm log file succeeds.

    2. Querying and Uploading Alarm Log Files

      1. In Figure 1, click Query. All the alarm log files that are created and saved in the server are displayed, as shown in Figure 3.
        Figure 3 Querying alarm log files

        NOTE:

        Enter the File Name of the alarm log file to be queried in the Query Setting area, and then click Query. The queried alarm log files are displayed. The fuzzy search is not supported.

      2. Select the alarm log files to be uploaded to the LMT, and then click Upload, the alarm log file is loaded.
        NOTE:

        If you select Zipped in the Upload Setting area, the selected alarm log files will be uploaded to the local PC as compressed files.

  • Through MML
    1. You can run the following commands on the Local Maintenance Terminal to manage alarm log files.

      • Run the LST ALMLOGFILE command to query the alarm log file list.
      • Run the ULD ALMLOG command to load alarm log files.
