Check whether the data configurations of the SS7 signaling points of the BSC are the same as those of the MSC

1) Log in to the LMT and then choose Alarm Maintenance > Browse Alarm. Then, view the Local Signaling Point Code and SCCP Dest Signaling Point Code in the alarm location information. Here, the Local Signaling Point Code is the code of the originating signaling point and the SCCP Dest Signaling Point Code is the code of the destination signaling point.

2) On the BSC Attributes tab page, click the A Signaling Point button in the Other Data area.

3) In the displayed dialog box, check whether the data configurations of the BSC (the originating signaling point) are the same as the data configurations of the MSC (the destination signaling point). The parameters involved in the data configurations are Encoding Scheme and Network Indicator.

Y => The data configurations of the SS7 signaling points of the BSC are the same as those of the MSC.

N => The data configurations of the SS7 signaling points of the BSC are different from those of the MSC.
