Check whether local management rights are obtained

1) Log in to the LMT and then choose BTS Maintenance > Query BTS Running Status.

2) In the displayed dialog box, set Object Type to Site and select the Site Type. The sites that meet the condition are displayed in the Candidate sites area.

3) Move the sites to be queried from the Candidate sites area to the Selected sites area. In the Query Status area, set the Local Management Rights. Then, click OK.

4) In the displayed dialog box, check whether the Local Management Rights is set to Yes.

Y => The Local Management Rights is set to Yes. The local management rights are obtained.

N => The Local Management Rights is set to No. The local management rights are not obtained.
