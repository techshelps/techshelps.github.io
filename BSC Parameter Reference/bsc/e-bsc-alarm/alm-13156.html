ALM-13156 EMU Boolean 23 alarm

Description

The alarm is reported when the level of the Boolean value equals to the alarm threshold.

Attribute

Alarm ID Alarm Severity Alarm Type
13156 Warning Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

162:GMU
3 Board No.

Number of the fault board

0:GMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255:None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

30:BTS3900E;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

The alarm configured by the user is generated and the corresponding impact occurs.

System Actions

None.

Possible Causes

  • There is a configuration alarm.
  • The sensor module is damaged.
  • The DEMU module is damaged.

Procedure

  1. Check if there is a configuration alarm.

    Y=>Disable the Boolean alarm according to user's configuration.

    N=>misinformation,Go to Step2.

  2. Check if the Boolean alarm needs to be configured.
    1. Check if the Boolean alarm needs to be configured.

      Y=>Go to Step 3.

      N=>Go to Step b.

    2. Disable the Boolean alarm and see if the alarm is cleared.

      Y=>Alarm handling is completed.

      N=>The alarm is not cleared. Please contact Huawei Customer Service Center.

  3. Check if the output contact of the sensor is changed.
    1. Check if the output contact of the sensor is changed.

      Y=>Go to Step b.

      N=>Go to Step 4.

    2. Handle the Boolean value alarm according to the actual configuration and see if the alarm is cleared.

      Y=>Alarm handling is completed.

      N=>The alarm is not cleared. Go to Step 4.

  4. Check if the sensor and its connecting cables are faulty.
    1. Check on site if the sensor and its connecting cables are faulty.

      Y=>Go to Step b.

      N=>Go to Step 5.

    2. Replace the faulty sensor or re-connect its connecting cables and see if the alarm is cleared.

      Y=>Alarm handling is completed.

      N=>The alarm is not cleared. Go to Step 5.

  5. Replace the damaged Module, and check whether the alarm is cleared.

    Y=> The alarm is cleared. End the alarm handling.

    N=> The alarm is not cleared. Contact the Huawei Customer Service Center.
