Identify the board that reports the alarm

1) Log in to the LMT and view the Subrack No. and Slot No. in the alarm location information.

2) Select the BSC on the Management Tree tab page.

3) On the BSC Device Panel tab page, view the board panel that corresponds to the subrack number and slot number.

a. If the board panel is displayed as GEIUB/GEHUB/GEXUB/GOIUB/GFUGB/GOGUB, the board is an Abis interface board. The peer NE is a BTS.

b. If the board panel is displayed as GEIUA/GFGUA/GOGUA, the board is an A interface board. The peer NE is an MSC.

c. If the board panel is displayed as GEIUP/GOIUP, the board is a Pb interface board. The peer NE is a PCU.

d. If the board panel is displayed as GEPGU/GFGUG, the board is a Gb interface board. The peer NE is an SGSN.

e. If the board panel is displayed as GEIUT/GOIUT, the board is an Ater interface board.
