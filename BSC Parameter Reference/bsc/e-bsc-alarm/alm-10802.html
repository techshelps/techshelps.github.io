ALM-10802 Standby Extended Boolean 2 Fault Alarm

Description

This alarm is reported when the standby extended Boolean port is faulty, the alarm level configuration is incorrect, or the TCU is faulty.

Attribute

Alarm ID Alarm Severity Alarm Type
10802 Major Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

95:DTCU
3 Board No.

Number of the fault board

0-3:USCU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255:None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;

20:DBS3036;

21:BTS3036;

22:BTS3036A;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

Subject to the specific external alarms defined by the user.

System Actions

None.

Possible Causes

  • The sensor on the standby extended Boolean port is faulty.
  • The TCU is faulty.

Procedure

  1. Check whether the standby extended Boolean 1 is faulty.
    1. Check whether the sensor for the standby extended Boolean 1 is faulty.

      Y => The sensor is faulty. Replace the standby extended Boolean 1. Go to procedure 2.

      N => The sensor is normal. Go to procedure 3.

  2. Check whether the alarm is cleared.

    Y => The alarm is cleared. No further action is required.

    N => The alarm is not cleared. Go to procedure 3.

  3. Reset the TCU.
    1. Reset the TCU on the LMT.
    2. Check whether the alarm is cleared.

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to procedure 4.

  4. Replace the TCU.
    1. Replace the TCU manually.
    2. Check whether the alarm is cleared.

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Contact Huawei.
