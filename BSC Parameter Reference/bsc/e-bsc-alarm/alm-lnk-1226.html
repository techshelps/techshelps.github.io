Check whether the license matches with the configured signal point.

Log in the MML client, type the MML command DSP ESN. Set the Display Type to GEN (By Generate). Then, set the Net Indicator, Net Structure, OPC (signaling point of the GMPS in the BSC), and DPC (signaling point of the MSC). Get the configured ESN.

Type the LST LICENSE command, type the license file name.Obtain the ESN information of the license, Compare the obtained ESN with the configured ESN.
