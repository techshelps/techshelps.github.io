ALM-4810 Main Control Module Configuration Inconsistency Alarm

Description

The alarm is reported when the configuration of the BSC is inconsistent with that of the actual installed equipment.

Attribute

Alarm ID Alarm Severity Alarm Type
4810 Major Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

64:DTMU/DOMU

87:GTMU
3 Board No.

Number of the fault board

0-5:DTMU/GTMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm Sub-Category Number

Sub No. 0: The E1/T1 port configured on the BSC is incompatible with that on the DTMU.

Sub No. 1: The rack type configured on the BSC is inconsistent with the actual rack type.

Sub No. 2: The transmission mode configured on the LMT or SMT is inconsistent with the optical transmission mode configured on the DMCM module.
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

12:BTS3012;

13:BTS3006C;

14:BTS3002E;

15:BTS3012AE;

16:BTS3012 II;

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

  • Sub No. 0: The secondary link on the incompatible E1 port and the extension cabinet group cannot work normally when this alarm is generated.
  • Sub No. 1: The normal running status of the boards in the extension cabinet group may be affected.
  • Sub No. 2: The extension cabinet cannot communicate normally with the BSC.

System Actions

None.

Possible Causes

  • Sub No. 0: The main control board has different types. Some main control boards support four E1/T1 ports, whereas others support eight E1/T1 ports. If the main control board supports four E1/T1 ports, this alarm is reported when the board supporting other quantity of E1/T1 ports is configured on the BSC.
  • Sub No. 1: The DIP switch in the rack of the extension cabinet group is not correct, or the actual rack type in the extension cabinet group is inconsistent with that configured on the BSC.
  • Sub No. 2: The transmission mode configured on the LMT is inconsistent with the optical transmission property configured on the DMCM module.

Procedure

  1. Sub No. 0
    1. Change the secondary link port or internal cascading port.

      Change the secondary link port or internal cascading port, and check whether the alarm is cleared.

      Y => The alarm is cleared. The alarm handling is complete.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.

    2. Change the DTMU to 8E1DTMU.

      Change the DTMU to 8E1DTMU, and check whether the alarm is cleared.

      Y => The alarm is cleared. The alarm handling is complete.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.

  2. Sub No. 1
    1. Check whether the DIP switches of the rack in the extension cabinet group are correctly set.

      Y => Go to step c.

      N => Go to step b.

      The settings of the DIP switches are as follows (for the BTS3012, BTS3012AE, and BTS3012II):

      Check whether the DIP switches of the DCSU are set correctly.

      There are totally 14 DIP switches on the DCSU. The settings of the DIP switches are described as follows:

      The SW11 is used for terminal matching selection. The setting of the SW11 is as follows:

      In a single cabinet, SW11 is set to all ON.

      In combined cabinets, all SW11s of the cabinets are set to all ON.

      In the case of two combined groups, all SW11s of the cabinets are set to all ON.

      In the case of three combined groups, the SW11 of the main cabinet in the main cabinet group is set to all OFF, and the SW11s of all the other cabinets are set to all ON.

      The SW1 is used for determining the main cabinet in the main cabinet group. The setting of the SW1 is as follows:

      All ON indicates that the selected cabinet is the main cabinet in the main cabinet group.

      All OFF indicates other circumstances.

      The SW2–SW5 are used for configuring the main and extension cabinets and for terminal matching selection. The settings of the SW2–SW5 are as follows:

      SW2, SW3, SW4, and SW5

      All On indicates that the selected cabinet is the main cabinet.

      All OFF indicates that the selected cabinet is the extension cabinet.

      The SW8 is used for configuring the cabinet numbers. The setting of the SW8 is as follows:

      1 2 3 4

      ON ON ON ON Indicates that the selected cabinet is the main cabinet in the main cabinet group.

      OFF OFF ON ON Indicates that the selected cabinet is the main cabinet in the extension cabinet group 1.

      ON OFF OFF ON Indicates that the selected cabinet is the main cabinet in the extension cabinet group 2.

      The SW9 and SW10 are used for configuring the cabinet types. The settings of the SW9 and SW10 are as follows:

      SW9 Bit SW10 Bit

      1 2 3 4 1 2 3 4

      ON ON ON ON ON ON ON ON Indicates that the cabinet type is BTS3012.

      OFF ON ON ON ON ON ON ON Indicates that the cabinet type is BTS3012AE.

      ON OFF ON ON ON ON ON ON Indicates that the cabinet type is BTS3012II.

    2. Reconfigure the DIP switches of the rack.

      Reconfigure the DIP switches of the rack on the basis of the actual rack type, and check whether the alarm is cleared.

      Y => The alarm is cleared. The alarm handling is complete.

      N => The alarm is not cleared. Go to step c.

    3. Check the rack type configured on the BSC.

      Check whether the rack type configured on the BSC is consistent with the actual rack type.

      Y => Consistent. Contact Huawei Customer Service Center.

      N => Inconsistent. Go to step d.

    4. Reconfigure the rack type on the BSC LMT.

      Reconfigure the rack type on the BSC LMT. Ensure that the configured rack type is consistent with the actual rack type. Check whether the alarm is cleared.

      Y => The alarm is cleared. The alarm handling is complete.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.

  3. Sub No. 2:
    1. On the LMT, set the transmission mode of the rack to E1 transmission mode. Ensure that the optical transmission property is consistent with the data configured.

      Y => The alarm is cleared. The alarm handling is complete.

      N => The alarm is not cleared. Go to step b.

    2. Replace the DMCM.

      Y => The alarm is cleared. The alarm handling is complete.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.
