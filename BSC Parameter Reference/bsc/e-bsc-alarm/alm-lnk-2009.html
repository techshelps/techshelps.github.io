Check the frequency data according in the following steps.

1) In the management tree of the LMT, right-click the cell where the TRX is located under the BSC6000 node. A shortcut menu is displayed.

2) Choose Configure Cell Property on the shortcut menu. The Configuring Cells Properties dialog box is displayed.

3) In the Configure Cell Property dialog box, click Next.

4) Select the cell, and then click Set Cell Attributes. The Set Cell Attributes dialog box is displayed.

5) In the Set Cell Attributes dialog box, select the TRX.

6) Click TRX Config. The Configure TRX Property dialog box is displayed.

7) In the Configure TRX Property dialog box, choose Channel Attributes tab.

8)Check whether the frequency configured for this TRX is valid for the TRX type.
