ALM-2208 Clock Reference Abnormal Alarm

Description

The BTS3X reports this alarm when its clock is set to lock to the BSC clock or an external synchronization clock but actually fails to lock to either.

Impact on the System

There might be an offset between the clock of BTS3X and that of other BTSs upon occurrence of this alarm, causing abnormal HO(handoffï¼‰, network selection and reselection.

System Actions

None.

Possible Causes

  • Upper level clock reference is abnormal.
  • The clock of the transmission link is not available.
  • The clock module is faulty.

Procedure

  1. Check upper level clock reference

    Check whether the BSC clock or an external synchronization clock is normal.

    Y=> Go to Step 2.

    N=> Correct the BSC clock or an external synchronization clock. The alarm handling ends.

  2. Check the clock of the BTS3X transmission link

    Test the frequency offset of the transmission link clock of BTS3X using a frequency meter. Check whether the measured frequency offset is at least 0.05 PPM.

    Y=> The transmission link clock is abnormal, indicating that the E1 or optical transmission link or the clock source is not available. Perform a loopback test on the transmission link segment by segment to locate and clear the fault. The alarm handling ends.

    N=> The transmission link clock is normal. Go to Step 3.

  3. Replace the TMU

    Replace the TMU and check whether the alarm is cleared.

    Y=> The alarm handling ends.

    N=> Huawei Customer Service Center.
