Check the configuration of the hopping frequency

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Right-click a cell and choose Configure Cell Property on the shortcut menu.

The Configuring Cells Properties dialog box is displayed.

3) Select a cell and click Next, and then click Set Cell Properties.

The Set Cell Attributes dialog box is displayed.

4) Click HOP at the bottom of the page.

The Set FH Attributes dialog box is displayed.

5) Check whether RF FH is chosen in the FH Mode option.

If so, check which frequency band these hopping frequencies belong to, 900M or 1800M.

Note:If the board property is 900M high band, it indicates the DRU is of valid GSM900M system, and the corresponding ARFCN should be within the range of 1-124.If the board property is 1800M high band, it indicates the DRU is of valid DCS1800M system, and the corresponding ARFCN should be within the range of 512-885.If the board property is 900M low band, it indicates the DRU is of valid R-DCS1800M system, and the corresponding ARFCN should be within the range of 955-1023 and 0-54.
