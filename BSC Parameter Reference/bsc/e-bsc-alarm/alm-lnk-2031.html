Handle it according to the settings of the Boolean alarm

Do as follows to set the appropriate Enable/Disable status and Valid Level.

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Right-click the site in the BSC6000 management tree and choose Configure Site Board Property on the shortcut menu.

3)Click Configure Site Board Attributes.

The Site Device Property dialog box is displayed.

4)Select Configure Site Board attributes tab and set the Board Parameters Configuration Enable to Yes.

5) Under the Configure Site Board Attributes tab, you can set the status and level of the Boolean values.

For the Boolean values that are being used, set the Alarm Disabled Configuration to Enabled, and then set an appropriate value for each Boolean value.

For the Boolean values that are not being used, set the Alarm Disabled Configuration to Disabled.
