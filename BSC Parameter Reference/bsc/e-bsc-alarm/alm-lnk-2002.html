Check the Channel Type

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Select a cell and click the Cell Attributes tab.

3) Select a TRX under the Assigned TRXs at the bottom of the Cell Attributes tab page and click TRX Config.

The Browse TRX Attributes dialog box is displayed.

4) Select Channel Attributes and check the following items:

Whether the Abis Mode of the Main BCCH or the combined BCCH is configured with Auto Allocate.

Whether there is only one Main BCCH or combined BCCH for a TRX within a cell.

Whether the CBCH (SDCCH/8) is configured in Channel No.0, 1, 2, or 3.
