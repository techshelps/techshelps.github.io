Check whether the DIP switches of BTS3002C is correct

For a BTS3002C that works in optical transmission mode, check whether digit 1 of DIP switch S1 is set to OFF. For a BTS3002C that works in E1 transmission mode , check whether digit 1 of DIP switch S1 is set to ON.
