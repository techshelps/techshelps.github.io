Check whether the power type is consistent with that of the DTRU power type

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Right-click a TRX and choose Configure TRX Prop on the shortcut menu.

The Configure TRX Property Wizard is displayed.

3) Click Set TRX Attributes.

The Configure TRX Property dialog box is displayed.

4) Select Device Attributes and check whether the power type is configured correctly.
