Set the link code of the port of the BSC to the same value as that of the port of the peer equipment

1) Log in to the LMT and then choose Alarm Maintenance > Browse Alarm. Then, view the Subrack No., Slot No., and Port No. in the alarm location information to confirm the locations of the BSC interface board and of the port.

2) Select the BSC on the Management Tree tab page.

3) According to the Subrack No. and Slot No., select the corresponding BSC interface board on the BSC Device Panel tab page.

4) Right-click the BSC interface board and then choose Configure Board Attributes from the shortcut menu.

5) Click Next.

6) In the displayed dialog box, set the Line Code to the same value as that of the port of the peer equipment.

Note: The value of Line Code can be B8ZS or HDB3.
