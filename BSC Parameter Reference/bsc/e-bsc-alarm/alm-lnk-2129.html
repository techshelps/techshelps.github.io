Replacing a PMU Board

Note：The replacement of the PMU will cause the interruption of the management to the power module; the AC/DC power module turns into the float charging without current limitation. Meanwhile, the interruption will cause a communication failure alarm at the IOMU.

Note：Wearing the antistatic wrist strap and gloves when performs the operation; make sure that the grounding terminal grounds to the earth.

The procedure for replacing a PMU board is as follows:

1) Record the management configuration parameters of the faulty PMU.

If the communication fails, look for the history record. Make sure that the management configuration parameters be the same with the actual system configuration.

2) Remove the screws.

Loosen the two fasteners on the panel of the PMU board in an anti-clockwise direction.

3) Remove the faulty module from the slot quickly (The PMU is hot-pluggable.)

4) Push the new module into corresponding slot smoothly through the rail.

5) Make sure that the new module works normally.

Check whether the indicator of the module displays correctly after five minutes when the module is in position:

Y=> If the RUN displays green and the ALM (red) in the panel fades, it means the new module works normally. Go to step 6).

N=> If the RUN does not display green or the ALM (red) displayed, it means that the new module is not in position. Reinstall the board.

6) Tighten the screws.

Tighten the two screws on the panel of the module in a clockwise direction and power on the module when the module functions well.

7) Configure the management parameters of the PMU board.

Configure the PMU board by referring to the configuration parameters of the faulty PMU recorded in step 2) through the Site Maintenance Terminal System

8) Testing.

Check whether the analog value, Boolean value, and the alarm value collected by the PMU display normally.

Y=> If the analog value, Boolean value, and the alarm value do not display normally, reinsert the PMU board.

N=> If the analog value, Boolean value, and the alarm value display normally, go to step 9).

9) Replacing PMU board succeeds.
