Checking the BTS Version

There are three ways of checking the software version:

1) Choose Site Maintenance > Maintain Site > Query Site Running Software Version in the menu bar of the LMT.

2) Right-click the BTS in the LMT management tree and choose Query Site Running Software Version on the shortcut menu.

3) Right-click a board on the right panel of the LMT and choose Query Board Information on the shortcut menu.
