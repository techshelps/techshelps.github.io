Impact on the System

The BTS hardware may be destroyed if the alarm is reported.

System Actions

None.

Possible Causes

  • The environment temperature is higher than the upper temperature threshold.
  • The setting of the upper temperature threshold is improper.
  • The temperature sensor is faulty.
  • The cable connecting to the temperature sensor is faulty.
  • The lightning protection board is faulty.
  • The environment monitoring module is faulty.

Procedure

  1. Judge the causes according to other alarm information.

    View the temperature of the modules to judge if the environment temperature is normal.

    Y=> Go to Step 2.

    N=> The temperature is abnormal. Adjust it normal.

  2. Find out other causes.
    1. Make sure that the upper temperature threshold of the DEMU is right.

      Y=> Go to Step c.

      N=> Go to Step b.

    2. Reset the upper temperature threshold. Check if the alarm is cleared.

      Y=> The alarm is cleared.

      N=> The alarm is not cleared. Go to Step c.

    3. Check whether the lightning protection board works normally.

      Y=> The alarm is cleared.

      N=> The alarm is not cleared. Go to Step d.

    4. Check whether the temperature sensor works normally.

      Y=> The alarm is cleared.

      N=> The alarm is not cleared. Go to Step e.

    5. Make sure that the cable of the temperature sensor is properly connected.

      Y=> The alarm is cleared.

      N=> The alarm is not cleared. Go to Step f.

    6. Reset the environment monitoring module by software or hardware. Check if the alarm is cleared.

      Y=> The alarm is cleared.

      N=> The alarm is not cleared. Please contact Huawei Customer Service Center.
