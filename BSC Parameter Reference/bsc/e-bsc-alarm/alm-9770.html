ALM-9770 Parameter set collision alarm

Description

Attribute

Alarm ID Alarm Severity Alarm Type
9770 Warning Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

93:FMU

94:FMUA
3 Board No.

Number of the fault board

0-8:FMU/FMUA
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

0: Speed Adjusting Mode Setting Conflict

1: Rotating Speed Setting Conflict

2: Air Inlet Overtemperature and Undertemperature Alarm Threshold Setting Conflict

3: Speed Adjusting Control Setting Conflict

4: Environment Monitoring Alarm Suppress Setting Conflict

5: Boolean Alarm Level Setting Conflict

6: Analog Alarm Level Setting Conflict
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

None.

System Actions

None.

Possible Causes

The parameter settings of GSM fans do not match that of UMTS fans

Procedure

  1. If the alarm sub-category number is 0, check whether the adjustment mode of GSM fan speed is the same as that of the UMTS fan speed.

  2. If the alarm sub-category number is 1, check whether the information about the adjustment mode of GSM fan speed is the same as that of the UMTS fan speed.

  3. If the alarm sub-category number is 2, check whether the GSM overtemperature and undertemperature alarm thresholds at the air inlet are the same as the UMTS configurations.

  4. If the alarm sub-category number is 3, check whether the temperature of starting the adjustment of GSM fan speed, temperature of stopping the adjustment of UMTS fan speed, percentage of adjusting the GSM fan speed, and maximum GSM fan speed are the same as the UMTS configurations.

  5. If the alarm sub-category number is 4, check whether the status of GSM channels 1 to 7 environmental alarms and the configurations of channels 0 to 3 reserved Boolean are the same as the UMTS configurations.

  6. If the alarm sub-category number is 5, check whether the level of GSM channels 1 to 5 Boolean alarms and the level of channels 0 to 3 reserved Boolean alarms are the same as the UMTS configurations.

  7. If the alarm sub-category number is 6, check whether the upper and lower alarm thresholds of the GSM ambient temperature sensor and ambient humidity sensor are the same as the UMTS configurations.
