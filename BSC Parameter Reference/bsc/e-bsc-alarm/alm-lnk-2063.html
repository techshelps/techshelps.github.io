Reset the upper temperature limit

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Right-click a site and choose Configure Site Board Property on the shortcut menu.

The Configure Site Board Property Wizard is displayed.

3) Click Configure Site Board Attributes.

The Site Device Property dialog box is displayed.

4) Right-click the DPMU and choose Configure Board Attributes on the shortcut menu.

The Site Base Board Property dialog box is displayed.

5) In the Configure Site Board Attributes tab page, the high temperature alarm threshold can be viewed. If the threshold is less than 50°C, modify the value as the setting is inappropriate.

(Note:Set the Board Parameter Configuration Enabled to Yes before modification, and then adjust the temperature alarm threshold.)
