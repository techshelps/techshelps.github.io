4194 Board type mismatch alarm

Description

The alarm is reported when the type of a board actually installed in a slot of the rack is different from that of the board configured in the same slot at the BSC side.

Attribute

Alarm ID Alarm Severity Alarm Type
4194 Major Fault

Alarm Character

Impact on the System

The DTRU in the slot where alarm is generated cannot work.

System Actions

The DTRU is back to normal when the alarm is cleared.

Possible Causes

The board type actually installed in the slot of the rack is different from the data configured in the slot at the BSC side.

Procedure

  1. Replace the board according to the system configuration.
    1. Check the type of the board in the faulty slot in the system configuration.
    2. Replace the board according to the queried board type.
    3. Check whether the alarm is cleared:

      Y => The alarm is cleared. The alarm handling ends.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.
