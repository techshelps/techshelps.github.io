340 Cell PS Service Faulty

Description

This alarm is generated when cell distribution is failed, the cell is forbidden due to no license, the Um interface is blocked, or the Gb interface is blocked.

Attribute

Alarm ID Alarm Severity Alarm Type
340 Critical Fault

Parameters

ID Name Meaning
1 CellId

ID of the cell where the alarm is generated
2 Cause

0=Cell distribution failure,1=The cell is forbidden due to no license,2=The Um interface is blocked,3=The Gb interface is blocked
3 CellName

Cell Name of the cell where the alarm is generated
4 CGI

Cell Global Identifier of the cell where the alarm is generated
5 Site No.

Number of the site where the cell generating an alarm is located
6 Site Name

Name of the site where the cell generating an alarm is located
7 Alarm Attribute

0=Normal, Alarms period are longer than the Transient Threshold.

1=Transient Count, The times and the period of the alarms whose periods are shorter than the Transient Threshold are summed up. If the sum result is longer than the Alarm Occurrence Period Threshold or Alarm Occurrence Times Threshold, an alarm so called Transient Count Alarm is triggered. Because Transient Count Alarm is based on accumulative result, the recovery of the alarm may occur at least one Summing Cycle later. The Summing Cycle can be query by MML "LST STATSLDWIN".

Impact on the System

The cell cannot provide packet services.

System Actions

None.

Possible Causes

  • Cause 0: The cell distribution fails.
  • Cause 1: The cell is forbidden due to no license.
  • Cause 2: The Um interface is blocked.
  • Cause 3: The Gb interface is blocked.

Procedure

  1. Check the value of the parameter "Possible cause" in the information used for locating the fault.
    1. "Possible cause" is 0. Add the GDPUP to increase the number of packet DSPs available.
    2. "Possible cause" is 1. Contact Huawei Customer Service Center to apply for a license that supports this feature.
    3. "Possible cause" is 2. Go to 2.
    4. "Possible cause" is 3. Go to 3.
  2. Run the MML command DSP PSCELL to check whether the cell Um administration state is blocked.

    Y => Adjust the BTS to unblock the Um interface. For example, unblock the blocked cell.

    N => The alarm is cleared.

  3. Run the MML command DSP PSCELL to check whether the cell Gb administration state is blocked.

    Y => Go to 4.

    N => The alarm is cleared.

  4. If the Gb interface board is the GEPUG, check whether the following types of parent alarms exist.
    1. Log in to the LMT. Choose Alarm Maintenance > Browse Alarm to check whether any of the following parent alarms is reported.

      331 NSVC Faulty

      332 NSVL Faulty

      333 NSE Faulty

      342 PTP BVC Faulty

      Y => Go to b.

      N => Go to 5.

    2. If any of these alarms is reported, rectify the fault according to the alarm help. Then, check whether the alarm is cleared.

      Y => The alarm handling is complete.

      N => Go to 5.

  5. Check whether the alarm is cleared.

    Y=> The alarm handling is complete.

    N => Contact Huawei Customer Service Center.
