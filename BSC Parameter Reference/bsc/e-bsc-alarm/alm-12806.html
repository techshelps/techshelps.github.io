ALM-12806 Board Software Fault

Description

This alarm is reported when a fault that cannot be cleared occurs during the running of the base station software. The faults include memory release request failure, task creation failure, system object (such as queue) creation failure, message sending failure, and DPD configuration file loading failure.

Impact on the System

When the alarm is generated, the services carried by the base station are disrupted.

System Actions

The system may automatically reset.

Possible Causes

  • BTS3900E module software running fault
  • DSP running fault
  • Mailbox fault
  • Frame number or timeslot number error
  • DPD configuration file nonexistent or damaged

Procedure

  1. Check the software version.

    On the BSC, check whether the software version is correct.

    Y=> The software version is correct. Go to step 2.

    N=> The software version is incorrect. Upload the Base Station Version again.

  2. Reset the base station.

    On the BSC or SMT, Reset the Site in Level 4, and then check whether the alarm is cleared.

    Y=> The alarm is cleared. End the alarm handling.

    N=> The alarm is not cleared. Contact Huawei Customer Service Center.
