upgrade the software of the BT

To upgrade the software of the BTS, do as follows:

1) Put the G3BTS34.30000.06.0303 software in a corresponding directory.

2) In the BSC Data Auto Configuration System, choose BSC > Dynamic Configuration without Command > BTS Software Index in the menu bar.

The Dynamic Configuration without Command dialog box is displayed.

3) Click BTS Software Index.

The BTS Software Configuration dialog box is displayed.

4) Click Add.

The Add Configuration Item dialog box is displayed.

5) Select the corresponding software and click OK.

The BTS Software Configuration dialog box is displayed.

6) Click the software and then click OK.

The index configuration is complete.

7) Log in to the Site Maintenance System.

8) Click a site and double-click Multi-site Software Load in the right pane.

The Multi-site Software Load dialog box is displayed.

9) Select the board and its software, and then click Load.

10) When the preceding operation is complete, click Multi-site Software Activation.

The Multi-site Software Activation dialog box is displayed.

11) Select the board, its software, and the site, and then click OK.
