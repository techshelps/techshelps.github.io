Query the battery current

To query the current of the storage batteries, do as follows:

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Right-click a site in the BSC6000 management tree and choose Monitor site Resource on the shortcut menu.

The Monitor Site Resource dialog box is displayed.

3) Choose Current of storage batteries in the Resource Type drop down list and click OK.
