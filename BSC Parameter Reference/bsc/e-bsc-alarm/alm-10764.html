ALM-10764 Temperature Too High Alarm

Description

This alarm is reported when the cabinet temperature is above the upper temperature limit (68°C by default).

Attribute

Alarm ID Alarm Severity Alarm Type
10764 Major Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

74:DHEU

95:DTCU
3 Board No.

Number of the fault board

0:DHEU/DTCU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255:None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

15:BTS3012AE;

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

The heat dissipation in the BTS3012AE cabinet is affected. The hardware of the BTS3012AE might be damaged.

System Actions

None.

Possible Causes

  • The cabinet temperature is too high.
  • The upper temperature limit is not properly set.
  • The temperature sensor is faulty.
  • The cables that connect the temperature sensor are faulty.
  • The fan is faulty.
  • The DHEC is faulty.

Procedure

  1. Check whether the upper temperature limit of the DHEU is properly set.

    Y=> Go to step 3.

    N=> Go to step 2.

  2. Reset the upper temperature limit.

    Observe whether the alarm is cleared after the upper temperature limit is reset.

    Y=> The alarm is cleared. Alarm handling is complete.

    N=> The alarm is not cleared. Go to step 3.

  3. Check whether the temperature sensor is normal.

    Y=> Go to step 4.

    N=> Go to step 6.

  4. Check whether the cabling of the temperature sensor is secure.

    Y＝> Go to step 6.

    N=> Go to step 5.

  5. Reconnect the cables of the temperature sensor.

    Observe whether the alarm is cleared after the reconnection.

    Y＝> The alarm is cleared. Alarm handling is complete.

    N=> The alarm is not cleared. Go to step 7.

  6. Replace the temperature sensor.

    Observe whether the alarm is cleared after the replacement.

    Y＝> The alarm is cleared. Alarm handling is complete.

    N=> The alarm is not cleared. Go to step 7.

  7. Check whether the fan is normal.

    Y＝> Go to step 9.

    N=> Go to step 8.

  8. Replace the fan.

    Observe whether the alarm is cleared after the replacement.

    Y=> The alarm is cleared. Alarm handling is complete.

    N=> The alarm is not cleared. Go to step 9.

  9. Replace the DHEC.

    Observe whether the alarm is cleared after the replacement.

    Y=>The alarm is cleared. Alarm handling is complete.

    N=> The alarm is not cleared. Please contact Huawei Customer Service Center.
