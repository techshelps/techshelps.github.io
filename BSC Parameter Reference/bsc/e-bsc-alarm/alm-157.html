157 License Inferior Limit

Description

This alarm is generated when the license file expires.

Attribute

Alarm ID Alarm Severity Alarm Type
157 Minor Fault

Parameters

ID Name Meaning
1 Alarm Attribute

0=Normal, Alarms period are longer than the Transient Threshold.

1=Transient Count, The times and the period of the alarms whose periods are shorter than the Transient Threshold are summed up. If the sum result is longer than the Alarm Occurrence Period Threshold or Alarm Occurrence Times Threshold, an alarm so called Transient Count Alarm is triggered. Because Transient Count Alarm is based on accumulative result, the recovery of the alarm may occur at least one Summing Cycle later. The Summing Cycle can be query by MML "LST STATSLDWIN".

Impact on the System

The license expires after 60 days. Then, all the dynamic allocation operations are prohibited. In addition, the license-controlled functions and resources are restricted.

System Actions

None.

Possible Causes

The license file expires.

Procedure

  1. On the MML client, run the MML command DSP LICENSE to query the information about the license file on the XPU. Check whether the license file expires according to the "running deadline" in the "Attrib" in the returned messages.

    Y=> Go to 2.

    N=> Contact Huawei Customer Service Center.

  2. Apply for the latest license file and activate the license.
    1. Copy the license file to "\bsc6000\Common\SWM\LICENSE" in the LMT installation directory. On the MML client, run the MML command DLD LICENSE to download the specified license file on the FTP server to the GOMU or GBAM server.
    2. On the MML client, run the MML command ACT LICENSE to activate the new license file. Then, check whether the alarm is cleared.

      Y=> The alarm handling is complete.

      N=> Contact Huawei Customer Service Center.
