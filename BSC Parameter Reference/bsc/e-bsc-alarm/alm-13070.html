ALM-13070 Mains Input Abnormal

Description

This alarm is reported when there are mains faults, such as AC power failure, lack of phase, overvoltage, and undervoltage.

Impact on the System

  • When there is no AC power, there are two cases: if the batteries are configured, the batteries supply power to the BTS. Before the batteries run down, the BTS works properly; if the secondary load shutdown is enabled, the secondary equipment is powered off and the services on the secondary equipment are disrupted when threshold for secondary load shutdown is reached.

  • When mains undervoltage occurs, the DC output power decreases, and the life of power system is shortened.
  • When mains overvoltage occurs, the power module may stop providing power output and the batteries supply power to the BTS. In this case, the life of power system is shortened.

System Actions

None.

Possible Causes

  • The alarm threshold is inappropriate.
  • The MCB for AC input is OFF.
  • The cable connections are faulty.
  • The AC power network or power distribution is faulty.
  • The monitoring unit is faulty.

Procedure

  1. Check for the problem in the alarm information.

    If the alarm is related to AC mains overvoltage or undervoltage, go to procedure [2].

    If the alarm is related to AC power failure or lack of phase, go to procedure [3].

  2. Check the threshold of the mains voltage alarm.
    1. Query the threshold of the mains power alarm. Check whether the threshold is appropriate.

      Y => The threshold is appropriate. Go to procedure 4.

      N => The threshold is inappropriate. Go to step b.

    2. Set an appropriate threshold for the alarm.

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to procedure 4.

  3. Check the setting of the AC MCB.
    1. Check whether the AC MCB is set to OFF.

      Y => The AC MCB is set to OFF. Go to procedure 4.

      N => The AC MCB is set to ON. Go to step b.

    2. Set the AC MCB to ON.

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to procedure 4.

  4. Check the connections between the AC power network and the cables in the equipment room.
    1. Check whether the AC power network and the cables in the equipment room are properly connected.

      Y => The AC power network and the cables in the equipment room are properly connected. Go to procedure 5.

      N => The AC power network and the cables in the equipment room are not properly connected. Go to step b.

    2. Reconnect or replace the cables.

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to procedure 5.

  5. Check the AC power network and power distribution.
    1. Check whether the AC power network or power distribution is functional.

      Y => The AC power network or power distribution is functional. Go to procedure 6.

      N => The AC power network or power distribution is not functional. Go to step b.

    2. Clear the fault in the AC power network or power distribution.

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to procedure 6.

  6. Replace the monitoring unit.

    Y => The alarm is cleared. No further action is required.

    N => The alarm is not cleared. Contact Huawei Customer Service Center.
