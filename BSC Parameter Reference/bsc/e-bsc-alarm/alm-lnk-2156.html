Replace the BTS3900B Module

1) Block the cell related to the faulty BTS3900B module.

Log in to the LMT. Click the site under the BSC. Right-click a cell, and choose Modify Administrative State on the shortcut menu. The Modify Administrative State dialog box is displayed. Select LOCKED to block the related cells. Click OK.

2) Power off the BTS3900B cabinet.

3) Disconnect the power cable and the Ethernet cable connected to the BSC.

4) Install a new BTS3900B module. Reconnect the Ethernet cable and the power cable.

5) Power on the BTS3900B cabinet.

6) Unblock the cell.

In the Modify Administrative State dialog box, select UNLOCKED to unblock the cell. Click OK.

7) Check the BTS3900B module replacement.
