Check whether there is no traffic for a long time

To check the traffic statistics in M2000, do as follows:

1) Log in to the M2000.

2) Choose Performance > Query Result > New Query (or Real Query).

The Query dialog is displayed.

3) Select Counter Selection and the TRX to be queried, and then click Query.

Check whether there has been a long time since the traffic flow is interrupted.
