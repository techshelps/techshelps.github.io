Check whether the Boolean alarm needs to be configured.

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Click a site in the BSC6000 management tree.

The Site Device Panel tab page is displayed.

3) Double-click the DPMU.

The Browse site Board Attributes dialog box is displayed.

4) Select the Configure Site Board Attributes tab and check whether the Enabled/Disabled status and the value of Valid Level is the required status and level.
