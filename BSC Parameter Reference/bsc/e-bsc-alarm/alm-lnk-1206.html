Change the threshold

Log in to the LMT.

Right-click the TRX you want to modify in the Management Tree tab page, and then choose Configure TRX Attributes on the shortcut menu. The Configure TRX Attributes Wizard dialog box is displayed.

Click Set TRX Attributes. The Configure TRX Attributes dialog box is displayed.

Click the Freqency Attributes tab, and change Flow Control Start Threshold and Flow Control end Threshold.
