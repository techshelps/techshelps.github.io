ALM-6502 Over Analog Input Signal 4 Alarm

Description

The alarm is reported when the external analog input signal 1 is higher than the threshold the user sets.

Attribute

Alarm ID Alarm Severity Alarm Type
6502 Warning Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

70:DEMU
3 Board No.

Number of the fault board

0-2:DEMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255:None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

12:BTS3012;

15:BTS3012AE;

16:BTS3012 II;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

The impacts depend on the configuration of the user.

System Actions

None.

Possible Causes

  • There are corresponding sensor alarms.
  • The distribution cables and lightening protection board are faulty.
  • There is software configuration error.
  • The environment monitoring module is faulty.

Procedure

  1. Judge the causes according to other alarm information.

    Check if there is corresponding sensor alarm.

  2. Judge the causes according to other alarm information.
    1. Check if the corresponding analog input signals to see if they exceed the thresholds.

      Y=> Go to alarm handling of corresponding analog input signals.

      N=> Go to Step b.

    2. Check the sensor. Replace the faulty one. Check if the alarm is cleared.

      Y=> The alarm is cleared.

      N=> The alarm is not cleared. Go to Step c.

    3. Make sure that the distribution cables of the DEMU front panel are properly connected. Check if the alarm is cleared.

      Y=> The alarm is cleared.

      N=> The alarm is not cleared Go to Step d.

    4. Make sure that the lightening protection board is normal. Check if the alarm is cleared.

      Y=> The alarm is cleared.

      N=> The alarm is not cleared. Go to Step e.

    5. Check whether the environment monitoring module is faulty.

      Y=> Replace the environment monitoring module.

      N=> Please contact Huawei Customer Service Center.
