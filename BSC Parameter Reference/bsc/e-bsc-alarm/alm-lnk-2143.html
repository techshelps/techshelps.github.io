Handle the abnormal TRX TCH release

1 Solution to the low success rate of TCH assignment

Check whether the data related to TCH assignment is correctly configured.

2 Solution to the low success rate of the incoming cell handovers

1) Solution to the low success rate of the internal incoming cell handovers

a. Ensure that there is no cell near the alarm cell that shares the same frequency or the BSIC.

b. Ensure that there is no alarm of the abnormal clock reference source.

c. Check the parameter "Receive quality performance measurement" to ensure that there is no high BER in the source cell and the target cell.

2) Solution to the low success rate of the external incoming cell handovers

a. Ensure that there is no cell near the alarm cell that shares the same frequency or the BSIC.

b. Ensure that there is no alarm of the abnormal clock reference source.

c. Check whether the data configuration related to the external incoming cell handovers is correct.
