Set the frame format of the port at the BSC the same as that at the peer equipment

1) Log in to the LMT and then choose Alarm Maintenance > Browse Alarm. Then, view the Subrack No., Slot No., and Port No. in the alarm location information to determine the locations of the BSC interface board and of the port.

2) According to the Subrack No. and Slot No., select the corresponding BSC interface board on the BSC Device Panel tab page.

3) Right-click the BSC interface board and then choose Configure Board Attributes from the shortcut menu.

4) Click Next.

5) Set the related parameters based on the type of BSC interface board.

a. If the BSC interface board is a GEIUx board, set the Frame Format parameter to the same value as that at the peer end in the displayed dialog box.

b. If the BSC interface board is a GOIUx board, click Next in the Configure Board Attributes dialog box. The Configure Board Attributes dialog box is updated. In the updated Configure Board Attributes dialog box, set the following parameters to the same values as those at the peer end: TX Frame Format, RX Frame Format, TX J0 Byte, RX J0 Byte, TX J1 Byte, RX J1 Byte, TX J2 Byte, and RX J2 Byte.
