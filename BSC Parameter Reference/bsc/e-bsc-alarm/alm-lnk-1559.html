Check whether the data configurations of the SS7 signaling links are consistent with the physical connections

1) Log in to the LMT and then select the BSC on the Management Tree tab page.

2) On the BSC Attributes tab page, click A Signaling Link.

3) In the displayed dialog box, check the following parameters of the SS7 signaling links between the BSC and the MSC:

If the BSC uses the BM/TC Combined configuration mode, record the values of the A Subrack No., A Slot No., and A Port No. parameters.

If the BSC uses the BM/TC Separated configuration mode, record the values of the BM Subrack No. of Ater Connection Path, A Subrack No., A Slot No., A Port No., and XPU Slot No. parameters.

4) In the BSC equipment room, check whether the data configurations of the SS7 signaling links are consistent with the physical configurations (board configuration and cable connection).
