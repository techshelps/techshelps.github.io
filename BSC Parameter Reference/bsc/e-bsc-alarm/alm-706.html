706 IP-based interface board Port Working Mode Fault

Description

Inform this alarm when working mode of FE port isn't 100M&full duplex,or working mode of GE port isn't 1000M&full duplex.

Attribute

Alarm ID Alarm Severity Alarm Type
706 Minor Fault

Parameters

ID Name Meaning
1 Subrack No.

Number of the subrack of the IP-based interface board
2 Slot No.

Number of the slot of the IP-based interface board
3 Port No.

Number of the port of the IP-based interface board
4 Alarm Attribute

0=Normal, Alarms period are longer than the Transient Threshold.

1=Transient Count, The times and the period of the alarms whose periods are shorter than the Transient Threshold are summed up. If the sum result is longer than the Alarm Occurrence Period Threshold or Alarm Occurrence Times Threshold, an alarm so called Transient Count Alarm is triggered. Because Transient Count Alarm is based on accumulative result, the recovery of the alarm may occur at least one Summing Cycle later. The Summing Cycle can be query by MML "LST STATSLDWIN".

Impact on the System

This maybe leads to packet loss.

System Actions

when the working mode is normal ,system will restore the alarm.

Possible Causes

Working mode of FE port isn't 100M&full duplex,or working mode of GE port isn't 1000M&full duplex.

Procedure

  1. Check whether the interface board over IP is configured, as required. That is, the FE interface must be in 100 Mbit/s full-duplex mode and the GE interface must be in 1000 Mbit/s full-duplex mode.

    Y=> Go to 2.

    N=> Modify the configuration of the interface board over IP.

  2. Check whether the peer interface board connected to the interface board over IP is configured, as required. That is, the FE interface must be in 100 Mbit/s full-duplex mode and the GE interface must be in 1000 Mbit/s full-duplex mode.

    Y=> Go to 3.

    N=> Contact the maintenance engineers of the peer equipment. The alarm handling is complete.

  3. Check whether only the transmission equipment supporting simplex mode exits in the transmission network between the interface board over IP and the peer equipment, for example, the half-duplex Ethernet hub.

    Y=> Request the maintenance engineers of transmission network to replace the half-duplex Ethernet hub with a full-duplex Ethernet switch. The alarm handling is complete.

    N=> Contact Huawei Customer Service Center.
