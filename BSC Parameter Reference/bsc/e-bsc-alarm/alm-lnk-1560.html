Reset the local process and core process of the server

The server can be GBAM or GOMU. The following paragraphs take the GOMU as an example to describe how to reset the local process and core process of the server.

1) Log in to the LMT and then choose BSC Maintenance > Maintain Device > Maintain GOMU.

2) In the displayed dialog box, select the process to be reset and then click Reset.
