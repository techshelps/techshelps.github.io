Description

Impact on the System

The DTMU runs abnormally.

System Actions

None.

Possible Causes

The main clock module hardware is faulty.

Procedure

  1. Reset the DTMU and check whether the alarm is cleared.

    Y=>The alarm is cleared. Alarm handling is complete.

    N=>Go to Step 2.

  2. Replace the DTMU and check whether the alarm is cleared.

    Y=>The alarm is cleared. Alarm handling is complete.

    N=>The alarm is not cleared. Please contact Huawei Customer Service Center.
