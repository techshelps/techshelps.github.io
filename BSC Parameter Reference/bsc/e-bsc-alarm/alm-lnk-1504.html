Check whether the E1/T1 cable between the BSC and the peer equipment is correctly connected

1) View the Subrack No. and Slot No. in the alarm location information to determine the location of the interface board that reports the alarm.

2) Based on the type of interface board, check whether the E1/T1 cable is correctly connected according to the following rules.

a. If the interface board is GEIUT and if the BSC adopts the BM/TC separated mode and remote GTCS configuration, the rules for connecting the GEIUT in the GMPS/GEPS to the GEIUT in the GTCS are as follows:

The TX wire of the E1/T1 cable for the GEIUT in the GMPS/GEPS must be connected to the RX wire of the E1/T1 cable for the GEIUT in the GTCS.

The RX wire of the E1/T1 cable for the GEIUT in the GMPS/GEPS must be connected to the TX wire of the E1/T1 cable for the GEIUT in the GTCS.

b. If the interface board is of other types, the rules for connecting the E1/T1 cable between the BSC and the peer equipment (DDF or peer NE) are as follows:

The TX end of the E1/T1 cable for the BSC interface board must be connected to the RX end of the E1/T1 cable for the peer equipment.

The RX end of the E1/T1 cable for the BSC interface board must be connected to the TX end of the E1/T1 cable for the peer equipment.
