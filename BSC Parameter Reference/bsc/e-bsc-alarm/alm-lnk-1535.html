Check whether the data configuration of the port of the BSC is the same as the data configuration of the port of the peer NE

1) Log in to the LMT and then view the Subrack No., Slot No., and Port No. (or Tributary Unit No.) in the alarm location information.

2) Select the BSC on the Management Tree tab page.

3) On the BSC Device Panel tab page, right-click the corresponding GEIUx/GOIUx board according to the Subrack No. and Slot No. in the alarm location information, and then choose Query Interface Board Port Information from the shortcut menu.

4) Based on the type of interface board, perform the following operations:

a. If the board is a GEIUx board, set the Subrack No., Slot No., Query Object, and Port No. in the Board Location area. On the Query E1T1 Property tab page, click Query. Then, check whether the Tx Frame Format is the same as the Rx Frame Format.

b. If the board is a GOIUx board, set the Subrack No. and Slot No. in the Board Location area. In addition, set the Query Object to Port and set the Port No. to the same value as the Tributary Unit No. in the alarm location information. On the Query OIU Port Status tab page, click Query. Then, check whether the TX Frame Format, TX J0 Byte, TX J1 Byte, and TX J2 Byte are the same as the RX Frame Format, RX J0 Byte, RX J1 Byte, and RX J2 Byte respectively.

Y => The data configuration of the port of the BSC is the same as the data configuration of the port of the peer NE.

N => The data configuration of the port of the BSC is different from the data configuration of the port of the peer NE.
