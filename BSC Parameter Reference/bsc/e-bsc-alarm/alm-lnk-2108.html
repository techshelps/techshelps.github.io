Check BS_AG_BLKS_RES

To check whether the parameters of the reserved blocks are set correctly, do as follows：

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Select a cell in the BSC6000 management tree.

The Cell Attributes tab page is displayed.

3) Click Idle Mode on the right of the page.

The Browse Idle Parameter dialog box is displayed.

Check whether the BS_AG_BLKS_RES is set correctly.

Note：

a.If the channel type is combined BCCH, check whether the BS_AG_BLKS_RES is less than or equal to 2. In this case, if the transmission of system information is desired at the extension BCCH, check whether the BS_AG_BLKS_RES is greater than 0.

b.If the channel type is SDCCH8_CBCH, check whether the BS_AG_BLKS_RES is greater than or equal to 1.
