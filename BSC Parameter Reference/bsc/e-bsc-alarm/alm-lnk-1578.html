Check whether the C2 byte configuration of the BSC interface board is the same as that of the interface board of the peer NE

1) Log in to the LMT and view the Subrack No. and Slot No. in the alarm location information.

2) Select the BSC on the Management Tree tab page.

3) On the BSC Device Panel tab page, right-click the corresponding GOIUx board according to the Subrack No. and Slot No. in the alarm location information, and then choose Query Interface Board Port Information from the shortcut menu.

4) In the displayed dialog box, specify Subrack No. and Slot No., and set Query Object to Sub-board in the Board Location area. On the Query OIU Sub-board Status tab page, click Query. Then, check whether the value of RX C2 Byte is 0x02.

Y => The value of RX C2 Byte is 0x02. The C2 byte configuration of the BSC interface board is the same as that of the interface board of the peer NE.

N => The value of RX C2 Byte is not 0x02. The C2 byte configuration of the BSC interface board is different from that of the interface board of the peer NE.
