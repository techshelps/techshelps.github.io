ALM-5652 PSU Safeguard Alarm

Description

This alarm is reported when the power module in the Sidepower system has a temperature alarm or a fan alarm. The yellow indicator is on when this alarm is generated.

Attribute

Alarm ID Alarm Severity Alarm Type
5652 Minor Fault

Impact on the System

None.

System Actions

None.

Possible Causes

The temperature of the power module in the Sidepower system exceeds the upper threshold (55 ℃).

Procedure

  1. Check for the over voltage or under voltage alarms of the –48 V DC power.
    1. Check whether the –48V abnormal voltage alarms exist:

      Y => The precedent alarms exist. Go to step b.

      N => The precedent alarms do not exist. Go to procedure 2.

    2. Handle the –48 V abnormal voltage alarms.
    3. Check whether the alarm is cleared:

      Y => The alarm is cleared. The alarm handling ends.

      N => The alarm is not cleared. Go to procedure 2.

  2. Replace the faulty PSU.
    1. Replace the faulty PSU.
    2. Check whether the alarm is cleared:

      Y => The alarm is cleared. The alarm handling ends.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.
