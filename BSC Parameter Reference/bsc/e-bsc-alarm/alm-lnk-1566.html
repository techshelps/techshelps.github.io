Check the status of board clock

1) Log in to the LMT and then select the BSC on the Management Tree tab page.

2) Choose BSC Maintenance > Maintain Device > Query BSC Board Clock State. Set the parameters in the Location area and then click Query. In the area that displays the query result, check whether the State is Normal.

Y => The State is Normal. The board clock is normal.

N => The State is Abnormal. The board clock is not normal.
