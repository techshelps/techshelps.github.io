Load the BTS Version

1） Save the latest software version of the BTS3900E to the related directory.

2） Choose BTS Maintenance > Download BTS Software on the LMT. Add the BTS software saved in Step 1. Click Download to download the software.

3） Choose BTS Maintenance > Configure BTS Software on the LMT. Fill in the related parameters. Click Add, and then click Submit.

4） Choose BTS Maintenance > Load BTS Software on the LMT. Select the BTS type and configured BTS software version. Operate according to the procedure for BTS software loading, and observe the loading progress.

5） When the progress is 100%, choose BTS Maintenance > Activate BTS Software on the LMT. Select the BTS type, and then select the software and board to be activated. Operate according to the procedure for the activation.
