ALM-3052 LAPD alarm

Description

This alarm is triggered upon disconnection of the LAPD link.

Attribute

Alarm ID Alarm Severity Alarm Type
3052 Major Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

135:MFU
3 Board No.

Number of the fault board

0:MFU 0

1:MFU 1
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255:None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board or module is located

7: BTS3001C-108/BTS3001C-116

8: BTS3001C-204
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

All services supported by the local BTS3001C are interrupted upon the occurrence of this alarm.

System Actions

When this alarm occurs, the system shuts off the power amplifier automatically.

Possible Causes

  • The transmission line is not in proper connection.
  • The data configuration is incorrect.
  • Inner container hardware of the BTS fails.

Procedure

  1. Check where there is any correlated alarm

    Check whether the ALM-3152 E1 Local alarm is reported on the LMT.

    Y=> Handle the ALM-3152 E1 Local alarm according to the corresponding instructions to check whether it is cleared. The alarm is cleared, Alarm handling is completed. The alarm is not cleared, Go to Step 2.

    N=> Go to Step2.

  2. BER test

    Loopback transmission: If the BTS3001C adopts E1 transmission mode, then perform loopback test for the E1 cable from the BSC or the senior BTS. If the BTS3001C adopts optical transmission mode, perform loopback test for the optical fiber from the BSC or the senior BTS and test the timeslot corresponding to the LAPD to see whether there are any error codes.

    Y=> There is a fault on the transmission line. Repair the transmission line. The alarm handling ends.

    N=>Go to Step 3.

  3. Check the data configuration

    Check whether the physical connection of the faulty site is consistent with the port configuration.

    Y=>The physical connection of the site is consistent with the port configuration, Go to Step 4.

    N=>The physical connection of the site is not consistent with the port configuration. In this case, modify the data configuration and change the value consistent with the physical connection . The alarm handling ends.

  4. Reset the BTS

    Perform level-4 reset on the BTS3001C through the local maintenance system. Check whether the alarm is cleared.

    Y=> The alarm handling ends.

    N=>Contact Huawei Customer Service Center to replace the faulty hardware.
