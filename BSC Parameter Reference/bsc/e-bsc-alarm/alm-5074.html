ALM-5074 MPT Minor Clock Abnormal

Description

This alarm is reported when the local MPT E1 clock or the system clock for each baseband board has no output or is faulty.

Attribute

Alarm ID Alarm Severity Alarm Type
5074 Minor Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

87:GTMU
3 Board No.

Number of the fault board

0:GTMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

0: E1 clock

1: system clock
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

  • Minor
  • 1、 If the MPT has no system clock, the baseband boards do no have a system clock. In this case, no services can be launched.
  • 2、 If the E1 clock is faulty, the local E1s/T1s on the board are unavailable.

System Actions

None.

Possible Causes

There maybe hardware fault existing with the board.

Procedure

  1. Reset the MPT.
    1. Reset the MPT.
    2. Check whether the alarm is cleared:

      Y => The alarm is cleared. End alarm handling.

      N => The alarm is not cleared. Go to Step 2.

  2. Replace the MPT.

    Replace the MPT. Check whether the alarm is cleared

    Y => The alarm is cleared. End alarm handling.

    N => The alarm is not cleared. Please contact Huawei Customer Service Center.
