Load the Required DTMU Software

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Choose Site Maintenance > Load site Software in the menu bar.

The Load Site Software dialog box is displayed.

（Note： Make sure that the DTMU software is loaded to the OMU with the correct software version before step 2).）

3) Select the correct site in the Site Type, the software version in the Software Type drop down list, and click the DTMU in the browser pane.

4) Click Next.

5) Perform the activation of software when the software is loaded.
