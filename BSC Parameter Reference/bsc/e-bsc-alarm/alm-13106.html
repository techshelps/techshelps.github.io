ALM-13106 Burglar Alarm

Description

This alarm is reported when a person enters the equipment room or the door of the cabinet is open.

Attribute

Alarm ID Alarm Severity Alarm Type
13106 Critical Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

162:GMU
3 Board No.

Number of the fault board

0:GMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

1: Door status alarm

2: Infrared alarm
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

30:BTS3900E;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count
13 Problem 1: Door status alarm 2: Infrared alarm

Problem

1: Door status alarm

2: Infrared alarm

Impact on the System

The equipment may be damaged or stolen.

System Actions

None.

Possible Causes

  • The configuration is incorrect.
  • The cable connection is faulty.
  • The related sensor is faulty.
  • The corresponding monitoring unit is faulty.

Procedure

  1. Check the configuration.
    1. Check whether the configuration at the remote end is correct:

      Y => The configuration is correct. Go to procedure 2.

      N => The configuration is incorrect. Go to step b.

    2. Correct the configuration, and then check whether the alarm is cleared:

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to procedure 2.

  2. Check the cable connection between the related sensor and the monitoring unit.

    a: Check whether the cable is properly connected between the sensor and the monitoring unit:

    Y => The cable is properly connected. Go to procedure 3.

    N => The cable is not properly connected. Go to step b.

    1. Reconnect or replace the cable, and then check whether the alarm is cleared:

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to procedure 3.

  3. Replace the sensor.
    1. Replace the sensor, and then check whether the alarm is cleared:

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to step b.

    2. Replace the monitoring unit, and then check whether the alarm is cleared:

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.
