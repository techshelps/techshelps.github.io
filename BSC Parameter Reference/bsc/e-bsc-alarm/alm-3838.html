ALM-3838 Over Humidity Alarm

Description

This alarm is reported when the equipment room humidity exceeds the upper limit.

Attribute

Alarm ID Alarm Severity Alarm Type
3838 Critical Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

13:EAC
3 Board No.

Number of the fault board

0:EAC0
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255:None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

9: BTS3002C
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

The BTS equipment room humidity is too high, which may affect the services.

System Actions

None.

Possible Causes

  • The environment temperature is lower than the set lower limit.
  • The lower limit is not properly set.
  • The temperature sensor and its connecting cable is faulty.
  • The main board of the environment alarm box is faulty.

Procedure

  1. Check the set lower limit of the environment alarm box and see if it is properly set.

    Y=>Go to Step 3.

    N=>Go to Step 2.

  2. Re-set a proper lower limit and see if the alarm is cleared.

    Y=>The alarm is cleared. Alarm handling is completed.

    N=>The alarm is not cleared. Go to Step 3.

  3. Check if the equipment room temperature is too low.

    The normal temperature range of the BTS30 and the BTS312 is -5oC - 45oC. Check if the equipment room temperature is below the set lower limit.

    Y=>Go to Step 4.

    N=>Go to Step 5.

  4. Turn on the air conditioner to raise the temperature and see if the alarm is cleared.

    Y=>The alarm is cleared. Alarm handling is completed.

    N=>The alarm is not cleared. Go to Step 5.

  5. Check the temperature sensor and its connecting cable and see if the connection is OK.

    Y=>Go to Step 7.

    N=>Go to Step 6.

  6. Connect the temperature sensor cable and see if the alarm is cleared.

    Y=>The alarm is cleared. Alarm handling is completed.

    N=>The alarm is not cleared. Go to Step 7.

  7. Replace the temperature sensor and its connecting cable and see if the alarm is cleared.

    Y=>The alarm is cleared. Alarm handling is completed.

    N=>The alarm is not cleared. Go to Step 8.

  8. Replace the main board of the alarm box and see if the alarm is cleared.

    Y=>The alarm is cleared. Alarm handling is completed.

    N=>The alarm is not cleared.Please contact Huawei Customer Service Center.
