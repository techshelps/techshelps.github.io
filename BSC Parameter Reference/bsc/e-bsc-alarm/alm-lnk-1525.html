Unblock the TRX

1) Log in to the LMT and then view the Site Name and Cell Name in the alarm location information.

2) On the Management Tree tab page, select the main BCCH TRX of the corresponding cell.

Note: The main BCCH TRX is generally the first TRX of the cell.

3) Right-click the main BCCH TRX and then choose Modify Administrative State from the shortcut menu.

4) In the displayed dialog box, set the Modified Status in the Operation Status area to Unlock.

5) Click the Apply button.
