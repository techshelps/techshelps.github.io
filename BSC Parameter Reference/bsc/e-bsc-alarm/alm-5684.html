ALM-5684 Power system alarm

Description

This alarm is reported when the power system is faulty.

Attribute

Alarm ID Alarm Severity Alarm Type
5684 Major Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

72:DPMU

133:APM/APMU
3 Board No.

Number of the fault board

Value range:0-8:DPMU/APM/APMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255: None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

15:BTS3012AE;

Number of the rack where the faulty board is located.

Number of the subrack where the faulty board is located.

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

If N+1 backup mechanism is not used for the power system, the normal operation of the entire system will be affected.

System Actions

None.

Possible Causes

  • The undervoltage protection mechanism for input voltage is used by the PSU.
  • The overvoltage protection mechanism for output voltage is used by the PSU.
  • The overtemperature protection mechanism is used by the PSU.
  • The fans are blocked.
  • The system is overloaded.

Procedure

  1. You can locate the fault of the power system by viewing the indicators on the panel of the PSU.

    Probable Causes Troubleshooting

    1. The power input indicator (green) is off.

      Probable Causes : 1. There is no input power. Troubleshooting: Make sure there is input power.

      2 .The polarity of the input power is wrong or the fuse is burnt.

      Troubleshooting: Replace the faulty PSU.

    2. The protection indicator (yellow) is on.

      Probable Causes : 1. The DC input voltage exceeds the threshold.

      Troubleshooting: Make sure the DC input voltage is within the normal value.

    3. The DC output voltage exceeds the overvoltage threshold.

      Troubleshooting: Replace the faulty PSU.

      The power supply module is overloaded or short-circuited. Thus, the current-limiting protection mechanism is triggered and the DC output voltage exceeds the current-limiting threshold. The module restores to normal operation after the load is decreased to normal value or after the short-circuit is cleared.

      Probable Causes : 1. The fans are blocked. Remove the object that blocks the normal operation of the fans.

    4. The air inlet or outlet is blocked, or the dust is accumulated on the dust filter or the barrier.

      Troubleshooting: Remove the object that blocks the air inlet or outlet. Or clean the dust on the dust filter or the barrier.

    5. The temperature is too high. There is a heating object around the air inlet.

      Troubleshooting: Remove the heating object to reduce the temperature of the environment.

    6. The PSU is not securely inserted. Troubleshooting: Re-insert the PSU.
    7. The fault indicator (red) is on.

      Probable Causes : The fans stop. Troubleshooting: Replace the faulty fans.

      Y=> The alarm is cleared. The alarm handling is complete.

      N=> The alarm persists. Please contact Huawei Customer Service Center.
