ALM-3934 Load Power Off Alarm

Description

This alarm is reported when some parts of the equipment is powered off.

Attribute

Alarm ID Alarm Severity Alarm Type
3934 Critical Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

85:APM
3 Board No.

Number of the fault board

0:PMU0
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

0:low voltage power off

1:MCU controlled power off

2:high temperature power off
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

9: BTS3002C
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

The services may be interrupted.

System Actions

None.

Possible Causes

  • When the DC power supply fails, or the temperature inside the cabinet is too high, the MCU or the PMU of the BTS power off the some of the less important devices.
  • 0: low voltage power off
  • 1: background control power off
  • 2: high temperature power off

Procedure

  1. Judge the causes of the alarm.

    MCU controlled power off: Go to Step 2.

    Low voltage power off: Go to Step 2.

    High temperature power off: Go to Step 3.

  2. Check if the AC power supply fails.

    Check on site if the AC power supply input to the BTS fails.

    Y=>Go to Step "Handle the problem in AC power supply failure and see if the alarm is cleared".

    N=>Go to Step 4.

    Handle the problem in AC power supply failure and see if the alarm is cleared.

    Y=>Alarm handling is completed.

    N=>Go to Step 4.

  3. Lower the temperature inside the cabinet

    Lower the temperature inside the cabinet and see if the alarm is cleared.

    Y=>Alarm handling is completed.

    N=>The alarm is not cleared. Go to Step 4.

  4. Replace the PMU and see if the alarm is cleared.

    Y=>Alarm handling is completed.

    N=>The alarm is not cleared. Please contact Huawei Customer Service Center
