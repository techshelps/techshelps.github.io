Check whether the network communication between the BSC and the CBC is normal

On the computer with the CBC, choose Start > Run.

In the displayed dialog box, enter cmd.

In the DOS window, enter ping xx.xx.xx.xx(xx.xx.xx.xx indicates the IP address of the XPUC) to check whether the communication between the CBC and the BSC is normal.
