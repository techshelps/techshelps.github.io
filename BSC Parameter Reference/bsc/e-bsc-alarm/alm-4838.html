ALM-4838 Rack Type Mismatch Alarm

Description

This alarm is reported when the rack type is not consistent with that in the data configuration.

Attribute

Alarm ID Alarm Severity Alarm Type
4838 Major Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

64:DTMU/DOMU

87:GTMU
3 Board No.

Number of the fault board

0-5:DTMU/GTMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255: None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

12:BTS3012;

13:BTS3006C;

14:BTS3002E;

15:BTS3012AE;

16:BTS3012 II;

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

The boards in the extension cabinet group can not work correctly.

System Actions

None.

Possible Causes

  • The DIP switch of the rack in the extension cabinet group is wrongly set.
  • The rack type is not consistent with that in the data configuration.

Procedure

  1. Check whether the DIP switch of the rack in the extension cabinet group is correctly set.

    Y => Go to procedure 3.

    N => Go to procedure 2.

    The settings of the DIP switches are as follows (for the BTS3012, BTS3012AE, and BTS3012II):

    Check whether the DIP switches of the DCSU are set correctly.

    There are totally 14 DIP switches on the DCSU. The settings of the DIP switches are described as follows:

    The SW11 is used for terminal matching selection. The setting of the SW11 is as follows:

    In a single cabinet, SW11 is set to all ON.

    In combined cabinets, all SW11s of the cabinets are set to all ON.

    In the case of two combined groups, all SW11s of the cabinets are set to all ON.

    In the case of three combined groups, the SW11 of the main cabinet in the main cabinet group is set to all OFF, and the SW11s of all the other cabinets are set to all ON.

    The SW1 is used for determining the main cabinet in the main cabinet group. The setting of the SW1 is as follows:

    All ON indicates that the selected cabinet is the main cabinet in the main cabinet group.

    All OFF indicates other circumstances.

    The SW2–SW5 are used for configuring the main and extension cabinets and for terminal matching selection. The settings of the SW2–SW5 are as follows:

    SW2, SW3, SW4, and SW5

    All On indicates that the selected cabinet is the main cabinet.

    All OFF indicates that the selected cabinet is the extension cabinet.

    The SW8 is used for configuring the cabinet numbers. The setting of the SW8 is as follows:

    1 2 3 4

    ON ON ON ON Indicates that the selected cabinet is the main cabinet in the main cabinet group.

    OFF OFF ON ON Indicates that the selected cabinet is the main cabinet in the extension cabinet group 1.

    ON OFF OFF ON Indicates that the selected cabinet is the main cabinet in the extension cabinet group 2.

    The SW9 and SW10 are used for configuring the cabinet types. The settings of the SW9 and SW10 are as follows:

    SW9 Bit SW10 Bit

    1 2 3 4 1 2 3 4

    ON ON ON ON ON ON ON ON Indicates that the cabinet type is BTS3012.

    OFF ON ON ON ON ON ON ON Indicates that the cabinet type is BTS3012AE.

    ON OFF ON ON ON ON ON ON Indicates that the cabinet type is BTS3012II.

  2. Reconfigure the DIP switch of the rack.

    Reconfigure the DIP switch of the rack based on the actual rack type, and check whether the alarm is cleared.

    Y => The alarm is cleared. The alarm handling is complete.

    N => The alarm is not cleared. Go to procedure 3.

  3. Check the rack type configured on the BSC.

    Check whether the rack type configured on the BSC is consistent with the actual rack type

    Y => Consistent. Contact Huawei Customer Service Center.

    N => Inconsistent. Go to procedure 4.

  4. Reconfigure the rack type in the data configuration on the BSC side.

    Reconfigure the rack type in the data configuration on the BSC side. Ensure that the configured rack type is consistent with the actual rack type. Check whether the alarm is cleared.

    Y => The alarm is cleared. The alarm handling is complete.

    N => The alarm is not cleared. Contact Huawei Customer Service Center.
