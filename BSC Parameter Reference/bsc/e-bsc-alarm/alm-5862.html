ALM-5862 Rack Low Temperature Alarm

Description

This alarm is triggered when the internal temperature of the cabinet is lower than the threshold(0 °C).

Attribute

Alarm ID Alarm Severity Alarm Type
5862 Major Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

72:DPMU
3 Board No.

Number of the fault board

Value range:0-8:DPMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255: None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

3:BTS30;

5:BTS312;

10:BTS3012A;

11:BTS3006A;

12:BTS3012;

13:BTS3006C;

14:BTS3002E;

16:BTS3012 II;

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

The services may be interrupted due to low environment temperature of the boards.

System Actions

None.

Possible Causes

  • The internal temperature of the cabinet is lower than 0 °C.
  • The temperature sensor is damaged.

Procedure

  1. Check the internal temperature of the cabinet
    1. Log in to the BSC6000 Local Maintenance Terminal.
    2. Click a site and choose Site Operation > Configure Site Property > Extend.

      Check the Upper Temperature Thrsh. and Lower Temperature Thrsh. , and then check whether the temperature is lower than the threshold.

      Y=> Go to Step 2.

      N=> The temperature is normal. Go to Step 3.

  2. Check the air conditioner and the fan

    Check the air conditioner and the fan. Clear the faults if any. View whether the alarm is cleared after the internal temperature of the cabinet resumes normal.

    Y=> The alarm handling ends.

    N=> Go to Step 3.

  3. Check the temperature sensor.

    Check the temperature sensor. Replace the temperature sensor if faulty. View whether the alarm is cleared.

    Y=> The alarm handling ends.

    N=>Please contact Huawei Customer Service Center.
