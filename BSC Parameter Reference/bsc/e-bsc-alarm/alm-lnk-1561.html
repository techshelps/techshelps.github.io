Check whether the BSC board is faulty

1) Log in to the LMT and then choose Alarm Maintenance > Browse Alarm. Then, view the Site Name in the alarm location information.

2) Select the corresponding BTS on the Management Tree tab page.

3) In the Site Description area, view the type of BSC Abis interface board that is connected to the BTS, and the subrack number and slot number of the BSC Abis interface board.

4) On the BSC Device Panel tab page, check whether the BSC Abis interface board in the corresponding subrack and slot and the GXPUM in the subrack are highlighted in red.

Y => The board is highlighted in red. The board is faulty.

N => The board is not highlighted in red. The board is normal.
