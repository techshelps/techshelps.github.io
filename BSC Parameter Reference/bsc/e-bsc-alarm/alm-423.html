423 Extension Signaling Link Failure

Description

This alarm is reported when the extension signaling link between the BSC and the BTS is disconnected.

Attribute

Alarm ID Alarm Severity Alarm Type
423 Critical Fault

Parameters

ID Name Meaning
1 Site No

Number of the BTS where the faulty extension signaling link is configured
2 Site Name

Site Name
3 Alarm Attribute

0=Normal, Alarms period are longer than the Transient Threshold.

1=Transient Count, The times and the period of the alarms whose periods are shorter than the Transient Threshold are summed up. If the sum result is longer than the Alarm Occurrence Period Threshold or Alarm Occurrence Times Threshold, an alarm so called Transient Count Alarm is triggered. Because Transient Count Alarm is based on accumulative result, the recovery of the alarm may occur at least one Summing Cycle later. The Summing Cycle can be query by MML "LST STATSLDWIN".

Impact on the System

The communication services are affected. If the period of link disconnection exceeds 10 minutes, the PTU is automatically reset.

System Actions

After the alarm is generated, the BTS attempts to connect to the BSC periodically.

Possible Causes

  • The transmission link is faulty.
  • The GEHUB/GFGUB/GOGUB is not properly installed.
  • The BTS is powered off.
  • The PTU board in the BTS is faulty.
  • The format of the E1 frame on the GEHUB/GFGUB/GOGUB does not match.

Procedure

  1. Check whether the number of the E1 port connected to the BTS is consistent with the configured data.

    Y => Go to 3.

    N => Reconfigure the data.

  2. Check whether the GEHUB/GFGUB/GOGUB is loosely inserted.

    Y => Insert the GEHUB/GFGUB/GOGUB tightly.

    N => Go to 3.

  3. Check whether the Abis transmission line is faulty.

    Y => Repair the Abis transmission cable.

    N => Go to 4.

  4. Check whether the BTS is powered off.

    Y => Power on the BTS.

    N => Go to 5.

  5. Check whether the PTU is faulty.

    Y => Replace the PTU board.

    N => Go to 6.

  6. Check whether the frame format configured on the GEHUB/GFGUB/GOGUB is consistent between the local and peer ends.

    Y=> Please contact Huawei Customer Service Center.

    N=> Set the frame format correctly and ensure that the frame format of the GEHUB/GFGUB/GOGUB is consistent with that of the BTS.
