Check whether the version of the software on the faulty board is the latest version.

On the LMT, choose BSC Maintenance > Query Software > Query Board Version Information.

In the Query Board Version Information dialog box, select the faulty board. Then select Software and Hardware Board and Patch Information and click Query. When the output window is displayed, query the information of the board.
