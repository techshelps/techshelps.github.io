Query the temperature of the built-in battery, battery load, and external battery cabinet

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Choose Site Maintenance > Maintain Site > Monitor site Resource.

The Monitor Site Resource dialog box is displayed.3) Select Temperature of storage batteries in the Resource Type drop down box.

You can query the temperature of the built-in battery, battery load, and external battery cabinet.
