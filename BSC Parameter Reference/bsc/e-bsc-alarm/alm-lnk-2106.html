Check channel attributes

To check the channel attributes, do as follows:

In the Navigation Tree on the left, click site. In the Site Device Panel area, double-click the TRX to be queried. The Browse TRX Attributes window is displayed. In the browse TRX Attributes window, click the Channel Attributes tab.

Check the following parameters in the Channel Attributes tab page:

a.Check whether the Abis Mode of the primary BCCH or combined BCCH is set to Auto Allocate.

b.Check whether there is only one TRX is configured with the primary BCCH or combined BCCH.

c.Check whether the CBCH (SDCCH/8) is configured in timeslot 0, 1, 2, or 3.
