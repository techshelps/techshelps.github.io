ALM-5748 Battery Cabinet Heater Fault Alarm

Description

This alarm is reported when the battery cabinet heater is abnormal.

Attribute

Alarm ID Alarm Severity Alarm Type
5748 Major Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

133:APM/APMU
3 Board No.

Number of the fault board

Value range:0-8:APMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255: None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

13:BTS3006C;

14:BTS3002E;

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

The BTS mains supply may be abnormal, which causes the interruption of services.

System Actions

None.

Possible Causes

This alarm is reported when the battery cabinet heater is abnormal.

Procedure

  1. Check whether the switch of the AC heater is turned on.
    1. Check on site whether the switch of the AC heater is turned on:

      Y => Go to procedure 2.

      N =>Turn on the switch of the AC heater. Go to procedure 2.

  2. Check the heating board and the relay.
    1. Turn off the switch of the AC heater, MCB of the battery loop, and MCB of the system.
    2. When the ambient temperature is below 5â„ƒ and the pins on the relay are interconnected, measure the voltage of two pins of each relay in serial connection on the heating board with a multimeter to see whether they are short-circuited.

      Y => Replace the heating board in the battery cabins.

      N => Replace the relay on the heating board in the battery cabin.

    3. After ensuring that the replaced cables of the heating board and the relay are correctly connected, turn on the general switch of the system, switch of the AC heater, and MCB of the battery loop in the AC power-on sequence. Use a calibrated outdoor thermometer to check whether the temperature of the battery cabin arises:

      Y => The alarm is cleared. The alarm handling ends.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.
