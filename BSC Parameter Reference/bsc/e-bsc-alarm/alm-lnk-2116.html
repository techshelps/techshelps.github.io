Check the BSC data configuration

To check BSC data configuration, do as follows:

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Double-click a MTR on the Site Device Panel.

The Browse TRX Attributes dialog box is displayed.

3) Check whether the frequency configured for this TRX is a valid value of GSM900M system.
