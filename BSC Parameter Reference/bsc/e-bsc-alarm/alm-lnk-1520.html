Reset the BSC board

1) View the Subrack No. and Slot No. in the alarm location information to determine the location of the board to be reset.

2) Use one of the following modes to reset the board.

GUI mode:

a. Log in to the LMT and then select the BSC on the Management Tree tab page.

b. On the BSC Device Panel tab page, right-click the board to be reset and then choose Reset BSC Board from the shortcut menu.

MML mode:

Run the MML command RST BRD.
