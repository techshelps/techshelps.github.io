Stop the timeslot loopback

1) Log in to the LMT and then select the faulty BTS on the Management Tree tab page.

2) In the Site Description area on the Site Device Panel tab page, view the subrack No., slot No., and port No. of the Abis interface board that is connected to the BTS.

3) Log in to the LMT and then select the BSC on the Management Tree tab page.

4) On the BSC Device Panel tab page, right-click the corresponding board and then choose Interface Board Port Timeslot Loopback from the shortcut menu. Then, click the Query button.

5) In the area that displays the query result, select the corresponding timeslot and click Stop.
