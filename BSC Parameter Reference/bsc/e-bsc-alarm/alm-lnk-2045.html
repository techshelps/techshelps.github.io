Reset the environment monitoring module by software or hardware

To reset the environment monitoring module by software, do as follows:

1) Start the LMT.

2) Right-click the DEMU and select the Board Reset short-cut menu.

To reset the environment monitoring module by hardware, do as follows:

Press the RST button on the front panel of the DEMU.
