Check whether the MTP3 link is deactivated

1) Log in to the LMT and then choose Alarm Maintenance > Browse Alarm. Then, view the SCCP Dest Signaling Point Code in the alarm location information.

2) Choose BSC Maintenance > Maintain Transmission and Signaling > Maintain MTP3 Link.

3) In the displayed Maintain MTP3 Link dialog box, click the MTP3 Link State tab. In the Location and Operation area, do as follows:

a. Set the Operation parameter to Query by Signaling Point.

b. Set the DPC parameter to the same value as the SCCP Dest Signaling Point Code.

4) Click the Query button.

5) In the area that displays the query result, check whether the Actived parameter of every MTP3 link is set to Active.

Y => The Actived parameter of every MTP3 link is set to Active. All the MTP3 links for the SCCP destination signaling point are activated.

N => The Actived parameter of every MTP3 link is set to Inactive. None of the MTP3 links for the SCCP destination signaling point is activated.
