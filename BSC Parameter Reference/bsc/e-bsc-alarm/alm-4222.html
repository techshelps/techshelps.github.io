ALM-4222 VSWR Prompt Alarm

Description

If the VSWR of the antenna system exceeds the Carrier Module VSWR warning alarm threshold, the VSWR warning alarm is generated.

Attribute

Alarm ID Alarm Severity Alarm Type
4222 Warning Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

88:DRRU

92:DRFU

96:MRRU

100:MRFU

101:GRFU

102:GRRU
3 Board No.

Number of the fault board

0-71:DRRU/DRFU/MRRU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255:None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

Once the alarm is generated, there is a decrease in the coverage area of the Carrier Module.

System Actions

None.

Possible Causes

  • The VSWR of the antenna system is too high.
  • The Carrier Module is faulty.

Procedure

  1. Check the connector of the cable connecting to the antenna port of the Carrier Module.

    Check whether the connector of the cable is securely connected to the antenna port of the Carrier Module.

    Y=> The connector is securely connected. Go to step 2.

    N=> The connector is not securely connected. Tighten the connector, and then check whether the alarm is cleared. If the alarm is cleared, the alarm handling is complete; otherwise, go to step [2].

  2. Check the connector of the cable connecting to the antenna port.

    Check whether the jumper connector on the tower or the jumper connector under the tower is securely connected to the feeder.

    Y=> The connector is securely connected. Go to step 3.

    N=> The connector is not securely connected. Tighten the connector, and then check whether the alarm is cleared. If the alarm is cleared, the alarm handling is complete; otherwise, go to step [3].

  3. Replace the jumper.

    Replace the jumper of the antenna system.

    Y=> The alarm is cleared. The alarm handling is complete.

    N=> The alarm persists. Go to step 4.

  4. Replace the Carrier Module.

    Replace the Carrier Module, and then check whether the alarm is cleared.

    Y=> The alarm is cleared. The alarm handling is complete.

    N=> The alarm persists. Go to step 5.

  5. Replace the antenna.

    Replace the antenna, and then check whether the alarm is cleared.

    Y=> The alarm is cleared. The alarm handling is complete.

    N=> The alarm persists. Please contact Huawei Customer Service Center.
