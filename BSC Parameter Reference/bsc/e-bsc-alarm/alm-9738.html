ALM-9738 Temperature high or low alarm

Description

The alarm is reported when the temperature of the air inlet and fan box exceeds the thresholds. Air inlet’s Upper thrsh.is 60oC, Fan box’s Upper thrsh.is 80oC, Air inlet’s Lower thrsh. is -20oC,Fan box’s Lower thrsh. Is -20oC,Air inlet’s Upper limit is 70oC,and Air inlet’s Lower limit is -45oC.

Impact on the System

The services may be interrupted or the hardware may be destroyed when the alarm is reported. All other normal fans auto run in full speed after the alarm.

System Actions

None.

Possible Causes

  • The environment temperature is over high or low.
  • The alarm thresholds are set incorrectly.
  • The temperature sensor is faulty.
  • The fan box is faulty.

Procedure

  1. Query the current temperature of the cabinet.
    1. Check whether the temperature of the cabinet is too high.

  2. Check whether the temperature sensor alarm exists.
    1. View the active alarm and check whether alarm 9374 exists, which is the temperature sensor alarm:

      Y => The precedent alarms exist. Go to step b.

      N => The precedent alarms do not exist. Go to procedure 3.

    2. Handle the temperature sensor alarm.
    3. Check whether the alarm is cleared:

      Y => The alarm is cleared. The alarm handling ends.

      N => The alarm is not cleared. Go to procedure 3.

  3. Check whether the alarms related to the fans exist.
    1. View the active alarm, and check whether alarm 9736 exists, which is the fan fault alarm:

      Y => The precedent alarms exist. Go to step b.

      N => The precedent alarms do not exist. Go to procedure 4.

    2. Clear the fan fault alarm.
    3. Check whether the alarm is cleared:

      Y => The alarm is cleared. The alarm handling ends.

      N => The alarm is not cleared. Go to procedure 4.

  4. Check whether the air conditioner in the equipment room is working normally.
    1. Check on site whether the air conditioner in the equipment room is working normally on site:

      Y => The air conditioner is functional. Go to procedure 5.

      N => The air conditioner is faulty. Go to step b.

    2. Restore the air conditioner.
    3. Check whether the alarm is cleared:

      Y => The alarm is cleared. The alarm handling ends.

      N => The alarm is not cleared. Go to procedure 5.

  5. Check whether the alarm is cleared when the indoor temperature is below 30℃.
    1. Check whether the alarm is cleared when the indoor temperature is below 30℃.

      Y => The alarm is cleared. The alarm handling ends.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.
