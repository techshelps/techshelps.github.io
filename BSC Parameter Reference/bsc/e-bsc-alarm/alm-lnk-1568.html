Check whether the link codes of the ports of both the BSC and the peer equipment are configured correctly

1) Log in to the LMT and then choose Alarm Maintenance > Browse Alarm. Then, view the Subrack No., Slot No., and Port No. in the alarm location information.

2) Log in to the LMT and then select the BSC on the Management Tree tab page.

3) On the BSC Device Panel tab page, right-click the corresponding GEIUx/GOIUx board according to the Subrack No. and Slot No. in the alarm location information, and then choose Query Interface Board Port Information from the shortcut menu.

4) Set the Subrack No., Slot No., Query Object, and Port No. in the Board Location area. Click Query. Then, check whether the RX Line Code and the TX Line Code are the same.

Y => The link code of the port of the BSC is the same as the link code of the port of the peer NE.

N => The link code of the port of the BSC is different from the link code of the port of the peer NE.
