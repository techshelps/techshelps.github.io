ALM-13108 Fan Stalled

Description

This alarm is reported when a fan is faulty.

Attribute

Alarm ID Alarm Severity Alarm Type
13108 Critical Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

162:GMU
3 Board No.

Number of the fault board

0:GMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

1: TCU-monitored fan stalled

2: FMU-monitored fan stalled
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

30:BTS3900E;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count
13 Problem 1: TCU-monitored fan stalled 2: FMU-monitored fan stalled

Problem

1: TCU-monitored fan stalled

2: FMU-monitored fan stalled

Impact on the System

Heat dissipation of the BTS does not work properly. In a severe case, the hardware of the BTS may be damaged.

System Actions

None.

Possible Causes

  • The fan is damaged.
  • The cable connected to the fan is faulty.
  • The fan monitoring board is faulty.

Procedure

  1. Reset the faulty board.
    1. On the LMT, reset the faulty board.
    2. Check whether the alarm is cleared:

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to procedure 2.

  2. : Observe the LED on the fan monitoring board.

    a: Observe the red LED on the fan monitoring board.

    b: Check whether the red LED is blinking:

    Y => Go to procedure 3.

    N => Go to procedure 5.

  3. Check the operating status of the fans.
    1. Remove the fan subrack quickly, and observe the fans.
    2. Find the stalled fan.

      c: Remove the cable that is connected to the fan, and install the cable again. Ensure that the cable is securely connected. Then, power on the fan subrack.

    3. Check whether the alarm is cleared:

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to procedure 4.

  4. Replace the fan or the fan subrack.
    1. If replacement of an individual fan is applicable, replace only the faulty fan. Otherwise, replace the fan subrack.
    2. Check whether the alarm is cleared:

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Go to procedure 5.

  5. Replace the fan monitoring board.
    1. Replace the fan monitoring board, and then power on the fan subrack again.
    2. Check whether the alarm is cleared:

      Y => The alarm is cleared. No further action is required.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.
