ALM-6166 Parameter set collision alarm

Description

Attribute

Alarm ID Alarm Severity Alarm Type
6166 Warning Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

70:DEMU/EMU
3 Board No.

Number of the fault board

Value range:0-8:DEMU/EMU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

0: Configuration Data Sending Conflict

1: Relay Parameter Setting Conflict

2: Sensor Alarm Information Clearing Conflict

3: Alarm Suppression/Enabling Setting Conflict

4: Boolean Alarm Level Setting Conflict
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

None.

System Actions

None.

Possible Causes

The parameter settings of GSM EMU do not match that of UMTS EMU.

Procedure

  1. If the alarm sub-category number is 0, check whether the GSM -48 V upper and lower alarm thresholds, +24 V upper and lower alarm thresholds, upper and lower alarm thresholds of the temperature sensor, upper and lower alarm thresholds of the humidity sensor, and upper and lower alarm thresholds of external analog sensors 1 to 4 are the same as the UMTS configurations.

  2. If the alarm sub-category number is 1, check whether the status of the GSM relay is the same as that of the UMTS relay.

  3. If the alarm sub-category number is 2, check whether the alarm cleared status of GSM sensors is the same as that of UMTS sensors.

  4. If the alarm sub-category number is 3, check whether the configurations of GSM analog alarms, Boolean alarms, and reserved Boolean alarms are the same as the UMTS configurations.

  5. If the alarm sub-category number is 4, check whether the high level and low level of GSM dedicated Boolean alarms and common Boolean alarms are the same as the UMTS configurations.
