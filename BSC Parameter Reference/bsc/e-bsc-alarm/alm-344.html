344 FAULTY DSP OVER LIMIT

Description

This alarm is generated when 16 or more DSPs among the 22 DSPs on the GDPUP are faulty.

Attribute

Alarm ID Alarm Severity Alarm Type
344 Critical Fault

Parameters

ID Name Meaning
1 Subrack No.

Subrack No. of the GDPUP where the alarm is generated.
2 Slot No.

Slot No. of the GDPUP where the alarm is generated.
3 DSPNum

Number of faulty DPS units on one GDPUP
4 Alarm Attribute

0=Normal, Alarms period are longer than the Transient Threshold.

1=Transient Count, The times and the period of the alarms whose periods are shorter than the Transient Threshold are summed up. If the sum result is longer than the Alarm Occurrence Period Threshold or Alarm Occurrence Times Threshold, an alarm so called Transient Count Alarm is triggered. Because Transient Count Alarm is based on accumulative result, the recovery of the alarm may occur at least one Summing Cycle later. The Summing Cycle can be query by MML "LST STATSLDWIN".

Impact on the System

  • The faulty DSP cannot provide services.
  • The system performance deteriorates.

System Actions

If the GDPUP is not faulty, the DSPs are reset automatically.

Possible Causes

  • The software loading to the DSP fails.
  • The configuration of the DSP fails.
  • The DSP fails to process services.
  • The DSP fault is detected.

Procedure

  1. Wait for at least three minutes for the system actions. Then, Check whether the DSP is faulty. Calculate the number of faulty DSPs and check whether the number exceeds the upper limit.

    Y => Reboot the GDPUP. Go to 2 after the board runs properly.

    Y => The alarm is cleared. The alarm handling is complete.

  2. Check whether the DSP is faulty. Calculate the number of faulty DSPs and check whether the number exceeds the upper limit.

    Y => Go to 3.

    Y => The alarm is cleared. The alarm handling is complete.

  3. Replace the GDPUP. Check whether the alarm is cleared after the board runs properly.

    Y=> The alarm handling is complete.

    N => Contact Huawei Customer Service Center.
