Modify the settings of the extension link port or internal cascading port.

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Right-click a site in the BSC6000 management tree and select Add or Delete Site Multiple Chain on the shortcut menu.

The Add or Delete Site Chain Wizard is displayed.

3) Click Add or Delete Chain.

The Add and Delete Site Chain dialog box is displayed.

4) Choose a site port in the Site Port List and click Delete Port.

5) When the site port is deleted, click New Port to add a new site slave chain.

The Add Site Slave Chain dialog box is displayed.

6) Select a GEIUB port in the GEIUB Port No. drop down list and a site port in the Site Port No. drop down list.

7) Click OK.
