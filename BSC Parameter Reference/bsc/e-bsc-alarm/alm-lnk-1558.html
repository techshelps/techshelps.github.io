Check whether the TMU software version is the same as the target version

1) Log in to the LMT and then choose Alarm Maintenance > Browse Alarm. Then, view the Site No. in the alarm location information.

2) Choose BTS Maintenance > Query Board Running Software Version.

3) In the displayed dialog box, set the Site Type and then move the sites to be queried from the Candidate Sites area to the Selected Sites area.

4) Set the Board Type to TMU and then click Start.

5) In the Result area, check whether the Software Version is the same as the target version.
