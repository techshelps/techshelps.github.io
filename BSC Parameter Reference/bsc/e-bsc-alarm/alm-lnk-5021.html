Check whether the IP connection between the IP-Interface-Boards of BSC and the SGSN is normal

[1] Ping the IP address of the remote NSVL. Then, check whether the IP addresses of the local NSVL and remote NSVL are normal.

Y => Normal.

N => Abnormal. Go to [2].

[2] Run the LST IPRT MML command. Then, check whether the IP route is added.

Y => The IP route is added. Go to [3].

N => The IP route is not added. Add an IP route.

[3] Check whether the network connection between the interface board in IP mode of the BSC and the interface board in IP mode of the SGSN is normal.
