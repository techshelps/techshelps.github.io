419 Bar code of BTS IP board config error

Description

The querying of the IP address of the configured active and standby Transmission Interface Boards based on the electronic label of Transmission Interface Board reported by the BTS fails. The alarm is reported.

Attribute

Alarm ID Alarm Severity Alarm Type
419 Critical Event

Parameters

ID Name Meaning
1 Electric Label

The electric label of the BTS
2 Alarm Attribute

0=Normal, Alarms period are longer than the Transient Threshold.

1=Transient Count, The times and the period of the alarms whose periods are shorter than the Transient Threshold are summed up. If the sum result is longer than the Alarm Occurrence Period Threshold or Alarm Occurrence Times Threshold, an alarm so called Transient Count Alarm is triggered. Because Transient Count Alarm is based on accumulative result, the recovery of the alarm may occur at least one Summing Cycle later. The Summing Cycle can be query by MML "LST STATSLDWIN".

Impact on the System

The BTS that supports IP transmission cannot obtain the IP address. Therefore, the BTS does not work, and all the services of the BTS are disrupted.

System Actions

None

Possible Causes

The electronic label of the active and standby Transmission Interface Boards configured on the LMT is different from the electronic label of the active and standby Transmission Interface Boards in the BTS.

Procedure

  1. Check whether the electronic label of the Transmission Interface Board indicated by the alarm is consistent with the electronic label of the Transmission Interface Board before delivery.

    Y => Go to 2;

    N => Go to 4.

  2. Check whether the electronic label of the Transmission Interface Board in the BTS is consistent with the electronic label of the Transmission Interface Board before delivery.

    Y => Go to 5;

    N => Go to 3.

  3. Reconfigure the electronic label of the Transmission Interface Board in the BTS on the basis of the electronic label of the Transmission Interface Board before delivery. Then, check whether the BTS is operational.

    Y => If the alarm is caused by the incorrect configuration of the electronic label, the alarm handling is complete;

    N => Go to 5.

  4. If the electronic label of Transmission Interface Board is faulty, contact Huawei Customer Service Center.

  5. If the Transmission Interface Board is faulty, contact Huawei Customer Service Center.
