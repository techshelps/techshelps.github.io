102 Disrupted Connection with the CBC

Description

This alarm is generated when the connection between the BSC and the CBC is disrupted.

Attribute

Alarm ID Alarm Severity Alarm Type
102 Major Fault

Parameters

ID Name Meaning
1 Subrack No.

Number of the subrack where the XPUM board is located
2 Logical Slot No.

Number of the slot where the XPUM board is located
3 CBC Port

Port number of cell broadcast center
4 CBC IP

IP address of cell broadcast center
5 CBC IP

IP address of cell broadcast center
6 CBC IP

IP address of cell broadcast center
7 CBC IP

IP address of cell broadcast center
8 Alarm Attribute

0=Normal, Alarms period are longer than the Transient Threshold.

1=Transient Count, The times and the period of the alarms whose periods are shorter than the Transient Threshold are summed up. If the sum result is longer than the Alarm Occurrence Period Threshold or Alarm Occurrence Times Threshold, an alarm so called Transient Count Alarm is triggered. Because Transient Count Alarm is based on accumulative result, the recovery of the alarm may occur at least one Summing Cycle later. The Summing Cycle can be query by MML "LST STATSLDWIN".

Impact on the System

When the connection with the CBC is disrupted, the BSC cannot receive new cell broadcast commands. This does not affect the execution of the received broadcast commands.

System Actions

The BSC sends the connection establishment request to the CBC every 0.5s to attempt to restore the connection.

Possible Causes

  • The network communication between the BSC and the CBC is faulty.
  • The CBC software is faulty or not running.

Procedure

  1. Check whether the network communication between the BSC and the CBC is normal.

    Y => Go to 2.

    N => Check the network connection between the CBC and the BSC to ensure that the connection is normal.

  2. Check whether the CBC software runs normally.

    Y => See the section "Routine Maintenance and Troubleshooting" in the User Guide in the cell broadcast database. If the fault cannot be rectified, contact Huawei Customer Service Center.

    N=> Run the Cbs.exe and Huawei.exe. For details, see "Software installed in CBS" in the Cell Broadcast Center Installation Manual in the cell broadcast database.1
