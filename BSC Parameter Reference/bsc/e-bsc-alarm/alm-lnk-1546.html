Query the BTS information according to the alarm location information

1) Log in to the LMT and then choose Alarm Maintenance > Browse Alarm. Then, view the Subrack No., Slot No., and Port No. in the alarm location information, that is, the subrack number and slot number of the Abis interface board, and the number of the port occupied by the BTS that is connected to Abis interface board.

2) Select the BSC on the Management Tree tab page.

3) According to the Subrack No. and Slot No., select the corresponding Abis interface board on the BSC Device Panel tab page.

4) Right-click the Abis interface board and then choose View Board from the shortcut menu.

5) According to the Subrack No., Slot No., and Port No., view the Direct Site Index and Direct Site Name on the Abis Interface E1 tab page in the displayed dialog box.

For example, if the Subrack No. = 1, Slot No. = 14, and Port No. = 1, perform the following steps to view the BTS information:

a. Log in to the LMT and then select the BSC on the Management Tree tab page.

b. On the BSC Device Panel tab page, select the board that is located in subrack 1 and slot 14.

c. Right-click the board and then choose View Board from the shortcut menu.

d. According to the Subrack No. = 1, Slot No. = 14, and Port No. = 1, view the Direct Site Index and Direct Site Name on the Abis Interface E1 tab page in the displayed dialog box.
