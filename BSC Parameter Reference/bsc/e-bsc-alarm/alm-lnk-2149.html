GTMU&UELP

The GTMU has five DIP switches. Each DIP switch has four DIP bits. The DIP switches S1 and S2 should be set in pairs,the functions of the DIP switches are as follows:

  • S1 is used to set E1 impedance. Table 5 describes the settings of the DIP switch.
  • S2 is used to set the grounding of E1/T1 transmission. Table 6 describes the settings of the DIP switch.
  • S3 is reserved.
  • S4 is used for the selection of E1 bypass. Table 7 describes the settings of the DIP switch.
  • S5 is used to set the timeslots when the E1 link is bypassed. Table 8 describes the settings of the DIP switch.
