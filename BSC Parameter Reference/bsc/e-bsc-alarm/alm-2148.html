ALM-2148 TRX Hardware Critical Alarm

Description

A TRX reports this alarm when detecting that the power supply allocated to the power amplifier in it is not available, or neither phase-lock loop in a TRX is available, or loading of the DSP fails.

Attribute

Alarm ID Alarm Severity Alarm Type
2148 Major Fault

Parameters

ID Name Meaning

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

65:DTRU

135:TRX
3 Board No.

Number of the fault board

Value range: 0-71
4 PassNo

Number of the passage where the faulty board is located.

Alarm sub-category number

0:Low Forward Power Alarm

4:Abnormal AD9857 Internal PLL

5:Abnormal in Checking AD9857 Configuration

6:Abnormal in Checking AD6650 Configuration

7:Abnormal I2C Link

8:Abnormal of I2C in Acquiring LM80

9:Abnormal of I2C in Acquiring DS1624

10:Abnormal of AD6650 in Outputting 541K Clock

11:Abnormal of AD9857 in Extending PLL

255:None
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board or module is located

3: BTS30

5: BTS312

10: BTS3012A

11: BTS3006A
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

All the services on the TRX will be interrupted upon occurrence of this alarm.

System Actions

If the TRX carries a BCCH, the system will start BCCH mutual-assistance.

Possible Causes

  • The hardware of the TRX running abnormal.
  • The hardware of the TRX is faulty.

Procedure

  1. Reset the TRX and see if the alarm is cleared.

    Y=> The alarm is cleared. Alarm Handling is completed.

    N=> The alarm is not cleared. Go to Step 2.

  2. Replace the TRX and see if the alarm is cleared.

    Y=> The alarm is cleared. The alarm handling ends.

    N=> The alarm is not cleared. Please contact Huawei Customer Service Center.
