Reconfigure the rack type in the data configuration on the BSC side. Observe whether the alarm is cleared after the reconfiguration

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Right-click a site in the BSC6000 management tree and choose Site Operation > Delete Site.

The Delete Site Wizard is displayed.

3) Delete a site according to the Delete Site Wizard.

4) When the a site is deleted, choose Site Operation > Add Site to add a new site.

The Add Site Wizard is displayed.

5) Click Add BTS in the Add Site Wizard.

The Add New Site dialog box is displayed.

6) Select a proper site in the Site Type drop down list and click OK.
