4144 TRX VSWR alarm

Description

This alarm is generated when the reverse power from the power output of the TRX is too high. This alarm is generated when the cables concerning the power output port of the TRX (TX1, TX2, TCOM) are not properly connected. This alarm is generated for the Carrier Module if the VSWR exceeds the major VSWR alarm threshold of the Carrier Module. This alarm is generated when the antenna jumper is not connected to the antenna port of the Carrier Module.

Impact on the System

Once the alarm is generated, the service carried by the TRX is totally interrupted.

System Actions

The TRX shuts down the power amplifier automatically. For the main BCCH TRX, the mutual TRX aid is enabled.

Possible Causes

  • The cable is not securely connected to the power output port of the TRX.
  • The cable is not securely connected to the power input port of the RF front-end unit.
  • The cable between the power output port of the TRX and the power input port of the RF front-end unit is damaged.
  • The RF front-end unit is faulty.
  • The DTRU is faulty.

Procedure

  1. According to the alarm information, check the connections between the modules and the cable connectors.
    1. Check whether the connections between the modules and the cable connectors are loose.

      Y => A connection is loose. Go to step b.

      N => All the connections are secure. Go to 2.

    2. Tighten the loose connectors.
    3. Reset the TRX. Check whether the alarm is cleared.

      Y => The alarm is cleared. The alarm handling is complete.

      N => The alarm is not cleared. Go to 2.

  2. Check the RF connections and site configuration.
    1. Check whether the RF connections and the site configuration are matched.

      Y => The connections of RF cables, jumpers, and RF modules are matched with the configuration of the logical transmit mode. Go to 3.

      N => The connections of RF cables, jumpers, and RF modules are not matched with the configuration of the logical transmit mode. Go to step b.

    2. Adjust the RF connections according to site configuration.
    3. Reset the TRX. Check whether the alarm is cleared.

      Y => The alarm is cleared. The alarm handling is complete.

      N => The alarm is not cleared. Go to 3.

  3. Replace the TRX. Check whether the alarm is cleared.

    Y => The alarm is cleared. The alarm handling is complete.

    N => The alarm is not cleared. Go to 4.

  4. Check the VSWR of the related components.
    1. Measure the VSWR by using the related instrument (for example, the SiteMaster) at the antenna port of the duplexer. Check whether the fault occurs on the jumper, surge protector, feeder, TMA, or the antenna.

      Y => A component is faulty. Go to step b.

      N => All the components are normal. Contact Huawei Customer Service Center.

    2. Replace the faulty component.
    3. Reset the TRX. Check whether the alarm is cleared.

      Y => The alarm is cleared. The alarm handling is complete.

      N => The alarm is not cleared. Contact Huawei Customer Service Center.
