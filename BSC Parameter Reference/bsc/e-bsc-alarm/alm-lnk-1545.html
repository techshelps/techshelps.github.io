Check whether the cell is blocked

1) Log in to the LMT and then choose Alarm Maintenance > Browse Alarm. Then, view the Site Name and Cell Name in the alarm location information.

2) On the Management Tree tab page, select the corresponding site and cell.

3) Right-click the main BCCH TRX and then choose Modify Administrative State from the shortcut menu.

4) In the displayed dialog box, check whether the Current Status in the Operation Status area is Lock.

Y => The Current Status is Lock. The cell is blocked.

N => The Current Status is Unlock. The cell is not blocked.
