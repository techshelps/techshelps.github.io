ALM-10784 Parameter set collision alarm

Description

This alarm is reported when the parameter related to the heater monitoring board is set to different values for GSM and UMTS networks.

Attribute

Alarm ID Alarm Severity Alarm Type
10784 Warning Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

74:DHEU

95:DTCU
3 Board No.

Number of the fault board

0:DHEU/DTCU
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number 

0: Conflict between the settings of the high temperature and low temperature alarm thresholds for GSM and UMTS networks 

1: Conflict between the settings of heater control parameter for GSM and UMTS networks 

2. Conflict between the settings of temperature difference parameter related to the temperature control system for GSM and UMTS networks
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

None.

System Actions

None.

Possible Causes

The parameter settings of the monitoring board of the GSM heater do not match that of the monitoring board of the UMTS heater.

Procedure

  1. If the alarm sub-category number is 0, check whether the GSM overtemperature alarm thresholds and undertemperature alarm thresholds are the same as the UMTS configurations.

  2. If the alarm sub-category number is 1, check whether the power-on temperature threshold and power-off temperature threshold of the GSM heater are the same as the configurations of the UMTS heater.

  3. If the alarm sub-category number is 2, check whether the GSM overtemperature state threshold, undertemperature state threshold, lowest expected temperature at the air outlet, allowed temperature control error, difference between the normal temperature at the air inlet and the environment temperature, difference between the normal temperatures at the air outlet and at the air inlet,and difference between the high temperature at the air outlet and the environment temperature are the same as the UMTS configurations.
