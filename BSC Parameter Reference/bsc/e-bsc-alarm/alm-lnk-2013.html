Activate the Software

1) Log in to the BSC6000 Local Maintenance Terminal.

2) Choose Site Maintenance > Activate site Software in the menu bar.The Activate Site Software dialog box is displayed.

3) Select the correct site in the Site Type, the software version in the Software Type drop down list, and click the DTMU in the browser pane.

4) Click Next to complete the operation.
