124 No A Interface Circuit Configured in the BSC

Description

[1]The MSC sends a circuit maintenance (such as resetting a circuit) message to the BSC, but the BSC is not configured with the corresponding circuit, the BSC sends the "Unequipped Circuit" message to the MSC and reports this alarm. [2]When the MSC sends an assignment request or a switchover request to the BSC, if the circuit identification code (CIC) in the assignment request or in the switchover requests is not configured in the BSC, the BSC sends the "Unequipped Circuit" message to the MSC and reports the alarm.

Attribute

Alarm ID Alarm Severity Alarm Type
124 Minor Event

Parameters

ID Name Meaning
1 DPCGROUP

DPCGROUP of the circuit that is not configured
2 CicNum

BSC Circuit identification code (CIC) numbers of the circuits those are not configured
3 CIC

BSC Circuit identification code (CIC) of the circuit that is not configured
4 OSP

Original signaling point (OSP) of the subrack where the circuit is located
5 Alarm Attribute

0=Normal, Alarms period are longer than the Transient Threshold.

1=Transient Count, The times and the period of the alarms whose periods are shorter than the Transient Threshold are summed up. If the sum result is longer than the Alarm Occurrence Period Threshold or Alarm Occurrence Times Threshold, an alarm so called Transient Count Alarm is triggered. Because Transient Count Alarm is based on accumulative result, the recovery of the alarm may occur at least one Summing Cycle later. The Summing Cycle can be query by MML "LST STATSLDWIN".

Impact on the System

If the CIC in an assignment request that the MSC sends to the BSC is not configured in the BSC, the call fails. If the CIC in a switchover request that the MSC sends to the BSC is not configured in the BSC, the switchover fails.

System Actions

The BSC sends the "Unequipped Circuit" message to the MSC.

Possible Causes

  • The CIC configuration in the BSC is inconsistent with that in the MSC. For example, the number of configured CICs or CIC range is different.
  • The circuit state configured in the BSC is inconsistent with that configured in the MSC.

Procedure

  1. Check whether the CIC configuration (for example, the number of configured CICs or the CIC scope) in the BSC is not consistent with that in the MSC.

    Y => Delete redundant CIC data from the MSC, or add required CIC data into the BSC.

    N => Go to 2.

  2. Check whether the circuit state configured in the BSC is not consistent with that configured in the MSC.

    Y => Modify the circuit state configuration in the BSC or in the MSC and ensure that the BSC and the MSC are consistent in the circuit state configuration.

    N => Please contact Huawei Customer Service Center.
