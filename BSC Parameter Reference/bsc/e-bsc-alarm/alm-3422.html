ALM-3422 Temperature A Level alarm

Description

This alarm is triggered when the BTS3001C cannot work normally because the BTS3001C MPA temperature is too high.

Attribute

Alarm ID Alarm Severity Alarm Type
3422 Minor Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

82:MCI
3 Board No.

Number of the fault board

0:MCI0

1:MCI1
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

255:None

Index of the cell where the board or module is located

65535: None

Impact on the System

If this alarm occurs when the BTS3001C ambient temperature is not normal, all the internal modules of MPA, MMU and MFU of the BTS3001C may be damaged in high temperature. In this case, the services will be interrupted.

System Actions

None.

Possible Causes

  • The MPA module temperature is too high.
  • The MPA module is damaged.
  • The alarm detection circuit of the MTR is faulty.

Procedure

  1. Display board temperature

    Query the temperature of each BTS3001C board through the remote or local maintenance system. Check whether the MPA module temperature is about 30 °C higher than other boards.

    NOTE:

    It takes the maintenance system about two minutes to display the actual temperature value of the boards.

    Y=>The temperature is higher than 75 °C. Go to step 3.

    N=>The temperature is not higher than 75 °C. Go to step 2.

  2. Reset the BTS

    In the remote or local maintenance system,perform fourth-level reset

    on the BTS3001C and then check whether the alarm is cleared.

    Y=>It is a false alarm. The alarm handling ends.

    N=>The possible cause is that the hardware in the BTS3001C is faulty. Please contact Huawei Customer Service Center.

  3. Check temperatures of other boards

    Check whether the temperature of the other boards is within 20 °C to 40 °C.

    Y=>The PA temperature collection circuit of BTS3001C has hardware faults.Please contact Huawei Customer Service Center.

    N=>The temperatures of all the boards are over 60 °C. Go to step 4.

  4. Observe external environment

    Check whether the external environment temperature is over 40 °C.

    Y=>Go to step 5.

    N=>Go to step 6.

  5. Improve external environment

    Protect the BTS3001C installation environment from direct sunlight and bad ventilation. Install a sunshade or awning on the BTS3001C and place the BTS3001C at a site with good ventilation. After that, check whether the board temperature has dropped. If the HPA temperature has decreased to a normal temperature, the alarm will be cleared.

    Y=>The alarm handling ends.

    N=>The possible cause is that the hardware in the BTS3001C is faulty.Please contact Huawei Customer Service Center.

  6. Check BTS3001C working current

    If the external environment is normal but the BTS3001C internal temperature is still over 30 °C higher than the external temperature, the cause may be a fault in the temperature control circuit of the hot plate. Use a galvanometer to test the working current of the BTS3001C. Check whether the current is over 1A.

    Y=>The temperature control circuit of the hot plate has a fault. It is necessary to replace the BTS3001C. Please contact Huawei Customer Service Center.

    N=>The fault cause cannot be located.Please contact Huawei Customer Service Center.
