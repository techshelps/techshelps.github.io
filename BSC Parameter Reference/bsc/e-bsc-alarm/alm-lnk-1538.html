Activate the MTP3 link

1) Log in to the LMT and then choose Alarm Maintenance > Browse Alarm. Then, view the Subrack No. and Slot No. in the alarm location information.

2) Choose BSC Maintenance > Maintain Transmission and Signaling > Maintain MTP3 Link.

3) In the displayed Maintain MTP3 Link dialog box, click the MTP3 Operation tab. In the Location and Operation area, do as follows:

a. Set the Operation parameter to Activate.

b. Set the Subrack No. to the same value as the Subrack No. in the alarm location information.

c. Set the XPU Slot No. to the same value as the Slot No. in the alarm location information.

d. Set the CPU No. to the same value as the CPU No. in the alarm location information.

4) Click the Apply button.
