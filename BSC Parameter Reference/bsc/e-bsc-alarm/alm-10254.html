ALM-10254 TMA Communication Alarm

Description

Communication between the TMA and the Antenna and TMA Control Unit is abnormal.

Attribute

Alarm ID Alarm Severity Alarm Type
10254 Major Fault

Parameters

ID Name Meaning
1 Site No.

Number of the site where the faulty board or module is located

Value range: 0-2047
2 Board Type

Number of the type which the faulty board belongs to.

67:DATU

98:GATM
3 Board No.

Number of the fault board

0-35:DATU/GATM
4 PassNo

Number of the passage where the faulty board is located.
5 SubLink No.

Alarm sub-category number

0:TMA communication alarm 1

1:TMA communication alarm 2

2:TMA communication alarm 3

3:TMA communication alarm 4

4:TMA communication alarm 5

5:TMA communication alarm 6
6 Cell Index

Index of the cell where the board or module is located

0-2047: Cell index of the faulty cell in the data configuration

65535: None
7 Site Type

Type of the site where the faulty board is located.

12:BTS3012;

13:BTS3006C;

14:BTS3002E;

15:BTS3012AE;

16:BTS3012 II;

17:DBS3900 GSM;

18:BTS3900 GSM;

19:BTS3900A GSM;
8 RackNo

Number of the rack where the faulty board is located.
9 BoxNo

Number of the subrack where the faulty board is located.
10 SlotNo

Number of the slot where the faulty board is located.
11 Site Name

Site Name
12 Alarm Attribute

Alarm Character

0:Normal

1:Transient Count

Impact on the System

The BTS cannot communicate normally with the TMA. The TMA cannot be operated normally.

System Actions

None.

Possible Causes

  • The TMA is not installed.
  • The communication link is faulty.
  • The Antenna and TMA Control Unit is faulty.

Procedure

  1. Check if the TMA is powered on and in position.

    Y=>Go to Step 3.

    N=>Go to Step 2.

  2. Power on the TMA or install it. Check if the alarm is cleared.

    Y=>The alarm is cleared. Alarm handling is complete.

    N=>Go to Step 3.

  3. Check if the communication cable of the TMA is properly connected.
    1. Check if the communication cable of the TMA is properly connected.

      Y=>Go to Step 4.

      N=>Go to Step b.

    2. Connect the communication cable of the TMA properly. Check if the alarm is cleared.

      Y=>The alarm is cleared. Alarm handling is complete.

      N=>Go to Step 4.

  4. Check if the feeders are connected properly.
    1. Check if the feeders are connected properly.

      Y=>Go to Step 5.

      N=>Go to Step b.

    2. Connect the feeders properly. Check if the alarm is cleared.

      Y=>The alarm is cleared. Alarm handling is complete.

      N=>Go to Step 5.

  5. Check if the Antenna and TMA Control Unit runs normally.

    Reset the DATU, power it off and on again, and Replace the faulty Antenna and TMA Control Unit. Check if the alarm is cleared.

    Y=>The alarm is cleared. Alarm handling is complete.

    N=>Go to Step 6.

  6. Replace the faulty TMA. Check if the alarm is cleared.

    Y=>The alarm is cleared. Alarm handling is complete.

    N=>The alarm is not cleared.Please contact Huawei Customer Service Center.
