Interf.Calculation Period(SACCH period(480ms))

Parameter Name

Interf.Calculation Period(SACCH period(480ms))

Description

This parameter specifies the period during which interference levels are averaged.

Before the BTS sends the radio resource indication message to the BSC, the interference levels on the idle channels in the period specified by this parameter are averaged. The result is used to classify the interference levels on the idle channels into five interference bands.

For details, see GSM Rec. 08.08, 08.58, and 12.21.

GUI Value Range

[1,31]

Default Value

20

Unit

Milliseconds

Step

480 ms

Modification Rules

None

Configuration Policy

If the value of this parameter is too great, the average result cannot reflect the change correctly. If the value of this parameter is too small, the averaging is performed too frequently and resources are wasted.

Recommended Value

20

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Radio Resources

Caution

1. Interf. Band Threshold 0 is not measured during traffic measurement.

2. For all versions of BTS2X, and earlier versions of BTS30 03.1130, BTS3001C07.0301, and BTS3002C02.0820, the value of this parameter must range from 85 to 115. If the value is beyond the range, the related cells do not work.

3. For BTS3X 03.1130 and later versions, BTS3001c 07.0301 and later versions, 3002c 02.0820 and later versions, and double-transceiver BTSs, the value of this parameter ranges from 48 to 115.
