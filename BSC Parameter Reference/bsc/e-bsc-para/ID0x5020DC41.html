Support EDA

Parameter Name

Support EDA

Description

Extended dynamic allocation (EDA) enables the MS to be allocated with more timeslots on the uplink, thus improving the uplink rate and helping to transmit large amounts of data on the uplink.This parameter specifies whether the EDA is supported.

GUI Value Range

[Not Support,Support]

Default Value

Not Support

Unit

None

Step

-

Modification Rules

None

Configuration Policy

In normal cases, the GPRS/EGPRS downlink services outnumber the GPRS/EGPRS uplink services. In some special cases, however, requirements for the uplink bandwidth are high; for example, a large-sized email is sent through the GPRS/EGPRS. In this case, this parameter should be set to Support.

EDA enables a single MS to be allocated with four timeslots on the uplink. If the MS high multislot class function is supported, the MS with the high multislot class 34 can be allocated with five timeslots on the uplink, thus meeting the high requirements for bandwidth on the uplink. The parameter used for supporting the high multislot class function is Support High Multislot Class.

Recommended Value

Not Support
