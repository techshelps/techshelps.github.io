Send Mode

Parameter Name

Send Mode

Description

BTS3012 series:

1. In a hybrid cabinet group, the send mode of the inserted double-transceiver TRX cannot be Wide Band Combining if the main cabinet group is composed of the BTS3012 cabinets and the extension cabinet group is composed of the BTS30 cabinets or BTS312 cabinets.

2. The send mode of the TRXs in the BTS3006C and BTS3002E can be No Combining, Diversity Transmitter, DDIVERSITY, or DPBT.

3. The receive mode of the TRXs in the BTS3006C and BTS3002E can be Independent Receiver, Dividing Receiver, or Four Diversity Receiver.

4. The send mode of the QTRU can be No Combining.

5. The receive mode of the QTRU can be Dividing Receiver.

NGBTS series:

6. The send mode of the DRRU and DRFU can be PBT, Diversity Transmitter, DDIVERSITY, DPBT, or Transmitter Independent or Combining.

7. The send mode of the BTS3900E can be Main Diversity.

8. The receive mode of the BTS3900E can be Main Transmitter Independent or Combining.

9. The send mode of the MRRU and GRRU can be No Combining, Diversity Transmitter, or DDIVERSITY.

10. The receive mode of the DRRU, DRFU, MRRU, and GRRU can be Four Diversity Receiver or Main Diversity.

11. The send mode of the MRFU and GRFU can be Transmitter Independent or Combining, or Diversity Transmitter.

12. The receive mode of the MRFU and GRFU can be Four Diversity Receiver or Main Diversity.

13. For the DRRU, the Send Receive Mode must be set to Double Feeder (2TX + 2RX) or Double Feeder (2TX + 4RX) if the send mode is set to Diversity Transmitter.

14. For the DRRU, the Send Receive Mode must be set to Double Feeder (2TX + 4RX) Receiver if the receive mode is set to Four Diversity Receiver.

15. For the MRRU, and GRRU, the TRX in the board cannot be Diversity Transmitter or DDIVERSITY if the following conditions are met simultaneously: The Send Receive Mode parameter is set to Double Feeder (2TX + 2RX), the TRX is assigned two cells, and the BSC does not support RanSharing.

16. For the MRFU, and GRFU, the Associated RXU Board No. parameter must be configured if the send mode of the TRX is set to Diversity Transmitter or if the receive mode of the TRX is set to Four Diversity Receiver.

17. If the send mode of the TRXs in the MRRU, and GRRU is set to Diversity Transmitter or DDIVERSITY, the Send Receive Mode parameter must be set to Double Feeder (2TX + 2RX) or Double Feeder (2TX + 2RX).

18. If the receive mode of the TRXs in the MRRU, and GRRU is set to Four Diversity Receiver, the Send Receive Mode parameter must be set to Double Feeder (2TX + 4RX).
