Positive or Minus Offset

Parameter Name

Positive or Minus Offset

Description

This parameter specifies whether the frequency is increased (positive) or decreased (minus) in comparison with the central frequency of the receive path.

GUI Value Range

[Positive,Minus]

Default Value

Positive

Unit

None

Step

-

Modification Rules

None

Configuration Policy

If the frequency of the interference band is smaller than the central frequency of the receive path, the parameter should be set to Positive. Otherwise, the parameter should be set to Minus.
