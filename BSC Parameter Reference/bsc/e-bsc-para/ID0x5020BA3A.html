Threshold of HCS Signal Strength(dB)

Parameter Name

Threshold of HCS Signal Strength(dB)

Description

This parameter specifies the threshold of HCS signal strength.

The MS uses the signal strength in the MR and this threshold to calculate C31, which is used for cell reselection.

This parameter is an optional field in the information element PSI3/PSI3bis. The value nouse indicates that this parameter will not be coded in the information element.

GUI Value Range

[-110,-108,-106,-104,-102,-100,-98,-96,-94,-92,-90,-88,-86,-84,-82,-80,-78,-76,-74,-72,-70,-68,-66,-64,-62,-60,-58,-56,-54,-52,-50,-48,nouse]

Default Value

-110

Unit

dB

Step

-

Modification Rules

None

Configuration Policy

If the threshold of HCS signal strength is high, it is difficult for the cell to be selected.

If the threshold of HCS signal strength is low, it is easy for the cell to be selected.

Recommended Value

-110

Parameter Relationship

None
