Interf. Band Threshold 5 (-dBm)

Parameter Name

Interf. Band Threshold 5 (-dBm)

Description

This parameter specifies the threshold used for interference measurement.

The BSS measures the uplink quality of the idle channels in the serving cell, calculates and reports the interference on each idle channel. This helps the BSC assign channels.

According to the strength of interference signals, the interference signals are classified into six levels, including no interference. The values of the receive levels used for the classification of interference signals are called interference band thresholds. The BTS determines the interference level based on these thresholds and then sends a radio resource indication message to the Scathe BSC compares the busy and idle channels reported in the measurement report and in the radio resource indication message to assign the channels with less interference. The interference band measurement result provides reference for threshold setting and interference analysis. For details, see GSM Rec. 08.08, 08.58, and 12.21.

If the difference between the values of two thresholds is too small, the interference change is too obvious. If the difference between the values of two thresholds is too great, the interference change is not indicated.
