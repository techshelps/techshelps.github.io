Inner Flow Control Allowed

Parameter Name

Inner Flow Control Allowed

Description

This parameter specifies whether to enable the inner congestion control function.

The inner congestion control function can be used to monitor the BSC load in real time. Thus, the calls that exceed the processing capability of the BSC can be prevented from entering the system and from enabling the BSC to be overloaded. Therefore, the inner congestion can be prevented.

GUI Value Range

[No,Yes]

Default Value

No

Unit

None

Step

-

Modification Rules

None

Configuration Policy

When this parameter is set to Yes, the BSC calculates the current flow control levels of the BSC based on the message packets load and the CPU usage. In addition, the BSC disables some minor services based on different flow control levels. For example, the BSC reports flow control alarms, disables part of the BSC functions, and discards the access messages, paging messages, and incoming BSC handover messages.

Recommended Value

No

Parameter Relationship

Each time the BSC assigns a message packet, the number of message units in the assigned message packets is added to the filter queue. The length of the filter queue is determined by the value of Inner Flow Control Window Size.

If the average load of the BSC exceeds the value of Inner Flow Control Discard Begin Threshold, the access messages, paging messages, and incoming BSC handover messages are discarded proportionally. If the average load of the BSC exceeds the value of Inner Flow Control Discard All Threshold, all the access messages, paging messages, and incoming BSC handover messages are discarded.
