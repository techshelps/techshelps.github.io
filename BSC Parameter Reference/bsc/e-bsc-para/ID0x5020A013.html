Clock Server IP

Parameter Name

Clock Server IP

Description

This parameter specifies the IP address of the IP clock server. If the Abis interface is based on IP transmission, the BTS cannot obtain the clock from the Ethernet lines. To reduce the costs and difficulties in running the BTS, the Abis interface clock is based on IP CLOCK scheme. The IP CLOCK system can be divided into two parts: the server and the client. The server is independent from the BSC6000, and can be also placed inside the BSC6000 cabinet. The clock message packets are sent from the server, transmitted in the network, and received by the client together with the network transmission delay and jitter. After the processes of frequency deviation extracting, filtering, and phase locking, the clock messages are used to synchronize the clock at the client and that at the server end.
