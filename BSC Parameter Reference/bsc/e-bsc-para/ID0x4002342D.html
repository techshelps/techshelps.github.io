Lower Threshold of Measurement Range of External Analog 1

Parameter Name

Lower Threshold of Measurement Range of External Analog 1

Description

This parameter specifies the lower limit of measurement range of an external analog sensor. When the measurement range of an external analog sensor is lower than the threshold, an alarm is generated if Enable Analog Parameter Alarm Reporting is set to Open.

The value of this parameter is a floating-point value set by the user according to actual requirements.

GUI Value Range

[-32768.00,32767.00]

Default Value

-40.00

Unit

None

Step

-

Modification Rules

None

Configuration Policy

The measurement range may be inaccurate if this parameter is set to a too great or too small value. It is recommended that the value range according to the type of the actually used external analog sersor.

Recommended Value

-40.00

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Equipment

Caution

The external analog sensor 1 has two types of input: voltage and current.

Lower Limit of Measurement Range of External Analog 1 should be smaller than or equal to Upper Limit of Measurement Range of External Analog 1.
