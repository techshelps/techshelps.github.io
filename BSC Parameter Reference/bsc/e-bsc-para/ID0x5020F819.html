Transmission Mode

Parameter Name

Transmission Mode

Description

This parameter specifies the transmission mode of the LAPD link.The signaling link is categorized into terrestrial transmission link, satellite transmission link, and terrestrial satellite transmission link.

The differences are the error correction mode and license control mode.

Terrestrial transmission mode indicates that the LAPD signaling link uses the non-satellite link. The basic error correction mode is used, and the link is not controlled by the license.

Satellite transmission mode indicates that the LAPD signaling link uses the satellite link. The preventive cyclic error correction mode is used, and the link is controlled by the license.

Terrestrial satellite transmission mode indicates that the LAPD signaling link uses the terrestrial satellite link. The preventive cyclic error correction mode is used, and the link is controlled by the license.
