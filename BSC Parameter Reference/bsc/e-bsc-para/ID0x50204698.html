HwIII UL Filter Adjust Factor

Parameter Name

HwIII UL Filter Adjust Factor

Description

This parameter specifies the filter adjustment factor for uplink power control. If this parameter is set to a large value, the filtered values become smooth, thus reducing the impact of poor measurement reports on the filtered values. If this parameter is set to a small value, the filter values are close to the actual ones and thus the power control speed is increased. It is recommended that the default value be used.
