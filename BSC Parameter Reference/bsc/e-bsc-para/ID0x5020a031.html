HDLC Congestion Bandwidth Threshold(%)

Parameter Name

HDLC Congestion Bandwidth Threshold(%)

Description

This parameter specifies the congestion bandwidth threshold of the HDLC. Congestion bandwidth of the HDLC = Congestion bandwidth threshold of the HDLC x Configured bandwidth of the HDLC.

GUI Value Range

[1,100]

Default Value

85

Unit

%

Step

-

Modification Rules

None

Configuration Policy

This parameter indicates the congestion bandwidth threshold of the HDLC. When the bandwidth used by the traffic reaches this threshold, the LDR activities are triggered. The LDR activities consist of the PS speed limitation, AMRC, TCHH preferred, and TCHF-TCHH handover. If this parameter is set to a small value, the time to reach the congestion threshold is shortened. It is recommended that the parameter be set to 85.

Recommended Value

85
