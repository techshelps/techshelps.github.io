Connect Site Index

Parameter Name

Connect Site Index

Description

This parameter specifies the index number of another BTS connected to the port of the site. This parameter is configured only when BTSs are cascaded.

This parameter is set to 65535 when the upper-level node is a BSC rather than a BTS.

GUI Value Range

[0,2047]

Default Value

65535
