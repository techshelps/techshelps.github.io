Encryption Algorithm

Parameter Name

Encryption Algorithm

Description

This parameter specifies the encryption algorithm supported on the BSS side.

The value of this parameter has eight bits. The eights bits (from the least significant bit to the most significant bit) specify whether to support the A5/0, A5/1, A5/2, A5/3, A5/4, A5/5, A5/6, and A5/7 encryption algorithms respectively. If a bit is set to 1, you can infer that the BSS supports the corresponding encryption algorithm. If a bit is 0, you can infer that the BSS does not support the corresponding encryption algorithm.

The eights bits cannot be all zeros and the least significant bit must be 1.

GUI Value Range

[A5/7,A5/6,A5/5,A5/4,A5/3,A5/2,A5/1,A5/0]

A5/0

Caution

The settings of the parameter on the BSS side must be consistent with those on the NSS side. The MAP procedure on the MSC side must specify whether the BSC encrypts data and specify the encryption algorithm used by the BSC. The encyrption algorithm used by the BSC is set on the basis of the data configuration of the BSC and the capability of the MS.
