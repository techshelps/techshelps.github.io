Battery Charge Current Limited coefficient

Parameter Name

Battery Charge Current Limited coefficient

Description

This parameter specifies the maximum current coefficient for battery charging. During battery charging, the maximum charge current can be 0.05 to 0.25 times the capacity of the battery cabinet, that is, Battery Charge Current Limited coefficient can be 0.05 to 0.25. Considering the battery service life and charge effect, this parameter is set to 0.15 by default.

For example, when the capacity of the battery cabinet is 200 Ah and Battery Charge Current Limited coefficient is set to 0.15 by default, the maximum charge current is 30 A.
