J1 Mode

Parameter Name

J1 Mode

Description

This parameter specifies the display mode of the TX J1 Byte and Expect RX J1 Byte.

If this parameter is set to Character String, some characters may not be entered. In this case, you can switch to the hexadecimal format. You can also use the hexadecimal format to enter some control characters.

The Character String mode can be switched to the Hexadecimal mode and the Hexadecimal mode can be switched to the Character String mode. You can switch from the Character String mode to the Hexadecimal mode, but the reverse operation is not allowed except printable characters are used in the Hexadecimal mode.
