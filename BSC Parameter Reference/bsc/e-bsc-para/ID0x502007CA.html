Max BTS Num of Parallel Software Load

Parameter Name

Max BTS Num of Parallel Software Load

Description

This parameter specifies the maximum number of BTSs that download software simultaneously from the OMU under one BSC.

GUI Value Range

[50,200]

Default Value

100

Unit

None

Step

-

Modification Rules

None

Configuration Policy

This parameter should not be modified.A maximum of 20 BTS software versions can be downloaded simultaneously.

Recommended Value

100

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Equipment

Caution

1. In the earlier versions of M2000V200R006C01B060SPC006 and M2000V200R008C01B060SPC003, a maximum of 50 BTSs can download software simultaneously under one BSC through the M2000 client. In M2000V200R006C01B060SPC006 and M2000V200R008C01B060SPC003 and their later versions, a maximum of 100 BTSs can download software simultaneously under one BSC through the M2000 client. Therefore, when the BTS software is downloaded through the M2000 client, the maximum number of BTSs downloading software should take the minimum value of the parameter Max BTS Num of Parallel Software Load and the maximum number of BTSs supported by the M2000 client.

2. If more than 150 BTSs download software simultaneously, the CPU usage of the OMU increases. In this case, the services are likely to be affected.
