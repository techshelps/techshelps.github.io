Abnormal Voice MOS Threshold

Parameter Name

Abnormal Voice MOS Threshold

Description

This parameter specifies the MOS threshold when the voice is abnormal.

The BTS periodically calculates the voice MOS and sends it to the BSC through a self-defined measurement report. Based on this threshold, the BSC determines whether the voice within the period is normal. If the MOS is smaller than the threshold, the BSC concludes that the voice is abnormal and records the information in the CHR of the corresponding call.

GUI Value Range

[0,250]

Default Value

100

Unit

None

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to a too great value, it is likely that the normal voice is regarded as abnormal, which is of no benefit for locating faults.

If this parameter is set to a too small value, the abnormal voice is not likely to be detected.

Recommended Value

100

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Equipment

Caution

To eliminate the impact from the decimal digits, the unit of this parameter is MOS x 50.
