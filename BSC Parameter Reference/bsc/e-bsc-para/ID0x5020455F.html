MAX Consecutive HO Times

Parameter Name

MAX Consecutive HO Times

Description

This parameter determines the maximum number of consecutive intra-cell handovers allowed.

If the interval of two continuous intra-cell handovers is shorter than a specified threshold, the two handovers are regarded as consecutive handovers. If multiple consecutive intra-cell handovers occur, the intra-cell handover is forbidden for a period.

GUI Value Range

[1,20]

Default Value

3

Unit

Times

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to a too small value, the intra-cell handover may not be timely; if this parameter is set to a too great value, the system resources may be wasted when intra-cell handovers occur frequently.

Recommended Value

3

Parameter Relationship

None
