Receive Mode

Parameter Name

Receive Mode

Description

This parameter specifies the RF receive mode of the DTRU.

The RF receive mode can be Not Support, Independent Receiver, Dividing Receiver, Four Diversity Receiver, or Main Diversity.

The BTS3012, BTS3012AE, BTS3012II, BTS3006C, and BTS3002E do not support Main Diversity.

The DBS3900 GSM and BTS3900 GSM support Four Diversity Receiver and Main Diversity.

GUI Value Range
