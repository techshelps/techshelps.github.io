Inner Flow Control Discard All Threshold

Parameter Name

Inner Flow Control Discard All Threshold

Description

If the average load of the BSC exceeds the value of this parameter, the BSC starts to discard all the channel request messages, paging messages, and incoming BSC handover messages.

This parameter is set to 3000. When the BSC performs inner flow control and the number of message units currently assigned by the BSC is larger than 3000, the BSC begins to discard all the access messages, paging messages, and incoming BSC handover messages.

GUI Value Range

[200,5000]

Default Value

3000

Unit

None

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to a too small value, the unnecessary inner flow control occurs when the traffic volume does not reach the processing capability of the BSC.

If this parameter is set to a too great value, the inner flow control does not work.

Recommended Value

3000

Parameter Relationship

This parameter and Inner Flow Control Discard Begin Threshold determine the discarding ratio of messages. The formula for calculating the discarding ratio is as follows:

(Average Message Units Assigned by the BSC - Inner Flow Control Discard Begin Threshold) /(Inner Flow Control Discard All Threshold - Inner Flow Control Discard Begin Threshold) x 100%
