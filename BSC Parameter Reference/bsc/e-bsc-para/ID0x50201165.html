Alarm Voltage Definition 1(Hex)

Parameter Name

Alarm Voltage Definition 1(Hex)

Description

Each bit in the alarm voltage definition field indicates whether a Boolean alarm is triggered in high level or in low level.1 indicates that the alarm is triggered in high level and 0 indicates that the alarm is triggered in low level.

GUI Value Range

[0,FF]

Default Value

0
