PA Intelligent Adjust Voltage

Parameter Name

PA Intelligent Adjust Voltage

Description

This parameter determines whether the intelligent voltage adjustment on the multi-transceiver is supported. With this function, the transmit power of the transceiver is automatically adjusted according to the service requirement on power.

GUI Value Range

[No, Yes]

Default Value

No

Unit

None

Step

-

Modification Rules

None

Configuration Policy

None

Recommended Value

None

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Equipment

Caution

This switch is controlled by the item "Multi-Carrier Intelligent Voltage Function" of the BSC license. If the BSC license does not allow the use of the "Multi-Carrier Intelligent Voltage Function", PA Intelligent Adjust Voltage should not be set to Yes. This parameter applies only to 3900 series base stations. For other base stations, this parameter is not displayed and cannot be configured.
