Parameter Name

Timer "Sending SIB"(ms)

Description

This parameter specifies the length of the timer for the MTP2 link to control local congestion.

This timer is triggered when services are available on the MTP2 link.When the timer expires, the MTP2 congestion information is reported to the MTP3 layer and a system information block (SIB) is sent to the MSC.

This parameter only applies to the 64 kbit/s links. For the 2 Mbit/s links, the value of this timer is always 100.
