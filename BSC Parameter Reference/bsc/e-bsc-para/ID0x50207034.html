Special Alarm Switch Enabled

Parameter Name

Special Alarm Switch Enabled

Description

This parameter specifies whether to enable the special Boolean alarms. There are totally four types of alarms: water alarm, smoke alarm, infrared alarm, and door status alarm, which correspond to bit 0, bit 1, bit 2, and bit 3 respectively. By default, all the Boolean alarms can be enabled. The bit 0 indicates enable and the bit 1 indicates disable. The parameter values are as follows:

Gate Magnetism(Disabled), Infrared(Disabled), Smog(Disabled), Water Immersed(Disabled)

GUI Value Range

[Gate Magnetism(Disabled), Infrared(Disabled), Smog(Disabled), Water Immersed(Disabled)]
