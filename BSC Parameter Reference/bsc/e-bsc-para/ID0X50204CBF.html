Location Group Power Switch

Parameter Name

Location Group Power Switch

Description

This parameter specifies whether the TRX power amplifier of the current location or of all the locations is turned on.

GUI Value Range

[Only Current Working Location Trxes be Turned On,All Location Trxes be Turned On]

Default Value

All Location Trxes be Turned On

Unit

None

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to All Location Trxes be Turned On, the TRX power amplifier of all the location groups is turned on. If this parameter is set to Only Current Working Location Trxes be Turned On, only the RX power amplifier of the current location group is turned on.

Recommended Value

None

Parameter Relationship

Location Group No

BSC Feature

None

Parameter Class

Parameters About Equipment

Caution

This parameter is valid only in the DBS3900 and DBS3036.
