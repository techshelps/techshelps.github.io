Cell Extension Type

Parameter Name

Cell Extension Type

Description

This parameter specifies whether a cell is an extension cell and specifies how to implement the extended cell.

A double-timeslot extension cell regards the additional TDMA frame as access delay. Theoretically, TA equals 219, that is, a delay of about 120 kilometers.

Caution

If a cell is a double-timeslot extension cell, the Max TA should be set to 219. Otherwise, call failures may occur though there are signals.

For the double-timeslot TRX of the double-timeslot extension cell, only the even timeslot is functional. For the normal TRX of the double-timeslot extension cell, the eight timeslots are functional.

The RF hopping configuration of the double-timeslot extension cell is similar to the RF hopping configuration of a normal cell. If a cell is a double-timeslot extension cell and the FH mode is baseband FH, the TRXs in one MA group must be of the same type: double-timeslot TRX or normal TRX. Otherwise, an error may occur.
