TDD Cell Threshold

Parameter Name

TDD Cell Threshold

Description

This parameter specifies the threshold for determining whether the MR about a TDD cell is valid.

The measurement report is valid if the receive level of the TDD cell in the measurement report is greater than the value of this parameter.

After the valid measurement report is filtered, the TDD cell joins in the cell prioritization.
