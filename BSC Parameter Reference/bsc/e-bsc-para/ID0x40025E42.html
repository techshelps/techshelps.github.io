J2 Byte Alarm Report Switch

Parameter Name

J2 Byte Alarm Report Switch

Description

This parameter specifies whether to report the Trace Flag Mismatch on the Lower-Order Path of the Optical Port (LPTIM) alarm if inconsistency occurs on TX J2 Byte or Expect RX J2 Byte on the two sides during optical port interconnection.

GUI Value Range

Modification Rules

None

Configuration Policy

In some special cases, during the interconnection with optical equipment from other vendors, the communication between the equipment on the two sides is normal though TX J2 Byte and Expect RX J2 Byte on the two sides do not match. In this case, this parameter can be set to No, not to report the Trace Flag Mismatch on the Lower-Order Path of the Optical Port (LPTIM) alarm.
