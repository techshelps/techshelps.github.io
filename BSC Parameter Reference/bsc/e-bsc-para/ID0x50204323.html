BS_AG_BLKS_RES

Parameter Name

BS_AG_BLKS_RES

Description

This parameter specifies the number of CCCH blocks reserved for the AGCH. After the CCCH is configured, this parameter actually indicates the CCCH usage for AGCH and PCH.

This parameter affects the paging response time of an MS and the system performance.

GUI Value Range

[0,7]

Default Value

2

Unit

Blocks

Step

-

Modification Rules

None

Configuration Policy

None

Recommended Value

2

Parameter Relationship

None

BSC Feature

Channel Management

Parameter Class

Parameters About Radio Resources

Caution

As specified in the protocol, the value of this parameter must be greater than 0 in the following cases:

System information is transmitted on the extended BCH.

The system is configured with CBCHs.

The GSMR system is configured with NCHs.
