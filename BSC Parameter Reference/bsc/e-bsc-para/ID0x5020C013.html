Alpha Parameter

Parameter Name

Alpha Parameter

Description

This parameter is used for the open-loop power control.

The MS uses the Alpha parameter to calculate the output power of the uplink PDCH, namely, PCH.

When the MS uses the GPRS dynamic power control, this parameter determines the reduced level of the MS transmit power mapping to the path loss.
