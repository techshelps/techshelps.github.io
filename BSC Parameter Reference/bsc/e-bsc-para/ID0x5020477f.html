IBCA Dynamic Measure Neighbor Cell Allowed

Parameter Name

IBCA Dynamic Measure Neighbor Cell Allowed

Description

This parameter determines whether to ignore the interference of the neighbor cells whose information is not carried in the MR when you are configuring the neighbor cells of the serving cell. If the interference of the neighbor cells cannot be ignored, set the value to Yes; if the interference of the neighbor cells can be ignored, set the value to No. Generally, the neighbor cells with co-channel interference are set as dynamic measurement cells. For the layer 2 neighbor cells with co-channel interference, the value must be set to Yes.This parameter specifies whether to measure the neighboring cells with IBCA Dynamic Measure Neighbor Cell Flag set to Yes.

If this parameter is set to Yes, the Directed Retry parameter must not be set to Yes because the dynamic neighboring-cell measurement and the directed retry are mutually exclusive.
