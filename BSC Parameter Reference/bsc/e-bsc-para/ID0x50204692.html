Path Loss of Different Frequency Band(0.1dB)

Parameter Name

Path Loss of Different Frequency Band(0.1dB)

Description

This parameter determines the pass loss difference between frequency bands.

GUI Value Range

[0,255]

Default Value

79

Unit

None

Step

0.1dB

Modification Rules

None

Configuration Policy

If the main-BCCH frequency of cell where the current channel is located is low, while the frequency of the frequency band where the current channel is located is high, the transmit power of the MS and BTS is incremented by the Path Loss of Different Frequency Band based on power prediction.

If the frequency of the frequency band where the current channel is located is low, while the main-BCCH frequency of cell where the current channel is located is high, the transmit power of the MS and BTS is subtracted by the Path Loss of Different Frequency Band based on power prediction.

Recommended Value

None

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Radio Resources

Caution

This parameter is invalid for multiple-band cells only. For single-band cells, the value of this parameter is 0.
