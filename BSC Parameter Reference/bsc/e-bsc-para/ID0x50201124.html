Reference Clock Source Type

Parameter Name

Reference Clock Source Type

Description

This parameter specifies the work mode of the BTS clock.

In internal clock working mode, the BTS cannot trace the external clock. The internal clock signal of high precision is in free-run working mode.

In trace BSC clock mode, the clock signals of high precision within the BTS traces the clock signals transmitted on the BSC side.

In external sync. clock mode, the clock signals of high precision within the BTS traces the synchronous clock signals imported from outside.

In IP clock working mode, the clock synchronization is achieved through the handshake between the IP network equipment and the IP clock server.

In trace transmission clock working mode, when the BTS upgrades its transmission mode from E1 transmission to IP transmission, you can obtain the clock from the E1 cable in the transmission network given that no IP clock is available.

In trace GPS clock working mode, the BTS obtains the clock from the GPS card on the DGPS board, and thus the clock synchronization is fulfilled.
