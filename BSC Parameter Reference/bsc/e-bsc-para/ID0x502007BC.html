Reserved Resource Type

Parameter Name

Reserved Resource Type

Description

This parameter specifies the reserved resource type, 0: not reserved.

Bit 0 indicates the reserved Ater resources.

Bit 1 indicates the reserved DSP resources in the TC subrack.

Bit 2 indicates the reserved DSP resources in the BM subrack.

Other bits are reserved at present.

GUI Value Range

[Reserve DSP(BM) Resource, Reserve DSP(TC) Resource, Reserve Ater Resource]

Default Value

Reserve Ater Resource

Unit

None

-

Configuration Policy

Recommended Value

Reserve Ater Resource

None

Parameter Class

Parameters About Equipment

Caution

In A over IP or BM/TC combined mode, this parameter can only be set to Reserve DSP(BM) Resource. In BM/TC separated mode, this parameter can be set to Reserve DSP(BM) Resource, Reserve DSP(TC) Resource, or Reserve Ater Resource.
