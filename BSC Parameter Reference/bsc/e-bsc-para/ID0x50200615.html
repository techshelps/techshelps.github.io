Cell Broadcast No Msg Timer(s)

Parameter Name

Cell Broadcast No Msg Timer(s)

Description

At present, the BSC communicates with the CBC based on the TCP/IP protocol, so the BSC may not detect that the link to the CBC is disrupted.

To resolve this problem, Huawei defines this parameter. If the BSC does not receive a message from the CBC within this period of time determined by this parameter, the BSC regards the link to the CBC as being disrupted and begins to re-establish the link.
