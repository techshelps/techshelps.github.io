Backup Power Saving Policy

Parameter Name

Backup Power Saving Policy

Description

This parameter specifies the BTS backup power saving policy. If this parameter is set to Coverage Preferred, the BTS shuts down some TRXs and then reduces the power of the main BCCH TRX after a period of time. If this parameter is set to Capacity Preferred, the BTS reduces the power of the main BCCH TRX and then shuts down some TRXs after a period of time. If this parameter is set to Backup Power Preferred, the BTS shuts down some TRXs and reduces the power of the main BCCH TRX simultaneously.

GUI Value Range

[Coverage Preferred,Capacity Preferred,Backup Power Preferred]

Default Value

Coverage Preferred
