Configure backup OML

Parameter Name

Configure backup OML

Description

This parameter specifies whether to configure the backup OML.The OML backup function is enabled when the BTS is configured. An OML is configured in timeslot 31 of port 0 and port 1 respectively.The links are established on the two ports in turn after the BTS is reset.Once the OML is established on a port, it is always used unless the BTS is reset or the OML is disconnected. If the established OML is disconnected, the BTS automatically switches the OML over to another port to re-establish the OML.

Caution

(1) The double-transceiver BTSs of the BTS3012 series and the later BTS type such as the 3900 series base station support the OML backup function. The BTS3X series and BTS2X series do not support this function. (2) The OML backup function is supported only in TDM networking mode and HDLC networking mode and not supported in HDLC HUB networking mode (including the leaf BTS) and IP networking mode. (3) The OML backup function, the ring topology, and the Abis bypass function are mutually exclusive. (4) Only the OML backup function between port 0 and port 1 in the main cabinet group is supported. The OML backup function between other ports in the main cabinet group or between the main cabinet group and the extension cabinet is not supported. (5) The two OMLs in backup mode between the BSC and the BTS cannot be configured on the same E1/T1 (including the E1/T1 of the upper-level BTS). That is, if only one E1/T1 link of the upper-level BTS is connected to the BSC, and the OML backup function is required on its lower-level BTS, one slave link needs to be added to the upper-level BTS, or one slave link needs to be added to the lower-level BTS and directly connected to the BSC.
