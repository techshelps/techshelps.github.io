Bad Quality Disconnection Threshold

Parameter Name

Bad Quality Disconnection Threshold

Description

If the value of the uplink quality or downlink quality in the measurement report is greater than the value of this parameter over a period of time before the subscriber hooks on, the BSC considers the reason for the on-hook as bad quality and then measures the number of on-hooks due to bad quality.

GUI Value Range

[0,70]

55
