Enable External Switch 14 Alarm Reporting

Parameter Name

Enable External Switch 14 Alarm Reporting

Description

This parameter specifies whether to report an external switch alarm. When this parameter is set to Open, an external switch alarm is reported if the external switch sensor detects abnormal signals.

GUI Value Range

[Close,Open]

Default Value

Open

Unit

None

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to Open, a PDB output switch sensor alarm is reported.

If this parameter is set to Close, a PDB output switch sensor alarm is not reported.

Recommended Value

Open
