Auto Download Active Function Switch

Parameter Name

Auto Download Active Function Switch

Description

This parameter specifies whether to enable the BTS software automatic loading and activation function.If this parameter is set to Yes and Work Mode for Auto Downloading and Activation is set to BTS Software Auto Downloading and Activation, the BTS software is automatically loaded when the BTS is initialized or a BTS board is added. If Work Mode for Auto Downloading and Activation is set to TMU Software Auto Activation, the TMU software is automatically activated when the BTS is initialized or it is time for the BSC to perform the resource check.

If this parameter is set to Yes, the BTS software fails to be forcibly loaded through the LMT, MML client, or M2000. If this parameter is set to No, the BTS software can be successfully loaded and activated through the LMT, MML client, or M2000.

GUI Value Range

[No,Yes]

No

Configuration Policy

If the parameter is set to "YES", the BSC automatically activates the TMU version actually configured when the TMU version is rolled back.
