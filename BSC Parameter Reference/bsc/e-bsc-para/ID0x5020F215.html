Signaling Link Set Mask

Parameter Name

Signaling Link Set Mask

Description

This parameter specifies the way for the BSC6000 to implement the load sharing of signaling services between multiple MTP link sets of the DPC.

When only one link set is available between the BSC6000 and the DPC, Signaling Link Set Mask can be set to any value.

When two signaling link sets are available between the BSC6000 and the DPC, the number of 1s in Signaling Link Set Mask must be greater than or equal to 1. For example, the parameter can be set to B0001, B1100, or B1111.

When three or four signaling link sets are available between the BSC6000 and the DPC, the number of 1s in Signaling Link Set Mask must be greater than or equal to 2. For example, the parameter can be set to B0011, B1100, or B1111.

When five to eight signaling link sets are available between the BSC6000 and the DPC, the number of 1s in Signaling Link Set Mask must be greater than or equal to 3. For example, the parameter can be set to B1011, B1110, or B1111.

When 9 to 16 signaling link sets are available between the BSC6000 and the DPC, Signaling Link Set Mask must be set to B1111.

To avoid the overlapping of bits, the operator cannot set the two parameters to the same value. That is, the digit 1 in Signaling Link Set Mask and that in Signaling Link Mask cannot be in the same position. For example, if Signaling Link Set Mask is set to B0110, the parameter Signaling Link Mask can be set to B1001.
