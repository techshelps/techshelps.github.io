BSC Default DPC

Parameter Name

BSC Default DPC

Description

If this parameter is set to Yes, all the calls in the BSC access the MSC identified by this DPC if the MSC pool function is disabled. If this parameter is set to No, all the calls in the BSC do not access the MSC identified by this DPC if the MSC pool function is disabled. This parameter is invalid if the MSC pool function is enabled.

GUI Value Range
