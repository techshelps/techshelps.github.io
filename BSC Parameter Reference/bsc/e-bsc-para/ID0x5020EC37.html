Timer "Emergency Proving"(ms)

Parameter Name

Timer "Emergency Proving"(ms)

Description

This parameter specifies the length of the timer for the MTP2 link to control the start of the emergency alignment to wait for the response from the MSC.

When the MTP2 link is in the aligned state, this timer is triggered to wait for the emergency alignment response from the MSC. When the timer expires, the link is set to the service disrupted state and the initial alignment starts again.

This parameter only applies to the 64 kbit/s links. For the 2 Mbit/s links, the value of this timer is always 500.
