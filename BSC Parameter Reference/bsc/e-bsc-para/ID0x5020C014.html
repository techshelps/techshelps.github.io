Initial Power Level

Parameter Name

Initial Power Level

Description

This parameter specifies the initial power level.

This parameter determines the expected receive signal strength on the BTS when the MS uses the GPRS dynamic power control.

GUI Value Range

[0,31]

Default Value

12

Unit

dB

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to a higher value, the output power of the MS decreases; if this parameter is set to a lower value, the output power of the MS increases.

Recommended Value

12

Parameter Relationship

None

BSC Feature

GPRS&EGPRS
