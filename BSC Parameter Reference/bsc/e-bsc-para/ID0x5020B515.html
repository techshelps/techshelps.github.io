Subnetwork Protocol Type

Parameter Name

Subnetwork Protocol Type

Description

This parameter specifies the type of the subnet protocol. Two types are Gb over FR and Gb Over IP.

Gb Over FR: The Gb interface uses the Fr transmission mode. The data link layer on the Gb interface supports frame relay.

The SGSN and the BSS are connected through the virtual circuits of frame relay. The LLC PDUs from different users are multiplexed onto these virtual circuits.

The user data and signaling on the Gb interface can be transmitted through frame relay.

Gb Over IP: The Gb interface uses the IP transmission mode.

The FR connection is replaced by the IP network. The user data and signaling are transmitted through UDP packets.

The protocol stacks and functions of the NS layer and upper layers in Gb Over IP mode are the same as those in Gb Over FR mode.
