Diesel Engine Configure Enable

Parameter Name

Diesel Engine Configure Enable

Description

This parameter specifies whether to allow the setting of diesel engine parameters. If the parameter is set to Yes, the diesel engine parameters can be set.

GUI Value Range

[No,Yes]

Default Value

No

Caution
