Subboard Slot No.

Parameter Name

Subboard Slot No.

Description

This parameter specifies the number of a sub-board.

GUI Value Range

[0,3]

Default Value

None

Unit

None

Step

-

Modification Rules

None

Configuration Policy

When the GXPUT/GXPUM is configured, two sub-boards are automatically configured. The GSCU must exist before a GXPUT/GXPUM is added because the GSCU is required to perform arbitration.

When the GDPUC is configured, two DSP sub-boards are automatically configured in subslots 0 and 1.

When the GEIU is configured, one E1/T1 sub-board is automatically configured in subslot 0.

When the GOIU is configured, one optical sub-board is automatically configured in subslot 0.
