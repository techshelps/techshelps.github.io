Cell Broadcast Handshake Timer(s)

Parameter Name

Cell Broadcast Handshake Timer(s)

Description

This parameter specifies the interval at which the BSC sends handshake messages to the CBC.

GUI Value Range

[1,5]

Default Value

3

Unit

Seconds

Step

-

Modification Rules

None

Configuration Policy

None

Recommended Value

3

Parameter Relationship

If CB Interface Handshake is set to Yes, the BSC sends a handshake message to the CBC after the Cell Broadcast Handshake Timer expires to determine whether the communication with the CBC is normal.
