IntraCell Non-AMR TCHH-TCHF Ho ATCB Threshold

Parameter Name

IntraCell Non-AMR TCHH-TCHF Ho ATCB Threshold

Description

If the non-AMR full-rate speech ATCB is smaller than or equal to this threshold, the MS meets the conditions for non-AMR TCHF-TCHH handover.

GUI Value Range

[0,255]

Default Value

14

Unit

None

Step

-

Modification Rules

None

Configuration Policy

None

Recommended Value

14

Parameter Relationship

When the ATCB of the MS is smaller than the ATCB threshold, the MS meets the conditions for TCHH-TCHF handover. This parameter, together with the path loss threshold and receive quality threshold, determines the TCHH-TCHF handover.

BSC Feature

None

Parameter Class

Parameters About Radio Resources

Caution

This parameter can be configured only when HO Control Switch is set to HO Algorithm II; this parameter is invalid when HO Control Switch is set to HO Algorithm I.
