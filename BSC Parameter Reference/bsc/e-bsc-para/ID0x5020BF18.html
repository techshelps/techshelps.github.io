Transmission Delay of POC Service

Parameter Name

Transmission Delay of POC Service

Description

This parameter specifies the maximum transmission delay of the POC services.

The POC services have a strict requirement on the transmission delay. The network should support the detection of the POC service type and take measures to reduce the transmission delay to meet the requirement of the POC services.

If the service type carried in the received message is POC, the Transfer Delay in the ABQP must be lower than the value of this parameter.

POC:push to talk over cellular.
