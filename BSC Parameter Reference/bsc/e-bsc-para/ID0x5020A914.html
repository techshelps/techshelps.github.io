Alive Timer

Parameter Name

Alive Timer

Description

This parameter specifies the interval between two alive messages sent by the BSC. The BSC periodically sends an alive message to the SGSN. If the SGSN does not respond, the BSC resends the message.

For example, value 1 indicates that the timer value is one second, and value 255 indicates that the timer value is 255 seconds.

GUI Value Range

[1,255]

Default Value

3

Unit

Seconds

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to an excessive value, the alive message is not sent in time.

If this parameter is set to a modest value, too many alive messages are sent in a short period of time. The evaluation of the SGSN may be inaccurate when the SGSN is busy.

Recommended Value

3

None
