Two-way diversity gain

Parameter Name

Two-way diversity gain

Description

This parameter is one of the items in the formula for calculating the UL and DL balance performance measurement.

GUI Value Range

[0,800]

Default Value

350

Unit

dB

Step

-

Modification Rules

None

Configuration Policy

None

Recommended Value

350

Parameter Relationship

If the receive mode of a TRX is 4-way receive diversity, the diversity gain is Four-way diversity gain.

Otherwise, the BSC traverses all the CDUs configured for the BTS where the TRX is located. That is, the BSC checks all the TX and RX relations of all the TRXs on uplink tributaries 0 and 1 of each CDU to determine the diversity gain mode.

If a TRX number occurs more than once in the TRX Transmit and Receive Relationship table about uplink tributaries 0 and 1, the diversity gain mode is 2-way diversity gain. Otherwise, the diversity gain mode is no diversity gain.

1. If a TRX is not configured with a TMA, the formula for calculating the UL and DL balance performance measurement is as follows:

UL and DL balance performance measurement = DL receive level - UL receive level - (MS sensitivity - BTS sensitivity) - diversity gain

2. If a TRX is configured with a TMA, the formula for calculating the UL and DL balance performance measurement is as follows:

UL and DL balance performance measurement = DL receive level - UL receive level - (MS sensitivity - BTS sensitivity) - diversity gain - Loss and sensitivity

The diversity gain described in the preceding formulas varies with the RF receive mode.

If the RF receive mode is set to 4-way receive diversity, the diversity gain equals Four-way diversity gain.

If the RF receive mode is set to 2-way receive diversity, the diversity gain equals Two-way diversity gain.
