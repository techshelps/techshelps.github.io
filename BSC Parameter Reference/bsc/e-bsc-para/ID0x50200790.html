Uplink and downlink balance traffic measurement optimization switch

Parameter Name

Uplink and downlink balance traffic measurement optimization switch

Description

This parameter specifies whether to optimize the performance measurement about the uplink and downlink balance.

Telecom operators determine whether to adjust the related parameters to improve network quality based on the measurement results about the uplink and downlink balance on TCHs.

Parameter Relationship

The UL and DL balance performance measurement is calculated according to the following formula:

1. If a TRX is not configured with the tower mounted amplifier (TMA), the transmission loss of signals on the uplink is the same as that on the downlink. In this case, the formula for calculating the UL and DL balance performance measurement is as follows:

UL and DL balance performance measurement = DL receive level - UL receive level - (MS sensitivity - BTS sensitivity - diversity gain)

2. If a TRX is configured with the TMA, the reference point of the UL sensitivity and UL receive level moves up to the antenna. The path loss from the antenna to the top of the cabinet is offset and the UL sensitivity is improved. The Attenuation Factor configured for the CDU, however, counters the TMA gain. Therefore, the UL receive level remains unchanged. In this case, the formula for calculating the UL and DL balance performance measurement is as follows:

UL and DL balance performance measurement = DL receive level - UL receive level - (MS sensitivity - BTS sensitivity) - diversity gain - Loss and sensitivity
