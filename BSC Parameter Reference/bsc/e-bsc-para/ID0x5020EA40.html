IntraCell TCHH-TCHF Pathloss Offset Overlay

Parameter Name

IntraCell TCHH-TCHF Pathloss Offset Overlay

Description

This parameter specifies the difference of pass loss between the overlaid subcell and the underlaid subcell.

This parameter, along with the IntraCell AMR TCHH-TCHF Ho Pathloss Threshold, determines whether a condition for deciding a TCHH-TCHF handover is met.

GUI Value Range

[0,255]

Default Value

10

Unit

None

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to a too great value, only an MS close to the center of the overlaid subcell can be assigned TCHHs. Thus, timeslots are wasted, and network congestion cannot be relieved in time.

If this parameter is set to a too small value, an MS of bad quality on the edge of the overlaid subcell cannot be handed over from TCHH to TCHF. Thus, the quality of the call cannot be improved.

Recommended Value

10

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Radio Resources

Caution

This parameter is available only when HO Control Switch is set to HO Algorithm II. This parameter is invalid when the HO Control Switch is set to HO Algorithm I.
