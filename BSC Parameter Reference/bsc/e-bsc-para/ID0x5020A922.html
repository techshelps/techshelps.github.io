Parameter Name

T6

Description

This timer is started when the BSC sends a request for downloading the BSS PFC packet procedure to the SGSN to obtain QoS parameters.

After the BSC receives the CREATE_BSS_PFC message, the timer is stopped.

When the timer expires, the BSC resends the request for downloading the BSS PFC packet procedure.

For example, value 1000 indicates that the timer value is 1000 ms, and value 10000 indicates that the timer value is 10000 ms.

GUI Value Range

[1000,10000]

Default Value

3000

Unit

Milliseconds

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to an excessive value, the BSC cannot obtain the QoS parameters in real time.

If this parameter is set to a modest value, it is possible that the SGSN sends a response message, but the BSC does not receive it. In this case, the BSC resends the request.

Recommended Value

3000
