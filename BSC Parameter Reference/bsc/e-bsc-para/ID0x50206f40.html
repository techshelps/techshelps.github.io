Alarm Disable Configuration

Parameter Name

Alarm Disable Configuration

Description

This parameter specifies whether to enable or disable one route of alarm. The parameter is expressed in four bits, each of which indicates the enabling or disabling of one route of alarm. Bit 0 indicates that the alarm is enabled whereas bit 1 indicates that the alarm is disabled. The DTCU supports four routes of alarms: route 1 Boolean alarm (disabled), route 2 Boolean alarm (disabled), route 3 Boolean alarm (disabled), and route 4 Boolean alarm (disabled).

Route 1 Boolean alarm is the door status alarm.

Route 2 Boolean alarm is the AC lightning arrester alarm.

Route 3 Boolean alarm is the DC lightning arrester alarm.

Route 4 Boolean alarm is reserved.

GUI Value Range
