CM3 Modification in Paging Response

Parameter Name

CM3 Modification in Paging Response

Description

This parameter specifies whether to set CM3 to 0 forcibly in the Mobile Station Classmark information element of the PAGING RESPONSE message.

GUI Value Range

[0,3]

Default Value

3

Unit

None

Step

-

Modification Rules

None

Configuration Policy

1. If this parameter is set 1, CM3 is modified to 0 forcibly.

2. If this parameter is set to 2, the BSC searches for neighbor cells on the 1800 MHz frequency band in the BA1 and BA2 tables. If there is no neighbor cell on the 1800 MHz frequency band, modify CM3 to 0.

3. If there is a neighbor cell on the 1800 MHz frequency band or if this parameter is set to 0 or 3, CM3 is not modified.

Recommended Value

3
