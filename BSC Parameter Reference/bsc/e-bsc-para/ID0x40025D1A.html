High Order Path 1 Expect RX J1 Byte

Parameter Name

High Order Path 1 Expect RX J1 Byte

Description

This parameter specifies the J1 byte that is expected to be received. J1 byte, also called the trace byte of the high-order path on the optical port, is one of the optical port interconnection parameters. The local end receives the J1 byte continuously from the peer end.

When the Expect RX J1 Byte is equal to the Received J1 Byte, it indicates that the TX and RX ends are connected continuously. If the TX and RX ends are not connected continuously, the Trace Flag Mismatch on the High-Order Path of the Optical Port (HPTIM) alarm is reported.

GUI Value Range

[1 Byte,16 Bytes,64 Bytes]

16 Bytes
