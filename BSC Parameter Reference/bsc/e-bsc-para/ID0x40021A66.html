Channel 1 -48V Low Alarm Threshold

Parameter Name

Channel 1 -48V Low Alarm Threshold

Description

This parameter specifies the threshold of the supply voltage alarm for the first route -48 V power. When the Channel 1 -48V Alarm Mask is set to Open, a supply voltage alarm is reported if the supply voltage of the first route -48 V power is lower than the threshold.

GUI Value Range

[-40,-42]
