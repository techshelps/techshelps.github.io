Line Code

Parameter Name

Line Code

Description

This parameter specifies the line coding scheme for line transmission.

The E1 link codes are classified into HDB3 and AMI. Generally, HDB3 is used. High Density Bipolar 3 (HDB3) has the DC characteristics with a transmission distance of about several kilometers.

The T1 link codes are classified into B8ZS and AMI. Generally, B8ZS is used because only B8ZS can avoid the loss of clock signals.

GUI Value Range

[B8ZS,HDB3]

Default Value

HDB3

Unit

None

Step

-

Modification Rules

None

Configuration Policy

In the case of an interconnection with network elements such as MSC and BTS or in the case of an interconnection on the Ater interface, the line codes must be the same on the two sides to ensure that all the E1/T1 links are functional.

Recommended Value

HDB3

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Equipment

Caution

When the work mode is E1, only HDB3 and AMI are supported.

When the work mode is T1, only B8ZS and AMI are supported.
