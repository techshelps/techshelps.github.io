Send System Message 10 Allow

Parameter Name

Send System Message 10 Allow

Description

This parameter specifies whether sending the system information type 10 message on the VGCS/VBS TCH is allowed.

GUI Value Range

[No,Yes]

Default Value

Yes

Unit

None

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to YES, the system information 10 is sent on the VGCS/VBS TCH to improve the cell reselection performance of the MS in group receive mode. The system information 10 contains only the information about the neighbor cells within the same group call area. This mechanism ensures that the MS reselects only the cell within the same group call area and avoids call drops.

Recommended Value

Yes
