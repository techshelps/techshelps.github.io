TCH/F Uplink DTX

Parameter Name

TCH/F Uplink DTX

Description

This parameter specifies whether to allow the MS to use the Discontinuous Transmission (DTX) function. For details, see GSM Rec. 05.08.

GUI Value Range

[May Use,Shall Use,Shall not Use]

Default Value

Shall Use

Unit

None

Step

-

Modification Rules

None

Configuration Policy

The DTX function allows a transmitter to stop power transmission in the case of no voice transfer. This function has the following benefits:

1. On the uplink: decreasing the power consumption of the MS and reducing system interference

2. On the downlink: decreasing power consumption of the BTS, reducing system interference, and reducing intermodulation inside the BTS

3. From the network perspective, the inter-frequency interference is reduced and the network quality is improved.

TCH/F Uplink DTX is not restricted by the MSC.It is allowed MS to use DTX when this parameter is set to " Shall Use". It is not allowed MS to use DTX when this parameter is set to " May Use" or " Shall not Use".

Recommended Value

Shall Use

Parameter Relationship

None

BSC Feature

Discontinuous Transmission and Discontinuous Reception

Parameter Class

Parameters About Radio Resources

Caution

The DTX function is implemented on the basis of the following basic functions: voice activity detection (VAD), silence descriptor (SID), and comfort noise.

The Huawei BSC can control whether to enable the downlink DTX function. For details, see the description of the DL DTX parameter.
