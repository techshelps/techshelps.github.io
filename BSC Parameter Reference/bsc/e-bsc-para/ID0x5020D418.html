Work Mode

Parameter Name

Work Mode

Description

This parameter indicates the work mode (strategy) of the clock source.

In Free Run mode, clock source 0 is set to the clock source for the GGCU.

The system generally uses Automatic mode.

The Manual mode is generally used for test purposes.

GUI Value Range

[CLIENT,SERVER]

Default Value

CLIENT

Caution

Specifying the priority level is not required for the Automatic mode.

Specifying the priority level is required for the Manual mode.

The priority level for the Free Run mode is 0.
