UTRAN FDD CELL BA2

Parameter Name

UTRAN FDD CELL BA2

Description

This parameter specifies the frequencies of the neighbor 3G cells of a cell. The frequencies of the added neighbor 3G cells are listed in 3G Cell BA Table 2 in sequence and are sent in the system information message.
