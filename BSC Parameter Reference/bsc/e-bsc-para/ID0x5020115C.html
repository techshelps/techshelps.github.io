T200(10ms)

Parameter Name

T200(10ms)

Description

This parameter specifies the timer of the LAPD protocol. The value of timer specifies the time of frame responded and confirmed by the command frame that the LAPD link waits to send.

GUI Value Range

[6,240]

Default Value

12

Unit

None

Step

10ms

Modification Rules

None

Configuration Policy

For no call drop in ring rotating, two sets of data are recommended. It is suggested that the configuration is based on the recommended data.

When the transmission quality is good, the following data is recommended: ring II waiting time before rotate(s) equals 0; ring II try rotating duration time (s) equal 60; ring II rotating penalty time (s) equals 0; T200 (10 ms) equals 12; T203 (s) equals 1; N200 (times) equals 3.

When the transmission quality is poor, the following data is recommended: ring II waiting time before rotate(s) equals 30; ring II try rotating duration time (s) equal 60; ring II rotating penalty time (s) equals 0; T200 (10 ms) equals 24; T203 (s) equals 3; N200 (times) equals 3.

Recommended Value

12

Parameter Relationship

None
