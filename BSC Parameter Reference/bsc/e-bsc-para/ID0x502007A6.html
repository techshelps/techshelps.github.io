Ater Fault Protect Switch

Parameter Name

Ater Fault Protect Switch

Description

If this parameter is set to Open, when the transmission on the Ater interface is faulty, the BSC blocks the CIC circuits on the A interface; when the transmission fault on the Ater interface is rectified, the BSC unblocks the CIC circuits on the A interface.
