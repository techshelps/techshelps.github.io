Extended Cell Throughput Enhancement

Parameter Name

Extended Cell Throughput Enhancement

Description

This parameter specifies whether to enable the downlink throughput enhancement function in the double-timeslot cell. If this parameter is set to Open, the downlink PS services can be processed on the odd-numbered channels on the underlaid TRXs in the double-timeslot extended cell, and thus improve the downlink throughput. If this parameter is set to Close, the PS services cannot be processed on the odd-numbered channels on the underlaid TRXs in the double-timeslot extended cell.

GUI Value Range

[Open,Close]

Default Value

Close

Unit

None

Step

-

Modification Rules

Modifying this parameter leads to cell reset.

Configuration Policy

If this parameter is set to Open, then the downlink PS services can be processed on the odd-numbered channels on the underlaid TRXs in the double-timeslot extended cell, and thus improve the downlink throughput. If this parameter is set to Close, then the packet services cannot be processed on the odd-numbered channels on the underlaid TRXs in the double-timeslot extended cell.

Recommended Value

Open

Parameter Relationship

None

BSC Feature

GPRS&EGPRS

Parameter Class

Parameters About Radio Resources

Caution

This function applies to only the built-in PCU. At present, only the BTS3012 supports this function. For the BTS in TDM transmission mode, sufficient idle timeslots needs to be configured.
