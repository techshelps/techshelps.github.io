Reference Clock Source Type 2

Parameter Name

Reference Clock Source Type 2

Description

This parameter specifies the type of the clock reference source, that is, the type of the input clock source for the GGCU.

GUI Value Range

[2M external clock 0,2M external clock 1,LINE0,LINE1,NONE]

Default Value

NONE

Unit

None

Step

-

Modification Rules

None

Configuration Policy

When this parameter is set to NONE, the clock reference source for the GGCU is not configured.
