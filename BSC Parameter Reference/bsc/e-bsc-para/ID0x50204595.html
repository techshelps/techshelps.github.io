Intracell F-H HO Stat Time(s)

Parameter Name

Intracell F-H HO Stat Time(s)

Description

This parameter determines the period during which the triggering conditions of the intra-cell full-rate to half-rate handover are met.

The intra-cell full-rate to half-rate handover must conform to the P/N criterion. That is, the triggering conditions of the intra-cell full-rate to half-rate handover are met for P consecutive seconds with N measurement seconds.

This parameter corresponds to N of the P/N criterion. The triggering conditions of the intra-cell full-rate to half-rate handover are the F2H HO Threshold or the H2F HO Threshold. This parameter is used with the two parameters.

GUI Value Range

[1,16]

Default Value

5

Unit

Seconds

Step

-

Modification Rules

None

Configuration Policy

The greater the value of this parameter is set, the more difficult the AMR handover can be triggered.

Recommended Value

5

Parameter Relationship

None

BSC Feature

Handover, AMR

Parameter Class

Parameters About Radio Resources

Caution

This parameter is available only when HO Control Switch is set to HO Algorithm I. This parameter is invalid when the HO Control Switch is set to HO Algorithm II.
