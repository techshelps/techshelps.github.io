RSCP Offset

Parameter Name

RSCP Offset

Description

The RSCP is short for Received Signal Code Power, which is a performance counter of the 3G cells.

GUI Value Range

[0,63]

Default Value

3

Unit

dB

Step

-

Modification Rules

None

Configuration Policy

If both Inter-RAT Out BSC Handover Enable and Better 3G Cell HO Allowed are set to Yes, a cell can become a candidate cell when the RSCP of an adjacent 3G cell is greater than the total of the RSCP Threshold for Better 3G Cell HO and the value of this parameter during a period of time.

Recommended Value

3

Parameter Relationship

Better 3G Cell HO Allowed, RSCP Threshold for Better 3G Cell HO

BSC Feature

Handover
