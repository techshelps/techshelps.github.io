Switch Mode

Parameter Name

Switch Mode

Description

This parameter specifies the channel switch mode.

Single port switch refers to the channel switch that is determined by the BSC or the peer end based on its conditions.

Double port switch refers to the channel switch on both the BSC end and the peer end. The BSC sends a switch request to the peer end according to the APS protocol. After the peer end sends an acknowledgment, channel switch is initiated on both ends at the same time.

GUI Value Range

[Single Port Switch,Double Port Switch]

Default Value

Double Port Switch

Unit

None

Step

-

Modification Rules

None

Configuration Policy

The A interface board can be configured to single port switch mode only in BM/TC separated scenario. In other scenarios, the A interface board must be configured to double port switch mode.

Recommended Value

Double Port Switch
