Absolute Signal Strength Threshold

Parameter Name

Absolute Signal Strength Threshold

Description

This parameter specifies the absolute signal level threshold of the adjacent cell.

This function is used for automatic optimization of frequencies and adjacent cells. The system can process the measurement results according to this threshold and use them in subsequent NaStar analysis.

When the host detects in a measurement report that the signal strength in adjacent cells exceeds this value, the number of Measurement Reports with Signal Strength of Neighbor Cell Greater Than Absolute Level Threshold is incremented by one.

The value of this parameter ranges from 0 to 63 (corresponding to -110 dBm to -47 dBm).

GUI Value Range

[0,63]

Default Value

30

Unit

None

Step

2dBm

Modification Rules

None

Configuration Policy

None

Recommended Value

It is recommended that this parameter be set as follows:

30: urban areas without PBGT handover

25: suburbs with a single BTS

25: urban areas with PBGT handover enabled
