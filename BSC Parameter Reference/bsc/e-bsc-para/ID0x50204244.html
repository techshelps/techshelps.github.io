Use LAPDm N200

Parameter Name

Use LAPDm N200

Description

This parameter specifies whether the BSC sends the LAPDm N200 parameter to the BTS.

GUI Value Range

[No,Yes]

Default Value

No

Unit

None

Step

-

Modification Rules

None

Configuration Policy

Only the BTS3X in G3BTS32.30000.04.1130 or later and the double-transceiver BTSs support the LAPDm N200 parameter.

If this parameter is set to Yes, the BSC sends the LAPDm N200 parameter. If this parameter is set to No, the BSC does not send the LAPDm N200 parameter.

If a BTS does not support this parameter, the parameter should be set to No. Otherwise, the BTS cannot be initialized.

Recommended Value

No

Parameter Relationship

Timer N200:

N200 of establish, N200 of release, N200 of SACCH, N200 of SDCCH, N200 of FACCH/TCHH, and N200 of FACCH/TCHF
