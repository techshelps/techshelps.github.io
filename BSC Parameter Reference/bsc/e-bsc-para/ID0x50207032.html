Special Analog Alarm Disable Configuration

Parameter Name

Special Analog Alarm Disable Configuration

Description

This parameter consists of eight bits, each of which from left to right specifies whether to enable or disable a special analog alarm. The bit 0 indicates enable and the bit 1 indicates disable. Selecting bit 1 indicates that the analog alarm is disabled. The eight alarm parameter values are as follows:

Preserved Parameter4, Preserved Parameter3, Humidity Sensor(Disabled), Temperature Sensor(Disabled), Preserved Parameter2, Preserved Parameter1, Preserved Parameter0, -48 Voltage(Disabled)

GUI Value Range

[Preserved Parameter4,Preserved Parameter3,Humidity Sensor(Disabled),Temperature Sensor(Disabled),Preserved Parameter2,Preserved Parameter1,Preserved Parameter0,-48 Voltage(Disabled)]
