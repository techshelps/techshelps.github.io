MAIO Plan

Parameter Name

MAIO Plan

Description

This parameter specifies the MAIO list of the MA group that is allowed to be used in the cell. To fully utilize the frequency diversity gain, you are advised to use the 1×1 frequency reuse mode when the number of TCH frequencies involved in FH in each cell is less than eight. For accuracy purpose, however, the MAIO of each IBCA-enabled cell still uses the 1×3 frequency reuse mode so that some MAIOs are not used in this cell. This parameter records the MAIOs that can be used in this cell.
