MAX BSC Reset Message Retransmission on A-Interface

Parameter Name

MAX BSC Reset Message Retransmission on A-Interface

Description

This parameter specifies the maximum number of retransmitted BSC reset messages on the A interface.

When the BSC is reset, the BSC sends a reset message to the MSC and starts a timer.

If the time of waiting for the BSC reset response message on the A interface expires for the BSC, the BSC retransmits the message and restarts the timer. If the timer expires again and the number of message transmissions is smaller than the value of this parameter, the BSC continues to transmit the message.
