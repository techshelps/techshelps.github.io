Signaling Link Mask

Parameter Name

Signaling Link Mask

Description

This parameter specifies the way for the BSC6000 to implement the load sharing of signaling services between multiple MTP links in the link set.

When only one link is configured in the link set, Signaling Link Mask can be set to any value.

When two signaling links are configured in the link set, the number of 1s in Signaling Link Mask must be greater than or equal to 1. For example, the parameter can be set to B0001, B1100, or B1111.

When three or four signaling links are configured in the link set, the number of 1s in Signaling Link Mask must be greater than or equal to 2. For example, the parameter can be set to B0011, B1100, or B1111.

When five to eight signaling links are configured in the link set, the number of 1s in Signaling Link Mask must be greater than or equal to 3. For example, the parameter can be set to B1011, B1110, or B1111.

When 9 to 16 signaling links are configured in the link set, Signaling Link Mask must be set to B1111.

To avoid the overlapping of bits, the operator cannot set Signaling Link Set Mask (configured by using the ADD N7DSP command) and Signaling Link Mask in the link sets to the same value. That is, the digit 1 in Signaling Link Set Mask and that in Signaling Link Mask cannot be in the same position. For example, if Signaling Link Set Mask is set to B0110, the parameter Signaling Link Mask can be set to B1001.
