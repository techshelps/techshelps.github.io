IntraCell Non-AMR TCHF-TCHH Ho Qual. Threshold

Parameter Name

IntraCell Non-AMR TCHF-TCHH Ho Qual. Threshold

Description

If the MS receive quality is smaller than the receive quality threshold, you can infer that the MS meets the receive quality threshold.

GUI Value Range

[0,7]

Default Value

0

Unit

None

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set too large, the MSs that have poor speech quality at full rate meet the handover conditions but have the risk of being handed over to half-rate services with equally poor speech quality.

If this parameter is set too small, it is difficult for the MSs to meet the handover conditions and thus the network congestion cannot be alleviated.

Recommended Value

0

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Radio Resources

Caution

This parameter can be configured only when HO Control Switch is set to HO Algorithm II; this parameter is invalid when HO Control Switch is set to HO Algorithm I.
