Channel 2 Digital Control Signal Level

Parameter Name

Channel 2 Digital Control Signal Level

Description

This parameter specifies the external signal level at which an alarm is generated. If an alarm needs to be generated when the level of an external signal is high, set this parameter to High; otherwise, set this parameter to Low.

GUI Value Range

[High,Low]

Default Value

High
