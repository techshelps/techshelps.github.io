Upper Limit of Voltage Alarm for 24 V Power Supply(V)

Parameter Name

Upper Limit of Voltage Alarm for 24 V Power Supply(V)

Description

This parameter indicates the upper limit of voltage alarm for 24 V power supply. If the voltage of the 24 V power supply is higher than this parameter, the Abnormal Voltage alarm is generated.

GUI Value Range

[20.00,28.00]

Default Value

28.00

Unit

Volts

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to an overlarge or oversmall value, some alarms may not be reported. Huawei recommends that you retain the default value.

Recommended Value

28.00

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Equipment

Caution

If the environment monitor device uses the 48 V power supply, the alarms related to the 24 V power supply are not reported.

Upper Limit of Voltage Alarm for 24V Power Supply cannot be smaller than Lower Limit of Voltage Alarm for 24V Power Supply.
