Abis Mode

Parameter Name

Abis Mode

Description

This parameter specifies the configuration mode of RSL timeslots on the Abis interface.

The default value is Auto, that is, the BSC automatically assigns the RSL timeslots on the Abis interface.

Generally, this parameter should be set to Auto. For the convenience of users, however, they are allowed to assign RSL timeslots manually. If this parameter is set to Manual, the RSL timeslots cannot be adjusted during timeslot defragmentation.
