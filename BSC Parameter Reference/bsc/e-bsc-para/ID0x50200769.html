Congestion Ratio on Ater Resource(%)

Parameter Name

Congestion Ratio on Ater Resource(%)

Description

When assigning channels, the system decides whether to use full-rate channels or half-rate channels according to the congestion situations on the Ater interface.

If the resource usage on the Ater interface exceeds the value of this parameter, the Ater interface is congested.

GUI Value Range

[80,100]

Default Value

85

Unit

%

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to a higher value, the full-rate TCH is still preferably selected when the Ater interface is congested. Thus, the system capacity decreases.

If this parameter is set to a lower value, the half-rate TCH is selected when the Ater interface is not congested. Thus, the conversation quality is degraded.

Recommended Value

85

Parameter Relationship

None

BSC Feature

None
