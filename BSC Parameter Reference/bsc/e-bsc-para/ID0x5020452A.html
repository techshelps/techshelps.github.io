Sent Freq.of preprocessed MR

Parameter Name

Sent Freq.of preprocessed MR

Description

This parameter specifies the frequency at which the BTS sends the preprocessed MR to the BSC. After preprocessing the MRs, the BTS sends the preprocessed MRs to the BSC. For example, if this parameter is set to Twice every second, the BTS sends preprocessed MRs to the BSC every 0.5 second.

GUI Value Range

[Not Report,Twice Every Second,Once Every Second,Once Every Two Second,Once Every Four Second]

Default Value

Twice Every Second

Unit

None

Step

-

Modification Rules

None

Configuration Policy

This parameter should be set on the basis of the data rate and flow on the Abis interface. If the preprocessed MR is sent at a high frequency, the flow on the Abis interface is increased.

Recommended Value

Twice Every Second

Parameter Relationship

MR Preprocessing

BSC Feature

Handover and power control

Parameter Class

Parameters About Radio Resources

Caution

If this parameter is set to Do not report, the BTS does not send the preprocessed MR to the BSC. Thus, the handover and the power control cannot be performed, and the conversation quality is severely affected.
