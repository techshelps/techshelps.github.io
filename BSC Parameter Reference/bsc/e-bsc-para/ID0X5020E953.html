ANT_B ALD Current Detection Mode

Parameter Name

ANT_B ALD Current Detection Mode

Description

his parameter indicates the mode in which the alarm related to the ANT_B tributary on the RXU is generated. There are three alarm modes.

In mode 1, there is only one alarm severity. The alarm current is fixed and higher than the normal working current.

In mode 2, there are two alarm severities: minor and major. The "ALD Current Prompt Alarm" severity is minor while the "ALD Current Abnormity Alarm" severity is major. The minor alarm current and major alarm current are fixed and higher than the normal working current.

In mode 3, there are two alarm severities: minor and major. The "ALD Current Prompt Alarm" severity is minor whereas the "ALD Current Abnormity Alarm" severity is major. The minor alarm current is pulsed, and the major alarm current is fixed.

When the parameter is set to MODE_1(MODE_1), "ANT_B ALD Current Minor Alarm Occur Threshold[mA]", "ANT_B ALD Current Minor Alarm Clear Threshold[mA]", "ANT_B ALD Current Minor Alarm Over Current Duration[ms]", and "ANT_B ALD Current Minor Alarm Normal Current Duration[ms]" cannot be set.

When the parameter is set to MODE_2(MODE_2), "ANT_B ALD Current Minor Alarm Over Current Duration[ms]" and "ANT_B ALD Current Minor Alarm Normal Current Duration[ms]" cannot be set.

When the parameter is set to MODE_3(MODE_3), "ANT_B ALD Current Minor Alarm Occur Threshold[mA]" and "ANT_B ALD Current Minor Alarm Clear Threshold[mA]" cannot be set.
