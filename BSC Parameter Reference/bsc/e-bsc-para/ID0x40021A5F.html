High Temperature Alarm Threshold(℃)

Parameter Name

High Temperature Alarm Threshold(℃)

Description

This parameter specifies the temperature alarm threshold. When the temperature exceeds the threshold, a temperature alarm is reported if the Temperature Alarm Mask is set to Open.
