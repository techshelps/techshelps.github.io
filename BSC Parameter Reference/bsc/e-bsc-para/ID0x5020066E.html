Abis Flow Control Timer 1(s)

Parameter Name

Abis Flow Control Timer 1(s)

Description

The two timers T1 (Abis Flow Control Timer 1(s)) and T2 (Abis Flow Control Timer 2(s)) are set for adjusting the flow control level in the cell. When the flow control level in the cell changes, the values of MS MAX Retrans and Tx-integer(RACH Timeslot(equals to a TDMA frame, 4.615 ms)) in the system information are changed.

When the immediate assignment fails, the BSC starts T1 and T2.

GUI Value Range

[10,255]

Default Value

30

Unit

Seconds

Step

-

Modification Rules

None

Configuration Policy

None

Recommended Value

30

Parameter Relationship

The value of T1 must be smaller than the value of T2.

Before T1 expires, all the BTS overload messages received by the BSC are ignored. During the period after T1 expires and before T2 expires, if the BSC receives a BTS overload message, the BSC increases the flow control level on the Abis interface and restarts T1 and T2. When T2 expires, the BSC decreases the flow control level on the Abis interface. If the BTS is still overloaded, that is, the flow control level is not 0, the BSC restarts T2, if the BTS is not overloaded, the flow control on the Abis interface is complete.
