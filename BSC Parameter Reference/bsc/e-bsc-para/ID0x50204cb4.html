Report Downlink VQI Allowed

Parameter Name

Report Downlink VQI Allowed

Description

This parameter specifies whether to allow the BTS to report the downlink voice quality index (VQI). After this parameter is set to Enable, the BTS extracts the DTX_USED, RXLEV_VAL, RX_QUAL_FULL, MEAN_BEP, CV_BEP, and NBR_RCVD_BLOCKS from the Enhanced Measurement Reports, Then, the BTS calculates the VQI by the RTT formula based on the extracted information and reports the VQI to the BSC. The BSC measures traffic statistics about the related downlink VQI for observing and evaluating the downlink speech quality.

GUI Value Range

[No,Yes,Invalid]

Default Value

No

Unit

None

Step

-

Modification Rules

None

Configuration Policy

When you need to observe and evaluate the downlink speech quality, set this parameters to Enable.

Recommended Value

None

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Radio Resources

Caution

1. Ensure that the BTS in use supports the reporting of the downlink VQI before setting this parameter to Enable. This parameter can be configured for the following BTS versions: G3BTS32V302R007C01B019 and its later versions; BTS3000V100R001C05B091 and its later versions, and BTS3900V100R008C11 and its later versions.

2. The EMR function should be enabled in the cell.
