RXLEV_ACCESS_MIN

Parameter Name

RXLEV_ACCESS_MIN

Description

This parameter specifies the minimum receive level of an MS to access the BSS. For details. see GSM Rec. 05.08. The minimum access level of an MS is calculated as follows: Minimum access level (dBm) = Value of RXLEV_ACCESS_MIN - 110For example, if the value of the RXLEV_ACCESS_MIN parameter is 8, the minimum access level of the MS is -102 (8 - 110) dBm.

GUI Value Range

[0,63]

Default Value

8

Unit

Levels

Step

-

Modification Rules

None

Configuration Policy

If the value of this parameter is too small, the required level of received signals is low. Therefore, many MSs attempt to camp on the cell, thus increasing the load of the cell and the risk of call drops. In such a case, you must set the parameter based on the balance conditions of the uplink and downlink levels.

Recommended Value

8
