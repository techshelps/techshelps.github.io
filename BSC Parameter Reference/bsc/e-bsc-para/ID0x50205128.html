TRX Priority

Parameter Name

TRX Priority

Description

This parameter specifies the TRX priority. It is used for Huawei II channel assignment algorithm.

GUI Value Range

[Level0,Level1,Level2,Level3,Level4,Level5,Level6,Level7]

Default Value

Level0

Unit

None

Step

-

Modification Rules

None

Configuration Policy

The smaller this parameter is, the higher the TRX priority is. In other similar conditions, channels are allocated to the TRX with higher priority.

Recommended Value

None

Parameter Relationship

This parameter is used together with Allocation TRX Priority Allowed. The channel can be assigned based on the TRX priority only when the Allocation TRX Priority Allowed is set to Yes.
