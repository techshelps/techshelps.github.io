Parameter Name

Penalty Time after OtoU HO Fail(s)

Description

After an OL subcell to UL subcell handover fails, the call cannot be handed over from the OL subcell to the UL subcell during the Penalty Time after OtoU HO Fail.

GUI Value Range

[0,255]

Default Value

10
