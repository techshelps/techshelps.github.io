BTS Patch Version

Parameter Name

BTS Patch Version

Description

This parameter specifies the patch version loaded to the BTS. This parameter can be configured only when the parameter Auto Download Active Function Switch is set to Yes.

This parameter can be set to only a hot patch version. You can use either of the following methods to query the version number of the BTS hot patch.

Method 1: If the BTS hot patch is downloaded to the OMU through the MML command DLD BTSPATCH, you can run the MML command LST BTSPATCH to obtain the number of the hot patch package and the hot patch version according to the returned result.

Method 2: If the BTS hot patch is not downloaded to the OMU, you can decompress the BTS hot patch package. Then, check the patchcfg.xml file in the level-1 directory of the compressed package. The value after the label "PatchVersion" is the software version.
