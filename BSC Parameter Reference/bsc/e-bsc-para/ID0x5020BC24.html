THP1-ARP1 Priority Weight

Parameter Name

THP1-ARP1 Priority Weight

Description

In the Interactive service, this parameter specifies the priority weight for the THP1 and the ARP1. The value of this parameter determines the size of budget block and the priority of block scheduling.

GUI Value Range

[1,10]
