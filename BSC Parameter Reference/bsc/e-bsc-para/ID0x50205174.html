Power Overload Threshold In Ho. In(VA)

Parameter Name

Power Overload Threshold In Ho. In(VA)

Description

If the sum of the maximum output power of the power amplifier and the value of this parameter is greater than the total power of BTS, an incoming handover can be performed.

GUI Value Range

[0,50]

Default Value

8

Unit

Watt

Step

-

Modification Rules

None

Configuration Policy

None

Recommended Value

8

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Equipment

Caution

The value of the Power Overload Threshold In Ho. In parameter must be smaller than or equal to the value of the Power Overload Threshold In Ho. Out parameter. This is to ensure that other timeslots or cells do not carry heavy power load because of the power insufficiency handover.
