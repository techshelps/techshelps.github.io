Port Type

Parameter Name

Port Type

Description

If this parameter is set to TOP extend out port, the local E1/T1 ports are used to extend the transmission of other boards. That is, the timeslots on the source T1/T1 port are used by other boards through the extended E1/T1 port.

If this parameter is set to TOP extend in port, the local E1/T1 ports are used to extend the transmission of local boards. That is, the local boards can obtain the timeslots from other boards through the extended E1/T1 port.
