Fix Abis Prior Choose Abis Load Threshold(%)

Parameter Name

Fix Abis Prior Choose Abis Load Threshold(%)

Description

This parameter specifies the load threshold of Fix Abis resources. When the load of Fix Abis resources is lower than the value of this parameter, the TCHFs are preferentially allocated. Otherwise, the TCHFs or TCHHs are preferentially allocated according to load of Flex Abis resources. This parameter is valid only when the parameter Abis Resource Adjustment TCHH Function Switch is set to Yes.
