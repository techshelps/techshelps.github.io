CCCH Conf

Parameter Name

CCCH Conf

Description

This parameter specifies the CCCH configuration in a system. In a BCCH multi-frame, the number of CCCH message blocks may be 3, 9, 18, 27, or 36. The CCCH configuration determines the number of PCHs, AGCHs, and RACHs.

For the cell with one TRX, it is recommended that one combined CCCH be configured (in the system that has a small number of paging messages in an LA). For a cell with more than one TRX, the configuration of CCCH depends on the number of TRXs in the cell.

For a system that has both main and extended BCCHs, the number of non-combined CCCHs equals the number of BCCHs.

The BS_AG_BLKS_RES and CCCH Conf. parameters can be dynamically adjusted according to the type of channel 0 on the main BCCH TRX in the cell.

If this parameter is set to a value rather than 1 Compounding CCCH, the default value of BS_AG_BLKS_RES can be set to 2 and the value of BS_AG_BLKS_RES ranges from 0 to 7. If CCCH Conf is set to 1 Compounding CCCH, the default value of BS_AG_BLKS_RES can be set to 1 and the value of BS_AG_BLKS_RES ranges from 0 to 2.

If channel 0 on the main BCCH TRX is set to Combined BCCH or BCCH+CBCH, this parameter can be set to 1 Compounding CCCH.

If channel 0 on the main BCCH TRX is set to Main BCCH, this parameter can be set to N Non-compounding CCCH, where N indicates the sum of channels 0, 2, 4, and 6 that are configured as main BCCHs and BCHs.

GUI Value Range

[1 Non-compounding CCCH,1 Compounding CCCH,2 Non-compounding CCCH,3 Non-compounding CCCH,4 Non-compounding CCCH]

Default Value

If channel 0 on the main BCCH TRX is set to Combined BCCH or BCCH+CBCH, the default value of this parameter is 1 Compounding CCCH. If channel 0 on the main BCCH TRX is set to Main BCCH, the default value of this parameter is "N Non-compounding CCCH", where N is the total number of channels (channel 0, channel 2, channel 4, or channel 6) set to BCH and Main BCCH.

Unit

None

Step

-

Modification Rules

None

Configuration Policy

During the configuration of this parameter, the capacity of PCHs should be considered. Generally, ensure that the PCH capacity of each cell under one LAC is consistent, or at least ensure that the number of TRXs in the case of the minimum PCH capacity is greater than the sum of TRXs under the LAC.

Recommended Value

If channel 0 on the main BCCH TRX is set to Combined BCCH or BCCH+CBCH, the default value of this parameter is 1 Compounding CCCH. If channel 0 on the main BCCH TRX is set to Main BCCH, the default value of this parameter is "N Non-compounding CCCH", where N is the total number of channels (channel 0, channel 2, channel 4, or channel 6) set to BCH and Main BCCH.
