IntraCell TCHH-TCHF Ho Last Time(s)

Parameter Name

IntraCell TCHH-TCHF Ho Last Time(s)

Description

This parameter indicates that if the IntraCell TCHH-TCHF Ho Last Time meets the conditions for triggering AMR TCHF-TCHH handover within the IntraCell TCHF-TCHH Ho Stat. Time, the AMR TCHF-TCHH switchover is triggered.

This parameter, used with IntraCell TCHH-TCHF Ho Stat Time(s), equals the P values defined in the P/N criterion. If the IntraCell TCHF-TCHH Ho Last Time meets the conditions for triggering AMR TCHF-TCHH handover within the IntraCell TCHF-TCHH Ho Stat. Time, the AMR TCHF-TCHH handover is triggered.

GUI Value Range

[1,16]

Default Value

2

Unit

Seconds

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set too large, the MS takes a longer time to meet the conditions for triggering the AMR TCHH-TCHF handover. As a result, the AMR TCHH-TCHF handover cannot be performed immediately and the speech quality unnecessarily deteriorates.

If this parameter is set too small, the MS takes a shorter time to meet the conditions for triggering the AMR TCHH-TCHF handover. As a result, the AMR TCHH-TCHF handover might incur a wrong decision.

Recommended Value

2

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Radio Resources

Caution

This parameter can be configured only when HO Control Switch is set to HO Algorithm II; this parameter is invalid when HO Control Switch is set to HO Algorithm I.
