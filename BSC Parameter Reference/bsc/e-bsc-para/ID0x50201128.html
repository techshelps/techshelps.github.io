Transmission Mode

Parameter Name

Transmission Mode

Description

This parameter specifies whether terrestrial transmission or satellite transmission or terrestrial satellite transmissionis used between the BTS and the BSC. Different time delays in the previous two transmission modes have impact on the LAPD links.Satellite transmission mod and Terrestrial Satellite transmission mod are controlled by the license.

GUI Value Range

[Terrestrial Transmission,Satellite Transmission,Terrestrial Satellite Transmission]

Default Value

Terrestrial Transmission

Unit

None

Step

-

Modification Rules

Modifying this parameter leads to BTS reset.

Configuration Policy

None

Recommended Value

Terrestrial Transmission

Parameter Relationship

None

BSC Feature

None

Parameter Class

Parameters About Equipment

Caution

Modifying this parameter leads to the modification of the parameters T3168, T3192, and BS_CV_MAX.

If this parameter is set to Terrestrial Transmission, the values of the preceding three parameters are 500 ms, 500 ms, and 10.

If this parameter is set to Satellite Transmission, the values of the preceding three parameters are 1000 ms, 1000 ms, and 15.

If this parameter is set to Terrestrial Satellite Transmission, the values of the preceding three parameters are 1000 ms, 1000 ms, and 15.
