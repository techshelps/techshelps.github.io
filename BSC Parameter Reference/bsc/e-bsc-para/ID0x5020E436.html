Timeslot Mask

Parameter Name

Timeslot Mask

Description

The 64 kbit/s timeslots on the E1 port are numbered from 0 to 31. If a bit is set to 1, the corresponding 64 kbit/s timeslot must be exchanged between the E1 port and the TOP port. For the T1 scheme, the timeslots are numbered from 0 to 24.
