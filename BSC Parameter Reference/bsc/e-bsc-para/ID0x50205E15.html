T17(s)

Parameter Name

T17(s)

Description

T17 and T18 are used together for the flow control on the A interface.

When receiving the MSC overload message or single-point congestion message for the first time, the BSC increases the level of flow control on the A interface and starts timers T17 and T18.

GUI Value Range

[1,300]

Default Value

30

Unit

Seconds

Step

-

Modification Rules

None

Configuration Policy

None

Recommended Value

30

Parameter Relationship

The value of T17 must be smaller than the value of T18.

Before T17 expires, all the MSC overload messages received by the BSC are ignored.

During the period after T17 expires and before T18 expires, if the BSC receives an MSC overload message, the BSC increases the level of flow control on the A interface and restarts T17 and T18.

If T18 expires, the BSC decreases the level of flow control on the A interface. In this case, if the MSC is still overloaded, T18 is restarted. Otherwise, the flow control on the A interface is complete.
