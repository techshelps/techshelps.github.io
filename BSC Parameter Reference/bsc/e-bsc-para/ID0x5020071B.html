CB Interface Handshake

Parameter Name

CB Interface Handshake

Description

At present, the BSC communicates with the CBC based on the TCP/IP protocol, so the BSC may not detect that the link to the CBC is disrupted. The handshake mechanism is used to detect whether the communication link is normal.

GUI Value Range

[No,Yes]

Default Value

Yes

Unit

None

Step

-

Modification Rules

None

Configuration Policy

None

Recommended Value

Yes

Parameter Relationship

If this parameter is set to Yes, the BSC sends a handshake message to the CBC after the Cell Broadcast Handshake Timer expires to determine whether the communication link between the BSC and the CBC is normal.

BSC Feature

Cell Broadcast
