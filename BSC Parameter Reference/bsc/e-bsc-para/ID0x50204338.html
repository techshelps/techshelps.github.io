Measurement Report Type

Parameter Name

Measurement Report Type

Description

This parameter specifies the type of measurement report sent by an MS.

The Enhanced Measurement Report is a new downlink measurement report, reported by the MS.

Compared with the Normal Measurement Report, the Enhanced Measurement Report has some new measurement items, such as BER and FER. An MS can report up to six GSM neighbor cells in a Normal Measurement Report. Compared with the Normal Measurement Report, the Enhanced Measurement Report has more measurement items.

Configuration Policy

The Enhanced Measurement Report supports the measurement of the 3G neighbor cells to realize the interoperability between the 2G system and the 3G system, and thus ensures the service continuity.
