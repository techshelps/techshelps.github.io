Min Power Level For Directed Retry

Parameter Name

Min Power Level For Directed Retry

Description

This parameter is used to select the candidate cells during directed retry in HO Algorithm I. If the receive level of an adjacent cell is greater than the value of this parameter, the adjacent cell can be selected as a candidate cell for directed retry. The value of this parameter ranges from 0 to 63 (corresponding to -110 dBm to -47 dBm).

GUI Value Range

[0,63]

Default Value

16

Configuration Policy

If the receive level of an adjacent cell is greater than or equal to the value of this parameter, this adjacent cell can be selected as a candidate cell for directed retry.
