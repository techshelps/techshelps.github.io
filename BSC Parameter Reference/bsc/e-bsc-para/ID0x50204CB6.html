Downlink Received Level Threshold of SDCCH Call Drop Statistic

Parameter Name

Downlink Received Level Threshold of SDCCH Call Drop Statistic

Description

This parameter specifies the level threshold for measuring the number of call drops on SDCCH.

When call drop occurs on the SDCCH, the BSC determines whether the downlink receive level is lower than this threshold. If the downlink receive level is lower than this threshold and the TA is smaller than the TA threshold of the serving cell, the number of call drops on SDCCH (receive level) is incremented by one.

GUI Value Range

[0,63]

Default Value

6

Unit

None

Step

-

Modification Rules

None

Configuration Policy

The range [0, 63] corresponds to [-110, -47] dBm. The smaller the value, the lower the receive level.

Recommended Value

6
