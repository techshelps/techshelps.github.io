NCell Interf Type

Parameter Name

NCell Interf Type

Description

This parameter is unconfigurable:

1. If two cells not belonging to the same site have identical frequencies in their CA tables, the two cells are called neighbor cells with co-channel interference to each other (Co-frequency interference).

2. If two cells not belonging to the same site have adjacent frequencies in their CA tables, the two cells are called neighbor cells with adjacent-channel interference to each other (Adjacent frequency interference).

3. If two cells not belonging to the same site have both identical frequencies and adjacent frequencies in their CA tables, the two cells are called neighbor cells with co-channel and adjacent-channel interference to each other (Co-frequency and Adjacent frequency interference).

4. For other cases except those listed above, the two cells are called neighbor cells without interference to each other (No interference).

GUI Value Range

[Co-frequency interference,Adjacent frequency interference,Co-frequency and Adjacent frequency interference,No interference]

Default Value

No interference
