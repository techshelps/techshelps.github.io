Additional Reselect Param Indication

Parameter Name

Additional Reselect Param Indication

Description

This parameter Additional Reselect Param Indication (ACS) is used to inform an MS where cell reselection parameters can be obtained.

If this parameter is set to 0, the MS should obtain PI and other parameters for calculating C2 from other bytes of the system information type 4 message. If this parameter is set to 1, the MS should obtain PI and other parameters for calculating C2 from other bytes of the system information type 7 or 8 message.

If the parameter value is forced to 0, it indicates that the ACS in the IE Cell Selection Parameters of system information type 4 message is fixed to 0.

If the parameter value is forced to 1, it indicates that the ACS in the IE Cell Selection Parameters of system information type 4 message is fixed to 1.

If the parameter value is forced to 2, it indicates that the ACS in the IE Cell Selection Parameters of system information type 4 message is determined based on the coding scheme. If system information type 4 message can hold Rest Octets, then ACS is 0; otherwise, ACS is 1.
