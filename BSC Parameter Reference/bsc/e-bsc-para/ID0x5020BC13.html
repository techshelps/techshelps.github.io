Maximum Ratio Threshold of PDCHs in a Cell(%)

Parameter Name

Maximum Ratio Threshold of PDCHs in a Cell(%)

Description

This parameter specifies the maximum ratio of PDCHs in a cell. The total number of TCHs and PDCHs available in a cell is fixed. The PDCH ratio is equal to PDCHs / (TCHs + static PDCHs). This parameter determines the proportion of PDCHs to the total number of TCHs + PDCHs.

GUI Value Range

[0,100]

Default Value

30

Unit

%

Step

-

Modification Rules

None

Configuration Policy

If this parameter is set to an excessive value, there are excessive PDCHs and insufficient TCHs. This affects CS services.

If this parameter is set to a modest value, there are insufficient PDCHs and excessive TCHs. This affects PS services.

Recommended Value

30

Parameter Relationship

This parameter affects the dynamic PDCH conversion as the parameter Maximum PDCH numbers of carrier does. If the proportion of PDCHs in a cell reaches the value of Maximum Ratio Threshold of PDCHs in a Cell or the PDCHs on a TRX reaches the value of Maximum PDCH numbers of carrier, then the TCH cannot be converted into the PDCH.

BSC Feature
