Configuring the 8K Reference Clock for the GEPS

This describes how to configure the 8 kHz clock provided by the GGCU as the 8 kHz reference clock for the GEPS. The 8 kHz reference clock serves as the synchronization clock for the boards in the GEPS.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalMandatory

Prerequisites

  • The GSCU in the GEPS is configured. For details, see Configuring the GSCU.
  • The GGCU in the GMPS is configured. For details, see Configuring the GGCU.
  • The system clock is configured.

Preparation

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the edge of the GEPS.
  2. Choose Configure 8K Reference Clock from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure GSCU 8K Reference Clock dialog box

  3. Set the parameters by referring to Table 1.
  4. Click Finish. The configuration of the 8 kHz reference clock for the GEPS is complete.
  5. Repeat 1 through 4 to configure the 8 kHz reference clock for multiple GEPSs.
