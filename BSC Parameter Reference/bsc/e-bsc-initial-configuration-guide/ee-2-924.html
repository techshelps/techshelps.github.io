Configuring the TRX Four-in-One Output

This describes how to configure the DFCU and its antenna feeder attributes for the BTS3012, BTS3012 II, or BTS3012AE to achieve the four-in-one output of TRXs.

ScenarioBSC initial configuration
Mandatory/OptionalOptional. Only one DFCU is configured when the four-in-one output of TRXs needs to be implemented in the BTS3012, BTS3012 II, or BTS3012AE.
NOTE:
  • The DFCU can independently provide the four-in-one output of TRXs.
  • The DFCU cannot be bound to the QTRU.
  • The DFCU should not be configured in slot 20 of the DAFU subrack in the BTS3012, BTS3012 II, or BTS3012AE. The slots on both sides of the DFCU slot should not be configured with the DFCU or the DFCB.
  • The TRX to be tuned must be configured on downlink tributary A of the DFCU. Tributary B should not be configured with TRXs.
  • The frequency band type of the DFCU must be the same as that of the TRX in the BTS.

Prerequisite

The frequency band type of the TRX in the BTS is specified.

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the BTS.
  2. Choose Add or Delete Site Board. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Add or Delete Site Board dialog box

  3. Click Add or Delete Board. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Site Device Attributes dialog box

  4. Right-click an empty slot, and then choose Add Board > DFCU from the shortcut menu. A DFCU is added.
  5. Right-click DFCU, and then choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 3.

    Figure 3 Set Antenna Feeder Property dialog box

  6. For details, see Table 1 to set the parameters, as shown in Figure 4.

    Figure 4 Setting antenna feeder attributes

  7. Click OK to return to the dialog box shown in Figure 2.
  8. Click OK to return to the dialog box shown in Figure 1.
  9. Click Finish. The configuration of the antenna feeder attributes of the DFCU is complete.
