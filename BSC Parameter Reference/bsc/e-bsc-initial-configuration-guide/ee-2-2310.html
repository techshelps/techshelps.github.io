Configuring Links on the A Interface

This describes how to configure the M3UA link set, M3UA links, and M3UA routes on the A interface for the communication between the BSC and the MSC.

  1. Configuring the M3UA Link Set
    This describes how to configure the M3UA link set on the A interface.
  2. Configuring the M3UA Link
    This describes how to configure the M3UA link. The M3UA link configured for the A interface implements the IP transmission between the BSC and the MSC. The M3UA link configured for the Abis interface implements the transmission between BSCs.
  3. Configuring the M3UA Route
    This describes how to configure the M3UA route for implementing the IP transmission between the BSC and the MSC.
