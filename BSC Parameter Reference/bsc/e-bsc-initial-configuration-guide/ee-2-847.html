Configuring the BSC EAC

This describes how to configure the EAC for the BSC cabinet. The EAC gathers Boolean values, analog values, and alarm threshold information and reports them to the LMT.

ScenarioBSC initial configuration
Mandatory/OptionalOptional. The EAC should be configured when the BSC needs to collect external alarm information.
NOTE:

One BSC cabinet can be configured with only one EAC.

Prerequisite

The subrack that holds the EAC is configured.

Preparation

None.

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the edge of a subrack.
  2. Choose Configure BSC EAC from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Add BSC EAC dialog box (switch)

    NOTE:

    You can use the default settings of the parameters or modify them according to actual requirements.

  3. Click Add. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Add BSC EAC dialog box (analog and alarm threshold)

    NOTE:

    You can use the default settings of the parameters or modify them according to actual requirements.

  4. Click Finish. The configuration of the BSC EAC is complete.
