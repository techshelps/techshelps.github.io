Configuring the Links on the Gb Interface (Gb over FR)

This describes how to configure the NSE, BC, NSVC, and PTPBVC on the Gb interface for the FR transmission between the BSC and the SGSN.

  1. Configuring the SGSN Node
    This describes how to configure the SGSN node on the Gb interface and determine the management status of the SGSN.
  2. Configuring the NSE
    This describes how to configure the Network Service Entity (NSE) on the Gb interface and determine the subnetwork protocol type.
  3. Configuring the BC
    This describes how to configure the bearer channel (BC) on the Gb interface in FR transmission mode. The BC is responsible for the data transmission between the BSC and the SGSN.
  4. Configuring the SGSN Clock
    This describes how to configure the SGSN clock in Gb over FR mode. The SGSN clock is responsible for the synchronization between the GEPUG and the SGSN.
  5. Configuring the NSVC
    This describes how to configure the Network Service Virtual Connection (NSVC) for service transmission on the Gb interface in Gb over FR mode. The NSVC and the BC are bound together to implement the data transmission between the BSC and the SGSN.
  6. Configuring the PTPBVC
    This describes how to add a PTPBVC on the Gb interface to bind the GPRS cell with the NSE.
