Configuring the GEIUP/GOIUP

This describes how to add a GEIUP/GOIUP in the GMPS/GEPS and configure its attributes.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalOptional. The GEIUP/GOIUP is configured when the BSC is configured with the external PCU.

Prerequisite

The subrack has idle slots to hold the GEIUP and GOIUP.

The procedures for configuring the GEIUP and GOIUP are the same. The following procedure takes how to add a pair of active and standby GEIUPs in the GMPS as an example.

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the BSC6000 root node.

  2. Choose Configure BSC Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure BSC Attributes dialog box

  3. On the Basic Data tab page, set PCUType to Outer.
  4. On the BSC6000 Local Maintenance Terminal, click the BSC Device Panel tab. Select a cabinet number from the Current Cabinet drop-down list.
  5. Right-click an idle slot among slots 14-27 holding the GMPS.
  6. Choose Add Board. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Add Board dialog box

  7. Set FuncType to Pb interface board, and set Board Type to GEIUP. Then, set other parameters by referring to Table 1.
  8. Click Finish. The addition of the GEIUP is complete.
  9. Right-click GEIUP, and then choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 3.

    Figure 3 Configure Board Attributes dialog box (1)

  10. Set Load Key by referring to Table 1.
  11. Click Next. A dialog box is displayed, as shown in Figure 4.

    Figure 4 Configure Board Attributes dialog box (2)

  12. Set the attributes of the port on the GEIUP by referring to Table 2.
  13. Click Finish. The configuration of the GEIUP is complete.
