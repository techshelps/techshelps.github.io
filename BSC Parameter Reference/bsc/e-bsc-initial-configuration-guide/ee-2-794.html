Configuring a Power Distribution Box

This describes how to add a power distribution box and configure its attributes through the LMT.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalMandatory
NOTE:

Any subrack in the cabinet can be configured as the management subrack for the power distribution box.

Prerequisite

The subrack that manages the power distribution box is configured.

Preparation

Table 1 Example of the data negotiated and planned for configuring a power distribution box

Parameter

Example

Source

Cabinet No.

0

BSC internal planning

Management Subrack No.

0

BSC internal planning

Power Box Type

Normal Power Distribution Box

BSC internal planning

Procedure

  1. On the BSC6000 Local Maintenance Terminal, click the BSC Device Panel tab. Select a cabinet number from the Current Cabinet drop-down list.
  2. Right-click in the power distribution box area, and then choose Add Power Distribution Box from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Add Power Distribution Box dialog box

  3. Set the parameters by referring to Table 1.
  4. Click Finish. The addition of a power distribution box is complete.
  5. Right-click in the power distribution box area, and then choose Configure Power Distribution Box Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Configure Power Distribution Box dialog box

  6. For the parameters in the dialog box shown in Figure 2, you can use the default settings or change them as required.
  7. Click Finish. The configuration of the power distribution box is complete.
  8. Repeat 1 through 7 to configure power distribution boxes for more cabinets.
