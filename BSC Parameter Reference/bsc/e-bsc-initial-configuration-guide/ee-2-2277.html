Initial Configuration Scenario: BM/TC Combined, Built-in PCU

When the BM and TC are configured in the same subrack and the built-in PCU is used, you need to follow the fixed procedures and methods to perform the BSC initial configuration.

Prerequisites

  • The BSC global data is ready. For details, see BSC Global Data (BM/TC Combined, Built-in PCU).
  • The A interface data is negotiated for interconnection. For details, see A Interface Data Negotiated for Interconnection (A over Non-IP).
  • The Gb interface data is negotiated for interconnection. For details, see Gb Interface Data Negotiated for Interconnection.

Procedure

  1. Configure the BSC (BM/TC combined).

    Mandatory For details, see Configuring the BSC (BM/TC Combined).

  2. Configure the built-in PCU.

    Mandatory For details, see Configuring the Built-in PCU.

  3. Configure the BTS.

    Mandatory For details, see Configuring the BTS.

  4. Configure the reciprocal relations between cells.

    Mandatory For details, see Configuring the Reciprocal Relations Between Cells.

  5. Back up the LMT local data.

    Mandatory For details, refer to Backing Up LMT Local Data.

  6. Activate the license file.

    Mandatory For details, refer to Activating the BSC License.

  7. Load the local configuration files.

    Mandatory For details, refer to Loading and Restoring the Local Data File.
