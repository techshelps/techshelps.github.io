BTS3900/BTS3900A Initial Configuration Guide

This describes how to configure a BTS3900 or BTS3900A for the BSC. This configuration enables the radio transmission and reception functionality of the BTS and fulfills radio coverage of the cells. The configuration also enables the BSC to perform centralized control and management of the BTS and allocate radio resources for the BTS.

  • Procedure for BTS3900/BTS3900A Initial Configuration
    This describes the procedure for BTS3900/BTS3900A initial configuration. The procedure involves obtaining data before configuration, adding a BTS, and adding other optional features of a BTS.
  • Data to Be Negotiated and Planned for BTS3900/BTS3900A Initial Configuration
    This describes the data to be negotiated and planned for the initial configuration of a BTS3900/BTS3900A. Before configuring a BTS3900/BTS3900A, you need to obtain the data of the BTS, which consists of site configuration, site attributes, RXU chain attributes, board attributes, cell attributes, cell broadcast parameters, and TRX attributes.
  • Adding a BTS3900/BTS3900A (TDM)
    This describes how to add a BTS3900/BTS3900A in TDM transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, TRX attributes.
  • Adding a BTS3900/BTS3900A (IP)
    This describes how to add a BTS3900/BTS3900A in IP transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, TRX attributes, BTS clock, and route.
  • Adding a BTS3900/BTS3900A (HDLC/HUB)
    This describes how to add a BTS3900/BTS3900A in HDLC/HUB transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, TRX attributes.
  • Configuring a BTS3900/BTS3900A
    This describes how to configure a BTS3900/BTS3900A. The procedure and precautions for some complicated configurations are also described.
  • Typical Scenarios of Configuring a BTS
    This describes typical scenarios of configuring a BTS for the BSC.
  • Reference for TRX Power Configuration
    TRX power configuration specifications serve as only the reference for on-site configurations. You can adjust specific configuration based on the actual situation.
  • Mapping Rule of the RFU Cabinet
    The mapping rules of the RFU cabinet are classified into the following types: Mapping Rule for One Cabinet, Mapping Rule for Two Cabinets, and Mapping Rule for Three Cabinets.
  • Customized Alarm Ports of the DBS3900/BTS3900/BTS3900A Base Station
    This describes the mapping between internal alarm IDs of the BTS and environment alarm ports and cables of each monitoring device. The monitoring devices are the EMUA, UPEU, UEIU, APMI, and FMUA.
