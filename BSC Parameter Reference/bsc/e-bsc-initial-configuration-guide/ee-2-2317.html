Configuring the Built-in PCU

When the built-in PCU is configured, the related BSC boards and Gb interface links should be configured on the basis of the transmission mode (FR or IP) used on the Gb interface. These boards and links are responsible for the communication between the BSC and the SGSN.

Procedure

  • In Gb over FR mode, the procedure for configuring the built-in PCU is as follows:
    1. Configure the GDPUP. For details, see Configuring the GDPUP.
    2. Configure the GEPUG (Gb over FR). For details, see Configuring the GEPUG (Gb over FR).
    3. Configure the Gb interface links (Gb over FR). For details, see Configuring the Links on the Gb Interface (Gb over FR).
  • In Gb over IP mode, the procedure for configuring the built-in PCU is as follows:
    1. Configure the GDPUP. For details, see Configuring the GDPUP.
    2. Configure the GFGUG (Gb over IP). For details, see Configuring the GFGUG (Gb over IP).
    3. Configure the Gb interface links (Gb over IP). For details, see Configuring the Links on the Gb Interface (Gb over IP).
