Data to Be Negotiated and Planned for BTS3900E GSM Initial Configuration

This describes the data to be negotiated and planned for the initial configuration of a BTS3900E GSM. Before configuring a BTS3900E GSM, you need to obtain the data of the BTS. The data consists of site configuration, site attributes, RXU link attributes, board attributes, cell attributes, cell broadcast parameters, and TRX attributes.

The data to be obtained before configuring the BTS is listed as follows.

  • For the configuration data of the TDM-based BTS3900E GSM, see Table 1.
  • For the configuration data of the IP-based BTS3900E GSM, see Table 2.
  • For the configuration data of the HDLC/HUB-based BTS3900E GSM, see Table 3.
NOTE:
  • CGI is comprised of MCC, MNC, LAC, and CI and is displayed in the format of "MCC-MNC-LAC-CI". In the following example, CGI is set to 460-01-8240-2.
  • If FH is applied, you need to obtain the FH serial number of each cell. FH serial numbers vary with different cells under the same BTS.
