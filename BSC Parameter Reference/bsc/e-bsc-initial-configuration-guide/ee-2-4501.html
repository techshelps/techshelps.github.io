BTS3900B GSM Initial Configuration Guide

This describes how to configure a BTS3900B for the BSC. This configuration enables the radio transmission and reception functionality of the BTS and fulfills the radio coverage of the cells. The configuration also enables the BSC to perform centralized control and management of the BTS and allocate radio resources for the BTS.

  • Adding a BTS3900B GSM BTS (IP)
    This describes how to add a BTS3900B GSM in IP transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, TRX attributes and BTS clock.
