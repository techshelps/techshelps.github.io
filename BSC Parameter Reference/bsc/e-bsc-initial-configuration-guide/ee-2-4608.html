Adding BTS Modules

This describes how to add BTS modules. You need to add other modules (except RF modules) of a BTS3900E GSM, including the APMU and the DTCU.

Mandatory/OptionalOptional

Procedure

  1. Add the APMU.
    1. In the Site Device Attributes dialog box, right-click a slot and then choose Add Board > APMU from the shortcut menu, as shown in Figure 1.

      Figure 1 Adding the APMU

    2. Right-click the added APMU, and then choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 2.

      Figure 2 Basic Attributes tab page

    3. Click the Configure Site Board Attributes tab to display the tab page, as shown in Figure 3.

      Figure 3 Configure Site Board Attributes tab page

    4. Set Board Parameter Configuration Enabled to Yes and configure parameters listed in Table 2. If a battery cabinet is configured, set Battery Cabinet Configuration Allowed to Yes.

      Table 2 Configurations of required APMU parameters

      Board Parameter Configuration

      Battery Cabinet Configuration
      • APMU Board Application Scenario
      • DPSU 0
      • DPSU 1
      • DPSU 2
      • Battery Type
      • Charge Coefficient
      • Battery Capacity
      CAUTION:

      The parameters related to the battery cabinet must be configured in accordance with the actual situation. Otherwise, the battery cabinet cannot work properly.

      Associated RXU Board No. must be Board No. of the BTS3900E.

    5. After the configuration is complete, click OK to return to the Site Device Attributes dialog box.
  2. Add the DTCU.
    1. In the Site Device Attributes dialog box, right-click a slot and then choose Add Board > DTCU from the shortcut menu, as shown in Figure 4.

      Figure 4 Adding the DTCU

    2. Right-click the added DTCU and then choose Configure Site Board Attributes from the shortcut menu. After data configuration is complete, click OK to return to the Site Device Attributes dialog box.

      NOTE:

      Associated RXU Board No. must be Board No. of the BTS3900E.

  3. Add the DEMU.
    1. In the Site Device Attributes dialog box, right-click a slot and then choose Add Board > DEMU from the shortcut menu, as shown in Figure 5.

      Figure 5 Adding the DEMU.

    2. Right-click the added DEMU and then choose Configure Site Board Attributes from the shortcut menu. After data configuration is complete, click OK to return to the Site Device Attributes dialog box.

      NOTE:

      Associated RXU Board No. must be Board No. of the BTS3900E.
