Configuring the GDPUX

This describes how to add a GDPUX in the GMPS/GEPS/GTCS and configure its attributes.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalMandatory

Prerequisite

The GMPS/GEPS/GTCS has an idle slot to hold the GDPUX.

NOTE:

In BM/TC separated mode, the GDPUX configured in the GTCS is used to process the speech service; the GDPUX configured in the GMPS or GEPS is used to provide the packet switching function for the BTS in IP or HDLC/HUB transmission mode.

Preparation

The following procedure takes how to configure the GDPUX in the GTCS as an example.

Procedure

  1. On the BSC6000 Local Maintenance Terminal, click the BSC Device Panel tab.
  2. Right-click an idle slot among slots 8-13 holding the GTCS, and then choose Add Board from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Add Board dialog box

  3. Set FuncType to Other board and Board Type to GDPUX and then click Finish to finish adding the GDPUX.
  4. Right-click GDPUX, and choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Configure Board Attributes dialog box

  5. Set Load Key by referring to Table 1.
  6. Click Next. A dialog box is displayed, as shown in Figure 3.

    Figure 3 Configure Board Attributes dialog box (DSPTC)

    NOTE:
    • When configuring the DSPTC attributes, select a DSPTC number first, and then configure the attributes of this DSPTC.
    • If you select Configure All, all the DSPTCs will be configured on the basis of the settings of the current DSPTC displayed in the DSPTC No. drop-down list.

  7. Set the parameters in the dialog box shown in Figure 3 as required.
  8. Click Finish. The configuration of the GDPUX is complete.
