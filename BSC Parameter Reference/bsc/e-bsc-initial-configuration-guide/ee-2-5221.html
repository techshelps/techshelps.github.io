Configuring the BTS Route

This describes how to configure the route for the BTS in IP transmission mode.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalOptional. When the BTS uses the IP transmission mode, you need to configure the route for the BTS.

Prerequisite

The BTS in IP transmission mode is configured.

Preparation

Table 1 Example of the data negotiated and planned for configuring the BTS route

Parameter

Example

Source

Gateway IP Address

172.20.6.1

Network planning

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the BTS in IP transmission mode.
  2. Choose Configure Site Route from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure Site Route dialog box

  3. Click Set Site Router. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Configure Site Router Record-BTS1 dialog box

  4. Click Add Record. A dialog box is displayed, as shown in Figure 3.

    Figure 3 Add SiteRouter dialog box

  5. Set the parameters by referring to Table 1.

    NOTE:

    By default, Destination IP Address is set to IP Address and Subnet Mask to IP Mask for the IP interface board of the configured BTS.

    Gateway IP Address and the port IP address of the PTU of the BTS must be configured on the same network segment.

  6. Click OK to return to the dialog box, as shown in Figure 2.
  7. Click OK to return to the dialog box, as shown in Figure 3.
  8. Click Finish. The configuration of the BTS route is complete.
