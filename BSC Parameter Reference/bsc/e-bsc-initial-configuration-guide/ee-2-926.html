Configuring the TRX Combination Output for the S12 Cell

This describes how to configure two DFCUs and one DFCB for the BTS3012, BTS3012 II, or BTS3012AE to achieve the TRX combination output for the S12 cell.

ScenarioBSC initial configuration
Mandatory/OptionalOptional. Two DFCUs and one DFCB are configured when the TRX combination output for the S12 cell needs to be implemented in the BTS3012, BTS3012 II, or BTS3012AE.

Prerequisite

The frequency band type of the TRX in the BTS is specified.

NOTE:
Two DFCUs (DFCU 0 and DFCU 1) and one DFCB should be configured to provide the TRX combination output for the S12 cell.

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the target BTS.
  2. Choose Add or Delete Site Board from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Add or Delete Site Board dialog box

  3. Click Add or Delete Board. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Site Device Attributes dialog box

  4. Right-click an idle slot and then choose Add Board > DFCB from the shortcut menu. A DFCB is added.
  5. Right-click DFCB and choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 3.

    Figure 3 Set Antenna Feeder Property dialog box

  6. See Table 1 to set the parameters, as shown in Figure 4.

    Figure 4 Setting DFCB antenna feeder attributes

  7. Click OK to return to the dialog box shown in Figure 2.
  8. Right-click an idle slot.
  9. Choose Add Board > DFCU from the shortcut menu. A DFCU is added.
  10. Right-click DFCU and choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 5.

    Figure 5 Set Antenna Feeder Property dialog box

  11. See Table 1 to set the parameters, as shown in Figure 6.

    Figure 6 Setting antenna feeder attributes

  12. Click the Extended Connection Attributes tab. The tab page is displayed, as shown in Figure 7.

    Figure 7 Extended Connection Attributes tab page (DFCU 0)

  13. See Table 1 to set the parameters, as shown in Figure 8.

    Figure 8 Setting extension connection attributes of DFCU 0

  14. Click OK. The configuration of the extension connection attributes of the DFCU is complete.
  15. Repeat 8 through 14 to configure DFCU 1.

    CAUTION:

    Cascading Tributary among the Extended Connection Attributes on the tab page of DFCU 1 should be set to B.
