Configuring the GDPUP

This describes how to add a GDPUP in the GMPS/GEPS and configure its attributes.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalOptional. The GDPUP is configured only when the BSC is configured with the built-in PCU.

Prerequisite

The GMPS/GEPS has an idle slot to hold the GDPUP.

Preparation

NOTE:

The following procedure takes how to configure a GDPUP in the GMPS as an example.

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the BSC6000 root node.
  2. Choose Configure BSC Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure BSC Attributes dialog box

  3. On the Basic Data tab page, set PCUType to Inner.

    NOTE:

    On the LMT, PCUType is set to Inner by default.

  4. On the BSC6000 Local Maintenance Terminal, click the BSC Device Panel tab. Select a cabinet number from the Current Cabinet drop-down list.
  5. Right-click an idle slot among slots 8-13 holding the GMPS.
  6. Choose Add Board from the shortcut menu. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Add Board dialog box

  7. Set FuncType to Other board and Board Type to GDPUP and then click Finish. The addition of the GDPUP is complete.
  8. Right-click GDPUP, and choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 3.

    Figure 3 Configure Board Attributes dialog box

  9. Set Load Key by referring to Table 1.
  10. Click Finish. The configuration of the GDPUP is complete.
