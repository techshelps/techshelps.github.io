Configuring the SS7 Signaling Points

This describes how to configure the SS7 signaling points of the BSC. The OPC and the DPC must be configured. If there is no direct physical link between the OPC and the DPC, you need to configure Signaling Transfer Points (STPs).

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalMandatory
CAUTION:
  • The OPC must be configured before the DPC is configured.
  • The OPC and the DPC must be configured before the STP is configured.
  • The names and codes of signaling points must be unique.
  • The BSC can be configured with up to four OPCs, 32 DPCs, and 32 STPs.

Prerequisite

None.

Procedure

  1. Configure the SS7 OPC.
    1. On the BSC6000 Local Maintenance Terminal, right-click the BSC6000 root node.

    2. Choose Configure SS7 Signaling Point from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Configure SS7 Signaling Point dialog box (1)

    3. On the OPC tab page, click Add OPC. A dialog box is displayed, as shown in Figure 2.

      Figure 2 Add OPC dialog box

      NOTE:
      If an OPC already exists, OPC Name and Encoding Scheme of the OPC are displayed by default in the dialog box shown in Figure 2.

    4. Set the parameters of the OPC by referring to Table 1.
    5. Click OK to return to the dialog box shown in Figure 1.
  2. Configure the SS7 DPC.
    1. In the dialog box shown in Figure 1, click the DPC tab to display the tab page, as shown in Figure 3.

      Figure 3 Configure SS7 Signaling Point dialog box (2)

    2. Click Add DPC. A dialog box is displayed, as shown in Figure 4.

      Figure 4 Add DPC dialog box

      NOTE:

      If there is no direct physical link between the original signaling point and the destination signaling point, you need to configure STPs. In the dialog box shown in Figure 4, set Using STP to Yes.

    3. Set the parameters of the DPC by referring to Table 1.
    4. Click OK to return to the dialog box shown in Figure 1.
  3. Check whether there is a direct physical link between the OPC and the DPC.

    If...

    Then...

    There is a direct physical link between the OPC and the DPC,

    The task is complete.

    There is no direct physical link between the OPC and the DPC,

    Go to 4.
  4. Configure the STP.
    1. In the dialog box shown in Figure 1, click the STP tab to display the tab page, as shown in Figure 5.

      Figure 5 Configure SS7 Signaling Point dialog box (3)

    2. Click Add STP. A dialog box is displayed, as shown in Figure 6.

      Figure 6 Add STP dialog box

    3. Set the parameters of the STP by referring to Table 2.
    4. Click OK to return to the dialog box shown in Figure 1.
  5. Click Finish. The configuration of the SS7 signaling points is complete.
