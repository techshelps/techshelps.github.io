Configuring the Remote NSVL

This describes how to configure the remote NSVL (NSVL on the SGSN side) in Gb over IP mode. The remote NSVL is responsible for the data transmission between the BSC and the SGSN.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalOptional. The remote NSVL is configured only when the Gb interface uses the IP transmission mode.

Prerequisites

  • The NSE is configured. For details, see Configuring the NSE.
  • The GFGUG is configured. For details, see Configuring the GFGUG (Gb over IP).

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the BSC6000 root node.
  2. Choose Configure PCU Data > Configure Remote NSVL from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure Remote NSVL dialog box

  3. Click Add. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Add Remote NSVL dialog box

  4. Set the parameters by referring to Table 1.
  5. Click OK to return to the dialog box, as shown in Figure 1.
  6. Click Finish. The configuration of the remote NSVL on the Gb interface is complete.
