Configuring the BTS Clock

This describes how to configure the clock for the BTS in IP transmission mode on the BSC side.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalMandatory. When the BTS uses the IP transmission mode, you need to configure the IP clock for the BTS.
NOTE:

The clock signals of the BTS in IP transmission mode are provided by the IP clock server instead of being extracted from the BSC.

Prerequisite

The BTS in IP transmission mode is configured.

Reference Clock Source Type of the BTS should be set to IP Time. For details, see Configuring the BTS Clock Mode.

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the BSC6000 root node.
  2. Choose Configure BSC Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure BSC Attributes dialog box 1

  3. Click the IP Transfers tab to display the tab page, as shown in Figure 2.

    Figure 2 Configure BSC Attributes dialog box 2

  4. Set Clock Server IP by referring to Table 1.
  5. Click Finish. The configuration of the BTS clock is complete.
