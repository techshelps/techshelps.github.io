Configuring the External PCU

This describes how to configure the GEIUP/GOIUP and links on the Pb interface for the communication between the BSC and the PCU.

  1. Configuring the GEIUP/GOIUP
    This describes how to add a GEIUP/GOIUP in the GMPS/GEPS and configure its attributes.
  2. Configuring Links on the Pb Interface
    This describes how to add a PCU and configure E1 connections and signaling links on the Pb interface. These E1 connections and signaling links are responsible for the communication between the BSC and the PCU.
