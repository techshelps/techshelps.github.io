Configuring the TOP Switching Parameters of 3900 Series BTSs

This describes how to configure the TDM over Packet (TOP) switching parameters on the backplane to share the E1/T1 timeslots between the GSM network and the UMTS network. That is, the E1/T1 timeslots in the GSM and UMTS networks can be shared.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalOptional

Prerequisite

Working Standard of the BTS is GSM&UMTS. The TRX board types are MRRU and MRFU.

The DBS3900 GSM, BTS3900 GSM, BTS3900A GSM, DBS3036, BTS3036, and BTS3036A support the TOP switching parameters.

Preparation

After the TOP switching parameters are configured, some E1/T1 timeslots on the GTMU panel can be used by the UMTS network through the virtual E1/T1 port on the backplane. Likewise, the E1/T1 timeslots for the UMTS network can also be used by the GSM network through the virtual E1/T1 port.
NOTE:

The following procedure takes how to configure the TOP switching parameters on the DBS3900 GSM as an example.

TOP Board Slot No.

Slot number configured for the UMTS controlling board that shares the E1/T1 timeslots with the GTMU.

7

Port Type
The port has two types: Out Port and In Port.
  • Out Port indicates that the E1/T1 port can be used for other boards during transmission. That is, the timeslots on the E1/T1 port are provided to other boards through the port on the backplane.
  • In Port indicates that the E1/T1 port is used for the current board during transmission. That is, the timeslots can be obtained from other boards and then used by the current board through the port on the backplane.

When the GSM uses certain E1/T1 timeslots of the UMTS, the Port Type of the GSM is set to In Port.

When the UMTS uses certain E1/T1 timeslots of the GSM, the Port Type of the GSM is set to Out Port.

Out Port

Switch Source E1/T1 Port No.
Number of the E1/T1 port on the GTMU panel. The value range is 0 to 7.
  • When Port Type is Out Port, this parameter indicates the E1/T1 port from which the timeslot is switched to the virtual E1/T1 port on the backplane.
  • When Port Type is In Port, use the default value 255 instead of configuring this parameter.

2

Timeslot Mask
Numbers of E1/T1 timeslots shared by the GSM and UMTS networks.
  • In T1 transmission mode, only timeslots 0 to 24 can be used.
  • In E1 transmission mode, timeslots 0 to 31 can be used.

16, 17, 18, and 19

Procedure

  1. On the Management Tree tab page of the BSC6000 Local Maintenance Terminal, right-click a BTS and then choose Configure Site Board Attributes from the shortcut menu. In the displayed dialog box, click Configure Site Board Attributes. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Site Device Attributes dialog box

  2. Click TOP Switch Parameters. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Set TOP Attributes dialog box

  3. Click Add Record. A dialog box is displayed, as shown in Figure 3.

    Figure 3 Add Top Para dialog box (1)

  4. Set the parameters by referring to Table 1. To configure the timeslot mask, click the TimeSlot Mask text box and select the related timeslot on the displayed page shown in Figure 4.

    Figure 4 Add Top Para dialog box (2)

  5. Repeat 3 through 4 to set the TOP switching parameters for more E1/T1 ports.
  6. Click OK to return to the dialog box, as shown in Figure 2.
  7. Click OK to return to Figure 1. Click OK to return to the upper-level menu. Then, click Finish.
