Procedure for Initial Configuration of a BTS3900E GSM

This describes the procedure for the initial configuration of a BTS3900E GSM. The procedure consists of obtaining data before configuration, adding a BTS, and adding other optional features of a BTS.

  1. Before initial configuration, obtain the BTS data required for the configuration. For details, see Data to Be Negotiated and Planned for BTS3900E GSM Initial Configuration.
  2. Add the BTS3900E GSM on the LMT.

    During the initial configuration, the involved complex substeps are listed as follows:
    • For details about how to add a BTS module, see Adding BTS Modules.
    • For details about how to add the RXU link and RF module (in star cascaded mode), see Adding RXU Links and RF Modules (Cascaded in Star Mode).
    • For details about how to add the RXU link and RF module (in link cascaded mode), see Adding RXU Links and RF Modules (Cascaded in Link Mode).
    • For details about how to set antenna attributes, see Setting the Attributes of the Antenna.
    • For details about how to configure the neighboring cell relation, see Configuring the Reciprocal Relations Between Cells.
    • For details about the reference to TRX power configuration, see Reference for the Configuration of TRX Power.
  3. For details about how to configure other optional features of the BTS based on the actual situations, see Configuring the Optional BTS Attributes.
  4. For some typical configuration scenarios, do as follows:
    • For details about how to configure BTS3900E GSM in O6 mode, see Configuring a BTS3900E GSM in O6 Mode.
    • For details about how to configure the BTS3900E GSM in S2/2/2 mode, see Configuring the BTS3900E GSM in S2/2/2 Mode.
