Configuring the GGCU

This describes how to configure the attributes of the GGCUs through the LMT. The GGCUs are displayed in slots 12 and 13 of the GMPS by default on the LMT.

ScenarioBSC initial configuration
Mandatory/OptionalMandatory

Prerequisite

None.

Preparation

Table 1 Example of the data negotiated and planned for configuring the GGCU

Parameter

Example

Source

Load Key

Server

BSC internal planning

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click GGCU.
  2. Choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure Board Attributes dialog box

  3. Set Load Key to Server.
  4. Click Finish. The configuration of the GGCU is complete.
