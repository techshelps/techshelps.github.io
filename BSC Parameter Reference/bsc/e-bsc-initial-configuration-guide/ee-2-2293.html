Configuring the BSC (BM/TC Combined)

This describes the procedures for BSC initial configuration on the LMT using the minimum configuration data when the BM and TC are configured in the same subrack.

  1. Configuring the BSC Global Data (BM/TC Combined)
    This describes how to configure the BSC global data when the BM and TC are configured in the same subrack. The BSC global data consists of BSC attributes, SS7 signaling point data, and the relation between the signaling points of the GMPS and GEPS.
  2. Configuring the BSC Devices (BM/TC Combined)
    This describes how to configure BSC devices on the LMT when the BM and TC are configured in the same subrack.
  3. Configuring the BSC Links (BM/TC Combined)
    This describes how to configure BSC links on the A interface for the signaling transmission between the BSC and the MSC when the BM and the TC are configured in different subracks.
  4. Configuring Clocks for the BSC (BM/TC Combined)
    This describes how to configure clocks for the BSC when the BM and TC are configured in the same subrack. The BSC clocks are responsible for the synchronization of boards in each subrack.
