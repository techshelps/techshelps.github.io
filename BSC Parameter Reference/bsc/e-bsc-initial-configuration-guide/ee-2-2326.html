Configuring the Links on the Gb Interface (Gb over IP)

This describes how to configure the NSE, local NSVL, remote NSVL, and PTPBVC on the Gb interface for the IP transmission between the BSC and the SGSN.

  1. Configuring the SGSN Node
    This describes how to configure the SGSN node on the Gb interface and determine the management status of the SGSN.
  2. Configuring the NSE
    This describes how to configure the Network Service Entity (NSE) on the Gb interface and determine the subnetwork protocol type.
  3. Configuring the Local NSVL
    This describes how to configure the local NSVL (NSVL on the BSC side) in Gb over IP mode. The local NSVL is responsible for the data transmission between the BSC and the SGSN.
  4. Configuring the Remote NSVL
    This describes how to configure the remote NSVL (NSVL on the SGSN side) in Gb over IP mode. The remote NSVL is responsible for the data transmission between the BSC and the SGSN.
  5. Configuring the PTPBVC
    This describes how to add a PTPBVC on the Gb interface to bind the GPRS cell with the NSE.
