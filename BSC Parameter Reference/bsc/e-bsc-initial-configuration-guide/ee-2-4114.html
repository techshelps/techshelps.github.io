Configuring the SGSN Node

This describes how to configure the SGSN node on the Gb interface and determine the management status of the SGSN.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalOptional. The SGSN node is configured only when the built-in PCU is configured.

Prerequisites

  • The GXPUM is configured.
  • The built-in PCU is configured.

Preparation

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the BSC6000 root node and then choose Configure PCU Data > Configure SGSN Node from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure SGSN Node dialog box

  2. Click Add. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Add SGSN Node dialog box

  3. Set related parameters by referring to Table 1. Click OK to return to the dialog box, as shown in Figure 1.

    NOTE:

    To modify Operator Name, perform the following operations: Right-click the BSC6000 node on the Local Maintenance Terminal BSC6000, and then choose Configure BSC Attributes from the shortcut menu. In the displayed Configure BSC Attributes dialog box, modify Operator Name on the Basic Data tab page.

  4. Click Finish. The configuration of the SGSN node is complete.
