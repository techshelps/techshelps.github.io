Configuring the BTS ABIS MUX

This describes how to configure ABIS MUX to improve the IP transmission efficiency over the Abis interface.

ScenarioBSC initial configuration
Mandatory/OptionalOptional. The BTS ABIS MUX should be configured when the ABIS MUX technique is used to improve the IP transmission efficiency over the Abis interface.
NOTE:

The BSC ABIS MUX function is available for use only when the GFGUB board is configured with the IP-based BTS and the BTS is configured with the BTS ABIS MUX function.

Prerequisite

The GFGUB is configured with an IP-based BTS. In addition, the Abis MUX Global Enable Switch parameter of the IP-based BTS is set to Open.

Procedure

  1. Right-click an IP-based BTS on the Management Tree tab page and then choose Configure BTS ABIS MUX from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure BTS ABIS MUX dialog box

  2. Click Add. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Add BTS ABIS MUX dialog box

  3. Set the parameters by referring to Table 1.
  4. Click OK to return to the dialog box, as shown in Figure 1.
  5. Click Finish. The configuration of the BTS ABIS MUX is complete.
