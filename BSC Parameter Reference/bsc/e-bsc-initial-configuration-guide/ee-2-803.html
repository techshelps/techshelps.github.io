Configuring the GXPUM

This describes how to add GXPUMs in slots 0 and 1 of the GMPS/GEPS and configure their attributes.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalMandatory

Prerequisite

None.

Preparation

The following procedure takes how to add a GXPUM in the GMPS as an example.

Table 1 Example of the data negotiated and planned for configuring the GXPUM

Parameter

Example

Source

Load Key

Server

BSC internal planning

Procedure

  1. On the BSC6000 Local Maintenance Terminal, click the BSC Device Panel tab. Select the cabinet where the GMPS is located from the Current Cabinet drop-down list.
  2. Right-click an idle slot among slots 0-1 in the GMPS, and then choose Add Board from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Add Board dialog box

  3. Set FuncType to Other board and Board Type to GXPUM.
  4. Click Finish. The addition of the GXPUM is complete.
  5. Right-click GXPUM, and then choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Configure Board Attributes dialog box (GXPUM)

  6. Set Load Key by referring to Table 1.
  7. Click Next. A dialog box is displayed, as shown in Figure 3.

    Figure 3 Configuring the switch of the GXPUM port

  8. Enable the alarm reporting function of ports as required.
  9. Click Next. A dialog box is displayed, as shown in Figure 4.

    Figure 4 Configuring attributes of the GXPUM port

  10. When the GXPUM adopts the virtual port mechanism, the related parameters should be set. Otherwise, skip this step.

    NOTE:

    The panel of the GXPUM is connected to external network, which is used to process the short message service. The parameters MAC Address of External Network and IP Address of External Network should be configured in the GXPUM, thus separating the internal network from the external network.

  11. Click Finish. The configuration of the GXPUM is complete.
