Configuring a BTS3900/BTS3900A

This describes how to configure a BTS3900/BTS3900A. The procedure and precautions for some complicated configurations are also described.

  • Adding BTS Modules
    This describes how to add modules other than RF modules for the DBS3900, BTS3900, and BTS3900A, including the APMU, DTCU, FMUA, FMU, DEMU, and GATM.
  • Adding the RXU Chain and RF Module (Star Topology)
    This describes how to add the RXU chain and RF module when the BBU and the RF module are connected in star topology.
  • Adding the RXU Chain and RF Module (Chain Topology)
    This describes how to add the RXU chain and RF module when the BBU and the RF module are connected in a chain topology.
  • Adding the RXU Chain and RF Module (Ring Topology)
    This describes how to add the RXU chain and RF module when the BBU and the RF module are connected in a ring topology.
  • Setting Antenna Attributes
    This describes how to set antenna attributes, including the antenna TX/RX mode and TMA. If the BTS is configured with the GATM and RET antenna, you also need to set the attributes of the RET antenna.
  • Configuring TRX Receive and Transmit Modes
    This describes how to configure TRX receive and transmit modes. This ensures that the configured TRX receive and transmit modes are in accordance with the actual RF cable connections and antenna cable connections of the BTS.
