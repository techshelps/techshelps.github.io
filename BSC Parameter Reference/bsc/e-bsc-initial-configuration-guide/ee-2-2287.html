Configuring the BSC (BM/TC Separated)

This describes how to configure the BSC global data, devices, links, and clock on the BSC6000 Local Maintenance Terminal when the BM and TC are configured in different subracks.

  1. Configuring the BSC Global Data (BM/TC Separated)
    This describes how to configure the BSC global data when the BM and TC are configured in different subracks. The BSC global data consists of BSC attributes, SS7 signaling point data, and the relation between the GMPS/GEPS and OSP.
  2. Configuring the BSC Devices (BM/TC Separated)
    This describes how to configure the BSC devices on the LMT when the BM and TC are configured in different subracks.
  3. Configuring the BSC Links (BM/TC Separated)
    This describes how to configure the BSC links on the Ater and A interfaces when the BM and TC are configured in different subracks. These links are responsible for the signaling transmission within the BSC and between the BSC and other NEs. BSC links are the links on the Ater interface and A interface.
  4. Configuring Clocks for the BSC (BM/TC Separated)
    This describes how to configure clocks for the BSC when the BM and TC are configured in different subracks. The BSC clocks are responsible for the synchronization of boards in each subrack.
