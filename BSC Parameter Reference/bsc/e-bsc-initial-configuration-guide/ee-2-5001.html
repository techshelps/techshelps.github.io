Procedure for DBS3900 Initial Configuration

This describes the procedure for DBS3900 initial configuration. The procedure involves obtaining data before configuration, adding a BTS, and adding other optional features of a BTS.

  1. Before initial configuration, obtain the BTS data required for the configuration. For details, see Data to Be Negotiated and Planned for DBS3900 Initial Configuration.
  2. Add the DBS3900 on the LMT.

    During the initial configuration, the involved complex substeps are listed as follows:
    • For details about how to add a BTS module, see Adding BTS Modules.
    • For details about how to add the RXU chain and RF module (in star topology), see Adding the RXU Chain and RF Module (Star Topology).
    • For details about how to add the RXU chain and RF module (in chain topology), see Adding the RXU Chain and RF Module (Chain Topology).
    • For details about how to add the RXU chain and RF module (in ring topology), see Adding the RXU Chain and RF Module (Ring Topology).
    • For details about how to set antenna attributes, see Setting Antenna Attributes.
    • For details about how to configure the TRX receive and transmit mode, see Configuring TRX Receive and Transmit Modes.
  3. For details about how to configure other optional features of the BTS based on customer requirements, see Configuring the Optional BTS Attributes. The configurations of optional features are describes as follows:
    • For details about how to configure the neighboring cell relation, see Configuring the Reciprocal Relations Between Cells.
    • For details about how to configure a slave chain for the BTS, see Configuring the Slave Chain.
    • For details about how to configure the customized alarm, see Configuring Customized Alarms of the Base Station.
    • For details about how to configure the TOP switching parameters, see Configuring the TOP Switching Parameters of 3900 Series BTSs.
  4. In addition, some configuration references are available.
    • For details about the reference to TRX power configuration, see Reference for TRX Power Configuration.
    • For details about customized alarm ports of the DBS3900. For details, see Customized Alarm Ports of the DBS3900/BTS3900/BTS3900A Base Station.
  5. For typical configuration scenarios, the following section is listed for reference.
    • For details about how to configure the DBS3900 in S2/2/2, see Configuring the DBS3900 in S2/2/2 Mode.
