Configuring the Subrack-OPC Mapping

This describes how to assign an OPC to the GMPS/GEPS to establish the mapping between the subrack and the OPC.

ScenarioBSC initial configuration
Mandatory/OptionalMandatory
NOTE:

The GMPS is displayed by default on the LMT. You need to configure its OPC on the LMT. For the GEPS, you need to configure it and its OPC on the LMT. The GTCS does not require an OPC.

Prerequisite

The SS7 OPC is already configured.

Preparation

Table 1 Example of the data negotiated and planned for configuring the subrack-OPC mapping

Parameter

Example

Source

Subrack No.

0

BSC internal planning

OPC

B1

Network planning

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the BSC6000 root node.

  2. Choose Configure SS7 Signaling Point from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure Subrack-OSP Mapping dialog box

  3. Set the OPC for the GMPS by referring to Table 1.
  4. Click Finish. The configuration of the GMPS-OSP mapping is complete.
