Configuring Service Vlan Mapping

This describes how to configure the Vlan mapping on the BSC side.

ScenarioBSC initial configuration
Mandatory/OptionalOptional
NOTE:

The function of Service Vlan mapping is automatically generated when an IP base station is configured. This function can be modified but cannot be added or deleted. The TDM base station does not support this function.

When IP transmission mode is configured for a BTS, Vlan Flag, VLan Id, PHB, and DSCP can be modified in this function.

When the IP PM is configured on the BTS, DSCP for the same service type on the BTS and BSC associated with the Vlan mapping must be the same.

Prerequisite

The BTS in IP transmission mode is configured.

Procedure

  1. In the Management Tree of the BSC6000 Local Maintenance Terminal, right-click the corresponding IP base station.
  2. Choose Configure Service Vlan Map from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure ServiceVlan Map dialog box

  3. Click Modify. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Modify ServiceVlan Map dialog box

  4. Set Vlan Flag, VLan Id, PHB, and DSCP.

    NOTE:

    After modifying the DSCP of the static PDCH in dynamic mode, you need to re-create a PDCH by re-activating the cell of the PDCH so that the modification can take effect.

  5. Click OK to return to the dialog box, as shown in Figure 1.
  6. Click Finish. The configuration of the Service Vlan mapping is complete.
