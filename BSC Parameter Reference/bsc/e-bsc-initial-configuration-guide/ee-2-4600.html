BTS3900E GSM Initial Configuration Guide

This describes how to configure a BTS3900E GSM for the BSC. This configuration enables the radio transmission and reception functionality of the BTS and fulfills the radio coverage of the cells. The configuration also enables the BSC to perform centralized control and management of the BTS and allocate radio resources for the BTS.

  • Procedure for Initial Configuration of a BTS3900E GSM
    This describes the procedure for the initial configuration of a BTS3900E GSM. The procedure consists of obtaining data before configuration, adding a BTS, and adding other optional features of a BTS.
  • Data to Be Negotiated and Planned for BTS3900E GSM Initial Configuration
    This describes the data to be negotiated and planned for the initial configuration of a BTS3900E GSM. Before configuring a BTS3900E GSM, you need to obtain the data of the BTS. The data consists of site configuration, site attributes, RXU link attributes, board attributes, cell attributes, cell broadcast parameters, and TRX attributes.
  • Adding a BTS3900E GSM (TDM)
    This describes how to add a BTS3900E GSM for the BSC and configure its attributes, logical TRXs, and cell attributes.
  • Adding a BTS3900E GSM (IP)
    This describes how to add a BTS3900E GSM in IP transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, TRX attributes, BTS Clock and BTS Route.
  • Adding a BTS3900E GSM (HDLC/HUB)
    This describes how to add a BTS3900E GSM for the BSC and configure its attributes, logical TRXs, and cell attributes.
  • Configuring a BTS3900E GSM
    This describes how to configure a BTS3900E GSM. The procedure and precautions for some complex configurations during the addition of 3900 series base stations are described.
  • Typical Scenarios of Configuring a BTS3900E GSM
    This describe the typical scenarios of configuring a BTS3900E GSM. The typical scenarios consist of configuring the O6 of a BTS3900E GSM and configuring the S2/2/2 of a BTS3900E GSM.
  • Reference for the Configuration of TRX Power
    TRX power configuration specifications serve as only the reference for on-site configurations. You can adjust configuration data based on the actual situation.
