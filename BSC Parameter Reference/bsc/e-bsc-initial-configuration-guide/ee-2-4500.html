Common BTS Initial Configuration Guide

This describes how to configure BTS3X series and double-transceiver BTSs for the BSC. This configuration enables the radio transmission and reception functionality of the BTSs and fulfills the radio coverage of the cells. The configuration also enables the BSC to perform centralized control and management of the BTSs and allocate radio resources to the BTSs.

  • Adding a Common BTS (TDM Transmission Mode)
    This describes how to add a BTS of the BTS 3X family and a BTS of the double-transceiver BTS family in TDM transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, and TRX attributes.
  • Adding a Common BTS (IP Transmission Mode)
    This describes how to add a common BTS in IP transmission mode and configure the IP clock and route for the BTS.
  • Adding a Common BTS (HDLC/HUB Transmission Mode)
    This describes how to add a common BTS in HDLC/HUB transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, and TRX attributes.
  • Configuring the DFCU/DFCB
    This describes how to configure the DFCU and the DFCB. The DFCU/DFCB can provide the four-in-one and six-in-one outputs of TRXs.
  • Typical Scenarios of BTS Configuration
    This describes the typical scenarios of BTS initial configuration on the BSC side.
