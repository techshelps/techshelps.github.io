Configuring the BSC Devices (BM/TC Combined)

This describes how to configure BSC devices on the LMT when the BM and TC are configured in the same subrack.

  1. Adding a GBSR Cabinet
    This describes how to add a GBSR cabinet and configure its attributes through the LMT. In BM/TC combined mode, the GBSR cabinet must be configured on the BSC side.
  2. Adding a GEPS
    This describes how to add a GEPS and configure its attributes on the LMT.
  3. Configuring a Power Distribution Box
    This describes how to add a power distribution box and configure its attributes through the LMT.
  4. Configuring the BSC Boards
    This describes how to add BSC boards or configure the attributes of the BSC boards. When the BM and the TC are configured in the same subrack, the GDPUX, GEIUA/GOIUA, GEIUB/GOIUB, GEHUB, GFGUB/ GOGUB, GXPUM, GXPUT, GGCU, GTNU, and GSCU must be configured.
  5. Configuring the BSC EAC
    This describes how to configure the EAC for the BSC cabinet. The EAC gathers Boolean values, analog values, and alarm threshold information and reports them to the LMT.
  6. Configuring the Subrack Communication Link
    This describes how to configure the communication links between the GMPS and the GEPS or between GEPSs. The links are responsible for the TDM switching between subracks.
