BSC Initial Configuration Guide (Based on LMT)

Purpose

This document describes how to perform the BSC initial configuration on the LMT. The contents of the document include task scenarios, working principles, and operating procedures of the BSC initial configuration. This document can be used as a guide to configure data during deployment.

Intended Audience

This document is intended for:

  • Field engineers
  • Network operators
  • System engineers

Organization

  • Changes in BSC Initial Configuration Guide (Based on LMT)
    This provides the changes of the BSC Initial Configuration Guide (Based on LMT).
  • Definition of BSC Initial Configuration
    BSC initial configuration refers to the configuration performed after BSC installation. BSC initial configuration is based on the hardware configuration, network planning, and data negotiated with other devices.
  • Scenarios of BSC Initial Configuration
    The procedure for BSC initial configuration varies with scenarios.
  • Configuring the BSC (BM/TC Separated)
    This describes how to configure the BSC global data, devices, links, and clock on the BSC6000 Local Maintenance Terminal when the BM and TC are configured in different subracks.
  • Configuring the BSC (BM/TC Combined)
    This describes the procedures for BSC initial configuration on the LMT using the minimum configuration data when the BM and TC are configured in the same subrack.
  • Configuring the BSC (A over IP)
    This describes the procedures for BSC initial configuration on the LMT using the minimum configuration data when the A over IP mode is used.
  • Configuring the Built-in PCU
    When the built-in PCU is configured, the related BSC boards and Gb interface links should be configured on the basis of the transmission mode (FR or IP) used on the Gb interface. These boards and links are responsible for the communication between the BSC and the SGSN.
  • Configuring the External PCU
    This describes how to configure the GEIUP/GOIUP and links on the Pb interface for the communication between the BSC and the PCU.
  • Configuring the BTS
    This describes how to configure a BTS for the BSC. This configuration enables the radio transmission and reception functionality of the BTS and fulfills the radio coverage of the cells. The configuration also enables the BSC to perform centralized control and management of the BTS and allocate radio resources for the BTS.
  • Principles of Configuring the BSC Data
    This describes the principles of configuring the BSC data.
