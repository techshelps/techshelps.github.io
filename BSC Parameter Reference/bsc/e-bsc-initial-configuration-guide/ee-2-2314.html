Configuring Clocks for the BSC (A over IP)

This describes how to configure clocks for the BSC in A over IP mode. The BSC clocks are responsible for the synchronization of boards in each subrack.

  1. Configuring the System Clock
    This describes how to configure the system clock, that is, the input clock for the GGCU. In A over IP mode, the system clock for the BSC can be extracted only by the GGCU from the BITS clock. The GGCU processes the input clock and generates the 8 kHz clock, which is used as the transmission synchronization clock and the time synchronization clock for the boards in the GMPS or GEPS.
  2. Configuring the 8 kHz Reference Clock for the GMPS
    This describes how to configure the 8 kHz clock provided by the backplane of the GGCU as the 8 kHz reference clock for the GMPS. The 8 kHz reference clock serves as the synchronization clock for the boards in the GMPS.
  3. Configuring the 8K Reference Clock for the GEPS
    This describes how to configure the 8 kHz clock provided by the GGCU as the 8 kHz reference clock for the GEPS. The 8 kHz reference clock serves as the synchronization clock for the boards in the GEPS.
