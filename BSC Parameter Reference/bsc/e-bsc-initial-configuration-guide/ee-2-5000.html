DBS3900 Initial Configuration Guide

This describes how to configure a DBS3900 for the BSC. This configuration enables the radio transmission and reception functionality of the BTS and fulfills radio coverage of the cells. The configuration also enables the BSC to perform centralized control and management of the BTS and allocate radio resources for the BTS.

  • Procedure for DBS3900 Initial Configuration
    This describes the procedure for DBS3900 initial configuration. The procedure involves obtaining data before configuration, adding a BTS, and adding other optional features of a BTS.
  • Data to Be Negotiated and Planned for DBS3900 Initial Configuration
    This describes the data to be negotiated and planned for the initial configuration of a DBS3900. Before configuring a DBS3900, you need to obtain the data of the BTS, which consists of site configuration, site attributes, RXU chain attributes, board attributes, cell attributes, cell broadcast parameters, and TRX attributes.
  • Adding a DBS3900 (TDM)
    This describes how to add a DBS3900 in TDM transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, TRX attributes.
  • Adding a DBS3900 (IP)
    This describes how to add a DBS3900 in IP transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, TRX attributes, BTS clock, and route.
  • Adding a DBS3900 (HDLC/HUB)
    This describes how to add a DBS3900 in HDLC/HUB transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, TRX attributes.
  • Configuring the DBS3900
    This describes how to configure a DBS3900. The procedure and precautions for some complicated configurations are also described.
  • Typical Scenarios of Configuring a BTS
    This describes typical scenarios of configuring a BTS for the BSC.
  • Reference for TRX Power Configuration
    This provides specifications related to TRX power configuration. The data configuration, however, depends on actual applications.
  • Customized Alarm Ports of the DBS3900/BTS3900/BTS3900A Base Station
    This describes the relation between the BTS internal alarm IDs and the environment alarm ports/cables of each monitoring device. The monitoring devices are the EMUA, UPEU, UEIU, APMI, and FMUA.
