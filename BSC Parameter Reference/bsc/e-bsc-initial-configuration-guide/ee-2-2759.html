Adding a BTS

This describes how to add a BTS3900B GSM in IP transmission mode for the BSC, and how to configure the site device attributes, cell basic attributes, and TRX attributes.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalMandatory
NOTE:
  • The BTS in IP transmission mode does not support the Ring Mode , Config AbisByPass or Backup OML function, and does not support cascading configuration.
  • The clock of BTSs in IP transmission mode is provided by the IP clock server instead of being extracted from the BSC.
  • A BTS in IP transmission mode supports two transmission types. One transmission type is based on the device IP address. The other transmission type is based on the port IP address. Select a specific transmission mode by setting BTS Communication Type. The LMT uses the port IP address of a BTS for transmission by default. A BTS3900B GSM BTS uses the port IP address for communication.
  • The IP address of a BTS3900B GSM can be configured on the same network segment as the port IP address of the BSC but cannot be configured on the same network segment as the device IP address of the BSC.
  • A BTS3900B GSM BTS does not support the configurations of timeslot monitoring and slave chains. In addition, this BTS does not support the switchover to the configuration of the separate mode.

Prerequisites

  • The GFGUB/GOGUB is configured. If a pair of active and standby GFGUBs/GOGUBs are configured, their device IP addresses and port IP addresses are configured. For details, see Configuring the GFGUB/GOGUB.
  • The GXPUM/GXPUT is configured for the GMPS or GEPS where the GFGUB/GOGUB is located. For details, see Configuring the GXPUM and Configuring the GXPUT.

Procedure

  • Through GUI
    1. On the BSC6000 Local Maintenance Terminal, right-click GFGUB in the GEPS.
    2. Choose Add Site from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Add Site dialog box (1)

    3. Click Add Site. A dialog box is displayed, as shown in Figure 2. Set related parameters by referring to Table 1.

      Figure 2 Add New Site dialog box

    4. Set the parameters by referring to Table 1.

      NOTE:
      The following IP addresses are not allowed to be configured:
      • IP addresses as all zeros or all 255s on network segments 80.x.x.x, 82.x.x.x, and 90.x.x.x
      • Loopback IP address 127.x.x.x
      • Multicast IP addresses of class D, that is, IP addresses from 224.0.0.0 to 239.255.255.255
      • Reserved IP addresses of class E, that is, IP addresses from 240.0.0.0 to 247.255.255.255
      • Reserved IP address starting from 248.***.***.***
      • Network addresses and broadcast addresses of class A, class B, and class C

    5. Click OK. A dialog box is displayed, as shown in Figure 3.

      Figure 3 Add Site dialog box (2)

    6. Click Next. A dialog box is displayed, as shown in Figure 4.

      Figure 4 Add Site dialog box (3)

    7. Select a BTS, and then click Add Cell. A dialog box is displayed, as shown in Figure 5.

      Figure 5 Add New Cell dialog box

    8. Set the parameters by referring to Table 1.

      NOTE:

      To add multiple cells at a time, set Number of Cells. The added cells are named in the format of current Cell Name plus sequence number.

    9. Click OK. A dialog box is displayed, as shown in Figure 6.

      Figure 6 Add Site dialog box (4)

    10. Click Next. A dialog box is displayed, as shown in Figure 7.

      Figure 7 Add Site dialog box (5)

    11. Click Site Attributes. A dialog box is displayed, as shown in Figure 8.

      Figure 8 Site Device Attributes dialog box

    12. Right-click the BTS3900B area box, and then choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 9.

      Figure 9 Basic Attributes of Site Board dialog box

    13. Click the FE port parameter of BTS tab to display the tab page, as shown in Figure 10.

      Figure 10 FE port parameter of BTS tab page

    14. Set the parameters by referring to Table 1.

      NOTE:

      BTS3012, BTS3012AE, BTS3012_II, DBS3900, BTS3900, BTS3900A, BTS3900B, and BTS3900E do not support that Muplex Mode is set to HALF.

      The parameter Peer Port IP address index specifies the index of the valid IP address for a BSC port. The BSC supports the setting of multiple IP address under one Port No.. The Peer Port IP address index maps the valid IP address of the BSC port. After the value of Peer Port IP address index is specified, the BSC IP Address in the IP Interface Board Parameter of BTS tab page is generated automatically and the BSC IP Address cannot be changed.

    15. Click the IP Interface Board Parameter of BTS tab to display the tab page, as shown in Figure 11.

      Figure 11 IP Interface Board Parameter of the BTS tab page

    16. Set the parameters of the BTS3900B by referring to Table 1.

      NOTE:

      You can obtain the BTS Interface Board Bar Code by running the DSP BTSESN command.

    17. Click OK to return to the dialog box, as shown in Figure 8.
    18. Right-click the BTS3900B and then choose Binding logical TRX from the shortcut menu. A dialog box is displayed, as shown in Figure 12.

      Figure 12 Binding logical TRX dialog box

    19. Set the parameters by referring to Table 1.
    20. Click OK to return to the dialog box, as shown in Figure 8.
    21. Click OK in Figure 8 to return to the dialog box, as shown in Figure 7.
    22. Click Next. A dialog box is displayed, as shown in Figure 13.

      Figure 13 Add Site dialog box (6)

    23. Click Cell Attributes. A dialog box is displayed, as shown in Figure 14.

      Figure 14 Set Cell Attributes dialog box

    24. Set the basic attributes of the cell by referring to Table 1.

      NOTE:
      • For two cells, the values of at least one of the following parameters should be different: MCC, MNC, LAC, and CI.
      • If you select GPRS Support, the BSC must be configured with the built-in PCU. If the BTS does not support the GPRS function or the GPRS services are unavailable in the cell, do not select GPRS Support.

    25. In the TRX Attributes group box, select the allocated TRXs, as shown in Figure 15.

      Figure 15 TRX Attributes group box

    26. Click Frequency Config. A dialog box is displayed, as shown in Figure 16. Then, select a frequency.

      Figure 16 Set Cell Frequency dialog box

    27. Click OK to return to the dialog box, as shown in Figure 14.
    28. Click TRX Config. A dialog box is displayed, as shown in Figure 17.

      Figure 17 Configure TRX Attributes dialog box

    29. Move a frequency corresponding to the TRX from Available Frequencies to Assigned Frequencies.
    30. Click the Channel Attributes tab. A tab page is displayed, as shown in Figure 18. Then, set the channel types corresponding to the eight channel numbers.

      Figure 18 Channel Attributes tab page

    31. Click the Device Attributes tab. A dialog box is displayed, as shown in Figure 19.

      Figure 19 Device Attributes tab page

      NOTE:

      The BTS3900B supports seven power levels ranging from 0 to 6. The difference between two adjacent power levels is 2 dB.

    32. Set Power Level and Power Type.
    33. Click OK to return to the Set Cell Attributes dialog box.
    34. Click OK to return to the dialog box, as shown in Figure 13.
    35. Click Finish. The configuration of the BTS is complete.
    36. On the Management Tree tab page of the BSC6000 Local Maintenance Terminal, right-click the target BTS and choose Site Operation > Set Site Active State from the shortcut menu. A dialog box is displayed, as shown in Figure 20.

      Figure 20 Set Site Active State dialog box

    37. Under Site Active State, select Activated. Then, click Finish to complete the BTS activation.
