Configuring the M3UA Route

This describes how to configure the M3UA route for implementing the IP transmission between the BSC and the MSC.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalMandatory

Prerequisites

  • The DSP for the M3UA entity is configured. For details, see Configuring the M3UA Entities.
  • The M3UA link set is configured.

Preparation

Table 1 Example of the data negotiated and planned for configuring the M3UA route

Example

Source

MSC001

Negotiation with the peer

BSC internal planning

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the BSC6000 root node.
  2. Choose Configure M3UA Data > Configure M3UA Entity from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure M3UA Route dialog box

  3. Click Add. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Add M3UA Route dialog box

  4. Set the parameters of the M3UA route by referring to Table 1.
  5. Click OK to return to the dialog box, as shown in Figure 1.
  6. Click OK. The M3UA route is configured.
