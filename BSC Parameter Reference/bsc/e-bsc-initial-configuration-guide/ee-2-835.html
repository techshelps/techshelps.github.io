Configuring the 3G External Cell

This describes how to add a 3G external cell for the BSC and set the attributes of the cell.

ScenarioBSC initial configuration
Mandatory/OptionalOptional. A 3G external cell should be configured when the handover from a cell under the BSC to the 3G external cell is required.

Prerequisite

Inter-RAT Out BSC Handover Enable, Inter-RAT In BSC Handover Enable, and Inter-RAT Cell Reselection Enable are set to Yes.

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click 3G External Cells.
  2. Choose Add 3G External Cell from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Add 3G External Cell dialog box

  3. Click Add ExtCell. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Adding a 3G external cell

    NOTE:

    To add 3G external cells in batches, set the number of 3G external cells in .

  4. Set the parameters by referring to Table 1.
  5. Click OK to return to the dialog box shown in Figure 1.
  6. Click Finish. The configuration of the 3G external cell is complete.
