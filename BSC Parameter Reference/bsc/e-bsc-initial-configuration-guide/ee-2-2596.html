Configuring the Multi-Transceiver Units

This describes how to configure the multi-transceiver units for the BTS supporting the separate mode.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalOptional. The multi-transceiver units should be configured if Support Separate is selected during BTS configuration.
NOTE:
  • The Support Separate option is available only for the BTS3012, BTS3012 II, and BTS3012AE. If Support Separate is selected, you need to configure the DTRU or QTRU.
  • For the 3900 GSM BTS family, DBS3036, BTS3036, and BTS3036A, Support Separate is selected by default. The DRRU/GRRU/MRRU/XRRU/DRFU/GRFU/MRFU/XRFU can be configured. The DRRU/GRRU/MRRU/XRRU is configured on the 3900 GSM BTS family and the DBS3036, BTS3036, and BTS3036A; the DRFU/GRFU/MRFU/XRFU is configured on only the BTS3900 GSM, the BTS3900A GSM and the BTS3036, the BTS3036A; the GRRU/GRFU supports only the GSM mode; the MRRU/MRFU supports the GSM, UMTS and GSM&UMTS modes.
  • On one DTRU/DRRU/DRFU, one physical TRX maps two logical TRXs; on one QTRU/MRRU/GRFU/MRFU, one physical TRX maps six logical TRXs; on one GRRU, one physical TRX maps eight logical TRXs.
  • The TRX boards XRRU and XRFU are reserved for the 3G network.

Prerequisite

The BTS supporting the separate mode is configured.

Procedure

  • Configuring a QTRU
    1. On the BSC6000 Local Maintenance Terminal, right-click a cell under the BTS that supports the separate mode.
    2. Choose Add TRX from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Add TRX dialog box

    3. Select a BTS, and then click Set Site Device. A dialog box is displayed, as shown in Figure 2.

      Figure 2 Site Device Attributes dialog box

    4. Right-click an empty slot.
    5. Choose Add Board > QTRU from the shortcut menu.
    6. Right-click the QTRU.
    7. Choose Binding logical TRX. A dialog box is displayed, as shown in Figure 3.

      Figure 3 Binding logical TRX dialog box

    8. Set the parameters by referring to Table 1.
    9. Click OK to return to the dialog box shown in Figure 2.
    10. Right-click the DDPU.
    11. Choose Configure Board Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 4.

      Figure 4 Set Antenna Feeder Property dialog box

      NOTE:

      In the TRX Transmit and Receive Relationship area, the first column indicates the number of the DTRU/QTRU. The second column indicates the downlink transmission channel of the DTRU/QTRU. The DTRU has two transmission channels, numbered 0 and 1. The QTRU has only one six-in-one transmission channel and the value is Null. The third column indicates the selection of the transmit/receive main and diversity, and the value can be A or B.

    12. Set the parameters by referring to Table 1.
    13. Click OK to return to the dialog box shown in Figure 2.
    14. Click OK to return to the dialog box shown in Figure 1.
    15. Select a cell and click Set Cell TRX. A dialog box is displayed, as shown in Figure 5.

      Figure 5 Configure Cell TRX dialog box

    16. Click Cell Frequency. Set the frequencies by referring to Table 1, as shown in Figure 6.

      Figure 6 Set Cell Frequency dialog box

      NOTE:

      On one QTRU, the absolute value of the bandwidth difference between the highest frequency and the lowest frequency must not be greater than 12.5 Mbit/s.

    17. Click OK to return to the dialog box shown in Figure 5.
    18. Select a logical TRX of the QTRU, and then click TRX Attributes to assign frequencies for it. Repeat this step until all the six logical TRXs of the QTRU are assigned frequencies.
    19. Click OK to return to the dialog box shown in Figure 1.
    20. Click Finish. The configuration of the QTRU is complete.
  • Configure a GRRU
    1. On the BSC6000 Local Maintenance Terminal, right-click a cell under the DBS3900.
    2. Choose Add TRX from the shortcut menu. A dialog box is displayed, as shown in Figure 7.

      Figure 7 Add TRX dialog box

    3. Select a BTS, and then click Set Site Device. A dialog box is displayed, as shown in Figure 8.

      Figure 8 Site Device Attributes dialog box

    4. Right-click the RXU chain, and then choose Add RXU > Add GRRU. A dialog box is displayed, as shown in Figure 9.

      Figure 9 Add GRRU dialog box

    5. Set the parameters, and click OK. The addition of the GRRU is complete.
    6. Steps 6 through 9 are optional. Right-click the GRRU.
    7. Choose Configure RXU Attributes from the shortcut menu. A dialog box is displayed, as shown in Figure 10.

      Figure 10 Basic Attributes of Site Board dialog box

    8. Click the Configure Site Board Attributes tab. A tab page is displayed, as shown in Figure 11.

      Figure 11 Configuring the GRRU attributes

    9. Set the parameters, and click OK to return to the dialog box shown in Figure 8.
    10. Right-click the GRRU.
    11. Choose Bind Logical TRX from the shortcut menu. A dialog box is displayed, as shown in Figure 12.

      NOTE:

      The GRRU integrates the functions of the TRX and antenna. A and B are the number of antenna tributaries.

      Figure 12 Binding logical TRX dialog box

      NOTE:

      The absolute value of the difference between the number of As and the number of Bs cannot be greater than 1.

    12. Set the parameters by referring to Table 2.
    13. Click OK to return to the dialog box shown in Figure 8.
    14. Click OK to return to the dialog box shown in Figure 7.
    15. Select a cell and click Set Cell TRX. A dialog box is displayed, as shown in Figure 13.

      Figure 13 Configure Cell TRX dialog box

    16. Click Cell Frequency. Set the frequencies by referring to Table 1, as shown in Figure 14.

      Figure 14 Set Cell Frequency dialog box

      NOTE:

      On one GRRU, the absolute value of the bandwidth difference between the highest frequency and the lowest frequency must not be greater than 12.5 MHz.

    17. Click OK to return to the dialog box shown in Figure 13.
    18. Select a logical TRX of the GRRU, and then click TRX Attributes to assign frequencies for it. Repeat this step until all the six logical TRXs of the QTRU are assigned frequencies.
    19. Click OK to return to the dialog box shown in Figure 8.
    20. Click Finish. The configuration of the GRRU is complete.
