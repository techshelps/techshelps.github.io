Configuring the M3UA Link

This describes how to configure the M3UA link. The M3UA link configured for the A interface implements the IP transmission between the BSC and the MSC. The M3UA link configured for the Abis interface implements the transmission between BSCs.

ScenarioBSC initial configuration and BSC capacity expansion
Mandatory/OptionalMandatory

Prerequisites

  • The DSP for the M3UA entity is configured. For details, see Configuring the M3UA Entities.
  • The mapping between the subrack and the signaling point is configured. For details, see Configuring the Subrack-OPC Mapping.
  • The GMPS/GEPS is configured with the GXPUM. For details, see Configuring the GXPUM.
  • The M3UA link set is configured.
  • The GMPS/GEPS is configured with the GFGUA/GOGUA and the GFGUB/GOGUB. For details, see Configuring the GFGUA/GOGUA and Configuring the GFGUB/GOGUB.
NOTE:
  • When the M3UA link is configured on the GFGUA/GOGUA, the IP transmission between the BSC and the MSC is implemented. When the M3UA link is configured on the GFGUB/GOGUB, the transmission between BSCs is implemented.
  • When the M3UA link is configured on the GFGUA/GOGUA, SCCP Subsystem Number of the corresponding M3UA destination entity is set to 254. When the M3UA link is configured on the GFGUB/GOGUB, SCCP Subsystem Number of the corresponding M3UA destination entity is set to 143 and moreover the link set of this destination entity must be configured.

Procedure

  1. On the BSC6000 Local Maintenance Terminal, right-click the GFGUA in the GMPS.
  2. Choose Configure M3UA Link > Add M3UA Link from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

    Figure 1 Configure M3UA Link dialog box

  3. Click Add. A dialog box is displayed, as shown in Figure 2.

    Figure 2 Add M3UA Link dialog box

  4. Set the parameters of the M3UA link by referring to Table 1.
  5. Click OK to return to the dialog box, as shown in Figure 1.
  6. Click OK. The M3UA link is configured.
