Configuring High-Speed Signaling

This describes how to configure a 2 Mbit/s signaling link on the BSC6000 Local Maintenance.

Prerequisite

The high-speed signaling is not license-controlled.

To improve the reliability of signaling links, on each Ater interface board (in the case of BM/TC separated mode) or A interface board (BM/TC combined mode), the total bandwidth cannot exceed 2M, and the number of signaling links cannot exceed two. It is recommended that the signaling links be distributed evenly on multiple interface boards.

Context

The high-speed signaling link meets the requirements of signaling transmission on the A interface when the traffic load is heavy.

Procedure

  • BM/TC combined
    1. Right-click a GEIUA on the BSC Device Panel of the BSC6000 Local Maintenance Terminal and choose Configure SS7 Signaling Link from the shortcut menu. A dialog box is displayed, as shown in Figure 1.

      Figure 1 Configure SS7 Signaling Link dialog box (1)

      NOTE:

      If the system prompts you to configure the E1/T1 connection, perform 2 through 4 to configure the E1/T1 connection for the GEIUA. Otherwise, go to 5.

    2. Right-click a GEIUA on the BSC Device Panel of the BSC6000 Local Maintenance Terminal and choose Configure A Interface E1/T1 from the shortcut menu. A dialog box is displayed, as shown in Figure 2.

      Figure 2 Configure A interface E1/T1 dialog box

    3. Click Add. A dialog box is displayed, as shown in Figure 3.

      Figure 3 Add A Interface E1/T1 dialog box

      CAUTION:

      The DPC Group No. maps the Port No.. Configure the Start CIC of this port.

      The start CICs of different ports within a DPC group cannot be duplicate.

    4. Click OK to return to the Figure 2 dialog box. Click Finish to complete the configuration.
    5. Click Add in the dialog box shown in Figure 1. A dialog box is displayed, as shown in Figure 4.

      1. On the MTP2 tab, set Rate Type to 2Mbit/s, and select the timeslots for A Timeslot Mask.

        Figure 4 Add SS7 Signaling Link dialog box (3)

      2. On the MTP3 tab page, as shown in Figure 5. set SLC and SLC Send.

        Figure 5 Add SS7 Signaling Link dialog box (3)

    6. Click OK to return to the dialog box shown in Figure 1. Click Finish to complete the configuration.
  • BM/TC separate
    1. Right-click a GEIUT on the BSC Device Panel of the BSC6000 Local Maintenance Terminal and choose Configure Ater Connection Path from the shortcut menu. A dialog box is displayed, as shown in Figure 6.

      Figure 6 Configure Ater Connection Path dialog box

    2. Click Add. A dialog box is displayed, as shown in Figure 7.

      Set the parameters according to the location of the GEIUT in the GTCS.

      Figure 7 Add Ater Connection Path dialog box

    3. Right-click a GEIUA on the BSC Device Panel of the BSC6000 Local Maintenance Terminal and choose Configure SS7 Signaling Link from the shortcut menu. A dialog box is displayed, as shown in Figure 8.

      Figure 8 Configure SS7 Signaling Link dialog box (1)

      NOTE:

      If the system prompts you to configure the E1/T1 connection, perform 2 through 4 to configure the E1/T1 connection for the GEIUA. Otherwise, go to the next step.

    4. Click Add. A dialog box is displayed, as shown in Figure 9.

      1. On the MTP2 tab, set Rate Type to 2Mbit/s, and select the timeslots for A Timeslot Mask and Ater Timeslot Mask.

        Figure 9 Add SS7 Signaling Link dialog box (1)

        NOTE:

        In BM/TC separated mode when the GTCS is configured remotely, you are advised to configure the SS7 signaling link on the main GTCS. In addition, TS1 should not be selected in Ater Timeslot Mask.

      2. On the MTP3 tab page, as shown in Figure 10. set SLC and SLC Send.

        Figure 10 Add SS7 Signaling Link dialog box (2)

    5. Click OK to return to the dialog box shown in Figure 8. Click Finish to complete the configuration.
