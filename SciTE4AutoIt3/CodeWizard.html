Overview

This utility generates AutoIt code for the following functions:

·     MsgBox
·     InputBox
·     SplashText
·     SplashImage
·     GUISetBkColor
·     GUICtrlSetColor
·     GUICtrlSetBkColor
·     GUICtrlSetFont
·     GUICtrlSetCursor

The user can select from various options to create the specific code they require

It merges the previous CodeWizard script (by Giuseppe Criaco) with the CFCCodeWizard (by Gary Frost).

See the Help file included with CodeWizard itself for more details.
