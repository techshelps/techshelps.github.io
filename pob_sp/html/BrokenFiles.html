Por qué algunos archivos estãn dañados o incompletos luego de la descarga

Hay dos razones que pueden causar que los archivos parezcan dañados: cuando el servidor Web devuelve la longitud del archivo y cuando la longitud es desconocida. En el primer caso, si el archivo no se carga completamente y la conexión se corta, Portable Offline Browser reanuda la descarga (estã intentando reanudar desde el punto en que se cortó (Función Retomar).)

De todas formas, como no se conoce la longitud del archivo, no hay una forma confiable de confirmar si el archivo se cargó completamente. En este caso, luego de que se cortó la conexión, el archivo se guarda en el disco tal como estã. Si el archivo es HTML, GIF o JPEG, entonces Portable Offline Browser busca la parte final del archivo. If it is missing, Portable Offline Browser tries to redownload the file to make sure it is complete.
