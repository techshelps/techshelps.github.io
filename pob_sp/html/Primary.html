.primary and Descr.wd3 files

Durante la descarga, Portable Offline Browser puede crear archivos con la extensión .primary y archivos descr.wd3.

El objetivo de los archivos .primary es el de ayudar durante la actualización de un sitio Web cargado. Cuando Portable Offline Browser debe cargar archivos sin modificar, verificarã cada archivo para ver si ha cambiado desde la última descarga, De no ser así, el archivo no se cargarã. Si se trata de un archivo HTML, Portable Offline Browser deberã tener la copia original con el vínculo HTML a seguir. Los archivos .primary son los archivos HTML originales sin modificar.

Los archivos primary también se usan cuando no se cargan los archivos existentes. Para seguir vínculos, Portable Offline Browser explora los archivos HTML .primary

Nota:Los archivos .primary no aumentan el trãfico. Portable Offline Browser carga sólo archivos normales, con archivos .primary creados posteriormente.

Las versiones de Portable Offline Browser a partir de 2.0 no generan archivos .primary a menos que usted agregue la siguiente línea al campo de URLs del Proyecto:

Additional=KeepPrimary

Los archivos Descr.wd3 se encuentran en cada directorio que crea Portable Offline Browser, y contienen información respecto de todos los archivos descargados en el directorio:

- nombre de archivo

- URL original del archivo

- fecha de la última modificación en el servidor Web (para buscar actualizaciones)

- archivo de tipo MIME, tal como fue devuelto por el servidor Web (esto es útil para la navegación sin conexión)

- un signo, indicando si se creó o no el archivo .primary.

Aunque todos estos archivos se pueden borrar, esto no se recomienda ya que puede hacer que las actualizaciones de sitio Web y navegación sin conexión sean menos confiables.
