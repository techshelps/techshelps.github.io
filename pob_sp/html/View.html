Exploración del Sitio Descargado Sin Conexión

Mientras descarga un un Proyecto se agregarã al ícono del Proyecto una pequeña flecha verde hacia adelante verde. Al finalizar la descarga, el ícono del Proyecto vuelve a su estado normal. Ahora puede ver los archivos descargados sin conexión.

Hay varias maneras de hacer esto:

  1. Puede hacer clic con el botón derecho sobre el Proyecto y seleccionar Explorar para abrir el sitio Web descargado en el Navegador Interno o en el navegador por defecto de su computadora, si el Navegador Interno estã apagado.

  2. Puede seleccionar Explorar Con y elegir el navegador deseado desde el menú de contexto descrito arriba (por favor, vaya a Opciones | Herramientas | Navegadores para configurar una lista de exploradores disponibles por primera vez). El sitio Web descargado se abrirã usando el navegador Web seleccionado.

  3. Puede hacer clic sobre la pestaña Mapa para ver el mapa del sitio Web descargado. El primer archivo se resaltarã automãticamente. Hacer clic con el botón derecho sobre cualquier archivo, directorio o servidor mostrarã un menú de contexto que permite explorar como se describe en 1 y 2 arriba.

  4. Si configura una lista de editores y visores en Opciones |Herramientas , podrã usar estas herramientas con cualquier archivo listado en el Mapa por medio del menú de contexto de botón derecho.

Luego de disfrutar de la navegación sin conexión, podrã borrar el Proyecto y todos sus archivos descargados. Para ello, presione la tecla Borrar o haga clic sobre el botón Borrar en la barra de tareas o haga clic sobre el botón Borrar sobre la Cinta de opciones - pestaña Inicio. Portable Offline Browser le preguntarã si desea borrar los archivos descargados.

Tenga cuidado - si responde "No", los archivos se mantendrãn en el disco duro y usted deberã borrarlos manualmente para liberar espacio en disco.

Si elige no borrar el Proyecto con los archivos descargados, serã guardado para usted entre sesiones de Portable Offline Browser.
