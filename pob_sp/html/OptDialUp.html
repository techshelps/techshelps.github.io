Diãlogo de Opciones: Internet - Discado

Para usar el discador interno en Portable Offline Browser, habilítelo y configure sus propiedades en el diãlogo de Opciones, Internet | Sección discado.

Para habilitar el discador, marque la casilla Habilitar discador interno.

Cuando estã habilitado, el discador iniciarã la conexión de acceso teléfonico a Internet y llamarã al Proveedor de Servicios de Internet cuando se inicie un Proyecto.

Estã disponible la siguiente configuración del discador:

  • Seleccionar conexión

    Para iniciar una conexión con un Proveedor de Servicios de Internet, Portable Offline Browser utiliza las conexiones estãndar de Windows en la carpeta Mi PC | Acceso remoto. Si usted no tiene esa carpeta, deberã configurar el servicio de Windows correspondiente. Portable Offline Browser usa el nombre de usuario y la contraseña provistos, así que antes de utilizar el discador de Portable Offline Browser asegúrese de que la conexión estã funcionando.

  • Nombre de inicio de sesión

    Al llamar al Proveedor de Servicios de Internet, Portable Offline Browser usarã el nombre de usuario y la contraseña provistos. Si son requeridos, por favor ingréselos aquí.

  • Contraseña

    Al llamar al Proveedor de Servicios de Internet, Portable Offline Browser usarã el nombre de usuario y la contraseña provistos. Si son requeridos, por favor ingrese la contraseña aquí.

  • Reintentar conexión rota

    Este parãmetro determina cuãntas veces el discador intentarã restaurar una conexión por módem rota.

  • Intervalo entre intentos

    La computadora remota puede estar ocupada o no estar respondiendo llamadas temporalmente. Por favor, ajuste este parãmetro para que el número de intentos de discado no se termine rãpidamente.

También puede usar el Discador externo para conexiones que no son por módem, como DSL, Satélite, etc. Simplemente coloque la ruta y el nombre de archivo del programa discador con sus parãmetros en las casillas de discado y cortado correspondientes.

Por defecto, Portable Offline Browser espera 20 segundos a que el discador externo se inicie. Si necesita mãs retraso, especifíquelo en la línea de comando:

c:\dir\ext_dialer.exe /param1 /param2 /delay=50

Donde 50 es el retraso en segundos.
