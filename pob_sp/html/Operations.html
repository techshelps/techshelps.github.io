Cómo organizar el ãrbol de Proyectos

El ãrbol de Proyectos organiza Proyectos agrupãndolos en carpetas. Cada Proyecto o carpeta puede ser eliminado o modificado de ser necesario.

La creación de un Proyecto nuevo se explica aquí. Para crear una nueva carpeta, haga clic en Nuevo y luego en Carpeta sobre la Cinta de Opciones - Pestaña Inicio.

Las operaciones siguientes requieren de la selección del Proyecto o carpeta deseada. Es posible seleccionar múltiples Proyectos o carpetas de una vez manteniendo presionadas las teclas CtrloShift mientras los selecciona con el botón izquierdo del ratón.

También puede mover cualquier Proyecto o carpeta a otra ubicación usando arrastrar y colocar.

Las operaciones con el Portapapeles son útiles si desea transferir un Proyecto a otra computadora -cópielo al portapapeles, péguelo en algún archivo de texto (p.ej. usando Bloc de notas), transfiera el archivo de texto a la otra computadora, y pegue el Proyecto en Portable Offline Browser. También puede pegar el Proyecto en la ventana Archivo | Plantillas para crear una nueva Plantilla basada en la configuración de ese Proyecto.

La última operación Borrar puede deshacerse utilizando el botón Deshacer en la barra de herramientas de Acceso Rãpido.

Para crear carpetas anidadas, use la tecla Ctrl. Por ejemplo, para crear una nueva carpeta dentro de una existente, seleccione la carpeta, haga clic en Nuevo en la Cinta de Opciones - Pestaña Inicio, mantenga apretada la teclaCtrl , haga doble clic sobreNueva carpeta y suelte la tecla Ctrl.

También puede arrastrar y colocar una carpeta dentro de otra manteniendo presionada la tecla Ctrl mientras arrastra la carpeta
