Cómo comenzar

Luego de que Portable Offline Browser ha sido instalado y se ejecuta por primera vez, hay cuatro pasos simples para la exploración sin conexión.

Paso 1.La primera vez que ejecuta Portable Offline Browser, debe configurar los parãmetros proxy del servidor. Esto se requiere si su navegador WWW se conecta normalmente a través de proxy. Si usted no estã seguro sobre el servidor proxy o su configuración, por favor consulte con su administrador de red.

Necesitarã saber lo siguiente:

  1. Tipo de servidor proxy: sin proxy. proxy HTTP, RTSP, Usuario@Sitio, servidor proxy SOCKS 4 o SOCKS 5

  2. Si hay un proxy, su dirección (puede ser IP o nombre DNS) y su número de puerto

  3. Si se trata de un proxy HTTP o SOCKS 5, nombre de usuario y contraseña, de ser necesario.

Si usted tiene un servidor proxy, deberã seleccionar el diãlogo Opciones, hacer clic sobre el nodo Servidor Proxy e ingresar la configuración enumerada arriba.

El primer paso es necesario únicamente cuando usted inicia por primera vez, o si la configuración de su red ha cambiado.

Los pasos 2-4 son necesarios cuando desea descargar un sitio Web para exploración son conexión.

Paso 2.Cómo crear un nuevo un Proyecto para descarga.

El Proyecto define desde qué pãgina web debe comenzar la descarga y cuãntas pãginas Portable Offline Browser deberã descargar.

Haga clic sobre el botón Asistente sobre la Cinta de Opciones - pestaña Inicio y siga los pasos para crear un Proyecto.

Paso 3. Esperar a que la descarga se complete.

El nuevo Proyecto se crearã y comenzarã la descarga. La flecha verde sobre el ícono del Proyecto indicarã que la descarga estã en curso. Cuando la descarga se complete, la flecha verde desaparecerã.

Paso 4..Cómo ver archivos descargados sin conexión.

Ahora el sitio Web estã disponible para la exploración sin conexión. Haga clic con el botón derecho sobre el Proyecto y seleccione Explorar. Portable Offline Browser detectarã la mejor forma de desplegar el sitio tanto en el Navegador Interno. como en su explorador WWW por defecto.

Puede repetir los pasos 2-4 en cualquier momento. Portable Offline Browser es compatible con la creación de un número casi ilimitado de Proyectos, que pueden ser descargados en paralelo.

También hay una forma semi manual de descargar los sitios para la exploración sin conexión. Haga clic aquí para leer mãs al respecto.
