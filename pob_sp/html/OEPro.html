Portable Offline Browser

Actualmente, Portable Offline Browser tiene las siguientes funciones adicionales:

  • Un directorio de descarga aparte para carpetas de Proyecto

  • Carpetas anidadas en el ãrbol de Proyectos

  • Macros de URL

  • Múltiples URLs en cada Proyecto

  • Resaltado de sintaxis para las URLs, la macros de URL y parãmetros adicionales. Autocompletado de Macros de URL.

  • Habilidad de mantener copias de las pãginas Web originales son vínculos no traducidos (archivos .primary)

  • Habilidad de mantener copias antiguas de archivos cargados

  • Administrador de cola de descargas

  • Habilidad de cambiar el directorio en que se guardan los Proyectos, plantillas y mapas

  • Diseño mejorado del panel de Mensajes

  • Soporte de protocolo MMS (flujo de archivos Microsoft Media Player)

  • Soporte de protocolo RTSP (flujo de archivos Real Player)

  • Soporte de protocolo PNM (versión antigua de flujo de archivos Real Player)

  • Extracción de Datos - post procesamiento del sitio Web descargado

  • Habilidad de prohibir el acceso de otras computadoras al servidor HTTP interno

  • Función de niveles de vínculos con otros servidores

  • Exportación automãtica de Proyectos descargados

  • Referentes personalizados , cookies y post solicitudes

  • Descarga o Exportación de archivos directamente desde el caché de MS IE

  • Exportación de sitios descargados a un directorio sencillo, sin subdirectorios.

  • Exportación de sitios descargados a un archivo MHT o a un archivo de ayuda HTML (.CHM)

  • Eliminado automãtico de archivos de Proyecto antiguos

  • Personalización de archivos Contents.htm antes de exportar el sitio Web

  • Ventana adicional de descripción de Proyecto para proveer de comentarios multilínea.

  • Habilidad de imprimir el sitio Web completo

  • Filtrado de imãgenes por dimensiones (mín/mãx y lista de soporte de dimensiones no deseadas)

  • Modos de verificación de archivo avanzados: Verificación de tamaño de archivo y Saltear archivos Media existentes

  • Habilidad de recolectar direcciones de correo electrónico desde sitios Web descargados

  • Habilidad de impedir que los directorios de descarga se sobrecarguen

  • Propiedades de Proyecto | Filtros de Contenidos para cargar sólo pãginas con palabras clave

  • Exploración simple de formularios Web

  • Sustitutos URL

  • Habilidad de limitar el número de conexiones por Proyecto y pausar descarga individuales

  • Editor interno con destaque de sintaxis para archivos HTML, CSS y secuencia de comandos Java

  • Función Auto Guardar para descargar y guardar pãginas mientras navega sin conexión

Nota:HTTPS requiere que el sistema tenga instalado Microsoft Internet Explorer 4.0 o posterior Usted puede descargar la última versión desde http://www.microsoft.com/windows/ie/.
