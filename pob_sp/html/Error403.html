¿Por qué recibo el mensaje "403 Prohibido"?

El servidor no permite que se cargue el documento. Hay varios casos en los que aparece este problema:

  • El sitio estã protegido por contraseña y usted necesita ingresar la contraseña en el diãlogo Propiedades de Proyecto, sección Avanzadas | Contraseñas.

  • Usted estã intentando descargar un listado de directorio y el servidor no se lo permite. Esto sucede cuando la URL termina con una barra "/" . Desafortunadamente, no existe solución a este problema.

  • La sección del sitio no estã habilitada para descargas.

  • El sitio no permite la descarga de navegadores sin conexión. Para superar esta limitación, abra el diãlogo Opciones y seleccione "Internet Explorer 5.0" en el campo de Agente de identificación de usuario.
