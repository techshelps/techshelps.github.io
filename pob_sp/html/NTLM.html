¿Cómo manejar la autenticación NTML?

Portable Offline Browser es compatible con la autenticación NTML tanto para sitios Web como para servidores proxy.

Durante la autenticación, el servidor verifica tres parãmetros - Dominio, nombre de usuario y contraseña. Para ingresarlos correctamente, por favor complete el campo nombre de usuario en Propiedades de Proyecto (Avanzadas | Contraseñas) o en el diãlogo Opciones (Proxy | HTTP) de la siguiente forma:

dominio\nombredeusuario

  • nota de la "\" barra invertida entre el dominio y el nombre de usuario.
