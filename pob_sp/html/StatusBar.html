Barra de Estado

La Barra de Estado se usa para mostrar la información de Portable Offline Browser respecto de todas descargas y la un Proyecto seleccionada.

La primer columna de la Barra de Estado muestra el estado interno de Portable Offline Browser:

Listo - Portable Offline Browser estã listo para descargar o se encuentra descargando.

Suspendiendo - Portable Offline Browser estã en modo Suspender luego de apretar el botón Suspender en la Cinta de Opciones - Pestaña Inicio. En este modo Portable Offline Browser no descargarã ningún archivo hasta que haga clic sobre el botón Suspender nuevamente.

Analizando - Portable Offline Browser estã analizando o un documento HTML o una clase Java. El número al lado del mensaje Analizando es el número de archivos HTML o Clase que quedan en la cola de anãlisis.

La columna siguiente estã vacío si no se seleccionó un Proyecto. De otra forma, contiene información referente al Proyecto seleccionado - estadísticas de la última descarga o número de archivos descargados.

La cuarta columna contiene la velocidad de descarga global actual para Portable Offline Browser. Se computa como el promedio de los últimos cinco segundos.

El grãfico de velocidad de descarga muestra los últimos 60 segundos. Pase el ratón por encima para ver mãs detalles.
