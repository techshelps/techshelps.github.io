  • Directorio de Descarga

    Asignar un directorio a la carpeta. Todos los Proyectos ubicados en esta carpeta usarãn este directorio para almacenar archivos y sitios descargados.

    Esta función le permite especificar varias ubicaciones donde se guardarãn los archivos descargados. Por ejemplo, puede agrupar los Proyectos en carpetas con diferentes directorios de descarga para regular el espacio usado en disco en diferentes unidades de disco duro o restringir el acceso de otros usuarios (si tiene instalado Windows NT).

    Desmarque esta opción si desea que esta carpeta use el directorio de descarga por defecto del diãlogo Opciones.
