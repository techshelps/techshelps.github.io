Parãmetros de Línea de Comando

Portable Offline Browser soporta parãmetros de línea de comandos para permitir la integración con otros programas tales como programadores de terceros, etc.

Los Parãmetros de Línea de Comando permiten que uno o mãs Proyectos comiencen a ser descargados cuando se ejecuta Portable Offline Browser.

El formato es: <Nombre_carpeta>/<Número_Proyecto_en_Carpeta>. Los números de Proyecto se cuentan desde cero. Así, si hay cinco proyectos en la carpeta "Deportes" y usted desea comenzar a descargar el primer Proyecto y el último en la carpeta, deberã tipear:

pob.exe Deportes/0 Deportes/4

También puede tipear proyectos de carpetas distintas:

pob.exe Deportes/3 Auto/5 Espacio/1

Los Proyectos especificados deberãn estar separados por espacio(s).También puede poner "todos" en lugar del número para comenzar la descarga de todos los Proyectos en la carpeta:

pob.exe Deportes/todos

Puede usar el parãmetro "/cerrar"para salir de Portable Offline Browser unos segundos después de que se haya completado la descarga. Esto es útil si desea descargar algunos proyectos de forma automãtica y luego cerrar Portable Offline Browser para guardar memoria y otros recursos. /CloseNoCountdown forzarã a Portable Offline Browser a cerrarse en silencio, sin desplegar el diãlogo de cuenta regresiva de cierre.

pob.exe Deportes/3/cerrar

Si desea que su computadora se apague unos segundos después de que todas las descargas se hayan completado, puede usar el parãmetro "/apagar".

pob.exe Deportes/3/apagar

Si desea que su computadora se apague unos segundos después de que todas las descargas se hayan completado, puede usar el parãmetro "/colgar".

pob.exe Deportes/3/colgar

Para explorar un Proyecto en el Navegador Interno, ingrese su Carpeta/Número luego del parãmetro /Explorar:

pob.exe /Explorar Deportes/3

Para respaldar un Proyecto, ingrese su Carpeta/Número luego del parãmetro /Respaldar=<archivo>:

pob.exe /Respaldar=c:\archivo.boe Deportes/3

Puede habilitar el registro de actividad automãtico a un archivo usando la sintaxis:

pob.exe /log=c:\dir\archivo.log

Para detener el registro, comience otra copia de Portable Offline Browser con el parãmetro “/nolog".

Para filtrar la cantidad de información del registro, use el parãmetro /LogFilter=1111101. Cada número habilita (1) o (0) deshabilita la salida de la siguiente información: conexión, progreso de descarga, información detallada (encabezados, etc.), desconexión, mensajes de cola, accesos al servidor web interno y URLs rechazadas.

Si el archivo de registro excede los 100 MBs, Portable Offline Browser lo borra por defecto, y comienza desde el principio. Para cambiar el límite, use el parãmetro /MaxLogSize=10, en el que el número especifica el número límite en megabytes del archivo de registro.

Para pausar la descarga actual en forma temporal:

pob.exe /Suspender

Si suspendió descargas a archivos .wdq, puede proveer el nombre del archivo .wdq en la línea de comando para cargarlo en Portable Offline Browser:

pob.exe "c:\ruta\micola.wdq"

Para suspender la descarga actual a un archivo, use el siguiente parãmetro de línea de comando:

pob.exe /SuspendToFile=c:\dir\archivo.wdq

De la misma forma puede restaurar el archivo de respaldo de un Proyecto:

pob.exe "c:\ruta\mybackup.boe"

Al comenzar, Portable Offline Browser carga los marcadores/favoritos/hotlists encontrados en su sistema. Esto puede tomar un rato. Si desea saltear la carga, use el siguiente parãmetro:

pob.exe /NoBookmarks

Portable Offline Browser crea, por defecto, Mapas de Proyecto para cada Proyecto al descargarlo. El Mapa simplemente lista todos los archivos descargados para un Proyecto. Se usa al borrar un Proyecto con sus archivos, para respaldar/exportar/limpiar/imprimir/estadísticas/encontrar contenidos/extracción de datos y otras funciones. De todas formas, si desea saltear el soporte Mapas, comience Portable Offline Browser de la forma siguiente:

pob.exe /NoMap

Si habilitó el icono de bandeja, pero desea ocultarlo (Portable Offline Browser se ejecutarã de forma invisible en ese caso), use:

pob.exe /NoTray

Puede agregar un nuevo proyecto desde la línea de comando:

pob.exe "/new=url=http://www.server.com/file.htm,template=template name,folder=Projects folder name"

Si especifica ejecutar=verdadero en la línea de arriba, Portable Offline Browser crearã un proyecto y comenzarã la descarga automãticamente:

pob.exe "/new=url=http://www.server.com/file.htm,url=POST=p1=v1&p2=v2,url=Remitente=http://www.srv.com/,run=true,level=3"

Crear un nuevo Proyecto, descargarlo, exportar y borrar ambos Proyectos y archivos descargados:

pob.exe "/new=url=http://www.server.com/file.asp,template=Template1,folder=MyFolder,run=true,level=3,url=Additional=DeleteProjectFiles;DeleteProjectSettings;AutoExport=c:\export\"

Usted puede comenzar Secuenciador al ejecutar Portable Offline Browser:

pob.exe /StartSequencer

Puede buscar contenidos en los archivos de Proyecto y guardar la lista de archivos encontrados en un archivo de texto:

pob.exe "/FindContents=Folder/5,c:\directory\file.txt,keyword1 keyword2,05/24/2003 5:20 PM"

Apagado de la pantalla Asistente al comienzo:

pob.exe /NoWizard

Puede iniciar Portable Offline Browser con el Navegador Interno maximizado:

pob.exe /MaximizeIB

Puede iniciar Portable Offline Browser minimizado:

pob.exe /Minimize

Para cerrar la aplicación en curso de Portable Offline Browser en forma inmediata (sin esperar a que la descarga o exportación actual finalicen) use:

pob.exe /Exit

Para esperar a que las descargas o procesos de exportación finalicen y luego salir de Portable Offline Browser use:

pob.exe /Close

o

pob.exe /CloseNoCountdown

Para cambiar el límite de velocidad actual:

pob.exe /Speed=2

El parãmetro puede estar entre 0 y 3, donde 0 es la velocidad de fondo y 3 la mãxima.

Para cambiar el número de conexiones a Internet use:

pob.exe /Channels=10

Para cambiar la demora entre descargas:

pob.exe /DownloadsDelay=5

Para ejecutar una nueva instancia de Portable Offline Browser:

pob.exe /Instance
