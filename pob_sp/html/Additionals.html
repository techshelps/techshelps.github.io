Lista de parãmetros de comando Additional=

Puede colocar el comando Additional= en una línea separada luego de todas las URLs en el campo de Direcciones del diãlogo Propiedades de Proyecto. Son de utilidad para llevar a cabo varias operaciones automãticas en sitios Web descargados una vez que la descarga se ha completado:

Additional=DeleteOldFiles

Elimina archivos que ya no estãn en el servidor de la copia local.

Additional=CountCRC

Realiza una mejor verificación de las modificaciones contando los CRCs de los archivos descargados.

Additional=Print

Imprime un sitio descargado automãticamente una vez que se completó la descarga.

Additional=CollectEMails=c:\dir\file.txt

Encuentra todas las direcciones de correo electrónico y las guarda en un archivo de texto específico. Mãs detalles aquí.

Additional=AutoExport=...

Realiza una Exportación automãtica cuando se complete la descarga. Haga clic aquí para mãs detalles.

Additional=DeleteProjectFiles

Elimina todos los archivos descargados. Es de utilidad cuando se lo combina con el parãmetro Auto exportar, por ejemplo.

Additional=SuspendToFileOnLimit=c:\dir\file.txt

Suspende la descarga de un Proyecto a un archivo si se lo debe detener debido a un límite en las Propiedades del proyecto | sección Avanzadas.

Additional=FinalizeAllFilesOnStop

Termina de cargar y guardar archivos aún incompletas cuando se detiene un Proyecto, o cuando se debe detener un Proyecto de forma automãtica debido a los Límites.

Additional=DisableScripts

Desactiva la extracción de vínculos desde secuencias de comandos Java y VB.

Additional=DisableJava

Desactiva la extracción de applets Java (archivos .class).

Additional=KeepPrimary

Portable Offline Browser crearã siempre una copia de cada archivo HTML, agregando la extensión ".primary". Estos archivos ".primary" son los archivos HTML originales con vínculos sin traducir, dado que se encuentran en un servidor Web. Los archivos HTML comunes tienen todos los vínculos traducidos para permitir la navegación sin conexión.

Additional=ReuseFTP

Utiliza la misma conexión FTP para descargar diferentes archivos (puede que no funcione en algunos servidores FTP).

Additional=ConvertPOSTToFileName

Agrega contenidos de solicitudes POST al nombre del archivo guardado luego del símbolo @. Esto ayuda a evitar sobrescribir archivos con contenidos diferentes cuando se permite el procesamiento de formularios HTML o se utilizan Macros de URL en solicitudes POST.

Additional=SkipIFrames

No procesa etiquetas IFRAME, que normalmente contienen publicidad.

Additional=DepthFirst

Añade nuevas URL a la parte superior de la cola de descarga, de forma que las imãgenes se carguen en primer lugar, etc.

Additional=NoMovedDuplicates

Detiene el procesamiento del comando 302 Moved HTTP si la misma URL se ha descargado.

Additional=AbsoluteURLs

Otra forma de Traducción de Vínculos - convierte todos los vínculos a direcciones URL absolutas en línea.

Additional=AddOriginalURL

Añade un comentario a cada pãgina Web descargada con la URL de la pãgina original.

Additional=NoMap

Deja de agregar archivos descargados al Mapa de Proyecto.

Additional=DoNotParseFiles

Sólo cargar archivos, no vínculos a seguir.

Additional=DoNotParseExistingFiles

Deja de tomar vínculos de archivos descargados previamente.

Additional=ParseIncludedScripts

Este comando incluye archivos .js externos en pãginas Web que las vinculan. Entonces, todas las secuencias de comandos serãn evaluadas en conjunto para conseguir incluso mãs vínculos en algunos sitios de secuencias de comandos complejas. También es necesario permitir cãlculos de secuencia de comandos en el cuadro de diãlogo Propiedades del proyecto - sección Avanzadas.

Esta función no estã activada por defecto, ya que enlentece el procesamiento de los archivos descargados.

Additional=SkipDisposition

Algunos servidores pueden devolver un nombre de archivo alternativo que difiere de la URL. Por ejemplo, la URL http://www.srv.com/img.asp?id=17 puede ser presentado como flower.jpg. En este caso, Portable Offline Browser crea ambos archivos: img.asp@id=17 y flower.jpg. El primer archivo sólo contendrã un pequeño código de redirección al segundo archivo. Este comando bloquea tal comportamiento y los nombres de archivo alternativos se ignoran. Esto es de utilidad cuando distintas URL con distinto contenido presentan el mismo nombre de archivo.

Additional=MoveOldFiles=c:\dir\

No sobrescribir archivos que han cambiado desde la última descarga, sino que se muevan a la carpeta especificada. De todas formas, es mãs fãcil usar la función Copias de Archivos en el diãlogo Propiedades de Proyecto para guardar copias antiguas de archivos modificados.

Additional=KeepCurrentURLs

Si se inicia el Proyecto dos veces, no se cargarãn los vínculos que aún se estã cargando.

Para combinar varios parãmetros en una línea utilice el símbolo de punto y coma:

Additional=Print;DeleteOldFiles

o

Additional=AutoExport=c:\export\;101;DeleteProjectFiles
