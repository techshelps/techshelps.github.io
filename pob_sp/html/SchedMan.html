Administrador de Programación

El Administrador de Programación es una herramienta útil para inspeccionar o cambiar el orden en que sus Proyectos son programados para descargas automãticas.

Representa los Proyectos en una línea de tiempo en uno de los cuatro modos elegidos:

Diariamente - muestra todos los Proyectos programados para la descarga diaria. (línea de tiempo de 0-24 horas)

Semanalmente - para Proyectos que se deben cargar cada semana. (línea de tiempo de domingo a sãbado)

Mensualmente - sólo para Proyectos que se deben descargar cada mes. ((línea de tiempo desde el día 1º al día 31º)

Anualmente - para programación anual. (línea de tiempo de enero a diciembre)

Cada Proyecto se muestra con una flecha azul. Los detalles del Proyecto se muestran en el panel superior cuando el ratón pasa sobre la flecha azul correspondiente.

Usted puede cambiar la programación del Proyecto moviendo esta flecha con el ratón (el botón izquierdo del ratón deberã mantenerse presionado). Arrastrar un Proyecto fuera del Administrador de Programación cambiarã su estado de programación a Manual. Hacer doble clic sobre un Proyecto mostrarã el panel de Programación en el diãlogo Propiedades de Proyecto.

También puede arrastrar un Proyecto desde el ãrbol de Proyecto hacia el Administrador de Programación para configurar su estado de programación y tiempo.

Secuenciador

Otra función muy útil es el Secuenciador. Usted puede arrastrar y colocar Proyectos sobre él para descargarlos de a uno. Cuando se descarga un Proyecto de la secuencia, comienza la descarga del siguiente.

Usted puede comenzar y detener el secuenciador usando el cuadro Suspender. Si la casilla Bucle estã marcada, el secuenciador comenzarã con el primer Proyecto luego de que finalice la descarga del último.

Haga doble clic sobre un Proyecto en el Secuenciador para seleccionarlo en ãrbol de Proyectos. Esto le permitirã comenzarlo o detenerlo manualmente. Utilice el menú de contexto de botón derecho para cambiar el orden de la secuencia, o para eliminar Proyecto.
