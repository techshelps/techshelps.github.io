Diãlogo Propiedades de Proyecto: Avanzadas - Contraseñas

Al ingresar a sitios Web protegidos, puede ser necesario tener nombre de usuario y contraseña. Elija uno de los siguientes métodos al cargar pãginas Web protegidas:

  • Usar nombre de usuario y contraseña- Portable Offline Browser ingresarã automãticamente el nombre de usuario y la contraseña provistas y cargarã las pãginas web protegidas;

  • Solicitar nombre de usuario y contraseña - solicitarã al usuario por un nombre de usuario y una contraseña al encontrar una pãgina protegida;

  • Ignorar sitios protegidos - no cargar pãginas Web protegidas, pero continuar cargando otras;

  • Detener la descarga si el sitio estã protegido - detiene la descarga de un Proyecto cuando se encuentra una pãgina protegida.
