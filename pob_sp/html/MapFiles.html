Archivos de Mapa

Cada Proyecto tiene su propio Mapa. El Mapa es simplemente una lista de todos los archivos descargados para el Proyecto. Por ejemplo:

c:\download\www.server.com\articles\file001.htm

c:\download\www.server.com\default.htm

c:\download\www.server.com\images\banner.gif

c:\download\www.server.com\images\pixel.gif

Cuando Portable Offline Browser descarga un Proyecto, cada archivo descargado se agrega al Mapa de Proyecto.

Portable Offline Browser usa la lista de archivos en un Mapa de Proyecto durante ciertas operaciones con Proyectos -exportar, eliminar, respaldar, buscar, imprimir y algunas otras.

Todos los archivos de Mapa se almacenan en el directorio definido en el diãlogo Opciones | sección Ubicaciones de Archivo - directorio de Proyectos, Mapas y Plantillas. Todos tienen la extensión.map .

Los archivos con la extensión .map son archivos de texto simples. Cada nombre de archivo se coloca en una línea aparte. Puede haber algunos parãmetros adicionales (tipo MIME, nivel, etc.). Se separan del nombre de archivo con una tabulación.

Si un archivo de Mapa de un Proyecto se pierde por alguna razón, es fãcil restaurarlo. Simplemente seleccione el Proyecto y presione Ctrl-F5 en el teclado. Portable Offline Browser comenzarã a descargar el Proyecto en el modo "No cargar archivos existentes". Esto agrega todos los archivos existentes y agregados recientemente al Mapa de Proyecto.
