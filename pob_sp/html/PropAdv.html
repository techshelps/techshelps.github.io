Diãlogo Propiedades de Proyecto: Avanzadas

Aquí encontrarã los siguientes parãmetros de Proyecto:

  • Limitar la descarga por tamaño de archivos cargados

    Si estã activa, Portable Offline Browser detendrã la descarga del Proyecto cuando el tamaño de los archivos descargados sea mayor que el valor ingresado.

  • Limitar la descarga por número de archivos

    Si estã activa, Portable Offline Browser detendrã la descarga del Proyecto cuando el número de archivos descargados exceda el valor ingresado.

  • Limitar la descarga por tiempo

    Si estã activa, Portable Offline Browser detendrã la descarga del Proyecto cuando la descarga tome mãs tiempo que el valor ingresado (en minutos).

  • Verificar la Integridad de archivos HTML

    Si estã activa, Portable Offline Browser verificarã la integridad de los archivos HTML, GIF y JPEG descargados cuando el servidor no informa el tamaño de información. Si un archivo parece no estar completo, serã recargado inmediatamente.

  • Explorar todos los subdirectorios posibles

    Con esta configuración activada, cuando Portable Offline Browser encuentra un vínculo a http://www.srv.com/dir1/dir2/somefile.htm, intentarã solicitar los contenidos de /dir1/ and /dir2/ . Si el servidor permite listados de directorio, Portable Offline Browser podrã descargar mãs archivos.

  • Suprimir errores de sitios Web

    Existen algunos sitios que protegen ciertas pãginas de los navegadores fuera de línea. Las marcan como pãginas no existentes o error, mientras que el contenido de la pãgina aún estã presente en el sitio. Portable Offline Browser ignorarã el estado de error de la pãgina con esta configuración activada. Usted puede desactivarla para sitios que no usan esa protección.

  • Cargar mapas de imagen del lado del servidor

    Algunas pãginas Web usan mapas de imagen - cuando diferentes zonas de una única imagen contienen vínculos a distintas pãginas. Algunos de esos mapas de imagen no pueden ser recuperados fãcilmente y Portable Offline Browser debe preguntar al servidor sobre cada una de las zonas de 5x5 pixeles - qué vínculo estã asociado con esa zona.

  • Explorar Formularios HTML

    Cuando estã activada, Portable Offline Browser trata de seguir vínculos en Formularios HTML simples que no requieren intervención del usuario.

  • Evaluar cãlculos de secuencia de comandos

    Cuando estã activada, Portable Offline Browser intenta evaluar todos los cãlculos de secuencia de comandos Java para encontrar aún mãs vínculos. Esta opción estã desactivada por defecto, ya que este proceso toma mãs recursos del CPU al analizar los archivos HTML descargados.

  • Sustitutos URL

    Es una función innovadora que le permite cambiar sobre la marcha vínculos que descarga Portable Offline Browser.

    Usted puede beneficiarse de muchas formas de los Sustitutos URL. Un ejemplo.

    Usted puede estar descargando de un servidor "www.server.com" que tiene vínculos a "server.com", y ambos sitios tienen contenidos idénticos. Usted puede convertir todos los vínculos a server.com para que apunten al primer servidor. Esto impedirã que usted tenga dos copias del mismo servidor en su disco duro.
