Parãmetros del campo de URLs del Proyecto

El campo de URLs del Proyecto puede contener múltiples URLs y los siguientes parãmetros para ajustar su Proyecto: Adicionales

Especifica la exportación automãtica, el tratamiento posterior de los archivos descargados, etc. Lea mãs aquí.

AcceptCharset

AgregaAccept-Charset: a los encabezados de petición. Ejemplo: Accept-Charset=en

AcceptLanguage

AgregaAccept-Language: a los encabezados de petición. Ejemplo: Accept-Language=en

Canales

Limita la cantidad de conexiones simultãneas que puede usar el Proyecto. Ejemplo: Channels=2

ChannelsPerServer

Limita el número de conexiones simultãneas por servidor. Ejemplo: ChannelsPerServer=3

Cookie:

Configura cookie personalizada. Ejemplo: Cookie=param1=value&param2=othervalue

DeleteAfterParsing

Eliminar archivos descargados por mãscara. Ejemplo: DeleteAfterParsing=*.asp*,*www.srv.com*

GlobalReferer

Fuerza el uso del remitente especificado para todos los vínculos que se descargan para el Proyecto. Ejemplo: GlobalReferer=http://www.srv.com/

IgnoreLogoutLinks

Saltea todos los vínculos similares a cerrar sesión, salir, etc. Ejemplo: IgnoreLogoutLinks

Nivel

Establece el nivel de configuración personalizada para todas las URL que siguen el parãmetro. Ejemplo: Level=1

ModifiedSince

Solicita sólo los archivos modificados después de la fecha especificada. La fecha debe seguir un formato estricto: ModifiedSince=Mon, 06 May 2002 02:35:36 GMT

POST

Especifica la solicitud de POST para las URLs dadas. Ejemplo: POST=param1=val1&param2=val2

Proxy

Fuerza el uso de la configuración personalizada del servidor proxy para el Proyecto. Ejemplo: Proxy=1,192.168.1.1:3128. The first number is 0 for no proxy, 1 - HTTP, 2 - SOCKS4, 3 - FTP, 4 - SOCKS5, 5 - RTSP, 6 - CONNECT. Luego sigue server:port También puede usar un archivo Proxy={:file=c:\proxies.txt} para especificar una lista de servidores proxy a usar. Cada línea deberã tener el mismo formato, por ejemplo: 1,192.168.1.1:3128. El primer número y la coma se pueden omitir. Esto implica que proxy HTTP por defecto.

Remitente

Utiliza la URL especificada como remitente para la dirección de inicio: Remitente=http://www.srv.com/

BrowseBeforeWithDelay

Abre la URL especificada en el navegador interno, espera el número especificado de segundos y luego comienza a descargar el Proyecto. Esto es útil cuando primero se necesita obtener una cookie del sitio. Ejemplo: BrowseBeforeWithDelay=5;http://www.site.com

RunAfter

Ejecuta el programa especificado luego de que se completa la descarga del Proyecto. Ejemplo: RunAfter=notepad.exe c:\file.txt

RunBefore

Igual que arriba, pero antes de que comience la descarga del Proyecto.

RunBeforeWithDelay

Igual que arriba, pero espera el número especificado de segundos después de ejecutar el programa, y luego comienza la descarga del Proyecto. Ejemplo: RunBeforeWithDelay=5;c:\program.exe

TextPipe

Ejecuta TextPipe con el filtro especificado cuando se completa la descarga del Proyecto. Ejemplo: TextPipe=c:\path\filter.fll

UserAgent

Configurar la identificación del agente de usuario. Ejemplo: UserAgent=Portable Offline Browser/6.9

WarnIfNotUpdatedForDays

Muestra un mensaje de advertencia si el Proyecto no se ha descargado en los últimos días. Esto sucede cada vez que un usuario explora el Proyecto sin conexión. Ejemplo: WarnIfNotUpdatedForDays=5

SuspendToFileEvery

Respalda la cola de descarga del Proyecto cada el número especificado de segundos. Ejemplo: SuspendToFileEvery=60;c:\file.wdq

hWndParam

Enviar mensaje WM_COPYDATA al identificador de la ventana especificada para cada archivo descargado. Ejemplo: hWndParam=9578

Demora

Configurar demora personalizada entre descargas. Ejemplo: Delay=2-5

Interrupción de conexión

Configurar Interrupción de conexión personalizada en segundos. Ejemplo: Timeout=60

SkipParsingBefore

Analizar sólo una parte de las pãginas Web descargadas luego del texto especificado. Ejemplo: SkipParsingBefore=<body

SkipParsingAfter

Analizar únicamente una parte de las pãginas Web descargadas. Cuando se encuentra el texto especificado, se saltea el resto del archivo. Ejemplo: SkipParsingAfter=</html>

SkipParsingFiles

No analizar (extraer vínculos y hacer que funcionen fuera de línea) los archivos que coinciden con la(s) palabra(s) clave(s). Aquí puede especificar una o mãs palabras clave. Sepãrelas con una coma. Las palabras clave deberían coincidir con una parte del archivo URL. Ejemplos:

SkipParsingFiles=*.pdf,*.exe

SkipParsingFiles=/test/
