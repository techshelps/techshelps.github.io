Cómo guardar pãginas durante la exploración

A veces sólo es necesario guardar un conjunto de pãginas Web pequeño para la exploración sin conexión. Si usted no desea perder tiempo ajustando la configuración del Proyecto para descargar sólo los archivos deseados, puede usar la función Auto Guardar.

Aquí se detalla el proceso paso a paso:

Paso 1.Haga clic sobre el botón Nuevo...en la barra de herramientas para crear un Proyecto nuevo. Portable Offline Browser abre el cuadro de diãlogo Propiedades de Proyecto. Ingrese la URL de la pãgina o sitio donde desea comenzar la exploración en el campo de URLs. Haga clic en el botón OK para guardar el Proyecto

Paso 2.Haga clic en el botón Auto Guardar () en la barra de herramientas del Navegador Interno. Este botón permanecerã presionado.

Paso 3.Haga clic sobre el botón Explorar en la Cinta de Opciones - Pestaña Inicio para comenzar a explorar el Proyecto. Aunque el Proyecto aún no ha sido descargado, Portable Offline Browser mostrarã la pãgina Web sin conexión. Todos los archivos faltantes serãn descargados automãticamente y agregado al Proyecto seleccionado. Puede visitar otras pãginas en este sitio Web u otros, y todos serãn descargados de la misma forma. Al finalizar la descarga de todas las pãginas deseadas, haga clic en botón Auto Guardar nuevamente para detener el proceso de descarga.

También puede usar el modo Auto Guardar cuando ha descargado un sitio Web, pero olvidó varias pãginas o archivos y desea que estos se agreguen al Proyecto sin tener que ajustar la configuración y descargar nuevamente.

Ademãs, algunas pãginas Web pueden incluir secuencias de comandos muy complejas, animaciones, etc. Si Portable Offline Browser no pudo extraer y descargar algunos vínculos de forma correcta, explorar el sitio sin conexión en modo de Auto Guardar corregirã la situación.
