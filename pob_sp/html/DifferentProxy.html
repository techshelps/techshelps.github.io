Cómo usar distinta configuración de servidor proxy para Proyectos

Usted puede describir los parãmetros del servidor proxy para un Proyecto en su campo de direcciones URL en una línea separada después de la URL de inicio:

http://www.server.com/

Proxy=1,proxy.mycompany.com:3128

Por favor, no utilice espacios. El primer número es el tipo de servidor proxy:

0 - sin proxy, conexión directa

1- proxy HTTP

2 - SOCKS4

3 - proxy FTP Usuario@sitio

4 - SOCKS5

5 - proxy RTSP

6 - proxy tipo CONNECT

Luego coloque el símbolo de coma seguida de la dirección del servidor proxy, el símbolo dos puntos y el número de puerto del servidor proxy.
