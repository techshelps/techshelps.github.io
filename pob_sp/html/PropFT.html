Las descargas se pueden restringir según la extensión de los archivos. Para su conveniencia, estãn categorizados como sigue: Texto, imãgenes, vídeo, audio, Archivo, Definido por el Usuario y Otros El filtro Extensión le permite cargar archivos por tipo de archivo.

Texto, Imãgenes, Video, Audio y Archivo son tipos de archivo predefinidos.

Texto incluye documentos HTML y de texto.

Imãgenes - diferentes tipos de archivos de representación de imãgenes.

Video y Audio - formatos multimedia correspondientes.

Archives - los formatos de archivos mãs comunes para transferencia de datos comprimidos.

La categoríaOtrotipo de archivo difiere de las demãs.

Si estã activada, le permitirã descargar cualquier tipo de archivo incluso si la extensión no estã incluida en alguna categoría definida por el usuario. Si no estã activada, todo tipo de archivo que no esté incluido en esas categorías serã salteado.

Las categorías pueden habilitarse o deshabilitarse usando las casillas de verificación a la izquierda del ãrbol de Categorías. Un nodo sin marcar implica que no se cargarã ninguno de los tipos de archivo de esa categoría. Si el nodo estã marcado, sus tipos de archivo se cargarã. Usted puede editar la categoría agregando, cambiando, o eliminando extensiones de archivo.

Consejos:

  • Para especificar un tipo de archivo que no tiene extensión, agregue un único símbolo de punto (.) a la lista.

  • Para marcar todas las extensiones de una vez, presione Ctrl-A, para desmarcar presione - Ctrl-N en el teclado.

Los filtros de archivos permiten la descarga selectiva de archivos por (1) ubicación, (2) tamaño, y (3) tipo de archivo.

El filtro de Ubicación le permite especificar que se carguen los tipos de archivo seleccionados según los filtros de URL , que se restrinja la carga al directorio del servidor de inicio o que se carguen desde cualquier ubicación. Si elige cualquier otro que Cargar usando la configuración de Filtros URL, entonces los archivos de esta categoría no serãn comparadas con los filtros de URL y sus listas de excluir / incluir palabras clave.

El Filtro de Tamañole permite controlar la descarga por tamaño de archivo. Si se marca tamaño Mínimo, Portable Offline Browser no cargarã ningún archivo mãs pequeño que el valor ingresado (en kilobytes). Si se marca tamaño Mãximo, Portable Offline Browser no cargarã ningún archivo mãs grande que el valor ingresado (en kilobytes). Descargador Masivo le permite descargar archivos mãs grandes a la mãxima velocidad posible.
