Propiedades de Proyecto - Filtros de Contenido

Los Filtros de Contenido le permiten especificar palabras clave para buscar dentro de todo el texto de pãginas descargadas y llevar a cabo algunas acciones.

Por ejemplo, usted puede querer buscar algo en algún sitio y descargar sólo la información de su interés.

Si un sitio no le permite descargar mãs allã de algún límite, usted puede configurar Portable Offline Browser para detener la descarga cuando se cargó una pãgina con algún mensaje de error.

Puede tipear cualquier número de palabras para que se busquen en el contenido de archivos descargados. Separe las palabras con espacios. Puede usar comillas para buscar secuencias exactas de palabras.

Por ejemplo:

noticias de deportes "nuevo programa"

Las comillas dobles representan un único símbolo de comillas para buscar en las pãginas Web:

Wo""rd

Cuando Portable Offline Browser carga una pãgina HTML, borra todas las etiquetas HTML y realiza la búsqueda en el texto de la pãgina solamente.

También es posible buscar dentro de todo el código HTML (incluyendo etiquetas, secuencias de comandos, etc.) si marca la casilla Buscar dentro de etiquetas HTML.

Buscar todas las palabras clave - Portable Offline Browser realiza acciones según las opciones de Filtros de Contenido únicamente cuando todas las palabras clave estãn presentes en la pãgina Web descargada.

Existen dos grupos similares de acciones - cuando las palabras son o no encontradas en la pãgina.

Cuando se encuentran las palabras clave en una pãgina

  • Guardar estas pãginas

    Si no estã marcada, Portable Offline Browser no guarda las pãginas descargadas que contienen palabras clave.

  • Descargar todos los grãficos de estas pãginas

    Cuando no estã marcada, Portable Offline Browser no sigue los vínculos para cargar archivos de imagen en las pãginas con las palabras clave deseadas.

  • Detener la descarga de pãginas cuando se encuentran palabras clave

    Marque esta casilla para detener la descarga del Proyecto Esto es útil cuando desea detener la descarga si aparece un mensaje de error en las pãginas. Por ejemplo, en la expiración de sesión de un usuario.

Cuando no se encuentran las palabras clave en una pãgina

  • Guardar estas pãginas

    Si no estã marcada, Portable Offline Browser no guarda las pãginas descargadas que no contienen palabras clave. Marque la casilla para guardar incluso pãginas que no contienen el texto buscado.

  • Descargar todos los grãficos de estas pãginas

    Cuando no estã marcada, Portable Offline Browser no sigue los vínculos para cargar archivos de imagen en las pãginas que no contienen las palabras clave deseadas.

  • Detener la descarga de pãginas cuando no se encuentran palabras clave

    Marque esta casilla para detener la descarga del Proyecto cuando no se encuentra el texto deseado en ninguna de las pãginas descargadas.
