Diff_Format Initialization Variable

See Also!ALink()

Controls the format used with the Show Differences (File) command on the Tools menu.

Syntax

Diff_Format = {SS|UNIX|Visual}

Example

Tells VSS to use a visual (side-by-side) difference display:

Diff_Format = Visual

Remarks

When you use the Show Differences (File) command to show the differences between two files, VSS can display the file differences in three ways.

  • SS Lists lines added, deleted, or changed.

  • UNIX Displays the standard output form of the UNIX Diff utility.

  • Visual Shows two files side by side, with different lines color-coded.

If you do not specify a preferred format, VSS Explorer defaults to Visual, and the command line defaults to the SS display. However, if you set this variable, both VSS Explorer and the command line switch their default to your setting. In any case, you can always override the default with command-line options and in the SourceSafe Options dialog box on the Tools menu.
