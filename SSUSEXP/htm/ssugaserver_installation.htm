Server Installation

See Also

As the VSS administrator, you use this method to install VSS to a computer that is accessible to all users.

To perform a server installation:

  1. Insert the VSS compact disc in the CD-ROM drive of the server or a machine connected to the server.

  2. Double-click Setup.exe.

  3. Follow the setup instructions in the setup wizard.

  4. When asked what type of installation you'd like, click Server.
