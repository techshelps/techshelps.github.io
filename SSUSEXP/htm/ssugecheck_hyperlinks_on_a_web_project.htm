Check Hyperlinks on a Web Project

See Also

You can use this command to check Web links in your HTML files, either the files in your working folder or those in a VSS project. VSS checks links that jump to other files in your project and links to external pages.

The links checked are HTML tags, including the <A HREF> tag. Links to other Web addresses created using code, such as VBScript or Java, are not checked.

To check Web project hyperlinks:

  1. Select a Web site project. Web site projects have a this icon: ()

  2. On the Web menu, click Check Hyperlinks.

  3. If you want to check links to external pages, click the Check External Links on World Wide Web box.

  4. Click OK.

    When the link check is complete, VSS displays a dialog box with three list boxes:

    • The top box shows all files in which errors were found.

    • The middle box shows links (jumps to other files within the Web project) that appear to be invalid in the selected file in the top box.

    • The bottom box shows links found, but not checked.
