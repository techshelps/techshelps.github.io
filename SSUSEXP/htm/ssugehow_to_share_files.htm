How to Share Files

Note   To view projects sharing a file, select the file, then go to the Links tab in the Properties command on the File menu.

To share a file:

  1. In VSS Explorer, select the project in which you want to share a file.

  2. On the SourceSafe menu, click Share to display the Share dialog box.

  3. In the File to share list, select the file you want to share. If the file you want to share is not displayed, select the project containing the file from the Projects list.

  4. Click Share.

  5. Click Close.
