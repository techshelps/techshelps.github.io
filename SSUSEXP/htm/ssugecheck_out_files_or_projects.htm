Check Out Files or Projects

See Also

Note   Set your working folder before using this command. See How to Set a Working Folder for more information.

To check out a file or project:

  1. Select the item(s) you want in VSS Explorer.

  2. On the SourceSafe menu, click Check Out.

  3. If you haven't previously set a working folder, VSS will ask you if you'd like to do so now. Click OK, then set your working folder.

  4. If you have set options to display a dialog box for the Check Out command (using the Command Dialogs tab in the Options command on the Tools menu), VSS displays the Check Out dialog box. Click OK to check out the item(s).

  5. VSS places the item(s) in your working folder. You can now edit the item(s).
