Command Dialogs Options Tab

In this tab you can specify whether command dialog boxes are displayed when you perform VSS commands.

If you do not use the options provided on the Get Latest Version dialog box, for example, you can turn the dialog box off. When the Get Latest Version command is performed, the dialog box is not shown.

You can specify whether a dialog box appears in the following file commands:

  • Get Latest Version

  • Check Out

  • Undo Check Out

  • Show Differences

  • History

  • View and Edit

You can specify whether a dialog box appears in the following project commands:

  • Get Latest Version

  • Check Out

  • Undo Check Out

  • Show Differences

  • History

  • Deploy

  • Create Site Map

To display the Command Dialogs tab:

  1. On the Tools menu, click Options.

  2. Click the Command Dialogs tab.

Important   Override any setting you make in this dialog box by holding down the SHIFT key while performing any of the preceding commands. The command dialog box appears even if the corresponding Options check box has not been selected.
