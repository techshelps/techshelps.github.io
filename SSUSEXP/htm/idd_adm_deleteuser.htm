Delete User Command (Users Menu)

Deletes a user from VSS Administrator's user list.

Remarks

To delete a user

  • Click the user you want to delete in VSS Administrator's user list, and then on the Users menu, click Delete. Click OK to confirm the deletion.

The user is deleted from the list, and is no longer able to run VSS.

If the deleted user has files checked out, a warning is not generated. To unlock those files, run VSS as the Admin user and use the Undo Check Out command.
