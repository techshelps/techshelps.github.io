Use <File> originally checked out to <Local Folder>? (Y/N)

This warning has the following cause and solution:

  • From the command line, you are attempting to check in a file that is multiply checked out to you.  VSS is asking you to confirm which checkout to check in.

    Type Y if the indicated local folder location is correct.  The check in will be performed.

    Type N if the indicated local folder location is not correct.  VSS will repeat this message for each checkout record for the file.
