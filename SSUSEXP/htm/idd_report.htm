Report Dialog Box

Creates a formatted report of selected information that you can send to the Clipboard, a file, or a printer.

Dialog Box Options

Report To

  • Printer Sends report to printer.

  • File Sends report to File dialog box.

  • Clipboard Places report on Clipboard.

OK

Confirms the report selection.

Cancel

Cancels the report.

Preview

Shows a preview of the specified report, so that you can determine if it is what you wanted before committing it to the printer queue or a file.
