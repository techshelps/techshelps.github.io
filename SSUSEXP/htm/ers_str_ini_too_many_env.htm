Too many SS.INI environment strings

This warning has the following cause and solution:

  • VSS stores many configurations in a SS.INI file that is individualized for each user.  For each INI file setting, VSS will store an environment string.  This is typically the name of the computer when this setting was made.  This usually occurs because of the ‘Dir (<environment string>) = <Some local folder>’ INI file setting.  VSS allows around 25 unique environment strings.

    Manually edit the \USERS\<Username>\SS.INI file for the user who gets this message.  Delete any unused INI file settings that have multiple environment strings. 

    If you count the number of times you receive this message, it indicates the number of environment strings that need to be removed.  If you get this upon start up, it will tell you what lines contain the excessive environment strings.
