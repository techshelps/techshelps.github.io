Integration Options Dialog Tab

Displays information about the integration between your Visual Basic project and the corresponding VSS project.

To display the Integration Options tab:

  • On the Add-Ins menu, click SourceSafe, then Options, and then click the Advanced button. From this dialog box, select the Integration tab.

Access Rights

You must have Read access right to use this dialog box.

Remarks

The Integration Options tab is most useful for seeing how your VSS and Visual Basic projects are connected. However, you can also set a couple options on this tab: whether to display intermediate dialog boxes for the History and Show Differences command, and whether to prompt for the VSS database to use when opening a new project.

Tab Options

SourceSafe project

Shows the full name of the VSS project associated with this Visual Basic project.

Database name

Displays the name (if any) of the VSS database the project is stored in.

Database path

Displays the path to the VSS database the project is stored in.

Data specifier

Displays the data directory for a specific project. (Your VSS Administrator can set up different data directories for different projects using the Data_Path variable.)

Display dialog box for

Check the box corresponding to the command if you want an options dialog box displayed when you select that command.

Choose SourceSafe database

Set to Use Default Database to always look in the default VSS database for your project information. Set to Prompt to display a prompt dialog box, allowing you to specify the database name, path, and data specifier each time you open a project.

OK

Sets the options as specified on the tab.

Cancel

Closes the Integration Options tab.
