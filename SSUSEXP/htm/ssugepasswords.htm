Passwords

See Also

Use the Change Password command to set and change your password. To change your password, you must first know what it is. If you forget your password, ask your VSS administrator to change it for you.

Upon log in, VSS prompts for a password if the operating system prevents it from determining user identity. It will not prompt for a password if the VSS administrator has specified the user network name is the same as the user log in name.

Note   While your VSS password can be the same as or different from your operating system password, it is not meant to replace the operating system password.
