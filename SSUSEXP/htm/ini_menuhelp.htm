Menu_Help Initialization Variable

Indicates whether the VSS status bar displays automatic help on selected menu items and icons.

Syntax

Menu_Help = {Yes|No}

Example

Instructs VSS to always display a Help line on the status bar when a menu item or icon is selected:

Menu_Help = Yes

Remarks

When this variable is set to Yes, as you scroll through a menu or click an icon, the status bar at the bottom of VSS Explorer shows a brief description of the currently selected command.
