Branch Shared Files

See Also

Branching a file breaks the shared link, making the file in that project independent of all other projects.

Note   To view the branch history of a file, click the Paths tab in the Properties command on the File menu.

To branch a file:

  1. Select the file(s) you want in VSS Explorer.

  2. On the SourceSafe menu, click Branch to display the Branch dialog box.

  3. Type a comment describing the branching in the Comment box, if desired.

  4. Click OK.
