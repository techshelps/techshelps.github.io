MKSS

See Also

Mkss.exe creates an empty VSS database in an old format.  Mkss.exe is installed in the ADMIN subfolder by default.

To create a new VSS database:

  1. Create a folder.

  2. Run the Mkss.exe utility on the folder.

  3. Run the Ddconv.exe utility on the folder.

Syntax

MKSS [folder]
