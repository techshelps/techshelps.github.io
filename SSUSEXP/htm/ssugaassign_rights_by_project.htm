Assign Rights by Project

See Also

To assign rights by project:

  1. From the VSS Administrator Tools menu, click Rights by Project to display the Project Rights dialog box.

  2. Under Projects, select a project.

  3. Under Users, select a user. VSS Administrator displays the current rights for the user in the selected project.

  4. You can add or delete rights by selecting or clearing the User rights check boxes.

  5. To add a user to the currently selected project, click Add User. VSS Administrator displays the Add Users for Project dialog box, with a list of users who currently do not have rights in the selected project. Select the check box next to the right(s) you would like to assign the user, then click OK.

  6. To delete a user from the currently selected project, click Delete User.

    When you delete a user's access to a project, VSS Administrator performs the deletion with no confirmation.

  7. Click Close.

Use the Rights Assignments for User command to view all the project security rights of a particular user.
