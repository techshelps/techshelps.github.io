Properties

VSS allows you to view or edit certain file or project properties by using the Properties command on the File menu. For example, you can change the file's type, its comments, and specify whether to store only the latest version to save disk space.

For information on how to view or modify file or project properties in VSS, refer to:

  • Append End-of-Line to all Text Files

  • Edit Properties

  • View Deleted Files List

  • View File Links

  • View File Paths

  • View File Properties

  • View Size of Selected File
