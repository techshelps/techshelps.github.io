View Check In Information

See Also

This displays information about when the file was checked in.

To view checkin information on a file:

  1. In VSS Explorer, select the file you want to view checkin information on.

  2. On the Tools menu, click Show History to display the History Options dialog box.

  3. Click OK to display the History of File dialog box. VSS displays checkin information, such as version of the file, the user, date of the checkout, and action taken on the file.
