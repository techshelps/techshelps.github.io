No other user available to copy rights from

The VSS Administrator can copy the rights from one user to another. This error has the following cause and solution:

  • You have attempted to copy rights to the currently selected user. However, there is no other existing user that you can copy rights from. (Note that you cannot copy rights from the Admin user.)

    You can only copy user rights when you have two or more users other than the Admin.  If you currently have only one user, you need to assign the rights to this user using the Rights by Project command or the Rights Assignments for User command on the Tools menu.
