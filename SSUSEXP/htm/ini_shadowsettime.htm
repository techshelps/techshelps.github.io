Shadow_SetTime Initialization Variable

See Also

Controls the date and time given to files in the shadow folder. Used in the SRCSAFE.INI file only.

Syntax

Shadow_SetTime = {Current|Mod|Update}

Example

Stamps shadow files with the date and time they are placed into the shadow folder (the default):

Shadow_SetTime = Current

Remarks

This variable affects the date and time stamp given to files in the shadow folder, in the same way SetTime affects files in your working folder.

You can also set this variable in the SRCSAFE.INI file with the File Date/Time option, which is accessed by clicking Options on the Tools menu, then selecting the Shadow Folders tab.
