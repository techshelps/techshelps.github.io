-S Command-Line Option

See Also

The Smart_Mode initialization variable in SS.INI puts VSS in SmartMode. With SmartMode enabled:

  • After you create a project, SmartMode sets that project as your current project.

  • When you delete or destroy a file, VSS automatically deletes the corresponding local file.

  • When you rename a file, VSS automatically renames the corresponding local file.

The -S option overrides the Smart_Mode variable for a particular command.

The following table lists ways to use the -S option.

Option Description
-S Enables SmartMode.
-S- Disables SmartMode.
