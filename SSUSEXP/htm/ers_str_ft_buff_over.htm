Adding the name <Group Name> would overflow the maximum File_Types length of <Maximum Length>.  Please decrease the name size or remove other names.

This error has the following cause and solution:

  • The combined length of the filter values on the File Types tab in the SourceSafe Options dialog box on the Tools menu is too long. VSS uses the Types Included in Group field to create a filter for the Share and Add dialog boxes. This error occurs when the combined length of the groups is too long. The total buffer length is 511 characters.

    To remedy and prevent this error, shorten the File Group name or delete other File Group names from the list.
