Open Database Dialog Box

See Also

This dialog lets you browse for a SRCSAFE.INI file associated with a VSS database.

Dialog Box Options

File Name

Enter the file name to search for. When looking for a VSS database, use SRCSAFE.INI.

List files of type

Filter the file list box by displaying files only of the selected type. SourceSafe Database is the default.

Folders

Select the folder to look in.

Drives

Select the drive letter to look on.

OK

Selects the database file listed in the File Name box.

Cancel

Cancels the browse operation.

Network

Lets you map a drive letter to a different network server, if necessary.
