Options Dialog Box

You can customize a number of VSS options using the Options dialog box. Some tasks you can customize in VSS include:

  • How command dialogs are displayed

  • How command-line options function

  • How VSS displays file difference information

  • How routine actions are carried out, such as checkin and checkout

  • How and what VSS Explorer is set to display (for example, the toolbar, status bar, and so on)

For information on the Options Dialog box, refer to:

  • Command Dialogs Options Tab

  • Command Line Options Tab

  • Difference Options Tab

  • File Types Options Tab

  • General Options Tab

  • Local Files Options Tab

  • View Options Tab

  • Warnings Options Tab
