VSS Administrator Tasks

As the VSS administrator, it is your job to make the VSS database as useful to VSS clients as possible. There are various ways in which you can change the behavior of VSS, for example to allow clients to use multiple file checkout, or to take advantage of VSS's new Web features.

As the administrator, you can do any of the following:

  • Create a New Database

  • Create Web Site Projects

  • Create Shadow Folders

  • Customize the SRCSAFE.INI or SS.INI File

  • Enable Keyword Expansion

  • Enable Multiple Checkouts

  • Open a Database

  • Set Default File Types
