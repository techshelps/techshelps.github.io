Customize VSS

You can customize the behavior of VSS in many ways. You can change the way that VSS behaves to optimize its performance, change the display, or insert expanding keywords into your files. Customizing VSS can increase your productivity, as well as make VSS run more smoothly for Macintosh and UNIX users.

You can customize fonts in VSS Explorer using the Font command on the Tools menu. You can also customize the VSS toolbar using the Customize Toolbar command on the Tools menu.
