Cloak Initialization Variable

See Also

Hides cloaked projects from recursive  Get, Check Out, Check In,  Undo Check Out, and Project Difference commands.

Syntax

Cloak = {Yes|No}

Example

Instructs VSS to cloak a project so that a recursive Get command, for example, would not affect that project:

Cloak = Yes

Remarks

No is the default setting. This variable can also be set from the File General Properties tab (when a project is selected)
