Open a Database

To open a VSS database:

  1. From the VSS Administrator Users menu, click Open SourceSafe Database to display the Open SourceSafe Database dialog box.

  2. Select a database from the list.

    Note   If the database you want is not in the list of available databases, click Browse. Navigate to the SRCSAFE.INI file for an existing VSS database, then click the SRCSAFE.INI file. Click Open to add the database to the list.

  3. Click Open on the Open SourceSafe Database dialog box.
