Filter string exceeds the maximum length of <number>

This error has the following cause and solution:

  • The combined length of the filter values on the File Types tab in the SourceSafe Options dialog box on the Tools menu is too long.

    VSS uses the File Types Included in File Group field to create a filter string for the Share and Add Files dialog boxes. This error occurs when the combined length of the file groups is too long. The total buffer length is 511 characters. To correct this error, reduce the length of your file-type lists.
