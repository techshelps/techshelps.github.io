Temp_Path Initialization Variable

See Also

Sets the folder where VSS stores temporary files.

Syntax

Temp_Path = path to Visual SourceSafe temporary folder

Example

Indicates that the temp path is a folder called Temp immediately under the folder that contains SRCSAFE.INI:

Temp_Path = Temp

Remarks

This variable can also be set in your SS.INI file by using the Folder for Temporary Files option that can be accessed from the Tools menu, by clicking Options, then selecting the General tab.
