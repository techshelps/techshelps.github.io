SiteMap Initialization Variable

See Also

Specifies the location of the HTML file generated when a user selects the Create Site Map command from the VSS Web menu.

Syntax

SiteMap = file path

Example

Specifies that the Sitemap.html file is the auto-generated site map for the project under which this variable appears:

SiteMap = c:\webproj\sitemap.html

Remarks

This variable can also be set from the Web Sites tab that can be accessed from the Tools menu, by clicking Options, then selecting Administrator.
