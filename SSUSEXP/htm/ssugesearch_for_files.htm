Search for Files

Ordinarily, the list view shows all the files in the current project. However, you can also use the Search command to see only files that match certain criteria, such as all files with the *.c extension (wildcard search), or to view only checked-out files. Searches can be recursive, to search all the subprojects under the current project, or global searches, to search the entire project list.

Search strings can be up to 63 characters long.
