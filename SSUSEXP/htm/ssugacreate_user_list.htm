Create a User List

See Also

To create a VSS user list:

  1. Select the desired database in VSS Administrator. For more information, see Open a VSS Database.

  2. From the VSS Administrator Users menu, click Add User to display the Add User dialog box.

  3. Enter the first user's name in the User name box. User names can be up to 31 characters long, cannot begin or end with a space, and cannot include any punctuation.

  4. Enter the user's password in the Password box. Passwords can be up to 15 characters long, and can contain any characters.

  5. To give the user read-only rights, select the Read only check box.

    Note   If you do not select this check box, the new user has read/write rights. See Security Access Rights for more information.

  6. Click OK.

  7. Repeat steps 1-6 until you've created your entire VSS user list.
