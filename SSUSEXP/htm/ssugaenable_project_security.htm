Enable Project Security

See Also

To enable project security:

  1. From the VSS Administrator Tools menu, click Options to display the SourceSafe Options dialog box.

  2. Click the Project Security tab.

  3. Select the Enable Project Security check box to enable project security.

  4. Under Default user rights, clear the check boxes next to the access rights you do not want to grant.

By default, when you enable project security, new users have the full set of access rights to the database.

You can establish a new set of default user rights by selecting only the check box next to the highest level right you want to use as a default.
