Lock SourceSafe Database Dialog Box

See Also

Allows you to lock all users out of the VSS database while you run a utility, upgrade the executables, or back up the database.

Dialog Box Options

Lock all users out of Visual SourceSafe

Select this box to lock all users out of the VSS database. Remember to clear the box when you are finished with your administrative duties.

Note that the act of selecting the check box in the Lock SourceSafe Database dialog box is what actually locks users out. After this box is selected, no one will be allowed to log on to the database. The users listed who are already logged on will not be affected you will need to notify them manually to log off.

X users logged in

This section of the dialog box shows the users who are currently logged on.

Close

Closes this dialog box.
