Change a User Password

For an explanation of this procedure, see Change Passwords.

To change a user's password:

  1. In the VSS Administrator user list, select a user.

  2. On the Users menu, click Change Password to display the Change Password dialog box

  3. In the New password box, type the new password.

  4. In the Verify box, type the new password again to verify it.

  5. Click OK.

Note   If you tell VSS to use the user's network username, this bypasses the VSS logon dialog box.
