Using VSS with UNIX and Macintosh Computers

See Also

To use VSS with UNIX or Macintosh computers, contact MetroWerks (Macintosh) or MainSoft (UNIX). The default VSS installation is simple to create and use, but may not be optimized for performance. Therefore, you can customize your system to improve performance:

  • Copy as much information as possible from the network to your hard disk. This change is useful for all network users, but is essential for Macintosh users on Intel networks. You can use the VSS Netsetup.exe program (on the VSS server) to copy the VSS executable files to your local computer.

  • Fine-tune your initialization files for performance. Most fine-tunings involve compromises, such as disabling certain features that involve time-consuming overhead. None of these optimizations provides as much benefit as copying information from the network to your hard disk.
