Advanced Get Options Dialog Box

Sets default Check Out command options.

To display the Advanced Get Options dialog box:

  • On the Tools menu, click Get, and then click the Advanced button.

Dialog Box Options

Replace Writable

Determines VSS's behavior when you attempt to get a file that is already marked as writable in your working directory.  Select one of the following in the list box:

  • Ask Asks you what to do in each case.

  • Replace VSS replaces the file with a read-only version.

  • Skip Skips the Get operation.

  • Merge Merges the changes between the checked out-file and the one you are now getting or checking out.

Default Uses the default set on the Local Files Options tab, via the Tools menu, Options command.

Set File Time

Changes the time assigned to the Get operation.  Select one of the following in the list box:  current time, the last file modification time, the last file check-in time, or the default time (set on the Local Files Options tab).

End of Line

Specifies the end-of-line character that VSS gives to each line in the file it retrieves.  Select one of the following in the list box: line feed (LF) for the UNIX platform, carriage return (CR) for the Macintosh, carriage-return/line-feed (CR/LF) for MS-DOS/Windows, or the default (set on the Local Files Options tab).

Make Writable

Retrieves the file and makes it writable.  This is not equivalent to a Check Out operation, because the Get command is a non-exclusive one or more people can perform the Get command on the same file.  You cannot check in changes after a Get operation with this option set.
