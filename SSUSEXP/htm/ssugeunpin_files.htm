Unpin Files

See Also

To unpin a file:

  1. In VSS Explorer, select the file you want to unpin.

    Note   VSS displays a pushpin icon next to a pinned file.

  2. On the Tools menu, click Show History to display the History of File dialog box.

  3. Select the pinned version, then click Unpin.

  4. Click Close.

VSS unpins the file so the most recent version is the current version in the project.
