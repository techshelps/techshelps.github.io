Get Latest Version

See Also

Note   When you Get a file, the file is placed in your working folder as read-only — any modifications you make to the file at this point cannot be saved in the project.

To get the most recent version of a file or project:

  1. Select the item(s) you want. You can select a file, multiple files, or a whole project.

  2. On the SourceSafe menu, click Get Latest Version.

  3. If you haven't previously set a working folder, VSS will ask you if you'd like to do so now. Click OK, then set your working folder.

  4. If you have specified in options that the Get Latest Version dialog box be displayed, VSS displays the dialog box. Click OK to get the item you selected. It is copied from the current project into your working folder.
