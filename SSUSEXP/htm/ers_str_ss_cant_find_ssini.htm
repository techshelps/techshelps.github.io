Cannot find the SS.INI file for user <user name>.

VSS cannot find the SS.INI file for this user. The default location is vss\users\<username>.

This error has the following causes and solution:

  • The SS.INI file is not where VSS expects to find it. (See your VSS Administrator if you have moved this file.)

  • The user's SS.INI has been renamed or deleted.

    You can copy an SS.INI file from another user into this user's folder, then modify it to customize VSS. Your VSS Administrator must then set VSS to look in the proper place.
