How to Set a Working Folder

You must specify a working folder to perform any action that copies a file out of VSS, including the Check Out, Check In, Undo Check Out, Get Latest Version, and Merge commands.

To set a working folder:

  1. Select the project for which you want to set a working folder.

  2. On the File menu, click Set Working Folder to display the Set Working Folder dialog box.

  3. Select the drive with the folder that you'd like to use.

  4. In the Folders box, navigate down to the folder you'd like to use as a working folder.

  5. Click OK. The path to your working folder is displayed in VSS Explorer above the list view.
