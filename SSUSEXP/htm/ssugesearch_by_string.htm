Search by String

See Also

Search strings can be up to 63 characters long.

To search by a string of text:

  1. On the Tools menu, click Find in Files to display the Find dialog box.

  2. Type the string of text for which you are searching.

    Note   If you're using an asterisk in your text string as part of a wildcard search, select the Use Regular Expressions check box. If you're using an asterisk as an ordinary text character, clear the Use Regular Expressions box.

  3. If you want your search to be case-sensitive, select the Match Case check box.

  4. Click OK. VSS will start the search.
