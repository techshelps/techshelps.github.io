File <File> already exists. Overwrite?

This warning has the following cause and solution:

  • You are trying to copy a file over an existing writable file as part of a Check out or Get command. This typically occurs in integration.

    Click Yes if you want the copy in your local folder to be overwritten by the file you are doing the Get or Check out on. 

    Click No if you want to retain the copy in your local folder.
