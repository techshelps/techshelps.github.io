Options Command (Tools Menu)

See Also

Enables you to customize the VSS environment for all users.

Remarks

Click the tab with the information you want. Each tab contains groups of options you can set to alter VSS's behavior in certain situations. The tabs in the SourceSafe Options dialog box change the behavior for all users in VSS Administrator's user list. (Individual users can override some settings by using the Options command on VSS Explorer's Tools menu.)

Dialog Box Tabs

General

Sets general options applicable to a number of commands in VSS.

Project Security

Sets security options, such as whether project security is enabled, and the default user rights assignment.

Web Projects

Designates a project as a Web project.

Web

Sets how certain Internet information is transmitted for Web projects.

File Types

Sets how file type information is determined for all users.
