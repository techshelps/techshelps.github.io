View Files without a Working Folder

See Also

You can view the master copy of a file without placing a local copy of the file in your working folder.

To view a file:

  1. Click the file you want in VSS Explorer.

  2. On the Edit menu, click View File to display the File dialog box

  3. Click the View SourceSafe's copy of this file box, if it isn't already selected.

  4. If you want to specify an editor, click the Advanced button and do so in the last text box.

  5. Click OK.
