The File Types string has a maximum length of <Number> characters. It currently has <Number> characters and needs to be shortened.

This error has the following cause and solution:

  • You are adding files types in the File types included in file group on the File Types tab of the Tools-> Options dialog.  VSS has a limit of 511 characters for any setting in the SS.INI file and you've exceeded this limit.

    Reduce the number of File Groups or the number of file masks used (in any one of the File Groups you are using) to 511 or less characters.
