Shadow_ReadOnly Initialization Variable

See Also

Controls whether files in the shadow folder are read-only. Used in the SRCSAFE.INI file only.

Syntax

Shadow_ReadOnly = {Yes|No}

Example

Specifies that all VSS files in the shadow folder are read-only (the default):

Shadow_Readonly = Yes

Remarks

This variable turns on the read-only flag for all files that VSS places in the shadow folder.

You can also set this variable in the SRCSAFE.INI file with the Set Read Only Flag For All Files option, which is accessed by clicking Options on the Tools menu, then selecting the Shadow Folders tab.
