Cannot find SS.INI file

VSS cannot find your SS.INI file. This error has the following causes and solution:

  • The SS.INI file is not where VSS expects to find it. (See your VSS Administrator if you have moved this file.)

  • You have inadvertently deleted your SS.INI file.

    You can copy an SS.INI file from another user into your VSS folder, then modify it to customize VSS. Your VSS Administrator must then set VSS to look in the proper place.
