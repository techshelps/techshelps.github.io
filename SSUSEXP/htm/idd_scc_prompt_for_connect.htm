Connect to VSS Dialog Box

Allows the user to specify the location of the database they want to connect to.

To display the Connect to VSS dialog box:

  • On the Add-Ins menu, click SourceSafe, and then click Open New SourceSafe Project.

Dialog Box Options

SourceSafe database path

Type the full path and name of the VSS database you want to connect to. Use the Browse button to browse for the full name.

Data path (optional)

Type the identifier for the data project you want. See your VSS Administrator for more assistance with this identifier. (Your VSS administrator must set up this identifier using the Data_Path variable.)

OK

Connects to the specified database, if available.

Cancel

Cancels the database connection.
