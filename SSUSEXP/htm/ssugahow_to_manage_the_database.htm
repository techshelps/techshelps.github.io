How to Manage the Database

As the VSS administrator, you maintain the VSS database by performing the following tasks:

  • Analyze the Data Folder

  • Archive Databases

  • Clean Temporary Folder

  • Lock a Database

  • Restore Databases

  • Undo Checkout of a File or Project

  • Using DDUPD to Take Advantage of Performance Enhancements and Label Promotion

  • Work with Multiple Databases
