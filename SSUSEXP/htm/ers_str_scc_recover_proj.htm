Project <Project> has been deleted and must be recovered before opening

See Also

This error has the following cause and solution:

  • You are trying to act on a project that has been deleted. This error will occur in the integration because it can’t connect to the project in VSS (the project was deleted from VSS).

    Before you can do anything with the project, you must recover the project in VSS using the Recover command. 

    Use the following steps in the VSS Explorer:

    • Select the parent project that contained the deleted project.

    • On the File menu, click Properties.

    • Click on the Deleted Items tab.

    • Select the deleted project, then click Recover.
