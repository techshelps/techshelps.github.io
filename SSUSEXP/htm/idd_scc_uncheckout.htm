Undo Check Out from <Project Name> Dialog Box

Overwrites your changes to a file, and copies the latest checked in version to your development environment working directory.

To use the Undo Check Out command:

  • On the Visual Basic Tools menu, click Undo Check Out.

Dialog Box Options

Files to Un-Check Out

Lists the currently checked out files.  Click the files whose check out you want to undo, or click Select All to select all the listed files.

Advanced

Displays the Undo Checkout Advanced Options.

OK

Confirms the selection.

Cancel

Cancels the selection.

Select All

Selects all files listed.
