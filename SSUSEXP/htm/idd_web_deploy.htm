Deploy Dialog Box

Allows you to set options for your Web site project deployment.

Dialog Box Options

If you enable the Deploy command dialog box on the Command Dialogs tab in the SourceSafe Options dialog box (Tools menu), or hold down the SHIFT key when you click the Deploy command, you have the following options:

To

Type the path where you want VSS to place the files.  You can Browse for the appropriate location.

Make Writable

Sets the read-write flag so the file is writable.  This is not the default behavior, but may be useful if you want to try a temporary change in the file, or want to rename the file after getting it.
