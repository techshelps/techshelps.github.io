<File> is an Explorer link file

This message has the following cause and solution:

  • You dragged a .lnk file into VSS.

    When you drag a .lnk file into VSS, you need to tell VSS whether to add the .lnk file itself to the database, or add the file that the .lnk file points to. If you want to add the file referred to by the .lnk file, click Yes. Otherwise, click No.
