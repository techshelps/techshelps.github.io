Login Dialog Box

Logs you on to VSS.  The Login dialog box is displayed when you start VSS.

To log on to VSS:

  1. In the User Name box, type your username.

  2. Type your VSS password (if you have one).

  3. Select the VSS database you want to log in to, and press ENTER to log on to VSS.  (You can Browse for a different database.)

Access Rights

Any VSS user can use this dialog box.

Remarks

If you are unsure of your VSS username or password, see your VSS administrator.
