Refresh List View

See Also

While you are using VSS, other people may change the contents of the current project. For instance, they may check out and check in files. To stay up-to-date, VSS automatically refreshes its list view periodically. You can change how often VSS autorefreshes by using the File List section on the View tab of the Options command on the Tools menu.

You can also refresh the list view manually.

To refresh the list view manually:

  • From the View menu, click Refresh File List, or press the F5 key.
