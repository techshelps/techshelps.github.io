Source Code Control Options Dialog Box

Customizes your source code control provider environment.

To use the Options command:

  • After you've added a development environment project to your source code control provider, on the Add-Ins menu, click the name of your source code control provider, then click Options.

Remarks

The Options command displays the Source Code Control Options dialog box.

Dialog Box Options

  • Get checked in version of project files when opening a project?

  • Check in files when closing the project?

  • Add files to source code control when adding them to Visual Basic?

  • Remove files from source code control when deleting them from Visual Basic?

For each option, you can specify Yes, No, or Ask.  Ask is the default.

Advanced

Displays advanced options.

OK

Confirms selections.

Cancel

Cancels selections.
