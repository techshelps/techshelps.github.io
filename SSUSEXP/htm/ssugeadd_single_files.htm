Add Single Files

See Also

You can add single files, multiple files, and folders to VSS. You can also add a file using drag and drop. See Add Files and Folders Using Drag and Drop.

To add a single file to VSS:

  1. Select the project you want to add files to.

  2. On the File menu, click Add Files to display the Add File dialog box.

  3. In the Drives list, select the drive where the file resides.

  4. In the Folders list, select the folder where the file resides.

  5. In the List files of type list, select a file type. This is an optional step used to filter the list down to a more manageable size.

  6. In the File name list, select a file and click Add to add it to VSS.

    Note   Files already in the current project are not displayed in the File name list.

  7. VSS displays a dialog box in which you can enter a comment. Enter applicable comment, if any, then click OK.

  8. Click Close to close the Add File dialog box.
