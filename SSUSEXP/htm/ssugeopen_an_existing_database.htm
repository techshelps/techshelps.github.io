Open an Existing Database

See Also

When you run VSS, you must connect to a VSS database where your files are stored. This should happen automatically, but if it doesn't or if you need to connect to a different database, follow the steps below. If a database has not been set up, contact your VSS administrator.

Note   If you've never opened a database before, from the VSS Logon dialog box, click Browse to search for an existing database, then click OK.

To open an existing VSS database from within VSS:

  1. On the File menu, click Open SourceSafe Database to display the Open SourceSafe Database dialog box.

  2. Select a database from the Available databases list.

    — Or —

    To search for an existing database, click Browse. After you locate the database, select its SRCSAFE.INI file, then click Open.

    Note   If the database you want is not in the Available databases list, or you cannot find it using Browse, consult your VSS administrator.

  3. Click Open in the Open SourceSafe Database dialog box.
