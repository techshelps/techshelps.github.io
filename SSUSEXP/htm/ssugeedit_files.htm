Edit Files

See Also

Use the Edit File command to check out and edit a file in one operation. VSS automatically displays the file in the editor associated with that file. For instance, a file with the .doc file type is associated with Microsoft Word, thus the file is opened in Word. To select a different editor, click the Advanced button on the File dialog box.

To edit a file:

  1. Select the file you want in VSS Explorer.

  2. On the Edit menu, click Edit File to display the File dialog box.

  3. Click the Check out this file and edit in your working folder box, if it isn't already selected.

  4. If you want to specify an editor, click the Advanced button and do so in the last text box.

  5. Click OK to display the Check Out dialog box for files and projects.

  6. Specify your working folder in the To box.

    — Or—

    Click Browse to search for and set a working folder.

  7. Click OK.
