You currently have file <File> checked out

See Also

This warning has the following cause and solution:

  • From the command line, you are attempting to check out a file you already have checked out.  This message will only appear if Multiple Checkouts is turned off.

    If you want to check out the file a second time, you will have to get the VSS Administrator to enable Multiple Checkouts.  Otherwise, check in the file before you attempt to check it out again.
