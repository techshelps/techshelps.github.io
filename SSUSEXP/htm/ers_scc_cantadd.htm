Unable to use the selected source code control project.

This message has the following cause and solution:

  • When you add a host project to source code control, you must select a source code control project into which the files will be placed. The source code control provider checks the selected source code control project to make sure it is suitable for the operation. If a problem is found, this error occurs.

Follow the rules imposed by the source code control provider regarding suitable projects for the Add command.
