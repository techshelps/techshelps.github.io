Advanced Comment Dialog Box

Advanced Options

Store Only Latest Version

Indicates you do not want to save old versions of the file(s) being added to the VSS database.  VSS keeps a file history of change records with comments.  However, the older versions of the file are not stored, and you cannot recover them.

Note   To increase performance and save disk space, use this option when storing executable programs in VSS.

Check Out Immediately

Checks out the file to you immediately after adding it to VSS.

Remove Local Copy

Deletes the file from your working folder after adding it to VSS.

File Type

Select an option from the list box:

  • AutoDetect VSS determines the file type.

  • Binary VSS treats the file as a binary file.

  • Text VSS treats the file as a text file.

Note   When adding a folder, VSS ignores the File Types field. However, VSS filters the files added during a folder Add Files operation by the Relevant_Masks setting on the File Types tab in the SourceSafe Options dialog box.
