Create Site Map Dialog Box

See Also

Allows you to create a map of your Web site, using HTML files checked into your VSS project as targets for HTML jumps.

Dialog Box Options

To

Specifies the file containing the resulting HTML site map code. Click Browse to search the file system for a suitable location.

Check in to VSS

Check this box to have the resulting file checked into the VSS project.

OK

Confirms your selections.

Cancel

Cancels your selections.
