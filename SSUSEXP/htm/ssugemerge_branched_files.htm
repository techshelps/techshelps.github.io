Merge Branched Files

See Also

To merge files:

  1. Select the file or project you want in VSS Explorer.

  2. On the SourceSafe Menu, click Merge Branches to display the Merge To dialog box.

  3. In the Projects box, select the project that has the version of the file from which you want to merge changes.

  4. Click Merge to display the Comment dialog box.

  5. If desired, enter a comment.

  6. Click OK.

Note   If you use the Find function while in Visual Merge, only one of the three Visual Merge panes will be searched, namely the pane that the cursor is in.
