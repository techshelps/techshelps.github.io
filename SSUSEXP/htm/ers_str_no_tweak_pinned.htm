You cannot set the Store Latest Version flag for the file <File name> because it is pinned to an explicit version.

This warning has the following cause and solution:

  • You were attempting to change the Store only latest version property of a file that is pinned.

    Find out what version the file is pinned to by using the ss dir <File Name> command.  Then use the ss unpin <File name> command to unpin the file.  At this point you will be able to use the ss properties –L <File Name> command to change the Store only latest version property.  Then pin the file to the version it was originally pinned to.
