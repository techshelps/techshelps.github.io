Share Dialog Box

See Also

Allows you to specify a project to share with the current project.

To display the Share dialog box:

  1. In the project pane of VSS Explorer, select a project into which you want to share another project.

  2. On the SourceSafe menu, click Share.

  3. In the Share with Project dialog box, select a project to share and click Share.

Dialog Box Options

New Name

Enter the name of the project you are sharing with the current project.

Recursive

Check to make the Share operation recursive (include all subprojects and their files).

Comment

Enter a comment for this operation.

OK

Returns to the Share with Project dialog box, which shows the selected project to share with the selected files.

Cancel

Cancels the Share command.
