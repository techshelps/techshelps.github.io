The project <Project name> was not found. Would you like to browse for the project?

VSS tried to open a project that could not be found in your VSS database.  This error has the following cause and solution:

  • You moved or renamed the VSS project in VSS Explorer, then opened your Visual Basic or Visual C++ project that refers to the old VSS project name.

    Click Yes to browse for the project if you know the name of the new project.

    Click No if you don't want the project to be under source code control.  If you click No, you must manually re-add the project to VSS in order to use source code control features in the future.
