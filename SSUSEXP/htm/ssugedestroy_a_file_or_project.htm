Destroy a File or Project

See Also

Warning   When you destroy a file permanently, you cannot recover it within VSS. All previous history information pertaining to that file is lost.

To destroy a file or project:

  1. Select the file or project in VSS Explorer.

  2. On the File menu, click Delete to display the Delete dialog box.

  3. Select the Destroy Permanently check box.

  4. Click OK.
