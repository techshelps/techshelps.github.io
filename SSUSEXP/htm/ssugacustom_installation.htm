Custom Installation

See Also

This option enables you to install only the parts of VSS that you want. It provides the following choices, and shows the amount of space required on your hard disk for each choice:

  • Create VSS database

  • Client program install

  • Administrative program install

  • Enable VSS integration

  • Help files

  • Conversion utilities (Pvcs and Delta)

  • Network client setup

Use the custom installation option when you want to omit parts of the server installation.

Note   The custom installation option is the only way to install the VSS database on your local hard disk and then have it accessible for integration with Visual Basic and Visual C++. For more information on VSS integration with Visual Basic and Visual C++, see documentation for those products.

To perform a custom installation:

  1. Insert the VSS compact disc in the CD-ROM drive.

  2. Double-click Setup.exe.

  3. Follow the setup instructions in the setup wizard.

  4. When asked what type of installation you'd like, click Custom.

  5. Select the custom options you'd like to install, then click Continue.
