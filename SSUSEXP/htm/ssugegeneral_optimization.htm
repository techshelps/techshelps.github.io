General Optimization

See Also

You can set these options on the Local Files tab in the Options dialog box on the Tools menu:

  • Copy keyword-expanded files into working folder — Useful if you use keyword expansion. Selecting this text box forces VSS to copy the file containing expanded keywords back into the working folder after it has been checked into the database. Clearing this check box may increase speed, as it tells VSS not to get a file after checkin.

  • Compare files by — Controls how VSS determines if you already have the latest copy of a file in your working folder. The Time setting sometimes does not behave as you expect when you try to get earlier versions. A solution is to occasionally delete your local copy to force a Get operation to occur.
