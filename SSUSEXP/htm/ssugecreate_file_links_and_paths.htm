Create File Links and Paths

See Also

Sharing a file creates links from the file to multiple projects. For more information on sharing, see Sharing Files. Branching a file creates different paths from project to project. For more information on branching, see Branch and Share Files and Projects.

  • To create a file link, see How to Share Files.

  • To create a file path, see How to Branch Shared File.

Note   To view projects sharing a file, click the Links tab in the Properties command on the File menu. To view the branch history of a file, click the Paths tab in the Properties command on the File menu.
