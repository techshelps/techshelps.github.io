Add Folders

See Also

You can add folders by following the steps below or by using drag and drop. See Add Files and Folders Using Drag and Drop.

To add an entire folder to VSS:

  1. Select the project you want to add a folder to.

  2. On the File menu, click Add Files to display the Add File dialog box.

  3. Select a folder from the Folders list.

  4. Click Add to display the Add Folder dialog box.

  5. Type a comment in the Comment box to describe the file or folder you are adding.

  6. If you want subfolders included, click Recursive.

  7. Click OK to add the folder with a comment.

    — Or —

    Click Cancel to cancel the operation and return to the Add File dialog box. Click Close.
