Delete a User

See Also

When you delete a user from the user list, VSS automatically deletes the SS.INI file that it created for that user.

Note   You can't delete the Admin user.

To delete a user:

  1. From the VSS Administrator user list, select a user.

  2. On the Users menu, click Delete User.

  3. VSS displays a warning box asking "Are you sure you want to delete?"

  4. Click OK.
