Restore Databases

See Also

You can now restore databases using the VSS Administrator program. In previous versions of VSS, Archive and Restore were only available from the command line.

To restore a database:

  1. Open the database into which you want to restore information. See How to Open a Database for more information.

  2. On the Archive menu, click Restore Projects… to display the Restore Wizard.

  3. Click Browse to select the archived file you want to restore.

  4. Click Open.

  5. In step 1 of the wizard, click Next.

  6. In step 2 of the wizard, select the project or projects you'd like to restore.

  7. Click Next.

  8. In step 3 of the wizard, specify where you want the project(s) restored to. You may also add a comment in the Comment box if you'd like.

  9. Click Finish to complete the restore.
