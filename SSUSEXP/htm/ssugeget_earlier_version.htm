Get Earlier Version

See Also

Note   When you Get a file, the file is placed in your working folder as read-only — any modifications you make to the file at this point cannot be saved in the project.

To get an earlier version of a file or project:

  1. Select the item(s) you want to see an older copy of.

  2. On the Tools menu, click Show History to display the History Options dialog box.

  3. Click OK to display the History dialog box.

  4. Select the version of the item you want a copy of.

  5. Click Get to display the Get dialog box.

  6. If you haven't previously set a working folder, VSS will ask you if you'd like to do so now. Click OK, then set your working folder.

  7. Click OK in the Get dialog box. The selected item is then copied from the current project into your working folder.
