Search with a Wildcard

See Also

Search strings can be up to 63 characters long.

Note   You stay in search mode until you explicitly cancel the search.

To search by a wildcard name:

  1. On the View menu, click Search Wildcard Search to display the Search for Wildcard dialog box.

  2. In the Wildcard box, type the part of the file name you know, using an asterisk (*) for parts you don't know.

    For example, if you're looking for a file you think is called TodaysProject, but you can't recall the entire file name, enter Today*.* as your wildcard search. VSS will identify all files that start with the words Today, helping you narrow the search.

    You can also search using multiple masks, separating your search with semicolons, such as *.exe; ?.dll; !ga*.exe.

  3. Select a Search Area.

  4. Click OK to start the search.
