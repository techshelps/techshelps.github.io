<File name> is checked out, replace?

This warning has the following cause and solution:

  • You chose to get a file when it was already checked out.

    When you get a file, a new copy is written to your working folder. If you already have the file checked out, you may have made changes that will be overwritten. In this situation, VSS gives you some options:

    • Click Replace to overwrite the working folder file.

    • Click Merge to merge changes from the working folder file into the VSS database file.

    • Click Leave to leave the working folder file alone (the Get command will be ignored for that file).

    • Click Cancel to cancel the Get operation.

    • Check Apply to all items to have your choices in this dialog box apply to all other files you currently have checked out.
