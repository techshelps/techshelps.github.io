Perform an Exclusive Checkout

See Also

You can check items out exclusively, checking them out so that no one else can check them out while you have the file or project.

There are two ways to achieve this. Your VSS administrator can set an administrative option to disable multiple checkouts for an entire project all the time, or you can do the following.

To perform an exclusive checkout:

  1. Select the file or project you want to check out.

  2. On the SourceSafe menu, click Check Out to display the Check Out dialog box.

    Note   If this dialog box does not appear, you need to display it using the Command Dialogs tab in the Options command on the Tools menu.

  3. Click Advanced.

  4. Clear the Allow Multiple Checkouts check box.

  5. Click OK.

Note   This exclusively checks the file or project out on a session-by-session basis only.

How to Tell if a File is Exclusively Checked Out:

There are three ways to tell if a file is exclusively checked out, including:

  • A red outline around the file icon.

  • The Check Out Status tab in the File Properties dialog box changes. A line is added under the file name that says "Exclusive Check Out."

  • On the command line, the SS Status command indicates the status:
    Ssus.rc    Charry v572    4/08/98 2:37 (Exclusive Check Out)
