Lock a Database

See Also

Locking a database does not automatically lock out those users presently logged on. The VSS administrator should ask users to quit VSS before locking the database.

To lock a VSS database:

  1. From the VSS Administrator Tools menu, click Lock SourceSafe Database to display the Lock Visual SourceSafe Database dialog box.

  2. Select the Lock all users out of Visual SourceSafe check box.

  3. Click OK.

    Important   You will have to clear the Lock all users out of Visual SourceSafe check box before your users will be able to access VSS again.
