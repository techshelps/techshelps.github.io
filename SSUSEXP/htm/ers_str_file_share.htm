File <File> is already open.

This warning has the following cause and solution:

  • Either another user or VSS has the file locked.

    If VSS has the file locked, you can attempt to release the lock by performing a status check of the file or the entire database.

    At a Command Prompt, type the following:

    SS Status $/ -R

    If the file remains locked, contact your network administrator.

    If problems like this continue, you may need to enable VSS locking.
