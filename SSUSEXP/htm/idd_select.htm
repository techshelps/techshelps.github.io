Select Dialog Box

Many of VSS's most common commands such as Get Latest Version, Check Out, and Check In can work on more than one file at once.  For them to do so, however, you must select multiple items before choosing the menu command you want to execute.

To select multiple items with the Select dialog box:

  1. Click in the file pane of VSS Explorer, and then on the Edit menu, click Select.

    VSS displays the Select dialog box.

  2. In the Item box, type a filename using wildcard characters.  For example, type *.* to select all items. 

    Click Select or press ALT+S.

  3. To add additional items to the selection list, separate items with a semicolon in the list.

  4. Click Close to confirm the selection and close the dialog box.

To select items with the mouse:

  • Select an item in VSS Explorer.

    Or

    Hold down the CTRL key as you choose each item you want; to select a block of items, click the first item, hold down the SHIFT key, and click the last item.

To select items with the keyboard:

  • Press SHIFT+SPACEBAR to select each item you want.  Use the arrow keys to move to each.

To deselect items with the Invert Selection command:

  • On the Edit menu, click Invert Selection to change selected items to deselected items, and vice versa.
