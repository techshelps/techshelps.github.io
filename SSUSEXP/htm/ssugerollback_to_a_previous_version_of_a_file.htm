Rollback to a Previous Version of a File

See Also

Caution   When you roll back a file, you lose all changes made to that file after the version you roll back to. For example, if you click on version 5 and then click Rollback, all changes since version 5 are lost.

To rollback a file:

  1. Select the file in VSS Explorer.

  2. On the Tools menu, click Show History to display the History Options dialog box.

  3. Click OK to display the History of File dialog box.

  4. Select a previous version of the file, then click Rollback.
