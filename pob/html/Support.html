Obtaining Technical Support

Technical Support is available via e-mail at support@metaproducts.com.

When requesting support, please remember to state which Windows platform you are using.

Although we have tried to ensure that Portable Offline Browser is as bug free as possible, no software is perfect. If you do find a bug, please let us know. We can’t fix the problem until we know about it.

New versions of Portable Offline Browser are released when new features are suggested, or a bug is found. New releases can always be found on our web site at http://www.metaproducts.com.
