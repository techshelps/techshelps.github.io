Suspending to a File

A download may be temporarily stopped and restarted at a later time using the Suspend to File button on the Ribbon - Tools tab. This will immediately stop current downloads and save the downloading queue state to a specified file.

You may resume the suspended download at any time by selecting Resume from File on the Ribbon - Tools tab. This will not affect any other downloading activity in Portable Offline Browser. All stopped or delayed downloads recorded in the desired file will be added to the downloading queue.

You can also suspend individual Project downloads - right-click a Project that is being downloaded and choose Download | Suspend Project to File. Other Projects will continue downloading.
