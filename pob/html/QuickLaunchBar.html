Address Bar

The Address Bar allows you to begin downloading a Web site in seconds. It works like a speedy wizard.

All you have to do is type URL of the desired Web site and change the Level value, if necessary, by pressing <Tab> key and then the <Up> and <Down> arrows on the keyboard. When done, press the <Enter> key or click the Download button.

Portable Offline Browser will create and immediately begin downloading a new Project with the entered URL and Level values. All other Project parameters will be taken from Defaults.

The Project will be created in the selected Folder.
