How to backup all my Projects, settings, etc.?

If you want to save all Projects, downloaded files and settings you need to make a backup of the download directory (as specified in the Options dialog | Advanced | Directories) and of the directory where Portable Offline Browser is installed. The most important files are with .dat, .map and .mpx extensions.

You may also need to backup other download directories, because every Projects folder may have it's own directory to keep downloaded files.
