Default Properties

Each newly created Project accepts properties of the Default Template, except the URL. The Default Template can easily be changed by clicking the Application button and choosing Templates from the menu.

For more information see the Templates section.

This feature makes adding a new Project very easy. For example, if you prefer downloading Web sites with the download Level set to 10 in most cases, just change the Level setting in the Default Template. This new Level setting will be used as the default for all future Projects unless you explicitly change it.
