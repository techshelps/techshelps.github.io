Internal Browser Toolbar

The Internal browser toolbar provides basic functionality when using the Internal browser.

Buttons (from left to right):

  • Full screen expands Internal browser to the full screen. All other areas of the Portable Offline Browser screen become hidden. To restore Internal browser, click this button again.

  • Back returns to the previously viewed page.

  • Forward undoes the Back operation.

  • Stop stops loading a Web page.

  • Refresh reloads the current Web page.

  • Home opens a list of Portable Offline Browser Projects as a Web page.

There is an address bar to the right of the buttons. An Internet address (URL) may be typed here to open the corresponding page in Internal browser.
