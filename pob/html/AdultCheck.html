How to download Adult check and CyberAge - protected Web sites?

Log on to the protected site with your Web browser. Then copy the address (URL) of the first page in the members area.

Add a new Project to Portable Offline Browser and paste the starting URL, then press the Enter key and type on the new line Referer=, then paste the URL again. You should have the following two lines in the Starting addresses field:

http://www.site.com

Referer=http://www.site.com

Set up all other Project settings as necessary and click OK. Now you can download the site.
