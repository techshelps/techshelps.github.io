Templates

The purpose of Templates is to simplify Projects creation. You can create as many templates as you want with customized properties and choose them when creating a new Project using the New... button (Ribbon - Home tab).

The newly created Project accepts the selected Template settings which could be altered at any time using the Project Properties dialog.

One of the Templates is called Default Template. It will be used when a new Project is created with the help of Drag-and-Drop, paste operation, Address Bar, etc.

To edit the list of Templates and their settings click the Application Button and select Templates from the menu.
