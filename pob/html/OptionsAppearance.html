Options dialog: Appearance

Here you will find different parameters that affect the look and behavior of the Portable Offline Browser user interface.

  • Run in system tray

    When checked, Portable Offline Browser will appear as a small icon in the system tray area near the Windows clock. Otherwise, the main window will be shown in the taskbar. You can use this setting to hide Portable Offline Browser and make it run in the background. When Portable Offline Browser is in the system tray, you can easily invoke the main window by clicking the Portable Offline Browser tray button. To hide it again, click the Minimize button on the window caption.

  • Display splash screen

    When checked, Portable Offline Browser will show the product splash screen while loading.

  • Hide main window when using the key

    Click the field and press the desired keys combination on keyboard to set the hotkey. When using this feature, it is convinient to hide the main window to tray.

    If you are not using the tray option, you may still hide Portable Offline Browser screen. To show it, start Portable Offline Browser again - this will simply show the running application window.

  • Double click behavior

    This option determines the default behavior when double-clicking on a Project in the Projects tree. You may select to view a downloaded Project in the default Web browser or to open the Project properties dialog box.
