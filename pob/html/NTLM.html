How to handle NTLM authentication?

Portable Offline Browser supports NTLM authentication for both Web sites and proxy servers.

While authenticating, the server checks three parameters - Domain, user name and password. In other to enter them correctly, please fill the user name field in the Project Properties (Advanced | Passwords) or in Options dialog (Proxy | HTTP) in the following way:

domain\username

  • note the "\" backslash between the domain and username.
