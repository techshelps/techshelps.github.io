Project Properties dialog: Limits

Allows to stop downloading when exceeding one of the following parameters:

  • Limit downloading by size of loaded files

    If checked, Portable Offline Browser will stop downloading the Project when the size of the downloaded files is greater than the entered value.

  • Limit downloading by number of files

    If checked, Portable Offline Browser will stop downloading the Project when the number of downloaded files exceeds the entered value.

  • Limit downloading by time

    If checked, Portable Offline Browser will stop downloading the Project when the download takes longer than the entered value (in minutes).
