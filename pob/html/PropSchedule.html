Project Properties dialog: Advanced - Scheduling

Scheduling starts a Project download automatically at a specified time.

There are a few scheduling settings:

  • Manually - no automatic Project download;

  • Daily - start downloading every day at specified time;

  • Weekly - start downloading on a specified weekday;

  • Monthly - download every month at specified day;

  • Yearly - download once in a year;

  • Every - start loading the Project every entered time interval. For example if the Time box contains 1:00, Portable Offline Browser will repeat Project downloads every hour.

The time should be a 24-hour value. For example, to set downloading to 5 PM you should enter 17:00.

The day number should be from 1 to 31.

The Project may be set to start downloading automatically only once. After the download is complete the Download interval will return to Manually.
