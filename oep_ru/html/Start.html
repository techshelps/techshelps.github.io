Начало работы

После того, как Offline Explorer Pro установлен и впервые запущен, есть три простых шага для автономного просмотра.

Шаг 1. Создание нового Проекта для загрузки.

Проект определяет, с какой web-страницы должна начаться загрузка и сколько страниц должен загрузить для вас Offline Explorer Pro.

По умолчанию при запуске Offline Explorer Pro откроется окно Мастера нового Проекта. Если это не происходит, нажмите кнопку Мастер на Ленте | вкладка Главная и следуйте по шагам для создания Проекта. В конце мастера Вам будет предложено начать загрузку нового Проекта. Новый созданный Проект будет добавлен к Дереву проектов на левой стороне экрана Offline Explorer Pro.

Шаг 2. Ожидание завершения загрузки.

Новый Проект будет создан и начнёт загружаться. Зелёная стрелка на значке Проекта показывает, что выполняется процесс загрузки. Когда загрузка завершена, зелёная стрелка исчезнет.

После начала загрузки Проекта Offline Explorer Pro начинает скачивать файлы, на которые ссылается URL Проекта (начальный адрес). Каждая ссылка в файле проверяется на соответствие условиям, установленным в свойствах Проекта: уровень, типы файлов, размеры файлов, URL-фильтры, и т.д. Поскольку Offline Explorer Pro скачивает ограниченное количество файлов за один раз, другие файлы помещаются в очередь ожидания загрузки. В строке состояния отображается количество обработанных (загруженных) файлов и количество файлов в очереди. Загрузка Проекта будет завершена, когда все файлы будут загружены и в очереди Проекта больше не останется файлов.

Шаг 3. Автономный просмотр загруженных файлов.

Теперь web-сайт доступен для автономного просмотра. Нажмите правой кнопкой мыши на загруженном Проекте и выберите Обзор. Offline Explorer Pro детектирует лучший способ для отображения сайта во встроенном браузере или в Вашем стандартном WWW-браузере.

Шаги 1-3 Вы можете повторить в любое время. Offline Explorer Pro поддерживает создание почти неограниченного количества Проектов, которые могут загружаться параллельно.
