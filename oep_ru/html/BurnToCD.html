Как записать загруженный web-сайт на CD?

Offline Explorer Pro непосредственно не имеет возможности записи CD, но может подготовить загруженный web-сайт (Проект) для записи на CD.

Выделите пожалуйста нужный Проект и нажмите кнопку Экспорт на Ленте - вкладка Главная. Будет выполнено копирование загруженного сайта в некоторый каталог, который и должен быть позже записан на диск CD-R или CD-RW. В диалоге Экспорт рекомендуется выбрать "Joliet - CD-R 64 symbols" в поле формата имён файлов.

Вы можете также создать Автозапускаемый CD, который будет автоматически запущен для просмотра, когда Вы вставите его в привод. Удостоверьтесь, что отмечена опция "Создать файл содержимого" для упрощения просмотра с CD.

Теперь нажмите кнопку Экспорт и ждите окончания процесса. После этого Вы можете записать экспортированный каталог используя Вашу любимую программу записи на CD.
