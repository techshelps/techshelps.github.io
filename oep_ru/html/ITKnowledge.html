Как загрузить сайты www.ITKnowledge.com или earthweb.com

Сайт www.ITKnowledge.com сложно загрузить. Он использует для аутентификации пользователей формы и cookies.

Однако Offline Explorer Pro версий 1.1 и выше поддерживает загрузку файлов с этого сайта. Например, вы можете скачать книги со следующим URL: http://www.itknowledge.com/reference/library/1556225466/ewtoc.html. Необходимо сделать следующее:

  • Щелкните на кнопке Новый Проект на Ленте - вкладка Главная и создайте новый проект с вышеприведённым адресом URL.

  • В окне Свойств Проекта установите значение уровня до 999, затем перейдите в раздел Дополнительно | Пароли.

  • Введите логин и пароль в соответствующие поля. Нажмите кнопку ОК.

  • Откройте Настройки и установите флажок в "Использовать MS IE Cookies". Нажмите кнопку ОК.

Теперь вы можете начать загрузку Проекта. Однако, если вы хотите иметь только текст книги без других (бесполезных) файлов и сайтов, я рекомендовал бы сделать следующие дополнительные шаги:

  • Откройте ещё раз диалог Свойства Проекта и перейдите в раздел Фильтры URL | Сервер.

  • Выберите "Загружать файлы только с начального сервера".

  • Щёлкните на узле Каталог и выберите "Маска".

  • В нижнем поле выберите вкладку "Включаемые слова" ("Included keywords"). Затем добавьте слова по списку:

    PSUser

    images

    library

    archive

    standard

  • Перейдите на узел Имена файлов и выберите "Маска".

  • В нижнем поле выберите вкладку "Исключаемые слова" ("Excluded keywords"). Затем добавьте слова по списку:

    ^ITK

    ^AITK

    ^CITK

  • Нажмите кнопку ОК.

Теперь будут загружены только необходимые файлы.
