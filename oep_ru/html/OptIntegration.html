Настройки: Дополнительно | Взаимодействие

Эта категория позволяет Вам интегрировать Offline Explorer Pro с Windows.

  • Отслеживать буфер обмена

    С этой отмеченной настройкой Offline Explorer Pro будет отслеживать содержимое буфера обмена. Если в нём появляется URL (строка, начинающаяся с http://, https://, ftp://, mms://, mmst://, pnm:// or rtsp://), то URL будет добавлен как новый Проект.

  • Использовать контекстное меню MS Internet Explorer

    С этой отмеченной настройкой Offline Explorer Pro добавит два элемента в контекстное меню MS Internet Explorer на правой кнопке мыши. Download with Offline Explorer Pro позволяет Вам добавить ссылку как Проект Offline Explorer Pro и загрузить его. Download the current page with Offline Explorer Pro создаёт новый Проект с установкой URL как у адреса текущей страницы в MS IE.

При появлении нового URL в буфере обмена Windows Offline Explorer Pro создаёт новый Проект. Откроется диалог Свойства нового Проекта со стандартными настройками. Но Вы можете изменить это поведение, используя:

  • Использовать шаблон по умолчанию

    Когда эта опция отмечена, то Offline Explorer Pro автоматически добавит новые Проекты в дерево Проектов. Все параметры для Нового Проекта, кроме URL и комментария, будут взяты из шаблона по умолчанию. Вы также можете включить Автоматически начинать загрузку. Загрузка добавленного нового Проекта будет автоматически начата без Вашего участия.

  • Автоматически начинать загрузку

    Отметьте эту настройку, чтобы автоматически без Вашего участия запускать все вставляемые Проекты. Примечание: в этом случае опция Использовать шаблон по умолчанию также должна быть включена.
