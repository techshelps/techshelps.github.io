Настройки: Интернет - Дозвон

Для использования дозвона, встроенного в Offline Explorer Pro, включите его и установите его свойства в диалоге Настройки, раздел Интернет | Дозвон.

Для включения дозвона отметьте поле Включить встроенный дозвон.

Если дозвон включен, то инициализируется выбранное модемное соединение с Интернет и выполняется набор номера Internet Service Provider (поставщик услуг Интернет) при запуске загрузки любого Проекта.

Доступны следующие параметры набора номера:

  • Выбор соединения

    Для инициализации соединения с провайдером Offline Explorer Pro использует стандартные Windows-соединения из папки Мой компьютер | Удалённый доступ. Если у Вас нет этой папки, то Вы должны установить соответствующий сервис Windows. Offline Explorer Pro использует указанные имя пользователя и пароль, так что обеспечьте, чтобы перед использованием Offline Explorer Pro дозвон работал вручную.

  • Имя пользователя

    При дозвоне провайдеру, Offline Explorer Pro будет использовать указанные имя пользователя и пароль. Если они требуются, то введите пожалуйста здесь имя пользователя.

  • Пароль

    При дозвоне провайдеру, Offline Explorer Pro будет использовать указанные имя пользователя и пароль. Если они требуются, то введите пожалуйста здесь пароль.

  • Повтор соединения

    Этот параметр определяет – сколько раз встроенный инструмент дозвона будет пробовать восстанавливать оборванное модемное соединение.

  • Интервал между попытками

    Удалённый компьютер провайдера может быть занят и временно не отвечать на запросы. Пожалуйста установите этот параметр так, чтобы количество попыток набора номера не было бы быстро использовано.

Вы можете использовать Внешний дозвон для немодемных соединений, подобно DSL, Satellite, и другим. Просто разместите путь и имя файла программы дозвона с её параметрами полей набора номера и отключения соединения.

По умолчанию Offline Explorer Pro ожидает 20 секунд инициализации внешнего инструмента дозвона. Если Вам необходимо большее время задержки, то определите его в командной строке:

c:\dir\ext_dialer.exe /param1 /param2 /delay=50

Здесь время задержки составляет 50 секунд.
