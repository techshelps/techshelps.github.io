Параметры командной строки

Offline Explorer Pro поддерживает параметры командной строки для обеспечения интеграции с другим программным обеспечением, таким как сторонние планировщики, и так далее.

Параметры командной строки позволяют запуск загрузки одного и более Проектов, тогда как Offline Explorer Pro закрыт.

Формат такой: <Имя_папки>/<Номер_Проекта_в_этой_папке>. Номера Проектов всегда считаются от нуля. Таким образом, если есть пять Проектов в папке "Sports", и Вы хотите запустить загрузку первого и последнего проектов в папке, то Вы должны ввести:

oe.exe Sports/0 Sports/4

Вы можете указать Проекты из разных папок:

oe.exe Sports/3 Auto/5 Space/1

Перечисляемые Проекты должны отделяться друг от друга пробелом (пробелами). Вы также можете ввести "all" вместо номера Проекта для запуска загрузки всех Проектов в папке:

oe.exe Sports/all

Вы можете использовать параметр "/close"для выхода из Offline Explorer Pro через несколько секунд после завершения всех загрузок. Это полезно, если Вы хотите автоматически загрузить несколько Проектов и затем выйти из Offline Explorer Pro для сохранения памяти и других ресурсов. /CloseNoCountdown заставит Offline Explorer тихо закрыться без отображения диалога обратного отсчёта времени.

oe.exe Sports/3 /close

Если Вы хотите, чтобы Ваш компьютер выключился через несколько секунд после завершения всех загрузок, то Вы можете использовать параметр "/shutdown".

oe.exe Sports/3 /shutdown

Если Вы хотите, чтобы Offline Explorer Pro отключил модемное соединение с Интернет через несколько секунд после завершения всех загрузок, то Вы можете использовать параметр "/hangup".

oe.exe Sports/3 /hangup

Для просмотра некоторого Проекта во Встроенном браузере примените Папка/номер после параметра /Browse:

oe.exe /Browse Sports/3

Для выполнения резервного копирования Проекта добавьте к его Папку/номер после параметра /Backup=<file>:

oe.exe /Backup=c:\file.boe Sports/3

Вы можете включить автоматическое создание и ведение журнала используя синтаксис:

oe.exe /log=c:\dir\file.log

Для остановки записи журнала запустите другую копию Offline Explorer Pro с параметром “/nolog”.

Для фильтрации информации, содержащейся в журнале, используйте параметр /LogFilter=1111101. Каждое число разрешает (1) или запрещает (0) вывод следующей информации: соединение, прогресс загрузки, подробную информацию (заголовки и прочее), отключение, сообщения очереди, доступ к встроенному web-серверу и отклоненные URL.

По умолчанию, если размер журнала превышает 100 Мб, то Offline Explorer Pro очищает его полностью и начинает с нуля. Для изменения предела используйте параметр /MaxLogSize=10, где число определяет предельный размер журнала в мегабайтах.

Для приостановки текущей загрузки:

oe.exe /Suspend

Для продолжения загрузки, используйте параметр /Resume.

Если Вы ранее приостановили загрузку в файл .wdq, то Вы можете применить в командной строке имя файла .wdq для продолжения загрузки в Offline Explorer Pro:

oe.exe "c:\path\myqueue.wdq"

Для приостановки текущей загрузки в файл используйте следующий параметр командной строки:

oe.exe /SuspendToFile=c:\dir\file.wdq

Таким же способом Вы можете восстановить Проект из файла резервной копии:

oe.exe "c:\path\mybackup.boe"

При запуске Offline Explorer Pro загружает закладки/избранное/hot-списки, найденные в Вашей системе. Это может занять некоторое время. Если Вы хотите пропустить загрузку, то используйте этот параметр:

oe.exe /NoBookmarks

По умолчанию Offline Explorer Pro создаёт Карту Проекта для каждого Проекта при его загрузке. В Карте просто перечисляются все файлы, загруженные Проектом. Этот список используется при удалении Проекта со всеми его файлами, для резервного копирования/экспорта/очистки/печати/статистики/поиска по содержанию/извлечения данных и в других функциях. Однако если Вы хотите пропустить поддержку Карт, запустите Offline Explorer Pro используя следующий параметр:

oe.exe /NoMap

Если Вы выбрали опцию сворачивания иконки в трей, но хотите скрыть программу (Offline Explorer Pro работает в данном случае незаметно), используйте:

oe.exe /NoTray

Вы можете добавить новый Проект из командной строки:

oe.exe "/new=url=http://www.server.com/file.htm,template=имя_шаблона,folder=имя_папки_проекта"

Если Вы определите в вышеупомянутой строке run=true то Offline Explorer Pro создаст Проект и автоматически начнёт его загрузку:

oe.exe "/new=url=http://www.server.com/file.htm,url=POST=p1=v1&p2=v2,url=Referer=http://www.srv.com/,run=true,level=3"

Создание нового Проекта, его загрузка, экспорт, удаление Проекта и загруженных файлов:

C:\Program Files\Offline Explorer Pro\oe.exe "/new=url=http://www.server.com/file.asp,template=Template1,folder=MyFolder,run=true,level=3,url=Additional=DeleteProjectFiles;DeleteProjectSettings;AutoExport=c:\export\"

Вы можете указать последовательность запуска по время работы Offline Explorer Pro:

oe.exe /StartSequencer

Вы можете производить в файлах Проектов поиск по содержанию и сохранять список найденных файлов в текстовый файл:

oe.exe "/FindContents=Folder/5,c:\directory\file.txt,keyword1 keyword2,05/24/2003 5:20 PM"

Отключение открытия Мастера новых проектов при запуске программы:

oe.exe /NoWizard

Вы можете запустить Offline Explorer Pro с максимизированным встроенным браузером:

oe.exe /MaximizeIB

Вы можете запустить Offline Explorer Pro минимизированным:

oe.exe /Minimize

Для выхода из выполняющегося Offline Explorer Pro немедленно (без ожидания завершения текущей загрузки или завершения экспорта) используйте:

oe.exe /Exit

Ожидание завершения текущей загрузки, ожидание завершения текущей операции экспорта и выход Offline Explorer Pro:

oe.exe /Close

или

oe.exe /CloseNoCountdown

Изменение текущего предела скорости:

oe.exe /Speed=2

Значение параметра может быть от 0 до 3, где 0 – фоновая скорость и 3 - максимальная.

Изменяйте общее количество Интернет-соединений (каналов загрузки) используя:

oe.exe /Channels=10

Для изменения времени задержки между загрузками:

oe.exe /DownloadsDelay=5

Запуск новой копии Offline Explorer Pro:

oe.exe /Instance

Чтобы пропустить показ страницы приветствия во внутреннем браузере:

oe.exe /NoWelcomePage

Пропустить сохранение файлов Проектов при выходе (может быть полезно при работе с несколькими копиями программы):

oe.exe /NoSaveToFile

Использовать другой каталог для Проектов, Карт и шаблонов:

oe.exe /ProjectsDir=c:\OEProjects\
