Лента: Главная | Импорт

Используйте меню Главная | Импорт для добавления нового Проекта, получая URL из Избранного разных web-браузеров, Закладок или Hotlist.

В зависимости от браузеров, установленных на Вашем компьютере, Вы можете увидеть следующие меню:

MS IE Favorites - отображает список Избранного Microsoft Internet Explorer.

Navigator Bookmarks - отображает список закладок Netscape Navigator 2.x и 3.x.

Communicator Users Bookmarks - отображает список закладок Netscape Communicator, разделённый на группы зарегистрированных пользователей.

Netscape 6.x-7.x Users Bookmarks - отображает список закладок браузера Netscape, разделённый на группы зарегистрированных пользователей.

Opera Hotlist - отображает Hotlist браузера Opera 2.x - 9.х.

Mozilla Firefox - отображает список закладок браузера.
