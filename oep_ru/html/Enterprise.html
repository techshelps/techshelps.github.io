Offline Explorer Enterprise

Offline Explorer Enterprise объединяет лёгкость использования Offline Explorer с мощью Offline Explorer Pro и добавляет:

  • Способность загружать до 100 миллионов URL в Проекте

  • OLE-автоматизация интерфейса включает управление загрузками из других приложений

  • Возможность обновления Проектов с удалёнными файлами

  • Создание файлов Google Site Map

  • Многопоточный анализ с использованием всех ядер CPU

Offline Explorer Enterprise - идеальный продукт для компаний, которые хотят иметь свои собственные решения для автономного просмотра. Мы поместили в него весь наш опыт автономного просмотра (начиная с января 1997 года).

Возможность огромных закачек

Версия Enterprise была оптимизирована для использования как можно более малой памяти во время выполнения загрузок. Мы переработали очерёдность анализа и формат карты файлов. Файлы Карт больше не загружаются в память, а постоянно содержатся на диске. Offline Explorer Enterprise читает только необходимые части карты для отображения в Карте Проекта или для добавления новых вхождений.

Файлы Карты имеют теперь расширение .mpx. Они несовместимы с другими версиями Offline Explorer. Offline Explorer Enterprise при необходимости преобразует старые файлы карт в новый формат, но обратная операция не выполняется. Так что, если Вы устанавливаете Offline Explorer или Offline Explorer Pro поверх версии Enterprise, то некоторая информация карт может быть утеряна. Файлы Проектов и другие форматы файлов - те же самые, поэтому все настройки, Проекты и шаблоны будут сохраняться.

Создание Ваших собственных решений для автономного просмотра

OLE-автоматизация открывает Offline Explorer Enterprise для разработчиков решений. Теперь можно написать программу, используя Visual C++, Visual Basic, Delphi или другую COM-совместимую среду разработки для запуска новых загрузок и обработки загруженных web-сайтов через Offline Explorer.

Offline Explorer Enterprise обеспечивает полный доступ к его папкам, Проектам и загрузкам. Полная помощь по OLE-автоматизации подробно рассматривается в разделе Содержание.

Мы планируем расширять интерфейс OLE-автоматизации в каждой новой версии. Если Вы нуждаетесь в подробной информации или если Вы хотите добавить к этому больше функциональных возможностей, то пожалуйста напишите нам.

Обновление Проектов с удалёнными файлами

Теперь Вы можете удалить загруженные Проектом файлы, а Offline Explorer Enterprise всё равно сможет загрузить только новые или обновлённые файлы с web-сайта.

Информация о файлах, которые Вы удаляете, сохраняется в компактной базе данных.

Например, если Вы загрузили коллекцию огромных PDF-файлов, которые записали на CD, Вы можете удалить эти файлы с Вашего жёсткого диска используя команду Файл | Удалить | Только файлы Проекта. Когда Вы в следующий раз запустите загрузку Проекта, Offline Explorer Enterprise определит изменение любого из удалённых файлов и загрузит только новые или обновлённые файлы.

Эта возможность сохраняет свободное место на диске, предотвращая хранение дубликатов файлов.
