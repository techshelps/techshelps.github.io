Вкладки служат для переключения между деревом проектов, картой, уровнями и менеджером очереди.

Проекты

Дерево Проектов помогает организовать Проекты и выполнение загрузок.

Читайте больше об организации Проектов здесь.

Карта

Карта обеспечивает графическое представление структуры загруженного сайта, отображая - какие web-серверы были загружены, их каталоги и файлы. Каждый файл, каталог или сервер могут быть открыты с помощью заранее выбранных браузеров или инструментов. Они также могут быть удалены с диска.

Для просмотра списка доступных инструментов выберите меню Вид или щёлкните в Карте правой кнопкой мыши на файле, каталоге или сервере.

Есть два режима отображения карты:

  • Отображаются все загруженные web-сайты.

  • Отображаются только web-сайты, загруженные выделенным Проектом. В этом режиме вкладка Карта использует файл карты Проекта.

Для изменения режима карты выберите из меню Настройки | Дополнительно.

Вы также можете показать меню Проводника Windows для каждого файла в карте. Удерживая нажатой клавишу Ctrl, щёлкните правой кнопкой мыши на файле или папке в карте.

Уровни

Эта вкладка показывает вам web-страницы со всеми их ссылками. Некоторые ссылки, возможно, ещё не загружены - они отображаются серым цветом. Для загрузки отсутствующего файла выберите его в дереве уровней и щёлкните по кнопке Загрузить. Также вы можете выбрать множество файлов, используя сочетания клавиш Shift+click или Ctrl+click.

Очередь

Менеджер очереди показывает список всех файлов, которые будут загружены. Если вы запускаете сразу загрузку нескольких проектов, каждый из них будет иметь собственную очередь. Вы можете выбрать нужный Проект из списка вверху вкладки Очередь.

Вы можете остановить загрузку любого файла в очереди используя щелчок правой кнопкой мыши на файле и выбирая в контекстном меню Остановить.

Обратите внимание, что все файлы в очереди имеют цвет. По умолчанию Offline Explorer Pro назначает разные цвета для разных типов файлов (основываясь на расширениях файлов). Вы также можете щёлкнуть правой кнопкой мыши на любом файле и использовать контекстное меню - подменю Цвета для выбора и назначения разных цветов.
