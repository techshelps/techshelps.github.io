Организация дерева Проектов

Дерево Проектов организовывает Проекты группировкой по папкам. Каждый Проект или папка могут быть при необходимости удалены или изменены.

Процесс создания нового Проекта поясняется здесь. Для создания новой папки кликните по кнопке Новая папка на Ленте, вкладка Главная.

Последующие операции требуют выбора нужного Проекта или папки. Выбрать сразу много Проектов или папок можно удерживанием клавиши Ctrl или Shift и выбирая Проекты или папки левой кнопкой мыши.

Вы можете также переместить любой Проект или папку в другое место используя drag-and-drop (перетаскивание мышью).

Операция с буфером обмена полезна в случае, если вы хотите переместить Проект на другой компьютер - копируйте его в буфер обмена, а затем вставьте в некоторый текстовый файл (например, используя Notepad), далее переместите этот текстовый файл на другой компьютер и затем вставьте Проект в Offline Explorer Pro. Вы также можете вставить Проект в окно Файл | Шаблоны для создания нового шаблона, базирующегося на настройках Проекта.

Последняя операция Удаление может быть отменена через пункт Отменить удаление на панели быстрого доступа.

Только в версии Professional:

Для создания вложенных папок используйте клавишу Ctrl. Например, для создания новой папки внутри существующей, выберите папку, кликните по кнопке Новая папка на Ленте, вкладка Главная; удерживая клавишу Ctrl, дважды щёлкните на кнопке Новая папка и после этого отпустите клавишу Ctrl.

Вы также можете использовать drag-and-drop для перетаскивания отдельной папки внутрь другой удерживая клавишу Ctrl при перетаскивании папки.

Чтобы развернуть или свернуть все папки используйте соответственно сочетания клавиш Ctrl [Numpad+] и Ctrl [Numpad-].
