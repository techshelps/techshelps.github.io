Настройки: Дополнительно

Здесь Вы найдёте различные параметры, затрагивающие просмотр и поведение пользовательского интерфейса Offline Explorer Pro.

  • Карта

    Выбор для отображения файлов, которые были загружены для выбранного Проекта на закладке Карта или отображение всех web-сайтов и файлов в каталоге загрузки.

  • Автосохранение

    Если опция отмечена, Offline Explorer Pro сохраняет все изменения, сделанные в Настройках, Defaults, дереве Проектов и настройках Проекта в указанный интервал времени. Эта опция отключена и по умолчанию установлено значение времени – пять минут.

  • Работа в области списка задач

    Когда Offline Explorer Pro запускает любой инструмент для обзора, правки или просмотра загруженных файлов, то окно приложения инструмента будет минимизировано, максимизировано или оставлено в обычном виде согласно этой настройке.

  • "Х" кнопка минимизирует окно программы

    Когда опция отмечена, Offline Explorer Pro будет появляться в виде маленького значка в системном трее возле часов Windows. В противном случае главное окно будет доступно через панель задач. Вы можете использовать эту настройку для того, чтобы скрыть Offline Explorer Pro и указать ему работать в фоновом режиме. Когда Offline Explorer Pro находится в системном трее (системном лотке), то Вы можете легко вызвать главное окно программы, щёлкнув на значке Offline Explorer Pro в трее. Чтобы скрыть его снова, щёлкните по кнопке Minimize в заголовке окна программы.

  • Заставка при запуске программы

    Когда опция отмечена, Offline Explorer Pro будет отображать при запуске заставку программы.

  • Двойной щелчок на проекте

    Эта опция определяет заданное по умолчанию поведение при двойном щелчке на Проекте в дереве Проектов. Вы можете выбрать просмотр загруженного Проекта в стандартном web-браузере или открытие диалогового окна Свойства Проекта.
