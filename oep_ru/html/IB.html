Встроенный браузер

Offline Explorer включает панель Встроенного браузера для простого и быстрого просмотра загруженных web-сайтов.

По умолчанию, когда Вы выбираете пункт Обзор в контекстном меню, загруженная страница будет открываться во Встроенном браузере.

Вы можете легко управлять Встроенным браузером, используя его панель инструментов.

Встроенный браузер может быть легко включен или отключен через Ленту - вкладка Вид.

Вы можете открывать множество вкладок для просмотра сразу нескольких страниц. Нажмите клавишу V, чтобы увидеть полный список всех открытых вкладок.

Для просмотра Проекта в новой вкладке, щёлкните правой кнопкой на Проекте и выберите "Просмотр в новом окне".
