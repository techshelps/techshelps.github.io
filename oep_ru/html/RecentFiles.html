Как получить список последних загруженных файлов Проекта?

Есть простой способ увидеть, какие файлы были обновлены (загружены) в течение последней загрузочной сессии для некоторого Проекта.

Выберите пожалуйста нужный Проект, кликните по кнопке найти содержание на Ленте, вкладка Главная, отметьте "Только в недавно загруженных файлах" и нажмите кнопку Поиск. Вы получите список всех файлов, которые были обновлены или недавно загружены в последней сессии Проекта. Теперь Вы можете дважды щёлкнуть на нужном файле для его просмотра во встроенном браузере.
