Размещение загруженного web-сайта на USB-диске для просмотра его на других компьютерах

Вы можете подготовить web-сайт к автономному просмотру без наличия любого дополнительного программного обеспечения. Всё, в чём Вы нуждаетесь, это - браузер (MS Internet Explorer, Netscape, Opera, и так далее). Когда загрузка сайта на Ваш жёсткий диск завершена, пожалуйста выберите соответствующий Проект, кликните по кнопке Экспорт на Ленте - вкладка Главная и выполните экспорт сайта на диск Вашего USB (например, диск G:\).

Удостоверьтесь пожалуйста, что отмечена опция "Иcпoльзoвaть cтaндapтныe pacшиpeния для извecтныx типoв фaйлoв" перед выполнением экспорта. Вы также можете отметить опцию "Сделать автозапускаемый CD". Если система поддерживает автозапуск диска USB, то сайт будет автоматически открыт для просмотра.

Если автозапуск не работает с диском USB, то у Вас будет возможность открыть файл contents.htm на диске USB для начала просмотра сайта. В большинстве систем достаточно двойного щелчка в Проводнике Windows на файле contents.htm во время обзора содержимого диска USB.
