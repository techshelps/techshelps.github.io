Управление расписанием

Управление расписанием – это полезный инструмент для осмотра или изменения порядка, в котором запланированы автоматические загрузки ваших Проектов.

Он разделяет Проекты на зоны времени в одном из четырёх режимов:

Ежедневно - отображает все Проекты, запланированные для ежедневной загрузки. (Формат времени 0 - 24 часовой).

Еженедельно - для Проектов, которые должны загружаться каждую неделю. (Формат времени – с Воскресенья до Субботы).

Ежемесячно - только Проекты, которые должны быть загружены каждый месяц. (Формат времени – с 1-го по 31-й день).

Ежегодно - для планирования ежегодной загрузки. (Формат времени – с Января по Декабрь).

Каждый Проект отображается с синей стрелкой. Подробности Проекта отображаются на верхней панели, когда курсор мыши находится над этой синей стрелкой.

Вы можете изменять запланированное время загрузки Проекта перемещая его стрелку Вашей мышью (левая кнопка мыши должна быть нажата и удерживаться в этом состоянии). Перетаскивание Проекта из Управления расписанием изменит его состояние на состояние загрузка Вручную. Двойной клик мышью на Проекте отобразит панель Расписание в диалоге Свойства Проекта.

Вы можете перетащить Проект из Дерева Проектов в Управление расписанием для того, чтобы запланировать его состояние и время.

Последовательность запуска

Другая полезная возможность – это Последовательность запуска. Вы можете перетащить и бросить Проекты на нём для того, чтобы загрузить их один за другим. Когда Проект из последовательности загружен, начинает загружаться следующий.

Вы можете запустить и остановить последовательность запуска, используя поле Пауза. Если поле Цикл отмечено, то последовательность запуска запустит первый Проект после того, как последний завершит загрузку.

Двойной клик мышью на Проекте в Последовательности запуска приводит к выбору Проекта в дереве. Это позволит Вам запустить или остановить загрузку вручную. Используйте контекстное меню на правой кнопке мыши для изменения порядка в последовательности запуска или для удаления Проектов.
