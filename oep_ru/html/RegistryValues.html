Недокументированные значения реестра

Ключи системного реестра.

Offline Explorer Pro:

HKEY_CURRENT_USER\Software\Metaproducts\Offline Explorer Pro\Parameters

Если вы хотите изменить параметры реестра, сначала выйдите из Offline Explorer Pro. Далее выполните команду regedit.exe и перейдите к вышеуказанному ключу реестра. Сделайте изменение и затем запустите Offline Explorer Pro.

Таблица 1. 

ЗначениеТипДействие
ParsingPriorityDWORDОпределяет приоритет выполнения анализа: 0 - обычный, 1 - низкий, 2 – самый низкий, 3 - бездействие.
ParsingMaxDWORDОграничение очереди анализа (в файлах). Если очередь для анализа превышает указанный предел, загрузка новых файлов приостанавливается.
OvrSymbolString%& по умолчанию – определяет имена каталогов предохранения от перегрузки.
MaxOvrFilesDWORDМаксимальное количество файлов в каталоге, при превышении этого предела создаются подкаталоги.
LogLimitDWORDМаксимальное количество строк в журнале, при превышении этого предела журнал очищается.
ClientIDStringRTSP ClientID: содержание поля.
DefaultDocStringdefault.htm по умолчанию. Стандартное имя файла, если URL заканчивается символом наклоненной вправо черты.
ASPAsHTMLDWORD1 - Offline Explorer экспортирует все ASP-файлы как HTML.
GlobalAcceptLanguageStringДобавление Accept-Language: во все заголовки HTTP-запросов.
ExportReplaceCharStringЗамена символов для экспорта 8+3.
SkipHTMLOnExportDWORD1 - Offline Explorer не добавляет тег <HTML> в web-страницы.
HideCheckNewVersionDWORD1 прячет на панели инструментов кнопку Проверка обновлений и аналогичную установку в Настройках.
SkipIndexInDDDWORD1 - Offline Explorer Pro не создаёт индексный файл default.htm в каталоге Download.
