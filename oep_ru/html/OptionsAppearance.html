Настройки: Внешний вид

Здесь вы найдете различные параметры, которые влияют на внешний вид и поведение пользовательского интерфейса Offline Explorer Pro.

  • Работа в системном трее

    Когда опция отмечена, Offline Explorer Pro будет появляться в виде маленького значка в системном трее возле часов Windows. В противном случае главное окно будет доступно через панель задач. Вы можете использовать эту настройку для того, чтобы скрыть Offline Explorer Pro и указать ему работать в фоновом режиме. Когда Offline Explorer Pro находится в системном трее (системном лотке), то Вы можете легко вызвать главное окно программы, щёлкнув на значке Offline Explorer Pro в трее. Чтобы скрыть его снова, щёлкните по кнопке Минимизировать в заголовке окна программы.

  • Заставка при запуске программы

    Когда опция отмечена, Offline Explorer Pro будет отображать при запуске заставку программы.

  • Скрыть главное окно по нажатию клавиши

    Щелкните в этом поле и нажмите нужную комбинации клавиш на клавиатуре, чтобы назначить горячую клавишу. Эта функция используется, чтобы скрыть главное окно в трей.

    Если Вы не используете опцию трея, Вы все равно можете скрывать экран Offline Explorer Pro. Чтобы показать его, запустите Offline Explorer Pro еще раз - тогда отобразится окно запущенного приложения.

  • Двойной щелчок на проекте

    Эта опция определяет заданное по умолчанию поведение при двойном щелчке на Проекте в дереве Проектов. Вы можете выбрать просмотр загруженного Проекта в стандартном web-браузере или открытие диалогового окна Свойства Проекта.
