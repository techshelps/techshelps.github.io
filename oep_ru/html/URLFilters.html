Свойства Проекта: Фильтры URL

Фильтры URL позволяют Вам точно указать – какие ссылки должны быть загружены Offline Explorer Pro. Они группируются по четырём категориям - Протокол, Сервер, Каталог и Имя файла.

Есть также список Пропускать адреса во время загрузки для определения Вами полных имён файлов, которые будут исключены из Проекта загрузки. Ключевые слова в этом списке не поддерживаются. Вы можете также определить каталоги и серверы для их пропуска. Для этого добавьте пожалуйста полный URL сервера или имя каталога, заканчивающееся наклоненной вправо чертой.

Примеры:

http://www.server.com/dir/file.html

Пропуск ссылок, имеющих точно такой URL, как указан выше.

http://www.server.com/dir/

Пропуск всех ссылок в .../dir/ папке.

http://www.server.com/dir/file*

Пропуск всех ссылок, которые начинаются с вышеуказанного участка. Например .../file.htm, .../file.asp?id=5 и так далее.

Пропуск списка URL доступен также из карты проекта.

В разделе Фильтры URL - Имя файла вы можете также фильтровать ссылки по их имени, используя ключевые слова подобно этому:

link:Football

Примечание: Offline Explorer Pro пропускает URL если любая настройка Фильтров URL или Фильтра файлов не разрешает его загрузку (даже если все другие настройки Проекта разрешают загрузку URL).
