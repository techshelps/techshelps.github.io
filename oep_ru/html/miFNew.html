Диалог Новый проект

Диалог Новый Проект позволяет Вам создать новый Проект или Папку в дереве Проектов.

Для создания новой Папки выберите значок Folder в закладке Стандартные и нажмите кнопку OK. Новая папка будет добавлена в конец дерева Проектов.

В версиях Pro и Enterprise, нажатие и удерживание клавиши Ctrl приведёт к созданию новой папки внутри выбранной в дереве Проектов папки.

Вы можете создать новый Проект просто выбрав значок Project в закладке Стандартные или выбрав шаблон на вкладке Шаблоны. Когда выбор сделан, нажмите кнопку OK для появления диалога Свойства Проекта. После изменения настроек Нового Проекта нажмите кнопку OK и новый Проект будет добавлен в текущую папку дерева Проектов.
