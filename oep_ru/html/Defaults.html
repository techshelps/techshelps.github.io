Свойства по умолчанию

Каждый создаваемый новый Проект принимает свойства шаблона по умолчанию, кроме URL. Шаблон по умолчанию может быть легко изменён щелчком по кнопке Приложение и выбором Шаблоны из меню.

Для получения дополнительной информации смотрите раздел Шаблоны.

Эта возможность делает создание нового Проекта очень простым. Например, если Вы предпочитаете загружать web-сайты с установкой значения уровня в большинстве случаев равным 10, то можно изменить установку значения уровня в Стандартном шаблоне на такое значение. Это установленное значение уровня будет использоваться как стандартное значение для всех будущих новых Проектов, если Вы его не измените.
