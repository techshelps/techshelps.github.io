Загрузка файлов BitTorrent

Вы можете использовать Offline Explorer Pro для загрузки протокола BitTorrent. Это можно сделать двумя способами:

  1. Нажмите кнопку Открыть файл Torrent и выберите файл .torrent на вашем жёстком диске. Будет создан Проект, включающий путь к этому файлу в поле URL. Также вы увидите диалог выбора файлов для загрузки.

  2. Создайте новый Проект для загрузки ссылки в файл .torrent или сайт с такими файлами. Если вы разрешили загрузку протокола BitTorrent в диалоге Свойства Проекта - Фильтры URL - раздел Протокол, Offline Explorer Pro будет следовать по ссылкам в файлах .torrent, загружать их и начнёт загрузку файлов, указанных в трекерах .torrent.

Когда вы видите загрузку файлов BitTorrent в панели загрузочных соединений, Вы можете, дважды щелкнув по этому соединению, просмотреть дополнительные подробности, выбрать файлы для загрузки и пропуска, и так далее.
