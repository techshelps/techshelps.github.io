Свойства папки

Вы можете ввести имя папки, которое появится в дереве Проектов.

  • Каталог загрузки для папки

    Назначение каталога для папки. Все Проекты, размещённые в этой папке, будут использовать этот каталог для хранения файлов и сайтов.

    Эта возможность позволит Вам указать несколько мест для хранения загруженных файлов. Например, Вы можете группировать ваши Проекты в папках с разными каталогами загрузки, чтобы с пользой использовать свободное место на разных дисках или ограничить доступ других пользователей (если у вас установлен Windows NT).

    Снимите флажок с опции, если хотите, чтобы для этой папки использовался каталог загрузки по умолчанию из общих настроек.
