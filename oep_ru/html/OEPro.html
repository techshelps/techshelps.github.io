Offline Explorer Pro

Мы получили много запросов о введении новых возможностей в Offline Explorer; однако, не все из них полезны для большинства пользователей. Для того, чтобы сохранить Offline Explorer простым и удобным в использовании, мы решили разработать специальную версию, названную Offline Explorer Pro.

Появление новой версии не подразумевает прекращение разработки стандартной версии. У нас есть планы развития на обе эти версии, поэтому новые версии Offline Explorer и Offline Explorer Pro периодически будут выпускаться.

В настоящее время Offline Explorer Pro имеет следующие возможности, не включенные в стандартную версию:

  • Отдельные каталоги загрузки для папок Проектов

  • Вложенные папки в Дереве Проектов.

  • Макрокоманды URL.

  • Множество URL в каждом Проекте.

  • Подсветка синтаксиса для URL, макрокоманд URL и дополнительных параметров. Автозавершение ввода макрокоманд URL.

  • Способность сохранять копии оригинальных исходных web-страниц с неизменёнными ссылками (файлы .primary).

  • Возможность сохранить старые копии загруженных файлов.

  • Менеджер очереди загрузок.

  • Возможность изменения каталога для хранения Проектов, шаблонов и карт Проектов.

  • Улучшенный дизайн панели сообщений.

  • Поддержка протокола MMS (потоковые файлы Microsoft Media Player).

  • Поддержка протокола RTSP (потоковые файлы Real Player).

  • Поддержка протокола PNM (потоковые файлы старой версии Real Player).

  • Извлечение данных - обработка web-сайтов после их загрузки.

  • Возможность запрета доступа к встроенному HTTP-серверу с других компьютеров.

  • Возможность настройки уровня ссылок, загружаемых с других серверов.

  • Автоматический экспорт загруженных Проектов.

  • Настраиваемые referrers, cookies и post-запросы.

  • Загрузка или экспорт файлов непосредственно в кэш MS IE.

  • Экспорт загруженных файлов в обычный каталог без вложенных подкаталогов.

  • Экспорт загруженных файлов в архивный MHT-файл или файл HTML Help (.CHM).

  • Автоматическое удаление старых файлов Проекта.

  • Настройка файлов Contents.htm перед экспортом web-сайтов.

  • Дополнительное окно описания Проекта для того, чтобы добавить к нему многострочный комментарий.

  • Способность печати web-сайта полностью.

  • Фильтр изображений по их размеру в пикселях (поддержка установки мин/макс размеров для их загрузки и список нежелательных для загрузки размеров).

  • Режимы дополнительной проверки изменения файлов: проверка размеров файлов и пропуск существующих медиа-файлов.

  • Способность сбора адресов E-mail из загруженных web-сайтов.

  • Способность предотвращать переполнение каталога загрузки.

  • Свойства Проекта | Фильтры Содержания для загрузки только страниц, содержащих ключевые слова.

  • Исследование простых Web-форм.

  • Возможность замены адресов URL.

  • Способность ограничения количества соединений в Проекте и паузы индивидуальных загрузок Проекта.

  • Встроенный редактор с подсветкой синтаксиса HTML, CSS и JavaScript-файлов.

  • Возможность автосохранения для сохранения страниц при их просмотре.

Примечание: HTTPS требует установленного в системе Microsoft Internet Explorer 4.0 или более поздней версии. Вы можете загрузить последнюю версию по ссылке http://www.microsoft.com/windows/ie/.
