Как установить Offline Explorer Pro без прав администратора на Windows XP/Vista/7/8/10?

Есть только одна вещь, которую нужно сделать после установки - пожалуйста, разрешите доступ для чтения/записи к папке, в которую установлен Offline Explorer Pro. По умолчанию она находится в папке Program Files.

Для некоторых пользователей без прав администратора могут быть отключены операции записи в этой папке. В этом случае Offline Explorer Pro, возможно, будет не в состоянии сохранять свои данные и настройки Проектов при выходе из программы.

Другой путь состоит в изменении папки размещения Проектов, Карт и Шаблонов в некоторое иное место, полный доступ к которому разрешён пользователю. Вы можете выполнить это, используя диалог Настройки | Дополнительно | Размещение.
