Свойства Проекта: Фильтры содержания

Фильтры Содержания позволяют Вам определить ключевые слова для поиска внутри всего текста загруженных страниц и выполнить некоторые действия.

Например, у Вас может возникнуть желание найти что-то на некотором сайте и загрузить только информацию, которая Вас интересует.

Если некоторый сайт не позволяет Вам загружать выше некоторого предела, то Вы можете указать Offline Explorer Pro остановить загрузку, когда загружена страница с определённым сообщением об ошибке.

Вы можете ввести любое количество слов для поиска их в содержимом загруженных файлов. Разделяйте слова пробелом. Вы можете использовать символы кавычек для поиска точных последовательностей слов.

Например:

sport news "new software"

Двойные кавычки представляют одиночный символ кавычек для поиска внутри web-страниц:

Wo""rd

Когда Offline Explorer Pro загрузит HTML-страницу, это очистит все HTML-теги и произведёт поиск только в тексте страницы.

Также возможен поиск в пределах всего HTML-кода (включая тэги, сценарии и так далее), если вы отметите поле Поиск внутри тэгов HTML.

Искать все слова - Offline Explorer Pro выполняет действия согласно настройкам фильтров содержания только в случае, когда все ключевые слова представлены на загруженной web-странице.

Есть две подобных группы действий - когда ключевые слова найдены или не найдены на странице.

Если слова найдены на странице

  • Сохранять эти страницы

    Когда опция отключена, Offline Explorer Pro не сохраняет загруженные страницы, которые содержат ключевые слова.

  • Загружать изображения с этих страниц

    Когда опция отключена, Offline Explorer Pro не следует по ссылкам для загрузки файлов изображений на страницах, которые содержат указанные ключевые слова.

  • Остановить загрузку если слова не найдены

    Отметьте это поле для того, чтобы остановить загрузку Проекта. Это полезно, если вы хотите остановить загрузку при появлении на страницах сообщений об ошибках. Например, после истечения срока пользовательской сессии.

Когда слова не найдены в странице

  • Сохранять эти страницы

    Когда опция отключена, Offline Explorer Pro не сохраняет загруженные страницы, которые не содержат ключевых слов. Включите опцию для сохранения также и страниц, на которых отсутствует текст, который вы ищете.

  • Загружать изображения с этих страниц

    Когда опция отключена, не следует по ссылкам для загрузки файлов изображений на страницах, которые не содержат указанные ключевые слова.

  • Остановить загрузку если слова не найдены

    Отметьте это поле для того, чтобы остановить загрузку Проекта, когда нужного текста не найдено на любой из загруженных страниц.
