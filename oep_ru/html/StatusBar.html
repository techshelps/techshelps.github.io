Строка состояния

Строка состояния используется для отображения Offline Explorer Pro информации о всех загрузках и загрузке выбранного Проекта.

Первый столбец строки состояния отображает внутреннее состояние Offline Explorer Pro:

Готово - Offline Explorer Pro готов к началу загрузки или в этот момент уже завершил загрузку.

Пауза - Offline Explorer Pro находится в режиме пауза после щелчка по кнопке Пауза на Ленте - вкладка Главная. В этом режиме Offline Explorer Pro не будет загружать никаких файлов до вторичного нажатия на кнопку пауза.

Анализ - Offline Explorer Pro выполняет анализ HTML-документа или Java-класса. Число, следующее за сообщением Анализ – это количество HTML-файлов или классов в очереди для анализа.

Следующий столбец пуст, если никакого Проекта не выделено. В противном случае, столбец содержит информацию о выделенном Проекте - статистику последней загрузки или количество загружаемых файлов.

Четвёртый столбец содержит информацию о текущей полной скорости загрузки Offline Explorer Pro. Она вычисляется как среднее значение скорости загрузки файлов за последние пять секунд.

Диаграмма изменений скорости загрузки показывает статистику за последние 60 секунд. Подведите курсор мыши на этот участок для того, чтобы увидеть больше подробностей.
