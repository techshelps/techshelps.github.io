Свойства Проекта: Типы файлов

Загрузка может быть ограничена заданными расширениями файлов. Для вашего удобства они разбиты по категориям: Текст, Изображения, Видео, Аудио, Архивы, Пользовательские и Другие. Фильтр расширений позволяет вам загружать нужные типы файлов.

Текст, Изображения, Видео, Аудио и Архивы являются предопределёнными типами файлов.

Текст - включает HTML и текстовые документы.

Изображения - разные типы графических файлов.

Видео и Аудио - соответствующие мультимедиа-форматы.

Архивы - наиболее распространённые форматы архивов для передачи сжатых данных.

Пользовательские - любой из введённых вами типов файлов для вашего удобства.

Другие - категория других типов файлов, отличающихся от прочих.

Если опция включена, то это разрешит загрузку любых типов файлов, даже если их расширения не перечислены ни в одной из категорий (предопределённых или пользовательских). Если опция не включена, то любой файл, не перечисленный в категориях, будет пропущен (не загружен).

Категории можно включить или отключить используя поля слева в дереве Категорий. Неотмеченный элемент означает – ни один из типов файлов из этой категории загружаться не будет. Отмеченный элемент разрешает загрузку файлов. Вы можете редактировать категорию, - добавляя, изменяя или удаляя расширения файлов.

Советы:

  • Чтобы задать тип файла, не имеющего никакого расширения, добавьте символ одиночной точки (.) в список.

  • Чтобы отметить все расширения сразу, нажмите на клавиатуре Ctrl-A; для отмены - Ctrl-N.

Типы файлов позволяют выполнять произвольную загрузку файлов по (1) их местонахождению, (2) размеру, и (3) типу.

Фильтр Расположение позволяет вам определить типы файлов для загрузки согласно фильтрам URL, ограничить загрузку начальным каталогом сервера или загружать файлы отовсюду. Если вы не выберете Загружать согласно фильтрам URL, то файлы из этой категории не будут проверяться на соответствие любым фильтрам URL и их спискам ключевых слов исключить/включить.

Фильтр Размеры файлов позволяет вам управлять загрузкой исходя из размеров файлов. Если отмечено Минимум, то Offline Explorer Pro не будет загружать никаких файлов, размер которых меньше указанного значения (в килобайтах). Если отмечено Максимум, то Offline Explorer Pro не будет загружать никаких файлов, размер которых больше указанного значения (в килобайтах). Загрузка с помощью Mass Downloader позволит загружать большие файлы с максимально возможной скоростью.
