Создание нового Проекта

Есть несколько способов добавить новый Проект:

  1. Мастер. Нажмите кнопку Мастер на Ленте (вкладка Главная). Далее шаг за шагом следуйте инструкциям для точной настройки вашей загрузки.

  2. Кнопка Новый проект. Кликните новый Проект на Ленте, вкладка Главная, и двойным кликом по значку Проект, или нажмите кнопку Новый Проект... в меню приложений. Введите необходимую информацию в диалоговом окне Свойства и нажмите ОК.

  3. Копировать и Вставить из буфера обмена. Вставьте URL (web-адрес) или копируйте Проект из Буфера обмена. Добавьте web-адрес выбирая Вставить из Ленты - вкладка Главная. Все свойства Проекта, кроме вставленного URL, будут установлены по умолчанию.

    Offline Explorer Pro добавит новый Проект автоматически, если в диалоге настроек отмечено "Дополнительно | Взаимодействие | Отслеживать содержимое буфера обмена".

  4. Импорт Избранного. Адрес Проекта может быть импортирован из браузеров Netscape Navigator, Netscape Communicator Bookmarks, Microsoft Internet Explorer Favorites или Opera HotLists. Кликните по кнопке Импорт на Ленте, вкладка Главная, просмотрите список через меню и выберите URL. Новый Проект будет создан с использованием заданной по умолчанию информации Свойств, кроме URL и заголовка.

  5. Drag-and-Drop (перетаскивание). Перетащите URL из Вашего web-браузера в дерево Проектов Offline Explorer и будет создан новый Проект с этим URL. Все другие Свойства будут установлены стандартными (по умолчанию).

Каждый новый Проект появляется в выбранной папке или в самой верхней стандартной папке, если никакой папки не было выбрано. (Заданная стандартной папка может быть переименована по вашему желанию). Если не определено иначе, то каждый новый Проект будет использовать шаблон по умолчанию, который может быть в любое время изменён выбором шаблона из меню кнопки приложений. К свойствам Проекта можно обращаться нажатием кнопки Свойства на Ленте, вкладка Главная или используя пункт Свойства в контекстном меню Проекта. Доступ к контекстному меню можно получить нажатием правой кнопки мыши в любом месте на вкладке Проект.
