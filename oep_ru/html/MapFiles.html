Файлы Карты

Каждый Проект имеет собственную Карту. Карта – это просто список всех загруженных для Проекта файлов. Например:

c:\download\www.server.com\articles\file001.htm

c:\download\www.server.com\default.htm

c:\download\www.server.com\images\banner.gif

c:\download\www.server.com\images\pixel.gif

Когда Offline Explorer Pro загружает Проект, каждый загруженный файл добавляется в Карту Проекта.

Offline Explorer Pro использует список файлов в Карте Проекта при выполнении некоторых операций с Проектами - экспорт, удаление, резервное копирование, поиск, печать и некоторых других.

Все файлы карт хранятся в каталоге, установленном в диалоге Настройки | раздел Размещение - Каталог Проектов, шаблонов и карт Проектов. Все они имеют расширение .map (.mpx в Offline Explorer Enterprise).

Файлы с расширением .map – это обычные текстовые файлы. Имя каждого файла размещается в отдельной строке. Имеются некоторые дополнительные параметры (тип MIME, уровень, и т.д.). Они отделяются символом tab от имени файла.

Если файл Карты Проекта по некоторым причинам утерян, то восстановить его легко. Просто выберите Проект и нажмите на клавиатуре Ctrl-F5. Offline Explorer Pro начнёт загрузку Проекта в режиме "Не загружать существующие файлы". Это добавит все существующие и недавно загруженные файлы в Карту Проекта.
