Загрузка Проекта

Загрузка Проекта означает, что Offline Explorer Pro начинает получать файлы из Интернета в соответствии с настройками Свойств Проекта. Чтобы начать загрузку Проекта, сначала выберите Проект, который Вы хотите загрузить. Затем:

  1. Нажмите кнопку Старт на Ленте - вкладка Главная.

  2. Нажмите Загрузить в главной панели инструментов.

  3. Нажмите клавишу F5 на клавиатуре.

  4. Если Проект создан с помощью адресной строки, загрузка Проекта начинается автоматически.

  5. Загрузка Проекта может быть запланирована через панель Расписание из Свойства Проекта - Дополнительно. Она определяет когда и как часто будет автоматически загружаться Проект. Смотрите Расписание для получения дополнительной информации.

  6. Параметры командной строки могут использоваться для загрузки Проектов при запуске Offline Explorer Pro.

Как только начинается загрузка, Offline Explorer Pro начинает загружать файлы, находящиеся в списке адресов (URL) Проекта. Каждая ссылка в файле проверяется на соответствие ограничениям, установленным в свойствах проекта: ограничение количества, типов файлов, размеров файлов, фильтров URL и так далее. Так как Offline Explorer загружает одновременно ограниченное количество файлов, то другие файлы помещаются в очередь для ожидания своей загрузки. В строке состояния отображается количество обрабатываемых (загружаемых) файлов и количество файлов, ожидающих свою очередь. Загрузка Проекта завершена, когда все файлы загружены и в очереди Проекта не остаётся файлов.

Во время загрузки Проекта к значку Проекта добавляется маленькая зелёная стрелка. Когда загрузка завершена, значок Проекта принимает свой обычный вид. Загруженные файлы станут доступны автономно.

Offline Explorer Pro может одновременно загружать несколько Проектов. Файлы, поставленные всеми Проектами в Очередь для загрузки, совместно используют одни и те же каналы загрузки.

У каждого Проекта есть собственная очередь загрузки, так что файлы разных Проектов будут загружаться параллельно. Версии Offline Explorer Pro и Enterprise позволяют Вам ограничить максимальное количество соединений, которые будут использоваться Проектом. Вы можете сделать это с помощью контекстного меню на закладке Очередь или добавив в поле URL следующую строку:

Channels=3

Вы можете ограничить количество одновременных соединений с каждым из серверов для предотвращения перегрузок:

ChannelsPerServer=2

Для остановки загрузки Проекта выберите Проект и нажмите кнопку Стоп панели инструментов Быстрого доступа или щёлкните Стоп на Ленте - вкладка Главная. Для остановки загрузки всех Проектов сразу щёлкните Остановить все на Ленте - вкладка Инструменты.

Загрузка может быть временно приостановлена для освобождения канала доступа в Интернет выбором Загрузка | Пауза. К значкам всех загружаемых Проектов добавляется жёлтый значок паузы (как у VCR - видеомагнитофона). Для возобновления загрузки ещё раз выберите Загрузка | Пауза. При паузе загрузки Offline Explorer Pro не будет немедленно отключать каналы соединений. Вместо этого Offline Explorer Pro будет ожидать, пока полностью не загрузится каждый из загружающихся в текущий момент файлов, так что пройдёт некоторое время, прежде чем загрузка Проекта будет приостановлена.

Загрузки в Offline Explorer Pro могут быть также задержаны и на более продолжительное время. В течение времени задержки будут доступны другие загрузки в Offline Explorer Pro, а также компьютер может быть перезапущен. Смотрите раздел Приостановить в файл для получения подробной информации.

Вы можете управлять ходом загрузки даже ещё более точно в версиях Offline Explorer Pro и Enterprise. В них имеется закладка Очередь, показывающая все URL, которые Offline Explorer Pro загружает для выбранного Проекта. Правый клик на URL в списке позволяет прервать загрузку, копировать URL или Referer, а также выполнить другие действия.

Менеджер Очереди позволяет вам также ограничивать количество соединений для выбранного Проекта и вводить в режим пауза индивидуальные Проекты.

Когда вы загружаете потоковые файлы (протоколы MMS, PNM и RTSP) вы можете сохранить частично загруженный поток. Нажмите в панели соединений на нужном файле правой кнопкой мыши и выберите Завершить в появившемся меню.
