Настройки: Инструменты

Загруженные файлы можно легко редактировать или просматривать используя обычные инструменты правки. Например, вы можете выбрать из множества графических программ программу для редактирования файлов JPEG.

Чтобы добавить инструмент, кликните по кнопке приложения и выберите Настройки из меню, перейдите в раздел Инструменты. Все инструменты в Offline Explorer Pro подразделяются на браузеры, средства просмотра и редакторы. Они будут появляться в отдельных разделах, когда Вы нажимаете правой кнопкой мыши на файле в Карте Проекта или на Проекте в дереве Проектов.

Название инструмента будет отображаться в контекстном меню. Путь должен содержать полное имя исполняемого файла соответствующего приложения. Оно должно быть включено в полный путь к файлу. Также вы можете нажать кнопку выбора файла рядом с полем ввода для облегчения размещения инструмента. Вы можете использовать параметр %windir% для определения каталога Windows в пути.

Нажмите эту кнопку для того, чтобы добавить новый инструмент в список. Поля Название и Файл будут очищены. Новый инструмент будет по умолчанию отмечен (включен) в списке Инструментов.

Нажмите кнопку Удалить для удаления выбранного инструмента из списка Инструментов. Эта кнопка также может использоваться для правки документа. Чтобы выполнить это, выберите инструмент, который нужно изменить и нажмите кнопку Удалить. Далее перейдите к полям Название или Файл для правки и сделайте необходимые изменения, а затем нажмите кнопку Добавить.

Кнопка Обнаружить поможет найти используемые в вашей системе браузеры и автоматически добавит их в список Инструменты - Браузеры.

Когда вы нажмёте правой кнопкой мыши на файле в закладке Карта, появившееся контекстное меню позволит вам выбрать инструмент. Выбранный инструмент запустит приложение и откроет выбранный файл.

Окно инструментов. Когда Offline Explorer Pro запускает любой инструмент для открытия в браузере, редактирования или просмотра загруженных файлов, то окно инструментального приложения в зависимости от этого параметра будет иметь размер минимизированный, развернутый или нормальный.
