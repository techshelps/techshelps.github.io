To quickly reassign the server/client displays/client displays on the same system

Method 1

  1. From the NVIDIA Control Panel Select at Task pane, under Workstation, click View system topology.

  2. Right click the server display icon and then from the context menu uncheck Treat as server.

  3. Right click the display that you want to use as the server and then from the context menu check Treat as Server.

  4. Right click any displays that you want to be clients, then from the context menu check Treat as Client.

Method 2

  1. From the NVIDIA Control Panel Select at Task pane, under Workstation, click Synchronize Displays.

  2. Click one of the client displays that you want to be the server display.

  3. Under Select displays to lock to the server, select the check boxes for any displays that you want to lock to the server as clients, then click Apply.
