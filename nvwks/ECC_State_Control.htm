ECC State Control

Natural environmental factors can sometimes cause a bit-error in data transmission and storage. Newer NVIDIA GPUs incorporate Error Correction Code (ECC) which checks, and in some cases corrects, these errors.

With GPUs that support ECC, you can turn ECC On or Off.
