About Maximus Technology

NVIDIA® Maximus™ technology uses two classes of NVIDIA GPUs in the system – one or more NVIDIA Quadro GPUs dedicated to Interactive Design, and one or more NVIDIA Tesla or Quadro GPUs dedicated to computational simulation or computational rendering.

This combination addresses the needs of users who want to leverage both a design and creation application together with a parallel computing application on a single workstation. Professional visualization applications that incorporate compute features also benefit from NVIDIA Maximus technology.

Requirements

  • Operating System: Windows 7 64-bit

  • Fermi-based or Kepler-based solutions

Fermi-based

One or more of the following NVIDIA Quadro products:

  • Quadro 400

  • Quadro 600

  • Quadro 2000 (D)

  • Quadro 4000

  • Quadro 5000

  • Quadro 6000

One or more of the following NVIDIA Tesla products in combination with a supported Fermi-based Quadro product:

  • Tesla C2075

Kepler-based

One or more of the following NVIDIA Quadro products:

  • Quadro 410

  • Quadro K600

  • Quadro K2000 (D)

  • Quadro K4000

  • Quadro K5000

  • Quadro K6000

One or more of the following NVIDIA Tesla products in combination with a supported Kepler-based Quadro product:

  • Tesla K20

  • NVIDIA Quadro Professional driver, Release 337 or later

 

Set Up

Using the NVIDIA Control Panel, you can configure each GPU to operate in the usage mode most suited to your particular application. See Processing Modes for detailed information.
