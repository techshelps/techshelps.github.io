To set up color space conversion

  1. From the NVIDIA Control Panel Select at Task pane, under Workstation, click Manage SDI output settings.

  2. Click the Color Settings tab.

  3. Check Override the default color space conversion.

  4. Click the Initialize the color space conversion matrix with list arrow and then click one of the pre-defined color-space standards to use as a starting point.

  5. In each color-space text box, either enter values directly or use the corresponding up and down arrows to change the values.

  6. Click OK when finished.

Note: Color space conversion control is not available with the NVIDIA Quadro FX 4000 SDI card.
