To change the usage mode of a GPU

  1. From the NVIDIA Control Panel Select a Task pane, under Workstation, click Manage GPU Utilization.

  2. Under Usage mode, select the appropriate option for each Quadro card.  

    • For each Quadro card or SLI group, you can select whether the Quadro card or SLI group handles professional graphics and compute (CUDA) needs, or professional graphics needs only.     

    • Tesla cards are automatically dedicated to compute needs, and cannot be changed.

  3. Click Apply when done.

The Usage Mode setting applies to all applications and programs, but you can set the usage mode for a specific program by clicking the Manage 3D Settings link at the bottom of the page and changing the CUDA-GPUs setting for your program.
