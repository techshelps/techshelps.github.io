Quadro Plex D2 Mosaic Connections for Passive Stereo

Passive 3D stereo uses two projectors to display overlapping images on the same screen. One projector displays the right-eye view and the other projector displays the left-eye view. The screen is then viewed with special glasses that filter each image to separate eyes, creating the perception of depth.  

This section describes the display positions and corresponding connections to use when configuring Mosaic with sync-capable GPUs for the following passive 3D stereo configurations:

  • 2-Display 4-Connections (Also available on a dual Quadro FX configuration)

  • 3-Display 6-Connections

  • 4-Display 8-Connections
