Manage EDID

The Manage EDID dialog lets you

  • View the connector information.

  • Get an EDID from a connected display and save it to disk.

  • Start or stop applying an EDID from a file to a connector.

Connector information (all tabs)

This section provides the following information:

  • Graphics card name

  • Connector type

  • Display name (if connected), and whether the display is active or inactive

  • EDID status, such as if the display or monitor EDID is used, if an EDID from a file is used, or if no display is connected  

Export tab

Export EDID opens the Save dialog that you can use to save the selected display's EDID in a file.

Unload tab

Unload EDID starts unloading or removing the EDID from the selected display connectors.

Load tab

  • EDID file: Click Browse to open the Choose File dialog where you can browse and select a file containing the EDID information that you want to apply.  A filename with any file extension is supported, as long as the contents conform to the VESA EDID data format.

  • Connector to be forced lists types of video signals. Choose the ones associated with your graphics card connector and to which your EDID will be applied.
