ToolTip

It's similar to the MessageBox management.If the optional fields are not provided, default values are used.
Some options require IE5+  (see the AutoIt Help).

Example:
#Region --- CodeWizard generated code Start ---
;ToolTip features: Text=Yes, X Coordinate=Default, Y Coordinate=Default, Title=Yes, Info
If Not IsDeclared("sToolTipAnswer") Then Dim $sToolTipAnswer
$sToolTipAnswer= ToolTip("Text",Default,Default,"Title",1,1)
#EndRegion --- CodeWizard generated code End ---
