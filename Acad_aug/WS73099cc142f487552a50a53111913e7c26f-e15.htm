You can start a drawing that uses predefined settings from a default drawing template file.

When you use a default drawing template file, new drawings automatically use the settings defined in the specified file. That template file displays each time you start a new drawing using the QNEW command.

You can specify the location of the default drawing template file using the Options dialog box.
