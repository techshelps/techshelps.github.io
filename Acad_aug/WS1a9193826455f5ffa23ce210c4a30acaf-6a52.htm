Copy, Offset, or Mirror Objects

ConceptProcedure
 
 
 

You can create duplicates of objects in your drawing that are either identical or similar to selected objects.

Topics in this section

  • Copy Objects

    You can create duplicates of objects at a specified distance and direction from the originals.

  • Create an Array of Objects

    You can create copies of objects in a rectangular or polar (circular) pattern called an array.

  • Offset an Object

    Offset an object to create a new object whose shape parallels the shape of the original object.

  • Mirror Objects

    You can flip objects about a specified axis to create a symmetrical mirror image.
