To save a block definition in the Block Editor

  1. Click Insert tab Block panel Block Editor. At the Command prompt, enter bedit.
  2. Click Block Editor tabOpen/Save panelSave Block. At the Command prompt, enter bsave
  3. To save the block definition in the drawing, click File menu  Save.To save the block definition in the drawing, at the Command prompt, enter qsave.
  4. (Optional) If you are finished using the Block Editor, click Close Block Editor.
