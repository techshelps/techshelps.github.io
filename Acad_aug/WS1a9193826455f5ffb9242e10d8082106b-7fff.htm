You can save layer settings as named layer states. You can then restore, edit, import them from other drawings and files, and export them for use in other drawings.

Topics in this section

  • Save, Restore, and Edit Layer States

    You can save the current layer settings to a layer state, make changes to the layer state, and restore them to the drawing later.

  • Import and Export Layer States

    You can import layer settings from other drawings and export layer states.
