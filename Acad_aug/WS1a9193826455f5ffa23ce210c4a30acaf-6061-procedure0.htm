To apply a named page setup to a sheet created from scratch

  1. Click View tab Palettes panel Sheet Set Manager.
  2. In the Open Sheet Set dialog box, select a sheet set. Click Open.
  3. In the Sheet Set Manager, right click in the Sheets area. Click New Sheet.
  4. In the New Sheet dialog box, enter a number and sheet title. Click OK.
  5. In the Sheet Set Manager, under Sheets, double-click the new sheet.
  6. Click Output tabPlot panelPage Setup Manager. At the command prompt, enter pagesetup.
  7. In the Page Setup Manager, under Page Setups, select a named page setup. Click Set Current.

    Named page setups are not enclosed in asterisks.

  8. Click Close.
 Toolbar: Standard 
 Command entry: SHEETSET
