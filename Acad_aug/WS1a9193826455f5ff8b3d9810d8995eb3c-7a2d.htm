You can add point lights, spotlights, and distant lights and set the location and properties of each.

Topics in this section

  • Control the Display of Lights

    The display of lights can be turned on and off in the drawing.

  • Adjust Light Placement

    After a light has been placed in a scene the position and target can be modified.

  • Control Light Properties

    Each light you add to the drawing is listed by name and type in the Lights in Model palette (LIGHTLIST). Lights in blocks and xrefs, the sun, and default lighting are not included.
