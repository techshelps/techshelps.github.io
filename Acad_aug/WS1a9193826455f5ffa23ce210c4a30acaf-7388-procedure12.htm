To display selected layers while turning off all other layers

  1. Click Home tabLayers panelIsolate. At the Command prompt, enter layiso.
  2. Select an object on the layer you want to isolate.
  3. Press ENTER.

    The selected layer is isolated.

    NoteTo restore layers to the layer state before you isolated them, use the LAYUNISO command. Any layer settings you changed are preserved.
