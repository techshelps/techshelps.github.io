You can add hyperlinks that provide jumps in your drawings to specific files or websites.

Topics in this section

  • Overview of Using Hyperlinks in a Drawing

    Hyperlinks provide a simple and powerful way to quickly associate a variety of documents (such as other drawings, bills of materials, or project schedules) with a drawing.

  • Use a Hyperlink to Access a File or a Web Page

    Hyperlinks can point to files that are stored locally, on a network drive, or on the Internet, or to named locations in drawings, such as views.

  • Use a Hyperlink to Create an Email Message

    Typically you use hyperlinks in a drawing to associate graphical objects with related documents. In some cases, however, it might be useful to insert a hyperlink that creates an email message when the hyperlink is activated.

  • Use Hyperlinks in Blocks

    Hyperlinks can be associated with blocks, including nested objects contained within blocks. If the blocks contain any relative hyperlinks, the relative hyperlinks adopt the relative base path of the current drawing when you insert them.
