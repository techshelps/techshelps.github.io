To use grips to modify a table

  1. Click a gridline to select the table.
  2. Use one of the following grips:
    • Upper-left grip. Moves the table.
    • Upper-right grip. Changes the width of the table and all columns proportionally.
    • Lower-left grip. Changes the height of the table and all rows proportionally.
    • Lower-right grip. Changes both the height and the width of the table and changes the columns and rows proportionally.
    • Column grip (at the top of the column head row). Widens or narrows adjacent columns without changing the width of the table.
    • CTRL+ a column grip. Changes the width of the column to the left of the grip and widens or narrows the table to accommodate the change.

    The minimum column width is the width of a single character. The minimum row height for an empty table is the height of a line of text plus the cell margins.

  3. Press ESC to remove selection.
