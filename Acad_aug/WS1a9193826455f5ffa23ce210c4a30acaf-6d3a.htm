A lookup action assigns custom properties and values to a dynamic block.

When you add a lookup action to a dynamic block definition and associate it with a lookup parameter, it creates a lookup table. For information about adding lookup tables to dynamic blocks, see Use Lookup Tables to Assign Data to Dynamic Blocks.
