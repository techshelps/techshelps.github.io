To specify the display of rewind thumbnails

  1. Display a wheel.
  2. Right-click on the wheel and click SteeringWheel Settings.
  3. In the SteeringWheels Settings dialog box, select one of the following:
    • Never - Only display thumbnails for view changes made with the SteeringWheels.
    • On Demand When the Bracket Is Moved Over an Empty Frame - Display thumbnails on demand for a previous view change when using the Rewind UI.
    • Automatically When a View Change Occurs - Display thumbnails for all view changes saved to the navigation history.
  4. Click OK.
