To edit a link template

  1. Click DbConnect menu Templates Edit Link Template.
  2. In the Select a Database Object dialog box, select a link template to edit. Click Continue.
  3. Select one or more key fields from the list by clicking their check boxes.
  4. Click OK to save your changes to the link template.
