To unload or reload images

  1. Click Insert tab Reference .
  2. In the External References palette, in the File References pane, right-click the image name, and then click Unload or Reload.

    The status of the selected image changes.

    All instances of the selected attached images are unloaded or reloaded.

 Toolbar: Reference 
 Command entry: EXTERNALREFERENCES
 Shortcut menu: Select an image to load or unload and right-click in the drawing area. Click Image  External References.
