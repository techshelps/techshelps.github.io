To rename a DGN mapping setup

  1. In the DGN Mapping Setups dialog box , select the mapping setup you want to rename.
    NoteThe Rename button is disabled when the Standard mapping setup is selected.
  2. Click Rename.
  3. Enter new mapping setup name.
  4. Click anywhere outside the text box.

You can also right-click the mapping setup name from the mapping setup list, click Rename and then continue with step 3.

 Command entry: DGNMAPPING
