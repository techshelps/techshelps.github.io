To select a linetype from .lin file

  1. In the Modify DGN Mapping Setup dialog box, click Linetype tab.
  2. Click the drop-down list in the cell on the right-hand column.
  3. Click Other... button.
  4. Click Load button to select a linetype from the acad.lin (acadiso.lin) file.

If several linetypes are loaded in the Load or Reload Linetypes dialog box, select a single linetype in the Select Linetype dialog box to populate the current cell. After loading several linetypes, the additional linetypes are added to the drop-down list so you can just select from the drop-down list for the other cells.
