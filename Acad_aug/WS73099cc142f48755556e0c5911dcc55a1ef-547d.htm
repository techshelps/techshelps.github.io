A dimensional constraint applied within the Block Editor using the BCPARAMETER command is called a constraint parameter.

Topics in this section

  • Overview of Constraint Parameters

    While you can use both dimensional constraints and constraint parameters in a block definition, only constraint parameters will display editable custom properties for that block reference.

  • Add a Horizontal Constraint Parameter

    A horizontal constraint parameter controls the X distance between two points on one object, or between two objects.

  • Add a Vertical Constraint Parameter

    A vertical constraint controls the Y distance between two points on one object, or between two objects.

  • Add an Aligned Constraint Parameter

    An aligned constraint controls the distance between two points on an object, one point and one object, or two line segments.

  • Add an Angular Constraint Parameter

    An angular constraint controls the angle between two line or polyline segments, or the angle of an arc.

  • Add a Radial Constraint Parameter

    A radial constraint parameter controls the radius of a circle, arc, or polyline arc segment.

  • Add a Diameter Constraint Parameter

    A diameter constraint parameter controls the radius of a circle, arc, or polyline arc segment.
