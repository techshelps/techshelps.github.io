You can create DWFx files (DWF and XPS) to publish your drawings on the web or across an intranet.

You can use the DWFx ePlot (XPS Compatible) plot configuration file to plot to a DWFx file. Each DWFx file can contain one or more drawing sheets.

DWFx files can be opened, viewed, and plotted by anyone using the following applications:

  • With Internet Explorer, you can view and print the 2D geometry of DWFx files.
  • With Autodesk Design Review, you can view the complete DWFx file; zoom and pan within the DWF file; turn layers on and off; and mark up the drawings.
