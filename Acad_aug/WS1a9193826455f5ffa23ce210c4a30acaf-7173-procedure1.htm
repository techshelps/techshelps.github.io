To correct typing errors in dynamic input tooltips

  • When a dynamic input tooltip displays the red error outline, the current entry is selected. Type over the selected text to replace it. You can also use the RIGHT ARROW, LEFT ARROW, BACKSPACE, and DELETE keys to correct your entry. After you make the correction, press TAB, comma (,), or a left angle bracket (<) to remove the red outline and complete the coordinates.
  • If you type the @ or # or * prefixes in a pointer input tooltip and then want to change it, you can just type the character you want. There is no need to backspace.
