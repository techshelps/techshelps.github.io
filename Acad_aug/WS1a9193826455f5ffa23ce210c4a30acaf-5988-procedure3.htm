To search the Data View window for a particular value

  1. Right-click a cell in the column that you want to search. Click Find.
  2. Enter a value to search for in Find What.
  3. Select a direction to search the Data View window:
    • Up. Searches the column records from the selected cell through the first record.
    • Down. Searches the column records from the selected cell through the last record.
  4. Click Find Next.
  5. Repeat step 4 to find additional occurrences of the specified value.
NoteThe Find option limits its search to records contained in the same column as the currently selected cell. There is no way to issue a global Find command that searches the entire database table for a specific value. You must select individual cells in any additional columns you want to search, and then click Find.
