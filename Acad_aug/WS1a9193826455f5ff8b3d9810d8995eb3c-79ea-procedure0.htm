To select a light using the light list

  1. Click Render tab Lights panel Lights In Model..At the Command prompt, enter lightlist
  2. In the Lights in Model window, select the light name.

    If the light has a light glyph, the light displays as selected. By double clicking the name of the light, the Lighting Properties palette is displayed.

    NoteDistant lights do not display a glyph in the drawing. Use the Lights in Model window to select a distant light.
