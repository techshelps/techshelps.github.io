Commands

DVIEW

Defines parallel projection or perspective views by using a camera and target.

PLAN

Displays the plan view of a specified user coordinate system

System Variables

VIEWDIR

Stores the viewing direction in the current viewport, expressed in UCS coordinates.

VIEWTWIST

Stores the view rotation angle for the current viewport measured relative to the WCS.
