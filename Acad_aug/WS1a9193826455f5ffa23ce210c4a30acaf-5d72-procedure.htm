To overlay an xref

  1. Click Blocks & References tabReference panelDWG. At the Command prompt, enter xattach.
  2. In the Select Reference File dialog box, select the file you want to attach and then click Open.
  3. In the External Reference dialog box, under Reference Type, select Overlay.
  4. Specify the insertion point, scale, and rotation angle. Click Specify On-Screen to use the pointing device.

    Using the Overlay option excludes any nested xrefs.

  5. Click OK.
