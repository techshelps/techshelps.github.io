To render a selection set of objects

  1. Click Render tab Render panel Advanced Render Settings.
  2. Choose a render preset to control the quality and speed of the rendered output.

    The Draft preset is very fast but its quality is low. Presentation generates a high quality image but takes substantially longer to render.

  3. Set the render procedure to Selected.
  4. Select the objects in the model that you want to render.
  5. Render the scene.
 Toolbar: Render
