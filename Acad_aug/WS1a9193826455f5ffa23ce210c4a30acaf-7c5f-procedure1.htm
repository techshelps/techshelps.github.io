To set a background for a named view

  1. At the Command prompt, enter view.
  2. In the View Manager, select an existing named view from the Model Views list.
  3. On the General panel, click the Background Override list and select Solid, Gradient, or Image.
  4. In the Background dialog box, set the colors or choose a bitmap image to use for the background. Click OK.
  5. Click OK to close the View Manager.
  6. On the ribbon, select the named view from the drop-down list on the View tab Viewports panel.
  7. Render the scene.
