To preview an image that has not been attached

  1. Click Blocks & References tabReferences panelImage. At the Command prompt, enter imageattach.
  2. In the Select Image File dialog box, select an image file to display a preview.

    If a preview of the image is not displayed, click Show Preview.

  3. Click Open.
  4. In the Image dialog box, click Details.

    The file details of the image are displayed.
