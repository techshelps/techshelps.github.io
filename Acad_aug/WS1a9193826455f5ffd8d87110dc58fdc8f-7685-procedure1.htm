To disable or remove a procedural map from a material

  1. Click Render tab 3D Palettes Materials.
  2. In the Materials window, Available Materials in Drawing panel, select a material swatch or create one by clicking the Create a New Material button.
  3. Select the Realistic material type.
  4. Select Diffuse Map and select a procedural map as the Map type (Checker, Gradient Ramp, Marble, Noise, Speckle, Tile, Waves or Wood).
  5. Select the drop down arrow in the Nested Map Navigation. The Diffuse and the procedural map name are displayed. Example: Diffuse : Checker.
  6. Clear the Diffuse Map check box.
  7. In the Nested Map Navigation panel, click the drop-down arrow.

    The Diffuse and the procedural map name are grayed out and disabled.

  8. Select the Diffuse Map check box.
  9. Select the drop down arrow in the Nested Map Navigation panel. The Diffuse and procedural map name are enabled.
  10. In the Diffuse Map channel panel, click the Delete Map Information from Material button to delete the Texture Map assigned to the material. It is an icon with the red x.
  11. Select the drop down arrow in the Nested Map Navigation panel. The procedural map name is no longer displayed in the navigation list.
