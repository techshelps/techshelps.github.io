Specify custom positions, distances, and angles for block geometry.

Topics in this section

  • Overview of Using Action Parameters
  • Add a Point Parameter

    A point parameter defines an X and Y location for the block in the drawing.

  • Add a Linear Parameter

    A linear parameter shows the distance between two anchor points.

  • Add a Polar Parameter

    A polar parameter shows the distance between two anchor points and displays an angle value.

  • Add an XY Parameter

    An XY parameter shows the X and Y distances from the base point of the parameter.

  • Add a Rotation Parameter

    A rotation parameter defines an angle.

  • Add a Flip Parameter

    A flip parameter flips objects.

  • Add an Alignment Parameter

    An alignment parameter defines an X and Y location and an angle.

  • Add a Visibility Parameter

    A visibility parameter controls the visibility of objects in the block.

  • Add a Lookup Parameter

    A lookup parameter defines a custom property that you can specify or set to evaluate a value from a list or table you define.

  • Add a Base Point Parameter

    A base point parameter defines a base point for the dynamic block reference relative to the geometry in the block.
