You can use linetypes to distinguish objects from one another visually and make your drawing easier to read.

Topics in this section

  • Overview of Linetypes
  • Load Linetypes

    At the start of a project, you load the linetypes that are required for the project so that they are available when you need them.

  • Set the Current Linetype

    All objects are created using the current linetype, which is displayed in the Linetype control on the Properties toolbar.

  • Change the Linetype of an Object

    You can change the linetype of an object by reassigning it to another layer, by changing the linetype of the layer the object is on, or by specifying a linetype for the object explicitly.

  • Control Linetype Scale

    You can use the same linetype at different scales by changing the linetype scale factor either globally or individually for each object.

  • Display Linetypes on Short Segments and Polylines

    You can center the pattern of a linetype on each segment of a polyline, and you can control how the linetype is displayed on short segments.
