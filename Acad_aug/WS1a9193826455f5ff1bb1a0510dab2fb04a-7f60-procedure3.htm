To create annotative multiline text

  1. Click Annotate tabText panelMultiline Text. At the command prompt, enter mtext.
  2. Specify opposite corners of a bounding box to define the width of the multiline text object. The In-Place Text Editor is displayed.
  3. Do one of the following:
    • On the Text Formatting toolbar, in the Text Style control, click the arrow and select an existing annotative text style from the list.
    • Click the Annotative button on the toolbar to create annotative multiline text.
  4. Enter the text.
  5. On the Text Formatting toolbar, click OK.
