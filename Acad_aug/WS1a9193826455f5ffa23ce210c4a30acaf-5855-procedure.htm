To obtain a root certificate

  1. To request the root certificate, contact the organization or individual who attached a digital signature to the current file.
  2. When you receive the root certificate, click Start menu (Windows) Settings Control Panel Internet Options.
  3. In the Internet Properties dialog box, Content tab, click Certificates.
  4. In the Certificate Manager, click Import to run the Certificate Manager Import Wizard.
  5. Follow the on-screen instructions to obtain a root certificate for the current file.
