Texture maps are useful for creating many kinds of materials, such as wood grains and wall surfaces.

You can used the following file types to create texture maps:

  • BMP, RLE, or DIB
  • GIF
  • JFIF, JPG, or JPEG
  • PCX
  • PNG
  • TGA
  • TIFF

. To display he Texture Map Property Settings specific to Texture map, click the Click for Texture Map Settings button.

To return to the top level map after making changes to the property settings, at the top of the properties settings window, next to the Nested Map Navigation name display, click the Up One Level to Parent Map button
