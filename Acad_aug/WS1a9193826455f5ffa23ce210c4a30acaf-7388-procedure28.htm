To lock or unlock a layer by using the Layers toolbar

  1. Click Home tab Layers panel Layers Properties
  2. In the Layer Properties Manager, click the padlock icon for the layer name that you want to lock or unlock.

    If the padlock is open, the layer is unlocked.

  3. (Optional) Specify the degree of fading for all locked layers with the LAYLOCKFADECTL system variable.
 Toolbar: Layers 
 Command entry: LAYER, LAYLOCKFADECTL
