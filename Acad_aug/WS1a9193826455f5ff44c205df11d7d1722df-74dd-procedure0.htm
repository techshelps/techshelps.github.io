To display a list of the underlays attached to a drawing

  1. Click Insert tab Reference .
  2. In the External References palette, click the List View button.
 Toolbar: Reference 
 Command entry: EXTERNALREFERENCES
 Shortcut menu: Select an underlay and right-click in the drawing area. Click External References.
