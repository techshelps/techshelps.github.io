To specify the Distance Type property for a stretch action associated with an XY parameter

  1. Click Insert tab Block panel Block Editor. At the Command prompt, enter bedit.
  2. In the Block Editor, select a stretch action associated with an XY parameter.
  3. On the Properties palette, Overrides area, Distance Type, select an option from the drop-down list.

Depending on the override you apply, the stretch action for the block is limited to the X axis, the Y axis, or both.
