To add an XY parameter to a dynamic block definition

  1. Click Insert tab Block panel Block Editor. At the Command prompt, enter bedit.
  2. Click Block Editor tab Action Parameters panel XY.In the Block Editor, in the Block Authoring Palettes window, Parameters tab, click the XY Parameter tool.
  3. Follow the Command prompts to specify the parameter type.
  4. Specify a base point for the parameter.
  5. Specify an endpoint for the parameter.
  6. To add an action now, double-click the alert icon. Follow the prompts to associate an action with the parameter and a selection set of geometry.
  7. Click Block Editor tab Open/Save panel Save Block. At the Command prompt, enter bsave.
  8. If you are finished using the Block Editor, click Close Block Editor.
