Creates ramps using colors, maps, and blends.

You can make highly customized gradients with the Gradient Ramp procedural map. Gradient Ramp uses several colors, sub-procedural maps or a combination of both to create shades or ramps from one to another.

An example of the Gradient Ramp map

To display the map property settings specific to Gradient Ramp, click the Click for Gradient Ramp Settings button. For more information, see Gradient Ramp Property Settings.
