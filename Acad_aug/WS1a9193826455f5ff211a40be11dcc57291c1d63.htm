A linear parameter shows the distance between two anchor points.

When you insert a linear parameter, grip movement is constrained along a preset angle. In the Block Editor, a linear parameter looks similar to an aligned dimension.

You can associate the following actions with a linear parameter:

  • Move
  • Scale
  • Stretch
  • Array

The linear parameter has a property called Chain Actions. For more information about allowing chained actions for a parameter, see Allow Chained Actions for Dynamic Blocks.
