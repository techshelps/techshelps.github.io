Commands

MODEL

Switches from a layout tab to the Model tab.

MSPACE

Switches from paper space to a model space viewport.

PSPACE

Switches from a model space viewport to paper space.

System Variables

LAYOUTREGENCTL

Specifies how the display list is updated in the Model tab and layout tabs.

MAXACTVP

Sets the maximum number of viewports that can be active at one time in a layout.

TILEMODE

Makes the Model tab or the last layout tab current.
