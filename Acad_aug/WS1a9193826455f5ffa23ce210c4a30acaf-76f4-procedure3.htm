To control display of silhouette edges

  1. Click View tab 3D Palettes panel Visual Styles Manager.
  2. With the current visual style selected in the sample images, in Fast Silhouette Edges, Visible, select Yes or No.
  3. If Yes is selected, specify a width for silhouette edges.
 Toolbar: Visual Styles 
 Command entry: VISUALSTYLES
