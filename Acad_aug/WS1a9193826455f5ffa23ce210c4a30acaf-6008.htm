There are several ways to position a drawing on the paper. You can specify the printable area, set the position of the plot, and set the drawing orientation.

Topics in this section

  • Specify the Printable Area

    The printable area is displayed by a dashed border in a layout. The plotter and paper size you select determine the printable area.

  • Set the Position of the Plot

    The printable area of a drawing sheet is defined by the selected output device and is represented by the dashed line in a layout. You can change the position of plot relative to the printable area or to the edge of the paper.
