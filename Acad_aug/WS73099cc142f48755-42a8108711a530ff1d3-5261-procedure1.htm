To switch to a different UCS

  1. Click View tab Views panel ViewCube. At the Command prompt, enter navvcube.
  2. On the UCS menu located below the ViewCube tool, click the down arrow next to the current UCS name.
  3. Select a named UCS from the list.
 Command entry: NAVVCUBE
