To add a procedural map to a material

  1. Click Render tab 3D Palettes Materials.
  2. In the Materials window, Available Materials in Drawing panel, click the Create a New Material button.
  3. Select the Realistic material type.
  4. Select Diffuse map, and select a procedural map as the map type (Checker, Gradient Ramp, Marble, Noise, Speckle, Tile, Waves or Wood).
  5. For additional map property settings, click the Click for <map type> Settings button.
  6. Make changes to theTexture Map Property Settingsmap property settings.
  7. To return to the top-level map, at the top of the properties settings window, click Up One Level to Parent Map.
  8. To display a larger view of the material swatch, click Preview Map Channel Procedural Results.
  9. In the Nested Map Navigation panel, click the drop-down arrow.

    The procedural map is displayed. Example: Diffuse : Checker

NoteEach procedural map type has different property settings available. For more information, see Map Property Settings in the Command Reference.

After making changes to the material you can apply materials to objects and faces.
