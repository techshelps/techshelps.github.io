To specify that DWF, DWFx, or PDF files are published with password security

  1. Click Output tabPublish panelPublish. At the command prompt, enter publish.

    The Publish dialog box appears.

  2. Click Publish Options.

    The Publish Options dialog box appears.

  3. In the General DWF Options pane, under Password Protection, select one of the following from the drop-down list:
    • Select Specify Password, and under Password, enter a password to use for the DWF or DWFx file.
    • Select Prompt for Password. You will be prompted for a password or phrase to open the published DWF or DWFx file.

    DWF or DWFx passwords are case-sensitive. The password or phrase can consist of letters, numbers, punctuation marks, or non-ASCII characters.

    ImportantKeep a list of passwords and their corresponding DWF or DWFx file names in a safe place. If you lose or forget the password, it cannot be recovered.
  4. Click OK.
  5. In the Publish dialog box, contine with publishing tasks, and then close the dialog box.
