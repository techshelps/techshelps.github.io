To specify the display size of grips in the Block Editor

  1. Click Insert tab Block panel Block Editor. At the Command prompt, enter bedit.
  2. Click Block Editor tab Manage panel Dialog box launcher.
  3. In the Block Editor Settings dialog box, under Parameter and Grip Size, specify the grip size.
  4. Enter an integer from 1 to 255 (pixels). Click OK.
 Command entry: BESETTINGS
