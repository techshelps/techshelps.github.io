To switch dictionaries while checking spelling

  1. Click Annotate tabText panelCheck Spelling. At the Command prompt, enter Spell
  2. In the Check Spelling dialog box, click Dictionaries.
  3. Do one of the following:
    • To change the main dictionary, select a dictionary from the Current Main Dictionary list.
    • To change the custom dictionary, select a dictionary under Current Custom Dictionary.
  4. Click Close.
 Command entry: SPELL
