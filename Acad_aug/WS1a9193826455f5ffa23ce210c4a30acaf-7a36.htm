You can adjust the application interface and drawing area to match the way you work.

Topics in this section

  • Set Up the Drawing Area

    You can adjust the color and display schemes used in the application and drawing windows, and control the behavior of general features such as zoom transitions.

  • Set Options for 3D Modeling with Perspective Projection

    You can set specific display options when working with 3D models using perspective projection.

  • Switch Between Model Space and Layouts

    You can control how you change between model space and one or more layouts.

  • Specify Application Fonts

    Change the fonts used in the application window and in the text window.

  • Specify the Behavior of Dockable Windows

    Windows such as the Properties palette, tool palettes, and DesignCenter can be docked, anchored, or floated.

  • Control the Display of Toolbars

    Similar to dockable windows, you can control the behavior of toolbars.
