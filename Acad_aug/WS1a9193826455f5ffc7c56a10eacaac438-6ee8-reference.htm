ConceptQuick Reference
 
 
 

Commands

CONVERTOLDMATERIALS

Converts materials created in previous drawing file formats to the current format.

MATERIALATTACH

Associates materials with layers.

MATERIALMAP

Displays a material mapping gizmo to adjust the mapping on a face or an object.

MATERIALS

Shows or hides the Materials window.

MATERIALSCLOSE

Closes the Materials window.

System Variables

MATSTATE

Indicates whether the Materials window is open or closed.

3DCONVERSIONMODE

Used to convert material and light definitions to the current product release.
