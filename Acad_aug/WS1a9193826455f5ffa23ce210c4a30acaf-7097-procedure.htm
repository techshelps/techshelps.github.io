To offset a point from a temporary reference point

  1. At a prompt for a point, enter from. Alternately, press SHIFT and right-click to display the object snap menu, and then choose From.
  2. If you want to offset from a location on an existing object, specify an object snap method. Then select the object snap point.
  3. Enter a relative coordinate (i.e. @1,1).
