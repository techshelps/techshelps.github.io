To specify that dynamic block reference can be exploded

  1. Use one of the following procedures to open a block definition in the Block Editor:
    • To open an existing block definition in the Block Editor
    • To open a drawing file saved as a block (not dynamic) in the Block Editor
    • To open a drawing file saved as a dynamic block in the Block Editor
  2. Click Insert tab Block panel Block Editor. At the Command prompt, enter bedit.
  3. In the Block Editor, make sure nothing is selected.
  4. In the Properties palette, under Block, click Allow Exploding.
  5. On the drop-down list, select Yes or No.
  6. Click Block Editor tab Open/Save panel Save Block. At the Command prompt, enter bsave.
  7. (Optional) If you are finished using the Block Editor, click Close Block Editor.
