To see several views at the same time, you can split the drawing area of the Model tab into separate viewing areas called model space viewports. You can save arrangements of model space viewports for reuse at any time.

Topics in this section

  • Set Model Space Viewports

    On the Model tab, you can split the drawing area into one or more adjacent rectangular views known as model space viewports.

  • Select and Use the Current Viewport

    When you use multiple viewports, one of them is the current viewport, which accepts cursor input and view commands.

  • Save and Restore Model Tab Viewport Arrangements

    Arrangements of model viewports can be saved and restored by name.
