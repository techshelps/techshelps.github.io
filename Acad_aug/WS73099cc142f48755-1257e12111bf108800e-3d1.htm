The application and drawing status bars provide useful information and buttons for turning drawing tools on and off.

Topics in this section

  • Application Status Bar

    The application status bar displays the coordinate values of your cursor, drawing tools, navigation tools, and tools for Quick View and annotation scaling.

  • Drawing Status Bar

    The drawing status bar displays several tools for scaling annotations.
