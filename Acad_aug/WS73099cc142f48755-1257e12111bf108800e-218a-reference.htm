Commands

MESH

Creates a 3D mesh primitive object such as a box, cone, cylinder, pyramid, sphere, wedge, or torus.

MESHPRIMITIVEOPTIONS

Displays the Mesh Primitive Options dialog box, which sets the tessellation defaults for primitive mesh objects.

System Variables

DIVMESHCYLAXIS

Sets the number of subdivisions around the perimeter of the mesh cylinder base.

DIVMESHCYLBASE

Sets the number of radial subdivisions from the center of the mesh cylinder base to its perimeter.

DIVMESHCYLHEIGHT

Sets the number of subdivisions between the base and the top of the mesh cylinder.

DRAGVS

Sets the visual style that is displayed while creating 3D solid and mesh primitives and extruded solids, surfaces, and meshes.
