To import a layout from a template in Quick View

  1. On the status bar, click .

    A row of Quick View images of model space and layouts in a drawing is displayed at the bottom of the application.

  2. Right-click a Quick View image. Click From Template.
  3. In the Select File dialog box, select a DWT or DWG file to import a layout from. Click Open. In the Insert Layout(s) dialog box, select a layout to import.
  4. Enter a new name for the layout. Click OK.
 Command entry: QVLAYOUT
