To redefine a stretch or polar stretch action's stretch frame set in a dynamic block definition

  1. Click Insert tab Block panel Block Editor. At the Command prompt, enter bedit.
  2. In the Block Editor, double-click a stretch or polar stretch action.
  3. Follow the Command prompts to define a new stretch frame and add objects to or remove objects from the action's selection set.
  4. Click Block Editor tab Open/Save panel Save Block. At the Command prompt, enter bsave.
  5. (Optional) If you are finished using the Block Editor, click Close Block Editor.
 Command entry: BACTIONSET
