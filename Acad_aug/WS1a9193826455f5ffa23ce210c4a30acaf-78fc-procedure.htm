To begin a new drawing using a wizard

  1. If necessary, set the STARTUP system variable to 1 and the FILEDIA system variable to 1.
  2. Click File menuNew.At the Command prompt, enter new.
  3. In the Create New Drawing dialog box, click Use a Wizard.
  4. Click Quick Setup or Advanced Setup.
  5. Complete the wizard pages using the Next and Back buttons to move forward and backward.
  6. On the last page, click Finish.
