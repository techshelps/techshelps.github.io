ConceptProcedureQuick Reference
 
 
 

You can choose a different drawing template file each time you create a new drawing with the Select Template dialog box.

If you require different settings each time you create a new drawing, such as an imperial template file and a metric template file, you can use the Select Template dialog box to create a new drawing.

NoteAt the bottom-right corner of the Select Template dialog box, there is an Open button with an arrow button next to it. If you click the arrow button, you can choose between two internal default drawing templates, metric or imperial.
