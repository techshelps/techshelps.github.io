To add a flip parameter to a dynamic block definition

  1. Click Insert tab Block panel Block Editor. At the Command prompt, enter bedit.
  2. Click Block Editor tab Action Parameter panel Flip.In the Block Editor, in the Block Authoring Palettes window, Parameters tab, click the Flip Parameter tool.
  3. (Optional) Follow the Command prompts to specify the following for the parameter:
    • Type
    • Label
    • Description
    • Properties displayed for block reference
    NoteYou can also specify and edit these properties in the Properties palette at a later time, after you've added the parameter to the block definition.
  4. Specify the first point of the reflection line for the flip parameter.

    This first point of the reflection line is the default location for the flip parameter grip.

  5. Specify the second point of the reflection line for the flip parameter.
  6. Specify a location for the parameter label.
  7. To add an action now, double-click the alert icon. Follow the prompts to associate an action with the parameter and a selection set of geometry.
  8. Click Block Editor tab Open/Save panel Save Block. At the Command prompt, enter bsave.
  9. If you are finished using the Block Editor, click Close Block Editor.
