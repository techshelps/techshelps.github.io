You can control many aspects of how an object is plotted by using plot styles.

Topics in this section

  • Overview of Plot Styles

    A plot style controls an object's plotted properties.

  • Choose a Type of Plot Style Table

    A plot style table is a collection of plot styles assigned to a layout or the Model tab. There are two types of plot style tables: color-dependent plot style tables and named plot style tables.

  • Switch the Type of Plot Style Table

    You can change which type of plot style table, color-dependent or named, is used in a drawing.

  • Assign Plot Style Tables to Layouts
  • Manage Plot Style Tables

    You can use the Plot Style Manager to add, delete, rename, copy, and edit plot style tables.
