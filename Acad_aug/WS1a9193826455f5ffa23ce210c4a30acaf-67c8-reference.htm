Commands

UCS

Manages user coordinate systems.

XEDGES

Creates wireframe geometry from the edges of a 3D solid, surface, mesh, region, or subobject.

System Variables

ELEVATION

Stores the current elevation of new objects relative to the current UCS.

FACETRES

Adjusts the smoothness of shaded and rendered objects and objects with hidden lines removed.

ISOLINES

Specifies the number of contour lines per surface on objects.
