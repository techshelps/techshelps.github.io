You can create text using several methods, depending on your needs.

Topics in this section

  • Overview of Creating Text

    The text you add to your drawings conveys a variety of information. It may be a complex specification, title block information, a label, or even part of the drawing.

  • Create Single-Line Text

    You can use single-line text to create one or more lines of text, where each text line is an independent object that you can relocate, reformat, or otherwise modify.

  • Create Multiline Text

    A multiline text (mtext) object includes one or more paragraphs of text that can be manipulated as a single object.

  • Create and Edit Columns in Multiline Text

    You can create and edit multiple columns using the In-Place Text Editor column options and column grips.

  • Import Text from External Files

    You can insert TXT or RTF text files into your drawing by either importing the text or dragging a file icon from Windows Explorer.
