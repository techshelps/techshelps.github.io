To create a ruled mesh

  1. Click Home tabModeling panelRuled Surface. At the Command prompt, enter rulesurf.
  2. Select an object to act as the first defining curve.
  3. Select a second object as the second defining curve.

    Mesh segments are drawn between the defining curves. The number of segments equals the value set for SURFTAB1.

  4. Erase the original curves if necessary.
 Command entry: RULESURF
