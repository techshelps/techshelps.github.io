You can access a calculator function as you work with the program. You can use either the QuickCalc calculator interface or the command prompt calculator.

Topics in this section

  • Use the QuickCalc Calculator

    With the QuickCalc calculator, an interface that looks and functions like a hand-held calculator, you can perform mathematical, scientific, and geometric calculations, convert units of measurement, manipulate the properties of objects, and evaluate expressions.

  • Use the Command Prompt Calculator

    By entering an expression in the command prompt calculator, you can quickly solve a mathematical problem or locate points in your drawing.
