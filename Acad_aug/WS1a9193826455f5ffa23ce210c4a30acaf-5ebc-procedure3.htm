To turn background publishing on or off using the options dialog box

  1. Click Tools menu  Options.At the command prompt, enter options.
  2. In the Options dialog box, Plot and Publish tab, under Background Processing Options, select or clear the Enable Background Plot When Publishing option.
  3. Click OK.
 Command entry: OPTIONS
