ConceptProcedure
 
 
 

Actions define how the geometry of a dynamic block reference will move or change when the custom properties of the block reference are manipulated in a drawing.

Topics in this section

  • Overview of Using Actions
  • Add a Move Action

    A move action causes objects to move a specified distance and angle.

  • Add a Scale Action

    A scale action scales a block’s selection set.

  • Add a Stretch Action

    A stretch action causes objects to move and stretch a specified distance in a specified location.

  • Add a Polar Stretch Action

    A polar stretch action rotates, moves, and stretches objects a specified angle and distance.

  • Add a Rotate Action

    A rotate action causes its associated objects to rotate.

  • Add a Flip Action

    A flip action allows you to flip a dynamic block reference about a specified axis called a reflection line.

  • Add an Array Action

    An array action causes its associated objects to copy and array in a rectangular pattern.

  • Add a Lookup Action

    A lookup action assigns custom properties and values to a dynamic block.
