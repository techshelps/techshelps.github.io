To add an angular constraint to an arc block definition

  1. Click Insert tab Block panel Block Editor. At the Command prompt, enter bedit.
  2. Click Block Editor tabDimensional panelAngular. At the Command prompt, enter bcparameter
  3. Specify an arc.
  4. Specify the dimension line location.
  5. Follow the Command prompts to specify the following for the constraint:
    • Value or name and value
    • Number of grips
  6. Press Enter.
 Command entry: BCPARAMETER
