Use a Hyperlink to Start a New Drawing
Expand All
Expand all collapsed sections.
Collapse All
Collapse all expanded sections.
ConceptProcedureQuick Reference
 
 
 

When you activate a hyperlink that points to a drawing template (DWT) file, a new drawing file is created based on the template rather than opening the actual template file.

This provides a convenient mechanism for creating a set of standardized drawing templates that you can share with others. Because the hyperlinked template files open a new drawing, there is no risk of others accidentally overwriting the original template.
