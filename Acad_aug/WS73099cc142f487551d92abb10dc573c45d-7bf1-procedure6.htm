To align and space leaders

  1. Click Annotate tabMultileaders panelAlign. Enter mleaderalign at the Command prompt.
  2. Select the multileaders to be aligned. Press ENTER.
  3. Specify a starting point in the drawing to begin the alignment. The point you select is the position of the landing head.
  4. If you want to change the spacing of the multileader objects, enter s and specify one of the following spacing methods:
    • Distribute. Spaces content evenly between two selected points.
    • Use Current. Uses the current spacing between multileaders.
    • Make Parallel. Places content so that each of the last line segments in the selected multileaders are parallel.
  5. In the drawing click a point to end the alignment.
 Toolbar: Multileader 
 Command entry: MLEADERALIGN
