To change the current view to the XY plane

  1. Click View menu 3D Views Plan View.
  2. Select one of the following options:
    • Current (for the current UCS)
    • World (for the WCS)
    • Named (for a saved UCS)
    NotePLAN changes the viewing direction and turns off perspective and clipping; it does not change the current UCS. Any coordinates entered or displayed subsequent to the PLAN command remain relative to the current UCS.
 Command entry: PLAN
