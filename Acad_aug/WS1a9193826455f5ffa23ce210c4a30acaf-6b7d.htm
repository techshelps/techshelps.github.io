At the Select Objects prompt, you can select one or more objects individually.

Use the Pickbox Cursor

When the square pickbox cursor is in position to select an object, the object is highlighted. Click to select the object.

You can control the size of the pickbox in the Options dialog box, Selection tab.

Select Objects Close Together

It is difficult to select objects that are close together or lie directly on top of one another. The example shows two lines and a circle that lie within the pickbox.

If selection preview is turned on, you can cycle through the objects by rolling over the object on top to highlight it, and pressing and holding Shift and then pressing Spacebar continuously. When the required object is highlighted, left-click to select it.

If selection preview is turned off, hold down Shift + Spacebar and click to cycle through these objects, one after the other, until the one you want is selected. Press Esc to turn off cycling.

Remove Selection from Objects

Remove objects from the current selection set by holding down Shift and selecting them again.
