To display the object snap menu

  1. Enter any command that prompts you to specify a point. For example, enter line.
  2. At the From Point prompt, hold down SHIFT  and right-click.

    The object snap menu is displayed, and you can click an object snap option.

 Shortcut menu: On the status bar, right-click the object snap button . Click Settings.
