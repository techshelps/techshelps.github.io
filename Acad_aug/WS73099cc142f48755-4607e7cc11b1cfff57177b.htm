The 2D Navigation wheel is for basic navigation of 2D views.

With this wheel you can access basic 2D navigation tools; it is particularly useful when you do not have a pointing device with a scroll wheel. The wheel includes the Pan and Zoom tools.

The 2D Navigation wheel wedges have the following options:

  • Pan. Repositions the current view by panning.
  • Zoom. Adjusts the magnification of the current view.
  • Rewind. Restores the most recent view orientation. You can move backward or forward by clicking and dragging left or right.
