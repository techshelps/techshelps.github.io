Use Hyperlinks in Fields

The Hyperlink field assigns a hyperlink to any piece of text.

The hyperlink works the same way as a hyperlink attached to an object. When the cursor pauses over the text, a hyperlink cursor is displayed, along with a tooltip that describes the hyperlink. Hold down the CTRL key and click to follow the link.

NoteThe Hyperlink field uses an absolute path to a file; the HYPERLINK command can create a hyperlink with a relative path.

The fields that point to sheet and view titles and numbers can be assigned a hyperlink when they are created. When these items are changed or moved in the Sheet Set Manager, the hyperlinks associated with them still jump to the correct location.

Hyperlinks in fields are converted from links to DWG files into links to DWF files when published to a multi-sheet DWF file format.
