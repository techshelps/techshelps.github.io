To validate a 3D solid object

  1. Click Home tabSolid Editing panelSolid Editing flyoutCheck.At the Command prompt, enter solidedit. Then enter b (Body) and c (Check).
  2. Select the 3D solid object.
  3. Press Enter to complete the command.

    If the object is a valid 3D object, a message is displayed at the Command prompt. If it is not valid, you continue to be prompted to select a 3D solid.
