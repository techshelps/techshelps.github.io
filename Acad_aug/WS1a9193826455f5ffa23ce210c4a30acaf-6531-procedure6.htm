To convert the lists in a multiline text object to plain text

  1. If the ribbon is active, double-click a multiline text object to open the MTEXT ribbon contextual tab. If the ribbon is not active, the In-Place Text Editor is displayed.
  2. Right-click in the editor. Click Bullets and Lists. Remove the check mark next to Allow Bullets and Lists.

    The plain text lists retain their bullets, numbers, or letters. If you add an item to the list, the number or letter sequence does not change.
