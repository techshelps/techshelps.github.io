ConceptProcedureQuick Reference
 
 
 

You can use a page setup to specify the settings for your plot job. When you select a page setup in the Plot dialog box, the settings from the page setup are added to the Plot dialog box. You can choose to plot with those settings, or change any of the settings individually and then plot.

Any settings you specify in the Plot dialog box can be saved as a new named page setup by clicking the Add button in the Page Setup area.

Any settings specified in the Plot dialog box, whether you've applied a page setup from the Page Setup list, or changed the settings individually, can be saved to the layout for use the next time you plot.
