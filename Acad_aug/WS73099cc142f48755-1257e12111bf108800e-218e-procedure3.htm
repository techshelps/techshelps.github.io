To create a mesh cone with the height and orientation specified by the axis endpoint

  1. Click Mesh Modeling tabPrimitives panelMesh Cone. At the Command prompt, enter mesh. Then enter c (Cone).
  2. Specify the center point of the base.
  3. Specify the radius or diameter of the base.
  4. At the Command prompt, enter a (Axis endpoint). Specify the endpoint and rotation of the cone.

    This endpoint can be located anywhere in 3D space.

 Toolbar: Smooth Mesh Primitives
 Command entry: MESH
