You can change text content, formatting, and properties such as scale and justification.

Topics in this section

  • Overview of Changing Text

    Text, whether created with TEXT, MTEXT, or MLEADER can be modified like any other object.

  • Change Single-Line Text

    You can change the contents, formatting and properties of single-line text.

  • Change Multiline Text

    You can change the location and content of multiline text objects with the Properties palette, the In-Place Text Editor, and grips.

  • Find and Replace Text

    You can easily find and replace text with the FIND command

  • Change Text Scale and Justification

    Several commands are available for changing the scale of one or more text and attribute objects, or their insertion points, simultaneously without changing the location of the objects.
