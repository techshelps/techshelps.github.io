Examine Individual Objects with ViewCube

ConceptProcedureQuick Reference
 
 
 

You can examine individual objects of a model using the ViewCube tool.

With the ViewCube tool, you can define the center of a view based on one or more selected objects. After you select an object or objects and you use the ViewCube tool is used to change the orientation of the model, the model rotates around the center of the view. The center of the view is calculated by the extents of the selected objects.
