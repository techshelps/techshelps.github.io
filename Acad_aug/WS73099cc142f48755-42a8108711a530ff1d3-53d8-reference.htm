Commands

NAVVCUBE

Controls the visibility and display properties of the ViewCube tool.

OPTIONS

Customizes the program settings.

System Variables

NAVVCUBEDISPLAY

Controls the display of the ViewCube tool in the current viewport when the 3D graphics system is active.

NAVVCUBELOCATION

Identifies the corner in a viewport where the ViewCube tool is displayed.

NAVVCUBEOPACITY

Controls the opacity of the ViewCube tool when inactive.

NAVVCUBESIZE

Specifies the size of the ViewCube tool.
