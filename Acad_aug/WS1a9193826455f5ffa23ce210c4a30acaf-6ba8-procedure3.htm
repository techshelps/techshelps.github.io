To change the linetype assigned to a layer

  1. On the Layers toolbar, click the Layer Properties Manager button.
  2. In the Layer Properties Manager, click Load and click one or more linetypes to load, and then click OK.

    You can hold down Ctrl to select several linetypes or Shift to select a range of linetypes.

  3. Select a layer from the list and then click Details to expand the dialog box.
  4. Select a linetype from the Linetype list.
  5. Click OK to close the dialog box.

    To open the Layer Properties Manager, click Format menu  Layer.To open the Layer Properties Manager, at the Command prompt, enter layer.
