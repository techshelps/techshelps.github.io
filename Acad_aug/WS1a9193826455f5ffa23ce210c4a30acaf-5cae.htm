Topics in this section

  • Link OLE Objects in Drawings

    When you link information from another document in a drawing, the information can be updated when it changes in the source document.

  • Embed OLE Objects in Drawings

    When you embed information from another document in a drawing, the information is not updated when it changes in the source document.
