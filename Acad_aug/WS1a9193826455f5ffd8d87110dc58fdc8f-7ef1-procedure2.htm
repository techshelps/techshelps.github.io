To create a new material using the Advanced material type

  1. Click Render tab Materials panel Materials.

    The Materials window is displayed.

  2. In the Materials window, below the swatches, click Create a New Material.
  3. In the Create New Material dialog box, enter a name and an optional description. Click OK.
  4. In the Materials Editor section, Type drop-down list, select Advanced.
  5. Specify the material color options. You can select one or more colors from the following:
    • Click Ambient to specify the color that appears on those faces lighted by ambient light alone, or select By Object to use the color of the object to which the material is attached.
    • Click Diffuse to assign a color to the material, or select By Object to use the color of the object to which the material is attached.
    • Click Specular to specify the color of a highlight on a shiny material, or select By Object to use the color of the object to which the material is attached.
  6. (Optional) Click the first lock (between Ambient and Diffuse) to set the ambient color to the diffuse color. Click the second lock (between Diffuse and Specular) to set the specular color to the diffuse color.

    When a color is locked to another color, changing one changes the other.

  7. Use the sliders to set the properties for the shininess, opacity, refraction, translucency, and so on.
  8. (Optional) Add a map style and a procedural map type. (See Use Map Channels for Added Realism and Understand Map Types.)
