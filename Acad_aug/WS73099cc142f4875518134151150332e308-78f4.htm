Get Information about Using Autodesk Impression

You can find information in a variety of locations:

  • Jump Start Tour. Available from the Autodesk Impression Help menu, this quick tutorial is displayed across the bottom of the Autodesk Impression canvas, or drawing area. It includes sample illustrations that give you hands-on guidance through some key activities in Autodesk Impression. Short animations demonstrate the steps.
  • Quick Start Guide. Available from the Autodesk Impression Help menu, the illustrations and animations in the Quick Start Guide give you an overview of what you can do with Autodesk Impression.
  • Autodesk Impression Help System. Once you are familiar with Autodesk Impression, the Autodesk Impression Help system is useful for obtaining more detailed information. Become familiar with the Help organization, so you can quickly find general descriptions, procedures, details about dialog boxes and palettes, and definitions.
  • Autodesk Impression Community Site. For the latest news and contact with other Autodesk Impression customers, go to http://impression.autodesk.com/impression-community.

How to Obtain Autodesk Impression

Autodesk Impression is only available through subscription to specific Autodesk products. For more informaiton, see the Autodesk Impression website.

If you are a subscription customer and want to download Autodesk Impression, go to the Autodesk Subscription website.

If you are not a subscription customer, but want to learn more about subscription, go to the Services & Support > Subscription website

Accessing Autodesk Impression Versions

When multiple versions of Autodesk Impression are installed on your system, the Export to Impression feature automatically exports your drawing to the most recent version.

If you have not installed the most recent version of Autodesk Impression and are a subscription customer, you can upgrade.

If your trial version of Autodesk Impression expires and you wish to obtain Autodesk Impression and continue using the Export to Impression feature, become a subscription customer.

If you have an old version of Autodesk Impression installed on your system, if desired, you can set it as the default and continue using the Export to Impression feature.

When you export to an older version of Autodesk Impression, some features do not function as they do in the current version:

  • You cannot export a Window or Display View to Autodesk Impression Release 1. Only the extents of the drawing are exported.
  • Plot style information is lost. The rendered drawing uses only the stroke type.
