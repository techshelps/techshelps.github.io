To import an Autodesk 3ds MAX file

  1. Click Blocks & References tabImport panel3DS File. At the Command prompt, enter 3dsin.
  2. In the 3D Studio File Import dialog box, select the file that you want to import.
  3. Click Open.
 Command entry: 3DSIN
