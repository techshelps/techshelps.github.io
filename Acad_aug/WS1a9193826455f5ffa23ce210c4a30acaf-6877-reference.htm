Commands

EXTRUDE

Extends the dimensions of a 2D object or 3D face into 3D space.

PEDIT

Edits polylines and 3D polygon meshes.

REGION

Converts an object that encloses an area into a region object.

SWEEP

Creates a 3D solid or surface by sweeping a 2D object along a path.

System Variables

DELOBJ

Controls whether geometry used to create 3D objects is retained or deleted.

SURFU

Sets the surface density for PEDIT Smooth in the M direction and the U isolines density on surface objects.

SURFV

Sets the surface density for PEDIT Smooth in the N direction and the V isolines density on surface objects.
