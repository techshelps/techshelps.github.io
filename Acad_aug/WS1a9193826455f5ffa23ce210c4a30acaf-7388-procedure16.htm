To turn a layer on and off by using the Layers toolbar

  1. Click Home tab Layers panel Layer Properties
  2. In the Layer Properties Manager, click the light bulb for the layer name that you want to turn on or off.

    If the light bulb is yellow, the layer is turned on.

 Toolbar: Layers 
 Command entry: LAYER
