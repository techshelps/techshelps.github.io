To move objects in 3D space along a specified axis

  1. Click Home tabModify panel3D Move. At the Command prompt, enter 3dmove.
  2. Select the objects and subobjects you want to move using the following methods:
    • Press and hold Ctrl to select subobjects (faces, edges, and vertices).
    • Release Ctrl to select entire objects.
  3. When you have selected all objects, press Enter.

    The Move gizmo is displayed attached to the cursor.

  4. Click to place the Move gizmo, specifying the base point for the move.
  5. Hover the cursor over an axis handle on the gizmo until it turns yellow and the vector is displayed. Then click the axis handle.
  6. Click or enter a value to specify the distance of the move.
 Toolbar: Modeling
