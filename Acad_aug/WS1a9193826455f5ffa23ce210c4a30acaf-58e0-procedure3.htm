To export a set of queries from the current drawing

  1. In the dbConnect Manager, in the tree view, right-click a drawing file. Click Export Query Set.
  2. In the Export Query Set dialog box, in the Save In list, select the directory where you want to save the query set.
  3. Under File Name, specify a name for the query set. Click Save.
