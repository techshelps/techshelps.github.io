To change existing multiline text to annotative or nonannotative

  1. Double-click a multiline text object. The In-Place Text Editor is displayed.
  2. Click the Annotative button on the toolbar to change existing multiline text to annotative or nonannotative.

    When the Annotative button is depressed, the text is annotative. When the button is not depressed, the text is nonannotative.

  3. Click OK to save the changes.
 Shortcut menu: Select and right-click the text. Click Properties to change the annotative property of the text.
