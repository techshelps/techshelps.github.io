To hide action bars for all parameters

  1. Click Insert tab Block panel Block Editor. At the Command prompt, enter bedit.
  2. Click Block Editor tabAction Parameters panelHide All Actions. At the Command prompt, enter bactionbar.
 Command entry: BACTIONBAR
