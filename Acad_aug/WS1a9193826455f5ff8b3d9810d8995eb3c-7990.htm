A luminaire object is a helper object that assembles a set of objects into a light fixture.

A luminaire object groups and manages the components of a light as a whole. Light fixtures can be represented by embedding photometric lights in blocks that also contain geometry. A luminaire object assembles a set of light objects into a light fixture.

Tool palettes of photometric lights are available to provide easy access to create a luminaire object. Luminaire objects are offered through plug-ins.
