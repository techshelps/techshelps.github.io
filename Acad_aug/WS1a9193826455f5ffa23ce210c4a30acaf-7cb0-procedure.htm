To adjust the position of a map on an object or a face

  1. Select an object.
  2. Click Render tab Materials panel Material Mapping drop-down Planer.
    NoteFrom the Material Mapping drop-down list, you can select Box, Cylindrical, and Spherical material mappings.
  3. A material mapper grip tool is displayed on the object.
  4. Use the grip tools and the representation of the object to adjust the display of the image on the object or face.
 Toolbar: Mapping , Render 
 Command entry: MATERIALMAP
