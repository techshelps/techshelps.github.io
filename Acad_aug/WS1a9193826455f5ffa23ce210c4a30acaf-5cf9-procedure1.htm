To change the names of layers, dimension styles, and other named objects

  1. Click Tools tabDrawing Utilities panelRename. At the Command prompt, enter rename.
  2. In the Rename dialog box, select the named object type and then the item you want to rename.
  3. Enter the new name in the Rename To box below the old name.
  4. Click Rename To. Click OK.
 Command entry: RENAME
