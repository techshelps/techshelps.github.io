ConceptQuick Reference
 
 
 

One additional named plot style table is installed for you to use beyond the default plot style table. All named plot style tables have an .stb extension.

  • acad.stb: Default plot style table
  • Monochrome.stb: All colors plot as black
  • None: No plot style table applied
NoteNamed plot style tables are available only if the drawing has been set to use named plot style tables.
