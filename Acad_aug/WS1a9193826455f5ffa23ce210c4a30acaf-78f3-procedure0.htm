To start a drawing by selecting a template file

  1. Click the Application menu, and click New menu Drawing.
  2. In the Select Template dialog box, select a template from the list.
  3. Click Open.

    The drawing is opened as drawing1.dwg. The default drawing name changes to reflect the number of new drawings that have been opened. For example, if you open another drawing from a template, the default drawing name is drawing2.dwg.

    To start a new drawing with no template file, click the arrow next to the Open button. Select one of the “no template” options from the list.
