Commands

CLASSICLAYER

Opens the modal Layer Properties Manager.

LAYER

Manages layers and layer properties.

LAYERPALETTE

Opens the modeless Layer Properties Manager.

PURGE

Removes unused items, such as block definitions and layers, from the drawing.

System Variables

CLAYER

Sets the current layer.

LAYERDLGMODE

Sets the flavor of the Layer Properties Manager that is defined for the use of the LAYER command.
