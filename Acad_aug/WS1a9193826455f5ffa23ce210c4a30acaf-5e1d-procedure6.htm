To publish selected objects in your model to a 3D DWF or 3D DWFx file

  1. Click Output tab Export to DWF/PDF panel 3D DWF. At the Command prompt, enter 3ddwf
  2. In the Export Data dialog box, specify a name and location for the DWF or DWFx file.
  3. Click the Tools button at the upper right.
  4. Click Tools menu  Options.At the command prompt, enter options.
  5. In the 3D DWF Publish dialog box, Objects To Publish, click Selected Model Space Objects and then click the Select Objects button.
  6. In the model, select the objects that you want to publish.
  7. Press ENTER or the SPACEBAR when you’re done selecting objects.
  8. Click OK.
  9. In the Export Data dialog box, click Save.
  10. (Optional) Click Yes to open the Autodesk Design Review and view the published 3D DWF or 3D DWFx file.
