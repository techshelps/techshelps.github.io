To change the order of standards files associated with the current drawing

  1. Click Manage tab Cad Standards panel Configure. At the Command prompt, enter standards.
  2. In the Configure Standards dialog box, Standards tab, select the standards file whose position you want to change in Standards Files Associated with the Current Drawing.
  3. Do one of the following:
    • Click the up arrow button (Move Up) to move the standards file up one position in the list.
    • Click the down arrow button (Move Down) to move the standards file down one position in the list.
  4. (Optional) Repeat steps 2 and 3 if you want to change the position of other standards files in the list.
  5. Click OK.
 Command entry: STANDARDS
