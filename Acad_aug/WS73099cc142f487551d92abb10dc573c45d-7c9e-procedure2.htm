To sort extracted data in a column

  1. Click Insert tab Linking & Extraction panel Extract Data.At the Command prompt, enter dataextraction
  2. In the Data Extraction wizard on the Refine Data page, click Sort Columns Options.
  3. In the Sort Columns dialog box, select a column name from the drop-down list.
  4. In the Order column, click Ascending or Descending. Click OK.

    The order of the data in the column is changed.

  5. (Optional) To sort additional columns, click Add.
  6. Repeat steps 2 and 3.
  7. Click Insert tab Linking & Extraction panel Extract Data.At the Command prompt, enter dataextraction
