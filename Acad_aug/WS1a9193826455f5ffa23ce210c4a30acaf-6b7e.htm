You have a wide range of options when you need to select objects for editing operations.

Topics in this section

  • Select Objects Individually

    At the Select Objects prompt, you can select one or more objects individually.

  • Select Multiple Objects

    At the Select Objects prompt, you can select many objects at the same time.

  • Prevent Objects from Being Selected

    You can prevent objects on specified layers from being selected and modified by locking those layers.

  • Filter Selection Sets

    You can use object properties or object types to include objects in a selection set, or to exclude them.

  • Customize Object Selection

    You can control several aspects of selecting objects, such as whether you enter a command first or select objects first, the size of the pickbox cursor, and how selected objects are displayed.

  • Group Objects

    A group is a saved set of objects that you can select and edit together or separately as needed. Groups provide an easy way to combine drawing elements that you need to manipulate as a unit.
