To use fog / depth cueing

  1. Select a camera and open its Property palette.
  2. Turn on its front or back clipping plane.
  3. Set the clipping plane offset value.
  4. Click Visualize tabRender panelEnvironment.At the Command prompt, enter renderenvironment.
  5. Turn on Enable Fog and set the color of the fog or depth cueing.

    Light colors are typically used for fog while dark colors are used for depth cueing.

  6. Set the Near and Far Distance values to specify where the fog starts and ends.
  7. Set the fog or depth cue opacity with the Near and Far Fog Percentage values. Click OK.
  8. Render the model.
 Command entry: RENDERENVIRONMENT
