To create a custom render preset from the Render Presets Manager

  1. Click Render tabRender panelRender Presets drop-downRender Presets Manager.
  2. In the Render Presets Manager, select an existing standard or custom preset.
  3. Click the Create Copy button and specify a name for your new preset. Click OK.

    The new preset appears in the tree view under the Custom Render Presets branch.

 Command entry: RENDERPRESETS
