To detach an image

  1. Click Insert menu  External References.At the Command prompt, enter externalreferences.
  2. In the External References palette, right-click the image name.
  3. Click Detach.

    The image is no longer linked to the drawing file, and all instances of the image are removed from the drawing.

 Command entry: EXTERNALREFERENCES
