To change an object’s property setting to ByLayer

  1. Click Home tabModify panelSet to ByLayer. At the Command prompt, enter setbylayer.
  2. Enter S to specify which properties to change to ByLayer or select objects using a standard selection method and press ENTER.
 Command entry: SETBYLAYER
