To unstack text

  1. Double-click the multiline text object you want to edit.
  2. In either the MTEXT ribbon contextual tab or In-Place Text Editor, select the stacked text.
  3. Click Stack on the Text Formatting toolbar.
  4. To save your changes and exit the editor, use one of the following methods:
    • Click OK on the toolbar.
    • Click in the drawing outside the editor.
    • Press CTRL+ENTER.
