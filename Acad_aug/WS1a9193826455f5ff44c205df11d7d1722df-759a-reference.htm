System Variables

DGNOSNAP

Determines whether object snapping is active for geometry in DGN underlays that are attached to the drawing.

DWFOSNAP

Determines whether object snapping is active for geometry in DWF or DWFx underlays that are attached to the drawing.

PDFOSNAP

Determines whether object snapping is active for geometry in PDF underlays that are attached to the drawing.

UOSNAP

Determines whether object snapping is active for geometry in DWF, DWFx, PDF, and DGN underlays that are attached to the drawing.
