Commands

ATTSYNC

Updates block references with new and changed attributes from a specified block definition.

INSERT

Inserts a block or drawing into the current drawing.

PROPERTIES

Controls properties of existing objects.

RESETBLOCK

Resets one or more dynamic block references to the default values of the block definition.

System Variables

BTMARKDISPLAY

Controls whether or not value set markers are displayed for dynamic block references.

GRIPDYNCOLOR

Controls the color of custom grips for dynamic blocks.

GRIPTIPS

Controls the display of grip tips when the cursor hovers over grips on dynamic blocks and custom objects that support grip tips.
