The Autodesk® ShowMotion® navigation tool provides an onscreen display that can be used to create and play back cinematic camera animations. These animations can be used for presentation purposes or to navigate through a design.

Topics in this section

  • Overview of ShowMotion
  • Create and Modify Shots and Shot Sequences

    You create and modify shots from ShowMotion. As you create or modify a shot, a thumbnail is generated and added to a shot sequence.

  • Play Back a Shot

    You can view shots individually or together in a shot sequence.
