To plot a layout

  1. Click Output tab Plot panel Plot.At the Command prompt, enter <plot>
  2. Do one of the following:
    • Right-click a layout tab. Click Plot.
    • Hold down SHIFT to select several layout tabs. Right-click a layout tab. Click Publish Selected Layouts.
  3. Select settings in the Plot or the Publish dialog box. Click OK or Publish.
 Toolbar: Standard 
 Command entry: PLOT
