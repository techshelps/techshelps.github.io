To disable or remove a texture map from a material

  1. Click Render tab 3D Palettes Materials.
  2. In the Materials window, in the Available Materials in Drawing panel, select a material swatch or create one by clicking the Create a New Material button.
  3. Select the Realistic material type.
  4. Select Diffuse Map and select Texture Map as the map type.
  5. Click Select Image and select an available image.
  6. In the Nested Map Navigation, click the drop-down arrow.

    The Diffuse and the Texture Map image name are displayed.

  7. Clear the Diffuse Map check box.
  8. In the Nested Map Navigation panel, click the drop down arrow.

    The Diffuse and the Texture Map image name are greyed out and disabled.

  9. Select the Diffuse Map check box.
  10. In the Nested Map Navigation panel, click the drop down arrow.

    The Diffuse and Texture Map image name are enabled.

  11. In the Diffuse Map channel panel, click the Delete Map Information from Material button to delete the Texture Map assigned to the material.
  12. In the Nested Map Navigation panel, click the drop down arrow.

    The texture map image name is no longer displayed in the navigation list.
