To edit a sheet list table

  1. Click View tab Palettes panel Sheet Set Manager.
  2. In the Sheet List tab, select a cell in an existing sheet list table.
  3. Right-click the cell. Click Edit Sheet List Table Settings.
  4. In the Edit Sheet List Table Settings dialog box, do one or more of the following:
    • Change the Table Style if you don’t like the current format.
    • Edit the Title Text for the table.
    • Add, remove, or change the order of the column entries.
    • Change the data type or heading text for column entries.
  5. Click OK to close the dialog box and change the table to the drawing.
 Shortcut menu: Select any cell in the sheet list table Right-click to open the shortcut menu Edit Sheet List Table Settings.
