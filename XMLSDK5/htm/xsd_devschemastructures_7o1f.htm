Data Type Definitions

Data type definitions are created from another base type using either extension or restriction.

Simple type definitions describe the content.

Complex type definitions describe the structure or the content model of the XML document.

The following can be complex data types:

  • Elements that contain other elements.
  • Elements that contain both elements and text.
  • Elements that contain only text.
  • Elements that are empty.
