Microsoft XML Core Services (MSXML) 5.0 for Microsoft Office - Digital Signatures

Supported Transforms and Their Namespace URIs

MSXML supports the following transforms in processing an XML digital signature:

Transform Element in Signature Template Namespace URI
SHA1 <SignatureDigest> http://www.w3.org/2000/09/xmldsig#sha1
DSA with SHA1 (DSS) <SignatureMethod> http://www.w3.org/2000/09/xmldsig#dsa-sha1
RSA with SHA1 <SignatureMethod> http://www.w3.org/2000/09/xmldsig#rsa-sha1
HMAC with SHA1 <signatureMethod> http://www.w3.org/2000/09/xmldsig#hmac-sha1
Canonical XML without comments <CanonicalizationMethod> http://www.w3.org/TR/2001/REC-xml-cl4n-20010315
Canonical XML with comments <CanonicalizationMethod> http://www.w3.org/TR/2001/REC-xml-cl4n-20010315#WithComments
XSLT <Transformation> http://www.w3.org/TR/1999/REC-xslt-19991116
XPath <Transformation> http://www.w3.org/TR/1999/REC-xpath-19991116
Enveloped Signature <Transformation> http://www.w3.org/2000/09/xmldsig#enveloped-signature
Base64 encoding <Transformation> http://www.w3.org/2000/09/xmldsig#base64
