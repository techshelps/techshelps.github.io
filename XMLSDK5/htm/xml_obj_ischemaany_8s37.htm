namespaces Property

Returns a string collection of namespaces. The namespaces are used by the <anyAttribute> declaration to validate the well-formed XML.

[Script]

Implementation Syntax

var oNamespaces = oISchemaAny.namespaces;

Parameters

None.

Return Values

oNamespaces
An object. The collection of namespaces for the any particle. The namespaces returned can be: ((##any|##other)|List of (anyURI|(##targetNamespace|##local))).
[Visual Basic]

Implementation Syntax

Parameters

None.

Return Values

[C/C++]

Implementation Syntax

HRESULT get_namespaces(ISchemaStringCollection* namespaces);

Parameters

namespaces [out,retval]
An object. The collection of namespaces for the any particle.

Return Values

S_OK
The value returned if successful.
E_POINTER
The value returned if the namespaces object is NULL.
E_FAIL
The value returned if something else is wrong.
