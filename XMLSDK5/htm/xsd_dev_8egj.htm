XML Schemas Overview

XML schemas are powerful and complex tools for creating structure and validating that structure in compliant XML documents. Schemas are XML files that typically have an .xsd file name extension. Schemas describe the contents of XML documents using valid XML; elements and attributes are declared using element and attribute elements.

An organization and its partners can benefit by using schemas to exchange data (through schema-compliant XML documents). Schemas are also useful for keeping XML documents consistent throughout an organization.
