Creating a Document Map from an XML Schema

It is often useful for users to see a pictorial representation of XML Schema document restrictions. Creating a document map for an XML Schema can provide a quick reference to an XML Schema document. You can use the SOM to programmatically read the parts of the XML Schema. The values returned are then placed in various formats, such as a graphics or text file.

The Walk the SOM sample application presents a document map in a tab-indented format.
