Output for the reset Example

When you build and run the reset example, you should get the following output.

errorItem: Element content is invalid according to the DTD/Schema.
Expecting: price.

errorItem: Element content is invalid according to the DTD/Schema.
Expecting: pub_date.

errorItem (reset): Element content is invalid according to the DTD/Schema.
Expecting: price.
