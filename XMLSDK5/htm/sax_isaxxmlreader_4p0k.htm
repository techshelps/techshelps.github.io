putErrorHandler Method (C/C++)

Allows an application to register an ErrorHandler. If the application does not register an ErrorHandler, all error events reported by the reader are ignored. It is strongly recommended that all Simple API for XML (SAX2) applications implement an ErrorHandler to avoid unexpected bugs. An application can register a new or different handler while in the middle of parsing a document, and the reader will immediately start using the new handler.

Syntax

HRESULT putErrorHandler(
   [in] ISAXErrorHandler * pHandler);

Parameters

pHandler [in]
The ErrorHandler to use.

Return Values

S_OK
The value returned if the ErrorHandler is successfully registered.
