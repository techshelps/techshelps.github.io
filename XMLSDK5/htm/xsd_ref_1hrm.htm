Quantifier

One of the following characters or strings: ?, *, +, {n,m}, or {n,}. For more information about these characters or strings, see Piece.

Syntax

quantifier :: = [?*+] | ('{'quantity'}')

Syntax

quantity :: = quantRange | quantMin | QuantExact

Syntax

quantRange :: = QuantExact ',' QuantExact

Syntax

quantMin :: = QuantExact ','

Syntax

QuantExact :: = [0-9]+

Example

The following are examples of allowed data for regex.

Regex Allowed data
a * x x, ax, aax, aaax…
a ? x ax, x
a + x ax, aax, aaax…
(a | b) + x ax, bx, aax, abx, bax, bbx, aaax, aabx, abax, abbx, baax, babx, bbax, bbbx, aaaax…
