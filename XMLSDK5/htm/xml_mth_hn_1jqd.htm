Microsoft XML Core Services (MSXML) 5.0 for Microsoft Office - DOM Reference

Resource Files (doc1.xml and doc2.xml)

Use the following resource files for the importNode example.

Input XML Data File (doc1.xml)

<?xml version="1.0" encoding="utf-8"?>
<doc>
   <a class="1">doc1</a>
   <b class="1">author1</b>
</doc>

Input XML Data File (doc2.xml)

<?xml version="1.0" encoding="utf-8"?>
<doc>
   <a class="2">doc 2</a>
   <b class="2">author 2</b>
</doc>
