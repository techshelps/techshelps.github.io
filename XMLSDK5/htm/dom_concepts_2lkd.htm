Microsoft XML Core Services (MSXML) 5.0 for Microsoft Office - DOM Developer's Guide

Introduction to the DOM

The Document Object Model (DOM) presents an easily processed standardized interpretation of an XML document to applications and scripts. The DOM implementation in Microsoft® XML Core Services (MSXML) 5.0 for Microsoft Office allows you to load or create a document; gather errors, if any; access and manipulate the information and structures contained within the document; and save the document back out to an XML file, if necessary.

This introduction contains the following topics:

  • DOM and MSXML
  • Modeling Documents as Node Trees
  • Working with the DOM
  • Additional MSXML Features
