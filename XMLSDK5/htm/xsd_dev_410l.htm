Microsoft XML Core Services (MSXML) 5.0 for Microsoft Office - XML Schemas

Version and Conformance

MSXML versions 4.0 and later comply with the World Wide Web Consortium (W3C) 2001 XML Schema Recommendation.

Version Support
MSXML No support for schemas.
MSXML 2.0 Support for XML-Data Reduced (XDR) schemas.
MSXML 2.6 Support for XDR schemas.
MSXML 3.0 Support for XDR schemas.
MSXML 4.0 Support for XML Schema definition language (XSD) and XDR schemas.
MSXML 5.0 Support for XML Schema definition language (XSD) and XDR schemas.
