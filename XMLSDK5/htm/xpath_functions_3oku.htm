string-length Function

Returns the number of characters in the string.

number string-length(string)

Remarks

If the argument is omitted, it returns the number of characters in the string-value of the context node.

The following function call returns 4:

string-length("abcd")

The following function call returns 5:

string-length("ab cd")

If an argument is not of type string, it is first converted to a string and then evaluated.
