To add the stocks.xml resource file to the project

  1. Create a new C++ source file. For detailed instructions on how to do this, see Set Up My Visual C++ Project. Name the new file stocks.xml
  2. Copy the resource listing above and paste it into the source file you just created.
    Note   You can also copy the file into the project's main directory using Windows Explorer (or a command prompt).

Next, build and run the loadDOMsmart project. The result should be the output shown in the following topic.
