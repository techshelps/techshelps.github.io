Microsoft XML Core Services (MSXML) 5.0 for Microsoft Office - XSLT Developer's Guide

XSLT Developer's Guide

This guide is intended to help you, the XSLT developer, understand the basic principles and techniques of programming with XSLT using the MSXML 5.0 SDK.

The following topics provide more information about XSLT.

Topic Description
Starter Kit Provides a high-level overview for beginning XSLT developers. This includes a Hello, World! XSLT example to get you started writing an XSLT style sheet.
Concepts Examines various features of XSLT in detail.
How Do I? Provides a series of task-based exercises as a hands-on introduction to writing XSLT solutions.
