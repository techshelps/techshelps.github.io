getEntityResolver Method (C/C++)

Returns the current EntityResolver.

Syntax

HRESULT getEntityResolver(
   [out, retval] ISAXEntityResolver ** ppResolver);

Parameters

ppResolver [out, retval]
Pointer to the EntityResolver currently in use.

Return Values

S_OK
