XDR Schema Developer's Guide

XML-Data Reduced (XDR) schema is a subset of ideas described in the XML-Data specification, specifically the World Wide Web Consortium (W3C) XML-Data Note and the Document Content Description (DCD) initiative for XML.

MSXML 2.0 and later provide XDR schema support.
