Microsoft XML Core Services (MSXML) 5.0 for Microsoft Office - Digital Signatures

CAPICOM and CryptoAPI

[This feature was first implemented for MSXML 5.0.]

CAPICOM and CryptoAPI are frameworks supported by Microsoft to enable applications to offer cryptographic and other security-related services. CryptoAPI is C-based. CAPICOM is COM-based and can be used in scripting. For more information, see Cryptography under the Security section in the MSDN Library.
