To verify that multi-GPU acceleration is enabled and working

  1. From the NVIDIA Control Panel navigation tree pane, under 3D Settings, select Set Multi-GPU configuration.

  2. From the menu bar, click 3D Settings, then click the drop-down menu item Show Multi-GPU Visual Indicator.

The following images indicate that multi-GPU technology is active.

  • A vertical green bar on the left side of the game screen appears if multi-GPU AFR rendering is enabled and working.

  • The antialiasing level appears on the game screen if multi-GPU antialiasing is enabled and working.

  • A green square appears on the primary - or focus - display when the game is viewed in full-screen mode.

If no image appears, then the system is not in multi-GPU mode.
