To configure unique 3D settings for my applications and games

  1. From the NVIDIA Control Panel navigation tree pane, under 3D Settings, select Manage 3D settings.

    The Manage 3D Settings page appears.

  2. Click the Program Settings stab.

    The Program Settings tab allows you to assign unique 3D settings to a game when it is launched; the game will automatically use this setting whenever it is launched.

  3. Under Select a Program to Customize, click the list arrow and then select the game for which you want to customize the preset 3D settings.   

    • If you want to select a game that is installed on your system but is not listed, click the Add button and then from the Open dialog box browse to locate the game that you want to add, then click Open.

    • If you want the list to show only those games that are installed on your system, check the Show only programs found on my computer check box.

       Note: The driver can still detect any games that are played and apply the appropriate 3D settings, even if the installed-game list filter did not detect the game on the system.

  4. In the Specify the settings box, make adjustments as needed for each listed feature.

    • To change a feature setting, click the corresponding setting, then click the setting that you want to use from the drop-down list that appears.   

    • To restore the default settings, from the Select a Program list select the program that you want to restore and then click the Restore button that is adjacent to the Select a Program list.  

    NOTE: Clicking the Restore Defaults link at the top of the page will restore all the defaults for the entire Manage 3D Settings page.

  5. Click Apply when done.
