To turn profile notifications on

When this feature is turned on, a notification will slide out on the desktop - just above the system tray - whenever a 3D workstation program is launched. The notification lets you know which workstation application profile is applied.

Note: This feature is not available on all computers, and applies only to windowed 3D programs running on NVIDIA Quadro cards.

  1. From the NVIDIA Control Panel navigation tree pane, under 3D Settings, select Manage 3D settings.

    The Manage 3D Settings page appears.

  2. Click the Preferences tab.

  3. Click the Show profile notification check box.

  4. Click the How long arrow and select the amount of time you want the notification to appear when a 3D program is launched.

  5. Click Apply.
