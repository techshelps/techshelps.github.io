NVIDIA PhysX Configuration

NVIDIA® PhysX® is a powerful physics engine that can utilize GPU acceleration to provide amazing real-time physics effects.

PhysX GPU acceleration is available on GeForce 8 series and later GPUs. In order to enable PhysX GPU acceleration, all the GPUs in your system must be PhysX-capable. See www.NVIDIA.com for the complete list of supported GPUs.
