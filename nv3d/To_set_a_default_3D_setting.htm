To set default 3D settings

  1. From the NVIDIA Control Panel navigation tree pane, under 3D Settings, select Manage 3D settings.

    The Manage 3D Settings page appears.

  2. Click the Global Settings tab.

    The Global Settings tab allows you to assign global settings that will be applied to all 3D games.

  3. If you have an NVIDIA Quadro product installed, then click the Global Presets list arrow and select one of the pre-installed global settings for workstation applications.

    The Global Presets list does not appear for GeForce products.

  4. In the Settings box, click the setting corresponding to the feature that you want to change, then from the drop-down list click the setting value that you want to use.

    • To restore the default global settings, click the Restore button on the bottom corner of the Global Settings tab.  

    • To restore the default settings for a global preset (workstation products), select the global preset and then click the Restore button that is adjacent to the Global Presets list.

    NOTE: Clicking the Restore Defaults link at the top of the page will restore all the defaults for the entire Manage 3D Settings page.

  5. Click Apply when done.
