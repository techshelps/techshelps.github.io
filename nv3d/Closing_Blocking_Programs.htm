Closing Blocking Programs

Some programs do not support changing the graphics mode while the application is running. If such a program is running a 3D process, you will not be able to change SLI settings unless you first close the "blocking" program.

Bring the opened program up on your screen and then close it using any method available for that program, such as:

  • Clicking File->Exit from the program's menu bar.

  • Clicking the Close icon (X) at the upper corner of the program window.

If you cannot locate a blocking program on the Windows desktop, task bar, or by pressing [Alt+tab], close the program using the Windows Task Manager:

  1. Press [Ctrl+Alt+Del] , then click Start Task Manager.

  2. From the Windows Task Manager Applications tab, click the blocking program, then click End Task.

    If you do not see your program in the Applications tab, click the Processes tab and then click the program's .exe name and click End Process.
