To verify that PhysX acceleration is enabled and working

  1. From the NVIDIA Control Panel navigation tree pane, under 3D Settings, select Set PhysX configuration.

  2. From the menu bar, click 3D Settings, then click the drop-down menu item Show PhysX Visual Indicator.

At the corner of the game screen, the text "PhysX>GPU" or "PhysX>CPU" indicates the type of PhysX acceleration the game is using. If no text appears, then the game is not using PhysX acceleration.
