The ActiveX Designer's Type Information does not match what was saved. Unable to Load.

   

This error has the following causes and solutions:

  • The project could not be loaded because the ActiveX designer's Type information doesn't match what was saved. This can be due to a corrupt file.

    Attempt to load a backup copy of the project.

For additional information, select the item in question and press F1 (in Windows) or HELP (on the Macintosh).
