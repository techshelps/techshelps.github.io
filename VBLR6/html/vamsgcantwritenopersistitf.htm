Illegal parameter. Can't write object because it does not support persistence.

   

This error has the following causes and solutions:

  • You attempted to write an object to a PropertyBag object, but the object doesn't support one of the ActiveX persistence interfaces.

    To fix this, you must have access to the code for the object. It must a Visual Basic-created object and have its Persistable property set to True. Also, the Class must be either Public or Public Createable and be in an ActiveX Dll, ActiveX Exe, or UserControl project.

For additional information, select the item in question and press F1 (in Windows) or HELP (on the Macintosh).
