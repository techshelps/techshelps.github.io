TechSmith User Design Initiative

The TechSmith User Design Initiative is the perfect opportunity for you to help us improve TechSmith products. If you choose to opt-in, we collect anonymous information about your hardware configuration and how you use TechSmith products. This allows us to identify trends and usage patterns.
