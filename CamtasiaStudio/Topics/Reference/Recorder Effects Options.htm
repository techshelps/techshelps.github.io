Reference : Recorder Effects Options

Recorder Effects Options

Recorder > Effects menu > Options.

The Effects Options tabbed dialog box is used to set recording options such as system time and date stamps, mouse audio effects, caption, and cursor effects.

Camtasia Recorder permanently embeds these effects into the recording. Adding similar effects in Camtasia Studio allows those effects to be edited.

Important Message about Recording the Effects

Before you record, enable the corresponding option from the Effects menu or the corresponding toolbar. If the option is not enabled, the effects will not be included in the recording.
