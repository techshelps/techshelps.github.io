Produce and Share a Video : Produce Using Custom Production Settings : Production Wizard Video Size

Production Wizard Video Size

Camtasia Studio > Produce and Share button > Custom production settings > Choose video file format > Next > Encoding Options > Video Size.

Video Size Options

Select the final dimensions for the video. The default size originally set in the Editing Dimensions dialog box is recommended.

Option Description
Editing dimensions video size (Recommended) Produces a video at the dimensions selected in the Editing Dimensions dialog box.
Custom size Enter a custom width and height to produce the video.
