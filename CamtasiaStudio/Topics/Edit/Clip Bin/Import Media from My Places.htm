Import Media from My Places

Import your files from a shortcut in My Places, such as OneDrive for Business.

  1. In Camtasia Studio, select File > Import from My Places.
    You may be prompted to sign in to your TechSmith Account.
  2. Select a location from My Places.
  3. If you do not have any locations in My Places or to import from a new location, see Add or Remove a Place.
  4. Browse to select the file to import.
  5. Click Import.

The file opens in the clip bin.
