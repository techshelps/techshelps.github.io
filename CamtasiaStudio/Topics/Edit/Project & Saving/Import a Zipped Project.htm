Import a Zipped Project

When a zipped project is imported, Camtasia Studio unzips it to a folder that you choose. At that time, the project is ready to open within Editor.

  1. In Editor, choose File > Import Zipped Project.
  2. The Import Zipped Project File dialog box appears. Click Browse and select a zipped project file.
  3. Select a project destination folder.
  4. Check the Open project after import option to open the project in Editor.
  5. Click OK.
