Change the Recorder Hotkeys

  1. In Camtasia Recorder, select Tools > Options > Hotkeys tab.
  2. Select a recording function.
  3. Choose the shortcut keystrokes.
  4. Click OK.

Default Hotkeys

Option Default Hotkey
Record/Pause F9
Stop F10
Marker CTRL + M
ScreenDraw CTRL+SHIFT+D
Select Region None
Hide Tray Icon None
Mute None
