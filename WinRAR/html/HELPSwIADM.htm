Switch -IADM - request administrative access for SFX archive

Use -IADM if you wish to create SFX archive, which will request the administrative access when started in Windows Vista and later. This switch does not change behavior of SFX archive in older Windows versions.

This switch is the command line equivalent of "Request administrative access" option in SFX options: Advanced dialog.

Example

WinRAR a -sfx -iadm install.exe c:\data
