Switch -MA[4|5] - specify a version of archiving format

By default WinRAR creates archives in RAR 5.0 format. Use -ma4 to create RAR 4.x archives. Use -ma5 or just -ma in case you need to override -ma4 saved in configuration and force WinRAR to use RAR 5.0 format.

This switch is used only when creating a new archive. It is ignored when updating an existing archive.

Example

Create RAR 4.x solid archive and compress SrcDir using 2 MB dictionary:

WinRAR a -ma4 -md2m data SrcDir
