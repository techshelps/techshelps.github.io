Command D - delete files from archive

Delete specified files and folders from an archive. This is the command line equivalent of the archive management command Delete.

Notes

  1. If this command removes all files from archive, the empty archive is removed.
  2. This command is not applicable to volumes, because volumes cannot be modified.

Example

delete all *.dmy files from archive dummy.rar

WinRAR d dummy.rar *.dmy
