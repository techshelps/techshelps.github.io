"Search results" dialog

This dialog contains items found in Find files command. You may select one or more files and then press "Extract To" to unpack them. You may also press "View" to view a file under the cursor or "Locate" to place the cursor to this file in the main WinRAR window.

Because of technical limitations operations with found files are started with some delay if search is active. WinRAR will perform the command only when the currently processing archive is closed.
