"Extract without confirmation" command

This command is accessible both in file management and in archive management modes. If WinRAR is inside of archive, you need to select archived files and folders, which are to be extracted. If you browse uncompressed files in the file management mode, select files and folders which will be searched for archives to unpack. Then run "Extract without confirmation" command using either Commands menu, "Extract" button on the toolbar or Alt+W keyboard shortcut. The selected files will be extracted to the current folder or to the default folder for extracted files, if it is specified in Paths settings dialog. If you need to set a different destination folder or some additional options, use Extract to the specified folder command instead.

Command line equivalent: command "x"
