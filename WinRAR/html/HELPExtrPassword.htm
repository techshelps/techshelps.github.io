Enter password for extracted file

You are trying to extract an encrypted file or view contents of encrypted archive. Enter the password or cancel the extraction.

If you unpack several archives at once, you can enable "Use for all archives" option to apply the entered password to all archives. This option is not available if only one archive is processed. Another possible application for this option is skipping encrypted archives. Just enter an empty string as a password and set "Use for all archives" on.

If you already saved some passwords, you can access them either using the drop down list associated with "Enter password" field or utilizing the autocomplete feature. Press "Organize passwords..." button to open "Organize passwords" dialog, where you can save frequently used passwords.

If saved passwords are protected with master password, you need to enter the master password and press "OK" to access them. If entered password does not match the master password, this dialog will treat it as a usual password for archive operations.
