Switch -OW - process file security information

Use this switch when archiving to save file security information and when extracting to restore it.

WinRAR stores file owner, group, file permissions and audit information with this switch, but only if you have necessary privileges to read them. Note that only NTFS file system supports the file based security.

Shell mode equivalent of this switch is the "Save file security data" option in the Archive name and parameters dialog.

This switch is supported only for RAR archives.
