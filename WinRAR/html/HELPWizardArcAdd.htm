Wizard: Choose the archive

Here you need to select an archive to create or update.

If you create a new archive, usually WinRAR will propose some name for it. You may either accept this name or enter another. If "Default folder for archives" option in WinRAR settings is not blank, WinRAR will propose to create an archive in this folder. Otherwise the proposed name does not include the path and archive will be created on Desktop. If you wish to create it in another folder, press "Browse" button, find a folder to place the archive, enter an archive name in the "File name" field and press "Open".

If you wish to add files to already existing archive, you may also use "Browse" to locate the archive to update or drag the archive icon and drop the icon to this dialog.

After you selected the archive name, press "Next" button at the bottom of dialog.
