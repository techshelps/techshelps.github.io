Command C - add an archive comment

This command allows to add a comment to an archive. Maximum comment length is 256 KB for RAR archives and 32768 bytes for ZIP archives. It is also possible to add an archive comment through archive management command Add archive comment.

Because archive volume modification is disabled, this command is not applicable to volumes. You may use switch -z<file> to add a comment to a volume when creating it.

Examples

  1. add comment to the archive distrib.rar

WinRAR c distrib.rar

  1. Comments may be added from a file by using the -z<file> switch.

WinRAR c -zinfo.txt dummy
