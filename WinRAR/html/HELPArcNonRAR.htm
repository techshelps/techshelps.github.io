Support of other archive formats

Additionally to complete support of RAR and ZIP, WinRAR provides basic operations for 7Z, ARJ, BZ2, CAB, GZ, ISO, JAR, LZ, LZH, TAR, UUE, XZ, Z files created by other tools: view contents, extract files, show comments and archive information. You do not need to have any external programs to handle these formats. Several types of ZIPX archives are supported.

Also WinRAR can combine files split to multiple parts with .001, .002, ... extensions. You can open .001 file as a usual archive and then unpack its contents. All parts must be stored in the same folder before starting extraction.
