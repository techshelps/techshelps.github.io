Switch -N@<listfile> - read additional filter masks from list file

Similar to -n<file> switch, but reads filter masks from the list file.

This switch does not replace usual list files or file masks, which still need to be entered in the command line. It is an additional filter limiting processed files only to those matching the include mask specified in -n switch.

Example

WinRAR a -r -n@inclist.txt text Project Info @listfile.txt
