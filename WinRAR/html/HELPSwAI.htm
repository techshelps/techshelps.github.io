Switch -AI - ignore file attributes

If this switch is used when extracting, WinRAR does not set general file attributes stored in archive to extracted files. This switch preserves attributes assigned by operating system to a newly created file. It affects archive, system, hidden and read-only attributes.

If this switch is used when archiving, predefined values, typical for file and folder, are stored instead of actual attributes.

Supported only for RAR and ZIP formats when extracting and only for RAR format when archiving.
