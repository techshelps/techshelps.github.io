Switch -P[pwd] - set password

Set password <pwd> to encrypt files when archiving or to decrypt when extracting. Passwords are case-sensitive. Maximum password length is 127 characters. Longer passwords are truncated to this length. If you omit a password in command line, you will be prompted to enter it.

In the shell mode a password may be entered through Enter default password dialog or in the Archive name and parameters dialog.

Example

add the contents of the folder "games" to the archive " secret", using the password ZaBaToAd

WinRAR a -pZaBaToAd -r secret games\*.*

Notes

  1. Use 8 characters or longer passwords for higher security. Avoid passwords which are easy to guess.
  2. Remember your passwords. If you forget a password to encrypted archive, you will not be able to unpack it.
