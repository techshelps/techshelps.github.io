Switch -AO - add files with "Archive" attribute set

This switch is designed for backup purposes. If it is used when archiving, only files with "Archive" file attribute will be added to archive. This switch does not affect folders, so all matching folders are added regardless of their attributes. You can also specify -ed switch if you prefer to omit all folder records.

Example

add all disk C: files with "Archive" attribute set to the 'f:backup' and clear files "Archive" attribute

WinRAR a -r -ac -ao f:backup c:\*.*
