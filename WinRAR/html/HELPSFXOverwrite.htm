SFX commands: Overwrite

Overwrite=[n]

If 'n' is 0, the user will be asked before overwriting existing files. It is the default behavior and the Overwrite command does not need to be specified for it. If 'n' is absent or equal to '1', all files are overwritten without a confirmation. If 'n' is equal to '2', all existing files will not be overwritten.

Example

Overwrite=1
