In desktop UI, you can use IME pad with a mouse or touch screen.

IME pad is a common framework for IME pad applets, which can be used to enter Chinese characters and/or symbols in different ways. There are four applets provided by traditional Chinese IME pad: Hand writing applet, Strokes applet, Radical applet, and Symbols applet.

¡@

Open IME pad

  1. Right-click the input language icon ( or ).
  2. Click IME Pad.

Note: Enter, Space, Del, Esc and the arrow buttons on the right of the dialog box have the same functions as the corresponding keys on the keyboard.
