Microsoft IME has a default dictionary to select candidates automatically for faster input and less need of selecting candidates. Since each user has different requirements, there is a need to change these phrases manually. For the purpose of research and product improvements, Microsoft would like to collect the phrases you modified manually. You can turn on the mis-conversion report feature to record the phrases you modified manually, and click the Mis-conversion Report button later to send modifications to Microsoft for the purpose of research and product improvement.

These reports will be used for purpose solely in research and product improvements.

Turn on/off mis-conversion report

  1. On the taskbar, right-click the input language icon ( or ), and then click Properties.
  2. On the General tab, in the Privacy setting section, select or clear the Save mis-conversions to file check box.
  3. Click OK.

Note: If you turn off mis-conversion report, all collected data will be removed.

View mis-conversion data

Applicable to: Microsoft New Phonetic

  1. On the taskbar, right-click the input language icon ( or ).
  2. Click Mis-conversion Report.

Edit mis-conversion data

Applicable to: Microsoft New Phonetic

  1. On the taskbar, right-click the input language icon ( or ).
  2. Click Mis-conversion Report.
  3. Select the item you want to edit, and then click Edit.
  4. In the Correct conversion result box, enter the desired characters.

Remove mis-conversion data

Applicable to: Microsoft New Phonetic

  1. On the taskbar, right-click the input language icon ( or ).
  2. Click Mis-conversion Report.
  3. Select the item you want to remove, and then click Remove.¡C

Send mis-conversion data

Applicable to: Microsoft New Phonetic

  1. On the taskbar, right-click the input language icon ( or ).
  2. Click Mis-conversion Report.
  3. Click Send mis-conversion data.

Note:

You can click Details to see information about what will be sent to Microsoft.

You can also click Update to keep your IME updated and optimal.
