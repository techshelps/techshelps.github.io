On Desktopo mode, you can set input method properties with the property sheet for the input method.

Open property sheet

Follow the steps below to open the property sheet for an input method:

  1. Switch to the input method that you want to set properties for.
  2. On the taskbar, right-click the input language icon ( or ).

  3. Click Properties to open the property sheet for the input method.

Note: Available tabs and items on the property sheet vary depending on the input method you are using.
