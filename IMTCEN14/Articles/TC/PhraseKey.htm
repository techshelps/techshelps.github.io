Phrase input key

Show all

　

The phrase input key serves as a delimiter between phrases when you are typing with Microsoft New Phonetic. That is, you can control word-breaking rules to improve conversion accuracy.

For instance, to prevent from converting "電子郵件" to "電子游件", you can type "電子郵件". In that way, "電子" is considered a phrase and so is "郵件".

Set the phrase input key feature

  1. Switch to Microsoft New Phonetic.
  2. On the taskbar, right-click the input language icon ( or ), and then click Properties.
  3. On the property sheet, click the Advanced tab.
  4. Click Phrase input key.
  5. To turn off the phrase input key feature, select OFF; to select a phrase input key, select TAB or SPACE.
  6. Click OK.
