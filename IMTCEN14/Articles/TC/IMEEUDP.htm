By using the user-defined phrases feature, you can define phrases which are not in the main dictionary of Microsoft New Phonetic, and assign shortcut keys for each defined phrase. This will reduce the inconvenience of modifying characters all the time.

Turn on the user-define phrase feature

Before using the user phrase tool, you must turn on the user-defined phrase feature. This feature is turn on by default.

Follow the steps below to turn on the user-defined phrase feature:

  1. On the taskbar, right-click the input language icon ( or ), and then click Properties.
  2. Click the Dictionary tab.
  3. Select the Use the phrases defined in user phrase tool check box.

  4. Press OK.

Open user phrase tool

On the taskbar, right-click the input language icon ( or ), and then click User Phrase to open the user phrase tool.

Add user-defined phrases

You can add user-defined phrases with the user phrase tool in either of the following ways:

Enter a phrase directly

  1. On the Edit menu, click Insert. Or, double-click the first blank row in the table. The Phrase Editor dialog box opens.

  2. In the Phrase box, enter a new phrase.
    • If the phrase contains more than 9 characters or no-reading (phonetic) characters, you must assign the phrase a shortcut key.
    • You can select reading (phonetic) for polyphonic characters.
  3. Click OK.
  4. To edit a phrase, double-click the phrase item. To add a new phrase, repeat steps 1-3.
  5. On the File menu, click Save.

Import a text file

If you need to create a large number of new phrases, we recommend you to create a text file (*.TXT) with each phrase in a single line, and separate the text and phonetic symbols for each phrase. If you do not enter the phonetic symbols, the user phrase tool will automatically add them. Follow these steps to import new phrases:

  1. Save the new phrases in a text file (*.TXT).
  2. Open the user phrase tool.
  3. On the File menu, click Import.
  4. Find and select the text file, and then click Open. New phrases will be imported into the user phrase library.
  5. On the File menu, click Save.

Add user-defined phrase while typing

You can add user-defined phrases while typing. When you are using Microsoft New Phonetic and the composition window is open, press Shift+Left/Right Arrow to select the phrase, and then press Enter to add the selected phrase to the user phrase tool.

You can open the user phrase tool to check whether the selected phrase has been added successfully.
