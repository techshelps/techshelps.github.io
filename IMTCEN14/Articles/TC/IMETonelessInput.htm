If the toneless feature is turned on, you can press any tone key to replace any tone mark. Microsoft New Phonetic will select the correct phrase automatically.

Turn on the toneless feature

  1. Switch to Microsoft New Phonetic.
  2. On the taskbar, right-click the input language icon ( or ), and then click Properties.
  3. On the property sheet, click the Advanced tab.
  4. Select ON for Toneless. (The default setting is OFF.)
  5. Click OK.

¡@

Differences between Toneless key and Toneless:

If the toneless feature is turned on, all tone keys are treated as toneless wildcard, and you can press any tone key for quick input. If the toneless key feature is turned on with a key set as toneless key, tone marks are still effective, and you can use the toneless key only when you are not sure which tone mark should be used.
