The intelligent learning feature learns and records your modification on homophones during input. Your preference will be automatically applied in the future. Intelligent learning saves the phrases that you often use in learning records.

Microsoft New Phonetic supports export and import of learning records.

You can turn on the new phrase learning feature for Microsoft New Phonetic to add frequentlye used phrases to the self-learning learned phrases list in the user phrase tool.

You can also use learning wizard to register new phrases in existing documents.

For more information, see Dictionary tab.
