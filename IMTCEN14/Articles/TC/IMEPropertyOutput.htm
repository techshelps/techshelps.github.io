In the Output settings section, you can set character set and output language.

Character set

　

This setting allows you to select the character set, and it supports Unicode and CNS11643 to display and enter more Chinese characters. Please make sure the required font files have been installed before using the function.

Character sets include:

  • BIG5 characters

    • A standard of Chinese computer industry wildly use in Taiwan and Hong Kong, a general character encoding which is different from code page concept using in Windows.

  • Unicode characters

    • A character encoding standard developed by Unicode Consortium. Unicode represents each character with a byte or above, Unicode allow to represent all characters in different languages with single word.

    • Characters of CJK Unified Ideographs Extension A

      • Defined in Unicode code position 3400 - 4DFF.

    • Characters CJK Unified Ideographs Extension B

      • The Encoding method is divide the Unicode code position D800 - DFFF into 2 areas. The upper area D800 - DBFF includes 1,024 code positions, lower area DC00 - DFFF also has 1,024 code positions. totally can be cross-defined 1,024×1,024＝1,048,576 code positions. This type of method is defined under Unicode as Surrogate.

    • Characters of the HKSCS

      • Traditional Chinese computer systems in Hong Kong are mainly using BIG5 character set, and BIG5 character set doesn't content many of Hong Kong frequent use characters. In order to meet the specific need of the unique Hong Kong characters, the Hong Kong SAR Government has defined the Hong Kong Supplementary Character Set (HKSCS) includes Chinese characters used in Hong Kong but are not contained in the BIG5 standard character set.

In addition, you can select Allow the input sequence defined by CNS 11643 to meet your requirements.

Output language

Applicable to: Microsoft New Phonetic when you are using a physical keyboard

You can write articles in simplified Chinese using Microsoft New Phonetic without the need of learning other pinyin systems. When simplified Chinese is set as the output language, Microsoft New Phonetic will convert simplified Chinese characters automatically.

Opotions:

  • Traditional Chinese (Default)
  • Simplified Chinese
