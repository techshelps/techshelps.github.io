In the General settings section, you can change settings for Default input mode, UI font setting, UI language setting, Query the input sequence of, and Prompt associated phrases of the input character.

Default input mode

Set Chinese or alphanumeric as the default input mode when an IME is started.

Options:

  • Chinese (Default)
  • Alphanumeric

UI font setting

Chang the font size in the composition window and the candidate window.

Options:

  • Normal font (Default)
  • Large font
  • Extra large font

UI language setting

Set the interface language of an IME to Chinese or English.

Options:

  • Chinese
  • English
  • Auto detect (Default)

Query the input sequence of

When using Microsoft Quick or Microsoft ChangJie, you can query phonetics, radicals or readings of another IME by current IME.

Note: Microsoft New Phonetic does not support this feature.

Options:

  • (None) (Default)
  • Phonetic
  • ChangJie
  • Quick
  • Unicode
  • Chinese Traditional DaYi (version 6.0)
  • Chinese Traditional Array (version 6.0)

Prompt associated phrases of the input character

If this option is selected, after you enter a Chinese character, the input method will prompt associated phrases.

Note: For Microsoft Quick and Microsoft ChangJie, this feature is turned on by default; for Microsoft New Phonetic, this feature is turned off by default.
