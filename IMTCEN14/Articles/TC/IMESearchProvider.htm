When you are using Microsoft New Phonetic in desktop UI, you can use search plug-in, with which you can search for information about the content you are typing.

Before you use the search plug-in function for the first time, you must add at least one search provider. Search providers are the search engines that provide search services.

Add a search provider

  1. Switch to Microsoft New Phonetic.
  2. On the taskbar, right-click the input language icon ( or ).
  3. Click Select Search Provider, and then click Add Search Provider.

  4. Select the desired search provider and install it.

Search provider setting

If you have added multiple search providers, you can set one as the default search provider.

  1. Switch to Microsoft New Phonetic.
  2. On the taskbar, right-click the input language icon ( or ).
  3. Click Select Search Provider, and then click Search Provider Setting.
  4. Select the search provider that you want to set as the default search provider, and then click Set Default.

You can also click Remove to remove non-default search providers.

Search using a search provider

You can search for information about characters in the composition or candidate window. To search characters in the candidate window, first press Up/Down arrow key to select a keyword. Press Ctrl+F8 to start searching with the default service provider.

If you have added multiple search providers, after you select a keyword, follow the steps below to select a search provider:

  1. On the taskbar, right-click the input language icon ( or ).
  2. Click Select Search Provider, and then select the search provider you want to search the keyword with.
