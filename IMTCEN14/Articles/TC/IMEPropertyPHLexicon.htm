  1. Open the Property sheet for Microsoft New Phonetic.
  2. Click the Ditionary tab.

  3. Select the desired options/settings.
  4. Click OK.

Enable personal regulating

Selecting the Enable personal regulating check box will turn on the intelligent learning feature. The intelligent learning feature learns and records your modification on homophones during input. You can select Fixed candidate order or Dynamic candidate order based on usage to customize candidate order.

Export: Save the learning records to a file.

Import: Import existing learning records file into Microsoft New Phonetic.

Delete: You can clear the B>Enable personal regulating check box to turn off the intelligent learning feature. And then you can click Delete ro delete all learning records.

Enable new phrase learning

Selecting the Enable new phrase learning check box will turn on the new phrase learning feature. Microsoft New Phonetic will add frequently used phrases, which has been entered for more than five times, to the self-learned phrases list in the user phrase tool. Those new phrases have higher priority in the candidate window.

You can clear the B>Enable new phrase learning check box to turn off the new phrase learning feature. And then you can click Delete ro delete self-learning phrases, so that Microsoft New Phonetic will convert phrases according to the original dictionary.

Invoke Learning Wizard

You can click the Invoke Learning Wizard button for the learning wizard to analyze documents.

The learning wizard can analyze HTML, DOC or plain text files and add new phrases into user-defined phrases list.

Use the phrases defined in user phrase tool

Selecting the Use the phrases defined in user phrase tool check box will turn on the user-defined phrase feature.

Invoke User Phrase Tool: Click the Invoke User Phrase Tool button to open th user phrase tool for you to edit user-defined phrases.

Delete: You can clear the B>Use the phrases defined in user phrase tool check box to turn off the user-defined phrase feature. And then you can click Delete ro delete all user-defined phrases.

Plug-in lexicons

The Plug-in lexicons section lists the lexicons installed and you can set whether to use the installed lexicons.
