You can enter full-width punctuations using the Ctrl key or the symbol keyboard. When using Microsoft New Phonetic with a physical keyboard, you can also enter full-width punctuations by using the leading key or selecting from the same type of punctuations.

Note: When you are using Microsoft New Phonetic with the touch keyboard, these methods are not applicable. You can use the punctuation keys in Chinese/English input view or number/punctuation view to enter punctuations. For more information, see Microsoft New Phonetic.

Use the Ctrl key

Press and hold the Ctrl key, and then press the key corresponding to the punctuation you want to enter. For example, press Ctrl+Comma to enter "¡A" .

If a punctuation corresponding to a shifted key, you need to press Shift before press the punctuation key. For example, press Ctrl+Shift+? to enter "¡H" .

Use symbol keyboard

The symbol keyboard allows you to enter full-width punctuations by directly clicking punctuation keys.

For more information, see Symbol keyboard.

Use leading key

Applicable to: Microsoft New Phonetic using physical keyboard

In Chinese input state, press the leading key (` ), and then type the punctuation.

For example, press `+[ to get "¡e".

Select from the same type of punctuations

Applicable to: Microsoft New Phonetic using physical keyboard

Before confirm and close the composition window, you can open the candidate window to manually select punctuations. If you cannot find the desired punctuation on the keyboard, you can also enter a same type punctuation on the keyboard, and then follow the steps to select a punctuation mark from the candidate window.

  1. Press the Left or Right Arrow key or use the mouse to move the cursor to the left of the punctuation you want to change. (Note: Some applications do not support mouse operations.)

    2. Press the Down Arrow key to open the candidate window.

  2. Press the number key to the desired punctuation, or click the punctuation directly to select and confirm. You can also press the Up or Down Arrow key to select the punctuation, and then press Enter to confirm.
