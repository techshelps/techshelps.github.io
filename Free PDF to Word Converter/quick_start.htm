Quick Start

After you have started Smart PDF Converter you need to:

  1. Click on Add File in order to load in a PDF file.
  2. Set an output format from the drop down menu.
  3. Choose an output folder.
  4. Press the Convert button.

In order to make use of Smart PDF Converter's flexibility and full features, please review the Advanced Options section.
