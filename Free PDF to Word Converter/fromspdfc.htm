Creating a PDF File from the Main Window of Smart PDF Converter

  1. Start Smart PDF Converter by either double clicking its desktop icon or by selecting it from the Windows Start menu

  2. Select the Convert To PDF tab and click Add File in order to load in a DOC, XLS, HTML, TXT or RTF file. To convert other file types look here

  3. Choose an output folder.

  4. Click Convert

Note: This method will work for MS Office documents such as DOC, XLS and also HTML, TXT, RTF and some others. To convert other file types look here
