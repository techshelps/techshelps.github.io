Creating PDF Files directly from Windows Explorer

  1. Find the file you want to convert in My Computer. Click with the right mouse button on the file and select Convert to PDF.

  2. Select a name for you new PDF file and click Save

  3. Your PDF file is ready!

    4. Note: This method will work for MS Office documents such as DOC, XLS and also HTML, TXT, RTF and some others. To convert other file types look here
