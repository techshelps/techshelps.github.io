Setting the Properties of a New PDF File

These properies are optional so you can leave them blak if you don't need them. When you create a PDF file a Save As window opens that enables you to select not only the name and location of the newly created PDF file but also three groups of additional properties:

  • Description
    • Title
    • Author
    • Subject
    • Keywords

  • Security
    • Set User Password
    • Set Owner Password
    • Allow Content Copying and Extraction
    • Allow Form Filling, Signing and Commenting
    • Allow Changing the Document
    • Allow Printing
    • Set Encryption Level to Low or High

  • Advanced
    • Select PDF Page Layout: You can set it to Single Page, Continuous, Facing or Continuous Facing.
    • Select PDF Page Mode: You can set it to Normal, Outlines, Thumbnails or Full Screen.
    • Document Compression: You can select between No compression and the Flate algorithm.
    • PDF Images Compression: You can select among the Flate algorythm, JPEG, CCITT3, CCITT32d and CCITT4
    • JPEG Quality: Set the quality of the JPEG images in the PDF
    • PDF Version: You can select between v1.3 (Acrobat 4) and v1.4 (Acrobat 5)
    • Alternate EMF Rendering
