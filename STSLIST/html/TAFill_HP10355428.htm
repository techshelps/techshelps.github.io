Troubleshoot automatically filling cells with data

I get the message "Cells in the destination region are read-only"

You are dragging the fill handle to include cells that are read-only.

I get the message "Rows were skipped because new rows contained invalid values in required fields"

You are dragging the fill handle to include rows that do not have any values in one or more columns that are marked required.

I get the message "You cannot perform this operation on a calculated column"

You cannot fill values in a calculated column by dragging the fill handle. A calculated column gets its values by evaluating a formula. To view or edit the formula of a calculated column, click a cell in the column, then click again.

I get the message "Attempting to fill cells or rows with invalid values"

You are attempting to fill cells with values that would fail the cell or row validation rules.

I get the message "Some data was truncated"

You are attempting to fill cells with values that are longer than the maximum length allowed in the destination cells.
