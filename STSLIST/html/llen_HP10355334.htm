LEN

LEN returns the number of characters in a text string.

Syntax

LEN(text)

Text   is the text whose length you want to find. Spaces count as characters.

Example

Col1Formula Description (Result)
Phoenix, AZ=LEN([Col1]) Length of the string (11)
 =LEN([Col1]) Length of the string (0)
A     One=LEN([Col1]) Length of the string, which includes 5 spaces (8)
