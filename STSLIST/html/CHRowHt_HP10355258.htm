Change row height

  1. Right-click a row header, and click Row Height.
  2. In the Row Height dialog box, type a number between 1 and 409.

  Notes  

  • Changing the height of a row changes the height of all rows in the list.
  • You can also drag the top or bottom boundary of the row header until the row is the height you want. To size a row so that the contents are fully visible, move the cursor to the bottom boundary of the row header. When the resizing cursor appears, double-click the boundary line.
