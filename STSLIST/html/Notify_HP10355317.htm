Get notified when contents of a list are modified

Receive notification when one of the following is changed: row, document, or cell

  1. Right-click the row header or cell for which you want to enable notification, and click Alert me.
  2. On the New Alert page, specify how and when you want to be notified.

Receive notification if any changes are made to the list or document library

  1. Right-click the top-left corner of the list or library, and click Alert me.
  2. On the New Alert page, specify how and when you want to be notified.
