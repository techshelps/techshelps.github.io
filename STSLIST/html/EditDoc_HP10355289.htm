Edit a document

When you open a document for editing, SharePoint Foundation 2010 can perform several required tasks for you automatically, such as checking out the file, downloading it to your computer, and opening the relevant program.

  • Right-click the row header or any cell in the row corresponding to the document that you want to edit, point to Document, and then click Edit Document.

    If check out of the document is required before editing, SharePoint Foundation 2010 notifies you that you are about to check out and edit the document. Click OK to continue.

The document is opened using the appropriate Office application. You can edit the document, and save the document back to the library or to a different location. When you close a checked out document, you are given the option of either checking the document back in or updating the file on the server while keeping it checked out.

 Note   You cannot edit a document that has been checked out by another user. You also cannot edit any of the columns in the row corresponding to the document until it is checked back in.
