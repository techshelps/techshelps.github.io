Check in or out a document

To check in a document that is checked out to you, but not currently open

  1. Right-click the row header or any cell in the row corresponding to the document that you want to check in, point to Document, then click Check in Document.
  2. On the Check In page, select whether you want to keep the document checked out to you after checking it in to the server or release it so that others can work on it.
  3. Optionally, in the Comments box, you can describe the changes you made to the document.
  4. Click OK.

To check out a document

  • Right-click the row header or any cell in the row corresponding to the document that you want to check out, point to Document, then click Check Out Document.

 Note   You cannot check out a document that is currently checked out by another user.
