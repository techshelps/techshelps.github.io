Approve or reject pending items

If a list is set up to require content approval, users with the appropriate list management rights can review the items that other users have added to the list and either approve or reject them before the information is displayed to the public. Only approved items are displayed to the site users.

  1. In Datasheet View, check the Approval Status column to see if any items are waiting for approval.
  2. Change to Approve/Reject Items view:
    1. In the ribbon, click the List tab, and then in the Current View group, click Approve/reject Items from the Current View drop-down menu.
    2. Make sure you are still in Datasheet View. Click Datasheet View, if necessary.
  3. Optionally, you can filter the list to display only pending list items. Click the drop-down arrow in the column header of the Approval Status column and click Pending.
  4. To approve or reject a single row, click the current status of the item in the Approval Status column, click the down arrow, and then click either Approved or Rejected in the drop-down menu.

    To update adjacent rows with the same status or comment, enter the value in the first row, and drag the fill handle to include the rows you want. You can also copy and paste values in the columns.

  5. Type your comments in the Approver Comments column.

 Note   You can also change the status of an approved or rejected item back to pending. This will un-publish the item so that users visiting the site can no longer see it.
