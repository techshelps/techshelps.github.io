INT

Rounds a number down to the nearest integer.

Syntax

INT(number)

Number   is the real number you want to round down to an integer.

Example 1

Formula Description (Result)
=INT(8.9) Rounds 8.9 down (8)
=INT(-8.9) Rounds -8.9 down (-9)
