Add a row or column

Add a row

By default, the star (*) row — a blank row marked by an asterisk (*) in the row header — is displayed after the last row in a list. You can use this row to add a new item to the list.

 Note   You cannot insert a blank row between two existing rows. If the view is read-only, the star row is not displayed.

To add a row, perform either of the following steps:

  • If the blank row is not visible, right-click any cell and then click Add Row, or,
  • In the ribbon, click the List tab, and then in the Datasheet group, click New Row.

Add a column

  1. Right-click a cell in the list.
  2. Click Add Column.
  3. In the Create Column page, specify the column details, and click OK.

    The new column is added to the right of the existing columns. To move the column to a new location, drag its column header to the location you want.
