Keyboard shortcuts

You can use shortcut keys for quick access to frequently used commands or operations. The topics below list the shortcut keys available in Access Web Datasheet. You can also use access keys to move focus to different parts of the view without using the mouse.

Select parts of the view

To do thisPress

Move selection from left to right, then down. If multiple cells are selected, move focus to the next cell within the selection.

TAB

Move selection from top to bottom, then right. If multiple cells are selected, move focus to the next cell within the selection.

ENTER

Move selection from right to left. If the current cell is the leftmost cell, SHIFT+TAB selects the last cell in the previous row.

SHIFT+TAB

Select the cell above the current cell. If the current cell is the topmost cell, SHIFT+ENTER selects the last cell in the previous column.

SHIFT+ENTER

Move the selection in the direction of the arrow key.

Arrow keys

Extend or reduce the selection in the direction of the arrow key.

SHIFT+Arrow key

Move the selection to the last cell in the direction of the arrow key. If there are blank cells in that direction, move to the last cell preceding the first blank cell.

CTRL+Arrow key

Extend selection to the last cell in the direction of the arrow key. If there are blank cells in that direction, extend selection to the last cell preceding the first blank cell.

CTRL+SHIFT+Arrow key

Select the first cell in the current row.

Home

Select the first cell in the first row.

CTRL+Home

Select the last cell in the last row.

CTRL+End

Extend selection to the beginning of the current row.

SHIFT+Home

Extend selection to the end of the current row.

SHIFT+End

Extend selection to the beginning of the list.

CTRL+SHIFT+Home

Extend selection to the end of the list.

CTRL+SHIFT+End

Extend selection to the entire column.

If text in a cell is selected, and the column supports rich HTML formatting, clear formatting.

CTRL+SPACEBAR

Extend selection to the entire row.

SHIFT+SPACEBAR

Select the entire list.

CTRL+A

Scroll to the next list page.

PAGE DOWN

Display the previous list page.

PAGE UP

Extend a selection down one page.

SHIFT+PAGE DOWN

Extend a selection up one page.

SHIFT+PAGE UP

Scroll to display the current cell.

CTRL+BackSpace

Carry out the shortcut menu command containing the underlined letter.

Underlined letter on a shortcut menu

Undo changes to rows that have not yet been committed.

If a shortcut menu is visible, close the shortcut menu without carrying out a command.

Format a cell that supports rich HTML text

Move focus to the Font list.

CTRL+SHIFT+F

Move focus to the Font size list.

CTRL+SHIFT+P

Apply Bold style to selection.

CTRL+B

Underline selection.

CTRL+U

Italicize selection.

CTRL+I

Justify selection to the left.

CTRL+L

Justify selection to the center.

CTRL+E

Justify selection to the right.

CTRL+R

Add bullets to selection.

CTRL+SHIFT+L

Increase indent.

CTRL+M

Decrease indent.

CTRL+SHIFT+M

Sort and filter data

To do thisPress

Display the filter drop-down list for the selected column.

ALT+SHIFT+DOWN

Close the filter drop-down list.

ALT+SHIFT+UP

Move to the previous item in the filter drop-down list, when the list is displayed.

UP

Move to the next item in the filter drop-down list, when the list is displayed.

DOWN

Move to the first item in the filter drop-down list, when the list is displayed.

HOME

Move to the last item in the filter drop-down list, when the list is displayed.

END

Move one page up or down in the filter drop-down list, when the list is displayed.

Page Up, Page Down

Execute the selected item in the filter drop-down list, when the list is displayed.
