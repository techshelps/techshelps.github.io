Add, edit, or delete cell contents

Make changes one cell at a time

You can add, edit, or delete cell contents in edit mode.

  1. If the cell that you want to edit is not selected, click the cell.

     Note   If the cell contains a hyperlink, you will navigate to the corresponding Web address.

  2. Click the cell again to enter edit mode, then make changes to the cell.

     Note   You can also press F2 to enter edit mode.

  3. Press TAB or ENTER when done.

 Note   Press CTRL+Z to undo the last change you made to the cell. Press ESC to discard the changes you made to the cell and exit edit mode.

Bulk edit cells

You can bulk edit cells, rows, or columns in one of the following ways:

  • By copying and pasting data.
  • By dragging the fill handle.
