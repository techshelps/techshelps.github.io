SQRT

Returns a positive square root.

Syntax

SQRT(number)

Number   is the number for which you want the square root.

Remark

If number is negative, SQRT returns the #NUM! error value.

Example

Formula Description (Result)
=SQRT(16) Square root of 16 (4)
=SQRT(-16) Square root of 16. Because the number is negative, an error is returned (#NUM!)
=SQRT(ABS(-16)) Square root of the absolute value of the number (4)
