Automatically fill cells with data

You can fill a range of cells with a specific value, a range of values, or with linearly increasing values.

Copy contents of a cell to other cells within the column

For example, the Year column in row 1 contains the value 1980, and you want to copy it to rows 2, 3, 4, and 5.

  1. Select the cell whose contents you want to copy.
  2. Position the mouse pointer on the drag fill handle at the bottom right corner of the cell.

  3. When you see the drag cursor (+), drag until all the destination cells are included. You can drag above or below the selected cell.

  4. When you release the mouse button, the contents of the first cell will be copied into the destination cells.

Copy a range of values to other cells within the column

For example, you want to fill the cells in the Quarter column with the values 1, 2, 3, 4, 1, 2, 3, 4, and so on.

  1. Type 1 in the first row, 2 in the second row, 3 in the third row, and 4 in the fourth row.
  2. Select all four cells.
  3. Position the mouse pointer on the drag fill handle at the bottom right corner of the selection.
  4. When you see the drag cursor (+), hold down CTRL and drag until all the destination cells are included. You can drag above or below the selected cell.

Linearly increasing values for a range of cells within the column

In certain text and numeric columns, you can fill cells with a series of linearly increasing values that increment upwards from the source cell or cells. Column types that do not support this operation are listed in the notes below.

For example, you want to fill the cells in the Year column with the values 1910, 1920, 1930, and so on.

  1. Type 1910 in the first row, and 1920 in the second row.
  2. Select the Year cell of the first and second rows.
  3. Position the mouse pointer on the drag fill handle at the bottom right corner of the selection.
  4. When you see the drag cursor (+), drag until all the destination cells are included. You can drag above or below the selected cell.

 Note   If the initial series contains a single value, you must hold down CTRL while dragging the fill handle to generate a series.

  Notes  

  • You can automatically fill more than one column in a single operation. The selected cells in each column will be used to fill the destination cells.
  • Columns with the following data types do not support filling with linearly increasing values:
    • Multiple lines of text
    • Choice
    • Lookup
    • Yes/No (check box)
    • Person or Group
    • Hyperlink or Picture
    • Calculated
  • If you try to fill a cell that is read-only or attempt to fill a cell with an invalid value, the fill operation will fail for that cell.
