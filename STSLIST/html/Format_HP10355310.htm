Format data

You can format the contents of a column if it supports multiple lines of text and rich text.

To enable support for rich text for a column:  

  1. Right-click in the column you want to enable for rich text and click Edit/Delete Column.
  2. Under Name and Type, select Multiple lines of text if it's not already selected.
  3. Under Additional Column Settings, select Rich text (Bold, italics, text alignment).
  4. Click OK.

Once those settings are made, you can format the text in one or more cells, or just a portion of the text in a cell:

  1. Select the cells or the text that you want to format.
  2. If the task pane is not open, in the ribbon, click the List tab, and then click Show Task Pane.
  3. Apply the format you want. You can change font settings, text and background color, and indentation and alignment. You can also organize text into a list of numbered or bulleted items.

  Notes  

  • You can clear rich text formatting by selecting text, then pressing CTRL+Space. However, CTRL+Space does not remove the alignment setting.
  • You cannot format read-only cells or the Total row.
  • Formatting does not apply to rows hidden as a result of a filter.
