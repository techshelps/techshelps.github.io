Make Datasheet view the default view

When you first create a list, a new "standard" view is defined for it by default. To change to Datasheet view, in the Ribbon, under List Tools, click List, and then click Datasheet View. However, if you want to make Datasheet view the default view for a list, you must first create a new Datasheet view of the list, then make that view the default view.

To create a new Datasheet view of a list and make it the default view:  
  1. In the ribbon, click the List tab, then click Create View.
  2. On the Create View page, under Choose a view format, click Datasheet View.
  3. On the Create Datasheet View page, type a name for the view. You may want to include the word "Datasheet" in the view name to help you remember that it is a Datasheet view.
  4. Select the Make this the default view check box.
  5. Check that the Create a Public View option is selected so other visitors can see this view.
  6. At the bottom of the page, click OK.
To make an existing Datasheet view the default view:  
  1. In the ribbon, click the List tab, and then click List Settings.
  2. On the List Settings page, under Views, click the name of the view that you want to make the default.
  3. On the Edit View page, in the Name section, select the Make this the default view check box.

     Note   If the view is already set as the default view, or if it is set as a personal view, you will not see the check box.
