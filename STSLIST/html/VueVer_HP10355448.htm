View versions of a document

If the document versions setting of a document library is set to store versions of documents, then you can view information on the different versions of each document in the library.

View document versions

  • Right-click the row header or any cell in the row corresponding to the document whose versions you want to view, point to Document, and then click Document Versions.

Change document version settings

  1. In the ribbon, click the Library tag, and then in the Settings group, click Library Settings.
  2. On the List Settings page, under General Settings, click Versioning settings.
  3. On the Document Library Versioning Settings page, under Document Version History, select the versioning scheme you want, and then click OK.
