Create an Access report

When you create a report in Microsoft Access with a Datasheet view, you create a linked table with the data in the Datasheet view and generate a report based on the data.

  1. In the ribbon, click the List tab, and then in the Datasheet group, click Show Task Pane.
  2. Click Report with Access.
  3. Choose to export the Datasheet to an existing database or a new database and click OK.

Access prompts you to specify the location of the database and creates a linked table that links to the data in SharePoint Foundation 2010 data. Then, it generates a simple report that displays all the fields and records in the linked table. For more information on reports in Access, search for "Create a report" in Access help.
