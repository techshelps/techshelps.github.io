Select a cell, row, column, or list

Select a single cell

  • Click the cell, or press the arrow keys to move to the cell.

     Note   If you click a cell that contains a hyperlink, you will navigate to the corresponding Web address. To select a cell containing a hyperlink, use one of the keyboard shortcuts.

Select the contents of a cell

  • Select the cell, click inside the cell, and then select the text in the cell.

Select a range of cells

  • Click the first cell of the range, and then drag to the last cell. Or, click the first cell in the range, and then hold down SHIFT and click the last cell in the range.

 Note   You can scroll to make the last cell visible.

Select a row or column

  • Click the row or column header.

Select multiple rows or columns

  • Click the first row or column header and then drag to the last row or column. Or, select the first row or column; then hold down SHIFT and select the last row or column.
