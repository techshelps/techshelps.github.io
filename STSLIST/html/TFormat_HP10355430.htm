Troubleshoot formatting data

I do not see the formatting buttons in the task pane.

The formatting does not apply to all rows in the column.

I copied text into a multiple line of text column, but the formatting does not work.

I get a message about a cell exceeding its maximum size.

I do not see the formatting buttons in the task pane.

To enable formatting of text, the column must support multiple lines of text as well as rich text.

To enable support for rich text for a column:  

  1. Right-click in the column you want to enable for rich text and click Edit/Delete Column.
  2. On the Change Columns page, in the Name and Type section, select Multiple lines of text if it's not already designated as the column type.
  3. In the Additional Column Settings section, select Rich text (Bold, italics, text alignment).
  4. Click OK.

Once those settings are changed, you can format the text in one or more cells, or just a portion of the text in a cell:

  1. In the ribbon, click the List tab, and then in the Datasheet group, click Show Task Pane.
  2. Select the cells or the text that you want to format.
  3. In the Task Pane, use the Rich Text tools to apply the format you want. You can change font settings, text and background color, and indentation and alignment. You can also organize text into a list of numbered or bulleted items.

The formatting does not apply to all rows in the column.

When you select an entire column and apply formatting, the Total row and rows that are hidden due to filtering do not get formatted.

 Top of Page

I copied text into a multiple line of text column, but the formatting does not work.

This could be because the formatting of the source text is not fully recognized by the list. Make sure the source text does not include formats that are not supported by the list.

To see what formats are supported, select a cell in a column that supports rich text formatting, and browse the buttons below the toolbar in the task pane.

 Top of Page

I get a message about a cell exceeding its maximum size.

A formatted cell contains text and information about the formatting. If the data and the formatting information exceed the maximum size of the cell, you will see this message. Try reducing the text or the number of formats applied to the text.
