Edit a hyperlink

This command is available only if the selected cell has a hyperlink.

  1. Right-click the cell that has a hyperlink, point to Hyperlink, then click Edit Hyperlink.
  2. In the Edit Hyperlink dialog box, type a valid URL in the Type the Web address text box. A valid URL must begin with a protocol, such as http:// or ftp://, and must not exceed 255 characters.
  3. Type the display text in the Type the description text box.
