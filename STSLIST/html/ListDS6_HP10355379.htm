Reconvert a determined character string

 Note   This command is available only if support for Japanese, Simplified Chinese or Traditional Chinese Input Method Editor is enabled on your operation system or through Microsoft Office Language Settings.

You can reconvert a character string that has already been input and converted.

  1. Place the insertion point at the beginning or in the middle of the characters you want to reconvert.
  2. Right-click the cell, and then click Reconvert on the shortcut menu.
  3. Click the desired character, and press ENTER twice.
