PRODUCT

Multiplies all the numbers given as arguments and returns the product.

Syntax

PRODUCT(number1,number2,...)

Number1, number2, ...   are 1 to 30 numbers that you want to multiply.

Remarks

  • Arguments that are numbers, logical values, or text representations of numbers are counted; arguments that are error values or text that cannot be translated into numbers cause errors.
  • If an argument is a column reference, only numbers are counted. Empty arguments, logical values, text, or error values are ignored.

Example

Col1Col2Col3Formula Description (Result)
51530=PRODUCT([Col1], [Col2], [Col3]) Multiplies the numbers (2250)
51530=PRODUCT([Col1], [Col2], [Col3],2) Multiplies the numbers and 2 (4500)
