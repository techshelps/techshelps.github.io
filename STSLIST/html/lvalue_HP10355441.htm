VALUE

Converts a text string that represents a number to a number.

Syntax

VALUE(text)

Text   is the text enclosed in quotation marks or a column reference containing the text you want to convert.

Remarks

  • Text can be in any of the constant number, date, or time formats. If text is not in one of these formats, VALUE returns the #VALUE! error value.

Example

Formula Description (Result)
=VALUE("$1,000") Number equivalent of the string (1000)
=VALUE("16:48:00")-VALUE("12:00:00") The serial number equivalent to 4 hours and 48 minutes, which is "16:48:00"-"12:00:00" (0.2 or 4:48)
