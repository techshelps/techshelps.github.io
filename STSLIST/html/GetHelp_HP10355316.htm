Get help in Access Web Datasheet

  • To get help on Access Web Datasheet, right-click any cell inside the Datasheet view, and click Help.

     Important   The Access Web Datasheet Help topics only cover functionality that is available in Datasheet view. For general assistance with SharePoint lists, click Help ().

  • To get help on the SharePoint page containing the Datasheet view, click Help ().
  • To get help when a List View Web Part is displayed in Datasheet view, click the Web Part menu, and then click Help. The Help topics cover Web Part Pages, Web Part properties, and Web Part connections.

  Notes  

  • Several dialog boxes and status bar messages include links to context-sensitive help information. You can also click Help () on the task pane toolbar.
  • If the Datasheet view is in focus, pressing F1 will display help on Datasheet view. Otherwise, it will display Internet Explorer Help.
