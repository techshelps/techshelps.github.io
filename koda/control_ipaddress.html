IPAddress

TAIPAddress

Creates an IPAddress control on the form.

Properties:
Left,TopPosition of control relative to it's parent.
Width,HeightSize of control in pixels.
NameUnique name of control, this defines the name of variable where control handle will be kept.
TextIP address as text here.
VisibleThis is True by default. Set it to False to generate hidden control.

Remarks:

None.
