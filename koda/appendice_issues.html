Known issues

Here the list of known issues, that very unlikely will be fixed.

Windows 98 SE

  • Code, that keep only one instance of Koda, don't work.
  • Koda can't open executed *.exe (like itself) when trying to add it's icons.

Windows Vista/Seven

  • Koda can load 16-bit icon libraries, but icons from them will not be visible in the script, since it's support was removed in Vista. So, better convert them into 32-bit ones. This can be done in the ImageList editor.
