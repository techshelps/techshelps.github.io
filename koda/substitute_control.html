Substitute controls

Five controls in Koda realization are substitute controls - Menu, ContextMenu, TrayMenu, Dummy and ImageList.

Because, for example, a menu is not actually a control, Koda places a small square control onto form. Selecting this control enable you to change menu's properties, many of which defines menu structure. This “empty control” itself will not be visible on the result form, only the resulting menu.

Having those controls as separate “pseudocontrol” have some advantages: you can have two or more menus on one form, but only one “attached” to main form will be generated; you can attach context menu to one or other control etc.
