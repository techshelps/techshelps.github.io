Color editor

Color properties can be chosen in two ways - selecting one of predefined colors from combobox or choosing color from standard color dialog (called by a doubleclick on the property).

Predefined color names are in the tables below. Some colors are system colors, that is, colors that the user can change from Windows Control Panel.

In near future is planned support to those colors via include, so your GUI controls can have colors set in the color theme by the user.
