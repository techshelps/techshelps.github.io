Import

Under this menu item are placed file converters from different formats to Koda own form format.

Currently, only one AutoIt GUI item is available. It allows to convert any AutoIt GUI script to Koda format.

Note: this is not perfect yet, and was basically made for old GUI scripts conversion, or those ones for what kxf file is missing. It does not support multiple GUIs in one file. So, if you have more then one, copy your GUI part manually to a new file and try to convert it.
