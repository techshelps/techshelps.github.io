Page Tab

Use the Page tab to change the printing preferences for Solid PDF Creator printer.

Your options are the following:

  • Page Size – Select the predefined paper size for the PDF file output. If you want to add your own page size, click the Custom Forms button. To edit the custom page size, select the size in the list and click the Custom Forms button.

  • Page Orientation – Select either Portrait or Landscape page orientation.

  • Printer Margins – Allows you to specify print margins. You can also click Get Margins From Printer and select one of the printers you have installed in your Windows to use its print margins.
