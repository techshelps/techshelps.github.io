Product Overview

Solid PDF Creator  is a utility for creating portable document format (PDF) files from other formats. PDF is a widely accepted document standard for creating universally readable documents that maintains document integrity. You can also add password security to PDF files and compress files to smaller sizes.  Solid PDF Creator  now has an option to save archive PDFs as PDF/A files compliant with ISO PDF/A-1b standard.

To evaluate the program, simply wait for the  Evaluate button to activate, then click that button. To remove the  Evaluate button and to be kept apprised of updates to the program, click  Buy Now to purchase  Solid PDF Creator.

System Requirements

  • Windows Vista® (32 bit), 2003 or XP

  • Microsoft® Office (any version) required to create PDF files from .doc, .xls, .ppt, ...

  • Single product license for single system. Site licenses are available.
