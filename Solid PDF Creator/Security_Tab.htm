Security Tab

The security setting you specify  will be used as predefined in Solid PDF Creator unless you change them for specific documents as described in  Selecting Create PDF Options .

You can specify the security settings in the following way:

  • Encryption Level – specify any value if you want to apply password protection to your PDF file. The encryption level determines the type of security you can choose. Note that higher encryption levels only work with later versions of Adobe Reader. If you do not select an encryption level,  High will be selected when you choose a password option.

  • Require user password to open the document – A user password restricts who can open and view the document. To set this type of password, select the check box to enable the password entry boxes. Then enter the password you want to use in the two boxes shown. Make sure you use the same password in both boxes.

    Select from the following options:

  • Allow printing  – to allow the user to print the PDF.

  • Allow editing of comments – to allow the users to make changes to the PDF document.

  • Allow content copying or extraction – to enable copying of content, including text and images.

  • Allow adding or editing of comments  – to enable adding or editing of comments.
