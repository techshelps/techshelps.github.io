Starting Solid PDF Creator

There are two ways to create PDF files with Solid PDF Creator.

1.  To start the program, first open your original document using any Windows program. When you are ready to create the PDF, select  File|Print, then select Solid PDF Creator as your printer.

  After you click OK, the Create PDF dialog box will open.

2. You can drag and drop your document onto the Solid PDF Creator icon on the desktop to create your PDF file.

Selecting Create PDF Options

The Create PDF dialog box allows you to select different saving options and start the PDF creation process.

The following choices are available:

  • The Prompt for Document Properties – Select this check box to specify the following parameters for the specific PDF file you are creating:

    • Document Information –  to fill in document information such as title, subject, author, and keywords. See Document Information Tab section for more details.

    • Security –  to apply the document security settings; See the Security Tab section for detailed information.

    • View – to define a view including page layout, window position and magnification level that appears when a user opens the document in a PDF viewer.

    • Optimization – to control the compression and font (embedding) settings for the PDF output file.  You will find the recommendations on optimizing in the Optimization Tab section.

If the Prompt for Document Properties is not selected, the prompt will not display and the PDF file will be created with the settings that are currently specified for Solid PDF Creator.  To configure Solid PDF Creator  behavior  to your specifications please see Configuring Solid PDF Creator Using the Preferences or Properties Dialog.

  • Open converted file after completion –  Select this check box to have the PDF automatically open after conversion. The program in which it will open depends upon your computer setup - either Acrobat Reader or Adobe Acrobat (if that program is on your computer). If you choose not to have Solid PDF Creator open the viewer, you can find the PDF file in the folder you selected using Save in drop-down menu and open it at another time. 

Creating PDF files

In the Create PDF dialog box first make sure you have assigned a File Name and a location where the file will be saved using the Save In drop-down menu. Then click the Create button.  If you have selected  the Prompt for Document Properties check box a Properties dialog will display requesting the appropriate information.  Finally, a dialog will display during the actual conversion process to show the progress of the conversion.  

The Status shows the creation processing results. Click on  Details to get more information about the creation process.
