Configuring Solid PDF Creator Using the Preferences or Properties Dialog

Accessing Preferences or Properties dialog box

The Preferences/Properties dialog is accessed outside of Solid PDF Creator. There are two ways of accessing this dialog:

  • Global Settings. Go to Windows  Printers & Faxes folder and right-click on  Solid PDF Creator. Select  Printing Preferences... . These settings will be used in Solid PDF Creator unless you change them for specific documents using the Document-Specific Settings (see below) or  Create PDF Options on the main screen.

  • Document-Specific Settings. After you select File|Print to print your document, click the Properties button.

The parameters on the Save, Compliance, Document Information, Security, Page , View and Optimization tabs  can be specified to configure Solid PDF Creator.
