Compliance Tab

The compliance information you specify will be used in Solid PDF Creator unless you change it for specific documents as described in  Selecting Create PDF Options . This tab provides an option to save archive PDF files as PDF/A files compliant with ISO PDF/A-1b standard.  It also allows you to specify the behavior you wish to have (warning and continue creation or cancel creation) if the file cannot be saved as PDF/A.

 

Selections include the following:

  • PDF – Select this radio button to create PDF without compliancy restrictions.

  • PDF/A-1b (ISO 19005-1 compliant long-term archiving standard) – Select this radio button to create the PDF file compliant with ISO PDF/A-1b standard.

  •  When not compliant –  Specify the desired program behavior in the situation when Solid PDF Creator detects that the created PDF will not meet the selected ISO standard.  You can choose:

    • Cancel PDF creation –  to stop creating. File creation will be cancelled and no warning message is displayed.

    • Display warning with option to continue – to display a  message asking you to choose between canceling the conversion or going ahead with the creation of a non-compliant file.
