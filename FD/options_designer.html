Designer options

Show grid - show grid on the designing form.

Snap to grid - when control created this will be automatically aligned to closest grid "line".

Grid size - horisontal and vertical grid size. Default is 8 x 8.

Advanced event editor - when working with "OnEvent" mode you may prefer this advanced mode.

Reflect form style changes - in this mode changes in form styles will visible immediately.

Don't show unsafe styles - setting of some styles can screw you form and you can't return it to previous state until form will be reloaded. This option is on by default and this is recommended setting, change it to you own risk.
