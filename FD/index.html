Welcome to the Koda help file.

Koda save it's form definition in the standard XML file with KXF extension. It's include all allowed control properties..... Interface is include 3 parts: main form, property inspector and form list.

Designer features

At the moment designer have not Undo function, so please be careful and more frequently save your work!

Multiple controls handling

You can cut, copy, paste and delete multiple controls at once.

When you select multiple controls, you can change some properties, that common for all selected controls, at once.

Container controls

Two controls in Koda realization are container controls - Tab and Group. This meaning that any other control placed onto them will have it as parent. This result into some features:

- subcontrols will move together with container control;
- position of subcontrols is relative to container control;
- if you want to place controls that was created outside container control onto container control, you have to copy/cut those controls and paste them onto container control, drag'n'drop will not work in this case;
