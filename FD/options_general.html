General

Run method - method for running created form. This can be:
Associated - when your have Autoit correctly installed or, at least have AU3 extension associated, this is default method
Manual - in this case you can select path to Autoit executable.

Open form after run - group of options, controlling way after program start. This can be
New - new form will be created just after start.
Last - last saved form will be opened
Manual - you can set form that will be always open after start.

Associate group - here two buttons. Associate button will associate KXF extension in the system, Unassociate - remove association.

Create backup - this checkbox allowed to create backup file every time you save form. Backup have the same name as form with additional BAK extension appended.
