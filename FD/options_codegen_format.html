Formatting options

Indent - set of code indent options

Indent by - character for indent, can be Tab or Space. You also can set, how many chosen characters will be represent one indent step.

Initial indent - whole generated code will be indented by given number characters, chosen above.

Scope - the scope in which code will be generated. This can be Default (no additional scope keyword will be generated), Global and Local (for use GUI in functions).

Data separator - char that will be used as separator for List, Listview and other controls. Useful when default "|" pipe char can't be used.

Strip "A" prefix from default name - this tell codegen to remove A-prefix that represent Autoit controls in Delphi object model. This is just cosmetic option.
