Language

This is one of most obvious page. In list you can select desired language and click "Apply" button. Under list in green showed current active language.

If you want to create you own language file, choose language on what new one will be created (don't forget to click "Apply"). Then click "Template" button - new language XML file will be created under Language folder. It will have all settings and already translated messages from base file plus new strings that may be added since previous release. Also, some strings can be removed or renamed.

Translate strings and rename file to lang_xxx.xml, where xxx - 3-letter code of you language (eng, rus, fra etc). If you need to change flag icon, please download language kit from here.
