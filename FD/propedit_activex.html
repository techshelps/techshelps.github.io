ActiveX objects

Warning! Save you form before trying to choose objects! Though Koda will try to handle errors caused by ActiveX components, this realm in many cases unpredictable, and some components can freeze or crash Koda, so it will be too late to save your work.



ActiveX objects browser can take some time when opening for scanning your registry for installed components, that have IDispatch interface and can be embedded into GUI.

When you select component in list, it's path, ID and GUID is shown below the list. Koda at once will try to create this object in preview area. You have to control status line to see if object was created successfully or not.
