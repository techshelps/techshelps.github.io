ListView items editor





Editor is pretty similar to TreeView editor, but here you can only create 2-level tree, where first level is listview captions, second - listview subitems.

Click "Item" button to create new listview item. Click "Subitem" button to create subitem under selected item. Click "Delete" button to delete item or subitem.

On right side you can change item properties. At the moment this is only caption. Items are change immediately while you change text in the input box.
