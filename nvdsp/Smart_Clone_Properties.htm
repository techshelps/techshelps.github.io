Smart Clone Properties

The Smart Clone Properties window lets you specify a section of a display (or Mosaic virtual display) to be cloned.

Smart Clone Properties is available only with NVIDIA Quadro 2000 and later products, and NVIDIA Quadro K2000 and later products.

NOTE: If you change your display DPI settings from the Windows Control Panel->Display page, the Smart Clone Properties window may not resize properly. If this happens, restart your computer and then reopen the Smart Clone Properties window.

Clone Source

The display icon represents the source clone display for the smart clone pair. If multiple smart clone pairs have been set up, then each pair is represented by a display icon. Select the icon for the clone pair that you want to configure.

Pixel Perfect Clone

Select this option to clone an area of the source display. A box appears on the source display that indicates the region that will be cloned. You can drag the box to reposition it over the region that you want cloned. The region size is equal to the size of the target display, and cannot be changed.

Allow panning:  Select this setting for pan and scan clone functionality.

Scaled Clone

Select this option if you want to specify the size and position of the region on the source display to clone.

Specify the size and location of the region by either clicking and then dragging the grab points of the region box on the desktop, or by entering values into the X/Y/Width/Height boxes. The X and Y boxes determine the origin (top left corner) of the region.

  • The minimum allowable region size is 640x480 pixels.

  • The maximum allowable region size is the size of the source display.

  • If the region size is smaller than the target display, it is scaled up to fill the display.

  • If the region size is larger than the target display, it is scaled down to fit within the display.

Default:  When clicked, the X/Y/Width/Height boxes are set to the size of the source display, which corresponds to the maximum allowable region size.
