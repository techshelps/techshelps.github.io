To lock a clone region

You can specify a section of the source display to be cloned and locked. Once locked, you cannot dynamically pan the area on the second display.

These instructions assume you have enabled Smart Clone on your displays.

  1. From the NVIDIA Control Panel navigation tree pane, under Display, click Set up multiple displays to open the page.

  2. Right-click the display icon box of any smart clone group, then click Smart clone properties.

    The Smart clone properties window opens, and shows icons of the clone source displays for all clone pairs in your system.

  3. Under Clone source, select the clone source display for the clone pair that you want to lock.

    4. A box appears on the source display that indicates the region that will be cloned.

  4. Configure your Smart Clone properties.

    By default, both pixel perfect clone and Allow panning are selected, which clones a section of the source display which you can pan on the target display.  To modify the size of the clone region and lock it, do the following:

    1. Select Scaled clone.

    2. Specify the size and location of the region by either entering values into the X/Y/Width/Height boxes, or by clicking and then dragging the grab points of the region box on the desktop.

  5. Repeat steps 3 through 5 for any other smart clone pair that you want to lock.

  6. When finished, click Apply.
