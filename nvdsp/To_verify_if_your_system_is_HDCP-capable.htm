To verify if your system is HDCP-capable

  1. From the NVIDIA Control Panel navigation tree pane, under Display, click View HDCP Status to open the associated page.

  2. Select the icon representing the display that you want to verify is HDCP-capable.

    The page shows one of three possible results:

    • This display supports HDCP.

    • This display does not support HDCP as configured.

    • This display does not support HDCP.

  3. Be sure to notice any error or alert messages that explain conditions that prevent or could prevent your system from being HDCP-capable.

  4. If you add or remove a display, then exit and then re-enter the View HDCP Status page so that is will update to reflect the status of the new configuration.
