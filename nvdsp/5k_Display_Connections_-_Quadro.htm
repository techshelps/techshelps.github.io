Connecting 5K Monitors to Quadro GPUs

This section provides guidance on how to properly connect one or more 5K monitors to the supported Quadro GPUs.

 

Basic Requirements

  • Each 5K monitor requires two DisplayPort connections in order to properly view the full 5k resolution.

  • The following Quadro GPUs can be configured to support 5K monitors, provided the product has a DisplayPort output and supports DisplayPort 1.2:

    • Quadro K2000, K2200, K4200, K5000, K5200, K6000 and later

    • NVS 510
