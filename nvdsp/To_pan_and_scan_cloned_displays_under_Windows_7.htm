To pan and scan cloned displays (Windows 7 and later)

Pan and scan clone mode is available only with NVIDIA Quadro 6000/5000/4000/2000/600 and later workstation solutions.

  1. Make sure you have securely connected the additional displays you want to use to your computer and enabled them.

    Displays which you intend to enable in pan and scan clone mode must be connected to the same GPU and in extended mode first, and any other displays which are enabled must be in extended mode.   

    If your displays are not enabled, enable them as follows:

    1. Verify that your displays are plugged in and turned on.

    2. From the NVIDIA Control Panel navigation tree pane, under Display, click Set up multiple displays to open the page.

    3. Check or clear the check boxes to select the displays that you want to use.

    4. In non-SLI modes, you can select up to two displays per GPU.

    5. In SLI mode, you can select up to two displays connected to GPUs within the same SLI group. Under 4-way SLI mode, only GPUs that have two display connectors can be used to drive displays. Typically, display connectors lined up on the same slot position are connected to the same GPU.

    6. Click Apply to activate the new display configuration.

  2. From the NVIDIA Control Panel navigation tree pane, under Display, click Set up multiple displays to open the page.

  3. If any displays are in Clone mode, right-click the display icon box of the Clone group, click Extend, then click Apply.

  4. Right-click the display icon for the display that you want to use as the source display for pan and scan clone.

  5. From the pop-up menu click Smart clone with, then click the display you want to use as the target clone display.

  6. Click Apply.

    7. The icon numbers corresponding to your displays appear in the same box icon.

  7. Repeat these steps for any other pair of displays that you want to set up for pan and scan clone, depending on the capabilities of the GPU.

  1. Note: If you are using SLI technology, then full SLI acceleration will not be achieved on displays in clone mode.

    Note: If any one of your displays isn't showing an image, make sure to repeat steps 1. and 2. If that doesn't work, restart your computer and repeat the above steps.
