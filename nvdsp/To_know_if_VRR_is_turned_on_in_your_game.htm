To know if G-SYNC is enabled for your game

To verify whether G-SYNC is enabled for your game, click Display from the NVIDIA Control Panel menu bar, then select Show G-SYNC Visual Indicator.

The indicator will appear on your screen to let you know whether G-SYNC is turned on.

If G-SYNC is not turned on, see How to set up G-SYNC for instructions on how to turn G-SYNC on if you want to use it.
