To lock or pan a Mosaic clone region

After you have cloned a Mosaic virtual display, you can either pan the source display, or specify a section of the source display to be locked. Once locked, you cannot dynamically pan the area on the second display.

These instructions assume you have enabled Smart Clone on a Mosaic virtual display.

  1. From the NVIDIA Control Panel navigation tree pane, under Display, click Set up multiple displays to open the page.

  2. Right-click the clone group box and select Smart clone properties.

    The Smart Clone Properties window opens, and shows icons of the clone source displays for all Smart Clone pairs in your system.

  3. Under Clone source, select the clone source display for the clone pair that you want to lock.

    4. A box appears on the source display that indicates the region that will be cloned.

  4. Configure your Smart Clone properties:

    • To create a clone display without any scaling, select the pixel perfect clone option.

    This option clones a centered area of the source display. The area is equal to the size of the target display.  

    Select Allow panning if you want to pan the Mosaic display.  

    • To create a clone display that is scaled, select the Scaled clone option.

    If you selected Scaled clone, then specify the size and location of the region by either entering values into the X/Y/Width/Height boxes, or by clicking and then dragging the grab points of the region box on the desktop.

  5. When finished, click Apply.
