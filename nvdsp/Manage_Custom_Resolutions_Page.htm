Customize

Note: Use this page only if you are an advanced user and are knowledgeable about display timings and their effects.

This page allows you to apply custom resolutions to your display. This is useful if your display requires a resolution that Windows does not offer.

From this page you can:

  • Create a custom resolution.

  • Click Create Custom Resolution and then use the Create Custom Resolutions dialog box to specify your custom timings. See Custom Resolutions Dialog Box for a description of each option.

    The created custom resolution appears in the Resolutions box, is checked, and also appears in the Resolution box on the Change Resolution page.

  • Edit an existing custom resolution.

  • From the Resolutions list box, click the custom resolution that you want to edit and then click the edit icon or press Enter on your keyboard to open the Create Custom Resolutions dialog box where you can edit the setting.  See Custom Resolutions Dialog Box for a description of each option.

  • Delete an existing custom resolution.

  • From the Resolutions list box, click the custom resolution that you want to delete and then click the delete icon or press Delete from your keyboard.

    Note: You cannot delete a custom resolution if it is currently applied.

  • Export an existing custom resolution

    • From the Resolutions list box, click the file icon by the custom resolution that you want to export, then use the Export Settings dialog to save the settings to a file in XML format.  

  • Enable resolutions not exposed by the display.

  • Checking this check box clears the Windows Hide modes that this monitor cannot display check box. Any unsupported resolutions listed in the Resolution box become available for selection.
