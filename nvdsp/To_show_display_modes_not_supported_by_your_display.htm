To show display modes not supported by your display

  1. From the NVIDIA Control Panel navigation tree pane, under Display, click Change resolution to open the Change Resolution page.

  2. If applicable, select the display for which you want to expose the additional display modes.

  3. Click Customize.

  4. Check the Enable resolutions not exposed by the display check box.

    The additional modes appear as available in the Resolutions box.

    WARNING:  This check box enables you to select display modes that cannot be correctly supported by this display. However, doing so may lead to an unusable display and/or damaged hardware.  When this check box is selected, unsupported displays will appear as selectable values.

  5. To use the additional modes, check the check box for any of the modes that you want to use, then click OK.

    The modes now appear in the Change Resolution page.

  6. In the Change Resolution page, select the resolution you want to use, then click Apply.
