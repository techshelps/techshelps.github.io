Connecting 5K Monitors to GeForce GPUs

This section provides guidance on how to properly connect one or more 5K monitors to supported GeForce GPUs.

 

Basic Requirements

  • Each 5K monitor requires two DisplayPort connections in order to properly view the full 5k resolution.

  • The following GeForce GPUs can be configured to support 5K monitors, provided the product has a DisplayPort output and supports DisplayPort 1.2:

    • GeForce 600, 700, and TITAN series GPUs based on the Kepler and Maxwell architecture

    • GeForce 900 series GPU and later
