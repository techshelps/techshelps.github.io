To export a custom resolution you created

  1. From the NVIDIA Control Panel navigation tree pane, under Display, click Change resolution.

  2. Click the image that represents the display you want to affect.

  3. Click Customize, then from the Customize dialog box, click the file icon by the custom resolution that you want to export.

  4. Use the Export Settings dialog to save the settings to a file in XML format.
