HTML Designer

The HTML Designer includes tools that enable you to create and edit HTML documents in two unique views:

  • Design view - where you can edit HTML pages graphically, insert elements, use drag-and-drop to position and size them, and see the results instantly.
  • HTML view - where you can edit HTML markup directly, use pop-up lists to insert elements and properties as you edit, and format and debug code.
