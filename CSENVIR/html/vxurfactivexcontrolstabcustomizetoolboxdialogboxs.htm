COM Controls, Customize Toolbox Dialog Box

This tab of the Customize Toolbox dialog box displays a list of all COM controls registered on your machine and allows you to change which of them are displayed in the Toolbox window. To sort the list of controls, click on any column heading. You can open the Customize Toolbox dialog box from the Tools menu.

Check box

When checked, indicates that the integrated development environment (IDE) has confirmed the entry for the control in your machine's registry and has added it to the Toolbox window, or will add it when you click OK. If this check box is cleared, the control does not currently appear in the Toolbox window or will be removed when you click OK.

Name

Displays the name of the control.

Path

Displays the full path to the control. You can sort by this column and then look for the installation path to identify default controls that shipped with the product.

Library

Displays the library that contains the control.

Last Modified

Displays the date when the control was last modified.

Control description

Displays a control's development language and version, and its Toolbox icon.

Browse

Displays the Browse dialog box, so that you can select COM controls (.ocx files) or .dll files stored on your machine or network that are not yet registered on your machine. When you select a control, it is registered and displayed on the Name, Path, and Last Modified list with its check box selected.

Reset

Restores the Toolbox to its default state and refreshes the current tab. Default controls appear with their check boxes selected.
