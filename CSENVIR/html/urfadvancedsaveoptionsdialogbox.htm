Advanced Save Options Dialog Box

Use this dialog box to specify non-default language encoding options and to register your line-ending character preferences for an open file. To access this dialog box, open a file and choose Advanced Save Options from the File menu.

Encoding

Use this list box to apply a new language encoding scheme to the target file.
