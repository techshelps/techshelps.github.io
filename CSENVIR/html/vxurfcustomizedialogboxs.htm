Customize Dialog Box

The Customize dialog box allows you to create custom toolbars, add and remove items from existing toolbars, and change the appearance of the toolbars displayed. You can access this dialog box by selecting Toolbars from the View menu. The following topics provide detailed information about tasks that can be performed in the Customize dialog box.
