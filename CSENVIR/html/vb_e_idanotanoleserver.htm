File <name> cannot be added to the Toolbox.

The file cannot be added to the Toolbox. This error message occurs when the file is:

  • Not an .ocx or .dll.
  • Corrupted.
  • Not a valid ActiveX control.
  • Missing or contains invalid version information.

To correct this error

  • Verify that the file is an ActiveX control and try again.

    —or—

  • Re-install the product that installed the file.

    Note   The product that installed the file may not be the current program.
