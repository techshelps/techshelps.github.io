Clipboard Ring, Toolbox Window

The Clipboard Ring tab and General tab appear by default whenever the Toolbox window opens. The Clipboard Ring tab stores the last twelve items that have been added to the system Clipboard Ring using Cut or Copy. These items can then be dragged and dropped from this tab to an editor or designer. The Toolbox is available from the View menu.

To cycle through the contents of the Clipboard Ring tab, press CTRL+SHIFT+V while working in an editor. This pastes the most recent item from the Clipboard Ring tab into the active file at the insertion point. Keep pressing CTRL+SHIFT+V to cycle through the Clipboard Ring until the desired item is found.

Tip   You can add custom tabs and tools to the Toolbox, and rename, remove, and organize tools and tabs.
