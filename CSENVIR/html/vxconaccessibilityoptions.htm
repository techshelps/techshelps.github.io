Accessibility Options

The integrated development environment contains features that make it easier for people who have low vision to read and for people who have limited dexterity to write.

To automate the entry of text and graphics

  1. Select Options from the Tools menu.
  2. Choose General from the Environment folder.
  3. Select Enable autocompletion.
  4. Choose OK.

To improve toolbar usability and accessibility, you can enlarge or add text to toolbar buttons.

To display Large icons on toolbars

  1. Choose Customize Toolbars from the Tools menu.
  2. In the Customize dialog box, select the Options tab.
  3. Under Other, select the Large icons option.

To assign text to toolbar buttons

  1. Choose Customize Toolbars from the Tools menu.
  2. In the Customize dialog box, select the Toolbars tab.
  3. Select and enable the toolbar containing the button image you want to change.
  4. Leaving the Customize dialog box open, click on the button you want to change.

    A black square or rectangle indicates that the button is selected for modification.

  5. Select the Commands tab.
  6. Choose Modify Selection.
  7. Choose Image and Text.

To modify the button's displayed text

  1. Re-select Modify Selection.
  2. Adjacent to Name, insert a new caption for the selected button.
