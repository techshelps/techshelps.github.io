Browse With Dialog Box

You can preview a document in a non-default browser by selecting a browser-compliant document in Project Explorer and then choosing Browse With from the File menu. Use the Open With dialog box to edit a document. The browser is the default browser until you change this setting.

Browser List

Lists the available browsers. Select the browser with which to browse the document, or choose Add to create a new program entry.

Browse

Choose Browse to view the a document in the browser selected from the Browser List.

Add

Select to use the Add Program dialog box to add a new browser to Browser List. You can either type the path to a program in Program Name or navigate to its location using Browse. Type the name that you wish to associate with the program under Friendly Name. This name will appear in the Browser List.

Remove

To remove a program from the Browser List, select the program and click Remove.

Set as Default

To specify a new default browser, select a program from the Browser List and click Set as Default.

Size of Browse Window

As an alternative to the default browser window size, you can choose either 640 x 480, 800 x 600, or 1024 x 768 from the drop-down list.
