The Image Height must be a positive integer greater than zero.

The Image Height property has a value that is not a positive number greater than zero.

To correct this error

  1. Open this HTML page for editing in Design view of the HTML Designer.
  2. Right-click on the image and select Properties from the shortcut menu.
  3. The Properties window displays the properties of the selected image. Scroll to the Image Height property and enter a positive integer greater than zero.

    The HTML markup of the selected <IMG> element is modified. Save these changes to your document.
