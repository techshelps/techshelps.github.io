General, HTML Designer, Options Dialog Box

Select the General page under HTML Designer in the Options dialog box to adjust the default view in which HTML, Active Server, and Web Form pages open in the HTML Designer.

Note   These settings supplement the settings on the pages listed under Text Editor, HTML. You can access the Options dialog box from the Tools menu.

Start HTML pages in

HTML View

When this option is selected, HTML pages open in HTML view of the HTML Designer, so that you can begin editing their markup immediately.

Design View

When this option is selected, HTML pages open in the Design view of the HTML Designer, so that you can position elements and see the effects of font and style settings. This is the default view.

Set Active Server pages in

HTML View

When this option is selected, Active Server (.asp, .aspx) pages open in HTML view. This is the default view.

Design View

When this option is selected, Active Server pages open in Design view.

Start Web Form pages in

HTML View

When this option is selected, Web Form pages open in HTML view.

Design View

When this option is selected, Web Form pages open in Design view. This is the default view.
