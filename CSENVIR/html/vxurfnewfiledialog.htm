New File Dialog Box

Use this dialog box to create a new file. Files created using this dialog box are displayed in Project Explorer under the Miscellaneous Files node.

Note   Files created using the New File dialog box will not be added to any open projects. To open this dialog box select New from the File menu and then choose File.

Categories

Displays categories of files from which you can choose a file Template. Select a category and then select a file template from the list on the right.

Note   If only one category exists, this dialog box will contain only one frame; Templates (see below).

Templates

After choosing a folder from Categories, select a file type from Templates. A brief description of the selected file template appears beneath Categories.

Displays items in Templates as large icons.

Displays items in Templates as small icons.
