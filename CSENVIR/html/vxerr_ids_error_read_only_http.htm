A Web page accessed via HTTP cannot be modified directly. Save the desired HTML source code as a new file, and then edit this new file.

To modify HTML source code from a Web page loaded into the editor via the HTTP protocol, you must first save a local copy of the page. You can modify this local copy as needed.

Note   The HTML protocol does not permit a Web page to be opened and then overwritten with a modified copy of itself on the same active Web server.

To correct this error

  1. Select Save As from the File menu to save a local copy of the Web page.
  2. Open and edit this copy of the Web page.
