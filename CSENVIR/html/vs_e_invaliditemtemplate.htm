Template <name> is not valid for the selected project.

This error generally occurs when an unknown template name has been entered for the New File command, File.NewFile, or for the Add New Solution Item command, File.AddNewSolutionItem.

To correct this error

  • Re-enter the command and its arguments.

    —or—

  • Check the “New File Command” or “Add New Solution Item Command” topics for the correct syntax.
