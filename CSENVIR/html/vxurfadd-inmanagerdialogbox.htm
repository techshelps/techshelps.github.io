Add-In Manager Dialog Box

The Add-In Manager dialog box allows you to view a list of registered Add-ins in the IDE, and allows you to load or unload them in the IDE. You can specify the Add-in's behavior and command line options, if applicable. You can access the Add-In Manager dialog box on the Tools menu.

Available Add-ins

A list of all Add-in components that are registered in the IDE.

Startup

Determines whether the Add-in loades at startup.

Command Line

Determines whether the Add-in loads for command-line build operations.

Description

A short description of the add-in created by its author. This might include implementation instructions, caveats, usage tips, support, version and licensing information, and so forth.
