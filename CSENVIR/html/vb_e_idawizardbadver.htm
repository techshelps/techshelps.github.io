Wizard file <name> format is not recognized.

The wizard cannot be opened. This error is generally caused by a corrupted wizard file or a wizard that does not match the version number of the application.

To correct this error

  • Verify that the wizard file has the same version number as the version of the application listed as the value of VSWIZARD, such as VSWIZARD 7.0.

    —or—

  • Re-install the application to repair the corrupted file.
