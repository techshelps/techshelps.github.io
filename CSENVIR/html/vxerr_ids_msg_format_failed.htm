Could not reformat the document. The original format was restored.

The HTML or XML in the document is not well formed, or you have attempted to make types of style changes to an entire page that can only be applied to some portions of the page.

To correct this error

  • Check the syntax of the CSS styles and style attributes used on your page.
  • Check the syntax of the HTML elements to which these styles apply.
  • Make sure that all CSS style attributes are applied to elements that permit those attributes.
