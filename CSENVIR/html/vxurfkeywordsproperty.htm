Setting the keywords Property

Provides keywords that are inserted into the meta name=keywords tag in the <HEAD></HEAD> of your active Web page.

To edit keywords for a Web page

  1. Select the page in the HTML Designer.
  2. From the drop-down menu at the top of the Properties Window, select Document.
  3. Enter and edit values for the keywords property.

For example, if you type sports, recreation, outdoor as values for the keywords property, the following tag is inserted into the <HEAD></HEAD> section of the page.:

<meta name=keywords content="sports, recreation, outdoor">

Search keywords are used in queries by internet search engines to identify the content of your page, so that it can be included in or excluded from their result lists.
