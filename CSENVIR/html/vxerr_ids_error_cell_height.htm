The Cell Height must be an integer greater than zero.

The Height property of a cell in an HTML table has a value that is not a positive number greater than zero.

To correct this error

  1. Open the HTML page for editing in Design view of the HTML Designer.
  2. From the View menu, select Other Windows and choose Document Outline. The Document Outline window opens in HTML Outline view.
  3. In the Document Outline window, click the <BODY> node to display all HTML elements on the page. You can click any element to select it in Design view. Click the <TD> node for the table cell that needs editing.
  4. The Properties window displays the properties of the selected cell. Scroll to the Height property and enter a value that is a positive number greater than zero.

    The HTML markup of the selected <TD> element is modified. Save these changes to your document.
