IMG Property Page

After you insert an image, you can change its properties. To display the IMG Property Pages, select an image in Design View or place the insertion point within the <IMG> tag, and then choose Property Pages from the View menu.

Image source

Specifies the name, including the full path, of a graphics file to insert as the image. Click the adjacent ellipse (...) button to open the Select Image dialog box and browse to an image.

Alternate text

Specifies text that the browser will display if the graphic is not available. In some browsers, such as Internet Explorer, the alternate text is also displayed in a ToolTip when the user holds the mouse pointer over the graphic.

Appearance

Alignment

Specifies how the image is aligned with the text around it.

Note   Absolute image positioning attributes take into account the font size of adjacent lines of text. For example, assigning an image the positioning value valign=middle will align the midpoint of the image with the baseline of the current line of text. The value valign=absmiddle will align the midpoint of the image with the midpoint of the current line.

Option Alignment
Left Aligns the image with the left margin. Adds the align=left attribute to the IMG tag.
Example: <IMG src="" align=left>
Right Aligns the image with the right margin. Adds the align=right attribute to the IMG tag.
Example: <IMG src="" align=right>
Texttop Aligns the top of the image with the top of the text in the current line. Adds the align=texttop attribute to the IMG tag.
Example: <IMG src="" align=texttop>
Absmiddle Aligns the middle of the image with the absolute middle of the text in the current line. Adds the align=absMiddle attribute to the IMG tag.
Example: <IMG src="" align=absMiddle>
Baseline Aligns the bottom of the image with the baseline of the text in the current line. Adds the align=baseline attribute to the IMG tag.
Example: <IMG src="" align=baseline>
Absbottom Aligns the bottom of the image with the absolute bottom of the text in the current line. Adds the align=absBottom attribute to the IMG tag.
Example: <IMG src="" align=absBottom>
Bottom Aligns the bottom of the image with the bottom of the text in the current line. Adds the align=bottom attribute to the IMG tag.
Example: <IMG src="" align=bottom>
Middle Aligns the middle of the image with the baseline of the text in the current line. Adds the align=middle attribute to the IMG tag.
Example: <IMG src="" align=middle>
Top Aligns the top of the image with the top of the tallest element in the current line. Adds the align=top attribute to the IMG tag.
Example: <IMG src="" align=top>

Note   The Absmiddle, Baseline, Absbottom, and Texttop options require browsers that support HTML 4.0.

Border size

Specifies the width in pixels of the border surrounding the image.

Height

Specifies the height in pixels of the space the image will occupy. If a height is not specified, the image displays at its intrinsic height. As you change this value, the image displayed scales accordingly.

Width

Specifies the width in pixels of the space the image will occupy. If width is not specified, the image displays at its intrinsic width. As you change values, the image displayed scales accordingly.

Horizontal space

Specifies the horizontal space between the left and right sides of the image and the surrounding text. The default value is zero pixels.

Vertical space

Specifies the vertical space between the top and bottom of the image and the surrounding text. The default value is zero pixels.

Image Map

Local hotspot information

Specifies the name and location of the <MAP> block containing the coordinates for each hotspot in the image. Enter the URL of the file containing the <MAP> block (or click the browse (...) button to browse to a file), a hash mark (#), and the name assigned to the <MAP> block. If the <MAP> block is in the current file, include just the hash mark and map block name. For example, the following within an <IMAGE> tag would reference a map file named image: USEMAP="#image.map". A file that contains a <MAP> block typically has a .map extension.

Hotspot information on server

Specifies that a <MAP> block is to be processed on the server. When a user clicks an image map, the x and y coordinates of the click location are sent to the server along with a URL for the map file. The ISMAP attribute of the <IMAGE> tag directs the browser to process the image map on the server specified in the URL.

Note   Most Web browsers can process both client-side and server-side image maps. However, older browsers can process only server-side image maps. Consequently, it is common to include both options. For example:
<A HREF ="cgi-bin/imagemap/Mymap.map">
<IMG SRC="Mapimg.gif" USEMAP="#Mymap.map" ISMAP></A>
In this instance, "Mapimg.gif" is an image file in the local directory referenced by this Web page. Browsers that support client-side image maps will also look for the map file "Mymap.map" in the local directory before using the URL in the <A HREF> tag to locate the map file on the server. Since client-side processing is faster then server-side processing, make sure that the ISMAP attribute is placed last.
