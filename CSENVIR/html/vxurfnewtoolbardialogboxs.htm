New Toolbar Dialog Box

This dialog box allows you to name and create a new toolbar. When you choose OK, the new toolbar appears in the integrated development environment (IDE). The toolbar name is also added to the Toolbars list in the Toolbars tab of the Customize dialog box. You can access this dialog box by selecting Toolbars on the View menu, and then choosing Toolbars in the Customize dialog box.

Toolbar name

Displays a default name for the new toolbar, which you can then edit.
