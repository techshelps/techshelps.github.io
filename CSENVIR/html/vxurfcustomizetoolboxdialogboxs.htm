Customize Toolbox Dialog Box

Use the Customize Toolbox dialog box to add or remove controls, objects, and other items displayed in the Toolbox window.

You can open the Customize Toolbox dialog box from the Tools menu.

The Customize Toolbox dialog box displays available components on tabs. Each tab contains a category of components recognized on your machine and ready to add to the Toolbox. To add a control, check the check box beside it; controls are added to the active tab in the Toobox. To remove a control, uncheck its check box.

Because this dialog box is extensible, applications can add tabs containing their own components. The IDE provides the following components:

  • COM Controls

To Do this
Add a control that is not registered on your machine Click Browse and navigate to the control file.
Sort your list of controls by name, location, or date modified Click the appropriate column heading.
