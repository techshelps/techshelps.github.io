Keywords, DOCUMENT Property Pages Dialog Box

By choosing Keywords on the DOCUMENT Property Pages dialog box, you can enter search keywords for the current document. Search keywords are used by internet search engines to index your Web pages. To access this dialog box, open a document for editing and do not select any elements on the page. Choose Property Pages from the View menu, and then click Keywords to enter search keywords for your document.

Note   Separate keywords with commas. The keywords you enter are stored in the META KEYWORDS tag within the <HEAD> of your HTML document. For example, if you enter web,design on the Keywords tab, the following tag is entered in your document:

<meta name=keywords content="web,design"
