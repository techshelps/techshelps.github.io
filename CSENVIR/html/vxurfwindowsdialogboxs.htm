Windows Dialog Box

This dialog box allows you to display, close, and arrange documents that are open in editors and designers. You can display this dialog box by choosing Windows from the Window menu.

Name

Displays the name of documents, including file extension, and windows with the currently active item selected.

Path

Displays the full path for documents. Windows typically do not have a path.

Activate

Makes the selected document the active document in the editor or designer.

Close Windows

Closes the selected documents and windows in the list.

Tile Horizontally

Stacks documents one on top of the other when the integrated development environment (IDE) is in MDI mode.

Tile Vertically

Displays documents side-by-side when the IDE is in MDI mode.
