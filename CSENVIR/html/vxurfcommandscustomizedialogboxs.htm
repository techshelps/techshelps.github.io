Commands, Customize Dialog Box

This dialog box allows you to add and remove commands on toolbars as well as change the images used for toolbar buttons. You can access this page by selecting Toolbars from the View menu, and then choosing Customize.

Categories

Displays the categories under which commands are organized.

Commands

Displays the commands and command images by category. You can select a command and drag and drop the command onto toolbars in the integrated development environment (IDE).

Description

Displays a description of the currently selected command.

Modify Selection

When a command is selected from a toolbar, the toolbar displays a list of commands that allow you to change the display of the button on the toolbar. For example, you can change the image or accelerator keys, as well as specify whether to display text instead of an image for the command.

Keyboard

Displays the Keyboard pane of the Environment options in the Options dialog box, which allows you to change and define shortcut key combinations for commands.
