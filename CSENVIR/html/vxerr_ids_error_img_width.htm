The Image Width must be a positive integer greater than zero.

The Image Width property has a value that is not a positive number greater than zero.

To correct this error

  1. Open this HTML page for editing in Design view of the HTML Designer.
  2. Right-click the image and select Properties from its shortcut menu.
  3. In the Width field, enter a positive integer greater than zero.

    The HTML markup of the selected <IMG> element is modified. Save these changes to your document.
