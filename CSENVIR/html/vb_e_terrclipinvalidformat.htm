Clipboard format is not valid.

The Clipboard only accepts data in certain specified formats. This error generally occurs when you try to place data from a component on the Clipboard, such as an icon from the COM Classic property browser, but the data is in a format incompatible with the Clipboard.

To correct this error

  • Consult the documentation for the component to determine whether you can use the Clipboard for transferring data from the component.

    —or—

  • Select another item from the property browser and try again.
