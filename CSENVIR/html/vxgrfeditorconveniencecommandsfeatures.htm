Editor Convenience Commands and Features

The Code Editor has several new commands and convenience features to facilitate the development process.

  • Word wrap   When text extends beyond the visible code pane, it can automatically wrap to the next line.
  • Incremental search   An incremental search is performed on a letter-by-letter basis as you type the characters. To activate an incremental search, press CTRL+I.
  • Go Back/Go Forward buttons   The Go Back and Go Forward buttons are used to jump to the previous or the next location.
  • Code Outlining   Select a region of code and hide it under a single line, and then expand or collapse that region.
  • Collapse to definition   (Available only in C# and Visual C++) All procedures (methods, functions, ifdefs, code blocks, etc.) are collapsed and hidden under their definition, making it easier to read and navigate throughout your code. To collapse to definition, press CTRL+M and then CTRL+O.
  • Collapse current block   Hide selected code under the first selected line, and then expand or collapse it by clicking the symbol next to it. To collapse the current block, choose Outlining from the Edit menu, and then choose Toggle Outlining Expansion.
  • Clipboard ring   When text is cut or copied, the fragments are stored in an environment clipboard known as the Clipboard Ring where you can pick and choose among them and paste back into your code.
  • IntelliSense   When a function or statement is entered into the Code Editor, its complete syntax and arguments are shown in a ToolTip. In addition, languages that support early binding display available functions, statements, constants, and other values in a list that you can choose from.
  • Line numbering   Display line numbers in code by clicking that option in Tools, Options, Text Editor. You can do this on a per-language basis or for all languages. After line numbering has been activated, you can go to a particular line by clicking Go To in the Edit menu, or by pressing CTRL+G.
