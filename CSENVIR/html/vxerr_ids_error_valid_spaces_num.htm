The values for Border/Horizontal/Vertical Spaces must be either positive integers or zero.

You have selected the Properties for an image and entered values in the Border or Horizontal Space or Vertical Space fields that are not positive integers or zero.

To correct this error

  1. Open this HTML page for editing in Design view of the HTML Designer.
  2. Right-click the image and select Properties from its context menu.
  3. In the Border, Horizontal Space, and Vertical Space fields, enter either zero or positive integers.
