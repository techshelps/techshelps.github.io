The value for Margins must be 1 or 0.

The Margins property of an HTML element has been set to a value other than 1 or 0.

To correct this error

  1. Open this HTML page for editing in Design view of the HTML Designer.
  2. Right-click on the element and select Properties from the context menu.
  3. The Properties window displays the properties of the selected cell. Scroll to the Margins property and enter 1 (show) or 0 (conceal).

    The HTML markup of the selected element is modified. Save these changes to your document.
