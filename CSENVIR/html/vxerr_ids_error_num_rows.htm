The Number of Rows must be an integer greater than zero.

You have entered a value in the Number of Rows field of the Insert Table dialog box that is not a positive number greater than zero.

To correct this error

  1. Open this HTML page for editing in Design view of the HTML Designer.
  2. Click at the location on the page where you want to insert a Table.
  3. From the Table menu, select Insert Table.
  4. Enter a value in the Number of Rows field that is a positive integer greater than zero.
