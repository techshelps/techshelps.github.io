The Horizontal Space value must be a positive number greater than zero.

You have selected the Properties for an image and entered a value in the Horizontal Space field that is not a positive integer greater than zero.

To correct this error

  1. Open this HTML page for editing in Design view of the HTML Designer.
  2. Right-click the image and select Properties from its context menu.
  3. In the Horizontal Space field, enter a positive integer greater than zero.
