Could not open this page in Design View. Nested double quotes are not permitted in an attribute value.

You have included an item in double quotes within a quoted attribute value.

To correct this error

  • Use single quotes within double quotes: 
    VALUE="The 'King' of Rock and Roll"

    - or -

  • Place an escape character before each nested double quote: 
    VALUE="The \"King\" of Rock and Roll"

    - or -

  • Use the HTML markup &quot; for the double quote mark: 
    VALUE="The &quot;King&quot; of Rock and Roll"
