Color and Margins, DOCUMENT Property Pages Dialog Box

By choosing Color and Margins on the DOCUMENT Property Pages dialog box, you can change the display properties of the current document. To access this dialog box, open a document for editing and do not select any elements on the page. Choose Property Pages from the View menu, and then click General to access the following options:

Color

Background

Sets the color for the current document's background. Select a color from the drop-down list or choose the ellipse (...) button to select a color in the Color Picker dialog box.

Text

Sets the color for text in the current document. Select a color from the drop-down list or choose the ellipse (...) button to select a color in the Color Picker dialog box.

Link

Sets the text color for non-visited hyperlinks in the current document. Select a color from the drop-down list or choose the ellipse (...) button to select a color in the Color Picker dialog box.

Visited link

Sets the text color for visited hyperlinks in the current document. Select a color from the drop-down list or choose the ellipse (...) button to select a color in the Color Picker dialog box.

Active link

Sets the text color for the active hyperlink as you select it in the current document. Select a color from the drop-down list or choose the ellipse (...) button to select a color in the Color Picker dialog box.

Preview

Displays text strings illustrating the current settings for link colors. As you select properties, the preview strings change to reflect the current values.

Page margins

Top

Sets the size in pixels for top margin of the current document.

Bottom

Sets the size in pixels for bottom margin of the current document.

Left

Sets the size in pixels for left margin of the current document.

Right

Sets the size in pixels for right margin of the current document.
