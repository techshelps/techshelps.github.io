General, Toolbox Window

The General tab and Clipboard Ring tab appear by default whenever the Toolbox window opens. Use the General tab to store default and custom controls. The Toolbox is available from the View menu.

Initially, the General tab of the Toolbox window contains only the Pointer tool. As an editor or designer becomes active, more tabs and tools become available.

Tip   You can rename, remove, and organize Toolbox tabs and their items, and add custom tabs and items.
