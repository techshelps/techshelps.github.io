To disable Mosaic or modify a Mosaic configuration

 

  1. From the NVIDIA Control Panel Select a Task pane, under Workstation, click Set Up Mosaic to open the associated page.

  2. If not already expanded, click the expand icon (+) for the Mosaic setup you want to modify.

    3. The expanded Mosaic setup box shows all the configuration information for that setup, including the display arrangement.  

    You can also click Expand all to expand all of the Mosaic configuration boxes.

  3. From the individual Mosaic configuration box, click Disable to disable Mosaic, or click Modify and then follow the steps in the Mosaic Setup wizard to make changes to that Mosaic configuration.

If there are multiple Mosaic configurations, you can also click the top Disable all to disable all of the Mosaic configurations on the system at one time.

See Set Up Mosaic Wizard for guidance on using the wizard.
