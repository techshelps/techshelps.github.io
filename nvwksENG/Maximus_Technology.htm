Manage GPU Utilisation

Engineers, artists, designers, and scientists can interact with high-performance visuals while performing simulations or renderings on the same workstation system – at the same time.  

This is accomplished using two NVIDIA GPUs in the system – an NVIDIA Quadro GPU dedicated to Interactive Design, and an NVIDIA Tesla or Quadro GPU dedicated to computational simulation or computational rendering.
