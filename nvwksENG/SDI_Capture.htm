SDI Capture

 

The NVIDIA® Quadro® SDI Capture card captures uncompressed 8-bit, 10-bit and 12-bit video  directly to the Quadro SDI-enabled GPU memory.

With the NVIDIA Control Panel you can view the signal status of each input capture port as well as the signal status of the output sync port of the SDI capture card.  

  • For help on using this feature, see How do I view the SDI input status.

  • For reference information on this feature, see the Reference: SDI Input.
