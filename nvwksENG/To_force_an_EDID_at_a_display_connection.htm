To force an EDID at a display connection

  1. From the NVIDIA Control Panel Select a Task pane, under Workstation, click View system topology.

  2. Locate the Quadro FX card with the connector on which you want to force a specific EDID, then click the corresponding Quadro FX [+] icon to expand the branch.

  3. Locate the connector type, then click EDID to open the Manage EDID dialogue.   

  4. Click the Load tab.

  5. Click Browse to locate and select the EDID file that you want to apply.

    6. The filename can have any extension but the contents must conform to the VESA EDID data format.

  6. Under Connector to be forced, click the arrow and select the connector type.

  7. Select the displays to which to apply the EDID.

  8. Select the check box indicating that you have read the warnings, then click Load EDID.

The EDID from the file is now applied to the selected connectors.
