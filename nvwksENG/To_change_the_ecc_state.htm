To change the ECC state

  1. From the NVIDIA Control Panel Select a Task pane, under Workstation, click Manage GPU Utilisation.

  2. For each Quadro and Tesla card, select the Enable Error Correction Code selection box to enable ECC, or clear the selection box to disable ECC.

    3. ECC guidelines:

    • ECC Enabled: Recommended for high-precision, GPU-accelerated computational applications.     

    • ECC Disabled: Recommended for GPU-accelerated, ray tracing applications.

  3. Click Apply when finished.

 

Note: You can also change the ECC state for each GPU using the Change ECC State page.  See ECC State Control for more information about ECC.
