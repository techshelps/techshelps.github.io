To check the ECC state of your GPU

  1. From the NVIDIA Control Panel, click the System Information link at the bottom left corner of the NVIDIA Control Panel.

  2. Click the Display tab, then under the Components column select the GPU that you want to check.

    3. The Error Correction Code (ECC) line in the Details column shows whether ECC is turned On or Off for the selected GPU.
