To set up your SDI output

  1. From the NVIDIA Control Panel Select a Task pane, under Workstation, click Manage SDI output settings.

  2. If you are using more than one NVIDIA Quadro FX SDI card (as in the NVIDIA Quadro Plex 1000), under Select SDI output display, click the display icon corresponding to the display you want to configure, then follow the remaining instructions for that display.

    • This option does not appear if your system contains only one NVIDIA Quadro SDI card.

    • If this option does not appear and your system does contain more than one NVIDIA Quadro SDI card, see To enable multiple SDI cards for instructions on enabling the cards.

  3. Click the SDI Signal Settings tab and choose a method for determining the format of the SDI output - either using internal timing or synchronised to an external signal source.

    • To use internal timing, select Free running (internal timing), then click the list arrow and choose from the list of available SDI signal formats.

    • To synchronise to an external signal source, make sure the house sync is connected to the INPUT BNC connector on the graphics card, then select the Synchronised to a house sync signal radio button and set up the synchronisation and signal formats as follows:

      • If you are connected to an NVIDIA Quadro FX 4500/5500 SDI card and you have both SDI and composite house syncs connected, select the corresponding House sync type radio button for the one you want to use.

      • If you are connected to a later NVIDIA Quadro SDI card that has just one input sync connector, click the House sync type list arrow and select the type of input signal you are using, then click Detect.

        • This must be done whenever the signal source changes.

      • To synchronise the pixel scanning of the SDI output to the external signal using genlock, select Pixel-accurate synchronisation.

      • To synchronise the frame rate of the SDI output to the external signal using frame lock, select Frame-accurate synchronisation using this format, then click the list arrow and choose from the list of available SDI signal formats.

      • To introduce a delay in the SDI output, enter the pixel or line delay values in the appropriate HSYNC or VSYNC delay boxes.

  4. Check the Terminate the analogue signal at this connector check box if

    • the system is a standalone system synchronised to a house sync signal, or 

    • the system is the last in a chain of systems connected to the same house sync signal.

    Note: This control is not available with the NVIDIA Quadro FX 4000 SDI card.

  5. Choose the SDI output data format by clicking the SDI output format list arrow and then selecting from the list of available colour formats.

  6. Click Apply.
