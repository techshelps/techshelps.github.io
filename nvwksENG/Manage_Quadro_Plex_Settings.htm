NVIDIA Mosaic

NVIDIA® Mosaic Technology takes multiple displays and Quadro or NVS GPUs and presents them as a single logical GPU to the operating system.

When using sync-capable GPUs and system, the Mosaic configuration offers the synchronised multi-GPU output required for stereoscopic or multi-projector installations.

Either way, you can use multiple displays to create a larger virtual canvas.
