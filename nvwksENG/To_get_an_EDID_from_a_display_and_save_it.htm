To get an EDID from a display and save it

  1. From the NVIDIA Control Panel Select a Task pane, under Workstation, click View system topology.

  2. Locate the display with the EDID that you want to save, then click the corresponding Display [+] icon to expand the branch.

  3. At the EDID Source property, click Monitor.

    The Manage EDID dialogue page opens to the Export tab.

  4. Select the display containing the EDID you want to save, then click Export EDID to open the Save file dialogue box where you can save the EDID information in a file.

  5. Click OK when finished.
