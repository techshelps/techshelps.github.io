To set up colour space conversion

  1. From the NVIDIA Control Panel Select a Task pane, under Workstation, click Manage SDI output settings.

  2. Click the Colour Settings tab.

  3. Check Override the default colour space conversion.

  4. Click the Initialise the colour space conversion matrix with list arrow and then click one of the pre-defined colour-space standards to use as a starting point.

  5. In each colour-space text box, either enter values directly or use the corresponding up and down arrows to change the values.

  6. Click OK when finished.

Note: Colour space conversion control is not available with the NVIDIA Quadro FX 4000 SDI card.
