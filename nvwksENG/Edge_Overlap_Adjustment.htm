Edge Overlap Adjustment

Multiple projection systems can be used to create a virtual canvas where multiple displays are projected onto a larger screen. In order to eliminate the hard edge between adjacent displays, NVIDIA Edge Overlap repeats the edges of adjacent displays so that the display edges can be overlapped to create a seamless composite projection.

  • For help on using the edge-overlap feature, see How Do I.

  • For reference information, see Reference.

NOTE: Edge overlap adjustment is available only with Windows XP.
