Manager Tools
The Manager Tools screen contains the Forecasting feature you can use to forecast sales data, sales mixes based on sales history and future events that may affect volume.
 
Features in the Manager Tools screen include:
•
Forecasting — Use sales history, future events, and projected growth to create sales forecasts that will affect staffing recommendations.
•
Import — Import sales, employee punches, and employee checkouts, for labor and payroll related tasks.
•
Export — Export sales, labor and payroll data to an external system, such as a payroll system.
•
Site Notifications — View all site related alerts and notifications.
•
Activity Log — View the activity of other users in the MenuLink Client.
•
Reports — View and print Labor-related reports.
