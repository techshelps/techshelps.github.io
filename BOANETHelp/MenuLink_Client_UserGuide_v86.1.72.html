Edit an Existing Schedule
You can easily edit an existing schedule by entering the Employee Schedule and selecting the Existing tab, in the Employee Schedule dialog. Using the drop-down arrow select the appropriate date from the Schedule Start Date calendar and make changes, as needed.
 
To Edit an Existing Schedule:
1.
In the Scheduling screen, click Employee Schedule.
2.
In the Employee Schedule dialog, select the Existing tab.
3.
Click the drop down arrow to access the Schedule Start Date drop-down list.
4.
Select an existing schedule from the list. Each existing schedule appears in the list, based on the schedule’s start date.
5.
Modify the schedule, by adding, deleting or modifying shifts as necessary.
6.
Click Save to keep your changes.
