CreateCabinet

Creates Cabinet.

Public Function CreateCabinet (

ByVal CabinetName As String,

ByVal DBPath As String,

ByVal DoBpath As String) As Long

 

Parameters  

In/out

Name

Data type

Description

In

CabinetName

String

Cabinet Name

In

DBPath

String

DataBase Path.

In

DoBPath

String

DoB Path.

 

Description

This function is called from OmniDocs Desktop. It is used to create local cabinet.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-5

Cabinet Already Exist.

2004

Out of Diskspace.

 

Example

Dim CabinetName          As  String

Dim DBPath                  As  String

Dim DoBPath                As String

Dim lRetVal                   As  long

CabinetName = “TestCabinet”

DBPath =”C:\TestCabinet.Mdb”

DoBPath = “C:\DoB\”

lRetVal = gPanDb. CreateCabinet(CabinetName,DBPath,DoBPath)

If lRetVal <> NG_SUCCESS Then

        MsgBox gPandDb.ErrDescription, vbInformation

        Screen.MousePointer = vbNormal

        Exit Sub

Else

        MsgBox “Success” 

  End If
