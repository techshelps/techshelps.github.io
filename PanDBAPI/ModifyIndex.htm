ModifyIndex

This function is used to modify the global index present in the cabinet.

Public Function ModifyIndex(ByRef IndexProp() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In/Out

IndexProp

Byte[TagIndex]

Properties of Index to be modified

 

Description

This function is called on both local and shared cabinet. It is called when Global index properties  is modified from desktop in local cabinet and from admin in shared  cabinet.

 

Return value

Name

Description

 1

In case of success

-2

Invalid operation in  read only cabinet

-3

Cabinet not connected

-201

 

Insufficient rights for the current operation

 

Example

Dim IndexInfo       As TagIndex

Dim IndexProp     As Byte

Dim lRetVal         As  Long

Dim gPDb           As PanDb.pdbCls


Set gPDb = New PanDb.pdbCls

TagIndex.IndexId =  32

TagIndex.Length= 2

TagIndex.Name = “remark”

TagIndex.Type = 3

 

Redim IndexProp(Len(IndexInfo) -1)

Call CopyMemory(IndexProp(0),IndexInfo,Len(IndexInfo))

lRetVal=gPDb.ModifyIndex(IndexProp)

if lRetVal <> NG_SUCCESS Then

     MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

End If
