SetGroupProp

Sets the properties of a group.

Public Function SetGroupProp(ByRef GroupProp() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In/Out

 

GroupProp

Byte[TagGroup]

Byte array specifying the properties of the group to be set.

 

Description

This function is called only for Shared cabinet. It is called whenever properties of an existing group are changed (except in case as users add/remove to/from the group)  or when privileges for a group are set/changed.

 

Return value

Name

Description

 1

In case of success

-1

Invalid cabinet name

-3

Cabinet not connected

-201

Insufficient rights for the current operation

-139

group with the given name already exist.

-133

User not logged in

 

Example

    Dim lRetVal                 As Long

    Dim bGroupProp()        As Byte

    Dim udtGroupProp       As TagGroup

    Dim gPDb                   As PanDb.pdbCls


    Set gPDb = New PanDb.pdbCls

    udtGroupProp.Comment = "Member of Application group"

    udtGroupProp.GroupName = "My Group"

    ReDim bGroupProp(Len(udtGroupProp) - 1)

    Call CopyMemory(bGroupProp(0), udtGroupProp, Len(udtGroupProp))

    lRetVal = gPDb.SetGroupProp(bGroupProp)

    If lRetVal <> 1 Then

        MsgBox gPDb.ErrDescription

    Else

        MsgBox "Success"

    End If
