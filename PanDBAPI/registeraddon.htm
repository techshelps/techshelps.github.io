RegisterADDOn

Used to register the ADDOn.

Public Function RegisterADDOn(ConfigFile As String) As Long

 

Parameters

In/out

Name

Data type

Description

In

ConfigFile

String

Path of Configuration File.

 

Description

This function is called from OmniDocs Desktop. It is used to compile the stored procedures for that particular ADDOn specified by “Config File” that contains the name of Stored Procedures.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

Error

In case of Failure

 

Example

Dim ConfigFile               As String

Dim lRetVal                   As  long

ConfigFile   = “C:\PrgramFiles\OmniDocsServer\Config.ini”

lRetVal = gPanDb. RegisterADDOn(ConfigFile)

If lRetVal <> NG_SUCCESS Then

        MsgBox gPandDb.ErrDescription, vbInformation

        Screen.MousePointer = vbNormal

        Exit Sub

Else

        MsgBox “Success”  

End if
