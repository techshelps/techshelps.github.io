DeleteFolder

Called to delete a folder permanently from the cabinet.

Public Function DeleteFolder (ByVal FolderId As Long, ByVal DeleteRefrenceFlag As String) As Long

Parameters

In/out

Name            

Data type

 Description

In

FolderId

Long

Index of folder being deleted

In

DelteReferenceFlag

String

Whether reference also has to be deleted

 

Description

This function is called from both local and shared cabinet. It is called when a folder is permanently deleted.

 

Return value

Name

Description

 1

In case of success

-1

Invalid operation

-2

Invalid operation in read only cabinet

-3

Cabinet not connected

 

Example

Dim FolderId             As Long

Dim DelRefFlag         As String

Dim lRetval               As  Long

Dim gPDb                 As PanDb.pdbCls


Set gPDb = New PanDb.pdbCls

FolderId = 67

DelRefFlag = “Y”

lRetVal=gPDb.DeleteFolder(FolderId,DelRefFlag)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

Endif
