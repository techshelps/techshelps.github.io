GetAttachment

Gets Attachment.

Public Function GetAttachment(

                ByVal DocId                      As Long,

                ByVal VersionNo               As Single,

                ByVal PageNo                   As Integer,

                ByVal AttachmentName     As String,

                ByVal FileName As String) As Long

 

Parameters

In/out

Name

Data type

Description

In

DocumentId

Long

Document Index.

In

VersionNo

Single

Version Number.

In

PageNo

Integer

Page Number.

In

AttachmentName

String

Attachment Name.

In

FileName

String

File Name.

 

Description

This function is used to get the attachment on the empty document, specified by the FileName.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-3

Cabinet not connected

 

Example

Dim lRetVal             As Long

Dim lDocId              As long

Dim sngVersionNo   As Single

Dim iPageNo           As integer

Dim sAttachmentName As string

Dim sFileName       As string

LDocId = 10

SngVersionNo = 12.2 

IPageNo =2

SAttachmentName = “Important”

SFileName = “C:\Temp.doc”

lRetVal = gPanObj.GetAttachment(lDocId, sngVersionNo, iPageNo, sAttachmentName, sFilename)

   If lRetVal <> NG_SUCCESS Then

      MsgBox gPanObj.ErrDescription

   Else

       MsgBox  “Success”                    

   End If
