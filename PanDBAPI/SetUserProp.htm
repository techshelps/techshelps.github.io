SetUserProp

To set properties of a user like username , password

 

Public Function SetUserProp(ByRef UserProp() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In,Out

 

UserProp

Byte[TagUser]

Properties of user

 

Description

This function is called through Admin to change any existing user’s properties and it is  called from Desktop whenever user changes his password.

 

Return value

Name

Description

1

In case of success

-1

Invalid cabinet name

-3

Cabinet not connected

-201

Insufficient rights for the current operation

-131

User does not exist.

-133

User not logged in

 

Example

    Dim lRetVal                     As Long

    Dim lUserId                     As Long

    Dim bUserProp()             As Byte

    Dim udtUserProp             As TagUser

    Dim gPdb as Pandb.Pdbcls

    set gPdb = new Pandb.PdbCls

    lUserId = 5

    udtUserProp.FamilyName = "Kumar"

    udtUserProp.UserName = "Sanjay"

    udtUserProp.PersonalName = ""

    ReDim bUserProp(Len(udtUserProp) - 1)

    Call CopyMemory(bUserProp(0), udtUserProp, Len(udtUserProp))

    

    lRetVal = gPdb.SetUserProp(lUserId, bUserProp)

    If lRetVal <> 1 Then

        MsgBox gPdb.ErrDescription

    Else

        MsgBox "Success"

    End If
