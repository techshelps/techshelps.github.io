LockObject

Used to Lock an Object.

LockObject(ByVal ObjectId As Long, _

ByVal VersionNo As Single, _

ByVal ObjectType As String, _

ByVal LockMessage As String) As Long

 

Parameters

In/out

Name

Data type

Description

In

ObjectId

Long

Index of the object to be locked

Out

VersionNo

Single

Version no the object

In

ObjectType

String

Type of object

In

LockMessage

String

Any message associated

 

Description

This function is called in both local and shared cabinet from Admin and Desktop.

In admin this is called when any folder is explicitly locked, and in desktop this function is

Internally called when any document is opened is OmniViewer.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-4

Invalid Operation

-3

Cabinet not connected

-201

Insufficient rights for the current operation

 

Example

Dim ObjectId             As Long

Dim VersionNo         As Single

Dim ObjectType        As String

Dim LockMessage    As String

Dim lRetVal              As Long

ObjectId  = 47

VersionNo = 1

ObjectType = “F”

LockMessage = “”  

lRetVal=gPDb(miCurCabId).LockObject(ObjectId,VersionNo,ObjectType,LockMessage)

  if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb(miCurCabId).ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Endif
