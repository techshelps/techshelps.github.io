DeleteUser

Called when a User is deleted

Public Function DeleteUser(ByVal UserId As Long) As Long

 

Parameters

In/out

Name

Data type

Description

In

UserId

Long

Index of user being deleted

 

Description

This function is called from Omnidocs Admin.It is called when any User is deleted. It is passed with the Index of the User that has to be deleted.

 

Return value

Name

Description

NG_SUCCESS

In case of success

-1

Invalid operation

-3

Cabinet not connected

 

Example

Dim UserId         As Long

Dim lRetval         As  Long

UserId = 12

LRetVal =gPanDb. DeleteUser (UserId)

If lRetVal <> NG_SUCCESS Then

    MsgBox gPanDb.ErrDescription, vbInformation

Else

     MsgBox “Success”

Endif
