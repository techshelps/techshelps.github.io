AddKeywordsinDict

Adds Keywords in cabinet .

Public Function AddKeywordsinDict(ByRef Keywords() As String,ByVal AuthoriseFlag As String) As Long

 

Parameters

In/out

Name

Data type

Description

In

AuthoriseFlag

String

Authorize flag ('Y' for Authorized)

In

Keywords

String

Keywords list.

 

Description

This function call is used to create keywords in the cabinet . It is used both for local and shared cabinet .

 

Return value

Name

Description

 1

In case of success

-3

Cabinet not connected.

-2

Invalid operation in read only cabinet

 

Example

Dim Keywords()         As String

Dim AuthoriseFlag     As String

Dim gPDb                 As PanDb.pdbCls


Set gPDb = New PanDb.pdbCls    

 

ReDim Keywords(0)

KeyWords(0) = “Remark”

AuthoriseFlag = “Y”

lRetVal=gPDb.AddKeywordsinDict(KeyWords, AuthoriseFlag)

  if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Endif
