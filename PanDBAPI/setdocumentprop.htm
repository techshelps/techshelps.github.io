SetDocumentProp          

Set properties of a document                                                                              

SetDocumentProp( ByVal DocId         As Long,

ByRef DocProp()                               As Byte,

Optional ByVal FolderId As Long = 0) As Long

 

Parameters  

In/out

Name

Data type

Description

In

DocId

Long

Index of the concerned document

In,Out

DocProp

Byte[TagDocument]

[In]Properties to be set

[Out]Properties of folder after set

 

Description  

This function is called form Desktop for both shared and local cabinet. It is called when a new document (empty doc) is created by right click menu, or when either comments or sharing is added/changed. 

 

Return value  

Name

Description

NG_SUCCESS

In case of success

-1

Invalid cabinet name

-2

Local cabinet is readonly

-3

Cabinet not connected

-31

Document with the same name already exist

-201

Insufficient rights for the current operation

-133

User not logged in

 

Example

    Dim lRetval                           As Long

    Dim bDocProp()                    As Byte

    Dim udtDocProp                   As TagDocument

    Dim lDocId                           As Long

    udtDocProp.DocName = "MyDoc"

    udtDocProp.Doctype = “TIF”

    udtDocProp.Author = “Mukesh”

    udtDocProp.sLinked = “Y”

    udtDocProp.ExpiryDateTime =  'Format$(Now, "dd/mm/yyyy hh:nn:ss")

    ReDim bDocProp(Len(udtDocProp) - 1)

    Call CopyMemory(bDocProp(0), udtDocProp, Len(udtDocProp))

    lRetval = PanObj.SetDocumentProp(9, bDocProp)

    If lRetval <> 1 Then

        MsgBox PanObj.ErrDescription

    Else

 MsgBox "Success"

    End If
