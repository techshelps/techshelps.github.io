CheckForDataDefExistence

Check whether data definition exists or not.

Public Function CheckForDataDefExistence(FieldList() As Byte,

DataDefCount                                                            As Long,

DataDefList() As Byte)                                                As Long

 

Parameters

In/out

Name

Data type

Description

In

FieldList()

Byte

Byte Array of Field List

In

DataDefList()

Byte

Byte array of DataDef List

In

DataDefCount

Long

Number of Data Definition

 

Description

This function is used to check the existence of Data Definition with the same fields’ type exists. This function is used in both local cabinet and shared cabinet.

 

Return value

Name

Description

NG_SUCCESS

In case of success

-3

Cabinet not connected

 

Example

Dim FieldList()              As Byte

Dim lNoFields               As Long

Dim udtField()               As TagDataDefField

Dim DataDefCount        As Long

Dim DataDefList()         As Byte           

LNoFields = 2

ReDim FieldList(((iNoFields) * Len(udtField(0))) - 1)

Call CopyMemory(FieldList(iCount * Len(udtField(0))), udtField(iCount), Len(udtField(0)))

lRetVal=gPanObj. CheckForDataDefExistence(FieldList,DataDefCount, DataDefList)

If lRetVal <> NG_SUCCESS Then

    MsgBox gPanObj.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Else

            If DataDefCount > 0 then

                Msgbox “Data Definition Exists!”

            End if

End if
