DeleteKeywordsFromDocument

Delete Keywords From Document.

Public Function DeleteKeywordsFromDocument (ByVal DocumentId As Long,  

ByRef Keywords() As String) As Long

 

Parameters

In/out

Name

Data type

Description

In

DocumentId

Long

Document Index.

In/Out

KeyWords ()

String Array

Array of Keywords to be deleted.

 

Description

This function is called from OmniDocs Desktop. It is used to delete keywords from the document.

 

Return value

Name

Description

NG_SUCCESS

In case of success

-1

Invalid cabinet name

-3

Cabinet not connected

-2

Invalid Operation in Read-Only Cabinet.

 

Example

Dim DocumentId                   As  long

Dim sDelKeyWord()              As  String

Dim lRetVal                          As  long

Redim sDelKeyWord(2)

sDelKeyWord(0) =”A”

sDelKeyWord(1) =”X”

DocumentId = 483

lRetVal = gPanDb. DeleteKeywordsFromDocument(DocumentId, sDelKeyWord)

If lRetVal <> NG_SUCCESS Then

        MsgBox gPandDb.ErrDescription, vbInformation

        Screen.MousePointer = vbNormal

        Exit Sub

Else

        MsgBox “Success”  

  End If
