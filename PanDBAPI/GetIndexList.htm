GetIndexList

Returns list of Global Indexes present in the cabinet.

Public Function GetIndexList( ByVal IndexFlag As String, ByVal StartPos As Long, ByVal FetchCount As Long, _
ByRef RetrievedRecCount As Long, ByRef TotalRecCount As Long, ByRef IndexList() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In

IndexFlag

String

Flag to specify index type 

In

StartPos

Long

Starting position for fetching of data

In

FetchCount

Long

Number of values to be fetched in each batch

Out

RetrievedCount

Long

Values fetched in each batch

Out

TotalCount

Long

Total retrieved count

Out

IndexList

Byte[TagIndex]

List of Indexes in the form of byte array

 

Description

This function is called form both local and shared cabinet. This call is used to fetch the global indexes of the cabinet. This call supports batching also. If 'FetchCount' is set as 0 then it is taken as '32000' by default.

 

Return value

Name

Description

 1

In case of success

-3

Cabinet not connected

-4

Invalid operation

 

Example

Dim IndexFlag                   As String    

Dim StartPos                    As Long

Dim FetchCount                As Long

Dim RetrievedCount          As Long

Dim TotalCount                As Long

Dim IndexList()                 As Byte

Dim lRetVal                     As Long

Dim gPDb                       As PanDb.pdbCls


Set gPDb = New PanDb.pdbCls

IndexFlag =  “G” ' for global index

StartPos = 0

FetchCount = 10

lRetVal=gPDb.GetIndexList(IndexFlag,StartPos,FetchCount,RetrieveCount,TotalCount,IndexList)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Endif
