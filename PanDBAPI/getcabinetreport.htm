GetCabinetReport

Gets Cabinet Report.

Public Function GetCabinetReport(

                ByRef NoOfFolders As Long,

                ByRef NoOfDocuments As Long,

                ByRef NoOfDataClasses As Long,

                ByRef NoOfGlobalIndexes As Long,

                ByRef NoOfDataTypeIndexes As Long,

                ByRef NoOfAuthorisedKeywords As Long,

                ByRef NoOfUnAuthorisedKeywords As Long,

                ByRef CabinetSize As Long) As Long

Description

This function is used to take the report of the cabinet. This function is called from OmniDocs Desktop. This function returns the number of folders, documents, dataclasses, global indexes and all the other important information regarding cabinet.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-3

Cabinet not connected
