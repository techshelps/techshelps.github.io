SetFolderProp      

Set properties of a folder                                                                         

SetFolderProp(ByVal FolderId                       As Long,

                        ByRef FolderProp() As Byte) As Long

 

Parameters  

In/out

Name

Data type

Description

In

FolderId

Long

Index of folder whose properties are to be set

In,Out

FolderProp

Byte[TagFolder]

[In]Properties to be set

[Out]Properties of folder after set

 

Description  

This function is called form Admin and Desktop for both shared and local cabinet. It is called when existing folder’s properties are changed (not at the time of creation), or when rights are set on that folder. In desktop it is called when a new folder is created. It also checks that folder for which properties are to set should not be any system folder.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-1

Invalid cabinet name

-3

Cabinet not connected

-12

Invaild folder name

-16

Folder with the same name already exist

-201

Insufficient rights for the current operation

-133

User not logged in

 

Example

    Dim lRetval                                     As Long

    Dim bFolderProp()                           As Byte

    Dim lFolderId                                  As Long

    Dim udtFoldprop                             As TagFolder

    udtFoldprop.Comment = “comments added”

    udtFoldprop.ExpiryDateTime = “12 june 2002;06:24:00 PM”

    udtFoldprop.FolderName = "NewFolder"

    udtFoldprop.FolderType = “G”

    udtFoldprop.ImageVolIndex = 1

    ReDim bFolderProp(Len(udtFoldprop) - 1)

    Call CopyMemory(bFolderProp(0), udtFoldprop, Len(udtFoldprop))

    lRetval = PanObj.SetFolderProp(102, bFolderProp)

    If lRetval = 1 Then

        MsgBox "Success"

    Else

        MsgBox PanObj.ErrDescription

    End If
