AddDataDefinition

Adds Data Class to the cabinet.

Public Function AddDataDefinition(

                ByVal DataDefName As String, _

                ByVal DataDefComment As String, _

                ByRef FieldList()  As Byte, _

                ByRef DataDefIndex As Long, _

                ByVal Unused As String, _

                Optional CheckExistenceOnly As String) As Long

 

Parameters

In/out

 Name

Data type

Description

In  

DataDefName

String

Name of the dataclass.

In  

DataDefComment

String

Comments on the dataclass.

Out

FieldList

 Byte[TagDataDefField] 

Specifies a byte array for the field list structure.

Out  

DataDefIndex

Long

Index of the dataclass.

In

Unused

String

Stores an important information of the dataclass.

In  

CheckExistenceOnly

String

Check whether already exists or not.

 

Description

This function is used to create the dataclass. To create the dataclass the following fields namely data class name, comment, field list, date definition index, unused and check existence only have to be passed.

 

Return value

Name

Description

 1

In case of success

-3

Cabinet not connected.

-2

Invalid operation in read only cabinet.

-137

Data definition already exists with the same name

-151

Data definition already exists with the same fields.

 

Example

Dim DataDefName            As String

Dim DataDefComment      As String

Dim DataDefFieldList()      As Byte

Dim DataDefIndex            As Long

Dim Unused                    As String

Dim CheckExistencenly   As String

Dim udtDefField()            As TagDataDefField

Dim lCount                     As Long

Dim gPDb                     As PanDb.pdbCls

Set gPDb  = New PanDb.pdbCls

DataDefName = “myclass”

DatDefComment = “”

 

ReDim DataDefFieldList((Len(udtDefField) * (UBound(udtDefField) + 1)) - 1)

For lCount = 0 To UBound(udtDefField)
    Call CopyMemory(DataDefFieldList(lCount*Len(udtDefField(lCount))),udtDefField(lCount),len(udtDefField(                  lCount)))
Next lCount

 

lRetVal=gPDb.AddDataDefinition(DataDefName,DataDefComment,DataDefFieldList,DataDefIndex,””,”Y”,)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Endif
