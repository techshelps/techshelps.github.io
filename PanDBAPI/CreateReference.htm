CreateReference

It is called to create a reference of object ( folder or document ) in another folder .

Public Function CreateReference(ByVal ObjectId As Long, ByVal ObjectType As String, ByVal ParentId As Long, ObjectProp() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In

ObjectId

Long

Index of document/folder whose reference is to be created

In

ObjectType

String

Type of object 'D' or 'F'

In

ParentId

Long

Index of parent of object

Out

 

ObjectProp

Byte[TagDocument]

Properties of reference created in the form of the byte array

 

Description

This function is called form local and shared cabinet. It is called  when a reference of a document or a folder is created.

 

Return value

Name

Description

 1

In case of success

-1

Invalid operation

-2

Invalid operation in read only cabinet

-3

Cabinet not connected

 

Example

Dim  ObjectId           As Long

Dim ObjectType        As String

Dim  ParentId           As Long

Dim ObjectProp()      As Byte

Dim lRetVal              As Long

Dim gPDb                As PanDb.pdbCls


Set gPDb = New PanDb.pdbCls

ObjectId = 67

ObjectType = “D”

ParentId = 23

lRetVal=gPDb.CreateReference (ObjectId,ObjectType,ParentId,ObjectProp)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Endif
