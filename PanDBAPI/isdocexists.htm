IsDocExists

Used to check whether document exists or not.

IsDocExists(DocumentProp() As Byte,

NoOfDocument As Long) As Long

 

Parameters  

In/out

Name

Data type

Description

In

DocumentProp

Byte[TagDocument]

Index of the document

In

NoOfDocuments

Long

Number of documents

 

Description  

This function is used to verify whether a particular document(s) exists. This function is  supplied With document properties, if in document properties FolderId is passed then it searches the document in that folder else if nothing passed it searches the document in entire cabinet.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-3

Cabinet not Connected

-4

Invalid Operation

-201

Insufficient rights for the current operation

 

Example

Dim DocInfo                  As  TagDocument

Dim DocumentProp       As   Byte

Dim NumberOfDocs      As   Longs

DocInfo.DocumentId = 0         

DocInfo.FolderId = 54

DocInfo.NoOfPages = 0

DocInfo.Owner  = 8

DocInfo.DataDefId   = 0

DocInfo.DocumentSize  = 0

DocInfo.FTSDocId  = 0

DocInfo.CheckOutBy = 0         

DocInfo.FinalizedBy = 0

DocInfo.FiledBy  = 0

DocInfo.AccessType = “I”

DocInfo.DocType  = “txt”

DocInfo.ExpiryDateTime = “11/13/12 1:00:05 PM”

DocInfo.DocName  = “mydoc”

DocInfo.Author = “Nikesh”

DocInfo.CheckInCheckOutFlag = “Y”

DocInfo.LimitCount = 50

NumberOfDocs  = 1

Call CopyMemory(DocumentProp(0),DocInfo,Len(DocInfo))

lRetVal=gPDb(miCurCabId).IsDocExist(DocumentProp,NumberOfDocs)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb(miCurCabId).ErrDescription, vbInformation

    Screen.MousePointer = vbNormal

Endif
