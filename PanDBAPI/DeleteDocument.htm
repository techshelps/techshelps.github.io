DeleteDocument

Called when a document is to be deleted permanently.

Public Function DeleteDocument (ByVal DocId As Long, ByVal DeleteRefrenceFlag As String) As Long

 

Parameters

In/out

Name

Data type

Description

In

DocId

Long

Index of the document to be deleted.

In

DeleteReferenceFlag

String

Whether reference also has to be deleted

 

Description

This function is called from both local and shared cabinet .It is called when a document is permanently deleted. In case of normal delete, document is just moved in trash.

 

Return value

Name

Description

 1

In case of success

-1

Invalid operation

-2

Invalid operation in read only cabinet

-3

Cabinet not connected

 

Example

Dim DocId                 As Long

Dim DelRefFlag          As String

Dim lRetVal               As  Long

Dim gPDb                 As PanDb.pdbCls


Set gPDb = New PanDb.pdbCls

DocId = 67

DelRefFlag = “Y”

lRetVal=gPDb.DeleteDocument(DocId,DelRefFlag)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

End If
