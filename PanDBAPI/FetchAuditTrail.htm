FetchAuditTrail

Fetches audit log as per criteria given for fetching the audit log.

Public Function FetchAuditTrail(ByVal Category As String, ByVal ObjectId As Long, ByRef ObjectType As String,
ByVal UserIndex As Long, ByVal DateFrom As String, ByVal DateTo As String,ByRef NoOfEntries As Long, ByRef AuditLog() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In

Category

String

'C' for Cabinet level log, 'F' for folder level log, 'D' for document level log and 'A' for Action Item log

In

ObjectId

Long

Index of the object for which audit log is being fetched

In

 

ObjectType

String

Type of the object for which audit log is being fetched - 'D' for document level audit log, 'F' for folder level log, 'C ' for Cabinet level audit log, 'D' for dataclass and 'U' for user level log. 

In

 

UserIndex

Long

Index of the user fetching audit log

In

 

DateFrom

String

The start date from which onwards log is required

In

 

DateTo

String

The date upto which log is required

Out

NoOfEntries

Long

Total number entries fetched

Out

AuditLog

Byte[TAGAUDITLOGEX]

Audit log as byte

 

Description

This function is called for both local and shared cabinet from both admin and desktop.

To get audit log from admin user has to be a member of supervisor group.

 

Return value

Name

Description

NG_SUCCESS

In case of success

-3

Cabinet not connected

-133

User not logged in

 

Example

    Dim Category              As String

    Dim ObjectId              As Log

    Dim ObjectType          As String

    Dim UserIndex            As Long

    Dim DateFrom            As String

    Dim DateTo                As String

    Dim NoOfEntries         As Long

    Dim AuditLog()            As Byte

    Dim gPDb                   As PanDb.pdbCls

    Dim lRetVal                 As Long


    Set gPDb = New PanDb.pdbCls
 

    Category = “F”

    ObjectId =  54

    ObjectType = “F”

    UserIndex = 13

    DateFrom = “11/09/02”

    Dateto      = “11/20/02”

 

  lRetVal=gPDb.FetchAuditTrail(Category,ObjectId,ObjectType,,UserIndex,Datefrom,DateTo,NoOfEntries,AuditLog)

  if lRetVal <> NG_SUCCESS Then

       MsgBox gPDb.ErrDescription, vbInformation

       Screen.MousePointer = vbNormal

       Exit Sub

  End If
