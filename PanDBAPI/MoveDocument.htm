MoveDocument

For moving the document  from one folder to another within a cabinet or across cabinets.

Public Function MoveDocument( ByVal DocId As Long, ByVal DestinationCabId As Integer, ByVal DestFoldId As Long, _
ByVal DuplicateNameFlag As String, ByRef DocumentProp() As Byte, ByVal MoveKeyWord As String, ByVal MoveGlobalIndex As String, ByVal MoveDataClass As String) As Long

 

Parameters

In/out

Name

Data type

Description

In

DocId

Long

Index of the document being moved

In

DestinationCabId

Integer

Index of destination cabinet 

In

DestFoldId

Long

Index of destination folder

In

DuplicateNameFlag

String

Whether duplicate name is to be allowed

In,Out

 

DocumentProp

Byte

Properties of the document in a byte array

In

MoveKeyWord

String

Whether keywords associated with the document also have to be moved

In

MoveGlobalIndex

String

Whether global indexes associated with the document also has to be moved

In

MoveDataClass

String

Whether dataclass associated with the document also has to  be moved

 

Description

This function is called on both local and shared cabinet. It is called when any document is moved to some other folder either in the same cabinet or other cabinet. Document properties are returned after getting successfully moved. If the document is tried to move to some other cabinet then it is  copied rather than moved, in which case 'CopyDocument' is called internally in this case.

 

Return value

Name

Description

 1

In case of success

-2

Invalid operation in  read only cabinet

-3

Cabinet not connected

-31

Duplicate document name

-201

 Insufficient rights for the current operation

 

Example

 Dim DocId                           As Long

 Dim DestinationCabId          As Integer

 Dim DestFoldId                   As Long

 Dim DuplicateNameFlag      As String

 Dim DocumentProp()           As Byte

 Dim MoveKeyWord              As String

 Dim MoveGlobalIndex          As String

 Dim MoveDataClass            As String

 Dim DocInfo                       As TagDocument

 Dim gPDb                         As PanDb.pdbCls


 Set gPDb = New PanDb.pdbCls

 DocId = 89

 DestinationCabId = 2

 DestFoldId = 46

 DuplicateNameFlag = “Y”

 MoveKeyWord = “Y’

 MoveGlobalIndex = “Y”

 MoveDataClass = “Y”

 

DocInfo.DocumentId = 123          

DocInfo.FolderId = 54

DocInfo.NoOfPages = 1

DocInfo.Owner  = 8

DocInfo.DataDefId   = 0

DocInfo.DocumentSize  = 100

DocInfo.CheckOutBy = "Richa"          

DocInfo.AccessType = “I”

DocInfo.DocType  = “txt”

DocInfo.ExpiryDateTime = “11/13/12 1:00:05 PM”

DocInfo.DocName  = “mydoc”

DocInfo.Author = “Nikesh”

DocInfo.CheckInCheckOutFlag = “Y”

    

Redim DocumentProp(len(DocInfo) - 1)

Call CopyMemory(DocumentProp(0),DocInfo,Len(DocInfo))

lRetVal=gPDb.MoveDocument(DocId,DestinationCabId,DestFolderId,DuplicateNameFlag,DocumentProp,MoveKeyWord,MoveGlobalIndex,MoveDataClass)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

     Exit Sub

End If
