GetKeywordListForDocument

Returns the list of keywords associated with document.

Public Function GetKeywordListForDocument(ByVal DocId As Long, ByVal AliasAlso As String, KeywordList() As Byte, _
KeywordCount As Long) As Long

 

Parameters

In/out

Name

Data type

Description

In

DocId

Long  

Document Id

In

AliasAlso

String

If Aliases of Keyword also required

Out

KeywordList

Byte[TagKeyword]

List of  keyword associated with document in the form of the bytearray

Out

KeywordCount

Long

Count of Keyword associated

 

Description

This function is called for both local and shared cabinet in desktop.It is used to fetch the list of all keywords or alias associated with a document.

 

Return value

Name

Description

 1

In case of success

-3

Cabinet not connected

 

Example

Dim DocId                 As Long

Dim AliasAlso           As String

Dim KeywordList()     As Byte

Dim KeywordCount    As Long
Dim gPDb                As PanDb.pdbCls


Set gPDb = New PanDb.pdbCls

DocId  = 67

AliasAlso = “Y”

lRetVal=gPDb.GetKeywordListForDocument(DocId,AliasAlso,KeywordList,KeywordCount)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

End if
