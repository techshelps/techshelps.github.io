SetRights

Set rights on a document or a folder for a user

Public Function SetRights(ByVal ObjectId As Long, ByVal ObjectType As String, ByVal GrpUserCount As Long, ByRef GrpUserRights() As Byte) As Long

 

Parameters

In/out  

Name

Data type

Description

 In

ObjectId

Long

Index of object for which rights are being set

 In

ObjecType

String

Type of Object

 In

 

GrpUsrCount

Long

Number of users

Out

 

GrpUserRights

Byte[TagACL]

Information including the rights to set about the user

 

Description

This function is internally called by the functions PostDocument and PostDocumentForFileFlow.

While posting a document (with which data class is associated) to another cabinet, if the user posting it, is either the owner or supervisor then this function is called to set the rights on the document for the recipient.

 

Return value

Name

Description

1

In case of success

-1

Invalid cabinet name

-3

Cabinet not connected

-47

Invalid Rights String

-131

User does not exist.

-133

User not logged in

 

Example

Dim udtSetProp           as TagACL

Dim  gPdb                  as Pandb.Pdbcls
Dim DDId                    as long

Dim  lRetVal                as long

Dim SetProp()              as byte

 

set gPdb = new Pandb.PdbCls

DDId = 2

udtSetProp.GrpUserId = RecipientUserid

udtSetProp.GrpUserType = "U"

udtSetProp.GrpUsrRights = "100000"

ReDim SetProp(Len(udtSetProp) -1)

 

Call CopyMemory(SetProp(0), udtSetProp, Len(udtSetProp))

lRetVal = gPdb.SetRights(DDId, "T", 1, SetProp)

 

If lRetVal <> 1 Then

            MsgBox gPdb.ErrDescription

Else

            MsgBox "Success"

End If
