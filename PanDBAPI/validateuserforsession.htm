ValidateUserForSession

Validation of user.

ValidateUserForSession(

ByVal CabinetName        As String,

ByVal lUserSessionId      As Long,  

ByVal password              As String,

lUserIndex                      As Long,

sUserName                    As String,

bSuper As String)           As Long

 

Parameters

In/out

Name

Data type

Description

In

CabinetName

String

Name of the cabinet user to trying to login

In

LuserSessionId

Long

User’s login session Id

In

Password

String

Password entered by the user

Out

LuserIndex

Long

User index of the user being validated

Out

SuserName

String

Name of the user being validated

Out

Bsuper

Long

Whether User is a member of Supervisor Group or Not.

 

Description

This function is internally called by the Pandb function SetUserSessionId.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-4

Invalid operation for local cabinet

-10000

Could not connect to server: Insufficient licences

 

Example

Dim lRetVal               As long

Dim CabinetName      As String

Dim lUserSessionId    As long

Dim password            As string

Dim lUserIndex          As long

Dim sUserName        As string

Dim bSuper              As Boolean

CabinetName = “Omnidocs”

lUserSessionId = 193

Password = “Supervisor”

lRetVal = ValidateUserForSession(CabinetName, lUserSessionId, password, lUserIndex,                  UserName, bSuper)

If lRetVal = NG_SUCCESS Then

Msgbox “Success”

Else

            Msgbox "Invalid SessionId. User may not exists or is disconnected."

End If
