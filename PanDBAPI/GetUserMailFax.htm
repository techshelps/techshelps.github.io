GetUserMailFax

Called to retrieve the E-mail and Fax numbers of the user whose index is passed as an input paramter.

Public Function GetUserMailFax(ByVal userId As Long, ByRef MailId As String, ByRef Fax As String ) As Long
 

Parameters

In/out

Name

Data type

Description

In

UserId

Long

Index of the user whose properties are to be fetched

Out

MailId

String

 

Out

Fax

String

 

 

Description

This function is called through Admin when a Fax number or Mail Id is to be fetch for an existing user. Returns the MailId and and Fax of the user.

 

Return value

Name

Description

 1

In case of success

-3

Cabinet not connected

-4

 Invalid Operation

 

Example

Dim userId        As Long

Dim MailId        As String

Dim Fax            As String

Dim lRetVal       As Long

Dim gPDb         As PanDb.pdbCls

 

Set gPDb = New PanDb.pdbCls

UserId = 7

lRetVal=gPDb.GetUserMailFax(userId,MailId,Fax)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

    Screen.MousePointer = vbNormal

End if
