GetDocumentProperties

This call is used to fetch the properties of the particular version of the document.

Public Function GetDocumentProperties(ByVal DocumentId As Long, ByVal VersionNo As Single, ByRef DocumentProp() As Byte, Optional ByVal FolderId As Long = 0) As Long

 

Parameters

In/out

Name

Data type

Description

In

DocumentId

Long

Index of document

In

VersionNo

Single

Version of document

Out

 

DocumentProp

Byte

Properties of document in the form of byte array as a return parameter

In

FolderId

Long

Index of folder containing the document

 

Description

This function is called in desktop in both  local and shared cabinet. It is used to fetch the properties of a particular version of a document. If version is passed as '0' then it will get the properties of the latest version of the document.

 

Return value

Name

Description

 1

In case of success

-3

Cabinet not connected

-4

Invalid operation

 

Example

Dim DocumentId            As Long

Dim VersionNo              As Single

Dim DocumentProp()     As Byte

Dim lRetVal                  As  Long

Dim gPDb                    As PanDb.pdbCls


Set gPDb = New PanDb.pdbCls

DocumentId = 98

VersionNo = 1

lRetVal=gPDb.GetDocumentProperties(DocumentId,VersionNo,DocumentProp)

if lRetVal <> NG_SUCCESS Then

     MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Endif
