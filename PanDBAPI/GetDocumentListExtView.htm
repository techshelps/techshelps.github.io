GetDocumentListExtView

Fetches the documents of the folder whose index is passed in the parameter.
Public Function GetDocumentListExtView(ByVal FolderId As Long,ByVal RecursiveFlag As String, LastDocument() As Byte, ByVal FetchCount As Long, ByVal TotalRecCountFlag As String, ByVal WithThumbNail As String, ByVal Orderby As String, ByVal DocRefFlag As Integer, RetrievedRecCount As Long, TotalRecCount As Long, ResultStr() As Byte) As Long

Parameters

In/out

Name

Data type

Description

In

FolderId

Long

Index of folder

In

 

RecursiveFlag

String

'Y' if  documents are to be fetched from the sub-folders

In

 

LastDocument

Byte[TagDocument]

Byte array containing information related to the last Document fetched

In

FetchCount

Long

Number of records to be fetched in a single batch

In

TotalRecCountFlag

String

Whether total no of records have to be fetched or not

In

 

WithThumbNail

String

Whether document is to be fetched with thumbnail or not

In

 

Orderby

String

Order in which result is desired

In

 

DocRefFlag

String

Whether document references have to be fetched or not

Out

RetrievedRecCount

Long

count of documents fetched

Out

TotalRecCount

Long

Total records fetched in this criteria

Out

ResultStr

Byte[TagDocument]

Result list of documents in the form of byte array

 

Description

Use this to get all document of  the specific folder .

This function is called in desktop from both local and shared cabinet. 'RecursiveFlag' should be set to “Y” if documents belonging to subfolders also has to be fetched. 'DocRefFlag' should be set to “Y” if references also have to be fetched. It supports fetching in batches but the logic here is not same as with the call GetKeywords. If FetchCount is passed as 0 then then it is by default taken as 32000.In this function after making the first call ,the last document fetched in the previous call has to be sent as 'LastDocument' in the subsequent calls. If 'RetrievedRecCount' is less than or equal to 'FetchCount' then this means that there are no more documents left to be fetched.

 

Return value

Name

Description

 1

In case of success

-3

Cabinet not connected

-4

Invalid operation

Dim RetrievedRecCount        As Long

Dim TotalRecCount              As Long

Dim ResultStr()                    As Byte

Dim OrderBy                        As String

Dim gPDb                           As PanDb.pdbCls

     MsgBox gPDb.ErrDescription, vbInformation
