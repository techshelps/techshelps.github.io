SearchFolderEx

Search folders as per given criteria and returns folder list along with folder properties. 

 

Public Function SearchFolderEx(FolderId() As Long, ByVal IncludeSubFolders As String,ByVal FolderName As String, ByVal sOwner As String, ByVal sAuthor As String, ByVal CreationDateRange As String, ByVal AccessedDateRange As String, ByVal RevisedDateRange As String, ByVal ExpirydateRange As String, ByVal SearchOnDD As String, DDCriteria() As Byte, ByVal SearchScope As Integer, ByVal DataDefCount As Long, DataDefList() As Long, ByVal FoldRefFlag As String, ByVal sOrderBy As String, ByVal PrevFolderCount As Long, FoldList() As Long, FolderCount As Long, ByVal StartPos As Long, ByVal FetchCount As Long, TotalRecCount As Long, ResultList() As Byte) As Long

Description

This function is almost same as 'SearchFolderWithData' with two parameters missing which are used to get dataclass field values if attached. This function is called when normal search folder is made. It returns only folder properties of all the folders retrieved.

 

Return value

Name

Description

 1

In case of success

-1

Invalid cabinet name

-3

Cabinet not connected

-201

Insufficient rights for the current operation

-133

User not logged in
