SetCabinetProp

Set cabinet properties                                                                                 

SetCabinetProp(ByRef CabinetProp() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In,Out

CabinetProp

Byte[TagCabinet]

[In]Cabinet properties to be set

[Out]Cabinet Properties set

 

Description                                                                                                                               

This function is called from admin for shared cabinet,though it is capable of handling local cabinet also it is currently not being used for this. It is called when cabinet properties are changed (that is security level is changed,not called for change in rights).In the parameter passed to this function security level is set to 0 if No security is set, it is set to 1 if Cabinet level security is set and is set to 2 if Object Level security is set.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-1

Invalid cabinet name

-2

Local cabinet is read only

-3

Cabinet not connected

 

Example

Dim lRetval                       As Long

Dim udtCabProp               As TagCabinet

Dim bCabProp()                As Byte

udtCabProp.Name = "MyCabinet"

udtCabProp.ImageVolIndex = 0

udtCabProp.CreationDateTime = Now

udtCabProp.Type = “B”

udtCabProp.SiteIndex = 0

udtCabProp.SecurityLevel = 1

ReDim bCabProp(Len(udtCabProp) - 1)

Call CopyMemory(bCabProp(0), udtCabProp, Len(udtCabProp))

lRetval = PanObj.SetCabinetProp(bCabProp)

If lRetval <> 1 Then

               MsgBox PanObj.ErrDescription

Else

                 MsgBox "Success"

End If

End Sub
