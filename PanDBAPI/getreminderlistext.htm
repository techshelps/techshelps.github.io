GetReminderListExt        

Get information about routes

GetReminderListExt(ByRef SearchCriteria() As Byte,

                                ByRef LastReminder() As Byte,

                                ByVal GetAllForUser As String,

                                ByVal ForUserId As Integer,

                                ByVal SetByUser As Integer,

                                ByVal NoOfRecordsToFetch As Long,

                                ByRef RetrievedCount As Long,

                                ByRef TotalNoOfRecords As Long,

                                ByRef Results() As Byte

                                ) As Long

 

Parameters

In/out

Name

Data type

Description

In

SearchCriteria

Byte[TagReminderEx]

Search criteria for reminders

In

LastReminder

Byte[TagReminderEx]

Last reminder fetched

In

GetAllForUser

String

 

In

ForUserId

Integer

Index of user for which generated reminders are to fetched

In

SetByUser

Integer

Index of user by whom reminders generated should have been set

In

NoOfRecordsToFetch

Long

Batch count

Out

RetrievedCount

Long

Reminders fetched

Out

TotalNoOfRecords

Long

Total numbers of reminders in the category

Out

Result

Byte[TagReminderEx]

Reminders generated

 

Description

This function is called in desktop in shared cabinet only. It is called when user tries to view

the reminders generated for him. It supports fetching of reminders in batches. If SetByUser

is passed as 0 than reminders set by all are fetched. If TotalNoOfRecords is greater than

RetrievedCounts in return values than this means that more reminders are remaining to be

Fetched.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-3

Invalid cabinet name

-4

Invalid operation for local cabinet

-47

Invalid Object Index

 

Example

Dim SearchCriteria() As Byte

Dim LastReminder() As Byte

Dim GetAllForUser As String

Dim ForUserId As Integer

Dim SetByUser As Integer

Dim NoOfRecordsToFetch As Long

Dim RetrievedCount As Long

Dim TotalNoOfRecords As Long

Dim Results() As Byte

Dim SearchInfo  As TagReminderEx

SearchInfo.RemDateTime = Format$(Now, mDateFormat)

GetAllForUser = “T”

ForUserId =  8

SetByUser = 0

NoOfRecordsToFetch = 10

Call CopyMemory(SearchCriteria(0),SearchInfo,Len(SearchInfo));

lRetVal=gPDb(miCurCabId).GetReminderListExt(SearchCriteria,LastReminder,GetAllForUser,ForUserId,SetByUser,NoOfRecordsToFetch,RetrievedCount,TotalNoOfRecords,Result,SearchInfo)

  if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb(miCurCabId).ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Endif
