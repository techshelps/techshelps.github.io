ChangeIndexValue

Called to change the value of  global index associated with document .

ChangeIndexValue (ByVal DocumentId As Long, ByRef IndexList() As Byte) As Long

 

Parameters

In/out             

Name

Data type

Description

In

DocumnetId

 Long

Index of document,with which the associated global indexes are being changed

In

IndexList  

 

Byte[TAGSETINDEXVALUE]

Byte array specifying the attributes for the global index

 

 

Description

This function is called for both local and shared cabinet.

It is called when any global indexes values attached with a document are changed.

 

Return value

Name

Description

 1

In case of success

-1

Invalid operation

-2

Invalid operation in read only cabinet

-3

Cabinet not connected

 

Example

Dim DocId                                 as Long

Dim btChangeIndexValues()       as Byte

Dim gPDb                                 as PanDb.pdbCls
Dim lRetVal                               as long 

Dim udtChangeIndexValueInfo()  as TAGSETINDEXVALUE

 

Set gPDb = New PanDb.pdbCls

Redim udtChangeIndexValueInfo(0)

 

udtChangeIndexValueInfo(0).lIndexId = 4

udtChangeIndexValueInfo(0).iIndexType = 3

udtChangeIndexValueInfo(0).sNewIndexValue = "6"
udtChangeIndexValueInfo(0).sOldIndexValue = "4" 

 

ReDim btChangeIndexValues(Len(udtChangeIndexValueInfo(0)) - 1)
CopyMemory btChangeIndexValues(0), udtChangeIndexValueInfo(0), Len(udtChangeIndexValueInfo(0)

lRetVal=gPDb.ChangeIndexValue(DocId,btChangeIndexValues)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

    Screen.MousePointer = vbNormal

End if
