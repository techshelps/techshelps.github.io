UnAuthoriseKeywords

Unauthorize all the key words given in the list

 

Public Function UnAuthoriseKeywords(ByRef Keywords() As String, ByVal KeywordCount As Long) As Long

 

Parameters

In/out

Name

Data type

Description

In/Out

Keywords

String

Specifies an array for the keyword structure

In

KeywordCount

Long

Specifies the Number of Keywords to

Unauthorise.

 

Description

This function is called when to unauthorize authorized keyword .This is called in both local and shared cabinet. For local cabinet it is called from Omnidocs Client and for shared cabinet it is called from admin.

 

Return value

Name

Description

 1

 

In case of success

-2

 

Invalid operation in readonly cabinet

-3

Cabinet not connected

 

Example

    Dim lRetVal                     As Long

    Dim sKeywords()              As String

    Dim lKeywordCount          As Long

    ReDim sKeyword(5)          As String

   Dim gPdb as Pandb.Pdbcls

    Set gPdb = new Pandb.PdbCls

    sKeyword(0) = "keyword1"

    sKeyword(1) = "keyword2"

    sKeyword(2) = "keyword3"

    sKeyword(3) = "keyword4"

    sKeyword(4) = "keyword5"

    sKeyword(5) = "keyword6"

    lKeywordCount = 6

    lRetVal = gPdb..UnAuthoriseKeywords(sKeywords, lKeywordCount)

    If lRetVal <> 1 Then

        MsgBox gPdb.ErrDescription,vbinformation

    Else

        MsgBox " Success",vbinformation

    End If
