GetUserSessionId

Used to get the SessionId of the User.

GetUserSessionId(ByVal CabinetName           As String,

                              ByVal sUserName           As String,

                              lUserSessionId As Long) As Long

 

Parameters

In/out

Name

Data type

Description

In

Cabinetname

String

Name of cabinet for login

In

SuserName

String

Username trying to login

Out

LuserSessionId

Long

Session Id of user as return parameter

 

Description

This function is called from OmniAcquire. This function is called whenever user tries to login in any cabinet. It first checks if user is already logged in into the same cabinet (through some other Application), if yes it returns the previous sessionId.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-3

Cabinet not connected

-4

Invalid Operation

-1000

Could not connect to server: Insufficient licences

Could not create object of NGAuthentication DLL

-201

Insufficient rights for the current operation

 

Example

Dim CabinetName As String

Dim sUserName As String

Dim lUserSessionId As Long

CabinetName = “OurSharedCabinet”

SUserName = “Deepak”

lRetVal=gPDb(miCurCabId).GetUserSessionId(CabinetName,sUserName,lUserSessionId)

  if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb(miCurCabId).ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Endif
