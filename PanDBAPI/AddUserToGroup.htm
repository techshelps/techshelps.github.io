AddUserToGroup

Adds user to the paticulat group.

Public Function AddUserToGroup(ByVal userId As Long,ByVal GroupId  As Long) As Long

 

Parameters

In/out

Name

Data type

Description

In

UserId

Long

User Index

In  

GroupId

Long

Group Index

 

Description

This function call is used to add the user to the particular group.

 

Return value

Name

Description

1

In case of success

-3

Cabinet not connected.

 

Example

Dim userId       As Long

Dim GroupId    As Long

Dim lRetval      As Long

Dim gPDb       As PanDb.pdbCls

 

Set gPDb = New PanDb.pdbCls

UserId = 8

GroupId = 3

lRetVal=gPDb.AddUserToGroup(UserId,GroupId)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

endif
