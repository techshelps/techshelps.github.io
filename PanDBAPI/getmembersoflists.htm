GetMembersOfLists          

Returns the users of a list

GetMembersOfLists(ByRef Lists() As Byte,

                                 ByRef Members() As Byte,

                                 ByVal lOrderBy As Long,

                                 lMemberCount() As Long) As Long

 

Parameters

In/out

Name

Data type

Description

In,Out

Lists

Byte[TagList]

[In]List(s) for which user

[Out]List with startpos and EndPos set

In

Members

Byte[TagMember]

List of members fetched

Out

LorderBy

Byte

Order in which result is desired

Out

LmemeberCount

Long

Total number of members fetched

 

Description

This function is called in shared cabinet only in desktop. It fetches all members belonging to all lists in one common byte array,with each list’s StartMemberPos and EndMemberPos set to their respective members in that common member array.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-3

Cabinet not connected

-4

Invalid operation

 

Example

Dim Lists()                         As Byte

Dim Members()                  As Byte

Dim lOrderBy                     As Long

Dim lMemberCount()          As Long

Dim ListInfo()                    As TagList

ListInfo(0).ListIndex = 12

ListInfo.ListName = “Mylist”

LOrderBy =  1

Call CopyMemory(Lists(0),ListInfo,Len(ListInfo))

lRetVal=gPDb(miCurCabId).GetMembersOfList(Lists,Members,lOrderBy,lMemberCount)

  if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb(miCurCabId).ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Endif
