PurgeAuditTrail

Deletes the desired log permanently from database.

Public Function PurgeAuditTrail(ByVal Category As String, ByVal DateFrom As String, ByVal DateTo As String) As Long

 

Parameters

In/out

Name

Data type

Description

In

Category

String

Level of log to be purged - 'C for cabinet, 'D' for document, 'F' for folder and 'A' for Action Item.

In

DateFrom

String

The start date from which onwards log is to be purged

In

Dateto

String

The end date upto which log is to purged

 

Description

This function supports call for both shared and local cabinet. It is called in admin by supervisor and deletes the log for the supplied category between the given dates

   

Return value

Name

Description

 1

In case of success

-1

Invalid cabinet name

-2

Invalid operation in read-only cabinet

-3

Cabinet not connected

-201

Insufficient rights for the current operation

-133

User not logged in

 

Example

    Dim Category                     As String * 1

    Dim StartDate                    As String

    Dim EndDate                     As String

    Dim lRetVal                       As  Long

    Dim gPDb                         As PanDb.pdbCls

 

    Set gPDb = New PanDb.pdbCls

    Category = “F”

    StartDate  = “11/01/02”

    EndDate   = “11/20/02”

   lRetVal=gPDb.PurgeAuditTrail(Category,StartDate,EndDate)

   if lRetVal <> NG_SUCCESS Then

       MsgBox gPDb.ErrDescription, vbInformation

         Screen.MousePointer = vbNormal

        Exit Sub

    End If
