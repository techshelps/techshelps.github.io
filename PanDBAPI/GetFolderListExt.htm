GetFolderListExt

To get the list of folder in a specific folder or the whole cabinet.

Public Function GetFolderListExt(ByVal FolderIndex As Long, ByVal RecursiveFlag As String, ByVal StartPos As Long,ByVal FetchCount As Long, ByVal TotalRecCountFlag As String, ByVal Orderby As String, ByVal FolderType As String, ByRef RetrievedRecCount As Long, ByRef TotalRecCount As Long, ByRef ResultStr() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In

FolderIndex

Long

Index of folder

In

 

RecursiveFlag

String

'Y' for fetching folder list recursively ie sub folders also

In

 

StartPos

Long

Starting position for fetching

In

 

FetchCount

Long

Number of records to be fetched each time

In

 

TotalRecCountFlag

String

Whether 'TotalRecCount' has to be fetched

In

 

Orderby

String

Order in which result is desired

In

 

 FolderType

String

The type of folders to be fetched

Out

RetrievedRecCount

Long

Records fetched

Out

 

TotalRecCount

Long

Total records in this criteria

Out

ResultStr

Byte

Byte array specifying the list of folders fetched along with their properties

 

Description

To get the list of folders for particular folder or in the cabinet as a whole. If 'FolderIndex' is 0 then it will get folder list of  cabinet. It will fetch the folder list in batches which is specified by parameter 'FetchCount'.By 'FolderType' we can specify which type of folders are to be retrieved 'OrderBy' specify the order in which we want folder result to be sorted.

 

Return value

Name

Description

 1

In case of success

-3

Cabinet not connected

 

Example :

Dim gPDb                         As PanDb.pdbCls

Dim lFolderId                     As Long

Dim sOrderBy                    As String

Dim lRetrievedRecCount     As Long

Dim lTotalRecCount           As Long

Dim btFolderList()              As Byte

Dim lRetVal                       As Long

Dim udtaFolderList()           As TagFolder

Dim  lFetchcount                As Long

Dim lTempRetCount            As Long


Set gPDb = New PanDb.pdbCls

lFolderId = 2

sOrderBy = 'N'         'for sorting on folder name

lFetchcount = 10

ReDim udtaFolderList(0) As TagFolder

lRetVal = gPDb.GetDocumentListExtView (lFolderId, “N”, 0, lFetchcount, “Y”, sOrderBy,"", lRetrievedRecCount ,lTotalRecCount, btFolderList)

If lRetVal <> NG_SUCCESS Then

      MsgBox gPDb.ErrDescription, vbInformation

      GetFolderList = gPDb.ErrNumber

      Exit Function

ElseIf lRetrievedRecCount > 0 Then

      If lRetrievedRecCount > lFetchCount And lFetchCount > 0 Then

         lTempRetCount = lRetrievedRecCount - 1

      Else

         lTempRetCount = lRetrievedRecCount

      End If

ElseIf lRetrievedRecCount = 0 Then

      GetFolderList = NG_SUCCESS

      Exit Function

End If

ReDim udtaFolderList(lTempRetCount - 1) 

   

For lCount = 0 To lTempRetCount - 1

      CopyMemory udtaFolderList(lCount), btFolderList(lCount * Len(udtaFolderList(0))), Len(udtaFolderList(0))

Next lCount
