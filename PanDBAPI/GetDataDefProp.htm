GetDataDefProp

This function fetches the properties of the dataclass whose index is passed as an input parameter.

Public Function GetDataDefProp(ByVal DataDefId As Long, ByRef DataDefName As String, ByRef DataDefComment As String, ByRef Unused As String, ByRef Rights As String, ByRef IndexCount As Long, ByRef IndexList() As Byte) As Long

                

Parameters

In/out

Name

Data type

Description

In

 

DataDefId

Long

Index of dataclass whose properties are to be retrieved

Out

 

DataDefName

String

Name of dataclass whose properties are to be retrieved

Out

 

DataDefComment

String

Comment,if any associated with dataclass

Out

Unused

String

The useful data

Out

 

Rights

String

Rights on that dataclass

Out

 

IndexCount

Long

Field Count of that dataclass

Out

 

Indexlist

Byte

Byte array specifying the attributes for each field of the dataclass 

 

Description

This function is called form both admin and desktop and for both local and shared cabinet. It is used to fetch all information concerned with a dataclass along with the field information. 

 

Return value

Name

Description

 1

In case of success

-2

Invalid operation in readonly cabinet

-3

Cabinet not connected

-4

Invalid operation

 

Example

Dim DataDefId                  As Long

Dim DataDefName            As String

Dim DataDefComment      As String

Dim Unused                     As  String

Dim IndexList()                 As Byte

Dim Rights                       As String

Dim IndexCount                As Long

Dim lRetVal                      As Long

Dim gPDb                        As PanDb.pdbCls

Set gPDb = New PanDb.pdbCls

DataDefId = 5

 

lRetVal=gPDb.GetDataDefProp(DataDefId,DataDefName,DataDefComment,Unused,Rights.IndexCount,IndexList)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

    Screen.MousePointer = vbNormal

Endif
