MoveReference

This function is used to move the reference either folder or document from one folder to another folder.

Public Function MoveReference(ByVal ObjectId As Long, ByVal ObjectType As String, ByVal ParentId As Long, ByVal DestinationId As Long, ByRef ObjectProp() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In

ObjectId

Long

Properties of form

In

ObjectType

String

Type of object

In

ParentId

Long

Index of Parent Folder

In

DestinationId

Long

Index of destination

Out

 

ObjectProp

Byte

Properties of the object after move

 

Description

This function is called on both local and shared cabinet. It is called when reference of any object is moved. It returns the properties of object after it has been moved successfully in the output parameter 'ObjectProp'.

 

Return value

Name

Description

 1

In case of success

-2

Invalid operation in  read only cabinet

-3

Cabinet not connected

-201

Insufficient rights for the current operation

 

Example

 Dim ObjectId              As Long

 Dim  ObjectType        As String

 Dim  ParentId             As Long

 Dim DestinationId       As Long

 Dim ObjectProp()       As Byte

 Dim lRetval                As Long

 Dim gPDb                 As PanDb.pdbCls


 Set gPDb = New PanDb.pdbCls

 ObjectId = 69

 ObjectType = “D”

 ParentId = 40

 DestinationId = 48

 lRetVal=gPDb.MoveReference(ObjectId,ObjectType,ParentId,DestinationId,ObjectProp)

 if lRetVal <> NG_SUCCESS Then

     MsgBox gPDb.ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

     Exit Sub

 End If
