AddOnProcessRequest

Used to pass a reference object to AddOn.

Public Function AddOnProcessRequest(sObjToUse As String,

XmlIn As String,

XmlOut As String)

 

Parameters

In/out

Name

Data type

Description

In

SobjToUse

String

Object of AddOn.

In

Xmlin

String

Xml Input

Out

XmlOut

String

Xml Output

 

Description

This function is called from OmniDocs Desktop. It is used to pass the connection object of database to AddOn.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

Error

In case of Failure

 

Example

Dim sObjToUse             As String

Dim XmlIn                      As String

Dim XmlOut                   As String

Dim lRetVal                   As  long

sObjToUse   = “FileView.clsFileView”

XmlIn = According To the AddOn

lRetVal = gPanDb.AddOnProcessRequest(sObjToUse, XmlIn, XmlOut )

If lRetVal <> NG_SUCCESS Then

        MsgBox gPandDb.ErrDescription, vbInformation

        Screen.MousePointer = vbNormal

        Exit Sub

Else

        MsgBox “Success”        

End if
