GetGroupProp

Fetches properties of a group.

Public Function GetGroupProp(ByVal GroupId As Long, ByRef GroupProp() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In

GroupId

Long

Index of Group

Out

 

GroupProp

Byte[TagGroup]

Properties of group in the form of byte array

 

Description

This function is called form admin for shared cabinet. This  is called when group properties are required .It takes the group index and returns the properties in the form of a byte array.

 

Return value

Name

Description

 1

In case of success

-3

Cabinet not connected

-4

Invalid operation

 

Example

Dim GroupId                         As Long

Dim GroupProp()                   As Byte

Dim lRetVal                          As Long

Dim gPDb                            As PanDb.pdbCls


Set gPDb = New PanDb.pdbCls

GroupId = 2

 

lRetVal=gPDb.GetGroupProp(GroupId,GroupProp)

if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb.ErrDescription, vbInformation

    Screen.MousePointer = vbNormal

Endif
