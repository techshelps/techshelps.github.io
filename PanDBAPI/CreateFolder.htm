CreateFolder

 

It is called to create a new folder with properties passed

Public Function CreateFolder(FolderProp() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In/Out

 

FolderProp

Byte [TagFolder]

[In] Folder properties to be set for the new created folder [Out]Properties set

 

Description

This function is called in both local and shared cabinet .It is passed with the properties of  the folder that has to be created and returns the properties which have been set by this call.

 

Return value

Name

Description

 1

In case of success

-1

Invalid cabinet name

-2

Local cabinet is read-only

-3

Cabinet not connected

-12

Invalid folder Name

-201

Insufficient rights for the current operation

-133

User not logged in

 

Example

Dim FolderInfo       As   TagFolder

Dim Folder            As   Byte

Dim gPDb             As   PanDb.pdbCls


Set gPDb = New PanDb.pdbCls

FolderInfo.ParentId  = 0

FolderInfo.Owner  = 7

FolderInfo.SiteIndex = 0

FolderInfo.LockedByUser  = 7

FolderInfo.AccessType = “I”

FolderInfo.FolderType  = “G”

FolderInfo.EnableLog    =”Y”

FolderInfo.UserRights   = “101011”

FolderInfo.ExpiryDateTime = “11/13/12 1:00:03 PM”

FolderInfo.FolderName  = “MyDocs”

FolderInfo.Comment  = “userFolder”

FolderInfo.OwnerName  = “Rashmi”

 

Redim Folder(len(FolderInfo) - 1)

Call CopyMemory(Folder(0), FolderInfo, Len(FolderInfo))

 

lRetVal = gPDb.CreateFolder(Folder)

If lRetVal <> NG_SUCCESS Then

        MsgBox gPDb.ErrDescription, vbInformation

        Screen.MousePointer = vbNormal

        Exit Sub

End If
