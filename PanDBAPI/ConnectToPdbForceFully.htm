ConnectToPdbForceFully

Connects the user to Cabinet forcefully if already connected.

Public Function ConnectToPdbForceFully(ByVal CabinetName As String,

ByVal UserName      As String,

ByVal password        As String,

ByVal RelogFlag       As String,

FolderCount             As Long,

FolderList() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In

CabinetName

String

Cabinet name with which to connect

In

 

UserName

String

User who wants to connect

In

 

Password

String

Password for that user

In

 

RelogFlag

String

Y or N

Out

FolderCount

Long

Number of folders in the cabinet connected

Out

FolderList

Byte[TagFolder]

List of all the folders which are in the     cabinet connect

 

Description

This function is called when some other user is already connected with the cabinet and we get error –6019  in connecttopdb cabinet  and after that we try to connect forcefully after disconnecting previous user.

All the parameters are same as connectToPdb except RelogFlag which is set to “Y”

to connect forcefully,that is other user is disconnected.

 

Return value

Name

Description

NG_SUCCESS

In case of success

-1

Invalid Ini entry

-3

Cabinet not connected

-7

 

Incorrect User name password

-112

 

Ini entry not found

-201

Insufficient rights for the current operation

-133

User not logged in

-999

Installation Expired

Dim CabinetName As String

Dim UserName     As String

Dim password       As String

Dim FolderCount   As Long

Dim FolderList()    As Byte

Username = “Abhiseik”
