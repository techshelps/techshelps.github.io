GetFTSText

Called to fetch the Full Text Search text of a document and coordinates of the words of that text.

Public Function GetFTSText(By Val lDocId, ByRef sOcrBuffer, ByRef sCoOrdinatesBuffer) As Long

Parameters

In/out  

Name           

Data type

Description

In

lDocId

Long

Document  index of the folder.

Out

 

sOcrBuffer

String

String buffer cotaining the OCRed text of the document

Out

sCoOrdinatesBuffer

String

 

String buffer cotaining the coordinates of all words of the OCRed text

 

Description

This function is called when OCRed text and coordinates of all the words of the text is to be fetched for a specific document..

 

Return value

Name

Description

 1

In case of success

-1

In caseof failure

 

Example

Dim lRetVal                        As Long

Dim gPDb                           As PanDb.pdbCls

Dim hFile                            As Long

Dim sOcrBuffer                   As string

Dim sCoOrdinatesBuffer     As string

 

Set gPDb = New PanDb.pdbCls

lRetVal = gPDb(iDBindex).GetFTSText(lDocId, sOcrBuffer, sCoOrdinatesBuffer)

 

If lRetval <> NG_SUCCESS Then

        Screen.MousePointer = vbNormal

        MsgBox gPDb.ErrDescription, vbInformation

        Exit Sub

 End If
