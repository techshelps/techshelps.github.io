AssociateIndexwithDoc

Associates global Index with the document.

Public Function AssociateIndexwithDoc(ByVal DocumentId As Long,ByRef IndexList() As Byte) As Long

 

Parameters

In/out

Name

Data type

Description

In   

DocumentId

Long

Document Index

In   

IndexList

Byte (TagIndex)

Specifies the byte array for attributes for each global index

 

Description

This function call is used to associates global index with the document.

 

Return value

Name

Description

 1

In case of success

-3

Cabinet not connected

-2

Invalid operation in read only cabinet

 

Example

Dim btIndexValues()              As Byte

Dim udtIndexValueInfo()         As TagIndex

Dim lDocId                            As Long

Dim gPDb                             As PanDb.pdbCls


Set gPDb = New PanDb.pdbCls

LDocId = 100    ‘ Document ID

 

‘udtIndexValueInfo contains global index info

udtIndexValueInfo.lIndexId = 5
udtIndexValueInfo.iIndexType = 3
udtIndexValueInfo.sClassType = "G" 
udtIndexValueInfo.sIndexValue = "12"

 

'Copy the indexes to be added into a byte array
ReDim btIndexValues((UBound(udtIndexValueInfo) + 1) * Len(udtIndexValueInfo(0)) - 1)

For iCount = 0 To UBound(udtIndexValueInfo)

    CopyMemory btIndexValues(iCount * Len(udtIndexValueInfo(0))), udtIndexValueInfo(iCount),     Len(udtIndexValueInfo(0))

Next iCount

      

 'Associate new indexes with their corresponding values

lRetVal = gPDb.AssociateIndexwithDoc(lDocId, btIndexValues)

If lRetVal <> 1 Then

      MsgBox gPDb.ErrDescription, vbInformation

      Exit Function

End If
