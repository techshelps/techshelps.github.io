GetDocIndexForName

Fetches index of a document corresponding to its name                       

GetDocIndexForNameEx(

                ByVal FolderId                 As Long,

                ByVal DocumentName     As String,

                ByVal DocumentType       As String,

                ByRef DocIndex As Long) As Long

 

Parameters

In/out

Name

Data type

Description

In

FolderId

Long

Index of folder containing the document

In

DocumentName

String

Name of document

In

DocumentType

Single

Type of document

Out

DocIndex

Long

Index of document as return parameter

 

Description

This function is called in WhiteHall and is used to fetch the index of a document corresponding to its name. It can be called for both local and shared cabinet.

 

Return value 

Name

Description

NG_SUCCESS

In case of success

-2

Invalid operation in read only cabinet

-3

Cabinet not connected

-4

Invalid operation

 

Example

Dim FolderId              As Long

Dim DocumentName  As String

Dim DocumentType   As String

Dim DocIndex           As Long

FolderId = 67

DocumentName = “Photograph.tif”

DocumentType = “I”

lRetVal=gPDb(miCurCabId).GetDocIndexForNameEx(FolderId,DocumentName,DocuemntType,DocIndex)

    if lRetVal <> NG_SUCCESS Then

    MsgBox gPDb(miCurCabId).ErrDescription, vbInformation

     Screen.MousePointer = vbNormal

Endif
