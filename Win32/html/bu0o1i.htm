TBSAVEPARAMS

The TBSAVEPARAMS function specifies the location in the registry where the TB_SAVERESTORE message stores and retrieves information about the state of a toolbar.

HKEY hkr;

LPCTSTR pszSubKey;

} TBSAVEPARAMS;
