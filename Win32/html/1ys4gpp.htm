LVN_COLUMNCLICK

The LVN_COLUMNCLICK notification message notifies a list view control's parent window that a column was clicked. This notification message is sent in the form of a WM_NOTIFY message.

LVN_COLUMNCLICK

pnmv = (NM_LISTVIEW FAR *) lParam;

Parameters

pnmv

Pointer to an NM_LISTVIEW structure. The iItem member is - 1, and the iSubItem member identifies the column. All other members are zero.

Return Values

No return value.

See Also

NM_LISTVIEW, WM_NOTIFY
