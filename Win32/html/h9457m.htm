Icon Types

The operating system provides a set of standard icons that are available for any application to use at any time. The Windows header files contain identifiers for the standard icons ¾ the identifiers begin with the "IDI_" prefix.

Each standard icon has a corresponding default image associated with it. The user can replace the default image associated with any standard cursor at any time.
