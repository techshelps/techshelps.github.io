TreeView_GetItem

The TreeView_GetItem macro retrieves some or all of a tree-view item's attributes. You can use this macro or explicitly send the TVM_GETITEM message.

BOOL TreeView_GetItem(

hwnd,  
pitem  
);  

Parameters

hwnd

Handle to the tree-view control.

pitem

Pointer to a TV_ITEM structure that specifies the information to retrieve and receives information about the item. When the TVM_GETITEM message is sent, the hItem member identifies the item to retrieve information about and the mask member specifies the attributes to retrieve.

If mask specifies the TVIF_TEXT value, the pszText member must contain the pointer to the buffer that receives the item text and the cchTextMax member must specify the size of the buffer.

If mask specifies the TVIF_STATE value, the stateMask member indicates which item states are to be returned.

Return Values

Returns TRUE if successful or FALSE otherwise.
