ImageList_GetImageInfo

The ImageList_GetImageInfo function retrieves information about an image.

BOOL ImageList_GetImageInfo(

HIMAGELIST himl,  
int i,  
IMAGEINFO FAR *pImageInfo  
);  

Parameters

himl

Handle to the image list.

i

Index of the image.

pImageInfo

Pointer to an IMAGEINFO structure that receives information about the image. The information in this structure can be used to directly manipulate the bitmaps for the image.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero.
