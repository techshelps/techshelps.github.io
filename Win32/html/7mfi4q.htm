TCM_INSERTITEM

The TCM_INSERTITEM message inserts a new tab in a tab control. You can send this message explicitly or by using the TabCtrl_InsertItem macro.

TCM_INSERTITEM

wParam = (WPARAM) (int) iItem; \\index of the new tab

lParam = (LPARAM) (const TC_ITEM FAR*) pitem; \\pointer to struct specifying tab attributes

Parameters

iItem

Index of the new tab.

pitem

Pointer to a TC_ITEM structure that specifies the attributes of the tab.

Return Values

Returns the index of the new tab if successful or - 1 otherwise.

See Also

TabCtrl_InsertItem, TC_ITEM
