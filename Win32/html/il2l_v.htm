IsValidSecurityDescriptor

The IsValidSecurityDescriptor function validates a SECURITY_DESCRIPTOR structure. Validation is performed by checking the revision level of each component in the security descriptor.

BOOL IsValidSecurityDescriptor(

PSECURITY_DESCRIPTOR pSecurityDescriptor // address of security descriptor
);  

Parameters

pSecurityDescriptor

Points to a SECURITY_DESCRIPTOR structure that the function validates.

Return Values

If the structure of the security descriptor is valid, the return value is nonzero.

If the structure of the security descriptor is not valid, the return value is zero. To get extended error information, call GetLastError.
