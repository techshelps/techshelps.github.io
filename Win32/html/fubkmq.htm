CharPrev

The CharPrev function returns a pointer to the preceding character in a string. This function supersedes the AnsiPrev function.

LPTSTR CharPrev(

LPCTSTR lpszStart, // pointer to first character
LPCTSTR lpszCurrent // pointer to current character
);  

Parameters

lpszStart

Pointer to the beginning of the string.

lpszCurrent

Pointer to a character in a null-terminated string.

Return Values

If the function succeeds, the return value is a pointer to the preceding character in the string, or to the first character in the string if the lpszCurrent parameter equals the lpszStart parameter.
