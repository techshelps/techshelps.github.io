DOC_INFO_1

The DOC_INFO_1 structure describes a document that will be printed.

typedef struct _DOC_INFO_1 { // dci1

LPTSTR pDocName;

LPTSTR pOutputFile;

LPTSTR pDatatype;

} DOC_INFO_1;

Members

pDocName

Points to a null-terminated string that specifies the name of the document.

pOutputFile

Points to a null-terminated string that specifies the name of an output file.

pDatatype

Points to a null-terminated string that identifies the type of data used to record the document.
