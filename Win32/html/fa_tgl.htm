ImageNtHeader

The ImageNtHeader function locates the IMAGE_NT_HEADERS structure in a Win32 image and returns a pointer to the data.

PIMAGE_NT_HEADERS ImageNtHeader(

IN LPVOID ImageBase  
);  

Parameters

ImageBase

The base address of an image that is mapped into memory by a call to the MapViewOfFile function.

Return Values

If the function succeeds, the return value is a pointer to an IMAGE_NT_HEADERS structure.

If the function fails, then the return value is NULL. To retrieve extended error information, call GetLastError.

Remarks

The ImageNtHeader function locates the IMAGE_NT_HEADERS structure in a Win32 image and returns a pointer to the data.
