RFC1157VarBindList

The RFC1157VarBindList structure represents an SNMP variable bindings list as defined in RFC 1157.

typedef struct {
RFC1157VarBind *list;
UINT len;
} RFC1157VarBindList;

Members

list

A pointer that references an array to access individual variable bindings.

len

Contains the number of variable bindings in the list.

For more information, see RFC 1157.
