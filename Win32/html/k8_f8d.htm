LVN_DELETEITEM

The LVN_DELETEITEM notification message notifies a list view control's parent window that an item was deleted. This notification message is sent in the form of a WM_NOTIFY message.

LVN_DELETEITEM

pnmv = (NM_LISTVIEW FAR *) lParam;

Parameters

pnmv

Pointer to an NM_LISTVIEW structure. The iItem member identifies the deleted item, and the other members are zero.

Return Values

No return value.

See Also

NM_LISTVIEW, WM_NOTIFY
