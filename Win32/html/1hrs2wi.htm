CF_HDROP Format

The global memory object contains a DROPFILES structure. If the object was copied to the clipboard as part of a drag and drop operation, the pt member of DROPFILES includes the coordinates of the point where the drop occurred. The pFiles member is the offset to a double - null-terminated list of filenames. An application can retrieve information from the data object by passing the object's handle to the DragQueryFile and DragQueryPoint functions.
