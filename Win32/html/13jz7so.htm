SOCOLUMN

Contains information about the columns in a spreadsheet.

typedef struct SOCOLUMNtag {

WORD wStructSize; // see below

LONG dwWidth; // see below

char szName[40]; // see below

} SOCOLUMN, VWPTR * PSOCOLUMN;

Members

wStructSize

Size, in bytes, of the structure.

dwWidth

Width, in characters, of the column.

szName

Null-terminated string specifying the name of the column.
