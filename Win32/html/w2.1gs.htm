ListView_Update

The ListView_Update macro updates a list view item. If the list view control has the LVS_AUTOARRANGE style, this macro causes the list view control to be arranged. You can use this macro or explicitly send the LVM_UPDATE message.

BOOL ListView_Update(

HWND hwnd,  
int iItem  
);  

Parameters

hwnd

Handle to the list view control.

iItem

Index of the item to update.

Return Values

Returns TRUE if successful or FALSE otherwise.
