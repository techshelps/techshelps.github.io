IMN_SETSENTENCEMODE

The IMN_SETSENTENCEMODE message is sent when the sentence mode of the input context is updated. The application receives this notification message through the WM_IME_NOTIFY message.

Return Values

No return value.

Remarks

The application can get information about the sentence mode by using the ImmGetConversionStatus function.
