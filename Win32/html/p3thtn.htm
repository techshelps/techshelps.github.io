PROVIDOR_INFO_1

The PROVIDOR_INFO_1 structure identifies a print provider.

typedef struct _PROVIDOR_INFO_1 { // pi1

LPTSTR pName;

LPTSTR pEnvironment ;

LPTSTR pDLLName ;

} PROVIDOR_INFO_1;

Members

pName

Points to a null-terminated string that is the name of the print provider.

pEnvironment

Points to a null-terminated environment string specifying the environment the provider dynamic-link library (DLL) is designed to run in.

pDLLName

Points to a null-terminated string that is the name of the provider DLL.
