MENUHELPID

The MENUHELPID structure is the final data written to the RT_MENU resource for a menu or submenu if the resInfo member of the PopupMenuItem structure is set to MFR_POPUP.

typedef struct MENUHELPID {

DWORD helpID;

} MENUHELPID;

Member

helpID

Specifies a numeric expression indicating the identifier used to identify the menu during WM_HELP processing.
