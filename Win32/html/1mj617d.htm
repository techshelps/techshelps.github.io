SERVER_INFO_1016

The SERVER_INFO_1016 structure contains information about the specified server, including name, platform, type of server, and associated software.

typedef struct _SERVER_INFO_1016 {
BOOL sv1016_hidden;
} SERVER_INFO_1016, *PSERVER_INFO_1016, *LPSERVER_INFO_1016;

Members

sv1016_hidden

Determines whether the server is visible to other computers in the same network domain. This member can be one of the following values:

Value Meaning
SV_VISIBLE The server is visible.
SV_HIDDEN The server is not visible.
