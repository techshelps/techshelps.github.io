Ported LAN Manager Function Groups

Ported LAN Manager functions use the standard-call calling convention and return a DWORD status value. The following modifiers are used in the function declarations:

#define NET_API_STATUS DWORD

#define NET_API_FUNCTION __stdcall

The ported LAN Manager functions can be divided into the following groups:

·Access functions

·Alert functions

·ApiBuffer functions

·Audit functions

·Configuration functions

·Error Logging functions

·File functions

·Group functions

·Local Group functions

·Message functions

·Remote Utility functions

·Replicator functions

·Schedule functions
