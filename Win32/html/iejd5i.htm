TBN_ENDADJUST

The TBN_ENDADJUST notification message notifies a toolbar's parent window that the user has stopped customizing a toolbar. This message is sent in the form of a WM_NOTIFY message.

TBN_ENDADJUST

pnmhdr = (NMHDR FAR *) lParam;

Parameters

pnmhdr

Pointer to an NMHDR structure that contains information about the notification message.

Return Values

No return value.

See Also

NMHDR, WM_NOTIFY
