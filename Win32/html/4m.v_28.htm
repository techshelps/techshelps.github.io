Statistics Functions

Windows NT and LAN Manager accumulate a set of operating statistics for workstations and servers from the time that the Workstation or Server service is started. The NetStatisticsGet2 is called to get those statistics. Because Windows NT and LAN Manager 2.x workstations collect a different set of statistics, the caller must know whether the server is Windows NT or LAN Manager 2.x (which can be discovered using the NetServerGetInfo function) and interpret the returned buffer accordingly.

NetStatisticsGet2 returns a STAT_WORKSTATION_0 structure when workstation statistics are requested; it returns a STAT_SERVER_0 structure when server statistics are requested.
