TreeView_GetRoot

The TreeView_GetRoot macro retrieves the topmost or very first item of the tree-view control. You can use this macro, or you can explicitly send the TVM_GETNEXTITEM message with the TVGN_ROOT flag.

HTREEITEM TreeView_GetRoot(

hwnd  
);  

Parameters

hwnd

Handle to the tree-view control.

Return Values

Returns the handle to the item if successful or NULL otherwise.

See Also

TreeView_GetNextItem, TVM_GETNEXTITEM
