SNMP Utilities

The following table summarizes the SNMP utilities and files that are available on the Microsoft Windows Resource Kit.

Filename Description
MIBCC.EXE SNMP MIB Compiler
PerfMIB SNMP to Performance Monitor
SNMPUTIL.EXE A very simple SNMP manager application that implements Get, Get Next, Walk, and Trap
LMMIB2.MIB Defines approximately 150 objects that include information on statistics, share, session, user, and logon
MIB_II.MIB Defines 171 objects essential for either configuration or fault analysis
