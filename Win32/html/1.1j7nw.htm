LVM_GETTEXTBKCOLOR

The LVM_GETTEXTBKCOLOR message retrieves the text background color of a list view control. You can send this message explicitly or by using the ListView_GetTextBkColor macro.

LVM_GETTEXTBKCOLOR

wParam = 0;

lParam = 0;

Return Values

Returns the background color of the text.
