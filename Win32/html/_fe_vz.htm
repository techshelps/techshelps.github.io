STM_GETIMAGE

An application sends an STM_GETIMAGE message to retrieve the handle of the image associated with a static control.

STM_GETIMAGE

wParam = (WPARAM) fImageType; // image-type flag

lParam = 0; // not used; must be zero

Parameters

fImageType

Value of wParam. Specifies the type of image to retrieve. This parameter can be one of the following values:

IMAGE_BITMAP
IMAGE_CURSOR
IMAGE_ENHMETAFILE
IMAGE_ICON

Return Values

The return value is the handle of the image, if any; otherwise, it is NULL.
