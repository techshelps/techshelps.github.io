Finding Header, Library, and Compatibility Information

In the Win32 Programmer's Reference, the QuickInfo button provides a convenient way to get the following information:

·Platform support for Windows NT, Windows 95, Win32s

·Import library

·Header file

·Unicode support

·Platform notes

Detailed information on platform differences is included in the documentation, but the QuickInfo button provides the only means to get information about headers, libraries, and Unicode support.

To open the QuickInfo box

·Click the button at the top of the reference topic displayed on the screen. The QuickInfo box appears.

[Note]

This functionality is now available in this .CHM version of this Manual.

But if the Quick Info link is not working, to get information about headers, libraries and Unicode support, open the Search pane, enter the function’s name in the text field, and hit the Show button.

The list of occurrences will contain an  entry with __QI suffix .

View this entry to obtain the required information.
