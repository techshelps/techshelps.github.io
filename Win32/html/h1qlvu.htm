Header_GetItemCount

The Header_GetItemCount macro retrieves a count of the items in a header control. You can use this macro or explicitly send the HDM_GETITEMCOUNT message.

int Header_GetItemCount(

hwndHD  
);  

Parameters

hwndHD

Handle to the header control.

Return Values

If the operation succeeds, the return value is the number of items.

If the operation fails, the return value is - 1.

Remarks

The Header_GetItemCount macro is defined as follows:

#define Header_GetItemCount(hwndHD) \

(int)SendMessage((hwndHD), HDM_GETITEMCOUNT, 0, 0L)
