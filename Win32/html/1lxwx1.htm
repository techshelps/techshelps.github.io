RasGetAutodialEnable

[New - Windows NT]

The RasGetAutodialEnable function indicates whether the AutoDial feature is enabled for a specified TAPI dialing location. For more information about TAPI dialing locations, see the Win32 Telephony (TAPI) Programmer's Reference.

DWORD RasGetAutodialEnable (

DWORD dwDialingLocation, // identifier of the TAPI dialing location
LPBOOL lpfEnabled // pointer to variable that receives AutoDial state for this location
);  

Parameters

dwDialingLocation

Specifies the identifier of a TAPI dialing location.

lpfEnabled

Pointer to a BOOL variable that receives a nonzero value if AutoDial is enabled for the specified dialing location, or zero if it is not enabled.

Return Values

If the function succeeds, the return value is zero.

If the function fails, the return value is a nonzero value.
