PSYM_ENUMMODULES_CALLBACK

typedef BOOL (CALLBACK *PSYM_ENUMMODULES_CALLBACK)(

LPSTR ModuleName,  
ULONG BaseOfDll,  
PVOID UserContext  
);  

Parameters

ModuleName

The name of the module.

BaseOfDll

The base address where the module is loaded into memory.

UserContext

The value passed to SymEnumerateModules.

Return Values

If a value of TRUE is returned, the enumeration will continue.

If a value of FALSE is returned, then the enumeration will stop.

Remarks

The calling application gets called once per module until all modules are enumerated, or until the enumeration callback function returns FALSE.
