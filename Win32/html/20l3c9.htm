SnmpUtilVarBindFree

The SnmpUtilVarBindFree function frees any allocated data associated with an RFC1157VarBind structure.

VOID SnmpUtilVarBindFree(

RFC1157VarBind *VarBind // variable binding to free
);  

Parameters

VarBind

[in/out] Points to an RFC1157VarBind structure whose allocated data should be freed.

Return Values

No return value.

See Also

RFC1157VarBind, SnmpUtilVarBindListFree
