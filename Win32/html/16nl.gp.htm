Dragging an Image

Dragging an image involves calls to the ImageList_BeginDrag, ImageList_DragMove, and ImageList_EndDrag functions. ImageList_BeginDrag begins a drag operation by combining an image with the current mouse cursor, setting a hot spot in the image, and drawing the image in its initial position. The function also prevents the system from updating any other parts of the display. ImageList_DragMove drags the image to a new location. ImageList_EndDrag ends a drag operation, permitting the system to update any part of the display.

The remainder of this section provides an example using four functions that demonstrates how to drag an image. The first function shows how to draw an image in a window's client area, and subsequent functions show how to drag the image.
