TreeView_GetDropHilite

The TreeView_GetDropHilite macro retrieves the tree-view item that is the target of a drag-and-drop operation. You can use this macro, or you can explicitly send the TVM_GETNEXTITEM message with the TVGN_DROPHILITE flag.

HTREEITEM TreeView_GetDropHilite(

hwnd  
);  

Parameters

hwnd

Handle to the tree-view control.

Return Values

Returns the handle to the item if successful or NULL otherwise.
