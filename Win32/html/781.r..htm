TabCtrl_SetItemSize

The TabCtrl_SetItemSize macro sets the width and height of tabs in a fixed-width or owner-drawn tab control. You can use this macro or explicitly send the TCM_SETITEMSIZE message.

DWORD TabCtrl_SetItemSize(

HWND hwnd,  
int cx,  
int cy  
);  

Parameters

hwnd

Handle to the tab control.

cx and cy

New width and height, in pixels.

Return Values

Returns the old width and height. The width is in the low-order word of the return value, and the height is in the high-order word.
