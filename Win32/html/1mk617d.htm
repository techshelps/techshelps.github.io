SERVER_INFO_1017

The SERVER_INFO_1017 structure contains information about the specified server, including name, platform, type of server, and associated software.

Members

sv1017_announce

Specifies the network announce rate, in seconds. This rate determines how often the server is announced to other computers on the network.
