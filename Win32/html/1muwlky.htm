TCM_DELETEALLITEMS

The TCM_DELETEALLITEMS message removes all items from a tab control. You can send this message explicitly or by using the TabCtrl_DeleteAllItems macro.

TCM_DELETEALLITEMS

wParam = 0; // not used; must be zero

lParam = 0; // not used; must be zero

Return Values

Returns TRUE if successful or FALSE otherwise.
