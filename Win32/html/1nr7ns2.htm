SHARE_INFO_1006

The SHARE_INFO_1006 structure contains information about the shared resource, including name of the resource, type and permissions, and number of connections.

typedef struct _SHARE_INFO_1006 {
DWORD shi1006_max_uses;
} SHARE_INFO_1006, *PSHARE_INFO_1006, *LPSHARE_INFO_1006;

Members

shi1006_max_uses

Indicates the maximum number of concurrent connections that the shared resource can accommodate (unlimited if the value specified in shi1006_max_uses is -1).
