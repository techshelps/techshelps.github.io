TBM_CLEARTICS

The TBM_CLEARTICS message removes the current tick marks from a trackbar.

TBM_CLEARTICS

wParam = (WPARAM) (BOOL) fRedraw;

lParam = 0;

Parameters

fRedraw

Redraw flag. If this parameter is TRUE, the trackbar is redrawn after the tick marks are cleared.

Return Values

No return value.
