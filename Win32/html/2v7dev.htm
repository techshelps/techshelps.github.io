EMRANGLEARC

The EMRANGLEARC structure contains members for the AngleArc enhanced metafile record.

typedef struct tagEMRANGLEARC
{
EMR emr;
POINTL ptlCenter;

DWORD nRadius;
FLOAT eStartAngle;
FLOAT eSweepAngle;
} EMRANGLEARC, *PEMRANGLEARC;

Members

emr

Base structure for all record types.

ptlCenter

Logical coordinates of circle's center.

nRadius

Circle's radius, in logical units.

eStartAngle

Arc's start angle, in degrees.

eSweepAngle

Arc's sweep angle, in degrees.
