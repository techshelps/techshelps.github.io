TreeView_GetFirstVisible

The TreeView_GetFirstVisible macro retrieves the first visible item in a tree-view control. You can use this macro, or you can explicitly send the TVM_GETNEXTITEM message with the TVGN_FIRSTVISIBLE flag.

HTREEITEM TreeView_GetFirstVisible(

hwnd  
);  

Parameters

hwnd

Handle to the tree-view control.

Return Values

Returns the handle to the item if successful or NULL otherwise.

See Also

TreeView_GetNextVisible, TreeView_GetPrevVisible, TVM_GETITEMRECT, TVM_GETNEXTITEM
