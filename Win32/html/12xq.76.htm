USER_INFO_1011

The USER_INFO_1011 structure contains user information for network accounts.

typedef struct _USER_INFO_1011 {
LPWSTR usri1011_full_name;
} USER_INFO_1011, *PUSER_INFO_1011, *LPUSER_INFO_1011;

Members

usri1011_full_name

Points to a Unicode string that contains the full name of the user. This string can be a null string, or it can have any number of characters before the terminating null character.
