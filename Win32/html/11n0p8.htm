LVN_INSERTITEM

The LVN_INSERTITEM notification message notifies a list view control's parent window that a new item was inserted. This notification message is sent in the form of a WM_NOTIFY message.

LVN_INSERTITEM

pnmv = (NM_LISTVIEW FAR *) lParam;

Parameters

pnmv

Pointer to an NM_LISTVIEW structure. The iItem member identifies the new item, and the other members are zero.

Return Values

No return value.

See Also

NM_LISTVIEW, WM_NOTIFY
