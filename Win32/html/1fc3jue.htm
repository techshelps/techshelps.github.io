TabCtrl_HitTest

The TabCtrl_HitTest macro determines which tab, if any, is at a specified screen position. You can use this macro or explicitly send the TCM_HITTEST message.

int TabCtrl_HitTest(

HWND hwnd,  
TC_HITTESTINFO FAR *pinfo  
);  

Parameters

hwnd

Handle to the tab control.

pinfo

Pointer to a TC_HITTESTINFO structure that specifies the screen position to test.

Return Values

Returns the index of the tab or - 1 if no tab is at the specified position.

See Also

TC_HITTESTINFO, TCM_HITTEST
