LVM_DELETECOLUMN

The LVM_DELETECOLUMN message removes a column from a list view control. You can send this message explicitly or by using the ListView_DeleteColumn macro.

LVM_DELETECOLUMN

wParam = (WPARAM) (int) iCol;

lParam = 0;

Parameters

iCol

Index of the column to delete.

Return Values

Returns TRUE if successful or FALSE otherwise.

See Also
