TabCtrl_GetRowCount

The TabCtrl_GetRowCount macro retrieves the current number of rows of tabs in a tab control. You can use this macro or explicitly send the TCM_GETROWCOUNT message.

int TabCtrl_GetRowCount(

HWND hwnd  
);  

Parameters

hwnd

Handle to the tab control.

Return Values

Returns the number of rows of tabs.

Remarks

Only tab controls that have the TCS_MULTILINE style can have multiple rows of tabs.
