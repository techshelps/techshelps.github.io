Overlay Mask

Every image list includes a list of images to use as overlay masks. An overlay mask is an image drawn transparently over another image. Any image can be used as an overlay mask. You can specify up to four overlay masks per image list.

You add the index of an image to the list of overlay masks by using the ImageList_SetOverlayImage function, specifying the handle of the image list, the index of an image, and the index of an overlay mask. Note that the indices for the overlay masks are one-based rather than zero-based.

You draw an overlay mask over an image using a single call to the ImageList_Draw or ImageList_DrawEx function. The parameters include the index of the image to draw and the index of an overlay mask. You must use the INDEXTOOVERLAYMASK macro to specify the index of the overlay mask.
