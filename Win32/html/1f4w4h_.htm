SetCommConfig

[Now Supported on Windows NT]

The SetCommConfig function sets the current configuration of a communications device.

BOOL SetCommConfig(

HANDLE hCommDev, // handle of communications device
LPCOMMCONFIG lpCC, // address of comm. configuration services
DWORD dwSize // size of structure
);  

Parameters

hCommDev

Identifies the open communications device.

lpCC

Points to a COMMCONFIG structure.

dwSize

Specifies the size, in bytes, of the structure pointed to by lpCC.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.
