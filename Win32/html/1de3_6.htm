TAPE_ERASE

The TAPE_ERASE structure describes the partition to be erased.

typedef struct _TAPE_ERASE { // ter

ULONG Type;

} TAPE_ERASE;

Members

Type

Specifies how to erase the tape. This member must have one of the following values:

Value Description
TAPE_ERASE_LONG Erases the entire partition.
TAPE_ERASE_SHORT Erases only the partition's header block.
