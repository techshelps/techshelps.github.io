SOBORDER

Contains information about the border around a cell in table.

typedef struct SOBORDERtag {
WORD wWidth; // see below
SOCOLORREF rgbColor; // see below
WORD wFlags; // see below
} SOBORDER, VWPTR * PSOBORDER;

Members

wWidth

Width, in twips, of the border.

rgbColor

Color of the border. This member can be a RGB color value. To set this member, use the SORGB macro.

wFlags

Type of border and the edges to which it applies. This member can be a combination of the SO_BORDERNONE, SO_BORDERDOUBLE, SO_BORDERHAIRLINE, SO_BORDERTHICK, SO_BORDERSHADOW, and SO_BORDERDOTTED values.
