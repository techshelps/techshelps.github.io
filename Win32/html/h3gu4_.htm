TCM_GETCURSEL

The TCM_GETCURSEL message determines the currently selected tab in a tab control. You can send this message explicitly or by using the TabCtrl_GetCurSel macro.

TCM_GETCURSEL

wParam = 0; // not used; must be zero

lParam = 0; // not used; must be zero

Return Values

Returns the index of the selected tab if successful or - 1 if no tab is selected.
