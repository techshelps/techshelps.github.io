ListView_DeleteColumn

The ListView_DeleteColumn macro removes a column from a list view control. You can use this macro or explicitly send the LVM_DELETECOLUMN message.

BOOL ListView_DeleteColumn(

HWND hwnd,  
int iCol  
);  

Parameters

hwnd

Handle to the list view control.

iCol

Index of the column to delete.

Return Values

Returns TRUE if successful or FALSE otherwise.
