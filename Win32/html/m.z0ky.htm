SOGetInfo

Retrieves information about a spreadsheet.

VOID SOGetInfo(

WORD wInfo,  
void VWPTR *pInfo,  
HPROC reserved  
);  

Parameters

wInfo

Information type. If this parameter is the SOINFO_COLUMNRANGE value, the function retrieves the range of columns for a spreadsheet to read.

pInfo

Address of a 32-bit variable that receives the first column number in the low-order word and the second column number in the high-order word.

reserved

Reserved; do not use.

Return Values

None.
