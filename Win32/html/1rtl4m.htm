ListView_SetTextBkColor

The ListView_SetTextBkColor macro sets the background color of text in a list view control. You can use this macro or explicitly send the LVM_SETTEXTBKCOLOR message.

BOOL ListView_SetTextBkColor(

HWND hwnd,  
COLORREF clrText  
);  

Parameters

hwnd

Handle to the list view control.

clrText

New text color.

Return Values

Returns TRUE if successful or FALSE otherwise.
