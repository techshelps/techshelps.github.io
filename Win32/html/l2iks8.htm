FILE_INFO_2

The FILE_INFO_2 structure specifies an identification number to a file, device, or pipe.

typedef struct _FILE_INFO_2 {
DWORD fi2_id;
} FILE_INFO_2, *PFILE_INFO_2, *LPFILE_INFO_2;

Members

fi2_id
