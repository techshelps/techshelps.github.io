VwStreamReadFunc

Outputs characters, cells, or fields, depending on the current section type and file contents.

INT VwStreamReadFunc(

SOFILE hFile,  
HPROC reserved  
);  

Parameters

hFile

Handle of the file for execute input and output (XIO) routines.

reserved

Reserved; do not use.

Return Values
