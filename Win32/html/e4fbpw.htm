TabCtrl_DeleteItem

The TabCtrl_DeleteItem macro removes an item from a tab control. You can use this macro or explicitly send the TCM_DELETEITEM message.

BOOL TabCtrl_DeleteItem(

HWND hwnd,  
int iItem  
);  

Parameters

hwnd

Handle to the tab control.

iItem

Index of the item to delete.

Return Values

Returns TRUE if successful or FALSE otherwise.
