UDM_SETBASE

The UDM_SETBASE message sets the radix base for an up-down control. The base value determines whether the buddy window displays numbers in decimal or hexadecimal digits. Hexadecimal numbers are always unsigned, and decimal numbers are signed.

UDM_SETBASE

wParam = (WPARAM) nBase;

lParam = 0;

Parameters

nBase

New base value for the control. This parameter can be 10 for decimal or 16 for hexadecimal.

Return Values

The return value is the previous base value. If an invalid base is given, the return value is zero.
