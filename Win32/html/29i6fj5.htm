ImmIsIME

[Now Supported on Windows NT]

The ImmIsIME function checks whether the specified handle identifies an IME.

BOOL ImmIsIME(

HKL hKL  
);  

Parameters

hKL

Handle to the keyboard layout to check.

Return Values

If the function succeeds, the return value is a nonzero value.

If the function fails, the return value is zero.
