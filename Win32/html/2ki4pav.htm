AbortPrinter

The AbortPrinter function deletes a printer's spool file if the printer is configured for spooling.

BOOL AbortPrinter(

HANDLE hPrinter // handle to printer object
);  

Parameters

hPrinter

Identifies the printer from which the spool file is deleted.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.

Remarks

If the printer is not configured for spooling, the AbortPrinter function has no effect.

The printer handle identified by the hPrinter parameter is returned by the OpenPrinter function.
