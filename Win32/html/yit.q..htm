TreeView_SetIndent

The TreeView_SetIndent macro sets the width of indentation for a tree-view control and redraws the control to reflect the new width. You can use this macro or explicitly send the TVM_SETINDENT message.

BOOL TreeView_SetIndent(

hwnd,  
indent  
);  

Parameters

hwnd

Handle to the tree-view control.

indent

Width, in pixels, of the indentation. If this parameter is less than the system-defined minimum width, the new width is set to the system-defined minimum.

Return Values

No return value.
