TabCtrl_SetImageList

The TabCtrl_SetImageList macro assigns an image list to a tab control. You can use this macro or explicitly send the TCM_SETIMAGELIST message.

BOOL TabCtrl_SetImageList(

HWND hwnd,  
HIMAGELIST himl  
);  

Parameters

hwnd

Handle to the tab control.

himl

Handle of the image list to assign to the tab control.

Return Values

Returns the handle to the previous image list or NULL if there is no previous image list.
