ListView_DeleteItem

The ListView_DeleteItem macro removes an item from a list view control. You can use this macro or explicitly send the LVM_DELETEITEM message.

BOOL ListView_DeleteItem(

HWND hwnd,  
int iItem  
);  

Parameters

hwnd

Handle to the list view control.

iItem

Index of the list view item to delete.

Return Values

Returns TRUE if successful or FALSE otherwise.
