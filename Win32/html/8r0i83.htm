Computers Key

Underneath the Computers key is a list of computers that have entries in the policy file. The steps to use in locating a Computers key follow:

1.Determine whether there is a key that is the same as the computer name. If there is one, merge the registry image beneath it into the HKEY_LOCAL_MACHINE key.

2.If the key is not there, determine whether there is a key called ".default". (This appears in the System Policy Editor user interface as Default Computer.) If a default key exists, merge the registry image beneath it.

3.If neither of the two steps work, do nothing.
