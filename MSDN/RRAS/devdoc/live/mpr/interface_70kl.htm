About the Routing Protocol Interface

[This is preliminary documentation and subject to change.]

The following documentation describes the integration of third-party routing protocols into the Microsoft® Windows NT® Routing and Remote Access Service, a multiprotocol router. It defines the interface between the router manager and the routing protocol dynamic link library (DLL).

Using this interface, vendors can implement routing protocols; for example, IGRP, NLSP, BGP, as user-mode DLLs that work with Windows NT Routing and RAS (RRAS).
