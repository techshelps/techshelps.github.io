About Tracing

[This is preliminary documentation and subject to change.]

The following documentation describes the implementation of the common tracing DLL, which provides a uniform mechanism for generating diagnostic output for the Microsoft® Windows NT® Routing and RAS components as well as any other application that wishes to use the DLL. The DLL provides dynamic configuration change, allowing a user to direct output to a console or to a specified file. In the case of files, the user can specify the maximum size for the file.
