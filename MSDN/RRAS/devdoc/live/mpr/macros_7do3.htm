In addition, the ID for the IP router manager is:

IPRTRMGR_PID

This ID can be used instead of a routing protocol ID for MIB calls with the IP Router Manager. This ID is used for MIB-II, Forwarding MIB, and some enterprise specific information. This ID is also listed in iprtrmib.h.

IPX Protocols

IPX_PROTOCOL_RIP

IPX_PROTOCOL_SAP

IPX_PROTOCOL_NLSP

In addition, the ID for the IPX router manager is:

IPX_PROTOCOL_BASE

Use this ID instead of a routing protocol ID for MIB calls with the IPX Router Manager.
