About Routing and Remote Access Service

[This is preliminary documentation and subject to change.]

The following documentation describes the Application Programming Interface (API) for Microsoft® Windows NT® Routing and Remote Access Service (RAS). The Routing and RAS API has the following components.

  • RAS Administration
  • Router Administration
  • Router Configuration
  • Routing Protocol Interface
  • Routing Table Manager
  • Plugin Interface
  • Extensible Authentication Protocol
  • Tracing
