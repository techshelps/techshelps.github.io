\SAMPFILE

This directory contains the sample files that can be used via Money or the OFC test application. In order to use a particular file in conjunction with Money, add the following key to the registry:


RequestFile

The value of the key is the complete path and filename of the OFC request file. Example: 


    c:\ofccert\sampfile\sonrq\sonrq1.ofc

Of course you can create new OFC requests or modify the files we have provided, but in order to use this testing hook, it must have the filename extension OFC. The location for the RequestFile key is:


HKLM\SOFTWARE\Microsoft\FASTXS\1.1\Providers\OFC

We have included sample files for the following: SONRQ, ACCTRQ, MAILRQ, STMTRQ, INTRARQ, INTERRQ, PAYEERQ, PAYMTRQ, and PAYIQRQ.
