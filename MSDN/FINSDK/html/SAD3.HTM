SERVER does not respond in time

STEPS:

  1. Note: For this test the server will have to force a "time-out" response.
  2. Create a new Money file in your c:\temp directory called CRA5A.MNY
  3. Create Accounts (Checking, Savings, and/or Credit Card)
  4. In the Bank Place select the Accounts tab and choose the "Select online services…" button
  5. Online enable the account(s) for POL, BOL or both according to your server specific information
  6. On the Connect Tab in the Bank Place - Press connect
  7. After the Money client has connected to the server, the server should post a "time-out" response
  8. Verify that Money enters a state of Crash Recovery after receiving the "time-out" response
  9. Copy the following files from c:\temp to your c:\ofccert\data directory

· request.txt rename to CRA5ARQ.OFC

· if response.txt exists rename to CRA5ARS.OFC
