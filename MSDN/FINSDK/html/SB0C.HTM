Execution

After launching the DRIVER.EXE program, choose "Load Test DLLs…" and load the necessary DLL. Select all the tests and choose "Execute". After all the tests have completed and passed, choose "Save" and save the contents to the c:\ofccert\data directory with a unique name. If any of the tests fail, you should investigate why the test fails and fix the problem. If the reason the test fails is due to a provider specific implementation decision, make a note of these and send those back to us also. We expect that you will need to run and re-run the suite of tests several times to flush out issues.
