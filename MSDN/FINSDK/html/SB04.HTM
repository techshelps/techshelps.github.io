\DLLS

We have compiled several different versions of the XSOFC DLL that write the OFC request and response to disk. There are 2 steps required to use the these versions of the DLL.

  1. Create a C:\TEMP directory.
  2. Copy the appropriate version of the XSOFC.DLL over the one that Money installs.

We have included several different versions of the DLLs

Directory

Purpose

SECURE

1. Security enabled

2. Payment method - Payee

SECURE_UK

1. Security enabled

2. Payment method - Interbank

NOSECURE

1. Security disabled
2. Payment method - Payee

NOSECURE_UK

1. Security disabled

2. Payment method - Interbank


After you complete your call to your OFC server you will find REQUEST.TXT and RESPONSE.TXT in the C:\TEMP directory you created.
