DTCLIENT Algorithm

DTCLIENT = DTCLIENT previous call

STEPS:

  1. Create a new Money file in your c:\temp directory called, CRA1A.MNY
  2. Create Accounts (Checking, Savings, and/or Credit Card)
  3. In the Bank Place select the Accounts tab and choose the "Select online services…" button
  4. Online enable the account(s) for POL, BOL or both according to your server specific information
  5. On the Connect Tab in the Bank Place - Press connect
  6. When the Call Summary is displayed press (CTRL+ALT+DEL) - select "End Task" for Money's process.
  7. copy the following files from c:\temp to your c:\ofccert\data directory

· CRA1A.OFD (Do Not Delete)

· CRA1A.OFE (Do Not Delete)

· request.txt rename to CRA1ARQ.OFC

· if response.txt exists rename to CRA1ARS.OFC

  1. Delete the response.txt and request.txt files from your c:\temp directory
  2. Open Money and salvage the file
  3. Go to the Bank Place and press connect again
  4. This time press the continue button on the "Call Summary" dialog
  5. Copy the following files from c:\temp to your c:\ofccert\data directory

· request.txt rename to CRA1BRQ.OFC

· if response.txt exists rename to CRA1BRS.OFC

  1. Delete the response.txt and request.txt files from your c:\temp directory
