\DRIVER

This is the directory that contains the executable, dynamic link libraries for certifying an OFC Server. The directory contains:

Filename

Description

BOL.DLL

Bank Online DLL

DRIVER.EXE

Driver for executing the tests

POL.DLL

Pay Online DLL

INTER.DLL

Interbank Transfer DLL

DRIVER.REG

Registry settings for the driver

OFCCERT.INI

INI file for the driver


Details for setup of the driver in the Server Certification Setup section.
