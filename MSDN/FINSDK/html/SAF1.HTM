Creating Active Statement Files

Institutions that wish to implement Active Statement Technology on their web sites have two choices when it comes to generating the OFC file; develop the code internally, or leverage Microsoft's publicly distributed Parse & Build code to generate the files.

Microsoft currently offers public Parse & Build code that may be used by Institutions implementing Active Statement Technology to generate the OFC file. This code is written in ANSI C and therefore available for use on a variety of platforms (Unix, etc.). Functions are defined within the P&B code that create the ACCTSTMT record format.

Note Refer to the document OFCPB.DOC in the download area for more information on using Microsoft's P&B code to generate the ACCTSTMT records.
