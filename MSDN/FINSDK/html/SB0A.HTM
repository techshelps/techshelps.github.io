How Server Certification Works

We have tried to make server certification as straight forward as possible, but in order to be flexible our certification requires a couple of steps.

  • Setup and Execution of the Test DLLs for your respective components. A complete list is at the end of this document.
  • Execution and documentation of the results from running manual tests via Money97.
  • Inspecting OFC code for specification implementation issues.
  • Sending the OFC files created when running the automated tests in step 1 back to Microsoft.
  • Fixing and resolving any problems that are uncovered.
