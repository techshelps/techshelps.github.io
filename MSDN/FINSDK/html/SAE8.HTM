How OFC Works

Microsoft designed OFC with input from financial institutions that wanted to deliver online banking services directly to their customers. With Microsoft Money 97, financial institutions now have the option to connect directly to customers via the OFC specification using either the Internet or a private dial-up connection. Money 97 implements the OFC specification by using standard Internet protocols (HTTP or TCP/IP) and standard Internet security protocols (PCT or SSL) to connect to the bank's server. Financial institutions that choose initially to make the connection via a private dial-up network can switch to the Internet connection easily at a later date.
