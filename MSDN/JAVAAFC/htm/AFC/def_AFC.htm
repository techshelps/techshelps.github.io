Microsoft Application Foundation Classes

Microsoft Application Foundation Classes (AFC) is a set of Java class libraries that provide user interface components and graphics effects for use in applets and applications using Java. AFC is licensed to be redistributed with the Microsoft Win32 VM for Java. AFC can also be redistributed with AFC102.ZIP for browsers that only support JDK 1.0.2. The following are links to detailed class references and overview articles explaining how to use AFC in your Java applets and applications.
