CHAPTER 8

Registering, Licensing, and Upgrading an ActiveX Designer

ActiveX designers must be registered in the system-wide registry so that hosts and other objects can find them. ActiveX designers also support licensing, which allows you to protect your proprietary product from unauthorized use. Upgrading allows you to distribute newer versions of your ActiveX designers which work with earlier versions in a clean, predictable manner.

This chapter discusses the following topics:

  • Registering an ActiveX designer.
  • Licensing the visual designer and run-time object.
  • Upgrading and releasing a new version of an ActiveX designer.
