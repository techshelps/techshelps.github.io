CHAPTER 2

Using ActiveX Designers with Visual Basic

Microsoft Visual Basic version 5.0 supports multiple kinds of designers, including ActiveX designers. Previous versions of Visual Basic supported only a single kind of designer, the built-in Forms designer.

To write ActiveX designers that work properly in the Visual Basic development environment, you need to understand how Visual Basic and Visual Basic programmers work with ActiveX designers.

This chapter covers the following topics:

  • ActiveX designers and other Visual Basic objects.
  • Creating an application with an ActiveX designer.
  • Lifetime of visual designer objects.
