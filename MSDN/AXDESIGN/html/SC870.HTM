ActiveX and OLE Support

ActiveX technologies provide the following features required by ActiveX designers:

  • Embedding.
  • In-place activation.
  • The ability to work as an in-process server.
  • Programmability using Automation.

In addition, ActiveX designers must implement a class factory to create instances of the design-time and run-time objects.

See Chapter 4, "Supporting ActiveX Interfaces," for additional information on ActiveX and COM requirements.
