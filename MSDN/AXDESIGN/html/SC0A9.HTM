Managing Properties

Most ActiveX Designers need to enable end users to view and change properties of objects. To provide this capability, the visual designer supports one or more of the following groups of interfaces:

  • IDispatch and either IProvideClassInfo or IProvideDynamicClassInfo.
  • IConnectionPointContainer, IConnectionPoint, and IPropertyNotifySink.
  • IPerPropertyBrowsing, IPropertyPage, IPropertyPage2, IPropertyPageSite, and ISpecifyPropertyPages.

This section describes the levels of property support that an ActiveX Designer may implement and shows how to implement each of these approaches. Refer to the   OLE Programmer's Reference for details of the property management interfaces.
