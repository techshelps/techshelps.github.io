Chapter One:

Status Bars and Toolbars: Organizational Common Controls

Our look at the new elements of the Microsoft Windows 95 user interface and the programming techniques you'll need to work with them begins with the new common controls. To get you started, this chapter does double duty. First it provides some basic information about the common controls as a group, including common control window classes, window styles, and notifications. Then the chapter moves on to introduce you to the two most popular common controls: status bars and toolbars.
