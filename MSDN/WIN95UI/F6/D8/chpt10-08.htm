HKEY_USERS

This key stores information about the default user configuration and contains a branch for each user of the computer. The default configuration is supplied for new users on the local computer and for the default current user if the user hasn't changed preferences. Because Windows 95 also supports HKEY_CURRENT_USER, applications can access the user-specific information the same way they do under Windows NT. Each user's information is stored in a separate file, which can be stored locally or on a network server. Windows 95 can copy this file to the user's current system so that settings can move from one computer to another with the user.
