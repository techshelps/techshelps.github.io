IShellLink::SetHotkey

Syntax:

SetHotkey (WORD wHotkey);

Parameter:

  • wHotkey is a pointer to the hot key.

Description:

Sets the hot key for the shortcut object. This allows you to specify that your shortcut is activated whenever the corresponding hot key is pressed. For example, you could set up a shortcut to a utility that backs up a specific directory when the user presses a key combination such as Alt-B.
