IShellLink::GetIconLocation

Syntax:

GetIconLocation (LPSTR pszIconPath, int cchIconPath, int *piIcon);

Parameters:

  • pszIconPath is a pointer to a text buffer that receives the icon location.
  • cchIconPath is the maximum number of characters in the icon location.
  • piIcon is a pointer to the icon.

Description:

Gets the icon location for the shortcut object.
