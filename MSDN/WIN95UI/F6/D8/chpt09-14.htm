IShellLink::GetDescription

Syntax:

GetDescription (LPSTR pszName, int cchMaxName);

Parameters:

  • pszName is a pointer to a text buffer that receives the description.
  • cchMaxName is the maximum number of characters that can be used in the description.

Description:

Gets the description of the shortcut object. The description string is copied into the buffer to which the pszName parameter points.
