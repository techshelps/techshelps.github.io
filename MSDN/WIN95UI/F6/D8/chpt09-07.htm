IShellLink::QueryInterface

Syntax:

QueryInterface (REFIID riid, LPVOID FAR *ppvObj);

Parameters:

  • riid is the identifier of the interface requested. To access the IShellLink interface, this parameter should be IID_IShellLink.
  • ppvObj is the pointer to the variable that receives the interface. This parameter is filled in with a pointer to the IShellLink interface. The returned pointer can then be used to access the other member functions provided by IShellLink.

Description:

Returns a pointer to the requested interface (IShellLink, in this case) if the interface is supported. Otherwise, it returns an error. Once this pointer is retrieved, the application can use it to gain access to the other member functions provided for the interface.
