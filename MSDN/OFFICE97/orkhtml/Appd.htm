This appendix describes the files installed by Microsoft Office Setup (Windows) or Microsoft Office Installer or drag-and-drop installation (Macintosh). The file lists are in Microsoft Excel workbooks included with the Office Resource Kit Tools and Utilities.

Note   The file size values in the workbooks included with the Office Resource Kit Tools and Utilities represent the number of bytes used by the files listed. The actual disk space used by these files may differ, depending on the sector size of your disk. For example, a file containing 5,000 bytes may take 16,384 bytes of disk space on a 1­GB disk with 16­KB sectors, but the same file may take 32,768 bytes of disk space on a 2­GB disk with 32­KB sectors.

Office 97 File Lists

   Office 97 for Windows Setup supports several distinct deployment strategies. For information about these strategies, see "Choosing Configuration and Installation Options" in Chapter 4, "Installing Microsoft Office"

Custom and Typical Installations

When you choose a Typical installation, Setup installs a subset of Office components that includes the most commonly used Office features. To see which components are included in a Typical installation, run Setup and choose a Custom installation. In the check box options displayed by Setup, the components that are selected by default are the components that are installed during a Typical installation.

  Tools and Utilities   The Office Resource Kit Tools and Utilities includes the workbooks Complete.xls and Typical.xls. These workbooks list complete lists of the the files installed during Custom installation (with all options selected) and Typical installation, respectively. For more information, see "Informational Workbooks for Windows," in Appendix A, "Microsoft Office Resource Kit Tools and Utilities."

Run from Network Server Installation

When you choose a Run from Network Server installation, Setup leaves most Office components on the network server. However, some Office components are copied to your hard disk.

  Tools and Utilities   The Office Resource Kit Tools and Utilities includes the workbook Network.xls. This workbook lists the files installed during a Run from Network Server installations. For more information, see "Informational Workbooks for Windows" in Appendix A, "Microsoft Office Resource Kit Tools and Utilities."

Run from CD Installation

When you choose a Run from CD installation, Setup leaves most Office components on the Office CD. However, some Office components are copied to your hard disk.

  Tools and Utilities   The Office Resource Kit Tools and Utilities include the workbook Runfrmcd.xls. This workbook lists the files installed during a Run from CD installations. For more information, see "Informational Workbooks for Windows" in Appendix A, "Microsoft Office Resource Kit Tools and Utilities."

Office 97 Service Release 1 File Lists

Office 97 (Windows) Service Release 1 (SR-1) is available in two forms:

  • The Office 97 SR-1 Enterprise Update

  • The SR-1 patch.

The SR­1 Enterprise Update is a complete Office installation which replaces the previous Office 97 product. The SR-1 patch contains only the changes made in SR-1 and is used to upgrade Office 97 files in-place to SR-1 versions. For information about both products, see "Installing the Office 97 Service Release 1" in Chapter 4, "Installing Microsoft Office."

SR­1 Enterprise Update

The Office 97 SR­1 Enterprise Update contains the complete SR­1 version of the Office 97 product. The SR­1 CD is identical to the Office 97 CD in structure and is installed in the same way. There are a number of individual files that are modified to incorporate the changes made in SR­1. The Sr1diff.xls workbook in the Office Resource Kit Tools and Utilities lists all the files that have been changed in the SR­1 Enterprise Update.

  Tools and Utilities   The Office Resource Kit Tools and Utilities include the workbook Sr1diff.xls. The worksheet "SR­1 Enterprise Update" in this workbook lists the Office 97 files that are changed in the SR­1 Enterprise Update. For more information, see "Informational Workbooks for Windows," in Appendix A, "Microsoft Office Resource Kit Tools and Utilities."

SR­1 Patch

The Office 97 SR-1 patch is a single, compressed file that contains only the changes made to Office 97 in SR-1, plus a program that can update Office 97 files and incorporate the changes. The Sr1diff.xls workbook in the Office Resource Kit Tools and Utilities lists all the files that are affected when the SR-1 patch program is run.

  Tools and Utilities   The Office Resource Kit Tools and Utilities include the workbook Sr1diff.xls. The worksheet "SR­1 Path" in this workbook lists the Office 97 files affected by the SR­1 patch. For more information, see "Informational Workbooks for Windows," in Appendix A, "Microsoft Office Resource Kit Tools and Utilities."

Office 98 File Lists

   Office 98 (Macintosh) supports two distinct installation methods: installation with Microsoft Office Installer or drag-and-drop installation. For information about both methods, see "Rolling Out Office 98 for the Macintosh" in Chapter 4, "Installing Microsoft Office."

Microsoft Office Installer includes three installation options: Easy Install, Custom Install, and Remove. These and other installation options are described below.

Easy Install and Custom Install Installations

When you choose Easy Install, Microsoft Office Installer installs a subset of Office components that include the most commonly used Office features. To see which components are included in Easy Install, run Microsoft Office Installer and choose Custom Install. In the check box options displayed by Microsoft Office Installer, the components that are selected by default are the components that are installed during Easy Install. These same components are also installed by drag-and-drop installation.

  Tools and Utilities   The Office Resource Kit Tools and Utilities include the workbook Macoff98.xls. The worksheet "Custom and Easy Install" in this workbook lists the Office 98 files installed during Custom Install (with all options selected) and Easy Install or drag-and-drop installation, respectively. For more information, see "Informational Workbook for the Macintosh," in Appendix A, "Microsoft Office Resource Kit Tools and Utilities."

Run from Network Server and Run from CD Installations

Although Microsoft Office Installer does not explicitly support installing to a network server, you can install Office to a network server and run Office applications from the server. You can also run Office applications directly from the Office CD. In both cases, Office copies a small number of files to the local hard disk.

  Tools and Utilities   The Office Resource Kit Tools and Utilities include the workbook Macoff98.xls. The worksheet "Run from Server or CD" in this workbook lists the files installed locally when running Office 98 from a network server or from the Office CD. For more information, see "Informational Workbook for the Macintosh," in Appendix A, "Microsoft Office Resource Kit Tools and Utilities."

Remove Installation Option

The Remove option in Microsoft Office Installer removes all Office files, except the files installed in the System folder and Office application files created after installation is complete such as the Microsoft Word Normal template.

  Tools and Utilities   The Office Resource Kit Tools and Utilities include the workbook Macoff98.xls. The worksheet "Files not removed" in this workbook lists the Office 98 files that the Remove option in Microsoft Office Installer does not remove. For more information, see "Informational Workbook for the Macintosh," in Appendix A, "Microsoft Office Resource Kit Tools and Utilities."
