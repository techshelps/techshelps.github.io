ReferenceStyle Property

Applies To

Application object.

Description

Returns or sets how Microsoft Excel displays cell references and row and column headings in either A1 or R1C1 reference style. Can be one of the following XlReferenceStyle constants: xlA1 or xlR1C1. Read/write Long.

Example

This example displays the current reference style.

If Application.ReferenceStyle = xlR1C1 Then
    MsgBox ("Microsoft Excel is using R1C1 references")
Else
    MsgBox ("Microsoft Excel is using A1 references")
End If
