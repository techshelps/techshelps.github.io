AutoSortOrder Property

Applies To

PivotField object.

Description

Returns the order used to sort the pivot field automatically. Can be one of the following XlSortOrder constants: xlAscending or xlDescending (or xlManual if automatic sorting is disabled). Read-only Long.

Example

This example displays a message box showing the AutoSort parameters for the Product field.

With Worksheets(1).PivotTables(1).PivotFields("product")
    Select Case .AutoSortOrder
        Case xlManual
            aso = "manual"
        Case xlAscending
            aso = "ascending"
        Case xlDescending
            aso = "descending"
    End Select
    MsgBox " sorted in " & aso & _
        " order by " & .AutoSortField
End With
