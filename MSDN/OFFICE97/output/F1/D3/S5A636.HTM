MarkerStyle Property

Applies To

LegendKey object, Point object, Series object.

Description

Returns or sets the marker style for a point or series in a line chart, scatter chart, or radar chart. Read/write Long.

Can be one of the following XlMarkerStyle constants.

Constant

Description

xlMarkerStyleNone

No markers

xlMarkerStyleAutomatic

Automatic markers

xlMarkerStyleSquare

Square markers

xlMarkerStyleDiamond

Diamond-shaped markers

xlMarkerStyleTriangle

Triangular markers

xlMarkerStyleX

Square markers with an X

xlMarkerStyleStar

Square markers with an asterisk

xlMarkerStyleDot

Short bar markers

xlMarkerStyleDash

Long bar markers

xlMarkerStyleCircle

Circular markers

xlMarkerStylePlus

Square markers with a plus sign

xlMarkerStylePicture

Picture markers


Example

This example sets the marker style for series one in Chart1. The example should be run on a 2-D line chart.

Charts("Chart1").SeriesCollection(1).MarkerStyle = xlMarkerStyleCircle
