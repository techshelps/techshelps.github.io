LeftHeader Property

Applies To

PageSetup object.

Description

Returns or sets the left part of the header. Read/write String.

Remarks

Special format codes can be used in the header text. For more information, see Formatting Codes for Headers and Footers.

See Also

CenterFooter property, CenterHeader property, LeftFooter property, RightFooter property, RightHeader property.

Example

This example prints the date in the upper-left corner of every page.

Worksheets("Sheet1").PageSetup.LeftHeader = "&D"
