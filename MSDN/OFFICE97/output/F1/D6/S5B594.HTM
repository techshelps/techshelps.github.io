MaximumScale Property

Applies To

Axis object.

Description

Returns or sets the maximum value on the axis. Read/write Double.

Remarks

Setting this property sets the MaximumScaleIsAuto property to False.

See Also

MaximumScaleIsAuto property, MinimumScale property, MinimumScaleIsAuto property.

Example

This example sets the minimum and maximum values for the value axis.

With myChart.Axes(xlValue)
    .MinimumScale = 10
    .MaximumScale = 120
End With
