Item Method

Applies To

Axes collection object, ChartGroups collection object, DataLabels collection object, LegendEntries collection object, Points collection object, SeriesCollection collection object, Trendlines collection object.

Description

Returns part of the specified collection. The Item method is the default method for collections. For example, the following two lines of code are equivalent:

myChart.SeriesCollection.Item(1)
myChart.SeriesCollection(1)
