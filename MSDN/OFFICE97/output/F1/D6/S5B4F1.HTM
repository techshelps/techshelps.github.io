AxisTitle Property

Applies To

Axis object.

Description

Returns an AxisTitle object that represents the title of the specified axis. Read-only.

See Also

ChartTitle property, HasTitle property.

Example

This example adds an axis label to the category axis in myChart.

With myChart.Axes(xlCategory)
    .HasTitle = True
    .AxisTitle.Text = "July Sales"
End With
