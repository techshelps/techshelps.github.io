SortBy Property

Applies To

FileFind object.

Description

Macintosh only. Returns or sets the method used to sort the files returned from the specified file search. Can be one of the following MsoFileFindSortBy constants: msoFileFindSortByAuthor, msoFileFindSortByDateCreated, msoFileFindSortByDateSaved, msoFileFindSortByFileName, msoFileFindSortByLastSavedBy, msoFileFindSortBySize, or msoFileFindSortByTitle. Read/write.
