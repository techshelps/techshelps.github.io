DateCreatedFrom Property

Applies To

FileFind object.

Description

Macintosh only. Returns or sets the date on which the file to be searched for was created; this will be the lower limit for the file search. Read/write Variant.

Remarks

You can use the DateCreatedTo property to set an upper limit for the file search, or you can omit it to conduct an open-ended search.

See Also

DateCreatedTo property, DateSavedFrom property, DateSavedTo property.
