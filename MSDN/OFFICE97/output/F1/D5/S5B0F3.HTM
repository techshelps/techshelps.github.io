GetRecipientFromID Method

Applies To

NameSpace object.

Description

Returns a Recipient object identified by the specified entry ID (if valid). This method is used for ease of transition between MAPI and OLE/Messaging applications and Outlook.

Syntax

expression.GetRecipientFromID(EntryID)

expression An expression that returns a NameSpace object.

EntryID Required String. The entry ID of the recipient.
