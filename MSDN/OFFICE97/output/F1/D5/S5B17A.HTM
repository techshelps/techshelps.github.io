ReminderSoundFile Property

Applies To

AppointmentItem object, MailItem object, TaskItem object.

Description

Returns or sets the path and file name of the sound file to play when the reminder occurs for the appointment or task. This property is only valid if the ReminderOverrideDefault and ReminderPlaySound properties are set to True. Read/write String.

See Also

ReminderOverrideDefault property, ReminderPlaySound property.
