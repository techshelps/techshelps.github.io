FormDescription Object

Description

Contains the general properties of an Outlook form. The properties of an Outlook form are displayed on the Properties page of a form in design mode. To see the Properties page in design mode, open the Outlook item, click Design Outlook Form on the Tools menu, and then click the Properties tab in item's window.

Using the FormDescription Object

Use the FormDescription property to return the FormDescription object associated with an Outlook item.

Properties

Category property, CategorySub property, Comment property, ContactName property, DisplayName property, Hidden property, Icon property, Locked property, MessageClass property, MiniIcon property, Name property, Number property, OneOff property, Password property, Template property, UseWordMail property, Version property.

Methods

PublishForm method.
