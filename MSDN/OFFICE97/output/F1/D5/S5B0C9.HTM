Email2AddressType Property

Applies To

ContactItem object.

Description

Returns or sets the address type (such as EX or SMTP) of the second e-mail entry for the contact. This is free-form text field, but it must match the actual type of a existing mail transport. Read/write String.

See Also

Email1AddressType property, Email3AddressType property.

Example

This example sets "SMTP" as the address type for the second e-mail entry of a contact.

myItem.Email2AddressType = "SMTP"
