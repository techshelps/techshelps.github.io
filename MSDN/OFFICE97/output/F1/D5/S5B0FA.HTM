HideFormPage Method

Applies To

Inspector object.

Description

Hides a form page in the inspector.

Syntax

expression.HideFormPage(PageName)

expression An expression that returns an Inspector object.

Index Required String. The display name of the page to be hidden.

Example

This example hides the Message page of the currently open item.

Set myInspector = myOlApp.ActiveInspector
myInspector.HideFormPage("Message")
