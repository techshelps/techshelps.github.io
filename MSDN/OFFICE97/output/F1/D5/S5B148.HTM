OtherAddressCountry Property

Applies To

ContactItem object.

Description

Returns or sets the country portion of the other address for the contact. Read/write String.

Remarks

This property is parsed from the OtherAddress property, but may be changed or entered independently should it be parsed incorrectly. Note that any such changes or entries to this property will be overwritten by any subsequent changes or entries to OtherAddress.

See Also

OtherAddress property, OtherAddressCity property, OtherAddressPostalCode property, OtherAddressPostOfficeBox property, OtherAddressState property, OtherAddressStreet property.
