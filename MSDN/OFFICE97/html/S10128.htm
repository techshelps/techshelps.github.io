The Recent File List Subkey

The Recent File List subkey corresponds to the list of the most recently used files at the bottom of the File menu. This list changes each time the user quits Microsoft Excel. The following is a list of entries in a typical Recent File List subkey:

Entry        Value
File1        "C:\ACCTG\CLOSING.XLW"
File2        "C:\MONTHLY\JANSALES.WK3"
File3        "C:\ACCTG\SCRATCH.XLS"
File4        "C:\EXCEL\AMORT.XLS"
File5        ""
File6        ""
File7        ""
File8        ""
File9        ""
