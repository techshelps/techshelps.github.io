SCL: Window Zoom Magnification (A0h)

This record stores the window zoom magnification.

Record Data

Name

Size

Contents

Numerator of a reduced fraction

6

dscl

2

Denominator of a reduced fraction


The magnification is stored as a reduced fraction. For example, if the magnification is 75 percent, nscl = 03h and dscl = 04h (3/4 = 0.75 = 75%). If the magnification is 11 percent, nscl = 0Bh (11 decimal) and dscl = 64h (100 decimal). If the BIFF file does not contain the SCL record, the magnification is 100 percent.
