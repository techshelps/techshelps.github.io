Adding an ActiveX Control to a Form

Once an ActiveX control has been registered, you can add it to a form.

Û To add an ActiveX control to a form

  1. Open the form in Design view.

  2. Click the More Controls tool in the toolbox, or click ActiveX Control on the Insert menu.

    A menu appears that lists all the registered ActiveX controls in your system.

  3. In the list, click the ActiveX control you want to add.

    Note   Some of the controls that appear in the list of ActiveX controls may not be compatible with Microsoft Access. Before adding a control to a form, be sure to verify that it is compatible. For more information, see “Support for ActiveX Controls” earlier in this chapter.

  4. If you used the More Controls tool in the toolbox, click on the form where you want to place the control. If you used the ActiveX Control command on the Insert menu, the control is automatically placed on the form for you.

    Microsoft Access creates the control.

Tip If you plan to use the control frequently, you can add it to the toolbox or to any toolbar. For more information on customizing toolbars, see Chapter 1, “Creating an Application.”
