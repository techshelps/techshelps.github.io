The Assistant Subkey

Specifies location information for the Microsoft Office Assistant.

Setting

Indicates

AsstPath

The path where the files for the installed assistant are located

AsstSourcePath

The path where the files for additional assistant options are located

OfficeStartupPath

The path where the assistant startup files are located
