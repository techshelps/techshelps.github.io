Other Microsoft Excel File Formats

Although chart records are written as part of the Book stream, they are documented in another article. Microsoft Excel creates several other files, some of which are documented. The workspace file (.XLW extension in Microsoft Windows) and the toolbar file (.XLB extension in Microsoft Windows) are not documented.

This article contains BIFF documentation for Microsoft Excel version 5.0, Microsoft Excel 95, and Microsoft Excel 97 only. Earlier versions of BIFF documentation are available on the Development Library (published by the Microsoft Developer Network). The library is a CD-based reference source for Windows-based developers.
