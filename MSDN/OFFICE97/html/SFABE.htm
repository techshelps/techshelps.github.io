Displaying Help Topics with Microsoft Excel

In Microsoft Windows, Help files are displayed using Microsoft Windows Help (WINHLP32.EXE), usually located in the WINDOWS directory.

To display a Help topic in Microsoft Excel, use one of the following:

  • The Visual Basic Help method or the HELP macro function
  • The Visual Basic MsgBox function or the ALERT macro function
  • Help for custom commands and toolbar buttons
  • Help for dialog boxes
