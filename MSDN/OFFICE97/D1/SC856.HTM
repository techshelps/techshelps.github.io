The Purpose of the Office Compatible 97 Help Topic

This feature helps users transfer their skills from Office to the Office Compatible 97 product. The

Help topic contains three standard sections which quickly orient users to the product. Section 1 contains a brief description of the Office Compatible program. Section 2 describes the Office features which the product supports, and it discusses any changes that have been made to the standard Office design. (Note that any such changes must be approved by Microsoft prior to testing.) Section 3 explains how to use the product together with Microsoft Office.
