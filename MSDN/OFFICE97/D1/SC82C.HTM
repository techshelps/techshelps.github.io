Modal Products

Some applications can operate in several distinct modes. For example, a database application might include a "Forms Design" mode, a "Query" mode, and a "Data Entry" mode. Every mode must fully comply with the relevant Office Compatible 97 guidelines. However, this requirement will be waived for simple modes (e.g., a configuration mode) if they cannot be reasonably expected to support a full Office Compatible 97 interface. In such cases, apply for an exemption by sending email to: OFFCOMP@MICROSOFT.COM.
