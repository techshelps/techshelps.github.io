Help Dropdown Menu Entries

As illustrated below, the Office Compatible 97 Help Dropdown Menu contains several entries, whose names and locations are standardized.

Figure 4. Help Dropdown Menu

Menu Name: Help-><Application_Name> Help

Location: See Figure 4.

Accelerator Key: F1

Purpose: Activates a list of Help topics. The topics may be organized by Contents, an Indexed list, or in any other manner which is suitable to the product.

Testing Status: Mandatory for File-Based and Non-File-Based Standalone applications. Other applications that include Help (or a substantially comparable entry) must use the standard name, location, and accelerator key.

Menu Name: Help-><Application_Specific_Entries>

Location: Application specific Help entries must be aggregated in one or more distinct groups which are located beneath <Application_Name> Help and above About <Applicaton_Name>.

Accelerator Key: Developer's discretion

Purpose: A list of one or more application specific entries which are not standardized by Office Compatible 97, since they vary from one product to another.

Testing Status: Not required, but products that include application specific entries, must place them in the standard location.

Menu Name: Help->About <Application_Name>…

Location: See Figure 4.

Accelerator Key: CTRL+A

Purpose: Displays the About dialog box.

Testing Status: Mandatory for File-Based and Non-File-Based Standalone applications. Other applications that include About (or a substantially comparable entry) must use the standard name, location, and accelerator key.
