Features Which Are Subject to Office Compatible 97 Testing

In general, Office Compatible 97 products must support the following features in a manner that is substantially similar to Office. Since some products cannot be expected to include these features, however, the requirements are tailored to several broad categories of products. Read Product Categories Within Office Compatible 97 for more information.

  • Microsoft Windows Logo features. Office Compatible 97 products must pass Windows logo testing.
  • Microsoft Intellimouse ™. Office Compatible 97 products must support the Intellimouse ™, a new version of the Microsoft mouse which dramatically improves ease of use within Office.
  • Main Menu Bar & Accelerator Keys
  • Dropdown Menus & Accelerator Keys
  • Pop-up Menus
  • Basic Dialog Boxes: Open, Save As, & Print
  • HTML Support
  • Standard Toolbar & Toolbar Configuration Dialog Box
  • Office Compatible Help Topic. A special topic that tells customers how they can use their Office skills to become productive right away.
