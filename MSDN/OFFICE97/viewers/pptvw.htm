Microsoft® PowerPoint® Viewer 97

Microsoft PowerPoint Viewer 97 is a small application that enables PowerPoint users to share their presentations with people who do not use Microsoft PowerPoint for Windows®. The PowerPoint Viewer allows users to view and print PowerPoint presentations exactly as they appear in PowerPoint. Features include the power to view animations, graphics effects, action settings, hyperlinks and custom shows. The PowerPoint Viewer allows users to read and print PowerPoint presentations, but it does not allow them to edit PowerPoint presentations.
