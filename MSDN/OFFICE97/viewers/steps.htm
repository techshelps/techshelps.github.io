Microsoft® Excel Viewer 97 Step-by-Step Instructions
Installation To obtain Microsoft® Excel Viewer, download VIEWER.EXE, run it to extract the files it contains and automatically begin the Setup process.

Installation of Microsoft Excel Viewer is designed to take place from within Microsoft Windows. If Windows is not running, please start it before continuing.

Make sure to close all applications first and make sure you are not running any utilities or virus protection before proceeding.

If you have the Microsoft Excel Viewer self-extracting archive VIEWER.EXE, run it to extract the setup files and automatically start Setup. If you received Microsoft Excel Viewer on floppy disks, run the SETUP.EXE program on Disk 1.

You can now select a folder in which to install the Microsoft Excel Viewer. The default folder location for the Microsoft Excel Viewer is C:\PROGRAM FILES\XLVIEW. NOTE: Do not install Microsoft Excel Viewer in the same directory as Microsoft Excel. Doing so will cause file conflicts.

Click the "Installation" button to install the Microsoft Excel Viewer.

  To run Microsoft Excel Viewer, click the Start button on the Windows 95 Taskbar, and then point to Programs and choose Microsoft Excel Viewer. Alternatively, you can run Microsoft Excel Viewer by double-clicking on the XLVIEW.EXE file in your \PROGRAM FILES\XLVIEW folder.
  For information on using Microsoft Excel Viewer, please refer to the online Help.
NOTE: If you already have a full version of Microsoft Excel installed on your computer, DO NOT install Microsoft Excel Viewer in the same directory. Doing so will cause file conflicts.
