ExLinkAtom (4049)

This atom contains information about an OLE linked object.

Offset

Type

Name

Contents

exObjId

Unique external object identifier

flags

Stores the way the link is updated. This can be changed with the links dialog in the edit menu. The valid values are:

1 - automatic

3 - manual

5

bool1

unavailable

Set if the linked object is not available
