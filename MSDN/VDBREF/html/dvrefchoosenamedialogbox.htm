Choose Name Dialog Box

      

Enables you to change the system-assigned name of a new table that has not yet been created in the database. This dialog box appears when you:

  • Right-click the Tables folder and choose New Table from the shortcut menu.

  • Right-click the database diagram background and choose New Table from the shortcut menu.

    –or–

  • Click the New Table button on the toolbar.

Options

Enter a name for the table

Displays the system-assigned name of the table. Type a new name for the table in the text box.

OK

Creates the table in your database diagram with the name you specified. The table is created in the database when you save the table or diagram.

Cancel

Cancels the creation of the new table.

For more information about creating tables, see Adding a Table to a Database Diagram.
