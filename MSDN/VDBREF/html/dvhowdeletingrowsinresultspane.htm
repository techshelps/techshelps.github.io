Deleting Rows in the Results Pane

If the data in the Results pane can be modified, you can also delete existing rows from the table or tables represented by the result set.

To delete a row

  1. To select the row or rows you want to delete, select the box to the left in the Result pane grid.

  2. Press DELETE.

    Caution   Rows you delete in this way are permanently removed from the database and cannot be recalled.
