Saving a Stored Procedure

A stored procedure must be saved before you can run it. If you attempt to run an unsaved stored procedure, a message prompts you to save it first. When you save the stored procedure, it is added to the Stored Procedures folder in Data View.

To save a stored procedure

  • From the File menu, choose Save.

Stored procedures must have unique names. If the name beside CREATE PROCEDURE in the procedure text is already assigned to another database object, a message prompts you to choose another name for the stored procedure.
