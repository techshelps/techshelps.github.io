Deleting a Stored Procedure

When you no longer need to retrieve the information specified by a stored procedure, or the tables or columns it references have been deleted from your database, you can delete the stored procedure.

To delete a stored procedure

  1. In Data View, expand the Stored Procedures folder.

  2. Right-click the stored procedure that you want to delete.

  3. Choose Delete from the shortcut menu.

  4. A message prompts you to confirm the deletion. Choose Yes.

The stored procedure is deleted from the database and Data View.
