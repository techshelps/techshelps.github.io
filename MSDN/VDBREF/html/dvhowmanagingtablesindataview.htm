Managing Database Tables in Data View

You can use Data View to open, design, edit, copy, and delete tables in your database. For each data connection, Data View displays any tables in the database associated with the data connection. You can expand the Tables folder to see the columns and triggers included in the table.

Double-clicking a table icon in Data View opens the table and displays its data, which you can edit. You can drag a table from Data View to add it to an open database diagram, where you can edit the table's column properties. You can also drag a table to a query in the Query Designer, to add the table to the query definition.

To work with tables in Data View

  1. Expand the Tables folder and select a table to work with or create a new table by right-clicking the Tables folder and choosing New Table from the shortcut menu.

  2. Right-click the name of the table and choose one of the following options.
