Differences Between Visual Studio Enterprise and Professional Edition

      

The Microsoft® Visual Database Tools are available as a component in many of the Microsoft Visual Studio™ applications, including Microsoft® Visual InterDev™, Visual Basic®, and Visual C++®. The Visual Studio suite is available in two editions: Professional and Enterprise.

The Professional version of Visual Studio 6.0 offers developers a great solution for accessing data. The Enterprise version builds on the Professional box by offering the ability to manipulate the structure of the database using the full features of the Visual Database Tools.
