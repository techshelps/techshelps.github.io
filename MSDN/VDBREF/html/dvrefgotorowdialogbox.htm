Go To Row Dialog Box

      

Allows you to navigate to a specific row in the Results pane. This dialog box appears when you right-click in the Results pane and choose Row.

Options

Row

Enter the row number of the row you want to navigate to. If the query has finished fetching query results, the total number of rows in the result set is displayed. If you enter a number higher than the total number of rows in the result set, the Query Designer moves to the last row.
