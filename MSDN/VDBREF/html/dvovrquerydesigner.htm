What You Can Do in the Query Designer

Using the Query Designer, you can:

  • Create queries to retrieve data from any ODBC-compliant database.

  • Specify elements, such as which tables (or views) and columns to display, how to order query results, and what values to search for.

    Note   In Oracle you can use synonyms as well. For details, see Query Designer Considerations for Oracle Databases.

  • Preview query results in the Results pane.

  • Join tables to create multi-table queries.

  • Edit databases directly by updating, inserting, or deleting rows.

  • Create queries that modify a database by updating, adding, or deleting rows, or by copying rows.

  • Create special-purpose queries, such as parameter queries in which search values are provided when the query is executed.

If you are familiar with SQL, you can also enter SQL statements directly or edit the SQL statements created by the Query Designer.
