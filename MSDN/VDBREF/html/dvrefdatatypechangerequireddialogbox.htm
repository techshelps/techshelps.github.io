Datatype Change Required Dialog Box

      

Appears when you change the data type, length, scale, or precision of a column that participates in a relationship.

Options

Yes

Changes the data type property of the related columns shown in the list so that existing relationships are preserved.

No

Cancels the data type change and restores the previous data type to the column you just changed.

Note   Although you cannot undo changes made to a table or database diagram, the changes are not saved to the database until you save the table or diagram. You can discard any unsaved changes by closing all open diagrams without saving them.
