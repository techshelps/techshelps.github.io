Formatting a Database Diagram

You can format your database diagram in a number of convenient ways. You can choose to have the diagram's layout automatically changed to best show the tables you're currently working on, or automatically display labels for relationship lines. The database diagram can display annotations you create for the diagram and its objects. You can also change the zoom setting of the diagram to display just the amount of the diagram you'd like to see.

Add notes to a database diagram Annotating Diagrams
