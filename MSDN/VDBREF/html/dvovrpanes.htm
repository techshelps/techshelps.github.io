Query Designer Window

The Query Designer provides a series of panes in which you can design, edit, and test your query. The panes are synchronized so that changes you make in one are reflected in the others, providing you with both a graphical and textual version of your query at all times.

Each pane provides different features for creating your query. For details about how each pane is used, see the topics listed in the following table

To See
Add tables or views to your query, select columns to work with, and join tables Diagram Pane
Specify options for displaying columns, sorting the results, searching, and grouping results Grid Pane
Enter SQL commands directly SQL Pane
View the results of your most recent Select query, and edit data directly Results Pane
