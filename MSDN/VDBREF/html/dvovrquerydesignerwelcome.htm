Query Designer

The Microsoft® Visual Database Tools Query Designer is a powerful tool for creating SQL commands using a simple, intuitive visual interface. You can create even complex, multi-table queries by simply dragging tables into your query, selecting options, and entering values.

With the Query Designer you can:

  • Work visually or with SQL commands   The Query Designer includes both graphical panes that display your query visually and an SQL pane that displays the SQL text of your query. You can work in either the graphical or SQL panes — the Query Designer keeps both views synchronized so they are always current.

  • Create database-aware commands   The Query Designer is designed to be aware of differences between databases such as Microsoft® SQL Server™ and Oracle, so it can generate and recognize database-specific SQL commands.

  • Join related tables   If you add more than one table to your query, the Query Designer automatically determines how the tables are related and constructs the appropriate join command in your query.

  • Query or update databases   You can use the Query Designer both to return data using SQL SELECT commands or to create queries that update, add, or delete records in a database, such as UPDATE, INSERT, and DELETE.

  • View and edit results immediately   You can execute your query and work with a recordset in a grid that allows you to scroll through and edit records in the database.
