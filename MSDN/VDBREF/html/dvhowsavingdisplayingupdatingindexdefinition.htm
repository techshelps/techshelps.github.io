Saving, Displaying, and Updating an Index Definition

Your index is automatically saved in the database (and defined in the database catalog) when you save your table or database diagram. The index is available for modification as soon as you create it.

To display and update index definitions

  1. In your database diagram, select the table whose index definitions you want to display.

  2. Right-click the table, and choose Properties from the shortcut menu.

  3. Choose the Indexes/Keys tab.

  4. To update the definition, select the field you want to update and change its value or setting.

Your changes are saved to the database when you save the table or diagram.
