Returning Query Results (Edit) Dialog Box

      

Notifies you that the Query Designer cannot save changes to the row you have edited because a query is still in progress. This dialog box appears when you execute a query, and then before the query has been completed, you edit a row in the Results pane and attempt to save it by moving to another row in the grid.

When working with some databases, the Query Designer waits until the query is finished before it saves the row you have edited and moves to the requested row. To help you estimate how long you must wait, the dialog box displays a progress counter.

Cancel

Choose this button to specify that you want to cancel your attempt to move to another row. The Query Designer returns you to the row you have edited, and you can save the changes later when the query is complete.

If you do not choose Cancel, the Query Designer saves your changes after the query has finished.
