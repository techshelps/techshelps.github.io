Make Your Messages Searchable

Place words that are both searchable and relevant to the central meaning at the beginning of the message string. For example: 

Recycle Bin cannot store some of the items you are about to delete.
   NOT:  Some of the items you are about to delete cannot be 
         stored in the Recycle Bin.

Endpoint was not found.
   NOT:  No endpoint was found.

Move you specified requires moving text cards.
   NOT:  That move requires moving text cards.

Log file filename is full.
   Not:  The filename log file is full.

Computer name as entered is not valid. 
   NOT:  The computer name you typed is invalid.
   NOT:  Invalid computer name. 

Chat application cannot start.
   NOT:  The application cannot start" in Chat component.
