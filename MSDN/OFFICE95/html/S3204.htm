InsertAddress

Syntax

InsertAddress

Remarks

Opens the Address Book window of the default Personal Address Book and inserts the selected address at the insertion point.

InsertAddress is available only if Windows 95 and either Microsoft Exchange or Schedule+ version 2.0 are installed. In Word version 6.0, InsertAddress is unavailable and generates an error.
