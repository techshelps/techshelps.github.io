Rotation Property

Applies To

Chart Object.

Description

Returns or sets the rotation of the 3-D chart view (the rotation of the plot area around the z-axis, in degrees). The value of this property must be between 0 and 360, except for 3-D Bar charts, where the value must be between 0 and 44. The default value is 20. Applies only to 3-D charts. Read-write.

See Also

Elevation Property, Perspective Property, RightAngleAxes Property.

Example

This example sets the rotation of Chart1 to 30 degrees. The example should be run on a 3-D chart.


Charts("Chart1").Rotation = 30
