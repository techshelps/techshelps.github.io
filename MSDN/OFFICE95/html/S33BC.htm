ViewRibbon, ViewRibbon()

Syntax

ViewRibbon [On]

ViewRibbon()

Remarks

The ViewRibbon statement displays or hides the Formatting toolbar. This statement is included for compatibility with the previous version of Word.

Argument

Explanation

On

Specifies whether to display or hide the Formatting toolbar:

0 (zero) Hides the Formatting toolbar.

1 Displays the Formatting toolbar.

Omitted Toggles the Formatting toolbar on and off.


The ViewRibbon() function returns the following values.

Value

Explanation

0 (zero)

If the Formatting toolbar is hidden

–1

If the Formatting toolbar is displayed
