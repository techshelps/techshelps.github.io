InsertColumnBreak

Syntax

InsertColumnBreak

Remarks

Inserts a column break at the insertion point. If the insertion point is in a table, InsertColumnBreak splits the table by inserting a paragraph mark above the row containing the insertion point. The instruction InsertColumnBreak is the same as the instruction InsertBreak .Type = 1.
