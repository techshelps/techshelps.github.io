NextMisspelling

Syntax

NextMisspelling

Remarks

Finds the next misspelled word. If automatic spell checking is turned on, NextMisspelling opens the shortcut menu containing spelling suggestions. For an example, see DocumentHasMisspellings(). In Word version 6.0, NextMisspelling is unavailable and generates an error.

See Also

DocumentHasMisspellings(), ToolsGetSpelling, ToolsLanguage, ToolsOptionsSpelling
