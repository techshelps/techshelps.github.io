MailMergeInsertMergeSeq

Syntax

MailMergeInsertMergeSeq

Remarks

Inserts a MERGESEQ field into a main document at the insertion point. A MERGESEQ field inserts a number based on the sequence in which data records are merged (for example, when merging records 50 to 100, inserts 1 when merging record number 50).
