ShowNextHeaderFooter

Syntax

ShowNextHeaderFooter

Remarks

If the insertion point is in a header, moves to the next header within the
current section (for example, from an odd header to an even header) or to the
first header in the following section. If the insertion point is in a footer, ShowNextHeaderFooter moves to the next footer. If the insertion point is in the last header or footer in the last section of the document, or is not in a header or footer, an error occurs.
