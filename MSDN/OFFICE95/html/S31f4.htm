HelpWordPerfectHelp

Syntax

HelpWordPerfectHelp

Remarks

In Windows, used with a Dialog or Dialog() instruction to display the Help For WordPerfect® Users dialog box (Help menu), as shown in the example for this entry. On the Macintosh, HelpWordPerfectHelp is not available and generates an error.

Example

This Windows example displays the Help For WordPerfect Users dialog box:


Dim dlg As HelpWordPerfectHelp
x = Dialog (dlg)
