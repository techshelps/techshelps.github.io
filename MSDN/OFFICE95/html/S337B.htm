ToolsLanguage

Syntax

ToolsLanguage .Language = text [, .Default]

Remarks

Sets the language format for the selected text. The language format identifies the text for the proofing tools in Word so that rules appropriate to the specified language are used.

Argument

Explanation

.Language

The name of the language

.Default

Adds the specified language format to the definition of the active template's Normal style


The following list includes valid settings for .Language. Note that languages marked with an asterisk (*) are valid only in Word version 7.0.
