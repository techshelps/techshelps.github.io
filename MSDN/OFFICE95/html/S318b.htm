FilePrintOneCopy

Syntax

FilePrintOneCopy

Remarks

On the Macintosh with QuickDraw GX installed, prints a single copy of the active document. This statement corresponds to the Print One Copy command that is added to the File menu if you install QuickDraw GX. In Windows, and on the Macintosh if QuickDraw GX is not installed, FilePrintOneCopy is not available and generates an error.
