GoToNextItem

Syntax

GoToNextItem

Remarks

Moves the insertion point to the next item specified by Item.

Argument

Explanation

Item

The item you want to go to. The following statements are available:

GoToNextAnnotation

GoToNextEndnote

GoToNextFootnote

GoToNextPage

GoToNextSection

GoToNextSubdocument


See Also

EditGoTo, GoToHeaderFooter, GoToPreviousItem
