ShowPrevHeaderFooter

Syntax

ShowPrevHeaderFooter

Remarks

If the insertion point is in a header, moves to the previous header within
the current section (for example, from an even header to an odd header) or
to the last header in the previous section. If the insertion point is in a footer, ShowNextHeaderFooter moves to the previous footer. If the insertion point is in the first header or footer in the first section of the document, or is not in a header or footer, an error occurs.
