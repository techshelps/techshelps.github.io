SetFileCreatorAndType

Syntax

SetFileCreatorAndType File$, Creator$ [, Type$]

Remarks

On the Macintosh, sets the creator (application signature) and file type for the specified file.

Argument

Explanation

File$

The path and filename of the file whose creator and type you want to set.

Creator$

A four-character application signature (for example, MSWD for Word) or a string returned by the MacID$() function.

Type$

A four-character string specifying a file type (for example, W6BN for a Word document) or a string returned by the MacID$() function.


In Windows, SetFileCreatorAndType is not available and generates an error.

See Also
