DBEngine Property

Applies To

Application Object.

Description

You can use the DBEngine property in Visual Basic to access the current DBEngine object and its related properties.

Setting

The DBEngine property is set by Microsoft Access and is read-only.

Remarks

The DBEngine property of the Application object represents the Microsoft Jet database engine. The DBEngine object is the top-level object in the Data Access Object (DAO) model and it contains and controls all other objects in the hierarchy of data access objects.
