FileNumber

Syntax

FileNumber

Remarks

Opens one of the files in the list of recently used files at the bottom of the File menu. (Word defines "used" as either opened or saved.) File1 opens the first document in the list (the most recently used), File2 opens the second document, and so on through File9. Nine is the maximum number of filenames that can be listed on the File menu. If no file is listed, or if Number is greater than the number of files listed, an error occurs. If the specified file is already open, Word activates the document.

You can control the number of files to be listed on the File menu with the Recently Used File List option on the General tab in the Options dialog box (Tools menu).
