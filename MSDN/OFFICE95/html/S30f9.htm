DrawBringInFrontOfText

Syntax

DrawBringInFrontOfText

Remarks

Moves the selected drawing object from the layer behind text to the layer in front of text. The object is placed behind any other drawing objects already in front of the text layer. An error occurs if the object is already part of the drawing layer above text or if the selection is not a drawing object.
