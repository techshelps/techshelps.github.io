CentimetersToPoints Method

Applies To

Application Object.

Description

Converts a measurement in centimeters into points (0.035 centimeters).

Syntax

object.CentimetersToPoints(centimeters)

object

Required. The Application object.

centimeters

Required. Specifies the centimeter value to convert to points.

See Also

InchesToPoints Method.

Example

This example sets the left margin of Sheet1 to 5 centimeters.


Worksheets("Sheet1").PageSetup.LeftMargin = _
        Application.CentimetersToPoints(5)
