ToolsCustomize

Syntax

ToolsCustomize [.Tab = number]

Remarks

Displays the Customize dialog box (Tools menu) with the specified tab selected.

Argument

Explanation

.Tab

The tab to select:

0 (zero) Toolbars

1 Menus

2 Keyboard
