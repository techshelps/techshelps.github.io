DocWindowPosTop, DocWindowPosTop()

Syntax

DocWindowPosTop Position

DocWindowPosTop()

Remarks

The DocWindowPosTop statement positions the active document window or icon so that it is Position points from the upper border of the workspace. If the active document is maximized, Word cannot perform this action and generates an error. The DocWindowPosTop() function returns the vertical position in points. For an example, see DocWindowPosLeft.
