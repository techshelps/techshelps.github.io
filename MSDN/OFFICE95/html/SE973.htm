Getting Help

Help on Visual Basic for Applications is available in a separate help file. In Excel 5, get to it by pressing F1 or choosing the Help/Contents menu item, and then choosing Programming With Visual Basic. In Excel 7, choose Help/Microsoft Excel Help Topics, then use the Contents, Index, or Find tabs to search for the topic. To get help on a VBA keyword while editing a module sheet just select the keyword in the code, then press F1. VBA help will automatically search for the keyword and display help on it.
