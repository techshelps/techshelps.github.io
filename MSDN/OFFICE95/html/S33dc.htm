Word Error Messages

These error messages are generated outside WordBasic, by Word itself. Word always displays an error message box for these messages and waits for the user to choose the OK button, regardless of whether a macro contains error-trapping statements. Once the user responds, Word returns control to the WordBasic macro, and the error can be trapped and handled like any other. Note that the Error statement cannot create these error conditions, nor can it be used to display these error messages.
