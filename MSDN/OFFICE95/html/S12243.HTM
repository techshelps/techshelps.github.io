WhatsThisButton Property

Applies To

Form.

Description

You can use the WhatsThisButton property to specify whether a Whats This button is added to a form’s title bar.

Note You can’t display the Whats This button on the title bar of a form unless the MinMaxButtons property is set to None.

Setting

The WhatsThisButton property uses the following settings.

Setting

Description

Visual Basic

Yes

A Whats This help button appears on the title bar.

True (-1)

No

(Default) A Whats This help button doesn’t appear on the title bar.

False (0)


You can set the WhatsThisButton property in the form’s property sheet, a macro, or using Visual Basic.

This property can be set only in form Design view.

Remarks

Choosing the Whats This button from the title bar of a form causes the Help Select mouse pointer to appear. With the Help Select mouse pointer, you can click any control to access its custom Help topic specified by the control’s HelpContextID property. If the control doesn’t have a custom Help topic, the form’s custom Help topic is displayed. If neither the form or the control has a custom Help topic, Microsoft Access Help is displayed.
