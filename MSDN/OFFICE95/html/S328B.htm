MicrosoftApplication

Syntax

MicrosoftApplication

Remarks

Starts the specified Microsoft application if it is not running or switches to the application if it is already running.

Argument

Explanation

Application

The application you want to go to. The following statements are available:

MicrosoftAccess (not available on the Macintosh)

MicrosoftExcel

MicrosoftFoxPro

MicrosoftMail

MicrosoftPowerPoint

MicrosoftProject

MicrosoftPublisher (not available on the Macintosh)

MicrosoftSchedule (not available on the Macintosh)

MicrosoftSystemInfo


See Also

AppActivate, AppIsRunning()
