User-Defined Data Type

Description

Any data type you define using the Type statement. User-defined data types can contain one or more elements of any data type, array, or a previously defined user-defined type. For example:


Type MyType
    MyName As String    ' String variable stores a name.
    MyBirthDate As Date    ' Date variable stores a birthdate.
    MySex as Integer    ' Integer variable stores sex (0 for 
End Type    ' female, 1 for male.
