DrawInsertWordPicture

Syntax

DrawInsertWordPicture

Remarks

Opens a temporary document (a Word Picture object) and displays the Picture and Drawing toolbars. When the user closes the document or clicks Close Picture on the Picture toolbar, the object is embedded in the document that was active when DrawInsertWordPicture was run. The instruction DrawInsertWordPicture is the same as the instruction InsertObject .Class = "Word.Picture.6".
