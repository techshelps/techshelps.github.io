FileExit

Syntax

FileExit [Save]

Remarks

Quits Word. FileExit (Windows) is the same as FileQuit (Macintosh).

Argument

Explanation

Save

Determines whether Word saves each document before closing it if it is "dirty" — that is, if changes have been made since the last time the file was saved:

0 (zero) or omitted Prompts the user to save each changed document.

1 Saves all edited documents before quitting.

2 Quits without saving changed documents.

The Save argument also controls whether a prompt appears if a document has a routing slip or if Word is printing in the background. Prompts appear if Save is 0 (zero) or omitted; otherwise, Word quits without routing documents or finishing print jobs.
