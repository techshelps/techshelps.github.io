EditReplaceClearFormatting

Syntax

EditReplaceClearFormatting

Remarks

Clears any formats specified for replacement text. Before specifying replacement formats with EditReplaceFont, EditReplacePara, and so on, it's a good idea to include the EditReplaceClearFormatting statement to ensure that any unwanted formats are not included in the replacement formatting. For an example, see EditReplace.
