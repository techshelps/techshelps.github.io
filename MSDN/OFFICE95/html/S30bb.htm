CreateSubdocument

Syntax

CreateSubdocument

Remarks

Converts the selected outline headings into subdocuments. If the active document is not in master document or outline view, or if the selection spans a subdocument boundary, an error occurs. An error is also generated if the first paragraph selected is not a heading.
