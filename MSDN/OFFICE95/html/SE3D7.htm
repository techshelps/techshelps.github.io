Display3DShading Property

Applies To

CheckBox Object, DrawingObjects Collection, DropDown Object, DropDowns Collection, GroupBox Object, GroupBoxes Collection, ListBox Object, ListBoxes Collection, OptionButton Object, OptionButtons Collection, ScrollBar Object, ScrollBars Collection, Spinner Object, Spinners Collection.

Description

True if the control uses 3-D visual effects. This property applies only to controls on worksheets and charts. Read-write.

Example

This example sets check box one on Sheet1 to use 3-D shading.


Worksheets("Sheet1").CheckBoxes(1).Display3DShading = True
