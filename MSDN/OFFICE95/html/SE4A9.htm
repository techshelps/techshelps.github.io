IncludeProtection Property

Applies To

Style Object.

Description

True if the style includes the FormulaHidden and Locked protection properties. Read-write.

See Also

IncludeAlignment Property, IncludeBorder Property, IncludeFont Property, IncludeNumber Property, IncludePatterns Property.

Example

This example sets the style attached to cell A1 on Sheet1 to include protection format.


Worksheets("Sheet1").Range("A1").Style.IncludeProtection = True
