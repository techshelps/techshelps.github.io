LeftPara, LeftPara()

Syntax

LeftPara

LeftPara()

Remarks

The LeftPara statement aligns the selected paragraphs with the left indent.

The LeftPara() function returns the following values.

Value

Explanation

0 (zero)

If none of the selection is left-aligned

–1

If part of the selection is left-aligned

1

If all the selection is left-aligned
