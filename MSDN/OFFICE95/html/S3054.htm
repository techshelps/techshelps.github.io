AOCEAddRecipient

Syntax

AOCEAddRecipient Recipient$ [, AddressType] [, MailSlot$]

Remarks

Adds the specified recipient address to the mailer attached to the active document. AOCEAddRecipient is available only on the Macintosh and only if PowerTalk™ is installed.

Argument

Explanation

Recipient$

The address of the recipient

AddressType

Specifies the list to add the recipient to:

13 or omitted "To" list

14 "Cc" list

15 "Bcc" list (the recipient's name appears in neither the "To" nor "Cc" lists)

MailSlot$

Specifies how the user will receive the document (for example, "AppleTalk®" or "Fax")
