FormatSectionLayout

Syntax

FormatSectionLayout [.SectionStart = number] [, .VertAlign = number] [, .Endnotes = number] [, .LineNum = number] [, .StartingNum = number] [, .FromText = text] [, .CountBy = text] [, .NumMode = number]

Remarks

Applies section formatting to the selected sections. This command exists for compatibility with earlier versions of Word. The arguments for the FormatSectionLayout statement are duplicated in FilePageSetup.
