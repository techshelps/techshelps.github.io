UNLOCKED.NEXT
UNLOCKED.PREV

Macro Sheets Only

Equivalent to pressing TAB or SHIFT+TAB to move to the next or previous unlocked cell in a protected worksheet. Use these functions when you want to control which cell is active on a protected sheet.

Syntax

UNLOCKED.NEXT( )
UNLOCKED.PREV( )

Related Functions

CELL.PROTECTION Controls protection for the selected cells

PROTECT.DOCUMENT Controls protection for the active document
