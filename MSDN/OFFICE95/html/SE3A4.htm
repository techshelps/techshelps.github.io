DataLabel Property

Applies To

Point Object, Trendline Object.

Description

Accessor. Returns a DataLabel object that represents the data label associated with the point or trendline. Read-only.

See Also

ApplyDataLabels Method, HasDataLabel Property.

Example

This example turns on the data label for point seven in series three in Chart1, and then it sets the data label color to blue.


With Charts("Chart1").SeriesCollection(3).Points(7)
    .HasDataLabel = True
    .ApplyDataLabels type:=xlValue
    .DataLabel.Font.ColorIndex = 5
End With
