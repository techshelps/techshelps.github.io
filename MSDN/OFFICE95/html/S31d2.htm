GetDocumentVarName$()

Syntax

GetDocumentVarName$(VariableNumber)

Remarks

Returns the name of a document variable set with SetDocumentVar or SetDocumentVar().

Argument

Explanation

VariableNumber

The number of the document variable, from 1 to the total number of document variables stored in the active document (you can obtain the total using CountDocumentVars()).


For an example, see CountDocumentVars().

See Also

CountDocumentVars(), GetDocumentVar$(), SetDocumentVar
