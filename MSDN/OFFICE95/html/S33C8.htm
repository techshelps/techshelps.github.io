VPage

Syntax

VPage [Count]

Remarks

Scrolls the active document vertically. VPage corresponds to clicking the vertical scroll bar above or below the scroll box.

Argument

Explanation

Count

The amount to scroll, in screens:

Omitted One screen down

> 0 (zero) The specified number of screens down

< 0 (zero) The specified number of screens up


See Also

HPage, VLine, VScroll
