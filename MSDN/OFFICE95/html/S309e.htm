Converter$()

Syntax

Converter$(FormatNumber)

Remarks

Returns the class name of the file format associated with FormatNumber.

Argument

Explanation

FormatNumber

A number corresponding to a file format listed under Save File As Type in the Save As dialog box (File menu): 0 (zero) corresponds to the first format, 1 to the second format, and so on.

FormatNumber can also be a value returned by ConverterLookup(). Note that in some cases, the format number returned by ConverterLookup() does not match the position of the format name in the list box. In Windows for example, ConverterLookup("Excel Worksheet") returns 101.


Example

This example inserts at the insertion point a list of file format class names available to Word. The instructions run until a$ is an empty string (""), indicating that the end of the list of available formats has been reached.


x = 0
a$ = Converter$(x)
While a$ <> ""
    Insert a$ + Chr$(13)
    x = x + 1
    a$ = Converter$(x)
Wend
