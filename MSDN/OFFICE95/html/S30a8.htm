CountAutoTextEntries()

Syntax

CountAutoTextEntries([Context])

Remarks

Returns the number of AutoText entries defined for the specified context.

Argument

Explanation

Context

The context in which to count AutoText entries:

0 (zero) or omitted Normal template and any loaded global templates

1 Active template

Note that if Context is 1 and the active template is the Normal template, CountAutoTextEntries() returns 0 (zero).


For an example, see AutoTextName$().
