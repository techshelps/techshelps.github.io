DemoteList

Syntax

DemoteList

Remarks

Demotes the selected paragraphs by one level in a multilevel list. If the selected paragraphs are formatted as a bulleted list or as a numbered list that isn't

multilevel, DemoteList increases the indent. If the selected paragraphs are not already formatted as a numbered or bulleted list, an error occurs.
