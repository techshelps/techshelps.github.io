Chart Property

Applies To

ChartObject Object.

Description

Accessor. Returns a Chart object that represents the chart contained in the object. Read-only.

Example

This example adds a title to the first embedded chart on Sheet1.


With Worksheets("Sheet1").ChartObjects(1).Chart
    .HasTitle = True
    .ChartTitle.Text = "1995 Rainfall Totals by Month"
End With
