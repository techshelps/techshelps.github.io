GetDirectory$()

Syntax

GetDirectory$([Directory$,] Count)

Remarks

Returns the name of the subfolder specified by Count in the specified folder.

Argument

Explanation

Directory$

The path of the folder containing the subfolder; if omitted, the current folder is assumed.

Count

A number in the range 1 to CountDirectories(), where 1 corresponds to the first subfolder, 2 to the second, and so on.


Example

This example defines an array with the names of the subfolders in the current folder. In a dialog box definition, you could specify this array in a ListBox instruction to present a list of the subfolders.


Dim subdirs$(CountDirectories())
For i = 1 To CountDirectories()
    subdirs$(i) = LCase$(GetDirectory$(i))
Next
