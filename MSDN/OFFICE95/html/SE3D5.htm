DirectPrecedents Property

Applies To

Range Object.

Description

Accessor. Returns a Range object that represents the range that contains all of the direct precedents of a cell. This may be a multiple selection (a union of Range objects) if there is more than one precedent. Read-only.

See Also

Dependents Property, DirectDependents Property, Precedents Property, ShowPrecedents Method.

Example

This example selects the direct precedents of cell A1 on Sheet1.


Worksheets("Sheet1").Activate
Range("A1").DirectPrecedents.Select
