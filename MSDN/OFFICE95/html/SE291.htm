Where to Find Information About Visual Basic for Microsoft Excel

For general information about Microsoft Excel, see Getting Results with Microsoft Excel for Windows 95 or Getting Results with Microsoft Office for Windows 95.

For in-depth information about using Visual Basic in Microsoft Excel, read the Microsoft Excel/Visual Basic for Windows 95 Programmer's Guide, available from Microsoft Press. If you're new to Visual Basic in Microsoft Excel, or if you prefer a task-oriented approach to learning, you might want to start by reading Microsoft Excel Visual Basic Step by Step, a self-paced tutorial also available from Microsoft Press. Another Microsoft Press book, Developing Microsoft Excel Solutions, contains expert development strategies, tips, tricks, and sample Microsoft Excel/Visual Basic macros and applications.
