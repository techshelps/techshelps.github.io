ShowAnnotationBy

Syntax

ShowAnnotationBy ReviewerName$

Remarks

Displays the annotations by ReviewerName$ when the annotation pane is open. To display all annotations, specify "All Reviewers" as ReviewerName$. If ReviewerName$ does not match a reviewer in the list at the top of the annotation pane, Word displays all annotations.

Example

This example instructs Word to display annotations by Sara Levine only, and then displays the annotation pane:


ShowAnnotationBy "Sara Levine"
ViewAnnotations 1
