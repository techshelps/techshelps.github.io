Mailer Property

Applies To

Workbook Object.

Description

Accessor. Returns a Mailer object that represents the PowerTalk mailer attached to the workbook. Available only in Microsoft Excel for the Apple Macintosh with the PowerTalk mail system extension installed. Read-only.

Remarks

The Mailer object contains the properties needed to mail workbooks with PowerTalk. To mail a workbook, turn on the mailer with the HasMailer property, set the mailer properties, and then send the workbook and mailer with the SendMailer method.

See Also

BCCRecipients Property, CCRecipients Property, Enclosures Property, HasMailer Property, Received Property, SendDateTime Property, Sender Property, SendMailer Method, Subject Property, ToRecipients Property.

Example

This example sets up the Mailer object for workbook one and then sends the workbook.


With Workbooks(1)
    .HasMailer = True
    With .Mailer
        .Subject = "Here is the workbook"
        .ToRecipients = Array("Jean")
        .CCRecipients = Array("Adam", "Bernard")
        .BCCRecipients = Array("Chris")
        .Enclosures = Array("TestFile")
    End With
    .SendMailer
End With
