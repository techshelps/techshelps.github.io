JustifyPara, JustifyPara()

Syntax

JustifyPara

JustifyPara()

Remarks

The JustifyPara statement justifies the selected paragraphs. The JustifyPara() function returns the following values.

Value

Explanation

0 (zero)

If none of the selection is justified

–1

If part of the selection is justified or there is a mix of alignments

1

If all the selection is justified
