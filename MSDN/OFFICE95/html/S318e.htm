FilePrintPreviewPages, FilePrintPreviewPages()

Syntax

FilePrintPreviewPages PagesAcross

FilePrintPreviewPages()

Remarks

The FilePrintPreviewPages statement specifies whether to display one or two columns of pages in page layout view and print preview. PagesAcross can be 1
or 2 only; for more control over the grid of page images, use ViewZoom. The FilePrintPreviewPages() function returns the number of columns of pages currently displayed. If the active document is not in either page layout view or print preview, an error occurs.
