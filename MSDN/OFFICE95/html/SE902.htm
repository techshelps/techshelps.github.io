ROUTE.DOCUMENT

Macro Sheets Only

Routes the document using the defined routing slip information.

Syntax

ROUTE.DOCUMENT()

Remarks

If there is no routing slip, returns #N/A. If an error occurs or routing is not enabled for the system, returns #VALUE!.

Related Functions
