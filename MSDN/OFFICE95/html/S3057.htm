AOCECountRecipients()

Syntax

AOCECountRecipients(AddressType)

Remarks

Returns the number of recipients in the mailer attached to the active document. AOCECountRecipients() is available only on the Macintosh and only if PowerTalk is installed.

Argument

Explanation

AddressType

The type of recipients to count:

0 (zero) All recipients

13 "To" recipients

14 "Cc" recipients

15 "Bcc" recipients


See Also

AOCEAddRecipient, AOCEGetRecipient$()
