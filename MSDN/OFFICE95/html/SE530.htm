Message Property

Applies To

RoutingSlip Object.

Description

Returns or sets the message text of the routing slip. This text is used as the body text of mail messages used to route the workbook. Read-write.

See Also

Subject Property.

Example

This example sends BOOK1.XLS to three recipients, one after the other.


Workbooks("BOOK1.XLS").HasRoutingSlip = True
With Workbooks("BOOK1.XLS").RoutingSlip
    .Delivery = xlOneAfterAnother
    .Recipients = Array("Adam Bendel", "Jean Selva", "Bernard Gabor")
    .Subject = "Here is BOOK1.XLS"
    .Message = "Here is the workbook. What do you think?"
End With
Workbooks("BOOK1.XLS").Route
