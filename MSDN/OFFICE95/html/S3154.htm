EditReplaceTabs

Syntax

EditReplaceTabs [.Position = text] [, .Align = number] [, .Leader = number]

Remarks

On the Macintosh, when followed by an EditReplace instruction in which .Format is set to 1, specifies tab setting for the replacement text. For argument descriptions, see FormatTabs. In Windows, EditReplaceTabs is not available and generates an error.
