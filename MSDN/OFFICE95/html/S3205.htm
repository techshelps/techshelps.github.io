InsertAnnotation

Syntax

InsertAnnotation

Remarks

Inserts an annotation mark at the insertion point, opens the annotation pane, and places the insertion point in the annotation pane. If text is selected, the annotation mark is inserted immediately after the selected text, which appears shaded when the insertion point is in the associated annotation in the annotation pane. If the document window is too small to display the annotation pane, an error occurs.
