HasMailer Property

Applies To

Workbook Object.

Description

True if the workbook has a mailer. Available only in Microsoft Excel for the Apple Macintosh with the PowerTalk mail system extension installed. Read-write.

See Also

Mailer Property, MailSystem Property, SendMailer Method.

Example

This example replies to the sender of the active workbook.


Set original = ActiveWorkbook
If original.HasMailer Then
    original.Reply
    original.Mailer.Subject = "Here's my reply"
    ActiveWorkbook.SendMailer
End If
