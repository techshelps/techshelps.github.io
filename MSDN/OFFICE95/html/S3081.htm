BorderTop, BorderTop()

Syntax

BorderTop [On]

BorderTop()

Remarks

The BorderTop statement applies or removes a top border for the selected paragraphs, table cells, or graphic. Note that when you apply a top border to
a series of paragraphs or table rows, the border appears only above the first paragraph or row in the series. If you want a border to separate each paragraph
or row, use BorderInside.

The BorderTop() function returns either 0 (zero) or 1, depending on whether the selected graphic or all the selected paragraphs or table cells are formatted with a top border.

For complete descriptions of arguments and return values, see BorderBottom.
