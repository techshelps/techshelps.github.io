ChartTitle Property

Applies To

Chart Object.

Description

Accessor. Returns a ChartTitle object that represents the title of the specified chart. Read-only.

See Also

AxisTitle Property, HasTitle Property, Title Property.

Example

This example sets the text for the title of Chart1.


With Charts("Chart1")
    .HasTitle = True
    .ChartTitle.Text = "First Quarter Sales"
End With
