SpellChecked, SpellChecked()

Syntax

SpellChecked [On]

SpellChecked()

Remarks

The SpellChecked statement identifies the selected text as either checked for spelling errors or not checked. If no text is selected, SpellChecked and SpellChecked() have no effect.

In Word version 6.0, SpellChecked and SpellChecked() are unavailable and generate errors.

Argument

Explanation

On

Specifies whether to identify the selection as checked or not checked for spelling errors:

1 Identifies the selection as checked

0 (zero) Identifies the selection as not checked

Omitted Toggles the setting


The SpellChecked() function returns the following values.

Value

Explanation

0 (zero)

If the selection is not checked for spelling

-1

If only part of the selection is checked for spelling

1

If all the selection is checked for spelling
