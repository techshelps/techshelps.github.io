RGB Function

Description

Returns a whole number representing an RGB color value.

Syntax

RGB (red, green, blue)

The RGB function syntax has these named arguments:

Part

Description

red

Number in the range 0 to 255, inclusive, that represents the red component of the color.

green

Number in the range 0 to 255, inclusive, that represents the green component of the color.

blue

Number in the range 0 to 255, inclusive, that represents the blue component of the color.


Remarks

Application methods and properties that accept a color specification expect that specification to be a number representing an RGB color value. An RGB color value specifies the relative intensity of red, green, and blue to cause a specific color to be displayed.

The value for any argument to RGB that exceeds 255 is assumed to be 255.
