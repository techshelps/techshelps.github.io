Eqv Operator

Description

Used to perform a logical equivalence on two expressions.

Syntax

result = expression1 Eqv expression2

The Eqv operator syntax has these parts:

Part

Description

result

Any numeric variable.

expression1

Any expression.

expression2

Any expression.
