Minute Function

Description

Returns a whole number between 0 and 59, inclusive, representing the minute of the hour.

Syntax

Minute(time)

The time argument is any numeric or string expression, or any combination, that can represent a time. If time contains Null, Null is returned.

See Also

Day Function, Hour Function, Now Function, Second Function, Time Function, Time Statement.

Example

This example uses the Minute function to obtain the minute of the hour from a specified time. In the development environment, the time (date literal) is displayed in short time format using the locale settings of your code.


MyTime = #4:35:17 PM#        ' Assign a time.= Minute(MyTime)    ' MyMinute contains 35.
