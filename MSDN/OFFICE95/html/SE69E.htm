SplitRow Property

Applies To

Window Object.

Description

Returns or sets the row number where the window is split into panes (the number of rows above the split). Read-write.

See Also

SplitColumn Property, SplitHorizontal Property, SplitVertical Property.

Example

This example splits the active window so that there are 10 rows above the split line.


Workbooks("BOOK1.XLS").Worksheets("Sheet1").Activate
ActiveWindow.SplitRow = 10
