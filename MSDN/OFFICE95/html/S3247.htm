Lof()

Syntax

Lof([#]FileNumber)

Remarks

Returns the length of an open sequential file, in bytes. FileNumber is the number specified in the Open instruction that opened the file for input. For more information about sequential files, see Chapter 9, "More WordBasic Techniques," in Part 1, "Learning WordBasic."

Example

This example uses Open to prepare the file MYDATA.TXT for sequential input and then displays the size of the file MYDATA.TXT in the status bar:


Open "MYDATA.TXT" For Input As #1
    Size = Lof(1)
    Print Size; " bytes"
Close #1
