FormatChangeCase

Syntax

FormatChangeCase [.Type = number]

Remarks

Changes the case of the selected text according to the specified type of capitalization. The arguments for the FormatChangeCase statement correspond to the options in the Change Case dialog box (Format menu).

Argument

Explanation

.Type

The type of capitalization you want to apply:

0 (zero) or omitted Sentence case (capitalizes the first character in each selected sentence).

1 lowercase (changes selected characters to lowercase).

2 UPPERCASE (changes selected characters to uppercase).

3 Title Case (capitalizes the first letter of each selected word).

4 tOGGLE cASE (toggles the capitalization of each selected letter).
