Introduction

This manual is an alphabetic reference for Visual Basic for Microsoft Excel. The reference contains topics for all functions, methods, objects, properties, and statements in the language, and is essentially identical to the information contained in Microsoft Excel Visual Basic Help.

Unless otherwise noted, each topic in this manual is applicable to both Microsoft Excel 5 and Microsoft Excel for Windows 95. At the beginning of the reference, there is a section with lists of methods, objects, and properties that are new or changed in Microsoft Excel for Windows 95. References to other online and printed material refer to documentation for Microsoft Excel for Windows 95.
