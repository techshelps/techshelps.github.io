FileAOCESendMail

Syntax

FileAOCESendMail

Remarks

Displays a dialog box in which you can set options for sending the active document to the the recipients specified in the attached mailer. Use AOCESendMail to set options and send the document without displaying a dialog box. FileAOCESendMail is available only on the Macintosh and only if PowerTalk is installed.
