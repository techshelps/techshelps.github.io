InsertDrawing

Syntax

InsertDrawing

Remarks

If Microsoft Draw is installed on your computer, starts Microsoft Draw and displays a blank window. When the user exits Microsoft Draw, the drawing is embedded in the active document. The instruction InsertDrawing is the same as the instruction InsertObject .Class = "MSDraw".
