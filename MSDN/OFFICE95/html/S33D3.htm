WindowPane()

Syntax

WindowPane()

Remarks

Returns the following values.

Value

Explanation

1

If the active window is not split or if the insertion point is in the top pane of the active window

3

If the insertion point is in the bottom pane of the active window (for example, the footnote pane, the annotation pane, or the lower of two document panes)


Example

This example moves the insertion point to the top pane of the active document if the active window is split and the insertion point is in the bottom pane:


If WindowPane() = 3 Then OtherPane
