MailMergeOpenHeaderSource

Syntax

MailMergeOpenHeaderSource .Name = text [, .ConfirmConversions = number] [, .ReadOnly = number] [, .AddToMru = number] [, .PasswordDoc = text] [, .PasswordDot = text] [, .Revert = number] [, .WritePasswordDoc = text] [, .WritePasswordDot = text]

Remarks

Attaches the specified header source to the active document, which becomes a main document if it is not one already. The header record in the header source is used in place of the header record in the data source. For argument descriptions, see FileOpen.
