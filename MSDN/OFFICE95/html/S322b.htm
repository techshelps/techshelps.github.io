InsertWordArt

Syntax

InsertWordArt

Remarks

If Microsoft WordArt is installed on your computer, starts WordArt. When the user exits WordArt, the WordArt object is embedded in the active document. The instruction InsertWordArt is the same as the instruction InsertObject .Class = "MSWordArt.2".
