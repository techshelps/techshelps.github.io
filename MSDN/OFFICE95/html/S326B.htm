MailMergeInsertSkipIf

Syntax

MailMergeInsertSkipIf .MergeField = text , .Comparison = number [, .CompareTo = text]

Remarks

Inserts a SKIPIF field into a main document at the insertion point. If the comparison in the SKIPIF field is true, Word cancels the current merge document and skips to the next data record. In general, it is preferable to specify query options instead of using SKIPIF fields. For argument descriptions, see MailMergeInsertIf.
