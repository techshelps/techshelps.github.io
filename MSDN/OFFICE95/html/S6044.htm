Preparing a Workbook to Compile to an Add-In

You create a source workbook for an add-in the same way you create any other workbook. However, before you compile the source workbook to an add-in, there are several unique issues associated with add-ins that you need to be aware of.

Note

Because add-in applications cannot be edited, it's extremely important that you keep a copy of your original workbook so that you can update its procedures or add new procedures.
