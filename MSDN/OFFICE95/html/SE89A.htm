SET.CRITERIA

Macro Sheets Only

Equivalent to choosing the Set Criteria command from the Data menu in Microsoft Excel version 4.0. Defines the name Criteria for the selected range on a sheet or macro sheet.

Syntax

SET.CRITERIA( )
