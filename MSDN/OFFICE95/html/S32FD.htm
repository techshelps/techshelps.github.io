SetAutoText

Syntax

SetAutoText Name$, Text$ [, Context]

Remarks

Defines a text-only AutoText entry. Unlike an EditAutoText instruction that uses .Add, SetAutoText does not require a selection.

Argument

Explanation

Name$

The name of the new entry

Text$

The text to be associated with the entry

Context

Specifies the availability of the entry:

0 (zero) or omitted Normal template (available to all documents)

1 Active template (available only to documents based on the active template)

Note that if Context is 1 and the active template is the Normal template, SetAutoText generates an error.


Example

This example defines the AutoText entry "Disclaim" in the active template; "Disclaim" contains the text assigned to text$:


text$ = "No warranty is either expressed or implied."
SetAutoText "Disclaim", text$, 1
