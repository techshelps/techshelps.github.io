Appendix C

Microsoft Word Application Programming Interface

The Microsoft Word application programming interface (Word API) is the doorway into the internal functionality of Microsoft Word. Using a programming language such as Microsoft Visual C++ (Windows) or Symantec® THINK C (Macintosh), the Word API, and the tools provided on the Microsoft Word Developer's Kit disk, you can create add-ins that interact directly with Microsoft Word. This appendix covers the concepts, tools, and methods for programming Word add-in libraries (WLLs).
