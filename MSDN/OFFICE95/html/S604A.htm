Compiling the Add-In

After you've debugged your source workbook and addressed the referencing issues discussed in the preceding section, you're ready to compile your add-in.

To compile an add-in

1. Switch to a Visual Basic module in the workbook you want to compile.

2. On the Tools menu, click Make Add-In.

3. Click the icon for the folder in which you want to save the file.

4. In the File Name box, enter the name you want for the compiled add-in file.

5. Click Save.
