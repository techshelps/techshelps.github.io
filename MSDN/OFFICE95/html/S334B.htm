TableSplitCells

Syntax

TableSplitCells [.NumColumns = text]

Remarks

Splits each selected table cell. Word generates an error if the selection or insertion point is not within a table.

Argument

Explanation

.NumColumns

Specifies the number of cells to split each selected table cell into. If omitted, Word splits each cell into two cells.
