PASTE.PICTURE.LINK

Macro Sheets Only

Equivalent to holding down the SHIFT key and choosing the Paste Picture Link command from the Edit menu in Microsoft Excel version 4.0 or to using the camera tool. Pastes a linked picture of the Clipboard contents. This picture is linked, so changes to the source data will be reflected in the picture.

Syntax

PASTE.PICTURE.LINK( )

Related Functions

COPY.PICTURE Copies and pastes data or object

COPY.PICTURE Creates a picture of the current selection for use in another program

CREATE.OBJECT Creates an object

PASTE Pastes cut or copied data

PASTE.PICTURE Pastes a picture of the currently copied area
