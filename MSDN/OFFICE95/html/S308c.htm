CheckBox

Syntax

CheckBox HorizPos, VertPos, Width, Height, Label$, .Identifier

Remarks

Creates a check box in a custom dialog box.

Argument

Explanation

HorizPos, VertPos

The horizontal and vertical distance of the upper-left corner of the rectangle containing the check box and its associated label from the upper-left corner of the dialog box, in increments of 1/8 and 1/12 of the System font (Windows) or the dialog font (Macintosh).

Width, Height

The width and height of the check box, in increments of 1/8 and 1/12 of the System font (Windows) or the dialog font (Macintosh).


Argument

Explanation

Label$

The label associated with the check box. An ampersand (&) precedes the character in Label$ that is the access key for selecting and clearing the check box.

.Identifier

Together with the dialog record name, .Identifier creates a variable whose value corresponds to the state of the check box. The form for this variable is DialogRecord.Identifier (for example, dlg.MyCheckBox). DialogRecord.Identifier can return the following values:

0 (zero) The check box is cleared.

1 The check box is selected.

–1 The check box is filled with gray.

The identifier string (.Identifier minus the period) is also used by statements in a dialog function that act on the check box, such as DlgEnable and DlgVisible.


To see an example of CheckBox in a dialog box definition, see Begin Dialog¼End Dialog.
