Introduction

Welcome to Microsoft Access Language Reference, your reference for programming in Microsoft® Access. This book contains information on the Visual Basic® for Applications programming language and its use within Microsoft Access. It also contains information on Microsoft Access language elements that enhance the Visual Basic programming language and make it easier for you to create great database applications with Microsoft Access.

This book contains reference information on programming with Visual Basic in Microsoft Access only. It does not contain reference information on programming with data access objects. For more information, see online Help and the following two books:

  • Microsoft Office 95 Data Access Reference, available in the Microsoft Access Developer’s Toolkit, contains reference information on programming with data access objects and their methods and properties, and with Microsoft Jet Database Engine SQL.
  • Building Applications with Microsoft Access for Windows® 95, available with Microsoft Access 95, is a guide to designing and creating database applications with Microsoft Access.

The information included in this manual is the best available at the time of publication; in some cases, information in online Help may be more up-to-date.
