Sqr Function

Description

Returns the square root of a number.

Syntax

Sqr(number)

The number named argument can be any valid numeric expression greater than or equal to 0.

Example

This example uses the Sqr function to calculate the square root of a number.


MySqr = Sqr(4)    ' Returns 2.
MySqr = Sqr(23)    ' Returns 4.795832.
MySqr = Sqr(0)    ' Returns 0.
MySqr = Sqr(-4)    ' Generates run-time error.
