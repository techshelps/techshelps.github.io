CountToolsGrammarStatistics()

Syntax

CountToolsGrammarStatistics()

Remarks

Returns the number of statistics that are stored when you check grammar with the ToolsGrammar statement. You can use this number to define the size of a two-dimensional array that you fill using the ToolsGrammarStatisticsArray statement.

For an example, see ToolsGrammarStatisticsArray.

See Also

ToolsGrammar, ToolsGrammarStatisticsArray, ToolsOptionsGrammar
