FileSendMail

Syntax

FileSendMail

Remarks

Opens a message window for sending the active document through Microsoft Mail. Use the ToolsOptionsGeneral statement with the .SendMailAttach argument to control whether the document is sent as text in the message window or as an attachment. FileSendMail is available only if Microsoft Mail is installed.
