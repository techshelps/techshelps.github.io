JavaNamespaceFlag, JavaNamespaceFlags

The following flags are associated with this enumeration:

JNF_TYPESONLY Internal use only
JNF_STATICONLY Only static class members
JNF_INVOLVED Internal use only
JNF_IMPORT Names that can appear in import statements
JNF_PACKAGE Internal use only
JNF_INCLUDEPROTECTED Internal use only
JNF_CONSTRUCTORSONLY Constructors only
