Introduction (Application Wizard)

   

The Application Wizard provides a step-by-step way to create a Windows application. Various features can be selected during the course of the wizard, which can add additional capabilities to the application the wizard generates. The wizard generates all code and forms needed to produce a working "Notepad" style application. Once the wizard is complete, you can modify the application to fit the needs of your project. When the wizard has completed its process, a report can be displayed that provides helpful suggestions on features you can add or change in the wizard's output project.

Options

Profile Selection

A list that allows you to select a profile. A profile is a file that stores your settings from a previous wizard session. A button is provided next to the list to choose a profile file from your computer to use instead of those listed in the list.
