Connect Information (Data Form Wizard)

   

Collects the necessary information to connect to an ODBC data source. This step only appears if you chose Remote (ODBC) for the database type in the Database Type step of the wizard.

Options

DSN (Data Source Name)

Displays a list of DSNs currently installed on your computer to which you can connect. Use the 32-bit ODBC icon in the Control Panel to install new DSNs.

UID (User ID)

Allows you to type the User ID required to connect to this data source. If the data source does not require a User ID, leave this blank.

PWD (Password)

Allows you to type the password required to connect to the data source. If the data source does not require a password, leave this blank.

Database

Allows you to type the name of the database to which you want to connect.

Driver

Displays a list of drivers currently available on your computer that you can use to create a connection to a Remote data source. This option is not available if you select a DSN.

Server

Allows you to type the name of the server where the database resides. This option is not available if you select a DSN.
