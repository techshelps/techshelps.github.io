Application Features (Application Wizard)

   

Allows you to specify features to add to your application's form.

Options

Features List

A list of features that you can add to the application generated by the wizard. A check box next to each feature in the list indicates if the wizard will add it to your form. The following is a list of available features to add to your form:

Feature Description
Menu When selected, the wizard adds a pre-defined menu to the form it generates.
Edit When selected, the wizard adds an edit control to the form that fills the contents of the window. By selecting the edit box option the wizard uses the edit control to create a "Notepad" style application.
Toolbar When selected, the wizard adds a toolbar and pre-defined buttons to the form it generates.
Status Bar When selected, the wizard adds a status bar to the form it generates.
