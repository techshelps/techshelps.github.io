Application Type (Application Wizard)

   

Allows you to determine the type of application the wizard will generate.

Options

Form Based Application

When selected, the Application wizard generates a form-based application. The application developed by the wizard is a simple "Notepad" type application.

Form Based Application with Data

When selected, the Application Wizard starts the Data Form wizard to step you through the process of creating a single form application based on a database.

Note   This step of the Application wizard is only available in the Visual J++ Professional edition and the Enterprise edition of Visual Studio.
