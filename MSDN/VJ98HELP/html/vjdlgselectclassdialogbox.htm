Select Class Dialog Box

   

The Select Class dialog box allows you to select a class from the classes in your project.

With the Select Class dialog box you can:

  • Select a class from the tree that you wish to use.

  • View your projects in either Class View or Project View.

Options

Project View

This option allows you to view your project's classes in a way similar to the Project Explorer. Project View displays the projects and class files that make up your project. You can select a class file and any of the classes that are defined inside the class file. To select a class you can either double-click the name of the class or click the class name and then click the OK button.

Class View

This option allows you to view your project's classes in a way similar to the Class Outline. Class View displays a simple list of each available class in the currently selected project. To select a class you can either double-click the name of the class or click the class name and then click the OK button.

Note   When the Select Class dialog box is displayed from the J/Direct Call Builder, two additional objects are available for selection. These objects are the Windows Clipboard and a predefined class called Win32.
