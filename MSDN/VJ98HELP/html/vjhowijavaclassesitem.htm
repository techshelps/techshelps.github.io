IJavaClasses::Item

Returns an IJavaClass item from the collection. The VARIANT index provided specifies either the name of a class to search for or an integer index.

Syntax

HRESULT Item(
[in] VARIANT index,
[out, retval] IJavaClass** ppClass
);

Function Item(ByVal index) As IJavaClass

Parameters

index

Variant. Class item index or name of a class.
