Applets Tab (Customize Toolbox Dialog Box)

   

Displays a list of all of the applets that are found on the Classpath or are registered in the Java Package Manager (JPM) on your machine and are available to add to your Toolbox. Applets that are part of your compiled solution appear and can be added to the Toolbox. Applets that belong to projects in a solution are removed when you close the solution and are re-added when you open the solution. The applets that have a check mark to the left of their names are already added to the Toolbox.

Note   To use these options, you must have both Microsoft Visual J++ and Microsoft Visual InterDev installed.

You can:

  • Add an applet to the Toolbox, by selecting it in the Name and Package list and clicking the check box to the left of the control.

  • Remove an applet from the Toolbox by clearing the check box.

  • Group your list of applets by name and package by clicking the appropriate column heading.

Options

Name and Package List

Displays the available applets: their names and the package to which it belongs.

Description

Displays the current package in which the applet is located or a custom description that is defined by the applet.

Reset

Restores the Toolbox to its default state and refreshes the currently open tab. The default applets appear with a check in the check box to the left of them.
