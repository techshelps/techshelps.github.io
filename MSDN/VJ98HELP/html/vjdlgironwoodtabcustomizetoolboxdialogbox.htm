WFC Controls Tab (Customize Toolbox Dialog Box)

   

Allows you to add or remove WFC components from the toolbox.

With the WFC Tab you can:

  • View a list of the installed WFC components on your system.

  • Add or remove WFC components from the toolbox.

Options

Installed Components List

A check box list of the WFC components currently installed on your system. A check mark in the check box next to the component name indicates that the component is currently displayed in your toolbox. The list also displays the package where the component is located.

Note   To have a component displayed in the list of installed components, the path to the component must be defined in either the Classpath or the Java Package Manager.
