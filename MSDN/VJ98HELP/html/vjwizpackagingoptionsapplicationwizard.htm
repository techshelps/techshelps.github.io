Packaging Options (Application Wizard)

   

Defines the type of package to create for the application you are creating with the wizard.

Options

Class Files

When this option is selected, the Application wizard will not put the project into a package file of any type. Use this option if you want to use a method other than packaging for distributing your application. This option is selected by default.

EXE File

When this option is selected, an EXE file will be created when your project is compiled and built. Use this option if you want to create a file that a user can use to run your application.

Cabinet File

When this option is selected, a cabinet file (CAB) will be created when your project is compiled and built. A CAB file is a compressed file that contains all of the important information for your project. Use this option if you plan to distribute your project through the Internet.

Deploy to (URL)

When this option is selected, you can define a URL to deploy your application to. The text box located below this option allows you to specify a URL to deploy your application to.
