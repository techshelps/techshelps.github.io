Database Type (Data Form Wizard)

   

Allows you to select the type of database you want to use as a data source for this form.

Options

Database Formats List

Lists the database formats available to Visual J++. The available database formats are:

  • Access   Any version of the Microsoft Access .MDB database.

  • ODBC   Any ODBC compliant database driver. You should select this option to connect with any ISAM database (dBase, FoxPro, Paradox) or remote data source (SQL Server, Oracle). Although you can use the Microsoft Access ODBC driver, it is recommended that you select the Access option to gain better performance.
