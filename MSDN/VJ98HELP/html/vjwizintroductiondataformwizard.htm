Introduction (Data Form Wizard)

   

The Data Form Wizard is designed to automatically generate WFC forms that contain individual bound controls and methods used to manage information derived from local or remote data sources. You can use the Data Form Wizard to create single recordset forms to manage the data from a single table, or Master/Detail type forms to manage more complex one-to-many data relationships.

This step of the wizard allows you to choose a profile to retrieve settings from a previous Data Form Wizard session.

Options

Profile Selection

A list that allows you to select a profile. A profile is a file that stores your settings from a previous wizard session. The ellipsis button, located next to the list, allows you to choose a profile other than those provided in the list.
