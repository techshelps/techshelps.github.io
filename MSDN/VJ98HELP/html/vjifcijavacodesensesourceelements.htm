IJavaCodeSense::SourceElements

This property provides access to a collection of one or more named Java source code elements (classes, methods, or fields).

Syntax

HRESULT SourceElements(
[in] BSTR bstrName
[in] JavaNameSearchFlags flags
[out, retval]IJavaSourceElements **retval
);

Property SourceElements(ByVal bstrName As String, ByVal flags As Long) As IjavaSourceElements

Parameters

bstrName

Required string. Class, method, or field.

flags

Long. Reserved for internal use only.

Remarks

If more than one element matches the specified name (for example, overloaded methods and classes in different packages), all such source elements are included in the returned collection.
