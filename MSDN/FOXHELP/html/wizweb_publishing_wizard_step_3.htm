Web Publishing Wizard: Step 3 - Choose Style

See Also

You can select a Data Layout and a Visual Style for your web page.

You can modify the selected layout by using the Options button. You can override the default visual style settings and add your own HTML tags by using the Advanced button.

Data layouts

You can choose a data layout to use in your web page. You can view a description of each layout by right-clicking the layout and choosing View Description from the shortcut menu.

Visual styles

You can apply any visual style to any layout you select in the Data Layout list. You can select more than one visual style at a time; however, the attributes of one style might take precedence over the attributes of another. You can control which style has precedence by ordering the styles in the list. You can view a description of each style by right-clicking the style and choosing View Description from the shortcut menu.

Options

Opens the Layout Options dialog box so that you can modify specific properties of the data layout you chose.

Advanced

Opens the Advanced dialog box so that you can modify the appearance of your web page by adding a cascading style sheet (.css), colors, text, and HTML tags.

Preview

Use this button to view a sample of your web page in the selected data layout and visual style(s). The preview is limited to 10 records per page.
