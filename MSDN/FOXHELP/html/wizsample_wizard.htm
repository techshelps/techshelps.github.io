Sample Wizard

This sample wizard is a demonstration of a do-it-yourself wizard that is created by using the files in the Wzsample folder. The Sample Wizard creates an HTML document using data from a table or view.

To run the Sample Wizard

  • Type the following command in the Command window:
    DO (HOME() + 'Wizards\Wzhtml.app')

You can create your own wizard using your own data and files in the Sample Wizard folder as templates for the wizard you want to create.
