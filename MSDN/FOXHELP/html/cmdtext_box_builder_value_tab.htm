Value Tab, Text Box Builder

See Also

The Value tab in the Text Box Builder specifies a field in which to store the text box's value. This tab also has the Format tab and the Style tab.

When you choose OK, the builder closes, applying the property settings from every tab.

Tab Options

Field Name

Specifies a table or view field in which to store the text box's value. Choose the dialog button to display the Open dialog box so you can select a file. This option corresponds to the ControlSource property.
