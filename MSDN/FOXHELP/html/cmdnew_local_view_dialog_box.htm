New Local View Dialog Box

See Also

Lets you create a view of local tables or other views in the database using either the Local View Wizard or View Designer. Local views can include native Visual FoxPro .dbf format tables, existing local or remote views, any tables in a .dbf format, and other tables stored on local servers.

The New Local View dialog box only displays the Local View Wizard option when you have checked Prompt for Wizards in the Projects tab of the Options dialog box.

This dialog box appears when you select New Local View from the Database menu.

Dialog Box Options

View Wizard

Opens the Local View Wizard, which creates a local view. Available only if you check Prompt for Wizards in the Projects tab of the Options dialog box.

New View

Opens the View Designer so you can create a local view.

For more information, see "Combining Local and Remote Data in a View" in Chapter 6, Querying and Updating Multiple Tables in the User's Guide or "Creating a Local View" in Chapter 8, Creating Views in the Programmer's Guide.
