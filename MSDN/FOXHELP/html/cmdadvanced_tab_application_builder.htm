Advanced tab, Application Builder

See Also

Enables you to specify the Help file name and the default directory for your application. You can also specify the inclusion of a standard toolbar and a Favorites menu in your application.

Tab Options

Help file

Enables you to specify the name and path to the Help file for your application.

Default data directory

Enables you to specify the default directory for the data files of your application.

Menus

Standard toolbar

Specifies whether the Standard Toolbar appears in your application.

Favorites menu

Specifies whether the Favorites menu is available from your application.

Cleanup

Runs a routine to reconcile any changes you’ve made in the Application Builder with the active project.

If you choose OK, the builder closes, applying the property settings from all tabs.
