Print Options Dialog Box

See Also

Lets you print a text file, report, label, or the contents of the Command window or Clipboard.

This dialog box appears when you select Options from the Print dialog box.

Dialog Box Options

Print What

Type

Specifies the type of file to print. You may select Report, Label, Command Window, File, ASCII File, Clipboard, or any open text file (.qpr, .prg, or .txt).

Note   The ASCII File option bypasses the printer driver and sends the contents of the selected file directly to the printer. Make sure that the file contains valid printer commands. Sending invalid printer commands to your print device may cause your printer to fail and to require a reset.

File

Specifies the path and file name of the file you want to print. Type the path and name, or click the dialog button to display the Print File dialog box, in which you can search directories and select the file to print.

Print Options

Line Numbers

Prints line numbers in the left margin. This option is enabled for the Command window, and the Clipboard.

Page Eject Before

Advances the printer to the top of the next page before printing the file.

Page Eject After

Advances the printer to the top of the next page after printing the file. This option is not enabled for reports and labels.

Restore Environment

For running FoxPro version 2.x reports, ensures that the data environment settings previously saved with the report are used to run the report. This option does not apply to Visual FoxPro reports because they use their own data environments.

Print Setup

Displays the Windows Print Setup dialog box, in which you can specify a local or network printer, page orientation, paper size, and additional printing options.

Options

Displays the Report and Label Print Options dialog box, in which you can create expressions using the Expression Builder to select the records you want to print.

For more information, see Chapter 7, Designing Reports and Labels in the User's Guide.
