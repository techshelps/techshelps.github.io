Database Wizard: Step 1 – Select a Database

You can choose a Visual FoxPro database template or a remote Visual FoxPro (.dbf) or Access (.mdb) database template to use as a template for your new database.

Specify a local Visual FoxPro database template by choosing from the list, and then click Next.
