Layout Tab, Option Group Builder

See Also

The Layout tab in the Option Group Builder specifies button layout (vertical or horizontal), spacing, and border style. This builder also has the Buttons tab and Value tab.

When you choose OK, the builder closes, applying the property settings from every tab.

Tab Options

Button Layout

Specifies vertical or horizontal layout of option group buttons by adjusting the Width property.

Spacing Between Buttons

Specifies spacing between buttons by adjusting the Width property or the Height property.

Border Style

Specifies a single-line border around the option group, or no border. This option corresponds to the BorderStyle property.
