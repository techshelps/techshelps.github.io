Reports Tab, Application Builder

See Also

Enables you to specify friendly names for reports in your application.

Tab Options

Name

Enables you to specify the friendly name for the selected report.

Appear in Print Reports dialog

Specifies whether the friendly name of the selected report appears in the Print Reports dialog box of your application.

Add

Enables you to add an existing report to your application.

Edit

Enables you to edit the selected report in the Report Designer.

Remove

Enables you to removes the selected report from your application.

If you choose OK, the builder closes, applying the property settings from all tabs.
