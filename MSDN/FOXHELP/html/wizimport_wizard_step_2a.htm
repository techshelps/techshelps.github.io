Import Wizard: Step 2a – Describe Data

See Also

Allows you to manually add column breaks or delete column breaks if you have chosen Fixed Width in the Data Format area of Step 2. If you have chosen Delimited from the Data Format area of Step 2, select what delimiter separates the text strings in the selected file. View the effect of the delimiter selected in the Field grid.

Note   Step 2a of the Import Wizard only appears if you have chosen to import a text (.txt) file.
