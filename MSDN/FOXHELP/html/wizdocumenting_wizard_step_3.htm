Documenting Wizard Name: Step 3 – Set Indentation

See Also

Allows you to specify how the Documenting Wizard will indent lines in your program. The wizard can indent specific types of lines, and you can adjust how many spaces to use when indenting.

Options

Type of Indent

Choose whether the wizard should indent using tabs or spaces. To turn off indentation, choose No Change.

Spaces

If you are indenting using spaces, set the number of spaces used for each level of indentation.

Text to indent

Specify the type of line that the wizard should indent. Choices are Comments, Controls structures (such as loops, if blocks, and DO CASE structures), and statements that are split across multiple lines.
