Layout Options Dialog Box

See Also

The Layout Options Dialog box enables you to modify the default values of the select data layout. You can save any changes you make as a new data layout and style specification by using the Save Settings button in the Advanced dialog box.

Border tab

This tab provides a way for you to select the background and border colors and border width. You can also specify a .gif or .jpg file as a background image.

Font tab

This tab enables you to specify one or more HTML-defined fonts in case a user doesn't have a particular font. This can help provide good cross-platform browser behavior.

You can also specify the font name and change the default font color.

Other tab

This tab enables you to specify the name of an HTML-defined class to use in your web page.

The Other tab can also provide access to properties settings that are specific to a particular layout choice.

Reset Defaults

Use this button to restore your selections to the original wizard default values.
