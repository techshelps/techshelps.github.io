New File Dialog Box

See Also

Lets you create a new form, report, label, or query, depending on the type of file currently selected in the Project Manager.

If you check Prompt for Wizard in the Projects tab of the Options dialog box, Visual FoxPro prompts you with the option of running a wizard if there is a wizard associated with the selected file type.

Note   File names preceded by a slashed circle are excluded when the project is built. If there is no slashed circle by the file name, that file will be included in the build. The main program file name is in bold.

This dialog box appears when you select New File from the Project menu.

Dialog Box Options

File Type Wizard

Opens a wizard that creates the selected file type. If there is a group of wizards to choose from, Visual FoxPro will prompt you to select the wizard you want. Displayed only if you check Prompt for Wizard in the Projects tab of the Options dialog box. For more information, see "Using Wizards" in Chapter 1, Getting Started in the User's Guide.

New File Type

Displays a dialog box, designer, or editing window so you can create the selected file type.
