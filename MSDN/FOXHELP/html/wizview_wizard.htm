Local View Wizard

See Also

Creates views using native data. If you do not have a database open, you are prompted to open a database or create one.

To run the Local View Wizard

  1. From the Wizards submenu on the Tools menu, choose Query.

  2. In the Wizard Selection dialog box, choose Local View Wizard.

Wizard Steps

Step 1 – Select Fields
Step 2 – Relate Tables
Step 2a – Include Records
Step 3 – Filter Records
Step 4 – Sort Records
Step 4a – Limit Records
Step 5 - Finish

For more information, see "Using Wizards" in Chapter 1, Getting Started, "Relating Tables" in Chapter 3, Collecting Tables into a Database, and "Selecting the Records You Want" in Chapter 4, Retrieving Data, all in the User's Guide.
