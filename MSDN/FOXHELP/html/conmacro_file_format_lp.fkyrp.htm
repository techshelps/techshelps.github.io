Macro File Format (.fky)

See Also

File Header

Byte offset Description
01 – 03 Signature, Hex 79FF
04 – 15 Ignored
16 – 17 Number of macros (binary)
18 – end The macros

Individual Macros

Byte offset Description
00 – 19 Macro name
20 – 21 Macro length (in keystrokes, binary)
22 – 23 Keystroke (two bytes, binary)
24 – end Macro keystrokes
