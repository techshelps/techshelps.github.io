Credits Tab, Application Builder

See Also

Enables you to specify application production credits, including author, company, version, copyright, and trademark.

Tab Options

Author

Specifies the name of the application author.

Company

Specifies the name of the company writing or using the application.

Version

Specifies the build edition of the application.

Copyright

Specifies any copyright information.

Trademark

Specifies any trade or service mark.

Because these entries are retained as text, you can enter any information you wish and not only information prompted by the tab option labels.

If you choose OK, the builder closes, applying the property settings from every tab.
