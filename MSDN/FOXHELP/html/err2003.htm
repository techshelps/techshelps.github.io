OLE Changed Properties stream is the wrong version. Subclassed properties ignored. (Error 2003)

The names and values of the OLE object properties could not be read from the form (.scx) or class (.vcx) file.

  • The information is missing from the file or the file is corrupt.

    Reapply your recent changes and save the file in the Form Designer or Class Designer. The correct information is written out.

  • The version number for the OLE object has changed.

    Contact the vendor of the OLE object.
