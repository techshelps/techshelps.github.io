No key columns specified for the update table "name".  Use the KeyFieldList property of the cursor (Error 1492)

No key column(s) have been specified in the KeyFieldList property for a base table that is being updated.

Modify the KeyFieldList property to include key columns for each table that is being updated. For most views, you can use the View Designer to set the view update properties and then save the view back into the database. For more information, see CURSORGETPROP( ).
