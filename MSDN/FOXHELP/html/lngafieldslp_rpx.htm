AFIELDS( ) Function Example

The following example creates an array named gaMyArray containing information about the fields in the customer table. The names of the fields are displayed.

CLOSE DATABASES
OPEN DATABASE (HOME(2) + 'Data\testdata')
USE Customer     && Open customer table

gnFieldcount = AFIELDS(gaMyArray)  && Create array
CLEAR
FOR nCount = 1 TO gnFieldcount 
   ? gaMyArray(nCount,1)  && Display field names
ENDFOR
