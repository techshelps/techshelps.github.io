Language Differences Between dBASE IV and Visual FoxPro

See Also

The Visual FoxPro language supports practically all the commands and functions available in dBASE IV. However, there are several dBASE IV commands and functions that are not supported in Visual FoxPro, and others that behave in a different manner in Visual FoxPro.

The following topics describe the dBASE IV commands and functions that are not supported in Visual FoxPro and those that behave differently in Visual FoxPro.
