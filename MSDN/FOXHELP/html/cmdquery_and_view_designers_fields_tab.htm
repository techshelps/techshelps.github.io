Fields Tab, Query and View Designers

See Also

Specifies fields, aggregate functions, and other expressions for query or view output. This tab appears in the Query Designer and the View Designer.

Tab Options

Available Fields

Lists all the available fields in the query's or view's tables.

Functions and Expression

Specifies a function or expression. You can either select a function from the list or type an expression directly into the box. Click the Add button to add it to the Selected Output box.

Selected Fields

Lists the selected fields, aggregate functions, and other expressions that appear in the query or view results. You can rearrange the order of the output by dragging the vertical double arrow to the left of the fields.

Add

Adds the selected item to the Selected Output box from the Available Fields box or the Functions/Expressions box.

Add All

Adds all fields to the Selected Output box from the Available Fields box.

Remove

Removes the selected item from the Selected Output box.

Remove All

Removes all items from the Selected Output box.

Properties

Displays the View Field Properties dialog box, which allows you to specify options for fields in a view just as you do for fields in database tables. This option is only available in the View Designer.
