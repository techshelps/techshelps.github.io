WWW Search Page Wizard: Step 6 – Set Up Result Page

See Also

Because a Web results page is not really very different from a search page, you can place both a header and background image on this page as well. The placement of the images is the same as that for the search page images: the header image will appear between the title and description, and the background image will be tiled throughout the entire page.

On this page, you can also specify the maximum number of records returned on the result page. The default value is 10, which is a typical number of return entries for a Web search page.

Finally, you can specify the data source name of the ODBC data source that the Web server will use to manage the search and search results.
