Remote View Wizard: Step 2 – Select Fields

See Also

In the Tables box, the wizard lists the tables available from the data source you specified in the previous step. In the list of tables, you select a table, and then the wizard lists fields that are available from that table. You can select as many fields as you need and move them into the Selected fields box. If you want fields from another table, you can continue to select tables and select more fields.
