Query Designer Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Query Designer. These commands appear when you right-click the Query Designer.

Shortcut Menu Options

Run Query

Executes the SQL select statement you built, and sends the results to the output destination you specified.

View SQL

Displays the SQL statement your query or view is building in a read-only text file.

Remove Table

Removes the selected table from the top pane of the designer window. This command is enabled when you have at least one table in the Query or View Designer selected.

Add Table

Displays the Add Table Or View dialog box so you can add a table or view to the designer window.

Output Settings

Displays the Query Destination dialog box, which lets you send the results of your query to seven different outputs.

Help

Displays Help on the selected item.
