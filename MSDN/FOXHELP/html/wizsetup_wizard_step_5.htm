Setup Wizard: Step 5 – Specify Default Destination

See Also

The setup routine will place your application in the directory you specify in the Default directory box. Avoid selecting directory names that are already used by Windows programs (such as Visual FoxPro, Windows itself, and so on).

If you specify a name in the Program group box, the setup program will create a program group for your application when the user installs it, and will make the application available on the user's Start menu.

By selecting options under User can modify, you can allow the user to change the names of the default directory and program group that the installation process creates.

For more information, see "Using Wizards" in Chapter 1, Getting Started, in the User's Guide.
