Generate Menu Dialog Box

See Also

Lets you specify a name for the generated menu program and then generate the program. If you have not saved your menu, you are prompted to do so.

This dialog box appears when you select Generate from the Menu menu.

Dialog Box Options

Output File

Specifies the path to the generated menu program. Choose the dialog button to select a directory and file name, or substitute a file name of your choice by typing over the name displayed.

Generate

Produces a menu program file, which has the .mpr extension.
