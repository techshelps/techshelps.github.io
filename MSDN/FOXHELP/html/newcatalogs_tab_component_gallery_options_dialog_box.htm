Catalogs Tab, Component Gallery Options Dialog Box

Displays the names of the views that are currently available in the Component Gallery. This is the list that displays in the View Type drop-down list.

Tab Options

Global

Sets a catalog to contain any type of folder. You can access global catalogs and their contents no matter what other catalogs are open.

Default

Enables a catalog to automatically open when the Component Gallery starts. All catalogs that are marked as default are available whenever you select the default view.

New

Displays the Open Catalog Dialog Box dialog box so you can add a catalog to the list.
