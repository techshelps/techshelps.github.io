Main Form Shortcut Menu

Allows you to control the appearance and behavior of the Component Gallery window. The menu provides the following functions when you right-click the Component Gallery toolbar:

Descriptions

When checked, displays the item description box at the bottom of the window.

Always on Top

When checked, the Component Gallery window remains on top, even when other windows are active. When unchecked, the window does not remain on top.

Auto Expand

When checked, the Component Gallery automatically expands the folder tree to display the second level of nodes. When unchecked, only the catalog folder items are visible.

Parent Class Toolbar

Places a button on the Visual FoxPro toolbar that allows you to view and edit method code in the parent class.

Restore Defaults

Restores the default Descriptions, Always on Top, and AutoExpand settings.

Font

Sets the font attributes of all controls in the Component Gallery.

Export

Saves the current view to a table or file.

Add-ins

Displays add-in programs, applications, and classes.

New Window

Launches a new instance of the Component Gallery with the same view and selected item.

New Class Browser

Launches a new instance of the Class Browser with the same view and selected class or member.

Refresh

Refreshes the Component Gallery by regenerating the current view. You can also use the F5 function key.

Help

Opens the Help file.
