Data tab, Application Builder

See Also

Enables you to specify the data source and the format for forms and reports you include in your application. The grid displays the tables used in your application.

Tab Options

Database Wizard button

Enables you to create a new database for your application using the Database Wizard. When the wizard closes, the tables in the new database are listed in the grid.

Table Wizard button

Enables you to create a new table for your application using the Table Wizard.

Select

Enables you to select an existing database or table to use in your application.

Clear

Enables you to remove a table from the grid.

Generate

Enables you to generate a report or form from the selected table, in the specified Form style or Report style.

Form Style

Enables you to select a form style to use for one or more tables listed in the grid.

Report Style

Enables you to select a report style for one or more tables listed in the grid.

To use different styles for each form or report, select the forms or reports and the styles, then generate the reports. Repeat this procedure for as many different styles as you want in your application.

If you choose OK, the builder closes, applying the property settings from all tabs.
