General Field Window

See Also

This window is displayed when you double-click a General field's type designator in a browsed table, or press CTRL+PAGE UP or CTRL+PAGE DOWN to open the General Field window from the keyboard.

With this window displayed, you can create or insert OLE objects by using the Insert Object command on the Edit menu. You can edit the existing object by double-clicking it or by selecting Bitmap Image object Edit or object Open from the Edit menu.
