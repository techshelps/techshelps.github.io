SYS(23) - FoxPro EMS Memory Usage

See Also

Returns the amount of EMS memory (in 16K segments) that is currently being used by the Standard (16-bit) version of FoxPro for MS-DOS.

Syntax

SYS(23)

Returns

Character

Remarks

Zero is returned if no EMS memory is in use.

SYS(23) always returns 0 in the Extended (32-bit) version of FoxPro for MS-DOS and Visual FoxPro.
