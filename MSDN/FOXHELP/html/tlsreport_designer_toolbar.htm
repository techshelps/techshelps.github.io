Report Designer Toolbar

See Also

The Report Designer toolbar is displayed when the Report Designer is open. For information about displaying and manipulating toolbars, see "Using Toolbars" in Chapter 1, Getting Started, in the User's Guide.

This toolbar contains the following buttons.

Button Description
Data Grouping Displays the Data Grouping dialog box so you can create data groups and specify their properties.
Report Controls Toolbar Shows or hides the Report Controls toolbar.
Color Palette Toolbar Shows or hides the Color Palette toolbar.
Layout Toolbar Shows or hides the Layout toolbar.

For more information, see "Using Toolbars" in Chapter 1, Getting Started, "Changing Control Colors" and "Creating a Report Layout" in Chapter 7, Designing Reports and Labels, "Aligning Controls" in Chapter 8, Managing Data Through Forms, and "Grouping Data on Your Layout" in Chapter 9, Importing and Exporting Data, all in the User's Guide; and "Formatting Field Controls" in Chapter 12, Adding Queries and Reports, in the Programmer's Guide.
