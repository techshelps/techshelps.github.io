Component Gallery Window

See Also

The Component Gallery window contains a toolbar plus the following options:

Window Options

Catalog (Left) Pane

Displays the Component Gallery hierarchy of catalogs and folders. Select a catalog or folder here to display the items of the container in the Object panel to the right.

Object (Right) Pane

Displays items contained in a catalog or folder.

For information on using items in the Component Gallery, see the Component Gallery overview topic.

Description Box

Provides information on the selected item. The Visual FoxPro status bar reports Component Gallery process status.
