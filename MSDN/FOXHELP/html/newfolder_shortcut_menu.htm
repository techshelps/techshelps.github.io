Folder Shortcut Menu

Provides folder-specific and general functionality to Component Gallery folder items. The menu provides the following functions when you right-click a Component Gallery folder:

Find

Launches the Find dialog box defaulting to search for items in the selected folder.

Cut

Moves the selected folder to the Gallery clipboard. The folder is dimmed (similar to Windows Explorer behavior). When a Paste operation occurs, the folder is removed from that location (its record marked as Deleted). This menu item is disabled for folder items contained in dynamic folders. Cut is always disabled for Catalog folders.

Copy

Places a copy of the selected folder to the Gallery clipboard. The Folder is not dimmed. Unlike a Cut, when a Paste operation occurs, the Component Gallery creates a new copy of the gallery item.

Paste

Moves the folder from the Gallery clipboard to the selected folder. This menu item is disabled in a dynamic folder. If a folder already exists with the same name, the new folder is named "Copy of " + folder name.

Paste Link

Pastes a link of the folder in the Gallery clipboard and selects the new folder. The new link is named "Link to" + name. This menu item is disabled in a dynamic folder.

Create Link

Makes a link of the folder and selects the new linked folder. This menu item is disabled in a dynamic folder.

Remove

Deletes the selected folder from the catalog. A message box is first displayed confirming the operation. This menu item is disabled in a dynamic folder.

Rename

Renames the selected folder. This menu item is disabled in a dynamic folder.

Backup

Saves the current catalog contents to the Component Gallery backup folder.

Restore

Restores the last saved version of the current catalog.

Properties

Displays the Folder Properties dialog box to view or edit specific properties of the selected folder. This menu item is disabled in a dynamic folder.

Note   When a folder item represents a URL, the browser page shortcut menu displays.
