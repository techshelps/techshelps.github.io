Folder Properties Dialog Box

See Also

Allows you to customize the behavior of Component Gallery folders.

This dialog appears when you click Properties on the shortcut menu of a selected folder.

General Tab

Enables specifying settings that are global for all catalogs. The following settings are available in this tab:

Name

Displays the value in the Text field as the component label in the Component Gallery.

Description

Optional information from the Desc field that displays in the description pane of the Component Gallery.

Picture

Value in the Folderpict field which specifies the path and file name of the graphic used to represent the selected item in the gallery.

Item picture

Value in the Picture field which specifies the path and file name of the graphic used to represent items in this catalog that have no specified picture.

Item desc

Value in the Typedesc field which contains a description that applies to all items contained in this catalog or folder.

Node Tab

A node is any selectable gallery item. Enables specifying settings that are global for all catalogs. The following settings are available in this tab:

ID

The name of the table plus the value in the ID field for the selected folder.

Link ID

If the selected item is a link, specifies the name of the source item.

Class name

The value in the Classname field which specifies the name of the class of the selected folder.

Class library

The value in the Classlib field which specifies the name and path to the class library containing the gallery item.

Item class

The value in the Itemclass field which specifies the default class of items in the selected folder that have no previously assigned class.

Dynamic folder

Value in the Filename field which contains the target expression, filename, or URL that is launched when you select this dynamic folder item.

Scripts Tab

This tab provides a place to specify events or properties that are global for all folders. The following areas are available in this tab:

Script

The value in the Script field which contains any event script related to this catalog and its items.

Properties

The value in the Properties field which can contain value assignments to properties of the item.

Comments Tab

Enables specifying settings that are global for all catalogs. The following settings are available in this tab:

Comments

The value in the Comment field which contains user comments about usage, characteristics, or anything.

User

The value in the User field which contains user comments about usage, characteristics, or anything.
