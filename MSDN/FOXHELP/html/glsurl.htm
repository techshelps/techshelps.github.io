URL

Uniform Resource Locator. Identifies the full path of a document, graphic, or other file on the Internet or on an intranet. A URL expresses the protocol (such as FTP or HTTP) to be accessed and the file's location. A URL may also specify an Internet e-mail address or a newsgroup. Some examples of URLs are:

http://www.someones.homepage/default.html

ftp://ftp.server.somewhere/ftp.file

gopher://server.name

file://Server/Share/File.doc

See also FTP, HTTP, World Wide Web (WWW).
