If you have FoxPro 2.6 for the Macintosh or Visual FoxPro 3.0 for the Macintosh files that you want to modify in Visual FoxPro version 5.0 or later, you must copy them to the Windows platform, and then convert or recompile them.

Converting FoxPro for the Macintosh Files

To use Visual FoxPro for the Macintosh files in Visual FoxPro version 5.0 or later, first copy the files to a Windows disk, and then convert or recompile your files.

To convert files to Visual FoxPro version 5.0 or later

  1. Copy the Visual FoxPro for the Macintosh file to a Windows disk.

  2. From the File menu, choose Open, select a Visual FoxPro 3.0 file to convert, and choose OK.

     – Or – 

  • Use a command such as MODIFY PROJECT or MODIFY FORM.

    The Conversion Options dialog box appears.

Using FoxPro 2.6 for the Macintosh Files

Converting from FoxPro 2.6 for the Macintosh to Visual FoxPro version 5.0 or later for Windows requires a full conversion of project and other files after you copy the files to the Windows platform. For details of conversion from FoxPro 2.6 files see, Converting FoxPro 2.6 Files.
