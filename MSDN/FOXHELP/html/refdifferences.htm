Differences Between Visual FoxPro and Previous Versions

See Also

Visual FoxPro menus and features have been modified in version 5.0. In addition, Visual FoxPro, versions 5.0 and later differ from Visual FoxPro 3.0 and FoxPro 2.6 in some terminology, language, database, and interface features. If you’re looking for a FoxPro 2.6 or Visual FoxPro 3.0 feature in Visual FoxPro 5.0 or later, check the following locations.
