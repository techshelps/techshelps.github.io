Documenting Wizard

See Also

The Documenting Wizard formats and produces text files from your code in projects and program files. For example, the wizard can indent your source code, capitalize reserved words and variables, and produce a report of the keywords in the program.

The wizard does not modify your code and only creates a text file to document the code. If you want the wizard to perform specific tasks, see Customizing the Documenting Wizard.

For general information about wizards, see "Using Wizards" in Chapter 1, Getting Started, in the User's Guide.

To run the Documenting Wizard

  • From the Wizards submenu on the Tools menu, choose Documenting.

Wizard Steps

Step 2 – Set Capitalization
Step 3 – Set Indentation
Step 4 – Add Headings
