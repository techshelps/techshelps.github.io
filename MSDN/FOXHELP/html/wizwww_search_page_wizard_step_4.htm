If you want to add graphics to your Web page, you can specify the images (.gif or .jpg) you want as the header and background images. These images are optional, but add to the visual interest of a page.

The header image appears between the title and description. The background image is tiled as the background for the page.

Tip   Use small and simple images for the background. Large or busy images do not display well when tiled for background.

Using the image files you specify, the wizard creates a subdirectory (IMG) to store the images referenced in your page.

To allow your Web page visitor to download the data as text, you can select Provide the ability to download the result set as a file. If you select this option, the wizard adds a check box to the Web page for the visitor to select and download the data found by the query.
