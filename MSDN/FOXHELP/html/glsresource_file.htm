resource file

1) A Visual FoxPro database table (Foxuser.dbf) used to store information about the current state of the program, such as the location and size of the command window, current keyboard macros, which toolbars are displayed, and so on. 2) A file, often a DLL, that contains information used in a program that might need to be changed, such as text strings, bitmaps, fonts, and so on. For example, Visual FoxPro stores the literal text of its error messages, window captions, and dialog box controls in a run-time resource file so that you can swap resource files if you want to use the program in a different language.
