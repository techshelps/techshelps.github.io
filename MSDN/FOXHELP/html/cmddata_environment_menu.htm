Data Environment Menu

See Also

Contains commands that allow you to view and modify a data environment. This menu appears when you open a form, report, or label, or create new ones, and choose Data Environment from the View menu. The Data Environment window is also displayed.

Data Environment Menu Commands

Add

Displays the Add Table or View dialog box so you can add a table or view to the Data Environment Designer window.

Remove

Removes the selected table or view from the Data Environment window.

Browse

Displays in a Browse window the table or view currently selected in the Data Environment Designer. In Browse mode you can examine or edit table or view contents.

For more information, see Chapter 7, Designing Reports and Labels or Chapter 8, Managing Data Through Forms in the User's Guide, and Chapter 9, Creating Forms and Chapter 12, Adding Queries and Reports in the Programmer's Guide.
