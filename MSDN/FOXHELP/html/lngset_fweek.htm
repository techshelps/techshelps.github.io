SET FWEEK Command

Example   See Also

Specifies the requirements for the first week of the year.

Syntax

SET FWEEK TO [nExpression]

Arguments

nExpression

Specifies a value which determines the requirements for the first week of the year. The following table lists the values nExpression can assume and the corresponding requirements for the first week of the year:

nExpression First week requirement
1 (Default) The first week contains January 1st.
2 The larger half (four days) of the first week is in the current year.
3 The first week has seven days.

If you omit nExpression, the first week of the year is reset to 1 (the first week contains January 1st).

Remarks

The first week of the year can also be set with the First Week of Year list box in the Regional tab of the Options dialog.
