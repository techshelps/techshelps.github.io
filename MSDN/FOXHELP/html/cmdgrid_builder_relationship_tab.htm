Relationship Tab, Grid Builder

See Also

The Relationship tab in the Grid Builder specifies a relationship between grid fields and table fields. This builder also has the Grid Items tab, Style tab, and Layout tab.

When you choose OK, the builder closes, applying the property settings from every tab.

Tab Options

Key Field in Parent Table

Specifies the key field of the controlling table in a relationship. This option corresponds to the LinkMaster property. To find a table, click the dialog button, which displays the Open dialog box.

Related Index in Child Table

Specifies the index tag name of the controlled table in a relationship. This option corresponds to the ChildOrder property.
