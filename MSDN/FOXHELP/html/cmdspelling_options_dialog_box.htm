Spelling Options Dialog Box

See Also

Controls the rules Visual FoxPro uses to check spelling. This dialog box appears when you choose Options in the Spelling dialog box.

Dialog Box Options

Always Suggest

Displays a list of suggested corrections for each unmatched word when you check spelling.

Ignore Words In UPPERCASE

Skips words in which each character is an uppercase letter.

Ignore Extra Spaces

Skips extra spaces between words.

Dictionary

Language

Specifies the language of the active dictionary.

Type

Specifies the Normal, Concise, Complete, Medical, or Legal dictionary.
