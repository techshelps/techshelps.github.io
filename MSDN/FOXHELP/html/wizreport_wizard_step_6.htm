Report Wizard: Step 6 – Finish

See Also

If the number of fields that you have selected cannot fit on a single line within the width of your report, the fields will wrap to the following line. If you do not want the fields to wrap, clear Wrap fields that do not fit. Click the Preview button to view the report without leaving the wizard.

If the table you selected is from a database, this step enables you to use display settings stored in the database.

After you save the report, you can open and modify it like any other report in the Report Designer.
