OLE Drag-and-Drop Shortcut Menu

See Also

Contains shortcuts to several commands useful when doing OLE drag-and-drop operations. This menu appears when you drop the item on the target form, control, or Data Environment.

Shortcut Menu Options

Copy Here

Copies text from the drag source to the drop target.

Create Grid Here

Creates a Grid control from a Visual FoxPro table dropped on a form.

Create Image Here

Creates an Image control from a graphic file dropped on a form.

Create Label Here

Creates a Label control from text dropped on a form.

Link Here

Adds a Visual FoxPro table dropped on the Data Environment to the Data Environment.

Move Here

Moves text to the drop target, removing it from the drag source.

Set Control.Caption

Sets the Caption property for a non-container control to the text dropped on the control.

Set Form.Icon

Sets the Icon property for a form to the graphic file dropped on the form.

Set Form.MouseIcon

Sets the MouseIcon property for a form to the graphic file dropped on the form.

Set Form.Picture

Sets the Picture property for a form to the graphic file dropped on the form.

Cancel

Cancels the drag and drop operation.
