Command Window Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Command window. These commands appear when you right-click on the Command window.

Shortcut Menu Options

Cut

Removes selected text to the Clipboard. Once you cut something, you can paste it elsewhere in the current application or in other applications. This command corresponds to the Cut command in the Edit menu.

Copy

Duplicates selected text and puts it on the Clipboard. Once you copy something to the Clipboard, you can paste it elsewhere in the current application or in other applications. This command corresponds to the Copy command in the Edit menu.

Paste

Places from the Clipboard cut or copied sections of text at the insertion point. This command corresponds to the Paste command in the Edit menu.

Build Expression

Displays the Expression Builder dialog box. This command can only be accessed in the Command window through the shortcut menu.

Execute Selection

Executes the currently highlighted command or set of commands in the Command window. This command can only be accessed in the Command window through the shortcut menu.

Clear

Clears the Command window of all text. This command does not correspond to the Clear command in the window menu. If you select Clear, you cannot select Undo from the Edit menu to unclear the Command window.

Properties

Displays the Edit Properties dialog box, which allows you to set editing options for the current editing window or set color and font options for all editing sessions. This command corresponds to the Properties command in the Edit menu.
