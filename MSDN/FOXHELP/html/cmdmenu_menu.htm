Menu Menu

The Menu menu contains commands you use to create or modify menu systems.

Menu Commands

Quick Menu

Loads the main Visual FoxPro menu system into the Menu Designer so you can use it as the basis for creating a menu system.

Insert Item

Inserts a new row in the Menu Designer window. Shortcut: CTRL+I

Insert Bar

Displays the Insert System Menu Bar dialog box, which allows you to add menu items to the Menu Designer window.

Delete Item

Deletes the selected row from the Menu Designer window. Shortcut: CTRL+E

Generate

Displays the Generate Menu dialog box so you can specify a name for the generated menu program and then generate the program.

Preview

Displays a menu system as you design it without compiling menu program code.
