Documenting Wizard: Step 4 – Add Headings

See Also

Allows you to specify whether the Documenting Wizards inserts headings into your programs. Headings are comments about the code that are relevant to the code that they precede. They can be placed at the beginning of files, procedures, class definitions, and methods to make your formatted code easier to read.

Specify the type of code block for which you want to insert headings.
