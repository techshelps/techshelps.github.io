Help Menu

The Help menu contains options for accessing online Help and information on how to get technical assistance.

Help Menu Commands

Microsoft Visual FoxPro Help Topics

Opens up the Help file specified with the SET HELP TO command. The help file can be an HTML Help file (.chm) or a Winhelp file (.hlp). The default setting for SET HELP TO is MSDN.

  • Regardless of what file Help is set to, the Contents, Index, and Search menu items will open the MSDN Library (if installed).

  • If no Help file is specified with SET HELP TO, the MSDN Library is opened (if MSDN is installed).

  • If there is no Help file set with SET HELP TO and the MSDN Library is not installed, this menu item is dimmed and you can't start Help from this menu or by pressing F1.

Contents

Opens the MSDN Library (if installed) with the Contents tab selected.

Index

Opens the MSDN Library (if installed) with the Index tab selected.

Search

Opens the MSDN Library (if installed) with the Search tab selected.

Microsoft on the Web

Displays the Microsoft on the Web Submenu, which provides access to helpful Microsoft Web pages.

Technical Support

Displays information about Microsoft Developer Technical Support.

About Microsoft Visual FoxPro

Displays the About dialog box, which contains information about Visual FoxPro and your system.
