Window Menu, Debugger Window

See Also

Provides access to the active Debugger windows.

Menu Commands

Clear Output Window

Deletes all text displayed in the Debug Output window.

Cascade

Allows you to view all the open Debugger windows in a stair-step fashion, one stacked on top of the other.

Restore to Default

Displays the windows in their original positions in the Debugger window and clears all the watch expressions. Also, all the data that is saved in the debugger configuration file is cleared.

Trace

Activates the Trace window.

Watch

Activates the Watch window.

Locals

Activates the Locals window.

Call Stack

Activates the Call Stack window.

Output

Activates the Debug Output window.

Note   If the windows are docked, there is no visual indication that they have been activated, although associated menu items are enabled or disabled.
