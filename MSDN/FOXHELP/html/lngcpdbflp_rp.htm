CPDBF( ) Function

See Also

Returns the code page with which an open table has been marked.

Syntax

CPDBF([nWorkArea | cTableAlias])

Arguments

nWorkArea

Specifies the work area number.

cTableAlias

Specifies the table alias.

Include nWorkArea or cTableAlias to specify a table open in a work area other than the current work area. CPDBF( ) returns 0 if a table isn't open in the work area you specify. If a table doesn't have the alias you specify with cTableAlias, Visual FoxPro generates an error message.

Remarks

DISPLAY STRUCTURE also displays the code page with which an open table has been marked.

For additional information on code pages and Visual FoxPro's international support, see Code Pages Supported by Visual FoxPro, and Chapter 18, Developing International Applications, in the Programmer's Guide.
