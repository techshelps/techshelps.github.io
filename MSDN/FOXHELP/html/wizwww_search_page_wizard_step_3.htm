WWW Search Page Wizard: Step 3 – Set Search Page Options

See Also

The wizard adds the title and description you provide to the search page. The title you provide appears at the top of the page in a Web browser. The title should describe what your page is for, but should be succinct.

The description is the short paragraph that follows the title and centered below the title. You can use the description to state the purpose of the site and how to do a search.

You can also provide the user with tips on how to enter information in the search page, what to expect after the query is run, or about the information in your table. For example, if your data is in all capitals, you can tell them to press CAPS LOCK before entering their search criteria.
