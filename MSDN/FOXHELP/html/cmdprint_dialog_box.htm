Print Dialog Box

See Also

Lets you print a text file, report, label, or the contents of the Command window or Clipboard.

This dialog box appears when you select Print from the File menu.

Shortcut

CTRL+P

Dialog Box Options

Printer

Name

Displays the names of the printers currently installed on your machine.

Status

Displays whether or not the chosen printer is the default printer, and if it is ready to print.

Type

Displays the model of the selected printer. You specify the printer model when you first install the printer.

Where

Displays the network printer path.

Comment

Displays comments about the printer.

Properties

Displays the system Printer Properties dialog box, which allows you to set options for your printer.

Print Range

All

Specifies that all the pages in the document print.

Pages

Allows you to specify which pages of the document print.

Selection

Prints the highlighted selection in the document.

Copies

Number of Copies

Allows you to specify the number of copies of the document to be printed.

Collate

Specifies that if multiple copies of a document are printed, the printed copies will be collated.

For more information, see Chapter 7, Designing Reports and Labels in the User's Guide.
