Data Environment Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Data Environment Designer. These commands appear when you right-click on the Data Environment Designer.

Shortcut Menu Options

Add

Displays the Add Table or View dialog box, which allows you to select one or more existing tables or views to add to the Data Environment.

Remove

Removes the selected table or view from the Data Environment. This option only appears when you select a table.

Browse

Displays the selected table or view in a Browse window.

Form Designer

Brings the Form Designer to the front. This command appears only when the Data Environment Designer is opened within the Form Designer.

Report Designer

Brings the Report Designer to the front. This command appears only when the Data Environment Designer is opened within the Report Designer.

Properties

Displays the Properties window, which allows you to view and edit the properties, events, and methods for the data environment.

Code

Displays the Code window, which allows you to write, display, and edit form, event, and method code.

Help

Displays Help on the selected item.
