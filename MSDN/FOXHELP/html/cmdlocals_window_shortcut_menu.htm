Locals Window Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Locals window of the Debugger. These commands appear when you right-click the Locals window.

Shortcut Menu Options

Public

Displays public variables in the Locals window.

Local

Displays local variables in the Locals window.

Standard

Displays standard variables in the Locals window.

Objects

Displays objects in the Locals window.

Docking View

Allows you to specify whether the docking feature for the window is on or off. If unchecked, you cannot dock the window.

Hide

Hides the Locals window.

Font

Displays the Font dialog box where you can set the font type, style, and size. This command corresponds to the Font command in the Format menu.

Help

Displays Help on the selected item.
