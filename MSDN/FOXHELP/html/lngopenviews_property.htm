OpenViews Property

Applies To   See Also

Determines the type of views associated with a form set, form, or report's data environment that are opened automatically. Available at design time; read-only at run time.

Syntax

DataEnvironment.OpenViews[ = nExpression]

Settings

nExpression

The settings for the OpenViews property are:

Setting Description
0 (Default) Local and Remote. Both local and remote views for the data environment for the form set, form, or report are opened automatically.
1 Local only. Only local views for the data environment for the form set, form, or report are opened automatically.
2 Remote only. Only remote views for the data environment of the form set, form, or report are opened automatically.
3 None. No views are opened for the data environment of the form set, form, or report.

Remarks

The OpenViews property applies when the AutoOpenTables property is set to true (.T.) or the OpenTables method is executed.
