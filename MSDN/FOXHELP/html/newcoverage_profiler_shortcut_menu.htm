Coverage Profiler Shortcut Menu

See Also

Provides right-click access to features from the Add-Ins, Statistics, and Options dialog boxes.

Dialog Box Options

Use Smart Pathing

Sets Coverage Profiler to certain assumptions about the location of files in your project or application.

If you check this option, Coverage Profiler assumes folders in which you have already Located files contain additional source files it encounters later in the coverage log.

Uncheck this option for more complete control or if you have source files with the same name in different folders.

Cascade Windows

When Coverage Profiler runs in the coverage frame environment, this option cascades Coverage and Profiler windows.

Display Font

Opens the Options dialog box so you can specify the font used in the Source List pane of the Coverage Profiler window.

Code Font

Opens the Options Dialog Box so that you can specify the font used in the Source Code pane of the Coverage Profiler window.

Statistics

Opens the Statistics Dialog Box so that you can view summaries of the Coverage Profiler details and review Coverage Profiler statistics relative to a specific Visual FoxPro project.

Add-Ins

Opens the Add-Ins dialog box so that you can specify Add-Ins to use with Coverage Profiler.

View Source Log

Displays the current log file in the Visual FoxPro editor.

Fast Zoom Mode

Available only in Zoom mode, changes the results of single-clicks and arrow scrolling. This option enables you to avoid parsing long code blocks.

When you select Fast Zoom mode, source code of marked files displays immediately when you click the file in the Source List pane. When you select an unmarked file, you must double-click the file in the Source list to view the source code in the Source Code window (as the Source Code window label prompts you).
