Component Gallery Options Dialog Box

See Also

Enables you to specify global settings for catalogs and views in the Component Gallery.

Tab Options

The options available in the Component Gallery Options dialog box are grouped on the following tabs.

Tab To access these features
Standard Enables you to specify global defaults. This tab is always available.
Catalog Enables you to add catalogs to the Component Gallery and to specify whether catalogs are global or default.
Dynamic View Enables you to add, edit, or remove views from your view list. This tab is always available.
