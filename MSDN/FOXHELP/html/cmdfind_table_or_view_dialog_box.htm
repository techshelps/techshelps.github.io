Find Table or View Dialog Box

Allows you to select a table or views you can't locate in the database schema displayed in Database Designer. Use this dialog box to find tables or views in the database schema.

This dialog box appears when you select Find Object from the Database menu.

Dialog Box Options

Tables or Views

Shows a list of objects in your database layout that you can find. If the table or view you want does not appear in this list, it has not been added to the database.

Find button

Selects and displays the object in the database schema.
