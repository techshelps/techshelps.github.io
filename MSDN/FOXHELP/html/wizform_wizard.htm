Form Wizard

See Also

Creates a form for data entry from a single table. The size of the form is based on your Maximum design area setting on the Forms tab of the Options dialog box.

To run the Form Wizard

  1. From the Wizards submenu on the Tools menu, choose Form.

  2. In the Wizard Selection dialog box, choose Form Wizard.

Wizard Steps

Step 1 - Select Fields
Step 2 - Choose Form Style
Step 3 - Sort Records
Step 4 - Finish

If you create a form within a database, the Form Wizard can use input mask and format settings stored in the database. After you save the form, you can open and modify it later like any other form in the Form Designer.

For more information, see "Using Wizards" in Chapter 1, Getting Started, in the User's Guide.
