EXE Version Dialog Box

Displays information about the build number, as well as other information which you supply.

Dialog Box Options

Version Number

Major

Displays a four-digit number that can be edited. This number corresponds to a major version.

Minor

Displays a four-digit number that can be edited. This number corresponds to a minor version.

Revision

Displays an editable four digit number. This number corresponds to incremental updates.

Auto Increment

When checked, this option allows for automatic updates to the Revision number with every build of the program.

Version Information

Type

Lists seven categories of version information which you can define. These categories are Comments, Company Name, File Description, Legal Copyright, Legal Trademarks, Product Name, and Language ID.

Value

Displays the information entered about a single category. Click a category in the Type list to display, enter, or edit the information.
