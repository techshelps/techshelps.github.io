_WWidthP( ) API Library Routine Example

The following example displays the height and width in pixels of the frontmost window.

Visual FoxPro Code

SET LIBRARY TO WWIDTHP 
=WDIMEN()

C Code

#include <pro_ext.h>

void putLong(long n, int width)
{
   Value val;

   val.ev_type = 'I';
   val.ev_long = n;
   val.ev_width = width;

   _PutValue(&val);
}

void FAR Example(ParamBlk FAR *parm)
{
   WHANDLE wh = _WOnTop();

   _PutStr("\nheight ="); putLong(_WHeightP(wh), 5);
   _PutStr("\nwidth  ="); putLong(_WWidthP(wh), 5);
}

FoxInfo myFoxInfo[] = {
   {"WDIMEN", Example, 0, ""},
};
FoxTable _FoxTable = {
   (FoxTable FAR *) 0, sizeof(myFoxInfo)/sizeof(FoxInfo), myFoxInfo
};
