Append From Dialog Box

See Also

Lets you add records to the current table or view from another table, a spreadsheet, or text file. This dialog displays when you select Append Records from the Table menu.

Dialog Box Options

Type

Specifies the file type from which to append records.

From

Provides space for you to type the name of the file from which to append records. If you do not know the name, choose the dialog button to display the Open dialog box and select the desired file.

To

Displays the path and file name of the table or view to which the records will be appended.

Options

Displays the Append From Options dialog box, in which you specify fields to append, and build an expression which records to be appended must meet.

For more information, see "Appending Data" in Chapter 9, Importing and Exporting Data in the User's Guide.
