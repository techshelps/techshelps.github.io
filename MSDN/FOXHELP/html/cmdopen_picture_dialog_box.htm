Open Picture Dialog Box

See Also

Lets you open an existing graphic file. Corresponds to the GETPICT( ) function.

This dialog box appears when you issue ? GETPICT( ) in the Command window.

Dialog Box Options

Look in

Allows you to specify the folder that contains the file you want to open.

File name

Allows you to specify the file to open.

Files of type

Allows you to specify the file type to list.

Picture

The area that displays a preview of the selected graphic when the Preview check box is selected.

Preview

Allows you to preview the selected graphic file.
