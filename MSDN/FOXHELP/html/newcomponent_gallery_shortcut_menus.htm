Component Gallery Shortcut Menus

Component Gallery shortcut menus are divided into sections that contain groups of behaviors. The top section of a shortcut menu provides options that are common to all gallery items of the same type. The next sections provide options like Copy and Paste that are common to other groups of gallery items. The last item in each shortcut menu, Properties, opens a Component Gallery Properties dialog box. You can view or modify characteristics of the selected item by clicking the Properties option of any shortcut menu.

Note   Changes to properties here affect only the selected Component Gallery object. To change properties of the referenced object, use the Modify option from the shortcut menu for an item.

The Component Gallery provides the following shortcut menus:

Main Form shortcut menu

Displays when you right-click the Component Gallery toolbar.

Folder shortcut menu

Displays when you right-click a catalog or folder.

Item shortcut menu

Displays when you right-click a gallery item.

Background shortcut menu

Displays when you right-click the background of either the Component Gallery Catalog pane or Object pane in the Component Gallery Window.
