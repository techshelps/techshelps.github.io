Column Header Dialog Box

See Also

Sets height and other options for the Column Header band in a label or multiple-column report. This dialog box appears when you double-click the Column Header band separator bar in the Report Designer.

Dialog Box Options

Height

Specifies band height. Type a value in the box, click the UP ARROW or DOWN ARROW key to change the displayed value in increments of .05, or drag the band's separator bar with the mouse.

Constant Band Height

Prevents the band from stretching to accommodate lengthy data or shrinking to absorb blank lines that have been removed.

Run Expression

On Entry

Specifies that an expression be evaluated before the contents of the column header are printed. Type the expression in the box, or choose the dialog button to display the Expression Builder dialog box.

On Exit

Specifies that an expression be evaluated after the contents of the column header are printed. Type the expression in the box, or choose the dialog button to display the Expression Builder dialog box.

For more information, see "Modifying Your Layout" and "Customizing Your Layout" in Chapter 7, Designing Reports and Labels in the User's Guide and "Refining Page Layout" in Chapter 12, Adding Queries and Reports in the Programmer's Guide.
