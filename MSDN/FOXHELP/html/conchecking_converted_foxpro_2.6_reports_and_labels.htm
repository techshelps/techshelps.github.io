Checking Converted FoxPro 2.6 Reports and Labels

See Also

Converting a FoxPro 2.6 report or label to a Visual FoxPro file changes the table structure of the .frx or .lbx file and adds records to support the data environment of the report or label.

Once you convert a report to Visual FoxPro format, you can add functionality to it. For information on differences between FoxPro 2.6 reports and Visual FoxPro reports, see Differences Between Visual FoxPro and Previous Versions. For details on creating or modifying reports, see Report Designer.
