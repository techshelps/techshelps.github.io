Label Wizard

See Also

Creates labels from a table.

To run the Label Wizard

  • From the Wizards submenu on the Tools menu, choose Label.

Wizard Steps

Step 1 – Select Tables
Step 2 - Choose Label Type
Step 3 – Define Layout
Step 4 – Sort Records
Step 5 - Finish

For more information, see "Using Wizards" in Chapter 1, Getting Started, in the User's Guide; "Indexing Tables" in Chapter 7, Working with Tables, "Importing Data" in Chapter 9, Importing and Exporting Data, and "Understanding Code Pages in Visual FoxPro" in Chapter 18, Developing International Applications, all in the Programmer's Guide.
