WWW Search Page Wizard: Step 5 – Select Result Fields

See Also

The wizard lists the fields available for display on the results page. The list includes all of the fields in the selected search table. Select the fields in the order you want them to appear on the results page. For example, if you want the results to display the product picture, product name, category, and description from your Products table, first choose the fields storing the product picture, then the product name field, and so on.

You can select up to five fields.
