Files Tab, Project Information Dialog Box

See Also

Displays an alphabetical list of all project files, their type and include/exclude status, date and time last modified, and code page information.

Tab Options

Type

Displays an icon representing the file type.

Name

Displays the file name.

Last Modified

Displays the date and time the file was last modified.

Included

If you select this box, the file is included. If you clear the box, the file is excluded. For more information about including and excluding files in a project, see Project Menu.

Code Page

Displays the code page number. For more information about code pages, see "Working with Code Pages" in Chapter 18, Developing International Applications in the Programmer's Guide.

Update native code page

Updates the code pages for each file listed.
