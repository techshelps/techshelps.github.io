Report Wizard

See Also

Creates reports using one free table or one table or view in a database. The wizard prompts you with simple questions as you go through a brief series of steps, in which you specify the source and fields that you want to use to create controls on your report.

To run the Report Wizard

  1. From the Wizards submenu on the Tools menu, choose Report.

  2. In the Wizard Selection dialog box, choose Report Wizard.

Wizard Steps

Step 1 – Select  Fields
Step 2 – Group Records
Step 3 - Choose ReportStyle
Step 4 - Define Report Layout
Step 5 - SortRecords
Step 6 -Finish

For more information, see "Using Wizards" in Chapter 1, Getting Started, in the User's Guide and "Indexing Tables" in Chapter 7, Working with Tables, in the Programmer's Guide.
