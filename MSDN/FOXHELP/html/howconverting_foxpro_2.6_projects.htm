Converting FoxPro 2.6 Projects

See Also

To use FoxPro 2.6 projects (.pjx files) and FoxPro catalogs (.fpc files), you must convert them to Visual FoxPro projects. Once you convert them you can access your files from the Project Manager and visual designers, and add Visual FoxPro event model functionality.

To convert a FoxPro 2.6 project

  • From the File menu, choose Open, select Project, locate the project you want to convert, and then choose OK.

     – Or – 

  • Use the MODIFY PROJECT command.

    The Conversion Options dialog box appears.

If you choose functional conversion, the conversion process performs these steps:

  • Copies the original project files to a new subdirectory named OLDx in the original .pjx directory and automatically increments x as you convert more projects.

  • Converts the project and all associated screens, reports, and labels to Visual FoxPro format.

  • Moves code snippets to appropriate event and method code.

  • Sets form and control properties.

  • Creates a file with an .spr extension for each converted screen. This file is different from a FoxPro 2.6 .spr file. For more information, see Checking the .SPR File.

  • Places files in the following locations.
    FoxPro 2.6 files Project Manager tab
    .prg and .spr files Code tab
    .scx and .sct files Documents tab
    .frx and .lbx files Documents tab

After converting a project, Visual FoxPro opens the Project Manager so you can access all the project files. To ensure that your project converted as you intended, see Checking Converted FoxPro 2.6 Projects.
