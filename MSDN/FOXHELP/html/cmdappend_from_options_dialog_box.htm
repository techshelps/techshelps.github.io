Append From Options Dialog Box

See Also

Specifies fields to append and allows you to build an expression which records to be appended must meet. This dialog box appears when you choose Options in the Append From dialog box.

Dialog Box Options

Criteria

Fields

Displays the Field Picker dialog box, where you select fields to append.

For

Displays the Expression Builder dialog box, where you create a For expression that every record must meet to be appended.
