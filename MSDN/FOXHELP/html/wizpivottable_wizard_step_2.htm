PivotTable Wizard: Step 2 – Define Layout

See Also

The pivot table will contain a column for each unique value in the field that you drag to the Columns box, and a row for each unique value in the field that you drag to the Rows box. If you drag a field to the Page box, the pivot table will contain a drop-down list from which you can select the different pages. There will be one page for each unique value that exists in the field you drag to the Page box.
