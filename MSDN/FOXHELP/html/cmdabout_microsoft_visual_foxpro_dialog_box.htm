About Microsoft Visual FoxPro Dialog Box

Displays the Visual FoxPro copyright, licensing, and version information, and your product identification number.

To display this dialog box

  • From the Help menu, choose About Microsoft Visual FoxPro.

Dialog Box Options

System Info

Displays the Microsoft System Information tool which contains your system configuration information. To access help information about Microsoft System Information, press F1 or, choose Help Topics from the Microsoft System Information Help menu.
