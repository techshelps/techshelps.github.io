Insert System Menu Bar Dialog Box

Allows you to add menus and menu options to the Menu Designer window. This dialog box appears when you choose Insert Bar from the Menu menu or select the Insert Bar button on the Menu Designer window.

Menu Bar Options

Menu List Box

Lists the available menu names and menu option names, depending on whether you have chosen Usage or Prompt in the Order by area.

Order By

Usage

Designates that the Menu List Box display the available menu items in the order that they would appear on their respective menu titles. For example, all of the File menu items are listed first, in the order that they appear on the actual File menu. Edit menu items appear next, and so on.

Prompt

Designates that the Menu List Box display the available menu items in alphabetical order.

Insert

Inserts the selected menu item on the designated row in the Menu Designer window.
