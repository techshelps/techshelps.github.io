Save As Dialog Box

See Also

Saves a new file or saves an existing file with a different name. Corresponds to the PUTFILE( ) function.

The Save As command is enabled for the following file types:

  • Project

  • Query

  • Connection (see the following note)

  • View (see the following note)

  • Form

  • Report

  • Label

  • Program

  • Text File

  • Menu

    Note   When you choose Save or Save As for a connection or view, the Save dialog box appears.

This dialog box appears when you select Save As from the File menu.

Dialog Box Options

Code Page

This option is available if you are saving a text file or a table that has no code page. The Code Page dialog box is displayed so that you can attach the correct code page. If your existing text file uses the extended character set, choose Code Page and make the appropriate selections.
