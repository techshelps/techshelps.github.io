Application Wizard (5.0): Step 5 – Finish

See Also

Allows you to specify what happens when you choose Finish. You can save your application.

  • Selecting Save project will generate the project, menu, and program files and return to Visual FoxPro. Choose this option if you intend to work with the project at a later time.

  • Selecting Save project and modify it will generate the project, menu, and program files and will open the newly created project after returning to Visual FoxPro. Choose this option if you want to display the new project in the Visual FoxPro Project Manager.

  • Selecting Save application and run it will generate the project, menu, program files, and build the application as an .app file, and will then run the newly generated .app file. Choose this option to compile the new option and run it.

If you choose Framework only at Step 1, you can save or save and modify your project.
