Microsoft on the Web Submenu

Provides access to Microsoft Web pages of interest. To use the options on this submenu, you must be able access to the World Wide Web using any of the popular browsers. When you choose an option, Visual FoxPro loads the default browser and links directly to the page associated with your choice.

Free Stuff

Displays a Web page from which you can download software, such as new drivers and early release versions of new utilities.

Product News

Displays the Visual FoxPro home page, which contains links to the latest information about Visual FoxPro, including training opportunities, Visual FoxPro events, and product case studies.

Frequently Asked Questions

Displays the Microsoft Support Online Frequently Asked Questions Web page for Visual FoxPro, with links to FAQ pages for all versions of the product.

Online Support

Displays the Visual FoxPro Support Options Web page, which provides links to Microsoft Visual FoxPro newsgroups and reference information.

MSDN Online

Displays the MSDN home page.

Visual Studio Product News

Displays the Visual Studio home page.

Send Feedback

Displays the Submit Feedback Web page, in which you can fill out a survey that asks for usage information and product suggestions.

Best of the Web

Displays the Microsoft Best of the Web home page.

Search the Web

Displays a Web page from which you can search the entire World Wide Web.

Web Tutorial

Displays Microsoft's Internet Tutorial Web page. This page teaches you about the Internet and the World Wide Web, explains how to surf the Web, and provides references for terminology, etiquette, and other topics.

Microsoft Home Page

Displays the Microsoft Home Page, from which you can access information about Microsoft products, obtain product support, and download software.
