Export Options Dialog Box

See Also

Specifies which records and fields to export. This dialog box is displayed when you choose Options in the Export dialog box.

Dialog Box Options

Criteria

These options control which records and fields export and the conditions under which they export.

  • Scope   Displays the Scope dialog box, in which you specify how many records to export.

  • For   Displays the Expression Builder, in which you create a logical expression which every record must meet to be exported.

  • While   Displays the Expression Builder, in which you create a logical WHILE expression that must be true in order for records to be exported.

  • Fields   Displays the Field Picker dialog box, where you choose fields to be exported.

For more information, see "Exporting Data" in Chapter 9, Importing and Exporting Data in the User's Guide.
