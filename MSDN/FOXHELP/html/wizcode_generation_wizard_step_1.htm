Code Generation Wizard: Step 1 – Select Source and Destination

See Also

Allows you to select the source Visual Modeler (.mdl) file and the destination Visual FoxPro class library(.vcx) file of classes that are not mapped in Visual Modeler. If you specify a library file that doesn't exist, the wizard will create it.

In the model file, you can create predefined class mapping for a class in its property sheet.

Current Model is checked if you have the selected model open in Visual Modeler.
