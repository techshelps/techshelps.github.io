Table Designer

See Also

Use the Table Designer to create and modify database tables, free tables, fields, and indexes. The Table Designer helps you implement advanced features such as validation rules, default values, and default control classes.

Tab Options

  • Fields   Displays table fields in a scrollable grid.

  • Indexes   Contains a scrollable grid for defining indexes.

  • Table   Displays the information about the table and allows you to specify triggers and record-level rules.
