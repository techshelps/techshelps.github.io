Insert Object Reference Dialog Box

See Also

Allows you to see the name of all objects in the designer and insert a fully qualified reference to the object in your code.

This dialog box appears when you choose Objects List from the shortcut menu of a Code window in the Form Designer or Class Designer.

Dialog Box Options

Objects list

All the objects contained in the highest level container in the Form Designer or Class Designer. Contained objects are indented to indicate the container hierarchy.

Insert

Inserts a string with the full object reference at the cursor in the editing window.
