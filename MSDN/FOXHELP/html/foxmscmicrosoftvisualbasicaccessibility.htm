Microsoft Visual FoxPro Accessibility

Follow these steps to enhance accessibility to Visual FoxPro:

Enlarging Toolbar Buttons

To view enlarged toolbar buttons

  1. From the View menu, click Toolbars.

  2. Select the Large buttons check box.

Enlarging Text in the Code Window

To view enlarged text in the Code Window

  • From the Format menu, click Enlarge Font.

-or-

  • From the Format menu, click Font and choose the font size you want.

-or-

  1. With the Command window active, right-click and choose Properties from the Shortcut menu.

  2. On the Edit Properties dialog box, click the Font button and choose the font size you want.

Enlarging Text in the Debugger

To view enlarged text in the Debugger

  1. From the Options menus, click the Debug tab and then click the Font button.

  2. Select the font and font size you want.
