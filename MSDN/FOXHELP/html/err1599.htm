Data session #number was forced to ROLLBACK all transactions to avoid deadlock (Error 1599)

A deadlock was detected from another data session attempting to lock an index or memo file held by data session #number. A forced ROLLBACK was completed, leaving data session number in an error state.

Issue the appropriate number of ROLLBACK commands to clear the error state. These situations can be avoided by setting SYS(3052) on for index and memo files.
