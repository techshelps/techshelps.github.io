Visual FoxPro Transporter Dialog Box

See Also

Allows you to transport forms, labels, and reports across different FoxPro platforms and versions. This dialog box only appears if you are trying to convert Macintosh or MS-DOS files which have never contained Windows records. The Transporter creates Windows platform records from the Macintosh or MS-DOS records so that the Converter can then convert the files to the Visual FoxPro 6.0 file format. The Converter calls the Visual FoxPro Transporter dialog box when you click the Continue button in the Visual FoxPro Converter dialog box.

Dialog Box Options

Transport

Creates a Windows version of the Macintosh or MS-DOS records so that the Visual FoxPro Converter will recognize the file and convert the file to the correct file format.

Don't Transport

Cancels the file transport and conversion process. You can still open the file in the older version of FoxPro.

Font

Opens the Font dialog box so you can choose the default font for your transported screen, report or label.

Transport Objects From

Allows you to choose the appropriate platform from the drop-down list.
