LineSlant Property

Applies To   See Also

Specifies which way a line slants, from upper left to lower right or lower left to upper right. Available at design time and run time.

Syntax

Line.LineSlant[ = cSlant]

Settings

cSlant

The settings for the LineSlant property are:

Setting Description
\ (Default) Line slants from upper left to lower right.
/ Line slants from lower left to upper right.
