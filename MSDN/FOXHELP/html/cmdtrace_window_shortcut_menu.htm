Trace Window Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Trace window of the Debugger. These commands appear when you right-click the Trace window.

Shortcut Menu Options

Open

Displays the Open dialog box, which allows you to select a file.

Resume

Available if program execution is suspended. Allows you to continue execution of the program at the current line of code.

Step Into

Allows you to execute the next line of code or move into the next executable line of code in a called procedure.

Step Over

Allows you to execute the next line of code. If the next line of code calls a function, method, or procedure, then the function, method, or procedure is executed in the background.

Step Out

Allows you to continue executing the code in a procedure without stepping through the code line by line. Program execution is suspended again on a line of code following the procedure call in the calling program.

Set Next Statement

Places the current line marker on the line of code with the cursor. This line of code will then be executed when you step or resume execution.

Run to Cursor

Allows you to execute code from the current line indicator to the line of code with the cursor. Click the line of code you want to suspend execution on to place the cursor on that line.

Trace Between Breaks

Toggles trace between breakpoints.

Docking View

Allows you to specify whether the docking feature for the window is on or off. If unchecked, you cannot dock the window.

Hide

Hides the Trace window.

Font

Displays the Font dialog box where you can set the font type, style, and size. This command corresponds to the Font command in the Format menu.

Help

Displays Help on the selected item.
