Field Picker Dialog Box

See Also

Limits the number of active fields in a table so you can customize what is displayed. Fields that are not active cannot be accessed, examined, or modified. Data in the table itself is not affected. The selections you make in this dialog box are ignored unless you select Only Fields Specified by Field Filter in the Work Area Properties dialog box.

This dialog box appears when you choose Field Filter in the Work Area Properties dialog box, or Fields in the Quick Report dialog box, Append From Options dialog box, or Export Options dialog box.

Dialog Box Options

All Fields

Displays all field names in the current table.

Selected Fields

Displays field names you select in the All Fields box and copy with the Move button.

From Table

Specifies which table's fields are displayed in the All Fields box.

Move

Copies the selected fields from the All Fields box to the Selected Fields box.

All

Copies all fields from the All Fields box to the Selected Fields box.

Remove

Removes the selected fields from the Selected Fields box.

Remove All

Removes all fields from the Selected Fields box.
