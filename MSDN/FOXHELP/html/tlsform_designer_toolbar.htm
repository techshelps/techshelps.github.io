Form Designer Toolbar

See Also

The Form Designer toolbar is automatically displayed with the Form Designer. For information about displaying and manipulating toolbars, see "Using Toolbars" in Chapter 1, Getting Started, in the User's Guide.

This toolbar contains the following buttons.

Button Description
Set Tab Order Switches between Design mode and Tab Order mode for setting the tab order of objects. Disabled unless the form contains one or more objects.
Data Environment Displays the Data Environment Designer.
Properties Window Displays a window reflecting settings for the current object.
Code Window Displays the Code window for the current object so you can view and edit the code.
Form Controls Toolbar Shows or hides the Form Controls toolbar.
Color Palette Toolbar Shows or hides the Color Palette toolbar.
Layout Toolbar Shows or hides the Layout toolbar.
Quick Form… Runs the Form Builder, which gives you an easy, interactive way to add fields to the form as controls and define the form's style and layout.
Auto Format Runs the AutoFormat Builder, which gives you an easy, interactive way to apply formatting styles to selected controls. One or more controls must be selected to enable this button.

For more information, see "Using Toolbars" in Chapter 1, Getting Started, and "Adding Controls to a Form" in Chapter 8, Managing Data Through Forms, in the User's Guide; and the View Menu topic.
