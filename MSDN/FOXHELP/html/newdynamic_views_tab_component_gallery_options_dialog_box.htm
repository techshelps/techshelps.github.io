Dynamic Views Tab, Component Gallery Options Dialog Box

Displays the names of the Dynamic Views currently available in the Component Gallery. You can specify dynamic views in the Component Gallery Options dialog box.

Tab Options

New

Enables you to create a new Dynamic View in the Component Gallery.

Edit

Enables you to edit an existing Dynamic View by displaying the Edit View dialog box for the selected view.

Remove

Enables you to remove the selected Dynamic View from the Component Gallery.
