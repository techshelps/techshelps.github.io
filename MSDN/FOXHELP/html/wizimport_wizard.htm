Import Wizard

See Also

The Import Wizard imports data into Visual FoxPro tables from other file formats.

To run the Import Wizard

  • From the Wizards submenu on the Tools menu, choose Import.

Wizard Steps

Step 1 – Identify Data
Step 1a – Describe Data
Step 2 - Determine Data Format
Step 2a – Describe Data
Step 3 - Define Imported Fields
Step 3a - Specify International Options
Step 4 - Finish

For more information, see "Using Wizards" in Chapter 1, Getting Started, in the User's Guide; and "Importing Data" in Chapter 9, Importing and Exporting Data, and "Understanding Code Pages in Visual FoxPro" in Chapter 18, Developing International Applications, both in the Programmer's Guide.
