Setup Wizard: Step 4 – Specify Setup Options

See Also

The Setup Wizard creates installation dialog boxes with the title you specify in the Setup dialog box caption box. It also places the copyright statement in the Copyright information dialog box that the user can access from the About command on the Setup application's control menu. The Post-setup executable entry is optional. You can specify an application, such as a web registration utility, that you want the user to run immediately after installing your application.

Note   In order to advance to Step 5, you must move the insertion point outside the Setup dialog box caption and Copyright information boxes.
