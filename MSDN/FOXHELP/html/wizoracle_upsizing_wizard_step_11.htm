Oracle Upsizing Wizard: Step 11 – Finish

See Also

In the final step, you can choose to:

  • Upsize without generating SQL code.

  • Only generate and save SQL code for upsizing.

  • Upsize and save generated SQL code.

To upsize your database

  1. Select the upsizing option you want.

  2. Choose the Finish button.

    The Oracle Upsizing Wizard creates devices and databases if necessary, and begins exporting Visual FoxPro objects to Oracle.

The first and third upsizing options are available only if you have the necessary permission on the server. If you choose either of the options that save generated SQL code, all the SQL code generated by the Oracle Upsizing Wizard is stored on your hard disk.

Note   Upsizing can take a long time, depending on the size of your data, the amount of network traffic and the number of concurrent demands being handled by your server. Large tables can require several hours to export.
