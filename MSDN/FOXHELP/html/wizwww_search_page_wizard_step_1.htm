WWW Search Page Wizard: Step 1 – Select Table

See Also

The wizard lists the available tables. To display the tables from your database, select it in the Databases and Tables box. If your database does not appear in this list, choose Browse to see more files.

Choose a table that has at least one index key in order to ensure optimal performance during searches.

Note   Choose a table located on the HTTP server, or one that has the same structure and relative path as the one you intend to use on the server. If you choose a table that has a different structure or path than the file on the server, data might not be available during the search.
