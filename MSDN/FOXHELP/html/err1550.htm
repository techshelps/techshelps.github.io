.DBC internal consistency error (Error 1550)

An internal database error has occurred.

  • The database is corrupt.

    Try VALIDATE DATABASE to diagnose or repair the problem.

  • The database container index (.dcx) is corrupt or not synchronized with the database container file (.dbc), usually as a result of an abnormal system shut-down.

    Rebuild the index by closing the database, deleting the index file , and reopening the database.

    If the problem is not corrected, record your recent actions while using Visual FoxPro and then call Microsoft Support.
