SYS(2023) - Temporary Path

See Also

Returns the path on which Visual FoxPro stores its temporary files.

Syntax

SYS(2023)

Returns

Character

Remarks

You can specify the path on which temporary files are stored by including the special TMPFILES configuration item in your Visual FoxPro configuration file.

For additional information about the special TMPFILES configuration item, see Chapter 3, "Configuring Visual FoxPro," in the Installation Guide.
