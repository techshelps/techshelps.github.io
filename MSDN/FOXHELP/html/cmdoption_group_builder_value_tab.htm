Value Tab, Option Group Builder

See Also

The Value tab in the Option Group Builder specifies a table or view field in which to place the option group's value. This builder also has the Buttons tab and Layout tab.

When you choose OK, the builder closes, applying the property settings from every tab.

Tab Options

Field Name

Specifies a table or view field in which to store the option group's value. By default, the list includes fields from a table. Choose the dialog button to display the Open dialog box so you can select another file. This option corresponds to the ControlSource property.
