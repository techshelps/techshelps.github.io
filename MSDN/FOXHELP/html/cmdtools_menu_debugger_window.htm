Tools Menu, Debugger Window

See Also

Provides access to the Debugger dialog boxes.

Menu Commands

Breakpoints

Opens the Breakpoints dialog box so you can add, delete, enable or disable breakpoints.

Event Tracking

Opens the Event Tracking dialog box so you can specify which events are written to the Debug Output window or a file when they occur.

Coverage Logging

Opens the Coverage dialog box so you can turn coverage logging on or off and specify a file for the coverage log.
