Remote Automation Connection Manager

See Also

The Remote Automation Connection Manager (Racmgr32.exe) in the Windows 95 System folder or the Windows NT System32 folder provides a way to configure and store the registry settings for remote data connections including server configuration, client access configuration, System Security Policy options, and authentication levels.

To establish or manage remote automation connections

  1. Install any component .dll or .exe file you want to use on both your local and target remote computers.

  2. Use the Remote Automation Connection Manager to configure the server.

  3. Specify the COM class, network connection criteria, and System Security Policy options.

  4. Use the Remote Automation Connection Manager to configure the remote computer.

  5. For detailed information, see the section Using Remote Automation in Chapter 16, “Adding OLE,” in the Programmer's Guide.
