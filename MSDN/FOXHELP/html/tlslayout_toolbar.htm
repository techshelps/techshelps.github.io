Layout Toolbar

See Also

Use the Layout toolbar to align and manipulate the locations of controls on a report or form. For information about displaying and manipulating toolbars, see "Using Toolbars" in Chapter 1, Getting Started, in the User's Guide.

This toolbar contains the following buttons.

Button Description
Align Left Sides Aligns selected controls on the leftmost edge. Enabled when you select more than one control.
Align Right Sides Aligns selected controls on the rightmost edge. Enabled when you select more than one control.
Align Top Edges Aligns selected controls on the topmost edge. Enabled when you select more than one control.
Align Bottom Edges Aligns selected controls on the bottommost edge. Enabled when you select more than one control.
Align Vertical Centers Aligns the selected controls' centers on a vertical axis. Enabled when you select more than one control.
Align Horizontal Centers Aligns the selected controls' centers on a horizontal axis. Enabled when you select more than one control.
Same Width Adjusts the width of the selected controls to the widest control.
Same Height Adjusts the height of the selected controls to the tallest control.
Same Size Adjusts the size of the selected controls to the largest control.
Center Horizontally Aligns the selected controls' centers on a vertical axis in the middle of the form.
Center Vertically Aligns the selected controls' centers on a horizontal axis in the middle of the form.
Bring to Front Positions the selected controls in front of all other controls.
Send to Back Positions the selected controls behind all other controls.

For more information, see "Using Toolbars" in Chapter 1, Getting Started, and "Aligning Controls" in Chapter 8, Managing Data Through Forms, both in the User's Guide.
