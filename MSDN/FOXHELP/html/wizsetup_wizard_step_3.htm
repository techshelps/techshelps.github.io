Setup Wizard: Step 3 – Create Disk Image Directory

See Also

The wizard creates a distribution subdirectory containing disk images for each type of disk you specify.

If you select 1.44 MB 3.5-inch, the wizard will create images for s.5 inch floppy disks in your distribution directory.

If you select Websetup (compressed), the wizard will create a densely compressed setup designed for fast download from a web site.

If you select the Netsetup option, the wizard will create a single directory that will contain all the files.

You can create the disk image directory before you run the wizard. If you want the wizard to create the directory for you, type a directory name in the text box. If you have run the Setup Wizard before, the wizard uses the location you specified last time as the default.
