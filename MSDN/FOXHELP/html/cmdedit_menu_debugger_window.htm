Edit Menu, Debugger Window

See Also

Contains options for editing text.

Menu Commands

Cut

Removes selected text to the Clipboard.

Copy

Duplicates selected text and puts it on the Clipboard.

Paste

Places cut or copied text from the Clipboard at the insertion point.

Select All

Selects all text in the active Debugger window.

Find

Displays the Find dialog box so you can search for text.

Find Again

Searches for the last target specified in the Find dialog box.
