Quick Report Dialog Box

See Also

Lets you select fields and a field layout for a report.

You can also use Visual FoxPro wizards to quickly create a report. These include the Report Wizard and the One-to-Many Report Wizard.

Dialog Box Options

Field Layout

Shows column layout on the left and row layout on the right. Choose column layout to display the fields from left to right across the page. Choose row layout to display the fields one below the other down the page.

Titles

Determines whether field names will be displayed as titles in label controls either above or next to the corresponding field.

Add Alias

Automatically adds the alias to all field names in the Report Designer window.

Add Table to Data Environment

Automatically adds the table to the data environment.

Fields

Displays the Field Picker dialog box, in which you select fields to display in your report.

For more information, see "Adding Controls with Quick Report" in Chapter 7, Designing Reports and Labels in the User's Guide.
