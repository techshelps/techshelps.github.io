Toolbars Shortcut Menu

See Also

Contains shortcuts to other toolbars. These commands appear when you right-click a toolbar.

Shortcut Menu Options

Color Palette

Displays the Color Palette toolbar.

Database Designer

Displays the Database Designer toolbar.

Debugger

Displays the Debugger toolbar. This command is enabled in the Debugger.

Form Controls

Displays the Form Controls toolbar.

Form Designer

Displays the Form Designer toolbar.

Layout

Displays the Layout toolbar.

Print Preview

Displays the Print Preview toolbar.

Query Designer

Displays the Query Designer toolbar.

Report Controls

Displays the Report Controls toolbar.

Report Designer

Displays the Report Designer toolbar.

Standard

Displays the Standard toolbar at the top of the Visual FoxPro screen toolbar.

View Designer

Displays the View Designer toolbar.

Toolbars

Displays the Toolbars dialog box.

Customize

Displays the Customize Toolbar dialog box.
