FillStyle Property

Applies To   See Also

Specifies the pattern used to fill shapes and figures created with the Circle and Box graphics methods. Available at design time and run time.

Syntax

Object.FillStyle[ = nStyle]

Settings

nStyle

Specifies the pattern used to fill a shape or figure. The settings for the FillStyle property are:

Setting Description
0 Solid.
1 (Default) Transparent. The FillColor property is ignored.
2 Horizontal Line.
3 Vertical Line.
4 Upward Diagonal. Upward diagonal lines from upper left to lower right.
5 Downward Diagonal. Downward diagonal from lower left to upper right.
6 Cross. Crossed vertical and horizontal lines forming squares.
7 Diagonal Cross.
