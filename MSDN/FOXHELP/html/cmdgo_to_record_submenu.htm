Go To Record Submenu

See Also

Contains options that let you position the record pointer on a specific record.

Go To Record Submenu Commands

Top

Positions the pointer on the first record in the table or view.

Bottom

Positions the pointer on the last record in the table or view.

Next

Positions the pointer on the record immediately after the current record.

Previous

Positions the pointer on the record immediately before the current record.

Record #

Displays the Go To Record dialog box, in which you specify a record number.

Locate

Displays the Locate Record dialog box, in which you can find an indexed or unindexed field or find more than one record in a table or view.
