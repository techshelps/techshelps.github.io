Forms Tab, Application Builder

See Also

Enables you to specify menu types, the menu from which a form is called, toolbars, and whether a form can be instanced multiple times.

For each form listed, check the options you want to apply.

Tab Options

Name

Enables you to specify the friendly name for the selected form.

Add

Enables you to select a form to add to your application. When you select a form to add, the check box options are enabled.

Edit

Enables you to modify the selected form in the Form Designer.

Remove

Removes a form from your application.

Single Instance

Specifies whether you can open only a single instance of a form in the application.

Use Navigation Toolbar

Specifies that the builder include a navigation toolbar for the highlighted form.

Use Navigation Menu

Specifies that the builder include a navigation menu for the highlighted form.

Appear in File New dialog

Specifies that the form name appears in the built application's New dialog box.

Appear in File Open dialog

Specifies that the form name appears in the built application's Open dialog box.

If you choose OK, the builder closes, applying the property settings from all tabs.
