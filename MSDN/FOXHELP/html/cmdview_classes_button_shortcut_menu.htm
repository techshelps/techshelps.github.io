View Classes Button Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Form Designer. This menu appears when you click the View Classes button of the Form Controls toolbar.

Shortcut Menu Options

Add

Displays the Open dialog box, which allows you to select a Visual Class Library to display on the Form Controls toolbar.

Standard

When selected, displays the standard controls for the Form Controls toolbar.

ActiveX Controls

When selected, displays ActiveX controls on the Form Controls toolbar.
