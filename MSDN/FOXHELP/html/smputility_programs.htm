Utility Programs

These tools are provided with Visual FoxPro to help speed the process of application development.

Utility Description
Addlabel.app Adds label sizes to the new label dialog box in the label designer.
Analyzer.app Dynamically traces the structure and symbols in Visual FoxPro application files
Code Analyzer Provides visual controls, a menu, and keyboard options to navigate through application file code by using output from the Documenting Wizard.
Cpzero.prg Removes the code page mark from any file that has a table file structure.
Gendbc.prg Generates a program that can re-create a database.
Genhtml.prg Generates a Web page based on data in a table.
Hexedit.app Examines or changes address, byte, and ASCII values of data in a file.
Racmgr32.exe Configures and stores the registry settings for remote data connections including server configuration, client access configuration, System Security Policy options, and authentication levels
Transformer Maps values of properties of existing forms, class libraries, and projects according to specific conditions.
