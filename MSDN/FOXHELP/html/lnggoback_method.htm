GoBack Method

Applies To   See Also

Navigates backwards in the history list of an Active Document host.

Syntax

HyperLink.GoBack( )

Remarks

The GoBack method is supported only when an Active Documents is hosted in an Active Document container.

The GoBack method is ignored when executed outside of an Active Document host. For example, it is ignored when an Active Document is run in the Visual FoxPro runtime or in an interactive Visual FoxPro session.
