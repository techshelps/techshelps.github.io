Visual FoxPro

Explore the Samples

The Visual FoxPro sample applications are designed to let you see how the sample works, how it is coded, and how you can apply its features in your own applications.

Be sure to install the Samples before trying to open or run them from here. For more information on installing samples, click here.

To open a sample application project in the Project Manager, click Open. To run a sample application, click Run.

Solutions Sample Open Run
Includes over 120 examples that illustrate particular features of Visual FoxPro including Foundation Classes, Access/Assign methods, OLE drag and drop, and much more. For more information, click here.

Tasmanian Traders Sample Open Run
An integrated Visual FoxPro application with logon security, order entry functionality, order history, invoices, and other real world implementations. For more information, click here.

Client/Server Sample Open Run
An application that illustrates some of the features that make Visual FoxPro an ideal development tool for client/server applications, including offline views, upsizing, and concurrency. For more information, click here.

Pool Manager OLE Server Sample Open
An example of a remote job manager that you can direct to perform certain tasks such as printing reports. The pool manager processes will do the task and send a notification when the job is finished. For more information, click here.

Gopher OLE Server Sample Open
Simulates a smart search business object which locates a customer that may be contained in 1 of many different databases. For more information, click here.

FoxISAPI OLE Server Sample Open
Manages sending selected records of Visual FoxPro data as HTML back to an Internet browser. For more information, click here.
