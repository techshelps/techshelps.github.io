Import Wizard: Step 3 – Define Imported Fields

See Also

Allows you to specify the name of the columns in the table, the data type of the columns, and the width of the columns. Click below the existing heading of the column in the Field grid and edit the information listed in the Name, Type, Width, and Decimal boxes.

If you don't want to import a specific column, click below the existing heading, and then press DELETE. The space for the column will still be visible (without a column heading name), but when you finish the wizard and import the data, that column will be ignored.
