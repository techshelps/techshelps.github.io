View Tab, Options Dialog Box

See Also

Contains options for controlling the display of Visual FoxPro features.

When you choose Set As Default, which appears on every tab in the dialog box, Visual FoxPro saves all the options on all tabs.

Tab Options

Show

Status bar

Specifies whether Visual FoxPro displays the status bar at the bottom of the main window. Corresponds to the SET STATUS BAR command.

Clock

Specifies whether the system clock appears in the status bar. You must choose the Status Bar option for this option to have effect. Corresponds to the SET CLOCK STATUS command.

Command results

Specifies that Visual FoxPro displays the results of certain commands including APPEND FROM, AVERAGE, CALCULATE, COUNT, INDEX, SUM, and TOTAL, in the main Visual FoxPro window, the system message window, the graphical status bar, or a user-defined window. Corresponds to the SET TALK command.

System messages

Specifies that Visual FoxPro displays certain system messages in the status bar. This option corresponds to the SET NOTIFY command.

Recently used project list

Specifies that the 1, 2, 3, 4 commands appear on the File menu, allowing you to directly open one of the four projects most recently used. This option is disabled if Resource file in the File Locations tab has been unchecked or if SET RESOURCE has been set to OFF.

Open last project on startup

Specifies that Visual FoxPro opens the most recently used project when starting. This option is disabled if Resource file in the File Locations tab has been unchecked or if SET RESOURCE has been set to OFF.
