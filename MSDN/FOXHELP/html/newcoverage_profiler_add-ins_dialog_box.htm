Coverage Profiler Add-Ins Dialog Box

See Also

Enables you to locate and run .app, .fxp, or .prg files to change or enhance the Coverage Profiler. You can use these Add-Ins to add controls, queries, dialog boxes, and other enhancements to the Coverage Profiler.

Dialog Box Options

Add-Ins

Allows you to select an Add-In program to run.

Register this Add-In after running

Adds the current Add-In to a drop-down list so that you can access the Add-In in the future.

Note   You can set the Coverage Profiler to register new Add-Ins by default by selecting the Register Add-Ins when run check box in the Coverage Profiler Options dialog box.

Run

Starts the Add-In program.
