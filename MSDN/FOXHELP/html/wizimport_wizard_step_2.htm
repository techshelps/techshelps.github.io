Import Wizard: Step 2 – Determine Data Format

See Also

Allows you to specify the type of separator for the data, and the point at which the Import Wizard should begin importing fields.

Data Format

Delimited

Specifies that the fields in the file are separated by a character.

Fixed Width

Specifies that the fields in the file are aligned in columns.

Field Names In Row

Allows you to specify which row contains the field names for the file. The Import Wizard automatically replaces spaces with underscores and reduces field name size to fit the 10-character maximum.

Begin Import at Row

Allows you to specify the row with which to begin importing data. This option allows you to avoid importing header data.

Worksheet

Specifies the name of the worksheet if you are importing a worksheet.

Field Grid

For a text file, displays the record number and data as it is found in the file. For other types of files, displays the file data as it will appear when you have completed the Import Wizard.
