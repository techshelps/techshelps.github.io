Checking the .SPR File

See Also

FoxPro 2.6 screens (.scx) file conversion creates a new (.spr) file. If the original .scx contains preprocessor directives, a global include file is also created. The new files enable Visual FoxPro to handle FoxPro 2.6 preprocessor directives, and to run the converted screen if other application components expect an .spr file.

If the original .scx file contains preprocessor directives, the conversion process:

  • Creates a global include file that contains all #DEFINE statements found in a screen.

  • Places a statement in the new .spr file that references the include file.

  • Places code representing the FoxPro 2.6 #INSERT preprocessor directive directly into the form.

For more information on the location of specific code segments, see Differences Between Visual FoxPro and Previous Versions.
