READ Compatibility Properties and Events

See Also

The READ compatibility of a form or form set is controlled by its WindowType property. The WindowType property of a form set overrides the WindowType settings of all forms it contains. Initially, converted screens have a WindowType property setting of 2 (Read) or 3 (Read Modal) and corresponding Read properties and events, listed in the table below.

With the WindowType property set to 2 or 3, converted forms provide the following properties and events for backward compatibility.
