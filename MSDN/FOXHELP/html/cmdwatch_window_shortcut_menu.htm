Watch Window Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Watch window of the Debugger. These commands appear when you right-click on the Watch window.

Shortcut Menu Options

Insert Watch

Inserts a new blank watch expression.

Delete Watch

Deletes the highlighted watch expression.

Docking View

Allows you to specify whether the docking feature for the window is on or off. If unchecked, you cannot dock the window.

Hide

Hides the Watch window.

Font

Displays the Font dialog box where you can set the font type, style, and size. This command corresponds to the Font command in the Format menu.

Help

Displays Help on the selected item.
