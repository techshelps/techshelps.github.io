Database Designer Toolbar

See Also

The Database Designer toolbar is displayed when the Database Designer is active. For information about displaying and manipulating toolbars, see "Using Toolbars" in Chapter 1, Getting Started, in the User's Guide.

This toolbar contains the following buttons.

Button Description
New Table Creates a new table using a wizard or designer.
Add Table Adds an existing table to a database.
Remove Table Removes the selected table from the database or deletes it from the disk.
New Remote View Creates a remote view using a wizard or a designer.
New Local View Creates a local view using a wizard or a designer.
Modify… Opens the selected table or query in the Table Designer or Query Designer.
Browse Displays the selected table or view in the Browse window for editing.
Edit Stored Procedures Displays a Microsoft Visual FoxPro stored procedure in the Editing window.
Connections Displays the Connections dialog box so you can access available connections or add new connections through the Connections Designer.

For more information, see "Introducing Visual FoxPro Designers," "Using Toolbars," and "Using Wizards," in Chapter 1, Getting Started, and "Creating a New Database" and "Relating Tables" in Chapter 3, Collecting Tables into a Database, both in the User's Guide.
