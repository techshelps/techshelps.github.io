Value Tab, Edit Box Builder

See Also

The Value tab in the Edit Box Builder specifies a field in which to store the edit box's value. This builder also has the Format tab and Style tab.

When you choose OK, the builder closes, applying the property settings from every tab.

Tab Options

Field Name

Specifies a table or view field in which to store the edit box's value. Type the field name, or click the dialog button to display the Open dialog box, where you can select another table or view. In the Edit Box Builder, click the drop-down arrow next to the Field Name box to select a field.

The default contents of this tab are all the fields in the data environment. This option corresponds to the ControlSource property.

For more information, see "Setting the Data Environment" in Chapter 9, Creating Forms, in the Programmer's Guide.
