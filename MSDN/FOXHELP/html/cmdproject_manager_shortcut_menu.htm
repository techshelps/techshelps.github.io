Project Manager Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Project Manager. Items that appear on shortcut menus depend on what you right-click in the Project Manager.

Shortcut Menu Options

Expand All

Expands all collapsed file subgroups in the Project Manager, so you can see the files that the subgroups contain. This command is only available in the Project Manager through the shortcut menu.

Exclude

In a project, excludes a file that was previously included. Excluded files are not compiled into the application. Enabled only when you select an included file in Project Manager. Excluded files appear with a slashed circle before their names in the Project Manager. Excluded files are listed in the Project Manager window for your reference, but you must manually distribute them if the application needs them. This command corresponds to the Exclude command in the Project menu.

Include

In a project, includes a file that was previously excluded. Enabled only when you select an excluded file. All included files are compiled into the .app or .exe file, making them read-only. If you want to write to a file at run-time, you should mark them as excluded. This command corresponds to the Include command in the Project menu.

Set Main

Specifies that the selected program, menu, query, or form is the application's main program; the program that is initially executed in a compiled application. You must select a program, form, or menu in the Project Manager to enable this command. When you have selected a main program, this command displays a check mark next to it whenever you have selected the main program in the Project Manager. The main program name is displayed in bold letters in the Project Manager. This command corresponds to the Set Main command in the Project menu.

Rename

Displays the Rename File dialog box where you can rename the selected file. Enabled when you have a file selected in the Project Manager. This command corresponds to the Rename File command in the Project menu.

Edit Description

Displays the Description dialog box in which you modify the text that appears as the file description at the bottom of the Project Manager window. This command corresponds to the Edit Description command in the Project menu.

Project Info

Displays the Project Information dialog box, in which you view and edit information about the project and its files. This command is enabled when you select a file in the Project Manager window. The Project Info shortcut command corresponds to the Project Info command in the Project menu.

Code Page

Displays the Code Page dialog box for cross-platform data sharing. This command is enabled when you select a text file, query, icon, or bitmap file name in Project Manager.

Display Description/Path

Displays or hides the description and path information found at the bottom of the Project Manager. This command appears when you right-click in the Description/Path area of the Project Manager. This command can only be accessed in the Project Manager through the shortcut menu by right-clicking in the Description/Path area or on the Project Manager title bar.

Move

Allows you to move the position of the Project Manager using the arrow keys. This command can only be accessed in the Project Manager through the shortcut menu by right-clicking in the Description/Path area or on the Project Manager title bar.

Size

Allows you to adjust the size of the Project Manager window using the arrow keys. This command can only be accessed in the Project Manager through the shortcut menu by right-clicking in the Description/Path area or on the Project Manager title bar.

Close

Closes the Project Manager window. This command can only be accessed in the Project Manger by right-clicking in the Description/Path area or on the Project Manager title bar. This command corresponds to the Close command in the File menu.

Undock

Undocks the Project Manager when it is in the docked mode. This command can only be accessed in the Project Manager through the shortcut menu by right-clicking anywhere on the docked Project Manager.

Builder

Runs the Application Builder.

Build

Displays the Build Options dialog box, in which you create a custom application or refresh an existing project. This command can only be accessed through the shortcut menu while Project Manager is in docked mode. When the Project Manager is undocked, this command corresponds to the Build command in the Project menu or the Build button in the Project Manager window.

Help

Displays Help on the selected item.
