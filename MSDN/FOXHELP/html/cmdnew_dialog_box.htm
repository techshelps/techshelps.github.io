New Dialog Box

See Also

Lets you create new files using designers and wizards.

Visual FoxPro provides a range of designers to help you create classes, connections, databases, forms, labels, menus, queries, local and remote views, reports, and tables. You may also use a Visual FoxPro wizard to create many of these files, including a variety of reports, forms, and queries. New program or text files are created simply by typing them in the Editing window. The following table shows you the file types and how each can be created.

Shortcut

CTRL+N

The New dialog box appears when you select New from the File menu.

Dialog Box Options

File Type

New File

Displays a dialog box, designer, or editing window so you can create the selected file type.

Wizard

Opens a wizard that creates the selected file type. If there is a group of wizards to choose from, Visual FoxPro will prompt you to select the wizard you want.

For more information, see "Using Designers" and "Using Wizards" in Chapter 1, Getting Started in the User's Guide.
