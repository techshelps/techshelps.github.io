WWW Search Page Wizard: Step 7 – Finish

See Also

After specifying the options for your pages, you are ready to create them. Choose Finish to display the Save As dialog box and provide a name and the directory for your Web pages. The wizard creates three files in the directory you specify, as listed in the following table.

File created by wizard Description
Name.htm HTML file used as the search page.
Name.htx HTML file used as the results page. The page includes a template that is filled in with the records returned by the search.
Name.idc Specifications for your search, including the SQL statement that will be used to find the records the user is seeking.
