Connection Wizards Overview

See Also

The Visual FoxPro Connection Wizards include the Code Generation Wizard and the Reverse Engineering Wizard. These wizards let you easily manage transfers between Visual FoxPro class libraries and Microsoft Visual Modeler models and provide the following functions:

  • Create and save new object models.

  • Locate and select an object model or class to transfer.

  • Modify object models.

  • Create libraries and class definitions.

  • Identify and resolve naming and inheritance conflicts.

  • Save exported Visual FoxPro libraries and classes to Visual Modeler default diagrams and model files.

An object modeler describes the elements and characteristics of a project or application and provides class diagramming, code generation, and reverse engineering so you can more directly design or maintain an application at a high level of abstraction.

The Code Generation Wizard and the Reverse Engineering Wizard connect Visual FoxPro libraries and classes to the object model abstractions and tools of Visual Modeler.
