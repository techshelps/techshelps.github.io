Remote View Wizard

See Also

Creates views using remote data (ODBC). You must have an existing database to store your view in as well as an existing data source or connection. If you do not have a database open, you are prompted to select an existing database or create one.

Before you run the Remote View Wizard, you can set options for remote view and connection defaults on the Remote Data tab of the Options dialog box. You can create connections in the Connection Designer.

To run the Remote View Wizard

  1. From the Wizards submenu on the Tools menu, choose Query.

  2. In the Wizard Selection dialog box, choose Remote View Wizard.

Wizard Steps

Step 1 - Choose Data Source
Step 2 – Select Fields
Step 3 – Relate Tables
Step 3a - IncludeRecords
Step 4 – Sort Records
Step 5 - FilterRecords
Step 6 -Finish

For more information about defining data sources, connections, or ODBC, see Chapter 1, Installing Visual FoxPro, in the Installation Guide and Chapter 6, Querying and Updating Multiple Tables, in the User's Guide. For more information about wizards, see "Using Wizards" in Chapter 1, Getting Started, also in the User's Guide.
