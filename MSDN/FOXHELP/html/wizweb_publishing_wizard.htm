Web Publishing Wizard

See Also

Enables you to display data from a table or view in an HTML document. You can use or share the file that the Web Publishing Wizard creates in your browser or edit it in a text editor.

To run the Web Publishing Wizard

  • From the Wizards submenu on the Tools menu, choose Web Publishing.

In the Web Publishing Wizard you can specify the layout for the data as well as the layout and style of the head and body of the HTML page. You can use the default layouts and visual styles that come with the wizard or customize those layouts and styles. In addition, you can load an existing cascading style sheet (.css) file.
