Layout Tab, List Box Builder

See Also

The Layout tab in the List Box Builder controls column width and display. The other List Box Builder tabs are: List Items tab, Style tab, and Value tab.

When you choose OK, the builder closes, applying the property settings from every tab.

Tab Options

Adjust the Width of the List Box to Display All Columns

Specifies that all columns will be displayed, and the list box will be automatically sized as required. This option corresponds to the Width property.

Column Grid

Shows the columns defined on the List Items tab. You can adjust the column width by dragging the right edge of a column heading with the mouse.

You can hide a column by double-clicking a column heading if you have chosen "Fill the list with fields from a table or view" in the List Items tab. This option corresponds to the ColumnWidths property.
