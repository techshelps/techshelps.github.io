Files Tab, Visual FoxPro Transformer

See Also

Specify one or more project (.PJX), form (.SCX), or class library (.vcx) files, or folders on which to run conversion rules.

Tab Options

Add File

Specify the target project, form, or class library file.

Add Folder

Specify the target folder. You can use wildcard characters as templates for any file or folder you specify.

Remove

Remove the highlighted file from the Selected File list.

Search Subdirectories

Directs the Transformer to search all folders contained in the listed target folder.

Create Log Only

Directs the Transformer to only create a log of the process you define in the Rules tab. This checkbox also displays on the Log tab.

Log To File

Directs the Transformer to copy the process log to a file you specify in the edit box by accepting the default file, TRANSFRM.LOG, or by using the Dialog button.
