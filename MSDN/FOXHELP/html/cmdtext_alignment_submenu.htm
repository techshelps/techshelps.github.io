Text Alignment Submenu

See Also

Adjusts alignment and spacing of text within a field or label control in a report or label.

These commands are available when you work with a report or label. To enable them, you must select a field or label control.

Submenu Commands

Left

Left-justifies text in the selected control.

Center

Centers text in the selected control.

Right

Right-justifies text in the selected control.

Single Space

Displays selected text with no blank lines between lines of text in a label control, text or program file, or the Command Window.

1 1/2 Space

Displays selected text with one and a half blank lines between lines of text in a label control, text or program file, or the Command Window.

Double Space

Displays selected text with two blank lines between lines of text in a label control, text or program file, or the Command Window.
