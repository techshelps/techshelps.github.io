Print Setup Dialog Box

See Also

Selects a printer, printer path, and paper setting. The settings you can adjust in this dialog box depend on the printer you have installed, and these settings are saved with the report or label. You can also change the settings for the current print job; these settings are not saved with the report file.

Dialog Box Options

Printer

Name

Displays the names of the printers currently installed on your machine.

Status

Displays whether or not the chosen printer is the default printer, and if it is ready to print.

Type

Displays the model of the selected printer. You specify the printer model when you first install the printer.

Where

Displays the network printer path.

Comment

Displays comments about the printer.

Orientation

Portrait

Orients the paper vertically for printing.

Landscape

Orients the paper horizontally for printing.

Paper

Size

Specifies the paper size to use. You can print either to paper or envelopes.

Source

Specifies the paper source (for example, upper tray) in the printer.

For more information, see Chapter 7, Designing Reports and Labels in the User's Guide.
