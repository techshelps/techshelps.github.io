Graph Wizard: Step 4 – Finish

See Also

If you choose to save the graph to a table, the wizard creates a table with a single record containing a General field that stores an Automation object for your graph. After the wizard is finished, it leaves your new table open. To view the graph, browse the new table and double-click the Olegraph field.

To create a legend for the graph, select Add a legend to the graph. To preview the graph, click the Preview button. To graph null values, select Show Null Values.

If you choose to create a query, the wizard saves a query that sends its results to a graph. After the wizard saves the query, you can open and modify it like any other query in the Query Designer.
