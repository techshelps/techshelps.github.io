Call Stack Window Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Call Stack window of the Debugger. These commands appear when you right-click the Call Stack window.

Shortcut Menu Options

Ordinal Position

Allows you to show or hide the ordinal position indicator.

Current Procedure

Allows you to show or hide the current procedure indicator.

Call Stack Indicator

Allows you to show or hide the call stack indicator.

Docking View

Allows you to specify whether the docking feature for the window is on or off. If unchecked, you cannot dock the window.

Hide

Hides the Call Stack window.

Font

Displays the Font dialog box where you can set the font type, style, and size. This command corresponds to the Font command in the Format menu.

Help

Displays Help on the selected item.
