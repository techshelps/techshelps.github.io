Graph Wizard

See Also

The Graph Wizard creates a graph from a Visual FoxPro table using Microsoft Graph.

The larger your table is, the longer it will take the wizard to process your graph. You might consider limiting the scope of records in a large table by creating a view before you run the Graph Wizard.

To run the Graph Wizard

  1. From the Wizards submenu on the Tools menu, choose Query.

  2. In the Wizard Selection dialog box, choose Graph Wizard.

Wizard Steps

Step 1 – Select Fields
Step 2 – Define Layout
Step 3 - Select Graph Style
Step 4 - Finish

For more information, see "Using Wizards" and "Running a Query" in Chapter 4, Retrieving Data, both in Chapter 1, Getting Started, in the User's Guide.
