Coverage Profiler Tool Buttons

See Also

Contains the following Tool buttons:

Button Description
Open Displays the Open dialog box so you can open a coverage log.
Save Displays the Save As dialog box so you can save the Coverage Profiler application information as a table (.dbf).
Statistics Displays the Coverage Profiler Statistics dialog box so you can view summaries of the Coverage Profiler details and review Coverage Profiler statistics relative to a specific Visual FoxPro project.
Add-Ins Displays the Coverage Profiler Add-Ins dialog box so you can specify Add-Ins to use with the Coverage Profiler.
Options Displays the Coverage Profiler Options dialog box to specify Coverage Profiler settings.
Coverage Mode Displays coverage results in the lower pane of the Coverage Profiler window or in the Zoom window.
Profile Mode Displays profiler results in the lower pane of the Coverage Profiler window or in the Zoom window.
Preview Mode Displays the Coverage Profiler window as a single dialog box of two panes.
Zoom Mode Displays the Coverage Profiler as two separate dialog boxes that you can arrange and resize.
