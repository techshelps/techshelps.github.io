Background Shortcut Menu

Provides folder-specific and general functionality to the Object pane on the right. The menu is not available in the Catalog pane on the left; clicking the Catalog pane background displays the Component Gallery Properties dialog box for the selected catalog or folder. The menu provides the following functions when you right-click a Component Gallery folder:

View

Displays standard submenu to control the display format of gallery items in the Catalog pane.

Paste

Pastes clipboard folder or item in the Gallery clipboard and selects it. This menu item is disabled until an item is copied to the clipboard. If the folder or item already exists in the folder , the new folder is named "Copy of " + folder/item name. This menu item is disabled when used in a dynamic folder.

Paste Link

Pastes a link of the folder/Item in the Gallery clipboard and selects the new folder/Item in the form "Link to" + name. This menu item is disabled when used within a dynamic folder.

New Item

Displays a submenu of choices for New items in current folder. This menu item is disabled in a dynamic folder.

Properties

Displays the properties dialog for the current catalog or folder.
