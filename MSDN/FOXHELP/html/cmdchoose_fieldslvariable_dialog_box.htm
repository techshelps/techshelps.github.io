Choose Field/Variable Dialog Box

See Also

Specifies a table field or variable.

Dialog Box Options

Choose a Field or Variable

Enter the desired field or variable.

Fields

Displays fields in the table specified in the Field box of the Report Picture dialog box. Double-click the desired field to move the field to the Choose a Field or Variable box.

Variables

Displays available variables. Double-click the desired variable to move it to the Choose a Field or Variable text box.

Comment

Allows you to add comments. Comments are entirely for your reference and do not appear in the printed report or label.
