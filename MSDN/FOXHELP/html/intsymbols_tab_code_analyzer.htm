Symbols Tab, Code Analyzer

See Also

This tab displays the symbols available in your documented code and provides a way to navigate through the code and between symbols. The tab contains the following controls.

Tab Options

Goto

 Opens a text window or moves the cursor to the first occurrence of the symbol selected in the Files grid.

Next

 Moves the text window cursor to the next occurrence of the symbol selected in the Files grid.

Symbols

 Displays all the symbols available in the documented file.

Files

 Displays the procedures and files in which the selected symbol occurs.
