SET MACKEY Command

See Also

Specifies a key or key combination that displays the Macro Key Definition dialog box.

Syntax

SET MACKEY TO [KeyLabelName]

Arguments

KeyLabelName

Specifies the key or key combination that displays the Macro Key Definition dialog box. See ON KEY LABEL for the key labels to use.

Remarks

Use SET MACKEY to change the default key combination for displaying the Macro Key Definition dialog box. Choose Macros from the Tools menu to display this dialog box.
