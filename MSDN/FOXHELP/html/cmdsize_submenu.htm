Size Submenu

See Also

Changes control size. You can change control sizes individually or resize a group of controls to match each other. You can resize any report control except labels.

These commands are available when you work with a report, label, form, or class, and create and select a control.

Submenu Commands

The buttons for the Size commands are located on the Layout toolbar.

To Fit

Sizes the object to fit its contents. Available only when you are working with forms.

To Grid

Sizes the object in grid increments when you drag it if the Snap to Grid command is checked on the Format menu.

To Tallest

Sizes all selected objects to the height of the tallest selected object.

To Shortest

CTRL+ Sizes all selected objects to the height of the shortest selected object.

To Widest

Sizes all selected objects to the width of the widest selected object.

To Narrowest

CTRL+ Sizes all selected objects to the width of the narrowest selected object.

For more information, see "Resizing Controls" in Chapter 7, Designing Reports and Labels in the User's Guide.
