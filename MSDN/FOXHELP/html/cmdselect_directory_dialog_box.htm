Select Directory Dialog Box

See Also

Allows you to set the home directory of the project. This dialog box appears when you choose the dialog button on the Project tab of the Project Information dialog box. The dialog button is associated with the Home option.

Dialog Box Options

Directory

Displays a directory tree, where you click on the directory you want to set as the project home directory.

Drive

Allows you to select a drive so you can choose a directory on that drive.

Select

Makes the selected drive/directory combination the project home directory. This directory is then displayed in the Home box of the Project tab.
