Create Dialog Box

Appears when you select Project, Database, or Table in the New dialog box and then choose New File. Allows you to specify the directory, name, and type of the new file.

Dialog Box Options

Code Page

Displays the Code Page dialog box so that you can specify the code page needed to view the file correctly. This command applies only to text files such as program (.prg), query (.qpr), and text (.txt) files.

For more information, see "Using Designers" and "Using Wizards" in Chapter 1, Getting Started in the User's Guide.
