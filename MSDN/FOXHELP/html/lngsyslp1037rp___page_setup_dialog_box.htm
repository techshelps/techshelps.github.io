SYS(1037) - Page Setup Dialog Box

See Also

Displays the Page Setup dialog box

Syntax

SYS(1037)

Returns

Character

Remarks

SYS(1037) displays the Page Setup dialog box, allowing you to make adjustments to printer settings such as the paper size and orientation. The settings you can adjust in this dialog depend on the printer you've installed.

SYS(1037) always returns the empty string.
