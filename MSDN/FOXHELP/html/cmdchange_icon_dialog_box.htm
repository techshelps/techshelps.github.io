Change Icon Dialog Box

See Also

Specifies an icon to be used for a linked or embedded object on a form or in a General field of a Visual FoxPro table. This dialog box is displayed when you select Display as Icon in the Insert Object dialog box or the Paste Special dialog box and choose Change Icon.

Dialog Box Options

Icon

  • Current   Specifies that the current icon is to be used.

  • Default   Specifies that the default icon is to be used.

  • From File   Specifies the file from which the new icon is to be obtained. Type the path and file name in the box, and then choose OK to show all available icons. Scroll through the icons, select the icon you want to use, and then choose OK.

Label

Specifies the label that will be displayed beneath the icon. Type the desired label in the box.

Browse

Displays the Browse dialog box so you can select a file containing the icon you want.
