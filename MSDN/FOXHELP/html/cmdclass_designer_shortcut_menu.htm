Class Designer Shortcut Menu

See Also

Contains shortcuts to several commands that are useful in the Class Designer. These commands appear when you right-click the Class Designer.

Note   These shortcut menu options are context sensitive. Not all of the following options appear each time you open the shortcut menu.

Shortcut Menu Options

Cut

Removes selected text or object to the Clipboard. Once you cut something, you can paste it elsewhere in the current application or in other application. This command corresponds to the Cut command in the Edit menu.

Copy

Duplicates selected text or object and puts it on the Clipboard. Once you copy something to the Clipboard, you can paste it elsewhere in the current application or in other applications. This command corresponds to the Copy command in the Edit menu.

Paste

Places from the Clipboard cut or copied sections of text at the insertion point. This command corresponds to the Paste command in the Edit menu.

Undo

Reverses the last editing change. When you save a file, the Undo command is disabled until you change the file in some way. The Undo command reverses keystrokes, cut, paste, or clear actions, moving of text, indent or outdent, and replacement.

Properties

Displays the Properties window. This command corresponds to the Properties command in the View menu.

Builder

Opens the appropriate builder for the selected control. Several different Builders can appear depending on the control selected in the Class Designer. If you select an item in the Class Designer which does not have a Builder associated with it, you will receive a dialog box message stating that no builders of that type are registered. This table lists the types of builders available in the Class Designer.

Builder Name Item Select on Form
Form Builder Nothing
Auto Format Builder* Multiple Controls
Edit Box Builder Edit Box Control
Command Group Builder Command Group Control
Text Box Builder Text Box Control
Combo Box Builder Combo Box Control
Option Group Builder Option Group Control
List Box Builder List Box Control
Grid Builder Grid Control

* The AutoFormat Builder corresponds to the Builder Lock button on the Form Controls toolbar.

Code

Displays the Code window. Use the Code window to write, display, and edit form, event, and method code.

Help

Displays Help on the selected item.
