DynamicFontName Property

Applies To   See Also

Specifies the name of the font used to display text in a Column object. The logical expression is re-evaluated at run time each time the Grid control is refreshed. Available at design time; read-write at run time.

Syntax

Column.DynamicFontName[ = cName]

Settings

cName

Evaluates to the name of a font.

Remarks

At design time, a list of available fonts is displayed when you select the FontName property in the Properties window and click the down arrow to the right of the Properties Settings box.
