Setup Wizard

See Also

The Setup Wizard creates distribution disks from the files in your distribution tree. To learn about the distribution process and how to prepare your distribution tree before running the Setup Wizard, see Chapter 26, Creating Distribution Disks, in the Programmer's Guide.

For general information about wizards, see "Using Wizards" in Chapter 1, Getting Started, in the User's Guide.

To run the Setup Wizard

  • From the Wizards submenu on the Tools menu, choose Setup.
