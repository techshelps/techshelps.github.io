API library revision mismatch.  Rebuild library (Error 1711)

The API library you are using does not match your version of Visual FoxPro.

  • You are using an older version of an .FLL file.

    Rebuild the .FLL file, or obtain an updated version.

  • The ActiveX control calling the Visual FoxPro API is out of date.

    Rebuild the ActiveX control or obtain an updated version.

For more information, see Chapter 28, Accessing the Visual FoxPro API, in the Programmer’s Guide.
