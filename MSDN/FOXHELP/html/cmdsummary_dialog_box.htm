Summary Dialog Box

See Also

Sets height and other options for the Summary band in a report or label. This dialog box appears when you double-click the Summary band separator bar in the Report Designer. To see the Summary band, you must select Summary band in the Title/Summary dialog box.

Dialog Box Options

Height

Specifies band height. Range is from 0.2083 to 20.00 inches. Type a value in the box, click the UP ARROW or DOWN ARROW key to change the displayed value in increments of .05, or drag the band's separator bar with the mouse.

Constant Band Height

Prevents the band from stretching to accommodate lengthy data or shrinking to absorb blank lines that have been removed. This option cannot be unselected in the Summary dialog box.

Run Expression

On Entry

Specifies that an expression be evaluated before each new group header is printed. Type the expression in the box, or choose the dialog button to display the Expression Builder dialog box.

On Exit

Specifies that an expression be evaluated after each new page is printed. Type the expression in the box, or choose the dialog button to display the Expression Builder dialog box.

For more information, see "Modifying Your Layout" and "Customizing Your Layout" in Chapter 7, Designing Reports and Labels in the User's Guide and "Refining Page Layout" in Chapter 12, Adding Queries and Reports in the Programmer's Guide.
