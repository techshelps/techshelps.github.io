Report and Label Print Options Dialog Box

See Also

This dialog box appears when you click Options in the Print Options dialog box.

Dialog Box Options

Scope

Allows you specify how much to print. Choose All, Next, Record, or Rest from the drop-down list. Choose a number from the spinner on the right when you select Next or Record from the Scope drop-down list.

For

Displays the Expression Builder, in which you create a logical expression which every record must meet to be printed.

While

Displays the Expression Builder, in which you create a logical expression that must be true in order for records to print.

For more information, see Chapter 7, Designing Reports and Labels in the User's Guide.
