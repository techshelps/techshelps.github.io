Color Palette Toolbar

See Also

Use the Color Palette toolbar to specify colors for a control on a form or report. For information about displaying and manipulating toolbars, see "Using Toolbars" in Chapter 1, Getting Started, in the User's Guide.

This toolbar contains the following buttons.

Button Description
Foreground Color Sets the default foreground color for controls.
Background Color Sets the default background color for controls.
Other Colors Displays the Windows Color dialog box, where you can define custom colors.

For more information, see "Using Toolbars" in Chapter 1, Getting Started, "Changing Control Colors" in Chapter 7, Designing Reports and Labels, and "Customizing Forms" in Chapter 8, Managing Data Through Forms, all in the User's Guide.
