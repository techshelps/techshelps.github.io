Debug Output Window Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Debug Output window of the Debugger. These commands appear when you right-click the Debug Output window.

Shortcut Menu Options

Save As

Displays the Save As dialog box, which allows you to save the contents of the Debug Output window.

Clear

Clears the contents of the Debug Output window.

Docking View

Allows you to specify whether the docking feature for the window is on or off.

Hide

Hides the Debug Output window.

Font

Displays the Font dialog box where you can set the font type, style, and size. This command corresponds to the Font command in the Format menu.

Help

Displays Help on the selected item.
