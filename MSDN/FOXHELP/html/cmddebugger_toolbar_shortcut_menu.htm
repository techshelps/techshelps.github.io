Debugger Toolbar Shortcut Menu

See Also

Contains shortcuts to several commands useful in the Debugger window or FoxPro Frame. These commands appear when you right-click the Debugger Window toolbar or the Debugger frame.

Shortcut Menu Options

Trace

Displays or hides the Trace window.

Watch

Displays or hides the Watch window.

Locals

Displays or hides the Locals window.

Call Stack

Displays or hides the Call Stack window.

Output

Displays or hides the Debug Output window.

Menu

Displays or hides the Visual FoxPro Debugger window menu.

Toolbar

Displays or hides the Visual FoxPro Debugger toolbar.

Status

Displays or hides the Visual FoxPro Debugger status bar.
