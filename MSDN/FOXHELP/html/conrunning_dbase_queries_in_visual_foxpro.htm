Running dBASE Queries in Visual FoxPro

See Also

A dBASE query is a file with a .qbe extension that consists primarily of the dBASE IV commands required to execute the query. You can run a dBASE query (.qbe) in Visual FoxPro without making changes.

dBASE queries are brought into the Project Manager as programs under the Code tab; you cannot modify them in the Query Designer.

To run a dBASE query (.qbe)

  • In the Command window, type:

    DO <query name>.QBE

     – Or – 

  • In the Project Manager, choose Code, select the name of the query, and then choose Run.
