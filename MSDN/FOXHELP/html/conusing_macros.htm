Using Macros

See Also

If you find yourself using the same keystrokes over and over, or if you would like to automate some keystroke tasks for your users, you can record and save these keystrokes in macros by using the Macros dialog box. This section covers the following topics:

  • Recording macros

  • Editing macros

  • Clearing macro definitions

  • Creating, saving, and restoring macro sets

  • Setting a default macro set

Note To use macros, any keyboard you use must have function keys F1 to F9, and a CTRL key or an ALT key.

To access the Macros dialog box

  • From the Tools menu, choose Macros.
