Print Preview Toolbar

See Also

Changes the previewed page and zooms the view in or out. For information about displaying and manipulating toolbars, see "Using Toolbars" in Chapter 1, Getting Started, in the User's Guide.

This toolbar contains the following buttons.

Button Description
First Page Displays the first page.
Previous Page Shows how the previous page of the report will be printed.
Page Number Displays the Go To Page dialog box.
Next Page Shows how the next page of the report will be printed.
Last Page Shows how the last page of the report will be printed.
Zoom Scales the text in the preview window by 75%, 50%, 25%, or 10% of its actual size.
Close Preview Closes Print Preview window.

For more information, see "Using Toolbars" in Chapter 1, Getting Started, and "Previewing Your Results" in Chapter 7, Designing Reports and Labels, both in the User's Guide.
