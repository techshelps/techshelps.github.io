Standard Tab, Component Gallery Options Dialog Box

Enables you to specify default behavior settings that are global for all catalogs and folders, or items. The following settings are available in this tab:

Tab Options

Global Defaults

Enable item renaming

Enables select-delay label editing within the Component Gallery panes. Renaming is always available from the shortcut menu.

FFC Builder Lock

Enables you to specify that a builder opens when you drop a class on a form.

Drag and drop to desktop

Enables gallery items to open as samples in the Visual FoxPro desktop window.

Advanced editing enabled

Enables the Type, Scripts, Views, and Comments advanced editing tabs of all Item Properties dialog boxes.

Catalog default behavior

Open file as new view

Changes default double-click behavior of gallery items from Modify to Run.

Add file to current view

Changes default double-click behavior of gallery items from Open to Add.

Item default behavior

Modify item file

Sets the default double-click behavior of gallery items to Modify.

Run item file

Sets the default double-click behavior of gallery items to Run.
