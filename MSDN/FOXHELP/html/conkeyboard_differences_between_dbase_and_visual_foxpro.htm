Keyboard Differences Between dBASE and Visual FoxPro

See Also

When a Browse window is active in dBASE, the F1 through F10 function keys provide different functionality. In Visual FoxPro, the function keys always provide the same functionality.

The first table below lists the standard dBASE and Visual FoxPro actions for each function key.

The second table lists the dBASE actions for the function keys when a Browse window is active. These actions are not supported in Visual FoxPro.

Visual FoxPro and dBASE Function Key Assignments

Key Action
F1 Activate online Help
F2 Set (displays the View window)
F3 List
F4 Dir(ectory)
F5 Display Structure
F6 Display Status
F7 Display Memory
F8 Display
F9 Append
F10 Activate/deactivate the system menu bar (same as Alt)

dBASE Browse Window Function Key Assignments
(Not supported in Visual FoxPro)

Key Action
F1 Activate online Help
F2 Toggle between Browse and Edit mode
F3 Previous field
F4 Next field
F5 No effect
F6 Extend select (select text in a memo field)
F7 Move (move selected text in a memo field)
F8 Copy (copy selected text in a memo field)
F9 Zoom memo field window to full screen
F10 Activate the Browse menu bar
DOWN ARROW Activate the append record prompt
