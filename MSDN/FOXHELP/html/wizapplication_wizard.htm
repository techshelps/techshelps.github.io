Application Wizard (5.0)

See Also

Creates a fully functional application including a project file, tables, reports, forms, and a complete project-specific class library. Alternatively, creates an application framework that includes only the project file and a starter class library subclassed from the Visual FoxPro base classes, ready for you to populate with new or existing tables and documents.

After running the Application Wizard, you can modify and customize the application or framework as required for your application. For details, see the Application Wizard Object Members, Application Wizard Object Methods, and Application Wizard Object Properties topics.

To run the Application Wizard (5.0)

  1. From the Wizards menu on the Tools menu, choose Application.

    You can also run the Application Wizard (5.0) by double-clicking the New Application item in the Component Gallery.
