Include File Dialog Box

See Also

Allows you to specify an include file containing predefined compile-time constants for a user-defined class, form, or form set.

This dialog box appears when you select Include File from the Class or Form menus.

Dialog Box Options

Include File

Specifies a header file of predefined compile-time constants for a user-defined class, form, or form set. Type the name of the include file in the box, or use the dialog button to select an existing include file.
