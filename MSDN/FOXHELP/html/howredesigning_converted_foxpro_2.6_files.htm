Redesigning Converted FoxPro 2.6 Files

See Also

You can make additional changes to your converted files to take full advantage of Visual FoxPro features. To redesign your application, review the following features:

  • New commands and functions, base classes, properties, events, and methods.

  • Enhanced multiuser support.

  • Better connectivity.

When you know which features you want to change, see Differences Between Visual FoxPro and Earlier Versions for information on how FoxPro 2.6 features map to Visual FoxPro features.

For additional information, see Chapter 6, Creating Databases, as well as the sections “Using Data Sessions” and “Using Null Values” in Chapter 7, Working with Tables, and “Using Transactions” in Chapter 22, Optimizing Client/Server Performance, all in the Programmer's Guide.
