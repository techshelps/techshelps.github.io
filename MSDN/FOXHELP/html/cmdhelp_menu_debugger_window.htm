Help Menu, Debugger Window

Contains options that open Visual FoxPro online Help, direct you to technical assistance, and display information about your computer's configuration.

Menu Commands

Contents

Displays the online Help table of contents.

Search For Help On

Displays a dialog box so that you can search for Help topics by keyword.

Technical Support

Displays information about Microsoft Product Support Services and common questions about Visual FoxPro.

Office Compatible

Displays information about Visual FoxPro compatibility with Microsoft Office.

About Microsoft Visual FoxPro

Displays information about Visual FoxPro and your system.
