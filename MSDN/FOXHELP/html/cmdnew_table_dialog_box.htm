New Table Dialog Box

See Also

Lets you create a new table in the database using the Table Designer or the Table Wizard.

The New Table dialog box only displays the Table Wizard option when you have checked Prompt for Wizards in the Projects tab of the Options dialog box.

This dialog box appears when you select New Table from the Database menu.

Dialog Box Options

Table Wizard

Opens the Table Wizard, which creates a table for you. The wizard provides sample fields that make it easy to create a table structure. Available only if you check Prompt for Wizards in the Projects tab of the Options dialog box.

New Table

Opens the Table Designer so you can create a table in the database.

For more information, see "Adding Tables to a Database" in Chapter 3, Collecting Tables into a Database in the User's Guide.
