Scope Dialog Box

See Also

Specifies which records Visual FoxPro acts on in the current table or view. This dialog box appears when you choose Scope from the Export Options dialog box.

Dialog Box Options

All

The action affects all records in the table or view.

Next

The action affects a range of records beginning with the current record and continuing for the number of records you specify. Type a value of 1 to act on the current record.

Record

The action affects only the specified record number.

Rest

The action affects a range of records beginning with the current record and ending with the last record in the table or view.
