Application Wizard (5.0): Step 2 – Choose Database

See Also

Allows you to select an existing database or create a new database using a database template for the application. This step appears only when you choose Complete application in Step 1.

If you select an existing database, the database specified will be included in the final Project file. If you select a database template, the database will contain sample data, in addition to forms and reports appropriate for that database. Using a database template will allow you to create a fully functional database application that is ready to use immediately.
