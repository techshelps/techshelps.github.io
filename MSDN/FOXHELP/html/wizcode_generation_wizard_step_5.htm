Code Generation Wizard: Step 5 – Finish

See Also

Allows you to import the model and immediately use it in Visual FoxPro. You can optionally view the class library in the Class Browser.

Also allows you to update the existing Visual Modeler model from changes made in this wizard. Updates are made to naming changes but not to inheritance changes.

You can create a log file that contains a listing of all the changes you made in the wizard session. The log file has the same name as your modeler file, but a .log extension. The file is saved in the folder that contains your connection wizard application.
