Microsoft Office Compatible 97

Visual FoxPro is a Microsoft Office Compatible 97 product, which means that many of its basic features (including toolbars, menus, and accelerator keys) are similar to those used by Microsoft Office. If you are already using Office or an Office Compatible 97 product, then you will see that many tasks can be completed in a similar manner in Visual FoxPro. These similarities will make it easier for you to use Office Compatible 97 products together.

Look for the Microsoft Office Compatible 97 logo when purchasing software. For more information about the Microsoft Office Compatible 97 program, and for a complete listing of Microsoft Office Compatible 97 products, please see our Web site at http://www.microsoft.com/msoffice/ofccomp or call Microsoft Customer Service at 1-800-426-9400. Customers outside the United States should contact their local Microsoft office.

Office Compatible Features Supported by Visual FoxPro

Visual FoxPro includes the following Office Compatible features:

  • Windows NT and Windows 95 logo design features set.

  • Main menu bar and accelerator keys.

  • Drop-down menus with accelerator keys.

  • Pop-up menus.

  • Basic dialog boxes, including Open, Save As, and Print.

  • HTML support through the WWW Search Page Wizard.

  • Standard toolbar and toolbar configuration dialog boxes.

  • Office Compatible Help Topic.

Using Visual FoxPro with Microsoft Office

You can integrate Visual FoxPro for Windows with other Office applications in many ways. The following list describes some of the ways you can use Visual FoxPro:

  • From Visual FoxPro, launch Microsoft Excel and open a spreadsheet using Automation. You can do this using the Visual FoxPro interface or programmatically using CREATEOBJECT( ) or GETOBJECT( ) functions.

  • Embed a Microsoft Word document in an OLE container control and place it on a form.

  • Drag selected text from a Microsoft Word document and drop it in a RichEditBox control.

Note   Other Microsoft Office products use CTRL+Y for redo, but in Visual FoxPro use CTRL+R for redo, and for backward compatibility with earlier versions of FoxPro, use CTRL+Y to add a record to a Browse window.
