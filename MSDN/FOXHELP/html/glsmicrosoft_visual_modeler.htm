Microsoft Visual Modeler

The Microsoft Visual Modeler is an object modeler that provides a graphical user interface for constructing and modifying the contents of an object model. Visual Modeler allows you to view the structure of an object model from several levels of abstraction, to change the properties of components in the model, and to create new object models. Visual Modeler is available in Microsoft Visual Studio - Enterprise Edition.

The Visual FoxPro provides two new connection wizards that enable you to import and export visual modeling information. They are the Code Generation Wizard and the Reverse Engineering Wizard. For details see Connection Wizards Overview.
