Class Menu

The Class menu contains options that allow you to create and edit properties and methods, specify include files, and control general class attributes.

Class Menu Commands

New Property

Creates a new property. Displays the New Property dialog box.

New Method

Creates a new method. Displays the New Method dialog box.

Edit Property/Method

Opens the Edit Property/Method dialog box where you can edit an existing property or method. Enabled when you add a new property or method.

Include File

Lets you specify a file that contains compiler directives. Displays the Include File dialog box.

Class Info

Displays the Class Info dialog box, which shows all properties and methods of class members, their design-time appearance, and whether or not they are protected.
