Management Information Format

To add or maintain items in a Microsoft Systems Management Server (SMS) database, the SMS system uses data in the Management Information Format (MIF). MIF files are ASCII text files that allow you to add objects with new architectures (for example, printers or hardware routers) to the SMS database; add new objects (for example, a new computer to the Personal Computer architecture) with existing architectures; and add new groups and attributes to existing objects.

You can also extend the SMS Administrator to display custom graphics for custom objects and groups in its Sites window and Properties window. If you want the new object to appear in the Sites window of the SMS Administrator, you must define at least seven specific attributes in the Identification group.
