Expanded Tool Functionality

Microsoft SQL Server version 6.5 provides enhancements for administration tools such as SQL Enterprise Manager. Enhancements to SQL Enterprise Manager include a Database Maintenance Plan Wizard, a built-in database transfer management interface for copying objects between databases, support for enhancements such as table dump and load and point-in-time recovery, enhanced menu options, and graph controls. Additionally, external applications can be added to the Enterprise Manager's menu and executed.

SQL Server 6.5 also provides new tools, including:

  • SQLTrace, a graphical utility for monitoring and recording database activity.
  • SQL Server Web Assistant for HTML publishing using SQL Server data.
  • Microsoft Query for querying tables and viewing data.
  • Microsoft Distributed Transaction Coordinator for administering distributed transactions.
