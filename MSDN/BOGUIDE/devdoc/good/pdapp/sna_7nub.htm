Requirements for Microsoft SNA Server Applications

To develop server applications for SNA Server, the following hardware is required:

  • Minimum of 20 MB of total system RAM. This value is 4 MB more than the required minimum for Windows NT Server. It is highly recommended that the server be equipped with enough RAM for optimal performance and for situations where loads are heavy.
  • Minimum of 11 MB free hard-disk space.
  • Network adapter.
  • Supported SNA communications adapter(s).

In addition to the required hardware, the following software is required:

  • Microsoft Windows NT Server version 3.5 or later.
  • Network software and protocols, including one or more of the following: Microsoft networking (the network software included in Windows NT), NWLink or equivalent IPX/SPX services, Microsoft TCP/IP or the equivalent Microsoft Win32 Windows Sockets version 1.1 interface, Banyan VINES Enterprise Client version 5.52 or later for Microsoft Windows NT, and services for the Apple Macintosh.
