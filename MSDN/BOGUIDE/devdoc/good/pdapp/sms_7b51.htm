Desktop Management Interface

In general, the Microsoft Systems Management Server (SMS) system can use any Management Information Format (MIF) file that complies with version 4.5 of the Desktop Management Interface (DMI) Specification. However, SMS treats some aspects of the MIF syntax in a unique way. Although the full DMI 4.5 syntax is accepted by the SMS MIF parser, some fields are not used, and the SMS system may use some of the information in a way that differs from the DMI 4.5 Specification. For example, the DMI 4.5 Specification permits only one component per MIF file whereas SMS supports multiple components.
