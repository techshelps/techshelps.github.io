Proxy Server

Microsoft® Proxy Server tightly integrates with the Windows NT Server platform and Internet Information Server to deliver high-performance Internet access to every desktop in your organization in an easy, secure, and cost-effective way. By sharing a single, secure gateway, Proxy Server eliminates the need to share one dedicated machine for the Internet among multiple users, or to run multiple Internet lines into the organization to provide each desktop with a separate connection.

Proxy Server supports a wide range of applications and protocols, including those with live, streaming audio and video (such as RealAudio and VDOLive), LDAP, IRC, SMTP, and Microsoft SQL Server, providing transparent access from any desktop. Its caching capability stores frequently accessed Internet data on your LAN where users can access it as quickly as other information on the local network. The Auto Dial tool in Proxy Server automatically connects to your Internet service provider (ISP) whenever a user needs information not already stored in the local cache.

Proxy Server offers many features including:

  • Integrates easily with existing networks.
  • Supports both TCP/IP and IPX/SPX protocols.
  • Integrates tightly with the Windows NT Server networking, security, and administrative interface.
  • Supports open standards for broad compatibility.
  • Complies fully with the CERN-proxy standard.
  • Supports the Secure Sockets Layer (SSL) for secure data communication.
  • Includes WinSock Proxy.
  • Delivers high-performance access to the Internet and your intranet.
  • Provides secure, managed access.
  • Integrates tightly with Windows NT Server user authentication.
