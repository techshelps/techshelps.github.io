Important Windows CPI-C Calls

Several CPI-C calls and Windows enhancements are of particular importance to developers and should be reviewed before using Microsoft SNA Server. These calls and enhancements include:

  • Set_Processing_Mode (cmspm)
  • Specify_Windows_Handle (xchwnd)
  • Wait_For_Conversation (cmwait)
  • WinCPICCleanup
  • WinCPICExtractEvent
  • WinCPICIsBlocking
  • WinCPICSetBlockingHook
  • WinCPICSetEvent
  • WinCPICStartup
  • WinCPICUnhookBlockingHook

SNA Server generally permits one outstanding Windows SNA asynchronous call per connection and one blocking verb per thread. The exception is when an asynchronous Receive call is issued.
