SQL Distributed Management Objects

SQL Distributed Management Objects (SQL-DMO) is a set of 32-bit Component Object Model (COM) objects for the Microsoft® Windows® 95 and Windows NT® operating systems. The SQL-DMO object model includes objects, properties, methods, and collections that are used to write programs that can administer multiple SQL Servers distributed across a network. SQL-DMO objects are Automation compatible and can range from simple Microsoft® Visual Basic® scripts to complex Microsoft® Visual C++® applications.
