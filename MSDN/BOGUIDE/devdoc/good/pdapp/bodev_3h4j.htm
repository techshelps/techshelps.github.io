BackOffice Server Components

The Microsoft® BackOffice® family of server software includes the following BackOffice Server components:

  • Microsoft Windows NT Server—a multipurpose network operating system that delivers broad application server and networking capabilities within a secure environment.
  • Microsoft Internet Information Server—a complete FTP, Gopher, and World Wide Web information service specifically designed to run on Microsoft Windows NT and within the BackOffice environment.
  • Microsoft Index Server—works with Windows NT Server version 4.0 and Internet Information Server version 2.0 to provide an organization access to all of the documents stored on its intranet or Internet site.
  • Microsoft Proxy Server—the easy and secure way to bring Internet access to every desktop in an organization.
  • Microsoft Exchange Server—a complete messaging and workgroup software solution.
  • Microsoft SQL Server—a high-end database platform designed to be the center of a complete information management system.
  • Microsoft Systems Management Server (SMS)—the most comprehensive solution for managing networked personal computers.
  • Microsoft SNA Server—the easiest system to connect personal computers to IBM mainframes and AS/400 mid-range systems.

The components of BackOffice Server were designed to work together and, consequently, share a solidly defined and supported, truly open architecture: Microsoft's widely-endorsed Windows Open Services Architecture (WOSA).
