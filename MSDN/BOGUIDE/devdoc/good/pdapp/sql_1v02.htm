System Requirements and Recommendations for SQL Server

To develop applications for Microsoft® SQL Server, the following hardware is required:

  • Intel 80486/25 or higher processor; Intel Pentium or Pentium Pro processor; RISC-based microprocessors such as MIPS R4000 and Digital Alpha AXP; or PowerPC
  • At least 60 MB of available disk space
  • CD-ROM drive
  • 16 MB RAM required (32 MB RAM or more is recommended)
  • Network adapter card and software

In addition to the required hardware, the following software is required:

  • Microsoft® Windows NT® version 3.5 or later (Windows NT version 3.51 or later is recommended)
  • Microsoft SQL Server version 6.0 or later
  • Microsoft Platform Software Development Kit (SDK)
