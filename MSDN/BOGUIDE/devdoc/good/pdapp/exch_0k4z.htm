Administrative Functions

Microsoft Exchange Server administrative functions are designed to allow you to administer, monitor, and optimize a gateway's performance using minimal system resources. The three administrative functions your gateway can perform are:

  • Error handling. Gateways can generate or store detailed error information.
  • Message tracking. Event tracking traces the progress of messages traveling through the gateway to Microsoft Exchange Server or to the foreign mail system.
  • Performance monitoring. Gateways can log performance statistics to the Windows NT Performance Monitor.

Microsoft also provides you with the ability to configure gateways. Gateway setup programs allow users to install, configure, and remove gateways within the Microsoft Exchange Server environment.

For more information about gateway programming, see "Gateway Programmer's Reference" in the "Database and Messaging Services" section of the Microsoft Platform SDK.
