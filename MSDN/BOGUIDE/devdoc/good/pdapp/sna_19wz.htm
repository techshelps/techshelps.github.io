Developing for the Microsoft Windows 3.x Environments

To develop client applications for SNA Server in a Windows version 3.x environment, the following hardware is required:

  • Minimum of 4 MB of total system RAM.
  • Minimum of 4 MB free hard-disk space.
  • Network adapter.

In addition to the required hardware, the following software is required:

  • Microsoft Windows for Workgroups or Windows version 3.x.
  • Networking software and protocols, including one or more of the following: the networking software included in Windows for Workgroups; Microsoft® LAN Manager version 2.x; Novell NetWare (IPX.COM) version 3.10 or later, or any open data-link interface (ODI) driver; Microsoft TCP/IP or the equivalent Microsoft Win32 Windows Sockets version 1.1 interface or later; and Banyan VINES version 5.52 or later.
