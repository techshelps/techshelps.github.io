Developing for the Microsoft Windows 95 Environment

To develop client applications for SNA Server in a Windows 95 environment, the following hardware is required:

  • Minimum of 6 MB of total system RAM. This is 2 MB more than the recommended minimum for Windows 95.
  • Minimum of 7 MB free hard-disk space.
  • Network adapter.

In addition to the required hardware, the following software is required:

  • Microsoft Windows 95.
  • Networking software and protocols, including one or more of the following: Microsoft networking (the network software included in Windows NT), NWLink or equivalent IPX/SPX services, Microsoft TCP/IP or equivalent Microsoft Win32 Windows Sockets version 1.1 interface, and Banyan VINES Enterprise Client version 5.52 or later for Microsoft Windows NT.
