Microsoft Index Server

Microsoft® Index Server requires Windows NT Server version 4.0 or later and Internet Information Server version 2.0 or later, in addition to the following:

  • A 32-bit Intel-based processor (such as Intel 80486/25 or higher), Intel Pentium or Pentium Pro, or supported RISC-based microprocessor such as the MIPS R4x00, Digital Alpha, or PowerPC.
  • Free disk space amounting to 40 percent of the total size of the documents to be indexed, in order to store the document indexes.
  • At least 12 MB of system memory (RAM) for Intel-based systems (16 MB recommended); at least 16 MB for RISC-based systems.
  • VGA or higher resolution monitor.
  • Optionally, one or more network adapter cards, if you want to use Windows NT Server with a network.
