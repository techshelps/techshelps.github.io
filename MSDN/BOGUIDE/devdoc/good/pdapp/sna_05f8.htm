Developing for the MS-DOS Environment

To develop client applications for SNA Server in a MS-DOS environment, the following hardware is required:

  • Minimum of 640 kilobytes (KB) of total system RAM.
  • Minimum of 2.2 MB free hard-disk space.
  • Network adapter.

In addition to the required hardware, the following software is required:

  • MS-DOS operating system version 3.2 or later.
  • Microsoft LAN Manager version 2.x, Novell NetWare (IPX.COM) version 3.10 or later, or Banyan VINES version 5.52 or later.
