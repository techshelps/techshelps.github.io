Document Conventions

This reference uses the following typographical conventions:

  • UPPERCASE text represents statement and clause names, structure names, file names, and paths.
  • Bold text indicates an interface, method, function, structure, or other keyword. Within discussions of syntax, bold type indicates that the text must be entered exactly as shown.
  • Italic text indicates a new or unfamiliar term, or a reference to a source outside of the Microsoft BackOffice Programmer’s Reference.
