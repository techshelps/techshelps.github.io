Microsoft SNA Server

Microsoft® Systems Network Architecture (SNA) Server requires at least 20 MB of system memory. It also requires a minimum of 7 MB of available hard disk space beyond the requirements for Windows NT Server. Additional memory may be required, depending on the number of users that are simultaneously running SNA Server and their level of usage.

The following table shows the recommended processor and amount of memory for a given number of users and type of usage.

Users, usage Processor RAM
Up to 25 users 486DX/33 16 MB
25–500 users, light or intermittent usage 486DX/66 32 MB
500–1500 users, light or intermittent usage Pentium or RISC 64–128 MB
1–200 users, heavy usage 486DX/66 32 MB
1–600 users, heavy usage Pentium or RISC 64–128 MB
1500–2000 users, up to 10000 sessions Symmetrical Multi-Processor (SMP) or fast RISC 64–128 MB

For more information about planning the design for an SNA Server system, see the Microsoft SNA Server Planning Guide.
