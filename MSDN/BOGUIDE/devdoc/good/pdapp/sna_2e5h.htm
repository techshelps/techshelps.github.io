3270 Emulator Interface

The Microsoft® SNA Server SDK supports the development of 3270 emulation client applications. The development of 3270 emulation clients depends on a familiarity with and understanding of SNA Server concepts and operation. Developers must also be aware of specific interface definitions used by the 3270 emulation client to communicate with SNA Server components. The following topics present these issues in greater detail.

For more information about topics not included in this overview or for a complete review of all 3270 Emulator Interface topics, see the Microsoft SNA Server documentation in the "Internet, Networking, and Distributed Services" section of the Microsoft Platform SDK.
