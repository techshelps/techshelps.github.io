Microsoft Exchange Server

In addition to the development requirements for your application, you should anticipate the run-time system requirements for Microsoft® Exchange Server version 4.0 or later. These requirements are shown in the following table.

Users Processor Disk space RAM

Note  The preceding system requirements are based on the assumption that Microsoft Exchange Server is installed and running on top of Windows NT Server. These requirements incorporate general Windows NT Server requirements but may vary depending on your specific system environment.

For more information about processor or memory requirements for Microsoft Exchange Server, see the Microsoft Exchange Server Concepts and Planning Guide.
