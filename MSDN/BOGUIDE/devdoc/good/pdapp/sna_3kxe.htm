Other CPI-C Development Issues for Microsoft SNA Server

There are several other operational and administrative tasks that CPI-C performs that are relevant to proper incorporation of CPI-C calls in your application. The following issues are important:

  • CPI-C maintains a set of internal values called characteristics for each conversation. Some characteristics affect the overall operation of the conversation, such as the conversation type. Others affect the behavior of specific calls, such as the receive type.
  • The information required for two CPI-C programs to communicate is stored in memory as a table, called the side information table. This table is maintained by the system administrator and contains entries for information such as the Partner LU Name and Security User ID and Password.
  • Design optimization and fulfillment of the requirements for developing applications in a Windows version 3.x, Windows NT Server, or OS/2 Client or Server environment.
