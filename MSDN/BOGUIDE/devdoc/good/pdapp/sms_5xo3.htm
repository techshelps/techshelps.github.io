System Requirements and Recommendations for SMS

To develop applications for Microsoft® Systems Management Server (SMS), the following hardware is recommended:

  • Intel 80486/25 or higher processor; Intel Pentium or Pentium Pro processor; RISC-based microprocessors such as MIPS R4000 and Digital Alpha AXP; or PowerPC
  • Super VGA (800 x 600) 256-color display
  • At least 100 MB space on the partition containing Windows NT Server files
  • CD-ROM drive
  • 32 MB RAM or more
  • Network adapter card

In addition to the recommended hardware, the following software is required:

  • Microsoft® Windows NT® Server version 3.5 or later
  • Systems Management Server version 1.1 or later
  • Microsoft® BackOffice™ Software Development Kit (SDK)

Applications using the SMS API are compatible only with primary site databases for SMS version 1.1 or later. Sites running SMS version 1.0 are not supported.
