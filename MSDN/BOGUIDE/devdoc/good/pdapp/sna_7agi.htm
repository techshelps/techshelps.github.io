APPC Programming and File Transfer

Advanced Program-to-Program Communications (APPC) is an application programming interface (API) that allows peer-to-peer communications in a Systems Network Architecture (SNA) environment. By using APPC, programs distributed across a network can work together—communicating and exchanging data—to accomplish a single processing task. A processing task can query a remote database, copy a remote file, or send or receive electronic mail, as well as perform other administrative tasks.
