System Requirements and Recommendations for Microsoft Exchange Server

To develop applications for Microsoft® Exchange Server, the following hardware is recommended:

  • Intel 80486/25 or higher processor; Intel Pentium or Pentium Pro processor; RISC-based microprocessors such as MIPS R4000 and Digital Alpha AXP; or PowerPC
  • Super VGA (800 x 600) 256-color display
  • At least 50 MB space on the partition containing Microsoft® Windows NT® Server files for the SDK, plus the space necessary for the Microsoft® Visual C++® version 4.0 development system, Windows NT Server, and Microsoft Exchange Server components required by your development
  • CD-ROM drive
  • 32 MB RAM or more
  • Network adapter card

In addition to the recommended hardware, the following software is required:

  • Microsoft Windows NT version 3.51 or later, with Service Pack 4
  • Microsoft Exchange Server must be running on the domain
  • Microsoft Platform SDK
  • Microsoft Exchange Client
  • Microsoft® BackOffice™ Software Development Kit (SDK)
  • Microsoft Visual C++ version 4.0; earlier versions are not supported

Sample applications may require the Microsoft Exchange Server Administrator program to be running on the development computer.
