Microsoft Proxy Server

Microsoft® Proxy Server requires Windows NT Server version 4.0 or later, Internet Information Server version 2.0 or later, and the Windows NT Server 4.0 Service Pack (provided on the Proxy Server CD-ROM). In addition, Proxy Server requires the following:

  • A 486/33-MHz or higher Intel-based processor, Intel Pentium or Pentium Pro, or supported RISC-based microprocessor compatible with Windows NT Server 4.0.
  • At least 15 MB of available hard disk space for Intel-based systems (20 MB for RISC-based systems).
  • At least 16 MB of system memory (RAM).
  • CD-ROM drive.
  • VGA, Super VGA, or video graphics adapter compatible with Windows NT Server 4.0.
