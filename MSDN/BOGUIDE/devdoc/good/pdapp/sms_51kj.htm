Registry Keys and Values

The Windows NT registry keys and values required for ISV application launching are as follows:

  • Microsoft Systems Management Server (SMS) Applications Key (HKEY_LOCAL_MACHINE or HKEY_CURRENT_USER)
  • Vendor Key, which is the full name of the vendor.
  • Application Key

    This key can contain values for the application name, application description, the name of the executable launched when the application is selected from the Tools menu, the directory from which the application will start, the state of the launched application, and the order in which the application will appear in the Tools menu.

  • Context Keys

    These keys specify whether the application is available in the General context or a specific context. An application can be registered in more than one context. Each context key contains additional values that specify when the application will be present in the Tools menu, when the application will be present but disabled, and the arguments to be passed on the command line that launches the application.
