New Features in Microsoft SQL Server Version 6.5

Microsoft® SQL Server version 6.5 builds on the foundation of performance, reliability, and scalability provided by SQL Server version 6.0. SQL Server 6.5 enhances that foundation as well as extends it into new areas such as the World Wide Web. New features in SQL Server 6.5 can be grouped into several categories:

  • Expanded tool functionality
  • Enhanced SQL Server architecture
  • Improved data processing and verification
