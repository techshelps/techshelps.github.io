Internet Information Server

Microsoft Internet Information Server (IIS), which installs and runs as a native Windows NT service, is a powerful platform for a new generation of World Wide Web applications that must interact with other BackOffice components and applications. Microsoft® Internet Information Server offers many features including:

  • Installation integrated into Windows NT Server version 4.0 installation
  • Low system management outlays
  • Scalable and portable design
  • High-performance architecture
  • High reliability
  • Integrated with Windows NT Server 4.0 directory and security services
  • Support for open development solutions
  • Access to BackOffice components
  • Integration with existing databases

Enhancements to Internet Information Server version 2.0 deliver improved programmability using the Internet Server API (ISAPI) and improved database programmability with the Internet Database Connector (IDC) for building Internet applications. In addition, support for both NCSA and CERN style image map files facilitates porting from UNIX systems.

By extending the power of Microsoft BackOffice server products to the Internet, Internet Information Server makes it easy to create powerful solutions that improve communications and deliver high-impact Web sites. IIS leverages Microsoft technology to help you develop a new generation of interactive Web-based applications.
