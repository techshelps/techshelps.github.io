Developing for the Macintosh Environment

To develop client applications for SNA Server in a Macintosh environment, the following software and hardware are required:

  • Apple Macintosh System 6.0.2 or later and Finder version 6.0 or later (System version 7.0 and Finder version 7.0 can be used).
  • Standard Macintosh platform with AppleTalk access.
  • Emulator software and any hardware or additional software required for that emulator.

For information about emulator software for the Macintosh, see the SNA Server Companion Product Catalog, ISVCATAL.DOC, in the \COLLATRL\3rdParty directory on the SNA Server compact disc.
