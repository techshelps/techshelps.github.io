Articles and References For Using Java with Native Code

The Microsoft Win32 VM for Java enables access to native code libraries and components from Java. The following articles and references show how this can be accomplished and describe the best approach based on your needs.

These articles provide overview information on the various paths you can take in mixing Java and native code.
