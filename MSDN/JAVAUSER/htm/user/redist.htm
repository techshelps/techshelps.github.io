Redistributable File List for Microsoft SDK for Java Version 3.0 Pre-Release 2

The Microsoft SDK for Java version 3.0 pre-release 2 is not redistributable; see the End-User License Agreement.

The Microsoft AFC included in this SDK, however, is redistributable. For more information about the AFC, go to Microsoft Application Foundation Classes. The JDK 1.1-based AFC class libraries are installed by default with the latest Microsoft virtual machine. The JDK 1.0.2-compatible AFC files that follow are redistributable together under the terms of the AFC License Agreement:

  • afc102.zip
  • afcsetup.exe
  • all samples under \AFC102\Samples
