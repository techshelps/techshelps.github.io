Class Direct3dRMUserVisual

public class Direct3dRMUserVisual implements 
            IDirect3dRMUserVisual
{
}

Applications use the Direct3dRMUserVisual class to manage user visual objects.

Note This class does not provide an init method. Use the Direct3dRM.createUserVisual method to create and initialize Direct3dRMUserVisual objects.

The Direct3dRMUserVisual class inherits the following methods from the Direct3dRMObject class:

addDestroyCallback
duplicate
deleteDestroyCallback
getAppData
getClassName
getName
setAppData
setName
