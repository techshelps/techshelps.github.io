About com.ms.xml.dso

This package provides a Data Source Object (DSO) applet class that can be used to provide data for data binding by using XML files.

Classes

Class XMLDSO
This class is an applet that can be used with an <APPLET> tag to load XML files and provide the data contained in these files for data binding.
