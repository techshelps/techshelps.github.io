Using the Java Language Package Manager

The Java language supports the concept of Java packages to manage the name space of classes, but does not provide for installing or managing packages. The Microsoft Win32 VM for Java (Microsoft VM) provides package management services with the Java Language Package Manager. Internet Explorer 4.0 uses the Java Language Package Manager to provide code download services for Java applets.

The following articles provide overview information on package management as well as information on how to produce your own distribution units that will install using the Java Language Package Manager services.
