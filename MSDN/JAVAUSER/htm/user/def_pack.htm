About the Microsoft Packages

The following Microsoft packages are in the Microsoft SDK for Java:

Package com.ms.activeX
Provides access to ActiveX controls.
Package com.ms.awt
Provides access to Win32® window-management and graphics features.
Package com.ms.awt.peer
Provides access to an interface supporting Win32® window-management features.
Package com.ms.com
Provides access to Component Object Model (COM) services from Java.
Package com.ms.directX
Provides access to Microsoft® DirectX™ services and features.
Package com.ms.dll
Provides additional support for Microsoft® J/Direct™, the technology that allows access to Windows® dynamic-link libraries (DLLs).
Package com.ms.fx
Provides access to graphics features of the Application Foundation Classes (AFC).
Package com.ms.io
Provides access to file dialog boxes, specialized input and output streams, and a class for manipulating path strings.
Package com.ms.io.clientstorage
Provides access to client storage management and enforcement of client storage limitations.
Package com.ms.lang
Provides access to Microsoft® Win32® system-level features.
Package com.ms.license
Provides classes that support Java object licensing files.
Package com.ms.object
Provides classes and interfaces used to facilitate data transfer.
Package com.ms.object.dragdrop
Provides classes and interfaces for implementing visual drag/drop operations.
Package com.ms.security
Provides access to classes that control the security system.
Package com.ms.security.auditing
Provides an interface for supplying information about an audited security request to a security auditor.
Package com.ms.security.permissions
Provides access to classes that represent permission and request objects that are used to perform security checks.
Package com.ms.service
Provides access to Microsoft® Windows NT® system services.
Package com.ms.ui
Provides access to user interface components of the Application Foundation Classes (AFC).
Package com.ms.ui.resource
Provides access to Win32-style resources from Java applets and applications.
Package com.ms.ui.event
Provides support for the Delegation Event model in the Application Foundation Classes (AFC).
Package com.ms.util
Provides access to version control, task manager, and timer services from Java.
Package com.ms.util.cab
Provides access to cabinet file creation and extraction classes.
Package com.ms.util.InputMethod
Provides access to Input Method Editor (IME) java components.
Package com.ms.vm
Provides a class that implements a weak reference to an object. A weak reference tracks an object without preventing it from being garbage collected.
Package com.ms.xml.dso
Provides a Data Source Object (DSO) applet class that can be used to provide data for data binding using XML files.
Package com.ms.xml.om
Provides classes that implement the Microsoft XML Object Model in Java.
Package com.ms.xml.parser
Provides the Microsoft® XML Parser, which is a validating XML parser written in Java.
Package com.ms.xml.util
Provides base classes for Microsoft XML object technology, including input and output stream objects specifically designed for XML.
