hWndMenu Property

object.hWndMenu

Sets the window handle for a voice menu. Whenever this window is the foreground window, the voice menu is automatically activated; otherwise, it is deactivated. If this property is set to NULL, the menu is global — that is, it remains active regardless of the foreground window until the application explicitly deactivates it.

object

Voice-menu object.

This property should be set before setting the Active property on the voice-menu object.

This property defaults to NULL, resulting in the voice-menu being a global voice-menu that is active regardless of which window currently has the input focus. For more information, see the description of the IVCmdMenu::Activate member function. The hWndMenu property is equivalent to the hWndListening parameter of IVCmdMenu::Activate.
