You create a voice-menu object by using the IVoiceCmd::MenuCreate member function with a pointer to a VCMDNAME structure that identifies the menu to open. MenuCreate returns a pointer to the IVCmdMenu interface on a voice-menu object for the menu.

When calling MenuCreate to create a new voice menu, you can specify whether the voice-command object stores the menu in the voice-menu database or deletes the menu when the voice-menu object is released.
