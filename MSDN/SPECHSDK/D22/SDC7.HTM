IVCmdAttributes Interface

The IVCmdAttributes interface allows an application to retrieve and set the attributes of a voice-command site. The application must call the IVoiceCmd::Register member function before it can call any of the functions in IVCmdAttributes.

A voice-command application may not need to use IVCmdAttributes functions if a voice-navigation application is installed on the user's computer. The voice-navigation application typically monitors and controls the attributes of a voice-command site.

Changing an attribute of a site affects all applications using the site. Settings are saved between uses of the site, even if the user shuts off his or her computer in the meantime.

The IVCmdAttributes interface is supported by the voice commands object. It has these member functions:
