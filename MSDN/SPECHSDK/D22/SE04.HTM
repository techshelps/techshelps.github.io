CommandSpoken Property

object.CommandSpoken

Identifies the voice command most recently recognized. An application must call the MenuCreate method before checking this property.

· Example:

CmdHeard = Vcmd.CommandSpoken

If CmdHeard > 0 Then

LCommand.ListIndex = CmdHeard - 1

End If

CmdHeard = 0

object

Voice-command object.

The voice-command object sets the CommandSpoken property to the command's unique identifier when the user speaks the command.

If the application has a mechanism to receive notifications, CommandSpoken can be ignored. For more information, see the description of the Callback property.

If there is no way to receive notifications, the application must continually poll this attribute to see if a command was spoken. The application usually polls this attribute twice per second. After the application has found a valid value, it should write a zero into the attribute to clear it for future commands.
