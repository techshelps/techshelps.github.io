IVCmdAttributes::MicrophoneSet

HRESULT MicrophoneSet(PTSTR pszMicrophone);

Sets the name of the current microphone for a voice-command site.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
VCMDERR_INVALIDMODE
VCMDERR_NOTSUPPORTED
VCMDERR_OUTOFMEM
VCMDERR_VALUEOUTOFRANGE

pszMicrophone

[in] Address of the string that contains the name of the microphone to set. If the current microphone is unknown, this parameter can be an empty string.

The microphone name for a site is saved between uses of the site, even if the user shuts down the computer in the meantime.

If a voice-navigation application is installed on the user's computer, an application may not need to set the microphone name.
