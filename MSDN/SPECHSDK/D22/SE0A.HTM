ListSet Method

object.ListSet(listName as String, numberOfElements as Long, listData as String)

Sets the phrases in a list for a voice command. A list is a named set of words or phrases. By embedding the name of a list in a command string, the user can speak any word or phrase in the list at the given point in a command string. For example, a list named "myfriends" might contain the names Linda, Tom, and Mary. If the list name were embedded in a command string such as, "Send mail to <Names>," the user could say "Send mail to Linda," "Send mail to Tom," or "Send mail to Mary."

· Example:

Dim Names

Dim szNULL as String

szNULL = Chr(0)

.

.

.

Names = "Larry" & szNULL & "Mike" & szNULL & "Gib" & szNULL & "Doug" & szNULL & "George" & szNull

Call Vmenu.ListSet("name", 5, Names)

object

Voice-menu object.

listName

Name of the list to set, such as "name" or "week day". The list name must appear in the command string for at least one command on the menu.

numberOfElements

Number of phrases (null-terminated strings) in the list specified by listData.

listData

List of null-terminated strings. Each string is a possible phrase for a voice command.

The ListSet method calls the IVCmdMenu::ListSet member function to set a list for a voice command.

Using ListSet establishes a list of phrases that can be spoken in a voice command, such as "Send mail to <name>." Typically, the list contains information that changes dynamically at run time, such as the 10 people to whom the user most recently sent electronic mail. For best results, a list should have fewer than 20 entries. Having more than 20 entries in a list can reduce the accuracy of recognition.
