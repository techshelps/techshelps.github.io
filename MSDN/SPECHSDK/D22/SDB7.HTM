Interfaces

The voice-command object supports the following interfaces.

Interface

Description

IVCmdAttributes

Controls attributes of a voice-command site, such as the audio device, the microphone, the speech-recognition mode, and whether speech-recognition is awake and enabled for the site. To get the address of the IVCmdAttributes interface, use the IVoiceCmd::QueryInterface member function with the IID_IVCmdAttributes interface identifier.

IVCmdDialogs

Displays Windows dialog boxes that allow a user to configure the speech-recognition engine. These dialog boxes are defined by the engine.

IVoiceCmd

Registers an application to use voice commands on a site, allows an application to create voice-menu objects and voice-menu enumerators, and controls recognition of voice commands.
