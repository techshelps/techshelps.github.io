Awake Property

object.Awake

Awakens or puts speech recognition to sleep for a site. Setting the Awake property to TRUE awakens speech recognition for the site; setting it to FALSE puts speech recognition to sleep.

· Example:

Vcmd.Awake = False

object

Voice-command object.

When speech recognition is asleep, it recognizes commands only from sleep menus, which are designated to be active when speech recognition is asleep and inactive otherwise. For more information, see the description of the IVCmdMenu::Activate and IVCmdAttributes::EnabledSet member functions.
