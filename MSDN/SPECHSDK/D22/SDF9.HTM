IVoiceCmd::MenuDelete

HRESULT MenuDelete(PVCMDNAME pName);

Deletes a menu from the voice-menu database. A menu cannot be deleted if it is currently open and the application is actively listening for its commands.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
VCMDERR_MENUACTIVE
VCMDERR_MENUDOESNOTEXIST
VCMDERR_MENUOPEN
VCMDERR_OUTOFMEM

pName

[in] Address of a VCMDNAME structure that identifies the menu to delete.

This function deletes the storage in the database for the menu (if it exists) and releases the voice-menu object that was created by the IVoiceCmd::MenuCreate member function. After a menu is deleted, the pointer to its IVCmdMenu interface is invalid, so it should be set to NULL.
