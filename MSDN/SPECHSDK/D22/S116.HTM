Interfaces and Member Functions

Voice-command interfaces contain functions that allow an application to implement simple speech recognition, or command and control. There are the following interfaces.

Interface

Description

IVCmdAttributes

Controls the attributes of a voice-command site.

IVCmdDialogs

Displays Windows dialog boxes that allow the user to configure the speech-recognition engine.

IVCmdEnum

Enumerates the menus in the voice-menu database.

IVCmdMenu

Allows an application to activate or deactivate a voice menu, request training for the menu, or define the commands in the menu.

IVCmdNotifySink

Used by a voice-command object to notify an application of a command or utterance, a menu activation, or a change in the current VU level on a voice-command site.

IVoiceCmd

Registers an application with a voice-command object and controls recognition of voice commands.


This section describes, in alphabetical order, the interfaces and functions that support voice commands.

In addition to the interfaces described in this section, each object supports IUnknown, which is a common interface supported by all OLE component objects. For information about IUnknown, see section, "Common Elements of the Speech Application Programming Interface."
