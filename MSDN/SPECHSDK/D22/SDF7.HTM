IVoiceCmd::CmdMimic

HRESULT CmdMimic(PVCMDNAME pMenu, PTSTR pszCommand);

Supplies a voice-aware application with the equivalent of a spoken voice command.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
VCMDERR_CANNOTMIMIC
VCMDERR_INVALIDCHAR
VCMDERR_MENUDOESNOTEXIST
VCMDERR_OUTOFMEM
VCMDERR_VALUEOUTOFRANGE

pMenu

[in] Address of a VCMDNAME structure identifying the menu that contains the command to mimic.

pszCommand

[in] Address of a string that contains the command to mimic.

CmdMimic parses the command string and eliminates white space and punctuation, and then the member function compares the result with each command on the voice menu until it finds a match. The comparison is case-insensitive, and the command string can include phrases from lists. If the string matches a command in the voice menu, it is recognized. Otherwise, the function returns an error.

An application can call CmdMimic to play back voice macros to another application, somewhat like playing back keystrokes and mouse messages in Windows.

The voice menu must be active before an application can mimic its commands.
