You create a voice-menu enumerator by using the IVoiceCmd::MenuEnum member function. MenuEnum returns a pointer to an IVCmdEnum interface on the object. When calling MenuEnum, you can enumerate only menus that are already open, only those that are active, or all menus in the database.

The voice-menu enumerator persists until all references to it are released, even if the voice-command object is released first.
