Voice-Menu Object

A voice-menu object contains interfaces that an application uses to manage its voice menus. A voice menu contains a set of commands that a speech-recognition engine can recognize and that the application can respond to. An application must create a voice-menu object to represent a voice menu; only then can the engine use the menu to recognize commands. An engine can use more than one voice menu for an application, but the application must create a separate voice-menu object for each menu.
