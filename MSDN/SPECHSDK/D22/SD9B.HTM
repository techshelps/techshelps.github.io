Voice-Menu Database

The voice-command object maintains a database of voice menus that have been created by applications. The first application to create a voice-command object causes the database to be opened. The database remains open until the last voice-command object is released.

An application can use a voice menu from the database or add new menus to the database by using voice-menu objects. It can enumerate the voice menus in the database by using a voice-menu enumerator. For more information about voice-menu objects and voice-menu enumerators, see "Voice-Menu Object" later in this section and "Voice-Menu Enumerator" later in this section.
