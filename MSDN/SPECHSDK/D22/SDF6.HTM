IVoiceCmd Interface

The IVoiceCmd interface allows an application to register with a voice-command object and to control recognition of speech. The application must call the IVoiceCmd::Register member function before it can call certain other functions, such as those in IVCmdDialogs.

The IVoiceCmd interface is supported by the voice-command object. It has these member functions:

CmdMimic

MenuCreate

MenuDelete

MenuEnum

Register
