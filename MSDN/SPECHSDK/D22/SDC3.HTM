IVCmdMenu

This interface is supported by the voice-menu object.

Member function

Description

Activate

Activates the voice menu so that its commands can be recognized.

Add

Adds one or more commands to a voice menu.

Deactivate

Deactivates the selected menu so that its commands are no longer recognized.

EnableItem

Permanently enables or disables a menu item. When a command is disabled by using EnableItem, it is not compiled into the menu.

Get

Gets information about one or more commands in a voice menu.

ListGet

Gets the phrases stored in the current list for the selected voice menu.

ListSet

Sets the phrases in a list for a voice command.

Num

Indicates the total number of commands on a voice menu.

Remove

Removes the specified commands from the voice menu.

Set

Sets information for one or more commands in a voice menu.

SetItem

Temporarily enables or disables a command on a voice menu. If a command is disabled by using SetItem, it is compiled into the menu but not recognized.

TrainMenuDlg

Displays a dialog box to train the engine for the selected menus.
