Interfaces

The multimedia audio-destination object supports the following interfaces.

Interface

Description

IAudio

Allows an audio-destination or audio-source object to manage its internal buffer and the control attributes of the audio device it represents.

IAudioDest

Sends information and data to an audio-destination object.

IAudioDestNotifySink

Notifies a text-to-speech engine of changes to the internal buffer of an audio-destination object.

IAudioMultiMediaDevice

Allows an audio-destination or audio-source object to access features specific to multimedia devices.
