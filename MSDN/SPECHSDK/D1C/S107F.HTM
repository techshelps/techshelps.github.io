IAudioFileNotifySink::Posn

HRESULT Posn(QWORD qwProcessed, QWORD qwLeft);

Sent about sixteen times a second so the application knows how much data has been processed.

qwProcessed

[in] Number of bytes that have already been read from the queue, or written to output wave files.

qwLeft

[in] Number of bytes of audio that are left in the queue to be read. The number is only valid for file audio-source objects. This will always be 0 for file audio-destination objects.
