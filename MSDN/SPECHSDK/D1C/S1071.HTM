IAudioDestNotifySink::BookMark

HRESULT BookMark(DWORD dwMarkID, BOOL fFlush);

Notifies the engine that the audio-destination device has encountered a bookmark in the audio stream.

· Return value is ignored.

dwMarkID

[in] Unique bookmark identifier.

fFlush

[in] TRUE if the bookmark notification has been sent because the engine called the IAudio::Flush member function to clear the object's internal buffer or FALSE if the object has encountered the bookmark while playing back audio data.

Bookmarks can be set by calling the IAudioDest::BookMark member function.
