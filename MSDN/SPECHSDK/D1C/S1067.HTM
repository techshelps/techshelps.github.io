IAudio::UnClaim

HRESULT UnClaim(void);

Releases the multimedia wave device.

· Returns NOERROR if successful, or one of these error values:

E_FAIL
AUDERR_NEEDWAVEFORMAT
AUDERR_NOTCLAIMED

UnClaim generates an AudioStop notification when the multimedia wave device is closed.

In the case of the multimedia audio-destination object, UnClaim releases the device only after all audio buffers have been sent. For this reason, there may be a delay between when UnClaim is called and when the device is actually closed. An application should call UnClaim only after making all calls to the IAudioDest::DataSet member function.
