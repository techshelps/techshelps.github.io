IAudioSourceIntrumented Interface

(New for 3.0)

The IAudioSourceInstrumented interface controls the instrumentation provided by the instrumented audio source object.

Member function

Description

AudioSource

Pass the IUnknown for the audio source that inputs to the instrumented version.

RegistryGet

Returns the registry key that can be used to externally control the instrumentation.

RegistrySet

Changes the registry key that can be used to externally control the instrumentation.

StateGet

Returns the current that the instrumented version is in.

StateSet

Changes the instrumentation mode.

WaveFileReadGet

Gets the wave file that is to be read from.

WaveFileReadSet

Sets the wave file that is to be read from.

WaveFileWriteGet

Gets the wave file that is to be written to.

WaveFileWriteSet

Sets the wave file that is to be written to.
