IAudioSource Interface

The IAudioSource interface gets information and data from a digital-audio source, such as a file or an audio device. When the engine gets the address of the IAudioSource interface for an audio-source object, that object already represents a particular audio source. If the audio-source object represents a multimedia wave-in device, the IAudioSource interface is already open to a particular WAVEFORMATEX structure and device.
