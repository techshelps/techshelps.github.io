IAudio::ToFileTime

HRESULT ToFileTime(PQWORD pqWord, FILETIME *pFT);

Converts a QWORD value representing the time, in bytes, to a Win32 FILETIME structure.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
AUDERR_NEEDWAVEFORMAT

pqWord

[in] Address of a variable that contains the time, in bytes, returned by an engine or by the IAudio::TotalGet or IAudio::PosnGet member function.

pFT

[out] Address of a variable that receives the Win32 FILETIME value of

pqWord.

An application can use ToFileTime to correlate real-time events with a time value returned by an engine. The value returned by the engine is an estimate that should be very accurate within a few seconds of the present, but becomes less accurate farther away from the present, due to time variations in the audio card's clock.
