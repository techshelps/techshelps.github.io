IAudioFile::Set

HRESULT Set(LPCWSTR pszFile, DWORD dwID);

Flushes any files in the queue, and sets the current file. This works for the file audio-source and the file audio-destination.

pszFile

[in] Unicode file name to read from. For the file audio-source, this must be a valid .WAV file with the same format as the speech recognition engine uses.

dwID

[in] Identifier that is passed to the application when the file is read/written, and when all of the data is finished.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
