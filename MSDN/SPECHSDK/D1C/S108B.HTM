IAudioSourceInstumented::RegistrySet

HRESULT RegistrySet(WCHAR *pszKey);

DOES NOT WORK ON WINDOWS95.

Sets the key of the registry controlling the instrumentation.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pszKey

[in] Address of a buffer that contains the key. If pszKey is an empty string, "", then the registry will not be used for instrumentation control.

See IAudioSourceInstrumented::RegistryGet.
