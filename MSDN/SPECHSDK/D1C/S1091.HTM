IAudioSourceInstumented::WaveFileWriteSet

HRESULT WaveFileWriteSet(WCHAR *pszWaveFile);

Sets the file to read from.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pszWaveFile

[in] Address of a buffer that contains the file name. If pszFileName is an empty string, "", then no file will be written to. If a file name was already set then the old file will be closed and the new one opened in its place.

See IAudioSourceInstrumented::WaveFileWriteGet.
