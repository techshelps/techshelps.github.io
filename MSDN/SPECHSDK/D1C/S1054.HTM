IAudioFile

(New for 3.0)

This interface is supported by the file audio-source and file audio-destination object.

Member function

Description

Add

Adds a file to the queue of files.

Flush

Clears the queue of files.

RealTimeGet

Returns information about how quickly audio is allowed to pass through the object.

RealTimeSet

Specified how quickly audio is allowed to pass through the object.

Register

Passes a pointer to a notification interface to the audio object so the object can notify the application when files are finished, or the current position has changed.

Set

Clears the file queue and sets a file.
