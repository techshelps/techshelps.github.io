IAudio::WaveFormatSet

HRESULT WaveFormatSet(SDATA dWFEX);

Sets the wave format for the object for subsequent audio, if this is allowed by the audio device.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
AUDERR_BADDEVICEID
AUDERR_NODRIVER
AUDERR_NOTSUPPORTED
AUDERR_WAVEDEVICEBUSY
AUDERR_WAVEFORMATNOTSUPPORTED

dWFEX

[in] SDATA structure that contains the address of the new wave format for the audio device. The pData member specifies the address of a WAVEFORMATEX structure that contains the wave format information, and the dwSize member indicates the size of the information.

WaveFormatSet specifies the kind of audio data that will be received by the audio source or sent to the audio destination. Most applications use pulse code modulation (PCM), but if the specified format is compressed or encoded audio, the data for the IAudioDest::DataSet and IAudioSource::DataGet member functions will be compressed.

If audio is currently being played, WaveFormatSet has no effect.

Some audio devices allow an application to set their wave format once, after which the device becomes locked. If an audio device does not allow an application to set its wave format, the function returns AUDERR_WAVEFORMATNOTSUPPORTED.
