Interfaces

The multimedia audio-source object supports the following interfaces.

Interface

Description

IAudio

Allows an audio-destination or audio-source object to manage its internal buffer and the control attributes of the audio device it represents.

IAudioSource

Gets information and data from an audio-source object.

IAudioSourceSusp

Pause/resume the audio.

IAudioSourceNotifySink

Notifies a speech-recognition engine of changes to the internal buffer of an audio-source object.
