IAudioSourceNotifySink::DataAvailable

HRESULT DataAvailable(DWORD dwBytes, BOOL fEOF);

Notifies the engine that the amount of data in the audio-source object's internal buffer has changed.

· Return value is ignored.

dwBytes

[in] Number of bytes of data that are currently available in the internal buffer.

fEOF

[in] TRUE if an end-of-file character has been reached and no more data is forthcoming from the audio source or FALSE if some data might remain. An audio device always returns FALSE.

For best performance, this notification should not be called by the audio-source object more than 16 times per second.

This notification is not sent when the amount of data changes as a result of the engine's own actions, such as calls to the IAudio::Flush or IAudioSource::DataGet member function.
