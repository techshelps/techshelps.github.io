IAudioDest Interface

The IAudioDest interface allows an engine to send information and data to an audio destination, such as a file or an audio wave-out device. When the engine gets the address of IAudioDest for an audio-destination object, that object already represents a particular audio destination. If the audio-destination object represents a multimedia wave-out device, IAudioDest is already open to a particular WAVEFORMATEX structure and device.
