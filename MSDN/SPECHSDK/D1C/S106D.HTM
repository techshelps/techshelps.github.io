IAudioDest::FreeSpace

HRESULT FreeSpace(DWORD *pdwBytes, BOOL *pfEOF);

Returns the number of bytes that are free in the object's internal buffer.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
AUDERR_NEEDWAVEFORMAT

pdwBytes

[out] Address of a variable that receives the number of bytes of space that are available in the audio-destination object's internal buffer.

pfEOF

Address of a variable that receives TRUE if an end-of-file character has been reached and no more data can be sent to the audio destination or FALSE if more data exists. If the audio destination is a file, this parameter receives TRUE if there is not enough disk space to finish writing the file. If the audio destination is a wave-out device, this parameter always receives FALSE.
