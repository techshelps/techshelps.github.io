IAudio::Start

HRESULT Start(void);

Informs an audio object to start playing or recording data.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
AUDERR_ALREADYSTARTED
AUDERR_NEEDWAVEFORMAT
AUDERR_NOTCLAIMED

An audio-destination object starts playing data from its internal buffer, and an audio-source object starts recording digital-audio data.

The internal buffer has a finite size (a minimum of 2 seconds of audio). If the internal buffer of an audio-destination object runs out of data, the user will hear gaps in the audio being played. If the buffer of an audio-source object fills up, data is lost.

For an audio-source object, if the IAudio::Claim member function has not been called, Start makes an implicit call to Claim. If the device is not available, Start continues to try to initiate recording until the IAudio::Stop member function is called. This implementation reduces the work required by the engine in obtaining a device which may not be available immediately. For additional error values that Start may return if it makes an implicit call to Claim, see the description of Claim.
