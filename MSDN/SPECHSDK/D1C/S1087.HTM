IAudioSource::DataGet

HRESULT DataGet(PVOID pBuffer, DWORD dwGetSize, DWORD *pdwCopied);

Reads digital-audio data from the internal buffer of an audio-source object, removing data from the buffer as it is read.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
AUDERR_NEEDWAVEFORMAT

pBuffer

[out] Address of a buffer that receives data from the internal buffer.

dwGetSize

[in] Number of bytes of data to write to the buffer.

pdwCopied

[out] Address of a variable that receives the number of bytes written to the buffer.

The audio-source object contains an internal FIFO buffer of digital-audio data. As audio data is received from the audio source, it is added to the end of the buffer. DataGet takes data from the beginning of the buffer.

If the buffer contains fewer than the requested number of bytes, DataGet returns the entire contents of the buffer.

An engine can use the data retrieved with DataGet as input for training dialog boxes or any other dialog boxes that use digital-audio data.
