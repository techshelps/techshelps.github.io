IVDctAttributes::ThresholdSet

HRESULT ThresholdSet(DWORD dwThreshold);

Sets the threshold level for a speech-recognition engine.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_NOTSUPPORTED
SRERR_VALUEOUTOFRANGE

dwThreshold

[in] Threshold level. If the value is out of range for the engine, an error is returned and the attribute is not changed. An application can specify SRATTR_MINTHRESHOLD or SRATTR_MAXTHRESHOLD for minimum or maximum allowable values.

The threshold level is a value from 0 to 100 that indicates the threshold below which an utterance is rejected as unrecognized. A value of 0 indicates that the engine matches any utterance to the closest phrase match. A value of 100 indicates that the engine must have determined absolutely that an utterance is the recognized phrase. The optimum threshold for the average user should be 50.
