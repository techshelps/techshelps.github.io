IVDctText::TextGet

HRESULT TextGet (DWORD dwStart, DWORD dwNumChars, PSDATA pData);

Gets the text from the internal buffer. It returns memory allocated by CoTaskMemAlloc. If the ANSI version of the function is called then the buffer size might vary on DBCS languages.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

dwStart

[in] Starting character. If this exceeds the size of the buffer then an error is returned.

dwNumChars

[in] Number of characters to get.

pData

[out] The SDATA structure is filled in with a pointer to the memory and the size of the memory. The application must free the memory with CoTaskMemFree.
