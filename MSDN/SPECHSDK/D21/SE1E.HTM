Inverse Text Normalization

"Inverse text normalization" is a feature in the voice dictation module that automatically formats text spacing, capitalization, and converts some phrases to a more symbolic form. Example: Inverse text normalization is what converts the spoken phrase, "the time is eight oh five p m period" to "The time is 8:05 PM." Applications can control the inverse text normalization by passing a Voice Dictation context free grammar into the Voice Dictation module. A description of the Voice Dictation Context Free Grammar is given later.
