IVDctText::TextSelSet

HRESULT TetSelSet (DWORD dwStart, DWORD dwNumChars);

Sets the position of the selection. Applications should keep this in sync with the text they display to the user. If the selection's screen coordinates change the application should call IVDctGUI::SetSelRect.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

dwStart

[in] Starting character of the selection.

dwNumChars

[in] Number of characters of selection. If dwNumChars is 0 then the selection is actually an insertion point.
