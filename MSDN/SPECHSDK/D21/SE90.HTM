IVoiceDictation::SessionSerialize

HRESULT SessionSerialize(PISTORAGE pISessionData);

SessionSerialize saves the current state of dictation to IStorage provided. The information includes all of the text in the buffer, results objects associated with the text, and bookmarks. An application can re-load a session during the Register call or by calling SessionSerialize.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pISessionData

[in] Storage to write all of the session data in.
