IVoiceDictation::TopicAddString

HRESULT TopicAddString(PCTSTR pszTopic, LANGUAGE *pLanguage);

Adds a new topic.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pszTopic

[in] Topic name. If this is NULL or an empty string the general topic, "General", will be used. If the topic exists then an error is returned.

pLanguage

[in] Language to use for the topic.

The language and topic name will be compiled into a grammar format to be passed to the speech recognition engine. It will be the standard grammar format for dictation.
