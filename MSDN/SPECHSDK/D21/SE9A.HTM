VDCTMEMORY

typedef struct {

DWORD dwMaxRAM;

DWORD dwMaxTime;

DWORD dwMaxWords;

BOOL fKeepAudio;

BOOL fKeepCorrection;

BOOL fKeepEval;

} VDCTMEMORY, *PVDCTMEMORY;

Provides information about memory restrictions for a voice dictation session.

dwMaxRAM

Maximum RAM, in bytes, to be used for storing the recognition results (including digital audio recordings, N-best list, etc.). If this is set to 0 when IVDctAttributes::MemorySet() is called then the current value isn't changed.

dwMaxTime

Maximum time, in seconds, for which the recognition results (including digital audio recordings, N-best list, etc.) will be maintained. If this is set to 0 when IVDctAttributes::MemorySet() is called then the current value isn't changed.

dwMaxWords

Maximum words for which the recognition results (including digital audio recordings, N-best list, etc.) will be maintained. If this is set to 0 when IVDctAttributes::MemorySet() is called then the current value isn't changed.

fKeepAudio

If TRUE then Voice Dictation will keep the ISRResAudio data around for new utterances. If FALSE then it discards the audio right away.

fKeepCorrection

If TRUE then Voice Dictation will keep the ISRResCorrection data around for new utterances. If FALSE then it discards the correction information right away.

fKeepEval

If TRUE then Voice Dictation will keep the ISRResEval data around for new utterances. If FALSE then it discards the evaluation information right away.
