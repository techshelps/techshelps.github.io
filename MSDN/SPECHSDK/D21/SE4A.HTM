IVDctAttributes::TimeOutSet

HRESULT TimeOutSet(DWORD dwIncomplete, DWORD dwComplete);

Sets the time-out values for a speech-recognition engine.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_NOTSUPPORTED
SRERR_VALUEOUTOFRANGE

dwIncomplete

[in] Number of milliseconds that the speech-recognition engine should wait before discarding an incomplete phrase because the user has stopped speaking. An application can specify SRATTR_MINTOINCOMPLETE or SRATTR_MAXTOINCOMPLETE for minimum or maximum allowable values.

dwComplete

[in] Number of milliseconds that the speech-recognition engine should wait before regarding a phrase as complete after the user has stopped speaking. An application can specify SRATTR_MINTOCOMPLETE or SRATTR_MAXTOCOMPLETE for minimum or maximum allowable values.

If a time-out value is set to a value that is out of range for the engine, an error is returned and the attribute is not changed.
