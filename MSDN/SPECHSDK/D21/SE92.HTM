IVoiceDictation::SiteInfoSet

HRESULT SiteInfoSet(PCTSTR pszSite, PVDCTSITEINFO pSiteInfo);

Given the name of a site, this sets properties about the site. If any applications are using the site then the properties of the site won't be changed until all applications are closed.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pszSite

[in] Site name. If this is NULL or an empty string the default site, "Local PC", will be used. If the site does not exist it will be created.

pSiteInfo

[in] New site information.
