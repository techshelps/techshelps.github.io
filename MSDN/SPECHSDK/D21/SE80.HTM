IVDctText::Lock

HRESULT Lock();

Locks the text object so that nothing except the application can modify it. An application shouldn't keep the text object locked for long. None of the other text functions will work until Lock() is called. Calling lock will call ISRCentral::Lock and prevent the engine from doing any further processing.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
