IVDctAttributes::IsAnyoneDictating

HRESULT IsAnyoneDictating(HWND hWnd, TCHAR *pszName, DWORD dwNameSize, DWORD *pdwNameNeeded);

An application passes in a window handle, and this returns whether or not the application (or anyone) has activated dictation for the window handle, and which application activated dictation.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
S_FALSE

hWnd

[in] Window which dictation is active for. If this is NULL then it checks to see if a global dictation session is activated.

pszName

[out] Pointer to a string to be filled with the name. This will not be filled in if *pdwNameNeeded > dwNameSize.

dwNameSize

[in] Number of bytes available in the data buffer pointed to by pszName.

pdwNameNeeded

[out] Filled in with the number of bytes needed for the entire name.
