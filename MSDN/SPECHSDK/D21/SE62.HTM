IVDctGUI::SetSelRect

HRESULT SetSelRect(RECT *pRect);

SetSelRect sets the rectangle in screen coordinates for where the selection or insertion point currently is. An application should call this whenever the window is moved or the insertion/selection coordinates change. If the user has not locked the correction window in place then the correction window will position itself near the selection rectangle. Applications will often call this in response to a selection change notification.

pRect

[in] Pointer to the rectangle containing screen coordinates of the caret or selection.
