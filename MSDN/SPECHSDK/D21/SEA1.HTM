Built In Commands

Voice Dictation provides the following built in commands. More commands may be available, depending upon the language.

"Capitalize". Capitalizes the next word.

"Capitalize-That". Capitalizes the previous word.

"Choose-one" through "Choose-ten". If the correction window is visible this replaces the last word with the Nth entry in the list.

"Cross-Out". Deletes the previous word.

"Go-Back-<1..9>". Moves the cursor back one through nine words.

"Go-Forward-<1..9>". Moves the cursor forward one through nine words.

"Go-To-The-End". Moves the cursor to the end of the document.

"Hide-Window". Hides the correction window.

"Show-Window". Shows the correction window.

"Stop-Dictating". Stops dictation and turns on speech recognition. Start Dictating will be supplied by Voice Commands.

"Today's-date". Type in today's date.
