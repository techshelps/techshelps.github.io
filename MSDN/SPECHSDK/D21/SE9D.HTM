VDICTGUIF_XXX Values

The following flags are used with the member functions of the IVCmdGUI interface to identify the state of the correction window.:

Value

Meaning

VDCTGUIF_VISIBLE

The correction window is visible.

VDCTGUIF_DONTMOVE

The correction window has been locked down so it doesn't move with the text when IVCmdGUI::SetSelRect() is called.

VDCTGUIF_ADDWORD

Shows a modeless dialog that allows users to add words.
