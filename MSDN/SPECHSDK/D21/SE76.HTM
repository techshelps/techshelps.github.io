IVDctText

The IVDctText interface allows applications to query and set the contents of the virtual text edit box that all dictated words are entered into. Applications need to keep the virtual text field synchronized with the one they display to end users.

Almost all of the functions in IVDctText will fail until IVDctText::Lock is called. The Lock call prevents the dictation and inverse text normalization features from modifying the text while the application looks at it. When the application is finished it must call IVDctText::UnLock.
