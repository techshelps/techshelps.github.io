IVDctInvTextNorm::OptionsEnum

HRESULT OptionsEnum(PSTR *ppOptions, DWORD *pdwSize);

OptionsEnum returns a buffer containing the names and descriptions of all the inverse text normalization options available in this topic.

ppOptions

[out] Filled with a pointer to memory that the calling application must free with CoTaskMemFree(). The data is filled with an alternating sequence of NULL terminated strings. The first string is the name of the first option, followed by the description of the first option, followed by the name of the second option, description of the second option, etc.

pdwSize

[out] Filled in with the number of bytes of data in *ppOptions.
