VSRMODE_XXX Values

The following flags are used with the member functions of the IVCmdAttributes interface to identify the current mode.:

Value

Meaning

VSRMODE_DISABLED

Dictation and command and control are turned off globally on the system. The user does not want to use them.

VSRMODE_OFF

Speech recognition is temporarily off for the system.

VSRMODE_CMDPAUSED

Speech recognition is paused (listening for a wake-up phrase such as "computer"). The command and control recognition is handling the paused.

VSRMODE_CMDONLY

Command and control recognition is listening. Dictation is off.

VSRMODE_DCTONLY

Dictation is listening. Command and control is off.

VSRMODE_CMDANDDCT

Both command and control, and dictation are on and listening at the same time. This can only be set if the engine can handle simultaneous command and control, and dictation.
