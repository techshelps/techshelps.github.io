Inverse Text Normalization Features

Voice Dictation provides the following "inverse text normalization" features to modify the text. Some of them can be turned on/off.

Automatic Spacing. Place spaces between words and sentences.

Automatic Capitalization of Words. Learn the capitalization corrections from users and use that capitalization until the user changes it again.

Automatic Capitalization of Sentences. Automatically capitalize words at the beginning of a sentence.

Number to Digits. Converts "six hundred forty two" to "642". It also handles decimals.

Years. Converts "nineteen ninety four" to "1994".
