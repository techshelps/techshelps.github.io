IVoiceDictation::CFGSet

HRESULT CFGSet(LANGID lang, PCSTR pszTopic, PCSTR pszCFG);

CFGSet passes down the text for a Voice Dictation Context Free Grammar to the Voice Dictation module. The grammar is compiled and used for the topic. The CFG is persistent, so it will continue to be used by the topic until it's changed.

lang

[in] Language that the inverse text normalization is for.

pszTopic

[in] Topic which the context free grammar is specific to. If pszTopic is NULL, the new CFG will be used as the default CFG for any topics that do not have a topic-specific CFG.

pszCFG

[in] Context free grammar text. See "Voice Dictation Context Free Grammars" for a description of the text.
