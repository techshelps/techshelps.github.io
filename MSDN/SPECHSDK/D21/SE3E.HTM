IVDctAttributes::MemorySet

HRESULT MemorySet(VDCTMEMORY *pDctMemory);

MemoryGet tells Voice Dictation how much memory to use for the results objects (used for correction) for the current session. If the memory constraints are tightened MemorySet does not free up any allocated resources. Changing this will affect the settings for this session only. The changes are not permanent across invocations.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pDctMemory

[in] Pointer to a VDCTMEMORY structure specifying memory constraints.
