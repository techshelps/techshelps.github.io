IVDctText::BookmarkEnum

HRESULT BookmarkEnum(DWORD dwStart, DWORD dwNumChars,

PVDCTBOOKMARK *ppBookMark, DWORD *pdwNumMarks);

This examines a range of the text and returns all of the bookmarks contained within it. It returns full information on the bookmark. The application must free up the memory with CoTaskMemFree.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

dwStart

[in] Starting position in characters. Bookmarks positioned from dwStart to dwStart+dwNumChars, inclusive, will be returned.

dwNumChars

[in] Number of characters. Bookmarks positioned from dwStart to dwStart+dwNumChars, inclusive, will be returned.

ppBookMark

[out] Filled with a pointer to memory containing an array of bookmarks. If none were found this is set to NULL. The memory must be freed by the application with CoTaskMemAlloc.

pdwNumMarks

[out] Filled with the number of bookmarks in ppBookMark.
