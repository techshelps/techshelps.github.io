IVoiceDictation::TopicSerialize

HRESULT TopicSerialize(PCTSTR pszTopic, PISTORAGE pITopicData);

Writes information stored about a topic on the local hard disk to a storage. Applications can call TopicSerialize and TopicDeserialize to move information learned in topics from one machine to another. Much of the information in a serialized session is engine dependent, so if the session is opened on a PC using a different dictation engine than some information will be lost.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pszTopic

[in] Topic name. If this is NULL or an empty string the general topic, "General", will be used. If the topic does not exist then an error is returned.

pITopicData

[in] IStorage where the information will be serialized.
