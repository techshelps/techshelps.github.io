IVDctInvTextNorm

The IVDctInvTextNorm interface allows applications to control the inverse text normalization performed by the voice dictation object. "Inverse text normalization" is a feature in the voice dictation module which automatically formats text spacing, capitalization, and converts some phrases to a more symbolic form. Example: Inverse text normalization is what converts the spoken phrase, "the time is eight oh five p m period" to "The time is 8:05 PM."

Member function

Description

OptionsEnum

Enumerates the options for inverse text normalization.
