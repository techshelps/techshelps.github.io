IVDctGlossary

This interface is supported by the voice-dictation object.

Member function

Description

Add

Adds one or more glossary entries to voice dictation.

Get

Gets information about one or more glossary entries in voice dictation.

Num

Indicates the total number of glossary entries in voice dictation.

Remove

Removes the specified glossary entries from voice dictation.

Set

Sets information for one or more glossary entries in voice dictation.
