IVoiceDictation

The IVoiceDictation interface allows an application to register with a voice-dictation object and to control recognition of speech. The application must call the IVoiceDictation::Register member function before it can call certain other functions, such as those in IVDctDialogs.

The IVoiceDictation interface is supported by the voice-dictation object. It has these member functions:

Member function

Description

Activate

Starts dictation listening.

Deactivate

Stops dictation listening.

Register

Registers a notification sink with the dictation object.

SessionSerialize

Writes a dictation session to disk.

SiteInfoGet

Gets the dictation information pertaining to a site.

SiteInfoSet

Sets the dictation information pertaining to a site.

TopicAddGrammar

Adds a new topic. Takes a grammar object to define the topic.

TopicAddString

Adds a new topic. Parameters are a several strings such as the topic name and language.

TopicDeserialize

Reads all of the topic data to a file.

TopicEnum

Enumerates the topics stored on the PC's hard disk.

TopicRemove

Removes a topic from disk.

TopicSerialize

Writes all of the topic data to a file.
