IVDctAttributes::MemoryGet

HRESULT MemoryGet(VDCTMEMORY *pDctMemory);

MemoryGet fills in a VDCTMEMORY structure with the memory constraints used for the dictation session.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pDctMemory

[out] Pointer to a VDCTMEMORY structure that is filled in.
