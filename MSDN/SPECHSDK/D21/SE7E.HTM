IVDctText::GetChanges

HRESULT GetChanged(DWORD *pdwNewStart, DWORD *pdwNewEnd,

DWORD *pdwOldStart, DWORD *pdwOldEnd);

This informs the applications of the changes that have occurred since the last time the application called IVDctText::Unlock. It returns where the changes started and ended in the old text, and where they now start and end in the new text. Thus, if text was not inserted/deleted then pdwXXXStart == pdwXXXEnd. If text was deleted then pdwOldStart < pdwOldEnd. Likewise if text was inserted, pdwNewStart < pdwNewEnd. If several changes occurred since the last call to UnLock then they will all be wrapped up into one larger change.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pdwNewStart

[out] Filled with the beginning of the new region that was changed.

pdwNewEnd

[out] Filled with the end of the new region that was changed.

pdwOldStart

[out] Filled with the beginning of the old region that was changed.

pdwOldEnd

[out] Filled with the end of the old region that was changed.

Example: If the text was "This is a test" the last time the application called UnLock, and is now changed to "This was a test" then, the following values would be returned:

*pdwNewStart = 5

*pdwNewEnd = 8

*pdwOldStart = 5

*pdwOldEnd = 7
