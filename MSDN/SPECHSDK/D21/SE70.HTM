IVDctNotifySink::TextChanged

HRESULT TextChanged(DWORD dwReason);

Notifies the application that text has been added, removed, or changes have been made to the data. The application should call IVDctText::Lock followed by IVDctText::GetChanges to find out what text has changed.

· Return value is ignored.

dwReason

[in] This parameter can be a combination of one or more of the following values:

Value

Meaning

VDCT_TEXTADDED

Text was added.

VDCT_TEXTREMOVED

Text was removed.

VDCT_TEXTREPLACED

Text was replaced.

VDCT_TEXTMOVED

Text was moved.
