Common Error Values

The following errors may be returned by common member functions. If no error occurs, the member functions return NOERROR.

Error value

Description

CO_E_OBJNOTCONNECTED

The handler was created, but the object application is not running. The object application must be running to determine whether it supports an interface.

E_FAIL

The function failed.

E_INVALIDARG

One or more arguments are invalid.

E_NOINTERFACE

The interface is not supported by the object; *ppvObj is set to NULL.

E_OUTOFMEMORY

There is no more memory.

E_UNEXPECTED

An unexpected error occurred.

LEXERR_ALREADYINLEX

The lexicon already contains information for the given word.

LEXERR_ENGBUFTOOSMALL

The buffer that is to receive the additional information about the given word is too small.

LEXERR_INVALIDPRONCHAR

The string that contains the phonetic or phonemic transcription of the given word contains an invalid character.

LEXERR_INVALIDSENSE

The value that indicates the alternate pronunciation or meaning of the word is invalid.

LEXERR_INVALIDTEXTCHAR

The string that contains the word to pronounce contains an invalid character.

LEXERR_NOTINLEX

The given word is not in the lexicon.

LEXERR_OUTOFDISK

The disk does not contain enough space to save the information to be added to the lexicon.

LEXERR_PRNBUFTOOSMALL

The buffer that is to receive the phonetic or phonemic transcription for the given word is too small.

REGDB_E_IIDNOTREG

The object application is running, but riid cannot be remoted because no information exists in the registration database to indicate how it is to be done.

S_FALSE

For the Next function: Fewer elements were returned than requested by celt. In this case, unused slots in the enumeration are not set to NULL and *pceltFetched holds the number of valid entries, even if zero is returned.

For the Reset function: The enumeration sequence was not reset to the beginning.

For the Skip function: The number of elements skipped is fewer than celt.

S_OK

For the Next function: The requested number of elements were returned; *pceltFetched is set if the number is non-NULL. All requested entries are valid.

For the Reset function: The enumeration sequence was reset to the beginning.

For the Skip function: The number of elements skipped is equal to celt.


In addition to the error values described in the preceding list, member functions can return error values generated by lower-level component objects or calls to the Win32 API. For example, an IVoiceText member function might return an AUDERR_ error value generated by the audio-destination object. For more information, see "Object Levels and Error Values."
