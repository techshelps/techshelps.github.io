VOICECHARSET

typedef enum {

CHARSET_TEXT = 0,

CHARSET_IPAPHONETIC = 1,

CHARSET_ENGINEPHONETIC = 2

} VOICECHARSET;

Indicates which character set is being used by a member function for a string.

CHARSET_TEXT

ANSI or Unicode character set.

CHARSET_IPAPHONETIC

IPA character set. This set requires a wide character (WCHAR).

CHARSET_ENGINEPHONETIC

Engine-specific phonetic character set.
