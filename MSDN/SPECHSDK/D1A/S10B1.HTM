Pronunciation Lexicons

Both speech-recognition and text-to-speech engines can make use of a pronunciation lexicon, which is a database of pronunciations for words and phrases to be recognized or played. Some engines allow an application or the user to edit the pronunciation lexicon to add new pronunciations or to replace incorrect pronunciations with correct ones.

An engine can give the user direct access to its pronunciation lexicon by providing a Lexicon dialog box. The Lexicon dialog box displays entries in the pronunciation lexicon and allows the user to edit them. An application causes a speech-recognition engine to display its Lexicion dialog box by calling the ISRDialogs::LexiconDlg or IVCmdDialogs::LexiconDlg member function. Similarly, an application calls the ITTSDialogs::LexiconDlg or IVTxtDialogs::LexiconDlg member function to cause a text-to-speech engine to display its Lexicon dialog box.

If an engine supports the optional ILexPronounce interface, an application can use it to query and control the engine's pronunciation lexicon. The ILexPronounce::Get member function retrieves the pronunciation and part of speech of a given word, the ILexPronounce::Add member function adds the pronunciation and part of speech of a word to the lexicon, and the ILexPronounce::Remove member function removes the pronunciation and part of speech of the given word from the lexicon.

For information about how to use the ILexPronounce interface to adjust a pronunciation, see Appendix C, "Unicode and the International Phonetic Alphabet."
