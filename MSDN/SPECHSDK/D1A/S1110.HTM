Synchronizing Notifications

Engines need to provide accurate synchronization for some of the notifications sent to the ITTSBufNotifySink interface. To do this:

1. Allocate memory to store the notification information. Copy the notification information into the memory.

2. While the digital-audio output is passed to the IAudioDest::DataSet member function, call the IAudioDest::BookMark member function with the pointer to the memory used as the bookmark identifier.

3. When the audio surrounding the bookmark is played, the audio-destination object calls the IAudioDestNotifySink::BookMark member function with a bookmark identifier. Convert this to a pointer to notification information.

4. Reference the notification information and call the appropriate member function of the ITTSBufNotifySink interface.

5. Free the notification information.
