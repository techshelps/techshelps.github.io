International Phonetic Alphabet

The International Phonetic Alphabet (IPA) is a standard system for indicating specific sounds, or phonemes. The Unicode character set contains all single symbols and diacritics in the most recently published IPA revision, as well as a few symbols no longer in use.

If an engine supports the IPA, an application can adjust the pronunciation of words by passing the engine the IPA phonemes that represent the correct pronunciation, expressed as Unicode values.

This appendix lists the Unicode values for common IPA phonemes. An application developer can use the information in this appendix to adjust pronunciations used by a speech-recognition or text-to-speech engine that supports the IPA.
