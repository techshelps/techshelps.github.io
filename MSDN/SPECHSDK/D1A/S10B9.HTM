ILexPronounce::Add

HRESULT Add(VOICECHARSET CharSet, TCHAR *pszText, TCHAR *pszPronounce,
VOICEPARTOFSPEECH PartOfSpeech, PVOID pEngineInfo,
DWORD dwEngineInfoSize);

Adds the pronunciation, the part of speech, and optional engine information of the given word to the lexicon.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
LEXERR_ALREADYINLEX
LEXERR_INVALIDPRONCHAR
LEXERR_INVALIDTEXTCHAR
LEXERR_OUTOFDISK

CharSet

[in] Character set to use for the pronunciation.

pszText

[in] Address of a string that contains the word to pronounce.

pszPronounce

[in] Address of a null-terminated string that contains the phonetic or phonemic transcription of

pszText.

PartOfSpeech

[in] Value that indicates the part of speech of the word. For a list of possible values, see the description of the VOICEPARTOFSPEECH enumeration.

pEngineInfo

[in] Address of a buffer that contains additional information about the word to be added. If an engine can maintain additional information about a word and the application can provide the information in the format required by the engine, it can pass that information to the engine. Otherwise, this parameter can be NULL.

dwEngineInfoSize

[in] Size, in bytes, of the data pointed to by pEngineInfo.

If the lexicon already has a sense of the word with the specified pronunciation and part of speech, the pronunciation lexicon is not changed.
