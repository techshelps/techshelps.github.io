How a dictation engine passes up punctuation and numbers...

Generally, the Voice Dictation module will transcribe exactly what the engine passes up to it. However if it sees a backslash in a "word" then it puts whatever is to the left of the back-slash in the text buffer. This would be used for punctuation, like:

.\Period

;\Semi-colon

\\Back-Slash

If a number is passed up (because the recognizer supports continuous digits) then the number is added to the text buffer without alteration.
