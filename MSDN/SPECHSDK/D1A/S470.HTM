Relationships Between Phonemes

The following tables and diagrams illustrate the relationships between phonemes (consonants and vowels) with Unicode values and some examples in English. Phonemes without examples are typically used in languages other than English.

If a phoneme needs to be adjusted, it will typically be to an adjacent cell in a table or node in a diagram.

Rounded Vowels

A rounded vowel is pronounced with the lips making an approximately oval opening.

An unrounded vowel is pronounced without rounding the lips.
