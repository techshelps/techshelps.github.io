Developing a Speech Engine

This appendix contains information for developers of speech-recognition and text-to-speech engines. It describes how to develop an engine that is compatible with the speech application programming interface (API).

Before reading this document, you should read the section, "Low-Level Speech Recognition API" and "Low-Level Text-to-Speech API." You should also be familiar with the objects, interfaces, and member functions that support speech-recognition and text-to-speech.
