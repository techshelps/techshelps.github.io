Supporting Multiple Languages

To support multiple languages, an engine should:

· Provide one or more speech-recognition modes for each language, each with its own globally-unique identifier (GUID). If the engine can support multiple languages at one time, all of the modes should have the same engine GUID.

· Load the modes for a language when the application uses that particular language.

· Do not provide a mode for a language if the application's setup program does not install a mode for that language.

In some cases, one or more applications will need to have several languages active and simultaneously recognizing speech from the same audio source. This is especially true in multilingual countries or countries that do not have localized versions of all products.
