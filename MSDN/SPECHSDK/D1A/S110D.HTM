Registry

When the user installs the text-to-speech engine, the setup program should write the following registry entries:

1. A class identifier entry for each OLE COM object.

2. An engine identifier entry under the following key:

HKEY_LOCAL_MACHINE\Software\Voice\TextToSpeech\Engine

The engine identifier entry should be in the following format:

EngineName : REG_SZ : EngineGUID

In an application, the text-to-speech enumerator enumerates all of the strings under this key and allows applications to access the engines' text-to-speech modes.

3. Registry entries for any custom marshaling code.

For more information about class identifiers and registry entries for marshaling code, see the Win32 SDK.
