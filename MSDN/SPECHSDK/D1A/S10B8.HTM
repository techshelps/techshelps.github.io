ILexPronounce Interface

The ILexPronounce interface allows an application to query and control the pronunciation lexicon for a speech-recognition or text-to-speech engine. If a speech-recognition engine supports ILexPronounce, it also supports the International Phonetic Alphabet (IPA) character set. ILexPronounce may be used by other objects in the system, such as a lexicon object. This interface is optional, and may or it may not be supported by an engine.
