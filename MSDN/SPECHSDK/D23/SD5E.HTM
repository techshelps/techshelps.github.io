IVTxtNotifySink::AttribChanged

HRESULT AttribChanged(DWORD dwAttribute);

Notifies all applications on a voice-text site that a process has changed an attribute of the site.

· Return value is ignored.

dwAttribute

[in] Site attribute that was changed. This parameter can be one of these values:

Value

Meaning

IVTNSAC_DEVICE

Audio-output device

IVTNSAC_ENABLED

Enabled state of voice text

IVTNSAC_SPEED

Average talking speed

IVTNSAC_TTSMODE

Text-to-speech mode

IVTNSAC_VOLUME

Baseline speaking volume


The notification is sent only to applications that set the dwFlags parameter of the IVoiceText::Register member function to the VTXTF_ALLMESSAGES value when the application registered to use voice text on the site.
