Speaking Text

To cause the text-to-speech engine to speak voice text , an application must place the text into the playback queue by calling the IVoiceText::Speak member function. The following example puts text into the playback queue for the voice-text object, indicating that the text is a normal-priority statement.

// SpeakString - puts a string in the playback queue.

// Returns the result of calling IVoiceText::Speak.

// szSpeak - address of buffer containing the text to speak

HRESULT SpeakString(char *szSpeak)

{

HRESULT hRes;

if( *szSpeak == NULL )

return 0;

// Put the text in the playback buffer.

hRes = gpIVTxt->Speak(szSpeak,

VTXTST_QUESTION | VTXTSP_NORMAL, NULL);

return hRes;

}
