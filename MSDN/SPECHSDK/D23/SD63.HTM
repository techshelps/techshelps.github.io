IVoiceText Interface

The IVoiceText interface allows an application to register with a voice-text object and to control how voice text is spoken. The application must call the IVoiceText::Register member function before it can call any of the other member functions in this interface or certain other member functions, such as those in the IVTxtDialogs interface.

The IVoiceText interface is supported by the voice-text object. It has these member functions:

AudioFastForward

Register

AudioPause

Speak

AudioResume

StopSpeaking

AudioRewind
