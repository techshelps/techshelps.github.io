Average Talking Speed

A text-to-speech engine can speak text at various speeds. An application can set the engine's average talking speed by using the IVTxtAttributes::SpeedSet member function and can retrieve the current average speed by using the IVTxtAttributes::SpeedGet member function. The average talking speed is measured in words per minute.

To discover whether an engine is currently speaking text for a voice-text site, an application can call the IVTxtAttributes::IsSpeaking member function.
