Lexicon Dialog Box

A text-to-speech engine maintains a database of pronunciations called a pronunciation lexicon. The engine uses the pronunciation lexicon to determine how to pronounce words and phrases. The IVTxtDialogs::LexiconDlg member function displays a Lexicon dialog box that allows the user to view and edit the pronunciation lexicon. For example, the user can edit the phonetics of mispronounced words so that the engine pronounces them properly.
