General Dialog Box

The General dialog box gives the user control of the general features of the text-to-speech engine. The dialog box may also include controls that allow the user to open other dialog boxes provided by the engine. An application displays the General dialog box by using the IVTxtDialogs::GeneralDlg member function. Depending on the engine, the General dialog box may allow the user to perform the following tasks:

· Set the maximum amount of disk space allotted for text-to-speech output. (Typically, setting the disk space is relevant only for rendering concatenated wave files.)

· Set the speaker preferences supported by the engine.
