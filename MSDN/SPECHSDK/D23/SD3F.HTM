Translation Dialog Box

The Translation dialog box allows the user to control the translation techniques for symbols, currencies, abbreviations, and numbers. For example, the user can specify whether to speak "34" as "thirty-four," "three four," or the old English "four and thirty." This dialog box also allows the user to control the translation of abbreviations and case sensitivity and to manage how both affect the prosody of spoken text. An application displays the Translation dialog box by using the IVTxtDialogs::TranslateDlg member function.
