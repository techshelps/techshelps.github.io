IVTxtAttributes

This interface is supported by the voice-text object.

Member function

Description

DeviceGet

Returns the device ID of the wave-out audio device used for the site.

DeviceSet

Changes the wave-out audio device for the site.

EnabledGet

Indicates whether or not voice text is enabled for the site.

EnabledSet

Enables or disables all voice text for a site.

IsSpeaking

Indicates whether text is currently being spoken by the site.

SpeedGet

Gets the current average talking speed for a site, in words per minute.

SpeedSet

Sets the current average talking speed for a site, in words per minute.

TTSModeGet

Gets the GUID of the current text-to-speech mode for the site.

TTSModeSet

Sets the text-to-speech mode for the site.
