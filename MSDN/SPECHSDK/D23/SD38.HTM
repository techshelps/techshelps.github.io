Text-to-Speech Mode

A text-to-speech engine typically provides an assortment of modes that it can use to play speech in different languages, dialects, and speaking styles. Each voice-text site uses a single text-to-speech mode. An application can set the mode for a voice-text site by specifying the GUID of the new site in a call to the IVTxtAttributes::TTSModeSet member function. To retrieve the GUID of the mode that the site is currently using, an application calls the IVTxtAttributes::TTSModeGet member function.
