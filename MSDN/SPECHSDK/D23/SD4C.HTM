IVTxtNotifySink

This interface is a notification sink that is supported by an application.

Member function

Description

AttribChanged

Notifies applications that a site attribute has changed.

Speak

Notifies applications on a site that an application has added voice text to the queue.

SpeakingDone

Notifies all applications on a site that speaking is finished and no voice text remains in the queue.

SpeakingStarted

Notifies all applications on a site that speaking has started.

Visual

Notifies applications on a site which phoneme is currently being spoken.
