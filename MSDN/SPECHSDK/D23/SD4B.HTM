IVTxtDialogs

This interface is supported by the voice-text object.

Member function

Description

AboutDlg

Displays an About dialog box that identifies the text-to-speech engine and contains the copyright notice.

GeneralDlg

Allows general control of the text-to-speech engine and should invoke some of the other dialog boxes.

LexiconDlg

Allows users to display and edit their dictionary entries.

TranslateDlg

Allows the end-user to control symbols, currencies, abbreviations, and number-translation techniques, as well as translation of abbreviations and case sensitivity, and how both affect the prosody of text.
