IVTxtAttributes Interface

The IVTxtAttributes interface allows an application to retrieve and set the attributes of a voice-text site. The application must call the IVoiceText::Register member functions before it can call any of the member functions in the IVTxtAttributes interface.

A voice-text application may not need to use IVTxtAttributes member functions if a voice-navigation application is installed on the user's computer. The voice-navigation application typically monitors and controls the attributes of a voice-text site.

Changing an attribute of a site affects all applications using the site. Settings are saved between uses of the site, even if the user shuts down the computer in the meantime.

The IVTxtAttributes interface is supported by the voice-text object. It has these member functions:

DeviceGet

SpeedGet

DeviceSet

SpeedSet

EnabledGet

TTSModeGet

EnabledSet

TTSModeSet

IsSpeaking
