ISRResGraph Interface

The ISRResGraph interface allows an application to examine the recognition/alternative graph generated by the engine. An application that needs detailed results uses this interface. This interface may or may not be supported by an engine. The GetPhonemeNode member function is optional.

The interface assumes that the engine has created a word graph while recognizing an utterance. The graph has both a beginning and alternate recognition paths that were explored by the engine in the process of recognizing the utterance.
