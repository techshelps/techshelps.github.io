General Format of a Grammar

Context-free, dictation, and limited-domain grammars use the same general format, which consists of a header, followed by a list of information chunks. Each grammar can contain one or more information chunks that follow each other sequentially. Several chunks of the same type—for example, Sample Text—can be included in the same grammar. The chunk structure allows the grammar to be extensible and allows a speech-recognition engine to skip information that is not relevant or not known to the engine.

A grammar can have only one SRCK_LANGUAGE chunk, which identifies the language of the grammar. If the grammar does not have a SRCK_LANGUAGE chunk, the grammar will use the current speech-recognition mode's default language. Some engines may not be able to load a grammar if it is for the wrong language.

The rest of this appendix describes significant features of context-free, dictation, and limited-domain grammars and provides reference information for each structure.
