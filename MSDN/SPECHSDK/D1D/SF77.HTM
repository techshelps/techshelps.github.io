Interacts with

When a speech-recognition enumerator is created, it loads the registry entries for each engine installed on the system and creates an engine enumerator object to represent each engine.

The application calls the ISRFind::Find member function on the speech-recognition enumerator with the address of an SRMODEINFO structure that indicates the desired characteristics of the mode and the address of an SRMODEINFORANK structure that indicates the relative importance of each characteristic. As an alternative, the application can call member functions of the ISREnum interface to enumerate the modes.

The speech-recognition enumerator passes these calls to each engine enumerator until one of them returns a mode that fits the requested characteristics.

After the application has created an engine object, the speech-recognition enumerator is no longer needed and the application can release it.
