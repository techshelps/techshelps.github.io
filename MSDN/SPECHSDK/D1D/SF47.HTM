Providing Feedback

An application can improve feedback to the user by displaying the information received with the PhraseHypothesis notification, replacing the text from the previous PhraseHypothesis each time a new one is received.

If the engine supports the ISRGramInsertionGUI member function, an application can use this interface to display a small status window next to the insertion point in the user's document. As the insertion point moves, the application calls the ISRGramGUI::Show and ISRGramGUI::Move member functions to change the window's position.
