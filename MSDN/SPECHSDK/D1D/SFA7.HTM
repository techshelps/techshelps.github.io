ISRFind Interface

The ISRFind interface allows an application to search through speech-recognition modes, to select a mode that most closely matches the desired characteristics, and to create an engine object that represents the selected mode.

Because ISRFind selects a speech-recognition mode based on the relative importance of various characteristics, it may not always select the optimal mode (just as the Windows font-matching function does not always select the optimal font). ISRFind is best used to select a default speech-recognition mode. An application should also allow the user to select a mode, and should store its GUID so that the mode can be retrieved later.

The ISRFind interface for the speech-recognition enumerator calls ISRFind for all of the registered speech-recognition engines and then selects the closest result. If an engine does not supply an ISRFind interface, the speech-recognition enumerator uses the engine's ISREnum interface to simulate this functionality.
