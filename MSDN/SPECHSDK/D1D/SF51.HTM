ISRAttributes

This interface is supported by the speech-recognition engine.

Member function

Description

AutoGainEnableGet

Gets the state of the automatic gain for the incoming audio stream.

AutoGainEnableSet

Sets the state of the automatic gain for the incoming audio stream.

EchoGet

Gets the state of echo canceling on the incoming audio stream.

EchoSet

Sets the state of echo canceling on the incoming audio stream.

EnergyFloorGet

Gets the value of the current noise floor expected by a speech-recognition engine from the audio stream.

EnergyFloorSet

Sets the value of the current noise floor expected by a speech-recognition engine from the audio stream.

MicrophoneGet

Gets the name of the current microphone for this audio source.

MicrophoneSet

Sets the name of the current microphone for this audio source.

RealTimeGet

Gets the real-time setting for the engine.

RealTimeSet

Sets the real-time setting for the engine.

SpeakerGet

Gets the name of the current speaker for an audio source.

SpeakerSet

Sets the name of the current speaker for an audio source.

ThresholdGet

Gets the threshold level of the speech-recognition engine.

ThresholdSet

Sets the threshold level of the speech-recognition engine.

TimeOutGet

Gets the current time-out values for a speech-recognition engine.

TimeOutSet

Sets the time-out values for a speech-recognition engine.
