Interfaces

The speech-recognition results object should support the ISRResBasic interface, and may also support any of the following interfaces. The results object for an engine may support other interfaces defined by the engine vendor to provide additional functionality.

Interface

Description

ISRResAudio

(Optional.) Gets an audio recording of an utterance from an engine.

ISRResBasic

Provides general information about a speech-recognition results object. Member functions allow an application to obtain the phrase that was recognized or one of its alternatives, the time that a phrase began and ended, and the grammar that was involved in the recognition.

ISRResCorrection

(Optional.) Confirms that an utterance was recognized correctly, or provides the correct interpretation of an incorrectly recognized utterance.

ISRResEval

(Optional.) Reevaluates the recognition.

ISRResGraph

(Optional.) Provides information from a recognition/alternative graph generated by an engine. Member functions allow an application to obtain the most likely recognition path of all those attempted by the engine, the score for a path, or the phoneme or word for a particular node in a path.

ISRResMemory

(Optional.) Gets the amount of memory used by a particular kind of information in a recognition/alternative graph, or frees memory used by the information.

ISRResMerge

(Optional.) Merges information from two temporally adjacent speech-recognition results objects into a new object.

ISRResModifyGUI

(Optional.) Allows an application to display a small status window next to selected text for specific recognition results.

ISRResScore

(Optional.) Provides scores for recognition results.

ISRResSpeaker

(Optional.) Identifies the speaker assumed by the engine and allows an application to correct or validate the speaker.
