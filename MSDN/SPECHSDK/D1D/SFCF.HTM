ISRNotifySink::Sound

HRESULT Sound(QWORD qTimeStampBegin, QWORD qTimeStampEnd);

Notifies the application that the engine detected a sound (or a very garbled word).

· Return value is ignored.

qTimeStampBegin

[in] Time, in bytes, when the sound began.

qTimeStampEnd

[in] Time, in bytes, when the engine notified the application of the sound.

The engine determines how long to wait before sending this notification.
