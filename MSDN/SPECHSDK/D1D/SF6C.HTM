Interfaces

The grammar object should support the following interfaces. The grammar object provided for a particular engine may support other interfaces with additional functionality.

Interface

Description

ISRGramCFG

(Context-free grammar only.) Controls a context-free grammar. Member functions manage the lists of words that the grammar recognizes, or determine whether the grammar is linked to another grammar.

ISRGramCommon

Controls features common to context-free, limited-domain, and dictation grammars. Member functions activate and deactivate a grammar, archive it, set a bookmark in the incoming audio stream, control storage of speech-recognition results objects, and allow the user to train the grammar to recognize his or her voice.

ISRGramDictation

(Dictation grammar only.) Controls a dictation grammar. Member functions set the context for dictation and indicate the type of text and words that are likely to be spoken.

ISRGramNotifySink

Notifies an application about events related to speech recognition with a grammar. Notifications indicate that the engine has encountered a bookmark; that the user has started a phrase or the engine has recognized a phrase; that the engine is paused; that a speech-recognition results object was reevaluated or retrieved when an archived grammar was loaded; or that training is recommended.
