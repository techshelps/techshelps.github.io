To create a grammar object, use the ISRCentral::GrammarLoad member function on an engine object and pass information about the grammar, such as its format and where it is located. The grammar can be new or previously archived. You can optionally pass the address of a notification interface that the grammar object can use to send notifications back to the application. To receive notifications, an application must create a COM object that supports ISRGramNotifySink as an interface. How the application responds to notifications is up to the application developer.

The GrammarLoad member function returns the address of an IUnknown interface on the newly created grammar object.
