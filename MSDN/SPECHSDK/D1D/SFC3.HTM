ISRGramNotifySink Interface

The ISRGramNotifySink interface is used by a speech-recognition engine to notify an application of recognition-related events. An application specifies the address of this interface when it creates the grammar object. A grammar may or may not call the PhraseHypothesis and Training member functions.

An engine processes one "phrase" at a time and follows every PhraseStart member function with a PhraseFinish member function before going on to process the next phrase. This means that the engine may fall behind in recognitions if several phrases or commands are spoken in quick succession. However, the application need not be concerned about this. If the engine is paused, it does not go on to the next phrase.

The ISRGramNotifySink interface must match the ISRCentral interface from which the GrammarLoad member function is being called — ISRGramNotifySinkA for ISRCentralA (ANSI) or ISRGramNotifySinkW for ISRCentralW (Unicode).
