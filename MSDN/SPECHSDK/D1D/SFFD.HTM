ISRSpeaker::Merge

HRESULT Merge(TCHAR *pszSpeakerName, PVOID pSpeakerData,

DWORD dwSpeakerData);

Merges a speaker profile from another system into the speaker profile on the current system.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pszSpeakerName

[in] Address of a string that contains the name of the speaker.

pSpeakerData

[in] Address of the speaker profile data to merge with the profile of the speaker on the current system.

dwSpeakerData

[in] Size, in bytes, of the data pointed to by pSpeakerData.
