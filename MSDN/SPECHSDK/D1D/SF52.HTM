ISRCentral

This interface is supported by the speech-recognition engine.

Member function

Description

GrammarLoad

Loads an archived or newly created grammar and returns a grammar object.

ModeGet

Gets information specific to the current speech-recognition mode.

Pause

Pauses the speech-recognition engine.

PosnGet

Gets the exact byte in the audio stream that is currently being received.

Register

Registers an ISRNotifySink interface to notify the application of grammar-independent recognitions.

Resume

Reduces the pause count of a paused speech-recognition engine by 1.

ToFileTime

Converts a QWORD representing the time in bytes to a Win32 FILETIME structure.

UnRegister

Releases a notification interface through which an engine sent grammar-independent notifications to an application.
