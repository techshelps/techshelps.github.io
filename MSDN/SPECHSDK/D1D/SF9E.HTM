ISRCentral::UnRegister

HRESULT UnRegister(DWORD dwKey);

Allows an application to release a notification interface through which an engine sent grammar-independent notifications to an application. An application must call UnRegister before closing the engine object.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_INVALIDKEY

dwKey

[in] Key value returned by the Register member function for the notification interface when the application registered the interface.
