ISRSpeaker::Write

HRESULT Write(TCHAR *pszSpeakerName, PVOID pSpeakerData,

DWORD dwSpeakerData);

Writes speaker profile information from another system to the current system.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_SPEAKEREXISTS

pszSpeakerName

[in] Address of a string that contains the name of the speaker.

pSpeakerData

[in] Address of the speaker profile data to write.

dwSpeakerData

[in] Size, in bytes, of the data pointed to by pSpeakerData.

If the specified speaker already exists on the current system, the function returns SRERR_SPEAKEREXISTS. If this happens, the application can either call the ISRSpeaker::Merge member function to combine the speaker profiles or delete the current profile and call Write again.
