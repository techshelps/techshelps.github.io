ISRResGraph::GetWordNode

HRESULT GetWordNode(DWORD dwWordNode, PSRRESWORDNODE pNode,

PSRWORD pSRWord, DWORD dwMemSize, DWORD *pdwMemNeeded);

Retrieves information about the word for a given node in a recognition results graph.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_NOTENOUGHDATA
SRERR_NOTSUPPORTED
SRERR_VALUEOUTOFRANGE

dwWordNode

[in] Number of the desired node. The beginning node in a graph is always 1. An application can obtain the number of a particular node by checking the SRRESPHONEMENODE or SRRESWORDNODE structures of one node for next, previous, and alternative nodes.

pNode

[out] Address of an SRRESWORDNODE structure that receives information about the node.

pSRWord

[out] Address of an SRWORD structure that receives the word (if the allocated memory is large enough). If this parameter is NULL, no information is copied.

dwMemSize

[in] Number of bytes available in the structure specified by pSRWord. If the number is too small, the member function returns an error and fills pdwMemNeeded with the number of bytes needed to store the word.

pdwMemNeeded

[out] Number of bytes needed to store the entire word structure.
