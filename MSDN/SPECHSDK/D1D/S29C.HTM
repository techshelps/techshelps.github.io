Audio-Source Object

Before an application can use speech recognition, it must determine the origin of the audio to be recognized. Typical audio sources include a microphone, a telephone line, and a file.

To recognize audio from a wave-in device, the application can create an instance of the multimedia audio-source object, which is an OLE Component Object Model (COM) object that supports audio communication interfaces in common with the engine. To use audio from a different source (such as a file), or to manipulate the audio before it gets to the recognition engine, an application can use a custom audio-source object that you provide or one provided by a third-party vendor.
