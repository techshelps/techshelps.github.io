Improving Accuracy

Certain functions can be used to improve the accuracy of dictation. If an application has information about what kind of information is being dictated, or if it has information about the context, it should call the ISRGramDictation::Context member function whenever the user moves the insertion point, whether by using the mouse, the keyboard, or a voice command.

An application can also use the ISRGramDictation::Hint member function if it has information that an address or date is being entered, or the ISRGramDictation::Words member function if a likely set of words is being used.
