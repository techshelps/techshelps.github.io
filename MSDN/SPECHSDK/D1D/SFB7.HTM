ISRGramCommon::DeteriorationSet

HRESULT DeteriorationSet(DWORD dwMemory, DWORD dwTime, DWORD dwObjects);

Sets the values used by the engine to determine how long to retain data stored in speech-recognition results objects.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_NOTSUPPORTED
SRERR_VALUEOUTOFRANGE

dwMemory

[in] Amount of memory to allow for storing speech-recognition results objects.

dwTime

[in] Number of seconds to allow a results object to exist.

dwObjects

[in] Maximum number of results objects to allow at one time.

To save memory, an engine allows data in results objects to deteriorate, unless a results object is locked by using the ISRResMemory::LockSet member function.

If an object deteriorates, the application's requests for data will fail. An application can identify what data is available by calling the ISRResMemory interface.
