ISRGramNotifySink

This interface is a notification sink that is supported by an application.

Member function

Description

Bookmark

Notifies the application that the engine has encountered a bookmark in the incoming audio stream.

Paused

Reminds the application that the engine is paused.

PhraseFinish

Informs the application that the speaker has finished a phrase and the speech-recognition engine has determined the words that were spoken.

PhraseHypothesis

Informs the application that the engine has a hypothesis about what was said over the past few seconds but does not have enough information to commit to a recognition.

PhraseStart

Notifies the application that the engine has started processing audio data for a recognition result.

ReEvaluate

Notifies the application that the engine changed a speech-recognition results object as a consequence of the engine's background processing (rather than as a result of an explicit call of the ISRResEval::ReEvaluate member function by the application).

Training

Notifies the application that the engine requires training for the current speaker, the current grammar, or the current microphone.

UnArchive

Notifies an application that a speech-recognition results object that was archived with a grammar has been loaded with the grammar.
