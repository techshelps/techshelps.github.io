ISRGramDictation

This interface is supported by the grammar object of a dictation grammar.

Member function

Description

Context

Provides a speech-recognition engine with context—that is, the words that were spoken before and after the current utterance.

Hint

Gives the speech-recognition engine information about the subcontext in which a document is being used.

Words

Informs the speech-recognition engine that a particular set of words is likely to be spoken (the engine can ignore these).
