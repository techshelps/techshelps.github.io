IEnumSRShare::New

HRESULT New(DWORD dwDeviceID, GUID gModeID,

PISRCENTRAL *ppISRCentral, QWORD *pqwInstanceID);

Creates a new engine/audio-source pair that can be shared by other applications.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_INVALIDMODE
SRERR_OUTOFDISK
SRERR_WAVEFORMATNOTSUPPORTED

dwDeviceID

[in] Device identifier of the wave-in audio device to use. If the device is already in use, an error is returned. An application can obtain a list of devices by calling the waveInGetNumDevs and waveInGetDevCaps Windows functions.

gModeID

[in] GUID that identifies the desired speech-recognition mode. An application can use a speech-recognition enumerator to obtain the mode.

ppISRCentral

[out] Address of a variable that receives a pointer to the ISRCentral interface for the engine object of the newly created engine/audio-source pair.

pqwInstanceID

[out] Address of a variable that receives the instance identifier of the newly created engine/audio-source pair. Use this identifier later in a call to Detach to detach from this instance of the engine/audio-source pair.
