ISREnum Interface

The ISREnum interface allows an application to enumerate the different speech-recognition modes and to select a mode for which to create an engine object.

Calling ISREnum on an engine enumerator allows the application to iterate through the engine's list of known speech-recognition modes. Calling ISREnum on a speech-recognition enumerator calls engine enumerators for all of the speech-recognition engines on the system.

The ISREnum interface is provided by the speech-recognition enumerator and engine-enumerator objects. It has these member functions:

Clone

Select

Next

Skip

Reset


All OLE enumeration interfaces contain the member functions Clone, Next, Reset, and Skip. In the ISREnum interface, these member functions use SRMODEINFO structures as the elements in the enumeration sequence. For more information about Clone, Next, Reset, and Skip, see the section, "Common Elements of the Speech Application Programming Interface."
