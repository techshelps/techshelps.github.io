IEnumSRShare Interface

The IEnumSRShare interface allows an application to enumerate the current engine/audio-source pairs and either select an existing pair or create a new pair.

This interface is supported by the speech-recognition sharing object. It has these member functions:

Clone

Reset

Detach

Share

New

Skip

Next


All OLE enumeration interfaces contain the Clone, Next, Reset, and Skip member functions. In the IEnumSRShare interface, these member functions use SRSHARE structures as the elements in the enumeration sequence.
