ISRGramNotifySink::ReEvaluate

HRESULT ReEvaluate(PIUNKNOWN pIUnknown);

Notifies an application that the engine changed a speech-recognition results object as a consequence of the engine's background processing (rather than an explicit ReEvaluate call by the application).

· Return value is ignored.

pIUnknown

[in] Address of the IUnknown interface on the speech-recognition results object that was reevaluated.

The application can compare the IUnknown interface of the reevaluated results object against the list of existing results objects or use the information returned by the object to determine the text to which it applies. The application does not have to call the ISRResEval::ReEvaluate member function to obtain the new information.

The application does not have to (and should not) release pIUnknown because the ISRGramNotifySink::ReEvaluate member function does not call the AddRef member function. This allows the application to ignore ReEvaluate notifications.
