ISRSpeaker

This interface is supported by the speech-recognition engine.

Member function

Description

Delete

Deletes all information that a speech-recognition engine has about a speaker.

Enum

Enumerates the list of speakers known to the speech-recognition engine.

Merge

Merges a speaker profile from another system into the speaker profile on the current system.

New

Creates a new speaker, using the default settings provided by the engine for training.

Query

Gets the name of the currently selected speaker.

Read

Gets speaker profile information for the specified speaker.

Revert

Restores training to its original state at the beginning of a training session.

Select

Selects the specified speaker to be the current speaker on the system.

Write

Writes speaker profile information from another system to the current system.
