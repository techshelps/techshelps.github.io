ISRCentral::PosnGet

HRESULT PosnGet(QWORD *pqwTimeStamp);

Retrieves the exact byte in the audio stream that is currently being received (or as exact as possible given the audio device).

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_VALUEOUTOFRANGE

pqwTimeStamp

[out] Address of a variable that receives the byte in the audio stream that is currently being received.

This member function works only for real-time audio sources such as microphones; it does not work for sources such as files.

An application can use this member function to synchronize actions with events. For example, if the user clicks a mouse button, an application can call PosnGet to retrieve the byte of audio being played at that moment and later determine whether or not the mouse click occurred before or after that particular byte.
