ISRResMerge::Split

HRESULT Split(QWORD qwSplitTime, PIUNKNOWN ppIUnkLeft,

PIUNKNOWN ppIUnkRight);

Splits a speech-recognition results object into two parts, keeping the original object intact. An application can use this function to examine smaller parts of a recognition result and to perform training or correction on the parts.

If splitting the results object at qwSplitTime would result in only one object, then the function returns SRERR_VALUEOUTOFRANGE.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_NOTENOUGHDATA
SRERR_NOTSUPPORTED
SRERR_VALUEOUTOFRANGE

qwSplitTime

[in] Time stamp at which the result should be split. Because in many cases a word cannot be split in the middle, the engine maps this to the closest time that the result can be split.

ppIUnkLeft

[out] Address of a variable that receives a pointer to the IUnknown interface for the new results object created by the split. This is the portion of the original results object that occurred before the time specified in qwSplitTime. If no portion of the results object occurred before this time, ppIUnkLeft receives NULL. The original results object is not affected.

ppIUnkRight

[out] Address of a variable that receives a pointer to the IUnknown interface for the new results object created by the split. This is the portion of the original results object that occurred after the time specified in qwSplitTime. If no portion of the results object occurred after this time, ppIUnkRight receives NULL. The original results object is not affected.
