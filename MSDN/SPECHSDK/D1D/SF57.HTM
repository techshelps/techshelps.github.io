ISRGramCommon

This interface is supported by the grammar object.

Member function

Description

Activate

Activates a grammar so that the speech-recognition engine can begin using the grammar to recognize speech.

Archive

Compiles a grammar in a standard grammar format into the native format for the engine and stores the compiled grammar in a data buffer.

Bookmark

Sets a bookmark in the incoming audio stream.

Deactivate

Deactivates an active grammar so that the speech-recognition engine no longer uses the grammar to recognize speech.

DeteriorationGet

Gets the values used by the engine to determine how long to retain data stored in speech-recognition results objects.

DeteriorationSet

Sets the values used by the engine to determine how long to retain data stored in speech-recognition results objects.

TrainDlg

Displays a dialog box so that the speaker can train the grammar.

TrainPhrase

Queries the engine for a list of phrases the speaker should utter to improve recognition accuracy.

TrainQuery

Queries the engine for the kind of training that is needed for a particular grammar.
