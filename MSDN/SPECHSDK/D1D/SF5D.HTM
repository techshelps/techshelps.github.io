ISRResBasic

This interface is supported by the speech-recognition results object.

Member function

Description

FlagsGet

Gets the recognition flags for the specified phrase.

Identify

Gets a unique identifier for a results object so the application can retrieve the object at a later time.

PhraseGet

Gets the specified phrase from the list of most likely phrases provided by the speech-recognition engine for a recognition.

TimeGet

Gets the time stamps that mark the beginning and ending of the phrase.
