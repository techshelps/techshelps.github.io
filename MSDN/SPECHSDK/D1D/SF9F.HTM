ISRDialogs Interface

The ISRDialogs interface allows an application to display Windows dialog boxes to configure a speech-recognition engine. The controls and capabilities in these dialog boxes are determined by the particular speech-recognition engine that is being used. All dialogs supported by the engine are modal and calls to the ISRDialogs interface are synchronous. All of the ISRDialogs member functions use the same syntax.

This interface is supported by a speech-recognition engine object. It has these member functions:

AboutDlg

TrainMicDlg

GeneralDlg

TrainGeneralDlg

LexiconDlg
