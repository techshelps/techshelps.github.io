ISRGramInsertionGUI Interface

The ISRGramInsertionGUI interface is used by an application to display a small status window next to the insertion point in a document or edit field. The status window gives the user information about the progress of recognition. The window is hidden until the application specifies otherwise. An engine may or may not support this interface.

The information provided in the status window is defined by the engine developer, but it may include the text of recognitions and hypotheses, a VU meter, indication of the beginning and end of an utterance, and whether the engine is processing. Some engines allow the user to drag the status window away from the insertion point and drop it at another location.

The ISRGramInsertionGUI interface is supported by the speech-recognition engine object. It has these member functions:

Hide

Move

Show
