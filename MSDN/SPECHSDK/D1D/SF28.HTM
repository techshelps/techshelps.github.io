Engine Enumerator

If an application is not satisfied with the result of the ISRFind::Find member function, or if the application requires a list of the available speech-recognition modes, it should retrieve the address of the ISREnum interface on the speech-recognition enumerator object.

If the application has already created the speech-recognition enumerator and has the address of the ISRFind interface, it can call the QueryInterface member function of ISRFind with the IID_ISREnum interface identifier.

If the application has not yet created the speech-recognition enumerator, it can call the CoCreateInstance function with the CLSID_SREnumerator class identifier and the IID_ISREnum interface identifier. This returns the address of the enumerator's ISREnum interface, which includes the standard member functions of the IEnumX interface: Next, Clone, Skip, and Reset. The application can use these functions to examine the SRMODEINFO structures for all of the speech-recognition engines and modes installed on the system.
