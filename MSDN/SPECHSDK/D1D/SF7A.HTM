If an engine supports speech-recognition results objects, it creates a speech-recognition results object each time a phrase is recognized. The engine returns the address of the IUnknown interface on the results object with the corresponding ISRGramNotifySink::PhraseFinish or ISRGramNotifySink::PhraseHypothesis notification for the recognized phrase.

The engine doesn't fully create the results object until the application calls an interface on the object. If the application ignores the notification, the engine releases the results object automatically. If the application calls an interface on the results object — for example, IUnknown::QueryInterface — the engine finishes creating the results object and it is up to the application to release it.
