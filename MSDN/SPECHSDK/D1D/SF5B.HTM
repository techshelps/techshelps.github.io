ISRNotifySink

This interface is a notification sink that is supported by an application.

Member function

Description

AttribChanged

Notifies an application that an engine attribute has changed.

Interference

Notifies the application that the engine cannot recognize speech properly for a known reason.

Sound

Notifies the application that the engine detected a sound (or a very garbled word).

UtteranceBegin

Notifies the application that the engine has detected the beginning of an utterance or sound.

UtteranceEnd

Notifies the application that the utterance has ended.

VUMeter

Notifies the application of the correct VU (Volume Units) level, which indicates the loudness of the digital-audio stream.
