ISRResBasic::Identify

HRESULT Identify(GUID *pgIdentity);

Retrieves a unique identifier for the results object so that an application can identify the results object at a later point.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pgIdentity

[out] Address of a variable that receives the GUID of the results object, which uniquely identifies that object.

Each engine provides its own mechanism for generating a results GUID, probably incorporating the timestamps returned by the ISRResBasic::TimeGet member function (in the qwTimeStampBegin and qwTimeStampEnd parameters) and some other differentiator, such as the time as specified in a Win32 FILETIME structure when the results object was generated.
