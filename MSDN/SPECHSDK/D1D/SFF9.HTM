ISRResSpeaker::Validate

HRESULT Validate(WORDwConfidence);

Notifies a speech-recognition engine that the speaker was correctly recognized.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_NOTENOUGHDATA

wConfidence

[in] Application's level of confidence that it determined the name of the person who actually spoke. This parameter can be one of these values:

Value

Meaning

SRCORCONFIDENCE_SOME

The application is somewhat confident because the user has not indicated that the speaker was recognized incorrectly.

SRCORCONFIDENCE_VERY

The application is very confident because the user explicitly verified that the speaker was recognized correctly.
