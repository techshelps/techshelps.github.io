SRSHARE

typedef struct { // srs
QWORD qwInstanceID;
DWORD dwDeviceID;
SRMODEINFO srModeInfo;
} SRSHARE, *PSRSHARE;

Provides information about a shared engine/audio-source pair.

qwInstanceID

Instance identifier used to identify an instance of sharing an engine/audio-source pair.

dwDeviceID

Device identifier of the audio source.

srModeInfo

SRMODEINFO structure that provides details about the speech-recognition mode.
