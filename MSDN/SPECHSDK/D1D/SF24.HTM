Low-Level Speech Recognition

Why Use the Low-Level API?

The high level speech API (Voice Commands, Voice Text, and Voice Dictation) exist to make speech easy to use, but in the process prevent full control of speech recognition. In the low level API:

The application must pay attention to many more details than with the high level API.

The application can do anything that it wants to so long as the engine can support the feature.

If standard low-level API doesn't provide for a feature, then engines can add onto API.

The engine is in application's process space.

Multiple instances of engines can be run at once.

Audio comes from or goes to any media that the application desire.
