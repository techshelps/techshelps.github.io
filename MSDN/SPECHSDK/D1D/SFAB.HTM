ISRGramCFG::LinkQuery

HRESULT LinkQuery(TCHAR *pszLinkName, BOOL *pfExist);

Indicates whether an imported symbol used by this grammar has been linked to another grammar.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_INVALIDRULE
SRERR_NOTSUPPORTED

pszLinkName

[in] Address of a buffer that contains the name of the link.

pfExist

[out] Address of a variable that receives TRUE if the link exists, or FALSE if it does not.

If the engine does not support links, it should return SRERR_NOTSUPPORTED.
