ISRResCorrection::Correction

HRESULT Correction(PSRPHRASE pSRPhrase, WORD wConfidence);

Indicates to a speech-recognition engine what the speaker actually said to cause a particular recognition result.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_NOTENOUGHDATA
SRERR_NOTSUPPORTED

pSRPhrase

[in] Address of an SRPHRASE structure that contains the phrase that the speaker actually said.

wConfidence

[in] Application's level of confidence that it knows what the speaker actually said. This parameter can be one of these values:

Value

Meaning

SRCORCONFIDENCE_SOME

The application is somewhat confident because the speaker somehow corrected the application without explicitly telling it what the correct result was.

SRCORCONFIDENCE_VERY

The application is very confident because the speaker explicitly informed it of what he or she said.
