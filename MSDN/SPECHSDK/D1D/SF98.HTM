ISRCentral::ModeGet

HRESULT ModeGet(PSRMODEINFO pModeInfo);

Retrieves information about the current speech-recognition mode.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY

pModeInfo

[out] Address of an SRMODEINFO structure that receives information about the current speech-recognition mode.
