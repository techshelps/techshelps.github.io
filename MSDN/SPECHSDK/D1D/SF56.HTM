ISRGramCFG

This interface is supported by the grammar object of a context-free grammar.

Member function

Description

LinkQuery

Indicates whether or not an imported symbol used by this grammar has been linked to another grammar.

ListAppend

Appends new words to an existing list.

ListGet

Gets the list of words that the engine is currently using.

ListQuery

Indicates whether or not a list has had words set.

ListRemove

Removes words from an existing list.

ListSet

Fills in an empty list field in a grammar or replaces an existing list.
