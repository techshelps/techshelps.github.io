ISRGramCommon::Deactivate

HRESULT Deactivate(PTCHAR pszInfo);

Deactivates an active grammar so that the speech-recognition engine no longer uses the grammar to recognize speech.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_INVALIDRULE
SRERR_RULENOTACTIVE

pszInfo

[in] Address of a buffer that contains the name of the rule to deactivate for a context-free grammar. For a dictation or limited-domain grammar, pszInfo is not used and should be NULL. If the rule is already deactivated, pszInfo is ignored. If pszInfo is NULL, the context-free grammar deactivates the rule that has the rule name of "Start."

If the grammar is deactivated during an utterance, recognition might not stop until the utterance is finished. If this is the last active grammar to be deactivated, the engine should notify the audio-source object to stop providing digital-audio data. This saves resources and minimizes processor consumption.
