ISRGramCommon Interface

The ISRGramCommon interface provides control of features that are common to both context-free and dictation grammars. For control of more specific features, see the ISRGramCFG and ISRGramDictation interfaces.

The ISRGramCommon interface is supported by the grammar object. It has these member functions:

Activate

DeteriorationSet

Archive

TrainDlg

BookMark

TrainPhrase

Deactivate

TrainQuery

DeteriorationGet
