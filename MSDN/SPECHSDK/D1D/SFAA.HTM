ISRGramCFG Interface

The ISRGramCFG interface provides control of certain features that are specific to context-free grammars. This interface is not used by dictation grammars. For more general control of context-free and dictation grammars, use the ISRGramCommon interface.

The ISRGramCFG interface is supported by the grammar object. It has these member functions:

ListAppend

ListRemove

ListGet

ListSet

ListQuery
