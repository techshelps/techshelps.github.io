ISRCentral::Resume

HRESULT Resume(void);

Reduces the pause count of a paused speech-recognition engine by 1. When the engine's pause count reaches 0, the engine resumes processing incoming phrases.

· Returns NOERROR if successful, or E_OUTOFMEMORY if a low-memory condition occurs.

An application must call Resume each time it pauses the engine, calls the Pause member function, or encounters a bookmark, and to resume after auto-pause activation.
