ISRAttributes Interface

The ISRAttributes interface allows the user of an engine to control various attributes and parameters of speech recognition. Initially, attribute values are set to an optimal level as defined by the engine. The application can change these values or maintain them.

This interface is supported by the speech-recognition engine object. It has these member functions:

AutoGainEnableGet

RealTimeGet

AutoGainEnableSet

RealTimeSet

EchoGet

SpeakerGet

EchoSet

SpeakerSet

EnergyFloorGet

ThresholdGet

EnergyFloorSet

ThresholdSet

MicrophoneGet

TimeOutGet

MicrophoneSet

TimeOutSet
