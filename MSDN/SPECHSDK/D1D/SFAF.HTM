ISRGramCFG::ListRemove

HRESULT ListRemove(TCHAR *pszListName, SDATA dWord);

Removes words from an existing list.

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_OUTOFMEMORY
SRERR_INVALIDCHAR
SRERR_INVALIDLIST
SRERR_NOTENOUGHDATA
SRERR_NOTSUPPORTED

pszListName

[in] Address of a buffer that contains the name of the list.

dWord

[in] An SDATA structure that contains a sequential list of SRWORD structures. (This is not an array because the size of each structure varies according to its contents.)

If a specified word is not in the list, it is skipped. If the engine does not support lists, it returns SRERR_NOTSUPPORTED.
