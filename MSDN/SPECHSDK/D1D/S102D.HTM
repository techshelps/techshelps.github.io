SRHEADER

typedef struct {
DWORD dwType,
DWORD dwFlags
} SRHEADER, * PSRHEADER;

Defines the header of a grammar.

dwType

Type of grammar. This member can be one of these values:

Value

Meaning

SRHDRTYPE_CFG

Context-free grammar

SRHDRTYPE_DICTATION

Dictation grammar

SRHDRTYPE_LIMITEDDOMAIN

Limited-domain grammar


dwFlags

If the data stored in the grammar is in Unicode, this member is set to SRHDRFLAG_UNICODE. If the flag is not set, the data is stored in ANSI.
