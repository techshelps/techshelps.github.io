ISTPhoneConv Interface

(New for 3.0)

The ISTMicWizard interface allows an application to convert between IPA and other phonemic or phonetic character sets.

Member function

Description

FromIPA

Converts from IPA to the phone set.

Get

Gets the text for converting the current version.

SetByLang

Sets the phone set based upon the language.

SetByText

Sets the phone set based upon a text file.

ToIPA

Converts from the phone set into IPA.
