ISTPhoneConv::Get

HRESULT Get(SDATA *pData);

Gets the text representation of the rules being used.

pData

[out] pData->pData is filled with a pointer to a NULL-terminated Unicode string allocated with CoTaskMemAlloc(). The application should free the memory. pData->dwSize is filled with the number of bytes in the string (including the NULL termination).

· Returns NOERROR if successful, or one of these error values:

E_INVALIDARG
E_FAIL - Buffer is too small.
