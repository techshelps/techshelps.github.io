MICWIZARDINFO structure

typedef struct {
BOOL fHasRunMicWizard;
DWORD dwWaveInDevice;
DWORD dwMicrophoneType;
WCHAR szMicString[64];
} MICWIZARDINFO, *PMICWIZARDINFO;

fHasRunMicWizard

TRUE if the current user has run the microphone wizard before, FALSE if they have not.

dwWaveInDevice

Wave input device number they have chosen to use, of WAVE_MAPPER if unknown.

dwMicrophoneType

Microphone type. This can be one of the following:

Value

Meaning

STMWI_UNKNOWN

Unknown

STMWI_CLOSETALK

Close-talk or headset microphone.

STMWI_EARPIECE

Microphone with an element near the ear or cheek bone.

STMWI_HANDSET

Telephone-style handset.

STMWI_CLIPON

Microphone clips onto shirt.

STMWI_DESKTOP

Microphone sits on desktop.

STMWI_HANDHELD

Microphone that is meant to be held in the user's hand, a few centimeters from his/her face.

STMWI_TOPMONITOR

Microphone that sits on top of the computer monitor.

STMWI_INMONITOR

Microphone built into the monitor.

STMWI_KEYBOARD

Microphone in the keyboard.

STMWI_REMOTE

Microphone in the remote control.


szMicString

String specifying the microphone name. The wizard does not currently obtain this information from the user, but when USB microphones become common it will obtain the information from the USB microphone.
