Phoneme Converter

The phoneme converter converts between Unicode IPA and a more human-readable phoneme set. Applications can use the phoneme converter to display pronunciations to end users.

To run the Microphone Setup Wizard, the application does a CoCreateInstance on the wizard's object, CLSID_STPhoneConv, and queries for the ISTPhoneConv interface. It then calls ISTPhoneConv->SetByLang() to specify the character set, and ISTPhoneConv->To/FromIPA() to convert.

The Microphone Setup Wizard understands only ISTPhoneConv.
