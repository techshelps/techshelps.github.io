Changing Help

The Microphone Setup Wizard ships with a list of mail-order microphones vendors WWW addresses. If you wish a different list, your application should write a new help file and replace $(WINDOWS)\Speech\Speech.hlp and Speech.cnt.

The contexts used are:

232 - Buying a microphone

235 - Adjusting the playback volume

245 - Microphone is too quiet.

246 - Microphone is too loud.
