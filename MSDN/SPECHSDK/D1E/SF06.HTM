ITTSFind Interface

The ITTSFind interface allows an application to search through text-to-speech modes to select a mode that most closely matches the desired characteristics and to create an engine object that represents the selected mode.

Because the ITTSFind interface selects a text-to-speech mode based on the relative importance of various characteristics, it may not always select the optimal mode (just as the Windows font-matching function does not always select the optimal font). The ITTSFind interface is best used to select a "default" text-to-speech mode. An application should also allow the user to select a mode and should store its GUID so the mode can be retrieved later.

The ITTSFind interface for the text-to-speech enumerator uses ITTSFind for all of the registered text-to-speech engines and then selects from the closest result. A text-to-speech engine may also supply an ITTSFind interface. However, if it does not, the ITTSFind interface of the text-to-speech enumerator uses the engine's ITTSEnum interface to simulate this functionality.
