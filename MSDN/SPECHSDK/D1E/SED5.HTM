ITTSEnum

This interface is supported by the text-to-speech enumerator and the engine enumerator.

Member function

Description

Clone

Returns another enumerator containing the same enumeration state as the current one.

Next

Retrieves the specified number of items in the enumeration sequence.

Reset

Resets the enumeration sequence back to the beginning.

Select

Selects a text-to-speech mode and creates an engine object to represent the selected mode.

Skip

Skips a specified number of elements in the enumeration sequence.
