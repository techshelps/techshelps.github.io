ITTSCentral::AudioPause

HRESULT AudioPause(void);

Immediately pauses text-to-speech output for an engine object.

· Returns NOERROR if successful, or the <E_OUTOFMEMORY> error value otherwise.

Calling AudioPause affects output for the current engine object only. When the engine object is first created, text-to-speech output is not paused.

Unless the engine is paused, it immediately plays any text sent to it using the ITTSCentral::TextData member function. To defer playing of text, you should call AudioPause before calling TextData. Any text that is sent to the engine is not played until the application calls the ITTSCentral::AudioResume member function.
