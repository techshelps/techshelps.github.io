ITTSDialogs Interface

The ITTSDialogs interface allows an application to display Windows dialog boxes to configure a text-to-speech engine. The controls and capabilities in these dialog boxes are determined by the particular text-to-speech engine that is being used. All dialog boxes supported by the engine are modal, and calls to the ITTSDialogs interface are synchronous. All of the ITTSDialogs member functions use the same syntax.

The ITTSDialogs interface is supported by the text-to-speech engine object. It has these member functions:
