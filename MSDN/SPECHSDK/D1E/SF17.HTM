Ctx

\Ctx=string\

Sets the context for the text that follows, which determines how symbols are spoken.

· Example: \ctx="Unknown"\

string

String that specifies the context. This parameter can be one of these strings:

Context string

Description

"Address"

Addresses and/or phone numbers

"C"

Code in the C or C++ programming language

"Document"

Text document

"E-Mail"

Electronic mail

"Numbers"

Numbers, dates, times, and so on

"Spreadsheet"

Spreadsheet document

"Unknown"

Context is unknown (default)

"Normal"

Normal/default speech mode.
