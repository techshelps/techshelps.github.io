ITTSCentral Interface

The ITTSCentral interface provides centralized control of a text-to-speech engine, allowing for actions such as text entry and conversion of phonemes.

The ITTSCentral interface is supported by the text-to-speech engine object. It has these member functions:

AudioPause

PosnGet

AudioReset

Register

AudioResume

TextData

Inject

ToFileTime

ModeGet

UnRegister

Phoneme
