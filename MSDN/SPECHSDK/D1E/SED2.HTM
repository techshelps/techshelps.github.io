ITTSBufNotifySink

This interface is a notification sink that is supported by an application.

Member function

Description

Bookmark

Notifies the application that the engine has encountered a bookmark in the text being played.

TextDataDone

Notifies the application that the text in a particular text-to-speech buffer has been sent to the audio-destination object or that the application called the ITTSCentral:AudioReset member function.

TextDataStarted

Notifies the application that a particular text-to-speech buffer has reached the beginning of the text-to-speech buffer queue.

WordPosition

Notifies the application which text is currently being played.
