ITTSEnum Interface

The ITTSEnum interface allows an application to enumerate the different text-to-speech modes and to select a mode for which to create an engine object.

Calling ITTSEnum on an engine enumerator allows an application to recurrently traverse through its list of known text-to-speech modes. Calling ITTSEnum on a text-to-speech enumerator calls engine enumerators for all of the text-to-speech engines on the system.

The ITTSEnum interface is supported by the text-to-speech enumerator and the engine enumerator. It has these member functions:

Clone

Next

Reset

Select

Skip


All OLE enumeration interfaces contain the functions Clone, Next, Reset, and Skip. In the ITTSEnum interface, these functions use TTSMODEINFO structures as the elements in the enumeration sequence. For more information about Clone, Next, Reset, and Skip, see the section, "Common Elements of the Speech Application Programming Interface."
