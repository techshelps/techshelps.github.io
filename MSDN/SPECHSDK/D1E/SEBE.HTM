Engine Enumerator

If an application is not satisfied with the result of the ITTSFind::Find member function, or if the application requires a list of the available text-to-speech modes, it should retrieve the address of the ITTSEnum interface on the text-to-speech enumerator object.

If the application has already created the text-to-speech enumerator and has the address of the ITTSFind interface, it can call the QueryInterface member function of ITTSFind with the IID_ITTSEnum interface identifier.

If the application has not yet created the text-to-speech enumerator, it can call the CoCreateInstance function with the CLSID_TTSEnumerator class identifier and the IID_ITTSEnum interface identifier. This returns the address of the enumerator's ITTSEnum interface, which includes the standard IEnumX member functions Next, Clone, Skip, and Reset. The application can use these member functions to examine the TTSMODEINFO structures for all of the text-to-speech engines and modes installed on the system.
