TTSMODEINFORANK

typedef struct { // ttsmir
DWORD dwEngineID;
DWORD dwMfgName;
DWORD dwProductName;
DWORD dwModeID;
DWORD dwModeName;
DWORD dwLanguage;
DWORD dwDialect;
DWORD dwSpeaker;
DWORD dwStyle;
DWORD dwGender;
DWORD dwAge;
DWORD dwFeatures;
DWORD dwInterfaces;
DWORD dwEngineFeatures;
} TTSMODEINFORANK, *PTTSMODEINFORANK;

Ranks the members of the TTSMODEINFO structure in order of importance.

dwEngineID

Relative importance of a matching engine identifier.

dwMfgName

Relative importance of the specified manufacturer name.

dwProductName

Relative importance of the specified product name.

dwModeID

Relative importance of a matching mode identifier.

dwModeName

Relative importance of the specified mode name.

dwLanguage

Relative importance of the specified language.

dwDialect

Relative importance of the specified dialect. This value is stored in the LANGUAGE structure of TTSMODEINFO.

dwSpeaker

Relative importance of the requested speaker.

dwStyle

Relative importance of the requested speaking style.

dwGender

Relative importance of the requested gender.

dwAge

Relative importance of the requested age. The text-to-speech mode that is closest to the requested age is selected.

dwFeatures

Relative importance of the requested features. Modes that lack the features requested are eliminated from consideration.

dwInterfaces

Relative importance of the requested interfaces. Modes that lack the interfaces requested are eliminated from consideration.

dwEngineFeatures

Relative importance of the requested engine features. Modes that lack the features requested are eliminated from consideration.

Each member of the TTSMODEINFORANK structure contains a priority value for the corresponding feature. Values are relative to each other — that is, a higher value indicates a higher priority, but the value itself is not meaningful.

The ITTSFind::Find member function searches for the text-to-speech mode that most closely matches the attributes specified in TTSMODEINFO, using the rankings of TTSMODEINFORANK to select the mode with the most desirable features.
