ITTSAttributes

This interface is supported by the text-to-speech engine.

Member function

Description

PitchGet

Gets the current baseline pitch for a text-to-speech mode.

PitchSet

Sets the baseline pitch for a text-to-speech mode.

RealTimeGet

Gets the real-time setting for the engine.

RealTimeSet

Sets the new percentage of processor time that the engine expects to use during constant speech.

SpeedGet

Gets the current baseline average talking speed for a text-to-speech mode, in words per minute.

SpeedSet

Sets the baseline average talking speed for a text-to-speech mode, in words per minute.

VolumeGet

Gets the current baseline speaking volume for the text-to-speech mode.

VolumeSet

Sets the baseline speaking volume for a text-to-speech mode.
