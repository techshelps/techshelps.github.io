ITTSAttributes::PitchGet

HRESULT PitchGet(WORD *pwPitch);

Retrieves the current baseline pitch for a text-to-speech mode.

· Returns NOERROR if successful, or one of these error values:

<E_INVALIDARG>
<E_OUTOFMEMORY>
<TTSERR_NOTSUPPORTED>
<TTSERR_VALUEOUTOFRANGE>

pwPitch

[out] Address of a variable that receives the current baseline pitch, in hertz, for a text-to-speech mode.

If the text being spoken contains text-to-speech control tags, the actual pitch of the speaking voice typically fluctuates above this baseline. The Pit tag changes the baseline pitch setting for the mode, the same as when the PitchSet member function is called. The default baseline pitch is the optimal pitch for the mode.
