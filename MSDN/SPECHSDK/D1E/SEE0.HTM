Interfaces

The engine enumerator should support the ITTSEnum interface and may support the ITTSFind interface. The engine enumerator may support other interfaces defined by the engine vendor to provide additional functionality.

Interface

Description

ITTSEnum

Enumerates and selects a text-to-speech mode.

ITTSFind

(Optional) Finds and selects the text-to-speech mode that most closely matches the requested characteristics.


For more information about common enumeration functions, see the section, "Common Elements of the Speech Application Programming Interface."
