The engine enumerator is created when the application calls the ITTSFind::Find member function on a text-to-speech enumerator. The application passes the address of a TTSMODEINFO structure that indicates the desired characteristics of the mode and the address of a TTSMODEINFORANK structure that indicates the relative importance of each characteristic. As an alternative, the application can call ITTSEnum member functions to enumerate the modes.

The text-to-speech enumerator creates an engine enumerator for each engine registered on the system and calls each engine enumerator.
