ITTSAttributes Interface

The ITTSAttributes interface allows the user of an engine object to control various attributes and parameters of text-to-speech. Initially, attribute values are set to an optimal level as defined by the engine. The application can change these values or maintain them.

The ITTSAttributes interface is supported by the text-to-speech engine. It has these member functions:

PitchGet

SpeedGet

PitchSet

SpeedSet

RealTimeGet

VolumeGet

RealTimeSet

VolumeSet
