ITTSCentral::ModeGet

HRESULT ModeGet(PTTSMODEINFO pttsInfo);

Retrieves information about the current text-to-speech mode, such as certain qualities of the voice.

· Returns NOERROR if successful, or one of these error values:

<E_INVALIDARG>
<E_OUTOFMEMORY>

pttsInfo

[out] Address of a TTSMODEINFO structure that receives information about the current text-to-speech mode.
