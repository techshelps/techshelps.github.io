ITTSNotifySink::Visual

HRESULT Visual(QWORD qTimeStamp, TCHAR cIPAPhoneme,
TCHAR cEnginePhoneme, DWORD dwHints, PTTSMOUTH pTTSMouth);

Notifies an application which phoneme is being used in the current digital-audio stream.

· Return value is ignored.

qTimeStamp

[in] Time, in bytes, when the phoneme starts. An application can ignore this parameter.

cIPAPhoneme

[in] IPA phoneme that is being played. To get the IPA phoneme character, you must use the Unicode ('W') version. This parameter is always NULL for the ANSI ('A') version of this function.

cEnginePhoneme

[in] Engine-specific phoneme that is being played.

dwHints

[in] Kind of sentence in which the phoneme is being played. This parameter can be one of these values:

Value

Meaning

TTSNSHINT_COMMAND

The phoneme is part of a command.

TTSNSHINT_EMPHASIS

The phoneme is part of a word being emphasized.

TTSNSHINT_EXCLAMATION

The phoneme is part of an exclamation.

TTSNSHINT_QUESTION

The phoneme is part of a question.

TTSNSHINT_STATEMENT

The phoneme is part of a statement.


pTTSMouth

[in] Address of a TTSMOUTH structure that contains information about the mouth position corresponding to the current phoneme. If the mouth position is unknown or not supported, this parameter is NULL.

An application can use the information returned by Visual to synchronize character animation with spoken text.
