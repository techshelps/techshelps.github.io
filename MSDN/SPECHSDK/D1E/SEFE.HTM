ITTSCentral::UnRegister

HRESULT UnRegister(DWORD dwKey);

Allows an application to release a notification interface through which an engine sent text-to-speech notifications to an application.

· Returns NOERROR if successful, or one of these error values:

<E_INVALIDARG>
<E_OUTOFMEMORY>
<TTSERR_INVALIDKEY>

dwKey

[in] Key value returned by the Register function for the notification interface when the application registered the interface.

An application must call UnRegister before closing the engine object.
