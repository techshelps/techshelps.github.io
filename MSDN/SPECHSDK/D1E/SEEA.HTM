ITTSAttributes::SpeedGet

HRESULT SpeedGet(DWORD *pdwSpeed);

Retrieves the current baseline average speed for a text-to-speech mode.

· Returns NOERROR if successful, or one of these error values:

<E_INVALIDARG>
<E_OUTOFMEMORY>
<TTSERR_NOTSUPPORTED>
<TTSERR_VALUEOUTOFRANGE>

pdwSpeed

[out] Address of a variable that receives the baseline average speed for the voice, in words per minute.

If the text being played contains text-to-speech control tags, the actual speed of the speaking voice fluctuates above and below this baseline. The Spd tag changes the baseline speed setting for the mode, the same as when the SpeedSet member function is called. The default baseline speed is the optimal speed for the mode.
