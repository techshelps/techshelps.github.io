Interfaces

The engine object should support the ITTSAttributes, ITTSCentral, and ITTSDialogs interfaces, and it may also support the ILexPronounce interface. An engine object may support other interfaces defined by the engine vendor to provide additional functionality.

There are the following interfaces.

Interface

Description

ILexPronounce

(Optional) Allows an application to query and control the pronunciation lexicon for a speech-recognition or text-to-speech engine.

ITTSAttributes

Controls the attributes of a text-to-speech engine. Member functions allow an application to adjust the pitch, speed, and volume of the voice and the engine's share of the processor.

ITTSBufNotifySink

Notifies an application of changes to the buffer that contains the text being spoken.

ITTSCentral

Controls an engine object. Member functions allow an application to send text to the engine; inject speech-inflection tags (control tags) into text as it is played; convert Unicode text to a phonemic representation; pause, resume, and reset the audio output; obtain information about the text-to-speech mode; register or release a notification interface; retrieve the time that a byte in the audio stream was played; and convert the time to a Win32 FILETIME structure value.

ITTSDialogs

Displays Windows dialog boxes that allow the user to configure the text-to-speech engine, such as controlling how symbols and currencies are pronounced.

ITTSNotifySink

Notifies an application of engine-specific events related to processing text into speech, such as a change of attributes, the time that audio starts or stops playing, and hints for synchronizing animation with the phoneme that is being spoken.
