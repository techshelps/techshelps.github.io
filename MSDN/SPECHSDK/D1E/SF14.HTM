Conventions

This appendix uses the following typographic conventions.

Example

Description

Chr

Bold type indicates speech-inflection keywords.

string

Italic indicates placeholders for information you supply, such as a character or context string.

[[option]]

Double square brackets indicate items that are optional.

[[option...]]

Three dots (an ellipsis) following an item indicate that more items having the same form may appear.

"C"

Quotation marks are required to delimit strings.
