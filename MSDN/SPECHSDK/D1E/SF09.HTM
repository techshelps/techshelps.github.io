ITTSNotifySink Interface

The ITTSNotifySink interface is used by an engine to return status information to the application. This interface is passed to the engine by a text-to-speech enumerator when it registers the engine. The Visual member function is optional and may or may not be supported by an engine.

The ITTSNotifySink interface is supported by an application. It has these member functions:

AttribChanged

AudioStart

AudioStop

Visual
