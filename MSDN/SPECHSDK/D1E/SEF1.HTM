ITTSBufNotifySink::TextDataStarted

HRESULT TextDataStarted(QWORD qTimeStamp);

Notifies the application that a particular text-to-speech buffer has reached the beginning of the text-to-speech buffer queue.

· Return value is ignored.

qTimeStamp

[in] Time, in bytes, when the engine started reading text from a text-to-speech buffer. An application can ignore this parameter.
