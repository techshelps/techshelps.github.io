The ActiveX Technology Family

ActiveX comprises the following technologies:

Name Role Former Name
ActiveX controls Live objects that can be inserted into Web pages as well as any Visual Basic, Visual C++, Visual FoxPro, Microsoft Access, or other application that is an ActiveX control container OLE controls
ActiveX documents Documents that can be displayed by Web browsers using document viewers DocObjects
ActiveX server framework Lets you provide customized Web pages, using content from a database or customized in other ways Includes the Internet Server API (ISAPI), as well as other server-side extensions
Internet ActiveX controls A set of ActiveX controls for building browser and Internet communications capabilities into your applications  
Internet Data Download Services APIs to access data over the Internet  
Active Scripts VBScript and other scripting languages that connect controls and add interactive functionality to Web pages  
Code Download and Verification Automatic downloading, verification, and installation  
HTML extensions To support controls and scripting  

All of these technologies have one goal: to enable you to produce Internet applications and content that is more beautiful and more interactive than ever before—and to do it using tools and techniques you already know.
