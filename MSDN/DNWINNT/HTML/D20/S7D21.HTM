To start the Messages database from the Windows NT command line

  1. In Program Manager, open the Main program group.
  2. Double-click the Command Prompt icon.
  3. At the command line, change to the directory containing the database, which could be either on your own computer or on a network server.
  4. Type msarn110 winntmsg.mdb

    The Messages startup screen appears within the Microsoft Access main application window.
