System-Information Messages

System-information messages from the Executive provide status information on conditions within a process that should be noted but that will not stop the application from running. All that you can do with these messages is read the information and choose the OK button. If one of the following messages appears frequently while you are working in the same application, contact either the supplier of the application or your system administrator to adjust the configuration of your Windows NT computer.
