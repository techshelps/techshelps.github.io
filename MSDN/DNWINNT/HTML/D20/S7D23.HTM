Running the Database Under Windows NT

Detailed procedures on how to use the Find, Notes, and Print buttons on the Messages form to work in the database are provided in the online Help files that accompany this database. They include information on the following topics:

  • Searching for information in the Messages database
  • Using a variable as a search criterion
  • Initiating a new search
  • Adding and removing notes to the Messages database
  • Setting up to print
  • Printing a database report
