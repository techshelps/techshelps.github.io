Windows NT Files Installed for SNMP

This table describes the files that are installed when you install SNMP.

Filename

Description

DHCPMIB.DLL

DHCP extension-agent DLL. Only installed on DHCP servers. This DLL is available for Windows NT 3.5 and later.

INETMIB1.DLL

Internet MIB II extension-agent DLL

LMMIB2.DLL

LAN Manager MIB 2 extension-agent DLL

MGMTAPI.DLL

SNMP Management API library

MIB.BIN

Binary file containing all of the MIBs

SNMP.EXE

SNMP Agent Service; proxy agent that listens for requests and hands them off to the appropriate SNMP DLL

SNMPTRAP.EXE

Receives SNMP traps and forwards them to MGMTAPI.DLL

WINSMIB.DLL

WINS extension-agent DLL. Only installed on WINS servers. This DLL is available for Windows NT 3.5 and later.
