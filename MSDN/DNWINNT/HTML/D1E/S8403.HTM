Specifying Virtual Printer Memory

You can change the amount of virtual memory that your PostScript printer has available for storing fonts. The PostScript driver uses a default setting recommended by the printer manufacturer for virtual memory.

To adjust the amount of virtual memory for your printer, in the PostScript Printer Setup dialog box, type the amount of virtual memory you want to use in the Printer Memory (kilobytes) box. (Choose the Setup option from the Printer Properties dialog box.) To determine the right value, copy the TESTPS.TXT file (supplied with the Windows NT Resource Kit) to the printer, and enter in this dialog the recommended virtual memory value printed on the resulting page.
