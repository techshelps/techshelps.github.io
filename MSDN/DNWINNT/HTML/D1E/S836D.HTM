Driver Files

Drivers make device independence possible for Windows NT applications, providing the hardware-specific interface between physical devices and Windows NT. The Setup program can install several kinds of drivers for Windows NT, including the following:

  • Comm drivers
  • Mouse drivers
  • Printer drivers
  • Display drivers
  • Multimedia drivers
  • Sound drivers
  • Keyboard drivers
  • Network drivers
  • System drivers

Note Multimedia and printer drivers are optional.
