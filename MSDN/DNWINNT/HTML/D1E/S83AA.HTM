Print Manager Tips

You use Print Manager to create local printers, connect to other servers' printers, install and configure printer drivers, share printers on the network, pause printers and documents, and configure spooler options.

Most of these tasks are straightforward, and you can find the information you need by using the online help or the Windows NT documentation. But a couple of Print Manager issues are a little trickier, and are discussed in the following sections. These issues are deciding which method to use to establish a printer, and determining which printer driver configuration dialog box to use for certain configuration options.
