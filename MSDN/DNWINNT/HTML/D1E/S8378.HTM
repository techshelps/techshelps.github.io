Plotter Driver Files

A plotter is any device used to draw charts, diagrams and other line-based graphics. To install a new or additional plotter driver, double-click the Printers icon in the Control Panel window.

Windows NT includes the following plotter drivers. For more information about printing, see Chapter 6, "Printing."
