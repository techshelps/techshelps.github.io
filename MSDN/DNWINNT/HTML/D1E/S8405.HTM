Microsoft Mail

Windows NT includes an electronic mail application, Microsoft Mail, that can be used to exchange information with other Windows NT computers. Mail also can work interactively with many other Windows-based applications.

This chapter describes the components that make up Mail and explains how to use Mail functionality from within other Windows-based applications, such as Microsoft Word for Windows and Microsoft Excel for Windows. This chapter also describes customization features you can use to tailor Mail for your office, including custom commands, custom message types, custom menus, and the Messaging Application Program Interface (MAPI).
