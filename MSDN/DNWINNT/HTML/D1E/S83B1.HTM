Windows NT Printing Model

Windows NT has a modular printing architecture. The modularity makes Windows NT an ideal printing platform because third-party software developers can meet specialized needs by adding one or two modules, leaving the others in place. The modular structure is generally transparent to people using the system.

This section describes the Windows NT printing model. The main focus is on how jobs are handled when Windows NT acts as a print server, with additional information about how Windows NT acts as a print client.
