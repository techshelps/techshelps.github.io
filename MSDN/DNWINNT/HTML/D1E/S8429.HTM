Schedule+ Interoperability

Schedule+ has a flexible architecture that makes communication with other schedule and calendar systems possible. Developers can modify their schedule and calendar software to share information with Schedule+ by using the Schedule+ Interchange format, described below.

The Schedule+ Interchange format is a text file with an .SCH extension. An interchange file can be imported to Schedule+ with the File/Import Appointments command; a Schedule+ calendar can be exported with the File/Export Appointments command. The file format supports importing and exporting projects, tasks, appointments, meetings, and notes.
