OS/2 Subsystem

The OS/2 subsystem supports OS/2 1.x character-based applications on x86-based computers. This subsystem isn't supported on RISC-based computers; however, OS/2 real-mode applications can run on a RISC-based computer in the MS-DOS environment.

Bound applications, which are applications designed to run under either OS/2 or MS-DOS, will always run in the OS/2 subsystem if it is available.

For more information about using OS/2 1.x applications on Windows NT, see Chapter 16, "OS/2 Compatibility."
