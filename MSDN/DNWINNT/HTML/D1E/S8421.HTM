Microsoft Schedule+

Microsoft Schedule+ is an application that lets you plan and schedule meetings and appointments with others in your workgroup. It works together with Microsoft Mail to perform key functions, such as sending meeting-request messages to other workgroup members.

This chapter describes the components that make up Schedule+ and shows how the application works with Microsoft Mail. It also describes the custom message types used for Schedule+ and includes a section that answers common questions about Schedule+.

For a description of the features of Schedule+ and information about how to use the application, see Chapter 8, "Schedule+" in the Windows NT System Guide.
