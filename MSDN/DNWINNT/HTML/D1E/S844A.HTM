HKEY_LOCAL_MACHINE\SECURITY Subtree

The HKEY_LOCAL_MACHINE\SECURITY subtree contains security information for the local computer, including user rights, password policy, and the membership of local groups, as set in User Manager.

The subkey HKEY_LOCAL_MACHINE\SECURITY\SAM is mapped to HKEY_LOCAL_MACHINE\SAM, so changes made in one automatically appear in the other.

If you want to change global group membership or other security-related items, use User Manager or User Manager For Domains.

Caution The information in this database is in binary format and should not be changed using Registry Editor. Errors in this database may prevent users from being able to log on to the computer.
