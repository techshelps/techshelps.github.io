PostScript Printer Description Files

The Windows NT PostScript driver supports Adobe version 4.0-compatible PostScript printer description files. The files described in this section provide additional PostScript description information for specific printers. To install a new or additional printer driver for your printer, double-click the Printers icon in the Control Panel window.

Windows NT includes the following Adobe PostScript printer description files. For more information about printing, see Chapter 6, "Printing."
