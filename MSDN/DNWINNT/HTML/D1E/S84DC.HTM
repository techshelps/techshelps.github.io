Registry Entries for Device Drivers

The following device drivers have additional value entries that can be specified in the Registry, in addition to changes that you can make using the Devices or Drivers icon in Control Panel. These types of services and drivers appear in the Registry and are described in this section:

  • File system drivers and recognizers
  • Disk, serial, and parallel port devices
  • Keyboard and mouse devices
  • SCSI miniport devices
  • Sound cards
  • Video display devices

Tip You can view current settings for these device drivers by choosing the Devices button in WinMSD.
