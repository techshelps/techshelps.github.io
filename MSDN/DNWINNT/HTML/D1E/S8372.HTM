Introducing Windows NT Sequence Files

Windows NT includes an online introduction, Introducing Windows NT. The introduction demonstrates how to use the built-in networking, security, and workgroup features of Windows NT to make your job easier. To view the online introduction, double-click the Introducing Windows NT icon in the Main program group.
