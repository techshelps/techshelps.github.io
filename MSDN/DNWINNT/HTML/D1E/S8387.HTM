Using the FAT File System with Windows NT

The Windows NT FAT file system works the same as it does with MS-DOS and Windows. In fact, you can install Windows NT on your existing FAT partition.

Note Remember that you cannot use Windows NT with any compression or partitioning software that requires drivers to be loaded by MS-DOS. Rather, you must have Windows NT-specific versions of the drivers to enable Windows NT to read the disk.

You can move or copy files between FAT and NTFS volumes. When you move or copy a file from NTFS to FAT, permissions and alternate streams are lost.
