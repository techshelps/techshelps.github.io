Multimedia and Sound Entries for Users

Values related to user preferences for multimedia items in Control Panel are found in the following Registry path:


HKEY_CURRENT_USER\Control Panel

The information here is for troubleshooting reference. All changes should be made using the Devices, Drivers, MIDI Mapper, and Sound icons in Control Panel. The following lists the default entries. All are REG_SZ data types.

Multimedia subkey

Default value entries

MMCPL

H=230
NumApps=20
W=442
X=88
Y=84

Sound

Beep=yes

Sounds
