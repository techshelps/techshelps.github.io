Connecting Computers to the Internet with RAS

Windows NT Server's Remote Access Service (RAS) can also be used to provide remote clients with Internet gateway. This requires additional communication equipment that allows multiple remote clients to dial into the RAS computer. This type of configuration allows communication to the Internet as illustrated in the following figure.

Figure 20.5 Remote Client Internet Gateway

On small LANs, you can also use RAS and a PPP connection in place of a router. This allows you to connect both LAN clients and RAS clients to the Internet.
