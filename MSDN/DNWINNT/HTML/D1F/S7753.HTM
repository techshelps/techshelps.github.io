Windows NT on the Internet

Now you can use Windows NT to connect to the Internet without having to remember arcane commands. You can even set up an Internet server on a Windows NT computer, taking advantage of Windows NT's scalability, easy interface, and ease of configuration. All you need is Windows NT, the Internet server services that are included in this resource kit, and a connection to the Internet.

Software for many client and server Internet services is also available in the public domain (free) software. In the case of client software especially, more powerful and better-supported commercial programs have been written based on the public domain tools, and are available through the usual software outlets and also through the Internet itself.
