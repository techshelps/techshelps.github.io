Integrating Multiple Internet Services on One Windows NT Computer

You don't need to set up a different server for every Internet service you want to provide. One Windows NT server can provide several different services. You can set up a separate directory (or even a separate partition) for each service, which makes it easier to keep track of the files for the different services. Or, you can keep files for several different services in one directory. The only real restrictions are the resources of the computer: bandwidth, disk size, memory, and so forth.
