Other Useful Tools

We have included on the floppy disk with this book a number of additional tools which you should find handy from time to time when you look at the performance of Windows NT systems and their applications. You'll want to refer to the documentation on the Windows SDK for the full details; here we'll just make sure you know they exist. They are:

  • PView
  • PMon
  • WPerf
  • wt command in the Windbg debugger
