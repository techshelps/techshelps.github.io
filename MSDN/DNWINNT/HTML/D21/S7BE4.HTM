Registry Entries for Network Services

This section describes parameters for Windows NT services under the HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Services subkey.

Some of these services also have configuration information stored under HKEY_LOCAL_MACHINE\SOFTWARE. These values are described in "NetRules Subkey Entries," earlier in this chapter.

Note Wherever possible, choose the Services icon in Control Panel or use Server Manager in Windows NT Advanced Server to change values for these services.
