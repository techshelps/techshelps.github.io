Checklist for USB

1. USB included on PC 98 system

Required

2. All USB hardware complies with USB 1.0 specifications

Required

3. Connections use USB icon

Required

4. Devices and drivers support maximum flexibility of hardware interface options

Recommended

5. USB host controller meets either OpenHCI or UHCI specification

Required

6. USB host controller can wake the system

Required

7. System and devices comply with USB power management requirements

Required

8. USB devices meet requirements in related USB device class specification

Required
