Power Management for Monitors

This section summarizes the specific power management requirements for monitors.

11. Monitor complies with device class power management reference specification
Required

The Display Device Class Power Management Reference Specification, Version 1.0 or higher, provides definitions of the OnNow device power states (D0–D3) for graphics adapters and monitors. The specification also covers device functionality expected in each power state and the possible wake-up event definitions for the class, if any. Monitors must support the D0, D2, and D3 power states. The D1 power state is optional for monitors.
