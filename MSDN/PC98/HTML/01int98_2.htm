Basic PC 98 System Types

The design requirements and recommendations in this document provide guidance for building systems that run the Windows family of operating systems. The system type designated as “Basic PC 98” is not a set of definitions for the minimum, optimal, or best hardware required to run Windows 98 or Windows NT. Instead, this specification describes the hardware that will deliver to end users an optimal level of performance when running Windows-based applications under Windows 98 or Windows NT.
