Resources for Accessibility Design

This section lists some documents and services related to accessibility design.

Microsoft Windows Guidelines for Accessible Software Design

Describes techniques for developing software applications that are usable by people with disabilities. This document is included on the Microsoft Developer Network (MSDN) CD-ROM. To obtain additional copies or for information about other Microsoft products and services for people with disabilities, contact:
