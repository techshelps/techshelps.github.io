Chapter 20: Network Communications

This chapter presents PC 98 requirements and recommendations for network adapters and related technologies.

Tips for Selecting High-Performance Network Adapters. For PC manufacturers who want to select high-performance components, the following are design features to look for in network adapters:

  • Adapter supports bus mastering, especially for use with Pentium Pro processors

  • PCI adapter properly supports higher-level PCI commands for intelligent data transfer

  • Drivers are tuned for 32-bit performance
