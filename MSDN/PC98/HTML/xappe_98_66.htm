UNDI CLEANUP

Op-Code: PXENV_UNDI_CLEANUP
Input: ES:DI points to a t_PXENV_UNDI_CLEANUP parameter structure.
Output: PXENV_EXIT_SUCCESS or PXENV_EXIT_FAILURE will be returned in AX, with the CF set accordingly. The status field in the parameter structure will be set to one of the values represented by the PXENV_STATUS_xxx constants.
Description: This call will uninstall the Interrupt 1Ah hook and will prepare the network adapter driver to be unloaded from memory. This call must be made just before unloading the Universal NIC Driver. The rest of the API will not be available after this call executes.

This service cannot be used in protected mode.
