Installation Service Request

To enter this state, the client must have an IP address. Also, the client must have received one or more extended DHCPOFFER messages and therefore know the IP address of one or more installation servers. The client selects one of these installation servers and sends a DHCPREQUEST message to the server on port 4011. Otherwise the format of this message is the same as an extended DHCPDISCOVER.
