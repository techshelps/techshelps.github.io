GetGValue

The GetGValue macro retrieves an intensity value for the green component of a 32-bit red, green, blue (RGB) value. 

BYTE GetGValue(
  DWORD rgb     // 32-bit RGB value
);

Parameters

rgb
Specifies an RGB color value.

Return Values

The return value is the intensity of the green component of the specified RGB color.

Remarks

The intensity value is in the range 0 through 255.

The GetGValue macro is defined as follows: 

#define GetGValue(rgb)   ((BYTE) (((WORD) (rgb)) >> 8))
