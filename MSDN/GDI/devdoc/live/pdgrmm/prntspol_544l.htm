WaitForPrinterChange

The WaitForPrinterChange function is obsolete. This function is provided only for compatibility with 16-bit versions of Windows. Win32-based applications should use the FindFirstPrinterChangeNotification, FindNextPrinterChangeNotification, and FindClosePrinterChangeNotification functions to monitor the occurrence of requested changes on a printer or print server.when one or more requested changes occur on a printer or print server.
