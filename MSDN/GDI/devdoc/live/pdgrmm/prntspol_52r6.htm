EndPagePrinter

The EndPagePrinter function indicates the end of one page and the beginning of the next page for the specified printer. 

BOOL EndPagePrinter(
  HANDLE hPrinter   // handle to printer object
);

Parameters

hPrinter
Handle to the printer for which the page will be concluded.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.

Remarks

The printer-object handle hPrinter is obtained by calling the OpenPrinter function.

Each page in a print job begins with a StartPagePrinter function call and ends with an EndPagePrinter function call. The data for each page is written to the print file by using the WritePrinter function.
