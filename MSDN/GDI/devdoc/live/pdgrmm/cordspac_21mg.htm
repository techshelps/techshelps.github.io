GetWindowExtEx

This function retrieves the x-extents and y-extents of the window for the specified device context. 

BOOL GetWindowExtEx(
  HDC hdc,        // handle of device context
  LPSIZE lpSize   // address of structure receiving window extents
);

Parameters

hdc
Handle to the device context.
lpSize
Pointer to a SIZE structure. The x- and y-extents in page-space units are placed in this structure.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero.

Windows NT: To get extended error information, call GetLastError.
