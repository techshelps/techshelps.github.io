Test Case: C5.01.00

Title: Verify properties in Object Browser

Platforms: Windows NT 3.51, 4.0, Windows 95
Requirements: Office97 app, OLE control. Container must have Control Toolbox
Sample Apps: Microsoft Word 97, DBGrid Control
Test Category: CONTROL

Overview:

These are steps on verifying control properties and methods using the Object Browser.

Steps:

  • Start Office97 app
  • Create new document
  • Choose Tools Macro Visual Basic Editor (Alt+F11)
  • Inside the Visual Basic Editor, choose View Object Browser (F2)
  • Search and find control to verify (note: this only works with registered controls)
  • Verify classes and members of control

 

Result:

All methods and properties that should be visible are available in the Object Browser.

Summary:

The Object Browser tool is a utility to help verify available methods and properties in a control.
