Test Case: C4.01.00

Title: Add Reference from TLB, OLB, DLL file

Platforms: Windows NT 3.51, 4.0, Windows 95
Requirements: Office97 app, OLE control. Container must have Control Toolbox
Sample Apps: Microsoft Word 97, DBGrid Control
Test Category: CONTROL

Overview:

This test verifies that the control can referenced from the VBA/VBA environment from the type library.

Steps:

  • Start Office97 app
  • Create new document
  • Choose Tools Macro Visual Basic Editor (Alt+F11)
  • Inside the Visual Basic Editor, choose Tools References
  • Drop down file filter and choose ActiveX controls (*.TLB, OLB, DLL)
  • Browse to find TLB, OLB, DLL file
  • Click OK to add reference

 

Result:

Reference will be added to Visual Basic environment and to the document. Verify refernce in tools references dialog. Use the Object Browser (F2 in VBE) to verify the existing of the control in the document.

Summary:

This test verifies that a reference can be added to the document from the TLB, OLB, DLL file..
