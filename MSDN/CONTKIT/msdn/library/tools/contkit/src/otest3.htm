Test Case: C2.01.00

Title: Adding Controls: Inline (Microsoft Word)

Platforms: Windows NT 3.51, 4.0, Windows 95
Requirements: Office97 app, OLE control. Container must have Control Toolbox
Sample Apps: Microsoft Word 97, DBGrid Control
Test Category: CONTROL

Overview:

This test verifies that the control can be added inline in a Word document via the control toolbox. Note that the control must be registered before it can be added.

Steps:

  • Start Microsoft Word 97
  • Create new document
  • Type some text
  • Position the cursor in between characters
  • Choose View Toolbars and ensure Control Toolbox is visible
  • Click on bottom right button for "More Controls"
  • Scroll to find the control to Add
  • Click the control to add while holding the shift key.

 

Result:

Control will be added to document in line (in between text characters).

Summary:

This test verifies that a control can be added to an Word 97 document inline using the control toolbox.
