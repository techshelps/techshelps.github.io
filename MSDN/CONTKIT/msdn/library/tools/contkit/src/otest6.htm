Test Case: C2.05.00

Title: Adding Controls: Verifying Missing License

Platforms: Windows NT 3.51, 4.0, Windows 95
Requirements: Office97 app, OLE control. Container must have Control Toolbox
Sample Apps: Microsoft Word 97, DBGrid Control
Test Category: CONTROL

Overview:

This test verifies that a control that requires a license will behave correctly when license information is missing.

Steps:

  • Start Office97 app
  • Create new document
  • Choose View Toolbars and ensure Control Toolbox is visible
  • Click on bottom right button for "More Controls"
  • Scroll to find the control to Add
  • Click the control that requires license.

Result:

Error Message, depending on application

Application

Invalid License Error Message
Excel Cannot Insert Object
Word The server application, source file, or item cannot be found. Make sure the application is properly installed, and that it has not been deleted, moved, or renamed.
Powerpoint License Information for this component cannot be found. You do not have an appropriate license to use this functionality in the design environment.
Access you don't have the license required to use this ActiveX control.... The control could not be created because it is not properly licensed.

Summary:

This test verifies that a control that requires a license fails appropriately when license information is not present.
