Object Browser

Once you have inserted your control on a form or report, Access creates a reference in the VBA project to your control's type library. If you haven't inserted your control onto any form or report in the database project, you can explicitly set a reference to the control. Design any module, and open the references dialog using the Tools | References menu item. From any code window you can invoke the Object browser using F2 or the appropriate menus. The control's type library will be listed in the Project\Library dropdown at the top of this window. Ensure that only the correct members are listed that you wish to expose to the user. Also, if you have specified any context sensitive help for these members, verify that the correct text is displayed when a member is selected and that the help topic links (if supplied) appear and jump to the help file successfully and with the correct context ID.
