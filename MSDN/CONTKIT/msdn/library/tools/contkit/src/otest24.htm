Test Case: C8.02.00

Title: InPlace Activation of Document with control

Platforms: Windows NT 3.51, 4.0, Windows 95
Requirements: Office97 app, OLE control.
Sample Apps: Microsoft Office97 application, DBGrid Control
Test Category: CONTROL

Overview:

This test covers the ability to view or use a control while the application is in place active.

Note: ActiveX controls in an Excel worksheet will be disabled when Excel is in-place active, i.e., when Excel is activated in-situ as an OLE embedding inside containers like Word or as an ActiveX document inside Internet Explorer or in Office Binder.

Steps:

  • Start Office97 app
  • Create new document
  • Choose View Toolbars and ensure Control Toolbox is visible
  • Click on bottom right button for "More Controls"
  • Add control to the document/slide/worksheet
  • Choose Insert Object and insert another object type that supports inplace activation (Word, Excel)
  • Verify properties and control

Summary:

This test verifies that the control can be manipulated while inplace active. Note: If control is added to embedded object, presentation is blank when embedded object is deactivated.
