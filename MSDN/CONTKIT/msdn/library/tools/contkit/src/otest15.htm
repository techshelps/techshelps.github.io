Test Case: C6.00.00

Title: Verify AboutBox in Control Toolbox

Platforms: Windows NT 3.51, 4.0, Windows 95
Requirements: Office97 app, OLE control. Container must have Control Toolbox
Sample Apps: Microsoft Word 97, DBGrid Control
Test Category: CONTROL

Overview:

These are steps on verifying control properties and methods using the Control Toolbox

Steps:

  • Start Office97 app
  • Create new document
  • Add a control
  • right click and select Properties
  • Click About button

 

Result:

AboutBox stock method will be called on control

Summary:

This tests whether the Office application can invoke the AboutBox method on the control.
