Building ActiveX Controls for Internet Explorer 4.0

This document covers features of Microsoft® Internet Explorer 4.0 that an ActiveX™ Control writer should take into account when targeting Internet Explorer 4.0 as a container. Some of the material in this document discusses existing technologies that have increased relevance with Internet Explorer 4.0; other sections discuss new technologies and services that are available to a control in Internet Explorer 4.0. For an in-depth description of these technologies and reference documentation, the Internet Client SDK is the best source of information. It is assumed that the reader is already familiar with ActiveX Controls and associated COM technologies.
