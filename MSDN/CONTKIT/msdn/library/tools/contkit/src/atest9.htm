Invisible-At-Runtime Controls

These controls typically show some design-time representation of the control on the design surface, but don't allow re-sizing the control. Ensure that your control responds correctly in both the case where the user tries to resize the control using the mouse a dragging the grab handles of the bounding rectangle around the control, and by setting the Height and Width properties in the merged property sheet for the extender object. These properties can be found on the Format tab of the property sheet. Regardless of how the user attempts to modify the height and width properties, this type of control should "snap back" to its original size.
