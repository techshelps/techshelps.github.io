Test Case: C2.02.00

Title: Adding Controls: Floating (Microsoft Word)

Platforms: Windows NT 3.51, 4.0, Windows 95
Requirements: Office97 app, OLE control. Container must have Control Toolbox
Sample Apps: Microsoft Word 97, DBGrid Control
Test Category: CONTROL

Overview:

THIS IS THE SAME AS THE STANDARD CASE FOR WORD:

This test verifies that the control can be added via the control toolbox. Note that the control must be registered before it can be added.

Steps:

  • Start Office97 app
  • Create new document
  • Choose View Toolbars and ensure Control Toolbox is visible
  • Click on bottom right button for "More Controls"
  • Scroll to find the control to Add
  • Click the control to add

 

Result:

Control will be added to document/slide/sheet floating above text. Drag the control to verify that it is floating.

Summary:

This test verifies that a control can be added to a Word 97 document floating in the text using the control toolbox.
