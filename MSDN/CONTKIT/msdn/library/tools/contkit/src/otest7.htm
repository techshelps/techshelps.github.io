These tests relate to controls that have the 'Insertable' key present in the registry. Note also that Office97 does not recognize Component Categories.

Test Case: C3.00.00

Title: Insertable Controls: Standard

Platforms: Windows NT 3.51, 4.0, Windows 95
Requirements: Office97 app, OLE control. Container must have Control Toolbox
Sample Apps: Microsoft Word 97, DBGrid Control
Test Category: CONTROL

Overview:

This test verifies that the control can be added via the control toolbox. Note that the control must be registered before it can be added.

Steps:

  • Start Office97 app
  • Create new document
  • Choose Insert Object
  • Select the appropriate control
  • Click OK

Result:

Control will be added to document/slide/sheet.

Summary:

This test verifies that an Insertable control can be added to an Office97 document using the Insert Object dialog.
