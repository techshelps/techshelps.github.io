Test Case: C8.00.00

Title: Object Presentation in different views

Platforms: Windows NT 3.51, 4.0, Windows 95
Requirements: Office97 app, OLE control
Sample Apps: Microsoft Office97 application, DBGrid Control
Test Category: CONTROL

Overview:

This test covers presentation of the control in multiple views. Note that Word, Excel, and Powerpoint do not support IViewObject and therefore will have unpredictable results in Window New Window and the splitter pane. Access97 does support this interface and will not have the problems.

Steps:

  • Start Office97 app
  • Create new document
  • Choose View Toolbars and ensure Control Toolbox is visible
  • Click on bottom right button for "More Controls"
  • Add control to the document/slide/worksheet
  • Choose View and select each option, then verify control presentation. Options will vary depending upon the application.

Summary:

This test verifies that the control is displayed correctly in multiple views of the document.
