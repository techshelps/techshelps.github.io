Microsoft NetShow Theater Server Administration SDK

Microsoft® NetShow™ Theater exposes the Server Administration SDK to facilitate the development of high bandwidth multimedia streaming applications. Using this SDK, it is a simple task to write Web- or Microsoft Windows®-based applications to take advantage of the various NetShow Theater services and to interact with its server components for performing an assortment of tasks, including:

  • Configuring and administering the NetShow Theater servers.
  • Monitoring multiple NetShow Theater server configurations.
  • Monitoring events triggered by a NetShow Theater server.
  • Managing lists of content titles.
  • Managing client connections.

In addition, the SDK allows you to

  • Customize user interfaces for administering the NetShow Theater server. You can automate the UI to fit the needs of novice users, or enable complex functionality for advanced users.
  • Integrate NetShow Theater server streaming capabilities with other applications.
  • Create customized administration and content management solutions for your targeted business: for example, corporate training courses, hospitality, and entertainment services for hotels, cruise ships, and airlines.

The NetShow Theater Server Administration SDK documentation is organized by the Microsoft ActiveX® controls it exposes. For each control a brief introduction is first presented to illustrate how to use the SDK programmatically. The discussion is then followed by a detailed presentation of the related API references. Samples and code excerpts are provided to demonstrate the techniques used to create applications built on the NetShow Theater platform.

Note: The term Theater Server refers to one or more computers, incorporating the title server and up to fourteen content servers. It is also referred to as the Video Server.
