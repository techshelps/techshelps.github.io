SourceLink Property

Returns a String specifying the path to the current content file.

Syntax

MediaPlayer.SourceLink

Remarks

Multiple presentation files (clips) can be referenced in one redirector (.asx) file. Use the SourceLink property to determine where the current clip file resides.

This property does not return valid information if FileName has not been set.
