Client Properties

Property Access Description
ID R Returns a Long value specifying the NetShow Theater client identifier.
IPAddress R Returns a String value specifying the IP address of the client.
Status R Returns a Long value specifying the status of the client.
Subscriber R Returns a String value specifying the name of the host computer of the client.
TitleName R Returns a String value specifying the title that the client is viewing.
