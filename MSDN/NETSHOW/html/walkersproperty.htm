Walkers Property

The MediaServer.Walkers property returns a Walkers collection object of the installed title walkers in a NetShow Theater server.

Syntax

MediaServer.Walkers

Example

Sub enumWalkers(server As String)
    Dim walker As IMSrvWalkerEntry
    'NOTE: you must connect to a NetShow Theater Server first
    ‘      in order to enumerate walkers.
    MediaServer1.SetServer server
    For Each walker In MediaServer1.Walkers
        MsgBox walker.Extensions
    Next
End Sub
