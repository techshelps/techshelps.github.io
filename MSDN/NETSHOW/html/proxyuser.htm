ProxyUser Property

Returns a String value specifying the user ID that is used for proxy authentication.

Syntax

NSUnicastMgr.ProxyUser

Remarks

NetShow supports HTTP stream distribution among NetShow servers. For example, the server connected to this instance of the Unicast Manager control could be distributing streams to multiple downstream servers. It may try, using HTTP, to connect to a server behind a firewall. This property returns the username used if this server is challenged by a proxy server.

See Also
