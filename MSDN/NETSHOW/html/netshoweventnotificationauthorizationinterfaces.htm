NetShow Event Notification and Authorization Interfaces

The interface descriptions in the reference section of this SDK specify the requirements for COM-compliant interfaces that must be implemented by the developer, except for INSSEventAuthorizationCallback, which is implemented by the NetShow server. All methods are listed in Vtable order. The NetShow Event Notification and Authorization interfaces include the following;

  • INSSEventAuthorization
  • INSSEventHandler
  • INSSEventAuthorizationCallback
