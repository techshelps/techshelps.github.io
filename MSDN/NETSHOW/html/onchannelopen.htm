OnChannelOpen Event

This event is fired when a station is opened.

Syntax

NSChannelMgr_OnChannelOpen( Station )

Parameters

Station

A Channel object.

Example

NSChannelMgr_OnChannelOpen Station

Example

Sub NSChannelMgr_OnChannelOpen( TheStation )
  MsgBox "Station " & TheStation.Name & " opened"
End Sub
