FastReverse Method

Scans rapidly backward through the current presentation.

Syntax

MediaPlayer.FastForward

Parameters

This method has no parameters.

Remarks

This method performs the same action as holding down the Rewind button on the playbar. Fast reverse can also be achieved by setting the Rate property to a negative value and playing the title.
