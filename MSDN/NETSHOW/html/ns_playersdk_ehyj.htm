FastForward Method

Scans rapidly forward through the current presentation.

Syntax

MediaPlayer.FastForward

Parameters

This method has no parameters.

Remarks

This method performs the same action as holding down the FastForward button on the playbar. Fast forward can also be achieved by setting the Rate property to a higher value and playing the title.
