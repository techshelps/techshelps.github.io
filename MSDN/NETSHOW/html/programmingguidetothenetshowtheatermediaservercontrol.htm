Programming Guide to the NetShow Theater MediaServer Control

The MediaServer control exposes properties and methods, and supports events used to perform content management and administration tasks on title and content servers. MediaServer control functionality includes:

  • Managing and monitoring client connections to the Title Server.
  • Supervising connections between the Title Server and Content Servers.
  • Managing Content Server disk drives.
  • Managing NetShow Theater titles.
  • Setting bandwidth and bit rate configurations.
