NetShow Event Notification and Authorization API

[This is preliminary documentation and subject to change.]

The Microsoft® NetShow™ Unicast Server provides an API for developing event notification and authorization plug-in components. These components extend the NetShow server providing tools to authorize content playing, server state monitoring, and transaction auditing.

In this portion of the SDK, we present a brief overview of authorization and the general foundation of the NetShow Event Notification and Authorization components. We then describe how to use the Event Notification and Authorization API to create these components.

The discussion is divided into the following topics.

Subject Description
About the Event Notification and Authorization API This section introduces the general concept of authorization, and discusses the architecture of an event notification and authorization component.
Using the Event Notification and Authorization API This section describes how the NetShow server interacts with the event handling and authorization component, and how you register the componnet in the system registry.
Event Notification and Authorization Reference This section describes the specifications for implementing the COM interfaces of the notification and authorization component.
Event Notification and Authorization API Samples This section discusses three  sample applications that illustrate how to develop NetShow authentication event notification and authorization plug-in components to implement pay per view and pay per minute business plans, and perform membership validation.
