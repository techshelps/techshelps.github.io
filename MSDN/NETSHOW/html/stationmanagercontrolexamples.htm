Station Manager Control Examples

Simple_Example A simple example showing how to use the Station Manager Control to list stations.
Multicast ASF A more detailed example that shows how to use the Station Manager Control to create a station and start a multicast stream based on an .asf file. See the Receive Multicast ASF sample in the Media Player SDK for an example of how to receive a multicast by using an .asx file.

These examples illustrate how to use HTML and Visual Basic Scripting Edition, with attributes exposed by Microsoft NetShow. They may not be fully functional within the Platform SDK framework. To download the complete suite of NetShow products, which includes working samples, visit the NetShow web site at (http://www.microsoft.com/NetShow)
