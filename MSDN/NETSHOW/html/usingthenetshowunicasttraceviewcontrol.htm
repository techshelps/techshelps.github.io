Using the NetShow Unicast TraceView Control

You can program the TraceView control to perform the following functions:

  • To pause and resume event monitoring.
  • To clear the display of the event logging.
  • To select event types for monitoring. The selection includes the Server, Client, Admin, and/or Alert events.

To enable the TraceView control embedded in your applications, you must initialize the control by calling its SetServer method. This connects the TraceView control to the server and lets the control start to receive events for display.

The CLSID of NetShow Unicast TraceView is 24DCD5F2-6DBB-11D1-BEF5-00805FBE84A6
