EnableProxy Property

Returns or sets a Boolean value indicating whether connecting to a proxy server is enabled.

Syntax

NSUnicastMgr.EnableProxy

Remarks

NetShow supports HTTP stream distribution among NetShow servers. For example, the server connected to this instance of the Station Manager control could be distributing streams to multiple downstream servers. It may try, using HTTP, to connect to a server behind a firewall. This property is used to set or return a value that determines whether or not this server can connect to a proxy server.
