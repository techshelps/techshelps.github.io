Properties for Station Manager Control

The Station Manager Control supports the following properties:

Property Read/Write Description
Channels R Gets a Channels object.
HostAddresses R Gets a HostAddresses object.
Clients R Gets a Clients object.
Streams R Gets a Streams object.
Server R Gets the name of the server for the connection.
EnableHttpStreaming RW Enables or disables Http streaming.
EnableProxy RW Enables or disables the proxy server.
ProxyHostName RW Returns the proxy host name.
ProxyUser R Returns the user ID used for proxy authentication.
