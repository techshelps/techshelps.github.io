An example version 3.0 ASX file containing relative links
<ASX version = "3.0">
   <Banner href = "graphics/logo1.jpg">
   <MoreInfo href = "category1/category1.htm" />
   </Banner>
   <Entry>
      <Ref href = "mms://NSserver/myfile.asf" />
   </Entry>
</ASX>

In these examples, the first ASX file, the file in version 1.0 syntax, can exist anywhere accessible to the user. There must be a link on a Web page that points to this ASX file so the user can click on it. The version 3.0 ASX file must exist in the metafiles virtual directory on the Web server named mywebserver. The file logo1.jpg must be in a subdirectory named graphics of the physical directory which is defined as the metafiles virtual directory. The Web document category1.htm must reside in a subdirectory to the metafiles directory named category1.
