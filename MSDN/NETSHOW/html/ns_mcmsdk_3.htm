Client Object Reference

The Client object contains methods and properties for each client.

The Client object has the following method:

  • Disconnect

The Client object has the following properties:

Property Read/Write Description
Address R Returns the address of the Client object.
ChannelID R Returns the station ID of the Client object.
DeliveryMode R Returns the delivery mode of the Client object.
Id R Returns the ID of the Client object.
