ServerEvents Property

A Boolean value specifying whether server events are enabled or disabled for this instance of the Netshow Unicast Manager Admin control.

Syntax

NSUnicastMgr.ServerEvents

Remarks

The default value for ServerEvents is True. Server events are OnServerOffline, OnServerOnline, and OnEventBacklogReached.

Example

If NSUnicastMgr.ServerEvents = False Then
  MsgBox "Server events are disabled"
End If

See Also

OnServerOffline, OnServerOnline
