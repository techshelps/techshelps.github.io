VirtualRoot.DirectoryPath Property

Returns the virtual root directory path on the server.

Syntax

NSUnicastMgr.VirtualRoot.DirectoryPath

Example

' display a list of all virtual root directory paths
NSUnicastMgr.Connect "LocalHost"
For i = 0 to NSUnicastMgr.VirtualRoots.Count
   Document.Write NSUnicastMgr.VirtualRoots( i ).DirectoryPath, "<BR>"
