INSSEventAuthorization

In addition to the methods implemented by IUnknown, the following methods must be implemented for the INSSEventAuthorization interface;

Method Description
GetAuthorizedEvents Specifies the events that will trigger calls to the AuthorizeEvent method.
AuthorizeEvent Sent when a specified event is requested.
