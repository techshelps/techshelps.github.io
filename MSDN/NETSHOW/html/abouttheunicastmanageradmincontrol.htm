About the Netshow Unicast Manager Admin Control

The NetShow Unicast Manager Admin control is an ActiveX control used to administer a NetShow unicast server. It manages a set of objects for providing various services. For example, the Client object represents a client connected to the NetShow server and the Clients object represents a collection of the Client objects. The AuthenticationPlugin object authenticates clients for the server.

At present, the NetShow Unicast Server can only run on computers running Microsoft Windows NT® Workstation or Microsoft Windows NT Server version 4.0 operating systems. However, the NetShow Unicast Manager admin control can run on computers running either Microsoft Windows NT 4.0 or Microsoft Windows® 95. For more information about NetShow, visit the NetShow home page at http://www.microsoft.com/netshow/.
