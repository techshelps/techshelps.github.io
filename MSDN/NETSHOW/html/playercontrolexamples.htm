Player Control Examples

Sample Description
VB Script Sample This sample demontrates embedding and control of the player in a simple HTML page. Compare the VB Script in this sample with the next sample, which accomplishes the same this using JScript.
JScript Sample This sample demontrates embedding and control of the player in a simple HTML page. Compare the JScript in this sample with the next sample, which accomplishes the same this using VB Script.
Full Sample This sample builds on the concepts demonstrated in the simple sample. It uses text boxes to display some of the title properties. It also captures state change events and uses them to display the state of the control. A "timer" object is incorporated to update text boxes with packet status information every two seconds.

These examples illustrate how to use HTML and Visual Basic® Scripting Edition, with attributes exposed by Microsoft Media Player. They may not be fully functional within the Platform SDK framework.
