Distributing Theater Server Administration Applications

Distributing applications created by using this SDK requires loading and registering the following files on the target computer:

  • NSTAdmin.ocx
  • Msrvobj.dll
  • Mmsprxy.exe
  • Mmsprxyps.dll
  • Nslstctl.dll
  • Any walker (file conversion utility) files you wish to use.

These files are installed and registered during SDK setup. By default, they are located in the Program Files\Microsoft NetShow\Theater\Admin folder.

Your setup program should copy these files to the computer running your application, and register the DLLs by using the Regsvr32.exe utility. For Mmsprxy.exe, however, you should use the “Mmsprxy.exe /RegServer” command to register the application.
