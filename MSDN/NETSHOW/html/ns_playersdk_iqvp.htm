ChannelURL Property

Returns a String value of the location of the station metafile.

Syntax

MediaPlayer.ChannelURL

Remarks

Remarks

This property only pertains to ASF content being multicast.

The station URL is the Uniform Resource Locator to the NetShow station (.nsc) file. The station URL follows the syntax of typical URLs, except that it can also include the version number, which is separated from the rest of a URL with a # (pound sign). For example, a station URL might be:

http://myserver/myfile.nsc#54

where 54 is the version desired. If the version of the .nsc file is on the local computer, a new .nsc file is not downloaded.
