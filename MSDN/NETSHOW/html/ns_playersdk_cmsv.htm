Volume Property

Returns or sets a Long value of the volume, in hundredths of decibels.

Syntax

MediaPlayer.Volume

Remarks

The value ranges from –10,000 to 0 (zero). Zero specifies full volume and –10,000 specifies no volume. The default value for this property is -600.

Not all devices support 10,000 discrete steps of volume.
