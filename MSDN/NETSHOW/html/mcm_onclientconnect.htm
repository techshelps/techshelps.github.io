OnClientConnect Event

This event is fired when a client is connected.

Syntax

NSChannelMgr_OnClientConnect( Client )

Parameters

Client

A Client object.

Example

Sub NSChannelMgr_OnClientConnect( Client )
  MsgBox "Client at " & Client.Address & "connected"
End Sub
