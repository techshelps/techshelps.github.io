Programming Guide to the NetShow Theater View Controls

The NetShow Theater Administration View controls provide an easy-to-use interface for monitoring NetShow Theater Server activity. These controls have a graphical user interface and support built in features, such as drag-and-drop functionality, for end users. They also expose methods and properties and support events for programmatically manipulating the controls. The View controls include:

ServerView Control

The ServerView control is a tree view control managing one or more title servers, their content servers and content disks. Expanding each top node of the tree displays a clients icon, specifying the number of clients connected to the title server, and an icon for each content server managed by the title server. Expanding each content server node displays the disks managed by that content server.

ServerView also supports the following functions, which are accessed by right-clicking a selection in the control:

  • Connecting and disconnecting title servers.

  • Taking content disks on and off line, as well as rebuilding content disks and pausing their operation.

  • Accessing property sheets used to manage and configure titles, servers, clients, and logging.
ClientView Control

The ClientView control is a control that displays the title being played, the TCP/IP address, status, and ID number for each client connected to the server. ClientView supports the following functions, which are accessed by right clicking on a selected client:

  • Disconnecting the selected client.

  • Accessing a property sheet containing specific information about the selected client.
TitleView Control

The TitleView control is a control that displays information about each of the titles available on the server. Information displayed on the TitleView control includes name, size, creation date, length, author, copyright, and attributes associated with each title. Functions are accessed by right-clicking the control include:

  • Adding, deleting, and renaming titles.

  • Playing selected titles.

  • Selecting the fields to be displayed by the control.

  • Accessing a property sheet with additional information about the title.
TraceView Control

The TraceView control is a control that displays a list of events fired by the NetShow Theater server. TraceView displays the date and time the event was triggered and a description of the event. Functions are accessed by right-clicking the control include:

  • Pausing and resuming the trace.

  • Clearing all entries from the control.

  • Access to a property sheet that allows the user to select the type of events listed in the control. Events are categorized into the following types: Server, Client, Admin, and Alert.
