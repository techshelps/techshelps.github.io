Important
  • The default value for the Value attribute is inherited from the ASX element. If there is no PreviewDuration defined for the ASX element, the value inherited from the ASX element is 10 seconds.
  • When in Preview Mode, the client renders a clip from the beginning of the clip. If there is a StartTime or StartMarker element for this clip, the client renders the clip in Preview Mode starting at the point defined by one of these element.
  • Clips stop normally if they are shorter than the Value of the PreviewDuration element.
