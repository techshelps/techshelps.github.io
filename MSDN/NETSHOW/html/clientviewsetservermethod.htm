ClientView.SetServer Method

The ClientView.SetServer method creates a connection to a NetShow Theater server and monitors the events from the proxy server.

Syntax

ClientView.SetServer( servername [, eventproxy] )

Parameters

servername

A String of the title server to connect to.

eventproxy

An optional String value specifying the name of the server where the NetShow Monitor Service is running
