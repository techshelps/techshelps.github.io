Stream.Stop Method

Stops a stream.

Syntax

NSChannelMgr.Streams(index).Stop

Return Values

Returns an ActiveX HRESULT error code.

Example

Dim Streams
Set Streams = NSChannelMgr.Streams
' stop first stream on server
Stream(0).Stop
