MaxBandwidth Property

The MediaServer.MaxBandwidth property returns or sets a Long value specifying the currently configured maximum aggregate bandwidth, in bits per second, for a particular NetShow Theater system.

Syntax

MediaServer.MaxBandwidth

See Also

MaxTitleBitRate, MinTitleBitRate, MaxPhysicalBandwidth, OnMaxBandwidth
