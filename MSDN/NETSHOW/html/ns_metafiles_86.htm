Attributes

Href (This is a required attribute)

Specifies the URL to a graphic file associated with an ASX or Entry element. The graphic file is displayed in the client next to the information defined by the associated element.

Style (This is a required attribute)

Specifies how the graphic file logo is displayed by the player.

ICON

The graphic specified with the “ICON” style replaces the icon displayed on the Display panel next to the Show or Clip title.

If the value of the Style attribute is “ICON,” the graphic specified in the Href attribute is displayed next to the title of the show or clip. If the parent element of the Logo element is ASX, the Logo graphic is positioned to the left of the Show title in the Display panel. If the parent element is Entry, the Logo graphic is positioned to the left of the Clip title.

MARK

The “MARK” style is only valid if the parent element of the Logo element is ASX. The “MARK” style graphic is displayed in the lower right corner of the Video area (allowing for a six pixel border) while the player is connecting to a server and opening a piece of content. The “MARK” style graphic is not displayed once the title has started playing.

The default value for this attribute is “ICON.”
