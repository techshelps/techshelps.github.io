Using the View Controls

This section of the SDK demonstrates how, in just a few simple steps, you can create administration and player applications by incorporating the ClientView, ServerView, TitleView, and TraceView controls.

The procedures in the Simple Administration Application work for Microsoft Visual Basic 5.0. However, the concepts can also be easily adapted for Web-based administration applications that uses HTML and script languages. The Simple Player Application illustrates the techniques used to embed a TitleView control and the Windows Media Player control in a Web page.
