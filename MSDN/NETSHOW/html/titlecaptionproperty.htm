Title.Caption Property

The Title.Caption property returns or sets a String of the title caption.

Syntax

Title.Caption

Remarks

An example of this property is “Waiting for Guffman.” The Description property can contain any other identifying information and the Name property is used to identify the title on the server, such as “WaitGuff.vod.”
