Stream.GetID Method

Gets a stream ID, in the form of a unique long integer.

Syntax

NSChannelMgr.Streams(index).GetID

Return Values

Returns a stream ID, as a Long value.

Example

Dim Streams
Set Streams = NSChannelMgr.Streams
Stream1 = Streams(0)
MsgBox "First stream ID: " & Stream1.GetID
