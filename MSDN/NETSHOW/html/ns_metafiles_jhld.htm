Working with ASX Files

An ASX file can be sent in e-mail, used as a link reference on a Web page, created dynamically using Active Server Pages (ASP), or exist as a stand-alone file on a hard disk drive. An ASX file, in its simplest form, is a link to an Active Streaming Format (ASF or .asf) file or stream. When a user opens an ASX file, Microsoft® Media Player is started. The player then attempts to open the file or stream defined in the ASX file. An ASX file also can be used to provide the user with a link to another ASX file, to an Active Server Page, or to a NetShow station (NSC) file. An NSC file is used to define real-time content, or NetShow stations to Media Player.
