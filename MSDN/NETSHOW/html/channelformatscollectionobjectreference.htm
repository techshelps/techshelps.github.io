ChannelFormats Collection Object Reference

The ChannelFormats collection object is a collection of ChannelFormat objects.

The ChannelFormats collection object has the following method:

Methods Description
Add Adds a ChannelFormat object to a Channel object.

The ChannelFormats collection object has the following properties:

Properties Read/Write Description
Item R Returns the specified ChannelFormat object.
Count R Returns the number of ChannelFormat objects.
