Important
  • The EndMarker element can have a value for either the Number or Name attribute, but not both.
  • In Preview Mode, reaching an EndMarker stops the preview. This is true even if the PreviewDuration has not elapsed.
  • An EndMarker element takes precedence over a Duration element.
  • An EndMarker element defined within a Ref element, which is defined within an Entry element, will take precedence over an EndMarker element defined within that Entry element.
  • It is possible to define an EndMarker that specifies the end before the beginning. You can specify a marker for the content to stop rendering that is placed earlier in the content than a marker defined for the StartMarker element. In this case, no content plays. No error is generated.
