Installing the Controls in Microsoft Visual Basic

Visual Basic 5.0 provides you with visual tools to install ActiveX controls into your project. To have all the controls installed, you simply bring up the Component dialog box from the Project menu or right click on the Toolbox pane. Then scroll down the Controls panel to select the NetShow Theater Administration Controls. The selected item corresponds to the NSTAdmin.ocx file and contains the MediaServer, ServerView, ClientView, TitleView, and TraceView controls. The controls will show up as five icons in the Toolbox pane.
