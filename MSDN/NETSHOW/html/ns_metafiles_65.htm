Attributes

ClientSkip

If the value of the ClientSkip attribute is “no,” the player does not allow the user to skip the piece of content defined by the Entry element using the player controls.

The default value is “yes.”

SkipIfRef

The SkipIfRef attribute pertains only to Entry elements that are included through the use of an EntryRef element. The EntryRef element references another ASX file for logical inclusion in the current file. If the value of the SkipIfRef attribute for an Entry element is “yes,” the player ignores this pulled-in Entry, and moves on to the next Entry element, if any. The next Entry element can be the next Entry in the original file, or the next Entry in the ASX file referenced in the EntryRef element.

The default value is “no.”
