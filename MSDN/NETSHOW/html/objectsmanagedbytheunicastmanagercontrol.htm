Objects Managed by the Unicast Manager Admin Control

The Unicast Manager control manages the following objects:

Object Description
AuthenticationPlugins Collection of AuthenticationPlugin objects.
AuthenticationPlugin Contains information about each authentication plug-in.
Clients Collection of Client objects.
Client Contains information on each client.
EventPlugins Collection of EventPlugin objects.
EventPlugin Contains information about each event notification/authorization object.
VirtualRoots Collection of VirtualRoot objects.
VirtualRoot Contains information on each virtual root.
