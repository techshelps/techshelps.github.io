Using File Transfer

Microsoft® NetShow™ file transfer uses an ActiveX control to receive files by using multicast routing. See the Microsoft ActiveX SDK for more information about ActiveX controls.

In the interest of simplicity, this tutorial focuses on using a scripting language, such as Microsoft Visual Basic® Scripting Edition, to explain how to use the control. See the Visual Basic Script home page for more information about Visual Basic Scripting Edition (VBScript). See the Microsoft ActiveX SDK for more ways to use ActiveX controls. For example, the Microsoft ActiveX Control Pad is used for rapid application development using ActiveX controls.
