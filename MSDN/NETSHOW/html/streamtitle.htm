Stream.Title Property

Returns or sets the title of a Stream object.

Syntax

NSChannelMgr.Streams(index).Title

Example

Dim Streams
Set MyStream = NSChannelMgr.Streams.Find( "Product Rollout" )
MsgBox "Stream Title is: " & MyStream.Title
