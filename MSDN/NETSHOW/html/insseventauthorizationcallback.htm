INSSEventAuthorizationCallback

In addition to the methods implemented by IUnknown, the INSSEventAuthorizationCallback interface exposes the following method:

Method Description
OnEventAuthorized Reports results of the authorization to the server.

Remarks

This interface is implemented by the server to handle authorization of events triggered by calls to the INSSEventAuthorization::AuthorizeEvent method.
