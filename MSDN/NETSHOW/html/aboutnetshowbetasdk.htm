About the NetShow Beta SDK

The NetShow SDK encompasses a wide variety of technologies. Some sections of this documentation are structured to meet the needs of Web authors wishing to add streaming media to their Web sites. Other sections are geared toward ISVs developing Web-based software solutions.

This documentation is divided into based on the corresponding system components. The NetShow SDK is comprised of the following components:

Component Description
NetShow Unicast Manager Control A Microsoft ActiveX® control that exposes attributes for configuring, monitoring, and managing NetShow unicasting services.
NetShow Authentication API A COM-compliant API for developing packages that authenticate users before allowing them access to NetShow content.
NetShow Event Notification and Authorization API A COM-compliant API for developing packages that require the server to obtain authorization before performing tasks and to send notification when the task is complete.
NetShow Station Manager Control An ActiveX control that exposes attributes for configuring, monitoring, and managing NetShow multicasting services.
NetShow File Transfer Control An ActiveX control used to receive multicast file transfers.
NetShow Encoder An Automation object that exposes the functionality used to create ASF streams.

The documentation for each component is divided into four sections:

  • The About section describing the architecture of the control or API.

  • The Using section describing ways to implement the control or API.

  • The Reference section with complete descriptions of each interface, method, property, event, or structure exposed by the control or API.

  • The Sample section, containing a description of each sample, along with pointers or links to sample code and/or scripts that demonstrates using the control or API.
