ProxyHostName Property

Returns or sets a String value specifying the proxy host name.

Syntax

NSChannelMgr.ProxyHostName

Remarks

NetShow supports HTTP stream distribution among NetShow servers. For example, the server connected to this instance of the Station Manager control could be distributing streams to multiple downstream servers. It may try, using HTTP, to connect to a server behind a firewall. This property returns the name of the proxy server to which this server is currently connected.
