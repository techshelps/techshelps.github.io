Streams.Item Property

Returns the Stream object as specified by the Index value.

Syntax

NSChannelMgr.Streams.Item( Index )

Example

NSChannelMgr.Connect "LocalHost"
Dim Streams
Set Streams = NSChannelMgr.Streams
MsgBox Streams.Item(0).Title
