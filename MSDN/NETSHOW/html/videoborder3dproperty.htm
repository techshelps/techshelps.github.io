VideoBorder3D Property

Returns or sets a Boolean value specifying whether or not the 3D video border effect is enabled.

Syntax

MediaPlayer.VideoBorder3D

Remarks

This property can have the following values:

Value Description
True The 3D video border is displayed.
False (Default) The 3D video border is not displayed.

The 3D video border is a one pixel wide

See Also

VideoBorderColor, VideoBorderWidth
