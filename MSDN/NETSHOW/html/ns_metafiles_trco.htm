Logo

The Logo element specifies the URL for a graphic file that is associated with an ASX or Entry element (a show or clip). These graphic files are displayed in the player as they are defined by the Logo element attributes. Logo elements defined within an ASX element (where the ASX element is the parent element) are displayed for the duration of a playlist. All graphic styles except “ICON” are displayed in the View area of the player. The “ICON” style images are displayed next to the title of the clip or show in the Display panel, depending on the parent of the Logo element. If no “ICON” style Logo elements are included in the ASX, the graphic displayed next to the title is the icon associated with the file name type.

Logo formats follow the Microsoft® Internet Explorer Channel guidelines as follows, with all sizes in pixels:

ICON - 16 height x 16 width

MARK - 32 height x 80 width

Media Player supports GIF, BMP, and JPG graphic formats.
