File Transfer Methods

The following methods are attached to the File Transfer object:

Method Description
FtsStartFileTransferReceive Starts a file transfer receive operation.
FtsCancelFileTransferReceive Cancels an ongoing file transfer receive operation.
FtsRegisterStatusCallback Registers a status callback.
FtsRevokeStatusCallback Revokes a registered status callback.
