RecordStop Method

Stops recording to an .asf file.

Syntax

NSASFRte.RecordStop

Parameters

This method has no parameters.

Return Values

This method has no return value.

Remarks

Use this method to stop recording streaming data to an .asf file. Specify the .asf file name with the RecordFileName property.
