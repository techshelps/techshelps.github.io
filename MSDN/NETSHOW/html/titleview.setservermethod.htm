TitleView.SetServer Method

The TitleView.SetServer method connects to a NetShow Theater server.

Syntax

TitleView.SetServer( servername [, eventproxy ])

Parameters

servername

This parameter is a String value indicating the name of the title server to connect to.

eventproxy

An optional String value specifying the name of the server where the NetShow Monitor Service is running
