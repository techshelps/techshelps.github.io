OnRestripeProgress Event

The MediaServer_OnRestripeProgress event is fired every 30 seconds to show the title redeployment progress.

Syntax

MediaServer_OnestripeProgress( datetime, percentage )

Parameters

datetime

A Date value specifying the GMT time when the event occurred.

percentage

A Long value specifying the percentage of restriping completed.
