Encoder Methods

Encoder Automation exposes the following methods:

Method Description
LoadASD Loads a configuration into the Encoder from an .asd file.
RecordStart Starts recording to an .asf file.
RecordStop Stops recording to an .asf file.
SendScript Inserts a script command to the ASF stream.
Start Starts an encoding session.
Stop Stops an encoding session.
