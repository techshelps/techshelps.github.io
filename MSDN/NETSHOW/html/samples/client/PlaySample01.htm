Simple Player Control Example

This sample shows how to play an ASF file. You have two choices.

  • One is a file on your disk named Sample.asf that can be found on c:\netshow\asfroot\sample.asf. This is the normal location of the NetShow Content Root for ASF files. If you install NetShow Server you will get this sample. If you install NetShow Tools, you can create a sample file with the NetShow ASF Editor. If you do not have a file named Sample.asf in that location, you can change the HTML code of this sample to point to any location that holds an ASF file.
  • The other is a file on the Internet called Johnk.asf that is a non-endorsement of Microsoft NetShow by John Keister of "Almost Live".
