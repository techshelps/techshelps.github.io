IsNetEnabled Property

Returns a Boolean value specifying whether or not the Encoder is using the network.

Syntax

NSASFRte.IsNetEnabled

Remarks

This property is False if the Encoder is recording to an .asf file but is not sending a stream over a network. Use this property to determine whether the IPPort and NetShowServer values are valid.
