Using the Station Manager Control

Microsoft® NetShow™ uses a Microsoft ActiveX® control to manage the flow of content from a NetShow server. See the Microsoft ActiveX SDK for more information about ActiveX controls.

To use the Station Manager Control (Nschmgr.ocx), you must install NetShow Server, or copy the Nschmgr.ocx control to your computer and register it with the RegSvr32 application. Nschmgr.ocx requires Microsoft Windows NT® Service Pack 3 to operate correctly.
