Applications of Event Notification and Authorization Components

Using the NetShow Event Notification and Authorization API, you can develop plug-in components to perform a wide-range of authorization related tasks, including:

  • Restricting access to protected material.
  • Billing customers for viewing NetShow content.
  • Performing usage analysis of NetShow presentations.
  • Tracking client preferences for selectively targeted advertising.
