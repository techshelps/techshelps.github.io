StreamDescriptor Object Reference

The StreamDescriptor object contains methods and properties for each stream descriptor format.

The StreamDescriptor object has the following properties:

Property Read/Write Description
Description R Returns the description of the StreamDescriptor object.
Height R Returns the height value of the StreamDescriptor object.
Type R Returns the type of StreamDescriptor object.
Width R Returns the width value of the StreamDescriptor object.
