Using ASX Elements

The most important part of successfully using ASX files is the correct syntax of the ASX elements. Syntax errors in an ASX file can cause anything from a single attribute being overlooked, to the ASX file not being recognized as valid and failing to work at all. Almost as important is the order in which the elements appear in the ASX file. The attributes of some elements override the attributes of similar elements in different sections of the ASX file. Following is a list of all ASX elements that link to the sections about them. Each section contains a definition of the element, its attributes and their values, and special conditions related to the element:

  • Abstract
  • ASX
  • Author
  • Banner
  • Base
  • Copyright
  • Duration
  • EndMarker
  • Entry
  • EntryRef
  • Event
  • Logo
  • MoreInfo
  • Preview/Duration
  • Ref
  • StartMarker
  • StartTime
  • Title
