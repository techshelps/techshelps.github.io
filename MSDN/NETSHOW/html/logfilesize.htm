LogFileSize Property

Description

Returns or sets a Long value specifying the maximum log-file size for this NetShow Unicast Manager Admin control.

Syntax

NSUnicastMgr.LogFileSize

Remarks

When the maximum size is reached, the log file is reset.

The value of the LogFileSize property must be zero when you specify a non-zero value for LogFilePeriod.
