Important
  • If a MoreInfo element is used in an ASX file, the mouse cursor changes to a pointing hand when placed over the title (of the clip if in an Entry element or of the Show if in an ASX element) or Banner graphic (if the MoreInfo element is a chile element to a Banner element). Also, the URL defined in the href attribute of the MoreInfo element is displayed in the Status bar so that users know to click in the area to go to the URL displayed.
  • It is recommended that you use a Target attribute if using MoreInfo elements with ASX files that link to content displayed as an embedded control in a Web page or another container. If you do not use a Target attribute, or set the value of the Target attribute to _self, the page containing the embedded control is replaced with the URL defined in the MoreInfo element.
