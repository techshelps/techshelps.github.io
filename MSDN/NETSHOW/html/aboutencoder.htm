About the Encoder

The Encoder takes audio signals, video signals, and/or script commands, and encodes them into an ASF stream. This stream can then be saved to a file and broadcast by using NetShow™ Server.

The Encoder can be used by an administrator without any programming. However, to automate encoding or run multiple encoders, a programming interface is provided by using Automation.

For more information about using the Encoder as an application, see the NetShow Tools Guide.
