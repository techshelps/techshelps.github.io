Copyright

The Copyright element is a text string that specifies the copyright information for an ASX or Entry element. The copyright information is displayed in the Details area of the player and the Properties dialog box.

When used in the ASX element the copyright information is displayed only on the Show tab of the Properties dialog box. If you use the Copyright element in an Entry element, the text is displayed in the Clip bar of the Display area and on the Clip tab of the Properties dialog box.
