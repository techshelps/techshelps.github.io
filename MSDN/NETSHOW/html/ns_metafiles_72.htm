Attributes

Href (This is a required attribute)

The value of the href attribute is a URL to a referenced ASX file.

ClientBind

The value of the ClientBind element either allows or inhibits the users ability to add the URL defined in the href attribute of the EntryRef element to the Media Player Favorites list. If the value is “no,” users cannot add this URL to the Favorites list in the player.

The default value is “yes.”
