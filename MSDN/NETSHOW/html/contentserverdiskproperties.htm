ContentServerDisk Properties

Property Access Description
ID R Returns a Long value specifying the disk identifier of a content server.
Name R Returns a String value specifying the drive letter of a content server disk.
Rebuilt R Returns a Double value specifying the number of bytes rebuilt when rebuilding is in process.
Size R Returns a Double value specifying the content server disk size in bytes.
Status R Returns the status of a content server disk.
