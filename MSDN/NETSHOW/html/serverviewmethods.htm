ServerView Methods

Method Description
AddServer Adds a server to the tree.
DiskOffLine Takes a disk offline.
DiskOnLine Takes a disk online.
GetItemObject Returns a ServerView control object that contains information about the item.
PauseRebuild Pauses the rebuild of a disk.
Rebuild Rebuilds a disk.
RemoveServer Disconnects a server and removes it from the tree.
ShowProperties Shows the properties of a node.
ShowProperties Shows the properties of a title server.
