INSSEventHandler::OnHeartbeat

The INSSEventHandler::OnHeartbeat method is called by the server at each heartbeat period.

Syntax

HRESULT OnHeartbeat ( );

Parameters

None

Return Values

This method should return S_OK.

Remarks

The following code illustrates a skeleton implementation of the INSSEventHandler::OnHeartbeat method:

HRESULT
CEventHandler::OnHeartbeat() 
{
  // Increment counter for each client.
  return( S_OK );
}
