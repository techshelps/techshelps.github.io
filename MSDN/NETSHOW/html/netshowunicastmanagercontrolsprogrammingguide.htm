NetShow Unicast Manager Controls Programming Guide

[This is preliminary documentation and subject to change.]

Microsoft® NetShow™ Unicast Manager, packaged in the Nsunimgr.ocx file, embodies two Microsoft ActiveX® controls: NetShow Unicast Manager Admin and NetShow Unicast TraceView. The Admin control also manages various objects. This portion of the Microsoft NetShow SDK document describes the functionality exposed by these controls for creating custom administration, monitoring applications, and integrating NetShow into other software products. The document strives to lay a conceptual foundation and to provide practical guidelines for programming with the NetShow Unicast Manager controls. The discussion is divided into the following topics:

Topic Description
About the Netshow Unicast Manager Admin Control Introduces the architecture and utility of the NetShow Unicast Manager Admin control.
Using the Netshow Unicast Manager Admin Control Presents a guideline to using the NetShow Unicast Manager Admin control in your applications.
Reference of the Netshow Unicast Manager Admin Control Presents API references of the NetShow Unicast Manager Admin control, including events, methods, properties, and objects.
About the NetShow Unicast TraceView Control Introduces the architecture and utility of the NetShow Unicast TraceView control.
Using the NetShow Unicast TraceView Control Presents a guideline to using the NetShow Unicast TraceView control in your application.
Reference of the NetShow Unicast TraceView Control Presents API references of the NetShow Unicast TraceView control, including events, methods, and properties.
Samples of using the NetShow Manager Admin and TraceView controls Sample applications that illustrate how to use the Admin and TraceView controls in HTML, VBScript, and C/C++.

The NetShow Unicast Manager controls require that Nsunimgr.ocx and Nsoobj.dll be present and registered on the computer.
