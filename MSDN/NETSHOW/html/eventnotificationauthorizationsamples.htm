Event Notification and Authorization Samples

The NetShow SDK provides three sample components, written in C++, for developing event notification and authorization plug-in components

Sample Description
Pay-Per-Minute Authorizer A plug-in component that authorizes a client to stream a title based on a pay-per-minite payment schedule.
Pay-Per-View Authorizer A plug-in component that authorizes a client to stream a title based on a pay-per-view payment plan.
Memebership Authorizer A plug-in component that grants permissions to members only to stream a title.

To test the SDK sample, you should have the appropriate ActiveX controls installed in your system. It is recommended that you run these samples on a machine that has NetShow installed. To download the complete suite of NetShow products visit the NetShow web site http://www.microsoft.com/NetShow
