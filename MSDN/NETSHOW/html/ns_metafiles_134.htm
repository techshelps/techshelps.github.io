Attributes

Count

The value of the Count attribute is an integer representing the number of times the player will repeat, or loop through, the play of an Entry element. If no Count attribute is defined in the Repeat element the content in the associated Entry element repeats an infinite number of times. A value of zero (0) does not represent an infinite loop, but instead causes the Repeat element to be ignored.
