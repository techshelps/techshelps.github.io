Unicast Manager Admin Control Properties

The Unicast Manager control supports the following properties.

Property Read/Write Description
AdminEvents RW Enables or disables receipt of administration events.
AlertEvents RW Enables or disables receipt of alert events.
AuthenticationPlugins* R Returns a collection of authentication plug-ins.
ClientEvents RW Enables or disables receipt of client events.
Clients R Obtains a collection of all active clients.
EnableHttpStreaming RW Enables or disables Http streaming on this NetShow server.
EnableLogging RW Enables or disables logging.
EnableProxy RW Returns or sets a value indicating whether the proxy server is enabled.
EventPlugins* R Returns a collection of event handler/authorization plug-ins.
EventsProxy RW Returns or sets the name of a proxy monitor.
LogFileDirectory RW Returns or sets the directory of a log file.
LogFilePeriod RW Returns or sets the period of time logged events will be kept.
LogFileSize RW Returns or sets the maximum size for a log file.
MaxBandwidth RW Returns or sets the maximum aggregate bandwidth for the server.
MaxClients RW Returns or sets the maximum number of client connections.
MaxFileBitRate RW Returns or sets the maximum bit rate per file.
ProxyHostName* RW Returns the proxy host name.
ProxyUser* R Returns the user ID used for proxy authentication.
Server R Returns the name of the server currently connected.
ServerEvents RW Enables or disables receipt of server events.
ServerStatus * R Returns a value specifying the current status of the server.
VirtualRoots R Obtains a collection of all virtual roots.

* New in NetShow version 3.0.
