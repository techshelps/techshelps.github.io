ContentServerDisk.ID Property

The ContentServerDisk.ID property returns a Long value specifying the ID for a content server disk.

Syntax

ContentServerDisk.ID

Example

The following Visual Basic code outputs the name for each disk in a content server disk collection.

Sub enumDisks(server As String)
    Dim Disk As IMSrvCServerDisk
    Dim CServer As IMSrvCServer
    
    MediaServer1.SetServer server
    
    For Each CServer In MediaServer1.CServers
        For Each Disk In CServer.Disks
            MsgBox "Disk " & Disk.Name & " is on " & CServer.Name
        Next
    Next
End Sub
