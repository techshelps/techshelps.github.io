NetShow Station Manager Programming Guide

[This is preliminary documentation and subject to change.]

The Station Manager Control component of the Microsoft® NetShow™ SDK provides conceptual information about the NetShow Station Manager Control, as well as guidelines for using it in HTML, Microsoft Visual Basic®, and Microsoft Visual C++® programming environments. It is divided into the following parts:

Part Description
About the Station Manager Control Architecture of the control
Using the Station Manager Control Guide to using and programming the control
Station Manager Control Reference Reference to the control events, methods, and properties
Station Manager Control Samples Sample applications using Station Manager controls
