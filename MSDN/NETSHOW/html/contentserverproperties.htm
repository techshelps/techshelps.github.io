ContentServer Properties

Property Access Description
Disks R Returns a ContentServerDisks collection.
ID R Returns a Long value specifying the identifier of the content server.
Name R Returns a String value specifying the name of the host computer of the content server.
Status R Returns a Long value specifying the status of the content server. S_OK or zero means that the server is running normally.
