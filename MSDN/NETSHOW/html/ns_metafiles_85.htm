Important
  • When the Logo element is used to include graphics with the content delivered to the client, it is important to remember that the size of the View area is adjusted automatically to allow space for the graphic. If using an ASX file to define a playlist, each clip in the playlist can cause the player to resize. This can be an undesirable effect. If the dimensions of a video clip differ from another clip in a playlist the player window is resized anyway.
