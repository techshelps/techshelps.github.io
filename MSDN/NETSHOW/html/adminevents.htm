AdminEvents Property

A Boolean value specifying whether administration events are enabled or disabled for this instance of the Netshow Unicast Manager Admin control.

Syntax

NSUnicastMgr.AdminEvents

Remarks

The default value for AdminEvents is True. Administrative events are OnAdminBandwidthLimit, OnAdminClientLimit, OnAdminFileBitRateLimit, and OnAdminSecurityEnabled.

Example

If NSUnicastMgr.AdminEvents = False Then
  MsgBox "Administrative events are disabled"
End If

See Also

OnAdminBandwidthLimit, OnAdminClientLimit, OnAdminFileBitRateLimit, OnAdminSecurityEnabled
