ServerView.ShowServerProperties Method

The ServerView.ShowServerProperties method displays the properties of a NetShow Theater server.

Syntax

ServerView.ShowServerProperties( page )

Parameters

page

One of the four pages of the Properties dialog box as described in Remarks.

Remarks

This method requires specifying one of the following property pages.

Page Description
ServerViewClientsPage Clients properties dialog
ServerViewGeneralPage General properties dialog
ServerViewLoggingPage Logging properties dialog
ServerViewTitlesPage Titles properties dialog
