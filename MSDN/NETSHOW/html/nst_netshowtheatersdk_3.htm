TraceView.SetServer Method

The TraceView.SetServer method connects to a NetShow Theater title server.

Syntax

TraceView.SetServer( servername [, eventproxy])

Parameters

servername

A String value indicating the title server to connect to.

eventproxy

An optional String value specifying the name of the server where the NetShow Monitor Service is running. Reserved for future use.
