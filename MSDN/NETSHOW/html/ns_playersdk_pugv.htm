Balance Property

Returns or sets a Long value indicating the stereo balance.

Syntax

MediaPlayer.Balance

Remarks

The default value 0 indicates that the sound is balanced equally between the left and right speakers. A value of –10,000 indicates that all sound is going to the left speaker and 10,000 that all sound is going to the right speaker.
