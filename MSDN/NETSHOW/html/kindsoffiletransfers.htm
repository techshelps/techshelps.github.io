Kinds of File Transfers

The file transfer control supports two kinds of file transfer services:

  • Multicast file transfers. One sender transfers files to many receivers over a corporate intranet or the Internet, using multicast addresses and valid unused ports. The receiver "listens" to the same multicast IP address and port to receive the file.
  • Unicast file transfers. One sender transfers files to one receiver over a corporate intranet or the Internet, using non-multicast addresses and valid unused ports. The receiver "listens" to one of its own IP addresses and the same port.
