Important

If you create an ASX file with multiple clips (Entry or EntryRef elements) and set the value of the BannerBar attribute to “AUTO,” the player will resize for each piece of content, which it does when the size of the video media differes from clip to clip. This may not be a desired effect. If you want the size of the Microsoft Media Player window to stay the same (except when the video size changes) you can use the “FIXED” value for the BannerBar attribute.
