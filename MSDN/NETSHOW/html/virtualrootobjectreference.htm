VirtualRoot Object Reference

The VirtualRoot object contains properties for each current virtual root.

The VirtualRoot object has the following properties:

Property Read/Write Description
AliasName R Returns the alias used by the virtual root.
DirectoryPath RW Returns the virtual root directory path on the server.
MaxBandWidth RW Sets the maxium bandwidth allowed.
MaxClients RW Sets the maximum number of clients allowed.
