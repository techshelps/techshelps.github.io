About Encoder Automation

The unique features of the Encoder are exposed through Automation (formerly called OLE Automation). The exposed methods and properties allow client applications written in Microsoft® Visual Basic® or Microsoft Visual C++® to control the activities of the Encoder.

Note that the methods and properties exposed through automation do not use the same programming techniques as do Microsoft ActiveX® controls.

For more information about programming the Encoder, see Using Encoder Automation.

For more information about the specific methods and properties of the Encoder, see the Encoder Automation Reference.
