Title Properties

Property Access Description
Attributes R Returns a Long value specifying an attribute of the title.
Author RW Returns or sets a String value specifying the author of the title.
BitRate R Returns a Long value specifying a bit rate, in bits per second, of the title.
Blocks R Returns a Long value specifying the number of blocks in the title.
Caption RW Returns or sets a String value specifying the caption of the title.
Copyright RW Returns or sets a String value specifying the copyright information.
CreationTime R Returns a Date value specifying the time when the title was added to NetShow Theater.
Description RW Returns or sets a String value specifying the description of the title.
Format R Returns a Long value specifying a format code of the title.
Name RW Returns or sets a String value specifying a name to identify the title.
PlayTime R Returns a Date value specifying the total playing time, in seconds, of the title.
Size R Returns a Double value specifying the size, in bytes, of the title.
Version R Returns a Long value specifying the format version of the title.
