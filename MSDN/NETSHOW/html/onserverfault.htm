OnServerFault Event

This event is fired when a server fault occurs.

Syntax

NSUnicastMgr_OnServerFault( datetime, Status )

Parameters

datetime

A Date value indicating the time the event occurred, expressed in Greenwich Mean Time (GMT).

Status

An HRESULT value indicating the type of fault that occurred.
