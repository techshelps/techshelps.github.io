EnablePositionControls Property

Returns or sets a Boolean value specifying whether or not the position buttons on the controls panel are enabled.

Syntax

MediaPlayer.EnablePositionControls

Settings

The EnablePositionControls property can have the following values:

Setting Description
True (Default) The position controls are enabled.
False The position controls are disabled.
