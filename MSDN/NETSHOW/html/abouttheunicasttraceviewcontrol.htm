About the Netshow Unicast TraceView Control

The NetShow Unicast TraceView control provides an easy-to-use interface for monitoring NetShow unicast activity. Its grapical user interface is used to display a list of events initiated by the NetShow server. The display includes a description of events and the date and time when they were triggered. The TraceView control also exposes methods and supports events for programmatically manipulating the TraceView control.
