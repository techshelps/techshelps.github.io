File Transfer Examples

The following samples show how to use the File Transfer Service. All of the samples require a specific IP address and port for a NetShow server. To use these samples, you will need to modify the address and port so that they point to the server from which you will receive the files.

Sample Description
Receive Multicast with Cancel and Error-Checking This sample demonstration shows how to receive a multicast file using the File Transfer Service, using error-checking.
Receive Multicast with Progress This sample demonstration shows how to receive a multicast file using the File Transfer Service, displaying progress and transfer statistics.
Receive Multicast with Sender Verification This sample demonstration receives a multicast, and verifies the sender.
Receive Unicast This sample demonstration shows how to receive files from a unicast server, using the File Transfer Service.
Total Demo This sample rolls the features of the other samples into one unified demonstration.

These samples demonstrate using HTML and Visual Basic® Scripting Edition, with attributes exposed by Microsoft NetShow FTS control. These samples will not be fully functional without installing the NetShow server. To download the complete suite of NetShow products visit the NetShow web site (http://www.microsoft.com/NetShow)
