Ref

The Ref element specifies a Uniform Resource Locator (URL) for a content stream. The URL can point to either an Advanced Streaming Format (ASF or .asf) file for on-demand content, or a .nsc file that provides the player with information about a NetShow station. A NetShow station is a real-time stream. The Ref element can appear only as a child element of an Entry element.
