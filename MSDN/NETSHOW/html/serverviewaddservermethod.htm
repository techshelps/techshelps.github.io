ServerView.AddServer Method

The ServerView.AddServer method adds a server to the tree, and optionally specifies the server for the NetShow Monitor Service.

Syntax

ServerView.AddServer( servername [, eventproxy ] )

Parameters

servername

A String value specifying the name of the NetShow Theater title server.

eventproxy

An optional String value specifying the name of the server where the NetShow Monitor Service is running.

Remarks

Specifying the server for the NetShow Monitor Service is not implemented in the current release.
