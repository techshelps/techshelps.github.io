OnStreamStart Event

This event is fired when a Stream object is started.

Syntax

NSChannelMgr_OnStreamStart( Stream )

Parameters

Stream

A Stream object.

Example

Sub NSChannelMgr_OnStreamStart( Stream )
  MsgBox Stream.Desc & " started"
End Sub

See Also

Stream.Start, Stream.Stop, Stream.Close
