ServerView.OnDiskStatusChange Event

The ServerView.OnDiskStatusChange event is fired when the status of a content server disk changes.

Syntax

ServerView_OnDiskStatusChange( selection, status )

Parameters

selection

A Long value specifying the current selection in ServerView.

status

A MSRVDISKSTATUS specifying the status of the content server disk.
