Introduction

This section provides an overview of the extended Microsoft Component Object Model (COM) attributes. These attributes are used to provide marshaling information between Java objects and COM objects. If a development environment generates these attributes, the Microsoft Win32 VM for Java (Microsoft VM) will use them to marshal parameters between COM and Java.
