DriveType Property		  Language Reference 
Version 3 

See Also                  Applies To

Description
Returns a value indicating the type of a specified drive.
Syntax
object.DriveType

The object is always a Drive object.

Remarks
The following code illustrates the use of the DriveType property: 
Function ShowDriveType(drvpath)
  Dim fso, d, t
  Set fso = CreateObject("Scripting.FileSystemObject")
  Set d = fso.GetDrive(drvpath)
  Select Case d.DriveType
    Case 0: t = "Unknown"
    Case 1: t = "Removable"
    Case 2: t = "Fixed"
    Case 3: t = "Network"
    Case 4: t = "CD-ROM"
    Case 5: t = "RAM Disk"
  End Select
  ShowDriveType = "Drive " & d.DriveLetter & ": - " & t
End Function
