QLLC
Qualified logical link control. The protocol that permits SNA sessions to occur over X.25 networks.
queued TP
An invokable TP that can be started by only one incoming allocate command at a time. Incoming allocate commands that arrive while the queued TP is running do not start the program again, but are queued until the program issues another RECEIVE_ALLOCATE or until it finishes execution.
