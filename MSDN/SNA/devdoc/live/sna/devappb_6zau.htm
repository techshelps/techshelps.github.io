SNA Modem Status Interface

This section describes the interface to the modem status used by the Microsoft® SNA Server-supplied SDLC and X.25 SNALinks. This interface provides a set of RASmon-like modem lights. The modem status interface is intended primarily for the Microgate cards with internal modems, but can be used with any SDLC/X.25 link service to show the modem status.

This interface is intended primarily for IHVs who wish to provide drivers for their own dumb cards that will be directly compatible with the SNA Server-provided SDLC and X.25 SNALinks. It can also be used by IHVs who intend to write their own SNALinks but who wish to ensure that their drivers conform to the standard SNA Server model.
