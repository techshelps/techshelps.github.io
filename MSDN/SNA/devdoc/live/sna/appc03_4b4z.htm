Active Link Information

Active link information is defined or returned as described here.

Defined by IBM ES for OS/2 Version 1.0

link_name
Local logical link station.
dlc_name
Data link control (DLC) name set to one of the following:

ETHERAND

IBMTRNET

IBMPCNET

SDLC

TWINAX

X25DLC

adapter_name
Adapter number for this link.
fq_adj_cp_name
Fully qualified cp_name in adjacent node.
adjacent_node_type
NN, EN, or LEN.
cp_cp_session_support
Specifies whether the link supports CP-CP sessions.
connection_type
HOST, PEER, or BOTH.
tg_number
Transmission group number.
effective_capacity
Highest bit rate transmission supported.
cost_per_connect_time
Relative value of using link.
cost_per_byte
Relative cost of transmitting byte over link.
propagation_delay
Indicates amount of time for signal to travel length of link. Set to one of the following:

MINIMUM

LAN

TELEPHONE

PACKET_SWITCHED

SATELLITE

MAXIMUM

security
Set to one of the following:

NONSECURE

PUBLIC_SWITCHED_NETWORK

UNDERGROUND_CABLE

SECURE_CONDUIT

GUARDED_CONDUIT

ENCRYPTED

GUARDED_RADIATION

user_defined_1
User-defined TG characteristics.
user_defined_2
User-defined TG characteristics.
user_defined_3
User-defined TG characteristics.

Returned by SNA Server

Members

link_name
Connection name.
dlc_name
DLC name set to one of the following:

IBMTRNET

SDLC

DFT

X25DLC

adapter_name
Always set to zero.
fq_adj_cp_name
Always set to EBCDIC spaces.
adjacent_node_type
Always set to LEN.
cp_cp_session_support
Always set to NO.
connection_type
HOST or PEER from connection configuration.
tg_number
Always returns zero.
effective_capacity
Always set to zero.
cost_per_connect_time
Always set to zero.
cost_per_byte
Always set to zero.
propagation_delay
Always set to MAXIMUM.
security
Always set to NONSECURE.
user_defined_1
Always set to zero.
user_defined_2
Always set to zero.
user_defined_3
Always set to zero.
