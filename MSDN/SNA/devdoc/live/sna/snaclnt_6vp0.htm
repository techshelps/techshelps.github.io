SNA Server Client Setup for Windows NT and Windows 95

This section provides information regarding installation of the Microsoft® SNA Server Client binaries in the Win32® environments of Microsoft® Windows NT® and Microsoft® Windows® 95. This information will enable you to consolidate SNA Server into your product's own setup, simplifying operation for the end user. SNA Server versions 3.0 and later support SNA Server versions 2.0 and 2.1 Client binaries for Win32 environments, although it is recommended that you release updated binaries with each normal product release. This section describes the use of SNA Server versions 3.0 and later binaries, which are recommended for all new applications.
