OLE DB Enumerator Object

The Enumerator object allows OLE DB consumers to find OLE DB service providers, OLE DB data providers, and other enumerator objects.

The following interfaces of the Enumerator object are supported by the current version of the Microsoft® OLE DB Provider for AS/400 and VSAM:

  • IParseDisplayName
  • ISourcesRowset
  • ISupportErrorInfo

The PROGID or provider string for the Microsoft OLE DB Provider for AS/400 and VSAM is "SNAOLEDB".
