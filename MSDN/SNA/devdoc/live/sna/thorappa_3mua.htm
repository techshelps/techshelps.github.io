Host CCSID and SNA OLE DB Provider

The Host CCSID registry setting at the OLE DB provider level should be configured with the SNA OLE DB Management Console snap-in. Valid values are any CCSID except 65535. It is not valid to use 65535 because the OLE DB Provider must pass the host a code page, and 65535 does not correspond to any code page. The Host CCSID registry setting at the OLE DB Provider level must be specified and cannot be empty.
