TPSTART

An autostarted TP that runs as an application under Microsoft® Windows NT® requires the support of the TPSTART program, which is installed with the Microsoft® SNA Server software in the SYSTEM subdirectory of the SNA Server root directory. Therefore, the TPSTART program must be started on a Windows NT-based client before an autostarted invokable TP can be started as an application. Starting TPSTART can be accomplished by using standard Windows NT methods, such as including TPSTART in the Startup group on the client.
