Code Samples Using the RUI API

The following files can be used to build simple 3270 emulators using the RUI API. Study the comments in the .C files for information about how to run the emulators.

File name API Used Type of file Target operating system
RUI3270.C RUI Source code for RUI3270.EXE Microsoft® Windows NT®, Microsoft® Windows® 95, OS/2, or Microsoft® MS-DOS®
RUI3270.MAK RUI Makefile Windows NT or Windows 95
RUI3270D.MAK RUI Makefile MS-DOS
RUI3270O.MAK RUI Makefile OS/2
WRUI3270.C RUI Source code for WRUI3270.EXE Windows version 3.1 or Windows for Workgroups
WRUI3270.DEF RUI Definition file Windows version 3.1 or Windows for Workgroups
WRUI3270.LNK RUI Link file Windows version 3.1 or Windows for Workgroups
WRUI3270.MAK RUI Makefile Windows version 3.1 or Windows for Workgroups
WRUI3270.RC RUI Resource definition file Windows version 3.1 or Windows for Workgroups
