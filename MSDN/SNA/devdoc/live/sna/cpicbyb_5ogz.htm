About The CPI-C Guide

The Microsoft SNA Server CPI-C Programmer's Guide provides information required to develop C-language applications that use the Common Programming Interface for Communications (CPI-C) to exchange data in a Systems Network Architecture (SNA) environment.

This guide is intended for the programmer writing applications that use CPI-C to exchange data. It provides conceptual information and detailed reference information.

To use this guide effectively, you should be familiar with:

  • Microsoft® SNA Server.
  • One of the following operating environments:
    • Microsoft® Windows NT®
    • Microsoft® Windows® 95
    • Microsoft Windows graphical environment version 3.0 or later
    • IBM OS/2 version 1.2 or later
  • SNA concepts.

This guide covers the following topics:

Introduction to CPI-C
Introduces the fundamental concepts of CPI-C.
Writing CPI-C Applications
Contains general information about writing CPI-C applications.
CPI-C Calls
Describes each CPI-C call in detail.
Extensions for the Windows Environment
Describes extensions to Windows CPI-C.
Common Return Codes
Lists all of the return codes in integer order.
Sample CPI-C TPs in the SDK
Describes the CPI-C transaction program samples in the SNA Server SDK.
