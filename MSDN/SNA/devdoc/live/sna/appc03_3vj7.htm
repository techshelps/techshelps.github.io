LU 0 to 3 Information

LU 0 to 3 information is defined or returned as described here.

Defined by IBM ES for OS/2 Version 1.0

Members

lu_dlc_name
DLC name set to one of the following:

ETHERAND

IBMTRNET

IBMPCNET

SDLC

TWINAX

X25DLC

lu_adapter_number
Adapter number for host link.
link_id
Name of local logical link station being used.

Returned by SNA Server

lu_dlc_name

IBMTRNET
