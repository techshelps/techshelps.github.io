OLE DB Session Object

The Session object is created by a DataSource object. The Session object is used to create one or more Rowset or Index objects.

The following interfaces of the Session object are supported by the current version of the Microsoft® OLE DB Provider for AS/400 and VSAM:

  • IDBCreateCommand
  • IDBSchemaRowset
  • IGetDataSource
  • IOpenRowset
  • ISessionProperties
  • ISupportErrorInfo
