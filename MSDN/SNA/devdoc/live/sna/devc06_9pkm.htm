Integrated Link Service Setup

In Microsoft® SNA Server version 3.0 and later, SNA Explorer supports installation and configuration of link services. For an example of this new process, install the DLC/802.2 or IBM SDLC link service in SNA Server 3.0 or later. The benefits of updating your link service to use the new procedure are:

  • Setup using the new integrated link service provides full support for remote installation or reconfiguration of the link service.

    The link service retains access to physical hardware through an optional link service-provided helper DLL running on the target server, to support application-specific adapter detection or diagnostic functions.

  • It is integrated with the look and feel of SNA Explorer, for greater ease of use.
  • It eliminates the older Microsoft® Windows NT® Setup Engine and .INF files.

This section updates the setup information for integrated link services, providing the information you need to develop an integrated link service installation and configuration procedure.
