OLE DB and ADO Library Files

The following library files are on Microsoft Developer Network CD-ROMs and as a downloadable Software Development Kit from the Microsoft World Wide Web site at http://www.microsoft.com/data/oledb/download.htm provided as part of the OLE DB 1.5 SDK:

File name Description
ADOID.LIB GUID library for client C++ classes
OLEDB.LIB The OLE DB library
OLEDBD.LIB The OLE DB debug library

The OLEDB.LIB and OLEDBD.LIB library files are also included in the SDK\SAMPLES|OLEDB\LIB subdirectory on the SNA Server 4.0 CD-ROM.
