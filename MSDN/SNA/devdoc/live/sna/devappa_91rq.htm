Initialization

Device drivers under Windows NT should perform all initialization required at start of day when they are loaded by Windows NT. Configuration information for device drivers is stored in the Configuration Registry under Windows NT. For more details, refer to the documentation supplied with the Windows NT Device Driver Kit.

The SNALinks for dumb cards are implemented by the following files:

SDLC SNALink
SNAroot\SYSTEM\IBMSDLC.DLL
X.25 SNALink
SNAroot\SYSTEM\IBMX25.DLL

To bind to one of these, an installation script should mention the appropriate DLL in the IHVDLL registry entry that the script creates for the link service.
