Operating Systems

This guide contains information relating to five operating systems: Microsoft® Windows NT®, Microsoft® Windows® 95, Windows version 3.x, Microsoft® MS-DOS®, and OS/2. At present, Microsoft® SNA Server only supports the development of client applications for OS/2, Windows 3.x, Windows 95, and Windows NT. The MS-DOS interface is described here for completeness, but MS-DOS client application development is not supported. Under Windows NT and Windows 95, support for client applications is provided only for the Win32® subsystem.

Information presented here can be different for different operating systems. When a section of text applies only to specific operating systems, this is indicated by a header preceding that text. The end of such a section is indicated by one of the following:

  • A header specifying other operating systems.
  • A header stating that subsequent information applies to all operating systems.
  • The end of a topic.

Text not explicitly designated applies to all operating systems.
