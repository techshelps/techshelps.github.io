SNALink Structure

An SNA Server SNALink consists of:

  • The link-specific protocol code provided by the IHV
  • A Base
  • A DMOD

The DMOD, Base, and the IHV link-specific component of an SNA Server SNALink are implemented as dynamic-link libraries (DLLs). The executable component SNALINK.EXE is used to start an SNALink. This component determines from the SNA Server configuration information which link support DLL (for example, IHVLINK.DLL) is required for the SNALink and dynamically loads it before entering the Base scheduler.
