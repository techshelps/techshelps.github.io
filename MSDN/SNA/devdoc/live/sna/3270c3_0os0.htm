The SSCP Connection

The application's SSCP connection to the local node provides access to the SSCP session between the Microsoft® SNA Server secondary LU and the host SSCP. This section describes:

  • How an application opens and closes its SSCP connection.
  • The use of the SSCP connection.

For simplicity, this section describes the SSCP connection as if an application only uses a single SNA Server LU (and therefore a single SSCP connection); in practice, applications can use multiple LUs.
