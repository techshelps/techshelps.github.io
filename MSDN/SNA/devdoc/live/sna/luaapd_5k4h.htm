Code Samples Using the SLI API

The following files can be used to build simple 3270 emulators using the SLI API. Study the comments in the .C files for information about how to run the emulators.

File name API Used Type of file Target operating system
SLI3270.C SLI Source code for SLI3270.EXE Microsoft® Windows NT®, Microsoft® Windows® 95, OS/2, or Microsoft® MS-DOS®
SLI3270.MAK SLI Makefile Windows NT or Windows 95
SLI3270O.MAK SLI Makefile OS/2
WSLI3270.C SLI Source code for WSLI3270.EXE Windows version 3.1 or Windows for Workgroups
WSLI3270.DEF SLI Definition file Windows version 3.1 or Windows for Workgroups
WSLI3270.LNK SLI Link file Windows version 3.1 or Windows for Workgroups
WSLI3270.MAK SLI Makefile Windows version 3.1 or Windows for Workgroups
WSLI3270.RC SLI Resource definition file Windows version 3.1 or Windows for Workgroups
