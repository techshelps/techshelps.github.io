OLE DB Index Object

The Index object is created by a Session object. The Index object is treated as a special case of the Rowset object under SNA OLE DB.

The following interfaces of the Index object are supported by the current version of the Microsoft® OLE DB Provider for AS/400 and VSAM:

  • IAccessor
  • IColumnsInfo
  • IConvertType
  • IRowset
  • IRowsetChange
  • IRowsetIndex
  • IRowsetInfo
  • ISupportErrorInfo

In addition to these standard OLE DB interfaces, the following additional interfaces of the Index object are also supported by the current version of the OLE DB Provider for AS/400 and VSAM:

  • IColumnsRowset
  • IRowsetIdentity
  • IRowsetLocate
  • IRowsetResynch
