Session Information

Session information is defined or returned as described here.

Defined by IBM ES for OS/2 Version 1.0

Members

link_id
Name of local logical link station.
type_of_session
SSCP_PU_SESSION, SSCP_LU_SESSION, or LU_LU_SESSION.
fq_pc_id
Fully qualified procedure correlation identifier of the session.
conversation_group_id
Unique identifier of the session.
type_of_pacing
FIXED or ADAPTIVE.

Returned by SNA Server

Members

link_id
Connection name.
type_of_session
SSCP_PU_SESSION is never returned.
fq_pc_id
Set to zero.
conversation_group_id
Set to zero.
type_of_pacing
Always returns FIXED.
