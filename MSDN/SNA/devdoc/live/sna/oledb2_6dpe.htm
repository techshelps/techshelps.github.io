OLE DB View Object

The View object is created on a Rowset object. The View object is used to expose simple operations such as sorting and filtering a rowset by applying a view. Views may be applied when opening a Rowset object or applied to an existing Rowset object.

The following interfaces of the View object are supported by the current version of the Microsoft® OLE DB Provider for AS/400 and VSAM:

  • IAccessor
  • ISupportErrorInfo
  • IViewChapter
  • IViewFilter
  • IViewRowset
  • IViewSort
