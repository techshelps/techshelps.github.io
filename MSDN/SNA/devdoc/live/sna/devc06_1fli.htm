Setup Information

This section describes both the new integrated link service installation provided with Microsoft® SNA Server version 3.0 and later and the older Microsoft® Windows NT® INF-based link service installation procedures used in SNA Server 2.x. In SNA Server 3.0 and later, SNA Explorer supports installation and configuration of link services. Although we will continue to support the existing INF-based link service setup procedure, there are clear benefits in updating your link service to the new procedure.
