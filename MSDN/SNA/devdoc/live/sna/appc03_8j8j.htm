Differences by Information Type

Differences to the SNA Server DISPLAY verb are described in this section by information type. For each information type, there is a topic that describes:

  • The information defined by IBM ES for OS/2 version 1.0.
  • The information returned by SNA Server.

Note  SNA Server does not support all of the information types supported by IBM ES for OS/2 version 1.0. If an information type is not listed in this section, it is not supported by SNA Server.
