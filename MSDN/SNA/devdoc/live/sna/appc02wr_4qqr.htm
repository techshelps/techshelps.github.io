Sync Point Level 2 Support in SNA Server

The following topics describe additions to Microsoft® SNA Server that enable Sync Point Level 2 support to the SNA Server LU 6.2 protocol stack.

This section assumes familiarity with the existing SNA Server APPC basic and mapped conversation interfaces. It does not attempt to explain the SNA formats and protocols for implementing Sync Point protocols, which are described in SNA LU6.2 Reference: Peer Protocols published by IBM (Document SC31-6808-1).
