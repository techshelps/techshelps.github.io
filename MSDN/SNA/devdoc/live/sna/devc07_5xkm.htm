SNA Server DMOD Logging Macros

The logging macros provided with SNA Server are relatively easy to use because each log call is a single line of code. In the simplest case, only a message number is required; the text of the logged message is taken from the message file. Parameters can also be supplied as required.

For the text and meaning of each of the log messages included in the SNA Server message file, see the Microsoft SNA Server Reference. Examples of messages logged by the SNA Server 802.2 DLC SNALink are shown in Examples.
