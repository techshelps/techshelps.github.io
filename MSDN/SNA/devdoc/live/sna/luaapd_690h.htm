Specifying a File Name for Table G for Code Conversion

The sample emulators use a user-defined table referred to as Table G in Microsoft® SNA Server for converting between ASCII and EBCDIC characters. The sample applications require that the file name of this table be specified. To specify the file name, use the COMTBLG registry or environment variable, as described in Appendix C of the Microsoft SNA Server Reference.

A sample Table G file, COMTBLG.DAT, can be found on the SNA server in the SYSTEM subdirectory of the SNA Server root directory. To use this file, copy it to the client computers where the sample programs will be run, and specify the file name with the COMTBLG variable.
