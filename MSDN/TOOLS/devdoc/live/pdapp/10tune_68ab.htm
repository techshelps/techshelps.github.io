Performance Tuning Win32-based Applications

After your application is complete, you can focus on improving its performance on Windows NT. Even if you are completely familiar with the design and implementation of the application code, you will need tools for performance tuning.

This overview briefly discusses the tools you can use to help you find performance problems in your application. To provide an example, this overview describes how several of these tools were used to find a performance problem in the Solitaire game.

These tools are flexible and can address a wide variety of performance issues. The tools discussed are:

  • Performance monitor
  • Call attributed profiler (CAP)
  • FIOSAP profiler
  • APIMON
  • PROFILE
