The Monitor Menu

You use the Monitor menu to specify one or more types of DDE information that DDESpy is to display. The following tasks can be specified:

  • Monitoring string handle data
  • Monitoring posted DDE messages
  • Monitoring callbacks
  • Monitoring Errors

You can use the Filters option to choose the type of DDE messages and types of DDE callbacks to monitor.

The DDE protocol passes information by using shared memory. The contents of the shared memory depend on the type of DDE transaction. Several structures have been defined to allow applications using DDE to access the information in shared memory. DDESpy displays the contents of the appropriate structure for the DDE activity being monitored.
