Creating Bitmaps, Cursors, and Icons

With the Image Editor (IMAGEDIT.EXE), you can create graphical images to represent files, windows, cursors, and other objects in your Windows applications. The Image Editor provides you with a set of drawing tools for creating commonly used shapes.

The Image Editor can create the following types of images:

  • Bitmaps, which are graphical objects used to create, manipulate, and store images.
  • Cursors, which represent the position of a pointing device, such as a mouse.
  • Icons, which represent objects such as files, folders, shortcuts, applications, and documents.

For more information, see the help file IMAGEDIT.HLP, included with the Image Editor.
