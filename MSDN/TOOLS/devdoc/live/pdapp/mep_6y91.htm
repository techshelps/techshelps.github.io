Accessing the Help File

Before you can open the help file, you must ensure that the help system has been configured properly.

    To Verify the Help Configuration
  • Check that MEP.HLP is located in a directory specified by the PATH environment variable.
  • Check that the TOOLS.INI file is located in the directory specified by the INIT environment variable.
  • Check that the TOOLS.INI file contains the following lines:
    [mep-mhelp]
helpfiles: mep.hlp

[mep]
load: mhelp.dll

The following table gives the keystrokes for the most common help tasks.

Help Task Key
Go to the help contents. ALT+A contents F1
Get help for the word at the cursor. F1
Get help for a specific topic. ALT+A topic F1
Get previously viewed help. ALT+F1
Get the next topic in the help file. CTRL+F1
Move to the next hotspot. TAB
Move to the previous hotspot. SHIFT+TAB
Get help for the highlighted hotspot. ENTER
Move between windows. F6
Exit split-window help. ESC
Exit help and return to previous file. F2
