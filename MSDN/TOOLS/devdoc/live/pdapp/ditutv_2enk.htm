Sizing and Aligning a Group

    To make all the selected buttons the same size as the anchor button
  1. From the Arrange menu, click Same Size.
  2. Click the top option to make the buttons the same width.
  3. Click Same Size again.
  4. Click the bottom option to make the buttons the same height.

All the buttons become the same size as the anchor button.

The two Push Button items on the Arrange menu control the alignment of push buttons. One centers buttons across the bottom of the dialog box, and the other arranges them down the right side. Use the proper option to arrange the buttons down the right side.
