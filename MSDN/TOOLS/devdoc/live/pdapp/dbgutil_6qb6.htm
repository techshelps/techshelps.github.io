Using Windows Debug Monitor

Windows 95 Only

The Windows Debug Monitor (DBWIN.EXE) displays debug messages sent by your application and by the system DLLs. By default, output is sent to the DBWIN window. You can change this default by clicking either Output to COM1 or Output to COM2 from the Options menu.

To control the types of messages that are displayed, click Settings from the Options menu, and select the messages to view.
