Using the Image Editor

Before creating a new image, you first need to design it. After you have a design, follow these steps to create the image:

    To Create An Image
  1. Click New on the File menu. The Resource Type dialog box appears.
  2. Select either Bitmap, Cursor, or Icon, depending on the image type.
  3. For bitmaps, set the width, height, and colors. For cursors and icons, select the target device.
  4. Select colors from the palette, and drawing tools from the tool box.
  5. Draw the image in the drawing area with the selected tools.
  6. Save the file and exit the Image Editor.
