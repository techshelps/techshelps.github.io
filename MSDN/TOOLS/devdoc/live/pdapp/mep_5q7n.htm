File Operations

Description Function Key
Save the file or open a new file. Setfile F2
Reread the file, discarding edits. Refresh SHIFT+F7
Merge a file or program output. Paste SHIFT+INS
Print all or part of file. Print CTRL+F8
Save all modified files. Saveall unassigned
Exit with or without saving. Exit ALT+X
Toggle no-edit state of file. Noedit unassigned
