The Kernel Debugger (KD)

This section describes basic procedures for using KD, the kernel debugger. The actual debugger name is based on the platform:

  • I386KD.EXE for x86 machines
  • MIPSKD.EXE for MIPS machines
  • ALPHAKD.EXE for Alpha machines
  • PPCKD.EXE for Power PC machines.
