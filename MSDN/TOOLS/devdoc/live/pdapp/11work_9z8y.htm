Using the Working Set Tuner

WST consists of the following files:

File Description
WST.INI Working set tuner initialization file
WST.DLL Working set tuner DLL
WST.LIB Working set tuner import library
WSTDUMP.EXE Allows collecting or clearing of WST data and stopping data collection at any time
WSTUNE.EXE Utility for creating the packing list from collected data
WSTCAT.EXE Concatenates multiple data files

Using the WST is similar to using CAP, from which it is derived.

The following steps are involved in using WST:

  • Preparing your application
  • Collecting WST data
  • Analyzing WST data
  • Linking with the packing list
