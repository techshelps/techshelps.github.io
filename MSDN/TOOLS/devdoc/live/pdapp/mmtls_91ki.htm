IDF Editor

Windows 95 Only

The IDF Editor (IDFEDIT.EXE) enables you to edit instrument definition files (IDF). Use the following steps to edit an existing .IDF file.

    To edit an .IDF file
  1. Click Open on the File menu. The instruments are displayed in the left pane, and the associated definition data is displayed in the right pane.
  2. Click any definition data that you wish to change and type in new values.
  3. To add new instruments, click New Instrument on the File menu.
