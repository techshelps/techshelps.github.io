Creating Fonts

With the Font Editor (FONTEDIT.EXE), you can modify existing fonts to create new fonts to use in your applications. You can use the Font Editor to create raster fonts. The Font Editor cannot create vector or TrueType fonts.

The Font Editor can do the following:

  • Edit letters, numbers, and other characters in a font.
  • Modify the height, width, and character-mapping of a font.
  • Change information in the font header.

For more information, see the help file FONTEDIT.HLP, included with the Font Editor.
