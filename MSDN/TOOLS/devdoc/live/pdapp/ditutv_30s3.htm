Creating Dialog Boxes

The Dialog Editor automates the process of creating dialog boxes. You can see your dialogs as you design them without writing and compiling your own resource scripts. The Dialog Editor requires a mouse or equivalent pointing device.

The Dialog Editor works much like a paint program. You design dialogs interactively, placing and moving graphical elements right on the screen. When you complete your design, the Dialog Editor automatically creates the files you need to incorporate the dialogs in your program.

This overview explains how to create dialog boxes and include them in an application. You can use the Dialog Editor's online Help to get more information on specific features.

Note  A dialog box that has not been compiled and exists only in resource script form is referred to as a dialog.
