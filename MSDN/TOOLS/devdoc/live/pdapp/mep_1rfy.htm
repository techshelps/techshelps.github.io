Command Manipulation

Description Function Key
Begin a function argument. Arg ALT+A
Turn on the meta command prefix. Meta F9
Cancel current argument. Cancel ESC
Recall the last selection. Lastselect CTRL+U
Recall the last text argument. Lasttext CTRL+O
Type the next character literally. Quote CTRL+P
Reverse the last edit. Undo ALT+BKSP
Toggle between box and stream mode. Boxstream CTRL+B
Assign a key or define a macro. Assign ALT+=
Execute command or macro list. Execute ALT+E
Toggle macro recording. Record unassigned
Display key or macro assignment. Tell CTRL+T
Display or clear status bar. Message unassigned
