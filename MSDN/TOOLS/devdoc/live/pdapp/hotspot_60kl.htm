Including Hypergraphics in a Help File

To include a hypergraphic in a help topic, use the bmc, bml, and bmr statements. These statements insert the graphic into the help file at the specified position. The syntax is as follows:

{bXmc filename.shg}

{bXml filename.shg}

{bXmr filename.shg}

Hotspots must reference valid context strings. The help compiler displays a warning if the hyperlink references a nonexistent context string.
