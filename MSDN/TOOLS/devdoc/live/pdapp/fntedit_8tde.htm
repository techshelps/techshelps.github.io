Using the Font Editor

To create a new font file with the Font Editor, you must open a preexisting font file. The font file ARTM1111.FNT is supplied with the Font Editor. After you open a font, the Font Editor displays the following features:

Feature Description
Character window Displays a copy of the character you want to edit.
Character-viewing window Displays two instances of the character in its normal size, so you can examine the effects of your edits.
Character information Displays the character value, and the character width and height, in pixels.
Font-viewing window Displays normal-size copies of the characters in the font.

You can edit a font in the following ways:

  • By turning individual pixels in a character on and off
  • By adding and deleting columns or rows of pixels in a character.
  • By changing the width of a character in a variable-pitch font.
  • Changing the height of the font characters.
  • Changing the maximum width of a variable-pitch font.
  • Changing the character width of a fixed-pitch font.
  • Changing a font from fixed-pitch to variable-pitch.
  • Changing the weight of the font.
  • Changing the font-file header information. This header includes information such as the filename, typeface name, and copyright, as well as the font metrics.
