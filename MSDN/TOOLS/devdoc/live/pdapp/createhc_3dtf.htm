Creating Help Files

Microsoft Windows help provides online help for users working with a Win32-based application. Windows help provides a practical way to present information about your application in a format users can access easily.

The Help Topics dialog box contains a Contents tab, and Index tab, and a Find tab. The Contents tab contains a hierarchical view of the information that the user can use to navigate the information. The view contains books, which can contain topics or other books. The Index tab contains keywords for one or more help files. The Find tab provides full-text search capability for any help file.
