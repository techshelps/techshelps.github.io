Using the Microsoft Editor

The Microsoft Editor (MEP.EXE) is a console-based text editor that is specialized for editing code. It contains many built-in editing features. To customize MEP, you can use the initialization file, TOOLS.INI. You can also create editor extensions to add custom features to MEP.

This overview is not meant to be comprehensive. It discusses the following topics:

  • The MEP Command Line
  • The TOOLS.INI File
  • Basic Editing
  • Accessing the Help File
  • Support Programs
  • MEP Editing Function Reference

For more information, please see the MS-DOS help file, MEP.HLP, included with MEP.
