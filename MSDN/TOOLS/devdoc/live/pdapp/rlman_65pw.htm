Converting to and from Unicode Text with UCONVERT

To convert a file to Unicode from an ANSI, OEM, or other code page, use the Unicode Converter (UCONVERT). To perform the conversion, use the following steps:

  • Click Open Source File on the File menu. Specify the file you want to convert.
  • Click Convert Now on the Conversion menu.
  • Click Save As from the File menu. Specify a filename for the converted file.

You can use the Conversion menu to specify conversion options. For more information, see the help file UCONVERT.HLP, included with the Unicode Converter.
