Capture Screen

Windows 95 Only

The Capture Screen tool (CAPSCRN.EXE) enables you to capture video sequences and images from your computer screen. Capture Screen compresses video during capture, storing the sequence using run-length encoding. Capture Screen stores only those portions of the image that change from frame to frame.

The following table describes some of the commands contained on the Capture Screen menus:

Command Description
Set Capture File Enables you to choose the capture file.
Save Single Frame Saves a single frame.
Play Captured Video Plays the captured video.
Set Capture Window Sets the screen area to capture.
Preferences Sets options for video capture and Capture Screen.
Capture Starts capturing video from the capture area. Press ESC to end video capture.
