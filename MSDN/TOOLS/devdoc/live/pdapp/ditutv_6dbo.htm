Changing Control Size and Placement

You can also use items on the Arrange menu to size and place groups of controls. Using items on the Arrange menu you can:

  • Align controls along an edge (click Align)
  • Distribute controls evenly with respect to an anchor control (click Even Spacing)
  • Set controls to the same size (click Same Size)
  • Distribute push buttons evenly along the bottom or side of the dialog (click Push Buttons)
  • Change grid and spacing values for editing controls (click Settings)

You might want to experiment with these concepts with the dialog you have just created. Click the Aligning icon in the help file for more information.
