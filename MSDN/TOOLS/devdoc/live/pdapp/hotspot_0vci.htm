Using the Hotspot Editor

To create a hypergraphic, follow these steps:

    Creating a Hypergraphic
  1. Load an existing graphic from a file (.BMP, .DIB, or .WMF) or paste the graphic from the clipboard.
  2. Move the mouse to the position for the hotspot. Create the hotspot by holding down the mouse button and dragging the bounding rectangle until the hotspot is as large as desired.
  3. Save the file in .SHG format.

You can also specify attributes for each hotspot. Click Attributes on the Edit menu. You can set the following attributes:

  • Link type (jump, pop-up, or macro)
  • Context string (or macro, if the link type is a macro)
  • Visible or invisible hotspot borders
  • Hotspot identifier
  • Coordinates
