AVI Editor

Windows 95 Only

The AVI Editor (AVIEDIT.EXE) enables you to edit audio-video interleaved (AVI) files. An AVI file can contain several different types of data, such as video sequences, and multiple soundtracks. You can easily access these components with the AVI Editor.
