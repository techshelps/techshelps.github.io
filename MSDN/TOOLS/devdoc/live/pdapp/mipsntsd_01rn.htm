Canceling NTSD Commands

To cancel the current command, use SYS RQ on the target keyboard (ALT-SYS RQ on enhanced keyboards) for RISC platforms; for x86 platforms, use CTRL+C. This action is most useful for halting commands that generate long output listings on the display.

If NTSD is running, program execution is stopped. Entering CTRL+C will not affect commands that pass execution control to the application being debugged.
