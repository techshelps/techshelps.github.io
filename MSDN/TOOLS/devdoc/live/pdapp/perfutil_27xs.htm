MemSnap

The Memory Snapshot utility (MEMSNAP.EXE) logs system memory usage to a log file. The following is the command-line syntax for MEMSNAP:

memsnap [LogFile]
LogFile
The name of the log file. The default is MEMSNAP.LOG.

The log file contains the following information for each active process:

  • Process ID
  • Process Name
  • Working Set
  • Paged Pool
  • Non-Paged Pool
  • Pagefile
  • Commit
  • Handles
  • Threads
