PerfMtr

Windows NT only

PerfMtr can display a variety of system performance information. The command-line syntax is:

perfmtr

PerfMtr begins by displaying CPU usage. Use the following keystrokes to change the type of performance information displayed:

Key Displays
C CPU usage
V VM usage
F File Cache usage
R Cache Manager read and write operations
P Pool usage
I I/O usage
X x86 VDM stats
S Server stats

The information is periodically updated, and will scroll off the screen. Type H to repeat the header for the current information type. You can also type Q to quit.
