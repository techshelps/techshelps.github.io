SubPlotLabelLocationType Constants

The VtSubPlotLabelLocationType constants provide methods for displaying the subplot label.

Constant Description
VtChSubPlotLabelLocationTypeNone No subplot label is displayed.
VtChSubPlotLabelLocationTypeAbove The subplot label is displayed above the pie.
VtChSubPlotLabelLocationTypeBelow The subplot label is displayed below the pie.
VtChSubPlotLabelLocationTypeCenter The subplot label is centered on the pie.
