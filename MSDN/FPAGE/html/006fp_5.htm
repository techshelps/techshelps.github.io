The Folders View

To display the file and folder structure of a FrontPage web, choose the Folders view. In this view, you can manage files and folders much in the same way as you would in the Windows 95 (or Windows NT) Explorer.

The left side of the Folders view, called the All Folders pane, shows a hierarchical list of the folders in your FrontPage web. Clicking on a folder in the All Folders pane displays its contents in the Contents pane. Using the plus (+) and minus (–) signs in the All Folders pane, you can expand and collapse folders to show or hide their subfolders.

The Contents pane displays information about the individual files. Clicking on a column header sorts the list by that column; clicking the column header a second time reverses the sort order.

You can right-click any file to display a shortcut menu, from which you can open, cut, copy, rename, or delete the selected file. From the shortcut menu, you can also view properties or add a web task for the selected file.

To arrange files and folders, simply drag and drop them between folders using the mouse. You can drag and drop single files or any selection of files. When you move or rename files or folders, FrontPage will update any hyperlinks that point to them.
