Dynamic HTML Features

Dynamic HTML gives authors enhanced control of the elements on their pages. Because it does not require fetching information from the Web server, Dynamic HTML is very efficient and presents the user with a lively, compelling page without requiring time-consuming network activity. FrontPage Dynamic HTML form features are supported by Microsoft Internet Explorer 4.0 and other Web browsers that recognize Dynamic HTML.

The FrontPage Editor includes the following Dynamic HTML features for forms:

  • tab ordering

  • form field labeling

  • access keys

This topic describes Dynamic HTML in forms. For information on other FrontPage Dynamic HTML features, see “Creating Active Pages,” on page 208.
