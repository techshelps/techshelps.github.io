Access Keys

A form field's access key is the keyboard key corresponding to an underlined letter in the field's label. In Web browsers that support Dynamic HTML, a user can select a field that has an access key by clicking ALT+key, where key is a single underlined letter in the field's label. For example, if a check box field has the label “small”, a user could select the check box field by typing ALT+S.

To assign an access key to a form field, type a label before or after the field and underline the single access key character, select the field and its label, choose Form Field on the Insert menu, and click Label.
