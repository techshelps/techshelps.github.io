The All Files View

Choose this view to see a complete list of all the files in the current FrontPage web, including information about each individual file. The All Files view does not show folders, but lists all of the files located in the folders and subfolders of your FrontPage web. Clicking on a column header sorts the list by that column; clicking the column header a second time reverses the sort order.

You can right-click any file to display a shortcut menu, from which you can open, cut, copy, rename, or delete the selected file. From the shortcut menu, you can also view properties or add a web task for the selected file.
