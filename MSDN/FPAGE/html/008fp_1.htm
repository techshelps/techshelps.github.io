Creating Hyperlinks in FrontPage

A hyperlink is a pointer from text or from an image to a page or other type of file on the World Wide Web. In World Wide Web pages, hyperlinks are the primary way to navigate between pages and among Web sites. When you create a text hyperlink, the FrontPage Editor underlines the text and displays it in a special color to indicate the presence of the hyperlink.

The most common hyperlink is from one page to another in the same FrontPage web. This gives users a way to move around the pages within your Web site. To create a hyperlink to a page or file within the current FrontPage web, choose Hyperlink on the Insert menu. The Create Hyperlink dialog box lists all the pages, other files, and folders in your FrontPage web. Double-click a folder to view its contents. Select the page or file that you want to create a link to (called the target) and click OK.

Open Page Symbol

FrontPage uses the open page symbol in the Create Hyperlink dialog box to identify currently opened pages. You can browse a list of any open page’s bookmarks in the Bookmark field of this dialog box.

In the URL field of the Create Hyperlink dialog box, FrontPage includes a drop-down list of special values that you can use to quickly create hyperlinks to pages in your FrontPage web.

The value… creates a hyperlink to…
Home Page The home page of the FrontPage web.
Up One Level The parent, in the FrontPage web’s navigational structure, of the page you are currently editing.
Previous Page The left peer, in the FrontPage web’s navigational structure, of the page you are currently editing.
Next Page The right peer, in the FrontPage web’s navigational structure, of the page you are currently editing.

To learn about the navigational structure of FrontPage webs, see “FrontPage Webs and the Microsoft FrontPage Explorer”.

To quickly create a hyperlink in the current FrontPage web… Drag a page or other file from the FrontPage Explorer and drop it anywhere on a page in the FrontPage Editor. The FrontPage Editor creates a hyperlink to that page or file.

World Wide Web button

Authors often want to give users of their FrontPage web a way to surf to pages at other locations on the World Wide Web, for example in a list of favorite Web sites. To create a hyperlink to a page on the World Wide Web, choose Create Hyperlink and click the World Wide Web button in the Create Hyperlink dialog box. This opens your Web browser. Surf to a page that you want to link to, and return to the Create Hyperlink dialog box using ALT+TAB. The location of the page will be displayed in the URL field. Click OK to create a hyperlink to the page.
