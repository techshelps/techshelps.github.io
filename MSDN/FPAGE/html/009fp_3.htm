Viewing and Setting Image Properties

Images have properties that you can set in the FrontPage Editor. These properties specify the image’s type, give information to browsers about how to display the image, control how the image aligns with text, and set the image’s default hyperlink.

To set an image’s properties in the FrontPage Editor, select the image and choose Image Properties on the Edit menu. Some important image properties are:

  • The image source, which is the URL or, if the image is not on a server, the file name of the image.

  • The image type, GIF or JPEG. You can convert an image from one type to another in the Image Properties dialog box.

  • The image dimensions, including its width and height in pixels, and its size in bytes.

  • The image alignment. When you insert an image, it is placed at the insertion point and is treated, for display purposes, like a character in text. By setting its alignment, you can control how the image aligns with text and other images. For example, the alignment selection left places the image in the left margin and wraps the text that follows the image down the image's right side.

  • The image border thickness, which sets a border of the specified thickness in pixels around the image.

  • The alternate low-resolution image, which is a small-sized image that some browsers display while the larger-sized main image is being loaded from the Web server.

  • The image alternate text, which is displayed by browsers when image display is disabled or not available.

  • The image default hyperlink, which is the hyperlink to follow when a user clicks in any area in the image not covered by a hotspot. You can browse your FrontPage web from the Image Properties dialog box and change this hyperlink.

  • The number of times to repeat, for video clips.

  • When to start playing, for video clips.
