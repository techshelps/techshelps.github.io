What’s Ahead

You have been introduced to the FrontPage Explorer and FrontPage Editor, the Web site management and page editing components that make up Microsoft FrontPage.

In the FrontPage Explorer, you learned how to create a new Web site and how to add pages and apply a graphical theme.

In the FrontPage Editor, you learned how to add text and formatting to a page, and how different page components can enhance the appearance of a FrontPage web.

In Lesson 2, you will learn more about Web page design using the FrontPage Editor, including forms and form fields, image maps, and advanced hyperlinks.
