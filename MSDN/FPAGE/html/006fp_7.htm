Navigation View

The Navigation view is most useful for constructing a new Web site, because it provides a graphical outline of your site’s structure and can automatically create navigation bars that point to the most important pages in your FrontPage web.

The Navigation view has a split-screen view that displays the structure of your FrontPage web in the upper half of the screen (the Navigation pane), and a familiar Windows Explorer-like file and folder list in the lower half of the screen (the Files pane).

The structure of the FrontPage web is displayed as a diagram, similar to an organizational chart. The home page is at the top, with its linked pages on the next levels.

For larger Web sites, creating a consistent and intuitive structure can take hours of work. If you create your FrontPage web in the Navigation view, then FrontPage can do this work for you.

To learn more about the Navigation view and how a FrontPage web structure is created, take “The FrontPage Tutorial”.
