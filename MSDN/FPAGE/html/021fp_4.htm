Frames Views

The FrontPage Editor has five views. Three views — Normal, HTML, and Preview — are available for all pages. (See “The FrontPage Editor’s Views” for details). When you edit a frames page, there are two additional views: No Frames and Frames Page HTML.

Frames pages can define alternative HTML for Web browsers to display if they do not support frames and frames pages. You use the No Frames view to edit the alternative HTML for the currently open frames page.

In the FrontPage Editor, you can display a different page in each frame of the currently open frames page. Using the HTML view, you can edit the HTML for all of the pages that are opened in your frames page. To edit the HTML of the frames page itself, use the Frames Page HTML view.
