Modifying SQL Queries

If testing the query returns results that you did not intend to display, or if they do not match the search results in your database management tool, you can make changes to the existing SQL query in the database region on your results page. You can also add, remove, or rearrange the columns on the results page.

To modify the query, open the results page in the FrontPage Editor. Right-click the placeholder icon for the database region and choose Database Region Properties from the shortcut menu.

When the Database Region Wizard opens, proceed to the Enter the SQL String for the Query input field. Although you can manually edit the code in this field, it is strongly recommended that you regenerate an SQL query in Microsoft Access, switch to the SQL view, and then copy and paste the new code into FrontPage using the Paste SQL From Clipboard button in the Database Region Wizard.

Note Clicking the Paste SQL From Clipboard button clears the paste buffer and replaces the entire contents of the SQL query input field. Any parameter that you manually inserted after pasting must be re-inserted into the new SQL text.
