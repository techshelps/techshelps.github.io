Page Transition Effects

You can apply a page transition effect to the active page. These are movie-like effects that can occur when the user surfs to your page or away from it. You can further limit a transition effect to happen only when the user enters or leaves your Web site.

You apply a page transition effect on the Format menu by pointing to Page Transition. In the Page Transitions dialog box, select the event, such as entering your Web site, that will trigger the effect, and select the type of effect.

To choose a page transition effect, create one and preview it. You can preview a transition effect in the FrontPage Editor’s Preview tab (if you have Microsoft Internet Explorer, version 4.0 installed) or by choosing Preview in Browser on the FrontPage Editor’s File menu (if you have Microsoft Internet Explorer, version 4.0 or another Web browser that supports Dynamic HTML installed).
