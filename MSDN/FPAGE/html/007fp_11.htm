Editing Text

In the FrontPage Editor, you enter and format text, create lists, specify paragraph styles, and adjust the indentation and alignment of your paragraphs using a familiar word-processor interface. FrontPage automatically generates HTML that reflects the paragraph styles and formats you chose. FrontPage also includes standard word processing commands, such as finding and replacing text, a spelling checker and a thesaurus.

Minimizing White Space Between Lines of Text Each time that you press ENTER, FrontPage creates a new paragraph. Most Web browsers add white space before and after each paragraph. To start a new line of text without adding white space, press SHIFT+ENTER to insert a line break. Because Web browsers do not add extra white space between lines separated by line breaks, your lines will be spaced more closely together.

Using the File command on the FrontPage Editor’s Insert menu, you can insert the contents of a file onto your page. If the file is not an HTML file, the FrontPage Editor converts it to HTML. You can insert many types of files, including Microsoft Word, Microsoft Excel, and WordPerfect. You can paste or, when supported, drag selections from one of these applications onto a page in the FrontPage Editor. If you paste or drag a range of cells from a Microsoft Excel spreadsheet, they are added to the page as a new table.
