Creating Thumbnail Images

A thumbnail is a small version of an image on a page that contains a hyperlink to a full-sized version of the same image. Putting a thumbnail image on a page with a hyperlink to its full-sized counterpart reduces the amount of information to download over the network before the complete page can be viewed in a Web browser. Users who want to view the full-sized image linked to a thumbnail can click the thumbnail. The Web browser then follows the hyperlink, causing the full-sized image to be loaded over the network and displayed.

You can quickly create a thumbnail image in the FrontPage Editor. Select the image on a page and choose AutoThumbnail on the Tools menu. FrontPage creates the thumbnail image, inserts it in place of the larger image, and creates a hyperlink to the full-sized image from the thumbnail image.
