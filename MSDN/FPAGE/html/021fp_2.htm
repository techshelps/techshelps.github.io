Creating Frames Pages

You create a new frames page in the FrontPage Editor using the New Frames Page command on the Frame menu. When you choose this command, FrontPage opens the New dialog box. You create a frames page by selecting a template in the Frames tab of this dialog box. Once you have created a frames page, you can add or remove frames and reposition borders in the FrontPage Editor to customize the frames page.

Creating or Resizing Frames To quickly resize a frame, click and drag one of its borders. To create a new frame, hold down CTRL and drag one of the borders.

Along with listing all frames page templates, the Frames tab displays a preview of each frames page and a description of how the frames page could be used. For example:

  • Banner and Contents creates a banner frame at the top, in which you display your corporate logo or other general information. The left frame is the contents frame. This is where the page containing hyperlinks is displayed. The right frame is the main frame. Each time that a user clicks a different hyperlink in the contents frame, the page displayed in the main frame will change.

  • Header creates a contents frame on top and a main frame beneath it. The contents frame could contain an image map containing hyperlinks to pages that will be displayed in the main frame.

When the FrontPage Editor displays a new frames page, each frame is displayed as a bordered region with the following three buttons:

  • Set Initial Page sets the initial page. This is the page that is displayed in a frame when a frames page is first displayed in a Web browser.

    When you click Set Initial Page, FrontPage displays the Create Hyperlink dialog box. When you set an initial page, you are actually creating a hyperlink to the page. See “Creating Hyperlinks and Bookmarks”, for details on using the Create Hyperlink dialog box.

  • New Page creates a new page and sets it as the initial page for the frame.

    When you click New Page, FrontPage creates a blank new page, opens it in the frame, and makes it the initial page for the frame. When you save the frames page, FrontPage prompts you for a name and title for the page.

  • Help displays a help topic explaining how initial pages and frames work.
