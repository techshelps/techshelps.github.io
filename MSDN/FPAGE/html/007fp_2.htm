Learning About the FrontPage Editor

This chapter will give you an overview of the FrontPage Editor and describe page design and text and table editing in the FrontPage Editor. To learn about other FrontPage Editor features, see the following chapters:

  • “Creating Hyperlinks and Bookmarks”, describes creating and editing hyperlinks using the FrontPage Editor.

  • “Images and Image Maps”, describes the FrontPage Editor’s comprehensive set of tools for editing images and for creating image maps.

  • “Creating Active Pages”, describes the built-in FrontPage components that you can insert to add dynamic functionality to your pages.

  • “Forms and Form Fields”, describes how you create forms using the FrontPage Editor and how you collect information from forms using the FrontPage form handlers.

  • “Creating and Using Frames”, describes creating frames and frames pages using the FrontPage Editor.
