Opening a FrontPage Web

When you first open the FrontPage Explorer, it displays the Getting Started dialog box, a quick starting point from which you begin your work in FrontPage.

Previously opened FrontPage webs will appear in the list of recently opened webs in this dialog box. Select a FrontPage web from the list and click OK to open it.

If the FrontPage web you want to open is not yet on the list, click the More Webs button. This will display the Open FrontPage Web dialog box, where you can select a Web server or disk location from the drop down list (or type a new location directly into the field). Click the List Webs button to find all FrontPage webs at that location. To open a FrontPage web, double-click its name in the list.
