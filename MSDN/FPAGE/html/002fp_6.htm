About This Book

This book teaches you the basics of Microsoft FrontPage 98 and introduces you to the Internet and the World Wide Web.

  • The “FrontPage Tutorial” will show you how easy it is to use Microsoft FrontPage to get a new Web site up and running. In three hands-on lessons, it will introduce you to FrontPage using easy to follow, step-by-step procedures.

  • “FrontPage and the World Wide Web” contains an overview of the Internet and the World Wide Web and describes how you use FrontPage during each stage of creating your Web site. Among the topics covered are: creating or importing FrontPage webs, applying themes, adding navigational structure, publishing webs, creating and editing pages, adding tables, images, hyperlinks, and active page elements, using frames, creating forms, and FrontPage database integration.

  • The “Glossary” has a set of definitions of FrontPage, Internet and World Wide Web terms. You can also use this glossary online, by clicking Glossary from any online Help window.
