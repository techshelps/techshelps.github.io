Push Button Fields

A push button field submits a form to a form handler or resets a form to its initial state. Because FrontPage automatically adds a Submit and Reset Push Button field to each new form, you only need to create a Push Button field if the default fields have been deleted.

When you configure a push button field, you can supply a label, which is displayed on the button and sent to the form handler as the field’s value, and a name, which is also sent to the form handler. Naming and labeling push button fields is useful if you are using a custom form handler that processes a form based on which button is clicked.
