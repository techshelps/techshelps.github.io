FrontPage Online Help

In Microsoft FrontPage 98, you can get online Help where and when you need it:

Help button

  • When you want to know more about a screen item, such as a command or toolbar button, click the Help button on the toolbar and then click the screen item for an online Help topic describing the item.

  • In a FrontPage dialog box, click Help for a Help topic describing the dialog box.

  • Select any command or dialog box and click F1 for a Help topic describing the command or dialog box.

  • Choose Microsoft FrontPage Help from any Microsoft FrontPage Help menu. You will find a list of all Help topics, an index of Help terms, and a Find tab, in which you can search for Help topics that contain one or more words of interest.

Using the Microsoft FrontPage Help menu, you can also get the latest news and information about Microsoft FrontPage 98. Choose Microsoft on the Web to visit the FrontPage World Wide Web site. At this Web site, you will find the latest information about FrontPage, answers to frequently asked questions, a list of Internet service providers that support the Microsoft FrontPage Server Extensions, and much more.
