Text Box Fields

Text box fields give users a way to enter text in a form. There are two types of text box fields: one-line text boxes and scrolling text boxes.

One-line text boxes are single-line form fields into which users can type text. Use a one-line text box for a single line of information, such as a user’s name or address. You can also configure a one-line text box as a password field. Most Web browsers will display the contents of a password field back to a user as asterisk characters, to protect the confidentiality of the text.

Scrolling text boxes are multiple-line form fields into which users can type many lines of text. Use a scrolling text box for information of unlimited length, such as general comments or questions about your product or organization.

When a form containing a scrolling or one-line text box field is submitted by a user in a Web browser, the name and the contents of the text box field are sent to the form handler.
