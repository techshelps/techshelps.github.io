Image Editing

Along with commands that create hotspots, the Image toolbar contains commands that edit images. When you edit an image using one of these commands, you change its appearance or its size.

Saving an edited image changes the image permanently in your FrontPage web. When you save a page containing an edited image, FrontPage prompts you to save the image. If the image is used in more than one place in your FrontPage web, FrontPage prompts you to save a copy of it. If you do not save a copy of an image after editing it, the image’s appearance will change on every page on which it is used.
