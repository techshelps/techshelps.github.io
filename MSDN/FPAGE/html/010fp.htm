Creating Active Pages

Active pages are pages that change based on user actions and that have a dynamic, engaging appearance. Using ActiveX, Java, Plug-ins, CGI scripting, and other methods, Web designers have created new and exciting tools for making Web pages come alive. Because FrontPage comes with a full set of active page elements, you can take advantage of these latest active page techniques, including Dynamic HTML, without having to write a line of code.

This topic gives you a tour of the FrontPage components and other active page elements that you can insert on your pages using the FrontPage Editor.
