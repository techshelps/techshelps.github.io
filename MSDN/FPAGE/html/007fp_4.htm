Creating Pages

You create a new page in the FrontPage Editor using the New command on the File menu. When you select this command, FrontPage opens the New dialog box, which displays all the built-in templates and wizards and any templates that you have added:

When you select a template or wizard in the New dialog box, it is displayed in the Preview field.

Once you have selected the template or wizard on which to base your page, click OK, and FrontPage creates the page and opens it in the FrontPage Editor.
