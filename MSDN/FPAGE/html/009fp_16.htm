Using GIF Images for Bulleted Lists

You can use a GIF image as the bullet characters in a bulleted list. To select a GIF image to use as bullets, select an item in a list and chose List Properties from the shortcut menu. In the Image Bullets tab of the List Properties dialog box, click Use images from current theme or click Specify Image, then click Browse to select the image in the Image dialog box.

When you use GIF images as bullets, the bulleted list is formatted in HTML as a table.
