Creating Tables

To create a table in the FrontPage Editor:

  • Use the Insert Table command on the Standard toolbar. This command is also on the Tables menu.

  • Use the Draw Table command on the Tables toolbar.

The Insert Table command on the Standard toolbar quickly creates a table with regularly-spaced rows and columns. To use this command, click the Insert Table button and drag the number of rows and columns that you want the table to have.

If you choose Insert Table command on the Tools menu, the FrontPage Editor displays the Insert Table dialog box. This lets you set initial properties for the table at the same time that you choose the number of rows and columns in the table. The Insert Table command will create a table anywhere on the page, including inside a table cell.

Draw Table button

The Draw Table command on the Table toolbar lets you draw a table with irregularly spaced rows and columns. To create a new table with this tool, click the Draw Table button, then click and drag on the page. This creates the outside border of the table.
