Resetting an Image

If you do not like the results of a set of image editing commands, you can undo them using the Reset command on the Image toolbar. This command undoes all image editing commands that have been applied to an image since the last time it was saved or, for a new image, since it was inserted on the page.

Can I edit a page’s background image? Yes. Image editing commands work on a page’s background image when no images on the page are selected. You can use the following commands on the background image: Washout, Black and White, Rotate Left, Rotate Right, Reverse, Flip, More Contrast, Less Contrast, More Brightness, Less Brightness, Bevel, and Reset. If the background image is used on other pages, you are prompted to save a copy of it.
