The FrontPage Server Extensions Resource Kit

The Microsoft FrontPage Server Extensions Resource Kit is an HTML document that is in the \SERK folder of your Microsoft FrontPage 98 CD-ROM. It is designed to make it easy for Internet Service Providers or corporate intranet Web server administrators to install and maintain the FrontPage Server Extensions on a Web server computer.

If you are a Web author or administrator, you do not need to read this document to use FrontPage. However, if you are curious about the FrontPage Server Extensions, you can learn more about them by reading the overview of the Server Extensions, in the file \SERK\intro.htm on the FrontPage 98 CD-ROM.
