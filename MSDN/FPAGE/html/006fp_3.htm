Creating FrontPage Webs

You create new FrontPage webs in the New FrontPage Web dialog box, which is displayed by choosing Create a New FrontPage Web in the Getting Started dialog box, or by choosing New FrontPage Web on the FrontPage Explorer’s File menu.

Click the wizard or template you want to use to view a description of it just below the list box.

In the New FrontPage Web dialog box, you have three options for creating a new FrontPage web:

  • One Page Web creates a FrontPage web with a single page (the home page). Use this option to create a new web site from scratch, with no generic or pre-existing content.

  • Import an Existing Web will launch the Import Web Wizard, which is used to import an existing web into a new FrontPage web. The wizard will guide you through the process of importing an existing Web site.

  • From Wizard or Template lists the web wizards and web templates you can choose from to create a new FrontPage web with suggested content.

A web wizard is an interactive tool that guides you through the creation of a FrontPage web. Based on your responses to a series of questions, web wizards create professional-looking, but generic FrontPage webs that you can customize. If applicable, FrontPage webs created by wizards may contain pre-designed forms, tables, images, navigation bars, hyperlinks, and other page elements.

A web template is a generic FrontPage web, modeled after popular World Wide Web site styles. You customize a FrontPage web created from a web template by adding, deleting, or editing its pages as needed.

Adding Custom FrontPage Web Wizards and Templates With the Microsoft FrontPage Software Developer’s Kit, you can create custom FrontPage web wizards and templates and add them to the FrontPage Explorer’s New FrontPage Web dialog box.

The FrontPage Software Developer’s Kit is available in the \SDK folder on the Microsoft FrontPage 98 CD-ROM.
