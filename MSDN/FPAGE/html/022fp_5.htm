Formatting the Results Page

The next step is to create and format the Active Server Page that will display the query results in a Web browser. You can add the database region to an existing page in your FrontPage web, or add the region on a blank page that you can later customize.

Û To Insert a Database Region on the Results Page

Show FrontPage Editor button

  1. On the FrontPage Explorer’s Tools menu, choose Show FrontPage Editor, or click the Show FrontPage Editor button on the toolbar.

    The FrontPage Editor is displayed.

  2. Open or create the page that will display the database search results in the Web browser and position the cursor where you want the query results table to be displayed.

  3. On the FrontPage Editor’s Insert menu, point to Database, then click Database Region Wizard.

  4. When the Database Region Wizard is displayed, proceed to the panel containing the ODBC Data Source Name (DSN) field, then enter the DSN in this field.

    The name you enter here should match the DSN that was specified on the Web server and on your workstation.

  5. Leave the Username and Password fields blank for this example and click Next.

    You may need to specify a username and password if your actual database file or database server have been secured.

  6. Switch to Microsoft Access and select the entire code text in the SQL View window

  7. On the Edit menu, choose Copy.

  8. Switch back to the Database Region Wizard and paste the SQL code into the Enter the SQL String for the Query input field by clicking Paste SQL From Clipboard.

  9. Click Next.

  10. Click the Add Field button.

    The Add Field dialog box is displayed. In this dialog box, you enter an actual database field name. While Microsoft Access shows “friendly” names for database fields, the FrontPage Database Region Wizard requires you to enter the actual database field name (without spaces). To look up the actual field names, switch to the Design View in Microsoft Access.

  11. Type CategoryName (without a space between the two words), then click OK.

    CategoryName is the first field you are adding from the Northwind Traders sample database. Repeat steps 10 – 11 for the ProductName and ProductSales fields.

  12. Click Finish to create the database region.

    FrontPage inserts a database region placeholder icon on the current page where your results table will be displayed when users display the page in a Web browser.

When the actual page is fetched by the Web browser, the results table will be expanded to contain the database query results. Each database field will be a column in the table and each returned database record is displayed as a row in the table.
