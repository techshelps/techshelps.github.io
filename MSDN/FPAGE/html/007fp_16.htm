Other Table Editing Techniques

Here are some other table-editing techniques:

  • Drag any cell border to resize the cell.

Eraser button

  • Use the Eraser tool on the Tables toolbar to delete one or more cell borders.

  • Add a caption to a table using the Insert Caption command on the Table menu.

  • Delete cells using the Delete Cells command on the FrontPage Editor’s Table menu.

  • Convert text to a table using the Convert Text to Table command on the FrontPage Editor’s Table menu.

See the FrontPage Online Help system for more table-editing procedures.

Why does FrontPage sometimes prevent me from erasing borders? FrontPage generates tables that conform to HTML syntax. In HTML, some arrangements of cells are not permitted. If you try to erase a border in a way that the syntax of HTML does not allow, FrontPage will not erase the border.
