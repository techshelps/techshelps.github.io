Workstation Configuration

When the Data Source Name has been defined on the Web server, and the database file has been imported to your FrontPage web, you can proceed to configure the DSN on the workstation.

Û To Configure the DSN on the Workstation

  1. In the Windows NT or Windows 95 Control Panel, double-click the 32-bit ODBC (ODBC on some systems) icon to display the ODBC Data Source Administrator dialog box.

  2. In the System DSN tab, click Add to specify a new Data Source Name.

  3. For this example, select Microsoft Access Driver (*.mdb).

  4. Click Finish.

    The ODBC Microsoft Access 97 Setup dialog box is displayed.

  5. Enter the DSN in the Data Source Name field.

    The name you enter here should match the DSN that was specified by your server administrator or ISP on the Web server.

  6. Click Select.

  7. In the Select Database dialog box, point to the disk location of the database file you imported to your FrontPage web (for example, C:\inetpub\wwwroot\northwind.mdb), then click OK.

  8. Click OK to close the ODBC Microsoft Access 97 Setup dialog box.

  9. Click OK to close the ODBC Data Source Administrator dialog box.

When the Data Source Name has been defined on the Web server, and the database file has been imported to your FrontPage web, the next step is to build a query using your database management tool.
