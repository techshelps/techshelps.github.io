Shared Borders

Pages can include shared borders, which usually contain navigation bars (graphical elements that contain hyperlinks to other pages). FrontPage maintains a single set of borders — top, bottom, left, and righ — for the entire FrontPage web.

You can include shared borders on pages in two ways:

  • In the FrontPage Explorer you can specify that one or more shared borders should be included in all pages that are part of the FrontPage web’s navigational structure (as displayed in the Navigation view). Choose Shared Borders on the Tools menu and, in the Shared Borders dialog box, select the shared borders to include. For example, if you specify the Top border in this way, it will appear on all pages that are part of the FrontPage web’s navigational structure.

  • In the FrontPage Editor you can specify to include one or more shared borders on the currently opened page. Choose Shared Borders on the Tools menu, and, in the Shared Borders dialog box, click Set for this page only and select the shared borders to include.

You edit the contents of shared borders in the FrontPage Editor. Click inside it on a page and edit the region as you would edit any other part of the page. When you save the page, all edits to shared borders are also saved.

Each shared border is used throughout a FrontPage web. When you edit it, you are changing its appearance on every page that it is included in. For example, if you add your e-mail address to the Bottom shared border in one page, all other pages that include the Bottom border will now display your e-mail address.
