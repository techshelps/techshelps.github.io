Setting Table Properties

You can complete the design of your table by setting properties on the entire table and on selected cells.

The Table Properties command opens the Table Properties dialog box. Use this dialog box, to set the alignment of the table on the page, specify a table border and select its color, choose a background color or image for the table, and precisely specify the table’s size.

The Cell Properties command opens the Cell Properties dialog box. This dialog box is similar to the Table Properties dialog box, but lets you override global table properties for the currently selected cells.
