Hover Buttons

Hover buttons are animated buttons. When the mouse pointer moves over a hover button, or when a hover button is clicked, the button comes alive. It glows, changes colors, or displays some other effect to attract the user. You assign a hyperlink to a hover button and specify its text message. You can also display an image on a hover button.

You create a hover button on the Insert menu by pointing to Active Elements and then clicking Hover Button. In the Hover Button dialog box, you supply the button text and the hyperlink, and you select the type of effect and the effect’s colors. Along with text, you can display an image on a Hover Button.

The best way to configure a hover button is to create one and preview it, trying various effects and color schemes until it matches your needs. To preview a hover button, choose Preview in Browser on the FrontPage Editor’s File menu.
