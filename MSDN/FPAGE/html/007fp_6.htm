Designing Pages

The FrontPage Editor contains many features that help you design your pages. You can apply a built-in theme to a page. You can assign border regions to a page and edit them, insert navigation bars and banners, and set properties on a page, such as background color or sound.

A common way to lay out a page is to create a table to lay out the page and add all content inside the table’s cells. For a discussion of table-editing in FrontPage, see “Creating and Editing Tables”.
