Do Not Ref-Count Core Components During Installation

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

Core components must not be ref-counted in the registry by application installers.

See Also

For the most up-to-date and complete list of Windows 95 and Windows NT core components, see http://www.microsoft.com/windows/thirdparty/winlogo.htm.
