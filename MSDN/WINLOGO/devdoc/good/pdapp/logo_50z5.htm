Use the Registry Correctly

[This is preliminary documentation and subject to change.]

Designed for Microsoft® Windows® Logo requirements that relate to making appropriate use of the Microsoft® Windows® registry are listed below. The list is just a summary. To qualify for the Designed for Windows Logo, you must comply with all stipulations of the requirement reference text to which the list refers.

  • Register native data file types.
  • Do not add to Win.ini or System.ini.
  • Support informational keys in the registry.
