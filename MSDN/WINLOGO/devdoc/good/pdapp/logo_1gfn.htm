Make a Development Tool Capable of Generating Logo-Compliant Applications

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

If Windows is one of the compiler or development tool target platforms, the application must be capable of generating applications that can meet all of the Designed for Microsoft Windows Logo requirements.

Exception

Development tools that create Java applications do not need to be capable of generating applications that can meet all of the Designed for Microsoft Windows Logo requirements (see Java Applications).
