Use TAPI 2.1 for Telephony As Soon As It Becomes Available

[This is preliminary documentation and subject to change.]

Best Practice   (Designed for Microsoft® Windows® Logo)

As soon as TAPI 2.1 is available, telephony applications should use TAPI 2.1 for Windows 95, Windows NT 4.0 Workstation, and Windows NT 4.0 Server. TAPI 2.1 is the only version that runs on all three platforms.
