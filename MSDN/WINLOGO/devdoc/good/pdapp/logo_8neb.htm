Use the Canonical/International Form for the Number Dialed in Telephony-Enabled Applications

[This is preliminary documentation and subject to change.]

Best Practice   (Designed for Microsoft® Windows® Logo)

A telephony enabled application should pass the number dialed to tapiRequestMakeCall in canonical/international form.

Remarks

This best practice is done so that the call-manager application placing the call can properly apply the user's Dialing Properties settings.
