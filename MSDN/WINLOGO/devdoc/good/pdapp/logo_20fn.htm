Avoid Operating System Differences

[This is preliminary documentation and subject to change.]

Best Practice   (Designed for Microsoft® Windows® Logo)

You should ensure that the differences between the Windows 95 and the Windows NT versions of your application are minimal.

Therefore, it is recommended that you review your application to ensure that it:

  • Does not write to different registry locations on Windows 95 and Windows NT (registry values may also be different).
  • Does not install different files (.dll, .exe, and so forth) on the two operating systems (or additional files on Windows 95).
  • Does not install different versions of files that are common to both Windows 95 and Windows NT.
  • Does not make operating-system-specific calls on Windows 95 that are not available on Windows NT.
