Provide Attended and/or Silent Installation

[This is preliminary documentation and subject to change.]

Requirement  (Designed for Microsoft® Windows® Logo)

The installer must either lead the user through the installation process in a user-attended mode (not just instruct the user to copy or decompress files) or provide a facility for an unattended (silent) installation requiring no user input. Preferably (though not required), the application will provide both options.
