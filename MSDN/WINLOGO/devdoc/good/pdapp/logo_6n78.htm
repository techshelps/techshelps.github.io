Provide Consistent, Up-to-date Windows Support

[This is preliminary documentation and subject to change.]

Providing consistent, up-to-date support for Windows standards and technologies makes your application easier to use and administer, and helps to avoid incompatibility problems. Designed for Microsoft® Windows® Logo guidelines that relate to this goal are presented in the following categories:

  • General
  • File System
  • Operating-System Migration
  • Accessibility and User Interface
  • Internet
  • OLE / COM and COM+
  • Telephony
  • OnNow / ACPI
  • Directory Services
