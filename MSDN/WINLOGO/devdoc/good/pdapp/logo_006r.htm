Apply Dialing Properties to Numbers in Telephony-centric Applications

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

Telephony-centric applications must apply the user's Dialing Properties settings to numbers to be dialed. This can be accomplished by passing the number to the lineTranslateAddress function before the call is placed.

This number must be in canonical/international form unless the number to be dialed is an internal extension or the user has otherwise explicitly instructed the application to dial the digits "as is" without applying Dialing Properties. Even so, they should pass to lineTranslateAddress so that tone/pulse dialing settings can be applied.
