Support Long File Names for Files Exposed in Windows Explorer

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

If a file name is fully exposed in Microsoft® Windows® Explorer, it must be a fully supported long file name (LFN). Applications are allowed to use "labeling schemes" in which a user is saving, for example, a report type or a game state without actually creating a file that is exposed to the user in Windows Explorer.
