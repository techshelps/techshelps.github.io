Submit a Sample Application for Testing of Your Development Tool

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

Development tools are required to submit a sample application in uncompiled form for testing. When compiled, this sample application must demonstrate the following:

  • Win32® PE format
  • Large and small icons for its executable file
  • The ability to save/retrieve files with long file names
  • The ability to save/retrieve files with UNC file names
  • OLE container and/or server drag and drop functionality

Remarks

Sample applications are not required to meet any of the other Designed for Microsoft Windows Logo requirements, such as install/uninstall, and so forth.
