Use Microsoft Installer Technology

[This is preliminary documentation and subject to change.]

Requirement  (Designed for Microsoft® Windows® Logo)

Starting with the releases of Windows NT 5.0 and Windows 98, applications must use the new Microsoft installation technology for installing and uninstalling. This technology will greatly facilitate meeting all of the other install and uninstall requirements for the Designed for Microsoft Windows Logo program. This technology will also make your application far easier to manage in a complex environment.
