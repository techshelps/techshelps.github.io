File System

[This is preliminary documentation and subject to change.]

These are guidelines that relate to providing consistent, up-to-date file system support for Microsoft® Windows®:

  • Do not assume a hard-drive size limit of 2 GB. (Required)
  • Support the Universal Naming Convention (UNC). (Required)
  • Support long file names for files exposed to users. (Required)
  • Support long file names for files exposed in Windows Explorer. (Required)
  • Use long file names in user-interface elements. (Required)
  • Test LFN functionality on Windows NT FAT, NTFS, and compressed NTFS partitions. (Required)
  • Be file-system independent. (Best Practice)
  • Hide program files and directories that users do not need to see. (Best Practice)
