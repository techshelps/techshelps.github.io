Other Registry Keys Used in Policy

[This is preliminary documentation and subject to change.]

HKEY_CURRENT_USER Keys

HKEY_CURRENT_USER\
  Control Panel\
    Appearance

Current
 Specify the Desktop Color Scheme.

HKEY_CURRENT_USER\
  Control Panel\
    Desktop

TileWallpaper
 Specify the tiling of Desktop wallpaper. 
Wallpaper
 Specify the path of Desktop wallpaper.

HKEY_CURRENT_USER\
  Software\
    Microsoft\
      Windows NT\
        CurrentVersion\
          Winlogon\

ParseAutoexec
 Include environment variables contained in Autoexec.bat in user's environment. 
RunLogonScriptSync
 Run logon scripts synchronously (wait for completion).

HKEY_CURRENT_USER\
  Software\
    Microsoft\
      Windows\
        CurrentVersion\
          Explorer\
            Tips\

Show
 Show welcome screen and tips at logon.

HKEY_LOCAL_MACHINE Keys

HKEY_LOCAL_MACHINE\
  Software\
    Microsoft\
      Windows NT\
        CurrentVersion\
          Winlogon\

LegalNoticeCaption
 The caption of a pre-logon dialog box/banner. 
LegalNoticeText
 The text of a pre-logon dialog box/banner. 
ShutdownWithoutLogon
 Disable the Shutdown button in the logon dialog. 
DontDisplayLastUserName
 Disable display of last logged-on user name. 
RunLogonScriptSync
 Force the logon script to run synchronously (wait for it to finish). 
DeleteRoamingCache
 Delete cached copies of roaming profiles. 
SlowLinkDetectEnabled
 Automatically detect slow network connections. 
SlowLinkTimeOut
 Set timeout for slow network connections. 
ProfileDlgTimeOut
 Set timeout for user profile dialog boxes.

HKEY_LOCAL_MACHINE\
  System\
    CurrentControlSet\
      Control\
        FileSystem\

NtfsDisable8dot3NameCreation
 Disable creation of 8.3 file names for long names. 
NtfsAllowExtendedCharacterIn8dot3Name
 Allow extended characters in 8.3 file names. 
NtfsDisableLastAccessUpdate
 Do not update the last access time for files that are only read (increases system performance).

HKEY_LOCAL_MACHINE\
  System\
    CurrentControlSet\
      Control\
        Print\

DisableServerThread
 Disable browse thread on this computer. 
SchedulerThreadPriority
 Set scheduler priority. 
BeepEnabled
 Enable beep on remote print server job error.

HKEY_LOCAL_MACHINE\
  System\
    CurrentControlSet\
      Control\
        Update\

UpdateMode
 Controls remote update of remote policies. 
NetworkPath
 Path for policy file used in manual update mode. 
Verbose
 Controls display of error messages in update. 
LoadBalance
 Controls load balancing.

HKEY_LOCAL_MACHINE\
  System\
    CurrentControlSet\
      Services\
        LanManServer\
          Parameters\

AutoShareWks
 Creates administrative workstation drive shares. 
AutoShareServer
 Creates administrative server drive shares.

HKEY_LOCAL_MACHINE\
  System\
    CurrentControlSet\
      Services\
        RemoteAccess\
          Parameters\

AuthenticateRetries
 Set maximum number of unsuccessful authentication retries. 
AuthenticateTime
 Set maximum time limit for authentication. 
CallbackTime
 Set wait interval for callback. 
AutoDisconnect
 Set idle time before RAS auto-disconnect.
