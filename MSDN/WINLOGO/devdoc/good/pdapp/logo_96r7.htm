Use DirectX Compatibly with Windows NT Versions

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

All applications that use Microsoft DirectX® must fail gracefully on Windows NT® 3.51 but must function properly on Windows NT 4.0 and later.

See Also

For more information about DirectX 5.0, see http://www.microsoft.com/directx.
