Support Microsoft Transaction Server

[This is preliminary documentation and subject to change.]

Best Practice   (Designed for Microsoft® Windows® Logo)

Your application should provide support for Microsoft Transaction Server. Among the requirements for Microsoft Transaction Server support are:

  • Deploy the application as a Microsoft Transaction Server package.
  • Use Microsoft Transaction Server for context management.
  • Use a Microsoft Transaction Server resource dispenser.
  • Use role-based security.

See Also

For information about Microsoft Transaction Server, see the appropriate topics in the Platform SDK and at http://www.microsoft.com/com.
