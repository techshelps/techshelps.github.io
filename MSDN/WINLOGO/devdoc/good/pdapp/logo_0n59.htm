New Areas of Focus of the Logo Program

[This is preliminary documentation and subject to change.]

There are several areas where the Designed for Microsoft® Windows® logo program has introduced new requirements, recommendations, and suggested practices for Windows applications:

  • Supporting networked and managed environments. As more and more personal computers are operating in large, complex networked environments, applications must take a share of the responsibility to help administrators manage these environments for everyone's benefit.
  • Managed software installation. Microsoft is adding new managed software installation technology to Windows 95, Windows 98, and Windows NT. This technology makes it much easier to install and uninstall your application correctly. For more information, see Introduction to New Microsoft Installer Technology .
  • Avoiding DLL Version Conflicts. Take up a little more disk space in order to save your users from potential problems.
  • Upcoming COM+ Technology. COM+ will make your COM objects easier than ever to use and re-use.
  • Upcoming Directory Services Technology. Take advantage of the new directory services provided with Windows NT 5.0.
