Save Data to the Best Locations

[This is preliminary documentation and subject to change.]

One of the things that often confuses and frustrates users is the difficulty of locating files and programs on their computers. Designed for Microsoft® Windows® Logo requirements that relate to installing programs and saving data in locations that minimize such confusion and frustration are listed below.

This list is just a summary. To qualify for the Designed for Windows Logo, you must comply with all stipulations of the requirement reference text to which the list refers.

  • Install applications to the \Program Files directory.
  • Do not install executables or DLLs in the root directory.
  • Separate user data from application bits.
  • Query the registry for directory names.
