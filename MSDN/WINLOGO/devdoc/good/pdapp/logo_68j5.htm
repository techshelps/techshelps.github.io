Testing Rules

[This is preliminary documentation and subject to change.]

The requirements listed below relate to how an application is tested for Designed for Microsoft® Windows® Logo compliance.

This list is just a summary. To qualify for the Designed for Windows Logo, you must comply with all stipulations of the requirement reference text to which the list refers.

  • Bundled applications are tested individually.
  • Add-on products included in a suite are tested both individually and with the suite.
  • Pretest application installation.
