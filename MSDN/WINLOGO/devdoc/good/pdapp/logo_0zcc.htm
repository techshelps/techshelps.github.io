Do Not Decrement or Remove Core Components During Uninstall

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

The uninstaller must not decrement or remove any core component, in particular MicrosoftFoundation Class Library (MFC) DLLs, Open Database Connectivity (ODBC), and DAO DLLs.

See Also

For the most up-to-date and complete listing of all Windows NT and Windows 95 core components, see http://www.microsoft.com/windows/thirdparty/winlogo/.
