Use WSALookupServiceBegin in Clients to Find Windows Sockets Server Objects

[This is preliminary documentation and subject to change.]

Recommendation   (Designed for Microsoft® Windows® Logo)

To find the Windows Sockets objects in the Directory Service, Windows Sockets clients should look up server endpoints using the WSALookupServiceBegin function and related functions.

See Also

For more information about Windows NT 5.0 Directory Services, see http://www.microsoft.com/ntserver/info/directoryservices.htm.
