Support AutoPlay of CD-ROMs

[This is preliminary documentation and subject to change.]

Requirement  (Designed for Microsoft® Windows® Logo)

Products distributed on CD-ROM must use the AutoPlay feature to begin setup or to launch the program the first time the application is run. It is up to the vendor whether AutoPlay is enabled on subsequent insertions of the CD-ROM. In the case of products distributed on multiple CD-ROMs, subsequent CD-ROMs must either use the AutoPlay feature or behave during installation like a subsequent floppy disk (that is, continue installation without prompting the user for action). It is not acceptable to require the user to go to the Run command on the Windows Start menu to start the subsequent CD-ROM during installation.
