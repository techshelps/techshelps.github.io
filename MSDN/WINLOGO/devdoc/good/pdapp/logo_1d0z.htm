Use DirectX 5.0 for Multimedia Capabilities

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

Games and multimedia applications must support Microsoft DirectX® 5.0 (or later) as described below or use the Microsoft Win32® multimedia subsystem API.

  • Applications with 2-D graphics output should be produced using Microsoft DirectDraw® in either window or full-screen mode.
  • Applications providing sound playback should use Microsoft DirectSound®.
  • Applications with game support for keyboard, mouse, and gaming devices (such as joysticks and game pads) should use Microsoft DirectInput®.
  • Applications with hardware accelerated 3-D graphics should be accessed through Microsoft® Direct3D®.
  • Applications with advanced services for media integration, animation, or streaming of 2-D, 3-D, video, and audio should support Microsoft® DirectX® media (Microsoft ActiveMovie™ 2.0).
