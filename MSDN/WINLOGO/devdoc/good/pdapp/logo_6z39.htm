Document the Keyboard User Interface

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

The keyboard user interface must be documented.

Remarks

Implementation information may be provided in your standard documentation that is included with the application, or it may be orderable separately. In the latter case, it is strongly recommended that the standard documentation inform the user that this additional documentation is available.

Exception

It is not necessary to document elements that simply follow the Windows conventions for elements such as standard menus and controls.
