Use Direct3D and DirectSound Compatibly with Windows NT 4.0 and Windows 95

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

All applications that use the Microsoft Direct3D® or Microsoft DirectSound® API must provide a comparable user experience on both Windows NT® 4.0 and Windows 95.

See Also

For more information about Microsoft DirectX 5.0, see http://www.microsoft.com/directx.
