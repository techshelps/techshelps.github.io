Provide Meaningful Functionality on Both Windows 95 and Windows NT

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

For a utility product to receive the Logo, it must include meaningful functionality for both Windows 95 and Windows NT® environments in the same package (box).

Some products, such as disk utilities, implement operating-system-specific functionality that cannot be implemented on the other platform. For example, a set of utilities might include a disk defragmenter that operates on Windows NT but not on Windows 95. Nonetheless, such products must still include meaningful functionality for the other operating system.
