Upcoming Directory Services Technology

[This is preliminary documentation and subject to change.]

You may well want to take advantage of the new directory services being introduced with Windows NT® 5.0.

Designed for Microsoft® Windows® Logo guidelines in the Directory Services section recommend that you use a product-specific Directory Services object in the configuration/services container, use RpcNsBindingLookupBegin in RPC clients to find RPC server objects, and use WSALookupServiceBegin in clients to find Windows Sockets server objects. Locate COM objects in the Class Store transparently in COM and DCOM, and publish only computer-specific parameters in the registry.
