Use REG_EXPAND_SZ Strings on Windows NT 5.0

[This is preliminary documentation and subject to change.]

Best Practice   (Designed for Microsoft® Windows® Logo)

For Windows NT 4.0, Windows 95, and Windows 98, the REG_SZ notation should be used for all registry entries requiring string values. However, for Windows NT 5.0, the REG_EXPAND_SZ notation should be used whenever possible for all string entries containing asterisks.
