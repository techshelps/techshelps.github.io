Using the Logo

[This is preliminary documentation and subject to change.]

Before using the Logo, please read the Microsoft® Windows® and Microsoft® Windows NT® trademark guidelines available at:

http://www.microsoft.com/windows/thirdparty/winlogo/

Once your product has passed testing and you have received a copy of the Microsoft® Windows NT® and Windows® 95 Logo License Agreement signed by Microsoft, you can download an electronic copy of the Logo artwork kit from a secure web server using a password provided to you by VeriTest. This kit also contains electronic copies of the Logo and Logo usage guidelines.

Once signed by Microsoft, the Microsoft® Windows NT® and Windows® 95 Logo license allows you to use the Logo on packaging, advertising, your Web site, and other promotional materials.

If your product box will be printed before you obtain the license...

If your product box will be printed before you obtain the license to use the Designed for Microsoft® Windows NT® and Windows® 95 Logo, you can order Logo stickers from Special Effects, Seattle, WA (an outside vendor) at specialeffex@msn.com. The stickers will be delivered when you receive the license to use the Logo.
