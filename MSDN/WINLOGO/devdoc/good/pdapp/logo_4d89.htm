Publish Only Machine-Specific Parameters in the Registry

[This is preliminary documentation and subject to change.]

Recommendation   (Designed for Microsoft® Windows® Logo)

Your application should publish only machine-specific parameters in the registry. User-specific parameters should be published on the User object for the user in question. Global application parameters should be published in the Configuration/Services container.

See Also

For more information about Windows NT 5.0 Directory Services in general, see http://www.microsoft.com/ntserver/info/directoryservices.htm. See also Use a product-specific DS object in the configuration/services container.
