Generate Browsable Output

[This is preliminary documentation and subject to change.]

Best Practice   (Designed for Microsoft® Windows® Logo)

Your application should produce documents that can be viewed from a browser. You can do this either by implementing a Save As HTML feature or by adding ActiveX document (docobj) support to the application.

See Also

For more information about adding ActiveX documents, see the ActiveX SDK at http://www.microsoft.com/IntDev/sdk.
