All Requirements

This section lists all requirements for the Designed for Microsoft® Windows® Logo, but does not include any recommendations or best practices.

The list is just a summary. To qualify for the Designed for Windows Logo, you must comply with all stipulations of the requirement reference text to which this list refers.
