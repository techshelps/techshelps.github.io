Use and Redistribute the 32-bit Virtual Machine for Java Applications

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

To qualify for the Designed for Microsoft Windows Logo, a Java application must use and redistribute the Microsoft Win32® virtual machine for Java when running on a Windows-based computer.

See Also

For information about using and redistributing the Microsoft Win32 virtual machine for Java, see http://www.microsoft.com/java or the Microsoft SDK for Java.
