Do Not Assume That Profile Settings Have Been Initialized

[This is preliminary documentation and subject to change.]

Recommendation   (Designed for Microsoft® Windows® Logo)

If an application finds that user settings or data are not present in the user's profile, the application should recover gracefully and regenerate the necessary settings based on the application defaults.

Remarks

In a multiuser environment, an application can be installed on a computer without the opportunity for setup to initialize HKEY_CURRENT_USER (HKCU) for the specific user starting the application.
