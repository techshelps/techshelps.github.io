Installing and Removing Applications

[This is preliminary documentation and subject to change.]

The guidelines in this section relate to making applications easy to install and uninstall. They particularly apply to installing and removing entire applications, as opposed to individual components:

  • Use Microsoft installer technology. (Required)
  • Provide graphical 32-bit setup. (Required)
  • Provide a fully automated uninstaller. (Required)
  • Provide attended and/or silent installation. (Required)
  • Detect versions when installing. (Required)
    • Maintain user settings across version changes. (Best Practice)
  • Create working shortcuts during installation. (Required)
  • Support autoplay of CD-ROMs. (Required)
  • Support Add/Remove Programs if not on CD-ROM. (Required)
  • Check operations in advance when installing. (Required)
  • Avoid rebooting the system. (Best Practice)
  • Register uninstaller and make it appear in Add/Remove Programs. (Required)
  • Uninstall from Add/Remove Programs. (Required)
  • Remove all application files during uninstall. (Required)
  • Remove all references from the Start Menu during uninstall. (Required)
  • Remove registry entries during uninstall. (Required)
  • Remove uninstaller during uninstall. (Required)
