Store Hardware-Independent Machine Settings in HKEY_LOCAL_MACHINE

[This is preliminary documentation and subject to change.]

Best Practice   (Designed for Microsoft® Windows® Logo)

Information concerning a computer's logical (as opposed to physical) configuration should be stored in the HKEY_LOCAL_MACHINE (HKLM) hive of the registry.

Such information is distinct from information that pertains to a specific user (which should be stored in HKEY_CURRENT_USER) and from information that pertains to the physical computer hardware (which should be stored in HKEY_CURRENT_CONFIG).
