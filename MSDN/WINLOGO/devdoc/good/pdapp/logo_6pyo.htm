Save Data to the Best Locations

[This is preliminary documentation and subject to change.]

One of the things that often confuses and frustrates users is the difficulty of locating files and programs on their computers. Designed for Microsoft® Windows® Logo recommendations and best practices that relate to installing programs and saving data in locations that minimize such confusion and frustration are listed below. The list is just a summary. To comply with the recommendations and best practices, you must follow all stipulations of the reference text to which the lists refer.

  • Do not write to Windows system directories. (Recommended)
  • Use Microsoft update packs to upgrade system components. (Recommended)
  • Do not install globally shared DLLs. (Recommended)
  • Store user data under the user's Profile folder. (Recommended)
  • Do not assume that profile settings have been initialized. (Recommended)
