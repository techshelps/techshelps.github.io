Register Native Data File Types

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

Native data file types (if applicable) must be registered as follows:

[HKEY_CLASSES_ROOT]
.(file type extension)
    (Default) = REG_SZ:FileTypeID

See Also

For further details on registering file types and application data, see the Windows Interface Guidelines for Software Design, Chapter 10 (available from Microsoft Press).
