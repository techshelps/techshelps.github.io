Games and Multimedia Applications

[This is preliminary documentation and subject to change.]

Games and multimedia applications are a special case primarily because of their use of DirectX technologies, which are supported differently on different versions of Microsoft® Windows®.

Special Eligibility for the Designed for Windows 95 Logo

Products that use Direct3D or DirectSound are eligible for the Designed for Windows® 95 Logo. No other products are eligible.

Products that use the Direct3D and DirectSound APIs without updating Windows NT using SP3 may be granted the Designed for Windows® 95 Logo provided that the product does not load on Windows NT and displays a meaningful user message.

Special Requirements for Games and Multimedia Applications

  • Use DirectX compatibly with Windows NT Versions. (Required)
  • Use Direct3D and DirectSound compatibly with Windows NT 4.0 and Windows 95. (Required)
  • Use DirectX 5.0 for multimedia capabilities. (Required)
