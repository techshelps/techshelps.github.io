Managed Software Installation

[This is preliminary documentation and subject to change.]

With Windows NT® 5.0, Microsoft is delivering new installation and component management technology that will greatly simplify the installation and uninstallation of applications. This technology will function as a gateway to the registry, handling reference counting, component tracking, and other aspects of installation and uninstallation. Microsoft is working with major vendors of installation tools to ensure that toolkits will be available to take advantage of this new technology.

  • Use of the new managed installation technology for installation and uninstallation is a requirement for the Windows NT® 5.0 release.

Today's installation requirements and the recommended changes in registry usage reflect incremental steps toward a model in which registry entries will be managed by a central operating system service.

Applications that are developed using the new installation technology will be compliant by default with the majority of the installation and uninstallation Logo requirements.

See the section entitled Be Easy to Install and Remove for the specific Designed for Microsoft® Windows® Logo guidelines relating to installation and uninstallation.
