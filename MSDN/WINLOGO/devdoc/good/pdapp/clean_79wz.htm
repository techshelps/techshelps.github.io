Registry Keys for Standard Folder Locations

[This is preliminary documentation and subject to change.]

The following registry keys contain the locations of various standard folders. These locations can be set by administrators as a part of policy.

HKEY_CURRENT_USER\
  Software\
    Microsoft\
      Windows\
        CurrentVersion\
          Explorer\
            User Shell Folders\

Desktop
 Specify the folder displayed on the Desktop. 
NetHood
 Specify the folder displayed when navigating Network Neighborhood. 
Programs
 Specify the folder displayed by Start menu, Programs. 
Start Menu
 Specify the folder displayed as the Start menu. 
Startup
 Specify the Startup folder (displayed in Start menu, Programs, Startup).

HKEY_LOCAL_MACHINE\
  Software\
    Microsoft\
      Windows\
        CurrentVersion\
          Explorer\
            User Shell Folders\

Common Programs
 Specify the common groups folder displayed in Start menu, Programs. 
Common Desktop
 Specify a folder of shared items displayed on the desktop. 
Common Start Menu
 Specify a folder of shared items displayed in the Start menu. 
Common Startup
 Specify a folder of shared startup items.
