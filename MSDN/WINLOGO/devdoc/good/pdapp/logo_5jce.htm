Extensively Test Any Utility That Uses the Exclusive Volume Locking Function

[This is preliminary documentation and subject to change.]

Requirement   (Designed for Microsoft® Windows® Logo)

Utility products that use the Exclusive Volume Locking API to access the 8.3 aliases directly and manipulate them (such as disk optimizers, file utilities, and antivirus software), must be tested extensively to ensure that these products function properly and do not damage the file allocation table or the long file name structure.
