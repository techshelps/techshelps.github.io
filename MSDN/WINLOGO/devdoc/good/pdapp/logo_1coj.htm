Save Data to the Best Locations

[This is preliminary documentation and subject to change.]

One of the things that often confuses and frustrates users is the difficulty of locating files and programs on their computers. These guidelines relate to installing programs and saving data in locations that minimize such confusion and frustration:

General

  • Install applications to the \Program Files directory. (Required)
  • Do not install executables or DLLs in the root directory. (Required)
  • Separate user data from application bits. (Required)
  • Query the registry for directory names. (Required)
  • Store user data under the user's Profile folder. (Recommended)
  • Do not assume that profile settings have been initialized. (Recommended)

Avoid DLL Versioning Problems

  • Do not write to Windows system directories. (Recommended)
  • Use Microsoft update packs to upgrade system components. (Recommended)
  • Do not install globally shared DLLs. (Recommended)
