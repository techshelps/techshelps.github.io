Installation and Setup

For general installation issues, refer to the Getting Started card included in both the Professional and Enterprise editions of Visual Studio.

You also can refer to the Readme files, located in the root of the installation compact disc. The Readme files contain detailed information on installation issues for all of the products in Visual Studio 6.0.

The Readme files are in HTML format and can be viewed with an Internet browser like Internet Explorer 4. For information on locating Readme files, refer to the topic Locating Readme Files.
