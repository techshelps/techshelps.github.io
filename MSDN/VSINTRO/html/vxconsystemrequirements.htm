System Requirements

The system requirements for Visual Studio are as follows:

Visual Studio, Professional Edition

  • Pentium personal computer with 90 MHz or higher processor

  • Microsoft Windows 95 or later operating system or Microsoft Windows NT Workstation operating system version 4.0 (Service Pack 3) or later

  • 24 MB of RAM (32 MB recommended) for Windows 95; 24 MB of RAM (32 MB recommended) for Windows NT Workstation 4.0

  • Microsoft Internet Explorer 4 and Internet Explorer 4 Service Pack 1 (included)

Hard disk space required for installation:

  • Visual Basic (54 MB Typical, 94 MB Maximum)

  • Visual C++  (266 MB Typical, 370 MB Maximum)

  • Visual FoxPro (85 MB Typical, 90 MB Maximum)

  • Visual InterDev  (81 MB Typical, 98 MB Maximum)

  • Visual J++ (86 MB Typical, 104MB Maximum)

Additional hard disk space required for the following products:

  • Internet Explorer (43 MB Typical, 59 MB Maximum)

  • MSDN Library (57 MB Typical, 493 MB Maximum)

  • Windows NT Option Pack (20 MB Typical, 200 MB Maximum)
  • CD-ROM drive

  • VGA or higher-resolution monitor (Super VGA recommended)

  • Microsoft Mouse or compatible pointing device

Visual Studio, Enterprise Edition

  • Pentium personal computer with 90 MHz or higher processor

  • Microsoft Windows 95 or later operating system or Microsoft Windows NT Workstation operating system version 4.0 (Service Pack 3) or later

  • 24 MB of RAM (32 MB recommended) for Windows 95; 24 MB of RAM (32 MB recommended) for Windows NT Workstation 4.0

  • Microsoft Internet Explorer 4 and Internet Explorer 4 Service Pack 1 (included)

Hard disk space required for installation:

  • Visual Basic (116 MB Typical, 135 MB Maximum)

  • Visual C++  (302 MB Typical, 403 MB Maximum)

  • Visual FoxPro (85 MB Typical, 90 MB Maximum)

  • Visual InterDev  (81 MB Typical, 98 MB Maximum)

  • Visual J++ (86 MB Typical, 104MB Maximum)

Additional hard disk space required for the following products:

  • Internet Explorer (43 MB Typical, 59 MB Maximum)

  • MSDN Library (57 MB Typical, 493 MB Maximum)

  • Windows NT Option Pack (20 MB Typical, 200 MB Maximum)

  • Microsoft SNA Server 4.0 (50 MB Typical, 100+ MB Maximum)

  • Microsoft SQL Server 6.5 (80 MB Typical, 95 MB Maximum)
  • CD-ROM drive

  • VGA or higher-resolution monitor (Super VGA recommended)

  • Microsoft Mouse or compatible pointing device
