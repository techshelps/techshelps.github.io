Microsoft Visual FoxPro Accessibility

Enlarging Toolbar Buttons

To enlarge toolbar buttons

  1. On the View menu, click Toolbars.

  2. Select the Large buttons check box.

Enlarging Text in the Code Window

To enlarge text in the Code window

  • On the Format menu, click Enlarge Font.

  • -or-

  • On the Format menu, click Font and choose the font size you prefer.

-or-

  1. With the Command window active, right-click and choose Properties on the Shortcut menu.

  2. In the Edit Properties dialog box, click the Font button and choose the font size you prefer.

Enlarging Text in the Debugger

To enlarge text in the Debugger

  1. On the Options menu, click the Debug tab, and then click Font.

  2. Select the font and font size you prefer.
