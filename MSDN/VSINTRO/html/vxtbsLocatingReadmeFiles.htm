Locating Readme Files

The location of the Readme*.htm files is determined by the installation option you selected.

Visual Studio default installation: If you chose the default installation, then you will find the Readme*.htm files at drive:\..\Program Files\Microsoft Visual Studio.

Visual Studio stand-alone product: If you installed a stand-alone product, such as Visual Basic or Visual InterDev, and you chose the default installation, then you will find Readme*.htm files at drive:\..\Program Files\productname.

Custom installation: If you selected Custom installation and cannot find the Readme files, then look in compact disc 1. All Readme files in the Visual Studio suite of products are HTML files (*.htm) stored at the root of CD 1.

Use your Internet browser to open and view the files.
