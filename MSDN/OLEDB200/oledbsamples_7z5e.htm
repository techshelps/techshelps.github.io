Rowset Viewer Menus and Toolbar

Four menus are built into Rowset Viewer: the main menu and a separate context menu for each window in the MDI child window. All menu items display an explanation of the functionality in the status bar.

Main Menu

The Main menu displays all OLE DB CoType objects, with all interfaces and methods for the object in the drop-down list. All supported interfaces are enabled, and interfaces that are not supported by the provider are disabled.

Command Menu

Right-click the Command window to display common methods and actions of the command object.

Rowset Menu

Right-click Rowset (data grid) to display common methods and actions of the rowset object.

Notification Menu

Right-click the Notification window to display common methods and actions for filtering and receiving notifications.

Toolbar

Use the toolbar for actions that are used frequently. ToolTips are available to help you determine the action of each item.
