ReleaseRows

HRESULT ReleaseRows(
const HROW *pRows, 
ULONG cRows);

Parameters

pRows

Array of HROWs to release.

cRows

The number of HROWs in the array.

Return Code

 S_OK

 Method succeeded.

E_FAIL

Returned if the object is not attached.

Comments

Uses IRowset::ReleaseRows to release the HROWs in the given array.
