Storage Alternatives

Microsoft® PivotTable® Service can store multidimensional data in three data cube storage scenarios:

  1. Create a data cube and load data from an OLEDB rowset provider or a SQL relational database. Microsoft PivotTable Service can be disconnected from the data source after the data cube is created and stored locally on the client as a MOLAP cube.

  2. Create a data cube definition and store it as a ROLAP cube. Dimensions and data are not stored.

  3. Create a data cube definition with dimension definitions and store it locally.
