Document Conventions

This document uses the following typographic conventions.

Convention Description
Bold Bold type indicates language keywords (for example, interfaces, properties, and methods; registry keys and subkeys) and other syntax that must be typed exactly as shown.
italics Italic type indicates variables or placeholders for information you supply, or word emphasis.
UPPERCASE Uppercase letters indicate flags, enumeration constants, data types, and structures. 
Monospace
 This font is used for program code and error message text.
