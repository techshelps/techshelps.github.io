Data Conversion Library

The OLE DB Data Conversion Library manages data type conversions for OLE DB consumers and providers. The OLE DB Data Conversion Library uses the IDataConvert interface to expose data conversion methods to consumers. Included in the documentation of the Data Conversion Library is a list of data type conversions Supported by MSDADC.DLL. For details about conversions, see "Data Type Conversions Supported by MSDADC.DLL."
