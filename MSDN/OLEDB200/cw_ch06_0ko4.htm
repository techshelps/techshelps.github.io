IDataSourceListener::dataMember Removed

Notifies a data consumer that a member has been removed from the enumeration of well-known data members.

HRESULT dataMemberRemoved(
DataMember   bstrDM);

Parameters

bstrDM [in]

Name of the data member that has been removed.

Return Code

S_OK

The method succeeded.

E_FAIL

A provider-specific error occurred.
