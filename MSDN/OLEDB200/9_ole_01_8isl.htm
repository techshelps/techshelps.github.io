Programming OLE DB with the SQL Server OLE DB Provider

SQLOLEDB, the Microsoft® SQL Server™ Microsoft SQL Server OLE DB provider, exposes interfaces to consumers wanting access to data on one or more SQL Servers.

When developing an OLE DB consumer, select a provider developed for the data source you want to consume. Use SQLOLEDB to develop an optimized OLE DB consumer for SQL Server databases.

SQLOLEDB is an OLE DB–compliant provider.

The provider recognizes ANSI SQL and Transact-SQL statements as text for OLE DB commands.
