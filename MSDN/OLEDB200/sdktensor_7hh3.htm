Microsoft OLE DB for OLAP Overview

Microsoft® OLE DB for OLAP (Online Analytical Processing) is a set of objects and interfaces that extends OLE DB to provide access to multidimensional data sources.

OLE DB for OLAP (Online Analytical Processing) is a set of COM (Component Object Model) objects and interfaces designed to facilitate communication between providers and consumers of multidimensional data. OLE DB for OLAP is an extension of OLE DB version 2.0 to provide access to multidimensional data sources.

Microsoft PivotTable Service

The Microsoft® PivotTable® Service is an OLE DB for OLAP provider. It is an in-process desktop OLAP server designed to provide online and offline data analysis functionality, and online access to OLAP data as a client of the Microsoft Decision Support Services OLAP Server (which is not included with Microsoft OLE DB for OLAP).

Microsoft PivotTable Service contains data analysis, cube building, and smart cache management functionality. It provides interfaces to data as well as functionality to manipulate data.
