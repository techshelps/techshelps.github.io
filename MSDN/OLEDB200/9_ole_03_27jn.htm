System Requirements

SQLOLEDB requires a system running:

  • The Windows 95® operating system on Intel® computers.
    – or -
    The Windows NT® operating system (version 4.0 or later) on Intel or Alpha computers.

  • An appropriately installed and configured SQL Server client Net-Library.

SQLOLEDB supports OLE DB consumers that access SQL Server databases. To use SQLOLEDB, a SQL Server must be installed and running in your enterprise.

Consumer Development

A C or C++ compiler supporting OLE/COM component construction is required to develop an SQLOLEDB consumer.
