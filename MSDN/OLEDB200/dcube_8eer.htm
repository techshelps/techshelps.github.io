Rowset Objects

Rowset objects are described in the OLE DB specification.

Supported Interfaces

  • IAccessor

  • IColumnsInfo

  • IConvertType

  • IDBAsynchStatus

  • IRowset

  • IRowsetLocate

  • ISupportErrorInfo

Microsoft PivotTable Service Issues

The OLE DB SDK includes a component that is used by Microsoft® PivotTable® Service for rowset support. This component may impose additional limitations (and may be replaced by a different tool in future versions).

Microsoft PivotTable Service uses the OLE DB Simple Provider component distributed with the OLE DB SDK for rowset support, which may impose other limitations. A different tool may be implemented in a later release.

All member properties are supported with Flattened Rowsets, as defined in the OLE DB for OLAP specification.

Limitations

All rowsets are read-only.
