Session Properties

The DBPROPSET_SESSION property set contains the following properties. All of these properties are in the SESSION property group.

Property ID Description
DBPROP_SESS_AUTOCOMMITISOLEVELS Type: VT_I4

Typical R/W: R only

Description: Autocommit Isolation Levels

Specifies a bitmask that indicates the transaction isolation level while in autocommit mode. The values that can be set in this bitmask are the same as those that can be set for DBPROP_SUPPORTEDTXNISOLEVELS.
