Wrapper Class Reference

This chapter describes the wrapper classes that accompany the ActiveX Control Writer’s SDK for OLE DB. You can find the source code for the wrapper classes in the Asbsdk\Inc and Absdk\Src\Adbctl directories of the SDK. You can also load the Adbctl sample (Adbctl.dsw) into Microsoft Visual C++ and use the ClassView pane to view the wrapper classes in the SDK.

In this chapter, the wrapper classes are categorized by usage:

  • Rowsets

  • Miscellaneous

  • IStream

  • String Handling

  • Bookmarks

  • Persistence

  • Property Pages

  • Additional Data Access Support

  • Advise Connections
