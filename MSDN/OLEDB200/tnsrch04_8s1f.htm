Chapter 4

OLE DB for OLAP Objects and Schema Rowsets

This chapter describes the objects and rowsets available through the OLE DB for OLAP interfaces. The following objects, which are unique to OLE DB for OLAP or differ from generic OLE DB objects when used with OLE DB for OLAP, are described in the following sections:

  • Enumerator object

  • Data source object

  • Session object

  • Dataset object

OLE DB for OLAP adds a number of properties to the DBPROPSET_DATASOURCEINFO property group of a data source object. These properties give the consumer information about variations among multidimensional providers. These properties are listed and explained in the section “Data Source Object.”

OLE DB for OLAP uses a number of schema rowsets that are new to OLE DB. They are listed below, and they are described in full in the section “Session Object.”

  • CUBES rowset

  • DIMENSIONS rowset

  • HIERARCHIES rowset

  • LEVELS rowset

  • MEASURES rowset

  • MEMBERS rowset

  • PROPERTIES rowset

The description of a dataset’s axes also requires descriptions of the MDAXISINFO structure and the layout of the axes rowsets. These are described in the section “Dataset Object.”

Finally, the rowset object plays an essential role in OLE DB for OLAP. A discussion of this appears in the section “Role of the Rowset Object in OLE DB for OLAP.”
