IGetDataSource::GetDataSource

Returns an interface pointer on the data source object that created the session.

HRESULT GetDataSource (
   REFIID       riid,
   IUnknown **   ppDataSource);

Parameters

riid

[in]
The IID of the interface on which to return a pointer.

ppDataSource

[out]
A pointer to memory in which to return the interface pointer. If GetDataSource fails, it must attempt to set *ppDataSource to a null pointer.

Return Code

S_OK
The method succeeded.

E_FAIL
A provider-specific error occurred.

E_INVALIDARG
ppDataSource was a null pointer.

E_NOINTERFACE
The object that created this session did not support the interface specified in riid.

See Also

ICommand::GetDBSession, IRowsetInfo::GetSpecification
