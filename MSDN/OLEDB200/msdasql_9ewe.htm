Introduction

The OLE DB SDK contains an OLE DB provider for ODBC data. This document describes the ODBC Provider, explains the procedures for installing, configuring, and testing the provider, and lists the limitations of the current implementation.

OLE DB Specification Compliance

The ODBC Provider is compliant with version 1.5 and above of the OLE DB Specification, which is included in the OLE DB SDK.

ODBC Version

Make sure that you use the latest version of ODBC (ODBC 3.51) with the ODBC Provider.
