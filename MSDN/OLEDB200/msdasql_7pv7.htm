Installation Notes

The Setup program installs the following files to the hard disk of your workstation or server. Provider writers need to build an installation program that will copy the below files (and any other desired OLE DB Core Components, such as the root enumerator) and register the DLLs (using resvr32 dllname).

\Program Files\Common Files\System\OLE DB or
<Install Dir>\REDIST

File name Description Required by
MSDADC.DLL OLE DB data conversion object for the Data Conversion Library ODBC provider

OLE DB Core Component
MSDAENUM.DLL OLE DB enumerator object OLE DB Core Component
MSDAER.DLL OLE DB error object ODBC provider
MSDAERR.DLL Resource file for OLE DB Error Object Resource Library OLE DB error object
MSDASQL.DLL ODBC provider
MSDASQLR.DLL Resource file for ODBC Provider Resource Library ODBC provider
MSDATL.DLL Memory/Array manager Temp table service provider, error object, ODBC provider
MSDATL2.DLL Memory/Array Manager version 2 ODBC provider
MSDATT.DLL Temporary table service provider ODBC provider

<Install Dir>\INCLUDE

File name Description Required by
MSDASQL.H Provider-specific defined ODBC provider
MSDADC.H Data conversion defined ODBC provider
