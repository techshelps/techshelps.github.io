IMDFind Interface

This is an optional interface on the Dataset object. It has methods to:

  • Find the ordinal of a cell in the dataset, given the array of members that identify it (FindCell).

  • Find the tuple ordinals on an axis, given the constituent member names and a starting tuple ordinal (FindTuple).
interface IMDFind : public IUnknown
{
HRESULT FindCell(
   ULONG       ulStartingOrdinal,
   ULONG       cMembers,
   LPCOLESTR*  rgpwszMember,
   ULONG*      pulCellOrdinal);

HRESULT FindTuple(
   ULONG       ulAxisIdentifier,
   ULONG       ulStartingOrdinal,
   ULONG       cMembers,
   LPCOLESTR*  rgpwszMember,
   ULONG*      pulTupleOrdinal);
};
