Running the Sample Consumer

To run the sample consumer, click Start, point to Programs, point to OLE DB SDK, and click Sample Client For Text Provider, or run the <install directory>\BIN\SAMPCLNT.EXE file from Windows Explorer. The data file, CUSTOMER.CSV, must be in the <install directory>\BIN directory as well. The sample consumer logs the results of its run to the file <install directory>\BIN\SAMPCLNT.OUT; view this file to see the results of the run.
