IDataSource::getDataMemberName

Returns the name of a well-known data member.

HRESULT getDataMemberName(
long   lIndex
DataMember*   pbstrDM);

Parameters

lIndex [in]

An index for the list of data member names.

pbstrDM [out]

A string that describes the data member name.

Return Code

S_OK

The method succeeded.

E_FAIL

A provider-specific error occurred.

Comments

A zero-based index parameter is specified. The index must fall between 0 and GetDataMemberCount –1, or this method returns an error. The default data member ("") should never be returned from this method.
