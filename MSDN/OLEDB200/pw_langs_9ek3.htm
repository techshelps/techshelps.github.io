Language Differences

Although the process of building a provider with the Microsoft OLE DB Simple Provider Toolkit is generally the same in Microsoft Visual Basic, Microsoft Visual C++, and Microsoft Visual J++, with the major portion of the work consisting of implementing the OSP functions, there are some implementation differences between the visual languages. The following topics provide implementation information and programming guidelines for building a Simple Provider in each of the Microsoft visual languages:
