Physical Information Schema Rowsets

This section details the contents of each schema rowset in the physical information schema. These schema rowsets are primarily concerned with the physical database metadata. The contents of such metadata are not defined in ANSI SQL92; all definitions in this section are OLE DB extensions to ANSI SQL92.

The following rowsets are in the physical information schema. All providers that support IDBSchemaRowset must support the PROVIDER_TYPES schema rowset. Support for other schema rowsets in the physical information schema is optional.
