IMDDataset::FreeAxisInfo

Frees the array of MDAXISINFO structures returned by IMDDataset::GetAxisInfo.

HRESULT FreeAxisInfo(
   ULONG            cAxes,
   MDAXISINFO*      rgAxisInfo);

Parameters

cAxes[in]

Number of elements in the rgAxisInfo array.

rgAxisInfo[in]

A pointer to an array of MDAXISINFO structures.

Return Code

S_OK
The method succeeded.

E_FAIL
A provider-specific error occurred.

E_INVALIDARG
cAxes is not zero and rgAxisInfo is NULL.

cAxes is zero and rgAxisInfo is not NULL.

Comments

rgAxisInfo must point to the array returned by IMDDataset::GetAxisInfo. If not, the effect of this method is undefined.

If cAxes is zero or rgAxisInfo is a null pointer, then this method does nothing.
