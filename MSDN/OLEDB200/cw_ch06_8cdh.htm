Overview of API Reference

This chapter provides reference information about the Application Programming Interfaces (APIs) that ActiveX control writers use and implement with OLE DB. The chapter includes discussions of the following:

  • IRowPosition interface

  • IRowPositionChange interface

  • IRowsetFind interface

  • CoTypes, structures, and enumerated types

You can find these interfaces in Oledb.h.

Two other interfaces are also described:

  • IDataSource

  • IDataSourceListener

These interfaces are in \INCLUDE\Msdatasrc.h.
