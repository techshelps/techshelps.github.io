Introduction

This document introduces control writers to OLE DB control writing techniques. It isn't a comprehensive study of OLE DB; rather, it discusses how to create functional ActiveX® controls by using OLE DB. In addition, the document is a reference for migrating controls written using obsolete ICursor interfaces to OLE DB.

To get a full description of topics introduced in this document, refer to the OLE DB Programmer’s Reference.

In addition, you may refer to the following sources, which can be found in the Microsoft Developer Network™ (MSDN) in the Platform Software Development Kit (SDK):

  • Automation Programmer’s Reference

  • COM Programmer’s Reference
