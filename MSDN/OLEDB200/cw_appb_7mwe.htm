Advise Using IUnknown

HRESULT Advise(
IUnknown *pUnkAdvise);

Parameter

pUnkAdvise

Pointer IUnknown of the control to advise on. If this is NULL, then any current advise connection is disengaged.

Comments

Call to establish an IPropertyNotifySink connection with a control. This is called by the base class for the first control prior to passing the PPM_NEWOBJECTS to the PageDialogProc.
