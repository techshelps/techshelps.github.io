IDataInitialize

IDataInitialize is called to provide a common entry point for consumers to obtain a data source object (DSO) or get persistent data source information.

Method Description
CreateDBInstance Creates a data source object. Analogous to CoCreateInstance.
CreateDBInstanceEx Creates a data source object. Analogous to CoCreateInstanceEx.
GetDataSource Given initialization string, instantiates and returns a data source object.
GetInitializationString Given a data source object, returns stringed connection information.
LoadStringFromStorage Loads an initialization string from storage.
WriteStringToStorage Writes an initialization string to storage.

For more information about IDataInitialize, see “IDataInitialize” in the Data Link API Reference.
