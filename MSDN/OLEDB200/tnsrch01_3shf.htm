Future Directions

OLE DB for OLAP version 1.0 does not define any new interfaces or language clauses for updating cube schemas. However, providers can achieve the same effect by supporting existing mechanisms for updating rowsets in order to provide updatable schema rowsets. Vendors may also define proprietary language extensions for Data Definition Language (DDL) functionality.

Note   DDL functionality is not a goal for OLE DB for OLAP version 1.0. It is a goal for the next major version and should be considered part of the future direction of the OLE DB for OLAP specification.
