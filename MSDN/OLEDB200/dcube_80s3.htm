Dataset Objects

Dataset objects are described in the OLE DB for OLAP specification.

Supported Interfaces

  • IAccessor

  • IColumnsInfo

  • IConvertType

  • IDBAsynchStatus

  • IMDDataset

  • ISupportErrorInfo

Microsoft OLE DB for OLAP PivotTable Service Issues

IMDFind is not supported.

All cell properties are supported.

All member properties defined as columns in the schema rowset are supported.

Limitations

All query results are read-only.
