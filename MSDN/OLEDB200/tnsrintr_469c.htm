Document Conventions

This manual uses the following typographic conventions.

Format Used for
WIN.INI Uppercase letters indicate file names, MDX statements, macro names, and terms used at the operating-system command level. 
#include <io.h>
 Sans serif font is used for sample command lines and program code, or BNF constructs.
Argument Italicized words indicate information that the user or the application must provide, or word emphasis.
IMDFind Bold type indicates that syntax must be typed exactly as shown, including method names.

In addition, the following words and phrases have specific meanings in the OLE DB for OLAP specification:

  • Unless otherwise specified, the word cube refers to the metadata object in a multidimensional schema, not the geometrical object.

  • The phrase tabular data provider or TDP refers to any data provider that exposes its data in the form of an OLE DB rowset. The phrase multidimensional data provider or MDP refers to any data provider that exposes its data as a multidimensional hypercube. Any TDP that exposes OLE DB for OLAP is, by definition, an MDP as well as a TDP.
