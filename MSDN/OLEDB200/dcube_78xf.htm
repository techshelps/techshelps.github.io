Session Objects

Session objects are described in the OLE DB specification.

Supported Interfaces

  • IDBCreateCommand

  • IDBCreateSession

  • IDBSchemaRowset

  • IGetDataSource

  • IOpenRowset

  • ISessionProperties

  • ISupportErrorInfo

Microsoft PivotTable Service Issues

None.

Limitations

IOpenRowset is not supported.

ISessionProperties is not fully tested.
