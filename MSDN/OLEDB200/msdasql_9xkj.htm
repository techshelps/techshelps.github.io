Predefined ODBC Data Sources

As part of the ODBC Provider installation, the following 32-bit ODBC data sources are created:

  • OLE_DB_NWind_Jet. This data source points to the NorthWind Access database shipped with the OLE DB SDK. This is \msdasdk\bin\oledb\nwind.mdb.

  • OLE_DB_NWind_SQL. This data source points to the Microsoft SQL Server "pubs" database. The server is set to "." and the database name is set to "pubs". You will need to modify this data source to use it on your system.
