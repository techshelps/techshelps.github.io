System Requirements

The Microsoft OLE DB Provider for Oracle requires a system running:

  • The Windows 95 operating system on Intel computers

    – or –

    The Windows NT operating system (version 4.0 with Service Pack 3 or later) on Intel or Alpha computers.

  • Oracle client software (appropriately installed and configured).

  • An operational Oracle server running in your enterprise.

Note   Oracle networking components installed with an Oracle client, and Oracle7 Release 7.3.3 or later (including Oracle8.x) are also required. Oracle products are available from the Oracle Corporation.

Consumer Development

A C or C++ compiler supporting OLE/COM component construction is required to develop a consumer.
