Customizable development environment

The VBA environment now includes command bars, a new type of menu/toolbar system. This new system offers dockable and free-floating toolbars that are fully customizable, including the ability to let you create and modify custom toolbars. Developers can modify any built-in menu bars and shortcut menus. For example, the undocked toolbar shown in Figure 6-1 contains numerous buttons.
