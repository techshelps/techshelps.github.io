APPENDIX A Coding Conventions

MARK HURST

PETER MORRIS

This appendix describes the coding conventions we used in preparing most of the source code for this book. These conventions are based on extracts from our in-house (TMS) Visual Basic programmer’s manual, VB Vogue. You can use this appendix purely to look up type or scope prefixes you don’t understand, but you might also want to use our conventions in your own code. Or you might decide to adapt them. The important thing is that you have coding conventions and you apply them uniformly across your code.
