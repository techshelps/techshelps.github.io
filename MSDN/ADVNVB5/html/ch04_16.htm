HTTP Transfer Limitations

When you receive an HTML document via the OpenURL method, you don’t receive any of the embedded objects in these documents. These include graphic images (JPG and GIF), sounds, video files, and other multimedia content. What you do receive is the raw HTML source. Displaying raw HTML in a text box won’t give you the same results as viewing the file in a Web browser.

For best results in viewing the full content of an HTML file, you should use either Microsoft Internet Explorer or the Microsoft Web Browser ActiveX control.
