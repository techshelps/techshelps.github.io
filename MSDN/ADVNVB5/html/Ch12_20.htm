Listing the logins

The stored procedure sp_helplogins will list all logins, but only the system administrator can use it. However, here’s a SELECT statement, which anyone can execute, that does the same job.

SELECT name, suid FROM master.dbo.syslogins

name                           suid
------------------------------ ------ 
sa                             1      
probe                          10     
DuncanG                        11     
SallyG                         12     
SimonR                         13     
ColinT                         14     
GlennM                         15     
repl_publisher                 16382  
repl_subscriber                16383

You can see the special logins for the replication processes and for the probe account, which is used for a two-phase commit between servers.
