Building Solutions Using Data Access Technologies Overview

Four real-world applications make up the set of solution samples. Each of these applications provides a solution for a common, real-world scenario. In addition, each application demonstrates specific MDAC features.

The four applications are:

  • Online Catalog and Order Placement
    Enables consumers to browse catalog products and place orders over the Internet.

  • MIS Inventory Management and Executive Reports (next release)
    Provides MIS employees a way to manage a list of product items and status codes within the master database and generates statistical reports.

  • Warehouse Order Fulfillment (next release)
    Provides warehouse employees a way to record the shipping of orders and prints reports.

  • Contact Management and Sales Forecasting (next release)
    Enables traveling salespeople to manage contacts with clients and record sales forecasts offline.

The applications all use a common database and related tables, so the results carried out in one application can appear in another application.
