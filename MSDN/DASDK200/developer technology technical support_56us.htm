If your question isn't urgent

Send e-mail to a Microsoft Technical Support engineer.

With Web Response, you can conveniently submit questions via e-mail. This option is ideal for detailed questions or questions with attached files. You can submit questions 24 hours a day and will receive a response from an engineer within one business day. To submit your question, go to http://support.microsoft.com/support/ and select Web Response.

Standard No-Charge Web Response

Standard No-Charge Support incidents may be submitted for all eligible retail Microsoft products. If you are using Microsoft Data Access Components in conjunction with a retail Microsoft product, see that product's individual help file for the amount of Standard No-Charge Support available.
