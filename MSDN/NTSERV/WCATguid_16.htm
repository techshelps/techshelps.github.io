Server Requirements

A WCAT server must consist of the following hardware and software:

· A computer running Windows NT Server 4.0.

· At least 220 MB of unused space on the computer’s hard disks. This disk space is required for the prepared WCAT server files. If you plan to use larger files in your WCAT test, allow more unused disk space.

· Transmission Control Protocol/Internet Protocol (TCP/IP) installed and bound to the adapter connected to the network on which the WCAT test is run. For more information on installing and binding TCP/IP, see the topic “TCP/IP Procedures” in Control Panel Help.
