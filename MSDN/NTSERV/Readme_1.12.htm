1.12 Internet Explorer 3.02

 

To set up Internet Explorer 3.02 on x86-based computers:

1. Click Start, and then click Run.

2. In the Run dialog box, type cd_driveletter: \apps\ie30x\setup.exe or network address\apps\ie30x\setup.exe and choose the OK button. (The subdirectory is still called Ie301. Shouldn’t that change to Ie302? And is the Setup.exe file from the correct date?)

3. Follow the instructions on screen to complete Setup.

 

For information about using Internet Explorer, see the Internet Explorer online Help file.
