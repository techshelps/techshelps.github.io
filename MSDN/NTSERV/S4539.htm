Troubleshooting HTTP with Netmonitor

By using the Microsoft netmonitor utility, you can reveal the contents of a specific frame. By examining the HTTP messages, you can determine whether the correct messages were sent.

Netmonitor captures the full frame before it reaches the network components and the browser client. Thus, you can isolate the symptoms of events that occur before the frames reach the host computer.

For more information about netmonitor, see the Windows NT Server Concepts and Planning Guide.
