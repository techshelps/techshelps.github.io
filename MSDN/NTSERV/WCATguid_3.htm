Introduction

This guide provides the information you need to install, configure, and run the Microsoft® Web Capacity Analysis Tool (WCAT). The Microsoft Web Capacity Analysis Tool is included on the Microsoft Windows NT® Server Resource Kit Version 4 Supplement 1 CD-ROM.

The Microsoft Web Capacity Analysis Tool User Guide includes the following sections:

  • “WCAT: A Tool to Test IIS Server and Client Capacity”

  • “Components of a WCAT Test”

  • “WCAT Prepared Tests”

  • “Before Installing WCAT”

  • “Installing WCAT”

  • “Verifying the WCAT Installation”

  • “Running WCAT”

  • “Using Performance Monitor Counters in a WCAT Test”

  • “Analyzing the WCAT Log”

  • “Designing Custom Tests”

The Microsoft Web Capacity Analysis Tool (WCAT) and the Microsoft Web Capacity Analysis Tool User Guide are provided “as is” without warranty of any kind. For details, see “Resource Kit Support Policy” in the introduction to the Supplement 1 book.
