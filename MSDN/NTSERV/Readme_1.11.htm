1.11 Crystal Reports Event Log Viewer

 

To set up Crystal Reports for the Windows NT Server Resource Kit at a later date:

1. Click Start, and then click Run.

2. In the Run dialog box, type cd_driveletter: \apps\crystal\crw45ntr.exe or network address\apps\crystal\crw45ntr.exe and choose the OK button.

3. Follow the instructions on screen to complete Setup.

 

For information about using Crystal Reports, see the README.HLP file in the \APPS\CRYSTAL sub-directory on the Resource Kit CD or the RKTOOLS.HLP online Help file.
