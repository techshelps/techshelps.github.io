Lose Internet Connectivity Or Cannot Use Browser

When you connect to a tunnel server, you might lose connectivity to the Internet. This is because the PPTP tunnel server becomes your default gateway, and your computer no longer uses the default gateway to the Internet. For more information about the default gateway problem and its solution, see the section “Cannot see servers on your network.”

You can solve this problem in one of two ways:

· You can disable the default gateway to the PPTP tunnel server

To Disable The Default Gateway To The PPTP Tunnel Server

1. In the Dial-Up Networking folder, right-click the PPTP server connection icon.

2. Click the Server Types tab.

3. Click TCP/IP Settings.

4. Clear the Use Default Gateway on Remote Networks check box.

5. You may need to configure routes to servers on the remote network.

· You can configure your Web browser’s proxy server settings

Many networks allow access to the Internet through a proxy server. To access Web pages on the Internet, you must configure your Web browser’s proxy server settings as if you were physically connected to the target network. For example, if you are using Internet Explorer, follow the procedure below.

To Configure Internet Explorer Proxy Server Settings

1. Right click the Internet Explorer icon.

2. Click Properties.

3. Click the Connection tab.

4. In the Proxy server box, select Connect through proxy server.

5. In the Proxy server box, click Settings.

6. In the Servers box, add the proxy servers you use on your corporate network. If you do not know these settings, contact your network administrator.
