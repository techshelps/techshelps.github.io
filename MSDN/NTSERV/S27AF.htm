Monitoring the Inetinfo Working Set

You can monitor the size and effectiveness of a working set of the Inetinfo process by using several tools that monitor processes, such as:

Process Viewer (Pviewer.exe)

Process Explode (Pview.exe)

Process Monitor (Pmon.exe)

Performance Data Log Service (PerfLog) (Pdlcnfig.exe and Pdlsvc.exe)

Performance Monitor (Perfmon.exe)

These tools are described in detail in the Windows NT Workstation Resource Guide, Chapter 10, "About Performance Monitor," and Chapter 11, "Performance Monitoring Tools."
