The Mouse Doesn't Work Correctly

If the mouse worked correctly before you set up Network Client, but doesn't work afterward, the mouse might be using the same interrupt setting as the network adapter. Try assigning a different interrupt to the network adapter.

For information about determining the correct interrupt setting for the network adapter, see the documentation that came with the network adapter. For information about how to change the interrupt setting, see "Checking and Changing Network-Adapter Settings," earlier in this section.
