Using PPTP to Connect to a PPTP Server by Dialing an ISP

With PPTP, you can connect your workstation to a remote network by tunneling through the Internet to a PPTP server on that network. To do so, you must make two connections. First, you must connect to the Internet through an ISP. Next, you must create a tunnel to the target network. This section explains how to make these connections using Windows NT and Windows 95 -based clients.
