Conclusion

The Web Capacity Analysis Tool is a tool for evaluating Web servers. It features an easy-to-use setup involving one controller, and a client application for use in a quiet network solely dedicated for performance evaluation. It also provides an extendible way of modifying the test parameters and pages transferred. Appendix: Script File Format This appendix describes the format of the Script file for a Web Capacity Analysis Tool test. The Script file is submitted to the Web Capacity Analysis Controller for execution. The Script file is identified by the filename extension .scr.
