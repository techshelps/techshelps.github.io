Synchronizing User Accounts at Terra Flora

The primary benefit that Terra Flora receives by using DSMN is that user accounts are synchronized between Windows NT and NetWare servers. (Please refer to the network diagram located inside the back cover of the Windows NT Server Resource Kit Supplement 1. There are two NetWare servers represented on the Terra Flora network diagram in the California domain, named CANW410DIV01 and CANW312DPT01.)
