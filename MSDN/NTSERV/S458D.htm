T1 or T3 connection

Standard measurements of network bandwidth.

tag file

A configuration file that contains information about a corresponding file on a Gopher server or links to other servers. This information is sent to clients in response to a Gopher request.

TCP/IP

See Transmission Control Protocol/Internet Protocol.

Telnet (VTP)

A terminal emulation protocol for logging on to remote computers. Once referred to as Virtual Terminal Protocol (VTP). Defined in RFC 854, among others. See also Telnet Server.

Telnet Server

A server that runs a Telnet service, enabling remote users to log on and run programs. See also Telnet.

TFTP

See Trivial File Transfer Protocol.

token ring network

A local area network formed in a ring (closed loop) topology that uses token passing as a means of regulating traffic on the line.

transforms

Rules the administrator creates to add, remove, and modify domain names appended to inbound and outbound messages.

Transmission Control Protocol/Internet Protocol (TCP/IP)

A set of networking protocols that provide communications across interconnected networks made up of computers with diverse hardware architectures and various operating systems. TCP/IP includes standards for how computers communicate and conventions for connecting networks and routing traffic.

Trivial File Transfer Protocol (TFTP)

A file transfer protocol that transfers files to and from a remote computer running the TFTP service. TFTP was designed with less functions than FTP. Defined in RFC 1350, among others. See also File Transfer Protocol.
