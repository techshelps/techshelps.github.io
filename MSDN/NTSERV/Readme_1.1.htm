1.1 Hardware-Platform and Other Issues

The following is a list of issues relating to various Resource Kit utilities that are included in this Supplement 2 release of the Windows NT Server Resource Kit. Most of them refer to utilities that do not work on one hardware platform or another. Hopefully, most of these issues will be resolved by the next release.

 

dbWeb - Works only on x86-based computers.

DiskSave - Works only on x86-based computers.

NLMonitor - Works only on x86-based computers.

NLTest - Works only on x86-based computers.

NTDETECT.COM (INSTALLD.CMD) - Works only on x86-based computers.

UPTOMP.EXE - Works only on x86-based computers.

RUNEXT - Works only on x86-based computers.

TweakUI - Works only on x86-based computers.

WCAT: Web Capacity Analysis Tool - Works only on x86-based computers.
