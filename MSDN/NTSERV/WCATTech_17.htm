Testing the Web Capacity Analysis Tool

Network monitoring and performance testing have confirmed that the Web Capacity Analysis Tool is transmitting, receiving, and monitoring data as designed. We tested WCAT on many tasks, such as processing requests from ISAPI extension and CGI applications, using Secure Sockets Layers protocol and maintaining persistent connections. However, the Web Capacity Analysis Tool is offered "as is" in accordance with the disclaimer of warranty in the Windows NT Server Resource Guide Supplement 1.
