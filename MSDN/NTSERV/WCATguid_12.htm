Table 2 CGI and ISAPI Tests

Name of test Description
CGI25 Tests the response of the server to a CGI application. In this test, 25 percent of client requests are for files that require processing by a CGI application. The remaining 75 percent of client requests are taken from the 1ktest01 test, described in Table 1.
CGI50 Tests the response of the server to a CGI application. In this test, 50 percent of client requests are for files that require processing by a CGI application. The remaining 50 percent of client requests are taken from the 1ktest01 test.
CGI75 Tests the response of the server to a CGI application. In this test, 75 percent of client requests are for files that require processing by a CGI application. The remaining 25 percent of client requests are taken from the 1ktest01 test.
ISAPI25 Tests the response of the server to an ISAPI extension. In this test, 25 percent of client requests are for files that require processing by an ISAPI extension. The remaining 75 percent of client requests are taken from the 1ktest01 test.
ISAPI50 Tests the response of the server to an ISAPI extension. In this test, 50 percent of client requests are for files that require processing by an ISAPI extension. The remaining 50 percent of client requests are taken from the 1ktest01 test.
ISAPI75 Tests the response of the server to an ISAPI extension. In this test, 75 percent of client requests are for files that require processing by an ISAPI extension. The remaining 25 percent of client requests are taken from the 1ktest01 test.
