Cannot Resolve Names Using DNS

The PPTP Dial-Up Networking upgrade also includes a WinSock upgrade. If you did not upgrade WinSock before upgrading Dial-Up Networking, you might have problems reaching DNS servers. You will need to perform the WinSock upgrade.

To Perform A Winsock Upgrade

1. In Windows Explorer, go to the Windows\Msdun directory.

2. Double-click Wsockupd.exe to start the upgrade program.

Note After you have upgraded WinSock, you may need to reinstall any applications that made modifications to your older version of WinSock.
