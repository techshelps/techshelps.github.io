5.0 Directory Structure of the Windows NT Server Resource Kit Supplement 2 CD

The following is the directory structure of the Windows NT Server Resource Kit Supplement 2 CD. The subdirectory names on the CD correspond to the categories from which you can choose during the Setup process. When installed, the files for the utilities that you selected during Setup will be located on your hard disk under the \NTRESKIT directory (unless you assigned a different directory name) or one of its subdirectories.
