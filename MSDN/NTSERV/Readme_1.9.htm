1.9 Microsoft dbWeb

 

To set up Microsoft dbWeb:

1. Click Start, and then click Run.

2. In the Run dialog box, type cd_driveletter: \apps\dbweb\setup.exe or network address\apps\dbweb\setup.exe and choose the OK button.

3. Follow the instructions on screen to complete Setup.

 

For information about using Microsoft dbWeb, see the RKTOOLS.HLP online Help file.
