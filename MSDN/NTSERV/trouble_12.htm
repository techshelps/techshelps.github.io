Troubleshooting PPTP on a Windows 95-based PPTP Client

This section describes a few monitoring tools to help you diagnose connection problems that can occur with Windows 95-based PPTP clients. It also outlines some general troubleshooting steps you can take and describes some common connection problems.

Caution If you have serious problems with your PPTP client, you might want to remove PPTP and reinstall it. If so, follow the steps outlined in the procedure “To remove Dial-Up Networking” in the section “PPTP doesn’t work after ISDN installation.” Never remove PPTP components by using Network in Control Panel.
