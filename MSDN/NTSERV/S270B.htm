Limiting Bandwidth

If the bandwidth on your server is not sufficient to handle the load imposed by Internet Information Server, you can limit the amount of bandwidth Internet Information Server uses. Limiting bandwidth guarantees that some proportion of the server's bandwidth remains available to other server services. This Internet Information Server feature is intended for servers that handle electronic mail or are domain or file servers, in addition to running Internet Information Server.

The following sections explain how to limit the bandwidth used by IIS services, how to monitor the effect of the limitation, and how to interpret the monitoring data you collect.
