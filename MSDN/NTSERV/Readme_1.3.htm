1.3 Index Server

 

To set up Index Server:

1. Click Start, and then click Run.

2. In the Run dialog box, type cd_driveletter: \apps\index\hardware platform\language filename

or network address\apps\index\hardware platform\language filename and choose the OK button.

For example, type D: \apps\index\i386\idxsveng.exe to obtain the International English version for use on an x86-based computer.

For information on the Index Server installation variables, see the INDEX.TXT file in the

\APPS\INDEX subdirectory.

3. Follow the instructions on screen to complete Setup.
