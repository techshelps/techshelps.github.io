Monitoring ISAPI Extensions

ISAPI scripts are compiled into DLLs that run in the Internet Information Server process, Inetinfo.exe. You can monitor the processor and memory use of the Inetinfo process, but it is difficult to distinguish the resource use attributable to ISAPI from that of the rest of the services running under Inetinfo. It is also extremely difficult, though technically possible, to associate individual threads in Inetinfo with ISAPI requests.
