Installing PPTP

For PPTP to succeed, a PPTP driver must be installed on the RAS servers—and on the client or communications server—for PPTP tunneling to succeed.

To Install PPTP

  1. Click Start, point to Settings, and click Control Panel.
  2. Double-click Network.
  3. Click the Protocols Tab.
  4. Click Microsoft PPTP Driver, and then click Add.
  5. Click Have Disk if loading the software from disks or a CD.

Or, click OK if loading from the network.

  1. When prompted, type the full path to the Windows NT Server PPTP files, and then click OK.
  2. Enter the number of connections you want available to PPTP (that is, for Virtual Private Networks).

  1. Click the port on which you want to install the PPTP driver, and click OK.

You must restart your computer to complete the DNS installation.
