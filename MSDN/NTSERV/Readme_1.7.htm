1.7 Microsoft TechNet

 

To set up Microsoft TechNet at a later date:

1. Click Start, and then click Run.

2. In the Run dialog box, type cd_driveletter: \apps\technet\setup.exe or network address\apps\technet\setup.exe and choose the OK button.

3. Follow the instructions on screen to complete Setup.

 

For information about using Microsoft TechNet, see the TechNet online Help file.
