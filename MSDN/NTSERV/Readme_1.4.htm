1.4 Desktop Themes for Windows NT 4.0

 

To set up Desktop Themes for Windows NT 4.0 at a later date:

1. Click Start, and then click Run.

2. In the Run dialog box, type cd_driveletter: \apps\themes\setup.exe or network address\apps\themes\setup.exe and choose the OK button.

3. Follow the instructions on screen to complete Setup.

 

For information about using these desktop themes, see the RKTOOLS.HLP online Help file.
