1.2 General Setup Instructions

 

To set up the standard Resource Kit utilities:

1. Start Windows NT Server and insert the Windows NT Server Resource Kit Supplement 2 CD in the CD-ROM drive.

The Windows NT Server Resource Kit Setup master setup screen appears. Notice that options that are only available on certain hardware platforms are not highlighted.

2. Click the Install Resource Kit option.

3. Follow the instructions on screen to complete Setup of the standard utilities.

 

To set up additional items at a later time from the CD or a network sharepoint:

1. Click Start, and then click Run.

2. In the Run dialog box, type cd_driveletter: msetup or network address\msetup.exe and choose the OK button.

The Windows NT Server Resource Kit Setup master setup screen appears. Notice that options that are only available on certain hardware platforms are not highlighted.

3. Click the option that you want to install.

4. Follow the instructions on screen to complete Setup of your selected item.

 

The Setup program provides two options for setting up the Windows NT Server Resource Kit on your hard disk: Typical Setup and Custom/Complete Setup. Typical Setup installs all the Resource Kit files onto your hard disk and requires a maximum of 52 MB of free space. Custom/Complete Setup, however, enables you to choose specific items and minimize the amount of space used on your hard disk.

 

Setup creates a Resource Kit 4.0 folder on the Start menu with shortcuts for the installed utilities. For more information about these utilities, choose the Resource Kit Tools Overview (RKTOOLS.HLP) shortcut.

 

Setup also adds the \NTRESKIT directory (or the directory name you choose for installing the utilities) to the PATH System environment variable. (If you want to change your PATH, you can do so by using the System option in Control Panel after the Resource Kit installation is complete.)

 

Some Resource Kit utilities require separate or additional setup besides the steps described above. The following sections describe some special setup requirements and how to set up items at a later date. For more information about each of these utilities and others with additional requirements, see the Overview of Windows NT Resource Kit Tools (RKTOOLS.HLP) online Help file.
