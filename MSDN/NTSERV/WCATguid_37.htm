Creating a Performance Counter Input File

The performance counter input file is a formatted file that lists the Performance Monitor counters you want to monitor during a WCAT test. You can use a performance counter input file in any WCAT test, and you can use the same performance counter input file for many different WCAT tests.

The performance counter input file can have any name, but the name must be in the format testname.pfc. By installation default, the file is located in the \Scripts directory on the WCAT controller, along with the configuration file, script file, and distribution file.
