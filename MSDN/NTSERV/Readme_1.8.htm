1.8 Client-based Administration Tools for Windows NT Server

 

To set up the Client-based Administration Tools for Windows NT Server at a later date:

1. Click Start, and then click Run.

2. In the Run dialog box, type cd_driveletter: \apps\clients\setup.exe or network address\apps\clients\setup.exe and choose the OK button.

3. Follow the instructions on screen to complete Setup.

 

For information about using these tools, see each tool's individual Help file.
