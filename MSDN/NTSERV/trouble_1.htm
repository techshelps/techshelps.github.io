Troubleshooting PPTP on a Windows NT-based PPTP Client

This section describes a few monitoring tools to help you diagnose connection problems that can occur with Windows NT-based PPTP clients. It also outlines some general troubleshooting steps you can take and describes some common connection problems.

This section does not describe installation and connection problems that can occur with Windows NT-based PPTP servers. For information about such problems, refer to the resources listed in Table 1.
