1.13 Service Pack 3 for Windows NT 4.0

 

To set up Service Pack 3:

1. Click Start, and then click Run.

2. In the Run dialog box, type cd_driveletter: \apps\sp3\hardware platform filename or network address\apps\sp3\hardware platform filename and choose the OK button.

For example, type D: \apps\sp3\nt4sp3_i.exe to obtain the Service Pack for use on an x86-based computer. Notice that Service Pack 3 no longer supports the MIPS (and PPC?) hardware platform .

3. Follow the instructions on screen to complete Setup.
