Verifying Your Configuration

If you are having problems connecting to a PPTP server, make sure you have correctly configured your PPTP client. This section provides a list of steps you should take to make sure you have correctly configured your PPTP client. If you still cannot connect to a PPTP server, see the section “Troubleshooting Your PPTP Connection.”

Verify the following:

· Make sure that your workstation and its applications support the protocols native to the target network (the network you will be tunneling to).

· If you are tunneling through the Internet, make sure your Internet connection is working properly.

· In Network in Control Panel, make sure that you have added the PPTP protocol.

· In Network in Control Panel, make sure that you have configured at least one VPN port.

· Check the phone book entry for the PPTP server to make sure it is for a VPN device, not a modem.

· Make sure you have a valid account on the PPTP server.

· Check the phone book entry for the PPTP server to make sure it includes the IP address or host name of the PPTP server.

· Check the phone book entry for the PPTP server to make sure it includes all the protocols used on the target network.

· If the target network uses TCP/IP, check the phone book entry for the PPTP server to make sure the TCP/IP settings are correctly configured for the target network.

· If you are making two connections, make sure that the second connection is a VPN connection.
