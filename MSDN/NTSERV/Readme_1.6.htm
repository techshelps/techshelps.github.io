1.6 Web Capacity Analysis Tool

 

To set up the Web Capacity Analysis Tool (WCAT) at a later date:

1. Click Start, and then click Run.

2. In the Run dialog box, type cd_driveletter: \apps\wcat\i386\wcsetup.exe or network address\apps\wcat\i386\wcsetup.exe and choose the OK button.

3. Follow the instructions on screen to complete Setup.

 

For information about using WCAT, see the RKTOOLS.HLP online Help file.
