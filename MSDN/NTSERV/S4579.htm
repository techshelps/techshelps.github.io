How To Obtain the BackOffice and NT Server 4.0 Resource Kit Utilities CDs

While TechNet includes the full document set for the BackOffice and NT Server 4.0 Resource Kit, the utilities CDs are available separately from your local ordering center or subsidiary. This is due to the size of these utilities. We apologize in advance for any inconvenience this may bring and hope to have these utilities included in our regular editions in the future. In the meantime, TechNet subscribers in North America may order the BackOffice and NT Server Resource Kit Utilities CDs by calling 1-800-344-2121 and asking for the part number(s) shown below. There is a $10.00 (US) shipping and handling charge for each order.

Solution Providers: If you receive TechNet as a benefit of the Microsoft Solution Provider program, please calling 1-800- SOLPROV in the United States or your local subsidiary to order these parts.
