Log File

The log file (extension: .log) contain results of the test in a standard format that spreadsheets applications can read them easily. The information in log files consists of the configuration data (given as input), summary statistics from each client, and a summary of all client statistics. It also includes the average values of any performance counters requested in the .pfc file. The Web Capacity Analysis tool creates a log file for each test. The log file can be processed off-line. This makes it possible to run the controller in a completely automated manner, and process logged data at a later date.
