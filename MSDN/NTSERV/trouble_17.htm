Route

Your computer maintains a routing table, which tells it where to send outgoing packets in order to reach certain subnets. In some cases, you need to view or modify your routing table.

You can use the route utility to add, change, or delete routes. Route has the following options:

· route displays help about the utility.

· route print displays a list of all routes your workstation recognizes.

· route add enables you to add routes.

· route delete enables you to delete routes.

· route change enables you to modify routes.

For more information about the route utility, see Chapter 12, “Troubleshooting Tools and Strategies,” in the Windows NT Server Networking Guide.
