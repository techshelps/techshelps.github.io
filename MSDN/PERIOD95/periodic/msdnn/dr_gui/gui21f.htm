Adding and Removing Property Pages

Dear Dr. GUI:

I am writing an application that uses property sheets. One of the property pages in the sheet has a set of radio buttons that control the total number of property pages displayed. Basically, these radio buttons allow users to dynamically remove and add property pages to the property sheet.

So far, I am able to dynamically remove the property page, but adding it dynamically is a little slow. It waits until I change to another property page before I see the newly added page. I would like for it to happen immediately, like after I click to select the radio button to add. Is there any way I can flush to request to add the page as soon as possible? Any hacks?

Dr. GUI replies:

This is a bug. The tab control is not being invalidated when an item is added. You can easily work around this, though. After calling AddPage, invalidate the tab control: 

m_pSendPS->GetDlgItem(AFX_IDC_TAB_CONTROL)->Invalidate();

This will be fixed for a future release, so you might want to include this line in your code: 

#if (_MFC_VER <= 0x300)... #endif.
