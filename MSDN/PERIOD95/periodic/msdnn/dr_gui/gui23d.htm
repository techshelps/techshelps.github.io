Dr. GUI replies:

The preferred way of handling the Media Player is as an OLE object. There is no DDE (dynamic data exchange) support in Media Player, so the way to add DDE is to put a wrapper in Visual Basic around the Media Player using MCI.VBX. This means looking at the samples for DDE and the samples using the MCI (media control interface) control and combining them. You may wish to create your own particular look to the interface. For example:

· using 3-D buttons with pictures of actions

· creating a distinctive background
