Microsoft Solutions Platform and Kit Available

May 9, 1995

Microsoft has introduced two products designed to make it faster and easier for developers to create custom, integrated client-server solutions. The Microsoft Solutions Platform and Microsoft Solutions Development Kit (MSDK) provide tools and information for solutions using components of Microsoft Office, Microsoft BackOffice, and Microsoft Visual Tool Suite.

The Solutions Platform includes Microsoft Office Professional version 4.3; Microsoft Project version 4.0; the Visual Basic programming system version 3.0; the Visual C++ development system version 2.0; the Visual FoxPro database management system version 3.0; Microsoft SQL Server, Workstation Edition version 4.21, and the MSDK (also available separately).

Details and information on pricing and ordering can be found in the Internet version of this newspaper. Go to http://www.microsoft.com and open Developer Network OffRamp/Developer Network News.
