ATL for Windows CE Reference

The ATL for Windows CE Reference contains documentation for all the Active Template Library classes supported in Windows CE 2.0.

Note Windows CE versions prior to 2.0 did not support the Active Template Library.

Each reference topic includes an At a Glance table containing the following information.

At a Glance

Header file: Header file in which the class is defined.
Platforms: Targeted platforms.
Windows CE versions: Windows CE versions that include this API element.
Complete documentation: Link to the complete documentation as originally published. Usually, this is a link to the Microsoft Platform SDK documentation.

When there is a link to original documentation, comments in the Remarks section are specific to Windows CE.

Windows CE supports almost all of the standard Active Template Library classes. Because Windows CE runs on platforms that are very different from the desktop Windows platforms, it has some different requirements. In order to meet these requirements, the designers of Windows CE have made changes to a few of the classes in the library.

For more information, see Using ATL Global Functions and Macros in Windows CE, Using ATL Classes in Windows CE, and What's Different in ATL for Windows CE.
