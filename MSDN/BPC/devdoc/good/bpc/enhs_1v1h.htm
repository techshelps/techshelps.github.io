Graphical Time Line

[This is preliminary documentation and subject to change.]

Presenting the user with a graphical representation of the enhancement stream time line enables the user to better visualize where in the stream an event occurs. This can be accomplished, for example, by creating a Microsoft® ActiveX® control that reads an enhancement stream and displays the events in a horizontal time line. The following illustration presents an example of a control that uses arrows to indicate events.
