EnhEvents.ShowLength

[This is preliminary documentation and subject to change.]

The ShowLength property specifies the length of the interactive episode, in seconds.

Syntax

object.ShowLength [ = dLength ]

Parts

object
Object expression that resolves to an EnhEvents object.
dLength
Double that indicates the show length, in seconds. The default value is 0.
