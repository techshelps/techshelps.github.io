Indication of Bandwidth Utilization

[This is preliminary documentation and subject to change.]

Allocating the limited broadcast bandwidth is a part of any stream-authoring tool. You can extend this functionality by providing the user with a graphical or numerical display of the running total of allocated bandwidth.

A graphical display, such as a bar or pie graph, helps the user visualize how much bandwidth is left and can influence their enhancement design decisions. For example, if a repetitions of a particular file download are taking up half the available bandwidth, the user can choose to reduce the file size by making enhancement pages more simple or to reduce number of times that the file is transmitted.
