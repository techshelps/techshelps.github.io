ipenhsnd Type Library

[This is preliminary documentation and subject to change.]

The ipenhsnd type library, Ipenhsnd.dll, provides objects your application can use to transmit enhancement stream events and files to broadcast clients. Included are cabinet utilities that enable your application to package and compress enhancement files for more efficient transmission.

The ipenhsnd type library contains the following objects:

  • ipsend, transmits enhancement stream events to broadcast clients.
  • EnhCab, creates and enumerates cabinet (.Cab) files.
