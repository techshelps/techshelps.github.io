About Show Reminders

[This is preliminary documentation and subject to change.]

Your application can schedule show reminders as tasks in the Task Scheduler. These tasks specify the application to run, typically Microsoft® WebTV® for Windows®, and command-line arguments for that application.

When the Task Scheduler runs a show reminder task, it sends these command-line parameters to the specified show reminder application. If the application is not currently running, the Task Scheduler starts it.

For more information, see Show Reminder Applications and Show Reminder Format.
