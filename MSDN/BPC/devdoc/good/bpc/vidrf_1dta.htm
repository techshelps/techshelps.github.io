Adding the Video Control to an Application

[This is preliminary documentation and subject to change.]

The Video control is an ActiveX control that you can use to display video in an application. Because the Video control uses ActiveX, it can be called from any COM-supporting programming environment.

To use the Video control in an application, you must first add it to your project. How you do this differs depending on which programming environment you use. For detailed information about how to add an ActiveX control to your project, see the documentation for your programming environment.

The following topics describe how to add the Video control to some common programming environments:

  • Adding the Video Control to a Web Page
  • Adding the Video Control to a Microsoft Visual Basic Project
  • Adding the Video Control to a Microsoft Visual C++ Project
  • Adding the Video Control to a Microsoft Visual J++ Project
