Stream Compiler Object Library

[This is preliminary documentation and subject to change.]

The Stream Compiler Object Library, Stream.dll, contains a set of COM objects that provide functionality to create, edit, and save enhancement streams. You can use these objects to create enhancement stream authoring tools such as an enhancement stream editor or play-back application.

The stream compiler exposes the following objects:

Object Description
EnhEvent An enhancement event object.
EnhEvents A collection of enhancement event objects.
Annc An announcement object.
