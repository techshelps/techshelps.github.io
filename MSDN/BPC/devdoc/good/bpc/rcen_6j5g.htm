Enhancements on the Client

[This is preliminary documentation and subject to change.]

The enhancement client software is part of the Microsoft® Windows® 98 operating system and is installed when the user selects the Broadcast Data Services component. This software receives and handles enhancement streams. When this software is installed and the user tunes to an interactive show, the broadcast client displays the appropriate enhancement pages and responds to enhancement events.

The following topics provide an overview of the enhancement client architecture and its components:

  • Enhancement Client Architecture illustrates the reception and flow of enhancement data on the broadcast client.
  • Enhancement Control describes the object used to handle enhancements on the client.
  • Location of Enhancement Files on the Client discusses where to place and find enhancement files on the client.
  • Enhancement Filter covers the announcement filter that processes enhancement announcements.

For more information about the head-end components used to create and broadcast the enhancement stream, see Enhancements at the Head End.
