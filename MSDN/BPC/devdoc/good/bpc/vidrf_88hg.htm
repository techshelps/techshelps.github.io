Video Access Server

[This is preliminary documentation and subject to change.]

The video access server, Vidsvr.exe, is a component of Broadcast Architecture that manages video devices by using the Microsoft® DirectShow™ filter graph. Much of the functionality exposed by the Video control, such controlling devices, is actually implemented in the video access server.

The object definition language file, Vidsvr.odl, defines interfaces exposed by both the video access server and the Video control. To utilize the functionality of the video access server, call the corresponding methods, properties, and events of the Video control.
