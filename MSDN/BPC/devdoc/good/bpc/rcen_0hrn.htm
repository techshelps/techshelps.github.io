Using Enhancements

[This is preliminary documentation and subject to change.]

By adding enhancements to video, you can make a presentation richer in content and more interactive. The following sections explain the technical processes involved in creating, transmitting, and receiving an interactive show:

  • Creating an Enhancement Page provides information on creating an enhancement page.
  • Creating an Enhancement Stream describes how to create a script that synchronizes the transmission and display of your enhancements with the interactive show.
  • Using the Stream Compiler Objects illustrates how to use the stream compiler objects to manipulate enhancement streams. This section covers such topics as adding events to an enhancement stream, merging enhancement streams, and so on.
  • Broadcasting an Enhancement describes how to use the ipsend object to broadcast enhancements to clients.
  • Creating a Cabinet File explains how to use the EnhCab object to create and populate cabinet files.
  • Monitoring the Status of an Enhancement Control describes how to monitor the status of an enhancement control running in the same process as an ipsend object.

For more information about creating custom tools to generate and edit the enhancement stream, see Building Enhancement Authoring Tools.
