Using Guide Data

[This is preliminary documentation and subject to change.]

To maintain current information, new data must be uploaded to the Guide database. This uploading is performed by Guide database loaders. For more information, see the following topics:

  • Moving Data to the Program Guide. This topic discusses dynamic-link libraries (DLLs) used by the loader application to map various data formats to the model used by the broadcast client.
  • Updating the Guide Database. This topic describes the general process by which a loader DLL updates Program Guide data in the Broadcast Architecture database.
