Setting a Reminder That Appears in WebTV for Windows

[This is preliminary documentation and subject to change.]

Microsoft WebTV for Windows provides a user interface that displays a information about currently set reminders. These reminders appear when the user opens the search page in Microsoft® WebTV® for Windows® and clicks My Reminders in the list of categories. In addition, a reminder icon appears in the Program Summary panel when the user clicks an episode in the Program Guide for which a reminder is set.

Users can use the MyReminders list to view, edit, or delete show reminders. The reccommended way to set reminders that appear in the WebTV for Windows user interface is to use ITVViewer::SetReminder.

However, you can use the IScheduledItems::Add method of TSS to set reminders that appear in WebTV for Windows if the reminders meet the following standards:

  • Ensure that the show reference contains the start date, channel number, tuning space, station call letters and title.
  • Set the application flag to tvviewer. For more information, see Using IScheduledItems to Schedule a Show Reminder.
  • Only set one-time, weekly, or daily reminders.
  • If it is a record reminder, follow the procedures described in Setting a Record Reminder.
  • Limit the number of reminders you add. The reminder list displayed by WebTV for Windows has a limit of 50 reminders.
  • Do not add duplicate reminders that specify the same show reference.
  • Set the reminder to run no more than 30 minutes before the show starts.
