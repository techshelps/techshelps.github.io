WebTV for Windows

[This is preliminary documentation and subject to change.]

Microsoft® WebTV® for Windows® is the user interface for Broadcast Architecture. It hosts several controls such as Microsoft® Internet Explorer and the Microsoft® ActiveX® control for video (the Video control). WebTV for Windows uses these controls to present data such as live video, Program Guide information, and enhancements. You can expand WebTV for Windows functionality by writing custom controls that interact with WebTV for Windows.

For more information, see the following topics:

  • About WebTV for Windows, which describes WebTV for Windows and the interfaces it exposes.
  • Using WebTV for Windows, which explains how to use the interfaces exposed by WebTV for Windows.
  • WebTV for Windows Reference, which provides detailed reference information.
