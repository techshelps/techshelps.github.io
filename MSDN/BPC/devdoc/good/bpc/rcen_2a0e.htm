Annc.SessionVersion

[This is preliminary documentation and subject to change.]

The SessionVersion property contains the session version number.

Syntax

object.SessionVersion [ = lversion ]

Parts

object
Object expression that resolves to an Annc object.
lversion
Long that contains the version number. This can be any non-zero integer. The default value is 1.

Remarks

The version identifier is used by the SAP/SDP protocol for tracking purposes.

You do not have to set the value of this property in order for the announcement file generated by Annc.Store to be valid. The default value is sufficient.
