Using the Microsoft Multicast Router

[This is preliminary documentation and subject to change.]

Applications can configure the MMR using a COM based API implemented in Msbdnapi.dll. The include file Msbdnapi.h includes all the definitions needed to access the interfaces in this COM object. For details on how applications use Msbdn.dll to configure the MMR, see Configuring the Microsoft Multicast Router.

Once an application has configured the MMR, it can send data to it in TCP/IP packets. The MMR uses the configuration information to determine how to route the TCP/IP packets. For information on sending data to the MMR, see Sending Data to the Microsoft Multicast Router.
