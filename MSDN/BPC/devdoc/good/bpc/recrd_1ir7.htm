Specifying Key Codes

[This is preliminary documentation and subject to change.]

To enable your application to send command sequences that perform specific VCR operations, you must define a list of key codes in your application that the remote control recognizes. Key codes are one-byte values that represent either operations that a VCR performs or numbers from zero to nine. The necessary key codes for operations you must define are the ones that cause the VCR to start and stop recording. Your application uses the key codes that represent numbers from zero to nine to tune a VCR to a specific channel. Some remote controls might require command sequences that put the remote controls in either automatic or manual mode.
