Running WebTV for Windows from the Command Line

[This is preliminary documentation and subject to change.]

Typically you do not run WebTV for Windows from the command line. Instead you use the user interface built into WebTV for Windows and Windows 98. However, WebTV for Windows does support command-line arguments. You can use this syntax to run WebTV for Windows from the command-line or to create applications that control WebTV for Windows by sending command lines to the shell.
