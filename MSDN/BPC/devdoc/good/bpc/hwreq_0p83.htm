Input Devices

[This is preliminary documentation and subject to change.]

For Broadcast Architecture, user input devices such as keyboards and mice change to accommodate their use in a living room setting. For example, keyboards can be used from the lap or a coffee table. Pointing devices can be hand-held, because a flat surface might not be available. Users can expect to use a remote control for channel surfing and other control of entertainment systems.
