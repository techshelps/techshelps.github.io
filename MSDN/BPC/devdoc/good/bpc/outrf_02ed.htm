Virtual Interface Reference

[This is preliminary documentation and subject to change.]

This section documents the functions that make up the interface between the Microsoft® Multicast Router (MMR) and a virtual interface, which sends data streams to a broadcast output device. Such a virtual interface communicates with the input unit of a broadcast data encoder. This section also documents structures that hardware vendors use to implement the virtual interface.

The virtual interface is documented in four groups:

  • Virtual Interface Functions
  • MMR Functions
  • Virtual Interface Structures
  • Sample Virtual Interface DLL
