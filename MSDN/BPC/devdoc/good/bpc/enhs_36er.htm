Stream Authoring Tool Strategies

[This is preliminary documentation and subject to change.]

The form and function of the enhancement stream authoring tool you create depends largely on your user's needs. The following topics describe some common authoring scenarios and the types of tools you can create to serve them.

  • Generic Stream Authoring Tools, which describes general purpose stream authoring tools.
  • Custom Stream Authoring Tools, which describes stream authoring tools customized for a particular enhancement format.
  • Live Stream Authoring Tools, which describes authoring tools designed to produce enhancements on the fly for live-broadcast shows
