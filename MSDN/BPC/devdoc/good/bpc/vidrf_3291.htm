BPCVid.Close

[This is preliminary documentation and subject to change.]

The Close method sets all inputs and outputs to NULL.

Syntax

object.Close()

Parameters

object
Object expression that resolves to a BPCVid object.

Remarks

This method does not affect device tuning.
