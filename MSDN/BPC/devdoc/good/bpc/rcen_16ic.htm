Annc Object

[This is preliminary documentation and subject to change.]

The Annc object wraps an enhancement announcement. The fields of the announcement are stored as properties in the object. The Annc object also provides file-access methods that enable your application to save an announcement to a file or load an announcement from a file.

Saving an announcement to a file enables you to broadcast the announcement at a later time, as is typically the case. Announcements are saved as text files. By convention, announcement files have the extension .ann.

For more information see the following topics:

  • Creating Enhancement Announcements explains how to create announcement files.
  • Enhancement Announcement Format describes the SAP/SDP fields specific to enhancement announcements. These fields correspond to the properties of the Annc object.
  • Annc contains reference topics which describe the methods and properties of the announcement object.
