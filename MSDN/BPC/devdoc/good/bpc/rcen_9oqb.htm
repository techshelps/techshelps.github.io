Triggers

[This is preliminary documentation and subject to change.]

Triggers are notifications that are sent to broadcast clients at specific times during an interactive show. Triggers instigate or trigger an action on a client. For example, you can use triggers to automatically change the content an enhancement displays, as in an advertisement rotator.

Triggers are passed to the Enhancement control, which handles Broadcast Architecture–defined triggers itself and which forwards user-defined triggers to the scripts that handle them.

For more information about the triggers currently defined by Broadcast Architecture, see Enhancement Triggers.
