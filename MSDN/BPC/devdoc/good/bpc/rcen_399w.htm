Enhancement Sender Object

[This is preliminary documentation and subject to change.]

The enhancement sender object, ipsend, is an object that you can use to transmit the enhancement stream over a broadcast medium such as a Microsoft Multicast Router, other multicast router or LAN. The ipsend object supports three separate Internet Protocol (IP) connections, one each for enhancement announcements, triggers, and FTS transmissions.

For more information on working with ipsend, see Broadcasting an Enhancement.
