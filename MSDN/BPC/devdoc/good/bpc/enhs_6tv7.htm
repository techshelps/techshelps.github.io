Templates

[This is preliminary documentation and subject to change.]

Enhancement stream templates are Enhancement Stream language files that contain standard or blank stream events. For example, if a show is repeatedly broadcast, such as a weekly situation comedy, the enhancements for each episode may contain the same elements, for example, each episode might broadcast the same HTML cast pages.

If the enhancement streams created by your users follow a pattern, using a template enables your tool to more quickly build the stream because it does not have to re-create the basic stream framework. The tool simply modifies events already in the template to fit the user's needs.

You can extend this functionality even further by providing a way for users to create and define their own custom templates.
