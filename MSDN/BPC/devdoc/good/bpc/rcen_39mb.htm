Using the Stream Compiler Objects

[This is preliminary documentation and subject to change.]

The stream compiler objects are a set of Component Object Model (COM) objects that provide functionality to create, edit, and save enhancement streams. You can use these objects to create enhancement stream authoring tools.

The following topics illustrate how to use the stream compiler objects to implement various types of stream-authoring functionality:

  • Adding Events to an Enhancement Stream explains how to use the stream compiler objects to add events to an enhancement stream.
  • Locating an Event in the Enhancement Stream explains how to find a particular event in the enhancement stream.
  • Editing and Deleting Events in an Enhancement Stream explains how to change or remove an event from the enhancement stream.
  • Creating Enhancement Announcements describes how to create the enhancement announcement files that are transmitted during an announcement event.
  • Adding User-Defined Comments to an Event explains how to add subevents to an event using the stream compiler objects.
  • Editing the Dependencies of an Event describes how to modify the dependencies files of an event. You can use this to set custom dependencies for the event, or to modify the dependency list generated by the stream compiler objects.
  • Including One Enhancement Stream in Another describes how to include enhancement streams within other enhancement streams. You can use this technique to combine enhancement streams, or to build a playlist that runs several shows.
  • Merging Enhancement Streams explains how to combine events from two or more enhancement streams into a single stream.
  • Converting Events to Low-level Syntax describes high-level and low-level stream syntax and explains how to convert an enhancement stream or event to low-level format.
  • Loading or Saving an Enhancement Stream explains how to read an enhancement stream from a text file and save the stream to disk.

For more information on building stream-authoring applications, see Building Enhancement Authoring Tools.
