Add Events to the Stream

[This is preliminary documentation and subject to change.]

An enhancement stream comprises the many events of an interactive show. Your authoring tool should provide a way for users to add new events, such as triggers, announcements, and file transmissions, to the enhancement stream.

If you are using the stream compiler objects to build your authoring tool, you can use the EnhEvents.Add and EnhEvents.AddText methods to add an event to an enhancement stream.

For more information on using the stream compiler objects to add events to the stream, see Adding Events to an Enhancement Stream.
