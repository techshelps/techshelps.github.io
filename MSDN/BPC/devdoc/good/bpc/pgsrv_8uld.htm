About Guide Data

[This is preliminary documentation and subject to change.]

Guide data comes from various locations in various formats. Often, the Program Guide combines information from multiple broadcast sources, each of which has its own channel numbering scheme. For more information on working with Guide data from multiple sources, see the following topics:

  • Multiple Data Sources, which describes possible sources of Program Guide data.
  • Overlapping Channels, which explains Broadcast Architecture concepts on integrating Program Guide data from multiple sources into a single data set.
