Sample Virtual Interface DLL

[This is preliminary documentation and subject to change.]

This section describes source code for the sample dynamic-link libraries (DLLs) for virtual interfaces (VIFs) that are provided as part of the Microsoft® Platform Software Development Kit (SDK). These DLLs are provided to give hardware vendors assistance in writing their own virtual interface DLLs. To be of maximum value, this section requires the user to have the Main.cpp source code files close at hand. You can find these files in the Mc_out and Nulloss directories.

Note  Virtual interface DLLs are components of a broadcast server and therefore only run on the Microsoft® Windows NT® Server version 4.0 operating system or later operating systems.
