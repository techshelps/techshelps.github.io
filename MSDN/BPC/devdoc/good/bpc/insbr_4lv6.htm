Working with the Microsoft Multicast Router

[This is preliminary documentation and subject to change.]

The Microsoft® Multicast Router (MMR) is a network server application that provides services to forward network traffic from a local computer network to a variety of unidirectional broadcast encoders. Network administrators can configure the MMR using a Microsoft® Management Console (MMC) snap-in. In addition applications can configure the MMR by using the Msbdnapi.dll Component Object Model (COM) object. The following section describes how to write programs that use the Msbdnapi.dll COM object.
