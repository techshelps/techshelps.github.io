Using Guide Database Loaders

[This is preliminary documentation and subject to change.]

Your application can either use a Guide database loader provided Broadcast Architecture or a custom loader. The loader currently provided with Broadcast Architecture adds television-schedule data for the Gemstar G-Guide system to the Guide database. To work with other types of data, you must have a custom loader for that data.

  • Writing a Custom Loader
