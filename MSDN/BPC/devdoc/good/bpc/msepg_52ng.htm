About the Enhancement Video Control

[This is preliminary documentation and subject to change.]

The Enhancement Video control displays video in HTML enhancement pages hosted by Microsoft® WebTV® for Windows®. This control is implemented as the Msepg4 class in the Msepg.ocx object library. The Enhancement Video control wraps the Video control, extending the Video control's functionality by automatically creating and handling a connection to WebTV for Windows when an instance of the Enhancement Video control is created.

Note  The classes in Msepg.ocx other than Msepg4 are reserved for use by Broadcast Architecture and are not supported for use by external applications.

The Enhancement Video control accepts an HTML parameter, INTENT, which indicates how the Enhancement Video control is used in the current Web page. If you use the Enhancement Video control to display video in an enhancement, INTENT should be set to the value ENHANCE_VIDEO. For more information on INTENT, see Using the Enhancement Video Control.
