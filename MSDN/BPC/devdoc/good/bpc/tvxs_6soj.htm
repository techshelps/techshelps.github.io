Controlling WebTV for Windows

[This is preliminary documentation and subject to change.]

Once your control has obtained a reference to Microsoft® WebTV® for Windows®, as described in Connecting to WebTV for Windows, the control can call the methods WebTV for Windows exposes to programmatically control WebTV for Windows. For example, a channel-surfing control might automatically retune WebTV for Windows to a new channel every three minutes.

WebTV for Windows also provides methods that your control can call to query the state of WebTV for Windows. For example, you can query whether WebTV for Windows is currently displaying closed caption information.

For more information about the methods exposed by WebTV for Windows, see ITVViewer.
