Loadstub

[This is preliminary documentation and subject to change.]

The Loadstub component, Loadstub.exe, starts a loader dynamic-link library (DLL). Typically, Loadstub is automatically started as a task in the Task Scheduler in the Microsoft® Windows® 98 operating system. Using this stub application provides a common interface for loader DLLs that themselves may be quite divergent in approach.

For more information on working with Loadstub, see the following topics:

  • About Loadstub, which describes what Loadstub does.
  • Using Loadstub, which details parameters your loader can set for Loadstub.
