About Television System Services

[This is preliminary documentation and subject to change.]

The TSS object library, Tssadmin.dll, provides a variety of television-related utility methods, as well as a television-related collection. The following topics group these items by related functionality:

  • About the Property Methods describes the methods used to return information about the current TSS installation.
  • About the Enhancement Loader Methods describes the methods that load and manipulate enhancement information in the Guide database.
  • About the Show Reference Methods describes the methods that create and parse show references.
  • About the Scheduled Items Collection describes the collection of show reminders.

For reference information on the interfaces that provide these methods, see the topics on the ITelevisionServices and IScheduledItems interfaces. For more information on the methods that TSS contains but that are reserved for future use, see Reserved Methods in ITelevisionServices.
