Video Enhancements

[This is preliminary documentation and subject to change.]

The term interactive video refers to the combination of a video program and multimedia elements such as hypertext links, graphics, text frames, sounds, and animations. The following sections are directed to those who plan to create or display interactive video using Broadcast Architecture.

These sections provide an overview of the enhancement components and their interactions, instructions on using the enhancement components to create and receive enhancement streams, and reference information for the enhancement components:

  • About Enhancements explains what enhancements are and describes the various components of the enhancement system.
  • Using Enhancements explains how to create and transmit interactive shows.
  • Enhancement Reference contains reference material on the enhancement announcement format, on the triggers that initiate enhancement actions, and on objects used to create enhancement streams.

To locate additional information on enhancing video, especially on designing and creating enhancements, see Further Information on Data Services for the Client.
