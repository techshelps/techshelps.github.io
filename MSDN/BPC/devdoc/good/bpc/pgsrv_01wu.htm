E Abbreviation

[This is preliminary documentation and subject to change.]

The E Abbreviation field contains a bit field that indicates how to abbreviate episode titles for smaller display areas. For example, "Mars: Close Encounters with the Red Planet," appears as "Mars: Close Encounters" when this field has a binary value of 1110000. This field's values are as follows:

Key: Not applicable
Index: No
Data type: Number
Field size: Long integer
Default value: Zero
Required: No
