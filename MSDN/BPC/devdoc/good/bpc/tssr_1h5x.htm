Television System Services Reference

[This is preliminary documentation and subject to change.]

The primary Automation class in Television System Services (TSS), Tssadmin.dll, is ITelevisionServices, derived from the IDispatch interface. For error handling, ITelevisionServices supports the IErrorInfo Automation interface. To locate more information on IDispatch and IErrorInfo, see Further Information on Television Services for the Client.

TSS also provides the IScheduledItems interface, which contains methods that you can use to set, edit, or delete show reminders.
