EnhEvents Collection

[This is preliminary documentation and subject to change.]

The EnhEvents collection represents the enhancement stream as a whole. It contains the complete set of EnhEvent objects that define the enhancement stream. Using EnhEvents, you can create enhancement authoring tools that programmatically add, edit, and delete events in the enhancement stream.

In addition to containing the EnhEvent objects, EnhEvents exposes properties to set global variables relating to the stream as a whole, such as the bandwidth required to transmit the stream. EnhEvents also provides utilities such as syntax validation and time format conversion.
