ShowLength

[This is preliminary documentation and subject to change.] 

To specify the length of an interactive show, use the reserved variable, ShowLength.

Syntax

ShowLength = hh:mm:ff ;

where hh is an integer denoting hours, mm is an integer denoting minutes, and ff is an integer denoting frames.

Example

In the following example, the show is 30 minutes long:

ShowLength = 00:30:00.00;
