LeadTime

[This is preliminary documentation and subject to change.] 

To specify lead time for an interactive show, use the reserved variable, LeadTime.

The reserved variable LeadTime ensures that files downloaded to a viewer's computer finish downloading before they are triggered to appear in TV Viewer or a browser. The value you place in this variable indicates the number of seconds that you want as a buffer between the finish of a file download to the viewer's computer and the activating of that file in the show.

Syntax

LeadTime = x ;

where x is an integer representing seconds.

Example

The following illustrates a lead time of five seconds:

LeadTime = 00:00:05.00;
