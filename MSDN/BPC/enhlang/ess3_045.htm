ShowName

[This is preliminary documentation and subject to change.] 

To specify the name of an interactive show, use the reserved variable, ShowName. The value for this setting names the directory of your interactive show when it is broadcast to the viewer's computer. All interactive shows are deposited into their own directory within the Spool directory. For example, if the name of the show was The Vanishing Serengeti, the directory that the show would be broadcast to would be C:\Windows\Spool\The Vanishing Serengeti.

Syntax

ShowName = "text" ;

where text is the name you want for the interactive show.

Note   The text of the show name must be enclosed in quotes.

Example

ShowName = "The Vanishing Serengeti";
