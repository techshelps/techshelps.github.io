Event Statements

[This is preliminary documentation and subject to change.] 

The Enhancement Stream language supports three different kinds of event statements. For more information, click on the event type of interest.

Announcements.   Inform a viewer's computer of either an upcoming interactive show or an interactive show that is currently being broadcast.

Data Downloads   Events that broadcast files to the viewer's computer for later use.

Triggers   Events that manipulate the components of an interactive show once they have been downloaded to the viewer's computer.
