DHTML Page Designer

Allows you to design an HTML page for use in a DHTML application. Using the designer, you can:

  • Create a new HTML page, or edit the contents of an existing page.

  • Determine which elements on the page are programmable and explore all the dynamic elements on the page.

  • Access the Code Editor window to write code for each programmable element on the page.

There are two panes to the page designer: the treeview pane and the detail pane.

Treeview Pane

Displays a hierarchical representation of all of the elements within an HTML page. For each element, the page designer lists the ID (if one exists), the type of control, and in some cases the beginning of the content for that element. If an element does not contain an ID, you can assign one to it in the Properties window. Elements with IDs are indicated in bold.

All elements in the treeview are children of the Body element and the Document object.

Detail Pane

Presents a drawing surface on which you can create a new page or edit the contents of an existing page. You can add HTML elements to the page, position elements, and set properties that control their physical appearance.
