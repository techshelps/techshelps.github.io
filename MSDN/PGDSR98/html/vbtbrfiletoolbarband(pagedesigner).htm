File Toolbar Band (DHTML Page Designer)

Contains buttons that are frequently used when working with pages in the designer. The items on this toolbar are available whenever you have a page designer open.

You can click a toolbar button once to carry out the action represented by that button.

Toolbar Buttons

DHTML Page Designer Properties

Opens the DHTMLPage Properties dialog box, in which you can determine how your page will be saved: as a separate HTML file or within the .dsr file for the designer. In addition, you can choose an external HTML file if you want to edit an existing page, or you can indicate that you want to create a new page.

Launch Editor

Opens Notepad or your computer's default HTML editor and displays the contents of the HTML page you are currently editing in the designer. You can make changes to the contents of the file and save it. This option only works when you have chosen to save your page in an external HTML file, or when the SourceFile property is set for the page.
