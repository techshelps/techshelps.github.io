DHTMLPage Properties Dialog Box

Allows you to specify how your page should be saved, and whether you want to use an existing HTML page or create a new one.

Options

Save Mode

Save HTML Inside the Designer

Saves the HTML file as part of the designer when you run the project. When the HTML is saved within the designer, you can't edit the page in an external editor. Visual Basic will create an external HTML file for the designer when it builds the project.

Save HTML in an External File

Saves the HTML file as an external file, to the location you specify. In this case, you can either enter a new file name, if you want to create a page from scratch, or you can select an existing page and modify it. Saving your HTML in an external file allows you to edit the page in an HTML or text editor. The value shown in the text box for this setting becomes the value of the designer's SourceFile property.

New

Opens the Create HTML File dialog box, used to specify a new file name to which you want to save your HTML file. Use this option when you want to create an HTML page from scratch.

Open

Brings up the Open dialog box, from which you can select an existing HTML page to import into the designer and edit. Use this option when you want to edit an HTML file that already exists.
