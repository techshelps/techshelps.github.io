DHTML Page Designer Shortcut Menus

Shortcut Menu Commands

Delete

Deletes the currently selected element. Available when you have selected any element from the treeview, except for the BODY element or Document object.

Cut

Removes the selected element and places it on the Clipboard. You must select at least one element for this command to be available.

Copy

Copies the selected element control onto the Clipboard. You must select at least one element for this command to be available.

Paste

Inserts the contents of the Clipboard at the current location.

View Code

Opens the Code Editor window and displays the event procedure for the selected HTML element. The Code Editor window shows the default event for the element. Available when you have selected any element that has an ID, from either pane of the designer.

Properties

Opens the Properties Pages dialog box for the selected element. Available when you have selected any element from either pane of the designer. If the selected element has no property page, a None tab will appear in the dialog box.

Absolute Position

Toggles the positioning mode of the current HTML element between absolute and relative. In absolute positioning, the element is placed on the page at the exact location you indicate. A series of parameters in the HTML for the element indicate the exact placement of the element.

If absolute positioning is off, the element is placed on the page in the next available space, with layout running from left to right and from top to bottom, across the relative width of the screen on the end user's computer.
