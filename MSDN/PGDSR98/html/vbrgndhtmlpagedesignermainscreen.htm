DHTML Page Designer Main Screen

Select from the following links to see help for the different areas of the page designer:

  • Treeview or detail panes of the designer window

  • Format toolbar band — Buttons used to format the elements on your page using standard HTML styles.

  • File toolbar band — Buttons used to set designer properties and launch external HTML editors for the current page.

  • Element toolbar band — Buttons used to add DIV, SPAN, hyperlink, and table elements to your page.

  • Design toolbar band — Buttons used to control the placement and positioning of the elements on your page.
