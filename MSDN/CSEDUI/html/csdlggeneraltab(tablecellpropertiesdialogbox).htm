General Tab (Table Cell Properties Dialog Box)

   

Allows you to set width, height, color, alignment, border colors, column span, and row span for table cells.

Background Color

Sets the background color for the cell. Select a color from the drop-down list, or choose Browse to select a color from the Color Picker dialog box. This option overrides any background colors previously set for the table using the General Tab of the Table Properties dialog box.

Alignment

Horizontal

Sets the left-to-right alignment of text in the cell.

Vertical

Sets the top-to-bottom alignment of text in the cell.

Dimensions

Height

Indicates the minimum height of the table in pixels. The table may appear larger in order to display contents.

Width

Sets the width of the table cell. Choose a unit for Width in the adjacent drop-down list.

Span

Columns

Specifies the number of columns that the cell extends across in the table.

Rows

Sets the number of rows that the cell extends down in the table.

Borders

Specifies the border color HTML attributes bordercolor (solid), bordercolorlight (highlight), and bordercolordark (shadow) for the table cell.

Color

Specifies a solid border color for the cell. Select a color name from the drop-down list, or choose a color from the Color Picker dialog box. If Highlight and Shadow are set, the Color setting is ignored.

Highlight

Sets the color for the bottom and right sides of the table cell. Select a color from the drop-down list, or choose a color from the Color Picker dialog box. Use this option with the Shadow option to create the appearance of a three-dimensional table cell.

Shadow

Sets the color for the top and left sides of the table cell. Select a color from the drop-down list, or choose a color from the Color Picker dialog box. Use this option with the Highlight option to create the appearance of a three-dimensional table cell.

Note   Use a light color for Highlight and a dark color for Shadow to maximize the three-dimensional appearance.

Wrap Text

Specifies whether text in cells is wrapped within the cell's borders. If unchecked, all text in each cell will appear on the same line. If checked, the browser will automatically word wrap text in each cell.
