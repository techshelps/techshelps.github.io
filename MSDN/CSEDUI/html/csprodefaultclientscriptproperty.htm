defaultClientScript Property

      

Specifies the language used to create client event handlers.

Settings

Choose VBScript or JavaScript.

Remarks

This setting is stored in a <META> tag and read by the editor when you double-click the name of an event handler in the Script Outline window. Changing this setting does not affect existing scripts.

Note   The value of this property is not used by the browser to determine the default language for client scripts.
