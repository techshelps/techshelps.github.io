alink Property

   

Specifies the color for an active link — a link currently selected by the user.

Settings

Specify a color by name (for example, "red") or hex value (for example, "ff0000"). Click the button to the right of the property value to display the Color Picker dialog box.

Remarks

When a link is not selected, it is displayed in the color specified by the link property. If the user has already visited the page specified in the link, the link is displayed in the color specified by the vlink property.

The value of this property is stored as the alink attribute of the <BODY> tag.
