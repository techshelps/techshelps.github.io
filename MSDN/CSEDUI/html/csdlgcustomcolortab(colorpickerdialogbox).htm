Custom Color Tab (Color Picker Dialog Box)

   

Allows you to select a color that is not one of the pre-defined colors from the Named Color or Safety Palette tabs. You create colors by mixing red, green, and blue (RGB).

This tab is primarily useful for creating colors to be displayed by browsers on computers that support a 16-bit color depth or higher. On computers with a lower color depth, most custom colors are displayed by dithering.

Options

Red

Specifies the amount of red in the custom color.

Green

Specifies the amount of green in the custom color.

Blue

Specifies the amount of blue in the custom color.

Color

Displays the hex value for the current color.
