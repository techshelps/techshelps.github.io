Color and Margins Tab (Page Properties Dialog Box)

   

Allows you to set the color for the page background, normal text, and link text.

You can use the Color tab to:

  • Select colors from the drop-down lists.

  • Choose a custom color.

  • View the color combinations as they would appear on the page.

Options

Color

Background

Sets the background color for the page. Select a color from the drop-down list, or choose Browse to select a custom color.

Text

Sets the normal text color for the page. Select a color from the drop-down list, or choose Browse to select a custom color.

Link

Sets the link color for links that the user has not visited. Select a color from the drop-down list, or choose Browse to select a custom color.

Visited Link

Sets the link color for links that the user has already traversed. Select a color from the drop-down list, or choose Browse to select a custom color.

Active Link

Sets the link color for links that the user has selected. Select a color from the drop-down list, or choose Browse to select a custom color.

Preview

Allows you to preview the interactions between the different page color options.

Page Margins

Note   Browsers typically have default values (usually 15 pixels) for page margins. If you are viewing the Color and Margins tab of the Page Properties dialog box while in Design view of the HTML editor, the default settings in the page margins boxes are set to 15 to reflect browser functionality. In Source view, the values are set to null. However, if you explicitly set a margin value, that value replaces the default value in either view.

Top

Specifies the margin for the top of the page in pixels. Overrides the default margin.

Bottom

Specifies the margin for the bottom of the page in pixels. Overrides the default margin.

Left

Specifies the left margin for the page in pixels. Overrides the default margin.

Right

Specifies the right margin for the page in pixels. Overrides the default margin.
