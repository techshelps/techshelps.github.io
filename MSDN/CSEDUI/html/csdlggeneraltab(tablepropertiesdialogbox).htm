General Tab (Table Properties Dialog Box)

   

Allows you to set the color, height, width, alignment, border size, border color, and other options for a table.

Options

Background Color

Sets the background color for the table. Select a color from the drop-down list, type an RGB value, or choose Browse to select a color from the Color Picker dialog box.

Background Image

Specifies the image file for the background graphic of the table. The image is tiled behind the table if the table exceeds the length and width of the image. Type the directory path and name for the file, or search for the file by choosing Browse.

Note   If you set a background color and also specify a background image, the background image will display instead of the background color.

Dimensions

Height

Indicates the minimum height of the table in pixels. The table may appear larger in order to display contents.

Width

Indicates the total width of the table. Choose a unit for the width in the adjacent drop-down list.

Layout

Alignment

Specifies the placement of the table on the page. By default, tables are aligned on the left margin of the page.

Cell Padding

Specifies the amount of space between cell contents and the cell border. Cell padding can range from 0-999 pixels.

Cell Spacing

Specifies the amount of space, in pixels, between the table border and the cells in the table and each cell from other cells. Cell spacing can range from 0-999 pixels.

Borders

Specifies the border color HTML attributes bordercolor (solid), bordercolorlight (highlight), and bordercolordark (shadow) for the table.

Size

Specifies the size, in pixels, of the outside table border. The border can range from 0-99 pixels. Choose a larger number to increase the width of the outside table border. Choose a smaller number to decrease the width of the outside table border.

Color

Specifies a solid border color for the table. Select a color name from the drop-down list, or choose a color from the Color Picker dialog box. If Highlight and Shadow are set, the Color setting is ignored.

Highlight

Sets the color for the bottom and right sides of each table cell and the color for the left and top sides of the table border. Select a color from the drop-down list, or choose a color from the Color Picker dialog box. Use this option with the Shadow option to create the appearance of a three-dimensional table border.

Shadow

Sets the color for the top and left sides of each table cell and the color for the right and bottom sides of the table border. Select a color from the drop-down list, or choose a color from the Color Picker dialog box. Use this option with the Highlight option to create the appearance of a three-dimensional table border.

Note   Use a light color for Highlight and a dark color for Shadow to maximize the three-dimensional appearance.
