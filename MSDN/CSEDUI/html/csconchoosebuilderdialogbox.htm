Choose Builder Dialog Box

   

Allows you to select and launch a utility (wizard, control, builder, and so on). The list of utilities displayed in this dialog box depends on what task you are performing at the time the dialog box is displayed, and on what utilities are installed on your computer.

For example, if you have chosen a command that starts a wizard, the Choose Builder dialog box shows you all the currently-installed wizards that are available in the current context. In some cases, only one choice might be displayed.

Options

Double-click the name of the utility to launch.
