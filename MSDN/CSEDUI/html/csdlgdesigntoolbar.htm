Design Toolbar

   

Contains buttons to specify how Design view in the HTML Editor displays your document and to specify absolute positioning for elements. The buttons on the toolbar are available only in Design view.

Buttons

Toolbar Button Name Description
Visible Borders Displays a 1-pixel border around HTML elements that support a BORDER attribute and have it set to zero, such as tables, table cells, and divisions.
Show Details Displays glyphs for HTML elements that do not have a visual representation, such as comments, scripts, and anchors for absolutely-positioned elements.
Absolute Positioning Converts the selected element into an absolutely positioned one (one that can be positioned using x and y coordinates), or converts an absolutely positioned element into a static one (positioned according to where it appears in the HTML text).
Lock Prevents an absolutely positioned element from being inadvertently moved.
Absolute Mode Specifies that when new elements are dragged from the ToolBox, they are positioned absolutely.

Note   Absolute mode applies only when you are working in Design view.
Snap to Grid Specifies that elements are aligned using an invisible grid. You can set grid spacing in the Options dialog box, and the grid will be changed the next time you open a document.
