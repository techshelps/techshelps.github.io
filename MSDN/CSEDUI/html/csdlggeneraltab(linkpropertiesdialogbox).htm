General Tab (Link Properties Dialog Box)

   

Allows you to specify Uniform Resource Locators (URLs), create bookmarks, select target frames, and specify pop-up text for links. For more information, see "Insert Hyperlink Dialog Box."

Options

Link Type

Specifies the type of link. For example, you could choose to create a mailto:, news:, http:, telnet:, or ftp: link.

Link

Sets the URL for the link. To search for a URL, choose Browse.

Create Bookmark

Specifies the name for the bookmark. You can create a bookmark by entering the bookmark name in the text box and then selecting (other) from the Link Type drop-down list.

Frame Target

Specifies the target window for the link.

Pop-up Text

Specifies the title that appears in a ToolTip when the mouse pointer hovers over the link.
