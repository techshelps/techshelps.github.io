Safety Palette Tab (Color Picker Dialog Box)

   

Allows you to select a color from a palette of 216 colors optimized for cross-platform use.

The safety palette features colors that are available on most computers used for browsing Web sites. If you select colors from only this palette, you minimize the chance that the browser will encounter a color it cannot display and substitute another color with unexpected results in your application.

Tip   For maximum compatibility with all types of browsers and computers, you can limit yourself to the 16 basic colors displayed in the Named Colors tab.

Options

Select a color from the color squares.

Color

Displays the hex value for the currently selected color.
