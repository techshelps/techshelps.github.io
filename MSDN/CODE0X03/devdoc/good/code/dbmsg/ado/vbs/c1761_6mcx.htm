ADOVBS Sample 
============= 
This sample demonstrates the use of ADO from VBScript and HTML and how ADO, VBScript and HTML on the client can be used to build a 2-tier weblication for the Intranet. In this sample, some of the VBScript and HTML code is generated at run-time to in order to generate fields based on the schema information in the Recordset.
