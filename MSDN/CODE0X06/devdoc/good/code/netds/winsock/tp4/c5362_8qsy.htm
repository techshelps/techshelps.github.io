README.TXT

This directory includes the following. 
 
- client.c and server.c files : sample WinSock applications. 
- makefile : Creates the executables for Client and Server 
 
The actual tp4 binaries can be found in the Windows NT driver library. 
I.e.  drvlib\protocol\tp4setup on the CD with the operating system. 
 
The tp4 Users Guide is provided on the Win32 SDK as doc\misc\msosi.doc. 
This is provided for people who are building solutions using tp4.
