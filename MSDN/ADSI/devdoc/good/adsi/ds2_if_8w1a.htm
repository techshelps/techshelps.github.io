IADsSession

The IADsSession interface enables a COM object in Active Directory to manage the information describing a connected session on a file service over a network.

Properties accessed by IADsSession are defined on the ADSI Session Object.

When to Implement

Implement IADsSession only when you are providing an Active Directory implementation for a directory service that represents open sessions across a network for a file service. If your directory service does not support this functionality, return E_NOTIMPL for this interface.

Under the Windows NT implementation, this interface is implemented on a dependent ADSI Session Object.

When to Use

Call the methods and properties of this interface when you are managing data the connected sessions of a software file service available through a network.
