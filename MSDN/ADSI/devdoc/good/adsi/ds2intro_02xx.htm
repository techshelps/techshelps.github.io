What You Should Know Before Reading This Guide

This guide assumes you are familiar with the Component Object Model (COM) and Automation, and that you know how to program in Visual Basic®, Java, or C/C++. For more information on these and other, related technologies, see ADSI References.

Some of the terms used in this guide are unique to the ADSI or directory services environments. Other terms will be familiar but may have slightly different meanings in these environments. For a list of ADSI terms and their definitions, see Glossary

This guide includes numerous code examples that demonstrate key features of ADSI and that may prove useful in your own code. For quick access to the code examples, see Code Examples.
