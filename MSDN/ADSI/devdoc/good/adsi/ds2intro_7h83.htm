What This Guide Contains

This guide explains the ADSI directory service architecture and the Active Directory Service Interface definitions. The first section presents information needed by ADSI client application writers. The second section presents information needed by ADSI provider writers.

Throughout the client application section, examples of writing to Active Directory Service Interfaces or to Active Directory Service Interface objects are provided for several language platforms, notably Visual Basic, Java, and C/C++. Resources for additional material are listed in ADSI References.
