ADSI Interface Definitions

The following ADSI interfaces are grouped by function.

  • Standard ADSI interfaces that provide the basic Active Directory object management functions. See Interfaces for Active Directory Object Management.
  • ADSI interfaces that use persistent information from a directory service. See Interfaces for Active Directory Objects Represented in the Underlying Directory Service.
  • ADSI interfaces that use dynamic information from a directory service. See Interfaces for Active Directory Objects Not Represented in the Underlying Directory Service.
  • ADSI interfaces that allow an ADSI client to establish its credentials to a server for a namespace and use security features supported by the directory service. See Security Interfaces.
  • ADSI interfaces that allow non-Automation clients low-overhead access to directory objects. See Non-Automation Interfaces.
