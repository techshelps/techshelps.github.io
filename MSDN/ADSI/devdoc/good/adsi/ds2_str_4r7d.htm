ADS_SORTKEY

The ADS_SORTKEY structure specifies information about sorting a query. 

typedef struct  _ads_sortkey
    {
    LPWSTR pszAttrName;
    LPWSTR pszReserved;
    BOOLEAN fReverseOrder;
    }    ADS_SORTKEY, *PADS_SORTKEY;

Members

pszAttrName
Pointer to a stream containing the name of the attribute.
pszReserved
Reserved for future use.
fReverseOrder
Reverse the order of the sorted results.
