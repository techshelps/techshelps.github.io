IADsResource

The IADsResource interface enables a COM object in Active Directory to manage information about a specific open resource in use by a file service, usually over a network.

Properties accessed by IADsResource are defined on the ADSI Resource Object.

When to Implement

Implement IADsResource only when you are providing an Active Directory implementation for a directory service that supports managing open resources over a network for a file service. If your directory service does not support this functionality, return E_NOTIMPL for this interface.

When to Use

Call the methods on this interface when you are referencing information about a currently open resource.

Methods in VTable Order

get_GUID Gets the object's globally unique identifier.

get_Parent Gets the object's ADsPath string for its parent container.
get_Schema Gets the ADsPath string to the schema class definition object for this object.
GetInfo Reloads the changes on this object and its dependents.
SetInfo Saves the changes on this object and its dependents.
Get Gets the value for a property by name.
Put Sets the value for a property by name.
GetEx Gets the value for a single or multi-valued property by name.
PutEx Sets the value for a single or multi-valued property by name.
GetInfoEx Reloads specific property values for this object as they exist in the underlying directory service.

IADsResource Properties Description
get_User Name of user for the Resource.

get_Path File system path of opened resource.
