IADsFileShare

The IADsFileShare interface enables a COM object in Active Directory to represent a published file sharing point of a file service in a directory.

Properties accessed by IADsFileShare are defined on the ADSI FileShare Object.

When to Implement

Implement IADsFileShare only when you are providing an Active Directory implementation for a directory service that publishes the file share points of services on a network. If your directory service does not support this functionality, return E_NOTIMPL for this interface.

When to Use

Call the methods on IADsFileShare when you are referencing or publishing information about a file share point for a file service.
