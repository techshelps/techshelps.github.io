IADsPrintJobOperations::Pause

Temporarily halts the processing of the print job currently active. To continue the processing, use IADsPrintJobOperations::Resume.

HRESULT IADsPrintJobOperations::Pause(void);

Return Values

This method supports the standard return values E_FAIL and E_UNEXPECTED, as well as the following:

S_OK
The pause command was successfully issued.

QuickInfo

  Windows NT: Use version 4.0 and later.
  Windows CE: Unsupported.
  Header: Declared in iads.h.
