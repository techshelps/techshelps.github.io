Interfaces for Active Directory Objects Represented in the Underlying Directory Service

The following interfaces provide access to data in a namespace that is persistent. This information includes user accounts, file directories, file shares, organizational hierarchies, and even such lists as print jobs waiting in a print queue.

IADsUser IADsService IADsLocality
IADsMembers IADsFileService IADsO
IADsGroup IADsFileShare IADsOU
IADsCollection IADsPrintQueue
IADsComputer IADsPrintJob
IADsDomain
