ADsFreeEnumerator

Frees an enumerator object previously created through ADsBuildEnumerator.

HRESULT ADsFreeEnumerator(
  IEnumVARIANT * pEnumVariant  //Enumerator object
);

Parameters

pEnumVariant
[out] Pointer to the IEnumVARIANT interface on the enumerator object to be freed.

Return Values

This method supports the standard return values E_FAIL and E_UNEXPECTED, as well as the following:

S_OK
The enumerator object has been freed successfully.

QuickInfo

  Windows NT: Use version 4.0 and later.
  Windows CE: Unsupported.
