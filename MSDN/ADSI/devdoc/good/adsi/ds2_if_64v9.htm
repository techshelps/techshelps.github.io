IADsService

The IADsService interface enables a COM object in Active Directory to manage the information stored in the underlying directory service describing a software service provided by a specific computer. For managing aspects of the service that are not persistently stored, such as controlling the service over the network, use IADsServiceOperations methods. For software services that are specifically file services, use the similarly designed IADsFileService and IADsFileServiceOperations interfaces.

Properties accessed by IADsService are defined on the ADSI Service Object.

When to Implement

Implement IADsService only when you are providing an Active Directory implementation for a directory service that publishes the availability of services on a network. If your directory service does not support this functionality, return E_NOTIMPL for this interface.

Under the Windows NT implementation, this interface is implemented on an WinNTService object.

When to Use

Call the methods and properties of this interface when you are managing information describing a software service available through a network.
