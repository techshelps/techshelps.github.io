IADsPrintJobOperations

The IADsPrintJobOperations interface enables a COM object in Active Directory to support methods that control a print job across a network.

IADsPrintJobOperations does not provide access to any properties.

When to Implement

Implement IADsPrintJobOperations only when you are providing an Active Directory implementation for a directory service that supports examining and controlling print jobs over a network. If your directory service does not support this functionality, return E_NOTIMPL for this interface.

When to Use

Call the methods and properties of this interface when you are browsing through or controlling print jobs over a network.
