IADsGroup

The IADsGroup interface enables a COM object in Active Directory to manage the information that describes and manages group membership.

Properties accessed by IADsGroup are defined on the ADSI Group Object.

When to Implement

Implement IADsGroup only when you are providing an Active Directory implementation for a directory service that supports group memberships.

When to Use

Call the methods on this interface to reference the description of a group membership.
