IADsServiceOperations

The IADsServiceOperations interface enables a COM object in Active Directory to manage the information that is not stored in the underlying directory service for a software service provided by a specific computer. For managing service information that is persistently stored, use IADsService properties and methods. For software services that are specifically file services, use the similarly designed IADsFileServiceOperations and IADsFileService interfaces.

Properties accessed by IADsServiceOperations are defined on the ADSI Service Object.

When to Implement

Implement IADsServiceOperations only when you are providing an Active Directory implementation for a directory service that supports controlling software services over a network. If your directory service does not support this functionality, return E_NOTIMPL for this interface.

When to Use

Call the methods and properties of this interface when you are managing the password or controlling a software service available on your network.
