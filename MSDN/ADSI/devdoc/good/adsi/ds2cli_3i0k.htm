ADSI Development Environment

The ADSI SDK contains the following files of interest to ADSI clients:

  • ADSI DLLs
  • ADSI OLE DB Provider DLLs
  • ADSI Provider DLLs for WinNT, LDAP, NDS, Netware 3.x
  • ADSI libraries
  • ADSI type library for Automation controllers
  • ADSI .odl files
  • ADSI include files
  • ADSI .dbg files
  • ADSI .hlp file
