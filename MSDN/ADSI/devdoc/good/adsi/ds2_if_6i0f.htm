IADsO

The IADsO interface enables a COM object in Active Directory to manage the data representing the organization to which an account belongs. This interface is one of several that provide support for directory services that organize accounts by country, locality (geographical location), organization (company), and organizational unit (department).

Properties accessed by IADsO are defined on the ADSI Organization Object.

When to Implement

Implement IADsO when you are providing an Active Directory implementation for your directory service that organizes directory entries by organization.

When to Use

Call the methods on this interface when you are expanding a directory that organizes accounts by physical location.
