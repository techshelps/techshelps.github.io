Interfaces for Active Directory Objects Not Represented in the Underlying Directory Service

The following interfaces provide methods that do not access a directory service's persistent data, but provide methods for namespaces that can work with volatile information, such as issuing commands to control processes over a network.

IADsComputerOperations IADsServiceOperations
IADsPrintQueueOperations IADsFileServiceOperations
IADsPrintJobOperations IADsResource
IADsSession
