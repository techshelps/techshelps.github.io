Select Folder Dialog Box

   

Use this dialog box to specify which folder you want Visual Component Manager to copy your component files to.  This dialog box appears when you click Add to toolbox… on the shortcut menu.

Options

Path

Specifies the full directory path of the folder you want to add your component files to. The display is updated when you change the path by using the Folders and Drive options, described below.

When the drive and folder you wish to use appears correctly in the Path box, click OK to close the dialog box and add your components to the selected folder and the toolbox of your development environment.

Folders

Lists the folders on your current disk drive. Use the scrollbar to move up or down, and double click on a folder to select it.  The selected folder appears in the Path text box. Click the New Folder button to add a new sub-folder to the current path.

Drives

Shows your current disk drive.  Click the arrow to drop down the list box and show all disk drives on your machine, and network drives that have been mapped to a drive letter. Click another disk to choose folders from a different disk drive. Click the Network… button to select a network drive that has not yet been mapped to a drive letter on your local machine.
