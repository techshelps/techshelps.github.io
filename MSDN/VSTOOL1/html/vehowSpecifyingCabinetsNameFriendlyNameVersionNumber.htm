Specifying a Cabinet's Name, Friendly Name, and Version Number

A cabinet file has a file name, friendly name, and a version number. These attributes are used by the Internet Explorer code download and by the Java Package Manager. With the Microsoft development environment, you can specify these attributes for your cabinet files.

To specify the file name, friendly name, and version number of a cabinet file

  1. Display the property pages of a distribution unit.

  2. In the Cabinet Properties tab, you can specify the file name, friendly name, and version number of the cabinet file.

  3. You can also set the compression algorithm used.

  4. Click OK.
