Item Keywords Dialog Box

   

Use this dialog box to pick from a list of currently available keywords when constructing a search string in the Find Items in Visual Component Manager dialog box, or when assigning keywords to a component from the Publish Wizard. You can also click the plus (+) sign to add new keywords to the list. See Finding Components for more information about keywords and searching.

Options

Add a New Keyword (+)

Displays the Add a Keyword dialog box, where you can add a new keyword to the repository database.

Available Keywords

Lists keywords currently available in the Visual Component Manager database.

List Keywords in

Lists the databases from which you can choose keywords.

Move All Keywords to the Right (>>)

Moves all keywords listed in the Available Keywords list to the Selected Keywords list.

Move All Keywords to the Left (<<)

Moves all keywords listed in the Selected Keywords list to the Available Keywords list.

Move the Selected Keyword to the Right (>)

Moves keywords selected in the Available Keywords list to the Selected Keywords list.

Move the Selected Keyword to the Left (<)

Moves keywords selected in the Selected Keywords list to the Available Keywords list.

Selected Keywords

Lists the keywords you have selected for this operation.
