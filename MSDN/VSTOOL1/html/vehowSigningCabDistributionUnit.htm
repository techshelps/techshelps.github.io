Signing a Cabinet (.cab) Distribution Unit

A cabinet file can be signed. The Microsoft development environment provides a convenient means to apply a digital signature to a cabinet distribution unit.

To sign an in-place cabinet (.cab)

  1. Display the property pages of a distribution unit.

  2. In the Cabinet Properties tab, select the Sign Cabinet check box.

  3. If you have a personal certificate and private key (for example, from Verisign, Inc.), click Use Certificate file, and then specify the certificate, the private key file name, and the timestamp server URL (the timestamp is optional).

    –or-

    If you just want to sign the cabinet file, with the Visual Studio test certificate, click Use Test Certificate. This certificate is for testing purposes only. Contact an authorization company to get a certificate and private key.

  4. Click OK.
