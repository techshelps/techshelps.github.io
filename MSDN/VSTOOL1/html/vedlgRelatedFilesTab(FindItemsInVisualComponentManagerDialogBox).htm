Related Files Tab (Find Items in Visual Component Manager Dialog Box)

   

Use this tab to specify whether you want to search all components in the current database, or only components that contain particular related files.

You can search based on sample code, source code, documentation, or Help files, or any combination of these related files. You can also specify that you want to search only items that contain a specific related file.

Options

All Components

Specifies, when selected, that you want the search to include all available components, regardless of related files.

Components that include

Enables, when selected, a list of related file types by which you can narrow your search. Select the check box next to a related file type to include it in the search.

An item named

Provides a place to enter the name of a particular file. Only components that contain the file you enter will be included in the search.
