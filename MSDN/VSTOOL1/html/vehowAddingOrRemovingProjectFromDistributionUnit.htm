Adding or Removing a Project From a Distribution Unit

You can add and remove items and projects from packaging projects in the Project Explorer.

To add a project to a distribution unit project

  1. In the Project Explorer, choose the distribution unit project to which you want to add a project.

  2. On the Project menu, click Add Project Outputs.

  3. In the Add Project Outputs dialog box, choose the project and item that you want to add to the distribution unit.

  4. Click Open.

    When you build the project, the outputs of the added project will be included.

To remove a project from a distribution unit project

  1. In the Project Explorer, choose the project or item in the distribution unit project that you want to remove.

  2. On the Edit menu, click Remove.
