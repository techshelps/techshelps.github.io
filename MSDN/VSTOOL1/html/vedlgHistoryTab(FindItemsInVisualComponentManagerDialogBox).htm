History Tab (Find Items in Visual Component Manager Dialog Box)

   

Use this tab to indicate that you want to search for components based on a particular publication history or usage limits.

Options

All Components

Specifies, when selected, that you want the search to include all available components regardless of history.

Components

Enables, when selected, a list of history criteria by which you can search. You can select an item in the list to include only those components that meet particular history requirements.

Published or Updated by

Provides a place to enter the name of a particular person. Select Published or Updated from the list, then enter the name by which you want to search.

Between

Specifies, when selected, that you want to search for items published within a given date range. Enter those dates in the boxes provided.

In the last

Specifies, when selected, that you want to search for item published within the time you specify. Enter a number, and then specify days, months, or years in the list provided.

Has been used

Specifies, when selected, that you want to limit the search according to the number of times the component has been used since publication. Select greater than (>), less than (<), greater than or equal to (>=), or less than or equal to (<=), and then enter a number of times. Only those components that meet the specified usage history will be included in the search.
