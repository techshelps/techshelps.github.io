Setting Java Permissions for a Cabinet (.cab) Distribution Unit

With the Microsoft development environment, you can sign cabinet files with Java permissions. You can specify certain permissions within the system to restrict the cabinet file.

To set Java permissions for a cabinet distribution unit

  1. Display the property pages of a distribution unit.

  2. Click the Java Permissions tab.

  3. In the Java Permissions dialog box, select the Sign Cabinet with Java Permissions check box.

  4. Choose your permission level. You can pick Low, Medium, High, or Custom. If you choose Custom, you can select the individual permissions from the Permissions list.

  5. If the cabinet (.cab) distribution unit contains Java components and ActiveX controls, select the Contains ActiveX controls check box. If your cabinet file contains only ActiveX controls, do not use Java permissions.

  6. Click Apply or OK.
