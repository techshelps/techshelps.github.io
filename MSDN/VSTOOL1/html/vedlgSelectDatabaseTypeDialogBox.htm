Select Database Type Dialog Box

   

Use this dialog box to indicate which type of Visual Component Manager database you want to create or open. This dialog box appears when you point to Repository on a shortcut menu, and then click New or Open on the shortcut menu.

Options

Microsoft Access Database (MDB)

Specifies, when selected, that you want to create or open a Microsoft® Access type database.

SQL Server Database

Specifies, when selected, that you want to create or open a Microsoft® SQL Server™ database.
