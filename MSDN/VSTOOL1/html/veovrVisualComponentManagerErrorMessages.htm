Visual Component Manager Error Messages

   

This section contains reference information on Visual Component Manager error messages. These error messages include the following.

3003 - Cannot paste <folder>

3009  - <Component> must be saved before it can be published

3010 - <File> is already in the list

3012 - Unable to create TypeLibInfo from tlbinf32.dll

3014 - Unable to register <component>!

3015  - <Component> does not require Registration!

4042  - <Template> does not contain any Controls

4044  - <Menu template> does not contain any Menus

4049 - Template Failed to load correctly!

4064 - Unable to load the requested document

4078 - This operation can only be performed when a project is loaded

4079 - This operation can only be performed when a component is loaded

4081 - Current Component cannot accept Menus

4087 - Database cannot be loaded!

4094 - Can't Paste a Cut Folder onto itself!

4103  - <Component> already exists

4113 - Current Component cannot accept Controls

4124 - Item no longer exists in the gallery!

4125 - Error Extracting File

4130 - Project failed to build!

4135  - <Menu> could not be added

4138  - <Item> already exists as a folder

6025 - Can't Remove the Primary File
