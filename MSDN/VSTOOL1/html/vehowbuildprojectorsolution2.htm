Building a Solution or Project

When you build your solution or project, you are building the active (default) build configuration. The two possible build configurations are: Debug or Release.

You can control the order in which files are built in the Build Order dialog box. For more information, see the Visual J++ documentation.

To build a solution

  • On the Build menu, click Build Solution.

To build a project or specific package

  1. In the Project Explorer, select the project or distribution unit.

  2. On the Build menu, click Build <project>.
