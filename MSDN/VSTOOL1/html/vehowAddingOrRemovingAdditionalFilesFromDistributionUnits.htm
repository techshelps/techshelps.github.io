Adding or Removing Additional Files from Distribution Units

You can add files to a distribution unit project to be included in the distribution unit after the project is built.

To add additional files to a distribution unit project

  1. In the Project Explorer, select the distribution unit in which you want to place the added items.

  2. On the Project menu, click Add Item.

  3. Browse around and find the items you want to add. Select the items and click Open.

To remove additional files from a distribution unit project

  1. In the Project Explorer, select the file in the distribution unit project.

  2. On the Edit menu, click Remove.
