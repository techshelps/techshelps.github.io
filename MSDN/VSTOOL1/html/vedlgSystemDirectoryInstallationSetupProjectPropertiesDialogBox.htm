System Directory Installation (Setup Project Properties Dialog Box)

   

Use this dialog box to place shared files in the Windows System directory of the target computer when the setup (.exe) extracts itself.

If you place these shared files in a different directory, the uninstaller for your setup (.exe) may not recognize when other programs share that resource file, and may inappropriately uninstall the file. Keeping all the common shared files in the same standard directory, the Windows System directory, ensures that the proper files will remain.

Options

Select the files to place into the target computer's Windows System directory

Displays a list of the files you can place in the Windows System directory when the setup (.exe) distribution unit extracts itself on the target computer. Select the files you wish to place in that directory.

This list includes files that are added as Additional Files or are project outputs.
