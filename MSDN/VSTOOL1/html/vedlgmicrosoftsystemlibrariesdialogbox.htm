Microsoft System Libraries (Setup Properties Dialog Box)

   

Use this dialog box to add Microsoft System Libraries to your setup executable distribution unit. Add the files here to set the entire library for installation on the target computer receiving the setup distribution unit.

Simply adding these files to the setup file with the Additional Files feature will not enable the libraries to self-extract and install on the target computer as part of the setup.

Options

Description

Displays the description of the selected Microsoft System Library file.

File List

Displays the files that are placed in the setup file for the selected Microsoft System Library file. This shows the directory path where the file is coming from. You can place the cursor in this list and press the right arrow button to see the rest of the list.

Listing

Displays a list of Microsoft System Libraries available to include in the setup (.exe) distribution unit. The selected library displays the details for the name, description, and file list.

Name

Displays the name of the selected Microsoft System Library file.
