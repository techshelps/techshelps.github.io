3001 - <Database> is already open

   

The Visual Component Manager database that you are attempting to open is already in use.

Try the following:

  • Check to see if you already have this database open.

  • If the database is not open, use the ODBC Data Source Administrator (in the Windows Control Panel) to view the DSN configuration for the database. If it is set for exclusive use only, turn the Exclusive flag off.
