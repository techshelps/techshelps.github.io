Add Annotation Dialog Box

   

Use this dialog box to enter textual annotations or comments to a component's properties. Annotations can be used for search criteria in the Description Tab of the Find Items in Visual Component Manager Dialog Box.

This dialog box is displayed when you click Add Annotation on the Annotations Tab of the Component Properties Dialog Box.

Options

Annotation by

Provides a place to enter the author of the annotation.

Annotation pane

Provides a place to enter the text of the annotation.
