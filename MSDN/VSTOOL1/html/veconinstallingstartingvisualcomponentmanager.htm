Installing and Starting Visual Component Manager

Visual Component Manager is automatically installed when you first install Visual Studio, Microsoft® Visual Basic, Microsoft® Visual J++™, Microsoft® Visual InterDev™, or Microsoft® Visual C++®. In Visual Basic, however, you must first add it to the toolbar in order to access it.

To add Visual Component Manager to the Visual Basic toolbar

  1. On the Visual Basic Add-ins menu, click Add-in Manager.

  2. In the add-ins list, scroll down and highlight Visual Component Manager.

  3. Under Load Behavior, select the Load on Startup check box.

  4. Click OK to add Visual Component Manager to the Visual Basic toolbar.

To start Visual Component Manager from Visual Basic

  • On the Visual Basic toolbar, click the Visual Component Manager icon.

    –or-

  • On the View menu, click Visual Component Manager.

To start Visual Component Manager from Visual J++ or Visual InterDev

  • On the Visual J++ or Visual InterDev toolbar, click the Visual Component Manager icon.

    –or-

  • On the View menu, click Visual Component Manager.

To start Visual Component Manager from Visual C++

  • On the Tools menu, click Visual Component Manager.
