Launch (Utility Project Settings Dialog Box)

   

Use this dialog box to specify the debugger to run when the active project is triggered in a build (F5) action.

You can set the order that the Microsoft development environment uses to execute the build commands in the Build Order dialog box.

Options

Arguments

Provides a place to enter any command line arguments for the debugger you want to run when the active project is built.

Program

Provides a place to enter the full path name for the debugger you wish to run when the active project is built.
