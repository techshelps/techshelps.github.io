Related Files Tab (Component Properties Dialog Box)

   

Use this tab to display and enter related files that are published as part of the item.

Options

Component Contains

List the types of files that may be related to the selected component. Select the check box next to a file type to indicate that the item contains related files of that type. The listed file types include:

  • Sample Code.

  • Source Code.

  • Documentation.

  • Help Files.

An item named

Use this text box to enter the name of a particular file. The search results will include all components which contain that particular item as a related file.
