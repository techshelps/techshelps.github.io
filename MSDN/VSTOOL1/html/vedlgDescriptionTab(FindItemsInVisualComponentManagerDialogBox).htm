Description Tab (Find Items in Visual Component Manager Dialog Box)

   

Use this tab to enter descriptive criteria to search for a particular component or other item in Visual Component Manager.

Options

Named

Provides a place to enter a component name. You can use the * wildcard character to specify any string of 0 or more characters in the component name.  For example, GRAPH* will find components named GRAPHICS.DLL, GRAPHDATA.OCX, GRAPH.EXE, and so on.

Containing text

Provides a place to enter one or more keywords or search strings. Click the ellipses (…) to display the Item Keywords dialog box, where you can select from the list of all keywords in your database. You can also use the * wildcard character to specify any string or any single character, respectively.

Of type

Lists types of items for which you can search.

Containing text in

Lists properties you can search for text. A check mark next to any of these properties means the search will return components for which that property contains the keyword or search string you entered in the Containing text box. The searchable properties are:

  • Keywords

  • Description

  • Annotations

Look In

Lists the available databases in which you can search.
