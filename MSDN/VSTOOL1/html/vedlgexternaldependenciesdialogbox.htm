External Dependencies (CAB Properties Dialog Box)

   

Use this dialog box to specify that the components in the cabinet file that you are building contain dependencies on other cabinet files that may be located on the intranet or Internet.

Options

Add

Displays the Add/Edit External Dependency dialog box, where you can add an external dependent cabinet's URL, friendly name, and version number.

Delete

Removes the selected dependency entry from the list of Dependencies on the Web.

Dependencies on the Web

Displays the list of your dependency cabinet files. For each dependent cabinet file, you can see the friendly name, URL, and version number.

Edit

Displays the Add/Edit External Dependency dialog box, where you can edit the external dependent cabinet's URL, friendly name, and version number.
