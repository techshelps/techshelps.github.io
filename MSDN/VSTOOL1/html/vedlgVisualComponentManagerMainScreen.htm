Visual Component Manager Main Screen

   

Use the main screen to select commonly used commands, and view the databases, folders, subfolders, and individual items stored by Visual Component Manager. These elements are briefly described below.

Options

Folders

Select from this list to set the focus of the explorer pane to a different database or folder.

Up One Level

Moves the focus up one level in the explorer pane.

Add a New Folder

Creates a new folder in the current database and inserts it into the explorer pane.

Publish a New Component

Starts the Visual Component Manager Publish Wizard to step you through publishing a new component.

Search

Opens the Find Items in Visual Component Manager dialog box, allowing you to search for items of any type. You can search by name, description, keywords, property values, or any combination of these.

Cut

Moves the current folder or item to the Clipboard.

Copy

Copies the current folder or item to the Clipboard.

Paste

Copies the group or item in the Clipboard to the selected folder.

Show/Hide Explorer Pane

Shows or hides the explorer pane in the Visual Component Manager user interface.

Show/Hide Properties Pane Button

Shows or hides the properties pane in the Visual Component Manager user interface.

Explorer

Hierarchically lists the databases, folders, and subfolders currently loaded.

Contents

Displays the items (components and other published files) in the currently selected folder of the repository database.

Properties

Displays the properties of the currently selected item. For COM components, the properties pane converts to a browse pane to display the object hierarchy when you select Browse from the shortcut menu.
