Adding Projects or Files to the Deployment Target

Deployment targets contain references to the projects and files that you want to be deployed. In the Deployment Explorer you can see the projects and files that will be deployed listed under each deployment target.

Once the files are listed in the appropriate service node, you can deploy that deployment target.

To add a project to an existing deployment target

  1. In the Deployment Explorer, expand the deployment target.

  2. Select a service node to which you want to add your project. For example, you can choose Web Content, Controls & Applets.

  3. On the Project menu, click Add Project Outputs.

  4. In the Add Project Outputs dialog box, select the projects that you want to add to the service.

  5. Click OK.

To add a file to an existing deployment target

  1. In the Deployment Explorer, expand the deployment target.

  2. Select a service node to which you want to add your file. For example, you can choose Web Content, Controls & Applets.

  3. On the Project menu, click Add Files.

  4. In the Add Files dialog box, select the files that you want to add to the service.

  5. Click OK.
