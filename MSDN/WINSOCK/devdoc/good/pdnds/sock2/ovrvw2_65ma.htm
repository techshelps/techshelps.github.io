QOS Templates

It is possible for QOS templates to be established for well-known media flows such as H.323, G.711, and others. The WSAGetQOSByName function can be used to obtain the appropriate QOS structure for named media streams. It is up to each service provider to determine the appropriate values for each element in the QOS structure, as well as any protocol or media-dependent QOS extensions. The documentation for the WSAGetQOSByName function will be periodically updated with a list of flow specifications and general descriptions as they become well-known. The WSAGetQOS ByName function can also be used to enumerate the set of known QOS template names.
