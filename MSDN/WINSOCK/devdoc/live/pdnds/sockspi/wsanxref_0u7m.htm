Option Profiles

OSI protocols typically have a large number of options. These options may include preferred class, alternative classes, timers, checksums, lifetime, and others. An option profile name may be used to select a set of options available in a particular vendors OSI protocol implementation. The Windows Sockets specification does not define the options that may be selected.

Option profile names are specified in the OSI address structure. If an option profile name is not specified, the OSI protocol will use its default option profile.
