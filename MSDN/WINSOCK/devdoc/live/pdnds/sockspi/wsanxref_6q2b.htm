ATM Controls

ATM point-to-point and point-to-multipoint connection setup and teardown are natively supported by the Windows Sockets 2 specification. In fact, Windows Sockets 2 QOS spec and protocol-independent multipoint/multicast mechanisms were designed with ATM in mind, along with other protocols. See section 2.7 and appendix D of the Windows Sockets 2 API specification for Windows Sockets 2 QOS and multipoint support, respectively. Therefore, no ATM specific ioctls need to be introduced in this document.
