Using Socket/WSASocket/WSPSocket

To create a socket that is bound to a DECnet service provider, the following values should passed to the socket call.

#include <ws2dnet.h>

SOCKET WSAAPI socket ( int af, int type, int protocol );

SOCKET WSAAPI WSASocket(int af, int type, int protocol, ...)

SOCKET WSPAPI WSPSocket(int af, int type, int protocol, ...)
af = AF_DECnet DECnet address family
type = SOCK_SEQPACKET Sequenced packets (message-oriented)
protocol = DNPROTO_NSP DECnet NSP transport protocol
