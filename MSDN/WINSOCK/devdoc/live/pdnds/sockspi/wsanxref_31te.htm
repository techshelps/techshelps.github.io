RSVP

The Resource Reservation Protocol (RSVP) implementation of the Windows Sockets 2 Protocol-Specific Annex, found at ftp://ftp.microsoft.com/bussys/winsock/winsock2/wsanx203.doc, is not supported by Microsoft. For information on how Microsoft implements RSVP for quality of service (QOS), please refer to the Generic Quality of Service (GQOS) documentation in the Platform SDK.
