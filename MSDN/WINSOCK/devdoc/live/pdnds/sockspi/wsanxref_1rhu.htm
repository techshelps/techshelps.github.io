Quality Of Service

The provider specific parameters in the Flow Spec are encoded in a Type/Length/Data format to enable multiple parameters to be provided. The OSISPECFLOWPARAM structure is used to encode the parameter.

The following parameter types are defined:

Parameters Type Meaning
OSI_PARAM_ID_X25_CALL_FACILITY u_char[] ISO 8208/X.25 call facilities.

OSI_PARAM_ID_X25_CALL_FACILITY
This parameter specifies the ISO 8208/X.25 call facilities to be used. The format of the data is the same as is coded in an X.25 call request/call confirm packet. For further details see the X.25 specification.
