GQOS API

[This is preliminary documentation and subject to change.]

The GQOS API is the programmatic interface to the GQOS service provider (GQOS SP). Under most circumstances, the GQOS API is the only interface that programmers will require to create QOS-aware or QOS-enabled applications. The GQOS API is the hot-water drip in the virtual coffee-making system that is GQOS; (almost) all things that happen in the GQOS sequence are a result of GQOS API calls percolating down (and sometimes back up) through GQOS components, in the end creating fixed-bucket sized servings of data that help keep the network from getting lethargic.

For more information on the programming elements included in the GQOS API, see GQOS Reference.
