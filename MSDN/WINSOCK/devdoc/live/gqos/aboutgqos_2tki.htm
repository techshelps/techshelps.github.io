Subnet Bandwidth Manager

[This is preliminary documentation and subject to change.]

The Subnet Bandwidth Manager (SBM) is the GQOS component that provides resource management and policy-based admission control for QOS-aware applications that use local 802 subnets (Ethernet, for example). The SBM is based on an IETF-draft that defines SBM functionality and its general implementation.

SBM is implemented in Windows NT 5.0 through the Admission Control Service (ACS). ACS is a Windows NT service that resides on a Windows NT 5.0 server.
