MMCN_DESELECT_ALL

[This is preliminary documentation and subject to change.]

Sent to the virtual list view when all items of an owner-data result pane are deselected.

Parameters

dataobject
NULL.
arg
0
param
0

Return Values

Not used.

Remarks

When an item is deselected in a virtual list view, unlike a regular list view, there is no way to know which item was deselected. Only -1 is passed, indicating deselect all. Because of this, this notification is not sent for virtual list views.
