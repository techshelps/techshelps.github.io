IConsoleVerb::GetDefaultVerb

[This is preliminary documentation and subject to change.]

Gets the snap-in's default verb. 

HRESULT GetDefaultVert(
  MMC_CONSOLE_VERB peCmID,    // Pointer to the default verb
);

Parameters

peCmID
[out] Pointer to where the snap-in's default verb is returned.

Return Values

S_OK
The default verb was returned successfully.

Remarks

The MMC_CONSOLE_VERB enumeration defines the set of default verbs.
