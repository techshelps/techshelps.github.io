IComponent::Destroy

[This is preliminary documentation and subject to change.]

Releases all references to the console.

HRESULT Destroy(
  LONG cookie  // reserved
);

Parameter

cookie
Reserved. Not in use at this time.

Return Values

S_OK
Indicates that all the references to the console were successfully released.
E_UNEXPECTED
An unexpected error occurred.

Remarks

The snap-in is in the process of being unloaded. Release all references to the console.

Notes to Callers

Only the console calls this method.
