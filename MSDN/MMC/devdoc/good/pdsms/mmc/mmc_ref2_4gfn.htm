IPropertySheetProvider::AddExtensionPages

[This is preliminary documentation and subject to change.]

Collects the pages from the extension snap-ins.

HRESULT AddExtensionPages( );

AddExtensionPages has no parameters.

Return Values

S_OK
The pages were successfully collected.
E_UNEXPECTED
An unexpected error occurred.

Remarks

The primary snap-in must have added at least one page before extensions can add pages.
