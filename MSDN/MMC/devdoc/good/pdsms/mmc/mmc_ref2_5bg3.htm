MMCN_ADD_IMAGES

[This is preliminary documentation and subject to change.]

Sent to the snap-in's IComponent implementation to add images for the result pane.

Parameters

lpDataObject
[in] Pointer to the data object of the currently selected scope pane item.
arg
Pointer to the result pane's image list (IImageList).
param
Specifies the HSCOPEITEM of the item that was selected or deselected.

Return Values

Not used.

Remarks

The primary snap-in should add images for both folders and leaf items. Extension snap-ins should add only folder images.

See Also

IComponent::Notify, IComponentData::Notify, IExtendControlbar::ControlbarNotify
