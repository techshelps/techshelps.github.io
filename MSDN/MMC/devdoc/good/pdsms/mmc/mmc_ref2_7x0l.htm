MMCN_QUERY_PASTE

[This is preliminary documentation and subject to change.]

Sent to the snap-in before pasting into the snap-in's folder to determine if the snap-in can accept the data.

Parameters

pdataobject
The dataobject of the selected item provided by the snap-in.
arg
The dataobject of the item(s) provided by the source snap-in that need to be pasted.
param
Not used.

Return Values

Not used.
