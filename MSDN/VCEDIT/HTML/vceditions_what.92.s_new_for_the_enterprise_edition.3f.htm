What's Included in the Visual C++ Enterprise Edition?

The Enterprise Edition of Microsoft Visual C++ 6.0 provides many tools and components for building and validating enterprise-level distributed Component Object Model (COM) applications.  It includes all the features that are in Professional Edition, plus the following features:

  • Microsoft Transaction Server

  • Visual Database Tools

  • SQL Editor

  • SQL Debugger

  • MFC Database Classes

  • ADO Data Binding

  • AppWizard and Data Sources

  • Remote Automation Components

  • MFC Data Binding

  • Visual SourceSafe

  • Remote Automation
