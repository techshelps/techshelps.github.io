Visual C++

Welcome to Microsoft Visual C++ version 6.0! Visual C++ is available in three editions: Standard, Professional, and Enterprise.


The documentation for Visual C++ includes: the Visual C++ Programmer's Guide, the Visual C++ User's Guide, Visual C++ Tutorials, the Microsoft Foundation Classes and Templates, and Languages and Libraries for Visual C++.

Visual C++ is a member of the Visual Studio 6.0 family of development products, which includes:

  • Visual Basic
  • Visual C++
  • Visual FoxPro
  • Visual InterDev
  • Visual J++
  • Visual SourceSafe
  • MSDN Library
