The SQL Editor

Feature Only in Enterprise Edition   This feature is supported only in Visual C++ Enterprise Edition. For more information, see Visual C++ Editions.

The integrated Text editor in Microsoft Visual C++ Enterprise Edition color codes SQL keywords to help you minimize syntax and spelling errors when editing SQL source files. This color coding, when combined with the ability of Data View to generate skeletal triggers and stored procedures, simplifies the creation of both SQL scripts and SQL procedures.

For more information on SQL editing, see Using the Text Editor.
