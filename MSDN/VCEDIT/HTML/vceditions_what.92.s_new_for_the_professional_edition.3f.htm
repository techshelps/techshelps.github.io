What's Included in the Visual C++ Professional Edition?

The Professional Edition of Microsoft Visual C++ version 6.0 provides features that let you develop and distribute commercial-quality software products.

The Professional Edition includes the following features (see the Visual C++ Documentation Map for another high-level look at the Visual C++ product):

Programming Features

  • Microsoft Foundation Classes

  • Active Template Library

  • Support for Common Controls in Internet Explorer

  • Compiler COM Support

  • OLE DB Provider and Consumer Templates

  • Component Object Model (COM)

  • Active Document Containment

  • Samples

  • C/C++ Run-Time Libraries

The Internet

  • Internet Server API (ISAPI) Extensions

  • CHttpConnection (and other CHttp* MFC classes)

  • CInternetConnection (and other CInternet* MFC classes)

ActiveX Controls

  • Composite Control Fundamentals

  • Developing MFC ActiveX Controls

  • Developing ATL ActiveX Controls

  • Ready-to-Use ActiveX Controls

Project Features

  • Wizard Support for Creating Applications

  • Integrated Development Environment and more IDE topics

  • Debugging Your Application

  • Source Code Profiler

  • Component and Object Gallery

  • InstallShield

Optimizations

  • Inline Optimizing of Your Code

  • Compiler Optimization Options

  • Linker Optimizations (/OPT)

  • C++ Exception Handling

Database Support

  • ActiveX Controls for Databinding

  • ADO

  • Read-only versions of Microsoft Visual Database Tools (Enterprise Edition has full-feature Visual Database Tools)
