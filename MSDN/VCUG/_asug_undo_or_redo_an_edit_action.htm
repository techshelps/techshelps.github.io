Undoing or Redoing an Edit Action

Home Page (Text Editor) |  Overview |  How Do I ... Topics |  FAQ |  Reference

Use the Undo command to undo previous editing actions. Use the Redo command to reapply editing actions that have been undone. Redo is unavailable unless you have used the Undo command.

The number and scope of editing actions you can undo is determined by the size of the Text editor’s UndoRedoSize buffer in the registry.

Note   You can also undo automated edits. For example, if you have used ClassWizard to add a command handler, you can undo the ClassWizard edits by choosing AutomatedEdit in the Undo drop-down list.

To undo an edit action

  • From the Edit menu, choose Undo.

To redo an edit action

  • From the Edit menu, choose Redo.
