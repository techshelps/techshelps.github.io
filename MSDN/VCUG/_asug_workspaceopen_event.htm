WorkspaceOpen Event

Home Page (Objects) |  Overview |  FAQ |  Reference

Applies to: Application object

Occurs after a workspace opens.

Syntax

object_WorkspaceOpen

Parameters

object

An expression that evaluates to an Application object.

Remarks

Developer Studio fires this event after a workspace opens.

For an overview about using event code, see the introduction to events.

Example

Following is a sample event handler for the WorkspaceOpen event. Copy this into your macro file, and then insert the appropriate code.

sub    Application_WorkspaceOpen()
   ' Insert code to handle the event here
end sub
