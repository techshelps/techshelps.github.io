Overview: Resource Editors

Home Page (Resource Editors) | How Do I ... Topics

The Resource editors share techniques and interfaces to create and modify application resources quickly and easily. You can use the Resource editors to create new resources, modify existing resources, copy existing resources, and delete old resources. The Resource editors are functionally consistent for ease of use.

You can edit all of the Microsoft Windows resources that your application uses. These resources include:

  • Accelerator tables

  • Binary data information

  • Bitmaps, icons, and other image files

  • Cursors

  • Dialog boxes

  • HTML pages

  • Menus

  • String tables

  • Toolbar resources

  • Version information

When you create or open a resource, the appropriate editor opens automatically. You can also open a resource (.rc) file in text format.

The Resource editors have many commands and procedures in common. For example, once you learn how to create and open a dialog box, you know the steps for creating and opening any of the other resources.
