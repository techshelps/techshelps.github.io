Viewing the Contents of the Thread Information Block

Home Page (Debugger) |  Overview |  How Do I... Topics

You can view the contents of the Thread Information Block (TIB) for the current thread. The debugger uses a pseudoregister called TIB for this purpose. You can view the contents of this pseudoregister in the Watch window or Quickwatch dialog using the standard procedures for displaying registers (discussed in Viewing the Value of a Variable or Contents of a Register).

For information on Thread Information Blocks, see "Under the Hood" in the May 1996 edition of the Microsoft Systems Journal.
