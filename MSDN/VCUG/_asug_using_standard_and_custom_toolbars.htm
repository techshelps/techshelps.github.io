Using the Source-Code Control Toolbar

Home Page (Source-Code Control) |  Overview |  How Do I ... Topics

You can access the integrated source-code control commands from buttons on a standard toolbar.

To display the Source-Code Control toolbar

  1. On the Tools menu, click Customize, then Toolbars.

  2. From the Toolbars list, select Source Control.

  3. Click Close.

You can also remove buttons from the toolbar, add other buttons, or add source-code control buttons to other toolbars. For information on customizing toolbars, see the section Working with Toolbars.
