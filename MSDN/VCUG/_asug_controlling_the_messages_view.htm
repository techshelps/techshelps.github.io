Controlling the Messages View

Home Page (Spy++) | Overview | How Do I ... Topics

With Spy++, you have considerable control over the content of the Messages view. You can start and stop displaying the messages at any time, and you can specify:

  • Which message types you want to see.

  • Which windows you want to monitor.

  • The display format for message lines.

These settings are available from the Message Options dialog box and apply only to the selected Messages view.
