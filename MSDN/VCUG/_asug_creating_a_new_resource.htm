Creating a New Resource

Home Page (Resource Editors) |  Overview |  How Do I ... Topics

You can create a resource as a new default resource, or as a resource patterned after a template. Creating a new default resource is as easy as clicking the appropriate button on the Resource toolbar. For more information on creating a resource from a template file, see Using Resource Templates.

When you create a resource, it is assigned a unique symbol name and value. If you need to change the symbol value, you can use the ID box on the resource’s property page. For more information on the property page, use the ? button.

To create a new resource

  1. From the Insert menu, choose Resource.

  2. Select a resource from the Resource Type list box and click New.

    Visual C++ creates the new resource and assigns it a unique ID, for example, IDD_Dialog1. You will probably want to customize this ID.
