Viewing Resources

Home Page (Resource Editors) |  Overview |  How Do I ... Topics

You can access your project resources from the ResourceView pane of the Project Workspace. Select the ResourceView tab in the Project Workspace if ResourceView is not the top-most pane.

To view a Resource

  1. From the ResourceView pane, expand the folder for the project whose resources you want to view.

  2. Expand the folder containing the resource you wish to view.

    For example, if you want to view a dialog resource, expand the Dialog folder.

  3. Double-click the resource, for example, IDD_ABOUTBOX.

The resource opens in the appropriate editor. For example, for dialog resources, the resource opens inside the Dialog editor.

You can also open a resource file as text.
