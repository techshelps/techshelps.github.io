Moving a String from One Segment to Another

Home Page (String Editor) |  Overview | How Do I ... Topics

To move a string from one segment to another

  1. Select the string you want to move.

  2. From the View menu, choose Properties.

  3. Change the string’s value in the ID box so that it falls in the range you want.

    For example, to move a string with a name of IDS_MYSTRING and a value of 100 to a segment in the 200 range, type the following in the ID box:

    IDS_MYSTRING=201
  4. Press ENTER.
