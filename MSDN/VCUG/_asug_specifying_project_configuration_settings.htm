Specifying Project Configuration Settings

Home Page (Projects) |  Overviews |  How Do I... Topics |  NMAKE Reference

The options chosen in the Project Settings dialog box can be applied to multiple projects, but apply only to the selected project configuration. For instance, you can specify settings for the Release configuration that do not apply to the Debug configuration. You can also specify that settings be applied to all current project configurations.

To specify project settings

  1. On the Project menu, click Settings.

  2. On the General tab, select the project or projects.

  3. In the Settings For pane, select the configuration, or select All Configurations.

  4. Click OK to save the settings.

    Note   If you specify settings that are incompatible with the project type that you chose when you created your project, you may not get the results that you expect.
