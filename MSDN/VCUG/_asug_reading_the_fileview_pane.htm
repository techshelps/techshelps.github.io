Reading the FileView and ClassView Pane

Home Page (Source-Code Control) |  Overview |  How Do I ... Topics

The FileView pane of the Project Workspace window displays all the files that are currently in the project workspace, and the ClassView pane displays all the classes. If a project is under source-code control, the file icons are grayed, and if a file is checked out, a check mark appears to the left of the file icon.

If all the files associated with a class are checked out, a red check mark will appear to the left of the class icon. If some, but not all, files associated with a class are checked out, a black check mark will appear to the left of the class icon.

For more information see: Examining the property pages for a file,Examining the file’s history, Viewing the changes to a file.
