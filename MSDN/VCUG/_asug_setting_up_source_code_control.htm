Setting Up Source-Code Control

Home Page (Source-Code Control) |  Overview |  How Do I ... Topics

Until you install a source-code control system that conforms to the Microsoft Common Source-Code Control Interface, the Visual C++ source-code control menu commands will not appear. In addition, either you or the installation program must make the correct entries in the Registry.

To use Visual C++ integrated source-code control:

  1. Install a source-code control system that conforms to the Microsoft Common Source Code Control Interface.

  2. Ensure that the installation program for the source-code control system writes the correct information to the Registry, and if it doesn’t, make the correct entries.

  3. Complete all the administrative tasks required by your source-code control system. These tasks may include designating locations for master versions of files, creating network connections, setting permissions for drives and/or directories, or adding user information and permissions to the system.
