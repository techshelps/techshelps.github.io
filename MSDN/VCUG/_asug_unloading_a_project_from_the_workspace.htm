Unloading a Project from the Workspace

Overviews |  How Do I... Topics |  NMAKE Reference

Unloading a project from the workspace can enable you to work more efficiently. For example, your workspace may contain multiple projects with multiple configurations. To reduce the time it takes to open and close the workspace, you can unload projects that you aren't currently working on.

Unloading a project does not delete it from the workspace. If a project is currently unloaded, it is simple to load it again.

To unload or load a project

  1. On the FileView tab of the Project Workspace, position the cursor over the project you wish to unload or load, and right-click the mouse button.

  2. From the shortcut menu, choose Unload Project or Load Project.

    When the project is unloaded, its icon changes to a folder and the words '– not loaded' appear next to it.
