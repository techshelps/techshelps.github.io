Previewing the HTML Resource

Home Page (HTML Resource) |  Overview |  How Do I ... Topics

As you code your HTML page, you'll undoubtedly want to see how it will appear when viewed with a Web browser. You can do so by using the Preview feature, which launches your current system browser to display the page.

To preview an HTML resource

  • With the HTML resource open in the Text editor, right-click in the page and choose Preview.

Note   In order to preview the HTML page, the resource must be an external file (this is the default). To verify that this is the case, right-click the resource in the ResourceView pane and choose Properties, then ensure that the External file check box is selected.
