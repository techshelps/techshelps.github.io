Document Windows

Home Page (User Interface) |  Overview |  How Do I ... Topics

The following windows are document windows:

  • Editor windows, either for text or resources

  • The resource browser window (except when it is used for active project resources in the project workspace)

Document windows are associated with the project workspace. Visual C++ records their positions, sizes, any selections made in them, any splitting of windows, and so on, once you close a project. When you open the project again, these characteristics are restored.

When you are working with documents using the text editor or other resource editors, you can switch to full-screen mode. With full-screen mode, you can see more text at once.
