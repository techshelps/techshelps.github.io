Formatting the Layout of a Dialog Box

Home Page (Dialog Editor) |  Overview |  How Do I ... Topics

The Dialog editor contains special tools for layout. These tools help align and arrange controls in the correct place.

You can use the Dialog editor in three different states for moving controls: with the guides and margins on (default setting), with the layout grid on, or without any snapping or alignment features.
