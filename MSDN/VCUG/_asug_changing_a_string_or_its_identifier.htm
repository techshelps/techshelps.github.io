Changing a String or Its Identifier

Home Page (String Editor) |  Overview |  How Do I ... Topics

To change a string or its identifier

  1. Select the string you want to edit.

  2. From the View menu, choose Properties, and modify the string in the Caption box.

  3. In the ID box, modify the string’s identifier:
    • Type a new ID, or select one from the list.

    • To change a string’s value, type the ID followed by an equal sign and the new value; for example:
      IDS_ERROR_MSG=2350
  4. Press ENTER.
