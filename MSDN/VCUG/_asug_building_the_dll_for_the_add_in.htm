Building the DLL for the Add-in

Home Page (Add-ins) |  Overview |  How Do I ... Topics |  FAQ |  Reference

If you use the Developer Studio Add-in Wizard to create an add-in, the wizard creates source files for the add-in but does not build the DLL from these files.

To build the DLL for an add-in

  1. Open the project for the add-in.

  2. On the Build menu, click Build dll, where dll is the name of the DLL.
