Overview: Source-Code Control

Home Page (Source-Code Control) |  How Do I ... Topics

The Visual C++ environment provides the functionality for integrating a system to manage your source code files. If you install a source-code control system that conforms to the Microsoft Source-Code Control Interface, you can directly access source-code control functionality from the Visual C++ menus.
