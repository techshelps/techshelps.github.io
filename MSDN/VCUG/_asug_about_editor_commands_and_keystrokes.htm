About Editor Commands and Keystrokes

Home Page (Text Editor) |  Overview |  How Do I ... Topics |  FAQ |  Reference

You can access Text editor commands in many ways: from the menus, from the Edit toolbar, and from the shortcut menu. There are many Text editor commands, but not all of them appear on the menus and toolbars. However, you can drag any command to a menu or toolbar, which allows you to customize the Editor with the commands you find most useful.

To see a complete list of commands, choose Keyboard from the Help menu, and choose Edit in the drop-down list. All of the Text editor commands are listed, along with the current key binding and a brief description. For information on customizing menus and toolbars, see the Customizing Visual C++ Overview.

The Advanced menu (available from the Edit menu) contains some of the most useful edit commands, including Incremental Search, Format Selection, Tabify Selection, Untabify Selection, Make Selection Uppercase, Make Selection Lowercase, and View Whitespace.

The Edit toolbar contains other useful commands. To display the Edit toolbar, choose Customize on the Tools menu. Click Toolbars, then select the Edit check box, and click Close.
