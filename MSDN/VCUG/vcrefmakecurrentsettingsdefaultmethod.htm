MakeCurrentSettingsDefault Method

Home Page (Objects) |  Overview |  FAQ |  Reference

Applies to: Configuration object

Changes the default settings for this configuration to the values of the current settings.

Syntax

object.MakeCurrentSettingsDefault

Parameters

object

An expression that evaluates to a Configuration object.

Remarks

The default settings for a project are initially set up by the AppWizard with which you created your project. This method allows you to change those defaults by applying settings changes to a configuration and then calling this method. To restore settings to the default values (the values you defined as the defaults), the user can click the Reset button in the Project Settings dialog box.

Tip   This method is particularly useful in writing Custom AppWizards.

Example

The following example adds the /WX switch as a default setting for every project in the workspace:

for each proj in Application.Projects
   for each cfg in proj.Configurations
      cfg.AddToolSettings "cl.exe", "/WX"
      cfg.MakeCurrentSettingsDefault
