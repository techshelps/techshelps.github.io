Home Page: Add-ins for the Visual C++ Developer Studio

Developer Studio add-ins are in-process COM components (DLLs) that you write in Visual C++ or Visual Basic. With add-ins, you can automate tasks in Developer Studio by adding commands to perform these tasks, by adding toolbar buttons to carry out these commands, and by responding to Developer Studio events.
