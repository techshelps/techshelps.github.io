Adding and Editing Controls in a Dialog Box

Home Page (Dialog Editor) |  Overview |  How Do I ... Topics

One of the first steps in creating a new dialog box (or making a dialog box template) is to add controls to the dialog box. Controls can be arranged to fit a certain size, shape, or alignment, or they can be moved around to work within the dialog box. You can edit certain properties of the control directly. It’s also easy to delete a control.
