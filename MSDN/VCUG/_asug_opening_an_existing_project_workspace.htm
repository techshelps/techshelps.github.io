Opening an Existing Project Workspace

Overviews |  How Do I... Topics |  NMAKE Reference

To open an existing project workspace

  1. On the File menu, click Open Workspace.

  2. Select the drive and directory containing the project workspace that you want to open.

  3. Select the .dsw file for the project workspace from the File Name list and click OK.

To reopen a recently used project workspace

  • On the File menu, click Recent Workspaces, and then click the name of the recently used workspace.

    -or-

    On the File menu, click the name of the workspace.

Note   You can customize the menu items for recently used files and workspaces. You can specify how many items to list and whether these menu items are displayed on submenus or on the File menu itself. For more information, see Customizing Menu Items for Recently Used Files and Workspaces.

To close a project workspace

  • On the File menu, click Close Workspace.
