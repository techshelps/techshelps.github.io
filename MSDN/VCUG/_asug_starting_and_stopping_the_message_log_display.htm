Starting and Stopping the Message Log Display

Home Page (Spy++) | Overview | How Do I ... Topics

When a Messages view window is active, a Start or Stop Logging choice appears on the Spy++ Messages menu, and the Start or Stop Logging toolbar button becomes active.

To stop the message log display

  • From the Messages menu, choose Stop Logging.

To start the message log display

  • From the Messages menu, choose Start Logging.
