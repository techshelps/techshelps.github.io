To start Spy++

  • From the Tools menu, choose Spy++.

Note   You can run only one copy of Spy++. Attempting to run additional copies of Spy++ will bring the currently running Spy++ to the front.

Spy++ is a read-only program. Using Spy++ does not change program operation, but can slow program execution.

When Spy++ starts, it opens a window titled “Windows 1,” which shows a tree view of all windows and controls in the system. There are also three other views available in Spy++: Messages, Processes, and Threads.
