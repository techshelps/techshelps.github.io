How Do I Find the Current Key Combination For a Command?

Home Page (Text Editor) |  Overview |  How Do I ... Topics |  FAQ |  Reference

To find the current key combination for a command

  1. Choose Keyboard Map from the Help menu.

    This displays the Help Keyboard dialog box.

  2. Choose Edit from the drop-down list, and scroll to the command you want.

    If no key combination is displayed, you can assign one. For more information, see Customizing Keyboard Shortcuts.
