About Keyboard Shortcuts

Home Page (User Interface) |  Overview |  How Do I ... Topics

Keyboard shortcuts offer an alternative method of performing actions for users who prefer the keyboard to the mouse. Some of the commands in Visual C++ are assigned shortcut keys by default. Other commands do not have any default shortcut key assigned to them.

You can also change the existing keyboard shortcuts or create new keyboard shortcuts for commands that do not have keyboard shortcuts by default.
