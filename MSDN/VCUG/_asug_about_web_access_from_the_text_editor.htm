About Web Access from the Text Editor

Home Page (Text Editor) |  Overview |  How Do I ... Topics |  FAQ |  Reference

You can open an Internet address from within the Text editor. Place the cursor on the address and right-click to display the shortcut menu. One of the available commands will be Open, with the Internet address next to it. If you click Open, your browser will start, access the Internet address, and display the appropriate page.

Another way to open an Internet address is to type it into the Find tool, then press CTRL+SHIFT+G.
