How Do I ... Topics: Resource Editors

Home Page (Resource Editors) |  Overview

The Resource editors share techniques and interfaces that enable you to create and modify application resources quickly and easily. You can use the Resource editors to create new resources, modify existing resources, copy existing resources, and delete old resources. The Resource editors are functionally consistent for ease of use.
