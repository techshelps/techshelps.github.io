Picking up a Color from an Image to Use Elsewhere

Home Page (Graphics Editor) |  Overview |  How Do I ... Topics

The color-pickup tool makes any color on the image the current foreground color or background color, depending on whether you press the left or the right mouse button. To cancel the color pickup tool, choose another tool.

To pick up a color

  1. From the Graphics toolbar, select the color-pickup tool.

  2. Select the color you want to pick up from the image. After you pick up a color, the Graphics editor reactivates the most recently used tool.

  3. Draw using the left mouse button for the foreground color, or the right mouse button for the background color.
