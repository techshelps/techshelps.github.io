Moving a String from One Resource Script File to Another

Home Page (String Editor) |  Overview |  How Do I ... Topics

To move a string from one resource script file to another

  1. Open the String Editor windows in both resource script files.

  2. Select the string you want to move.

  3. From the Edit menu, choose Cut.

  4. Place the cursor in the target String Editor window.

  5. From the Edit menu, choose Paste.

Note   If the ID or value of the moved string conflicts with an existing ID or value in the destination file, either the ID or the value of the moved string changes. If a string exists with the same ID, the ID of the moved string changes. If a string exists with the same value, the value of the moved string changes.
