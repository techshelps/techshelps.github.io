Examining File Status on Property Pages

Home Page (Source-Code Control) |  Overview |  How Do I ... Topics

Each file in a project has a property page associated with it. The property page includes information about the file, including its current status in the source-code control system if it is under source-code control.

Note If the file is not under source-code control, no status information appears on the property page.

To examine a property page from the FileView pane

Select the file in the FileView pane of the Project Workspace window, and press ALT+ENTER.

– or –

Select the file in the FileView pane of the Project Workspace window, click the right mouse button to display the shortcut menu, and choose Properties.

– or –

Select the file in the FileView pane of the Project Workspace window, and from the View menu, choose Properties.

To examine a property page from a source editor window

Press ALT+ENTER.

– or –

Click the right mouse button to display the shortcut menu and choose Properties.

– or –

From the View menu, choose Properties.
