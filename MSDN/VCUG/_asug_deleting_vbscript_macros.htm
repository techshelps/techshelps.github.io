Deleting VBscript Macros

Home Page (Macros) |  Overview |  How Do I Topics |  FAQ |  Reference

When you no longer need a macro, you can delete it.

To delete a macro

  1. On the Tools menu, click Macro.

  2. In the Macro File box, select the file containing the macro you want to delete. If this file is not in the list, first install the file, then select it in the Macro File box.

  3. In the Macro Name box, select the macro you want to delete.

  4. Click the Edit button.

  5. Delete the Sub block containing the macro's code.
