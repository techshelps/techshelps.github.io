Checking Files In When Closing the Workspace

Home Page (Source-Code Control) |  Overview |  How Do I ... Topics

You can choose to have Visual C++ prompt you to check in files when you close the current workspace.

To check in files when closing the workspace

  1. From the Tools menu, choose Options.

  2. Click the Source Control tab.

  3. Select the Check In Files When Closing The Workspace check box.

  4. Click OK.
