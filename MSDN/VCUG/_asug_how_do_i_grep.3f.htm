Use the Find in Files command.

To grep

  1. From the File menu, choose Find in Files.

  2. In the Find what box, type the string you want to find, or click the button to the right of the Find what box to select from a list of regular expressions.

  3. In the In files/file types box, specify the types of files in which you want to search.

  4. In the In folder box, specify the folder where you want to begin the search. Use the Browse button to the right of the box to display a dialog where you can navigate to the folder you want.

    Tip   You can search recursively through a directory structure by selecting the Look in subfolders check box.

  5. Select any of the other Find options.

  6. Click Find.

Tip   You can direct the output from a Find in Files search to a second output window by selecting the Output to pane 2 check box. This allows you to keep the output from two separate Find in Files searches available.
