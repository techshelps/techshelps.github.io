Saving an Image in a Different Format

Home Page (Graphics Editor) |  Overview |  How Do I ... Topics

When you create a new image using the New command on the File menu, the image is always in a bitmap format. You can, however, save your image as a GIF or JPEG.

To save an image in a different file format

  1. From the File menu, choose Save As.

  2. In the File Name box, type the name you want to give the file and the extension that denotes the file format you want. For example, myfile.gif.

  3. Click Save.
