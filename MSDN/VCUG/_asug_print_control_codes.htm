Print Control Codes

Home Page (Text Editor) |  Overview |  How Do I ... Topics |  FAQ |  Reference

The print control codes are used in the Header and Footer boxes of the Print Setup dialog box. You can use these control codes to add the information listed in the table below to your source code listings.

To print this information Use this code
Filename &f
Page number &p
Current time &t
Current date &d
Left aligned &l
Centered &c
Right aligned &r
