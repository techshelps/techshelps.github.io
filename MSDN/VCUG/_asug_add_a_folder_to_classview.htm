Adding a Folder to ClassView

Home Page (Working with Classes) |  Overview |  How Do I ... Topics

It's easy to organize your project classes simply by adding a new folder to ClassView. The new folder can contain any classes belonging to the project.  The new folder cannot contain non-class files such as text or resource files. To organize other types of files, go to the related view, for example FileView or ResourceView. For more information, see Project Workspace Views.

To add a folder to ClassView

  1. In ClassView, point your cursor to the project to which you want to add a folder.

  2. Click the right mouse button, and from the shortcut menu choose New Folder.

You can nest folders under folders you create, but not under a class.

Moving Classes into a Different Folder

Once you've created a new folder for a project, simply drag any classes you want into the folder to organize your project classes. Note that you cannot move classes from one project into a folder under a different project.
