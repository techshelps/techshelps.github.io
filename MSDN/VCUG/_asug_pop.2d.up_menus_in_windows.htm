Shortcut Menus in Windows

Home Page (User Interface) |  Overview |  How Do I ... Topics

Most windows contain a shortcut menu with commands appropriate for the window in its current state. The shortcut menus provide quick access to the most commonly used commands for the window you are working in. For example, the shortcut menu for an editor window displays the Cut, Copy, and Paste commands while editing but displays the Toggle Breakpoint and QuickWatch commands while debugging.

Right-click in any window to display the shortcut menu.

There is also a shortcut menu associated with the dock along the border of the application window. If you right-click in the dock area or on a toolbar, the menu displays commands to show or hide all of the docking windows, and to customize the toolbars.
