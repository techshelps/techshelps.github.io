Editing VBScript Macros

Home Page (Macros) |  Overview |  How Do I Topics |  FAQ |  Reference

With the macro editor, you can edit an existing macro or add to a macro you just recorded.

To edit a macro

  1. On the Tools menu, click Macro.

  2. In the Macro File box, select the file containing the macro you want to edit. If this file is not in the list, first install the file, then select it in the Macro File box.

  3. In the Macro Name box, select the macro you want to edit.

  4. Click the Edit button.

  5. Make the appropriate changes.

  6. Save the macro.
