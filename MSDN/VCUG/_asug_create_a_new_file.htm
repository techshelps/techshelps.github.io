Creating a New File

Home Page (Text Editor) |  Overview |  How Do I ... Topics |  FAQ |  Reference

The New command creates a new source file. Creating a source file does not affect other open source files. You can create source files of many types, including Visual C++ header or source files, HTML pages, Active Server pages, and VBS (macro) files.

Note   The Text editor automatically inserts a small amount of boilerplate text into new HTML and Active Server pages, as well as VBS files.

To create a new source file

  1. From the File menu, click New.

  2. Select the kind of source file you want to create.

  3. Select or clear the Add to Project check box, if necessary.

  4. Type a name in the File name box.

  5. Type a path in the Location box.

  6. Click OK.
