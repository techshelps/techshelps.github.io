Using Go To

Home Page (Text Editor) |  Overview |  How Do I ... Topics |  FAQ |  Reference

The Go To dialog box allows you to jump quickly to several different items in a file, including bookmarks, definitions, errors or tags, specific lines, or references.

To use the Go To dialog box

  1. From the Edit menu, choose Go To.

  2. In the Go To what box, select the type of item you want.

  3. Enter any additional information required.

  4. Click one of the navigation buttons: Go To, Previous, or Next.

Note   If the Go To what item is undefined, the additional selection criteria box is unavailable. For example, if you have not defined any bookmarks, the Enter Bookmark Name box appears dimmed.
