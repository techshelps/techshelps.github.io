How Can I Pretty-Print My Code?

Home Page (Text Editor) |  Overview |  How Do I ... Topics |  FAQ |  Reference

You can select sections of code to automatically indent (AutoIndent), but the Text editor does not automatically reformat to merge or break up lines of code.

To quickly indent sections of code, first set the Smart Indent settings you want, then select the code you want to format.

  1. Choose Options from the Tools menu, click the Tabs tab, then change the Smart Indent options

  2. Choose Advanced from the Edit menu, and then choose Format Selection.
