Converting an Image From One Format to Another

Home Page (Graphics Editor) |  Overview |  How Do I ... Topics

You can open GIF or JPEG images in the Graphics editor and save them as bitmaps. Also, you can open a bitmap file and save it as a GIF or JPEG.

To convert an image from one format to another

  1. Open the image in the Graphics editor.

  2. From the File menu, choose Save As.

  3. In the File name box, type the file name and the extension that denotes the format you want.

  4. Click Save.
