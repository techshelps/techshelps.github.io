Posted DDE Messages

Overview

The DDE protocol uses the MONMSGSTRUCT structure to post DDE messages. The information displayed for posted DDE messages is similar to the information displayed for sent DDE messages. DDESpy displays the following information from this structure:

  • Task

  • Time

  • Handle of receiving window

  • Transaction type (posted)

  • Message type

  • Handle of sending application

  • Other message-specific information
