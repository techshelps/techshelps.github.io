Specifying File Settings

Home Page (Projects) |  Overviews |  How Do I... Topics |  NMAKE Reference

In general, settings for a project configuration affect all files belonging to that project. However, for each individual file, you can specify settings that are different from, or additional to, the project configuration's settings. You can specify file settings for files within a specific project, or for files from multiple projects. Within the projects in your project workspace, you can select any combination of files.

Note   You can also specify input dependencies on individual files as part of a custom build process. This ensures that custom build steps run in the order you need them to.

To set file options for a project or projects

  1. On the Project menu, click Settings and then select the General tab.

  2. Under the Settings For drop-down control, expand the project or projects that contain the files you wish to specify settings for.

  3. Select the files, and then, on any of the available tabs, select the options that you want to apply to these files.

    The available tabs depend on the files or configurations that you have selected. Only those tabs with settings common to the selections appear. Similarly, within each tab, only the settings common to all the files selected are available.

Note   If you specify settings incompatible with the project type that you chose when you created your project, you may not get the result that you expect.
