Customizing Menu Items for Recently Used Files and Workspaces

Home Page (Customizing) |  Overview |  How Do I ... Topics

Files and project workspaces that you have used recently can be opened by selecting the name of the file or workspace from a menu. You can choose whether the recently used files and workspaces are listed on the File menu or on submenus of the File menu. You can also customize the number of files and workspaces listed.

To customize the menu items for recently used files and workspaces

  1. On the Tools menu, click Options, then click the Workspace tab.

  2. To list recently used files and workspaces on submenus of the File menu, select the Show Recently Used Items on Submenus check box.

     – or – 

    To list recently used files and workspaces on the File menu itself, clear the check box.

  3. To change the number of recently used files listed, type the number you want in the Recent File List Contains box.

  4. To change the number of recently used workspaces listed, type the number you want in the Recent Workspace List Contains box.
