How Do I ... Topics: Macros

Home Page (Macros) |  Overview |  FAQ |  Reference

You can automate your work in Visual C++ by recording VBScript macros that perform routine tasks. You can quickly record and playback a macro, similar to the way previous versions of Visual C++ provided a keystroke recording feature. Or, you can save your macro with a unique name so that you can edit the resulting VBScript macro file and rerun the macro whenever you want.
