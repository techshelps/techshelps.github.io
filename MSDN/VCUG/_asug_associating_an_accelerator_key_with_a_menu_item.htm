About Associating an Accelerator Key with a Menu Item

Home Page (Accelerator Editor) |  Overview |  How Do I ... Topics

Many times, you want a menu item and a keyboard combination to issue the same program command. You do this by assigning the same resource identifier to the menu item and to an entry in your application’s accelerator table. You then edit the menu item’s caption to show the name of the accelerator. For more information on menu items and accelerator keys, see Associating a Menu Item with an Accelerator Key.
