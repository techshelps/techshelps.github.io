Navigating in Files

Home Page (Text Editor) |  Overview |  How Do I ... Topics |  FAQ |  Reference

The Text editor provides a variety of methods with which to move around in a source file. In addition to the regular mouse movement and page controls, the Text editor includes an assortment of commands that enable you to move to almost any location in a file. Furthermore, the editor includes several advanced navigation features, such as virtual space, matching group delimiters, and matching conditional statements.
