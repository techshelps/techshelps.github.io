Assigning a WizardBar Command to a Toolbar Button

Home Page (Working with Classes) |  Overview |  How Do I ... Topics

You can make WizardBar commands available from any toolbar, to save screen real estate or to create a more customized environment.

To assign a WizardBar command to a toolbar button

  1. Ensure that the toolbar where you want the WizardBar command to appear is visible.

  2. From the Tools menu, choose Customize, and if necessary click the Commands tab to select it.

  3. In the Category list, select WizardBar.

    The Commands list displays all available WizardBar commands.

  4. Select the WizardBar command you want, and drag it onto the toolbar.

In the Button Appearance dialog box, select the image you want for the toolbar button, and click OK.
