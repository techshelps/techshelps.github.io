About Graphical Resources

Home Page (Graphics Editor) |  Overview |  How Do I ... Topics

Graphical resources are the images you define for your application. In addition to creating new graphical resources, you can import existing images and add them to your project, and you can open files that are not part of a project for stand-alone editing.

What do you want to do?

Draw or erase freehand

Draw lines or closed figures

Select an area of an image

Edit selected parts of an image

Flip an image

Resize an image

Create a custom brush

Set image properties

Save an image in a different format

Convert an image from one format to another
