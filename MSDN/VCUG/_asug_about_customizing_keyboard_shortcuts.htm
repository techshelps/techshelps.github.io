About Customizing Keyboard Shortcuts

Home Page (Customizing) |  Overview |  How Do I ... Topics

Keyboard shortcuts offer an alternative method of performing actions for users who prefer the keyboard to the mouse. In Visual C++, you can:

  • Display a list of current keyboard shortcuts.

  • Delete existing assignments for shortcut keys.

  • Replace default shortcut keys with different ones.

  • Assign shortcut keys to commands that have none by default.

  • Assign multiple shortcut keys to a command.

Using these assignments, you can choose your own set of shortcut keys — ones that are familiar and natural for you to use, or that have some easily remembered value.
