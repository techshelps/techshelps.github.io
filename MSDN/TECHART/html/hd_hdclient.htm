HDClient

HDClient is an n-tier design, client-side COM component of the HelpDesk sample application. The client applications use this component to communicate with HDServer, which communicates directly to the database executive, DBExec, as shown in the object model.

The HDClient component contains the following classes:

  • CAdmin

  • CLocation

  • CPriority

  • CRequest

  • CReqDetail

  • CReqDetType

  • CReqStatus

  • CSkill

  • CStdResp

  • CTech

  • CUser
