Overview of CDO Rendering

This section is an introduction to the Microsoft® Collaboration Data Objects (CDO) Rendering Library version 1.2. It describes rendering and how it is accomplished using the objects in the library. It also provides a short description of Automation and a conceptual overview of the CDO Rendering Library.

The CDO Rendering Library is associated with the CDO Library, and the two are interdependent. CDO Rendering entails displaying CDO objects and collections over the World Wide Web. For more information on CDO, see the CDO Library Overview.
