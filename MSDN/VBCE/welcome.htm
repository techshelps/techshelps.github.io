Welcome to Windows CE!


Windows CE Toolkit for Visual Basic 5.0 contains all the tools you need to develop great applications for platforms that run Windows CE.


The Windows CE emulation environment and remote tools, built into the familiar Visual Basic development environment, can help you port existing applications or develop entirely new ones for a wide range of devices that run Windows CE.


The Windows CE documentation is now HTML-based. Start reading now by clicking on one of the categories you see at the right. Or use the InfoViewer Contents pane for a hierarchical view of the Table of Contents, much as it would appear in a printed book.


For the latest information about developing for Windows CE, visit the Microsoft Windows CE World Wide Web site, http://www.microsoft.com/ windowsce/developer.
