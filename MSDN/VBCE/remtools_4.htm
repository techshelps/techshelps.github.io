Starting Windows CE Control Manager

    To start Windows CE Control Manager

  • In the Visual Basic IDE, select the Windows CE menu and click Control Manager

    – Or –

  • Click the Start button, point to Programs, click Visual Basic 5.0, click Windows CE, and then click Control Manager.

    – Or –

  • At the Windows NT command prompt, type "C:\Program Files\DevStudio\vb\vbce\cectlmgr.exe" and press Enter.
