Creating an Installation Program

Use the Windows CE Application Install Wizard to create an installation program for your application. The Install Wizard creates a setup program (setup.exe), a directory containing a readme file, and several other files and directories.

    To run the Windows CE Application Install Wizard

  • From the Windows CE menu, select Application Install Wizard.

    To create an installation program for your Windows CE application

  • The Application Install Wizard leads you through the process to create an installation program. For step-by-step instructions on using the Application Wizard, see Walking Through the Application Wizard.

Note Using Notepad, you can customize the installation program by adding or changing code in the .INF file.

For more information about CAB wizard and .INF files, refer to the Microsoft Windows CE SDK Programmer's Guide.
