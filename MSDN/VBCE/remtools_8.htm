Adding and Removing Existing Controls

    To install or uninstall a control listed in the control manager box

  1. Select the control in the box.
  2. Under Emulation or Device, select Install. The control will be installed

    – Or –

    Under Emulation or Device, select Uninstall. The control will be uninstalled.

Note This menu item is different from the Install Control menu item, which installs a new control from a selected directory .
