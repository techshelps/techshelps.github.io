Error

Occurs whenever an error accurs in background processing (for example, failed to connect, or failed to send or receive in the background).

Syntax

object.Error(long lNumber, BSTR bstrDescription)

The parts of the Error event syntax are described in the following table.

Part Description
object An object expression that evaluates to an object.
lNumber Indicates error code from WSAGetLastError().
bstrDescription Contains an error description.
