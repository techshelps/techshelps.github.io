File System Controls

The Windows CE Toolkit for Visual Basic 5.0 File System controls work together to provide basic file input and output functionality, as well as the ability to manipulate files and directories.

The FileSystem control provides access to the File object through the File control and to the FileSystem object through the FileSystem control.
