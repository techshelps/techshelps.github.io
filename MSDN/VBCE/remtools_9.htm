Windows CE Spy

When you start the Windows CE Spy utility, the remote device downloads a Windows list to the desktop PC. The Windows list displays the windows that are open on the remote device. When you select a window and click Messages from the File menu, Windows CE Spy opens a secondary window that contains information about the messages in that window's message queue.

Use Windows CE Spy to:

  • Display a tree hierarchy that represents the relationships between system objects.
  • View messages that are being sent or posted to windows on the remote device.
