What Controls, Properties, Events, and Methods are Unsupported

The following intrinsic controls located in the Toolbox are not supported by the Windows CE Toolkit for Visual Basic 5.0.

  • Directory List Box
  • Drive List Box
  • File List Box
  • Image
  • Picture Box
  • Data
  • OLE

Note If you select a control that is not supported in Windows CE, an error message will be displayed.
