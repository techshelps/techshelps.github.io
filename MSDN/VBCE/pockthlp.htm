Pocket Help Files for Windows CE

Microsoft® Windows® CE uses Pocket Help to display Help files for Windows® CE-based programs. Pocket Help files are authored in Hypertext Markup Language (HTML). The Help engine supports jumps, basic formatting, tables, and bitmaps. This chapter describes how to create Pocket Help files and what tools you will need.

Keep in mind that space on the device is extremely limited. You may want to restrict the online Help to reminders for common tasks and key commands, and place more detailed information in another medium. Windows CE compression reduces file size by 30 to 40 percent.
