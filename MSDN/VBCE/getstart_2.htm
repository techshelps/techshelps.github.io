Installing the Windows CE Toolkit for Visual Basic 5.0

The Windows CE Toolkit for Visual Basic 5.0 is an add-on that can only be installed after the installation of Visual Basic 5.0.

    To install Windows CE Toolkit for Visual Basic 5.0

  1. Verify that Visual Basic 5.0 is already installed on your system.
  2. Run the Setup.exe program for the Windows CE Toolkit for Visual Basic 5.0. It will be installed in a subfolder, named vbce, located within the folder for Visual Basic 5.0. See the readme file for any additional information.

When you run the installation program for the Windows CE Toolkit for Visual Basic 5.0, the Windows CE Platform SDK, including the emulation environment, will also be installed.

Setup will also ask if you wish to install Windows CE Services. Choose Yes, if CE Services is not already installed on your system. This will allow you to connect to a remote device.
