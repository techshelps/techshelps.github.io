Downloading Runtime Files

When running a project in emulation or on a device, the runtime files are automatically downloaded.

The following files will be downloaded to the connection:

  • pvb.exe
  • pdm.dll
  • vbscript.dll
  • vbsen.dll
  • pvbrt.dll
  • regsvr.exe
  • vbrmon.dll

    To download runtime files to a device

  1. Make sure your PC is connected to the device before you download the runtime files.
  2. Make sure your target is Remote Device in Project Properties.
  3. On the Windows CE menu, choose Download Runtime Files.

    To download runtime files to the emulator

  1. Make sure your target is Emulator in Project Properties.
  2. On the Windows CE menu, choose Download Runtime Files. The emulator will be automatically opened and the runtime files will be downloaded to the emulator.
