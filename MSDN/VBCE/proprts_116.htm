HWnd

Returns a handle to a form or control.

Syntax

object.hWnd

The object placeholder represents an object expression that evaluates to an object.

Remarks

The Microsoft Windows CE operating environment identifies each form and control in an application by assigning it a handle, or hWnd. Many Windows operating environment functions require the hWnd of the active window as an argument.

Note Because the value of this property can change while a program is running, never store the hWnd value in a variable.
