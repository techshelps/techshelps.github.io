Windows CE Heap Walker

The Windows CE Heap Walker (ceheapwk.exe) enables you to view detailed information about heap IDs and flags for processes that are running on the remote device. You can use the Windows CE Heap Walker to determine if an application is releasing all of the memory it is allocating. Windows CE Heap Walker is a useful tool to determine if an application has memory leaks.
