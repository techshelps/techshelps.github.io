Installing Controls for a Device

    To install a new device control not listed in the Control Manager box

  1. On the Windows CE menu, click Control Manager.
  2. Click on Device, and then click Install Controls. An Open dialog box displays the location of the controls.
  3. Find the directory where the controls reside. The default directory for the Windows CE controls is:
  4. \Program files\DevStudio\vb\vbce\bin\cpuname\controls.
  5. Click on the control you want to install.
