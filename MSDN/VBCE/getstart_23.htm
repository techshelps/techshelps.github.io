Uninstalling the Windows CE Toolkit for Visual Basic 5.0

If you need to restore the original Visual Basic installation, you can uninstall the Toolkit.

    To uninstall Windows CE Platform SDK

Before you uninstall, make sure you do not have other applications open, such as Windows CE Toolkit for Visual C++, that are dependent on the Windows CE Platform SDK.

  1. On the Install/Uninstall tab of Add/Remove Programs, select Windows CE Platform SDK.
  2. Click Add/Remove, and follow the on-screen instructions.

    To uninstall Windows CE Toolkit for Visual Basic 5.0

  1. On the Install/Uninstall tab of Add/Remove Programs, select Windows CE Toolkit for Visual Basic 5.0.
  2. Click Add/Remove, and follow the on-screen instructions.

    To uninstall Windows CE Services

  1. On the Install/Uninstall tab of Add/Remove Programs, select Windows CE Services.
  2. Click Add/Remove, and follow the on-screen instructions.
