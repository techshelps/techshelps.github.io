Len

Description

Returns the number of characters in a string or the number of bytes required to store a variable.

Syntax

Len(string | varname)

The Lens yntax has these parts:

Part Description
string Any valid string expression. If string contains Null, Null is returned.
varname Any valid variable name. If varname contains Null, Null is returned.

Note The LenB s used with byte data contained in a string. Instead of returning the number of characters in a string, LenB returns the number of bytes used to represent that string.
