Installing Controls Using the Windows CE Control Manager

Several new controls are included with the Windows CE Toolkit for Visual Basic 5.0. These controls must be installed for emulation or for use on the remote device by using the Control Manager. When you start the Windows CE Control Manager utility, the Control Manager downloads a list of installed controls. When a control is available for use on an Visual Basic form, it is shown in the Toolbox. To add an installed control to the Toolbox, click Project on the Visual Basic menu and choose Components, then check the box next to the name of the component you wish to add. When you are finished, update your system to synchronize, so that both the emulation and the device have the latest updated controls. Controls from third-party vendors should be installed according to the instructions provided with the control. For step-by-step instructions on installing a control, see the following topics:
