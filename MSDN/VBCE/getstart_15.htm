Building a Project

When you build a Windows CE project, the toolkit compiles your code and creates a Visual Basic (.pvb) file in the location specified in the Local Path box of the Project Properties dialog.

    To build a project

  • Choose the Make Project button on the toolbar.

    – Or –

  • Choose Press F5, to run the application.

    – Or –

  • On the Main menu, choose Run.

    – Or –

  • On the File menu, choose Make Project1 (where project1 is the name of your project).

Note The Make Project menu item on the File menu builds the .pvb file in the location pointed to in the Project Properties dialog box.

Windows CE Toolkit for Visual Basic 5.0 does not check errors during design. Instead, an error message will be displayed when you run your application.
