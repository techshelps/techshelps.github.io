Starting the Windows CE Registry Editor

    To start the Windows CE Registry Editor

  • In the Visual Basic IDE, select the Windows CE menu and click Registry Editor

    – Or –

  • Click the Start button, point to Programs, click Visual Basic 5.0, click Windows CE, and then click Registry Editor.

    – Or –

  • At the Windows NT command prompt, type "C:\Program Files\DevStudio\vb\vbce\ceregedit.exe" and press Enter.

When you start the Windows CE Registry Editor, the Windows CE Desktop Emulation Environment window appears in the upper-left corner of your monitor, then the Windows CE Registry Editor window appears.
