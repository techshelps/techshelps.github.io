Starting Windows CE Zoom

    To start Windows CE Zoom

  • In the Visual Basic IDE, select the Windows CE menu and click Zoom.

    – Or –

  • Click the Start button, point to Programs, click Visual Basic 5.0, click Windows CE, and then click Zoom.

    – Or –

  • At the Windows NT command prompt, type "C:\Program Files\DevStudio\vb\vbce\cezoom.exe" and press Enter.

The Windows CE Zoom window appears, with a dialog box that contains a progress bar. This dialog box displays "Initializing communication with device" and then "Connecting to device" as your desktop computer and the remote device establish a communications link. The desktop computer then downloads a bitmapped screen capture from the remote device.
