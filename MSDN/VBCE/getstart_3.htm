Supported Features

The language chosen for the Windows CE Toolkit for Visual Basic 5.0 is a subset of Visual Basic 5.0. This section lists the language features supported in the Windows CE Toolkit for Visual Basic 5.0. For information on how features supported in this toolkit differ from those supported in Visual Basic, see the following topics.
