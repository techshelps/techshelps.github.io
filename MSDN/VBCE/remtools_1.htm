Windows CE Application Install Wizard

Use the Windows CE Application Install Wizard to create an installation program for your application. The Install Wizard creates an installation program (setup.exe), a directory containing a readme file, CAB and INF files.

For more information about CAB Wizard and .INF files, refer to Microsoft Windows CE SDK Programmer's Guide.

    To run the Windows CE Application Install Wizard

  • On the Windows CE menu, select Application Install Wizard.

    The Application Install Wizard guides you through the steps necessary to create an installation program. For step-by-step instructions on completing the wizard, see Walking Through the Application Wizard.
