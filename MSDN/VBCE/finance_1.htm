Remarks

    To create a Finance Object

  • The Finance object is a non-visual object. It cannot be dropped onto a form. To use the Finance object, use the CreateObject() statement to create the object.
Dim Financial
Set Financial = CreateObject("pVBFinFunc.pVBFinFunc")
Text1.Text = Financial.Pmt(0.0083, 48, 12000)
