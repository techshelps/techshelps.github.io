Using the View Menu

The View menu on the debugger displays the following commands:

  • Toolbar

    Displays the Toolbar.

  • Status Bar

    Displays the Status Bar.

  • Quick Watch

    Displays the Quick Watch dialog box with the current value of the selected expression.

    Use this command to check the current value of a variable, property, or other expression for which you have not defined a watch expression. Select the expression from either the Forms window or the Watch window, and then choose the Quick Watch command. To add a watch expression based on the expression in the Quick Watch dialog box, choose the Add button.
