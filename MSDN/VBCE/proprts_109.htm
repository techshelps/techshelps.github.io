HelpCommand

Returns or sets the type of online Help requested.

Syntax

object.HelpCommand [ = value ]

The parts of the HelpCommand property syntax are described in the following table.

Part Description
object An object expression that evaluates to an object.
value A value or constant that specifies the type of Help, as described in Settings.

Settings

The settings for value are described in the following table.

Value Description
&H1 Displays Help for a particular context. When using this setting, you must also specify a context using the HelpContext property.

Data Type

Integer
