Hardware and System Requirements

In order to run the Windows CE Toolkit for Visual Basic 5.0 on a remote device, you need the following items.

  • PC with a 486/66 MHz or higher processor; Pentium or higher processor recommended.
  • Microsoft Windows NT® Workstation operating system version 4.0.
  • Microsoft Visual Basic, Professional or Enterprise edition, version 5.0.
  • 24 MB of RAM; 32 MB recommended.
  • Hard-disk space required:
    • Minimum installation: 50 MB.
    • Complete installation: 80 MB
  • CD-ROM drive.
  • VGA or higher-resolution monitor; Super VGA recommended.
  • Microsoft Mouse or compatible pointing device.

Note You need Windows NT to run emulation, since Windows CE is Unicode only and NT provides the necessary Unicode environment.

If you need to install Internet Explorer when you bring up Windows CE Books Online, you can install it by running the msie302.exe in the /msie302 directory.
