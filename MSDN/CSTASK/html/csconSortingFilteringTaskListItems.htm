Sorting and Showing Task List Items

Items in the Task List can be sorted or arranged to create custom views and help you focus on specific tasks. You can:

  • Sort by Priority, Category, Checked, Description, File, or Line.

  • Display a subset of tasks by filtering by Category, Current File, or Checked status.

Each time you arrange or sort the Task List, the current state is saved.

To return to the previous state

  • Select Previous View from the shortcut menu.

    You can only go back one previous view.

To sort tasks

  • Click the task heading.

    Items are sorted in ascending order. As new task items are added to the Task List, they are arranged according to the current sorting scheme.

To view your tasks by a particular category

  1. Right-click the Task List window.

  2. Choose Show Tasks and then select a category, such as User.

    The category types vary with the development environment you are using to build a project.

  3. Select the category you want to view.

    The Task List displays only those tasks for the category you selected.

To view all of your tasks

  1. Right-click the Task List window.

  2. Choose Show Tasks.

  3. Choose All.
