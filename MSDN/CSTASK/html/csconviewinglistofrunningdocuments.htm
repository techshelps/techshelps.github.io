Viewing a List of Running Documents

The Running Documents window displays a list of HTML documents that are active in any attached process capable of displaying HTML, such as Internet Explorer. This window allows you to debug scripts or components embedded in HTML pages using the script debugger.

You can attach to any script or applet currently running, which can include scripts and components in pages that are not part of your current project. You can debug scripts running locally on your computer or attach to processes running on remote computers, such as an .asp page running on a server.

If you aren't attached to any processes, the Running Documents window will be empty.

To view the Running Documents window

  • Choose Running Documents from the Debug Windows submenu, available from the View menu.
