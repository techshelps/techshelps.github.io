Checking and Deleting Task List Items

You can only check user-defined or named shortcut task list items. You can delete user-defined or named shortcut items directly; other types of task list items are deleted by the application that generated them. For example, you can delete a Comment task from the Task List by deleting it in the source code. Still other types of task list items such as Compiler tasks can be checked but not deleted.

To mark a user-defined or named shortcut task as completed

  • Click the Checked column in the Task List window.

    Checking the item as completed puts a line through the item.

To delete a user-defined or named shortcut

  • Select the entire row in the Task List and press DEL.

Caution   You cannot undo deletion of a task.

To choose multiple contiguous tasks

  • Select the entire row of the first task, press SHIFT, and select the last contiguous task you want.

To choose multiple discontiguous tasks

  • Select the entire row of the first task, press Ctrl, and select the other tasks you want.
