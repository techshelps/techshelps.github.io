Adding and Deleting Named Shortcuts in the Task List

Depending on the development language you are using, you may be able to add named shortcuts to the Task List from within the application's editor.

Named shortcuts make it easier to find your code.

To add a shortcut to the Task List

  1. Right-click the line of code in the Text Editor window.

  2. Select Toggle Shortcut.

    The text on the selected line of code becomes the name of the task and is added to the Task List.

  3. If you want, you can edit the description field of the new shortcut.

    Editing the description will not change the actual line of code.

To go to the shortcut from the Task List

  • Double-click the task in the Task List.

To delete a shortcut

  1. Right-click the line of code in the editor.

  2. Choose Toggle Shortcut.

    You can also delete the shortcut directly in the Task List.
