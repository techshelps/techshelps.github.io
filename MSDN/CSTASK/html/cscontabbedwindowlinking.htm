Linking Windows

The development environment allows you to drag tool windows onto each other to create linked windows. Tool windows are windows defined by each development language (for example, the Toolbox). There are two types of tool windows:

  • Linked windows appear as one window with the title bar separating each window from the other.

  • Tabbed linked windows appear as one window with a set of tabs that take you to the other linked windows.

You can create and separate linked or tabbed linked windows and can reorder the tabs of tabbed linked windows.

To create a linked window

  • Click the title bar of the tool window you want to link, and then drag it to one side of the window to which you want to link.

    Windows can be linked horizontally or vertically or to the right or left side of the destination window.

To separate linked windows

  • Click the title bar and drag the window out of the linked window

    -or-

  • Double-click the title bar.

To create tabbed linked windows

  • Click the title bar of the window you want to link, and then drag it to the title bar of the window with which you want to create a tabbed link.

    A tab outline appears when you drag the window over the appropriate place.

To separate tabbed windows

  • Click the tab of the window you want to separate, and then drag it out of the tabbed linked set

To reorder tabs

  • Drag the tab you want to move to the new location.

    The active tab locks to the desired location, and the other tabs move to the right or left of it.
