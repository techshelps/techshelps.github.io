Adding, Deleting, and Renaming Toolbox Tabs

By default, the Toolbox contains two tabs: the General tab and the HTML tab. When you add a Toolbox item, it is displayed on the current tab. You can, however, customize your own tabs in the Toolbox so that you can organize your tools for the tasks you want to perform.

To add a new tab to the Toolbox

  1. Right-click the Toolbox.

  2. Select Add Tab.

    A blank tab is added to the Toolbox.

  3. At the blinking insertion point at the bottom of the Toolbox, type a name for the tab and press ENTER.

When you create a new tab, a pointer tool, , is automatically added as the first item.

To delete a tab that you've added to the Toolbox

  1. Right-click the tab in the Toolbox that you want to delete.

  2. Select Delete Tab.

To rename a tab that you've added to the Toolbox

  1. Right-click the tab in the Toolbox that you want to rename.

  2. Select Rename Tab.

  3. Type the new name for the tab.

To reorder tabs in the Toolbox

  • Drag them up or down to a new position.
