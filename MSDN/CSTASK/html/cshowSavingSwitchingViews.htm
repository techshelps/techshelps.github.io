Creating Custom Workspaces

You can create custom workspaces called window layouts.

To save a window layout

  1. Arrange the windows in your workspace.

  2. Select Define Window Layout from the View menu.

  3. In the Define Window Layout dialog box, type the name of the new window layout.

    The window layout is saved when you change views so you can return to it.

To open a saved window layout

  • Choose the Define Window Layout command on the View menu.

To apply a view

  1. Select Define Window Layout from the View menu.

  2. In the Define Window Layout dialog box, select the view you want to apply from the Views list.

  3. Click the Apply button.

Your window layout also gets saved each time you switch among design, debug, and Full Screen modes. For example, you set up your design workspace, save the window layout, and then change to debug mode. While in debug mode, you display the Task List and then switch back to your defined window layout. When you return to debug mode, the Task List is still displayed because the development environment saved your window layout. The default design and default debug window layouts are always available in the Define Window Layout dialog box or the Load/Save Window UI box on the Window UI toolbar.
