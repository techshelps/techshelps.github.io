IntelliMouse Support

With the Microsoft® IntelliMouse™, you can take advantage of additional navigational features in the development environment. The IntelliMouse is a pointing device with a wheel control between the two buttons. The wheel control is both a wheel and a button that makes common navigation functions easy to perform with the mouse.

You can:

  • Rotate the wheel to scroll through a window more quickly than using the scroll bars.

  • Press the wheel and move the mouse to smoothly scroll vertically. This is called panning.

  • Click the wheel button to place it in autoscroll mode and navigate without pressing the mouse button.

You can use the IntelliMouse wheel in these areas:

  • Text Editor window

  • Immediate window

  • Locals window

  • Object Browser

  • Project Explorer

  • Properties window

  • Watch window

  • Web documents
