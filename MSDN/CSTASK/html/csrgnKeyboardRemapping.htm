Keyboard Remapping

The keyboard mapping scheme defines the keystroke combinations that can be used while working in the integrated development environment. You can choose from several predefined schemes or define your own.

Predefined keyboard mapping schemes include schemes that match Microsoft® Developer Studio® 5 and 6, Microsoft Visual Basic® 5 and 6, and Microsoft Visual C++® 2.

To choose a predefined keyboard mapping scheme

  1. Select Options from the Tools menu.

  2. In the Options dialog, select the General, Keyboard node.

  3. Choose a scheme from the Keyboard Mapping Scheme dropdown.

To define a new keyboard mapping scheme

  1. Select Options from the Tools menu.

  2. In the Options dialog, select the General, Keyboard node.

  3. Modify the keyboard mapping scheme as desired, then enter a name for the new scheme in the Keyboard Mapping Scheme dropdown listbox.

  4. Click the Save button.
