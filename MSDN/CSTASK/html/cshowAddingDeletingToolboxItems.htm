Adding and Deleting Toolbox Items

To add new Toolbox items

  1. In the Toolbox, click the tab to which you want to add the item.

  2. From the Tools menu, choose Customize Toolbox.

    Although you can drag items into your Toolbox, using the Customize Toolbox dialog box allows you to browse the registry and to locate controls and applets.

    Note   You can also select text or HTML elements in the Editor and drag them to the Toolbox to create HTML Fragments.

  3. Click the tab that corresponds to the type of item you want to add.

    Items with a check mark are currently in your Toolbox but may not appear on the current tab.

  4. Select the check box to the left of the each item you want to add.

    If the item you want does not appear in the list, choose Browse to locate it.

  5. Click OK to add the item or items.

    They will appear on the appropriate tab in the Toolbox.

To delete an item from the Toolbox

  1. From the Tools menu, choose Customize Toolbox.

  2. Click the tab that lists the item you want to delete.

  3. Clear the check box to the left of the item.

  4. Click OK to delete the item.

    The item will disappear from the appropriate tab in the Toolbox.
