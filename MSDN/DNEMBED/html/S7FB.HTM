Chapter 4: Building Applications

  • To build an Embedded SQL application for the Windows NT or the Windows 95 operating systems, you must compile the application on a computer that is running Windows NT or Windows 95.
  • To build an Embedded SQL application for the 16-bit Windows operating system, you must compile the application on a computer that is running MS-DOS or 16-bit Windows.
  • To build an Embedded SQL application for the MS-DOS operating system, you must compile the application on a computer that is running MS-DOS version 6.22 or later.
