About This Manual

The information in this manual describes Embedded SQL for C and provides details about building applications for the Windows® operating system (16-bit), the Windows NT™ operating system, the Windows 95 operating system, and the MS-DOS® operating system. This manual also includes a directory of Embedded SQL statements.

This manual is written for programmers who are using the C programming language to develop applications for Microsoft SQL Server and who are familiar with the Transact-SQL® and C languages. You should also be familiar with the Windows NT, Windows 95, 16-bit Windows, or MS-DOS operating system.
