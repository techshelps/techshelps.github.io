Appendix A: Advanced Programming

Most of the features that are used in writing Embedded SQL code are discussed in Chapter 2, "Embedded SQL Programming." Appendix A provides supplemental information for developing Embedded SQL applications and discusses:

  • Datatype mappings from Transact-SQL to C, and vice versa
  • The SQLDA data structure for data input and output
  • The contents of the SQLCA data structure
  • The EXEC statement and how to use it to selectively bypass the creation of an access plan
