Prepare the Test Site and Equipment

In the space allotted for the test lab, the installation team should perform the following tasks:

1. Make sure the test computer meets your organization's standards and the Windows NT Workstation 4.0 minimum standards for operation: a 486 or higher processor with 12 MB RAM and 90 MB of free disk space. If not, perform the hardware upgrades now.

2. Defragment the hard disk and scan it for viruses.

3. Back up and verify key data and configuration files, such as INI, Autoexec.bat, and Config.sys files. Also back up the Windows and DOS directories, and all files in the root directory. Make a system startup disk containing Command.com, Sys.com, and Fdisk.exe.
