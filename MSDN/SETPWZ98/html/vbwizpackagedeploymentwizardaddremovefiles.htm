Package and Deployment Wizard — Included Files

Displays a list of the files that will be included in the package and allows you to add additional files to the package or to remove unwanted files.

Options

Included Files List

Displays the name and current location of each file that will be included in the package. The main project executable file is initially selected.

To remove files, clear the checkbox to the left of the file name. If you remove a file with dependencies, all of its dependency files are also removed. (Removed files remain listed. If you have removed a file mistakenly, check its check box.)

When you move the mouse cursor over an entry in this list, a tooltip appears, listing the files for which the given file is needed or stating that the given file was manually added to the list.

Add

Displays the Add File dialog box where you can locate and choose a file to add to your package. The Add File dialog box is a standard Windows dialog box. When you add a file, the wizard searches for dependencies of the file and adds them as well.

Help

Displays the help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the packaging portion of the Package and Deployment Wizard.

Back

Moves you to the previous step.

Next

Moves you to the next step.

Finish

Creates a package using the settings you selected and displays a report at the end of the process. Available only when you have provided enough information for the Package and Deployment Wizard to successfully create a package.

If you chose a pre-existing script, the Finish button is available because the Package and Deployment Wizard assumes that the script contains all of the information needed to create the package.
