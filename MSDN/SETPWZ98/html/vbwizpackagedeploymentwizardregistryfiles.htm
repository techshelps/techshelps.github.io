Package and Deployment Wizard — Registry Information

Lists the registry files in your package. You can choose how this information should be conveyed to the user's computer during installation. The wizard can simply copy the registration files to the user's machine, without performing the necessary registration, or it can extract the information from the registry files without copying the files to the user's computer and store it in the registry upon installation. The wizard can also do both.

This screen only appears if your package includes one or more .reg or .vbl files.

Options

Registration Files List

Displays the name and current location of each registry file, and provides columns in which you can indicate whether to copy the file to the user's computer, update the registry, or both. Make your selections by selecting a file in the table and choosing Yes or No from the Copy Files and Update Registry columns.

When you move the mouse cursor over an entry in this list, a tooltip appears, listing the files for which the given file is needed or stating that the given file was manually added to the list.

Help

Displays the help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the packaging portion of the Package and Deployment Wizard.

Back

Moves you to the previous step.

Next

Moves you to the next step.

Finish

Creates a package using the settings you selected and displays a report at the end of the process. Available only when you have provided enough information for the Package and Deployment Wizard to successfully create a package.

If you chose a pre-existing script, the Finish button is available because the Package and Deployment Wizard assumes that the script contains all of the information needed to create the package.
