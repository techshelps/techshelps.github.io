Package and Deployment Wizard — Package to Deploy

Allows you to choose the package you want to deploy.

This screen will not be displayed if there is only one package that can be deployed for the current project.

Options

Package

Displays a list of packages from which you can choose the one you want to deploy. Each package is identified by the name of the corresponding packaging script. Only packages for the current project are shown.

Help

Displays the help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the deployment portion of the Package and Deployment Wizard.

Back

Moves you to the previous step. Available only if there are previously saved deployment scripts for the current project.

Next

Moves you to the next step.

Finish

Deploys your package using the settings you selected and displays a report at the end of the process. Available only when you have provided enough information for the Package and Deployment Wizard to successfully deploy the package.

If you chose a pre-existing script, the Finish button is available because the Package and Deployment Wizard assumes that the script contains all of the information needed to deploy the package.

Note   If you chose to deploy a different package from that specified in the script, the Finish button is unavailable because the new package will require deployment information not stored in the script.
