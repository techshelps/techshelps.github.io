Missing Files Dialog Box (Package and Deployment Wizard)

Lists files that are needed for your package but which the Package and Deployment Wizard was unable to locate. Such files may be needed by your project or needed by another file in the package.

Options

Missing Files List

Lists the files that the Package and Deployment Wizard cannot find.

When you move the mouse cursor over an entry in this list, a tooltip appears, listing the files for which the given file is needed.

Browse

Displays the Locate dialog box where you can search for your file.

OK

Allows you to proceed without the files that are missing.

Cancel

Closes the dialog box and returns you to the appropriate screen. Choose this option if you want to exit the wizard and investigate the problem, then re-run the packaging process for this project.

Help

Displays the help topic for this step. You can also press F1 for help.
