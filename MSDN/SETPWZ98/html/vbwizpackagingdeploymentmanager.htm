Manage Scripts Dialog Box (Package and Deployment Wizard)

Allows you to rename, duplicate, or delete selected package and deployment scripts in the current project.

Warning   If you delete a packaging script, the deployment portion of the Package and Deployment Wizard will no longer recognize the package created from that script as one that it can deploy. You will have to recreate the package in order to be able to deploy it. Delete only those scripts you are certain you no longer need.

Options

Package Scripts Tab

Displays a list of the package scripts for the current project.

Deployment Scripts Tab

Displays a list of the deployment scripts for the current project.

Rename

Displays the Rename dialog box where you can give a new name to the selected script.

Duplicate

Creates a new script identical to the selected script and displays the Duplicate dialog box where you can give a name to the new script.

Delete

Allows you to delete the selected script.

Warning   When you click this button, you will be prompted to confirm the deletion. If you confirm, the script will be deleted immediately. This deletion cannot be undone.

Close

Closes the dialog box.
