Package and Deployment Wizard — Deployment Method

Allows you to choose the way you want to deploy the selected package.

Options

Deployment Type

Lists the types of deployments supported for the given package:

Folder — Deploys the package to the folder you indicate on a local or network drive.

Web Publishing — Deploys the package to a Web server via Web Publishing technology.

Floppy — Deploys the package to floppy disks. This option is only available when you have packaged your application into multiple .cab files.

Description

Describes the selected option.

Help

Displays the help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the deployment portion of the Package and Deployment Wizard.

Back

Moves you to the previous step. Available only if there is more than one package which can be deployed for the current project.

Next

Moves you to the next step.

Finish

Deploys your package using the settings you selected and displays a report at the end of the process. Available only when you have provided enough information for the Package and Deployment Wizard to successfully deploy the package.

If you chose a pre-existing script, the Finish button is available because the Package and Deployment Wizard assumes that the script contains all of the information needed to deploy the package.

Note   If you chose a different deployment method from that specified in the script, the Finish button is unavailable because the new deployment method will require information not stored in the script.
