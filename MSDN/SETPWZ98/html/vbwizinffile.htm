INF File

Contains instructions for installing the ActiveX components on the user's machine. The .inf file contains:

  • A list of all file dependencies needed for Internet download setup.

  • A list of linked .cab files and their locations.

  • Version information.

  • Class IDs for components.

  • Platform specific information.

  • Destination directory specifics.

  • Registry information, including Safe for Initialization and Scripting entries.
