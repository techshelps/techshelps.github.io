SafeArrayGetElemsize

UINT SafeArrayGetElemsize( 
  SAFEARRAY FAR*  psa  
);

Parameter

psa
Pointer to an array descriptor created by SafeArrayCreate.

Return Value

Returns the size (in bytes) of the elements of a safe array.
