SafeArrayGetDim

UINT SafeArrayGetDim( 
  SAFEARRAY FAR*  psa  
);

Returns the number of dimensions in the array.

Parameter

psa
Pointer to an array descriptor created by SafeArrayCreate.

Return Value

Returns the number of dimensions in the array.
