Reading Order

Text entered in Visual Basic has either an RTL (Middle East) or an LTR (Latin) reading order. Reading order describes the order in which words are displayed in mixed text. It is instructive to note that reading order pertains to the word order, not the order of the entered characters. When Arabic is the keyboard language, new characters will always flow in a right-to-left manner. Conversely, when Latin is the keyboard language, characters flow in a left-to-right manner.
