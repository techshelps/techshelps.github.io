RightToLeft Property (DataList Control)

      

Bidirectional Features

The DataList control has configurable alignment and reading order of list box items, and automatic positioning of the vertical scroll bar.

When the RightToLeft property is set to True:

  • Reading order of DataList box items is RTL

  • List box items are right aligned.

  • If the list area is smaller than the number of items to be displayed, a vertical scroll bar appears automatically at the left of the list box.

When the RightToLeft property is set to False, item reading order is LTR, items are left aligned, and the vertical scroll bar is positioned at the right of the list box.
