Form Object Used in Bidirectional Programming

Forms:

  • Control the layout of menus.

  • Determine the form caption reading order.

  • Establish the default RightToLeft property value for newly added controls that have a RightToLeft property.

  • Determine printing behavior.

Note   Bidirectional form behavior applies to all forms: MDI (multiple document interface) forms, MDI child forms, or non-MDI forms.

Forms are RTL or LTR depending on the setting for their RightToLeft property. A form with a RightToLeft property set to False behaves exactly as it's described in Visual Basic 6.0 documentation when not running in a bidirectional 32-bit Microsoft Windows environment (this is the default). If the RightToLeft property is set to True, several actions occur:

  • Menus and their associated command names (or items) are right-aligned with RTL reading order. They appear at the right edge of the form in an overall RTL layout.

  • Form captions have an RTL reading order and are right-aligned in the title bar.

  • Any text that the Print method prints to the form will begin at the right boundary of the form and have RTL reading order.
