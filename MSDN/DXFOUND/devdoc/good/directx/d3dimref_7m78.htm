D3DLINE

The D3DLINE structure describes a line for the D3DOP_LINE opcode in the D3DOPCODE enumerated type.

typedef struct _D3DLINE { 
    union { 
        WORD v1; 
        WORD wV1; 
    }; 
    union { 
        WORD v2; 
        WORD wV2; 
    }; 
} D3DLINE, *LPD3DLINE;

Members

wV1 and wV2
Vertex indices.

Remarks

The instruction count defines the number of line segments.
