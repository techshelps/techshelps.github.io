RGB_GETBLUE

The RGB_GETBLUE macro retrieves the blue component of a D3DCOLOR value.

RGB_GETBLUE(rgb)    ((rgb) & 0xff)

Parameters

rgb
Color index from which the blue component is retrieved.

Return Values

Returns the blue component.
