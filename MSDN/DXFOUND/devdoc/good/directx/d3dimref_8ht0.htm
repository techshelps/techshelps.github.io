D3DVECTOR

The D3DVECTOR structure defines a vector for many Direct3D and Direct3DRM methods and structures.

typedef struct _D3DVECTOR { 
    union { 
        D3DVALUE x; 
        D3DVALUE dvX; 
    }; 
    union { 
        D3DVALUE y; 
        D3DVALUE dvY; 
    }; 
    union { 
        D3DVALUE z; 
        D3DVALUE dvZ; 
    }; 
} D3DVECTOR, *LPD3DVECTOR;

Members

dvX, dvY, and dvZ
Values of the D3DVALUE type describing the vector.
