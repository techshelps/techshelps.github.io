Interfaces

This section contains reference information for the COM interfaces provided by Direct3D's Immediate Mode. The following interfaces are covered:

  • IDirect3D2
  • IDirect3DDevice
  • IDirect3DDevice2
  • IDirect3DExecuteBuffer
  • IDirect3DLight
  • IDirect3DMaterial2
  • IDirect3DTexture2
  • IDirect3DViewport2

Some stub methods that were exposed in previous releases of DirectX are no longer exposed. For more information, see Unimplemented Methods.
