D3DRECT

The D3DRECT structure is a rectangle definition.

typedef struct _D3DRECT { 
    union { 
        LONG x1; 
        LONG lX1; 
    }; 
    union { 
        LONG y1; 
        LONG lY1; 
    }; 
    union { 
        LONG x2; 
        LONG lX2; 
    }; 
    union { 
        LONG y2; 
        LONG lY2; 
    }; 
} D3DRECT, *LPD3DRECT;

Members

lX1 and lY1
Coordinates of the upper-left corner of the rectangle.
lX2 and lY2
Coordinates of the lower-right corner of the rectangle.
