Axes Collection (ADO MD)

            

An Axes collection contains the Axis objects that define a Cellset object.

Remarks

A Cellset object contains an Axes collection. Once the Cellset is opened this collection will contain at least one Axis. See the Axis object topic for a more detailed explanation of how to use Axis objects.

Note   The filter axis of a Cellset is not contained in the Axes collection. See the FilterAxis property topic for more information.
