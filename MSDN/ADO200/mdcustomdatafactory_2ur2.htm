Customization File Logs Section

   

This section contains a log file entry, which specifies the name of a file that records errors during the operation of the DataFactory.

Syntax

A log file entry is of the form:

err=FileName

Part Description
err Literal string—Indicates this is a log file entry.
FileName A complete path and file name. The typical file name is c:\msdfmap.log.

The log file will contain the user name, HRESULT, date, and time of each error.
