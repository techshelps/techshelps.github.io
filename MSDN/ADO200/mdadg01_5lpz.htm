Remote Data Service Developer's Guide

   

The Remote Data Service (RDS) is designed for building data-centric Web applications. It incorporates existing technologies, such as ActiveX®; Microsoft® Internet Information Server (IIS); Microsoft® Visual Basic®, Scripting Edition; JScript™; and OLE DB, and introduces innovations such as enhanced data binding for Web pages, client-side caching, and the ability to invoke custom business objects.

There are many related technologies that you use in an RDS application. The Developer's Guide describes the various parts of a three-tiered RDS application and provides code examples to show you how to use those technologies.
