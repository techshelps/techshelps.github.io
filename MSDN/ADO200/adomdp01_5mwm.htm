ParentSameAsPrev Property (ADO MD)

         

The ParentSameAsPrev property indicates whether the parent of this member is the same as the parent of the immediately preceding member.

Return Values

ParentSameAsPrev returns a Boolean value and is read-only.

Remarks

This property is supported only on Member objects belonging to a Position object. An error occurs when this property is referenced from Member objects belonging to a Level object.
