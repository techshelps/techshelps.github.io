onReadyStateChange (Event) Method (RDS)

   

This method is called whenever the value of the ReadyState property changes.

Syntax

onReadyStateChange

Parameters

None

Remarks

The ReadyState property reflects the progress of an RDS.DataControl object as it asynchronously fetches data into its Recordset object. Use the onReadyStateChange method to monitor changes in the ReadyState property whenever they occur. This is more efficient than periodically checking the property's value.
