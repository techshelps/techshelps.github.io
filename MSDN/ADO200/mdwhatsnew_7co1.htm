ADO for Windows Foundation Classes

   

You can now access data using the ADO application programming interface with Microsoft® Visual J++™ (the Java programming language).

ADO for Windows Foundation Classes (ADO/WFC) supports all the standard ADO methods, properties, objects, and events. However, operations that require a VARIANT as a parameter and show excellent performance in a language like Microsoft® Visual Basic®, display lesser performance in a language like Visual J++. For that reason, ADO/WFC also provides accessor functions on the Field object that take native Java data types instead of the VARIANT data type.

This feature will be available with the release of Visual J++ version 6.
