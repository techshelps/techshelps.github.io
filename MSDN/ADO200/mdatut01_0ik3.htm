Remote Data Service Tutorials

   

The Remote Data Service tutorials provide step-by-step procedures that walk you through using the most common data access features.

  • RDS tutorial. This tutorial illustrates using the RDS programming model to query and update a data source.

  • Building a Simple Remote Data Service Application. This tutorial walks you through building a simple Remote Data Service application that provides access to live data in a Microsoft® Access database.

  • Address Book. This tutorial shows you how to create a simple Remote Data Service application that provides access and updatability to a Microsoft® SQL Server™ database.
