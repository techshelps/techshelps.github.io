Repository Programmer's Guide for Visual Basic

You can use Microsoft Repository with Microsoft Visual Basic 6.0. The Visual Basic package includes:

  • The Microsoft Repository software.

  • The MDO Model (an information model that lets you store the structure of Visual Basic applications in the repository).

  • The Repository Add-in for Visual Basic (a Visual Basic add-in that automatically maintains the structure of your Visual Basic projects in the repository).

The following topics describe how Visual Basic, Microsoft Repository, and the Repository Add-in for Visual Basic interact, and some typical ways you can use Repository with Visual Basic.
