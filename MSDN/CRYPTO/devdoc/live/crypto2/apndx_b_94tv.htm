Global Changes

  • Changed "Digest" to "Hash", "digest" to "hash" and "DIGEST" to "HASH".
  • Prefixed all data structures with one of the following to avoid name conflicts with other include files:

    CERT_, PCERT_, _CERT_,

    CRYPT_, PCRYPT_, _CRYPT_,

    CMSG_, PCMSG_, _CMSG_

  • The *_OBJID constants were renamed to szOID_* constants. For example,

    AUTHORITY_KEY_IDENTIFIER_OBJID renamed to szOID_AUTHORITY_KEY_IDENTIFIER.

  • In wincrypt.h, all data structures and APIs having to do with SET were removed.
  • In wincrypt.h, all data structures having to do with SPC were moved to signcde.h.
  • The certificate and message encoding types were consolidated into a single DWORD. For more information, see Certificate and Message Encoding Types.
