CERT_RDN

The CERT_RDN structure is a Relative Distinguished Name (RDN) which consists of an array of CERT_RDN_ATTR structures.

typedef struct _CERT_RDN {
    DWORD            cRDNAttr;
    PCERT_RDN_ATTR   rgRDNAttr;
} CERT_RDN,         *PCERT_RDN;

Members

cRDNAttr
The number of elements in the array rgRDNAttr.
rgRDNAttr
An array of CERT_RDN_ATTR structures.
