CERT_EXTENSIONS

The CERT_EXTENSIONS structure contains an array of extensions.

typedef struct _CERT_EXTENSIONS {
    DWORD              cExtension;
    PCERT_EXTENSION    rgExtension;
} CERT_EXTENSIONS,    *PCERT_EXTENSIONS;

Members

cExtension
The number of elements in the array rgExtension.
rgExtension
An array of structures, each holding CERT_EXTENSION information about a certificate or CRL.
