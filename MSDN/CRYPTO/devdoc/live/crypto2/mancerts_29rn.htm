Advanced Certificate Store Concepts

Microsoft® Windows NT® 5.0, adds new functionality to CryptoAPI for working with certificate stores. This includes the use of collection certificate stores, logical system certificate stores, and the ability to store links to contexts in stores rather than duplicates of the contexts.

For details on collection stores, see Logical and Collection Certificate Stores.

For details on logical system stores, see System Certificate Stores as Logical Stores.

For details on using links, see Using Certificate, CRL, and CTL Links.

For details about the registry locations of system logical stores and default physical stores associated with them, System Store Locations.
