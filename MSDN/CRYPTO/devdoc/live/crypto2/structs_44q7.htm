CERT_POLICY_INFO

The CERT_POLICY_INFO structure contains an object identifier string specifying the policy, and an optional array of policy qualifiers.

CERT_POLICY_INFO is a component of CERT_POLICIES_INFO.

typedef struct _CERT_POLICY_INFO {
    LPSTR                        pszPolicyIdentifier;
    DWORD                        cPolicyQualifier;  
    CERT_POLICY_QUALIFIER_INFO*  rgPolicyQualifier; 
} CERT_POLICY_INFO,            *PCERT_POLICY_INFO;

Members

pszPolicyIdentifier
An object identifier string specifying the policy.
cPolicyQualifier
The number of elements in the array rgPolicyQualifier.
rgPolicyQualifier
An array of structures, each holding CERT_POLICY_QUALIFIER_INFO information.
