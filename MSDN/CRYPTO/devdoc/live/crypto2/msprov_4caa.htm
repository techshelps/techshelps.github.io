Using the CryptoAPI with the Microsoft PROV_DSS_DH Provider

This section describes how the CryptoAPI can be used to generate D-H keys, exchange D-H keys, and export a private D-H key.

This provider can also be used to hash data, generate DSS keys, sign data with DSS keys, and verify DSS signatures. For information on how to do this, see Using the CryptoAPI with the Microsoft DSS Cryptographic Provider.
