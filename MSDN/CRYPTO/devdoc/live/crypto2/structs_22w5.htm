CRYPT_ATTRIBUTE_TYPE_VALUE

The CRYPT_ATTRIBUTE_TYPE_VALUE structure is used to specify an attribute having only a single value. The Value's CRYPT_OBJID_BLOB is in its encoded representation.

typedef struct _CRYPT_ATTRIBUTE_TYPE_VALUE {
    LPSTR                       pszObjId;
    CRYPT_OBJID_BLOB            Value;
} CRYPT_ATTRIBUTE_TYPE_VALUE,  *PCRYPT_ATTRIBUTE_TYPE_VALUE;

Members

pszObjId
The object identifier that specifies the attribute type data contained in the Value blob.
Value
The encoded attribute.
