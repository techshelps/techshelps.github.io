szOID_NEXT_UPDATE_LOCATION

When szOID_NEXT_UPDATE_LOCATION is used for the lpszStructType with CryptEncodeObject, CryptEncodeObjectEx, CryptDecodeObject, or CryptDecodeObjectEx the following details apply.

Object Identifier String

"1.3.6.1.4.1.311.10.2"

Corresponding Data Structure being encoded or decoded

pvStructInfo points to a CERT_ALT_NAME_INFO structure.

Additional Details

Used with certificate trust lists to get the location for the most recent, time valid CTL. Commonly, the choice used in the CERT_ALT_NAME_INFO is a URL indicating the location.
