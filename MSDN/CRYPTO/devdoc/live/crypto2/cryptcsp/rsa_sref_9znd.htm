KP_CLEAR_KEY

This parameter specifies the "clear key" data. The value is only used when generating an export encryption key and should be ignored if using a domestic cipher.

When using PCT, the clear key data ranges from 0 to 16 bytes. When using SSL 2, the clear key data will be either 0 or 11 bytes. This parameter is not used by SSL 3.0 or TLS 1.0.
