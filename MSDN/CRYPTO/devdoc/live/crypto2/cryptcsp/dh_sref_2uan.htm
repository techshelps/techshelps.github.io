KP_ALGID (Diffie-Hellman)

This parameter is used to change the key from an agreed key to the required pre-master secret key. Two new algorithm types have been added for use with this value, CALG_SSL3_MASTER and CALG_TLS1_MASTER.

For more information on pre-master secret keys, see CALG_SCHANNEL_MASTER_HASH (Diffie-Hellman).
