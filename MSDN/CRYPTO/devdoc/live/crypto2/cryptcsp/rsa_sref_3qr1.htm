KP_SERVER_RANDOM

When using SSL 3.0 or TLS 1.0, this parameter specifies the "server random" data. When using PCT, it specifies the "connection id" data. In both cases, this data originates in the "ServerHello" message.

When using PCT, SSL 3.0, or TLS 1.0, this data is always 32 bytes long. This parameter is not used by SSL 2.
