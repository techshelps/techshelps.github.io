Default CRYPT_OID_INFO Tables by dwGroupId

The following default tables are supplied with predefined entries that cover commonly used key and group ID combinations. These tables can be expanded per user need by adding combinations with CryptRegisterOIDInfo User added combinations can be removed from the tables using CryptUnregisterOIDInfo.

Table by dwGroupId Description
CRYPT_HASH_ALG_OID_GROUP_ID Hash Algorithm Table
CRYPT_ENCRYPT_ALG_OID_GROUP_ID Encryption Algorithm Table
CRYPT_PUBKEY_ALG_OID_GROUP_ID Public Key Algorithm Table
CRYPT_SIGN_ALG_OID_GROUP_ID Signature Algorithm Table
CRYPT_RDN_ATTR_OID_GROUP_ID RDN Attribute Table
CRYPT_EXT_OR_ATTR_OID_GROUP_ID Extension or Attribute Table
CRYPT_ENHKEY_USAGE_OID_GROUP_ID Enhanced Key Usage Table
CRYPT_POLICY_OID_GROUP_ID Policy Table
