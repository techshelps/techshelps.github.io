RSAPUBKEY

The RSAPUBKEY structure contains information specific to the particular public key contained in the key blob. 

typedef struct _RSAPUBKEY {
    DWORD magic;
    DWORD bitlen;
    DWORD pubexp;
} RSAPUBKEY;

Members

magic
This should be set to "RSA1" (0x31415352) for public keys and to "RSA2" (0x32415352) for private keys.

Notice that the hex values are just an ASCII encoding of "RSA1" and "RSA2."

bitlen
Number of bits in the modulus. In practice, this must always be a multiple of eight.
pubexp
The public exponent.
