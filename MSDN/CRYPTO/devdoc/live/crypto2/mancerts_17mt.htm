Certificate Management Example Code

The following example code demonstrates how to use many of the certificate functions. This code uses some cryptographic function calls that are described in earlier sections and in the "CryptoAPI Reference."

For examples of code used to open certificate stores, see Example Code for Opening a Certificate Store.

For more general certificate store code examples, see Example Code for Fundamental Certificate Store Operations.

For examples of code for certificate verification, see Example Code for Certificate Verification Operations.

For example code for user defined functions including functions used in many of the examples, see Example Code for User Defined Functions.

For code to list system store locations, system stores or the physical stores that are members of a system store, see Example Code for Listing System and Physical Stores.
