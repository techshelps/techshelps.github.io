New Low-Level Cryptographic Message Functions and Data Structures

Most of the cryptographic message functions and data structures were included in the SDR version of wincrypt.h. However, no documentation was provided on their usage. Therefore, for the purpose of this Summary of Changes, assume that all the low-level cryptographic message functions were added after the SDR. All the functions start with the CryptMsg prefix. The data structures are prefixed with CMSG_, PCMSG_ or _CMSG_.

For an overview, see Using Low-Level Message Functions. For details, see Low Level Message Functions.
