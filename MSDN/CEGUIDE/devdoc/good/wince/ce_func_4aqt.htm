CeCopyFile

The CeCopyFile function copies an existing file to a new file. This is the RAPI version of the CopyFile function.

Syntax

BOOL CeCopyFile(LPCWSTR lpExistingFileName, LPCWSTR lpNewFileName, BOOL bFailIfExists);

At a Glance

Header file: Rapi.h
Platforms: H/PC
Windows CE versions: 2.0 and later

Remarks

When writing applications for Windows CE version 1.0 and 1.01, use the PegCopyFile function.

For more information about using RAPI functions, see Invoking Functions from a Desktop Computer.
