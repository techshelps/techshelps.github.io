IDccManSink::OnLogError

The IDccManSink::OnLogError method indicates that the connection manager failed to start communications between the desktop PC and the Windows CE device.

Syntax

STDMETHODIMP IDccManSink::OnLogError(void);

At a Glance

Header file: Dccole.h
Platforms: H/PC
Windows CE versions: 1.0 and later

Parameters

None.

Return Values

This method should always return NOERROR.

Remarks

The IDccManSink::OnLogError method is implemented by a client application and is called if the Windows CE connection manager failed to start the Remote Access Service (RAS) communications between the desktop PC and the Windows CE device.

OnLogError may also be called whenever a general Windows CE connection manager related error occurs.

For example, OnLogError is called if the communications port — selected in the Communications settings of the Windows CE connection manager Communications Properties — is already in use or is not available.

For more information, see Receiving Connection Notification.
