CommandBar_InsertComboBox

The CommandBar_InsertComboBox function inserts a combo box into the command bar.

Syntax

HWND CommandBar_InsertComboBox(HWND hwndCB, HINSTANCE hInst, int iWidth, UINT dwStyle, WORD idComboBox, int iButton);

At a Glance

Header file: Commctrl.h
Platforms: H/PC
Windows CE versions: 1.0 and later

Parameters

hwndCB
The command bar's window handle. This handle is returned by the CommandBar_Create function when the command bar is created.
hInst
Handle to the application instance.
iWidth
Width, in pixels, of the combo box.
dwStyle
Window styles to apply to the combo box. The WS_VISIBLE and WS_CHILD styles are applied automatically. For descriptions of other styles you can apply, see Combo Box Styles and Window Styles.
idComboBox
Identifier of the combo box to insert.
iButton
Zero-based index of a button in the command bar. CommandBar_InsertComboBox inserts the combo box to the left of this button.

Return Values

The window handle to the combo box. If this function fails, it returns a value of NULL.

See Also

CommandBar_Create, CommandBar_InsertButton, CommandBar_InsertMenubar
