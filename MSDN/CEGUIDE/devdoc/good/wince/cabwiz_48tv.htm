CEUTIL: Helper DLL for Windows CE Services

Use CEUTIL, a utility DLL, to handle the desktop registry entries for Windows CE Services. CEUTIL encapsulates the registry top-level locations, to ensure forward-compatibility for applications. It also provides helper functions for browsing device partnerships and querying the currently connected, or selected, device settings. In general, this DLL is a replacement for, and compatible with, the Win32 registry application programming interface (API) used when referring to any subkeys under the Windows CE Services root.

Use CEUTIL to do the following tasks:

  • Register desktop file filters
  • Register desktop synchronization services
  • Access device-partnership settings used for both file filters and synchronization services
  • Add custom menu items

For a list of the CEUTIL functions, see Lists of Functions and Interfaces.
