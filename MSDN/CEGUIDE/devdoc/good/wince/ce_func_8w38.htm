CommandBands_Height

The CommandBands_Height macro retrieves the height in pixels of a command bands control.

Syntax

UINT CommandBands_Height (HWND hwndCmdBands);

At a Glance

Header file: Commctrl.h
Platforms: H/PC
Windows CE versions: 2.0 and later

Parameters

hwndCmdBand
Window handle to the command bands control. This handle is returned when the command bands control is created by the CommandBands_Create function.

Return Values

The height, in pixels, of the command bands control.

Remarks

Because the command bar occupies part of the client area of an application's main window, you should call this function to determine the actual size of the client area.
