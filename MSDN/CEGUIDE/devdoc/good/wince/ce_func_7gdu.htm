CeSetFilePointer

The CeSetFilePointer function moves the file pointer of an open file. This is the RAPI version of the SetFilePointer function.

Syntax

DWORD CeSetFilePointer(HANDLE hFile, LONG lDistanceToMove, PLONG lpDistanceToMoveHigh, DWORD dwMoveMethod);

At a Glance

Header file: Rapi.h
Platforms: H/PC
Windows CE versions: 2.0 and later

Remarks

When writing applications for Windows CE versions 1.0 and 1.01, use the PegSetFilePointer function.

For more information about using RAPI functions, see Invoking Functions from a Desktop Computer.
