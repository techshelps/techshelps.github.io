CeGetWindowLong

The CeGetWindowLong function retrieves information about the specified window. CeGetWindowLong also retrieves the 32-bit (long) value at the specified offset into the extra window memory of a window. This is the RAPI version of the GetWindowLong function.

Syntax

LONG CeGetWindowLong(HWND hWnd, int nIndex);

At a Glance

Header file: Rapi.h
Platforms: H/PC
Windows CE versions: 2.0 and later

Remarks

When writing applications for Windows CE versions 1.0 and 1.01, use the PegGetWindowLong function.

For more information on using RAPI functions, see Invoking Functions from a Desktop Computer.
