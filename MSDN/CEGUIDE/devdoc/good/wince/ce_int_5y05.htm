IDccManSink::OnLogActive

The IDccManSink::OnLogActive method indicates that a connection is established between the client application and the connection manager.

Syntax

STDMETHODIMP IDccManSink::OnLogActive(void);

At a Glance

Header file: Dccole.h
Platforms: H/PC
Windows CE versions: 1.0 and later

Parameters

None.

Return Values

This method should always return NOERROR.

Remarks

The IDccManSink::OnLogActive method is implemented by a client application and is called when a socket connection is established between the client application and the connection manager.

When this notification occurs, the Windows CE connection manager starts any applications that are registered with the registry key HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows CE Services\AutoStartOnConnect.

During this notification is a good time for the client application to initialize the remote application programming interface (RAPI) by calling the RAPI function CeRapiInit. A call to CeRapiInit does not return until a connection is established between the desktop PC and the Windows CE device.

For more information, see Receiving Connection Notification, Registry-Based Notification and Invoking Functions from a Desktop Computer.
