CommandBar_Destroy

The CommandBar_Destroy function destroys the command bar.

Syntax

void CommandBar_Destroy(HWND hwndCB);

At a Glance

Header file: Commctrl.h
Platforms: H/PC
Windows CE versions: 1.0 and later

Parameters

hwndCB
The command bar's window handle. This handle is returned by the CommandBar_Create function when the command bar is created.

Remarks

The CommandBar_Destroy function calls the DestroyWindow function to destroy the command bar. If the command bar's parent window is being destroyed, it is not necessary to call CommandBar_Destroy.
