CeSHCreateShortcut

The CeSHCreateShortcut function creates a shortcut. This is the RAPI version of the SHCreateShortcut function.

Syntax

DWORD CeSHCreateShortcut(LPWSTR lpszShortcut, LPWSTR lpszTarget);

At a Glance

Header file: Rapi.h
Platforms: H/PC
Windows CE versions: 2.0 and later

Remarks

When writing applications for Windows CE versions 1.0 and 1.01, use the PegSHCreateShortcut function.

For more information about using RAPI functions, see Invoking Functions from a Desktop Computer.
