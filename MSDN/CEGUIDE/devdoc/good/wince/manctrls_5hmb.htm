Foundation Controls

The controls described in this chapter are all common controls that contain or manage other controls. For example, a command bar can contain menus, combo boxes, and buttons. A command band can contain a variety of controls including command bands. All of the controls described in this chapter are designed as containers for other controls.

The following controls are described in this chapter:

  • Command bars
  • Property sheets
  • Rebars
  • Command bands
  • Tab controls
  • Toolbars
