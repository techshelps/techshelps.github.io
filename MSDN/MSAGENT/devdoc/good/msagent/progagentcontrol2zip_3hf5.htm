The Agent Object Model

The Microsoft Agent Object Model consists of the following objects:

  • Request
  • Agent (control)
  • Characters (collection)
  • Character
  • Commands (collection)
  • Command
  • Balloon
  • SpeechInput
  • AudioOutput
  • CommandsWindow
  • PropertySheet
