Can I redistribute Microsoft Agent and its characters?

Microsoft Agent can be downloaded directly from the Microsoft site by including the Microsoft Agent control's CLSID in an HTML <OBJECT> tag on your Web page, which will automatically offer to download and install the control when a user loads the page. Details on licensing and use are included in the Microsoft Agent End User License agreement that displays when Microsoft Agent is installed. This license does not include provisions for redistribution. Application developers who wish to include Microsoft Agent and any of its components as part of their application must obtain a redistribution license. For more information about redistribution licenses, see the Licensing and Redistribution document at http://www.microsoft.com/workshop/prog/agent/licensing.htm.
