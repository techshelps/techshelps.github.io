How do I remove Microsoft Agent from my system?

To uninstall Microsoft Agent, open the Control Panel, then open the Add/Remove Programs object. You should find Microsoft Agent included in the list. Select the entry, then click the Add/Remove button.

The Microsoft Agent uninstall utility removes only the Microsoft Agent services. If you also installed the Lernout & Hauspie® TruVoice Text-To-Speech Engine for Microsoft Agent, select this and click the Add/Remove button. In addition, if you also installed Microsoft Command and Control speech recognition engine and you know that no other application uses it, you should select and remove the following entries:

Microsoft Command & Control Engine

Microsoft Speech Lexicon

Microsoft Speech API (3.0)
