When I attempt to install Microsoft Agent on Windows NT 4.0, I get the following error: Process (Regsvr32 /s Progra~1\Micros~2\AgentCtl.dll). Error while creating this file. Cannot find this file.

Installation of Microsoft Agent requires the proper installation of Regsvr32.exe (and other OLE dlls). The best way to ensure that all the correct system files are present is to install Microsoft Internet Explorer 3.02 or later.
