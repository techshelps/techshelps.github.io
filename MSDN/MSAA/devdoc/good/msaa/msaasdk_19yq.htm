About the Reader

[This is preliminary documentation and subject to change.]

This document provides a complete reference to the objects, functions, and other API elements that comprise Microsoft® Active Accessibility. Active Accessibility objects are OLE component objects that conform to the OLE Component Object Model (COM).

To get the most out of this documentation, you should be familiar with the following material.

  • C and C++ programming concepts.
  • OLE programming concepts, including: COM, interface queries, and calling functions.
  • The Win32 application programming interface as presented by the Microsoft Platform Software Development Kit (SDK).
