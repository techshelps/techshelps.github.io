Alt Tab Window

  • Description

    The Alt Tab window is what appears when the user presses the alt and tab keys together. It displays a dialog-link window with a series of icons displayed in one or more rows. Each icon represents a running application.

  • Properties and Methods

    DefaultAction is "Switch"

    DoDefaultAction switches to the application referenced by the item

  • Events

    Normal window and focus events are sent with this object

Carets

  • Description

    Carets are the insertion point in editable text. Carets are owned by applications, but the operating system is aware of them. Not all applications use the system-provided caret APIs, opting instead to draw and redraw the caret themselves. In such cases, Active Accessibility can't provide information.

    HitTest

    Role

    State

  • Events
    • CREATE/DESTROY

    • STATECHANGE when shown or hidden.
