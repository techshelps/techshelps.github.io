M

minidriver

A device driver that contains hardware-specific code related to a specific hardware device. The Win32 Driver Model allows hardware manufacturers to implement minidrivers.

miniport driver

A 32-bit installable driver. Windows NT was the first operating system to use miniport drivers.

motherboard

See system board.

MPEG

Acronym for Motion Picture Experts Group, used when referring to one of several standard video-compression schemes.
