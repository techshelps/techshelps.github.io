S

SCAM

Acronym for SCSI configured automatically.

SCSI

Acronym for small computer system interface. An I/O bus designed as a method of connecting several classes of peripherals to a host system without requiring modifications to generic hardware and software.

SPI

Acronym for SCSI-3 parallel interfaces.

spin down

A power management capability in which a hard drive shuts down its spindle motor.

static resources

Device resources, such as IRQ signals, DMA channels, I/O port addresses, and memory addresses, that cannot be configured or relocated.

system board

The primary circuit board in a PC that contains most of the basic components of the system. Also referred to as motherboard or planar.
