Ensuring Secure Data Transmissions

   

Under normal unsecured data transmission circumstances, the information your application transmits could be captured by someone able to read the contents of (or "sniff") network packets. Anyone with the ability to watch raw traffic on the Internet wires is able to see your data.

To protect sensitive data transmissions, you should consider one of the following technologies, described in this section:

  • Secured Communications and SSL
  • Using the Microsoft CryptoAPI
  • Using the Microsoft Secure Electronic Transactions Framework (SET)
