Using Visual Studio

Thanks for purchasing Microsoft Visual Studio 6.0! The book you received with your purchase of a full Visual Studio product suite is designed to help you use the suite's tools in concert to meet your development goals. (These books are also available online in this documentation.)

Developing for the Enterprise, included with the Enterprise Edition, is a complete guide for developing large-scale applications for managing mission-critical data.

Developing for Windows and the Web: Guide to Visual Studio 6.0, Professional Edition, included with the Professional and Enterprise Editions, provides an ideal starting place if you're planning a new application or want to expand your development skills.

The Visual Studio 6.0 family of development products includes:

  • Visual Basic
  • Visual C++
  • Visual FoxPro
  • Visual InterDev
  • Visual J++
  • Visual SourceSafe
  • MSDN Library
