ActiveX Control Constants

The following constants are recognized by the ActiveX controls. As a result, they can be used anywhere in your code in place of the actual values.

  • BorderStyle Constants

  • MousePointer Constants

  • OLEDragDrop Constants

Use the Object Browser to view the intrinsic constants you can use with methods and properties. From the View menu, choose Object Browser, select the appropriate control library, and then the Constants object. You can scroll through the constants that appear in the Members pane.

Note   Prefixes for the constants change with the specific control or group of controls. However, the description remains the same unless indicated.
