While attempting to execute a Command object which is based on a stored procedure, the Field and Parameter objects required cannot be retrieved. Note that if the stored procedure performs an insert or delete, executing the Command object will add or delete data from the database. If the stored procedure performs a SELECT, you may execute the Command object without any side effects.
