Toolbar (Webclass Designer)

Contains buttons that are shortcuts to some menu items frequently used when creating webclasses. You can click a toolbar button once to carry out the action represented by that button.

Toolbar Buttons

Cut

Removes the selected webitem or event and places it on the Clipboard.

Copy

Copies the selected webitem or event onto the Clipboard.

Paste

Inserts the contents of the Clipboard at the current location.

Delete

Deletes the currently selected webitem or event.

Add HTML Template WebItem

Displays the Add HTML Template dialog box, from which you can select an HTML template file to insert into the webclass. The designer displays the template and all of the attributes of HTML tags within it that are capable of launching server requests.

Add Custom WebItem

Adds a custom webitem to the webclass. You can create events for the custom webitem and write code for it.

Add Custom Event

Adds a custom event to the selected template or custom webitem.
