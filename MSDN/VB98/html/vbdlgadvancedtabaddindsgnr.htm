Advanced Tab (Add-In Designer)

Allows you to add further options when using the Add-In Designer to begin the development process for an add-in.

To display the Add-In Designer

  1. Select the AddInDesigner Object on the Designers tab of the Components dialog on the Project menu.

  2. Select the AddInDesigner Object from the Add ActiveX Designer command on the Project menu.

Window Elements

Satellite DLL Name

Enter the name of the file containing resources for the add-in. The resource file must be in the same folder as the registered .dll containing the add-in itself.

Registry Key for Additional Add-In Data

Enter the full path of the registry key containing the values listed in the list box below.

Add-In Specific Data

Enter values you want stored in the registry key listed in the list box above. The only two types of data permitted are String and DWORD.
