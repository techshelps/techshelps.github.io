Visual Basic found an instance of 'item' in a binary form file. VBX's cannot be converted from binary form files in the 32-bit version

Versions of Visual Basic later than 5.0 cannot convert VBXs in binary form files to the equivalent ActiveX controls.

You can use versions prior to Visual Basic 5.0 to convert the form file from binary to a text format form, then load it into the newer version of Visual Basic.
