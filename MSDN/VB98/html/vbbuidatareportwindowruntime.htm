Microsoft Data Report Window (Run Time)

Allows you to view the report as it will look when printed and to export your report to HTML or text.

Uses halftone palette.

Options

Print

Begins printing a hard copy of the report. You can print only to the current printer. Change the printer from your operating system and then return to the Microsoft Data Report and print.

Export

Exports selected range to an HTML file or text file.

Zoom

Allows you to view your report from 10% to 500% of the actual size of the printout.

Next Page

Moves you to the next page of the report.

Previous Page

Moves you to the previous page of the report.
