Function Property

      

Returns or sets the function to use for the DEAggregate object.

Syntax

object.Function [=value]

The Function property syntax has these parts:

Part Description
object An object expression that evaluates to an item in the Applies To list.
value A constant or value that specifies the function to use, as described in Settings.

Settings

The settings for value are:

Constant Description
deAny Any. Returns the default operation of the selection.
deAverage Average. Obtains the average of the values in the selection.
deCnt Count. Returns the count of the number of records in the selection.
deMax Maximum. Returns the highest value of the selection.
deMin Minimum. Returns the lowest value of the selection.
deSDev Standard Deviation. Returns the standard deviation of the selection.
deSum Sum. (Default) Returns the sum of all the values in the selection.
