Import/Export Command (VisData File Menu)

Allows you to import tables from other databases or export tables and SQL query results to another database.

Dialog Box Options

Import

Loads subsequent dialogs from which you can choose the data type and table that you can import from another database.

Export Table(s)

Exports all tables selected in the table box.

Export SQL Results

Exports the results from the current SQL query in the SQL Statement window.

List

Lists the available tables.
