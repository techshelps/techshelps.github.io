Status Property (AsyncProperty Object)

      

Returns protocol-specific status information about an AsyncRead operation.

Syntax

object.Status

The Status property syntax has this part:

Part Description
object An object expression that evaluates to an object in the Applies To list.

Remarks

The Status property returns a string specific to the StatusCode property and can be an empty string. For example, for vbAsyncStatusCodeBeginConnecting, the string is the TCP/IP port address.
