Define Color Dialog Box

   

Allows you to create customized colors.

Dialog Box Options

Color Screen with Slider

Allows you to choose your color by sliding the bar up and down, from white to black.

Color Preview Box

Allows you to preview your color.

Hue

Adjusts the hue from 0-239 with 0 representing black and 239 representing white.

Sat

Adjusts the color saturation from 0-240 with 0 representing black and 240 representing black.

Lum

Adjusts the luminescence from 0-240 with 0 representing black and 240 representing white.

Red

Adjusts the amount of red in the color from 0-255.

Green

Adjusts the amount of green in the color from 0 to 255.

Blue

Adjusts the amount of blue in the color from 0 to 255.

Add Color

Adds your custom color to the Color Palette.
