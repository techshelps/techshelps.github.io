FirstWeekOfYear Property

      

Sets or returns the week considered the first week of the year. Read/write both at design time and run time.

Syntax

object.FirstWeekOfYear [= value]

The FirstWeekOfYear property syntax has these parts:

Part Description
object An object expression that evaluates to an object in the Applies To list.
value Optional enumerated Integer. A value that determines the week considered the first week of the year, as described in Settings.

Settings

The settings for value are:

Constant Setting Description
fmtWeekUseSystem 0 Use NLS API setting.
fmtFirstJan1 1 The week containing January 1. This is the default value.
fmtFirstFourDays 2 The first week of the year containing at least four days.
fmtFirstFullWeek 3 The first full week of the year.

Remarks

This property is only used when formatting dates.
