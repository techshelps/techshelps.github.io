Grouping Tab (Command Properties Dialog Box)

The Command Properties dialog box is divided into six tabs. The General, Parameters, and Advanced tabs are used for setting properties of the Command and Parameter objects, while the Relation, Grouping, and Aggregates tabs are used for setting properties that are specific to Command hierarchies.

On the Grouping tab, specify the properties for a grouping hierarchy. You can also create, modify, and delete the grouping of a Command object.

Note   The properties on this tab do not apply to Visual InterDev.

Group Command Object

If selected, the Command object is grouped, and all controls on the tab are enabled. If not selected, the Command object is not grouped, and all controls on the tab are disabled.

Grouping Definition

This frame contains the properties for the grouping hierarchy. If Group Command Object is not selected, the contents of this frame are disabled.

Grouping Command Name

The name of the grouping Command object that is created when a Command object is grouped. This Command object contains the grouped data, while the current Command object contains the detail data. It is recommended that you change this to a meaningful, unique name. For example, if grouping by country, a logical name would be ByCountry.

Fields in Command

Lists all Field objects that exist in the source Command object.

Arrows

Use > to move the selected Field object from the Fields in Command list to the Fields Used for Grouping list. This is disabled if no items are listed in the Fields in Command list.

Use >> to move all Field objects from the Fields in Command list to the Fields Used for Grouping list. This is disabled if no items are listed in the Fields in Command list.

Use < to move the selected Field object from the Fields Used for Grouping list to the Fields in Command list. This is disabled if no items are listed in the Fields Used for Grouping list.

Use << to moves all Field objects from the Fields Used for Grouping list to the Fields in Command list. This is disabled if no items are listed in the Fields Used for Grouping list.

Fields Used for Grouping

Contains the Field objects by which the Command object is grouped.

Note   If the Command object is a child in a relation hierarchy, the Field objects used in the relation are automatically included here and cannot be removed.
