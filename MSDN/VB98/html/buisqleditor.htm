SQL Editor

A modal window that allows you to open, save, cut, copy, and paste stored procedures and triggers.

To open the SQL Editor

  1. In Data View, right-click the Stored Procedures folder or any stored procedure in that folder.

  2. Choose Design from the shortcut menu.

Font Colors

The SQL Editor window contains a single font that is color-coded. As you move from line to line by pressing ENTER or by using the mouse or keyboard, the text is colored based on the type of element.

Element Color
keyword red
argument blue
text black
punctuation green
