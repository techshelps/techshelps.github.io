BorderStyle Constants (ActiveX Controls)

Constant Value Description
ccNone 0 (Default) No border or border-related elements.
ccFixedSingle 1 (Default for ListView control) Fixed single. There is a single line border around the control.

Note   The cc prefix is reserved for the ActiveX controls. The prefixes for the constants may change with the specific control or group of controls. However, the description remains the same unless indicated.
