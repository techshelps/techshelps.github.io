Stored Procedure Tab (Visual Basic Batch T-SQL Debugger Dialog Box)

Tab Options

Procedure Name

Allows you to select from the available stored procedures.

Query

Displays the SQL string that calls the stored procedure.

Parameters

Allows you to select from the parameters included in the selected stored procedure.

Data Type

Displays the data type of the selected parameter.

Value

Allows you to type the value of the selected parameter.
