User Credentials Dialog Box (Webclass Designer)

Prompts you for your network user login information when the webclass creates the virtual directory for your project on a network share. Microsoft Internet Information Server must associate this information with any virtual directory it creates so that it can authenticate future requests to that directory. This prompt appears the first time you run the project.

Dialog Box Options

UserID

Allows you to enter your user ID for verification. Must be a fully qualified user name — that is, if the user account is a member of a domain, the domain must also be specified in the format domain\user.

Password

Allows you to enter your password for verification.
