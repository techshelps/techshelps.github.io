FirstApp (FirstApp.vbp)

This sample application demonstrates the usage of the Data Control with other data aware controls. It provides a simple introduction to assigning properties at run-time.

Background

The sample is comprised of a single form with a DataGrid control connected to a Data control. For more information, see "Developing an Application in Visual Basic," in the Programmer's Guide.

File Description
Form1.frx Binary data for Form1.frm
Form1.frm Form for the application
Firstapp.vbp The project file for the application.

To Run

From the Visual Basic File menu, choose Open Project and select the Firstapp.vbp file, which is listed in the Samples directory. Press F5 or choose Start from the Run menu to run the application.
