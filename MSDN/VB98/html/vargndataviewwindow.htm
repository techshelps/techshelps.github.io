Data View Window

   

Use the Data View Window to access the structure of your database and manipulate it. You can:

  • create and modify tables and views.

  • create and modify database diagrams.

  • create and modify stored procedures and triggers.

It also lets you create and manipulate Oracle functions and synonyms.

The Data Environment, on the other hand, is used in the project to access data from within your application.

The Data View Window looks at all of the Data Environments added to your project and displays the connections from the Data Environments.

You can also:

  • drag and drop objects from the Data View Window into the Data Environment to automatically create commands in the Data Environment.

  • Drag and drop commands from the Data Environment Window onto Forms and Reports.

You can open a Data View Window from:

  • The toolbar.

  • The View menu.
