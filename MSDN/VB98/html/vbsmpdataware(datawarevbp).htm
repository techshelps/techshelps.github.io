Dataware (Dataware.vbp)

The Dataware (Dataware.vbp) sample demonstrates how to create classes that can act as sources or consumers of data. For more information, see "Creating Data-Aware Classes" in "Programming with Objects"," in the Visual Basic Programmers Guide.

File Description
Dataware.vbp Main project for the sample application.
MySource.cls Data source class.
MyCons.cls Data consumer class.
Dataform.frm Form to display the output of the source and consumer classes.

From the Visual Basic File menu, choose Open Project and select the Dataware.vbp file, which is listed in the Samples directory. Press F5 or choose Start from the Run menu to run the application.
