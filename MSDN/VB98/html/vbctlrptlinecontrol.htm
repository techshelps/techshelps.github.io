Line Control (Data Report Designer)

            

A Line control is a graphical control displayed as a horizontal, vertical, or diagonal line.

Syntax

RptLine

Remarks

The Data Report designer version of the Line control is similar to the standard Visual Basic intrinsic Line control in displaying lines on any application. The Data Report version, however, uses a LineSlant property to determine which direction the line slopes instead of an X, Y coordinate system to determine the appearance of the line.
