Format Property (StdDataFormat Object)

      

Sets or returns a format string. Read/write both at design time and run time.

Syntax

object.Format = formatstring

The Format property syntax has these parts:

Part Description
object An object expression that evaluates to an object in the Applies To list.
formatstring Required. A string specifying the desired format. The Format property supports the same set of format expressions as the Format function.

Remarks

If you use complex format strings you will need to unformat the data yourself.

The formatting process occurs before the Format event. The unformatting process is handled by the database and occurs after the Unformat event.
