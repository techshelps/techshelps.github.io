Using Add-In Designer

Use the Add-In Designer to begin the development process for an add-in by setting the properties that determine load behavior, name, description, target application and version. The designer contributes code to the add-in's dll or exe file to register the add-in correctly for the target application.

Initializing a new add-in

  1.  Create an Add-In project by selecting New Project from the File menu and then AddIn in the New Project dialog box.

  2. On the General tab enter basic information about the add-in like description, load behavior, and indicate which applications host the add-in.

  3. Select the Advanced tab to enter additional information on localization resources, and registry settings.

  4. To add code to your add-in, double-click the Add-In Designer.

  5. To build your add-in, select Make exe or Make dll from the File menu.
