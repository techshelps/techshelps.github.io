General Tab (Options Dialog Box)

Using the Options dialog box you can change the global settings of your Data Environment designer. The properties are divided between two tabs: General and Field Mapping.

On the General tab, specify the display and operation options for your Data Environment designer.

Show properties immediately after object creation

Specifies whether the Properties dialog box appears immediately after the Command or Connection object is added.

If selected, the properties dialog box appears after an object is added. If not selected (the default), no properties dialog box appears after an object is added.

This setting is stored in the system registry.

Prompt before deleting object

Specifies whether a confirmation dialog box displays when you delete an object. If selected (the default), a confirmation displays after an object is deleted. If not selected, no confirmation displays after an object is deleted.

Show status bar

If selected (the default), the Data Environment designer's status bar displays. If not selected, no status bar displays.

Show system objects

If selected (the default), the database's system tables display in the Database Objects box. If not selected, only user-created tables show.

Disable Warnings

If selected, all warning messages are disabled for the Data Environment designer. If not selected (the default), warning messages display, as appropriate.

Note   It is recommended that warning messages remain enabled. Once disabled, the system will not alert you of effects of your requests, such as a deletion that may result in a broken connection.

Prompt before executing command

If selected (the default), the system prompts you before executing a Command object. If not selected, the system executes Command objects without providing any notice.
