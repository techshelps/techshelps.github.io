Zoom Dialog Box (VisData)

Allows you to view and modify text and memo data.

This is an expanded view of a text or memo field. You can display this dialog box from the Dynaset/Snapshot, Table, and Data Control forms by double-clicking a field or by pressing F4. If the recordset is not updatable or is not in edit mode, you will only be able to view the data. If it is updatable, you can make and save changes.

Dialog Box Options

Save

Saves the new text in the field.

Close without Changes

Cancels any changes made to the data.
