Data Environment Toolbar Buttons

   

The following table describes the toolbar icons of the Project – DataEnvironment window.

Toolbar of the Project – DataEnvironment window

Icon Control Description
Add Connection Adds a new Connection object to the DataEnvironment object.
Add Command Adds a new Command object to the DataEnvironment object.
Insert Stored Procedures Accesses the Insert Stored Procedures dialog box. From this dialog box, you can create one or more Command objects from a list of stored procedures that are available in a particular connection.
Add Child Command Adds a Child Command to the currently selected Command object.
Delete Deletes the currently selected Command or Connection object.
Refresh Refreshes the currently selected Command or Connection object. The Refresh functionality differs when performed on Connection and Command objects, as follows:
  • Refreshing a Connection object closes the connection, if it was open, and then reopens it. The connection information — a list of tables, stored procedures, views, or synonyms — is also refreshed.

  • Refreshing a Command object refreshes all its stored meta-data, such as field and parameter information. Any changes to the table or stored procedure's definition is reflected in the Command object after the refresh.

    Caution   All user-defined property settings are lost during the refresh. For Field objects, the Caption and Control properties are reset to their defaults. For Parameter objects, the Name, Direction, DataType, HostDataType, Precision, Scale, Size, and Value properties are reset.
Design Accesses the Query designer for a Command object when the command source is text.
Properties Accesses the Properties dialog box for the selected Command, Connection, or Field object.
View Code Accesses the Code Editor window, in which you can write code.
Options Accesses the Options dialog box.
Arrange by Connections Displays the objects grouped by connection, with all the Command objects listed beneath the Connection object to which they are associated.
Arrange by Objects Groups objects of the same type together. This enables you to quickly locate specific objects within a DataEnvironment object.
