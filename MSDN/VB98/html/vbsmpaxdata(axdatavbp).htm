AXData (AXData.vbg)

The AXData sample demonstrates how to create ActiveX components that can act as data sources for other controls or objects. It includes a UserControl that mimics the ADOData control, plus an ActiveX DLL data source that demontrates the use of an OLE DB Simple Provider. For step-by-step instructions on creating this sample, see  "Creating Data Sources," in Creating ActiveX Components.

File Description
MyDataControl.vbp Project file for the ActiveX data source control.
MyData.ctl User control for the MyDataControl project.
DataSourceTest.vbp Test project for demonstrating ActiveX data sources.
DataSourceTest.frm Form for the DataSourceTest project.
MyDataComponent.vbp Project file for the ActiveX data source DLL.
MyOSPObject.cls Class to implement an OLE DB Simple Provider.
MyDataSource.cls Class to act as a data source for the MyDataComponent project.
Customer.txt Data file for the OLE DB Simple Provider.

To Run

From the Visual Basic File menu, choose Open Project and select the AXData.vbg file, which is listed in the Samples directory. Press F5 or choose Start from the Run menu to run the application.
