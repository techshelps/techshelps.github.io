Create Virtual Root Dialog Box (Webclass Designer)

Allows you to confirm the name of the virtual directory that Visual Basic creates for your project when you first run a new project, if the project directory is not already configured as a virtual directory. A virtual directory, or virtual root, is a directory outside your Web server's home directory that appears to browsers as a subdirectory of the home directory.

Dialog Box Options

Virtual Root Name

Allows you to change the name of the virtual root that Visual Basic will create for your webclass. You cannot change the location.
