Data Report (DataRept)

The Data Report sample demonstrates the functionality in the Data Report designer. Using a Data Environment as a data source, the application displays a hierarchical report of customers and their orders. The report further uses the Grouping feature of the Data Environment to group the companies by country.

File Description
deNwind.Dsr Data Environment designer file.

prjNwind.vbp Project file.

To Run

The Data Report uses the Northwind database (Nwind.mdb). The Data Environment is configured to access the database from its default location when installed from the Visual Studio CD. If Nwind.mdb is moved, you must change the ConnectionString property of the Data Environment's Connection.

To change the ConnectionString

  1. Open the Data Environment designer.

  2. Right click the deNwind icon to display, and click Connections to display the deNwind Properties dialog box.

  3. Next to the ConnectionString box, click Build.

  4. Click the Data Source box, and type the full path to the new location of the Nwind.mdb file.

  5. Click Test Connection. If the connection passes, the path is correct.

Once the ConnectionString property is correct, run the project. On the form, click the Show Report button to display the data report.

Click the Export Report button to export the report to an HTML file.

Click the Templates button to print (to the Immediate window) the templates contained by the designer.
