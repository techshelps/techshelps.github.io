Connect to WebItem Dialog Box

Allows you to choose a custom or template webitem with which you want to associate the selected HTML tag attribute. The process of connecting an attribute to a webitem tells the webclass to invoke the webitem's Respond event whenever the element represented by the attribute's HTML tag is selected in the browser.

Dialog Box Options

WebItem List Box

Lists the available templates and custom webitems for the webclass. You can select any of these webitems to associate your HTML tag attribute with it. When an attribute is associated with a webitem, it fires that webitem's Respond event when the tag element is selected in the browser.

OK

Associates the attribute with the selected webitem.
