Aggregates Tab (Command Properties Dialog Box)

The Command Properties dialog box is divided into six tabs. The General, Parameters, and Advanced tabs are used for setting properties of the Command and Parameter objects, while the Relation, Grouping, and Aggregates tabs are used for setting properties that are specific to Command hierarchies.

On the Aggregates tab, define the aggregates associated with a relation or group-based hierarchy.

Note   The properties on this tab do not apply to Visual InterDev.

Aggregates

Lists the aggregates defined for the Command object.

Add

Inserts additional aggregates into the Aggregates box.

Note   Once an aggregate is added, you can define it in the Aggregate Settings and Grand Total Command frames.

Remove

Deletes the selected aggregate from the Aggregates box. If no aggregate is selected, this is disabled.

Aggregate Settings

This frame contains the control group used to change the aggregate properties of the selected aggregate. When no aggregate is selected, this is disabled.

Name

This is the name of the Field object that will be added to the Command object. At run time, this field will contain the calculated aggregate value.

You can change this to a more meaningful, unique name that describes the aggregate. For example, if you are creating an aggregate that provides the average number of items on each order, an appropriate name could be "AverageOrderNumber."

Function

Contains the aggregate function to perform on a field object, you can use one of the following operations for each aggregate that you create:

Operation Description
Any Returns the default operation of the selected field.
Average Obtains the average of the values in a particular field.
Count Returns the count of the number of records in the selection.
Maximum Returns the highest value in a selected field.
Minimum Returns the smallest value in a selected field.
Standard Deviation Returns the standard deviation of the selection.
Sum Returns the sum of all the values in the selected field.

Aggregate On

Specifies what the aggregate is based upon — grand total, grouping, or child command.

The contents of this list vary, depending on the definition of the Command object. Grouping is only available if the Command object is grouped, child Command objects are listed only if the current Command object is a parent, and Grand Total is listed only if the current Command object is a top-level object.

Field

Fields upon which you can create an aggregate. The fields listed are dependent upon the Aggregate On selection.

Grand Total Command

This frame contains the control group used if the Grand Total constraint is selected in Aggregate On.

Name

A name you specify for the Grand Total Command. It is recommended that you change this to a more meaningful, unique name. The Grand Total Command Name can be the same as the Aggregate Settings Name.
