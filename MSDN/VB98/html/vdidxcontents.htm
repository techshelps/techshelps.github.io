Visual Data Manager (VisData)

VisData is an application that was created using Visual Basic 5.0. A built version of VisData is used as an add-in accessible from the Add-Ins menu. In the Professional and Enterprise editions the source code for VisData is available in the samples directory.

Note   VisData is accessed by choosing the Visual Data Manager command on the Add-Ins menu.

This Help file includes information on the VisData interface: menu items, dialogs, and forms. It also contains a topic about VisData as a sample. It is recommended that you browse this Help file often to gain the most use of VisData.
