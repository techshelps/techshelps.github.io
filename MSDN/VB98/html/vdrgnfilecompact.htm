Compact MDB Commands (VisData File Menu)

Compacts a Jet .mdb file and creates an encrypted or decrypted file. If you choose the same name for the destination file as the source, the source will be overwritten with the new file.

7.0 MDB

Creates a Microsoft Access version 7.0 .mdb file with the same encryption status as the one you’re compacting.

2.0 MDB

Creates a Microsoft Access version 2.0 .mdb file with the same encryption status as the one you’re compacting.
