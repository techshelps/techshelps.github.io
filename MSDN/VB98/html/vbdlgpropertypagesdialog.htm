Property Pages Dialog Box

   

Allows you to change a control's properties at design time.

Dialog Box Options

Tabs

Visual Basic creates a tabbed dialog box that acts like a form by writing code to handle updating property values when a user changes values in the control.

You can add Property Pages to your project using the Add Property Pages command on the Project menu.

OK

Validates the Property Pages settings and closes the Property Pages dialog box.

Apply

Validates the Property Page without closing the dialog box.
