ReportWidth Property

      

Returns or sets the width of the report, in twips.

Syntax

object.ReportWidth[= integer]

The ReportWidth property syntax has these parts:

Part Description
object Required. An object expression that evaluates to an object in the Applies To list.
integer Optional. A numeric expression that specifies the width of the report in twips.

Remarks

The DataReport object's Height, Width, Top, and Left properties determine the print preview window's size, while the ReportWidth property values specifies the actual report width.
