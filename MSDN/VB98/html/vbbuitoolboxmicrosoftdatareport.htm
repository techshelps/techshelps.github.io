Data Report Designer Controls (Microsoft Data Report)

Controls that you can use with your Microsoft Data Report.

Note   Although the Data Report designer controls appear in the Visual Basic toolbox, they do not work the same way as their analogous Visual Basic controls. These controls can be used only in the Microsoft Data Report.

At design time, you can change the properties of the controls in the Properties window. You can change the:

  • ForeColor and BackColor

  • Caption and text

  • Fonts

  • Size

  • Location and alignment

  • Border color and Border style

Data Report Designer Controls

   Pointer

The only item in the Microsoft Data Report Toolbox that doesn't draw a control. When you select the pointer, you can only resize or move a control that has already been drawn on a report.

   rptLabel

Allows you to have text that cannot change, such as a caption under a graphic. You cannot use the Label control to display the contents of a field.

   rptTextBox

Bound to a field in a database. Set the field using the DataField property after setting the DataMember property. The control is always in multiline mode.

   rptImage

Displays a graphical image from a bitmap, icon, or metafile on your report.

   rptLine

Used to draw a variety of line styles that you can use to separate sections of your report and make it easier to read. Horizontal lines appear as HTML <HR> tags when exporting to an HTML file.

   rptShape

Allows you to draw a variety of shapes on your report. You can choose a rectangle, rounded rectangle, square, rounded square, oval, or circle.

   rptFunction

Calculates and displays figures using one of four built-in functions: Sum, Average, Minimum, Maximum, Row Count, Value Count, Standard Error, and Standard Deviation.
