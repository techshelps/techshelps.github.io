Delete WebItem Dialog Box

Displays a list of all the HTML tag attributes that have been connected to the webitem you have chosen to delete. If you proceed to delete the webitem, the link between each attribute and the webitem will be disconnected.

Dialog Box Options

Template

Lists the template to which each tag attribute is connected.

Tag

Lists the name of each HTML tag attribute that currently references the webitem.

Navigate

Moves you to the currently selected attribute in the designer window.

OK

Deletes the webitem, removing the link between it and all of the listed attributes.
