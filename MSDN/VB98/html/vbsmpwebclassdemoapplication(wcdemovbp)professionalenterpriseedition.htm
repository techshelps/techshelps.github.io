Webclass Demo Application (wcdemo.vbp), Professional and Enterprise Edition

The Webclass Demo (wcdemo.vbp) sample application demonstrates key concepts involved in building an IIS application. See "Developing IIS Applications," in the Building Internet Applications book of the Component Tools Guide for more information on the concepts in this sample.

File Description
Explorer.bmp An image file that is added to the start page for the application in response to a webclass event.
Page11.htm The HTML page that acts as the basis for the webclass's template.
Wcdemo.vbp Main project for the sample application.
Wcdemo_WebClass1.asp The Active Server Page that hosts the webclass on the server, generates its run-time component, and launches the first event in its lifecycle.
Webclass1.dsr Designer for the webclass in the application. The designer contains information about the templates, events, and event procedures associated with the application.
Webclass1.dsx A binary version of the application's designer.

To Run

From the Visual Basic File menu, choose Open Project and select the wcdemo.vbp file. Compile, then press F5 or choose Start from the Run menu to run the project.
