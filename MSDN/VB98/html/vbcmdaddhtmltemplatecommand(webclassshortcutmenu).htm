Add HTML Template Command (Webclass Shortcut Menu)

Displays the Add HTML Template dialog box, from which you can select an HTML template file to insert into the webclass. The template and all of the attributes of HTML tags within it that are capable of launching server requests are displayed in the designer window.

Available only when you select the webclass, the HTML Template WebItems folder, or the Custom WebItems folder in the Treeview panel of the Webclass designer.
