Parameters Tab (Command Properties Dialog Box)

The Command Properties dialog box is divided into six tabs. The General, Parameters, and Advanced tabs are used for setting properties of the Command and Parameter objects, while the Relation, Grouping, and Aggregates tabs are used for setting properties that are specific to Command hierarchies.

On the Parameters tab, add to a Command object's Parameters collection.

Parameters

This list box contains all the Parameter objects that are associated with the Command object. Select a parameter and edit its properties, as appropriate.

Parameter Properties

This frame contains the control group with which you can change the property values of the selected Parameter object. This is disabled when no Parameter object is selected.

Name

The name of the Parameter. It is recommended that you change this to a more meaningful, unique name.

Direction

Specifies whether the parameter is an input or output parameter, or both, or if the parameter is the return value from the procedure. Select a value from combo box: Input, Output, Input and Output, Return Value, or Unknown.

Data Type

Specifies the data type to which a Data Environment Parameter object is converted.

Precision

The maximum number of digits, or precision, of the Data Environment Parameter object.

Note   This applies to numeric fields and is disabled when the parameter is of a non-numeric type.

Scale

The maximum number of digits to the right of the decimal point of a Data Environment Parameter object.

Note   This only applies to numeric fields and is disabled when the parameter is of a non-numeric type.

Size

The maximum size, in bytes, of the Parameter object.

Host Data Type

The data type that is used when the Parameter object is referenced by the host application. Changing the host data type affects the data type used in building the typelib information for the host.

Required

When selected, this indicates that a parameter value is required when executing the Command object.

Value

A value for the Parameter object. The Data Environment uses this value for design-time data binding, and run-time Command object execution.
