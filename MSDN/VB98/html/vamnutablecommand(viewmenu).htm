Table Command (View Menu)

   

Displays a submenu that enables you to collapse and expand the selected window in the Diagram pane. Used in the Database, Query, and View designers.

Column Properties

Shows a set of properties for each column in the table. You can add, edit, and delete columns in this view. New tables inserted into a database diagram are displayed in this view.

Column Names

Shows the name of each column in the table. You can edit and delete columns in this view. Existing tables that are added to a database diagram are displayed in this view.

Keys

Shows the name of each column in the table that participates in a primary key, foreign key, or unique constraint. You can rename or delete columns in this view.

Name Only

Shows the table name and, if applicable, an asterisk to indicate that the table was modified.

Custom

Shows the table as defined by your custom view. You can use the Modify Custom View command in the Diagram menu to pick the column properties you want to display for all custom view tables.
