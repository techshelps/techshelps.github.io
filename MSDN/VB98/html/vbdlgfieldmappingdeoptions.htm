Field Mapping Tab (Options Dialog Box)

The Options dialog box enables you to change the Data Environment designer's global settings. The properties are divided between two tabs: General and Field Mapping.

On the Field Mapping tab, change the association of a control with a data type. This information is used when a Command or Field object is dragged from the Data Environment designer and dropped onto a form or report.

Default Control Association

This frame contains the control group that specifies drag information.

Show all data types

If selected, all specific field types display in the list, in addition to the field type categories. For example, the Category/Data Type contains both general and detailed information such as Binary, adBinary, adLongBinary, adVarBinary, Boolean, and so forth, and Control shows TextBox, <Use default>, <Use default>, <Use default>, CheckBox, and so forth.

If not selected (the default), only the field type categories display in the box. For example, the Category/Data Type contains general information such as Binary and Boolean, while Control shows TextBox and CheckBox.

Category/Data type

Once an item is selected in the Category/Data Type and Control list, this shows the field type category or specific field data as read-only data.

Control

Once an item is selected in the Category/Field Type and Control list, this shows the control associated with the selected field type category or specific field data. The list contains all ActiveX controls available on your system as well as a base set of intrinsic controls. You can select a control from this list.

Drag and Drop Field Captions

If selected, when a Command or Field object is dragged to a form, a label control is automatically created next to it. The caption is specified in the Caption field from the General tab of the Field Properties dialog box.

If not selected, no label control is created. The type of control created is based on the Caption field type; therefore, you can change the type of control created for the captions.
