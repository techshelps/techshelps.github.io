Make Project Group Dialog Box

   

Allows you to choose the available projects that you want to build.

Dialog Box Options

Project List

Lists the projects that are available. You can choose as many projects as you want to build together.

Use Default Build Options

When checked, uses the options set in the Make tab of the Project Properties dialog box. If you have not changed the options, Visual Basic will use the default values and the project name for the build file names.

Build

Builds the project files.
