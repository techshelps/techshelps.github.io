Window Menu (VisData)

Arranges or activates windows.

Use the items on this menu to arrange all of the child windows or icons, or to make a specific  child window active.

Commands

Tile

Tiles the open windows so that you can see them all at one time.

Cascade

Arranges the windows in a step-like manner with their title bar  visible so you can click the window you want.

Arrange Icons

Arranges the icons so that all are visible.

Windows Lists the open windows.
