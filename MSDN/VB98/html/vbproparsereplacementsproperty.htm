RescanReplacements Property

      

Sets or returns a value that determines whether to rescan for tokens contained within replacement text. You will always scan once, and if RescanReplacement=TRUE, then replacement text that contains tokens will be continually scanned until there are no tokens left.

Syntax

object.RescanReplacements[=boolean]

Part Description
object An object expression that evaluates to an object in the Applies To list.
boolean A Boolean expression that indicates whether or not to rescan tokens within replacement text.

Remarks

See the ProcessTag event, which allows you to replace specified strings with replacement text.
