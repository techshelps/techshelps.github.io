Insert Stored Procedures Dialog Box

When Insert Stored Procedures is selected, the Insert Stored Procedures dialog box appears and allows you to select one or more stored procedures to insert into your DataEnvironment object.

To insert stored procedures into your DataEnvironment object, use the arrows to move one or more stored procedures from the Available list to the Add list, and then click Insert. A new Command object is created for each selected stored procedure. The name of the Command object defaults to the name of the stored procedure.

Connection

The Connection object from which to display the available stored procedures.

Available

Contains all available stored procedures from the selected Connection object.

Arrows

Use > to move the selected stored procedure from the Available list to the Add list. This is disabled if no items are listed in the Available list.

Use >> to move all available stored procedures from the Available list to the Add list. This is disabled if no items are listed in the Available list.

Use < to move the selected stored procedure from the Add list to the Available list. This is disabled if no items are listed in the Add list.

Use << to moves all stored procedures from the Add list to the Available list. This is disabled if no items are listed in the Add list.

Add

The stored procedures listed here are added to the current DataEnvironment object once Insert is clicked.

Insert

Inserts the stored procedures in the Add list into the current DataEnvironment object.

Close

Closes the Insert Stored Procedures dialog box.

Note   If you have not moved stored procedures to the Add list and clicked Insert, no stored procedures are added to your DataEnvironment object. In this case, clicking Close acts as a cancel operation.
