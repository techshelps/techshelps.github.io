GetDataMember Event

      

Occurs when a data consumer requests a new data source.

Syntax

Private Sub object_GetDataMember(DataMember As String, Data As Object)

The GetDataMember event syntax has these parts:

Part Description
object An object expression that evaluates to an object in the Applies To list.
DataMember A string containing the name of the data member to be bound as a data source.
Data An object reference to an ADO RecordSet object or a class which implements the OLEDBSimpleProvider interface.

Remarks

The GetDataMember event is available only when an object’s DataSourceBehavior property is set to vbDataSource or vbOLEDBProvider (for classes only). You can add code to the GetDataMember event procedure to initialize a data member or to select from multiple data members within an object.
