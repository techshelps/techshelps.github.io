Login Dialog Box (VisData)

   

Allows you to log on as a different user.

Use when:

  • The initial CreateWorkspace call fails due to an invalid logon (The user administrator must have a password).

  • You choose Workspace from the File menu.

You must enter a valid user name and password combination or choose Cancel. If you cancel the opening instance of this dialog, VisData will close. If you cancel after choosing Workspace from the File menu, you will remain logged on.
