'Item' is in binary format and cannot be loaded into VB

Binary format files created with versions of Visual Basic prior to version 5.0 cannot be loaded without being converted. Only Visual Basic versions 2.0 and 3.0 support binary format file conversion. If you have access to a copy of Visual Basic 2.0 or 3.0, you can load the file and save it as text, then load the file into the newer version of Visual Basic.
