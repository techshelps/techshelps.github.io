Displaying Property Qualifiers

[This is preliminary documentation and subject to change.]

Property qualifiers can only be displayed for non-system properties.

    To display property qualifiers
  1. Select the class in the Class Explorer, and click the Properties tab in the Class Viewer.
  2. Double-click the property whose qualifiers you want to display.

    —or—

    Right-click the property whose qualifiers you want to display, and select Property qualifiers from the menu that appears.

    In both cases, the Qualifiers window appears. If the class has no property qualifiers, the window is blank.

To close the Qualifiers window, click the OK or Cancel button.
