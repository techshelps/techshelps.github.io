Browsing for a Namespace

[This is preliminary documentation and subject to change.]

    To browse for a namespace:
  1. Click the Browse for Namespace button. The Browse for Namespace dialog box appears:

  1. By default, the Machine Name box shows the local machine. To select a namespace on another machine, type that machine name in the Machine Name box and press Enter.

    —or—

    Click the Network Neighborhood button, select the remote machine from the list that appears, and then click OK.

    The WBEM Login dialog box appears.

  2. Enter the appropriate login information (see Opening the ActiveX Applications for instructions), and click OK. This returns you to the Browse for Namespace dialog box.
  3. In the Browse for Namespace dialog box, click the plus sign next to the root folder to display the namespaces on the selected machine.
  4. Click the namespace you want to use, then click OK. This returns you to the CIM Studio where the class inheritance tree for the selected namespace appears.
