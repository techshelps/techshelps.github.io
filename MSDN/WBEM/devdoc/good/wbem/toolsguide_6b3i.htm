To View a Class Definition

[This is preliminary documentation and subject to change.]

Select a class from the standard or graphic views to display the properties of that class. The following screen shows the class definition for the ComputerSystem class.

The Parent <class> represents the class from which this class is derived. In this case, it is the "System" class. Selecting this link takes you to an HTML page that provides a description of the parent class.

Select the link to the namespace that you are currently browsing. In this example, \\<MachineName>\root\default allows you to jump back to the standard view for this namespace.
