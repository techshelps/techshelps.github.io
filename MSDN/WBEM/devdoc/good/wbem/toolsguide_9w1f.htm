Wizards

[This is preliminary documentation and subject to change.]

The three icons above the Class Viewer invoke the following wizards:

C++ Code Generator Wizard generates framework provider C++ code for a selected class. See C++ Code Generator Wizard.
MOF Generator Wizard inserts selected class definitions and/or instances from the database into a text file in the MOF syntax. See MOF Generator Wizard.
MOF Compiler Wizard imports class definitions or instances described in a MOF file into the current database. See MOF Compiler Wizard.

When selected, each wizard opens a series of screens that guide you through the operation.
