Adding Instances of a Class

[This is preliminary documentation and subject to change.]

    To add an instance of a class
  1. In the Class Explorer, select the class for which you want to create an instance.
  2. Click the Create instance button in the Class Viewer. The viewer displays data for the new instance, and the Save button becomes active.
  3. Enter values for the key property fields. For information on editing properties, see Editing Class Properties.
  4. When you have finished editing the key fields, click the Save button.

Note  You cannot add instances of the following classes:

  • Classes that do not have key fields (abstract classes)
  • Classes that have dynamic instance providers
