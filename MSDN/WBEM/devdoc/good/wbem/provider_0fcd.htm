Registering the Provider with CIMOM

[This is preliminary documentation and subject to change.]

    To register the provider with CIMOM
  1. Open WBEM CIM Studio and double-click the MOF Compiler icon.
  2. In the first screen, select Compile a MOF file then click the Next button.
  3. In the next screen, type <drive>\Prov\myprov.mof or use the Browse button to select the file. Click the Next button.
  4. Click the Finish button on the next screen that appears. When compilation is complete, a confirmation message appears. Click OK to dismiss the message.
