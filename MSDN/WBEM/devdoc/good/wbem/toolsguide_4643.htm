Object Browser Functions

[This is preliminary documentation and subject to change.]

This section describes the viewing and editing functions you can perform in the WBEM Object Browser.

Note  If an error occurs while you are editing object properties or qualifiers (for example, if you enter a property type and value that are incompatible), a message box appears with a brief description of the problem. Click Advanced to view the error object's properties. The property display contains more details on the problem that occurred.
