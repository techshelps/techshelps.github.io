logical consumer
An instance of a class derived from the __EventConsumer system class. The properties of the class describe the action to be taken upon receipt of a notification. A component of the permanent consumer architecture. Also see permanent consumer, event consumer provider interface, event consumer provider object, event filter, consumer-filter binding, and sink.
