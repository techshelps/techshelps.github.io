Logging

[This is preliminary documentation and subject to change.]

    To configure logging
  1. Select one of the following options:

    Disable logging - turns off logging.

    Error logging - sends only error messages to the log files.

    Verbose logging - sends general tracing messages as well as error messages to the log files.

  2. In the Maximum log file size (bytes) field, enter the maximum size of the log files.
  3. In the Logging directory field, enter the directory where the log files reside. The default is the \Logs subdirectory of the WBEM installation directory.
  4. Click Apply to save changed information and remain in the WBEM Administrator.

    —or—

    Click OK to save changed information and close the WBEM Administrator.

Changes to the logging settings take effect within one minute of the change.
