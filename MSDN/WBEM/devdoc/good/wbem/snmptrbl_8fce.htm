Installation and Registration

[This is preliminary documentation and subject to change.]

The following topics describe errors that can occur during installation and registration. These errors are associated with the SNMP device namespace configuration.

  • Dynamic class provider not registered in the SNMP device namespace.
  • Dynamic instance provider not registered in the SNMP device namespace.
  • Dynamic class provider incorrectly registered in the SNMP device namespace.
  • Dynamic instance provider incorrectly registered in the SNMP device namespace.
  • No classes loaded into the SNMP device namespace.
  • No classes loaded into the SMIR for an SNMP dynamic class provider-backed SNMP device namespace.
