CIM Studio Functions

[This is preliminary documentation and subject to change.]

This section describes the functions that can be performed in the CIM Studio.

Note  If an error occurs while you are editing class properties or qualifiers (for example, if you enter a property type and value that are incompatible), a message box appears with a brief description of the problem. Click the Advanced button to display the error object's properties. The property display contains more details on the problem that occurred.
