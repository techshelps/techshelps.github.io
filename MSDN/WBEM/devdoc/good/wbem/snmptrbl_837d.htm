The set of instances returned by the SNMP providers changes intermittently.

[This is preliminary documentation and subject to change.]

Further description:

The SNMP instance provider may use the services of the SNMP class provider to supply class definitions that define the SNMP MIB schema definitions used to access an SNMP device. The set of classes returned by the SNMP class provider can change intermittently due to network connectivity.
