Deleting Instances of a Class

[This is preliminary documentation and subject to change.]

    To delete an instance of a class
  1. Open the instance table for the desired class. For instructions, see Displaying an Instance Table.
  2. Click the blank box to the left of the instance you want to delete. This selects the entire instance entry.
  3. Click the Delete instance button in the Class Viewer. A confirmation message appears. Click OK to delete the instance. The instance is deleted from the table.
