The SNMP agent receives an SNMP protocol data unit but does not return a response.

[This is preliminary documentation and subject to change.]

Further description:

The SNMP agent does not respond to an SNMP protocol operation because the SNMP protocol data unit is not a valid authenticated packet.

Solutions:

  • For the SNMPv1 protocol operations GET and GET-NEXT, make sure that the AgentReadCommunityName qualifier of the SNMP device namespace instance or SNMP-compatible class is valid for that SNMP agent device.
  • For the SNMPv1 protocol operation SET, make sure that the AgentWriteCommunityName qualifier of the SNMP device namespace instance or SNMP-compatible class is valid for that SNMP agent device.
