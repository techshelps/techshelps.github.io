Editing Object and Property Qualifiers

[This is preliminary documentation and subject to change.]

    To edit object and property qualifiers
  1. Select the object in the Object Explorer.
  2. Display the Qualifiers (object or property) window you want to edit.
  3. Click the field you want to change.
  4. If a down arrow appears when you select the field, click the arrow and choose a value from the list that appears.

    If no down arrow appears, type the new value in the field.

    Click the I, C, and O fields to select or cancel these settings.

    The Origin field is read-only. It indicates where the qualifier originated: local (for qualifiers defined in the current object) or propagated (for qualifiers propagated from another object).

    When you start typing, the text turns blue. The Save button becomes active to indicate that the value has not been committed yet.

  5. When you have finished editing, click the OK button. This closes the Qualifiers window.
  6. In the Object Viewer, click the Save button. A confirmation message appears. Click Yes to save the changes.
