Displaying a Class Inheritance Tree

[This is preliminary documentation and subject to change.]

By default, the CIM Studio sets the namespace to root\DEFAULT. This is the default namespace on the local machine. You can view the class inheritance tree for any namespace in your system or on a network by specifying the namespace path in one of the following ways:

Specifying the Namespace in the Classes in Box

Selecting the Namespace from the History List

Browsing for a Namespace
