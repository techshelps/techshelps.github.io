SNMP Providers

[This is preliminary documentation and subject to change.]

The Web-Based Enterprise Management (WBEM) SDK includes two Simple Network Management Protocol (SNMP) providers:

  • An instance provider that enumerates all instances of a particular class in a namespace.
  • A dynamic class provider that enumerates all classes in a particular namespace and generates class definitions.

A WBEM namespace can be created that represents an SNMP Management Information Base (MIB) view associated with an SNMP agent. By creating such namespaces and registering the SNMP providers in each namespace, WBEM applications can access both static and dynamic SNMP information in a Common Information Model (CIM) schema.
