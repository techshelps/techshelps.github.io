Displaying Property Qualifiers

[This is preliminary documentation and subject to change.]

Property qualifiers can only be displayed for non-system properties.

    To display property qualifiers
  1. Select the object in the Object Explorer, and click the Properties tab in the Object Viewer.
  2. Double-click the property whose qualifiers you want to display.

    —or—

    Right-click the property whose qualifiers you want to display. Select Property qualifiers from the menu that appears.

    In both cases, the Qualifiers window appears. If the object has no property qualifiers, the window is blank.

To close the Qualifiers window, click the OK or Cancel button.
