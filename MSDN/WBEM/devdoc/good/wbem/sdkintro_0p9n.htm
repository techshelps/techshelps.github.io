Using the WBEM SDK

[This is preliminary documentation and subject to change.]

The Web-Based Enterprise Management (WBEM) Software Development Kit (SDK) contains the header files and Interface Definition Language (IDL) files for creating management applications, providers, and schema extensions. There are four IDL files that are included in the WBEM SDK: Wbemcli.idl, Wbemtran.idl, Wbemprov.idl, and Wbemdisp.idl.

The WBEM SDK contains:

  • Applications for viewing and changing data in the Common Information Model Object Manager (CIMOM) repository
  • Object providers for accessing data and events
  • Open Database Connectivity (ODBC) adapter
  • Managed Object Format (MOF) language and compiler
  • Sample Component Object Model (COM) programs
