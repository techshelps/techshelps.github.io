CIM Object Manager (CIMOM) Reference

The CIM Object Manager (CIMOM) provides access to information about objects in the management environment. Through CIMOM, applications can query for and make changes to static information in the CIMOM repository and dynamic information maintained by object providers.

The CIMOM Reference provides discussions of the following core features of CIMOM:

  • C/C++ APIs, a complete listing of the COM interfaces that are used by C or C++ management applications and object providers to access CIMOM services.
  • Visual Basic APIs, a complete listing of the interfaces that are used by Visual Basic and VBA applications to access CIMOM services.
  • System Class Reference, a complete listing of the classes that are defined by CIMOM.
  • System Property Reference, a complete lislting of the properties that are supported by CIMOM.
  • Standard Qualifier Reference, a complete listing of the qualifiers that are supported by CIMOM.
  • Appendixes, a description of the command line syntax used by CIMOM and the MOF compiler and the Backus Naur Form (BNF) syntax for MOF, SQL1, and object paths.
