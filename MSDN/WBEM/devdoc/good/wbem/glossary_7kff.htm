Kerberos
A security mechanism that provides authentication, data integrity, data privacy, and mutual authentication.
key
A value used to identify an object within the scope of a namespace, such as a drive letter in the scope of a system. A property which is a key has the key qualifier set to true.
key property
A property that provides a unique value. Key properties are designated by the Key qualifier, a standard qualifier defined by WBEM. All classes that define multiple instances must specify a key.
key qualifier
A qualifier that must be attached to every property in a class that serves as part of the key for that class.
