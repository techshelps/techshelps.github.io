The SNMP provider returns an error result because the qualifier values of the SNMP device namespace instance or SNMP-compatible class do not match the expected type or values.

[This is preliminary documentation and subject to change.]

Further description:

The SNMP provider uses a number of qualifiers to determine the transport context used to communicate with an SNMP agent device. All mandatory qualifiers must be present and all optional qualifiers that are present must be defined using the correct type and values.

Solution:

Verify that the namespace qualifier values are of the expected types and values. See Creating an SNMP Namespace for information on the qualifier values and types.
