Editing Class or Property Qualifiers

[This is preliminary documentation and subject to change.]

    To edit class or property qualifiers
  1. Open the Qualifiers (class or property) window you want to edit. For instructions, see Displaying Class Qualifiers or Displaying Property Qualifiers.
  2. Click the field you want to change.
  3. If a down arrow appears when you select the field, click the arrow and choose a value from the list that appears.

    If no down arrow appears, type the new value in the field.

    Click the I, C, and O fields to select or cancel these settings.

    The Origin field is read-only. It indicates where the qualifier originated: local (for qualifiers defined in the current object) or propagated (for qualifiers propagated from another object).

    When you start typing, the text turns blue. This indicates that the value has not been committed yet.

  4. When you have finished editing, click the OK button to close the Qualifiers window.
  5. In the Class Viewer, click the Save button. A confirmation message appears. Click Yes to save your changes.
