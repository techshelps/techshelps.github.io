Editing Class Properties

[This is preliminary documentation and subject to change.]

    To edit a property field
  1. Select the class in the Class Explorer, and click the Properties tab in the Class Viewer.
  2. Click the field you want to change.
  3. If a down arrow appears when you select the field, click the arrow and choose a value from the list that appears.
  4. —or—

    If no down arrow appears, type the new value in the field.

    When you start typing, the text turns blue, and the Save button becomes active. This indicates that the value has not been committed yet.

    If you edit a Value field that contains an array, a dialog box appears that shows each value in the array. Edit the values you want to change. Click OK to return to the Properties page.

    If the property type is an object, the Value field contains either <empty> (which represents NULL) or an Object button. To change this field, do one of the following:

    • Click the Object button to open the Embedded Object dialog box. You can edit properties and qualifiers from this box in the same way as in the Class Viewer Properties page. Click OK to return to the Class Viewer.
    • Right-click the Object button, and select Set to <empty>.
    • Right-click the <empty> value, and select Create object. This opens the Embedded Object dialog box. Enter the name of the class to which the object belongs. The Properties page for that class appears. You can edit properties and qualifiers the same way as in the Class Viewer Properties page. Click OK to return to the Class Viewer.
  5. When you have finished editing, click the Save button. A confirmation message appears. Click Yes to save the changes. After the changes are saved, the text turns black.
