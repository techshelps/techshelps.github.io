Changing Permissions for Users or Groups

[This is preliminary documentation and subject to change.]

    To change permissions for a user or group
  1. Double-click the user/group you want to change.

    —or—

    Select the user/group you want to change, then click the Edit button.

    The WBEM User File and Registry Permissions dialog box appears.

  2. Make the desired changes to the WBEM File and Registry Permissions, then click OK to return to the WBEM Security for Windows NT dialog box.
  3. Click Apply to save the new data and remain in the WBEM Administrator.

    —or—

    Click OK to save the new data and close the WBEM Administrator.
