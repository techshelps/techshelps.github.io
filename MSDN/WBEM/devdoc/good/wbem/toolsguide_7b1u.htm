WBEM User Manager

[This is preliminary documentation and subject to change.]

The WBEM User Manager allows you to create authorized WBEM users, and to set their access privileges. Use this application to manage user authentication and access to CIM objects for WBEM. (On Windows NT systems, use the WBEM Administrator tool to control access to WBEM installation directories and the registry.)
