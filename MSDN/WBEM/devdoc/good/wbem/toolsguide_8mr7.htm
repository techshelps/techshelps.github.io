Viewing Class Definitions and Instances

[This is preliminary documentation and subject to change.]

By default, the HTML Viewer sets the namespace to root\default, which is the default namespace on the local machine. You can view the class inheritance tree for any namespace in your system or on a network by specifying the namespace path.

For example, specifying \\Sapper7\root\default displays the class inheritance tree for the default namespace on the machine Sapper7. If no machine is specified, the default is the local machine. You can specify a chain of namespaces, separating each with a backslash:

\\<machine>\root\<namespace>\<namespace>\<namespace>

The HTML Viewer can also display a table of all instances of a class. You can also define and display custom views of instances.
