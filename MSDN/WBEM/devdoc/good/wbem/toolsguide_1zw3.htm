Displaying Properties of Regular Objects

[This is preliminary documentation and subject to change.]

    To display properties of regular objects
  1. Select the object in the Object Explorer.
  2. Click the Properties tab in the Object Viewer.

The Properties page displays the properties for a selected regular object. Properties attached to an object by the system are preceded by a double underscore (for example, __SERVER). Nonsystem properties are either defined by the user or inherited from parent classes.
