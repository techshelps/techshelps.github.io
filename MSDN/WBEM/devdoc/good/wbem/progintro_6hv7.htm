Specifying Object Paths

[This is preliminary documentation and subject to change.]

Web-Based Enterprise Management (WBEM) object paths are used to access:

  • Namespaces
  • Classes
  • Instances

Each object on the system has a unique path that identifies it locally or over the network. Object paths are conceptually similar to Universal Resource Locators (URL). Object paths can be fully qualified or relative to a certain location, such as the local machine or a particular namespace on the local machine.

To view the Backus-Naur Form definition for object paths, see Object Path BNF.
