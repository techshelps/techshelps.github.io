WBEM Security for Windows NT

[This is preliminary documentation and subject to change.]

On Windows NT systems, use the WBEM Security for Windows NT tab to set security for the WBEM installation directories and the registry.

The WBEM Security for Windows NT tab shows the directory where WBEM is installed and the file system used on the drive where WBEM is installed. This tab also lists the currently defined users, the group to which they belong, and their permissions to access WBEM files and the registry. For example:

Note  Directories must reside on an NTFS drive in order to be secured. Installations with file allocation table (FAT) file systems cannot be secured.
