DWbemContext.DeleteAll

[This is preliminary documentation and subject to change.]

The DWbemContext.DeleteAll method removes all named context values from the current object, thus emptying it.

DeleteAll( ) As Long

Parameters

None.

Return Values

WBEM_E_FAILED Unspecified error.
WBEM_NO_ERROR Success.
