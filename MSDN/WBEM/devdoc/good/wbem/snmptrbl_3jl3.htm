SNMP MIB Mapping

[This is preliminary documentation and subject to change.]

The following topics describe errors associated with SNMP MIB mapping.

  • The SNMP provider returns an error for a class because some mandatory property qualifiers are not present or some qualifiers are not of the correct type and value.
  • The SNMP provider returns an error for a class because some mandatory qualifiers are not present within the class definition or some qualifiers are not of the correct type and value.
  • The SNMP provider returns an error for a class because some properties of a CIM MOF class definition do not contain valid values for some property qualifiers.
  • The SNMP provider returns a subset of the properties present for a class definition because the object_identifier qualifier value is incorrect with respect to the SNMP MIB definition.
  • The SNMP provider returns an error for a CIM MOF class because none of the properties of the class definition have read access.
  • The SNMP provider returns an error for a CIM MOF class because none of the properties of a given CIM MOF class definition have write access.
  • The SNMP provider returns an error for a CIM MOF class because some properties of a given CIM MOF class definition are marked as keyed and no associated key_order qualifier is present, or the key_order values are invalid.
