Deleting Class Properties

[This is preliminary documentation and subject to change.]

    To delete a property
  1. Select the class in the Class Explorer, and click the Properties tab in the Class Viewer.
  2. Click the blank box to the left of the property you want to delete. This selects the entire property entry.
  3. Press the Delete key. A confirmation message appears. Click OK to delete the property. This activates the Save button in the Class Viewer.
  4. Click the Save button. A confirmation message appears. Click Yes to confirm the deletion.
