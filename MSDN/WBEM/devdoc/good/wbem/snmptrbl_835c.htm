The AgentAddress qualifier of the SNMP device namespace instance or SNMP-compatible class is an IP address but the operation hangs for a period of time before unsuccessfully returning.

[This is preliminary documentation and subject to change.]

Further description:

If the AgentAddress qualifier has been entered incorrectly, the SNMP providers assume the value is a resolvable host name. The providers then unsuccessfully attempt to resolve the host name.

Solution:

Make sure the AgentAddress qualifier is really a dotted decimal IP address.
