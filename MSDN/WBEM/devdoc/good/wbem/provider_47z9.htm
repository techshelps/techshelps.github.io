Generating Skeleton Code

[This is preliminary documentation and subject to change.]

    To generate skeleton code for the class
  1. In the left frame, make sure the check box next to the MyClass class is checked. Double-click the CPP Generator icon to run the C++ Code Generator Wizard.
  2. Click the Next button on the first screen. In the second screen enter A simple class in the Description field.
  3. Click the Next button. In the next screen, change the provider name to MyProv. Enter My first provider as the provider description. Change the directory to <drive>:\Prov (or use the Browse button to select this directory).
  4. Click the Finish button. The skeleton code files are generated. A confirmation message appears. Click OK to dismiss the message.
