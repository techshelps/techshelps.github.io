System Requirements for the ActiveX Applications

[This is preliminary documentation and subject to change.]

The system requirements for running the ActiveX applications are as follows:

  • Microsoft® Windows NT® version 4.0 or higher or Windows® 95. ActiveX™ components are not supported on Windows NT® 3.51.
  • Internet Explorer version 3.01 or higher. Note that the original version of Windows 95 includes a version of Internet Explorer that is labeled 4.0, but is actually an older version. You must update Internet Explorer if you are still using this version. You can download the latest version of Internet Explorer from http://www.microsoft.com.
  • Screen resolution is no less than 800x600.
  • VBScript version 2.0 must be installed on your machine. You can download VBScript from http://www.microsoft.com/vbscript.
