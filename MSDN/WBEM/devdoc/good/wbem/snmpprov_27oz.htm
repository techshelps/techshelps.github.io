RFC MIBs

[This is preliminary documentation and subject to change.]

The WBEM SDK includes a set of SNMPv1 Request for Comments (RFC) MIBs (including MIB-II definitions, RFC 1213) that have been mapped to CIM schema definitions for use with the SNMP providers. The mapped information is in two forms:

  • Static MOF files

    These files are located in <Wbem-directory>\Snmp\Mofs\Snmpv1\Rfcs, where <Wbem-directory> is the directory in which WBEM is installed. To use these files for static class operation, they must first be loaded into CIMOM.

  • SMIR MOF files

    These files are located in <Wbem-directory>\Snmp\SMIR_MOFS\Snmpv1\Rfcs. To use these files for dynamic class operation, they must first be loaded into the SNMP Module Information Repository (SMIR) which is an SNMP schema database.

Note that the MIB text files are also included with the SDK in the <Wbem-directory>\Snmp\Mibs\Snmpv1 directory.

For instructions on loading MOF and SMIR MOF files into CIMOM or the SMIR, see Loading Static MOF Files and SMIR MOF Files.
