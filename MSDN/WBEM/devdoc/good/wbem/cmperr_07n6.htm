Fatal Error 102

[This is preliminary documentation and subject to change.]

<102, Fatal>:"Invalid argument on command line after <last correct argument>"
Command-line syntax error. There are some extra arguments or an invalid switch on the command line. The <last correct argument> parameter refers to the last argument that was read correctly on the command line, and can possibly be smi2smir, which is the name of the executable file for the SNMP module information compiler.
