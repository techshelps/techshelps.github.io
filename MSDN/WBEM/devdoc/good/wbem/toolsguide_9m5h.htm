Applications Guide

[This is preliminary documentation and subject to change.]

The Web-Based Enterprise Management (WBEM) Software Development Kit (SDK) applications include:

  • ActiveX™ Applications:
    • WBEM Object Browser

      The Object Browser is primarily used to view objects in a WBEM Common Information Model Object Manager (CIMOM) database. You can also edit property values, edit object and property qualifiers, and run operations on objects in the Object Browser.

    • Common Information Model (CIM) Studio

      The CIM Studio is used to view classes in a CIMOM database. You can also perform editing functions on classes, properties, qualifiers, and instances, run operations on classes, generate and compile Managed Object Format (MOF) files, and generate framework provider code in the CIM Studio.

  • WBEM HTML Viewer

    The HTML Viewer is used to view classes, properties, qualifiers, and instances in a CIMOM database. This tool is designed for use by advanced system managers and developers, particularly those writing providers, who do not have access to Microsoft® Internet Explorer with ActiveX™.

  • WBEM Event Registration Viewer

    The WBEM Event Registration Viewer is a graphical interface for viewing event consumer, filter, and timer system class properties and creating instances of them, as an alternative to doing so programmatically.

  • WBEM Event Viewer

    The WBEM Event Viewer displays events for all instances of registered consumers. You can also start the WBEM Event Registration Viewer from the Event Viewer in order to register consumers for events.

  • WBEM User Manager

    The WBEM User Manager allows you to create authorized WBEM users and set their access privileges.

  • WBEM Administrator

    The WBEM Administrator allows you to configure CIMOM database backups, enable and configure logging, and to set security for the WBEM installation directories and the registry.

This document provides a brief description of these applications and how to use them.
