Common Information Model (CIM) Studio

[This is preliminary documentation and subject to change.]

The CIM Studio is designed primarily for use by developers, particularly those who are writing providers, and for advanced system managers. Using the CIM Studio, you can do the following:

  • Display the class inheritance tree (schema) in a specified CIMOM database.
  • Search for a specific class.
  • Display the definition of an existing class (properties, qualifiers, and methods (operations)).
  • Display instances of an existing class (standard or custom views).
  • Extend and modify the existing schema by adding or deleting class definitions.
  • Modify a class by adding, editing, or deleting properties, qualifiers, and methods in a class definition.
  • Add or delete instances of a class.
  • Run the C++ code generator, MOF file generator, and MOF file compiler on selected classes and/or instances.
