Rerooting an Object Tree

[This is preliminary documentation and subject to change.]

When the Object Browser opens, it searches for an instance of the System class or any class derived from System. If found, this class becomes the root of the initial object tree. If no such class is found, the Object Browser displays a dialog box asking for a class name. Type a class name in the dialog box, and click OK. The specified class becomes the root of the initial object tree.
