Adding Class or Property Qualifiers

[This is preliminary documentation and subject to change.]

    To add qualifiers to a class or property
  1. Open the Qualifiers window for the desired class or property. For instructions, see Displaying Class Qualifiers or Displaying Property Qualifiers.
  2. Click the blank Name field, and enter the new qualifier name.
  3. Click the Type field, then click the down arrow and choose a type from the list that appears. When you select the type, a default value may appear in the Value field.
  4. Click the I, C, and O fields to select or cancel these settings.

    The Origin field is read-only. It indicates where the qualifier originated: local (for qualifiers defined in the current object) or propagated (for qualifiers propagated from another object).

  5. If you want a value other than the default in the Value field, click the field and enter the desired value. When you select this field and begin typing, a list box may appear from which you can choose the appropriate value.
  6. When you have finished editing, click the OK button to close the Qualifiers window.
  7. In the Class Viewer, click the Save button. A confirmation message appears. Click Yes to save the changes.
