HAVING COUNT
A clause in the WQL SELECT statement used to set a minimum limit on the number of discrete events that produce an aggrevate event, reducing overall event traffic.

Also see GROUP BY, ISA, WHERE, WITHIN, and the SELECT Statement HAVING COUNT Clause.
