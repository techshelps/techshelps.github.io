The SNMP provider returns a subset of the properties present for a given class definition because some entity is dropping a percentage of SNMP packets between the source and destination.

[This is preliminary documentation and subject to change.]

Further description:

The SNMP provider is returning a small percentage of NULL properties for a given instance of a class because the SNMP operation and response packets are being dropped between the source and destination.

Solutions:

See the following topics:

  • The SNMP agent does not respond to an SNMP protocol operation because the SNMP operation protocol data units are being dropped before reaching their destination.
  • The SNMP agent does not respond to an SNMP protocol operation because the SNMP response protocol data units are being dropped before reaching their destination.
  • The SNMP agent does not respond to an SNMP protocol operation because the SNMP agent is dropping the SNMP protocol data units.
