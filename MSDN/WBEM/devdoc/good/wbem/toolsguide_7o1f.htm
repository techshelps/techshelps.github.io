Repository Backups

[This is preliminary documentation and subject to change.]

    To set CIMOM repository backup intervals
  1. In the Idle time before attempting repository backup field, enter the amount of time (in minutes) that you want CIMOM to wait between accesses to the repository before considering doing a backup. If this check fails (that is, if the repository is constantly being accessed), no backup occurs.
  2. In the Maximum backup interval (minutes) field, enter the amount of time between repository backups. This setting is checked only if the idle time check succeeds.
  3. Click Apply to save changed information and remain in the WBEM Administrator.

    —or—

    Click OK to save changed information and close the WBEM Administrator.

Changes to backup settings take effect immediately.
