WBEM Event Registration Viewer

[This is preliminary documentation and subject to change.]

The WBEM Event Registration Viewer is designed for developers and system managers. It provides a graphical interface for what may also be done programmatically. The Event Registration Viewer allows you to display and modify the event consumers, filters, and timers for a given namespace and any bindings between filters and consumers.

Using the Event Registration Viewer, you can do the following:

  • View properties of the defined consumer, filter, and timer system classes and instances.
  • Add or delete event consumer instances.
  • Add or delete event filter instances.
  • Add or delete event timer instances.
  • Edit instance properties.
  • Register consumers for events by binding consumer and filters.
