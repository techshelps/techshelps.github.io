Dynamic class provider not registered in the SNMP device namespace.

[This is preliminary documentation and subject to change.]

Solution:

Create an instance of __WIN32Provider using the following instance template:

instance of __WIN32PROVIDER
{
    Configuration = 0;
    Provider = "MS_SNMP_CLASS_PROVIDER";
    ProviderClsId = "{3296BF00-A1FC-11cf-8C8B-00AA00A4086C}";
    MethodSet = 332817;
} ;
