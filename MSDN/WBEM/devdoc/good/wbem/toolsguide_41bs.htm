Specifying the Namespace in the Objects in Box

[This is preliminary documentation and subject to change.]

The format for specifying a namespace in the Objects in box is as follows:

\\<machine>\root\<namespace>

For example, specifying \\EDKDOCS1\root\DEFAULT displays the object tree for the default namespace on the machine EDKDOCS1. If no machine is specified, the default is the local machine. You can also specify a chain of namespaces, separating each with a backslash:

\\<machine>\root\<namespace>\<namespace>\<namespace>
