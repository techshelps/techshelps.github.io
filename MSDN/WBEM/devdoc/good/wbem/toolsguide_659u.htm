WBEM Object Browser

[This is preliminary documentation and subject to change.]

The WBEM Object Browser is designed for use by system managers. The Object Browser displays the object tree in a CIMOM database, showing the relationships (that is, associations) between the objects. You can browse the objects in the database by traversing these relationships in either tree or graphical formats.

Using the Object Browser, you can do the following:

  • Display the object tree contained in a specified CIMOM database.
  • Reroot the object tree.
  • Display properties, methods (operations), and associations for a selected object.
  • Display custom views of object data.
  • Display instances of grouped objects.
  • Display property and object qualifiers.
  • Run operations on a selected object.
