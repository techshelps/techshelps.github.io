Localization Support

[This is preliminary documentation and subject to change.]

CIMOM supports services that enable developers to localize classes created for the Microsoft® Windows NT®, Microsoft® Windows® 95, and Windows 98 platforms. Localization support exists for the following elements of a class:

  • Class name
  • Property name
  • Property value
  • Descriptive help text
  • Qualifier name
  • Qualifier value

Existing classes and instances can be localized after they have been introduced and remain unaffected. Localization services are provided entirely through WBEM classes specified with Unicode MOF files; the CIMOM API is not involved.

For a detailled description of CIMOM's localization support, see Localization Services.
