SNMP Compiler Error Messages

[This is preliminary documentation and subject to change.]

This section describes the various error conditions that can result when the SNMP information module compiler parses a Management Information Base (MIB) module.

The error messages have three severity levels:

  • Fatal errors
  • Warnings
  • Information messages

The severity level is shown in parentheses at the beginning of the error message description.
