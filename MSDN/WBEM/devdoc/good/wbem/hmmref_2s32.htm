IWbemQualifierSet::EndEnumeration

[This is preliminary documentation and subject to change.]

Call the IWbemQualifierSet::EndEnumeration method when you plan to terminate enumerations using IWbemQualifierSet::BeginEnumeration and IWbemQualifierSet::Next . We recommend this call, but it is not required. It immediately releases resources associated with the enumeration.

HRESULT IWbemQualifierSet::EndEnumeration( );

Parameters

None.

Return Values

WBEM_NO_ERROR Success.

See Also

IWbemQualifierSet::BeginEnumeration, IWbemQualifierSet::Next
