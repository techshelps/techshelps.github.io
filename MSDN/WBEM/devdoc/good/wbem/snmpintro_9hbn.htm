Internet Engineering Task Force (IETF) Documents

[This is preliminary documentation and subject to change.]

The following tables list Internet Engineering Task Force (IETF) Request for Comments (RFC) describing SNMPv1 and SNMPv2C. All of these documents can be freely obtained from ftp://ds.internic.net/rfc.
