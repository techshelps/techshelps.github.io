Displaying an Object Tree

[This is preliminary documentation and subject to change.]

By default, the Object Browser sets the namespace to root\DEFAULT, which is the default namespace on the local machine. You can view the object tree for any namespace in your system or on a network by specifying the namespace path in one of the following ways:

Specifying the Namespace in the Objects in Box

Selecting the Namespace from the History List

Browsing for a Namespace

Using the Go to Namespace Function

Note  While the Object Browser runs with any database, a database that has no associations shows only a single object, the root selected by the user. This type of database can be viewed more effectively in the CIM Studio.
