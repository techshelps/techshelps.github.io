Displaying an Instance Table

[This is preliminary documentation and subject to change.]

You can open an instance table for any class in the namespace. An instance table shows all instances of the selected class in the namespace, as well as their common properties.

    To display the instance table
  1. Select a class in the Class Explorer.
  2. Click the Instances button in the Class Viewer. The instance table for the selected class appears. If there are no instances of a selected class, the table displays the message "No Instances Available."

To return to the class definition display, click the Instances button again.
