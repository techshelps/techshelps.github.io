Viewing Namespaces

[This is preliminary documentation and subject to change.]

In CIMOM, namespace objects are used to control the scope and visibility of instances of a particular class. A namespace is a logical data set consisting of classes and all instances that belong to those classes, thus forming the logical database.

Every machine that has CIMOM installed has a root namespace, \\<machinename>\Root, and potentially more namespaces defined under the root namespace in a tree hierarchy. For example, the Win32 schema and objects are located in the default namespace under the root namespace (\\<machinename>\root\DEFAULT). For a more complete discussion see Namespaces in the CIMOM Reference.

    To open the HTML Viewer:
  1. Click Start.
  2. Select Programs, then select WBEM SDK.
  3. Double-click the WBEM HTML Viewer icon.

When you are prompted to enter the name of the machine you wish to browse, type in the target machine name (\\machine) and select Go. The HTML Viewer presents a list of all namespaces available on the target machine. There is one link for each available namespace. The root\DEFAULT namespace is a special case, since it is more frequently used. It has an additional link to a graphical view of the most interesting classes in this namespace.

Note  After selecting the Go button, Netscape can fail to display a list of the namespaces initially. To try again, select the reload button.
