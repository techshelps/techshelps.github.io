Loading MOF Files

[This is preliminary documentation and subject to change.]

The SNMP compiler can generate MOF files in a format suitable for loading into the SMIR, or in a format suitable for loading as static classes into a CIM namespace. A SMIR MOF file contains the name of its associated namespace. A static MOF file does not contain this information, and the namespace must be specified when loading the MOF file.

For instructions on loading MOF and SMIR MOF files, see Loading Static MOF Files and SMIR MOF Files.
