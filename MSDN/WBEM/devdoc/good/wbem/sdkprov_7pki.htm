Win32 Provider

[This is preliminary documentation and subject to change.]

The Win32 Provider retrieves and updates data using the Win32 API, enabling management applications to access this data through WBEM. The Win32 provider retrieves its information by making Win32 function calls and querying the system registry.

Because the Win32 Provider is registered automatically as part of the WBEM installation process, applications need not register it independently. All of the classes and instances defined in the Win32 schema are available through the CIMOM repository.
