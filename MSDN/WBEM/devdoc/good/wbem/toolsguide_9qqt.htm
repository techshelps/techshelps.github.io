Using Make Root to Reroot the Object Tree

[This is preliminary documentation and subject to change.]

To change the object tree's root, right-click the object you want to be the new root. Select Make Root from the menu that appears. Because only regular objects can be made the root of the tree, the Make Root option is not available if grouping nodes are selected.
