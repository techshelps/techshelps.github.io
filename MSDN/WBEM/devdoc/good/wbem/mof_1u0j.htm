MOF Data Types

[This is preliminary documentation and subject to change.]

The MOF language supports many data types that can be used to assign values to properties, qualifiers, and method parameters. Some of these data types map to Automation data types. All MOF data types are case-insensitive.

When a property is lacking an initial value, it is assigned the value of NULL. When a qualifier is lacking a value, it is assigned the value of TRUE. Parameters lacking initial values remain unassigned.

The MOF language also supports the following categories of data types:

  • Boolean
  • Numeric
  • String
  • Array
  • Embedded Object
  • Ref
  • Void
