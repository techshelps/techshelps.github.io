WBEM HTML Viewer

[This is preliminary documentation and subject to change.]

The HTML Viewer is designed for use by advanced system managers and developers, particularly those who are writing providers, and who do not have access to Microsoft® Internet Explorer with ActiveX™.It provides HTML-based browser views of a CIM Object Manager (CIMOM) schema. The HTML Viewer is automatically installed during a complete installation of the WBEM SDK.

This interface requires Microsoft® Internet Information Server (IIS) and an Internet browser. Because the HTML Viewer uses HTML frames, it is recommended that you use only Internet browsers supporting frames, such as Microsoft® Internet Explorer 3.0 and newer versions and Netscape® Navigator 2.0 and newer versions).

Note  If you have Microsoft® Internet Explorer with ActiveX™, use the WBEM Object Browser.

Using the HTML Viewer, you can do the following:

  • Display the class inheritance tree (schema) in a specified CIMOM database.
  • Search for a specific class.
  • Display the definition of an existing class (properties, qualifiers, and methods (operations)).
  • Display instances of an existing class (standard or graphic views).
