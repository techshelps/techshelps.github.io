Enabling Logging

[This is preliminary documentation and subject to change.]

The following registry key is the parent of all logging-related information:

HKEY_LOCAL_MACHINE\SOFTWARE\MICROSOFT\WBEM\PROVIDERS\SNMP\LOGGING

The following values are associated with this key:

Value name Data type Description
LOGGING REG_DWORD One of the following integer values:

'0'        Default. Turns off all debugging output.

'1'        Enables debugging output.
