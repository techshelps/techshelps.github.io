Deleting Class or Property Qualifiers

[This is preliminary documentation and subject to change.]

    To delete qualifiers
  1. Open the Qualifiers window for the desired class or property. For instructions, see Displaying Class Qualifiers or Displaying Property Qualifiers.
  2. Click the blank box to the left of the qualifier you want to delete. This selects the entire qualifier entry.
  3. Press the Delete key. A confirmation message appears. Click OK to delete the qualifier. This activates the Save button in the Class Viewer.
  4. Click the OK button to close the Qualifiers window.
  5. In the Class Viewer, click the Save button. A confirmation message appears. Click Yes to confirm the deletion.
