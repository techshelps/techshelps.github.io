Editing Object Properties

[This is preliminary documentation and subject to change.]

You can edit the Value field for most properties.

    To edit a property value
  1. Display the Properties page for the object you want to change. For instructions, see Displaying Properties of Regular Objects or Displaying Properties of Association Objects.
  2. Click the property value you want to change.
  3. If a down arrow appears when you select the value, click the arrow, then choose a value from the list that appears.

    If no down arrow appears, type the new value in the field.

    When you start typing, the text turns blue. The Save button becomes active; this indicates that the value has not been committed yet.

    If the property type is an array, clicking the Array button in the Value field opens a dialog box in which you can enter each value in the array. When you have finished entering values, click OK to return to the Properties page.

    If the property type is an object, the Value field contains either <empty> (which represents NULL) or an Object button. To change this field, do one of the following:

    • Click the Object button to open the Embedded Object dialog box. You can edit properties and qualifiers from this box the same way as on the Object Viewer Properties page. Click OK to return to the Object Viewer.
    • Right-click the Object button, and select Set to <empty>.
    • Right-click the <empty> value, and select Create object. This opens the Embedded Object dialog box. Enter the name of the class to which the object belongs. The Properties page for that class appears. You can edit properties and qualifiers the same way as on the Object Viewer Properties page. Click OK to return to the Object Viewer.
  4. When you have finished editing, click the Save button. A confirmation message appears. Click Yes to save the changes. After the changes are saved, the text turns black.

Note  Some property values are read-only and cannot be edited.
