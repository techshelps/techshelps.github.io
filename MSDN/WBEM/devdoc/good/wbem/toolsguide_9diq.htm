Class Viewer

[This is preliminary documentation and subject to change.]

The right frame in the CIM Studio is the Class Viewer. The Class Viewer displays details on a selected class. Using the tabs, you can display the properties and methods defined for a class. For additional information on these views, see Displaying the Definition of an Existing Class. Selecting a class in the Class Explorer automatically updates the Class Viewer display.

The Class Viewer includes the following buttons:

Create instance button creates a new instance of a selected class. See Adding Instances of a Class.
Delete instance button deletes the selected instance. See Deleting Instances of a Class.
Back/Forward buttons display previously-viewed pages.
Save button commits changed information to CIMOM.
Instances button displays a table of all instances of a selected class. See Displaying an Instance Table.
Views button displays custom views of class instance data. See Creating and Using Custom Views of Instances.

The CIM Studio also includes a Help button (). For help on the WBEM ActiveX applications, double-click this button.
