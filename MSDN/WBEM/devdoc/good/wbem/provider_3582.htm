Trying the Provider

[This is preliminary documentation and subject to change.]

    To try the provider
  1. Open CIM Studio and select the newly-defined class in the left frame.
  2. Click the Multi-Instance Viewer button to invoke the instance enumeration in the provider. An instance table for the class appears.
  3. To invoke the GetObject implementation, double-click an instance in the table.

Also, try the sample program WBEMDump as follows:

Wbemdump root\default MyClass  <- This will do the enumeration
