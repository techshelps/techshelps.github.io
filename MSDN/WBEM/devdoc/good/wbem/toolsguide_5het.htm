Compiling a MOF File

[This is preliminary documentation and subject to change.]

    To compile a MOF file
  1. Double-click the MOF Compiler Wizard icon. A welcome screen appears. Select Compile a .MOF file.
  2. Click the Next button. The following screen appears:

Type the path and file name for the MOF file you want to compile (or use the Browse button to select the file), and the namespace where the MOF file should be loaded.

  1. Click the Next button. The following screen appears:

Select or cancel the selection of the appropriate compiler options. You can modify classes and instances that are already in the namespace, or you can add new classes and instances to the namespace.

  1. Click Finish to compile the file. The WBEM Login dialog box appears.

    Enter the appropriate login information (see Opening the ActiveX Applications for instructions), and click OK. A message box informs you of successful compilation. Click OK to dismiss the message. The resulting database file is created in the same directory as the MOF file.
