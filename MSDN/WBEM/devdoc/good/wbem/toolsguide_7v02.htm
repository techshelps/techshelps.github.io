WBEM Event Viewer

[This is preliminary documentation and subject to change.]

As CIMOM generates events, the event objects are sent automatically to consumers registered for the object type. The WBEM Event Viewer displays the events for all instances of registered consumers.

With the WBEM Event Viewer, you can do the following:

  • View CIMOM-generated events and event information, such as the event's date and time, class, point of origin, and a description.
  • View event instance properties.
  • Start the WBEM Event Registration Viewer.
  • Clear the display.
