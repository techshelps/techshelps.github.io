Adding Users or Groups

[This is preliminary documentation and subject to change.]

    To add a user or group
  1. Click the Add button. The WBEM User File and Registry Permissions dialog box appears:

  1. Under Target User or Group, type the Name of user or group you want to add.
  2. Select one of the WBEM File and Registry Permissions options:

    Use

    Allows the user/group read and execute access to all WBEM components.

    Modify

    Allows the user/group full access (read/write/execute) to all files, directories, and the registry.

    No Access

    Denies the user/group access to all WBEM components.

  3. Select the Submit Self-Installing MOFs option if you want the user/group to have this privilege.
  4. Click OK to return to the WBEM Security for Windows NT dialog box.
  5. Click Apply to save the new data and remain in the WBEM Administrator.

    —or—

    Click OK to save the new data and close the WBEM Administrator.
