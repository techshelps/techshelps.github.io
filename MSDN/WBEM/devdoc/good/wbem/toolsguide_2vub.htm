Viewing Class Properties

[This is preliminary documentation and subject to change.]

While you cannot modify any properties of a system class, you can view its properties.

    To view class properties
  1. In the left pane of the Event Registration Viewer, click the class, and then click the View Class Properties button.

    —or—

  2. In the left pane of the Event Registration Viewer, right-click the class, and then select View Class Properties from the pop-up menu.

    The View Class Properties dialog box appears.
