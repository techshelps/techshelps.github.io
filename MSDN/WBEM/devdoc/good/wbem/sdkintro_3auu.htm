Propprov

[This is preliminary documentation and subject to change.]

Propprov is an example of a property provider DLL. This provider simply returns a property of type SINT32 or STRING. This is the easiest provider to write, and makes a good starting point for developers.

For more information on Propprov, see the Readme.txt file in the Samples\Propprov subdirectory of the WBEM installation directory.

WBEMDump

[This is preliminary documentation and subject to change.]

WBEMDump shows the entire contents of the CIMOM database in a hierarchical view. CIMOM supports a number of options to specify which information is shown. To view these options, open a command prompt window and use the cd command to move to the \Samples\WBEMDump subdirectory of the WBEM installation directory. Then enter the string shown below, followed by the ENTER key:

Wbemdump /?
