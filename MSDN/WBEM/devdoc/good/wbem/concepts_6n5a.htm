Identity and Location

[This is preliminary documentation and subject to change.]

Web-Based Enterprise Management (WBEM) classes and instances are identified and located using a formatted string known as an object path. For classes, an object path identifies the server on which the class resides, the namespace to which it is assigned, and the name of the class. A namespace is a key-scoping mechanism, a unit for grouping classes that operate in the same managed environment. The object path for instances specifies a key in addition to the server, namespace, and class.
