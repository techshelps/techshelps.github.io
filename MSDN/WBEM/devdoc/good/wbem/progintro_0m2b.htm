Programming Basics

[This is preliminary documentation and subject to change.]

The Web-Based Enterprise Management (WBEM) SDK provides tools and programming interfaces that enable management applications and providers to be created quickly and easily. Although they implement a diffferent set of features, applications and providers perform some of the same basic tasks due to their use of the single CIMOM API.

For example, most WBEM developers need to:

  • Understand the IUnknown interface
  • Understand how to use output parameters
  • Understand asynchronous processing
  • Understand semisynchronous processing
  • Understand how to specify object paths
  • Understand the date/time format
  • Understand how to handle errors
  • Understand how to create namespaces, classes, and instances
