Using the Go to Namespace Function

[This is preliminary documentation and subject to change.]

The object tree can contain objects from other namespaces. If an object is from another namespace, the object's full path appears in the tree (including the machine name if it resides on a different machine). You can display the tree for that object's namespace by using the Go to Namespace function.

    To use the Go to Namespace function
  1. Right-click the object whose namespace you want to display.
  2. Select the Go to Namespace option. The object tree for the namespace appears rooted at the selected object.
