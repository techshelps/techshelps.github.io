Providers and Drivers

The Web-Based Enterprise Management (WBEM) SDK contains several built-in object providers and application drivers. The built-in object providers enable management applications to access data and event notifications from a variety of sources such as the system registry and an SNMP device. The drivers enable applications written with various programming interfaces to access the CIMOM repository without having to be rewritten.

The Providers and Drivers section of the WBEM SDK documentation contains:

  • SDK Providers, a description of the providers that ship with the WBEM SDK.
  • WBEM ODBC Driver, a description of the technology that allows ODBC applications to access database information through WBEM.
  • DMI Support, an object provider that supplies information to applications from Desktop Management Interface (DMI) objects.
  • SNMP Support, an object provider that supplies information to applications from Simple Network Management Protocol (SNMP) devices.
