Class Explorer

[This is preliminary documentation and subject to change.]

The left frame in the CIM Studio is the Class Explorer. The Class Explorer shows the selected namespace (root\DEFAULT in the sample screen in the preceding topic) and the schema defined for that namespace as a class inheritance tree. Clicking the plus and minus boxes expands and contracts the tree. The shaded icons next to the class names indicate the type of objects defined by the class:

Class that defines regular objects
Class that defines association objects

Use the check boxes to select or cancel the selection of classes for the wizard operations performed in the CIM Studio. You can also select or clear all check boxes by right-clicking anywhere in the Class Explorer, and then choosing Select all or Unselect all from the menu that appears.

For non-wizard operations, select classes by clicking the class name in the Class Explorer.

The Class Explorer includes the following buttons:

Browse for Namespace button allows you to browse and select namespaces on the local machine or a remote machine. See Browsing for a Namespace.
Search for Class button allows you to search for a specific class in the namespace. See Searching for a Class.
Add class button adds a new class to the current database. See Adding Class Definitions.
Delete class button deletes a selected class from the current database. See Deleting Class Definitions.
