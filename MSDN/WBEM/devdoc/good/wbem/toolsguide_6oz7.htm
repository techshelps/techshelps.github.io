Displaying Properties of Association Objects

[This is preliminary documentation and subject to change.]

    To display the properties of an association object
  1. On the Associations page, double-click the icon for the association object you want to display.
  2. Click the Properties tab. The property details for the selected association object appear.

You can also right-click an object in the Object Explorer, then select Association from the menu that appears. The Object Viewer displays the details of the association object that links the selected object to the object above it.
