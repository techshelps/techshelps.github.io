Mapping the DESCRIPTION Clause

[This is preliminary documentation and subject to change.]

Each SNMPv1 and SNMPv2C MIB object, whether in a scalar or table collection, contains a possibly empty DESCRIPTION clause. The SNMP DESCRIPTION clause maps to the CIM property qualifier description.

Each TABLE and ENTRY object within an SNMP table definition contains a possibly empty DESCRIPTION clause. The TABLE and ENTRY DESCRIPTION clauses are concatenated and the result maps to the CIM class qualifier description.
