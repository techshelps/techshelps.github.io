Fatal Error 1101

[This is preliminary documentation and subject to change.]

<1101, Fatal>: "<fileName><line#>: The length of the range is too long for the compiler (Does not fit into 4 bytes)"
Module semantic error in range or size specification, specific to neither SNMPv1 nor SNMPv2C. The size of a set of ranges must be less than 232. The size of a set of ranges is defined as the difference between the largest value and the smallest value in the set.
