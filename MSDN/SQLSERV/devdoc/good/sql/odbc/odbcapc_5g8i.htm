Error Messages

For errors that occur in the data source and that are detected and returned by SQL Server, the SQL Server driver returns an error message that is based on the message that is returned by SQL Server. For errors that occur in the SQL Server driver or the Driver Manager, the SQL Server driver returns an error message that is based on the text that is associated with the SQLSTATE. For a list of error messages that can be returned by SQL Server (but not by the SQL Server driver), see the description column of the sysmessages system table in the master database in SQL Server.
