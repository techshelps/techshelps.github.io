Installing the SQL Server Driver

The ODBC SQL Server driver is installed automatically when you install the SQL Server 6.5 client software on a computer running Windows NT, Windows 95, or Windows 3.11. For more information about installing SQL Server 6.5 client software, see Microsoft SQL Server Setup.

If you need to reinstall just the ODBC SQL Server driver, you can run the ODBC setup program from the ODBC subdirectory of the appropriate client install directory.

To use the SQL Server driver, you usually add a data source for each installation of SQL Server in which you want to access data (see Connecting to a SQL Server Data Source.)

If you are using Microsoft SQL Server 4.21a or 6.0, the INSTCAT.SQL file included with this driver includes minor fixes to the catalog stored procedures. For information about installing the updated catalog stored procedures, see the following section.
