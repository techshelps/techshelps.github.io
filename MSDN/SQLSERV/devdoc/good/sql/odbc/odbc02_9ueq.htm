SQL Server

To use the SQL Server driver to access data in Microsoft SQL Server databases, you must have Microsoft SQL Server 4.21a or later. The catalog stored procedures must be installed on your SQL Server. You may need to install the catalog stored procedures shipped with this driver when you use version 4.21a or 6.0 of Microsoft SQL Server (see the Upgrading the Catalog Stored Procedures topic). For information about the hardware and software required by SQL Server, see Microsoft SQL Server Setup.
