Performance Improvements

The driver has the following new performance-related features:

  • Improved Connect Time—The driver batches informational queries to the server when making a connection that reduces the number of trips to the server. The fast connect option is no longer needed or supported.
  • The driver makes better use of memory (eliminating copies) for faster processing of results.
  • 64-bit Arithmetic—The 32-bit driver uses 64-bit arithmetic for data conversions to and from money, decimal, and numeric datatypes.
