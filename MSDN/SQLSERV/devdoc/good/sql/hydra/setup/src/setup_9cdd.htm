Setup Errata

The following changes should be made to Microsoft SQL Server Setup for Microsoft SQL Server 6.0.

  • On page 99, the Orgname entry should be OrgName.
  • On page 101, the [Charset] section title should be [CharSet].
  • On page 101, the value for the 850 Multilingual character set should be cp850.
  • On page 107, the unattended setup initialization file is incorrect. The correct file is in A Sample Unattended Setup Initialization File.
