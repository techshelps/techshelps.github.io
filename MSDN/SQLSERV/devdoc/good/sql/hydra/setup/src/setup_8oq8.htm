Server Installation

During SQL Server 6.5 installation, the following new components are installed in addition to those provided by SQL Server 6.0. Each component is described in detail later in this document:

  • Microsoft Distributed Transaction Coordinator (MS DTC), running as a Windows NT service
  • Microsoft Query
  • Simple Network Management Protocol Management Information Base (SNMP MIB) and SNMP agent
  • SQL Server Web Assistant
  • SQL Trace
