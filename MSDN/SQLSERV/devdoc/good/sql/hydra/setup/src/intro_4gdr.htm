About This Document

What's New in SQL Server 6.5 provides information supplementary to the Microsoft SQL Server 6.0 documentation set. In many cases, especially when a feature has been enhanced, it is important that you read the version 6.0 information about that feature in addition to reading about the enhancements in What's New in SQL Server 6.5.

Important SQL Server Books Online contains the most up-to-date information for Microsoft SQL Server 6.5. See Books Online for information that changed or was unavailable at the time the documentation was printed.

The new SQL Server 6.5 features and more are detailed in the following parts:

  • Part 2, What's New for Setup
  • Part 3, What's New for Administrators
  • Part 4, What's New for Transact-SQL
  • Part 5, What's New for OLE Automation Stored Procedures
  • Part 6, What's New for SQL-DMO
  • Part 7, What's New for DB-Library for C
  • Part 8, What's New for DB-Library for Visual Basic
  • Part 9, What's New for Open Data Services
