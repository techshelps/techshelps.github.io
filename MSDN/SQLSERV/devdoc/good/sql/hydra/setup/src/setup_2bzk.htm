Client Installation

The following components are new for SQL Server 6.5 and are optional for client installations:

  • MS DTC (client utility support installed)
  • Microsoft Query
  • Other ODBC client drivers for heterogeneous replication use
  • SQL Server Web Assistant
  • SQL Trace

Note To install on a Windows 95 - based client, you must have an AUTOEXEC.BAT file before installing. This is the only place that setup sets the path, and it will not create the file for you.
