Running the Setup Program

Important Earlier versions of Microsoft SQL Server should be shut down before installing or upgrading to version 6.5. Also, before upgrading, stop replication and make sure that the log is empty.

    To install or upgrade to SQL Server 6.5
  • From the SQL Server CD, open the appropriate platform directory and run setup.

    For example, to install SQL Server on a Windows NT 3.51 or later, Intel®-based system, run setup from the i386 subdirectory on the SQL Server CD.

The default installation directory for SQL Server 6.5 is MSSQL. You can change the installation directory during setup.
