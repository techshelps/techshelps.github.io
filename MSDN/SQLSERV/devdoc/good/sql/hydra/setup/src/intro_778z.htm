ODBC Enhancements

The ODBC SQL Server driver includes support for enlisting in MS DTC transactions using a new driver-specific connection option, the ability to log performance data and long-running queries, and support for connecting to fallback servers. Many other enhancements have also been included. For details about ODBC enhancements, see Programming ODBC for Microsoft SQL Server.

Note that the ODBC Help file contains version 6.0 information. For version 6.5 information, see Programming ODBC for Microsoft SQL Server.
