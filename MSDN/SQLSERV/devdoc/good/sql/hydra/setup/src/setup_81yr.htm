MSSQLServer Service and SQL Executive Requirements

The requirements for MSSQLServer service and SQL Executive accounts can be set by using the Windows NT User Manager application:

  • The account must be a member of the Administrators local group.
  • The account must have the Password Never Expires attribute.
  • The account must be granted the following advanced User Rights:
    • Log on as a service
    • Act as part of the operating system
    • Replace a process level token
    • Increase quotas
