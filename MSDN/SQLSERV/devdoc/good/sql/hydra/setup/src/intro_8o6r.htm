DB-Library for Visual Basic Enhancements

SQL Server 6.5 includes a new DB-Library for Visual Basic OLE custom control, a 32-bit OLE custom control for creating DB-Library for Visual Basic projects using 32-bit Visual Basic 4.0.

DB-Library for Visual Basic includes the same enhancements as DB-Library for C, except for MS DTC support. It also includes the new SqlCursorRowStatus function for returning the row fetch status value for a row in the cursor fetch buffer.

New programming examples are provided in the SQL Server Workstation System in the \PTK\SAMPLES directory. Additional information about the samples is included in the Workstation's README.TXT file.
