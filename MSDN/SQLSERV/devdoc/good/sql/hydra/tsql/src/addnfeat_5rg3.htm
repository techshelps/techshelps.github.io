Utilities (version 6.5)

SQL Server contains several utilities, such as isql and SQLMAINT, that are included for performing certain maintenance, administration, and debugging tasks.

isql is a command-line query tool that is used for MS-DOS®-based connectivity and batch processing.

SQLMAINT is a command-line utility that is used to run automated consistency checks and backups. For example, it can create status reports and send them to an e-mail account.
