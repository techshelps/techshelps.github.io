ODBC Driver Support

ODBC drivers for Microsoft Access and ORACLE are included on the SQL Server 6.5 CD. Microsoft technical support will support problems encountered when using these drivers for replication. For information about Microsoft technical support options, see the service and support card included in your SQL Server 6.5 package.

If you encounter problems trying to replicate to ODBC data sources using other ODBC drivers, please contact the ODBC driver vendor for support.
