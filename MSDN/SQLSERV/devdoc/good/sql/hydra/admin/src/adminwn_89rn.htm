The following table describes the commands that are new for SQL Server 6.5.

New menu / Command Description
File / (all) Contains commands from the toolbar for the active window.
File / Print Setup Opens the Print Setup dialog box, which is used to configure the default printer.
View / Tools Bar Displays or hides the floating, dockable Tools Bar for external tools.
View / Refresh Updates information in the currently selected folder if applicable for a server in the Server Manager window.
Server / Remote Servers Opens the Manage Remote Servers dialog box, which is used to manage remote servers and logins, and subscribers (including ODBC subscribers).
Server / Distributed Transaction Coordinator Opens the MS DTC Transactions, Trace, Statistics, or Configuration windows for servers with MS DTC installed. These windows are used to administer MS DTC.
Server / SQL Mail Opens a shortcut menu, which is used to stop, start, and configure SQL Mail. To open the menu, select the SQL Mail icon for a server and right-click the selection.
Server / SQL Executive Opens a shortcut menu, which is used to stop, start, and configure SQL Executive. To open the menu, select the SQL Executive icon for a server and right-click the selection.
Server / SQL Server Opens a shortcut menu, which is used to stop, start, pause, continue, and administer SQL Server. To open the menu, select a server and right-click the selection.
Tools / External Tools Opens the External Tools dialog box, which is used to add or remove external tools that can be run in SQL Enterprise Manager.
Manage / Extended Stored Procedures Opens the Manage Extended Stored Procedures dialog box, which is used to add or remove extended stored procedures from the master database only.
Object / Transfer Opens the Transfer Objects dialog box, which is used to transfer objects from one database to another.
Help / Database Maintenance Wizard Starts the Database Maintenance Plan Wizard.
