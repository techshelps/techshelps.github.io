Copying the MIB to an SNMP Workstation

For SNMP applications to monitor the status of a SQL Server, a copy of the MSSQL-MIB must be placed on the monitoring workstation and loaded into the SNMP application. The MSSQL-MIB enables the SNMP application to access the SQL Server SNMP extension agent on the computer that is running SQL Server and to monitor that SQL Server.

    To copy the SQL Server MIB to an SNMP workstation
  1. In the C:\MSSQL\SNMP directory on the computer that is running SQL Server, locate the MSSQL.MIB file.
  2. Copy the MSSQL.MIB file to the appropriate directory on the monitoring workstation.
  3. Repeat for all workstations that will be monitoring SQL Server.

For information about loading the MSSQL-MIB and monitoring SQL Server, see the documentation for your SNMP application.
