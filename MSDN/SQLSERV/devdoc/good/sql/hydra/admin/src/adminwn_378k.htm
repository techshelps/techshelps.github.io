Database Maintenance Plan Wizard

The Database Maintenance Plan Wizard sets up and schedules core maintenance actions for a selected database. These maintenance actions include checking data allocation, data and index linkages, and system data; updating optimizer information and reorganizing data and index pages; and performing database and transaction log backups. Maintenance actions can be scheduled for automatic execution at preset daily or weekly intervals.

    To run the Database Maintenance Plan Wizard

  • From SQL Enterprise Manager, choose the Wizard icon from the toolbar or, from the Help menu, choose Database Maintenance Wizard and then follow the on-screen instructions.
