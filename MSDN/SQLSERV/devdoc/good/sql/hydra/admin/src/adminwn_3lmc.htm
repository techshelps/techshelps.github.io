SQLServer-Licensing Object

The SQLServer-Licensing object contains three counters.

Counter Description
Client Count Specifies the number of client workstations that are connected to SQL Server.
Client Limit Specifies the maximum number of client workstations that can connect to SQL Server. This number reflects the number of separate machines that can connect, not the number of connections that can be made.
Max Client Count Specifies the maximum number of concurrent client connections that can be made to SQL Server during a SQL Performance Monitor monitoring session.
