About Microsoft Access Subscribers

SQL Server 6.5 includes an ODBC driver that supports Microsoft Access subscriptions to SQL Server replication. The ODBC driver exists only for Intel-based computers. To install the driver, run SETUP.EXE from the \I386\ODBCREPL directory on the SQL Server CD-ROM.

To replicate to Microsoft Access subscribers, you must assign to the MSSQLServer service the same domain user account as is assigned by SQL Executive to have necessary permissions to connect to an .MDB file over the network. Use the services application in Control Panel to do this.

Note When you register a Microsoft Access DSN on a remote server, supply a UNC path (not a redirected drive letter).
