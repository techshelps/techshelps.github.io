External Tools Support

You can launch any Windows NT-based application from SQL Enterprise Manager. External applications can be added and run from the Tools menu and from the floating, dockable Tools Bar.

To add an application, from the Tools menu, choose External Tools and complete the dialog box that appears. Note that [SVR] and [DBN] can be used in the Parameters box to represent the current server and database. When the tool is run, SQL Enterprise Manager will substitute the current server and database names, passing them as parameters to the application.

The floating, dockable Tools Bar can be displayed or hidden. To change the current setting, from the View menu choose Tools Bar. When the Tools Bar command is selected, the floating, dockable Tools Bar is displayed.

For more information about adding and running external tools, see the SQL Enterprise Manager online Help.
