Query Interface Renamed

For SQL Server 6.5, the query interface in SQL Enterprise Manager is named SQL Query Tool. For SQL Server 6.0, the command on the Tools menu and the icon on the toolbar are called Query Analyzer.

To use the query tool, from the Tools menu, select SQL Query Tool or from the SQL Enterprise Manager toolbar, choose the SQL Query Tool icon.

For more information, see the online Help for ISQL/w.
