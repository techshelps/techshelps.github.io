Graph Controls

Several SQL Enterprise Manager windows provide enhanced charting capabilities. The charts have been refined and buttons have been added that you can use to control graph characteristics and to print the graphs. The following dialog boxes have new option buttons and enhanced charting capabilities.

Dialog box Has a chart of
Manage Databases Databases
New Database Available database devices
Manage Database Devices Devices
New Database Device Available storage space
Edit Database Device Device space usage

The option buttons are:

Graph Properties
Opens the Graph Properties dialog box, in which you can set graph label, solid color, legend display, and graph type properties.

Bar Graph Options
Opens the Bar Graph Options dialog box, in which you can set the graph mode to stretched or selectable, and the orientation to vertical or horizontal.

Print Options
Opens the Print Options dialog box, in which you can set color, formatting, and font options, and can print the graph.
    To print a graph
  • In SQL Enterprise Manager, from the File menu choose the Print Setup command and then choose the Print button.
