Extended Stored Procedures Manager

SQL Enterprise Manager includes an extended stored procedures manager. With the extended stored procedures manager, you can add extended stored procedures to or drop existing extended stored procedures from SQL Server. The extended stored procedures manager can be used only when the master database is the current database.

Execute permission defaults to the system administrator.

    To use the extended stored procedures manager
  1. From SQL Enterprise Manager, in the Server Manager window select a server, open its Databases folder, and select the master database.
  2. From the Manage menu, choose Extended Stored Procedures.

    The Manage Extended Stored Procedures dialog box appears.

  3. Complete the dialog box to add or drop a procedure.

For more information about using the Extended Stored Procedures Manager, see the online Help for SQL Enterprise Manager.
