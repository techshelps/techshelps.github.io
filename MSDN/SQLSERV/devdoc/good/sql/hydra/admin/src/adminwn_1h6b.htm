SERVERS.BIN File Replaced by Registry Entries

For SQL Server 6.5, registration information for servers is maintained in the operating system Registry for both Windows NT- and Windows 95-based computers. Server registration information includes the server groups that are used in the Server Manager window of SQL Enterprise Manager, the servers that are registered with SQL Enterprise Manager, and associated data. Putting server registration information in the operating system Registry is a more standard location for this type of information.

For more information about the new server Registry-based storage, see Remote Server Registration Storage Configuration, earlier in this document.

For SQL Server 6.0, registration information for servers is maintained in a SERVERS.BIN file.

The first time SQL Enterprise Manager is run after upgrading a SQL Server 6.0 installation to SQL Server 6.5, the information from the existing SERVERS.BIN file is automatically moved to the Registry.
