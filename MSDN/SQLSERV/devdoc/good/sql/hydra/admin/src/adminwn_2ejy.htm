SQLServer-Log Object

The SQLServer-Log object includes the following counters to obtain additional statistics.

Counter Description
Max Log Space Used %
(per db instance)		 Specifies the maximum percentage of space in a transaction log that has been used for each database during the monitoring session.
Max Log Size (MB)
(per db instance)		 Specifies the maximum size of the transaction log, in megabytes, that has been allocated for each database during the monitoring session.
