Uninstalling Replication

SQL Server 6.5 automates uninstalling replication from SQL Enterprise Manager. Once replication is installed on a server, the Install Publishing menu option becomes Uninstall Publishing. Uninstalling replication causes all publications for the current server to be deleted and the distribution database to be dropped.

    To uninstall replication
  1. From the Server menu, choose Replication Configuration/Uninstall Publishing.

    The Uninstall Replication dialog box appears.

  2. Choose Yes.
