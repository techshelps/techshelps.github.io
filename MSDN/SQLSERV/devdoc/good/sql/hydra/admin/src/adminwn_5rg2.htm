SQL Enterprise Manager

SQL Enterprise Manager provides many new and enhanced features. You can restore data in a database to a specified point in time. You can easily transfer objects from one server to another. External tools can be added to the SQL Enterprise Manager toolbar. These and many more enhancements and new features in SQL Enterprise Manager are detailed in the following sections.

Note SQL Enterprise Manager for SQL Server 6.5 administers SQL Server 6.0 and 6.5 servers. SQL Enterprise Manager for SQL Server 6.0 cannot administer SQL Server 6.5 servers. To connect to a SQL Server 6.5 server, you must upgrade to a SQL Server 6.5 client installation.
