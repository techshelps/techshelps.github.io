Installing Microsoft SQL Server Using Systems Management Server

You can use Microsoft Systems Management Server (SMS) to automatically install Microsoft SQL Server version 6.5 on multiple Windows NT Server computers in your organization.

The Microsoft SQL Server CD-ROM contains the following files (located at the root of the CD-ROM) to help you deploy SQL Server with SMS.

Filename Description
SMSQL65.PDF Is a Package Definition Format (PDF) file that automates creating a SQL Server package in SMS. The SQL Server package can then be distributed and installed on SMS computers. The PDF file also enables SMS to detect SQL Server on SMS computers and report that information in the SMS computer inventory.
SMSSQL65.CMD Is a batch file that detects the platform of the computer and runs the appropriate version of setup. The SMSSQL65.CMD batch file installs the complete SQL Server product, including SQL Server Books Online.
SMSSQL65.INI Is the installation script that is used for unattended setup. This file can easily be edited to specify any drive, subdirectory, or installation option. For more information, see Unattended Setup.
