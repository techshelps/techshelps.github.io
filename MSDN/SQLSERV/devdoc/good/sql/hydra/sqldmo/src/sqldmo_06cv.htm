EnumLoginMappings List

Description

For every login in SQL Server, lists corresponding user information for every database that has a user mapped to the login. List is returned in a QueryResults object.

Visual Basic

QueryResults = SQLServer.EnumLoginMappings

C++

HRESULT pSQLServer->EnumLoginMappings ( LPSQLOLEQUERYRESULTS* ppQueryResults );

Elements


Element Type Description

QueryResults QueryResults Returned QueryResults object

QueryResults

This method will return a results set for each login.
