ServerMessage Method

Description

This event method is called when a low-severity message that does not cause a SQL-DMO error is received from SQL Server.

C++

HRESULT pServerSink->ServerMessage (
long lSeverity,
long lNumber,
long lState,
SQLOLE_LPCSTR strMessage );

Elements


Element Type Description

lSeverity Long Server message Severity
lNumber Long Server message Number
lState Long Server message state
strMessage String Server message text
