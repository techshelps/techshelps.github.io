Application Properties


Property Description

GroupRegistrationServer
    String
    Write: Yes		 The computer to use for group registration information. The default is to use local group registration information. Changing this property does not automatically refresh the Application.ServerGroups collection.
GroupRegistrationVersion
    Long
    Write: No		 The version number of the group registration information.
VersionBuild
    Long
    Write: No		 The SQL-DMO component object layer build number.
UseCurrentUserServerGroups
    Boolean
    Write: Yes		 Determines if group registration information is stored per user. If True, group registration information is stored separately for each Windows 95 or Windows NT user. If False, group registration information is stored globally for the computer. Changing this property does not automatically refresh the Application.ServerGroups collection.
