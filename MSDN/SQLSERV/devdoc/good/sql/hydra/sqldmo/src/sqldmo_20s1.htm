Script Method

Description

Generates a series of Transact-SQL statements (script) to create the underlying database.

Visual Basic

strScript = Database.Script (
[[ScriptType :=] tScriptType,]
[[ScriptFilePath :=] strFile] )

C++

HRESULT pDatabase->Script (
SQLOLE_SCRIPT_TYPE tScriptType = SQLOLEScript_Default,
SQLOLE_LPCSTR strFile = NULL,
SQLOLE_LPBSTR pstrScript = NULL );

Elements


Element Type Description

strScript String Generated Transact-SQL script
tScriptType SQLOLE_SCRIPT_TYPE Type of script to generate
strFile String Full path and filename to use to store the Transact-SQL script

Remarks

The generated Transact-SQL script can be used for database recovery.
