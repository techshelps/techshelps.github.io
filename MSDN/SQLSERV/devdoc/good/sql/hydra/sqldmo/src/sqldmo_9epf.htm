Article Properties


Property Description

IncludeInsertColumnNames
    Boolean
    Write: Yes		 Determines if INSERT statements used by replication include the column list.
OwnerQualifyTable
    Boolean
    Write: Yes		 Determines if the table for this article is owner qualified. If True, the table is owner qualified, and the owner of the table in the subscribing database must be the same as the owner of the table for this article. If False, the table is not owner qualified.
