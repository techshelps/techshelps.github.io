EnumLoginMappings List

Description

For every login that maps to a user in this database, lists a single-row results set describing that mapping. List is returned in a QueryResults object.

Visual Basic

QueryResults = Database.EnumLoginMappings

C++

HRESULT Database->EnumLoginMappings ( LPSQLOLEQUERYRESULTS* ppQueryResults );

Elements


Element Type Description

QueryResults QueryResults Returned QueryResults object

QueryResults

This method will return a results set for each login.
