AddObjectByName Method

Description

Adds an object name to the list of copied objects.

Visual Basic

Transfer.AddObjectByName (
[ObjectName :=] idObject,
[ObjectType :=] tObjectType )

C++

HRESULT pTransfer->AddObjectByName (
SQLOLE_LPCSTR idObject,
SQLOLE_OBJECT_TYPE tObjectType );

Elements


Element Type Description

idObject Identifier Name of the object
tObjectType SQLOLE_OBJECT_TYPE Type of the object; valid values are:

SQLOLEObj_UserDefinedDatatype
SQLOLEObj_View
SQLOLEObj_UserTable
SQLOLEObj_StoredProcedure
SQLOLEObj_Default
SQLOLEObj_Rule
SQLOLEObj_Trigger
