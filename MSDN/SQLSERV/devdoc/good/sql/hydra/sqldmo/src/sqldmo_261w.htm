RemoteLoginFailed Method

Description

This event method is called when a remote login to a remote SQL Server fails.

C++

HRESULT pServerSink->RemoteLoginFailed (
long lSeverity,
long lNumber,
long lState,
SQLOLE_LPCSTR strMessage );

Elements


Element Type Description

lSeverity Long Severity of the server message
lNumber Long Number of the server message
lState Long State of the server message
strMessage String Text of the server message
