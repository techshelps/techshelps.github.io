PercentComplete Method

Description

This event method is called when a running Dump or Load method (of a Database or TransactionLog object) sends a new percent complete message.

C++

HRESULT pBackupSink->PercentComplete (
SQLOLE_LPCSTR strComplete,
long lPercentComplete );

Elements


Element Type Description

strComplete String Percent complete message
lPercentComplete Long Percent complete
