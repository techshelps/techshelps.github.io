What's New for DB-Library for Visual Basic

SQL Server 6.5 includes several new and enhanced functions for DB-Library for Visual Basic. These functions include SqlCursorOpen and SqlCursorRowStatus, and they are designed for use with cursor operations. The new SqlSetLFallback function is designed for fallback server support. The following table lists the new and enhanced functions, which are explained in this part. Features are in alphabetical order.
