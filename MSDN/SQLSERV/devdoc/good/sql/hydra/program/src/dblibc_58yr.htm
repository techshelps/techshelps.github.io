What's New for DB-Library for C

SQL Server 6.5 includes several new and enhanced functions for DB-Library for C. The new functions dbenlisttrans and dbenlistxatrans are designed for use with Microsoft Distributed Transaction Coordinator (MS DTC). The new DBSETLFALLBACK function is designed for fallback server support. The following table lists the new and enhanced functions, which are explained in this part. Features are in alphabetical order.

Note that the DB-Library for C Help file contains version 6.0 information. Read this section for the most up-to-date version 6.5 information.
