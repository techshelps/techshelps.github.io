Default Values for a LOGINREC Structure

If you do not call DBSETLAPP, the application parameter of the LOGINREC structure is set to the calling application name by default instead of to NULL, as it is in SQL Server 6.0.

If you do not call DBSETLHOST, the workstation parameter of the LOGINREC structure is set to the host workstation name by default instead of to NULL, as it is in SQL Server 6.0.
