XA Switch for Microsoft SQL Server

The object file containing the XA switch is XASWITCH.OBJ. Typically, this file is linked with each XA program that calls XA functions. For the procedures for linking the XA switch file with your XA application, see your TP monitor documentation.

The variable name that represents the XA switch is msqlsrvxa1.

The descriptive name for the XA switch is “MS SQL Server XA_1”.
