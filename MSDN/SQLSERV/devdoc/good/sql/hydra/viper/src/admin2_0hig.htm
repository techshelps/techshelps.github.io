Display Refresh

Use the Display Refresh slider to control how often the transaction list, trace messages, and statistical information are sent from the MS DTC process to the graphical interface. More frequent updates increase administrative overhead on the running transactions but also provide more current information on the running MS DTC. These are the values on the slider:

  • Update the display every 20 seconds. (Shown as "Infrequently" on the slider.)
  • Update the display every 10 seconds.
  • Update the display every 5 seconds (default).
  • Update the display every 3 seconds.
  • Update the display every 1 second. (Shown as "Frequently" on the slider.)
