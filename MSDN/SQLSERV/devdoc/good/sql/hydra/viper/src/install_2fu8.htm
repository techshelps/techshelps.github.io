Setup

You install the Microsoft Distributed Transaction Coordinator (MS DTC) by using the Microsoft SQL Server 6.5 setup program. The setup program provides the following MS DTC installations:

  • Complete MS DTC service on a database server
  • MS DTC client utility on a client computer

The SQL Server setup program automatically installs the complete MS DTC service as a component of SQL Server.

Installing the MS DTC client utility on a client computer is required only when the client application resides on the same computer, and the client application begins an MS DTC transaction with the C or C++ BEGIN TRANSACTION statement. The MS DTC client utility is not required if the client application resides on the same computer as a complete MS DTC service.
