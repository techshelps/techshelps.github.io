The Log Group

The MS DTC log file is called MSDTC.LOG.

Caution Do not reset the MS DTC log file while MS DTC has unresolved transactions. Unresolved transactions are displayed in the MS DTC Transactions window in SQL Enterprise Manager while MS DTC is running.

    To change the size or location of the log file
  1. Choose the Stop button to stop the MS DTC.
  2. Wait for the Status indicator to show Stopped before making any changes.
  3. To change the log file size, use the Size slider.

    To change the log file location, use the Drive and Directory boxes.

    After changing the size or location of the log file, choose the Reset Log button.

  4. Choose the Start button to restart the MS DTC.
