Converting MS DTC Installations

Occasionally, you may want to reconfigure one or more computers on a distributed transaction system. For example, you may want to add a SQL Server database to a computer that was previously configured as a client machine.

To convert a networked computer from one installation option to the other:

  1. Remove the current MS DTC installation (see Removing an MS DTC Installation).
  2. Rerun the Microsoft SQL Server setup program and choose the installation option you want. See Installing the MS DTC Service and Installing the MS DTC Client Utility.
