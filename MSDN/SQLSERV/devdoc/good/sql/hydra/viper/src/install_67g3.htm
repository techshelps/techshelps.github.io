MS DTC Files

The complete MS DTC components are contained in these files.

File Description
ADME.DLL Maintains contacts between various MS DTC components
DAC.EXE MS DTC user interface control program
DTCCM.DLL Connection manager DLL
DTCTRACE.EXE MS DTC trace program
DTCTRACE.DLL MS DTC trace DLL
DTCUIC.DLL Client side of administrative component
DTCUIS.DLL MS DTC administrative component DLL
DTCUTIL.DLL MS DTC name service DLL
ENUDTC.DLL String resource DLL
LOGMGR.DLL Log manager DLL
MSDTC.EXE MS DTC Windows NT service executable
MSDTCPRX.DLL MS DTC interface proxy DLL
MSDTCTM.DLL Transaction manager state machine DLL
UTIL.LIB Name service/Memory management library
XOLEHLP.DLL MS DTC helper APIs DLL

The MS DTC client utility includes the interfaces that applications use to begin MS DTC transactions. The MS DTC client utility consists of these files.

File Description
ADME.DLL Maintains contact between various MS DTC components
DTCCFG.CPL Remote client configuration utility
DTCCM.DLL Connection manager DLL
DTCHELP.LIB Helper library
DTCUTIL.DLL MS DTC name service DLL
ENUDTC.DLL String resource DLL (one per national language supported)
MSDTCPRX.DLL MS DTC interface proxy DLL
SVCSRVL.DLL Helper for DTCCFG.CPL
UTIL.LIB Name service/Memory management library
XOLEHLP.DLL MS DTC helper APIs DLL
