Trace

Use the Trace slider to control the level of trace information sent from the MS DTC process to the graphical interface. These are the settings on the slider:

  • Send no traces.
  • Send only error traces.
  • Send error and warning traces.
  • Send error, warning, and informational traces (default).
  • Send all traces.

Sending no traces increases MS DTC's speed performance. Sending all traces decreases MS DTC's speed performance.
