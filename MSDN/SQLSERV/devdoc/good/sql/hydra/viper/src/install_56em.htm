Removing an MS DTC Installation

Removing MS DTC is useful, for example, as a preliminary step in converting from a server installation to a client installation, or vice versa, on a networked computer.

  • To remove a complete MS DTC installation from a server, you must run the SQL Server setup program and remove SQL Server from its host computer.
  • To remove an MS DTC client utility from a client computer, you must run the setup program from your Microsoft SQL Server CD.

For more information, see Microsoft SQL Server Setup.
