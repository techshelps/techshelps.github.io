Document Conventions

This documentation uses the following conventions to distinguish elements of text:

Convention Purpose
UPPERCASE Represents statement and clause names, structure names, datatypes, filenames, and paths.
SMALL CAPITALS Represent key names, such as ctrl.
bold Represents functions, configuration-file entries, and syntax.
italic Represents database names, table names, column names, index names, binding types, and variables that appear in text.
monospace Represents examples and sample program code.
