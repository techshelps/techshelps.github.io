Starting MS DTC from a Client

You can start the complete MS DTC service on remote servers from a client node. Doing so is useful if you want to use the client node as a central console for administrating MS DTC transactions. You must have administrative privileges for each SQL Server on which an MS DTC service you want to start resides.

To start a remote MS DTC service from a client node, use SQL Server Enterprise Manager. The procedure is the same as that described in Starting MS DTC from the Server.
