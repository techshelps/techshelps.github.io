About Guide to Microsoft Distributed Transaction Coordinator

This book is organized as follows:

  • Chapter 1, Introduction provides an introduction to transactions written by Jim Gray
  • Chapter 2, Setup provides instructions for installing and setting up MS DTC
  • Chapter 3, Administration provides details about the MS DTC user interface in SQL Enterprise Manager
  • Chapter 4, Application Development provides information about how to use MS DTC with Transact-SQL and C or C++ applications
  • Chapter 5, MS DTC Programmer's Reference provides information about he supported ITransaction methods
