The MS DTC Configuration Dialog Box

Use the MS DTC Configuration dialog box to:

  • Change MS DTC display parameters.
  • Reset the MS DTC log.
  • Start and stop the MS DTC.

You can change display parameters while the MS DTC is running. You must stop the MS DTC before resetting the log and timers.

    To access the MS DTC Configuration dialog box
  • Click the right mouse button on the MS DTC icon in the Server Manager window and from the menu, select Configure.
