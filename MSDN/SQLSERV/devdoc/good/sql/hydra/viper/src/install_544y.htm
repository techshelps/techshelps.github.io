Starting MS DTC from the Server

You can start MS DTC from the server in several ways¾by using SQL Enterprise Manager, SQL Service Manager, the Services application in Control Panel, or from the command prompt.

In each case, MS DTC starts as a service under the Windows NT Service Control Manager by using the default startup options written to the Windows NT Registry by the setup program.

    To start MS DTC from SQL Enterprise Manager
  1. From the Microsoft SQL Server 6.5 program group, start SQL Enterprise Manager.
  2. If you have not yet registered the server you want to use, from the Server menu, choose Register Server to do so.
  3. From the Server Manager window, choose the plus sign (+) beside the server group that you want to open.
  4. Click the right mouse button on the server you want to use.
  5. From the shortcut menu, choose Start.
  6. In the Server Manager window, choose the plus sign (+) beside the name of the server that hosts the MS DTC you want to start.

    The available Server services display.

  7. Right-click the Distributed Transaction Coordinator icon to display this shortcut menu:

  8. To start MS DTC either:
    • Choose Start. The Distributed Transaction Coordinator icon turns green once MS DTC has started.
    • Or
    • Choose Configure. When the MS DTC Configuration dialog box displays, choose the Start button.
    To start MS DTC from SQL Service Manager
  1. From the Microsoft SQL Server 6.5 group, choose SQL Service Manager.

    The SQL Service Manager dialog box appears.

  2. If the service is a remote service, select or enter the remote server name in the Server box, and then choose the Connect button.
  3. In the Services box, select MS DTC.
  4. Double-click the Start / Continue light.
    To start MS DTC from Control Panel
  1. From the Main group, choose Control Panel.
  2. From Control Panel, choose Services.
  3. From the Services dialog box, select MS DTC, and then choose Start.
  4. Choose Close.

Note From the Services dialog box, you can also set MS DTC to automatically start. For information, see your Microsoft Windows NT documentation.

    To start MS DTC from the command prompt
  • Type the following from the command prompt:

    net start msdtc
