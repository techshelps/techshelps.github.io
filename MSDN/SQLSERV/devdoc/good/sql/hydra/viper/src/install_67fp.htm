Complete MS DTC Service

An installation of the complete MS DTC service on a database server includes these components:

  • Distributed transaction coordinator service
  • Extensions to SQL Server Enterprise Manager for graphical management of the MS DTC
  • Interfaces that applications use to begin MS DTC transactions

When you install the complete MS DTC service, the MS DTC creates the log file MSDTC.LOG. This log file is used to record the outcome of all transactions in which the local MS DTC participates.

Caution You should never modify or delete the MSDTC.LOG file.
