About the Example Programs

The Microsoft SQL Server Programmer's Toolkit (PTK) includes sample MS DTC applications for Transact-SQL, ODBC, and DB-Library. The samples are located in the \PTK\SAMPLES\MSDTC directory on the SQL Workstation CD.

The examples use the Microsoft SQL Server pubs database that is installed by Microsoft SQL Server. To use the example programs, identical copies of the pubs database must exist on two systems. The example programs use an MS DTC distributed transaction to update the address of an author on both databases.
