Using Views to Export Data

Using views, you can export data to other applications. For example, suppose you want to use stores and sales tables to analyze sales data by stores in Microsoft® Excel. To do this, you can create a view based on the stores and sales tables. Then you can export the data defined by the view by using the bulk copy program utility (bcp).

For more information on exporting data, see Importing and Exporting Data in Working with Data. For details on bcp, see the Microsoft SQL Server Administrator's Companion and the bcp command-line utility in the Microsoft SQL Server Transact-SQL Reference.
