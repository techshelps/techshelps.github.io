Dropping Constraints

You can remove constraints from a table definition by dropping them.

    To drop a constraint
  • In SQL Enterprise Manager's Server Manager window, double-click the appropriate table, choose the Advanced Features button, choose the tab representing the type of constraint you want to drop, select the constraint from the drop-down list, and then choose the Remove button.

    (For details, search for "constraints" in SQL Enterprise Manager Help.)

Or

  • Use the ALTER TABLE statement.

    For example: 

    ALTER TABLE authors
DROP CONSTRAINT UPKCL_auidind

    drops the constraint created earlier in the example in "PRIMARY KEY constraints."

    (For details, see the ALTER TABLE statement in the Microsoft SQL Server Transact-SQL Reference.)
