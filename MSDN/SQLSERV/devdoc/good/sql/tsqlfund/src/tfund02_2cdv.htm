Renaming Views

You can rename a view, subject to the following guidelines:

  • The view to be renamed must be in the current database.
  • The new name must follow the rules for identifiers.
  • You can rename only views that you own.
  • The database owner can change the name of any user's view.
    To rename a view
  • Execute the sp_rename system procedure.

    For example: 

    sp_rename accounts, owing

    renames the accounts view the owing view.

    (For details, see the sp_rename system stored procedure in the Microsoft SQL Server Transact-SQL Reference.)
