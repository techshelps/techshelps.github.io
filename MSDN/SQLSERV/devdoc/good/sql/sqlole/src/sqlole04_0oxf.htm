    Write: No		 The login of the database owner.

    Write: No		 The SQLServer object.
Size
    Long
    Write: No		 The total storage space in megabytes allocated for this database. This includes both used and available space.

    Write: No		 The unused storage space in kilobytes available for new data.

    Float
    Write: No		 The unused storage space in megabytes (accurate to two decimal places) available for new data.

UserProfile

    Write: No		 The database profile information for the currently connected user.
Version

    Write: No		 The version of the database from sysdatabases.
