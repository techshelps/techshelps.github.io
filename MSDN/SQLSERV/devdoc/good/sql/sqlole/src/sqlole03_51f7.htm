EnumSubscriberSubscriptions List

SQLServer Object

Description

Lists the subscription information for a subscriber SQL Server. List is returned in a QueryResults object.

Visual Basic

QueryResults = SQLServer.EnumSubscriberSubscriptions (
[Subscriber :=] idSubscriberServer,
[[SourceDatabase :=] idDatabase] )

C++

HRESULT pSQLServer->EnumSubscriberSubscriptions (
SQLOLE_LPCSTR idSubscriberServer,
LPSQLOLEQUERYRESULTS* ppQueryResults,
SQLOLE_LPCSTR idDatabase = NULL );
