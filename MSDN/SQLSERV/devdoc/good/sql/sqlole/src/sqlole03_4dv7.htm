Property Description

Application
    Application
    Write: No		 The Application object.
Default
    Boolean
    Write: Yes		 Determines if unused space on this database device should be included in the default device pool. When the user creates a new database or extends an existing database, and either does not specify a database device or specifies the default database device, SQL Server uses available device space from the default device pool for database storage. If True, unused space on this device is included in the default device pool. If False, either this device is not included in the default device pool, or this is a dump device.
