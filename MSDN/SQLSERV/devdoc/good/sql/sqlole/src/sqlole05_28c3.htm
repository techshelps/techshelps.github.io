EnumDependencies List

SQLServer.Database.Table Object

Description

Lists the database objects that depend on this Table, or database objects that this Table depends on, in dependency order. List is returned in a QueryResults object.

Visual Basic

QueryResults = Table.EnumDependencies ( [[DependencyType :=] tDependency] )

C++

HRESULT pTable->EnumDependencies (
LPSQLOLEQUERYRESULTS* ppQueryResults, SQLOLE_DEPENDENCY_TYPE tDependency = SQLOLEDep_Parents );
