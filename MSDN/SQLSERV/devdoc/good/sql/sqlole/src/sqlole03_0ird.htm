EnumHistory List

SQLServer Object

Description

Lists SQL Server task history. List is returned in a QueryResults object. Set the properties of a created HistoryFilter object as desired and pass it to the EnumHistory method.

Visual Basic

QueryResults = SQLServer.EnumHistory ( [[HistoryFilter :=] HistoryFilter] )

C++

HRESULT pSQLServer->EnumHistory (
LPSQLOLEQUERYRESULTS* ppQueryResults,
LPSQLOLEHISTORYFILTER HistoryFilter = NULL );
