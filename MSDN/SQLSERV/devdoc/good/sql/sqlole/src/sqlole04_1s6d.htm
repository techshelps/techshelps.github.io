ExecuteImmediate Method

SQLServer.Database Object

Description

Executes a Transact-SQL statement on a database. No results are returned to the application.

Visual Basic

Database.ExecuteImmediate (
[Command :=] strSQLStatement,
[[ExecType :=] tExecOption] )

C++

HRESULT pDatabase->ExecuteImmediate (
SQLOLE_LPCSTR strSQLStatement,
SQLOLE_EXEC_TYPE tExecOption = SQLOLEExec_Default );

Elements

Element Type Description

strSQLStatement String SQL statement
tExecOption SQLOLE_EXEC_TYPE Options for executing this SQL statement
