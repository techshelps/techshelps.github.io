ListUserPermissions List

SQLServer.Database.Table Object

Description

Returns a list of Permission objects containing the specified user's privileges for this table.

Visual Basic

PermissionList = Table.ListUserPermissions ( [UserName :=] idUser )

C++

HRESULT pTable->ListUserPermissions (
SQLOLE_LPCSTR idUser,
LPSQLOLEPERMISSIONLIST* ppPermissionList );

Elements

Element Type Description

PermissionList PermissionList Returned list of Permission objects
idUser Identifier User
