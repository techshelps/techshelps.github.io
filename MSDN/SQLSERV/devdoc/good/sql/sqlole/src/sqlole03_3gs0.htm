SwitchToMirrorTemp Method

SQLServer.Device Object

Description

Temporarily disables disk mirroring for a device. Until the ReMirror method is used, only the mirror device is used for storage, and the primary database device is temporarily disabled. Use the ReMirror method to reestablish disk mirroring.

Visual Basic

Device.SwitchToMirrorTemp

C++

HRESULT pDevice->SwitchToMirrorTemp ( );
