UnMirrorTemp Method

SQLServer.Device Object

Description

Temporarily stops disk mirroring for a device. Until the ReMirror method is used, only the primary database device is used for storage, and the mirror device is temporarily disabled. Use the ReMirror method to reestablish disk mirroring.
