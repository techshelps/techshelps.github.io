ListPermissions List

SQLServer.Database.DBObject Object

Description

Returns a list of Permission objects containing the object privileges for this object.

Visual Basic

PermissionList = DBObject.ListPermissions ( [[PrivilegeTypes :=] tPrivileges] )

C++

HRESULT pDBObject->ListPermissions (
LPSQLOLEPERMISSIONLIST* ppPermissionList ,
SQLOLE_PRIVILEGE_TYPE tPrivileges = SQLOLEPriv_AllObjectPrivs );

Elements

Element Type Description

PermissionList PermissionList Returned list of Permission objects
tPrivileges SQLOLE_PRIVILEGE_TYPE Types of object privileges to list
