ListBoundDatatypes List

SQLServer.Database.Default Object

Description

Returns a list of UserDefinedDatatype objects that are currently bound to this Default.

Visual Basic

UserDefinedDatatypeList = Default.ListBoundDatatypes

C++

HRESULT pDefault->ListBoundDatatypes ( LPSQLOLEUSERDEFINEDDATATYPELIST*
ppUserDefinedDatatypeList );

Elements

Element Type Description

UserDefinedDatatypeList UserDefinedDatatypeList Returned list of UserDefinedDatatype objects
