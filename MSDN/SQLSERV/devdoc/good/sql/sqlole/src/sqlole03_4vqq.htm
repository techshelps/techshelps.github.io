ReadBackupHeader Method

SQLServer Object

Description

Retrieves the dump header information for a Backup object. The dump header contains information about every dump currently stored in the dump device. The dump header information is returned in a QueryResults object.

Visual Basic

QueryResults = SQLServer.ReadBackupHeader ( [LoadSpec :=] Backup )

C++

HRESULT pSQLServer->ReadBackupHeader (
LPSQLOLEBACKUP Backup,
LPSQLOLEQUERYRESULTS* ppQueryResults );

Element Type Description

QueryResults QueryResults Returned QueryResults object
Backup Backup Backup object containing the backup specifications
