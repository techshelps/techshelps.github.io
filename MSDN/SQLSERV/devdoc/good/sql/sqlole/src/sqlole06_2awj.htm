ListSubSystemTasks List

SQLServer.Executive Object

Description

Returns a list of Task objects for a specific subsystem.

Visual Basic

TaskList = Executive.ListSubSystemTasks ( [SubSystem :=] strSubSystem )

C++

HRESULT pExecutive->ListSubSystemTasks (
SQLOLE_LPCSTR strSubSystem,
LPSQLOLETASKLIST* ppDBObjectList );

Elements

Element Type Description

TaskList TaskList Returned list of Task objects
strSubSystem String Subsystem to list tasks of. Can be "Distribution", "LogReader", or "Sync".
