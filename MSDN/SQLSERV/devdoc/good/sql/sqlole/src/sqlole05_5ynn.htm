EnumDependencies List

SQLServer.Database.Table.Trigger Object

Description

Lists the database objects that depend on this Trigger, or database objects that this Trigger depends on, in dependency order. List is returned in a QueryResults object.

Visual Basic

QueryResults = Trigger.EnumDependencies ( [[DependencyType :=] tDependency] )

C++

HRESULT pTrigger->EnumDependencies (
LPSQLOLEQUERYRESULTS* ppQueryResults, SQLOLE_DEPENDENCY_TYPE tDependency = SQLOLEDep_Parents );
