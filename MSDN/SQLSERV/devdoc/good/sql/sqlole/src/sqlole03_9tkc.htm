UnloadODSDLL Method

SQLServer Object

Description

Unloads an Open Data Services DLL, such as an extended stored procedure DLL, from SQL Server memory.

Visual Basic

SQLServer.UnloadODSDLL ( [DLLName :=] strDLL )

C++

HRESULT pSQLServer->UnloadODSDLL ( SQLOLE_LPCSTR strDLL );

Elements

Element Type Description

strDLL String Open Data Services DLL file
