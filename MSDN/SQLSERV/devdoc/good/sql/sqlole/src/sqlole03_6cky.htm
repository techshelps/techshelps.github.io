ProcessOutputBuffer Method

SQLServer Object

Description

Retrieves the SQL Server output buffer for a SQL Server process.

Visual Basic

strOutputBuffer = SQLServer.ProcessOutputBuffer ( [ProcessID :=] lProcess )

C++

HRESULT pSQLServer->GetProcessOutputBuffer (
long lProcess,
SQLOLE_LPBSTR pstrOutputBuffer );

Elements

Element Type Description

strOutputBuffer String Returned SQL Server output buffer
lProcess Long SQL Server process

Remarks

This is an indexed property.
