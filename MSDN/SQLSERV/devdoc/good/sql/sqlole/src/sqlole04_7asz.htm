ListUserPermissions List

SQLServer.Database.StoredProcedure Object

Description

Returns a list of Permission objects containing the specified user's privileges for this stored procedure.

C++

SQLOLE_LPCSTR idUser,
LPSQLOLEPERMISSIONLIST* ppPermissionList );
