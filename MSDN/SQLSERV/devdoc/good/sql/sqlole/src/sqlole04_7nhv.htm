DedicateLogDevices Method

SQLServer.Database.TransactionLog Object

Description

Marks database devices as usable only for transaction log information.

Visual Basic

TransactionLog.DedicateLogDevices ( [DeviceNames :=] mstrDevices )

C++

HRESULT pTransactionLog->DedicateLogDevices ( SQLOLE_LPCSTR mstrDevices );

Elements

Element Type Description

mstrDevices MultiString Delimited list of one or more database device usage specifiers. Each usage specifier consists of either the database device name alone, or the database device name, an equal sign, and a size in megabytes.

Reference

ALTER DATABASE, sp_logdevice
