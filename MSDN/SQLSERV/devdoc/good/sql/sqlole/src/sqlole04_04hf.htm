ListBoundDatatypes List

SQLServer.Database.Rule Object

Description

Returns a list of UserDefinedDatatype objects that are currently bound to this Rule.

Visual Basic

UserDefinedDatatypeList = Rule.ListBoundDatatypes

C++

HRESULT pRule->ListBoundDatatypes ( LPSQLOLEUSERDEFINEDDATATYPELIST*
ppUserDefinedDatatypeList );

Elements

Element Type Description

UserDefinedDatatypeList UserDefinedDatatypeList Returned list of UserDefinedDatatype objects
