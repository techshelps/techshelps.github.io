ListDatabasePermissions List

SQLServer.Database.Group Object

Description

Returns a list of Permission objects containing the statement privileges for this group.

Visual Basic

PermissionList = Group.ListDatabasePermissions ( [[PrivilegeTypes :=] tPrivileges] )

C++

HRESULT pGroup->ListDatabasePermissions (
LPSQLOLEPERMISSIONLIST* ppPermissionList ,
SQLOLE_PRIVILEGE_TYPE tPrivileges =
SQLOLEPriv_AllDatabasePrivs );

Elements

Element Type Description

PermissionList PermissionList Returned list of Permission objects
tPrivileges SQLOLE_PRIVILEGE_TYPE Types of statement privileges to list
