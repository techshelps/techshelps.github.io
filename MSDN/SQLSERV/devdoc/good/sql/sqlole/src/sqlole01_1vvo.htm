Getting Started

SQL Distributed Management Objects (SQL-DMO) are 32-bit Component Object Model (COM) objects for the Microsoft Windows 95 and Windows NT operating systems. The SQL-DMO objects are OLE Automation Compatible. The SQL-DMO object model includes objects, properties, methods, and collections that are used to write programs that can administer multiple SQL Servers distributed across a network. SQL-DMO programs can range from simple Visual Basic scripts to complex C++ applications. SQL-DMO is the foundation of SQL Enterprise Manager.
