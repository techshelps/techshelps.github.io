ListUserPermissions List

SQLServer.Database.DBObject Object

Description

Returns a list of Permission objects containing the specified user's privileges for this object.

Visual Basic

PermissionList = DBObject.ListUserPermissions ( [UserName :=] idUser )

C++

HRESULT pDBObject->ListUserPermissions (
SQLOLE_LPCSTR idUser,
LPSQLOLEPERMISSIONLIST* ppPermissionList );

Elements

Element Type Description

PermissionList PermissionList Returned list of Permission objects
idUser Identifier User
