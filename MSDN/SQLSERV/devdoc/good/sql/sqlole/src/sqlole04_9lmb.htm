EnumDependencies List

SQLServer.Database.StoredProcedure Object

Description

Lists the database objects that depend on this stored procedure, or database objects that this stored procedure depends on, in dependency order. List is returned in a QueryResults object.

Visual Basic

QueryResults = StoredProcedure.EnumDependencies (
[[DependencyType :=] tDependency] )

C++

HRESULT pStoredProcedure->EnumDependencies (
LPSQLOLEDBOBJECTLIST* ppQueryResults,
SQLOLE_DEPENDENCY_TYPE tDependency = SQLOLEDep_Parents );
