ListOwnedObjects List

SQLServer.Database.User Object

Description

Returns a list of DBObject objects that this user owns.

Visual Basic

DBObjectList = User.ListOwnedObjects (
[[ObjectTypes :=] tObjects,]
[[SortBy :=] tSortBy] )

C++

HRESULT pUser->ListOwnedObjects (
LPSQLOLEDBOBJECTLIST* ppDBObjectList,
SQLOLE_OBJECT_TYPE tObjects = SQLOLEObj_AllDatabaseObjects,
SQLOLE_OBJSORT_TYPE tSortBy = SQLOLEObjSort_Name );

Elements

Element Type Description

DBObjectList DBObjectList Returned list of DBObject objects
tObjects SQLOLE_OBJECT_TYPE Types of database objects to list
tSortBy SortType How to sort the list
