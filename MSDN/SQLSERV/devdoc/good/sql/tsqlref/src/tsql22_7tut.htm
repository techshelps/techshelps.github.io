sp_helpremotelogin System Stored Procedure

Reports information about a particular remote server's logins or about all remote servers' logins.

Syntax

sp_helpremotelogin [remoteserver [, remotename]]

where

remoteserver
Specifies the server about which remote login information is to be displayed.
remotename
Specifies a particular remote user on the remote server.

Examples

A.    Help on a Single Server

This example displays information about all remote users of the remote server ACCOUNTS.

sp_helpremotelogin ACCOUNTS
B.    Help on All Remote Users

This example displays information about all remote users of all remote servers known to the local server.

sp_helpremotelogin

Permission

Execute permission defaults to the public group.

Tables Used

master.dbo.sysremotelogins, master.dbo.sysservers

See Also
