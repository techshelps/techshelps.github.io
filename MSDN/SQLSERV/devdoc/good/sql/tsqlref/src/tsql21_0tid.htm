sp_droptype System Stored Procedure

Deletes a user-defined datatype from systypes.

Syntax

sp_droptype typename

where

typename
Is the name of a user-defined datatype that you own.

Remarks

The typename user-defined datatype cannot be dropped if tables or other database objects reference it.

Example

This example drops the user-defined datatype named birthday.

sp_droptype birthday

Permission

Execute permission defaults to the database owner and the datatype owner.

Tables Used

syscolumns, sysobjects, systypes, sysusers

See Also
