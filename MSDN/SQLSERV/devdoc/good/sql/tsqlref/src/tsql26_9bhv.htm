sysremotelogins (master database only)

Contains one row for each remote user who is allowed to call remote stored procedures on this SQL Server.

Column Datatype Description
remoteserverid smallint Remote server identification
remoteusername varchar(30) User's login name on a remote server
suid smallint Local server user ID
status smallint Bitmap of options

Index

csysremotelogins clustered, unique on remoteserverid, remoteusername
