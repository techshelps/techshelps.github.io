syssegments (all databases)

Contains one row for each segment (named collection of disk fragments). When SQL Server is installed, this table contains three entries: segment 0 (system) for system tables, segment 1 (default) for other objects, and 2 (log segment) for the transaction log (the syslogs table). All entries in sysusages contain both of these segments and their maps.

Column Datatype Description
segment smallint Segment number
name varchar(30) Segment name, used internally
status smallint Default segment

Index

This table does not have any indexes.
