sp_defaultlanguage System Stored Procedure

Changes a user's default language.

Syntax

sp_defaultlanguage login_id [, language]

where

login_id
Is the user's login ID.
language
Specifies the user's default language.

Remarks

A default language can be set by using either sp_defaultlanguage or sp_addlogin when the user's login ID is initially added to SQL Server. If a language is not specified, the user's default language is set to the server's default language.

If a user's default language is dropped from the server, the server's default language is used as the initial language setting and a message is displayed.

The system administrator can change anyone's default language by using sp_defaultlanguage. Users can change only their own default language.

Example

This example sets the default language for Claire to French.

sp_defaultlanguage Claire, french

Permission

Execute permission defaults to public for the user's own login only. Execute permission defaults to the system administrator for any login ID.

Tables Used

master.dbo.syslanguages, master.dbo.syslogins
