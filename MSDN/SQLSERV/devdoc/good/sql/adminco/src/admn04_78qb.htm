Windows- and Windows NT-based Clients

To set up a server connection entry that includes a server name, the correct DLL name for the client operating system (DBMSSPXN for Windows NT - based clients and DBMSSPX3 for Windows-based clients), and a connection string, use the Advanced button of the SQL Client Configuration Utility. The format of the connection string is as follows:

NIC_address,port,network_number

For example:

08012B327C21,33854,1

You can make a direct connection from a client by using the server name that you entered in the Server box when setting up the connection string using the SQL Client Configuration Utility. For example, if you set up the connection using the server name SQLNOVELL, you could make a direct connection by using the ISQL/w application. In the Connect Server dialog box specify the server SSQLNOVELL, enter a valid username and password, and choose the Connect button.
