Displaying Current Server Activity

Only the SA can open the Current Activity window.

    To display current server activity
  1. From the Server Manager window, select a server.
  2. From the toolbar, choose the Current Activity button.

    The Current Activity window appears.

The Current Activity window has three tabs:

  • The User Activity tab displays information about the current connections and locks. For more information, see Viewing User Activity.
  • The Detail Activity tab displays more detailed information about the process number, status, locks, and commands that the active users are running. For more information, see Viewing Detail Activity.
  • The Object Locks tab displays information about the objects that are locked and the kinds of locks that are present. For more information, see Viewing Object Locks.

While viewing any of the tabs in the Current Activity window, you can display a legend explaining the graphics used in the list. You can also update the display with current information.

    To display a legend explaining the graphics used in the list
  • Choose the Toggle Legend button.

    The Activity Legend appears.

    To update the list with current information
  • Choose the Refresh on-screen information button.

If you have SA privileges, you can also select an item from the list and:

  • Display detailed information about the selected process. For more information, see Viewing More Information.
  • Send a message to the connected user or computer. For more information, see Sending a Message.
  • Kill the selected process. For information see Killing a Process.
