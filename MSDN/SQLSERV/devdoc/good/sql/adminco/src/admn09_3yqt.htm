Removing a Group from a Database

When you drop a group, all users in that group are automatically removed from the group. Users who were members of the dropped group are still users in the database, and members of the public group.

    To remove a group from a database
  1. From the Server Manager window, select a server, and then open the Databases folder for that server.

    The databases of that server are listed.

  2. Select a database, and open the Groups/Users folder for that database.

    The groups and users of that database are listed.

  3. Select a group, and then press the Delete key.

    Or use the right mouse button to click on the group name, and then from the drop-down menu that appears, choose Delete.

    The group is removed from the list.
