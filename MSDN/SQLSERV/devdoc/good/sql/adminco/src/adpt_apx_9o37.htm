Appendixes

The appendixes provide supplemental information.

Appendix A, Interoperation with Microsoft SQL Server 4.2, discusses the simultaneous operation in an enterprise of Microsoft SQL Server 6.0 and Microsoft SQL Server version 4.2.

Appendix B, Estimating Database Size, contains formulas and examples that show how to estimate the size of tables and indexes in a database.

Appendix C, System Tables Diagram, presents a chart of the relationships between the SQL Server system tables.

Appendix D, RAID Technology, discusses using drive subsystems to tune a database server for optimal performance.

Appendix E, Backup and Recovery Guidelines for Production Environments, discusses backup and recovery of SQL Server databases running in production environments.

Appendix F, Accessibility for People with Disabilities, provides information about products and service that make Microsoft Windows NT, Windows, and SQL Server more accessible for people with disabilities.
