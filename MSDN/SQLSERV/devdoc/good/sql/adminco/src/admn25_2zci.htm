Error 17952

Level 18

Message Text

Failed to start Microsoft Mail session. Check the errorlog file in the SQL Server directory for details.

Explanation

The xp_startmail extended stored procedure could not start a Mail session for SQL Server. This error is received by the client that performs the xp_startmail extended stored procedure.

Action

Follow the directions given about using SQLMail in Chapter 18, "Setting Up Mail."

If this error occurs again, call your primary support provider for assistance.

For more information about troubleshooting SQLMail, see Chapter 18, "Setting Up Mail."
