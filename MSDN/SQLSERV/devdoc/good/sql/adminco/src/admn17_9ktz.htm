Accessing the Alerts Window

Alerts are administered from the Manage Alerts window of SQL Enterprise Manager, which can be accessed in several ways.

    To access the Manage Alerts window
  1. From the Microsoft SQL Server 6.0 program group, double-click the SQL Enterprise Manager icon.
  2. From the Server Manager window, select a server.
  3. Open the Manage Alerts window by using one of these techniques:
    • From the toolbar, choose the Manage Alerts button.

    • From the Server menu, choose Alerts.
    • In the Server Manager window, under the selected server, click the right mouse button on the SQL Executive icon, and then from the drop-down menu that appears, choose Edit Alerts.

    After you perform one of those techniques, the Manage Alerts window appears.

For simplicity, the procedures in this chapter assume that you will use the Manage Alerts button from the toolbar. However, you can substitute any of the other techniques.
