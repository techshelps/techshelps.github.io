Manual Replication

SQL Enterprise Manager provides full functionality for setting up and maintaining replication. Replication can be managed using this easy-to-use, graphical interface. However, as an option you can set up replication manually, primarily through the use of replication and scheduling stored procedures. This section provides a summary of the steps you will accomplish to set up replication manually, shows you a sample schema creation script and describes how to manually perform partitioning.
