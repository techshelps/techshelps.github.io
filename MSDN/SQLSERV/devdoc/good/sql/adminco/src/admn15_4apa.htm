Tables Used by Replication

Replication employs tables from publication databases, the distribution database, destination databases, and the master database.

In addition to the tables described here, SQL Executive uses the scheduling tables in the msdb database to handle scheduling of replication tasks. For information on the tables in the msdb database, see the Microsoft SQL Server Transact-SQL Reference.
