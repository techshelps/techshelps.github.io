SQLServer-Log Object

The following counters are part of the SQLServer-Log object, which includes statistics for SQL Server transaction logs. Each of these counters displays statistics for the transaction log for a specific database.

Log Size (MB)
The transaction log size.
Log Space Used (%)
The percentage of the transaction log that is full. When the log is full, transactions that require logging are rejected. You can use this statistic to set a Performance Monitor alarm based on a percent full threshold and then spawn a task to dump or truncate the transaction log or to send a message (for example, using net send).

You can use the alerter utility, sqlalrtr, to generate a SQL Server alert on a threshold for this statistic (or for any Performance Monitor statistic). Alert response can include email or pager email notification of an operator. For more information and configuration instructions, see Chapter 17, Managing Alerts.
