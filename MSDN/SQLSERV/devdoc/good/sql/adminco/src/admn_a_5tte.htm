Interoperation with Microsoft SQL Server 4.2

In most ways, SQL Server 6.0 servers and clients interoperate normally with SQL Server 4.2 servers and clients. This appendix discusses interoperation issues and exceptions.

Note The information in this appendix addresses administration and connectivity topics. For information about language, statement, query, and other compatibility issues between SQL Server 6.0 and SQL Server 4.2 databases, see the compatibility information provided in the Microsoft SQL Server Transact-SQL Reference.
