Monitoring Server Performance

SQL Performance Monitor¾the integration of Microsoft SQL Server 6.0 with the Windows NT Performance Monitor¾makes it possible for you to get up-to-the-minute activity and performance statistics about SQL Server.

SQL Performance Monitor obtains statistics from performance counters, several of which are grouped within a performance object.

When you install SQL Server, the setup program automatically provides six performance objects to SQL Performance Monitor.
