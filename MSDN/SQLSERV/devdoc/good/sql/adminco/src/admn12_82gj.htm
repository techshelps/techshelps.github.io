Basic Backup and Restore Operations

Backup and restoration are easy to set up and execute using SQL Enterprise Manager. The information in this section shows you how to back up a database or transaction log, execute a backup immediately, configure a backup for scheduled execution, customize the backup command that is executed, monitor scheduled backup tasks, and restore a database or apply a transaction log from a backup.
