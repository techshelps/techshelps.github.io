Replication Maintenance

With replication, the modification of data at one site can affect data being used at one or many remote locations. When a production system consists of multiple distinct locations, an administrator must consider the relationships between those locations when designing a method of maintaining data that resides on servers.

This section discusses some of the issues that should be considered when establishing a maintenance plan for servers that participate in a distributed data replication system.
