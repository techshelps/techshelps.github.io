pubs Database

The pubs database is a sample database provided as a learning tool. The pubs database is the basis of most of the examples in the Microsoft SQL Server documentation. This sample database is described in the Microsoft SQL Server Transact-SQL Reference.

After pubs has been used as a learning tool, you may need to reinstall it to restore its original tables, schema, and content. For instructions about how to reinstall pubs, see the Microsoft SQL Server Transact-SQL Reference.

If the pubs database is not required for learning purposes, you can drop it.
