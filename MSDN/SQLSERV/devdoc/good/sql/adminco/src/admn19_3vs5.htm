Monitoring Server Activity and Performance

This chapter shows you how to monitor SQL Server activity and performance using SQL Performance Monitor and SQL Enterprise Manager. It also explains how to use the sp_monitor system stored procedure to view SQL Server statistics and how to view the SQL Server and Windows NT logs.

This chapter focuses on server activity and performance. For information about database performance, integrity, and optimization, see the Microsoft SQL Server Database Developer's Companion.
