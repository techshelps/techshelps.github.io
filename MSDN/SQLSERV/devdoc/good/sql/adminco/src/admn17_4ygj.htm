Managing Alerts

SQL Enterprise Manager and the SQL Executive service provide an easy way to set alerts on SQL Server events. If the event occurs, SQL Executive will respond automatically, either by executing a task that you have defined or by sending an email and/or a pager message to an operator you have specified.
