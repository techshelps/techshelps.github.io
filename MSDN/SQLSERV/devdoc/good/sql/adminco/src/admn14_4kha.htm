Setting Up Replication

To establish replication between servers of your enterprise, you set each server's distribution, publication, and subscription options, publish data from publication servers, and subscribe to published data from subscription servers. This chapter explains how to perform these tasks. It also discusses planning and prerequisites and explains how to stop replication.

The procedures presented in this chapter assume you will be using SQL Enterprise Manager to perform replication administration tasks. If you need to set up replication manually, see the instructions in Chapter 15, Advanced Replication. For background and conceptual information about replication, see Chapter 13, Replication Concepts.
