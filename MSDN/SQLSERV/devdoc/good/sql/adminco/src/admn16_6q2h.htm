Viewing Task History

You can view information about the last run date of a task, the time and duration of the last run, the status of that run, and whether any email was sent by SQL Executive. You can also view any messages encountered while running the task.

    To view task history
  1. From the Server Manager window, select a server, and then from the toolbar, choose the Task Scheduling button.

    The Task Scheduling window appears.

  2. From the Task List, select the task to view history for.
  3. Choose the Task History button.

    The Task History dialog box appears. The following information is displayed:

    • Run Date shows the date the information is about.
    • Run Time shows when the task ran.
    • Status shows whether the task completed successfully.
    • Emailed indicates whether email was sent by SQL Executive after task execution.
    • Run Duration shows the amount of time the task took to run.
    • Message displays any messages that resulted from the task execution.
    • Clear All clears (erases) the history for the current task. (Only the task owner can do this.)
