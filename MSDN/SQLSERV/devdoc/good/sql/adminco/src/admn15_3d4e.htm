Monitoring Replication

Once replication has been set up, you can monitor replication activities several ways. You can:

  • Use SQL Performance Monitor to monitor replication-related counters.
  • Use SQL Enterprise Manager to view scheduled replication tasks and to see a history for each task.
  • Set up alerts on replication events. If the event occurs, SQL Enterprise Manager will respond automatically, either by executing a task that you have defined or by sending an e-mail and/or a pager message to an operator that you have specified.
