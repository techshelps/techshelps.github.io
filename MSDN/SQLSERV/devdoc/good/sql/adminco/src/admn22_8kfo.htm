Transaction Log Management

This chapter will assist you in the long-term management of Microsoft SQL Server transaction logs. It includes information about:

  • Separating data and log segments
  • Sizing and truncating the transaction log
  • Managing large transactions
  • Using the bulk copy utility
  • Automating transaction log dumps
  • Detecting long-running transactions

For more information about transaction log management, see Part 3, "Devices, Databases, and Drives."
