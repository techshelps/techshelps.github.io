Administration Overview

As an overview and orientation to SQL Server security administration, three different security summaries are presented here. Each summarizes the tasks you will perform to administer security under each of the three SQL Server login security modes: integrated, standard, and mixed. Each is intended to serve only as a high-level guide to the tasks that will be performed. For more detailed information about setting up each security element, see the other topics in this chapter. For background and conceptual information about SQL Server security, see Chapter 8, Security Concepts.

Also provided in this section is a quick guide to the tools you will use to manage SQL Server security.
