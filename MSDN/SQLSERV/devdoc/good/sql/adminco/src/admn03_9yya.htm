Registering a Server with SQL Enterprise Manager

To be administered by SQL Enterprise Manager, a server must first be registered with SQL Enterprise Manager. Registering a server provides SQL Enterprise Manager with the server name and tells SQL Enterprise Manager which server group the server belongs to. Once it is registered, a server is listed in SQL Enterprise Manager's Server Manager window.

When a new installation of SQL Enterprise Manager, is run for the first time, SQL Enterprise Manager automatically prompts you to register servers. All subsequent times you run SQL Enterprise Manager, to register servers you must follow the instructions provided in the following topics.
