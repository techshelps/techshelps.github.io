Starting the Client Configuration Utility

Depending on the operating system, the SQL Client Configuration Utility is started in slightly different ways.

    To start the SQL Client Configuration Utility from a Windows NT-based client
  1. Open the Microsoft SQL Server 6.0 program group.
  2. Double-click the SQL Client Configuration Utility icon.

    To start the SQL Client Configuration Utility from a Windows 95-based client
  1. Click the Start button, and then point to Programs.
  2. Point to the Microsoft SQL Server 6.0 folder.

    Note If the 16-bit client software is installed, point to the SQL Server Tools folder.

  3. Click the SQL Client Configuration Utility program.
    To start the SQL Client Configuration Utility from a Windows- or Windows for Workgroups-based client
  1. Open the SQL Server Tools program group.
  2. Double-click the SQL Client Configuration Utility icon.

For clients running the Windows or Windows for Workgroups operating systems, the utility can also be started by running W3DBVER.EXE from the \SQL60 directory.
