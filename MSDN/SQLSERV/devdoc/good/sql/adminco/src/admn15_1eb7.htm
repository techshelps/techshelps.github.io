Advanced Replication Examples

This section provides examples that tell you how to implement some advanced replication scenarios. These are more complex than the basic examples provided in Chapter 14, Setting Up Replication. As with the basic examples in that chapter, these advanced replication examples are provided to help orient you to the capabilities of SQL Server 6.0 replication.

You are not limited to these configurations¾many others are possible.
