Permissions of Database Owners

The database owner has full permission to do anything inside his or her database and must explicitly grant permissions to other users.

These permissions are automatically granted to the database owner and cannot be transferred to other users:

  • CHECKPOINT
  • DBCC
  • DROP DATABASE
  • GRANT and REVOKE statement permissions
  • LOAD DATABASE
  • LOAD TRANSACTION
  • SETUSER

Database owners can grant permissions on these statements to other users:

  • CREATE DEFAULT
  • CREATE PROCEDURE
  • CREATE RULE
  • CREATE TABLE
  • CREATE VIEW
  • DUMP DATABASE
  • DUMP TRANSACTION
