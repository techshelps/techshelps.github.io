Sending a Message

If you have SA privileges, you can send a message to a connected user or computer. This feature is available only for Microsoft networks.

    To send a message to a connected user or computer
  1. From the Server Manager window, select a server, and then from the toolbar, choose the Current Activity button.

    The Current Activity window appears.

  2. Select a tab (User Activity, Detail Activity, or Object Lock), and then select an item from the list in that window.
  3. Choose the Send Message button.

    The Send Message dialog box appears.

  4. In the Message box, type the text of the message.
  5. Under Destination, specify where the mail will be sent.
    • Select Using Username to send the message to the displayed username.
    • Select Using Hostname to send the message to the computer name. To change the computer that will receive the message, type a computer name in the Hostname box.

      To send a message to all users connected to the computer, type an asterisk (*) in the Hostname box. This sends a message to all users connected to the computer (not just users connected to SQL Server).

  6. Choose Send.
