Accessing the Task Window

Alerts are administered from the Task Scheduling window of SQL Enterprise Manager, which can be accessed several ways.

    To access the Task Scheduling window
  1. From the Microsoft SQL Server 6.0 program group, double-click the SQL Enterprise Manager icon.
  2. From the Server Manager window, select a server.
  3. Open the Task Scheduling window using one of these techniques:
    • From the toolbar, choose the Task Scheduling button.

    • From the Tools menu, choose Task Scheduling.
    • In the Server Manager window, under the selected server, double-click the SQL Executive icon.

    • In the Server Manager window, under the selected server, click the right mouse button on the SQL Executive icon, and then from the drop-down menu that appears, choose Edit Task.

    After you perform one of those techniques, the Task Scheduling window appears.

For simplicity, the procedures in this chapter assume you will use the Task Scheduling button from the toolbar. However, you can substitute any of the other techniques.
