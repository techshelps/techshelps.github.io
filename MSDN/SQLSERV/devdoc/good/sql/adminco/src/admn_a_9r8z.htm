Support for New Features

For a summary of new SQL Server 6.0 features, see the Release Notes provided with SQL Server Books Online.

As should be expected, SQL Server 4.2 servers do not support new SQL Server 6.0 features such as constraints, new keywords, the identity property, the decimal, numeric, float(b), and double precision datatypes, expanded ANSI SQL support and language enhancements, server cursors, constraints, and so on. For information on these features, see the Microsoft SQL Server Transact-SQL Reference.

SQL Server 4.2 servers are not supported as replication participants. They cannot act as publication servers or distribution servers and are not supported subscribers.
