Viewing Operators

You can view a list of all existing operators, or information about a specific operator.

    To view information about all operators
  1. From the Manage Alerts window, select the Operators tab.

    A list is displayed of all operators defined for this server.

    For each operator, the list displays the operator name, enabled status, email name, pager e-mail name, date and time the last email message was sent, and date and time the last page message was sent.

  2. To update the list with the latest information, choose the Refresh button.

    To view information about a specific operator
  1. From the Manage Alerts window, select the Operators tab.
  2. Select the operator name, and then choose the Edit Operator button. Or double-click an operator name.

    The Edit operator dialog box for that window appears, displaying the options and configuration settings for that operator. The Most Recent Notifications box displays statistics for the operator.
