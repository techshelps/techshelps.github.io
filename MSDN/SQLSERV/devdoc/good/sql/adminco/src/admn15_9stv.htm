Advanced Setup Tasks

Under certain conditions, advanced users may find it useful to:

  • Add custom stored procedures that are run when the log reader process encounters an INSERT, UPDATE, or DELETE transaction that is marked for replication in the transaction log of the publication database
  • Change the distribution database that a publication server uses
  • Edit the replication entries in the Windows NT Registry.
