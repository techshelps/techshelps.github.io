What Setup Installs

Although this documentation does not discuss installation, it is important to know how SQL Server is installed and what is installed with it. The setup program installs system devices and databases, system tables, system stored procedures, a character set and sort order, some default users, administrative tools and utilities, and, optionally, SQL Server Books Online.
