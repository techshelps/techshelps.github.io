Modifying a Task

After a task has been created, you can modify it. You can change all the information and options you specified when you created the task: the name of the task, the type of task, the command information for a task, the task schedule, and the task options.

    To modify a task
  1. From the Server Manager window, select a server, and then from the toolbar, choose the Task Scheduling button.

    The Task Scheduling window appears.

  2. From the task list, select a task, and then choose the Edit Task button or double-click on the name of the task.

    The Edit Task dialog box appears.

  3. Change the task information as appropriate.

    The information managed while modifying a task is the same information managed while creating a task. For details about the task information you can manage, see Creating a Task.

  4. Choose Modify.

    The task is modified.
