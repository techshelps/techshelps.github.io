Editing an Operator

Once you have created an operator, you can change any of the information that defines that operator.

    To edit an operator
  1. From the Manage Alerts window, select the Operators tab.

  2. Select the operator name, and then choose the Edit Operator button. Or double-click the operator name.

    The Edit Operator dialog box for that window appears, displaying the options and settings for that operator. The Most Recent Notifications box displays statistics for the operator.

  3. Change the options as necessary.

    For example, to select another alert for this operator to receive, select the alert name from the Alerts Assigned To This Operator list, and then select the Email and/or the Pager box.

    For information on the configuration options in this dialog box, see Creating an Operator.

  4. Choose OK.
