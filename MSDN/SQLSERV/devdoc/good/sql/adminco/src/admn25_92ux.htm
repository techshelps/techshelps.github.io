Error 17825 (ODS)

Severity Level

Message Text

Unable to close server-side connection

Explanation

This error occurs if SQL Server gets an error when attempting to close the network connection to the client. Normally, this should never happen. This error is sometimes a side effect of other, more serious errors. Focus on the more serious errors first.

Action

Look in the SQL Server error log for any errors that occurred at approximately the same time as error 17825. For further help in resolving this error, consult the Microsoft Knowledge Base.
