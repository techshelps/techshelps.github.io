Remote Servers and Users

A remote server is a SQL Server on the network that a user can access through his or her local server, allowing the user to execute stored procedures on the remote server. This chapter describes remote servers and their remote users, shows you how to set up and configure a remote server, and details how to add remote login IDs for those users who need to access the remote server.

When setting up remote servers and users, you will also need to grant object permissions for the stored procedures. To learn how to grant object permissions, see Chapter 9, Managing Security
