DB-Library Error Messages

This appendix describes all DB-Library error messages and severity levels. The information in this appendix is divided into two tables. The table in the following section lists the errors alphabetically, along with their severities. The table in the "Error Severities" section summarizes the error severity levels. Number values corresponding to the errors are passed to the currently installed, user-supplied error handler. For information about creating error handlers, see the description for dberrhandle in Microsoft SQL Server Programming DB-Library for C. To access these error definitions, include the SQLFRONT.H and SQLDB.H header files in your program.
