Modifying a Login ID

The SA can change the password, default language, and database access (including default database) for a login ID.

    To modify a login ID
  1. From the Server Manager window, select a server, and then open its Logins folder.
  2. Double-click a login ID.

    The Manage Logins dialog box appears, displaying information about that login ID.

  3. Change the appropriate information.
  4. Choose Modify.

For more information about the settings you can modify for a login ID, see Summary of Login ID Information.
