Expanding a Database Device

If you find you need additional space for a database device¾and if there is adequate space on the storage media¾you can expand the space allocated to that device.

    To expand a database device
  1. Open the Edit Database Devices window for the device.

    The Size (MB) box displays the current size of that device.

  2. In the Size (MB) box, type a larger number.

    A database device can be enlarged using this technique, but it cannot be shrunk.

  3. Choose OK.

You can also use the DISK RESIZE statement to increase the size of a database device. For information about the DISK RESIZE statement, see the Microsoft SQL Server Transact-SQL Reference.
