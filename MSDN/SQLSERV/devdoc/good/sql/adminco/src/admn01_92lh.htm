SQL Server Books Online

During installation of server software¾or of client software on 32-bit or 16-bit Windows-based computers¾the setup program gives you the option of installing SQL Server Books Online. Selecting this option installs the SQL Server Books Online software and adds the SQL Server Books Online icon to the Microsoft SQL Server 6.0 program group.

SQL Server Books Online provides on-screen access to the entire SQL Server 6.0 documentation set. A notable and powerful feature is its full-text search capability, which allows you to quickly search for a word or a phrase. You can search through the entire documentation set or through only selected books or topics. For more information on full-text search, see the online Help for SQL Server Books Online.

You can install SQL Server Books Online to run from the disk (which provides the best performance) or to run from the CD (which uses less disk storage space). You can also choose to not install SQL Server Books Online.
