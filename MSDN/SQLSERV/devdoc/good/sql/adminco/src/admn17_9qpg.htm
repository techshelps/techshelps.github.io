Canceling an Alert

If an alert is no longer applicable, you can drop it. Or you can disable an alert, which inactivates the alert but preserves the alert information.

    To drop an alert
  1. From the Manage Alerts window, select an alert.
  2. Choose the Delete Alert button.

    To disable or reactivate an alert
  1. From Manage Alerts window, select an alert, and then choose the Edit Alert button. Or double-click an alert name.

  2. Set the Enabled option.
    • To disable an alert, clear this option.
    • To reactivate a disabled alert, select this option.
  3. Choose OK.
