Disabling a Task

If you do not want a task to run but you do not want to delete it, you can temporarily disable it.

    To disable a task
  1. From the Server Manager window, select a server, and then from the toolbar, choose the Task Scheduling button.

    The Task Scheduling window appears.

  2. From the task list, select a task, and then choose the Edit Task button or double-click on the name of the task.

    The Edit Task dialog box appears.

  3. Clear the Enabled check box.
  4. Choose Modify.

    The task is disabled. It appears in the task list and retains all its settings but will not be run by SQL Executive.

    To reinstate a disabled task, repeat the preceding procedure, but in step 4 select the Enabled check box.
