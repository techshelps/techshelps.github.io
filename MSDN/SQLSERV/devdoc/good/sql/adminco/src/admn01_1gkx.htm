Overview

Microsoft SQL Server 6.0 is a multiuser relational database management system that provides a powerful client/server platform for delivering critical business applications efficiently across multiple environments. It combines the power, scalability, and manageability of the Microsoft Windows NT operating system with the reliability and advanced capabilities of high-end, client/server database management.

This chapter discusses what the setup program installs, provides "quick start" instructions on how to start and log in to SQL Server, explains how to use this documentation to learn how to administer SQL Server, and lists SQL Server 6.0 capacities.
