Configuring Servers for Replication

Before a server can participate in replication as a publisher, you must install or select a distribution database and also set the server's publication options. Before a server can participate in replication as a distributor, you must install a distribution database. Before a server can participate in replication as a subscriber, you must set the server's subscription options.
