Adding Remote Users

There are three ways to set up login IDs for a remote server:

  • A user can have a remote login ID that is different from his or her local login ID.

  • A group or all users can have their remote login IDs the same as their local login IDs.

  • A group or all users can be assigned to one remote login ID.

The techniques used to accomplish these login ID mappings are discussed in the topics that follow.
