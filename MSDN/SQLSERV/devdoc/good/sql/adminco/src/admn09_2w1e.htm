Logging In to a SQL Server

Before you can manage integrated security accounts, you must start SQL Security Manager and log in to a SQL Server.

    To start SQL Security Manager and log into a SQL Server
  1. From the Microsoft SQL Server 6.0 program group, choose the SQL Security Manager icon.

    SQL Security Manager starts, and the Connect Server dialog box appears.

  2. In the Server box, type or select the name of the server for which you will manage integrated security.

    The Server box contains a list of the last five servers logged in to. To see a list of servers on the network, choose the List Servers button.

  3. In the Login ID box, type your login ID. Then, in the Password box, type your password.

    For integrated security connections, you do not need to supply a login ID or password.

  4. Choose the Connect button.

    The SQL Security Manager window opens, and SQL Security Manager connects to the server.
