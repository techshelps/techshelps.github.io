Destination Database Tables

Destination databases maintain replication information in the MSlast_job_info table.

Table Description
MSlast_job_info Contains the job ID of the last job (from a batch) that was successfully applied. A job is a complete transaction stored in the distribution database.

Also, if present, this table contains the publication name, article name, and description of a job that has paused distribution while waiting for manual synchronization.


For more information about this table, see the Microsoft SQL Server Transact-SQL Reference.
