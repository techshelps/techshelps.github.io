Error 2519

Severity Level 16

Message Text

Database Corrupt: The last checkpoint in Sysdatabases is incorrect. Syslogs recorded a different checkpoint

Explanation

This error occurs when the DBCC CHECKCATALOG statement finds that a row in the sysdatabases table has not been updated to reflect a new checkpoint in syslogs.

Action

Sporadic appearance of this error is not a cause for concern. It will normally resolve itself as database use continues. If the error persists, contact your primary support provider. Have the output from the appropriate DBCC statements available for review.
