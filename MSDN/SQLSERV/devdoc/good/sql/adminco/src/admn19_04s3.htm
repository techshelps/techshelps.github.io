Viewing Object Locks

From the Object Lock tab, you can view information about object locks on the server.

    To view object locks
  1. From the Server Manager window, select a server.
  2. From the toolbar, choose the Current Activity button. Or from the Server menu, choose Current Activity.

    The Current Activity window appears.

  3. Select the Object Locks tab.

Each item in the Object Locks list displays an icon indicating the type of lock, the process ID of the process holding the lock, the user, the type of statement that is being executed against the locked object, and the name of the connected computer. A red background indicates a blocking lock.

The information for an item in the list can be expanded or contracted by choosing the plus ("") or minus ("-") adjacent to the item. To display a legend explaining the graphic elements used, choose the Legend button from the toolbar.
