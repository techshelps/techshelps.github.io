Windows Sockets Support for Windows- and Windows for Workgroups-based Clients

Microsoft TCP/IP-32 for Windows for Workgroups 3.11 is tested and supported for use with the Win16 TCP/IP Sockets Net-Library and is distributed with Windows NT Server version 3.5 or later. It allows Windows for Workgroups clients to use a 32-bit protected-mode stack and take advantage of TCP/IP services provided by Windows NT Server version 3.5 or later (such as the DHCP and WINS services).

While not officially tested and supported, third party TCP/IP products that properly implement the Windows Sockets specification may be used.
