Error 15080

Severity Level 17

Message Text 

Failure using SQLMapi, check SQL Server error log.

Explanation

The sp_processmail stored procedure failed to perform one of its mail functions.

Action

If the SQL Server error log contains errors 17903 and 17951, resolve those errors first and then run the sp_processmail stored procedure again. If errors 17903 and 17951 do not appear in the error log, close and restart the client application and run the sp_processmail stored procedure again.

If error 15080 occurs again, call your primary support provider for assistance.

For more information about troubleshooting SQLMail, see Chapter 18, "Setting Up Mail."
