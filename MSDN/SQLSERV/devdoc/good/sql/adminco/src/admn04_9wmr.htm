MS-DOS - based Clients

With MS-DOS - based clients, you cannot set up the connection information in advance, so you must enter the appropriate connection information each time you connect. Simply load the DBMSSPX.EXE TSR, and then connect to the server using the connection string instead of the server name. To make a direct connection using isql, type the following:

isql -Uusername -Ppassword -SNIC_address,port,network_number

For example:

isql -Usa -Pmysecret -S08012B327C21,33854,1
