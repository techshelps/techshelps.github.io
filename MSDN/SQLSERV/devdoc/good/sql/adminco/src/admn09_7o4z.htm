Managing Access for Windows NT Groups and Users

When a server is set for integrated or mixed security, after you set the server security options, you create the Windows NT groups and users that will be authorized to access that SQL Server and then provide those Windows NT groups and users with appropriate access to that SQL Server.

You manage Windows NT groups and users by using Windows NT User Manager. You provide them access to SQL Server by using SQL Security Manager. For more information, choose one of the following topics:

Creating Windows NT Groups and Users
