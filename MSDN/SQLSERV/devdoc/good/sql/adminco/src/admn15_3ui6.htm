Tables, Stored Procedures, and Scripts Used by Replication

This section presents descriptions of the tables, stored procedures, and scripts that are used by replication. Note that in most cases you will not need to directly modify these tables or run these stored procedures or scripts, since when you use the SQL Enterprise Manager to set up and maintain replication it handles those chores for you.
