About the Administrator's Companion

The Microsoft SQL Server Administrator's Companion provides background information about SQL Server administration, and provides instructions for administering SQL Server. It is written for system administrators who are responsible for overseeing and maintaining systems that run SQL Server. It contains information about creating and managing servers, devices, and databases. It explains SQL Server replication, backing up and recovering data, scheduling tasks, managing security, and monitoring performance. It also describes server and client configuration, details error conditions, and provides troubleshooting information.

This book assumes that you are familiar with Transact-SQL®, the enhanced version of SQL (structured query language). (For information about Transact-SQL, see the Microsoft SQL Server Transact-SQL Reference.) This book also assumes that you are familiar with the Microsoft® Windows NT™, Microsoft® Windows® 95, Microsoft Windows, and MS-DOS® operating systems.
