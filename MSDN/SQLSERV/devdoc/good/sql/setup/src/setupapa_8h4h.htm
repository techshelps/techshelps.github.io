ISO 8859-1

This is the default character set. ISO 8859-1 is also known as the Latin 1 or ANSI character set. It is compatible with the ANSI characters used by the Windows NT and Windows operating systems.

The ISO 8859-1 character set might be appropriate if you intend to use Windows NT - and Windows-based clients exclusively, or if you need to maintain exact compatibility with a SQL Server environment for UNIX or VMS.
