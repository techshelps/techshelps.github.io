Hardware Platforms

Microsoft SQL Server 6.0 can be installed on computers that are based on these microprocessor architectures:

  • Alpha AXP™ microprocessors
  • Intel® 32-bit x86-based microprocessors (such as the 80486 or the Pentium™)
  • MIPS® microprocessors

For more information about supported hardware, see the Windows NT Hardware Compatibility List provided with your Windows NT software.
