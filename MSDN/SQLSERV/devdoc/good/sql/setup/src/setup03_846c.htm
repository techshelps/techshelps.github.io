Name, Company, and Product ID

During installation the setup program will prompt you to supply a name, company, and product ID.

Name
Is the name of a user who is responsible for using or administering this server. This is a required entry.
Company
Is the name of your company or other organization. This is an optional entry.
Product ID
Is an identifier provided on the software's CD box. Although this is an optional entry, it is strongly recommended that you enter the product ID, because it is used for product support purposes.
