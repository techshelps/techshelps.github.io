Character Sets

The character set and sort order choices you make during installation determine the types of characters that SQL Server recognizes in your database data. For information about character sets, choose one of the following topics:

Note During an installation, selecting the default character set and sort order results in a rapid installation (in most cases, 5 to 10 minutes or less). Choosing an alternate character set or sort order can result in a somewhat longer installation time (but still less than 30 minutes).
