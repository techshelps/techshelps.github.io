Installing a Standalone Server

The procedure for installing SQL Server software on a standalone computer (one that does not have a network adapter) is nearly the same as the standard installation procedure, except all network and remote setup options are unavailable.

Important If the setup program is unable to start the network, it assumes that you're installing on a standalone computer. If you're installing SQL Server on a computer that has a network adapter and the setup program is unable to start the network, exit the setup program, correct your network problem, and then restart the setup program.
