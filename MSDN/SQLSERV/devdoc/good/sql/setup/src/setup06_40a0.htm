Installation Path

For Windows NT- and Windows 95-based clients, the default SQL Server client installation root directory is C:\SQL60, although you can specify another directory. These are the directories created under the root directory:

Directory Contents
C:\SQL60\BINN
 Windows NT - based client executable files
C:\SQL60\DLL
 Dynamic-link library files
C:\SQL60\INSTALL
 SQL Server Books Online files

The SQL Server setup program places copies of the DB-Library and Net-Library DLLs in the \WINDOWS\SYSTEM32 directory.
