Installing MS-DOS - based Clients

No setup program is provided for MS-DOS -based clients. To install the software, copy the files from the \CLIENTS\MSDOS directory on the SQL Server CD into a \SQL60\BIN directory you create on the client.

It is recommended that you add the \SQL60\BIN directory you create to the computer's path. You can also add an entry to the AUTOEXEC.BAT file to start the SQL Server client software each time the client starts.

To upgrade an existing client installation on a MS-DOS - based computer, first make sure that no SQL Server client components are running on the computer, and then perform the same steps as for a new installation. To be sure a TSR is not loaded, type the following from an operating-system prompt:

enddblib
