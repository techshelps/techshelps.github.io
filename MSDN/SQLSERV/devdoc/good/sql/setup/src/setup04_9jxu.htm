Installing SQL Server

After you've reviewed the information in Chapter 3, "Server Options," and after you've created a user account for SQL Server, you're ready to install Microsoft SQL Server.

You can install the software directly from the Microsoft SQL Server 6.0 CD, or you can copy the software onto a network drive and install from there. If you plan to install from a network drive, simply copy the entire contents of the CD into a shared network directory.
