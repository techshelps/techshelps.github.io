Starting SQL Server Manually

You can start SQL Server 6.0 using SQL Service Manager, SQL Enterprise Manager, or the Services application in the Control Panel. From an operating-system prompt you can start it as a service or as an executable file. You can also start it in a minimal configuration (useful if the setting of a configuration value¾for example, over-committing memory¾has prevented the server from starting).
