Dictionary Order, Case-insensitive, Accent-insensitive

With this sort order, characters are compared according to the following rules:

  • Uppercase and lowercase characters are treated as equivalent.
  • Characters with diacritical marks are treated as equal to their counterparts without diacritical marks.

For example:

A = a = à = á = â = Ä = ä = Å = å
Jose = jose = José = josé

With this sort order, lowercase characters are paired with their corresponding uppercase characters (for example, a and A) and are sorted before any corresponding uppercase character with a diacritical mark (for example, Ä).

For example:

A, a, à, á, â, Ä, ä, Å, å
