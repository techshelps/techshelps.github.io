[MasterPath]

These values correspond to those user types in the MASTER Device Creation dialog box of the setup program. An entry is required for each value.

Entry Description Value
MasterSize MASTER device size, in MB * 20 minimum. More is recommended. For example:
25
MasterDBFileName MASTER device file name A file name ending in the .DAT file name extension. For example:
MASTER.DAT
MasterDBPath The directory where the MASTER device will be placed The path. For example:
\SQL60\DATA
LogicalDBDrive The drive where the MASTER device will be created A drive letter. For example:
C:
*     MasterSize plus 9 MB must be less than the available disk space on the server being
    installed.
