Client Installation

This chapter discusses preinstallation requirements and provides installation instructions for Microsoft SQL Server client components on computers running the Windows NT, Windows 95, Windows, Windows for Workgroups, or MS-DOS operating systems. It also describes how to upgrade existing client installations.

For information on configuring an installed client, see the Microsoft SQL Server Administrator's Companion.
