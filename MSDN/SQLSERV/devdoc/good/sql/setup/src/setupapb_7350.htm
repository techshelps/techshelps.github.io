[NewOptions]

These values correspond to the check boxes at the bottom of the Installation Options dialog box of the setup program. An entry is required for each value.

Entry Description Value
AutoServerService Indicates whether SQL Server will be automatically started with the system CHECKED | NOTCHECKED
AutoExecutiveService Indicates whether SQL Executive will be automatically started with the system CHECKED | NOTCHECKED
BooksOnline Installation selection for SQL Server Books Online 1 Copy to hard disk
2 Run from CD
3 Do not install
