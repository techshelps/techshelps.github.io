Client Platforms

The SQL Server setup program can install 32-bit client software (including SQL Enterprise Manager) on computers running these operating systems:

  • Microsoft Windows NT Server version 3.5 or later. (Version 3.51 is recommended.)
  • Microsoft Windows NT Workstation version 3.5 or later. (Version 3.51 is recommended.)
  • Microsoft Windows 95.

Client software is also provided for computers running these 16-bit operating systems:

  • Microsoft Windows (throughout this book, Windows refers to Windows- and Windows for Workgroups-based clients).
  • MS-DOS
