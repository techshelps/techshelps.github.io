srv_langptr

Returns a pointer to the language request buffer.

Syntax

void *srv_langptr ( SRV_PROC * srvproc );

where

srvproc
Is a pointer to the SRV_PROC structure that is the handle for a particular client connection (in this case, the handle that received the language request). The structure contains the information that the ODS Library uses to manage communication and data between the Open Data Services server application and the client.

Returns

A pointer to the beginning of the language buffer. If there is no language request, returns NULL.

Remarks

The language request buffer can contain an SQL string or an ASCII string. The Open Data Services server application determines which type of string is present and what to do with it.
