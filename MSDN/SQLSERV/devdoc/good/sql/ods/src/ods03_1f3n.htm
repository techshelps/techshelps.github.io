Header Files

The following table details the contents of each Open Data Services header file.

Header file Contains
SRV.H All of the header files listed below. This file is the only header file that developer-defined routines need to include.
SRVAPI.H Function prototype definitions for all Open Data Services functions.
SRVCONST.H Open Data Services constant definitions.
SRVDBTYP.H DB-Library definitions.
SRVMISC.H Open Data Services miscellaneous definitions.
SRVSTRUC.H Open Data Services structure definitions.
SRVTOK.H Open Data Services token definitions.
SRVTYPES.H Open Data Services datatype definitions.
