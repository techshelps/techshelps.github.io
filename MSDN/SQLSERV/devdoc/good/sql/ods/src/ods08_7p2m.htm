Running the PROCSRV Sample Application

After you install Open Data Services and the sample applications, you can start the PROCSRV sample application.

    To start the PROCSRV sample application from an operating-system prompt
  • At the operating-system prompt, enter

PROCSRV

You can minimize this window after PROCSRV starts.

You can also use the service utility to install the PROCSRV application as a Windows NT service. Once PROCSRV is installed as a Windows NT service, you can control PROCSRV using:

  • The net start and net stop commands
  • The SQL Service Manager utility
  • The Services utility in the Windows NT Control Panel

When you start the PROCSRV application, it runs as a separate server application process in the SQL Server environment. SQL Servers and database clients can access PROCSRV in the same way as they access stored procedures on another SQL Server on the network.

The PROCSRV sample application listens for connection requests on the connection information specified in the Windows NT Registry. For details, see Network Setup Options for Open Data Services Server Applications.
