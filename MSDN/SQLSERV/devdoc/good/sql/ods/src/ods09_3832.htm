Running the GATEWAY Sample Application

    To run the GATEWAY application

At the operating-system prompt, enter

gateway -rregistry_key_name -sservername

where

registry_key_name
Is the name of the Registry key under which Open Data Services can find the Windows NT Registry ListenOn value. Open Data Services looks for this key in the following location:

HKEY_LOCAL_MACHINE\
Software\
Microsoft
MSSQLServer\
registry_key_name

servername
Is the name of the SQL Server GATEWAY connects to.

The GATEWAY sample application connects the database client to a SQL Server database.

For information about setting up network access to an Open Data Services server application, see Network Setup Options for Open Data Services Server Applications.
