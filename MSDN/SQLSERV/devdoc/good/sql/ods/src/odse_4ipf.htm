Miscellaneous

The IDS_HOSTDBMS string contains the name of the target DBMS. This is used in the Open Data Services ODBC driver for special cases. Currently the only defined use of this value is for IDS_HOSTDBMS=DB2. When the Open Data Services ODBC driver detects this value, it saves the default database as specified in the data source definition or in the ODBC SQLDriverConnect call. The Open Data Services ODBC driver uses this default database name in the SQLGetInfo call to provide the qualifier for the DB2 system catalogs. It is also used for the database name in a DB2 Create Table IN DATABASE clause.
