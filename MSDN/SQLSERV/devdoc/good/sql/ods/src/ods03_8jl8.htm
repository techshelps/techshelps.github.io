Data Retrieval

Function Description
srv_getuserdata Returns a pointer to private data space in an SRV_PROC structure. To access private data, set the pointer with srv_setuserdata.
srv_setuserdata Sets a pointer to private data space in an SRV_PROC structure. This pointer can be used later with srv_getuserdata.
srv_pfield Returns a field from an SRV_PROC structure.
srv_symbol Converts an Open Data Services token value to a readable string. Use this function to return data, event, status, or error-severity token types.
