About Programming Open Data Services

Microsoft SQL Server Programming Open Data Services describes how to create Open Data Services server applications. It describes the Open Data Services application programming interface (API), provides details about building Open Data Services server applications, provides a directory of all Open Data Services functions and macros, and includes sample code.

This book is written for programmers who are using the C programming language to develop server applications that can interact with Microsoft SQL Server and with client computers running applications based on the DB-Library™ or Microsoft® Open Database Connectivity (ODBC) API.

This book assumes that you are familiar with DB-Library and/or ODBC. It also assumes that you are familiar with Transact-SQL™, the enhanced version of SQL (structured query language). For information about Transact-SQL, see the Microsoft SQL Server Transact-SQL Reference.
