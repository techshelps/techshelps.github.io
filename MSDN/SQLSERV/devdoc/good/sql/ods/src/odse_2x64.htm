The Gateway Resources DLL

The gateway resources DLL encodes specific information about the capabilities of a gateway and its data source. This information is used by the Open Data Services ODBC driver to determine the handling of prepared statements, transaction commit/rollback, isolation levels, and other features of the ODBC interface.

For more information about the sample source code for a resources DLL, see the README.TXT file provided in the C:\SQL60\OPENDS\SAMPLES\ODSODBC directory. This sample is coded to allow the Open Data Services ODBC driver to connect directly to Microsoft SQL Server 6.0.
