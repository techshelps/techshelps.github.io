Building Server Applications

This chapter describes how to build and then install Open Data Services server applications for the Windows NT operating system.

This chapter describes how to precompile, link, and run the sample server applications, GATEWAY, PROCSRV, and XP.DLL, included with Microsoft Open Data Services. The necessary header files, along with a makefile and a build file, are provided in the package. You will use a similar process to compile and link the Open Data Services server applications that you develop.
