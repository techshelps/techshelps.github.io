Remote Stored Procedure Event Processing

Function Description
srv_paramdata Returns the value of a remote stored procedure parameter.
srv_paramlen Returns the length of a remote stored procedure parameter.
srv_parammaxlen Returns the maximum length of a remote stored procedure parameter.
srv_paramname Returns the name of a remote stored procedure parameter.
srv_paramnumber Returns the number of a remote stored procedure parameter.
srv_paramset Sets the output value for a remote stored procedure parameter.
srv_paramstatus Returns the status flags for a remote stored procedure parameter.
srv_paramtype Returns the datatype of a remote stored procedure parameter.
srv_returnval Defines an output parameter value for a language event.
srv_rpcdb Returns the database name component for the current remote stored procedure.
srv_rpcname Returns the procedure name component for the current remote stored procedure.
srv_rpcnumber Returns the number component for the current remote stored procedure.
srv_rpcoptions Returns the run-time options for the current remote stored procedure.
srv_rpcowner Returns the owner name component for the current remote stored procedure.
srv_rpcparams Returns the number of parameters for the current remote stored procedure.
