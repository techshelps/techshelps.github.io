Libraries

Open Data Services server applications require the following libraries:

Library Description Setup
OPENDS60.DLL * A dynamic-link library that contains Open Data Services functions for the Win32 API. Set the PATH environment variable to include the directory in which OPENDS60.DLL resides.
OPENDS60.LIB Contains the import definitions that Open Data Services server applications use. Set the LIB environment variable to include the directory in which OPENDS60.LIB resides.
*    For information about the OPENDS60.DLL library, see
    Installing an Open Data Services Server Application.

By default, the OPENDS60.DLL file is located in the C:\SQL60\DLL directory.
