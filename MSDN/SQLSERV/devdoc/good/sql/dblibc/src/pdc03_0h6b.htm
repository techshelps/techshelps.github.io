Libraries

DB-Library supports two memory models: medium and large. The following table shows the libraries for each model:

Memory model Library file
Large RLDBLIB.LIB

BLDBLIB.LIB (Borland)
Medium RMDBLIB.LIB

BMDBLIB.LIB (Borland)

When linking the application, append to the LIB environment variable the path where the library files reside.
