About Programming DB-Library for C

This book describes the Microsoft DB-Library for C application programming interface (API), a set of C functions used to create client applications that interact with Microsoft SQL Server.

This book describes DB-Library and provides details about building applications for the Windows NT operating system and about porting Windows-based applications to Windows NT system. It also provides a directory of all DB-Library for C functions and includes sample code.

The documentation is written for programmers who are using the C programming language to develop applications for SQL Server. To use this documentation effectively, you should already be familiar with Transact-SQL®, the enhanced version of SQL (structured query language). For information about Transact-SQL, see the Microsoft SQL Server Transact-SQL Reference.
