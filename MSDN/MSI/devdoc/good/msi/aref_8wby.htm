UnregisterClassInfo Action

[This is preliminary documentation and subject to change.]

The UnregisterClassInfo action manages the unregistration of COM class information with the system. The action unregisters the COM classes from the Class table for which the corresponding feature is currently selected to be uninstalled.

Database Tables Queried

Class Table

Properties Used

None

Sequencing Restrictions

The UnregisterClassInfo action must be sequenced after the InstallFiles action and before the RegisterClassInfo action.
