RAMDiskVolume Control Stylebit

[This is preliminary documentation and subject to change.]

If this bit is set the control lists RAM disks in its list of volumes.

Control Types That Can Have This Style

DirectoryCombo
VolumeCostList
VolumeSelectCombo

Associated Control Attribute

None.

Related Control Stylebits

CDROMVolume
FixedVolume
FloppyVolume
RemoteVolume
RemovableVolume

Numeric Value

0x00100000 = 1048567
