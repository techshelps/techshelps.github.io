REBOOT Property

[This is preliminary documentation and subject to change.]

If this property is set to "Force" the installation will reboot at the end, regardless of what the actions require.If the property is set to "Suppress" the installation will not reboot in case of reboots that happen at the end of the installation before the new program can be used. If the reboot is needed to continue the installation, the reboot will still take place.

Set By

Command line.

Used By

MsiEngine uses this value at termination to override the reboot decision made by the actions.

Default Value

None.

Remarks

This property is typically used by an administrator who installs several products at the same time and wants to have only one reboot at the end of the series of installations.
