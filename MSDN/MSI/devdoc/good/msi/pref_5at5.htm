TemplateFolder Property

[This is preliminary documentation and subject to change.]

Full path to the Template folder for the current user.

Set By

Installer at initialization.

Used By

Actions and Directory table.

Default Value

None.

Remarks

Common values for this property are C:\Winnt\Profiles\[LogonUser]\ShellNew\ (Windows NT) and C:\Windows\ShellNew\ (Windows 95 and Windows 98).
