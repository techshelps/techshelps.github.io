ODBCRemove Action

[This is preliminary documentation and subject to change.]

The ODBCRemove action removes the data sources, translators, and drivers that are listed for removal during the installation.

Database Tables Queried

This action queries the ODBCDataSource table, the ODBCTranslator table, or the ODBCDriver table for each data source, translator, or driver to be removed.

Properties Used

None

Remarks

If both the ODBC usage count and the file usage count become zero, the file is removed. Registration is dependent on ODBC usage counts and file removal based on SharedDLLs key ref counts.
