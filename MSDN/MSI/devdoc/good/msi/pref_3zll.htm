ROOTDRIVE Property

[This is preliminary documentation and subject to change.]

Set to the default drive for the install. This property supplies the default paths for any undefined root directories in the Directory table. When in ADMIN mode, this is set to the first writable connected network drive found. When no network drives are found or when not in ADMIN mode, the location of the writable local drive with the most available free disk space is used.

The value for this property must end with '\'.

Set By

Command line.

Used By

Directory Table use this value to determine the drive used for an application installation.

Default Value

None.
