RegisterUser Action

[This is preliminary documentation and subject to change.]

The RegisterUser action registers the user information with the installer to identify the user of a product.

Database Tables Queried

None

Properties Used

The RegisterUser action uses the USERNAME, the COMPANYNAME, and the ProductID.

Sequencing Restrictions

None

Remarks

The RegisterUser action is not performed during an Admin install. If the Product ID entered by the user has not been validated by the ValidateProductID action, then the ProductID property will not be set and this action does nothing.
