RegisterClassInfo Action

[This is preliminary documentation and subject to change.]

The RegisterClassInfo action manages the registration of COM class information with the system. In advertise mode, the action registers all COM classes from the Class table for which the corresponding feature is enabled. Else the action registers COM classes from the Class table for which the corresponding feature is currently selected to be installed.

Database Tables Queried

Class Table

Properties Used

None

Sequencing Restrictions

The RegisterClassInfo action must be sequenced after the InstallFiles and the UnregisterClassInfo actions.
