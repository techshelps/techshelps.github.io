RegisterProgIdInfo Action

[This is preliminary documentation and subject to change.]

The RegisterProgIdInfo action manages the registration of OLE ProgId information with the system. This action registers all ProgId information from the ProgId table for features that are connected to the extension information (in the Extension table) or the Class information (in the Class table) that are enabled to be advertised or selected to be installed.

Database Tables Queried

ProgId Table

Class Table

Extension Table

Properties Used

None

Sequencing Restrictions

The RegisterProgIdInfo action must be sequenced after the InstallFiles action and the UnregisterProgIdInfo action.
