TimeRemaining ControlEvent

[This is preliminary documentation and subject to change.]

The handler uses this event to broadcast the approximate time remaining, in seconds, for the current progress sequence.

Published By

Handler.

Argument

None.

Action on Subscribers

None.

Action by Publisher When Subscriber Activated

None.

Typical Use

A text control on a modeless dialog subscribes to this event (via the EventMapping table), using the TimeRemaining attribute to display the time remaining.
