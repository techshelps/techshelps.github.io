Sorted Control Stylebit

[This is preliminary documentation and subject to change.]

If this bit is set on a listbox or combobox, the items in the list are displayed in the order defined by the Ordering column of the corresponding table. If the bit is not set, the items are displayed in alphabetical order.

Control Types That Can Have This Style

ComboBox
ListBox
ListView

Associated Control Attribute

None.

Numeric Value

0x00010000 = 65536
