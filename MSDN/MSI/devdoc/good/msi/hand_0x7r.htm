DiskCost Dialog

[This is preliminary documentation and subject to change.]

This dialog box typically appears when the OutOfDiskSpace property is set indicating that there is not enough disk space for the currently selected installation.

This dialog box or a variation of it can also be activated from the Selection Dialog, to give the user an overview of the disk space needed on the different volumes.

The dialog box is modal and normally contains a VolumeCostList control and a PushButton that returns the user to the previous dialog box. It can contain other controls like static text or pictures.
