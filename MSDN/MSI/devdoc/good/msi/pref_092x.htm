DiskPrompt Property

[This is preliminary documentation and subject to change.]

The string displayed by a message box prompting for a disk. This string should include the name of the product and a placeholder for the text printed on the CD surface. See Authoring Disk Prompt Messages.

Set By

Authored into the Property table.

Used By

Advertised as a product property. Used to display Diskprompt dialog when a disk is not detected in any drive.

Default Value

None.
