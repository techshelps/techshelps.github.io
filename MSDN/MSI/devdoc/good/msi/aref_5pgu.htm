RMCCPSearch Action

[This is preliminary documentation and subject to change.]

The RMCCPSearch action uses file signatures to ensure that a user who acquires an upgrade version of a product has the installation for one of the qualifying upgrade products. The action requires the CCP_DRIVE property to be set to the root path on the removable volume that has the installation for any of the qualifying products. The action works on the CCPSearch table.

Database Tables Queried

CCPSearch Table

CCPSearch.Signature_ File signature

Properties Used

CCP_DRIVE

CCP_Success

Sequencing Restrictions

None

Remarks

Each file signature, Signature_, in the table is searched for under the path referred to by the CCP_DRIVE property using the DrLocator table. The absence of the signature from the Signature table denotes a directory. If any signature is determined to exist, the property CCP_Success is set to 1, and the RMCCPSearch action terminates.
