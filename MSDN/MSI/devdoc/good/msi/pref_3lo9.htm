LOGACTION Property

[This is preliminary documentation and subject to change.]

A list of action names, separated only by semicolons (no spaces), for which logging will occur.

Set By

Command line.

Used By

MsiEngine.Message uses this property to determine which actions will generate log information.

Default Value

None.

Remarks

The action names in this property are case-insensitive. The list may optionally end with a semicolon.
