UnregisterMIMEInfo Action

[This is preliminary documentation and subject to change.]

The UnregisterMIMEInfo action unregisters MIME-related registry information from the system. The action unregisters MIME info for servers from the MIME table for which the corresponding feature is currently selected to be uninstalled.

Database Tables Queried

MIME Table

Properties Used

None

Sequencing Restrictions

The UnregisterMIMEInfo action must be sequenced after the InstallFiles action and before the RegisterMIMEInfo action.
