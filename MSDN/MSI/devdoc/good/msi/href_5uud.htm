Default Control Attribute

[This is preliminary documentation and subject to change.]

This attribute describes if the given control is defined as the default control on the dialog. If it is the default, then pressing ENTER automatically activates this control.

Type

Boolean.

Access

Read and Write.

Minimum Size of the Record

At least one field.

Control Types That Have This Attribute

All controls.

Although this attribute is defined for every control, it is only practical for controls that have a button-like behavior.

Associated Control Stylebit

None.
