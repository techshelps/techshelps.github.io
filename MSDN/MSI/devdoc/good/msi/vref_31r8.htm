Persist Method

[This is preliminary documentation and subject to change.]

This is the Persist method of the MsiSummaryInfo object. This method formats and writes the previously stored properties into the standard SummaryInformation stream. It generates an error if the stream cannot be successfully written. This method may only be called once after all the property values have been set. Properties may still be read after the stream is written.

Syntax

object.Persist

Parts

object
MsiSummaryInfo object.
