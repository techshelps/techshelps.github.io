ActionText ControlEvent

[This is preliminary documentation and subject to change.]

The Handler uses this event to broadcast the name of the present action.

Published By

Handler.

Argument

None.

Action on Subscribers

The subscribers are shown when a new progress data arrives from the engine.

Action by Publisher When Subscriber Activated

None.

Typical Use

A Text control on a modeless dialog subscribes to this event using the text attribute to receive the name of the current action.
