ADVERTISE Action

[This is preliminary documentation and subject to change.]

The ADVERTISE action is a top-level action that is called to install or remove advertised components.

Database Tables Queried

The ADVERTISE action queries the following sequence tables for the action to execute, the condition for action execution, and the place of the action in the sequence.

AdvtUISequence Table

AdvtExecuteSequence Table

Properties Used

None

Remarks

Advertisement involves placing information with the system to allow either explicit or implicit Just-in-Time installation of features of a product when needed.

When the installer is in install mode, advertise information for all enabled features.
