ComboList Control Stylebit

[This is preliminary documentation and subject to change.]

If this bit is set on a combobox, the edit field is replaced by a static text field, therefore the user cannot enter a new value, can choose only one of the predefined values. Otherwise the combobox is drawn with an edit field.

Control Types That Can Have This Style

ComboBox

Associated Control Attribute

None.

Numeric Value

0x00020000 = 131072
