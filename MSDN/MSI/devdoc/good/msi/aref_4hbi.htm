ADMIN Action

[This is preliminary documentation and subject to change.]

The Admin action is a top-level action that is used to install an Admin image for workgroup use.

Database Tables Queried

The Admin action queries the following sequence tables for the action to execute, the condition for action execution, and the place of the action in the sequence.

InstallUISequence Table

InstallExecuteSequence Table

AdminUISequence Table

AdminExecuteSequence Table

AdvtUISequence Table

AdvtExecuteSequence Table

Properties Used

None

Sequencing Restrictions

The Admin action is not called from within the action table sequence.

Remarks

The application of transforms during the creation of an Admin image has no valid effect. Therefore to make a customized version of a product available to a workgroup, the transform should be applied during the installation of the product from the Admin image.
