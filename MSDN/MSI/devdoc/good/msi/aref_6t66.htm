RemoveShortcuts Action

[This is preliminary documentation and subject to change.]

The RemoveShortcuts action manages the removal of advertised shortcuts whose feature is selected for uninstall and non-advertised shortcuts whose component is selected for uninstall.

Database Tables Queried

Shortcut Table

Properties Used

None

Sequencing Restrictions

The RemoveShortcuts action must be sequenced after the InstallFiles action and before the CreateShortcuts action.
