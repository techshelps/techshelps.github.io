installer
The new installation technology from Microsoft. See also Microsoft Windows installer.
installer database
Contains the setup information for an application. The installer database is in the installer package.
installer function
A function you can use to enable installer functionality. These functions can be called by any application.
installer package authoring tool
Hides the complexity of the installer database by automating the creation and population of database tables. The installer package authoring tool calls access functions. See also access function, custom installer action, and installer database.
