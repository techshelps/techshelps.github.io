Summary Information Stream

[This is preliminary documentation and subject to change.]

The material in this section is intended for developers who are writing their own setup programs and developers who want to learn more about the Microsoft® Windows® installer database tables. For general information, see About Windows Installer.

To conveniently store information in Microsoft® Windows® Explorer, you can use the summary information stream, which is accessible through the IStream interface. The following topics provide information about how to use the summary information stream with the installer:
