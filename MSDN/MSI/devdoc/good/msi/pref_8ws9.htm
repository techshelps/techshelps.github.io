Installed Property

[This is preliminary documentation and subject to change.]

The Installed property indicates that a product is already installed.

Set By

This property is set by the Installer at initialization if the product is already installed.

Used By

The Installed property can be used in conditional expressions to determine if a product is already installed. For example, the property can be used in the conditional column of a sequence table or to differentiate between a first install and maintenance mode user interface.

Default Value

None.
