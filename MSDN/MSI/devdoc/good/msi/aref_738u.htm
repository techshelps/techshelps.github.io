UnregisterExtensionInfo Action

[This is preliminary documentation and subject to change.]

The UnregisterExtensionInfo action manages the unregistration of extension-related information with the system. The action unregisters the extension servers from the Extension table for which the corresponding feature is currently selected to be uninstalled.

Database Tables Queried

Extension Table

Verb Table

Component Table

Properties Used

None

Sequencing Restrictions

The UnregisterExtensionInfo action must be sequenced after the InstallFiles action and before the RegisterExtensionInfo action.
