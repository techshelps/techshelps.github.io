MsiEnableUIPreview

[This is preliminary documentation and subject to change.]

The MsiEnableUIPreview function enables preview mode of the user interface to facilitate authoring of user-interface dialog boxes.

UINT MsiEnableUIPreview(
  MSIHANDLE hDatabase,  // database handle
  MSIHANDLE *phPreview  // returned handle for UI preview capability
);

Parameters

hDatabase
Handle to the database.
phPreview
Pointer to a returned handle for user-interface preview capability.

Return Values

ERROR_INVALID_HANDLE
An invalid or inactive handle was supplied.
ERROR_INVALID_PARAMETER
Bad parameter.
ERROR_OUTOFMEMORY
Out of memory.
ERROR_SUCCESS
The function succeeded.
