SHORTFILENAMES Property

[This is preliminary documentation and subject to change.]

Setting the SHORTFILENAMES property causes short file names to be used rather than long file names. This property may be set during an ADMIN install to create a source image having only short file names.

Set By

This property can be set by the user at the command line to disable long file names.

Used By

Actions use this property for directory resolution.

Default Value

Long names will be used by the installer if the SHORTFILENAMES property is not set and the target volume supports long names. Short names will be used by the installer if the target volume does not support long names.

Remarks

When the SHORTFILENAMES property is set, the installer will create folders and install files using short file names from any short|long pairs listed in the File table or Directory table.
