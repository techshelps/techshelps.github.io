LIMITUI Property

[This is preliminary documentation and subject to change.]

If this property is set, the UI level used when installing this package is capped as Basic. This property can be used when a package has no authored UI to use the Installer's built in default UI.

Set By

Property table or command line.

Used By

Installer uses this value to determine which UI to display.

Default Value

None.

Remarks

On Windows NT 5.0, the LIMITUI property will not prevent an uninstall. If the LIMITUI property is set, a user can uninstall an application by the following methods:

  • msiexec /x <path to MSI or ProductCode>
  • right-clicking on the MSI file, then clicking Uninstall
  • Calling the MsiInstallProduct API
