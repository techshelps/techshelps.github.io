PRODUCTLANGUAGE Property

[This is preliminary documentation and subject to change.]

The numeric language ID.

Set By

Set on the command line.

Used By

Advertised as a product property, this value is used to apply the correct language transform in the case of a multi-language product. .

Default Value

0.

Remarks

The PRODUCTLANGUAGE property is applied only during the installation or advertisement of a product. Since the installer must use the same language for later maintenance of the product, the PRODUCTLANGUAGE property cannot be set in maintenance mode.
