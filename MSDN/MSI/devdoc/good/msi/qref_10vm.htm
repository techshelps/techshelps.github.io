MsiRecordSetInteger

[This is preliminary documentation and subject to change.]

The MsiRecordSetInteger function sets a record field to an integer field.

UINT MsiRecordSetInteger(
  MSIHANDLE hRecord,   // record handle
  unsigned int iField, // field of record
  int iValue           // value to set
);

Parameters

hRecord
Handle to the record.
iField
Specifies the field of the record to set.
iValue
Specifies the value to which to set the field.

Return Values

ERROR_INVALID_FIELD
An invalid field of the record was supplied.
ERROR_INVALID_HANDLE
An invalid or inactive handle was supplied.
ERROR_INVALID_PARAMETER
An invalid parameter was passed to the function.
ERROR_SUCCESS
The function succeeded.

Remarks

In the MsiRecordSetInteger function, attempting to store a value in a non-existent field causes an error.
