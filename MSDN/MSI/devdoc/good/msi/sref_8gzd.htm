MSI Source Type Summary Property

[This is preliminary documentation and subject to change.]

The MSI Source Type Summary Property is used to indicate the type of source file image. If this property is not present, it defaults to 0. The location of the image is determined by the SOURCEFILE property, which defaults to the location of the installer package. Note that this property is stored in place of the standard Count property.

  1. Tree image matching the Directory table, long file names.
  2. Tree image matching the Directory table, short file names.
  3. Cabinet and files matching the Media table, long file names.
  4. Cabinet and files matching the Media table, short file names.
