Creating the MsiInstall Object

[This is preliminary documentation and subject to change.]

For OLE to access the automation, the DLL must be self-registered. The OLE ProgID used to create the MsiInstall object is Msi.ApiAutomation. If the DEBUG build automation is registered, the ProgID Msi.ApiAutoDebug is registered as well. If the SHIP build is registered last, both may be used.
