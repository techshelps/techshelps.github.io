Working with Installer Properties

[This is preliminary documentation and subject to change.]

The installer uses a number of properties to control installation. A list of the properties can be found in the Property Reference topic.

Functions that get installer properties:

  • To obtain the current properties, call the MsiGetProperty function.
  • To obtain the current installation state, call the MsiGetMode function.
  • To obtain the language for the current installation, call the MsiGetLanguage function.

Functions that set installer properties:

  • To set properties, call the MsiSetProperty function.
  • To set the installation state, call the MsiSetMode function.
