INSTALL Action

[This is preliminary documentation and subject to change.]

The INSTALL action is a top-level action called to install or remove components.

Database Tables Queried

The INSTALL action queries the following sequence tables for the action to execute, the condition for action execution, and the place of the action in the sequence.

InstallUISequence Table

InstallExecuteSequence Table

AdminUISequence Table

AdminExecuteSequence Table

AdvtUISequence Table

AdvtExecuteSequence Table
