Image Control Attribute

[This is preliminary documentation and subject to change.]

This attribute is the name of the image in the Binary Table. If the image is set by the ImageHandle attribute, this attribute does not reflect the correct name anymore.

Type

String.

Access

Read and Write.

Minimum Size of the Record

At least 1.

Control Types That Have This Attribute

Bitmap
Icon

Associated Control Stylebit

None.

Related Control Attribute

ImageHandle
