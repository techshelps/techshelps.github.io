RegisterMIMEInfo Action

[This is preliminary documentation and subject to change.]

The RegisterMIMEInfo action registers the MIME-related registry information with the system.

Database Tables Queried

MIME Table

Properties Used

None

Sequencing Restrictions

The RegisterMIMEInfo action must be sequenced after the InstallFiles and the UnregisterMIMEInfo actions.

Remarks

In advertise mode, the action registers all MIME info for servers from the MIME table for which the corresponding feature is enabled. Else the action registers MIME info for servers from the MIME table for which the corresponding feature is currently selected to be installed.
