LeftUnit Property

[This is preliminary documentation and subject to change.]

If this property is set, the unit is placed to the left of the number instead of the usual right hand side. This is needed to accommodate languages where this is the accepted format.

Set By

The value is set in the Property Table.

Used By

The UI uses this property to display the size of files and the available space on different volumes.

Default Value

None.
