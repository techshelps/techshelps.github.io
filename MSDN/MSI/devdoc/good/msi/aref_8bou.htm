PublishFeatures Action

[This is preliminary documentation and subject to change.]

The PublishFeatures action manages the advertisement of the feature composition (feature-component mapping) within the system.

Database Tables Queried

The PublishFeatures action queries the FeatureComponents table, the Feature table, and the Component table.

Properties Used

None

Sequencing Restrictions

None

Remarks

The PublishFeatures action publishes the composition of all features that are selected to be advertised or installed.
