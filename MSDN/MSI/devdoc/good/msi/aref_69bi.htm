ForceReboot Action

[This is preliminary documentation and subject to change.]

The ForceReboot action is authored in the sequence if there is a need to reboot during the installation before it can continue. The action writes a registry key that causes the installer to start after reboot. It also runs all spooled operations before reboot.

Database Tables Queried

None

Properties Used

None

Sequencing Restrictions

The ForceReboot action must be preceeded by the InstallValidate action and placed at the point in the installation sequence where the reboot is required.
