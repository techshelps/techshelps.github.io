Last Saved By Summary Property

[This is preliminary documentation and subject to change.]

In an installer database the Last Saved By Summary Property is used to keep track of who made the last changes to the database. Initially the value for this summary property is set to NULL. However, should a user choose to make modifications to an installation the value for this summary property changes. The value is set to the value of the USERNAME property defined by the installer.

Transform
In a transform this property is used for a different purpose. It contains the platform and language ID(s) that a database should have after it has been transformed. In other words, it specifies what the Template property should be set to in the new database. See the description of the Template property for information about the form of this property.
