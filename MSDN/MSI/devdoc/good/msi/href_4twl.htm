PropertyValue Control Attribute

[This is preliminary documentation and subject to change.]

This attribute is the present value of the associated property. If this attribute is changed, the control updates to show the new state.

Type

String or Integer depending on the Integer style of the control.

Access

Read and Write.

Minimum Size of the Record

At least 1.

Control Types That Have This Attribute

Every active control.

Associated Control Stylebit

None.
