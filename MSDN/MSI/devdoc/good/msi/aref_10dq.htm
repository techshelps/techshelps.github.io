PublishProduct Action

[This is preliminary documentation and subject to change.]

The PublishProduct action manages the advertisement of the product information with the system. This action publishes the product if it is in advertise mode or if MsiGetProductInfo returns ERROR_INSTALL_PRODUCT for the product.

Sequencing Restrictions
