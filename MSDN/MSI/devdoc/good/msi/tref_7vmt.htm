Icon Table

[This is preliminary documentation and subject to change.]

This table contains the icon files. Each icon from the table is copied to a file as a part of product advertisement to be used for advertised shortcuts and OLE servers.

The Icon table has the following columns.

Column Type Key Nullable
Name Identifier Y N
Data Binary N

Columns

Name
Name of the icon file.
Data
The binary icon data in PE (.dll or .exe) or icon (.ico) format.

Remarks

This table is referred to when the PublishProduct action is executed.
