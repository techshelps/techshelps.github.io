LicenseAgreement Dialog

[This is preliminary documentation and subject to change.]

This modal dialog box displays the license agreement to the user. Typically the text of the agreement is displayed in a ScrollableText control.

The dialog box typically also contains a pair of option buttons that let the user accept or reject the agreement. For legal reasons it is advisable that neither of the buttons is selected initially, forcing the user to make an active choice. This can be achieved by not having an initial value for the property linked to the RadioButtonGroup. The author can also disable/enable the next button on this dialog box, depending on the choice made. Another possible situation is when the next button takes the user to different dialog boxes depending on the choice he or she makes.
