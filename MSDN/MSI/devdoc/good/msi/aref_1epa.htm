RegisterExtensionInfo Action

[This is preliminary documentation and subject to change.]

The RegisterExtensionInfo action manages the registration of extension related information with the system.

Database Tables Queried

Extension Table

Verb Table

Properties Used

None

Sequencing Restrictions

The RegisterExtensionInfo action must be sequenced after the InstallFiles and the UnregisterExtensionInfo actions.

Remarks

In advertise mode, the action registers all extension servers from the Extension table for which the corresponding feature is enabled. Else the action registers extension servers from the Extension table for which the corresponding feature is currently selected to be installed.
