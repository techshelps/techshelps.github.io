ProductCode Property

[This is preliminary documentation and subject to change.]

A unique identifier for the particular product release, represented as a string GUID, e.g. "{12345678-1234-1234-1234-123456789012}". This ID must differ for different versions and languages.

Set By

Authored into Property table. REQUIRED.

Used By

Advertised as a product property, as the primary method of specifying the product.

Default Value

None.
