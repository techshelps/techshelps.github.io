Using the Installer Database

[This is preliminary documentation and subject to change.]

The database enables the developer of an installation package to control the transfer of an application from a source to a target image. The layout of both the source and target images of the application are specified in the Directory table and the actions that install the application are specified in six sequence tables:

  • The InstallUISequence table
  • The InstallExecuteSequence table
  • The AdminUISequence table
  • The AdminExecuteSequence table
  • The AdvtUISequence table
  • The AdvtExecuteSequence table
