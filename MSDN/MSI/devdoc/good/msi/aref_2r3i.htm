PublishComponents Action

[This is preliminary documentation and subject to change.]

The PublishComponents action manages the advertisement of the components from the PublishComponent.

Database Tables Queried

This action queries the PublishComponent Table, the Feature Table, and the Component Table.

Properties Used

None

Sequencing Restrictions

None

Remarks

This action publishes all of the components from the PublishComponents table that belong to features that have been selected for advertisement or installation.
