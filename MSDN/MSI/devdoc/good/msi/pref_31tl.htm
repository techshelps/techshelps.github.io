Resume Property

[This is preliminary documentation and subject to change.]

Set when resuming a suspended install. This property can be used to determine what text to show the user in the UI. For example, when not in resume mode the user could be asked "Are you ready to install?" while in resume mode the user could be asked "Are you ready to complete your install?".

Set By

Installer at initialization.

Used By

UI conditionalization uses this value to determine which install dialog to display.

Default Value

None.
