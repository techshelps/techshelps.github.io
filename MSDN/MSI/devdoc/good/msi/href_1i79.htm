IndirectPropertyName Control Attribute

[This is preliminary documentation and subject to change.]

In case of an active control this attribute is the name of the associated indirect property. For inert controls this attribute returns Null.

An indirect property names an other property, and the control displays/changes the value of this second property.

Type

String.

Access

Read only.

Minimum Size of the Record

At least 1.

Control Types That Have This Attribute

Every control.

Associated Control Stylebit

None.
