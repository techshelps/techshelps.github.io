UnregisterFonts Action

[This is preliminary documentation and subject to change.]

The UnregisterFonts action unregisters installed fonts with the system. The RegisterFonts action is triggered when the component to which the Font.File belongs is selected for uninstall. This implies that fonts can be made private, shared, or available to the entire system by specifying such status for the components to which they belong.
