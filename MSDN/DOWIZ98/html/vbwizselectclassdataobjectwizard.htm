Data Object Wizard—Select Class

Allows you to select the Class to use as the data source for your control.

Options

Available Classes List

Displays a tree-view list of all classes in the project that were previously created by the Data Object Wizard.

Help

Opens the Help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Data Object Wizard.

Back

Moves you to the previous step.

Next

Moves you to the next step.

Finish

Accepts any selections you entered and the defaults for the remaining steps, then creates a control. The Finish button is available once you make enough choices for the Data Object Wizard to create the appropriate data object.
