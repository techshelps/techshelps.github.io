Data Object Wizard—Select User Control Type

Allows you to choose which type of User Control you want to build.

Options

Single Record

Creates a User Control that displays one record at a time.

Single Record Repeating

Creates a repeated User Control using the DataRepeator control.

Data Grid

Creates an enhanced Data Grid control using the Microsoft DataGrid control.

Listbox

Creates a ListBox control.

ComboBox

Creates a ComboBox control.

Help

Opens the Help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Data Object Wizard.

Back

Moves you to the previous step.

Next

Moves you to the next step.

Finish

Accepts any selections you entered and the defaults for the remaining steps, then creates a data form. The Finish button is available once you make enough choices for the Data Object Wizard to create the appropriate data object.
