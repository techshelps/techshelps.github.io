Data Object Wizard—Finished!

Indicates that all of the necessary information needed to build your class or user control has been collected and lets you give a meaningful name to your saved class or user control.

Options

Class Name

If you are creating a class, allows you to type a name for it.

User Control Name box

If you are creating a User Control, allows you to type a name for it. The name must be unique to the current project.

Save current settings as default

Saves the current settings to the registry so they can be used as default each time you use the Data Object Wizard, until you change the settings.

Note   The settings are saved by default, if you do not want your settings saved, you must clear this check box.

Help

Opens the Help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Data Object Wizard.

Back

Moves you to the previous step.

Next

Moves you to the next step. Not available in this step.

Finish

Accepts any selections you entered and the defaults for the remaining steps, then creates a class or user control.
