Data Object Wizard—Introduction

Using the Data Object Wizard, you can create classes and user controls that are bound to the classes.

Note   For the Data Object Wizard to generate the correct logic, your responses must be consistent with the database you are using.

Options

Skip this screen in the future

When selected, the Data Object Wizard will not show the Introduction screen the next time you start it. If you want to see this screen in the future, click the Back button on the first step of the wizard.

Help

Opens the Help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Data Object Wizard.

Back

The Back button is unavailable on this step.

Next

Moves you to the next step.

Finish

Accepts the defaults and builds the appropriate object. The Finish button is available once you make enough choices for the Data Object Wizard to create the appropriate object.
