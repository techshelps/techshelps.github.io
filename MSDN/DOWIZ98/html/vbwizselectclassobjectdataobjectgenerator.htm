Data Object Wizard—Create Object

Allows you to choose whether to build a Data Source Class or a User control.

Options

A Class to which other objects can bind data

Note   You must have a Data Environment in your project to create a Data Source Class. The Data Object Wizard uses the Data Environment Commands you specify to retrieve and update data. If you don't have a Data Environment in your project, a message box appears. If you click OK, you return to this step. Clicking Cancel causes you to exit the Data Object Wizard.

A user control bound to an existing class

Note   You must have an existing class created by the Data Object Wizard to use this option.

Help

Opens the Help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Data Object Wizard.

Back

Moves you to the previous step.

Next

Moves you to the next step.

Finish

Accepts any selections you entered and the defaults for the remaining steps, then creates a control. The Finish button is unavailable on this step because you have not made enough choices for the Data Object Wizard to create the appropriate control.
