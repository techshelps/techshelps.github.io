Can't perform this operation. (Error 3032)

Possible causes:

  • You tried to delete the only user account in the group Admins. The group Admins must have at least one user account. If you want to delete this account, create a new account and add it to the group Admins, or add an existing account to the group Admins, and then delete the account.

  • You tried to put a user in a group to which the user already belongs, by appending either one of the following:
    • A CdbGroup object to a CdbUser object's Groups collection that already has a CdbGroup object of the same name.

    • A CdbUser object to a CdbGroup object's Users collection that already has a CdbUser object of the same name.
