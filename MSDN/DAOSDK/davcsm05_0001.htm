CdbRecordset::Close Method

Closes an open Recordset object.

Syntax

VOIDClose(VOID);

Usage

#include <afxole.h>
#include <dbdao.h>
...
CdbRecordset      rst;
COleVariant      vVal;
rst.OpenRecordset(...);   // Open some recordset
vVal = rst.GetField(0L);   // Get the first field
rst.Close();              // Close the recordset
