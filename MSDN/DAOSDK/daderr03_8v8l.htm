Can't open database <name>. It may not be a database that your application recognizes, or the file may be corrupt. (Error 3049)

Possible causes:

  • You tried to open a non-Microsoft Jet database, such as a Btrieve, dBASE, or Paradox database or table. Instead, link the database or table to an existing Microsoft Jet database.

  • You tried to import or link a Paradox database or table, and the associated .px file couldn't be found. Make sure the .px file is in the same directory as the Paradox database or table, and then try the operation again.

  • If the specified database is a Microsoft Jet database, it is corrupted. You should attempt to repair the database. If the database can't be repaired, restore the database from a backup copy, or create a new database.
