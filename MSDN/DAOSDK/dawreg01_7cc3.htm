Microsoft Exchange Data Source Initialization Settings

The Jet\3.5\Engines\Exchange folder includes initialization settings for the Msexch35.dll driver, used for external access to Microsoft Exchange folders. The only entry in this folder is the following:

win32=<path>\MSEXCH35.dll

The Microsoft Jet database engine uses this Exchange folder to indicate the location of Msexch35.dll. The full path is determined at the time of installation. Values are of type String for Windows 95 and Windows NT 4.0, and of type REG_SZ for Windows NT 3.51.
