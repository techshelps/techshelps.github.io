The language-specific code page was not specified or could not be found. (Error 3649)

You have attempted to open a database that was created with a language that is not installed on your computer. You should determine what language was specified for this database when it was created and then make sure that language is installed on your system. If the database was created with DAO, the language was specified with the collating order argument of the CreateDatabase method.

Languages can be added to your system through the Regional settings of the Control Panel.
