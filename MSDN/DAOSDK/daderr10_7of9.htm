Btrieve: Can't find file Field.ddf. (Error 3353)

The Btrieve driver cannot find the Field.ddf file pointed to by the File.ddf file.

Possible solutions:

  • If the Field.ddf file exists, fix the reference to it in the File.ddf file using either Novell Xtrieve or another .ddf file builder.

  • If the Field.ddf file doesn't exist, create it using either Novell Xtrieve or another .ddf file builder.
