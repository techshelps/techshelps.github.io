Dynamic-Type CdbRecordset Object

      

This CdbRecordset type represents a query result set from one or more base tables in which you can add, change, or delete records from a row-returning query. Further, records that other users add, delete, or edit in the base tables also appear in your CdbRecordset.

This type is only available in ODBCDirect workspaces and corresponds to an ODBC dynamic cursor.
