Btrieve: Invalid IndexDeleteRenumber option in initialization setting. (Error 3339)

The setting for the IndexDeleteRenumber entry in the Btrieve key of the Windows Registry is not valid. The IndexDeleteRenumber entry determines whether remaining indexes are renumbered if one is deleted.

Valid settings are:

  • On Renumbers.

  • Off Does not renumber.

On is consistent with Btrieve version 5.1x behavior. Off is consistent with Btrieve version 6.0 behavior.
