Couldn't open table 'MSysAccounts' in the workgroup information file. (Error 3173)

This operation requires the System.mdw file, which contains the security and option settings for Microsoft Jet database tables.

Possible causes:

  • The SystemDB value of the \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Office\8.0\Access\Jet\3.5\Engines\Jet key in the Windows Registry points to a network drive that is not connected. Make sure the network drive is available, and then try the operation again.

  • The System.mdw file is corrupted. If you have a backup copy of System.mdw, copy it to the location specified in the SystemDB entry in the Registry.
