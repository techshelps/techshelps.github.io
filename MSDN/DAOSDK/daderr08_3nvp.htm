Invalid property value. (Error 3271)

Possible causes:

  • The setting for the Value property of the CdbParameter object is too long or is otherwise incorrect.

  • The CdbIndex object name is a zero-length string (" "). You must enter a name for the Index object when appending the object to the Indexes collection.

  • The CdbWorkspace object name is a zero-length string. You must enter a name for the CdbWorkspace object when appending the object to a Workspaces collection.
