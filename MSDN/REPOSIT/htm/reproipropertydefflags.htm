IPropertyDef Flags Property

See Also

A flag that specifies whether or not a column is created in the SQL table for the interface to which this property is attached. If no column is created, instances of this property are only attached to individual objects, when the property value is set for that particular object. By default, a column is created for each property.

See the PropertyDefFlags Enumeration for the symbolic and numeric values of this flag.

Dispatch Identifier:   DISPID_PropFlags (63)

Property Data Type:   long
