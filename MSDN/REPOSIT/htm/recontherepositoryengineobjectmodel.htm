The Repository Engine Object Model

Microsoft Repository is an Automation application that stores data for tools. The Repository engine object model includes classes that expose their properties and behavior through interfaces. Repository contains objects and relationships and their respective property values.

Because Microsoft Repository can accommodate data for any tool, its object model reflects a simple, fundamental view of data. This section describes Repository's fundamental object model and introduces both the classes and the interfaces of Repository's object model.
