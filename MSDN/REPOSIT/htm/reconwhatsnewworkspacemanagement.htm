What's New: Workspace Management

Workspaces, together with a powerful checkin-checkout mechanism, support the implementation of sophisticated team-development functionality. This functionality includes:

  • Document management.

  • High-end source code control.

  • Configuration management of complex interrelated structures such as Web sites, program representations, and metadata for data warehouses.
