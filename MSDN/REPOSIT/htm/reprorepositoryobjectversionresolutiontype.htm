RepositoryObjectVersion ResolutionType Property

See Also

This property indicates the resolution technique by which the Repository chose to give you a reference to the current version (rather than a reference to any other version of the same object). This is a read-only property.

Syntax

object.ResolutionType

The ResolutionType property syntax has the following parts.

Part Description
object An object expression that evaluates to a RepositoryObjectVersion object.

Remarks

The Repository automatically sets the value of the ResolutionType property whenever you retrieve an object version. There are four possible values:

  • VersionInWorkspace

  • PinnedVersion

  • LatestVersion

  • SpecifiedVersion
