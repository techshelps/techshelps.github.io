Documentation Map

The principal sections of the Microsoft Repository documentation are the Overview, Browser, Migration Wizard, Programmer's Guide and Reference. Many of the topics in the Programmer's Guide refer to the COM Reference and Automation Reference for further information about objects, interfaces, and interface members (properties, methods, and collections).
