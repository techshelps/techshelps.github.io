Depicting Relationships Between Objects

To depict relationships between objects, use arrows as shown in the following figure.

Diagrams of metadata use these standard conventions:

  • Show objects as dots

  • Show relationships as arrows

  • Show kinds of objects as labeled rectangles

The labels in the Kind of Data column constitute a portion of an object model for storing object models. The object model for storing object models is called the Type Information Model.
