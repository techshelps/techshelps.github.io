What's New: Version Management

Repository's new object and relationship versioning functionality enables complex configuration management operations at the information model level. The main features are:

  • Versioning of objects with efficient evolutionary storage of properties.

  • The ability to create and merge versions, which allows the user to track the evolution of an object through a graph of parallel or sequential historical versions.

  • Versioning of relationships to maintain the history of associations between object versions.
