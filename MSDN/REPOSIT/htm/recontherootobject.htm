The Root Object

The root object is the top-level object in the Repository. It is the object from which all Repository navigation begins. Like any Repository object, the root object can have any number of relationships with other objects. The following figure shows seven such relationships.

Each relationship connecting the root object to other objects must conform to a relationship type. The relationship type to which these relationships conform is created by the tool information model creator.
