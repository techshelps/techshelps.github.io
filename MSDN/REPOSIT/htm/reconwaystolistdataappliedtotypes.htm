Ways to List Data, Applied to Types

Repository stores metadata as data. This section uses data visualization techniques to demonstrate how to model metadata. (See Ways to List Data for a table listing and organizing the data about books and publishers.)

The table that follows lists and organizes the types of the bookseller's object model. The left column contains convenient groupings of like information, and the right column (Data) contains the information itself. The Data column describes particular classes (such as Book and Person) and particular relationship types (such as Authorship and Publication). The Kind of Data column thus reveals a portion of the object model for storing classes and relationship types.
