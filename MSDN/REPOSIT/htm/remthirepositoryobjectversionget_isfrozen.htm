IRepositoryObjectVersion::get_IsFrozen

Determines whether the current object version is frozen.

HRESULT get_IsFrozen(
VARIANT_BOOL *pbFrozen
);

Parameters

*pbFrozen

[out]
TRUE if the current object version is frozen; FALSE if it is not frozen.

Return Value

S_OK

The method completed successfully.

Error

Values

This method failed to complete successfully.
