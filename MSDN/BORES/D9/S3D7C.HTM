Checking MIF Files

You can check the syntax of a MIF file using the MIFCHECK utility. The MIFCHECK utility runs the same MIF parser used by the SMS components. Therefore, any MIF file accepted by MIFCHECK will also be accepted by the SMS system. MIFCHECK verifies that the specified MIF files adhere to the syntax of the DMI Specification version 4.5 with the exceptions noted in the preceding section.

The MIFCHECK utility is provided on the Microsoft BackOffice Resource Kit compact disc. See the SMSTOOLS.HLP file on the compact disc for more information.
