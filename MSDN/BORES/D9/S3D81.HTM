SNMP Export Files

The SMS Event to Trap Translator allows you to export trap configurations that you have defined in the SMS Administrator. You can export definitions in two formats: the configuration tool (CNF) format, and the trap text (TXT) format. The CNF format is used to export configurations that can be used with the Event Configuration tool to configure the Event to Trap Translator on computers running Windows NT. The TXT format is used to export configurations that can then be imported into other SNMP systems.
