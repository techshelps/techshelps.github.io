RPC Counter Tool

The RPC counter is a performance tool that checks the number of RPCs from an application to a Microsoft Exchange Server. The RPC counter is used with clients running Windows 3.1, Windows for Workgroups, Windows NT Workstation, and Windows 95.

It's always important to design your application for maximum performance. Minimizing the number of RPCs for your application is the most important factor in improving performance. The RPC counter can alert you to less efficient code processes so that you can optimize these areas. Using the RPC counter to minimize the number of RPCs can improve your application's performance and decrease the load on the Microsoft Exchange Server computer.

The RPC counter is useful for determining the number of RPCs that are generated by:

  • MAPI calls
  • User actions
  • An application

All these examples for using the RPC counter are useful in determining the effect of system performance on the Microsoft Exchange Server from various sources.
