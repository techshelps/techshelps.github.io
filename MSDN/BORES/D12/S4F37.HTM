Server IMC Traffic Chart View

The following counters are available in the Microsoft Exchange Server IMC Traffic chart view.

Note

The Internet Mail Connector must be running to use these counters.

Counter

Object

Description

Messages Entering MTS-IN

Internet Mail Connector

The number of messages received from the Internet that are waiting for final delivery to Microsoft Exchange Server.

Messages Entering MTS-OUT

Internet Mail Connector

The number of messages being sent to the Internet that are waiting to be converted to Internet mail format.

Messages Leaving MTS-OUT

Internet Mail Connector

The number of messages that are waiting for final delivery to the Internet.

Connections Inbound

Internet Mail Connector

The number of current simple mail transport protocol (SMTP) connections to the Internet Mail Connector established by other SMTP hosts.

Connections Outbound

Internet Mail Connector

The number of current SMTP connections that the Internet Mail Connector has established to other SMTP hosts.
