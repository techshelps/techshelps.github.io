Server Maintenance Installation

Once you've installed Microsoft Exchange Server, you can run Setup again to add or remove components as needed. For example, you may decide to add a connector on your server. When you run Setup, it detects whether Microsoft Exchange Server is already installed on the computer. If it is, Setup starts maintenance installation. Maintenance installation includes options for Add/Remove, Remove All, and Reinstall.

Adding or Removing Components

When you add components by selecting the Add/Remove option, Setup runs the installation (.ins) file for each selected component.

When you remove components by selecting the Add/Remove option or the Remove All option, Setup performs the following:

  • Deletes one or more specified components from the system by running the uninstallation (.uns) file for each component. If the Remove All option is selected, all Microsoft Exchange Server components are deleted.
  • Removes the service for each component and deletes references to the component in the registry and directory.

Reinstalling Microsoft Exchange Server

You can use the Reinstall option to upgrade Microsoft Exchange Server to a later version or to restore registry settings if the registry becomes corrupted. To update Microsoft Exchange Server, Setup performs the following.

  • Checks the version of the existing installation.
  • Overwrites the appropriate files.
  • Modifies the registry entries.
  • Sets values in the directory.

Setup does not overwrite templates because they can be customized by users. The database files for the information store or directory are not modified.
