Obtaining Information About a Folder

After you create a public folder, you can view information about its characteristics.

To obtain information about the type, location, and general description of a public folder

    1. Select the folder.

    2. From the File menu, choose Properties.

    3. Select the General tab.

To obtain information about the folder contact, your permission level, and the folder path

    1. Select the folder.

    2. From the File menu, choose Properties.

    3. Select the Summary tab.
