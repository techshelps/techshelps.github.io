Chapter 2 Setup

When you set up Microsoft Exchange Server, it's helpful to understand how Setup works so you can troubleshoot problems or customize your installation. If you understand how Setup works, it's easier to trace the cause of problems related to Setup. Understanding Setup also makes it possible for you to customize your installation quickly and easily so your client and server configurations suit your organization's needs. For example, you can run Setup unattended using customized settings or create default client messaging profiles.
