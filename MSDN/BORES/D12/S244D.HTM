Running Setup Unattended

There are two options that you can use to install the Microsoft Exchange Client quickly on several computers. You can distribute the Microsoft Exchange Client by running Setup from the command line or by using SMS.

Running Setup from the Command Line

There are several command-line parameters that you can use to perform an unattended batch mode installation. Running Setup from the command line is useful if you need to install the Microsoft Exchange Client quickly and don't want to be prompted to enter additional information. The following parameters can be used for batch Setup. These parameters are case-sensitive.

Command-line parameter

Description

Setup /Q

Installs the Microsoft Exchange Client using the Typical option. This cannot be used with MS-DOSâ or when installing a shared installation of the client.

/N "name"

The user's name (not case-sensitive).

/O "organization"

The organization name (not case-sensitive).


For example, typing the following at the command line installs the Microsoft Exchange Client for a user named Maria Black without prompting for additional information.

Setup /Q /N "Maria Black"

You can use the following command-line parameters to perform unattended maintenance installations. Like batch mode Setup, these parameters are useful if you need to reinstall, remove, or restore the Microsoft Exchange Client quickly and don't want to be prompted to enter additional information. These parameters are case-sensitive and can't be used together.

Command-line parameter

Description

Setup /R

Reinstalls the Microsoft Exchange Client.

Setup /U

Removes all Microsoft Exchange Client components.

Setup /Y

Restores the Microsoft Exchange Client default settings.


Using SMS to Distribute the Microsoft Exchange Client

If you use SMS to install applications on clients in your network, you can also use SMS to distribute the Microsoft Exchange Client software. SMS makes it easier to install Microsoft Exchange Client on many computers without user input. SMS packages (.pdf files) for installing the Microsoft Exchange Client are provided with the Microsoft Exchange Client software. SMS packages are provided for computers running the following operating systems:

  • Microsoft Windowsâ 95
  • Windows NT Server
  • Windows NT Workstation
  • Windows version 3.1
  • Windows for Workgroups

For information about installing .pdf files, see your SMS documentation.
