Installing Preview Pane Extension

To make Preview Pane Extension available to users, you need to copy it from the Microsoft BackOffice Resource Kit: Part One compact disc to a network share, then install it on a client computer running Windows NT Workstation or Windows 95.

To install Preview Pane Extension to a network share

    1. Locate the Tools\Cliext directory on the Microsoft BackOffice Resource Kit: Part One compact disc.

    2. Copy its contents to the server share where you want users to find it.

To install Preview Pane Extension on a Client Computer

    1. Connect to the network location where Preview Pane Extension is available for setup.

    2. Double-click Setup.exe.
