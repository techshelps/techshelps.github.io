Server IMC Queues Chart View

The following counters are available in the Microsoft Exchange Server IMC Queues chart view.

Note

The Internet Mail Connector must be running to use these counters.

Counter

Object

Description

Queued Inbound

Internet Mail Connector

The number of messages received from the Internet that are waiting to be converted for Microsoft Exchange Server.

Queued MTS-IN

Internet Mail Connector

The number of messages received from the Internet that are waiting for final delivery to Microsoft Exchange Server.

Queued MTS-OUT

Internet Mail Connector

The number of messages that are waiting to be converted to Internet mail format.

Queued Outbound

Internet Mail Connector

The number of messages that are waiting to be delivered to the Internet.
