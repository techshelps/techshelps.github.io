Using the Bulk Advanced Security Tool

Use the following procedures to decrypt and encrypt messages:

To decrypt messages in a folder

    1. In the Tools\Security directory on the Microsoft BackOffice Resource Kit: Part One compact disc, double-click Sectool.exe.

    2. From the Exchange Security menu, choose Unsecure Folder.

    3. Type your security password, and then choose OK.

    4. Select the message store containing the messages you want to decrypt, and then choose OK.

    5. Select the folder containing the messages you want to decrypt, and then choose Start Here. To select a subfolder, choose Open Folder.

    6. Select Process Subfolders to decrypt messages in the selected folder and its subfolders, and then choose OK.

To encrypt messages in a folder

    1. In the Tools\Security directory on the Microsoft BackOffice Resource Kit: Part One compact disc, double-click Sectool.exe.

    2. From the Exchange Security menu, choose Secure Folder.

    3. Type your security password, and then choose OK.

    4. Select the message store containing the messages you want to encrypt, and then choose OK.

    5. Select the folder containing the messages you want to encrypt, and then choose Start Here. To select a subfolder, choose Open Folder. If you are moving messages between sites or recipient containers, select the folder containing the messages you decrypted before the messages were moved.

    6. Select the options that you want to use for encrypting messages, and then choose OK.

Option

Description

Re-secure previous secure messages

Encrypts only messages that were previously decrypted by this tool.

Encrypt all messages in folder

Encrypts all messages in the folder that were not previously encrypted.

Encrypt Signed Messages

Encrypts messages that are secured with digital signatures. If selected, digitally signed messages will be encrypted but no longer signed.

Process Sub Folders

Encrypts all messages that were not previously encrypted in the folder and its subfolders.
