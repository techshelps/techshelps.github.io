Using Microsoft Exchange Server Address Export to MS Mail

Use the following instructions to export Microsoft Exchange Server Addresses to MS Mail.

    1. In the Administrator program, export the addresses from the required container to a .csv file.

    2. Run Eximwiz.exe.

    3. Select the .csv file as the input file, and then select an output file to write the MS Mail import instructions to.

    4. Select the user addresses that you want to process. If you want a majority of these user addresses processed, select the Add All option and then remove the addresses you don't want.

    5. Select one of the following options:

  • Stop after creating the output file.
  • Send the file through e-mail to an MS Mail administrator.
  • Import this information directly into a MS Mail postoffice on the network.

    6. Supply the additional information requested (for example, postoffice name, administrator ID and password for MS Mail directory import; and MAPI logon and recipient address information).
