Installing RPC Counter

Before you can use the RPC counter, you need to connect with the Emsmdb(32).dll file, which contains the necessary entry points for generating the RPC data. For details on RPC counter entry points, see "Understanding RPC Counter Entry Points" later in this appendix. The Emsmdb(32).dll file is part of the Microsoft Exchange Client and Microsoft Exchange Server installation and is located in the Windows\System(32) directory.

You can access RPC counter data through several sample applications, depending on the your computer's operating system.

To install the RPC counter sample applications

· Copy the appropriate file from the Tools\Rpc directory on the Microsoft BackOffice Resource Kit: Part One compact disc to your current working directory or somewhere in the system path.

For Windows NT Workstation and Windows 95, copy Rpccnt.exe.

For Windows 3.1 and Windows for Workgroups, copy Rpccnt16.exe.

When you run these files displays, the RPC counter appears in a dialog box.
