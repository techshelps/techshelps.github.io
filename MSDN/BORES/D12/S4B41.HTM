Optional Components

You can install optional components to benefit from added features, such as connectivity to other messaging systems and advanced security. For example, if your users need to send messages to Internet users, you can install the Internet Mail Connector. If you have Microsoft Schedule+ users on both Microsoft Mail for PC Networks and Microsoft Exchange Server, you can install the Schedule+ Free/Busy Connector so users can view free and busy times.

The following optional components can be installed on a Microsoft Exchange Server computer.

Internet Mail Connector

Enables users on Microsoft Exchange Server to send messages to and receive messages from users on the Internet. The Internet Mail Connector can also be used to connect sites over a simple mail transport protocol (SMTP) backbone.

Microsoft Mail Connector

Enables users on Microsoft Exchange Server to send messages to and receive messages from users on Microsoft Mail for PC Networks and Microsoft Mail for AppleTalk Networks. The Microsoft Mail Connector includes all components required to connect to Microsoft Mail postoffices, including the directory synchronization component and the Schedule+ Free/Busy Connector.

X.400 Connector

Can be configured to connect sites with Microsoft Exchange Server, or to route messages to foreign X.400 systems. The X.400 Connector conforms to the 1984 and 1988 CCITT X.400 standards. For more information, see Chapter 5, "X.400 Connector."

  • Key Management Manages security information used for digitally signing and encrypting messages sent between users within an organization. For more information, see Chapter 7, "Security."
