Installing Microsoft Exchange Server Address Export to MS Mail

The Microsoft Exchange Server Address Export to MS Mail tool is included on the Microsoft BackOffice Resource Kit: Part One compact disc.

To install the Microsoft Exchange Server Address Export to MS Mail tool

    1. Copy Eximwiz.exe from the Tools\Msmailex\Eximwiz directory to a directory of your choice.

    2. Copy Msmapi32.ocx from the Tools\Msmailex\Eximwiz directory to your System32 or System directory.
