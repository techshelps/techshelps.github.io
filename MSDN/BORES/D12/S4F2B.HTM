[Licensing]

This section contains entries that specify licensing information.

Entry

Description

Value

PerSeat=

Whether to set up licensing on a per-seat basis.

True or false

PerServer=

Whether to set up licensing on a per-server basis.

True or false

PerServerUsers=

Number of users licensed on this server. Required if PerServer= is used.

Number of users


Note

The following is required in the [Licensing] section if the Services setting in the [Components] section is set to true:

  • Either PerSeat= or PerServer= must be true.
  • If PerServer= is true, PerServerUsers= must be specified.
