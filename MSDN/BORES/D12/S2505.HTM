Setting Up Your Topology

Before you can run a load simulation test, you must set up the mailboxes to be used for the test.

To configure your topology

    1. In the Loadsim directory on your Microsoft Exchange Client computer's local disk, double-click Loadsim.exe.

    2. From the Configuration menu, choose Test Topology.

    3. Choose Add Server.

    4. In the Server Properties dialog box, type the organization name, the site name, and the server name for each server you're testing.

    5. Type in the number of user accounts you will use in the test.

To test public folder usage on the server, specify the number of public folders to use in the test.

    6. Choose OK twice to close the dialog boxes.

    7. From the File menu, choose Save As and save the .sim file to a directory on your local disk.

Option

Description

Organization

The organization name (up to 63 characters).

Site

The site name (up to 63 characters).

Server

The server name (up to 15 characters).

Number of accounts

The number of users that will be tested on this server.

Different public server

Another public folder server if the server containing the test mailboxes does not contain both a public and private information store.

Total public folders

The number of public folders for this test.

Root folders

The number of top-level public folders that will be used for this test.

Replicas in this site of each folder

The minimum and maximum number of public folder replicas that will be created in this site weighted to the average you select.

Replicas in other sites of each folder

The minimum and maximum number of public folder replicas that will be created in other sites weighted to the average you select.
