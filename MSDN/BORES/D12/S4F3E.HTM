Backing Up and Restoring Data

You must have the following permissions to back up and restore data. These permissions are set in User Manager.

  • To back up data, you must be a member of the Windows NT Server Backup Operators group on each server that is backed up. Only administrators can add users to this group.
  • To restore data from tape backup, you must be a member of the Windows NT Server Administrators group on each server being restored. Only administrators can add users to this group.
