Setting Up a Mailbox to Send on Behalf of Another Mailbox

Send on Behalf of permissions enable a user to send messages with the user's name after "Sent On Behalf Of" and the delegate's name after "From." These permissions can be set up using either Microsoft Exchange Client or the Administrator program. You can use the Administrator program to grant Send on Behalf of permissions if the mailbox owner is not available.

To grant Send on Behalf of permissions by using Microsoft Exchange Client

    1. From the Tools menu in the Microsoft Exchange Client, choose Options, and then select the Exchange Server tab.

    2. Choose Add.

    3. Under Type Name or Select from List, type or select the user's name.

    4. Choose Add, and then choose OK.

To grant Send on Behalf of permissions by using the Administrator program

    1. In the Administrator window, choose Recipients, and then select the name of the recipient you want to configure.

    2. From the File menu, choose Properties, and then select the Delivery Options tab.

    3. Under Give Send On Behalf Of permissions to, choose Modify.

    4. In the Address Book, select the recipient and then choose Add.
