Increasing Microsoft Exchange Server Security

In addition to using advanced security to protect messages, you can also increase the security of your Microsoft Exchange Server system by:

  • Restricting access to resources by setting permissions on directory objects and public folders. For information about setting permissions, see Appendix D, "Setting Permissions."
  • Changing permissions on shared directories created during server Setup to minimize the possibility that Microsoft Exchange Server files can be tampered with.
  • Configuring Microsoft Exchange Client to use encrypted RPCs so data sent between clients and servers can't be altered while traveling across the network.
