Using Server and Link Monitors

The Administrator program includes server and link monitors.

  • Use server monitors to monitor the Microsoft Exchange Server services, such as the message transfer agent (MTA) or any other Windows NT services that are running on a Microsoft Exchange Server. You should configure server monitors to send a notification if the services go down and automatically restart the service.
  • Use link monitors to verify the connections between servers and foreign systems.

For more information, see the Microsoft Exchange Server Administrator's Guide.
