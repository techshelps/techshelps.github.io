Microsoft Exchange Server Preview Pane Extension

Microsoft Windows 95 and Windows NT Workstation users can use the Microsoft Exchange Server Preview Pane Extension to add a third area for previewing the body of messages to the Microsoft Exchange Client Viewer. The contents of the To and Subject boxes can also be displayed in the preview pane message header, but no attachments or other boxes are available for preview. Users can set reminders on messages or folders. In addition, a user can determine the size of a folder and its subfolders.

Users can also use Preview Pane Extension to mark messages as read, set reminders on messages or folders, automatically synchronize offline folders on a specified schedule, view messages in a fixed font, and move or copy messages or folders in the Microsoft Exchange Client hierarchy.

A user customizes Preview Pane Extension using the Preview tab in the Options dialog box available from the Microsoft Exchange Client Tools menu. The extension dynamic-link library (DLL) for these options resides on the user's workstation.

Preview Pane Extension may appeal to more experienced users in your organization because it combines some of the functionality of Microsoft Schedule+ type reminder features with Microsoft Exchange Client folders and messages. However, new users will also find it easy to customize and use Preview Pane Extension.

Preview Pane Extension uses approximately 1 megabyte (MB) of space on the share where you make it available to users. Preview Pane Extension is a Microsoft Exchange Client extension that uses MAPI and supports users with Microsoft 32-bit operating systems. Preview Pane Extension supports advanced security through the Microsoft Exchange Advanced Securities Extensibility application programming interface (API). It also supports uniform resource locators (URL). When a user reads an Internet address text in the preview pane, the text can be used as an operational Internet link if the address text is valid.

When an extension is added to Microsoft Exchange, another module is loaded, which can impact overall performance. The preview pane generates an RPC for each item stored on a server that is previewed by a user. Also, each window resize can cause additional RPCs. A user may notice this when using Remote Access Service (RAS).

A Preview Pane Uninstall icon is included in the Microsoft Exchange program group when a user sets up Preview Pane Extension. Running the uninstall program removes all Preview Pane Extension files. However, if it fails to remove these files, you can delete the registry entries.

Preview Pane Extension stores its configuration data in HKEY_Local_Machine\Software\Microsoft\Exchange\Client\Extensions by adding the following value: 


4.0;c:\Windows\tripane\tripane.DLL;1;11010010000000;1101010.

You can disable Preview Pane Extension by removing the value.

It's a good idea to distribute Preview Pane Extension to a small group of users to test before making it available for wide release. A minimum of 30 days is recommended as a test period. Based on this experience, you can modify the documentation you send to users.
