Microsoft Exchange Server Address Export to
MS Mail Tool

This tool (Eximwiz.exe) converts Microsoft Exchange Server Bulk Export format (.csv) to a MS Mail import format. After creating the output file, you can use the tool to interactively import the addresses into a MS Mail postoffice available on the network or send the import file through e-mail to external recipients.

This tool is useful in situations where directory synchronization between a Microsoft Exchange Server implementation and MS Mail is not a viable option. For example, if you are trying to populate a trading partner's MS Mail system with relevant addresses (such as purchasing agents) from your global address list.
