Avoiding Recipient Ambiguity Problems

Recipient ambiguity problems can occur when using the Command Line Mail Sender. Recipient names entered in the To and Cc boxes must not be ambiguous; otherwise, the Command Line Mail Sender will fail.

To test for ambiguity of names

    1. Type the recipient names in the To box of a Microsoft Exchange Client standard send form.

    2. Choose To, or press CTRL+K.

If the names are immediately resolved without any prompting, they are not ambiguous.

    3. Repeat this test each time the Address Book is changed, and run the test from the computer where Command Line Mail Sender runs.
