Performance

Windows NT Performance Monitor can be used to help you troubleshoot performance problems that affect Microsoft Exchange Server. It provides information about performance statistics such as processor use, memory use, server throughput, queue size, and number of write/read operations per second. You can use the preconfigured Performance Monitor chart views that are included with Microsoft Exchange Server to help you improve your system's performance and identify bottlenecks.

The following are the most typical indications of performance bottlenecks.

  • Microsoft Exchange Client or Microsoft Exchange Server is slow.
  • Mail delivery is slow.

For information about the Performance Monitor chart views for Microsoft Exchange Server, see Appendix C, "Performance Monitor Chart Views."
