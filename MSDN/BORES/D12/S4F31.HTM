Periodically

  • Run information store maintenance. Check the IS Maintenance property page on the Server object in the directory to ensure that the maintenance routines run during off-peak hours and at least once per day.
  • Defragment the server offline. Take the server offline and use the defragmentation utility (Edbutil.exe) to defragment the directory and information store.
  • Check mailbox and public folder use as follows:
    • Check the Logons property pages in the private and public information stores.
    • Check the Mailbox Resources and Public Folder Resources property pages to see how individual users and public folders are using storage space.
  • Check storage limits. Monitor storage limit settings on mailboxes and public folders. You may need to adjust these limits periodically for certain users or public folders.
