Using the Proxy Conversion Tool

    1. Run Import.exe against an existing MS Mail postoffice.

    2. Type the following:


PERL <input_file > <output_file>

where input_file is the file created in step 1, and output_file is the .csv file for the conversions.

    3. Using the Administrator program, import the .csv file to add the addresses as custom recipients to a Microsoft Exchange Server Recipient container.
