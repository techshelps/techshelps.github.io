Setting Permissions on a Schedule

By default, all users can access the free and busy times for other users. However, a user must have permission to open another user's appointment book or view details of another user's busy times. Users can grant permissions to other users to access their appointments, contacts, events, or tasks. Schedule+ items marked as Private can't be accessed unless a user is an owner or delegate owner on the item.

To set up access to a user's schedule

    1. From the Tools menu in Schedule+, choose Set Access Permissions.

    2. Choose Add, double-click the delegate, and then choose OK.

    3. In the User role box, select a role, and then choose OK.

To view another user's schedule

    1. From the File menu in Schedule+, choose Open, and then choose Other's Appointment Book.

    2. Select a user name and then choose OK.
