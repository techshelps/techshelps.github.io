Setting the Alerter Service

You can configure a Microsoft Exchange Server to notify a computer or user with system alerts (for example, when disk space is low).

To configure system alerts

    1. In Control Panel, choose Server.

    2. Choose the Alerts button.

    3. Type the computer or user names to send alerts to in the New Computer or Username box.

    4. Choose Add to add computers or user names, or choose Remove to remove them.
