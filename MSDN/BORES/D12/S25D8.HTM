Installing the Proxy Conversion Tool

The MS Mail SMTP Gateway Proxy Migration tool (Perl.exe) is included on the Microsoft BackOffice Resource Kit: Part One compact disc.

To install the Proxy Conversion tool

· Copy the following files from the Tools\Msmailex\Proxy directory of the Microsoft BackOffice Resource Kit: Part One compact disc to a directory of your choice.

Perl.exe

Perl100.dll

Convert.pl

Go.bat (Sample file optional)

In.txt (Sample file optional)

Out.txt (Sample file optional)

Fail.txt (Sample file optional)

For more information on the PERL program, see the Microsoft Windows NT Resource Kit version 3.51.
