Creating Subfolders

You can create public folders that appear as subfolders. However, to do so, you must have been granted permission by the top-level folder owner, also known as the publishing author.

To create a subfolder

    1. In the Microsoft Exchange Client Viewer, select the parent folder.

    2. From the File menu, choose New Folder.

    3. If you are writing a MAPI function call, use CreateFolder() on the parent folder object.
