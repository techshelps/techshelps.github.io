Public Folders

When you establish a new site, public folder replication cannot occur until the directory has been replicated. If the directory replication interval is long, the information store may try to replicate public folders before the new site has been added to the directory. To ensure that public folders replicate successfully, you can wait for the directory to replicate the new site information or force directory replication to occur. You can do this by choosing Update Now in the General property page for the server directory.

Note

If you force directory replication to occur, choose a time when the server's performance is not critical and connection costs are low.
