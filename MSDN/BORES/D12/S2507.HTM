Setting Up Distribution Lists

To use distribution lists in your test, you must configure and import them before setting up your tests. This step is optional.

To configure distribution lists

    1. From the Configuration menu, choose Distribution Lists.

    2. Select the Use distribution lists check box.

    3. Under Distribution list specification, configure your distribution lists.

    4. Choose OK.

Option

Description

Use distribution lists

Indicates whether distribution lists will be used for the test.

Size of lists

The minimum and maximum number of users that will be included in a distribution list. You can also specify an average number of users in a distribution list.

Cover % of users (no overlap)

The percentage of the total number of simulated users that are members of distribution lists.

Create lists (allow overlap)

The number of distribution lists that are created for this test.

% local to same server

The percentage of users in distribution lists whose mailboxes are located on the server that is being tested. If you don't select this option, users added to distribution lists will be chosen randomly from the total number of simulated users.

If your site has multiple servers, this option allows you to control the grouping of users in distribution lists. For example, if you set the value to 50 percent, half the users in a distribution list will reside on one server, and the other half will be randomly selected from the remaining servers in your load simulation topology.
