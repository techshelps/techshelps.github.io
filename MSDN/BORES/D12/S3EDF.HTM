Web Sites

Visit the following Web sites for up-to-date information about the Microsoft Exchange Server product.

http://www.microsoft.com/exchange

The Microsoft Exchange Server Web site contains up-to-date information about Microsoft Exchange Server and links to other Microsoft BackOffice products.

http://www.microsoft.com/support/products/backoffice/msmail.htm

The Microsoft Exchange Server Technical Support Web site gives you access to the Microsoft Exchange Server knowledge base, support information, and frequently-asked questions.

http://www.microsoft.com/technet

The Microsoft TechNet Web site contains information about subscribing to Microsoft TechNet, which provides in-depth technical information about Microsoft business products, including Microsoft Exchange Server and other Microsoft BackOffice products.

http://www.microsoft.com/train_cert

The Microsoft Training and Certification Web site provides information about training options and the Microsoft Certified Professional Program.
