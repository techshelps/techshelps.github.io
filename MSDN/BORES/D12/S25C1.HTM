IMC Configuration Restorer

The IMC Configuration Restorer is a command-line tool that exports specific Internet Mail Connector configuration information into a file so that it can be imported to another Internet Mail Connector.

In large organizations, the Internet Mail Connector for each domain routing and encoding configuration is extensive. For an administrator to duplicate the configuration manually on another connector can take hours and can also be prone to error. The IMC Configuration Restorer can save valuable time by automating the process of copying Internet Mail Connector configuration information to another Internet Mail Connector.

The IMC Configuration Restorer saves the following information:

  • Per-domain routes
  • Per-domain content encoding
  • Security exception list (accept/reject by host)
  • Multipurpose Internet Mail Extensions (MIME) types
