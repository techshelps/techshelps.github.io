Using Preview Pane Extension

You can use Preview Pane Extension with Microsoft Exchange Server if you are running either the Windows 95 or Windows NT Workstation operating system. You can use it to preview the body text of messages (together with the To and Subject information), to set reminders on messages or folders, or to determine the size of a folder.

You can customize Preview Pane Extension as follows:

  • Set reminders on messages or folders.
  • Hide or display the preview pane message header.
  • Mark messages as read, either at the end of a time period, such as 5 seconds, or when you click off the message.
  • Use a filter to display only unread messages in the preview pane.
  • Synchronize offline folders automatically on a schedule.
  • Customize the client toolbar with Preview Pane Extension buttons.
  • Display messages with a fixed font.

To customize the Preview Pane Extension

    1. From the Microsoft Exchange Client Tools menu, choose Options, and select the Preview tab.

    2. Select the options you want, and then choose OK.

    3. From the Microsoft Exchange Client Tools menu, choose Customize Toolbar, and then add the Preview Pane Extension buttons you want to use.

To display the preview pane

· From the View menu, select Preview.

To hide the preview pane

· From the View menu, clear Preview.

Note

If you installed the Show/Hide Preview Pane button on your Microsoft Exchange Client toolbar, you can select it instead of choosing the Preview command.

To set a reminder on a message or folder

    1. In the Microsoft Exchange Client, select a folder or message.

    2. From the File menu, choose Reminder.

    3. In the Preview Pane Reminder dialog box, select the options you want, and then choose Save.

Note

If you installed the Reminder button on your Microsoft Exchange Client toolbar, you can click it instead of choosing the Preview Pane Reminder command.

When you move a message or folder that has a reminder, you lose the reminder. Reminders you set are private because reminder information is stored only on your local computer.

To copy or move a folder or message

    1. From the Tools menu, choose Options, and then select the Preview tab.

    2. Set up a Copy or Move button.

    3. From the Tools menu, choose Customize Toolbar, and then display the corresponding tool on the toolbar.

    4. In the Viewer, select a message.

    5. On the Microsoft Exchange Client toolbar, click the Copy or Move button.

To determine folder size

    1. In the Viewer, select a folder.

    2. From the File menu, choose Folder Size.

To view a message using a fixed font

    1. Right-click the text in the preview pane.

    2. From the shortcut menu, choose Fixed Font.

Preview Pane Extension Shortcut Menu

When you use the preview pane, you can find commands for that message on its shortcut menu.

To open a shortcut menu

· In the preview pane, right-click a message. Depending on the message, the following commands are available.

Command

Description

Open

Opens the message in normal mode so that you can identify and work with attachments.

Remind

Opens the Preview Pane Reminder dialog box, where you can set a reminder for the message.

Previous

Displays the previous message.

Next

Displays the next message.

Copy

Copies selected text to the Clipboard.

Select All

Selects all text in the message body.

Find

Opens the Find dialog box.

Fixed Font

Displays the message body in a monospace font.

Stop Reading

Stops the mark-as-read countdown timer. When you choose this command, the message will not be marked as read. You can also press the ESC key to stop the timer.


Options Dialog Box and Preview Tab Options Settings

Show Message Header in Preview Pane

Select this option to hide or display the preview message header. The preview pane message header displays the contents of the To and Subject boxes of a message.

Mark Messages as Read in Preview Pane

Select this option to mark messages as read. When this option is selected, you can modify the time period or defer marking messages. The default is 5 seconds. To adjust the time period, select the number in the Wait box and then type a new number.

Defer Marking Until Selection Changes

Select this option to mark messages as read only when you click off the message. This option is available only when the Mark Message as Read option is selected. You can use the two marking options together. If you don't want messages you read using the preview pane to be marked as read, clear Mark Message as Read in the preview pane. If you want to be certain you've read a message before it's marked, select both Mark Message as Read and Defer Marking Until Selection Changes.

Synchronize Every <variable> Minutes After Full Idle

Select this option to synchronize your folders with their offline counterparts every <variable> number of minutes. To adjust the time period, select the option, select the number displayed in the box, and then type a new number.

Before you can set a synchronization default time period using Preview Pane Extension, you must create at least one offline folder.

Process Secure Messages

Select this option to process secure messages in the preview pane. Secure messages have been encoded using the Microsoft Exchange Client Encrypt Message Contents and Attachments feature.

Reminder Administrator

Select this option to open the Preview Pane Reminder Viewer, where you can review your reminders.

Copy/Move Buttons Box

Select these options to copy or move a message to a folder. You can also specify an action to automatically follow any copy or move, such as opening the item below it, returning to Microsoft Exchange Server, or doing nothing. Choose New or Modify to open the Button Properties dialog box, and then select the options you want. Choose Delete to delete a button.

Before any options you set in the Copy/Move Buttons box are available, you must customize your Microsoft Exchange Client toolbar by choosing the Customize Toolbar command on the Tools menu. Preview Pane Extension buttons are in the Customize Toolbar dialog box. After they have been added to the toolbar, ToolTips are available.

Message Reminder Viewer Dialog Box Options

Use this dialog box to view reminders you've set. You can select a way to view them (either View Reminders as They Expire, View All Message Reminders, View All Folder Reminders, View Message Reminders By Date, or View Folder Reminders By Date) by choosing a new filter.

When you choose View All Message or All Folder Reminders, double-click an item to read it. When you choose View All Messages or All Folder Reminders By Date, you can change the date from its default (the default is the current date).

When you work in the Reminder Viewer, you can use the following buttons:

Button

Description

Close

Closes the Reminder Viewer.

Open

Opens a message if a message reminder is selected

Properties

Opens the Set Reminder dialog box, where you can revise a reminder

Disable

Maintains reminder information but doesn't display a reminder

Snooze

Delays the reminder according to the time you add using the Snooze dialog box.

Select All

Selects all messages and folders.

Delete

Deletes the selected reminders. Use the Delete button with care; there is no Undo to recreate reminders that you delete.


Right-click on a message or folder in the Reminder Viewer to open a shortcut menu that displays the commands you can use with that message or folder.

Message Reminder Dialog Box Options

Disabled

Select to disable, but not lose, reminder information you set in the When and Occurs boxes. In the upper-right corner, check the information to verify you've selected the number of messages or folders you want to set a reminder on.

When Box

Use this area to set up the reminder.

Remind at

You can type a time, such as 12:00 P.M., or set the time and date by using any one or all of the Time Remaining boxes.

On

Click the > button to open the calendar if you want to change the day displayed in the On box. When you open the calendar, double-click the date you want. Use the buttons in the upper-right and upper-left corners of the calendar to move back or forward one month per click.

Time Remaining

You can use the Days, Hours, Minutes, and Seconds boxes to set a reminder. Otherwise, the boxes display the time remaining before the reminder appears.

Occurs

Use the features available in this area to set a recurring reminder for one of the following intervals: every 15 or 30 minutes, hourly, daily, weekly, monthly, yearly, or by the weekday. Select By Weekday to enable the days of the week check boxes. The default recurring reminder is None. Choose Save to save your reminder. Choose Viewer to open the Message Reminder Viewer.

.Dat Files

Two files, Reminder.dat and Buttons.dat, contain the dynamic data that is written when a user uses Preview Pane Extension. Buttons.dat contains all the button settings and customized options for a user; Reminder.dat contains all the reminders.
