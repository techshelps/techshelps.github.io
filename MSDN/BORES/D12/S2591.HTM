Appendix E Tools

Microsoft Exchange Server provides a variety of tools that can be used to troubleshoot problems in your organization. These tools are in the Tools directory on the Microsoft BackOffice Resource Kit: Part One compact disc. The following tools are described in this appendix. Additional tools are available on the compact disc.

Name

Directory

Description

Mailbox Cleanup Agent

Cleanup

Configures different age limits on mailbox folders.

Microsoft Exchange Server Preview Pane Extension

Cliext

Adds a third area for previewing the body of messages in the Microsoft Exchange Client.

Event Log Tools

Eventlog

Monitors and filters events and sends alerts when specified events occur.

Import Header Tool

Headers

Creates import or export templates for the objects selected from the directory.

IMC Configuration Restorer

Imccopy

Exports specific Internet Mail Connector configuration information into a file so that it can be imported to another Internet Mail Connector.

IMC Sample Extension DLL

Imcext

Intercepts inbound messages from simple mail transport protocol (SMTP) hosts and selectively reroutes those messages to other SMTP hosts before they are processed by the Internet Mail Connector.

IMC Configuration Extractor

Imcsave

Extracts Internet Mail Connector messages and configurations from the Microsoft Exchange Server information store and directory. These messages and configurations are then restored on another computer.

Microsoft Exchange Address Export to MS Mail

Msmailex/Eximwiz

Converts Microsoft Exchange Bulk Export format (.csv) to MS Mail import format.

RPC Counter

Rpc

Measures remote procedure call (RPC) traffic between the client and the server.

Bulk Advanced Security Tool

Security

Encrypts and decrypts messages.

Command Line Mail Sender

Sendmail

Sends mail messages and file attachments from the command line.

Mailbox Statistics

Storstat

Collects statistics about e-mail usage.

MS Mail SMTP Gateway Proxy Migration

Msmailex/Migsmtp

Preserves current Internet addresses of the recipients and adds them as secondary proxies to the Microsoft Exchange address list, eliminating the need for end users to republish their new Internet address for their business associates.

MS Mail Proxy to Microsoft Exchange Proxy Conversion

Msmailex/Proxy

Converts existing SMTP/MHS/X.400 addresses from MS Mail Export format to Microsoft Exchange Bulk Import format (.csv) to be added as custom recipients to a Microsoft Exchange recipient container
