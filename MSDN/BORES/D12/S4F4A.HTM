Installing the Mailbox Cleanup Agent

    1. In the Tools\Cleanup directory on the Microsoft BackOffice Resource Kit: Part One compact disc, double-click Setup.exe.

    2. Provide the following information:

  • The Microsoft Exchange Server computer where the Mailbox Cleanup Agent will reside. This can be any server in the site.
  • The site name.
  • The container where the Mailbox Cleanup Agent will be located. You can place it in any recipient container in the site (for example, the Recipients container).
  • The service account and service account password. This account is the security context the agent runs under. It can be any account, but it's a good idea to use the service account that you are using for the Microsoft Exchange Server services.
  • The service directory specifying where the Mailbox Cleanup Agent will be installed. The path to this directory must include a drive letter.

    3. In the Microsoft Exchange Server Administrator program, find the Mailbox Cleanup Agent in the container you specified in the previous step and open its property pages. Configure the containers and schedule pages.

    4. After Setup is completed, start the Mailbox Cleanup Agent service using Services in Control Panel.

Note

If you start the service before configuring the agent's property pages, the service stops and logs an event indicating that it has not been configured.
