Using Forms Designer with Public Folders

You can use custom forms to create surveys, classified ads, or order forms that you want to manage by working with public folders. Use the Microsoft Exchange Forms Designer to create forms and applications, and then work in Microsoft Visual C++â to extend the functionality of these forms and applications. The Forms Designer even contains sample applications that you can adopt as-is or modify to suit your purposes.

For more information, see the Microsoft Exchange Server Application Designer's Guide. You can also visit the Application Farm Web site at:

Considering All Storage Options

When you store forms, there are two locations to choose from:

  • Public folders
  • Organization forms library

Public folders allow many users to view and store information and are useful when you have many people adding information or posting forms.

The organization forms library is more restrictive than public folders because only an administrator can install the forms. However, this can be changed by allowing all users to have full ownership permissions for the organization forms library.
