[Site]

This section contains entries that specify the names of directory objects in the site, such as the site and server names.

Entry

Description

Value

ExistingServerName=

Name of the server in the existing site.

Server name

SiteName=

Name of the new site if ExistingServerName is not provided.

Site name

SiteProxyName=

Proxy name of the new site. Used to determine the site addresses for connectors.

Site proxy name

InternetSiteName=

Internet site name if the Internet Mail Connector is being installed.

Internet site name (for example, Litware.com)
