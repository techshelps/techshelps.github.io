Setting Up a Delegate User on a Mailbox

You can assign another user to act as a delegate for a user's mailbox or Microsoft Schedule+. A delegate can perform various activities using someone else's mailbox or schedule depending on the permissions the mailbox owner grants to the delegate. The procedures in this section describe how to grant a user Delegate permissions for someone else's mailbox and open another user's mailbox as a delegate.

To grant a user Delegate permissions for a mailbox

    1. Log on to the Microsoft Exchange Client as the mailbox owner.

    2. In the folder list, select the mailbox or a mailbox folder.

    3. From the File menu, choose Properties, and then select the Permissions tab.

    4. To add a name to the Name/Role list, choose Add.

    5. Under Type Name Or Select From List, type or select the name, choose Add, and then choose OK.

    6. In the Name/Role list, select the name of the user to whom you want to grant permissions.

    7. Under Permissions, select a predefined role in the Roles box, or select the individual permissions.

You must grant a minimum of Read Items (Reviewer role) permission to allow another user to open the mailbox.

Choose OK.

To open another user's mailbox as a delegate

    1. From the Tools menu in the Microsoft Exchange Client, choose Services.

    2. In The Following Information Services Are Set Up In This Profile box, select Microsoft Exchange Server, and then choose Properties.

    3. Select the Advanced tab, and then choose Add.

    4. In the Add Mailbox box, type the name of the additional mailbox you have permission to access.

    5. Choose OK until all open dialog boxes are closed.

Note

When a delegate opens the mailbox, the delegate can open only the folders the user has granted permission for.
