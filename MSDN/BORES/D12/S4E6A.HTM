Installing Address Book Details Templates for Languages

You can customize the directory details template to change the language of the Address Book. The language details templates show text provided by the Microsoft Exchange Server (such as user properties in the Address Book) in the language on the Microsoft Exchange Client. Details templates for English, French, German, and Japanese are installed automatically. Installing language details templates does not affect server performance.

Note

Templates are not replicated across sites.

You can also customize the text shown in the display dialog box of the Address Book, such as what type of information is stored in the directory and the format the information is displayed in.

For more information about customizing details templates, see the Microsoft Exchange Server Administrator's Guide.
