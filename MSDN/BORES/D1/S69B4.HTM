Installing sp_drop_fkeys

If you ran the BackOffice Resource Kit CD-ROM Setup, you can access Spdropfk.sql from the Bork\Sql\Sprocs directory on your computer. Otherwise, install it from the BackOffice Resource Kit CD-ROM.

To install sp_drop_fkeys

Get the Spdropfk.sql script from the Sql\Sprocs directory on the BackOffice Resource Kit CD-ROM.

sp_drop_fkeys is installed from the script Spdropfk.sql. At the command prompt, use the isql utility to run the Spdropfk.sql script. For example:

C:> ISQL /Usa /Ppassword /Sserver_name /ilocation\spdropfk.sql

where:

/Ppassword

Is the password for the SA login.

/Sserver_name

Is the name of the server on which SQL Server resides.

/ilocation

Is the full path to the Spdropfk.sql script.
