Part 7 - Data Warehousing

Many Microsoft SQL Server customers are already using SQL Server for data warehousing. Part 7 discusses data warehousing and how you can begin implementing it for your enterprise. The following topics are included:

  • Online transaction processing (OLTP) and data warehousing
  • Data warehouse design
  • Star-schema design
  • Data marts
  • Architecture of a data warehouse
  • Constructing a data warehouse
  • Arcadia Bay demonstration
