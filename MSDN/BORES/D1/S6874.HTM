* The 1.02.3231 and 1.02.3731 drivers are no longer shipped with any Microsoft products and are considered obsolete. They were ODBC version 1.0 drivers, and as such, only had Win16 drivers.


Customers who have the SQL Server versions listed above will have the corresponding ODBC driver version in the SQL Server client utilities. The drivers are also shipped with several applications or development languages, such as Microsoft Office, Visual Basic, and Visual C++.

Users can also obtain self-extracting zip files of drivers from ftp://ftp.microsoft.com/developr/odbc/public.
