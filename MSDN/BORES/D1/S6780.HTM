Cost-Effective Licensing

User-based licensing models of traditional client/server RDBMs, in which customers are charged for the number of concurrent users accessing the database, have proven unworkable for the Internet. Microsoft SQL Server Internet Connector, sold separately from Microsoft SQL Server, provides a simple, flexible, and cost-effective way for customers to license their databases for use on the Internet or corporate intranets.

Internet Connector licensing is an open solution, providing access to Microsoft SQL Server from IIS or from third-party Web servers. Internet Connector allows an unlimited number of Internet and intranet connections to a single SQL Server.

Internet Connector licensing is supported for version 6.0 and version 6.5 of Microsoft SQL Server.
