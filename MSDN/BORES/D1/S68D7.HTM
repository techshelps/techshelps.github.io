ODBC ANSI Upgrade Changes

When you upgrade from Microsoft SQL Server version 6.0 and its associated Microsoft ODBC drivers to Microsoft SQL Server 6.5 Service Pack 2 and its associated Microsoft ODBC driver, changes in behavior are introduced. Review the "Open Database Connectivity" article in Part 6, "Programming for Microsoft SQL Server." If you are using drivers other than Microsoft ODBC drivers, consult the documentation from your driver vendor.
