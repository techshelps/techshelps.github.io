Conventions Used in This Guide

The following table summarizes the typographical conventions used in this guide.

Convention

Description

bold

Menus and menu commands, command buttons, toolbar buttons, tab and dialog box titles and options, command-prompt commands, and portions of syntax that must be typed exactly as shown. Bold is also used for elements of a SQL Server database such as table, column, view, index, device, and server names, data types, and configuration and database options as well as Transact-SQL stored procedures.

Initial capitals

Names of applications, file names, and directories in a path.

Italic

Information you provide, terms that are being introduced, and book titles.

Monospace

Example code, statements, and commands, program code and program output.
