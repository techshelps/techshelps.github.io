Windows NT Services

It is recommended that all services be turned off because services use valuable system resources and therefore potentially limit SQL Server performance. Candidates for disabling are:

  • Alerter
  • ClipBook Server
  • Computer Browser
  • Messenger
  • Network DDE
  • Network DDE DSDM
  • Schedule

In most cases, the following services should remain enabled:

  • EventLog
  • MSSQLServer
  • Server
  • SQL Executive
  • Workstation

You can turn off services from the Services dialog box, which is accessed by clicking Services from Control Panel.
