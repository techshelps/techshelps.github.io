Installing the SQL Server Object Comparison Tool

The SQL Server Object Comparison tool needs approximately 4.2 MB of free hard-drive space for file installation. The SQL Server Object Comparison tool is a 32-bit application that requires Microsoft Windows 95, Microsoft NT Server version 4.0 or Microsoft NT Workstation version 4.0. Microsoft SQL Server version 6.5 is the target server platform for comparisons. The server that will be used as the source or target of a comparison must be configured as an ODBC data source.

The following files are created in the SQL Server Object Comparison install directory when you run a comparison:

  • Machine.ini (used internally by SQL Server Object Comparison)
  • Results.log (a logging file with full logging)
  • Results.txt (a logging file containing only PASS or FAIL)

To install the SQL Server Object Comparison tool

  • Run Setup.exe from the Sql\Sqlcomp\Disk1 directory of the BackOffice Resource Kit CD-ROM.
