Stopping SQL Trend Analyzer Data Collection

To stop SQL Trend Analyzer data collection

From the command prompt, go to the Sql\Sqltrend directory on the BackOffice Resource Kit CD-ROM.

Run the following command:

stopsqltrend server,sa_password

For example:

stopsqltrend clown,red

This will stop SQL Trend Analyzer data collection on the server Clown (SA password "red").
