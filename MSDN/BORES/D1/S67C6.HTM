Publishing with SQL Workbook Publisher

SQL Workbook Publisher is a Microsoft Excel 97 add-in that makes it easy for users to easily analyze data stored in SQL Server databases. Users create workbooks that can be populated with data from SQL Server databases. SQL Workbook Publisher enables these workbooks to be saved to SQL Server databases, where they can be shared with many people within an organization. With a few mouse clicks, users can obtain data stored in a SQL Server database in a formatted Excel workbook.

You can install SQL Workbook Publisher on your Microsoft Internet Information Server to enable users to browse workbooks saved on SQL Server from your corporate Intranet. With Microsoft Internet Explorer, users can view existing workbooks saved on one or more SQL Workbook Publisher–enabled servers.

SQL Workbook Publisher is included on the BackOffice Resource Kit CD-ROM, along with detailed documentation about its usage.
