sp_help_sourcetext Procedure

sp_help_sourcetext is a stored procedure that addresses formatting problems encountered with the system stored procedure sp_helptext. The source text of views, triggers, stored procedures, rules, and defaults are stored in the syscomments table. sp_helptext can be used to view that text, providing it was not saved in encrypted format. The text is stored as a series of 255-byte segments. When sp_helptext prints the source text of an object, it treats these arbitrary 255-byte boundaries as newlines. This breaks up the displayed source in ways that make it difficult to read.

sp_help_sourcetext addresses that problem by not treating the 255-byte boundaries as newlines, reproducing the original source text. sp_help_sourcetext can be used as a functional replacement for sp_helptext.

This version of sp_help_sourcetext works on SQL Server versions 4.21a and later.

sp_help_sourcetext is created as a stored procedure in the master database.
