Devices

The SQL Server term for a physical file used by the server is device. There are three types of devices:

  • Disk dump devices—define SQL Server dump files that reside on disk drives.
  • Tape dump devices—define SQL Server dump files that reside on tape drives.
  • Database devices—The physical disk files that hold databases, or portions of databases.

Devices are built of 2K pages, just like databases are.
