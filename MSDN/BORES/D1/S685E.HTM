Controlling Thread Priority

An item that controls thread priority is the System application "tasking" option of the Windows NT 3.51 Control Panel. This allows you to control the degree of foreground priority boost that normal priority class applications receive. A foreground application is one that is made active by selecting it on the Windows NT desktop, therefore bringing it to the foreground. All other applications running are then termed background applications.

Note In Windows NT 4.0, you can change the performance boost in the Performance tab in the System Properties dialog box in Control Panel.
