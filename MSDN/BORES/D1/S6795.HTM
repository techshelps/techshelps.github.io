sysdevices

The information about devices is stored in the system table master.dbo.sysdevices. If a user selected the name, phyname, low, and high columns for a default master device, the output would be: 

NAME
PHYNAME
LOW
HIGH

master
c:\mssql\data\master.dat
0
12799


Low is the starting virtual page number for the master device (0), and high is the ending virtual page number (12799). If you then execute the following DISK INIT statement, another row would be added to the sysdevices table with the following information (see the result set), where LOW = VDEVNO*16777216, and HIGH = LOW + SIZE – 1.



DISK INIT NAME = 'sample', PHYSNAME='d:\sqldata\sample.dat', VDEVNO=1, SIZE=20480




NAME
PHYNAME
LOW
HIGH
sample
d:\sqldata\sample.dat
16777216
167797695
