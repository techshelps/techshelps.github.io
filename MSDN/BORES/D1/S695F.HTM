Changing the OEM Code Page of Windows NT and Windows 95

You can change the OEM code page by altering the value in:

HKEY_LOCAL_MACHINE
    \SYSTEM
        \CurrentControlSet
            \Control
                \Nls
                    \CodePage
                        \OEMCP

to the code page you want. The change takes effect after you restart Windows NT or Windows 95. By default, the OEM code page is set to 437.

Caution Modifications to the Registry can be risky, and should be carefully considered. In this instance, the OEM code page can be changed by changing the country settings. Additionally, it is risky changing the OEM code page without setting the related ANSI code page.
