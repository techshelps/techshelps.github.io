Vbsql.ocx Replaces Vbsql.vbx

SQL Server 6.5

A Visual Basic version 3.0 project that uses the 16-bit DB-Library for Visual Basic, Vbsql.vbx, should be ported to the new DB-Library for Visual Basic OLE custom control, Vbsql.ocx. For more information, see "Porting an Old DB-Library for Visual Basic Project" in SQL Server 6.5 Books Online.
