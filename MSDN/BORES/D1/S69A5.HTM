Installing Multi.xls

If you ran the BackOffice Resource Kit CD-ROM Setup, you can access Multi.xls from the Bork\Sql\Examples directory on your computer. Otherwise, install it from the BackOffice Resource Kit CD-ROM.

To install Multi.xls

Get Multi.xls from the Sql\Examples directory on the BackOffice Resource Kit CD-ROM.

Start Excel 97.

The macro has been optimized for Excel 97.

On the File menu, click Open.

Find Multi.xls and click Open.

Click Enable Macros.
