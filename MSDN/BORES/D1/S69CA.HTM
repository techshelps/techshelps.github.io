Importing and Summarizing SQL Trend Analyzer Data

To import and summarize SQL Trend Analyzer information for reporting

From the command prompt, go to the Sql\Sqltrend directory on the BackOffice Resource Kit CD-ROM.

Run the following command:

import server,sa_password

For example:

import clown,red

This will stop SQL Trend Analyzer data collection on the server Clown (SA password "red"), import the data from the collection file directory into SQL Server, and summarize the data.

Note SQL Trend Analyzer will remember the path that contains the collection data.
