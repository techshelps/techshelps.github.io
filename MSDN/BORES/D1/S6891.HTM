Planning a Design

When you decide to develop a data warehouse, you need to have a clearly defined business case. In this business case you should:

  • List the players.
  • Explain the expected value of the data warehouse to the enterprise.
  • Outline the framework and direction of the implementation process.
  • Define the enterprise's most vital business functions.
  • Evaluate the enterprise's overriding facts and subject-oriented dimensions.

Once you have built your business case, evaluate the available technology. You need the following:

  • One or more Windows NT servers
  • One or more SQL Server licenses
  • SQL Server data extraction or transfer tools
  • Microsoft decision support applications
