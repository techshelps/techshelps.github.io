SQL Trend Analyzer

SQL Trend Analyzer is a tool designed to automatically collect, store, and analyze historical SQL Server performance information obtained from SQL Trace. This data can be used for performance forecasting, performance analysis, capacity planning, statistical trend analysis, and so on. The data can also be combined with other performance data obtained from tools such as Windows NT Performance Monitor to identify causes of macroscopic performance bottlenecks, such as saturated I/O subsystems, paging, or saturated CPUs.

SQL Trend Analyzer works by collecting SQL Server command information on a specified server, importing this data into a SQL database, and then summarizing this data into useful information. There are three main components to SQL Trend Analyzer:

  • Setup.cmd (the setup and startup script)
  • Import.cmd (the collection and summarization script)
  • SQL Trend Reports97.xls (Visual Basic for Applications macros for front-end reporting and analysis)
