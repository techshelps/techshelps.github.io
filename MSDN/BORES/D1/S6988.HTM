Part 9 - Tools

Part 9 describes each of the tools for SQL Server shipped on the BackOffice Resource Kit CD-ROM:

  • Adduser.sql script
  • Arcadia Bay demonstration
  • DB Checker
  • Deadlock demonstration
  • Filltabl utility
  • Firstsql.cpp
  • Multiserver Excel macro
  • sp_help_set_options procedure
  • sp_help_sourcetext procedure
  • sp_drop_fkeys procedure
  • SQL Load Simulator
  • SQL Server Object Comparison tool
  • SQL Trend Analyzer
  • SQL Workbook Publisher
  • SQL-DMO code samples
  • Sqlhdtest utility

    Note Most tools are available for Intel platforms only and are available only in English.
