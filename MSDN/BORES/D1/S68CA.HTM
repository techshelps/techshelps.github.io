Books Online

SQL Server 6.5

SQL Server 6.5 Books Online uses the Infoview.exe program to view text.

SQL Server does not have to be installed in order to read Books Online; Books Online can be installed independently:

Create a directory C:\Sqlbks.

Copy the \Sqlbks65\Sqlbooks.* files and \I386\Infoview.exe from the CD-ROM into that directory. You will not be able to view Books Online using Msin32.exe.

Create a Program Manager item with:

  • A command prompt of C:\Sqlbks\Infoview.exe Sqlbooks.mvb.
  • A working directory of C:\Sqlbks.

SQL Server 6.0

SQL Server 6.0 Books Online uses the Msin32.exe program to view text.
