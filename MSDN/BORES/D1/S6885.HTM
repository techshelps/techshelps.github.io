Data Access Objects

With Data Access Objects, you can manipulate databases in several database formats.

You can use Data Access Objects (DAO) to manipulate databases in either the native Jet database engine .mdb format or in other installable ISAM database formats, including Microsoft FoxPro, dBASE, Microsoft Excel, Btrieve, Paradox, and delimited Text.

In addition, you can use DAO to access Microsoft SQL Server and any other database that can be accessed with an ODBC driver using the same DAO code. This can be done through the Jet engine or directly through ODBC.
