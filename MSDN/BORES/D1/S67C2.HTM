Integrating Database-Web Connectivity

Microsoft offers several tools you can use to integrate databases with your intranet content.

Companies choosing to adopt World Wide Web technology need to integrate it with their existing information technologies, such as databases. Web-database integration software helps Webmasters and database administrators solve two business problems:

  • Publishing snapshots of database information on a Web page
  • Using a Web browser as a database front end

Microsoft has created Web-database integration tools to help you solve these problems. Choose the combination that works best for you.

You must use the special license pack, SQL Server Internet Connector, when integrating a Web server with Microsoft SQL Server. For more information, see the "Microsoft SQL Server Internet Connector" section, later in this article.
