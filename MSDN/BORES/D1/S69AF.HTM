Implementing sp_help_sourcetext

Use sp_help_sourcetext instead of sp_helptext to get a more readable display of the source text of stored procedures, views, triggers, rules, and defaults.

Usage

sp_help_sourcetext objname

where:

objname

Specifies the object about which the definition is to be displayed. Objname is a required parameter.

Remarks

If the objname given specifies an object created with the encryption option, sp_help_sourcetext does not return a result set. If the objname given specifies an object that either does not exist or is not the type of object for which the source text is stored, sp_help_sourcetext does not return a result set. If more than one user has created an object with the same name as specified in the objname parameter, sp_help_sourcetext will return the text of the object with the smallest object ID, not all of them.
