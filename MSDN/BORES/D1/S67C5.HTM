Publishing with SQL Server Web Assistant

The following illustration shows how users access a database with SQL Server Web Assistant. It is assumed that SQL Server 6.5 is already installed.

Start SQL Server Web Assistant either on the server or on a 32-bit SQL Server client.

(The illustration shows the simpler, less common case in which the SQL Server Web Assistant runs on the server.)

Proceed through each wizard-like screen, making sure to schedule data refresh times.

At the specified scheduled time or data change instance, SQL Server executes the query and updates the old HTML file with the converted data. At this point, the HTML file can be viewed with a Web browser.
