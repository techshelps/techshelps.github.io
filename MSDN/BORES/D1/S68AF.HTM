RAISERROR and @@ERROR (*2701 or SETERROR Option)

SQL Server 6.5

The RAISERROR statement sets @@ERROR to 0 if the severity is between 1 and 10 inclusive. Messages with severity levels of 10 and under are not errors, but they do provide additional information. If you set the msg_id by using the WITH SETERROR option, the RAISERROR statement assigns the msg_id to @@ERROR regardless of severity.

SQL Server 6.0

@@ERROR is set to 50,000 for messages with severity levels 10 and under.
