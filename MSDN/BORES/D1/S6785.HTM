Devices

Devices are normal operating files used to store SQL Server databases. Multiple databases can be stored on a single device.

The master system device contains four databases:

  • The master system database is used to track system-wide information.
  • The tempdb system database is used for temporary storage.
  • The model system database is used as a template for newly created databases.
  • The pubs sample database is used as an example throughout SQL Server documentation.

A single database can span multiple devices.

The msdb system database is stored on two devices:

  • The msdbdata system device contains only data.
  • The msdblog system device contains only the transaction log.

These system devices and databases are shown as they are created by SQL Server Setup.

A typical database, like mydb, is also stored on two devices:

  • The database data device contains only data.
  • The database log device contains only the transaction log.

A typical database can be expanded in one of two ways:

  • An existing device can be expanded, and then the database can be expanded onto the new space on the existing device.
  • A new device can be created, and then the database can be expanded onto the new device.
