Introduction

Welcome to the Microsoft SQL Server Resource Guide. This guide is designed for people who are, or want to become, expert users of Microsoft® SQL Server™.

If you are responsible for designing a database solution, administering one or more SQL Server databases, or programming applications to streamline database activities, this guide is for you. It has been compiled largely from articles written by Microsoft staff in answer to frequently asked questions and to provide insights and usage ideas.

In addition to the information in this guide, various SQL Server utilities are included on the Microsoft BackOffice™ CD-ROM. Throughout the guide, you will learn about different uses for SQL Server. Demonstrations of these product applications can be found on the CD-ROM. If you develop SQL Server database systems, you can pick up code snippets from the CD-ROM. Also included on the CD-ROM are product add-ons and stand-alone applications you can use to simplify your administration duties.

The Microsoft SQL Server Resource Guide supplements the documentation included as part of the Microsoft SQL Server product and does not replace the documentation as the source for learning how to use Microsoft SQL Server.
