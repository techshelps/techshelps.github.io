About the Resource Guide

Part 1, Product Guide and Strategies

Discusses the benefits of SQL Server, how it enforces data integrity, and its scalability on the Windows NT® Server operating system. Additionally, you will gain insight on using SQL Server for Web solutions.

Part 2, Understanding SQL Server Architecture and Components

Describes in greater detail the architecture of SQL Server, the internal structure of the database, and Microsoft Distributed Transaction Coordinator.

Part 3, Planning and Implementing Your SQL Server Solution

Provides information to help you plan, design, and implement your SQL Server architecture. Backup and recovery, database-Web connectivity, efficient queries, and hardware planning are discussed here.

Part 4, Maintaining Your SQL Server Solution

Demonstrates everyday maintenance tasks on your SQL Server. You will learn tips about upgrading, managing replication, checking database consistency, scheduling database tasks, and tuning.

Part 5, Integrating with BackOffice

Describes how you can use SQL Server with the Microsoft BackOffice family.

Part 6, Programming for Microsoft SQL Server

Provides information that will help you incorporate your program into the SQL Server framework.

Part 7, Data Warehousing

Discusses the use of SQL Server as a data warehouse. A demonstration of a data warehouse using Microsoft technology is included on the CD-ROM.

Part 8, Reference

Provides information about changes to SQL Server from version to version, SQL Server code pages, database size estimates, and locking.

Part 9, Tools

Discusses the SQL Server tools included on the BackOffice Resource Kit CD-ROM, including installation instructions, descriptions, and usage.
