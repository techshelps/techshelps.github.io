Installing Filltabl

If you want to run only Filltabl, you can install the Filltabl.exe file. Filltabl.exe is an Intel executable for Windows NT or Windows 95 clients. If you ran the BackOffice Resource Kit CD-ROM Setup, you can access Filltabl from the Start menu. Otherwise, install it from the BackOffice CD-ROM.

To install Filltabl

Copy Filltabl.exe to the Windows\System (Windows 95), or the Winnt\System32 (Windows NT Server or Windows NT Workstation) directory on your client.

To customize Filltabl, for example if you are a DB-Library developer, copy Filltabl.c, Errhndl.c, and Filltabl.mak to your DB-Library development system.
