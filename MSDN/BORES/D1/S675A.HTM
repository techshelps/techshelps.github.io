Clusters

Windows NT Server offers many cluster-wide services: security, naming, performance monitor, software asset management (SMS), transactions (MS DTC), distributed objects (DCOM), node and resource failover, and automatic IP address assignment (DHCP). Companies like Compaq, Digital Equipment Corporation, Hewlett Packard, Intel, NCR, Tandem, and SGI are building on Windows NT Server and are working with Microsoft to make Windows NT Server clusters even more powerful. Many additional services are planned, including an industry-standard, cluster application programming interface (API).
