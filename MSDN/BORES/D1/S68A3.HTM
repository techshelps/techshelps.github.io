FROM Clause and Table Names (*110)

SQL Server 6.5

Errors are reported when redundant table names appear in the FROM clause. If two table names are not identical, the database IDs and table IDs are compared to determine whether or not they are the same table.

SQL Server 6.0

Redundant table names in the FROM clause are supported. SQL Server uses string comparisons alone to determine whether two table names identify the same table.
