Using Different FILLFACTOR Values

Other values for FILLFACTOR reduce the number of rows per page on data pages and leaf index pages. To compute the correct values when using FILLFACTOR, multiply the size of the available data page (2,016) by the FILLFACTOR. For example, if your FILLFACTOR is 75 percent, your data page would hold 1,512 bytes. Use this value in place of 2,016 in Step 2 for both clustered and nonclustered index calculations.
