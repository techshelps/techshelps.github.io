Installing SQL-DMO Code Samples

If you ran the BackOffice Resource Kit CD-ROM Setup, you can access the SQL-DMO code samples from the Bork\Sql\Examples directory on your computer. Otherwise, install it from the BackOffice Resource Kit CD-ROM.

To install SQL-DMO code samples

Get Dmosamp.vbp and Dmosamp.frm from the Sql\Examples directory on the BackOffice Resource Kit CD-ROM.

Start Visual Basic.

The sample was written using Visual Basic version 4.0.

Load the Dmosamp.vdp file.

This will automatically pick up the other program files, provided they are in the same directory.

Select the Dmosamp.frm from the Project window to view the code, and then click on the form controls to bring up the Code window.

On the Run menu, click Start. Alternatively, press the F8 key to step through the code.
