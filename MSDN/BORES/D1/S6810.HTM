Installing Replication

After SQL Server is installed, installing replication using SQL Enterprise Manager requires three steps: installing the distribution server, configuring publication servers, and configuring subscription servers.

The following procedure assumes a local distribution server when installing replication from SQL Enterprise Manager.

To install the distribution server

Click on a server.

On the Server menu, point to Replication Configuration, and then click Install Publishing.

To configure a publication server

Click on a server.

On the Server menu, point to Replication Configuration, and then click Publishing.

To configure a subscription server

Click on a server.

On the Server menu, point to Replication Configuration, and then click Subscribing.

Alternatively, on the Server menu you can point to Replication, and then click Topology to configure both publication and subscription servers. If you choose to use the Topology method, be sure to configure the publication server first. Otherwise, you will receive error 18483 because SA is not yet defined as a remote login at the site:

Error 18483: Unable to connect to site '<publication server>'
