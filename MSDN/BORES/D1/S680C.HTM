Upgrade to SQL Server 6.5

To upgrade your SQL Server 6.0 to SQL Server 6.5

Make sure you are logged on to Windows NT as administrator or someone with administrative privileges.

Check the network capabilities.

If the server being upgraded has network capabilities, continue with step 3.

If the server being upgraded does not have network capabilities, you can do one of two things:

  • Stop the workstation service to force the upgrade to use local named pipes.

    To do this, perform the following steps:

    In Control Panel, double-click Services.

    Click the Workstation service.

    Click Close.

  • Leave the Workstation service running and install the MS Loopback Adapter.

    To do this, perform the following steps:

    In Control Panel, double-click Network.

    Click Adapters, then click Add.

    Double-click MS Loopback Adapter.

Regardless of your choice, you will need to restart Windows NT once the installation completes.

Run SQL Server 6.5 Setup.exe to upgrade SQL Server.

Specify the same drive and directory for all SQL Server files to be copied during the upgrade.
