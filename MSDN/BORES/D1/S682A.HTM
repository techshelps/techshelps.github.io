Master Device Location

The location of the master device file, Master.dat, can be changed from the default path, C:\Mssql\Data, to any other system location. Consideration should be given to changing the location of this file to a dedicated physical disk unit or disk volume. This location will help to isolate database I/O away from system I/O. In addition, by default, the tempdb database, which is 2 MB in size, takes its initial space allocation from the master device.
