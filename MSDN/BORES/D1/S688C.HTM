Data Warehouse Characteristics

The defining characteristics of a data warehouse are:

  • Subject-oriented design

    Data warehouses are organized around major subjects integral to the enterprise. A data warehouse concerns itself with subjects like customers, vendors, or products.

  • Nonvolatility of data

    Data in data warehouse databases is static. Data is loaded into the warehouse at one time and is processed there, but does not change as it is accessed for DSS queries.

  • Historical time value of data

    In a data warehouse, data is accurate in the context of a specific historical time.

  • Integrated data types

    Before it is loaded into a data warehouse, data from varied sources is integrated into one set format.
