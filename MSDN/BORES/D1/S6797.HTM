Logical and Virtual Page Numbers

Each logical page number in a database relates to an underlying virtual page number in a device. For example, if you execute the following statements:

DISK INIT
NAME = 'sample',
PHYSNAME = 'c:\mssql\data\sample.dat',
VDEVNO = 8,
SIZE = 2048
GO
CREATE DATABASE sampledb ON sample = 4
GO

You create the sample device, then map the sampledb database to that device. Both the device and the database are the same size. The sample device has virtual page numbers from 134217728 to 134219775, and the sampledb database has logical page numbers from 0 to 2047.
