Arcadia Bay Demonstration

Arcadia Bay is a demonstration, included on the BackOffice Resource Kit CD-ROM, of the data warehousing capabilities of Microsoft SQL Server. It includes a star-schema database that works with or without the Arcadia Bay Order Entry System database. It also includes an intranet-based front end so users can drill down into the data from a Web browser.

Arcadia Bay requires the following installed software:

  • Windows NT Server version 4.0 with SP2 or later
  • SQL Server version 6.5 with SP1 or later
  • Internet Information Server version 2.0
  • Internet Explorer version 3.0 or later
  • Exchange Server (optional)

The SQL Mail feature of SQL Server should be configured to log in to a predefined Exchange account, and it should be started on SQL Server.
