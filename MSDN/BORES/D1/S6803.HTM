Upgrading to SQL Server 6.5

Be sure you have the latest and greatest Microsoft database solution—upgrade to Microsoft SQL Server version 6.5 today!

Most of the utilities shipped with this resource guide require SQL Server version 6.5. If you are using SQL Server versions 6.0 or 4.2x, you'll need to upgrade to use the utilities.

This article describes some upgrade issues you should be aware of and offers backup and configuration strategies that can be performed before upgrading to SQL Server 6.5. Be sure to review the Microsoft SQL Server documentation about upgrading SQL Server before performing any SQL Server upgrade. The following topics will be discussed:

  • Back Up
  • Configuration
  • Troubleshooting
