Restarting SQL Trend Analyzer Data Collection

To restart SQL Trend Analyzer data collection

From the command prompt, go to the Sql\Sqltrend directory on the BackOffice Resource Kit CD-ROM.

Run the following command:

startsqltrend server,sa_password,'path of collection file'

For example:

startsqltrend clown,red,'c:\temp'

This will restart SQL Trend Analyzer data collection on the server Clown (SA password "red") and collect data in the temp directory.
