ODBC Clients

Because ODBC is a Windows API, ODBC clients use the ANSI 1252 code page. Any ODBC data source used to connect to a server running an ANSI 1252 code page should have the Convert OEM to ANSI characters check box activated. Any ODBC data source connecting to a server running a 437 or 850 code page should have the Convert OEM to ANSI characters check box activated.
