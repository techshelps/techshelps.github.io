Moving X.400 Connectors

Follow the steps below if you have one or more X.400 Connectors installed on Server A.

To move X.400 Connectors

On Server B, install the required MTA transport stacks.

On Server B, display the properties for each X.400 Connector.

Change the MTA transport stack so that it is a stack installed on Server B.

On Server A, delete the MTA transport stacks.

On Server B, request a full directory update.

On the directory service object under Configuration, Servers, Server B, choose Update Now, and then select Refresh all items in the directory.
