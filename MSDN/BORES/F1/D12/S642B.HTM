Installing Mapindl

Use the following procedure to install Mapindl.

To install Mapindl

From the Exchange\<operating system\platform>\Developr\MAPIndl directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy Mapindl.dll to the Windows\System directory on your computer.

Make sure that Msvcrt.dll and Mfc42.dll exist somewhere in your computer's path because these files are required before you can run Mapindl.dll. If you do not have these files, they can be obtained from Microsoft Visual C++ version 4.2.

Note Mapindl.dll only needs to be present on the computer that originates the MAPI message and attaches the Lotus Notes DocLink. Standard OLE activation in Windows 95 and Windows NT will be used to activate Lotus Notes and display the document.

At the command prompt, switch to the directory where Mapindl.dll resides.

Type the following command:

REGSVR32 MAPINDL.DLL

Typing this command registers the DLL with your system. You can test the installation by starting the Lotus Notes database (Mapindl.nsf), and composing and saving a main topic document. After you reopen the document, select Send MAPI Notification.
