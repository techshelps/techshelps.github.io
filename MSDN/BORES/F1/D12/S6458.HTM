Using PFTreeInfo

You can perform the following tasks by using PFTreeInfo:

  • Count messages recursively.
  • Count subfolders recursively.
  • Count messages and subfolders recursively.

The following figure illustrates the PFTreeInfo user interface.

To count messages recursively

Select the Count messages check box.

Choose Start Count.

To count subfolders recursively

Select the Count sub folders check box.

Choose Start Count.

To count messages and subfolders recursively

Select the Count messages and Count sub folders check boxes.

Choose Start Count.
