Moving Microsoft Exchange Server to Another Computer with a Different Hardware Platform

Follow the steps outlined in the "Moving Microsoft Exchange Server to Another Computer with the Same Hardware Platform" section. However, rather than copying the entire Exchsrvr directory from the Original Server to the New Server, do the following.

From your backed-up copy of the Exchsrvr directory of the Original Server, copy the following subdirectories to the Exchsrvr directory on the New Server: Dsadata, Dxadata, Imcdata, Mdbdata, and Mtadata. If you overwrote all the files from the Exchsrvr directory of the Original Server, you would destroy your executable files.
