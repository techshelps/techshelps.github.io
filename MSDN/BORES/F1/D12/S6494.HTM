Using the Microsoft Mail for Windows Tools

The following table lists the Microsoft Mail for Windows tools included on the BackOffice Resource Kit, Part Two compact disc. More complete descriptions of each utility follow this table.

Warning To prevent the loss of important data, always back up the Microsoft Mail for Windows database before you use any of the utilities described in this document.

Tool

Description

Acctonme.exe

Creates an Admin.nme file from the following existing Microsoft Access files: Access.glb, Access2.glb, and Access3.glb.

Dumpfld.exe

Displays the contents of a folder (.fld) file.

Dumpidx.exe

Displays the contents of a folder index file (.idx files, including Foldroot.idx).

Dumpkey.exe

Lists owners of calendar files.

Escntlr.exe

Removes, installs, and displays the status of enhanced security on a version 3.0 or 3.2 postoffice.

Fixfld.exe

Recovers corrupted folder files.

Fixidx.exe

Recreates a folder index file.

Gimport.exe

Creates a group from a file.

Listuser.exe

Displays the mailbox names and associated eight-digit ID numbers.

Movefold.exe

Enables you to transfer folders to another postoffice.

Mvcal.exe

Renames a corrupted calendar file located on the server.

Netdump.exe

Displays the external postoffice and gateway routing files.

Revert.exe

Resets the unread count for a Microsoft Mail message file (.mmf).
