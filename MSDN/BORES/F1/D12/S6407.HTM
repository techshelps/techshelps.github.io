Installing Internet Idioms

To install the Internet Idioms tool

  • From the Exchange\<operating system\platform>\Client\Inetxidm subdirectory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy Inetxidm.exe and Inetxidm.dll to your system directory (Windows\System on Microsoft Windows 95; \Windows\System32 on Windows NT).

    Note Before trying to run Internet Idioms, verify that you have Msvcrt40.dll in your system directory. You must have this dynamic-link library (DLL) to run Internet Idioms.
