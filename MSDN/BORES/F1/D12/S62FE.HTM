Exporting Mailboxes, Custom Recipients, and DLs

To export mailboxes, custom recipients and distribution lists, follow the steps below.

From the Support\Samples\Csvs\Mailbox directory on the Microsoft Exchange Server compact disc, copy the Mailbox.csv file to your computer.

From the Support\Samples\Csvs\Custom directory on the same compact disc, copy the Custom.csv file to your computer.

From the Support\Samples\Csvs\Dl directory on the same compact disc, copy the Dl.csv file to your computer.

For each of these three files:

In Windows NT Explorer, from the File menu, choose Properties, and then clear the Read-Only check box.

To export the Mailbox.csv file

In the Administrator window for Server A, choose Directory Export from the Tools menu.

Choose Export File, and then select Mailbox.csv.

Choose Container, and then select the Recipients container.

Under Export objects, select the Mailbox check box.

Choose Export.

Edit the Mailbox.csv file, replacing all instances of Server A with Server B.

Save the file.

Repeat these steps for each top-level container. Use different file names for each top-level container and do not specify Mailbox.csv as a file.

To export the Custom.csv file

In the Administrator window, choose Directory Export from the Tools menu.

Choose Export File, and then select Custom.csv.

Choose Container, and then select the Global Address List container.

Under Export objects, select the Custom Recipient check box.

Choose Export.

To export the Dl.csv file

In the Administrator window, choose Directory Export from the Tools menu.

Choose Export File, and then select Dl.csv.

Choose Container, and then select the Global Address List container.

Under Export objects, select the Distribution list check box.

Choose Export.
