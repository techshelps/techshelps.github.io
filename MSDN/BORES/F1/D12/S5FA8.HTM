Server Space Client Extension Tool

The Server Space Client Extension tool enables you to display the total amount of space consumed on a Microsoft Exchange Server computer by your Microsoft Exchange Server mailbox folders. This tool is useful in situations where the amount of space allocated for each mailbox on the server is limited. The Server Space Client Extension Tool is also useful in situations where someone other than the owner manages a mailbox , because it displays the amount of space consumed by each server-based account in your viewer.
