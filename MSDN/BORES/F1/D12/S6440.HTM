Installing the Assistant

To install the Assistant on Windows 95

  • Run Setup from the Exchange\<operating system\platform>\Client\Schedaba directory on the Microsoft BackOffice Resource Kit, Part Two compact disc.

To install the Assistant on non-Windows 95 operating systems

Copy Schedaba.exe to your Windows directory.

Copy all the dynamic-link libraries (DLLs) from Client\Schedaba to your Windows\System directory.

Run Schedaba.reg to add an entry for the tool to the registry.
