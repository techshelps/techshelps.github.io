Before Using the NLS Configuration Tool

Before using this tool, you must log on to an account with administrator rights, then follow the steps below.

At the command prompt, switch to the directory containing the Setupnls.bat file, and then type: Setupnls

This command updates a variety of NLS files in your System32 directory and writes a Setupnls.log file to your Temp directory. You can review this file before proceeding further.

Reboot your computer so that Windows NT Server can update the files. This also makes the outdated files available so that they can be deleted by Cleannls.

At the command prompt, switch to the directory containing the Cleannls.bat file, and then type: CLEANNLS

This deletes the outdated NLS files from your SYSTEM32 directory.

Note Correct Korean sorting requires either the Korean version of Windows NT Server version 3.51, or any language version of Windows NT Server version 4.0 or later.
