6.4 Microsoft SQL Server Directory Structure

Directory

Description

\Dbcheck

A utility that checks the integrity of databases

\Docs

A Help file for the SQL Server utilities

\Examples

Example files for the SQL Server tools and utilities

\Filltabl

A utility that generates a large amount of data for a table

\Loadsim

An application that simulates a large number of user connections to SQL Server

\Sprocs

Stored procedures that drop foreign key constraints for a referenced table, report the current state of the SET options for a connection, and address formatting problems encountered with sp_helptext

\Sqlcomp

A tool that performs object comparisons across a subset of SQL Server object types

\Sqlhdtst

A utility that performs a high-intensity, multithreaded file system stress test

\Sqltrend

A tool that collects, stores, and analyzes performance information obtained from SQL Trace

\Swp

A Microsoft Excel add-in that allows users to analyze data stored in SQL Server databases
