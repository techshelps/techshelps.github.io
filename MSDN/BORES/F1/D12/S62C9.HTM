Procedure

Use the following procedure to update Windows NT Server version 3.51 to Service Pack 4.

To update Windows NT Server 3.51 to Service Pack 4

  • Using the built-in scheduling service in Windows NT Server, schedule the Ntsp4upd.bat file to run when no users are logged on to the server (some time at night is usually best). This batch file takes one argument—the name of the server where the installation directory resides (and from which the Windows NT Server 3.51 Service Pack 4 will be installed). An example of a typical argument might be ntsp4upd cisvr002.

    Note that the batch file uses drive X, so any previous connections made to this drive will be disconnected during the setup process.

This batch file will apply the Service Pack unattended and reboot the server.
