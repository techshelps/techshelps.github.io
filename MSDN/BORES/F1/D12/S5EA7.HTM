Using the Backup Tool

The Backup tool prevents you from having to manually verify whether all Microsoft Exchange Server backups succeeded or failed. This tool uses a script that searches the Windows NT Exchange backup log files and then reports:

  • Statistics on each Microsoft Exchange Server computer that is backed up.
  • The size of each file backed up.
  • The amount of time each backup took.

This script extracts the data into a Microsoft Excel spreadsheet, which is routed to the daily Microsoft Exchange operations report. Therefore, only one report is needed to provide the backup status for all Microsoft Exchange Server computers.
