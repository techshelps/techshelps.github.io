Mvcal Tool

A user's online calendar file is not named with the same eight-digit ID number as the user's Microsoft Mail message file, nor is it named with the user's name (for security reasons). You can use the Mvcal tool to locate a specific user's online calendar file (if, for example, the file is damaged).

When to Use Mvcal

Use this program if a .cal file is corrupted or damaged, and you want to find the owner of that calendar.

Side Effects of Using Mvcal

This tool will find the calendar file for the user name specified and rename it <username>.bad. After the calendar file is renamed, a new file will be created the next time the user logs on. This new calendar file will be created from the user's local calendar file.

To run Mvcal

  • Use the following command-line syntax format:

    mvcal <user name> [-d<drive> -p<password>]

The following table lists the command-line options available in Mvcal, and their descriptions:

Option

Description

<user name>

Displays the name of the user with the corrupted calendar file.

<drive>

Displays the path to the Microsoft Mail database (the default is drive M).

<password>

Shows the password for the Schedule+ administrator (not the Microsoft Mail administrator).


Example of Renaming a Calendar File

To rename John Doe's calendar file on drive N

  • Type the following at the command prompt:

    mvcal JohnDoe -dn -ppassword

    where the Schedule+ administrator's password is "password."
