Escntlr Tool

The Escntlr tool enables you to turn off enhanced security and turn it on for the postoffice. This tool can only be used with Microsoft Mail 3.0 and 3.2 postoffices. Enhanced security is a double-encrypted password that stops Microsoft Mail 2.x clients from accessing the postoffice. It is unrelated to Advanced Security, which is a method of creating a connection to the postoffice that a user cannot access.

When to Use Escntlr

Use this program to display or change the status of enhanced security.

Side Effects of Using Escntlr

Enhanced security will be added or removed, and the files containing the user accounts will be modified.

To run Escntlr

  • Use the following command-line syntax format:

    escntlr <admin> -p<password> [-s] [-r or -i] [-d<drive>] [-?]

The following table lists the command-line options available in Escntlr and their descriptions:

Option

Description

<admin>

The name of the administrator for this postoffice (required).

-p<password>

Specifies the password for the administrator.

<admin>

Specifies the name of the administrator for this postoffice (required).

-s

Displays the status of enhanced security only. This overrides all other options, except for the drive option (-d).

-r

Removes enhanced security if installed.

-i

Installs enhanced security if not installed.

-d<drive>

Specifies the drive where the Microsoft Mail database is located (the default is drive M).

-?

Displays the Help screen.


Note As with all Microsoft Mail utilities, it is highly recommended that you back up your Microsoft Mail database before running Escntlr. At a minimum, you should back up the Glb subdirectory by typing the following command from the root of the Microsoft Mail database directory:

xcopy \glb \bak\ /s /v

This command copies the entire GLB subdirectory to a new directory called Back.

Removing Enhanced Security

The setup program for Microsoft Mail 3.2 removes or adds enhanced security to the postoffice.

To remove enhanced security

  • Type the following at the command prompt:

    escntlr admin -p<password> -r

Installing Enhanced Security

To install enhanced security on drive N

  • Type the following at the command prompt:

    escntlr admin -p<password> -i -dn

Viewing Enhanced Security Status

To view the status of enhanced security on drive N

  • Type the following at the command prompt:

    escntlr -dn -s
