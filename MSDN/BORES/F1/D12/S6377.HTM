Windows AT Command Scheduler and the Windows NT Schedule Service

When you use the Windows AT Command Scheduler, the Windows NT Schedule service runs all jobs that have been scheduled. Because batch jobs are run in the context of the Schedule service, Windows NT security must be considered. When configuring the Schedule service, configure the account to be a member of the Windows NT Backup Operators group. This enables a full backup of the information store and directory.
