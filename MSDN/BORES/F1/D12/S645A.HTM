Using the Phone Book Search Tool

The following procedures outline how to perform specific functions with the Phone Book Search tool, including:

  • Searching by e-mail alias
  • Searching by phone number

Note Microsoft Exchange Server computers do not have an index for phone numbers; therefore this search could be time-consuming. Adding an index on the phone number attribute speeds up this task.

To search by e-mail alias

At the command prompt, switch to the directory where Phone.exe is installed.

Type Phone <Microsoft Exchange Server computer name> <e-mail alias> (for example, Phone exchange1 adelew).

You can also use wildcards to search for multiple phone numbers (for example, Phone exchange1 adelew*).

To search by phone number

At the command prompt, switch to the directory where Phone.exe is installed.

Type Phone <Microsoft Exchange Server computer name> <phone number>.
