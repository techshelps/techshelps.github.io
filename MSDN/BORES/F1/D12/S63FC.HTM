Installing the Event Log Tools

To install the event log tools

  • From the Exchange\<operating system\platform>\EventLog directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy the files into a directory on a computer that is running the services you want to monitor.

    Note To have Event Scan alert you through e-mail, you must install it on a computer that has MAPI installed (for example, a Microsoft Exchange Server computer or a Microsoft Exchange Client computer).
