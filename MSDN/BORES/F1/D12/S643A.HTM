Installing the Proxy Conversion Tool

The Microsoft Mail SMTP Gateway Proxy Migration tool (Perl.exe) is included on the Microsoft BackOffice Resource Kit, Part Two compact disc.

To install the Proxy Conversion tool

  • From the Exchange\Common\Proxy directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy the following files to a directory of your choice.
    • Perl.exe
    • Perl100.dll
    • Convert.pl
    • Go.bat (Sample file optional)
    • In.txt (Sample file optional)
    • Out.txt (Sample file optional)
    • Fail.txt (Sample file optional)

For more information on the PERL program, see the Microsoft Windows NT Resource Kit version 3.51.
