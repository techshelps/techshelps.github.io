Using AutoFill

After the Autofill tool is installed, it runs automatically. However, AutoFill needs to build the list of frequently used names before it can guess a name. For example, the first time you type a name, nothing will happen; however, the next time (after check names has run) when you type the first letter, AutoFill guesses the name.

To delete names from the AutoFill list

  • Using a text editor, delete the name from the Names.dat file.

Registry Settings

The following registry settings are used by AutoFill. If you modify a registry setting to customize AutoFill, you must restart the Microsoft Exchange Client.

Field name

Type

Description

bAliasNm

REG_DWORD

This value can be defined by the user.

If set at 1 (true), AutoFill stores the Alias names in the users list of saved names.

bAlways Guess

REG_DWORD

This value can be defined by the user.

1 AutoFill32 guesses the name while you are backspacing.

0 Backspace functions normally.

BDisplayNm

REG_DWORD

This value can be defined by the user.

If set to 1 (true), AutoFill stores the display names in the user's list of saved names.

bExactMatch

REG_DWORD

This value can be defined by the user.

If set to 1 (true), AutoFill only guesses a name if it is unique. If set to 0 (false), AutoFill gives a best guess.

Data File

REG_SZ

This value can be defined by the user.

The name and location of the text file AutoFill uses to store the display names.

Fields

REG_DWORD

This value cannot be defined by the user.

Query Depth

REG_DWORD

This value can be defined by the user.

The number of characters that are typed before AutoFill guesses the name.

Tbl Depth

REG_DWORD

This value cannot be defined by the user.


To change the number of characters to wait before AutoFill guesses names

  • Modify the Query Depth registry value. If Query Depth is set to zero, AutoFill guesses after the first character is typed. If Query Depth is set to a non-zero value, AutoFill waits until that number of characters are typed before guessing.

To keep AutoFill from loading

  • Remove the AutoFill32 registry key from the following location:

    Software\Microsoft\Exchange\Client\Extensions

    This registry key registers the AutoFill extension with the Microsoft Exchange Client.

To set AutoFill to guess only if the characters entered map to a unique name

  • Set the bExactMatch registry value to 1 (true).

To set AutoFill to guess while backspacing

  • Set the bAlways Guess registry value to 1 (guess).

    If this value is set to 0, AutoFill does not guess while backspacing.

    Note If you set AutoFill to guess while you backspace, the tool highlights text that you backspace over. This means that the text is erased if AutoFill cannot make a guess or when you press the Delete key.

To set whether the display name or alias name is stored

  • To store the display name, set bDisplayNm to 1 (true). To store the alias name, set bAliasNm to 1 (true). You can set both registry settings to true to store both names.
