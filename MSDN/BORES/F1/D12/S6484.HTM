Using the Server Space Client Extension Tool

After you install the Server Space Client Extension tool, Server Space appears as a new command on the Tools menu.

Note You can specify the amount of disk space that can be allocated to each user on a given server. Personal and public folders are displayed with "N/A" to indicate that items stored in these folders do not count towards a user's allocated disk space.

To display the total amount of space consumed by a user's mailbox folders on a Microsoft Exchange Server computer

  • From the Tools menu, choose Server Space.

To add the Server Space Client Extension button to your toolbar

From the Tools menu, choose Customize Toolbar.

Under Available Buttons, select the Server Space button, and then choose Add.

Position the button by using the Move Up and Move Down buttons.
