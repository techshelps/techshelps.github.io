Using PFTreeInfo

Note that this tool is designed for the Microsoft Win32 environment only.

The following is a list of functions performed by the PFTreeInfo tool.

  • Specifies a profile to log on to.
  • Uses the tree control to step down the folder's subfolders and choose any folder to be counted.
  • Counts all the messages, or subfolders, or both.
  • Starts the counting process when you choose the Start Count button.
  • Indicates that the counting process is running by showing a ticking clock.
  • Updates the counting result every half-second.
  • Stops the counting process at any time when you choose the Stop Count button.
  • Adds all the folders that you access to the list box.
