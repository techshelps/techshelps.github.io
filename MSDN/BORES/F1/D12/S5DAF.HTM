Chapter 3 Optimizing Performance

Your system's performance is also affected by the server's software and hardware configurations. You can minimize performance bottlenecks by balancing server hardware resources to support the expected load and adjusting software settings. This chapter suggests ways you can maximize the number of users supported while keeping response times low. Keep in mind that if you are supporting a live user population, you must be more cautious with the changes you make.

For information about troubleshooting performance, see Chapter 7, "Troubleshooting Microsoft Exchange Server." For more information about optimizing performance, see the Microsoft Exchange Server Concepts and Planning Guide.
