Listuser Tool

The Listuser tool displays a list of all users on a given postoffice, including their associated eight-digit user ID numbers (these numbers are used by the Microsoft Mail system to identify each individual user). Listuser works with Microsoft Mail version 2.1 and later postoffices. This tool also displays the size of the .mmf files on the server, and can search for a specific name on the postoffice.

When to Use Listuser

Use this program whenever you need to know the ID number associated with any of your users.

Side Effects of Using Listuser

There are no side effects when using Listuser, because it is a read-only tool.

To run Listuser

  • Use the following command-line syntax format:

    listuser <admin> -p<password> [-d<drive>] [-f] [-x] [-n<name>] [-s] [-?]

The following table lists the command-line options available in Listuser, and their descriptions:

Option

Description

<drive>

The drive where the Microsoft Mail database is located (the default drive is M).

-f

Displays the user's full name in addition to the alias (optional).

-x

Displays the delete status of the users and any deleted users (optional).

-n<name>

Displays only those names that match the <name> parameter. This option by itself searches on the alias and the hexademical ID. If the option is specified, the full name field is searched.

-s

Displays the size of any .mmf files that are on the server.

-?

Displays the Help window.


To determine a user's ID number

You can also determine a user's ID number by performing the following steps:

Run Microsoft Mail and log on as the user.

From the Options menu, choose Folders, and then choose Private Storage.

Select Local-Storage.

Type c:\ (or whichever local drive is available).

Quit Microsoft Mail.

At the MS-DOS prompt, type Dir C:\*.idx. The file name displayed is the user's ID number.

To return the folders back to the server, restart Microsoft Mail.

From the Options menu, choose Folders, and then choose Private Storage.

Select Server.
