Dumpkey Tool

A user's online calendar file is not named with the same eight-digit ID number as the user's mail message file, nor is it named with the user's name (for security reasons). You can use the Dumpkey tool to find out which online calendar files belong to which users.

To run Dumpkey

  • Copy the Dumpkey.exe file to the Cal subdirectory of the postoffice by typing the following at the command prompt (this example assumes that the Microsoft Mail database is located on drive M):

    copy dumpkey.exe m:\cal\dumpkey.exe

To use this tool correctly, follow the command-line syntax format shown below:

dumpkey schedule.key

When to Use Dumpkey

You can use this program if you want to find the owner of a particular calendar (.cal) file on the server.

Side Effects of Using Dumpkey

There are no side effects to using the Dumpkey tool because it is a read-only file.
