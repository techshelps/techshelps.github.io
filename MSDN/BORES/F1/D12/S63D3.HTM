Installing AutoFill

Use the following procedure to install Autofill.

To install Autofill

From the Exchange\<operating system\platform>\Client\Autofill directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, run Setup by typing:

setup [/Ppath /Llocation]

where path is the destination path for the installation (the default is C:\AutoFill32), and location is the path to Autofill32.dll (the default is the current directory). (For command-line Help, type Setup /?).

For example, you can use the following syntax to install AutoFill into the C:\Exchange\AutoFill directory. and copy AutoFill32.dll from E:\Tools\Autofill.

setup /PC:\Exchange\AutoFill /LE:\Tools\Autofill

Close and restart the Microsoft Exchange Client.
