Moving Details Templates

Use the procedure below to move details templates.

To move details templates

On Server A, create a directory called Templates.

From the Support\Samples\Tpl directory on the Microsoft Exchange Server compact disc, copy Tpl.ini and Tpl.csv to your Templates directory on Server A.

Remove the Read-Only attributes from these files.

Edit the Tpl.ini file. Under the [Import] and [Export] sections, modify the Basepoint entry by replacing the words "ORG" and "SITE" with your organization and site names. Save the file.

At the command prompt, switch to the Templates directory and type the following:

C:\Exchsrvr\BIN\ADMIN /e tpl.csv /d Server A /o tpl.ini

Copy the Templates directory from Server A to Server B.

On Server B, at the command prompt, type the following:

C:\Exchsrvr\BIN\ADMIN /i tpl.csv /d Server B /o tpl.ini.
