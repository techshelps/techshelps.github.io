Using the Proxy Conversion Tool

Run Import.exe against an existing Microsoft Mail postoffice.

Type the following:

PERL <input_file > <output_file>

where input_file is the file created in step 1, and output_file is the .csv file for the conversions.

Using the Microsoft Exchange Server Administrator program, import the .csv file to add the addresses as custom recipients to a Microsoft Exchange Server Recipient container.
