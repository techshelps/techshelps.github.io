Moving the KM Server

The following steps are applicable only if you already have the KM server running on Server A and security is enabled for the Microsoft Exchange mailboxes on Server A.

To move the KM server

Remove the KM server from Server A.

In Control Panel, double-click Services, and then stop the Key Management Server service on Server A.

In the Exchkm directory on the Microsoft Exchange Server compact disc, run Setup.exe.

Install the KM server on Server B. Select the option to create a startup disk.

After the installation is completed, stop the Key Management Server service on Server B.

From the KM Server Startup disk for Server A, copy the Kmspwd.ini file onto another disk for Server B. Label this disk "KM Server Startup Disk: Server B."

From the backup of the Security directory on Server A, copy the Security\Mgrent directory to the Security directory of Server B.

Insert the new KM Server Startup disk for Server B (labeled "KM Server Startup Disk: Server B") into the disk drive on Server B.

On Server B, start the Key Management Server service.

As long as the .epf files are still on your users' computers and their profiles have not changed, your users can read all previously encrypted mail.
