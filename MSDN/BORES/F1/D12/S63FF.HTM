Installing GSO

Upon installation, the Gsoinst.exe file creates the appropriate registry entry in the list of Microsoft Exchange Client extensions.

To install the GSO tool

From the Exchange\<operating system\platform>\Client\GloblSnd subdirectory on the Microsoft BackOffice Resource Kit, Part II, compact disc, copy Gso.dll and Gsoinst.exe to your computer.

At the command prompt, switch to the directory where you copied the files.

Type gsoinst <Gso.dll pathname>.

For example, if you copied the files to your C:\Exchange directory, switch to this directory and then type:

gsoinst c:\exchange\gso.dll

If you are running the Microsoft Exchange Client or Schedule+, quit these programs.

Start (or restart) the Microsoft Exchange Client to load Gso.dll.

To manually load Gso.dll rather than running Gsoinst.exe

Create a new string value under: HKEY_LOCAL_MACHINE\Software\Microsoft\Exchange\Client\Extensions and give it the value:

4.0;<path>;1;00000100000100;1000010;MSEMS

where <path> is the full path name to Gso.dll.
