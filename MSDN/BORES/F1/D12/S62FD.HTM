Backing Up Data

Before moving Server A to Server B, follow the steps below to back up your data on Server A.

To back up Microsoft Exchange Server data

Stop all Microsoft Exchange Server services on Server A and back up the entire Exchsrv subdirectory. If the logs and database files are on different drives, back up these directories as well.

If the KM server is installed, back up the Security directory.

From the KM Server Startup disk, back up the Kmpswd.ini file.

Start all Microsoft Exchange services on Server A.
