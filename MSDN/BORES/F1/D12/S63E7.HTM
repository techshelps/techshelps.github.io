Using CleanSweep

To run the CleanSweep tool

At the command prompt, type RegEdit.

Beneath the HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Exchange\
Client\Extensions registry key, add the following key value: Clnswp.

Note that the value for this key value should be: 4.0;c:\windows\system32\
clnswp32.dll;2 if you copied Clnswp32.dll to the proper directory (c:\Windows\System32) when installing the tool.

To delete rules, permissions, forms, views, or rules reply templates

In the Microsoft Exchange Client Viewer, select the folder or message store object that contains the items you want to delete.

From the Tools menu, choose CleanSweep.

Select the items you want to delete, and then choose OK.
