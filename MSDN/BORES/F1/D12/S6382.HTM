File Dump

The File Dump utility generates formatted output of various database file types. The correct syntax for using this utility is edbutil /m[mode-modifier]
<file name>.

Parameters and Switch Options

Description

[mode-modifier]

An optional letter designating the type of file dump to perform. Valid values are: h (dump database header) and k (dump checkpoint file).

<file name>

The name of file to dump, where the type of the specified file should match the dump type being requested.
