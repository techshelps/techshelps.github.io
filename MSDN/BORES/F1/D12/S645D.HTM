Installing Profgen

To install the Profgen tool

  • From the Exchange\<operating system\platform>\Admin\Profgen subdirectory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy Newprof.exe to your computer.

    Note Before installing Profgen, it is recommended that you read the installation section below that applies to your operating system ("Installing Profgen for Microsoft Windows NT," "Installing Profgen for Windows 95," or "Installing Profgen for Windows 3.x").
