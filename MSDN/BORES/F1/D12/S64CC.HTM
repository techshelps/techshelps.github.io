1.0 Setup for the Resource Kit Software

In the root directory of the Microsoft BackOffice Resource Kit: Part Two CD-ROM, there is the overall README.WRI file that you are currently reading and a SETUP.EXE file for installing most of the Resource Kit software. Utilities that are not installed by the main Setup program have their own installation program in their own directory or subdirectory.

The following is a list of the main directories on the Microsoft BackOffice Resource Kit: Part Two CD-ROM. For a list of all the first- and second-level directories on the CD-ROM, see Section 6.0, "Directory Structure of the Microsoft BackOffice Resource Kit: Part Two CD-ROM."

\Alpha

\apps

\Exchange

\Help

\I386

\Libs

\Sms

\Sna

\Sql

\Win95

Important Log on as a member of the Administrators group before starting Setup because many of the tools require administrator privilages.

Finally, these tools have not been localized: they have been written and tested in English only. Using these tools with a different language version of Microsoft® Exchange Server, Microsoft Exchange Client, or Systems Management Server may produce unpredictable results.
