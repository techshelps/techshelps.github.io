Natural Language Support Configuration Tool

The Natural Language Support (NLS) Configuration tool automates the process of configuring a Microsoft Exchange Server computer to support a variety of languages. On a practical basis, it enables a Microsoft Exchange Server computer to support Microsoft Exchange Clients running on operating systems configured for different languages. The tool automates this process by installing code-page files for languages not supported by default in the Windows NT language list.

The NLS tool is also useful for those who want to manually install individual code pages on an as-needed basis for Windows, Microsoft Exchange-DOS and Apple Macintosh operating environments.
