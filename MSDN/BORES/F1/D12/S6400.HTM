Using GSO

To change the Deferred Send/Internet Attachment encoding settings

In the Microsoft Exchange Client Viewer, from the Tools menu, select Options.

Select the Global Send tab.

You can now set the time for the Deferred Send option, as well as the type of encoding for attachments intended for the Internet.

Note If you select the Check for existing settings before putting global settings on message option, then Gso.dll will determine whether you have already set Deferred Send or Internet attachment encoding options for the message. If so, Gso.dll will not replace the old settings with the new global settings.
