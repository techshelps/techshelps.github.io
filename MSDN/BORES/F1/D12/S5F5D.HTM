PFAdmin Tools

The PFAdmin tools have been expanded to include reporting capabilities for interpersonal message (IPM) types within the entire public folder tree, single folder, or subtree. The report is provided as a .csv file that includes headings to describe the IPM type.

PFAdmin is actually comprised of three separate tools:

  • Public Folder ACL Replication tool
  • Re-Home Public Folder tool
  • Public Folder Item Type tool
