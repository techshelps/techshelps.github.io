Installing the EFD Handcode Agent

To install the EFD Handcode Agent

From the Exchange\<operating system\platform>\Developr\EFDhandc subdirectory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy Efdcoder.exe to your computer.

From the Start menu, choose Run, and then choose Browse to search for the folder or subdirectory where you copied the file.

Select the EFD handcode folder, and then select Setup.exe.

When you run Setup.exe, the Visual Basic 4.0 32-bit run times are automatically loaded.
