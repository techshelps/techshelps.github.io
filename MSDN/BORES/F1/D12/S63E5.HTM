Using the Bulk Advanced Security Tool

Use the following procedures to decrypt and encrypt messages.

To decrypt messages in a folder

In the Exchange\<operating system\platform>\Security\Sectool directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, double-click Sectool.exe.

From the Exchange Security menu, choose Unsecure Folder.

Type your security password, and then choose OK.

Select the message store containing the messages you want to decrypt, and then choose OK.

Select the folder containing the messages you want to decrypt, and then choose Start Here. To select a subfolder, choose Open Folder.

Select Process Subfolders to decrypt messages in the selected folder and its subfolders, and then choose OK.

To encrypt messages in a folder

In the Exchange\<operating system\platform>\Security\Sectool directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, double-click Sectool.exe.

From the Exchange Security menu, choose Secure Folder.

Type your security password, and then choose OK.

Select the message store containing the messages you want to encrypt, and then choose OK.

Select the folder containing the messages you want to encrypt, and then choose Start Here. To select a subfolder, choose Open Folder. If you are moving messages between sites or recipient containers, select the folder containing the messages you decrypted before the messages were moved.

Select the options that you want to use for encrypting messages, and then choose OK.
