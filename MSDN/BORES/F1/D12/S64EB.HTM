2.3 Microsoft SQL Server Tool Notes

The Microsoft SQL Server Resource Guide and Help file are up-to-date. No additional changes are documented in these release notes.

2.3.1 Arcadia Bay Demonstration

The Arcadia Bay demonstration on the BackOffice Resource Kit CD-ROM is a self-extracting ZIP file. The Arcadia Bay files will be extracted into the Bork\Sql\Arcadia directory when the ZIP file is opened.

2.3.2 SQL Trend Analyzer

The SQL Trend Analyzer tool on the BackOffice Resource Kit CD-ROM is a self-extracting ZIP file. The SQL Trend Analyzer files will be extracted into the Bork\Sql\SQLTrend directory when the ZIP file is opened.
