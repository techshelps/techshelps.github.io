1.3 Additional Setup Information

This section contains information on installing applications that are not installed by the main Resource Kit Setup program, or that require additional configuration. Instructions for installing many of the tools are in the Exchtool.hlp and Smstools.hlp Help files.

1.3.1 Additional Setup Information on Systems Management Server Tools

Some of the Systems Management Server tools access DLL files in a directory that is not in your path. These tools must be run from the directory in which the DLL files are located. In most cases, the DLL files are located in Sms\Site.srv\<platform>.bin, where <platform> is I386 or Alpha. The tools that require this are documented in Smstools.hlp.

For information on the NetBoot floppy, installing Performance Monitor counters, or installing Visual Test 4 and upgrading to Systems Management Server 1.2, see the Smstools.hlp Help file.

To set up Pollinv.exe:

Start Windows NT and insert the Microsoft BackOffice Resource Kit: Part Two CD-ROM in the CD-ROM drive.

In Program Manager, choose Run from the File menu, or choose the Start button, and then choose Run.

Type cd_driveletter:\SMS\Ntinvcol\platform\instpoll.exe, and then choose OK.

Follow the instructions on the screen to complete Setup.

Manually start the service by typing the following at the command line prompt: net start sms_nt_inventory_file_collector

1.3.2 Additional Setup Information on Microsoft Exchange Server Tools

This section contains information on installing applications that are not set up by the main Resource Kit Setup program, or that can be installed using several installation procedures, or that require additional configuration. Instructions for installing many of the tools are in the Exchtool.hlp. Help file. Some instructions in the Help file apply only to Windows 95. Additional setup information for these tools is included here.

To install Crystal Reports for Exchange:

Start Windows 95 or Windows NT and insert the Microsoft BackOffice Resource Kit: Part Two CD-ROM in the CD-ROM drive.

In Program Manager, choose Run from the File menu.

Type cd_driveletter: msetup, then choose OK.

From the Microsoft BackOffice Resource Kit screen, click More MS Exchange Tools.

Click the tool you wish to install.

Follow the instructions on the screen to complete Setup.

Or use the Setup program in \Exchange\<operating system\platform>\Admin\Crystal to set up this program.

To install the Schedule+ Address Book Assistant on Non-Windows 95 platforms:

Copy Schedaba.exe from cd_driveletter:\Exchange\<operating system\platform>\Client\Schedaba to your Windows directory.

Copy all of the dynamic-link libraries (DLLs) from cd_driveletter:\Exchange\<operating system\platform>\Client\Schedaba to your Windows\System directory.

Run Schedaba.reg to add an entry for the tool to the registry.

To install the Simple Mail Handler Tool:

For all operating systems except Windows NT Workstation, copy Mergeini.exe and the DLL for your operating system to the \Windows\System directory. For Windows NT Workstation, copy Mergeini.exe and the DLL to the system32 directory.

Run the Mergeini.exe tool to add the Simple Mail Handler to the list of services available.

In Control Panel, choose Mail and Fax.

Select the Services tab, and then choose Add.

Select Sample Mail Handler, and then choose OK.
