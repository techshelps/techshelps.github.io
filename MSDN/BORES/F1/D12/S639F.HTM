Summary

Microsoft Exchange Server provides an application design environment that is ideal for rapidly changing business requirements. It enables all users, from novices to skilled programmers, to create custom applications to improve efficiency and make it easier to communicate. It provides a scalable design environment that makes it easy to modify those applications quickly and cost effectively. It also offers reliable, versatile options for connecting to the Internet and X.400 systems to distribute those applications to users virtually anywhere.
