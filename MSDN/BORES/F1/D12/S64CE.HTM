1.2 Setting Up the Resource Kit Tools

The Setup program for the Microsoft BackOffice Resource Kit: Part Two requires Microsoft Windows NT® version 3.51 or later. In Windows NT 4.0 or later, the Setup program autoruns.

To set up the Resource Kit tools on Windows NT 3.51:

Start Windows NT and insert the Microsoft BackOffice Resource Kit: Part Two CD-ROM in the CD-ROM drive.

In Program Manager, choose Run from the File menu.

Type cd_driveletter: setup, then choose OK.

The Microsoft BackOffice Resource Kit: Part Two Setup program displays a License Agreement dialog box. If you agree to accept the terms of the license agreement, Setup continues.

Follow the instructions on the screen to complete Setup.

There are two options for installing the Microsoft BackOffice Resource Kit: Part Two on your hard disk: Typical Setup and Complete\Custom Setup. Typical Setup installs all the Resource Kit files onto your hard disk and requires a maximum of 27 MB of free space. Use Complete\Custom Setup to choose specific components and minimize the amount of space used on your hard disk.

Setup creates program groups for the BackOffice Resource Kit with icons for some of the tools. For more information about these tools, choose the BackOffice Tools Help (Borktool.hlp) icon in the BackOffice ResKit program group.

Setup also adds the directory name you choose for installing the tools to the path System environment variable. (If you want to change your path double-click the System icon in Control Panel after the Resource Kit installation is complete.)

Setup installs the Microsoft Systems Management Server Client applications in a different directory.
