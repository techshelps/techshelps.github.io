1.1 Hardware-Platform and Other Issues

The following is a list of issues relating to various Resource Kit utilities that are included in this release of the BackOffice Resource Kit, Part Two. These are utilities that were not available at the time of release, or that do not work on one hardware platform or another. Hopefully, most of these issues will be resolved by the next release.

Attmwiz Tool - Not available.

Dupproxy.exe - Works only on Windows NT.

Exchbkup.exe - Not available.

PFReports (various .rpt files for Crystal Reports) - Not available.

Phone.exe - Works only on Windows NT.

Storstat.exe - Not available
