Saving Public Folders

Follow the procedure below to save your public folders.

To save public folders

Log on to the Microsoft Exchange Client by using an account on Server A that has at least Read access to all public folders. If necessary, use the Administrator program to grant this access permission on public folders.

Add "Personal Folders" to the current profile using PublicFoldersOnServerA.pst as the file name.

Create a new personal folder called "Public Folders on Server A."

Select all public folders and copy them to the Public Folders on Server A folder that you just created. This should also copy any forms associated with the public folders.
