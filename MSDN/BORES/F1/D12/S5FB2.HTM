SwitchForms Tool for Microsoft Outlook

With the SwitchForms tool, you can easily switch between Microsoft Exchange Client and Microsoft Outlook. Although you can view the same e-mail folders in both applications, each application installs its own forms for composing and reading messages. If you need to run both Microsoft Exchange Client and Outlook on the same system, use SwitchForms to reinstall the correct forms.

The SwitchForms tool is intended for:

  • Helpdesk personnel who support and run both applications.
  • Microsoft Exchange Server public folder application designers who need to test their applications under both Microsoft Exchange Client and Outlook.
  • Systems professionals who run Microsoft Exchange Client for production use but need to evaluate Outlook.
