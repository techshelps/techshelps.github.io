Installing Preview Pane Extension

To make Preview Pane Extension available to users, you need to copy the appropriate files from the Microsoft BackOffice Resource Kit, Part Two compact disc to a network share, and then install them on a client computer running Windows NT Workstation or Windows 95.

To install Preview Pane Extension to a network share

  • From the Exchange\<operating system\platform>Client\Tripane directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy all files to the server share where you want users to find them.

To install Preview Pane Extension on a client computer

Connect to the network location where Preview Pane Extension is available for setup.

Double-click Setup.exe.
