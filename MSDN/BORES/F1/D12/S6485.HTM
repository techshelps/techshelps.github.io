Installing Simple Mail Handler

Use the following procedure to install the Simple Mail Handler tool.

To install Simple Mail Handler

For all operating systems except Windows NT Workstation, from the Exchange\<operating system\platform>\Client\SMH directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy Mergeini.exe and the .dll for your operating system to the Windows\System directory on your computer.

For Windows NT Workstation, copy Mergeini.exe and the appropriate .dll to the System32 directory on your computer.

Run Mergeini.exe to add the Simple Mail Handler to the list of services available.

In Control Panel, double-click the Mail and Fax icon.

Select the Services tab, and then choose Add.

Select Sample Mail Handler, and then choose OK.
