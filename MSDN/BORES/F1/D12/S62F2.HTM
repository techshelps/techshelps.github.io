Recreating DLs on the New Server

You only need to re-create DLs if you have manually moved the mailboxes to Server B. This is because the process of manually moving the mailboxes involved deleting the users on Server A. When you did this, the users were removed from the DLs. This is also the reason why all the DLs were exported.

To re-create the DLs

In the Administrator window, choose Directory Import from the Tools menu.

Select the Dl.csv files, and then choose Import.
