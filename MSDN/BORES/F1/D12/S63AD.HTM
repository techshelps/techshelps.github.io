Deleting Menus

[Deleted Menus] 

;Examples 
DeleteHelp=2.0;help 
DeleteUndo=2.0;edit,undo 
tag=version;menu path 
tag Identifies the command to someone reading the INI file, 
but serves no other purpose. 
version Identifies the version of Schedule+ with which the 
command is compatible; 2.0 is the current version. 
menu path Path to an existing menu to be removed. The name is 
case insensitive, and ampersands are ignored. Commas 
are used to specify the path to the menu. To 
specify an absolute menu position use '#' followed by 
the number for the position.
