Installing Profgen for Windows 95

To install Profgen for Windows 95

Copy Profgen.exe to C:\Exchange.

Copy the Profgen.ini file to C:\Windows.

In the Profgen.ini file, edit the PathToExeFile and PathToPRFFile entries to point to the Newprof.exe and .prf files, respectively.

Normally these entries are: PathToExeFile=c:\exchange\newprof.exe PathToPRFFile=H:\Win95.prf where h:\ is the user's home directory on the server.

Edit the Load= line in the Win.ini file as follows. Load=c:\exchange\profgen.exe
