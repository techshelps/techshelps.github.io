EFD Handcode Agent

The EFD Handcode Agent enables you to paste or copy your handcode text when adding it to your EFD project script file. By using simple scripting commands, you can play the script, and your handcode is automatically inserted into your EFD project source-code files.

You can also customize your EFD project with Microsoft Visual Basic® code and then return to EFD as needed. When you return to EFD, the EFD changes are merged with your Visual Basic hand-coded changes automatically.
