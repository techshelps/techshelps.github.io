Installing DL Import

To install the DLImport Tool

  • From the Exchange\<operating system\platform>\Admin\DLImport directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy all the files to your computer.
  • At the command prompt, type: DLImport -fi

Installing and Running the Metrics Service

The DLImport -fi command instructs the Metrics service to install itself as a service on the system and set the account value to what you specify in the command. Running this command forces the new registry defaults into the local registry.

To start the Metrics service

  • At the command prompt, type:

    Metrics -fi \u<domain>\<user_name>\p<user_password>
