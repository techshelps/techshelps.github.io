Installing the Management Chain List Tool

There is only one step to installing the Management Chain List tool. However, before you install the tool, make sure that you:

  • Have already created a default Microsoft Exchange profile; or
  • Create a new profile by using a name of your choice and specify it by using the /p command-line switch.

To install the Management Chain List tool

  • From the Exchange\<operating system\platform>\Admin\MgmtList subdirectory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy the Mgmnt.exe file to a directory on your computer.
