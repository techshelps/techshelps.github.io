Installing Eximwiz

Installing the Eximwiz tool is a simple process. After installation, the Eximwiz tool is ready to use as a command-line utility.

To install the Eximwiz tool

From the Exchange\<operating system>\Admin\Eximwiz2 subdirectory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy all files to your computer.

Copy Msmapi32.ocx to your system directory (C:\Windows\System on Windows 95 or C:\Windows\System32 on Windows NT).

At the command prompt, switch to the subdirectory that contains Eximwiz, then type Setup.exe.
