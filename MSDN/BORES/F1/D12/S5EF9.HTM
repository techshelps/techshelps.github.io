Mailbox Cleanup Agent

You can use the Mailbox Cleanup Agent to control client use of disk space on Microsoft Exchange Server computers. The Mailbox Cleanup Agent gives administrators the flexibility to delete and move outdated messages in users' mailboxes.

The Mailbox Cleanup Agent consists of a Windows NT Server service, a configuration object in the Microsoft Exchange Server directory, and a mailbox agent. Configuration information is contained in the Mailbox Cleanup Agent directory object in the Add Ins container. The mailbox agent originates the status reports that are sent to users after their mailbox has been cleaned. One instance of the Mailbox Cleanup Agent controls an entire site.
