Buddy Dropper Tool

Buddy Dropper (Dropper) enables you to automate files, the registry, applications, and other tasks for an array of Microsoft Windows 95 and Windows NT servers. It helps manage large mixed-platform and mixed-language environments.

When you use the Dropper tool, you can specify a hierarchy of server nodes. A node can be an individual server or a server group. Because each server node contains a hierarchy of actions and Dropper variables, actions are configured by the script creator to perform tasks, and variables enable data to be shared from server to server.
