Deleting Mailboxes from the Original Server

Follow the procedure below to delete mailboxes from Server A.

To delete mailboxes

In the Administrator window for Server A, choose Server Recipients.

From the menu bar, choose View, and then choose Mailboxes.

Choose Columns, and then add Home Server to the Show the following columns list.

Move the position of the Home Server entry so that it is the second entry in the list.

Select each container that includes Microsoft Exchange mailboxes.

Delete all users who have Server A as their home server.

Recreating Mailboxes on the New Server

Before recreating mailboxes on Server B, make sure that the Microsoft Exchange Directory service is running on Server B.

To re-create mailboxes

In the Administrator window for Server B, select Server B.

In the right pane, double-click Directory Service.

Choose Update Now.

Select Refresh all items in the directory, and then choose OK.

This causes all changes made to the directory on Server A to be replicated to Server B. Note that after the replication is completed, the mailboxes that were deleted from Server A should not be displayed on Server B.

In the Administrator window, choose Directory Import from the Tools menu.

Choose Import File, and then select the Mailbox-ServerB.csv file.

Make sure that the server displayed in the MS Exchange Server box is
Server B.

Under Account creation, select Use select container if it is not specified in the file. Do not select the Create Windows NT account check box.

Choose Import.

Repeat this step for each file that contains mailbox information. All mailboxes that were originally on Server A should now be re-created on Server B.
