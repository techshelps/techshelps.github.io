Installing Mailbox Statistics

You install Mailbox Statistics by copying its files from the Microsoft BackOffice Resource Kit, Part Two compact disc to a Microsoft Exchange Server computer.

To install Mailbox Statistics

  • From the Exchange\<operating system\platform>\Admin\Storstat directory, copy all the files in the directory to your Microsoft Exchange Server computer.

This directory contains StorStat.exe, which can be used by Windows 95 and Windows NT Workstation users.
