AT & T Mail Migration Wizard

The AT&T Mail Migration Wizard (Attmwiz.exe) migrates Microsoft Access and PMX mail messages into a local Microsoft Exchange Client personal folder or the Microsoft Exchange Server-based mailbox. This wizard supports AT&T Mail Access Plus version 2.7 and earlier, AT&T PMX Starmail, AT&T PMX PC, and AT&T PMX Term. The mail is still accessible in the native environment.

To run the AT & T Mail Migration Wizard (ATTMWIZ), you must have the Microsoft Exchange Client installed, plus one of the following operating systems:

  • Microsoft Windows version 3.1
  • Windows for Workgroups 3.11
  • Windows 95
  • Windows NT 3.51 or later
