Installing RPC Counter

Before you can use the RPC counter, you need to connect with the Emsmdb(32).dll file, which contains the necessary entry points for generating the RPC data. For details on RPC counter entry points, see "Understanding RPC Counter Entry Points." The Emsmdb(32).dll file is part of the Microsoft Exchange Client and Microsoft Exchange Server installation and is located in the Windows\System(32) directory.

You can access RPC counter data through several sample applications, depending on your computer's operating system.

To install the RPC counter sample applications

  • From the Exchange\<operating system\platform>\Developr\RPCCount directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy the appropriate files (listed below) to your current working directory or somewhere in the system path.

    For Windows NT Workstation and Windows 95, copy Rpccnt.exe.

    For Windows 3.1 and Windows for Workgroups, copy Rpccnt16.exe.
