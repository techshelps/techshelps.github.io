Installing the Microsoft Mail SMTP Gateway Proxy Migration Tool

The Microsoft Mail SMTP Gateway Proxy Migration tool is included on the Microsoft BackOffice Resource Kit, Part Two compact disc.

To install the Microsoft Mail SMTP Gateway Proxy Migration tool

  • From the Exchange\<operating system\platform>\Migrate\Migsmtp directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy Migsmtp.exe to a Microsoft Exchange Server computer.
