Using Write or WordPad to View This Document

You can view this document using either Write or WordPad.

If you enlarge the Write or WordPad window to its maximum size, this document will be easier to read. To do so, click the Maximize button in the upper-right corner of the window. Or open the Control menu in the upper-left corner of the Write window (press ALT+SPACEBAR), and then choose the Maximize command.

To move through the document, press PAGE UP or PAGE DOWN or click the arrows at the top and bottom of the scroll bar along the right side of the Write or WordPad window.

To print the document, choose the Print command from the File menu.

For Help on using Write or WordPad, press F1.

To read other online documents, choose the Open command from the File menu.
