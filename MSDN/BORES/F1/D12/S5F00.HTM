Using the Mailbox Migration Tool

Before running the Mailbox Migration tool, you must have mailbox owner permissions on both source and destination mailboxes. If you are migrating a large number of users, the best method is to create a Windows NT account for this purpose. Then grant mailbox ownership permissions to the recipient's container on the source and destination servers.

Setting Mailbox Permissions

Follow these steps to set permissions on the mailbox:

From the Microsoft Exchange Administrator window, choose Tools from the toolbar, then select Options.

Select Permissions, then select Show Permissions Page for All Objects and Display Rights for Roles on Permissions Page.
