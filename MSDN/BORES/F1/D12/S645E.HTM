Installing Profgen for Microsoft Windows NT

To install Profgen for Windows NT

  • Add the following line to the Windows NT domain logon script:

    \\Server Name\Share Name\Profgen.exe C:\EXCHANGE\Newprof.exe
    -P H:\Winnt.prf

    where H:\ is the user's home directory.

    (This assumes that Newprof.exe is present in the C:\Exchange directory. You can also find Profgen.exe on your local hard disk instead of on the server.)
