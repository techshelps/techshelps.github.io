Installing the UNIX Mail Source Extractor

To install the UNIX Mail Source Extractor tool

From the Exchange\Samples\UnixSE subdirectory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy the UNIX source code to your computer.

Because the UNIX Mail Source Extractor is provided as source code, a C compiler on the host computer, or a compiler capable of compiling to the host computer is required to produce the migrate file. For more information, see the compiler documentation for your version of UNIX.
