Bulk Advanced Security Tool

The Bulk Advanced Security tool (Sectool.exe) encrypts and decrypts messages in folders and subfolders in a user's private information store or personal folder file (.pst file). This tool is most useful for moving messages in security-enabled mailboxes between sites and between recipient containers. It can also be used to secure messages in confidential folders. You can do the following with the Bulk Advanced Security tool:

  • Decrypt all messages in a folder and its subfolders.
  • Encrypt all messages in a folder and its subfolders.
  • Encrypt only messages that were previously decrypted with this tool.
