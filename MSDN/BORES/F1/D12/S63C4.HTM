Using ASpiriN

To run ASpiriN

From the Start menu, choose Run, and then choose Browse to search for the folder or subdirectory where you copied the file.

Select the Aspirin folder, and then select Aspirin.exe.

To open a hexadecimal or binary-format ASN.1 file

From the File menu, choose Open.

Select Hex if the file contains ASN.1 data encoded as hexadecimal ASCII.

Select Binary if the file contains ASN.1 data encoded as raw binary.

To produce ASN.1 data files from a Microsoft Exchange Server MTA log file

  • The Microsoft Exchange MTA logs messages and other information in the Exchsrvr\Mtadata directory as BF*.log files. These files contain, among other pieces of data, ASN.1 hexadecimal dumps of messages passing through the MTA. These ASN.1 blocks can be viewed by using ASpiriN if they are parsed out of a BF*.log file. To view these ASN.1 blocks, using the following procedures:

    From the File menu, choose Parse MTA Log.

    Select a log file, such as Bfo.log, and then choose OK.

After you follow the steps above, ASpiriN produces one hex file in the same directory as the log file for each block in the log file. Some, but not all blocks, are ASN.1 data. You can view the ASN.1 hex files by following the procedure outlined earlier.
