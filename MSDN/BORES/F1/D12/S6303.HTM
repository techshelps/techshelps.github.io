Installing the KM Server

Follow the procedure below to install the KM server:

To install the KM server

If you were using the KM server on Server A, install the KM server on Server B. Select the option to create a startup disk. After the installation is completed, stop the Key Management service on Server B.

From the KM Server Startup disk for Server A, copy the Kmspwd.ini file to another disk for Server B. Label this disk "KM Server Startup Disk: Server B."

On the new server, rename the Security\Mgrent subdirectory to Mgrent.original.

From the backup of the Security directory on Server A, copy the Security\Mgrent directory to the Security directory of Server B.

Insert the new KM Server Startup disk for Server B (labeled "KM Server Startup Disk: Server B") into the disk drive on Server B.

Start the Key Management Server service on Server B.
