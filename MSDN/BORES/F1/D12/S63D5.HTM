Installing AutoSig

Installing the AutoSig tool is a simple process. After installation, you can use the auto signature, because the AutoSig tool automatically adds entries to your registry keys to enable this feature.

To install AutoSig

From the Exchange\Common\Autosig directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy autosig.reg to your Windows\System directory.

From the Start menu, choose Run.

Select Autosig.reg from your Windows\System directory.

Under the [HKEY_CURRENT_USER\Software\Microsoft\Autosig]registry key, AutoSig adds the following new entry: "Path"="C:\\Windows"

Under the [HKEY_CURRENT_USER\Software\Microsoft\Windows\
CurrentVersion\ProfileReconciliation\Autosig] registry key, Autosig adds the following new entries:

"CentralFile"="Config"

"DefaultDir"="*windir"

"LocalFile"="" (blank)

"MustBeRelative"=dword:00000000

"Name"="Autotext.sig"

"RegKey"="Software\\Microsoft\\Autosig"

"RegValue"="Path"
