Backup Tool for Microsoft Exchange

The Backup tool for Microsoft Exchange provides an automated way to report whether a backup was successful or if it failed.

To ensure that all of your Microsoft Exchange data is backed up, you should back up the following files:

  • Microsoft Exchange Server Directory files (Dir.edb).
  • Private Microsoft Exchange Database files (Priv.edb).
  • Public Microsoft Exchange Database files (Pub.edb).

This tool prevents you from inadvertently ignoring a failed backup.
