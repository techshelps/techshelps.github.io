AutoFill Tool

The AutoFill Tool assists in resolving the display names you type in the To and Cc boxes of a mail message in the Microsoft Exchange Client.

When you check names (from the Tools menu, choose Check Names), names that are resolved successfully are added to the AutoFill list. This creates a list of names that you send mail to. AutoFill uses the list to guess the name after a few characters have been typed, and enters the remainder of the recipient's name automatically.
