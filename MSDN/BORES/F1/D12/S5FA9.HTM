Installing the Server Space Client Extension Tool

There are two types of installation methods for the Server Space Client Extension tool:

  • Installation for Microsoft Windows clients
  • Installation for Windows NT clients

Installation for Windows Clients

To install the Server Space Client Extension tool on Microsoft Windows clients, follow the procedure below.

From the Exchange\<operating system\platform>\Client\SrvSpace directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy Spcext32.dll to the Exchange directory on your computer.

Start Notepad and open the Windows\Exchng.ini file.

Add the following line under the [Extensions] section:

Server Space Extensions=4.0;c:\exchange\spcext16.dll;2;01010000000000

Save the file and close Notepad.

Restart the Microsoft Exchange Client.

Installation for Windows NT Clients

To install the Server Space Client Extension tool on Windows NT clients, follow the procedure below.

From the Exchange \<operating system\platform>\Client\Srvspace directory on the Microsoft BackOffice Resource Kit, Part Two compact disc, copy Spcext32.dll to the Exchange directory on your computer.

At the command prompt, switch to the Winnt35\System32 directory.

Type Regedt32, and then press Enter.

Switch to the HKEY_LOCAL_MACHINE window, and then locate this section: Software, Microsoft, Exchange, Client, Extensions.

Double-click Edit, and then choose Add Value.

For the Value Name, type Server Space, and then choose REG_SZ for Data Type.

Choose OK.

Paste the following into the String field: 4.0;c:\exchange\Spcext32.dll;1;01010000000000

Choose OK to add the entry, and then close the Registry Editor.

Restart the Microsoft Exchange Client.
