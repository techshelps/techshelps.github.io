Message 41

NetView Management Connection information not found in value

EXPLANATION: In the configuration file, there was no connection specified for sending and receiving NetView management data.

The connection to use for NetView management data can be specified using SNA Server Manager. To specify the connection, from the Options menu, choose NetView, and then, in the NetView Options dialog box, select a connection name. For more information, see the Microsoft SNA Server Administration Guide.

ACTION: This message probably indicates an internal error in SNA Server Manager. Provide network support personnel with a copy of the SNA Server configuration file. For information about the configuration file, see the Microsoft SNA Server Administration Guide.
