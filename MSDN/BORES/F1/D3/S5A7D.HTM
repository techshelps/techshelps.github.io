Message 618

Failed to access configuration file value, rc = value

EXPLANATION: The process cannot read the configuration file. The file name shown should be COM.CFG. If anything else appears here, it indicates an internal error. Normally the Windows NT return code is shown. However, three additional values can also be returned here:

612 No configuration file is available at present.

616 Failed to obtain name of configuration file.

618 Invalid or corrupted configuration file.

ACTION: Contact support services at your site. For return codes 616 and 618, check that the file COM.CFG exists in the SYSTEM\CONFIG subdirectory of the SNA Server root directory, and that it is a valid configuration file (try opening it using SNA Server Manager). If you still have a problem, contact support services at your site.
