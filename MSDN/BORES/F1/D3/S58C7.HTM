TS Profile 7

TS Profile 7 is supported on PLU-SLU sessions using LU type 6.2. This profile specifies the following session rules:

  • PLU-SLU and SLU-PLU normal flows are paced.
  • Sequence numbers are used on the normal flows.
  • BIND and UNBIND are supported.

The TS Usage subfields defining the options for this profile are as follows:

  • Pacing counts.
  • Maximum RU sizes on the normal flows.
