Message 333

CLEAR packet received from network on channel value

Cause = value

Diagnostic = value

EXPLANATION: A CLEAR packet has been received from the X.25 network. The connection using the indicated channel of the switched virtual circuit is broken. The displayed Cause and Diagnostic are the cause and diagnostic fields (shown in decimal) from the CLEAR packet.

A CLEAR may also be the response to a CALL packet indicating that the switched virtual circuit could not be completed. A cause code value of zero indicates that the CLEAR originated from the host.
