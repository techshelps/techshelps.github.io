Message 228

FRMR sent (value)

Connection = value

Local MAC address = value

Local ring number = value

Remote MAC address = value

Routing data = value

Source SAP = value

Destination SAP = value

Connection data = N(S) N(R) PFRC LRNR FRMD CRCV CSND LKST
value

EXPLANATION: A FRMR frame was sent on this connection indicating that the remote station sent a frame containing an error. The connection will be deactivated.

The FRMR code takes the following values:

01 Invalid N(R) value received

02 Maximum I-field length exceeded

03 Invalid or unsupported command/response received

04 I-frame received when not permitted

ACTION: See message #226 for details of the connection data.

If this problem persists, contact support personnel for your network.
