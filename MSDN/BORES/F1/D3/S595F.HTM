APPC session activation failure: BIND negative response sent

Sense data = value

EXPLANATION: An APPC session has failed to activate because an error was detected in the SNA BIND request sent by the partner LU. The sense data gives more information about the nature of the error.

This error is usually caused by a mismatch between the SNA Server configuration and the configuration of the partner LU's system (for example, the mode name specified in the SNA BIND request may not be configured in SNA Server).

ACTION: Provide network support personnel with the information contained in this message.
