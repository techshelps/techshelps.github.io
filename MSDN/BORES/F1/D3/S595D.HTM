Message 15

APPC session activation failure: host session set-up error

Sense data = value

Connection = value

LU alias = value

PLU alias = value

Mode name = value

LU address = value

TH = value

RH = value

RU = value

EXPLANATION: An APPC session has failed to activate because the host has sent an SNA NOTIFY request indicating a problem connecting to the partner LU. The sense data contains more information about the reason for the failure.

This error is usually caused by a temporary error at the host (for example, in an SNA Server network consisting of multiple subdomains, the connection between the SSCP host and the host containing the partner LU may have failed). This error may also be caused by a mismatched configuration.

ACTION: Provide network support personnel with the information contained in this message.
