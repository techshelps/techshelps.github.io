Host Security Integration Components

Host Security Integration is comprised of components packaged as three setup options. Each one works with the Windows NT primary domain or resource domain. These include:

  • Windows NT Account Synchronization Service
  • Host Account Synchronization Service
  • Host Account Cache

Figure 5.1 This figure shows the Host Security Integration components and where they may reside in a security domain.

These services are installed using two methods. The Windows NT Account Synchronization Service and the Host Account Cache are installed using the Host Security Setup located on the SNA Server CD-ROM. The Host Account Synchronization Service is installed during SNA Server Setup. The latter is one of the available options in the Microsoft SNA Server - Select Components dialog box. For more information see the Microsoft SNA Server Getting Started.
