Sense Data Specific to LU Type 6.2

The LU type 6.2 sense codes are specified and interpreted by SNA components within SNA Server, and, in general, are presented as return codes and as parameters to specific verbs.

The following sense codes are carried on responses flowing on LU type 6.2 sessions.

Code

Meaning

0814

Bracket bid reject — RTR forthcoming.

0819

RTR not required — receiver of RTR request has nothing to send.

0835

Invalid parameter — request contained a fixed or variable field which is invalid. Bytes 2 and 3 contain a 2-byte binary count which indexes the field error (zero origin).

0846

ERP message forthcoming: The received request was rejected for a reason to be specified in a forthcoming FMH-7 request — which itself carries SNA sense data (see the following table for a list of these codes).

088B

BB not accepted — BIS reply requested.

2010

BIS protocol error —receiver detected a BIS error; for example, two BIS replies received.
