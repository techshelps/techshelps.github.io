Message 177

The sending of the phone number from SNA Server to the modem failed: no CTS. Check modem state, power, and cabling, for the modem used by connection value through link service value.

EXPLANATION: The dialing process failed and there is no CTS signal from the modem.

ACTION: Check the modem connections and the modem. See the Microsoft SNA Server Administration Guide for more information on the use of modems.
