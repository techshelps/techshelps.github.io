Chapter 10 Performance Monitor Counters

Performance Monitor counters are useful for evaluating the demand on and performance of the servers in your subdomain. You can obtain benchmarks and use the information to estimate hardware requirements for future growth.

The Performance Monitor is a graphical tool included in Windows NT. The Performance Monitor allows you to collect performance data on your computer or other computers on the network. To start Performance Monitor for SNA Server, make sure that all the services you want to monitor are active. Then, go to the Windows NT Administrative Tools and select Performance Monitor.
