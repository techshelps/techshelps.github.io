Message 647

SNA Server could not initiate LAN communications due to a LAN connection failure.

EXPLANATION: The first mailslot broadcast issued a return code 2136 — NERR_NetworkError. This means that although the primary or backup server (node) started, the LAN is down. Any services started on the server cannot communicate with clients or with other nodes, but you can work locally on the server.

ACTION: Contact the LAN administrator at your site to get the LAN working.
