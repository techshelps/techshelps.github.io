Message 162

(value) Protocol Violation

EXPLANATION: The adjacent SDLC link station caused a protocol violation. This is probably a configuration or internal logic error in the remote system.

SUBCODE INFORMATION:

X'0146' The SDLC frame was too short — there is no address or control.

X'014A' Received NR value is invalid.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, SNA Server trace file(s), and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
