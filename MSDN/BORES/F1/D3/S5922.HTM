Lists of Available Variables

The following table outlines the Registry variables that can be set on a computer running SNA Server. For detailed information about each variable, see the sections that follow.

Table 8.1 Registry Variables for SNA Server

Variable

What is affected

ALLOCINACT:REG_SZ:{ YES | NO }

Allocation of inactive LUs from a pool

ALLOWBBRQE:REG_SZ:{ YES | NO }

Use of begin bracket, exception response

CLEANUP:REG_SZ:{ YES | NO }

Deactivation method for sessions

COMBUF:REG_DWORD:number

Size of buffer pool available

NO4004:REG_SZ:{ YES | NO }

Handling of sense code 4004

NOINITCNOS:REG_SZ:{ YES | NO }

Time at which CNOS negotiation is performed

BINDRETRY:REG_DWORD:number

Retry time-out for BIND attempts for autoactivated sessions

NORETRY:REG_SZ:{ YES | NO }

Retry of BIND

NOTERMSELF:REG_SZ:{ YES | NO }

Termination of 3270 sessions

RENEGLIMITS:REG_SZ:{ YES | NO }

Automatic CNOS renegotiation when a BIND fails

RETRYCNOS:REG_SZ:{ YES | NO }

Automatic CNOS renegotiation when partner resets session limits

SESTIMEOUT:REG_SZ:number

Time at which unused contention winner sessions are deactivated

NONOTIFYRSP:REG_SZ:{ YES | NO }

How SNA Server responds to the lack of a Notify response.

SyncConfigFiles:REG_SZ:{ ON | OFF }

SnaBase: copying of configuration file from primary to backup servers

ACCPTFMT:REG_SZ:{ 0 | 1}

X.25: use of addresses on CALL_ACCEPT

LUWIDSUP:REG_SZ:{ YES | NO }

APPC: support of logical unit of work identifier

application:REG_SZ:localLUalias

CPI-C: the local LU alias to be used by an application

APPCTPN:REG_SZ:TPname

CPI-C: the local TP name

COMTBLG:REG_SZ:filename

CSV: name of file containing Table G (for code conversion)

COMCPT:REG_DWORD:path

CSV: path of code page (for code conversion)

SNAMAXLUALUS:REG_DWORD:number

LUA: number of frames in the protocol window

LOCALLUS:REG_MULTI_SZ:localLUname

No longer used in SNA Server 3.0

COMPOP12:REG_SZ:{ YES | NO }

Suppression of level 12 pop-up warnings

Unique5250Names:REG_SZ:{ YES | NO }

Uniqueness of session IDs for 5250 emulators

FrameType:REG_DWORD:number

DIX or 802.3 framing in an Ethernet LAN; significant when communicating via a router

BufferAuditInterval:REG_DWORD:number

Auditing of buffer pool at regular intervals

DloadMatchTPOnly:REG_SZ:{ YES | NO }

Method for matching requested TP against available TPs

DloadMatchLocalFirst:REG_SZ:{ YES | NO }

Method for matching requested TP against available TPs

ConfigFilePath:REG_SZ:pathname

Path used when the server accesses the local configuration file

NumberOfDevices:REG_DWORD:ports

Number of RAS ports available to SNA Remote Access Service
