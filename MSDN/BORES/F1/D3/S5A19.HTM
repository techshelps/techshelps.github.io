Message 338

CLEAR packet sent on channel value. Diagnostic = value

EXPLANATION: The X.25 link service has sent a CLEAR request to the X.25 network, breaking the switched virtual circuit on the indicated channel.

A diagnostic code of zero is sent when the virtual circuit is being cleared because the connection using it is no longer active. Non-zero values are sent when the X.25 link service detects an incorrect or unexpected packet from the host or the X.25 network.

ACTION: If the CLEAR was unexpected, restart the link service with tracing on, and try to reproduce the error. Contact the host administrator if the problem occurs again.
