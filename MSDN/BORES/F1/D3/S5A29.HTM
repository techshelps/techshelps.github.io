Message 361

Failed to open help file value, rc = value

EXPLANATION: The help file shown could not be opened. The return codes generated are listed here.

SUBCODE INFORMATION:

OpenFileOnPath failed. Check that the file exists in a directory in the PATH environment variable, and that you have permission to access the file.

Could not read file header. The help file may have been corrupted.

Too many open help files. This error should not occur during normal use. Contact support services at your site.

Bad appended file. This error should not occur during normal use. Contact support services at your site.

The file accessed was not a help file. You may have copied another file to this file by mistake.

Incompatible version. The version of the help file is incompatible with the version of the program you are using. Make sure you are using the help file that was provided with the program.

Memory allocation failed. There was not enough memory available to open the help file. Try to clear more memory and then retry.

ACTION: Check the return code and take the indicated action.
