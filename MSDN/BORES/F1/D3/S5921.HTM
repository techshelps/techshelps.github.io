Setting Variables on Different Operating Systems

The following table shows how Registry and environment variables are set on a server and on the various types of SNA Server clients. For information about the Windows NT Registry, see the Windows NT documentation. For information about environment variables for the Windows™ version 3.x, MS-DOS®, or OS/2® operating systems, see the appropriate operating system documentation.

Operating system and SNA Server software

Method for setting Registry variables or environment variables

Windows NT
with SNA Server or SNA Server client

Change values in the Windows NT Registry by using the Registry editor, REGEDT32.

Windows 95
with SNA Server client

Change values in the Windows 95 Registry by using the Registry editor, REGEDIT.

Windows version 3.x
with SNA Server client

Add or change lines in the WIN.INI file.

MS-DOS
with SNA Server client

Add or change lines in the SNA.INI file.

OS/2
with SNA Server client

Use set commands, either at the command prompt or in CONFIG.SYS.
