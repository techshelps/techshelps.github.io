SNA Server Remote Access Service:
SNA Server of Variable

The Registry variables that affect the SNA Remote Access Service on a computer running SNA Server are found in the following location in the Registry:

HKEY_LOCAL_MACHINE
SOFTWARE
Microsoft
RAS
OTHER DEVICES
INSTALLED
SnaRas

There is one user-configurable variable:

NumberOfDevices:REG_DWORD:ports Default: 10

Specifies the number of RAS devices (ports) that can be configured for SNA Remote Access Service.
