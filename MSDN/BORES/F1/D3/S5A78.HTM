Message 613

Invalid computername or type of SnaBase service

EXPLANATION: An attempt was made to start an invalid type of SNA Server software or client software for the specified computername. The possible types are as follows:

  • The primary server, on which the primary configuration file is stored. There can be only one primary server in a subdomain.
  • Backup server(s), on which extra copies of the latest configuration file should be stored. There can be more than one backup server in a subdomain.
  • Member servers, on which there is no SNA Server configuration file. There can be more than one member server in a subdomain.
  • Client computers, which run client software, not SNA Server software.

ACTION: To change the role of the server, use SNA Server Manager.
