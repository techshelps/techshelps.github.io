Message 4

(value) Message Sequence Error

EXPLANATION: The server (node) received an invalid or unexpected message from a remote node on an SNA session. The remote system has a possible configuration or internal logic error.

SUBCODE INFORMATION:

X'1130' SNA pacing count error. The pacing count held exceeds the receive pacing window specified in the BIND, caused by a remote LU sending more than one pacing request within the window.

X'1132' The node received an FMD or DFC request when the data traffic state was not active.

X'1133' The node received a request with an invalid sequence number.

X'1139' The node received an STSN session control request while there was active data traffic on the session.

X'1182' The node received a BIND response when no BIND request was outstanding.

X'1183' The node received an UNBIND request for an inactive session, or an UNBIND response when no UNBIND request was outstanding.

X'11B2' The node received an unexpected BIS reply from a partner LU.

X'11B3' The node received an unexpected BIS request from a partner LU.

X'11B5' The node received an unexpected RTR from a partner LU.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, SNA Server trace file(s), and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
