Message 3

(value) Node Configuration Error

EXPLANATION: The server (node) detected a configuration file problem caused by a logic error in the configuration program. These subcodes should not occur during normal operation.

SUBCODE INFORMATION:

X'1100' Incorrect Record Reference Number.

X'1101' Invalid length of product data record.

X'1102' Number of configured control blocks is valid.

X'1103' Error detected while resetting a control block.

X'1106' Configuration file is corrupted.

X'1107' Operator interface queue thresholds invalid.

X'1173' The SNA Server Manager program is not configured to allow permission to change session limits.

X'117A' Not enough room in configuration table.

ACTION: Provide network support personnel with the SNA Server configuration file. For information about the configuration file, see the Microsoft SNA Server Administration Guide.
