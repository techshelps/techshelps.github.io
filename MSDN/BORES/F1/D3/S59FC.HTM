Message 303

(value) Resource Allocation Error

EXPLANATION: This is an internal error and should not occur during normal operation.

SUBCODE INFORMATION:

X'2505' L3 clock handler has run out of QCBs.

X'2506' Run out of QCBs on clock initialization.

X'2521' Buffer manager cannot allocate a QCB.

X'2522' Buffer manager cannot provide an RX buffer.

X'2523' Buffer manager cannot release an RX buffer.

X'2524' Buffer manager cannot release a TX buffer.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, the SNA Server configuration file, and the number and description of the displayed subcode. For information about SNA Server log files and the configuration file, see the Microsoft SNA Server Administration Guide.
