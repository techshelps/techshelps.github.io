SnaServer

The SNA protocol engine of SNA Server implements the SNA protocol profiles of PU 2.0 and PU 2.1. SnaServer is the component of SNA Server that interacts with client computers and other nodes on the SNA network, using the four DLLs. SnaServer also provides up-to-date status to SNA Server Manager, providing real-time display information on servers, clients, connections, LUs, and users.
