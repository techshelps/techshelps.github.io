TN3270 Performance Monitor Counters

This section describes the Performance Monitor counters for TN3270 service. The object is TN3270 Server. The following are the counters for the TN3270 Server object:

Connections Active

The number of TCP connections currently active.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

RU Throughput

This is the total number of RUs sent and received by TN3270 service.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

RU Throughput/sec

The rate at which RUs are being sent and received by TN3270 service.

Counter Type: PERF_COUNTER_COUNTER

Default Scale: 1 Counter Size: 4 bytes

RUs Inbound

The total number of RUs sent to the IBM host by TN3270 Service.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

RUs Inbound/sec

The rate at which RUs are being sent to the IBM host by TN3270 Service.

Counter Type: PERF_COUNTER_COUNTER

Default Scale: 1 Counter Size: 4 bytes

RUs Outbound

The total number of RUs received from the IBM host by TN3270 Service.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

RUs Outbound/sec

The rate at which RUs are being received from the IBM host by TN3270 Service.

Counter Type: PERF_COUNTER_COUNTER

Default Scale: 1 Counter Size: 4 bytes

Sessions Active

The number of TN3270 clients currently using an LU.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

Sessions Configured

The number of sessions available to TN3270 Service.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

Sessions Ended LULU

The number of TN3270 client sessions that were terminated while in session with an IBM host application.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

Sessions Ended Normal

The number of TN3270 client sessions that were terminated normally.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

Sessions Failed

The number of TCP/IP connections that were actively terminated by TN3270 Service.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

Sessions Failed Inactivity

The number of TCP connections that were actively terminated by TN3270 Service due to inactivity.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

Sessions Failed No Resource

The number of TN3270 client sessions that were rejected because no SNA Resource could be obtained.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

Sessions Succeeded

The number of TN3270 client sessions that have successfully obtained an LU since TN3270 Service was started.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

TCP Bytes Received

The total number of TCP bytes received by TN3270 Service.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

TCP Bytes Received/sec

The rate at which TCP bytes are being received by TN3270 Service.

Counter Type: PERF_COUNTER_COUNTER

Default Scale: 1 Counter Size: 4 bytes

TCP Bytes Sent

The total number of TCP bytes sent by TN3270 Service.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

TCP Bytes Sent/sec

The rate at which TCP bytes are being sent by TN3270 Service.

Counter Type: PERF_COUNTER_COUNTER

Default Scale: 1 Counter Size: 4 bytes

TCP Throughput

The total number of TCP bytes received and sent by TN3270 Service.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes

TCP Throughput/sec

The rate at which TCP bytes are being received and sent by TN3270 Service.

Counter Type: PERF_COUNTER_COUNTER

Default Scale: 1 Counter Size: 4 bytes

Users Active

The number of different TCP/IP addresses currently active in the TN3270 Service.

Counter Type: PERF_COUNTER_RAW_COUNT

Default Scale: 1 Counter Size: 4 bytes
