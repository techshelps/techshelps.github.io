Message 18

APPC session activation failure: BIND negative response or UNBIND request received

Sense data = value

Connection = value

LU alias = value

PLU alias = value

Mode name = value

EXPLANATION: An APPC session has failed to activate because the partner LU rejected the SNA BIND request sent to initiate the session. The sense data gives more information on the reason for the rejection.

This error is usually caused by a mismatch between the SNA Server configuration and the configuration of the partner LU's system (for example, the mode name sent in the SNA BIND request was not acceptable to the partner LU).

ACTION: Provide network support personnel with the information contained in this message.
