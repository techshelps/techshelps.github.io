Message 336

CALL CONNECTED response received on channel value

Remote (called) NUA = value

Local (calling) NUA = value

Facility data = value

EXPLANATION: A switched virtual circuit has been connected successfully on the specified channel. This packet is an "OK" response to a previous CALL request packet sent by the X.25 link service.

The returned facility data is displayed in hexadecimal. The values may have been negotiated (altered) by either the remote host or the X.25 network itself.

Note The locNUA may be filled in by the X.25 network itself, even if it has not been configured and was not present on the initial CALL packet.

No action is necessary.
