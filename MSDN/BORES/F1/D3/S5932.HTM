The following Registry variables affect the SNA Server Distributed Link Service.

LocalIpPort:REG_DWORD: Default: 0

LocalIpxPort:REG_DWORD: Default: 0

LocalVinesPort: REG_DWORD: Default: 0

These Registry variables specify the port number the distributed link service uses to connect to the remote computer. If set to 0, it will use dynamic ports.
