SNA Server Performance Counters

The following list describes objects and counters in SNA Server that may be useful for evaluating demand and performance on an SNA Server computer.

The counters are chosen via the Object pull-down box. The objects that are specific to SNA Server are:

  • SNA Adapter <adapter name>
  • SNA Connections
  • SNA Logical Unit Sessions
  • TN3270 Server

Many of the objects have common counters, such as Data Bytes Received/Sec and Data Bytes Transmitted/Sec. Each counter is discussed below only once with a reference given to each object that has the counter.

Adapter Failures

The number of times since startup that a network adapter encountered an error condition.

Object location:

SNA Adapter adaptername

Connection Failures

The number of times since startup that a connection has failed.

Object location:

SNA Adapter adaptername

Databytes Received/Sec

This is the number of data bytes received per second.

Object location:

SNA Adapter adaptername

SNA Connections

SNA Logical Unit Sessions

Databytes Transmitted/Sec

This is the number of data bytes transmitted per second.

Object location:

SNA Adapter adaptername

SNA Connections

SNA Logical Unit Sessions

Frames Received/Sec

The number of data frames received per second. A frame is an information structure organized by one of the various protocols related to SNA.. Frames contain multiple bytes of data.

Object location:

SNA Adapter adaptername

Frames Transmitted/Sec

The number of data frames transmitted per second.

Object location:

SNA Adapter adaptername

Successful Connects

The number of times since startup that a successful connection has been made.

Object location:

SNA Adapter adaptername

Throughput Bytes/Sec

The total number of bytes flowing through SNA Server per second. This includes both incoming and outgoing bytes, and is a good indicator of how heavily your SNA Server computers are loaded.

Object location:

SNA Adapter adaptername
