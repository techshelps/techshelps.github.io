Message 160

(value) Resource Allocation Error

EXPLANATION: Severity 16.

SUBCODE INFORMATION:

X'014B' SNA Server is unable to send a message to the node due to a buffer shortage. This is an internal error that should not occur during normal operation.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, internal trace file(s), and the dump file. For information about SNA Server log, trace, and dump files, see the Microsoft SNA Server Administration Guide.
