LUA API: SNA Server of Variable

For SNA Server or a Windows NT client, the Registry variable that affects the conventional Logical Unit Application programming interface (LUA) API is found in the following location:

HKEY_LOCAL_MACHINE
SYSTEM
CurrentControlSet
Services
SnaBase
Parameters
Client

This is the variable name and description:

SNAMAXLUALUS:REG_DWORD:number

This variable sets the maximum number of frames in the protocol window. Setting this number lower means that fewer buffers will be required on the system. It is unlikely that this variable would be required for a Windows NT-based system.
