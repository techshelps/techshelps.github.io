Request Errors (Category X'10')

Sense codes in this category represent an imbalance in half-session capabilities. They indicate that the RU was delivered to the intended half-session but could not be processed. The following codes are in this category:

Code

Meaning

1003

Function not supported (also used instead of X'0826', which is not supported by SNA Server).

1005

Parameter modifying a control function is invalid.

1007

Category not supported (SSCP data for host printer).
