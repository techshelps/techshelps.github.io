Message 8

(value) Internal Protocol Violation

EXPLANATION: A protocol violation has occurred at the interface between the server (node) and another SNA component (for example, the emulator or link service).

SUBCODE INFORMATION:

X'1105' Insufficient space to copy data into node buffer.

X'1112' Received unknown message type at the node interface.

X'1144' Received illegal message at the node interface.

X'1151' Received unrecognized message from the link service component.

X'1174' Received invalid session limits message from SNA Server Manager.

X'1175' Received invalid CNOS command.

X'1188' Received invalid CNMS message.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, SNA Server trace file(s), and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
