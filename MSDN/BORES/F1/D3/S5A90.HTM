Message 638

Possible Replication problem: the primary SNA Server was started on a system with file replication active but not replicating the SNA configuration file.

EXPLANATION: Check to see if this computer is supposed to support replication of SNA Server files. If not, ignore the message. If SNA Server file replication is intended, set up file replication with Windows NT.

ACTION: For information about setting up file replication, see your Windows NT documentation.
