Message 229

SABME received when connection active

Connection = value

Local MAC address = value

Local ring number = value

Remote MAC address = value

Routing data = value

Source SAP = value

Destination SAP = value

Connection data = N(S) N(R) PFRC LRNR FRMD CRCV CSND LKST
value

EXPLANATION: A SABME frame was received from the remote station while the connection was activated. The connection will be deactivated. This may be caused by an error at the remote station causing it to reset.

ACTION: See message #226 for details of the connection data.

If this problem persists, contact support personnel for your network.
