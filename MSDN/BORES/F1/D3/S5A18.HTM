Message 337

RESTART packet sent. Diagnostic = value

EXPLANATION: The X.25 link service has sent a RESTART request to the X.25 network. Any active connections are lost.

A RESTART packet is normal whenever communication between the X.25 link service and the X.25 network is first established, but otherwise may indicate that a serious failure has occurred.

ACTION: If the RESTART was unexpected, restart the link service with tracing on, and try to reproduce the error. Contact the host administrator if the problem occurs again.
