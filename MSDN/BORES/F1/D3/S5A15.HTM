Message 334

RESET packet received from network on channel value

Cause = value

Diagnostic = value

EXPLANATION: A RESET packet has been received from the X.25 network. The connection using the indicated channel of the switched virtual circuit is broken. The displayed Cause and Diagnostic are the cause and diagnostic fields (shown in decimal) from the RESET packet.

A value of zero for the Diagnostic indicates that the RESET originated from the host.

ACTION: If the RESET was unexpected, restart the link service with tracing on, and try to reproduce the error. Contact the host administrator if the problem occurs again.
