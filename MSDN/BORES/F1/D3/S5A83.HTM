Message 624

Creating dump file value for value

EXPLANATION: A process (value) ended abnormally due to a protection violation or similar condition and a dump file value has been created that contains information about the state of the process at the time of the exception. Register information is contained in your Windows NT root directory in the file DRWTSN32.LOG.

ACTION: For more information on diagnostics, refer to the Microsoft SNA Server Administration Guide. Provide network support personnel with the event log file(s) related to SNA Server, the dump file, and the Dr. Watson log file.
