Message 120

(value) Shortage Of Buffers

EXPLANATION: There are insufficient buffers for the component to continue processing.

ACTION: Use the COMBUF Registry variable to allocate more buffers to the component. If increasing the size of the buffer pool does not solve the problem, it may indicate an internal error. Provide network support personnel with the event log file(s) related to SNA Server, as well as the dump file.

For information about SNA Server log and dump files, see the Microsoft SNA Server Administration Guide. For information about COMBUF, see the Appendix C, "Registry and Environment Variables for SNA Server."
