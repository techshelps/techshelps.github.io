Message 5

(value) Unsupported Session Control Request

EXPLANATION: The server (node) received an unexpected session control request. This is a protocol violation by the remote system. The remote system has a serious configuration or internal logic problem.

SUBCODE INFORMATION:

X'1137' The node received an SDT session control request.

X'1138' The node received an STSN session control request.

X'113A' The node received an RQR session control request.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, SNA Server trace file(s), and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
