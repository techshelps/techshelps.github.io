Message 13

(value) Message Sequence Error (SENSE = value)

EXPLANATION: The server (node) received an invalid or unexpected message from a remote node during an SNA session. The cause is probably a logic error in the remote system. The SNA sense data shown gives details of the type of error.

SUBCODE INFORMATION:

X'1141' Session-level protocol violation in an SNA message from the partner LU.

X'1142' Session-level protocol violation on host SSCP-to-LU session.

X'11B1' A BID request has failed the SNA receive request checks.

X'11C2' A partner LU has violated LU 6.2 protocols.

ACTION: See IBM Systems Network Architecture: Formats for an explanation of the SNA codes. Provide network support personnel with the event log file(s) related to SNA Server, SNA Server trace file(s), and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
