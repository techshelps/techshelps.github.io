Introduction

Welcome to the Microsoft SNA Server Resource Guide. This guide is designed for people who are, or want to become, expert users of Microsoft SNA Server. This guide is a technical supplement to the documentation included as part of the SNA Server product and does not replace that information as the source for learning how to use SNA Server.

This guide is included in the Microsoft BackOffice Resource Kit. The Microsoft BackOffice Resource Kit for SNA Server includes many tools you can use to support your SNA Server installation. For more information about SNA Server tools, see the SNATOOLS.HLP file on the Microsoft BackOffice Resource Kit CD-ROM.
