Message 26

APPC session activation failure: session limit error

Connection = value

LU alias = value

PLU alias = value

Mode name = value

EXPLANATION: An APPC session has failed to activate because the session limits for the mode have not been negotiated correctly. This error is usually caused by a mismatch between the SNA Server configuration and the configuration of the partner LU's system (for example, the requested mode's session limits may not be configured to correspond to the local mode's limits).

ACTION: Correct the configuration error or contact support personnel for your network.
