Message 633

The sharename COMCFG does not exist, or the SNA Server configuration file COM.CFG does not exist.

EXPLANATION: When a primary or backup server is started, it must be sharing the configuration file on a share called \\servername\COMCFG, where servername is the name of the server. This is checked by the SnaBase at startup.

A SnaBase started as a backup server will also check the file replication information and may log message #634 as well.
