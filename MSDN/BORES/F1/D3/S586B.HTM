WINAPPC

APPC is an API that allows peer-to-peer communications in an SNA environment using LU 6.2 over either dependent or independent LUs. Through APPC, programs distributed across a network can work together, communicating with each other and exchanging data, to accomplish a single processing task such as querying a remote database, copying a remote file, or sending and receiving electronic mail.

A detailed description of APPC APIs is found in the Microsoft SNA Server APPC Programmer's Guide.
