Message 643

Invokable TPs can only be executed in the background when LAN Manager local security is active

EXPLANATION: You can execute invokable TPs only in the background when local security is active on the network. This message only applies to OS/2-based clients.

ACTION: Reconfigure the TP to run without a screen group, if it can run in that mode. Otherwise, load the TP on a server that is not running local security.
