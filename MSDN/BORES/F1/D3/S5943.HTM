Environment Variables Used on Clients Running
Windows Version 3.x, MS-DOS, or OS/2

The environment variables in the following sections can be set on an SNA Server client running the Windows version 3.x, MS-DOS, or OS/2 operating systems. For information about accessing or modifying these variables, see "Setting Variables on Different Operating Systems" earlier in this appendix.

Note The following sections do not include the variables used for configuring an invokable TP. For information about those variables, see Chapter 2 of the Microsoft SNA Server APPC Programmer's Guide or Chapter 2 of the Microsoft SNA Server CPI-C Programmer's Guide.
