Chapter 3 SNA Server Support

This chapter explains how SNA Server works within SNA networks. The following information is included:

  • A summary of the types and number of SNA sessions supported by SNA Server.
  • A list of the SNA commands supported by SNA Server. This section also shows the type of SNA sessions in which the commands flow.
  • The transmission service (TS) and function management (FM) profiles supported by SNA Server.
  • A list and explanation of the five categories of SNA sense codes generated by SNA Server. This section also describes logical unit (LU) type 6.2-specific sense data.

For specific information about an application programming interface (API) supported by SNA Server, see the programmer's guide for that API:

  • Microsoft SNA Server APPC Programmer's Guide
  • Microsoft SNA Server CPI-C Programmer's Guide
  • Microsoft SNA Server LUA Programmer's Guide
