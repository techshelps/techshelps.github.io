Message 2

(value) Internal Error

EXPLANATION: An internal logic error occurred. These subcodes should never occur during normal operation.

SUBCODE INFORMATION:

X'1' An invalid message was sent by SNA Server Manager to a local server on the internal administration connection.

X'2' An unknown component has attempted to log an error.

X'6' The administration connection between SNA Server Manager and the local server (node) was lost while processing a command from SNA Server Manager.

X'7' Error in the format of the message sent from SNA Server Manager to the local server (node).

X'8' Buffer shortage or queue length exceeded. The administration connection between SNA Server Manager and the node has been damaged.

X'11E0' The local server (node) has found an error in its internal count of the number of messages in transit internally.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, SNA Server trace file(s), and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
