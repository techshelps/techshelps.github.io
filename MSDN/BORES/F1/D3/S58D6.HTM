LUSTAT Sense Codes

The following sense codes are sent by a 3270 emulator on LUSTAT requests.

Code

Meaning

0001 B000

Host initialized local copy soft error recovery.

081C 0000

Soft error changed to hard error (LU1/LU3).

081C B000

Soft error changed to hard error (local copy).

082B 0000

SLU session changed from SSCP to PLU (SYSREQ).
