Message 335

Incoming CALL packet received on channel value

Remote (calling) NUA = value

Local (called) NUA = value

Facility data = value

EXPLANATION: An incoming CALL packet indicates that a remote computer is attempting to establish an X.25 connection with a local server. This will be rejected unless an active local server has been configured to receive incoming calls to address locNUA and has an X.25 connection configured with address remNUA.

The "Facility data" is a hexadecimal representation of the facility data on the CALL packet. If the requested facilities are invalid or not supported by the X.25 link service, the CALL is rejected.

No action is necessary.
