Message 101

(value) Internal Protocol Violation

EXPLANATION: The component logging this error received an invalid message from another SNA Server component. This is an internal error and should not occur during normal operation.

SUBCODE INFORMATION:

0B49 Wrong length for Locality Data.

0A08 Invalid message discarded.

0A40 Unrecognized Open request type.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, internal trace file(s), the dump file, and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
