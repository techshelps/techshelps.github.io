Message 557

DLOAD has timed out

EXPLANATION: The node requested dynamic loading of an APPC TP, but did not receive a response from the TP within the specified time. This could be caused by the following conditions:

  • The TP started but failed before issuing a Receive_Allocate (check for messages logged by the TP).
  • The "Timeout for Starting TPs" for the local APPC LU used by this TP is too short.
  • The TP is operator-started, and the operator did not start it.
