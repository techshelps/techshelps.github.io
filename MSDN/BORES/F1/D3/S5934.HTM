X.25 Link Service: SNA Server of Variable

The Registry variable that affects the X.25 link service is found in the following location in the Registry:

HKEY_LOCAL_MACHINE
SYSTEM
CurrentControlSet
Services
X.25LinkServiceName
Parameters
ExtraParameters

This is the variable name and description:

ACCPTFMT:REG_SZ:{ 0 | 1 } Default: 0

When this variable is set to 1, calling and called addresses are omitted on a CALL_ACCEPT packet that is sent in response to an incoming call. Setting this variable to 1 is a workaround for a problem when the host rejects standard X.25 CALL_ACCEPT packets.
