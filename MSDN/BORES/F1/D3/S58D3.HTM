State Errors (Category X'20')

Sense codes in this category indicate a sequence number error, or an RH or RU that is not allowed in the receiver's current state. This condition prevents delivery of the request to the intended half-session. The following codes are in this category:

Code

Meaning

2001

Sequence number error

2002

Chaining error

2003

Bracket error

2004

Direction error

2005

Data traffic reset

2006

Data traffic quiesced

2007

Data traffic not quiesced
