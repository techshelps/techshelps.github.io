Message 51

User ID value not found

EXPLANATION: A remote APPC TP supplied a user ID not recognized by SNA Server. This error occurs when a remote APPC TP attempts to establish a conversation with an SNA Server TP configured to use conversation-level security.

ACTION: Check the user ID against the configured IDs.

This message may indicate that an unauthorized remote user is attempting to access your programs from a remote LU that imitates the remote LU that you configured.
