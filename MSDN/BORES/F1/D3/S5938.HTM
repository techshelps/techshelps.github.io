CSV API: SNA Servers of Variables

For SNA Server or a Windows NT-based client, the Registry variables that affect the CSV API are found in the following location:

HKEY_LOCAL_MACHINE
SYSTEM
CurrentControlSet
Services
SnaBase
Parameters
Client

These are the variable names and descriptions:

COMTBLG:REG_SZ:filename

This variable specifies the name of the file that contains Table G for code conversion. Table G is used with the CONVERT verb.

For more information about CSV, see the Microsoft SNA Server APPC Programmer's Guide.

COMCPT:REG_SZ:path

This variable specifies the directory in which the code page file (for code conversion) is located. For example, if the code page file is located in the directory C:\PAGEDIR, the setting to use is as follows:

COMCPT:REG_SZ:C:\PAGEDIR

For more information about code pages, see the Microsoft SNA Server APPC Programmer's Guide.
