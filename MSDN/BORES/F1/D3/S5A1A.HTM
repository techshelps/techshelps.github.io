Message 339

RESET packet sent on channel value. Diagnostic = value

EXPLANATION: The X.25 link service has sent a RESET request to the X.25 network on the indicated permanent virtual circuit channel. The connection using the circuit was lost.

A diagnostic code of zero is sent when the virtual circuit is being reset because the connection using it is no longer active. Non-zero values are sent when the X.25 link service detects an incorrect or unexpected packet from the host or the X.25 network.

ACTION: If the RESET was unexpected, restart the link service with tracing on, and try to reproduce the error. Contact the host administrator if the problem occurs again.
