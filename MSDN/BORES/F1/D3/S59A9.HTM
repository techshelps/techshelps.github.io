Message 152

(value) Open from node rejected, rc = value

SUBCODE INFORMATION:

X'0100' The SDLC link service rejected an Open request from the local server (node), which indicates an internal error in SNA Server.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, as well as the return code displayed with this error. For information about SNA Server log files, see the Microsoft SNA Server Administration Guide.
