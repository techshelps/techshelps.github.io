Message 103

(value) Configuration File Format Error

EXPLANATION: The node configuration data read from the configuration file is invalid.

SUBCODE INFORMATION:

X'0A00' The record was not found in the configuration file, COM.CFG.

X'0A06' The destination name was not found in the configuration file.

X'0A09' The resource ID was incorrect.

X'0A43' The Record Reference Number was repeated in the configuration file.

X'0A44' The Reference Number in the configuration file was too big.

X'0A45' The record was not found in the configuration file.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, the SNA Server configuration file, and the number and description of the displayed subcode. For information about SNA Server log files and the configuration file, see the Microsoft SNA Server Administration Guide.
