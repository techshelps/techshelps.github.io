Message 14

APPC session activation failure: INITSELF negative response received

Sense data = value

Connection = value

LU alias = value

PLU alias = value

Mode name = value

LU address = value

EXPLANATION: An APPC session has failed to activate because the host has rejected the SNA INITSELF request sent to initiate the session. The sense data contains more information about the reason for the rejection.

This error is usually caused by a mismatch between the host configuration and the SNA Server configuration. It may also be caused by some temporary error at the host (for example, the partner LU may be temporarily unavailable).
