Message 589

Failed to display a pop-up, rc = value

EXPLANATION: A CreateProcess call for the pop-up process failed. The return code is shown.

ACTION: Check the return code and attempt to free more memory. A return code of 999 indicates that a pop-up could not be displayed on an MS-DOS-based system because an application was already using the foreground screen.

This error is caused by a resource shortage on the system.

Also, provide network support personnel with the event log file(s) related to SNA Server. For information about SNA Server log files, see the Microsoft SNA Server Administration Guide.
