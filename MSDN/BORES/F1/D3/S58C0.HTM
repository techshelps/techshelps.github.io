Session Support

SNA Server supports three types of SNA communications sessions:

  • SSCP-PU

    The SNA Server PU Services Manager communicating with a host systems services control point (SSCP).

  • SSCP-LU

    An SNA Server LU communicating with a host SSCP.

  • PLU-SLU

    SNA Server LUs communicating with host or peer LUs.
