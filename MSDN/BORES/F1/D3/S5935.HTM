Registry Variables Used on SNA Server, a Windows NT Client, or a Windows 95 Client

The Registry variables in the following sections can be set on SNA Server or on a Windows NT client. MaxThreads can be set only on a Windows NT client, and SnaBackupDomain and BackupSponsors can only be set on Windows 95 and Windows NT clients. To access or modify these variables, use the Registry Editor, REGEDT32 (Windows NT) or REGEDIT (Windows 95).

Note The following sections do not include the variables used for configuring an invokable TP. For information about these variables, see Chapter 2 of the Microsoft SNA Server APPC Programmer's Guide or Chapter 2 of the Microsoft SNA Server CPI-C Programmer's Guide.
