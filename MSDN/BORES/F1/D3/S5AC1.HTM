Message 697

SNA Server failed to add the service value to the list value. The VnsAddSStMember() function failed with VINES error value.

EXPLANATION: SnaBase failed to add its name to the list of SNA Servers in the Windows NT domain. This step is necessary for the server to function correctly in a Banyan VINES network.

Most common error codes:

155 = The net/host is unreachable, communication problem between SNA Server machine and VINES Server.

1001 = Bad name, incorrect syntax appeared in a StreetTalk name.

1013 = No such object, StreetTalk cannot find this object.

1015 = Object already exists, an object with the same name already exists in the network.

1016 = Not an object name, the name specified is not a name StreetTalk knows.

1033 = Access denied, a user tried to change StreetTalk information without proper authorization.

1801 = User has not logged on to the VINES network.

ACTION: Contact your support personnel.
