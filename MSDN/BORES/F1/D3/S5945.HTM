CPI-C API

[ApplicationName]
APPCLLU=localLUalias

For this variable, substitute the name of the application for ApplicationName, and the local LU alias for localLUalias. Type the square brackets around the application name, and place the application name on a separate line, as shown. This variable specifies the local LU alias to be used with a CPI-C application (a TP).

APPCTPN=TPname

For this variable, substitute the name of the TP, in ASCII, for TPname. This variable specifies the local TP name.
