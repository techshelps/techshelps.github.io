Message 586

The workstation service must be started before the SnaBase service can be started

ACTION: Use the NET START WORKSTATION command (for a client) or NET START SERVER (for a server) before starting the SnaBase service. Or use the Windows NT Control Panel to start the appropriate services in order. For a client, start the workstation service, and then the SnaBase service. For a server, start the workstation service, then the Server service, and then the SnaBase service.
