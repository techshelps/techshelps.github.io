Message 352

Unable to open the configuration file, rc = value

EXPLANATION: A Windows NT error occurred while attempting to open the configuration file to obtain configuration information for the component that logged this message.

This error could be due to one of the following:

  • You have not configured the server using SNA Server Manager
  • The configuration file is corrupt
  • This is a member server and the primary/backup servers are not available

ACTION: Check the Windows NT return code shown on this message. See your Windows NT documentation for information on return codes.
