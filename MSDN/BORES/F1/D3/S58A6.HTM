SOGA Channel-Attachment Technology

SNA Server supports two channel-attachment technologies—Bus & Tag and ESCON (Enterprise System Connection). Both offer speed improvements over high-speed token-ring, Ethernet, and FDDI.

Bus & Tag Provides Greatest Flexibility

Bus & Tag channel attachment uses parallel copper cable and offers speeds up to 4.5 megabytes per second. This is a significant improvement over high-speed token rings, which only provide16 megabits per second.

SNA Server's channel attachment capability provides deployment flexibility because of the many channel vendors offering solutions. The hardware vendors that offer Bus & Tag products that comply with SOGA include Barr Systems, Bus-Tech, Cabletron, Computerm, and Polaris Communications. Contact information and white papers for these companies is located in the SNA Server Companion Product Catalog at http://www.microsoft.com/sna.

ESCON Provides Greatest Performance

A second channel attachment option is ESCON (Enterprise System Connection). ESCON uses standard fiber cable and increases the speeds offered by channel attachment up to 200 megabits per second (roughly 17 megabytes per second). Currently, Polaris Communications has released a PCI ESCON solution for SNA Server. Amdahl, Bus-Tech, and Dr.Materna have also announced support for SNA Server ESCON products.

SNA Server takes advantage of these high-speed channel attachment options by supporting up to 250 concurrent host connections. Each channel device can support multiple host PUs. SNA Server supports up to 5,000 concurrent users running up to 15,000 concurrent sessions.
