WINSLI

WINSLI implements the Session Level Interface (SLI) API to access dependent LUs. It can drive either LUA LUs or 3270 LUs configured in SNA Server Manager.

The most common application for RUI and SLI is 3270 terminal emulation for access to SNA mainframes. The next most common application is a broad class of LU0 applications that support a variety of custom applications in the financial services industry.
