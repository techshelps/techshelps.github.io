About the Resource Guide

This guide includes the following chapters:

Chapter 1, "SNA Server Architecture," describes the client-server and management architecture of SNA Server. It includes how SNA Server computers communicate with other servers and clients on the network.

Chapter 2, "SNA Open Gateway Architecture," describes the architecture of SNA Server in central and branch network configurations.

Chapter 3, "SNA Server Support," describes how SNA Server works within SNA networks.

Chapter 4, "Network Management," describes the support that SNA Server provides for centralized network management.

Chapter 5, "Host Security Integration Architecture," describes how Host Security Integration works with SNA Server, Windows NT, and host computers.

Chapter 6, "Printer Definition Files," describes the format of files you can use to specify the capabilities of printers made available through SNA Server Host Print Service.

Chapter 7, "Sample Host Definitions," presents samples for SNA Server connections to IBM host systems.

Chapter 8, "Registry and Environment Variables for SNA Server," describes the Registry variables and environment variables that can be set for SNA Server.

Chapter 9, "Event Viewer Messages," lists the error and audit messages for SNA Server.

Chapter 10, "Performance Monitor Counters," describes the Performance Monitor Counters used to monitor SNA Server activity.

This guide also includes an index.
