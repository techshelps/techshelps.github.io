Message 591

TP has not been configured

EXPLANATION: A TP issued the RECEIVE_ALLOCATE verb with a TP name that is not known to the system.

ACTION: Check the TP name against the configuration. The TP should supply a name that matches the Full TP Name field in an Invokable TP record. For information about TPs, see the Microsoft SNA Server APPC Programmer's Guide or the Microsoft SNA Server CPI-C Programmer's Guide.
