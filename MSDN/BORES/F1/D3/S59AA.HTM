Message 154

(value) Internal Message Sequence Error

EXPLANATION: The node received an invalid or unexpected message. This is an internal logic error in the node component detected by the link service.

SUBCODE INFORMATION:

X'0147' Link not open — cannot receive messages.

X'0148' Station not open — cannot receive messages.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, internal trace file(s), the dump file, and the number and description of the displayed subcode. For information about SNA Server log, trace, and dump files, see the Microsoft SNA Server Administration Guide.
