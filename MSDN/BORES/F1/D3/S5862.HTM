SNA Server Web Sites

You can visit the following Web sites for current information about SNA Server:

http://www.microsoft.com/sna The SNA Server Web site contains current information about SNA Server including helpful information about planning, migration, and deployment.

http://www.microsoft.com/TechNet/ The Microsoft TechNet Web site contains information about subscribing to the Microsoft TechNet, which provides in-depth technical information about Microsoft's business products, including SNA Server and other Microsoft BackOffice products.

http://www.microsoft.com/train_cert/ The Microsoft Training and Certification Web site provides information about training options and the Microsoft Certified Professional Program.
