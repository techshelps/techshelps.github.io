Message 343

Unexpected QLLC control field (value) received on connection value

EXPLANATION: The indicated connection has been lost. The "control field" value is a hexadecimal representation of the QLLC (qualified logical link control) control field from the packet causing the problem. QLLC is one of the protocols commonly used for transporting SNA data over X.25 virtual circuits.

ACTION: Restart the link service with tracing on, and try to reproduce the error. Contact the host administrator if the problem occurs again.
