Message 310

(value) Illegal packet received from network

EXPLANATION: An unexpected or invalid packet was received from the network.

SUBCODE INFORMATION:

X'2511' Unrecognized qualified data packet.

X'2555' Illegal Reset Indication packet.

X'2556' Illegal Reset Confirm packet.

X'2557' Illegal Restart Indication packet.

X'2558' Illegal Incoming Call packet.

X'2559' Illegal Call Connected packet.

X'255A' Illegal Clear Indication packet.

X'255B' Illegal Clear Confirm packet.

X'255C' Unrecognized packet type.

X'255E' Received an unexpected or invalid packet on a PVC.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server and SNA Server trace file(s) such as Data Link Control trace files for the server. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
