Message 230

Remote station not responding to value commands

Connection = value

Local MAC address = value

Local ring number = value

Remote MAC address = value

Routing data = value

Source SAP = value

Destination SAP = value

EXPLANATION: The remote station could not be contacted because it is not answering TEST or XID frames. This could be caused either by an incorrectly configured remote MAC address or remote SAP address, or by a fault in the intervening network (for example, a bridge fault).

ACTION: Check the configuration details, then contact support personnel for your network
