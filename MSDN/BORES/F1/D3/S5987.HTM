Message 53

Password mismatch, User ID = value

EXPLANATION: An invalid password was sent by a remote APPC TP during an attempt to establish a conversation with an SNA Server TP configured to use conversation-level security. The user ID supplied by the remote TP is shown. An error in the remote TP or in the SNA Server configuration can cause this error to occur.

ACTION: Check with the remote user whose user ID is shown on this message.

This message may indicate that an unauthorized remote user is attempting to access your programs from a remote LU that imitates the remote LU that you configured.
