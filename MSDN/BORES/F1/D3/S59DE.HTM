Message 238

Connection value statistics

Station log IFTX IFRX ERTX ERRX T1EX
value

SAP log FRTX FRRX FDSC DLST NBUF
value

Adapter log LNER INER BTER ACER ABDL LFRM RXCG FCER FQER TKER
value

EXPLANATION: This message logs the LAN LLC statistics when a connection is terminated. This information is provided to help evaluate the performance of the LAN or isolate problems in the operation of the LAN. Descriptions of the fields are as follows:

Station log:

IFTX Number of I-frames transmitted

IFRX Number of I-frames received

ERTX Number of I-frames transmitted with an error resulting

ERRX Number of I-frames received with an error resulting

T1EX Number of times T1 timer expired

SAP log:

FRTX Number of frames transmitted

FRRX Number of frames received

FDSC Number of frames discarded

DLST Number of times data lost

NBUF Number of buffers available in buffer pool

Adapter log:

LNER Number of line errors

INER Number of internal errors

BTER Number of burst errors

ACER Number of A/C errors

ABDL Number of times abort delimiter transmitted

LFRM Number of lost frames

RXCG Number of times receiver congested

FCER Number of frame copied errors

FQER Number of frequency errors

TKER Number of token errors

ACTION: Refer to the IBM Token Ring Network Architecture Reference for descriptions and explanations of the fields. No action is necessary.
