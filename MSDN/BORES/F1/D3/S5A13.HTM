Message 332

RESTART packet received from network.

Cause = value

Diagnostic = value

EXPLANATION: A RESTART packet has been received from the X.25 network. All virtual circuits are cleared and reset, and any active connections will be lost. The displayed Cause and Diagnostic are the cause and diagnostic fields (shown in decimal) from the RESTART packet.

A RESTART packet is normal whenever communication between the X.25 link service and the X.25 network is first established.
