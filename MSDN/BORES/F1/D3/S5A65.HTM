Message 594

DLOAD free queue contention is relieved

EXPLANATION: The SNA subsystem succeeded in queuing an ALLOCATE request to the TP. This means that the condition indicated by the previous message #576 (no more DLOAD queue entries available) cleared. This is not an error condition, but is logged at the same level as

#576 to ensure that both messages will appear.
