Message 630

Failed to get SNA Server path from base driver on an OS/2-based client.

ACTION: On the OS/2-based client, check that the device driver line for COMBASE.SYS is correct in the CONFIG.SYS file. This line should have been added by the installation program, and should have the following form:

DEVICE=C:\COMMSRV\DEV\COMBASE.SYS C:\COMMSRV

In this line, both instances of C:\ should be replaced by the actual path of the COMMSRV home directory.

For SNA Server 1-User, this line should be in the following form:

DEVICE=C:\COMMWS\DEV\COMBASE.SYS C:\COMMWS

In this line, both instances of C:\ should be replaced by the actual path of the COMMWS home directory.
