Message 255

Buffer Shortage

EXPLANATION: There are insufficient buffers for the component to continue processing.

ACTION: Use the COMBUF Registry variable to allocate more buffers to the component.

If increasing the size of the buffer pool does not solve the problem, an internal error may be occurring. Provide network support personnel with the event log file(s) related to SNA Server, SNA Server trace file(s) such as Data Link Control traces of the server or the affected link service, and the dump file.

For information about SNA Server log, trace, and dump files, see the Microsoft SNA Server Administration Guide. For information about COMBUF, see Appendix C, "Registry and Environment Variables for SNA Server."
