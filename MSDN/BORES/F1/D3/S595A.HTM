Message 12

(value) Negative Response from Remote System (SENSE = value)

SUBCODE INFORMATION:

X'1118' The local server (node) received an SNA negative response on an SNA session from a remote system, which indicates a protocol violation by the 3270 emulator. The SNA sense code shown gives details of the type of error. See IBM Systems Network Architecture: Formats for an explanation of the sense code.

ACTION: If the error is accompanied by functional problems, use internal tracing to determine the cause of the problem. If the problem is associated with use of a user application (HLLAPI or APPC), report it to the application developer. Otherwise, check the SNA Server configuration, especially the configuration of the 3270 LU, and see that it corresponds to the configuration of the remote system.

If the problem persists, provide network support personnel with the event log file(s) related to SNA Server, SNA Server trace file(s), and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
