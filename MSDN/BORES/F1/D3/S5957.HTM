Message 9

(value) Internal Message Sequence Error

EXPLANATION: Invalid or unexpected message received by the server (node) from another SNA Server component (for example, the emulator or link service).

SUBCODE INFORMATION:

X'1111' Received message from unrecognized component.

X'1117' Received an invalid message from a 3270 emulator.

X'1160' Unexpected response from link service.

X'1161' Unexpected message from link service.

X'1180' Unexpected message from emulator.

X'1181' Mode inconsistency: close received when not expected.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, SNA Server trace file(s), and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
