Message 76

APPC error received:

Sense data = value

TP_ID = value

Conv_ID = value

Log data value

EXPLANATION: The partner APPC TP sent an error to the local TP by issuing an [MC_]SEND_ERROR or [MC_]DEALLOCATE (ABEND) verb. The message is also logged if the partner TP failed to start. The TP_ID and Conv_ID in this message were shown in a previous message. This message is followed by one or more messages containing log data supplied by the TP. The SNA sense data provides details about the error type.

ACTION: See IBM Systems Network Architecture: Formats for an explanation of the sense data.
