Message 19

(value) Security Violation: local LU = value, remote LU = value

EXPLANATION: This message may indicate that an unauthorized remote user is attempting to access your programs from a remote LU that imitates the remote LU that you configured.

SUBCODE INFORMATION:

X'11B4' The remote LU sent incomplete security key data for session-level security or has used the wrong key.

ACTION: Check that the remote LU is configured to use session-level security and is using the correct security key, that is, the same session-level security password that is configured for the remote APPC LU in the local SNA Server configuration.
