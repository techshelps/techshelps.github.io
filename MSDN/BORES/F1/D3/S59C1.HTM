Message 188

Warning: Non-productive receive timeout on connection value through driver value. Contact with the host may have been lost.

EXPLANATION: The indicated connection, using the indicated SDLC driver, has not received anything from the host for 1 minute. The connection may have been lost. If the connection has been lost, after 3 minutes, it fails and message #23 is logged. If the host is very slow at polling its remote nodes, it is possible that the connection is not lost.

ACTION: If the message persists, contact your host administrator.
