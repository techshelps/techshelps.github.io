Message 360

(value) Unexpected error logged

EXPLANATION: An internal error. An SNA Server component has used an internal message that does not appear in SNA Server. Although logged at level 12 to ensure that it is not ignored, this does not necessarily indicate a level 12 problem.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, internal trace file(s), and any information displayed in this message. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
