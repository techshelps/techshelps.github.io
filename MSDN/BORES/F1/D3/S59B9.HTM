Message 178

The sending of the phone number from SNA Server to the modem failed; DSR steady. Check state of modem used by connection value through link service value.

EXPLANATION: The dialing process initiated by SNA Server failed; the modem is not dialing when the dial command is sent by SNA Server.

ACTION: Check the modem connections and the dial string sent to the modem. If the connections and dial string are OK, then the modem may not be functioning correctly. See the Microsoft SNA Server Administration Guide for more information on the use of modems.
