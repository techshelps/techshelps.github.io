Message 182

Connection failure, code = value

Cause Data = value

Link Role = value

Remote Node Type = value

EXPLANATION: Cause data is a list of the failure causes from the Failure subvector in the alert. Link role and remote node type are as described in the Link Connection Subsystem Configuration Data subvector in the alert.

A connection failure can be due to a station outage, rather than a link outage. If it is due to a station outage — which can be recognized by outage code values of X'80' or higher — then message #183 will also be logged.

ACTION: No action is necessary.
