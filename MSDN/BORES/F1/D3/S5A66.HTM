Message 595

Buffer pool has been extended to value kilobytes

EXPLANATION: The buffer pool emptied and had to be extended. This condition is caused by heavy usage of the node. If it occurs too frequently it may cause performance problems. The pool size will be increased in increments of 16 K until it reaches 128 K.

ACTION: The 128 K limit can be increased by using the COMBUF Registry variable. For information about COMBUF, see Appendix C, "Registry and Environment Variables for SNA Server."
