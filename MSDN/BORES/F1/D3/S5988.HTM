Message 54

User ID not supplied

EXPLANATION: A remote APPC TP has not supplied a user ID when attempting to establish a conversation with an SNA Server TP configured to use conversation-level security. An error in the remote TP or in the SNA Server configuration can cause this error to occur.

ACTION: Check whether the remote LU is configured to use conversation-level security.

This message may indicate that an unauthorized remote user is attempting to access your programs from a remote LU that imitates the remote LU that you configured.
