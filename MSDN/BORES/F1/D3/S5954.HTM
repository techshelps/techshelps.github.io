Message 6

(value) Internal Network Management Logic Error (rc = value)

EXPLANATION: The server (node) detected a run-time error or resource shortage caused by internal logic errors in SNA Server Manager, the Common Service Verbs, or the 3270 emulator.

SUBCODE INFORMATION:

X'1184' Insufficient resources for concurrent network management activity. This is probably an SNA Server Manager program logic error, but can be caused by a large number of applications simultaneously issuing TRANSFER_MS_DATA.

X'118A' LU name specified by SNA Server Manager not found.

X'118B' Unexpected message.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, SNA Server trace file(s), and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
