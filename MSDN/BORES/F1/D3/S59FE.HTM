Message 305

(value) Initialization Failure: configuration error

EXPLANATION: This is an internal error in SNA Server (but it may be caused by a corrupted configuration file).

SUBCODE INFORMATION:

X'250D' Failure in configuration initialization.

X'250E' Failure to access a configuration record.

ACTION: Check that the configuration file can be opened using SNA Server Manager, and that the X.25 link service configuration appears to be correct. If it does, provide network support personnel with the event log file(s) related to SNA Server, and the SNA Server configuration file. For information about SNA Server log files and the configuration file, see the Microsoft SNA Server Administration Guide.
