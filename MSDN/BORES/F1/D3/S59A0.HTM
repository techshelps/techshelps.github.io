Message 104

(value) Message Sequence Error

EXPLANATION: This is an internal logic error. An SNA Server component received an unexpected message from another SNA Server component.

SUBCODE INFORMATION:

X'0B43' Invalid or unexpected message received.

ACTION: Provide network support personnel with the event log file(s) related to SNA Server, internal trace file(s), and the number and description of the displayed subcode. For information about SNA Server log and trace files, see the Microsoft SNA Server Administration Guide.
