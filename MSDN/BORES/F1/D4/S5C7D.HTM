HDECOMP.EXE: Package File Decompressor

HDECOMP decompresses an SMS package file (.PKG).

Note HDECOMP requires access to the file SMSUNZIP.DLL, which resides in the \site.srv\platform.bin directory. Therefore, this command must be run from that directory, or the directory must be included in the path.

Runs On

Alpha, x86

Files Required for HDECOMP

HDECOMP.EXE
SMSUNZIP.DLL

HDECOMP Syntax

hdecomp source destination

source

The SMS package file to decompress.

destination

The directory in which the decompressed files are to be placed.

Both source and destination must be explicitly specified.

For a usage message, run the command with no arguments (/? Is not accepted).
