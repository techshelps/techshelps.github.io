SMSTRACE.EXE: SMS Log File Viewer

SMSTRACE is a GUI application that views SMS log files and gets real-time updates as lines are appended to the log files.

SMSTRACE works only with the SMS 1.2 final release, not earlier versions, because of DLL incompatibility.

You can use SMSTRACE.ATD with SMSADDIN to add SMSTRACE to the SMS Administrator's Tools menu.

Runs On

Alpha, x86

Files Required for SMSTRACE

base.dll
smsapi.dll
SMSTRACE.EXE
SMSMFC.DLL

This tool must be run from the directory in which the DLLs are located: SMS\SITE.SRV\platform.BIN.

SMSTRACE Syntax

smstrace
