Chapter 3 - Troubleshooting Remote Control

The remote control feature in SMS is used to directly monitor and control the computers in the SMS inventory database. Many problems using the remote control features of SMS are related to difficulties locating the client computer, difficulties with the remote viewing session itself, or both. The greatest number of potential problems involve failure to locate the client, and fall into three categories:

  • Inventory problems related to remote control
  • Client Remote Control Agent configuration problems
  • Network problems

The relatively few problems involving the remote viewing session itself fall into two categories: display problems and problems with passing keystroke information.

This chapter discusses known solutions to specific problems.

Note Many potential problems can be avoided by using the latest Service Pack for SMS.
