TRACER.EXE: Text File Viewer

TRACER reads, in real time, changes made to a text file.

Runs On

Alpha, x86

Files Required for TRACER

TRACER.EXE
MSVCRT20.DLL

The file MSVCRT20.DLL is included with Windows NT 3.51, but not with Windows NT 4.0. You can copy the file from the smssetup\platform directory of the Back Office Resource Kit CD-ROM.

TRACER Syntax

tracer log_file

log_file

The file to read.
