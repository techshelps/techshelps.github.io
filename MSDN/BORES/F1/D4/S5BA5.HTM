Categories of Tools

The tool descriptions are organized into the following categories:

  • System setup tools
  • Administration tools
  • Troubleshooting tools
  • Windows NT–based tools
  • Job and client tools

The tools in each category are as follows:

System Setup Tools

  • CHECKRAS.EXE: RAS Connection Checker
  • COPYVT.BAT: Visual Test 4 Installer
  • SMS_NT_INVENTORY_FILE_COLLECTOR: Inventory File Collector
  • SMGENIE.EXE: Event Log Repeater
  • SMSADDIN.EXE: Application Adder
  • SMSBOOT.EXE: Remote Windows NT Rebooter
  • TRAPMON.EXE: SNMP Trap Monitor
  • Remote Site Setup and Upgrade

  • MIFNAME.EXE: MIF File Renamer
  • NEWSVCAC.EXE New Service Account Creator
  • NLSMSG*.EXE: NLS Message Maker
  • PDFWIZ.EXE: PDF Wizard
  • PGCSET32.EXE: PGC Status Setter
  • RSAPPCTL.EXE: Shared Applications Database Resetter
  • SETCLCFG.EXE: Client Configuration Setter
  • SETGUG.EXE: Global User Group Setter
  • SKIPINV.EXE: Inventory Skipper
  • SMSMAINT.SQL: Database Maintainer
  • SQLPWD.EXE: SQL Password Changer

  • WRITEPDF.EXE: PDF Writer
