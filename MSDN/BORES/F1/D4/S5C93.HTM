NADUSER.EXE: NAD User

NADUSER is a Win16 application that, when run on a client, can indicate:

  • User global group membership
  • Network application database (NAD) applications available to a user based on the user's global group membership

Runs On

x86

Files Required for NADUSER

NADUSER.EXE
NADAPI.DLL
SMSNET.DLL

This tool must be run from the directory in which the DLLs are located: SMS\SITE.SRV\platform.BIN.

NADUSER Syntax

naduser
