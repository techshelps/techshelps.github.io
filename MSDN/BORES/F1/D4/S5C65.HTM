DUMPSEND.EXE: Display Send Request

DUMPSEND displays the contents of a send request file, which comprises these records:

Send request data

Destination site, priority, job name, job ID, and out-box location

Cancel

Specifies whether this send request is canceled

Action code

Specifies whether this send request is retrying or deleted

Address (multiple
records allowed)

Encrypted names, as well as gateway and destination addresses

Package file

The name of the package file

Instruction file

The name of the instruction file

Sender

Information about the sender's history: status, time of last event, start time, number of times a sender took this send request, and the size of this send request

Access

File access information

SSPS

Information about the remote site installation


Runs On

Alpha, x86

Files Required for DUMPSEND

DUMPSEND.EXE

DUMPSEND Syntax

dumpsend [/p] filename

/p

Pause before DUMPSEND exits. If you run DUMPSEND in a command prompt window that closes after the command is finished, using /p allows you to see DUMPSEND's output before the window closes. Then press any key to close the command prompt window.

filename

The name of the send request file whose contents you wish to display.
