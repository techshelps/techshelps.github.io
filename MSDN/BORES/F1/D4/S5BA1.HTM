Chapter 4 - Tools

Introduction

This chapter explains how to use, and in some cases install, the Microsoft® Systems Management Server tools provided in the Microsoft BackOffice™ Resource Kit.

This chapter is written for people who are already familiar with Microsoft Windows NT®. For general information about Windows NT, including keyboard equivalents to menu and mouse actions, see the Windows NT online help.

Disclaimers

Many of these tools were developed internally at Microsoft. Any use of any of these tools is at your own risk. These tools are provided "as is" without warranty of any kind. Microsoft disclaims any implied warranty of merchantability and/or fitness for a particular purpose.

These tools have not been localized: they are written and tested in English only. Using these tools with a different-language version of Windows NT might produce unpredictable results.

Many of these tools are designed to work only if you have logged on as a member of an Administrators group.

Most of these tools are for use on the SMS server, not on SMS clients. None of these tools are for use on Windows 95 or any other platforms.
