NADRESET.BAT: NAD Resetter

SMS network applications, packages, and program groups are listed in a network applications database (NAD). The NAD is replicated from the parent site to child sites. If a child site is detached from a parent site, it retains the parent site's NAD; this prevents a large-scale deletion of program groups and program items from the clients that may still have access to these items.

When you no longer need the program groups and program items from the former parent site, remove them by running NADRESET.BAT. This utility will create a directory named 10000000.NAD in SMSROOT\SITE.SRV\SITECFG.BOX, and then create the following files in that directory:

TRANSLOG.CLR
ALL.RST

Once this is done, the Applications Manager will remove any NAD items that were not created locally. This will be done in the next polling period of Applications Manager; generally the local site and logon servers will be updated within five to six minutes. Then the 10000000.NAD directory and the files it contains are deleted by the Applications Manager.

Runs On

x86

Files Required for NADRESET

NADRESET.BAT

NADRESET Syntax

nadreset smsroot

smsroot

The drive letter and directory where SMS is installed; for example, D:\SMS.

/?

Provides a usage statement.
