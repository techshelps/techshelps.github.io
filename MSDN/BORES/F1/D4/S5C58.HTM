WRITEQDF.EXE: QDF Writer and Reader

Given a query ID, WRITEQDF either writes a Query Definition File (QDF) from the information about a query in the database, or reads a specified QDF into SMS, where it can be manipulated in the SMS Administrator's Query window.

Runs On

Alpha, x86

Files Required for WRITEQDF

WRITEQDF.EXE

WRITEQDF Syntax

writeqdf server account password database {QDF | query_ID}
[/import | /export]

server

The computer running the instance of SQL Server containing database.

account

The account to use when accessing database.

password

The password for account. If the password is blank or contains spaces, it must have quotation marks around it; for example, "Harry Browne".

database

The name of the database containing the query information.

QDF

A Query Definition File (QDF) to read into SMS.

query_ID

A query ID for which to write a QDF.

/import

Specifies that a QDF is being read.

/export

Specifies that a QDF is being written. This is the default behavior.

/?

Provides a usage statement.
