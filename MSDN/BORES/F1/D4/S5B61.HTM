The SMS Registry

The following tables show the main registry keys used by SMS:

Note This document does not include all keys and values created by or used by SMS.

SMS may use registry keys created by the operating system or other applications. SMS may also create new keys or add values to existing keys created by the operating system or other applications.

In addition, if Program Group Control is being used, SMS will modify the registry as needed to run the application.

In HKEY_CLASSES_ROOT:

Primary key

Immediate subkeys

On Computer Types

Description

\SMS\

Applications, Groups, PGC_Version, SMS_INI, Tasklist.

Windows-based computers running as SMS clients.

Information about the SMS client setup and software.


In HKEY_LOCAL_MACHINE\Software:

Primary key

Immediate subkeys

On Computer Types

Description

\Microsoft\SMS\

Applications, Client Services, Components, Compression, Identification, Setup, Sites, SQL Server, and Tracing.

Applications and Client Services are present only on SMS clients; Identification and Tracing are present on both SMS clients and servers; all other keys are present only on SMS servers.

Information about the SMS system and its components.

\Microsoft \SNMP_EVENTS\

EventLog.

Windows NT computers running as SMS clients.

Information about the Event to Trap Translator.


In HKEY_LOCAL_MACHINE\System\CurrentControlSet\:

Primary key

Immediate subkeys

On Computer Types

Description

\Services\

SMS_Hierarchy_ Manager, SMS_Executive, SMS_Site_Config_ Manager, SMS_Inventory_ Agent_NT, SMS_Package_ Command_Manager_NT, SMS_Client_Config_ Manager.

SMS servers.

Information about the SMS Services.

\Services\BH\

Enum, Linkage, Parameters, Performance, Security.

Computers running Network Monitor.

Information about Network Monitor.

\Services\NMAgent\

Linkage, Parameters, Performance, Security.

Computers running the Network Monitor Agent.

Information about the Network Monitor Agent.

\Services\SNMP\

Parameters.

Computers running the Windows NT SNMP service.

Configuration information supplied to the Windows NT SNMP service. The main keys are created by Windows NT, but SMS modifies some values.


In HKEY_CURRENT_USER\SOFTWARE\MICROSOFT:

Primary key

Immediate subkeys

On Computer Types

Description

\SMS\

Applications, Login, MachineProperties, Main, MDIs, Preferences, SecurityAdmin, Sender Manager, Tables, ToolBar.

Windows NT computers running the SMS Administrator.

Information about the current user settings in the SMS Administrator.

\Network Monitor\

Configuration.

Computers running Network Monitor.

Network Monitor configuration information.

\Windows NT \CurrentVersion \Windows\

None.

Windows NT computers running as SMS clients.

This is a Windows NT key, but SMS modifies the LOAD value in this key and uses it to start the SMS client software at logon time.
