ACCTCHG.EXE: Account Changer

ACCTCHG changes the information that SMS services use to access the SQL Server database. ACCTCHG affects only the services on the specified site server. ACCTCHG does not make any changes to SQL Server itself; this must be done manually while running ACCTCHG:

Run ACCTCHG, specifying a site server. ACCTCHG shuts down that site and waits.

Using SQL Server tools, make whatever changes you wish; for example, change the SQL Server account and password, rename the SMS database, or move the database to another SQL Server.

Resume ACCTCHG by returning focus to the window running ACCTCHG and pressing any key.

ACCTCHG prompts you for the following information to be used from now on by the SMS services in the specified site:

  • SQL Server name
  • SQL Server database name
  • SQL Server account
  • SQL Server password

ACCTCHG restarts the site.

Runs On

Alpha, x86

Files Required for ACCTCHG

ACCTCHG.EXE

ACCTCHG Syntax

acctchg site_server [/v]

site_server

The name of the site server whose SMS services will be affected.

/v

Verbose mode. More output is displayed on the screen.
