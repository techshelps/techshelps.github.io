REMFPROT.EXE: F-PROT TSR Remover

If the virus-scanning software F-PROT is installed on a computer running Windows 3.11, the call to the F-PROT TSR must be removed from AUTOEXEC.BAT and WIN.INI before upgrading to Windows 95. REMFPROT removes the call to the F-PROT TSR, and a few other TSRs, from AUTOEXEC.BAT and WIN.INI.

REMFPROT returns errorlevel 1 if it fails to remove the TSR calls.

Runs On

x86

Files Required for REMFPROT

REMFPROT.EXE
