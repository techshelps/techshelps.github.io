MIFCHECK.EXE: MIF Checker

MIFCHECK is an MIF parser and syntax checker that verifies text MIF files before submitting them to an SMS system. MIFCHECK reports syntax and semantic errors, and is designed to help ISVs writing MIF files for new components for the Inventory Agent to report.

Runs On

Alpha, x86

Files Required for MIFCHECK

MIFCHECK.EXE

MIFCHECK Syntax

mifcheck [/dump] file

/dump

Dumps the parsed information to the console. If MIFCHECK is run without the /dump option, the MIF files are checked for syntax errors.

file

A MIF text file.

/?

Provides a usage statement.
