SMSBOOT.EXE: Remote Windows NT Rebooter

SMSBOOT reboots a remote Windows NT computer.

Runs On

Alpha, x86

Files Required for SMSBOOT

SMSBOOT.EXE

SMSBOOT Syntax

smsboot -n:remote_computer [-d:delay] [-f] [-m:"message"] [-r]

-n:remote_computer

Specifies the NetBIOS machine name of a remote computer to reboot. The local computer cannot be rebooted by SMSBOOT.

-d:delay

Specifies the number of seconds before shutdown of the remote computer. The default is 30.

-f

Does not wait for applications with pending changes. The default is to wait for each application with a pending change.

-m:"message"

Specifies a message to be displayed on the remote computer's console. The default is "Remote shutdown initiated."

-r

Does not prompt the remote computer's console to reboot after shutdown. The default is to prompt before rebooting.

/?

Provides a usage statement.
