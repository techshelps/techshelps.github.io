Enable Logging

For the PCM service to log trace messages, the TRACING registry key and Enabled value must be added to the registry of the client computer, as follows:

HKEY_LOCAL_MACHINE

\SOFTWARE

     \Microsoft

     \SMS

     \TRACING

Enabled 0x00000001

In addition, the following registry setting is needed to construct the proper path to the log file to be used by the PCM Service:

HKEY_LOCAL_MACHINE

\SOFTWARE

     \Microsoft

     \SMS

     \IDENTIFICATION

    

     "Installation Directory" "C:\MS\SMS" (data type REG_SZ)

Note There is a space between Installation and Directory.

Also, the SMS_PACKAGE_COMMAND_MANAGER_NT key must be created, with the Enabled and TraceFilename values set as follows:

HKEY_LOCAL_MACHINE

\SOFTWARE

     \Microsoft

     \SMS

     \TRACING

     \SMS_PACKAGE_COMMAND_MANAGER_NT

Enabled 0x00000001

     TraceFilename \MS\SMS\LOGS\pacman.log (data type REG_SZ)

After these changes have been made, restart SMS_PACKAGE_COMMAND_MANAGER_NT so it reads and uses the new settings. Because the change must be made manually at the client computer, you may prefer to enable logging on only to those computers on which you encounter problems with jobs run by the PCM service.
