PDFWIZ.EXE: PDF Wizard

PDFWIZ is a package definition file (PDF) wizard. Use PDFWIZ to make an application easier to distribute and install with SMS. Starting PDFWIZ opens a window on your console with directions for its use.

The window prompts for package information, supported platforms, whether to inventory, and so forth, and generates a PDF that can be loaded into SMS. PDFWIZ does not depend on any SMS DLLs, so PDFWIZ can run on any computer running Windows.

If the SMS Admin user interface is installed, you may prefer to use SMS Admin to create the package, and then extract the package from the database using WRITEPDF.EXE.
