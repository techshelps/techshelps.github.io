HCOMP.EXE: Package File Compressor

HCOMP compresses an SMS package file (.PKG) using the same compression algorithm used when SMS creates a package in SMS Administrator.

HDECOMP decompresses files compressed by HCOMP.

HVALCOMP validates files compressed by HCOMP.

Note HCOMP, HDECOMP, and HVALCOMP require access to the files SMSZIP.DLL and SMSUNZIP.DLL, which reside in the \site.srv\platform.bin directory. Therefore these commands must be run from that directory, or the directory must be included in the path.

Runs On

Alpha, x86

Files Required for HCOMP

HCOMP.EXE
SMSZIP.DLL

HCOMP Syntax

hcomp source destination

source

An SMS package file (.PKG) to be compressed.

destination

The name to give to the compressed SMS package file.

/?

Provides a usage statement.
