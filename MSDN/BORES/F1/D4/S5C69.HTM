ERROR32.EXE: Error Code to Message

ERROR32 displays the text of a message for the specified Windows NT (Win32 subsystem, driver, or service) error code. If no application or event-log message DLL is specified, the message text comes from the system message table.

Runs On

Alpha, x86

Files Required for ERROR32

ERROR32.EXE

ERROR32 Syntax

error32 code [:code] [source_name | message_DLL] [insertion_string[...]] [*]

code

Specifies an error code (in decimal) for a Windows NT error, or an event code for an application or system source (such as SMS). The error code is assumed to be a Win32 error unless source_name or message_DLL is specified.

:code

Specify another code to form a range; for example, 3:11 specifies the codes from 3 to 11, inclusive.

source_name

Specifies an application (such as SMS) or system name registered for the Windows NT Event Log. Cannot be used with message_DLL.

message_DLL

Specifies a full or partial path to a Windows NT event_log message DLL file (such as NETMSG.DLL). Cannot be used with source_name.

insertion_string

Specifies one or more strings to be inserted into the displayed message.

*

Displays messages for all matching languages and severities.

ERROR32 Example

The following command returns the message text from the system message table for error code 2:

error32 2

The system cannot find the file specified.
