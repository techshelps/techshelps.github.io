Deploying Windows NT 4.0 with SMS

The setup program for Windows NT 4.0 allows a completely scripted installation. In addition, the installation can be completely customized. You can even specify per-computer settings in your scripted installation. The Windows NT Workstation Resource Guide provides information on customizing your installation. The Windows NT Server Resource Kit includes the Windows NT Workstation Resource Guide as an online book.

Before you use SMS to deploy the Windows NT Workstation or Windows NT Server operating system to SMS clients, you'll need to complete the following tasks:

  • Select the sites containing client computers that you want to upgrade.
  • Create a source directory for the Windows NT Workstation and/or Windows NT Server files.
  • Import the PDF to distribute the operating system. A prepared PDF is available on the Windows NT CD-ROM.
  • Edit the Windows NT answer files as needed for your organization.
  • Use queries to identify which clients have the memory, disk space, and microprocessor required to run the new operating system. Also, query for legacy applications that may not run with the new operating system because they require access to shared memory.
  • Use the query results to create a machine group that contains all the computers you will upgrade to the new operating system.
  • Create a job to distribute the operating system.
  • Monitor the SMS job status.
  • Evaluate distribution results.
