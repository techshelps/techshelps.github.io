After Deployment

After the package has been distributed to all servers, you can delete the compressed files for the package from the following folders:

  • sms\site.srv\sender.box\tosend
  • sms\site.srv\despoolr.box\store

The names of the compressed files for a package all begin with the package ID number.

After the job status for distributing Office 97 changes to Complete, make sure the new software has been successfully installed on the client computers. To do so, you can use the following SMS Software Audit query:

[QDF]

Version=1.0

[Query Definition]

Name=MS Office 97 Pro Installation Present

Comment=

Architecture=Personal Computer

[Rule 1]

AGroup=MICROSOFT|SOFTWARE|1.0

Attribute=Software Name

Operator=9

Value=Microsoft Office 97 Professional 8.0

[Query]

Query Rule Part 1=Rule 1
