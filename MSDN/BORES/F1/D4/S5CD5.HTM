SRVINFO.EXE: Server Info

SRVINFO gathers and displays various information about a computer, such as available disk space and partition types. SRVINFO can be used on a remote computer.

Runs On

Alpha, x86 under Windows NT 4.0; x86 only under Windows NT 3.51

Files Required for SRVINFO

SRVINFO.EXE

SRVINFO Syntax

srvinfo server

server

The name of a server about which you want information.
