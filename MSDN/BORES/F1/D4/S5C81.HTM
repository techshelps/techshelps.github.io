HVALCOMP.EXE: Compressed Package File Validator

HVALCOMP validates a compressed package file (.PKG).

Note HVALCOMP requires access to the files SMSZIP.DLL and SMSUNZIP.DLL, which reside in the \site.srv\platform.bin directory. Therefore, this command must be run from that directory, or the directory must be included in the path.

Runs On

Alpha, x86

Files Required for HVALCOMP

HVALCOMP.EXE
SMSZIP.DLL
SMSUNZIP.DLL

HVALCOMP Syntax

hvalcomp source

source

The SMS package file to validate.

Run with no arguments for a usage message (/? Is not accepted).
