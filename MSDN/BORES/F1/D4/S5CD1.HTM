SHRPERM.EXE: Set Share Permissions

SHRPERM sets share permissions on Windows NT Server or Windows NT Workstation.

Runs On

Alpha, x86

Files Required for SHRPERM

SHRPERM.EXE

SHRPERM Syntax

shrperm \\machine\share user_or_group:permission[…]

\\machine\share

A universal naming convention (UNC) path to the server share or workstation share.

user_or_group

A user name or user group name.

permission

A permission setting:

r or read

Read(RX)

c or change

Change(RXWD)

f, full, or all

Full control
