SENDCODE.EXE: Send Code

SENDCODE sends a numeric code to an SMS service, causing it to perform a specific task. The following table correlates the codes with the applicable SMS services and tasks.

Code

Services

Task

128

SMS_EXECUTIVE
SMS_HIERARCHY_MANAGER
SMS_SITE_CONFIG_MANAGER

Starts memory allocation tracking

129

SMS_EXECUTIVE
SMS_HIERARCHY_MANAGER
SMS_SITE_CONFIG_MANAGER

Stops memory allocation tracking

130

SMS_EXECUTIVE
SMS_HIERARCHY_MANAGER
SMS_SITE_CONFIG_MANAGER

Dumps memory allocation data to files C:\ALERTER.MEM, APPMGR.MEM, HMAN.MEM, SCMANMEM.LOG, SCHED.MEM, SMSEXEC.MEM

131

SMS_EXECUTIVE

Dumps the SQL cache for the SMS_EXECUTIVE service and lists the status for each SQL connection in C:\SQLCACHE.LOG

132

All SMS services

Toggles the net pause support for an SMS service

133

SMS_EXECUTIVE

Rereads the registry and resets the SMS tracing (logging) settings for location and size

192

SMS_SITE_CONFIG_MANAGER

Does a watchdog cycle (verifies the current installation, checks for new logon servers, restarts any stopped components, etc.)

193

SMS_SITE_CONFIG_MANAGER

Does a watchdog cycle and writes a status file (.CT2)

194

SMS_SITE_CONFIG_MANAGER

Does a watchdog cycle as if an upgrade were happening (this should be preceded by a SENDCODE 234 to be most effective)

195

SMS_SITE_CONFIG_MANAGER

Does a watchdog cycle and writes a user group MIF

196

SMS_SITE_CONFIG_MANAGER

Does a watchdog cycle and modifies logon scripts (only if Automatically Configure Logon Scripts and Use All Detected Servers are enabled)

234

SMS_SITE_CONFIG_MANAGER

Stops and de-installs services at this site (on the site server and all logon and helper servers)

235

SMS_SITE_CONFIG_MANAGER

De-installs the site (de-installs all components and removes files and shares from the site server and all logon and helper servers)


Runs On

Alpha, x86

Files Required for SENDCODE

SENDCODE.EXE

SENDCODE Syntax

sendcode SMS_service code

SENDCODE Example

To trigger a watchdog cycle in SMS_SITE_CONFIG_MANAGER:

sendcode SMS_SITE_CONFIG_MANAGER 192
