Conventions in the Resource Guide

The following tables summarize the typographic conventions used in this book.

Convention

Meaning

bold

Characters that you type exactly as shown, including commands and switches.

Italic

Variables for which you supply a specific value. For example, SYSTEMROOT in a path name could refer to C:\WINNT35 or to some other path where Windows NT is installed on a specific computer.

ALL CAPITALS

Names of paths, directories, files, and acronyms. Unless otherwise indicated, you can use uppercase or lowercase when you type paths, directories, or file names.

Monospace type

Examples of code.

[ ] (square brackets)

In syntax descriptions, square brackets enclose optional items. If you include the item, type only the information between them, not the square brackets themselves.

{choice1 | choice2} (braces)

In syntax descriptions, braces enclose items which require a choice, such as {yes | no}. Type only one of the choices, not the braces or the dividing line.

Menu | Command (menus)

In referring to menu choices, Edit | Paste, for example, means to select the Edit menu and then choose the Paste command from it.


Keyboard Conventions

Convention

Meaning

SMALL CAPITALS

Names of keys that you press, such as TAB.

+ (a plus sign between two or more key names)

Indicates that you must press the keys at the same time; for example, ALT + TAB.

, (a comma between two or more key names)

Indicates that you must press each key consecutively; for example, ALT, F, X.


Terminology Conventions

Convention

Meaning

"Command prompt"

refers to the point on the MS-DOS screen where you enter commands, typically designated by "C:\>" (or other characters indicating the current drive and directory). To reach the command prompt in Windows NT, double-click the MS-DOS Prompt icon in Program Manager.

"Command-line application"
or
"console-based application"

refers to an application designed to run under MS-DOS or in an MS-DOS window, and which cannot run in a Windows or Windows NT window.

"Windows-based application"
or
"GUI application"

refers to an application that must run in a Windows® or Windows NT window. All such applications follow similar conventions for the arrangement of menus, dialog box styles, and keyboard and mouse use.


Procedural Conventions

  • Procedures describing how to do a task are preceded by a triangular bullet ( Û) for a single step or numbers for multiple steps.
  • An instruction to "type" information means to press the key or keys and then press the ENTER key.
  • An instruction to "select" information means to highlight directories, file names, text boxes, menu bars, and options, or to select options in a dialog box.
  • An instruction to "choose" something means to carry out a command from a menu or dialog box with mouse or keyboard actions.
  • "Note" alerts you to supplementary information.
  • "Caution" alerts you to possible data loss or breaches of security.
