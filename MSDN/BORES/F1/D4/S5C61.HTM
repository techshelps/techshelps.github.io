BUGDATA.EXE: Bug Data Collector

BUGDATA collects the SMS Registry, log files, and site control image, and puts them in a compressed file for subsequent analysis. HDECOMP.EXE can decompress the file.

Runs On

Alpha, x86

Files Required for BUGDATA

BUGDATA.EXE

BUGDATA Syntax

bugdata [\\server] [output_file]

server

The computer name of the SMS site server from which data will be gathered. The default is the local computer.

output_file

The name of the compressed output file to be written. The default is BUGDATA.SMS.
