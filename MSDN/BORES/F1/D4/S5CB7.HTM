KILL.EXE: Task Killer

KILL kills the application specified by a process ID, which can be gotten from TLIST.

Runs On

Alpha, x86

Files Required for KILL

KILL.EXE

KILL Syntax

kill [-f] {process_ID | pattern}

-f

Forces the process to die.

process_ID

The process ID of the task to be killed. Use TLIST to get a valid process ID.

pattern

A complete task name, or a regular expression pattern that matches a task name. KILL compares the pattern to task names and window titles.
