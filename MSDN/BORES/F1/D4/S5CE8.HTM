BADPENT.EXE: Bad Pentium Detector

BADPENT detects and reports clients that exhibit the FPU problem seen in some Pentium CPUs. BADPENT creates a MIF file, BADPENT.MIF, in the NOIDMIFS directory. There is package definition file, BADPENT PDF, in SMS\Clients\Win16, along with BADPENT.EXE. Use BADPENT PDF to create a package containing BADPENT to clients.

Runs On

x86

Files Required for BADPENT

BADPENT.EXE
BADPENT.PDF

BADPENT Syntax

badpent [/a] [/v]

/a

All. Reports good Pentium CPUs as well as bad ones.

/v

Verbose mode. More output is displayed on the screen.
