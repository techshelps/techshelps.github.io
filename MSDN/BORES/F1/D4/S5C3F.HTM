SETGUG.EXE: Global User Group Setter

SETGUG sets the Global User Group (GUG) reporting interval by using a PCM job. This is useful for reducing the number of User Group enumerations performed in a complex trust model.

Runs On

Alpha, x86

Files Required for SETGUG

SETGUG.EXE

SETGUG Syntax

setgug [/e {on | off}] [/i days]

/e

Specifies whether Global User Group enumeration is on or off for this site

/i

Specifies the number of days between enumeration cycles. The minimum is 1.
