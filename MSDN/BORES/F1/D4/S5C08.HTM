CHECKSMS.SQL: Status and Configuration Reporter

CHECKSMS.SQL provides a quick dump of data (resource usage, status, and configuration) about an SMS database and its SQL Server that can be read, saved, or pasted into records and reports. This information is helpful when troubleshooting SQL problems, planning for hardware upgrades, or restoring SQL Server or an SMS database.

CHECKSMS.SQL also provides a set of commented SQL commands that are commonly used when working with the SQL Server in an SMS hierarchy.

Runs On

Alpha, x86

Files Required for CHECKSMS

CHECKSMS.SQL

CHECKSMS Syntax

To run checksms.sql, open it in an ISQL/w command window and execute it.

Additional directions are in the checksms.sql file.
