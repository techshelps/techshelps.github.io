Chapter 4 Network Management

This chapter describes the support that SNA Server provides for centralized network management. The following topics are discussed:

  • Connections used for the NetView® and Response Time Monitor (RTM) facilities
  • Link alerts
  • Link statistics
  • Application-generated alerts, and alerts generated through the NVAlert and NVRunCmd services
  • RTM statistics
  • Local logging of network management data

For information about configuring SNA Server to work with NetView and/or RTM, and information about configuring the NVAlert and NVRunCmd services, see the Microsoft SNA Server Administration Guide.
