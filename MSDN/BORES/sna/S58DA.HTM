Link Alerts for SDLC and Token Ring

When a Synchronous Data Link Control (SDLC) or token ring connection fails, SNA Server logs diagnostic information, called link alerts, about the connection failure. The alerts are logged in a log file that can be viewed using the Windows NT Event Viewer. To find an SNA Server log, in the Event Viewer, from the Log menu, make sure Application is selected. Under the "Source" heading, the application log file will have a name starting with "SNA."

The link alerts are also used to build a network management vector transport (NMVT) alert, which includes probable causes and suggested actions for the alert. If a connection on the server running SNA Server has been designated for NetView, and the connection is active, the NMVT alert will be sent on that connection.

  • Descriptions of individual alerts produced by the SDLC link service.

  • Descriptions of individual alerts produced by the token ring link service.
