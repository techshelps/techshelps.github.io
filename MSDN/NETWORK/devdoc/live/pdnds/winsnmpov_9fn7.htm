WinSNMP Communications Management Concepts

[This is preliminary documentation and subject to change.]

The WinSNMP Manager API provides network transport independence for SNMP-based network management applications that execute in the Microsoft Windows NT® operating environment. The topics in this section cover concepts of communications management that apply to programming with WinSNMP.
