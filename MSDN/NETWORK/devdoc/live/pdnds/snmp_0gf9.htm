Configuring the SNMP Service

On occasion, you may need to reconfigure SNMP. In these instances, you need to know community names in your network, the trap destination for each community, and IP addresses or computer names for SNMP management hosts before you use or reconfigure SNMP services. The following topics contain information relevant to configuring the SNMP service:

  • Community names
  • Host names and IP addresses
  • Configuring SNMP security
  • Configuring SNMP agent information
