DLC Command Completion

When synchronizing thread execution with the completion of a command, you can check for command completion using any combination of the following three methods:

  • Set the ulCompletionFlag member of the LLC_CCB structure to a nonzero value and issue an LLC_READ command to read the command completion status.
  • Provide an event handle in the hCompletionEvent member. The protocol driver sets the event to the signaled state upon command completion to notify the application. Use one of the wait functions to monitor the event.
  • Poll the uchDlcStatus member.

    Windows NT: This method is unreliable and should not be used. The system can update the uchDlcStatus member before setting the other return values.
