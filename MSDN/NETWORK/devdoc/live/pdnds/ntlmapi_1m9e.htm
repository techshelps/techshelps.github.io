ADMIN_OTHER_INFO

The ADMIN_OTHER_INFO structure contains error code information in administration messages. 

typedef struct _ADMIN_OTHER_INFO {
    DWORD  alrtad_errcode;
    DWORD  alrtad_numstrings;
}ADMIN_OTHER_INFO, *PADMIN_OTHER_INFO, *LPADMIN_OTHER_INFO;

Members

alrtad_errcode
Specifies the error code for the new message in the administration message.
alrtad_numstrings
Specifies the number (0-9) of consecutive Unicode strings in the administration message.
