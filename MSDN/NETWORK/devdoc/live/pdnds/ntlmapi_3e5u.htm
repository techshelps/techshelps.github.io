SHARE_INFO_0

The SHARE_INFO_0 structure contains information about the shared resource, including name of the resource, type and permissions, and number of connections.

typedef struct _SHARE_INFO_0 {
    LPWSTR    shi0_netname;
} SHARE_INFO_0, *PSHARE_INFO_0, *LPSHARE_INFO_0;

Members

shi0_netname
A Unicode string containing the sharename of a resource.
