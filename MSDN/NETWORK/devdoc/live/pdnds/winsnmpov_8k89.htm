Software Requirements for the WinSNMP Manager API

[This is preliminary documentation and subject to change.]

A WinSNMP manager application must access the WinSNMP Manager API through the dynamic-link library WSNMP32.DLL.

The following files are required to support the functionality of the WinSNMP Manager API.

Filename Description
WSNMP.LIB WinSNMP Library
WSNMP32.DLL Provides WinSNMP interface
WINSNMP.H WinSNMP header file
SNMPTRAP.EXE Receives SNMP traps and forwards them to MGMTAPI.DLL, the Microsoft SNMP Manager API library
SNMPAPI.DLL Provides SNMP utilities
