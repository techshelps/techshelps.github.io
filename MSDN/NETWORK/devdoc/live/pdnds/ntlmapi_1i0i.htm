USER_INFO_0

The USER_INFO_0 structure specifies information about a user account. 

typedef struct _USER_INFO_0 {
    LPWSTR    usri0_name;
}USER_INFO_0, *PUSER_INFO_0, *LPUSER_INFO_0;

Members

usri0_name
Specifies the name of the user account. For the NetUserSetInfo function, this member specifies the new name of the user. The number of characters in the name cannot exceed the value of UNLEN.
