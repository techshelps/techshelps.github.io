GROUP_INFO_1002

The GROUP_INFO_1002 structure specifies information about a group, including name, identifier, and resource attributes.

typedef struct _GROUP_INFO_1002 {
     LPWSTR  grpi1002_comment;
} GROUP_INFO_1002, *PGROUP_INFO_1002, *LPGROUP_INFO_1002;

Members

grpi1002_comment
A Unicode string that contains a remark for the group. This element can be a null string. The comment can have as many as MAXCOMMENTSZ characters.
