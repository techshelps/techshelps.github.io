SHARE_INFO_1501

The SHARE_INFO_1501 structure contains information about the shared resource, including name of the resource, type and permissions, and number of connections.

typedef struct _SHARE_INFO_1501 {
    DWORD    shi1501_reserved;
    PSECURITY_DESCRIPTOR    shi1501_security_descriptor;
} SHARE_INFO_1501, *PSHARE_INFO_1501, *LPSHARE_INFO_1501;

Members

shi1501_reserved
Must be zero.
shi1501_security_descriptor
Specifies the security descriptor associated with this share.
