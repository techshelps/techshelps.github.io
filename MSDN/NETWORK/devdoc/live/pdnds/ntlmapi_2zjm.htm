SHARE_INFO_1

The SHARE_INFO_1 structure contains information about the shared resource, including name of the resource, type and permissions, and the number of connections.

typedef struct _SHARE_INFO_1 {
    LPWSTR    shi1_netname;
    DWORD     shi1_type;
    LPWSTR    shi1_remark;
} SHARE_INFO_1, *PSHARE_INFO_1, *LPSHARE_INFO_1;

Members

shi1_netname
A Unicode string containing the sharename of a resource.
shi1_type
This member can be one of following types.
Value Meaning
STYPE_DISKTREE Disk drive
STYPE_PRINTQ Print queue
STYPE_DEVICE Communication device
STYPE_IPC Interprocess Communication (IPC)

shi1_remark
Points to a Unicode string containing an optional comment about the shared resource.
