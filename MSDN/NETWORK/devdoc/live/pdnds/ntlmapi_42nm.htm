LOCALGROUP_INFO_1

The LOCALGROUP_INFO_1 structure renames the local group and specifies a comment describing the local group.

typedef struct _LOCALGROUP_INFO_1 {
    LPWSTR   lgrpi1_name;
    LPWSTR   lgrpi1_comment;
}LOCALGROUP_INFO_1, *PLOCALGROUP_INFO_1, *LPLOCALGROUP_INFO_1;

Members

lgrpi1_name
A Unicode string that specifies the local group to which a user belongs. The constant GNLEN specifies the maximum number of characters in the string. For the NetLocalGroupSetInfo function, this member is ignored
lgrpi1_comment
A Unicode string that points to a Unicode string that contains a remark for the local group. This element can be a null string. The comment can have as many as MAXCOMMENTSZ characters.
