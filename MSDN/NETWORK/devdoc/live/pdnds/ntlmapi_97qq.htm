SERVER_INFO_1538

The SERVER_INFO_1538 structure contains information about the specified server, including name, platform, type of server, and associated software. 

typedef struct _SERVER_INFO_1538 {
    BOOL    sv1538_enablefcbopens;
} SERVER_INFO_1538, *PSERVER_INFO_1538, *LPSERVER_INFO_1538;

Members

sv1538_enablefcbopens
Specifies whether several MS-DOS File Control Blocks (FCBs) are placed in a single location accessible by the server. This saves resources on the server.
