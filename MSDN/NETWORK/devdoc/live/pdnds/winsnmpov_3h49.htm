About the WinSNMP Manager API

[This is preliminary documentation and subject to change.]

The WinSNMP Manager API is an interface for the development of SNMP-enabled network management applications.

The WinSNMP Manager API provides the following features:

  • SNMP-enabling technology for network management applications
  • SNMP version 1 (SNMPv1) and SNMP version 2C (SNMPv2C) support

The WinSNMP Manager API supports 32-bit applications, and it executes in single- and multi-threaded environments. Support for the WinSNMP Manager API is available for applications that execute in the Windows NT® operating environment, version 4.0 and later.
