REPL_IDIR_INFO_0

The REPL_IDIR_INFO_0 structure specifies subdirectories in the main import directory of the replicator.

typedef struct _REPL_IDIR_INFO_0 {
    LPWSTR         rpid0_dirname;
} REPL_IDIR_INFO_0, *PREPL_IDIR_INFO_0, *LPREPL_IDIR_INFO_0;

Members

rpid0_dirname
Specifies the name of a particular subdirectory of the main import directory.
