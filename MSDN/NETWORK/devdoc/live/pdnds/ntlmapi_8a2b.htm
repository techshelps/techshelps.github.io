Net Function Groups

Net functions use the standard-call calling convention and return a DWORD status value. The following modifiers are used in the function declarations:

#define NET_API_STATUS   DWORD
#define NET_API_FUNCTION __stdcall

The Net functions can be divided into the following groups:

  • Access functions
  • Alert functions
  • ApiBuffer functions
  • Audit functions
  • Configuration functions
  • Directory Service (Ds) functions
  • Distributed File System (Dfs) functions
  • Error Logging functions
  • File functions
  • Group functions
  • Local Group functions
  • Message functions
  • Remote Utility functions
  • Replicator functions
  • Schedule functions
  • Server functions
  • Service functions
  • Session functions
  • Share functions
  • Statistics functions
  • Use functions
  • User functions
  • Workstation and workstation user functions
