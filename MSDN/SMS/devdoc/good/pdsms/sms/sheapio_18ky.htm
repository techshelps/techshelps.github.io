Setting Scalars for a New Program Item Folder

When your application creates a program item (F_PROGITEM folder), it must set the following scalars:

Scalar name Data type Requirement
Description String Required.
Command line String Required.
Run minimized Integer Required.
Platforms Integer Required.
Configure script file String Required.
Search local flag Integer Required.
Drive mode Integer Required.
Drive letter String If Drive mode is set to PI_DRIVEMODE_SPECIFIC_LETTER, this scalar is required.
Registry name Integer Required.
Display Icon in Program Group Integer Required.
Icon Binary. Required.
