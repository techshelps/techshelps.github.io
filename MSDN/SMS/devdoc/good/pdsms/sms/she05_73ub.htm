Supported Context Names

As stated previously, the context that applies to each of the ISV applications is the internal class name of the active SMS MDI window. The currently supported contexts are listed below:

Context Description
General Any child window or none at all
SMSMachine Machine Properties window
SMSSites Sites window
SMSSiteGroups Site Groups window
SMSMachineGroups Machine Groups window
SMSQueryResults Query Results window
SMSPackages Packages window
SMSJobs Jobs window
SMSQueries Queries window
SMSAlerts Alerts window
SMSTrapWnd SNMP Traps window
SMSArchWnd Events window
SMSProgramGroups Program Groups window
