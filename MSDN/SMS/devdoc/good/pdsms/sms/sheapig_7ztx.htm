The SMS Demonstration Database

The Platform SDK and the Microsoft Developer Network (MSDN) CD-ROMs include SMS demonstration site data in a separate SQL database. It is designed to be installed as a separate SMS database so that it does not override any previous SMS operations you have done. It also allows you to easily look at both pre-configured data and your real network by logging out and logging back into SMS with a different database name. The sections that follow describe how to install the SMS demonstration database.
