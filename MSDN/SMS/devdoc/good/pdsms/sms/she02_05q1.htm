Extending the Inventory

To help you store and manage inventory information, Systems Management Server provides the following predefined architectures:

  • Personal Computer
  • SMSEvent
  • PackageLocation
  • JobDetails
  • UserGroups

To extend the inventory, you can also define custom architectures.

By using ASCII text files in the Management Information Format (MIF), you can add objects with new (custom) architectures to the SMS database, add new objects with existing architectures, and add new groups and attributes to existing objects. For example, you may want to add "Printers" to your list of architectures, add a new workstation with a Personal Computer architecture, or include a new device driver group in the inventory of a workstation.
