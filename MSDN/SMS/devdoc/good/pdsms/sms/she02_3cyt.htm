Verifying the Syntax of a MIF File

By using the MIFCHECK utility, you can check the syntax of a MIF file. The MIFCHECK utility runs the same MIF parser used by the SMS components. Because the same parsing engine is used by both the Inventory Processor and MIFCHECK, any MIF accepted by MIFCHECK will also be accepted by the SMS system. MIFCHECK verifies that the specified MIF files adhere to the syntax of the DMI Specification version 4.5—with the exceptions noted in the previous section.

The MIFCHECK utility is provided on the Microsoft Systems Management Server CD-ROM in the platform subdirectory for the processor type of your computer in the \PSSTOOLS directory (for example, the \PSSTOOLS\X86 directory if you want to run MIFCHECK on a computer with an x86 processor).
