Filter Types

There are eight types of filters:

Type Tag Description
ARCHITECTURE_FILTER Architecture filter Specifies that machine folders of the specified architectures are retrieved in a site, site group, or machine container.
ATTRIBUTE_FILTER Attribute filter Specifies which attributes (represented as scalars within group folders) are retrieved for machine folders in a site, site group, or machine container.
GROUP_FILTER Group filter Specifies which group folders are retrieved for machine folders in a site, site group, machine, or machine group container.
JOB_FILTER Job filter Specifies which job folders are retrieved in a job container.
MACHINE_FILTER Machine filter Specifies which machine folders are retrieved in a site, site group, or machine container.
PACKAGE_FILTER Package filter Specifies which package folders are retrieved in a package container.
SITE_FILTER Site filter Specifies which site folders are retrieved in a site container.
SITELIMIT_FILTER SiteLimit Filter Specifies that machine folders within the specified site and/or domain are retrieved in a machine container.

The types of filters that your application can set are dependent on the container. For example, filters of type ARCHITECTURE_FILTER, MACHINE_FILTER, GROUP_FILTER, and ATTRIBUTE_FILTER can be set on a machine container.
