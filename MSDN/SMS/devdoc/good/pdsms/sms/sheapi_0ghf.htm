Expression Token Operators

An expression token contains a relational operator (such as "is equal to" or "is like") that defines how the expression's value should be compared with the specified attribute within the objects stored in the inventory. The relational operators are defined in QRYCODES.H.

The types of relational operators available depend on the data type of the attribute. The data types are numerical, string, and date/time.
