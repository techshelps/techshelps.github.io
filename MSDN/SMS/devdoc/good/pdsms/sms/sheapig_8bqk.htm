RESDLL.DLL

The code in this directory demonstrates how to override and extend graphics in the SMS Administrator. In this example DLL, the default SMS icons for Packages, Macintosh workstations, PC Servers, User Groups, Macintosh Operating System, Mouse, and any non-Macintosh or non-PC system are replaced by the "eye" icon.

The techniques demonstrated in this sample application are documented in Overriding and Extending Graphics in the SMS Administrator.
