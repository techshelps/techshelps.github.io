Filter Types

Type Tag Description
ARCHITECTURE_FILTER Architecture filter Specifies that machine folders of the specified architectures are retrieved in a site, site group, or machine container.
ATTRIBUTE_FILTER Attribute filter Specifies the attributes (represented as scalars within group folders) that are retrieved for machine folders in a site, site group, or machine container.
GROUP_FILTER Group filter Specifies the group folders that are retrieved for machine folders in a site, site group, machine, or machine group container.
JOB_FILTER Job filter Specifies the job folders that are retrieved in a job container.
MACHINE_FILTER Machine filter Specifies the machine folders that are retrieved in a site, site group, or machine container.
PACKAGE_FILTER Package filter Specifies the package folders that are retrieved in a package container.
SITE_FILTER Site filter Specifies the site folders that are retrieved in a site container.
SITELIMIT_FILTER SiteLimit filter Specifies that machine folders within the specified site and/or domain are retrieved in a machine container.
