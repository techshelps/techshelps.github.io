Naming the prodspec.INI File

The name of the prodspec.INI file should be as unique as possible. A suggested way to do this is to use the first letters of the product’s name. For example:

Product File name
Microsoft Windows NT NTSPEC.INI
Microsoft SQL Server SQLSPEC.INI
Microsoft Exchange EXCHSPEC.INI
Microsoft SNA Server SNASPEC.INI

Note that you do not need to use “SPEC” as the last four letters of the name. Any name can be chosen.

Additionally, you should choose a file name that conforms to MS-DOS 8.3 file naming conventions to ensure that the package can be installed successfully on 8.3 file systems.
