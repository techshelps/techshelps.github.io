Desktop Management Interface (DMI) Compatibility

In general, the SMS system can use any MIF file that complies with version 4.5 of the Desktop Management Interface (DMI) specification. However, SMS treats some aspects of the MIF syntax in a way that is unique to SMS. The full DMI 4.5 syntax is accepted by the SMS MIF parser; however, some fields are not used. For additional information about the MIF format, refer to the Desktop Management Interface Specification version 4.5 (DMI45.DOC) provided in the \MSSDK\DOC subdirectory of the Platform SDK CD-ROM.
