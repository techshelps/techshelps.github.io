Steps for Creating a Sample Parser DLL

This section shows the steps necessary to create a parser. The following example uses the Finger protocol to show the steps.

    To create a parser DLL
  1. Set up your build environment. For this example, build environment requirements are:

    Windows NT

    Microsoft® Visual C++®

    Win32® SDK

  2. Copy the REMAPI files provided in the \MSSDK\SAMPLES\SYSMGMT\SMS\NETMON\REMAPI subdirectory of the Platform SDK CD-ROM.
  3. Rename REMAPI.* to FINGER.*.
  4. Edit FINGER.C and change all references of "RemAPI" to "Finger."
  5. Edit FINGER.H and change all references of "RemAPI" to "Finger."

    Note FINGER.H is where you make your data structures within the #pragma pack(1) lines. You need this pragma because you will get DWORD aligned unless you tell the compiler to pack.

  6. Edit the makefile and change all references of "RemAPI" to "Finger."
  7. Edit the FINGER.DEF file and change all references of "RemAPI" to "Finger."
  8. Save all files and do an NMAKE. Your output should look like this:
