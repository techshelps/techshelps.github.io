Remove Package From Server Jobs

Remove Package From Server jobs do not remove packages from clients; that is done by using deinstall options with the Run Command on Workstation job. The Remove Package From Server job is used to remove packages from SMS site servers, distribution servers, and servers that have had a package installed using a Share Package on Server job.

There is no specific PDF for this type of job. Any package in SMS can be the basis of a Remove Package From Server job.
