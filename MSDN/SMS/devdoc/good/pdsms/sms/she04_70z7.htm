SMSVIEW Samples

With the SMSVIEW program, you can create database views on a site database. Using the Microsoft Desktop ODBC Database Driver, you can use these views in other applications (such as Microsoft Word for Windows) to view the computer inventory and generate reports.

SMSVIEW is provided with the Microsoft Systems Management Server product.

The SMS SDK provides samples for viewing, extracting, and using the computer inventory and other information in the site database. Samples are provided for the following applications:

  • Microsoft Word for Windows version 6.0
  • Microsoft Excel version 5.0
  • Microsoft Access version 2.0

The samples are provided on the Microsoft® Platform SDK CD-ROM in the \MSSDK\SAMPLES\SYSMGMT\SMS\SMSVIEW directory.
