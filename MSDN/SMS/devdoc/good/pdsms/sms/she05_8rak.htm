Using the Application Registration Tool

To assist ISVs with integrating their applications into the Registry and creating the structure described previously, SMS provides the SMSADDIN.EXE tool. SMSADDIN.EXE is a Windows-based executable file that you use in two ways to register applications:

  • Interactively, to create and test Add-in Tool Definition (ATD) files, which are ASCII text files containing the Registry data for the application.
  • As part of the application's setup program, to register the application for launching from the SMS Administrator console.
