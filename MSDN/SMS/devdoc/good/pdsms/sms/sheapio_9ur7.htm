Machine Group Folder Scalars

A machine group folder contains a set of scalars that store information about the machine group. Machine group folder scalars have read-only access. Therefore, an application cannot create a machine group folder.

A machine group folder has the following scalars:

Name
String. Read-only access. Name of the machine group. This is the alias you can use to refer to the set of computers defined in the machine group. The name is not case-sensitive. Spaces are allowed.
Comment
String. Read-only access. Text that contains any comments used to describe the machine group.
