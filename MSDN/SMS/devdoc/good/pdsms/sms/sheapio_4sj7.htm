Inventory Rules Folder Scalars

An inventory rules folder contains a single scalar that stores information about the package's availablity for inventory. Inventory rules folder scalars have both create and read access. Therefore, an application can create an inventory rules folder.

An inventory rules folder has the following scalar:

Inventory this package
Integer. Create access. A flag that specifies that the package be included in the SMS software inventory, using the rule defined by the tokens in the inventory rules folder. TRUE specifies that the package is included in the inventory. FALSE specifies that the package is not included in the inventory.

The Inventory this package scalar is used by the SMS API engine to determine whether to set the Inventory package scalar for the package folder that contains the inventory rule folder. Note that the SMS API engine will set the Inventory package scalar for a package folder to TRUE only if the following conditions are met:

  • Inventory this package scalar is set to TRUE.
  • The inventory rules folder contains at least one expression token.
