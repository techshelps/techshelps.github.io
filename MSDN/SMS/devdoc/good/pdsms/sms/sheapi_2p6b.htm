Scalar Functions

The scalar functions enable an application to read and set the value of a scalar. Currently, scalars can be set only on folders that support creation and on machine filters that will be committed to the site database as persistent filters in a filter container.

SmsGetNextScalar

SmsGetScalarByName

SmsGetScalarCount

SmsSetScalar

Note that after scalars have been retrieved by using SmsGetNextScalar, the scalar list for a folder or persistent filter can be rewound by using SmsRewind. To get information (such as name, data type, access rights, and so on) about the scalars within a folder, use the SmsDescribeFolder function.
