Example PDF Inventory Section

The following is an example of a PDF inventory section:

[Setup Package for Inventory]
InventoryThisPackage=TRUE
Detection Rule Part 1=File 1
Detection Rule Part 2=AND
Detection Rule Part 3=File 2

[File 1]
FILE=ntoskrnl.exe
COLLECT=FALSE

[File 2]
FILE=ntspec.ini
COLLECT=FALSE
;Checksum details for NT 4.0 Workstation ntspec.ini
CRC=0, 267, 38501

In this example, SMS inventories the product if the NTSPEC.INI file with a file length of 267 bytes and CRC 38501 is present and the file NTOSKRNL.EXE is also present on the inventoried machine. The NTOSKRNL.EXE file was chosen as a file that is expected to exist only on Microsoft Windows NT, but no CRC is specified because this file is likely to change until the product is finalized. Using two files in this way provides a high probability of accurate package recognition.
