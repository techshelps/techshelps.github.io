Status MIF File Creator (ISMIF)

SMS provides a tool, ISMIF, that writes a generic status MIF file from information given through a command line and/or a dialog box (ISMIF16 only). Two versions of the tool are provided: ISMIF16.EXE and ISMIF32.EXE.

ISMIF16 is a 16-bit application that opens a dialog box where you can type any information not specified on the command line; items specified on the command line appear in the corresponding fields of the dialog box.

ISMIF32 is a 32-bit application that has no dialog box; any information must be specified on its command line.
