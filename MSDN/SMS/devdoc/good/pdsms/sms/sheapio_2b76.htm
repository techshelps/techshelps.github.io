Domain Folder (F_DOMAIN)

A site contains one or more domains. Therefore, a site folder contains one or more domain folders that represent the site's domains. Domain folder scalars have read-only access. Therefore, an application cannot set scalar values for a domain folder.

Property Value Description
Tag "Domain" String. A description of the folder type.
Folder type F_DOMAIN Integer. Folder type.
ID domain. For example, "REDDOM". String. The name of the domain.
Active filters None Integer. Filters that are directly activated on the folder.
Immediate subfolders F_MACHINE Integer. Types of subfolders that the folder type can contain.
Number of scalars 1 Integer. Number of scalars for the folder type.
