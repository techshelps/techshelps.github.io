Machine Group Folder (F_MACHINEGROUP)

Machine group folders contain machine folders which, in turn, contain group folders. Machine group folders can also contain other machine group folders.

A machine group folder has the following properties:

Property Value Description
Tag "Machine Group" String. A description of the folder type.
Folder type F_MACHINEGROUP Integer. Folder type.
ID Machine group ID. For example, "MyGroup". String. The name of the machine group.
Active filters None
Immediate subfolders F_MACHINEGROUP

F_MACHINE

 Integer. Types of subfolders that the folder type can contain.
Number of scalars 2 Integer. Number of scalars for the folder type.
