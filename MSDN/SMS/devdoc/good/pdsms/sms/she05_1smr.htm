Context Keys

Each application key must contain at least one context key. General context applications are always available on the Tools menu regardless of the current context. Limited context applications are restricted to a particular context. If the General context key is specified for an application, all other context keys that may be listed for the application are ignored. If no context keys are found for an application, then the application's menu item will not appear under any context.

Each context key contains three additional values that pertain to the application:

PresentRule

EnableRule

Arguments
