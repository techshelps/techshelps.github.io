Date and Time Values

If the attribute used for an expression uses a date/time data type, you must specify a date that the SMS API engine can use to evaluate the expression. You must specify the value according to the units specified by the date/time operator. For example, if you used the "year is after" operator, you would specify the year by using four digits (for example, 1994). The dates and times must be entered using the date and time formats supported by Microsoft SQL Server. For more information about date and time formats for Microsoft SQL Server, see the Microsoft SQL Server documentation.
