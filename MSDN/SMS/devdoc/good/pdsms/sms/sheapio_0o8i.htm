Site Folder (F_SITE)

A site folder represents a site in the SMS system. A site folder has the following properties:

Property Value Description
Tag "Site" String. A description of the folder type.
Folder type F_SITE Integer. Folder type.
ID sitecode. For example, "RTM". String. The site code. A three-character code used by the SMS system to identify the site. The site code is specified when the site is created, and cannot be changed.
Active filters None. Integer. Filters that are directly activated on the folder. These filter types are used to retrieve the subfolders in the folder.
Immediate subfolders F_SITE

F_DOMAIN

 Integer. Type of subfolders that the folder contains directly beneath it in hierarchy.
Number of scalars 5 Integer. Number of scalars that the folder contains.

A site folder can contain subfolders. A site can contain one or more domains. The domains are represented as domain folders within a site folder. A primary site can also contain other sites—a primary site database contains the current site (the primary site that uses the current database as a site database) and all sites beneath it in the site hierarchy. The child sites are represented as site folders within a site folder.

A site folder also has scalars that store information about the site.
