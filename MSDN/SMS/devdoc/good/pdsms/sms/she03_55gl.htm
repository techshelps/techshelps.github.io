Adding Inventory Rules to the PDF File

After the prodspec.INI file has been created, you should add inventory rules to the PDF file that specify the name of the prodspec.INI file and its CRC value (or some other unique characteristic). You can generate file CRCs using the create package function in the SMS Administrator console or with the SMS Resource Kit utilities. If you use the CRC to define the inventory rule, then very little of the prodspec.INI file needs to change when a new version of the product ships.

You can increase the accuracy of package recognition by specifying additional files in the PDF inventory section which are likely to exist only in the target package.

SMS identifies the package by searching for the file or files specified in the PDF inventory section and, if found, comparing the files’ characteristics with those specified in the PDF file.

For more information, see Example PDF Inventory Section.
