Folder Types

There are 14 types of folders:

F_DOMAIN

F_GROUP

F_INSTALLJOB

F_INVENTORY

F_MACHINE

F_MACHINEGROUP

F_PACKAGE

F_PROGITEM

F_REMPKGJOB

F_SITE

F_SITEGROUP

F_SRVINSTALL

F_SYSTEMJOB

F_WCL

Each type of folder represents a specific type of object. For example, F_SITE folders represent sites. A folder can contain scalars, subfolders, or both. The scalars and subfolders represent properties of the object.

In addition, queries are represented as persistent machine filters in a filter container. For more information about persistent filters and filter containers, see Filter Containers.
