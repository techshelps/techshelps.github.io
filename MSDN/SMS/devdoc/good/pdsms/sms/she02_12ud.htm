How MIF Files are Used to Maintain Inventory in an SMS Database

To add or maintain items in an SMS database, the SMS system uses data in the Management Information Format (MIF). For example, computer inventory, events, package locations, job status, and user groups are all reported to the SMS database as MIF files.

This section provides the following topics:

  • Overview of the Inventory Collection Process
  • How Inventory History is Used to Maintain the SMS Inventory
  • Desktop Management Interface (DMI) Compatibility
  • How SMS Processes MIF Files

For additional information about the MIF format, refer to the Desktop Management Interface Specification version 4.5 (DMI45.DOC) provided in the \MSSDK\DOC subdirectory of the Microsoft® Platform SDK CD-ROM.
