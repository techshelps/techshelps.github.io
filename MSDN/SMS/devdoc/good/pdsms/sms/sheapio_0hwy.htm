Remove Package From Server Job Folder (F_REMPKGJOB)

A remove package job folder (F_REMPKGJOB) represents a Remove Package From Server job.

A Remove Package From Server job removes a package that has been installed on servers. A Remove Package From Server job removes the compressed version of the package stored at the receiving site server(s) and the uncompressed package files at the distribution servers. It does not, however, remove the package source directory stored on the originating server.

Using the SMS API, your application can perform the following actions:

  • View existing Remove Package From Server jobs
  • Delete existing Remove Package From Server jobs
  • Create new Remove Package From Server jobs

A remove package job folder has the following properties:

Property Value Description
Tag "Remove package job" String. A description of the folder type.
Folder type F_REMPKGJOB Integer. Folder type.
ID jobID. For example, "RTM00001". String. An eight-character identifier that the SMS system assigns to the job. Note that the first three characters are the site code for the site where the job was created.
Active filters None
Immediate subfolders None Integer. Types of subfolders that the folder type can contain.
Number of scalars 20 Integer. Number of scalars for the folder type.
