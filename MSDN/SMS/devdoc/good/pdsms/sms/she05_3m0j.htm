Testing Registry Settings

    To test Registry settings
  1. Restart the SMS Administrator console. Then open the window in which you want to work and display the Tools menu.

    Depending on the PresentRule and EnableRule settings, applications defined in the General Context will appear in the Tools menu above the separator line; applications defined in a specific context will appear below the separator line when the MDI window corresponding to that context is active.

  2. Choose the application you want to launch.

    SMS forms a command line from the application's Registry information and passes the command to the underlying operating system for execution. If SMS cannot form the command line, the launch fails and a message box appears describing the problem.

    If an application fails to execute for any reason, a message box appears with information that allows you to determine the cause of the failure.
