System Requirements

To build and run applications that use the SMS API, your computer must meet the following requirements:

  • Microsoft Windows NT version 3.5 and later.
  • Systems Management Server version 1.1 and later.

Note that applications using the SMS API are compatible with primary site databases for SMS version 1.1 and later. Sites running SMS version 1.0 are not supported.

For information about requirements for running applications using the SMS API, see Installing Your Application.

For information about installing the SMS API components, see Installing the SMS API Components.
