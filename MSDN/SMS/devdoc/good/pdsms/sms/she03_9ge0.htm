ismif32 Syntax

ismif32  [-f filename]  [-c manufacturer]  [-p product]  [-v version]  [-l locale]  [-n serial_number]  [-d description]  [-s success]  [-q]  [-?]
-f filename
Specifies the name of the MIF file that will be created. A .MIF extension is automatically appended. The default name is SETUP.MIF.
-c, -p, -v, -l, -n, -d
Each of these options can specify a text string. Strings containing spaces must be surrounded by quotation marks. The description string (-d) must be localized.
-s success
0 or 1. 1 means success.
-q
Quiet mode. No output is displayed on the screen.
-?
Provides a usage statement.
