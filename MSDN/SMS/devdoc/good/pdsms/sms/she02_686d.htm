Maintaining Objects from a Custom Architecture

To maintain the properties of objects from a custom architecture, you make changes directly to the object's MIF file. The SMS Inventory Agent program detects the changes when it reads the MIF file, and the SMS system updates the object's inventory in the database.

Note When making changes to a MIF file, make sure to specify values for all of the attributes defined in the previous MIF file; otherwise the values for those attributes not specified will be set to NULL.
