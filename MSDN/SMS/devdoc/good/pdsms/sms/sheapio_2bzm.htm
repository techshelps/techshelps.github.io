Site Group Folder (F_SITEGROUP)

Site group folders contain site folders or other site group folders.

A site group folder has the following properties:

Property Value Description
Tag "Site Group" String. A description of the folder type.
Folder type F_SITEGROUP Integer. Folder type.
ID Site group ID. For example, "MyGroup". String. The name of the site group.
Active filters None
Immediate subfolders F_SITEGROUP

F_SITE

 Integer. Types of subfolders that the folder type can contain.
Number of scalars 2 Integer. Number of scalars for the folder type.
