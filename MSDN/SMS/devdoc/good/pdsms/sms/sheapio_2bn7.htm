Site Group Folder Scalars

A site group folder contains a set of scalars that store information about the site group. Site group folder scalars have read access. Therefore, an application cannot create a site group folder.

A site group folder has the following scalars:

Name
String. Read-only access. Name of the site group. This is the alias you can use to refer to the set of sites defined in the site group. The name is not case-sensitive. Spaces are allowed.
Comment
String. Read-only access. Text that contains any comments used to describe the site group.
