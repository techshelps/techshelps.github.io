Workstation Command Line Folder (F_WCL)

A workstation command line folder contains the properties of a workstation command line.

Workstation command lines are used for Run Command On Workstation jobs. A Run Command On Workstation job runs a specified package command line on individual computers. A command line can be used to install a package (by running a script or starting an application's setup program) or to run a program (such as a virus scan utility). You can define multiple command lines for a package. When you create a Run Command On Workstation job for a package, you select the specific command line you want to use.

Using the SMS API, your application can perform the following actions:

  • View existing workstation command lines within a package
  • Create workstation command lines within a new package

A workstation command line folder has the following properties:

Property Value Description
Tag "Workstation Command Line" String. A description of the folder type.
Folder type F_WCL Integer. Folder type.
ID CommandName. For example, "Complete". String. The name for the workstation command line. The Name scalar is used as the ID. This scalar has a maximum length of 255 characters. Within a specific package folder, the Name scalar must be unique.
Active Filters None
Immediate subfolders None Integer. Types of subfolders that the folder type can contain.
Number of scalars 4 Integer. Number of scalars for the folder type.
