Numerical Values

If the attribute used for an expression token uses a numerical data type, you must specify a numeral that the SMS API engine can use to evaluate the expression.

Numerical entries can be specified as decimal (specify the decimal value) or hexadecimal (start the hexadecimal number with 0x).

If you type a nonnumerical character (such as z or !), SMS will not be able to evaluate the expression and complete the query.
