Container Types

There are six types of containers:

C_JOB

C_MACHINE

C_MACHINEGROUP

C_PACKAGE

C_SITE

C_SITEGROUP

Each type of container supports specific types of objects. For example, the C_SITE container contains sites, which are represented as site folders. The site folders contain scalars and subfolders that represent the contents of a site (namely, domains and computers).

In addition, queries are represented as persistent machine filters in a filter container. For more information about persistent filters and filter containers, see Filter Containers.
