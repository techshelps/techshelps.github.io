Console Application Integration

SMS provides support for launching ISV applications from a Tools menu in the SMS Administrator console in a context-specific manner. (The term context refers to the internal class name of the active SMS multiple document interface (MDI) window.) Launching an application from the SMS Administrator console requires registering information about the application in the Windows NT Registry.

This section provides an overview of the ISV application registration and launch process, a detailed description of the required Registry information, and instructions on using the application registration tool provided by SMS.
