Site Group Container (C_SITEGROUP)

A site group container is used to gain access to site groups in the current site's site database. The site group container is analogous to the Site Groups window in the SMS Administrator. Thus, a site group container contains, at the topmost level, site group folders. Site group folders contain site folders which, in turn, contain site folders and/or domain folders. The domain folders contain machine folders, group folders, and so on. Site group folders can also contain other site group folders.

A site group serves as an alias for a set of sites. Site groups can be used to specify the site limits for Run Command On Workstation, Share Package On Server, and Remove Package From Server jobs.

A site group container has the following properties:

Property Value Description
Tag "SiteGroup container" String. A description of the container type.
Container type C_SITEGROUP Integer. Container type.
Applied filters ARCHITECTURE_FILTER

MACHINE_FILTER

ATTRIBUTE_FILTER

GROUP_FILTER

 Integer. Filters that can be applied to the container type.
Active filters None. Integer. Filters that are directly activated on the container. These filter types are used to retrieve the top-level folders in the container.
Immediate subfolders F_SITEGROUP Integer. Types of subfolders that the container type can contain.
