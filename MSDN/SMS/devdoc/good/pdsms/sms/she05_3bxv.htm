Updating Existing ATD Files

    To update an existing ATD file
  1. Run SMSADDIN.EXE.
  2. Click the Open button on the toolbar and type the name of the ATD file to be modified in the dialog box that appears.
  3. Change the settings in the SMS Add-In Registration window as desired. Then choose the Register button (check mark) on the toolbar.
  4. Click the Save button on the toolbar to save the new settings in the ATD file.
  5. To make the change effective, you must restart the SMS Administrator console.
