Microsoft® JScript™
input Property		  Language Reference 
Version 3 

See Also                  Applies To

Description
Contains the string against which a search was performed. Read-only.
Syntax
RegExp.input
Remarks
The value of input property is modified any time the searched string is changed.

The following example illustrates the use of the input property:

function inputDemo()
{
  var s;
  var re = new RegExp("d(b+)(d)","ig");
  var str = "cdbBdbsbdbdz";
  var arr = re.exec(str);
  s = "The string used for the match was " + RegExp.input; 
  return(s);
}
