Properties Tab (Event Builder Dialog Box)

Displays and allows editing of the properties of the event being added or edited. The Name property is required.

Tab Options

Name

The name of the event.

Arguments

The arguments of the event procedure.

  • If this box is blank, the event will have no arguments.

  • To add an argument, click the "+"; this opens the Add Argument dialog box.

  • To delete an argument, select it and click the "x" button.

  • When an argument is selected, the up arrow and down arrow keys move the argument up or down the list.
