Add Argument Dialog Box

Allows adding an argument to a method or event.

Dialog Box Options

Name

The name of the argument.

By Val (Event Builder and Method Builder only)

When selected, allows you to pass the argument by value.

Data Type

The data type of the property. Displays a list of all possible data types and known classes and collections of the project and allows you to type the name of a data type not in the list. Any data types that you add to the list remain in the list for the project.

Array (Event Builder and Method Builder only)

Adds ( ) to make the argument an array. Available only for data types that have array arguments. Not available if Optional is selected.

Optional (Method Builder only)

Allows you to add additional parameters or arguments.

Default Value (Method Builder only)

Allows you to use the default value of the argument.
