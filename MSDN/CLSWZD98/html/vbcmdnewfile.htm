New Commands (Class Builder File Menu)

Allows adding a new class or collection, or a new member to the currently selected class or collection.

Class Command

Adds a new class to the currently selected class or collection, and displays the Class Module Builder dialog box.

Collection Command

Adds a new collection to the currently selected class or collection, and displays the Collection Builder dialog box.

Toolbar shortcut: 

Property Command

Adds a new property to the currently selected class or collection, and displays the Property Builder dialog box.

Toolbar shortcut: 

Method Command

Adds a new method to the currently selected class or collection, and displays the Method Builder dialog box.

Toolbar shortcut: 

Event Command

Adds a new event to the currently selected class or collection, and displays the Event Builder dialog box.
