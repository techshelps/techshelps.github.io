Events Tab (Class Builder Utility Dialog Box)

Displays all the events of the currently selected class or collection, as well as the arguments and their data types. The list of events can be sorted by clicking on the column header of the column to sort by. Events can be added with the New command on the File menu or by clicking the New Event button on the toolbar; this action displays the Event Builder dialog box. Events can be edited by clicking the Events command on the Edit menu or by right-clicking the event and choosing Properties. Events can also be renamed, cut, copied, and deleted.
