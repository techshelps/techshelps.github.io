All Tab (Class Builder Utility)

Displays all the members (properties, methods, events, and enums) of the currently selected class or collection, as well as their data type, arguments, property declarations, and values. The list can be sorted by clicking on the column header of the column to sort by. Properties, methods or events, and enums can be added, renamed, cut, copied, and deleted. The All tab combines the information from the Properties tab, the Methods tab, the Events tab, and Enums into a single display.
