Properties Tab (Class Builder Utility)

Displays all the properties of the currently selected class or collection, as well as its data type and property declaration. The list of properties can be sorted by clicking on the column header of the column to sort by. Properties can be added with the New command on the File menu or by clicking the New Property button on the toolbar; this action displays the Property Builder dialog box. A property can be edited by clicking the Properties command on the Edit menu or by right-clicking the property and choosing Properties. Properties can also be renamed, cut, copied, and deleted.
