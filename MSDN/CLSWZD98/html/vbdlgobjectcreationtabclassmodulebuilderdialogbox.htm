Object Creation Tab (Class Module and Collection Builder Dialog Boxes)

Displays and allows editing of the object creation options of the class module or collection being added or edited.

Tab Options

Class_Initialize Event

The object is created when the parent is initialized. This option slows down the startup of the parent, but does not slow down the accessing of the object property.

Property Get on first access

The object is not created until the object property Get procedure is used. This option slows down all object property accesses including the initial one, but does not slow down the initialization of the parent.
