Name and Password Required

The operating system was unable to validate your identity. For example, this alert will appear if your operating system is Microsoft® Windows NT® and it is set to use Basic Authentication only or if you are using both Basic Authentication and NT Challenge/Response and NT Challenge/Response failed.

To identify yourself to the operating system:

  • Verify the name in the Web application box. Is this the application you wanted to view or modify?

  • Enter your Windows NT domain name, a backslash, and your user ID in the User name box. For example, a user specified in the operating system as "myusername" in a domain called "mydomain" would enter this: mydomain\myusername.

  • Enter your network password in the Password box.

For more information, see Adding Security.
