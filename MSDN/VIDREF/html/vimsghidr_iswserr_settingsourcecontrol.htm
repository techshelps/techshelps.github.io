Cannot establish source control for the Web application.

You aren't able to establish source control for the Web application. Typically, this is because Microsoft® Visual SourceSafe™ was not set up properly on your computer or on the server. The second sentence of this error describes the specific problem, as reported by the Microsoft® FrontPage® Server Extensions. This error is usually of the form:

[Server Error:   error text]

  • To correct this error, correct the condition described in the second sentence of this error.

For more information, see Working with Multiple Developers and Source Control.
