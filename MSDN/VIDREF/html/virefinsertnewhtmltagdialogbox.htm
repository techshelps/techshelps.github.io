Insert New HTML Tag Dialog Box

      

Use this dialog box to add HTML tags to the Modified HTML Tags node or to any HTML tag in the CSS Editor window. You can then select these tags and specify style settings for the tags.

Note   When you specify style settings for an HTML tag, the style settings will apply to all instances of the tag. You can not have multiple copies of the same HTML tag in the CSS Editor.

Options

HTML tag

Select to choose an HTML tag from the list or type a tag name. The tag will appear in the Modified HTML Tags node of the CSS Editor window under the currently selected tag.

For example, you could select the <BODY> tag to set styles for the tag.
