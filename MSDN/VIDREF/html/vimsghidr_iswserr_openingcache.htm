Cannot open the local state file for the project <name>. The local state of files in this project has been lost.

You're trying to open an existing Web project, but Microsoft® Visual InterDev™ is not able to open the information on the local state of your project files. Visual InterDev stores information on any changes you've made to files in local mode, but this information has been lost.

  • To correct this error, you must recreate any of your local files and changes. You will lose changes you've made to the files in local mode.

For more information, see Working Locally.
