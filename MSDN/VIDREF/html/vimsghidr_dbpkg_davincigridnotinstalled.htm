Cannot find the parameter grid control.

The component of Microsoft® Visual InterDev™ that contains the parameter grid control was not installed properly on your computer. You use the parameter grid control when you select a stored procedure in the Data View window and then choose the Execute command. The parameter grid is the dialog box in which you enter the parameters for the stored procedure.

  • To correct this error, you must re-install Visual InterDev.

For more information, see the instructions on installing Visual InterDev.
