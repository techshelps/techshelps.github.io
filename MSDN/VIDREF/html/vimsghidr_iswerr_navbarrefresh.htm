Navigation bars will now use the navigation structure information on the local Web server for local files.

Microsoft® Visual InterDev™ will use the navigation structure information for the Web project on the local Web server each time you refresh, recalculate links, or synchronize your Web project.

Using the navigation structure on the local Web server allows you to refresh, recalculate links, and synchronize your Web project faster. However, any changes you make to the navigation bar links for local files may not appear when you preview your pages.

To use the most current information to generate navigation bars, check this option and refresh the Web project.
