Reorder Global Navigation Bar Dialog Box

      

Use global navigation bar pages to create a navigation bar containing the most used or most useful links for a site. You can easily change the order in which these links appear on the global navigation bar.

Options

Order of pages in global navigation bar

Lists all of the pages currently marked as Global Navigation Bar pages in the order that the pages will appear on a global navigation bar.

Move up

Allows you to move the page selected in the Order of pages in Global Navigation bar list box above the previous page in the list.

Move down

Allows you to move the page selected in the Order of pages in Global Navigation bar list box below the next page in the list.
