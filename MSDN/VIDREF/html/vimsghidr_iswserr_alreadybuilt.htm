There is already a built component in this folder with the same name as <name>.

You are trying to add a built output file to a folder in your Web project and a built component with the same name from another project or the same project already exists in the folder. Built outputs come from other applications such as Microsoft® Visual J++™, and can be added to your Web project. This error occurs while you're adding built outputs; another output has the same name as one you're adding.

  • To correct this error, add the built output file to a different folder.

    -or-

    Remove the previous project output dependency.

For more information, see Build a Package.
