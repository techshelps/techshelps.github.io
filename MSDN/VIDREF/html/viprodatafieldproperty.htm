DataField Property

            

Specifies the field that binds the current control to a particular recordset. Available at design time.

Remarks

You must specify a Recordset control in the Recordset property before setting the DataField property. The choice of valid recordsets is determined by which Recordset controls exist on the page.

This property is mapped to the Field drop-down list box on the General tab for the following dialog boxes: Checkbox Properties, Label Properties, Listbox Properties, OptionGroup Properties, Textbox Properties.
