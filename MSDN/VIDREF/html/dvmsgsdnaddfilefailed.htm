Could not add files to the project.

Site Designer has created the files for the new pages in the site diagram but was unable to add the pages to the Web project. The files are located in the working directory for the Web application. Usually this is the result of a lost connection with the master Web server.

  • To correct this error, add these pages manually to the Web application using the Add to Master Web command on the Project menu.

    -or-

    Create the pages listed manually and then add the pages to the master Web server.

For more information, see Adding Files and Creating Pages.
