TitleFont Property

            

Specifies the name of the font for the header row of the grid. Available at design time.

Remarks

The TitleFont property sets the value of the FACE attribute of the <FONT> tag. You can select from the available list or type in a different value that is supported by the FACE attribute. The following example uses more than one default font:

Verdana, Arial, Helvetica

This property is mapped to the Name(s) box on the Format Tab of the Grid Properties dialog box when the Header row option is selected. The default value is Arial.
