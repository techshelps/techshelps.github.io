Zoom Dialog Box

      

Appears when you choose Zoom from the View menu. This dialog box enables you to select a preset zoom level or type a custom zoom level to change the magnification of the link diagram.

Options

Zoom to

Select a zoom level. To specify a custom zoom level, select Custom and type a percentage from 10% to 400%. To make the link diagram fit completely in the window, choose Fit. When zooming to fit, Link View will not make the diagram smaller than 10% or larger than 100%.
