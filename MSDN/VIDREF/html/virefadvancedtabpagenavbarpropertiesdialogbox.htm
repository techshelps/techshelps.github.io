Advanced Tab (PageNavbar Properties Dialog Box)

      

Use this tab to set the target frame for the navigation bar. You can also specify to use the navigation structure of a different page to generate the navigation bar links.

For more information, see Designing Site Navigation.

Options

Alternate page

Select to use the navigation structure of a different page to generate the navigation bar links for the current page. For example, you could choose to use the navigation structure of the home page to generate navigation bar links for the current page.

Note   This option accepts only project-relative URLs. For example, if you choose to use the navigation structure of default.asp to generate navigation bar links, you would enter default.asp rather than /default.asp.
