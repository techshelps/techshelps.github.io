ListField Property

            

Specifies the field that supplies the control with text to display on the page. Available at design time.

Remarks

You must specify a Recordset control in the RowSource property before setting the ListField property. The choice of valid recordsets is determined by which Recordset controls exist on the page.

This property is mapped to the List field box on the Lookup tab for the following dialog boxes: Listbox Properties and OptionGroup Properties.
