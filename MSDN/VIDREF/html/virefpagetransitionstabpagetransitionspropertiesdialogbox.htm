Page Transition Tab (PageTransitions Properties Dialog Box)

      

Use this tab to specify the duration and type of transition that you want to occur when the page is loaded into the Web browser or replaced when the user links to another page.

Note   If the page is displayed within a frame, the page transitions do not play. These transitions can only occur on the main page read by the browser, not within frames.

Options

Page Enter

This option group specifies the type and duration of the transition that plays when the page is loaded into the Web browser.

Transition

Select a transition from the list of possible transitions. This list reflects the transitions available with Microsoft® Internet Explorer 4.0. The default value is No transition.

For a list of transitions, see the Internet SDK topic, "Visual and Transition Filters Reference."

Duration

Select the amount of time it takes for the transition to play. The default value is 1 second. Although you can set the duration to a maximum of 59.99 seconds, you typically choose from 1 to 3 seconds. The duration of a transition is not affected by the speed of the user's machine but it can affect the smoothness of the display.

Note   The shorter the time the smoother the transition appears to flow. The longer the time the more you notice the incremental steps the transition takes.

Preview

Select to play the transition for Page Enter in the Preview box.

Page Exit

This option group specifies the type and duration of the transition that plays when the current page is replaced by another page in the Web browser. For example, when a user chooses a link on the page, the transition replaces the current page with the next page.

Transition

Select a transition from the list of possible transitions. This list reflects the transitions available with Internet Explorer 4.0. The default value is No transition.

Duration

Select the amount of time it takes for the transition to play. The default value is 1 second. Although you can set the duration to a maximum of 59.99 seconds, you typically choose from 1 to 3 seconds. The duration of a transition is not affected by the speed of the user's machine but it can affect the smoothness of the display.

Preview

Select to play the transition for Page Exit in the Preview box.
