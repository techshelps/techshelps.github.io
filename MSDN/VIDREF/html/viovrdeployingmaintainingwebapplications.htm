Deploying and Maintaining Web Applications

When you are done developing and testing your Web application, you can make the tested version of your application available to end users on the production server.

Depending on your Web application, you might want to complete one of the following three tasks:

To Use
Publish a new Web application Deploying a New Web Application
Update an existing Web application Maintaining Existing Application
Deploy server components or components as a transaction package Deploying an Integrated Web Solution
