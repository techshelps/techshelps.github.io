RowFontSize Property

            

Specifies the font size for the data rows of the grid. Available at design time.

Remarks

The RowFontSize property sets the value of the SIZE attribute of the <FONT> tag. This property is mapped to the Size box on the Format Tab of the Grid Properties dialog box when the All detail rows option is selected.

The value ranges from one to seven or can be set to +1, +2, +3, -1, -2 or –3. The default value is 2.
