Refreshing the diagram will discard any changes that have been made. Continue?

The site diagram may have been modified and the changes have not been saved. Refreshing the site diagram will return the diagram to the last saved version of the file.

  • If you choose Yes, the site diagram will refresh and all changes will be lost.

  • If you choose No, the site diagram will not refresh and you can save the diagram before trying to refresh again.

For more information about refreshing Web application files, see Getting Master Copies Locally.
