HTML Tags and Link Diagrams

      

Link View identifies the in links and out links for each item in a link diagram using different link information based on whether the item is an internal item or an external item.

For items existing in the current Web project, Link View builds the link diagram using the link information located on the Web project server in the link index file. This information, however, does not reflect changes other users may have made to the project since the link diagram was created. To retrieve the latest link information from the Web server, use the Refresh command.

For items located outside the current Web project, Link View dynamically verifies items by searching the World Wide Web for the item's HTML tags using HTTP, FTP, and Gopher. When verifying external items, Link View uses the most current link information for the item.
