Unable to open the local Web application <name> for this project.

You're trying to open the local Web application for a project, but are unable to do so. The second sentence of this error describes the specific problem, as reported by the Microsoft® FrontPage® Server Extensions. This error is usually of the form:

[Server Error:  error text]

  • To correct this error, correct the condition described in the second sentence of this error.
