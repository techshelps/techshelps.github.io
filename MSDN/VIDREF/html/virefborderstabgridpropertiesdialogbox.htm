Borders Tab (Grid Properties Dialog Box)

      

Use this tab to set the border and color options of the Grid design-time control.

Options

Border style

Custom

Select to specify that the table border should be a double line. An example is displayed to the left of this option. This is the default border style.

Thin

Select to specify that the table border should be a thin line. An example is displayed to the left of this option. The border of the table is turned off, Cell separation is set to 1, and Grid background is used to set the color of the border.

3D

Select to specify that the table border should have a three-dimensional look. An example is displayed to the left of this option.

None

Select to specify that the table should not have a border. An example is displayed to the left of this option.

Gridlines

All

Select to specify that the table should display all possible gridlines. This is the default setting for Gridlines.

Horizontal only

Select to specify that the table should display only horizontal gridlines.

Vertical only

Select to specify that the table should display only vertical gridlines.

None

Select to specify that the table should not display any gridlines.

Color

Border

Select to specify the color of the table border when Border style is set to Custom. When Border style is set to Thin, use Grid background to set the border color.

Highlight

Select to specify the color of the highlight if Border style is set to 3D.

Grid background

Select to specify the background color of the table. This color setting does not affect the color of the table cells.

Shadow

Select to specify the color of the shadow if Border style is set to 3D.

Border size

Select to specify the width of the table border.

Cell separation

Select to specify the amount of space between the cells in the table.
