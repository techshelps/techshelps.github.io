Copying Web Structure Dialog Box

      

Use this dialog box to specify how to treat conflicts between the existing site structure file on the destination Web server and the site structure file being copied during the Copy Project process.

The site structure file stores navigation bar link information for the pages in a Web project.

Note   When you replace or merge the destination Web server and the source Web server site structures you can impact the navigation bar links for the project.

Options

Destination site structure

Select to protect the site structure files on the destination Web server from being overwritten with the source Web project's site structure file.

Source site structure

Select to overwrite the destination Web project's site structure file with the source Web project's site structure.

Merge site structures

Select to merge the site structure file for the destination Web project with the site structure file for the source Web project.
