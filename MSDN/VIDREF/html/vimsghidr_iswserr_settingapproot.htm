Cannot set the application root for this Web application.

Microsoft® Visual InterDev (through Microsoft® Internet Information Server) is trying to set a root for your Web application when the application is first started, but is unable to do so. The second sentence of this error describes the specific problem, as reported by the Microsoft® FrontPage® Server Extensions. This error is usually of the form:

[Server Error:   error text]

  • To correct this error, correct the condition described in the second sentence of this error.

For more information, see Creating Web Projects.
