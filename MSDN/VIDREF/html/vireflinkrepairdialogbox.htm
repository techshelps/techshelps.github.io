Link Repair Dialog Box

      

Use this dialog box to specify whether you want Microsoft® Visual InterDev to repair the links to a renamed or moved file or folder automatically. This dialog box appears after you have renamed or moved an existing file or folder.

Options

Do you want to update the links in files that refer to this file/folder?

When you rename or move a file or folder you might break links in other files that reference that changed file or folder.

Yes

Select to fix links in all files in the project that references the changed file or folder. This option fixes links in both the local and master copies of the file regardless of the project’s mode.

No

Select to rename the file, but do not update the links.

Cancel

Select to cancel the rename or move.

Use as default response

Select this option if you think you will always use the same choice each time you rename or move a file or folder.
