Integrated Samples

To enhance your Web application, you can include features from other products in your Web pages.

You can use integrated components, such as Java applets, to add functionality to your Web site not yet supported by dynamic HTML. Or, by using a guest book scenario, you can obtain information feedback from the people visiting your site.

The samples in this section show different scenarios for accomplishing these tasks.
