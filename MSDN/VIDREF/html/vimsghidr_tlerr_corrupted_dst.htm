The file contains insufficient HTML tags or incomplete layout information. Themes and layouts cannot be applied.

Microsoft® Visual InterDev™ can only apply themes and layouts to files that contain valid HTML tag information.

Files must contain the <HTML> and <BODY> tags as well as all end tags before themes and layouts can be applied. To correct this error

  • Open the selected file in the HTML editor and verify that the file contains the <HTML>, <BODY>, </BODY>, and </HTML> tags. Also, verify that the file is not missing any end tags and then apply themes and layouts again.

If a file already has a layout, Visual InterDev can only apply themes and layouts again if the file contains all of the following Layout controls: Layout Header Start, Layout Header End, Layout Footer Start and Layout Footer End. In the HTML editor, the layout controls appear as Add Your Content Above and Add Your Content Below on a page.

To correct this error:

  1. Open the selected file in the HTML editor and verify that each layout Add Your Content Above has a corresponding Add Your Content Below and vice versa. If one or more layout controls are missing:

  2. Delete the remaining layout controls and corresponding PageNavbar controls in Source view.

  3. Delete each <!--VI6.0Layout = "layout name"--> from the file. For example, <!--VI6.0Layout = "Left 1"-->.

  4. Apply themes and layouts again.

For more information, see Site Consistency and Laying Out Pages.
