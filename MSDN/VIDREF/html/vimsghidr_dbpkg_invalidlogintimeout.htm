Valid range for 'SQL login time-out' is 1-65535.

You entered a value in the SQL login time-out field in the Query options group of the Data View tab of the Data Tools Options that is outside the legal range of seconds.

  • To correct this error, choose the Options command on the Tools menu. In the Data Tools section, select the Data View tab and enter a value for seconds between 1 and 65535.

For more information, see Data View Options (Data Tools-Options Dialog Box).
