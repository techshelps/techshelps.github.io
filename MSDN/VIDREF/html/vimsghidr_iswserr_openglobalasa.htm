Microsoft Visual InterDev is attempting to open the Global.asa file, but is unable to do so. The second sentence of this error describes the specific problem, as reported by the FrontPage Server Extensions. This error is usually of the form:

[Server Error:   error text]

  • To correct this error, correct the condition described in the second sentence of this error.
