Check In to Source Control Dialog Box

      

Use this dialog box to check in stored procedures to source control. You can enable source control for stored procedures in the Data View window, and check out and check in these stored procedures.

For more information on using source control with stored procedures, see Adding a Stored Procedure to Source Control in Data View.

This dialog box is displayed when you right-click a checked-out stored procedure in the Data View window, and then choose the Check In command.

Options

Comment

Enter a description or explanation to accompany the check in for this stored procedure. For example; "Updated CustomerDiscount column to reflect new discount."
