Data Range Footer Design-Time Control

Creates script that works with the script generated by the Data Range Header control to display records from a record set on a Web page.

Note   This control is included for compatibility for Microsoft® Visual InterDev™ 1.0 only. For complete details, see your Visual InterDev 1.0 documentation.

In Visual InterDev 6.0, use the Grid design-time control instead of the Data Range Header and Footer controls.

Remarks

The Data Range Footer creates an HTML form with buttons that allow a user to move through the pages of records in a record set.
