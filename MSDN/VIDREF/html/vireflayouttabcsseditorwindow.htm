Layout Tab (CSS Editor Window)

      

Use this tab to size text or image elements, specify text placement and spacing, and to set cursor styles for elements. For more information, see CSS Attributes.

Note   The CSS Editor allows you to modify HTML tags supported by Microsoft® Internet Explorer version 4.0 or later. HTML tags not supported by Internet Explorer will be converted to UNKNOWN when you save the .css file. For more information, see HTML References.

Options

Scale

Width

Select to set the width of text or image elements and the unit of measure. Images are scaled to fit the specified width if necessary. Leaving this option set to nothing preserves the aspect ratio of the image.

Height

Select to set the height of text or image elements and the unit of measure. Images are scaled to fit the specified height if necessary. Leaving this option set to nothing preserves the aspect ratio of the image.

Text layout

Alignment

Select to set the justification or positioning of text within an element. Choose from left, right, center, and justified.

Indent

Select to set the indent that appears before the first formatted line. Use the associated list box to set the unit of measure. Indent accepts negative values and percentages. Indentations are not inserted in the middle of elements that have been broken by other elements, such as the <BR> tag in HTML.

Spacing between

Letters

Select to set additional spacing between text letters. Choose Normal or enter a specific spacing unit. Letters accepts negative values but does not accept percentages.

Lines

Select to set additional spacing between lines of text in a paragraph. Choose Normal or enter a specific spacing unit. Percentage values are set in relation to the font size of the element. Lines does not accept negative values.

Page break

Before

Select to set when page breaks occur before elements. To specify that page breaks occur before the element only if necessary, choose Auto. To specify that the page break always occur before the element, choose Always. To have one or more page breaks occur before the element until a left page is reached, choose Until blank left page. To have one or more page breaks occur before the element until a right page is reached, choose Until blank right page.

After

Select to set when page breaks occur after elements. To have the page break occur after the element only if necessary, choose Auto. To have the page break always occur after the element, choose Always. To have one or more page breaks occur after the element until a left page is reached, choose Until blank left page. To have one or more page breaks occur after the element until a right page is reached, choose Until blank right page.

Text line shift

Select to set the vertical position of text elements. Choose from subscript and superscript.

Cursor style

Select to set the type of cursor that displays for the mouse pointer. To display the correct cursor type based on the current context, choose Auto. To display the platform cursor, choose Default. Other options include crosshair, hand, move, resize, text, and wait.
