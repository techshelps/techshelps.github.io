Source Control Login Dialog Box

      

Use this dialog box to login to your source control project when you first access a database whose stored procedures have been put under source control. You can enable source control for stored procedures in the Data View window, and check out and check in these stored procedures. For more information on using source control with stored procedures, see Adding a Stored Procedure to Source Control in Data View.

This dialog box is displayed when you expand the Stored Procedures folder for a data connection under source control in the Data View window.

Note   If you cancel this dialog box, you can log in later by refreshing the Stored Procedures folder in the Data View window.

Options

Login ID

Enter your login ID for this source control project.

Password

Enter your password for this source control project.
