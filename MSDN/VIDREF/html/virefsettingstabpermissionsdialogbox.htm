Settings Tab (Permissions Dialog Box)

      

Use this tab to specify that the permissions for the Web application be unique from the permissions of the root Web application on the master Web server.

Options

Use same permissions as the root Web application

Select this option to use the permissions that were established with the root Web application on the master Web server. If you select this option, the settings in the other tabs show the permissions for the root Web application.

To change the permissions for the root Web application on the master server, you need administrator permission on the master Web server and a Web project that points to the root Web application.

Use unique permissions for this Web application

Select this option to specify permissions for users, groups, or computers that are different from those specified for the root Web application. If this option is selected, you can change users, groups, or computers. To apply this option and then specify unique users, groups, or computers, choose Apply and then choose the appropriate tab.
