Scripting Object Model

Expand this node in the contents to see a complete listing of the objects, properties, events, and methods that make up the Microsoft® Visual InterDev™ Scripting Object Model.

Note   These items apply to the Scripting Object Model only. For a list of design-time controls, see Design-Time Controls. For a list of properties that apply to the design-time controls, see Design-Time Controls Properties.

Important   The scripting object model is implemented using script stored in the Script Library. Do not alter the contents of the library, or components in the scripting object model will not work properly.

For a discussion of the Scripting Object Model and design-time controls, see The Scripting Object Model and Scripting with Design-Time Controls and Script Objects.
