General Tab (PageNavbar Properties Dialog Box)

      

Use this tab to specify the pages that appear as links on the navigation bar. The PageNavbar design-time control uses the navigation structure designed in the site diagrams for a Web project to determine parent page, sibling page, child page, and global navigation bar links.

For more information, see Designing Site Navigation.

Options

Type

Select the navigation structure types you would like to appear as links on this navigation bar. Use the sample site diagram to view the types of links that will appear based on the options you select below.

Global navigation bar

Select to generate links to pages designated as global navigation bar pages in the site diagrams for the Web project.

First level pages

Select to generate links to pages one level below the home page.

Parent level pages

Select to generate links to pages one level above the current page.

Sibling pages

Select to generate links to pages at the same level as the current page.

Back and next pages

Select to generate links to pages at the same level as the current page that use the previous and next arrows. To change the order of the previous and next links, reorder the pages in the site diagram.

Children pages

Select to generate links to pages one level below the current page.

Banner

Displays the page label, page title, or file name for the page overlaid on an image. The image is provided by the theme applied to the page. To change the image used for the banner, apply a different Microsoft® Visual InterDev™ theme or apply your own custom theme.

Additional pages

Home

Select to include the home page as a link on the navigation bar.

Parent

Select to include the parent page of the current page as a link.
