Incompatible version of the FrontPage Server Extensions. You need at least version XX.XX.XXXX.XX

This version of Site Designer cannot use the version of FrontPage Server Extensions currently on the Web server. Usually this is the result of Microsoft® FrontPage® 97 or older server extensions in use on a Web server. This can also occur if FrontPage 98 Beta server extensions are in use.

  • To correct this error, upgrade to the released version of the FrontPage 98 Server Extensions.

For more information, see "An Overview of the FrontPage Server Extensions" in the FrontPage Server Extensions Resource Kit help.
