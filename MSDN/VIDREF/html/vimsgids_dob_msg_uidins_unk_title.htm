Error inserting Unique ID.

The identifier is not valid. Usually this is the result of entering an invalid character, such as :, ;, #, (), [], or ?, in the unique ID name. Unique ID names contain only one # character at the beginning of the name.

  • To correct this error, re-enter the unique ID name.

    -or-

    Enter a different unique ID.

For more information, see Insert New Unique ID Dialog Box.
