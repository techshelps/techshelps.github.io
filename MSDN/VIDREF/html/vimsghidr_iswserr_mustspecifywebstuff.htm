You've bypassed the Web Project wizard by placing an existing .vip file in the \Program Files\Microsoft Visual Studio\VintDev98\Templates\Web Projects folder and used this .vip file to open a Microsoft® Visual InterDev™ project.

In the Open or Create Web Project dialog box that's displayed, you must enter values in both the Web server URL field and the Web name field. Neither can be left blank.
