CSS file is not valid.

The cascading style sheet (.css) file is invalid. If the invalid .css file is saved, the data in the file may be lost. Note that the .css file is not modified until it is saved. Therefore if you close the file without saving it, no data will be lost.

  • To correct this error, close the .css file without saving it.

For more information on .css files, see Customizing Page Appearance.
