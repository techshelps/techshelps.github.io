Delete Pages Dialog Box

      

Allows you to remove pages from the site diagram and the navigation bars for a site. You can also choose to delete pages from the site diagram and the Web project.

Options

Remove these pages from all navigation bars

Removes the selected pages from the site diagram. These pages will no longer be available as links on any of the site's navigation bars. This option does not remove files from the Web project.

Delete these pages from the Web project

Deletes the selected pages from the master Web application and removes the pages from the site diagram. These pages will no longer be available as links on any of the site's navigation bars.
