You are working in local mode. Any changes you make to your site diagrams, including the addition of new pages, will be made on the master Web server.

When you work in local mode, Microsoft® Visual InterDev™ saves your changes only to your local machine. Site Designer, however, saves changes to the master Web server regardless of the project mode.

Pages created with the New HTML Page and New ASP Page commands are added to the master Web server when you save the site diagram. Any navigation structure changes you make in a site diagram are also added to the master Web server when you save a site diagram.

Note   When you use site diagrams, your changes will always be available for all users in the Web application when you save.

For more information, see Visual InterDev Modes in Site Designer.
