Permissions Dialog Box

Use this dialog box to set browse, administration, and authoring permissions for your Web application.

Through this dialog box, Microsoft® Visual InterDev™ exposes the security model defined by Microsoft® FrontPage® for Web applications hosted on a server running the FrontPage Server Extensions. For more information about the FrontPage security model, see the FrontPage documentation.

Note   If the Use unique permissions for this Web application option is selected on the Settings tab, you can change the users, groups, or computers permissions for the Web application.

To change the permissions for the root Web application on the master server, you need administrator permission on the master Web server and a Web project that points to the root Web application.

For more information, see Security.

Add, edit, or remove unique permissions specified for a computer or group of computers(1) Computers Tab (Permissions Dialog Box)

(1) This dialog box is not available if your master Web server is on an operating system using the File Allocation Table file system (FAT), such as Windows® 95, instead of Windows NT® File System (NTFS).
