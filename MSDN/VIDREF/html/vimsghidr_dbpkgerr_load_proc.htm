Failed to load source for stored procedure.

You are trying to open or design a stored procedure from the Data View window, but Microsoft® Visual InterDev™ is unable to load the source code text for the stored procedure into the editor.

  • To correct this error, open the stored procedure in your database and view its source code. Look for problems that might cause the source code to be unable to be loaded into the editor.

For more information, see Working with Stored Procedures in Data View and Stored Procedures.
