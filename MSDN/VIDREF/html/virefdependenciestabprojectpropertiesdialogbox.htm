Dependencies Tab (Project Properties Dialog Box)

      

Use this tab to view dependent projects and project output files, and to delete the project dependencies.

Note   Project properties are read-only while you are offline. To change these settings, you must be connected to the network.

Options

Dependent projects

Displays a list of the projects whose outputs will be added automatically to the selected Web project when you build the Web project. To add a project to the list, use the Add Project Outputs Dialog Box.

Files

Displays a list of the output files for the project specified in the Dependent projects list box. These output files are updated and added to the Web project when you build the project.

Remove Dependent Project

Removes the selected project from the Dependent projects list.
