The Query cannot be run until it has been designed. Choose Design from the shortcut menu.

You've added a query to your database project, but haven't designed the query. That is, you haven't added the database fields and criteria that define a set of records in the connected database. This can happen if you add a query using the Add Item command, but then immediately close the query without designing it. When you then choose the Open command, this error occurs.

  • To correct this error, select the query in the database project and choose Design from the shortcut menu. Then build the query in the Query Designer.

For more information, see Designing Queries.
