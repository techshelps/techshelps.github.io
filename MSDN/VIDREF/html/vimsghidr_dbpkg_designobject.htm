Failed to design object.

You're using the Design command in Data View to open a table, view, or database diagram, or the Design command to open a local query in a database project, but the Database Designer or Query Designer can't be used to design the specified object. The object may be corrupt, or the designer may not be installed correctly.

  • To correct this error, open the object in the database directly, and make sure it's not corrupt.

    -or-

    Re-install the Microsoft® Visual Database Tools by reinstalling Microsoft® Visual InterDev™.

For more information, see Working with Database Objects.
