Cannot set component registration or project reference information in the master Web application.

Microsoft® Visual InterDev™ is trying to set component registration or project reference information for the master Web application, but is unable to do so. Visual InterDev stores information on what components need to be registered. The second sentence of this error describes the specific problem, as reported by the Microsoft® FrontPage® Server Extensions. This error is usually of the form:

[Server Error:   error text]

  • To correct this error, correct the condition described in the second sentence of this error.

For more information, see Managing Web Projects.
