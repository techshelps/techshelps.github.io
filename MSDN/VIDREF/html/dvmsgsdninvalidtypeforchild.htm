Parent pages must be an .htm or .asp file in your Web project.

A parent page has been renamed or mapped to a URL outside of the current Web project. Site Designer only allows .htm or .asp files located in the current Web project to be parent pages.

  • To correct this error, change the file extension of the parent page to .htm or .asp.

    -or-

    Detach the child pages from the parent page in the site diagram and then change the URL of the page to point to a page outside of the current Web project.
