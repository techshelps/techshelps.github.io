Cannot build file while offline.

You're trying to build an output file while in offline mode. A file can only be built in online mode, or if it's a personal file or it has been checked out. Built outputs come from other applications such as Microsoft® Visual J++™, and can be added to your Web project. This error occurs while the file is being built.

  • To correct this error, switch to online mode and build the file.

For more information, see Build a Package.
