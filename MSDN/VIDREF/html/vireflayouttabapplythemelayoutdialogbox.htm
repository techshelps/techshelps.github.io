Layout Tab (Apply Theme and Layout Dialog Box)

      

Layouts provide a consistent approach to navigation bar placement on a page. Use this dialog box to select the placement of navigation bars on a page and to preview a graphical description of the layout before applying it to a page.

Options

Preserve current layout

Select to retain the existing layouts applied to pages in a Web project. Pages will use the layout selected in the Apply layout and reapply theme box.

Apply layout and theme

Lists the current layouts available from the path specified in the Directory box. When you apply a layout, Microsoft® Visual InterDev™ automatically reapplies the theme selected on the Theme tab.

Preview

Displays a graphical description of the selected layout. The graphic illustrates where the navigation bars will be located on the page and identifies the types of navigation links that will be used by default with that layout.

Directory

Lists the path to the layout files displayed in the Apply layout and theme box.

Restore Default Directory

Select to return the path listed in the Directory box to the Visual InterDev default Layouts directory.

Browse

Select to choose a set of layouts in a different directory.
