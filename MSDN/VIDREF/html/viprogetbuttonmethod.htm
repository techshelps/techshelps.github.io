getButton Method

            

Returns one of the Button objects that is created by the RecordsetNavbar object or the OptionGroup object.

Syntax

object.getButton(nIndex)

Parameters

object

A RecordsetNavbar or OptionGroup script object.

nIndex

An integer that specifies which button object is returned. The following values determine which button is returned for the RecordsetNavbar.

Value Specifies Button for
0 First record
1 Previous record
2 Next record
3 Last record

Example

objLastButton = RecordsetNavbar1.getButton(3);
objLastButton.value = "Very Last";   // change caption of the "Last" button
