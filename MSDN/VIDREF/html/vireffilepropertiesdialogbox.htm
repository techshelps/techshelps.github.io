File Properties Dialog Box

      

Use this property page to view file information for the working Web and master Web project, as well as file statistics and source control information. You can also set installation and registration options for components and select build configuration types.

Note   File properties are read-only while you are offline. To change these settings, you must be connected to the network.

To See
View the name of the current file, the location of the working copy, the working Web server URL, the master Web server URL, and file statistics General Tab (File Properties Dialog Box)
View source control information about a file and determine whether its Web project has been added to source control Source Control Tab (File Properties Dialog Box)
Set options for installation and registration of server and client components Component Installation Tab (File Properties Dialog Box)
