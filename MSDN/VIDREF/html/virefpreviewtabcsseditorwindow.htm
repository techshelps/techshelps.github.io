Preview Tab (CSS Editor Window)

      

Use this tab to view the affect of the CSS settings on a sample page. For more information, see CSS Attributes.

Note   The CSS Editor allows you to modify HTML tags supported by Microsoft® Internet Explorer version 4.0 or later. HTML tags not supported by Internet Explorer will be converted to UNKNOWN when you save the .css file. For more information, see HTML References.

Options

HTML page to preview the style sheet with

Select the file used to preview the selected styles. Enter a file path or URL in the text box or choose Browse to search for a file.

Use Default Page

Select to use the CSS Editor default page to preview your style settings against.

Style sheet applied

Select to view how the current styles affect the file specified in HTML page to preview the style sheet with text box. When not selected, the page is viewed using the standard HTML tags attributes as defined by the default browser.
