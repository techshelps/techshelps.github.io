Alignment Property

            

Specifies whether the buttons are aligned vertically or horizontally.

Settings

The settings for Alignment are:

Setting Description
0 – Vertical Buttons are aligned vertically.
1 – Horizontal Buttons are aligned horizontally.

Remarks

This property is mapped to the Alignment option on the Format Tab of RecordsetNavbar Properties dialog box and to the Align option on the General Tab of OptionGroup Properties dialog box.
