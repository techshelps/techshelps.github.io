Valid range for 'Limit SQL output' is 1-9999.

You entered a value in the rows box of the Limit SQL output field in the Query options group of the Data View tab of the Data Tools Options that is outside the legal range of rows.

  • To correct this error, choose the Options command on the Tools menu. In the Data Tools section, select the Data View tab and enter a value for rows between 1 and 9999.

For more information, see Data View Options (Data Tools-Options Dialog Box).
