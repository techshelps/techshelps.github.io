bubbleEvent Method

            

Sends event notification for a standard event from a DHTML scriptlet to the host application.

Syntax

window.external.bubbleEvent()

Remarks

Use this method to pass a standard DHTML event (such as onbuttonclick, onmousemove, or onkeypress) from a DHTML scriptlet to the host application. For more details, see Exposing Events in DHTML Scriptlets.
