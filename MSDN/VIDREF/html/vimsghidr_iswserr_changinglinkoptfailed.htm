Cannot change the link repair option.

You're trying to change the Link repair option in the Web Projects section on the Projects tab of the Options dialog box, but are unable to do so. The second sentence of this error describes the specific problem, as reported by the Microsoft® FrontPage® Server Extensions. This error is usually of the form:

[Server Error:   error text]

  • To correct this error, correct the condition described in the second sentence of this error.

For more information, see Web Project Options (Projects-Options Dialog Box) and Link Repair Dialog Box.
