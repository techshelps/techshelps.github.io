Apply Theme and Layout Dialog Box

      

Use this dialog box to provide a consistent visual design and approach to navigation bar placement in pages in your Web project.

Note   Apply Theme and Layout dialog box options are read-only while you are offline. To change these settings, you must be connected to the network.
