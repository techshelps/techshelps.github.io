The specified file name is not valid.

This file name cannot be used for a page. Usually this is the result of invalid characters in the file name, a blank file name, a file name that includes a folder name, or a file name that is too long.

  • To correct the file name, enter a new file name for the page or re-enter the file name.

For more information, see "To change the name of a file or folder" in Windows help.
