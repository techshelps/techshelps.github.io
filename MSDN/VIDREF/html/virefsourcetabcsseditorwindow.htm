Source Tab (CSS Editor Window)

      

Use this tab to view the source generated by the style sheet based on the CSS settings. For more information, see CSS Attributes.

Note   The CSS Editor allows you to modify HTML tags supported by Microsoft® Internet Explorer version 4.0 or later. HTML tags not supported by Internet Explorer will be converted to UNKNOWN when you save the .css file. For more information, see HTML References.

Options

Style sheet source

Displays the source CSS script for the style options selected. You can copy and paste from within this list box to add the source CSS into individual HTML pages.
