Link Diagram Legend

      

Items in the link diagram are joined by lines that represent links between the items. Arrows communicate the direction of the links, either in or out.

You can understand the link relationships between items by analyzing the graphical representation of those relationships in a link diagram.

Broken links between items, either selected or unselected, are displayed as red lines:

The origin of a link is represented by a nub on a link line. A link line that passes behind an item is not linked to that item.

Link Diagram Layouts

Link View builds link diagrams in one of two layouts: horizontal layout and radial layout.

  • In horizontal layout, the link diagram displays in links to the left of the central item and displays out links to the right of the central item.

  • In radial layout, Link View displays either in links or out links, but not both, arranged in a circle around the central item.

The horizontal layout appears by default when you build a new link diagram. You switch between these layouts by using the Change Diagram Layout button on the Link View toolbar. For more information on Link View layouts, see Changing a Link Diagram Layout.

Expanding and collapsing items in a link diagram provides different results depending on the layout you are working in.

  • In horizontal layout, expanding links to the left of the central item only expands the in links for an item. Expanding links to the right of the central item expands only the out links for an item.

  • In radial layout, you can expand and collapse only in links or out links in a diagram.
