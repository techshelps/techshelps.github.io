Master Web Server Tab (Project Properties Dialog Box)

      

Use this tab to view the Web project name, master Web URL, master Web server name, version of server extensions, and server version. This tab also indicates whether source control is enabled for this Web project.

Options

Master Web server

Web name

Displays the name of the master Web project.

URL

Displays the full URL path for the master Web project. Use this path to view the root of the Web project in a browser.

Server properties

Server name

Displays the name of the master Web server.

Extensions version

Displays the version information for the Microsoft® FrontPage® Server Extensions on the master Web server.

Server version

Displays the name and version of the master Web server software.

Source control

Project name

Displays the name of the source control project for the Web project.
