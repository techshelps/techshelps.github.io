The database object <name> already exists. Please choose a unique name.

You're trying to add a database object to a database using the Add Database Item command on the Project menu or one of the Add commands in Data View, and an object with that name already exists.

  • To correct this error, choose a different name for the new object.

    -or-

    Rename the existing object in the database.

For more information, see Managing Databases in Data View.
