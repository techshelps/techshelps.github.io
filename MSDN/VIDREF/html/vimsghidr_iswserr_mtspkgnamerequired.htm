Adding this component to the Microsoft Transaction Server requires a package name

You're trying to add a component to the Microsoft® Transaction Server, but you did not specify a package name for this component.

  • To correct this error, right-click the component in the Project Explorer, choose Properties from the shortcut menu, and then choose Add to Microsoft Transaction Server package on the Component Installation tab. Enter a name in the Package name box.

For more information, see Deploying an Integrated Web Solution and specifically, the "Packaging Components for Microsoft Transaction Server" section in this topic.
