Project Properties Dialog Box

      

Use this dialog box to set options that affect all files in this Web project, such as link repair options, project output files, project launch options, and the default script language. You can also view information such as the Web project name and Web server URL.

To See
View the working Web server details, set link repair options, and Microsoft® Visual InterDev™ mode options for a Web project General Tab (Project Properties Dialog Box)
View the master Web server details, master Web URL, and version information Master Web Server Tab (Project Properties Dialog Box)
View the project outputs, the project output files, and to delete project output files Dependencies Tab (Project Properties Dialog Box)
Set the page that loads when you first launch the debugger, set the launch method, and specify parameters Launch Tab (Project Properties Dialog Box)
Set the scripting language for server and client script, change the theme and layout for a Web project, set the Browser Object Model, and select the data-binding method for design-time controls Editor Defaults Tab (Project Properties Dialog Box)
View the name of the current theme and layout applied to the Web project and to change the default navigation bar link text for pages such as home and parent Appearance Tab (Project Properties Dialog Box)

Note   Project properties are read-only while you are offline. To change these settings, you must be connected to the network.
