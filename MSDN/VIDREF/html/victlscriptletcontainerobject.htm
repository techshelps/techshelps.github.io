DHTML Scriptlet Container Object Reference

                        

Remarks

The scriptlet container object is used to host DHTML scriptlets in an application. It provides properties and events that are available as standard extensions on all DHTML scriptlets. When you work with DHTML scriptlets in your application, you can use the following properties and events of the scriptlet container object.

Properties
event
readyState
scrollbar
url

Events
onclick
ondblclick
onkeydown
onkeypress
onkeyup
onmousedown
onmousemove
onmouseup
onreadystatechange
onscriptletevent
