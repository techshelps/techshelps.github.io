Cannot set default theme name.

You're trying to set a default theme name for your Web project using the DefaultThemeName property, but are unable to do so. The second sentence of this error describes the specific problem, as reported by the Microsoft® FrontPage® Server Extensions. This error is usually of the form:

[Server Error:  error text]

  • To correct this error, correct the condition described in the second sentence of this error.

The DefaultThemeName property is set in the Properties window for the Web project.
