Error inserting HTML tag

The HTML tag is not valid. Usually this is the result of entering an HTML tag that is not recognized by Microsoft® Internet Explorer 4.0 or entering an invalid character in the HTML tag.

  • To correct this error, re-enter the HTML tag.

    -or-

    Enter an HTML tag recognized by Internet Explorer 4.0.

For more information about the HTML tags recognized by Internet Explorer 4.0, see the HTML documentation.
