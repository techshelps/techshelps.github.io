Visual InterDev Error Messages

Only error messages generated by Microsoft® Visual InterDev™ are available here. If you get an error message that is not described here, press F1 to see the associated help topic.

Visual InterDev is dependent upon many components. For example, message may also come from your database server, from the Microsoft® Visual Database Tools, Link View, and Site Designer. For information about component error messages, see the documentation for that component.
