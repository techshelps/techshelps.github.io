Unable to undo checkout of file.

You're trying to undo a check out of a file on the master Web application, but are unable to do so. The second sentence of this error describes the specific problem, as reported by the Microsoft® FrontPage® Server Extensions. This error is usually of the form:

[Server Error:   error text]

  • To correct this error, correct the condition described in the second sentence of this error.

For more information, see Working with Multiple Developers.
