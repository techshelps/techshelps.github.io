Format Tab (Grid Properties Dialog Box)

      

Use this tab to set the font and color attributes of the text appearing in the Grid design-time control. Users can modify the Header, all of the detail rows, or specific columns.

Options

Header row

Select to make font and color changes to only the header row of the grid.

All detail rows

Select to make font and color changes to the detail rows of the grid.

Specific column

Select to specify a single column in the grid to make font and color changes to.

Font

Name(s)

Select to specify the font name to be applied to the selected section of the grid.

Size

Select to specify the font size to be applied to the selected section of the grid. The font size ranges from 1 to 7; the default size is 3.

Color

Select to specify the font color to be applied to the selected section of the grid.

Bold

Select to make the font bold for the selected section of the grid.

Italic

Select to make the font italic for the selected section of the grid.

Wrap text

Select to fit the text within the column width. This option is selected by default.

Color

Row color

Select to specify the color for the rows of the grid.

Alternating row color

Select to specify the color for every other row of the grid. This can also be adjusted by applying a style name in the General tab. Selecting a style will set Row color and Alternating row color to match the style. Not all styles will set an alternating row color.

Alignment

Select to specify the alignment of the text for the selected section of the grid. The default alignment is Left.

Fixed column width

Select to specify that the column width should stay a fixed size. If this option is not selected, the browser automatically sizes the columns for best fit.
