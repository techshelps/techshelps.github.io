Unable to apply themes, layouts, or project defaults to the new files.

The changes to the site diagram have been saved, but Site Designer could not apply themes and layouts or the Web project defaults to the new .htm or .asp files.

This error appears with two different messages:

  • Unable to apply themes and layouts to the new files.

    To correct this error, reapply the appropriate theme and layout to each new file.

  • Unable to apply project defaults to the new files.

    To correct this error, open each new file in the HTML editor and save the files again.

For more information, see Applying Themes, Laying Out Pages, and Project Properties Dialog Box.
