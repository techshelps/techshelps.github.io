Cannot create the Global.asa file. A folder with this name already exists.

Microsoft® Visual InterDev™ can't create the Global.asa file that is needed for your application, because a folder with this name already exists. The Global.asa file contains data connection information and other general information about your application.

  • To correct this error, rename the existing Global.asa folder to a different folder name.

For more information on the Global.asa file, see Sharing Dynamic Information and The Data Environment.
