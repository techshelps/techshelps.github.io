setAlignment Method

            

Determines whether the object's buttons are aligned vertically or horizontally.

Syntax

object.setAlignment(nAlign)

Parameters

object

A RecordsetNavbar or OptionGroup script object.

nAlign

An integer that determines the alignment. nAlign accepts the following values:

Value Description
0 Align buttons vertically
1 Align buttons horizontally

Remarks

To return the value, use the getAlignment method.

Example

In the following line, the buttons are aligned horizontally on the RecordsetNavbar object.

RecordsetNavbar1.setAlignment(1);
