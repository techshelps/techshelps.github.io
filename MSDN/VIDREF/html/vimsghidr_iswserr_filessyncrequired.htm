Cannot access server information for the file <name>.

You're trying to access a file on the server from your project, but information for this file is not available. The file may no longer exist on the server. For example, someone may have deleted a file from the Web server. This error typically occurs when you're trying to check out a file or get the latest version of a file.

  • To correct this error, use the Refresh command to synchronize your project with the server.

For more information, see Updating the Local Web Application.
