Folder name is too long. The complete folder path must be less than 260 characters.

You're trying to add a new folder or rename an existing folder in a Web project, and the folder name (with path) is 260 characters or greater.

  • To correct this error, use a shorter name for the project. You may need to revise your folder structure if the path name is too long.

For more information, see Reorganizing Project Structure.
