Theme Tab (Apply Theme and Layout Dialog Box)

      

Themes provide a consistent visual design for the pages in a Web site. Use this dialog box to select a theme and to preview the theme before applying it to a page.

Options

Preserve current theme

Select to retain the existing themes applied to pages in a Web project. Pages without a theme will use the theme selected in the Apply theme box.

Apply theme

List the current themes available from the path specified in the Directory box.

Preview

Displays a sample page showing the color scheme, navigation button graphics, bullet graphics, font, and banner style for the selected theme.

Directory

Lists the path to the theme files displayed in the Apply theme box.

Restore Default Directory

Select to return the path listed in the Directory box to the Microsoft® Visual InterDev™ default Themes directory.

Browse

Select to choose a set of themes in a different directory.
