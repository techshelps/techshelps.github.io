Debug setup: Debug user name or password not valid.

To debug server script, you must be able to provide the name and password of an Windows NT® user who has administration privileges on the server. The user name you provide is used to set the identity under which the Microsoft® Transaction Server (MTS) package for the out of process application runs. This error indicates that the name or password you provided:

  • Was not recognized.

  • Identifies a user who does have appropriate privileges for debugging.

For details about setting up a user, see Debugging Remotely.
