Insert New Class Dialog Box

      

Use this dialog box to add a new class to the Classes node or to any HTML tag in the CSS Editor window. You can then select the class and specify style settings that share format across different elements or that allow format variations for different instances of the same element.

Options

Class name

Select to specify a name for a set of styles that control the appearance of a category of elements on a page.

For example, you can create a class named Enhanced to modify the styles for several tags.

Apply only to the following tag

Select to set a specific tag that the class applies to.
