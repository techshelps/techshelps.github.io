Design-Time Controls Properties

Expand this node in the contents to see a complete listing of the properties that apply to Microsoft® Visual InterDev™ design-time controls.

These properties apply to the design-time controls only. For a list of properties that apply to the scripting object model, see Scripting Object Model Properties.

For a discussion of the scripting object model and design-time controls, see The Scripting Object Model. Also, Scripting with Design-Time Controls and Script Objects includes the introspective Creating Forms with Design-Time Controls.
