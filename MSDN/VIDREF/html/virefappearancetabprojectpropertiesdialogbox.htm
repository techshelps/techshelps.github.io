Appearance Tab (Project Properties Dialog Box)

      

Use this tab to change the default theme and layout applied to a Web project or to change the default navigation bar labels for links such as the home page.

Options

Default theme and Layout

Theme

Displays the name of the current theme applied by default to all new .htm and .asp files created in the Web project. To select a different theme, choose the Change button.

Layout

Displays the name of the current layout applied by default to all new .htm and .asp files created in the Web project. To select a different layout, choose the Change button.

Change

Displays the Apply Theme and Layout dialog box, which allows you to select a different default theme or layout for the Web project. Themes used in existing pages in the Web project will not change themes, but all new pages added to the project will use the new theme.

Navigation bar labels

Use the PageNavbar design-time control or layouts to add navigation bars to the pages in your Web project.

Home

Select to set the default navigation bar label for the home page of a Web project.

Previous

Select to set the default navigation bar label for a previous page.

Next

Select to set the default navigation bar label for a next page.

Parent

Select to set the default navigation bar label for the parent page.
