New Timeline Dialog Box

      

Use this dialog box to specify a new timeline. This dialog box is available by choosing New from the Timelines Properties Dialog Box.

Options

Timeline Name

Enter a name for the new timeline or leave the default name as provided, "Timeline n", where n is an auto-incremented number to ensure uniqueness. The name you enter can have up to 64 characters and must be unique for the page.

Automatically play Timeline when loaded

Select to play the timeline as soon as the page is loaded. This option is cleared by default. If you don't use this option, you need to reference the timeline and its play method in the script on your page.
