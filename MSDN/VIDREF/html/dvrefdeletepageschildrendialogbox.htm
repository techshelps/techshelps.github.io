Delete Pages and Children Dialog Box

      

Allows you to remove pages from the site diagram and the navigation bars for a site. You can also choose to delete pages from the site diagram and the Web project.

Note   To delete or remove a parent page without removing or deleting the child pages, you must detach all the child pages from the parent page before using the Delete or Remove commands.

Options

Remove these pages and all pages below them from all navigation bars

Removes the selected pages and their child pages from the diagram. These pages will no longer be available as links on any of the site's navigation bars. This option does not remove files from the Web project.

Delete these pages and all pages below them from the Web project

Deletes the selected pages and their child pages from the master Web application and removes the pages from the site diagram. These pages will no longer be available as links on the any of the site's navigation bars.
