Edit Timeline Dialog Box

      

Use this dialog box to change the name and AutoPlay state for the timeline. This dialog box is available by choosing Edit from the Timelines Properties dialog box.

Options

Name

Enter a new name for the timeline. The name you enter can have up to 64 characters and must be unique for the page.

Automatically play Timeline when loaded

Select to play the timeline as soon as the page is loaded. This option is select by default. When selected, you can still reference the timeline in the script on your page. If you clear this option, you need to call its Play() method through script.

Note   You can stop, pause, play, or seek a timeline, plus access its play state and elapsed time through script.
