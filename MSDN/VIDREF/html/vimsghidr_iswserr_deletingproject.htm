Cannot delete project <name> from the Web server.

You are trying to delete a project from the Web server using the Remove Project command, but are unable to do so. For example, you may not have permissions to delete projects on the Web server. The second sentence of this error describes the specific problem, as reported by the Microsoft® FrontPage® Server Extensions. This error is usually of the form:

[Server Error:   error text]

  • To correct this error, correct the condition described in the second sentence of this error.

For more information, see Reorganizing Project Structure.
