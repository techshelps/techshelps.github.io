PageNavbar Properties Dialog Box

      

Use this dialog box to set the navigation bar links, the appearance of the navigation bar, and other options for the PageNavbar design-time control.

The PageNavbar control uses the navigation structure designed in the site diagrams for a Web project to determine parent page, sibling page, child page, and global navigation bar links.

For more information, see Designing Site Navigation.

This dialog box appears when you select the PageNavbar control and then choose Property Pages from the View menu. You can also access this dialog box by right-clicking on the PageNavbar control on an HTML page in the HTML Editor.
