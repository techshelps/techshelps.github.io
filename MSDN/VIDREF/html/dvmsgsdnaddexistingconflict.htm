Adding this page to the global navigation bar will detach it from its parent. Continue?

Global navigation bar pages cannot have parent pages. Site Designer will automatically detach the selected page from its parent. Usually this is the result of selecting a child page to be a global navigation bar page.

  • If you select No, the page will not become a global navigation bar page and will not appear as a link on the global navigation bar.

For more information, see Adding a Page to all Site Navigation Bars and Site Diagram Legend.
