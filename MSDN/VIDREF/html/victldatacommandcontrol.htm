Data Command Design-Time Control

Creates commands to browse or modify a data source using a project data connection.

Note   This control is included for compatibility for Microsoft® Visual InterDev™ 1.0 only. For complete details, see your Visual InterDev 1.0 documentation.

In Visual InterDev 6.0, use the Recordset design-time control instead of the Data Command control.

Remarks

In Visual InterDev 1.0 projects, use the Data Command control to provide lookup functionality or to modify the data in a database. Use the Data Range Header and Data Range Header controls when you want to publish multiple rows of data in your page.
