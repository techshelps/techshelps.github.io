CPropertyPage::CancelToClose

Call this member function after an unrecoverable change has been made to the data in a page of a modal property sheet. This function will change the OK button to Close and disable the Cancel button. This change alerts the user that a change is permanent and the modifications cannot be canceled.

The CancelToClose member function does nothing in a modeless property sheet, because a modeless property sheet does not have a Cancel button by default.
