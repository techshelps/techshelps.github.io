CPoint::CPoint

Use this constructor to construct a CPoint object. If no arguments are given, x and y members are not initialized.

Syntax

CPoint( );

CPoint( int initX, int initY );

CPoint( POINT initPt );

CPoint( SIZE initSize );

CPoint( DWORD dwPoint );
