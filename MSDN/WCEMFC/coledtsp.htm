COleDateTimeSpan

The COleDateTimeSpan class represents a relative time, or time span. A COleDateTimeSpan keeps time in days. This class is used with its companion class COleDateTime.

At a Glance

Header file: Afxdisp.h
Platforms:
Versions: 2.0 and later
Complete documentation: Visual C++ documentation

Remarks

MFC for Windows CE does not support the Format member function of the COleDateTimeSpan class.
