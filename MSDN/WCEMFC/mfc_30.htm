Support Classes

MFC for Windows CE supports the following classes the same way the standard MFC library does, with the one exception that you can't use the SetCursor member function of the CRectTracker class.

CCmdUI

CDataExchange

CLongBinary

COleDispatchDriver

CPropExchange

CRectTracker

CWaitCursor

MFC for Windows CE doesn't support the COleCmdUI, CDaoFieldExchange, CDBVariant, CFieldExchange, and COleDataObject support classes.

To see where the support classes fit into the Windows CE class hierarchy, see Classes Not Derived from CObject.
