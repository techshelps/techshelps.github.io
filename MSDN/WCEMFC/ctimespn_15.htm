CTimeSpan Comparison Operators

Use these operators to compare two relative time values. They return nonzero if the condition is true; otherwise zero.

BOOL operator ==( CTimeSpan timeSpan ) const;

BOOL operator !=( CTimeSpan timeSpan ) const;

BOOL operator <( CTimeSpan timeSpan ) const;

BOOL operator >( CTimeSpan timeSpan ) const;

BOOL operator <=( CTimeSpan timeSpan ) const;

BOOL operator >=( CTimeSpan timeSpan ) const;
