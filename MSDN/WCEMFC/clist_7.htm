CList::GetAt

Call this member function to retrieve the element (or a reference to the element) associated with a given position. The position parameter is a key for the list. It is not the same as an index, and you cannot operate on the position value yourself.

Syntax

TYPE& GetAt( POSITION position );

TYPE GetAt( POSITION position ) const;
