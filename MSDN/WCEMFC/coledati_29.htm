COleDateTime::DateTimeStatus

The DateTimeStatus enumeration specifies the values that can be used in the COleDateTime::m_status data member to indicate whether the COleDateTime::m_dt data member contains a valid value. This enumeration is used with the COleDateTime::GetStatus and COleDateTime::SetStatus functions.

enum DateTimeStatus
valid =0,
invalid = 1,
null = 2
}
