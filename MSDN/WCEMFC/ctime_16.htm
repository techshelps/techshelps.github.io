CTime Comparison Operators

Use these operators to compare two absolute times and return nonzero if the condition is true; otherwise zero.

Syntax

BOOL operator ==( CTime time ) const;

BOOL operator !=( CTime time ) const;

BOOL operator <( CTime time ) const;

BOOL operator >( CTime time ) const;

BOOL operator <=( CTime time ) const;

BOOL operator >=( CTime time ) const;
