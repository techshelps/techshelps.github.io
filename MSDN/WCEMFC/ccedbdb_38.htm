CCeDBDatabase::m_CEOID

This data member contains the object identifier of the database, with respect to the object store.

CEOID m_CEOID;

At a Glance

Header file: Wcedb.h
Platforms:
Versions: 2.0 and later

Remarks

When you call either the Create or Open member function, it sets the CCeDBDatabase object's m_CEOID member to a non-NULL value.

Every object in the object store has a unique object identifier. A database is one type of object that can exist in the object store. Its object identifier is stored in its m_CEOID data member. For more information about object identifiers, see About Object Identifiers.
