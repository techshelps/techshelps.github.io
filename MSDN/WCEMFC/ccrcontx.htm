CCreateContext

The framework uses the CCreateContext structure when it creates the frame windows and views associated with a document.

{

CRuntimeClass* m_pNewViewClass;
CDocument* m_pCurrentDoc;

CView* m_pLastView;
CFrameWnd* m_pCurrentFrame;
CCreateContext();

};
