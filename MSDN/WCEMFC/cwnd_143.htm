CWnd::OnSysKeyDown

This member function is called by the framework, if the CWnd object has the input focus, when the user holds down the ALT key and then presses another key. If no window currently has the input focus, the active window's OnSysKeyDown member function is called. The CWnd object that receives the message can distinguish between these two contexts by checking the context code in nFlags.

Syntax

afx_msg void OnSysKeyDown( UINT nChar, UINT nRepCnt, UINT nFlags );
