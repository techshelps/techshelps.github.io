CCeDBDatabase Data Members

Data Members
m_bAutoSeekNext A flag that specifies whether to call SeekNext when calling ReadCurrRecord. The default is FALSE.
m_bEOF The end-of-file flag. Set when m_bAutoSeekNext is set to TRUE and ReadCurrRecord reads past the end of the database.
m_CEOID The database's object identifier.
m_hHandle The database handle.
