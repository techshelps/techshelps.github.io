COleDispatchDriver::COleDispatchDriver

Use this constructor to construct a COleDispatchDriver object.

The default constructor creates a COleDispatchDriver object but does not connect the IDispatch interface. Before using COleDispatchDriver object, you should connect an IDispatch interface to it by using either COleDispatchDriver::CreateDispatch or COleDispatchDriver::AttachDispatch.

The form COleDispatchDriver(LPDISPATCH lpDispatch, BOOL bAutoRelease = TRUE) connects the IDispatch interface.

The third form is the copy constructor.

Syntax

COleDispatchDriver( );

COleDispatchDriver( LPDISPATCH lpDispatch, BOOL bAutoRelease = TRUE );

COleDispatchDriver( const COleDispatchDriver& dispatchSrc );
