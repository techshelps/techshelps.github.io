COleStreamFile

A COleStreamFile object represents a stream of data (IStream) in a compound file as part of COM Structured Storage. An IStorage object must exist before the stream can be opened or created unless it is a memory stream. COleStreamFile objects are manipulated exactly like CFile objects.
