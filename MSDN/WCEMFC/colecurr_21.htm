COleCurrency::CurrencyStatus

The CurrencyStatus enumeration specifies the values that can be used in the COleCurrency::m_status data member to indicate whether the COleCurrency::m_cur data member contains a valid value. This enumeration is used with the COleCurrency::GetStatus and COleCurrency::SetStatus functions.

enum CurrencyStatus{
valid = 0,
invalid = 1,
null = 2,
};
