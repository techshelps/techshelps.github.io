Global MFC Functions Not Supported in Windows CE

Windows CE doesn't support the following MFC global functions.

AfxDaoInit AfxOleLockControl
AfxDaoTerm AfxOleSetEditMenu
AfxDbInit AfxOleSetUserCtrl
AfxDbInitModule AfxOleTypeMatchGuid
AfxGetInternetHandleType AfxOleUnlockApp
AfxInitRichEdit AfxOleUnlockControl
AfxNetInitModule AfxOleParseURL
AfxOleCanExitApp AfxThrowDaoException
AfxOleGetMessageFilter AfxThrowDBException
AfxOleGetUserCtrl AfxThrowInternetException
AfxOleLockApp
