CSize::CSize

Use this constructor to construct a CSize object. If no arguments are given, cx and cy members are not initialized.

Syntax

CSize( );

CSize( int initCX, int initCY );

CSize( SIZE initSize );

CSize( POINT initPt );

CSize( DWORD dwSize );
