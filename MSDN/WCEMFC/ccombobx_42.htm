CComboBox::SetItemDataPtr

This member function sets the 32-bit value associated with the specified item in a combo box to be the specified pointer (void*). This pointer remains valid for the life of the combo box, even though the item's relative position within the combo box might change as items are added or removed. Hence, the item's index within the box can change, but the pointer remains reliable.

Syntax

int SetItemDataPtr( int nIndex, void* pData );
