CCeDBProp::GetFiletime

This member function retrieves the value of a property, when the property's data type is a FILETIME. (CeDBProp::Type_Filetime).

Syntax

FILETIME GetFiletime( ) const;

At a Glance

Header file: Wcedb.h
Platforms:
Versions: 2.0 and later

Return Values

The value of the FILETIME property.

Remarks

You can only call GetFiletime on a property of the type CeDBProp::Type_Filetime.
