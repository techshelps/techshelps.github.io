CWnd::OnHScroll

This member function is called by the framework when the user clicks a window's horizontal scroll bar. The SB_THUMBTRACK scroll-bar code typically is used by applications that give some feedback while the scroll box is being dragged.

Syntax

afx_msg void OnHScroll( UINT nSBCode, UINT nPos, CScrollBar* pScrollBar );
