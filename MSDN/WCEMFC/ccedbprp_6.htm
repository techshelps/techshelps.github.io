CCeDBProp::GetLong

This member function retrieves the value of a property, when the property's data type is an unsigned long integer (CCeDBProp::Type_Long).

Syntax

long GetLong( ) const;

At a Glance

Header file: Wcedb.h
Platforms:
Versions: 2.0 and later

Return Values

The value of the property.

Remarks

You can only call GetLong on a property of the type CeDBProp::Type_Long.
