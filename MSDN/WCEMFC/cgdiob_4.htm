CGdiObject::CreateStockObject

Use this member function to retrieve a handle to one of the predefined stock pens, brushes or fonts.

Syntax

BOOL CreateStockObject ( int nIndex );

At a Glance

Header File Afxwin.h
Platforms
Versions 1.0 and later
Complete documentation Visual C++ documentation

Remarks

Windows CE doesn't support the following stock objects in the nIndex parameter.

ANSI_FIXED_FONT

ANSI_VAR_FONT

DEFAULT_PALETTE

OEM_FIXED_FONT

SYSTEM_FIXED_FONT
