CTime::GetMonth

Call this member function to retrieve the month, based on local time, in the range 1 through 12 (1 = January). This function calls GetLocalTm, which uses an internal, statically allocated buffer. The data in this buffer is overwritten as a result of calls to other CTime member functions.

Syntax

int GetMonth( ) const;
