CDocTemplate::AddDocument

Call this member function to add a document to a template. The derived class CSingleDocTemplate overrides this function. If you derive your own document-template class from CDocTemplate, your derived class must override this function.

Syntax

virtual void AddDocument( CDocument* pDoc );
