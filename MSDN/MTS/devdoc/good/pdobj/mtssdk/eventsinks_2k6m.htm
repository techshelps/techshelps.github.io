IMtsPackageEvents::OnPackageActivation

Indicates that a package has been activated.

HRESULT OnPackageActivation(
LONGLONG        perfCount,
REFGUID            guidPackage,
LPCOLESTR    sPackageName);

Parameters

perfCount
The current value of the high-resolution performance counter, if one exists.

guidPackage
The GUID associated with the package.

sPackageName
A string indicating the name of the package.
