ITransactionOutcomeEvents::Commit

This event is raised when the transaction committed.

HRESULT Committed (
BOOL
 
fRetaining,
 
XACTUOW *
 
pNewUOW,
 
HRESULT
 
hr);

Parameters

fRetaining [in]
Indicates whether retaining Commit was specified. Will be FALSE.

pNewUOW [in]
Always NULL.

hr [in]
Always S_OK.

Return Values

S_OK
Success.
