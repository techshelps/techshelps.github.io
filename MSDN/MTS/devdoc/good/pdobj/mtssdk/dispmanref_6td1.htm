IDispenserDriver::DestroyResource

Holder asks resource dispenser to destroy a resource.

HRESULT IDispenserDriver::DestroyResource (
const RESID
 
ResId);

Parameters

ResId [in]
The resource that DispMan is asking resource dispenser to destroy.

Return Values

S_OK
Success.

E_NOTIMPL
For resource dispensers which do not support numeric RESIDs.

E_FAIL
Failure.
