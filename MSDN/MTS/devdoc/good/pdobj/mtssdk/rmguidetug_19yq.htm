More Information on Microsoft Distributed Transaction Coordinator

See the following documents for more information about topics related to Microsoft Distributed Transaction Coordinator (MS DTC):

  • Microsoft Distributed Transaction Coordinator (MS DTC) Administrator’s Guide and Programmer’s Reference— describes how to install, configure, and administer MS DTC. It also describes how to write programs that use MS DTC transactions.
  • Transaction Processing: Concepts and Techniques by Jim Gray and Andreas Reuter (Morgan Kaufmann Publishers, ISBN 1-55860-190-2)—the definitive work on transaction processing
  • Inside OLE, second edition by Kraig Brockschmidt (Microsoft Press ISBN 1-55615-843-2)—provides an introduction to OLE. Chapters 1 and 2 describe the basic concepts behind the Microsoft Component Object Model (COM).
