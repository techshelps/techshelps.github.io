SRESID

A Unicode string version of RESID, used in the TrackResourceS, UntrackResourceS and DestroyResourceS methods. Strings are sometimes useful when only a small amount of information needs to be recorded about a resource, and the entire description of the resource can be contained in the SRESID. In particular, using the SRESID can sometimes eliminate the need for a map in the resource dispenser when the resource represents a relationship between two (or more) things.
