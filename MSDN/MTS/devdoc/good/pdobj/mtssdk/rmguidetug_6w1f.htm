Microsoft Distributed Transaction Coordinator Features

Microsoft Distributed Transaction Coordinator (MS DTC) can be used to:

  • Update two databases from a single application.
  • Update a database and a file from a single application.
  • Update a geographically distributed database.
  • Update a database partitioned for scalability.

You can also use an OLE Transactions-compliant resource manager with an X/Open-compliant transaction-processing monitor.
