Dispenser Manager (DispMan)

DispMan is the controlling executive that manages the loading and unloading of resource dispensers, interfaces with the MTS Executive for context, and it controls the Inventory Statistics Manager.

DispMan polls each holder every 10 seconds to allow them to readjust their resource inventory. Each holder calls the Inventory Statistics Manager to suggest inventory levels for each type of resource. As a result, the holder may ask the resource dispenser to either create or destroy some inventory.
