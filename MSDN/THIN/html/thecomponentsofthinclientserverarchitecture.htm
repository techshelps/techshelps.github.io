The Components of Thin-Client/Server Architecture

As you saw in the first two chapters, WinFrame supports enterprise application deployment using a thin-client/server architecture. The key pieces of the WinFrame system, as it exists today, include the ICA protocol, the universal thin client, and the Win-Frame MultiWin multiuser application server technology.
