The Client Side: The ICA Web Client

The Citrix WinFrame Web Client, available in a 16-bit version for Windows 3.x users and a 32-bit version for Windows 95 and Windows NT users, automatically configures most browsers when it’s installed. The WinFrame Web Client is smaller than the standard WinFrame Client, which is about 250 KB. After downloading the client, you must exit from your browser; otherwise, the installation process will not complete properly. When you restart the Web browser, the WinFrame Web Client extensions are fully integrated into the Web browser and available for use.
