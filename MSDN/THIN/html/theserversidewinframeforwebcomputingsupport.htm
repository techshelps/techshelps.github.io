The Server Side: WinFrame for Web Computing Support

WinFrame Web Computing support is automatically loaded during installation of the WinFrame server for thin-client/server computing. This section describes some of the features of Win-Frame Web Computing support for your thin-client/server infrastructure. These features include the following:

  • The ability to allow anonymous users as well as explicit users to access your applications

  • ALE

  • Security

These features are covered in depth later in this chapter.
