Class Store Provider for ADSI/LDAP

The Class Store Provider object for ADSI/LDAP is implemented by the COM libraries as an in-process COM object. It accesses a class store in the Active Directory.

The class store provider for ADSI/LDAP, like all class store provider objects, is a moniker object. Typically, clients obtain an initialized provider object by calling MkParseDisplayName on an element of the class store search path.
