Link Tracking and NTFS

[This is preliminary documentation and subject to change.]

The Windows NT File System (NTFS), version 5.0 supports Object Identifiers (OIDs) on files. A file can be opened by specifying its OID. Part of the link source's LSID is an NTFS 5.0 Object ID.

The Windows NT 5.0 link-tracking service only maintains links as long as a link source remains on NTFS 5.0 volumes.
