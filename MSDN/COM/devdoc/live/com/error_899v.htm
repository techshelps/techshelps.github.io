HRESULTs

The values of some HRESULTs have been changed in COM to eliminate all duplication and overlapping with Win32 error codes. Those that duplicate Win32 error codes have been changed to FACILITY_WIN32 and those that overlap remain in FACILITY_NULL. Following is a list of their 32-bit values:

HRESULT Value
E_UNEXPECTED 0x8000FFFF
E_NOTIMPL 0x80004001
E_OUTOFMEMORY 0x8007000E
E_INVALIDARG 0x80070057
E_NOINTERFACE 0x80004002
E_POINTER 0x80004003
E_HANDLE 0x80070006
E_ABORT 0x80004004
E_FAIL 0x80004005
E_ACCESSDENIED 0x80070005
