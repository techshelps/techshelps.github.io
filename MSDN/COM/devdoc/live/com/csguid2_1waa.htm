Class Store Manager

To interpret the class store search path (computer, user, OU) and conduct searches in all class stores listed in the search path, the Class Store Manager object is implemented as part of the COM client-side libraries. This object allows read-access to a set of class stores according to the calling user credentials. Modifications to class stores require direct access to a specific class store object.

The Class Store Manager manages the client-side class store objects by implementing IClassAccess (named and created by monikers). It provides an extensibility hook for third-party providers to create new Class Store objects.
