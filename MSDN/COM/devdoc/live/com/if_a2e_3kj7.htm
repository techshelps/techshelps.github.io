IDirectWriterLock::HaveWriteAccess

[This is preliminary documentation and subject to change.]

Indicates whether the write lock has been taken.

HRESULT HaveWriteAccess(void);

Return Values

S_OK
The storage object is currently locked for write access.
S_FALSE
The storage object is not currently locked for write access.

QuickInfo
