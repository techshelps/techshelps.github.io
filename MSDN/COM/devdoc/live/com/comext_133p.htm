COM Security and Windows 95

Prior to the release of the DCOM-enabled version of COM for Windows 95 there were no security issues related to COM on Windows 95 because COM on Windows 95 had no security. The DCOM-enabled version of COM for Windows 95 includes a subset of the COM security found on Windows NT. For more information, see:

  • Call-level Security in Windows 95
  • Remote Connections and Activation in Windows 95
  • Security on Windows 95-only Networks
