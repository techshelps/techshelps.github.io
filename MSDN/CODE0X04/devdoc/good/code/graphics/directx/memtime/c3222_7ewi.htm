MEMTIME 
 
This is an x86-only sample. 
 
Memory bandwidth tests.  Times various operations on system and video 
memory. Memtime is intended to show the advantages and disadvantages of  
different drawing methods.  
 
BUILDING WITH MICROSOFT VISUAL C++ 4.0 AND HIGHER 
 
From the File/Open Workspace menu, open the makefile file and reply 
'Yes' when asked if you wish to create a wrapper project.
