Setup Sample (x86 only) 
----------------------- 
 
This sample is an example of how to use DirectXSetup APIs  
to install DirectX subsystem and DirectX drivers.  It also  
serves as a working template for ISV's to install a game. 
                                                           
Note that this program will not run until the \Mssdk\Redist\DirectX5  
directory is installed into the same directory as setup.exe.  The  
entire tree must be copied in order to setup correctly.
