The class is a specialization of template class vector for elements of type bool. It alters the definition of four member types (to optimize the packing and unpacking of elements) and adds two member functions. Its behavior is otherwise the same as for template class vector.

In this implementation, if partial specializations are not supported or if bool is not a distinct type, the class should be referred to by the synonym _Bvector.
