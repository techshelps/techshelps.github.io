_variant_t::Detach

Microsoft Specific

VARIANT Detach( ) throw( _com_error );

Return Value

The encapsulated VARIANT.

Remarks

Extracts and returns the encapsulated VARIANT, then clears this _variant_t object without destroying it. This member function removes the VARIANT from encapsulation and sets the VARTYPE of this _variant_t object to VT_EMPTY. It is up to you to release the returned VARIANT by calling the VariantInit function.
