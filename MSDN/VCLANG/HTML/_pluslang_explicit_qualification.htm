Explicit Qualification

A name in a class or namespace can be accessed using an explicit qualifier.

Syntax

id-expression :

unqualified-id
qualified-id

nested-name-specifier :

class-or-namespace-name
:: nested-name-specifieropt

class-or-namespace-name :

class-name
namespace-name

namespace-name :

original-namespace-name
namespace-alias

This is very similar to using the scope operator to resolve access to a member of a class. For more information, see Qualified Names in Chapter 4.
