Member-Access Control

With C++, you can specify the level of access to member data and functions. There are three levels of access: public, protected, and private. This chapter explains how access control applies to objects of class type and to derived classes. This chapter includes the following topics:

  • Controlling access to class members

  • Access specifiers

  • Access specifiers for base classes

  • Friends

  • Protected member access

  • Access to virtual functions

  • Multiple access
