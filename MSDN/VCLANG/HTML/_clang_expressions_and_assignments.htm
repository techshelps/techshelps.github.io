Expressions and Assignments

This chapter describes how to form expressions and to assign values in the C language. Constants, identifiers, strings, and function calls are all operands that are manipulated in expressions. The C language has all the usual language operators. This chapter covers those operators as well as operators that are unique to C or Microsoft C. The topics discussed include:

  • L-value and r-value expressions

  • Constant expressions

  • Side effects

  • Sequence points

  • Operators

  • Operator precedence

  • Type conversions

  • Type casts
