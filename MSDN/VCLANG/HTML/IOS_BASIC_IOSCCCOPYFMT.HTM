basic_ios::copyfmt

basic_ios& copyfmt(const basic_ios& rhs);

The member function reports the callback event erase_event. It then copies the fill character, the tie pointer, and the formatting information from rhs into *this. Before altering the exception mask, it reports the callback event copyfmt_event. If, after the copy is complete, state & exceptions() is nonzero, the function effectively calls clear with the argument rdstate(). It returns *this.
