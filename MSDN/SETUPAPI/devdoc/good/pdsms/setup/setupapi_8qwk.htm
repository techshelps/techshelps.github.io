SetupRemoveInstallSectionFromDiskSpaceList

[New for Windows NT 4.0 Service Pack 3.]

The SetupRemoveInstallSectionFromDiskSpaceList function searches an Install section of an INF file for CopyFiles and DelFiles lines, and removes the file operations specified in those sections from a disk-space list.

BOOL SetupRemoveInstallSectionFromDiskSpaceList(
  HDSKSPC DiskSpace,     // handle to the disk-space list
  HINF InfHandle,        // handle to INF file
  HINF LayoutInfHandle,  // optional, handle to the layout INF file
  PCTSTR SectionName,    // specifies the Install section name
  PVOID Reserved1,       // must be zero
  UINT Reserved2         // must be zero
);

Parameters

DiskSpace
Supplies a handle to a disk-space list.
InfHandle
Supplies a handle to an open INF file that contains the Install section. If LayoutInfHandle is not specified, the INF file must also contain the section specified by SectionName. This handle must be for a Windows 95- or Windows NT 4.0-style INF file.
LayoutInfHandle
This optional parameter supplies the handle to the INF file that contains the SourceDisksFiles section(s). Otherwise, that section is assumed to exist in the INF file specified by InfHandle.
SectionName
Supplies the name of the section to be added to the disk-space list.
Reserved1
Must be 0.
Reserved2
Must be 0.

Return Values

If the function succeeds, the return value is a non-zero value.

If the function fails, the return value is zero. To get extended error information, call GetLastError.
