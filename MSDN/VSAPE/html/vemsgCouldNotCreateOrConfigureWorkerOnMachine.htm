Could not create or configure a worker on machine

   

An error occurred when trying to create a worker on the named computer.

Try the following:

  • Confirm that the remote computer is still turned on and logged on to the network.

  • Confirm that a valid APE (client or server) installation exists on the computer.

  • Confirm that you have valid security rights to the remote computer.

  • Confirm that the necessary DCOM and Remote Automation files and settings exist on the computer.

For More Information   For information on installing APE components, see Installing Application Performance Explorer.
