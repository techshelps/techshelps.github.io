Configure Remote Client Machines Dialog Box

   

Use this dialog box to specify each remote client machine that is available for the test.

Options

Add to Test

Adds the test machine entered in the Client Machine Name box to the list of client test machines available for the test.

Client Machine Name

Provides a place to specify the name of a remote client machine. Enter the name and then click Add to Test to add this client machine to the list of remote client test machines.

Client Test Machines

Displays the list of active remote client machines. To delete a listed machine, select the machine name and click Remove. To eliminate a listed machine from the test but still keep the machine in the list, clear the check box beside the machine name.

Remove

Removes the selected client test machine from the machines available for the test.

Use Client Manager Machine as Client Machine

Specifies that the client manager machine should also be used as a client machine (in addition to the listed client test machines). This option is the default when no remote clients are selected. However, when remote clients are specified, you can additionally use the client manager machine as a client.
