General (Client Options Dialog Box)

   

Use the General tab to configure the delay between calls and how long the test will run.

Options

Continue Until Stopped

Specifies that the test will run continuously until you click Stop.

Delay After Calls: (ms)

Specifies the time in ms that each client process waits between calls.

+/-

Specifies the amount of delay variation (using Delay After Calls as a base) for data returned to the client objects.

Number of Calls

Specifies the duration of the test as a certain number of calls.

Number of Minutes

Specifies the duration of the test in minutes.
