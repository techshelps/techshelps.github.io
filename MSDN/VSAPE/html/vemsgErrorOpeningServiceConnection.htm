Error opening service connection

   

An error occurred while trying to establish a database connection. Error text appended to this error message describes the specific error that occurred.

Try the following:

  • If you are using the Jet database, make sure that the Apetest.mdb database exists at the location where APE is installed on all the computers that provide workers.

  • If you are using a SQL Server database, make sure that the SQL Server database is set up correctly and that the correct user ID and password is specified at the start of the test.

For More Information   For information on installing the APE SQL Server database, see Installing Application Performance Explorer.
