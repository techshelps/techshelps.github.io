Create Log File Dialog Box

   

Use this dialog box to specify the name and location of the log file.

Options

File name

Provides a place to specify the log file name.

Save

Creates the new log file in the specified folder.

Save as type

Lists the available log file types (*.log is recommended).

Save in

Lists the folders where you can save the log file.

Save in list

Lists the items in the current Save folder.
