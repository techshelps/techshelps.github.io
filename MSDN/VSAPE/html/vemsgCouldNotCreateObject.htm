Could not create object

   

This standard error is reported when there has been a failure in creating a COM object — in this case while the APE Manager is preparing to start a test.

Try confirming that the installation of all local APE components and any server installations being used in the current profile were successful. If not, you may need to reinstall either the APE client or server, as appropriate.

For More Information   For information on installing APE, see Installing Application Performance Explorer.
