Customizing Application Performance Explorer

   

You can modify the APE components to more closely model your enterprise application. The complete Visual Basic source code and project files for each component (except Aemanagr) are located in the ..\Program Files\Microsoft Visual Studio\Common\Tools\Ape\Source subdirectory of your Visual Studio installation.

Note   The compatible executables and DLLs for Application Performance Explorer are shipped in the ..\Program Files\Microsoft Visual Studio\Common\Tools\Ape\Source\AECompat directory. Except for the MTS Service (Aemtssvc.dll) and the Expediter (Aeexpdtr.dll), all components use Project, not Binary, compatibility.
