Unexpected error

   

An unexpected error occurred in APE.

Try the following:

  • Close and restart the APE Manager.

  • If the problem persists, confirm that the installation of all local APE components and any remote installations being used in the current profile were successful. If not, you may need to reinstall an APE client or server, as appropriate.

For More Information   For information on installing APE components, see Installing Application Performance Explorer.
