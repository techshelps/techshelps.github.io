Error posting to account

   

An error occurred while trying to create the MTS Account Service.

Try the following:

  • Confirm that MTS is installed correctly on the computer set up to run the APE Services. This is configured using the Application Services Options dialog box within the APE Manager.

  • Confirm that the APE Package has been installed correctly into MTS. See the APE setup help for information on how to do this.

For More Information   For information on installing APE components, see Installing Application Performance Explorer.
