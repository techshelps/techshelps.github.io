Application Performance Explorer Main Window

   

Use the APE Main Window to select the test Profile, start, stop, and close the test, and clear the log file. Additionally, you can click any of the five large buttons to display additional dialog boxes to completely configure your APE test.

Options

Clear

Clears the summary window. As a general guideline, you should delete the summary information prior to running another test. After deleting the summary information, the log file still retains all of the detailed run-time statistics.

Client Options

Displays the Client Options dialog box.

Client Service Connection

Displays the Application Service Connection Options dialog box.

Database Connection

Displays the Database Connection Options dialog box.

Database Options

Displays the Database Options dialog box.

Profile

Displays a list of available test profiles within the current profile collection. You can select any profile, modify it using the configuration dialog boxes, and then save it as a customized profile with a unique name.

To load a profile

  • Select a profile from the Profile list.

To save the selected profile

  • Click Save.

To delete the selected profile

  • Click Delete.

To select a different profile collection

  • From the File menu, click Open Profile Collection.

Service Options

Displays the Application Service Options dialog box.

Start

Begins the APE test based on the current profile settings.

Stop

Stops the currently running APE test. The run-time statistics up to the time of stopping remain available.
