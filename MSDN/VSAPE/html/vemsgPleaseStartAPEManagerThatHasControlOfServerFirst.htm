Please start the APE Manager that has control of the server first

   

This message will normally appear when you have another instance of the APE Manager already running.

Try the following:

  • Close down the version of APE that you are currently running and use the instance that was already running.

  • If the preceding does not fix this problem, or no other instance seems to be already running, start up the Windows Task Manager and confirm that another instance isn't still in memory. If another instance is in memory, close it down and try again.
