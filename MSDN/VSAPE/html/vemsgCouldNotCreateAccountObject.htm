Could not create account object

   

An error occurred while trying to create the MTS Account Service.

Try the following:

  • Confirm that MTS is installed correctly on the computer running APE Services. This is determined using the Application Services Options dialog box within the APE Manager.

  • Confirm that the APE Package has been installed correctly into MTS.

  • Confirm that you have security rights to call the MTS package.

For More Information   For information on installing APE, see Installing Application Performance Explorer.
