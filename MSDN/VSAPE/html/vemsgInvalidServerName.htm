Invalid Server Name

   

The server name specified in the Application Service Connection Options dialog box cannot be found.

Try the following:

  • From the APE Manager View menu, click Service Connection Options. In the Server Name box, enter the name of a valid network computer with either an APE client or server installed.

  • Confirm that the remote computer is running AutMgr32.exe if you are using Remote Automation and have the necessary security rights set to allow for the use of distributed COM.
