Concurrency (Client Options Dialog Box)

   

Use the Concurrency tab to configure the number and location of client processes.

Options

Allow Externally Hosted Clients

Specifies whether additional client processes can be added to a test machine by other APE tests. If this check box is selected, other APE tests can start client processes on one or more of your test machines. If this check box is cleared, the test profile is essentially locked and the test will run with the configured client workload.

Client Processes per Machine

Specifies the number of client processes created on each machine.

Configure

Displays the Configure Remote Client Machines dialog box, where you can modify the list of remote client machines.

Use Remote Client Machines

Specifies the use of client processes on remote machines (instead of local).

When this check box is selected, you can configure Application Performance Explorer (APE) to start client processes on remote machines. Selecting this box displays the Configure Remote Client Machines dialog box, where you can create a list of remote machines.
