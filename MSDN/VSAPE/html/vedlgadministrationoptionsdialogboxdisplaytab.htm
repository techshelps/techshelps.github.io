Display (Administration Options Dialog Box)

   

Use the Display tab to request views of various APE progress statistics as the test is running. Displaying progress requires considerable machine resources and may distort the run times of your test.

Options

Display Client Progress

Specifies that you want to display the client progress.

Display Expediter Progress

Specifies that you want to display the expediter progress.

Display Job/Pool Manager Progress

Specifies that you want to display the queuing and pooling progress.

Display MTS Progress

Specifies that you want to display the MTS progress.
