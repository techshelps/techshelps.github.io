Log Setup (Administration Options Dialog Box)

   

Use the Log Setup tab to configure the logging options.

Options

Browse

Displays the Create Log file dialog box, where you can select an existing log file.

Clear Log

Deletes the current log file.

Flush log file to disk every: (KB)

Specifies that the log records are held in memory up to the kilobyte threshold, at which time the log records are written to the disk log file. Writing log records directly to disk can slow the currently running APE test. Holding the log records in memory postpones the disk I/O and helps your test run faster.

Log File

Specifies the log file name. All log records and error messages generated by the components are written to this file.

Maximum log file size: (MB)

Specifies the maximum disk space that will be used for the log file. If the log file size is exceeded, the earliest log entries are deleted and the newest log entries are appended to the end of the file.

Overwrite the file at the start of each new APE session

Indicates that the APE Manager should overwrite the log file every time a test runs. If this check box is cleared, the APE Manager appends all new log records to the existing file.

View

Displays the current log file in Notepad.
