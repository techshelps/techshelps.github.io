Logging (Administration Options Dialog Box)

   

Use the Logging tab to select the event types that are logged.

Options

Log Client Events

Specifies that client events are logged.

Log Database Events

Specifies that database events are logged.

Log Expediter Events

Specifies that expediter events are logged.

Log Job/Pool Manager Events

Specifies that queuing and pooling events are logged.

Log MTS Events

Specifies that MTS events are logged.

Log Summary Information

Specifies that the APE Manager prints summary information to the log file. The summary information includes a profile summary that is at the beginning of the log, and a performance summary that is at the end of the log.

Log Worker Events

Specifies that worker events are logged.
