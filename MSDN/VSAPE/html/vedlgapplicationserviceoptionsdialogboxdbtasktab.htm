DB Task (Application Service Options Dialog Box)

   

Use the DB Task tab to configure the database resource workload.

Options

Database Query

Specifies that a named database query will be part of the test. When Database Query is checked, you must specify a valid query name.

Perform Database Task

Specifies that database access is part of the test.

Query

Provides a place to specify the name of the query that will be part of the test.

SQL

Displays the SQL statements that compose the query.

Write Distributed MTS Transactions

Specifies that the test uses Microsoft® Transaction Server transactions.
