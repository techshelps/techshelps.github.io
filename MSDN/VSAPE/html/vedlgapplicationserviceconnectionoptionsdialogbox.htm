Application Service Connection Options Dialog Box

   

Use this dialog box to configure the remote application server.

Options

Connection Type

Specifies the type of connection used to connect to remote machines in order to launch the server components and remote clients. The available options include:

  • Distributed COM

  • Remote Automation

Note   Every machine that hosts remote APE components (such as workers, services, or clients) must be running Microsoft® Windows NT® 4.0. The APE Manager machine must be running either Windows NT 4.0 or Windows® 95 (or higher).

Local (COM)

Specifies using APE server components on the local machine.

RA Protocol

Specifies the type of Remote Automation protocol for connecting to remote machines. The available options include:

  • TCP/IP

  • Named Pipes

Note   To use Remote Automation, the Automation Manager must be running on every machine except the APE Manager machine.

Remote

Specifies using APE server components on a remote machine. When Remote is checked, you must specify a valid network server name.

Server Name

Provides a place to specify the name of the remote machine where the APE server components will run.
