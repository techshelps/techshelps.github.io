User's Guide

The Visual J++ User’s Guide contains the following topics:

  • Creating Projects   Explains how to create, manage, and build a Visual J++ project. Also shows how to import a Visual J++ 1.1 project.

  • Designing Forms   Shows how to create and modify forms using the Rapid Application Development (RAD) features of the Forms Designer.

  • Editing Code   Explains how to use IntelliSense features, such as Statement Completion, in the Text editor. Provides information on using Class Outline and the Component Builder to modify your code, and discusses the features of the Object Browser to view Java and COM-based components.

  • Accessing Data   Shows how to use the WFC data controls and the Data Form Wizard to access data on a form.

  • Introducing Wizards and Builders   Provides an overview of the Visual J++ wizards and builders.

  • Debugging Applications   Discusses how to debug applications, applets, and components.

  • Packaging and Deploying Projects   Shows how to package and deploy your projects.

  • Managing Projects with Source Code Control   Explains how to use a source code control program, such as Microsoft Visual Source Safe, with your projects.

  • Compiling Applications with JVC   Discusses how to use the JVC compiler and provides a reference for command-line options.

  • Viewing Applications with JVIEW and WJVIEW   Discusses how to use the JVIEW and WJVIEW tools to view applications and applets.
