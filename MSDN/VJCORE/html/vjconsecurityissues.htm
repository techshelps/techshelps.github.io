Security Issues

Although J/Direct is a very powerful feature for stand-alone Java applications and trusted intranet Web applications, it is clearly too powerful to be used by normal Java applets on the Web. This section describes how J/Direct works with the security system of the Microsoft VM to prevent untrusted code from abusing the power provided by J/Direct.
