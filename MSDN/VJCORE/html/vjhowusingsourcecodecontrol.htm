Managing Projects with Source Code Control

The Visual J++ environment integrates source control systems (such as Visual SourceSafe) into the Project Explorer and other menus and windows. You can use source control to manipulate source file (item) locations and versions, set up your source control project hierarchy to match your development directory tree, and more effectively manage team projects by delaying additions, deletions, and renames to the source control database.
