Setting Project Options

   

Visual J++ provides a number of options that allow you to configure your application for debugging, optimize your project's compiled output, expose classes within your project as COM objects, and specify the paths for your class's dependencies. You can find all of these options in the <Project> properties dialog box (where <Project> is the name of your currently selected project). You can display these options by clicking the Project menu and selecting the <Project> Properties menu command.
