Navigation Keyboard Shortcuts for the Auto, Locals, and Watch Windows

   

You can use the following keys to view or hide member variables of a selected class or array or to move around in the Auto, Locals, or Watch windows:

To Use the
Collapse the member variables list LEFT ARROW
View the member variables RIGHT ARROW
Toggle between hiding and viewing the variables ENTER
Move upward in the member variables list UP ARROW
Move downward in the member variables list DOWN ARROW
Move up an expanded list LEFT ARROW
Move down an expanded list RIGHT ARROW

Note   You can change the size of a column by dragging the border of the column header to the right to make the column larger or to the left to make it smaller.
