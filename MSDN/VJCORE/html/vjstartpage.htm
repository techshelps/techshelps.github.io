Visual J++

Welcome to this preview edition of Microsoft Visual J++, the tool that offers a high-productivity visual environment and integrated high-performance components to create, test, tune, and deploy Java code. Although not all features are enabled in this release, we encourage you to explore and try the product.

Visual J++ is a member of the Visual Studio family of development products which includes:

  • Visual Basic
  • Visual C++
  • Visual FoxPro
  • Visual InterDev
  • Visual J++
  • Visual SourceSafe
  • MSDN Library
