Browsing Packages and Libraries with the Object Browser

The Object Browser provides a convenient way to view the contents of Java packages and COM libraries. You can quickly browse Java or COM-based components without actually adding them to your project, even if you don't have the source code for these components.

Using the Object Browser, you can:

  • View the classes and members contained in the packages and libraries

  • Select the packages and libraries to be browsed

  • Filter the classes and members

  • Group and sort the classes and members

To open the Object Browser, point to Other Windows on the View menu and click Object Browser.
