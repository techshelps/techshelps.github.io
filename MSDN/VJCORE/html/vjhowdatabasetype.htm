Database Type Step

In the Database Type step of the Data Form Wizard, you specify the format of your database.

To specify the database format

  1. Select the Access option for any Microsoft Access .mdb file. Click Next to specify the database file.

  2. Select the ODBC option to access a database through ODBC, such as an ISAM database (dBase, FoxPro, or Paradox) or a remote data source (SQL Server or Oracle). Click Next to specify the ODBC connection information.

    Note   Although you can use the Microsoft Access ODBC driver to connect to an Access database, selecting the Access option provides better performance.
