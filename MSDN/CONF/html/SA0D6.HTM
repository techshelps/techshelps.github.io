Builders and Wizards

A builder is a Microsoft Access component that simplifies a task. For example, the Expression Builder allows you to create simple or complex expressions. This is different from a wizard, which asks you questions to perform a task.

There are a number of different builders available in Microsoft Access 2.0. These range from the Expression Builder, which is used to generate complex expressions, to the menu Builder, which is used to create menus and menu structures for your application.

In this section, we will discuss three different builders: the Expression Builder, the Code Builder, and the Menu Builder. There are other builders available. Please refer to the documentation or online Help.
