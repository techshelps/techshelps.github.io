When our query is submitted, we have some simplified measures of data volume and times. The measurements are shown below.

  • Query transmission: 57 bytes
  • Processing transmission: 0 bytes
  • Processing time on server: 1 seconds
  • Results transmission: Less than 6 bytes
  • Latency: None assumed
  • Elapsed time: At 14.4K baud, less than 1.5 seconds

This performance level is acceptable. If large result sets are returned, the performance would be unacceptable.
