Types of Integration

When integrating an AS/400 environment and a LAN-based PC environment, there are two possible architectures that may be used. One involves leaving the data on the AS/400, and accessing that data directly from the client PCs using the tools provided in Back Office. The other, described in detail in this paper, involves using the ShowCase Data Distributor and the components of Microsoft's BackOffice to migrate data from the AS/400 to a SQL Server running on a Microsoft Windows NT™ system.
