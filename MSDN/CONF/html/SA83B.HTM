Architecture

Figure 8: Time Analysis Architecture.

The architecture for the Time Tracking System is based on three steps:

  • Information is entered into a file, e.g., a meeting is entered into Microsoft Schedule+.

  • An E-Form, developed in Visual Basic and Microsoft Electronic Forms, reads the .CAL file for meeting and imports the information. Additional information is then captured, and the form is saved to a Microsoft Project file on the SQL Server.

  • The Briefing Book uses embedded Microsoft Excel graphs to display management information. The Briefing Book was written in Visual C++ and is an OLE 2.0 Container.
