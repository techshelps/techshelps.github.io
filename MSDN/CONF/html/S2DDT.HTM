Microsoft Managerment Console: Overview

Microsoft® Management Console (MMS) is an ISV-extensible, common console framework for management applications. MMS will be released as part of the next major release of Windows NT®. This paper introduces MMC and provides an overview of the MMC user interface (including sample screen shots) and MMC architecture. It also explains the concept of management Snap-Ins, how they relate to the console, and general interface guidelines for developers.

MMC is a core part of Microsoft's future management strategy. Microsoft development groups will use MMC for future management applications.
