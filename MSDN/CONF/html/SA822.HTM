Solution Developer

Below are requirements for any Solution Developer, ISV or IHV who vends applications and hardware that are compatible with and exploitive of the Microsoft Windows NT Server Platform and the BackOffice set of applications: Mail/Exchange, SQL, Systems Management Server, and SNA. There are two groups of ISVs that fall under this single initiative:

  • NT Server Compliance Program. The NT Server Compliance Program is for server hardware and peripheral vendors whose equipment works with NT and also for Software that has a server side element that works on the NT Server itself. In its initial incarnation for software it is analogous to the Novell "yes" program for measuring whether and application works against an NT server; but does not require an active process that runs on NT. Over time, the server application "points of light" will serve as a set of guidelines for ISV's who wish to retain their logo.

  • BackOffice Solution Program. A superset of the NT Server Compliance program, this is for software programs that exploit one or more of the members of the BackOffice family of products. It includes SQL applications, Network Management Applications that work with Systems Management Server, Host Emulator products that work with SNA Server, and Simple Messaging enabled applications.
