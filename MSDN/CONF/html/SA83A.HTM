Time Analysis

The Time Sheet has been saved, it can be analyzed. Each night, specifically designed Briefing Books are compiled and distributed to each developer and team leader. See the section on Briefing Books for additional information.

Figure 7: Time Analysis Briefing Book.

The Briefing Book presents the information in various levels of detail so that it can be analyzed. Pie charts display time by category and billing breakdown. Bar charts display time by date. In addition, drill-down capabilities display detail of categories.

The Briefing Book also summarizes tasks (compiled from Microsoft Project), defects, problem reports, and goals.
