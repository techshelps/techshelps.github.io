Description of Sessions:

Componentizing Data Access Through OLE DB: (No description available at this time)
Programming Your Web Server: In this session you will learn about the features and functionality of the Microsoft Proxy Server and how it can be used to build more efficient and secure solutions. You will also learn how the architecture can be extended and future developments.

Advanced IIS (ISAPI): This session will present a brief overview on the many differedetails on the Win32 Internet Server API (ISAPI) including security and multi-thread safety in your ISAPI applications and filters, and techniques to attain high performance and multi-processor scalable services. This talk will also cover the latest additions to the ISAPI interface, and look forward to some of the programmability interfaces in "K2"

Active Server Pages Programming 3: Creating and Using Scriptable Server-Side Components: In this session, we will cover the basic techniques that you can use to make your server or client application an automation controller. We'll do a complete code walk-through, with demos done in IE, Denali and with ScriptCMD.
