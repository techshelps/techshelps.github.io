Briefing Book Builder

The Briefing Book Builder is a Visual C++ application used to design the Books. The application allows the designer to place titles, boxes, graphics, and Microsoft Excel objects in the Book. Microsoft Excel objects contain a special module that automatically updates the information.

Figure 15: Briefing Book Builder.

The Briefing Book Builder creates Visual Basic .FRM and .MAK files. Once the files are developed, they can be opened in Visual Basic and additional functionality can be added. This allows Briefing Books to be developed by novice users and still maintain the flexibility for more advanced users.

Once the Briefing Book is complete, it is left uncompiled in a .MAK subdirectory. This allows the book to be updated with fresh data each night.
