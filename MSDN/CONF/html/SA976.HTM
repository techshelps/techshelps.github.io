Excel Pivot Wizard

The Excel Pivot Wizard guides users in creating a Microsoft Excel 5.0 PivotTable with data from a FoxPro table. The Excel Pivot Wizard outputs the table to an Excel spreadsheet, launches Excel, opens the spreadsheet, and creates a pivot table. Users have the option of embedding the pivot table in a FoxPro form or staying in Microsoft Excel to perform more analysis when the Wizard is done.
