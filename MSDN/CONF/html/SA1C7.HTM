Using CompuServe Mail

Windows 95 provides a wizard to install CompuServe Mail Service, which allows you to customize how the CompuServe MAPI driver makes a CompuServe connection and to choose the times your CompuServe MAPI driver will connect to CompuServe for the exchange of mail.

To install CompuServe Mail Service

  1. In the Add Services to Profile dialog box, click CompuServe Mail Service.
  2. In the CompuServe Mail Transport dialog box, click the CompuServe Dir button to define the directory where CompuServe connection information (CIS.INI) will be located, and then click OK.
  3. Click the Communications button and in the Setup Session Settings dialog box, enter the CompuServe connection information such as User ID number, password, telephone number.
  4. Click the Connection button and in the Connection Times dialog box enter the time you want Windows 95 to connect to CompuServe Mail. If you click "Exit of Mail," CompuServe mail will only be sent when you exit Microsoft Exchange.
