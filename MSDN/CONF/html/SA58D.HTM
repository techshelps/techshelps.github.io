Taking advantage of Directory Services

With the upcoming release of Microsoft's enhanced NT Directory Services, Dfs will be able to take advantage the following additional functionality:

  • Directory Services for storing existing administrative knowledge.
  • LDAP as the protocol for updating and querying for Dfs knowledge.
  • Directory Services store for providing root level fault tolerance in Dfs.
  • Directory Services for keeping all participating machines in any Dfs root synchronized in their perception of the Dfs structure.
  • Directory Services site topology for providing intelligent replica selection.
  • Content Replication Service for keeping alternate volumes synchronized with one another.
