Session Goal

The goal of this session is to provide Microsoft® Access developers with an understanding of the issues involved in developing efficient, optimized Microsoft Access version 2.0 applications for the client/server environment.

What You Will Learn

How Microsoft Access architecture operates and how it implements client/server computing.

How to optimize Microsoft Access to take advantage of its client/server capabilities.

How to use transaction processing system design issues as references for structural ideas and actual code segments optimizing client/server applications.
