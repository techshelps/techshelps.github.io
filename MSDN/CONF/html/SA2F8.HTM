Introduction

This session is designed to introduce developers to the use of messaging application programming interface (MAPI) in distributing data from databases. The session examines a more general problem than simply the use of MAPI to present a rich solution set to the developer. Topics covered in this session include:

  • Supplying Data Anywhere
  • Solutions
  • Data Access Models
  • RAS Behavior
  • Agents
  • Architecture of Agents
  • Result Sets
  • Implementing the Solution

For the code samples used in this session, see the basic project files, HOST.MAK and CLIENT.MAK, which demonstrate the agent concepts. The application EMPAPP.MAK demonstrates a simple application.
