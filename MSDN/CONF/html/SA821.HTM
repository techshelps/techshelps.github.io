The System Integrator or Analyst has the opportunity to build additional functionality with each BackOffice family product. Below are some examples.

  • ODBC tools

  • SQL PTK

    Open Data Services (ODS)

    DBLIB

    VBSQL, SQL OLE

  • Transact SQL

  • MAPI Send Messages

  • Embedded Transact-SQL for C to build applications that leverage the server infrastructure

Microsoft Mail and Exchange offers the following development tool sets:

  • MAPI/CMC let users forward documents from other BackOffice applications to various recipient platforms and devices

  • Eforms offers custom Message Types and Forms

  • Workgroup Templates offers access from other tools

  • MAPI Service Providers to access and offer messages and public folders from various external messaging platforms

  • GDK for building gateways to other messaging platforms

  • MAPI VBX

  • OLE/Messaging

  • OLE Schedule+

  • MAPI address book is extensible for user information and routing

  • Integration with SQL using Transact SQL

  • Walldata Rumba for Visual Basic
