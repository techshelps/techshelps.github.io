Wizards and Wizard Chaining

Like property pages, wizards work much the same in the console as they do in the shell. Extension Snap-Ins can also extend existing wizards. This is particularly beneficial in the case of wizards that create new objects. This allows extension Snap-Ins to chain onto the end of those wizards and create any additional objects that they require. As with property pages, wizards should always use the standard large size so as to allow extensions to chain onto the end of the wizard without having to account for size differences.
