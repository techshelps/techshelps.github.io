Alternate Sessions

Comparison of Windows NT and UNIX OS Architectures

In this session we'll present architecture diagrams, and we'll discuss user/kernel modes, the Executive, subsystem interaction, subsystems (POSIX, Win16, Win32...), and security (tokens, logon, subsystem ...). We'll compare Windows NT fibers, threads, and processes with UNIX solutions, and we'll contrast Services with Daemons.

Overview of Windows NT for UNIX Developers

In this session we'll describe the Windows NT memory model, file handling, Registry issues, exception handling, security issues, and priorities—all as they compare to their UNIX counterparts.
