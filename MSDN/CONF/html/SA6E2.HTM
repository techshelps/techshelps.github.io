Expression Builder

With the addition of the "DebugLibraries" option mentioned above, the Expression Builder can be used to refer to functions stored in library modules. To bring up the Expression Builder in the Code Module Window, select Build from the Shortcut menu.

Here the Expression Builder lists the name and arguments of the wlib_FillFromList function that ships in the wlib_Listbox module of WZLIB.MDA, the Wizard library function database.
