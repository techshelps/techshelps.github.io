Recommended Sessions

DCOM and Security

This session discusses how to use and work with COM's security infrastructure. We'll drill down into the various configuration options for COM security and provide detailed discussion of the various options for activation security and call-level security. In-depth coverage of the relevant APIs and interfaces is provided. Attendees should also attend session WCS-9, "Using Integrated Security Features of Windows NT."

Windows NT Directory Service Architecture (DS 1)

This session provides a technical discussion of the Windows NT 5.0 Directory Services features, including replication, schema, and naming. And we'll introduce Active Directory—the API set that will allow your application to access directory services from any vendor.

Windows NT Networking Structure and APIs from a UNIX Perspective

Come hear about Windows NT domain and workgroup models, protocols, APIs (such as RPC, sockets, NetBIOS, mail slots, named pipes, and more), and network security.
