Ad-hoc queries need lots of processing power

The solution to the problem of user access to the data created a new problem. By their very nature, the ad-hoc queries that users needed to run for decision support required a great deal of processing power. When running these queries directly against the databases on the AS/400, the response-time impact for on-line users was dramatic.

The only solutions to this problem are to upgrade the AS/400 to add more processing power or distribute the data to another platform for the decision support queries.
