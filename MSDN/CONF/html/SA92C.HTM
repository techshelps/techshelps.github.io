SQL Server MAPI Issues

Some aspects of using xp_sendmail that should be considered follow. The size of the result set may be above the message limits on the communications system. There may be security problems and confidentiality issues because of the possibility of e-mailing data by accident or intent to inappropriate persons.

The biggest problem of sending results is the inability to send some data types. Some data types cannot be sent as text—for example, binary large objects (BLOBS) and graphics.
