Functionality Requirements

The application was designed to:

  • Present formatted text with hyperlinks.

  • Present a full table of contents for all material in the content set.

  • Present bitmapped images, AVI video, and sound.

  • Allow the user to create and save lists of desired content items.

  • Allow a full text search through the content.

A full discussion of the techniques we used to create the front end application is presented in MM203.
