Class List

The Class List uses the Outline.OCX OLE control to display class in one or more VCX class library.

Select a class by clicking the class name or moving the cursor keys up or down.

Edit the selected class using the Class Designer by double clicking the class name.

Display the parent class of a specific class by double clicking the folder picture next to the class name. If the parent class is located in another directory, another instance of the Class Browser will be launched and automatically select the parent class in the new Class Browser form.
