The object is to demo TAP and show it doing something useful like FM radio transmission. Beware that we need to do the demo in 640 x 480 mode, so don't get too adventurous.

The point is that for engineers and scientists working on UNIX® in C, Microsoft® Windows NT™ operating system is easily as good a platform and, potentially, better.
