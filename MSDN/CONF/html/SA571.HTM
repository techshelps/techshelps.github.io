Unicode for Multi-lingual Indexing

Most Web pages today are authored in English, but many documents on an intranet are not. Since IIS can serve documents, multi-lingual indexing and querying features are a standard feature of Index Server. The query system was designed with localization in mind. It is completely modular and can dynamically load and unload language specific utilities.

The system can index multi-lingual documents and switch between languages as required, such as index an English paragraph, index a French paragraph, and then switch back to English. All index information is stored as Unicode characters and all queries are converted to Unicode before processing.
