Windows NT Server Today

Windows NT Server provides all the components necessary to support mission-critical applications. The system is built on a fully 32-bit microkernel foundation. It is multithreaded, offers preemptive multitasking, and provides memory protection for both applications and the operating system itself. It scales to run on hardware with up to 32 processors, 4 GB of RAM, and 17 million terabytes of disk space. In addition, Windows NT Server supports IntelX86, MIPS R4x00, DEC Alpha AXP, and IBM PowerPC processors.
