Description of Sessions:

New Security With Windows NT: This session will discuss the Kerberos authentication protocol support in Windows NT 5.0, Public key security components for Windows NT and Migration to new Security Technologies

Win32 Inet APIs: WinInet is an extension of the Win32 API that provides Win32 applications with easy access to common Internet protocols, including Gopher, FTP, and HTTP. By providing task-focused interfaces for Internet access, WinInet simplifies Internet access for Win32 developers. This session will explain how to use the WinInet functions to add Internet access to your Win32 application
