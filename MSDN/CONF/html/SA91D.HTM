Solution Model

Our solution model consists of two components or agents. One is at the remote PC and one is at the host PC. The agents communicate with each other. On the host or server database side, we may have one of the following agents:

  • A RAS agent
  • A MAPI agent

On the remote or client side, we may have one of the following agents:

  • Manual agent—for example, manual e-mail
  • Application-based RAS or MAPI agent
  • Microsoft Mail MAPI agent
