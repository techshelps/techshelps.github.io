Windows NT 4.0 RAS Enhancements

In addition to PPTP, other enhancements have been made to RAS that are included in Windows NT 4.0. The following is a brief summary:

  • PPP multilink Provides the ability to aggregate two or more physical connections to form one larger logical connection, improving bandwidth and throughput for remote connections. Software tools have also been added to facilitate ease of use.
  • Auto dial Makes it easier and faster for remote users to connect to their corporate networks.
  • Logon dial If activated, automatically connects the remote user to the corporate Intranet when the computer is turned on.
  • Restartable File Copy (RFC) Addresses the problem of having to dial in and start over if an interruption occurs while downloading a huge file off a server to a remote computer. RFC allows the transfer to continue where it left off, once the connection has been re-established.
  • RAS API enhancements Gives third-party companies more options to create their own custom solutions, applications, and utilities based on RAS.
