Microsoft Open Directory Services Interface: Active Directory

Open Interfaces for Managing and Using Directory Services

A White Paper from the Desktop and Business Systems Division

Abstract

The following paper details the vision of Microsoft for integrating multiple directory services through a well defined, open set of interfaces: Active Directory. The availability of a standard, open directory service administration and programming model for Windows platforms will encourage the inclusion of directory services in a wide range of commercial and customer-developed application

The Microsoft Business Systems Division series of white papers are designed to educate information technology (IT) professionals about Windows NT and the Microsoft BackOffice family of products. While current technologies used in Microsoft products are often covered, the real purpose of these papers is to give readers an idea of how major technologies are evolving, how Microsoft is using those technologies, and how this information affects technology planners.
