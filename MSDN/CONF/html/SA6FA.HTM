Property Builders

Certain form and control properties have builders available. For example, the Expression Builder can help set expressions for calculated fields or validation rules. You use a builder by clicking the dot-dot-dot button next to the property.

Another builder, the Query Builder, is especially useful for creating a Structured Query Language (SQL) statement for a record or row source setting. In version 1.x you had to type SQL into the property sheet (or use a saved query); you can now use the query by example (QBE) grid to create the queries your form needs.
