Data Access Criteria

The changes in communications technology have resulted in higher expectations from remote offices, telecommuters, and road warriors. They expect to request and receive data through whatever communications links are available. Today's communications link can include services such as MCI MAIL®, Internet, CompuServe®, America Online®, Windows NT Remote Access Service (RAS), and Microsoft® Mail.

The user expects that data transfer will be optimal and timely. Optimal means that the connection time is minimal and the data volume minimal. Timely for e-mail users means the response is in the same time frame that an e-mail response would be. Timely for modem connections means that the response is in minutes plus data-transfer time.
