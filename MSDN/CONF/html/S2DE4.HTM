Menus and Toolbars

The permanent MMC menus and the main application toolbar may not be extended by Snap-Ins, except in the case of OLE document objects. It is intended that these menus and the main toolbar be populated only with actions that affect the entire console, rather than actions which pertain to a particular Snap-In. A particular Snap-In may, however, create its own toolbar which can be docked anywhere the user wishes, including on the top of the console window, next to the main application toolbar. These Snap-In created toolbars may also contain "command bar" style drop-down menus which can expose actions appropriate to that Snap-In without relying solely on the context menus.
