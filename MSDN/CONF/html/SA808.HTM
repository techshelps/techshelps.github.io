Is There More?

This coverage of the integration and migration of data from an AS/400 to the SQL Server is just a beginning. The real value of such an integration is that your investment in dollars on the AS/400 is preserved and enhanced by migrating the data to a SQL Server. By being able to better utilize that data, it has more value to your company.

The limit to what can be done by migrating data has not been defined. You are only limited by your imagination.
