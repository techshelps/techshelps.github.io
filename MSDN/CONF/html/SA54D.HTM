HTTP URL Names

The next generation Windows NT Directory Services support access from Web browsers via the HTTP protocol and Microsoft Internet Information Server, the full-service Web server built-in to Windows NT Server. HTTP URL's are familiar to most users that have Web browsers, and are in the form HTTP://somedomain/
path-to-page. The next generation Windows NT Directory Services support access to its contents via HTTP URLs in which "somedomain" refers to a server running next generation Directory Services and "path-to-page" is the path through next generation Directory Services hierarchy to the object of interest, for example: 


HTTP://SomeServer.Myco.Com/BIN/Division/Product/Ssys/JamesSmith
