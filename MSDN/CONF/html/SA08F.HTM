Upsizing Microsoft Access–based Applications

Presented by: Christopher Payne

Christopher Payne is a Microsoft Product Manager for the Microsoft Access database management system. Chris has been with Microsoft for 4-1/2 years and has been focusing on research and on planning future versions of Microsoft Access for the past 2-1/2. In addition to working on Microsoft Access, Chris was the project lead of the Microsoft Access Upsizing Tools released in October of 1994.
