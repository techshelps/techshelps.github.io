Introduction

This session is intended to demonstrate how Microsoft® Back Office products can be used to migrate data from and integrate with an AS/400® environment. Products used to achieve this integration and migration include Microsoft Windows NT™ Server network operating system, SNA Server connectivity for IBM® enterprise networks, Microsoft SQL Server™ client-server database management system, System Management Services centralized management for distributed systems, and the ShowCase Data Distributor. Topics covered in this session include:

  • Data Migration Background

  • ODS Overview

  • General types of integration

  • Back Office Components Used

  • Data Migration Scenarios

  • Transfer Examples

  • Replication Examples

For a description of all of the files used in this session, see the file BF206.LST. Almost all of these files are used from the Windows NT Server or a Microsoft Windows® client, but AS/400 commands are also included for reference.
