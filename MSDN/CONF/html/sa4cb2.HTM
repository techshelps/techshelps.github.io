ICM 2.0 Features

After meetings with multiple industry leaders in the color field, Microsoft has designed ICM 2.0. The new API is a superset of the ICM 1.0 API that adds a new range of capabilities:

  • ICM 1.0 compatible
  • ICC compliant
  • Scalable: Simple functions for applications such as Microsoft Office, complete control for applications such as Adobe® PhotoShop®
  • Same APIs for Windows 95 and Windows NT® operating systems
  • Support for profile management at API and UI level
  • Bitmap v5 header support
  • Standard color space support: sRGB
  • Broader color space support: RGB, CMYK, LAB, and others
  • Broader support for bitmap formats
  • Improved palette handling
  • Device driver participation on the Windows 95 and Windows NT® operating systems
  • Support for multiple Color Management Modules (CMM)
  • Faster default CMM that supports all ICC-compliant profiles
  • Easier installation of profiles

This new version of ICM will be integrated in future releases of Windows 95 and Windows NT.
