Introduction

The Microsoft Windows NT® Security Model and the Microsoft® Internet Security Framework provide the infrastructure for building and deploying secure applications for the Internet and intranet with the same set of APIs. This theme paper will guide you through the sessions that will teach you how to secure your applications using the Windows NT Security model and the Microsoft Internet Security Framework.

If you are a developer building Internet applications, a content developer, or a client/server application developer, you can use the Microsoft Internet Security Framework and Windows NT Security to build secure applications.
