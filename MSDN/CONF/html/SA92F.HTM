Query Contents

The query that is sent to the host agent must contain sufficient information. The amount of information forwarded determines how far the agency extends. Some possible elements of the query request are:

  • Query SQL code
  • Database name
  • Login parameters
  • ODBC declarations for database
  • MDB file, which contains records that should be processed

Windows SQL integrated security is done based on the NT login. Since the host agent should not have to log off and log on for each query, integrated security should not be used.
