Replication engine to SQL Server

To overcome the problem of not enough capacity on the AS/400 to perform decision support queries, the data must be replicated to another DBMS such as the Windows NT based SQL Server. The ShowCase Data Distributor, built with ODS, is such a replication engine.

The Data Distributor is designed to provide high-speed transfers of data between the AS/400 and SQL Server, moving data in either direction under SQL control. To better support the decision support needs, the data can be separated, combined, transformed, or kept intact during the transfer.
