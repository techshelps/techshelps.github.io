Architecture

Figure 11: Testing Architecture.

The Testing Application is composed of the following architecture:

  • Test Script - The Test Script is developed using a custom Visual Basic application. ODBC is used to log the script to a SQL Server database.

  • Test Plan - The Test Plan is also a custom Visual Basic application. The program is set to always be on top so that it will float above the application being tested.

  • Test Results - The Test Results are a custom E-Form, developed using Visual Basic and Microsoft Electronic Forms Designer.
