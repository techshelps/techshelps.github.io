Query Forms

An example of a query form is presented in Figure 1.

Figure 1 - Sample Query Form

In this example, the user is searching the entire Web site for all documents that contain the phrase "NT Server" near the phrase "Performance Benchmarks."

To execute this query, the user clicks the "Execute Query" button.

Index Server allows a Webmaster or administrator to create customized search forms to help employees and other clients find documents at the local site. The forms can be modified to allow users to search by contents, or by other document properties such as author or subject. The query form is created using standard HTML and is just a Web page itself. Any user who knows how to create Web pages can put together a simple query form.
