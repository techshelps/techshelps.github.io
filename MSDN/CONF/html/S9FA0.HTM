Recommended Sessions

New Security with Windows NT

This session describes new security features integrated with the Windows NT Directory Service. We'll explain how Kerberos authentication is integrated with Windows NT and directory support to provide strong authentication using public key certificates, and we'll provide an overview of security configuration management and a summary of features to be included in the next release of Windows NT.

Microsoft Internet Security Framework

This session provides an overview of technologies that allow you to build a secure Web site. We'll discuss Authenticode, SSL, and certificate technologies.
