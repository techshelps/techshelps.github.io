RFC822 Names

RFC822 names are in the form somename@somedomain and are familiar to most users as Internet e-mail addresses, for example JamesSmith@myco.Com. The next generation Windows NT Directory Services provide a "friendly name" in RFC822 form for all objects. For example, a user can use a "friendly name" as an e-mail address, suitable for display on a business card, and as the name used to log on.
