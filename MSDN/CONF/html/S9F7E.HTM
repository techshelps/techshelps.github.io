Programming a Distributed Message Queue Application

Do you want to see how simple it is to develop a distributed message queue application? In this session, you will see two MSMQ applications, one developed in C++ and the other developed in Visual Basic using MSMQ ActiveX components.

Both applications will be run together, showing how they can work together even though they were written in two different languages. And if disconnects are an integral part of your network, you will see that either machine can be shut down without loss of information.
