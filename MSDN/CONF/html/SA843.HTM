Architecture

Figure 14: Problem/Resolution Architecture.

The Problem/Resolution System is based on the following architecture:

  • All E-Forms were developed using Visual Basic and the Microsoft Electronic Forms Designer. The Problem Report form is based on an E-Form included in Microsoft Workgroup Templates.

  • The Message Manager is a Microsoft Access database application, which uses Messaging Application Programming Interface (MAPI) to read messages and store information in an attached SQL Server database.
