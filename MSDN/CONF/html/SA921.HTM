Microsoft Access Model

In the Microsoft Access model, the database engine is on a remote PC. The engine obtains data from permanent tables over an Ethernet or a modem connection. Temporary data is saved in the TEMP directory on the PC. (The TEMP directory may be on a local drive or a RAM drive.) This situation means that, if a query must check each record, every record will be transferred over the connection. This volume of data on a slow link can take hours.
