Abstract: Web-enabling Your Line-of-business Application

Tailoring your application for the Web can be as simple using a browser to view existing documents, or launching the client program of a client-server application from inside a browser instead of from the user's desktop. Or you can employ all the capabilities that Microsoft offers through the active WindowsÒ platform and Internet technologies. This paper provides an overview of the technologies and products you use to add Internet capabilities to your existing business applications. Briefly, the implementation options are:

  • Internet publishing
  • Server-browser applications
  • Custom client applications
