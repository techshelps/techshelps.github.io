The following articles were distributed at the 1997 Professional Developers Conference (PDC). They are now part of the new Microsoft SQL Server Developer's Resource Kit, which you can find on the Library at MSDN Library\Windows Resource Kits\Microsoft SQL Server Developer's Resource Kit.

The SQL Server Developer’s Resource Kit is a compilation of technical materials targeted at developers of large-scale Microsoft SQL Server applications. The goal of the Resource Kit is to provide developers with the key technical information needed to design and to develop great applications on SQL Server.
