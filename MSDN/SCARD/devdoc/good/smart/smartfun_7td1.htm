SCardForgetCardType

The SCardForgetCardType function removes an introduced smart card from the smart card subsystem.

LONG SCardForgetCardType(
  IN SCARDCONTEXT hContext,  
  IN LPCTSTR szCardName  
);

Parameters

hContext
Supplies the handle that identifies the resource manager context. The resource manager context is set by a previous call to SCardEstablishContext.
szCardName
Supplies the friendly name of the card to be removed from the smart card database.

Return Values

If the function… The return value is…
Succeeds SCARD_S_SUCCESS.
Fails An error code (see Error Codes for a list of all error codes).

Remarks

SCardForgetCardType is a database management function. For a description of other database management functions, see Smart Card Database Management Functions.
