ISCard::LockSCard

The LockSCard method claims exclusive access to the smart card.

HRESULT LockSCard(void);

Return Values

The possible return values are the following:

Value Meaning
S_OK Operation completed successfully.

Remarks

In addition to the COM error code listed above, this interface may return a smart card error code if a smart card function was called to complete the request. For information on smart card error codes, see Error Codes.
