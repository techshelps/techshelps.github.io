Wizard Manager Shortcut Menu

Move All Off Screen

When selected, the Wizard Manager moves the currently visible step out of the visible screen frame. This is usually the last task you perform after you've completed the creation of a Wizard. Otherwise, the last step you were working on remains visible in the frame.

For example, if the Finished! step is visible when you save the Wizard, and you don't click the Move Step button, the Finished! step will show when the Wizard is started.

Toolbar button:

Add

When selected, the Wizard Manager adds a new step to the end of the list before the Finished! step.

Toolbar button:

Insert

When selected, the Wizard Manager creates a new step before the currently selected step. If no step is currently selected, the button does nothing.

Toolbar button:

Delete

Deletes the currently selected step from the Wizard Manager. You can also use the delete key.

Toolbar button: None

Move Up

When selected, the Wizard Manager moves the selected step up one position from its current position in the list. If no step is currently selected, the button does nothing.

Toolbar button:

Move Down

When selected, the Wizard Manager moves the selected step down one position from its current position in the list. If no step is currently selected, the button does nothing.

Toolbar button:

Edit Name

When selected, allows you to edit the name of the currently selected step.

Toolbar button: None

Refresh

When selected, the list of steps is refreshed to reflect any changes made to it.

Toolbar button:

New Wizard

When selected, a new Wizard is created from the Wizard template.

Toolbar button: None

Help

When selected, invokes Help on this topic.

Toolbar button: None

Show Toolbar

(Toggle) When checked, the Wizard Manager toolbar is displayed. When unchecked, the toolbar is hidden.

Toolbar button: None
