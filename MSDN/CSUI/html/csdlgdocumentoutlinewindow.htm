Document Outline Window

   

A window with a flat or hierarchical outline for the currently selected document window. The name of the window changes to reflect the editor that is open. For example, in a script editor, the Script Outline window displays the scripting object model and the script blocks for the associated HTML page.

Use the Document Outline window to view the contents of other editor windows —along with the Document Outline window associated to them — that do not currently have focus.

If you open another document window that has an outline view associated with it, the contents of the Document Outline window changes to that of the new window.

You can dock the Document Outline window. The Document Outline window displays when you select Document Outline from the Windows submenu.

For information about the specific Document Outline windows, click See Also.
