Open With Dialog Box

   

Allows you to edit documents with any external or internal editor you choose.

Options

Available Editors List

Lists all environment-intrinsic editors and any other editors that you have added using the Add button.

This list is saved per file type and user only.

Add

Displays the Add Viewer dialog box where you can search for and add an editor to the Available Editors list.

Remove

Deletes the selected editor name from the list. You cannot delete any of the environment-intrinsic editors from the list.

Set as Default

Sets the selected editor as your default editor.
