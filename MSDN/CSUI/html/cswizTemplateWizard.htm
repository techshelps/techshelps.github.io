Template Wizard

Use this wizard to add custom files to a project based on template files you create. Templates can be a single .htm, .asp, code, or other text file or templates can contain multiple files. The Template wizard prompts users for information based on parameters you specify in the template files. When you complete the Template wizard, files based on the custom template files are added to the current project.

Template files can contain text, source code, URLs, and other information. Relative URLs between template files are automatically fixed when the files are added to the project.

For more information, see Creating Custom Templates.
