Property Pages Dialog Box

   

Allows you to change a control's properties at design time.

Options

Tabs

Creates a tabbed dialog box (property page) that allows users to change property values for a control at design time. The property page acts like a form by letting you write code that updates the property values when the user changes them.

OK

Validates the Property Pages settings and closes the Property Pages dialog box.

Apply

Validates the Property Page without closing the dialog box.
