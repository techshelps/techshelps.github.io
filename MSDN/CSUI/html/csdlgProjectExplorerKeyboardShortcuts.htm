Project Explorer Keyboard Shortcuts

   

Use the following key combinations in the Project Explorer.

Note   You can change or create keyboard shortcuts in Keyboard of the Environment options in the Options dialog box.

To Press
Display documentation on the Project Explorer F1
Close the Project Explorer window. SHIFT+ESC
Display the shortcut menu. SHIFT+F10
Expand a list and then selects a subentry in the list each time you press it. RIGHT ARROW
Move down the list from project to project. TAB
Move down the list one entry at a time. DOWN ARROW
Move the insertion point to the beginning of the next line. SHIFT+ENTER
Move up the list one entry at a time. UP ARROW
Move up the list from project to project. SHIFT+TAB
Open the designer. SHIFT+F7
Open the selected file from the list, or expand and collapse the list to show its subentries. ENTER
Open the editor for the selected file. F7
Select a subentry in the list and then move up the list each time you press the LEFT ARROW until the subentry list collapses into a folder. LEFT ARROW
Select the first file in the list. HOME
Select the last file in the list. END
