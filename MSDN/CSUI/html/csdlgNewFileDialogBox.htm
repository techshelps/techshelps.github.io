New Tab (New File and Open File Dialog Boxes)

   

Displays files from all project types that you can create or open without having a project. The files can contain snippets of code, notes, or other information you may want to use in your project. The development environment remembers the last location and file type you used and displays it when you display the tab.

Note   Your file is not saved unless you choose Save from the File menu.

Options

Folders

Lists the available folders. The contents of this list vary with the application you are using. Although a General folder is always displayed, it may not be used by the current application.

Contents

Lists the content of the selected folder.
