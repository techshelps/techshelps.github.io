New Tab (Add Item Dialog Box)

   

Allows you to add new project items to the current project only.

Options

Folder List

Lists the items from the currently selected project in the Project Explorer.

Items List

Lists all of the items in the selected folder.

Name

Displays a default name for the new item you want to add. You can rename the item by typing a name in the box.

Add Folder

Available when the Existing tab is active.
