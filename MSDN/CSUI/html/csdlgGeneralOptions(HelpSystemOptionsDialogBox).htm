Help System Options (Environment — Options Dialog Box)

   

Allows you to:

  • Choose the language in which your documentation is displayed.

  • Choose which documentation you want to view. The documentation consists of groups of multiple files called collections. For example, the Microsoft Developers Network (MSDN) collection consists of documentation files for multiple applications, which you may have installed on your computer, plus additional files. If you do not specify one, the latest collection (based on its timestamp) is chosen by default.

  • Choose both the preferred language and the preferred documentation.

  • Change the preferred documentation for all of the languages you have installed on your machine.

Options

Preferred Language

Displays a list of the languages for documentation that you have installed on your machine. For example, if you installed an English version of MSDN and then also installed a German version, the Preferred Language list would contain both English and German. You can choose the language you prefer to use.

Preferred

Displays a list of the collections installed on your machine from which you can choose the one you prefer to use. For example, suppose you have a version of the Microsoft Developers Network (MSDN) documentation that contains a topic called "Inside OLE" and have recently installed an new version of MSDN that does not include this topic. Because you have both versions on your machine, you can set the original MSDN as your preferred collection because you want easy access to the "Inside OLE" topic. The <Default> collection is the newest collection.
