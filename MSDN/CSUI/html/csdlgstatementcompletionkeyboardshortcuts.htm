Statement Completion Keyboard Shortcuts

   

Use the following key combinations in the Statement Completion list.

To Press
Move the cursor up one list item at a time and select it. UP ARROW
Move the cursor down one list item at a time and select it. DOWN ARROW
Move the cursor up one page in the Statement Completion list. PAGE UP
Move the cursor up one page in the Statement Completion list. PAGE DOWN
Move to the beginning of the Statement Completion list. CTRL+PAGE UP
Move to the end of the Statement Completion list. CTRL+PAGE DOWN
Hide the Statement Completion list, move the focus to the text editor, and move the cursor up one line at a time in the Text Editor window. SHIFT+UP ARROW
Hide the Statement Completion list, move the focus to the text editor, and move the cursor down one line at a time in the Text Editor window. SHIFT+DOWN ARROW
Hide the Statement Completion list, move the focus to the text editor, and move the cursor up one page at a time in Text Editor window. SHIFT+PAGE UP
Hide the Statement Completion list, move the focus to the text editor, and move the cursor down one page at a time in the Text Editor window. SHIFT+PAGE DOWN
Hide the Statement Completion list, move the focus to the text editor, and move to the top of the Text Editor window. SHIFT+CTRL+PAGE UP
Hide the Statement Completion list, move the focus to the text editor, and move to the bottom of the Text Editor window. SHIFT+CTRL+PAGE DOWN
