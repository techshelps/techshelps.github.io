Task List (Environment—Options Dialog Box)

   

Allows you to set the options for adding and deleting tasks and for adding, deleting, and changing comment tokens, except TODO, in the Task List window. For TODO comments, you can change the priority level but you cannot rename or delete them.

CAUTION   Clicking Cancel cancels all of the changes made in the dialog box. If you made changes to more than one category, none of the changes will be saved.

Options

General

Confirm Deletion of Tasks

Displays a message box allowing you to confirm that you wanted to delete a task. A task must be editable to be deleted, but not all editable tasks can be deleted.

Warn When Adding a Task That Won't Be Shown

When checked, provides a message box to warn you when a task you are adding will not be shown in the Task List. For example, the message would appear if you chose to show only Smart Editor tasks (using the options on this dialog box) and later enter a User task.

Comment Tokens

Token List

Displays a list of comment tokens that includes UNDONE and HACK, and allows you to add comment tokens to display in the Task List. Comment tokens are indicators in comments that tell the Task List to create a new task item when the token is typed in the editor.

Priority and Name

Displays the priority and name of the token that is currently selected in the Token List.

Add

Click to add a new token with the Name you typed and Priority you selected in the Priority and Name list.

Change

Click to make changes to the Priority you selected or to the Name you typed into the Priority and Name list.

Delete

Click to delete the selected item in the Token list. You cannot delete TODO.
