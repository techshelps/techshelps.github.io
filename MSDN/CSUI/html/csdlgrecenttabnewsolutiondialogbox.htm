Recent Tab (New Project and Open Project Dialog Boxes)

   

Displays a list and the locations of recently opened solutions and projects.

Options

File

Displays the list of recently opened projects.

Folder

Displays the path to the folder that contains the file.

Close Current Solution

Closes the current solution, creates a new solution with the name you typed in the Name box, and adds the project or file to the new solution.

Add to Current Solution

Adds the new project or file to the current solution.

Import Project Folder

Not available for this tab.
