International Settings Options (Environment – Options Dialog Box)

   

Allows you to change the language and fonts of the dialogs and tool windows in your development environment when you have more than one language version of the development environment installed on your machine. The changes become effective when you restart your application and remain until you change them.

Note   Application-specific options do not appear in the Options dialog box until the application is loaded into memory. Therefore, you may not see the same options you saw during your last session. Once you create a project or use a command that uses a particular application, the options appear in the Options dialog box and remain available as long as the application remains in memory. Common options are available all of the time.

CAUTION   Clicking Cancel cancels all of the changes made in the dialog box. If you made changes to more than one category, none of the changes will be saved.

Options

Language

Lists the available environment languages. Only the languages that you have installed on your machine appear in the list. For example, if you have both English and German  installed on the same machine, English and German both appear in the list.

Font

Lists the available fonts based on the language selected.

Note   Tool window title bars are not affected by changing the font size changes in this tab. You can change the font size of the tool window title bar by choosing Palette Title on the Appearance Tab of the Display Properties dialog box in the system Control Panel.

Size

Lists the available font sizes.

Sample

Displays a sample of the font and the language.

Reset All

Allows you to reset the Font and Font Size to the default Font and Font size for the currently selected language. For example, if Japanese is your current language and you selected a different and larger font, choosing Reset All changes the font type and size back to the defaults for Japanese.
