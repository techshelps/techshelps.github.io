Running Documents Window

   

Displays a list of documents that are currently loaded into the process you are running. For example, if you add an HTML frameset, the Running Documents window shows you which pages are currently loaded in the browser.

In the Running Documents window you can:

  • Double-click on a document to open an editor window for that document.

  • Expand and collapse the list to see the pages or documents within the top-level documents.

Elements

Documents List

Contains a hierarchical list of documents and their paths.
