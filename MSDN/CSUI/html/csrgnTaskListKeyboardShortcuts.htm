Task List Keyboard Shortcuts

   

Use the following key combinations in the Task List window.

Note   You can change or create keyboard shortcuts in Keyboard of the Environment options in the Options dialog box.

To Press
Cancel the edit. ESC
Close the Task List window. SHIFT+ESC
Copy the selected text in an editable field to the Clipboard. CTRL+C or CTRL+INS
Cut the selected text in an editable field and move it to the Clipboard. CTRL+X or SHIFT+DELETE
Delete the selected task item or selected text in an editable field without placing it on the Clipboard. DELETE
Display the shortcut menu. SHIFT+F10
Display documentation for the task item, if it is available. F1
Display documentation for the Task List window. CTRL+F1
Move to the next task column when in edit mode. TAB
Move to the previous task column when in edit mode. SHIFT+TAB
Move the cursor to the Description field and switch to edit mode. If you are in edit mode, pressing F2 accepts your edits and closes edit mode. F2
Move the cursor to the first displayed task item. Each time you press F12, you move to the next task item in the list. Only acts on tasks that are currently displayed. For example, if you are showing compiler errors, F12 cycles through compiler errors. F12 does not work with User-defined tasks. If your Task List shows a mixture of tasks, the ones not associated with a file are skipped. F12
Move the cursor to the previous task item in the Task List window. Provides F12 functionality in the reverse direction. SHIFT+F12
Move to the task in the associated text editor window.

In edit mode, choose Return to accept your edits and close edit mode.

 RETURN
Paste the Clipboard contents at the insertion point in an editable field. CTRL+V or SHIFT+INSERT
Place a checkmark in the Checked column to mark the task as completed. SPACE BAR on a selected, editable task item
Undo the last editing action in the editable field of the current task item. CTRL+Z or ALT+BACKSPACE
