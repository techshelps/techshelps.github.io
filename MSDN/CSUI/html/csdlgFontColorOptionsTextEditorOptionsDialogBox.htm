Font and Color Options (Text Editor — Options Dialog Box)

   

Allows you to pick the font for editor windows and the colors for various text elements in the editor window.

CAUTION   Clicking Cancel cancels all of the changes made in the dialog box. If you made changes to more than one category, none of the changes will be saved.

Options

Editor Font

Lists all the fonts you have installed for the editor. When the box first appears, the font already in use for the selected element is highlighted. A change to the font affects all elements of the editor. You cannot change the font for only some elements.

Size

Lists common point sizes for the highlighted font. Changing the size of the font affects all elements of the editor.

Display Items

Lists the items for which you can modify the foreground and background, as well as the bolding of syntactic elements.

Item Foreground

Lists the available colors you can choose for the foreground of the item you selected in the Display Items list. Changes to foreground color of Text changes the defaults for some elements such as Compiler Error, Keyword, or Operator.

Item Background

Lists the available colors you can choose for the background of the item you selected in the Display Items list. A change to the background color of Text also changes the defaults for some elements such as Compiler Error, Keyword, or Operator.

Bold

When selected, makes the selected text bold. Only available for syntactic elements such as Comments, Inactive Code, Keywords, and Numbers.

Sample

Displays a sample of the font you have selected. The sample shows the font, size, and colors you have specified. You can use this box to preview the results as you experiment with different formatting options.

Reset All

Returns the settings to the default settings for the language of the application you are using.
