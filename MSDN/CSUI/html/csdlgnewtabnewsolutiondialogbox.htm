New Tab (New Project, Add Project, and Open Project Dialog Boxes)

   

Allows you to select a new project based on the available project templates provided by each product.

Options

Template and Items Folders

Lists the project template and items folders. You can expand and collapse the list by clicking + or -.

Contents

Lists the content of the selected project or items folder.

Name

Allows you to select the default name or type a name for the new project.

Location

Displays the location of the default project directory with the project name added at the end.

Browse

Displays the Browse for Folder dialog box, where you can locate and open the project you want.

Close Current Solution

Closes the current solution, creates a new solution with the name you typed in the Name box, and adds the project or item to the new solution. Displayed only if a solution is open.

Add to Current Solution

Adds the new project or item to the current solution. Displayed only if a solution is open.

Open

Shows the solution and its projects and items in the Project Explorer.

Import Project Folder

Not available on this tab.

Show This Dialog in the Future (only appears on the New Project dialog box the first time you start your application)

When selected, automatically displays the New Project dialog the first time that you start your application. If you clear this box and want to view the New Project dialog box in the future, select the Show New Solution dialog from the General options of the Environment page of the Options dialog box.
