Define Window Layout Dialog Box

   

Allows you to save your current window settings, collectively called a layout, and reuse them whenever you like. You can add, delete, and rename layouts or you can select a layout and apply it.

Note   You cannot rename or delete the Default Debug, Default Design, or Default Full Screen layouts.

Options

View Name

Specify a name for the layout.

Views

Lists the available layouts.

Apply

Sets the specified layout to the interface without closing the dialog box.

Add

Saves the layout with the name you typed in the Layout Name box and adds the name to the Layouts list.

Delete

Deletes the selected layout in the Layouts list. Not available if Default Debug, Default Design, or Default Full Screen is selected in the Layouts list.

Rename

Renames the selected layout with the name you specified in the Layout Name box. Not available if Default Design, Default Debug, or Default Full Screen is selected in the Layouts list.
