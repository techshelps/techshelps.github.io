Existing Tab (New Project, Add Project, and Open Project Dialog Boxes)

   

Allows you to locate and select existing solutions or projects. When you add projects, they merge into one solution.

You can add as many projects as your system resources and memory can accommodate.

Options

Close Current Solution

Closes the current solution, creates a new solution with the name you typed in the Name box, and adds the project or item to the new solution.

Add to Current Solution

Adds the new project or file to the current solution.

Import Project Folder

Imports any folder or file on the disk and turns the contents into a project.
