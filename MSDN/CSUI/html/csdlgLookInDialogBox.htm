Look In Folders Dialog Box

   

Allows you to choose the files you want to include in your search. The options you choose are not affected by the New Search command in the Find and Replace dialog box.

Options

Available Folders

Displays a list of available folders, drives, and servers in which you can search. Double-click on any of the following to view a list the folders contained within.

Icon Contains a List of Available Folders
Your internal hard drive.
Another floppy drive.
A hard drive.
A CD-ROM drive.
A server to which you are connected.

Add to List

Adds the selected folders, drive, or server to the search criteria. Each item is searched in the order in which it appears in the list.

Remove from List

Deletes the selected folder, drive, or server from the search criteria.

List of folders to look in

Displays a list including the full path of the files you added and through which you want to search.

Look in files of type

Lists the type of files that each application allows to be searched.
