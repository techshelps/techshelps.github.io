Select Packages/Libraries Dialog Box

   

Allows you to select which packages and libraries you want displayed in the Object Browser.

Options

Current Settings

Displays a hierarchical tree showing the libraries and packages for browsing. The hierarchical tree contains topmost nodes, with libraries in nodes below it.

Note   The default display changes according to the development environment that you are using.

Libraries and packages that are selected for browsing appear in bold text. When you check the node of a library that you wish to browse, all of the nodes from the parent to root nodes are checked. When you clear a parent node, none the libraries and packages under it will be displayed in the browser. However, if there was a partial selection of libraries and packages under that node, the selection remains checked even though it will not be displayed in the browser.

Add

Displays the Add New Packages/Libraries dialog box, where you can add new packages or libraries.
