Other Java Packages Tab (Add New Packages/Libraries Dialog Box)

   

Allows you to add and remove packages that are not already registered. The packages you add persist only per machine. This is useful if you want to view packages that have not been registered on your system.

Options

Specified Package List

Displays a list of available packages, including the path to the file.

Browse

Displays the Browse dialog box where you can locate the nonregistered package files you want to display.

Remove

Removes the selected package from the list. You can use this button to remove a package from the list after the package has been registered in the system registry.
