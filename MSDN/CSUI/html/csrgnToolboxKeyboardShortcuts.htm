Toolbox Keyboard Shortcuts

   

Use the following key combinations to move around the Toolbox.

Note   You can change or create keyboard shortcuts in Keyboard of the Environment options in the Options dialog box.

To Press
Display documentation on the Toolbox F1
Close the Toolbox. SHIFT+ESC
Display the shortcut menu. SHIFT+F10
Move left through the controls in the current tab, one control at a time. LEFT ARROW
Move right through the controls in the current tab, one control at a time. RIGHT ARROW
Move to the next tab. (Available only in the Toolbox) CTRL+UP ARROW
Move to the previous tab. (Available only in the Toolbox) CTRL+DOWN ARROW
Move up through the controls, one page at a time. PAGE UP
Move down through the controls, one page at a time. PAGE DOWN
Move up through the controls in the current tab, one control at a time. UP ARROW
Move down through the controls in the current tab, one control at a time. DOWN ARROW
Place the selected control at the insertion point in an editor or the default location on a form. ENTER
