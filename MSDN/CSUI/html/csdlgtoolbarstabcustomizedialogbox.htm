Toolbars Tab (Customize Dialog Box)

   

Allows you to create, rename, delete, and reset your toolbars.

Options

Toolbars

Displays the toolbars built into the development environment and any toolbars you create. When a toolbar is displayed in the environment, a check mark appears to the left of it on this dialog box.

Note   The Menu bar cannot be hidden. It can only be reset.

New

Displays the New Toolbar dialog box where you type the name for your new toolbar in the Toolbar Name box.

Rename

Displays the Rename Toolbar dialog box where you type the new name for your toolbar. Only available if you select a user-defined toolbar.

Delete

Deletes a user-defined toolbar from your project.

Reset

Removes any changes to the built-in toolbars and resets them to their original state. Only available if you select a built-in toolbar.
