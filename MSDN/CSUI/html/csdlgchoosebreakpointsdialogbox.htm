Choose Breakpoints Dialog Box

   

Allows you to choose which breakpoints you want to add from the list of matching breakpoints.

Note   This dialog box appears only if the breakpoint name you typed is used more than once.

Options

Breakpoints List

Lists all of the breakpoints that match the expression listed in the Add Breakpoint dialog box. Check the breakpoints you want to add.

Choose All

Selects all of the breakpoints.

Clear All

Clears all of the breakpoints.
