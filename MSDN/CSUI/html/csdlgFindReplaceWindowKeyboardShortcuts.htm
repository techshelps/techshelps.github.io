Find Window Keyboard Shortcuts

   

Use the following keyboard shortcuts in the Find window.

Note   You can change or create keyboard shortcuts in Keyboard of the Environment options in the Options dialog box.

To Press
Clear the Find and Replace boxes and all options, and create a new search request. Hold down ALT and press F3 and N
Close the Find window and change focus to the editor window. ESC
Display documentation for the Find window. F1
Display the Find window where you can replace the specified text. CTRL+H
Display the Find window with the word under your cursor in the Find box. CTRL+F
Displays the shortcut menu. SHIFT+F10
Find next occurrence of text specified in the Find window. F3
Find previous occurrence of the text specified in the Find window. SHIFT+F3
Find the previous occurrence of the current word or selected text. CTRL+SHIFT+F3
Finds the next occurrence of the current word or selected text. CTRL+F3
Match uppercase and lowercase characters exactly as they appear in the Find combo box. Hold down ALT and press F3 and C.
Put the cursor in the Find box on the Standard toolbar. CTRL+SHIFT+F
Search in the reverse direction. Hold down ALT and press F3 and B.
Set the Find Whole Words option in the Find window. Hold down ALT and press F3 and W.
Stop a search in progress. Hold down ALT and press F3 and S
Turn on regular expression mode so that special characters can be used. Hold down ALT and press F3 and R
