1.3.1 File Transfer API

The File Transfer API provides a simple interface to lower-level APIs, including the WindowsSockets API. You use calls to the File Transfer API to transfer one or more files to a remote application through a VoiceView-enabled modem. An application on the remote computer directly receives the transferred files, or a system-provided, default file transfer application receives the files.

Chapter 2, "File Transfer API Guide," discusses concepts and programming topics for the File Transfer API. Chapter 3, "File Transfer API Reference," provides details on each of the items of the API.
