2.2.1.2 Getting File Transfer Engine Version

Use the xfGetVersion function call at any time in a file transfer application to determine the version of the currently running file transfer engineThe return value can be used to determine if a file transfer application is compatible with the current engine. The return value can also be used as input to the xfInitialize function call if the file transfer application is built for another version of the engine, but it is still compatible with the current version.
