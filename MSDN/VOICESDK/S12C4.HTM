HCONNECTION, LPHCONNECTION

#include <fte.h>

HCONNECTION hConnection; /* 32-bit connection handle */
LPHCONNECTION lphConnection; /* 32-bit pointer to connection handle */

Variables of types HCONNECTION and LPHCONNECTION specify a file transfer connection.

Parameters

hConnection

32-bit value used as a connection handle.

lphConnection

32-bit pointer to a connection handle.

Comments

A connection handle results from a call to the xfConnect function or from a file transfer engine call to the notification procedure. The connection handle remains valid until the application closes the connection with a call to the xfDisconnect function.
