1.3.2 Windows Sockets

The VoiceView Sockets extensions to the Window Sockets API provide a low-level interface to the communications layer of Microsoft Windows. You use sockets when you need to exercise fine control over the VoiceView hardware device and when the File Transfer API provides insufficient functionality for a particular application.

Chapter 4, "VoiceView Sockets API Guide," discusses concepts and programming topics for the VoiceView Sockets API. Chapter 5, "VoiceView Sockets API Reference," provides details on each of the items of the API.
