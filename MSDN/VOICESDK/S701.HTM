CHAPTER 4

Sockets API Guide

Windows Sockets provide a network applications programming interface based on the socket mechanism used by the Berkeley Software Distribution (BSD) of UNIX Ò. The Windows Sockets Version 1.1 specification defines the Windows-specific extensions to the standard BSD routines. VoiceView Sockets in turn provide an additional extension to Windows Sockets by including VoiceView-enabled hardware as communications devices.

This chapter describes the use of the extensions that VoiceView Sockets provide to Windows sockets. Chapter 5, "Sockets API Reference," includes details for different or extended items.
