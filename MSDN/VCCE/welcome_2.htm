Packages and Configurations

Windows CE Toolkit for Visual C++ 5.0 provides an overlay that allows you to simultaneously develop applications for multiple targets. On an x86 host machine, you can build programs from a single makefile for execution and debugging in the following environments:

  • Windows CE Emulation Environment (x86EM)
  • Windows CE SH3 target device (WCESH)
  • Windows CE MIPS target device (WCEMIPS)
