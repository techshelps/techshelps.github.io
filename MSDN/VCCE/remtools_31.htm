Remote Connection Server

The Remote Connection Server (Rsdkmon.exe) is a Visual C++ for Windows CE Version 1.0 tool that starts a connection with the remote device by running Dccman.exe, a program that starts the desktop PC Remote Access Service (RAS) connection. The Remote Connection Server functions as a communications placeholder. It acts as an empty remote client. This makes it easier to start the other remote tools.

The Remote Connection Server will not perform synchronization between the remote device and the desktop PC. If you need to perform synchronization, use the Version 1.0 H/PC Explorer. Use the Remote Connection Server instead of the H/PC Explorer when you want to establish a quick connection to the remote device.

Note The Remote Connection Server is a Version 1.0 tool that was part of Visual C++ for Windows CE. It is not part of the Windows CE Toolkit for Visual C++ 5.0.
