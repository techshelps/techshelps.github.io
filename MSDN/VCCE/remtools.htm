Windows CE Tools

After you install the Microsoft® Windows® CE Toolkit for Visual C++® 5.0, you will be able to use several new tools. The new tools will appear on the Microsoft® Visual Studio™ Tools menu, and enhance development environment functionality by enabling you to access a remote device from a desktop computer and perform a variety of Windows CE-specific tasks.

The following Windows CE tools are provided for use on your desktop computer:

Windows CE Spy

Windows CE Registry Editor

Windows CE Zoom

Windows CE Process Viewer

Windows CE Heap Walker

Remote Object Viewer

Remote Connection Server

When running multiple instances of the Windows CE tools, note the following general rules:

  • You can run multiple instances of Windows CE Zoom.
  • You can only run a single instance of Windows CE Spy, Windows CE Registry Editor, Windows CE Process Viewer, Windows CE Heap Walker, Remote Object Viewer, and Remote Connection Server.
