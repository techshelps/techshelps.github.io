Establishing a Remote Connection

Start any of the Windows CE tools to automatically initiate the appropriate remote connection. If the remote connection initialization fails, run Repllog.exe from the remote device.

    To establish a remote connection

  1. On the Visual Studio Tools menu, click [filename] where filename is the name of one of the Windows CE tools

    -or-

    at the Windows® NT command prompt, type "C:\Program Files\DevStudio\WCE\BIN\[filename]" where filename is the name and extension of any of the Windows CE tools.

    -or-

    on the Visual C++ 5.0 submenu, click [filename] where filename is the name of one of the Windows CE tools.

  2. Turn on your remote device.

Note To make sure you have a working remote connection, start Windows CE Services on your desktop computer, then turn on your remote device. Windows CE Services will try to establish a connection to the remote device; if it cannot establish a connection, run Repllog.exe from the remote device.

    To Run Repllog.exe from your remote device

  1. Tap Start to display the Start menu, and then tap Run.
  2. In the Run dialog box, type "Repllog.exe" in the Open field.
  3. Tap OK.
