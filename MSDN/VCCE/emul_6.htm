Changing the Size of the Emulation Window

If you want to change the display size of the emulation environment window, you'll have to run the Windows NT registry editor (regedit.exe).

    To change the size of the emulation window

  1. If you are running the emulation, close it by clicking the emulation environment Start button, then click Suspend.
  2. Click Start, and point to Run.
  3. In the Open field, type "Regedit" and click OK.
  4. In the Windows Registry editor, click HKEY_LOCAL_MACHINE.
  5. Click SOFTWARE.
  6. Click Microsoft.
  7. Click Windows CE Platform SDK.
  8. Click Emulation.
  9. Click H-PC.
  10. Click Screen Settings.
  11. Modify the SM_CXSCREEN and SMCYSCREEN values and close the registry editor.
  12. Start emulation.
