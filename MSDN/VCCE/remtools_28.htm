Remote Object Viewer

The Remote Object Viewer (Osviewer.exe) utility enables you to:

  • View the object store on the remote device.
  • Browse the emulation file system and object store.
  • Browse the remote device file system and object store.
  • Browse the root directory (containing Windows NT) of the desktop computer hard drive.
  • Delete, copy, and move files, folders, and databases.
  • Move files in and out of the emulation object store (Wceobstr.dat).
