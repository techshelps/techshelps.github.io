Windows CE SDK Environments

Windows® CE currently supports three specific target environments:

  • Windows CE emulation under Windows® NT on an x86 CPU
  • H/PC target platform with an SH CPU
  • H/PC target platform with a MIPS CPU

These environments are established when you install the Microsoft® Windows® CE Toolkit for Visual C++® 5.0, and each environment has a corresponding icon in the Visual C++ program group. Clicking on the icon opens a command line window and sets environment variables to appropriate values for the selected environment.
