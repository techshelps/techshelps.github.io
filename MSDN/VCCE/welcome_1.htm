Introduction

The Windows CE Toolkit for Visual C++ 5.0 is similar to Visual C++ 5.0 in both functionality and design. Several Visual C++ features have not been implemented because of inherent limitations in the Windows CE operating system. These limitations impose certain restrictions that are explained in the following table.

Visual C++ 5.0 Windows CE Toolkit for Visual C++ 5.0
Win32 API Subset of the Win32 API set.
C Runtime Functions Subset of the standard CRT.
CL native compiler CL is used for the native emulation environment.

Additional new compilers:
SHCL for remote SH3 devices (WCESH)
CLMIPS for remote MIPS devices (WCEMIPS)
Common Development Tools Uses the same tools as Win32 for the emulation environment, plus adds Remote Tools. Resource editor features are modified to support unique Windows CE formats, such as 4-color bitmap images.
Microsoft Foundation Classes Supports a subset of the MFC classes supported on Win32.
