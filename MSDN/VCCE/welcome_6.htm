Uninstalling the Windows CE Toolkit for Visual C++ 5.0

If you decide to remove the Windows CE Toolkit for Visual C++ 5.0, perform the following procedure.

    To uninstall the Windows CE Toolkit for Visual C++ 5.0

  • In the Visual C++ program group, click the Uninstall WCE Toolkit for Visual C++ 5.0 icon and follow the on-screen instructions

    -or-

  • from the \VC subdirectory of your Visual Studio installation, double-click Wceremove.exe and follow the on-screen instructions.

    To uninstall both Visual C++ 5.0 and the Windows CE Toolkit for Visual C++ 5.0

  1. Uninstall the Windows CE Toolkit for Visual C++ 5.0 by clicking the Uninstall WCE icon in the Visual C++ 5.0 program group, and then following the on-screen instructions

    -or-

    from the \VC subdirectory of your Visual Studio installation, double-click Wceremove.exe and follow the on-screen instructions.

  2. Uninstall Visual C++ 5.0 by first double-clicking the My Computer icon on your Windows desktop, and then double-clicking the Control Panel icon.
  3. On the Windows Control Panel, double-click Add/Remove Programs.
  4. On the Install/Uninstall folder, select Microsoft Visual C++ 5.0.
  5. Click Add/Remove and follow the on-screen instructions.

    To install any other Visual Studio add-on products

  1. Uninstall the Windows CE Toolkit for Visual C++ 5.0 by clicking the Uninstall WCE icon in the Visual C++ 5.0 program group and follow the on-screen instructions

    -or-

    from the \VC subdirectory of your Visual Studio installation, double-click Wceremove.exe and follow the on-screen instructions.

  2. Install any of the other Visual Studio add-on products.
  3. Reinstall the Windows CE Toolkit for Visual C++ 5.0.
