WCE ATL COM AppWizard

The WCE ATL COM AppWizard generates a starter application that helps you create COM/ActiveX components using the Active Template Library.

    To start a project using the WCE ATL COM AppWizard

  1. On the Visual Studio File menu, click New.
  2. In the New dialog box, click the Projects tab.
  3. Select WCE ATL COM AppWizard.
  4. Select the platforms you want to target from the Platforms checkbox list.
  5. Enter a text string in the Project Name field, and click OK.
