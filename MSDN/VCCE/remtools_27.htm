Viewing a Memory Dump for a ProcessID

To download and view the Memory Dump window for a selected processID, double-click a memory address within the heap.

Note You can display all of the Heap Walker windows by clicking Cascade or Tile on the Windows menu.

The Windows CE Heap Walker Process ID column headings are described in the following table.

Column heading Description
Address The memory address of the heap.
Hex A hex representation of the contents of the heap.
ASCII An ASCII representation of the contents of the heap.
