Resource Editor Features

Windows CE Toolkit for Visual C++ 5.0 does not support the Toolbar resource editor. The following Visual C++ controls in the dialog box resource editor are also not supported.

  • New Hot Key
  • New Animate
  • New Rich Edit
  • Windows Custom control

The Windows CE Toolkit for Visual C++ 5.0 resource editors have been customized to help:

  • Produce icons and bitmap images in the 4-color (two bit-per-pixel grayscale) format used by Windows CE devices.
  • Convert existing bitmap images to 4-color images.
  • Build dialog boxes that use the standard Windows CE features (toolbars, OK buttons and so on).
