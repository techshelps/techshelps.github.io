Remote Tools

The Windows CE Toolkit for Visual C++ 5.0 adds seven menu items to the Microsoft® Visual Studio™ Tools menu. Double-click any of these items to start a Windows CE remote tool from within the Visual Studio IDE. You can also start a remote tool by entering its file name in the Open field of the Windows NT Run dialog box, then clicking OK.

The following table provides a brief description of each of the Windows CE remote tools.

Windows CE Tool Description
Windows CE Spy Displays a graphical view, on the desktop computer, of Windows CE system processes, threads, windows, and window messages running on the remote device.
Windows CE Registry Editor Manages remote device registry entries from the desktop computer.
Windows CE Zoom Captures an image from the remote device and displays it on the desktop computer.
Windows CE Process Viewer Displays a graphical view, on the desktop computer, of Windows CE process and thread characteristics. The Process Viewer enables you to view memory usage and search for memory leaks.
Windows CE Heap Walker Enables you to view detailed information about heap IDs and flags for processes that are running on the remote device. The Windows CE Heap Walker is a useful tool to determine if an application has memory leaks.
Windows CE Object Viewer Displays a hierarchical view, on the desktop computer, of the Windows CE object store. The Remote Object Viewer enables you to transfer objects between the emulation object store and the remote device.
Remote Connection Server Used in version 1.0 to establish a communications link between the desktop computer and the remote device.
