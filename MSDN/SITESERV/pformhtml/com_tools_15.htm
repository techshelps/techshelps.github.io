Starting the Win32-based Pipeline Editor in Expert Mode

The Pipeline Editor can be started in Expert mode only from a command-line prompt.

Caution

In standard mode, the Pipeline Editor has built-in safeguards that prevent some actions that could disable a site’s pipeline. In Expert mode, these protections do not exist. It is recommended that you do not run the Pipeline Editor in Expert mode unless you specifically need the additional functionality it provides.

To start the Pipeline Editor in Expert mode
  1. On the Start menu, click Run.
  2. In the Open box, type the path to the PipeEditor.exe file, and add the switch /e to the end of the command.

    If you installed Commerce Server in the default location, you would type the following command:
    \Microsoft Site Server\Bin\PipeEditor.exe /e

  3. Click OK.
