StandardSManager.CreateShopperID Method

The CreateShopperID method creates and returns a new shopper ID.

Syntax

StandardSManager.CreateShopperID()

Parameters

None.

Remarks

The CreateShopperID method has no parameters, and returns the newly created shopper ID.

For information about StandardSManager modes, see StandardSManager Initialization Modes.
