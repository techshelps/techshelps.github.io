Usages Collection

This collection is a dispinterface on a class List object intended to contain IUsage dispinterfaces. When the parent object’s CDF string is assembled, the GetCDFString method is invoked on each interface to fetch the object's CDF string contribution.

Type

IList (VT_DISPATCH)

Contains

IUsage (VT_DISPATCH)

Default

Empty
