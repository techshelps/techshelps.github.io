OnEndPage Method

After it finishes executing an ASP page, the built-in ASP Server object automatically invokes this method on all objects it created during the execution of ASP script. The method performs all necessary memory de-allocations and object releases.

This method is standard for all classes designed to take advantage of the ASP environment and is not intended to be invoked directly by script in an ASP page. It is listed here strictly for completeness.
