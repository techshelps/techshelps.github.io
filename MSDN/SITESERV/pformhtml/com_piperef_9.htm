DecodeSMIME

The DecodeSMIME component decodes a business data object that was encapsulated in the S/MIME (secure MIME) format.

  • υRead From Field. Specifies the name of the field on the Transport Dictionary from which the encoded data is read.
  • υWrite To Field. Specifies the name of the field on the Transport Dictionary to which the decoded data is written.
  • υWrite Signature Certificate To Field. Specifies the name of the field on the Transport Dictionary to which the name of the signature certificate is written.
  • υWrite Crypt Certificate To Field. Specifies the name of the field on the Transport Dictionary to which the name of the certificate used for decryption is written.
