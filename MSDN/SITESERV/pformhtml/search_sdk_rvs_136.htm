Accessing IGatherLogs Objects

You access an IGatherLogs object by calling IBuildCatalog.Logs method, as in the following example, where objLogs is the name you give to the IGatherLogs object: 

Option Explicit 
On Error Resume Next 

Dim objSearchAdmin, objBuildServer, objCatalogs, objCatalog, objLogs 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1") 
Set objBuildServer = objSearchAdmin.BuildServer 
Set objCatalogs = objBuildServer.BuildCatalogs 
Set objCatalog  = objCatalogs("KMSampleCatalog1")
Set objLogs     = objCatalog.objLogs 

...

'Release objects 
Set objLogs     = Nothing 
Set objCatalog  = Nothing 
Set objCatalogs = Nothing 
Set objBuildServer = Nothing 
Set objSearchAdmin = Nothing
