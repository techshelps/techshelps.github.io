Using the Pipeline Editor Property Pages

The property pages available in the Pipeline Editor provide an easy interface for viewing and modifying pipelines, stages, and components.

The following property pages are available.

Element Property pages Description
Pipeline

(available in Win32® Pipeline Editor only)

 Pipe Properties Pipeline name; the type of pipeline the .pcf file can be used with; comments.
Stage

(available in Win32 Pipeline Editor only)

 Stage Properties Stage name and comments.
Component Component Properties,
Values Read and Written,
component-specific page		 Component name; Class ID; Program ID; description; values read from and written to the order form; values read from the PipeContext.

A component that has configurable properties may have an additional property page whose options are specific to that component.

To view property pages
  • υIn the Win32-based Pipeline Editor, right-click the name of the pipeline, stage, or component, and then on the shortcut menu, click Properties.
  • υIn the ASP-based Pipeline Editor, click the Edit link on a component.
