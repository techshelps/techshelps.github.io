DeleteProject Method

This method deletes a project on the server.

ReplicationServer.DeleteProject(Project)

Parameters

Project

The name of the project to delete.

Remarks

Example

The following example deletes the Project1 project. 

Option Explicit 
On Error Resume Next 

const CRS_ERROR_NO_PROJECTS    = 15100
const CRS_ERROR_PROJECT_NOT_FOUND  = 15101

dim ReplServer
set ReplServer = CreateObject("CrsApi.ReplicationServer")
ReplServer.Initialize("")

Err.Clear

ReplServer.DeleteProject("Project1")

if Err.Number = CRS_ERROR_NO_PROJECTS then
  Wscript.Echo "No projects defined!"
elseif Err.Number = CRS_ERROR_PROJECT_NOT_FOUND then
  Wscript.Echo "Project1 not found!"
end if

'Release Server object
set ReplServer = Nothing
