StopServer Method

The StopServer method stops a specified instance of a Membership virtual server.

IDL Definition

HRESULT StopServer ( [in] LONG lVirtServId ) ;

Parameters

lVirtServId

The Membership virtual server instance ID.

Return Values

A standard HRESULT value

Example

Set ObjCreator = CreateObject("ObjCreator.ObjCreator.1")
Set BrokServers = ObjCreator.CreateObjAuth("MemAdmin.BrokServers.1")

BrokServers.StopServer 1 '   stops server # 1
