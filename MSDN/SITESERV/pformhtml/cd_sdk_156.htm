AddEventSink Method

This method either creates a new event sink or opens an existing event sink.

Syntax

ReplicationServer.AddEventSink(Sink)

Parameters

Sink

The name of the new event sink.

Remarks

You can set the various event sink object properties, but the object is not created until you call the ReplicationEventSink.Commit method on the event sink.

Example

The following example add the new event sink TestSink. 

Option Explicit 
On Error Resume Next 

dim ReplServer
set ReplServer = CreateObject("CrsApi.ReplicationServer")
ReplServer.Initialize("")

dim ReplSink
set ReplSink = ReplServer.AddEventSink("TestSink")
...

'Release objects
set ReplSink   = Nothing
set ReplServer = Nothing
