Admin Related Values

These values all have the bits 0x1900 in common.

HRESULT Value (Hex) Status Code Name Description
0x80041901 NLADMIN_E_DUPLICATE_CATALOG A catalog with the name specified already exists.
0x00041902 NLADMIN_S_NOT_ALL_BUILD_CATALOGS_INITIALIZED At least one catalog definition failed to initialize.  This may be due to insufficient disk space or invalid registry configuration.
0x80041903 NLADMIN_E_FAILED_TO_GIVE_ACCOUNT_PRIVILEGE The account could not be granted a privilege to log on as service on the domain controller.
