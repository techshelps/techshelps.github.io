ProjectFilter Property

This property specifies the project that an event sink monitors.

Syntax

ReplicationEventSinkConfig.ProjectFilter[=Project]

Parameters

Project

The name of the project to monitor for events.

Remarks

You must have Site Server Publishing administrator privileges on the server to get or set this property.

The event sink gets all events for the specified project. If you subsequently call a RegisterForEvent method, only those events are monitored.

Example

The following example instructs the store-and-forward event sink to monitor events for the Project1 project.

Option Explicit 
On Error Resume Next

const CRS_ERROR_NO_MORE_ITEMS  = 0&80003B17

dim ReplServer
set ReplServer = CreateObject("CrsApi.ReplicationServer")
ReplServer.Initialize("")

dim Iterator
Iterator = 0
dim EvSink

do while True
  'Clear any error text
  Err.Clear
  'Get an event sink
  set EvSink = ReplServer.EnumEventSinks(Iterator)

  'Quit if empty object returned (no more instances)
  if IsEmpty(EvSink) then exit do

  'Quit if "No more items" error 
  dim ReplError
  ReplError = Err.Number
  if ReplError = CRS_ERROR_NO_MORE_ITEMS then exit do

  if EvSink.CLSID = "Crssink" then 
    EvSink.ProjectFilter("Project1")
  end if
Loop

'Release objects
set EvSink     = Nothing
set ReplServer = Nothing
