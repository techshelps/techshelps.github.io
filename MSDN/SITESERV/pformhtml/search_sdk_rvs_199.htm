ISearchCatalogs Interface

This interface is a list of search catalogs (collection of ISearchCatalog objects) that Search searches.

Hierarchy

ISearchAdmin.ISearchServer

The following table briefly describes this interface's methods and properties.

Method Description
Add Adds a catalog.
Remove Removes a catalog.
Property Description
Count The number of catalogs.
Item Accesses a catalog.
