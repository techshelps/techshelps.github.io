Pipeline Properties

You can view this property page only in the Win32-based Pipeline Editor.Use this property page to change the name of the pipeline, specify its transaction mode, or save notes about the pipeline.

  • υLabel. Specifies the name that the Pipeline Editor displays for this pipeline. You can change the name by typing a new name here.
  • υTransaction compatibility. Specifies what pipeline object this configuration file can be used with.
    • υAny pipeline. Indicates that this configuration file can be used with an OrderPipeline object, an MtsPipeline object, or an MtsTxPipeline object.
    • υRequires transacted pipeline (MtsTxPipeline). Indicates that this configuration file can be used only with an MtsTxPipeline object. The MtsTxPipeline object is registered with Microsoft® Transaction Server (MTS) as transactions required, and holds a transaction that spans all components in the pipeline.
    • υRequires non-transacted pipeline (MtsPipeline). Indicates that this configuration file can be used only with an MtsPipeline object. The MtsPipeline object is registered with MTS as transactions not supported.
  • υDescription. Provides a place for you to enter comments about this pipeline.
