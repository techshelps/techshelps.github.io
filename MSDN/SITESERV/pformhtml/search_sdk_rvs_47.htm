Count Property

This read-only property specifies the number of catalog definitions (IBuildCatalog interfaces).

Syntax

IBuildCatalogs.Count

Example

The following example displays the number of catalog definitions. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set objCatalogs = objBuildServer.BuildCatalogs

Wscript.Echo "There are " & objCatalogs.Count _
  & " catalog definitions."

'Release objects 
Set objCatalogs = Nothing 
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
