Software Requirements

To write components for Commerce Server, in addition to the software required to run Commerce Server you need the following software tools:

  • If you plan to develop components in C or C++: Microsoft Visual C++® version 4.1 or later.
  • If you plan to develop components in Java: A Java compiler such as Microsoft J++© version 1.0 or later.

You can use any compiler that supports VTABLE binding.
