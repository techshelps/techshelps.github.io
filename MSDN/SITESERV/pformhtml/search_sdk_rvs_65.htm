DefaultPluginAccountName Property

This read-only property specifies the administrative access account name.

Syntax

IBuildServer.DefaultPluginAccountName

Example

The following example displays the administrative access account name. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer

Wscript.Echo "The administrative access account name is " _
  & objBuildServer.DefaultPluginAccountName 

'Release Objects 
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
