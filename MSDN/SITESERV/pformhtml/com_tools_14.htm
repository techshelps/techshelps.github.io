Using the Win32-based Pipeline Editor in Expert Mode

The Win32®-based Pipeline Editor is a powerful tool in its standard mode, but it can also be run in Expert mode, which provides additional capabilities useful to developers of custom pipeline components and pipelines.

Caution

In standard mode, the Pipeline Editor has built-in safeguards that prevent some actions that could disable a site’s pipeline. In Expert mode, these protections do not exist. It is recommended that you do not run the Pipeline Editor in Expert mode unless you specifically need the additional functionality it provides.

In Expert mode, the Pipeline Editor provides the following additional functionality:

  • υCreate a custom pipeline without basing it on a preconfigured template file.
  • υInsert, move, and delete pipeline stages.
  • υCut, copy, and paste a stage, including its components.
  • υAssign globally unique identifiers (GUIDs) to stages.
  • υView the required components that are not displayed in standard mode.
  • υMark a component as Required if it is the last component in a stage.
