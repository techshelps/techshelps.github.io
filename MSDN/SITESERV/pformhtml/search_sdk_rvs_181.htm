BuildServer Property

This read-only property provides access to the IBuildServer interface of the host.

Syntax

ISearchAdmin.BuildServer

Example

The following example retrieves the IBuildServer object for the current host. 

Option Explicit 
On Error Resume Next 

Dim objSearchAdmin, objBuildServer 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer

...

'Release objects 
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
