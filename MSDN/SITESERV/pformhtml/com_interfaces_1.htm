Header Files

The following table lists and describes the header files that are included with the Commerce Server Software Development Kit (SDK).

File Description
billingapi.h Contains the definitions for the Billing Application Programming Interface (API) routines.
commerce.h Contains the interface definitions for the Commerce Server 3.0 objects.
comp_ids.h Contains the class identifiers (CLSIDs) for the pipeline component classes.
mspu_guids.h Contains the CLSIDs for the order processing pipeline objects and interfaces.
pipeline.h Contains the definitions for the pipeline interfaces.
pipecomp.h Contains the proxy/stub definitions for the pipeline interfaces.
pipe_stages.h Contains the globally unique identifiers (GUIDs) for the OPP stages.
