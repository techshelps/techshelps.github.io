GetTimeSaved Method

The GetTimeSaved method returns the date a project was last saved.

VBScript Syntax

IManager.GetTimeSaved name

Parameters

name

A string specifying the name of the project

Return Value

A Date (VT_DATE) value containing the date the project was saved.

Example 

' Using Windows Scripting Host Runtime Env  (cscript.exe)
Option Explicit

Dim Manager
Dim Time_Saved
Dim Projects

Set Manager = CreateObject("Push.Manager")
Set Projects = Manager.GetProjects
For Each Project in Projects
  wscript.echo Project
  wscript.echo "Saved " & Manager.GetTimeSaved(Project)
Next
