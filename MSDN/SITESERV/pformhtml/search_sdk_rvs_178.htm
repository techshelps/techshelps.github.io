ISearchAdmin Interface

This interface provides properties to access the other Search interfaces on a specific host.

ProgID: 

Search.SearchAdmin.1

interfaceThe following table briefly describes this object's properties.

Property Description
BuildServer The IBuildServer interface of the host.
HostName The name of the host being administered.
SearchServer The ISearchServer interface of the host.
