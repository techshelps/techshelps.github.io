Microsoft Site Server 3.0 SDK

Microsoft® Site Server is a comprehensive Web site environment that works with Microsoft® Windows NT® Server and Microsoft® Internet Information Server (IIS) version 4.0. It enables you to build cost-effective Web solutions for the advanced management and targeted delivery of information. This SDK describes the COM/Automation objects provided by the Site Server system.

You can also access this SDK on the Web, part of the Microsoft Platform SDK, to get late breaking information and updates.

To get an understanding of the essential architectures, concepts, and terms, developers should first read about the components they want to work with in the main Site Server documentation. This SDK assumes a familiarity with VBScript, or JScript, and Microsoft Windows® Scripting Host. The primary language used for syntax and examples is VBScript.

The components of this SDK are as follows:

Component Description
Content Deployment Program content replication involving content deployment, Internet retrieval, and component deployment.
Push Program the Microsoft Active Channel Server to automate the process of information delivery using channels.
Personalization and Membership Program the management of customer relationships, memberships, and personalized information.
Search Program the building of search catalogs used for searches on your Web site.
Ad Server Commerce Edition Only—Program the delivery of advertisements to dynamically built web pages.
Commerce and Wallet Commerce Edition Only—Program custom order processing components, and client payment components for the Microsoft® Wallet control.
