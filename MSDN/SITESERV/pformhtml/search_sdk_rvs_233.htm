Remove Method

This method removes a site from the list of sites.

Syntax

IServers.Remove(Site)

Parameters

Site

The site to remove.

Example

The following example removes Microsoft.com from the list of sites. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objServers 

Set objSearchAdmin  = CreateObject("Search.SearchAdmin.1")
Set objBuildServer  = objSearchAdmin.BuildServer 
Set objServers   = objBuildServer.Sites 

objServers.Remove("Microsoft.com") 

'Release objects
Set objServers  = Nothing 
Set objBuildServer = Nothing 
Set objSearchAdmin = Nothing
