ReplicationPostingAcceptor

The ReplicationPostingAcceptor object provides access to replication posting acceptor settings. The following table briefly describes the ReplicationPostingAcceptor object's methods.

Method Description
Enum Lists all Posting Acceptor parameters.
Get Retrieves a Posting Acceptor parameter.
Put Sets a Posting Acceptor parameter.
