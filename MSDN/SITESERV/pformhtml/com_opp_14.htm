Modifying the OrderForm Object

When you pass an OrderForm to the order processing pipeline (OPP), the OrderForm must already be initialized with certain elements of data.

You can modify an OrderForm in any of the following ways:

  • υBy direct assignment. To initialize the shopper_id name/value pair in a page, perform the following assignment:
    objOrderForm.shopper_id = objPage.GetShopperID()
  • υUsing the OrderForm object’s native methods. The OrderForm object implements the AddItem, ClearItems, and Clear methods, which are designed to add items to the items SimpleList, clear items from the items SimpleList, and clear the entire OrderForm, respectively.
  • υUsing the DBStorage object. The DBStorage object is designed to load a row of data from a database table into an OrderForm or Dictionary object.
