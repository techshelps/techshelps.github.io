Extensions Property

This read-only property specifies the list of filename extensions (collection of IExtensions interfaces) to crawl or avoid.

Syntax

IBuildCatalog.Extensions

Example

The following example displays the extensions that the KMSampleCatalog2 catalog definition specifies. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs, objCatalog, objExts 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set objCatalogs = objBuildServer.BuildCatalogs 
Set objCatalog  = objCatalogs("KMSampleCatalog2")

Wscript.Echo objCatalog.Name & " crawls the following extensions:"

For Each objExts in objCatalog.Extensions
  Wscript.Echo objExts.Extension
Next

'Release objects 
Set objExts        = Nothing 
Set objCatalogs = Nothing 
Set objBuildServer = Nothing 
Set objSearchAdmin = Nothing
