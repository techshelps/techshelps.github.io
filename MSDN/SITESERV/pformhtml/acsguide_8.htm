ASP Related COM Classes

The following classes define objects that are useful in constructing personalized channel subscriptions in an ASP environment.

CDFGenerator Class

This class of object can be used to build a user's custom subscription CDF stream in an ASP environment.  It integrates with the Personalization & Membership Active User Object system to retrieve the users selected channels.

CDFGenHelper Class

This class of object can be used in an ASP environment to aid other objects retrieve information about a project.  The object is intended to act as a mini-server, being persisted as an ASP application-scope object.  Objects of class CDFGenerator require instances of this class to b available when requesting a user's personalized channel subscriptions.
