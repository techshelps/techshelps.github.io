IBuildCatalogs Interface

This interface manages a list of catalog definitions (collection of IBuildCatalogSR_IBuildCatalog_Interface interfaces).

Hierarchy

ISearchAdmin.IBuildServer

The following table briefly describes this interface's methods and properties.

Method Description
Add Adds a catalog definition.
Remove Removes a catalog definition.
Property Description
Count The number of catalog definitions.
Item Accesses a catalog definition.
