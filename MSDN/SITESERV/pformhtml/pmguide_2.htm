Overview

The P&M feature SDK can be broken down into the following broad categories:

The P&M COM Classes

The P&M feature provides a set of COM/Automation compliant classes.  Instances of these classes fall into two categories: front-end ASP server-side COM objects, and administrative COM objects.  There are a number of examples demonstrating the use of these objects in various programming languages.

Samples

This SDK comes with two complete programming samples.  The first sample demonstrates how to customize a client DPA login screen to specify a particular realm.  The second is a client/server example that demonstrates how to go about using the Windows NT Security Support Provider Interface (SSPI) in conjunction with the P&M Membership Authentication server.
