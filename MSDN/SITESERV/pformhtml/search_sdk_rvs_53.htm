LoadConfiguration Method

This method restores a catalog build server configuration.

Syntax

IBuildServer.LoadConfiguration(Path)

Parameters

Path

The full pathname to the file containing the build server configuration information.

Remarks

By default, the Search save catalog build server configuration operations in both MMC and WebAdmin loads catalog build server configuration files with a SBC file extension.

Example

The following example loads the catalog build server configuration file C:\Backup\LocalHost.sbc. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer

objBuildServer.LoadConfiguration("C:\Backup\LocalHost.sbc") 

'Release objects 
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
