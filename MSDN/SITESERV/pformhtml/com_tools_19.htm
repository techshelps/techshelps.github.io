Before You Run the Site Packager

The following programs must be located either in the PATH or in the same directory from which you are running the Commerce Site Packager:

  • υcabarc.exe
  • υiexpress.exe
  • υmakecab.exe
  • υwextract.exe

These files are installed as part of the Commerce Server SDK installation. By default they are installed into the same directory as the Commerce Site Packager program (\Commerce\SDK\Commerce\bin).

Before you run the Commerce Site Packager, you must:

  • υCreate the site that you intend to package. The site must have the standard Commerce Server 3.0 site structure.
  • υPrepare the install script that installs the site after it has been copied to the remote computer.
  • υIf there are additional files other than the site itself that you want to bundle with the packaged site, copy those files to the same directory as the install script. (This is not a requirement, but makes it faster to set program options because the necessary files are in one place.)
