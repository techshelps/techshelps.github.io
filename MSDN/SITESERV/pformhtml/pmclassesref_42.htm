CreateServer Method

The CreateServer method creates a Membership virtual server instance.

IDL Definition

HRESULT CreateServer([out] VARIANT* plVirtServId);

Parameters

plVirtServId

on return, the address of a VT_I4 variable containing the new Membership virtual server instance identifier.

Return Values

A standard HRESULT value

Example

Set ObjCreator = CreateObject("ObjCreator.ObjCreator.1")
Set BrokServers = ObjCreator.CreateObjAuth("MemAdmin.BrokServers.1")
BrokServers.CreateServer InstanceID
WScript.Echo "New Instance: " & InstanceID
