DefinedColumns Key

Associated Value Type

Array (VT_ARRAY)

Contains

String (VT_BSTR)

Description

The Search agent (searchagent.vbs) uses the value of this key to set the DefinedColumns property of an instance of the MSSearch.Query object used to search a catalog.

Example

Channel.RefreshOptions("definedcolumns") = Array("itemname", ... )

Applies to Agents

Search
