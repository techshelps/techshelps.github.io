Count Property

This read-only property specifies the number of start addresses in the collection of start addresses.

Syntax

IStartPages.Count

Example

The following example displays the number of start pages for the KMSampleCatalog2 catalog definition. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs, objCatalog, objStartPages 

Set objSearchAdmin    = CreateObject("Search.SearchAdmin.1")
Set objBuildServer    = objSearchAdmin.BuildServer
Set objCatalogs    = objBuildServer.BuildCatalogs 
Set objCatalog     = objCatalogs("KMSampleCatalog2")
Set objStartPages  = objCatalog.objStartPages

Wscript.Echo objCatalog.Name & " has " & objStartPages.Count & " start pages."

'Release objects
Set objStartPages = Nothing 
Set objCatalog    = Nothing 
Set objCatalogs   = Nothing 
Set objBuildServer   = Nothing 
Set objSearchAdmin   = Nothing
