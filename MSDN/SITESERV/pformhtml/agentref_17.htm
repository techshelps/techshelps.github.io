RecurseDirectories Key

Associated Value Type

String (VT_BSTR)

Values

"YES" or "NO"

Description

This option can be used to have the agent recursively descend subdirectories starting with the directory pointed to by the Directory option. If set to "YES", each directory under the directory specified by the Directory option will become a class Channel object. The hierarchy of the directories will be preserved in the Channel object hierarchy.

Example

Channel.RefreshOptions("recursedirectories")="YES"

Applies to Agents

File System
