ICatalogSearchServerList Interface

Hierarchy

ISearchAdmin:IBuildServer:IBuildCatalogs:IBuildCatalog

This interface manages information about Search hosts (collection of IIndexDs interface objects) to which a catalog (ISearchCatalog interface) is propagated.

You access an ICatalogSearchServerList interface object by calling the IBuildCatalog.SearchServers property.
