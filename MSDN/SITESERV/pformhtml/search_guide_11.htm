ICatalogProperties Interface

Hierarchy

ISearchAdmin:IBuildServer:IBuildCatalogs:IBuildCatalog ISearchAdmin:ISearchServer:ISearchCatalogs:ISearchCatalog

This interface specifies the properties of a catalog.

You access ICatalogProperties interface objects by calling any of the following:

Property Description
IBuildCatalog.CurrentBuildProperties For information on the current build.
IBuildCatalog.PreviousBuildProperties For information on the previous build.
ISearchCatalog.Properties Properties of a catalog.
