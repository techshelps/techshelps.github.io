Add Method

This method adds a catalog definition (IBuildCatalog interface).

Syntax

IBuildCatalogs.Add(Catalog, Path)

Parameters

Catalog

The catalog definition to add.

Path

The full path to the catalog definition to add.

Example

The following example adds the catalog definition KMSampleCatalog3 from C:\Catalogs. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set objCatalogs = objBuildServer.BuildCatalogs

objCatalogs.Add("KMSampleCatalog3", "C:\objCatalogs") 

'Release objects
Set objCatalogs = Nothing 
Set objBuildServer = Nothing 
Set objSearchAdmin = Nothing
