IsBuilding Property

This read-only property specifies whether Search is building a catalog.

Syntax

IBuildCatalog.IsBuilding

Example

The following example displays whether Search is building any catalogs. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs, objCatalog 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set objCatalogs = objBuildServer.BuildCatalogs 

For Each objCatalog in objCatalogs
  If objCatalog.IsBuilding Then 
    Wscript.Echo objCatalog.Name & " is building."
  Else
    Wscript.Echo objCatalog.Name & " is not building."
  End If
Next

'Release objects
Set objCatalogs = Nothing 
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
