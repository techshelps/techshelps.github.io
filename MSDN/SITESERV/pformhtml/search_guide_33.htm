ISitePaths Interface

Hierarchy

ISearchAdmin:IBuildServer:IBuildCatalogs:IBuildCatalog:ISiteRestrictions:ISiteRestriction

This interface manages site path rules (collection of ISitePath interface objects) that tell Search whether to crawl or avoid certain paths.

You access an ISitePaths interface object by calling the ISiteRestriction.Paths property.
