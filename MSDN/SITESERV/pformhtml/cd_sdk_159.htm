ClearEvents Method

This method clears one or all event sinks.

Syntax

ReplicationServer.ClearEvents[(EventSink)]

Parameters

EventSink

The name of the event sink to clear. If Empty (""), all event sinks are cleared.

Remarks

You must have Site Server Publishing administrator privileges on the server to call this method.

See ReplicationServer.EnumEventSinks for a description of the default event sinks.

Example

The following example clears the database event sink.

Option Explicit 
On Error Resume Next 

dim ReplServer
set ReplServer = CreateObject("CrsApi.ReplicationServer")
ReplServer.Initialize("")

ReplServer.ClearEvents(CRSDBSink)

'Release Server object
set ReplServer = Nothing
