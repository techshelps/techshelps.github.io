Deleting Rows with DBStorage

You delete a row from a DBStorage object’s underlying storage in one of the following ways:

  • υYou can initialize an OrderForm or Dictionary object to describe the data you want to delete, and then pass the initialized object to the DBStorage object’s DeleteData method. In the following example, MSCSOrderFormStorage is a DBStorage object and mscsOrderForm is the customer’s basket. The DeleteData method deletes the contents of the customer’s order form storage by passing in the customer’s basket:
    REM Purchase was successful. Delete the orderform from storage
call MSCSOrderFormStorage.DeleteData(null, mscsOrderForm)
  • υAlternately, to delete an entire row based on the value stored in the column designated as the table’s key, pass the value for that column to DeleteDataKey. In the following example, MSCSShopperStorage is a DBStorage object in which the column shopper_id has been specified as the key. The example deletes the information for a given customer from the table: 
    shopper_id = Page.GetShopperID
call MSCSShopperStorage.DeleteDataKey(Null, shopper_id)
