The Pipeline Editor

The Pipeline Editor is an application used for editing Commerce Interchange Pipelines and Commerce Server order processing pipelines. There are two versions of the Pipeline Editor:

  • υThe Win32®-based Pipeline Editor. Used for editing a pipeline on the local computer or on a computer connected to a local area network (LAN).
  • υThe ASP-based Pipeline Editor. Used for editing a pipeline remotely over the Internet using a Web browser.

The functionality is very similar in the two versions. You can edit existing pipelines in either Pipeline Editor, but you can create a new pipeline only in the Win32-based Pipeline Editor.
