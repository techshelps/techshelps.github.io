Commerce Site Packager

The Commerce Site Packager is a utility that packages an existing Commerce Server site into a self-extracting executable file. This file can then be sent over the Internet for installation at a remote location.

The Site Packager program files are installed into the \Commerce\SDK\Commerce\bin directory as part of the Commerce Server SDK installation. (To install the SDK, you must choose the Custom option during Setup, and then select the SDK check box.)
