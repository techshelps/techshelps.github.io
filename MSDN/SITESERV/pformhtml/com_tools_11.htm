Values Read and Written

You can view this property page only in the Win32-based Pipeline Editor.Use this property page to view the values that this component reads from and writes to the order form and the values it reads from the PipeContext Dictionary object.

Note

The Pipeline Editor is unable to display Values Read and Written for components that do not support the IPipelineComponentDescription interface (such as Commerce Server 2.0 components).

  • υValues Read. Specifies the values that this component reads from the order form before it performs its function.
  • υValues Written. Specifies the values that this component writes to the order form after it has completed its function.
  • υContext Values Read. Specifies the values that this component reads from the PipeContext Dictionary object. The PipeContext is passed as a parameter to the pipeline object’s Execute method.
