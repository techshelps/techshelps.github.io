SearchAdmin Property

This read-only property specifies the parent ISearchAdmin object.

Syntax

IBuildServer.SearchAdmin

Remarks

This property is a convenient way to get to the parent interface of the current IBuildServer object.

Example

The following example displays the host name of the associated ISearchAdmin object. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objBuildServerParent 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer

Set objBuildServerParent = objBuildServer.SearchAdmin 

If objBuildServerParent is objSearchAdmin Then 
  Wscript.Echo "Identical ISearchAdmin interfaces"
Else
  Wscript.Echo "Error! The ISearchAdmin interfaces differ!"
End If

'Release Objects
Set objBuildServerParent = Nothing 
Set objBuildServer       = Nothing
Set objSearchAdmin       = Nothing
