The Channel Agent

The Active Channel Server agent is simply a script written in a language such as VBScript or JScript.  The only requirement made about a given language is that it must have an installed ActiveX Scripting engine.  By default, all machines with the Active Server Pages environment installed have scripting engines for VBScript and JScript.

The Active Channel Server comes with 6 pre-installed agents.   The files for these agents can be found in the directory "c:\microsoft site server\bin".

The agents are implemented in the following files:

FileSystemAgent.vbs
DatabaseAgent.vbs
BriefingAgent.vbs
PromotionsAgent.vbs
SearchAgent.vbs

These agents can be used as starting points for the construction of new agents.
