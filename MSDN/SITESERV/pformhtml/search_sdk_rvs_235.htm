Count Property

This read-only property specifies the number of sites in the list of sites.

Syntax

IServers.Count

Example

The following example displays the number of sites in the list. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objServers 

Set objSearchAdmin  = CreateObject("Search.SearchAdmin.1")
Set objBuildServer  = objSearchAdmin.BuildServer
Set objServers   = objBuildServer.Sites 

Wscript.Echo "There are " & objServers.Count & " sites in the list." 

'Release objects
Set objServers  = Nothing 
Set objBuildServer = Nothing 
Set objSearchAdmin = Nothing
