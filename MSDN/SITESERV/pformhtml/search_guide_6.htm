Managing Catalogs and Search Requests

To manage catalogs, you obtain a collection of them to which you can add, remove, and modify individual catalogs.

You can perform the following configurations and actions on a Search catalog object:

  • Get the name of the server that built the catalog.

  • Enable or disable searching on a catalog.

  • Update a catalog.

  • Set the maximum number of result rows for a query.

  • Get catalog properties.

  • Set the timeout value for a query to finish.

  • Determine the status of a catalog.
