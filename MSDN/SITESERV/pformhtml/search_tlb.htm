Interface Type Information

Type libraries contain data type information about interfaces. You can get type information about interfaces by accessing a type library using the VB Object Browser or the VC++ OLE/COM Object Viewer.

Site Server Search Administration 1.0 Type Library

This type library contains information about the Search administrative interfaces and is available as a resource in NLAdmin.dll.

Site Server Indexer Type Library

This type library contains information about the Search schema interfaces and is available as a resource in nlindex.dll.

Site Server Gatherer Type Library

This type library contains information about the Search Gatherer interfaces and is available as a resource in gather.dll.

Site Server Gathering Service Type Library

This type library contains information about the Search Gatherer service interfaces and is available as a resource in gthrsvc.exe.
