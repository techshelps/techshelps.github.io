StandardSManager.InitManager Method

The InitManager method initializes the StandardSManager object to the specified site key and mode.

Syntax

StandardSManager.InitManager(SiteKey, Mode)

Parameters

SiteKey
The name of the site.
Mode
A string that indicates how customer information is to be stored (see StandardSManager Initialization Modes).
