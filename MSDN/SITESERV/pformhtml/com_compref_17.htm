AdminSite.Version Property

The Version property returns the version of Commerce Server as an integer (2 for Commerce Server 2.0, and 3 for Commerce Server 3.0).

Syntax

AdminSite.Version

Example

The following example creates and initializes an AdminSite object for the Clocktower site, which is in the default Web site, then tests the version number of the site:

Set AdminSite = Server.CreateObject("Commerce.AdminSite")
Call AdminSite.Initialize("1", "Clocktower")
Call AdminSite.ReadDefaultProperties 

If AdminSite.Version = 3 Then
    REM Special processing for Commerce 3.0 site
End If

Parameters

None.
