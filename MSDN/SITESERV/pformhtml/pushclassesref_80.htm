Class Manager

Programmatic Identifier: Push.Manager.1

The Push.Manager.1 COM class, hereafter referred to as the Manager class, defines an Automation-compliant object that can be used to manage aspects of the Active Channel Server system, including projects and installed agents.

The Manager class exposes a dispinterface to the methods listed below. Therefore, all invocations must proceed through the IDispatch interface.

Dispinterface IManager

Methods

Name Description
Delete Deletes a specified project.
GetAgents Returns information about installed agents.
GetProjects Returns information about saved projects.
GetTask Returns task information about a specified project.
GetTimeSaved Returns the last time a specified project was saved.
