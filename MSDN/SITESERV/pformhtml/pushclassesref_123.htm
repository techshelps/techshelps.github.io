AutoInstall Property

The AutoInstall property is a string used when building the <SOFTPKG></SOFTPKG> element upon invocation of the GetCDFString method. If this property is not empty, the AUTOINSTALL attribute is generated, and the string is inserted as its value.

Type

String (VT_BSTR)

Default

None

Example

Softpkg.AutoInstall = 1
