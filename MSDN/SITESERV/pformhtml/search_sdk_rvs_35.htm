SearchServers Property

This read-only property specifies the list of search server names (collection of ICatalogSearchServerList interfaces) to which the catalog is propagated.

Syntax

IBuildCatalog.SearchServers

Example

The following example displays the search servers for all catalogs. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs, objCatalog, Server 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set objCatalogs = objBuildServer.BuildCatalogs 

For Each objCatalog in objCatalogs
  Wscript.Echo objCatalog.Name & " is propagated to search servers:"

  For Each Server in objCatalog.SearchServers
    Wscript.Echo "  " & Server.Name
  Next
Next

'Release objects 
Set Server   = Nothing 
Set objCatalog  = Nothing 
Set objCatalogs = Nothing 
Set objBuildServer = Nothing 
Set objSearchAdmin = Nothing
