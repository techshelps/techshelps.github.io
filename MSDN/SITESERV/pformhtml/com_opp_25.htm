Required Components

The order processing pipeline (OPP) contains required components, components that generally verify the integrity of work performed within a given stage.

Unless you run the Win-32® Pipeline Editor in expert mode, these components are not displayed. They are simply appended to the end of each stage. To run the Pipeline Editor in expert mode, type the path to the Pipeline Editor at the command line, followed by the /g switch.
