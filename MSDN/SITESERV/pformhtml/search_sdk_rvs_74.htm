Sites Property

This read-only property specifies a list of sites (IServers interface) with access frequency restrictions.

Syntax

IBuildServer.Sites

Example

The following example displays the names of the sites with access frequency restrictions. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, ServersObj, Site, Hit 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set ServersObj = objBuildServer.Sites 

Wscript.Echo "The following sites have timing restrictions: "

For Each Site in ServersObj
  Wscript.Echo Site.Name 
  Hit = Site.HitInterval

  If Hit = -1 Then
    Wscript.Echo "(Request docs simultaneously)"
  elseif Hit < 0 Then
    Wscript.Echo "(Request at most " & -Hit 
    & " docs simultaneously)"
  Else
    Wscript.Echo "(Wait " & Hit & " seconds between requests)"
  End If
Next 

'Release objects
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
