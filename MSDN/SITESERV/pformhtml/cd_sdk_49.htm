ProviderCLSID Property

This property specifies the ProgID or CLSID of the provider for the store-and-forward event sink.

Syntax

ReplicationEventSinkConfig.ProviderCLSID[=ProviderID]

Parameters

ProviderID

The ProgID or CLSID of the provider (the Content Deployment server from which files are replicated) for the store-and-forward event sink.

Remarks

You must have Site Server Publishing administrator privileges on the server to set this property.

Example

The following example displays the CLSID of provider for the store-and-forward event sink.

Option Explicit 
On Error Resume Next

const CRS_ERROR_NO_MORE_ITEMS  = 0&80003B17

dim ReplServer
set ReplServer = CreateObject("CrsApi.ReplicationServer")
ReplServer.Initialize("")

dim Iterator
Iterator = 0
dim EvSink

do while True
  'Clear any error text
  Err.Clear
  'Get an event sink
  set EvSink = ReplServer.EnumEventSinks(Iterator)

  'Quit if empty object returned (no more instances)
  if IsEmpty(EvSink) then exit do

  'Quit if "No more items" error 
  dim ReplError
  ReplError = Err.Number
  if ReplError = CRS_ERROR_NO_MORE_ITEMS then exit do

  if EvSink.CLSID = "Crssink" then 
    Wscript.Echo "CLSID for Crssink: " & EvSink.ProviderCLSID
  exit do
end if
Loop

'Release objects
set EvSink     = Nothing
set ReplServer = Nothing
