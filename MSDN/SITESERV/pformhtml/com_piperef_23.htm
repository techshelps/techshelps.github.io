ExecuteProcess

The ExecuteProcess component runs an executable file.

  • υPath Name of the executable. Specifies the complete path to the executable file that is to be run by this component. To locate the file, click Browse (Win32 only).
  • υArguments to the executable. Specifies the arguments that are to be passed to the executable file. The argument may contain %1, which will be replaced by the name specified in the Field Name of substitutable parameter field.
  • υField Name of substitutable parameter. (Optional) Specifies the name of a field on the OrderForm that is to be substituted for the %1 in the argument that is passed to the executable file.
