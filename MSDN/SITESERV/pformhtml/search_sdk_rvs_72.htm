ProxyUsageSetting Property

This read-only property specifies the proxy server configuration.

Syntax

IBuildServer.ProxyUsageSetting

Remarks

ProxyUsageSetting can have one of the following values:

0

Uses the default system proxy server.

1

Does not use a proxy server.

2

Uses the proxy server specified by the ProxyAddress property.

This property is set by the first argument to the SetProxySettings method.

Example

The following example displays the proxy usage setting. 

Option Explicit 
On Error Resume Next

Const UseSystemDefaultProxyServer = 0
Const DontUseProxyServer          = 1
Const UseSpecifiedProxyServer     = 2

Dim objSearchAdmin, objBuildServer, ProxySetting

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer

ProxySetting = objBuildServer.ProxyUsageSetting

If ProxySetting = DontUseProxyServer Then
  Wscript.Echo "Search does not use a proxy server"
End If

If ProxySetting = UseSystemDefaultProxyServer Then
  Wscript.Echo "Search is using the default system proxy server"
End If

If ProxySetting = UseSpecifiedProxyServer Then
  Wscript.Echo "Search is using the proxy server " & 
    objBuildServer.ProxyAddress & " 
    on port " & objBuildServer.ProxyPortNumber 
End If

'Release Objects
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
