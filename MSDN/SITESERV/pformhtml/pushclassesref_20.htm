DeleteOnRefresh Property

This property marks the object for deletion from a parent Channel object's collection. The deletion occurs when a process invokes the DeleteObjects method on the parent.

Type

Integer (VT_I4)

Default

0 (False)

Remarks

An agent or program must call the DeleteObjects method explicitly to delete the marked objects in collections.

Example

VBScript

Set IChannel = CreateObject("Push.Channel")
Set IItem = IChannel.AddItem
IItem.DeleteOnRefresh = TRUE
