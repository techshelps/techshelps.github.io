Stage Properties

You can view this property page only in the Win32-based Pipeline Editor.Use this property page to change the name of the stage or to save notes about the stage.

  • υLabel. Specifies the name that the Pipeline Editor displays for this stage. You can change the name by typing a new name here.
  • υDescription. Provides a place for you to enter comments about this stage.
