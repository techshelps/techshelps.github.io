CurrentPages Property

This read-only property lists the documents being crawled.

Syntax

IBuildCatalog.CurrentPages

Example

The following example displays the documents being crawled while building the KMSampleCatalog2 catalog definition. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs, objCatalog, Doc, i 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set objCatalogs = objBuildServer.BuildCatalogs 
Set objCatalog  = objCatalogs("KMSampleCatalog2")

Doc = objCatalog.CurrentPages

Wscript.Echo "objCatalog " & objCatalog.Name _
  & "is searching the following documents:" 

For i = Lbound(Doc) to Ubound(Doc)
  Wscript.Echo Doc(i)
Next

'Release objects 
Set objCatalog  = Nothing 
Set objCatalogs = Nothing 
Set objBuildServer = Nothing 
Set objSearchAdmin = Nothing
