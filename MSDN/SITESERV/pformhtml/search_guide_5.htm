Configuring the Search Server

After the catalogs are built, you can control which catalogs are used for searches and specify settings for the Search Server.

The Search Server, obtained by the SearchServer property, serves two purposes:

  1. Obtains a catalog collection for adding, removing, and configuring catalogs. This collection is obtained by the SearchCatalogs property.

  2. Provides the following configuration options, such as:
    • Gets the location of catalogs on the Search Server.

    • Sets the default catalog used for a search.

    • Sets the performance level of host resources dedicated to searching.
