Indexer (Exchange) Related Values

These values all have the bits 0x1100 in common.

HRESULT Value (Hex) Status Code Name Description
0x800411a0 EXCI_E_NO_CONFIG The configuration properties for Microsoft Exchange have not been set. Specify these properties, and then try again.
0x800411a1 EXCI_E_INVALID_SERVER_CONFIG The name of the Exchange Server specified in the start address does not match the Exchange Server specified in the Microsoft Exchange configuration properties. Change the name of the Exchange Server in the start address or configuration properties to match the other, and then try again.
0x800411a2 EXCI_E_ACCESS_DENIED Insufficient privilege to access Exchange server. Verify that the Exchange indexing configuration is valid and that the Search Service is running in the context of a user having administrative privileges on the Exchange server configuration node.
0x800411a3 EXCI_E_INVALID_EXCHANGE_SERVER The Exchange server name specified in the URL does not match the one specified in Exchange Indexing configuration.
0x800411a4 EXCI_E_BADCONFIG_OR_ACCESSDENIED Insufficient privilege to access Exchange server. Verify that the Exchange indexing configuration is valid and that the Search Service is running in the context of a user having administrative privileges on the Exchange server configuration node.
0x800411a5 EXCI_E_WRONG_SERVER_OR_ACCT The service cannot access the Exchange server. The configured Exchange server name may be invalid, the Exchange server may be temporarily unavailable, or the account used by the Search services may not have administrative access to the Exchange server.
0x800411a6 EXCI_E_NOT_ADMIN_OR_WRONG_SITE The service cannot access the Exchange Service. The account used by the Search services may not be an Exchange administrator, or the Site or Organization settings on the Search host may be invalid.
