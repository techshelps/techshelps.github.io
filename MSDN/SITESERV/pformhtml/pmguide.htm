Introduction

Welcome to the Personalization and Membership (P&M) Programmer's Guide.  This guide contains information and examples that will help you start programming with the various parts of this feature.  This guide is a part of the Site Server 3.0 Software Development Kit (SDK).  For this feature, the kit includes the following parts:

Programmer's Guide

The programmer's guide contains overview information and examples of using the various parts of the P&M feature of Site Server 3.0.

Programmer's Reference

This reference contains information on the various COM classes used to interact with the P&M system.  It also includes reference information required to configure the system

Samples

The kit comes with a set of samples that can be used to learn more about the P&M system.

In addition, the Programmer's Guide contains many examples of using the various COM classes.
