Name Property

This property specifies the name of the project.

Syntax

ReplicationProject.Name[=Name]

Parameters

Name

The name of the project.

Remarks

Name does not take effect until Commit is called.

Example

The following example retrieves the names of the existing projects. 

Option Explicit 
On Error Resume Next 

const CRS_ERROR_NO_MORE_ITEMS  = 0&80003B17

dim ReplServer
set ReplServer = CreateObject("CrsApi.ReplicationServer")
ReplServer.Initialize("")

dim Iterator
Iterator = 0
dim ReplProject
dim RepError
Wscript.Echo "The project names are:"

while True do
  Err.Clear
  set ReplProject = ReplServer.EnumProjects(Iterator)
  if IsEmpty(ReplProject) then exit do
  RepError = Err.Number
  if RepError = CRS_ERROR_NO_MORE_ITEMS then exit do
  
  Wscript.Echo ReplProject.Name
Loop
'Release objects
set ReplProject = Nothing
set ReplServer  = Nothing
