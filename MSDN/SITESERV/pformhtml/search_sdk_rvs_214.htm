ISearchServer Interface

This interface manages search catalogs (through the ISearchCatalogs interface) and catalog search server properties.

Hierarchy

ISearchAdmin

The following table briefly describes this interface's properties.

Property Description
CatalogsLocation The location of the Search catalogs.
DefaultCatalog The default Search catalog.
PerformanceLevel The amount of host resources used for searching.
SearchAdmin The parent ISearchAdmin interface.
SearchCatalogs The ISearchCatalogs interface on the host.
