ICatalogProperties Interface

This interface specifies the properties of a catalog.

Hierarchy

ISearchAdmin.IBuildServer.IBuildCatalogs.IBuildCatalog ISearchAdmin.ISearchServer.ISearchCatalogs.ISearchCatalog

The following table briefly describes this interface's properties (all are read-only).

Property Description
BuildDate The date and time when the catalog was built.
IndexSize The size, in megabytes, of the catalog.
NumberOfDocuments The number of documents in the catalog.
PropertyStoreSize The size, in megabytes, of the property store.
PropSeqNum The crawl number.
UniqueKeyCount The number of unique keys in the catalog.
