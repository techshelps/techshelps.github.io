SearchServer Property

This read-only property specifies the name of the search server (ISearchServer interface).

Syntax

ISearchCatalog.SearchServer

Example

The following example displays the name of the search server for the KMSampleCatalog1 catalog. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objSearchServer, objCatalogs, objCatalog 

Set objSearchAdmin  = CreateObject("Search.SearchAdmin.1")
Set objSearchServer = objSearchAdmin.SearchServer
Set objCatalogs  = objSearchServer.SearchCatalogs
Set objCatalog   = objCatalogs("KMSampleCatalog1") 

Wscript.echo "The search server is " & objCatalog.SearchServer 

'Release objects
Set objCatalog   = Nothing 
Set objCatalogs  = Nothing 
Set objSearchServer = Nothing
Set objSearchAdmin  = Nothing
