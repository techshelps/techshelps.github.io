Count Property

This read-only property specifies the number of search servers to which a catalog definition is propagated.

Syntax

ICatalogSearchServerList.Count

Example

The following example displays the number of search server to which all catalog definitions are propagated. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs, objCatalog, objServers 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set objCatalogs = objBuildServer.BuildCatalogs 

For Each objCatalog in objCatalogs 
  Set objServers = objCatalog.SearchServers 
  Wscript.Echo objCatalog.Name & " is propated to " & objServers.Count 
    & " servers." 
Next

'Release objects
Set objServers  = Nothing 
Set objCatalog  = Nothing 
Set objCatalogs = Nothing 
Set objBuildServer = Nothing 
Set objSearchAdmin = Nothing
