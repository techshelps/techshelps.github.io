Page.IncrShopperPerfCtrs Method

Note

The IncrShopperPerfCtrs property is included only for backward compatibility with sites created in Commerce Server 2.0, and will be discontinued in future releases.

The IncrShopperPerfCtrs method performs no operation in Commerce Server 3.0.

Syntax

Page.IncrShopperPerfCtrs()
