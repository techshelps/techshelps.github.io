AbortProp Method

This method aborts the propagatation of a build catalog.

Syntax

IBuildCatalog.AbortProp

Example

The following example begins a forced propagation of a catalog specified by the  KMSampleCatalog1 catalog definition, presenting an abort button. 

Option Explicit
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs, objCatalog
Dim Prompt, Buttons, Title, Response

Prompt = "Click OK to kill the forced propagation"
Buttons = vbOKOnly
Title = "Kill Forced Propagation?"

Err.Clear

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set objCatalogs    = objBuildServer.BuildCatalogs
Set objCatalog     = objCatalogs.Item("KMSampleCatalog1")

'Start forced propagation
objCatalog.ForceProp

'Present message box so user can cancel forced propagation operation
Response = MsgBox(Prompt, Buttons, Title)

If Response = vbOK Then
  objCatalog.AbortProp
End If

'Display its status
Wscript.Echo "Status: " & objCatalog.Status

'Release objects
Set objCatalog      = Nothing
Set objCatalogs     = Nothing
Set objBuildServer  = Nothing
Set objSearchAdmin  = Nothing
