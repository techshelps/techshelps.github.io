NDRRecipientKeywords Property

This property is a comma-separated list of words used when identifying the recipient line in mail items that were undeliverable.

IDL Definition

[propget]
HRESULT NDRRecipientKeywords( [out, retval]  VARIANT  *pVal /*VT_BSTR*/);
[propput]
HRESULT NDRRecipientKeywords( [in]  BSTR  newVal );

Type

BSTR

Default

" To, Delivery-To"
