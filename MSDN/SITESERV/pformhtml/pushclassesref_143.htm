Core Container Classes

Below is a brief description of various utility classes that are routinely used by Active Channel Server classes to organize information. Examples of use by Active Channel Server classes are listed as well. It should be noted that these objects are not true COM classes, but are native within the Active Channel Server COM classes. Consequently, they cannot be created independently of the various Push COM classes. However, dispinterfaces on each are returned by Push classes, so they act essentially like any other COM object.
