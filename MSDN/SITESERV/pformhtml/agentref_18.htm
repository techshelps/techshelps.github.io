FileModifiedLastTime Key

Associated Value Type

Date (VT_DATE)

Description

The File System agent will use this option to compare modification times for files found during the refresh cycle.  If the file has been modified since the date specified by the option, it will be included as a new item in the Channel object.

Example

Channel.RefreshOptions("filemodifiedlasttime") = #6/30/2001 23:00:00#

Applies to Agents

File System
