DefaultOrderSubtotal

The DefaultOrderSubtotal component, which usually appears in the Order Subtotal stage of an order processing pipeline, calculates the subtotal for an order and stores the resulting value in the order._oadjust_subtotal member of the OrderForm. This value represents the sum of the item._oadjust_adjustedprice name/value pair for every item in the OrderForm’s items SimpleList.
