SubLanguage Property

This property specifies the language dialect to which the list of ignored words applies.

Syntax

IStoplist.SubLanguage

Remarks

SubLanguage is an integer containing the Win32 sub-language ID to which the list of ignored words applies.

The Language and SubLanguage values must be unique in this Search schema.
