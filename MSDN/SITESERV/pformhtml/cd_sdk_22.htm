ReplicationClientError

The ReplicationClientError object provides ReplicationClient error information to your Content Deployment applications.

The following table briefly describes the ReplicationClientError object's properties.

Property Description
Description Textual description of error
Destination Destination computer of error
ErrorCode Content Deployment or system error code
FileName Filename associated with the error 
Project Project in which error occurred
Source Source machine on which error occurred
