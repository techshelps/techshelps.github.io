Creation Error

This property page displays an error to help you determine the reason that a pipeline component could not be loaded.

Try the following solutions:

  • υDelete the component from the pipeline and then reinsert it.
  • υMake sure that the DLL associated with the component is properly registered.
  • υIf neither of the above solutions corrects the problem, try reinstalling Commerce Server.
