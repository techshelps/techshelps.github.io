TempPath Property

This property specifies the directory where Search stores temporary files.

Syntax

IBuildServer.TempPath [=Directory]

Parameters

Directory

The full pathname to the directory where Search stores temporary files.

Example

The following example specifies that Search stores temporary files in C:\Temp. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer

objBuildServer.TempPath = "C:\Temp" 

...

'Release Objects
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
