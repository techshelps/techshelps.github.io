Certificate Management Tool

The Certificate Management Tool enables you to manage your certificates and those of your trading partners, and to generate certificate requests.

The EncryptPKCS, DecryptPKCS, DigitalSig, VerifyDigitalSig, EncodeSMIME and DecodeSMIME components used in Commerce Interchange Pipelines use certificates to perform operations on data stored in the business data object. To use these components, you must first configure the Certificate Management Tool to import and export certificates generated using the Microsoft Certificate Server.

Note

At this time, the Certificate Management Tool works only with the Windows NT Certificate Server. Support by other certificate authorities is currently in progress.

This section explains how to use the Certificate Management Tool to perform the following tasks:

  • υGenerate certificate requests.
  • υGenerate certificates using Microsoft Certificate Server.
  • υImport certificates from certificate chain (.chn) files.
  • υExport certificates to trading partners.
