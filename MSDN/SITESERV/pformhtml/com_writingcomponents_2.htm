Sources of Information on COM

Because both of the samples that are used in this section to illustrate component building rely largely on both Microsoft® Visual C++® and the Component Object Model (COM), this section assumes that you have a working knowledge of both.

If you are not familiar with COM, please consult the following resources.

  • υRogerson, Dale. Inside COM, Microsoft Press, 1997.
  • υChappell, David. Understanding ActiveX and OLE: A Guide For Developers and Managers, Microsoft Press, 1996.
Note

This section also assumes that you are using Visual C++ version 5.0 to create your custom components. You can, however, create pipeline components using any build environment that supports the creation of free-threaded COM server objects.
