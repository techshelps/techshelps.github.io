LogUnSub Property

This property specifies whether to log unsubscribe messages received.

IDL Definition

[propget]
HRESULT LogUnSub([out, retval] VARIANT *pVal /*VT_I4: 1=TRUE,0=FALSE*/);
[propput]
HRESULT LogUnSub([in] BOOL newVal);

Type

BOOL

Values

1=TRUE,0=FALSE
