DataTimeout Property

This property specifies how long,  in seconds, to wait for data to arrive when accessing a site.

Syntax

IBuildServer.DataTimeout

Remarks

The default duration is 20 seconds.

Example

The following example displays the current data timeout duration, in seconds. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer

Wscript.Echo "The current data timeout is " _
  & objBuildServer.DataTimeout & " seconds." 

'Release Objects
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
