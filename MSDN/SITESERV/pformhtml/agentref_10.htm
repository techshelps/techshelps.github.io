ChannelHREF Key

Associated Value Type

String (VT_BSTR)

Description

An Agent uses the integer value passed with this option to set the HREF property for each Channel object that the agent creates during the refresh cycle.

Example

' VBScript
Channel.RefreshType="file system"
Channel.RefreshOptions("channelhref") = "/channel1/page.htm"

Applies to Agents

File System
