Creating a Custom Pipeline in Expert Mode

Using the Win32-based Pipeline Editor in Expert mode, you can create a custom pipeline without basing it on one of the preconfigured pipeline template files (.pct).

Caution

In standard mode, the Pipeline Editor has built-in safeguards that prevent some actions that could disable a site’s pipeline. In Expert mode, these protections do not exist. It is recommended that you do not run the Pipeline Editor in Expert mode unless you specifically need the additional functionality it provides.

To create a new pipeline in Expert mode
  1. Start the Win32 Pipeline Editor in Expert mode.
  2. On the File menu, click New.
  3. In the Choose a Pipeline Template box, double-click Empty.pce.

    This opens an empty pipeline file.

  4. Right-click Empty Pipeline Template, and then on the shortcut menu, click Properties. In the Label field, type a new name for the pipeline. Select an option for Transaction Compatibility, add a description of the pipeline in the Description field, and then click OK.
  5. Right-click the name of the pipeline and then on the shortcut menu, click Insert Stage. Right-click New Stage, and then on the shortcut menu, click Properties. Set the stage properties, and then click OK.
  6. Right-click the name of the stage, and then on the shortcut menu, click Insert Component. For Stages, select All, click the name of the component you want to insert, and then click OK.
  7. To configure the new component, right-click the component, and then on the shortcut menu, click Properties. Modify the component’s properties as necessary, and then click OK.
  8. On the File menu, click Save As, and then type a name for the pipeline configuration file.

    When the Pipeline Editor saves the file, it adds the .pcf file extension.
