ConnectTimeout Property

This property specifies how long, in seconds, to wait for a connection when accessing a site.

Syntax

IBuildServer.ConnectTimeout

Remarks

The default duration is 20 seconds.

Example

The following example displays the current connection timeout duration, in seconds. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer

Wscript.Echo "The current connection timeout is " _
  & objBuildServer.ConnectTimeout & " seconds."

'Release Objects
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
