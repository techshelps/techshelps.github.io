BuildServer Property

This read-only property specifies the catalog build server (IBuildServer interface).

Syntax

IBuildCatalog.BuildServer

Example

The following example retrieves the catalog build server for a catalog definition and verifies its accuracy. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs, objCatalog, objBldSvr 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set objCatalogs = objBuildServer.BuildCatalogs 
Set objCatalog  = objCatalogs("KMSampleCatalog1")

Set objBldSvr = objCatalog.BuildServer

If objBuildServer is objBldSvr Then 
  Wscript.Echo "Identical IBuildServer interfaces"
Else
  Wscript.Echo "Error! The IBuildServer interfaces differ!"
End If

'Release objects 
Set objCatalogs = Nothing 
Set objBldSvr   = Nothing
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
