IPersistXML::LoadXML

The LoadXML method loads XML data from a BSTR.

Syntax

HRESULT LoadXML(
long Reserved,// in
BSTR bstrXML// in
);

Parameters

Reserved
This value is not used.
bstrXML
A BSTR that contains the XML data.

Include File

<pipeline.h>
<pipecomp.h>

Return Value

This method returns an HRESULT value.
