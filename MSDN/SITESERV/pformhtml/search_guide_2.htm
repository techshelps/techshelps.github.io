Using the Search Objects

This section provides an overview of programming with the Search objects. The following table lists the topics in this section and the interfaces from which the relevant objects are obtained.

Topic Interface
Configuring the Catalog Build Server IBuildServer
Managing Definitions and Building Catalogs IBuildCatalogs, IBuildCatalog
Configuring the Search Server ISearchServer
Managing Catalogs and Search Requests ISearchCatalogs, ISearchCatalog

If you create catalog defintions and catalogs by using Microsoft® Management Console MMC, Web Admin, or on the command line, they are included in the collection of catalog definitions and catalogs.

Using the Type Libraries

To get acquainted with Search objects, properties, and methods, programmers can peruse the Search type libraries by using the Microsoft Visual Basic® Object Browser or the Microsoft Visual C++® OLE/COM Object Viewer. The following type libraries are provided.

  • Site Server Search Administration 1.0 Type Library

  • Site Server Gathering Service Type Library

  • Site Server Gatherer Type Library

  • Site Server Indexer Type Library

For more information about the type libraries and the interfaces they describe, see Interface Type Information in the Search Programmer’s Reference Appendix.
