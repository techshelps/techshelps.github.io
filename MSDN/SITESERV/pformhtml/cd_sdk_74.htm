ReplicationItem

The ReplicationItem object provides access to individual replication items. The following table briefly describes the ReplicationItem object's read-only properties.

Property Description
Attributes Attributes of the item.
CreationTime When the item was created
LastAccessTime When the item was last accessed
LastModifiedTime When the item was last modified
Name The name of the item
SizeHigh The high size of the item
SizeLow The low size of the item
