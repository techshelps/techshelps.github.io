Page.URLArgs Method

The URLArgs method generates a series of query string arguments in an appropriate form for passing in a URL. The method applies URL encoding to the values passed in.

Syntax

Page.URLArgs(Arguments)

Parameters

Arguments
An optional list of name/value pairs.

Example

The following example:

Page.URLArgs("a", 1, "b", 2)

returns the string:

a=1&b=2

The following example (from Volcano Coffee’s Manager/xt_data_add_update.asp file) redirects the client browser to the page specified by the goto query string variable, and appends as query string arguments the product family ID and the attribute ID.

Response.redirect Cstr(Request.Form("goto")) & "?" & mscsPage.URLArgs("pfid", CStr(Request.Form("pfid")), "attribute_id", CStr(Request.Form("attribute_id")))
