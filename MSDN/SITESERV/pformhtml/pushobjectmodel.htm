Object Model

The primary hierarchy of objects implemented by the Active Channel Server COM classes is listed below.  The structure closely follows the Channel Definition (CDF) and Open Software Description (OSD) specifications on content relationships.  In each case, the parent object contains information only about its direct descendents.  A complete description of each object can be found by following the associated hyperlinks.
