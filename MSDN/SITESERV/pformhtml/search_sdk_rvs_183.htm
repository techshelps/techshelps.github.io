SearchServer Property

This read-only property provides access to the ISearchServer object of the host.

Syntax

ISearchAdmin.SearchServer

Example

The following example retrieves the ISearchServer object for the current host. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objSearchServer 

Set objSearchAdmin  = CreateObject("Search.SearchAdmin.1")
Set objSearchServer = objSearchAdmin.SearchServer

...

'Release objects 
Set objSearchServer = Nothing
Set objSearchAdmin  = Nothing
