Site.ConnectionStringMap Value

The ConnectionStringMap value contains a Dictionary object containing connection strings that are available to the site.

The connection string map provides an abstraction for database connection strings by mapping names to connection strings. The names are called connection names. The connection string map is saved in a Commerce Server site configuration as MSCSSite.ConnectionStringMap.

The connection string map can be created when a new Commerce Server site is created, using the Commerce Server Site Foundation Wizard.
