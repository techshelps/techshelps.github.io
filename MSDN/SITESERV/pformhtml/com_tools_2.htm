Starting the Pipeline Editor

Because the Win32®-based version of the Pipeline Editor is designed to be used locally, you start it from the Start menu. You start the Active Server Pages (ASP)-based Pipeline Editor from the browser by connecting to the site’s Manager page, and then clicking a button to launch the Pipeline Editor with the selected pipeline loaded.

To start the Win32-based pipeline editor
  1. On the Start menu, point to Programs, point to Microsoft Site Server, point to Commerce, and then click Pipeline Editor.
  2. On the File menu, click Open.
  3. Locate the .pcf file for the pipeline you want to edit. The default location for a site’s .pcf file is \Inetpub\wwwroot\sitename\Config.
  4. Select the .pcf file, and then click Open.
To start the ASP-based pipeline editor
  1. Start your browser and then type the URL of the Manager page of the site whose pipeline you want to edit.

    For example, to edit a pipeline for the Volcano Coffee site, you would type the following URL, substituting the name of the computer for servername: http://servername/vc30/manager/default.asp.

  2. In the System area on the site’s Manager page, select the pipeline you want to edit in the drop-down list, and then click Edit Pipeline.
