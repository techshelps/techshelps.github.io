Remove Method

This method removes a search server from the list of search servers to which a catalog definition is propagated.

Syntax

ICatalogSearchServerList.Remove(Server)

Parameters

Server

The server that is removed from the list.

Example

The following example removes the server KnowledgeServer1 from the list of servers to which the KMSampleCatalog1 catalog definition is propagated. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer, objCatalogs, objCatalog, objServers 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer
Set objCatalogs = objBuildServer.BuildCatalogs 
Set objCatalog  = objCatalogs("KMSampleCatalog1")
Set objServers  = objCatalog.SearchServers

objServers.Remove("KnowledgeServer1")

'Release objects
Set objServers  = Nothing 
Set objCatalog  = Nothing 
Set objCatalogs = Nothing 
Set objBuildServer = Nothing 
Set objSearchAdmin = Nothing
