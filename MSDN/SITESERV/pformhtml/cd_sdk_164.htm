DeleteRoute Method

This method deletes a route from the server’s routing table.

Syntax

ReplicationServer.DeleteRoute(Route)

Parameters

Route

The route to delete.

Remarks

You must have Site Server Publishing administrator privileges on the server to call this method.

Example

The following example deletes the Test route. 

Option Explicit 
On Error Resume Next 

const CRS_ERROR_NO_ROUTES    = 15110
const CRS_ERROR_ROUTE_NOT_FOUND  = 15109

dim ReplServer
set ReplServer = CreateObject("CrsApi.ReplicationServer")
ReplServer.Initialize("")

Err.Clear
ReplServer.DeleteRoute("Test")

if Err.Number = CRS_ERROR_NO_ROUTES then
  Wscript.Echo "No routes defined!"
elseif Err.Number = CRS_ERROR_ROUTE_NOT_FOUND then
  Wscript.Echo "Route Test not found!"
end if

'Release Server object
set ReplServer = Nothing
