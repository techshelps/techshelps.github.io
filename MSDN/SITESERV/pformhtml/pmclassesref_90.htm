MinTTL Property

This property defines the minimum time to live for dynamic objects.  It is in units of seconds.

IDL Definition

[propget] 
HRESULT MinTTL([out, retval] VARIANT *pData);
[propput] 
HRESULT MinTTL([in] VARIANT Data);

Type

VT_I4 (integer)
