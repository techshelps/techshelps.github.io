DependentType Property

This property, if defined, causes the construction of the <DEPENDENTTYPE VALUE="value"> child XML element upon invocation of the SoftPkg.GetCDFString method.

Type

String (VT_BSTR)

Default

None

Remarks

This type should be one of "ASSERT" or "INSTALL".

Example

SoftPkg.DependentType = "ASSERT"
