Datasource Object Overview

Note

The Datasource object is included in Commerce Server 3.0 only for backward compatibility with sites created in Commerce Server 2.0. New sites should be constructed using ADOs.

Although you can use the Datasource object to perform any operation that you need to perform on your site database, Commerce Server sites generally use this object in two contexts:

  • υTo execute queries that retrieve product information for display on a page.
  • υTo retrieve product information for storage in an OrderForm.
