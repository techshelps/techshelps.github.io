DeleteOnInstall Property

The DeleteOninstall property is a integer flag used when building the <SOFTPKG></SOFTPKG> element upon invocation of the GetCDFString method. If this property is 1, the <DELETEONINSTALL/> child element is generated.

Type

Integer (VT_I4)

Default

0 (false)

Example

Softpkg.DeleteOnInstall = 1
