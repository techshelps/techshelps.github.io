Content Deployment Programmer's Guide

Content Deployment enables replication of files, directories, and other components for these configurations:

  • From one directory to another on the same server.

  • From one server to another on an intranet, or on the Internet.

  • From one server to multiple servers on an intranet, or on the Internet.

There are three project types that specify the method of the replication: Content Deployment, Internet Retrieval, and Component Deployment.

These project types are fully described in the main Site Server documentation, as is how to run them by using Microsoft Media Console (MMC) or WebAdmin. Alternatively, developers can program Content Deployment objects to have the same functionality tailored to their needs. This Content Deployment SDK contains reference information for programming these objects in VBScript.
