Gatherer Service Related Values

These values all have the bits 0x0d00 in common.

0x80040d03 GTHR_E_UNABLE_TO_READ_REGISTRY Error reading registry value.
0x80040d04 GTHR_E_ERROR_WRITING_REGISTRY Unable to set value, the object is already deleted or not initialized properly.
0x80040d05 GTHR_E_ERROR_INITIALIZING_PERFMON Unable to initialize performance monitoring.
0x80040d06 GTHR_E_ERROR_OBJECT_NOT_FOUND Unable to find the specified object.
0x80040d07 GTHR_E_URL_EXCLUDED The specified URL is excluded and will not be cataloged.
0x80040d08 GTHR_E_CONFIG_DUP_CRAWL_ENTRY Duplicate crawl entry in registry.
0x80040d09 GTHR_E_CONFIG_DUP_NNTP_SERVER Duplicate NNTP server entry in registry.
0x80040d0a GTHR_E_CONFIG_DUP_PROJECT Duplicate project entry in registry.
0x80040d0b GTHR_E_CONFIG_DUP_EXTENSION Duplicate extension entry in registry.
0x80040d0c GTHR_E_CRAWL_TOO_DEEP The Gatherer attempted to follow a link deeper than the configured restrictions.

0x80040d0e GTHR_E_TOO_MANY_PLUGINS The Gatherer attempted to create more Gatherer plug-in objects than allowed.
0x80040d0f GTHR_E_INVALIDFUNCTION  Unable to call function in this context or the function is not implemented.

0x80040d14 GTHR_E_NAME_TOO_LONG The length of the name specified has exceeded allowed limits.

0x80040d2d GTHR_E_FILTER_MISMATCH Unable to filter the document, possibly caused by document type and extension mismatch.

0x80040d2f GTHR_E_FILTER_LOAD Unable to load the document info filter.
