IProtocol Interface

This interface contains the properties of a protocol that Search uses when crawling.

Hierarchy

ISearchAdmin.IBuildServer.IBuildCatalogs.IBuildCatalog.IProtocols

The following table briefly describes this interface's properties.

Property Description
Included Whether Search crawls this protocol building a catalog.
ProgIdHandler ProgID for the component that handles this protocol.
UrlName Protocol name.
