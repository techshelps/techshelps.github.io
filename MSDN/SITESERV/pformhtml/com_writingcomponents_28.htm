Billing Action Values

When a Billing Application Programming Interface (API) routine invokes the pipeline that you register, the routine identifies itself to the pipeline by writing a string representation of its name to the _billing_action name/value pair on the OrderForm. The following table contains the name/value pairs that Billing API routines write to this name/value pair, and the functions that these strings identify.

Value Function
BillableEvent ICPGenerateBillableEvent
CreditEvent ICPGenerateCreditEvent
AuthBillableEvent ICPBillAuthorizedUser
ExBillableEvent ICPGenerateBillableEventEx
ExAuthBillableEvent ICPBillAuthorizedUserEx
BillableUser ICPIsUserBillable
