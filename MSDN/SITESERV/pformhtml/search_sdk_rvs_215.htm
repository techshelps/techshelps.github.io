Accessing an ISearchServer Object

You access an ISearchServer object by calling the SearchAdmin.SearchServer property, as in the following example, where objSearchServer is the name you give to the new ISearchServer object: 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objSearchServer 

Set objSearchAdmin  = CreateObject("Search.SearchAdmin.1")
Set objSearchServer = objSearchAdmin.SearchServer

...

'Release objects 
Set objSearchServer = Nothing
Set objSearchAdmin  = Nothing
