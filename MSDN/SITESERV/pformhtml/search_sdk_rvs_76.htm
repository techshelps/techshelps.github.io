UserAgent Property

This property specifies the user agent used when crawling.

Syntax

IBuildServer.UserAgent [=Agent]

Parameters

Agent

The user agent.

Example

The following example displays the value of the user agent. 

Option Explicit 
On Error Resume Next

Dim objSearchAdmin, objBuildServer 

Set objSearchAdmin = CreateObject("Search.SearchAdmin.1")
Set objBuildServer = objSearchAdmin.BuildServer

Wscript.Echo "The user agent is: " & objBuildServer.UserAgent

'Release Objects
Set objBuildServer = Nothing
Set objSearchAdmin = Nothing
