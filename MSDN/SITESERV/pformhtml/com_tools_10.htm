Component Properties

You can view this property page only in the Win32-based Pipeline Editor.Use this property page to change the name of the component, to view the component’s Class ID or Program ID, and to save notes about the component.

  • υLabel. Specifies the name that the Pipeline Editor displays for this component. You can change the name by typing a new name here.
  • υClass ID. Specifies the globally unique identifier (GUID) that uniquely identifies this component on any computer. This ID does not change even if you change the component’s name.
  • υProgram ID. Specifies the easily recognizable ID for the component. This ID does not change even if you change the component’s name.
  • υDescription. Provides a place for you to type comments about this component.
