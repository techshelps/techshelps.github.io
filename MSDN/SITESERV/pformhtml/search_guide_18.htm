IGatherLogs Interface

Hierarchy

ISearchAdmin:IBuildServer:IBuildCatalogs:IBuildCatalog

This interface manages catalog build server logs (collection of IGatherLog interface objects) containing the names of the documents that Search successfully or unsuccessfully crawled, or avoided, while building a catalog.

You access an IGatherLogs interface object by calling the IBuildCatalog.Logs property.
