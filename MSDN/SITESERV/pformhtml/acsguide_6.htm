OSD Related COM Classes

The objects defined by these classes can be used to store content to be presented as OSD data to subscribers.  The classes are listed below, along with a brief description of each.  The OSD specification itself can be found at http://www.microsoft.com/standards.

SoftwarePackage Class

This class of object is the primary container in the OSD content hierarchy.  It can be used to manage data presentable using the SOFTPKG OSD XML element.

Implementation Class

This class of object can be used to manage data presentable using the IMPLEMENTATION OSD XML element.

Codebase Class

This class of object can be used to manage data presentable using the CODEBASE OSD XML element.
