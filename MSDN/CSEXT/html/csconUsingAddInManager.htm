Using Add-In Manager

Use Add-In Manager to select an add-in, load or unload it, and set the add-in's load behavior. You must first register an add-in before it is displayed in the Add-In Manager.

Loading or unloading an add-in

  1. Open Add-In Manager from the Tools menu.

  2. Select your add-in in the listbox. (Double-clicking toggles the Load/Unload setting).

  3. Select the Loaded/Unloaded option in the Load Behavior frame.

Setting an add-in's load behavior

  1. Open Add-In Manager.

  2. Select your add-in in the listbox.

  3. Select one of the following two options in the Load Behavior frame:
    • Choose Load on Startup to load the add-in automatically whenever the development environment is started from the desktop.

    • Choose Command Line to load the add-in whenever the environment is started from the DOS prompt or from a script.
