BuildDone Event

      

Occurs after building the solution.

Syntax

Event BuildDone (success As Boolean, updated As Boolean, cancelled As Boolean)

This syntax has these parts:

Part Description
success Boolean. True means the build succeeded, False means the build failed.
updated Boolean. True means build or deploy products were modified. False means build or deploy products were not modified.
cancelled Boolean. True means a user action cancelled the build. False means the build completed normally.

Remarks

Updated could be False while success is True, indicating that everything is up to date and no changes were made.
