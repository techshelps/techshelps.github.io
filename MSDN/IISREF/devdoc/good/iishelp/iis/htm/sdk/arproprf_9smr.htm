AccessSSLFlags

This single property contains the following Secure Socket Layer (SSL) file permission flags:

AccessSSL AccessSSLNegotiateCert
AccessSSL128 AccessSSLRequireCert
AccessSSLMapCert

The following table describes this property:

Data type Long
Default value Zero (no SSL permissions)
Default inheritance Inheritable

Accessible by IIS Admin Objects:

IIsWebService IIsWebServer IIsWebVirtualDir IIsWebDirectory
IIsWebFile

See Also

MD_SSL_ACCESS_PERM for hexadecimal values when setting flags in AccessSSLFlags.
