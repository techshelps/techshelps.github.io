DefaultDocFooter

This value specifies the custom footer that is appended to files that are returned to the client. The custom footer will be sent only if the EnableDocFooter property is True. This can be a string such as "This is a custom footer," or a reference to a file such as c:\wwwroot\footers\footer.htm.

The following table describes this property:

Data type String
Default value Empty string
Default inheritance Inheritable

Accessible by IIS Admin Objects:

IIsWebService IIsWebServer IIsWebVirtualDir IIsWebDirectory
IIsWebFile
