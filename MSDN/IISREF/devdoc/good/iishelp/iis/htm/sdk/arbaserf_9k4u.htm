MD_DEFAULT_LOGON_DOMAIN

This identifier specifies the default domain to which the user will log on, if no domain is specified by the user. This value is valid for Basic authentication only.

Attributes

Data type STRING
Default inheritance Inheritable
User type IIS_MD_UT_FILE

Values

Default Value
Empty string

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/MSFTPSVC IIsFtpService
/LM/MSFTPSVC/n IIsFtpServer
/LM/W3SVC IIsWebService
/LM/W3SVC/n IIsWebServer
/LM/W3SVC/n/ROOT IIsWebVirtualDir
/LM/W3SVC/n/ROOT/WebVirtualDir IIsWebVirtualDir
/LM/W3SVC/n/ROOT/WebVirtualDir/WebDirectory IIsWebDirectory
/LM/W3SVC/n/ROOT/WebVirtualDir/WebDirectory/WebFile IIsWebFile

Remarks

If the value of this identifier is an empty string, and the server is a domain controller, the server will use its own domain name as the default name. If the server is not a domain controller, the computer name of the server machine is used as a domain name.
