MD_MAX_CLIENTS_MESSAGE

This identifier contains the string that is sent to all clients that attempt to connect to the service or server once the service or server's maximum number of connections, specified in MD_MAX_CONNECTIONS, has been reached.

Attributes

Data type STRING
Default inheritance Inheritable
User type IIS_MD_UT_SERVER

Values

Default Value
Empty string

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/MSFTPSVC IIsFtpService
/LM/MSFTPSVC/n IIsFtpServer

See Also

MaxClientsMessage, MD_MAX_CONNECTIONS
