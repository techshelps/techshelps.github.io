IMSAdminBase Structures

The following structures are used by the IIS Admin Base Object methods.

MD_CHANGE_OBJECT Stores information about changes made to a key in the metabase.
METADATA_GETALL_RECORD Member of an array that receives multiple metabase entries.
METADATA_HANDLE_INFO Stores information about a handle to the metabase.
METADATA_RECORD Specifies or receives a single metabase data entry.
