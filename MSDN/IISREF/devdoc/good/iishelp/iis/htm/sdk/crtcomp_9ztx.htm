IVariantDictionary Interface

You can use the IVariantDictionary interface to retrieve values from the Application Contents, Application StaticObjects, Session Contents and Session StaticObjects collections.

Syntax

public interface asp.IVariantDictionary extends com.ms.com.IUnknown
{
    public abstract com.ms.com.Variant getItem(com.ms.com.Variant);
    public abstract void putItem(com.ms.com.Variant, com.ms.com.Variant);
    public abstract com.ms.com.IUnknown get_NewEnum();
    public abstract com.ms.com.Variant getKey(com.ms.com.Variant);
    public abstract int getCount();
}
