IInetLogInformation::GetParameters

The IInetLogInformation::GetParameters method retrieves the parameters related to the current operation.

virtual LPSTR STDMETHODCALLTYPE GetParameters(
  IN PCHAR pszParameters,  
  IN PDWORD pcbSize        
);

Parameters

pszParameters
[in] The parameter values to be retrieved.
pcbsize
[in] Size of the pszParameters parameter.
