ServerBindings

This property specifies a string that is used by IIS to determine what network endpoints are used by the server instance. The string format is IP:Port:Hostname. Both the IP and Hostname parameter of the string are optional. Any unspecified parameters default to an all-inclusive wildcard.

The following table describes this property:

Data type List (String)
Default value :80: for Web service,

:21: for FTP service
Default inheritance Inheritable

Accessible by IIS Admin Objects:

IIsFtpService IIsFtpServer IIsWebService IIsWebServer
