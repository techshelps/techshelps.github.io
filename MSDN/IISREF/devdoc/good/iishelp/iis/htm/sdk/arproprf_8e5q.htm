NetLogonWorkstation

This property specifies Microsoft® Windows NT® Workstation logon restrictions from a primary domain controller. The Windows NT Workstation logon field is empty if this value is 0, contains the Internet Protocol (IP) address if the value is 1, and contains the Domain Name System (DNS) name if the value is 2.

The following table describes this property:

Data type Long
Default value 0 (Workstation restrictions only)
Default inheritance Inheritable

Accessible by IIS Admin Objects:

IIsWebService IIsWebServer
