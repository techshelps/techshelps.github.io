LogFilePeriod

This property specifies how often Microsoft® Windows NT® should create a new log file. If this property is set to 0, a new file will be created when the maximum size designated in LogFileTruncateSize is reached. If this property is set at 1, 2, or 3, a new log file will be created daily, weekly, or monthly, respectively.

The following table describes this property:

Data type Long
Default value 1 (Create new log file daily)
Default inheritance Inheritable

Accessible by IIS Admin Objects:

IIsFtpService IIsFtpServer IIsWebService IIsWebServer
