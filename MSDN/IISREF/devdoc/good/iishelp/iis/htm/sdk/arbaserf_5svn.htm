MD_IISADMIN_EXTENSIONS

This identifier contains a list of CLSIDs for all registered administration extensions installed on the server.

Attributes

Data type MULTI_SZ
Default inheritance Not inheritable
User type IIS_MD_UT_SERVER

Values

Default Value
Not applicable

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/IIsAdmin/Extensions/DCOMCLSIDS Not applicable

Remarks

This identifier should be considered read-only.
