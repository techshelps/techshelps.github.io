Web Application Management Interface

You can manage your Web applications by creating custom C++ programs that use the IWamAdmin interface described in this section, or by using ASP scripts with the IIS Admin Objects, IIsWebVirtualDir and IIsWebDirectory.

This section includes:

  • About Web Application Management explains the overall process of Web application management using the IWamAdmin interface.
  • Using Web Application Management provides details on how you use the IWamAdmin interface to manage your Web applications.
  • IWamAdmin Interface Reference provides the syntax and details of the methods of the IWamAdmin interface.
