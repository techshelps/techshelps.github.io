Developer Samples

Welcome to the Developer Samples. This section is designed to expose you to the power and extensibility of IIS, one step at a time, by using clear and simple examples. ASP scripts, ISAPI extensions and filters, and Active Server components, both installable and custom, are included in the samples provided.

These samples have been organized by type and placed in the directories specified below. For each script sample, you will find an HTML page that provides an overview of the sample, as well as the script and any supporting files, such as database or text files. Each ActiveX™ and ISAPI project is placed in its own subdirectory, also with an HTML overview page.

For more information, see %mssdk%\Samples\Internet\IIS\Readme.htm.
