Administration Property Reference

Administration properties are associated with key values in the metabase. This reference provides a description for each property, including the data type, default value, default inheritance attributes, and usage notes. For each property, the IIS Admin Objects are listed that can access that property..

The Alphabetical Property List provides quick access to individual property descriptions.

The Property Inheritance Charts provide easy-to-use tables that relate each individual property with the Admin Objects that have access to it.

  • FTP Property Inheritance chart showing properties that apply to FTP services.
  • Web Property Inheritance chart showing properties that apply to Web services.
  • Miscellaneous Property Inheritance chart showing properties that apply to objects other than FTP or Web services.
