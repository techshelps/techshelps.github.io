MD_ANONYMOUS_PWD

This identifier specifies the anonymous user password associated with the account specified by MD_ANONYMOUS_USER_NAME. This password is only required if the identifier MD_ANONYMOUS_USE_SUBAUTH is set to FALSE.

The password is stored in an encrypted form in the metabase. It is unencrypted locally, so the value of this identifier, when queried, will be the unencrypted version of the password.

Attributes

Data type STRING
Default inheritance Inheritable
User type IIS_MD_UT_FILE

Values

Default Value
Random string

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/MSFTPSVC IIsFtpService
/LM/MSFTPSVC/n IIsFtpServer
/LM/W3SVC IIsWebService
/LM/W3SVC/n IIsWebServer

/LM/W3SVC/n/ROOT/WebVirtualDir IIsWebVirtualDir
/LM/W3SVC/n/ROOT/WebVirtualDir/WebDirectory IIsWebDirectory
/LM/W3SVC/n/ROOT/WebVirtualDir/WebDirectory/WebFile IIsWebFile
