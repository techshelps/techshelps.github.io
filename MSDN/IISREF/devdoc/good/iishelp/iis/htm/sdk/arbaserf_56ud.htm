MD_KEY_TYPE

This identifier specifies the type of a metabase key, and the IIS Admin Object that relates to that key. This identifier can be used to specify the key type, such as IIsWebServer or IIsWebVirtualDir, when creating or deleting IIS Admin Objects.

Attributes

Data type STRING
Default inheritance Not inheritable
User type IIS_MD_UT_SERVER

Values

Default Value
Not applicable
