IIS Metabase

The metabase stores your IIS configuration parameter values in a fast-access, memory-resident data store. The metabase is specifically designed for use with IIS and is faster, more flexible, and more expandable than the Windows NT® registry. You can use the IIS Admin Objects with Automation and scripting languages to administer your IIS installation by setting IIS configuration parameter values in the metabase.

Details of the IIS metabase are presented in the following sections:

  • Structure details the organization of data in the metabase.
  • Key Names and Paths describes how metabase keys are addressed.
  • Property Inheritance describes a metabase feature that simplifies configuration operations and reduces storage requirements.
  • Performance addresses optimization and locking strategies for the metabase.
  • Security provides strategy and implementation details for protecting and backing up the metabase.

For custom use and extension of the metabase with C++ programs, see IIS Admin Base Object.
