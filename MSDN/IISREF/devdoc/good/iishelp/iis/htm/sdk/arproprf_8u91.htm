AspCodepage

This property determines the default codepage for an application. The Web administrator can designate that a range of the namespace is written in a specific language and prevent individual ASP pages in an application from specifying different languages.

The following table describes this property:

Data type Long
Default value 0 (CP_ACP, System ANSI codepage)
Default inheritance Inheritable

Accessible by IIS Admin Objects:

IIsWebService IIsWebServer IIsWebVirtualDir IIsWebDirectory
