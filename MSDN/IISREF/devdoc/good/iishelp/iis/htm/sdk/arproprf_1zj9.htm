Miscellaneous Property Inheritance

This chart shows the properties associated with IIS Admin Objects other than those associated with FTP or Web services. For each property you can see the default setting of its inheritance attribute and the IIS Admin Objects you can use to access the value of the property in the metabase. The inheritance is indicated as:

Inh Inherits
Inh Contained in another inherited property
Inh* Contains more than one property
NoAtt Does not inherit
