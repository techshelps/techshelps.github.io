MD_ADV_CACHE_TTL

This identifier specifies the number of seconds that a password expiration notice will be cached. This has the effect of limiting the frequency at which users will be notified that their password is about to expire. For example, if the value for this identifier is set at its default, ten minutes, then a user whose password is close to expiring will be notified at most every ten minutes.

Attributes

Data type DWORD
Default inheritance Inheritable
User type IIS_MD_UT_SERVER

Values

Value Range
0 – 0xFFFFFFFF
Default Value
600 (10 minutes)

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/W3SVC IIsWebService
/LM/W3SVC/n IIsWebServer
