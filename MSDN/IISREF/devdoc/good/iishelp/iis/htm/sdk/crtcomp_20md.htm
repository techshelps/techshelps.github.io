Built-in Object Reference

You can use the Active Server Pages (ASP) built-in objects to provide a connection between your component and clients accessing them through IIS. The way you access the built-in objects from a component is determined by your development environment. This section defines the built-in objects from the perspective of three different languages: C++, Java, and Microsoft® Visual Basic®. ASP scripts always access the built-in objects through the Visual Basic Object Model. These views are provided in:

  • C++ Interfaces
  • Java Class and Interface Definitions
  • Visual Basic Object Model
