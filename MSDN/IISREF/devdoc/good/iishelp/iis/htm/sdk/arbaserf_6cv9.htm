MD_GREETING_MESSAGE

This identifier specifies a string that will be sent to the FTP client at the beginning of every FTP session.

Attributes

Data type MULTI_SZ
Default inheritance Inheritable
User type IIS_MD_UT_SERVER

Values

Default Value
Empty list

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/MSFTPSVC IIsFtpService
/LM/MSFTPSVC/n IIsFtpServer
