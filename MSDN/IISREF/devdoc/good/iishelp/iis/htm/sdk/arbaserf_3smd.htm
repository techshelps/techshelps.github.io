MD_ALLOW_REPLACE_ON_RENAME

This identifier indicates whether the rename command, when sent to the FTP server by a client, should replace the existing file with the newly renamed file if a name conflict exists.

Attributes

Data type DWORD
Default inheritance Inheritable
User type IIS_MD_UT_SERVER

Values

Value Range
TRUE | FALSE
Default Value
TRUE

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/MSFTPSVC IIsFtpService
/LM/MSFTPSVC/n IIsFtpServer
