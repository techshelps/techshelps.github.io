About the IIS Admin Base Object

You can use the IIS Admin Base Object to write applications, such as server administration or Web authoring tools, that check and update the server's configuration by manipulating keys and data in the metabase. You can also use the IIS Admin Base Object to store your IIS-related custom application configuration data in the fast-access metabase without filling up the system registry.

This section contains:

  • Keys and Data Entries, which describes how data is organized into hierarchical keys.
  • Handles to Keys, which discusses the use of handles to access keys.
  • Inheritance, which details a feature that minimizes data storage and increases the efficiency of your programs.
  • Reserved Identifiers and User Types, which explains how data is identified and categorized.

Remarks
