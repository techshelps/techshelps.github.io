MD_FRONTPAGE_WEB

This identifier specifies whether this server instance is being managed by Front Page Extensions. If the value is set to TRUE, Front Page Manager will create all necessary management files. If the value is set to FALSE, all management files for that server instance are deleted.

Attributes

Data type DWORD
Default inheritance Not inheritable
User type IIS_MD_UT_SERVER

Values

Value Range
TRUE | FALSE
Default Value
FALSE

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/W3SVC/n IIsWebServer
