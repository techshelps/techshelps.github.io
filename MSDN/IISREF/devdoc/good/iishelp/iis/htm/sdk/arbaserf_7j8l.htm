MD_EXIT_MESSAGE

This identifier specifies the string that is sent by the server to an FTP client after the quit command is received from the client.

Attributes

Data type STRING
Default inheritance Inheritable
User type IIS_MD_UT_SERVER

Values

Default Value
Empty string

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/MSFTPSVC IIsFtpService
/LM/MSFTPSVC/n IIsFtpServer
