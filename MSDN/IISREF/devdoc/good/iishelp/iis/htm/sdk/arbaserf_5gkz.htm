MD_LOGSQL_DATA_SOURCES

This identifier specifies the name of the ODBC data source to which Microsoft® Windows NT® will write data during logging events.

Attributes

Data type STRING
Default inheritance Inheritable
User type IIS_MD_UT_SERVER

Values

Default Value
HTTPLOG

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/MSFTPSVC IIsFtpService
/LM/MSFTPSVC/n IIsFtpServer
/LM/W3SVC IIsWebService
/LM/W3SVC/n IIsWebServer

Remarks

The logging module's CLSID determines whether Microsoft® Windows NT® will log to an ODBC data source, or to the file system.
