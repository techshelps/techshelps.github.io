AppDeleteRecursive

You can use the AppDeleteRecursive method of the IIsWebDirectory object to delete Web application definitions from a metabase key and all subordinate keys. If the applications are running, they will be shut down. All resources associated with the applications, such as out-of-process packages, will be released. This action cannot be undone.

Syntax

IIsWebDirectory.AppDeleteRecursive

Code Example

<% 
Dim DirObj 
Set DirObj = GetObject("IIS://LocalHost/W3SVC/1/ROOT/MyAppDir") 
' Delete the application at this directory 
' and all subordinate directories 
DirObj.AppDeleteRecursive 
%>
