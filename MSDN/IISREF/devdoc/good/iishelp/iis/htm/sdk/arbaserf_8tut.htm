MD_ASP_SCRIPTERRORMESSAGE

This identifier specifies a generic error message string that is sent to the client browser if the server encounters an error in an ASP script, and the identifer MD_ASP_SCRIPTERRORSSENTTOBROWSER is set to FALSE.

Attributes

Data type STRING
Default inheritance Inheritable
User type IIS_MD_UT_WAM

Values

Default Value
An error occurred on the server when processing the URL. Please contact the system administrator.

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/W3SVC IIsWebService
/LM/W3SVC/n IIsWebServer
/LM/W3SVC/n/ROOT IIsWebVirtualDir
/LM/W3SVC/n/ROOT/WebVirtualDir IIsWebVirtualDir
/LM/W3SVC/n/ROOT/WebVirtualDir/WebDirectory IIsWebDirectory
