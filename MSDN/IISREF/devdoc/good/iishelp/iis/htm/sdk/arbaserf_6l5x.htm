MD_ANONYMOUS_USER_NAME

This identifier specifies the user name that will be associated with all actions performed by an anonymous user. If the identifier MD_ANONYMOUS_USE_SUBAUTH is set to FALSE, then the MD_ANONYMOUS_PWD identifier must be set to the password associated with this account.

Attributes

Data type STRING
Default inheritance Inheritable
User type IIS_MD_UT_FILE

Values

Default Value
I_USR_Local machine name

Metabase Keys
