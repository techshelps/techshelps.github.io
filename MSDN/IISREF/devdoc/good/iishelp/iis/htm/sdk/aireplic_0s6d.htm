Replication Extension Interface

IIS includes a replication feature that enables you to replicate configuration information from one computer to other computers. You can develop custom interfaces to extend the replication process for your own needs. You can use replication in a Web farm, or in a clustered computer environment using Microsoft® Windows NT ® Server 4.0 Enterprise Edition. In either situation, replication helps you synchronize configurations among computers.

This section includes:

  • About Replication Extensions, which provides an overview of replication and custom replication extensions.
  • Using the Replication Extension Interface, which explains the replication process and how your custom interface interacts with it.
  • IMSAdminReplication Interface Reference, which provides details of the methods you implement in your custom replication extension interface.
