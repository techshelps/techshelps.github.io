AccessFlags

This single property contains the following file access permission flags:

AccessExecute AccessNoRemoteWrite
AccessNoRemoteExecute AccessRead
AccessNoRemoteRead AccessScript
AccessNoRemoteScript AccessWrite

All permissions are applicable to Web service, servers, directories, and files. AccessRead and AccessWrite are the only permissions applicable to FTP service, servers, and directories.

The following table describes this property:

Data type Long
Default value AccessRead = True
Default inheritance Inheritable

Accessible by IIS Admin Objects:

IIsFtpService IIsFtpServer IIsFtpVirtualDir IIsWebService
IIsWebServer IIsWebVirtualDir IIsWebDirectory IIsWebFile

See Also

MD_ACCESS_PERM for hexadecimal values when setting flags in AccessFlags.
