DontLog

This property specifies whether client requests should be written to the log file. By default (False), requests are written to the file. You can set this value to True to turn logging off.

The following table describes this property:

Data type Boolean
Default value False
Default inheritance Inheritable

Accessible by IIS Admin Objects:

IIsFtpService IIsFtpServer IIsFtpVirtualDir IIsWebService
IIsWebServer IIsWebVirtualDir IIsWebDirectory IIsWebFile
