MD_MIME_MAP

This identifier contains a list of Multipurpose Internet Mail Extensions (MIME) file extension associations. This information is used by the computer to determine what applications are capable of processing or viewing a given document type. Each string in the list specifies the MIME type, and the file extension that is associated with that type. The string format for each entry is MIMEType, Extension.

Attributes

Data type MULTI_SZ
Default inheritance Inheritable
User type IIS_MD_UT_FILE

Values

Default Value
Not applicable

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM IIsComputer
/LM/MimeMap IIsMimeMap
/LM/W3SVC IIsWebService
/LM/W3SVC/n IIsWebServer
/LM/W3SVC/n/ROOT IIsWebVirtualDir
/LM/W3SVC/n/ROOT/WebVirtualDir IIsWebVirtualDir
/LM/W3SVC/n/ROOT/WebVirtualDir/WebDirectory IIsWebDirectory
/LM/W3SVC/n/ROOT/WebVirtualDir/WebDirectory/WebFile IIsWebFile

Remarks

The /LM/MimeMap key establishes the default values for the machine. These default values can be overriden at the service or server level.
