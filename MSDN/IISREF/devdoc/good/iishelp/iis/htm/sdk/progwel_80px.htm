Programmer's Reference

Welcome to the IIS Programmer's Reference. This section provides detailed information about special event methods and interfaces available for components running on a Web server that supports Active Server Pages (ASP), the storage of your custom IIS configuration data and how you use built-in objects to manipulate that data, creating and maintaining logging modules, and a set of interfaces that you can use to create HTTP server extensions.

This section contains:

  • Creating Components for ASP, which provides overviews and object reference material to help you develop server components for use with ASP.
  • IIS Administration, which provides overviews and reference material to help you manage the IIS configuration programmatically.
  • Logging Interface, which provides overviews and reference material to help you generate server activity logs without interfering with other server processes.
  • ISAPI, which provides overviews and reference material to help you create extensions and filters for HTTP servers.
