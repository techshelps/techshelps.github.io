IServer Interface

The IServer interface enables your component to use the methods and properties of the Server object.

Syntax

public interface asp.IServer 
    extends com.ms.com.IUnknown
{
           //Methods
    public abstract java.lang.Object CreateObject(java.lang.String);
    public abstract java.lang.String MapPath(java.lang.String);
    public abstract java.lang.String HTMLEncode(java.lang.String);
    public abstract java.lang.String URLEncode(java.lang.String);
    public abstract void putScriptTimeout(int);
    public abstract int getScriptTimeout();
}
