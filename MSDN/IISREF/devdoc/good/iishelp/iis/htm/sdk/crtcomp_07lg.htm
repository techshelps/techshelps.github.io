Developing and Deploying Your Component

Once you have a plan for developing your component, you will need to implement this plan using an appropriate development environment. Although you can develop components using any environment that is capable of creating COM components, there are some minor discrepancies in how the various environments view COM components. This section will explain how to implement your component design in the C++, Microsoft® Visual Basic®, and Java development environments. It is divided into five areas:

  • Implementing Your Component
  • Deploying Your Component
  • Accessing the ASP Built-in Objects
  • Application Isolation Issues
  • Security Issues
