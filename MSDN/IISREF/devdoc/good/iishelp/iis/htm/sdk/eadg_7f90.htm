Advanced Web Application Development

This guide provides an overview of Web application design, and introduces the more advanced topics of security, data access, and Web-based transactions.

Note  The information contained in this section is excerpted from the forthcoming Internet Information Server Resource Kit, to be published by Microsoft Press.

This section contains:

  • Web Applications: An Overview: An introduction to the IIS programming model.
  • Security Ramifications for IIS Applications: Choosing the right access control to prevent unauthorized access to your site.
  • Accessing Data with Active Server Pages (ASP): An introduction to data access from a Web page with Microsoft® ActiveX™ Data Objects (ADO).
  • Debugging Your Application: Troubleshooting tips and tricks for ISAPI, and component developers.
