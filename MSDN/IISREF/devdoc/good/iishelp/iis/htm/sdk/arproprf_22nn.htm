Property Inheritance Charts

These charts provide a quick visual map of properties to IIS Admin Objects, and the inheritance attribute of the property. An X in a cell shows which objects can access a specific property. There are three charts.

  • The FTP Property Inheritance chart shows properties that apply to FTP services.
  • The Web Property Inheritance chart shows properties that apply to Web services.
  • The Miscellaneous Property Inheritance chart shows properties that apply to objects other than FTP or Web services.
