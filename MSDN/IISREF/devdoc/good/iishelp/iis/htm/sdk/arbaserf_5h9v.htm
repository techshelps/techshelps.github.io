MD_ALLOW_ANONYMOUS

This identifier enables anonymous FTP access to FTP server.

Attributes

Data type DWORD
Default inheritance Inheritable
User type IIS_MD_UT_SERVER

Values

Value Range
TRUE | FALSE
Default Value
TRUE

Metabase Keys

This identifier is available at the following metabase keys:

Metabase Path Key Type
/LM/MSFTPSVC IIsFtpService
/LM/MSFTPSVC/n IIsFtpServer

See Also

AllowAnonymous, MD_ANONYMOUS_ONLY
