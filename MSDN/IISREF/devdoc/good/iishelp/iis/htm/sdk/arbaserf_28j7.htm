MD_SECURE_BINDINGS

This identifier specifies a string that is used by IIS to determine what secure network endpoints are used by the server instance. The string format is IP:Port.

Attributes

Data type MULTI_SZ
Default inheritance Inheritable
User type IIS_MD_UT_SERVER

Values

Default Value
Empty string if no keys installed

:443: if keys installed

Metabase Keys

This identifier is available at the following metabase key:

Metabase Path Key Type
/LM/W3SVC/n IIsWebServer

Remarks

The IP component of each string is optional. If it is not provided, IIS assumes that any IP address is allowable (wildcard).
