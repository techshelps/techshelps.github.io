ISAPI

Internet Server Application Programming Interface (ISAPI) provides a set of interfaces that you can use to create extensions and filters for Microsoft® Internet Information Server (IIS). This document describes the implementation requirements of IIS. It provides general overview material on how to develop ISAPI extensions and filters, and reference information for the functions and structures available to you. This section contains the following broad categories:

  • ISAPI and the Web Application Architecture
  • Developing ISAPI extensions
  • Extension Reference
  • Developing ISAPI filters
  • Filter Reference
