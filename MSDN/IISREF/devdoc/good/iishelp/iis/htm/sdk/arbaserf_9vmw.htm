MD_ANONYMOUS_USE_SUBAUTH

This identifier indicates whether the sub-authenticator should be used for users attempting to access resources anonymously. If the value is FALSE, the administrator must set the password, using the MD_ANONYMOUS_PWD identifier, for anonymous access to function properly. If the value is TRUE, no anonymous user password is required.

Attributes

Data type DWORD
Default inheritance Inheritable
User type IIS_MD_UT_FILE

Values

Value Range
TRUE | FALSE
Default Value
TRUE
