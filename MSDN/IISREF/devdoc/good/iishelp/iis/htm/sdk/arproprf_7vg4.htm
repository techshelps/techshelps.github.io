AccessSSLRequireCert

When set to True, Secure Socket Layer (SSL) file access processing will request a certificate from the client. If no certificate is provided by the client, the connection will be closed. AccessSSLNegotiateCert must also be set to True when using AccessSSLRequireCert. This property is one of the SSL file permission flags contained in the AccessSSLFlags property.

The following table describes this property:

Data type Boolean
Default value False
Default inheritance Inheritable

Accessible by IIS Admin Objects:

IIsWebService IIsWebServer IIsWebVirtualDir IIsWebDirectory
IIsWebFile
