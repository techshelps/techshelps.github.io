Open Software Description Reference

This Open Software Description (OSD) reference contains all the elements that are defined by the Osd.dtd file. These elements can be used in OSD and Channel Definition Format (CDF) files to describe software. There are additional elements described in Microsoft Internet Component Download Reference that can be used to describe software in more detail and enhance information provided by the OSD elements.
