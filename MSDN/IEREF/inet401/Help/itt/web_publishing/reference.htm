Web Publishing Programmer's Reference

This section contains detailed descriptions of the interface methods and code examples for the Microsoft® Web Publishing Software Development Kit (SDK.) The examples in the Programmer's Reference are not complete samples and often omit most error processing.

Please read the Web Publishing Overview first for overview and general information about the Web Publishing SDK.

Note The Web Publishing interfaces (including the File Upload Control interface) cannot be used from a UI-less environment. For instance, if a pop-up dialog is created during a server-side run, the client never sees the dialog on its end and the process cannot complete.
