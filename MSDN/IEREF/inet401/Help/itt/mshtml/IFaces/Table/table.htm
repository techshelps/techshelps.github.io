IHTMLTable


This interface is used to specify that the contained content is organized into a table with rows and columns.

Valid tags within a table include:

  • TR
  • CAPTION
  • THEAD
  • TBODY
  • TFOOT
  • COLGROUP
  • COL

This element is a block element.
