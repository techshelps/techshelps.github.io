?XML

<?XML
 ENCODING="char-set"
 VERSION="n.n"
?>

Indicates an Extensible Markup Language (XML) document and specifies the version and character set for the current document. This tag should occur only once at the beginning of the CDF file.

ENCODING="char-set"
Character set used in the document. The default is "UTF-8". Additional information on character encoding and character sets is available in the CDF Parsing Rules and Hints section.
VERSION="n.n"
Major and minor version of XML the CDF file supports. If not specified, the default value is "1.0".

Number of allowed occurrences: One
Parent elements Child elements
None None

Example 

<?XML ENCODING="UTF-8" VERSION="1.0"?>
