About the Samples DHTML Overview Folder
----------------------------------------
This folder contains working samples that are specific to the Dynamic HTML overview
documents discussed throughout the Dynamic HTML section of the Internet Client SDK.


It is recommended that you view these samples as follows:

1)  From the Start Menu, launch the Internet Client SDK.
2)  Go to the Dynamic HTML section of SDK.
3)  Throughout each of the Dynamic HTML topics, "Show Me" buttons will be displayed.
    Each "Show Me" button demonstrates a working sample of the described feature
    of Dynamic HTML.
