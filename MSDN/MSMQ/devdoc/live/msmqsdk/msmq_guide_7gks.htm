Opening a Machine Journal

To read messages in a machine journal (you cannot send messages to a machine journal), you must specify the computer where the journal resides. When opening the machine journal, the format name of the queue should appear as follows:

MACHINE=MachineGUID;JOURNAL

Note  You cannot open a machine journal using a direct format name.
