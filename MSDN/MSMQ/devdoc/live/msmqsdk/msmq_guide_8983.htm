System Queues

System queues include journal, dead letter, and report queues. MSMQ or the MSMQ Administrator creates these queues, and only MSMQ can send messages to them. Applications have read-only access to messages from these queues.

Journal and dead letter queues are used to store copies of application messages. Report queues contain report messages that track the path of a specific application message.
