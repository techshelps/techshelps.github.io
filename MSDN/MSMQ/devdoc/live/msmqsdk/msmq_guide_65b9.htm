About the MSMQ Guide

The Microsoft® Message Queue Server (MSMQ) Guide outlines conceptual information on queues that can be used, messages that are available, how queues and messages are defined by properties, and information on other MSMQ services.

The following topics are in the MSMQ Guide:

  • About MSMQ
  • MSMQ Objects
  • MSMQ Queues
  • MSMQ Messages
  • MSMQ Computers
  • MSMQ Object Properties
  • MSMQ Transactions
  • Error Reporting
  • MSMQ Offline Support
  • MSMQ Security Services
  • MSMQ Connector Server
  • MSMQ Mail Services
  • MSMQ ActiveX Support
