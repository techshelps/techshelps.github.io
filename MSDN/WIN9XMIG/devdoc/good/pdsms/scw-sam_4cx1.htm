Requirements If You Use a Response File

[This is preliminary documentation and subject to change.]

If your Setup program uses a response file during its unattended mode, it must:

  • Create a master copy (template) of this file that will be used for all installations. It exists on the server as the response file used for all client installations of your application.
  • Within this master copy, use SCW variable names for fields that must be substituted during installation.

    For example, HomeDir=%SBSServer%

Note At present, %SBSServer% is the only SCW variable name defined. Future versions of SCW will provide others.
