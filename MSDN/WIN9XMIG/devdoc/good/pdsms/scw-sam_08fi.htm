Requirements for Your Application

[This is preliminary documentation and subject to change.]

To make use of SCW integration functionality, your application Setup program must comply with the following requirements:

  • It must provide an unattended or automated installation mode. This enables Small Business Server to automate all application installations on Small Business Server clients.
  • The option to eliminate prompt for reboot. Since more than one application will likely be installed on the client workstation during Setup, SCW postpones machine reboot until after all queued applications have been installed.
