Migration DLL Function Reference

[This is preliminary documentation and subject to change.]

You call the MIGRATE.DLL functions in order as you migrate your application from Windows® 95 and Windows 98 to Windows NT®.

  • QueryVersion
  • Initialize9x
  • MigrateUser9x
  • MigrateSystem9x
  • InitializeNT
  • MigrateUserNT
  • MigrateSystemNT
