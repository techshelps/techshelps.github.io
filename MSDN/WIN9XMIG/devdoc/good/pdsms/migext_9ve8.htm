Run-Time DLLs Included with Windows NT 5.0

[This is preliminary documentation and subject to change.]

Before running any migration extensions on the Windows 95 or Windows 98 system, Setup will copy the following files from the Windows NT compact disc to the Windows 95 or Windows 98 System directory.

SETUPAPI.DLL
Setup
MSVCRT.DLL
Run-time shipped with Microsoft® Visual C++® 5.0
MSVCRT40.DLL
Run-time shipped with Microsoft Visual C++ 4.0
MFC40.DLL
Microsoft Foundation Class Library 4.0
MFC42.DLL
Microsoft Foundation Class Library 4.2

For international environments, Setup copies the appropriate localized DLLs. For example, Setup copies MFC40JPN.DLL and MFC42JPN.DLL for a Japanese installation.

Debug versions of the run-time DLLs are not copied, nor can Setup guarantee that other run-time versions of an application will migrate correctly.

This list of files is not final. Setup may also install newer versions if they become available. However, if your migration extension needs a run-time DLL that is not installed by Windows NT, you must ship it with your migration DLL, and use the migration DLLs functionality to ensure the run-time DLL is installed.
