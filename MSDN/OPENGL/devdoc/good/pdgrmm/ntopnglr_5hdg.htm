POINTFLOAT

The POINTFLOAT structure contains the x and y coordinates of a point.

typedef struct _POINTFLOAT { // ptf   
    FLOAT      x; 
    FLOAT      y; 
} POINTFLOAT;

Members

x
Specifies the horizontal (x) coordinate of a point.
y
Specifies the vertical (y) coordinate of a point.
