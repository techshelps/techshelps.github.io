COM Object Reference


This reference defines the COM-based objects for Microsoft® NetMeeting™ 2.1, including the interfaces and methods through which a client application makes use of object instances.

The NetMeeting COM-based objects defined in this documentation enable developers to incorporate the NetMeeting API attributes of their choice into their own applications and providing any portion of the NetMeeting communication functionality to their users.
