Internet Locator Server Reference


This reference defines the COM-based Internet Locator Server (ILS) objects, formerly known as User Location Service (ULS). The objects enable your application to communicate with other applications over the Internet, similar to the directory service support provided within Microsoft® NetMeeting™ 2.1.

ILS is defined and represented by a set of COM-based objects. A client application makes use of these objects through predefined sets of interfaces to each instance of the object.

For more information and guidelines refer to the ILS Extended Attributes and Implementing Filters in ILS Queries tutorials.
