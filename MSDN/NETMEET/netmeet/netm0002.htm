What's New for NetMeeting 2.1


This release of the Microsoft® NetMeeting™ Software Developers Kit (SDK) contains the NetMeeting 2.0 APIs. These have not changed, so your applications written using the NetMeeting 2.0 APIs will continue to function as expected.

New features of this SDK include:

  • New and updated sample applications.
  • New tutorials that guide you through the following tasks.
    • Using the data channel.
    • Using extended attributes and filters in an ILS client.
    • Launching programs remotely.
    • Creating a call center.
  • Updated references with more complete information about using the NetMeeting SDK.
  • An updated Frequently Asked Questions section that addresses the issues we've been asked about most.
  • New information about installing and setting up NetMeeting.
  • Background information about the standards on which NetMeeting was developed
  • New examples demonstrating ways to use the SDK.

For information about NetMeeting, including what's new in the NetMeeting 2.1 product, see the Microsoft NetMeeting web site.
