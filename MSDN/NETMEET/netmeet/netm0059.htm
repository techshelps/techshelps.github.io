Are NetMeeting sample applications available?

In addition to the tutorials provided in this documentation, the NetMeeting SDK includes several sample applications that are located in the samples folder of your NetMeeting SDK directory. The Samples.txt file located in the NetMeeting SDK directory contains information to aid you in your use of these samples.

The NetMeeting SDK can be downloaded for free from the NetMeeting web site.

Where can I send feedback and questions about the NetMeeting SDK?

Support for Microsoft NetMeeting and the NetMeeting SDK is provided through standard Microsoft product support channels; see the Microsoft Support Online pages for details. In addition, the following newsgroup is available for you to interact and share information with other users of the SDK: //msnews.microsoft.com/microsoft.public.internet.activexconferencing.sdk.

We also welcome feedback about the NetMeeting SDK at msconf@microsoft.com.
