Conceptual Diagram


The NetMeeting COM objects are:

  • Conference Manager – manager of the overall conferencing system.
  • Local System – information about the local user and local computer.
  • Call – monitors and controls an incoming or outgoing call.
  • Conference – manages an instance of a conference.
  • Member – a participant in a conference.
  • Channel – a conduit for a specific type of media (application sharing, audio, data, file transfer, and video).
  • Audio Channel – a conduit for audio type media.
  • Video Channel – a conduit for video type media.
  • File Transfer Channel – manager of file transfer activity.
  • Application Sharing Channel – manager of shared applications.
  • Data Channel – a conduit for sending and receiving data between conference participants.
