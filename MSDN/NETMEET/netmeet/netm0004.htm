Dependencies


To run an application that uses the Microsoft® NetMeeting™ SDK, NetMeeting 2.0 or later must be installed on your computer.

The NetMeeting SDK runs only on Windows 95 or later and Windows NT 4.0 or later. Do not install this SDK on Windows NT 3.51 or Windows 3.1.

To implement application sharing on Windows NT 4.0, you must install Service Pack 3 and the application sharing driver as described in Enabling Application Sharing on Windows NT 4.0.

The NetMeeting control is implemented as a custom control called the ActiveX™ Conference Control. To use the control in your application, the NetMeeting application must be installed on your computer.
