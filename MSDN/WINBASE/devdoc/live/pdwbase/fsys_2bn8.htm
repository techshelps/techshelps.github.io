IDiskQuotaUser::GetQuotaUsedText

[This is preliminary documentation and subject to change.]

Retrieves the user's quota used value for the volume. This value is expressed as a text string, for example "10.5 MB".

HRESULT GetQuotaUsedText(
  LPWSTR pszText,
  DWORD cchText
);

Parameters

pszText
Pointer to the buffer to receive the text string.
cchText
Size of the buffer, in bytes.

Return Values

This method returns one of the following values.

Value Meaning
S_OK Success.
DQC_E_ACCESS_DENIED The caller has insufficient access rights.
DQC_E_LOCK_FAILED Failure to obtain an exclusive lock.
E_OUTOFMEMORY Insufficient memory.
E_FAIL Failure reading quota information.
