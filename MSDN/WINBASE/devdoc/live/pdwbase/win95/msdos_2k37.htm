MS-DOS Extensions Structures

This section provides information about the following structures.

DRIVE_MAP_INFO Get Drive Map Info returns information about the specified drive in this structure.
MID This structure, which is used by Get Media ID (Interrupt 21h Function 440Dh Minor Code 66h) and Set Media ID (Interrupt 21h Function 440Dh Minor Code 46h), has been updated to support compact disc (CD) file systems. For information about using these functions, see the Microsoft MS-DOS Programmer's Reference.
PARAMBLOCK This structure is needed by Lock/Unlock Removable Media.
