FOCUS_EVENT_RECORD

The FOCUS_EVENT_RECORD structure is used to report focus events in a console INPUT_RECORD structure. These events are used internally and should be ignored. 

typedef struct _FOCUS_EVENT_RECORD { // fer 
    BOOL bSetFocus; 
} FOCUS_EVENT_RECORD;
