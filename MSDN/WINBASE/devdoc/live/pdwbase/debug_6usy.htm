UNLOAD_DLL_DEBUG_INFO

The UNLOAD_DLL_DEBUG_INFO structure contains information about a dynamic-link library (DLL) that has just been unloaded. 

typedef struct _UNLOAD_DLL_DEBUG_INFO { // uddi 
    LPVOID lpBaseOfDll; 
} UNLOAD_DLL_DEBUG_INFO;

Members

lpBaseOfDll
Pointer to the base address of the DLL in the address space of the process unloading the DLL.

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in winbase.h.
