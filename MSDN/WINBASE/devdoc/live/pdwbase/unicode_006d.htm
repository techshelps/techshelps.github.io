Using Special Characters in Unicode

There are a few special characters and characters with unusual meanings in Unicode text strings. To avoid unexpected problems with these characters, use the rules provided in the following topics.

  • Null-terminated strings
  • Byte-order mark
  • Escape sequences and control characters
  • Line and paragraph separators
  • ASCII control codes 0x000D and 0x000A
  • Nonspacing characters and diacritics
