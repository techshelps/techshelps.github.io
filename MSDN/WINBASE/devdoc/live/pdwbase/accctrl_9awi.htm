TRUSTEE_FORM

The TRUSTEE_FORM enumeration type contains values that indicate how a TRUSTEE structure identifies the trustee.

typedef enum _TRUSTEE_FORM
{
    TRUSTEE_IS_SID,
    TRUSTEE_IS_NAME,
} TRUSTEE_FORM;
Enumerator Value Meaning
TRUSTEE_IS_SID The trustee is identified by a security identifier (SID).
TRUSTEE_IS_NAME The trustee is identified by a name.
