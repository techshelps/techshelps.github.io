PDH_RAW_COUNTER

The PDH_RAW_COUNTER structure returns the data as it was collected from the counter provider. No translation, formatting, or other interpretation is performed on the data.

typedef struct _PDH_RAW_COUNTER {
    DWORD       CStatus;
    FILETIME    TimeStamp;
    LONGLONG    FirstValue;
    LONGLONG    SecondValue;
    DWORD       MultiCount;
} PDH_RAW_COUNTER, *PPDH_RAW_COUNTER;

Members

CStatus
The status of the last collection operation for this counter. See Counter Status versus Function Status.
TimeStamp
The local time this data was collected.
FirstValue
The first raw counter value.
SecondValue
The second (if necessary) raw counter value.
MultiCount
The multi count (normally 1).
