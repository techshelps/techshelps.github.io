Very Large Memory (VLM)

[This is preliminary documentation and subject to change.]

The Very Large Memory (VLM) extensions provide 64-bit addressing for very large regions of memory on Microsoft® Windows NT®. Before working with VLM, you should be familiar with memory management and file mapping.
