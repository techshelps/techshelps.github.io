DBT_CONFIGCHANGECANCELED

The system sends the DBT_CONFIGCHANGECANCELED device message when a request to change the current configuration (dock or undock) has been canceled.

To send the DBT_CONFIGCHANGECANCELED device message, the system sends the WM_DEVICECHANGE message with wParam set to DBT_CONFIGCHANGECANCELED and lParam set to zero.

Parameters

lParam
lParam is zero when the WM_DEVICECHANGE message is sent with wParam set to DBT_CONFIGCHANGECANCELED.

Return Values

Return TRUE.

QuickInfo

  Windows NT: Requires version 5.0 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in dbt.h.
