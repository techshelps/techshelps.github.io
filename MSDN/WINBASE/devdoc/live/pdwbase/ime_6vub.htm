ImmGetOpenStatus

The ImmGetOpenStatus function checks whether the IME is open or closed.

BOOL WINAPI ImmGetOpenStatus(
  HIMC hIMC  
);

Parameters

hIMC
Handle to the input context.

Return Values

The return value is a nonzero value if the IME is open, and zero otherwise.
