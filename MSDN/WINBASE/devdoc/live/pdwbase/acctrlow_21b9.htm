GetAce

The GetAce function obtains a pointer to an ACE in an ACL.

An ACE is an access control entry. An ACL is an access control list. 

BOOL GetAce(
  PACL pAcl,         // pointer to access-control list
  DWORD dwAceIndex,  // index of ACE to retrieve
  LPVOID *pAce       // pointer to pointer to ACE
);

Parameters

pAcl
Points to an ACL structure containing the ACE to be retrieved.
dwAceIndex
Specifies the ACE to which a pointer is retrieved. A value of 0 corresponds to the first ACE in the ACL, 1 to the second ACE, and so on.
pAce
Points to a pointer the function sets to the address of the ACE.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.
