EXIT_PROCESS_DEBUG_INFO

The EXIT_PROCESS_DEBUG_INFO structure contains an exiting process's exit code. 

typedef struct _EXIT_PROCESS_DEBUG_INFO { // epdi 
    DWORD dwExitCode; 
} EXIT_PROCESS_DEBUG_INFO;

Members

dwExitCode
Specifies the exit code for the process.
