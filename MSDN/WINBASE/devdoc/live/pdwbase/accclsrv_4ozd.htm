DestroyPrivateObjectSecurity

The DestroyPrivateObjectSecurity function deletes a private object's security descriptor. This security descriptor must have been created by a call to the CreatePrivateObjectSecurity function. 

BOOL DestroyPrivateObjectSecurity(
  PSECURITY_DESCRIPTOR *ObjectDescriptor   // address of pointer to 
                                           // SECURITY_DESCRIPTOR
);

Parameters

ObjectDescriptor
Pointer to a pointer to the SECURITY_DESCRIPTOR structure to be deleted.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero. To get extended error information, call GetLastError.
