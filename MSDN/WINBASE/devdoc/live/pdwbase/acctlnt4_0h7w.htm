Windows NT 4.0 Access Control

This overview describes the Microsoft® Windows NT® version 4.0 access-control functions for working with security descriptors and access-control lists (ACLs). These functions are also supported on later versions of Windows NT.

The Windows NT 4.0 access-control functions are one of three sets of Microsoft® Win32® access-control functions. For information about when to use each set of functions, see Access-Control Reference.
