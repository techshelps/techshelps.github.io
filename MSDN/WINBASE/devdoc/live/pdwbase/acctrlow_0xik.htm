Low-Level Access-Control

This overview describes the low-level access-control functions for working with security descriptors and access-control lists (ACLs). Most low-level access-control functions are supported on all versions of Microsoft® Windows NT®. The low-level functions for working with object-specific ACEs are supported by Windows NT versions 5.0 and later.

The low-level access-control functions are one of three sets of access-control functions. For information about when to use each set of functions, see Access-Control Reference.
