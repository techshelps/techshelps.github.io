File Code Page

The AreFileApisANSI function determines whether the file functions are using the ANSI or OEM character set code page. The SetFileApisToANSI function causes the functions to use the ANSI code page. The SetFileApisToOEM function causes the functions to use the OEM code page.

By default, file functions use ANSI filenames. Functions exported by KERNEL32.DLL which accept or return file names are affected by the file code page setting.
