TOKEN_PRIMARY_GROUP

The TOKEN_PRIMARY_GROUP structure specifies a group security identifier (SID) for an access token. 

typedef struct _TOKEN_PRIMARY_GROUP { // tpg 
    PSID PrimaryGroup; 
} TOKEN_PRIMARY_GROUP;

Members

PrimaryGroup
Pointer to a SID structure representing a group that will become the primary group of any objects created by a process using this access token. The security identifier (SID) must be one of the group SIDs already in the token.
