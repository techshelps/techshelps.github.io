TOKEN_TYPE

The TOKEN_TYPE enumeration type contains values that differentiate between a primary token and an impersonation token. 

typedef enum tagTOKEN_TYPE { // tt 
    TokenPrimary = 1, 
    TokenImpersonation 
} TOKEN_TYPE;

QuickInfo

  Windows: Unsupported.
