I

Import media pool
Media Service for Windows NT puts a medium in an import media pool when Media Service for Windows NT recognizes the on-media identifier, but the medium is not cataloged in the current Media Service database.
Imported state
Media whose label types are recognized by Media Service for Windows NT, but whose label IDs are not cataloged by Media Service for Windows NT.
Incompatible state
Media that are not compatible with the library in which they were classified. This media should be immediately ejected from the library hardware unit. Media Service for Windows NT puts Incompatible media in the foreign media pool.

Independent Software Vendor
