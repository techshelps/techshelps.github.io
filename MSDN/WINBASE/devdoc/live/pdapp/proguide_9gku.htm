Roadmap to Removable Storage Manager Documentation

[This is preliminary documentation and subject to change.]

The Removable Storage Manager (RSM) for Microsoft® Windows NT® documentation covers the following functional areas:

  • Removable Storage Manager Overview

    Contains high-level information about RSM and the features it provides, as well as information about designing your own RSM application.

  • Developing a Media Label Library

    Contains a sample media label library and information on how to test and install a media label library.

  • RSM Media Handling

    Contains information on how RSM handles media, including information about allocating, deallocating, and importing media.

  • Removable Storage Manager Function Reference

    Contains RSM function reference pages.

  • Media Label Library Function Reference

    Contains media label library function reference pages.

  • Removable Storage Manager Structure Reference

    Contains RSM structure reference pages.

  • Glossary

    Contains RSM glossary terms and definitions.
