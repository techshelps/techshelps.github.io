Allocate
Mark media for use by an application. You can allocate media in the Available state.
Allocated state
Media that are in use and assigned to application media pools.
API
Application Programming Interface
Application media pool
Application media pools are created by applications. They determine which media can be accessed by which applications and set the policies for that media. There can be any number of application media pools in a Media Service system.
Available state
Media that can be allocated for use by applications.
