Testing Your Media Label Library

[This is preliminary documentation and subject to change.]

RSM includes a sample test utility that reads media and uses the media label library functions to verify that the information added to the MediaLabelInfo data structure is correct. RSM displays the information added to the data structure. The utility is hard-coded to read \\.\tape0 and uses the media label library, YourMediaLabelLibraryHere.dll. Replace YourMediaLabelLibraryHere.dll with the name of your media label library, including the absolute path, before using the test utility.
