Offline library
The offline library holds all media that are known by Media Service for Windows NT but are not in an online library or stand-alone drive. Media can only be moved from an offline library to an online library or stand-alone drive by the administrator physically inserting it into a library unit. A set of shelves is an example of an offline library. There is one offline library per Media Service system.
OMID
On-media identifiers are labels that are electronically recorded on each medium in a Media Service system. Media Service for Windows NT uses on-media identifiers to track media in the Media Service database. An application on-media identifier is a subset of the media label.
Online library
Online libraries are robotic library units. They are sometimes referred to as jukeboxes.
Operator request
A request for the operator to perform a task. This request can be issued by an application or by Media Service for Windows NT.
