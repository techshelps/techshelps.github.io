L

Library
A media storage location. There are three types of libraries: online, stand-alone drives, and offline. A medium always exists in some type of library.
Library request
A request for an online library or stand-alone drive to perform a task. This request can be issued by an application or by Media Service for Windows NT.
LMID
A logical representation of a physical storage location. Media Service for Windows NT generally defines sides of the logical media for use by the application. The intent of logical media is to allow the underlying physical medium of the data to change without affecting the application.
Logical media
Logical media are abstract representations of partitions of physical media. All media that are currently allocated in Media Service for Windows NT are represented as logical media. Logical media are referenced by LMIDs (logical media IDs).
