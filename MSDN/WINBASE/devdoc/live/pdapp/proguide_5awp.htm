Disaster Recovery

[This is preliminary documentation and subject to change.]

This section addresses the backup and recovery of a RSM database. The following are tasks that the administrator must complete to prepare for and execute the recovery process:

  1. Backing up the RSM database
  2. Tracking the RSM database backup media
  3. Mounting the media
  4. Recovering the database
  5. Updating the restored database

Note  The information in this section assumes that there has been no change to the system configuration that affects RSM.
