RSM Configuration

[This is preliminary documentation and subject to change.]

When configuring RSM applications, administrators and application developers should remember the following issues:

  • For performance reasons, administrators should not manage more than 1,000 media in a single RSM system.
  • RSM includes an autoconfiguration feature that maps each drive in a library unit to a changer. To avoid manual configuration, administrators should install changers and their drives on the same SCSI bus.
  • RSM does not support every existing piece of hardware. Before adding new hardware to an RSM system, administrators must check http://www.microsoft.com/isapi/hwtest/hcl.idc on the Microsoft Web site for the Microsoft® Windows NT® Hardware Compatibility List (HCL) that lists all the libraries and stand-alone drives supported by RSM.
