Database File Relocation

[This is preliminary documentation and subject to change.]

The database files can be relocated by creating the registry key REG_SZ (NtmsData) in the registry hierarchy at HKEY_LOCAL_SYSTEM\SYSTEM\CurrentControlSet\Control\NTMS. This allows the database to be relocated to a different local disk or a remote file service. In some cases, the RSM administrator might want to relocate the RSM database to a more reliable file server using RAID or mirroring technology.
