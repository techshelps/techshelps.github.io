Handling Optical/Rewritable Random-access Media in RSM

[This is preliminary documentation and subject to change.]

Media such as rewritable optical (EO), JAZ, and ZIP cartridges are example of media types that are members of this category. However, RSM supports only Optical EO media. RSM requires that these media have a file system written on the medium. RSM currently supports both FAT and NTFS file systems.
