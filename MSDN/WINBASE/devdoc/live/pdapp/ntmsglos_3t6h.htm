Scratch media pool
Scratch media pools contain media that are not allocated by an application and do not contain data. There is a one scratch media pool for each media type in a Media Service system.
Sequence numbers
Media Service for Windows NT assigns sequence numbers to media for visual reference by the administrator during inserting, ejecting, and offline tracking of media.
Stand-alone drive
An online drive that is not part of a library unit. Media Service for Windows NT treats stand-alone drives as online libraries with one drive and a port.
System media pool
Include scratch, foreign, and import media pools. RSM creates one scratch, one foreign, and one import media pool for each media type in an RSM system.
