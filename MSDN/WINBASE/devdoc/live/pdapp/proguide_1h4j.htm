RSM Media Pools

[This is preliminary documentation and subject to change.]

There are four types of media pools in RSM:

  • Foreign

    Media that is new or unrecognized

  • Import

    Media that have labels that are recognized by the system but are not in the RSM catalog

  • Scratch

    Media available for writing

  • Application

    Media used by a particular application on the system

All media in an application media pool are in the RSM catalog by default. Media in the available state can reside in both the scratch media pools and application pools. Note that there are individual Import, Foreign, and Scratch media pools for each media type detected on the system.

Media in Scratch, Import, or Foreign media pools are deleted from the RSM catalog when they are ejected from the library or drive.
