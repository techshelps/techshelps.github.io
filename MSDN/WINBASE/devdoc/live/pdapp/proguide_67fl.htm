Rewritable Disk Media

[This is preliminary documentation and subject to change.]

There are two classes of rewritable removable disk: rewritable optical disk (3.5", 5.25") and rewritable magnetic disk (floppy, ZIP, JAZ, and so forth). In Windows NT 5.0, RSM supports rewritable optical disks, but does not support the management of rewritable magnetic disk..

RSM accesses the disk drive at the raw driver level. On an RSM mount completion, the application can view the volume as a mapped drive (for example, E:\). As part of updating drive configuration, both the raw path and the mapped path are available in the RSM database so the application can access the drive with either handle.
