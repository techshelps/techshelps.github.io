RSM MMC Snap-in

[This is preliminary documentation and subject to change.]

The RSM MMC snap-in is a user interface to RSM that is a snap-in for the Microsoft Management Console (MMC) for Windows NT. Administrators can use the snap-in to add RSM objects, view and modify properties of objects, insert and eject media, perform inventories, mount and dismount media, and check status information. RSM objects can be physical objects (such as libraries, drives, and media) or groups of media (called media pools).

The MMC snap-in can help administrators perform the following tasks:

  • Handle operator requests
  • Point and click operations such as inserting media, ejecting media, drive cleaning, and dismounting media
  • Report status
  • Manage media pools manually
