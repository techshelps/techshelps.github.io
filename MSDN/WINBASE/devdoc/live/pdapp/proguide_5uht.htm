Mounting the Media

[This is preliminary documentation and subject to change.]

To allow the recovery application to access media, the media must be mounted in a drive. Most libraries have doors that allow the administrator to mount the media. If there is no door, the administrator can mount media from the import media pool in the RSM MMC snap-in or from the front panel of the library.

Recovery applications should use the RSM API to select and mount the appropriate media as part of their disaster recovery procedures.
