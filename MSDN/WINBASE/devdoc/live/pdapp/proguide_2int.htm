Installing A Media Label Library

[This is preliminary documentation and subject to change.]

You must install or provide instructions for installing your media label library in the installation program for your application. Your installation program must create the appropriate registry entries and copy your media label library to the appropriate directory. The following figure illustrates the registry hierarchy that your application must create. YourCompanyName represents your company or product name and must be unique.

Under the key for your media label library, the installation program must create two registry values: Description and Path. The previous figure displays these values in the right window pane. The Description registry value contains a description of the media label library. The Path registry value contains the absolute path of the media label library.

Remember the following rules when you develop the installation program for your application:

  • Installations of new media label libraries are restricted to members of the Administrator group.
  • The registry entries that contain the list of media label libraries are accessible only by members of the Administrator group.
  • The ExtMLL key is optional.
  • The directory that contains all the media label library DLLs is typically accessible only by members of the Administrator group. The default directory for RSM media label libraries is winnt\system32.

Note  If you distribute an update, users can install the new media label library through the RSM MMC snap-in.
