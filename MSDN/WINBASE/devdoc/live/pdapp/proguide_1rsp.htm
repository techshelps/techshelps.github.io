Inserting a CD-ROM into a library

[This is preliminary documentation and subject to change.]

When a CD-ROM is introduced into a library by an insert command or door access command, the CD-ROM is examined to determine its destination media pool. The identification process uses the CDFS label as identification. The following diagram shows the process and resulting media pool for the medium. Because CD-ROMs are not writable media, CD-ROMs never appear in a scratch pool.
