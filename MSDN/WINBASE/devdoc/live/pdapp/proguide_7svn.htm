Moving a CD-ROM Among Media Pools

[This is preliminary documentation and subject to change.]

Compact discs in general appear in the import pool. A user or application can create an application pool and move media to that pool, allowing RSM to continue to track the media. If the media is in the import pool and is ejected from the library or drive without moving it to an application pool, the media is deleted from the RSM catalog.

Also, because the media is read-only, RSM does not allow it to be placed in the Scratch pool, and does not attempt to label the media as it moves between pools. The following table shows what happens when media is moved from or to different media pools.

From/To Import Foreign Application
Import Not allowed Allowed – no action
Foreign Not allowed Not allowed
Application Not allowed Not allowed Allowed – no action
