Moving Optical/Rewritable Among Media Pools

[This is preliminary documentation and subject to change.]

The following table shows what happens when media is moved from or to different media pools.

From/To Scratch Import Foreign Application
Scratch Not allowed Not allowed Retains Scratch label until allocated.
Import Write NTFS Scratch label Not allowed Not allowed
Foreign Write NTFS Scratch label Not allowed Not allowed
Application Write NTFS Scratch label Not allowed Not allowed Retains current label. Allocated media may not be moved from a pool. Both sides must be Available.
