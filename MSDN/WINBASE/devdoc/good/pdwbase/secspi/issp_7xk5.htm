Security Support Provider Interface

The Security Support Provider Interface is a common interface to security providers. Security providers export a set of functions that can be used to gain a certain level of security support. This interface is not specific to Microsoft® Windows NT™. It is supported by RPC (Remote Procedure Call) in Windows NT and Windows 95, as well as RPC on MS-DOS®, Windows® version 3.11, and Macintosh® (both MC680x0 and PowerPC).
