Calling the NTLMSSP Security Provider

The Windows NT LAN Manager Security Support Provider (NTLMSSP) security provider is based on the NTLM (Windows NT LAN Manager) authentication protocol. NTLMSSP ships with the following products:

  • Windows NT versions 3.5 and later
  • Windows 95
  • The RPC SDK for Windows 3.1, MS-DOS, and Macintosh platforms
