bound control

A data-aware control that can provide access to a specific column or columns in a data source through an ADO Data control. A data-aware control can be bound to an ADO Data control through its DataSource and DataField properties. When an ADO Data control moves from one row to the next, all bound controls connected to the ADO Data control change to display data from columns in the current row. When users change data in a bound control and then move to a different row, the changes are automatically saved in the data source.
