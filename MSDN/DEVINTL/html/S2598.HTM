Appendix L Multilingual API Functions and Structures

This appendix consists of two tables. The first table lists the functions contained in the multilingual API, and the second table lists the structures contained in the multilingual API. Although Microsoft Windows NT version 3.x does not support the multilingual API, the next version of Windows NT, sometimes referred to as "Cairo," will contain full support for the API. Microsoft Windows 95 contains partial support for the API.
