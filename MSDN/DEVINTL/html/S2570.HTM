Appendix H Code Pages

This appendix lists Microsoft Windows, MS-DOS, and Macintosh code pages. Each of the Windows code pages, 1250 through 1257, is followed by a list showing the individual characters, the corresponding Unicode IDs, and the Unicode descriptive names. Note that the MS-DOS code pages include additional picture characters in code points 0x00 through 0x1F; these have been omitted.
