Appendix G DBCS/Unicode Mapping Tables

This appendix lists characters in several double-byte character set (DBCS) encodings and their corresponding Unicode code points. The following code pages are included:

Code Page Language
932 Japanese
936 Simplified Chinese
949 Korean
950 Traditional Chinese

Within each code page, the characters are grouped by the lead byte, which is shown in the upper-left corner of each block of characters. First digits of the trail bytes are given at the left, and second digits are given at the top. The corresponding Unicode code point is shown beneath each character. Consult The Unicode Standard, Version 1.0, Volume 2, for a listing of characters in order of Unicode code-point value.
