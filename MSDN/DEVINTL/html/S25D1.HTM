Appendix V International Address Formats

This appendix consists of one table, which lists the preferred address formats for selected countries. Details about formatting conventions, such as spacing and hyphenation requirements, are provided in the note following the address. Information enclosed in square brackets is considered optional. Note that formatting standards change from time to time, so you should always double check these formats with other sources.
