Sentence-Mode Bits

Constant Description
IME_SMODE_AUTOMATIC IME carries out the conversion in automatic mode.
IME_SMODE_NONE No sentence information.
IME_SMODE_PHRASEPREDICT IME uses phrase information to predict the conversion for the next character.
IME_SMODE_PLURALCLAUSE IME uses plural clause information to carry out the conversion.
IME_SMODE_SINGLECONVERT IME carries out the conversion in single-character mode.
