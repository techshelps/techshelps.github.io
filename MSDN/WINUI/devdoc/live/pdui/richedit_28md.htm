EM_GETOLEINTERFACE

The EM_GETOLEINTERFACE message retrieves an IRichEditOle object that a client can use to access a rich edit control's OLE functionality.

EM_GETOLEINTERFACE 
wParam = 0; 
lParam = (LPARAM) (LPVOID FAR *) ppObject;

Parameters

ppObject
Address at which the control stores a pointer to the IRichEditOle object. The control calls the AddRef function for the object before returning, so the calling application must call the Release function when it is done with the object.

Return Values

Returns a nonzero value if successful, or zero otherwise.

QuickInfo

  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
