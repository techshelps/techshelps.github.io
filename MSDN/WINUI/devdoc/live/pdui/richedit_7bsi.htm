FINDTEXT

The FINDTEXT structure contains information about text to search for in a rich edit control. This structure is used with the EM_FINDTEXT message.

typedef struct _findtext { 
    CHARRANGE chrg; 
    LPSTR lpstrText; 
} FINDTEXT;

Members

chrg
Range to search.
lpstrText
Null-terminated string to find.

QuickInfo
