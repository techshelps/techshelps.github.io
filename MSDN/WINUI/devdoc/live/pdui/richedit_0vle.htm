ENSAVECLIPBOARD

Contains information about objects and text on the clipboard.

typedef struct {
    NMHDR nmhdr;        
    LONG cObjectCount;  
    LONG cch;           
} ENSAVECLIPBOARD;

Members

nmhdr
Notification header.
cObjectCount
Number of objects on the clipboard.
cch
Number of characters on the clipboard.
