STM_SETIMAGE

An application sends an STM_SETIMAGE message to associate a new image (icon or bitmap) with a static control.

STM_SETIMAGE 
wParam = (WPARAM) fImageType;       // image-type flag 
lParam = (LPARAM) (HANDLE) hImage;  // handle to the image

Parameters

fImageType
Value of wParam. Specifies the type of image to associate with the static control. This parameter can be one of the following values:

IMAGE_BITMAP
IMAGE_CURSOR
IMAGE_ENHMETAFILE
IMAGE_ICON

hImage
Value of lParam. Handle to the image to associate with the static control.

Return Values

The return value is a handle to the image previously associated with the static control, if any; otherwise, it is NULL.

QuickInfo
