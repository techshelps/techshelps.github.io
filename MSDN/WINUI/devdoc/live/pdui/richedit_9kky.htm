FORMATRANGE

The FORMATRANGE structure contains information that a rich edit control uses to format its output for a particular device. This structure is used with the EM_FORMATRANGE message.

typedef struct _formatrange { 
    HDC hdc; 
    HDC hdcTarget; 
    RECT rc; 
    RECT rcPage; 
    CHARRANGE chrg; 
} FORMATRANGE;

Members

hdc
Device to render to.
hdcTarget
Target device to format for.
rc
Area to render to. Units are in TWIPS
rcPage
Entire area of rendering device. Units are in TWIPS
chrg
Range of text to format.

QuickInfo
