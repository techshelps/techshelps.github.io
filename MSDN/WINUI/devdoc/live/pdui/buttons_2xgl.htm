BN_UNHILITE

The BN_UNHILITE notification message is sent when the highlight should be removed from a button. The parent window of the button receives this notification message through the WM_COMMAND message. 

BN_HILITE

BN_UNHILITE is the same as the BN_UNPUSHED notification message.

Note  This notification message is provided for compatibility with applications written for 16-bit versions of Windows earlier than version 3.0. Win32-based applications should use the BS_OWNERDRAW button style and the DRAWITEMSTRUCT structure for this task.

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Requires version 2.0 or later.
  Header: Declared in winuser.h.
