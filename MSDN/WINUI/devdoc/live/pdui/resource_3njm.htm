MENUHEADER

The MENUHEADER structure contains version information for the menu resource. The structure definition provided here is for explanation only; it is not present in any standard header file.

struct MENUHEADER { 
    WORD wVersion; 
    WORD cbHeaderSize; 
};

Members

wVersion
Specifies the version number of the menu template. This member must be equal to zero to indicate that this is an RT_MENU created with a standard menu template.
cbHeaderSize
Specifies the size of the menu template header. This value is zero for menus you create with a standard menu template.

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
