Policies

A policy is specified as follows.

POLICY name
[KEY key name]
[... part definition statements ...]
END POLICY

name
Policy name as it should appear in the System Policy Editor list box. It may optionally be enclosed by double quotation marks. (Names with spaces must be enclosed by double quotation marks.)
key name
Optional registry key name to use for the policy. If a key name is specified, it will be used by all child parts of the policy, unless they specifically provide a key name of their own.
