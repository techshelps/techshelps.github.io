And several people asked…

Dear Dr. GUI:

I cannot get the Library Archive I recently received to install correctly on 16-bit Windows—do you know of any problem with this?

Dr. GUI replies:

The problem is that the 16-bit setup is writing incorrect information in the infoview.ini file. In addition, two Help files are not getting copied to the local drive, resulting in an error when trying to view them from InfoViewer's Help menu or from the Program Manager icons.

For a workaround, connect to http://microsoft.com/msdn/whatshot.htm.and read the section titled "Users will receive an error when attempting to launch the Library Archive CD on 16-bit platforms."
