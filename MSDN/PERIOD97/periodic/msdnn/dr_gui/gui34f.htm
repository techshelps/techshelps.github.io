Saving the screen

Dear Dr. GUI,

I was interested in writing a screen saver for Windows 95 and being new to this type of programming, I was wondering if you could give me a few references, hints, pointers, etc.

  1. Which of the SDKs will I need?
  2. Which language would you recommend for writing screen savers?
  3. Can you recommend any books which would be useful? Any other resources? Newsgroups, mailing lists, FAQs, code samples, etc.

Thank you in advance,
Glenn Schmid

Dr. GUI replies:

I’m glad you asked this question. Screen savers are clearly the next untapped growth market in software. (Actually, with Internet push technology, this is more than somewhat true. . . .) To answer your questions:

  1. To write screensavers for Windows 95, you’ll need the Win32 SDK.
  2. Dr. GUI recommends using C++ for this task. You’ll need C++ or at least C to interface with Windows, although it’s possible to write the real screen saver as an ActiveX control in any language, including Visual Basic or Java.
  3. A sample you might find useful is in the Microsoft Knowledge Base: “How to Write a 32-Bit Screen Saver” (Q160960). If that’s not enough, there is a whole slew of newsgroups for Windows programmers listed at http://www.microsoft.com/support/news/win32.htm.
