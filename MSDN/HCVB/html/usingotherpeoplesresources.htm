Using Other People’s Resources

The WinWatch program has a feature I discussed only briefly in Chapter 6: for each program or DLL you select, WinWatch shows all the resources of that module in a list box in the lower right corner of the screen. When you select a resource in the list box, WinWatch displays or otherwise handles the item in the Resource Information area (which is actually a picture box). The display varies, depending on the resource type.

It’s an adventure to look at resources in the Windows DLLs, Program Manager, and Visual Basic itself. You can learn a lot about resources and how programmers use them in other languages.
