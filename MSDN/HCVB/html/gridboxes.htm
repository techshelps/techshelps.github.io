Grid Boxes

Visual Basic’s Properties window offers a convenient color dialog box that pops up when you click on the down arrow for BackColor, ForeColor, or any other color property. Wouldn’t it be nice to include a similar dialog box in your own programs?

In fact, you might have noticed one when testing the Fun ’n Games program (FUNNGAME.VBP) discussed in Chapter 7. I skipped the Color dialog box in that chapter, but now I’m ready to talk about it and about how you could expand on the idea.
