.v (Virtual Machine Commands) 


.v[option]

Displays information about the cuurent virtual machine (VM).

option

Command option. Can be one of these option letters:

c

Displays the standard fields of the current VM's control block.

h

Displays the current VM's handle.

m

Displays the status of the current VM. Status information includes re-entry count, VM handle, critical section state, client registers, and top entries from the client's stack.

r

Displays the current VM's client registers if the 80386 debugger is running in protected mode.

s

Displays the current VM's virtual mode stack if the debugger is running in protected mode.

l

Displays a list of all valid VM handles.
