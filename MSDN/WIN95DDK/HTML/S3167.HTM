Setup Program

To set up the DDK using the Setup program, run Setup.exe from the CD-ROM.

Important

The DDK Setup program does not install the Windows Hardware Quality Labs (WHQL) compatibility tests. You will need to run Ctsetup.exe from the \Ct directory on the CD-ROM. For more information, see .

Where to Install

The first question the setup program asks you is where you want to install the DDK. By default, Setup chooses c:\Ddk as the installation directory.

Tip

You might want to create a compressed drive to store the DDK because most of the files are text files, which compress well.

What to Install

The installation program asks you next what you want to install. By default, all installation options are selected.
