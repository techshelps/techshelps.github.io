VK_PROCESSKEY

This is sent to an application as the wParam of a WM_KEYDOWN or WM_KEYUP message. When this virtual key is generated, the real virtual key is saved in the input context, if the messages generated by IME are stored in the input context. The system restores the real virtual key or posts the messages that are stored in the message buffer of the input context.

lParam

Must be 1.
