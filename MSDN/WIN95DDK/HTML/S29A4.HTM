OPTIONAL Step 3 -- Adding FAX Support

The Microsoft Fax modem drivers and T.30 protocol engine use several registry entries settings to control their behavior. These settings are determined the first time the fax software is run, using a combination of the following techniques:

  • Querying the modem for capabilities.
  • Querying the modem to get an ID, and then using this ID to look up an internal table.
  • And, new to Windows 95, reading the unimodem INF section for the particular modem installed. If an entry is present in the INF section, it overrides information obtained via the previous two methods.

This section describes how add fax entries to the modem's INF file, lists the major known incompatibility problems (and their workarounds, when available), and documents a fax communications I/O logging facility that is useful for troubleshooting modem-related problems.
