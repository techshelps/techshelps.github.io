RnaCloseMac 


DWORD
WINAPI RnaCloseMac(HANDLE hConn)

This function is called by an SMM to request RNA to close the adapter port and to unload the MAC.

hConn

The RNA connection handle.

  • SUCCESS or one of the following error codes:

ERROR_INVALID_HANDLE

The RNA connection handle is invalid.

ERROR_INVALID_PORT_HANDLE

The RNA connection handle is invalid.
