VariableTerminator

Characters that terminate a caller ID or distinctive ring response from modem.

Use a single line of one or more characters. For example, 


HKR,, VariableTerminator,,

A variable length response is composed of a fixed response (which is listed in the Responses registry entries) followed by a variable length string. This string is terminated by this Variable Terminator. The high bit of the first byte of the Response binary data registry entry must be set for the response to be interpreted as a variable length response.

See also Responses registry entry.
