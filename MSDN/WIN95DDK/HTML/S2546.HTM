CONFIGMG_Reenumerate_DevNode 


CONFIGRET CONFIGMG_Reenumerate_DevNode(DEVNODE dnDevNode, ULONG ulFlags)

Causes the enumerator of the specified device node to receive the CONFIG_ENUMERATE enumerator function.

  • Returns CR_SUCCESS if the function is successful. Otherwise, the return value can be CR_INVALID_FLAG. There is no checking of a valid devnode until later.

dnDevNode

The device node whose enumerator will be called.

ulFlags

Must be zero.

This function is asynchronous (that is, callable at interrupt time), reentrant, and fast.
