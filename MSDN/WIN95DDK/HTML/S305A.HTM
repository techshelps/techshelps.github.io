ISP_QUERY_REMOVE 


#include <isp.h>

typedef struct ISP_QUERY_REMOVE {
    ISP    ISP_q_remove_hdr;    // standard header
    PVOID  ISP_q_remove_pdcb;   // physical DCB pointer
    ULONG  ISP_q_remove_flags;  // flags; see below
} ISP_QUERY_REMOVE, *PISP_QUERY_REMOVE;

Contains information for a ISP_QUERY_REMOVE_DCB function.

ISP_q_remove_flags

Can be ISP_Q_REMOVE_FL_MEDIA_ONLY to indicate only the medium is going away.

See also DCB, ISP
