EnableDistinctiveRing

Command to enable distinctive ringing reporting.

One or more command strings can be used. For example, 


HKR, EnableDistinctiveRing, 1,, "at-sdr=7"

If modem does not have ability to report distinctive rings, this entry may be omitted. Distinctive ringing can be enabled or disabled from the modem's property page. The VoiceProfile registry entry can specify if distinctive ringing is supported, and what type of distinctive ringing is supported. If distinctive ringing is enabled, make sure that the correct distinctive ringing responses are listed in the Responses registry settings.

See also VoiceProfile registry entry.
