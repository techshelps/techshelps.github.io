End_Message_Mode 


include vmm.inc

mov     ebx, VMHandle
mov     eax, End_Message_Mode
VMMcall System_Control

Directs the virtual device to end message-mode processing. The virtual device may modify all general registers and flags.

  • This message cannot be failed. The virtual device must return carry clear.

VMHandle

Handle of the virtual machine leaving message mode.

See also Begin_Message_Mode, End_Message_Mode2
