IFSMgr_RunScheduledEvents 


IFSMgr_RunScheduledEvents

This service runs any pending events that were queued by requests to IFSMgr_SchedEvent or IFSMgr_QueueEvent and returns. It returns a status indicating whether or not any events were run.

None.

  • Carry flag clear, pending events were run.

    Carry flag set, no pending events.

Registers Used

None.
