MXDM_CLOSE

Sent to an audio mixer device driver to deallocate a specified open instance. This message is normally sent to an audio mixer driver in response to a mixerClose call.

  • The return value is an error code, or zero (MMSYSERR_NOERROR) if the operation is successful. A mixer driver should never fail the MXDM_CLOSE message.

dwParam1

Unused.

dwParam2

Unused.

An audio mixer driver must support this message. Note that the MXDM_CLOSE message will only be sent to a mixer driver if the corresponding MXDM_OPEN message succeeded.
