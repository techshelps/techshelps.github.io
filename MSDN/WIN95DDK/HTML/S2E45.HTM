Driver Return Values

The driver return values are listed in the following table.

Constant

Description

Value

DRV_BADPARAM1

Bad first parameter.

0xFFFFFFFF

DRV_BADPARAM2

Bad second parameter.

0xFFFFFFFE

DRV_BADSTRUCT

Bad structure.

0xFFFFFFFD

DRV_FAILURE

Message processing was not successful.

0x00000000

DRV_SUCCESS

Message processing was successful.

0x00000001
