Reset

The Reset key specifies a command that Unimodem will send to the modem to return it to a configuration that will work with MS-DOS–based and Windows-based communications programs.

One command string can be used. For example, 


HKR,, Reset,, "AT&F"

This entry is usually "AT&F" or "ATZ". Some modems may use "AT&F1" or "AT&F2" for IBM PC compatible configurations. It is important to ensure that the modem is in a state that MS-DOS–based communications program expect (for example, make sure echo is on).
