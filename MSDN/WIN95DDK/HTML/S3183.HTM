Libraries

The following list describes the purpose of each library in the Lib directory. Some individual device classes have their own library directories as well.

COMMCTRL.LIB

16-bit Windows common controls functions

DCIMAN.LIB

16-bit DCI client functions

DCIMAN32.LIB

32-bit DCI client functions

DIBENG.LIB

DIB engine functions

LIBENTRY.ASM

Source for LIBENTRY.OBJ

LIBENTRY.OBJ

16-bit code to set up segments for the LibEntry routine

LIBW.LIB

16-bit Windows library

MMSYSTEM.LIB

Multimedia functions

NETDI.LIB

Network installer library

PIFMGR.LIB

Program information file (PIF) functions

RASAPI32.LIB

Remote-access services functions (dial-up networking)

SETUPX.LIB

Setup routines

SHELL.LIB

The shell library (for more information, see the Win32 Software Development Kit documentation)

SPOOLSS.LIB

32-bit print spooler library

VFW.LIB

16-bit Video for Windows functions

VXDWRAPS.CLB

Wrapper functions for calling VxD services from C
