Logical Block Addressing

Logical Block Addressing (LBA) mode for IDE drives is implemented as defined in the AT Attachment Specification. If Windows 95 determines that the IDE drive supports LBA, this mode of addressing is used to access the drive. Once it is determined that the drive supports LBA, Windows 95 verifies that the drive can be accessed correctly using this mode. Windows 95 will not use LBA mode for the drive unless the drive is accessed in LBA mode through the BIOS.
