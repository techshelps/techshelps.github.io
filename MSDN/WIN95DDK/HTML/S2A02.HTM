LowestSendSpeed

Specifies the lowest over-the-wire speed that is acceptable when sending a fax, in bits per second.

Use a single command string. For example, 


HKR, Fax, LowestSendSpeed,,

A value of "0" indicates "use the negotiated settings." The default value is "0". The protocol engine uses this setting when determining whether to continue transmission at a lower speed after transmission errors (after receiving FTT, RTP, etc).
