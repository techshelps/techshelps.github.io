MASM Setup Batch File

The following information describes how to use New-vars.bat to set up MASM version 6.11c.

Syntax

new-vars

Parameters

There are no parameters.

Environment Variables

PATH

The directory containing MASM (usually \Masm611\Bin) is added to the beginning of PATH.

HELPFILES

The path and file name of MASM's help files (usually \Masm611\Help\*.hlp).

TMP

The temporary directory (usually \Windows\Temp).
