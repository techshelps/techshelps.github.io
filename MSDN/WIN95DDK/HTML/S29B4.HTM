Adding or Modifying the DestinationDirs Section

If your INF file has a [DestinationDirs] section, add the following items to it. Many modem INF files do not have a [DestinationDirs] directory until VoiceView support is added; in that case, you will have to create a [DestinationDirs] section and put all the following items in it. 


[DestinationDirs]
Register.Copy    = 17     ; INF File Directory
VV_Sys           = 11     ; VoiceView System Components
VV_Sock_Sys      = 11     ; Windows Sockets System Components
VV_App           = 10     ; File Transfer Application
VV_Help          = 18     ; File Transfer Help
