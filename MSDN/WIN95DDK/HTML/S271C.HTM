GDI API

The system default TrueType EUDC and typeface-specific TrueType EUDC fonts are hidden fonts from applications so that there is no way to handle the EUDC fonts in the applications except the following special GDI functions.

Special applications like EUDC editors and Control Panel must call these special functions whenever the applications directly add/modify/delete the EUDC fonts on the fly without any help of GDI.
