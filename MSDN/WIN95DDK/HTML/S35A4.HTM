mov [Timer_ID], ax

Register a timer callback to be called at periodic intervals. Limited to 32 timers.

  • Returns a 16-bit handle to the timer if successful; otherwise returns zero.

TimerParams

Address of VTDAPI_Timer_Parameters structure.
