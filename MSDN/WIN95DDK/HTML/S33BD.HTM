_GetMutexOwner 


include vmm.inc

VMMcall   _GetMutexOwner, <MutexHandle>
mov     Owner, eax

Retrieves the handle of the thread that owns the specified mutex. Uses C calling convention: Uses EAX, ECX, EDX, and Flags.

  • Returns the handle in EAX or zero if the mutex is unowned.

MutexHandle

Handle of the mutex.
