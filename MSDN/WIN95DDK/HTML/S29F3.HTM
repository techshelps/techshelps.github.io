Settings, SpeakerMode_Off, SpeakerMode_Dial, SpeakerMode_On, and SpeakerMode_Setup

Commands to set the modems speaker mode.

Use a single command string. For example, 


HKR, Settings, SpeakerMode_Off,, "M0"
HKR, Settings, SpeakerMode_Dial,, "M1"
HKR, Settings, SpeakerMode_On,, "M2"
HKR, Settings, SpeakerMode_Setup,, "M3"

If the modem does not support any or all of the SpeakerMode settings, the corresponding registry entry should not be included. The supported commands must be indicated in the SpeakerMode member of the Properties registry entry for the SpeakerMode commands to be used.

See also SpeakerMode in the Properties registry setting documentation.
