Settings, DialSuffix

The characters that will be appended to any dial string, before the Settings, Terminator characters.

Use a single command string. For example, 


HKR, Settings, DialSuffix,, ";"

The DialSuffix key should cause the modem to return to command state after dialing, if possible.

A null DialSuffix can be used (HKR, Settings, DialSuffix,,"") if a dial suffix should not be used. This null suffix would be used for devices which inadvertently listen for dialtone at each dial command. Note that for voice modems, there is a setting in the VoiceProfile registry key which tells Unimodem that it should use a null DialSuffix while dialing in voice mode.
