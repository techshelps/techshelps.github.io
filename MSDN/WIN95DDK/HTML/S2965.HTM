VKD_Enable_Keyboard 


include vkd.inc

VxDcall VKD_Enable_Keyboard

Enables the keyboard after it has been disabled by a previous call to the VKD_Disable_Keyboard service. This service is mainly for use by VMOUSE while it is initializing a PS/2 style mouse. Uses EAX, EDX, and Flags.

  • No return value.

See also VKD_Disable_Keyboard
