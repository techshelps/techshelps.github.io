Compatibility Tests

The hardware compatibility tests reside in the \Ct directory. They were created by the Windows Hardware Quality Labs (WHQL) for testing device drivers to obtain WHQL certification as "Designed for Windows 95."

To install the compatibility tests, run the Ctsetup.exe program in the \Ct directory on the CD-ROM. You can find set up instructions in the document \Ct\Doc\Dt.doc or \Ct\Doc\Ct.hlp.
