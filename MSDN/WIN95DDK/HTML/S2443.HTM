NdiGetOwnerWindow 


RETERR NdiGetOwnerWindow(hndi, lphwnd)
HNDI hndi;    // NDI handle
HWND FAR* lphwnd;    // address of window handle

The NdiGetOwnerWindow function returns the window handle of owner of the NDI in lphwnd.

lphwnd

Points to an HWND to receive the owner window handle.

  • The return value is zero (OK) if successful; otherwise it is a nonzero NDI error value.
