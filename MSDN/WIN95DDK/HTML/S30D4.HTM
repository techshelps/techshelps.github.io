IFSMgr_CheckHeapItem 


IFSMgr_CheckHeapItem(
 void * pMemptr,
 char * pFilename,
 unsigned int Line
 )

This service works only on debug builds of the IFS manager. On non-debug builds, it returns success without doing anything. This service checks the consistency of the heap allocation whose pointer is passed in. If there is any inconsistency, the IFS manager puts out an error message and breaks into the debugger. The debug message is generated in an infinite loop so you cannot continue from this point on.

pMemPtr

Supplies pointer to memory allocated from the heap whose consistency needs to be checked.

pFilename

Supplies pointer to source file that the service is being called from.

Line

Supplies line number in the source file this service is being called from.

  • Returns 1 if any inconsistency was discovered in the heap allocation. Returns 0 if there were no problems with the heap allocation.
