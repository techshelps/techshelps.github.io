Step 7 -- Getting Modem Certification from WHQL

Get the Microsoft Windows Hardware Quality Labs (WHQL) Modem Test Kit which gives step-by-step instructions for submitting your modem and INF file to WHQL for certification testing. During certification testing, you will get technical feedback on problems found during the testing.

When the certification tests are completed, you will get a Test Report.

When the certification tests are passed, you will get

  • A signed Windows Logo License Agreement and a Logo Kit containing camera ready artwork.
  • Addition of the modem product on the Windows 95 and/or Windows NT Hardware Compatibility Lists (HCL).
  • An opportunity to distribute your INF file through the Windows Driver Library (WDL).

To obtain the WHQL information about the latest modem certification tests, send mail to whqlinfo@microsoft.com, or call the fax-back phone number 206-635-2222 and enter the doc # 869 and step-by-step instructions for obtaining WHQL certification for modems will be faxed to you.
