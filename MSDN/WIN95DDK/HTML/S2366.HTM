Using the IShellPropSheetExt::ReplacePage Function

You can use the ReplacePage member function of the IShellPropSheetExt interface to replace property-sheet pages in Control Panels. This technique is explained in the "Shell Extensions" chapter of the Windows 95 Guide to Programming. This section gives only a brief overview of the procedure. Most developers will not replace property-sheet pages in Control Panels, because new functionality is often better exposed in a new page (rather than a replacement page), and because replacing a page can make the built-in Help information incorrect.

The ReplacePage member function is called only by control panels. The uPageID parameter of this function contains the identifier of the property-sheet page to be replaced. The CPLEXT.H header file defines the following identifiers:

Constant

Meaning

CPLPAGE_MOUSE_BUTTONS

Identifier for the mouse control panel's "Buttons" page

CPLPAGE_MOUSE_PTRMOTION

Identifier for the mouse control panel's "Motion" page

CPLPAGE_KEYBOARD_SPEED

Identifier for the keyboard control panel's "Speed" page.
