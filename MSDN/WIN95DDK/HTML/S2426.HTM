NDI_NOTIFY_UNBIND 


NDI_NOTIFY_UNBIND
hndiUnbound = (HNDI)wParam;    // handle of NDI object
wBindType = LOWORD(lParam);    // type of binding

The NDI manager sends NDI_NOTIFY_UNBIND to inform the driver installer that an upper or lower binding has been removed by NdiUnbind. The NDI_NOTIFY_UNBIND message is sent before NdiUnbind returns.

hndiUnbound

Value of wParam. Identifies the NDI object that was unbound from this NDI object.

wBindType

Value of the low-order word of lParam. Specifies which edge of the NDI that hndiUnbound was removed from.

Value

Meaning

NDIBIND_UPPER

Specifies that hndiUnbound was removed from the upper bindings of the given NDI object.

NDIBIND_LOWER

Specifies that hndiUnbound was removed from the lower bindings of the given NDI object.


  • A driver procedure that processes this message should return OK.
