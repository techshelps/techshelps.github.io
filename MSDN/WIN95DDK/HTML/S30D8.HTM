IFSMgr_Get_DOSTime

This service returns the current date and time in DOS format. The time returned is always the local time.

Nothing.

[EAX]

High word contains the date in DOS format. Low word contains the time in DOS format.

[EDX]

Millisecond count remainder in 2 seconds, always a value between 0 and 1999.

Register Usage

EAX, ECX, EDX, Flags.
