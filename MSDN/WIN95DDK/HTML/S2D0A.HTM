I wrote a network driver for Windows for Workgroups, and I want to port it over to Windows 95. What do I do now?

You will probably want to find out several things:

  • Differences between Windows for Workgroups and Windows 95. For more information, see Windows 95 and Windows for Workgroups in Chapter 1 in this Network pamphlet.
  • Miniport overview. For more information, look in the following location on the Microsoft Development Library CD: 


    DDKs
        Windows NT 3.51 DDK
            Network Drivers
                Design Guide
                    Part 2 NDIS Design Overview
                        Chapter 7 Miniport NIC Driver Operations
  • Information on setup and .INF files. For more information, see Chapters 2 and 3 in the Network pamphlet.

Microsoft recommends that you write a miniport driver for Windows 95. However, you will need the Windows NT 3.51 DDK to do so.
