CONFIGMG_CallBack_Device_Driver 


CONFIGRET CONFIGMG_CallBack_Device_Driver(CMCONFIGHANDLER Handler, ULONG ulFlags)

Call back the device driver, once per devnode. with a CONFIGMG_Register_Device_Driver.

Handler

The configuration handler that was registered in a call to the CONFIGMG_Register_Device_Driver function.

ulFlags

Must be zero.

  • Returns CR_SUCCESS if the function is successful. Otherwise, the return value can be CR_INVALID_API or CR_INVALID_FLAG.

A device driver calls this function when it needs to be called back once per devnode.

See also CONFIGMG_Register_Device_Driver
