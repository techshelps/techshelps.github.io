VPICD_Virtual_EOI 


include vpicd.inc

mov     eax, IRQHandle
mov     ebx, VMHandle
VxDcall VPICD_Virtual_EOI

Allows another VxD to send VPICD a virtual EOI for the specified IRQ. The EOI handlers are called like a normal EOI.

  • No return value.

IRQHandle

Handle of the IRQ to set.

VMHandle

Handle of the virtual machine.
