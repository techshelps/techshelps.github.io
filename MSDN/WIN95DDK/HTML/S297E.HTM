Determining the Type of Modem You Are Installing

The types of modems and the types of modem ID strings you can expect to obtain from each are listed in the table below. Once you have determined the type of modem you are installing, you know which types of modem ID strings you can obtain from the modem and use in your INF file.

Modem Type

Types of ID Strings That Can Typically Be Obtained

Non-Plug and Play

Unimodem ID strings only.

Plug and Play Serenum modems (modems that come in both external and internal ISA forms)

Serenum PNP HardwareID string and Unimodem ID string.

Plug and Play ISA modems

ISAPNP HardwareID string, ISAPNP CompatibleID string, and Unimodem ID string.

Plug and Play PCMCIA modems

PCMCIA PNP HardwareID string and Unimodem ID string.
