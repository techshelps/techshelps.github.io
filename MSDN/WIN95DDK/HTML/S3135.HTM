IFSMgr_FSDGetCurrentDrive 


IFSMgr_FSDGetCurrentDrive(
 pioreq pir
 )

This service returns the current drive for the VM and thread context that this request originally came in on. The IOREQ stores the state of the original request.

pir

Supplies the pointer to the IOREQ structure that this call originally came in on.

  • Returns the 1-based current drive number on the original context.
