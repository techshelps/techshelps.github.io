RESET_LATCH_BANK (Function 23)

Call With

EBX: Contains a Windows VM Handle.

EBP: Points to the Windows VM's Client Registers.

Return Values

Save anything that you use.

Remarks

The mini-VDD should restore the bank saved when SET_LATCH_BANK was called. Since manipulation of the latches is a synchronous operation, there is no need to check whether the bank was reset (with RESET_BANK) without the Main VDD being notified.
