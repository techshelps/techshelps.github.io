StopRecord

For modems that use the serial wave device, this command is called after TerminateRecord. It can be used to reset the modem's baud rate, if it was changed for playing.

For modems that use the wave wrapper wave device, this command is called after StartRecord, and should issue all necessary modem commands to stop playing the wave file.

One or more command strings can be used. For example, 


HKR, StopRecord,                1,, "None"
HKR, StopRecord,                2,, "NoResponse"

See also VoiceProfile
