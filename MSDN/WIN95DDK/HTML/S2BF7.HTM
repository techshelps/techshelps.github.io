MCI_ANIM_WINDOW_PARMS

See also MCI_WINDOW 


typedef struct {
    DWORD  dwCallback;
    HWND  hWnd;
    UINT  wReserved1;
    UINT  nCmdShow;
    UINT  wReserved2;
    LPCSTR  lpstrText;
} MCI_ANIM_WINDOW_PARMS;

The MCI_ANIM_WINDOW_PARMS structure contains parameters for the MCI_WINDOW message for animation devices. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

hWnd

Specifies a handle to the display window.

wReserved1

Reserved.

nCmdShow

Specifies how the window is displayed.

wReserved2

Reserved.

lpstrText

Specifies a long pointer to a null-terminated string containing the window caption.
