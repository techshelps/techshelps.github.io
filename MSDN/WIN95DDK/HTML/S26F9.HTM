ImeInquire 


BOOL ImeInquire(LPIMEINFO lpIMEInfo, LPTSTR lpszWndClass, 
    LPCTSTR lpszData)

Handles initialization of IME. It also returns an IMEINFO structure and the UI class name of IME.

  • Returns TRUE if successful, FALSE otherwise.

lpIMEInfo

Address of an IMEINFO structure.

lpszWndClass

Address of a buffer that receives the class name from IME. This class name is IME's UI class.

lpszData

Not used; must be NULL.
