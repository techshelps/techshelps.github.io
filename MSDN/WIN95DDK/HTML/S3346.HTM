Virtual Machine Execution Control

Introduction

A virtual device uses the functions described in this section to suspend, resume, and destroy virtual machines. Also, a virtual device receives the following messages when suspending, resuming, and destroying virtual machines:

Message

Description

Close_VM_Notify

A virtual machine is terminating.

VM_Resume

A virtual machine is resuming after having been suspended.

VM_Suspend

A virtual machine is being suspended.


For a description of these messages, see VMM Messages.

See also Close_VM_Notify, VM_Resume, VM_Suspend
