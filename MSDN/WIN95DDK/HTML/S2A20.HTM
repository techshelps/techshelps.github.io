StartRecord

Command that are issued to the modem to start recording audio (wave files) from the modem. These commands are issued during an active to call to record from the line and are issued when there is no active call to record from the handset.

One or more command strings can be used. For example, 


(1)
HKR,  StartRecord,    1,, "at+vgt=64"
HKR,  StartRecord,    2,, "at+fclass=8"

(2)
HKR, StartRecord,    1,, "at#vrx"
(3)
HKR, StartRecord,    1,, "at#spk=0"
HKR, StartRecord,    2,, "ata"
HKR, StartRecord,    3,, "at+vgr=134"
HKR, StartRecord,    4,, "at#vrx"

See also VoiceProfile
