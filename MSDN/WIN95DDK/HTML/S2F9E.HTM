Definitions

ASPI:

Advanced SCSI Programming Interface.

SRB:

SCSI Request Block: Used by both drivers or application programs to send commands to the ASPI manager.

SCSI:

Small Computer Systems Interface.

DLL:

Dynamic Link Library.

ASPI Client:

An application or driver that uses ASPI to access SCSI devices.

ASPI Manager:

The service layer which processes ASPI requests.
