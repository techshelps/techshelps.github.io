Mouse Application Programming Interface Functions

This section describes the API functions provided by VMOUSE. Note that the following functions are intended only for the system virtual machine:

VMDAPI_GET_MOUSE_INFO

VMDAPI_QUERY_WINDOWED_MOUSE_SUPPORT

VMDAPI_SET_MOUSE_EVENT_CALLBACK

VMDAPI_SET_MOUSE_FOCUS

VMDAPI_UPDATE_MOUSE_STATE

If a nonsystem virtual machine calls the preceding functions, the functions simply set the carry flag and return.
