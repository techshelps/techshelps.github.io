StaticVxD

StaticVxD=name

Identifies a static virtual device. The name is a string specifying the name of a static virtual device, that is, one loaded with the system rather than dynamically loaded. Such names typically start with an asterisk (*). This value replaces the corresponding DEVICE entry in the [386Enh] section of the SYSTEM.INI file.

This value is used in HLM\SYSTEM\CurrentControlSet\Services\VxD\device-class.
