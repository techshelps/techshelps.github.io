EnumPropPages

EnumPropPages=module-and-entry-point-name

Specifies the name of the property page handling function for this device class. The module-and-entry-point-name is a character string that specifies the name of the dynamic-link library containing the handler and optionally the exported name of the handler. If the name of the function is given, it is given after the DLL name and is preceded by a comma.

This value is used in HLM\SYSTEM\CurrentControlSet\Services\Class\device-class.
