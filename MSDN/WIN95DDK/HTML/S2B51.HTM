MODM_PAUSE

Sent to a MIDI output device driver to temporarily pause output. Playback of streams stops but no buffers are marked as done.

  • Returns zero if the operation is successful. Otherwise, it returns an error number. Possible error values include the following:

MMSYSERR_NOTENABLED

The driver failed to load or initialize.


dwParam1

Unused.

dwParam2

Unused.

The driver should halt MIDI playback in the current position. The driver should issue note-off messages all notes that are currently on.
