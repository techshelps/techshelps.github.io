Where can I find more information on the Windows 95 network architecture?

A good introduction to Windows 95 networking is in the Windows 95 Resource Kit. The CD-ROM version of Windows 95 contains the Windows 95 Resource Kit as a help file. The help file is located in the Admin\Reskit\Helpfile directory on the CD-ROM and is called Win95rk.hlp. Open this file and select the Windows 95 Reference section and then Windows 95 Network Architecture.

For a good overview of how the various components interface with the operating system, look in the following location on the MSDN Development Library CD: 


DDKs
    Windows NT 3.51 DDK
        Network Drivers
            Design Guide
                Part 2 NDIS Design Overview

In Part 2 NDIS Design Overview, you will find two chapters, Chapter 7, "Miniport NIC Driver Operations," and Chapter 15, "Full NIC Driver Operations," that give a good description of how a driver interacts with the operating system.
