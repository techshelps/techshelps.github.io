About Windows Hardware Compatibility Labs

The Windows Hardware Compatibility Labs (WHQL) group exists to ensure quality computer hardware, both systems and peripherals, for users of Windows 95. Devices that pass WHQL certification tests are qualified to display the "Designed for Microsoft Windows 95" logo, and are included in the Windows Hardware Compatibility List (HCL). In addition, device drivers are distributed for free through the Windows Driver Library (WDL).

For the information about working with WHQL to certify hardware devices and device drivers, see Device Driver Certification.
