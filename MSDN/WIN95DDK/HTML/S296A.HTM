VKD_Get_Msg_Key 


include vkd.inc

VxDcall VKD_Get_Msg_Key
jz      no_key_avaliable

Returns the next available key from the special message mode input buffer, and removes it from the buffer. The zero flag is clear if a key is available; otherwise, it is set. This service does not block. Uses EAX and Flags.

  • Returns the scan code in AL if the zero flag is clear, and a combination of these flags in AH:

Value

Meaning

MK_Alt

An ALT key is down.

MK_Ctrl

A CTRL key is down.

MK_Extended

The key is an extended key.

MK_Shift

A SHIFT key is down.


VMHandle

Handle of the virtual machine.
