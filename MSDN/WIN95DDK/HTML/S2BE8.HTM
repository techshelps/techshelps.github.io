MCI_SPIN

Starts the device spinning up or down. This command is part of the videodisc command set. The parameters and flags for this message vary according to the selected device.

  • Returns zero if successful. Otherwise, it returns an MCI error code.

DWORD lParam1

The following flags apply to all devices supporting MCI_SPIN:

MCI_NOTIFY

Specifies that MCI should post the MM_MCINOTIFY message when this command completes. The window to receive this message is specified in the dwCallback member of the structure identified by lParam2.

MCI_WAIT

Specifies that the spin up or spin down operation should finish before MCI returns control to the application.

MCI_VD_SPIN_UP

Starts the disc spinning.

MCI_VD_SPIN_DOWN

Stops the disc from spinning.


LPMCI_GENERIC_PARMS lParam2

Specifies a far pointer to the MCI_GENERIC_PARMS structure. (Devices with extended command sets might replace this structure with a device-specific structure.)
