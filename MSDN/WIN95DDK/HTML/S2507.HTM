CONFIGMG_Add_ID 


CONFIGRET CONFIGMG_Add_ID(DEVNODE dnDevNode, PFARCHAR pszID, ULONG ulFlags)

Add an hardware or compatible id during CONFIG_SETUP.

  • Returns CR_SUCCESS if the function is successful. Otherwise, the return value can be CR_INVALID_DEVNODE, CR_INVALID_POINTER or CR_INVALID_FLAG.

dnDevNode

Handle of a device node.

pszID

Points to a id to add.

ulFlags

Must be CM_ADD_ID_HARDWARE or CM_ADD_ID_COMPATIBLE.
