Message Table File

The message table file contains the assembled strings used by the VxD. To adapt a VxD for a given country, you create and assemble a message table source file. The file includes the VMM.INC and MSGMACRO.INC files and sets the constant CREATE_MESSAGE to TRUE before including the message declaration file. Because the message macros do not support creating a message table from a source file that contains references, your message table source file must not include references to the macros.

Note

VxD access to the message table files is not currently implemented.
