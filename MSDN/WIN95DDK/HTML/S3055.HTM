ISP_IOP_create 


#include <isp.h>

typedef struct ISP_IOP_create {
    ISP    ISP_i_c_hdr;       // Standard header
    USHORT ISP_IOP_size;      // size of IOP to allocate, in bytes
    ULONG  ISP_delta_to_ior;  // offset to IOR within IOP
    ULONG  ISP_IOP_ptr;       // address of IOP
    UCHAR  ISP_i_c_flags;     // allocation flags; see below
    UCHAR  ISP_pad2[1];       // Pad to DWORD boundary
} ISP_IOP_alloc, *PISP_IOP_alloc;

Contains information for a ISP_CREATE_IOP function.

ISP_i_c_flags

Allocation flags. Can be one of these values:

ISP_M_FL_EXTERNAL_IOP

Register the IOP instead of allocating it.

ISP_M_FL_INTERRUPT_TIME

Must be set if allocation is at asynchronous time.

ISP_M_FL_MUST_SUCCEED

If set, IOS will not fail the allocation.

ISP_M_FL_PERSISTENT_IOP

Allocation is for a long time IOP.

ISP_M_FL_SMART_ALLOC

Heap allocation if not MS-DOS pager.

ISP_M_FL_USE_SYSTEM_HEAP

Don't allocate from IOS heap.


See also IOP, ISP
