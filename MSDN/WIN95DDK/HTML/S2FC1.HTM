Compatibility With Enhanced BIOS Features

Ontrack recommends that the FASTDISK driver perform all BIOS detection algorithms prior to looking for XBIOS. This allows future versions of XBIOS to incorporate standardized BIOS enhancement features without requiring changes to the FASTDISK driver. If the FASTDISK driver looks for XBIOS first, there may be features in XBIOS that the FASTDISK driver will miss.
