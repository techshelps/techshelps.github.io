WNetCloseEnum

Closes an enumeration. 


DWORD WNetCloseEnum (
    HANDLE hEnum
    );

hEnum

This must be a handle obtained from WNetOpenEnum call.

  • WN_SUCCESS if the call is successful. Other error codes include:

WN_FUNCTION_BUSY

The MPR or NP is busy (possibly initializing). The caller should retry.

WN_NO_NETWORK

network is not present. This condition is checked for before hEnum is tested for validity.

WN_BAD_HANDLE

hEnum is not a valid handle.

WN_EXTENDED_ERROR

a network specific error occurred. WNetGetLastError should be called to obtain a description of the error.
