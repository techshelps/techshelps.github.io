Cl2lsEx

Use a single command string. For example, 


HKR, Fax, Cl2lsEx,,

If non-zero, driver will not issue AT+FDIS?, AT+FDIS=..., and AT+FPTS=... commands, and expects a non-standard FPTS response: FPTS:1 indicating good page and FPTS:1,2,0,0 indicating a bad page. The default is modem-specific, but typically 0.
