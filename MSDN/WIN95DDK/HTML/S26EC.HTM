IMN_CLOSECANDIDATE

This message is sent when an IME is about to close a candidates window. It notifies an application of the end of candidate processing. The UI window destroy a candidate window when it processes this message.

  • No return value.

lParam

Shows which candidate list should be updated. For example, if bit0 is 1, the first candidate list should be updated. If bit31 is 1, the 32nd candidate list should be updated.
