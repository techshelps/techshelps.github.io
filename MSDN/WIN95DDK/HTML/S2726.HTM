GAIJIPHYSICALFONTSIZE

Get the size of bitmap specified by the index.

  • Returns 1 is upon success, or zero for failure. Windows 95 always returns zero.

lpInData

The low-order word is the index to the system default WIFE EUDC font.

The high-order word always passes zero.

lpOutData

A pointer to the POINT structure, which will contain the size of bitmap.
