PageFile_Init_File 


VxDcall PageFile_Init_File
mov     max_size, ebx      ; maximum swap file size, in pages
mov     current_size, eax  ; current swap file size, in pages
or      ebx, ebx
jz      error_handler      ; ebx = 0 on error

Called by PAGESWAP to initialize the swap file. Can only be called at initialization time. Uses EAX, EBX, ECX, EDX, Flags.

  • Returns with zero in the EBX register if an error occurs. Otherwise, EAX contains the swap file's current size and EBX its maximum size, in pages.
