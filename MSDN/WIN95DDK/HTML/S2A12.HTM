EnableCallerID

Command to enable caller ID reporting.

One or more command strings can be used. For example, 


HKR, EnableCallerID, 1,, "at#cid=1"

If the modem does not have ability to report Caller ID, this entry may be omitted. The VoiceProfile registry entry can specify if the modem supports Caller ID. If caller ID is enabled, make sure that caller ID responses are in the Responses registry settings.

See also VoiceProfile registry entry.
