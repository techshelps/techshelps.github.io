VCache_Deregister

Deregisters a cache and its associated resources. Uses EAX, EBX, ECX, EDX, EDI, ESI, and flags.

  • No return value.

On entry, AH contains an FSD ID of the cache to deregister.

An FSD must call this function to tell VCache that it no longer needs any cache resources.

Each cache ID is unique. They are not recycled.
