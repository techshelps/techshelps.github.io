Pen Calibration Application

The pen calibration application extends the capability of the Pen icon in the Windows 95 Control Panel by providing a user interface for calibrating the pen device. This application enables the user to calibrate non-opaque digitizers with 640x480 resolution and obtains the calibration information required by the set of digitizers supported by the pen Unidriver. To find out how to change this application for digitizers of different resolutions or digitizers whose drivers have different calibration data requirements, see About the Pen Calibration Application.
