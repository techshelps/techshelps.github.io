About System-Internal Memory Management

System-internal memory management services are used internally, and should not be used by virtual devices. They are as follows:

Service

Description

_GetDemandPageInfo

Retrieves information used for demand paging.

_MMGR_Toggle_HMA

Enables or disables the HMA.

_SetLastV86Page

Sets last page in V86 address space.

Set_Physical_HMA_Alias

Sets the physical HMA alias.
