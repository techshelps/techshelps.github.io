VCache_AdjustMinimum

Adjusts the client's minimum buffer quota. Uses EAX, EBX, ECX, EDX, EDI, ESI, and flags.

  • No return value; however, the Carry flag is cleared if the quota is adjusted. If the quota is not adjusted, the Carry flag is set.

On entry, AH contains the FSD ID.

On entry, ECX contains the new buffer quota.
