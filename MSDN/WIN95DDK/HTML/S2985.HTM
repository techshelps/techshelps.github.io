Step 2 -- Creating The Modem INF File

Once you have validated your modem ID strings and have some idea how you are going to use the ID strings in your modem INF file, it's time to create the INF file. There are many examples of modem INF files in the WINDOWS\INF subdirectory of your Windows 95 computer. You can either edit one of those INF files to work with your modem, or build your INF file from scratch.
