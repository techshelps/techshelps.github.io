Geometry From the Ontrack Disk Manager XBIOS

Windows 95 supports the Ontrack Disk Manager XBIOS driver. If Windows 95 detects version 6.03 or later of the driver (both the Master Boot Record and CONFIG.SYS driver versions), the Windows 95 IDE driver obtains the actual geometry and sector skew factor from the driver using the driver's function interface. For more information about the interface supported by this driver, see Ontrack Computer Systems.
