DVM_STREAM_FREEBUFFER

Notifies a video capture driver to free data buffers on the capture device that were allocated using the DVM_STREAM_ALLOCBUFFER message.

  • Returns DV_ERR_OK if successful or an error code otherwise. The following errors are defined:

DV_ERR_INVALHANDLE

Indicates the specified device handle is not valid.

DV_ERR_NOTSUPPORTED

Indicates the device does not have on-board memory.


dwParam1

Address of the buffer to be freed.

dwParam2

Not used.
