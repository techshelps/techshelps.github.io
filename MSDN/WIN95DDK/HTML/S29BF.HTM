Understanding the Primary Commands Used by Unimodem/V

Unimodem/V uses the command values in the Init, Monitor, EnableCallerID, EnableDistintiveRing, VoiceDialNumberSetup, VoiceAnswer, and VoiceDialNumberSetup registry keys to control the initialization of your voice modem. The relationships between these commands are summarized in the following state transition diagram.

The TAPI functions called, and their parameters, when Unimodem/V performs the commands stored in these registry keys are listed in the following table, along with their parameters.
