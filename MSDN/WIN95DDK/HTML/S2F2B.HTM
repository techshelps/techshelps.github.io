How the Installation Process Works

Assuming that the printer INF file is modified as outlined above, the printer setup code will detect all dependent files that should be copied to the \Windows\System\Color directory and call the UpdateICMRegKey function to attempt to install the color profile in the registry. No other action is taken by the printer setup code. For the profile to be installed correctly, the profile contents must contain the tags described in Overview of ICC 3.0 Profile Tags Used by Windows 95.
