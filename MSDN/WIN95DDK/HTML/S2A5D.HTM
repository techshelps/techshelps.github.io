VMDAPI_GET_MOUSE_INFO 


include vmd.inc

mov     ax, VMDAPI_GET_MOUSE_INFO
call    pVMD_API_Entry
jc      errorhandler
mov     MouseType, al
mov     IRQ, ah
mov     PortNumber, cx

Retrieves information about the mouse hardware. Uses AX, CX, and Flags. Clears the carry flag upon success, sets it upon failure.

  • Returns the mouse-type identifier in AL, the IRQ that the mouse is using in AH, and the port to which the mouse is connected in CX (if the mouse is a serial or InPort mouse). The mouse-type identifier can be one of the following values:

VMD_Type_Bus

VMD_Type_PS2

VMD_Type_Serial

VMD_Type_Serial_w_port

VMD_Type_InPort

VMD_Type_Undefined

VMD_Type_HP
