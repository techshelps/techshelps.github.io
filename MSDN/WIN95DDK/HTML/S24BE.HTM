CurrentConfig

CurrentConfig=profile

Identifies the current hardware profile. The profile is a 4-digit, hexadecimal number that corresponds to one of the profile keys in the IDCONFIGDB branch. The system sets this number when it starts or when the user docks, undocks, or forces a switch between hardware profiles by using the the docking control panel.

This value is used in HLM\SYSTEM\CurrentControlSet\Control\IDCONFIGDB.
