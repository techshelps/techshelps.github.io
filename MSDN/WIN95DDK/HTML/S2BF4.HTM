MCI_ANIM_RECT_PARMS

See also MCI_PUT, MCI_WHERE 


typedef struct {
    DWORD  dwCallback;
    RECT  rc;
} MCI_ANIM_RECT_PARMS;

The MCI_ANIM_RECT_PARMS structure contains parameters for the MCI_PUT and MCI_WHERE messages for animation devices. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

rc

Specifies a rectangle.
