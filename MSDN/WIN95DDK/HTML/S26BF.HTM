ImmLockIMC 


LPINPUTCONTEXT WINAPI ImmLockIMC(HIMC hIMC)

Retrieves the address of the INPUTCONTEXT structure for the input context. IME uses the address to access the input context. This function increases the lock count for IMC.

  • Returns the address of the INPUTCONTEXT structure if successful, NULL otherwise.

hIMC

Handle of the input context.
