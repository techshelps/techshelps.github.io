VDMAD_Set_EISA_Adr_Mode 


include vdmad.inc

mov eax, Channel
mov ebx, DMA_Handle
mov cl, Mode
VxDcall VDMAD_Set_EISA_Adr_Mode

Sets the EISA extended mode. Uses flags

  • No return value.

Channel

Specifies the channel number. It must be in the range 0 to 7.

DMA_Handle

Specifies the DMA handle.

Mode

I/O mode. It can be one of the following values:

0

8-bit I/O, with count in bytes

1

16-bit I/O, with count in words and address shifted

2

32-bit I/O, with count in bytes

3

16-bit I/O, with count in bytes
