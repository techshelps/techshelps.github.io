InactivityScale

Scale for the InactivityTimeout. InactivityTimeout * InactivityScale = inactivity timeout in seconds.

A single DWORD hexadecimal value. For example, 


HKR,, InactivityScale, 1, 0a,00,00,00

If inactivity scale is omitted from an INF a default value of 10 seconds will be assigned by Win95. It is important to verify the InactivityScale and InactivityTimeout.

See also Settings, InactivityTimeout registry setting and the InactivityTimeout value in Properties setting.
