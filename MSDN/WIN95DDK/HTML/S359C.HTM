VTDAPI_Begin_Min_Int_Period 


; Obtain the entry point address in VTDAPI_Addr
mov cx, InterruptPeriod
mov ax, VTDAPI_Begin_Min_Int_Period
call VTDAPI_Addr

Sets the minimum interrupt period being used globally. Performs boundary checking on the structure elements passed, and uses VTD to set the minimum timer interrupt period.

  • Returns TIMERR_NOERROR in AX if successful, otherwise returns TIMERR_NOCANDO.

InterruptPeriod

The requested interrupt period in milliseconds.
