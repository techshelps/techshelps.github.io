VTD_Enable_Trapping 


include vtd.inc

mov ebx, VMHandle
VxDcall VTD_Enable_Trapping

Reenables timer I/O port trapping after the VTD_Disable_Trapping service has been called. This service must be made once for every call to VTD_Disable_Trapping. Only when every disable call has been matched by a call to this service will port trapping be re-enabled. Uses Flags.

  • No return value.

VMHandle

Virtual machine handle.
