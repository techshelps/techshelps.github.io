NPSGetProviderName 


DWORD NPSGetProviderName(
  HPROVIDER hProvider,
  LPCSTR FAR * lpszProviderName
  );

This service allows a provider to obtain its name from MPR for use in supporting services.

hProvider

The handle returned from a previous call to NPSGetNPHandle.

lpszProviderName

Returns a pointer to the NP's name. This pointer is valid for the duration of the NP DLL being loaded.

  • WN_SUCCESS if the call is successful.

WN_BAD_HANDLE

hProvider is not a valid NP handle.
