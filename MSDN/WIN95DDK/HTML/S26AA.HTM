IME Interface

In Windows 95, IMEs are provided as Dynamic Link Libraries (DLLs) like device drivers. Input Method Manager (IMM) should handle all of the installed IMEs. Because IMEs are changeable at running time without rebooting, IMM will have a structure to maintain all entry points of each IME.

The list of IME functions is the description of all of Far East common IME function functions. These functions should not be called by an application directly.
