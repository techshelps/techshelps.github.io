Get_Nest_Exec_Status 


include vmm.inc

VMMCall Get_Nest_Exec_Status

Retrieves information about whether the current thread is in nested ring 3 execution. Uses EAX, ECX, EDX, and Flags.

  • Returns with the Z-flag set if the thread is not in nested execution. Otherwise, returns the following:

Register

Meaning

EAX

Original ring 3 EIP

EDX

Original ring 3 CS

ESI

Original ring 3 SS:SP

ECX

Ring 0 stack pointer returned from Resume_Exec or Exec_Int, if ; waiting for completion of ring 3 code.


There should be no need for a virtual device to call this service.
