What chapters are in the Network pamphlet and what is discussed there?

Chapters

1 Windows 95 Network Device Drivers: An overview of the Network pamphlet, differences between Windows 95 and Windows for Workgroups and between Windows 95 and Windows NT, and changes to the networking functions for Windows 95.

2 Plug and Play Network Drivers: Plug and Play issues involving network drivers.

3 Network Driver Installer: Writing of .INF files for network drivers.

4 Multiple Provider Router: Discussion of the Multiple Provider Router layer, which unifies the interface to the network redirector across multiple network protocols.

5 Network Provider SPI: Description of the Network Provider Service Provider Interface (SPI). Different network vendors can write network providers to provide a consistent interface to access network resources, such as printers and file servers.

6 Windows 95 Password Providers: Description of a common means of using passwords with network resources.

7 Windows 95 Session Management Modules: How to add new drivers to Dial-Up Networking.
