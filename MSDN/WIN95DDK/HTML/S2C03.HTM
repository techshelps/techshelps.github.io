MCI_PLAY_PARMS

See also MCI_PLAY 


typedef struct {
    DWORD  dwCallback;
    DWORD  dwFrom;
    DWORD  dwTo;
} MCI_PLAY_PARMS;

The MCI_PLAY_PARMS structure contains parameters for the MCI_PLAY message. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

dwFrom

Specifies the position to play from.

dwTo

Specifies the position to play to.
