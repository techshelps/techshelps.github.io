AbortPlay

AbortPlay is the command issued to the modem to stop playing the audio data immediately. It does not play what is currently in it's buffer.

Use one or more command strings. For example, 


HKR,, AbortPlay,, "<h10><h18>at"

AbortPlay is called after StartPlay, and before StopPlay.

This command is only called for modems that use the serial wave device. It is not used for modems that use the wave wrapper.

See also StartPlay, StopPlay, TerminatePlay registry entries.
