Constants

The following set of tables list the constants defined by the pen Unidriver. These constants are defined in the header file \PEN\INC\PENDRV.H.

Each table below lists a different functional group of constants. The functional groups are:

  • Driver return values.
  • Data type values (which indicate the various types of pen packets).
  • OEM return values.
  • Pen driver client return values.
  • Hardware flags.
  • Miscellaneous pen Unidriver constant definitions.
