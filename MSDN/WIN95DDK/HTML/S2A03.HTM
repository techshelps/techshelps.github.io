Cl1FCS

Specifies the FCS checking behavior of the class1 driver.

Use a single command string. For example, 


HKR, Fax, Cl1FCS,,

0

Check (default and ONLY value for Win95 4.0.0.950)

1

Don't check -- assume there is no FCS

2

Don't check -- assume there is an FCS (default for Win95 4.0.0.951 and later)


After Windows 95 shipped, Microsoft was notified that there were several newer modems that did not report the correct FCS bytes. Binary file transfer will not work for these devices. The fix requires version 4.0.0.951 of the following fax components: AWFXIO32.DLL and AWCL1_32.DLL. In NT-SUR and future Windows 9x platforms, we fix this problem.
