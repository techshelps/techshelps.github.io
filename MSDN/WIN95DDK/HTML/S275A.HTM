_VCOMM_Map_Win32DCB_To_Ring0 


include vcomm.inc

mov     eax, Win32_DCB
mov     edx, Ring0_DCB
VxDcall _VCOMM_Map_Win32DCB_To_Ring0

The _VCOMM_Map_Win32DCB_To_Ring0 function can be called to convert a Win32-style DCB to a Ring 0 DCB.

  • No return value.

Win32_DCB

Address of a Win32-style DCB structure.

Ring0_DCB

Address of a Ring0 (or Win16-style) DCB structure.
