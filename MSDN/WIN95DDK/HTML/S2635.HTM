Certifying Your Device Driver

Windows Hardware Compatibility Labs (WHQL) activities ensure that quality computer hardware, both systems and peripherals, is available to users of Windows® 95. Only hardware that passes WHQL certification tests are qualified to display the "Designed for Microsoft Windows 95" logo. This logo is a customer's assurance that your hardware and associated device drivers will work with the Windows 95 family of products.

After your hardware is certified to receive the WHQL logo, you will be given a detailed test report. In addition, your hardware will be added to the Windows Hardware Compatibility List (HCL), and your hardware driver will be distributed free through the Windows Driver Library (WDL). To see the current HCL, access the WHQL-administered Web site at the URL http://www.microsoft.com/windows. Then use the Windows 95 Compatible Products link followed the by Hardware Compatibility List link.

To see the drivers currently in the WDL, use your Web browser to access the FTP site at the URL ftp://ftp.microsoft.com/services/whql.

In order to have you hardware tested by WHQL at Microsoft Corporation, follow the step-by-step instructions for your hardware category. The list of hardware categories supported by WHQL in Quarter 1, 1996 is listed below:

  • Audio adapters
  • Computer systems
  • Display adapters and monitors
  • Input devices
  • Microprocessors
  • Modems
  • Network adapters
  • Printers
  • Storage adapters and devices
  • Video capture adapters
  • Miscellaneous (hardware categories not listed above)

WHQL distributes step-by-step instructions for having WHQL test devices and drivers for each class of hardware listed above. You can obtain these instructions in several ways:

  • As downloadable files from the Internet FTP server ftp://ftp.microsoft.com/services/whql.
  • As downloadable files at the Web site http://www.microsoft.com/windows/thirdparty/hcl/home.htm.
  • As fax-back documents by calling the U.S. number 206-635-2222.
