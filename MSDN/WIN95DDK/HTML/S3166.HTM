System Requirements

The minimum requirements for using the DDK include this software and hardware:

  • Microsoft Windows 95 operating system
  • Microsoft® Win32® SDK
  • Microsoft Visual C++ version 2.0 (or equivalent 32-bit compiler) for x86
  • Microsoft MASM version 6.11c (or equivalent assembler)
  • CD-ROM drive
  • 16 megabytes (MB) RAM
  • 80MB hard disk space for the full installation (100MB more for all the compatibility tests).
