AnsiToOem 


WORD AnsiToOem(LPSTR lpSrc, LPSTR lpDst);

Translates a string in the ANSI character set to a string in the OEM character set. For most computers, the OEM character set is code page 437.

  • Return 0FFFF in AX if successful.

lpSrc and lpDst

Address of the input ANSI string, and address of the output OEM string.

The export ordinal for this function is 5.

The lpSrc and lpDst parameters may point to the same buffer.

The code page used to translate the character depends on the current code page. For most drivers, the default translation is code page 437 (the standard U.S. IBM-PC character set).

Strings longer than 64K may be translated.

See also AnsiToOemBuff, Enable, OemToAnsi
