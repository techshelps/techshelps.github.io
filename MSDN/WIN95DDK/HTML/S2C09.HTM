MCI_STATUS_PARMS

See also MCI_STATUS 


typedef struct {
    DWORD  dwCallback;
    DWORD  dwReturn;
    DWORD  dwItem;
    DWORD  dwTrack;
} MCI_STATUS_PARMS;

The MCI_STATUS_PARMS structure contains parameters for the MCI_STATUS message. When assigning data to the members in this structure, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the members.

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

dwReturn

Contains the return information on exit.

dwItem

Identifies the capability being queried.

dwTrack

Specifies the length or number of tracks.
