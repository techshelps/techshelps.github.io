Adding CopyFiles Sections

CopyFiles sections in an INF file copy files from an installation disk to the appropriate paths on the Windows 95 computer hard disk. To add VoiceView support for your modem, you must add the following five CopyFiles-type sections to your modem INF file. Do not change the CopyFiles sections listed below in any way.

Note also that these new CopyFiles sections must be referenced in a CopyFiles= item in the Install section (see Modifying the Install Section for an example of such a CopyFiles= item). 


[VV_Sys]
fte.dll
vvexe32.exe
wsvv.vxd

[VV_Sock_Sys]
wsock32.dll
wsock.vxd
vtdi.386

[VV_App]
filexfer.exe

[VV_Help]
filexfer.hlp
filexfer.cnt
