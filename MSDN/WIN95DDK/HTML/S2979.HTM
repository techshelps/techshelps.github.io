Files Only on the FTP Site (Not in the Windows 95 DDK)

Of the 24 files on the FTP site at the time of this Windows 95 DDK update release, 11 are not in the update release of the DDK. The documentation file Voicemod.doc is extremely important. It documents the changes to the Unimodem specification for voice modems. If you are working on voice modems, you must read this document.

File name

Description

Readme.txt

Describes files currently on the FTP site.

Awcfg32.dll

Debug DLL for FAX testing.

Info.xls

Feedback form for modem vendors to fill out.

Modemdev.ppt

Powerpoint presentations from the Modem Developers' Conference, help 11/3/95 in Seattle, WA.

Modemid.txt

Documentation for Modemid.exe.

Pnp.doc

The Plug and Play documentation from the Windows 95 DDK, which includes a chapter on how to use the Infedit tool and a reference for all types of sections and items that can be in a Windows 95 INF file.

Pnpcom.doc

External COM port Plug and Play specification.

Procalc.zip

Profile Calculator utility kit; Profile Calculator is a tool that presents a graphical user interface used to build and analyze VoiceProfile key values, which are hexadecimal values that represent bit vectors that can put in an INF file, such as the VoiceProfile bit vector.

Unimodv.doc

Unimodem/V specification.

Urgent.doc

Proposal for support of modems in future releases of Windows and in Windows NT.

Voicemod.doc

Unimodem/V specification update; the specification for changes to Windows 95 Unimodem to support voice modems. If you are working on voice modems, you must read this document.
