CONFIGMG_Read_Registry_Log_Confs 


CONFIGRET CONFIGMG_Read_Registry_Log_Confs(DEVNODE dnDevNode, ULONG ulFlags)

Retrieves logigal configuarations from the registry.

  • Returns CR_SUCCESS if the function is successful. Otherwise, the return value can be CR_INVALID_DEVNODE, CR_INVALID_FLAG, CR_OUT_OF_MEMORY, CR_INVALID_DATA, or CR_REGITRY_ERROR.

dnDevNode

Handle of a device node.

ulFlags

Must be zero.

This API is for the root enumerator, MCA and EISA bus driver. Instead of hardwarely detect the possible logical configurations, those enumerators rely on this API to read logical configurations from the hardware LogConfig subkey into the BASIC_LOG_CONF list.
