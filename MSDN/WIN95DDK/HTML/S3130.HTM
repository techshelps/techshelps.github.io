IFSMgr_CDROM_Detach 


IFSMgr_CDROM_Detach(
 unsigned long Drive,
 unsigned long VRP
 )

This service is called by the CD-ROM driver to detach a drive after a preceding attach call. The IFS manager looks for the drive with the correct system volume handle and then performs the detach on that volume.

Drive

Supplies the 0-based drive number of the drive the detachment needs to be done for.

VRP

Supplies the system volume handle of the volume to be detached.

  • Nothing.
