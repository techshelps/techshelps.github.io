CallerIDPrivate, CallerIDOutSide

Characters that the modem returns for Caller ID to indicate that the information is not sent because it has been blocked (private) or is out of area.

Use a single line - can be one or more characters. For example, 


HKR,, CallerIDPrivate,,"P"
HKR,, CallerIDOutSide,,"O"

These entries are case-sensitive.

See also EnableCallerID registry entry.
