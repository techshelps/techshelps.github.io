VMCPD_Get_Version 


include vmcpd.inc

VxDcall VMCPD_Get_Version
mov [Major], ah
mov [Minor], al
mov [Coprocessor_Flag], ecx

Retrieves the VMCPD version number. Uses EAX, ECX, and Flags.

  • Returns the version number in EAX. AH contains the major version number; AL contains the minor version number. ECX contains one of these values:

0

No coprocessor

2

80287

3

80387
