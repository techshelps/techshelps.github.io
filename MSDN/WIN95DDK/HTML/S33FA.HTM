End_Touch_1st_Meg

Disables the first 1 megabyte of memory for the current virtual machine. Virtual devices use this macro in conjunction with the Begin_Touch_1st_Meg macro to examine and modify addresses in the first megabyte. This macro is intended to be used for debugging purposes.

  • No return value.

This macro does not generate code unless the DEBUG symbol is defined. If the macro generates code, the code calls the Disable_Touch_1st_Meg service.

See also Begin_Touch_1st_Meg, Disable_Touch_1st_Meg
