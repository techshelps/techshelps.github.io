Building the Sample INF File

This topic walks through the process of using Infedit to build each of the INF file sections necessary to satisfy the requirements listed in Sample INF File Requirements. The procedures used to build several types of INF file sections with Infedit are shown.

  • To see an example of building a Manufacturer section, see Completing the Manufacturer Section.
  • To see an example of building a Logical Configuration section, which specifies the resource requirements of a device such as IRQs and I/O addresses, see Specifying Resource Requirements.
  • To see an example of building an Add Registry section, see Adding Registry Entries.
  • To see an example of building a Copy Files section, see Copying Files.
  • To see an example of editing an INF file header section, see Completing the INF File Header Section.
  • To see information about adding strings to a Strings section, see Adding Strings to the Strings Section.
