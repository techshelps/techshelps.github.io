Introduction to Drivers and Image Color Matching

This document provides an overview of how device color characterization profiles are installed, referenced in the registry, and selected and associated with printer drivers by Microsoft® Windows® 95 at run time. The Image Color Matching (ICM) technology provided with Windows 95 uses the industry standard ICC (International Color Consortium) version 3.0 specification for device color characterization profiles together with some Windows-specific optional tags in the profiles themselves. This document is not intended to be a replacement for the Windows 95 DDK documentation, but rather as a supplemental summary of those areas effecting printer support and ICM.
