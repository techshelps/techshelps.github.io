Documentation

The DDK documentation comes in two formats: Windows help (.hlp) files and Microsoft Word (.doc) files . If you want to print the documentation, you should use Microsoft Word for Windows or Microsoft Word Viewer to print the documentation because printing from Windows help breaks pages between each topic.

.HLP File

The entire DDK documentation set is contained in one help file, Ddk.hlp. You can use the full-text search feature of Windows help to locate topics of interest.

.DOC Files

The documentation is also available as Microsoft Word for Windows files for printing. You can obtain the Microsoft Word Viewer at no charge from Microsoft's Web site to print documents of interest.

There are separate documents for three of the four parts of the documentation (Getting Started, Programmer's Guide, and Kernel Services Guide). Of these, the Programmer's Guide is also available as separate files. The Design Guide is not available as a separate document because it is unlikely that you would print out all of the sections.
