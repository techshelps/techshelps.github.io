Providing a Device Information File

The setup disk for your pointing device must contain a device information file (.INF). The file provides information that the Windows 95 operating system needs to install your device driver software and to support your pointing device. The Windows 95 device driver kit (DDK) includes a sample mouse minidriver and accompanying device information file.
