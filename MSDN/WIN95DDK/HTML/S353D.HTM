VCache_CheckAvail

Checks the availability of a specified number of buffers. Uses EAX, EBX, ECX, EDX, and flags

  • Returns the number of available buffers in EAX. In addition, the Carry flag is clear if the specified number of buffers is available. If fewer buffers are available, the Carry flag is set.

On entry, AH contains the FSD ID.

On entry, ECX contains the number of buffers needed.

An FSD can call VCache_CheckAvail to decide whether to cache a data transfer or use a direct transfer instead.
