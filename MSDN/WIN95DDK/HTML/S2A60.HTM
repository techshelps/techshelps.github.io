VMDAPI_QUERY_WINDOWED_MOUSE_SUPPORT 


include vmd.inc

mov     ax, VMDAPI_QUERY_WINDOWED_MOUSE_SUPPORT
call    pVMD_API_Entry
jc      errorhandler

Discovers whether VMOUSE supports the mouse in nonsystem virtual-machine applications running in a window. Uses Flags.

  • Clears the carry flag, indicating that the mouse is supported in windowed, non-system virtual-machine applications.
