CompatibleIDs

CompatibleIDs=device-id

Identifies a list of compatible device IDs for a device that can be used for purposes of loading a default driver when a specific one from the vendor is not available. The device-id is a multiple-string value that specifies one or more valid device identifiers.

This value is used in HLM\ENUM\enumerator\device-id\instance and HLM\ENUM\ROOT\device-id\instance.
