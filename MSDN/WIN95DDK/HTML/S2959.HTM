KBINFO 


typedef struct tagKBINFO {
    BYTE  Begin_First_range;  /* see below */
    BYTE  End_First_range;    /* see below */
    BYTE  Begin_Second_range; /* see below */
    BYTE  End_Second_range;   /* see below */
    int   StateSize;          /* size, in bytes of ToAscii state block*/
} KBINFO;

Contains information about the keyboard.

Begin_First_range

Low value in the range of valid lead-byte values for double-byte character sets (DBCS).

End_First_range

High value in the range of valid lead-byte values for double-byte character sets (DBCS).

Begin_Second_range

Low value in the range of valid second-byte values for double-byte character sets (DBCS).

End_Second_range

High value in the range of valid second-byte values for double-byte character sets (DBCS).
