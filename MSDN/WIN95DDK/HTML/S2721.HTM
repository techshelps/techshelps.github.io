GAIJIFONTSIZE

Return the size of the typical EUDC character in the system default WIFE EUDC font.

  • Returns zero for failure. Windows 95 always returns zero.

lpInData

Not used.

lpOutData

A pointer to the POINT structure, which will contain the size of the typical EUDC character in the system default WIFE EUDC.

This subfunction returns the size that is specified by the StandardGaijiSize in the [UserFont] section of the userfont.ini file.

The format of the StandardGaijiSize key is as follows (units are physical dots):

StandardGaijiSize=<x-size>,<y-size>

The contents of a standard userfont.ini file is as follows:

[UserFont]

StandardGaijiSize=40,40
