\Inc16

The Inc16 directory contains include files required for building 16-bit drivers. The following list shows the use of each include file.

AVICAP.H

Video (.avi) capture

AVIFILE.H

Video file reads

AVIFMT.H

Video file format

AVIIFACE.H

Component Object Model (COM) interface to .avi files

CDERR.H

Common dialog box error-return codes

CMACROS.INC

MASM macros for interfacing to high-level languages

COGUID.H

Globally unique identifiers (GUIDs) for common objects (included by Compobj.h)

COLORDLG.H

Common dialog box color control id numbers

COMDEV.H

Miscellaneous definitions for communication devices

COMMCTRL.H

COM interface for the Windows common controls

COMMDLG.H

Common dialog box functions, types, and definitions

COMMON.VER

Resource definitions used to create version resources

COMPDDK.H

Implementiation of installable video compressors

COMPMAN.H

Communication with installable video compressors and decompressors

COMPOBJ.H

COM definitions

CPL.H

Control Panel extension dynamic-link library (DLL) definitions

CUSTCNTL.H

Support for custom controls

DCIDDI.H

Definitions for Microsoft- and Intel-defined Display Control Interface (DCI)

DCIMAN.H

Support for DCI clients

DDE.H

Dynamic Data Exchange structures and definitions

DDEML.H

Dynamic Data Exchange Management Library

DIBENG.INC

MASM device-independent bitmap (DIB) engine interface definitions

DIGITALV.H

Media Control Interface (MCI) digital-video command set

DIGITALV.RC

Resource definitions for MCI digital-video command set

DISPATCH.H

OLE Automation definitions

DISPDIB.H

Support for the DisplayDib function

DLGS.H

Common dialog box element identifier numbers

DRAWDIB.H

Routines for drawing device-independent bitmaps (DIBs) to the screen

DRIVINIT.H

Obsolete (use Print.h instead)

ERROR.H ,ERROR.INC

Microsoft® MS-DOS® error codes

FCE.H

Type definitions and function prototypes for Forced Configuration Edit (FCE) support

GDIDEFS.INC

Graphics device interface (GDI) definitions for device drivers

IDF.H

Format of the .IDF files (MIDI instruments)

INSTVER.H

SDK version-checking installation

LZDOS.H

Obsolete: Replaced by

#include <lzexpand.h>
