NPCloseEnum

Closes an enumeration. 


DWORD NPCloseEnum (
  HANDLE hEnum
  );

hEnum

This must be a handle obtained from NPOpenEnum call.

  • WN_SUCCESS if the call is successful. Otherwise, an error code is returned, which may include:

WN_NO_NETWORK

The network is not present. This condition is checked for before hEnum is tested for validity.

WN_BAD_HANDLE

hEnum is not a valid handle.
