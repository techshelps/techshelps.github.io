Modifying the Install Section

The following items must be added to the INF file section that installs the modem. The section for installing a modem will need to be modified to incorporate the following additions. These additions will modify the registry, copy required files to the appropriate directory, and update folders to support Windows 95 VoiceView functionality. 


AddReg      = VoiceView_Modem, VoiceView_Socket
CopyFiles   = VV_Sys, VV_Sock_Sys, VV_App, VV_Help
UpdateInis  = VView.Inis
Uninstall   = VoiceView_remove
