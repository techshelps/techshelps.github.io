ScsiPortConvertPhysicalAddressToUlong 


ULONG ScsiPortConvertPhysicalAddressToUlong(Address)
IN SCSI_PHYSICAL_ADDRESS  Address;

Truncates a SCSI_PHYSICAL_ADDRESS to a ULONG.

  • Returns the low-order part of the physical address.

Address

Specifies a value of type SCSI_PHYSICAL_ADDRESS.

See also ScsiPortGetPhysicalAddress
