VflatD_Reset 


include vflatd.inc

mov   dx, VflatD_Reset          ; Function number
call  VFlatDApiProcAdr
jc    errorhandler

Called when Video hardware has been changed by someone other than VFLATD. A display driver typically does not call this service. Uses EAX, and Flags.

  • Returns with the carry flag clear if successful; otherwise sets the carry flag.
