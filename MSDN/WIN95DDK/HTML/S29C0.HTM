Adding Section Names to the Install Section AddReg= Item

All of the additions you must make to add voice modem capability to a modem INF file are in AddReg-type sections. If you add entire new AddReg-type sections to the INF file, don't forget to add the name of the new section(s) to the AddReg= item in the INF Install section. A typical AddReg= item is shown below, with four AddReg-type sections named in it. You may have to add a fifth, and a sixth, etc. 


AddReg=All, MfgAddReg, Modem1.AddReg, INTERNAL
