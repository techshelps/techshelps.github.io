VPEND_SETCALIBRATION

Sets calibration values in the hardware information structure.

  • Returns DRV_FAILURE or DRV_SUCCESS.

lParam1

Pointer to a DRV_CALBSTRUCT structure that contains the calibration information to set in the hardware information structure.

lParam2

Should be set to zero.

For a definition of the members of the DRV_CALBSTRUCT structure see the "Reference" section in About the Pen Driver.
