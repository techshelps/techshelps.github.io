HDR

Prints header information from executable files.

Syntax

hdr [ options ] filename

Parameters

-d

Prints data relocation records.

-f

Prepends the file name on every output line.

-h

Prints the file header (default).

-m

Print only memory segment table information.

-j[options]

Prints specified tables.


d

Prints debug table.

e

Prints entry table.

f

Prints extended attributes.

i

Prints import module table.

I

Prints import procedure table.

r

Prints resident name table.

R

Prints resource table.


-p

Prints file seek positions.

-R

Same as -r, but also follows relocation chains.

-r

Prints both text and data relocations.

-S

Prints the file segment table with a header.

-s

Prints the symbol table.

-T

Prints the file type.

-t

Prints the text relocation records.

-y

Dumps DOS .SYM file.

-v

Prints everything (verbose).

-?

Prints usage summary.

filename

Specifies the filename
