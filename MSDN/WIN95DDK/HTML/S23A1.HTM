DiInstallClass 


RETERR DiInstallClass(LPCSTR lpszInfFileName, DWORD dwFlags)

Install the [ClassInstall] section of the specificed INF.

  • Returns OK if successful, otherwise an ERR_DI_xxx error code will be returned.

lpszInfFileName

The name of the INF file containing a [ClassInstall] section.

dwFlags

Flags to control the geninstall operation.

DI_NOVCP

This flag should be specificed if a VCP queue is allready opened.

DI_NOBROWSE

This flag should be specificed if no file browsing should be allowed in the event a copy operation cannot find a specificed file.


This API is generally called the the Device Manager when it installs a device of a new class.
