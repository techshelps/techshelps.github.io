Unhook_VMM_Fault 


include vmm.inc

mov     eax, FaultNo
mov     esi, OFFSET32 Hook_Proc
VMMCall Unhook_VMM_Fault
jc      Error_Handler

Removes the hook procedure for the specified ring 0 fault. Uses flags.

  • Clears the carry flag if successful; sets the carry flag on the following conditions:
    • Invalid fault number passed in EAX.
    • ESI does not point to a hook procedure.
    • Hook procedure could not be found.
    • A non-hook procedure was found in the chain of procedures.

FaultNo

Specifies the fault number.

Hook_Proc

Address of the hook procedure to be removed.

The specified hook procedure must have been declared using the BeginProc macro with the HOOK_PROC attribute in order to generate the necessary header.

See also BeginProc
