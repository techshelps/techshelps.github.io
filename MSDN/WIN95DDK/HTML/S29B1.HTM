OPTIONAL Step 4 -- Adding Voiceview Support

To add VoiceView support for your modem, you will have to make the following changes to your modem INF file sections:

  • Add several items to the Install section.
  • Add an AddReg section that puts your VoiceView modem commands and responses in the registry.
  • Add a CopyFiles section and a DestinationDirs section that copy the VoiceView support files to the appropriate paths on the hard disk.
  • Add an AddReg section that puts information in the registry that integrates the VoiceView components with Windows Sockets.
  • Add items to the [Strings] section.
  • Add an Update Ini File section and Uninstall section that add and remove VoiceView icons in the Windows 95 Shell.
