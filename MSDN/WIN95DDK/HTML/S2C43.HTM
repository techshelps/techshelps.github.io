DVM_STREAM_RESET

Notifies a video capture driver to stop input of a video stream and reset the current position to 0.

  • Returns DV_ERR_OK if successful or an error code otherwise.The DV_ERR_NOTSUPPORTED error code indicates the message is not supported.

dwParam1, dwParam2

Not used.

All pending buffers are marked as done and returned to the application.

When a device driver receives this message, it should return to the state established for DVM_STREAM_INIT.
