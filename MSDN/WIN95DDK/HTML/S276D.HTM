_VCOMM_GetModemStatus 


include vcomm.inc

VxDcall _VCOMM_GetModemStatus, <hPort, <OFFSET32 dwModemStatus>>
or      eax, eax
jz      error_handler

Retrieves the current value of the modem-status register. Uses the C calling convention. This is an asynchronous service.

  • Returns TRUE if successful, FALSE otherwise. For extended error information, use _VCOMM_GetLastError.

hPort

Handle of a communications resource.

dwModemStatus

Address of a 32-bit variable to receive the modem status. This can be a combination of the following values:

Value

Meaning

MS_CTS_ON

CTS is on.

MS_RING_ON

Ring indicator is on.

MS_RLDS_ON

RLSD is on.

MS_DSR_ON

DSR is on.


See also _VCOMM_GetLastError
