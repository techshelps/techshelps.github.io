DVM_STREAM_START

Notifies a video capture driver to start a video stream.

  • Returns DV_ERR_OK if successful or an error code otherwise.. Otherwise, it returns an error number. The DV_ERR_NOTSUPPORTED error code indicates the message is not supported.

dwParam1, dwParam2

Not used.

For VIDEO_IN channels, this message begins transferring the contents of the frame buffer to the system supplied buffers.
