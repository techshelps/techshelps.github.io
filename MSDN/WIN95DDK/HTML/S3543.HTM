VCache_GetStats

Retrieve cache statistics. Uses EBX, ECX, EDI.

  • Returns statistics in the following registers:

Register

Contents on exit

EBX

Number of cache misses to one of the last 26 discarded cache blocks since the last call to VCache_GetStats.

ECX

Number of cache hits to one of the last 26 LRU cache blocks since the last call to VCache_GetStats.

EDX

Base address of range reserved for VCache.

EDI

Number of cache blocks discarded since the last call to VCache_GetStats.


This service returns some statistics used by the memory manager to tune how much memory is available for VCache and how much for paging.

When the statistics are queried, the counts are cleared. Therefore, if an application other than the memory manager uses this service, system performance will suffer.
