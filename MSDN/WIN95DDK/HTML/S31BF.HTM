Disable_Touch_1st_Meg 


include vmm.inc

VMMcall Disable_Touch_1st_Meg

Disables the first 1 megabyte of memory for the current virtual machine. This is an asynchronous service.

  • No return value.

See the description of Enable_Touch_1st_Meg for more information.

Do not call this service directly. Use the End_Touch_1st_Meg macro instead.

This service has no effect in the retail version of Windows. It is intended to be used with the debugging version.

See also Enable_Touch_1st_Meg
