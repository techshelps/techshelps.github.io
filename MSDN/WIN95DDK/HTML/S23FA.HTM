Child List

This entry is a text entry that, like the resource map, is added via the addreg command from an [InstallSection.MF] section. These entries cause the configuration manager to create devnodes for every logical device contained in the child list. Configuration Manager, enumerates these devices. The INF entry would look like the example below. 


HKR,ChildList,LogicalDeviceID1,,CompatibilityIDList
