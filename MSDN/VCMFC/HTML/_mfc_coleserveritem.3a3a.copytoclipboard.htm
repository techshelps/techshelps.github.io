COleServerItem::CopyToClipboard

void CopyToClipboard( BOOL bIncludeLink = FALSE );

Parameters

bIncludeLink

Set this to TRUE if link data should be copied to the Clipboard. Set this to FALSE if your server application does not support links.

Remarks

Call this function to copy the OLE item to the Clipboard. The function uses the OnGetClipboardData member function to create a COleDataSource object containing the OLE item’s data in the formats supported. The function then places the COleDataSource object on the Clipboard by using the COleDataSource::SetClipboard function. The COleDataSource object includes the item’s native data and its representation in CF_METAFILEPICT format, as well as data in any conversion formats you choose to support. You must have implemented Serialize and OnDraw for this member function to work.
