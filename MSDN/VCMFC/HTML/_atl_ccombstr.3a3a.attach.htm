CComBSTR::Attach

void Attach( BSTR src );

Parameters

src

[in] The BSTR to attach to the object.

Remarks

Attaches a BSTR to the CComBSTR object by setting the m_str member to src.

Note   This method will assert if m_str is non-NULL.
