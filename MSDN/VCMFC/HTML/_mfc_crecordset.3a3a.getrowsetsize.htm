CRecordset::GetRowsetSize

DWORD GetRowsetSize( ) const;

Return Value

The number of rows to retrieve during a given fetch.

Remarks

Call this member function to obtain the current setting for the number of rows you wish to retrieve during a given fetch. If you are using bulk row fetching, the default rowset size when the recordset is opened is 25; otherwise, it is 1.

To implement bulk row fetching, you must specify the CRecordset::useMultiRowFetch option in the dwOptions parameter of the Open member function. To change the setting for the rowset size, call SetRowsetSize.

For more information about bulk row fetching, see the article Recordset: Fetching Records in Bulk (ODBC) in Visual C++ Programmer’s Guide.
