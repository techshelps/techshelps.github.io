CStatic::GetEnhMetaFile

HENHMETAFILE GetEnhMetaFile( ) const;

Return Value

A handle to the current enhanced metafile, or NULL if no enhanced metafile has been set.

Remarks

Call this member function to get the handle of the enhanced metafile, previously set with SetEnhMetafile, that is associated with Cstatic.
