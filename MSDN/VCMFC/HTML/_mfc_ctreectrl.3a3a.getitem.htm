CTreeCtrl::GetItem

BOOL GetItem( TVITEM* pItem );

Return Value

Nonzero if successful; otherwise 0.

Parameters

pItem

A pointer to a TVITEM structure, as described in the Platform SDK.

Remarks

Call this function to retrieve the attributes of the specified tree view item.
