CDaoTableDef::m_pDAOTableDef

Remarks

Contains a pointer to the OLE interface for the DAO tabledef object underlying the CDaoTableDef object. Use this pointer if you need to access the DAO interface directly.

For more information on accessing underlying DAO objects, see the article DAO Collections: Obtaining Information About DAO Objects in Visual C++ Programmer's Guide.
