CString::IsEmpty

BOOL IsEmpty( ) const;

Return Value

Nonzero if the CString object has 0 length; otherwise 0.

Remarks

Tests a CString object for the empty condition.

Example

The following example demonstrates the use of CString::IsEmpty.

// example for CString::IsEmpty
CString s;
ASSERT( s.IsEmpty() );
