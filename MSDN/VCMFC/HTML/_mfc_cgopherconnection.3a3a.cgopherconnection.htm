CGopherConnection::CGopherConnection

CGopherConnection( );

Remarks

This member function is called to construct a CGopherConnection object. You never create a CGopherConnection directly. Rather, call CInternetSession::GetGopherConnection, which creates a CGopherConnection object and returns a pointer to it.
