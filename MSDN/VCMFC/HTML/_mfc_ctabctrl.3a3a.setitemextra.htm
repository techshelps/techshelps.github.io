CTabCtrl::SetItemExtra

BOOL SetItemExtra( int nBytes );

Return Value

Nonzero if successful; otherwise zero.

Parameters

nBytes

The number of extra bytes to set.

Remarks

This member function implements the behavior of the Win32 message TCM_SETITEMEXTRA, as described in the Intenet Client SDK.
