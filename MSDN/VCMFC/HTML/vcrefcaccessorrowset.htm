CAccessorRowset

template < class TAccessor, class TRowset >
class CAccessorRowset : public TAccessor, public TRowset

Parameters

TAccessor

An accessor class.

TRowset

A rowset class.

CAccessorRowset encapsulates a rowset and its associated accessors in a single class. Class TAccessor manages the accessor. Class TRowset manages the rowset.
