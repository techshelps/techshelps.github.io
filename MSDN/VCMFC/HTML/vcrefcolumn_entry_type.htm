COLUMN_ENTRY_TYPE

COLUMN_ENTRY_TYPE ( nOrdinal, wType, data )

Parameters

nOrdinal

[in] The column number.

wType

[in] Data type of column entry.

data

[in] The corresponding data member in the user record.

Remarks

A specialized variant of the COLUMN_ENTRY macro that provides a means of specifying data type.
