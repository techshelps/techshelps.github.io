CString::MakeLower

void MakeLower( );

Remarks

Converts this CString object to a lowercase string.

Example

The following example demonstrates the use of CString::MakeLower.

// example for CString::MakeLower
CString s( "ABC" );
s.MakeLower();
ASSERT( s == "abc" );
