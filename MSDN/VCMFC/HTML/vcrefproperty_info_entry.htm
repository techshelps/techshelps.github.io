PROPERTY_INFO_ENTRY

PROPERTY_INFO_ENTRY( dwPropID )

Parameters

dwPropID

[in] A DBPROPID value that can be used in conjunction with the property set GUID to identify a property.

Remarks

Represents a specific property in a property set. This macro sets the property value of type DWORD to a default value defined in ATLDB.H. To set the property to a value of your choosing, use PROPERTY_INFO_ENTRY_VALUE. To set the VARTYPE and DBPROPFLAGS for the property at the same time, use PROPERTY_INFO_ENTRY_EX.
