Active Document Classes

Active documents can be displayed either in the entire client window of a Web browser, such as Internet Explorer 3.0, or in an Active container—such as the Microsoft Office Binder—that supports Active documents.

CDocObjectServer

Maps the Active document interfaces, and initializes and activates an Active document object.

CDocObjectServerItem

Implements OLE server verbs specifically for Active document servers.

COleDocObjectItem

Implements Active document containment.
