CTreeCtrl::GetParentItem

HTREEITEM GetParentItem( HTREEITEM hItem );

Return Value

The handle of the parent item; otherwise NULL.

Parameters

hItem

Handle of a tree item.

Remarks

Call this function to retrieve the parent of hItem.
