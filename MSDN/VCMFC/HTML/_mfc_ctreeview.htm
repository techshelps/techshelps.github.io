CTreeView

The CTreeView class simplifies use of the tree control and of CTreeCtrl, the class that encapsulates tree-control functionality, with MFC’s document-view architecture. For more information on this architecture, see the overview for the CView class and the cross-references cited there.

#include <afxcview.h>
