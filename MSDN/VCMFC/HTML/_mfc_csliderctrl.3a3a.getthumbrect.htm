CSliderCtrl::GetThumbRect

void GetThumbRect( LPRECT lprc ) const;

Parameters

lprc

A pointer to a CRect object that contains the bounding rectangle for the slider when the function returns.

Remarks

Call this function to retrieve the size and position of the bounding rectangle for the slider (thumb) in a slider control.
