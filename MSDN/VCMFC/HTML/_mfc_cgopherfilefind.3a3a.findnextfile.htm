CGopherFileFind::FindNextFile

virtual BOOL FindNextFile( );

Return Value

Nonzero if there are more files; zero if this is the last file, and the previous call to either FindFile or FindNextFile returned nonzero. To get extended error information, call the Win32 function GetLastError.

Remarks

Call this member function to continue a file search begun with a call to CGopherFileFind::FindFile.
