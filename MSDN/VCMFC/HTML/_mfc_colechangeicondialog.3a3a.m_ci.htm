COleChangeIconDialog::m_ci

Remarks

Structure of type OLEUICHANGEICON used to control the behavior of the Change Icon dialog box. Members of this structure can be modified either directly or through member functions.

For more information, see the OLEUICHANGEICON structure in the OLE 2.01 User Interface Library.
