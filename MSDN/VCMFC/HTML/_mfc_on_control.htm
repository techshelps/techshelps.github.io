ON_CONTROL

ON_CONTROL( wNotifyCode, id, memberFxn )

Parameters

wNotifyCode

The notification code of the control.

id

The command ID.

memberFxn

The name of the message-handler function to which the command is mapped.

Remarks

Indicates which function will handle a custom-control notification message. Control notification messages are those sent from a control to its parent window.

There should be exactly one ON_CONTROL macro statement in your message map for every control notification message that must be mapped to a message-handler function.

For more information and examples, seeMessage Handling and Mapping Topics in Visual C++ Programmer’s Guide.
