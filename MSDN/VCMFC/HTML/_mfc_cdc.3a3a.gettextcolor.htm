CDC::GetTextColor 

COLORREF GetTextColor( ) const;

Return Value

The current text color as an RGB color value.

Remarks

Retrieves the current text color. The text color is the foreground color of characters drawn by using the GDI text-output member functions TextOut, ExtTextOut, and TabbedTextOut.
