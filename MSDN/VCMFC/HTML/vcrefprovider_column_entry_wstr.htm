PROVIDER_COLUMN_ENTRY_WSTR

PROVIDER_COLUMN_ENTRY_WSTR( name, ordinal, member )

Parameters

name

[in] The column name.

ordinal

[in] The column number. Unless the column is a Bookmark column, the column number must not be 0.

member

[in] The member variable in the data class that stores the data.

Remarks

Represents a specific column supported by the provider. Use this macro when the column data is a null terminated Unicode character string, DBTYPE_WSTR.
