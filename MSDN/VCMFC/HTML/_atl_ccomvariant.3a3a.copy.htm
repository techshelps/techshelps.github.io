CComVariant::Copy

HRESULT Copy( const VARIANT* pSrc );

Return Value

A standard HRESULT value.

Parameters

pSrc

[in] A pointer to the VARIANT to be copied

Remarks

Frees the CComVariant object and then assigns it a copy of the specified VARIANT.
