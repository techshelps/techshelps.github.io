TN015: Windows for Pen Computing

The MFC 2.0 product for Windows 3.1 supports the Windows for Pen Computing environment that is available as an add-on to Windows 3.1. MFC 4.0 and later versions do not support the Pen Computing add-on to Windows 3.1, but do support Pen Computing on Windows 95. Windows NT does not yet provide the Windows for Pen Computing environment.
