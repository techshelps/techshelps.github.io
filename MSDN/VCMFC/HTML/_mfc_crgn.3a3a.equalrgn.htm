CRgn::EqualRgn

BOOL EqualRgn( CRgn* pRgn ) const;

Return Value

Nonzero if the two regions are equivalent; otherwise 0.

Parameters

pRgn

Identifies a region.

Remarks

Determines whether the given region is equivalent to the region stored in the CRgn object.
