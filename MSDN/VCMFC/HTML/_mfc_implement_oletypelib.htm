IMPLEMENT_OLETYPELIB

IMPLEMENT_OLETYPELIB( class_name, tlid, wVerMajor, wVerMinor )

Parameters

class_name

The name of the control class related to the type library.

tlid

The ID number of the type library.

wVerMajor

The type library major version number.

wVerMinor

The type library minor version number.

Remarks

Implements the control’s GetTypeLib member function. This macro must appear in the implementation file for any control class that uses the DECLARE_OLETYPELIB macro.
