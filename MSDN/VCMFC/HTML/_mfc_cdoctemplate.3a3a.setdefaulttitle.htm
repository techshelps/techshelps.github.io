CDocTemplate::SetDefaultTitle

virtual void SetDefaultTitle( CDocument* pDocument ) = 0;

Parameters

pDocument

Pointer to the document whose title is to be set.

Remarks

Call this function to load the document’s default title and display it in the document’s title bar. For information on the default title, see the description of CDocTemplate::docName in CDocTemplate::GetDocString.
