COleDateTimeSpan::m_span

Remarks

The underlying double value for this COleDateTime object. This value expresses the date/time-span in days.

Caution   Changing the value in the double data member will change the value of this COleDateTimeSpan object. It does not change the status of this COleDateTimeSpan object.
