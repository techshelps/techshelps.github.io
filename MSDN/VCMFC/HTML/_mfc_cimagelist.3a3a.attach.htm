CImageList::Attach

BOOL Attach( HIMAGELIST hImageList );

Return Value

Nonzero if the attachment was successful; otherwise 0.

Parameters

hImageList

A handle to an image list object.

Remarks

Call this function to attach an image list to a CImageList object.
