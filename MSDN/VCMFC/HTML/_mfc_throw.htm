THROW

THROW( exception_object_pointer )

Parameters

exception_object_pointer

Points to an exception object derived from CException.

Remarks

Throws the specified exception. THROW interrupts program execution, passing control to the associated CATCH block in your program. If you have not provided the CATCH block, then control is passed to a Microsoft Foundation Class Library module that prints an error message and exits.

For more information, see the article Exceptions in Visual C++ Programmer’s Guide.
