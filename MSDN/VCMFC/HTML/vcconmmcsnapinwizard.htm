MMC Snap-In Wizard

The Microsoft Management Console (MMC) Snap-In wizard is designed to easily create a framework for an MMC snap-in object, for use in an ATL project. The snap-in object is implemented with several ATL classes and macros.

This article discusses the following topics:

  • Overview of the MMC Snap-In Wizard

  • MMC Snap-In FAQ

This article is intended for developers who have prior experience developing snap-in objects and ATL projects. It is not intended as a tutorial but a supplement, regarding ATL support for MMC snap-ins, to the existing Platform SDK documentation. For more information, see Microsoft Management Console in the Platform SDK.
