CRect::Width

int Width( ) const;

Return Value

The width of CRect.

Remarks

Calculates the width of CRect by subtracting the left value from the right value. The width can be negative.

Note   The rectangle must be normalized or this function may fail. You can call NormalizeRect to normalize the rectangle before calling this function.
