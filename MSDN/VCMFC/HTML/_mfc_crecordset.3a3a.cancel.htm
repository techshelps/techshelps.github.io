CRecordset::Cancel

void Cancel( );

Remarks

Call this member function to request that the data source cancel either an asynchronous operation in progress or a process from a second thread. Note that the MFC ODBC classes no longer use asynchronous processing; to perform an asychronous operation, you must directly call the ODBC API function SQLSetConnectOption. For more information, see the topic "Executing Functions Asynchronously" in the ODBC SDK Programmer's Guide.
