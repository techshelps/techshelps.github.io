CEnumerator

class CEnumerator : public CAccessorRowset< CAccessor <CEnumeratorAccessor >>

Uses an OLE DB enumerator object, which exposes the ISourcesRowset interface to return a rowset describing all data sources and enumerators.

You can retrieve the ISourcesRowset data indirectly from this class. See the CatDB sample file ENUMDLG.CPP, where you can find an example of using ISourcesRowset data from the CEnumeratorAccessor class.

#include <atldbcli.h>
