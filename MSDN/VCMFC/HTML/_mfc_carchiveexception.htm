CArchiveException

A CArchiveException object represents a serialization exception condition. The CArchiveException class includes a public data member that indicates the cause of the exception.

CArchiveException objects are constructed and thrown inside CArchive member functions. You can access these objects within the scope of a CATCH expression. The cause code is independent of the operating system. For more information about exception processing, see the article Exceptions in Visual C++ Programmer's Guide.
