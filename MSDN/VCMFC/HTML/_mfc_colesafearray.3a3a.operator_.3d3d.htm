Remarks

This operator compares two arrays (SAFEARRAY, VARIANT, COleVariant, or COleSafeArray arrays) and returns nonzero if they are equal; otherwise 0. Two arrays are equal if they have an equal number of dimensions, equal size in each dimension, and equal element values.
