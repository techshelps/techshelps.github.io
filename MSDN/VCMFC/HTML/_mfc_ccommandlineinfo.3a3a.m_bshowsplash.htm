CCommandLineInfo::m_bShowSplash

Remarks

Indicates that the splash screen should be displayed. If TRUE, this means the splash screen for this application should be displayed during startup. The default implementation of ParseParam sets this data member to TRUE if m_nShellCommand is equal to CCommandLineInfo::FileNew.
