COleCurrency::m_cur

Remarks

The underlying CURRENCY structure for this COleCurrency object.

Caution   Changing the value in the CURRENCY structure accessed by the pointer returned by this function will change the value of this COleCurrency object. It does not change the status of this COleCurrency object.

For more information, see the CURRENCY entry in the Win32 SDK OLE Programmer’s Reference.
