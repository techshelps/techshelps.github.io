CInternetException::m_dwError

Remarks

The error that caused the exception. This error value may be a system error code, found in WINERROR.H, or an error value from WININET.H.

For a list of Win32 error codes, see Error Codes in the Win32 SDK. For a list of Internet-specific error codes, see the ActiveX SDK documentation.
