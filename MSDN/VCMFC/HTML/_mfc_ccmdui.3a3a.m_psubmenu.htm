CCmdUI::m_pSubMenu

Remarks

Pointer (of CMenu type) to the contained sub-menu represented by the CCmdUI object. NULL if the item is not a menu. If the sub menu is a pop-up, m_nID contains the ID of the first item in the pop-up menu. For more information, see Technical Note 21.
