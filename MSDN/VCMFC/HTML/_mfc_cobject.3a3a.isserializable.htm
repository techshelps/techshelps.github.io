CObject::IsSerializable

BOOL IsSerializable( ) const;

Return Value

Nonzero if this object can be serialized; otherwise 0.

Remarks

Tests whether this object is eligible for serialization. For a class to be serializable, its declaration must contain the DECLARE_SERIAL macro, and the implementation must contain the IMPLEMENT_SERIAL macro.

Note   Do not override this function.

Example

See CObList::CObList for a listing of the CAge class used in all CObject examples.

// example for CObject::IsSerializable
CAge a(21);
ASSERT( a.IsSerializable() );
