COleUpdateDialog

The COleUpdateDialog class is used for a special case of the OLE Edit Links dialog box, which should be used when you need to update only existing linked or embedded objects in a document.

For more information regarding OLE-specific dialog boxes, see the article Dialog Boxes in OLE in Visual C++ Programmer’s Guide.
