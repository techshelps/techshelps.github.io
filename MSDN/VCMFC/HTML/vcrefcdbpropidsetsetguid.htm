CDBPropIDSet::SetGUID

SetGUID( const GUID& guid );

Parameters

guid

[in] A GUID used to set the guidPropertySet field of the DBPROPIDSET structure.

Remarks

Sets the GUID field in the DBPROPIDSET structure. This field can be set by the constructor as well. Call this function if you use the default constructor for this class.
