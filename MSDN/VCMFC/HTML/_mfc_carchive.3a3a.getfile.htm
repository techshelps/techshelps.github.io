CArchive::GetFile

CFile* GetFile( ) const;

Return Value

A constant pointer to the CFile object in use.

Remarks

Gets the CFile object pointer for this archive. You must flush the archive before using GetFile.

Example

extern CArchive ar;
const CFile* fp = ar.GetFile();
