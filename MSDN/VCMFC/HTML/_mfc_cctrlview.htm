CCtrlView

The class CCtrlView and its derivatives, CEditView, CListView, CTreeView, and CRichEditView, adapt the document-view architecture to the new common controls supported by Windows 95 and Windows NT versions 3.51 and later. For more information on the document-view architecture, see Document/View Architecture Topics in Visual C++ Programmer's Guide.
