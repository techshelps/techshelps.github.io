COleClientItem::GetDrawAspect

DVASPECT GetDrawAspect( ) const;

Return Value

A value from the DVASPECT enumeration, whose values are listed in the reference for SetDrawAspect.

Remarks

Call the GetDrawAspect member function to determine the current “aspect,” or view, of the item. The aspect specifies how the item is to be rendered.
