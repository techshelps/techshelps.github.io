END_ACCESSOR

END_ACCESSOR( )

Remarks

Marks the end of an accessor entry. In the case of multiple accessors on a rowset, you need to specify BEGIN_ACCESSOR_MAP, and use the BEGIN_ACCESSOR macro for each individual accessor. The BEGIN_ACCESSOR macro is completed with the END_ACCESSOR macro. The BEGIN_ACCESSOR_MAP macro is completed with the END_ACCESSOR_MAP macro.
