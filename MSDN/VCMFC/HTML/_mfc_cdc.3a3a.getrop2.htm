CDC::GetROP2 

int GetROP2( ) const;

Return Value

The drawing mode. For a list of the drawing mode values, see the SetROP2 member function.

Remarks

Retrieves the current drawing mode. The drawing mode specifies how the colors of the pen and the interior of filled objects are combined with the color already on the display surface.
