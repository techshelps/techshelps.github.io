CSnapInPropertyPageImpl::OnReset

void OnReset( );

Remarks

This member function is called when the user clicks the Cancel button. When this function is called, changes to all property pages that were made by the user previously clicking the Apply Now button are discarded, and the property sheet retains focus.

Override this member function to specify what action the program takes when the user clicks the Cancel button.
