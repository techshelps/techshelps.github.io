CRichEditDoc::GetStreamFormat

int GetStreamFormat( ) const;

Return Value

One of the following flags:

  • SF_TEXT   Indicates that the rich edit control does not maintain formatting information.

  • SF_RTF   Indicates that the rich edit control does maintain formatting information.

Remarks

Call this function to determine the text format for streaming the contents of the rich edit. The return value is based on the m_bRTF data member. This function returns SF_RTF if m_bRTF is TRUE; otherwise, SF_TEXT.
