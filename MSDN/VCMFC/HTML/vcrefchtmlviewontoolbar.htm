CHtmlView::OnToolBar

virtual void OnToolBar( BOOL bToolBar );

Parameters

bToolBar

Nonzero if Internet Explorer's toolbar is visible or zero otherwise.

Remarks

This member function is called by the framework when the ToolBar property has changed.
