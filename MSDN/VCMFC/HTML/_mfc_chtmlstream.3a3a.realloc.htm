CHtmlStream::Realloc

virtual BYTE* Realloc( BYTE* lpMem, DWORD nBytes );

Return Value

A pointer to the memory block that was reallocated (and possibly moved), or NULL if the reallocation failed.

Parameters

lpMem

A pointer to the memory block to be reallocated.

nBytes

New size for the memory block.

Remarks

Called by the framework to reallocate memory. Override this function to implement custom memory reallocation. If you override this function, you'll probably want to override Alloc and Free as well.
