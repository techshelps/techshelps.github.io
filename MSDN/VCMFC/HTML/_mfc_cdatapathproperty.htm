CDataPathProperty

Class CDataPathProperty implements an OLE control property that can be loaded asynchronously. Asynchronous properties are loaded after synchronous initiation.

The class CDataPathProperty is derived from CAysncMonikerFile. To implement asynchronous properties in your OLE controls, derive a class from CDataPathProperty, and override OnDataAvailable.

For more information about how to use asynchronous monikers and ActiveX controls in Internet applications, see the following articles in Visual C++ Programmer's Guide:

  • Internet First Steps: ActiveX Controls

  • Internet First Steps: Asynchronous Monikers
