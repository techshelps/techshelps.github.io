SNAPINTOOLBARID_ENTRY

SNAPINTOOLBARID_ENTRY( id )

Parameters

id

[in] Identifies the toolbar control.

Remarks

Use this macro to enter a toolbar ID into the Snap-In object's toolbar ID map. The BEGIN_SNAPINTOOLBARID_MAP macro marks the beginning of the property map; the END_SNAPINTOOLBARID_MAP macro marks the end.
