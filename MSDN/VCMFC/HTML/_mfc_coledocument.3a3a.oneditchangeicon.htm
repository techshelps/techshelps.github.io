COleDocument::OnEditChangeIcon

afx_msg void OnEditChangeIcon( );

Remarks

Displays the OLE Change Icon dialog box and changes the icon representing the currently selected OLE item to the icon the user selects in the dialog box. OnEditChangeIcon creates and launches a COleChangeIconDialog Change Icon dialog box.
