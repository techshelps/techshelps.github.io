CFileFind::IsArchived

BOOL IsArchived( ) const;

Return Value

Nonzero if successful; otherwise 0.

Remarks

Call this member function to determine if the found file is archived. Applications mark an archive file, which is to be backed up or removed, with FILE_ATTRIBUTE_ARCHIVE, a file attribute identified in the WIN32_FIND_DATA structure.

See the member function MatchesMask for a complete list of file attributes.
