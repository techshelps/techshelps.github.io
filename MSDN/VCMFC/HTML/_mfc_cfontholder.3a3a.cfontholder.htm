CFontHolder::CFontHolder

CFontHolder( LPPROPERTYNOTIFYSINK pNotify );

Parameters

pNotify

Pointer to the font’s IPropertyNotifySink interface.

Remarks

Constructs a CFontHolder object. You must call InitializeFont to initialize the resulting object before using it.
