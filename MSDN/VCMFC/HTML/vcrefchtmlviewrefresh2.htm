CHtmlView::Refresh2

void Refresh2( int nLevel );

Parameters

nLevel

The address of the variable specifying the refresh level. The possible variables are defined in RefreshConstants, in the Platform SDK.

Remarks

Reloads the file that Internet Explorer is currently displaying. Unlike Refresh, Refresh2 contains a parameter that specifies the refresh level.

Applies to Internet Explorer and WebBrowser.
