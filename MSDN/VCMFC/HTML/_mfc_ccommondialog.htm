CCommonDialog

CCommonDialog is the base class for classes that encapsulate functionality of the Windows common dialogs:

  • CFileDialog

  • CFontDialog

  • CColorDialog

  • CPageSetupDialog

  • CPrintDialog

  • CFindReplaceDialog

  • COleDialog
