CSliderCtrl::GetPageSize

int GetPageSize( ) const;

Return Value

The size of a page for the slider control.

Remarks

Call this function to retrieve the size of the page for a slider control. The page size affects how much the slider moves for the TB_PAGEUP and TB_PAGEDOWN notifications.
