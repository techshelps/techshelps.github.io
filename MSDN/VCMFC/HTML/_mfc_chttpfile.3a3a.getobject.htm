CHttpFile::GetObject

CString GetObject( ) const;

Return Value

A CString object containing the name of the object.

Remarks

Call this member function to get the name of the object associated with this CHttpFile. Use this member function only after a successful call to SendRequest or on a CHttpFile object successfully created by OpenURL.
