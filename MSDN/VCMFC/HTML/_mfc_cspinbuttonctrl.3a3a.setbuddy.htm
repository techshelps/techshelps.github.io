CSpinButtonCtrl::SetBuddy

CWnd* SetBuddy( CWnd* pWndBuddy );

Return Value

A pointer to the previous buddy window.

Parameters

pWndBuddy

Pointer to the new buddy window.

Remarks

Call this function to set the buddy window for a spin button control.
