COleDispatchDriver::m_lpDispatch

Remarks

The pointer to the IDispatch interface attached to this COleDispatchDriver. The m_lpDispatch data member is a public variable of type LPDISPATCH.

For more information, see IDispatch in the OLE Programmer’s Reference.
