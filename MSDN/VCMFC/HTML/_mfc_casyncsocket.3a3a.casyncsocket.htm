CAsyncSocket::CAsyncSocket

CAsyncSocket( );

Remarks

Constructs a blank socket object. After constructing the object, you must call its Create member function to create the SOCKET data structure and bind its address. (On the server side of a Windows Sockets communication, when the listening socket creates a socket to use in the Accept call, you do not call Create for that socket.)
