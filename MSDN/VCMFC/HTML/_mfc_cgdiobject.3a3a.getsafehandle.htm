CGdiObject::GetSafeHandle

HGDIOBJ GetSafeHandle( ) const;

Return Value

A HANDLE to the attached Windows GDI object; otherwise NULL if no object is attached.

Remarks

Returns m_hObject unless this is NULL, in which case NULL is returned. This is part of the general handle interface paradigm and is useful when NULL is a valid or special value for a handle.
