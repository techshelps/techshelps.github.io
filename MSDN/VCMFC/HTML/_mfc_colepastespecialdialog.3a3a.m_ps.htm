COlePasteSpecialDialog::m_ps

Remarks

Structure of type OLEUIPASTESPECIAL used to control the behavior of the Paste Special dialog box. Members of this structure can be modified directly or through member functions.

For more information, see the OLEUIPASTESPECIAL structure in the OLE 2.01 User Interface Library.
