Objects and Controls

The ATL Object Wizard allows you to insert a variety of different objects and controls into your project, including:

  • Minimal COM objects

  • Full controls

  • Internet Explorer controls

  • Internet Explorer objects

  • Add-in objects

  • Dialog boxes

  • Property pages

  • Active Server components

  • Microsoft Transaction Server components

  • Component Registrar objects

For more information about these objects, see the article Creating an ATL Project. For step-by-step instructions on inserting a control and adding custom properties and events, see the ATL Tutorial.
