Remarks

IDBProperties is a mandatory interface for data source objects and an optional interface for enumerators. However, if an enumerator exposes IDBInitialize, it must expose IDBProperties. IDBPropertiesImpl provides an implementation for the IDBProperties interface. IDBPropertiesImpl implements IDBProperties by using a static function defined by BEGIN_PROPSET_MAP.
