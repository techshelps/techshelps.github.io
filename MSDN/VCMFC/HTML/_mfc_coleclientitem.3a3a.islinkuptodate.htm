COleClientItem::IsLinkUpToDate

BOOL IsLinkUpToDate( ) const;

Return Value

Nonzero if the OLE item is up to date; otherwise 0.

Remarks

Call this function to see whether the OLE item is up to date. A linked item can be out of date if its source document has been updated. An embedded item that contains links within it can similarly become out of date. The function does a recursive check of the OLE item. Note that determining whether an OLE item is out of date can be as expensive as actually performing an update.

This is called automatically by the COleLinksDialog implementation.

For more information, see IOleObject::IsUpToDate in the OLE documentation.
