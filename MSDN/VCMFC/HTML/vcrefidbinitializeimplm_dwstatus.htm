IDBInitializeImpl::m_dwStatus

DWORD m_dwStatus;

Remarks

Contains one or more of the following enum.

enum DATASOURCE_FLAGS {
   DSF_MASK_INIT         = 0xFFFFF00F,   
   DSF_PERSIST_DIRTY      = 0x00000001,   // Set if init string changes.
   DSF_INITIALIZED         = 0x00000010,   // Have we been initialized.
};

where,

DSF_MASK_INIT A mask to restore the uninitialized state.
DSF_PERSIST_DIRTY Set if object requires persistence.
DSF_INITIALIZED Set if data source is initialized.
