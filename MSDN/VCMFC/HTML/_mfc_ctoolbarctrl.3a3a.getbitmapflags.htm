CToolBarCtrl::GetBitmapFlags

UINT GetBitmapFlags( ) const;

Return Value

A UINT that has the TBBF_LARGE flag set if the display can support large toolbar bitmaps, clear otherwise.

Remarks

Call this function to retrieve the bitmap flags from the toolbar. You should call it after creating the toolbar but before adding bitmaps to the toolbar.

The return value indicates whether the display supports large bitmaps or not. If the display supports large bitmaps and if you choose to use them, call SetBitmapSize and SetButtonSize before adding your large bitmap using AddBitmap.
