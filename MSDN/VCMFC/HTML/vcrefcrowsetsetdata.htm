CRowset::SetData

HRESULT SetData( );

HRESULT SetData( int nAccessor );

Return Value

A standard HRESULT.

Parameters

nAccessor

[in] The number of the accessor to use for accessing the data.

Remarks

Sets data values in one or more columns of a row. In the case of the SetData function that accepts no arguments, all accessors are used for updating.
