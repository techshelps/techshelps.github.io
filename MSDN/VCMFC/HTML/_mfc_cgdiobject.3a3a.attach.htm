CGdiObject::Attach

BOOL Attach( HGDIOBJ hObject );

Return Value

Nonzero if attachment is successful; otherwise 0.

Parameters

hObject

A HANDLE to a Windows GDI object (for example, HPEN or HBRUSH).

Remarks

Attaches a Windows GDI object to a CGdiObject object.
