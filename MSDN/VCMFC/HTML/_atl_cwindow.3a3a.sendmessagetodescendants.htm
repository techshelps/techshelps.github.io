CWindow::SendMessageToDescendants

void SendMessageToDescendants( UINT message, WPARAM wParam = 0, LPARAM lParam = 0, BOOL bDeep = TRUE );

Parameters

message

[in] The message to be sent.

wParam

[in] Additional message-specific information.

lParam

[in] Additional message-specific information.

bDeep

[in] If TRUE (the default value), the message will be sent to all descendant windows; otherwise, it will be sent only to the immediate child windows.

Remarks

Sends the specified message to all immediate children of the CWindow object. If bDeep is TRUE, the message is additionally sent to all other descendant windows.
