CWnd::BringWindowToTop 

void BringWindowToTop( );

Remarks

Brings CWnd to the top of a stack of overlapping windows. In addition, BringWindowToTop activates pop-up, top-level, and MDI child windows. The BringWindowToTop member function should be used to uncover any window that is partially or completely obscured by any overlapping windows.

Calling this function is similar to calling the SetWindowPos function to change a window’s position in the Z-order. The BringWindowToTop function does not change the window style to make it a top-level window of the desktop.
