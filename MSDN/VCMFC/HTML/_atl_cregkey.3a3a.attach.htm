CRegKey::Attach

void Attach( HKEY hKey );

Parameters

hKey

[in] The handle of a registry key.

Remarks

Attaches an HKEY to the CRegKey object by setting the m_hKey member handle to hKey.

Note   Attach will assert if m_hKey is non-NULL.
