TRACE1

TRACE1( exp, param1 )

Parameters

exp

A format string as used in the run-time function printf.

param1

The name of the variable whose value should be dumped.

Remarks

See TRACE0 for a description of the TRACE1 macro.

Example

// example for TRACE1
int i = 1;
TRACE1( "Integer = %d\n", i );
// Output: 'Integer = 1'
