CString::MakeUpper

void MakeUpper( );

Remarks

Converts this CString object to an uppercase string.

Example

The following example demonstrates the use of CString::MakeUpper.

// example for CString::MakeUpper
CString s( "abc" );
s.MakeUpper();
ASSERT( s == "ABC" );
