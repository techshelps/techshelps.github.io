CDatabase::InWaitForDataSource

Remarks

In MFC 4.2, CDatabase::InWaitForDataSource became obsolete. The MFC ODBC classes now use only synchronous processing. To start an asychronous operation, you must directly call the ODBC API function SQLSetConnectOption. For more information, see the topic "Executing Functions Asynchronously" in the ODBC SDK Programmer's Guide.
