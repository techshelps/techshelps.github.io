TN018: Migrating OLE Applications from MFC 1.0 to MFC 2.0

This technical note provided guidelines for migrating MFC version 1 applications that use MFC/OLE to MFC 2.0. This version of MFC does not support OLE 1.0. OLE 1.0 support has been replaced by OLE 2.0 support. For more information about migrating your MFC/OLE1 application to MFC/OLE2 see Technical Note 41.
