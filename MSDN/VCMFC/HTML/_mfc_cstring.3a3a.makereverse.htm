CString::MakeReverse

void MakeReverse( );

Remarks

Reverses the order of the characters in this CString object.

Example

The following example demonstrates the use of CString::MakeReverse.

// example for CString::MakeReverse
CString s( "abc" );
s.MakeReverse();
ASSERT( s == "cba" );
