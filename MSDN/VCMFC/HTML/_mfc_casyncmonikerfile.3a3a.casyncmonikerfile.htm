CAsyncMonikerFile::CAsyncMonikerFile

CAsyncMonikerFile( );

Remarks

Constructs a CAsyncMonikerFile object. It does not create the IBindHost interface. IBindHost is used only if you provide it in the Open member function.

For a description of the IBindHost interface, see the ActiveX SDK.
