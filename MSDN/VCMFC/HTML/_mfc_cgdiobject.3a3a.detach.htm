CGdiObject::Detach

HGDIOBJ Detach( );

Return Value

A HANDLE to the Windows GDI object detached; otherwise NULL if no GDI object is attached.

Remarks

Detaches a Windows GDI object from a CGdiObject object and returns a handle to the Windows GDI object.
