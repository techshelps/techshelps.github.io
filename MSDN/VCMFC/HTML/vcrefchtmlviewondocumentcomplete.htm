CHtmlView::OnDocumentComplete

virtual void OnDocumentComplete( LPCTSTR lpszURL );

Parameters

lpszURL

A pointer to a string that evaluates to the URL, UNC file name, or a PIDL (a pointer to an item identifier list) that was navigated to.

Remarks

This member function is called by the framework to notify an application that a document has reached the READYSTATE_COMPLETE state.

Not every frame will fire this event, but each frame that fires an OnDownloadBegin event will fire a corresponding OnDocumentComplete event.

The URL indicated by lpszURL can be different from the URL that the browser was told to navigate to, because this URL is the canonicalized and qualified URL. For example, if an application specifies a URL of "www.microsoft.com" in a call to Navigate or Navigate2, the URL passed by OnNavigateComplete2 will be "http://www.microsoft.com/". Also, if the server has redirected the browser to a different URL, the redirected URL will be reflected here.
