CDatabase::Cancel

void Cancel( );

Remarks

Call this member function to request that the data source cancel either an asynchronous operation in progress or a process from a second thread. Note that the MFC ODBC classes no longer use asynchronous processing; to perform an asychronous operation, you must directly call the ODBC API function SQLSetConnectOption. For more information, see Asynchronous Execution in the Platform SDK.
