CWindowImpl::OnFinalMessage

virtual void OnFinalMessage( HWND hWnd );

Parameters

hWnd

[in] A handle to the window being destroyed.

Remarks

Called after receiving the last message (typically WM_NCDESTROY). The default implementation of OnFinalMessage does nothing, but you can override this function to handle cleanup before destroying a window. If you want to automatically delete your object upon the window destruction, you can call delete this; in this function.
