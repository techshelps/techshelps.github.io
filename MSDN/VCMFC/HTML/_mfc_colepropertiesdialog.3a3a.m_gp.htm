COlePropertiesDialog::m_gp

Remarks

A structure of type OLEUIGNRLPROPS, used to initialize the General page of the OLE Object Properties dialog box. This page shows the type and size of an embedding and allows the user access to the Convert dialog box. This page also shows the link destination if the object is a link.

For more information on the OLEUIGNRLPROPS structure, see the OLE documentation.
