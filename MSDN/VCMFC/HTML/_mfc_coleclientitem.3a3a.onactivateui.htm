COleClientItem::OnActivateUI

virtual void OnActivateUI( );

Remarks

The framework calls OnActivateUI when the object has entered the active UI state. The object has now installed its tool bar and menus.

The default implementation remembers the server’s HWND for later GetServerWindow calls.
