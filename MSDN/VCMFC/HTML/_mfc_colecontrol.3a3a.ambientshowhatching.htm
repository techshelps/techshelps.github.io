COleControl::AmbientShowHatching

BOOL AmbientShowHatching( );

Return Value

Nonzero if the hatched pattern should be shown; otherwise 0. If this property is not supported, this function returns nonzero.

Remarks

Call this function to determine whether the container allows the control to display itself with a hatched pattern when UI active. Note that the container is not required to support this property.
