CDataSource::GetInitializationString

HRESULT GetInitializationString( BSTR* pInitializationString, bool bIncludePassword = false );

Return Value

A standard HRESULT.

Parameters

pInitializationString

[out] A pointer to the initialization string.

bIncludePassword

[in] True if string includes a password. Otherwise, false.

Remarks

Retrieves the initialization string of a data source that is currently open. The resulting initialization string can be used to reopen this data source connection at a later time.
