DECLARE_OLECREATE_EX

DECLARE_OLECREATE_EX( class_name )

Parameters

class_name

The name of the control class.

Remarks

Declares a class factory and the GetClassID member function of your control class. Use this macro in the control class header file for a control that does not support licensing.

Note that this macro serves the same purpose as the following code sample:

   BEGIN_OLEFACTORY(CSampleCtrl)
   END_OLEFACTORY(CSampleCtrl)

See Also   BEGIN_OLEFACTORY, END_OLEFACTORY
