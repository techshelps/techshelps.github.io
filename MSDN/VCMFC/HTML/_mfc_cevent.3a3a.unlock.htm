CEvent::Unlock

virtual BOOL Unlock( );

Return Value

Nonzero if the thread owned the event object and the event is an automatic event; otherwise 0.

Remarks

Releases the event object. This member function is called by threads that currently own an automatic event to release it after they are done, if their lock object is to be reused. If the lock object is not to be reused, this function will be called by the lock object’s destructor.
