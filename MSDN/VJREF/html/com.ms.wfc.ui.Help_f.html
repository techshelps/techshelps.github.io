Help Fields

Fields Description
COMMAND_CONTENTS Specifies that the contents of the WinHelp or HTML Help file will be displayed.
COMMAND_FIND Specifies that the search page of the specified URL should be displayed.
COMMAND_INDEX Specifies that the index of the specified URL should be displayed.
COMMAND_TOPIC Specifies that the topic referenced by the specified URL should be displayed.
