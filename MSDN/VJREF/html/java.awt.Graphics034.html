Graphics.drawChars

Graphics.drawChars

Class Overview | Class Members | This Package | All Packages

Syntax
public void drawChars( char data[], int offset, int length, int x, int y )
Parameters
data
the array of characters to be drawn
offset
the start offset in the data
length
the number of characters to be drawn
x
the x coordinate of the baseline of the text
y
the y coordinate of the baseline of the text
Description
Draws the text given by the specified character array, using this graphics context's current font and color. The baseline of the first character is at position (x, y) in this graphics context's coordinate system.
