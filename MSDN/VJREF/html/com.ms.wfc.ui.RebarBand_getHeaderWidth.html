RebarBand.getHeaderWidth

Retrieves the width, in pixels, of the rebar band's header.

Syntax

public int getHeaderWidth()

Return Value

Returns the header width, in pixels, for this rebar band.

Remarks

The band header is the area between the edge of the band and the edge of the child control where rebar band text and images are displayed.
