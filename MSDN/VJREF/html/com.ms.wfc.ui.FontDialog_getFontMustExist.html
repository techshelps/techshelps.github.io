FontDialog.getFontMustExist

Retrieves a boolean value indicating whether the dialog box should indicate an error condition if the user attempts to select a font or style that does not exist.

Syntax

public final boolean getFontMustExist()

Return Value

Returns true if the dialog box indicates an error condition when the user tries to select a font or style that does not exist; otherwise, returns false.
