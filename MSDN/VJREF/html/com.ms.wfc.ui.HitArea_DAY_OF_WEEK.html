HitArea.DAY_OF_WEEK

Specifies a day abbreviation ("Fri", for example).

Syntax

public static final int DAY_OF_WEEK;

Remarks

If the hitArea field of a HitTestInfo object is set to HitArea.DAY_OF_WEEK, the time field is set to the corresponding date in the topmost visible row of the month calendar.
