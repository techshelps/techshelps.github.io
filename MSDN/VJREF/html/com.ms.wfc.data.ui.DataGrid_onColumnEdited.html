DataGrid.onColumnEdited

Triggers the columnEdited event.

Syntax

protected void onColumnEdited( ColumnEvent event )

Parameters

event

A ColumnEvent object that provides the event data.

Remarks

The columnEdited event occurs when a cell exits edit mode. For more information, see the DataGrid Control overview.

When overriding onColumnEdited, be sure to call super.onColumnEdited.
