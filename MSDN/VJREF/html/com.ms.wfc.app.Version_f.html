Version Fields

Fields Description
buildNumber Specifies the build number of the operating system.
major Specifies the major version number of the operating system.
minor Specifies the minor version number of the operating system.
platformId Specifies the platform identifier of the operating system.
text Specifies the operating system-defined version information.
WIN32_NT Determines whether the operating system is Windows NT.
WIN32_WINDOWS Determines whether the operating system is Windows 95 or later.
WIN32S Determines whether the operating system is Win32s running on a 16-bit version of Windows.
