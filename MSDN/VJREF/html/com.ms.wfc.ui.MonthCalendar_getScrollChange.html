MonthCalendar.getScrollChange

Retrieves the scroll rate for a month calendar control.

Syntax

public int getScrollChange()

Return Value

Returns an integer that represents the current scroll rate.

Remarks

The scroll rate is the number of months that the control moves its display when the user clicks a scroll button.
