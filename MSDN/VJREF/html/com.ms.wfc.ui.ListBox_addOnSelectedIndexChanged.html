ListBox.addOnSelectedIndexChanged

Assigns a delegate to the selectedIndexChanged event.

Syntax

public final void addOnSelectedIndexChanged( EventHandler value )

Parameters

value

The EventHandler delegate to be notified when the event occurs.

Remarks

Passing an event handler as a parameter to this method is equivalent to setting the selectedIndexChanged event to that method in the Properties window. The selectedIndexChanged event occurs when the selection in the list box changes.
