HitArea.PREV_MONTH_DATE

Specifies a date from the previous month (partially displayed at the beginning of the currently displayed month).

Syntax

public static final int PREV_MONTH_DATE;

Remarks

If the user clicks here, the month calendar scrolls its display to the previous month or set of months.
