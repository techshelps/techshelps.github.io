MouseEventHandler Delegate

Identifies the method handling the mouseDown, mouseEnter, mouseLeave, mouseMove, or mouseUp event of a Control control.

package com.ms.wfc.ui

public multicast delegate
void MouseEventHandler( Object sender, MouseEvent e );

Remarks

The syntax of the MouseEventHandler delegate specifies the signature for the event handler, where sender is the source of the event and e is a MouseEvent object that provides the event data.

When you create a MouseEventHandler delegate, you specify the method that will handle the event. Once the delegate is assigned to the event, the handler method is automatically invoked whenever the event is triggered.

For more information about delegates, see Handling Events in WFC.
