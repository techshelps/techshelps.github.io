Label.getBorderStyle

Retrieves a value indicating the kind of border the label has.

Syntax

public final int getBorderStyle()

Return Value

Returns an integer that specifies the kind of border the label has. Possible return values are constants defined in the BorderStyle class. If the value is BorderStyle.FIXED_3D, the label has a three-dimensional border.
