ExtenderInfo.getValue

Retrieves the current value of the extender property for the specified extender-extendee pair.

Syntax

public Object getValue( Object extender, Object extendee )

Parameters

extender

The object providing the extender property.

extendee

The object receiving the extender property.

Return Value

Returns the value of the extender property for the extendee object.

Remarks

This method invokes the extender object's get<ExtenderName> method to retrieve the value of the extender. The extender object manages the value of the extender property for each extendee object.
