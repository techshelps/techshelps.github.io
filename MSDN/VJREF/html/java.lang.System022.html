System.loadLibrary

System.loadLibrary

Class Overview | Class Members | This Package | All Packages

Syntax
public static void loadLibrary( String libname )
Parameters
libname
the name of the library.
Description
Loads the system library specified by the libname argument. The manner in which a library name is mapped to the actual system library is system dependent.

Exceptions
SecurityException if the current thread cannot load the specified dynamic library.
Exceptions
UnsatisfiedLinkError if the library does not exist.
