RichEditWordPunctuation Fields

Fields Description
CUSTOM Indicates that the control should use a custom-defined punctuation table.
LEVEL1 Indicates that the control should use the pre-defined Level 1 punctuation table as the default.
LEVEL2 Indicates that the control should use the pre-defined Level 2 punctuation table as the default.
