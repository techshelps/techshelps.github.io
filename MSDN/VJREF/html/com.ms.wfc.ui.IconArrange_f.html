IconArrange Fields

Fields Description
DEFAULT Specifies a default arrangement of icons in which icons remain in the position in which they were last placed by a user.
LEFT Specifies that icons are arranged along the left of the parent control or window.
SNAP_TO_GRID Specifies that icons are snapped to an imaginary grid, dividing the parenting control into invisible sections.
TOP Specifies that icons are arranged along the top of the parent control or window.
