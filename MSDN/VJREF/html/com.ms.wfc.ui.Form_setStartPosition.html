Form.setStartPosition

Specifies the initial position of this form.

Syntax

public final void setStartPosition( int startPosition )

Parameters

startPosition

The initial position of the form. This value must be one of the enumeration constants defined in the FormStartPosition class.

Remarks

The valid values for the startPosition parameter are FormStartPosition.MANUAL, FormStartPosition.WINDOWS_DEFAULT_LOCATION, FormStartPosition.WINDOWS_DEFAULT_BOUNDS, and FormStartPosition.CENTER_SCREEN.
