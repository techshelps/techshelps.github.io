DhWindowInfo.showChannelMode

Informs the browser to open a new window in theater mode and show the channel band.

Syntax

public boolean showChannelMode;

Remarks

In theater mode, the Internet Explorer main window fills the entire screen and displays a toolbar with a minimal set of navigational buttons. A status bar is also provided in the upper-right corner of the screen. Channels, Search, History, and Favorites browser bars appear (if enabled) as an autohide pane on the left edge of the screen in theater mode.
