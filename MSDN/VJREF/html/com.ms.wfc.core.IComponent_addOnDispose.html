IComponent.addOnDispose

Assigns a delegate to the dispose event.

Syntax

public void addOnDispose( EventHandler value )

Parameters

value

The EventHandler delegate to be notified when the event occurs.

Remarks

The dispose event occurs when the component is disposed of through the dispose method. Components that maintain references to other components can register dispose event handlers for the referenced components so that the references can be cleared when the components are disposed of.
