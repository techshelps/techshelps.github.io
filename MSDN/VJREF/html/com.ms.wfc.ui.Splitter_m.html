Splitter Methods

Constructors Description
Splitter Creates Splitter control.

Methods Description
addOnSplitterMoved Assigns a delegate to the splitterMoved event.

addOnSplitterMoving Assigns a delegate to the splitterMoving event.

getMinExtra Retrieves the minimum distance that must remain between the splitter and the edge of the opposite side of the container (or the closest control docked to that side).

getMinSize Retrieves the minimum distance that must remain between the splitter and the container edge that the control is docked to.

onSplitterMoved Triggers the splitterMoved event.

onSplitterMoving Triggers the splitterMoving event.

removeOnSplitterMoved Removes the delegate that is assigned to the splitterMoved event.

removeOnSplitterMoving Removes the delegate that is assigned to the splitterMoving event.

setMinExtra Sets the minimum distance that must remain between the splitter and the edge of the opposite side of the container (or the closest control docked to that side).

setMinSize Sets the minimum distance that must remain between the splitter and the container edge that the control is docked to.

setSplitPosition Sets the distance between the splitter and the container edge that the control is docked to.
