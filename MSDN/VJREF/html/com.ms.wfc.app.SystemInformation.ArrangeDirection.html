SystemInformation.ArrangeDirection Class

Defines constants used to denote the direction the operating system uses to arrange minimized windows.

package com.ms.wfc.app

public static class SystemInformation.ArrangeDirection

Remarks

Use the constants defined in this class with the constants in the SystemInformation.ArrangeStartingPosition class to determine the results of a call to the SystemInformation.getArrange method.
