DhBorderInfo.topWidthUnit

Specifies the unit label of the width of the top border as one of the DhUnits constants.

Syntax

public int topWidthUnit;

Remarks

This label is valid only if width is in value/unit form (such as 20px), and value is not NULL. For more information on unit label constants, see DhUnits.
