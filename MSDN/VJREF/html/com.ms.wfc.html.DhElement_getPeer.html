DhElement.getPeer

Retrieves the underlying HTML peer element for this class.

Syntax

public IUnknown getPeer()

Return Value

Returns the element peer underlying the HTML peer for this class.

Remarks

This method can return null if the element or its parent was never added to a document. The Java code that is running in a trusted environment can freely cast the resulting reference to a specific element peer type (for example, com.ms.wfc.html.om.IHTMLElement). This method always returns null if the code is running in a Web server environment.

Calling this method represents an advanced usage scenario. Typically, you would never use this method.
