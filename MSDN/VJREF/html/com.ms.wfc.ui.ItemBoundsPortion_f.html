ItemBoundsPortion Fields

Fields Description
ENTIRE Specifies that the bounding rectangle of the entire item, including the icon, the item text, and the subitem text (in Report view), should be retrieved.
ICON Specifies that the bounding rectangle of the icon or small icon should be retrieved.
ITEMONLY Specifies that the bounding rectangle of the icon or small icon and the item text should be retrieved.
LABEL Specifies that the bounding rectangle of the item text should be retrieved.
