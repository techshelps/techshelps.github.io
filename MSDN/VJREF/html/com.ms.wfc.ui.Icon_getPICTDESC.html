Icon.getPICTDESC

Retrieves a PICTDESC (Picture Descriptor) for this icon. 

Syntax

public final PICTDESC getPICTDESC()

Return Value

Returns a PICTDESC structure that describes this icon's image.

Remarks

The PICTDESC structure is used by OLE functions to create picture objects.
