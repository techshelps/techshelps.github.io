ItemActivation Fields

Fields Description
ONECLICK Specifies that the user must single-click to activate items, that the cursor changes to a hand with a pointing index finger, and that the item text changes color as the user moves the mouse pointer over the item.
STANDARD Specifies that the user must double-click to activate items and that no feedback is given as the user moves the mouse pointer over an item.
TWOCLICK Specifies that that the user must double-click to activate items and that the item text changes color as the user moves the mouse pointer over the item.
