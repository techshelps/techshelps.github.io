ExtenderInfo.setValue

Sets the value of the extender property for the specified extender-extendee pair.

Syntax

public void setValue( Object extender, Object extendee, Object value )

Parameters

extender

The object providing the extender property.

extendee

The object receiving the extender property.

value

The new value of the extender property.

Remarks

This method invokes the extender object's set<ExtenderName> method, if it exists. The extender object manages the value of the extender property for each extendee object.
