IDataStream.readFloats

Reads a specified number of float values from this stream into a float array.

Syntax

public float[] readFloats( int count )

public void readFloats( float[] buffer, int index, int count )

Parameters

count

The number of float values to read.

buffer

The array to read data into.

index

The starting index in the array.

Return Value

Returns a float array containing the data that was read.

Exceptions

IOException thrown if an I/O error occurs or if the stream does not support reading.

EOFException thrown if the end of the stream is reached while reading the data.

Remarks

The current position is advanced by count * 4 bytes.
