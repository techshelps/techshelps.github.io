Class Overview | Class Members | This Package | All Packages

Syntax
public static void load( String filename )
Parameters
filename
the file to load.
Description
Loads the specified filename as a dynamic library. The filename argument must be a complete pathname.

This method calls the load method in class Runtime.

Exceptions
SecurityException if the current thread cannot load the specified dynamic library.
Exceptions
UnsatisfiedLinkError if the file does not exist.
