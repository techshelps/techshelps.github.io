SplitterEvent Fields

Fields Description
splitX Specifies the x-coordinate of the upper-left corner of the splitter (in client coordinates).
splitY Specifies the y-coordinate of the upper-left corner of the splitter (in client coordinates).
x Specifies the x-coordinate of the mouse pointer (in client coordinates).
y Specifies the y-coordinate of the mouse pointer (in client coordinates).
