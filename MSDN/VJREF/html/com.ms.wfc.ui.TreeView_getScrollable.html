TreeView.getScrollable

Retrieves a boolean value indicating whether a scroll bar is added to the tree view control when there is not enough room to display all the tree view nodes in the control's client area.

Syntax

public boolean getScrollable()

Return Value

Returns true if scroll bars are added to the control when necessary to allow the user to see all the nodes; otherwise, returns false.
