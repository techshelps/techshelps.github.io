SplitterEventHandler Delegate

Identifies the method that will handle the splitterMoved and splitterMoving events of a Splitter control.

package com.ms.wfc.ui

public multicast delegate
void SplitterEventHandler( Object sender, SplitterEvent e );

Remarks

The syntax of the SplitterEventHandler delegate specifies the signature for the event handler, where sender is the source of the event and e is a SplitterEvent object that provides the event data.

When you create a SplitterEventHandler delegate, you specify the method that will handle the event. Once the delegate is assigned to the event, the handler method is automatically invoked whenever the event is triggered. For more information about delegates, see Handling Events in WFC.
