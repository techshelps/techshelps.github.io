Remarks

The columnResize event occurs when a column in the grid is resized, either programmatically or by the user. A ColumnResizeEvent object specifies the column that is being resized and the new width of the column, and provides a flag to cancel the event.

For information about the WFC event model, see Handling Events in WFC.
