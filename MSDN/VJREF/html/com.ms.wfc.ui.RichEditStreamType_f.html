RichEditStreamType Fields

Fields Description
PLAINTEXT Indicates that the stream is a plain text stream that includes spaces in places of OLE objects.
RICH_NOOLEOBJS Indicates that the stream is an RTF stream with spaces in place of OLE objects.
RICHTEXT Indicates that the stream is a Rich Text Format (RTF) stream.
TEXT_TEXTOLEOBJS Indicates that the stream is plain text with a textual representation of OLE objects.
