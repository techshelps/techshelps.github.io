DataStream.fromComStream

Retrieves an IDataStream object for an IStream object.

Syntax

public static IDataStream fromComStream( IStream comStream )

Parameters

comStream

The com.ms.com.IStream object to create an IDataStream object from.

Return Value

Returns an IDataStream object for the specified IStream object, or returns null if the comStream parameter is null.

Remarks

If you have a COM-compatible IStream object but need an IDataStream object, you can use this method to create an adapter.
