DhBrowser Methods

Methods Description
getAppCodeName Retrieves the browser's code name.

getAppName Retrieves the name of the browser.

getAppVersion Retrieves the version of the browser.

getBrowserLanguage Retrieves the browser language setting.

getCookieEnabled Determines whether client-side cookies are enabled in the browser.

getCpuClass Retrieves the CPU class string.

getOnLine Determines whether the browser is connected online.

getPeer Retrieves the underlying peer object for this class.

getPlatform Retrieves the platform setting.

getSystemLanguage Retrieves the system language setting.

getUserAgent Retrieves the Hypertext Transfer Protocol (HTTP) user-agent header string.

getUserLanguage Retrieves the user language setting.
