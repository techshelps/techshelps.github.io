MonthCalendar.removeMonthlyBoldedDate

Specifies that a date will not be displayed in bold every month in the month calendar.

Syntax

public void removeMonthlyBoldedDate( Time date )

Parameters

date

A Time object that represents the date to display in with no emphasis every month. If the specified date is not found in the list of dates to display in bold, this method has no effect. If the specified date occurs more than once in the bolded date list, only the first occurrence is removed.

Remarks

Call the updateBoldedDates method after calling this method.
