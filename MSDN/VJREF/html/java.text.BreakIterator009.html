public abstract int following( int offset )
Parameters
offset
the offset to begin scanning. Valid values are determined by the CharacterIterator passed to setText(). Invalid values cause an IllegalArgumentException to be thrown.
Returns
The first boundary after the specified offset.
Description
Return the first boundary following the specified offset. The value returned is always greater than the offset or the value BreakIterator.DONE
