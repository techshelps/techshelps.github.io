Locale.getDefaultCountry

Retrieves the country code for the principal country in this locale.

Syntax

public int getDefaultCountry()

Return Value

Returns an integer value that represents the default country code for this locale.

Remarks

This method is provided so that partially specified locales can be completed with default values.
