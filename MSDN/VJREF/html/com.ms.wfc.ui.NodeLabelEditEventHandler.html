NodeLabelEditEventHandler Delegate

Identifies the method that will handle the afterLabelEdit event and the beforeLabelEdit event of a TreeView control.

package com.ms.wfc.ui

public multicast delegate
void NodeLabelEditEventHandler( Object sender, NodeLabelEditEvent e );

Remarks

The syntax of the NodeLabelEditEventHandler delegate specifies the signature for the event handler, where sender is the source of the event and e is a NodeLabelEditEvent object that provides the event data.

When you create a NodeLabelEditEventHandler delegate, you specify the method that will handle the event. Once the delegate is assigned to the event, the handler method is automatically invoked whenever the event is triggered.

For more information about delegates, see Handling Events in WFC.
