StatusBarPanelAutoSize.SPRING

Specifies that the panel shares the the available status bar space (the space not taken up by panels with the StatusBarPanelAutoSize.NONE and StatusBarPanelAutoSize.CONTENTS settings) with other panels that have the StatusBarPanelAutoSize.SPRING setting.

Syntax

public static final int SPRING;
