SystemColor.getRGB

Class Overview | Class Members | This Package | All Packages

Syntax
public int getRGB()
Description
Gets the "current" RGB value representing the symbolic color. (Bits 24-31 are 0xff, 16-23 are red, 8-15 are green, 0-7 are blue).
