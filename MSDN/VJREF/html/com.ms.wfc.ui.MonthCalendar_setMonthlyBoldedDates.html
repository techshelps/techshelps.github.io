MonthCalendar.setMonthlyBoldedDates

Sets an array of Time objects that represent dates that are displayed in bold every month in the month calendar.

Syntax

public void setMonthlyBoldedDates( Time[] dates )

Parameters

dates

An array of Time objects that represent the dates to display in bold every month in the month calendar.

Remarks

Calling this method removes existing information about which dates to display in bold.
