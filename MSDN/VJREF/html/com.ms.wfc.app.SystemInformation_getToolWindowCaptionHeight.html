SystemInformation.getToolWindowCaptionHeight

Retrieves the height, in pixels, of a small caption.

Syntax

public static final int getToolWindowCaptionHeight()

Return Value

Returns an integer value that represents the height of a small caption.

Remarks

Use the getToolWindowCaptionHeight method to determine the height of a tool window caption in Windows. This method can assist in manipulating the caption area of a tool window.
