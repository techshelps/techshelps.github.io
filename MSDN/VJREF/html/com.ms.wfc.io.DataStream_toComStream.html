DataStream.toComStream

Retrieves an IStream object for an IDataStream object.

Syntax

public static IStream toComStream( IDataStream dataStream )

Parameters

dataStream

The IDataStream object to create an IStream object from.

Return Value

Returns a com.ms.com.IStream for the specified IDataStream object.

Remarks

If you have an IDataStream object but need a COM-compatible IStream object, you can use this method to create an adapter.

If the specified IDataStream object is null, this method returns null. Otherwise, the getComStream method is called to create an IStream object from the IDataStream object.
