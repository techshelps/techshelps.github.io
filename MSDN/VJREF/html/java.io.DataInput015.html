DataInput.readUTF

Interface Overview | Interface Members | This Package | All Packages

Syntax
public abstract String readUTF() throws IOException
Returns
a Unicode string.
Description
Reads in a string that has been encoded using a modified UTF-8 format.

For an exact description of this method, see the discussion in Gosling, Joy, and Steele, The Java Language Specification.

Exceptions
EOFException if this stream reaches the end before reading all the bytes.
Exceptions
IOException if an I/O error occurs.
Exceptions
UTFDataFormatException if the bytes do not represent a valid UTF-8 encoding of a string.
