Locale.setNegativeCurrencyMode

Sets the mode to use when displaying negative currency values.

Syntax

public void setNegativeCurrencyMode( int value )

Parameters

value

An integer value specifying which negative currency mode to use.

Remarks

This method takes one of the negative currency modes from the following table:

Value Description
0 ($1.1)
1 -$1.1
2 $-1.1
3 $1.1-
4 (1.1$)
5 -1.1$
6 1.1-$
7> 1.1$-
8 -1.1 $ (space before $)
9 -$ 1.1 (space after $)
10 1.1 $- (space before $)
11 $ 1.1- (space after $)
12 $ -1.1 (space after $)
13 1.1- $ (space before $)
14 ($ 1.1) (space after $)
15 (1.1 $) (space before $)
