RebarBand.getAllowVariableHeight

Retrieves a boolean value indicating whether the rebar control can adjust the height of this band.

Syntax

public boolean getAllowVariableHeight()

Return Value

Returns true if the rebar control can adjust the height of this rebar band; otherwise, returns false.
