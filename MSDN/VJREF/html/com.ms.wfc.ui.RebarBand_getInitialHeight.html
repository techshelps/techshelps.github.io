RebarBand.getInitialHeight

Retrieves the initial height of the rebar band, in pixels.

Syntax

public int getInitialHeight()

Return Value

Returns the initial height of the rebar band, in pixels.

Remarks

The value of this property is ignored if the allowVariableHeight property is set to false.
