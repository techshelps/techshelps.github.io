ItemCheckEventHandler Delegate

Identifies the method that will handle the itemCheck event of a CheckedListBox control.

package com.ms.wfc.ui

public multicast delegate
void ItemCheckEventHandler( Object sender, ItemCheckEvent e );

Remarks

The syntax of the ItemCheckEventHandler delegate specifies the signature for the event handler, where sender is the source of the event and e is a ItemCheckEvent object that provides the event data.

When you create a ItemCheckEventHandler delegate, you specify the method that will handle the event. Once the delegate is assigned to the event, the handler method is automatically invoked whenever the event is triggered.

For more information about delegates, see Handling Events in WFC.
