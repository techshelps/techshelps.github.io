FontSize Fields

Fields Description
CELLHEIGHT Represents the size of the font in logical units.
CENTIMETERS Represents the size of the font in centimeters.
CHARACTERHEIGHT Represents the size of the font in logical units.
EM Represents the size of the font as a percentage of the system default font's character height.
EX Represents the size of the font as a percentage of the system default font's cell height.
INCHES Represents the size of the font in inches.
MILLIMETERS Represents the size of the font in millimeters.
PIXELS Represents the size of the font in pixels.
POINTS Represents the size of the font specified in points or 1/72nds of an inch.
