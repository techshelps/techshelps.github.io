IExtenderProvider Interface

Defines the interface for extending properties to other components in a container.

package com.ms.wfc.core

public interface IExtenderProvider

Remarks

Any component that provides extender properties must implement IExtenderProvider. A visual designer can then call IExtenderProvider.canExtend to determine which objects in the container should receive the extender properties.

For more information about extender properties, see ExtenderInfo.
