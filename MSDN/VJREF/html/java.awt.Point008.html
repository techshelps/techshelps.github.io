Point.translate

Class Overview | Class Members | This Package | All Packages

Syntax
public void translate( int x, int y )
Parameters
dx
the distance to move this point along the x axis.
dy
the distance to move this point along the y axis.
Description
Translates this point, at location (x, y), by dx along the x axis and dy along the y axis so that it now represents the point (x + dx, y + dy).
