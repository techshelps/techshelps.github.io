FontDialog.getAllowScriptChange

Retrieves a boolean value indicating whether the user can change the character set specified in the Script combo box to display a character set other than the current one.

Syntax

public final boolean getAllowScriptChange()

Return Value

Returns true if the user can change the character set specified in the Script combo box; returns false if only fonts in the current character set can be displayed.
