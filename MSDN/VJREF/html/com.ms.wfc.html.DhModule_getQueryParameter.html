DhModule.getQueryParameter

Retrieves a query parameter.

Syntax

public String getQueryParameter( String strParamName)

Parameters

strParamName

The query parameter name to retrieve a value for.

Return Value

Returns the parameter value from the Request object. If the parameter does not exist, returns an empty string.
