DhStyleBase.getTextVerticalAlign

Retrieves the current text alignment setting.

Syntax

public int getTextVerticalAlign()

Return Value

Returns the current alignment as one of the following DhAlignment values:

  • BASELINE (default)
  • SUBSCRIPT
  • SUPERSCRIPT
