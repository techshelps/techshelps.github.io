Bitmap.getHandle

Retrieves the Win32 handle to this bitmap.

Syntax

public int getHandle()

Return Value

Returns the handle to this bitmap.

Exceptions

WFCException thrown if the Bitmap object has been disposed through a call to its dispose method.

Remarks

The handle returned by this method is not a copy of the bitmap handle. Therefore, this handle should not be freed by the caller.
