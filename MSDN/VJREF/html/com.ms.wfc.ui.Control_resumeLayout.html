Control.resumeLayout

Resumes normal layout logic.

Syntax

public final void resumeLayout()

protected final void resumeLayout( boolean performLayout )

Parameters

performLayout

Set to true if pending layout requests are immediately executed; set to false if pending layout requests are ignored.

Remarks

Calling this method forces an immediate layout if there are any pending layout requests. Also, calling this method with the performLayout parameter set to true forces an immediate layout if there are any pending layout requests.
