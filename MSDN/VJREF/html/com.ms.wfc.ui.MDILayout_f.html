MDILayout Fields

Fields Description
ARRANGE_ICONS Specifies that all minimized MDI child icons should be arranged within the parent window.
CASCADE Specifies that all MDI child windows should be cascaded in the available MDIClient region.
TILE_HORIZONTAL Specifies that all the MDI child windows should be tiled horizontally horizontally in the available MDIClient region.
TILE_VERTICAL Specifies that all MDI child windows should be tiled vertically in the available MDIClient region.
