HelpEvent Fields

Fields Description
component Specifies the control or menu item that the user wants Help on.
contextId Specifies the Help context identifier of the control.
contextType Specifies whether Help was requested for a menu item or for a control.
controlId Specifies the identifier of the control or menu item.
handled Specifies whether the event is handled.
mousePos Specifies the screen coordinates of the mouse pointer.
