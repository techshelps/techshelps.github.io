IComponentSite Methods

Methods Description
componentChanged Specifies that the associated component has changed.
getComponent Retrieves the component associated with the site.

getContainer Retrieves the container associated with the site.

getDesignMode Determines whether the associated component is currently in design mode.

getDisposing Determines whether the associated component is currently being disposed of.

getName Retrieves the name of the associated component that is maintained by the site.
getService Retrieves a service provided by the site.

setName Sets the name of the associated component that is maintained by the site.
