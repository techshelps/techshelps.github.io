IDataStream.writeInts

Writes the contents of the specified integer array to this stream.

Syntax

public void writeInts( int[] buffer )

public void writeInts( int[] buffer, int index, int count )

Parameters

buffer

The integer array containing the data to write.

index

The starting index in the array.

count

The number of integers to write.

Exceptions

IOException thrown if an I/O error occurs or if the stream does not support writing.

Remarks

The number of bytes written to the stream is buffer.length * 4. The current position is advanced by count * 4 bytes.
