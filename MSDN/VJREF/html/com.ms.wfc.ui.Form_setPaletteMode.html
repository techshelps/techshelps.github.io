Form.setPaletteMode

Specifies the palette mode for this form.

Syntax

public final void setPaletteMode( int paletteMode )

Parameters

paletteMode

The palette mode for this form. This value must be one of the enumeration constants defined in the FormPaletteMode class.

Exceptions

WFCInvalidArgumentException thrown if the paletteMode parameter is not a valid member of the FormPaletteMode class.

Remarks

The valid values for paletteMode parameter are FormPaletteMode.HALFTONE, FormPaletteMode.NONE, and FormPaletteMode.USE_CONTROL.
