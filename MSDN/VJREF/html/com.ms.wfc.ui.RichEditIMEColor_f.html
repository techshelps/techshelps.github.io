RichEditIMEColor Fields

Fields Description
backColor Specifies the background color of the input method editor (IME) composition window.
bold Enables or disables the use of boldfaced text in the input method editor (IME) input window.
italic Enables or disables the use of italics in the input method editor (IME) composition window.
protect Enables or disables the protected attribute on text in the input method editor (IME) input window.
strikeOut Enables or disables the use of strikeout text in the input method editor (IME) composition window.
textColor Specifies the text color for the input method editor (IME) composition window.
underline Enables or disables the use of underlining in the input method editor (IME) composition window.
