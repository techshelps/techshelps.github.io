CoordinateSystem Fields

Fields Description
ANISOTROPIC Specifies that logical units are converted to arbitrary units with arbitrarily scaled axes.
HIENGLISH Specifies that each logical unit is converted to 0.001 inch.
HIMETRIC Specifies that each logical unit is converted to 0.01 millimeter.
ISOTROPIC Specifies that logical units are converted to arbitrary units with equally scaled axes; that is, one unit along the x-axis is equal to one unit along the y-axis.
LOENGLISH Specifies that each logical unit is converted to 0.01 inch.
LOMETRIC Specifies that each logical unit is converted to 0.1 millimeter.
TEXT Specifies that each logical unit is converted to one device pixel.
TWIP Specifies that each logical unit is converted to one twentieth of a point (called a twip).
