DhEventInfo Fields

Fields Description
EVT_LEFTBUTTON Indicates that the left mouse button has been pressed.

EVT_NOBUTTON Indicates that no mouse button has been pressed.
EVT_RIGHTBUTTON Indicates that the right mouse button has been pressed.
