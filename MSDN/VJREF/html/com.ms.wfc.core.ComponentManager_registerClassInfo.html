ComponentManager.registerClassInfo

Registers a ClassInfo class for the specified component.

Syntax

public static void registerClassInfo( Class componentClass, Class classInfo )

Parameters

componentClass

The class of the component to register the ClassInfo for.

classInfo

The ClassInfo class to be associated with componentClass.

Remarks

To find a ClassInfo for a given component, WFC typically looks for an inner class named ClassInfo. However, if a component class cannot define an inner class, the registerClassInfo method can be used to create a custom mapping between the component and a ClassInfo class.
