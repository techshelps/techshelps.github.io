MonthCalendar.getTrailingForeColor

Retrieves the color used to display text for header and trailing-day days (days from the previous and following month that appear on the current calendar).

Syntax

public Color getTrailingForeColor()

Return Value

Returns a Color object that represents the color used to display text for header and trailing-day days
