SystemInformation.getVertScrollBarArrowHeight

Retrieves the height, in pixels, of the arrow image on the vertical scroll bar.

Syntax

public static final int getVertScrollBarArrowHeight()

Return Value

Returns an integer value that represents the height of a vertical scroll bar arrow.

Remarks

Use the getVertScrollBarArrowHeight method to determine the height of the arrow on a vertical scroll bar.
