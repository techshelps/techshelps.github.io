Locale.toString

Class Overview | Class Members | This Package | All Packages

Syntax
public final String toString()
Description
Getter for the programmatic name of the entire locale, with the language, country and variant separated by underbars. If a field is missing, at most one underbar will occur. Example: "EN", "DE_DE", "EN_US_WIN", "DE_POSIX", "FR_MAC"
