Brush.ACTIVECAPTION

Represents a Brush object that is the color of the active caption bar.

Syntax

public static final Brush ACTIVECAPTION;

Remarks

The color represented by this setting is based on the current system settings. If the system setting for this user interface element is modified, this object automatically reflects the new setting.
