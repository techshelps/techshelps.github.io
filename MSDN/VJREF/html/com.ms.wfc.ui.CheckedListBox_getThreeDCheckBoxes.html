CheckedListBox.getThreeDCheckBoxes

Retrieves a boolean value indicating whether the check boxes for the list items have a three-dimensional appearance.

Syntax

public final boolean getThreeDCheckBoxes()

Return Value

Returns true if the check boxes for the list items have a three-dimensional appearance; returns false if the check boxes have a flat appearance.
