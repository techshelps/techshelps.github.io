Edit Menu

Command Description
Undo Reverses the previous command selected from the Edit menu.
Cut Removes the selected text and places it onto the Clipboard.
Copy Copies the selected text onto the Clipboard.
Paste Inserts a copy of the Clipboard contents at the insertion point.
Delete Removes the selected text without placing it onto the Clipboard.
Clear… Clears the contents of either the input or output window. If no connection has been made, so no input window is open, this command is named Clear Output Window, and only clears the output window. If a connection has been made, this command displays a window from which the user can choose to clear the input window or the output window.
Select All Selects the entire contents of the input window.
Word Wrap Turns the word wrap feature on or off for the input window.
