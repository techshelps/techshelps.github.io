Running the ODBC Administrator

To open the ODBC Administrator from the Control Panel

  1. Click Start, point to Settings, and then click Control Panel. The Control Panel will open.

  2. Double-click the ODBC icon. The ODBC Administrator will run.

Note   The icon is labeled "ODBC" on workstations running the Microsoft® Windows NT® operating system, and "32-bit ODBC" on Windows® 95 operating systems.

When the ODBC Administrator is running, the ODBC Data Source Administrator dialog box appears.
