SQLGetConnectAttr (Attr Menu)

The word "Conn:" prefixes the values in the Attribute list to signify these are connection attributes (as opposed to statement attributes).

If you enter an Attribute value that is not recognized by ODBC Test, it is assumed to be a driver-specific connection or statement attribute. So that ODBC Test can correctly display the value returned for this attribute, you must specify the value's C data type in the Parameter Type box.
