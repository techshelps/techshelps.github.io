Making a Query with Query Demo

  1. From the Login menu, choose either Connect or Driver Connect to connect to a data source. If connection is successful, a connection window opens. The title bar displays the values for DSN, HDBC, and HSTMT. The SQL> box enables you to type in SQL statements.

  2. Type a valid SQL statement in the SQL> box. From the Query menu, choose Execute SQL or use the corresponding toolbar button. The SQL statement is executed. If successful, the result set for the query will appear. You can scroll through the result set using the mouse or the keyboard. (You must click in the result set window before you can use the keyboard to scroll the result set.)

  3. If you want to open more than one Statement window, choose New Query from the Query menu to open a new query window, or use the corresponding toolbar button.

  4. To free statement handles and disconnect from the connected data sources, choose Disconnect from the Login menu. The Disconnect From Data Source dialog box appears.

  5. Select all data sources you have connected to and click the Disconnect button.

  6. To exit Query Demo, choose Exit from the Login menu. Query Demo closes all open connections before exiting.
