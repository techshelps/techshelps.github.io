ODBC Menu Version Tab

Specifies the function menus to be displayed in the menu bar of ODBC Test, based on the version of ODBC selected. When the version is ODBC 3.0, the menus contain only ODBC 3.0 functions. When the version is ODBC 2.x, the menus contain only ODBC 2.x functions. When the version selected is Both 2.x and 3.0, the menus contain both ODBC 3.0 and ODBC 2.x functions.
