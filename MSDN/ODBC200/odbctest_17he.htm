Miscellaneous Tab

Logging Level. Specifies the level of error output ODBC Test generates.

Error Logging Level Error Output
W0 Return codes and ODBC Test error messages (TST0xxx).
W1 W0 output and ODBC Test warning messages (TST1xxx).
W2 W1 output and function name and argument values.

Font Button. Displays the Font dialog box in which you can specify the font and point size to be used by all text in the general output, connection, and rowset windows.
