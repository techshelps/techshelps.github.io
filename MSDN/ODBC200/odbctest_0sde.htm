Buffer Lengths Tab

Default Buffer Length. Specifies the default length of output buffers, such as OutConnectionString in SQLDriverConnect or ValuePtr in SQLGetStmtAttr.

Argument Display Length. Specifies the default length of arguments displayed as output by the W2 logging level.

Max Bind Col All Length. Specifies the maximum length of the ParameterValuePtr buffer when the Bind Col All function tool is used.
