File Menu

Command Description
Open Loads text files into the input window.
Save As Saves contents of the input window as a text file.
Save User Options Saves the user options set in the Tools/User Options dialog box in gator32.ini. The Unicode version of ODBC Test saves the user options in GATOR32W.INI. Saving the user options in this way ensures that the option will be valid when ODBC Test is next executed, even if the previous session of ODBC Test was terminated abnormally.
Exit Exits ODBC Test.
