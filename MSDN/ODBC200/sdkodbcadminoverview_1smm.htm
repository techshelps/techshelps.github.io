Microsoft ODBC Data Source Administrator

The ODBC Administrator manages database drivers and data sources. The ODBC SDK Setup program invokes the ODBC Administrator after it has finished copying the essential SDK program files.

The ODBC Administrator is a Control Panel device on workstations running the Microsoft® Windows NT® or Windows® 95 operating system. The ODBC Administrator is both a Control Panel device and a program item.

For information about support for Microsoft data access components, see the Getting Help from Microsoft Technical Support page.
