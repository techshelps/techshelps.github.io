SQLParamOptions

Conformance

Version Introduced:ODBC 1.0
Standards Compliance:Deprecated

Summary

The ODBC 2.0 function SQLParamOptions has been replaced in ODBC 3.x by calls to SQLSetStmtAttr.

Note For more information about what the Driver Manager maps this function to when an ODBC 2.x application is working with an ODBC 3.xdriver, see “Mapping Deprecated Functions” in Appendix G, “Driver Guidelines for Backward Compatibility.”
