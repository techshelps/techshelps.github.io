ADO for Windows Foundation Classes Overview

The new ActiveX® Data Objects for Windows Foundation Classes (ADO/WFC) components provide a robust, powerful set of data access services. Developers can use these components to write Java-based applications and components for Win32® platforms and premier hosting environments such as Microsoft® Internet Explorer 4.0. The following topics provide an overview of ADO/WFC and demonstrate how to use the components:

  • Architecture and Component Model

  • Programming Model

  • Programming Samples

Use these programming examples as building blocks. By combining several of these simple scenarios, you can create more complex application behaviors. These samples discuss various topics, including asynchronous execution, remote business object invocation, transactional data access, and metadata examination.

For updated information on the continuing evolution of ADO/WFC, visit the Microsoft Data Web site at http://www.microsoft.com/data.
