Appendix H

Setup Messages

The server Setup and client Setup programs create log files with information about the setup process. The Microsoft Proxy Server setup log is created as a file named C:\Mpssetup.log on the server computer. The client setup log is created as a file named C:\Mpcsetup.log the client computer.

Customers calling their support provider for assistance should have these log files available when calling. They contain valuable information beyond any error messages that are displayed to users.

The following is an alphabetized listing of messages that can be returned from the server and client Setup programs of Microsoft Proxy Server.

Call to LoadString with ID number failed.

[Client Setup]

An unexplained error in setup occurred.

Consult with your technical support team or network administrator for further assistance. If you contact your support provider regarding this error, be sure to have the setup log file, typically C:\Mpssetup.log, available when you call, and reference the ID number displayed with this message.

Call to LoadString with ID number failed err=error number.

[Client Setup]

An unexplained error in Setup occurred.

Consult with your technical support team or network administrator for further assistance. If you contact your support provider regarding this error, be sure to record the error number displayed with this message and have the setup log file, typically C:\Mpssetup.log, available when you call.

Could not identify disk drives configuration.

[Server Setup]

Indicates that setup was not able to recognize your current drive configuration.

Contact your technical support team for further assistance.

Either the logged account is not permitted to modify the system file: filename or the file is locked by another application.

[Server Setup]

Another Internet service is currently active on the server or the account you are using does not provide administrative access.

Use Internet Service Manager to manually stop all other Internet services. Also, verify that you have administrative rights for your current logon account before running Setup again.

Fail to wait for process termination.

[Server Setup]

A process was ended unexpectedly by the system.

Consult with your technical support team or network administrator for further assistance. If you contact your support provider regarding this error, be sure to have the setup log file (typically C:\Mpssetup.log) available when you call.

Contact your support provider for further assistance. Be sure to have information from the setup log (typically C:\Mpssetup.log) available when you contact your support provider.

Instances of Winsock DLLs appear in both system and windows directory. Setup cannot proceed until there is only one TCP/IP installation.

[Client Setup]

Setup found multiple copies of Winsock.dll. Only one copy of this file can be located within the Windows or System directories.

Delete or rename the additional copies of this file from the Windows and System directory and rerun Setup. If you have multiple TCP/IP client installations installed on this computer, or encounter further complications, contact your technical support team or network administrator for assistance.

Make sure you have enough disk space and that the logon account you are using is permitted to write to C:\Wspclnt\Msplat.txt. Also, this file must contain at least one valid range of address pairs to identify internal IP network clients.
