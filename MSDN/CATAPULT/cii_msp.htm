Installing the Search Components

If Microsoft® Index Server is not already installed, click the Download button.

You will be connected (if you have an Internet connection) to the Microsoft Index Server page, located at www.microsoft.com on the World Wide Web. Follow the instructions on that page to download, install, and configure Microsoft Index Server on your server. After installing Microsoft Index Server, you can reopen the Microsoft® Proxy Server Installation and Administration Guide, open the Search page, and perform a full-text search.

Note   You must have Internet connectivity in order to install Microsoft Index Server. For more information on installing Microsoft Index Server and using it to search the Microsoft Proxy Server Installation and Administration Guide, see “Server Installation.”
