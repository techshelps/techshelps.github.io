Remote-Boot Installations

For greater network security and manageability, Windows 95 can be set up to run from a server, either partially or completely (as in the case of remote booting (RIPL) from a ROM card).

For more information, see Chapter 4, "Server-based Setup for Windows 95," and Chapter 5, "Custom, Automated, and Push Installations."

Tips for the administrator

  • Although Server-based Setup automates the copying of files to the server, you might want to customize the shared directory by adding or removing files that are specific to your network.
