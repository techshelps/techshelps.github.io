2.0 SUMMARY OF WINDOWS 95 RESOURCE KIT UTILITIES

Tools for learning about Windows 95

  • Windows 95 Resource Kit online (WIN95RK.HLP, WIN95RK.CNT) for finding information quickly.
  • Administrator's online tour (TOUR4A.HLP, TOUR4ADM.CNT) for viewing the Windows 95 network and system management features described in the Windows 95 Resource Kit.
  • Information for Macintosh users (MACUSERS.HLP, MACUSERS.HLP) about the Windows 95 interface.
  • Description of Resource Kit utilities (OVERVIEW.HLP, OVERVIEW.HLP)
