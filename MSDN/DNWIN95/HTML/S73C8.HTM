Microsoft Knowledge Base

Microsoft Knowledge Base (KB) is a primary source of product information for Microsoft support engineers and customers, containing detailed how-to articles, plus answers to technical support questions, bug lists, fix lists, and documentation corrections. Knowledge Base articles are available on CD with a membership in the Microsoft Developer Network or a subscription to Microsoft TechNet, and available online from CompuServe (go mskb, or go mdkb for the developer Knowledge Base), the Internet, and Microsoft Download Service, described later in this appendix.
