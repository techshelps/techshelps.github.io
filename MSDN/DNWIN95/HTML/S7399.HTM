Microsoft Natural Keyboard Keys

The following table summarizes the shortcut keys available on the Microsoft Natural Keyboard™.

Shortcut

Action

WIN+R

WIN+M

Minimize All

Start Windows Explorer

WIN+F

Find files or folders

CTRL+WIN+F

Find computer
