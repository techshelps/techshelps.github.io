Backup Agents

Windows 95 includes backup agents for the Cheyenne® and Arcada® server-based backup systems. With the appropriate server software, Windows 95 can be easily backed up to a NetWare server with these agents.

For more information, see Chapter 16, "Remote Administration."

Tips for the administrator

  • To manage network computers remotely, enable the Remote Registry and Network Monitor agent, and assign remote administration privilege for each computer when installing Windows 95.
