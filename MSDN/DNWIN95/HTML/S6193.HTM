Briefcase

Briefcase is a tool that allows mobile users to easily track and update copies of files stored on two or more computers— usually an office computer and a portable computer. Users put the files that need to stay in sync into Briefcase before going on the road. When users return, connectto the network, and open Briefcase, Windows 95 prompts them to synchronize the files. Briefcase then updates the file on the network to match the file on the portable computer.

Tips for the administrator

  • Although Dial-Up Networking is a powerful productivity feature for users, it might present security and control concerns. To disable Dial-Up Networking, use system policies to centrally configure Windows 95. Also, you can use additional security, such as callback or hardware signature devices.
