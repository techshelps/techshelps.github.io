MS-DOS and Windows 16-bit Applications

Because of its 32-bit system components, Windows 95 provides a more stable environment than Windows 3.1 for running your Windows-based and MS-DOS –based applications.

For more information, see Chapter 22, "Application Support."

Tips for the administrator

  • Even though Windows 95 provides a more stable and functional environment for your existing applications, to take advantage of multitasking and long filenames, you'll want to upgrade to the latest 32-bit applications.
