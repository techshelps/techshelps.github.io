Chapter 9 Windows 95 on NetWare Networks

This chapter presents information for installing and configuring Windows 95 on Novell® NetWare® networks.

Note

For computers that use Microsoft Client for NetWare, all the files required for networking are included with Windows 95. However, Windows 95 does not include any Novell-supplied components required for real-mode NETX and VLM. For information about obtaining updates for Novell-supplied files, see "Obtaining Current Novell-Supplied Files" later in this chapter.

For information about configuring Novell-supplied components and running NetWare utilities, consult your Novell documentation. For information about licensing requirements, see your Novell NetWare license agreement.
