Guided Tour for Administrators

This chapter of the Windows 95 Resource Kit explains how Windows 95 can help you — network administrators and systems-support professionals — to realize benefits such as reducing support costs, increasing control of user desktops, improving user productivity, and achieving a smooth and easy migration within your organization.

Each page of the Guided Tour covers a Windows 95 feature that yields a specific benefit for managing and supporting corporate networks. For each feature described, the tour provides an illustrated summary plus tips for the administrator. Use this Guided Tour to learn how Windows 95 can reduce the total cost of ownership for personal computers in your organization.
