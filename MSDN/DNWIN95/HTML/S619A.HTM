Online Services

Similar to Windows 3.1, Windows 95 supports a wide variety of online services. For those unfamiliar with online services, The Microsoft Network introduces users to online features such as product information and chat forums.

For more information about messaging services in Windows 95, see Part 6, "Communications."

Tips for the administrator

  • Many of the messaging services in Windows 95 (including Microsoft Exchange, Microsoft Fox, and The Microsoft Network) are optional during installation, so you'll need to select them if you want them installed. You also need to set up the appropriate protocols and dial-up connections for access to the Internet.
