PR_EMS_AB_COMPUTER_NAME

The PR_EMS_AB_COMPUTER_NAME property contains the name of the computer running a mailbox agent or gateway service.

Quick Info

Header file: EMSABTAG.H
Applies to: Mailbox-Agent, Mail-Gateway, and Add-In objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8065
Property type: PR_EMS_AB_COMPUTER_NAME PT_TSTRING

PR_EMS_AB_COMPUTER_NAME_A PT_STRING8

PR_EMS_AB_COMPUTER_NAME_W PT_UNICODE
Access: Read/write
Unicode: Yes

Values

1
Minimum length
256
Maximum length
