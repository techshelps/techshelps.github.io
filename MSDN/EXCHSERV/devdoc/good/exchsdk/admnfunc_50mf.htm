Performance Monitoring

The Windows NT Performance Monitor is a utility that displays statistics for analyzing system performance. The Microsoft Exchange Server performance monitoring system enables a gateway to send statistics to the Windows NT Performance Monitor.

Using this mechanism to set up your gateway’s performance reporting requires a lot less effort than creating a performance monitoring system yourself. In addition, displaying performance information in the Windows NT Performance Monitor allows the gateway’s performance statistics to be directly compared and graphed against other system performance statistics such as memory footprints and CPU usage.
