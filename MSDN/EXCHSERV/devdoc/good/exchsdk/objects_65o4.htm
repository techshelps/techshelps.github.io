Remote-Address Object

The Remote-Address object represents any user outside of the Microsoft Exchange organization. For example, if you communicate with Internet or X.400 users, they are represented by this object.

This object inherits from a Mail-Recipient object, which means that it can also be contained within a Distribution-List object.

Because Remote-Address objects appear to MAPI as messaging user objects, use the MAPI IMailUser interface to access a Remote-Address object.

For more information on accessing Remote-Address objects through MAPI and through directory access functions, see Using MAPI with Directory Objects.
