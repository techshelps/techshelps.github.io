PR_FOLDER_DESIGN_FLAGS

The PR_FOLDER_DESIGN_FLAGS property contains a reason for the status of the folder.

Quick Info

Header file: EDKMDB.H
Applies to: Public folder objects (information store objects)
Identifier: 0x3fe2
Property type: PT_LONG
Access: Read/write

Remarks

The PR_FOLDER_DESIGN_FLAGS property is only present when the PR_DESIGN_IN_PROGRESS optional property is present. No access to the folder is possible during this time.
