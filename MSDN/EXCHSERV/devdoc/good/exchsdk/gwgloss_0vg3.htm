S

secondary proxy address
A proxy address used when receiving mail from the foreign system. A user can have more than one secondary proxy address. See also primary proxy address.
service control manager
A Windows NT Server process that manages all the services in the Windows NT Registry.
setup program
A utility program whose main function is to copy all necessary files from floppy disks or CD to the server’s hard disk. The setup program must also ensure that the components are recognized by each other and can communicate with each other.
Simple Mail Transfer Protocol (SMTP)
The message standard used by the Internet to transfer mail.
site
One or more Microsoft Exchange Server computers that provide services to a set of users. Sites can be centrally managed and can span physical locations.
site proxy address
The portion of a proxy address that corresponds to the site rather than to the individual user. For example, @trager.com might be the site proxy address for the Trager company.
