Microsoft Exchange Server Properties

The following properties are presented in alphabetical order. Microsoft Exchange Server objects have these properties, as well as their MAPI properties. For information on MAPI properties, see the MAPI Programmer’s Reference.

Information store properties are defined in the EDKMDB.H header file. Address book properties are defined in the EMSABTAG.H header file. Personal folder store properties are defined in the MSPST.H header file.

For a list of properties that apply to a certain object, see that object’s documentation.
