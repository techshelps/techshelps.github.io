SERVINST: Installing a Service

The SERVINST sample application demonstrates how to install a service of Windows NT Server on a local Microsoft Exchange Server computer or a Microsoft Exchange Client computer. After you create a new Microsoft Exchange Server service (a gateway or mailbox agent), you must install its profile, the service itself, and its extension DLL. For more information about installing a mailbox agent, see Using a Batch File to Install SMBAGENT.

    To build SERVINST
  1. Change to the directory containing the SERVINST source files:

    \BKOFFICE\SAMPLES\EXCHANGE\EDKINSTU

  2. Type NMAKE.
