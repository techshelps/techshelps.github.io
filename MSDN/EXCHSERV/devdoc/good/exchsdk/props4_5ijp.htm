PR_RULE_NAME

The PR_RULE_NAME property contains a name string that is used for provider self-documentation only.

Quick Info

Header file: EDKMDB.H
Applies to: Rule table objects (information store objects)
Identifier: 0x6682
Property type: PT_STRING8
Access: Read-only

Remarks

The PR_RULE_NAME property is not used or displayed by the Microsoft Exchange Client provider.
