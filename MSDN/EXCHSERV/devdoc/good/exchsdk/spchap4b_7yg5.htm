StartRecurringTime

The StartRecurringTime property returns the start time for a recurrence pattern.

Applies To

Recurring Appointment, RecurringEvent, and RecurringTask objects

Data Type

Time

Access

Read/write

Remarks

The StartRecurringTime and EndRecurringTime properties determine the range of time for the recurring item on every instance of the recurrence pattern. For example, for a recurring appointment from 9:00 A.M. to 10:00 A.M., the StartRecurringTime value is 9:00 A.M.
