Directory Objects

The Microsoft Exchange Server directory is exposed through a MAPI address book provider. Therefore, directory objects can be accessed through MAPI interfaces. This section provides reference information on accessing the most commonly used directory objects through MAPI. For an overview of this process, see Using MAPI with Directory Objects.

You can also use Microsoft Exchange Server directory access functions to manipulate directory objects. For reference information, see About the Directory Schema. For general information, see Using Directory Access Functions.

For general information on the Microsoft Exchange Server directory, see About the Directory. For information on using directory interfaces to manipulate directory objects and attributes, see Working with the Directory.
