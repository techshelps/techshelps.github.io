PR_RULE_LEVEL

The PR_RULE_LEVEL property contains the exit level of a rule.

Quick Info

Header file: EDKMDB.H
Applies to: Rule table objects (information store objects)
Identifier: 0x6683
Property type: PT_LONG
Access: Read/write

Remarks

When ST_EXIT_LEVEL is set on a rule, processing skips the rest of the rules.
