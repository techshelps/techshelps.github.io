About the Registry Editor

Windows NT stores its configuration information in a database (the registry) that is organized in a tree format. Registry Editor enables you to inspect and modify the registry.

    To run Registry Editor
  1. Click Start, and then click Run.
  2. Type regedt32 in Open.

To run Registry Editor from Windows NT Explorer, double-click Reget32.exe in the %SystemRoot%/System32 folder.
