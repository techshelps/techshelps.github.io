PR_SVR_GENERATING_QUOTA_MSG

The PR_SVR_GENERATING_QUOTA_MSG property contains the identity of the server that generates a message when a user exceeds the storage quota limit.

Quick Info

Header file: EDKMDB.H
Applies to: Message objects (information store objects)
Identifier: 0x3ff7
Property type: PT_STRING8
Access: Read-only
