Adding New Property Pages

The primary duties of a Microsoft Exchange Server administrator are basic operations such as creating mailboxes, managing the basic components of Microsoft Exchange Server, and directory object management. Administrators perform most of these tasks—including all server control and maintenance actions—by using property sheets to manipulate objects in the Microsoft Exchange Server directory.

Each property sheet is made up of one or more property pages. Within the property sheet, controls are visible only on the front property page. Additional property pages are displayed one at a time when the administrator selects a tab.
