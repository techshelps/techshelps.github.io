Using the Gateway Programmer’s Reference

This reference provides conceptual information about Microsoft® Exchange Server gateways and describes the gateway development tools that are included in the Microsoft Exchange Server Software Development Kit (SDK). Microsoft developers and third parties, including Microsoft Exchange Server customers with in-house customization expertise, can use these tools to create gateways between Microsoft Exchange Server and foreign e-mail systems.

The primary goal of this documentation is to provide gateway developers with the concepts and instructions needed to design and implement a Microsoft Exchange Server gateway.

For successful application development, this guide should be used in conjunction with the Messaging Application Programming Interface (MAPI) Programmer’s Reference, the Microsoft® Win32® SDK documentation, and the Microsoft Exchange Server Programmer’s Reference.
