PR_EMS_AB_USN_SOURCE

The PR_EMS_AB_USN_SOURCE property contains the PR_EMS_AB_USN_CHANGED property that the object had on the server from which it was replicated.

Quick Info

Header file: EMSABTAG.H
Applies to: All directory objects. These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8157
Property type: PT_LONG
Access: Read-only

Remarks

When an object is replicated to a new server, the source object’s PR_EMS_AB_USN_CHANGED property is copied to the destination object’s PR_EMS_AB_USN_SOURCE property. The destination object’s PR_EMS_AB_USN_CREATED and PR_EMS_AB_USN_CHANGED properties are set using the destination server’s USN counter, as though the object had been created or modified on the destination server.
