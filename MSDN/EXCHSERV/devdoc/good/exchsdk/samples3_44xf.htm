SMBAGENT Build and Install Options

Several options are available to you for building SMBAGENT and installing the mailbox agent. You can build all the executables files of the mailbox agent together or separately. Sample applications can be placed in the directories required for MBSETUP installation by executing the makefile in the MBSETUP directory, or they can be manually moved to the correct directories. If you use the batch file or keyboard commands to install the mailbox agent, it is not necessary to move the files.

Three installation methods are available:

  • Sample mailbox setup program You can use MBSETUP (the sample mailbox agent setup program) to install the mailbox agent. Its interactive user interface allows you to install the server without manually entering the four keyboard commands. This option allows you to answer questions on the forms, rather than type in command line instructions.
  • Keyboard commands You can type in keyboard commands to call each sample application.

·    Batch file You can create a batch file to complete the four install steps. This procedure is illustrated with a sample batch file and instructions for keyboard commands.
