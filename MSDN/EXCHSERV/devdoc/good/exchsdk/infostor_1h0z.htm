Configuring Public Folders

Configuring public folders consists of a combination of the following actions:

  • Setting ACLs on the public folder to control access. For more information, see Manipulating ACLs.
  • Configuring public folder replication. See the Microsoft Exchange Server Administrator’s Guide and the Microsoft Exchange Server Concepts and Planning Guide and Public Folder Replication.
  • Setting rules on the public folder. For more information, see Working with Rules.
