Interfaces Used for Incremental Change Synchronization

The incremental change synchronization interface consists of three OLE interfaces:

  • IExchangeExportChanges provides the interface for the synchronizer (for both hierarchy and contents data).
  • IExchangeImportHierarchyChanges provides the interface used by the collector for folder imports (hierarchy table).
  • IExchangeImportContentsChanges provides the interface used by the collector for message imports (contents table).
