PR_DESIGN_IN_PROGRESS

The PR_DESIGN_IN_PROGRESS property contains a flag that indicates a folder is being modified and can only be accessed by the owner.

Quick Info

Header file: EDKMDB.H
Applies to: Public folder objects (information store objects)
Identifier: 0x3fe4
Property type: PT_BOOLEAN
Access: Read/write

Values

0
FALSE = Folder is not being modified
1
TRUE = Folder is being modified
