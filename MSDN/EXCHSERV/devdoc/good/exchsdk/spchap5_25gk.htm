UserEntryId

The UserEntryId data member returns a unique identifier that is used to identify the user to the underlying mail system.

Applies To

Application and Schedule objects

Data Type

String

Access

Read-only

Remarks

The UserEntryId data member is an opaque identifier whose format is specific to the mail system. It uniquely identifies the currently logged-on Schedule+ user. Applications can use this data member to identify a user to the mail system, such as when sending mail.

Use the UserSearchKey data member to compare two user identities. This data member is set by the underlying mail system.
