IsUserDistributionList

The IsUserDistributionList property returns a Boolean value indicating that the user associated with an item is actually a group of users specified by a mail distribution list.

Applies To

Attendee object

Data Type

Boolean

Access

Read/write

Remarks

When an Attendee item is created and its UserEntryId property is set to a distribution list, the IsUserDistributionList property must be set to True on the Attendee item. If it is not, Schedule+ will interpret the Attendee object as a single user and will not expand the name to a list of individual users.
