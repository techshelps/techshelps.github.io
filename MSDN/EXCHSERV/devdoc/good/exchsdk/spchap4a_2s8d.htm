DailyAlarm

The DailyAlarm property returns a Boolean value that determines whether daily reminders are displayed.

Applies to

Schedule object

Data Type

Boolean

Access

Read/write

Remarks

The DailyAlarm property controls the Daily Reminder dialog box that appears every day if there is an active task for the day. This is a user option that can also be changed using the Options dialog box.

Note This option should not be modified unless it is important to do so.
