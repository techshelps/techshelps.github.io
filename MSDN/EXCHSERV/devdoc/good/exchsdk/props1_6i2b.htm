PR_CLIENT_ACTIONS

The PR_CLIENT_ACTIONS property contains the rule actions.

Quick Info

Header file: EDKMDB.H
Applies to: Message objects (information store objects)
Identifier: 0x6645
Property type: PT_BINARY
Access: Read/write

Remarks

The PR_CLIENT_ACTIONS property is used only on a deferred action message (DAM) to store the action data for the Microsoft Exchange Client to carry out when one or more deferred actions rules have fired on the mailbox. This property should not be used for other purposes. Developers who wish to provide custom client-side rule behavior should implement custom rule actions, which do not need to access this property directly and are called by the client when it has received a DAM.
