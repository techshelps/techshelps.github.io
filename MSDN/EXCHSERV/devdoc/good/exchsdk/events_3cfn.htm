About Event Logs

Choosing the Logs button displays the most recent log messages resulting from this particular event, in this particular folder. These messages are stored as properties—along with your script—in the hidden message in the application folder. You can have your script write messages to this log by assigning string values to the Script.Response intrinsic object.

Log data is displayed in a simple list box control. By default, there is a maximum of 32K of log text that can be stored. There is no mechanism to clear the logs; they replace each other—first in, first out—when the 32KB limit is reached.
