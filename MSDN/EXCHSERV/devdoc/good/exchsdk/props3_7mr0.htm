PR_EMS_AB_IS_MEMBER_OF_DL

The PR_EMS_AB_IS_MEMBER_OF_DL property contains the distinguished names of all distribution lists of which the object is a member.

Quick Info

Header file: EMSABTAG.H
Applies to: All directory objects. These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8008
Property type: PR_EMS_AB_IS_MEMBER_OF_DL PT_OBJECT
