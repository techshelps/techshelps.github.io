PR_EMS_AB_FILTER_LOCAL_ADDRESSES

The PR_EMS_AB_FILTER_LOCAL_ADDRESSES property determines if an address matches a local address space (ADMD, Country, PRMD, Organization, or Organization Units [1-4]). If a local mail recipient cannot be found in the directory, a nondelivery report is generated for the message.

Quick Info

Header file: EMSABTAG.H
Applies to: Site-Addressing and MTA objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x80ad
Property type: PT_BOOLEAN
Access: Read/write

Values

0
FALSE (default)
1
TRUE
