ExportToWatchPhones

The ExportToWatchPhones property returns a combination of flags indicating which of the contact’s telephone numbers should be exported to the Timex Data Link watch.

Applies To

Contact object

Data Type

Long

Access

Read/write

Remarks

The ExportToWatchPhones property is reserved for internal use by the Schedule+ user interface. It can have one or more of the following flags set.

Value Description
1 Business phone 1
2 Business phone 2
4 Home 1
8 Home 2
16 Fax
32 Mobile
64 Pager
128 Assistant
