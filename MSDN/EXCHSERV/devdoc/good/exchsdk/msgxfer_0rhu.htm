Gateway to Microsoft Exchange Server Message Transfer

The following steps are performed when the gateway transfers a message to Microsoft Exchange Server:

  1. Microsoft Exchange Server creates a new MTS message and assigns it a message ID. It also assigns the same message ID back to the PR_MTS_ID of the message the gateway delivered to its MTS-IN folder.
  2. The PR_CONTENT_CORRELATOR and PR_CONTENT_IDENTIFIER properties in the MTS envelope are assigned from the gateway transfer envelope’s PR_CONTENT_IDENTIFIER.
