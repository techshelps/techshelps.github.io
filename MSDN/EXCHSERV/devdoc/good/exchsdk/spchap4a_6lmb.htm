AutoProcessMeetingRequests

The AutoProcessMeetingRequests property returns a Boolean value that indicates whether Schedule+ should automatically process incoming meeting requests for the user.

Applies To

Schedule object

Data Type

Boolean

Access

Read/write

Remarks

Only the owner can write to the AutoProcessMeetingRequests property — everyone else has normal access. Schedule+ must be running in group-enabled mode.

If this property is set to True and Schedule+ receives a meeting request, the meeting is automatically booked onto the user’s schedule and a confirmation response is sent to the meeting request sender. If a conflicting meeting is already booked onto the user’s schedule when the request is received, the meeting is not booked and the request remains in the user’s Inbox.
