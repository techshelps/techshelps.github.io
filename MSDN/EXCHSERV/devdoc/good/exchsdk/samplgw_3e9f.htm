Installing Gateway Components

The following sample applications relate to installing gateway components. They are located in the \BKOFFICE\SAMPLES\EXCHANGE\EDKINSTU directory.

Note In practice, gateway developers usually combine this functionality into a single setup program. They are kept separate here to make the sample code simpler to understand.

ADDRINST: Installing an Addr-Type Object

EXTNINST: Installing an Administrator Extension

GWINST: Installing a Mail-Gateway Object

PERFINST: Installing Performance Monitoring

PROFINST: Installing a MAPI Profile

SERVINST: Installing a Service

TMPLINST: Installing an Address Entry Template

XLATINST: Installing a Message Conversion
