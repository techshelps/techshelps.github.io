PR_EMS_AB_COUNTRY_NAME

The PR_EMS_AB_COUNTRY_NAME property is for X.500 compatibility and is not used by Microsoft Exchange Server.

Quick Info

Header file: EMSABTAG.H
Applies to: Mail-Gateway and Country objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8069
Property type: PR_EMS_AB_COUNTRY_NAME PT_TSTRING

PR_EMS_AB_COUNTRY_NAME_A PT_STRING8

PR_EMS_AB_COUNTRY_NAME_W PT_UNICODE
Access: Read/write
Unicode: Yes
