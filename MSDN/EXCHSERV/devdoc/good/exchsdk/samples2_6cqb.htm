MBLSAMPL Syntax and Parameters

    To run MBLSAMPL

Type the following on the command line:

MBLSAMPL <profile name> [<password>] [/HELP or /?]

MBLSAMPL Parameters

Flag Settings
Profile Specifies the MAPI profile name associated with this logon.
Password Specifies the password for this profile, if one exists.
/HELP or /? Displays Help information.
