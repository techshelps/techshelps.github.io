Intended Audience

To develop gateways for the Microsoft Exchange Server environment, your background should include:

  • Extensive knowledge of the messaging system for which you are developing the gateway.
  • Proficiency with C or C++ programming in the Microsoft® Windows NT® Server network operating system environment.
  • Familiarity with basic features and capabilities of Microsoft Exchange Server and Microsoft Exchange Client. Reviewing the Microsoft Exchange Server Administrator’s Guide and the Microsoft Exchange Server Concepts and Planning Guide is highly recommended.
  • Familiarity with MAPI.
  • Familiarity with network environments.
