Importing Data into Microsoft Exchange Server

To successfully import a new object into the directory, all the required attributes for this object must be defined. Some of these attributes are already defined because they are imported. The remaining attributes must also be defined before the object is successfully imported into the directory.

The object’s class determines which attributes must be defined. For example, if you create a Mail-Recipient object, a value must be supplied for its Delivery Mechanism attribute, Common-Name attribute, and Object Class attribute. To locate a complete list of required attributes for a directory object, see the SCHEMA.HLP file included in this SDK, or use the Microsoft Exchange Server Administrator program.

    To find a complete list of required attributes for a directory object
  1. Start the Administrator program with the /RAW switch set.
  2. Select Raw Attributes from the File menu.
  3. In the List Attributes of Type box, select Required. All the required attributes will be displayed.

For more information about creating a process for exporting and importing data, see Creating a Directory Synchronization Agent.
