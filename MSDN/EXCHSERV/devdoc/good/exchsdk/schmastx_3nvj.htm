String(Numeric)

In this syntax strings match as in String(Printable), except that in a deviation from X.500, all space characters are ignored in comparisons. Corresponds to X.500 Numeric-String-Syntax. The Touchdown DS does not do value checking to ensure that only numerals and spaces appear in values of this syntax.

Value checking

The DS will accept any contents for a numeric string; it will not attempt to verify that they are indeed numeric. Individual attribute types may associate a minimum and maximum size with any string, and those size limits are enforced; any value outside the legal size range will not be accepted.
