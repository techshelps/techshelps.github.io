MAPI and Directory Access Pseudo-Attributes

The directory access functions that are used for importing define several pseudo-attributes that are not part of the Microsoft Exchange Server directory schema. Using these attributes can simplify some operations, such as assigning privileges to an account to determine its access to a newly created object. These attributes are described in Using Directory Access Pseudo-Attributes.

Although the standard directory schema attributes can also be accessed as MAPI properties (see Accessing Directory Attributes with MAPI), the directory access pseudo-attributes have no direct MAPI equivalents.
