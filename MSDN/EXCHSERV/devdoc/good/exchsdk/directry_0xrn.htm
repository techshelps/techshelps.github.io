Sample Applications Using Directory Access Functions

Several sample applications included with the Microsoft Exchange Server SDK use the directory access functions. Observing the code in these applications shows how to use the directory access functions. Because these sample applications are functional, you can also use them as they are or program new functionality into them. They include the following:

  • DIRSYNC: Importing and Exporting Mailboxes Uses the directory access functions to transfer address and other information between foreign systems and Microsoft Exchange Server.
  • DSEXPORT: Exporting Directory Objects Exports objects into a file, using command-line parameters.
  • DSIMPORT: Importing Directory Objects Imports objects from a file, using command-line parameters.
