GetGWNewMailEvent

The GetGWNewMailEvent function obtains a new mail event.

Quick Info

Header file: GWMAIN.H
Library: GWMAIN.LIB 

HANDLE GetGWNewMailEvent(
  VOID  
);

Parameters

None.

Return Values

A handle to a new mail event.

Remarks

If no events are pending, the GetGWNewMailEvent function returns NULL.
