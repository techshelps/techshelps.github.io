PR_EMS_AB_NETWORK_ADDRESS

The PR_EMS_AB_NETWORK_ADDRESS property contains a list of the names by which a server is known to the various transports in use by the network.

Quick Info

Header file: EMSABTAG.H
Applies to: Computer objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8170
Property type: PR_EMS_AB_NETWORK_ADDRESS PT_MV_TSTRING

PR_EMS_AB_NETWORK_ADDRESS_A PT_MV_STRING8

PR_EMS_AB_NETWORK_ADDRESS_W PT_MV_UNICODE
Access: Read/write
Unicode: Yes
