Using Template Objects

When importing objects into the directory, you can specify a template object that contains default information. This information — in the form of attributes — may be assigned to objects being imported. During the import operation, the import function reads the attributes associated with the template's mailbox and uses this data as defaults for attribute values not specified in the import file.

The advantage of specifying templates for importing recipients is that administrators can select a user that closely approximates the basic configuration to be inherited by all the users about to be imported.

An example of using an object template is located in Example Directory Access Import File and File Formats.
