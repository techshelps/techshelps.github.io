HrMAPICopyMessage

The HrMAPICopyMessage function copies a message from one folder to another folder.

Quick Info

Header file: EDKMAPI.H
Library: EDKMAPI.LIB 

HRESULT HrMAPICopyMessage(
  LPMAPIFOLDER lpSrcFolder,  
  LPMAPIFOLDER lpDstFolder,  
  ULONG cbeid,               
  LPENTRY lpeid              
);

Parameters

lpSrcFolder
Input parameter. Points to a MAPI IMAPIFolder interface containing the source folder.
lpDstFolder
Input parameter. Points to an IMAPIFolder interface containing the destination folder.
cbeid
Input parameter. Number of bytes in entry identifier.
lpeid
Input parameter. Points to entry identifier.

Return Values

See Return Values.

Remarks

For information on the IMAPIFolder interface, see the MAPI Programmer's Reference.
