Automatic File Copying

The BASEMAKE.INC file automatically copies compiled import libraries (LIB) and static libraries (LIB) into the default \LIB\I386 directory under the \BKOFFICE root directory. You can change this default behavior by editing the \BKOFFICE\include\BASEMAKE.INC file and modifying the Inference Rules section.

Trying to compile with incompatible libraries causes linker errors. For information about including only compatible libraries, see Choosing Runtime Libraries.
