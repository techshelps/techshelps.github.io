Intended Audience

To develop applications for the Microsoft Exchange Server environment, your background should include:

  • Proficiency with C or C++ programming in the Windows environment and the Microsoft® Windows NT® Server network operating system.
  • Familiarity with basic features and capabilities of Microsoft Exchange Server and the Microsoft Exchange Client. Reviewing the Microsoft Exchange Server Administrator's Guide, the Microsoft Exchange Server Concepts and Planning Guide, and the Microsoft Exchange Server Application Designer's Guide is highly recommended.
  • Familiarity with MAPI.
  • Familiarity with network environments.
