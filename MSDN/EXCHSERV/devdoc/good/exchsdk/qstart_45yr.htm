Quick Start Procedures

Topics in the Quick Start Procedures folder examine tasks commonly performed using the Microsoft Exchange Server SDK. Each topic is divided into three sections: design tasks, implementation tasks, and references to sample source code.

These procedures are not intended to supply the actual implementation details. Instead, they provide a high-level overview of design decisions you will need to make and refer you to the necessary topics.

The procedures in this section are divided into two categories: server applications and client applications.

Following the procedures is a reference list of programming tasks covered in the Microsoft Exchange Server Programmer's Guide.
