ItemIdPath

The ItemIdPath property returns a list of item identifiers that specify a path to the object through the Schedule+ table hierarchy.

Applies To

Alarm object

Data Type

String

Access

Read-only

Remarks

A path consists of a set of tuples (a table and an item identifier). Because tables can be nested, each table and item identifier specifies a level into the Schedule+ table hierarchy. The ItemIdPath and TablePath properties make up a path.
