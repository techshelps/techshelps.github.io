Implementing Extensions with the CAdminDialog Class

The Microsoft Exchange Server SDK provides a C++ interface that you can use to implement Administrator extension DLLs. It is defined in the exadmin.H file and is based on the MFC CDialog class. It handles the standard interface handshaking.

For examples that show how this class is used, see the sample applications QUEVIEW: Listing Gateway Folders and SMBAGENT: Creating a Sample Mailbox Agent.
