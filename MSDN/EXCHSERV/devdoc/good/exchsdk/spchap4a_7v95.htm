DefaultProjectPriority

The DefaultProjectPriority property returns the priority level assigned to new projects.

Applies To

Schedule object

Data Type

Long

Access

Read/write

Remarks

A Schedule+ priority is one of two types: 0-9 or A0-Z9. In each case, the priority is stored in a loword specifying the lower part, and a hiword specifying the upper part. Both parts are stored as ASCII characters.

This property is set by the user in the Schedule+ Options dialog box. For more information, see the Schedule+ Help.
