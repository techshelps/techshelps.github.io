Working with Text in Rich Text Format

The body of a message in a Microsoft Exchange Server information store is kept in the PR_RTF_COMPRESSED property. As the name implies, this property contains compressed text that includes rich text formatting information. For this reason, the body of any message that will at some time be stored in a Microsoft Exchange Server information store or transmitted by Microsoft Exchange Server can most efficiently be stored in this property and not, for example, in the PR_BODY property. This section describes various ways to work with text contained within or destined for the PR_RTF_COMPRESSED property.
