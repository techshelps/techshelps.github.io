PR_EMS_AB_REPORT_TO_OWNER

The PR_EMS_AB_REPORT_TO_OWNER property specifies whether the owner of a distribution list is informed that an oversized message has been submitted to the distribution list.

Quick Info

Header file: EMSABTAG.H
Applies to: Distribution-List objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8121
Property type: PT_BOOLEAN
Access: Read/write

Values

0
FALSE = Do not receive reports.
1
TRUE = Receive reports.
