Overview

The Schedule+ printing model formats printed information using different page sizes and formats. The application contains a print engine that can find print layouts and page formats in a directory and parse them at run time. Both the layout and the format are described by text files that have an extension of .PRT for the layout and .FMT for the format.

The layout specifies how items such as appointments, tasks, and calendars are laid out on a page. The format defines the size of the physical page, the number of virtual pages on a physical page, and so on.

The user interface lets you use a list box to select a format and a layout. Examples of formats are Daily and Trifold; examples of layouts are Normal and Avery.
