Organization

This reference is organized into the following sections:

  • Overview introduces the basic concepts of Microsoft Schedule+ and examines how to extend its functionality and customize its menus. It also describes the Schedule+ Automation interface, the objects it exposes, and the types of operations that can be performed using those objects.
  • Programming Tasks contains a selection of sample programming tasks. Source code samples are provided.
  • Objects provides a reference for Schedule+ Automation objects and their corresponding properties, data members, and methods in table format.
  • Properties contains an alphabetic listing of Schedule+ Automation properties, their data types, and the objects to which they apply.
  • Data Members contains an alphabetic listing of Schedule+ Automation data members, their data types, and the objects to which they apply.
  • Methods contains an alphabetic listing of Schedule+ Automation methods, their parameters, and the objects to which they apply.
