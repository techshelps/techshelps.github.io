PR_MEMBER_ID

The PR_MEMBER_ID property contains the identifier of a table member that has the described rights on a Microsoft Exchange Server folder or mailbox.

Quick Info

Header file: EDKMDB.H
Applies to: Public folder and mailbox objects (information store objects)
Identifier: 0x6671
Property type: PT_I8
Access: Read-only

Remarks

The PR_MEMBER_ID property returns an identifier unique to the table. A directory user identifier is associated with each member identifier and is given by the PR_MEMBER_ENTRYID property. This property is used by the IExchangeModifyTable interface to retrieve the directory entry identifier of a member with explicit rights on a folder.
