PR_EMS_AB_USER_PASSWORD

The PR_EMS_AB_USER_PASSWORD property is for X.500 compatibility and is not used by Microsoft Exchange Server.

Quick Info

Header file: EMSABTAG.H
Applies to: Organization, Organizational-Unit, and Person objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8153
Property type: PT_MV_BINARY
Access: Read/write

Values

1 (2 hex digits)
Minimum length
128 (256 hex digits)
Maximum length
