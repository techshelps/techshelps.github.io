Preliminary System Configuration

The Microsoft Exchange Server Administrator program must be installed and running on any computer onto which a new Microsoft Exchange Server application is installed. This computer does not need to be a Microsoft Exchange Server computer.

If your application is to run as a service of Windows NT Server, it is subject to the following prerequisite. The Microsoft Exchange Server administrator needs to set up a Windows NT account with the right to log on as a service before running the application's installation program. The documentation for your application's installation procedure should make this prerequisite clear.
