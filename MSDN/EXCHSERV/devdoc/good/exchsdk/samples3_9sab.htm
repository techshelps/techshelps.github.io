REPORT.LIB Functions

The NEWPST.LIB library contains the following functions:

Function Purpose
HrEDKReportSubject Returns a report subject.
HREDKReportMessageClass Returns a report message class
HREDKSetReportMessageClass Sets the report message class.
HrEDKEnvelopeMessageClass Returns an envelope message class.
HrEDKCopyReportProps Copies report properties.
HrReadReceipt Creates a read/non-read notification report.
HeEDKCreateReadReceipt Creates a RN/NRN report.
HrEDKCreateReadNotification
Report		 Creates a read notification report.
HrEDKCreateNonReadNotification
Report		 Creates a non-read notification report.
