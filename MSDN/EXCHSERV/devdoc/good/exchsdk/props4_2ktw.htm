PR_FOLDER_CHILD_COUNT

The PR_FOLDER_CHILD_COUNT property specifies the number of subfolders contained within a public folder object.

Quick Info

Header file: EDKMDB.H
Applies to: Public folder objects (information store objects)
Identifier: 0x6638
Property type: PT_LONG
Access: Read-only

Remarks

The PR_FOLDER_CHILD_COUNT property is always present even if there are no subfolders for the public folder object. For more information, see the PR_SUBFOLDERS property in the MAPI Programmer's Reference.
