PR_EMS_AB_AUTOREPLY_SUBJECT

The PR_EMS_AB_AUTOREPLY_SUBJECT property contains the subject line for an Out of Office message.

Quick Info

Header file: EMSABTAG.H
Applies to: Mail-Recipient objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x803e
Property type: PR_EMS_AB_AUTOREPLY_SUBJECT PT_TSTRING

PR_EMS_AB_AUTOREPLY_SUBJECT_A PT_STRING8

PR_EMS_AB_AUTOREPLY_SUBJECT_W PT_UNICODE
Access: Read/write
Unicode: Yes

Values

0
Minimum length
256
Maximum length
