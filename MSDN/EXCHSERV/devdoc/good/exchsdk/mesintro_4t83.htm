Software Requirements

To design applications for Microsoft Exchange Server, you need the following software installed on your system:

  • Microsoft Windows NT version 3.51 or above, with Service Pack 4.
  • Microsoft Exchange Server must be running on the domain.
  • Microsoft Platform SDK.
  • Microsoft Visual C++ version 4.0 is required. Earlier versions are not supported.
  • Sample applications may require the Administrator program to be running on the development computer. You may also need to install Microsoft Exchange Server and Microsoft Exchange Client.
