Directory Addressing Objects

For Microsoft Exchange Server to support a foreign address type, several objects must be created in the Microsoft Exchange Server directory:

  • An Addr-Type object.
  • A proxy generation DLL.
  • An address entry template for the Microsoft Exchange Server Administrator.
  • Per-message and Per-recipient options for messages sent to addresses of the foreign address type, with online Help for those options.
  • A data file describing the dialog box to be displayed by the Address-Template object for creating and editing addresses in the remote system’s address format.
  • A data file describing a program for validating the data entered into the address template dialog and converting it into an address in the foreign system’s address format.
