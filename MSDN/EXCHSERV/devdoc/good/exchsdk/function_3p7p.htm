GetGWDefaultStore

The GetGWDefaultStore function obtains a pointer to the default gateway store.

Quick Info

Header file: GWMAIN.H
Library: GWMAIN.LIB 

LPMDB GetGWDefaultStore(
  VOID  
);

Parameters

None.

Return Values

A pointer to a MAPI IMsgStore interface containing the default gateway store.

Remarks

For information on the IMsgStore interface, see the MAPI Programmer's Reference.
