Specifying the Template Object

You specify the template object — typically an existing recipient — with the pszCreateTemplate parameter in a structure passed to the BatchImport and DAPIStart functions. The template object can be expressed as the DN or RDN of an existing Microsoft Exchange Server directory object. The following is an example of a DN of an object:

bp->pszDefaultUser = "/o=Organization/ou=Site/cn=Recipients/cn=DefaultUser";
