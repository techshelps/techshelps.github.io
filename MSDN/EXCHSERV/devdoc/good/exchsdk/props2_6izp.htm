PR_EMS_AB_HELP_FILE_NAME

The PR_EMS_AB_HELP_FILE_NAME property contains the name of the Help file for the Display-Template object.

Quick Info

Header file: EMSABTAG.H
Applies to: Display-Template objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x803b
Property type: PR_EMS_AB_HELP_FILE_NAME PT_TSTRING

PR_EMS_AB_HELP_FILE_NAME_A PT_STRING8

PR_EMS_AB_HELP_FILE_NAME_W PT_UNICODE
Access: Read/write
Unicode: Yes

Values

1
Minimum length
13
Maximum length

Remarks

The PR_EMS_AB_HELP_FILE_NAME property is valid only when reading a property from the Microsoft Exchange Server address book. It is not valid after a recipient object has been copied to another type of address book. (e.g., a personal address book).
