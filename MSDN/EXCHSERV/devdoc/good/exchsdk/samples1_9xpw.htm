Exporting Directory Attributes with DSEXPORT

    To export directory attributes with DSEXPORT
  • To export all the recipients from the North America site of the Litware organization, type the following DSEXPORT command line:

    DSEXPORT /FILE=fran1.csv /DSA=BERKELEY /BASEPOINT=/o=Litware/ou="North America" /ALL_RECIPIENTS

  • To export only the distribution lists from the basepoint "o=BLACK11 /ou=Site1", type the following command on one line:

    DSEXPORT /FILE=export.csv /DSA=BERKELEY, /BASEPOINT=/o=BLACK11/ou=Site1 /DIST_LIST

  • To export mailboxes and remote recipients from the basepoint "/o=REDD22 /ou=Site10", type the following command on one line:

    DSEXPORT /FILE=export5.csf /DSA=BERKELEY, /BASEPOINT=/o=REDD22/ou=Site10 /MAILBOX /REMOTE_ADDRESS

  • To export objects of the mailbox and distribution lists class from the basepoint "/o=WHITE1 /ou=Site26" to the EXMAILBOX.CSV export file, type the following command on one line:

    DSEXPORT /file=exmailbox.csv /dsa=BERKELEY /BASEPOINT=/o=WHITE1/ou=Site26/CLASSES=MAILBOX%DL

  • To export hidden mailboxes residing on the server named BERKELEY1, type the following command on one line:

    DSEXPORT /FILE=export.csv /DSA=BERKELEY /SERVER=BERKELEY1 /BASEPOINT=o=m/ou=b /MAILBOX /HIDDEN

Note When exporting a distribution list or a custom recipient, avoid using the /SERVER flag because such objects do not have the Home-Server attribute and cannot be seen if /SERVER is passed.

When you have run DSEXPORT, you will have created an export file that will be stored at the file name indicated. The format of the export file is shown in the following example, which lists exported attributes and information for each recipient.

Directory Name,Primary Windows NT Account,Home-Server,E-mail 
address,E-mail Addresses,Members,Obj-Container,Hide from AB

MS:MICROSOFT/ALLISON2/3%SMTP:3@Allison2.Microsoft.com%X400:
c=US;a= ;p=Microsoft;o=Allison2;s=3;,,Recipients,
Remote,11,,11,11,11,,,SMTP:1@EX.COM,
MS:MICROSOFT/JEFFREY2/11%X400:c=US;a= ;p=Microsoft;o=Jeffrey2;s=11;%SMTP:1@EX.COM
,,Recipients,

In this export operation, fifteen attributes are exported per recipient. In some cases, the attributes contained no values, and are marked by consecutive separators (commas).

All correctly exported files must be in this exact format. For more information about formats, see Formatting DIRSYNC Import and Export Files.

For more information about export and import file formats, see File Formats.
