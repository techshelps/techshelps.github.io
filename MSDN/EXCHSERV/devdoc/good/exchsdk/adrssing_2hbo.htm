The Addr-Type Object

Every gateway has an associated address type that corresponds to the address format of the foreign system the gateway serves. This address type is represented by an Addr-Type directory object. The gateway and the Addr-Type object are associated by the Microsoft Exchange Server’s GWART. Your gateway’s setup program installs a the Addr-Type object and associated information when it installs the gateway.

The following components are associated with an Addr-Type object:

  • A proxy address generation DLL that enables the system to generate proxy addresses of this address type.
  • Address entry templates that assist the administrator or user in manually entering a recipient address of this address type. There are two types of templates: normal and MS-DOS.
  • Per-message and per-recipient options that enable users to set options in messages sent to addresses of this address type.
  • Online Help that provides instructions for using the property pages associated with per-message and per-recipient options.
