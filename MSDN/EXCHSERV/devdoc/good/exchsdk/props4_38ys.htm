PR_MTS_ID

The PR_MTS_ID property is the message transfer envelope identifier.

Quick Info

Header file: EDKMDB.H
Applies to: Message transfer envelope objects (information store objects)
Identifier: 0x0047
Property type: PT_BINARY
Access: Read/write

Remarks

The PR_MTS_ID property can be used by gateway developers to correlate messages with reports or to follow threads. This property is generated by the transport (such as the gateway) rather than by a client. If not supplied explicitly by a gateway, Microsoft Exchange Server generates a value for PR_MTS_ID automatically.

The PR_MTS_ID property has the same values and is synonymous with the MAPI PR_MESSAGE_SUBMISSION_ID property.

For more information on PR_MESSAGE_SUBMISSION_ID, see the MAPI Programmer's Reference.
