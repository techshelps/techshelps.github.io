Server Applications for Microsoft Exchange Server

Services of Microsoft Exchange Server interact with components such as the information store and the directory and offer custom services to clients. For example, a server application can be designed to search through folders in the information store for all messages containing a certain text string or property value. Server applications may function independently of clients, or they may include a client component. Server applications communicate with Microsoft Exchange Server through MAPI.
