PR_EMS_AB_ADMIN_DESCRIPTION

The PR_EMS_AB_ADMIN_DESCRIPTION property contains a short description of a directory object for the administrator.

Quick Info

Header file: EMSABTAG.H
Applies to: All directory objects. These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x804a
Property type: PR_EMS_AB_ADMIN_DESCRIPTION PT_TSTRING

PR_EMS_AB_ADMIN_DESCRIPTION_A PT_STRING8

PR_EMS_AB_ADMIN_DESCRIPTION_W PT_UNICODE
Access: Read/write
Unicode: Yes

Values

1
Minimum length
1024
Maximum length
