PR_PROFILE_MAX_RESTRICT

The PR_PROFILE_MAX_RESTRICT property restricts the maximum number of names the address book will return.

Quick Info

Header file: EDKMDB.H
Applies to: Profile objects
Identifier: 0x660d
Property type: PT_LONG
Access: Read/write

Remarks

The PR_PROFILE_MAX_RESTRICT property is used in the global sections of Microsoft Exchange Server profiles.
