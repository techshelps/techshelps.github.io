IExchangeModifyTable::GetTable

The GetTable method retrieves a pointer to an interface to a MAPI table object.

HRESULT GetTable(
  ULONG ulFlags,               
  LPMAPITABLE FAR * lppTable   
);

Parameters

ulFlags
Input parameter. Reserved for future use. Set this parameter to 0 (zero).
lppTable
Output parameter. Points to a MAPI IMAPITable interface containing the table object.

Return Values

See Return Values.
