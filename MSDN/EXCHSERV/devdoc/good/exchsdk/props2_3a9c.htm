PR_EMS_AB_HIDE_DL_MEMBERSHIP

The PR_EMS_AB_HIDE_DL_MEMBERSHIP property determines whether the individual members of a distribution list are to be listed (and able to be viewed by other users) or unlisted.

Quick Info

Header file: EMSABTAG.H
Applies to: Distribution-List objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x80b8
Property type: PT_BOOLEAN
Access: Read/write

Values

0
FALSE = Membership is not displayed.
1
TRUE = Membership is displayed.
