AnniversaryItemId

The AnniversaryItemId property returns an identifier for the anniversary of a contact in the RecurringEvents table.

Applies To

Contact object

Data Type

String

Access

Read/write

Remarks

The AnniversaryItemId property is used to link a Contact object to a RecurringEvent object. In this case, the recurring event is the anniversary of the contact. This is the ItemId property in a RecurringEvents table.
