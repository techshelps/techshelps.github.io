PR_PST_ENCRYPTION

The PR_PST_ENCRYPTION property contains the encryption level at which to create a file in the personal folder store.

Quick Info

Header file: MSPST.H
Applies to: Personal folder store objects
Identifier: 0x6702
Property type: PT_LONG
Access: Read/write

Values

PSTF_NO_ENCRYPTION
Do not encrypt.
PSTF_COMPRESSABLE_ENCRYPTION
Encrypt for potential compression.
PSTF_BEST_ENCRYPTION
Perform the most-rigorous encryption.
