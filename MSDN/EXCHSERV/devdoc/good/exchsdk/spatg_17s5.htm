Intended Audience

To use the Schedule+ Automation interface, your background should include:

  • Familiarity with basic features and capabilities of Schedule+.
  • Proficiency with Visual Basic programming in the Windows® environment. Reviewing the Microsoft Win32 SDK and Microsoft Visual Basic Programming System for Windows documentation is recommended.
  • Proficiency with the Active Messaging Library. Reviewing the Microsoft Active Messaging Library Programmer’s Reference is recommended.
  • Familiarity with Message Application Programming Interface (MAPI). Reviewing the MAPI Programmer’s Reference is recommended.
  • Expertise with network environments.
