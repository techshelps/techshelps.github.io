Installing a Typical Development Platform

The typical development environment for the Microsoft Exchange Server SDK includes the following products. To avoid conflicts, install them in the order listed.

  1. Windows NT Server Version 3.51 or later as a base platform.
  2. Microsoft Visual C++ Version 4.0 or later (earlier versions are incompatible with the Microsoft Exchange Server SDK).
  3. Microsoft Exchange Client
  4. Microsoft Exchange Server For developing and testing most types of applications, you will need the entire Microsoft Exchange Server product on your development computer or on a networked computer. Note that some applications require only the Microsoft Exchange Server Administrator program.
  5. Microsoft Exchange Server SDK See Installing the Microsoft Exchange Server SDK.
