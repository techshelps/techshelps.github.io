DefaultTaskDurationTypeUnit

The DefaultTaskDurationTypeUnit property returns the unit of time for the default duration of a task.

Applies to

Schedule object

Data Type

Long

Access

Read/write

Remarks

The DefaultTaskDurationTypeUnit property can have one of the following values.

Value Description
0 Minutes
1 Hours
2 Days
3 Weeks
4 Months
