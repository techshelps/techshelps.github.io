Quick Start Procedures: Server Applications

This section covers the following procedures:

  • Quick Start: Creating a Server Application

    Presents the basics of creating a gateway, a mailbox agent, or an add-in application.

  • Quick Start: Creating a Profile through MAPI

    Examines how to tailor profiles created to work with Microsoft Exchange Server.

  • Quick Start: Designing a Gateway

    Examines the most important issues to consider when designing a gateway.

  • Quick Start: Creating a Directory Synchronization Agent

    Discusses how to customize the DIRSYNC sample application.

  • Quick Start: Replicating Data from an External Database (such as SQL)

    Describes both the replication of information store data (such as transactions or Help Desk reports) and data intended for the Microsoft Exchange Server directory.
