Formatting for Addresses Delimited by Semicolons

Multivalued fields that are delimited by semicolons — such as X.400 addresses, distribution list members, and e-mail addresses — must follow these formatting conventions:

  • The entire distribution list membership field must be placed within a pair of double quotation marks.
  • The e-mail address mini-field that contains semi-colons must be placed within a pair of double quotation marks.
  • All quotation marks that are part of the field entry must themselves be placed within a pair of double quotation marks.

For example:

dl,Friends,\\
"SMTP:mariab@litware.com;""X400:C=US;A= ;P=NWTraders;G=Joe;S=Parkes"""\\
,,,,,,,,,,,,,,,,,,,,,

If your existing system requires that you avoid these formatting rules with semicolons, it's recommended that you specify X.400 addresses by using a slash ( / ), as shown in the following example:

dl,Friends,\\
SMTP:mariab@litware.com;X400:/C=US/A= /P=NWTraders/G=Joe/S=Parkes\\
,,,,,,,,,,,,,,,,,,,,,
