Adding an Application Object

In most cases, you will want to make a new application or service you write addressable. To do this, add it to the Microsoft Exchange Server directory using the BatchImport function or the DAPIWrite function.

    To add an application object to the directory
  1. Follow the steps in Adding an Object to the Directory.
  2. For a complete installation of an application, not just its addition to the directory, see Creating a Complete Setup Program.
