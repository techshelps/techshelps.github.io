Displaying Rules using RULEEDIT

When you run RULEEDIT with the /LIST switch, RULEEDIT displays the following values for each rule.

Values Displayed for Rules

Column Description
Provider Name Specifies the name of the provider used to create the rule.
Row number Specifies the cursor position: continuous 0 - n. This represents the order in which rules are stored.
Rules state Specifies whether the rules are Enabled (ST_ENABLED) or Disabled (ST_DISABLED).
Rule condition Specifies an expression that represents the rule restriction
Rule action Specifies the action(s) taken by the rule.
Rule level, Rule name Shows internal rule attributes

Note If no provider name is specified from the command line, RULEEDIT will list rules for all providers.
