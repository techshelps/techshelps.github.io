SentRanges

The SentRanges property returns a range of change numbers sent to the master schedule.

Applies To

Schedule object

Data Type

Double (multivalued)

Access

Read-only

Remarks

The SentRanges property is used by Schedule+ to synchronize schedules. Only the owner of the schedule file can read this property.
