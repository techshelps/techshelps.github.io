Sending Edited Text with POSTSMPL

    To send edited text with POSTSMPL
  1. Type POSTSMPL at a command prompt. The Choose Profile dialog box appears.

  2. Pull down the Profile Name combo box, if necessary, and select a profile for your messaging session. Click OK. A text edit box appears.

  3. Create a new file by typing text in the text edit box or open an existing file by choosing Open from the File menu.
  4. Save the contents by choosing Save from the File menu. The file must be saved before it can be posted or sent. If you change the file after it is saved, the changes will not be sent or posted unless the file is saved again.
  5. To include summary information with your file, select Summary Information from the File menu and fill in the properties.

  6. Choose Send from the File menu. A New Message box appears with your edited file as an attachment. Add any other text you wish to the message and select Send from the File menu or click the envelope-icon button.
