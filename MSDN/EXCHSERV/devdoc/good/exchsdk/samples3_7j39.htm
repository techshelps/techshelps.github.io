PROFINST: Installing a MAPI Profile

The PROFINST sample application demonstrates how to create a MAPI profile for a server application. The server application can then use the profile to log on to MAPI. For more information about installing a server application, see Using a Batch File to Install SMBAGENT.

    To build PROFINST
  1. Change to the directory containing the PROFINST source files:

    \BKOFFICE\SAMPLES\EXCHANGE\EDKINSTU

  2. Type NMAKE.
