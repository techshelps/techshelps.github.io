Microsoft Exchange Server to Gateway Message Transfer

When a message is transferred from Microsoft Exchange Server to the gateway, the following steps are performed:

  1. A new MAPI message, the MTE, is created.
  2. Microsoft Exchange Server copies the value of the PR_MESSAGE_SUBMISSION_ID property from the outgoing message to the MTE’s PR_MTS_ID property.
  3. Microsoft Exchange Server stores the outgoing message’s PR_SEARCH_KEY identifier in the MTE’s PR_CONTENT_SEARCH_KEY property.
