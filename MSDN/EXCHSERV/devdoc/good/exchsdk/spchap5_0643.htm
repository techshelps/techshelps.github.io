SupportedOperations

The SupportedOperations data member returns the operations available for the schedule.

Applies To

Schedule object

Data Type

Long

Access

Read-only

Remarks

The SupportedOperations data member determines what type of operations can be performed on the schedule.

The SupportedOperations data member returns the following values.

Value Description
1 Application can only determine the schedule owner.
2 Application can only access free/busy information for the schedule.
3 Application can only access appointments and tasks.
4 Application can access any part of the schedule.
