Running the CRARUN Launch Application

Once you have installed the CRARUN Launch application, you can use it to set up a rule that starts an application of your choice when it fires.

    To run the Launch application
  1. Start the Microsoft Exchange Client.
  2. From the Tools menu, select Inbox Assistant. A screen such as the following is displayed:

  3. Click Add Rule. The Edit Rule dialog box appears.

    

  1. Click in the Custom box. Pull down the combo box. The Launch custom rule should appear. Select Launch.
  2. The Program to Launch dialog box appears. Select a program to launch if the rule fires, for example, C:\WINNT\NOTEPAD.EXE.

  3. Click OK in the Program to Launch dialog box.
  4. Select a condition under which you want the rule to fire, for example, From a certain user. Then click OK in the Edit Rule dialog box.
  5. You should see the custom rule in the Conditions box of the Inbox Assistant dialog box. Click OK.

  6. Optionally, test the condition that fires the rule. For example, send a message from the selected user to the profile that contains the custom rule. The rule should fire; in this case, NOTEPAD.EXE should automatically start.
