PADMIN_InfoHasChanged

The PADMIN_InfoHasChanged Administrator extension function notifies the Microsoft Exchange Server Administrator program that some information has been altered on a property sheet.

Quick Info

Header file: ADMINEXT.H
Library: ADMINEXT.LIB 

VOID PADMIN_InfoHasChanged(
  HWNDPARENT hwndParent  
);

Parameters

hwndParent
Input parameter. Handle to the current property sheet window.

Return Values

None.

Remarks

Use PADMIN_InfoHasChanged in your Administrator extension DLL to have the Microsoft Exchange Server Administrator program enable the Apply button after the user has changed some data in the property sheet.

For more information on this function, see Using Administrator Extension Functions (Called by Extension DLL).
