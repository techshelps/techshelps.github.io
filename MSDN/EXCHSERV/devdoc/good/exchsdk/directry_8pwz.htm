Initializing Import Structures

The BIMPORT_PARMS structure used as a parameter for the BatchImport function must be initialized to zero before use. You can do this by calling the Win32 ZeroMemory function. By zeroing this structure, you ensure that default processing is invoked for fields that are not filled in. The following fields must then be filled in:

  • The BatchImport function needs to have an import file specified with the pszImportFile member.
  • Pointers to callback functions need to be nonzero if a callback is to be used.
