OwnerName

The OwnerName property returns the MAPI display name for the meeting owner of an item.

Applies To

Appointment and RecurringAppointment objects

Data Type

String

Access

Read/write

Remarks

The value of the OwnerName property is supplied by MAPI. The meeting owner is usually the user who creates the meeting, except when a delegate creates the meeting instead.
