PR_CONTENTS_SYNCHRONIZER

The PR_CONTENTS_SYNCHRONIZER property is used with the MAPI OpenProperty function to retrieve the IExchangeExportChanges interface. This interface is for exporting only a folder's contents changes, not its hierarchy changes.

Quick Info

Header file: EDKMDB.H
Applies to: Public folder objects (information store objects)
Identifier: 0x662d
Property type: PT_OBJECT
Access: Read-only

Remarks

The PR_CONTENTS_SYNCHRONIZER property is part of the incremental change synchronization (ICS) interface, which is not supported by all stores. See the MAPI Programmer’s Reference.
