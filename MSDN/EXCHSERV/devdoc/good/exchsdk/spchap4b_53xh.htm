IsRecurringInstance

The IsRecurringInstance property determines whether an item is an instance of a recurrence pattern.

Applies To

RecurringAppointment, RecurringEvent, and RecurringTask objects

Data Type

Boolean

Access

Read-only

Remarks

The IsRecurringInstance property is available only in merged tables, where items can be single items or instances of recurrence patterns.
