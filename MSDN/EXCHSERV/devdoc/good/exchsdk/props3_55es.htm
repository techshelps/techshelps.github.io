PR_EMS_AB_MDB_MSG_TIME_OUT_PERIOD

The PR_EMS_AB_MDB_MSG_TIME_OUT_PERIOD property contains the number of minutes a message can remain unread by a gateway before the information store will generate a nondelivery report.

Quick Info

Header file: EMSABTAG.H
Applies to: Mail-Gateway objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x80d0
Property type: PT_LONG
Access: Read/write

Remarks

The Microsoft Exchange Server information store transfers a message to a gateway by placing the message in the gateway's MTS-OUT folder. If a message remains unread in the MTS-OUT folder for more than the number of minutes set by the PR_EMS_AB_MDB_MSG_TIME_OUT_PERIOD property, the information store removes the message from the folder and generates a nondelivery report.
