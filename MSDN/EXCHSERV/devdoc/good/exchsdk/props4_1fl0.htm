PR_FOREIGN_REPORT_ID

The PR_FOREIGN_REPORT_ID property contains an identifier from the foreign system that is assigned to a gateway report received from a foreign system.

Quick Info

Header file: EDKMDB.H
Applies to: Report objects (information store objects)
Identifier: 0x6668
Property type: PT_BINARY
Access: Read/write

Remarks

The PR_FOREIGN_REPORT_ID property applies only to reports. The gateway assigns this property from a foreign report's report identifier.
