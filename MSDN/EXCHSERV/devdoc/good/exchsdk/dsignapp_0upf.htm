Microsoft Exchange Server Development Options

Microsoft Exchange Server provides a rich and complex development environment. The system is a MAPI service provider built on the Windows NT Server platform that employs the Active Messaging Library. You can use the following development tools:

  • Microsoft® Visual C++®
  • Microsoft® Visual Basic® Use Visual Basic with Active Messaging to access the Microsoft Exchange Server address book and information store, and to send and receive mail.
  • MAPI interfaces
  • Microsoft Exchange Server interfaces

Choosing the most appropriate tools depends on the application you plan to develop and the environment in which your application will run.
