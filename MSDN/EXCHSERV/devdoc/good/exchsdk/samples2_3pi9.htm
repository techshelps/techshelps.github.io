EXTDATVW: Viewing and Editing Extension Data

The EXTDATVW sample application creates an Administrator program extension that allows you to view and edit the values of an Extension-Data attribute of a Microsoft Exchange Server directory object. EXTDATVW is implemented as an Administrator extension DLL (EXTDATVW.DLL).

To use EXTDATVW, the Administrator extension DLL must be installed and EXTDATVW must be attached to the directory object. This adds the Ext Data Viewer property page to the object's property sheet. For more information about installing extension DLLs, see Installing the Administrator Extension DLL. For general information about Administrator extensions, see Administrator Program Extensions.

    To build EXTDATVW
  1. Change to the directory containing the EXTDATVW source files:

    \BKOFFICE\SAMPLES\EXCHANGE\EXTDATVW

  2. Type NMAKE.

To install the Administrator extension DLL, use the EXTNINST sample application.
