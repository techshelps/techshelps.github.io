PR_MAILBOX_OWNER_NAME

The PR_MAILBOX_OWNER_NAME property contains the name of the owner of a mailbox.

Quick Info

Header file: EDKMDB.H
Applies to: Mailbox objects (information store objects)
Identifier: 0x661c
Property type: PT_TSTRING
Access: Read-only

Remarks

The PR_MAILBOX_OWNER_NAME property is not defined for gateway mailboxes.
