JSClose

The JSClose function closes a jump window.

Quick Info

Header file: STATS.H
Library: STATS.LIB 

VOID JSClose(
  HJSWINDOW hJSWindow  
);

Parameters

hJSWindow
Input parameter. Handle to the jump window.

Return Values

None.

Remarks

JSClose closes the jump window.
