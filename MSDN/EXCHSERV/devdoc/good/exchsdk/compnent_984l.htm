About Mailboxes and the Private Information Store

The private information store consists of mailboxes that can belong to users or to applications. Mailboxes reside on the computer running Microsoft Exchange Server. Each newly created mailbox has four default folders visible to the Microsoft Exchange Client:

  • Deleted Items
  • Inbox
  • Outbox
  • Sent Items

The mailbox contains several other folders that are not visible and are used for storing information such as user-defined views. Users or applications can create new, visible folders in addition to the four default folders. The following diagram shows the internal architecture of a mailbox.

Mailbox architecture

Folders that are always present but are not visible to the user include the following:

  • Deferred Action Includes messages about actions that should be performed when resources are available to update the visible folders.
  • Schedule Contains Schedule+ information for the mailbox owner.
  • Shortcuts Contains information about Favorites folders.
  • Common Views Contains user-defined views.

The Finder and the Print Spooler are additional folders that are contained in the mailbox but are not visible to the user.
