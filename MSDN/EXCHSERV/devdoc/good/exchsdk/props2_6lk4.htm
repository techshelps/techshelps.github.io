PR_EMS_AB_EXTENSION_NAME_INHERITED

The PR_EMS_AB_EXTENSION_NAME_INHERITED property contains the name of an Admin-Extension object.

Quick Info

Header file: EMSABTAG.H
Applies to: All directory objects. These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x80aa
Property type: PR_EMS_AB_EXTENSION_NAME_INHERITED PT_MV_TSTRING

PR_EMS_AB_EXTENSION_NAME_INHERITED_A PT_MV_STRING8

PR_EMS_AB_EXTENSION_NAME_INHERITED_W PT_MV_UNICODE
Access: Read/write
Unicode: Yes

Values

1
Minimum length
255
Maximum length

Remarks

The PR_EMS_AB_EXTENSION_NAME_INHERITED property is the same as PR_EMS_AB_EXTENSION_NAME, except that the Admin-Extension object is inherited by all the object’s children. For example, this property is used to add a property page to all mail recipients under a specified container.
