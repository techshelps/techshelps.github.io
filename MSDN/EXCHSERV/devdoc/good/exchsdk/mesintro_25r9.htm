Using the Microsoft Exchange Server Programmer’s Reference

The Microsoft Exchange Server Programmer's Reference explains the concepts of writing applications for Microsoft® Exchange Server and describes the development tools included in the Microsoft Exchange Server Software Development Kit (SDK). These tools include code libraries, header files, sample applications, and sample code. Using this SDK, you can:

  • Develop stand-alone client applications and server applications for Microsoft Exchange Server.
  • Develop Administrator program extensions for Microsoft Exchange Server.
  • Use the sample applications to extend Microsoft Exchange Server client-server messaging and groupware.
  • Use the sample applications to build client applications and server applications customized for your environment.

You can also develop client extensions for Microsoft Exchange Server. For more information, see Client Extension Development Options and the Microsoft Exchange Client Extensions Programmer's Reference in the Messaging Application Programming Interface (MAPI) Programmer's Reference.
