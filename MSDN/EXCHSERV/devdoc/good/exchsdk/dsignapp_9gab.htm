Configuring Special Features

You may want to configure your source extractor to perform certain activities. For example, it is advisable to configure both an error log and online Help, which can help troubleshoot problems. Also consider configuring the following features:

  • Mail forwarding You can configure the source extractor to forward mail, depending on your existing system’s features.
  • Special migration folder If you configure this folder, only messages in the folder and in the Inbox are migrated.
  • Status file To prevent a mailbox’s contents from being imported twice, configure your source extractor to keep a log of migrated users.
  • Last message configuration A message informing users that their new mailboxes are available and not to use the old mailboxes anymore.
