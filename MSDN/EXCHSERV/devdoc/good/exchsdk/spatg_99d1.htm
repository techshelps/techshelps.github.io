Using the Microsoft Schedule+ Programmer’s Reference

The Microsoft Schedule+ Programmer’s Reference is designed for use with Microsoft® Schedule+ Automation (formerly OLE Automation). Automation is a feature of the Component Object Model (COM), an industry standard used by applications to expose objects to development tools and other applications. The Schedule+ Automation interface exposes objects that enable applications, such as those developed with Microsoft® Visual Basic® and Microsoft® Visual C++®, to manipulate Schedule+ information. This information includes appointments, contacts, tasks, projects, events, alarms, and other Automation controllers.

Using these tools, you can customize menus and write stand-alone applications that extend Schedule+ capabilities.

This reference should be used in conjunction with the following documents:

  • Microsoft Visual Basic Programming System for Windows Programmer’s Guide
  • Microsoft Active Messaging Library Programmer’s Reference
  • MAPI Programmer’s Reference in the Microsoft® Win32® Software Development Kit (SDK)
