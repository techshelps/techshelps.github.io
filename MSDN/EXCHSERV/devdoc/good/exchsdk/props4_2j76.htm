PR_RECIPIENT_NUMBER

The PR_RECIPIENT_NUMBER property contains an index into a table of recipients.

Quick Info

Header file: EDKMDB.H
Applies to: Message transfer envelope objects (information store objects)
Identifier: 0x6662
Property type: PT_LONG
Access: Read/write

Remarks

The PR_RECIPIENT_NUMBER property is on the recipient table of the message transfer envelope.

The X.400 system uses this number to deliver DRs and NDRs. Gateway developers can track messages by using this property to assign the same number to a recipient as occurs on the foreign system.
