Generating a Packed Property File

    To generate a file containing a binary packed property structure
  1. Write an input file that contains the information to be packed. The format of this text file is described in The CFGCODE Input File.
  2. Run CFGCODE, specifying the name of the input file, a name for the output file, and a configuration name using the /NAME switch. Output files often use the .DAT extension. You also specify a section of the input file with the /SECTION= switch. Extension-data owner tags are described in the section Using the Extension-Data Attribute.
