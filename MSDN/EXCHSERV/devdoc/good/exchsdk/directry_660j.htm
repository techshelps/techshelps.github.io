LDAP Protocol Operations

This section describes the protocol operations of LDAP and states whether each operation is implemented by the directory service of Microsoft Exchange Server. For more information on LDAP, see the documents Lightweight Directory Access Protocol (RFC1777) at http://directory service.internic.net/rfc/rfc1777.txt and The LDAP Application Program Interface (RFC1823) at http://directory service.internic.net/rfc/rfc1777.txt.
