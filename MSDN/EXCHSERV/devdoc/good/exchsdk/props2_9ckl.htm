PR_EMS_AB_DISPLAY_NAME_PRINTABLE

The PR_EMS_AB_DISPLAY_NAME_PRINTABLE property contains the printable string version of the display name.

Quick Info

Header file: EMSABTAG.H
Applies to: Mail-Recipient objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x39ff
Property type: PR_EMS_AB_DISPLAY_NAME_PRINTABLE PT_TSTRING

PR_EMS_AB_DISPLAY_NAME_PRINTABLE_A PT_STRING8

PR_EMS_AB_DISPLAY_NAME_PRINTABLE_W PT_UNICODE
Access: Read/write
Unicode: Yes

Values

1
Minimum length
256
Maximum length

Remarks

The PR_EMS_AB_DISPLAY_NAME_PRINTABLE property is valid only when reading a property from the Microsoft Exchange Server address book. It is not valid when reading a property from an object copied into another address book provider.

The printable string version of the display name is required because the normal display name may contain extended characters that cannot be displayed by some mail systems. The printable string display name supports only the first 128 ASCII characters.
