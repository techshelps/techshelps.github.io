GetGWMtsInFolder

The GetGWMtsInFolder function retrieves the inbound message folder, MTS-IN.

Quick Info

Header file: GWMAIN.H
Library: GWMAIN.LIB 

LPMAPIFOLDER GetGWMtsInFolder(
  VOID  
);

Parameters

None.

Return Values

A pointer to a MAPI IMAPIFolder interface containing the inbound message folder, MTS-IN.

Remarks

For information on the IMAPIFolder interface, see the MAPI Programmer's Reference.
