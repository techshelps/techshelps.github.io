PR_PARENT_SOURCE_KEY

The PR_PARENT_SOURCE_KEY property contains the PR_SOURCE_KEY property of the parent folder of a folder.

Quick Info

Header file: EDKMDB.H
Applies to: All folder objects (information store objects)
Identifier: 0x65e1
Property type: PT_BINARY
Access: Read-only

Remarks

The PR_PARENT_SOURCE_KEY property is used primarily by the incremental change synchronization (ICS) interface and contains the PR_SOURCE_KEY that uniquely identifies the parent folder of a folder.

For more information, see About Incremental Change Synchronization.
