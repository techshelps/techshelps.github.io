Logoff

The Logoff method logs off Schedule+.

Applies To

Application object

Syntax

objApplication.Logoff NotifyEveryone

Parameters

NotifyEveryone
Optional. Specifies whether to notify all MAPI applications when logging off a shared session.

Remarks

If Schedule+ is group-enabled, the Logoff method also logs off MAPI. You should call this method if you call the Logon method.
