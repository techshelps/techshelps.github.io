About Mailbox Use

Mailboxes can be handled by server applications in three different ways:

  • Maintains a separate mailbox Most server applications are either mailbox agents or gateways that maintain their own mailboxes in information stores on Microsoft Exchange Server.
  • Accesses other mailboxes Server applications can use mailboxes other than their own with delegate access and privileged access. For information on privileged access, see Information Store Operations.
  • Uses no mailbox An application can be configured as an add-in service that has no mailbox.
