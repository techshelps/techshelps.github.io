AlarmsEnabled

The AlarmsEnabled property returns a Boolean value that indicates whether the alarms processing feature has been enabled.

Applies To

Schedule object

Data Type

Boolean

Access

Read/write

Remarks

If the AlarmsEnabled property is set to True, meeting reminders are displayed. If this property is set to False, the user will never see alarm notification. This does not affect daily task reminders.
