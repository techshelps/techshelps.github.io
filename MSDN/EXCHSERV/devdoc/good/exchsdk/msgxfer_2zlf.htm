Using MAPI Features

A Microsoft Exchange Server gateway is a MAPI client with access to MAPI folders, address books, message transports, and message stores. A gateway must therefore use MAPI interfaces and method to interact with Microsoft Exchange Server. Microsoft Exchange Server uses the MAPI interface to provide:

  • An information store where messaging information is kept.
  • A message transfer agent that controls how messages are sent from originators to recipients.
  • The folders that contain messages transferred to and from the gateway.
