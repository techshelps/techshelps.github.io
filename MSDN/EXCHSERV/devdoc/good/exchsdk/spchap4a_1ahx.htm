AccessEffective

The AccessEffective property identifies the access control list (ACL) level that has been granted to the user viewing an item.

Applies To

All Item objects

Data Type

Long

Access

Read-only

Remarks

The AccessEffective property has the following values.

Value Description
0 No access
1 Minimal Read
2 Read
3 Create
4 Write
5 Owner
