PR_EMS_AB_HOME_PUBLIC_SERVER

The PR_EMS_AB_HOME_PUBLIC_SERVER property contains the name of the home public server for mailboxes on an information store.

Quick Info

Header file: EMSABTAG.H
Applies to: MHS-Message-Store objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x803f
