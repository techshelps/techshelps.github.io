AutoAlarmsTask

The AutoAlarmsTask property returns a Boolean value that indicates whether an alarm should be set for new tasks.

Applies To

Schedule object

Data Type

Boolean

Access

Read/write

Remarks

The AutoAlarmsTask property is usually set by users through the Schedule+ user interface. If this property is set to True, all new Task objects will automatically have their AlarmAmount property set to the value of the AlarmAmountDefaultTask property.
