Installing the Executable File

Your gateway’s executable file is copied into a directory specified by the administrator installing the gateway. You must install a gateway onto the computer on which you run the setup program, although this computer need not be the Microsoft Exchange Server computer. For example, if you want to run a gateway on Computer A, run the setup program and install the gateway’s files on Computer A — even though Computer B may be the actual Microsoft Exchange Server computer to which you will connect the gateway.
