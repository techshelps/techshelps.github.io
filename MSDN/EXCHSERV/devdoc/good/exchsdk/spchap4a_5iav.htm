AlarmsToRing

The AlarmsToRing property returns a table of Alarm objects.

Applies To

Schedule object

Data Type

Object (Table)

Access

Read/write

Remarks

Only the owner of the schedule file can read and write to the AlarmsToRing property. This table can be used to determine when the next alarm will ring. To do this, examine the Notify property of all the Alarm objects contained in the table.
