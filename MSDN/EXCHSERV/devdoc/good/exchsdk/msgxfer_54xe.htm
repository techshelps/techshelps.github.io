Client to Microsoft Exchange Server Message Transfer

The following steps are performed when a message is transferred from a client to Microsoft Exchange Server:

  1. When a client creates a message, MAPI assigns the PR_SEARCH_KEY message identifier.
  2. When the message is submitted to Microsoft Exchange Server, the PR_SEARCH_KEY property’s value is also maintained in the message’s PR_CONTENT_ID and PR_CONTENT_CORRELATOR properties.
  3. Microsoft Exchange Server creates a new message identifier and stores it in the message’s PR_MESSAGE_SUBMISSION_ID property.
