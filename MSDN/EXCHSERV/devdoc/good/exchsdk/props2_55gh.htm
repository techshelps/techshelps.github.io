PR_EMS_AB_EXTENSION_DATA

The PR_EMS_AB_EXTENSION_DATA property contains a list of data sections specific to an extension.

Quick Info

Header file: EMSABTAG.H
Applies to: All directory objects. These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x80a8
Property type: PT_MV_BINARY
Access: Read/write

Remarks
