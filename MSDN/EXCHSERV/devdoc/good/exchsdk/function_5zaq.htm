GetAdrBookPtr

The GetAdrBookPtr function gets the address book for the gateway.

Quick Info

Header file: GWMAIN.H
Library: GWMAIN.LIB 

LPADRBOOK GetAdrBookPtr(
  VOID  
);

Parameters

None.

Return Values

A pointer to a MAPI IAddrBook interface containing the address book for the gateway.

Remarks

For information on the IAddrBook interface, see the MAPI Programmer's Reference.
