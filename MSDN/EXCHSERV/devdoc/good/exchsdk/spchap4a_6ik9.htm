BossWantsCopy

The BossWantsCopy property returns a Boolean value indicating whether the owner, in addition to any delegates, would like to receive a copy of the meeting mail.

Applies To

Schedule object

Data Type

Boolean

Access

Read/write

Remarks

Only the owner can write to the BossWantsCopy property — everyone else has read-only access. For more information, see the Schedule+ Help.
