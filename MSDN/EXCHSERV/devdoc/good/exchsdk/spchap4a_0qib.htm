AutoAlarms

The AutoAlarms property returns a Boolean value that determines whether an alarm should be set for new appointments and events.

Applies To

Schedule object

Data Type

Boolean

Access

Read/write

Remarks

The AutoAlarms property is usually set by users through the Schedule+ user interface. If this property is set to True, all new Appointment and Event objects will automatically have their AlarmAmount property set to the value of the AlarmAmountDefault property.
