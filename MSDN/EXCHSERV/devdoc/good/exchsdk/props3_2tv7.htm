PR_EMS_AB_TARGET_ADDRESS

The PR_EMS_AB_TARGET_ADDRESS property contains the foreign system e-mail address of a custom recipient. For example, an Internet custom recipient's target address might be SMTP:PatParkes@trager.com.

Quick Info

Header file: EMSABTAG.H
Applies to: Remote-Address objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8011
Property type: PR_EMS_AB_TARGET_ADDRESS PT_TSTRING

PR_EMS_AB_TARGET_ADDRESS_A PT_STRING8

PR_EMS_AB_TARGET_ADDRESS_W PT_UNICODE
Access: Read/write
Unicode: Yes

Values

1
Minimum length
1123
Maximum length
