Hardware Requirements

The following hardware is recommended when using the Schedule+ Automation interface:

  • Personal computer with a 32-bit 486/25 or higher; Pentium, or supported RISC-based processor, such as the MIPS R4000, Digital Alpha AXP; and PowerPC (running Windows NT® Server for RISC, or Windows, Windows 95 or Windows NT Server for 486/25 or higher)
  • VGA 16-color display (Super VGA (800 x 600) 256-color display recommended)
  • One or more hard disks with 30 megabytes (MB) minimum free hard disk space on the partition containing Windows NT Server files (50 MB recommended)
  • CD-ROM drive
  • 12 MB RAM minimum for 486- or Pentium-based systems (16 MB recommended)
  • 16 MB RAM minimum for RISC-based systems
  • Network adapter card
