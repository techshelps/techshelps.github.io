RULEEDIT: Manipulating Rules

The RULEEDIT sample application demonstrates how to use an application to set, examine, and delete rules on a folder. For more information about rules, see About Rules and Sharing Information.

    To build RULEEDIT
  1. Change to the directory containing the RULEEDIT source files:

    \BKOFFICE\SAMPLES\EXCHANGE\RULEEDIT

  2. Type NMAKE.
