RPTDRIV: Monitoring Message Delivery

The RPTDRIV sample application demonstrates how to generate delivery/nondelivery reports and read/unread reports about messages traveling across a gateway. It also demonstrates using the REPORT library functions. For a simple procedure for setting up and verifying a gateway, see Importing and Exporting with DIRSYNC.

    To build RPTDRIV
  1. Change to the directory containing the RPTDRIV source files:

    \BKOFFICE\SAMPLES\EXCHANGE\RPTDRIV

  2. Type NMAKE.
