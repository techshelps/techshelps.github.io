ItemId

The ItemId property returns the item identifier for an object.

Applies to

All Item objects

Data Type

String

Access

Read-only

Remarks

Every item in the schedule has a unique item identifier. The value contained by the ItemId property string is set and managed by Schedule+. Applications can use this value as a unique global item identifier of an item. Only item identifiers returned by Schedule+ are valid.
