Restrictions on Import Values

The DAPIWrite and BatchImport import functions can import attribute values expressed only as 7-bit ASCII characters. In this respect, they follow the model of the Administrator program. Import functions map Common-Names (and other values such as server names, account names, and passwords) that fall outside the set of 7-bit ASCII characters.

Disallowed characters are mapped to other characters that look similar, or to spaces. For example, Å maps to A, and £ maps to L. The case of the original letter is preserved.
