FreeThreadParameter

The FreeThreadParameter function frees the thread context.

Quick Info

Header file: GWMAIN.H
Library: GWMAIN.LIB 

VOID FreeThreadParameter(
  LPVOID lpParameter  
);

Parameters

lpParameter
Input parameter. Points to the thread context to free.

Return Values

None.
