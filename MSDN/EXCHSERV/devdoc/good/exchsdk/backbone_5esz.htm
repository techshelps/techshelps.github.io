Backboning Scenarios

Backboning can be done between Microsoft Exchange Server sites in different organizations or between a Microsoft Exchange Server site and a foreign system that uses a MAPI client and can therefore process messages containing MAPI properties.

Site-to-site replication is not supported for Microsoft Exchange Server gateways. The sample gateway provided with this SDK cannot be used as a site connector. Replication between Microsoft Exchange Server sites must be handled by a site connector and not a gateway.
