IExchangeFolderACLs

The IExchangeFolderACLs interface supports modifying a public folder's Access Control List (ACL).

Quick Info

Header file: ACLCLS.H
Exposed by:
Corresponding pointer type: LPFOLDERACLS

Vtable Order

HrDelete

HrGet

HrInsert

HrModify

HrSeek

HrTell

Remarks

To get a pointer to this interface, call the HrFolderACLsOpen function.

For additional information on this interface, see Manipulating ACLs.
