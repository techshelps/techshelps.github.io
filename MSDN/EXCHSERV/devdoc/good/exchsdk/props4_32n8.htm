PR_LAST_MODIFIER_ENTRYID

The PR_LAST_MODIFIER_ENTRYID property contains the entry identifier of the last person to modify a message.

Quick Info

Header file: EDKMDB.H
Applies to: Message objects (information store objects)
Identifier: 0x6643
Property type: PT_BINARY
Access: Read-only

Remarks

The PR_IPM_FAVORITES_ENTRYID property is set by the information store every time changes are saved. It can be used by an application to display the address book entry identifier of the last user to modify a message.
