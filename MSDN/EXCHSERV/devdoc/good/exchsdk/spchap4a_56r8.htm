AmountEstimatedEffort

The AmountEstimatedEffort property specifies the estimated amount of effort required to perform a task.

Applies To

Task object

Data Type

Long

Access

Read/write

Remarks

The number to which the AmountEstimatedEffort property is set has meaning only to the application that sets it. Schedule+ does not interpret or process the value stored in this property. The unit of measure for this value is also application-specific.
