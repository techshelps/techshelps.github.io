PR_EMS_AB_FOLDERS_CONTAINER

The PR_EMS_AB_FOLDERS_CONTAINER property contains the distinguished name of the container into which new public folders are to be created.

Quick Info

Header file: EMSABTAG.H
Applies to: Information-Store-Cfg objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
