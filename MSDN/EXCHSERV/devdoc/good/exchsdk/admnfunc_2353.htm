Example: Performance Monitoring

The following topics show techniques used by the sample gateway and the sample performance monitor DLL for performance monitoring, such as installing counters in the Windows NT Registry so the Windows NT Performance Monitor will recognize them.

Installing the counters requires creating counter definition files, some initial setup of the registry, and loading the counter definition file information into the registry. It also requires that the performance monitor DLL be set up in a directory specific to your gateway.

For more information about installing counters, see Optimizing Windows NT in the Windows NT Server Resource Kit.
