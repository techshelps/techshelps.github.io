Creating Online Help

The online Help file supplied with the Administrator extension should provide administrators with instructions for setting up and configuring the application using the application-specific property pages.

For the property pages that display extension data, you need to create and supply the online Help file with your application. Follow the procedure below to create an online Help file for the property pages governed by the extension DLL.

    To create online Help files
  1. Write the Help file, using a text editor that can produce files in the required rich text format (RTF). Microsoft Word is recommended.
  2. Add hypertext jumps and other features you want your Help file to offer. For more information, refer to the Tools User's Guide of the Microsoft Win32 Software Development Kit.
  3. Save the Help text file in .RTF format and compile the .HLP file.
