PR_EMS_AB_ENCAPSULATION_METHOD

The PR_EMS_AB_ENCAPSULATION_METHOD property contains the default encapsulation instructions to be used for recipients on a gateway.

Quick Info

Header file: EMSABTAG.H
Applies to: Mail-Gateway objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x80a2
Property type: PT_LONG
Access: Read/write
