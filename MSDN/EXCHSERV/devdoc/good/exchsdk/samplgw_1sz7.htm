Gateway Sample Applications

Several sample applications illustrate gateway-related programming tasks. Each sample application is located in its own directory under \BKOFFICE\SAMPLES\EXCHANGE. They are:

Sample Purpose
LOOKUP Looks up proxy addresses.
PERFTEST Performance monitoring.
PROXYGEN Generates proxy addresses.
QUEVIEW Displays the contents of the gateway's incoming and outgoing folders.
RPTDRIV Driver for report generation.
XLATDRIV Driver for message translation.

See Installation Programs for more information about installing each of these sample applications. For information about setting up your development environment, see Development Environment.
