Group Enabling

Schedule+ can be run in group-enabled mode (also known as mail-enabled) or stand-alone mode. Group enabling lets the user send e-mail, meeting requests, and so on, but requires that the user be logged on to a MAPI-compliant e-mail system.

In group-enabled mode, the Logon and Logoff methods on the Application object can be used to log the user on or off when starting or quitting the application.

For the Visual Basic application to create meetings, access privileges, and so on, MAPI properties must be used. See the MAPI Programmer’s Reference for more information.
