AccessLists

The AccessLists property is a multivalued property that contains access levels of a table.

Applies To

AccessControls table

Data Type

Object (Table)

Access

Read/write

Remarks

Only the owner of the schedule file can read this table. The values in the AccessLists property have a one-to-one correspondence with values in the AccessControls property, which lists the access level assigned to each table specified by the AccessLists property. For more information on how to use the AccessLists property, see Overview.
