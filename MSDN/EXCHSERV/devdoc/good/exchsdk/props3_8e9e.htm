PR_EMS_AB_OWNER

The PR_EMS_AB_OWNER property contains the distinguished name of the mailbox that owns the object.

Quick Info

Header file: EMSABTAG.H
Applies to: Group-Of-Names, Add-In, Distribution-List, Mailbox-Agent, and Device objects (directory objects. These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x800c
Property type: PR_EMS_AB_OWNER PT_TSTRING

PR_EMS_AB_OWNER_A PT_STRING8

PR_EMS_AB_OWNER_O PT_OBJECT

PR_EMS_AB_OWNER_T PT_TSTRING

Access: Read/write
