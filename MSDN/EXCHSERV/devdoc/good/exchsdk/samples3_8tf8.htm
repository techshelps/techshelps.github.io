Using MBSETUP to Install SMBAGENT

To use the MBSETUP setup utility to install the sample mailbox agent, the files must be moved to the correct directories prior to running the program. Do this by executing the makefile in the \BKOFFICE\SAMPLES\EXCHANGE\MBSETUP directory, or by moving the files yourself. See the previous section if you did not complete this step.

    To install SMBAGENT using MBSETUP
  1. Gather the information necessary to install the mailbox server. The required information includes:
    • Display name (address-book entry) for the service
    • Service name used to start and stop service, which may or may not be the same as the display name
    • Service logon account
    • Password for that account
    • Installation directory path
    • Container for installation of the mailbox agent. Using any container other than the Recipients container for this installation can produce unexpected results.
  2. Confirm the organization name and the site name, which will be generated from the system information.
  3. Double-click the Sample Mailbox Agent Setup icon in the Microsoft Exchange Server SDK group on your desktop. This installs both SMBAGENT and SMBADMIN, and adds default configuration data to the directory. SMBAGENT should be installed in the Recipients container.
  4. In the Microsoft Exchange Server Administrator program, look in the Recipients container and find the newly installed service. Double-click the service to display its property sheet, then click on the Options tab. In the Options dialog box, enter a name for the topic root folder and a topic root folder comment. Also enter a time value in milliseconds for the Poll Inbox option. For example, enter 100,000 milliseconds to indicate that the service should poll its inbox every 10 seconds. Entries must be greater than 60,000.
