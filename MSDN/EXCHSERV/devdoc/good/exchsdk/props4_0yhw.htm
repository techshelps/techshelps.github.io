PR_NON_IPM_SUBTREE_ENTRYID

The PR_NON_IPM_SUBTREE_ENTRYID property contains the entry identifier of a top-level folder for the non-IPM subtree.

Quick Info

Header file: EDKMDB.H
Applies to: Public folder store objects (information store objects)
Identifier: 0x6620
Property type: PT_BINARY
Access: Read-only

Remarks

The non-IPM subtree is the area of the public store that is not seen by users. It always contains a standard set of folders and can also contain folders created by applications. Subfolders of the non-IPM subtree include a Schedule+ information folder that contains a subfolder for each site, an offline address book folder containing subfolders for each site, and a forms library folder containing forms in various languages. Some folders on the non-IPM subtree are replicated and others are not.
