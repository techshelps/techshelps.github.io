Hardware Requirements

The following hardware is recommended for Microsoft Exchange Server application development:

  • Intel 80486/25 or higher; Intel Pentium processor; RISC-based microprocessors such as MIPS R4000 and Digital Alpha AXP; PowerPC
  • Super VGA (800 x 600) 256-color display (recommended)
  • 50 MB of disk space (recommended)
  • CD-ROM drive
  • 32-MB RAM or more (recommended)
  • Network adapter card

In addition to a computer for application development, many developers use one or more extra computers to test their applications under varying memory and display configurations. Although extra computers are not required, they are recommended.
