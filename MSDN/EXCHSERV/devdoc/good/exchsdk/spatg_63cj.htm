Software Requirements

To design applications for Schedule+, you need the following software installed on your system:

  • Microsoft Windows NT Server version 3.51 or later, or Windows 95
  • Microsoft Win32 SDK
  • Microsoft Windows-compatible C or C++ compiler and development environment

The following list provides examples of development environments that support Automation:

  • Microsoft Visual Basic version 4.0 or later
  • Microsoft Visual Basic for Applications (included with Microsoft Access, Microsoft Excel, Microsoft Project, and Microsoft Visual FoxPro™)
  • Microsoft Access version 2.0 or later
  • Microsoft Excel version 5.0 or later
  • Microsoft Project version 4.0 or later
  • Microsoft Visual C++ version 1.5 or later (version 2.0 recommended)

Note Although you can use Microsoft Visual Basic version 3.0 as the development environment for Schedule+, you cannot get or set multiple properties in one call. Therefore, Visual Basic version 4.0 should be used when developing for the Schedule+ Automation interface.
