PR_EMS_AB_PROXY_GENERATOR_DLL

The PR_EMS_AB_PROXY_GENERATOR_DLL property contains the name of a proxy generation DLL. Refer to the Gateway Programmer’s Reference for details.

Quick Info

Header file: EMSABTAG.H
Applies to: Addr-Type objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x810f
Property type: PR_EMS_AB_PROXY_GENERATOR_DLL PT_TSTRING

PR_EMS_AB_PROXY_GENERATOR_DLL_A PT_STRING8

PR_EMS_AB_PROXY_GENERATOR_DLL_W PT_UNICODE
Access: Read/write
Unicode: Yes

Values

1
Minimum length
255
Maximum length

Remarks

The PR_EMS_AB_PROXY_GENERATOR_DLL property only contains the DLL file name of the proxy address, not the full path name. The DLL is found in the following path:

\\server\address\<address-type>\<computer>

where an example of <address-type> is SMTP and a <computer> is i386.
