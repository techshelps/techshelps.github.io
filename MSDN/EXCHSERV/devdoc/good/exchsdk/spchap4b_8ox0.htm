OwnerEntryId

The OwnerEntryId property returns the mail entry identifier for the meeting owner of an item.

Applies To

Appointment and RecurringAppointment objects

Data Type

String

Access

Read/write

Remarks

The value for the OwnerEntryId property is supplied by the MAPI entry identifier. The meeting owner is usually the user who creates the meeting, except when a delegate creates the meeting instead.
