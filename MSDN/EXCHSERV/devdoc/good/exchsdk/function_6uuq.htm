HrRemoveMessageConverter

The HrRemoveMessageConverter function deletes a message conversion DLL.

Quick Info

Header file: EXCHINST.H
Library: EXCHINST LIB 

HRESULT HrRemoveMessageConverter(
  LPCSTR lpszConverter  
);

Parameters

lpszConverter
Input parameter. Points to a null-terminated string containing the name of the message converter DLL to remove.

Return Values

See Return Values.
