Installing the Microsoft Exchange Server SDK

Once Microsoft Exchange Server is installed, the next step is to install the SDK itself. If you plan to install and build all the sample applications, at least 50 MB of disk space is required.

    To install the Microsoft Exchange Server SDK
  1. Change to the directory containing the setup program for the sample applications:

    \BKOFFICE\<computer type>\DEBUG\SAMPLES\EXCHANGE

  2. Type SETUP at the command prompt.

The setup program automatically sets several environment variables.

The Microsoft Exchange Server SDK includes a set of sample applications that are already compiled. They are located in the \BKOFFICE\BIN directory. See Building the Sample Applications.
