PR_EMS_AB_ADDRESS_ENTRY_DISPLAY_TABLE_MSDOS

The PR_EMS_AB_ADDRESS_ENTRY_DISPLAY_TABLE_MSDOS property contains the MS-DOS format display table for an address entry.

Quick Info

Header file: EMSABTAG.H
Applies to: Display-Template objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8047
Property type: PT_BINARY
Access: Read-only

Remarks

For more information about display tables, see the MAPI Programmer’s Reference.
