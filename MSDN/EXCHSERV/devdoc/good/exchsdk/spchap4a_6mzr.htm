Billing

The Billing property lists billing information for a task.

Applies To

Task object

Data Type

String

Access

Read/write

Remarks

The format of the Billing property is application-specific and Schedule+ does not interpret or process it. Typically, applications use this property to store billing infomation for a task, such as a client’s billing address, purchase order number, how much the task cost, hourly billing rate, invoice number, or any other billing information.
