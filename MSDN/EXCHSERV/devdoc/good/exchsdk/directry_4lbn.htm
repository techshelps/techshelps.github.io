Files Used with Directory Access Functions

The following header files provide directory access functionality.

Directory Access Files

File Description
DAPI.H Contains structure and function declarations.
DAPI.LIB Contains export library that modules must link to in order to use directory access functions.
DAPI.DLL Contains directory access functions.
DAPIMSG.H Contains the output from the message compiler, which are the error and warning codes that directory access functions work with. These are string resources in DAPI.DLL.
