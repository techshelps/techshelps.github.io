Using MAPI Folders

A Microsoft Exchange Server gateway uses the MAPI folders MTS-IN and MTS-OUT to submit messages to and receive messages from Microsoft Exchange Server. You can also design your gateway so it uses other MAPI folders to store messages while it processes them. Each MAPI folder has a contents table. The contents table for a folder created by a gateway will contain:

  • One row for each message contained within the folder.
  • Default content properties defined by MAPI in each row.
