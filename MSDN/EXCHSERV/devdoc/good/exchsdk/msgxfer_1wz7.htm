MTE Per-Message Properties

The following table shows the per-message properties of a message’s MTE. Many of these properties are native MAPI properties, but some are properties defined by Microsoft Exchange Server.

MTE Per-Message Properties

Property name Type Description
PR_ARRIVAL_TIME Date/
Time		 Date and time the message arrived at the gateway.
PR_CONTENT_SEARCH_KEY Binary Entry identifier of the message.
PR_FOREIGN_ID Binary Identifier of message within the foreign system. On outgoing messages from Microsoft Exchange Server, this property initially will not exist, but is added by the gateway.
PR_FOREIGN_REPORT_ID Binary Identifier assigned to the reports generated by the gateway in response to reports received from the foreign system.
PR_FOREIGN_SUBJECT_ID Binary Subject identifier assigned to reports generated by the gateway in response to reports received from the foreign system.
PR_MTS_ID Binary The Microsoft Exchange Server MTE identifier. If this property is not supplied by the gateway on incoming messages, it will be supplied by Microsoft Exchange Server.
PR_MTS_REPORT_ID Binary MTS identifier for reports.
PR_MTS_SUBJECT_ID Binary MTS identifier of the message for which a report was generated.
PR_ORIGINATOR_ADDR String E-mail address of message originator.
PR_ORIGINATOR_ADDRTYPE String Address type of message originator.
PR_ORIGINATOR_ENTRYID Binary Entry identifier of message originator.
PR_ORIGINATOR_NAME String Display name of message originator.
PR_RECIPIENT_NUMBER Long Index into the table of recipients.
PR_REPORT_DESTINATION_ENTRYID Binary Entry identifier of the immediate destination of the report.
PR_REPORT_DESTINATION_NAME String Display name of the immediate destination of the report.
PR_SUBJECT_TRACE_INFO Binary Trace information on the message report.
PR_TRACE_INFO Binary Trace information indicating how each messaging domain handled the message. See Gateway Trace Information for details.
