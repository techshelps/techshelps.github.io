G

gateway
A software service that allows Microsoft Exchange Server users to communicate with other mail systems, such as IBM PROFS. Gateways can be used to send messages between two Microsoft Exchange Server sites through an intermediate mail system through a process called backboning (backbone), but do not support directory replication.
gateway address routing table (GWART)
A table that contains information about the gateways available to Microsoft Exchange Server and the address types associated with each one. When a message needs to be routed through a gateway, the GWART is used to determine the appropriate gateway to use for each address type.
global address list
A list of names of all possible recipients in a Microsoft Exchange Server organization.
