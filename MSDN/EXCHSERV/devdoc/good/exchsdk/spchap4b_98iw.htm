TablePath

The TablePath property returns a table in a path to the object through the Schedule+ table hierarchy.

Applies To

Alarm object

Data Type

Long (multivalued)

Access

Read-only

Remarks

A path consists of a set of tuples (a table and an item identifier). Because tables can be nested, each table and item identifier specifies a level into the Schedule+ table hierarchy. The TablePath and ItemIdPath properties make up a path.
