HrMAPIWriteAttachmentToFile

The HrMAPIWriteAttachmentToFile function saves a message attachment to a file.

Quick Info

Header file: EDKMAPI.H
Library: EDKMAPI.LIB 

HRESULT HrMAPIWriteAttachmentToFile(
  LPMESSAGE pMessage,  
  ULONG iAttach,       
  HANDLE hFile         
);

Parameters

pMessage
Input parameter. Points to a MAPI IMessage interface containing the attachments.
iAttach
Input parameter. Attachment identifier.
hFile
Input parameter. Handle to file.

Return Values

See Return Values.

Remarks

For more information on this function, see the HrMAPIWriteAttachmentToFile function in EDKMAPI.C under the EDKMAPI code sample in Code Samples folder.

For information on the IMessage interface, see the MAPI Programmer's Reference.
