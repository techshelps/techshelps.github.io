PROFINST Syntax and Parameters

    To run PROFINST

Type the following on the command line:

PROFINST /SERVICE=<service name> /NAME=<profile name>
/TYPE=<profile type> [/DELETE] [/HELP or /?]

PROFINST Parameters

Flag Setting
/SERVICE Specifies service name, or displays name of service.
/NAME Specifies profile name of the service.
/TYPE Specifies profile type (GATEWAY or AGENT).
/DELETE Deletes the profile.
/HELP or ? Displays Help information.
