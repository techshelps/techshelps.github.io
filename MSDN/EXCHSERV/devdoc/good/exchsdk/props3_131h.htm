PR_EMS_AB_TRACKING_LOG_PATH_NAME

The PR_EMS_AB_TRACKING_LOG_PATH_NAME property contains the path name to where the message tracking logs are kept.

Quick Info

Header file: EMSABTAG.H
Applies to: Computer objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8146
Property type: PR_EMS_AB_TRACKING_LOG_PATH_NAME PT_TSTRING
