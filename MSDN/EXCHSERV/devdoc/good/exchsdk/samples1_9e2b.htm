ACLEDIT Permissions

The following table lists the possible values for the /RIGHTS parameter. The permissions are granted to the user to perform the actions that appear in the description column.

Permission Description
READANY (READONLY) Read any message in the folder. Default.
CREATE Create messages in the folder.
EDITOWNED Edit any messages owned by himself in the folder.
DELETEOWNED Delete any messages owned by himself in the folder.
EDITANY Edit any messages in the folder.
DELETEANY Delete any messages in the folder.
CREATESUBFOLDER Create subfolders within the folder.
OWNER Set permissions on the folder.
CONTACT Establishes the user as the contact person for the folder.
NONE No permissions.
READWRITE Read and edit any items.
ALL Includes all permissions except Contact permissions.
