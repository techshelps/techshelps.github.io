Using the Raw Properties Screen

The Raw Properties screen is part of the Microsoft Exchange Server Administrator program. By setting data in the Raw Properties screen, you can establish values as they would be set by components you have not yet developed. Data is normally set and manipulated by your application or its setup program, and data is passed from one component to the next. Establishing this artificial environment allows you to test them while they are being developed. For example, you could debug an extension DLL by changing the underlying data.

To view the Raw Properties screen, select any object and choose Raw Properties from the File menu. Note that the Raw Properties screen is available only if you start the Administrator program with the /raw switch set.
