AccessDefault

The AccessDefault property identifies the default access control list (ACL) level for objects.

Applies To

AccessControls table

Data Type

Long

Access

Read/write

Remarks

Only the owner of the schedule file can read or modify the AccessControls table. This property identifies the default ACL level for items in the table. You can override this default by specifying the ACLs on a per-table basis. The default should always be present.

The AccessDefault property has the following values.

Value Description
0 No access
1 Minimal read
2 Read
3 Create
4 Write
5 Owner
