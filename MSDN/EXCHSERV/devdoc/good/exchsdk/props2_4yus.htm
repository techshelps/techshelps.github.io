PR_EMS_AB_ALT_RECIPIENT

The PR_EMS_AB_ALT_RECIPIENT property contains the distinguished name of an alternate recipient mailbox that will receive all incoming mail for a mailbox.

Quick Info

Header file: EMSABTAG.H
Applies to: Mail-Recipient objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x804e
Property type: PR_EMS_AB_ALT_RECIPIENT PT_TSTRING

PR_EMS_AB_ALT_RECIPIENT_O PT_OBJECT
