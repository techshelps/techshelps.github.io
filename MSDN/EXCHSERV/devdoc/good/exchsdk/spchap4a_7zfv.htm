AlarmAmountDefaultTask

The AlarmAmountDefaultTask property specifies the default time interval before the start or end of a task when an alarm will ring.

Applies To

Schedule object

Data Type

Long

Access

Read/write

Remarks

The BeforeEnd property setting on the Task object determines whether the default time interval specified is before the start or end of a task.

Because this property is set by the user in the Schedule+ Options dialog box, applications read the value of this property, but do not write a value to it. For more information, see the Schedule+ Help.
