PR_EMS_AB_USN_CREATED

The PR_EMS_AB_USN_CREATED property contains the update sequence number (USN) for the object at the time it was created.

Quick Info

Header file: EMSABTAG.H
Applies to: All directory objects. These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8154
Property type: PT_LONG
Access: Read-only

Remarks

The PR_EMS_AB_USN_CREATED property of an object is set to the next USN from the counter when the object is created. For a newly created object, the PR_EMS_AB_USN_CHANGED and PR_EMS_AB_USN_CREATED properties are equal.
