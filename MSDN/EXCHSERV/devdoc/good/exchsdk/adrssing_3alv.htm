Providing Online Help for Message and Recipient Options

The online Help file supplied with the Addr-Type object provides the user with instructions for using the gateway’s message and recipient options. It is a normal Microsoft Windows Help file. It should be made available on all Windows-based clients, but not on Microsoft Exchange Server MS-DOS clients. The MS-DOS client does not expose those options and cannot use Windows Help files.

There are no special requirements for context identifier or topic titles for this Help file. To link a property sheet to its help file, put the context identifier for the topic in the control flags field of the property sheet’s DTCT_PAGE control in its .WTX file. For example, an entry in a WTX file that looks like this:

0,  0,  0,  0,  DTCT_PAGE,  3300,  0,  0,General

specifies a property sheet called “General” that uses a help topic with the context identifier 3300.

Microsoft Exchange Server handles the details of allowing users to access the contents of this file; all you have to do is ensure that a topic with that context identifier is present in the help file specified when you install the property page. See Creating the Options Template for details.
