Posting Edited Text with POSTSMPL

    To post edited text with POSTSAMPL
  1. Create the message as directed in Sending Edited Text with POSTSMPL.
  2. Post the message to an existing folder by choosing Post Message from the File menu. The Select Folder dialog box appears.

  3. Select the folder in which you want to post the file. Click OK.

  4. To verify that the folder has been posted, edit the folder by selecting the new posting. Your edited text should appear.

    To view the MAPI property values that were set on this message
  1. Activate the Microsoft Exchange Client.
  2. From the toolbar, select View, then Columns
  3. Select the columns representing the summary properties to verify the values.
