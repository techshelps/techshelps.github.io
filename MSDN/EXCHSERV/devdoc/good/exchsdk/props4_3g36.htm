PR_PROFILE_HOME_SERVER

The PR_PROFILE_HOME_SERVER property contains the name of a home server.

Quick Info

Header file: EDKMDB.H
Applies to: Profile objects
Identifier: 0x6602
Property type: PT_STRING8
Access: Read/write

Remarks

No backslash (\) characters are used in the values for this property.

The PR_PROFILE_HOME_SERVER property can be edited in the Edit Profile option of the Messaging Profiles dialog box, within which PR_PROFILE_DOWNLOAD is represented by the Server Name field on the General page of the Microsoft Exchange Server Information Service dialog box.

The PR_PROFILE_HOME_SERVER property is used in the global sections of Microsoft Exchange Server profiles.
