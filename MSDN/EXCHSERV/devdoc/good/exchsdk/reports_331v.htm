Nondelivery Report Properties

The following table shows the MAPI properties of nondelivery reports, and how these properties relate to the properties of the original message.

Nondelivery Report Property Original Message Property
PR_REPORT_DESTINATION_NAME PR_SENDER_NAME, PR_REPORT_NAME
PR_REPORT_DESTINATION_ENTRYID PR_SENDER_ENTRYID, PR_REPORT_ENTRYID
PR_MTS_SUBJECT_ID PR_MTS_ID
PR_DISPLAY_NAME PR_DISPLAY_NAME
PR_ADDRTYPE PR_ADDRTYPE
PR_EMAIL_ADDRESS PR_EMAIL_ADDRESS
PR_SEARCH_KEY PR_SEARCH_KEY
PR_ENTRYID PR_ENTRYID
PR_DISCRETE_VALUES Used for distribution lists with nondelivered members.
PR_ORIGINAL_DISPLAY_NAME Used for a nondelivered member of a distribution list.
PR_ORIGINAL_ENTRYID Used for a nondelivered member of a distribution list.

For details about the properties on reports, see the topic Required Report Message Properties in the MAPI Programmer’s Guide.
