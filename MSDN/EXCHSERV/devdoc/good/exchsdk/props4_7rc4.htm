PR_SPLUS_FREE_BUSY_ENTRYID

The PR_SPLUS_FREE_BUSY_ENTRYID property contains the entry identifier of the Schedule+ free/busy subtree top-level folder.

Quick Info

Header file: EDKMDB.H
Applies to: Public store objects (information store objects)
Identifier: 0x6622
Property type: PT_BINARY
Access: Read-only

Remarks

The PR_SPLUS_FREE_BUSY_ENTRYID property identifies the top-level folder. The Schedule+ free/busy subtree is not seen by users, but is replicated.

The Schedule+ free/busy subtree top-level folder is a subfolder of the non-IPM subtree. There is an individual subfolder for each site.
