Installation Programs

The programs described in the following topics install individual gateway components. All these programs are located in the \BKOFFICE\SAMPLES\EXCHANGE\EDKINSTU directory.

    To build the installation programs
  1. Change to the \BKOFFICE\SAMPLES\EXCHANGE\EDKINSTU directory.
  2. Type NMAKE.

These programs do not copy files. Instead, they take care of creating various objects in Microsoft Exchange Server. Therefore, when you run them, you must do the following:

  • Ensure that all the files you need are located in the appropriate directories.
  • Include the full path when entering parameters that use file names.
  • Ensure that the directory access function DLL (DAPI.DLL) is located on your path.

In all the installation programs, you must leave a space between the parameters, as shown in the syntax examples.
