PR_EMS_AB_ORGANIZATIONAL_UNIT_NAME

The PR_EMS_AB_ORGANIZATIONAL_UNIT_NAME property is for X.500 compatibility and is not used by Microsoft Exchange Server.

Quick Info

Header file: EMSABTAG.H
Applies to: Application-Entity, Application-Process, Device, Group-Of-Names, Organizational-Person, Organizational-Role, Organizational-Unit, and Residential-Person objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8102
Property type: PR_EMS_AB_ORGANIZATIONAL_UNIT_NAME PT_MV_TSTRING
