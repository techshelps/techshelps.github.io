tombstone
An object with the Is-Deleted attribute set that remains in the Microsoft Exchange Server for a set period of time. See also Exporting and Importing Changed Objects.
Transport-Neutral Encapsulation Format (TNEF)
A standard method defined by MAPI of passing textual and binary message data. The method involves wrapping unsupported MAPI properties in a binary attachment file that accompanies a message through the transport and decoding the file on the receiving side to reconstitute the MAPI properties.
