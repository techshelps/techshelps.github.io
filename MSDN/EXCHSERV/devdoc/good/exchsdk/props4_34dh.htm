PR_ORIGINATOR_ADDRTYPE

The PR_ORIGINATOR_ADDRTYPE property contains the address type of a message originator.

Quick Info

Header file: EDKMDB.H
Applies to: Message objects (information store objects)
Identifier: 0x665d
Property type: PT_TSTRING
Access: Read/write

Remarks

The PR_ORIGINATOR_ADDRTYPE property is used on the message transfer envelope.
