PR_EMS_AB_ADMD

The PR_EMS_AB_ADMD property determines whether the internal trace information is stripped.

Quick Info

Header file: EMSABTAG.H
Applies to: Mail-Gateway and MTA objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8049
Property type: PR_EMS_AB_ADMD PT_TSTRING

PR_EMS_AB_ADMD_A PT_STRING8

PR_EMS_AB_ADMD_W PT_UNICODE
Access: Read/write
Unicode: Yes

Values

1
Minimum length
16
Maximum length
