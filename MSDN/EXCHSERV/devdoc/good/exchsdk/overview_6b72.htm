Sharing Information

Microsoft Exchange Server allows users to share information either by sending e-mail or by sharing it with others using public folders. Applications can also send messages and access the public folders.

Information is organized and managed using rules, views, and the Find window.

  • Rules Can be automatically created or changed by your application. Microsoft Exchange Server uses rules to filter incoming mail and take actions such as moving selected objects to a particular folder, forwarding to another user, or deleting certain selected types of information. Auto-assistants are user interface options that can be used by your applications to create rules that sort items even when the user is not logged on.
  • Views Can be created or changed by custom applications. Rules allow users to organize and view information by selecting a predefined view or defining a custom view.
  • The Find window Allows applications or users to search through existing items stored in any folder by using a wide range of search criteria.
