About Gateway Setup Programs

A primary function of the setup program is to copy all necessary files from floppy disks or CD to the server’s hard disk. The setup program must also ensure that the gateway is recognized by Microsoft Exchange Server system components and can communicate with them. Your setup program can use the utility functions provided with the Microsoft Exchange Server SDK to manage the gateway’s configuration information.

Your setup program can include some or all of the following steps:

  • Collect and verify system installation requirements from the administrator. See Collecting System Requirements.
  • Install the gateway as a service of Windows NT Server using the Windows NT service control manager. See Installing the Gateway as a Service.
  • If you use an Administrator extension DLL to add property pages to the gateway’s directory object, the setup program can install its associated Admin-Extension object in the Microsoft Exchange Server directory. See Installing an Admin-Extension Object.
  • Certain files are required for gateways that use Administrator extensions or address types. The setup program can copy these directory-related files onto the server’s hard disk. See Copying Files Onto Disk.
  • Copy the required files for the gateway itself onto the computer on which the gateway will run (not necessarily a Microsoft Exchange Server computer). See Copying Files Onto Disk.
  • Set the Gateway bit in the Heuristics attribute on the Server object. See Setting the Server Heuristics Attribute.
  • Create the Mail-Gateway object in the Microsoft Exchange Server directory. See Creating the Directory Object.
  • Set the attribute values of the gateway object. See Initializing the Extension-Data Attribute.
  • Install address type and proxy address information, and rebuild the gateway address routing table. See Installing Addressing, Updating the Gateway-Proxy Field of the Site Object, and Rebuilding the Gateway Address Routing Table.
  • Generate proxy addresses for current recipients. See Generating Recipient Proxy Addresses.
  • Set up performance monitoring for your gateway. See Installing Performance Monitoring.
  • Reconfigure other system elements as required. See Reconfiguring the System.
