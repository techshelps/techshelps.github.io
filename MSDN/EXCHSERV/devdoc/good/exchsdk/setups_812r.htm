Installing Applications with Server and Client Components

In many cases, a server application is installed on a server and runs from that location only. However, some applications contain both server and client components. These applications need to be installed in two steps.

    To install an application with both a server and a client component
  1. Run a setup program on the server to install server application components. This step also copies files needed for client component installation to the server's public share, where they can be accessed by clients.
  2. When the server installation is complete, run a different setup program on the client to install and download client components from the server.
