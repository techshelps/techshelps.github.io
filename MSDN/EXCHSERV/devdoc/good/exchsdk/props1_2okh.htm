PR_ACL_DATA

The PR_ACL_DATA property contains the binary form of the PR_ACL_TABLE object, which is the access control list (ACL) table for a folder.

Quick Info

Header file: EDKMDB.H
Applies to: Folder objects (information store objects)
Identifier: 0x3fe0
Property type: PT_BINARY
Access: Read/write
