Specifying the Object Class

The first column specifies the type of object to be created. It must appear first and must have a name of Obj-Class. Options available for use as values in this column are shown in the following table. For an illustration, see Example: Importing Directory Data.

Import Object Types (with Object Types in Column 1)

Value Comment
mailbox Designates that a mailbox is to be created
remote Designates that a remote user is to be created
dl Designates that a distribution list is to be created.
