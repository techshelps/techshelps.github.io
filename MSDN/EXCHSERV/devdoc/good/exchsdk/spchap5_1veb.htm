UserAddress

The UserAddress data member returns the e-mail address of the currently logged-on Schedule+ user.

Applies To

Application and Schedule objects

Data Type

String

Access

Read-only

Remarks

The format of the UserAddress data member is a human-readable e-mail address that can be understood by the underlying mail system. For example, a valid value for this data member is SMTP:BrianW@Cyber.Com for an SMTP user on a TCP/IP network.
