PR_EMS_AB_ROUTING_LIST

The PR_EMS_AB_ROUTING_LIST property contains a list of entries describing the address types supported by a connector or gateway. The set of supported address types is called the connector's address space.

Quick Info

Header file: EMSABTAG.H
Applies to: Mail-Gateway, Site-Addressing, and Site-Connector objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8126
