PR_EMS_AB_PREFERRED_DELIVERY_METHOD

The PR_EMS_AB_PREFERRED_DELIVERY_METHOD property is for X.500 compatibility and is not used by Microsoft Exchange Server.

Quick Info

Header file: EMSABTAG.H
Applies to: Organization, Organizational-Person, Oganizational-Role, Organizational-Unit, and Residential Person objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x810d
Property type: PT_MV_LONG
Access: Read/write
