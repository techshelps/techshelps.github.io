PR_GW_ADMIN_OPERATIONS

The PR_GW_ADMIN_OPERATIONS property is used to pass instructions between gateway administration tools and gateways.

Quick Info

Header file: EDKMDB.H
Applies to: Message and public folder objects (information store objects)
Identifier: 0x6658
Property type: PT_LONG
Access: Read/write

Values

The PR_GW_ADMIN_OPERATIONS property has the following Folder values when used on a folder:

Bit number Mask name Meaning
0 ADMINOP_HIDDEN 0 – Messages can be viewed. Default.

1 – Messages cannot be viewed.
1 ADMINOP_DEFER 0 – No deferred processing. Default.

1 – Deferred processing OK.
2 ADMINOP_DELETE 0 – Messages in this folder cannot be deleted. Default.

1 – Message delete OK.
3 ADMINOP_NDR 0 – Delivery of messages in this folder cannot be aborted. Default.

1 – Delivery can be aborted.
4 – 15 Reserved. Must be 0.
16 – 31 Available for use by gateway developers.

The PR_GW_ADMIN_OPERATIONS property has the following values when used on a message, representing deferred processing to be done by the gateway:

Bit Number Mask Name Meaning
0 ADMINOP_HIDDEN Ignored. Must be 0.
1 ADMINOP_DEFER Ignored. Must be 0.
2 ADMINOP_DELETE 0 – No action. Default.

1 – Delete message. Do not send NDR.
3 ADMINOP_NDR 0 – No action. Default.

1 – Delete message. Send NDR.
4 – 15 Reserved. Must be 0.
16 – 31 Available for use by gateway developers.

Remarks

The PR_GW_ADMIN_OPERATIONS property is used to pass instructions between gateway administration programs and gateways. The QUEVIEW sample application is an example of a gateway administration program. It allows administrators to view and manipulate the contents of gateway mailbox folders like MTS-IN and MTS-OUT.

For more information, see QUEVIEW: Listing Gateway Folders.
