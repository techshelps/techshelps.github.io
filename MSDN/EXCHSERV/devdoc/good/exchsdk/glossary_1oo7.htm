G

gateway
A software service that allows Microsoft Exchange Server users to communicate with other mail systems, such as IBM PROFS. Gateways are used to send messages between two Microsoft Exchange Server sites through an intermediate mail system using a process called backboning, but do not support directory replication.
global address list
The directory of all Microsoft Exchange users and distribution lists (DLs) to whom messages can be addressed. The administrator creates and maintains this list. The global address list may also contain public folder names. Entries from this list can be added to a user’s personal address book.
globally unique identifier (GUID)
A Microsoft implementation of the DCE UUID. UUIDs uniquely identify objects, such as interfaces, manager entry-point vectors, and client objects. The RPC run-time libraries use UUIDs to check for compatibility between clients and servers and to select among multiple implementations of an interface.
