PR_EMS_AB_GARBAGE_COLL_PERIOD

The PR_EMS_AB_GARBAGE_COLL_PERIOD property contains the number of hours representing the interval at which garbage is collected.

Quick Info

Header file: EMSABTAG.H
Applies to: Directory-Cfg objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x80af
Property type: PT_LONG
Access: Read/write

Remarks

During garbage collection, objects older than the tombstone lifetime are permanently deleted from the system. The directory is then compacted for maximum efficiency.
