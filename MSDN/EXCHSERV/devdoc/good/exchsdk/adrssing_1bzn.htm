Address Entry Template Objects

If your gateway provides an Addr-Type object, it should also provide an address entry template. This template displays a dialog box in which the user enters information to create an address of the type supported by the gateway.

Templates make the address type generally available to users and allow users to create addresses by simply entering information in the dialog box. In this way the user does not need to know the details of address type syntax. The address entry template works together with a corresponding Address Syntax Program which converts the data in the dialog box specified by the address entry teplate dialog box into an actual proxy address string.

The address entry template is used by:

  • The administrator using the Microsoft Exchange Server Administrator program to create a new custom recipient object. The Administrator program command for this is New Custom Recipient from the File menu.
  • Users of the Microsoft Exchange Client to create one-off addresses either directly in e-mail messages or for storage in their personal address books.
