F

Favorites folder
A user-customized folder in the public information store that contains links to public folders for quick access. The Favorites folder is also used for synchronization with offline folders and as a means to provide an unread count on public folders. See also About the Favorites Folder.
Find window
A dialog box accessible from the Microsoft Exchange Client user interface that allows users to search for a specific text string or other message attributes in message properties.
folder
A MAPI object in an information store that is accessible for sharing. Folders can contain objects such as messages, files, and other folders.
folder hierarchy
Tree of public folders, private folders, and messages in the information store.
folder replication
foreign address
The address of an object in a foreign system, which is any mail system that is not a Microsoft Exchange Server system.
form
A MAPI object that is used to display a message of a particular message class in a structured format based on MAPI properties. User interaction with the form is controlled with a form server application. MAPI form objects implement the IMAPIForm and IMAPIFormInfo interfaces.
form manager
A MAPI service provider implemented as a DLL that manages form libraries using the IMAPIFormMgr interface.
form server
An application that manages a user’s interaction with a form, such as responding to menu commands.
