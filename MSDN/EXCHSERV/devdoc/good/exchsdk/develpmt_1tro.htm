Development Environment

Before you develop Microsoft Exchange Server applications, prepare your development environment by performing the following tasks:

  • Installing the necessary software
  • Setting environment variables. Optionally, this task includes setting up debug logging.

Other aspects of Microsoft Exchange Server application development include:

  • Compiling and linking your code with Microsoft Visual C++. You can use its interactive development environment or you can build applications from a command prompt with the command NMAKE.
  • Selecting the correct runtime libraries and include files.

To learn about developing applications for Microsoft Exchange Server by building its sample applications, in particular the Sample Gateway and Sample Mailbox Agent.
