EndRecurringDate

The EndRecurringDate property returns the end date for a recurrence pattern.

Applies To

RecurringAppointment, RecurringEvent, and RecurringTask objects

Data Type

Date

Access

Read/write

Remarks

The EndRecurringDate property does not have to be set if the recurrence pattern is indefinite.
