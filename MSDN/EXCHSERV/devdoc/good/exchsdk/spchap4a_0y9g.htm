AlarmAmount

The AlarmAmount property specifies the time interval before the start of an appointment, event, or task when an alarm will ring.

Applies To

Appointment, Event, RecurringAppointment, RecurringEvent, RecurringTask, and Task objects

Data Type

Long

Access

Read/write

Remarks

The AlarmAmount property is used in combination with the AlarmTypeUnit property to specify the time interval before the start or end of the object when the alarm will occur.
