DSIMPORT Syntax and Parameters

    To run DSIMPORT

Type the following on the command line:

DSIMPORT /FILE=<name of file to import from> /DSA=<name of directory service agent> /BASEPOINT=<DN of directory basepoint for import> [/CONTAINER=<name of container to import from>] [/HELP or /?]

DSIMPORT Parameters

Flag Settings
/FILE= Gives name of import file to receive information.
/DSA= Gives name of directory service agent (DSA) used to import directory objects.
/BASEPOINT= Gives DN of directory basepoint object.
/CONTAINER= Gives RDN of Directory container beneath BASEPOINT to which to import.
/HELP or /? Displays full Help information.
