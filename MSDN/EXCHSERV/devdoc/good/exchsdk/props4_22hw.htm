PR_IPM_PUBLIC_FOLDERS_ENTRYID

The PR_IPM_PUBLIC_FOLDERS_ENTRYID property contains the entry identifier of the top-level folder for the IPM subtree.

Quick Info

Header file: EDKMDB.H
Applies to: Public information store objects (information store objects)
Identifier: 0x6631
Property type: PT_BINARY
Access: Read-only

Remarks

The public information store is divided into the IPM subtree and the non-IPM subtree. The IPM subtree consists of a set of public folders that is replicated and seen by a user.

For more information, see Opening the Favorites Folder and Entry Identifiers of Special Folders.
