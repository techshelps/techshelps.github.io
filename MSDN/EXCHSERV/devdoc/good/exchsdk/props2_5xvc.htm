PR_EMS_AB_ADDRESS_SYNTAX

The PR_EMS_AB_ADDRESS_SYNTAX property contains the correct syntax to use when encoding the display table properties as a string.

Quick Info

Header file: EMSABTAG.H
Applies to: Address-Template objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8018
Property type: PT_BINARY
Access: Read-only

Remarks

The PR_EMS_AB_ADDRESS_SYNTAX property is valid only when reading a property from the Microsoft Exchange Server address book. It is not valid when reading a property from an object copied into another address book provider.

For more information, see Sample Address Syntax Program.
