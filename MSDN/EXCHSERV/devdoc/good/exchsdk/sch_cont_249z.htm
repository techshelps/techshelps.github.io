MHS-Link-Monitoring-Config

Class Information

Attribute Must Description
Monitor-Servers Indicates whether to Monitor servers or not
Monitoring-Alert-Delay The delay, in seconds, before alert recipients are notified of a problem
Monitoring-Alert-Units Units for Alert interval
Monitoring-Recipients The list of DNs of recipients to which to send monitoring information
Monitoring-Recipients-NDR The list of DNs of recipients from which NDR's should be received
Monitoring-Warning-Delay The delay, in seconds, before a monitoring warning is generated
Monitoring-Warning-Units Units for Warning interval
