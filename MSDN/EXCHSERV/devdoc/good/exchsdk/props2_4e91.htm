PR_EMS_AB_FOLDER_PATHNAME

The PR_EMS_AB_FOLDER_PATHNAME property contains the full path name of the folder.

Quick Info

Header file: EMSABTAG.H
Applies to: Public-Folder objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8004
Property type: PR_EMS_AB_FOLDER_PATHNAME PT_TSTRING

PR_EMS_AB_FOLDER_PATHNAME_A PT_STRING8

PR_EMS_AB_FOLDER_PATHNAME_W PT_UNICODE
Access: Read/write
Unicode: Yes

Remarks

The PR_EMS_AB_FOLDER_PATHNAME property is valid only when reading a property from the Microsoft Exchange Server address book. It is not valid when reading a property from an object copied into another address book provider.
