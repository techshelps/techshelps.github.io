PR_EMS_AB_MEMBER

The PR_EMS_AB_MEMBER property contains the list of members of a distribution list.

Quick Info

Header file: EMSABTAG.H
Applies to: Distribution-List and Group-Of-Names objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8009
Property type: PR_EMS_AB_MEMBER PT_OBJECT

PR_EMS_AB_MEMBER_A PT_MV_STRING8

PR_EMS_AB_MEMBER_O PT_OBJECT

PR_EMS_AB_MEMBER_T PT_MV_TSTRING

PR_EMS_AB_MEMBER_W PT_MV_UNICODE
Access: Read/write
Unicode: Yes

Values

1
Minimum length
256
Maximum length
