PR_TRACE_INFO

The PR_TRACE_INFO property contains trace information indicating how each messaging domain handled a particular message.

Quick Info

Header file: EDKMDB.H
Applies to: Message transfer envelope objects (information store objects)
Identifier: 0x6660
Property type: PT_BINARY
Access: Read/write

Remarks

The PR_TRACE_INFO property contains information about where a message has been since it was created. When the information associated with this property is prepared for transmittal across a foreign system, Microsoft Exchange Server encodes the information in a transmission-neutral encoding format (TNEF) with a little-endian byte order.
