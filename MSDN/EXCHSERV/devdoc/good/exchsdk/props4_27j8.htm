PR_SCHEDULE_FOLDER_ENTRYID

The PR_SCHEDULE_FOLDER_ENTRYID property contains the folder entry identifier for a schedule folder.

Quick Info

Header file: EDKMDB.H
Applies to: Mailbox objects (information store objects)
Identifier: 0x661e
Property type: PT_BINARY
Access: Read-only

Remarks

The schedule folder is a subfolder of a mailbox. It stores Schedule+ data and is not seen by the user. Schedule+ data is stored as a property on this nonstandard folder located under the root directory of a mailbox. This folder is not replicated.
