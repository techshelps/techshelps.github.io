HrMAPIWriteFileToStream

The HrMAPIWriteFileToStream function writes the contents of a file to a stream.

Quick Info

Header file: EDKMAPI.H
Library: EDKMAPI.LIB 

HRESULT HrMAPIWriteFileToStream(
  HANDLE hFile,      
  LPSTREAM lpStream  
);

Parameters

hFile
Input parameter. Handle to file.
lpStream
Output parameter. Points to an open Win32 IStream interface.

Return Values

See Return Values.

Remarks

For more information on this function, see the HrMAPIWriteFileToStream function in EDKMAPI.C under the EDKMAPI code sample in Code Samples folder.
