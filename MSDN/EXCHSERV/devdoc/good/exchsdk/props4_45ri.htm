PR_REPLICA_VERSION

The PR_REPLICA_VERSION property contains the version number of the installation of Microsoft Exchange Server.

Quick Info

Header file: EDKMDB.H
Applies to: Folder and message objects (information store objects)
Identifier: 0x664e
Property type: PT_I8
Access: Read-only

Remarks

When read on a private folder, the PR_REPLICA_VERSION property contains the version of the home server. When read on a public folder, it contains the version of the server on which folder replica resides.

The format for PR_REPLICA_VERSION is:

  • The HighPart member of the ULARGE_INTEGER structure contains the Microsoft Exchange major version in the high order word and the Microsoft Exchange minor version in the low order word.
  • The LowPart member of the ULARGE_INTEGER structure contains the Microsoft Exchange build number (integral part) in the high order word, and the build number (decimal part) in the low-order word.

For example:

To represent Microsoft Exchange 5.5, build 1635.09, the ULARGE_INTEGER of PR_REPLICA_VERSION would contain these values:

  • HighPart DWORD, high-order word contains: 5
  • HighPart DWORD, low-order word contains: 5
  • LowPart DWORD, high-order word contains: 1635
  • LowPart DWORD, low-order word contains: 09
