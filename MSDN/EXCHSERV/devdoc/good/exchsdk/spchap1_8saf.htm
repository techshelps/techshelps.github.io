Overview

Microsoft® Schedule+ enables users to schedule group meetings, manage their daily appointment schedules, track projects and tasks, maintain lists of contacts, and perform many other time- and resource-management tasks.

Using the Microsoft Schedule+ Programmer’s Reference and the Schedule+ Automation interface (formerly OLE Automation), you can extend Schedule+ functionality by writing stand-alone programs and by adding custom commands to Schedule+ menus.

This section covers the following topics:

  • Key concepts of writing programs using Automation, which provides the foundation for the Schedule+ Automation interface
  • Objects exposed by Automation
  • Types of operations that can be performed using these objects
  • Customizing Schedule+ menus
