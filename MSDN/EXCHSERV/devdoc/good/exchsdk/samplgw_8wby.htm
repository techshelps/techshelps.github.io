Sample Gateway Installation and Operation

The GWSETUP.MAK file in the \BKOFFICE\SAMPLES\EXCHANGE\GWSETUP directory creates a new GWSETUP directory, as well as the GWSETUP.INF file that is used for setting up the sample gateway.

Note Before you can install the sample gateway, you must build all the sample applications. See Building the Sample Applications for general information about the build process. If the sample applications do not exist in the corresponding \BKOFFICE\SAMPLES\EXCHANGE directories (not in the \BIN directory), INFLINE.EXE will fail when you attempt to install the gateway.
