BeforeEndDefaultTask

The BeforeEndDefaultTask property returns a Boolean value that indicates whether an alarm is set relative to the end rather than the start of a new task by default.

Applies To

Schedule object

Data Type

Boolean

Access

Read/write

Remarks

Typically, the BeforeEndDefaultTask property is set to False. The amount of time relative to the end of an item is set in the AlarmTypeUnit property.
