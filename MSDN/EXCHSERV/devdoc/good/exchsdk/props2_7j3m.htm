PR_EMS_AB_HELP_DATA32

The PR_EMS_AB_HELP_DATA32 property contains the binary Help information for the template for the 32-bit client.

Quick Info

Header file: EMSABTAG.H
Applies to: Display-Template objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x8010
Property type: PT_BINARY
Access: Read/write

Remarks

The PR_EMS_AB_HELP_DATA32 property is valid only when reading a property from the Microsoft Exchange Server address book. It is not valid when reading a property from an object copied into another address book provider.
