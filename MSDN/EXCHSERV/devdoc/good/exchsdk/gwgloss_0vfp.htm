E

embedded object
An object that is included in a second object. An object embedded in a Microsoft Exchange Server message and represented as an icon is called an attachment.
event logging
The recording to a file of any significant occurrences in the system or in an application.
EX
Address type for native Microsoft Exchange Server addresses.
extension data
Data with fields other than the standard configuration data provided by the Microsoft Exchange Server Administrator program, stored by a directory object. One extension data attribute can contain a number of extension data fields of varying data types. Extension data is commonly used when an application must expose information that is outside the application’s schema.
extension DLL
A DLL that controls the property sheets that display extension data. Also called Administrator extension DLL.
