PublishFreeBusy

The PublishFreeBusy property returns the number of months to publish free or busy information.

Applies To

Schedule object

Data Type

Long

Access

Read/write

Remarks

Only the owner can write to the PublishFreeBusy property. Free or busy information indicates when a user's appointment schedule is free or busy.

This property indicates the number of months that Schedule+ will make the user's free or busy information available to other users. For example, if this property is set to 3, other users will be able to view the free or busy information for that schedule starting with the current date and extending for the next three months.
