Gateway to Microsoft Exchange Server Report Transfer

The following steps are performed when the gateway transfers a report to Microsoft Exchange Server:

  1. Microsoft Exchange Server creates a new MTS report with and assigns it a report ID.
  2. Microsoft Exchange Server copies the report ID to the gateway report’s PR_MTS_REPORT_ID property for auditing purposes.
  3. Microsoft Exchange Server copies the gateway report’s PR_MTS_SUBJECT_ID to the MTS report’s subject ID, and the gateway report’s PR_PARENT_KEY to the MTS report’s PR_CONTENT_CORRELATOR property.
