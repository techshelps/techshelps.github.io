MBINST: Installing a Mailbox Agent

The MBINST sample application demonstrates how to install a Mailbox-Agent object in the Microsoft Exchange Server directory. To use it, Microsoft Exchange Server and the Microsoft Exchange Server SDK must both be active on the target computer. MBINST can be used to install the mailbox agent (MBA) created by the SMBAGENT sample application.

    To build MBINST
  1. Change to the directory containing the MBINST source files:

    \BKOFFICE\SAMPLES\EXCHANGE\EDKINSTU

  2. Type NMAKE.
