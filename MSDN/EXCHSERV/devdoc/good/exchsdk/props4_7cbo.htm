PR_GW_MTSIN_ENTRYID

The PR_GW_MTSIN_ENTRYID property contains the entry identifier for a gateway's MTS-IN folder.

Quick Info

Header file: EDKMDB.H
Applies to: Gateway message store objects (information store objects)
Identifier: 0x6628
Property type: PT_BINARY
Access: Read-only

Remarks

The MTS-IN folder is used to transfer messages from a gateway to the Microsoft Exchange Server information store.
