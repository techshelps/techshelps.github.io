RoleItemId

The RoleItemId property returns an item identifier for the Role object.

Applies To

Role object

Data Type

String

Access

Read/write

Remarks

The RoleItemId property is reserved for use by the Schedule+ Covey Seven Habits Tools. This property links a task to a role. All tasks belonging to a given role have the same RoleItemId property value.
