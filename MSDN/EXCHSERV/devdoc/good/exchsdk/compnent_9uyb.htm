About Personal Folders

The personal folder store is part of the Microsoft Exchange Client. This information store resides at the client computer and is used to store items that are owned by individuals and not shared by multiple users. The following diagram shows the personal folder store architecture.

Personal folder store architecture

A user’s personal folders are found on the client computer in a file with the extension .PST. Only folders below the Top of Personal Folders folder are visible. The other folders below the root (such as IPM_VIEWS) are not visible.
