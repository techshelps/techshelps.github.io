O

object
A set of information that consists of properties and functions used to manipulate the properties, collected into an interface (usually a DLL). Windows objects can be linked to or embedded into a document. They are created by using a Windows-based application with OLE capabilities.
one-off address
An address of any type that does not appear in a Microsoft Exchange Server address book. One-off addresses are either constructed once and not saved, or are saved in a user’s personal address book, inaccessible to Microsoft Exchange Server.
organization
A collection of servers, grouped into sites, that are linked together to provide workgroup connectivity, information, and messaging services. The organization is the largest administrative unit in a Microsoft Exchange Server system. It encompasses all the Microsoft Exchange Server computers in a company’s messaging infrastructure.
