Editing Extension Data with EXTDATVW

The following options are available on the Ext Dat Viewer property page.

Extension Data Text Box

Displays the data types and values of the data fields within the currently open value in the multivalued Extension-Data property, which are displayed in the text box.

Refresh Button

Displays the saved values of the fields shown in the Extension Data text box. EXTDATVW will display a dialog box asking whether to save changes first if the values displayed have not yet been saved. To save changes, click Yes, then click Apply.

Add Button

Adds new fields to the value displayed in the Extension Data text box. Select a data field in the Extension Data text box, then click Add. This adds a new data field in the position immediately before the selected field. Click Apply to save the new value.

Modify Button

Modifies a data field. Click the Modify button to edit the value of that data field. This option displays the Edit Value dialog box. Type the new value for the data field in the dialog box. After changing the value, click Apply to save changes.

Delete Button

Deletes a data field. First select the data field, then click the Delete button. Click Apply to execute the delete operation.
