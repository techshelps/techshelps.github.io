PR_EMS_AB_ADMIN_EXTENSION_DLL

The PR_EMS_AB_ADMIN_EXTENSION_DLL property contains the name of the Administrator extension DLL associated with an Admin-Extension object.

Quick Info

Header file: EMSABTAG.H
Applies to: Admin-Extension objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x804c
Property type: PR_EMS_AB_ADMIN_EXTENSION_DLL PT_TSTRING

PR_EMS_AB_ADMIN_EXTENSION_DLL_A PT_STRING8

PR_EMS_AB_ADMIN_EXTENSION_DLL_W PT_UNICODE
Access: Read/write
Unicode: Yes

Values

1
Minimum length
255
Maximum length

Remarks

The PR_EMS_AB_ADMIN_EXTENSION_DLL property holds the name of the DLL that adds one or more property pages to any object. This is the name of the DLL without the path, which is automatically prepended as:

\\<server>\Add-ins\<extension-name>\<machine>

For example,

\\davidand1\Add-ins\SMTP\I386\IMCADMIN.DLL
