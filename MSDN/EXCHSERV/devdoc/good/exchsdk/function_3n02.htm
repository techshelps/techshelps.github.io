GetGWMtsOutFolder

The GetGWMtsOutFolder function retrieves the outbound message folder, MTS-OUT.

Quick Info

Header file: GWMAIN.H
Library: GWMAIN.LIB 

LPMAPIFOLDER GetGWMtsOutFolder(
  VOID  
);

Parameters

None.

Return Values

A pointer to a MAPI IMAPIFolder interface containing the outbound message folder, MTS-OUT.

Remarks

For information on the IMAPIFolder interface, see the MAPI Programmer's Reference.
