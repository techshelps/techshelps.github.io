PR_EMS_AB_FILE_VERSION

The PR_EMS_AB_FILE_VERSION property contains the file version stamp for DLLs used by the object.

Quick Info

Header file: EMSABTAG.H
Applies to: Addr-Type, Admin-Extension, and DSA objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x80ac
Property type: PT_BINARY
Access: Read/write

Values

1 (2 hex digits)
Minimum length
8 (16 hex digits)
Maximum length

Remarks

The PR_EMS_AB_FILE_VERSION property contains a 64-bit file version identifier in Windows FILEVERSION format. Objects associated with DLLs use the PR_EMS_AB_FILE_VERSION property to check that the correct version of a DLL is being used.
