Microsoft Exchange Server Information Store Properties

Features that are specific to Microsoft Exchange Server information stores are accessed as properties and interfaces, and are defined in EDKMDB.H. Most properties on message store objects are read-only.

The information store properties declared by Microsoft Exchange Server in the EDKMDB.H file are listed in the following table. This table also indicates whether a function call to retrieve the value of a given property causes an RPC. For more information, see Reducing the Number of RPCs.
