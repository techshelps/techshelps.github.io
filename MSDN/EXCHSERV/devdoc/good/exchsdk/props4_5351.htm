PR_MEMBER_NAME

The PR_MEMBER_NAME property contains the display name of a member of the ACL table.

Quick Info

Header file: EDKMDB.H
Applies to: Public folder objects (information store objects)
Identifier: 0x6672
Property type: PT_TSTRING
Access: Read-only

Remarks

The PR_MEMBER_NAME property is used by IExchangeModifyTable interface to display the name of a member of an ACL table, which is a person or role with explicit rights on a folder or mailbox.
