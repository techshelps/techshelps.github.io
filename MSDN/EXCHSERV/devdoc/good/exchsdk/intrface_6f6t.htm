IExchangeManageStore

The IExchangeManageStore interface supports managing the folders and mailboxes in an information store.

Remarks

To get a pointer to this interface, call the QueryInterface method of the message store object. For additional information on this interface, see Logging On to a Mailbox Using the IExchangeManageStore Interface.
