HrMAPIGetPropLong

The HrMAPIGetPropLong function obtains the long value of a property from a property tag.

Quick Info

Header file: EDKMAPI.H
Library: EDKMAPI.LIB 

HRESULT HrMAPIGetPropLong(
  LPMAPIPROP lpObj, 
  ULONG ulPropTag,  
  ULONG * lpulProp  
);

Parameters

lpObj
Input parameter. Points to a MAPI IMAPIProp interface containing the object.
ulPropTag
Input parameter. Property tag.
lpulProp
Output parameter. Points to property variable.

Return Values

See Return Values.

Remarks

Use this function when the property type is PT_LONG.

For more information on this function, see the HrMAPIGetPropLong function in IPROP.C under the EDKMAPI code sample in the Code Samples folder.

For information on the IMAPIProp interface, see the MAPI Programmer's Reference.
