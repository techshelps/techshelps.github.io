PR_CONFLICT_ENTRYID

The PR_CONFLICT_ENTRYID property is the entry identifier of an original conflict message that contains conflicting versions of the message as attachments.

Quick Info

Header file: EDKMDB.H
Applies to: Notification message objects (information store objects)
Identifier: 0x3ff0
Property type: PT_BINARY
Access: Read/write

Remarks

Handling Message Conflicts.
