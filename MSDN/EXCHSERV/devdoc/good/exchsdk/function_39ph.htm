DAPIUninitialize

The DAPIUninitialize function terminates background processes.

Quick Info

Header file: DAPI.H
Library: DAPI.LIB 

VOID DAPIUnitialize(
  DWORD dwFlags  
);

Parameters

dwFlags
Unused. Must be 0 (zero).

Return Values

None.

Remarks

The SchemaPreload function is an example of a process terminated by DAPIUninitialize.
