PR_EMS_AB_GATEWAY_PROXY

The PR_EMS_AB_GATEWAY_PROXY property contains a list of proxy addresses that is unique for a domain.

Quick Info

Header file: EMSABTAG.H
Applies to: Site-Addressing objects (directory objects). These objects appear to MAPI as messaging user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x80b2
Property type: PR_EMS_AB_GATEWAY_PROXY PT_MV_TSTRING

PR_EMS_AB_GATEWAY_PROXY_A PT_MV_STRING8

PR_EMS_AB_GATEWAY_PROXY_W PT_MV_UNICODE
Access: Read/write
Unicode: Yes
