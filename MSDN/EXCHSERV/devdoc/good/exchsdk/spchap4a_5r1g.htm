BirthdayItemId

The BirthdayItemId property returns an identifier for the birthday of a contact in the RecurringEvents table.

Applies To

Contact object

Data Type

String

Access

Read/write

Remarks

The BirthdayItemId property is used to link a Contact object to a RecurringEvent object. In this case, the recurring event is the birthday of the contact. This is the ItemId property in a RecurringEvents table.
