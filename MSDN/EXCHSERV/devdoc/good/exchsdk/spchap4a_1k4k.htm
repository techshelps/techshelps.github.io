DelegateEntryId

The DelegateEntryId property returns a list of mail entry identifiers for delegates.

Applies To

Schedule object

Data Type

String (multivalued)

Access

Read/write

Remarks

Only the owner can write to the DelegateEntryId property — everyone else has read-only access. This property is used to assign delegates for the schedule’s owner. Because the property is multivalued, you can set it to any number of delegate entry identifiers. When a meeting request is received, each delegate in this property will receive meeting requests on behalf of the schedule’s owner.

This property must reflect the delegates specified in the AccessControls table. The values of this property should only be changed by the Schedule+ user interface.
