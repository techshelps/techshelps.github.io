Performing Preliminary System Configuration

Before you install a Microsoft Exchange Server gateway:

  • Microsoft Exchange Server must be installed and running.
  • The Microsoft Exchange Server Administrator program must be installed on the computer on which the gateway will run.

The computer from which you install the gateway does not have to be the Microsoft Exchange Server computer. This means that you can install a gateway remotely.

If your gateway is designed to operate as a service of Windows NT Server, an administrator needs to set up a Windows NT Server account with the “Log on as a service” right for the gateway before running its setup program. The documentation for your gateway’s setup procedure should make these prerequisites clear.
