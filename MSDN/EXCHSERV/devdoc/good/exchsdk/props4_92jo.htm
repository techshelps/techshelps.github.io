PR_EXCESS_STORAGE_USED

The PR_EXCESS_STORAGE_USED property contains the amount of storage actually used by a user who has exceeded the storage quota.

Quick Info

Header file: EDKMDB.H
Applies to: Message objects (information store objects)
Identifier: 0x3ff6
Property type: PT_LONG
Access: Read-only
