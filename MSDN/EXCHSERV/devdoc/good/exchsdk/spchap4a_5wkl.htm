DelegateName

The DelegateName property returns the display names of delegates listed in the DelegateEntryId property.

Applies To

Schedule object

Data Type

String (multivalued)

Access

Read/write

Remarks

Only the owner can write to the DelegateName property — everyone else has read-only access. Use this multivalued property to set the display names of delegates that have been set in the DelegateEntryId property. The order of the display names must correspond to the order of the entry identifiers in the DelegateEntryId property.

This property must reflect the delegates specified in the AccessControls table. The values of this property should only be changed by the Schedule+ user interface.
