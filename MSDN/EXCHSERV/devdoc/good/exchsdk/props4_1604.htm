PR_USER_ENTRYID

The PR_USER_ENTRYID property contains the entry identifier of a user.

Quick Info

Header file: EDKMDB.H
Applies to: Message store objects (information store objects)
Identifier: 0x6619
Property type: PT_BINARY
Access: Read-only

Remarks

The PR_USER_ENTRYID property contains the user's entry identifier, which is not always the same as the owner's entry identifier.
