PR_EMS_AB_MDB_BACKOFF_INTERVAL

The PR_EMS_AB_MDB_BACKOFF_INTERVAL property contains the number of minutes the information store waits before resending a message to the gateway.

Quick Info

Header file: EMSABTAG.H
Applies to: Mail-Gateway objects (directory objects). These objects appear to MAPI as messaging_user objects, which are accessible through the MAPI IMailUser interface.
Identifier: 0x80cf
Property type: PT_LONG
Access: Read/write

Remarks

When the number of unread messages in a gateway's mailbox folders (MTS-IN, MTS-OUT, etc) reaches the limit set by PR_EMS_AB_MDB_UNREAD_LIMIT, the information store stops the transfer of messages from the MTA to the gateway for the length of time specified by the PR_EMS_AB_MDB_BACKOFF_INTERVAL property.
