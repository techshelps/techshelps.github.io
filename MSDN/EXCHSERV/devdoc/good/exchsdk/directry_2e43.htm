Specifying Import Destinations

Objects are imported into containers in the Microsoft Exchange Server directory. The following table lists typical import destinations for objects most commonly imported using the directory access functions. Because more than one container of a given type may exist in a given directory information tree (DIT), specify the correct destination in the parameters to your function call.

Import Destinations for Objects Imported with Directory Access Functions

Object Import to
Mailbox Recipients container
Distribution List Recipients container
Custom Recipient Recipients container
Gateway Connections container
Mailbox Agent Recipients container
Address Type Address Types container
Address Template One-off Address Templates container

For more information about DIT structure, see About the Directory Information Tree.
