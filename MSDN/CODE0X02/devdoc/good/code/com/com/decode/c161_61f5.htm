SAMPLE:       DECODE: Error Code Decoder Tool 
 
DECODE is a decoder of 32 bit OLE error codes (HRESULTs) 
and Windows error codes. Enter OLE error codes in hexadecimal 
and Windows error codes in decimal. 
 
This tool may not be able to decode errors added after the shipping 
of the Win32 SDK in which it is included.
