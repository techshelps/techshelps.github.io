Using ActiveX Controls

This section provides an introduction and scenarios to help you get started with the ActiveX controls — formerly called OLE controls — included with the Microsoft Visual Basic Professional Edition. Start here for general information and for suggestions and code to use when developing your applications. More detailed reference information, including links to properties, methods, and events associated with each control, is included in the Language Reference.
