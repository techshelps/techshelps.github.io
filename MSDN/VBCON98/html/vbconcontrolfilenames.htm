ActiveX Controls File Names

To load an ActiveX control into the Toolbox, use the Components dialog box from the Project menu. The table below lists the names of components you will find in the dialog box along with the file name (.ocx) and the ActiveX controls contained by the component.

Note The presence or absence of components on your system depends upon which version of Visual Basic you are using.
