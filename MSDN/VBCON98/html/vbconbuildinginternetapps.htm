Building Internet Applications

Visual Basic offers several possibilities for how you can move your applications onto the Internet or a corporate intranet.

Chapters

Introduction to Internet Applications

Introduces the developmental approach for creating applications for the Internet using Dynamic HTML and Internet Information Server technology.
Developing DHTML Applications

Explains how to create, test, and deploy client-based Internet applications using Dynamic HTML.
Developing IIS Applications

Explains how to create, test, and deploy server-based Internet applications using Internet Information Server.
Downloading ActiveX Components

Explains how to package and deploy ActiveX controls, components, and user documents on the Web, through Internet Component Download.
