Active Directory Schema

[This is preliminary documentation and subject to change.]

The Microsoft® Windows NT® 5.0 Active Directory schema defines all directory service objects. Types of objects are classes and their properties are attributes. Classes are defined by Class-Schema objects and attributes are defined by Attribute-Schema objects.

  • Directory Service Terminology Useful directory service terms and their definitions.
  • Schema Class Table A table of all the schema class objects.
  • Reading a Schema Class Entry A guide to understanding the information available in each schema class entry.
  • Schema Classes The alphabetical list of all the classes in the schema and their definitions.
  • Schema Attribute Table A table of all the schema attribute objects.
  • Reading a Schema Attribute Entry A guide to understanding the information available in each schema attribute entry.
  • Schema Attributes The alphabetical list of all the attributes in the schema and their definitions.
