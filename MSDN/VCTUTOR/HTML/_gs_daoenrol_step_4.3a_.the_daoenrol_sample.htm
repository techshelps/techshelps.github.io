DaoEnrol Step 4: The DAOENROL Sample

The sample DAOENROL contains code that is very similar to the sample ENROLL.

Step 4 of the DAOEnrol tutorial is an exercise in reading and comparing source code. To complete Step 4 of the DaoEnrol tutorial:

  • Jump to the DAOENROL abstract (above) and, if you have not done so, copy the sample files to your hard drive.

  • Jump to the ENROLL sample (above) and, if you have not done so, click the jump STEP4 and copy the sample files to your hard drive.

  • Alternately open and examine the source code files for the two samples. Study the how the files vary to understand some of the differences in the implementation of the ODBC and DAO database classes.
