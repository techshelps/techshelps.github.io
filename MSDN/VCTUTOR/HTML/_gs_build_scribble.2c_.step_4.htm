Build Scribble, Step 4

How does Scribble behave with these new enhancements? Build the new version and find out.

To build Scribble

  1. From the Build menu, click Execute Scribble.exe.

  2. Click Yes when prompted to build new versions of files.

Draw some strokes, scroll to a new portion of the drawing, and draw some more strokes. Resize the window and scroll back and forth. Double-click the split box, or drag it to split the window into two panes.
