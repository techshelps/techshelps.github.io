Registering an OLE Component Application

Before you can preview the Container application (by running either the Step 2 executable source file, Contain.exe, or the version you’ll develop yourself by following the lesson steps), you need to install and register at least one component application for Container to access.

The Step 7 version of Scribble is a good sample application. Another example is the HIERSVR sample. Abstracts for both of these samples can be found in Visual C++ Samples under \ MFC Samples \ Tutorial Samples.

To register the Scribble server application

  • Run Scribble.exe Step 7 directly; or build and run the sample. Close the sample.

    Installing the Sample Files in Scribble Lesson 1 describes how to copy sample files and how to run a sample directly. Running Scribble briefly as a stand-alone application registers it in the system registry as an OLE server.
