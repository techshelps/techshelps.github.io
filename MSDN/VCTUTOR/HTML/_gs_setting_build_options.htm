Setting Build Options

When you create a new project, both Release and Debug build configurations are automatically created, although only one project configuration (the current default) is displayed in the Project Workspace window. Debug is set as the active project configuration. For Scribble, you shouldn’t need to change this default setting. However, for your own projects you’ll want to be able to build a Release version.

To change the default, display the Build toolbar. From the Default Configuration drop-down list on the Build toolbar, select the type of project you want to build. The project you select remains the active project until you change it again.
