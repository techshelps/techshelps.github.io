Build Scribble, Step 5

What does Scribble’s printing look like now? Build the new version of Scribble and find out.

To build Scribble

  1. From the Build menu, click Execute Scribble.exe.

  2. When prompted to build the files, click Yes.

Run the new version of Scribble. Draw some strokes, and then click Print Preview from the File menu. Switch back and forth between one-page and two-page display mode, or move to the previous or next page. The following figure shows this version of Scribble.

Scribble Version 5

Exit Scribble.

This completes Step 5 of the Scribble tutorial.
