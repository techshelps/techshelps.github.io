Previewing the Sample Applications

You can use the online documentation to preview sample applications, including tutorial samples, in their executable state. For example, if you’re simply reading through the Scribble tutorial without adding code, you can still run Scribble at each step to see what it looks like and how it behaves. You do not need to install any sample files to preview sample applications.

To run a sample application from the source files, follow the steps described in Installing the Sample Files earlier in this lesson. At step 4, follow the instructions to run the sample application.
