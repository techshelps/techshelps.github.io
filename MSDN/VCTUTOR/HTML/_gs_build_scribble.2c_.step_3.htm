Build Scribble, Step 3

Build the new version of Scribble and examine how Scribble behaves with a dialog box.

To build and test Scribble Step 3 Version

  1. From the Build menu, click Build Scribble.exe.

    When prompted to build the files, click Yes.

  2. Run the new version of Scribble. Draw some strokes with the default thick pen and the default thin pen. Then use the Pen Widths dialog box to change the thickness of the pens and draw some new strokes. The following figure illustrates the third version of Scribble with a variety of strokes drawn.

    Scribble Version 3

  3. Exit Scribble.

This completes Step 3 of the tutorial.
