Editing Code

You’ll continue to use the navigational support provided by ClassView and WizardBar as you edit your project source code. The integrated text editor provides full-featured editor support, including the following editor features:

  • Emulate another editor

  • Incremental searching

  • Automatic indentation and separate tabs and indent sizes

  • Full-screen editing mode

  • File navigation
