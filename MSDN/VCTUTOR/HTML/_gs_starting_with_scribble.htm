Starting with Scribble

The starter files AppWizard creates contain starter code for several C++ classes:

  • An application class

  • A document class

  • A view class

  • A frame window class

Details about these files are available in a text file (ReadMe.txt) that is created along with the starter files.

Without adding a line of code, you can build the starter application you created with AppWizard and run the resulting program, which exhibits much of the standard functionality you expect from a program written for Windows. See Building the Starter Application and Running the Starter Application, later in this lesson.

If you’re working along, adding code as you read, follow all directions in this lesson. When you finish, you’ll have a full set of starter files in your own project directory. If you’re reading along without adding any code, work through this lesson to familiarize yourself with AppWizard.
