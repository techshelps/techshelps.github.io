About Step 1

Step 1 teaches the basics of:

  • Using AppWizard to create an application with database support.

  • Using ClassWizard, ClassView, and the resource editors to bind controls on a form to data.

  • Using recordsets.

  • Using record views.

In the next topic, Understanding Recordsets and Record Views, you will learn the difference between recordsets and record views, and their respective roles in representing and manipulating the view to the data in a database. You will also get an introduction to the MFC classes that access database data and views.
