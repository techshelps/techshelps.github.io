Creating the Circle Control

The Circle sample tutorial uses the Circle control to explore the features and functions provided by the ActiveX control classes. Each lesson in this tutorial represents one step in the development cycle.

Note   You can find a finished example of this lesson's code in the CIRC1 sample source code directory.

The Circle control created at the beginning of this tutorial is a simple one: it meets minimum control requirements and provides only default behaviors.

In this lesson, you will preview the Circle control and learn how to:

  • Generate the Circle control template using ControlWizard.

  • Modify the default control bitmap.

  • Modify the default About Circ Control dialog box.

  • Build the control.

  • Test the control using Test Container.
