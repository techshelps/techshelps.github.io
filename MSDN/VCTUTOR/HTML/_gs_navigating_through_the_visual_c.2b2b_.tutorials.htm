Navigating Through the Visual C++ Tutorials

The tutorial lessons include two kinds of links: jumps to other parts of the online documentation set and jumps to topics in the tutorial lessons. The jumps to other parts of the online documentation are grouped into suggested reading lists in the lessons. These topics will introduce you to concepts that are necessary to perform the lesson. The jumps will take you to topics in one of the following information titles:

  • Visual C++ Programmer’s Guide   Information on using Visual C++ and the MFC classes.

  • Microsoft Foundation Class Reference   Reference information on the MFC classes and their member functions and data members.

  • Visual C++ User’s Guide   Information about using the development environment, including the resource editors and wizards.

  • Win32 SDK   Reference and conceptual information on the various application programming interfaces (APIs) included in the Win32 Software Development Kit.

If you click a jump going to one of these information titles and it isn't installed, you will be prompted to install it. Note that the jump identifies the name of the information title the topic is in. Jumps within the tutorial information title appear in the text of the lessons and do not list the name of the information title.
