The Tutorial Example: Container

The Container tutorial consists of two steps. The individual steps can be found under Samples \ MFC Samples \ Tutorial Samples \ Container in the online documentation or in the \SAMPLES subdirectory on the Visual C++ CD-ROM. The sample source files contain a subdirectory for each step: STEP1 and STEP2. Each step’s subdirectory contains complete source files and other files needed.

This lesson covers Step 1 of the OLE container tutorial. You will learn how to:

  • Preview the container application.

  • Create a starter OLE container application capable of visual editing.

  • Interpret the container code provided by AppWizard.

  • Coordinate the size of an embedded object within the server.

  • Add hit testing and selection to the AppWizard-created container application.

  • Implement activation of an embedded object.

  • Implement tracker rectangles for resizing and moving items.

  • Draw embedded and linked items.

  • Delete embedded items.

In Step 2, you will learn how to:

  • Implement the Copy and Paste commands on the Edit menu.

  • Implement smart invalidation.

  • Improve coordination with the server to determine the size of contained objects
