Use the Prebuilt StdReg32.mdb

If you are using Microsoft Access, follow these steps:

To use StdReg32.mdb with DaoEnrol

  1. Open the sample abstract topic for the STDREG sample.

  2. In the topic, click the link provided to copy sample project files.

  3. In the Sample Application dialog box, select StdReg32.mdb and click Copy.

  4. In the Copy dialog box, navigate to the directory where you want to copy the file and click OK.

    The development environment creates the directory for you, if necessary, and copies the file. StdReg32.mdb already contains the tables and records used in the tutorial.
