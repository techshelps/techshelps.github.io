The CSectionForm Record View Class

To examine the source code for class CSectionForm

  • Use ClassView to jump to the OnInitialUpdate member function of class CSectionForm.

    SectionForm.cpp opens inside the text editor, with your cursor just before the OnInitialUpdate member function:

    void CSectionForm::OnInitialUpdate()
{
m_pSet = &GetDocument()->m_sectionSet;
CRecordView::OnInitialUpdate();
}

    The base class function CRecordView::OnInitialUpdate opens the database if not already open, opens the recordset.
