job object
An object that allows groups of processes to be managed as a unit. Job objects are namable, securable, sharable objects that control attributes of the processes associated with them. Operations performed on the job object affect all processes associated with the job object. See also process.
join style
A geometric pen attribute that specifies how the ends of connected lines are joined. See also geometric pen.
junction point
A directory in a Distributed File System (Dfs) tree structure that points to, or references, a different Net (formerly LAN Manager) share. A junction point that has one or more shares for its storage has alternate paths. See also Distributed File System (Dfs), leaf volume, root volume, alternate path.
