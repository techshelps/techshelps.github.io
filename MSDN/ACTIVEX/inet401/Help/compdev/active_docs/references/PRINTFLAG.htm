PRINTFLAG

typedef enum  
{ 
    PRINTFLAG_MAYBOTHERUSER         =  1, 
    PRINTFLAG_PROMPTUSER            =  2, 
    PRINTFLAG_USERMAYCHANGEPRINTER  =  4, 
    PRINTFLAG_RECOMPOSETODEVICE     =  8, 
    PRINTFLAG_DONTACTUALLYPRINT     = 16, 
    PRINTFLAG_FORCEPROPERTIES       = 32, 
    PRINTFLAG_PRINTTOFILE           = 64  
} PRINTFLAG;

Specifies printing options. A combination of values from PRINTFLAG is passed in the grfFlags parameter of IPrint::Print.

PRINTFLAG_MAYBOTHERUSER
User interaction is permitted. If this flag is not set, no part of the printing process can interact with the user.
PRINTFLAG_PROMPTUSER
Object should prompt the user for job-specific printing options, using the normal print dialog for the object. Support for this option is required. Valid only if PRINTFLAG_MAYBOTHERUSER is set.
PRINTFLAG_USERMAYCHANGEPRINTER
User can change the target printer. If this flag is not set, the user can print only on the default printer. Valid only if PRINTFLAG_PROMPTUSER is set.
PRINTFLAG_RECOMPOSETODEVICE
Object should attempt to recompose itself to the indicated target device. If this flag is not set, the object should, if possible, retain any association that it already has with a compositional device.
PRINTFLAG_DONTACTUALLYPRINT
Object should carry out actions resulting from user actions or object recomposition, but it should not actually carry out the printing operation.
PRINTFLAG_FORCEPROPERTIES
Object should always display the print properties dialog box.
PRINTFLAG_PRINTTOFILE
Object should print to the file that is named in the portname member of the DVTARGETDEVICE structure.
