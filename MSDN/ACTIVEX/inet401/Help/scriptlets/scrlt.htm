Welcome to Scriptlet Technology

Microsoft® Scripting Components (scriptlets) provide you with an easy but powerful way to create controls from Web pages (.htm files) that use Microsoft Dynamic HTML. You can use scriptlets as ordinary controls in applications such as Microsoft Visual Basic, Microsoft Internet Explorer 4.0, and Microsoft Visual InterDev.
