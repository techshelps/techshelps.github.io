NdisAllocateBuffer

NdisAllocateBuffer creates a buffer descriptor mapping a specified virtual (sub)range within an already allocated, nonpaged memory block.

Syntax

VOID NdisAllocateBuffer(OUT PNDIS_STATUS Status,
OUT PNDIS_BUFFER *Buffer, IN NDIS_HANDLE PoolHandle,
IN PVOID VirtualAddress, IN UINT Length);
