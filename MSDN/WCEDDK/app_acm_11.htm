ACMDRV_MESSAGE_PARAMS

The ACMDRV_MESSAGE_PARAMS structure is for parameters passed to the ACM_IOControl function.

Syntax

typedef struct { DWORD dwDriverID; HANDLE hDriver; UINT uMsg; LPARAM lParam1; LPARAM lParam2; } ACMDRV_MESSAGE_PARAMS;

Members

dwDriverID
Instance identifier. This value is message-dependent.
hDriver
Driver handle.
uMsg
One of the ACM Driver Messages.
lParam1
The first message parameter. Dependent on message type.
lParam2
The second message parameter. Dependent on message type.

Remarks

The ACMDRV_MESSAGE_PARAMS structure is declared in the header file Wavedev.h.
