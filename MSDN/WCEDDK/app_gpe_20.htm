GPE::SetPointerShape

This function sets the shape of the pointer, the hot spot of the pointer, and (if it is multicolored) the colors to use for the cursor. GPE::MovePointer is called separately to move and/or hide the cursor.

Syntax

SCODE GPE::SetPointerShape( GPESurf *pMask, GPESurf *pColorSurf, int xHot, int yHot, int cx, int cy )

pMask points to a one bit per pixel surface with a width of cx and a height of (2 x cy). The top half defines the AND mask for the pointer while the lower half defines the XOR mask. Taken together, these masks provide two bits of information for each pixel of the pointer image. The following truth table determines what should be drawn on the display for different values in the AND and XOR masks:

AND mask value XOR mask value Result displayed on screen
0 0 Black.
0 1 White.
1 0 Transparent, pixel is unchanged
1 1 Inverts the pixel color

This supplies a black and white image while allowing for transparency and inversion of the pixels that make up the pointer.

If pColorSurf is not null, then the colors for the corresponding pixels of the color surface should be used instead of the XOR mask value for pixels where the AND mask value is 0.
