Windows CE Device Driver Reference

This section provides detailed information on the functions, structures, constants, and registry keys related to Windows CE device drivers:

Stream I/O Functions and Required Entry Points Functions and entry points for installable device driver DLLs.
PC Card Reference Functions that can be used to write installable device drivers for PC Card devices.
NDIS Network Driver Functions Functions that can be used by Miniport drivers.
Registry Keys Related to Windows CE Device Drivers Registry keys for installable device drivers, Miniport drivers, and printer drivers.
