Accelerated line segment function

Note that if an accelerated line draw function is implemented, it is obliged to draw lines with sub-pixel precision (the emulated function provided also does this).

The function whose address was placed in pLineParms->pLine is called once for each line segment with the start pixel on the destination surface defined by pLineParms->xStart and pLineParms->yStart.

The bresenham parameters for the line are passed in pLineParms->dM and pLineParms->dN, and the actual number of pixels to draw is indicated in pLineParms->cPels.
