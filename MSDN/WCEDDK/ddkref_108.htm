NDIS_BUFFER_TO_SPAN_PAGES

NDIS_BUFFER_TO_SPAN_PAGES determines how many physical pages of memory are being used to back a given buffer.

Syntax

ULONG NDIS_BUFFER_TO_SPAN_PAGES(IN PNDIS_BUFFER Buffer);

At a Glance

Header File: Ndis.h
Module: Nk
Platforms:
Versions: 2.0 and later
Complete Documentation: Windows NT Device Driver Kit

Remarks

The Windows CE version of NDIS_BUFFER_TO_SPAN_PAGES always returns 1.
