NdisResetComplete

NdisResetComplete forwards a reset request to an underlying driver.

Syntax

VOID NdisReset(OUT PNDIS_STATUS Status,
IN NDIS_HANDLE NdisBindingHandle);

At a Glance

Header File: Ndis.h
Module: Nk
Platforms:
Versions: 2.0 and later
Complete Documentation: Windows NT Device Driver Kit

Remarks

This function exists in Windows CE 2.0, but has not been enabled. It should not be used.
