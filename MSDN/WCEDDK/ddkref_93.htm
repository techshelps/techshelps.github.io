PC Card Window Attributes

Window attribute masks define memory or I/O window configurations. The following window attribute masks are defined:

WIN_ATTR_IO_SPACE
1 = I/O window, 0 = memory window
WIN_ATTR_ATTRIBUTE
1 = attribute, 0 = common
WIN_ATTR_ENABLED
1 = window enabled, 0 = window disabled
WIN_ATTR_16BIT
1 = 16 bit, 0 = 8 bit
WIN_ATTR_PAGED
1 = size is 16K multiple
WIN_ATTR_SHARED
1 = shared memory request
WIN_ATTR_FIRST_SHARED
1 = first shared request
WIN_ATTR_OFFSETS_SIZED
Card offsets window sized.
WIN_ATTR_ACCESS_SPEED_VALID
Causes window's access speed change.
