Creating Your Own Users and Groups

Now that you have a database containing your secured objects, create the users and groups that your application needs. The easiest way to do so is through the Microsoft Access User And Group Accounts dialog box, available from the Security submenu on the Tools menu.

Note As mentioned earlier, it’s best to create groups and then assign permissions to the groups rather than to individual users. This way, administering your workgroup becomes simply a matter of assigning users to the appropriate groups.
