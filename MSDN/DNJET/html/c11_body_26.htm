DAO SDK Files

The DAO 3.5 SDK is included in Visual C++ versions 4.2 and 5.0. Attempting to build new SDK libraries with older versions of MFC will result in a compile error.

The DAO SDK Setup program does everything required to start programming with dbDAO, including copying all necessary DLLs, updating the Registry, and setting up a directory structure for the header files, libraries, and sample programs.
