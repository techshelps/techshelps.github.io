The Microsoft Jet Multiuser Model

In a file-server environment, a multiuser database system using Microsoft Jet consists of the following components:

  • One or more database files residing on the network server

  • A locking information file (.ldb) for each database

  • A workgroup information file residing either on the network server or on each user’s workstation

  • A copy of Microsoft Jet running on the local workstation(s)
