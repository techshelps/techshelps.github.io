Driver Management Components

Microsoft Jet can access data through the ODBC standard. While the Microsoft Jet components contain no intrinsic ODBC services, they do call the standard ODBC Driver Manager. The ODBC32.dll dispatches calls to ODBC drivers for processing. This file is not a Microsoft Jet component.
