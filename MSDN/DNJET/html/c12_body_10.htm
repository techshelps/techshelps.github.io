Microsoft Jet Files

The following Microsoft Jet files, discussed in Appendix E, “Microsoft Jet Components,” are used by each Desktop Database Driver.

File Description
MSJINT35.DLL Contains all localized Microsoft Jet strings.
MSJET35.DLL Contains the main engine functionality.
MSJTER35.DLL Contains the Microsoft Jet error service.
VBAJET32.DLL Initializes the Visual Basic for Applications (VBA) expression service for Microsoft Jet.
VBAR332.DLL A run-time version of the VBA expression service.
