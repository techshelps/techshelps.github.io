Activating the ShowPlan Function

To activate the ShowPlan function, add the following key to the registry: \\HKEY_LOCAL_MACHINE\SOFTWARE\MICROSOFT\Jet\3.5\Engines\Debug. Under this key, add a setting called JETSHOWPLAN that has the String data type, making sure you use all capital letters in the name of the registry setting. To activate the ShowPlan function, set the value of the new registry setting to ON. To turn it off, set the value to OFF. Note that after you create this registry setting, you must re-initialize Microsoft Jet.
