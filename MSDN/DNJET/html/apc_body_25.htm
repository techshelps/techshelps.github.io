Lotus 1-2-3 ISAM Formats Settings

The following tables give the settings and values that Microsoft Jet 3.5 installs for the Lotus 1-2-3 formats. These settings are located in the \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Jet\3.5\ISAM Formats\Lotus version key of the Windows registry, where version is WK1, WK3, or WK4.

Note The WizardProject and ImportWizardFunction settings listed in the following tables are added by Microsoft Access.
