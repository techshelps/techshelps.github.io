Security

Microsoft Jet provides a sophisticated security model with which you can define your application’s users and groups, and set specific permissions for each object in the database for these users and groups. With Microsoft Jet, you can create workgroups to define users, groups, and passwords in a secure environment. This function is supplied in a stand-alone executable called Workgroup Administrator (WrkgAdm.exe), which is included with Microsoft Access.
