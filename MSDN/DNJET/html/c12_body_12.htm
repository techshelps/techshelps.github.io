ODBC Files

The Desktop Database Drivers include the ODBC Core components.

File Description
ODBC32.DLL ODBC Driver Manager
ODBCAD32.EXE ODBC Administrator
ODBCCP32.DLL
ODBCCP32.CPL		 ODBC Installer/ODBC Administrator Control Panel
ODBCCR32.DLL ODBC Cursor Library. Because the Desktop Database Drivers use the native scrollable cursors provided by Microsoft Jet, the Cursor Library is usually not needed. An application can, however, force use of the Cursor Library.
ODBCINT.DLL ODBC Core localized strings
ODBC16GT.DLL
ODBC32GT.DLL		 Thunking files for the Driver Manager
ODBC.INF
ODBCKEY.INF
ODBCSTF.DLL		 ODBC setup files
