Microsoft Jet 2.x ISAM Formats Settings

The following table gives the settings and values that Microsoft Jet 3.5 installs for the Microsoft Jet format. These settings are located in the \HKEY_LOCAL_MACHINE \SOFTWARE\Microsoft\Jet\3.5\ISAM Formats\Jet 2.x key of the Windows registry.

Setting Value
Engine Jet 2.x
IsamType 0
OneTablePerFile 00
IndexDialog 00
CreateDBOnExport 00
