Unsupported Tuning Functions

You can use the built-in ISAMStats and ShowPlan functions to measure and improve Microsoft Jet performance. These functions apply only to native Microsoft Jet database files. Similar functions do not exist for the other database formats.

Important The ISAMStats and ShowPlan functions are not supported as an official part of DAO. They have not been through the same quality testing as the supported components of Microsoft Jet. You use them entirely at your own risk. The results returned by the ISAMStats and ShowPlan functions may vary. The availability of these features in future releases of Microsoft Jet is not guaranteed.
