Assigning Permissions to Users for the Objects

The final step is to assign the actual permissions for objects to users and groups. This step tells Microsoft Jet which users can access which objects. Again, Microsoft Access offers the quickest way to accomplish this; use the User And Group Permissions dialog box (Tools menu, Security submenu).

Your new database is now secure. The only people who can get into the objects in the database are those to whom you assigned permissions, and anyone who is a member of the Admins group of the workgroup information file in place when you ran the User-Level Security Wizard.
