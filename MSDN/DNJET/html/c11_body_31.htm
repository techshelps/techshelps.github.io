Using Microsoft Visual J++ and the DAO Automation Interfaces

This section introduces the DAOSample application, a simple Java application that illustrates how to program with the DAO Automation interfaces in Visual J++. The DAOSample application includes the simple DAO applet that displays table entries. The application is written to run either as a Web page or as a stand-alone application.

See Also For information about how to install the DAOSample application, search the Visual J++ Help index for “DAOSample” and follow the instructions in the Help topic.
