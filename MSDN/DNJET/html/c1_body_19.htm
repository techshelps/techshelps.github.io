Engine Configuration Information

Versions of Microsoft Jet prior to 3.0 store configuration information in an initialization file that follows the Windows .ini file standard. (The one exception to this is version 2.5, which stores the location of the installable ISAM DLL in the registry.) Microsoft Jet 3.0 and 3.5 use the Windows registry exclusively. The registry is a structured file used by the Windows 95 and Windows NT operating systems to store information about the system’s configuration, including software components such as Microsoft Jet.
