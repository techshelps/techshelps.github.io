Microsoft Excel ISAM Formats Settings

The following tables give the settings and values that Microsoft Jet 3.5 installs for the Microsoft Excel formats. These settings are located in the \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Jet\3.5\ISAM Formats\Excel version key of the Windows registry, where version is 3.0, 4.0, 5.0, or 8.0.

Note The WizardProject and ImportWizardFunction settings listed in the following tables are added by Microsoft Access.
