The Object Model for ODBCDirect Workspaces

The object model for an ODBCDirect workspace includes a subset of the objects in a Microsoft Jet workspace, with the addition of a new object, the Connection object. Figure 9.1 shows the object hierarchy for ODBCDirect workspaces; the subsequent sections describe the objects themselves, to the extent that they differ from the objects in the Microsoft Jet object hierarchy.
