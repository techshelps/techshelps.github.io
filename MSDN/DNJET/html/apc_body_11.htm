FoxPro ISAM Formats Settings

The following tables give the settings and values that Microsoft Jet 3.5 installs for the Microsoft FoxPro formats. These settings are located in the \HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Jet\3.5\ISAM Formats\FoxPro version key of the Windows registry, where version is 2.0, 2.5, 2.6, 3.0, or DBC.
