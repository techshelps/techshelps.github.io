Types of Cluster Utility Functions

The cluster utility functions can be categorized by the type of work that they do. There are six types of utility functions:

·    Utility functions to access resource information.

  • Utility functions to access values of different types from the cluster database.
  • Utility functions to handle the parsing and construction of property lists that are passed to and from the control code functions of the Resource and Cluster APIs.
  • Utility functions to handle multiple properties using an array of RESUTIL_PROPERTY_ITEM structures, commonly referred to as a property table.
  • Utility functions to manage threads.
  • Utility functions to perform a variety of tasks.
