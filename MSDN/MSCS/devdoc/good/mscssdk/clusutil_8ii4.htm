ResUtilResourceTypesEqual

The ResUtilResourceTypesEqual utility function tests whether or not a resource type matches the resource type name of a specified resource.

BOOL WINAPI ResUtilResourcesTypesEqual(
  LPCWSTR lpszResourceTypeName,
  HRESOURCE hResource  
);

Parameters

lpszResourceTypeName
[in] Pointer to the resource type name to test.
hResource
[in] Handle of the resource to test.

Return Values

TRUE
The resource types are equal.
FALSE
The resource types are not equal.

Notes to Callers

ResUtilResourceTypesEqual compares the resource type name pointed to by lpszResourceTypeName with the resource type name of the resource identified by hResource. To perform the comparison, ResUtilResourceTypesEqual passes the CLUSCTL_RESOURCE_GET_RESOURCE_TYPE control code to the ClusterResourceControl function to retrieve the resource type. If the two resource type names are the same, the resource types are equal. Note that ResUtilResourceTypesEqual compares the resource type name and not the resource type display name.
