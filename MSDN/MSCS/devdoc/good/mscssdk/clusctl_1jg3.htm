CLUSCTL_RESOURCE_TYPE_VALIDATE_COMMON_PROPERTIES

The CLUSCTL_RESOURCE_TYPE_VALIDATE_COMMON_PROPERTIES control code is used to check whether or not specific properties are valid common properties for a resource type.

Quick Info

Type: External
Access: CLUS_ACCESS_READ
Input: Property list
Output: None
Global: CLUS_NOT_GLOBAL
Modify: CLUS_NO_MODIFY
Code: CLCTL_VALIDATE_COMMON_PROPERTIES

Notes to Callers

When you call ClusterResourceTypeControl with the CLUSCTL_RESOURCE_TYPE_VALIDATE_COMMON_PROPERTIES control code, pass a property list with the names and values of all the common read/write properties that you want to validate. A common property is a property that applies to all resource types.

For an overall description of the common resource type properties, see Common Resource Type Properties.

To build a property list, use the property list utility functions.

Notes to Resource DLL Writers

Do not handle the CLUSCTL_RESOURCE_TYPE_VALIDATE_COMMON_PROPERTIES control code. Instead, return ERROR_INVALID_FUNCTION from your ResourceTypeControl entry point function. CLUSCTL_RESOURCE_TYPE_VALIDATE_COMMON_PROPERTIES is handled by the Resource Monitor because it is aware of the valid formats and values for these properties.
