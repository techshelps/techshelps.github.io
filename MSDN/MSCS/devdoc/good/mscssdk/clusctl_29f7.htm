CLUSCTL_RESOURCE_GET_COMMON_PROPERTIES

The CLUSCTL_RESOURCE_GET_COMMON_PROPERTIES control code is used to request the read/write common properties for a resource.

Quick Info

Type: External
Access: CLUS_ACCESS_READ
Input: None
Output: Property list
Global: CLUS_NOT_GLOBAL
Modify: CLUS_NO_MODIFY
Code: CLCTL_GET_COMMON_PROPERTIES

Notes to Callers

When you call ClusterResourceControl with the CLUSCTL_RESOURCE_GET_COMMON_PROPERTIES control code, it returns a property list with the names and values of all of the properties that apply to resources of any type and can be modified using the CLUSCTL_RESOURCE_SET_COMMON_PROPERTIES control code. The following properties are returned:

DebugPrefix

Description

IsAlivePollInterval

LooksAlivePollInterval

PendingTimeout

PersistentState

RestartAction

RestartPeriod

RestartThreshold

SeparateMonitor

Type

For an overall description of the common resource properties, see Common Resource Properties.

To parse a property list, use the property list utility functions.

Notes to Resource DLL Writers

Do not handle the CLUSCTL_RESOURCE_GET_COMMON_PROPERTIES control code in your ResourceControl entry point function. Instead, have ResourceControl return ERROR_INVALID_FUNCTION. The Resource Monitor handles CLUSCTL_RESOURCE_GET_COMMON_PROPERTIES because it is aware of the valid formats and values for these properties.
