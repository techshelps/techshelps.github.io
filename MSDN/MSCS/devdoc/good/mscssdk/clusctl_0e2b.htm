CLUSCTL_NETINTERFACE_GET_COMMON_PROPERTIES

The CLUSCTL_NETINTERFACE_GET_COMMON_PROPERTIES control code is used to request the read/write common properties for a network interface.

Quick Info

Type: External
Access: CLUS_ACCESS_READ
Input: None
Output: Property list
Global: CLUS_NOT_GLOBAL
Modify: CLUS_NO_MODIFY
Code: CLCTL_GET_COMMON_PROPERTIES

Notes to Callers

When you call ClusterNetInterfaceControl with the CLUSCTL_NETINTERFACE_GET_COMMON_PROPERTIES control code, it returns a property list with the names and values of all of the properties that apply to all network interfaces and can be modified using the CLUSCTL_NETINTERFACE_SET_COMMON_PROPERTIES control code. The Description property is the only common property for network interfaces that can be modified.

To parse the property list returned by ClusterNetInterfaceControl, use the property list utility functions.
