CLUSCTL_NETWORK_GET_NAME

The CLUSCTL_NETWORK_GET_NAME control code is used to retrieve the name of a network.

Quick Info

Type: External
Access: CLUS_ACCESS_READ
Input: None
Output: Null-terminated Unicode string
Global: CLUS_NOT_GLOBAL
Modify: CLUS_NO_MODIFY
Code: CLCTL_GET_NAME

Notes to Callers

The name of a network is stored in the network's Name property. To retrieve the Name property along with the other common read-only network properties, call ClusterNetworkControl with the CLUSCTL_NETWORK_GET_RO_COMMON_PROPERTIES control code.
