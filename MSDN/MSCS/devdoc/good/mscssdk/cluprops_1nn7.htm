Description Property for Groups

The Description property provides remarks or comments about the group.

Quick Info

Type: Null-terminated Unicode string
Access: Read/write
Function: ClusterGroupControl
Structure: CLUSPROP_SZ
Control Code: CLUSCTL_GROUP_GET_COMMON_PROPERTIES

Remarks

When ClusterGroupControl processes the CLUSCTL_GROUP_GET_COMMON_PROPERTIES control code, it returns a property list that includes the Description property as one of the entries. The property value portion of the entry contains a CLUSPROP_SZ structure that is set as follows:

CLUSPROP_SZ DescriptionValue;
LPCWSTR DescriptionData = L "Description of group";
DescriptionValue.Syntax = CLUSPROP_SYNTAX_LIST_VALUE_SZ;
DescriptionValue.cbLength = sizeof(DescriptionData);
lstrcpyW(DescriptionValue.sz, DescriptionData);
