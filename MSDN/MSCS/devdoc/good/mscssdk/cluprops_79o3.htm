Path Property for File Shares

The Path property describes the path to the directory being shared.

Quick Info

Type: Null-terminated Unicode string
Access: Read/write
Status: Required
Function: ClusterResourceControl
Structure: CLUSPROP_SZ
Control Code: CLUSCTL_RESOURCE_GET_PRIVATE_PROPERTIES

Remarks

When ClusterResourceControl processes the CLUSCTL_RESOURCE_GET_PRIVATE_PROPERTIES control code, it returns a property list that includes the Path property as one of the entries. The property value portion of the entry contains a CLUSPROP_SZ structure that is set as follows:

CLUSPROP_SZ PathValue;
LPCWSTR PathData = L "\\NodeA\ShareDirectory";
PathValue.Syntax = CLUSPROP_SYNTAX_LIST_VALUE_SZ;
PathValue.cbLength = sizeof(PathData);
lstrcpyW(PathValue.sz, PathData);

QuickInfo

  Version: Use Windows NT Server Enterprise Edition 4.0.
  Windows CE: Unsupported.
