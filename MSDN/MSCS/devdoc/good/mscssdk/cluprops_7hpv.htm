JobCompletionTimeout Property for Print Spoolers

The JobCompletionTimeout property is the number of milliseconds before a pending job is deleted from the Print Spooler resource's queue.

Quick Info

Type: DWORD
Access: Read/write
Status: Optional
Function: ClusterResourceControl
Structure: CLUSPROP_DWORD
Control Code: CLUSCTL_RESOURCE_GET_PRIVATE_PROPERTIES

Remarks

When ClusterResourceControl processes the CLUSCTL_RESOURCE_GET_PRIVATE_PROPERTIES control code, it returns a property list that includes the JobCompletionTimeout property as one of the entries. The property value portion of the entry contains a CLUSPROP_DWORD structure that is set as follows:

DWORD JobCompletionTimeOutData = 200;
CLUSPROP_DWORD JobCompletionTimeOutValue;
JobCompletionTimeOutValue.Syntax = CLUSPROP_SYNTAX_LIST_VALUE_DWORD;
JobCompletionTimeOutValue.cbLength = sizeof(DWORD);
JobCompletionTimeOutValue.dw = JobCompletionTimeOutData;
