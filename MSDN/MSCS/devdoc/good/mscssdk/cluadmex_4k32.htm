IGetClusterDataInfo : IUnknown

The IGetClusterDataInfo interface is called by a Cluster Administrator extension to retrieve information about a cluster.

Vtable Order

GetClusterName Retrieves the name of the current cluster.
GetClusterHandle Retrieves a handle to the current cluster.
GetObjectCount Retrieves a count of the number of selected objects.

Remarks

To call the methods in the IGetClusterDataInfo interface, Cluster Administrator extensions use an interface pointer provided by Cluster Administrator. An extension can access this pointer by calling the IUnknown::QueryInterface method with the piData pointer passed to one of these methods:

IWEExtendPropertySheet::CreatePropertySheetPages

IWEExtendContextMenu::AddContextMenuItems

IWEExtendWizard::CreateWizardPages

IWEInvokeCommand::InvokeCommand
