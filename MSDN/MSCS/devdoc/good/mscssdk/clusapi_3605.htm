OfflineClusterResource

The OfflineClusterResource function takes an online resource offline.

DWORD WINAPI OfflineClusterResource(
  HRESOURCE hResource  
);

Parameters

hResource
[in] Handle to the resource to be taken offline.

Return Values

ERROR_SUCCESS
The operation was successful.
ERROR_IO_PENDING
The resource or one of the resources it depends on has returned ERROR_IO_PENDING from its Offline entry point function.

If the operation was unsuccessful, OfflineClusterResource can also return a Win32 error value.

QuickInfo
