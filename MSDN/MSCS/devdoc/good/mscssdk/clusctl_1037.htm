CLUSCTL_GROUP_GET_COMMON_PROPERTIES

The CLUSCTL_GROUP_GET_COMMON_PROPERTIES control code is used to request the read/write common properties for a group.

Quick Info

Type: External
Access: CLUS_ACCESS_READ
Input: None
Output: Property list
Global: CLUS_NOT_GLOBAL
Modify: CLUS_NO_MODIFY
Code: CLCTL_GET_COMMON_PROPERTIES

Notes to Callers

When you call ClusterGroupControl with the CLUSCTL_GROUP_GET_COMMON_PROPERTIES control code, it returns a property list with the names and values of the properties that apply to all groups and can be modified using the CLUSCTL_GROUP_SET_COMMON_PROPERTIES control code. The following properties are returned:

AutoFailbackType

Description

FailbackWindowStart

FailbackWindowEnd

FailoverThreshold

FailoverPeriod

PersistentState

For an overall description of the common group properties, see Common Group Properties.

To parse a property list, use the property list utility functions.
