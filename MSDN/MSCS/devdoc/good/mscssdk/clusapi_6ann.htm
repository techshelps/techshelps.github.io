Network Interface Management Functions

The functions in the Cluster API for managing a network interface in a cluster open and close the network interface, perform selected operations, and retrieve information. The following functions are included in the Cluster API for managing network interfaces:

Function Description
CloseClusterNetInterface Closes a network interface handle.
ClusterNetInterfaceControl Initiates an operation affecting a network interface.
  • GetClusterNetInterface
 Returns the name of a node's interface to a network in a cluster.
  • GetClusterNetInterfaceState
 Returns the current state of a network interface.
OpenClusterNetInterface Opens a handle to a network interface.
