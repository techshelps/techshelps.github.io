Cluster Utility Functions

The Cluster Utility functions provide a shortcut to implementing common application or resource DLL tasks. By calling a utility function, developers of these components can save significant time and effort.

The Cluster Utility functions are implemented in RESUTILS.DLL, a dynamic link library that is provided with Cluster Administrator. Applications and resource DLLs that want to use the Cluster Utility functions must include RESAPI.H in addition to CLUSAPI.H and link to RESUTILS.LIB.
