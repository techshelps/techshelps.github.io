IWEExtendContextMenu : IUnknown

The IWEExtendContextMenu interface is implemented by a Cluster Administrator extension to enhance a Cluster Administrator context menu.

Vtable Order

Add ContextMenuItems Adds items to a Cluster Administrator context menu.
