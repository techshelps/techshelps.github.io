CLUSCTL_NETINTERFACE_UNKNOWN

The CLUSCTL_NETINTERFACE_UNKNOWN control code indicates a request for an unknown operation. ClusterNetInterfaceControl always returns ERROR_SUCCESS when the dwControlCode parameter is set to this code.

Quick Info

Type: External
Access: CLUS_ACCESS_READ
Input: None
Output: None
Global: CLUS_NOT_GLOBAL
Modify: CLUS_NO_MODIFY
Control Code: CLCTL_UNKNOWN
