Directory Property for IIS Virtual Roots

The Directory property describes the physical location of the IIS Virtual Root.

Quick Info

Type: Null-terminated Unicode string
Access: Read/write
Status: Required
Function: ClusterResourceControl
Structure: CLUSPROP_SZ
Control Code: CLUSCTL_RESOURCE_GET_PRIVATE_PROPERTIES

Remarks

The data for the Directory property is a full directory path. When ClusterResourceControl processes the CLUSCTL_RESOURCE_GET_PRIVATE_PROPERTIES control code, it returns a property list that includes the Directory property as one of the entries. The property value portion of the entry contains a CLUSPROP_SZ structure that is set as follows:

CLUSPROP_SZ DirectoryValue;
LPCWSTR DirectoryData = L "C:\SCRIPTS";
DirectoryValue.Syntax = CLUSPROP_SYNTAX_LIST_VALUE_SZ;
DirectoryValue.cbLength = sizeof(DirectoryData);
lstrcpyW(DirectoryValue.sz, DirectoryData);
