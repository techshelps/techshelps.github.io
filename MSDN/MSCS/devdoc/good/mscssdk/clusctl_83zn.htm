CLUSCTL_NETINTERFACE_SET_COMMON_PROPERTIES

The CLUSCTL_NETINTERFACE_SET_COMMON_PROPERTIES control code is used to update the common properties of a network interface.

Quick Info

Type: External
Access: CLUS_ACCESS_WRITE
Input: Property list
Output: None
Global: CLUS_NOT_GLOBAL
Modify: CLUS_MODIFY
Code: CLCTL_SET_COMMON_PROPERTIES

Notes to Callers

The CLUSCTL_NETINTERFACE_SET_COMMON_PROPERTIES control code enables you to set one or more of the read/write properties that apply to all network interfaces. The following property may be modified with CLUSCTL_NETINTERFACE_SET_COMMON_PROPERTIES:

Description

To retrieve the names and current values of the common network interface properties, use the CLUSCTL_NETINTERFACE_GET_COMMON_PROPERTIES control code.

For an overall description of the common network interface properties, see Common Network Interface Properties.

To construct a property list to pass to ClusterNetInterfaceControl, use the property list utility functions.
