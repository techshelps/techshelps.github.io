V

Value list
A collection of data values that is used as the data portion of a property value in a property list. Each entry except the final one in a value list is made up of a CLUSPROP_VALUE structure and data. The final entry contains only the CLUSPROP_SYNTAX_ENDMARK value.
Virtual circuit
Form of communicating where, when a transmission is complete, the transmitted information is guaranteed to be received by the intended recipient or an error is returned. Also, messages are delivered in order and are never duplicated.
Virtual server
Collection of services which gives the appearance of an actual Microsoft® Windows NT® Server network operating system to clients. A virtual server is typically a group containing all of the resources to run a particular application, such as a Network Name resource and an IP Address resource.
