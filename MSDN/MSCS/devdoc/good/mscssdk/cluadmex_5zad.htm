IGetClusterDataInfo::GetClusterHandle

The IGetClusterDataInfo::GetClusterHandle method retrieves a handle to the current cluster.

HCLUSTER GetClusterHandle();

Parameters

None.

Return Values

If GetClusterHandle is successful, it returns a cluster handle.
