CLUSCTL_GROUP_GET_RO_PRIVATE_PROPERTIES

The CLUSCTL_GROUP_GET_RO_PRIVATE_PROPERTIES control code is used to request the read-only private properties for a group.

Quick Info

Type: External
Access: CLUS_ACCESS_READ
Input: None
Output: Property list
Global: CLUS_NOT_GLOBAL
Modify: CLUS_NO_MODIFY
Code: CLCTL_GET_RO_PRIVATE_PROPERTIES

Notes to Callers

When you call ClusterGroupControl with the CLUSCTL_GROUP_GET_RO_PRIVATE_PROPERTIES control code, it returns a property list with the names and values of the properties that apply only to the specific group and cannot be modified using the CLUSCTL_GROUP_SET_PRIVATE_PROPERTIES control code. Private properties can include:

  • Dynamic values generated as needed.
  • Static values stored in the cluster database.

To parse the property list returned by ClusterGroupControl, use the property list utility functions.

Note There are currently no private properties defined for groups.
