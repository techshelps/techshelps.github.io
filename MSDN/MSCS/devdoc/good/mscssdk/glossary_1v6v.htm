G

Generic Application resource
A cluster-aware or cluster-unaware application that is running on a cluster node and being managed as a cluster resource. Cluster Server provides a resource DLL to manage Generic Application resources.
Generic Service resource
A Microsoft® Windows NT® service that is running on a cluster node and is being managed as a cluster resource. Cluster Server provides a resource DLL to manage Generic Service resources.
Global Update Manager
Software component in the Cluster Service that provides a global update service that is used by other components in the Cluster Service.
Group
Collection of resources to be managed as a single unit. Typically a group contains all of the resources needed to run a specific application or service. When a group contains a Network Name resource and an IP Address resource, it can be accessed as a virtual server.
