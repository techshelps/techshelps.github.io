REGCLADM

The REGCLADM.EXE program registers a Cluster Administrator extension DLL with the cluster and the COM library.

REGCLADM 
  [/U]         
  [/Ccluster]  
[drive:][path] filename

Parameters

/U
Uninstalls the Cluster Administrator extension DLL specified by the drive, path, and filename parameters.
/Ccluster
Identifies the cluster in which to register the Cluster Administrator extension DLL. The default cluster is the cluster running on the local computer.
drive:path filename
Identifies the Cluster Administrator extension DLL to register.

Remarks

To run REGCLADM, specify the name of the cluster, a path to the Cluster Administrator extension DLL, and an option to uninstall the DLL if necessary. A Cluster Administrator extension DLL must be registered on each computer that is running Cluster Administrator.
