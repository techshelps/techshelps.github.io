N

Network Name resource
A friendly name of an entity that exists on a network. Cluster Server provides a resource DLL for managing Network Name resources. When a group contains a Network Name resource and an IP Address resource, it can be accessed by clients as a virtual server.
Network partition
Situation that occurs when there is a network failure and one node in a cluster cannot receive heartbeat signals from the other node. Also known as split brain syndrome.
No Remote Memory Access (NORMA)
Clustering architecture where all memory is local to each system and all communication between systems consists of message passing. Shared-disk architectures are sometimes included in this classification.
Node
Microsoft® Windows NT® Server network operating system that is an active or inactive member of a cluster.
