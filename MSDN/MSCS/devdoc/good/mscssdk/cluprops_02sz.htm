Private Properties for IIS Virtual Root Resources

Each Internet Information Server (IIS) Virtual Root resource is an IIS Website, an FTP site, or a Gopher site containing a collection of IIS Virtual Directories. Each Cluster Server IIS Virtual Root resource is dependent on an IP Address resource.

IIS Virtual Root resources have several private properties, some that are required and others that are optional. The following table lists these properties:

AccessMask Optional
Alias Required
Directory Required
ServiceName Required

Cluster Administrator presents the IIS Virtual Root resource properties on the property page associated with the Parameters tab for IIS Virtual Roots.
