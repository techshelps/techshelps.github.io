CLUSCTL_NETWORK_VALIDATE_COMMON_PROPERTIES

The CLUSCTL_NETWORK_VALIDATE_COMMON_PROPERTIES control code is used to check whether or not specific properties and property values are valid as common network properties.

Quick Info

Type: External
Access: CLUS_ACCESS_READ
Input: Property list
Output: None
Global: CLUS_NOT_GLOBAL
Modify: CLUS_NO_MODIFY
Code: CLCTL_VALIDATE_COMMON_PROPERTIES

Notes to Callers

When you call ClusterNetworkControl with the CLUSCTL_NETWORK_VALIDATE_COMMON_PROPERTIES control code, pass a property list with the names and values of all of the common read/write properties that you want to validate. A common property is a property that applies to all networks.

For an overall description of the common network properties, see Common Network Properties.

To build a property list, use the property list utility functions.
