IGetClusterGroupInfo : IUnknown

The IGetClusterGroupInfo interface is called by a Cluster Administrator extension to retrieve information about a group.

Vtable Order

GetGroupHandle Retrieves a handle to a group.

Remarks

To call the methods in the IGetClusterGroupInfo interface, Cluster Administrator extensions use an interface pointer provided by Cluster Administrator. An extension can access this pointer by calling the IUnknown::QueryInterface method with the piData pointer passed to one of these methods:

IWEExtendPropertySheet::CreatePropertySheetPages

IWEExtendContextMenu::AddContextMenuItems

IWEExtendWizard::CreateWizardPages

IWEInvokeCommand::InvokeCommand
