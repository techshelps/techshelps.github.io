DllRegisterCluAdminExtension

The DllRegisterCluAdminExtension function registers a Cluster Administrator extension DLL with the cluster by adding information to the cluster database.

STDAPI DllRegisterCluAdminExtension(
  HCLUSTER hCluster  
);

Parameters

hCluster
[in] Handle to the cluster to modify in the cluster database.

Return Values

NOERROR
The Cluster Administrator Extension DLL was successfully registered.

If DllRegisterCluAdminExtension is unsuccessful, it returns a Win32® error value.
