IWEExtendWizard : IUnknown

The IWEExtendWizard interface is implemented by a Cluster Administrator extension to enhance Cluster Administrator's Create Group or Create Resource wizard.

Vtable Order

CreateWizardPages Creates property sheet pages and adds them to the Create Group or Create Resource Wizard.

Remarks

The IWEExtendWizard interface is implemented by Cluster Administrator extensions.
