IIS Virtual Root resource
A virtual root to be used with Internet Information Server (IIS) that supports the WWW, FTP, and GOPHER services. IIS Virtual Root resources may have dependencies on IP Address resources, Network Name resources, and/or Physical Disk resources. Virtual roots containing access information are not supported. Cluster Server provides a resource DLL for managing IIS Virtual Root resources.
Interconnect
Private network that connects nodes in a cluster.
IP Address resource
A 32-bit number in dotted decimal format that represents an Internet Protocol (IP) address. Cluster Server provides a resource DLL for managing IP Address resources. When a group contains an IP Address resource and a Network Name resource, it can be accessed by clients as a virtual server.
