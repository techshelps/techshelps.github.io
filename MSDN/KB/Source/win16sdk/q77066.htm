PRB: Problems with Dr. Watson or with the Packager

 Last reviewed: July 23, 1997
Article ID: Q77066
3.10 WINDOWS kbprg kbprb

The information in this article applies to:

  • Microsoft Windows Software Development Kit (SDK) for Windows, version 3.1

SYMPTOMS

Neither Dr. Watson nor the Packager runs correctly.

CAUSE

An old version of the TOOLHELP.DLL file is in the main Windows directory (by default, C:\WINDOWS).

RESOLUTION

Verify that Windows has installed TOOLHELP.DLL in the Windows system directory (by default, C:\WINDOWS\SYSTEM). Delete the old TOOLHELP.DLL from the Windows directory.
