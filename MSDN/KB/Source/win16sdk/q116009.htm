Get MPEG Command Set for MCI from Microsoft Software Library

 Last reviewed: June 4, 1996
Article ID: Q116009
The information in this article applies to:
  • Microsoft Windows Software Development Kit (SDK) for Windows version 3.1

Motion Picture Experts Group (MPEG) is a compression format for use in motion video multimedia applications and drivers. This article documents the current MPEG command set for the Media Control Interface (MCI) and is contained in the file MCIMPG.EXE. 

You can find MCIMPG.EXE (size: 40488 bytes) 
                       , a self-extracting file, on these services:
  • Microsoft's World Wide Web site on the Internet 

          On the www.microsoft.com home page, click the Support icon
      Click Knowledge Base, and select the product

          Enter kbfile MCIMPG.EXE (size: 40488 bytes) 
                             , and click GO!
      Open the article, and click the button to download the file


  • Internet (anonymous FTP)

          ftp ftp.microsoft.com
      Change to the Softlib/Mslfiles folder
      Get MCIMPG.EXE (size: 40488 bytes) 

    

  • The Microsoft Network

          On the Edit menu, click Go To, and then click Other Location
      Type "mssupport" (without the quotation marks)
      Double-click the MS Software Library icon
      Find the appropriate product area
      Locate and Download MCIMPG.EXE

    

  • Microsoft Download Service (MSDL)

          Dial (206) 936-6735 to connect to MSDL
      Download MCIMPG.EXE (size: 40488 bytes) 

    
For additional information about downloading, please see the following
article in the Microsoft Knowledge Base:

    

       ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services


    In late 1993 and early 1994, Microsoft was approached by several hardware
and software companies to create an industry standard for MPEG on PCs.
While commercial MPEG products for PC are starting to appear, many
companies in the industry believe that a standard playback command set will
speed widespread acceptance of MPEG on the PC.

    
This MPEG command set is a subset of the "Digital Video Command Set
for the Media Control Interface" (Microsoft part #098-37538).
