Retrieve Line Number from Edit Control Sample Code

 Last reviewed: August 12, 1997
Article ID: Q77782

The information in this article applies to:

  • Microsoft Windows Software Development Kit (SDK) for Windows versions 3.0 and 3.1

In an edit control, the EM_LINEFROMCHAR message returns the line number of the line that contains a specified character.

Edicnt.exe is a file in the Microsoft Software Library that demonstrates using the EM_LINEFROMCHAR message to determine the line number of the line of an edit control selected using the mouse.

The following file is available for download from the Microsoft Software Library: 

 ~ Editcnt.exe (size: 20350 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base:
