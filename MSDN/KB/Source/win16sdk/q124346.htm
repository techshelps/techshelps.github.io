How to Obtain the Video for Windows Development Kit

 Last reviewed: August 5, 1996
Article ID: Q124346
The information in this article applies to:
  • Video for Windows Development Kit (DK) version 1.1

Win16 Availability

The Video for Windows Development Kit (DK) for Windows version 3.1 is available through the following channels:

  • The Multimedia JumpStart CD-ROM disc

    Please see the following article in the Microsoft Knowledge base for information about obtaining the JumpStart CD-ROM disc:

    ARTICLE-ID: Q124824 

       TITLE     : How to Obtain the Multimedia Jumpstart CD
  • The Microsoft Developer Network Development Platform (MSDN level 2), which is a quarterly CD subscription service.

    The Video for Windows DK resides on disc 6 of 20 in the October 1994 release of the MSDN Platform in the \WIN31\VFW\VFWDK directory. It may be located on a different disc in future releases.

    Please see the following article in the Microsoft Knowledge base for information about subscribing to the MSDN Platform: 

          ARTICLE-ID: Q116437

          TITLE     : MSDN Frequently Asked Questions (FAQ)


Win32 Availability

 


There is no separate Video for Windows Development Kit for Win32. Instead,
the Video for Windows DK has been incorporated in the Microsoft Win32 SDK
for Windows NT version 3.5.


Windows NT version 3.1 doesn't provide any Video for Windows functionality.
Developers must upgrade to Windows NT version 3.5.
