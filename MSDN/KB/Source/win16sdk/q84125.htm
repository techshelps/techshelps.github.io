Cannot Install Windows SDK Over Debugging Windows

 Last reviewed: July 23, 1997
Article ID: Q84125
3.10 WINDOWS kbsetup kbprg

The information in this article applies to:

  • Microsoft Windows Software Development Kit (SDK) for Windows version 3.1

The installation program for the Windows Software Development Kit (SDK) version 3.1 is incompatible with the debugging version of Windows.

To resolve this problem, run the D2N batch file to set up the standard (non- debugging) version of Windows, then run the SDK installation program.
