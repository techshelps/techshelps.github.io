Sample Source Code Demonstrates Playing Metafiles

 Last reviewed: August 12, 1997
Article ID: Q77649

The information in this article applies to:

  • Microsoft Windows Software Development Kit (SDK) for Windows, versions 3.0 and 3.1

Playmeta.exe is a file in the Microsoft Software Library that demonstrates techniques an application can use to open a metafile and "play" it into a device context.

Playmeta.exe can also be used to help identify errors in a given metafile because it pauses and displays a message box before acting on each record.

The following file is available for download from the Microsoft Software Library: 

 ~ Playmeta.exe (size: 19738 bytes)
