OL97: Notes Interface Will Not Connect from Outlook

 Last reviewed: February 26, 1998
Article ID: Q158243
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When running Microsoft Outlook 97, you can set up a connection to a Notes Mailbox. But when Outlook attempts to connect to the Notes Server, you may receive the following error message: 

   Unable to display folder, invalid entryID.
This error does not occur when running Microsoft Exchange Client.

CAUSE

The Notes Transport does not support enough of the Messaging Application Programming Iinterface (MAPI) specification to work with Outlook. NotesMail is not a supported configuration.

WORKAROUND

Use a Personal Store (PST) rather than a Notes Store.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
