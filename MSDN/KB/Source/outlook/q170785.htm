OL97: Invalid Page Fault Clicking Outlook Contacts Folder

 Last reviewed: January 29, 1998
Article ID: Q170785
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

In Microsoft Outlook 97, if you click the bottom folder in the folder list, and subsequently click each folder above, you may receive the following error message when you click the Contacts folder: 

   OUTLOOK has caused an Invalid Page Fault in module <unknown>
NOTE: This error also occurs when exiting Outlook 97.

CAUSE

This error occurs under the following circumstances:

  • You have either the Distributed Component Object Model for Windows 95 (DCOM95) or Microsoft Internet Explorer, version 4.0 installed. 

        -and-
  • You have the Handheld PC (H/PC) Explorer, version 1.1 installed. 

        -and/or-
  • You have WinFax Pro, version 8.0 or 8.01 installed.

RESOLUTION

There are several ways to resolve this problem.

Method 1

You can disable the H/PC Explorer and WinFax Pro extension used by Outlook. H/PC Explorer 1.1 installs an extension called PMail. WinFax Pro 8.0 and 8.01 install an extension called DCCExtension.

To disable or remove the extensions, follow these steps:

  1. Start Outlook.

  2. On the Tools menu, click Options.

  3. Click the General tab.

  4. Click Add-In Manager.

  5. Click to clear the extension you want to disable. If you want to remove an extension from your computer, click to select the extension, and then click Remove.

  6. Click OK and OK again to close all dialog boxes.

Method 2

You can remove DCOM95 or Microsoft Internet Explorer, 4.0 from your computer.

To remove DCOM95, follow these steps:

  1. On the Windows taskbar, click Start, point to Settings, and then click Control Panel.

  2. In Control Panel, double-click Add/Remove Programs.

  3. Click to select "DCOM for Windows 95" and click Add/Remove.

  4. When prompted, click Yes to restart your computer.

To remove Microsoft Internet Explorer 4.0, you must install Windows 95 into a clean folder. Unlike DCOM95, which can restore old OLE files, Microsoft Internet Explorer 4.0 cannot.

MORE INFORMATION

Both DCOM95 and IE 4.0 update the OLE files; it is these updates, in combination with the program extensions, that cause the problem.

Steps to Reproduce Problem

  1. Install DCOM95 (It can be located in Visual Basic, version 5.0, Visual C++, version 5.0, and Visual Studio 97).

  2. Install H/PC Explorer 1.1.

  3. Start Outlook.

  4. On the View menu, click Folder List.

  5. Click the bottom folder in the list.

  6. Click each subsequent folder above until you reach Contacts.
