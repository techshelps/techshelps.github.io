OL97: Reminders Sound when Not Set for Timex Data Link Watch

 Last reviewed: December 18, 1997
Article ID: Q178262
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

In Microsoft Outlook 97, when you download appointments to the Timex Data Link Watch and choose not to set reminders in the wizard, the reminders will still sound.

RESOLUTION

If you do not want to have reminders activated by the Data Link Watch, you can clear the reminder option AND set the time interval to 0 minutes.

-or-

You can keep the option selected AND change the time interval to 0 minutes.

MORE INFORMATION

The reminder option in the wizard is independent of the reminder you set in the Outlook appointment.

For example, if you create an appointment in Outlook and set the reminder to sound 15 minutes beforehand, the reminder may not sound on the watch. You must explicitly set the reminder in the wizard by selecting the reminder option and setting a time interval greater than 0 minutes.

However, if you clear the reminder option in the wizard, the watch will activate a reminder based on the time interval set in the reminder option.

If the wizard has the reminder option set for 15 minutes beforehand and you clear the option, the watch will still activate the reminder 15 minutes beforehand.
