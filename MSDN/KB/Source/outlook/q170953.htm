OL97: How to Use Outlook 8.01 with Office Run from Server

 Last reviewed: January 19, 1998
Article ID: Q170953
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article describes how to update a Microsoft Office 97 "Run from Server" installation to include Microsoft Outlook 97 version 8.01.

MORE INFORMATION

Outlook version 8.01 has no Run from Server Setup option; you must install it locally. To install Outlook version 8.01 locally in a Run from Server installation, follow thse steps:

  1. Use Add/Remove Programs in the Control Panel to remove the current installation of Office from each workstation.

  2. Install Outlook version 8.01 on each workstation.

  3. Make a copy of the Off97Pro.stf file found on the server, and modify it to NOT install Outlook, by removing the value "202" from line 56.

  4. Run Setup to install Office 97 on each workstation with the Run from Server option.
