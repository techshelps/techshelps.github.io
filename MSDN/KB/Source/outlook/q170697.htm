OL97: Cannot Forward a Fax in Outlook

 Last reviewed: January 19, 1998
Article ID: Q170697
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you attempt to forward a fax using Microsoft Outlook 97, you may receive an "Undeliverable" message from the System Administrator. 

   -or-
You may receive an error message that Outlook cannot schedule the fax for delivery.

RESOLUTION

You can resolve this problem using one of the methods below.

Method 1

Try printing the fax message to the Microsoft Fax printer. This will create a new fax message that contains the original information. Follow these steps to print to the Fax printer:

  1. In Outlook, open the fax message.

  2. On the File menu, click Print to open the Print dialog box.

  3. In the Name box, click to select "Microsoft Fax."

  4. Click OK.

  5. Step through the Compose New Fax wizard to send a new fax.

  6. Click Finish at the end of the wizard.

Method 2

  1. In Outlook, open the fax message.

  2. Select the contents of the fax and on the Edit menu, click Copy.

  3. On the File menu, click Close.

  4. On the File menu, click File and New and then Mail Message.

  5. Address the new mail message to the recipient and type in the subject. If you are sending to a fax recipient use the format [FAX: +1 (xxx) xxx-xxxx].

  6. Click in the message area and on the Edit menu, click Paste.

  7. Click Send to send the new message.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce the Problem

These steps assume you received a fax in Outlook.

  1. Open the fax message and on the message toolbar click Forward.

  2. On the message toolbar, click the Address Book and click to 

        select a name that has a fax number.


           -or-


        In the message "To" field, type the following notation for a fax:


           [FAX: +1 (xxx) xxx-xxxx]

  3. Click Send.

NOTE: The Exchange client that comes with Outlook also displays this problem.
