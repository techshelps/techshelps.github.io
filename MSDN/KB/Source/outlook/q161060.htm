OL97: Task Information Bar Does Not Show Overdue Status

 Last reviewed: January 19, 1998
Article ID: Q161060
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you open a Task that is overdue less than fifteen days, the default Task form displays an Information Bar with the number of overdue days. If the task is overdue by fifteen or more days, no Information Bar appears.

CAUSE

By design, Microsoft Outlook 97 limits the Information Bar warning to fourteen days.

WORKAROUND

You can identify all overdue tasks by selecting one of the Current View options within the Task Folder. All overdue tasks are displayed in the appropriate Overdue Task Color.
