OL97: Error Message or No Response Trying to Fax from Outlook

 Last reviewed: January 19, 1998
Article ID: Q169062
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you attempt to compose or send a fax from Microsoft Outlook 97, you may receive no response or an error message that the printer is not ready.

When you click Compose and then click New Fax, you may receive no response from a profile that contains the Fax service. 

   -or-
If you compose a message and click Print on File menu, and then select Microsoft Fax from the Printer list; after you click OK to print to the fax, you may receive the following error message: 

   Printer is not ready

   -or-
If you compose a message addressed to a fax recipient and click Send, the fax stays in your Outbox and is not sent.

CAUSE

Your profile has "Deliver new mail to the following location" set to (None).

RESOLUTION

Set your messaging system delivery point to a valid Personal Folder or a valid Exchange Server mailbox.

To set you messaging system delivery point, use the following steps:

  1. Open Outlook using your profile that contains the fax service.

  2. On the Tools menu, click Services and click the Delivery tab.

  3. In the "Deliver new mail to the following location" list, click to select Personal Folders or Mailbox.
