OL97: Go to Today and/or Group By Button Unavailable on Toolbar

 Last reviewed: January 19, 1998
Article ID: Q170603
The information in this article applies to:
  • Microsoft Outlook versions 8.00, 8.01, 97

SYMPTOMS

In Microsoft Outlook 97, the "Go to Today" toolbar button in the Calendar folder and/or the "Group By" toolbar button in the Inbox folder are missing or unavailable.

CAUSE

You have the following combination installed on your computer:

  • Microsoft Distributed Component Object Mode for Windows 95 (DCOM) or Microsoft Internet Explorer 4.0
  • and either
  • Microsoft H/PC Explorer 1.1
  • or
  • Winfax Pro 8.0/8.01

RESOLUTION

Choose the appropriate method from the following:

Method 1

Remove the H/PC Explorer extensions (PMail) and/or Winfax Pro extensions (DCC Extensions) from Outlook:

  1. On the Tools menu, click Options.

  2. Click the General tab.

  3. Click Add-In Manager.

  4. Select the extension, click Remove, then click OK.

  5. Click OK to close the Options dialog box.

Method 2

Remove DCOM:

  1. Click Start, point to Settings, then click Control Panel.

  2. Double-click Add/Remove Programs.

  3. Click DCOM for Windows 95.

  4. Click Add/Remove.

  5. When prompted, click Yes to restart your computer.

Method 3

Remove Internet Explorer 4.0.

For more information on removing Internet Explorer 4.0, please see the following articles in the Microsoft Knowledge Base: 

   Article-ID:  Q166322
   Title     :  Uninstalling Internet Explorer 4.0 Platform Preview 1.0

   Article-ID:  Q166313
   Title     :  Removing Internet Explorer 4.0 for Windows 95 Using
                Ieremove.exe

   Article-ID:  Q166314
   Title     :  Removing Internet Explorer 4.0 for Windows NT 4.0 Using
                Ieremove.exe

MORE INFORMATION

Removing either the H/PC Explorer or WinFax Pro extensions will not affect the use of the programs or the funtionality of Outlook.
