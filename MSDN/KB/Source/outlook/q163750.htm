OL97: Outlook Running With Eudora Pro 3.0 May Cause Errors

 Last reviewed: February 5, 1998
Article ID: Q163750
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you run Microsoft Outlook 97 on the same computer with Eudora Pro 3.0, 3.0.1, or Eudora Light 3.0.1 32 bit, you may receive one of the following error messages when using Outlook: 

   OUTLLIB.DLL is linked to missing export MAPI32.DLL.

   -or-

   The MAPI Spooler has shut down unexpectedly. Close all
   mail-enabled applications before attempting to log on again.

   -or-

   Ordinal 136 could not be located in the dynamic link library Mapi32.dll.

CAUSE

Eudora Pro includes a set of messaging functions called Extended Messaging Services Application Programming Interface (API) that may cause this problem.

RESOLUTION

To correct the problem turn off Eudora's MAPI service:

  1. On the Eudora Tools menu, click Options.

  2. In the Category list, click MAPI.

  3. Under, Use Eudora MAPI server, click Never and OK.

  4. Exit Eudora and restart Outlook.
