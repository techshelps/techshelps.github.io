OL97: How to Change the Font Size on the Calendar TaskPad

 Last reviewed: January 19, 1998
Article ID: Q170956
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article describes how to set the font size used for the TaskPad in the Calendar in Microsoft Outlook 97.

MORE INFORMATION

How to Change the TaskPad Display Font Size

  1. From the Outlook Day/Week/Month view, click the View menu, point to TaskPad Settings and click Format View.

  2. Click Font to change the font for Column headings, Rows, and AutoPreview.

This will affect the display font settings for the TaskPad; it will not affect the printed version of the TaskPad.

How to Change the TaskPad Font Size for Printing

The font size specified for printed Appointments controls the font size for printed TaskPad items. To set the font size for printed Appointments follow these steps:

  1. On the File menu click Print and then click Page Setup.

  2. In the Fonts area, click Font for Appointments, change the font size and click OK.

  3. Click Print to print the Calendar page.

On the printout the TaskPad font will be the same size as you set for Appointments.

NOTE: Changing the time increment for Appointments to less than 30 minutes will cause the Appointment font to revert to the 8-point Outlook default. This does not affect the font size for the TaskPad; it will retain the size specified for Appointments.

To Change the Time Increment for Appointments

  1. From Outlook's Day/Week/Month view, click the View menu and click Format View.

  2. In the Day area, change the Time scale by clicking a time interval from the list.
