OL97 ErrMsg: Installing ValuPack Components for Outlook

 Last reviewed: January 19, 1998
Article ID: Q173057
The information in this article applies to:
  • Microsoft Outlook, versions 8.02, 97
  • Microsoft Office 97 for Windows, Service Release 1

SYMPTOMS

If you attempt to install any Microsoft Outlook 97 component from the ValuPack folder, and the component has a separate setup for Microsoft Windows 95/NT 4.0 and Microsoft Windows NT 3.51, such as additional converters, cc:Mail, or the Timex Datalink converter, you may receive the following error after the files are extracted: 

   Error loading SETUPAPI.DLL
The above error occurs with the ValuPack from Microsoft Office 97, Service Release 1 Enterprise Update, and Microsoft Office 97 Service Release 1 Patch on CD-ROM.

With the original Office 97, the same scenario results in the following message: 

   Windows 95 or Windows NT 4.0 Beta 2 or greater is required for this
   installation.

CAUSE

You are using Windows NT 3.51 and chose to install the wrong version of the ValuPack files for your operating system.

RESOLUTION

You must install the files designed for your operating system. To guarantee that you install the correct files follow these steps:

  1. Insert your Microsoft Office 97 Service Release 1 compact disc in the CD-ROM drive.

  2. Double-click the ValuPack folder and then double-click the ValuPack Help file (Valupk8.hlp).

  3. In the "What do you want to know about?" list, click to select the component you want to install. For example, click the "Timex Data Link Watch Wizard."

  4. If separate links appear in the component's help topic for Windows 95/NT 4.0 and Windows NT 3.51, click the link for Windows NT 3.51 to install the correct files.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97, version 8.02. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. With Windows NT 3.51 as your operating system and Office 97 Service Release 1 installed, insert your Office compact disc in your CD-ROM drive.

  2. With Explorer navigate to and double-click to install the following file from the ValuPack: 

          D:\VaulPack\Timex\timexwiz.exe

After the files are extracted, you receive the error message described in the Symptoms section of this article.
