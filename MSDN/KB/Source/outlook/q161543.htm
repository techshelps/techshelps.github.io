OL97: How to Disable VB Code When You Open an Outlook Item

 Last reviewed: January 19, 1998
Article ID: Q161543

The information in this article applies to:

  • Microsoft Outlook 97

SUMMARY

Programmers can store Visual Basic Scripting Edition code in a Microsoft Outlook 97 item and configure the code to automatically run when you open the item. This article describes how to open an Outlook item without running any Visual Basic Scripting Edition code.

MORE INFORMATION

The behavior of Outlook may vary when you open an item that contains Visual Basic Scripting Edition code, as follows:

  • If you open an item that contains Visual Basic Scripting Edition code which is not published in a forms library, you receive a prompt to either enable or disable the code from running.
  • If you open an item that contains Visual Basic Scripting Edition code which is published in a forms library, you do not receive a prompt to disable the code from running.
To prevent any Visual Basic Scripting Edition code from running, hold down the SHIFT key while you open the item. For example, hold down SHIFT while you open an item to prevent the Visual Basic Script code for the Open event from running.

REFERENCES

For more information about disabling Visual Basic Scripting Edition code, type "disable visual basic macro" in the Office Assistant, click Search, and then click to view "Troubleshoot forms."
