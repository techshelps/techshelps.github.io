OL97: How to Import Act 2.x Files into Outlook

 Last reviewed: February 5, 1998
Article ID: Q167316
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article describes how you can import your Symantec ACT! 2.0 information into Microsoft Outlook 97.

MORE INFORMATION

Outlook does not provide a Symantec ACT! 2.0 converter; however, you can install an ACT! 2.0 import-only converter for Ou tlook from the Office 97 or Outlook 97 CD-ROM ValuPack folder. Once installed, the converter is available in the Import and Export Wizard for importing your ACT! 2.0 information into Outlook. You do not need ACT! 2.0 on your computer to use this import converter.

If you run Outlook from a CD-ROM, you cannot import or export from ACT! 2.0, even if you install converters from the Office 97 ValuPack.

For more information on how to use the ACT! 2.0 converter if you are running Outlook from a networked computer, please see the following article in the Microsoft Knowledge Base: 

   Article-ID:   Q16222
   Title     :   OL97: Contents of the Olreadme.txt Readme File
Follow these steps to install and use the ACT! 2.0 converter:

  1. Quit all programs.

  2. In the Office or Outlook CDROM ValuPack folder, double-click Valupk8.hlp to run the ValuPack help file.

  3. In the help topic, "What do you want to know about?" click "Microsoft Outlook import and export converters."

  4. In the help topic, "Microsoft Outlook import and export converters," click to install the Outlook import and export converters for your operating system.

    NOTE: This procedure installs additional converters for Symantec ACT! 2.0, NetManage ECCO Pro 3.0, 3.01, or 3.02, and Starfish SideKick 1.0/95. It also installs import and export converters for Outlook journal entries, notes or mail messages.

  5. When you receive the prompt, "Do you wish to install additional Import/Export translators?" click Yes and follow the prompt to install the translators.

  6. After the additional converters are installed, start Outlook.

  7. On the File menu, click Import and Export to start the Import and Export Wizard.

  8. Click to select, "Import from Schedule+ or another program or file", and click Next.

The Import and Export Wizard should now present you with the option to import ACT! 2.0 Contact Manager for Windows, information.

REFERENCES

For more information about transitioning from ACT! 2.0 to Outlook, type "act 2.0" in the Office Assistant, click Search, and then click to view "If you use Symantec ACT! 2.0."
