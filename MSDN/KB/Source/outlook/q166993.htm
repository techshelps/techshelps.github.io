OL97 Outlook Calendar Template for Word 97 Available on MSL

 Last reviewed: March 19, 1998
Article ID: Q166993
The information in this article applies to:
  • Microsoft Outlook 97
  • Microsoft Word 97 for Windows

SUMMARY

Olcalndr.exe contains a sample Microsoft Word 97 template for use with Microsoft Outlook 97 and Outlook 98. The template uses Visual Basic for Applications to automate creating Microsoft Outlook monthly calendars. It allows you to print the monthly view of your Outlook calendar, with or without weekends and in either portrait or landscape layout. Because the template uses the Word table format, you can easily modify your calendar before saving or printing. Using the Word for Windows "Save as HTML" converter, you can save the monthly calendar for publishing on a Web page.

NOTE: This template solution is designed to work only with U.S. versions of Microsoft Windows and Microsoft Office. It may not function correctly with non-U.S. versions of these products.
