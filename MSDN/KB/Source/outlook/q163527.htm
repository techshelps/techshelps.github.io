OL97: Cannot Print, Print Preview, or Delete Selected Group

 Last reviewed: January 19, 1998
Article ID: Q163527

The information in this article applies to:

  • Microsoft Outlook 97

SYMPTOMS

When you try to print, print preview, or delete in Microsoft Outlook 97, you may receive the following message: 

    This action will apply to all items in the selected groups.
If you click OK, the message closes without printing, print previewing, or deleting.

RESOLUTION

To work around this problem use a different view when grouping and selecting items. For example, the following steps allow you to delete all entries for a Contact from the Journal:

  1. In the Journal, select the Entry List view.

  2. On the View menu, click "Group By Box."

  3. Drag the Contact heading to the "Group By Box." This will group your Entry List by Contacts.

  4. Right-click the appropriate Contact name heading and from the context-
sensitive 
   menu, click Delete. This deletes all Journal entries for that particular
   Contact.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Outlook exhibits the above behavior under the following conditions:

  • You try to print, print preview, or delete items in your Journal, and you selected a group heading, instead of an individual item, and the current view is By Category or By Contact.
  • You try to print, print preview, or delete items in a mail folder, and you selected to group by categories, selected a group heading, and the current view is Message Timeline.

This behavior does not occur in Tasks, Contacts, Calendar, and Notes. You can successfully print or print preview when using equivalent settings in your Tasks, Contacts, Calendar, and Notes folders.

REFERENCES

For more information about setting views, type "views" in the Office Assistant, click Search, and then click to view "Views."

For more information about using print preview, type "views" in the Office Assistant, click Search, and then click to view "View pages in print preview."
