OL97: Microsoft Outlook Version Information

 Last reviewed: February 18, 1998
Article ID: Q172623
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article lists the different versions of Microsoft Outlook 97 and how you can determine which version you have.

MORE INFORMATION

This article describes version information for US versions of Outlook. The build numbers in this article are for US versions; in most cases international versions ship with later build numbers of Outllib.dll for the same release.

Version Information

Outlook 97 (8.0, 8.1 and 8.02)

Outlook 97, version 8.x, is a full-featured contact management program with calendar, scheduling, and contact management capabilities. Unlike Outlook Express, Outlook 8.x can use e-mail transports other than Internet Mail. For example, in addition to Internet mail, Outlook 8.x works with Exchange Server, Lotus cc:Mail, MSN Mail, Microsoft Mail, and other mail transport systems. Outlook 8.x can take full advantage of the capabilities of Microsoft Exchange Server mail systems.

Version 8.0

Outlook, version 8.0 is the original version. To obtain this version you must purchase either Office 97, Office 97 Small Business Edition, or the standalone version of Outlook 97.

Version 8.01

Outlook, version 8.01 is only available with Microsoft Exchange Server, version 5.0. To obtain this version you must purchase Microsoft Exchange Server. For more information on what features version 8.01 includes, please see the following article in the Microsoft Knowledge Base: 

   Article-ID: Q163743
   Title     : OL97: Information About Outlook Version 8.01

Version 8.02

Outlook, version 8.02, is available in three different configurations. You can obtain Outlook 8.02 from:

  • Office 97 Service Release 1 Enterprise Update (SR-1 Ent Upd)
  • Office 97 Service Release 1 Patch (SR-1 Patch)
  • Microsoft Exchange Server 5.0 Service Pack 1 (SP-1).

For information on which configuration of Outlook, version 8.02, is correct for you, please see the following article in the Microsoft Knowledge Base: 

   Article-ID: Q166024
   Title     : OL97: Outlook 8.02 Distribution Information
You can determine which configuration of Outlook 8.02 you have by checking for the existence of specific files that are installed by either the SR-1 Patch or SP-1 update tool and by viewing the properties of the Outllib.dll file to determine the create and modified dates.

Outlook, version 8.02 Configurations:

  • You have the SP-1 configuration of Outlook 8.02 if Help About shows Microsoft Outlook 97 (8.02.4212) and your hard disk contains the following file: 

          C:\Outlook\Client\Update.exe


             and/or

      The Outllib.dll file has a modify date of 6/12/97.
  • You have the SR-1 Patch configuration of Outlook 8.02 if Help About shows Microsoft Outlook 97 (8.02.4212) and your hard disk contains the following file: 

          C:\Program Files\Microsoft Office\Office\97sr1_<#>.txt


          where <#> is an integer that increments each time you run the patch.


             and/or

      The Outllib.dll modified date is 7/11/97.
  • You have the SR-1 Ent Upd configuration of Outlook 8.02 if the Help About shows Microsoft Outlook 97 (8.02.4212), and the Outllib.dll file's create and modified dates are both 7/11/97. The SR-1 Ent Upd updates the Outlook.exe file to 7/11/97, while the SR-1 Patch does not.

Version 8.03

Outlook, version 8.03 is only available with Microsoft Exchange Server, version 5.5. To obtain this version you must purchase Microsoft Exchange Server version 5.5. For more information on what features version 8.03 includes, please see the following article in the Microsoft Knowledge Base: 

   Article-ID: Q175676
   Title     : OL97: Summary of Changes in Outlook 8.03

Outlook Express

Microsoft Outlook Express is an update to Microsoft Internet Mail and News, which is distributed with Internet Explorer, version 3.x. Outlook Express is the Internet Explorer, version 4.0, equivalent of Microsoft Internet Mail and News. Outlook Express is not the same program as Outlook 97, versions 8.x.

Outlook Web Access

Outlook Web Access for Microsoft Exchange Server provides secure access to e-mail, and public folders on Microsoft Exchange Server using only a Web browser. For more information on Outlook Web Access, please see the following URL: 

   http://www.microsoft.com/outlook/documents/owa/web_acc.htm
