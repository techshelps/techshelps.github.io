OL97: Contents of the Outlook 8.03 ValuPack

 Last reviewed: January 19, 1998
Article ID: Q177603
The information in this article applies to:
  • Microsoft Outlook 97 version 8.03

SUMMARY

The ValuPack included with Outlook 97 version 8.03 contains items that enhance the functionality of Outlook as well as items not directly related to Outlook. The contents of the ValuPack differs from the ValuPack supplied with Outlook 97 version 8.0 and Office 97.

MORE INFORMATION

Installing ValuPack Components

In some cases there are special packages for Windows 95 and Windows NT. If you are using Windows NT, special care should be taken to select the file names which are appended with the letters "nt". For example, the Outlook converter package name for Windows 95 is "outcvt", while the package for Windows NT is named "outcvtnt."

To install the ValuPack components, follow these steps:

  1. Locate the appropriate subfolder within the ValuPack folder.

  2. Run Setup directly from the folder.

The following section outlines the items based on folder name and content. 

 Folder      Contents
 ------      --------

 CCMail      cc:Mail transport services for Windows 95 and Windows NT

 Convert     Personal Information Manager converter software for Windows 95
             and Windows NT

 Cursors     Animated Cursors for Windows 95 and Windows NT

 DataAcc     Installer for ODBC and other drivers for database conversion

 DCom95      Distributed Component Object Model elements (newest OLE
             functionality)

 FarEast     Support for 4 Far East language character sets, Japanese,
             Korean, Traditional Chinese and Simplified Chinese.

 Iexplore    Setup for Microsoft Internet Explorer version 3.02

 MSCam       Microsoft Camcorder to record actions, procedures, and
             sounds from your computer.

 MSFonts     150 additional TrueType fonts to help you create, view,
             and print great looking documents.

 Odbc        ODBC 3.0 Driver Manager has been updated with enhanced
             security.

 OffClean    The Upgrade Wizard that originally shipped with Office.

 Patch       Software patches originally shipped with Office97.

 RulesWiz    The Outlook Rules Wizard is a client/server
             messaging manager that allows users to manage their e-mail
             communication through defined inbox rules.

 Sounds      Add fun audio cues to screen elements as you work with them.
            Not available for Windows NT Workstation 3.51 or Windows NT
            Server 3.51.

 Template    A collection of Outlook forms for Windows 95 and Windows NT.
             Also included are PowerPoint Designs and Templates

 Timex       Timex Data Link Wizard for sharing appointment and other
             information on the Timex Data Link watch.
NOTE: The Microsoft Outlook Visual Basic Help file (Vbaoutl.hlp) is not available in the Microsoft Outlook version 8.03 ValuPack as stated in the VBScript Editor Help file.
