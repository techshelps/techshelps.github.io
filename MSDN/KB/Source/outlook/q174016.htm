OL97: Error in "Download Offline Address Book" Help Topic

 Last reviewed: January 19, 1998
Article ID: Q174016
The information in this article applies to:
  • Microsoft Outlook 97, version 8.03

SYMPTOMS

In Microsoft Outlook 97 version 8.03, the Help topic for "Download Offline Address Book", may not match the dialog box options.

CAUSE

The Help topic for this dialog box was not updated from the previous version of Outlook.

RESOLUTION

While the topics listed in the Help file do not match the option titles in the dialog box, the information provided is still correct.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97, version 8.03. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. On the Tools menu point to Synchronize and click, Download Address Book.

  2. Click Help. You will only see information on two options.

Results

The Outlook 8.03 dialog box contains 3 options. 

    - Download changes since last synchronization: This new option
      provides for downloading only changes to the address book rather
      than the whole address book file.

    - Full details: This option provides for downloading all information
      for the address book entries and is required for sending encrypted
      messages.

    - No details: This option provides for downloading only minimal
      information for the address book entries and is faster than
      option #2 above.
The Help file topic contains a summary and 2 options: 

    - "Download offline address book." This is the same as the Full details
      option above.

    - "Download offline address book without details information." This is
      the same as the No details option above.
