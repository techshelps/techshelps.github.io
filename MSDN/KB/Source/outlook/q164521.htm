OL97: Outlook Related ValuPack Components

 Last reviewed: January 19, 1998
Article ID: Q164521
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

The ValuPack, is a set of product enhancement components, on the Microsoft Office 97 Compact Disc. It contains several components related to Microsoft Outlook 97. This article describes those Outlook-related components.

MORE INFORMATION

Installing ValuPack Components

To install the ValuPack components, do the following:

  1. Insert your Microsoft Office 97 Compact Disc in the CD-ROM drive. If the CD does not Autorun, right-click the Start button, click Explore, and then click your CD-ROM drive.

  2. Double-click the ValuPack folder and then double-click the ValuPack Help file (Valupk8.hlp).

A list of the ValuPack components will be displayed. To access information about each component, double-click its name. Each component's information page contains a button for installing the component.

Those ValuPack components relating to Outlook are described below:

Data Access Pack

The Data Access Pack provides a set of drivers (ODBC and IISAM) to access external data from Microsoft Access and Microsoft Excel. One set of these drivers provides read-only access to Microsoft Exchange and Outlook mail message stores.

Lotus cc:Mail Information Service

The Lotus cc:Mail Information Service, provides a Messaging Application Programming Interface (MAPI) that allows the use of Outlook as an e-mail client to a cc:Mail postoffice.

For more information on Lotus cc:Mail, please see the following article in the Microsoft Knowledge Base: 

   Article-ID:   Q163290
   Title     :   OL97: Troubleshooting cc:Mail Connectivity in Outlook

Office 97 Sounds

The Office 97 Sounds, provide audible cues to various program functions in any Office 97 programs including Outlook. These sounds are not available for Windows NT Workstation 3.51 or Windows NT Server 3.51.

Office Templates, Forms, and Wizards

The Office Templates, Forms, and Wizards, provide help in creating Great looking Office documents and Outlook items. Those relating to Outlook, include templates for Notes and Journalizing as well as forms for Classified Ads, Sales Tracking, Training Management, Vacation Request, and While You Were Out.

Outlook Import and Export Converters

The Outlook Import and Export Converters, provide programs for Importing information from various Personal Information Manager (PIM) programs such as Symantec Act! 2.0; NetManage ECCO Pro 3.0, 3.01, or 3.02; and Starfish Sidekick 1.0 and 95. Also included are programs for importing and exporting information to and from Outlook journal entries, notes, and mail messages.

For more information on import and export converters, please see the following article in the Microsoft Knowledge Base: 

   Article-ID: Q158084
   Title     : OL97: Converters Included with Outlook

Outlook Software Patches

The software patches include a patch to replace the Exchange Inbox icon on the Windows desktop with an Outlook icon with similar properties that gives access to profile information.

For more information on replacing the Inbox icon, please see the following article in the Microsoft Knowledge Base: 

   Article-ID: Q160297
   Title     : OL97: How to Change Inbox Icon to Start Microsoft Outlook 97

Timex Data Link Watch Wizard

The Timex Data Link Watch Wizard, provides for exporting Outlook appointments, tasks, phone numbers, anniversaries, birthdays, reminders, time zone information, and current time, to the Timex Data Link Watch.

For more information on the Timex Watch Wizard, please see the following articles in the Microsoft Knowledge Base: 

   Article-ID: Q158084
   Title     : OL97: List of Converters Included with Microsoft Outlook

   Article-ID: Q160989
   Title     : OL97: Timex Watch Will Not Accept Extended Characters
