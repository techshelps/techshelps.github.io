OL97: Msg: No Recipients Have Reported Reading This Message

 Last reviewed: January 19, 1998
Article ID: Q177811
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you attempt to recall a sent message in Microsoft Outlook 97, you may receive the following message: 

   No recipients have reported reading this message.
This message may be erroneous. Some recipients may have read the message.

CAUSE

The message was sent without enabling Read Receipts.

RESOLUTION

Enable Read Receipts on the message before sending. To do this for a specific message, follow these steps:

  1. Prior to sending the message, click the Options tab.

  2. In the Tracking Options section, click to select "Tell me when this message has been read."

To enable Read Receipts for all messages:

  1. On the Tools menu, click Options.

  2. On the Sending tab, in the Tracking Options section, click to select "Tell me when all messages have been read."

RESULTS: If a recipient has read the message, the Recall This Message dialog box will read: 

   At least <number> recipients have already read this message.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce the Problem

  1. Compose and send a message to yourself.

  2. Read and then close the message.

  3. In the Sent Items folder, click to open the sent message.

  4. On the Tools menu, click to select Recall This Message.

RESULTS: The Recall This Message dialog box will read: 

   No recipients have reported reading this message.
