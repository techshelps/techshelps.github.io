OL97: ErrMsg Exporting Data to Timex DataLink Watch

 Last reviewed: January 19, 1998
Article ID: Q170656
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you attempt to Export data from Microsoft Outlook 97 to a Timex DataLink watch, you may receive the following error message: 

   Too much data to fit on CRT - reduce using '/NC' or '/bb' options.

CAUSE

This error messages occurs when you install Microsoft Office 97 to run from a CD-ROM or networked computer, and you install the Windows 95 version of the Timex software.

RESOLUTION

Install the Windows NT Workstation 3.51 version of the Timex software.

MORE INFORMATION

There are two versions of the Office 97 ValuPack Setup program that install files for the Timex DataLink Watch. One version is for Windows 95 and Windows NT Workstation 4.0; the other version is for Windows NT Workstation 3.51.

If you installed Outlook to run from a CD-ROM or a networked computer, install the Windows NT Workstation 3.51 version of the Watch Wizard Setup program.

To install the Timex Data Link Watch Wizard, run one of the two setup programs located in the \ValuPack\Timex folder on the Microsoft Office 97 CD-ROM.

For Windows NT 3.51, run: 

   Timexnt.exe
For Windows 95 or Windows NT 4.0, run: 

   Timexwiz.exe
The Timex DataLink watch is manufactured by Timex, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
