OL97 DocErr: Can't Automatically Add Faxes to the Journal

 Last reviewed: January 19, 1998
Article ID: Q162038
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

In the "What is journal?" Help topic (under "Using Journal" on the Contents tab), "Automatically record activities" displays incorrect information. The topic's picture displays "Fax" as an item that is automatically recorded. However, Outlook does not have the capability to automatically journal faxes.

This documentation error also occurs on page 121 of the "Getting Results with Microsoft Office 97" book.

This documentation error also occurs on page 81 of "Microsoft Outlook 97 Field Guide" (Microsoft Press).

MORE INFORMATION

To locate this documentation error using the Office Assistant, follow these steps:

  1. Type "journal" in the Office Assistant, click Search, and then click to view "Journal."

  2. In the Help topic, click "Automatically record activities." Note that under "Automatically record these items, "Fax" is displayed as an option.

REFERENCES

For more information about items that you can automatically add to the journal, type "journal" in the Office Assistant, click Search, and then click to view "Activities that can automatically be recorded in Journal."
