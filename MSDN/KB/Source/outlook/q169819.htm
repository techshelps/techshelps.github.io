OL97: Creating a From Rule for an Internet Mail Address

 Last reviewed: March 12, 1998
Article ID: Q169819
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

In Microsoft Outlook 97, Internet Mail addresses in the "From" field of Inbox Assistant rules or Rules Wizard rules may not work correctly. This article describes how to define Internet Mail addresses in the From field of the rule so that they are correctly recognized.

MORE INFORMATION

The information in the incoming message header may include more than just the Internet address in the From field. Therefore, typing or pasting just the Internet address into the From field may result in an incorrect compare and the address may not be recognized.

There are three methods to create a From rule for Internet Mail.

  • Use an entry in the Personal Address Book.
  • Use an entry in the Contact list.
  • Use the Create Rule feature. (Not available with Inbox Assistant)

Use an Entry in the Personal Address Book

Create an entry for the Internet addressee in the Personal Address Book as follows:

  1. On the Tools menu, click Address Book.

  2. In "Show Names from the" list, click Personal Address Book.

  3. On the File menu, click New Entry.

  4. In "Select the entry type" list, click Internet Address and click OK.

  5. In Display name, type the addressee's name.

  6. In E-mail Address, type or paste the addressee's e-mail address.

  7. Click OK to close the Address Book.

When defining the From rule, use the addressee's Display name as entered in the Personal Address Book instead of the Internet Address.

Use an Entry in the Contact List

Create an entry for the Internet addressee in the Contact list as follows:

  1. On the File menu, point to New and click Contact.

  2. Type the addressee's name in Full Name box.

  3. Type or paste the Internet address in the E-mail box.

  4. Enter any additional information and click Save and Close.

When defining the From rule, use the addressee's full name as entered in the Contact record instead of the Internet Address.

Use the Create Rule Feature

To use the Create Rule feature do the following:

  1. Open a message to which the rule is to apply.

  2. On the Tools menu, click Create Rule.

This inserts the correct Internet Mail address in the From field.
