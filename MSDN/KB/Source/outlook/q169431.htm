OL97: Word 97 Document to Change Outlook Folder Message Class

 Last reviewed: February 27, 1998
Article ID: Q169431
The information in this article applies to:
  • Microsoft Outlook 97
  • Microsoft Word 97 for Windows

SUMMARY

The Microsoft Word 97 Outlook Change Message Class document is a Word 97 document that uses Office automation to change the message class for all items in a Outlook 97 MAPI folder. With Outlook, you can create a new form for items such as Contacts and set a Contacts folder property so that all new Contacts use the new form. However, old Contacts will still use the old form. The Omsgclas.doc document allows you to change the message class held for each item in the folder so that all existing items will use a specific custom form. In addition to Contacts, this document works with other Outlook item folders.

The following files is available for download from the Microsoft Software Library: 

 ~ Omsgclas.exe (size: 40242 bytes)
