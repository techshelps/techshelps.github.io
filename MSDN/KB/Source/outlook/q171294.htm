OL97: HTML Formatted Mail Is Received as a Shortcut Icon

 Last reviewed: January 19, 1998
Article ID: Q171294
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

An e-mail message you received in Microsoft Outlook 97 consist of a blank message with a shortcut icon that, when you click it, starts your default Web Browser.

CAUSE

The message you received is formatted in Hypertext Markup Language (HTML) and Outlook does not read HTML.

MORE INFORMATION

Because the current version of Outlook does not read HTML, it attaches a shortcut to the document. Double-clicking the shortcut opens the message in your default Web Browser.
