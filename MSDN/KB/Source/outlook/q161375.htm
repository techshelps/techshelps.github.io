OL97: Err Msg Using Send Options by Using Right Mouse Button

 Last reviewed: January 19, 1998
Article ID: Q161375
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you use the right mouse button to click a resolved recipient name, and then click Send Options on the shortcut menu, you may receive the following error message: 

   Cannot perform the requested operation. The command selected is not
   valid for this recipient. To set Send Options for this message, click
   Properties on the File menu.

CAUSE

The Outlook Address Book and the Personal Address Book used with Outlook do not have the ability to provide send options for a particular address.

MORE INFORMATION

Addresses for an Outlook item are entered from a variety of sources, including the Address Books. An Address Book can potentially include send options for a particular name.

Currently, the typical address books (Outlook Address Book and Personal Address Book) used by Outlook do not provide sending options; therefore, for practically all addresses, you may receive the error message mentioned in the "Symptoms" section.
