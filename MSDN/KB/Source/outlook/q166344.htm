OL97: File Attachments Will not Open When Using Wordmail

 Last reviewed: January 19, 1998
Article ID: Q166344
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

You can not open a file attachment in an e-mail message if the following two conditions are true:

  • You are using Word as your e-mail editor.
  • The message was originally created with the sensitivity was set to Private.

RESOLUTION

To open the file attachment, disable Microsoft Word as your e-mail editor with the following steps:

  1. Start Outlook.

  2. On the Tools menu, click Options.

  3. On the E-mail tab, clear the "Use Microsoft Word as the Email Editor" check box, and then click Close

  4. Open the message containing the attached file.

  5. Double-click the attachment to open it.

MORE INFORMATION

Steps to Reproduce Problem

  1. Compose a new mail message using Word as the email editor.

  2. Attach a Word document or Microsoft Excel worksheet.

  3. Under the Options tab, set the sensitivity to Private.

  4. Send the message to yourself.

  5. Open the message and double-click the file attachment.

Result: The file attachment is not opened.

NOTE: This is not a problem when using the Microsoft Exchange Client with Word as the e-mail editor.
