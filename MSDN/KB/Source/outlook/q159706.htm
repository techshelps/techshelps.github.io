OL97: Using Microsoft Outlook with Different Mail Systems

 Last reviewed: January 19, 1998
Article ID: Q159706
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

Although it is optimized for use with the Microsoft Exchange Server, you can use Microsoft Outlook 97 with different mail systems. You can also use Outlook without a mail system and still take advantage of its personal information management features.

MORE INFORMATION

The table below indicates which mail systems are supported by Outlook. Depending on the mail system you use, some Outlook features may not be available. Only the Microsoft Exchange Server mail system allows you to use the entire Outlook feature set. 

Mail Service       Installed From

Internet Mail      Outlook via Typical Setup
MS Mail            Outlook via Typical Setup
Exchange Server    Outlook via Typical Setup
CompuServe         CompuServe Setup
MSN                MSN Setup
CC Mail            Office 97 ValuPack
Outlook is designed to work with any MAPI-compliant mail provider. However, several MAPI-compliant mail systems do not support enough of the MAPI specification to successfully run Outlook. Using a limited MAPI-compliance mail system with Microsoft Outlook is not supported by Microsoft; you must contact the vendor of the mail system for technical support.
