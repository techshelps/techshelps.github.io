OL97: How to Remove Page Border From Table Print Style

 Last reviewed: January 19, 1998
Article ID: Q161098
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

When you print based on the Table Style, Microsoft Outlook 97 automatically adds a page border. This article explains how to remove the page border from the Table Style.

MORE INFORMATION

Outlook adds a page border to the Table Style whenever you include header or footer information. By default, the header is empty and the footer includes the user's name, page number, and current date. The border disappears if you remove all header and footer information.

The following steps demonstrate how to print from the Table Style without a page border:

  1. Open the Inbox.

  2. On the Current View drop-down list, click the Messages View.

  3. On the File menu, click "Page Setup And Table Style" to open the Table Style dialog box.

  4. Click the Header/Footer tab to activate the header and footer settings.

  5. With the pointer, select and delete all header and footer entries.

  6. Click the Print button.
