OL97: Embedded Message Window Too Small and Cannot Be Changed

 Last reviewed: January 19, 1998
Article ID: Q165573
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you open an embedded mail message in Microsoft Outlook, the frame size may appear small and you may not be able to resize the window.

CAUSE

This problem occurs when either of the following conditions is true:

Case 1: The Microsoft Exchange mail client previously installed on the same computer had WordMail set as the default e-mail editor.

Case 2: Your forms registration is set for Exchange instead of Outlook.

RESOLUTION

To resolve this problem, use the appropriate method for your situation.

Case 1: Turn off WordMail in the Exchange client.

Case 2: Run the SwitchForms utility to register client forms with Outlook. This utility is included in the Microsoft Office 97 Resource Kit. It is also available as "Switch Forms Utility for E-mail" from the following Microsoft World Wide Web site: 

   http://www.microsoft.com/officefreestuff/outlook/default.htm
When you run SwitchForms, select install forms for Microsoft Outlook.

Microsoft also has a new utility that allows you to switch forms interactively between the two e-mail clients. The new utility is called FormsFix and is available on the same page on the Microsoft World Wide Web site (click "E-mail Forms Fix Utility").

The code for FormsFix is also included in the Outlook components patched by the Microsoft Office Service Release 1 (SR-1) Patch.

MORE INFORMATION

This problem only occurs on computers that have both the Microsoft Exchange client and Microsoft Outlook 97. Forms are registered with the last client program installed.

For more information on SwitchForms and FormsFix, please see the following articles in the Microsoft Knowledge Base:
