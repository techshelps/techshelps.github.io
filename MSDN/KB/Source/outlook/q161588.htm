OL97: Outlook Bar Use Requires a Pointing Device

 Last reviewed: January 19, 1998
Article ID: Q161588

The information in this article applies to:

  • Microsoft Outlook 97

SYMPTOMS

You do not have a mouse or other pointing device and you want to use the Microsoft Outlook 97 Outlook Bar to move through folders. You can find no shortcut keys for use with the Outlook Bar.

CAUSE

The Outlook Bar is designed for use with a pointing device. No shortcut keys are available for use with the Outlook Bar.

WORKAROUND

You can use the Go menu shortcut key to move through any folder on your system.

To open a folder without the use of a pointing device, follow these steps:

  1. Press ALT+G to open the Go menu.

  2. Press G to open the Go To Folder dialog box.

  3. Press TAB to access the Folder Name list, which includes all folders on the Outlook Bar.

    NOTE: The Folder Name list includes all folders on the Outlook Bar. If you press TAB a second time, you can select from a hierarchical list that includes all available folders.

  4. Press the UP ARROW or DOWN ARROW key to select the desired folder.

  5. Press ENTER to open the folder.

REFERENCES

For more information about shortcut keys, type "shortcut keys" in the Office Assistant, click Search, and then click to view "Shortcut Keys."
