OL97: Last Used Folder Moves to the Top of the Folder List

 Last reviewed: January 19, 1998
Article ID: Q168235
The information in this article applies to:
  • Microsoft Outlook 97
  • Microsoft Exchange Server, version 5.0

SYMPTOMS

After you open a mail message from any of your Microsoft Outlook 97 mailbox folders, the folder may move to the top of your Folder List. When you receive mail in your Inbox, the Inbox folder may move to the top of the Folder List. The last folder worked on may move to the top of the Folder List. This behavior only occurs with Mailbox folders, not Personal Folders.

CAUSE

The language setting on the computer running Outlook is different from the Exchange Server.

RESOLUTION

Set the Windows Regional Settings language on the client computer to match the language on the Exchange Server. Follow these steps to set the language on the client computer:

  1. On the Windows Taskbar, click Start and then Settings and Control Panel.

  2. In the Control Panel, double-click Regional Settings.

  3. On the Regional Settings tab, click to select the language that matches your Exchange Server language and then click OK.

If you change the language, you must restart your computer before the change takes effect.
