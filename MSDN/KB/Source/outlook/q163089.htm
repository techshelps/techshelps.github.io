OL97: Sent Mail Disappears with Focal Point Extensions Loaded

 Last reviewed: January 19, 1998
Article ID: Q163089
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you send a message using Microsoft Outlook 97, the message may seem to disappear. It does not appear in the Outbox or Sent Items folders.

CAUSE

This symptom may occur if you have Global Village Communication's Focal Point extensions on your system. Focal Point is software that integrates e-mail, voicemail, fax paging, data communications and Internet access. Focal Point software can come preinstalled on your computer.

RESOLUTION

You can use the Outlook Add-In Manager to disable the Focal Point extensions. To disable Focal Point extensions, follow these steps:

  1. On the Outlook Tools menu, click Options to open the Options dialog box.

  2. Under the General tab, click the Add-In Manager button to open the Add-Ins dialog box.

  3. In the list of add-ins, click to clear any add-in that references your Focal Point software.

  4. Click OK and OK to close all dialog boxes.

  5. On the File menu, click "Exit and Log Off."

  6. Press CTL+ALT+DELETE and view the Windows Task Manager.

  7. Make sure that mapisp32.exe has stopped running. If it is still running, either wait for it to stop or end the task.

  8. Restart Outlook.

REFERENCES

For more information about using the Add-In Manager, type "add-in manager" in the Office Assistant, click Search, and then click to view "Load, install, or remove an add-in."
