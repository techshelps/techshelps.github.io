OL97 ErrMsg: OLE Automation Error With Timex Data Link

 Last reviewed: January 19, 1998
Article ID: Q164533
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you start the Outlook 97 Import and Export Wizard and choose "Export to Data Link watch," you may receive the following error message: 

   OLE Automation Error: Unable to connect to server

CAUSE

You may have an old version of the OLE files installed on your computer.

RESOLUTION

You can resolve this problem by using the Maintenance Mode of Microsoft Office 97 or Outlook 97 Setup and selecting Remove All. Run Setup again to install Office or Outlook with the correct OLE files.

MORE INFORMATION

You may encounter this problem if you install an OLE patch for Windows after installing Office 97. Some patches may contain older OLE files than Office 97 programs provide.
