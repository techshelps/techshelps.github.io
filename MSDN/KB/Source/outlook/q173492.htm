OL97: Error in "When I type a name in all lowercase letters..."

 Last reviewed: January 19, 1998
Article ID: Q173492
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

In the "When I type a name in all lowercase letters, it doesn't stay that way" Help topic, the instructions for forcing the letters to remain in lowercase are incorrect. The single quotation mark is not automatically removed. You get different results if you use the TAB key to move to the next field than if you click in the next field. If you type a space between names or characters, the lowercase reverts to initial capital letters.

MORE INFORMATION

To locate this documentation error, type "contacts" in the Office Assistant, click Search, click to view "Troubleshoot contacts," and then click "When I type a name in all lowercase letters, it doesn't stay that way."
