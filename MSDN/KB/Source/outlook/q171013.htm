OL97: Added Internet E-mail Servers May Not Be Selected

 Last reviewed: January 19, 1998
Article ID: Q171013
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

Using the Internet Mail Enhancement Patch for Microsoft Outlook 97 (IMEP), when you add additional Internet Mail servers to your active profile, the additional servers may not be selected to check for new mail.

RESOLUTION

Before checking for new mail, make sure the added e-mail servers are selected.

  1. On the Tools menu, click "Check for New Mail On."

  2. Click to select any Internet E-mail servers that are not selected.

  3. Click OK.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Start Outlook using a profile that contains the IMEP service.

  2. On the Tools menu, click Services, and then click Add.

  3. Select Internet E-mail, click OK, and type the information for a new mail account.

  4. On the File menu, click "Exit and Log Off."

  5. Start Outlook using the same profile.

  6. On the Tools menu, click "Check for New Mail On."

The added Internet E-mail server may not be selected.
