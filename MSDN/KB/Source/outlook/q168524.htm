OL97: Outlook 8.01 Not in Select Update Kit for April 1997

 Last reviewed: January 19, 1998
Article ID: Q168524
The information in this article applies to:
  • Microsoft Outlook versions 8.01, 97

SUMMARY

The English version of Microsoft Outlook 97, version 8.01, was not included in the Select CD-ROM April 1997 Update Kit.

MORE INFORMATION

The April 1997 Select Update Kit is the first kit that included Microsoft Exchange, version 5.0. The English version of Exchange, version 5.0, will work with the standalone English version of Outlook 97, version 8.0, that was included in the February 1997 Select CD Kit in the application pool.

The May 1997 Select CD-ROM Update Kit includes Outlook 97, version 8.01, for Exchange, version 5.0, in both the application and server pools.
