OL97: Error in Outlook 97 Using Spelling Checker

 Last reviewed: January 19, 1998
Article ID: Q162083

The information in this article applies to:

  • Microsoft Outlook 97

SYMPTOMS

When you try to use the spelling checker feature in Microsoft Outlook 97, you may receive the following error message: 

   Error occurred while checking spelling. The shared Spelling
   component is configured incorrectly. Please run setup again.
This error does not occur in other Microsoft Office 97 or Microsoft Office 95 programs.

CAUSE

One or more of the Microsoft Office 97 spelling files are corrupted or you did not choose to install them.

RESOLUTION

You can correct this problem by using the Maintenance mode of Office 97 Setup to remove and/or add the Office 97 spelling files.

To Remove the Office 97 Spelling Files

  1. Run Office 97 Setup and click Add/Remove to open Maintenance Mode Setup.

  2. Click to select Office Tools, and click Change Option to open the Office Tools dialog box.

  3. Click to clear Spelling Checker check box.

  4. Click OK, and then Continue.

  5. Click the Windows 95 Start button, click Find, then Files or Folders.

  6. Search for the file mssp232.dll. If it is found, rename it to mssp232.old.

  7. Search for the file mssp2_en.lex. If it is found, rename it to mssp2_en.old.

To Install Office 97 Spelling Files

  1. Run Office 97 Setup and click Add/Remove to open Maintenance Mode Setup.

  2. Click to select Office Tools, and click Change Option to open the Office Tools dialog box.

  3. Click to select Spelling Checker.

  4. Click OK, and then Continue.

MORE INFORMATION

Outlook 97 cannot use the Office 95 spelling files; however, all other Office 97 programs can use the Office 95 spelling files, and all Office 95 programs can use the Office 97 spelling files.

REFERENCES

For more information about spell checking, type "spelling" in the Office Assistant, click Search, and then click to view "Use the spelling checker."
