OL97: Missing Date Completed Data on Task Export

 Last reviewed: January 19, 1998
Article ID: Q166405
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When exporting Task records from Microsoft Outlook 97, the "Date Completed" field is empty. This behavior occurs with all Export options and file types.

WORKAROUND

There is no known workaround.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. On the File menu, Click Import and Export.

  2. Click "Export to a file," and click Next.

  3. Click the Task folder and click Next.

  4. Click any file type option and click Next.

  5. Type a name for the Export file and click Next.

  6. Click Finish, type an appropriate date range and click OK.

RESULTS: View the exported data file. The Date Completed field will be present but will contain no data.
