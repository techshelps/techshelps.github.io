OL97: Errors in Russian Section of the Holiday File

 Last reviewed: January 19, 1998
Article ID: Q170742
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

There are two errors in the Russian section of the Microsoft Outlook 97 Holiday file, Outlook.txt:

  • Orthodox Christmas, is currently scheduled as recurring each April 27, but should be scheduled as recurring each January 7.
  • Orthodox Easter, is not currently scheduled but should be specifically scheduled on April 27, 1997, and thereafter on every first Sunday after the first full moon after Passover.

MORE INFORMATION

Outlook holidays are stored in a text file named Outlook.txt. For more information on Outlook holidays, please see the following articles in the Microsoft Knowledge Base:
