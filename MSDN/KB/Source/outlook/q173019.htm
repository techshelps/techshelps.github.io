OL97: CC:Mail Service Versions Shipped with Versions of Outlook

 Last reviewed: January 19, 1998
Article ID: Q173019
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article describes which versions of the cc:Mail transport service are packaged with the various versions of Microsoft Outlook 97.

MORE INFORMATION

 cc:Mail    Outlook version                  Required VIM files
 --------------------------------------------------------------------------
 Build 42   8.0                              Vdlwin.zip (Windows 95)
                                            Vdlw32.zip (Windows NT)

 Build 48   8.02                             Vdlw32.zip (Windows 95 and NT)
           Exchange Server 5.0 SP 1
           Office 97 SR1 Enterprise Edition
To download build 48 of the cc:Mail transport service, please see the following Microsoft World Wide Web site: 

   http://www.microsoft.com/OfficeFreeStuff/Outlook/dlpages/CCMailSp.htm
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
To download the required VIM files, please see the following Lotus Development Corporation World Wide Web site: 

   http://www2.support.lotus.com/ftp/pub/comm/ccmail/dev_tools/
NOTE: Build 48 of the cc:Mail transport service contains updated dual-mode VIM files in the Vdlw32.zip file. This build fixes the following error message when starting Outlook: 

   Invalid Page Fault in Keyboard.drv.
The cc:Mail transport service is manufactured by Transend Corporation, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

The VIM files are manufactured by Lotus Development Corporation, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
