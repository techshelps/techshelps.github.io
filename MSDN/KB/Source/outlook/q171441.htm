OL97: Notes Field Incomplete in Printed Contact

 Last reviewed: January 19, 1998
Article ID: Q171441
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you print a Microsoft Outlook 97 Contact, the Note field may not be complete.

CAUSE

Outlook prints only the first 255 characters of the Contact's Note field in all print styles, except the Memo Style.

RESOLUTION

To print the complete contents of the Contact's Note field, use the Memo Style to print the Contact. To do so, follow these steps:

  1. On the File menu, click Print.

  2. From the Print Style window, click Memo Style, and then click to select "Only selected items."

  3. Click OK to print the Contact.

RESULTS: Outlook prints the complete contents of the Note field for the Contact.

NOTE: The Phone Directory Style prints no Note field at all.

MORE INFORMATION

Steps to Reproduce Behavior

  1. In the Contacts folder, switch to Detailed Address Cards view.

  2. On the File menu, point to New, and then click Contact.

  3. Set up a new contact with a large amount of text (more that 255 characters) in the Notes field (the large unlabeled window at the bottom of the input form).

  4. Click Save, and then Close.

  5. Click to select the new Contact, and then, on the File menu, click Print.

  6. From the Print Style window, click any style, except Memo or Phone Directory Style, and then click to select "Only selected items."

  7. Click OK to print the Contact.

RESULT: Outlook prints only 255 characters of the Note field for the Contact.
