OL97: Printing Error "The Operation Failed"

 Last reviewed: January 19, 1998
Article ID: Q166493
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you attempt to print another user's calendar from within Microsoft Outlook 97, you may receive the following error message: 

   The Operation Failed.

CAUSE

You do not have read permission to the Task list and you are trying to read the other user's calendar using the Day/Week/Month view.

RESOLUTION

To resolve this problem, do one of the following:

  • Change the view that is being printed to something other than Day/Week/Month. 

        -or-
  • Have the other user give you permissions to the Task folder. 

        -or-
  • Change the page setup for the Daily Style so that it does not include the Task Pad. This can be done by following these steps:

    1. On the File menu, click Print.

    2. Click the Daily Style to select it and click Page Setup.

    3. Under Options, click to clear the Task Pad check box.
