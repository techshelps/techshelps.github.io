OL97: Cannot Add Shortcut for a Single File to Outlook Bar

 Last reviewed: January 19, 1998
Article ID: Q154296
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

In Microsoft Outlook 97, you cannot add a shortcut to the Outlook Bar for a single file or program executable.

CAUSE

Outlook Bar shortcuts can only point to file system folders or MAPI folders.

WORKAROUND

To improve access to frequently used individual files and programs:

  1. Create a file system folder containing shortcuts to your frequently used files or programs. 

        a. Click the My Computer icon in the Other group on the Outlook Bar.


        b. From the File List, select the location for the new Shortcuts
       Folder.


        c. On the File menu, point to New, and then click Folder.


        d. Type an appropriate name for the folder, and then click OK.


        e. Drag the new folder to the Outlook Bar to create an icon.

  2. Create shortcuts to your frequently used files or programs in the new folder. 

        a. In Outlook, locate the file or program you want to use.


        b. Select the appropriate file or program executable and drag it to the
       new icon on the Outlook Bar.

To use the new shortcut folder, click the icon on the Outlook Bar, and Then double-click the file or program in the Outlook view window.
