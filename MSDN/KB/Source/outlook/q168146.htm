OL97: Expression Service Error Using Timex Data Link Watch

 Last reviewed: January 19, 1998
Article ID: Q168146
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When exporting information to the Timex Data Link Watch in Microsoft Outlook 97, you may receive the following error: 

   Failed to start the expression service.  The Expression service
   may not be installed.  For Information about how to install
   conponR[HN
NOTE: This error message seems incomplete, however, the exact text above is populated in the message dialog.

CAUSE

This error may occur because of a missing or corrupted Vbajet32.dll.

RESOLUTION

Copy a new Vbajet32.dll file from your Outlook setup discs to the Windows\System folder on your computer, as follows:

  1. Quit all running programs.

  2. Click Start on the Windows Taskbar, point to Find and click Files or Folders to search for Vbajet32.DLL and rename it. 

        a. Click the file name to select it and on the File menu click
       Rename.


        b. Type a new name for the file, such as, vbajet32.old.

  3. On your setup disc, navigate to the following folder: 

          \OS\System\Vbajet32.dll

  4. Copy the Vbajet32.dll file to the following folder on your computer: 

          \Windows\System (if you are running Windows 95)

    -or- 

          \Winnt\System32 (if you are running Windows NT)

  5. Right-click the Vbajet32.dll file you copied, and click Properties on the shortcut menu.

  6. Click Read-only to deselect it, and click Archive to select it.
