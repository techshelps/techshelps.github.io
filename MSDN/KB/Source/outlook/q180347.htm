OL97: MAPI Service Error in GWMSPI.DLL Starting Outlook

 Last reviewed: February 9, 1998
Article ID: Q180347
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you start Microsoft Outlook 97, the following error message appears: 

   Unable to open your default mail folders. MAPI was unable to load the
   information service GWMSPI.DLL
If you click OK, Outlook fails to start.

CAUSE

You have installed client support for Novell GroupWise and Outlook 97 on the same computer and messaging services for both products were installed in the same profile.

RESOLUTION

Create two separate Windows Messaging profiles, one for Outlook and one for GroupWise.

MORE INFORMATION

The error message described in the Symptoms section occurs without respect to the installation order.
