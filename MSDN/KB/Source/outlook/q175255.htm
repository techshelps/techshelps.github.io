AutoArchive Generates Error when Personal Folders Disabled

 Last reviewed: January 19, 1998
Article ID: Q175255
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When AutoArchive starts in Microsoft Outlook 97, you receive a message similar to the following: 

   Unable to archive one or more folders. An error occurred while
   performing AutoArchive operations.
In addition, if several messages are selected for manual archiving, you receive a message similar to the following: 

   The message interface has returned an unknown error. If the problem
   persists, start Outlook again.

CAUSE

There is no Personal Folders service in the profile.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97. This problem was corrected in Outlook 97 version 8.03.

MORE INFORMATION

There are features in Outlook that require the use of the Personal Folders service and the presence of the associated .pst files; the AutoArchive feature is an example.
