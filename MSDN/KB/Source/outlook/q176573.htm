OL97: Setting Delayed Delivery Options

 Last reviewed: January 19, 1998
Article ID: Q176573
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

If you send and receive e-mail using Microsoft Exchange Server and your e- mail is delivered to your mailbox on the server, you can compose e-mail messages in Microsoft Outlook 97 and delay sending the messages.

MORE INFORMATION

Use either of the following methods to specify when your message is sent.

Method 1

  1. In your open message, click the Options tab.

  2. Under Delivery Options, click to select the "Do not deliver before" check box.

  3. In the "Do not deliver before" box, type a date and time or display the calendar and click to select a date. If you want to use a date months in the future, click the arrow to the right of the month name in the calendar heading and repeatedly click until the month you want appears.

  4. If you want the unread message to be deleted, click to select the Expires After check box and type a date and time in the Expires After box, or display the calendar and click to select a date. If you want to specify a date months in the future, click the arrow to the right of the month name in the calendar heading and repeatedly click until the desired month appears. Change the time if necessary. You may also specify that the message is never deleted by clicking None at the bottom of the calendar.

  5. Click Send to transfer the message to the Microsoft Exchange Server for processing.

Method 2

  1. In your open message, click Properties on the File menu.

  2. On the General tab, click Send Options, and click the Microsoft Exchange Server tab.

  3. Under Send This Item, click to select In, type a number, and then click to select a interval from the list. For example, if you want the message sent in two hours, type 2 and select hours from the list.

  4. If you want the unread message to be deleted, in the Expire This Item section, type a number and then click to select a interval from the list. For example, if you want the message deleted in two days, type 2 and select days from the list.

  5. Click OK twice.

  6. Click Send to place the message in your Outbox for processing.

The advantage to using this method is you can modify the message or delete it before it is actually sent.
