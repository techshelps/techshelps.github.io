OL97: Error in Help Topic "Add a Personal Address Book . . ."

 Last reviewed: January 21, 1998
Article ID: Q179648
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

When you click the Personal Address Book link in Step 5 of the Microsoft Outlook 97 Help topic "Add a Personal Address Book to a user profile" you receive the message 

   Cannot find the OUTLTIP.HLP i file. Do you want to try to find this
   file yourself?
If you click No, you receive the message 

  Cannot find the OUTLTIP.HLP i Help file. Check to see that the file
  exists on your disk. If it doesn't, you need to reinstall it.
If you click Yes, you receive an Open File dialog box. If you click either Open or Cancel, you receive the message 

  Cannot find the OUTLTIP.HLP i Help file. Check to see that the file
  exists on your disk. If it doesn't, you need to reinstall it.

CAUSE

The link to the Personal Address Book in Step 5 incorrectly incorporates the letter "i" from the word "in."

RESOLUTION

To read the Help topic in the link in Step 5:

  1. On the Help menu, click Contents and Index.

  2. On the Index tab in the first box, type Personal, click Personal Address Book in the list and then click Display.

  3. Click "About the Address Book" in the Topics Found list and then click Display.
