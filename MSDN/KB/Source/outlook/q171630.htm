OL97: Files Installed by the Internet Mail Enhancement Patch

 Last reviewed: January 19, 1998
Article ID: Q171630
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

This article lists the files installed or modified by the Internet Mail Enhancement Patch for Microsoft Outlook 97 (IMEP).

MORE INFORMATION

IMEP installs the following files: 

   C:\Program Files\Microsoft Office\Office\IMEPRead.txt
   C:\Program Files\Microsoft Office\Office\Outlimep.cnt
   C:\Program Files\Microsoft Office\Office\Outlimep.hlp
   C:\Windows\Addins\Minet.ecf (added if not present)
   C:\Windows\Inf\Imailab.inf
IMEP modifies the following files: 

   C:\Program Files\Microsoft Office\Office\Outlhlp.cnt
   C:\Windows\Help\Inetcomm.hlp
   C:\Windows\System\Inetab32.dll
   C:\Windows\System\Iso88591.trn
   C:\Windows\System\Mapisvc.inf
   C:\Windows\System\Minet32.dll
   C:\Windows\System\Norweg.trn
   C:\Windows\System\Outlcomm.dll
   C:\Windows\System\Sedish.trn
   C:\Windows\System\Usascii.trn
