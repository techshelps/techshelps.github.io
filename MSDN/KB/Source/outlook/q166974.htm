OL97: Error When Installing IMEP With Running Programs

 Last reviewed: January 19, 1998
Article ID: Q166974
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

Installing the Internet Mail Enhancement Patch (IMEP) for Microsoft Outlook 97, while other programs are running, may cause a system restart.

CAUSE

Some shared files will not be properly updated if other programs are running during installation.

RESOLUTION

Quit all running programs other than your Web browser before installing the IMEP.

MORE INFORMATION

The IMEP shares certain files with programs other than Outlook. To assure proper updating of these files you must quit all programs other than your Web browser before installation of the IMEP.

The Internet Mail Enhancement Patch (IMEP) for Microsoft Outlook 97 is available on the Microsoft Internet site at: 

 http://www.microsoft.com/OfficeFreeStuff/Outlook/
