OL97: Unable to Route E-mail Message

 Last reviewed: January 19, 1998
Article ID: Q173103
The information in this article applies to:
  • Microsoft Outlook 97

SUMMARY

Microsoft Outlook 97 does not have the ability to send e-mail messages to a list of recipients in a specific order.

The freeware utility, Routed Message Form, is a Microsoft Outlook 97 form that can be used to send a message sequentially to a list of Outlook recipients. It uses a custom RouteTo text field to keep track of the recipients while the form is being routed to the next person in the list.

MORE INFORMATION

For more information about this product and to download the file, go to the following Web address:
