OL97: ErrMsg: Your Changes Could Not Be Saved

 Last reviewed: January 5, 1998
Article ID: Q178765
The information in this article applies to:
  • Microsoft Outlook 97
  • Microsoft Windows NT

SYMPTOMS

If you open an Microsoft Excel workbook from within Microsoft Outlook 97, make changes and save the workbook, you may get the following error: 

   Your changes could not be saved to '<filename.xls>', but were saved to
   a temporary document named '<xxxxxxxx>'. Close the existing document,
   then open the temporary document and save it under a new name.
If you open a Microsoft Word document in Outlook, make changes, and save the workbook, you may get the following error: 

   Word cannot complete the save due to a file permission error (<path and
   filename.doc>)

RESOLUTION

Rename the temp file using Windows NT Explorer.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Outlook 97 and Microsoft Windows NT 4.0, Service Pack 3. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

After the error message has been displayed you can apply the following steps, as indicated in the error message.

Rename the Temp File in Windows NT Explorer

  1. Click the Start button, point to Programs, and then click Windows NT Explorer.

  2. Double-click the My Documents folder.

  3. Right-click the temp file and click Rename on the shortcut menu.

  4. Type the correct file name with the proper file extension, .xls for Excel files and .doc for Word documents. Press ENTER.

  5. Close Windows NT Explorer.

Steps to Reproduce Problem

  1. In a new Excel Workbook, press ALT+F11 to open the Visual Basic Editor.

  2. On the Insert menu, click Module.

  3. Close the Visual Basic Editor.

  4. Save the Workbook as Book1.xls in the My Documents folder, then close Excel.

  5. On the Outlook shortcut bar, click Other.

  6. Click My Computer, and in the Explorer window click the My Documents folder.

  7. Open the file Book1.xls and make some edits in the file.

  8. Click Save on the toolbar.
