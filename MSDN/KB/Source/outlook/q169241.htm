OL97: Error in Page Setup Using Acrobat PDFWriter

 Last reviewed: January 19, 1998
Article ID: Q169241
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you use the Page Setup option of Microsoft Outlook 97, you may receive the following error message: 

   An application error has occurred and an application error log is
   being generated.
                            OUTLOOK.exe
   Exception access violation (0x0000005), Address: 0306c361c
This error may occur if your printer driver is set to the Adobe Acrobat, version 3.01, PDFWriter for NT.

CAUSE

You are using a beta version of the PDFWriter for NT driver. The beta version 5 of this driver is known to cause errors with Outlook's Page Setup option.

RESOLUTION

Adobe is aware of this problem and is updating their printer driver.

MORE INFORMATION

Follow these steps to determine what version of the PDFWriter for NT driver you are using:

  1. On the Windows Taskbar, click Start and point to Settings and then click Printers.

  2. In the list of installed printer drivers use your right mouse button to click "Acrobat PDFWriter for NT" and on the shortcut menu, click Properties.

  3. Click the Page Setup tab, and then click About.

The Adobe PDFWriter for NT driver is manufactured by Adobe, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
