OL97: Connection Error Using MindSpring Internet Mail

 Last reviewed: January 19, 1998
Article ID: Q164245

The information in this article applies to:

  • Microsoft Outlook 97

SYMPTOMS

When you start Outlook 97 or initially connect to your MindSpring Internet Mail server, you may receive the following error message: 

    You could not be logged on to mail server: servername.
    Be sure your account information is accurate. If this problem
    persists, contact your Internet Service Provider.
Following this message, you may notice that sent mail does not leave your Outbox.

RESOLUTION

This problem may be caused by incorrect settings in the Internet Mail service properties. Follow these steps to troubleshoot this problem:

NOTE: If you use the Internet Mail Enhancement Patch, see the section following these steps.

  1. Click the Start button, point to Settings, and click Control Panel to open the Control Panel.

  2. Double-click the Mail And Fax icon. You may only have a Mail icon if you did not install Fax services.

  3. If you have more than one profile, click Show Profiles and on the General tab, click the profile that contains your MindSpring Internet Mail settings.

  4. Click Properties to open the Profile Properties dialog box. On the Services tab, you should see the Internet Mail service. If not, use the Add button to add Internet Mail.

  5. Under the Services tab, click Internet Mail, and then click the Properties button to open the Internet Mail Properties dialog box.

  6. Under the General tab and Mailbox Information, type pop.mindspring.com or pop.pipeline.com (depending on if you have a MindSpring or Pipeline account)in the Internet Mail Server box.

    NOTE: If your mail server information is different than listed above, use the server name provided by MindSpring.

  7. In the Account Name box, type your user e-mail name, e-mail password, and the rest of your Personal Information.

  8. Click Advanced Options and in the "Forward all outbound mail to the following mail server" box, type mail.mindspring.com or mail.pipeline.com (depending on if you have a MindSpring or Pipeline account).

    NOTE: If your forwarding (sending) mail server information is different than listed above, use the server name provided by MindSpring.

  9. Click OK, OK, OK, and Close to close all dialog boxes.

Incorrect Settings Using the Internet Mail Enhancement Patch Service

  1. From the Windows Desktop, click Start, point to Settings, and click Control Panel to open Control Panel.

  2. Double-click to open the Mail And Fax icon. You may only have a Mail icon if you did not install Fax services.

  3. If you have more than one profile, click Show Profiles and on the General tab, click the profile that contains your MindSpring Internet Mail settings.

  4. Click Properties to open the Profile Properties. On the Services tab, you should see the Internet Mail service. If not, use the Add button to add Internet Mail.

  5. On the Services tab, click Internet Mail, and then click Properties to open the Internet Mail Properties.

  6. Click the Servers tab and in the Outgoing Mail (SMTP) box type, IN LOWERCASE, mail.mindspring.com or mail.pipeline.com (depending on if you have a MindSpring or Pipeline account).

  7. In the Incoming Mail (POP3) box type, IN LOWERCASE, pop.mindspring.com or pop.pipeline.com (depending on if you have a MindSpring or Pipeline account).

    NOTE: If your mail server information is different than listed above, use the server names provided by MindSpring.

  8. In Account Name, type the user's e-mail name up to, but not including the @ symbol, and in Password, type in your password.

  9. Click OK, OK, and Close to close all dialog boxes.

Start Outlook and try to send a message.

MORE INFORMATION

MindSpring's Internet Mail system uses one server for receipt of messages and a second server for sending messages. In the Internet Mail service profile, the Internet Mail server box identifies the mail server from which you receive mail (incoming). The Advanced Options of the Internet Mail service profile enables you to name the sending e-mail server (outgoing).

To obtain your personal MindSpring e-mail account information contact MindSpring Customer Service at 800-719-4660 or MindSpring Technical Support at 800-719-4660.
