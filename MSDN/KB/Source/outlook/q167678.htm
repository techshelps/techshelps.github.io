OL97 Err Msg: Internal Application Error

 Last reviewed: January 19, 1998
Article ID: Q167678
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

If you click Properties on the File menu of an open, unsent e-mail message that has an attachment, and then click Send Options, you may receive the error: 

   Internal Application Error.

CAUSE

There may be corruption in the attached file.

RESOLUTION

To successfully send the attached file with the message, you must repair the damage. There are various methods for recovering different types of corrupted files.

MORE INFORMATION

For more information on ways to repair Microsoft Word documents, please see the following articles in the Microsoft Knowledge Base: 

   Article-ID:   Q156573
   Title     :   How to Recover Damaged Documents

   Article-ID:   Q87856
   Title     :   Troubleshooting Damaged Documents in Word for Windows
For more information on ways to repair Microsoft Excel files, please see the following article in the Microsoft Knowledge Base: 

   Article-ID:   Q142117
   Title     :   Excel: Summary of Methods to Recover Data from
                 Corrupted Files
For more information on ways to repair Microsoft Access databases, please see the following article in the Microsoft Knowledge Base: 

   Article-ID:   Q109953
   Title     :   How to Repair a Damaged Database
For more information on obtaining Microsoft Support Files from online services, please see the following article in the Microsoft Knowledge Base: 

   Article-ID:   Q119591
   Title     :   How to Obtain Microsoft Support Files from Online Services
For information on ways to repair corruption in files created in programs made by companies other than Microsoft, contact the manufacturer of the application.

REFERENCES

For more information about attachments, type "send attachments" in the Office Assistant, click Search, and then click to view "Insert a file into an item."
