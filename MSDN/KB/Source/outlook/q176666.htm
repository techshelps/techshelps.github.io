OL97: Holidays Duplicated After Importing Calendar

 Last reviewed: January 19, 1998
Article ID: Q176666
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

Holidays are duplicated in your default Microsoft Outlook 97 Calendar folder after importing a Calendar folder from a Personal Folders (PST) file.

CAUSE

Both your initial Calendar folder and the one you imported contained holidays inserted with the Add Holidays option. This occurs even though you chose the option "Do not import duplicates" or "Replace duplicates with items imported" during importing.

RESOLUTION

To Avoid Importing Duplicate Holidays

  1. On the File menu, click Import And Export.

  2. Click to select "Import from a personal folders file (.pst)," and then click Next.

  3. Type the name and path of the file you want to import or browse to the file by clicking Browse and then click Next.

  4. Click to select the folder you want to import.

  5. Click to select the folder you want to import the items into.

  6. Click Filter.

  7. On the Advanced tab click Field.

  8. Point to Frequently-used Fields and click Categories.

  9. From the Condition list click Doesn't Contain.

  10. In the Value field type "Holiday" without the quotation marks.

  11. Click Add To List then OK and Finish.

The holiday records will not be imported from the incoming file.

MORE INFORMATION

If holidays are duplicated in your Calendar, you can delete all holidays and re-import them.

To Delete All Holidays

  1. In the Calendar folder, in Current view, click to select By Category.

  2. Right-click the Holiday categories group bar and click Delete on the shortcut menu.

All items with a category of Holiday will be deleted.

To Re-import Holidays to Your Calendar Folder

  1. On the Tools menu, click Options.

  2. On the Calendar tab, click Add Holidays.

  3. Click to select the country whose holidays you want to add and click OK.

Your calendar will contain only one set of holidays.

Steps to Reproduce Problem

  1. On the File menu click, Import and Export.

  2. Click to select "Import from a personal folders file (.pst)" and then click Next.

  3. Type the name and path of the file you want to import or browse to the file by clicking Browse and then click Next.

  4. Click to select the folder you want to import.

  5. Click to select the folder you want to import the items into.

  6. Click OK and Finish.

RESULTS: The holiday records are duplicated from the imported file.
