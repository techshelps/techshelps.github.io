OL97: Error After Installing Crystal Reports Add-In

 Last reviewed: January 19, 1998
Article ID: Q174975
The information in this article applies to:
  • Microsoft Outlook 97

SYMPTOMS

After installing Crystal Reports for Microsoft Outlook 97, you may get the following error: 

   The add-in 'C:\Program Files\Seagate Software\Crystal Reports
   for Office...' could not be installed or loaded. Unable to load
   'C:\Program Files\Seagate Software\Crystal Reports for Office...'.
   You may be out of memory, out of system resources, or missing
   a .dll file.

CAUSE

The path to the Crext32.dll file is incorrect in the Crext.ecf file.

RESOLUTION

Open the Crext.ecf file and correct the path to the Crext32.dll file.

MORE INFORMATION

After downloading the Crystal Reports programs, you must install the Crext.ecf file as an Outlook Add-in. After this is done, the error above may appear. To remedy this situation, you must open the Crext.ecf file with a text editor and correct the path to the Crext32.dll file.

Downloading the Crystal Reports Programs

  1. On the Help menu point to "Microsoft on the Web, and click Free Stuff.

  2. On the Web page, scroll to and click to download "Crystal Reports Designer" and/or "Crystal Reports Viewer/Analyzer."

Installing the Crystal Reports Add-in

  1. On the Tools menu, click Options.

  2. On the General tab, click Add-In Manager and in the Add-Ins dialog box, click Install.

  3. Click to select the Crext.ecf file and then click Open. The default location is: 

          Program Files\Seagate Software\Crystal Reports for Outlook 97

  4. Click OK twice.

Correcting the Path to the Crext32.dll File

  1. In Windows Explorer navigate to Crext.ecf, right-click the file and click Open on the shortcut menu.

  2. In the Open With list, click to select either NotePad or WordPad and then click OK.

  3. Change the path to point to the correct location of the Crext32.dll file. The default location is: 

          Program Files\Seagate Software\Crystal Reports for Outlook 97\
