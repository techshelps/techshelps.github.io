Deinstall of SMS Administrator Does Not Clean Registry

 Last reviewed: April 23, 1997
Article ID: Q138236

The information in this article applies to:
  • Microsoft Systems Management Server versions 1.1 and 1.2

SYMPTOMS

After you deinstalling Systems Management Server Administrator, the registry still contain keys for Systems Management Server. The deinstall process leaves the Systems Management Server key with Components and Setup subkeys user: 

   HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server versions 1.1 and 1.2. We are researching this problem and will post new information in the Microsoft Knowledge Base as it becomes available.
