Hardware Inventory Fails on Removable Media Drives

 Last reviewed: April 24, 1997
Article ID: Q151861

The information in this article applies to:
  • Microsoft Systems Management Server versions 1.0 and 1.1

SYMPTOMS

If a client has a removable media drive, and inventory is run without a disk in that drive, inventory fails with the following error: 

     HARDWARE INVENTORY PHASE
     -[xx%]-
     Not Ready Reading Drive D
     Abort, Retry, Fail?

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server versions 1.0 and 1.1. We are researching this problem and will post new information in the Microsoft Knowledge Base as it becomes available.
