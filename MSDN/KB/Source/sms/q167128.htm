SMS: Network Ports Used by Remote Helpdesk Functions

 Last reviewed: May 27, 1997
Article ID: Q167128
The information in this article applies to:
  • Microsoft Systems Management Server, version 1.2

The Systems Management Server Helpdesk uses the following ports during its operations: 

   Purpose                  TCP/IP      IPX
   ------------------------------------------

   Verification of Rights     1761     0x8138
   Remote Control             1762     0x8238
   Remote Reboot              1761     0x8138
   Remote Chat                1763     0x845F
   File Transfer              1764     0x4100
   Remote Execute             1761     0x8138
