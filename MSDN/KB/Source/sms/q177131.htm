SMS: Cannot Remotely Monitor WinNT Client from Win95 NetMon

 Last reviewed: March 16, 1998
Article ID: Q177131
The information in this article applies to:
  • Microsoft Systems Management Server version 1.2

SUMMARY

When attempting to use Network Monitor remotely from Windows 95 to a Windows NT Server computer, you may receive the following error message: 

   Access Denied. You must have admin privledges on the machine
   you are connecting to.
NOTE: Privileges is spelled incorrectly in the error message.

MORE INFORMATION

You must use a computer running Windows NT Server to remotely monitor another Windows NT computer with Network Monitor.
