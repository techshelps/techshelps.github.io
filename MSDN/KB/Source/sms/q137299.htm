System Font Size Mismatch in Windows 95 Remote Control

 Last reviewed: April 18, 1997
Article ID: Q137299

The information in this article applies to:
  • Microsoft Systems Management Server version 1.1
  • Microsoft Windows 95

SYMPTOMS

When you remotely control a Windows 95 client, the system font size used in the Quick Windows Viewer may not correspond to the font size used on the Windows 95 client. As a result, the text for some controls will be truncated.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.1. This problem was corrected in the latest US Service Pack for Systems Management Server. For information on obtaining this update, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
