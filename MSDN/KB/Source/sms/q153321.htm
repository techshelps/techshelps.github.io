Network Monitor Err Msg: Parser DLL Is Not Valid Message

 Last reviewed: April 24, 1997
Article ID: Q153321

The information in this article applies to:
  • Microsoft Systems Management Server versions 1.0 and 1.1

SYMPTOMS

When stopping a capture in Network Monitor while a Remote Control session is going, the Network Monitor gives the following error message: 

   Parser DLL is not valid message

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.1. This problem has been corrected in Systems Management Server version 1.2.
