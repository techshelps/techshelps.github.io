SMS: Chat Window Shows "Unknown" for User

 Last reviewed: May 21, 1997
Article ID: Q161697
The information in this article applies to:
  • Microsoft Systems Management Server, version 1.2

SYMPTOMS

When attempting to use the Help Desk Remote Chat function to communicate with a Systems Management Server client computer running Windows NT Workstation, the remote username is displayed as "unknown" above the chat dialog box.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.2. This problem was corrected in the latest Microsoft Systems Management Server version 1.2 U.S. Service Pack. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K

MORE INFORMATION

This problem only occurs on client computers running Windows NT Workstation. Other clients capable of remote control will display "Remote User <USER> on Machine <MACHINE NAME>" when a remote chat session is established.
