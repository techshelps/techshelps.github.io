SMSRUNxx Doesn't Support Redirectors Returning a Null Username

 Last reviewed: April 24, 1997
Article ID: Q153572

The information in this article applies to:
  • Microsoft Systems Management Server versions 1.0 and 1.1

SYMPTOMS

SMSRUNxx will not launch any SMS applications.

CAUSE

SMSRUNxx checks for a null username returned from the network detection process. If the username is null, the network detection process determines that there isn't an active network connection and SMSRUNxx won't launch any SMS applications.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server version 1.1. It has been fixed in Systems Management Server version 1.2.
