SMS: Inventory Does Not Distinguish Individual Raid 5 Drives

 Last reviewed: March 12, 1997
Article ID: Q121207
1.00 1.10 1.20 WINDOWS kbnetwork kbui

The information in this article applies to:

  • Microsoft Systems Management Server versions 1.0, 1.1 and 1.2

SYMPTOMS

When running Inventory from a Systems Management Server on a machine that has a RAID 5 disk array, Systems Management Server reports only one drive controller.

STATUS

Microsoft has confirmed this to be a problem in Systems Management Server versions 1.0 and 1.1. We are researching this problem and will post new information in the Microsoft Knowledge Base as it becomes available.
