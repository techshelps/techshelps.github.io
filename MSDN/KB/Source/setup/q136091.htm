"Invalid Destination Folder" Error Performing Admin Install

 Last reviewed: January 16, 1997
Article ID: Q136091
The information in this article applies to:
  • Microsoft Access 97 for Windows
  • Microsoft Access version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Project 97 for Windows
  • Microsoft Project for Windows 95, version 4.1
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you run an administrative installation (Setup /a) of one of the programs listed at the beginning of this article, you may receive the following error message when you change the destination folder for the program 

   The specified folder:

   '<path>'

   is invalid, incomplete, too long, or write-protected. Enter a full
   path with the drive letter; for example 'C:\APPS'.
where <path> is the location on the server in which you are installing the program.

CAUSE

This error message appears when you change the destination for the administrative installation to one of the following:

  • a Universal Naming Convention (UNC) path

    -or-

  • a path that contains a long folder name

    -or-

  • a path with a folder name that contains a space character

For example, this error message appears if you enter the following destination for an administrative installation: 

   \\server\share\Msoffice

   -or-

   f:\Office 95
When you perform an administrative installation, you must install the program to a path using a drive letter (such as f:) and you must use 8.3 folder names.

RESOLUTION

To avoid receiving this error message when you perform an administrative installation, enter a destination that uses a drive letter and 8.3 folder names. The following examples are valid destinations for an administrative installation: 

   f:\Msoffice

   -or-

   f:\Word95
