OFF97: Can't Install Visual FoxPro or Microsoft Office 7.0

 Last reviewed: July 14, 1997
Article ID: Q159527
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Visual FoxPro, version 5.0

SYMPTOMS

When you attempt to install Microsoft Office for Windows 95, version 7.0 or Microsoft Visual FoxPro, version 5.0 on your computer, you may receive the following error messages: 

   Unable to load installable ISAM
   Can't load mspdox35.dll (error code=126)

   -and-

   Error while configuring ODBC drivers!
   The configuration of the ODBC Microsoft Paradox Driver (*.db) driver
   failed.

CAUSE

This problem may occur if the following conditions are true:

  • You have previously installed Microsoft Office 97 on this computer.

    -and-

  • You did not install the Paradox ODBC driver, included in the Microsoft Office 97 Data Access Pack, on this computer.

WORKAROUND

To prevent this problem from occurring, either:

  • Install the Data Access Pack included with Microsoft Office 97. To do this, run the file Dataacc.exe, located in the \ValuPack\DatAcc folder on your Microsoft Office 97 compact disc.

    Once the Data Access Pack is installed, the Setup programs for Microsoft Office 7.0 and Microsoft Visual FoxPro 5.0 should work correctly.

    -or-

  • Before running the Setup program for Microsoft Office 7.0 or Microsoft Visual FoxPro 5.0, rename the following files in your Windows\System folder: 

          Msjt3032.dll
      Odbcji32.dll
      Odbcjt32.dll
      Odbctl32.dll

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Microsoft Office versions 7.0 and 97 and Microsoft Visual FoxPro 5.0 install a set of database files, known as Jet files, on your computer. Jet files are designed to be used by any program that works with database files.

Microsoft Office 97 behaves differently with respect to Jet files than Microsoft Office 7.0 and Microsoft Visual FoxPro 5.0:

  • By default, Microsoft Office 97 does not install the Paradox driver: if you want the Paradox driver installed, you must install the Data Access Pack. This differs from Microsoft Office 7.0 and Microsoft Visual FoxPro 5.0, which automatically install the Paradox driver.
  • Microsoft Office 97 installs a newer version of the Jet files than the versions installed by Microsoft Office 7.0 and Microsoft Visual FoxPro 5.0.
  • Some Jet files have changed names from one version to the next: specifically, the Paradox driver's name can be either Mspdox35.dll or Mspdox32.dll, depending on which version of the Jet files is installed on your computer.

Because of the change in behavior with respect to the Paradox driver, the change in versions of the Jet files, and the change in file names, you may encounter problems when installing programs that use an older version of the Jet files, such as Microsoft Office 7.0 or Microsoft Visual FoxPro 5.0, after installing programs that use a newer version of the Jet files, such as Microsoft Office 97. Note that you will encounter problems only when you install an older program after you have installed Microsoft Office 97.

To prevent problems from occurring, use either of the workarounds shown in the "Workaround" section.
