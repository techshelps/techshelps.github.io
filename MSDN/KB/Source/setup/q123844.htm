Options Unavailable in Maintenance Mode Install

 Last reviewed: September 10, 1996
Article ID: Q123844
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.2, 4.3
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows, version 5.0c
  • Microsoft PowerPoint for Windows, versions 4.0a, 4.0c
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

When you run the Setup program in maintenance mode for any of the applications listed above, and you attempt to remove one of the application's components, if the check box for the item that you want to remove is cleared (it does not contain an "x") you will be unable to remove the item.

CAUSE

This behavior occurs if you have installed the component using an earlier version of the application's Setup program. For example, if you perform a complete installation of Microsoft Office version 4.0, and you later run the Microsoft Office version 4.2 Setup program, but you choose NOT to install Microsoft Excel, you cannot run the Microsoft Office version 4.2 Setup program in maintenance mode to remove Microsoft Excel.

If the item that you want to remove was not installed using the version of the application's Setup program that you are running, you cannot remove the item because it is not present in the application's SETUP.STF file.

WORKAROUND

To work around this behavior, run the Setup program in maintenance mode, select (check) the item that you eventually want to remove, and install it. You can then run the Setup program in maintenance mode, and clear the check box for the item to remove the item.
