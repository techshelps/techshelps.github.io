Product: Microsoft SNA Server for Windows NT v2.0 Update: Patch #39 Status: Confirmed

PKZIP 2.04g format -- uncompress using the "-d" option.

Date: April 18, 1994

Modules updated: 

   <snaroot>\SETUPSVC.DLL  04/12/94  02:20p  31744
Bugs fixed:

XXXX Fix setup problem on MS Windows NT Advanced Server 3.5

Update Procedure:

- For disk-based installation, copy this new module over the old version 

  located on SNA Server 2.0 Disk #1.
- For CD-ROM based installation: 

  1. copy the contents of the SNA Server 2.0 CD-ROM to the local disk

  2. replace this updated SETUPSVC.DLL over the original 2.0 version
     located on the local disk

  3. install SNA Server from the local disk
*********************** End of Readme *******************************

SNA Server 2.0 patches are located on:

  1. Library #2 of the MSSNA Compuserve forum.

  2. The Internet server under BUSSYS\SNASRVR\SUP-ED\FIXES\ (supported patches) OR on BUSSYS\SNASRVR\UNSUP-ED\FIXES\ (unsupported patches).
