Tab Function Does Not Work on 3270 Printer LU Page

 Last reviewed: March 13, 1998
Article ID: Q164651

The information in this article applies to:

  • Microsoft SNA Server, version 3.0.

SYMPTOMS

The tab button does not cycle through button options (OK, Cancel, Help) on the SNA Manager 3270 Printer LU page.

CAUSE

This is a problem in the SNA Server 3.0 Manager tool.

RESOLUTION

An update to SNA Server 3.0 Manager fixes this problem. The updated module is <snaroot>\system\snaexp.exe.

STATUS

Microsoft has confirmed this to be a problem in SNA Server, version 3.0.

A supported fix is now available for 3.0, but has not been fully regression- tested should be applied only to systems experiencing this specific problemand . Unless you are severely impacted by this specific problem, Microsoft recommends that you wait for the next Service Pack that contains this fix. Contact Microsoft Product Support Services for more information.
