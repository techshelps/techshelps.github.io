5250 Applet Defaults To a French Canadian Host Code Page

 Last reviewed: July 18, 1996
Article ID: Q143239
The information in this article applies to:
  • Microsoft SNA Server for Windows NT, versions 2.1 and 2.11

SYMPTOMS

After you install the 5250 applet on a Windows or Windows NT computer and the Country code is configured to Canada (Canadian English), the Host Code Page defaults to French Canadian.

In Windows 95, the Host Code Page defaults to French Canadian if the Country code in Control Panel Regional Settings is set to English (Canadian).

WORKAROUND

Select Session Configuration from the Session menu in the 5250 applet and change the Host Code Page.

RESOLUTION

Apply SNA Server 2.11 Service Pack 1.

STATUS

Microsoft has confirmed this to be a problem in SNA Server versions 2.1 and 2.11. This problem has been resolved in SNA Server 2.11 Service Pack 1.
