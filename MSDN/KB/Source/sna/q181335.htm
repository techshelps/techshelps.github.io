QFE Support Policy for SNA Server, Version 3.0

 Last reviewed: February 25, 1998
Article ID: Q181335
The information in this article applies to:
  • Microsoft SNA Server version 3.0

SUMMARY

The QFE policy period for SNA Server 3.0 will end 12 months after the release of SNA Server 4.0. SNA Server 4.0 was released on January 15, 1998. Therefore, the QFE policy period will end on January 15, 1999. Please take this date into consideration as you are developing your SNA Server deployment plans. If you have any questions on Microsoft QFE or SNA Server support, please contact Microsoft Technical Support.

MORE INFORMATION

The Microsoft policy for Quick Fix Engineering (QFE) is to provide bug fixes for 12 months following the release of the subsequent product version. Beyond the 12-month period, Microsoft will consider specific customer bug fix requests on a case-by-case basis. Please note that the Microsoft QFE policy is different from the support policy. Microsoft’s support policy is to continue supporting its customers as long as there is sufficient support demand.
