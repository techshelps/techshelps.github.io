SNA Patch 23 Corrects Printing Failure for 5250 Applet

 Last reviewed: April 23, 1996
Article ID: Q118302
The information in this article applies to:
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft SNA Server for Windows NT, version 2.0

SYMPTOMS

The File/Print feature of the 32-bit 5250 Applet for SNA Server 2.0 does not work.

RESOLUTION

Apply Patch 23.

If a WSF file has been created prior to applying patch 23 and the Applet has been configured to automatically load a WSF profile, you will still not be able to use the File/Print feature. If this is the case, re-save the WSF file after the patch has been applied.

STATUS

Microsoft has confirmed this to be a problem in SNA Server version 2.0. This problem was corrected in Patch 23, and may be obtained from the MSSNA forum of Compuserve.
