How to Obtain SNA Version 2.11 U.S. Service Pack

 Last reviewed: April 17, 1997
Article ID: Q164077
The information in this article applies to:
  • Microsoft SNA Server version 2.11 and U.S. Service Pack 2

To find out which bugs are fixed by this update, see the Readme.txt file that accompanies the Service Pack or query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K

AVAILABLE FORMATS FOR SNA Server 2.11 SERVICE PACK 2

Service Pack 2 is available as a single self-extracting archive file for Intel (x86) and ALPHA computers from Microsoft's Internet WWW and FTP sites.

NOTE: Microsoft is phasing out future development and engineering efforts for the Power-PC and MIPS platform. The contents of the compact disc are not available for MIPS and Power-PC platforms.

INSTRUCTIONS FOR DOWNLOADING

You can download the correct version for your computer platform from one of the following locations:

  • Microsoft's FTP Site: ftp.microsoft.com (198.105.232.1) 

          /bussys/winnt/sna-public/fixes/sna211/ussp2/
  • Microsoft's Internet WWW Site: 

          http://www.microsoft.com/SNASupport/

The following files comprise the updated SNA Server and client files for SNA 2.11 SP2:

   S20484    211sp2as.exe   Server Update (Alpha)                 11.40 MB

   S20487    211sp2ie.exe   Eicon WAN Services Drivers (Intel)     5.04 MB
   S20486    211sp2ea.exe   Eicon WAN Services Drivers (Alpha)     5.76 MB
If you are unable to access these files, contact Microsoft Product Support Services.

INSTRUCTIONS FOR DECOMPRESSING FILES

All of the above files are compressed in self-extracting archive format, containing SNA Server 2.11 SP2. To extract each file, copy the file to a separate empty directory on your hard drive and run the file using the "-d" option. For example: 

    211sp2is.exe -d <PATH>
where <PATH> is the destination directory for the service pack. If you wish to extract the service pack to the current directory, leave <PATH> blank.

After the service pack file has been decompressed, run the Update.exe program to install the updated SNA Server 2.11 server or client files, as appropriate.

NOTE: The Update program will automatically stop any SNA Server services that may be running.

INSTALLING SNA 2.11 SERVICE PACK 2

** TO BE APPLIED TO SNA SERVER 2.11 or 2.11 SP1 ONLY!! **

Both the SNA Server and SNA client files have been updated in 2.11 SP2. Updates may be applied in any order---to the server and/or to the clients. It is recommended that all systems running SNA Server 2.11 (that is, SNA Servers and client computers) are updated to SP2. It may not be necessary to update the client computers to SP2 unless you are experiencing a client problem that has been fixed in SP2.

ORDERING INFORMATION AND AVAILABILITY

Microsoft SNA Server 2.11 U.S. Service Pack 2 (SP2) compact disc is available by ordering from Microsoft at (800) 936-3500, using the following part number: 

   Disk Set #: 211-075-100
SNA 2.11 SP2 includes SNA Server 2.11 Service Pack 1 (SP1) as well as bug fixes implemented since 2.11 SP1. In addition, the 2.11 SP2 compact disc includes updated documentation, collateral, and software development kit (SDK) files. SNA 2.11 SP2 includes updated SNA Server and client software for I386 (Intel) and DEC Alpha platforms.

NOTE: Support for MIPS and Power PC platforms has been discontinued and is not included in SNA 2.11 SP2. If you are running SNA Server on a MIPS or Power-PC system and are experiencing a problem that has been fixed in 2.11 SP2, please contact Microsoft Support for an update.
