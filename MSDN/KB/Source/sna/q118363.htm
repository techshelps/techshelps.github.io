README.TXT: Patch 50 for SNA Server 2.0

 Last reviewed: April 23, 1996

The information in this article applies to:
  • Microsoft SNA Server for Windows NT, version 2.0

The following is the README.TXT file that accompanies SNA Server 2.0 patch 50 (SNA20.P50):

Note that this article reflects the text of a static document. Therefore, some of the information in this article may be outdated. For example, as of May 20, 1996, Microsoft no longer maintains support forums on CompuServe.

******************* Beginning of Readme ***************************

Product: Microsoft SNA Server for Windows NT v2.0 Update: Patch #50 Status: Confirmed

PKZIP 2.04g format -- uncompress using the "-d" option.

Date: July 14, 1994

Modules updated: 

        \CLIENTS\WIN3X\WINCPIC.DLL
        \SDK\LIB\WIN3X\WINCPIC.LIB
Bugs fixed:

2270 Win16 WINCPIC.DLL missing security entry points

SNA Server Windows 3.x WINCPIC.DLL and WINCPIC.LIB was missing the entry points for the security related API's, such as CMSCST, CMSCSU, and CMSCSP.

Update Procedure:

- Relink the application with the above WINCPIC.LIB file, and 

  run the application using the attached WINCPIC.DLL, which should
  be applied to the Windows client machine's <winroot>\SYSTEM
  directory.
*********************** End of Readme *******************************

SNA Server 2.0 patches are located on:

  1. Library #2 of the MSSNA CompuServe forum.

  2. The Internet server under BUSSYS\SNASRVR\SUP-ED\FIXES\ (supported patches) OR on BUSSYS\SNASRVR\UNSUP-ED\FIXES\ (unsupported patches).
