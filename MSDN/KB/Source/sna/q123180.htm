Large File Download Fails Over Eicon WAN Services

 Last reviewed: November 1, 1995
Article ID: Q123180
The information in this article applies to:
  • Microsoft SNA Server for Windows NT 2.0 and 2.1

SUMMARY

Downloading a 30MB host file fails after 96,000 records when you use the Passport Communications RJE Print Manager product (either the MS-DOS or Windows NT versions) over SNA Server 2.0.

SNA Server is configured with an SDLC link supported by Eicon WAN Services for Windows NT version 3 release 1. The RJE download problem coincides with the Eicon SDLC connection being dropped.

RESOLUTION

Upgrade to Eicon WAN Services for Windows NT version 3 release 1a (v3r1a).

NOTE: Eicon has a separate update for Windows NT 3.5.
