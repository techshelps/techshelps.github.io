SnaExp.exe Access Violation in CshrMemPool()

 Last reviewed: March 19, 1998
Article ID: Q171083
The information in this article applies to:
  • Microsoft SNA Server for Windows NT, version 3.0 Service Pack 1

SYMPTOMS

An SnaExp.exe access violation in CShrMemPool() occurs during shutdown of SNA Server Manager.

CAUSE

Terminating SNA Server Manager may occasionally cause an access violation to occur when it tries to access a pointer that is not available to ManageRoot.

RESOLUTION

SNA Server has been updated to correct this problem.

STATUS

Microsoft has confirmed this to be a problem in Windows NT SNA Server version 3.0 Server Pack 1. This problem was corrected in the latest SNA Server version 3.0 U.S. Service Pack.
