Product: Microsoft SNA Server for Windows NT v2.0 Update: Patch #22 Status: Confirmed

PKZIP 2.04g format -- uncompress using the "-d" option.

Date: March 5, 1994

Modules updated: 

  All Windows 5250 applet files for the SNA Server Windows 3.x client
  (see below for a complete file list).  Includes Patch #1
Bugs fixed:

1818 Windows 5250 applet problems with international keyboards

Keyboard translation problems with the 5250 applet have been observed on both German and Swiss keyboards.

1703 Win16 5250 session configuration dialog problems (Patch #1)

NOTE: The 5250 applet included with this patch displays the session configuration dialog title as "&Local LU Alias:". This display problem will be fixed in a subsequent release.

Update Procedure:

- Expand the pkzip file from it's own directory, using the "-d" switch. 

  For example:

     md C:\patch22
     copy sna20.p22 C:\patch22
     cd C:\patch22
     PKUNZIP -d SNA20.P22

  This will create a subdirectory tree of CLIENTS\WIN3X, and expand all of
  the Windows 5250 components into the correct directory structure.
- XCOPY the above directory over your existing SNA Server Windows client 
  directory and server's \CLIENTS\WIN3x directory, using the /S /E /V
  switches. For example:

     XCOPY c:\patch22\clients\win3x\*.* C:\SNA.WIN\*.* /s /e /v
