SNA Web Clients Only Available on 3.0 Service Pack 2 CD

 Last reviewed: February 4, 1998
Article ID: Q177489
The information in this article applies to:
  • Microsoft SNA Server, version 3.0 SP2

SUMMARY

After applying Service Pack 2 for SNA Server 3.0, there is a reference in the online SNA Server help file under the new feature Web Setup for Windows 95 Clients: 

   For more information, see the Web Setup for Windows 95 Clients online guide on the
   SNA Server Help contents.

MORE INFORMATION

The information on the setup for the web clients, as well as the files for the web clients themselves, is only available on the Service Pack 2 compact disk. These files are not included in the Service Pack available for download on the Internet.

For more information on obtaining Service Pack 2 on compact disk see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q168654
   TITLE: How to Obtain SNA Version 3.0 U.S. Service Pack 2
