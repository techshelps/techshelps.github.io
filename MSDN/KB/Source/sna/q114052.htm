Product: Microsoft SNA Server for Windows NT v2.0 Update: Patch #24 Status: Confirmed

PKZIP 2.04g format -- uncompress using the "-d" option.

Date: March 5, 1994

Modules updated: 

   \CLIENTS\WIN3X\WIN16CL.DLL  02/14/94  04:24p  55164
Bugs fixed:

1871 Allow Win3x client setup to continue if system files not found

Make the SNA Server Windows 3.x client setup more forgiving if the a system file cannot be found. This update allows the user to continue past this error and complete the client setup program.

Update Procedure:

- COPY the above file over your existing client setup file on your Win3x 

  client setup disk, or on the server's \SNA\CLIENTS\WIN3X directory.
