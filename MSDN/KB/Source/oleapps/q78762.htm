How to Obtain Additional WordArt Font Effects in WinWord

 Last reviewed: September 9, 1996
Article ID: Q78762
The information in this article applies to:
  • Microsoft WordArt version 1.0

SUMMARY

The Help menu for Microsoft WordArt included with Microsoft Word for Windows version 2.0 contains the following information: 

   TopLine, a Windows software program that provides additional font
   effects, is available from the creators of WordArt: Pixellite,
   5221 Central Avenue, Suite 105, Richmond, CA 94804.

MORE INFORMATION

WordArt was created by a division of the Pixellite Group. Ken Grant was the contact at the Pixellite Group for questions about TopLine. NOTE: The last known phone number for the Pixellite Group, (510) 527-6400 has been disconnected and as of 9/12/94, there is no forwarding number.

TopLine is an enhanced version of WordArt that you can install and use with Word for Windows as an object linking and embedding (OLE) server.

TopLine is manufactured by the Pixellite Group, vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

For more information about WordArt, search for "WordArt" using the Help menu.
