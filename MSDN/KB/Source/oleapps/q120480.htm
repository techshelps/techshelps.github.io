OrgChart for Windows: Error Reading Drive <x>:

 Last reviewed: September 9, 1996
Article ID: Q120480
The information in this article applies to:
  • Microsoft Organization Chart for Windows, version 1.0

SYMPTOMS

When you reactivate a Microsoft Organization Chart 1.0 object or start a new one, you receive the following error message 

   Error reading drive <x>:

   -or-

   System error cannot read from drive <x>
where <x> is the letter assigned to your CD-ROM drive.

CAUSE

This error occurs when a CD-ROM-based application was installed with an option to leave its fonts on the CD-ROM. It also occurs if any fonts were explicitly loaded in the Fonts Control Panel from the CD-ROM drive but the font files were not copied to the Windows SYSTEM subdirectory.

RESOLUTION

Use one of the methods below to prevent this error message.

Insert a Compact Disc Before Running Graph or Organization Chart

Microsoft Organization Chart looks to the CD-ROM drive only to see if a compact disc (CD) exists. If any CD (other than a music CD) is in the drive at the time you start Organization Chart, the above error message is not displayed.

Install the Fonts to the Local Hard Disk Drive

Many fonts that come on CDs are packaged with software that uses an installation program which allows you to choose where to store the fonts. If available, use the installation program to install the fonts on the local hard disk drive.

If no installation program is available, you can reinstall the fonts to the local hard disk drive with the following steps:

  1. In Program Manager, choose Control Panel.

  2. Double-click the Fonts icon.

  3. In the Installed Fonts list, select all of the fonts that came on the CD. To select more than one font, hold down the CTRL key and click each font with the mouse.

  4. Choose the Remove button to remove the selected fonts. Choose Yes or Yes To All to confirm their removal.

  5. Choose Add. Make sure that the option to Copy Fonts To Windows Directory is selected.

  6. From the Drives list, select the CD-ROM drive. Make sure the disk is in the CD-ROM drive.

  7. In the Directories window, locate the directory that contains the fonts to be installed.

  8. To select the fonts to install, hold down the CTRL key and click each font with the mouse.

MORE INFORMATION

If you choose Cancel twice when the error is displayed, the dialog box is closed and Organization Chart is started. If a font from the CD-ROM is used during the session, the same error occurs. Choosing Cancel repeatedly at this point eventually does select the font; however, after you exit and return to the client application and then later return to the Organization Chart object you created, the font format reverts to the System font.
