MS Graph: Moving and Selecting in the Datasheet

 Last reviewed: January 8, 1998
Article ID: Q64456
The information in this article applies to:
  • PowerPoint Graph version 1.0c
  • Microsoft Graph for Windows, versions 3.0, 5.0
  • Microsoft Graph for Windows 95, version 5.0

SUMMARY

The following information below is from the Graph Reference Tables topic of PowerPointHelp, version 2.0. 

Keyboard Key(s)           Function


Arrow keys                Moves to and selects one cell up, down,
                          right, left

SHIFT+Arrow key           Extends selection one cell up, down,
                          left, right

HOME                      Moves to first cell in row containing
                          active cell

SHIFT+HOME                Extends selection to start of row

END                       Moves to last occupied column of row
                          containing active cell

SHIFT+END                 Extends selection to end of row

SHIFT+SPACEBAR            Selects entire row containing selected
                          cell or cells
SHIFT+UP ARROW or 
SHIFT+DOWN ARROW          Extends selection of row to include
                          other rows

CTRL+SPACEBAR             Selects entire column containing selected
                          cell or cells
SHIFT+LEFT or SHIFT+RIGHT Extends selection of column to include 
                          other columns

CTRL+PGUP                 Moves to cell at current row, column 2

CTRL+PGDN                 Moves to farthest right cell with data

CTRL+SHIFT+HOME           Extends selection to cell in row 2, column 2

CTRL+SHIFT+END            Extends selection to last cell with data

CTRL+SHIFT+SPACEBAR       Selects all cells in the datasheet

SHIFT+BACKSPACE           Deselects everything but the active cell

PGUP, PGDN                Moves to next window of cells (up or down)

SHIFT+PGUP                Extends selection to next window of
                          cells (up)

SHIFT+PGDN                Extends selection to next window of
                          cells (down)

CTRL+PGUP                 Moves right to next window of cells

CTRL+PGDN                 Moves left to next window of cells

CTRL+SHIFT+PGUP           Extends selection right to next window
                          of cells

CTRL+SHIFT+PGDN           Extends selection left to next window
                          of cells

TAB                       Moves one cell to the right

SHIFT+TAB                 Moves one cell to the left

ENTER                     Moves to the next cell below the current
                          cell in the selection
For more information about this topic, refer to pages 104-105 of your "Microsoft PowerPoint for Windows User's Guide," version 2.0.
