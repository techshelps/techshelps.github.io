WordArt: Pressing SHIFT+INS Turns Mouse Pointer into Hourglass

 Last reviewed: September 9, 1996
Article ID: Q77009
The information in this article applies to:
  • Microsoft WordArt version 1.0

SUMMARY

In the WordArt Editing dialog box, if SHIFT+INS is pressed with no object in the Clipboard, the mouse pointer will turn into an hourglass and remain that way until the mouse is moved. The computer will not be hung, this is simply a cursor redraw problem.

MORE INFORMATION

This effect is a problem with Windows 3.0, not WordArt. Any dialog box that contains a multiline text field will exhibit the same effect. (This includes the entire application space of Windows Notepad.) The same effect will be noticed if the Clipboard contains only non-text information.

Steps to Reproduce Problem

  1. In Publisher, use the WordArt tool to create a WordArt frame.

  2. Capture the entire screen to the Clipboard by pressing ALT+PRINT SCREEN.

  3. Attempt to paste into the WordArt editor text field by pressing SHIFT+INS.

  4. The mouse pointer will turn to an hourglass and will stay that way until the mouse is moved.

WordArt is the font effects package bundled with Microsoft Publisher.
