Equation Editor Fonts Not Listed for LaserJet III on LPT1.OS2

 Last reviewed: October 18, 1996
Article ID: Q80032
The information in this article applies to:
  • Microsoft Equation Editor, version 1.0

SUMMARY

The fonts provided with the Microsoft Word for Windows version 2.0 Equation Editor may not be recognized if you install them for use with a Hewlett-Packard (HP) LaserJet III printer that is configured to the LPT1.OS2 port.

MORE INFORMATION

The MT Extra, MT Symbol, and Fences fonts may not appear in the Word for Windows font list if you install them for the HPPCL5A driver when it is connected to the LPT1.OS2 port. Because the fonts are not recognized by Word for Windows, the Equation Editor warns you that these fonts are not installed for the current printer when you open the Equation Editor.

Workaround

Word for Windows recognizes these fonts if they are also listed in the WIN.INI file under a section for the same printer configured to the LPT1 port. To do this, use the following steps:

  1. Open the Windows WIN.INI file in a text editor.

  2. Search for the section headed [HPPCL5A,LPT1.OS2].

  3. Select this entire section, and copy it to a new location in the WIN.INI file.

  4. Change the name of the section heading to [HPPCL5A,LPT1].

  5. Save the changes, exit Windows, then restart Windows.

Note: This problem does not occur if the fonts are installed for a LaserJet II on the LPT1.OS2 port.

Microsoft has confirmed this to be a problem in Word versions 2.0 2.0a 2.0a-CD 2.0b 2.0c for Windows. We are researching this problem and will post new information here as it becomes available.
