XL:Excel Worksheet and Chart Objects Unavailable After Install

 Last reviewed: February 3, 1998
Article ID: Q125232
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 5.0
  • Microsoft Excel for the Power Macintosh, version 5.0

SYMPTOMS

When you choose Object from the Insert menu in an application that supports OLE, after installing Microsoft Excel, Microsoft Excel 5.0 Worksheet and Microsoft Excel 5.0 Chart do not appear in the list of objects that you can install.

CAUSE

After you perform a workstation installation of Microsoft Excel, this behavior occurs if you attempt to insert a Microsoft Excel object in a document before you run Microsoft Excel.

WORKAROUNDS

To work around this problem, start Microsoft Excel before you attempt to insert a worksheet or chart object in a document in another application.
