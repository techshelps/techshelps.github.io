CG2: Inserting ClipArt Across Banyan Vines Network Fails

 Last reviewed: February 23, 1998
Article ID: Q155782
The information in this article applies to:
  • Microsoft Clipart Gallery for Windows 95, version 2.0

SYMPTOMS

When you try to insert a Microsoft Clipart Gallery image into another program, such as Microsoft PowerPoint or Microsoft Publisher, you may receive the following error message 

   Cannot find "<file>.pcs"
   Use the Organize button and choose "Update Pictures" to update you
   Gallery
followed by 

   The file "<file>.pcs>" is not available because the Clipart Gallery
   can't connect to <server location>. The server may be down, or you may
   not have access privileges
You are then presented with an option to remove the image from the gallery.

CAUSE

Microsoft ClipArt Gallery uses universal naming convention (UNC) paths to locate and insert shared ClipArt Gallery images. Because Banyan Vines does not support UNC paths, ClipArt gallery is unable to locate the file.
