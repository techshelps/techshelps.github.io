WordArt: Arched Top with Inflate/Deflate Bottom Text Shape

 Last reviewed: February 27, 1998
Article ID: Q136496
The information in this article applies to:
  • Microsoft WordArt for Windows 95, version 3.0
  • Microsoft Publisher for Windows 95, version 3.0

SYMPTOMS

If you create a WordArt object using the Inflate or Deflate Bottom shapes, the top of the WordArt object will be slightly arched instead of a straight line.

If you choose the Inflate or Deflate Top shapes, the bottom of the WordArt object will be a straight line.

CAUSE

This is a side effect of allowing these shapes to work correctly with accented characters such as E accent acute or O with an umlaut.

RESOLUTION

To straighten the tops of these shapes of WordArt objects, click the Ee button on the WordArts toolbar.
