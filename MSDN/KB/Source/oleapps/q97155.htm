Draw: Rotate/Flip Command Unavailable with Bitmapped Formats

 Last reviewed: September 9, 1996
Article ID: Q97155
The information in this article applies to:
  • Microsoft Draw version 1.0 and 1.01
  • Microsoft Windows operating system version 3.0 and 3.1
  • Microsoft Windows 95

SUMMARY

The Rotate/Flip command on the Draw menu in Microsoft Draw is unavailable for use on the following types of graphics files: 

   Bitmap (.bmp)
   PC Paintbrush (.pcx)
   Tagged image file format (.tif)
   Encapsulated PostScript with a bitmap header (.eps)
This command is available only when you are working with a Windows metafile (.wmf) or a computer graphics metafile (.cgm).

MORE INFORMATION

Microsoft Draw cannot rotate or flip bitmapped images. Because the .tif, .bmp, and .pcx files are bitmapped images, they cannot be rotated or flipped.
