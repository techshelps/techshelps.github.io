Documentation Error: Cannot Change Default Menu Settings

 Last reviewed: September 9, 1996
Article ID: Q83696
The information in this article applies to:
  • Microsoft Draw version 1.0

SUMMARY

There is a documentation error on page 11 of the "Microsoft Draw User's Guide." The section titled "To choose a default menu setting" provides instructions for changing the default menu settings in Microsoft Draw. This information is incorrect; default settings are NOT saved when you exit Draw. The next time you start Draw, the application defaults are restored.
