Equation Editor: Fonts Won't Print with Pacific Page

 Last reviewed: September 9, 1996
Article ID: Q79581
The information in this article applies to:
  • Microsoft Equation Editor version 1.0

SUMMARY

Prior to the release of version 3.2, the Pacific Page PostScript Emulation Cartridge did not support Adobe Type 1 downloadable fonts. If you use a version of this cartridge earlier than 3.2, Microsoft Windows version 3.0 cannot download the PostScript fonts supplied with the Microsoft Equation Editor; this can cause the printer to hang. This problem can occur anytime you use the Fences, MT Extra, MT Symbol or MT Symbol Italic fonts.

MORE INFORMATION

To determine which version of the Pacific Page PostScript Emulation Cartridge cartridge you have, take the printer offline, and press the plus (+) key. The version number should display on the printer's LCD screen.

If your cartridge is a version earlier than 3.2, choose one of the following options:

  1. Contact Pacific Data Products at (619) 552-0880 to order an upgrade for the cartridge.

  2. Use the printer in Hewlett-Packard LaserJet (PCL) mode with the PCL versions of the aforementioned fonts installed. Using these fonts in conjunction with scalable font technology provides the best output.
