DRW: Error Clicking How To on Help Menu in Microsoft Draw

 Last reviewed: February 27, 1998
Article ID: Q137654
The information in this article applies to:
  • Microsoft Draw for Windows 95, version 1.01
  • Microsoft Publisher for Windows 95, version 3.0
  • Microsoft Works for Windows 95, version 4.0

SYMPTOMS

In Microsoft Draw, if you click How To on the Help menu, you will receive the following error message: 

   The topic does not exist. Contact your application vendor for
   an updated help file. (129)
Microsoft does not have an updated help file at this time.

NOTE: The other commands on the Help menu function properly.

CAUSE

Microsoft Draw contains an incorrect pointer to this part of the Help file.

WORKAROUND

To work around this problem, use the following steps to start Microsoft Draw Help from outside of Draw:

  1. Double-click the Msdraw.hlp file from My Computer or Explorer. This file is typically located in the following folder: 

          Program files\Common files\Microsoft Shared\Msdraw

  2. In the contents screen, double-click the How To icon.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Draw version 1.01 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
