Microsoft Draw Cannot Access More Than 81 Fonts

 Last reviewed: September 9, 1996
Article ID: Q81444
The information in this article applies to:
  • Microsoft Draw version 1.0

SUMMARY

Because of the way Microsoft Draw version 1.0 displays the Font dialog box, it cannot display more than 81 fonts. If more than 81 fonts are installed, you can only choose from the first 81 fonts.

MORE INFORMATION

In Microsoft Draw, you select fonts by choosing the Fonts command from the Text menu; a cascading menu, which can display up to 81 fonts, appears. Because no more than 81 font names (3 columns, 27 fonts per column) can display on a standard VGA screen at one time, the rest of the font names disappear beyond the right edge of the screen. You can only choose from the first 81 fonts that actually appear on the screen.

NOTE: The edge of a fourth column of font names is visible on the far right- hand side of the screen. You can use these names to select fonts; however, the font names themselves are hidden except for the first three characters.

REFERENCES

"Microsoft Draw User's Guide," version 1.0, page 64
