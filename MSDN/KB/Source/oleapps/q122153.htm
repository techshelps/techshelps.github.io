Microsoft Organization Chart File Descriptions (Windows)

 Last reviewed: February 19, 1998
Article ID: Q122153
The information in this article applies to:
  • Microsoft Organization Chart for Windows, version 1.0

SUMMARY

When you install Microsoft Organization Chart 1.0, the following three files are placed in the <Windows>\MSAPPS\ORGCHART subdirectory: 

    ORGCHART.EXE     Program executable
    ORGCHART.HLP     Help files
    TEMPLATE.MSO     Default template
and the following file is placed in the <Windows>\SYSTEM directory: 

    MSOFEN.DLL
These files are the main files required to run Organization Chart. It can be run from within another application that supports OLE, or it can be run as a stand-alone application.

When you start Organization Chart, whether from within another application or as a stand-alone application, it will check your Windows directory for two files, MSOPREFS.FEN and MSOCLIP.FEN. These files contain settings that Organization Chart uses, including some of your custom settings.

Organization Chart files can be saved as files to be used later. When you save a file from Organization Chart, the default extension is .OPX.

MORE INFORMATION

The .FEN files are not installed during Setup, so they are not removed using the Remove feature of Maintenance Mode in Setup. If you experience odd behavior in Organization Chart and reinstalling the application does not correct the behavior, the problem may be caused by a setting in one of these .FEN files. They are not text files, so they cannot be edited.

To reset the settings files, delete MSOPREFS.FEN and MSOCLIP.FEN from your Windows directory.

New files with the standard defaults will be generated the next time you run Organization Chart.
