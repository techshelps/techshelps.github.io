OFF4: COMMAND+OPTION+SPACEBAR Doesn't Work in Equation Editor

 Last reviewed: February 3, 1998
Article ID: Q122547
The information in this article applies to:
  • Microsoft Equation Editor, version 2.0
  • Apple Macintosh System 7.1 or later

SYMPTOMS

In Equation Editor, running under Macintosh System 7.1 or later, the COMMAND+OPTION+SPACEBAR key combination does not insert a 1-point space as expected.

CAUSE

If you are using System 7.1 or later, COMMAND+OPTION+SPACEBAR is reserved for international keyboard support, so the system does not allow you to use this keystroke in applications.

WORKAROUND

To work around this problem, remove the U.S. System 6 keyboard template from the System suitcase.

Once this file is removed can should be able to create a 1-point space by pressing COMMAND+OPTION+SPACEBAR.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Equation Editor listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
