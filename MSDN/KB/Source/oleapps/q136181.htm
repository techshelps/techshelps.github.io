Organization Chart 2.0: Shadows Are Always Black

 Last reviewed: January 23, 1998
Article ID: Q136181

The information in this article applies to:

  • Microsoft Organization Chart for Windows 95, version 2.0
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 2.0
  • Microsoft PowerPoint for Windows, versions 4.0, 4.0a, 4.0c

SYMPTOMS

You can't change the shadow color of a Microsoft Organization Chart 2.0 object by changing the PowerPoint color scheme. The shadow color of a Microsoft Organization Chart 2.0 object is always black.

Note that this behavior is different from Organization Chart 1.0 objects.

WORKAROUND

To create an organization chart that has shadows that are a color other than black, use any of the following methods.

Method 1: Apply a Shadow to Everything in the Organization Chart

  1. Create an organization chart without any type of shadow.

  2. On the File menu, click Close And Return To Presentation.

  3. In PowerPoint 97, click the Shadow button on the Drawing Toolbar. In PowerPoint 4.0 or 7.0, click Shadow on the Format menu.

  4. In PowerPoint 97, click Shadow Settings and then click the Shadow Color button. In PowerPoint 4.0 or 7.0, click the shadow color you would like from the Color list box.

  5. Click OK.

This method will apply shadows to everything (including the lines and the title of the organization chart); however, the shadows can be any color.

Method 2: Convert the Organization Chart to a Picture and Recolor it

  1. Create an organization chart, and put shadows on the boxes that you want to have shadows.

  2. On the File menu, click Close And Return To Presentation.

  3. In PowerPoint, click Cut on the Edit menu.

  4. Click Paste Special on the Edit menu. Select Picture from the list of available data types.

  5. Click the Recolor button on the Picture toolbar (PowerPoint 97) or the Recolor on the Tools menu (PowerPoint 4.0 or 7.0).

  6. Select (put an "x" in) the check box to the left of the black color swatch. Choose the shadow color you would like from the list of colors to the right.

  7. Click OK.

This method will change everything that is black in the organization chart to the color of your choice.

Method 3: Ungroup the Organization Chart

  1. Create an organization chart, putting shadows on whatever boxes you want to have shadows.

  2. On the File menu, click Close And Return To Presentation.

  3. On the Draw menu, click Ungroup. You will see the following message: 

         This is an embedded object, not a group. Converting it to a
     PowerPoint drawing will discard embedded data or linking
     information. Convert to PowerPoint drawing?

    Note: In PowerPoint 97, the Draw menu is on the Drawing Toolbar.

  4. Click OK.

  5. Now you can click each shadow individually and click the Shadow button on the Drawing Toolbar (PowerPoint 97) or Colors And Lines on the Format menu (PowerPoint 4.0 and 7.0) to change the color of each shadow.

While this method is the most time-consuming, it provides the greatest amount of control over the colors on your organization chart. Note that once you ungroup the chart, you won't be able to edit it in Microsoft Organization Chart.
