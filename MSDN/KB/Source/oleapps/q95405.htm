Pasting Equations into Micrografx Designer, Charisma, or Draw

 Last reviewed: September 10, 1996
Article ID: Q95405
The information in this article applies to:
  • Microsoft Equation Editor version 1.0
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c
  • Microsoft Windows operating system versions 3.0, 3.1

SUMMARY

If you paste an equation from Microsoft Equation Editor into Micrografx Designer version 3.1 or Micrografx Charisma version 2.1, the characters of the equation either appear on top of one another or only the first character displays.

This problem does not occur with Micrografx Designer version 3.1 plus OLE or Micrografx Draw version 3.0 plus OLE (10/92 release).

MORE INFORMATION

Neither Micrografx Charisma version 2.1 nor Designer version 3.1 can interpret TrueType font technology; therefore, these applications substitute an outline font for the TrueType fonts used by Microsoft Equation Editor. Equation Editor treats these Micrografx outline fonts as a set of graphical objects; Designer and Charisma treat them as fonts. As a result, the Micrografx applications apply font properties to the pasted equations instead of symbol properties.

According to Micrografx, Inc., the Designer version 3.1 Plus OLE and Draw version 3.0 Plus OLE applications are compatible with Windows version 3.1. These applications support the Windows version 3.1 object-linking-and- embedding (OLE) protocol and TrueType font technology.

For more information or technical support with Micrografx Designer, Draw or Charisma, call Micrografx Technical Support.
