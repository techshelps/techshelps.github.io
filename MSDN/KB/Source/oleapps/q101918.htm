WordArt: Shaded Letters with Closed Areas Print Striped

 Last reviewed: September 9, 1996
Article ID: Q101918
The information in this article applies to:
  • Microsoft WordArt version 2.0
  • Microsoft Publisher version 2.0

SYMPTOMS

In WordArt 2.0 objects, letters that contain closed areas (such as O, B, A, and P) may print striped. This problem occurs if you format them for a 50-percent shading and print them to a PostScript printer. Narrow horizontal white stripes appear in the letters that contain closed areas. These stripes do not display on the screen.

STATUS

Microsoft has confirmed this to be a problem in Publisher version 2.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Create a WordArt object that includes some letters that have closed areas.

  2. Shade the letters 50-percent gray (the pattern in the second column of the second row in the Shading dialog box).

  3. Print the page to a PostScript printer.

The letters without closed areas print the 50-percent shading correctly, but the letters with closed areas contain horizontal white stripes.
