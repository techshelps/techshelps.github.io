Equation Editor: Insertion Point Disappears with MT Extra Font

 Last reviewed: September 10, 1996
Article ID: Q79927
The information in this article applies to:
  • Microsoft Equation Editor version 1.0

SUMMARY

If you use the MT Extra soft font, which is installed with the Equation Editor, the insertion point disappears with certain point sizes in Microsoft Word for Windows version 2.0.

MORE INFORMATION

The MT Extra soft font is installed during the Word for Windows Setup for use with the Equation Editor if Setup detects that a PostScript or Hewlett-Packard (HP) LaserJet series printer is installed.

If you use this font in certain point sizes, the insertion point disappears when you start a new line (press ENTER) or when the insertion point is on a line without any text. The insertion point appears as soon as you type some text. Another symptom of this problem is that you cannot select the paragraph marker at the end of a paragraph; the insertion point does not produce a highlight.

With the Postscript driver, 7-point, 14-point, and 24-point sizes exhibit this behavior. Any scalable points between the above screen fonts also have this problem. With HP LaserJet drivers, only the 7-point size has this problem.
