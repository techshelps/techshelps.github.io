OFF: Org Chart Does Not Retain Line Thickness Settings

 Last reviewed: February 3, 1998
Article ID: Q140994

The information in this article applies to:
  • Microsoft Organization Chart for Macintosh, version 1.0
  • Microsoft Organization Chart for Macintosh, version 2.0
  • Microsoft PowerPoint 98 Macintosh Edition
  • Microsoft PowerPoint for the Macintosh, version 4.0
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Office for the Macintosh, versions 4.2 and 4.2.1

SYMPTOMS

When you set the thickness of Organization Chart connecting lines to Hairline, and if you then close and re-open the organization chart, the connecting lines will change to 1 point thickness. This will occur with embedded Organization Chart objects as well as with Organization Chart files.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Organization Chart listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
