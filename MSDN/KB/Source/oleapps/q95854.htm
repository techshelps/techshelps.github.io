Rotated Text in MS Graph Does Not Print Bold on HP LaserJet 4

 Last reviewed: September 10, 1996
Article ID: Q95854
The information in this article applies to:
  • Microsoft Graph for Windows, versions 3.0, 3.0a, 3.0b

SUMMARY

In Microsoft Graph, the default font for data labels and chart titles is Arial Bold 8 point. If you rotate text in a chart, the text may lose its bold formatting. This occurs if you are using the Hewlett- Packard (HP) LaserJet 4 driver version 3.1.03 for Microsoft Windows operating system version 3.1.

To avoid this problem, follow these steps:

  1. From the File menu, choose Print Setup.

  2. Choose the Setup button and then the Options button.

  3. In the Options dialog box, change the graphics mode option to Raster and select the Print TrueType As Graphics check box.

  4. Choose the OK button three times to close the Options, printer driver, and Print Setup dialog boxes.

The rotated chart text should now regain its bold formatting.

MORE INFORMATION

This problem does not occur with the HP III driver. As an alternative workaround, you can print to the HP LaserJet 4 using the HP III driver.
