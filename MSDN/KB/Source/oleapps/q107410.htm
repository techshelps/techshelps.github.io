Refresh Command Err Msg: ClipArt...Could Not Read File...*.PCD

 Last reviewed: February 25, 1998
Article ID: Q107410
The information in this article applies to:
  • Microsoft ClipArt Gallery version 1.0

SUMMARY

If you choose the Refresh or Add command in the Microsoft ClipArt Gallery, you may receive the following error: 

   The ClipArt Gallery could not read the file <c:\windows\anyname>.pcd
where the path to the file and the filename (<c:\windows\anyname> in this example) may vary, but the extension of the filename is always .PCD.

If you receive this error, choose OK and the ClipArt Gallery will continue to add or refresh files. The file in question is either a corrupted Kodak Photo CD graphics file or a Playlist file created by an audio application called Pocket CD by Media Vision, Inc.

MORE INFORMATION

Kodak Photo CD graphics files and Pocket CD Playlist files use the same extension, .PCD. ClipArt Gallery cannot distinguish between the two and tries to read the Playlist file as a graphic, resulting in the error.
