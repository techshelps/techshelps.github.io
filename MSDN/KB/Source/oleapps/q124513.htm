Cannot Import Embedded Microsoft Excel Chart to Graph 5.0

 Last reviewed: September 10, 1996
Article ID: Q124513
The information in this article applies to:
  • Microsoft Graph for Windows, version 5.0
  • Microsoft Graph for Macintosh, version 5.0
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Macintosh, version 5.0

SYMPTOM

In Microsoft Graph 5.0, if you try to import a chart that is embedded in a Microsoft Excel 5.0 worksheet, you may receive the following error message: 

   File does not contain a chart. Chart will not be updated.

CAUSE

The Import Chart feature in Microsoft Graph 5.0 can only detect a chart when it is on its own sheet (called a chart sheet in Microsoft Excel). Microsoft Graph 5.0 cannot detect a chart that is embedded in a worksheet.

RESOLUTION

In Microsoft Excel 5.0, to create a chart that is on its own worksheet:

  1. Select the range of data that you want to be displayed in a chart.

  2. From the Insert menu, choose Chart, and then choose As New Sheet.

To create your chart, follow the prompts in the ChartWizard.

NOTE: In Microsoft Graph 5.0, when you choose Import Chart from the Edit menu, it will always import the first chart that it finds in a workbook.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Graph version 5.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
