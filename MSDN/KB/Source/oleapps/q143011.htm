OFF4: Org Chart Must Have Default Printer Selected (Mac)

 Last reviewed: February 3, 1998
Article ID: Q143011
The information in this article applies to:
  • Microsoft Organization Chart for Macintosh, version 1.0
  • Microsoft Office for Macintosh, versions 4.2, 4.2.1
  • Microsoft PowerPoint for Macintosh, version 4.0
  • Microsoft Word for Macintosh, versions 6.0, 6.0.1
  • Microsoft Excel for Macintosh, version 5.0

SYMPTOMS

When you attempt to start Microsoft Organization Chart version 1.0 for the Macintosh, you may receive a message similar to any of the following: 

   The server program, source file, or item cannot be found.
   Make sure the program is properly installed, and that it has
   not been deleted, moved, or renamed.

   -or-

   There is no valid selection for a printer to use. Select a printing
   device using the Chooser, then restart Microsoft Organization Chart.

   -or-

   Sorry, a system error occurred.
   "MS Organization Chart"
   error type 11

CAUSE

In order to run Microsoft Organization Chart 1.0, a default printer driver must be selected. Typically, this problem occurs just after installing your System Software. A default printer driver is not established during System Software setup.

MORE INFORMATION

Double-click the MS Organization Chart 1.0 icon located in the Microsoft folder. If you receive a Type 11 error or a message informing you that there is no valid print driver, use the following steps to select a default printer driver:

  1. On the Apple menu, click Chooser.

  2. Click a printer driver icon (located in the upper-left box in the Chooser window).

  3. Close the Chooser window.

    You may see the following message: 

          You have changed your current printer. Please choose "Page Setup..."
      in all of the open programs.

  4. Click OK to finalize the driver choice.

You should now be able to access the Organization Chart from within the Office programs.
