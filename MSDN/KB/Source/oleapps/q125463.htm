How to Change Columns to Pictures in Graph 5.0

 Last reviewed: January 20, 1997
Article ID: Q125463
The information in this article applies to:
  • Microsoft Graph 97 for Windows
  • Microsoft Graph for Windows, version 5.0

SUMMARY

If you use Graph 97 you can add an image (a bitmap for example) to certain types of data markers, the chart area, the plot area, or the legend in 2-D and 3-D charts, or the walls and floor in 3-D charts.

If you use Graph 5.0 to create a 2-D column chart, you can replace some or all of the columns with pictures. Additionally, you can format picture markers in bar and column charts by stacking, stretching, or stacking and scaling them.

MORE INFORMATION

Note that the following procedure work only with Microsoft Graph 5.0 or Graph 97 for Windows; it does not work with earlier versions of Graph or with Graph for the Macintosh.

Graph 97

Use the following steps to add a picture to the data markers in column, bar, area, bubble, 3-D line, and filled radar charts:

  1. Click the chart item that you want to add a picture to.

  2. Click the arrow next to Fill Color, click Fill Effects, and then click the Picture tab.

  3. To specify a picture, click Select Picture.

  4. In the Look in box, click the drive, folder, or Internet location that contains the picture, and then double-click the picture you want.

  5. On the Picture tab, select the options you want.

NOTE: To use a picture as a data marker in 2-D line, scatter, and unfilled radar charts, select the picture in a picture editing program, click Copy (Edit Menu), switch to Microsoft Graph or double-click the Microsoft Graph chart, click the data series, and then click Paste (Edit menu).

Graph 5.0

To replace your default columns or bars with graphics in Graph 5.0, use the following steps:

  1. Find a picture to bring into your document or presentation, for example, in Paintbrush.

  2. Select the picture, copy it from Paintbrush so that it is on the Clipboard and choose the Cut or Copy command from the Edit menu.

  3. Insert a Graph into your document or presentation.

  4. When the Graph application starts, click Chart Type on the Format menu.

  5. Choose 2-D from the Chart Dimensions section, and then select either Bar or Column.

  6. Click one of the columns in the graph.

    The whole series of columns is selected.

  7. On the Edit menu, click Paste.

Your picture will now replace the selected bars.

NOTE: The picture will be stretched to fit the area of the column it replaces. To format picture markers in bar and column charts, do the following:

  1. Activate the chart.

  2. Double-click the data series or individual picture marker you want to format.

  3. This displays the Format Data Series or Format Data Point dialog box. Select the Patterns tab.

  4. Under Picture Format, select the option you want: Stretch, Stack, or Stack And Scale To.

  5. Choose the OK button.
