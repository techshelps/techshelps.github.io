Error Accessing Module/OLE If Network Components Not Installed

 Last reviewed: September 18, 1996
Article ID: Q124582
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Project for Windows, version 4.0
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

In Microsoft Excel or Microsoft Project, when you create a new Visual Basic, Applications Edition, macro, or try to display a Visual Basic module, a dialog box with an exclamation point appears. This indicates that an error has occurred, but the dialog box does not include any text describing the error.

CAUSE

This problem occurs when you install Windows NT version 3.5 without any network components. When you do this, the blank dialog box appears when you attempt to load a Visual Basic module, or create a new Visual Basic macro in Microsoft Project or Microsoft Excel.

Note that OLE operations may also fail when you install Windows NT version 3.5 without network components. The error message you receive in this case varies depending on the application you are using.

WORKAROUND

To work around this problem, set up a network by doing the following: 

   1. In the Control Panel window, choose the Network icon.

   2. When the dialog box appears informing you that the network components
      are not installed and asks if you want to install them, choose OK To
      Install.

   3. When you are asked if you want Setup to autodetect your network card,
      choose No. You will then be presented with a list box with network
      card choices. Select the MS Loopback Adapter.

   4. Choose OK for the default frame type of 802.3.

   5. Choose OK for the default protocol of NWLink.

   6. In the Network Configuration screen, choose OK.

   7. Shut down and then restart your computer.

STATUS

Microsoft has confirmed this to be a problem in Windows NT Workstation and Windows NT Server version 3.5. This problem was corrected in the latest U.S. Service Pack for Windows NT version 3.5.
