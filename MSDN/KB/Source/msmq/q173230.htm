INFO: Message Queue Server Doesn't Support Non-Windows Platforms

 Last reviewed: January 28, 1998
Article ID: Q173230
The information in this article applies to:
  • Microsoft Message Queue Server, version 1.0

SUMMARY

Microsoft is not planning to provide Microsoft Message Queue Server (MSMQ) client or server software for platforms other than the Windows platform.

MORE INFORMATION

A Microsoft partner, Level8 Systems, provides an MSMQ application programming interface (API) client and gateway product, "FalconMQ," that allows access to an MSMQ enterprise from a number of platforms other than the Windows platform. Also, the FalconMQ product provides connectivity to the IBM MQSeries message queueing system.

See the MSMQ Web site listed below for more details on interoperability solutions and availability of MSMQ.

You can contact Level8 Systems at (212) 244-1234, or visit its Web site listed below.

REFERENCES

For more information, visit the Web sites at:

  • http://www.microsoft.com/msmq/
  • http://www.level8.com/
