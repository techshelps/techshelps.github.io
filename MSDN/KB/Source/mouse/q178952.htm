Err Msg: Setup Caused a Datatype Misalignment Fault...

 Last reviewed: January 7, 1998
Article ID: Q178952
The information in this article applies to:
  • Microsoft IntelliPoint software versions 2.0, 2.0a, 2.1
  • Microsoft Windows NT Workstation versions 3.51, 4.0
  • Microsoft Windows NT Server versions 3.51, 4.0
  • Microsoft Windows NT Server Enterprise Edition version 4.0

SYMPTOMS

When you attempt to install any of the versions of the IntelliPoint software listed above, you may receive an error message similar to the following: 

   Setup caused a datatype misalignment fault in module Kernel32.dll.

   If you choose Ignore, slower software emulation of misaligned accesses
   will be used.
If you click Ignore, you receive an error message similar to the following: 

   The image file a:\_setup.bin is valid, but is for a machine type other
   than the current machine.

CAUSE

This behavior can occur if you attempt to install the IntelliPoint software on a computer that uses a non-Intel processor.

RESOLUTION

To work around this behavior, do not install the IntelliPoint software on a computer that uses a non-Intel processor.

Without the IntelliPoint software installed, you can use the Microsoft IntelliMouse or Microsoft IntelliMouse TrackBall as a standard two-button pointing device in Windows NT 3.51 and 4.0.

MORE INFORMATION

The versions of the IntelliPoint software listed above are not supported on computers that use an Alpha, MIPS, or PPC processor.
