Wheel Does Not Scroll Smoothly in Windows NT Explorer

 Last reviewed: December 4, 1997
Article ID: Q177608
The information in this article applies to:
  • Microsoft IntelliPoint software versions 2.0, 2.0a, 2.1
  • Microsoft IntelliMouse version 1.0
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Server Enterprise Edition version 4.0

SYMPTOMS

When you roll the IntelliMouse wheel in Windows NT Explorer, the Windows NT Explorer window may not scroll smoothly.

CAUSE

This problem can occur if the following conditions exist:

  • You use two input devices on your computer.
  • Windows NT detects a standard PS/2 mouse before the IntelliMouse pointing device.

RESOLUTION

If you want to disable the wheel functions in Windows NT, follow these steps:

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Mouse.

  3. On the Wheel tab, click the Turn On The Wheel and Turn On The Wheel Button check boxes to clear them.

  4. Click OK, and then close Control Panel.

STATUS

Microsoft has confirmed this to be a problem in Microsoft IntelliPoint version 2.0, 2.0a, and 2.1. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
