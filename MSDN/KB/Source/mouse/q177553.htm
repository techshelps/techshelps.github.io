Cannot Configure the Odometer Feature in IntelliPoint

 Last reviewed: December 10, 1997
Article ID: Q177553
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.1

SYMPTOMS

When you change the configuration of the Odometer feature in the Mouse Properties dialog box, the configuration you specify may have no effect.

CAUSE

This issue can occur if you change the configuration and you then try to change the configuration again.

RESOLUTION

To resolve this issue, install the latest version of the IntelliPoint software. To do so, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q177274
   TITLE     : IntelliPoint Software 2.1: Point32 Update Available
To work around this issue, restart your computer, and then change the Odometer configuration. To do so, use the following steps:

  1. Click Start, click Shut Down, click Restart (or Restart The Computer), and then click OK (or Yes).

  2. Click Start, point to Settings, and then click Control Panel.

  3. Double-click Mouse.

  4. On the Productivity tab, click the Turn On The Odometer check box to clear it if it is already selected, or to select it if it is already cleared.

  5. Click OK, and then close Control Panel.
