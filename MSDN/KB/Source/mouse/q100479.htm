Mouse Driver 9.0 Does Not Work in Windows 3.0 Real Mode

 Last reviewed: September 16, 1996
Article ID: Q100479
The information in this article applies to:
  • Microsoft Mouse Driver for Windows, version 9.0
  • Microsoft Windows operating system version 3.0

The Microsoft Mouse driver 9.0 does not work with Windows 3.0 when running in real mode.

When working in real mode, the 9.0 driver completes the installation procedures, but when prompted to restart Windows, it stops responding (hangs) at the logo screen or the MS-DOS command prompt.
