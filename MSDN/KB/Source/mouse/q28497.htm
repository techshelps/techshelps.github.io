Interrupt Setting for the IBM PS/2 Model 30

 Last reviewed: September 16, 1996
Article ID: Q28497
The information in this article applies to:
  • Microsoft InPort Mouse, version 1.x
  • Microsoft Mouse Driver for MS-DOS, versions 6.x, 7.x, and 8.x

SUMMARY

When you install the mouse InPort interface card on a PS/2 Model 30, Microsoft recommends that you set the J4 interrupt to jumper 2 or 3.

Please check that other peripheral boards are not using the same interrupt setting as the mouse. Do not use pair 5 for the setting.
