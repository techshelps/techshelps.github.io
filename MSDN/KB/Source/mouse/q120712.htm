Creating New & Editing Existing Animated Cursors

 Last reviewed: September 12, 1996
Article ID: Q120712
The information in this article applies to:
  • Microsoft IntelliPoint Software Home Edition version 1.0
  • Microsoft IntelliPoint software version 1.0

SUMMARY

With both the Microsoft IntelliPoint Software Home Edition and the Standard IntelliPoint software, you can create additional animated cursors as well as edit existing animated cursors.

MORE INFORMATION

IntelliPoint uses a sequence of numbered cursor files to display animated cursors. You can create a new series of files to generate a new animated cursor, and you can edit an existing animated numbered series to modify an existing cursor. The examples below explain how to edit and create animated cursors.

Example: Editing Existing Animated Cursor

Home "Pop-art: Pop" includes the following cursor files:

POP01.CUR, POP02.CUR, POP03.CUR, POP04.CUR, and POP05.CUR

To edit this cursor, use the RENAME command to add two additional cursors (SARROW.CUR and SWAIT.CUR), replace two of the cursors in the animated sequence (POP01.CUR and POP03.CUR) with one (WNO.CUR) being at the beginning of the numbered series and the other (DWAIT.CUR) being in the middle:

  1. Rename POP01.CUR to POPAAA.CUR.

  2. Rename POP03.CUR to POPBBB.CUR.

  3. Rename WNO.CUR to POP01.CUR.

  4. Rename DWAIT.CUR to POP03.CUR.

  5. Rename SARROW.CUR to POP06.CUR.

  6. Rename SWAIT.CUR to POP07.CUR

Example: Creating New Animated Cursors for Microsoft Home Mouse

Create a numbered-series animated icons featuring a dancer by using the RENAME command to make the following changes:

  1. Rename DALL.CUR to DANCE_01.CUR.

  2. Rename DARROW.CUR to DANCE_02.CUR.

  3. Rename DEW.CUR to DANCE_03.CUR.

  4. Rename DNESW.CUR to DANCE_04.CUR.

  5. Rename DNS.CUR to DANCE_05.CUR.

  6. Rename DNWSE.CUR to DANCE_06.CUR.

  7. Rename DSTART.CUR to DANCE_07.CUR.

  8. Rename DWAIT.CUR to DANCE_08.CUR.

NOTE: You can use the COPY command in place of the RENAME command. Doing so preserves the original filename of the cursor for later use and creates a backup of the cursor file.

Numbered-series animated cursors are usually designated with 01, 101, 201, or 11 in a series of cursor files with the file extension .CUR. If you click the first numbered-series file in a list (for example, DANCE_01.CUR), a preview of the cursor animation is displayed. Selecting other grouping files (for example, DANCE_02.CUR [or any file without 1]) does not generate a preview of the animation.
