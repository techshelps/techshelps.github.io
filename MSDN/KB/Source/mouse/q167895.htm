Context Menu Is Displayed When EasyBall Button Is Pressed

 Last reviewed: May 1, 1997
Article ID: Q167895
The information in this article applies to:
  • Microsoft EasyBall version 1.0
  • Microsoft IntelliPoint software version 1.1

SYMPTOMS

When you press the button on the Microsoft EasyBall, a context menu may be displayed.

CAUSE

This behavior occurs when the right mouse button is configured as the primary mouse button.

RESOLUTION

To resolve this issue, configure the left mouse button to be the primary mouse button. To do so, follow these steps:

  1. In Control Panel, double-click Mouse.

  2. Click the Buttons tab.

  3. In the Button Configuration section, click Right-Handed.

  4. Click OK.

MORE INFORMATION

This behavior does not occur when you use the EasyBall with Microsoft IntelliPoint software version 2.0.
