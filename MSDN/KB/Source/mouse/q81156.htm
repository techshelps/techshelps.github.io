Norton Desktop for Windows and Microsoft Mouse Control Panel

 Last reviewed: September 16, 1996
Article ID: Q81156
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 8.10, 8.20, and 9.0

SUMMARY

If you have Norton Desktop for Windows on your machine and you run the Microsoft Mouse Setup program, Norton Desktop for Windows may prevent the Microsoft Mouse Setup program from creating a Mouse Control Panel under Microsoft Windows. This information applies to all versions of the Microsoft Mouse for MS-DOS systems.

MORE INFORMATION

If you do NOT have an existing Microsoft Mouse Control Panel under Microsoft Windows and you are running Norton Desktop for Windows, Norton Desktop for Windows will prevent the Microsoft Mouse Setup program from creating a Mouse Control Panel under Microsoft Windows.

To create the Mouse Control Panel under Microsoft Windows, do the following:

  1. Choose New from the File menu under Norton Desktop for Windows Program Manager.

  2. Create a new item (not a group) and fill in the following lines appropriately: 

          Title: Cpanel (or some other name)
      Program/Document/Script: C:\MOUSE\POINT.EXE*
      Startup Directory: C:\MOUSE*
      Description: Leave blank
      Shortcut Key: None

    * Please note that the Microsoft Mouse has three different default directory names according to which version of the Microsoft Mouse driver you are installing: MOUSE1, MOUSE, or BALLPNT. Adjust the above information accordingly.

If you have an existing Microsoft Mouse Control Panel under Microsoft Windows, then the Microsoft Mouse Setup program will be able to update the Control Panel. If problems occur, follow the steps above to manually install the Microsoft Mouse Control Panel.
