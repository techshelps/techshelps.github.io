Options Unavailable in Mouse Properties

 Last reviewed: October 11, 1996
Article ID: Q157459
The information in this article applies to:
  • Microsoft IntelliPoint software version 1.1

SYMPTOMS

After you install the IntelliPoint 1.1 software, the following items may be unavailable in the IntelliPoint Tools And Controls tool or the Mouse tool in Control Panel:

  • Button Selection on the Buttons tab
  • Set Orientation on the Orientation tab
  • Acceleration on the Speed tab

CAUSE

This behavior can occur if you were previously using a LogiTech serial mouse and the driver for the LogiTech mouse is still installed.

RESOLUTION

To resolve this issue, follow these steps:

  1. In Control Panel, double-click Mouse.

  2. Click the General tab, and then click Change.

  3. Select the appropriate mouse driver (such as Microsoft Serial Mouse), and then click OK.

  4. Click Close.

  5. When you are prompted to restart your computer, do so.
