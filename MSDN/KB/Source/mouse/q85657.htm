/B Not Valid for ORIENT.EXE in BallPoint Mouse Driver 8.2/8.2a

 Last reviewed: September 16, 1996
Article ID: Q85657
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, version 8.2

SUMMARY

The /B option, which is a valid switch for COMPASS.EXE in versions 8.0 and 8.1 of the BallPoint Mouse driver, is not valid for ORIENT.EXE in BallPoint Mouse driver versions 8.2 and 8.2a.

MORE INFORMATION

COMPASS.EXE in BallPoint Mouse driver versions 8.0 and 8.1 includes a /B option that is added to the AUTOEXEC.BAT file by the Setup program. This option causes the line 

   c:\ballpt\compass /b
to be removed from the AUTOEXEC.BAT file after it is run. The /B option is no longer necessary with ORIENT.EXE, because it will remove itself from the AUTOEXEC.BAT file after it is run. Adding the /B option to ORIENT.EXE in the AUTOEXEC.BAT file will produce the following error message: 

   Invalid command-line switch
