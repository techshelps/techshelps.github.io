Paintbrush Requires OS2MOUSE.SYS

 Last reviewed: September 16, 1996
Article ID: Q49340

SUMMARY

To run Microsoft Paintbrush in the OS/2 DOS Compatibility Box, you need to load the OS2MOUSE.SYS driver in the CONFIG.SYS file. This driver can be found on the Microsoft Mouse Mouse Menus/BOA disk.

The highest graphics mode the mouse functions at in the DOS Box is EGA. OS/2 Version 1.10 does not support the VGA graphics cursor.
