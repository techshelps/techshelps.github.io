CalComp Digitizer Drivers May Disable IntelliMouse Wheel

 Last reviewed: November 21, 1997
Article ID: Q177137
The information in this article applies to:
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliMouse TrackBall, version 1.0
  • Microsoft IntelliPoint software versions 2.0, 2.1

SYMPTOMS

The wheel on your Microsoft IntelliMouse or IntelliMouse Trackball may no longer function if you install one of the following CalComp digitizing tablet drivers:

  • CalComp Ultraslate digitizer, versions 4.0, 4.2
  • CalComp digitizer (all other models), version 3.3

CAUSE

The CalComp drivers may interrupt the reading of the mouse message queue.

RESOLUTION

To resolve this issue, add Mswheel.exe to the Startup folder. To do so, use the following steps:

  1. Right-click Start, and then click Open.

  2. Double-click the Programs folder.

  3. Double-click the Startup folder.

  4. On the File menu, point to New, and then click Shortcut.

  5. Type the following line, and then click Next 

          <drive>:\<windows>\system\mswheel.exe

    where <drive> is the drive on which Windows is installed and <windows> is the folder in which Windows is installed.

  6. Click Finish.

  7. Close all open windows, and then restart the computer.

MORE INFORMATION

To inquire about obtaining an updated driver for your digitizing tablet, contact CalComp Input Technologies.
