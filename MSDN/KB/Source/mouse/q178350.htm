How to Use the MousInfo Tool Included with IntelliPoint Software

 Last reviewed: December 19, 1997
Article ID: Q178350
The information in this article applies to:
  • Microsoft IntelliPoint software versions 2.0, 2.1
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliMouse TrackBall, version 1.0

SUMMARY

MousInfo is a 32-bit diagnostic tool that shows you if the computer recognizes the pointing device messages. This article describes how to use the MousInfo tool included with the IntelliPoint software.

MORE INFORMATION

To run the MousInfo tool, follow these steps:

  1. Click Start, click Run, type the following command in the Open box, and then press ENTER: 

          c:\msinput\mouse\mousinfo.exe

    IntelliPoint installs the Mousinfo.exe and Mousinfo.dll files in the Msinput\Mouse folder by default. If you installed IntelliPoint in a different folder, use that path when you type the command.

  2. On the Taskbar, click the MousInfo tool.

The MousInfo tool uses the following indicators for each function of the pointing device: 

   Indicator     Function
   ------------------------------------------------
   L             Left button
   M             Middle button
   R             Right button
   W             Wheel state
   X             Horizontal position of the pointer
   Y             Vertical position of the pointer
   ScrollLines   Scrolling setting configuration
When you press a button, the value of L, M, or R changes depending on the following button positions: 

   Value   Meaning
   ----------------------
   0       Button is up
   1       Button is down
When you roll the wheel, the value of W changes depending on the following wheel motions: 

   Value   Meaning
   ------------------------------------------------
   +       Wheel roll toward the front of the mouse
   -       Wheel roll toward the back of the mouse
When you move the pointing device, the value of X and Y changes depending on the pointer's position. The numbers that you see depend on your screen resolution.

The ScrollLines value can be from -1 to 32767. The value -1 is reserved for the "Scroll one screen at a time" setting.

You can display mouse messages if you right-click MousInfo, and then click Display Mouse Messages. When you turn the Display Mouse Messages feature on, you see the following messages: 

   Message            Meaning
   ------------------------------------------------------------------
   WM_xBUTTONUP       Left, middle, or right button is released
   WM_xBUTTONDOWN     Left, middle, or right button is pressed
   WM_xBUTTONDBLCLK   Left, middle, or right button is double-clicked
   WM_MOUSEWHEEL      IntelliMouse wheel is rolled
