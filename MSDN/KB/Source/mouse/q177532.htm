IntelliPoint Err Msg: Command or File Name Not Found

 Last reviewed: December 3, 1997
Article ID: Q177532
The information in this article applies to:
  • Microsoft IntelliPoint software versions 2.0, 2.1

SYMPTOMS

If you install the Microsoft IntelliPoint software in a folder whose name contains an extended character, you may receive the following error message when you start the computer in MS-DOS mode: 

   Command or file name not found.

CAUSE

The extended character in the folder name cannot be parsed when the Autoexec.bat file attempts to load the Mouse.exe file.

RESOLUTION

To resolve this issue, uninstall the IntelliPoint software, and then reinstall the IntelliPoint software in the default folder suggested by the IntelliPoint Setup program. For information about how to uninstall the IntelliPoint software, see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q166004
   TITLE     : No Uninstall Option in IntelliPoint Setup Program
