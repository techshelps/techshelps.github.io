Mouse Does Not Work with Diamond Video Snooper Utility Running

 Last reviewed: September 12, 1996
Article ID: Q122163
The information in this article applies to:
  • Microsoft Mouse driver for Windows, versions 8.x, 9.0, 9.01, 10.0
  • Microsoft IntelliPoint software version 1.0

SYMPTOMS

When you run the Video Snooper utility, which is supplied by Diamond Computer Systems with the VideoStar video capture board, your mouse no longer works. The mouse pointer disappears, and it reappears only when you quit Video Snooper.

STATUS

At this time, Diamond Computer Systems technical support has no additional information about this problem, so no workaround is currently available.

MORE INFORMATION

Microsoft has reproduced this problem on a computer with the following configuration:

  • Video Snooper version 1.02
  • Microsoft Mouse
  • A Microsoft Mouse driver between version 8.0 and version 9.01, or Microsoft IntelliPoint software version 1.0
  • Hewlett-Packard (HP) Vectra computer with an S3 chip set

NOTE: By default, Diamond Video Snooper runs when you start Windows.
