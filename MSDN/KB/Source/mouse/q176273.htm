Programs That Support Horizontal Scrolling w/ IntelliMouse Wheel

 Last reviewed: November 14, 1997
Article ID: Q176273
The information in this article applies to:
  • Microsoft IntelliPoint software versions 2.0, 2.0a, 2.1
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliMouse TrackBall version 1.0

SUMMARY

The IntelliMouse wheel supports only vertical scrolling in most programs.

MORE INFORMATION

When you roll the IntelliMouse wheel you can scroll through a document without clicking the scroll bar. However, the wheel does not generally support scrolling horizontally. Horizontal scrolling is available only in specific programs.

Programs that support horizontal scrolling include:

  • Windows Explorer

    Windows Explorer supports horizontal scrolling with the IntelliMouse wheel if there is no vertical scroll bar.

  • Microsoft Excel 97

    Microsoft Excel 97 supports a horizontal panning feature that you can use by holding the wheel button down and moving the mouse to the left or right.
