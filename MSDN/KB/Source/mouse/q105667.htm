Mouse Version 9.0 and Seiko Smart Label Printer Driver

 Last reviewed: September 16, 1996
Article ID: Q105667
The information in this article applies to:
  • Microsoft Mouse Driver for Windows, version 9.0
  • Microsoft Mouse Driver for MS-DOS, version 9.0
  • Microsoft Windows for Workgroups version 3.1
  • Microsoft Windows operating system version 3.1

SYMPTOMS

There are two problems that can occur if you are using an outdated Seiko Smart Label printer driver with the Microsoft Mouse driver 9.0:

  • In Windows, if you are using the version 9.0 Microsoft Mouse driver for Windows and version 1.09 or earlier of the Seiko Smart Label Windows printer driver, your computer may stop responding (hang) during the Windows boot sequence or when you try to print from Windows.
  • If you are using version 2.13 or earlier of the Seiko Smart Label MS-DOS printer driver, your computer either hangs or generates a sharing violation when you try to load the 9.0 Microsoft Mouse driver for MS-DOS.

The Seiko Smart Label MS-DOS driver loads as SLP.EXE in the AUTOEXEC.BAT file or SLP.SYS in the CONFIG.SYS file.

NOTE: These symptoms occur only if the Seiko Smart Label driver is resident in memory.

RESOLUTION

To correct this problem, update both the Windows and MS-DOS printer drivers to version 3.0 by contacting Seiko Technical Support.

NOTE: You can also work around this problem by using an earlier version of the Microsoft Mouse driver.
