IntelliPoint 2.0 Cursor Schemes Not Available in Windows NT

 Last reviewed: April 7, 1997
Article ID: Q164871
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.0
  • Microsoft Windows NT Server versions 3.5, 3.51
  • Microsoft Windows NT Workstation versions 3.5, 3.51

SYMPTOMS

In the Cursors tool in Windows NT Control Panel, the Microsoft IntelliPoint software cursor schemes are not available to all members of the Administrators group. Only the administrator logged on during the installation of the IntelliPoint software has access to the IntelliPoint cursor schemes. Other administrators have access only to the default Windows NT cursor schemes. The IntelliPoint cursor files are available, but the cursor schemes are not.

RESOLUTION

To work around this problem, each administrator wanting access to the IntelliPoint cursor schemes should log on to the computer and then install the IntelliPoint software.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft IntelliPoint software version 2.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
