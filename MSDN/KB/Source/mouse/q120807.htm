ErrMsg When Setting Primary Button: Error Setting System Value

 Last reviewed: September 12, 1996
Article ID: Q120807
The information in this article applies to:
  • Microsoft IntelliPoint Software Home Edition version 1.0
  • Microsoft IntelliPoint software version 1.0

SYMPTOMS

The following error occurs when you try to set the primary button in IntelliPoint Tools & Controls: 

   Error: setting system value. An error occurred while trying to set your
   selection.

CAUSE

This error occurs when you press both mouse buttons at the same time or when one of the mouse buttons sticks when you try to select the primary button.

RESOLUTION

Make sure that you click only one mouse button for the primary button.

NOTE: You can check the operation of the mouse buttons by setting one button as primary and then setting the other button as primary.

MORE INFORMATION

After you receive this error, the following message is displayed next to the Set Buttons button: 

   Unable to set buttons
