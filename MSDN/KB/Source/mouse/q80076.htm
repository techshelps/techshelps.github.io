Abnormal Colors in WordPerfect 5.1 Running Under MS-DOS 5.0

 Last reviewed: September 16, 1996
Article ID: Q80076
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 7.x, 8.x, and 9.0

SYMPTOMS

Colors change abnormally in version 5.1 of WordPerfect running under MS-DOS version 5.0.

CAUSE

This problem occurs if you load the EGA.SYS device driver in the CONFIG.SYS file on a system with an EGA video adapter. When you installed MS-DOS 5.0, the Setup program may have added the following line to your CONFIG.SYS file: 

   device=c:\dos\ega.sys
This line may cause abnormal color changes in the view document or print preview modes of WordPerfect 5.1 when loaded in conjunction with a Microsoft Mouse driver.

RESOLUTION

To solve the problem, remove or remark out the EGA.SYS line in the CONFIG.SYS file.
