Mouse 9.0 Reset Does Not Affect Orientation or Sensitivity

 Last reviewed: September 16, 1996
Article ID: Q101195
The information in this article applies to:
  • Microsoft Mouse Driver for Windows, version 9.0
  • Microsoft Windows operating system version 3.1

Choosing the Reset button when using Microsoft Mouse Manager version 9.0 in Windows 3.1 does not reset the preselected Orientation or Sensitivity settings to the default values.

The Reset button resets only those settings that appear on the first screen in Mouse Manager.
