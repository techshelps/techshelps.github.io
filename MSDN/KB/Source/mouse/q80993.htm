Packing List for Microsoft BallPoint 8.1 Update Kit

 Last reviewed: September 16, 1996
Article ID: Q80993
The information in this article applies to:
  • Microsoft BallPoint Mouse, version 1.0
  • Microsoft Mouse Driver for MS-DOS, version 8.1

SUMMARY

The following is the directory of the Microsoft BallPoint Version 8.1 Update Kit for both the 5.25-inch and 3.5-inch disk versions. Note that files ending with "$" (for example, MSWRITE.CO$) are compressed and must be decompressed with the DECOMP.EXE utility. This is only needed for the files that Setup does not automatically decompress.
