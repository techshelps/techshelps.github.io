Magnify Window May Not Follow Text as You Type

 Last reviewed: July 9, 1997
Article ID: Q153366
The information in this article applies to:
  • Microsoft IntelliPoint software version 1.1

SYMPTOMS

When you are using IntelliPoint version 1.1 with Microsoft Windows 95, the Magnify window may not follow the text as you type in a document.

RESOLUTION

If the program supports it, you can use the Zoom feature as an alternative to the Magnify feature.

Note that you may be able to improve the screen's readability by using the Accessibility features included with Windows 95. For information about the Accessibility features included with Windows 95, please see the "Accessibility Options for People with Disabilities" topic in Windows 95 Help.

MORE INFORMATION

The IntelliPoint Magnify Insertion Point feature works in Windows and Windows for Workgroups versions 3.x.
