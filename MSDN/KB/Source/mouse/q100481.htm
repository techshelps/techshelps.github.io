Mouse 9.0 Setup Doesn't Update Windows 3.0 Control Panel

 Last reviewed: September 12, 1996
Article ID: Q100481
The information in this article applies to:
  • Microsoft Mouse Driver for Windows, version 9.0
  • Microsoft Windows operating system versions 3.0, 3.00a
  • Microsoft IntelliPoint software version 1.0

When you install the Microsoft Mouse driver version 9.0 in Windows 3.0, the Mouse icon in Control Panel is not updated to the 9.0 Mouse icon.

The icon isn't updated because the Mouse 9.0 Setup program does not update the files associated with Windows 3.0 Control Panel. The settings that display in the Mouse dialog box of Control Panel are operable, but the Mouse 9.0 driver features (Screen Wrap, Magnify, and so forth) must be set with the Mouse Manager (MOUSEMGR.EXE) that is installed in the Mouse group during Setup.
