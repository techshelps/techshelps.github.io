Microsoft Mouse Doesn't Work on Commodore Mouse Port

 Last reviewed: September 16, 1996
Article ID: Q81203
The information in this article applies to:
  • Microsoft Windows operating system versions 2.x, 3.0, 3.00a, 3.1, 3.11
  • Microsoft Serial Mouse versions 1.0 and 2.0

SUMMARY

The Commodore PC 40-III, PC 20, and PC 10-III may not work with the Microsoft serial Mouse. These machines have a 9-pin port marked "mouse". This port is intended only for the Commodore 1352 mouse. The port does not work with any other mouse and must be disabled before a mouse connected to any other port can work in Windows.

This information applies to all Microsoft serial Mouse products for MS-DOS systems and to Microsoft Windows versions 2.03, 2.1, 2.11,, 3.0, 3.0a, and 3.1.

MORE INFORMATION

You can obtain further information and assistance by calling Commodore Technical Support.
