Err Msg: The Mousex32.exe File Is Linked to Missing Export...

 Last reviewed: February 19, 1998
Article ID: Q160959
2.00 WINDOWS kberrmsg kbtool

The information in this article applies to:

  • Microsoft IntelliPoint software version 2.0

SYMPTOMS

When you attempt to run the IntelliPoint Tools And Controls tool or the Mouse tool in Control Panel, you may receive the following error message: 

   C:\msinput\mouse\mousex32.exe. A device attached to the system is not
   functioning.
If you click OK, you receive the following error message: 

   The mousex32.exe file is linked to missing export cmtool32.dll :901

CAUSE

These errors can occur if any of the following conditions exist:

  • Both the Cmtool16.dll and Cmtool32.dll files exist on your hard disk.
  • Multiple versions of the Cmtool32.dll file exist on your hard disk.

RESOLUTION

To resolve this problem, follow these steps:

  1. Search your hard disk for the Cmtool16.dll and Cmtool32.dll files, and then rename all copies of these files that are found.

  2. Reinstall the IntelliPoint 2.0 software.

  3. Restart your computer.
