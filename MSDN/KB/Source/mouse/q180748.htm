Compaq Keys Stop Working After Installing IntelliPoint Software

 Last reviewed: February 9, 1998
Article ID: Q180748
The information in this article applies to:
  • Microsoft IntelliPoint software versions 1.0, 1.1, 2.0, 2.1

SYMPTOMS

When you install the Microsoft IntelliPoint software, the programmable keys on your Compaq Armada laptop computer may no longer function properly.

RESOLUTION

To work around this issue, quit the Point32 program. To do so, follow these steps:

  1. Press CTRL+ALT+DELETE to open the Close Program dialog box.

  2. Click Point32, and then click End Task.

NOTE: Because Point32 starts each time you start Windows, you must quit the Point32 program by following these steps each time you start Windows.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Point32 program is a tool installed by the IntelliPoint software.
