Earlier Versions of Generic Cad May Not Work with InPort

 Last reviewed: September 16, 1996
Article ID: Q32129
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, versions 7.x and 8.x
  • Microsoft InPort Mouse, version 1.x

SUMMARY

Generic Software, maker of Generic CAD, recommends Interrupt 3 for the J4 setting of the bus mouse.

Be aware that older versions (possibly releases 1 and 2) of Generic Cad may not work correctly with versions of the bus mice that include the InPort Interface. Because Generic CAD goes directly to the hardware (that is, it uses its own mouse driver), you must obtain an update to release 1 or 2.

MORE INFORMATION

To obtain an update, please contact Generic Software.
