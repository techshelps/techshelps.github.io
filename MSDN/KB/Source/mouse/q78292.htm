Problems with the Microsoft Mouse Driver 8.1 Setup

 Last reviewed: September 16, 1996
Article ID: Q78292
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, version 8.1

SUMMARY

A copy of the disk containing the software for the Microsoft Mouse driver version 8.1 does not let you run Setup if the volume label is anything except Setup. The Microsoft Mouse Setup program runs only on a disk made using the MS-DOS DISKCOPY command.

This information applies to version 8.1 of the Microsoft Mouse driver for MS-DOS based systems.

MORE INFORMATION

If you try to run Setup with a copied version of Microsoft Mouse 8.1 driver, the following message is displayed:

*****************************************************

Microsoft Mouse Setup

Insert Disk

Please insert the disk labeled SETUP into drive A.

Press Enter to continue or ESC to return to main menu.

******************************************************

When you copy the files from one disk to another, then try to run Setup, the copied disk prompts you to put in the Setup disk as outlined above. This is because the copied disk has no label, or is not labeled as Setup.

If you are not sure if the disk is labeled or you think it is labeled, type "label" (without the quotation marks) at the drive prompt that the disk is in. The label needs to be Setup for the Microsoft Mouse 8.1 Setup to work

Use one of the following two methods to label your disk:

Method 1

Type "label" (without the quotation marks) at the drive prompt that the disk is in. The following message is displayed: 

   Volume in drive ( Drive name ) has no label
   Volume Serial Number is ( some number )
   Volume Label (11 Characters, ENTER for none)?
-or- 

   Volume in drive ( Drive name ) is ( label name )
   Volume Label (11 Characters, ENTER for none)?
On the line where the message says "Volume Label (11 Characters, ENTER for none)?", type "setup" and press ENTER. This labels your disk as Setup. You can now run the Setup program.

Method 2

Change drives to the drive that the disk is in. At the drive prompt, type "Label Setup" (without the quotation marks), then press ENTER. This labels the copied disk as Setup. You can now run the Setup program
