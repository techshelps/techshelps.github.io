Errors Running Quicken for Windows with IntelliPoint 1.1

 Last reviewed: September 12, 1996
Article ID: Q141719
The information in this article applies to:
  • Microsoft IntelliPoint software version 1.1
  • Microsoft IntelliType software version 1.1
  • Microsoft EasyBall software version 1.0
  • Microsoft Windows 95

SYMPTOMS

When you run Quicken 4.0 for Windows or Quicken Deluxe 5.0 for Windows in Windows 95, you may receive general protection (GP) fault or segment load failure error messages at random intervals.

CAUSE

Point32.exe, a tool included with IntelliPoint software version 1.1, IntelliType software version 1.1, and EasyBall software version 1.0 is not compatible with Quicken 4.0 for Windows and Quicken Deluxe 5.0 for Windows.

RESOLUTION

Microsoft has released updated versions of the Point32.exe and Point32.dll files that correct this problem. For information about obtaining these updates, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q146216
   TITLE     : Updated Pointer Files Available
