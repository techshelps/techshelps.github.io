The IntelliMouse Wheel Does Not Respond with Pointix Engine

 Last reviewed: November 26, 1997
Article ID: Q177380
The information in this article applies to:
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliMouse TrackBall, version 1.0
  • Microsoft IntelliPoint software versions 2.0, 2.0a, 2.1

SYMPTOMS

When you start your computer, the IntelliMouse or IntelliMouse Trackball wheel may not respond, or some IntelliPoint features may not function correctly.

CAUSE

This behavior can occur if you install Pointix Engine for Windows 95, version 2.60 or earlier.

RESOLUTION

To resolve this behavior, use one of the following methods:

Uninstall Pointix Engine

To uninstall Pointix Engine, follow these steps:

  1. On the right side of the taskbar, right-click the Pointix Engine icon, and then click Exit Pointix.

  2. Click Start, point to Settings, and then click Control Panel.

  3. Double-click Add/Remove Programs.

  4. On the Install/Uninstall tab, click Pointix Engine <version> (where <version> is 2.60 or earlier), click Add/Remove, and then click Yes.

  5. Close the Add/Remove Programs dialog box.

Quit Pointix Engine Before Starting IntelliPoint

To quit Pointix Engine and then start the IntelliPoint software, follow these steps:

  1. On the right side of the taskbar, right-click the Pointix Engine icon, and then click Exit Pointix.

  2. Click Start, point to Find, and then click Files Or Folders.

  3. In the Named box, type "point32.exe" (without quotation marks), and then click Find Now.

  4. In the list of found files, double-click the Point32.exe file.

  5. In the Named box, type "mswheel.exe" (without quotation marks), and then click Find Now.

  6. In the list of found files, double-click the Mswheel.exe file.

  7. Close the Find dialog box.

NOTE: Pointix Engine restarts when you restart your computer. If you want to use this method, you must use it each time you start your computer.
