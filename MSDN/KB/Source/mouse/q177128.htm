Assigned Wheel Button Features Disabled When You Start PointCast

 Last reviewed: November 21, 1997
Article ID: Q177128
The information in this article applies to:
  • Microsoft IntelliMouse version 1.0
  • Microsoft IntelliMouse TrackBall, version 1.0
  • Microsoft IntelliPoint software versions 2.0, 2.1

SYMPTOMS

When you use the Microsoft IntelliPoint software to assign either the Double-Click feature or the Help (F1) feature to the IntelliMouse wheel button, that feature may be disabled when you start a 16-bit version of PointCast Business Network. The assigned wheel button feature is restored when you quit PointCast Business Network.

RESOLUTION

To resolve this issue, contact PointCast Inc. to inquire about obtaining a 32-bit version of PointCast Business Network software.
