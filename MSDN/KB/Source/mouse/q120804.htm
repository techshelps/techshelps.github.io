Differences Between Home & Standard Mouse IntelliPoint Options

 Last reviewed: September 12, 1996
Article ID: Q120804
The information in this article applies to:
  • Microsoft IntelliPoint software versions 1.0, 1.1
  • Microsoft IntelliPoint Software Home Edition versions 1.0, 1.1

SUMMARY

The Microsoft Home Mouse with IntelliPoint software Home edition contains a subset of the Standard Mouse with IntelliPoint software options. Options located under the IntelliPoint Manager for the Home edition are fewer and have different animation schemes.

MORE INFORMATION

The Standard edition of the IntelliPoint software includes the following options:

  • IntelliPoint Toolbar
  • Buttons=Set Buttons, Set Both Buttons, Short Cuts (secondary and both)
  • Orientation=tells the software which way is up
  • Clicking=Double Click and Click Saver
  • Speed=Pointer Speed, Acceleration, and Smart Speed
  • Appearance=Schemes for Pointer Library (Animal, Autos, Boating, Dance, Foodfight, Grow Black, Grow Reverse, Grow White, Hands, Large Black, Large Reverse, Large White, Music, Planes, Small Black, Small Reverse, Small White, Space, Sports, Windows Default, and Worktools)
  • Visibility=Sonar, Trails, and Vanish
  • Activity=Snap-To, Pointer Wrap, LineLock, and Focus
  • Special Effects=Include: Unassigned, Comix, Construction, Fright Night, Jungle, and Lazy Afternoon

The Home edition of the IntelliPoint software includes the following options:
  • Clicking=Double Click but NO Click Saver
  • Speed=Pointer Speed, Acceleration but NO Smart Speed
  • Appearance=same Schemes for Pointer Library as for Standard edition
  • Visibility=Sonar, Trails, but NO Vanish
  • Special Effects=Include: Unassigned, Magic, Nautical, Party Time, Pop-art, and Sci-fi

The following options are not included in the Home edition:
  • No IntelliPoint Toolbar
  • No Orientation
  • No Activity
