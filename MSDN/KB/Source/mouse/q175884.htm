IntelliPoint Features Stop Working After Adding New Mouse

 Last reviewed: October 29, 1997
Article ID: Q175884
The information in this article applies to:
  • Microsoft IntelliPoint software versions 2.0, 2.0a, 2.1

SYMPTOMS

When you add a new mouse using the Add New Hardware wizard, some features of the IntelliPoint software may stop working.

CAUSE

This behavior can occur if IntelliPoint files are replaced with earlier versions by the Add New Hardware wizard.

RESOLUTION

To resolve this behavior, reinstall the IntelliPoint software by following these steps:

  1. Insert the IntelliPoint software installation disk into the appropriate floppy disk drive.

  2. Click Start, and then click Run.

  3. In the Open box, type the following line, and then press ENTER 

          <disk drive letter>:\setup

    where <disk drive letter> is the drive letter of the disk drive containing the installation disk.

  4. Follow the Setup instructions to reinstall the IntelliPoint software.

MORE INFORMATION

The following features may stop working after you add a new mouse:

  • Acceleration
  • ClickLock
  • Orientation
  • Pointer Speed
