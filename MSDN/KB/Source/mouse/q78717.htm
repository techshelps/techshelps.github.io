Mouse Cursor Disappears in Word for Windows 2.0

 Last reviewed: September 16, 1996
Article ID: Q78717
The information in this article applies to:
  • Microsoft Mouse Driver for MS-DOS, version 8.1
  • Microsoft Word for Windows, version 2.0

SUMMARY

If you run Microsoft Word for Windows, version 2.0, the mouse cursor may disappear when you choose the Record Macro command.

MORE INFORMATION

If, in Microsoft Windows, version 3.0, black text is displayed on the white background, and you select a black color cursor in the Mouse Control Panel,the Macro Record command makes the mouse cursor invisible when the mouse cursor is moved into the edit region. However, when the Stop Recorder command is chosen, the mouse cursor reappears

To resolve this problem, change the mouse cursor color (for example, to white or transparent), or use the Windows Control Panel to change the color of the text buffer to a darker background.

Note: The mouse cursor disappears when you type a character, but reappears as soon as you move the mouse.
