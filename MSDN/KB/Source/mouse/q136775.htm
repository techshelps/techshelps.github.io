How to Obtain IntelliPoint 1.1 and IntelliType 1.1 Upgrades

 Last reviewed: December 23, 1997
Article ID: Q136775
The information in this article applies to:
  • Microsoft IntelliPoint software versions 1.0, 1.0a, 1.1
  • Microsoft IntelliType software versions 1.0, 1.1

SUMMARY

IntelliPoint 1.1 and IntelliType 1.1 drivers are available to registered owners of the IntelliPoint 1.0 and IntelliType 1.0 software. To obtain these upgrades, please contact the Microsoft Order Desk at (800) 360-7561. Make sure to have your product identification (PID) number ready when you call.

MORE INFORMATION

The 1.1 versions of these drivers contain 16-bit and 32-bit components, making them compatible with all Microsoft operating systems (including Microsoft Windows NT and Microsoft Windows 95).

To determine your PID number, use any one of the following methods:

At the MS-DOS Level

  1. Quit Windows (if it is running).

  2. Put the IntelliPoint or IntelliType driver disk in a floppy disk drive, and then change to that drive.

  3. Type "setup" (without quotation marks), and then press ENTER.

  4. In the first dialog box, choose Exit, and then press ENTER. Your PID number is listed in the next dialog box.

In Windows (Not Windows 95)

  1. Double-click the IntelliPoint Tools And Controls icon, or the IntelliType Manager icon, in the Microsoft Input Devices group.

  2. On the Control menu (to the left of the title bar), click About. Your PID number is listed in the dialog box that appears.

In Windows 95

  1. Click the Start button, point to Programs, point to Microsoft Input Devices, and then click either IntelliPoint Tools And Controls or IntelliType Manager.

  2. Use the right mouse button to click the title bar, and then click About on the menu that appears. Your PID number is listed in the dialog box that appears.
