Mouse Driver 9.0 and Patton & Patton Flow Charting 3.0

 Last reviewed: September 16, 1996
Article ID: Q106440
The information in this article applies to:
  • Microsoft Mouse driver for MS-DOS, version 9.0

SYMPTOMS

Patton & Patton Technical Support has confirmed that when you are using the Microsoft Mouse driver version 9.0 for MS-DOS with Patton & Patton Flow Charting 3.0, the left mouse button does not work in the edit mode of Flow Charting. All other mouse functions should work properly in Flow Charting 3.0.

RESOLUTION

  • Upgrade the MS-DOS Microsoft Mouse driver to version 9.01.

    -or-

  • Reinstall the MS-DOS Microsoft Mouse driver version 8.2 (MOUSE.COM).
