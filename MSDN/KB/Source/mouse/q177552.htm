Invalid Page Fault Using SmartSpeed or Focus Feature

 Last reviewed: December 10, 1997
Article ID: Q177552
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.1

SYMPTOMS

When you are using Microsoft Office 97, you may receive an Invalid Page Fault (IPF) error message in the Comctl32.dll file when you move your pointing device.

CAUSE

This problem can occur if you turn on the SmartSpeed or Focus feature in the IntelliPoint software.

RESOLUTION

To resolve this problem, install the latest version of the IntelliPoint software. To do so, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q177274
   TITLE     : IntelliPoint Software 2.1: Point32 Update Available
To work around this problem, turn off the SmartSpeed and Focus features. To do so, follow these steps:

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Mouse.

  3. On the StepSavers tab, click the Focus and the SmartSpeed check boxes to clear them.

  4. Click OK, and then close Control Panel.
