Err Msg: Software Multiple for Dynamic Resolution Is Off

 Last reviewed: September 12, 1996
Article ID: Q121126
The information in this article applies to:
  • Microsoft IntelliPoint software version 1.0
  • Microsoft Mouse driver versions 8.2, 9.0, 9.01

SYMPTOMS

During the Microsoft MS-DOS boot sequence, the system displays the following error message and then stops responding (hangs): 

   Software Multiple for Dynamic Resolution is off.

CAUSE

This problem occurs when loading the Microsoft Mouse driver version 8.2, 9.0, or 9.01, or the IntelliPoint software version 1.0. The problem is caused by the file LEMOUSE.COM, which is not removed from the AUTOEXEC.BAT file when the Microsoft Mouse software is installed.

RESOLUTION

To correct this problem, remove all references to LEMOUSE.COM in the AUTOEXEC.BAT file.

MORE INFORMATION

LEMOUSE.COM is the MS-DOS-level mouse driver for the Leading Edge mouse.
