IntelliPoint Software 2.1: Point32 Update Available

 Last reviewed: December 23, 1997
Article ID: Q177274
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.1

SUMMARY

New versions of the Point32.exe and Point32.dll files are available to correct problems with the versions of these files included with the Microsoft IntelliPoint software version 2.1.

NOTE: The IntelliPoint software version 2.1 must already be installed on your computer for this update to work correctly.

To obtain the IntelliPoint software version 2.1, please contact the Microsoft Order Desk at (800) 360-7561. Make sure to have your product identification (PID) number ready when you call. You must be a registered owner of the IntelliPoint software.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~ 21p32upd.exe (size: 192736 bytes)

   ARTICLE-ID: Q119591
