Rolling Wheel in Help Window Scrolls Text in Both Directions

 Last reviewed: December 4, 1997
Article ID: Q177531
The information in this article applies to:
  • Microsoft IntelliPoint software version 2.1

SYMPTOMS

When the mouse pointer is located inside a Help window that contains both horizontal and vertical scroll bars, if you roll the wheel on the Microsoft IntelliMouse, the text in the window may scroll both horizontally and vertically.

RESOLUTION

To resolve this behavior, download the most recent versions of the Point32.exe and Point32.dll files. These files are contained in the Microsoft IntelliPoint 2.1 Pointer Update file named 21p32upd.exe.

For information about how to obtain the 21p32upd.exe file, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q177274
   TITLE     : IntelliPoint Software 2.1: Point32 Update Available
