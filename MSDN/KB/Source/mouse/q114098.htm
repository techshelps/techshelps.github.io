Mouse Locate Causes Control Key Problem in File Manager

 Last reviewed: September 16, 1996
Article ID: Q114098
The information in this article applies to:
  • Microsoft Mouse driver for MS-DOS, version 9.01
  • Microsoft Mouse driver for Windows, version 9.01

SYMPTOMS

You cannot select multiple files in File Manager using the CTRL key with the Primary mouse button after setting the Microsoft Mouse 9.01 Locate feature shortcut key to CTRL+Primary.

When you try to use the CTRL+Primary shortcut within File Manager, the mouse jumps to the center of the screen and the files are not selected.

CAUSE

The Microsoft Mouse 9.01 Locate Feature CTRL + Primary shortcut key overrides the File Manager's CTRL+Primary key functionality for selecting multiple files.

RESOLUTION

To work around the problem, change the Mouse 9.01 Locate feature shortcut key to something other than CTRL+Primary.
