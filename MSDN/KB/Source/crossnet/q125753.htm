Some Windows NT Printer Information Not Displayed in WFWG

 Last reviewed: February 11, 1997
Article ID: Q125753
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5
  • Microsoft Windows for Workgroups version 3.11

SUMMARY

Neither the description nor the location of a Windows NT shared printer can be viewed when you attempt to connect to a Windows NT printer from a Windows for Workgroups client.

Because Windows for Workgroups was written before Windows NT, it is unaware of all information that can be obtained about the printer. Is simple does not know to request the printer description or location information.

MORE INFORMATION

You can set the printer description and location in the Windows NT Print Manager Printer Properties dialog box Description and Location fields. The description can be used to provide network users with information about the printer (for example, Accounting Department Printer). The location can be used to specify the printers physical location (for example, Room 101).

Once you have connected to the printer using Windows NT Print Manager, the printer Description and Location appear both in the Connect to Printer and Printer Properties dialog boxes. However, when using Windows for Workgroups version 3.11, the printer Description and printer Location of a shared Windows NT printer do not appear in the Connect Network Printer dialog box.

To work around this Windows for Workgroups limitation, use the maximum characters allowed in the Share Name (12) and Printer Name (32) fields when you create a Windows NT printer to provide additional information.
