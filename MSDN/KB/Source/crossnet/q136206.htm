Macro Recorder Not Included in Windows NT and Windows 95

 Last reviewed: September 9, 1996
Article ID: Q136206
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation versions 3.5, 3.51, and 4.0
  • Microsoft Windows NT Server versions 3.5, 3.51, and 4.0
  • Microsoft Windows 95
  • Microsoft Visual Test for Windows 95 and Windows NT, version 4.0
  • Microsoft Test for Windows, version 3.0a

SUMMARY

Macro Recorder (RECORDER.EXE), which was included with Windows and Windows for Workgroups, is not included with Windows NT or Windows 95. You can obtain the Macro Recorder functionality under Windows NT and Windows 95 using Microsoft Test for Windows.

You may purchase Microsoft Test for Windows from Microsoft.

MORE INFORMATION

Macro Recorder can be used to automate repetitive tasks by recording mouse clicks and keyboard selections. It is a 16-bit program that was not ported to 32-bit for inclusion in Windows NT or Windows 95. Although no utility packaged with Windows NT or the Windows NT Resource Kit performs an equivalent function, you can use Microsoft Test for Windows to achieve this functionality.

Microsoft Test for Windows is a tool developers can use to test Windows- based applications. It simplifies the task of creating high-quality, robust applications by automating most of the test process. While Microsoft Test for Windows is much more than a macro recorder, it can be used to automate processes in Windows NT and Windows 95.

For more information on Microsoft Test, query on the following words in the Microsoft Knowledge Base: 

   mstest
