Err Msg: "Invalid Parameter" or "Incorrect Parameter"

 Last reviewed: September 10, 1997
Article ID: Q149440
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows for Workgroups version 3.11
  • Microsoft Windows NT Server versions 3.5, 3.51, and 4.0

SYMPTOMS

When you log on to a Windows NT domain from a Windows 95 or Windows for Workgroups client, and the domain name listed is invalid, the error "Invalid Parameter" or "Incorrect Parameter" may appear.

CAUSE

An invalid name listed in the Domain field, such as "domain\server," may cause the "Incorrect Parameter" error.

Misspelled domain names may cause the "Invalid Parameter" error, rather than causing an error stating that no domain controller could be found.

RESOLUTION

Make sure the domain name is correct.
