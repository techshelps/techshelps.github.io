Company Name Preset to MICROSOFT When Installing WFW

 Last reviewed: September 9, 1996
Article ID: Q129547
The information in this article applies to:
  • Microsoft Windows NT Server version 3.5
  • Microsoft Windows For Workgroups version 3.11

SYMPTOMS

When you install Windows for Workgroups version 3.11 from the Windows NT Server version 3.5 compact disc (CD), the Company name is pre-set to MICROSOFT.

CAUSE

This problem occurs because the CD is read-only.

WORKAROUND

To work around this problem:

  1. Copy the files in the Windows NT Server CD \CLIENT\WFW\NETSETUP subdirectory to a directory on your local drive.

  2. Install Windows for Workgroups version 3.11 again.
