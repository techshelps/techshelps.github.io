Obtaining Latest Novell Client Windows Drivers and DLLs

 Last reviewed: September 9, 1996
Article ID: Q121462
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

Novell has new Windows and DOS client drivers and dynamic-link library (DLL) files with new naming schemes available. They are WINDRx.EXE and NWDLLx.EXE (where x stands for the latest revision) for Windows files and VLMUPx.EXE and NET33x.EXE for DOS. To obtain these files, contact your local Novell authorized reseller or call Novell Technical Support at (800) 638-9273.
