Total Disk Space on Banyan Vines Server Incorrectly Reported

 Last reviewed: September 9, 1996
Article ID: Q136430
The information in this article applies to:
  • Microsoft MS-DOS version 6.0
  • Microsoft Windows for Workgroups version 3.11
  • Microsoft Windows 95

MS-DOS programs running on a Banyan Vines Client report the wrong total free disk space on Banyan file servers with large disks when free space on those disks is small. The reported amount is often less than is actually available.

More information can be found in Banyan Technotes Bulletin number 288. This technote may be downloaded from the BANYAN forum on CompuServe or contact Banyan Vines technical support at (800) 222-6926.
