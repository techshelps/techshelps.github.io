Microsoft Internet Jump Start CD for Education

 Last reviewed: November 14, 1997
Article ID: Q147612

The information in this article applies to:

  • Microsoft Internet Jump Start CD for Education
  • Microsoft Windows NT Server version 3.51
  • Microsoft Internet Information Server version 1.0
  • Microsoft Internet Explorer for Windows 3.1, Windows 95, Windows NT, and Macintosh
  • Microsoft Word Internet Assistant for Windows version 1.0z and 2.0z
  • Microsoft Word Internet Assistant for the Macintosh (Beta)

SUMMARY

The Microsoft(R) Internet Jump Start CD for Education is a set of Microsoft software and tools for exploring and publishing on the Internet. This CD-ROM is available for no charge to K-12 educational institutions in the U.S. It provides all the software a school needs for Web page creation and publishing, including the network operating system and the Microsoft Internet Information Server to host a Web site that's your very own. Or, create your school's home page and let us host it free at the Global Schoolhouse site on the World Wide Web - all possible thanks to a collaboration between Microsoft, MCI and the Global SchoolNet Foundation.

MORE INFORMATION

The Internet Jump Start CD for Education includes:

  • Microsoft Windows NT (TM) 3.51 Server This is a special version of Windows NT Server limited to 2 File and Print connections at one time.
  • Microsoft Internet Information Server (IIS)
  • Microsoft Word Internet Assistant for Windows(R) (1.0z and 2.0z)
  • Microsoft Word Internet Assistant for the Macintosh(R) (Beta)
  • Microsoft Internet Explorer This includes versions for Windows(R) 3.1, Windows(R) 95, Windows NT, and Macintosh(R).
  • WebWhacker by the ForeFront Group, Inc. This tool downloads Web pages and creates local versions of them.
