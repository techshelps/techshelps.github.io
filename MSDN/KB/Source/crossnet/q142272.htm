Olicom 16/4 TR Drivers Cause Checksum Errors

 Last reviewed: September 9, 1996
Article ID: Q142272
The information in this article applies to:
  • Microsoft TCP/IP-32 for Windows for Workgroups versions 3.11, 3.11a, and 3.11b
  • Microsoft Windows for Workgroups version 3.11

SYMPTOMS

Windows for Workgroups version 3.11 client computers using the Olicom 16/4 Token Ring card experience checksum errors during connections to other devices.

CAUSE

The Olicom 16/4 Token Ring driver (OLITOK.DOS) dated 9/94 causes this problem.

RESOLUTION

Update the driver to (OLITOK16.DOS) dated 5/95. This driver is available from Olicom at 800-265-4266.
