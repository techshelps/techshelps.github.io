ADT95: Updated Data Outline Control Wizard Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q148146
The information in this article applies to:
  • Microsoft Access version 7.0a
  • Microsoft Access Developer's Toolkit version 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

The Wzdboutl.exe file contains an update of the Data Outline Control Wizard, which you can use in developing applications with Microsoft Access for Windows 95 version 7.0a. The original wizard code uses Visual Basic for Applications syntax that is no longer supported in Microsoft Access version 7.0a.

The Wzdboutl.exe file contains the following files: 

   Wzdboutl.mda   The updated Data Outline Wizard for Microsoft
                  Access for Windows 95 version 7.0a

   Readme.txt     A text file with installation instructions for the
                  updated Data Outline Wizard.
