ADT2: ADT Upgrade Rebate Coupon Lists Incorrect Requirements

 Last reviewed: April 7, 1997
Article ID: Q114877
The information in this article applies to:
  • Microsoft Access Developer's Toolkit version 2.0

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

The coupon labeled "$100 U.S./$135 CDN Upgrade rebate by mail!" located on the front of the Microsoft Access Developer's Toolkit version 2.0 box incorrectly states that in order to collect the $100 U.S/$135 CDN rebate, you must provide proof of ownership of both the Microsoft Access Distribution Kit version 1.1 and Microsoft Visual Basic version 3.0. This information is incorrect.

RESOLUTION

To collect the rebate, you need only provide proof of ownership of either the Microsoft Access Distribution Kit version 1.1 OR Microsoft Visual Basic version 3.0.

MORE INFORMATION

The proof of purchase criteria on the reverse of the coupon incorrectly reads: 

   1) Proof of ownership of previous version (one of the following):

      A) The title page of your manual for Microsoft Access
         Distribution Kit version 1.1 and Microsoft Visual Basic
         programming system version 3.0.

      B) A photocopy of your dated sales receipt for Microsoft Access
         Distribution Kit version 1.1 and Microsoft Visual Basic
         programming system version 3.0.
The criteria should read: 

   1) Proof of ownership of previous version (one of the following):

      A) The title page of your manual for Microsoft Access
         Distribution Kit version 1.1 or Microsoft Visual Basic
         programming system version 3.0.

      B) A photocopy of your dated sales receipt for Microsoft Access
         Distribution Kit version 1.1 or Microsoft Visual Basic
         programming system version 3.0.
Note that you need to provide proof of ownership of only one of the products to collect the rebate.
