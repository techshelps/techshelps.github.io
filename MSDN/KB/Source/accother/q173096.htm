ODE97: Summary of Changes in ODE Service Pack 2 (ODESP2)

 Last reviewed: February 23, 1998
Article ID: Q173096
The information in this article applies to:
  • Microsoft Office 97 Developer Edition Tools

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

This article lists the contents and changes in Microsoft Office 97 Developer Edition Service Pack 2 (ODESP2) and explains how to obtain this updated version.

MORE INFORMATION

How to Obtain ODESP2

 ~ ODESP2.EXE (size: 4135056 bytes) 
              is available for download from the Microsoft Software
   Library.
It is also available at the following locations:
  • On the Microsoft Web Site at the following address: 

          http://www.microsoft.com/OfficeDev/ProdInfo/ODESP2.htm
  • In the \ValuPack\ODBC\ folder on the Microsoft Office 97 Service Release 1(SR-1) Enterprise Update CD
  • In the \ValuPack\ODBC\ folder on the Microsoft Office 97 Service Release 1 (SR-1) Patch CD

NOTE: You must have the Microsoft Office 97 Developer Edition Tools (ODE) installed on your computer before you run the Setup program for ODESP2. If the ODE is not installed on your computer you will receive an error message.

NOTE: ODESP2 is language specific even though the Office setup program does not verify the language edition installed on your computer. This means, you should not mix language editions of ODESP2. For example, you should not Install a French edition of ODESP2 on a computer with an English edition of the ODE. If you need a particular language edition of ODESP2, please contact the local Microsoft subsidiary office appropriate for that language.

Fixes in ODESP2

ODESP2 fixes the following problems:

  • The error message, "The ODBC installer DLL (ODBCCCO33.DLL) is not installed on this system," which can occur when you install a run-time application built by the ODE Setup Wizard on a computer that uses the updated ODBC 3.0 Driver Manager.

    For information about ODESP2 and the new ODBC 3.0 Driver Manager, please see the following article in the Microsoft Knowledge Base: 

          ARTICLE-ID: Q172701

          TITLE     : ODE97: Setup Wizard Requires ODESP2 for Updated ODBC
                  Drivers


  • Inability to view an ActiveX control because it is not properly
   registered or the receipt of an Invalid Page Fault (IPF) when
   attempting to view an ActiveX control using the run-time version
   of Microsoft Access 97.

    

ODESP2 also contains the following updates:


  • Fixes included in ODESP1, which was previously released for the
   Traditional Chinese and Korean editions of Microsoft Access. Changes
   in ODESP1 ensure that the Office run-time files are copied to the
   correct folders on the computer. Also included is an updated language-
   specific edition of the MS Graph run-time application (MSGRAPH.EXE) and
   a previously missing language-specific dependency file (*.dep) for an
   ActiveX control.


  • The About box is updated to read "Version 8.0 (Service Pack 2)" so
   you can easily identify whether ODESP2 is installed on your computer.
