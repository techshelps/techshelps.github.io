ADT95: List of Redistributable Files for Microsoft ADT 95

 Last reviewed: May 30, 1997
Article ID: Q140549
The information in this article applies to:
  • Microsoft Access Developer's Toolkit version 7.0

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

This article lists the files that you can distribute with a run-time application that you create by using the Microsoft Access Developer's Toolkit (ADT) for Windows 95 version 7.0. These files are also listed in the Adteula.txt file, which is located in your \ADT folder.
