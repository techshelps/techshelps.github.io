ACC2: Exporting to Btrieve Help Topic Error

 Last reviewed: May 12, 1997
Article ID: Q127975
The information in this article applies to:
  • Microsoft Access version 2.0

SYMPTOMS

Moderate: Requires basic macro, coding, and interoperability skills.

The Microsoft Access online Help topic "Exporting a Table to a Btrieve Table" states the following information: 

   To export to a Btrieve table, you must have the data definition
   files FILE.DDF and FIELD.DDF, which tell Microsoft Access about
   the structure of your tables.
This information is incorrect.

RESOLUTION

If you do not have the FILE.DDF or FIELD.DDF files, Microsoft Access version 2.0 creates them during the export process. You do not need to have these files before you can export data to a Btrieve table.

REFERENCES

For more information about exporting to Btrieve, search for "Btrieve tables," and then "Exporting a Table to a Btrieve Table" using the Microsoft Access Help menu.

For more information about exporting to a Btrieve table, please see the following article in the Microsoft Knowledge Base:
