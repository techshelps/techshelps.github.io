ACC97: Microsoft Access 97 Menu Builder Wizard Available on MSL

 Last reviewed: November 12, 1997
Article ID: Q163235
The information in this article applies to:
  • Microsoft Access 97

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

Earlier versions of Microsoft Access contained a Menu Wizard to assist in creating custom menus. This wizard helps you create custom menus in Microsoft Access 97. 

 ~ Wzmenu80.exe (size: 1215893 bytes) 

   is available for download from the Microsoft Software Library. It
   contains the following files:

   Acmsetup.exe   Setup File
   Acmsetup.hlp   Setup File
   Admin.inf      Setup File
   Expand.exe     Setup File
   MenuBu~1.wri   MenuBuilder Help file in Microsoft Write format
   Msacccah.dll   Setup File
   Msodeusa.dll   Setup File
   Mssetup.dll    Setup File
   Msvcrt20.dll   Setup File
   Oleaut32.dll   Setup File
   Selfreg.dll    Setup File
   Setup.exe      Setup File
   Setup.inf      Setup File
   Setup.ini      Setup File
   Setup.LST      Setup File
   Setup.STF      Setup File
   Setup.tdf      Setup File
   Stdole2.tbl    Setup File
   WzMenu80.mde   MenuBuilder Add-in
   Readme.txt     Setup Readme File
