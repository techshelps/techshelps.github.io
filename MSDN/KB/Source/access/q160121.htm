ACC: Access 95/97 Setup Does Not Create Workgroup Admin Icon

 Last reviewed: July 16, 1997
Article ID: Q160121
The information in this article applies to:
  • Microsoft Access versions 7.0, 97
  • Microsoft Office 97 Professional Edition

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

The Setup program for Microsoft Access 7.0, Microsoft Access 97, and Microsoft Office 97, Professional Edition does not create a shortcut for the Workgroup Administrator program on Microsoft Windows 95 or Microsoft Windows NT 4.0 computers. This feature is by design, and differs from earlier versions of Microsoft Access.

MORE INFORMATION

The Workgroup Administrator program provides a simple user interface for creating or joining a new workgroup information file.

The following steps create a shortcut to the Workgroup Administrator under Programs on the Start Menu:

  1. Click the Start button, point to Settings, and then click Taskbar.

  2. In the Taskbar Properties dialog box, click the Start Menu Programs tab.

  3. Click the Advanced button.

  4. In the Explorer window that opens, double-click the Programs folder to display the contents.

  5. On the File menu, point to New, and then click Shortcut.

  6. In the Create Shortcut dialog box, click Browse.

  7. Locate the Wrkgadm.exe file, and then click Open. In Microsoft Access 97, the default location for the Workgroup Administrator file is the Windows System folder. In Microsoft Access 7.0, the default location is C:\MSOffice\Access.

  8. Click Next to advance to the "Select a Title for the Program" dialog box.

  9. Type the name you want to assign to the shortcut, such as MS Access Workgroup Administrator, and then click Finish.

  10. Close the Explorer window and the Taskbar Properties dialog box.

  11. Click the Start button, and then point to Programs. Note the new shortcut that appears there.

REFERENCES

For more information about the Workgroup Administrator program, search the Help Index for "Workgroup Administrator," or ask the Microsoft Access 97 Office Assistant.
