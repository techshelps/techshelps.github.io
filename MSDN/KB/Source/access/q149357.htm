ACC2: Parameter Typing Utility Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q149357
The information in this article applies to:
  • Microsoft Access version 2.0

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

The Parameter Typing Utility (Params.exe) can help automate the process of explicitly defining the data types of parameters. It loops through the queries in a database and prompts you for the parameter type of any parameters not explicitly defined. You have the option of looping through all queries or only the update and append queries. If you have installed the Microsoft Access version 2.0 Service Pack you, will probably want to run this utility. 

 ~ Params.exe (size: 33747 bytes) 

   is available for download from the Microsoft Software Library. It
   contains the following files:

   Params.mda    The Parameter Typing Utility library
   Params.txt    A text file with installation instructions
