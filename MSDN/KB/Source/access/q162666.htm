ACC97: Setup Program Prompts for Microsoft Office/Access 97 CD

 Last reviewed: April 30, 1997
Article ID: Q162666
The information in this article applies to:
  • Microsoft Access 97

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

When you run the Microsoft Office 97 Professional Edition, or the Microsoft Access 97 Setup program from the Add/Remove Programs icon in Control Panel, you are prompted with one of the following messages.

For Microsoft Office 97 Professional Edition

   Please insert your Microsoft Office 97 Pro CD1.

For Microsoft Access 97

   Please insert your Microsoft Access 97 CD1.
This behavior occurs even if your compact disc is in the CD-ROM drive.

RESOLUTION

Click OK when prompted to place the Microsoft Office 97 Professional Edition or the Microsoft Access 97 compact disc in the CD-ROM drive. The Setup program will start to read from the CD-ROM drive.

MORE INFORMATION

Steps to Reproduce Behavior

  1. Insert the Microsoft Office 97 Professional Edition or Microsoft Access 97 compact disc into your CD-ROM drive and do the following.

    In Microsoft Windows 95 or Microsoft Windows NT 4.0: 

        a. Click the Start button on the Taskbar, point to Settings, and then
       click Control Panel.


        b. In Control Panel, double-click Add/Remove Programs.


        c. In the Add/Remove Programs Properties dialog box, click Microsoft
       Access 97 or Microsoft Office 97, Professional Edition and then
       click Add/Remove.


           Note that you are prompted to insert the compact disc into the
       CD-ROM drive.

    In Microsoft Windows NT 3.51: 

        a. Open the program group for Microsoft Access or Microsoft Office.


        b. Double-click the Setup icon. Note that you are prompted to insert
       the compact disc into the CD-ROM drive.

REFERENCES

For more information about using the Setup program, search the Help Index for "Setup, adding/removing Microsoft Access components," or ask the Microsoft Access 97 Office Assistant.
