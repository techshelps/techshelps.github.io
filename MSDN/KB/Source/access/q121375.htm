ACC: Changing Printer Port Resets Printer Setup Information

 Last reviewed: August 29, 1997
Article ID: Q121375
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1, 2.0, 7.0, 97

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

The printer settings for forms and reports (such as a custom paper size or paper source) are reset if you change the printer's port in the Printers dialog box in Control Panel.

CAUSE

Microsoft Access saves printer port information along with the form or report. Changing the printer port causes Microsoft Access to reset the printer settings for that object.

RESOLUTION

Make sure that all users are using the same port for the default printer, so that there is no need to change ports.

MORE INFORMATION

Additional things that cause this behavior:

Switching from one print server to another.

Copying a database from one machine to another may also cause this if the printers on the different machine have different names, are connected to different ports or are network printers. (This behavior may differ when using Windows NT)

Steps to Reproduce Behavior

Microsoft Access 7.0 and 97:

  1. Start Microsoft Access, open the sample database Northwind.MDB, and then open the Sales By Category report in Design view.

  2. On the File menu, click Page Setup.

  3. Click the Page tab.

  4. In the Paper Size box, select "Legal 8 1/2 x 14 in," and then click OK.

  5. Save, and then close the report.

  6. Click the Start button, point to Settings, and then click printers. Select the printer. On the File Menu, click Properties.

  7. In the Properties dialog box, click the Details tab. Select a different Port. Click the Apply button, and then click OK.

  8. In Microsoft Access, open the Sales By Category report in Design view.

  9. On the File menu, click Page Setup.

  10. Click the Page tab. Note that the paper size is reset to "Letter 8 1/2 x 11 in."

In Microsoft Access versions 1.0, 1.1, and 2.0:

  1. Start Microsoft Access, open the sample database NWIND.MDB, and then open the Sales By Category report in Design view.

  2. On the File menu, click Print Setup.

  3. In the Paper Size box, select "Legal 8 1/2 x 14 in," then click OK.

  4. Save, and then close the report.

  5. Start Control Panel, and then double-click the Printers icon.

  6. In the Printers dialog box, click Connect. Select a different port in the Ports box, and then click OK. click Close.

  7. In Microsoft Access, open the Sales By Category report in Design view.

  8. On the File menu, click Print Setup. Note that the paper size is reset to "Letter 8 1/2 x 11 in."
