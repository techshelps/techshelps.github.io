ACC95: Microsoft Access 95 Sample Reports Available on MSL

 Last reviewed: October 31, 1997
Article ID: Q145777
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

The RptSampl.exe file contains a sample database with over 30 sample reports. These samples demonstrate how to create custom reports to organize and present information the way you want. Topics include dynamically sorting and filtering your data, creating complex totals, faxing reports, printing custom labels, customizing page layout, and many more. 

 ~ RptSampl.exe (size: 514845 bytes) 

   is available for download from the Microsoft Software Library. It
   contains the following files:

   RptSampl7.mdb     A Microsoft Access version 7.0 sample database
   Readme.txt        A text file with installation instructions
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base:
