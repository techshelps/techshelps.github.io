ACC: Data Source Dialog Box Does Not List ODBC Data Sources

 Last reviewed: June 3, 1997
Article ID: Q105512
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1, 2.0

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

When you are attempting to attach or import a table that is accessible through ODBC (such as a SQL Server, Sybase SQL Server, or ORACLE table), the Data Source dialog box does not list the table as a choice.

In order to make working with ODBC data sources as transparent as possible, there is no explicit choice for each source in the Data Source dialog box.

To select an ODBC data source, select SQL Database in the Data Source dialog box. Another dialog box will appear displaying a list of servers with available ODBC data sources.
