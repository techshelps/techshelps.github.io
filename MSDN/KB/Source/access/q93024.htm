ACC: Default Values for AutoLabel and AddColon Properties

 Last reviewed: April 2, 1997
Article ID: Q93024
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1, 2.0, 7.0, 97

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

The following table shows the default values for the AutoLabel and AddColon properties for the different form and report objects: 

   Toolbox object       AutoLabel    AddColon
   ------------------------------------------
   Text box                Yes         Yes
   Option group            Yes         No
   Toggle button           No          No
   Option button           Yes         No
   Check box               Yes         No
   Combo box               Yes         Yes
   List box                Yes         Yes
   Subform/Subreport       Yes         Yes
   Bound object frame      Yes         Yes
   Command button          No          No

REFERENCES

For more information about the AutoLabel and AddColon properties, search the Help Index on "AutoLabel property", or "AutoColon property" or ask the Microsoft Access 97 Office Assistant.
