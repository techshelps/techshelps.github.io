ACC2: New Events and Properties in MS Access 2.0

 Last reviewed: May 21, 1997
Article ID: Q113302
The information in this article applies to:
  • Microsoft Access version 2.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

Microsoft Access version 2.0 has many new events and properties. The new events and properties are listed below in alphabetical order.
