ACC95: Access 95 Replica to Regular DB Wizard Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q162861
The information in this article applies to:
  • Microsoft Access 7.0

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

The Wzrplrgl.mda file is an add-in for Microsoft Access 7.0 that enables you to convert the Design Master database from a replica set to its original, nonreplicated state. 

 ~ Wzrplrgl.exe (size: 459699 bytes) 

   file is available for download from the Microsoft Software Library. It
   contains the following files:

   Wzrplrgl.mda    Replica to Regular Database Wizard
   Readme.txt      Information and installation instructions
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
