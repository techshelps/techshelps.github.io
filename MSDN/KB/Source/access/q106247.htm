ACC1x: Error Using Form Wizard with VGA Mono Driver

 Last reviewed: May 14, 1997
Article ID: Q106247
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1

SYMPTOMS

The Main/Subform FormWizard generates an "Application Error" or general protection (GP) fault when you create a form.

CAUSE

This problem occurs when you are using the Main/Subform FormWizard if you have installed the VGA with Monochrome Display video driver (VGAMONO.DRV) in Microsoft Windows.

RESOLUTION

Use the standard VGA video driver (VGA.DRV).

STATUS

Microsoft has confirmed this to be a problem in Microsoft Access versions 1.0 and 1.1. This problem no longer occurs in Microsoft Access version 2.0.

MORE INFORMATION

Steps to Reproduce Problem

  1. Install the VGA with Monochrome Display video driver in Microsoft Windows.

  2. Create a new form with the Main/Subform FormWizard.

The error will occur when you choose the Next button after selecting the style for the form.
