ACC2: HP LaserJet 4P/4MP 31.V1.27 Updates Custom Paper Size

 Last reviewed: May 28, 1997
Article ID: Q138908
The information in this article applies to:
  • Microsoft Access version 2.0

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

When you create a report using the HP LaserJet 4P/MP 31.V1.27 release, if you use the Custom Paper Size option available for the printer driver, the current values stored in Control Panel will always be used when you print a report.

This limits your ability to create multiple-label reports, each with its own pre-defined Custom Paper Size. You can use only one Custom Paper Size, the one currently set in Control Panel.

CAUSE

If you use the HP LaserJet 4P/4MP 31.V1.27 and set a Custom Paper Size option, the setting is automatically updated to the current Control Panel setting, regardless of the setting you used when you created the report.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Access version 2.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Install the HP LaserJet 4P/MP driver and create a Custom Paper Size of 3 inches wide and 5 inches in length.

  2. Open the sample database NWIND.MDB and open the Customer Mailing Label report in Design view.

  3. On File menu, click Print Setup. Change the printer driver to the HP LaserJet 4P/MP and set the Paper Size to Custom Paper Size.

  4. On the File menu, click Print Preview. Choose All Customers. Note that the Page Size is displayed as 3 x 5.

  5. Save the report.

  6. Open Control Panel and change the Custom Paper Size to 6 inches wide.

  7. In the Database window, preview the Customer Mailing Label report. Note that the new Custom Paper Size is 3 x 6. The report automatically updates to the current settings in Control Panel.

This problem does not occur with Microsoft Access version 1.x, Microsoft Access version 7.0, or with any other version of the printer drivers for the HP 4 series printers.
