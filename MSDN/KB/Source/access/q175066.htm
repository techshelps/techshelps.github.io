ACC97: Microsoft Access 97 Sample Forms Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q175066
The information in this article applies to:
  • Microsoft Access 97

SUMMARY

Moderate: Requires basic macro, coding, and interoperability skills.

The FrmSmp97.exe file contains a sample database with over 30 form examples. These examples demonstrate how to create custom forms for data entry and non-data entry purposes. Topics include dynamically synchronizing two forms, opening multiple instances of a form, creating an AfterUndo event, lightweight forms, hyperlinks, using balloons, command bars and many more. 

 ~ FrmSmp97.exe (size: 1034464 bytes) 

   is available for download from the Microsoft Software Library. It
   contains the following files:

   Frmsmp97.mdb   The forms sample database.
   Readme.txt     This readme file.
   V35perf.doc    The Jet 3.5 Performance White Paper.
