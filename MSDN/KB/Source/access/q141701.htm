ACC95: TransferSpreadsheet Help Topic Error in MS Access 95

 Last reviewed: April 3, 1997
Article ID: Q141701
The information in this article applies to:
  • Microsoft Access version 7.0

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

The TransferSpreadsheet Action Help topic states the following about the its Has Field Names argument: 

   When you export Microsoft Access table or select query data to a
   spreadsheet, the field names are inserted into the first row of the
   spreadsheet if you have selected Yes for this argument.
This information is not entirely correct. The field names are always inserted into the first row of a spreadsheet when you export data using the TransferSpreadsheet action even if the action's Has Field Names argument is set to No.

REFERENCES

For more information about the TransferSpreadsheet action, search for "TransferSpreadsheet," and then "TransferSpreadsheet Action" using the Microsoft Access for Windows 95 Help Index.
