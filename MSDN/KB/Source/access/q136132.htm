ACC95: Components Installed by Each Type of Setup

 Last reviewed: April 3, 1997
Article ID: Q136132
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

This article lists the Microsoft Access for Windows 95 version 7.0 components installed by each of the Setup options: Typical, Compact, and Complete.
