ACC95: Setup Folder Structure for Network Installation

 Last reviewed: April 3, 1997
Article ID: Q136138
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

This article lists the default folder structure used by the Setup program for administrative (Setup /A) and workstation (Run from Network Server) installations.

MORE INFORMATION

Administrative Installation

 Folders (or Directories)                    Subfolders (or Subdirectories)
 --------------------------------------------------------------------------
 \\<server>\<share>\Msoffice                 Access
                                             Bitmaps
                                             ClipArt
                                             Excel
                                             Office
                                             OS
                                             Samples
                                             Template
                                             Workdir

 \\<server>\<share>\MSAPPS                   DAO
                                             MSGraph5
                                             MSInfo
                                             Msn
                                             Os
                                             Proof
                                             VBA

 \\<server>\<share>\Msoffice\Excel           Library

 \\<server>\<share>\Msoffice\Office          Msn

 \\<server>\<share>\Msoffice\OS              Fonts
                                             MSAPPS
                                             System

 \\<server>\<share>\Msoffice\OS\MSAPPS       DAO
                                             MSGraph5
                                             MSInfo
                                             Proof
                                             VBA

 \\<server>\<share>\Msoffice\Template        Database

Workstation Installation

 Folders (or Directories)                    Subfolders (or Subdirectories)
 --------------------------------------------------------------------------
 <drive>\msoffice                            Access
                                             Office

 <drive>\msoffice\Access                     Setup

 <drive>\msoffice\Office                     Msn

REFERENCES

For more information about installing Microsoft Access for Windows 95 on a network, see the topic "Microsoft Access Network Installation" in the Network.txt file included with Microsoft Access for Windows 95.
