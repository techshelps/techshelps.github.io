ACC95: Updated Vba232.dll Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q147529
The information in this article applies to:
  • Microsoft Access version 7.0
  • Standard, Professional, and Enterprise Editions of Microsoft Visual Basic, 32-bit only, version 4.0 Far East, for Windows

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

The Vba232a.exe file contains an update of Vba232.dll, which is included with Microsoft Access for Windows 95 version 7.0 and Microsoft Visual Basic version 4.0 Far East. The update fixes a problem with invalid page faults, which may occur in the following scenarios:

  • As you develop form modules in Microsoft Access, over time, you may receive an "Invalid Page Fault in VBA232.dll" error message.
  • When you print a code module using a Far East version of Microsoft Access or Visual Basic, you may receive an "Invalid Page Fault in VBA232.dll" error message.

For additional information on this problem in VB 4.0 Far East, please see the following article in the Microsoft Knowledge Base:
