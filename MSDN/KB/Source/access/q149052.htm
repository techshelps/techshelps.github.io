ACC95: "TransferText Action" Help Topic Error

 Last reviewed: April 3, 1997
Article ID: Q149052
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

The first sentence of the "Remarks" section of the "TransferText Action" Help topic states the following: 

   Remarks

   You can export the data in Microsoft Access select queries to
   text files, as long as these select queries don't contain
   parameters.
This information is incorrect.

MORE INFORMATION

You can use a parameter query with a TransferText action. However, when the TransferText action runs, you will be prompted to supply information for the parameters.

REFERENCES

For more information about the TransferText Action, search on the phrase "TransferText," and then view "TransferText Action" using the Answer Wizard from the Microsoft Access for Windows 95 Help menu.
