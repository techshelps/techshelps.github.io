ACC1x: Mail Label Wizard Shows Blank Line in Label Size List Box

 Last reviewed: May 28, 1997
Article ID: Q92890
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1

SYMPTOMS

In the Mailing Label ReportWizard, the Label Size list box has a blank line at the bottom. This blank line shows on standard VGA and super VGA monitors but does not show on model 8515/A monitors.

STATUS

This behavior no longer occurs in Microsoft Access version 2.0.
