ACC: Text Import Wizard Allows Only One Field for Primary Key

 Last reviewed: May 30, 1997
Article ID: Q152049
The information in this article applies to:
  • Microsoft Access versions 7.0, 97

SYMPTOMS

Moderate: Requires basic macro, coding, and interoperability skills.

When you import a text file, one of the options in the Import Text Wizard is the "Choose my own Primary Key" option. However, this option will allow you to select only one field for a primary key.

RESOLUTION

If you need a multiple-key index, select the "No Primary Key" option when you use the Import Text Wizard. Once the import is completed, open the table in Design view, select the fields you want for your primary key, and then click the Key button on the toolbar.

REFERENCES

For more information about importing text files, search the Help Index

for "Importing data," or ask the Microsoft Access 97 Office Assistant.
