PRB: Changing Show Ruler Option Changes Existing Forms

 Last reviewed: May 28, 1997
Article ID: Q109313
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1, 2.0

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

Changing the Show Ruler option in the Form & Report Design section of the Options dialog box overrides this setting in existing forms and reports.

CAUSE

This behavior occurs when the Show Ruler option is changed for a new form or report. For example, if you set the Show Ruler option to No for a new form or report, that setting will override a Show Ruler option of Yes that was set in any already existing form or report.

Note that this behavior also occurs with the Show Grid and Objects Snap To Grid options.

NOTE: This option was removed in Microsoft Access for Windows 95. Most Options have been moved to the Tools menu. However, Show Ruler, Show Grid and Objects Snap to Grid are available only on the View menu. If you change them on the View menu, the changes will also affect the next form or report you open.

MORE INFORMATION

Steps to Reproduce Behavior

  1. Create a new, blank form.

  2. If the ruler is not currently visible along the top of the form in Design view, choose Ruler from the View menu.

  3. Save and then close the form.

  4. From the View menu, choose Options. Select Form & Report Design, and then set the Show Ruler option to No.

  5. Open the form that you closed in step 3. Note that the ruler is not visible.
