ACC1x: Multiuser Report Error with Postscript Printer

 Last reviewed: May 14, 1997
Article ID: Q113920
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1

SYMPTOMS

When you open a report, the following error message occurs: 

   An error occurred while attempting to retrieve printer information
   for the <printer name and port>.
Note that the name of the actual postscript printer that you are using is displayed in the error message.

CAUSE

This error can occur when the report is opened from a shared database in a multiuser environment. For example, if a shared database is located on computer A, and users on computer A and computer B both have the shared database open, and both users try to open the same report, one of them may receive the error message stated earlier. This problem only occurs when a postscript printer driver is being used.

RESOLUTION

You may need to use a different, non-Postscript, printer driver.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Access versions 1.0 and 1.1. This problem no longer occurs in Microsoft Access version 2.0.
