ACC2: "Language Reference" of MS Access Version 2.0 Only Online

 Last reviewed: April 2, 1997
Article ID: Q112126
The information in this article applies to:
  • Microsoft Access version 2.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

Microsoft Access version 2.0 includes the Microsoft Access "Language Reference" manual in the Microsoft Access Help system. To view it, choose Contents from the Help menu, and then click the Language And Technical Reference icon.

If you want a printed version of the Microsoft Access "Language Reference" manual, your retail package contains a coupon you can use to order the manual directly from Microsoft for $9.95 plus $5.00 shipping (plus local sales tax, if applicable). Send the coupon and payment to: 

   Attn: Microsoft Access 2.0 Language Reference
   Microsoft Access for Windows Supplemental Offers
   P.O. Box 3022
   Bothell, WA 98041
The printed Microsoft Access "Language Reference" manual is also included in the Microsoft Access Developer's Toolkit.
