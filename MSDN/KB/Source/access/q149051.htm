ACC95: "Import or Link Spreadsheet" Help Topic Error

 Last reviewed: April 3, 1997
Article ID: Q149051
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

The "Import or Link Spreadsheet" Help topic states the following:

You can import or link data from a Microsoft Excel (versions 2.x, 3.0, 4.0, 5.0, and 7.0) spreadsheet.

This information is incorrect.

MORE INFORMATION

Microsoft Access for Windows 95 cannot import a Microsoft Excel 2.x spreadsheet. This is by design.

REFERENCES

For more information about importing or linking to spreadsheets, search on the phrase "import spreadsheet," and then view "How do I import or link a spreadsheet" using the Answer Wizard from the Microsoft Access for Windows 95 Help menu.
