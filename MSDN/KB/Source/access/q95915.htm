ACC1x: "You Must Run SETUP /A Before Running SETUP /N" Error Msg

 Last reviewed: May 7, 1997
Article ID: Q95915
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1

SYMPTOMS

When you attempt to set up a workstation for Microsoft Access using the SETUP /N command, you receive the following error message: 

   You must run SETUP /A before running SETUP /N

CAUSE

When the network server was set up, the command SETUP /A was issued from a temporary installation directory on the hard disk, rather than from the installation floppy disks.

RESOLUTION

Run SETUP /A from the installation disks in the floppy disk drive to set up the network server. Then you can run SETUP /N from the workstations.

STATUS

This behavior no longer occurs in Microsoft Access version 2.0.
