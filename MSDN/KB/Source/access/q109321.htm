ACC: Options Dialog Box Items Are Not Visible

 Last reviewed: May 28, 1997
Article ID: Q109321
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1, 2.0, 7.0, 97

SYMPTOMS

Novice: Requires knowledge of the user interface on single-user computers.

When you click Options on the Tools menu (or the View menu in versions 1.x and 2.0), some of the option items are not visible.

CAUSE

In Control Panel under the Appearance Tab in Windows 95 (or under the Colors icon in Windows version 3.x and Windows NT), the Window Text color and font (or the Window Text color in Windows version 3.x and Windows NT) has been set to the same color as the 3D Objects (or Button Face in Windows version 3.x and Windows NT) in the Message Box, which is the color Microsoft Access uses for the background color in the Options dialog box.

RESOLUTION

To work around this behavior, follow these steps:

  1. Start Control Panel.

  2. Double-click the Display icon (or Color icon in Windows version 3.x and Windows NT).

  3. Click the Appearance tab (or Color Palette in Windows version 3.x and Windows NT).

  4. Select 3D Objects (or Button Face in Windows version 3.x and Windows NT) in the Item box (or Screen Element box in Windows version 3.x and Windows NT).

  5. Choose a different color for the Item Color (or Button Face in Windows version 3.x and Windows NT).

  6. Click OK.

MORE INFORMATION

Steps to Reproduce Behavior

  1. Start Control Panel.

  2. Double-click the Display icon (or Color icon in Windows version 3.x Windows NT).

  3. Click the Appearance tab (or Color Palette in Windows version 3.x and Windows NT).

  4. Select Window (or Window Text in Windows version 3.x and Windows NT) in the Item box (or Screen Elements box in Windows version 3.x and Windows NT).

  5. Choose a color (for example, the light gray color) for the Item and Font color (or for the Window Text color in Windows version 3.x and Windows NT).

  6. Select 3D Objects (or Button Face in Windows version 3.x and Windows NT) in the Item box (or Screen Elements box in Windows version 3.x and Windows NT).

  7. Choose the same color you chose in step 5.

  8. Click OK. Close Control Panel.

  9. Start Microsoft Access and open any database. Click Options on the Tools menu (or the View menu in versions 1.x and 2.0).
