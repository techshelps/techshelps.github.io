ACC: How to Set the Opening Position of a Form on the Screen

 Last reviewed: April 2, 1997
Article ID: Q159683
The information in this article applies to:
  • Microsoft Access versions 2.0, 7.0, 97

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

This article describes how to set or change the location on the screen where a form opens.

MORE INFORMATION

If you open a form, drag it to a new location on the screen, and then close it, when you reopen the form it appears in the original location. If you want to change the opening position of a form, follow these steps:

  1. Open the form in Design view.

  2. On the View menu, click Properties (if that choice is not already selected).

  3. Set the AutoCenter property of the form to No.

  4. Save the form and close it.

  5. Open the form in Form view.

  6. Move the form to any location on the screen.

  7. On the File menu, click Save (or Save Form in version 2.0). When you close the form and reopen it, it will open in the same position on the screen.

  8. When you want the form to open in a new location on the screen, repeat steps 5-7.

REFERENCES

For more information about forms and their design, search the Help Index for "forms."
