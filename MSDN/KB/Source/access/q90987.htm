ACC1x: Common Dialog Boxes Allow Only Lowercase Typing

 Last reviewed: April 2, 1997
Article ID: Q90987
The information in this article applies to:
  • Microsoft Access versions 1.0, 1.1

You can type only lowercase letters in the File Name box in the New Database, Open Database, Compact Database, Encrypt/Decrypt Database, and Repair Database dialog boxes.

This behavior occurs because the File Name box in the common dialog box template has the lowercase flag set so as to allow more characters to be displayed in a given width. The behavior has no affect on the outcome of any task or process, because Microsoft Access is not case sensitive in filenames.
