ACC95: Function Template Wizard Available on MSL

 Last reviewed: October 21, 1997
Article ID: Q150348
The information in this article applies to: 

  - Microsoft Access version 7.0

SUMMARY

Advanced: Requires expert coding, interoperability, and multiuser skills.

The Funcwiz.exe file contains a Microsoft Access Add-in for automating the creation of code procedures. Common features such as DAO code and error handling are added automatically by the wizard based on options chosen by the user. 

 ~ Funcwiz.exe (size: 687988 bytes) 

   file is available for download from the Microsoft Software Library. It
   contains the following files:

   Acmsetup.exe    Setup file
   Acmsetup.hlp    Setup file
   Admin.inf       Setup file
   Expand.exe      Setup file
   Functi~1.mda    Function Template Wizard.mda
   Functi~1.doc    Function Template Wizard.doc
   Msacccah.dll    Setup file
   Msadtusa.dll    Setup file
   Mssetup.dll     Setup file
   Setup.exe       Setup file
   Setup.inf       Setup file
   Setup.ini       Setup file
   Setup.lst       Setup file
   Setup.stf       Setup file
   Setup.tdf       Setup file
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base:
