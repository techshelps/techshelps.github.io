ACC1: Microsoft Access 1.0 (1/14/93) Disks 1-3 Contents (1.44M)

 Last reviewed: April 2, 1997
Article ID: Q100155
The information in this article applies to:
  • Microsoft Access version 1.0

SUMMARY

The following lists the directory contents of Disks 1-3 shipped with Microsoft Access version 1.0 with a release date of 1/14/93 (3.5 inch, 1.44 MB).

Note that the WBTRCALL.DLL file was removed from this release of Microsoft Access. Due to this removal, the SETUP.EXE program was changed. The functionality of Microsoft Access was not changed in any way for this release.
