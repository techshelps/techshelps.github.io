ACC2: Tech*Ed 94 Sessions on Queries in MS Access 2.0

 Last reviewed: October 21, 1997

The information in this article applies to:

SUMMARY

Novice: Requires knowledge of the user interface on single-user computers.

During Tech*Ed 94, Microsoft held technical sessions on Microsoft Access version 2.0. These sessions covered topics from Introduction To Relational Databases to Creating Client-Server Applications. The information provided in these sessions has been compiled and is available in the Microsoft Software Library (MSL).
