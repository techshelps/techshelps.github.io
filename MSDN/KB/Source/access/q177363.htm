ACC: Error Message: Winhlp32 Caused an Invalid Page Fault...

 Last reviewed: March 11, 1998
Article ID: Q177363
The information in this article applies to:
  • Microsoft Access versions 7.0, 97
  • Microsoft Windows 95

SYMPTOMS

When you try to open a topic in Microsoft Access 97 Help, you may receive the following error message: 

   WINHLP32 caused an invalid page fault in module <unknown> at <address>.
When you click Close, Microsoft Access 97 Help quits.

CAUSE

This error message can occur if Windows 95 multilanguage support is enabled.

RESOLUTION

To work around this issue, disable multilanguage support before you use Microsoft Access 97 Help. To do so, follow these steps:

  1. Press CTRL+ALT+DELETE to open the Close Program dialog box.

  2. Click Internat, and then click End Task.

Multilanguage support is re-enabled the next time you start your computer.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

For information about how to enable multilanguage support in Windows 95, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q141306
   TITLE     : How to Enable Support for Multiple Languages in Windows 95
