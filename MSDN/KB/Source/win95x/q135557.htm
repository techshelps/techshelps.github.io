Desktop Themes Icons Are Different

 Last reviewed: August 30, 1996
Article ID: Q135557
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Plus! For Windows 95

SUMMARY

The minimized icon for Desktop Themes, as well as the icon on the Desktop Themes title bar, is different from the full-size icon.

MORE INFORMATION

The smaller (16 x 16) icon does not include the same detail as the full- size (32 x 32) icon for Desktop Themes. The full-size icon includes a sound horn element that is not included in the smaller icon.

The sound horn is a graphic element only, and does not indicate the availability of sound in Desktop Themes.
