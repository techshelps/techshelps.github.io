Details Tab Incorrectly Selected After Invalid Separator Page

 Last reviewed: August 30, 1996
Article ID: Q132615
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you click the General tab in the Properties dialog box for the printer so that you can configure the printer to use a separator page, you specify an invalid file name in the Separator page box, and receive the following error message: 

   You must select a valid separator page
After you click the OK button to clear the error message, the Details tab is automatically selected as the current tab.

RESOLUTION

Manually select the General tab, and then specify a valid file name in the Separator page box.

STATUS

Microsoft has confirmed this to be a problem in Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
