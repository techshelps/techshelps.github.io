Windows 95 Does Not Boot After Running MicroHelp UnInstaller

 Last reviewed: July 30, 1996
Article ID: Q117431
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

After you run MicroHelp UnInstaller version 2.0 under Windows 95, your system does not boot.

CAUSE

UnInstaller searches your system and deletes files associated with a specified application that you want to remove. However, if you run UnInstaller under Windows 95, it leaves the system unbootable.

RESOLUTION

To correct this problem, reinstall Windows 95.
