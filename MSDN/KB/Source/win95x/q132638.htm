Help Window in Carbon Copy Setup May Not Be Visible

 Last reviewed: August 21, 1995
Article ID: Q132638
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you click Help from the Setup program for Carbon Copy, the Help window may not be visible.

CAUSE

Carbon Copy Help is hidden behind the Setup program for Carbon Copy.

RESOLUTION

Use the taskbar or press ALT+TAB to switch to the Windows Help window.

MORE INFORMATION

In versions of Windows prior to Windows 95, after Windows Help starts, the Setup program for Carbon Copy brings the Help window to the foreground. Because of the ability of Windows 95 to run multiple programs at the same time, Windows Help may not be fully started when the Setup program tries to bring it to the foreground.
