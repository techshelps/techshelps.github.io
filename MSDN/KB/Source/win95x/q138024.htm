Problems Printing to NewGen Turbo PS Printers

 Last reviewed: June 19, 1997
Article ID: Q138024
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you print to a NewGen PS Turbo 480 printer using the Windows 95 Apple LaserWriter II NTX printer driver, you receive a blank test page, followed by PostScript error codes.

You may experience similar problems using NewGen PS Turbo 300, 380, and 400 models.

CAUSE

The Windows 95 PostScript printer drivers are not compatible with these older printers.

RESOLUTION

To use these printers with Windows 95, install a printer driver included with the printer by the manufacturer. For information about installing the drivers included with your printer, please consult the printer's documentation.
