How to Install Microsoft Backup

 Last reviewed: August 21, 1997
Article ID: Q152561
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

Microsoft Backup is a backup tool included with Windows 95. This tool is not installed by default when you install Windows 95. This article describes how to install Backup after Windows 95 is installed.

MORE INFORMATION

To install Backup, follow these steps.

NOTE: This procedure requires access to your original Windows 95 disks or CD-ROM or to the server-based-setup (SBS) shared installation of Windows 95.

  1. In Control Panel, double-click Add/Remove Programs.

  2. On the Windows Setup tab, click Disk Tools, and then click Details.

  3. Click the Backup check box to select it, and then click OK.

  4. Click OK.
