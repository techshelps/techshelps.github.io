Availability of Windows 95 OEM Service Release 2

 Last reviewed: October 3, 1997
Article ID: Q169625
The information in this article applies to:
  • Microsoft Windows 95 OEM Service Release version 2

SUMMARY

This article contains information about the availability of Windows 95 OEM Service Release 2 [OSR2] (Windows 95 version 4.00.950B).

MORE INFORMATION

OSR2 is available with the purchase of the following types of new hardware:

  • A new computer that contains the following components: 

        - Motherboard and CPU
    - Hard disk
    - Power supply
    - Case
  • A new hard disk.
  • A new motherboard and CPU.

For information about support options for OSR2, see the following article in the Microsoft Knowledge Base:
