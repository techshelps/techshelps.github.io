Leading Zero Not Shown While Changing Time

 Last reviewed: August 8, 1996
Article ID: Q134239
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you are changing the time using the Date/Time tool in Control Panel, the leading zero is not shown in the minutes field if the time is less than 10 minutes past the hour.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The leading zero is missing only when you are changing the minutes field. The zero is shown at all other times. Note that this problem does not affect the functionality of the clock in any way.
