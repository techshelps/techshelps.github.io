 Last reviewed: October 11, 1995
Article ID: Q137992
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

The following information is a copy of the information in the Readme.txt file in the Drivers folder on the Windows 95 CD-ROM. Setup does not copy this file to your hard disk.

MORE INFORMATION

Double-click on HCL95.HLP or HCL95.RTF to see a listing of PC peripherals that are compatible with the Microsoft(R) Windows(R) 95 operating system.

This directory contains drivers for the following devices, which are part of the HCL. Follow instructions outlined in the HCL95.HLP or HCL95.RTF to load drivers for these devices.
