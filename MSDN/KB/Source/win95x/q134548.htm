Pressing CTRL+ESC During Logon Opens the Tasks Dialog Box

 Last reviewed: September 30, 1997
Article ID: Q134548
95 WINDOWS kbui

The information in this article applies to:

  • Microsoft Windows 95

SUMMARY

Pressing CTRL+ESC when the Logon dialog box is displayed opens the Tasks dialog box, from which you can run programs.

This can be a problem if the System Administrator has set the "Logon must be validated" policy, since the Tasks dialog box allows users to run programs before logging on.

MORE INFORMATION

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
