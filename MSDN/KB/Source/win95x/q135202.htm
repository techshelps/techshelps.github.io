Backup Cannot Access Tape Drive in Safe Mode

 Last reviewed: August 22, 1995
Article ID: Q135202
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

If you start Windows 95 in Safe mode and then start Backup, your computer may stop responding (hang).

CAUSE

The protected-mode driver used by Backup to access the floppy disk drive controller cannot be loaded in Safe mode.

MORE INFORMATION

Backup does not function in Safe mode if the protected-mode driver is required to access the tape drive.
