Extra Serial Port Displayed in Device Manager

 Last reviewed: June 4, 1997
Article ID: Q158845
95 WINDOWS kbhw

The information in this article applies to:

  • Microsoft Windows 95 OEM Service Release versions 2, 2.1

SYMPTOMS

When you are using Windows 95 OEM Service Release 2 or 2.1, you may see an extra communications port in Device Manager. There is an exclamation point in a yellow circle next to the port. If you remove the port, it is added again the next time you restart your computer.

CAUSE

The computer's Plug and Play BIOS may be incorrectly reporting that the COM ports are not using resources though they were detected during Setup.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
