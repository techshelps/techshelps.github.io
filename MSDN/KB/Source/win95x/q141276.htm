How to View System and Hidden Files in Windows

 Last reviewed: December 8, 1997
Article ID: Q141276
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows NT Workstation version 4.0

SUMMARY

This article describes how to view hidden and system files in Windows Explorer, My Computer, and the Find tool. By default, Windows does not display these files.

MORE INFORMATION

To view hidden and system files, use the appropriate method.

Computers Without the Internet Explorer 4.0 Windows Desktop Update:

  1. Double-click My Computer.

  2. On the View menu, click Options.

  3. On the View tab, click Show All Files, and then click OK.

Computers With the Internet Explorer 4.0 Windows Desktop Update:

  1. Double-click My Computer.

  2. On the View menu, click Folder Options.

  3. Click the View tab.

  4. In the Advanced Settings box, click Show All Files, and then click OK.

Enabling the Show All Files option causes the following file types to be displayed: 

   .dll - Program extension
   .sys - System file
   .vxd - Virtual device driver
   .386 - Virtual device driver
   .drv - Device driver
Note that any hidden files are also displayed.
