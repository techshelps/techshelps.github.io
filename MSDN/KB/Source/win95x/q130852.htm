Text Assist Requires Updated File to Function in Windows 95

 Last reviewed: August 22, 1995
Article ID: Q130852
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

None of the programs in Text Assist, an application suite published by Creative Labs, functions in Windows 95.

CAUSE

The Text Assist application suite requires an updated file (CTSW.EXE) to work correctly in Windows 95.

RESOLUTION

To obtain the updated CTSW.EXE file, please contact Creative Labs.
