Windows 95 Software Compatibility List

 Last reviewed: December 3, 1997
Article ID: Q136660
The information in this article applies to:
  • Microsoft Windows 95

SUMMARY

The Windows 95 Software Compatibility List (SCL)--also known as the Application Compatibility List, or ACL--contains a list of applications that have been tested with the Microsoft Windows 95 operating system.

MORE INFORMATION

You can search for products that carry the Made for Windows 95 logo on the Microsoft Web site at: 

   http://www.microsoft.com/windows/thirdparty/winlogo/default.htm
The SCL is also included with the Windows 95 Support Assistant. For information about obtaining the Windows 95 Support Assistant, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q134363
   TITLE     : WW1169: Windows 95 Support Assistant Application Note
