No Start Menu Icons After Installing Borland C++ 4.02

 Last reviewed: August 30, 1996
Article ID: Q131001
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

After you install Borland C++ version 4.02, there are no Start menu icons for the application components.

CAUSE

This is a problem with the Borland C++ 4.02 Setup program.

RESOLUTION

To create Start menu icons, follow these steps:

  1. Click the Start button, point to Settings, then click Taskbar.

  2. Click the Start Menu Programs tab.

  3. Add the programs you want.

STATUS

Borland is aware of this problem and plans to revise future versions of the program.
