Contents of Windows 95 and Microsoft Plus! Startup Disks

 Last reviewed: March 17, 1998
Article ID: Q142943
95 WINDOWS kbtool kbref

The information in this article applies to:

  • Microsoft Windows 95
  • Microsoft Plus! for Windows 95

SUMMARY

This article lists the directory contents of the Windows 95 Startup disk, and includes information about files on the Startup disk that are updated by Microsoft Plus! when you install DriveSpace 3.

The files on the Startup disk should be dated either 7/11/95 or the date on which the Startup disk was created.

Contents of the Microsoft Plus! Startup Disk

If you install DriveSpace 3, two files on the Startup disk must be updated so that you can access DriveSpace 3 compressed volumes when you start your computer using the Startup disk. When you compress a volume with DriveSpace 3, you are prompted to update the Startup disk. If you reinstall Microsoft Plus! after you compress a volume with DriveSpace 3, or if you do not compress a volume with DriveSpace 3, you are not prompted to update the Startup disk.

An updated Startup disk contains the same files as listed above, with the exception of the following two files: 

   Scandisk.exe    137,836    07-14-95 4:40a
   Drvspace.bin     64,135    07-14-95 4:40a
