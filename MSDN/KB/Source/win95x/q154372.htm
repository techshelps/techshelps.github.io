Long File Names Lost with Ncopy.exe

 Last reviewed: August 2, 1996
Article ID: Q154372
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you copy files using Novell's Ncopy.exe, some or all long file names (LFNs) may be lost.

CAUSE

You are using an older copy of Ncopy.exe.

RESOLUTION

To work around this behavior, use either of the following methods:

  • Use Windows Explorer to copy files between NetWare servers.
  • Obtain an updated copy of Ncopy.exe (version 4.13, which is included with Novell NetWare 4.1).
