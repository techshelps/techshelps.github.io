System Agent Does Not Shut Down Startup Tasks

 Last reviewed: August 31, 1995
Article ID: Q134769
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Plus! For Windows 95

SYMPTOMS

Programs started by System Agent at startup that should stop when you use the computer do not stop.

RESOLUTION

Stop the program manually if it interferes with your use of the computer. If you start your computer at the same time every day, try setting System Agent to start the program at a specific time. This allows System Agent to stop the program when you use the computer.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Plus! for Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
