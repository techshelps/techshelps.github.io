Online Registration Wizard Cannot Locate SignUp Wizard

 Last reviewed: February 27, 1998
Article ID: Q137985
95 WINDOWS kbsetup kberrmsg

The information in this article applies to:

  • Microsoft Money for Windows 95, version 4.0
  • Microsoft Works for Windows 95, version 4.0
  • Microsoft Office Professional for Windows 95, version 7.0
  • Microsoft Publisher for Windows 95, version 3.0.
  • Microsoft Windows 95

SYMPTOMS

When you try to register a Windows 95-based program using the online Registration Wizard, you may receive the following error message: 

   Registration Wizard

   The Registration Wizard was unable to locate the SignUp Wizard which
   is used to transmit your registration information to Microsoft.
   Please make sure SignUp is properly installed then try the
   Registration Wizard again.

RESOLUTION

In order for the online Registration Wizard to work correctly, the MSN (The Microsoft Network) software component must be installed. To install the MSN software, follow these steps:

  1. In Control Panel, double-click the Add/Remove Programs icon.

  2. Click the Windows Setup tab.

  3. Click the "The Microsoft Network" check box to select it, and then click OK.
