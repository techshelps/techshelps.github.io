Reinstalling OSR2 Removes Language Indicator on Taskbar

 Last reviewed: November 8, 1996
Article ID: Q159020
The information in this article applies to:
  • Microsoft Windows 95 OEM Service Release versions 2, 2.1

SYMPTOMS

If you have two or more languages installed for the keyboard, the language icon may no longer appear on the taskbar after you reinstall Windows 95.

RESOLUTION

To reactivate the language icon, follow these steps:

  1. In Control Panel, double click Keyboard.

  2. Click the Enable Indicator On Taskbar check box to select it.

  3. Click OK.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Even though the language indicator does not appear on the taskbar, you can still switch languages by pressing ALT+SHIFT.
