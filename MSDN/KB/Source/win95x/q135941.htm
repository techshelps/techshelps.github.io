Write Protection Error Message with DriveSpace 3

 Last reviewed: August 31, 1995
Article ID: Q135941
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Plus! for Windows 95

SYMPTOMS

On a computer with a write-protected, mounted, compressed removable drive, you may receive a write error message on a blue screen from the compression driver.

CAUSE

This error occurs because the compression driver does not have write access to the host drive.

RESOLUTION

Remove the write protection from the removable drive and try the operation again.
