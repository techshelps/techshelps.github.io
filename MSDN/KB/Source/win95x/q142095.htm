MS Access 2.0 Font Boxes Too Small After Installing MS Plus!

 Last reviewed: January 5, 1996
Article ID: Q142095
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Plus! for Windows 95
  • Microsoft Access version 2.0

SYMPTOMS

After you install Microsoft Plus!, the Font Name and Font Size boxes in Microsoft Access version 2.0 may be too small to display the entire name and size of a font when you are viewing a form in Design view. Instead, you may see only the top half of each character in the Font Name and Font Size boxes. However, the functionality of the program is not affected in any way.

CAUSE

The Font And Window Sizes option in Desktop Themes is enabled, and you are using one of the following themes:

  • Dangerous Creatures
  • More Windows
  • Nature
  • Science
  • Sports
  • The Golden Era
  • Windows 95
  • Windows Default

These themes have smaller window sizes, and Microsoft Access 2.0 uses this setting to draw the Font Name and Font Size boxes.

RESOLUTION

In Desktop Themes, click the theme you want to use in the Theme box, and then click Apply. Click the Font And Window Sizes check box to clear it, and then click OK.
