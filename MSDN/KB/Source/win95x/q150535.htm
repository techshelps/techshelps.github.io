Cannot Create Backup Copies of Windows 95 Disks

 Last reviewed: August 12, 1997
Article ID: Q150535
95 WINDOWS kbother

The information in this article applies to:

  • Microsoft Windows 95

SUMMARY

You cannot make backup copies of your original Windows 95 disks.

MORE INFORMATION

Windows 95 disks are Distribution Media Format (DMF) disks and are not compatible with the MS-DOS COPY and DISKCOPY commands. You cannot create backup copies of the Windows 95 disks.
