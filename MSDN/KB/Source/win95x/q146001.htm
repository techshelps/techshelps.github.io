AMD (NexGen) Nx586 Processor Identified as 80386

 Last reviewed: May 28, 1996
Article ID: Q146001
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Plus! for Windows 95

SYMPTOMS

Windows 95 and other programs may identify the AMD (formerly NexGen) Nx586 processor as an 80386 processor. You may experience the following symptoms with this processor:

  • You may receive the following error message when you run Microsoft Plus! Setup: 

          MS Plus! requires a 486 processor or greater
  • The 3D Pinball game included with Microsoft Plus! may not run.
  • The Microsoft Hover! game included on the CD-ROM version of Windows 95 may not run.
  • Microsoft Fury3 for Windows version 1.0 may not run.
  • You may not be able to install Microsoft Encarta World Atlas for Windows, 1996 edition. For additional information about this problem, please see the following article in the Microsoft Knowledge Base: 

          ARTICLE-ID: Q142217

          TITLE     : Encarta World Atlas 1996: Setup Fails with NexGen CPU


CAUSE

 


Programs developed before the NexGen Nx586 processor may mistakenly
identify the processor as an 80386 processor.



RESOLUTION

 


To cause programs to correctly identify the Nx586 processor, obtain the
Windows CPU Identifier tool from NexGen. This tool is included in the
NexGen Tool Kits and Update Kits. These kits are available for downloading
from NexGen's BBS and their Internet site. For additional information,
please contact NexGen.


Also, if your computer's BIOS version is earlier than the following
versions, contact NexGen about obtaining a Flash BIOS upgrade:



   For PCI: p06w034n
   For VL : v06d033w
