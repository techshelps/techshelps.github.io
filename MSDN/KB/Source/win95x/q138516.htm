Problems with the Samp1.adm File

 Last reviewed: May 31, 1996
Article ID: Q138516
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

The following problems exist with the Samp1.adm file:

  • The Samp1.adm file referred to in the Policies.txt file on the Windows 95 CD-ROM is not available on the CD-ROM as is mentioned, and is currently not available from any other source.

    The Policies.txt file is located on the Windows 95 CD-ROM in the Admin\Reskit\Samples\Policies folder.

  • The version of the Samp1.adm file included on the Windows 95 Resource Kit disks is damaged. Attempting to use this file with System Policy Editor may cause Windows 95 to stop responding (hang).

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
