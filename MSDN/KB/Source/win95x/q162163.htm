Error Message During Resume on IBM Aptiva Computer

 Last reviewed: January 15, 1997
Article ID: Q162163
The information in this article applies to:
  • Microsoft Windows 95 OEM Service Release versions 2, 2.1

SYMPTOMS

When you press the hardware Suspend button on an IBM Aptiva computer with a Mwave driver, you may receive the following error message on a blue screen when you resume the computer: 

   0028:C002CFC8 in VXD VDMAD(01) + 00000514

RESOLUTION

To avoid this problem, use the Suspend command on the Start menu to suspend the computer instead of using the hardware Suspend button.
