Incorrect IBM Streamer Adapter Files on Service Pack 1 CD-ROM

 Last reviewed: September 19, 1997
Article ID: Q148490
95 WINDOWS kbnetwork

The information in this article applies to:

  • Microsoft Windows 95

SUMMARY

The Windows 95 Service Pack 1 CD-ROM, which contains the Windows Driver Library (WDL), contains incorrect files for IBM Streamer network adapters.

MORE INFORMATION

Incorrect files are included for the following network adapters:

  • IBM Auto LANStreamer PCI
  • IBM Auto LANStreamer MC 32
  • IBM LANStreamer MC 32
  • IBM EtherStreamer MC 32

The correct files are available on the Windows 95 CD-ROM in the Drivers\ Network\IBM folder. The correct files to use are:
  • Netstrm.inf (23.2K)
  • Streamer.vxd (68.2K)

If you do not have the CD-ROM version of Windows 95, the files are available from online services. Please see the following article in the Microsoft Knowledge Base for information about obtaining the STREAMER.EXE file:
