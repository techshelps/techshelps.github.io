Difficulty Installing System Shock

 Last reviewed: August 21, 1995
Article ID: Q132803
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you use the Add/Remove Programs Wizard in Control Panel to install System Shock (a program manufactured by Looking Glass and distributed by Origin), the program does not install properly.

CAUSE

You must run the installation program for System Shock from an MS-DOS prompt.

RESOLUTION

To install System Shock, start an MS-DOS prompt. At the command prompt, type "D:" (without the quotation marks), type "install" (in lowercase letters and without the quotation marks), and then press ENTER.
