No Error Message Received When Print File Is Not Created

 Last reviewed: January 25, 1996
Article ID: Q128823
The information in this article applies to:
  • Microsoft Windows 95 Preview Program

SYMPTOMS

When you try to print to a file, no file is created and you do not receive an error message.

CAUSE

If there is insufficient space on the disk to create the file, Windows 95 does not present you with an error message. Instead, the file is simply not created.

RESOLUTION

To work around this problem, print to a different disk or free some disk space by deleting unwanted files.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
