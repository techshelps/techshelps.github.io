Alacrity Equip+ Does Not Function Properly in Windows 95

 Last reviewed: August 22, 1995
Article ID: Q132775
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you try to install Alacrity Equip+ on a computer running Windows 95, you find that you are unable to install the program. If you are able to install the program, you find that many of the program's features may not function properly. In particular, the Tooltips, OCR, and Fax features may not function properly or may not function at all.

CAUSE

Alacrity Equip+ versions 6.0 and earlier are not completely compatible with Windows 95.

RESOLUTION

For information about a possible software upgrade that corrects these problems, contact your software manufacturer.

MORE INFORMATION
