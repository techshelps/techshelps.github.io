Configuration Backup Tool Does Not Restore All Settings

 Last reviewed: May 7, 1997
Article ID: Q142572
95 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Windows 95

SYMPTOMS

When you restore a backup of the Windows 95 registry using the Configuration Backup tool (Cfgback.exe), some settings are not properly restored.

RESOLUTION

Rather than using the Configuration Backup tool, use another method to back up and restore the registry. For additional information, please see the following articles in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q139437
   TITLE     : Windows 95 Emergency Recovery Utility

   ARTICLE-ID: Q132332
   TITLE     : How to Back Up the Registry

STATUS

Microsoft has confirmed this to be a problem in Configuration Backup version 4.0 for Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
