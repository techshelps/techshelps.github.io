ProShare 1.8 Does Not Work with Windows 95 TCP/IP

 Last reviewed: September 7, 1995
Article ID: Q129765
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

Intel ProShare version 1.8 cannot find the TCP/IP network protocol when you scan for it.

CAUSE

ProShare does not recognize and does not work with the Windows 95 TCP/IP network protocol.

RESOLUTION

To work around this behavior, use either of the following methods:

  • Contact Intel about a possible upgrade to ProShare.
  • Use a different network protocol.

MORE INFORMATION

Intel ProShare uses a digital camera board and an audio/ISDN board to provide video conferencing.
