System Agent Err Msg: Cannot Find the File Cmpagent.exe

 Last reviewed: January 26, 1996
Article ID: Q136837
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Plus! for Windows 95

SYMPTOMS

When you schedule a Compression Agent task in System Agent, you may receive the following error message when you click Run or OK: 

   Cannot find the file Cmpagent.exe. Make sure the path and filename
   are correct.

CAUSE

This problem can occur if your compressed drive has not been upgraded from DriveSpace 2 to DriveSpace 3. Compression Agent works only with DriveSpace 3.

RESOLUTION

Either upgrade the compressed drive to DriveSpace 3, or remove the Compression Agent task in System Agent.

To upgrade a compressed drive to DriveSpace 3, follow these steps:

  1. Click the Start button, point to Programs, point to Accessories, point to System Tools, and then click DriveSpace.

  2. On the Drive menu, click Upgrade.

  3. Follow the instructions on the screen.

To remove a Compression Agent task from System Agent, follow these steps:

  1. Double-click the System Agent icon on the taskbar.

  2. In System Agent, click the Compression Agent task you want to remove.

  3. On the Program menu, click Remove.

  4. When you are prompted to confirm the removal, click Yes.

  5. Close System Agent.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Plus! for Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Compression Agent is not compatible with versions of DriveSpace prior to DriveSpace 3.
