Iomega Jaz Tools Software May Not Work Properly on FAT32 Media

 Last reviewed: December 15, 1997
Article ID: Q159828
95 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Windows 95 OEM Service Release versions 2, 2.1

SYMPTOMS

When you use Iomega's Jaz Tools software on a FAT32-formatted Jaz cartridge, the following items may not function:

  • Software eject JAZ drive
  • Write protect disk
  • Password protect media

CAUSE

Iomega Jaz Tools software with a date prior to 6/25/96 is not compatible with the FAT32 file system in Windows 95 OEM Service Release 2 (OSR2).

RESOLUTION

Contact Iomega for an updated version of the Jaz Tools software.
