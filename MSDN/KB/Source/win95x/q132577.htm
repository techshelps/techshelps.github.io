Setup Program for Grammer Game Warns User of Unknown Shell

 Last reviewed: August 21, 1995
Article ID: Q132577
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you run the Setup program for Grammar Games (for Windows) by Davidson & Associates, Inc., you receive the following message: 

   Setup has detected a different program shell than Microsoft's Project
   Manager. If an error occurs during setup, check the Readme.txt file for
   information on how to temporarily change your program shell."

CAUSE

Grammar Games expects to find the Windows version 3.1 Program Manager running. However, Windows 95 uses a new shell by default. Program Manager is not used.

RESOLUTION

The warning is harmless, and you can ignored it.
