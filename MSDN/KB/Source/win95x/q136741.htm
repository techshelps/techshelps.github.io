Error Message: The Fax Server Could Not Be Removed

 Last reviewed: September 14, 1995
Article ID: Q136741
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

If you remove a fax device (local or network) from the list of available fax/modems on the Modem tab in the properties for Microsoft Fax, the fax device may not be removed from the list. If you try to remove the fax device again, you may receive the following error message: 

   The fax server could not be removed

CAUSE

The registry settings are changed, but the display was not updated properly.

RESOLUTION

Close the properties for Microsoft Fax and then open the properties again to refresh the list of available fax devices.
