Err Msg: This Program Cannot Be Run Due to Restrictions...

 Last reviewed: October 4, 1995
Article ID: Q137685
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you are installing Windows 95, you may receive the following error message: 

   This program cannot be run due to restrictions in effect on this
   computer. Please contact your system administrator.
When you click OK, Setup continues normally.

CAUSE

This error message can occur when you are reinstalling Windows 95 or upgrading over an earlier version of Windows 95 if System Policies have been used to disable the MS-DOS command prompt on the computer. Setup tries to run a real-mode version of ScanDisk, but cannot if the MS-DOS command prompt has been disabled.

RESOLUTION

To work around this problem, use either of the following methods:

  • Contact your system administrator to have the command prompt enabled.
  • Run Setup with the /IS switch to prevent Setup from running ScanDisk.
