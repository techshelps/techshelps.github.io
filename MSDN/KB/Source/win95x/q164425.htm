Cannot Use Keyboard to Set Connection Time in CompuServe Mail

 Last reviewed: February 28, 1997
Article ID: Q164425
95 WINDOWS kbui kbtool

The information in this article applies to:

  • Microsoft Windows 95

SYMPTOMS

You cannot use the keyboard to select the connection time when you are setting up the CompuServe Mail service for Microsoft Exchange.

RESOLUTION

Use a mouse or other pointing device to change the connection times.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
