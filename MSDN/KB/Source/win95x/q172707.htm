File Manager Shows Incorrect Date for Year 2000 or Later

 Last reviewed: December 4, 1997
Article ID: Q172707
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

File Manager displays an incorrect date for files created with a date of 01-01-2000 or later.

RESOLUTION

This issue is resolved by the following updated file for Windows 95 and Windows 95 OEM Service Release 2 (OSR2): 

   Winfile.exe version 4.00.951 (dated 3/11/97) and later
To install this update, follow these steps:

  1. Download the W95filup.exe file from the Microsoft Software Library an empty folder.

  2. In My Computer or Windows Explorer, double-click the W95filup.exe file you downloaded in step 1.

  3. Follow the instructions on the screen.

The following file(s) are available for download from the Microsoft Software Library: 

 ~ W95filup.exe (size: 245800 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
The following files are installed by W95filup.exe: 

   File name     Version    Date/Time        Size     Destination folder
   ---------------------------------------------------------------------
   Winfile.exe   4.00.951   3/11/97 09:51a   115,456  Windows\System

MORE INFORMATION

For information about a similar issue with Windows 3.1 and Windows for Workgroups, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q85557
   TITLE     : File Manager Shows Garbled Date for Year 2000 or Later
