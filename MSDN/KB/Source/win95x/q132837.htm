Toshiba 4500 May Not Resume Correctly if Suspended from Start

 Last reviewed: February 2, 1996
Article ID: Q132837
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

On a Toshiba 4500 computer, when you use the Suspend command on the Start menu, the computer may spontaneously restart during the resume phase.

CAUSE

An old version of Toshiba's Apm.sys driver may prevent programs from resuming properly.

RESOLUTION

Initiate the Suspend command by using the Suspend button on the portable computer, rather than by using the Suspend command on the Start menu. Also, contact Toshiba for an updated Apm.sys driver.

MORE INFORMATION

A similar problem has been reported on the Toshiba 4700cs computer. You can work around the problem by using the Suspend button on the computer rather than the Suspend command on the Start menu.
