Cannot Cancel Changes to Network Adapter Properties

 Last reviewed: August 30, 1996
Article ID: Q130823
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

After you modify the properties for the Intel EtherExpress 16 or 16TP network adapter in your computer, the Cancel button is unavailable and you cannot cancel the changes you have made. In particular, the following steps cause the Cancel button to become unavailable:

  1. In Network properties, double-click the Intel EtherExpress 16 or 16TP card to modify its properties.

  2. Click the Resources tab.

  3. Change any setting on the Resources tab.

  4. Click any other tab.

Note that the Cancel button does not become unavailable until you click a tab other than the Resources tab.

RESOLUTION

To cancel the changes you made to the network adapter properties, follow these steps:

  1. Click the Close button to close the network adapter properties.

  2. Click the Cancel button to close Network properties.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
