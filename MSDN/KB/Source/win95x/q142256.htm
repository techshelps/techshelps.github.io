PSI Hard Disk Controllers Incompatible with Windows 95

 Last reviewed: January 9, 1996
Article ID: Q142256
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

Windows 95 may stop responding (hang) during startup, even in Safe mode, if you are loading the graphical user interface (GUI). However, you can boot Windows 95 to a command prompt.

CAUSE

This problem can occur if your computer contains one of the following hard disk controllers by Perceptive Solutions, Inc. (PSI):

  • Easy Cache
  • Easy Cache Pro
  • Hyperstore
  • Hyperstore Easy Cache Pro

According to PSI Technical Support, these hard disk controllers are incompatible with Windows 95.

RESOLUTION

Contact PSI for information about upgrading your hard disk controller.
