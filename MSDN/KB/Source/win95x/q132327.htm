Net Watcher Does Not Disconnect User

 Last reviewed: August 31, 1995
Article ID: Q132327
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you use Net Watcher to disconnect a client user, the user receives no disconnect notification and can still access the network resource.

CAUSE

When a user is disconnected from a network resource by Net Watcher, Windows 95 automatically reconnects the user when he or she tries to access the network resource.

RESOLUTION

To disconnect a user so that he or she cannot reconnect, use either of the following methods:

  • Remove the access privileges to the resource.
  • Change the password for the resource.

STATUS

This behavior is by design to allow Windows 95 to restore a lost network connection without user input.
