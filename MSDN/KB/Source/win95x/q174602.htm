Computer Hangs Using Panasonic "Big 5" CD-ROM Drive

 Last reviewed: October 2, 1997
Article ID: Q174602
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you try to start your Windows 95-based computer, it may stop responding (hang).

CAUSE

This behavior can occur if you are using IDE busmastering drivers with the Panasonic "BIG 5" 5-disc CD-ROM changer (model SQ-TC510N).

RESOLUTION

To resolve this problem, contact Panasonic to update the device with the latest firmware revision.

To work around this problem, remove the CD-ROM changer. For information about removing your CD-ROM changer, contact Panasonic.
