MS Access 2.0 System Error When You Create Two New Databases

 Last reviewed: August 21, 1995
Article ID: Q132793
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Access version 2.0

SYMPTOMS

In Microsoft Access, when you create a database and accept the default name (for example, Db4.mdb), and then, with the Db4.mdb database open, you create another database, you receive the following error message: 

   System Error: The file you are trying to open is already in use by
   another program. Quit the other program, and then try again.

CAUSE

Microsoft Access tries to access the file (Db4.mdb) that is already open.

RESOLUTION

To open two databases at the same time, rename the first database before you open the second database.
