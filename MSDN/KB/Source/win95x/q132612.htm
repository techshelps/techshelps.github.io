Arcada Backup Hangs in MS-DOS Mode

 Last reviewed: August 21, 1995
Article ID: Q132612
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

Arcada Backup may stop responding (hang) when you are using it to back up or restore files.

RESOLUTION

Run Arcada Backup in MS-DOS mode. Specify a special configuration that does not load Emm386.exe but loads the necessary ASPI drivers. To create a special configuration, click the Advanced button on the Program tab in the program's properties.
