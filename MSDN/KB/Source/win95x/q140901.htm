Err Msg: Could Not Decode This Setup (.cab) File...

 Last reviewed: March 4, 1998
Article ID: Q140901
The information in this article applies to:
  • Microsoft Windows 95

SYMPTOMS

When you try to install Windows 95, or install a component that requires copying files from the original Windows 95 disks or CD-ROM, you may receive the following error message: 

   Setup has detected the following decoding error: "Could not decode
   this setup (.CAB) file." Setup will attempt to recover from this
   situation, click OK to continue.
When you click OK, Setup either proceeds or generates the same error message again.

CAUSE

This error can be caused by any of the following conditions:

  • There is a virus on your computer.
  • Your Windows 95 CD-ROM or disks are damaged.
  • Your CD-ROM drive is not functioning properly.

RESOLUTION

You may be able to resolve the problem using one of the following methods:

  • Check your computer for a virus using virus-detection software.
  • To determine if your Windows 95 CD-ROM is damaged, try copying the contents of the Win95 folder on the CD-ROM to an empty folder on the hard disk.

    If you can copy the Win95 folder, run Setup from the folder on the hard disk containing the files you just copied.

    For information about determining whether your disks are damaged or defective, please see the following article in the Microsoft Knowledge Base: 

          ARTICLE-ID: Q124970

          TITLE     : Troubleshooting DMF Issues

   If you determine that you have a damaged CD-ROM or disk, call the
   Microsoft Order Desk at (800) 360-7561 to order a replacement. If you
   are outside the United States, contact the Microsoft subsidiary for
   your country.


  • If your computer does not have a virus and your Windows 95 CD-ROM is
   not damaged, consult the CD-ROM drive's documentation or manufacturer
   to determine whether the CD-ROM drive and its software are configured
   correctly.
