BUG: Invalid Page Fault When Registering Visual Studio

 Last reviewed: March 19, 1997
Article ID: Q164541
The information in this article applies to:
  • Microsoft Visual Studio 97 on the following platforms: - Windows NT - Windows 95

SYMPTOMS

When you install Visual Studio on a system that does not have a default Internet browser installed and select the Register Microsoft Visual Studio option in the master setup dialog box, an Invalid Page Fault error occurs.

CAUSE

A default Internet browser is required to do online registration of Visual Studio.

RESOLUTION

Install Internet Explorer, version 3.0 (build 4.70.1215) or higher, prior to attempting to register Microsoft Visual Studio. To install the Internet Explorer manually, run Setup.exe from the Internet Explorer 3.0 (IE3.0) directory on the first Microsoft Visual Studio CD-ROM.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Several Visual Studio 97 components require the Internet Explorer COM object and other components from Internet Explorer 3.0 build 4.70.1215 or later. These components are installed when you install Internet Explorer, and Visual Studio setup checks to see if these components are installed. If they are not installed, then Visual Studio will launch the Internet Explorer build 4.70.1215 setup.

Steps to Reproduce Behavior

  1. Install a recent build of Internet Explorer 3.0 (build 4.70.1215 or later).

  2. Uninstall Internet Explorer. This removes the Internet Explorer executable but leaves the other components required by Visual Studio for viewing online documentation. Because all the expected Internet Explorer components remain, Visual Studio setup will not attempt to re-install Internet Explorer.

  3. Install Visual Studio.

  4. Select the "Register Microsoft Visual Studio" option.

NOTE: The application will terminate due to an Invalid Page Fault error. The fault will occur during any portion of the installation of any Visual Studio component if the "Register Microsoft Visual Studio" option is selected.
