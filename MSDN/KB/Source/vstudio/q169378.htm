DOC: Visual Basic 5.0 .DOC Files Not Included in Tools Directory

 Last reviewed: June 3, 1997
Article ID: Q169378
The information in this article applies to:
  • Microsoft Visual Studio, Enterprise Edition 97
  • Microsoft Visual Studio, Professional Edition 97

SUMMARY

Visual Basic 5.0 Books Online states the following: 

   "Chapters in this book are also available as Microsoft® Word documents,
   enabling you to copy or print multiple topics at once. If you don't have
   Word, you can view and print the documents using Word Viewer 7.1. Both
   the viewer Setup.exe and the Books Online .doc files are included in the
   Tools directory of your Visual Basic CD-ROM."
However, the documents and viewer are not included in the VB5.0\tools\docs directory of Visual Studio CD-ROM that includes Visual Basic 5.0.

MORE INFORMATION

Visual Studio 97 includes a coupon for a set of Visual Basic 5.0 documentation. You can send in the completed coupon to this printed documentation.

Another option is to print the documentation from the MSDN browser that comes with Visual Studio.

You may also download the files from the Visual Studio Owners site at: 

   http://www.microsoft.com/vstudio/owner/default.asp
The documentation is available as a single downloadable file in the past "Weekly Feature" section of the Visual Basic area.
