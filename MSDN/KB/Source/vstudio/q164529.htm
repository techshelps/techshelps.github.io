FILE: Updated Automation System Files

 Last reviewed: November 26, 1997
Article ID: Q164529
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Windows NT versions 3.51, 4.0

SUMMARY

Some applications and components require that you have newer versions of the Automation (formerly called OLE Automation) libraries installed on your system. Oadist.exe is a self-extracting Cabinet that will install the Automation libraries onto your system. Simply download the file Oadist.exe and double-click to run it. It will prompt you to be sure that you want to proceed. Answering "yes" will update the Automation libraries on your computer.

The following files are available for download from the Microsoft Software Library: 

 ~ Oadist.exe (size: 490096 bytes) 
 ~ Oadist2z.exe (size: 559499 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

MORE INFORMATION

The following files are updated by this program: 

   Filename       Version     Size (bytes)
   ------------   ---------   ------------
   OLEAUT32.DLL   2.20.4054   491,792
   COMCAT.DLL     4.71         22,288
   OLEPRO32.DLL   5.0.4055     32,528
   ASYCFILT.DLL   2.20.4056   120,592
   STDOLE2.TLB    2.20.4054    16,896
The Oadist.exe setup program does not work on Windows NT 3.51. If you need to install the updated Automation system files on Windows NT 3.51 you should download Oadist2z.exe. Oadist2z.exe is a self-extracting zip that contains a setup program named Oadist2.exe. Oadist2.exe uses a different method to install the Automation system files that also works on Windows NT 3.51. To use Oadist2.exe you need to download Oadist2z.exe and copy it into a new directory. When you run Oadist2z.exe, it extracts out two files. One is a readme and the other is the Oadist2.exe setup program.
