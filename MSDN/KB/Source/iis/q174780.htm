Filtering Adobe Acrobat Files for Use with Index Server

 Last reviewed: October 29, 1997
Article ID: Q174780

The information in this article applies to:

  • Microsoft Index Server version 1.1
-------------------------------------------------------------------------

SUMMARY

By default, Index Server version 1.1 will not filter Adobe Acrobat (.pdf) files. Index Server does not ship with an Adobe Acrobat filter.

MORE INFORMATION

Adobe now as a filter for Acrobat (.pdf) files. You can find the filter at http://www.adobe.com/prodindex/acrobat/ifilter.html
