List of Microsoft and Other HTML Editors

 Last reviewed: April 22, 1997
Article ID: Q143152
The information in this article applies to:
  • Microsoft Internet Information Server version 1.0

SUMMARY

This article lists several HTML (Hypertext Markup Language) Editor programs.

MORE INFORMATION

HTML Documents for World Wide Web (WWW) servers can be written on any text editor as long as you know the proper HTML syntax. However, HTML documents can be easily produced using an HTML editor such as Internet Assistant for Microsoft Word. Internet Assistant for Microsoft Word is available on WWW.MICROSOFT.COM. There are also numerous shareware HTML editors available on the Internet as listed below:
