IExplorer 2.0 Doesn't Return FTP Directory List with IIS

 Last reviewed: April 22, 1997
Article ID: Q147905

The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51
  • Microsoft Windows 95
  • Microsoft Internet Information Server version 1.0

SYMPTOM

The Microsoft Internet Explorer 2.0 connects to Internet Information Server (IIS) FTP service; however, it does not return directory listing. The Internet Explorer version 1.5 and Netscape viewer return the directory listing as expected.

RESOLUTION

In the IIS FTP service, select the Directories tabs on the Properties dialog box. Select Directory Listing Style as UNIX instead of MS-DOS. Click the Apply button for changes to take effect immediately.
