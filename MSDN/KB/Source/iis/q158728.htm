Gopher Types Not Implemented w/ Internet Information Server 2.0

 Last reviewed: October 30, 1997
Article ID: Q158728
The information in this article applies to:
  • Microsoft Internet Information Server version 2.0

SYMPTOMS

When you use Gdsset.exe to create a Gopher service tag file, you may get the following error message: 

   Unable to write tag file.

CAUSE

The Internet Information Server documentation incorrectly indicates the following tag file types are supported: 

Code       Type
----       ----
2          CSO Phone-book Server
6          uuencoded files
8          Telnet
T          TN3270
These Gopher tag types are not implemented and are not supported under the Internet Information Server version 2.0 Gopher services.
