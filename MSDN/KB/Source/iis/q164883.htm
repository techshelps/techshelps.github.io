Internet Information Server (IIS) Version 3.0 Features

 Last reviewed: November 20, 1997
Article ID: Q164883

The information in this article applies to:

  • Microsoft Internet Information Server version 3.0

SUMMARY

Internet Information Server (IIS) version 3.0 is available for download from the following Internet site: 

   http://www.microsoft.com/iis/
To see a summary of all the included features click Summary of Internet Information Server 3.0 Downloadable Features.
  • Microsoft Active Server Pages version 1.0 is an open, compile-free application environment in which you can combine HTML, scripts and ActiveX server components to easily build powerful Web-based business solutions. Active Server Pages includes support for VBScript and JScript, and is compatible with almost any scripting engine. Active Server Pages will enable many more people to create server-side Web applications for the first time.
  • Microsoft Index Server version 1.1 is an integrated search engine that provides full-text indexing and searching documents in any format. It includes built-in support for HTML, text, and Microsoft Office documents. Index Server enables users to quickly locate specific information on their intranet or Internet sites.
  • Microsoft NetShow version 1.0 enables Internet Information Server to provide one-to-many multimedia communication and information distribution. NetShow technologies provide live and on-demand audio and video content, while reducing impact on network bandwidth.
  • The FrontPage 97 Server Extensions allow one button publishing and graphical site management to help web publishers keep information organized. With FrontPage 97, organizations can more effectively develop and manage their corporate intranet and Internet sites.
  • Crystal Reports for Internet Information Server is the world-leading client/server report writer that lets you create presentation quality reports and integrate them into database applications. Crystal Reports for Internet Information Server allows you to report off your Web Server log files and includes pre-formatted Web log reports.

To download the features of IIS 3.0

  1. You can choose to download the features of IIS 3.0 over the Internet by following the steps outlined,

    -or-

    You can install IIS 3.0 by ordering the Microsoft Windows NT Service Pack 2 compact disc.

  2. If you choose to download, please fill out the Registration Form.

  3. Select platform, features, and languages desired.

  4. Choose download location.

  5. At the selected download location, you must download your desired features individually. You can remain on this page until all downloads are complete.

  6. If you chose to download features that were unavailable at the location chosen in step 4, this page lists secondary download locations where these features are available.
