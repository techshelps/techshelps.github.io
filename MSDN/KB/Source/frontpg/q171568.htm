FP: Can't Read Configuration File Installing Server Extensions

 Last reviewed: March 18, 1998
Article ID: Q171568
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SYMPTOMS

When you attempt to install FrontPage Server Extensions, you receive the following error message: 

   Error: Unable to read Microsoft Personal Web Server configuration file.
   Error Code 1114.

CAUSE

This error message appears when the configuration files for the Microsoft Personal Web Server have been removed. For example, this error will appear if you uninstall the Microsoft Personal Web Server before you uninstall the FrontPage Server Extensions.

RESOLUTION

Reinstall the Microsoft Personal Web Server, using either of the following methods.

Method 1: Run the Microsoft Personal Web Server Setup Program

Install the Microsoft Personal Web Server using the FrontPage compact disc. If the FrontPage Setup program does not automatically run when you insert the compact disc in the CD-ROM drive, use these steps to install the Personal Web Server:

  1. Click Start, and then click Run.

  2. Click Browse.

  3. Select the drive that contains the FrontPage compact disc.

  4. Select Setup.exe, and then click OK.

  5. Click "The Microsoft Personal Web Server for Windows 95."

  6. Follow the instruction on your screen.

Method 2: Download the Microsoft Personal Web Server

To download the Microsoft Personal Web Server, please see the following Microsoft World Wide Web site: 

   http://www.microsoft.com/ie
Click Download, and then click Personal Web Server.

NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
