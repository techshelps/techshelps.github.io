FP98: Beta Version of FP98 Not Qualifying Product for Upgrade

 Last reviewed: March 18, 1998
Article ID: Q180305
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

If you attempt to upgrade the beta version of FrontPage 98 to the released version, you will receive a message indicating that you are trying to upgrade from a non-qualifying product.

CAUSE

The beta version of FrontPage 98 does not qualify as an upgrade product for FrontPage 98. For additional information about what products qualify as upgrade products for the upgrade version of FrontPage 98, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q178487
   TITLE     : FP98: Requirements to Upgrade to FrontPage 98

RESOLUTION

To upgrade to FrontPage 98, remove the beta version of FrontPage 98 and then install the full retail version of FrontPage 98. If you previously installed FrontPage 97 or FrontPage 1.1, you can install the upgrade version of FrontPage 98 by following these steps.

NOTE: Because the Setup program for the upgrade version of FrontPage 98 searches for a qualifying product, you will need the FrontPage 97 compact disc or the FrontPage version 1.1 disks in order to complete the following steps.

  1. On the Windows Start menu, point to Settings, and then click Control Panel.

  2. Double-click the Add/Remove Programs icon.

  3. In the Add/Remove Programs dialog box, do the following: 

        a. In the list of installed programs, select FrontPage 98.


        b. Click Add/Remove.


        c. When you receive a message confirming that you are removing the
       files, click OK.


        d. When asked if you want to remove all files, click Yes To All.


        e. Click OK.

  4. Close the Add/Remove Programs dialog box and Control Panel.

  5. Install the upgrade version of FrontPage 98 by following these steps: 

        a. Insert the FrontPage 98 compact disc in your CD-ROM drive.


           NOTE: If the FrontPage 98 Setup program starts automatically, the
       Master Setup screen will appear. If FrontPage Setup does not start
       automatically, do the following:


           1) On the Windows Start menu, click Run.


           2) Click Browse.


           3) Open the CD-ROM folder and double-click the Setup.exe file.


        b. Click Install FrontPage 98.


        c. On the Welcome screen in the Install Shield Wizard, click Next.


        d. Enter your Name and Company Name, and then click Next.


           NOTE: You must fill in both boxes to continue setting up FrontPage 98.


        e. Click Yes.


        f. When you are prompted to enter the CD Key, type the number located
       on the back of the compact disc jewel case and then click OK.


        g. Click Continue.

  6. When the upgrade check fails, click Locate. Perform the upgrade check for the version of FrontPage you are upgrading: 

        - If you are upgrading from FrontPage 97 (compact disc), follow these steps:


          a. Remove the FrontPage 98 compact disc from the CD-ROM drive and then

             insert the FrontPage 97 compact disc.

         NOTE: If the FrontPage 97 Master Setup screen appears, close it.

      b. In the Locate Directory dialog box, select the CD-ROM drive and
         click OK.

      c. Follow the instructions on your screen.

    - If you are upgrading from FrontPage 1.1 (disks), follow these steps:

      a. Insert the FrontPage 1.1 setup disk (disk 1) in the floppy disk drive.

      b. In the Locate Directory dialog box, select the disk that contains
         the FrontPage 1.1 Setup disk and click OK.

      c. Follow the instructions on your screen.
