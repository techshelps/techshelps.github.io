FP97: "Hard Disk Search Failed" Error During Upgrade Check

 Last reviewed: October 14, 1997
Article ID: Q169083
The information in this article applies to:
  • Microsoft FrontPage 97 for Windows

SYMPTOMS

When you are attempt to install the upgrade version of FrontPage 97, you receive the following message, even though setup is still searching your hard disk: 

   Upgrade Check - Hard Disk Search Failed.

CAUSE

The FrontPage 97 Setup program searches the C:\Program Files\Microsoft FrontPage\bin folder for executable files. It Setup does not find the files, you will receive the error message described in the Symptoms section.

RESOLUTION

To resolve this problem and continue running the Setup program, click OK.

STATUS

Microsoft has confirmed this to be a problem in Microsoft FrontPage 97. We are researching this problem and will post new information here as it becomes available.
