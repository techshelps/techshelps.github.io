FP97: System Requirements for FrontPage 97 with Bonus Pack

 Last reviewed: March 18, 1998
Article ID: Q166360
The information in this article applies to:
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SUMMARY

This article lists the system requirements needed to run Microsoft FrontPage 97 and the Bonus Pack products included with Microsoft FrontPage.

MORE INFORMATION

Microsoft FrontPage

  • A personal or multimedia computer with a 486 or higher processor.
  • Microsoft Windows 95 operating system or Microsoft Windows NT Workstation 3.51 Service Pack 5 or later (will not run on earlier versions of Windows).
  • 8 megabytes (MB) of RAM memory for use on Windows 95; 16 MB for use on Windows NT Workstation; 16 MB recommended for Microsoft Personal Web Server.
  • 30 MB of available hard-disk space.
  • A CD-ROM drive.
  • A VGA or higher-resolution video adapter (Super VGA, 256-color recommended).
  • A Microsoft Mouse, Microsoft IntelliMouse, or compatible pointing device.
  • Internet features require you to obtain Internet access. Internet and other online access may require payment of a separate fee to a service provider.
  • TCP/IP compliant Winsock 1.1 or later.

Additional Hard-Disk Requirements for Bonus Pack Components

  • 11 MB for Microsoft Internet Explorer.
  • 1 MB for Microsoft Personal Web Server.
  • 1 MB for Microsoft Web Publishing Wizard.
  • 2 MB for Internet Mail and News.

Microsoft Image Composer

To use Microsoft Image Composer, you need:

  • 16 MB of RAM memory (32 MB recommended).
  • Hard-disk space required: 11 MB minimum; 17 MB complete.
  • VGA, 256-color (Super VGA TrueColor with 2 MB of video memory recommended).

Microsoft Image Composer Options
  • Windows-compatible tablet with mouse emulation for basic mouse support, or WinTab-compatible tablet for Full Pressure support.
  • TWAIN-compatible scanners or digital cameras.
