FP: Can't Find Executable Running Server Utilities

 Last reviewed: March 18, 1998
Article ID: Q148428
The information in this article applies to:
  • Microsoft FrontPage for Windows, versions 1.0, 1.0a
  • Microsoft FrontPage Personal Web Server, versions 1.0, 1.0a

SYMPTOMS

When you try to run the FrontPage TCP/IP Test or the Check Installation utility, you receive an "executable could not be found" error message.

CAUSE

When you install only the FrontPage client, icons or shortcuts for the FrontPage TCP/IP Test and Check Installation server utility programs are installed in the Vermeer group. However, the executable programs for the Check Installation (Fpchk.exe), the FrontPage TCP/IP Test (Tcptest.exe), and the Server Administrator (Fpsrvadm.exe) are not installed.

RESOLUTION

To use the server utilities, install both the FrontPage client and the FrontPage Personal Web Server.
