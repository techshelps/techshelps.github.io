FP98: How To Delete Web Names From Getting Started Dialog Box

 Last reviewed: March 18, 1998
Article ID: Q173641
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SUMMARY

In FrontPage 98, you can remove a Web from the list of Webs in the "Open an Existing FrontPage Web" list in the Getting Started dialog box in FrontPage Explorer. In contrast, in earlier versions of FrontPage, you can remove a Web by editing the Fpexplor.ini file. This article describes how to remove a Web from the "Open an Existing FrontPage Web" list in the Getting Started dialog box. This dialog box appears when you start FrontPage 98, when you click the Open FrontPage Web button on the toolbar, or when you click Open FrontPage Web on the File menu.

NOTE: The Getting Started dialog box will appear when you start FrontPage if the "Always open last Web" option was not selected the last time you opened a Web. If this option was selected the last time you opened a Web, then when you restart FrontPage, the last Web will open automatically rather than the Getting Started dialog box.

MORE INFORMATION

Use either of the following methods to delete a Web from the Getting Started dialog box.

NOTE: Using either of these methods does not remove the Web from the disk. These procedures merely remove the Web name from the "Open an Existing FrontPage Web" list in the Getting Started dialog box.

Method 1

  1. In the Getting Started dialog box, in the "Open an Existing FrontPage Web" list, click the Web you want to remove.

  2. Press DELETE.

  3. When you receive a message asking you to confirm that you want to remove the Web, click Yes.

Method 2

  1. In the Getting Started dialog box, in the "Open an Existing FrontPage Web" list, right-click the Web you want to remove.

  2. Click Remove From List on the menu that appears.

  3. When you receive a message asking you to confirm that you want to remove the Web, click Yes.

REFERENCES

For more information about removing Webs from the "Open an Existing FrontPage Web" list, click Help in the Getting Started dialog box.
