FP98: Nothing Happens Clicking Hover Button with Hyperlink

 Last reviewed: March 18, 1998
Article ID: Q179064
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

When you use the Hyperlink command on the Edit menu in FrontPage Editor to create a hyperlink for a hover button, the hyperlink will not work in your Web browser. In other words, nothing happens when you click the hyperlink.

CAUSE

The HyperText Markup Language (HTML) does not recognize the <Applet> tag as a source for an anchor (<A>) tag.

RESOLUTION

To create a hyperlink from a hover button, follow these steps:

  1. Start FrontPage Editor. Open the page that contains the hover button.

  2. Right-click the hover button and then click Java Applet Properties on the menu that appears.

  3. Click Browse.

  4. Select the page you want to use as the destination hyperlink.

  5. Click OK.

  6. Click OK.
