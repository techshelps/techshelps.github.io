FP98: Preview Tab Not Available in FrontPage Editor Window

 Last reviewed: March 18, 1998
Article ID: Q173419
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SUMMARY

The FrontPage 98 Editor features new tabs at the bottom of the screen so you can quickly switch between different views as you create your Web page. The Preview tab allows you to preview your work in progress without saving it back to the Web server. However, the Preview tab may not appear on all computers.

MORE INFORMATION

FrontPage uses controls installed with Microsoft Internet Explorer, version 3.0 or later, to display the Preview tab in FrontPage Editor. Therefore, if Internet Explorer is not installed, you will not see the Preview tab in FrontPage Editor.

Each time you start FrontPage Explorer, FrontPage reads the [Internet Explorer Integration] section of the Frontpg.ini file to determine whether to show or hide the Preview tab. In the following example, the DisablePagePreviewTab=0 statement tells FrontPage to show the Preview tab. If you change this value to 1, FrontPage will hide the Preview tab. Note that FrontPage will only show the Preview tab if you have installed Internet Explorer, version 3.0 or later.

NOTE: If you uninstall Internet Explorer after you install FrontPage, the Preview tab will be available when you run FrontPage. 

   [Internet Explorer Integration]
   UseAlternateThemePreview=1
   DisablePagePreviewTab=0
If you have installed FrontPage on a computer on which you are running Internet Explorer, version 2.0, Netscape Navigator, or Netscape Communicator, the Preview tab will not be available. If you then install Internet Explorer 3.0, the Preview tab will appear and you will not have to edit the Frontpg.ini file.

If you have installed FrontPage on a computer on which you have installed Netscape Navigator or Netscape Communicator, and you change the DisablePagePreviewTab setting to 1, the Preview tab will not be present. If you then install Internet Explorer, version 3.0, the Frontpg.ini file will be modified (that is, the DisablePagePreviewTab parameter will be set to 0), and the Preview tab will appear when you run FrontPage.
