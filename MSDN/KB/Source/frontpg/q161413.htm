FP: How to Install the FrontPage Personal Web Server

 Last reviewed: March 18, 1998
Article ID: Q161413
The information in this article applies to:
  • Microsoft Frontpage 98 for Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SUMMARY

The FrontPage Personal Web Server is installed when you perform a custom installation of FrontPage (select the Custom option during FrontPage Setup).

MORE INFORMATION

To Install the FrontPage Personal Web Server

  1. In FrontPage Setup, click Custom.

  2. Click to select the "Personal Web Server" check box.

If you have already installed FrontPage and you did not install the FrontPage Personal Web Server, you must reinstall FrontPage and follow steps 1 and 2.

To Install the Server Extensions

FrontPage 98

  1. Insert the FrontPage compact disc in the CD-ROM drive.

  2. On the Windows Start menu, click Run.

  3. Click Browse.

  4. Open the \ServExt folder.

  5. Double-click the executable file you want.

FrontPage 97

  1. Start FrontPage Setup.

  2. Click Custom.

  3. Click to select the Server Extensions check box.
