FP: Can't Create Directory if Home Directory is Root Directory

 Last reviewed: March 18, 1998
Article ID: Q164279
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack
  • Microsoft FrontPage for Windows, version 1.1

SYMPTOMS

When you attempt to install FrontPage Server Extensions, you receive the following error message: 

   Cannot create directory "<drive letter>:"

CAUSE

FrontPage cannot create the required files when the Web server's home directory is the root directory.

RESOLUTION

Create a subdirectory in the root directory on the server, map the new subdirectory as the root directory on the Web server, and then install the FrontPage Server Extensions.

For example, map the root directory, 

   d:
as 

   d:\webs
For more information about mapping the root directory on your Web server, refer to your Web server documentation.
