FP97: Web Servers Supported by FrontPage

 Last reviewed: March 18, 1998
Article ID: Q161158
The information in this article applies to:
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SUMMARY

The FrontPage 97 Server Extensions are designed to be used with the servers listed in the "More Information" section of this article.

MORE INFORMATION

UNIX

When you use any of these operating systems: 

   BSD/OS 2.1
   Digital UNIX 3.2c, 4.0 (Alpha)
   IRIX 5.3, 6.2
   Linux 3.0.3 (from Red Hat software)
   Solaris (SPARC) 2.4, 2.5
   SunOS (SPARC) 4.1.3, 4.1.4
   HP-UX 9.03, 10.01
the FrontPage Server Extensions will run on the following servers: 

   Apache 1.0.5, 1.1.1
   CERN 3.0
   NCSA 1.5, 1.5.2 (we do not support 1.5.1 or 1.5a)
   Netscape Commerce Server 1.12
   Netscape Communications Server 1.12
   Netscape Enterprise 2.0
   Netscape FastTrack 2.0

Windows NT

   Microsoft Internet Information Server 1.x, 2.0, 3.0 (for Intel,
     Alpha, and PPC processors)
   Microsoft Peer Web Services for Intel, Alpha and PPC
   O'Reilly & Associates WebSite Pro, 1.1, 1.0
   Netscape Commerce Server 1.12
   Netscape Communications Server 1.12
   Netscape Enterprise 2.0
   Netscape FastTrack 2.0
   FrontPage Personal Web Server 2.0.2.1112, 1.1.2.6

Windows 95

   Microsoft Personal Web Server 1.0
   FrontPage Personal Web Server 2.0.2.1112, 1.1.2.6
   O'Reilly & Associates WebSite Pro, 1.1, 1.0
