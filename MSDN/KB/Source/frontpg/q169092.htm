FP97: FrontPage Setup Does Not Install Extensions to MS PWS

 Last reviewed: March 18, 1998
Article ID: Q169092
The information in this article applies to:
  • Microsoft FrontPage 97 for Windows with Bonus Pack
  • Microsoft Personal Web Server, version 1.0

SUMMARY

After you install FrontPage 97, the FrontPage Server Extensions are not installed to the Microsoft Personal Web Server.

CAUSE

This problem typically occurs when

  • The computer's host name contains a space or an illegal character.
  • The Microsoft Personal Web Server is not running.

RESOLUTION

To resolve this problem, do the following:

  1. On the Start menu, point to Settings, and then click Control Panel.

  2. Right-click the Network icon, and then click Properties.

  3. Click the Identification tab.

  4. In the Computer Name box, type a name for the computer. NOTE: The name can contain any alpha-number character, including the hyphen (-).

  5. Click OK.

  6. Restart the computer.

  7. In Windows Explorer, double-click the FrontPage Server Administrator (fpsrvwin.exe) file. By default this file is located in the C:\Program Files\Microsoft FrontPage\bin folder.

  8. Install the Microsoft Personal Web Server Extensions. For more information about how to do this, please see the following article in the Knowledge Base.

    ARTICLE-ID: Q167931 

       TITLE     : FP97: How to Install FrontPage Server Extensions for
               PWS

  9. Close the Server Administrator.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
