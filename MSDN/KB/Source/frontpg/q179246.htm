FP98: Text Animations on Included Pages Not Functioning

 Last reviewed: March 18, 1998
Article ID: Q179246
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows

SYMPTOMS

When you use the Include Page feature in a Web document to include a page with text animations, the text animations do not work when viewed in the browser.

NOTE: You will also not see the text that was used to animate the object.

CAUSE

FrontPage 98 will include the page with the text animation, but it will not include the Dynamic HTML script that makes the animation function.

WORKAROUND

To work around this behavior, follow these steps:

  1. Open the page that contains the included page.

  2. Click on any blank line.

  3. Press the SPACEBAR.

  4. On the Format menu, point to Animation, and click the text animation option that you applied to the text in the included page.

  5. On the File menu, click Save.
