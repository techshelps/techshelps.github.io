FP: Error Installing FP to Dell Computer using Windows NT 4.0

 Last reviewed: March 18, 1998
Article ID: Q169085
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SYMPTOMS

When you install Microsoft FrontPage for Windows, you may receive one of the following error messages: 

   Exception number: c0000005 (access violation).
-or- 

   Exception number: oxc000005 0x9f000000 (access violation).
These errors occur when Setup is approximately 96 percent complete, while Setup is attempting to add shortcuts to the Start menu.

These errors may also occur on some Dell computers on which Microsoft Windows NT Workstation version 4.0 is preinstalled.

RESOLUTION

If you are using a Dell computer, you can download a patch to fix this problem.

For additional information, please see the following Dell Computer Corporation World Wide Web site: 

   http://www.dell.com/filelib/31be.htm
