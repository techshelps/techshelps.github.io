FP1: Invalid Page Fault Pressing TAB While Image Is Selected

 Last reviewed: March 17, 1998
Article ID: Q170952
The information in this article applies to:
  • Microsoft FrontPage for Windows, version 1.1

SYMPTOMS

In FrontPage Editor, if you select an image and then press TAB, you will receive the following error message if you are running Windows 95: 

   Invalid page fault in VT11IME.DLL
followed by this message: 

   Invalid page fault in MFC40.DLL
If you are running Microsoft Windows NT 4.0, you will receive this message: 

   Exception: access violation (0xC00000005)

RESOLUTION

To resolve this problem, use either of the following methods before you press TAB.

  • Click above or below the image to deselect the image.

    -or-

  • Press the Home, Page Up, Page Down, or Arrow key to deselect the image.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft FrontPage 97 for Windows with Bonus Pack.
