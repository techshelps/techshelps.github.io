FP97: Don't Need Web Server to Use FrontPage

 Last reviewed: January 12, 1998
Article ID: Q161154
The information in this article applies to:
  • Microsoft FrontPage 98 For Windows
  • Microsoft FrontPage 97 for Windows with Bonus Pack

SUMMARY

You do not need a Web server to work with FrontPage.

You can use FrontPage to create Web documents and manage webs directly on your hard disk; note, however, that WebBot browse-time components will not function without a Web server.

MORE INFORMATION

For more information about which features will not function on a disk-based Web, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q160227
   TITLE     : FP97: Some WebBot Components Don't Work on Disk-Based
               Web
FrontPage 98 and FrontPage 97 with Bonus Pack, both include two Web servers: the FrontPage Personal Web Server and the Microsoft Personal Web Server. To get full functionality from FrontPage, you should use FrontPage with a Web server.
