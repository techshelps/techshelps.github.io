FP98: Requirements to Upgrade to FrontPage 98

 Last reviewed: March 18, 1998
Article ID: Q178487
The information in this article applies to:
  • Microsoft FrontPage 97 for Windows with Bonus Pack
  • Microsoft FrontPage 98 for Windows

SUMMARY

This article outlines the items required to upgrade to FrontPage 98.

NOTE: THE BETA VERSION OF FRONTPAGE 98 DOES NOT QUALIFY AS AN UPGRADE PRODUCT. If you do not have one of the earlier, released versions of FrontPage, you will need to purchase the full version of Microsoft FrontPage 98.

MORE INFORMATION

In order to install the upgrade version of FrontPage 98, you must first have one of the following qualifying products:

  • FrontPage 97 for Windows
  • FrontPage for Windows, version 1.1

In addition, you must have all of the following system resources:
  • a 486 or higher processor.
  • Microsoft Windows 95, Microsoft Windows NT Workstation, or Microsoft Windows NT Server.
  • 16 megabytes (MB) of RAM for Windows 95, 32 MB of RAM for Windows NT.
  • 36 MB of hard disk space.
  • a VGA or higher resolution video adapter.

NOTE: Any online features require Internet access. Internet access requires the use of an Internet Service Provider, which may require payment of a separate fee.

In order to use FrontPage 98, you must have a valid 32-bit Transmission Control Protocol/Internet Protocol (TCP/IP) stack. To check your TCP/IP stack, run the Tcptest.exe file. This file is located in the folder in which you installed FrontPage. By default, this is the \Program Files\Microsoft FrontPage\Bin folder. If you installed FrontPage to a folder other than the default folder, you will need to specify that path when you run this file.

Not all types of TCP/IP are compliant with WinSock 1.1 specifications.
