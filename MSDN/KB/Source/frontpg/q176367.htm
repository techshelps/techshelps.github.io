FP1: IMS TwinTurbo Acceleration May Cause Display Problems

 Last reviewed: March 17, 1998
Article ID: Q176367
The information in this article applies to:
  • Microsoft FrontPage for the Power Macintosh, version 1.0
  • Microsoft Exchange Macintosh client, version 5.0
  • Microsoft Bookshelf 98 for the Power Macintosh

SYMPTOMS

When you run Microsoft FrontPage for the Power Macintosh, Microsoft Exchange, or Microsoft Bookshelf 98 for the Power Macintosh on a PowerComputing PowerTower Pro 225, with a IMS Twin Turbo Accelerator Card, you may lose the blinking insertion point.

WORKAROUND

Disable the Twin Turbo Acceleration Extension in Extensions Manager or remove the extension from the System Folder, and restart the computer.

MORE INFORMATION

If the IMS TwinTurbo Acceleration Extension version 3.8 or 4.0.2 is loaded into memory, FrontPage Editor and Exchange both lose the blinking insertion point on the page. In Bookshelf 98, QuickTime VR Panoramas will display scrambled video. This behavior has been observed under both System 7.6.1 and 8.0. on a PowerComputing PowerTower Pro 225.
