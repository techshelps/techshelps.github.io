BUG: Dupe Timestamp Values After Installing Alpha 4.21a SP3/4

 Last reviewed: May 2, 1997
Article ID: Q147840

The information in this article applies to:
  • Microsoft SQL Server, version 4.21a
BUG# NT: 13528 (4.21a Service Pack 3 and Service Pack 4 - Alpha)

SYMPTOMS

Duplicate timestamp values can be generated after installing Service Pack 3 or Service Pack 4 for SQL Server 4.21a on the Alpha platform.

WORKAROUND

Avoid using timestamps, or uninstall SQL Server Service Pack 3 or 4. If there are fixes in Service Pack 3 or Service Pack 4 that you require, you should upgrade to SQL Server 6.0.

STATUS

Microsoft has confirmed this to be a problem in Microsoft SQL Server version 4.21a Service Pack 3 and Service Pack 4 for Alpha platforms. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
