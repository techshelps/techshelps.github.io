BUG: DocErr: Books Online for Mixed Security Has Wrong Graphic

 Last reviewed: May 1, 1997
Article ID: Q137336

The information in this article applies to:
  • Microsoft SQL Server, version 6.0
BUG# NT: 11447 (6.00 DocErr)

SYMPTOMS

In Microsoft SQL Server version 6.0 Books Online, in the "Administrator's Companion," Part 4 - Security, Chapter 9 - Managing Security, Administration Overview, the article 'Setting Up Mixed Security,' shows a replication illustration. This is a documentation error; the graphic should show the mixed security illustration found on page 238 of the hard copy "Administrator's Companion" manual.

WORKAROUND

Refer to page 238 of the printed "Administrator's Companion" manual for the correct graphical reference for setting up mixed security.

STATUS

Microsoft has confirmed this to be a problem in the documentation for SQL Server version 6.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
