INF: Exceptions in SQL Server to ANSI X3.135-1986 Level 1

 Last reviewed: April 25, 1997
Article ID: Q64605

The information in this article applies to: 

  - Microsoft SQL Server version 4.2 for OS/2

Microsoft SQL Server meets the ANSI X3.135-1986 standard with several minor exceptions. For more information, see the SQL Server Technical Note entitled "Discussion of the ANSI SQL Standard and MS SQL Server" (Microsoft publication 098-20421).
