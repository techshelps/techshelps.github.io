INF: Size Considerations for Database and Log Devices

 Last reviewed: April 25, 1997
Article ID: Q60053

The information in this article applies to: 

  - Microsoft SQL Server version 4.2 for OS/2

The maximum size of a device assigned to a database or a log is limited to disk space available and any limitation placed by the operating system on file sizes (because for the operating system, devices are files). There is a minimum size limit of 2 MB for the device that will be used for a database, and a minimum of 1 MB for the size of a device that will be used for a log.
