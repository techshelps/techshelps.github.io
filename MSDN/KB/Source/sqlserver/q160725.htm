INF: How to Obtain Embedded SQL for C Version 6.5 Patch

 Last reviewed: November 7, 1997
Article ID: Q160725
The information in this article applies to:
  • Microsoft Embedded SQL for C, version 6.5

This article describes how to obtain the Microsoft Embedded SQL for C Server version 6.5 patch. This patch corrects problems with the Microsoft Visual C/C++ 4.2 compiler and DBCS when combining COBOL and C applications.

FILES AVAILABLE FROM ONLINE SERVICES

You can get the file from the Microsoft Software Library (MSL) in the following online services: 

File ID      Filename           Description
-------      --------           -----------

S20407       Esqlc.exe       Embedded SQL for C patch
  • The Microsoft Download Service 

          Dial (425) 936-6735 to connect to MSDL
      Download <Filename.exe>
  • Anonymous FTP via the Internet 

          ftp ftp.microsoft.com
      Change to the Softlib\Mslfiles directory
      Get <Filename.exe>
  • The Microsoft Network 

          On the Edit menu, click Go To, and then click Other Location.
      Type "mssupport" (without the quotation marks).
      Double-click the MS Software Library icon.
      Find the appropriate product area.
      Locate and Download <Filename.exe>.
  • Microsoft's World Wide Web Site on the Internet 

          On the www.microsoft.com home page, click the Support icon.
      Click Download a driver, patch, or sample file.
      Select SQL Server from the drop down list box and click GO!
      Scroll through the list of executables to find the file you want.

    The files in the MSL are compressed, self-extracting executable files. After you download the file(s) you need, simply run it to extract the files it contains. The expanded file Esqlc.exe contains the following files: 

       Nsqlprep.exe
   Readme.txt
   Sqlakw32.dll
    Please read the Readme.txt file included with this patch before using it.

    If you need more information or assistance regarding this patch, please contact your primary support provider.
