FIX: SQL Server 6.5 Service Pack 4 Fixlist

 Last reviewed: December 22, 1997
Article ID: Q178225
The information in this article applies to:
  • Microsoft SQL Server, version 6.5 Service Pack 4

The following is a list of fixes and other various improvements that have been made in the Microsoft SQL Server version 6.5 Service Pack 4. For more information, contact your primary support provider.

Please note that workarounds described in these articles have been provided for your information only. It is not necessary to implement these workarounds if you have the updated software.

For more information about each corrected problem in SQL Server version 6.5 Service Pack 4, use the Qxxxxxx number to query for the article in the Microsoft Knowledge Base.
