BUG: Error 4028 Does Not Report the Correct Volume Label

 Last reviewed: April 9, 1997
Article ID: Q158326
The information in this article applies to:
  • Microsoft SQL Server, version 6.5
BUG #: 16146 (6.5)

SYMPTOMS

The tape mount notification error (4028) does not report the correct volume label.

WORKAROUND

Ignore the volume label reported by error 4028.

STATUS

Microsoft has confirmed this to be a problem in Microsoft SQL Server version 6.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

In order to span media, the volume label must be the same. In the case where the volume label does not match, an error 4035 will occur.
