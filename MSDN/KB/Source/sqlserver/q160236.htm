BUG: DB-Library Connections Stop Responding with AfxOleInit

 Last reviewed: April 9, 1997
Article ID: Q160236
The information in this article applies to:
  • Microsoft SQL Server, version 6.5
BUG #: 16067 (Windows: 6.50)

SYMPTOMS

An application that calls AfxOleInit and establishes "make and break" DB-Library connections using the Named Pipe Net-Library from a computer running Windows 95 may stop responding on the call to dbclose().

WORKAROUND

To work around this problem, use a Net-Library other then Named Pipes.

STATUS

Microsoft has confirmed this to be a problem in Microsoft SQL Server version 6.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
