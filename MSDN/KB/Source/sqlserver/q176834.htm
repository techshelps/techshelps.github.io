BUG: DOCERR: Books Online Says That Binding Causes Re-Resolution

 Last reviewed: November 18, 1997
Article ID: Q176834
The information in this article applies to:
  • Microsoft SQL Server versions 6.0 and 6.5
BUG #: 17439 (6.5)

SYMPTOMS

The SQL Server Books Online, Administrator's Companion 6.0, Part 8 Troubleshooting, Chapter 21 Configuration Issues, Managing Stored Procedures, Re-resolution says that the binding and unbinding of defaults and rules should cause re-resolution. However, these operations actually cause recompilation, not re-resolution.

STATUS

Microsoft has confirmed this to be a problem in the Books Online documentation for SQL Server versions 6.0 and 6.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

For more information, see the following article in the Microsoft Knowledge Base:
