PRB: dbopen Error Code Documentation Incorrect

 Last reviewed: April 28, 1997
Article ID: Q95591

The information in this article applies to:
  • Microsoft SQL Server Programmer's Toolkit, version 4.2

SUMMARY

The dbopen documentation on page 167 of the "Programmer's Reference for C" incorrectly lists SQLESOCK and SQLEUHST as valid DB-Library (DB-Lib) error codes.

These error codes do not exist, and will not be returned to the DB-Lib error handler.
