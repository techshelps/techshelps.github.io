BUG: bcp_exec May Cause Unexpected EOF Message

 Last reviewed: April 29, 1997
Article ID: Q113728

The information in this article applies to:
  • Microsoft SQL Server Programmer's Toolkit, version 4.21
BUG# 9967 (4.21)

SYMPTOMS

When copying data into a table with at least one text field, the bcp_exec function may receive the following message from DB-Library (DB-Lib): 

   Unexpected EOF encountered in BCP data-file.
This is the case only if the datafile exceeds 64K and the program is using the Windows version of DB-Library (W3DBLIB.DLL).

WORKAROUND

Use the bcp utility to insert the data.

STATUS

Microsoft has confirmed this to be a problem in DB-Library version 4.21. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
