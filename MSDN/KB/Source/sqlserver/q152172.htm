INF: Microsoft SQL Server ODBC Driver Versions

 Last reviewed: April 8, 1997
Article ID: Q152172
The information in this article applies to:
  • Microsoft SQL Server, versions 4.21, 6.0 and 6.5

SUMMARY

This article discusses the different Microsoft SQL Server ODBC Driver versions, and the versions of SQL Server and operating systems which they support.

MORE INFORMATION

The following list shows which versions of Microsoft SQL Server ODBC Drivers shipped with different versions of Microsoft SQL Server. It also lists the operating system versions under which the drivers are certified to run, and the versions of SQL Server against which they are certified to work.

Driver Version and Date: 1.02.3231, 10/8/93 (Obsolete, see Note) Shipped with: OS/2 SQL Server 4.2B Supported SQL Servers: Microsoft OS/2 SQL Server 1.11 to 4.2B. 

                       Sybase SQL Server 4.0 to 4.9.2
Supported Operating Systems: Windows 3.1, Windows for Workgroups 3.11.

Driver Version and Date: 1.02.3731, 1/26/94 (Obsolete, see Note) Shipped with: SQL Server 4.21 Supported SQL Servers: Microsoft OS/2 SQL Server 1.11 to 4.2B. 

                       Microsoft SQL Server for Windows NT 4.20 to 4.21.
                       Sybase SQL Server 4.0 to 4.9.2
Supported Operating Systems: Windows 3.1, Windows for Workgroups 3.11.

Driver Version and Date: 2.00.1912, 7/14/94 Shipped with: SQL Server 4.21a Supported SQL Servers: Microsoft OS/2 SQL Server 4.2 to 4.2b. 

                       Microsoft SQL Server for Windows NT 4.2 to 4.21a.
Supported Operation Systems: Windows 3.1, Windows for Workgroups 3.11, 
                             Windows NT 3.1, 3.50, 3.51.
Driver Version and Date: 2.50.0121, 6/7/95 Shipped with: SQL Server 6.0 Supported SQL Servers: Microsoft SQL Server 4.21a and 6.0. Supported Operation Systems: Windows 3.1, Windows for Workgroups 3.11, 
                             Windows NT 3.50, 3.51, Windows 95.
Driver Version and Date: 2.50.0126, 8/17/95 Shipped with: SQL Server 6.0 Service Packs 1, 2, and 3 Supported SQL Servers: Microsoft SQL Server 4.21a and 6.0. Supported Operation Systems: Windows 3.1, Windows for Workgroups 3.11, 
                             Windows NT 3.50, 3.51, Windows 95.
Driver Version and Date: 2.65.0201, 4/3/96 Shipped with: SQL Server 6.5 Supported SQL Servers: Microsoft SQL Server 4.21a to 6.5. Supported Operation Systems: Windows 3.1, Windows for Workgroups 3.11, 
        Windows NT 3.50, 3.51, Windows 95.
Driver Version and Date: 2.65.0213, 7/30/96 Shipped with: SQL Server 6.5 Service Pack 1 Supported SQL Servers: Microsoft SQL Server 4.21a to 6.5. Supported Operation Systems: Windows 3.1, Windows for Workgroups 3.11, 
        Windows NT 3.50, 3.51, Windows 95.
Driver Version and Date: 2.65.0240, 12/30/96 Shipped with: SQL Server 6.5 Service Pack 2 Supported SQL Servers: Microsoft SQL Server 4.21a to 6.5. Supported Operation Systems: Windows 3.1, Windows for Workgroups 3.11, 
        Windows NT 3.50, 3.51, Windows 95.
Note: The 1.02.3231 and 1.02.3731 drivers are no longer shipped with any Microsoft products and are considered obsolete. They were ODBC 1.0 drivers, and as such only had Win16 drivers.

Customers who have the SQL Server versions listed above will have the corresponding ODBC driver version in the SQL Server client utilities. The drivers are also shipped with several applications or development languages, such as Visual Basic, Visual C++, and Microsoft Office.

Users can also obtain drivers from ftp.microsoft.com. Go to the developr/ODBC/public directory. The file sqldrv.exe is a self-extracting zip file containing the 2.00.1912 driver; the file sqlsrv32.exe is a self- extracting zip file containing the 2.50.0126 driver. The files ssrv6532.exe and ssrv6516.exe are self-extracting zip files containing the Win32 and Win16 versions of the 2.65.0201 driver, respectively.

The 2.50.0126 SQL Server driver is also in the SQL Server 6.0 Service Packs 2 and 3. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
