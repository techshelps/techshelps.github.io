INF: How to Obtain SQL Server v6.0 Service Pack 3

 Last reviewed: November 7, 1997
Article ID: Q154548
The information in this article applies to:
  • Microsoft SQL Server version 6.0, Service Pack 3

This article describes how to obtain Service Pack 3 for Microsoft SQL Server version 6.0. To see a complete list of bugs that are fixed by this update, see the FIXLIST.TXT file that accompanies the Service Pack.

NOTE: This Service Pack is for use only by holders of a current valid Microsoft SQL Server version 6.0 license. Most customers will not need to apply this Service Pack. This Service Pack addresses some specific problems that were discovered in SQL Server 6.0 after it began shipping. Microsoft recommends that most customers use the shipping version of 6.0 and only apply this Service Pack if you encounter one of the problems it corrects.

SQL Server 6.0 Service Pack 3 is available for Intel (x86), MIPS, and Alpha- based computers.

FILES AVAILABLE FROM ONLINE SERVICES

You can get the files from the Microsoft Software Library (MSL) in the following online services: 

File ID      Filename           Description
-------      --------           -----------

S20203       60SP3ALP.EXE       SP3 executable for Alpha
S20204       60SP3MIP.EXE       SP3 executable for MIPS
S20205       60SP3X86.EXE       SP3 executable for Intel
  • The Microsoft Download Service 

          Dial (425) 936-6735 to connect to MSDL
      Download <FILENAME.EXE>
  • Anonymous FTP via the Internet 

          ftp ftp.microsoft.com
      Change to the SOFTLIB\MSLFILES directory
      Get <FILENAME.EXE>
  • The Microsoft Network 

          On the Edit menu, click Go To, and then click Other Location.
      Type "mssupport" (without the quotation marks).
      Double-click the MS Software Library icon.
      Find the appropriate product area.
      Locate and Download <FILENAME.EXE>.
  • Microsoft's World Wide Web Site on the Internet 

          On the www.microsoft.com home page, click the Support icon.
      Click Download a driver, patch, or sample file.
      Select SQL Server from the drop down list box and click GO!
      Click Service Pack 3.
      Scroll through the list of executables to find the file you want.

    The files in the MSL are compressed, self-extracting executables. After you download the file(s) you need, simply run it to extract the files it contains. Note that the expanded file for each of the platforms contains both the README.TXT file for installation instructions of the Service Pack, and the FIXLIST.TXT file describing all bugs corrected with this patch.

    If you need more information or assistance regarding this Service Pack, please contact your primary support provider.
