INF: Location of Odbcss.h

 Last reviewed: April 15, 1997
Article ID: Q150178

The information in this article applies to:
  • Microsoft SQL Server, version 6.5
  • Microsoft Open Database Connectivity, version 2.65

SUMMARY

The SQL Server 6.5 "Books Online" section "Programming ODBC for SQL Server" mentions the header file Odbcss.h, which contains the #defines for the Microsoft SQL Server ODBC Driver's driver-specific options. This file is not found on the SQL Server 6.5 compact disc.

MORE INFORMATION

The file odbcss.h is only distributed in two locations. It is in the PTK directory of the SQL Workstation compact disc. It is also in SQL Server 6.5, Service Pack 2.
