INF: SQLDescribeParam Not Supported in Queries with Union Clause

 Last reviewed: March 8, 1997
Article ID: Q164424
The information in this article applies to:
  • Microsoft SQL Server, versions 6.0 and 6.5
  • Microsoft Open Database Connectivity, versions 2.5 and 3.0

The SQL Server driver does not support the use of SQLDescribeParam for queries with a union clause. The SQL Server driver documentation will be updated to include this information

Additional query words: ODBC parameter markers
