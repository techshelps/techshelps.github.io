INF: How to Obtain SQL Server 6.5 Service Pack 1

 Last reviewed: September 5, 1997
Article ID: Q154549
The information in this article applies to:
  • Microsoft SQL Server version 6.5, Service Pack 1

This article describes how to obtain Service Pack 1 for Microsoft SQL Server version 6.5. To see a complete list of bugs that are fixed by this update, see the FIXLIST.TXT file that accompanies the Service Pack.

NOTE: This Service Pack is for use only by holders of a current valid Microsoft SQL Server version 6.5 license. Most customers will not need to apply this Service Pack. This Service Pack addresses some specific problems that were discovered in SQL Server 6.5 after it began shipping. Microsoft recommends that most customers use the shipping version of 6.5 and only apply this Service Pack if you encounter one of the problems it corrects.

SQL Server 6.5 Service Pack 1 is available for Intel (x86), MIPS, Power PC (PPC) and Alpha-based computers.

FILES AVAILABLE FROM ONLINE SERVICES

The following file(s) are available for download:

65SP1X86.EXE (For Intel-based computers) 65SP1PPC.EXE (For PPC-based computers) 65SP1ALP.EXE (For Alpha-based computers) 65SP1MIP.EXE (For Mips-based computers)

These files can be found at the following Internet Location:

ftp://ftp.microsoft.com/bussys/sql/public/fixes/usa/SQL65/Sp1/

To extract the contents of the self-extracting .exe file after you download it, type the following from the command prompt or the Run command: 

      <path\filename> -d
where <path\filename> is the location and name of the downloaded file. For example, if you downloaded Sample.exe to C:\Download, type the following at the Run command: 

     c:\download\sample -d
If you need more information or assistance regarding this Service Pack, please contact your primary support provider.
