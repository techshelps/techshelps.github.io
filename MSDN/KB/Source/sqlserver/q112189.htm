INF: Setting Packet Size Using VBSQL

 Last reviewed: April 29, 1997
Article ID: Q112189

The information in this article applies to:
  • Microsoft SQL Server Programmer's Toolkit, version 4.2

SUMMARY

DB-Library (DB-Lib) version 4.20.50 introduced two new functions (DBSETLPACKET and dbgetpacket) that allowed a DB-Lib application to attempt to increase the packet size used to communicate with a remote server. By increasing packet size, it is sometimes possible to improve throughput of large BCPs and text/image data transfers. Version 4.21 of VBSQL.VBX introduces two new functions that can be used to set the packet size in Visual Basic-SQL applications.

MORE INFORMATION

Detailed information on these new functions is contained in the 4.21 PTK release notes
