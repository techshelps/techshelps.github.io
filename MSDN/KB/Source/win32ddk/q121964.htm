Unidrivers: Convert Win 3.1 .RC Files to Win NT .RC Files

 Last reviewed: September 18, 1995
Article ID: Q121964
The information in this article applies to:

- Microsoft Windows NT DDK version 3.10

In the Win32 Subsystem Driver Design Guide, driver developers are instructed to use a program called NTGPC to create the Windows NT .GPC extensions.

See the UNITOOL.HLP file for details on converting Windows version 3.1 RC files to Windows NT .RC files.
