Sample Net Print Provider Not Installed with Printer Samples

 Last reviewed: December 12, 1995
Article ID: Q140659
The information in this article applies to:
  • Microsoft Win32 Device Development Kit (DDK) for Windows 95

SUMMARY

The Windows 95 DDK contains a sample net print provider, but it is not installed with the printer samples.

MORE INFORMATION

If you want to install the sample net print provider, the net samples have to be installed. To do this, run the Windows 95 DDK's Setup program. Select the Network samples check box and press ENTER. The sample net print provider will now be installed in \Ddk\Net\Samples\Prntprov.

Note that the sample is not buildable as is; the sample is missing a makefile. To build the sample, you need to create your own makefile.
