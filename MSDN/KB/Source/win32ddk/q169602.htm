BUG: Installing a Plug and Play Network Card

 Last reviewed: July 17, 1997
Article ID: Q169602
The information in this article applies to:
  • Microsoft Win32 Device Development Kit (DDK) for Windows 95, version 4.0 on the following platform: x86

SYMPTOMS

When you install plug and play network cards on a machine with Windows 95 Service Pack 2, you may encounter an error where the operating system will prompt you to input the complete path for the file it is trying to copy.

RESOLUTION

On encountering this error, you will have to provide the correct path to the file the operating system is looking for, and then the installation will proceed uninterrupted.

STATUS

Microsoft has confirmed this to be a bug in the Microsoft products listed at the beginning of this article. We are researching this bug and will post new information here in the Microsoft Knowledge Base as it becomes available. 

Version             : 4.0
Platform            : WINDOWS
Hardware            : x86
Issue type          : kbbug
