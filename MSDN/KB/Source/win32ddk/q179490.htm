FILE: BusMast.exe Handles DMA on Bus Master Device Using WinNT

 Last reviewed: February 4, 1998
Article ID: Q179490
The information in this article applies to:
  • Microsoft Win32 Device Development Kit (DDK) for Windows NT, versions 3.1, 3.5, 3.51, 4.0

SUMMARY

The BusMast.exe file contains base code that demonstrates how to use operating system functions in Windows NT to handle DMA on a bus master device. The individual vendors who use this sample as a base for their driver development should complete the device specific code. Every effort has been made to ensure the code is correct. However, since there is no device on which to test this driver, it has not been debugged.
