PARSEPJL.OBJ Source Code Not Available

 Last reviewed: June 11, 1996
Article ID: Q151886
The information in this article applies to:

The information in this article applies to:

  • Microsoft Win32 Device Development Kit (DDK) for Windows 95, version 4.0

SUMMARY

Source code for PARSEPJL.OBJ is not available from Microsoft.

MORE INFORMATION

The Win95 DDK contains a print subsystem sample called PJLMON. PJLMON is a sample printer language monitor for bi-directional HP Laserjet printers. A language monitor is responsible for decoding device specific printer language into commands that Win95 can understand.

PARSEPJL.OBJ contains proprietary code and is not available for public use.
