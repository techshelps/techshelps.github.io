SAMPLE: Statest.exe Reads Status of Parallel Port

 Last reviewed: March 6, 1998
Article ID: Q182146
The information in this article applies to:
  • Microsoft Win32 Device Driver Kit (DDK) for Windows NT, versions 3.51, 4.0 on the following platform: 

        - x86

SUMMARY

StatTest.exe is a simple console test application that reads the status of the parallel port through the Parstat sample driver with the Parstat.ini install file.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library:

~ Statest.exe
