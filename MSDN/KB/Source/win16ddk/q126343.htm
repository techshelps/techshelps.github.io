Windows 3.1x Print Manager Not Designed to Be Replaced

 Last reviewed: February 23, 1995
Article ID: Q126343
The information in this article applies to:
  • Microsoft Windows Device Drvier Development Kit (DDK) for Windows version 3.1

The Windows version 3.1x Print Manager has no extensible documented interface. It was not designed to be replaced. The only documented interface is the WM_SPOOLERSTATUS message, which gives limited information concerning print jobs. WM_SPOOLERSTATUS is documented in the Windows version 3.1 Software Development Kit (SDK).

The Windows version 3.1x printing subsystem was not designed to have replacement print spoolers. The Windows version 3.1x Print Manager uses code that relies on GDI internals that are not documented and are proprietary. GDI also depends on code in the Windows version 3.1x Print Manager, which is also undocumented and proprietary.
