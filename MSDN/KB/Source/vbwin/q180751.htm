INFO: Error Messages Shared Between VBCE and VBScript

 Last reviewed: February 10, 1998
Article ID: Q180751
The information in this article applies to:
  • Windows CE Toolkit for Visual Basic 5.0, version 1.0

SUMMARY

The Windows CE Toolkit for Visual Basic 5.0 (VBCE) and Visual Basic Script (VBScript) share the following list of error codes and descriptions. When error handling is turned on in VBCE, the Source property of the Err object will return "VBScript runtime error" when encountering one of the errors listed below.

MORE INFORMATION

The error codes and messages are as follows:
