SUMMARY

The OLE Registration Manager (REGMAN.EXE) is a utility application designed to make it easy to modify OLE-related information in the Windows registration database (REG.DAT) or the Windows NT registry.

MORE INFORMATION

The registration database is a central location for OLE-related information. However, the standard registration editor (REGEDIT.EXE) was not designed specifically for OLE. REGMAN.EXE is a Windows utility that assists in the management of the registration database (REG.DAT). Unlike other registration management tools and editors, REGMAN.EXE focuses solely on OLE information stored in the registration database. With the advent of OLE 2.0, the amount and location of registration information is becoming increasingly complex. REGMAN offers a view of this information in a manner consistent with how an OLE container application would display it.

REGMAN.EXE is provided with an accompanying Help file, REGMAN.HLP. REGMAN.HLP discusses the basics of the OLE registration database and how OLE registry information is stored and used.
