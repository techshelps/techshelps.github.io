BUG: VB 4.0 Professional Edition Does Not Reinstall Help Files

 Last reviewed: May 14, 1996
Article ID: Q150701
The information in this article applies to:
  • Professional Edition of Microsoft Visual Basic, 16-bit and 32-bit, for Windows, version 4.0

SYMPTOMS

When Visual Basic is installed without the Help files, it is not possible to install the Help files of the Professional Edition of Visual Basic by running Setup.

STATUS

Microsoft has confirmed this to be an issue in the Microsoft products listed at the beginning of this article. Microsoft is researching this issue and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Uninstall and then reinstall Visual Basic with the Help files.

MORE INFORMATION

Steps to Reproduce Behavior

  1. Install the 16-bit or 32-bit Professional Edition of Visual Basic 4.0, using the Custom Install option. Click No when prompted to copy the Help files to your hard disk.

  2. Run Setup again to install a component of Visual Basic, and click Yes when prompted to copy the Help files to your hard disk.

  3. After installation, note that the Help files are not copied to your hard disk.
