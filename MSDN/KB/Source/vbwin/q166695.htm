PRB: Where is RegClean?

 Last reviewed: March 17, 1998
Article ID: Q166695
The information in this article applies to:
  • Microsoft Visual Basic Professional and Enterprise Editions for Windows, version 5.0 on the following platforms: - Windows NT - Windows 95

SUMMARY

The Windows Registry Analysis and Correction Utility, called RegClean, cannot be found on the Visual Basic 5.0 CD-ROM.

MORE INFORMATION

The RegClean utility was removed from the Visual Basic product. It is now being distributed as a downloadable utility.
