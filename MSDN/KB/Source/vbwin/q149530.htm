PRB: 16-Bit Projects Cannot Be Saved to a Novell File Server

 Last reviewed: May 22, 1996
Article ID: Q149530
The information in this article applies to:
  • Professional and Enterprise Editions of Microsoft Visual Basic, 16-bit only, for Windows, version 4.0

SYMPTOMS

In a system running under Windows for Workgroups connected to a Novell file server, saving a 16-bit Visual Basic project or project files results in the message, "Path not found."

CAUSE

The Novell file client is running outdated or mismatched NetWare drivers.

RESOLUTION

Obtain the latest NetWare client drivers from Novell and install these updated client drivers on your file server. The updated client file names are VLMUPKT*.EXE, where * is 1 to 5.

MORE INFORMATION

The following platform combinations of NetWare client software may produce the errors listed in the Symptoms section. If your current platform and NetWare client combination are among those listed, upgrade your NetWare client to resolve the problem. 

Platform                      NetWare Client


Windows for Workgroups        Novell NetWare Client Kit for MS-
                              DOS\Microsoft Windows,(NETX Driver Version)
                              any version

Windows for Workgroups        Novell NetWare Client Kit for MS-
                              DOS\Microsoft Windows,(VLM Driver Version)
                              any version prior to version 1.2a

Windows 95                    Novell NetWare Client Kit for MS-
                              DOS\Microsoft Windows, (NETX Driver Version)
                              any version

Windows 95                    Novell NetWare Client Kit for MS-
                              DOS\Microsoft Windows, (VLM Driver Version)
                              any version prior to version 1.2a

Software Combinations That Function Correctly

The following combinations of platform and NetWare client software may function correctly. If your current platform and NetWare client software combination are not among these you should contact your network software provider and ask for the latest drivers. 

Platform                      Network Client


Windows for Workgroups        Novell NetWare Client Kit for MS-
                              DOS\Microsoft Windows, (VLM Driver Version)
                              version 1.2a or higher

Microsoft Client for          Novell NetWare Client Kit for MS-
                              DOS\Microsoft Windows, NetWare Networks
                              (VLM Driver Version) version 1.2 or higher

Microsoft Client Service      N/A
   for NetWare

Novell NetWare Client         Novell NetWare Client Kit for MS-
                              DOS\Microsoft Windows for Windows NT
                             (VLM Driver Version) version 3.5b
Contact Novell to receive the latest version NetWare client software.
