INFO: Visual Basic 5.0 Fixes in Visual Studio 97 Service Pack 1

 Last reviewed: December 18, 1997
Article ID: Q168800
The information in this article applies to:
  • Microsoft Visual Basic Control Creation, Learning, Professional, and Enterprise Editions for Windows, version 5.0
  • Microsoft Visual Studio 97

SUMMARY

Microsoft Visual Studio 97 Service Pack 1 contains several fixes for Visual Basic 5.0. This article lists all of the Microsoft Knowledge Base articles documenting bug fixes that affect Visual Basic 5.0.

MORE INFORMATION

The following list contains the updated components and the bugs that were fixed for Visual Basic 5.0 in the Visual Studio 97 Service Pack 1. You can find all these articles quickly in the Microsoft Knowledge Base by searching on VB5FixlistSP1. If you are only interested in a specific article below, you can find it in the Microsoft Knowledge Base by searching on the article ID (for instance, Q167123.)
