Fixlist for Visual Basic 4.0a for Windows as of April 23, 1996

 Last reviewed: February 3, 1997
Article ID: Q150423
The information in this article applies to:
  • Standard, Professional, and Enterprise Editions of Microsoft Visual Basic, 16-bit and 32-bit, for Windows, version 4.0a

SUMMARY

Visual Basic 4.0a is a synchronization release of Visual Basic 4.0. that synchronizes the components shared between Microsoft Access For Windows 95 and Visual Basic 4.0. These components include the language engine (Visual Basic For Applications), the Jet database engine, and the Data Access Objects (DAO). This release also upgrades the following components: the data aware grid for both the 16-bit and 32-bit editions, the Remote Data Objects and Remote Data Control for the 32-bit only edition, and the Windows 95 controls.

To obtain this release, please call 1-800-360-7561. Although this release is free to all registered users, there is a $5 charge to cover shipping and handling.

The synchronized components contain bug fixes, which are outlined in the following table. This is not a comprehensive list and is current as of April 23, 1996. For more information about each issue, query in the Microsoft Knowledge Base on the article identifier (ID) number.

These articles can be faxed to you from Microsoft FastTips if you call (800) 963-4300 and follow the prompts. When prompted to enter the Item ID, enter the article ID number without the "Q".
