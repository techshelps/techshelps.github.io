INFO: CAB Files Distributed with Visual Basic 5.0

 Last reviewed: August 22, 1997
Article ID: Q172991
The information in this article applies to:
  • Microsoft Visual Basic Control Creation, Learning, Professional, and Enterprise Editions for Windows, version 5.0

SUMMARY

Microsoft provides digitally-signed cabinets (.CAB files) containing core Visual Basic 5.0 files for download on the Internet. To download these CAB files, type "http://activex.microsoft.com/controls/vb5/xxx.cab" where xxx.cab is the name of the CAB file you want to download.
