HOWTO: VB3: Search for Visual Basic Articles Using KBKeywords

 Last reviewed: March 10, 1998
Article ID: Q108753
The information in this article applies to:

- Microsoft Visual Basic for Windows, versions 2.0, 3.0

SUMMARY

The Microsoft Knowledge Base is categorized by using keywords. This article lists the keywords specific to articles in the Microsoft Visual Basic for Windows collection.

MORE INFORMATION

Microsoft Visual Basic Major and Minor KBKeywords

Each Knowledge Base article in the Visual Basic collection contains at least one product-specific keyword (called a KBKeyword) that places the article in an appropriate Visual Basic category. Each KBKeyword is composed of the concatenation of a major topic keyword and a minor topic keyword. For example, you can find all the general database programming articles by using the APrgDataOther keyword to query the Microsoft Knowledge Base.

Use the asterisk (*) wildcard to find articles that fall into the general categories or into an intermediate subcategory. For example, to find all the articles that apply to Visual Basic Forms and Controls regardless of whether they are standard, custom, or third-party controls, use PrgCtrls* as the keyword to query the Microsoft Knowledge Base. To find all advanced programming articles, query on APrg*. An article usually has only one KBKeyword, but it may have more.

Product-Specific Keywords

You can use the KBKeywords to organize Visual Basic articles or to search for specific groups of Visual Basic articles. For information about KBKeywords for other Microsoft developer products, please query the Microsoft Knowledge Base using these keywords: 

   dskbguide and kbkeyword
KB-wide Keywords

Each article in the Visual Basic collection also contains at least one generic, KB-wide keyword. The KB-wide keywords are standard throughout the Microsoft Knowledge Base, appearing in all Knowledge Base articles, regardless of product. You can use the KB-wide keywords to organize all Knowledge Base articles or to search for articles across several Microsoft products. For more information about these KB-wide keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles
