Authentication Requested by Proxy for Some Requests

 Last reviewed: December 30, 1997
Article ID: Q177502

The information in this article applies to:

  • Microsoft Proxy Server versions 1.0 and 2.0

SYMPTOMS

When you access certain Internet sites, your web browser may display a logon dialog box. If you click OK or Cancel repeatedly, you should be able to display the web page.

CAUSE

You may have insufficient directory permissions on the Proxy Server. This problem usually occurs when you access sites returning cookies.

WORKAROND

Configure all proxy users with Read/Execute access or Full Control access to the following directories and their subdirectories: 

   \Inetpub\Scripts\Proxy
   \Urlcache
Also, configure the System account with Full Control access to all of the directories.

MORE INFORMATION

Please refer to the online Proxy Server documentation for information on how to use Password Authentication with Proxy server.
