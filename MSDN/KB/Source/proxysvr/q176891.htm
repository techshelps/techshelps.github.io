Can Not Connect to Oracle 7.3 Server via Proxy Server 2.0

 Last reviewed: December 17, 1997
Article ID: Q176891

The information in this article applies to:

  • Microsoft Proxy Server version 2.0

SUMMARY

The Server Proxy service of Proxy Server version 2.0 can not be configured to allow remote SQL*NET clients to connect to an Oracle Server 7.3 running on Microsoft Windows NT Server.

MORE INFORMATION

To use the Server Proxy service in Proxy Server 2.0, the TCP port needed to establish communications between the server and client must be remotely bound to the network adapter on the Internet side of the Proxy Server.

The TCP port used by the SQL*NET client to connect to the Oracle Server database is dynamically configured at the time the client connects.

Because the Oracle server and the SQL*NET client communicate over dynamically configured TCP ports, it is not possible to connect the two through the Proxy Server.
