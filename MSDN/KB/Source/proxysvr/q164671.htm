Winsock Proxy Service Receives Access Violation

 Last reviewed: March 6, 1997
Article ID: Q164671
The information in this article applies to:
  • Microsoft Proxy Server, version 1.0

SYMPTOMS

The Winsock Proxy Service receives the following Dr. Watson error when used in an IPX environment: 

   WSPSRV.EXE
   Exception: access violation (0x0000005), Address: 0x776916ab

CAUSE

There was a problem with the Msafd.dll that caused this error to occur when Winsock was accessed in an IPX environment.

RESOLUTION

To resolve this problem, apply Windows NT 4.0 Service Pack 2, which contains the fix for this problem.

MORE INFORMATION

Microsoft has confirmed this to be a problem with Microsoft Proxy Server version 1.0. This problem was corrected in the latest U.S. Service Pack for Microsoft Windows NT version 4.0 .For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces):

S E R V P A C K
