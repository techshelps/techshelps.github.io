Your Credentials Were Not Authenticated Error with Chkwsp32

 Last reviewed: October 6, 1997
Article ID: Q170853
The information in this article applies to:
  • Microsoft Proxy Server versions 1.0 and 2.0

SYMPTOMS

After you install the Winsock proxy client and run Chkwsp32, you will get the following error message: 

   Your credentials were not authenticated.
   Client control protocol version MATCHES the server control protocol.

CAUSE

The above error indicates the proxy clients do not have appropriate rights to the Proxy Server Msp\Clients share.

WORKAROUND

Configure the Proxy Server so that the clients have appropriate rights to the Proxy Server Msp\Clients share, including both share and NTFS permissions.

Run Chkwsp32 again to verify client systems have appropriate rights. If the rights are correct, the above error message should not appear.
