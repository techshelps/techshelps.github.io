Server Error 500: System Cannot Find the Path Specified

 Last reviewed: December 17, 1997
Article ID: Q164194
The information in this article applies to:
  • Microsoft Proxy Server, versions 1.0 and 2.0

SYMPTOMS

When users are going through a proxy server that also hosts an Internet Information Server (IIS) with virtual servers, the proxy client may receive the following error: 

   HTTP 500 Server Error "System cannot find the specified path"

CAUSE

This error occurs if the WWW service has all HOME directories associated with virtual servers.

WORKAROUND

To work around this problem, use the following steps to create a new HOME directory that is not associated with a virtual server:

  1. Open the Internet Service Manager.

  2. Go to the properties for the WWW service.

  3. Click the Directories tab.

  4. Click Add.

  5. Point to a physical directory that you want to use.

  6. Make this a HOME Directory.

  7. Do not add it to a virtual server.

  8. Apply the changes and restart the WWW service.
