Err. Msg: System Is Being Monitored Remotely by Perfmon

 Last reviewed: December 30, 1997
Article ID: Q164142

The information in this article applies to:

  • Microsoft Proxy Server version 1.0

SYMPTOMS

If you set up or remove either Microsoft Proxy Server 1.0 or the evaluation version of Microsoft Proxy Server, you may get the following error message: 

   Setup has detected the system is being monitored remotely by
   Performance Monitor.

CAUSE

This is an issue with the Setup program and how it handles some DLL files associated with Performance Monitor.

WORKAROUND

  1. Change to the \%winnt%\system32 directory.

  2. Rename the Wspperf.dll file to Tmp1 and rename the Wspmon.dll file to Tmp2.

  3. Run the Setup program.

  4. Verify that Performance Monitor is still running correctly.

  5. Delete Tmp1 and Tmp2.
