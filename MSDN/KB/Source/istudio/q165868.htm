BUG: Cannot See FoxPro Remote Views in Data Tools Data View

 Last reviewed: December 9, 1997
Article ID: Q165868
The information in this article applies to:
  • Microsoft Visual InterDev, version 1.0

SYMPTOMS

When connecting a FoxPro database to the Data View, you won’t see any Remote Views in the Views folder.

CAUSE

The FoxPro version of the ODBC driver specifically excludes Remote Views from being identified.

STATUS

This behavior is a limitation of the ODBC driver and is by design.
