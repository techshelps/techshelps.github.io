PRB: Crash Dropping SYBASE Data Connection

 Last reviewed: December 9, 1997
Article ID: Q165835
The information in this article applies to:
  • Microsoft Visual InterDev, version 1.0

SYMPTOMS

When removing a data connection to SYBASE using the Intersolv ODBC driver, Visual InterDev will crash either by disappearing or with a fault in MSDEV.EXE.

CAUSE

This appears to be a problem with the Intersolv ODBC driver for Sybase versions 2.12 and 3.00.

RESOLUTION

There is no known workaround for this problem at this time.

STATUS

The Intersolv ODBC driver for Sybase versions 2.12 and 3.0 is manufactured by Sybase, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
