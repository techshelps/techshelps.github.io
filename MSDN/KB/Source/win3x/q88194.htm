Can't Create Swap Files on SYDOS Removable Hard Drives

 Last reviewed: November 23, 1994
Article ID: Q88194
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

According to Syquest technical support, its SYDOS removable hard drives do not allow Microsoft Windows version 3.1 to create either temporary or permanent swap files on the drives. The drive letter(s) for the SYDOS drives are not available in the Virtual Memory dialog box.

WORKAROUND

If you have a standard non-removable hard drive in addition to the SYDOS drive, you can create a swap file on that hard drive.

MORE INFORMATION

Syquest technical support has stated that it has no plans to correct this limitation. Please contact Syquest for more information.
