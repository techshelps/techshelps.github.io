Windows 3.1 and LAN Manager 2.1 NetBind Hangs

 Last reviewed: November 30, 1994
Article ID: Q82405
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

If you use Microsoft LAN Manager version 2.1 and NetBind is not started before Microsoft Windows, your computer may stop responding (hang or lock up).

NOTE: This problem may occur under a variety of situations and not just Windows.

WORKAROUND

To work around this problem, run NetBind before you start Windows.

MORE INFORMATION

NetBind is a utility that comes with LAN Manager version 2.1.
