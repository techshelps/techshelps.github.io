Windows 3.1 Top 10 Questions and Answers

 Last reviewed: July 29, 1996
Article ID: Q85388
This information applies to:
  • Microsoft Windows operating system version 3.1

The Microsoft Software Library contains a document with answers to the ten most frequently asked customer questions about using Windows. These questions and answers cover common setup, printing, applications, network, and usage issues.

The document is available in the Microsoft Software Library in ASCII text, Microsoft Write, and Microsoft Word for Windows formats. The document can be found by searching on the word WIN31QA2, the Q number of this article, or S13543. WIN31QA2 was archived using the PKWARE file compression utility.
