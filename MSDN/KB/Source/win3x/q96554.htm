WFWG: Microsoft Diagnostics (MSD) May Fail in Standard Mode

 Last reviewed: November 23, 1994
Article ID: Q96554
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.1

SYMPTOMS

If you run the Microsoft Diagnostics command (MSD) in standard mode of Windows for Workgroups version 3.1, and you select any option from the menu bar more than one time, MSD stops running and blinking characters appear in the top-left corner of the MS-DOS-session screen.

WORKAROUND

There is currently no workaround for this problem exists. To return to Windows, type "exit" (without the quotation marks) at the command prompt and then press ENTER.

STATUS

Microsoft has confirmed this to be a problem in Windows for Workgroups version 3.1.
