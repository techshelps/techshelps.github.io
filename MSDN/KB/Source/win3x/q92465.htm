APM.EXE Needed for APM With Twinhead 386SL Notebook

 Last reviewed: November 22, 1994
Article ID: Q92465
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

If "Intel 386SL Based System with APM" is the system type in Windows 3.1 on a Twinhead 386SL notebook, the Advanced Power Management (APM) icon may not appear in Control Panel unless APM.EXE is loaded in the CONFIG.SYS file.

MORE INFORMATION

According to Twinhead technical support, APM.EXE is required for the Advanced Power Management features to function correctly with Windows 3.1.

The APM.EXE file is available from the Twinhead bulletin board service (BBS) or Twinhead technical support.

For more information on installing Advanced Power Management for Windows 3.1, please see section 5.0 "Using Advanced Power Management (APM)" of the Windows 3.1 README.WRI.
