WRK Shows Incorrect Network Setting Dialog Box

 Last reviewed: November 23, 1994
Article ID: Q94376
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.1

The "Microsoft Windows for Workgroups Resource Kit" guide for Microsoft Windows for Workgroups (WFWG) version 3.1 incorrectly shows an Auditing button in Figure 9.2 "Network Setting Option From Control Panel - 386 Enhanced Mode " on page 9-17. Auditing capability is not provided in Windows for Workgroups version 3.1.

The correct button options in this dialog box are: Adapters, Logon, Networks, and Password.
