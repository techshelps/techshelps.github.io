Novell Printer Dialog Not Fully Repainted on EGA

 Last reviewed: November 21, 1994
Article ID: Q84156
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

When you Windows version 3.1 with Novell NetWare on an EGA display, some of the network specific dialog boxes are not repainted properly.

There is no solution to this problem.
