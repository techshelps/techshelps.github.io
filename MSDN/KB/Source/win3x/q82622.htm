DoDot Uses More System Resources with Windows 3.1

 Last reviewed: November 23, 1994
Article ID: Q82622
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

When the Halcyon Software DoDot program (a graphics image converter) is run in Microsoft Windows version 3.1, system resources in the Help/About dialog box drop significantly as compared to Windows 3.0.

When you quite the application the memory is recovered.

CAUSE

This problem is caused by the DoDot program. Halcyon Software is aware of the problem and is working on an upgrade.

RESOLUTION

Contact Halcyon Software for upgrade availability information.

MORE INFORMATION

Contact Halcyon Software for more information.
