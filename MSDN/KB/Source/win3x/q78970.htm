        Questions and Answers: Windows 3.0 Network Installation
        -------------------------------------------------------

1. Q. I want to set up a shared version of Windows on my network. How
      do I do this?

   A. The step-by-step procedure for installing Windows on a network
      is on pages 553 and 554 of the "Microsoft Windows User's Guide"
      for version 3.0. In addition, the following information may be
      useful to you:

      a. If the files from the Windows disks expand and install
         correctly on the hard disk and if they are dated 10/31/90,
         the DOS directory (DIR) command will show 283 files using 9.6
         megabytes (MB).

      b. On any network, these files must be set as read-only.

      c. On a Novell network, be sure to set rights correctly for both
         Windows directories. For the shared directory where the
         expanded files are stored, the search, open, and read rights
         need to be turned on. For the user's directory, which is
         chosen while running the Setup program with the forward slash
         N option (SETUP /N), all rights must be on, except for
         parent, which is optional.

2. Q. When I try Windows Setup from my network, it copies a few files
      and then gives me an error or locks up. What can I do?

   A. With certain network cards, especially ARC NET cards, the
      interrupt setting or adapter segment setting may cause problems
      for Windows Setup. The following network card settings work best
      with Windows.

      a. Set IRQ on the network card for an Interrupt from 3 to 8

      b. If your card uses memory areas in the adapter segment, DO NOT
         use location D000.

      If you are on a Novell network, the following rights need to be
      set for both Windows directories.

      a. For the shared directory where the expanded files are stored,
         the search, open, and read rights need to be turned on.

      b. For the user's directory, which is chosen while running the
         Setup program with the forward slash N option (SETUP /N), all
         rights must be set to on, except for parent, which is
         optional.
