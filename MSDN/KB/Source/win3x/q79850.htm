Buffer Lines Setting in Terminal Limited to 25-244 Lines

 Last reviewed: November 23, 1994
Article ID: Q79850
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

In Microsoft Windows Terminal, when the Columns option is set to 132, Terminal allows Buffer Lines to be set at any number between 0-399; however, it resets any value under 25 to 25, and any value over 244 to 244.

NOTE: If column option is set to 80, then the maximum buffers value is 399.
