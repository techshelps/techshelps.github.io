Text Does Not Appear with Picture Publisher 3.0

 Last reviewed: November 23, 1994
Article ID: Q88737
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

When you use Micrografx's Picture Publisher version 3.0 with Microsoft Windows 3.1, text may not appear when it is typed. This problem only occurs when a 256-color video driver is installed.

RESOLUTION

To correct this problem, either install a 16-color driver or upgrade to Picture Publisher version 3.1.

MORE INFORMATION

For more information, contact Micrografx.
