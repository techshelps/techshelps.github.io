WordPerfect's File Manager Incorrectly Maps Fonts

 Last reviewed: November 21, 1994
Article ID: Q85471
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

If you run version 1.0 of WordPerfect for Windows from Microsoft Windows operating system version 3.1, the WordPerfect's File Manager version 1.0 (date 11/4/91) will not correctly map display fonts.

For example, if you choose Arial as the font to display, Modern is displayed.

MORE INFORMATION

There currently is no workaround to this problem. WordPerfect is working on an upgrade to its existing release that may address this and other problems. Contact WordPerfect corporation for more information.

For more information on WordPerfect's File Manager, query on the following words: 

        wordperfect and gp and fault
