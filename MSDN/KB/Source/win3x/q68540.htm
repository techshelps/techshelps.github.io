Installing Windows Printer Description Files from the SDL

 Last reviewed: November 23, 1994
Article ID: Q68540
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

The Supplemental Driver Library (SDL) contains Windows Printer Description (WPD) files that require the Windows 3.0 PostScript Driver (PSCRIPT.DRV) for installation.

MORE INFORMATION

Use the following steps to install any of the WPD files listed at the bottom of this article:

Installation Procedure For 5.25 inch SDL disk set

  1. Install the PSCRIPT.DRV printer driver from your SDL Disk 4.

  2. Select PostScript Printer from the Installed Printers list box.

  3. Choose Configure.

  4. Choose Setup. (The PostScript Printer dialog box appears.)

  5. Choose Add Printer.

  6. Type the name of the drive that contains Disk 4 of the Supplemental Driver Library. The Control Panel displays a list of additional PostScript printers that can be installed.

  7. Select the printer from the Add Printer list box.

  8. Select Add. The Control Panel installs the printer.

  9. Select Done. The Printer pull-down box of the PostScript Printer dialog box displays the newly installed printer(s).

  10. Choose OK and proceed with normal printer setup in the Control Panel.
