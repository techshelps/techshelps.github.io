Cannot See Files on Shared CD-ROM Drives with Corel SCSI

 Last reviewed: November 22, 1994
Article ID: Q114477
The information in this article applies to:
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

After connecting to a shared CD-ROM drive from a remote Microsoft Windows for Workgroups workstation, your computer cannot access any files.

CAUSE

Corel Technical Support has confirmed this to be a problem with the Corel version of CD-ROM extensions (CORELCDX.COM) prior to version 2.0.

RESOLUTION

You can work around this problem by using one of the following methods:

  • Upgrade to CORELCDX.COM version 2.0.

    -or-

  • Use MSCDEX.EXE version 2.21 or later.

CORELCDX.COM is manufactured by Corel, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
