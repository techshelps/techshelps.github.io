Doc Err: Location of EXPAND.EXE for Windows 3.1

 Last reviewed: November 21, 1994
Article ID: Q83428
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

EXPAND.EXE can be found on Disk 3 of both the 3.5-inch, 1.44 MB, and the 5.25-inch, 1.2 MB, Windows 3.1 disk sets.

MORE INFORMATION

Chapter 4 of "Getting Started with Microsoft Windows" discusses the process of expanding and copying files from Windows disks to a hard drive. On page 61, the following is written:

To copy Windows files to your system and expand them:

  1. Copy EXPAND.EXE from Windows Disk 2 to your hard disk.

The step should state:

  1. Copy EXPAND.EXE from Windows Disk 3 to your hard disk.
