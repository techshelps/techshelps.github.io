Logitech Mouse Doesn't Work in Generic Cad with Windows 3.0

 Last reviewed: November 23, 1994
Article ID: Q64656
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

The Logitech C7 Serial Mouse does not work in Generic Cad version 3.00 while you are in Microsoft Windows version 3.0. This behavior occurs in standard- and enhanced-mode Windows 3.0. The mouse works correctly when used outside of Windows. This happens even when you use Logitech Mouse Driver version 4.10.

STATUS

Generic Software is researching this problem.

MORE INFORMATION

Contact Generic Software for more information.
