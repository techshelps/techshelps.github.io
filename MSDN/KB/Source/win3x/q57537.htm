Leading Edge Won't Take Mode CO80 Command

 Last reviewed: November 22, 1994
Article ID: Q57537
The information in this article applies to:
  • Microsoft Windows operating system versions 2.x, 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

The Leading Edge D3 using Leading Edge DOS Version 3.30 does not permit the entry of "Mode CO80" at the command line to allow Microsoft Windows to run with a monochrome VGA display.

WORKAROUND

This computer is supplied with software from Leading Edge to reconfigure the graphics adapter to run as a color VGA. The software is supplied on the Utilities Disk. After reconfiguration, the display is still in monochrome and Windows works properly.

This has not been tested by Microsoft. For more information, contact Leading Edge.
