Windows Network Setup User Directory Files

 Last reviewed: February 12, 1997
Article ID: Q95842
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

When you install Microsoft Windows 3.0 or Windows 3.1 from an Administrative, Shared directory on a local area network (LAN) using the Setup /N command, the Setup program prompts you for a drive and path. This creates a user directory on the specified drive with all the needed files.

MORE INFORMATION
