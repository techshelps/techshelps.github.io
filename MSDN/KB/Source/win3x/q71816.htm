AT&T 6286 Prints Corrupted Output in Standard Mode

 Last reviewed: November 22, 1994
Article ID: Q71816
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

The AT&T 60286 with Firmware (BIOS) 1.17 prints corrupted output from all Windows applications when the machine is running in standard mode. Real mode prints normally.

CAUSE

This is due to a problem in the BIOS.

RESOLUTION

An update of the BIOS to version 1.25 (or later) solves the problem.

If this problem occurs on an AT&T 6286, call AT&T product support for the latest BIOS upgrade.
