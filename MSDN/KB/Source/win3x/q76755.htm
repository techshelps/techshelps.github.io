Forest & Trees for Windows

 Last reviewed: November 21, 1994
Article ID: Q76755
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

Forest & Trees by Channel Computing, Inc., is an application that provides read-only access to a variety of database files, spreadsheets, and database servers.

MORE INFORMATION

Forest & Trees allows you to query a wide range of external database systems and use Windows Dynamic Data Exchange (DDE) to return the results to an Excel spreadsheet. Forest & Trees supports access to both local and network-based files.

Forest & Trees includes support for the following environments/databases:

  • AS/400
  • Oracle
  • Paradox
  • SQL Server
  • SQLBase
  • DB2
  • Btrieve
  • Lotus 1-2-3
  • Microsoft Excel
  • DBASE
  • R:BASE
  • DataEase
  • Q&A
  • Novell SQL
  • ASCII flat files

System requirements include an 80286 or higher processor, EGA or higher, 2 MB or more of RAM, Windows 3.0 or later. A mouse is optional.

For additional information on Forest & Trees for Windows, please contact Channel Computing, Inc.
