Underlining Spaces in Microsoft Windows Write

 Last reviewed: December 11, 1997
Article ID: Q25371
1.x 2.x 3.00 3.00a 3.10 3.11 WINDOWS kbusage kbtool

The information in this article applies to:

  • Microsoft Windows operating system versions 1.x, 2.x, 3.0, 3.0a, 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SUMMARY

To underline spaces in Microsoft Windows Write and keep them on the screen after you press ENTER, you must type a character at the end of the underlined spaces, as in the following steps:

  1. Press F8 or choose Underline from the Character menu.

  2. Type the desired number of underlined spaces.

  3. Press SHIFT+-(MINUS SIGN).

  4. Press ENTER.

The underlined spaces remain on the screen.

MORE INFORMATION

You can achieve the same effect by typing SHIFT+- multiple times for the desired number of underlined spaces.
