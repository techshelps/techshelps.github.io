Third-Party Utility That Checks Integrity of TrueType Fonts

 Last reviewed: April 26, 1996
Article ID: Q117702
The information in this article applies to:
  • Microsoft Windows operating system version 3.1
  • Microsoft Word for Windows, versions 6.0, 6.0a
  • Microsoft Word Assistant, version 1.0

Although Windows usually detects incorrectly installed fonts in the Fonts dialog box of Control Panel, it may not detect damaged (corrupted or bad) TrueType fonts.

You can use OKFonts to check the integrity of your TrueType fonts. This third-party utility scans your system for TrueType fonts and checks each character of every font. OKFonts displays each character of every font on the screen, then it prints a sample of each character.
