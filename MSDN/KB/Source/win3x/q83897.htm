Err Msg: Fatal Disk Error (with HP Advanced Link)

 Last reviewed: November 23, 1994
Article ID: Q83897
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

Hewlett-Packard's (HP) Advanced Link (any version) doesn't run in Microsoft Windows version 3.1 in 386 enhanced mode. When you attempt to run in 386 enhanced mode, the following error message is displayed: 

   FATAL DISK ERROR
This problem does not occur with Windows 3.0.

STATUS

HP is aware of this problem and is currently researching it.

MORE INFORMATION

The HP Advanced Link connectivity package allows a workstation to communicate over a standard communications port with a HP 2392 minicomputer.
