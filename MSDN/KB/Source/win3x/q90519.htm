Word Hangs When Linking Bitmap from Different Display Type

 Last reviewed: November 23, 1994
Article ID: Q90519
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.1

SYMPTOMS

A problem may occur when creating a link between two machines using ClipBook Viewer and Microsoft Word. If two machines have different display configurations and a bitmap is shared between them, Word may stop responding (hang).

WORKAROUND

To avoid this problem, paste the bitmap as a device independent image.

MORE INFORMATION

This has happened between two machines having the following displays:

  • 640 by 480, 16 colors (Server)
  • 1024 by 768, 256 colors (Client)
