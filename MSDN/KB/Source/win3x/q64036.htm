Unrecoverable Application Error in Windows Program Manager

 Last reviewed: November 23, 1994
Article ID: Q64036
The information in this article applies to:
  • Microsoft Windows operating system version 3.0, 3.0a

SYMPTOMS

The Microsoft Windows version 3.0 Program Manager reports an Unrecoverable Application Error if you do the following:

  1. Run Windows and open one other application (your choice).

  2. Activate Program Manager.

  3. From the File menu, choose New.

  4. Select the Program Item Radio button, then choose OK.

  5. Enter an invalid file path name in the Command Line.

  6. Select the Change Icon push button of the Program Item Properties dialog box.

  7. Select the View Next push button. You receive a message about no icons in file.

  8. Press ALT+ESC twice or until Program Manager has the focus if more than one other application is running.

STATUS

Microsoft has confirmed this to be a problem in Windows version 3.0. This problem was corrected in later versions of Windows
