AT&T 60386-25 Phoenix BIOS FB12 Needs Update to FB15

 Last reviewed: November 22, 1994
Article ID: Q64850
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

You may experience problems displaying non-Windows applications in Microsoft Windows version 3.0 if your AT&T 60386-25 computer has Phoenix BIOS version FB12.

Upgrading this BIOS to Phoenix BIOS version FB15 seems to correct this problem.

For information on obtaining an upgrade, query on the following words here in the Microsoft Knowledge Base: 

   obtain latest phoenix bios.
