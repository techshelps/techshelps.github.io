Windows 3.0 and Award BIOS for 286/386 Computers

 Last reviewed: November 23, 1994
Article ID: Q64823
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

According to Award, the following versions of the 286/386 BIOS work correctly with Microsoft Windows version 3.0:

  • 3.05 (may not work correctly in all situations)
  • 3.1x (where x refers to machine-specific versions)

MORE INFORMATION

You can upgrade your BIOS by contacting Award Software or a regional Award distributor.
