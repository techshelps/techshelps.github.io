Norton Desktop v1.0 Conflicts with SoundBlaster

 Last reviewed: November 21, 1994
Article ID: Q82365
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

After you install version 1.0 of Norton Desktop for Windows, and you restart Windows, the following error message may be displayed if you also have a Creative Labs SoundBlaster sound card installed: 

   The SoundBlaster 1.5 driver requires the Windows 3.1 version of
   VDMAD.386 to work correctly. An incorrect version is currently in
   use. Change the DEVICE= line in SYSTEM.INI to: DEVICE=*VDMAD, then
   restart your system."

RESOLUTION

Follow the instructions in the message to solve the problem.
