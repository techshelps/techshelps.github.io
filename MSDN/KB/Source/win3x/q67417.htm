PIF Files Created by Windows 3.0 Setup from P to R

 Last reviewed: November 30, 1994
Article ID: Q67417
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

The following are the default parameters for program information files (PIFs) created by the Microsoft Windows version 3.0 Setup program, in alphabetical order from P to R. These defaults may be modified by loading the PIF file into the Windows 3.0 PIF Editor, PIFEDIT.EXE.
