Starting Windows Applications in Full Screen Mode

 Last reviewed: November 23, 1994
Article ID: Q66961
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

There is no way to set the following Microsoft Windows applications to start in a full screen: 

Calculator      Calendar        Cardfile
Clipboard       Clock           Control Panel
File Manager    Notepad         Paintbrush
Recorder        Reversi         Print Manager
Solitaire       PIF Editor      Windows Setup
Write
