Norton Desktop for Windows Has a Repaint Problem

 Last reviewed: November 23, 1994
Article ID: Q82718
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

Norton Desktop for Windows (NDW) may have screen refresh problems browsing for a file to print when you are running MS-DOS Prompt.

WORKAROUND

To work around this problem, close both the print option and MS-DOS Prompt. This problem does not occur when MS-DOS Prompt is not being run.

It may be necessary to exit and restart Windows to prevent this problem from reoccurring, even if MS-DOS Prompt has been closed.
