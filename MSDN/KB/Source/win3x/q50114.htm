Lotus Error Unable to Locate 123.set from PIF file

 Last reviewed: November 23, 1994
Article ID: Q50114
The information in this article applies to:
  • Microsoft Windows operating system versions 2.x, 3.0, 3.0a, 3.1, 3.11

SYMPTOMS

When you attempt to run Lotus 1-2-3 from a PIF file, the following error message is displayed: 

   Unable to locate 123.set

CAUSES

The following can cause this message to occur:

  • You did not specify an initial directory in the PIF file.
  • The directory you specified is incorrect.

RESOLUTION

To correct this problem, set the initial directory in the PIF file to the directory that contains the Lotus 1-2-3 executable files.
