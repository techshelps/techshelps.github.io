MS-DOS Err Msg: Not Ready Reading Drive A

 Last reviewed: November 23, 1994
Article ID: Q85685
The information in this article applies to:
  • Microsoft Windows operating system version 3.1

SYMPTOMS

If you use Sysgen's Omnibridge controller card driver ONMIBRG.SYS version 3.3 or earlier with Windows version 3.1, the following MS-DOS error message is displayed after you type "win" (without the quotation marks) to execute Windows: 

   Not ready reading drive A
   Abort, Retry, Fail

CAUSE

This problem is caused by the Sysgen driver.

RESOLUTION

You can either updated the driver to version 3.5 or later (updates are available from Sysgen), or you can follow this procedure:

  1. Insert a disk in drive A.

  2. Retry several times.

  3. Remark out the OMNIBRG.SYS driver.

MORE INFORMATION

The Sysgen Omnibridge is a floppy drive controller card that allows high-density floppy support on XTs and above for up to four floppy drives. It can be the primary or secondary controller.
