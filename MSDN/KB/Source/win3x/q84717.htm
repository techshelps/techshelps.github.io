Gerber Software Requires Upgrade for Windows 3.1

 Last reviewed: November 23, 1994
Article ID: Q84717
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

When using Gerber Scientific's Composer version 3.0 with Microsoft Windows operating system version 3.1, you may receive the following general protection (GP) fault: 

   GPF in COMPOSER.EXE at 003A:09BC
You may also be missing the toolbox display.

Gerber has an upgrade to correct this problem. For more information on this upgrade, contact Gerber at (800) 336-1698 and ask for the Graphic Advantage group.

MORE INFORMATION

Gerber Scientific sells dedicated computer systems with Windows 3.0 for scientific plotting tasks.
