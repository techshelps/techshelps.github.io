Multimedia Personal Computing: The Microsoft View

 Last reviewed: May 1, 1997

The information in this article applies to:

  • Microsoft Windows operating system versions 3.0, 3.0a

At the Windows World exposition in Atlanta, Georgia in May, 1991, Microsoft distributed a document titled "Multimedia Personal Computing: The Microsoft View." This document describes multimedia, its application to business and home computing and the Microsoft commitment to help fulfill the vision of "Information at Your Fingertips"(TM) through multimedia technology.

The text of the document is available in two different formats in the Software/Data Library: MMBACKWW.EXE has a copy of the file in Word for Windows format and MMBACK.EXE has a copy of the file as plain text.
