Windows Err Msg: Incorrect Version of FINSTALL

 Last reviewed: November 21, 1994
Article ID: Q69253
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

You may get the error message 

   Incorrect version of FINSTALL
when installing soft fonts in Microsoft Windows version 3.0.

CAUSE

This is caused by a corrupted or improperly expanded FINSTALL.DLL soft font installation file.

RESOLUTION

To resolve this problem, expand the file and try the font installation again.

The FINSTALL.DLL has a compressed file size of 59212 bytes and an expanded file size of 104128 bytes. This file is located on the 1.2 MB 5.25-inch Disk 5 and the 720K 3.5-inch Disk 7.
