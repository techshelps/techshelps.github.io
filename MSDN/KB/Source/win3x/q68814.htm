Procom Propak S185 and 185-15 Hard Disk Drive Problem

 Last reviewed: November 22, 1994
Article ID: Q68814
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

When you attempt to access a Procom Propak S185-15 or 185-15 hard disk drive while you are in 386 enhanced mode of Microsoft Windows version 3.0, the following error message may be displayed after about 30 seconds: 

   Error reading drive x:, Abort, Retry, Fail
If you press "R" (without the quotation marks) to retry, Windows reads drive x: correctly.

Windows runs correctly in real and standard mode with the Propak S185-15 or 185-15 hard disk drive.

STATUS

Procom is aware of this problem and is researching a solution.

MORE INFORMATION

For more information, contact Procom product support.
