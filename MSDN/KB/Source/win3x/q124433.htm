Information About Improving Windows 3.1 Hard Disk Access

 Last reviewed: May 1, 1997
Article ID: Q124433
3.10 WINDOWS kbfile kbenv

The information in this article applies to:

  • Microsoft Windows operating system version 3.1

SUMMARY

The file WIN31DSK.EXE contains a paper that discusses two technologies contained in Microsoft Windows operating system version 3.1: the SMARTDrive 4.0 disk cache and 32-bit disk access. These technologies are designed to improve the performance of Windows by providing faster hard disk access. For information about obtaining WIN31DSK.EXE, refer to the section below.
