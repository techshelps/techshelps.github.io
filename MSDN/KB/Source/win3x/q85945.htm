Windows Paintbrush Does Not Print to a Plotter

 Last reviewed: November 23, 1994
Article ID: Q85945
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

Microsoft Windows Paintbrush will not print to a plotter. There is no plotter support in the Paintbrush application because plotters cannot print bitmaps, which is Paintbrush's native file format.

MORE INFORMATION

Windows 3.0

When you try to print from Paintbrush with a plotter driver selected in Windows 3.0 and 3.0a, the Print command in the File menu will be unavailable (dimmed).

Windows 3.1

In Windows 3.1 the Print command will be available, but the following error message will appear when you try to print: 

   The currently selected printer cannot print graphics
