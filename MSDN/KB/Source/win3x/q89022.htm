Event Managers or Launchers for Windows

The information in this article applies to:

  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

The following is a list of some of the Windows-based event managers or launch timers that allow you to set a given program or macro to activate at a predetermined time.

For example, you can use a communications program that is set up to start at 3 AM, execute a script to log onto a bulletin board service (BBS), and download messages, without requiring you to guide it through these tasks.
