IBM PC Network Baseband Adaptr/A Untested, Not on WDL

 Last reviewed: November 21, 1994
Article ID: Q97596
The information in this article applies to:
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SUMMARY

The IBM PC Network Baseband Adapter /A network interface card (NIC) driver, formerly available from the Microsoft WDL (Windows Driver Library), is an untested driver. It has been known to have problems and was removed from the WDL.

MORE INFORMATION

The IBMNET.EXE file (which contained the IBM PC Network Baseband Adapter /A driver) also contained other driver files. When expanded, IBMNET.EXE produced driver files for the following:

  • IBM PC Network Baseband Adapter /A
  • IBM PC Network Baseband Adapter
  • IBM PC Network Adapter II
  • IBM PC Network Adapter II/A

IBMNET.EXE was removed from the WDL, and none of these drivers is available from the WDL at this time.
