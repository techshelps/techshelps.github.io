Files Used for Remote Access Services (RAS)

 Last reviewed: March 1, 1995
Article ID: Q121274
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

SUMMARY

When you are troubleshooting Remote Access Services (RAS), it may be necessary to remove the files that were installed with the setup of the RAS client or the point-to-point (PTP) server update. The following listings are provided to aid you in removing all RAS files so that a fresh installation can be implemented.
