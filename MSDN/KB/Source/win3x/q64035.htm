Updated IS/Pensmith Driver Available from Kurta for Windows

 Last reviewed: November 22, 1994
Article ID: Q64035
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

The Kurta IS/ONE input system requires an updated Kurta IS/Pensmith driver to work with Microsoft Windows version 3.0. To obtain the new driver, call Kurta technical support. The Kurta IS/ONE driver is a digitizing tablet.

MORE INFORMATION

To install the new driver, do the following:

  1. From the Main group of Program Manager, choose Windows Setup.

  2. From the Options menu, choose Change System Settings.

  3. In the Change System Settings dialog box, select the "No mouse or other pointing device" option from the Mouse options bar.

  4. Setup will prompt you to put the Windows 3.00 Disk #1 in drive A. Insert the Kurta disk that contains the drivers in drive A and choose OK.
