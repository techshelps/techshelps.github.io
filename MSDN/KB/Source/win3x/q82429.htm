No-Squint II Cursor Enhancement Package and Windows

 Last reviewed: November 23, 1994
Article ID: Q82429
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

No-Squint II is a cursor enhancement package by SkiSoft Publishing Corporation. No-Squint II is a terminate-and-stay-resident (TSR) program that has the filename NOSQUINT.COM and date of 7-11-90.

When No-Squint II is loaded high (with MS-DOS version 5.0 or other memory managers) and you want to reload or run the TSR NOSQUINT.COM to get the information screen or to modify the hot key, improper and erratic cursor behavior results.

It is recommended that No-Squint II be loaded low prior to running the Microsoft Windows operating system version 3.1. The No-Squint TSR can then be run at any time.
