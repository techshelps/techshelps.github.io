Cannot Create Color Text in Write

 Last reviewed: November 30, 1994
Article ID: Q44767
The information in this article applies to:
  • Microsoft Windows operating system versions 2.x, 3.0, 3.0a, 3.1, 3.11

Microsoft Windows Write does not have the capability to create editable text in different colors. You can use the Clipboard to import color text and graphics into Write; however, the text and graphics are not editable beyond changing the size.
