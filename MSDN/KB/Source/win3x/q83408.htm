Windows 3.0a Printer Drivers: File Sizes & Supported Printers

 Last reviewed: November 22, 1994
Article ID: Q83408
The information in this article applies to:
  • Microsoft Windows operating system version 3.0a

SUMMARY

This article contains the printer drivers, version numbers, expanded and compressed file sizes, file dates, printer names (as they appear in the Installed Printers box), and a list of the printers supported with each driver supplied with Microsoft Windows version 3.0a.
