Windows Network Share Directory Files

 Last reviewed: February 12, 1997
Article ID: Q97477
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1

SUMMARY

The following is the directory listing for the Network Share Directory of Windows versions 3.0 (05/01/90), 3.0a (10/31/90), and 3.1 (3/10/92). For versions 3.0 and 3.0a, these directories are created by using the EXPALL.BAT file. (See page 553 in the "Microsoft Windows 3.0 User's Guide" for more information about EXPALL.BAT.) For version 3.1, these directories are created by using the SETUP /A command. (See page 9 in "Getting Started with Microsoft Windows" document number PC21670-0492 or page 9 of the "Windows 3.1 User's Guide" document number 50658).
