Pasting Pictures into Windows Write

 Last reviewed: November 23, 1994
Article ID: Q67295
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

Pictures pasted from the Clipboard into Windows Write are inserted on the current line and aligned to the left margin, regardless of where the cursor is positioned in the document.

Pictures can be repositioned horizontally by choosing Move Picture from the Edit menu. Pictures can also be treated as a paragraph of text. The picture can be moved using the Left, Right, Centered, or Justified command from the Paragraph menu. Also, the picture can be positioned more precisely with the Indent command in the Paragraph menu.
