Scando Version 2.0 Requires Real Mode Windows

 Last reviewed: November 23, 1994
Article ID: Q85600
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

Hammerlab Corporation has confirmed that Scando 2.0 is not compatible with Windows version 3.1. Scando 2.0 is incompatible because Windows 3.1 no longer supports real mode and Scando 2.0 requires real mode to operate correctly under Windows. Scando 2.0 is only compatible with Windows versions 3.0 and earlier.

MORE INFORMATION

Scando 2.0 is a scanning package that originally shipped with a runtime version of Microsoft Windows version 2.03.

Hammerlab Corporation states that Scando version 3.0, to be released in the near future, will be compatible with Windows 3.1.
