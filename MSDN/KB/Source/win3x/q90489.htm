WinLogin 1.0 File List with Expanded Filenames/Sizes

 Last reviewed: November 22, 1994
Article ID: Q90489
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft WinLogin version 1.0

SUMMARY

This article contains a listing of files included on the Microsoft WinLogin version 1.0 disk. This list includes the compressed filename, expanded filename, compressed file size, and uncompressed file size.
