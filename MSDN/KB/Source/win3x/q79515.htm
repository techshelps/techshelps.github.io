Serial Mouse Problems with Magnavox 286

 Last reviewed: March 12, 1997
Article ID: Q79515
3.00 3.00a WINDOWS kb3rdparty hbkw

The information in this article applies to:

  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

Earlier models of the Magnavox 286 personal computer may have trouble running Windows with a serial mouse. One problem may be hanging at the logo screen when trying to run Windows 3.0 in standard mode.

MORE INFORMATION

This problem stems from the fact that several models of the Magnavox 286 have a built-in bus port for the mouse. Windows expects to find the mouse on the bus port and it will hang during its boot if the mouse is connected elsewhere, such as on a serial port.

The Magnavox machines for which this information applies include: 

   Headstart 300
   Headstart 300 CD
   Maxstation 286
   Magnum GL
All of these machines are Magnavox-Philips 286-based machines.
