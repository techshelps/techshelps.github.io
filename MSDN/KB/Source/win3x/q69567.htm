Certain Color Combinations Make Text and Other Items Disappear

 Last reviewed: August 7, 1996
Article ID: Q69567
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11
  • Microsoft Windows 95
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you change color combinations in Windows, text may seem to disappear from dialog boxes and windows, and you may not be able to see the download status bar in MSN, The Microsoft Network, and Microsoft Internet Explorer.

CAUSE

This behavior can occur if you select colors that overlap for certain items. For example, this behavior occurs if you select white for both the Application Workspace (Window color in Windows 95) and Windows text color.

RESOLUTION

Windows 3.x and Windows for Workgroups 3.1x

To restore colors so that you can see the "missing" items, use either of the following methods. Note that the first method is the recommended method:

  • Remove the [Colors] section from the Win.ini file. To do so, follow these steps:

    1. Quit Windows.

    2. Use any text editor (such as Edit.com) to open the Win.ini file 

          in the Windows folder.

    3. Remove the [Colors] section from the file. Make sure to remove the 

          "[Colors]" section heading and all the entries up to the next
      section heading.

    4. Save and then close the Win.ini file.

    5. Restart Windows.

    This resets the colors to the original default settings.

  • Reset the colors within Windows. To do so, follow these steps:

    1. Double-click Control Panel, and then double-click the Color icon.

    2. Press DOWN ARROW, press CTRL+HOME, and then press ENTER.

    This resets the colors to the original default settings.

Windows 95

To restore colors so that you can see the "missing" items, follow these steps:

  1. Use the right mouse button to click the desktop, and then click Properties on the menu that appears. (If you cannot see the text in the menu, click the bottom of the menu.)

  2. Click the Appearance tab. (If you cannot see the labels on the tabs, click the third tab from the left.)

  3. Press DOWN ARROW, and then press ALT+A. This selects the first color scheme listed.

    NOTE: If the status bar is still not visible, repeat step 3, but this time press DOWN ARROW more than once.

  4. Choose the color scheme you want, and then click OK.
