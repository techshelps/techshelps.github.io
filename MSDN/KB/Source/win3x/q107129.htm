How to Disable 32-Bit File Access or 32-Bit Disk Access

 Last reviewed: December 11, 1997
Article ID: Q107129
3.11 WINDOWS kbusage kbhw

The information in this article applies to:

  • Microsoft Windows for Workgroups version 3.11

To disable 32-bit file access and start Windows for Workgroups version 3.11, type the following at the MS-DOS prompt and then press ENTER: 

   win /d:c
To disable 32-bit disk access and start Windows for Workgroups version 3.11, type the following at the MS-DOS prompt and then press ENTER: 

   win /d:f
