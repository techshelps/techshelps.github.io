Write Cannot Use Cut, Copy Accelerator Keys

 Last reviewed: November 23, 1994
Article ID: Q62642
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

When you start Microsoft Windows Write as an icon, the Edit Cut (SHIFT+DEL) and Edit Copy (CTRL+INS) accelerator keys do not initially function.

CAUSE

This is a problem with Write, not with Windows.

WORKAROUND

Press the ALT key twice to initialize the Write menu bar or do not start Write as an icon.

STATUS

Microsoft has confirmed this to be a problem in Windows Write version 3.0. This problem does not occur in later versions.
