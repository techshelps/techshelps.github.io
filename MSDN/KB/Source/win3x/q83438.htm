Printer Driver and Version Supplied in Windows SDL

 Last reviewed: November 21, 1994
Article ID: Q83438
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

The following is a summary of the printer drivers, version numbers, file sizes, file dates, printer names as they appear in the Installed Printers list box; a listing of any associated files; and a listing of the printers supported under each driver as supplied with the Windows 3.0 Supplemental Driver Library (SDL):

This information applies to Microsoft Windows versions 3.0 and 3.0a. It does not apply to later versions of Windows.

NOTE: All files on the SDL are NOT compressed.
