No Multimedia Features in Real Mode with Multimedia Windows

 Last reviewed: November 22, 1994
Article ID: Q76815
The information in this article applies to:
  • Microsoft Windows with Multimedia Extensions version 1.0

Multimedia features only work in standard or 386 enhanced mode. If Windows with Multimedia Extensions is run in real mode, animations, audio, and MIDI (musical instrument digital interface) features do not operate.

This information does not apply to later versions of Windows.
