Large Titles and Icons on EVEREX/AGI 286

 Last reviewed: November 22, 1994
Article ID: Q83973
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

When Microsoft Windows operating system version 3.1 is installed on an EVEREX/AGI 286 model 3602A laptop, the icons and icon titles appear to be twice the size of the same display under Windows 3.0. This happens with either the CGA or EGA drivers.

There is no solution to this problem.

MORE INFORMATION

This laptop has a gas plasma screen with the capacity of handling 640 x 400 pixel resolution and dimensions of 211mm X 132mm (8.3 inches across by 5.2 inches tall). Windows 3.1 otherwise runs fine except for the icons and words being oversized.
