Setup Word for Windows Before Running Program Over A Network

 Last reviewed: February 15, 1995
Article ID: Q91074
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0b, 6.0 6.0a 6.0c
  • Microsoft Windows for Workgroups versions 3.1, 3.11
  • Microsoft Windows operating system versions 3.1, 3.11 -------------------------------------------------------------------

SUMMARY

If you want to run Word for Windows from a network or shared workgroup drive, you must first run the Word Setup program and choose the Workstation Installation option. Word for Windows does not operate correctly if you start the program by running the WINWORD.EXE file from a network server. A general protection (GP) fault eventually occurs when you run Word for Windows in this manner.

MORE INFORMATION

For more information about installing Word for Windows on a network workstation, refer to Appendix A (pages 793-802) in the "Microsoft Word for Windows User's Guide."

For information about licensing Word for Windows for use on multiple workstations, contact your Microsoft dealer or Microsoft Sales Information Center (MSIC) at (800) 426-9400.

REFERENCES

"Microsoft Word for Windows User's Guide," version 2.0, pages 793-802.
