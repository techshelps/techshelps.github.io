Windows Err Msg: GP Fault in WSPDPBF.DRV

 Last reviewed: April 25, 1996
Article ID: Q94361
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11
  • Microsoft Access for Windows, version 2.0
  • Microsoft Bookshelf for Windows, 1994 Edition
  • Microsoft Excel for Windows, version 5.0
  • Microsoft Office for Windows, versions 4.x
  • Microsoft PowerPoint for Windows, version 4.0
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a
  • Microsoft Works for Windows, version 2.0, 3.0
  • Microsoft Project for Windows, version 4.0

SYMPTOMS

When you use the Diamond Speedstar 24x 256K video driver version 2.02 with Microsoft Windows Solitaire or any of the Microsoft Office products included above, you receive the following error message 

   <Program Name> has caused a general protection fault in
   WSPDPBF.DRV at 0001:0F38
where <Program Name> is the name of your program.

WORKAROUND

You can work around this problem by using the standard VGA video driver supplied with Windows 3.1 or by using any of the other Diamond Speedstar 24x drivers.

STATUS

Diamond has confirmed this to be a problem with version 2.02 of its 256K video driver. This problem has been corrected with the version 2.03. This driver is available by calling Diamond Speedstar Technical Support or by downloading the driver from CompuServe or American Online.

MORE INFORMATION

This error message can occur when you choose Options from the Word for Windows Tools menu, attempting to change the column width of a table using the mouse, or when you load Works for Windows or open a file within that program.
