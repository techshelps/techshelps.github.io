Setup Does Not Show a Sound Driver Option

 Last reviewed: November 22, 1994
Article ID: Q78037
The information in this article applies to:
  • Microsoft Windows with Multimedia Extensions version 1.0

The Windows with Multimedia Extensions Setup program only displays a Sound option when initially setting up Windows with Multimedia Extensions on a machine. After the initial installation, if Setup is run again from the command (MS-DOS) prompt, no sound option is displayed.

Any driver additions and deletions can be handled by selecting the Drivers icon in the Control Panel and making the appropriate changes. \ jimcam

This information applies to Windows with Multimedia Extensions version 1.0. This information does not apply to later versions of Windows.
