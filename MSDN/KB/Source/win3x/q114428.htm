ProAudio Spectrum 16 Card Plays Wave Files at High Speeds

 Last reviewed: November 22, 1994
Article ID: Q114428
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

Wave (.WAV) files being played in Microsoft Windows or Microsoft Windows for Workgroups play at higher speeds than normal.

CAUSE

This behavior may occur if you are using a MediaVision ProAudio Spectrum 16 sound card that has the number 650-0022-01 printed on the bottom of the card and if the U7 chip is imprinted with the number LM358.

RESOLUTION

Contact MediaVision technical support for information regarding updating the card's hardware.

MORE INFORMATION

The ProAudio Spectrum 16 sound card is manufactured by MediaVision, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
