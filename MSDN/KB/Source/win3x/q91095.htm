NET START from MS-DOS Prompt Can Hang Computer

 Last reviewed: November 21, 1994
Article ID: Q91095
The information in this article applies to:
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

If you start MS-DOS Prompt from Windows for Workgroups and use the NET START command, your machine may stop responding (hang). You also may be forced out of Windows for Workgroups and lose data from any open files in other applications.

RESOLUTION

Microsoft does not recommend that you use the NET START command inside of Windows for Workgroups. The NET START command should only be used before entering Windows for Workgroups.
