Visionware's X-Vision 4.0 Requires NetBIOS with Windows 3.1

 Last reviewed: June 6, 1996
Article ID: Q83974
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

If you run Visionware's X-Vision version 4.0 with the Microsoft Windows operating system version 3.1, you must have NetBIOS support loaded.

Visionware recommends installing Windows 3.1 for a Digital Equipment Corporation (DEC) Pathworks version 4.0 network.
