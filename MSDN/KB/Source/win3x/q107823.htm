Incorrect Sort Order in File Manager with Some Languages

 Last reviewed: May 21, 1996
Article ID: Q107823
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups version 3.1

SYMPTOMS

If you have the Swedish, Norway, Finnish, or Icelandic keyboard drivers installed for Windows, you may see an incorrect sort order in File Manager.

CAUSE

The incorrect sort order is caused by a problem with the LANGSCA.DLL file in Windows and Windows for Workgroups. (The LANGSCA.DLL file is the Finnish, Icelandic, Norwegian, and Swedish language driver used in Windows and Windows for Workgroups.)

STATUS

Microsoft has confirmed this to be a problem in Windows versions 3.1 and 3.11. This problem was corrected in Windows for Workgroups version 3.11.
