Post Office Directory Name Cannot Contain Extended Characters

 Last reviewed: November 21, 1994
Article ID: Q107799
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

SYMPTOMS

When you try to create a Windows for Workgroups 3.11 post office, you may receive the following error message: 

   That action could not be completed. Check that your post office has
   sufficient free space and that your network connection is active.

CAUSE

This error occurs if you try to create a post office in a directory that contains extended characters in its name.

RESOLUTION

To avoid this error, create the post office in a directory that does not contain any extended characters in its name. The default and preferred directory name to use for the post office is WGPO.
