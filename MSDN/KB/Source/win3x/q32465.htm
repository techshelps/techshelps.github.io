Switching from Header or Footer Dialog Box and Text in Write

 Last reviewed: November 21, 1994
Article ID: Q32465
The information in this article applies to:
  • Microsoft Windows operating system versions 2.x, 3.0, 3.0a, 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

When you create a header or footer in Microsoft Windows Write without using a mouse, you must use ALT+F6 to go back and forth between the dialog box and the header or footer text. The ALT+(underlined character in the dialog box) key code does not work until the dialog box is active.
