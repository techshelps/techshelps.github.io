WFWG Mail Password Case Sensitive Only When Changing Password

 Last reviewed: May 24, 1995
Article ID: Q130596
The information in this article applies to:
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

When you change your Windows for Workgroups Mail password, the OK button is unavailable, even though you typed the correct password in the Verify New Password box.

CAUSE

In the Change Password dialog box, the New Password and Verify New Password boxes are case sensitive.

RESOLUTION

Type your new password again, making sure to use the same case when you verify the password.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Windows for Workgroups versions 3.1 and 3.11. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

In the versions of Microsoft Mail shipped with Windows for Workgroups, the only place where passwords are case sensitive is the Change Password dialog box.
