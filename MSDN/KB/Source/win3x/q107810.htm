Err Msg in MS-DOS Session: Cannot Write to Drive C:/Cancel...?

 Last reviewed: March 30, 1995
Article ID: Q107810
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

SYMPTOMS

When you choose the MS-DOS Prompt icon, you receive one of the following error messages: 

   Cannot access drive C:

   -or-

   Cannot find drive C:
Or, if you can run the MS-DOS Prompt, you receive the following error message when you type "exit" (without the quotation marks) at an MS-DOS prompt: 

   Cannot Write to Drive C:

CAUSE

These errors are caused by a corrupted DOSAPP.INI file; deleting this file corrects the problem.
