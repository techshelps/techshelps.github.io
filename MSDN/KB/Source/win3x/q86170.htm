Previous Connections List in the WRK 3.1 May Be Empty

 Last reviewed: November 23, 1994
Article ID: Q86170
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

In the Network Assistant program, which is included with the Microsoft Windows Resource Kit (WRK) for Windows operating system version 3.1, the Previous Connections List option will not show previous connections made from the Windows 3.1 File Manager.

The only previous network connections that will appear on this list are those connections that were made from the Network Assistant program with the Network Connections option.

MORE INFORMATION

Both the Network Connections and the Previous Connections List options are accessible from the Network Assistant program, included with the WRK version 3.1. The Previous Connections List's information is saved in the NETAPP.INI file in the Windows directory. The File Manager previous connections are saved in the WINFILE.INI file in the Windows directory.
