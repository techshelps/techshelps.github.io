Capturing 320 x 200 MS-DOS Apps in Low Memory

 Last reviewed: November 21, 1994
Article ID: Q57638
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

When you attempt to capture the screen of an MS-DOS application running in 320 x 200 4-color mode (modes 4 or 5) under Windows version 3.0 386 enhanced mode in a low memory situation, it may be necessary to perform the capture operation twice.

If you attempt to capture a 320 x 200 screen and find that the Clipboard is blank, repeat the operation and it should work.

This problem occurs only in low memory situations under 386 enhanced mode.
