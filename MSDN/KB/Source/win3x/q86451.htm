Err Msg: "Abnormal Termination" with Adobe TypeAlign

 Last reviewed: May 17, 1996
Article ID: Q86451
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

When you try to run Adobe Systems TypeAlign under Windows version 3.1, the following error message may be displayed: 

   Application error, application requested abnormal termination.
   [close]

CAUSE

This error is a result of having an older version of TypeAlign on your system. Windows 3.1 is not compatible with TypeAlign versions earlier than version 2.1.

RESOLUTION

To run Windows 3.1 with an older version of TypeAlign on your system, you must temporarily disable TypeAlign by removing the following line from the [windows] section of your WIN.INI file: 

   load=talgndll.exe
To obtain an updated version of TypeAlign, contact Adobe Systems Inc.
