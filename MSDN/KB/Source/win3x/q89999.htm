Removing Time Separator Causes Incorrect Time in File Manager

 Last reviewed: November 23, 1994
Article ID: Q89999
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1
  • Microsoft Windows for Workgroups version 3.1

SYMPTOMS

If the colon (:) separator in the International - Time Format dialog box is removed, File Manager truncates the minutes and seconds of the file date.

MORE INFORMATION

If the separator is enabled, the File Manager time stamp has a reading such as "9:14:55 AM." If the separator is disabled, the File Manager time stamp has a reading such as "9." File Manager does not round the time to the nearest hour; for example, "9:59:59" becomes "9" when the separator is disabled.
