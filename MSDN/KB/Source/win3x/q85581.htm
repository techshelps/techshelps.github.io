Rumba 3270 Emulation May Cause GP Faults with Windows 3.1

 Last reviewed: November 23, 1994
Article ID: Q85581
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

If you run Wall Data's Rumba 3270 Emulation version 3.1.03 with Microsoft Windows version 3.1, a general protection (GP) fault may occur.

CAUSE

Rumba version 3.1.03 was written specifically to be compatible with Windows 3.0. Wall Data has confirmed that version 3.1.03 is not compatible with Windows 3.1.

RESOLUTION

To run Rumba with Windows 3.1, upgrade to Rumba version 3.1.04 dated 4/24/92. Contact Wall Data Technical Support for more information.
