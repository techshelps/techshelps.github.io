F1 Key Dysfunctional in Customize Toolbar Dialog Box

 Last reviewed: November 23, 1994
Article ID: Q89975
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.1

SYMPTOMS

In File Manager, you can add custom controls using the Customize Toolbar dialog box. However, if you try to access Help by pressing the F1 key, nothing happens. The Help button works correctly.

STATUS

Microsoft has confirmed this to be a problem in Windows for Workgroups File Manager version 3.1. This problem was corrected in Windows for Workgroups version 3.11.
