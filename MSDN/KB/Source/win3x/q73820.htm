Microsoft PS/2-Style Mouse Incompatibility on Citizen 286

 Last reviewed: November 23, 1994
Article ID: Q73820
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SYMPTOMS

The Microsoft PS/2-style mouse does not work in Windows 3.0 standard mode when plugged into the mouse port on a Citizen 286 computer.

CAUSE

The problem appears to be an interrupt conflict.

WORKAROUND

The mouse works if plugged into a serial port (that is, COM1 or COM2).

MORE INFORMATION

Citizen is aware of the problem and is researching it. Please contact Citizen for more information.
