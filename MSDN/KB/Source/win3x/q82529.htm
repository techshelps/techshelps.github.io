Using Norton Utilities with Erase Protect with Windows

 Last reviewed: November 23, 1994
Article ID: Q82529
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

Setting up a temporary swap file with Microsoft Windows operating system version 3.1 may fail if Norton Utilities's EP.EXE is running. This problem occurs only with Norton Utilities version 5.0 or earlier. EP.EXE is Norton's Erase Protect program.

This problem does not occur in Erase Protect from Norton Utilities version 6.0 or later.
