Money Manager by Expert Software May Cause GP Faults

 Last reviewed: November 23, 1994
Article ID: Q87440
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

Expert Software has confirmed that earlier versions of Money Manager may cause general protection (GP) faults when running under Windows 3.1. According to Expert Software, it does not use version numbers on most of its products. If the file MM.VIP is dated earlier than 3/25/92, then it will need to be upgraded to run correctly under Windows 3.1.

For more information, contact Expert Software technical support.
