DOC ERR: XMS Specification Available

 Last reviewed: May 1, 1997
Article ID: Q84426
3.10 3.11 WINDOWS kbref

The information in this article applies to:

  • Microsoft Windows operating system versions 3.1, 3.11

The "Microsoft Windows Resource Kit" manual for Windows operating system version 3.1 needs clarification on page 450. The "Microsoft Windows Resource Kit" says that the eXtended Memory Specification (XMS) is available from the Microsoft Information Center.

To obtain the XMS specification, see the "Instructions for Downloading" section below.

Instructions for Downloading

The file XMS.EXE contains the Extended Memory Specification version 3.0, including text-formatted XMS versions 2.0 and 3.0, sample source code for an XMS 2.0 driver, a C library for the XMS 2.0 functions, test programs for an XMS 2.0 driver, and HIMEM.SYS version 2.04. The following files are included in XMS.EXE: 

XMS      EXE       167,786 06-05-92   2:00a
CLIB     EXE        22,829 06-05-92   2:00a
OEMSRC   EXE        43,875 06-05-92   2:00a
SPEC     EXE        38,277 06-05-92   3:00a
TESTS    EXE        44,238 06-05-92   2:00a
HIMEM    SYS         6,261 08-15-88   2:04a
README   TXT         3,705 06-05-92   3:00a
        9 file(s)        326,971 bytes

You can find XMS.EXE (size: 167786 bytes) 
                    , a self-extracting file, on the following services:
  • Microsoft's World Wide Web Site on the Internet 

          On the www.microsoft.com home page, click the Support icon.
      Click Knowledge Base, and select the product.

          Enter kbfile XMS.EXE (size: 167786 bytes) 
                          , and click GO!
      Open the article, and click the button to download the file.


  • Internet (anonymous FTP)

          ftp ftp.microsoft.com
      Change to the Softlib/Mslfiles folder.
      Get XMS.EXE (size: 167786 bytes) 

    

  • Microsoft Download Service (MSDL)

          Dial (425) 936-6735 to connect to MSDL
      Download XMS.EXE (size: 167786 bytes)
