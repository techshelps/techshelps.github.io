MR BIOS Version 1.2 or Later Needed for Windows 3.1

 Last reviewed: November 22, 1994
Article ID: Q86188
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

According to Microid Research, MR BIOS version 1.2 or later should be used with Microsoft Windows version 3.1.

MORE INFORMATION

Steps to Find the BIOS Version on a Machine with MR BIOS

  1. After the system starts, press CTRL+ALT+ESC.

  2. In the upper-right corner of the CMOS screen, the BIOS version will be listed to the right of VER:.

  3. Press F10 to exit the CMOS screen.
