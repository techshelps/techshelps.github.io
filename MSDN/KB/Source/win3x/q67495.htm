How to Download the SDL Files from CompuServe

 Last reviewed: April 24, 1996
Article ID: Q67495
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a

SUMMARY

The following is a list of the files that contain the Supplemental Driver Library (SDL) on CompuServe and information about how to download the files.

MORE INFORMATION

Files

S12731.ARC - Windows 3.00 Supplemental Driver Library Disk 1 S12730.ARC - Windows 3.00 Supplemental Driver Library Disk 2 S12729.ARC - Windows 3.00 Supplemental Driver Library Disk 3 S12728.ARC - Windows 3.00 Supplemental Driver Library Disk 4 S12727.ARC - Windows 3.00 Supplemental Driver Library Disk 5 S12726.ARC - Windows 3.00 Supplemental Driver Library Disk 6 S12725.ARC - Windows 3.00 Supplemental Driver Library Disk 7

Downloading Files

To download any of these files, do the following:

  1. Type "GO MSL" (without the quotation marks) to go to the Microsoft Software Library.

  2. From the Microsoft Software Library menu, choose 4 Download A File.

  3. Enter the filename that you want to download.

  4. Select the appropriate transfer protocol from the Access Protocol menu.

  5. Set up your computer to receive the file that you have requested to be downloaded.

  6. After the file has been downloaded, you must convert the file from an archived file, using an "unarchiving" utility such as ARC-E, PKXARC.EXE, or PKUNZIP.EXE.

CompuServe Instructions

The ARC-E utility is available on CompuServe from the IBM New Users Forum. To download ARC-E and its documentation, do the following:

  1. Type "GO IBMNEW" (without quotation marks) to go to the IBM New Users Forum.

  2. Type "DOWN" to request that a file be downloaded.

  3. Type "ARC-E.COM" to request that the program file be downloaded.

    -or-

    Type "ARC-E.DOC" to request that the documentation file be downloaded.

  4. Select the appropriate transfer protocol.

  5. Set up your computer to receive the file that you have requested to be downloaded.

NOTE: The Windows 3.00 Supplemental Driver Library can be obtained from Microsoft Sales and Service by calling (800) 426-9400.
