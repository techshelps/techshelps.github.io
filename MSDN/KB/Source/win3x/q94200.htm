Windows Screen Savers Don't Activate with Icon Hearit

 Last reviewed: February 21, 1996
Article ID: Q94200
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

The screen savers provided with Microsoft Windows will not activate if Icon Hearit is loaded and the Animate Cursor On Startup option is chosen.

CAUSE

The Animate Cursor On Startup option in Moon Valley Software's Icon Hearit software moves the cursor, acting similar to a screen saver, and does not allow Windows screen savers to activate. Turning off this option allows the Windows screen savers to work.

WORKAROUND

To disable this option in Icon Hearit, do the following:

  1. Double click the Icon Hearit icon.

  2. From the Options menu, disable the Animate Cursor On Startup option.
