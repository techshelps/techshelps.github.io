Icon DoIt 3.0 Requires Update to Run with Windows 3.1

 Last reviewed: February 21, 1996
Article ID: Q82588
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

The animated icon feature in Icon DoIt does not work with Windows 3.1. Icon DoIt is an icon and cursor replacement/animation program manufactured by Moon Valley Software. This program also cannot read the existing icons in Program Manager, in Windows 3.1. (Icon DoIt could read the icons in Windows 3.0a.)

These problems have been corrected in Icon DoIt version 3.1.

For more information, contact Moon Valley Software.

Icon DoIt is manufactured by Moon Valley Software, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
