Okidata ML92 Prints Random Characters

 Last reviewed: February 28, 1995
Article ID: Q93051
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SUMMARY

Some versions of the Okidata ML92 and ML93 printer cannot print graphics; therefore, such versions can print from Windows only if you are using the Generic/Text Only (TTY.DRV) printer driver.

MORE INFORMATION

Versions 1.0, 2.0, and 2.1 of the printer are IBM graphics compatible. For these versions, you can use the Okidata ML 92-IBM, Okidata ML 93-IBM, or the IBM Graphics printer drivers.

Versions 3.0, 4.0, and 5.0 of printer are not IBM graphics compatible. For these versions, you must use the Generic/Text Only (TTY.DRV) printer driver.

Determining the Printer Version

To determine the printer version, hold down the linefeed button, turn the printer on, and hold down the LF button for two seconds. When the printer performs the self test, the first line of printed text lists the version number.
