Windows Workstation Requires Version 4.1

 Last reviewed: November 21, 1994
Article ID: Q87291
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

To run Automated Design Systems' Windows Workstation with Microsoft Windows, you must use Windows Workstation version 4.1.

If you use an earlier version of Windows Workstation, then you may have problems printing across the network, and may experience Novell shell problems.

MORE INFORMATION

Windows Workstation is a network application for Windows 3.0, 3.0a, and 3.1 that adds additional configuration control, security options, and printer control. It is available for Novell and LAN Manager and compatible networks.

For further information, contact Automated Design Systems.
