Font Corruption in Ventura Publisher 4.0

 Last reviewed: February 13, 1998
Article ID: Q91096
3.10 3.11 WINDOWS kb3rdparty kbdisplay kbprb

The information in this article applies to:

  • Microsoft Windows for Workgroups versions 3.1, 3.11
  • Microsoft Windows operating system versions 3.1, 3.11

SYMPTOMS

In Ventura Publisher version 4.0 under Microsoft Windows or Microsoft Windows for Workgroups, you may notice some font corruption in a few Ventura Publisher fonts. Text may display in one or two columns only. This may occur when using these fonts: CG Times, Univers, and Univers CD. If you change to a TrueType font included with Windows for Workgroups, the problem no longer occurs.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
