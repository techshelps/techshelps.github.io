Changing Portrait and Landscape Orientation in DayBook

 Last reviewed: November 23, 1994
Article ID: Q64085
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0

SYMPTOMS

When you change the paper orientation in DayBook, the change isn't recognized the first time you print. Any changes made are only recognized the second time you print.

CAUSE

This is due to a problem with the communication between DayBook and the Control Panel.

STATUS

Microsoft has confirmed this to be a problem in Daybook version 1.0. This problem does not appear in later versions.
