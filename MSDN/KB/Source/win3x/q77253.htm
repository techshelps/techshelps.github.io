Unable to Access Output Device in Lotus 1-2-3 for Windows

 Last reviewed: November 21, 1994
Article ID: Q77253
The information in this article applies to:
  • Microsoft Windows operating system versions 3.0, 3.0a, 3.1, 3.11

SUMMARY

When you try to print from Lotus 1-2-3 for Windows, you may receive the following error message: 

   Unable to access output device

MORE INFORMATION

The following conditions may cause this error:

  • The printer you selected is not connected to a port. In the Windows Control Panel, choose the Printers icon to assign the printer to a valid output port, or make the printer active on an output port.
  • There are not enough Windows system resources to print. Close one or more of your applications and try to print again. If you are using a high-end video drivers (1280 x 1024 resolution or 250,000 colors) try using the standard Windows VGA video driver.
