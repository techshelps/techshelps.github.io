WFWG Setup Leaves Two Sets of Mail and Schedule+ Icons

 Last reviewed: November 21, 1994
Article ID: Q106023
The information in this article applies to:
  • Microsoft Windows for Workgroups version 3.11

SYMPTOMS

When you upgrade from Windows for Workgroups version 3.1 to Windows for Workgroups version 3.11, you are left with two sets of Microsoft Mail and Microsoft Schedule+ icons: one set in the Main group and one set in the Network group.

RESOLUTION

Both sets of icons point to the same programs. You can delete either set of icons or keep both.

NOTE: Deleting a program icon does not delete the program files.
