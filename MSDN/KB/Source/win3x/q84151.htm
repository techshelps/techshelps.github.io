Changes to Windows Driver Library

 Last reviewed: November 22, 1994
Article ID: Q84151
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11

SUMMARY

The changes to the Microsoft Windows Driver Library (WDL) are documented in several Microsoft Knowledge Base articles, divided by the type of driver. The WDL offers support for many printers, displays, sound cards, and network card adapters not provided with Microsoft Windows operating system version 3.1.

For a summary of the changes to a particular driver, look up its type in the table below, then search on the corresponding article number in the Microsoft Knowledge Base. 

    Driver Type        Article Number
    -----------        --------------

    Audio                  Q90736
    Display                Q90734
    Miscellaneous          Q90737
    Network card adapters  Q91113
    Printer (A-G)          Q90776
    Printer (H-O)          Q90844
    Printer (P-Z)          Q90845
For a complete listing of the WDL's contents and instructions for obtaining the WDL, query on "Q82006" (without the quotation marks) in the Microsoft Knowledge Base.
