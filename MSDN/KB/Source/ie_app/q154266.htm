Read Notes Appear Under Taskbar

 Last reviewed: September 3, 1997
Article ID: Q154266
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95

SYMPTOMS

When you open a note that has been read, it may appear behind the taskbar so that it cannot be seen.

CAUSE

The rectangle used to remember the dimensions of the note is applied whenever a note window is created.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The x and y coordinates of a note are determined by the User.exe file to obtain the cascading effect. User.exe does not readjust the dimensions of the note to adjust for the taskbar if the notes become too big.
