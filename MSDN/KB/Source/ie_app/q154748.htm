Internet Explorer 2.0 for Macintosh Type 192 Error Message

 Last reviewed: December 17, 1997
Article ID: Q154748
The information in this article applies to:
  • Microsoft Internet Explorer version 2.0, 3.0 for Macintosh

SYMPTOMS

When you start Internet Explorer, you may receive the following error message: 

   The application program 'Internet Explorer 2.0' could not be
   opened because an error of type -192 occurred.

CAUSE

This error message occurs when you install the PowerMac version of Internet Explorer on a 68K-series Macintosh.

RESOLUTION

To resolve this problem, download and install the proper version of Internet Explorer for your particular Macintosh. For information about obtaining Internet Explorer for Macintosh, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q148874
   TITLE     : Microsoft Internet Explorer for Macintosh Available
