How to Obtain 128-Bit Security for Internet Explorer

 Last reviewed: October 14, 1997
Article ID: Q172714
The information in this article applies to:
  • Microsoft Internet Explorer version 3.02 for Windows NT 4.0

SUMMARY

This article describes how to upgrade Internet Explorer to support 128-bit secure transactions.

MORE INFORMATION

To upgrade Internet Explorer to support 128-bit secure transactions, follow these steps:

  1. If you have not already done so, install Internet Explorer 3.02 for Windows NT 4.0, which can be downloaded from the following Web site: 

          http://www.microsoft.com/ie/download/

  2. Install the 128-bit version of Service Pack 3 for Windows NT 4.0. Information about obtaining the 128-bit version of Service Pack 3 is available at the following Web site:

http://www.microsoft.com/NTServerSupport/Content/ServicePacks/Default.htm
