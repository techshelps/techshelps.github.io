Unable to Close the Edit Alias Dialog Box in Personal Web Server

 Last reviewed: February 2, 1998
Article ID: Q180301
The information in this article applies to:
  • Microsoft Personal Web Server version 4.0 for Windows 95

SYMPTOMS

If you install Personal Web Server from Microsoft Windows NT Option Pack, right-click a folder, click Sharing, click the Web Sharing tab, click Share This Folder, click Cancel, click the question mark (?), and then click OK or Cancel, the Edit Alias dialog box reappears and does not close.

RESOLUTION

To work around this behavior, press CTRL+ALT+DELETE, click Edit Alias in the Close Program dialog box, and then click End Task.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

You can use Personal Web Server to select folders on your computer and share the contents of the folders with other users on a network. For more information about Personal Web Server, view the "Welcome to Personal Web Server" Help topic in the Windows NT Option Pack documentation.
