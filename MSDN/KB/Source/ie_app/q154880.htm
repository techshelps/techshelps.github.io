Troubleshooting Internet Mail and News Setup

 Last reviewed: September 4, 1997
Article ID: Q154880
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95

SUMMARY

This article describes the Setup process and lists steps to troubleshoot problems you may encounter when you set up Internet Mail and News.

MORE INFORMATION

You can set up Internet Mail and News by double-clicking the Mailnews.exe file, or by installing the typical (Msie301r.exe) or full (Msie301mnt.exe) release of Microsoft Internet Explorer version 3.01. This extracts the Setup files to a temporary folder in your TEMP folder and runs the installation program. The temporary folder is named Mse<xxxx>.tmp, where <xxxx> is a randomly generated number. This folder is deleted when the installation is finished, along with any files that are used only for the installation process.

When the file copy process is finished, you may be prompted to restart the computer. During the restart process, the Wininit.ini file is processed to replace files that were in use at the time of installation. The RunOnce registry key is also processed and Grpconv.exe is run to update the shortcuts in the Start menu.

Troubleshooting Setup Problems

Setup may not work correctly if any of the following conditions exist. Use the appropriate troubleshooting steps listed below to determine the cause of the problem.

  • You may not have enough free space on your hard disk. Check the location from which you downloaded Internet Mail and News or Internet Explorer for the space requirements.
  • The Mailnews.exe file may be damaged or may not have been extracted correctly. To resolve this problem, follow these steps: 

        1. Download the file again, choosing to save it to disk.


        2. When the download is complete, double-click the Mailnews.exe file to
       run the installation again.
  • The W95inf16.dll and W95inf32.dll files do not match, or you have multiple versions of these files on your hard disk. To resolve this problem, follow these steps: 

        1. Search your drive for these files and rename or delete all that are
       found.


        2. Double-click the executable file you downloaded to run the
       installation again.
  • You may be running background terminate-and-stay-resident programs (TSRs) or other programs that are causing a problem with the installation. To resolve this problem, follow these steps: 

        1. Restart your computer. When you see the "Starting Windows 95"
       message, press the F5 key to start Windows 95 in Safe mode.


        2. Double-click the executable file you downloaded again to run the
       installation again.
