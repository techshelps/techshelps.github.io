Quitting News Empties Deleted Items Folder in Internet Mail

 Last reviewed: September 3, 1997
Article ID: Q154329
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95

SYMPTOMS

When you quit Internet News while Internet Mail is running, the Deleted Items folder in Mail is emptied.

CAUSE

The "Empty messages from the Deleted Items folder on exit" option is enabled in Internet Mail.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Mail and News share a common exit routine. Part of this routine involves purging the Deleted Items folder. There is currently no way to check which program was running and to purge only the appropriate folder.
