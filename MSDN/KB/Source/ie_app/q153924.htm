How to Increase Internet News Server Timeout Setting

 Last reviewed: September 2, 1997
Article ID: Q153924
1.00 WINDOWS kbhowto

The information in this article applies to:

  • Microsoft Internet Mail and News version 1.0

SUMMARY

The default server timeout setting for Microsoft Internet News is 30 seconds. This article describes how to increase the server timeout setting for Internet News. Increasing this setting may be necessary if your Internet connection is slow.

MORE INFORMATION

To increase the server timeout setting for Internet News, use the following steps:

  1. Click Start, point to Programs, and then click Internet News.

  2. On the News menu, click Options.

  3. Click the Server tab.

  4. Under News Server(s), click the news server you are using, and then click Properties.

  5. On the Advanced tab, move the Server Timeout slider to the right to increase the server timeout setting.

  6. Click OK, and then click OK again.

NOTE: The maximum server timeout setting is 5 minutes.
