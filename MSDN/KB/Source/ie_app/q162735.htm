Internet News Downloads Newsgroup List Every Time

 Last reviewed: September 9, 1997
Article ID: Q162735
1.00 WINDOWS kbenv

The information in this article applies to:

  • Microsoft Internet Mail and News version 1.0 for Windows 95
  • Microsoft Internet Mail and News version 1.0 for Windows NT 4.0

IMPORTANT: This article contains information about editing the registry. Before you edit the registry, you should first make a backup copy of the registry files (System.dat and User.dat). Both are hidden files in the Windows folder.

SYMPTOMS

The list of newsgroups may be downloaded every time you start Internet News or open the Newsgroups dialog box.

CAUSE

This behavior can occur if you have changed your user name or password on the news server. This causes the current list of newsgroups to be considered invalid, requiring a new list of newsgroups to be downloaded.

RESOLUTION

WARNING: Using Registry Editor incorrectly can cause serious problems that may require you to reinstall Windows 95. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use Registry Editor at your own risk.

NOTE: For information about how to edit the registry, view the Changing Keys And Values online Help topic in Registry Editor (Regedit.exe). Note that you should make a backup copy of the registry files (System.dat and User.dat) before you edit the registry.

Delete the following registry keys: 

   HKEY_CURRENT_USER\Software\Microsoft\Internet Mail and
   News\News\<server name>\Group List Dirty

   HKEY_CURRENT_USER\Software\Microsoft\Internet Mail and
   News\News\<server name>\Invoke on Startup
where <server name> is the name of the news server in which you changed your user name and password.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you change the user name and password combination for a given news server, Internet News assumes that the list of groups is no longer valid and marks the "Group List Dirty" and "Invoke on Startup" registry keys. If these values are present when you start Internet News, a new list of newsgroups is downloaded.
