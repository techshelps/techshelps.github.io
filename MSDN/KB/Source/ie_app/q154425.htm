The information in this article applies to:

  • Microsoft Internet Mail and News version 1.0 for Windows 95

SYMPTOMS

When you attempt to install Microsoft Internet Mail and News version 1.0, you may receive the following error message: 

   This version of Microsoft Internet Mail and News requires a later
   version of Microsoft Internet Explorer. Please update your current
   version and reinstall Internet Mail and News.

CAUSE

This error occurs if Microsoft Internet Explorer version 3.0 or later has not been installed on your computer.

RESOLUTION

Install Internet Explorer version 3.0 or later before you install Internet Mail and News. If you do not want to use Internet Explorer, you can uninstall it after you install Internet Mail and News.

MORE INFORMATION

Internet Mail and News requires the Comctl32.dll and Shdocvw.dll files included with Internet Explorer 3.0 or later. If these files are not present when you try to install Internet Mail and News, you receive the error message. When you remove Internet Explorer, these files are not removed from your computer.

NOTE: The Comctl32.dll file included with Internet Explorer 3.0 or later is a newer version of the Comctl32.dll file included with Microsoft Windows 95. Because the Comctl32.dll file is required by Windows 95, it is always present on your computer, even if Internet Explorer 3.0 or later has never been installed. However, to install Internet Mail and News, the version of the Comctl32.dll file included with Internet Explorer 3.0 or later must be present instead of the version included with Windows 95.

For information about how to obtain the latest version of Internet Explorer, see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q164475
   TITLE     : Availability of Internet Explorer 3.02 for Windows 95 and
               NT 4.0
