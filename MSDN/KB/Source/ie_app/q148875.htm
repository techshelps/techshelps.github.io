Microsoft Internet Explorer 2.1 for Windows 3.1 Available

 Last reviewed: May 1, 1997
Article ID: Q148875
2.10 WINDOWS kbtool kbfile

The information in this article applies to:

  • Microsoft Internet Explorer 2.1 for Windows 3.1

SUMMARY

Microsoft has released Microsoft Internet Explorer version 2.1 for Windows 3.1. You can use this product to browse and view HTML documents on a network, or on the World Wide Web or Internet (provided your network is connected to the Internet or provided you have an existing TCP/IP connection). Other services, such as Gopher and FTP, are also available.

Microsoft Internet Explorer 2.1 for Windows 3.1 is primarily intended for individuals using Microsoft Windows version 3.1 or 3.11 or Microsoft Windows for Workgroups version 3.1 or 3.11 who are connected to a local area network (LAN) or who have existing Internet connections (individuals who have an existing TCP/IP connection, a dialer, and an account with an Internet service provider).

Note that this is a 16-bit program, and that you do not need to have Win32s installed on your computer.

MORE INFORMATION

What's New in Version 2.1

  • Enhanced Frames
  • Animated GIFs
  • Background Sounds
  • Pragma: No-Cache Support
  • Dialer Scripting

Installation

To install Internet Explorer 2.1, follow these steps:

  1. Download the appropriate Internet Explorer 2.1 for Windows 3.1 file from one of the online services listed below to an empty folder.

  2. In File Manager, double-click the file you downloaded in step 1.

  3. Follow the instructions on the screen.
