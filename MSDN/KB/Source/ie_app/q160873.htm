Attached File Modifications Lost When File Is Closed

 Last reviewed: September 8, 1997
Article ID: Q160873
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95

SYMPTOMS

When you edit an attached file received with Internet Mail and save the file, the changes are missing when you next open the file.

CAUSE

Internet Mail does not support saving changes to attachments within a mail message.

RESOLUTION

To work around this behavior, save the attached file to disk and then make changes to the file you saved to the disk. To save the attached file to disk, right-click the attachment, and then click Save as.
