Cannot Use Right Mouse Button Features in Internet Explorer

 Last reviewed: September 8, 1997
Article ID: Q159490
The information in this article applies to:
  • Microsoft Internet Explorer versions 2.0, 2.01, 2.1, 3.0, 3.01, 3.02a for Windows 3.1

SYMPTOMS

When you right-click an area in Internet Explorer to open the shortcut menu, the Diamond InControl Tools utility may open instead.

CAUSE

The InControl Tools utility captures all right mouse button clicks.

RESOLUTION

To use the right mouse button normally in Internet Explorer, quit the InControl Tools utility while you are using Internet Explorer.

MORE INFORMATION

The Diamond InControl Tools utility is known to be included with the following video card: 

   Stealth64 Video 2001 Series
