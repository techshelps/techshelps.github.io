The information in this article applies to:

  • Microsoft Internet Explorer version 2.0 for Windows 95

SUMMARY

Japanese character set support for International Extensions for Microsoft Internet Explorer version 2.0 for Windows 95 is available as a self-extracting file named Ie2ijp.exe.

The extracted files are temporarily placed in the Windows\Temp folder and are deleted after installation is complete. This article lists the contents of the Ie2ijp.exe file and the cabinet file it contains.

The Ie2ijp.exe file is included with the Microsoft Internet Starter Kit version 2.0.
