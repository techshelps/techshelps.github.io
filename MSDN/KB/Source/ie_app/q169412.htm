Missing Favorites in Internet Explorer for Windows 3.1

 Last reviewed: September 9, 1997
Article ID: Q169412
The information in this article applies to:
  • Microsoft Internet Explorer versions 2.0, 2.01, 2.1, 3.0, 3.01, 3.02a for Windows 3.1

SYMPTOMS

After you add a new favorite in Internet Explorer, a previous favorite may be missing from the Favorites list.

CAUSE

Internet Explorer for Windows 3.1 is limited to displaying 16 favorites on the Favorites menu.

RESOLUTION

To view a list of all the favorites, use the appropriate method:

Internet Explorer Versions 2.0, 2.01, 2.1

Click Open Favorites on the Favorites menu.

Internet Explorer Versions 3.0, 3.01, 3.02a

Click Organize Favorites on the Favorites menu.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
