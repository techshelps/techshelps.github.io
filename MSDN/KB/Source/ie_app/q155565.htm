CTRL+G Does Not Function as Expected in Internet Explorer

 Last reviewed: September 5, 1997
Article ID: Q155565
The information in this article applies to:
  • Microsoft Internet Explorer version 3.0 for Windows NT 4.0

SYMPTOMS

In Internet Explorer 1.5 for Windows NT, the CTRL+G key combination provides a shortcut for the Find Next function. This shortcut key combination does not work in Internet Explorer 2.0 or 3.0.

CAUSE

The dialog box for finding text has been changed in newer versions of Internet Explorer. The CTRL+G shortcut is not supported by these new dialog boxes.

RESOLUTION

To find the next instance of the text you are searching for, click the Find Next button in the dialog box.
