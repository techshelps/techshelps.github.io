Cannot Install Internet Assistant After Installing Viewer

 Last reviewed: August 29, 1997
Article ID: Q150597
2.00 WINDOWS kbtool kberrmsg

The information in this article applies to:

  • Microsoft Internet Starter Kit version 2.0

SYMPTOMS

When you attempt to install the Microsoft Excel Internet Assistant or the Microsoft Word Internet Assistant from the Microsoft Internet Starter Kit, you may receive one of the following error messages: 

   For Microsoft Excel Internet Assistant:

    - Setup could not find Microsoft Excel installed on your system.
      Please install Microsoft Excel before installing this application.

    - This version of Excel is incompatible with the version of the
      Internet Assistant you are installing. This version of the Internet
      Assistant requires Excel for Windows 95, or Excel for Windows NT.

   For Microsoft Word Internet Assistant:

    - Setup could not find Microsoft Word installed on your system.
      Please install Microsoft Word before installing this application.

    - This version of Word is incompatible with the version of the
      Internet Assistant you are installing. This version of the Internet
      Assistant requires Word for Windows 95, or Word for Windows NT.

CAUSE

This issue may occur if you install the Microsoft Excel Viewer or the Microsoft Word Viewer from the Internet Starter Kit and you select Open With Viewer in the Set Default Application dialog box.

When you select Open With Viewer, you associate Microsoft Excel or Word files with the Viewer instead of the full retail version of the product. When you try to install an Internet Assistant, the Setup program checks the file association for its default files. If the association points to the Viewer instead of the full retail product, Setup assumes you do not have the full product installed.

RESOLUTION

NOTE: You must have the full retail version of Microsoft Excel or Word installed on your system to use an Internet Assistant.

To resolve this issue, restore Microsoft Excel or Word as the default application. To do so, use one of the following methods:

Method 1

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Add/Remove Programs.

  3. Click Microsoft Excel Viewer or Microsoft Word Viewer, and then click Add/Remove.

  4. Click Remove All. If you are prompted to confirm that you want to remove the Viewer, click Yes.

  5. If you are prompted to restart the computer, click Yes.

  6. Reinstall the Viewer. When the Set Default Application dialog box appears, click Open With Excel or Open With Word.

Method 2

Reinstall Microsoft Excel or Word.

NOTE: Editing the file types using Windows Explorer does not resolve this issue.

MORE INFORMATION

You can use the Microsoft Viewers to view files created with the full retail version of the product even though you do not have the product installed.
