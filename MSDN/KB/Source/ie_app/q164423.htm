Enclosures Not Attached When Finish Later Is Used

 Last reviewed: September 4, 1997
Article ID: Q164423
1.10 MACINTOSH kbtool

The information in this article applies to:

  • Microsoft Internet Mail and News version 1.1 for Macintosh

SUMMARY

When you create a new message in Internet Mail for Macintosh and then click Finish Later on the toolbar, any enclosures you added to the message are no longer attached.

MORE INFORMATION

When you click Finish Later, the message is moved to the Outbox so you can continue creating the message at a later time. To send the message when you finish creating it, click Send on the toolbar.

If an enclosure is attached to the message when you click Finish Later, you must reattach the enclosure while the message is in the Outbox. If you do not add reattach the enclosure, it will not be sent with the message.
