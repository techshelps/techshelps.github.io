Microsoft Internet Explorer 2.0 for Windows 95 Available

 Last reviewed: August 29, 1997
Article ID: Q140649
2.00 WINDOWS kbtool kbfile

The information in this article applies to:

  • Microsoft Internet Explorer version 2.0 for Windows 95

SUMMARY

Microsoft has released Microsoft Internet Explorer version 2.0 for Windows 95. Version 2.0 of Internet Explorer includes the following features:

  • Table support (including features not found in other browsers)
  • Secure Sockets Layer (SSL) security
  • In-line video
  • Network News Transport Protocol (NNTP) news reading and posting
  • Background sounds
  • Client pull
  • Hypertext Transfer Protocol (HTTP) cookie support
  • Improved user interface
  • RealAudio player included
  • Greatly increased speed
  • Ready to display Virtual Reality Markup Language (VRML) worlds, either full-screen or embedded in Web pages

MORE INFORMATION

Installing Internet Explorer 2.0

To install Microsoft Internet Explorer 2.0, use the following steps:

  1. Download the Msie20.exe file from one of the online services listed below into an empty folder.

  2. In My Computer or Windows Explorer, double-click the Msie20.exe file you downloaded in step 1.

  3. Follow the instructions on the screen.

Availability

You can download Msie20.exe, a self-extracting file, from the following online services:

  • Microsoft's World Wide Web site on the Internet 

          www.microsoft.com/ie/download/
  • Internet (anonymous FTP) 

          ftp ftp.microsoft.com
      Change to the Softlib/Mslfiles folder
      Get Msie20.exe
  • Microsoft Download Service (MSDL) 

          Dial (425) 936-6735 to connect to MSDL
      Download Msie20.exe
