Cannot Remove .doc or .xls Files from Internet Explorer Cache

 Last reviewed: September 2, 1997
Article ID: Q154636
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Internet Explorer version 3.0 for Windows NT 4.0

SYMPTOMS

After you access Microsoft Word or Microsoft Excel files using Internet Explorer 3.0, you are unable to remove the files from the Temporary Internet Files folder (the Internet Explorer cache) while Internet Explorer is running.

RESOLUTION

Quit Internet Explorer and then remove the files. To do so, follow these steps:

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Internet, and then click Settings on the Advanced tab.

  3. Click Empty Folder, and then click OK.

STATUS

This behavior is by design.

MORE INFORMATION

Windows NT does not allow files to be deleted while they are open. Both Microsoft Word and Microsoft Excel keep these files open while Internet Explorer is running, causing Windows NT to prevent the deletion.
