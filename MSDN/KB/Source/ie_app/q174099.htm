Print Command Unavailable in Mail Window

 Last reviewed: September 19, 1997
Article ID: Q174099
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.1 for Macintosh

SYMPTOMS

If the Print command is unavailable on the File menu in the Mail window, you must open a mail message in order to print it.

CAUSE

This feature is not available in versions of Internet Mail and News earlier than version 3.0a.

RESOLUTION

Upgrade to Internet Mail and News version 3.0a. To upgrade Internet Mail and News, connect to the following Microsoft Web site: 

   http://www.microsoft.com/ie/download/

MORE INFORMATION

You can read a mail message in the preview pane of the Mail window, but you cannot print it without opening it in versions earlier than 3.0a.
