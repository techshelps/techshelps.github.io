The information in this article applies to:

  • Microsoft Internet Explorer version 3.0 for Windows 3.1

SUMMARY

Microsoft Internet Explorer version 3.0 for Windows 3.1 is available as a self-extracting file named Dlbas30c.exe. The Dlbas30c.exe file contains the Internet Explorer 3.0 for Windows 3.1 Internet browser, a Transmission Control Protocol/Internet Protocol (TCP/IP) stack, and a dialer tool.

This article lists the contents of the Dlbas30c.exe file and the cabinet files it contains.

For information about obtaining the Dlbas30c.exe file, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q159584
   TITLE     : Microsoft Internet Explorer 3.0 for Windows 3.1 Available
