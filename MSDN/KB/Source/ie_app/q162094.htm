Features of Microsoft Web Publishing Wizard Version 1.1

 Last reviewed: August 27, 1997
Article ID: Q162094
The information in this article applies to:
  • Microsoft Web Publishing Wizard version 1.1 for Windows 95
  • Microsoft Web Publishing Wizard version 1.1 for Windows NT 4.0

SUMMARY

Web Publishing Wizard makes it easy to post Web pages on the World Wide Web or an intranet by automating the process of copying files from your computer to a Web server. New features in version 1.1 include:

  • Updates to WebPost service providers.
  • Support for multiple languages.

MORE INFORMATION

Updates to WebPost Service Providers

The following WebPost service providers included with previous versions of Web Publishing Wizard have been updated in version 1.1:

  • CompuServe
  • America Online (AOL)
  • Sprynet
  • GNN

The following WebPost service providers have been added to Web Publishing Wizard version 1.1:
  • America Online Primehost
  • Microsoft FrontPage

Support for Multiple Languages

Web Publishing Wizard version 1.1 supports the following languages:

  • French
  • German
  • Italian
  • Japanese
  • Spanish
  • U.S. English

Additional Information

Web Publishing Wizard version 1.1 is also included with the following programs:

  • Microsoft Office 97 for Windows
  • Microsoft Access 97 for Windows
  • Microsoft FrontPage 97
  • Microsoft Publisher 97
  • Microsoft Creative Writer

NOTE: Publisher 97 does not include the Webpost.exe file. When you use Publisher 97 to install Web Publishing Wizard, it is installed from the Microsoft World Wide Web site.

The following previous versions of Web Publishing Wizard are included with the following programs:

  • Web Publishing Wizard version 1.0 is included with Microsoft Internet Explorer Starter Kit version 3.0.
  • Web Publishing Wizard versions 1.0 and 1.0a are included with Microsoft Creative Writer.

NOTE: Web Publishing Wizard version 1.1 does not work with America Online (AOL) for Windows 95 (32-bit version). It results in the following error message: "AOL is unable to post webpage." The error message does not appear in the 16-bit version of AOL.
