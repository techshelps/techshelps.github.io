Err Msg: IEXPLORE Caused a General Protection Fault

 Last reviewed: March 18, 1998
Article ID: Q170935
The information in this article applies to:
  • Microsoft Internet Explorer version 3.02a for Windows 3.1

SYMPTOMS

When you close Internet Explorer after connecting to a secure Hyper Text Transfer Protocol (HTTPS) Internet site, you may receive the following error message: 

   Application Error
   IEXPLORE caused a General Protection Fault in module MSHTML16.DLL
   at 000B:957C.

RESOLUTION

To resolve this problem, upgrade to Internet Explorer 3.03 for Windows 3.1. To obtain Internet Explorer 3.03, please visit the following Microsoft Web site: 

   http://www.microsoft.com/ie/download/

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Explorer version 3.02a for Windows 3.1.
