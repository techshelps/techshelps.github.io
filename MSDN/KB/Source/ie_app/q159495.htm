TAB Key Does Not Function in Internet Explorer

 Last reviewed: September 4, 1997
Article ID: Q159495
The information in this article applies to:
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02a for Windows 3.1

SYMPTOMS

When you press the TAB key in Microsoft Internet Explorer for Windows 3.1, the text in the Address box is not selected. Earlier versions of Internet Explorer for Windows 3.1 support this functionality.

CAUSE

Internet Explorer for Windows 3.1 does not support this shortcut key feature.

RESOLUTION

To select the text in the Address box, click the Address box with the mouse.
