No News Server Setting Displays E-mail Name

 Last reviewed: December 16, 1996
Article ID: Q159869
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Macintosh

SYMPTOMS

When you use the Internet Configuration Utility to provide profile information to Internet Mail and News for Macintosh, if you do not enter a value in the News Server box, the Internet Configuration Utility inserts your e-mail name as the news server name.

STATUS

Microsoft is researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.
