Internet Mail and News 1.1 Runs on All Macintosh Computers

 Last reviewed: September 9, 1997
Article ID: Q162655
1.10 MACINTOSH kbsetup

The information in this article applies to:

  • Microsoft Internet Mail and News version 1.1 for Macintosh

SUMMARY

While you are installing Internet Mail and News 1.1 for Macintosh, you receive the following prompt: 

   Do you want "Internet Mail and News 1.1" to be able to run on all
   Macintosh computers?

MORE INFORMATION

Setup does not explain which components will be installed when you click Yes or No. If you click Yes, the file allocation table (FAT) version of Internet Mail and News is installed. This version runs on all Macintosh computers, including PowerPC and 68K computers. If you click No, the version of Internet Mail and News specific to the computer you are currently using is installed.
