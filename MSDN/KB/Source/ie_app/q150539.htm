The information in this article applies to:

  • Microsoft Internet Explorer version 2.0 for Windows 3.1
  • Microsoft Internet Explorer version 2.0 for Windows 95
  • Microsoft Internet Explorer version 2.0 for Macintosh

SUMMARY

Microsoft Internet Explorer is now available for the following platforms:

  • Windows 3.1x and Windows for Workgroups 3.1x
  • Windows 95
  • Macintosh

Users requesting support for Internet Explorer can call (425) 635-7123.
