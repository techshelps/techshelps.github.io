Dr. Watson Error Message After Closing Internet Explorer

 Last reviewed: August 28, 1997
Article ID: Q172835
The information in this article applies to:
  • Microsoft Internet Explorer version 3.02 for Windows NT 4.0

SYMPTOMS

When you quit Internet Explorer 3.02, you may receive the following Dr. Watson error message: 

   WangImg.exe
   Exception: access violation (0xc0000005), Address: 0x77b50b0e

CAUSE

Internet Explorer 3.02 has not completely closed the hosted Imager program.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Explorer 3.02 for Windows NT 4.0 and will post new information as it becomes available.

MORE INFORMATION

Dr. Watson for Windows NT is a program that detects and diagnoses program errors, and then logs the resulting diagnostic information. In the event of a program error, Dr. Watson for Windows NT starts automatically. Microsoft Technical Support personnel then can use the information logged by Dr. Watson to diagnose problems.

The Wangimg.exe file is a part of Imaging, an accessory included with Windows NT 4.0.
