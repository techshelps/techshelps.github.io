Err Msg: There Is Not Enough Memory to Run the Java VM

 Last reviewed: September 4, 1997
Article ID: Q164388
3.00 MACINTOSH kberrmsg kbenv

The information in this article applies to:

  • Microsoft Internet Explorer version 3.0 for Macintosh

SYMPTOMS

When you browse a Web page that attempts to run a Java program, you may receive the following error message: 

   There is not enough memory to run the Java VM
If you click OK, the Web page continues to load but the Java program does not run.

RESOLUTION

To resolve this issue, upgrade to the latest version of Internet Explorer for Macintosh. For information about how to obtain the latest version of Internet Explorer for Macintosh, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q148874
   TITLE     : Microsoft Internet Explorer for Macintosh Available

MORE INFORMATION

Java is an object-oriented programming language developed by Sun Microsystems. Java is a platform-independent, multiple-threaded programming environment designed for creating programs and applications for the Internet and Intranets.
