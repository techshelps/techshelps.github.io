Web Services May Be Stopped When Web Server Is Running

 Last reviewed: August 28, 1997
Article ID: Q158366
The information in this article applies to:
  • Microsoft Personal Web Server versions 1.0, 1.0a for Windows 95

SYMPTOMS

When you try to connect to a Microsoft Personal Web Server, you may receive one of the following error messages:

  • Could not open ftp://computername/sharename/

    -or-

  • Could not open http://computername/sharename/

CAUSE

Personal Web Server is running, but Hypertext Transfer Protocol (HTTP) or File Transfer Protocol (FTP) services may be stopped.

RESOLUTION

To start the required services, follow these steps:

  1. Click the Start button, point to Settings, and then click Control Panel.

  2. In Control Panel, double-click the Personal Web Server icon.

  3. Click the Services tab.

  4. Click the service you want to use, and then click Start.

  5. Click OK.

MORE INFORMATION

The FTP service is not started by default when the Personal Web Server services are installed.
