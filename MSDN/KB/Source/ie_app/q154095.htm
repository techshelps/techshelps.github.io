Internet Mail Automatically Opens Messages Top to Bottom Only

 Last reviewed: September 3, 1997
Article ID: Q154095
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95

SYMPTOMS

Regardless of the sort order (ascending or descending) in a message folder, Internet Mail opens the message listed below the current message in the folder automatically when you do one of the following:

  • You move a mail message from the list view.
  • You delete an open mail message.
  • You click Next Message on the View menu in an open message window.
  • You click the Next button on the toolbar in an open message window.

There is no option to open the message above the current message.

CAUSE

This behavior is by design. To open messages in a particular order, sort the message list in that order. The messages are opened from top to bottom.

STATUS

This feature is under review and will be considered for inclusion in a future release of Microsoft Internet Mail and News.
