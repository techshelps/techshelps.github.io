Error Message: Could Not Save This Message...

 Last reviewed: February 3, 1997
Article ID: Q163117
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.1 for Macintosh

SYMPTOMS

When you attempt to mark a message as Finish Later in the Inbox in Internet Mail for Macintosh, you may receive the following error message: 

   Could not save this message.
   Error 1634955892

CAUSE

The message was previously restored to the Inbox after being marked as Finish Later and deleted.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Finish Later is a feature for placing a message in the Outbox, but not sending it until it is opened and the Send/Post Message command is used.
