Color Box Does Not Display Correct Text Color

 Last reviewed: September 10, 1997
Article ID: Q170945
The information in this article applies to:
  • Microsoft Internet Mail and News versions 3.0, 3.0a for Macintosh

SYMPTOMS

When you type any text and then apply color formatting to the text, the Color Box may not display the color you applied.

CAUSE

The Color Box is not always refreshed immediately.

RESOLUTION

To work around this problem, either select the text you typed, or press ENTER after applying color formatting.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Mail and News versions 3.0 and 3.0a for Macintosh. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
