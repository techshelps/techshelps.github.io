How to Configure Microsoft Internet News for Windows 95

 Last reviewed: September 2, 1997
Article ID: Q153841
The information in this article applies to:
  • Microsoft Internet Mail and News version 1.0 for Windows 95
  • Microsoft Internet Mail and News version 1.0 for Windows NT 4.0

SUMMARY

You can use Microsoft Internet News to read bulletin board discussion groups (such as Usenet newsgroups) using Network News Transfer Protocol (NNTP)-based news servers. This article describes how to configure Internet News.

MORE INFORMATION

Running the Internet News client requires that you have an existing news server. If you do not know which news server you should use, contact your Internet service provider.

The first time you run Microsoft Internet News, you are prompted for the following information:

  • The name of your news server.
  • Your account name and password on your news server. Depending on your service provider, you might not need this.
  • Your full e-mail address. This is the address that people should use when responding to a message that you post to a newsgroup. For example, "johndoe@acme.com."

To change this information after Internet News is installed, follow these steps:

  1. On the News menu, click Options.

  2. Click the Server tab, click your server, and then click Properties. Modify the information as appropriate.

    NOTE: If your server is not listed, Click Add and enter your server information.

  3. Click OK to save the settings.

  4. Click OK.

Before you can view Internet newsgroups, you must establish an Internet connection. For information about connecting to the Internet, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q138789
   TITLE     : How to Connect to the Internet in Windows 95
NOTE: When you configure the news server for Internet News, make sure not to include a space before the server name. If you include a space, you receive an error message when you start Internet News or when you view a newsgroup. To correct this problem, remove the server from the list of available servers by following these steps:

  1. In the Newsgroups box, click either Outbox or Posted Items.

  2. On the News menu, click Options.

  3. Click the Server tab.

  4. Click the server with a space before its name, and then click Remove.

  5. Click Yes.

After you follow these steps, you can add the server again.
