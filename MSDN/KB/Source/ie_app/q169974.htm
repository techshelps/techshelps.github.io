How to Decode Files Using Internet Explorer for Macintosh

 Last reviewed: September 9, 1997
Article ID: Q169974
The information in this article applies to:
  • Microsoft Internet Explorer version 3.01a for Macintosh

SUMMARY

This article describes how to decode files using Internet Explorer version 3.01a for Macintosh.

MORE INFORMATION

Internet Explorer can automatically decode BinHex and MacBinary files during download. You can turn this option on or off. To change this option, perform the following steps:

  1. On the Edit menu, click Preferences.

  2. Click Download Options, and then click the Automatically Decode BinHex Files check box to select it or clear it.

When you enable this option, Internet Explorer uses its own internal routine to decode these files. If you want to use an external tool to decode these files, perform the following steps:

  1. On the Edit menu, click Preferences.

  2. Click File Helpers, and then click BinHex File or MacBinary File in the Description/Extensions list.

  3. Click Change.

  4. In the Handling area, click Browse, locate the external tool, and then click Open.

  5. Click OK, and then click OK again.
