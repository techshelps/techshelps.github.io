MSDN: Setup Fails with CD-ROM Shared on Novell Server

 Last reviewed: August 4, 1995
Article ID: Q106081
The information in this article applies to:
  • Microsoft Developers Network (MSDN), Disc 5

SYMPTOMS

The MSDN Disc 5 setup may fail if the shared CD-ROM drive is on a server running Novell NetWare version 3.11.

Attempting a manual installation may also fail due to an inability to copy all of the files.

RESOLUTION

Unloading and then reloading the following NetWare Loadable Modules (NLMs) on the server may correct the problem. 

NLM             Software (Source)


UNIV            Corel SCSI Pro
CDSRV           Corel SCSI Pro
MS_SCSI         Maynard Tape drive
MAYNSTRM        Maynard Tape drive
For everything to work, the NLMs need to be loaded in the following order: MS_SCSI, UNIV, CDSVR, MAYNSTRM. No changes are necessary on the client workstations.
