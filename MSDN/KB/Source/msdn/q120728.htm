FILE: Microsoft Development Library Demo Available in Softlib

 Last reviewed: February 3, 1997
Article ID: Q120728
The information in this article applies to:
  • Microsoft Development Library

SUMMARY

The Microsoft Development Library is a comprehensive reference for developers who are developing applications for Microsoft Windows. The Microsoft Development Library contains sample code, technical articles, specifications, utilities, product documentation, the Microsoft Knowledge Base, and more.

A demo of the Microsoft Development Library is now available for download from the Microsoft Software Library (MSL). This demo delivers the full functionality of the Microsoft Development Library applied to a subset of the Library's contents, along with a comprehensive listing of everything on the Library CD-ROM. The Library is available only through membership in the Microsoft Developer Network. For more information about the Microsoft Developer Network, please call (800) 759-5474.
