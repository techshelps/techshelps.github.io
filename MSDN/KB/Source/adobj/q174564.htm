HOWTO: Search for RDS Articles by Article Topic/Keyword

 Last reviewed: March 10, 1998
Article ID: Q174564
The information in this article applies to:
  • Remote Data Service for ADO versions 1.0, 1.1, 1.5

SUMMARY

The Microsoft Knowledge Base is categorized by using keywords. This article lists the keywords specific to articles in the Remote Data Service (RDS) collection, which includes the Microsoft products listed at the beginning of this article.

The Remote Data Service, with version 1.5, is now a feature of ActiveX Data Objects (ADO). Previously, the Remote Data Service was known as the Advanced Data Connector (ADC).

MORE INFORMATION

RDS Keywords

Each Knowledge Base article in the RDS collection may contain one or more product-specific keywords that place the article in an appropriate category.

In some cases, you can use the asterisk (*) wildcard to find articles that fall into the general subcategory.

Here are the topics and the corresponding keywords. The minor topics are in the indented list under each major topic. 

   Major Topic
      Minor Topic                                  Keyword
   ----------------------------------------------------------------

   RDS Product Keywords:
      RDS for all Development Environments         rdsall
      RDS within Visual Basic                      rdsvb
      RDS within Visual C++                        rdsvc

   General:
      General RDS Product Info                     rdsGenInfo
      RS Online Documentation                      rdsdoc
      Miscellaneous                                rdsmisc

   Bugs and Fixes:
      Remote Data Service Client 1.0 Bug           rdsClient10bug
      Remote Data Service Client 1.0 Fix           rdsClient10fix
      Remote Data Service Server 1.0 Bug           rdsServer10bug
      Remote Data Service Server 1.0 Fix           rdsServer10fix
      Remote Data Service Client 1.1 Bug           rdsClient11bug
      Remote Data Service Client 1.1 Fix           rdsClient11fix
      Remote Data Service Client 1.5 Bug           rdsClient15bug
      Remote Data Service Client 1.5 Fix           rdsClient15fix
      Remote Data Service Server 1.5 Bug           rdsServer15bug
      Remote Data Service Server 1.5 Fix           rdsServer15fix

Product-Specific Keywords

You can use the keywords to organize RDS articles or to search for specific groups of RDS articles. For information about keywords for other Microsoft developer products, please query the Microsoft Knowledge Base using these keywords: 

   dskbguide and kbkeyword

KB-Wide Keywords

Each article in the RDS collection also contains at least one generic, KB- wide keyword. These keywords are standard throughout the Microsoft Knowledge Base, appearing in all KB articles, regardless of product. You can use these keywords to organize all KB articles or to search for articles across several Microsoft products. For more information about these keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles
