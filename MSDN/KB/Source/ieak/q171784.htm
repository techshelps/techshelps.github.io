Corporate Version Installs Stack and Dialer

 Last reviewed: January 29, 1998
Article ID: Q171784
The information in this article applies to:
  • Microsoft Internet Explorer Administration Kit 3.2a

SUMMARY

When you install a customized Win16 Internet Explorer from floppy disks, a stack and dialer are installed. All other corporate media installations do not install a stack and/or dialer.

MORE INFORMATION

Corporate builds of Internet Explorer are meant to be used on a corporate network and thus there is no need for a stack and dialer to be used for dial-up access to an ISP or network.

The Win16 floppy disk version of a corporate build does install a stack and dialer for those people that need to connect to a network via RAS. The Win32 floppy versions do not install a stack and dialer because they are already a part of the operating system.

To not install a stack and dialer on a Win16 operating system, you must select to do a custom install and clear the option for stack and dialer.
