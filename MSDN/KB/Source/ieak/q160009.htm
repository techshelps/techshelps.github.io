Submit Process Takes Long Time on Windows for Workgroups

 Last reviewed: July 11, 1997
Article ID: Q160009
The information in this article applies to:

- Microsoft Internet Explorer Administration Kit version 3.2a

SYMPTOMS

When you go through the signup process on Microsoft Windows for Workgroups version 3.11, you must click the Submit button in order to finish the signup process. After you click the Submit button, nothing happens for approximately 15-20 seconds. After this time has elapsed, the signup process is done.

CAUSE

This is not reproducible on a 32-bit operating system (OS). This is just the delay that will be seen when you use a 16-bit OS connecting to a signup server.

STATUS

Microsoft has confirmed this to be a problem with Microsoft Internet Explorer Administration Kit version 3.2a. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
