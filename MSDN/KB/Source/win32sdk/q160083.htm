BUG: TabbedTextOut Does Not Work on Middle East Windows 95

 Last reviewed: January 15, 1998
Article ID: Q160083 
4.00    | 4.00
WINDOWS | WINDOWS NT kbprg kbbuglist

The information in this article applies to:

  • Microsoft Win32 Software Development Kit for: - Microsoft Windows NT version 3.51 - Microsoft Windows 95 version 4.0

SYMPTOMS

Under Hebrew/Arabic Win95, TabbedTextOut does not display text from the correct starting x position, especially when the starting x position is not zero (0).

RESOLUTION

Avoid using TabbedTextOut on Middle East (Hebrew/Arabic) Windows 95.

STATUS

Microsoft has confirmed this to be a bug in the Microsoft products listed at the beginning of this article. We are researching this bug and will post new information here in the Microsoft Knowledge Base as it becomes available.
