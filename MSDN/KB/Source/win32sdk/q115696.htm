HOWTO: Search for Win32 SDK Articles Using KBKeywords

 Last reviewed: March 10, 1998
Article ID: Q115696
The information in this article applies to:
  • Microsoft Win32 Software Development Kit (SDK) versions 3.1, 3.5, 3.51, 4.0
  • Microsoft Win32 Application Programming Interface (API)
  • Microsoft Win32s versions 1.25, 1.3, 1.30a, 1.3c

SUMMARY

The Microsoft Knowledge Base (KB) is categorized by using keywords. This article lists the keywords specific to articles in the Microsoft Win32 Software Development Kit (SDK) for Windows NT and Windows 95 Collection.

MORE INFORMATION

Microsoft Win32 SDK Major and Minor KBKeywords

Each KB article in the Win32 SDK collection may contain one or more product- specific keywords (called KBKeywords) that places the article in an appropriate Win32 SDK category. Each KBKeyword is composed of the concatenation of a major topic keyword and a minor topic keyword. For example, you can find all the BASE DLL articles by using BseDll as the keyword when you query the Microsoft Knowledge Base.

Use the asterisk (*) wildcard to find articles that fall into the general categories or into an intermediate subcategory. For example, to find all the articles that apply to GDI issues, query on Gdi*. An article usually has only one KBKeyword, but it may have more.

Product-Specific Keywords

You can use the KBKeywords to organize Win32 SDK articles or to search for specific groups of Win32 SDK articles. For information about KBKeywords for other Microsoft developer products, please query the Microsoft Knowledge Base using these keywords: 

   dskbguide and kbkeyword

KB-wide Keywords

Each article in the Win32 SDK collection also contains at least one generic, KB-wide keyword. The KB-wide keywords are standard throughout the Microsoft Knowledge Base, appearing in all KB articles, regardless of product. You can use the KB-wide keywords to organize all KB articles or to search for articles across several Microsoft products. For more information about these KB-wide keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles
