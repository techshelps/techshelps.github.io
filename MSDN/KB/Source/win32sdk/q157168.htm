BUG: Windows NT Version 4.0 Bug List - Kernel

 Last reviewed: November 25, 1997
Article ID: Q157168
The information in this article applies to:
  • Microsoft Windows NT, version 4.0

SUMMARY

This article lists the bugs in the implementation of the Win32 API on Windows NT 4.0 that may be relevant to Application or Device Driver developers (ISVs or IHVs). This article contains only those bugs that were known at the time of the Windows NT 4.0 release.

MORE INFORMATION

SSPI interface has different .dll names for WinNT and Win95; security.dll and secur32.dll. This should be changed to have one consistent .dll name.

Winlogon.exe displays symptoms of a memory leak.

Using services w/ user logon and Vines hangs system. Turning off the common login feature in the banyan redir works around this problem.

Winlogon does not recognize %Servername% variable when set on domain controller. Works fine if set on the local workstation.

Cannot take ownership of remote files with file manager when user has local admin rights; users access token is not checked completely for suitable owner Sids.

Security log cleared event doesn't have the right computer name for remote clear operations.

Winlogon: should not be able to ALT+TAB between "logging you on" and "main logon" windows.

Grant "Log on locally" right to "Everyone" vs. Guests, Users, etc., in the default Windows NT Server installation.

CACLS utility doesn't correctly handle file and user arguments containing extended characters.

CloseHandle on write-protected floppy causes blue screen in FASTFAT.SYS.

Fat: Async read past end of partition reports the wrong numBytes returned.

International characters dropped in filenames as long file name characters.

NTFS Stream Limits less than NT3.51.

GetProfileString, registry mapped ini fails to remove space.

ScsiPortCompleteRequest appears to abort requests that are posted after the complete request.

Case-sensitive searches unavailable using FindFirstFileEx on remote drives.

wglUseFontOutlines on alpha uses excessive memory.

GetForeGroundWindow returns before window is ready to receive input.

Incorrect rounding of a de-normalized result on axp.

Kernel32 failure after aborted end_session.

Console: ANSI ReadConsole inserts NULL character in buffer.

CreateProcessAsUser() does not treat adjusted quotas in thread-safe manner.

GetExitCodeProcess() always returns 0 from 16-bit apps launched from service.

MoveFileEx() should return FALSE on when attempting inter-machine copy.

Doing a 3-floppy (1.44) install on the LS-120 drives fails with INACCESSIBLE_BOOT_DEVICE.

Mirroring overwrites OEM bytes in boot sector of mirrored drive.

Disk size on the secondary IDE is wrong when a hard disk and CD-ROM are connected to the primary IDE.

PPC: NTBACKUP can't mount tape using Adaptec AHA2940W and HP C1533A tape drive.

Fujitsu M2513 MO drive has cache corruption on Adaptec 2742 SCSI adapters.

DPT SmartCache IV adapters fail scanner test at 10 MB/second transfer rate.

Can't install on Dell OptiPlex 4100/MTe with NEC IDE CD-ROM.

Accessing multiple platters simultaneously on Sanyo 3-CD changer (ATAPI) goes into endless loop.

CPQARRAY: Setup fails if installing on system with multiple PCI buses, and boot partition isn't on PCI bus 0.

Disk Administrator shows incorrect size for drive on secondary Compaq IDA RAID controller.

SCSISCAN: Won't find multiple scanners if multiple adapters are used.

Seagate ST32430N and ST5230N drives hang bus when disk cache enabled.

Crush dump does not work when using AIC-7770 SCSI adapter.

HP Scanners not visible to Deskscan on Buslogic BT-542B and BT-545S SCSI controllers.

Adaptec AHA-174x incompatible with Quantum XP32150 hard disk drives (bus timeouts).

HP Scanjet IIp scanners do not work on Future Domain TMC-860m and TMC-885m SCSI controllers.

Getting unknown Tape Format error when running End-of-Tape test with Sparrow cards on PPC.

NCR/Symbios 8100s adapter fails certain Fault Tolerance tests.

DPT controllers can lose requests when reset during a flush.

DPT 2124W SCSI controller (with cache) for PPC doesn't work with HP SCSI scanner.

ATAPI driver won't find devices attached to the IDE port on Soundblaster boards (SB16 and AWE32).

CDFS_FILE_SYSTEM may bugcheck with ATAPI CD changers while running HCT DISKLOAD.

NCR 53C710 SCSI controller might drop request that had an error.

Sony 6251a cannot access the second CD drive.

SCSIPORT: doesn't free PCI resources if error returned from HwFindAdapter.

SCSIPORT and class drivers don't support mini-port alignment requirements.

SCSI bus rescan does not work if only one device on bus and device powered off at boot up.

AMSINT disk driver async I/O performance degrades when outstanding requests exceed threshold.

SCSIPORT: IOCTL_SCSI_RESCAN_BUS doesn't find new LUNs, only new Targets.

SCSI class drivers might not be able to get the Inquiry data from SCSIPORT.

Buslog 946c causes data overrun on Pentium 133Mhz machine using SCSI Fast Synchronous transfers.

Incorrect error message when executing file on CD-ROM drive playing music.

There currently is no way to do a SCSI bus reset or device reset through a IOCTLs.

System sluggish and unstable when addressing certain devices via Trantor/Media Vision SCSI bus.

CD-ROM changer information is not updated when changing CDs.

Sony CDU77E doesn't recognize audio portion of HCT CD when configured as MASTER.

PCMCIA modem hangs Tadpole machine upon shutdown.

A bad Maxtor 130MB PCMCIA hard disk causes system to hang during boot.

Keyboard driver has problems with the KB 3270 (122-key) keyboard.

IBM Thinkpad 701c with Xircom Pocket Ethernet 3 Adapter causes crash when Parport.sys driver installed.

Event ID 28 in system log from 8042prt.sys on IBM PS/2 and PC Servers.

Unable to use two identical PCMCIA cards at the same time.

Brazilian keyboard layout has two characters not working.

Angia Safejack 28.8 V.34 PCMCIA Modem does not work on desktop.

Conner CTM4000 tape drive fails end-of-tape testing.

Cannot re-tension a Sony SDT-5200 firmware version 3.26 DAT tape.

4mmdat.sys with WangDat 1300Xl does not seek to correct partition.

DEC TLZ06 4MM DAT tape drives containing newer firmware are not recognized by 4mmdat.sys.

DLTTAPE: supports sequential filemarks but FeaturesHigh doesn't report this info.

4mmdat.sys with Archive Turbo Python does not seek to correct partition.

4mmdat Sony SDT-7000 partitioning is not working correctly.

Tandberg TDC 3520 only supports one partition but reports the current partition as 2.

PPTP: Driver does not check TDI interface state before trying to send on connection.

PPTP: Cleanup needs to be done on invalid assertions.

NDISWAN: calling from Eicon/Diehl S2M PRI to Xircom, MCI-23 PRI fails.

IoAllocateIrp() return status not always checked.

DIWAN: Eicon/Diehl still allocates resources after adapter is removed from system.

PCIMAC: Should return zero in mode fields of LINEADDRESSCAPS structure when callstate not supported.

RAS ISDN: Server occasionally thinks client connected after disconnect.
