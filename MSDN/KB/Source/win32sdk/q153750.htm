SAMPLE: DIALUP.EXE: Dial-Up Networking Using Windows 95 Shell

 Last reviewed: July 19, 1996
Article ID: Q153750
The information in this article applies to:
  • Microsoft Win32 Software Development Kit (SDK), version 4.0

SUMMARY

The DIALUP sample demonstrates how to programmatically manipulate Dial-Up Networking items in the Windows 95 shell. The sample shows how to enumerate existing connections, how to programmatically launch the Add New Connection Wizard, and how to create shortcuts to Dial-Up connections and the Add New Connection Wizard on the desktop. This is accomplished by using combinations of shell functions and OLE.

This sample is not compatible with Windows NT 4.0. The RAS APIs are available on Windows NT to perform all of the same functions that this sample demonstrates.
