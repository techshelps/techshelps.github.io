RUN /S Suppresses "Press Any Key to Continue" Message

 Last reviewed: August 28, 1995
Article ID: Q118915
The information in this article applies to:
  • Microsoft FoxPro for UNIX, version 2.6

SUMMARY

In FoxPro for UNIX, RUN /S will suppress the "Press any key to continue" error message on a serial terminal when the information fills more than one screen.

MORE INFORMATION

In the Command window, type "RUN ls -l /" (without the quotation marks). Note that after each screen is filled, the "Press any key to continue" message is displayed. To suppress this message, type "RUN /S ls -l /" (without the quotation marks).
