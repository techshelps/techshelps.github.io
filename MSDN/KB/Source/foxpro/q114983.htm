Wizards and Catalog Manager Cannot Be Distributed

 Last reviewed: June 27, 1995
Article ID: Q114983
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.6
  • Microsoft FoxPro for Windows, version 2.6

SUMMARY

The FoxPro wizards and Catalog Manager cannot be copied or distributed. The wizards and Catalog Manager are governed by the terms of the "Microsoft License Agreement" that is included with all Microsoft FoxPro version 2.6 for Windows and Microsoft FoxPro version 2.6 for MS-DOS.

MORE INFORMATION

The wizards and Catalog Manager are provided with both the Standard Edition and Professional Edition of FoxPro version 2.6 for Windows and FoxPro version 2.6 for MS-DOS. The wizards and Catalog Manager are considered to be part of FoxPro.

If an application is written in FoxPro and that application is distributed by providing source code or compiled code, using the run-time modules, building an .EXE file, or by any other means, the application must not include any of the wizard or Catalog Manager files. The list of files that can be distributed when the Distribution Kit is included in the "User's Guide" for the Distribution Kit. It is a violation of the Microsoft License Agreement to obtain or distribute any of the wizards or the Catalog Manager by any means other than by purchasing a copy of FoxPro version 2.6 for Windows or FoxPro version 2.6 for MS-DOS or as otherwise provided for in the Microsoft License Agreement.
