Maximum Table and File Sizes for Visual FoxPro

 Last reviewed: April 30, 1996
Article ID: Q130445
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

The maximum size of a free table (.DBF file) in Visual FoxPro is one gigabyte. This holds true for all operating systems (Windows, version 3.x, Windows 95, and Windows NT).

The maximum size of a file in Windows 95 is 2 gigabytes, because it uses the File Allocation Table. Windows NT supports much larger file sizes. However, Visual FoxPro does not take advantage of these capabilities because they were not part of the original design specification for Visual FoxPro.
