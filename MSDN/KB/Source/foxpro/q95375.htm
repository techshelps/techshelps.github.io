DEFINE WINDOW STYLE 'O' Clause Does Not Work

 Last reviewed: January 24, 1996
Article ID: Q95375
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6x

Page L3-431 of the FoxPro 2.x and page L3-25 of the Visual FoxPro "Language Reference" incorrectly describe the STYLE clause of the DEFINE WINDOW command. This page lists an outline font style ("O"); however, this font style is not available.
