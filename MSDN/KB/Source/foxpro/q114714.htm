BUG: "Position Is off the Screen" When Running Table Wizard

 Last reviewed: June 27, 1995
Article ID: Q114714
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6

SYMPTOMS

Running the Table Wizard causes the following FoxPro error message: 

   Line Number: 224
   Program: WIZ_TBL3
   Error: Position is off the screen.
   Source: @...

CAUSE

The ATI Ultra Windows 3.1 800 x 600 16/256 video driver is in use and conflicts with the Table Wizard. The file for this driver is ULTRA800.DRV. Its file size, date, and time are 96,624 bytes, 8/14/92, and 1:42:28 P.M.

WORKAROUND

To correct this problem:

  • Install one of the standard Microsoft video drivers that ship with Windows. These include drivers whose names begin with the text "VGA" or "Super VGA"

    -or-

  • Contact the video card manufacturer or vendor for a more recent version of the video driver.

STATUS

Microsoft has confirmed this to be a problem in FoxPro version 2.6. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. From the Run menu, choose Wizard. Then choose Table.

  2. Under Fields, double-click AccountID, moving AccountID to the Selected Fields list box. Choose Finish.

    Note that none of the push buttons in the dialog box can be chosen.

    Also, note that only the following radio buttons are displayed: 

        - Save Table
    - Save And Browse For Data Entry

    The following radio buttons are not displayed: 

        - Save And Create An AutoScreen For Data Entry
    - Modify Structure Of Table

  3. From the Control-menu box (in the upper-left corner) of the dialog box, choose Close.

The following FoxPro error message is now visible: 

   Line Number: 224
   Program: WIZ_TBL3
   Error: Position is off the screen.
   Source: @...
