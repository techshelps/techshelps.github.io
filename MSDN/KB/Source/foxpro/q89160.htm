Memory & Disk Space FoxPro 1.x Installation Requirements

 Last reviewed: April 17, 1995
Article ID: Q89160
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 1.0, 1.01, and 1.02

FoxPro versions 1.x for MS-DOS require 470K of free memory to load the single-user version. The multiuser version requires 460K of free memory.

FoxPro 1.x for MS-DOS requires 1.5 MB of free disk space to complete a basic installation that does not include FOXVIEW, FOXCODE, FOXDOC, or FOXBIND. The complete installation (basic and supplemental programs) requires 3 MB.

Additional reference words: FoxDos 1.00 1.01 1.02 1.x multi-user
