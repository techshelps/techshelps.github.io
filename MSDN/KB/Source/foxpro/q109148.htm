How to Transport Fox Macintosh Files to Windows or MS-DOS

 Last reviewed: April 30, 1996
Article ID: Q109148
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.5b
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, and 2.5b
  • Microsoft FoxPro for MS-DOS, versions 2.5, 2.5a, and 2.5b

To transport files from FoxPro for Macintosh to either FoxPro for MS-DOS or FoxPro for Windows, do the following:

  1. Copy the TRANSPRT.PRG program file from the FoxPro for Macintosh folder to the FoxPro for MS-DOS or FoxPro for Windows directory. This file will replace the version of the Transport program that is currently in that directory.

  2. Delete the TRANSPRT.FXP file that is located in the FoxPro for MS-DOS or FoxPro for Windows directory. When the Transport program is called, it will compile a new TRANSPRT.FXP file.
