The information in this article applies to:

SUMMARY

Fp0660.exe is a file that contains an updated copy of Microsoft FoxDoc that corrects two problems. Microsoft FoxDoc is a code-documentation utility distributed with Microsoft FoxPro for MS-DOS.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library:

~ Fp0660.exe

For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
The most recent version of FoxDoc, version 2.1e, corrects the following two problems in earlier versions of FoxDoc:
  • If you attempt to run FoxDoc from a network directory other than the directory in which FoxDoc is installed, a "FOXDOC.MSG file cannot be found" error occurs.
  • If the path to a file is specified using a macro substitution, FoxDoc appends a period to the generated filename and displays a "Program file not found" error message.

Microsoft FoxDoc is a code-documentation utility distributed with Microsoft FoxPro for MS-DOS. The most recent version of FoxDoc, version 2.1e, which is included on the enclosed FP0660 disk, corrects the following two problems in earlier versions of FoxDoc:
  • If you attempt to run FoxDoc from a network directory other than the directory in which FoxDoc is installed, a "FOXDOC.MSG file cannot be found" error occurs.
  • If the path to a file is specified using a macro substitution, FoxDoc appends a period to the generated filename and displays a "Program file not found" error message.

To update your copy of FoxDoc

  1. Copy the Foxdoc2.exe file from the enclosed FP0660 disk to the directory where FoxPro is installed.

  2. From the MS-DOS prompt, run the Foxdoc2.exe file with the -D and -O parameters, as follows: 

          foxdoc2 -d -o

When the Foxdoc2.exe file runs, it expands into the following seven files that overwrite any previous copies of the files in the appropriate directories: 

  Files          Description
  -------------------------------------------------------------------

  Foxdoc.exe     FoxDoc application file
  Foxdoc.msg     FoxDoc error message file
  Foxdoc.ovl     FoxDoc overlay file
  Fx2words.fxd   FoxDoc data file that contains FoxBASE(R) keywords
  Fxdprt.fxd     FoxDoc data file that contains installed printer
                 drivers
  Fxpwords.fxd   FoxDoc data file that contains FoxBASE+(R) keywords
  Prowords.fxd   FoxDoc data file that contains FoxPro keywords
