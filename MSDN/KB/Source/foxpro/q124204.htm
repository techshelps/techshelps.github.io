FIX: Splash Screen & Setup Display Version 2.6 Instead of 2.6a

 Last reviewed: September 22, 1997
Article ID: Q124204
2.60a MACINTOSH kbsetup kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

The setup program displays version 2.6 instead of version 2.6a in the following places:

  • The Introductory (also known as the splash) screen that appears briefly when FoxPro starts. It displays "Microsoft FoxPro 2.6 Setup."
  • The first setup dialog box, which displays the following text: 

          Welcome to the Microsoft FoxPro 2.6 Setup Program.


          Setup will install Microsoft FoxPro version 2.6 on you
      Macintosh..."
  • The title bars of all setup dialog boxes, which contain the text "Microsoft FoxPro 2.6 Setup."

RESOLUTION

Perform one of the following actions to correctly determine the version of FoxPro:

  • From the Apple menu, choose About Microsoft FoxPro.

    -or-

  • In the Command window, type "VERSION()" (without the quotation marks).

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Visual FoxPro 3.0b for Macintosh.
