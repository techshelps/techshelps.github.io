PRB: Unable to Select Printer Driver Check Box

 Last reviewed: April 17, 1995
Article ID: Q87689
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5 and 2.5a

SYMPTOMS

In Microsoft FoxPro, it is not possible to select the Printer Driver Setup check box.

CAUSE

The GENPD.APP file is missing or the _GENPD memory variable is not set properly.

RESOLUTION

To address this situation, do the following:

  1. Verify that the GENPD.APP file exists on your hard disk.

  2. In the Command window, issue the following command:

    _GENPD="<drive>:\<path>\GENPD.APP"

    (For example: _GENPD="C:\FOXPRO2\GENPD.APP")

  3. To make the change permanent, modify the CONFIG.FP file to define the _GENPD variable appropriately.
