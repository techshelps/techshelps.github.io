BUG: Name Edit Region Not Displaying Correctly in RQBE

 Last reviewed: April 29, 1996
Article ID: Q103596
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a

SYMPTOMS

When you are modifying a query that is going to a report or label, the Name edit region just below the Output popup appears for a split second before disappearing.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
