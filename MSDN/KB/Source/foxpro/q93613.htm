Using the FoxPro 2.0 Menu Bar in FoxPro 2.5 for Windows

 Last reviewed: June 27, 1995
Article ID: Q93613
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

You can use the FoxPro version 2.0 for MS-DOS menu bar in FoxPro version 2.5 or 2.5a for Windows by running the DOS.MPR program. The DOS.MNX file is installed in the FoxPro GOODIES\DOS subdirectory (by default, the path is C:\FOXPROW\GOODIES\DOS). You must open DOS.MNX as a menu and then generate the menu program.

The DOS.MPR menu bar differs from the FoxPro 2.0 for MS-DOS menu bar in two respects: it has a Text menu and it does not have a Run menu.

To access the Run menu, use FOXSTART.APP. To return to the default FoxPro for Windows menu bar, issue the SET SYSMENU TO WINDOWS command.

If a "Variable 'windows' not found" error message occurs, press any key to continue and FoxPro will display the default Windows menu. The variable used in the SET SYSMENU TO command can be any text.
