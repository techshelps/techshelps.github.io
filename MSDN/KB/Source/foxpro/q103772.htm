How to Change Keyboard Shortcuts in FoxPro for Windows

 Last reviewed: June 27, 1995
Article ID: Q103772
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

The keyboard shortcuts in FoxPro for Windows differ from those in FoxPro for MS-DOS because FoxPro for Windows uses the Windows-standard keystrokes--for example, pressing ENTER selects the option that has the focus.

FoxPro for Windows supports both an MS-DOS-style keystroke mode and a Windows-style keystroke mode. (At installation time, you can choose "Windows-standard" or "DOS-style" keystrokes.) For details on the keystroke differences, refer to the FoxPro online Help file.

You can change this mode at any time for a particular FoxPro session by issuing the appropriate command shown below: 

   SET KEYCOMP TO DOS

   -or-

   SET KEYCOMP TO WINDOWS
Note that each time FoxPro opens anew, the default KEYCOMP setting is reinvoked. To permanently change the KEYCOMP setting, add one of the following statements to the CONFIG.FPW file: 

   KEYCOMP=DOS

   -or-

   KEYCOMP=WINDOWS
