dBASE and FoxPro Keyboard Differences Not Listed

 Last reviewed: June 27, 1995
Article ID: Q119852
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6
  • Microsoft FoxPro for MS-DOS, version 2.6

SYMPTOMS

The differences in keyboard assignments for a Browse window in dBASE and FoxPro aren't listed in the online Help files for FoxPro 2.6 for Windows and MS-DOS. This error has been corrected in the online Help files for FoxPro 2.6a for Windows and MS-DOS.

MORE INFORMATION

The FoxPro 2.6a online Help system contains a table of dBASE actions for the function keys when a Browse window is active. These actions are not supported in FoxPro.

To go directly to this documentation, choose dBASE Help from the Help menu. Then do one of the following, depending on which online Help file is in use:

  • FOXHELP.HLP: Choose "Keyboard Differences"
  • FOXHELP.DBF: Scroll to the "Keyboard Differences" heading, located near the bottom of the topic.
