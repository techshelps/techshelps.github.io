PRB: Unable to Use Genius Mouse with FoxPro for MS-DOS

 Last reviewed: April 17, 1995
Article ID: Q86915
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.0

SYMPTOMS

Mouse problems occur when you are running Microsoft FoxPro for MS-DOS with a Genius mouse in Microsoft Mouse emulation mode.

RESOLUTION

Perform the following two steps to ensure that the mouse is configured properly in Microsoft emulation mode:

  1. Press and hold down any mouse button when you first turn on your computer. This sets the mouse to Microsoft emulation mode.

  2. Install and load the latest Microsoft Mouse driver before running FoxPro. The MMOUSE.COM program in the Goodies directory (by default, C:\FOXPRO2\GOODIES) is provided for this purpose.

MORE INFORMATION

The Genius mouse can emulate either a Mouse System or Microsoft mouse. The GMOUSE driver included with the Genius mouse does not work properly with FoxPro.
