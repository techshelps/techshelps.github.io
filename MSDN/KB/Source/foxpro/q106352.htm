PRB: Cannot Enter Value in Spinner Text Box

 Last reviewed: June 27, 1995
Article ID: Q106352
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6

On page U11-45 of the FoxPro for Windows "User's Guide," there are two documentation errors under "Spinner Range." Under "Lower...," the manual states, "You can also directly enter the value in the text box beside the push button."

This is incorrect. The Lower push button must be chosen first before you can enter the expression or procedure in the Code Snippet dialog box to set the lowest value. This error also appears under "Upper...." The Upper push button must first be chosen before you can enter the upper value.
