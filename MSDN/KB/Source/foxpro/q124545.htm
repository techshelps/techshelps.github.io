DOCERR: Installation and Configuration Manual Doc Error

 Last reviewed: March 15, 1995
Article ID: Q124545
The information in this article applies to:
  • Microsoft FoxPro for UNIX, version 2.6

Page 26 of the Installation & Configuration Manual for FoxPro for UNIX points the reader to Chapter 19 of the FoxPro Developers Guide for information on transporting files across platforms. This is an error in the documentation. There is no Chapter 19 of the FoxPro Developer's Guide for UNIX. The only place this information exists is in the FoxPro Developer's Guide for Macintosh.

For more information on transporting files across platforms, refer to the online Help topic: Transporting Files
