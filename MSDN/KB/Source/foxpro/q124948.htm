PRB: Incorrect System Date Stops Wizard Execution

 Last reviewed: February 21, 1995
Article ID: Q124948
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.6, 2.6a

SYMPTOMS

If FoxPro determines that the system date on the computer is incorrect, it displays a WAIT WINDOW with the following text: 

   The system date on your computer is inaccurate. Please reset it.
Pressing any key causes FoxPro to return to the Command window. You will not be able to run any wizards until the date has been correctly set.

WORKAROUND

Reset the date to the current day and date using the Windows Control Panel program.

STATUS

This behavior is by design.

MORE INFORMATION

Steps to Reproduce Problem

  1. Double-click the Windows Control Panel program, , located in the Main Program Group.

  2. Double-click the Date/Time icon, then set the Date to a year several years in the past.

  3. Start FoxPro and attempt to run a Wizard.

NOTE: This problem occurs with dates greater than or equal to 1980.
