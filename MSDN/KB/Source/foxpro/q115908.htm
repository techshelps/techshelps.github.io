Installation Dialog Box Doesn't Give Disk Space

 Last reviewed: June 27, 1995
Article ID: Q115908
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6

On page 1-1 of the "Installation and Configuration" manual, under "System Requirements, is the following text: 

   The hard disk space requirements for specific installation
   types are given in the Installation Type dialog
This is incorrect. The installation dialog box, titled, "Choose the type of installation by clicking one of these buttons:", does not list the hard disk space requirements.

NOTE: The dialog box displayed if you choose a Custom installation does list the requirements for each individual FoxPro component. However, none of the dialog boxes displayed if you choose Complete give the hard disk space requirements.
