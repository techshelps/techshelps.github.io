Array Not in Previous Example of @ ... GET-List

 Last reviewed: June 27, 1995
Article ID: Q95711
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

On page L3-80 of the "Language Reference" manual, the "Color Scheme" section of the @ ... GET-List command incorrectly mentions that the next two examples use an array created in the previous example. However, the previous example for this command does not define or use an array. An array is not needed for these examples because the list is created from a popup.
