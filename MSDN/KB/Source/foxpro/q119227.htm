DOWN ARROW Missing from dBASE Keyboard BROWSE Differences

 Last reviewed: June 27, 1995
Article ID: Q119227
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6a
  • Microsoft FoxPro for MS-DOS, version 2.6a

SUMMARY

The description in the FOXHELP.DBF online Help file of the dBASE Browse window function key assignments that are not supported in FoxPro doesn't include the DOWN ARROW key.

The FoxPro for Windows FOXHELP.HLP online Help file is correct. It includes the description of the DOWN ARROW key, as shown in the following table: 

   dBASE Browse Window Function Key Assignments
   (Not supported in FoxPro)
   --------------------------------------------

   Key           Action
   --------------------------------------------------------

   F1           Activate online help
   F2           Toggle between Browse and Edit mode
   F3           Previous record
   F4           Next record
   F5           No effect
   F6           Extend select (select text in a memo field)
   F7           Move (move selected text in a memo field)
   F8           Copy (copy selected text in a memo field)
   F9            Zoom memo field window to full screen
   F10           Activate the Browse menu bar
   DOWN ARROW    Activate the append record prompt

MORE INFORMATION

Location of Documentation Error

To go directly to this documentation error, do one of the following, depending on which FoxPro product you are using:

  • FoxPro for Windows: Use the SET HELP TO command to use the FOXHELP.DBF help file. Choose dBASE Help from the Help menu. Scroll to the "Keyboard Differences" heading, located near the bottom of the topic.
  • FoxPro for MS-DOS: Choose dBASE Help from the System menu. Scroll to the "Keyboard Differences" heading, located near the bottom of the topic.
