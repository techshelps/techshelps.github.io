Disk Space Requirements of FoxPro 2.5c for Macintosh

 Last reviewed: May 16, 1996
Article ID: Q116021
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.5c

SUMMARY

Listed below are the free disk (volume) space requirements in megabytes (MB) for complete and minimum installations of FoxPro for Macintosh.

MORE INFORMATION

Complete Installation

   Component                                         MB
   ------------------------------------------------------

   FoxPro and All Supplemental Files                 18.7
   Apple Shared Library Manager (ASLM)                0.2
   Fonts                                              1.1

   TOTAL (FoxPro, Supplemental Files,                ====
          and System files):                         20.0

Minimum Installation

   Component                                         MB
   -----------------------------------------------------

   FoxPro Only                                       4.4
   Apple Shared Library Manager (ASLM)               0.2
   Fonts                                             1.1
                                                     ====
   TOTAL (FoxPro and System Files)                   5.7
