Files Installed During FoxPro Ver 2.5 for MS-DOS Setup

 Last reviewed: April 17, 1995
Article ID: Q98358
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a

SUMMARY

Below is a list of the directories installed with FoxPro for MS-DOS and their sizes. A Library Construction Kit (LCK) update is shipped to update the libraries (.PLB files) from FoxPro version 2.0 for MS-DOS to FoxPro version 2.5 for MS-DOS.

NOTE: The FOXAPP directory with all the source code is always installed, independent of what is selected. For example, if the help file is selected to be installed, FoxApp will also be installed.

MORE INFORMATION

Basic Installation

   16-bit, 32-bit, and Help file:

   FOXPRO root directory      5,791,744 bytes
   FOXAPP                            688,128 bytes

   16-bit product only:

   FOXPRO root directory           3,234,630 bytes

   32-bit product only:

   FOXPRO root directory      3,350,210 bytes

   Help file only:

   FOXPRO root directory           3,929,007 bytes

Optional Installation

   FOXAPP                            688,128 bytes

      HELPTREE                        57,344 bytes

      SNIPFMT                        114,688 bytes
      LANGUAGE                       245,760 bytes
   RUN                                90,112 bytes
