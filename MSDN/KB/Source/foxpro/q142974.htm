SUMMARY

View.exe is a sample that provides an introduction to the concepts of parameterized views and how they can be applied in Visual FoxPro.

This document provides a basic understanding and overview of views, parameterized views, the flexibility and power of using views, and a practical application of these concepts.

The material explains how to control the display of related data using parameterized views and gives related step-by-step procedures that guide you through the process of creating and using parameterized views.
