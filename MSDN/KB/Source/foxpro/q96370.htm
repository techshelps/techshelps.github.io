FoxPro 2.5 Databases with General Fields in FoxPro 2.0

 Last reviewed: April 17, 1995
Article ID: Q96370
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.0

SUMMARY

The database structure in Microsoft FoxPro version 2.5 for Windows has been changed to allow the use of a general field. Because of this change, using version 2.5 database files that contain general fields in FoxPro version 2.0 for MS-DOS is not recommended.

MORE INFORMATION

FoxPro 2.0 can use databases created with FoxPro 2.5 that have general fields. The database structure for a general field will show as (UNKNOWN) in FoxPro 2.0. However, when the structure of the database is modified or when the general fields are referenced from within FoxPro 2.0, an "Internal Consistency Error" could occur.
