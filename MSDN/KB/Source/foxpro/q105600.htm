PRB: Cannot Move Main FoxPro for Mac Screen

 Last reviewed: May 17, 1996
Article ID: Q105600
The information in this article applies to:
  • Microsoft FoxPro for the Macintosh, version 2.5b

SYMPTOMS

When FoxPro starts, the main FoxPro screen is positioned in such a way that it cannot be moved because the title bar is hidden.

CAUSE

This behavior occurs because the FoxPro screen was redefined or moved so that the title bar became hidden.

RESOLUTION

To correct this problem, do one of the following:

  • Put the System Folder:Preferences:FoxPro Settings file in the Trash and then restart FoxPro. Doing this returns the position of the main FoxPro screen to its default size and position.

    -or-

  • Execute the ZOOM WINDOW SCREEN MAX command, and then resize the window.
