Font Dialog Box Behavior in FoxPro for Windows

 Last reviewed: June 26, 1995
Article ID: Q99135
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SUMMARY

When the Font dialog box is invoked, FoxPro for Windows calls the Windows Font Manager. The Windows Font Manager then presents a list of fonts and sizes.

The Font Manager will produce new font sizes when they are available. For example, you can select a font like FoxFont and type in a large size that is not on the list, such as 100, and Windows will resize the font to that point size.

MORE INFORMATION

Changing the font size sometimes does not change the font's appearance. For example, if you are using a font like Fixedsys, where the only size given is 9, changing the size to 8, 11, or even 15 does not change the physical appearance of the text on-screen. The font's on-screen appearance is the best approximation of the font that the Font Manager has available. However, despite its on-screen appearance, the font should print out in the correct size.
