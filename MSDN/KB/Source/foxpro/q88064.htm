FIX: "Program File Not Found" Error in FoxDoc

 Last reviewed: October 14, 1997
Article ID: Q88064
2.00 MS-DOS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, version 2.0

SYMPTOMS

In releases of Microsoft FoxPro version 2.0 for MS-DOS prior to the March 25, 1992 release, the error "Program file not found" occurs in FoxDoc.

CAUSE

Macro substitution is used to specify a path for FoxDoc. FoxDoc appends a period to the generated pathname.

STATUS

Microsoft has confirmed this to be a problem in Microsoft FoxPro version 2.0 for MS-DOS releases dated prior to March 25, 1992.

Specify the path without macro substitution to avoid this error with versions prior to the March 25, 1992 release or obtain the latest release to use the macro substitution.

This problem was corrected in the Microsoft FoxPro version 2.0 for MS-DOS release dated March 25, 1992.
