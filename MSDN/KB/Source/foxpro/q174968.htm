INFO: Programmer's Guide to Using ADO in Visual FoxPro

 Last reviewed: October 10, 1997
Article ID: Q174968
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0, 5.0a

SUMMARY

This article presents useful information for anyone wanting to take advantage of the ActiveX Data Objects (ADO) within Visual FoxPro 5.0. The following topics are covered:

  • How to find ADO Knowledge Base Articles
  • ADO Resources/Points of Information

MORE INFORMATION

How to Find ADO Knowledge Base Articles

ADO is not a technology that ships with one particular product, and it has been treated as a product in its own right. To find ADO articles in the Microsoft Knowledge Base on the Web, connect to the following: 

   http://support.microsoft.com/support/
Select "ActiveX Data Objects" as the product.

For the MSDN or other non-Web-based Knowledge Base search mechanisms, you can use the keyword adobj, which all ADO articles contain. Other keywords that offer greater granularity when searching are documented in the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q168337
   TITLE     : HOWTO: Searching for ADO Articles by Keyword

ADO Resources/Points of Information

ADO originally shipped with the OLE-DB Software Development Kit (SDK), so you will want to visit both the OLE-DB Web page and the ADO Web site: 

   http://www.microsoft.com/ado/
   http://www.microsoft.com/oledb/

REFERENCES

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q165492
   TITLE     : HOWTO: Using ADO with a Visual FoxPro Database
