PRB: Object Order Reverses in Screen Builder

 Last reviewed: April 18, 1995
Article ID: Q106491
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a, and 2.5b

SYMPTOMS

When the following steps are taken in the Screen Builder, the fields are not only sent to the back but their order is changed also:

  1. Use the CUSTOMER.DBF file in the TUTORIAL subdirectory.

  2. From the File menu, choose New Screen.

  3. From the Screen menu, choose Quick Screen and then OK.

  4. From the Edit menu, choose Select All.

  5. From the Screen menu, choose Send To Back.

  6. From the Edit menu, choose Select All.

CAUSE

This behavior is by design. Changing the field order in FoxPro for MS-DOS is accomplished by choosing Send To Back or Bring To Front.
