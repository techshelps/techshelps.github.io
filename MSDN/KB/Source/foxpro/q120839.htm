"Microsoft System Information" Utility Included on Disk 1

 Last reviewed: May 29, 1996
Article ID: Q120839
The information in this article applies to:

- Microsoft Visual FoxPro for Macintosh, version 3.0b - Microsoft FoxPro for Macintosh, version 2.6a

SUMMARY

The "Microsoft System Information" utility is included on Installation Disk 1. It reports extensive information about your Macintosh. This utility is also included with Visual FoxPro for Macintosh, version 3.0b

Click the utility's Save button to save a full report of all system information in a TeachText document.

MORE INFORMATION

The utility has a list box that contains four different areas: System, Disk Storage, Fonts, and System Extensions. The utility displays information for each area selected.

NOTE: You can also run this utility by double-clicking the MSInfo file, which is a "Microsoft System Information" document file located on Installation Disk 1.
