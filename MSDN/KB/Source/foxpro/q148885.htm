Visigenic Oracle ODBC Driver Support in Visual FoxPro

 Last reviewed: October 17, 1996
Article ID: Q148885
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0

SUMMARY

Visual FoxPro 3.0b ships with an Oracle ODBC driver that was provided by Visigenic Corporation. This is a 32-bit driver that will be installed with Visual FoxPro on Windows 95 and Windows NT. This driver ships with many other 32-bit Microsoft products and is also available for download.

For more information on how to obtain this driver, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q140255
   TITLE     : GE1263: Microsoft ODBC Drivers for 32-Bit Programs

MORE INFORMATION

Windows 95

As with all previous versions of the Oracle ODBC drivers, the Visigenic driver requires that the appropriate Oracle client software (SQL*Net) be installed on the system before the driver will work properly. Currently, Oracle does not have a version of SQL*Net that is supported on Windows 95; therefore the Visigenic driver is not supported on Windows 95.

Visigenic Corporation has identified a workaround for using their Oracle ODBC driver with Windows 95. For more information about this possible workaround, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q140498
   TITLE     : INF: Using the Visigenic Oracle Driver on WinNT & Win95

Windows NT

Under Windows NT, the Visigenic driver will work with either the 1.1x or the 2.1x versions of the Oracle 32bit SQL*Net libraries.
