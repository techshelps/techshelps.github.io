Object Menu in FoxPro for Windows, Not FoxPro for MS-DOS

 Last reviewed: April 29, 1996
Article ID: Q103775
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

The Object menu in FoxPro for Windows is available when either the Screen Builder or the Report Writer is open. As with the Text and Browse menus, the Object menu has access to fonts.

Specific items on the Object menu are enabled and disabled based on the currently selected object. For example, a rectangle drawn in the Screen Design window may have an outline or pen color, a background or fill color, and a fill pattern.
