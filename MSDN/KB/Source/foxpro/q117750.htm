Answer #2 of DKREADME.TXT Should Say FOXW2600.ESL

 Last reviewed: April 30, 1996
Article ID: Q117750
The information in this article applies to:
  • Microsoft FoxPro for Windows Distribution Kit, version 2.6

SUMMARY

In the DKREADME.TXT file, located in the main FoxPro directory (usually FPW26) and on Disk 1 of the Distribution Kit, the answer to Question 2 should refer to FOXW2600.ESL, not FOXW2500.ESL.

MORE INFORMATION

Part 5 of the DKREADME.TXT contains answers to commonly asked questions. Question 2 asks the following: 

   If my application uses the RUN command to execute external MS-DOS
   programs, where should I put FOXRUN.PIF?
The answer incorrectly states the following: 

   Place FOXRUN.PIF in the same directory as the FOXW2500.ESL file.
This answer should actually say to place FOXRUN.PIF in the same directory as FOXW2600.ESL, the support file included with FoxPro version 2.6 for Windows Distribution Kit.
