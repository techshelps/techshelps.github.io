How to Disable Snap to Grid Feature When Aligning Objects

 Last reviewed: June 27, 1995
Article ID: Q96345
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SUMMARY

To align objects without having them automatically aligned to the nearest grid lines in a screen form or report form, choose the Snap To Grid command on the Screen or Report menu. You can also disable the snap to grid feature by holding down the CTRL key while dragging an object to the desired position with the mouse.

MORE INFORMATION

By default, objects are automatically aligned to the nearest invisible grid lines within the Screen Design or Report Layout window. This feature is available only in the FoxPro for Windows.
