Differences Between FoxPro Platforms Described in Help

 Last reviewed: June 27, 1995
Article ID: Q99091
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a

Both the MS-DOS-style Help file, FOXHELP.DBF, and the Windows-style Help file, FOXHELP.HLP, include a section titled "Differences Between FoxPro for MS-DOS and Windows."

This section is broken into the following subtopics:

  • Features Unique to FoxPro for Windows
  • Online Help
  • Interface
  • Language
  • Text Editor
  • Power Tools
  • Keystroke Differences
  • Configuration File

NOTE: There is also a section called "Changes Since 2.0" in the FoxPro 2.5 for MS-DOS Help file. It explains differences between FoxPro 2.0 and 2.5 for MS-DOS.
