PRB: Visual FoxPro Does Not Run Under OS/2

 Last reviewed: May 16, 1996
Article ID: Q132416
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

SYMPTOMS

Attempting to run Visual FoxPro under OS/2 version 2.1 or OS/2 Warp version 3.0 generates the following Win32s errorS: 

   Missing Numeric CO-Processor Virtual Device Driver (VMCPD)

   Unexpected DOS Error:20

MORE INFORMATION

Visual FoxPro version 3.0 is designed to run under Win32s, Windows 95, and Windows NT.

NOTE: Visual FoxPro for Windows requires version 1.25.142.0 or later of Win32s.
