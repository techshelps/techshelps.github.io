Parts of FoxPro for Macintosh That Require FOXTOOLS.MLB

 Last reviewed: May 29, 1996
Article ID: Q111875
The information in this article applies to:

- Microsoft FoxPro for Macintosh, versions 2.5b, 2.6

The following FoxPro for Macintosh add-on components require access to FOXTOOLS.MLB: 

   FB+MIGRATE.APP
   FOXDOC.APP
   WIZARD.APP, WZ_FORM.APP, WZ_REPORT.APP
   Distribution Kit SetupWizard
   Spelling checker
Since FOXTOOLS.MLB is used by all these applications, it is automatically installed in the System Folder:Extensions folder during FoxPro's installation.

If FOXTOOLS.MLB becomes unavailable, errors will result when the components listed above are used.
