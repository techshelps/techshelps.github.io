HOWTO: Search for FoxPro Articles Using KBKeywords

 Last reviewed: March 10, 1998
Article ID: Q122791
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 5.0
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6x
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5x, 2.6x
  • Microsoft FoxPro for Macintosh, versions 2.5x, 2.6a
  • Microsoft Visual FoxPro for Macintosh, version 3.0b
  • Microsoft FoxPro for UNIX, version 2.6

SUMMARY

This article lists the standard FoxPro-specific query keywords. You can use these keywords to simplify and increase the accuracy of your queries when searching for FoxPro information in the Microsoft Knowledge Base.

MORE INFORMATION

Products and Platforms: 

   Keyword       Description
   ------------------------------------

   VFoxWin       Visual FoxPro for Windows
   VFoxMac       Visual FoxPro for Macintosh
   FoxWin        FoxPro for Windows
   FoxDos        FoxPro for MS-DOS
   FoxMac        FoxPro for Macintosh
   FoxUnix       FoxPro for UNIX
   LCK           Library Construction Kit
   DK            Distribution Kit
   CK            Connectivity Kit
Tools: 

   Keyword       Description
   -----------------------------

   CatMan        Catalog Manager
   SBuilder      Screen Builder
   MBuilder      Menu Builder
   RWriter       Report Writer
   LWriter       Label Writer
   ProjMan       Project Manager
Other: 

   Keyword       Description
   ---------------------------------------------------------------------

   XPlatForm     Cross-platform issues.
   FIXLISTx.xx   Bug fixed in the specified version. For example,
                 use fixlist2.60a for a bug fixed in version 2.60a.
   BUGLISTx.xx   Bug in the specified version. For example, use
                 buglist2.60a for a bug in version 2.60a.
   KBMaster      Reference listing of article ID numbers (Q numbers) and
                 titles of articles that discuss bugs and fixes.
   DiskDir       Directory listing.
   SoftLib       Microsoft Software Library files (can also use KBFILE).
