PRB: Illegal Operation for General Field with Report Writer

 Last reviewed: June 27, 1995
Article ID: Q123949
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6, 2.6a

SYMPTOMS

The following error message is displayed when you run or preview a report: 

   Illegal operation for general field

CAUSE

One or more of the Report Expressions in the report reference a general field. Although general fields are available as Report Expressions through the Report Writer Expression Builder interface, general fields cannot be used in a Report Expression. When you execute the report by previewing or printing it, you'll see the error message.

RESOLUTION

Remove all Report Expressions that have a reference to a general field. To display an image stored in a general field, use the Picture tool to add a Report Picture from Field. Then select the general field for display.

STATUS

This behavior is by design.
