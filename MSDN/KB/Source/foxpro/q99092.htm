Location of FoxPro for Windows Screen Generator Options

 Last reviewed: April 29, 1996
Article ID: Q99092
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SUMMARY

To access the FoxPro for Windows screen code generator options, such as Open Files and Close Files, click the More push button in the Generate Screen dialog box.

MORE INFORMATION

In FoxPro version 2.0 for MS-DOS, the Screen Builder dialog box contains options to tell the generator exactly what code to generate in the .SPR file. These options include opening and closing files and defining and releasing windows and modal screens.

In FoxPro for Windows, you can access these generator options by clicking the More button in the Generate Screen dialog box. When you click this button, the dialog box will expand to display the code generator options.

This design change was implemented to make the dialog box less cluttered and confusing.
