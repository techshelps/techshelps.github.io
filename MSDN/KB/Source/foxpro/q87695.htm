PRB: Unable to Build Executable File from Project

 Last reviewed: April 17, 1995
Article ID: Q87695
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.0

SYMPTOMS

The option to build an executable file from a project is unavailable.

CAUSE

Either one of the following causes this problem:

  • You do not have the FoxPro Distribution Kit (sold separately), which is required to build an executable file.

    OR

  • Your project does not include a SET MAIN program.

RESOLUTION

The two solutions below correspond to the two causes in the previous section:

  • Install the FoxPro Distribution Kit in your FOXPRO subdirectory. Verify that the FoxPro files are not marked as read-only before installing the FoxPro Distribution Kit.

    OR

  • Select the main menu or program, then choose Main from the Project menu.
