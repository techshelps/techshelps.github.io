Browse Flicker in FoxPro for Windows Under Windows 3.0

 Last reviewed: June 27, 1995
Article ID: Q104353
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

When FoxPro for Windows is running under Windows 3.0, the Browse window flickers when it first comes up. This flicker is not as apparent under Windows 3.1.

The flicker occurs because the screen is repainted twice on the initial display of a Browse window. This behavior is more apparent in Windows 3.0 than in Windows 3.1 because:

  • Windows 3.0 is slower overall than Windows 3.1 in repainting the screen.
  • The font the user is working with may need to be created on the fly in Windows 3.0.

Additional reference words: FoxWin 2.50 2.50a 3.00 3.10
