KB Articles Included w/ FoxPro for Windows/MS-DOS 2.6 DK

 Last reviewed: May 26, 1997
Article ID: Q119693
The information in this article applies to:
  • Microsoft FoxPro Distribution Kit for Windows, version 2.6
  • Microsoft FoxPro Distribution Kit for MS-DOS, version 2.6

SUMMARY

KBVIEW.APP is a utility that provides access to 36 Microsoft Knowledge Base articles. It is provided with the Microsoft FoxPro Distribution Kit version 2.6 for Windows and MS-DOS and is installed in the root FoxPro installation directory (usually FPW26 or FPD26).
