How to Create Floating Text Objects in Report Writer

 Last reviewed: April 29, 1996
Article ID: Q97053
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

In FoxPro for Windows, the Float As Band Stretches option in the Report Writer is not available with text objects as it is in FoxPro for MS-DOS.

To work around this situation, create the text objects as field objects. Use the field tool and position the object on the screen layout, and then enter the text in the expression box surrounded by quotations. Since this is now a field object, the Top - Field Can Stretch option is now available.
