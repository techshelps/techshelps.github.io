FIX: "Invalid BROWSE Setup" Error Browsing Database

 Last reviewed: October 14, 1997
Article ID: Q88138
2.00 MS-DOS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, version 2.0

SYMPTOMS

Issuing a BROWSE LAST command from the Database menu or in the View window causes an "Invalid Browse Setup -- Default Setup Used" message to appear.

CAUSE

The structure of the database was changed, followed by a BROWSE LAST command.

Microsoft has confirmed this to be a bug in Microsoft FoxPro version 2.0 for MS-DOS.

RESOLUTION

After deleting a field from the database, issue a BROWSE command from the Command window. Doing so saves the changes made to the BROWSE window in the FOXUSER resource field when the BROWSE window is closed. The saved information includes the field list, the size of each field, and the location and size of the BROWSE window.

This problem was corrected in Microsoft FoxPro version 2.5 for MS-DOS, and in Microsoft FoxPro version 2.5 for Windows.

REFERENCES

Microsoft FoxPro "Command & Functions" manual, BROWSE command chapter.
