HOWTO: Search for FoxPro Articles Using KBKeywords

 Last reviewed: March 10, 1998
Article ID: Q122923
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 5.0
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6x
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5x, 2.6x
  • Microsoft FoxPro for Macintosh, versions 2.5x, 2.6a
  • Microsoft Visual FoxPro for Macintosh, version 3.0b
  • Microsoft FoxPro for UNIX, version 2.6

SUMMARY

The Microsoft Knowledge Base (KB) is categorized by using keywords. This article lists the category and subcategory keywords specific to Microsoft FoxPro articles.

NOTE: For intuitive keywords to limit searches to specific platforms (for example, "VFoxWin" limits searches to Visual FoxPro for Windows articles), please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q122791
   TITLE     : Query Keywords Specific to the FoxPro Knowledge Base

MORE INFORMATION

Microsoft FoxPro KBKeyword Major and Minor Keywords

Each KB article in the FoxPro collection may contain a product-specific subcategory keyword (called a KBKeyword) that places the article in an appropriate FoxPro category. Each KBKeyword word is composed of the concatenation of a major topic keyword and a minor topic keyword.

For example, the keyword for the Interoperability major topic is Fxinterop. The keyword for the DDE minor topic is Dde. Therefore, to query for all articles about Interoperability using DDE, use the KBKeyword FxinteropDde.

NOTE: To retrieve all FoxPro articles on a major topic, place the asterisk (*) wildcard character at the end of the major topic keyword, For example, to query for all FoxPro Interoperability articles, use Fxinterop*.

KB-Wide Keywords

Each article in the FoxPro collection also contains at least one generic, KB-wide keyword. The KB-wide keywords are standard throughout the Microsoft Knowledge Base, appearing in all KB articles, regardless of product. You can use the KB-wide keywords to organize all KB articles or to search for articles across several Microsoft products. For more information about these KB-wide keywords, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles
