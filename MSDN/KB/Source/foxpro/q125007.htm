How To Make BEAUTIFY.APP Recognize New Words

 Last reviewed: August 7, 1995
Article ID: Q125007
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a

SUMMARY

The BEAUTIFY.APP application can "beautify" additional or new commands, functions, or other words, if they are entered into its PROWORDS.FXD text file.

MORE INFORMATION

The BEAUTIFY.APP beautifies the words listed in the PROWORDS.FXD file located in the main FoxPro directory. The PROWORDS.FXD file is a text file that can be modified in any text editor such as Word for Windows, Write, FoxPro, or the MS-DOS editor (EDIT.COM).

Open the PROWORDS.FXD in an editor, then add the new words, functions or commands to the list. Save the changes to PROWORDS.FXD.

The next time you run the BEAUTIFY.APP, it will beautify the new words added to the PROWORDS.FXD file.

Additional reference words: 2.50 2.50a 2.50b 2.60 2.60a FoxWin
