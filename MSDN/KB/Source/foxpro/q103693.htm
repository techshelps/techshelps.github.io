FoxPro for Windows Won't Run Under OS/2 Version 2.0

 Last reviewed: June 27, 1995
Article ID: Q103693
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

OS/2 version 2.0 does not support the 32-bit instruction set in its Windows simulator. Since FoxPro for Windows is a 32-bit product, it cannot be run under OS/2 2.0.

Additional reference words: FoxWin 2.50 2.50a OS/2 32 bit
