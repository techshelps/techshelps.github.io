PRB: FoxPro for Windows Distribution Kit Error with OS/2

 Last reviewed: April 30, 1996
Article ID: Q117432
The information in this article applies to:
  • Microsoft FoxPro for Windows Distribution Kit, versions 2.5, 2.5a, 2.5b, 2.6

SYMPTOMS

If you use the FoxPro for Windows Setup Wizard to create a set of distribution disks under OS/2, you receive the error message "Dynamic Library does not contain a function called GetPrivateProfileString."

CAUSE

Older versions of OS/2 for Windows do not include this function.

RESOLUTION

The OS/2 2.1 Service Pack, available from IBM, may correct the problem.

According to Technical Support at IBM, APAR number PJ10140 specifically references this problem with FoxPro. IBM lists this problem as corrected with the above-mentioned service pack.
