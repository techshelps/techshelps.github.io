BUG: Hide All Can Be Chosen During Welcome Splash Screen

 Last reviewed: May 16, 1996
Article ID: Q119688
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

Double-clicking the welcome ("splash") screen while pressing COMMAND+OPTION enables the Hide All command on the Window menu.

Choosing Hide All causes all FoxPro windows to disappear.

WORKAROUND

Do one of the following:

  • To quit FoxPro, press COMMAND+Q.

    -or-

  • To continue running FoxPro, choose Wiz_splash from the Window menu. This redisplays the welcome screen.

    NOTE: If you chose Proceed To The FoxPro Menu System And Command Window from the welcome screen, you will also need to do the following to restore the Screen and Command windows:

    1. From the Window menu, choose Screen. This restores the Screen, but 

          hides the Command window.

    2. From the Window menu, choose Command. This redisplays the Command 

          window in front of the Screen window.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The introductory, or "splash," screen is a dialog box titled "Welcome to Microsoft FoxPro version 2.6a for Macintosh."

This problem occurs in both FoxPro for Macintosh for the Motorola 680x0 processor, and FoxPro for Macintosh for the PowerPC processor.

Steps to Reproduce Problem

  1. Start FoxPro for Macintosh. If the introductory screen doesn't appear, do the following:

    a. Type "MODIFY FILE SYS(2019)" (without the quotation marks) in 

          the Command window.

    b. Comment out the line containing the CATMAN=ON or CATMAN=OFF 

          statement by placing an asterisk in front of it. Close the file,
      saving changes.

    c. Quit and then restart FoxPro.

  2. While pressing COMMAND+OPTION, double-click the welcome screen.

  3. From the Window menu, choose Hide All.

Note that all FoxPro windows disappear. FoxPro, however, is still running, as indicated by the FoxPro icon in the upper-right corner of the menu. Also, all FoxPro menu commands, except Wiz_splash, are grayed out (disabled).
