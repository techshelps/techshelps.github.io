OTHERWISE Used Instead of ELSE in IF Command

 Last reviewed: April 29, 1996
Article ID: Q95669
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

The keyword OTHERWISE is used incorrectly in three examples in the "Developer's Guide." OTHERWISE should be replaced with ELSE.

The examples are in the following sections of the DDE chapter: 

   Request on page D12-16
   Poke on page D12-17
   Advise on page D12-18
Note also that the keyword "END IF" on page D12-18 should be "ENDIF" with no space.
