FIX: DISPLAY STATUS Doesn't Display Processor Type in FPM

 Last reviewed: October 20, 1997
Article ID: Q114128
2.50b 2.50c MACINTOSH kbprg kbfixlist kbbuglist kbdocerr

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, versions 2.5x

SYMPTOMS

Page 389 of the FoxPro for Macintosh "Language Reference" incorrectly states that the DISPLAY STATUS command can be used to display the processor type of the current FoxPro environment. The FoxPro online Help for the DISPLAY STATUS command also states this. However, neither the DISPLAY STATUS nor LIST STATUS command will return processor information in FoxPro for Macintosh.

NOTE: These commands will return processor information in FoxPro for Windows and MS-DOS.

WORKAROUND

Use the SYS(17) function to return the processor type in FoxPro for Macintosh.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in FoxPro 2.6a for Macintosh.

REFERENCES

"Language Reference," version 2.5b, pages 388-389, 494 and 851 FoxPro for Macintosh, version 2.5b/2.5c online Help
