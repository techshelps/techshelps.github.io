Where to Find Info About Visual FoxPro File Structures

 Last reviewed: August 21, 1996
Article ID: Q136586
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0
  • Microsoft Visual FoxPro for Macintosh, version 3.0b

SUMMARY

Visual FoxPro file structures are described in the Help file. This information is also located in the 26spec.Pjx and 30spec.Pjx projects located in the Vfp\Filespec directory.

MORE INFORMATION

To find information about the .dbc, .frx, .lbx, .mnx, .pjx, .scx, and .vcx files types, search for "file structures" and then "Table Structure of Table Files" in the Help file. This topic contains some general information, and points you to field-by-field descriptions in the Vfp\Filespec directory.

To find information about the .cdx, .dbf, .fky, .fpt, and .idx files types, search for "file structures" and then the topic that describes the appropriate file type in the Help file.
