BUG: FoxPro for Mac Can't Locate Database When Loading .VUE

 Last reviewed: May 17, 1996
Article ID: Q109840
The information in this article applies to:
  • Microsoft Visual FoxPro for Macintosh, version 3.0b
  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.6

SYMPTOMS

When you are opening a .VUE (View) file, FoxPro for Macintosh will ask you to open each database because it cannot find them. It will also not restore the relations to each file if they were set before the View file was saved.

CAUSE

If the default folder is not on the drive that FoxPro for Macintosh is on, View files will not be restored properly.

RESOLUTION

To avoid this problem, make sure that FoxPro for Macintosh is on the same drive (volume) as the default folder.

STATUS

Microsoft has confirmed this to be a problem in FoxPro 2.5b for Macintosh. We are researching this problem and will post new information in the Microsoft Knowledge Base as it becomes available.
