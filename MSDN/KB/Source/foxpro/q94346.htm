No Print Option in FoxPro for Windows Page Preview

 Last reviewed: April 29, 1996
Article ID: Q94346
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

The FoxPro version 2.5 and 2.5a for Windows page preview does not have a Print button as most Microsoft Windows-based applications do.

This feature was not implemented in this release to provide consistency with Microsoft FoxPro version 2.5 for MS-DOS.

NOTE: A Print Preview performed through Visual FoxPro's interface is modal, which means that you can't subsequently access the menu in order to print the report.
