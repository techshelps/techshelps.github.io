PRB: Using Proportional Fonts with Printer Drivers in FoxPro

 Last reviewed: April 17, 1995
Article ID: Q99545
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a

SYMPTOMS

When you are using a printer driver that has proportional fonts (for example, CG Times or Univers), all printed text appears on one line and the lines overwrite each other.

CAUSE

The value of the Leading Points option in the Printer Driver Setup dialog box is incorrect.

RESOLUTION

To correct this problem, do the following:

  1. From the File menu, choose Printer Setup.

  2. Choose Printer Driver Setup.

  3. In the dialog box, choose either Edit or New.

  4. In the Fonts section, enter the correct size in the Leading Points box. For example, the correct size for 10 points is 6.66, not 0.80.
