See Also IMPORT Jump in Import Data Dialog Help Is Wrong

 Last reviewed: April 30, 1996
Article ID: Q120903
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6a
  • Microsoft FoxPro for Macintosh, version 2.6a

SUMMARY

In the FOXHELP.DBF online help file, the "Import Data Dialog" topic lists "IMPORT" in the See Also list box.

Choosing IMPORT from the list box should cause the IMPORT command topic to be displayed. Instead the "Import Data Dialog" topic is displayed again.

MORE INFORMATION

"Import Data Dialog" is located under "The FoxPro Interface" section, toward the bottom of the help file.
