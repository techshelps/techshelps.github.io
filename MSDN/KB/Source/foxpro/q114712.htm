PRB: dBASE RUN() Causes "File 'RUN.PRG' Does Not Exist."

 Last reviewed: June 27, 1995
Article ID: Q114712
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6
  • Microsoft FoxPro for MS-DOS, version 2.6

SYMPTOMS

Running the dBASE RUN() function causes the following error message: 

   File 'RUN.PRG' does not exist.

STATUS

This behavior is by design.

MORE INFORMATION

According to FoxPro online Help, the dBASE RUN() function is not supported in FoxPro. Use the FoxPro RUN command instead.

Steps to Reproduce Problem

Type the following in the Command window: 

   =RUN("C:\COMMAND.COM")

Additional reference words: FoxDos FoxWin 2.60 on-line help
