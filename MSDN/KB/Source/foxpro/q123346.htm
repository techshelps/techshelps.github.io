FILE: Migrat.exe Contains the FoxPro Migration Tool

 Last reviewed: February 19, 1998
Article ID: Q123346
2.5x WINDOWS kbtool kb3rdparty

The information in this article applies to:

  • Microsoft FoxPro for Windows, versions 2.5x

SUMMARY

Migrat.exe is a file that contains the FoxPro Migration application to help developers convert dBASE IV and Clipper code to FoxPro 2.5x compatible syntax.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~ Migrat.exe (size: 458512 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
The FoxPro Migration application helps developers convert dBASE IV and Clipper code to FoxPro compatible syntax. FoxPro 2.6 and 2.6a have built in commands providing compatibility with dBASE IV, so Microsoft no longer directly offers the Migration application to users. However, you can still obtain the files from the MSL.
