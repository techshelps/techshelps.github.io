FILE: Gphsamp.exe Shows Automation to DataSheet in MS Graph 8

 Last reviewed: February 19, 1998
Article ID: Q165172
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 5.0, 5.0a

SUMMARY

Gphsamp.exe is a file that demonstrates how to use OLE Automation to Microsoft Graph to change the data that Graph 8.0 is using for a graph.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~ Gphsamp.exe (size: 25470 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
Microsoft Graph 8.0 that ships with Microsoft Office 97 exposes the DataSheet to OLE automation. The Graphsmp form, which is available for download, is an example of how to use OLE automation to change values in the Graph 8.0 DataSheet. The sample uses the TestData database (Vfp\Samples\Data) that ships with Visual FoxPro. It uses the Employee and Order tables to chart the total yearly and quarterly sales for each salesperson. For more information on doing OLE automation to Microsoft Graph 8.0 please see the Vbagrp8.hlp file that ships with Microsoft Office 97.
