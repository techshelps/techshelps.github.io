The information in this article applies to:

  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5x, 2.6, 2.6a
  • Microsoft Visual FoxPro for the Macintosh, version 3.0b
  • Microsoft FoxPro for the Macintosh, version 2.5x, 2.6a
  • Microsoft FoxPro for UNIX, version 2.6

SUMMARY

The following is a list of technical-support telephone numbers useful to Microsoft FoxBASE+ and FoxPro developers.

MORE INFORMATION

Microsoft technical support is available for all versions of the following FoxBASE+ and FoxPro products on the MS-DOS, Microsoft Windows, Apple Macintosh, SCO UNIX, and SCO XENIX platforms: 

   FoxBASE+
   FoxPro
   FoxPro Distribution Kit
   FoxPro Connectivity Kit
   FoxPro Library Construction Kit
   Visual FoxPro
NOTE: Microsoft does not provide technical support for FoxBASE+ for UNIX. The Santa Cruz Operation (SCO) supports FoxBASE+ for UNIX. SCO's support number is (800) 726-UNIX.
