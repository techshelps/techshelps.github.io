FIXUSER.APP Ships Only with FoxPro for MS-DOS

 Last reviewed: June 27, 1995
Article ID: Q114353
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a, 2.5b, 2.6
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6
  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c

SUMMARY

FIXUSER.APP is a utility program shipped with FoxPro versions 2.0 and later for MS-DOS. This program restores the FoxPro default color sets to the FOXUSER.DBF file.

MORE INFORMATION

FoxPro for Macintosh does not support the use of color sets and will ignore any commands that save or load color sets.

FoxPro for Windows does support the use of color sets, but the use of color sets in Windows is not recommended. Instead, use individual color schemes or the Windows Control Panel to control colors.

Because the use of color sets is not supported or is discouraged on platforms other than MS-DOS, FIXUSER.APP is not shipped with FoxPro for Windows or FoxPro for Macintosh.

A FoxPro color scheme is a set of 10 color pairs. Each color pair controls the color of a specific attribute. Color schemes are referred to by number, with valid color scheme numbers ranging from 1 to 24. Some color scheme numbers are reserved by FoxPro. For example, color scheme 10 controls the colors in a Browse window.

A FoxPro color set contains information for all color pairs in all color sets 1 through 24.

REFERENCES

For more information about using colors in FoxPro, refer to the "Language Reference" or the online Help file.
