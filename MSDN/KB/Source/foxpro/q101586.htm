How to Access Close All Snippets Command on Screen Menu

 Last reviewed: June 27, 1995
Article ID: Q101586
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.5

The Close All Snippets command on the Screen menu in FoxPro for Windows does not automatically appear when a screen with screen snippets is open. (The Open All Snippets command remains on the menu instead.)

Page D2-14 of the "Developer's Guide" fails to state that you must press the SHIFT key before opening the Screen menu in order to make this command available. Instead, D2-14 simply states: 

   From the Screen menu you can choose Open All Snippets or Close All
   Snippets to open or close all code snippet editing windows at once.
