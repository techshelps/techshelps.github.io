Incorrect Program Example in Lang Ref for PRIVATE Command

 Last reviewed: April 29, 1996
Article ID: Q97636
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a

The program example demonstrating the PRIVATE command on page L3-752 in the "Language Reference" manual contains an error in the fifth line. If this incorrect line of code is executed, the error message "Unrecognized phrase/keyword in command" will appear.

To correct this problem, replace this incorrect line 

   ? val1, ? val2
with the following correct line: 

   ? val1, val2

Additional reference words: FoxDos FoxWin 2.50 2.50a errmsg err msg
