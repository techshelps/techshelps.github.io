FIX: Cannot Build Compact EXE

 Last reviewed: January 8, 1998
Article ID: Q124117
2.60 MS-DOS kbsetup kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, version 2.6

SYMPTOMS

Attempts to build a compact .EXE file fail in FoxPro version 2.6 for MS- DOS.

CAUSE

FOXT.EXE is required to build .EXE compact executable files. During Distribution Kit installation, the FOXT.EXE file was placed in an incorrect directory or was not installed at all. FOXT.EXE is installed only when the standalone EXE option is also installed.

RESOLUTION

Install the Distribution Kit with both the compact and standalone options. This ensures that the FOXT.EXE file is available to the Build command.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in FoxPro version 2.6a for MS-DOS.
