General Functionality of the Setup Wizard

 Last reviewed: October 7, 1995
Article ID: Q136692
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

SUMMARY

This article describes the Setup Wizard and the features that can be customized.

MORE INFORMATION

The Setup Wizard creates a setup routine and distribution disks for your application for both disk and network installations. Besides installing your files in the directory specified for installation, the Setup Wizard is a customizable installation routine that allows you to specify the title of the application being installed, a copyright statement in the About Setup dialog box, and a Program Manager item.
