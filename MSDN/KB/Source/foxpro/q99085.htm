FoxPro Applications Running w/ Windows for Pen Computing

 Last reviewed: June 26, 1995
Article ID: Q99085
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.5

SUMMARY

All FoxPro applications should run correctly in the Microsoft Windows for Pen Computing environment; for example, you should be able to access all buttons and menus through the pen.

If the cursor does not change into a pen when placed over a text input field, make sure that the Pen Palette is running and that Automatic Text Recognition (see Options in the Pen Palette) is enabled.

MORE INFORMATION

To input text, you can write over a text field using the pen.

NOTE: There is no support for inking (leaving the text unrecognized) in text fields.
