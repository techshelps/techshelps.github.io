PRB: AutoReport Fields Default to Chicago Font

 Last reviewed: June 3, 1996
Article ID: Q121241
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

Fields in a report created by Catalog Manager's AutoReport tool default to the Chicago font.

NOTE: This behavior does not occur in Visual FoxPro for Macintosh.

CAUSE

The Times New Roman TrueType font, which is normally used by AutoReport, is not installed.

NOTE: In Visual FoxPro for Macintosh, the default font is Geneva.

RESOLUTION

To install the Times New Roman font, do the following:

  1. Run the Microsoft FoxPro Setup (installation maintenance) program.

  2. When prompted to select an option, choose the Add/Remove button.

  3. Under Components Not Yet Installed, select TrueType Fonts, and then choose Continue.

STATUS

This behavior is by design.

MORE INFORMATION

TrueType fonts, including Times New Roman, are not installed when a minimum installation of FoxPro is performed.
