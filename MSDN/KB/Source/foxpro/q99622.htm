PRB: No Line Number in Text Editor Status Bar

 Last reviewed: June 27, 1995
Article ID: Q99622
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SYMPTOMS

Line numbers do not appear in the status bar of the text editor.

CAUSE

The preferences for the editor are set to support word wrapping. When the Word Wrap check box is selected in the Edit Preferences dialog box, FoxPro cannot accurately reflect the line number in relation to the current cursor position. Therefore, it displays only the column number in relation to the current cursor position.

RESOLUTION

  1. With the FoxPro Editor active, choose Preferences from the Edit menu.

  2. In the Edit Preferences dialog box, clear the Word Wrap check box.
