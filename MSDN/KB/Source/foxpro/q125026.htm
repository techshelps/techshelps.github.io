DOCERR: FOXTERM Environment Variable Instructions Incorrect

 Last reviewed: February 21, 1995
Article ID: Q125026
The information in this article applies to:
  • Microsoft FoxPro for UNIX, version 2.6

SUMMARY

The information on page 57 in the "Installation and Configuration" Guide about setting the FOXTERM environment variable for FoxPro for UNIX is incorrect.

MORE INFORMATION

The example on page 57 in the "Installation and Configuration" Guide reads: 

   FOXTERM=wy60FoxPro. export TERM
However, to correctly set the FOXTERM environment variable in the Bourne shell, the example on page 57 should read: 

   FOXTERM=wy60fox; export FOXTERM
