SAMPLE: Solution.exe Extends the Visual FoxPro Solutions Sample

 Last reviewed: February 19, 1998
Article ID: Q158051
5.00 WINDOWS kbusage kbappnote kbhowto kbfile

The information in this article applies to:

  • Microsoft Visual FoxPro for Windows, version 5.0

SUMMARY

Solution.exe is a sample that contains a reference to a sample form, Filtrlst.scx, which can be used to facilitate adding new components and filters to the sample.

The Visual FoxPro Solutions sample can easily be extended by adding new components.

Refer to Knowledge Base article, Q157723, "How To Extend the Visual FoxPro Solutions Sample" for more information about how to add a new form, query, view, or report to the Solutions sample.

This article contains a reference to a sample form Filtrlst.scx, which can be used to facilitate adding new components and filters to the sample. For best results, these files Filtrlst.scx and Filtrlst.sct should be placed in the Vfp\Samples\Solution directory. The form itself contains instructions about its use.
