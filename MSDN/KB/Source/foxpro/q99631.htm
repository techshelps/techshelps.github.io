The information in this article applies to:

SUMMARY

KBVIEW.APP is an utility that provides access to 101 Microsoft Knowledge Base articles. It is provided with the Microsoft FoxPro Distribution Kit version 2.5 for Windows and is installed in the root FoxPro for Windows installation directory.
