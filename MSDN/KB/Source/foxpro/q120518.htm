ERR: <Filename>.pjx Is Not an Object File

 Last reviewed: August 28, 1995
Article ID: Q120518
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.6, 2.6a

SYMPTOMS

When you insert a project inside another project and try to build the original project into an .APP or an .EXE file, the error message "<Filename>.pjx is not an object file" occurs.

CAUSE

The Project Manager assumes that the inserted project is an application that can be built into an .APP or an .EXE file.

RESOLUTION

Remove the project that was inserted into the original project. A project cannot be inserted into another project in FoxPro.
