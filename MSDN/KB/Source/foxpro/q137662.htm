UNCONF: Unable to Process Msjt2032.dll Running Setup Wizard

 Last reviewed: December 18, 1997
Article ID: Q137662
3.00 WINDOWS kbtool kbprb

The information in this article applies to:

  • Microsoft Visual FoxPro for Windows, version 3.0

SYMPTOMS

When running the Setup Wizard, the disk image directories are not created and the error message, 'Unable to process MSJT2032.DLL' is displayed.

CAUSE

The Msjt2032.dll file was corrupted.

RESOLUTION

Manually extract the Msjt2032.dll from the Visual FoxPro floppy disks to allow the Setup Wizard to complete the building of the disk images.
