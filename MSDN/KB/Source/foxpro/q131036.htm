FIX: Command+O Appears Twice in Macintosh Menu

 Last reviewed: October 20, 1997
Article ID: Q131036
2.50c MACINTOSH kbtool kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, version 2.5c

SYMPTOMS

When a Project is opened in FoxPro, the command key equivalent Command+O appears as a key combination when you choose Options from the Project menu and when you choose Open from the File menu.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in version 2.60a of FoxPro for Macintosh.

MORE INFORMATION

In version 2.50c of FoxPro for the Macintosh, if a Project window is active the Project menu is available. Use of the the key combination Command+O opens the Options dialog box for the Project. Any other time the key combination Command+O opens the File Open dialog box.
