BUG: Linked Word Document Doesn't Update Properly

 Last reviewed: June 26, 1995
Article ID: Q125006
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a

SYMPTOMS

When a Word 6.0 for Windows document is linked to a FoxPro general field, modifications to the original Word for Windows document do not appear in the general field.

CAUSE

The modifications to the original Word for Windows document occurred after its original end-of-file marker. Modifications before the original end-of- file marker are properly displayed in the FoxPro general field.

RESOLUTION

Manually relink the general field to the Word for Windows document.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. Type the following in the Command window:

    CREATE TABLE link (word_gen GENERAL) APPEND BLANK BROWSE

    NOTE: Do NOT quit FoxPro.

  2. Start Word 6.0 for Windows.

  3. From the File menu, choose New. Select the Normal template.

  4. Type 'This is a test' (without the quotation marks) in the document.

  5. From the File menu, choose Save. Save the document as LINK.DOC.

  6. From the Edit menu, choose Select All.

  7. From the Edit menu, choose Copy.

  8. Switch to FoxPro.

    NOTE: Leave Word for Windows and LINK.DOC open.

  9. Double-click (or press CTRL+PAGE DOWN) to open the general field.

  10. From the Edit menu, Paste Special.

  11. Select Microsoft Word 6.0 Document Object and press the Paste Link button.

  12. Close the general field by pressing CTRL+W, or double-clicking the control menu box in the upper left-hand corner of the general field. When prompted to save changes choose Yes.

  13. Switch back to Word for Windows.

  14. At the beginning of the document, type 'Adding text at the beginning.' (without the quotation marks).

  15. At the end of the document, type 'Adding text at the end.' (without the quotation marks).

  16. Switch back to FoxPro.

  17. Double-click or press CTRL+PAGE DOWN to open the general field.

Note that the new text at the beginning of the document appears in the general field, but that the text at the end of the document does not.
