FIX: "Type 1" Error While Printing to StyleWriter II

 Last reviewed: October 20, 1997
Article ID: Q115332
2.50b MACINTOSH kbprg kbfixlist kbbuglist kberrmsg

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, version 2.5b

SYMPTOMS

The following System 7 error message occurs when you are printing to a StyleWriter II printer: 

   The application "unknown" has unexpectedly quit, because an error of
   type 1 occurred.
NOTE: If the Apple Debug tool is installed, the following error occurs instead: 

   Bus Error at 01A2002E 'code 00060DC fox Root +072E'

STATUS

Microsoft has confirmed this to be a problem in FoxPro 2.5b for Macintosh. This problem was corrected in FoxPro 2.5c for Macintosh.

MORE INFORMATION

This error may occur under one of the following conditions:

  • More than 12 pages are printed.
  • The system runs out of memory.
  • The print job is canceled.
  • A report is printed.

Steps to Reproduce Problem

These steps assume the following:

  • FoxPro is installed in a folder (directory) named "FoxPro".
  • The volume (disk) name is "Macintosh HD".

  1. From the Apple menu, choose Chooser.

  2. From the list of drivers, select the LW Select 300.

  3. Close the Chooser.

  4. Start FoxPro.

  5. In the Command window, type the following: 

          MODIFY FILE "Macintosh HD:\FoxPro\Q&A Readme"

  6. From the File menu, choose Print. Choose OK.

  7. After four pages have spooled to the printer, press COMMAND+PERIOD to cancel the print job.

Note that one of the error messages described in the "Symptoms" section above is displayed.
