FIX: Screen Choice Not Staying Selected in Catalog Manager

 Last reviewed: October 20, 1997
Article ID: Q121634
2.60 WINDOWS kbui kbbuglist kbfixlist

The information in this article applies to:

  • Microsoft FoxPro for Windows, version 2.6

SYMPTOMS

After you run a screen from the Catalog Manager, the screen you selected in the list box does not stay highlighted. The first screen selection in the list box will become the new highlighted choice.

STATUS

Microsoft has confirmed this to be a problem in FoxPro version 2.6 for Windows. This problem was corrected in FoxPro version 2.6a for Windows.

MORE INFORMATION

Steps to Reproduce Problem

  1. Open the Catalog Manager.
2. Choose the Screen tab. 3. Create at least two screens. 4. Select the last screen and run it. 5. Close the screen.

Note that in the Catalog Manager, the screen selection at the top of the list is now highlighted.
