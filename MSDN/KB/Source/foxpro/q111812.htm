FoxPro for Macintosh Installer/Setup Memory Requirements

 Last reviewed: May 29, 1996
Article ID: Q111812
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.5b

SUMMARY

The FoxPro for Macintosh Setup program requires that at least 600K of memory be available in order for it to run.

NOTE: These memory requirements also apply to the Setup program for the Distribution Kit.

MORE INFORMATION

The Get Info dialog box in the Macintosh Finder shows that the Microsoft FoxPro Setup program has a Suggested, Minimum, and Preferred memory size of 600K.
