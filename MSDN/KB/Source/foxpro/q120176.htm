FIX: DELETED=ON in CONFIG.FPW Causes Subscript Error

 Last reviewed: October 20, 1997
Article ID: Q120176
2.50b 2.50c 2.60a MACINTOSH kbtool kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, versions 2.5x, 2.6a

SYMPTOMS

When you are building disks in the FoxPro for Macintosh Distribution Kit using the SetupWizard, a subscript error occurs.

CAUSE

The CONFIG.FPW file contains the setting DELETED=ON.

RESOLUTION

Set DELETED=OFF in the CONFIG.FPW file. Or, type SET DELETED OFF in the Command window before running SETUP.APP.

STATUS

Microsoft has confirmed this to be a problem in FoxPro versions 2.5x,2 .6a for Macintosh. This problem was corrected in Visual FoxPro for Macintosh, version 3.0b.
