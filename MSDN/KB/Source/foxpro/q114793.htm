Color Sets Not Supported on FoxPro for Macintosh

 Last reviewed: June 4, 1996
Article ID: Q114793
The information in this article applies to:
  • Microsoft Visual FoxPro for Macintosh, version 3.0b
  • Microsoft FoxPro for Macintosh, versions 2.5b and 2.5c

SUMMARY

FoxPro for Macintosh does not support the use of color set commands. Any command that creates or loads color sets will be ignored.

The SET COLOR SCHEME and SET COLOR TO commands are fully supported and should be used instead of SET COLOR SET.

MORE INFORMATION

Color sets are incorrectly referred to as being supported in the following sections of the FoxPro for Macintosh version 2.5b documentation:

  • "Language Reference," pages 281, 711, and 713
  • "Installation and Macintosh Features Guide," page 28
