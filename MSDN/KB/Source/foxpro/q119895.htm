Double-Clicking Right Mouse Button Activates the Menu Bar

 Last reviewed: July 5, 1995
Article ID: Q119895
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5x, 2.6, 2.6a
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6, 2.6a
  • Microsoft FoxPro for Macintosh, versions 2.5x, 2.6, 2.6a
  • Microsoft FoxPro for UNIX, version 2.6

SUMMARY

In any version of FoxPro, when the right (secondary) mouse button is double-clicked, either the File menu (Windows and Macintosh) or the System menu (MS-DOS and UNIX) will become highlighted. This is the expected behavior; however, this behavior is documented only in the online Help file.

MORE INFORMATION

This feature provides an additional way for a user to activate the menu bar. Pressing F10 or the ALT key also activates the menu bar.
