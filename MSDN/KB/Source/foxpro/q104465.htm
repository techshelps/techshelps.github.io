Zoom Option Not Available in Screen Builder

 Last reviewed: April 17, 1995
Article ID: Q104465
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a

SUMMARY

According to the online Help file, there is a Zoom option in the Attributes box under Type in the Screen Layout dialog box in the FoxPro 2.0, 2.5, or 2.5a for MS-DOS Screen Builder. This information is incorrect. There is no such option.

MORE INFORMATION

In the Screen Builder, when you choose Screen Layout from the Screen menu, select Window and then Type, there is an Attributes box. In this box, there are four options: 

   Close
   Float
   Shadow
   Minimize
If you press the F1 key for online Help, these items will be listed along with a Zoom option. This is an online Help error. There is no option to zoom the screen in the Attributes box.
