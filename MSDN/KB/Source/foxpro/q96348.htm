Using the Keyboard to Reorder Fields in Modify Structure

 Last reviewed: June 27, 1995
Article ID: Q96348
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, and 2.5a

To reorder fields in a database using the Modify Structure dialog box without using a mouse, use the CTRL key in combination with the UP ARROW or DOWN ARROW keys. To access the Modify Structure dialog box, choose Setup from the Database menu, then in the Setup dialog box, choose Modify.

To reorder the fields in the Modify Structure dialog box:

  1. Using the TAB key, place the insertion point on the shaded area to the left of the field name that is to be moved.

  2. Press CTRL+UP ARROW key to move the field up or press CTRL+DOWN ARROW key to move the field down.

  3. Continue to press the CTRL+UP ARROW key or CTRL+DOWN ARROW key combination until the field is in the desired position.

  4. When finished, use the TAB key to move to the OK button and press ENTER.
