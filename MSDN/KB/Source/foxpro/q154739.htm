PRB: Fax Manager Causes FoxPro for Mac System Menu Problem

 Last reviewed: December 22, 1997
Article ID: Q154739
2.60a MACINTOSH kbprg kb3rdparty kbtool kbenv kbprb

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

After installing Fax Manager, the File menu pop-up of the system menu appears blank. However, the File menu pad's hot keys function properly. Furthermore, the other system menu pads function normally.

CAUSE

There is a memory conflict between Fax Manager and FoxPro 2.6 for Macintosh.

WORKAROUND

Disable Fax Manager by removing the Fax Manager extension located in the Extension folder and placing it on the desktop. The Extension folder is located within the Macintosh System folder.

STATUS

This behavior is by design.

MORE INFORMATION

The memory conflict prevents FoxPro from properly displaying the contents of the File menu pad. The same problem may occur with other third-party fax or communication software programs. If this behavior occurs, quit FoxPro, disable any fax or communication software present on the computer, and restart FoxPro.
