BUG: Beautify Option Causes a General Protection Fault

 Last reviewed: April 30, 1996
Article ID: Q108713
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, and 2.6

SYMPTOMS

If you do a minimal installation, the Beautify option will not be initialized. If you add the following line to the CONFIG.FPW file 

   _beautify=c:\foxprow\beautify.app
where C:\FOXPROW is the path to a full installation of FoxPro for Windows, the Beautify option will be initialized. However, if you are running FoxPro version 2.5 or 2.5a and you then try to run Beautify on any program file, you will be prompted to locate the FOXDOC.FLL file. Once you have done this, you will be informed that FOXDOC.MSG is not found and a general protection GP) fault will occur if you choose the OK button.

If you are running FoxPro version 2.5, you will be informed that PROWORDS.FXD and FOXDOC.MSG could not be found. A dialog box with Cancel, Suspend, and Ignore buttons will appear. Choosing any of these buttons will cause the Beautify options window to hang. The window will close when you press ESC, but if you continue to repeat these steps, the GP fault will occur.

RESOLUTION

To resolve this error, do one of the following:

  • Reinstall FoxPro using a Complete Installation instead of a Minimum Installation.

    -or-

  • Copy PROWORDS.FXD, FOXDOC.MSG and FOXDOC.FLL to the directory in which you have your Minimum Installation of FoxPro.

STATUS

Microsoft has confirmed this to be a problem in FoxPro versions 2.5, 2.5a, 2.5b, and 2.6 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
