PRB: DB-Library Error 10040: DB-Library Network...

 Last reviewed: April 18, 1995
Article ID: Q115993
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5, 2.5a, 2.5b, 2.6
  • Microsoft FoxPro Connectivity Kit, versions 2.5, 2.6

SYMPTOMS

When you are trying to connect to a server in FoxPro for MS-DOS using the FoxPro Connectivity Kit, the following error message appears: 

   DB-Library error 10040: DB-Library network communication
   layer not loaded

CAUSE

This error message occurs when no named pipes system/network software is loaded.

RESOLUTION

Load DBNMPIPE.EXE at the MS-DOS prompt before starting FoxPro for MS-DOS.

For more information about making an SQL Server connection in FoxPro for MS- DOS, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q112482
   TITLE     : How to Make an MS-DOS Level SQL Server Connection
