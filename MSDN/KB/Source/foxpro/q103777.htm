APPEND GENERAL Won't Work w/ Paintbrush Objects & Win 3.0

 Last reviewed: April 29, 1996
Article ID: Q103777
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

When FoxPro for Windows is run under Windows version 3.0, the APPEND GENERAL <OLE field> FROM <filename> command will give the message "Error creating ole object."

This error occurs because Windows Paintbrush is not an OLE server in Windows 3.0.
