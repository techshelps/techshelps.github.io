List of Error Messages That Have Changed in Visual FoxPro

 Last reviewed: October 18, 1996
Article ID: Q134636
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

SUMMARY

The following list shows those error messages with the corresponding error numbers that have been modified (the wording has changed) in Microsoft Visual FoxPro. This list does not include error numbers new to Microsoft Visual FoxPro.
