Description of Maintenance Mode Setup of FoxPro for Mac

 Last reviewed: May 16, 1996
Article ID: Q121243
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, version 2.6a

SUMMARY

When a FoxPro installation already exists, the FoxPro Setup program starts in maintenance mode, prompting you to add or remove components of your current installation, reinstall FoxPro, or remove all FoxPro components of the current FoxPro installation.

Below is a full description of the maintenance mode Setup program.

MORE INFORMATION

Opening Screen Text

   Welcome to the Microsoft FoxPro 2.6 Setup installation maintenance
   program.

   This program allows you to make changes to your current
   installation of Microsoft FoxPro 2.6 Setup. Please select one of
   the following options:

Opening Screen Buttons and Descriptions

Button       Description


Add/Remove   Add new components, or remove installed components, from
             your current installation.

Reinstall    Repeat your last installation, restoring missing files
             and settings.

Remove All   Remove all previously installed components.

Exit Setup   (No description)
NOTE: The Reinstall button installs FoxPro in the same folder. To install FoxPro in a different folder, choose Remove All, then choose the Exit Setup button. Run Setup again. This time, Setup does not start in maintenance mode and will perform a fresh installation, prompting you for the folder in which to install FoxPro.

Description of Screen Displayed by Choosing Add/Remove

The dialog box contains two list boxes. The list box on the left has the following description: 

   Components Not Yet Installed

   Check any items from this list you would like to install.
The list box on the right has the following description: 

   Installed Components

   Check any items from this list that you would like to remove.
The items listed in either list box are:
  • FoxPro for Macintosh
  • Graphical Help
  • DBF Style Help
  • Documented Examples
  • Other Samples & Goodies
  • FoxDoc
  • GraphWizard & MSGraph
  • FoxBASE+ Tools
  • QuickTime Support
  • Spell Checker & Dictionary
  • TrueType Fonts 

        - Arial
    - Courier New
    - FoxFont
    - WingDings
    - Times New Roman
    - FoxPrint
  • Setup
  • Miscellaneous 

        - Microsoft System Information
    - Readme Files
  • FoxPro Professional Kits (only listed in the Professional version) 

        - Distribution Kit
    - Library Construction Kit
    - Connectivity Kit

This screen also displays the following information:
  • Description (of item selected in either list box)
  • Folder for currently selected option
  • Space required by Setup to install
  • Space available on <hard disk name>
