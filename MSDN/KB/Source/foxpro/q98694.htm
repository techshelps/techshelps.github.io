List & Sizes of Directories Installed w/ FoxPro for Win

 Last reviewed: April 29, 1996
Article ID: Q98694
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.5

SUMMARY

Below is a list of directories installed with FoxPro, along with their approximate sizes.

MORE INFORMATION

When you select Complete Install in the Install dialog box, the GOODIES directory is installed as packed. It must be expanded (unpacked) before it can be used.
