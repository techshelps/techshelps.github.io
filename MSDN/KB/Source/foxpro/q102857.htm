PRB: "Unrecognized Command Verb" from Help File Example

 Last reviewed: June 27, 1995
Article ID: Q102857
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.5

SYMPTOMS

If you copy and run the sample code for the "@ ... GET - Invisible Buttons" and "@ ... GET Command" topics in the Windows-style Help file, you will receive an "unrecognized command verb" syntax error. These examples have an extra carriage return in the middle of @ ... GET commands that are continued on multiple lines using semicolons.

RESOLUTION

To run these examples, remove the extraneous carriage returns.

Additional reference words: FoxWin 2.50 unrecognized command verb @ GET
