Available Installation Documentation

 Last reviewed: April 17, 1995
Article ID: Q99086
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a

The following documents describe the installation process for FoxPro versions 2.5 and later:

  • The "Installation and Configuration" manual describes the normal setup process.
  • INSTALL.TXT describes the Administrative (/a) and Network (/n) installation procedures. Installation of optional components is also described in this file.
  • README.TXT briefly describes the installation process, and then refers to INSTALL.TXT.
  • PSSKB.TXT has an "Installation" section with common setup questions answered.

NOTE: INSTALL.TXT, README.TXT, PSSKB.TXT are located on the main setup disk.
