List of Files Installed by LCK Incorrect in Dev. Guide

 Last reviewed: August 28, 1995
Article ID: Q112478
The information in this article applies to:
  • Microsoft FoxPro Library Construction Kit for MS-DOS, version 2.5
  • Microsoft FoxPro Library Construction Kit for Windows, version 2.5

SUMMARY

Page ix of the "Introduction" section of the Microsoft FoxPro version 2.5 Library Construction Kit "Developer's Guide" incorrectly lists the files installed in the FOXLCK and EXAMPLES subdirectories.

MORE INFORMATION

The following files were omitted from the list of files installed in the FOXLCK subdirectory: 

   DOSC7.MAK
   DOSWAT.MAK
   WINC7.MAK
   WINWAT.MAK
   README.TXT
The following files are installed in the EXAMPLES subdirectory; all the other files listed in the documentation are not installed: 

   EVENT.C
   EXAMPLES.DBF
   EXAMPLES.FPT
   FPATH.C
   FPATHC7.MAK
   FPATHWAT.MAK
   HELLO.C
   LOGIREPL.C
   MEMREPL.C
   MODIFER.C
   NEWVAR.C
   REVERSE.C
   SWATCH.C
