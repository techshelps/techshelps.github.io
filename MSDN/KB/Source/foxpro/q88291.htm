PRB: Screen Builder Message "Cannot Open <Screen> Line 560"

 Last reviewed: April 17, 1995
Article ID: Q88291
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.0

SYMPTOMS

In FoxPro Screen Builder, screen generation stops and the following error message is displayed: 

   Cannot open C:\FOXPRO2\<SCREEN NAME> Line number 560 Press any key
   to clean up and exit

CAUSE

FoxPro cannot open any more files. This error usually indicates that all available file handles are in use.

RESOLUTION

Modify the FILES statement in the CONFIG.SYS file to increase the number of available file handles.

Additional reference words: FoxDos 2.00 err msg
