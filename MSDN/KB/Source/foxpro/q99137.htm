How to Directly Access Help Information for a Command

 Last reviewed: April 29, 1996
Article ID: Q99137
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

To quickly access the Help file for information on a particular subject in FoxPro for Windows, do the following:

  1. Within the Command window or text file, highlight the particular command in question.

  2. Hold down the ALT key and then press the left mouse button. If the mouse pointer is pointing to the highlighted code, Help will be opened for that command; if the mouse pointer is pointing elsewhere, the main Help file screen will be displayed.

REFERENCES

"User's Guide," version 2.5, page U8-2
