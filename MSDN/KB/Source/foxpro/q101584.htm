Where to Find INSTALL.TXT in FoxPro Ver 2.5 Directory

 Last reviewed: April 17, 1995
Article ID: Q101584
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.5 and 2.5a

The Setup program for FoxPro version 2.5 for MS-DOS places the INSTALL.TXT file in the FoxPro version 2.5 program (root) directory. This file is not discussed in any of the documentation supplied with FoxPro version 2.5 for MS-DOS.

INSTALL.TXT contains information on the installation process of FoxPro version 2.5. The document is divided into three main sections:

  • Installing FoxPro for MS-DOS
  • Administrative Setup (/A)
  • Workstation Installation (/N)

"Installing FoxPro for MS-DOS" discusses the three methods of installing the MS-DOS product. The "Administrative Setup (/A)" section covers the basic network requirements for installing FoxPro and lists the steps to follow for installation. The "Workstation Installation (/N)" section documents the ten steps necessary for a workstation setup.
