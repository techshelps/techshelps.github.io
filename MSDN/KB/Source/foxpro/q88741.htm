Developer's Guide Refers to Nonexistent FOXPLUS.FP File

 Last reviewed: April 17, 1995
Article ID: Q88741
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.0

SUMMARY

Chapter 17, the FoxBASE+ compatibility section, of the Microsoft FoxPro version 2.0 "Developer's Guide," refers to a FOXPLUS.FP file in the GOODIES\MISC directory. This filename is incorrect; the text should refer to the CONFIG.FOX file.

Installing the CONFIG.FOX file configures FoxPro 2.0 to be almost completely compatible with FoxBASE+.
