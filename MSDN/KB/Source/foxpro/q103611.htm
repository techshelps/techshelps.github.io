PRB: Dialog Opens w/ REPORT FORM <Report> TO <File> & Win 3.0

 Last reviewed: June 27, 1995
Article ID: Q103611
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.5a

SYMPTOMS

When you send output from a report to a file with the command 

   REPORT FORM <report> TO <file>
the Print To File dialog box will appear and ask for a filename, even though you already supplied one.

CAUSE

This behavior is caused by a limitation in Windows version 3.0.

RESOLUTION

Upgrade to Windows version 3.1.

Additional reference words: FoxWin 2.50 2.50a Print To File dialog
