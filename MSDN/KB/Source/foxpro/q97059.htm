How to Underline Fields in Report Writer by Using Line Draw

 Last reviewed: December 14, 1995
Article ID: Q97059
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5x, 2.6x

SUMMARY

To underline fields in the Report Writer in FoxPro for Windows, you can use the Line Draw feature.

MORE INFORMATION

In FoxPro for MS-DOS, you can underline a field by selecting the Style and Underline options in the Report Expression dialog box. These options are not available in FoxPro for Windows. One way to work around this is to use the Line Draw feature.

You can also draw a line under a field by using the following command in a field expression directly under the field you want to underline: 

   REPLICATE("_",LEN(ALLTRIM(fieldname)))
For information about using the FONTSTYLE field to underline fields, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119233
   TITLE     : How to Underline Objects in a Report Form
