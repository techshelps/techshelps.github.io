PRB:Backspace/Delete TreeView Node's Name Leaves Vertical Line

 Last reviewed: October 29, 1996
Article ID: Q157852
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 5.0

SYMPTOMS

In Visual FoxPro 5.0, using the Backspace/Delete key to delete a TreeView control node's name leaves a vertical line.

STATUS

This is a display problem with the control itself.

MORE INFORMATION

Steps to Reproduce Behavior

  1. Run the Solution.app file in the \Samples\Solution directory.

  2. Choose ActiveX, ActiveX controls, and Add. Remove items in the TreeView control options, then press Run Samples.

  3. Click the New Root button in the sample.

  4. Click the newly created node to activate the node in edit mode.

  5. Type something in the node, then delete the node name using the Backspace/Delete key.
