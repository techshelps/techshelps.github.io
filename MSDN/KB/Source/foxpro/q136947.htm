PRB: Corrupted .vcx or .vct File Causes Visual FoxPro to Hang

 Last reviewed: September 20, 1995
Article ID: Q136947
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, version 3.0

SYMPTOMS

A corrupted .vcx file causes Visual FoxPro to hang (stop responding).

CAUSE

This usually happens because the associated .vct file, not the .vcx file, is corrupted.

STATUS

Microsoft is researching this behavior and will post new information here in the Microsoft Knowledge Base as it becomes available.
