PRB: Cannot Find FOXPRO.PR Error Message in FoxPro for UNIX

 Last reviewed: January 23, 1995
Article ID: Q124540
The information in this article applies to:
  • Microsoft FoxPro for UNIX, version 2.6

SYMPTOMS

When FoxPro for UNIX starts, you see a "Cannot find FOXPRO.PR" error message.

CAUSE

FoxPro for UNIX files have a set of default permissions, just like any other UNIX application. These files and their default permissions are documented in a file called foxpro in the /etc/perms subdirectory. This file can be viewed with any text editor. If these permissions are changed, FoxPro for UNIX may not run correctly.

WORKAROUND

Try the following basic troubleshooting steps:

  1. Contact the system administrator to set the permissions on the FoxPro files back to their defaults by typing:

    fixperm /etc/perms/foxpro

  2. Contact the system administrator to check the filesystems in single-user mode only by typing:

    fsck

  3. Contact the system administrator to re-link the UNIX kernel (again in single-user mode) by typing:

    ./link_unix

STATUS

This behavior is by design.
