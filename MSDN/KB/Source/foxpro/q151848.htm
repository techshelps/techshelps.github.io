Join Microsoft's FoxPro Add-On Component Builder Program

 Last reviewed: June 3, 1996
Article ID: Q151848
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b
  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c, 2.6a
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a, 2.5b, 2.6, 2.6a
  • Microsoft FoxPro for UNIX, version 2.6
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, 2.5b, 2.6, 2.6a

SUMMARY

This article describes how to obtain information on participating in the Microsoft Component Builders Program.

If you create Microsoft FoxPro add-ons, OLE Controls, toolkits, or any other product that works with Microsoft tools, be a part of the Microsoft Component Builders Program. Our program offers marketing opportunities and Information to vendors of add-on products for Microsoft development tools.

Participating in the Microsoft Component Builders Program means having access to registered developers and receiving important information on new products and technologies before it becomes news, giving you a jump on developing your products. Most important, you will have the support of Microsoft's expertise behind your products.

MORE INFORMATION

To learn more about the Component Builder Program, please send your company name, contact information, address, and product information to: cbisv@microsoft.com
