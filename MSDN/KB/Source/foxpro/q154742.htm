FILE: Codewiz.exe the Xbase Code Import Wizard for VFP 3.0

 Last reviewed: February 20, 1998
Article ID: Q154742
The information in this article applies to:
  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b, 5.0

SUMMARY

Codewiz.exe is a file that allows you to convert Xbase code written in Clipper, FoxPro, FoxBASE, or dBASE into a Visual FoxPro form. It converts the following Xbase commands into their equivalent objects in Visual FoxPro: 

   @ ... SAY
   @ ... GET
   USE
