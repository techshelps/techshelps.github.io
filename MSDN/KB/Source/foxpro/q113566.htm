PRB: ESC Key Does Not Quit Catalog Manager

 Last reviewed: April 30, 1996
Article ID: Q113566
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6
  • Microsoft FoxPro for MS-DOS, version 2.6

SYMPTOMS

Pressing the ESC key does not close the Catalog Manager.

CAUSE

The Catalog Manager is part of the FoxPro user interface. Keeping this in mind, it would be confusing to the user if the quit dialog box appeared after the ESC key was accidentally pressed.

RESOLUTION

To quit the Catalog Manager and return to the FoxPro Command window, do one of the following:

  • Double-click the Control-menu box (close box) for the Catalog Manager window.

    -or-

  • From the File menu, choose Command Window.
