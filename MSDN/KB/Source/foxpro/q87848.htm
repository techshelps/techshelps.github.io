Single-User FoxPro Not Designed for Network Installation

 Last reviewed: April 17, 1995
Article ID: Q87848
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 1.0, 1.01, 1.02, and 2.0

SUMMARY

The single-user version of FoxPro should not be installed on a network server.

MORE INFORMATION

Single-user FoxPro is not recommended for network installation. Although single-user FoxPro can efficiently access files located on a network (in EXCLUSIVE mode), it is not designed to be installed on a network server. If you install single-user FoxPro on a network server, you may experience severe performance degradation, especially with workstations that have the minimal amount memory.

FoxPro/LAN is specifically designed to operate efficiently in a network environment. For several users to share a single copy of FoxPro, FoxPro/LAN is required. Shared use of single-user FoxPro is specifically prohibited by the FoxPro license agreement.
