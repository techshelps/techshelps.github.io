FIX: APPEND NOMENU Not Recognized in FoxPro

 Last reviewed: September 22, 1997
Article ID: Q124595 
2.60    | 2.60a
WINDOWS | MACINTOSH kbprg kbfixlist kbbuglist kberrmsg

The information in this article applies to:

  • Microsoft FoxPro for Windows, version 2.6
  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

FoxPro supports the NOMENU option on the BROWSE, EDIT, and CHANGE commands. However, the NOMENU option on the APPEND command generates this error: 

   Unrecognized phrase/keyword in command

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft FoxPro version 2.6a for Windows and Microsoft FoxPro version 2.6a for Macintosh.
