Setup/Index Dialog Box Push Buttons in Different Order

 Last reviewed: June 26, 1995
Article ID: Q105588
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5, 2.5a, and 2.5b
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a, and 2.5b

In FoxPro for Windows, the order of push buttons in the Indexes area of the Setup dialog box (on the Database menu) differs from the order of these buttons in FoxPro for MS-DOS.

In FoxPro for Windows, the order of the push buttons in the Indexes area of the Setup dialog box is as follows: 

   Add
   Modify
   Set Order/No Order
   Remove
In FoxPro for MS-DOS,the order of the push buttons in the Indexes area of the Setup dialog box is as follows: 

   Add
   Modify
   Remove
   Set Order/No Order

Additional reference words: FoxDos FoxWin 2.00 2.50 2.50a 2.50b
