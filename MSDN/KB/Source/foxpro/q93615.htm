PRB: MS-DOS CD Command Doesn't Work in FoxPro for Windows

 Last reviewed: June 27, 1995
Article ID: Q93615
The information in this article applies to:
  • Microsoft FoxPro for Windows, versions 2.5 and 2.50a

SYMPTOMS

In Microsoft FoxPro for Windows, the following command does not change the working directory: 

   RUN /0 CD \

CAUSE

The FoxPro for Windows MS-DOS window is not an active window. When you issue a "RUN <MS-DOS command>" or "RUN <MS-DOS program>" command, FoxPro invokes FOXRUN.PIF to create a new MS-DOS virtual machine. Any changes to the MS-DOS environment affect only the MS-DOS virtual machine.

RESOLUTION

Use the SET DEFAULT TO command to change the directory.
