DOCERR: MODIFY PROJECT Is Missing NOWAIT in Syntax Section

 Last reviewed: May 23, 1996
Article ID: Q133295
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6a
  • Microsoft FoxPro for Macintosh, version 2.6a

There is a Documentation error in the help file of FoxPro version 2.6a for Windows and Macintosh in the MODIFY PROJECT command. The syntax section is missing the NOWAIT keyword. Additionally, in the clauses section, the NOWAIT topic incorrectly references MODIFY MENU when it should reference MODIFY PROJECT. These documentation errors exists in both the graphical and MS-DOS-style help files.

Additional reference words: 2.60a FoxMac
