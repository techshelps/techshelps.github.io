How to Use the REPLACE Command in FoxPro 1.x

 Last reviewed: April 17, 1995
Article ID: Q89178
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 1.0, 1.01, and 1.02

The REPLACE command replaces the data that was previously in one field with the data from an expression. Unless a FOR or WHILE clause is specified, only the current record is affected by the REPLACE command.

The REPLACE command does not replace any field or record if any open file is at the end of the file (EOF).

Additional reference words: FoxDos 1.00 1.01 1.x
