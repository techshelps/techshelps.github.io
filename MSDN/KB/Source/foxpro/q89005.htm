FoxPro 2.0 for MS-DOS Optional Installation Files

 Last reviewed: April 17, 1995
Article ID: Q89005
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, version 2.0

SUMMARY

The following is a list of the optional files that can be installed during FoxPro installation. File sizes and descriptions are included. Installing all of the options will require approximately 6.6 MB of space.

MORE INFORMATION

Filename            Description


FOXDOC              The automatic applications documentor--a
                    powerful cross-referencing and utility for source
                    code.

GENGRAPH            Interface between FoxPro's RQBE/SQL and the
                    optional FoxGraph 3D graphics package. Using RQBE,
                    GENGRAPH, and FoxGraph, direct graphical outputs
                    can be created from queries.
COMMFUNC directory These files are used in the examples found in the 
                    "Commands & Functions" manual.

SAMPLE directory    Large sample applications--Contains the ACCNTS,
                    CLIENTS, CONVERT, CREDIT, FAMILY, ORGANIZE,
                    RESTAURS, and TRANSACTION applications.
TUTORIAL directory Examples used in the "Getting Started" manual. 

GOODIES directory   Contains several subdirectories with application
                    source code and other utilities.

GOODIES\FNDATION    Example of the foundation READ--READ VALID. This
                    directory contains the EX1 and EX2 applications
                    that demonstrate the following:

                     - How to coordinate READ and BROWSE

                     - How to create applications with an event-driven
                       interface like the FoxPro interface

GOODIES\DEMO        Sources for the demo application. It contains the
                    pieces required to build the FoxPro 2.0 self-
                    running demo.

GOODIES\HELPTREE    HelpTree is an example that features the FoxPro
                    Menu Builder in action. The result is an index and
                    organizer for the FoxPro Help file.

GOODIES\SCREENS     Included are examples of two screens that can be
                    included in applications. One is a sample field
                    picker and the other is a sample file picker.

GOODIES\LASER       Concise example demonstrating how FoxPro's Power
                    Tools can be used to create full-featured
                    applications quickly and easily. Contains the
                    LASER1.APP, LASER2.APP, and LASER3.APP
                    applications.

GOODIES\ADDUSER     This is the source code for AddUser, a utility
                    that properly sets up a workstation to use
                    FoxPro/LAN effectively. Available with the LAN
                    version only.

GOODIES\FOXAPP      Source code for FOXAPP.

GOODIES\PDRIVERS    Source code for the printer drivers. All necessary
                    projects, screens, and menus are included.

GOODIES\PROCOLOR    ProColor is a utility that edits color sets.
                    Courtesy of C. Blaise Mitsutama.

GOODIES\SNIPFMT     Formatter for source code contained in screen and
                    menu "snippets." Requires FoxDoc.

GOODIES\MISC        Odds and ends: files that provide FoxBASE+ 2.1
                    compatibility, a routine to convert a version 1.02
                    FoxUser preference file to FoxPro 2.0 format, and
                    the latest mouse drivers.
