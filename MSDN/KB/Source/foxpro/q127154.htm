FIX: SYSMETRIC(9) Returns Incorrect Value in FoxPro for Mac

 Last reviewed: October 20, 1997
Article ID: Q127154
2.50c MACINTOSH kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, version 2.5c

SYMPTOMS

When the command ? SYSMETRIC(9) is issued in in FoxPro for the Macintosh, the value 1 is returned.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem has been fixed in version 2.60a of FoxPro for the Macintosh.

MORE INFORMATION

Steps to Reproduce Problem

In the Command window type ? sysmetric(9). This should yield the height of the window title. The command returns a value of 1.
