SAMPLE: Sched.exe OLE Automation Adds Schedule+ Appointments

 Last reviewed: February 19, 1998
Article ID: Q153156
3.00 3.00b WINDOWS kbinterop kbole kbfile kbwebcontent

The information in this article applies to:

  • Microsoft Visual FoxPro for Windows, versions 3.0, 3.0b

SUMMARY

Sched.exe is a sample that demonstrates how to use OLE Automation to add appointments to your Microsoft Schedule+ file.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~ Sched.exe (size: 24973 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
The Readme.txt file contained in the Sched.exe self-extracting file provides the location of the key elements of the process (within the form and class library) of adding Schedule+ appointments.
