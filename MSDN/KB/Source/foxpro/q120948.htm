FIX: Setup of Professional Edition PPC/68K Hangs on Power Mac

 Last reviewed: October 20, 1997
Article ID: Q120948
2.60a MACINTOSH kbsetup kbtshoot kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

On a Power Macintosh, Setup may hang when you are installing the combined Power Macintosh and 68K versions of FoxPro in the Professional Edition.

The following error message may also be displayed: 

   Unimplemented trap
NOTE: This problem has also been reported with a MicroMac accelerator card.

CAUSE

The Modern Memory Manager in Control Panel is turned on.

WORKAROUND

To work around this problem, do the following:

  1. From the Apple menu, choose Control Panel, and then choose Modern Memory Manager.

  2. Turn Modern Memory Manager off.

  3. Restart the Macintosh so that the new setting takes effect.

    NOTE: If the original attempt at installing FoxPro hung your Macintosh and prevented it from restarting normally, you may need to rebuild the Macintosh desktop.

  4. Run the FoxPro Setup program.

    If Setup detects the previous installation of FoxPro, choose the Remove All button. Run Setup again to reinstall FoxPro.

After FoxPro is installed, you can turn the Modern Memory Manager on again.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Visual FoxPro 3.0b for Macintosh.

MORE INFORMATION

This problem occurs only with the version of the Microsoft Setup application program included with FoxPro 2.6a for Macintosh.

If another Microsoft product that uses a newer version of Setup, such as Microsoft Excel 5.0, has already been installed on the Macintosh, FoxPro uses that version of Setup.

Steps to Reproduce Problem

  1. Verify that the Modern Memory Manager is turned on. To do this, choose Control Panel from the Apple menu, then choose Modern Memory Manager.

  2. Start the FoxPro Setup program.

  3. When prompted to select the version to install, choose Microsoft FoxPro For Macintosh And Power Macintosh, and then choose OK.

  4. Proceed with the installation.
