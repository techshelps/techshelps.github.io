FIX: Installing Optional Components "Overwrite FoxUser" Msg

 Last reviewed: September 22, 1997
Article ID: Q104262
2.50 MS-DOS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for MS-DOS, version 2.5

SYMPTOMS

When FoxPro installs its optional components, it always displays the following prompt, even if the FoxUser file does not exist: 

   Do you want to overwrite your FoxUser file?

STATUS

Microsoft has confirmed this to be a problem in FoxPro 2.5 for MS-DOS. This problem was corrected in FoxPro 2.5a for MS-DOS.
