PRB: Error 15: "Not a Table/Dbf" at Line 275 in GETCTRL

 Last reviewed: May 16, 1996
Article ID: Q123723
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c, and 2.6a

SYMPTOMS

When you use the Setup wizard in FoxPro for Macintosh, the following error may occur: 

   Error 15 - not a table/dbf @ line 275 in GETCTRL
             Setup Wizard Canceled
   Error 1 - File does not exist in cleanup

CAUSE

One of the .DBF files in the DKCONTRL folder that is used by the SETUP wizard has become corrupted.

RESOLUTION

Drag the DKCONTRL folder (found in the DKSETUP folder) to the TRASHCAN and rerun the Setup Wizard. The Setup wizard will rebuild DKCONTRL folder and its .DBF files.

STATUS

This behavior is by design.

Additional reference words: FoxMac 2.50b 2.50c 2.60a
