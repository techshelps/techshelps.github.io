FIX: SetupWizard Generates Extra Zeros in COMPRESS Command

 Last reviewed: October 14, 1997
Article ID: Q107517
2.50 2.50a WINDOWS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft FoxPro for Windows Distribution Kit, versions 2.5 and 2.5a

SYMPTOMS

The SetupWizard generates unnecessary zeros in the COMPRESS command line with certain settings of the SET POINT command . The zeros do not produce errors or cause any problems.

STATUS

Microsoft has confirmed this to be a problem in the FoxPro 2.5 and 2.5a Distribution Kit for Windows. This problem was corrected in the FoxPro 2.5b Distribution Kit for Windows.
