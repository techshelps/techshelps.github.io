PRB: Catalog Manager Bitmap Push Buttons Appear as Gray Boxes

 Last reviewed: April 30, 1996
Article ID: Q119699
The information in this article applies to:
  • Microsoft FoxPro for Windows, version 2.6

SYMPTOMS

When you are running FoxPro 2.6 for Windows under Windows NT, the AutoScreen, AutoReport, and Browse push button bitmaps in the Catalog Manager appear as dark gray boxes when the Catalog Manager opens an empty catalog.

CAUSE

Some versions of the ATI video drivers cause this display problem when they are running under Windows NT. For example, the Windows NT ATI 1024 x 768 265 Colors video driver causes this display problem.

RESOLUTION

To correct this problem, obtain the most recent ATI driver for the system.

The ATI drivers discussed here are manufactured by a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.
