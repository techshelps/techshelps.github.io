PRB: Novell Network Workstation Hangs When Running FoxPro

 Last reviewed: October 18, 1996
Article ID: Q89573
The information in this article applies to:
  • Microsoft FoxPro for MS-DOS, versions 2.0, 2.5, 2.5a

SYMPTOMS

A computer or workstation running the Novell EMSNETx or XMSNETx drivers stops responding (hangs) while you are attempting to load FoxPro/LAN (not single user). The machine does not hang when it runs the Novell NETx driver.

CAUSE

This problem occurs with Novell EMSNETx and XMSNETx drivers dated prior to 2/11/92.

RESOLUTION

An updated version of EMSNETx or XMSNETx is required.

These files, along with other Novell files, are available from Novell.

To purchase the "DOS Windows Client Kit," which contains the latest Windows and MS-DOS drivers, call Novell at (800) 873-2831 and select option 1, or call (801) 429-7000 and ask to speak with Fulfillment.
