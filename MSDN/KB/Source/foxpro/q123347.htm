FIX: Error Inserting OLE Object When Pasting Text from Word

 Last reviewed: September 22, 1997
Article ID: Q123347
2.60a MACINTOSH kbole kbinterop kbfixlist kbbuglist kberrmsg

The information in this article applies to:

  • Microsoft FoxPro for Macintosh, version 2.6a

SYMPTOMS

Inserting text directly from Microsoft Word version 6.0 for Macintosh into a general field in Foxpro can cause this error: 

   Error creating OLE object

RESOLUTION

Prior to pasting the information into the general field, choose Edit, Insert Object and insert a Word document into the general field. The contents of the clipboard will paste into the blank Word document without error.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Visual FoxPro 3.0b for Macintosh.

MORE INFORMATION

Steps to Reproduce Problem

  1. Create a new document in Microsoft Word version 6.0 for Macintosh for Power PC. Type a sentence. Then copy the sentence to the clipboard.

  2. Open a Foxpro for Macintosh table that includes a general field.

  3. Append a blank record, and browse the table. Double-click the general field to edit.

  4. Choose Edit, Paste to attempt to paste the information from the clipboard. An error will occur. The error occurs regardless of the type of paste; that is, it occurs with Edit, Paste and it occurs with Edit, Paste Special, Paste Link.
