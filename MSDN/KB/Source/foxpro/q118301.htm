Online Help Incorrectly Refers to "Creator File Types"

 Last reviewed: May 29, 1996
Article ID: Q118301
The information in this article applies to:
  • Microsoft FoxPro for Macintosh, versions 2.5b, 2.5c

The information found in the "Creator File Types" topic in the online Help files incorrectly refers to "Creator File Types."

Creators and file types are actually two different components of a file within the Macintosh operating system.

For example, the column headings in the table in the Help file should read as follows: 

   File Type          FoxPro Files          File Extension
   -------------------------------------------------------

   F+DB               Table                 .DBF

Locations of Documentation Errors

  • FoxPro 2.5b and 2.5c for Macintosh FOXHELP.HLP online help, "Creator Types" topic, located under "Reference Tables" in the Contents screen
  • FoxPro 2.5b and 2.5c for Macintosh FOXHELP.DBF online help, "Creator File Types" topic

Additional reference words: FoxMac 2.50b 2.50c on-line docerr
