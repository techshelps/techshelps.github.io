Membership Authentication Doesn't Work Through Netscape Proxy

 Last reviewed: February 3, 1998
Article ID: Q170500

The information in this article applies to:

  • Microsoft Commercial Internet System, version 1.0

SYMPTOMS

When a Netscape user who has installed the Microsoft Membership DPA Authentication Plug-in for Netscape Browser attempts to go through the Netscape Proxy to an MBS Protected site, the following message appears: 

   Authentication Failed. Please Retry.

WORKAROUND

To work around this problem, do one of the following:

  • Have end users upgrade to Microsoft Internet Explorer 3.02 with the DPA add-on for Membership.
  • Have the Internet service provider use another proxy server.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Commercial Internet System version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
