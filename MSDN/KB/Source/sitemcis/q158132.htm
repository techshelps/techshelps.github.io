INS Does Not Support Proxy Server/Remote Winsock

 Last reviewed: February 3, 1998
Article ID: Q158132
The information in this article applies to:
  • Microsoft Commercial Internet System, version 1.0

SYMPTOMS

Microsoft Internet News Server (INS) is not compatible with Microsoft Proxy Server.

CAUSE

INS runs as an Microsoft Windows NT service and only allows the use of a local system account. This means Microsoft Proxy Server (and other "smart" proxies that use account names) will not allow connections to pass through it.

If a proxy server does not use account names (also known as a "dumb proxy"), it should work with no problems.

STATUS

Microsoft has confirmed this to be a problem in Commercial Internet System INS version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
