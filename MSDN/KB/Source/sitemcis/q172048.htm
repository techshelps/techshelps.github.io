Err Msg: Cannot Stop the Procedure "SP"

 Last reviewed: March 12, 1998
Article ID: Q172048
The information in this article applies to:
  • Microsoft Site Server version 2.0 

        - Microsoft Usage Analyst

SYMPTOMS

Usage Analyst displays the following error message: 

   Cannot stop the procedure "SP" because it doesn't exist in the system
   catalogs.

CAUSE

The stored procedure that is attempting to run has been deleted from the catalog.

WORKAROUND

To work around this problem:

  • Run Dbsetup.exe and restore the stored procedures. This will reinstall any missing files that are needed.
