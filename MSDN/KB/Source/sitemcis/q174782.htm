Symbols Font Appears in Print Preview Using Tree View

 Last reviewed: February 2, 1998
Article ID: Q174782
The information in this article applies to:

- Microsoft Site Server version 2.0 

  - Microsoft Site Analyst

SYMPTOMS

When you select the tree view of Site Analyst, Print Preview is displayed using the Symbols font.

CAUSE

The system does not have the Arial fonts loaded on it.

WORKAROUND

Install the Arial fonts on the server from the installation disk.
