CRS: Pgiparse.dll Is Not Updated with MCIS SP2

 Last reviewed: February 2, 1998
Article ID: Q173534
The information in this article applies to:
  • Microsoft Commercial Internet System, version 1.0

SYMPTOMS

After you apply Microsoft Commercial Internet System Service Pack 2, the Pgiparse.dll file is not updated.

WORKAROUND

To work around this problem, do the following:

  1. Copy c:\%systemroot%\system32\pgiparse.dll <crs directory>.
2. Install Service Pack 2. 3. Copy <crs directory>\pgiparse.dll c:\%systemroot%\system32.

Microsoft Commercial Internet System 1.0 has Pgiparse.dll version 5.00.7494, and Microsoft Commercial Internet System SP2 has version 5.00.7726. 

Version           : 1.00
Platform          : WINDOWS
Issue type        : kbbug
