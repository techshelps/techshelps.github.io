InstallShield Setup Wizard Fails

 Last reviewed: March 6, 1998
Article ID: Q159939
The information in this article applies to:
  • Microsoft Commercial Internet System, version 1.0

SYMPTOMS

The following problem occurs when the registry entry EnableAlignmentFaultExceptions is enabled (that, when it is set to 1. The InstallShield Setup Wizard fails for all Commercial Internet System components.

This generates the following Dr. Watson log: 

   An application error has occurred.
The file noted in the dialog box is _INSO433._MP.exe. The OK and Cancel buttons are grayed out. No log is created.

WORKAROUND

To resolve this problem, modify the following registry value:

WARNING: Using Registry Editor incorrectly can cause serious problems that may require you to reinstall Windows NT. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use Registry Editor at your own risk.

  1. Start Registry Editor (Regedt32.exe).

  2. Under the HKEY_LOCAL_MACHINE subtree, go to the following subkey: 

          System\CurrentControlSet\Control\
      SessionManager:EnableAlignmentFaultExceptions

  3. Set this value to 0.

  4. Quit Registry Editor.
