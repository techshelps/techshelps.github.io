INFO: IE4 PP2 Win32 Virtual Machine for Java File List

 Last reviewed: December 9, 1997
Article ID: Q172437
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0
  • Microsoft Win32 Virtual Machine for Java
  • SDK for Java, version 2.0

SUMMARY

The current version (or build number) of the Microsoft Win32 Virtual Machine (VM) for Java is 2174.

MORE INFORMATION

Typing "jview" in a Command window displays the version of the currently installed Java VM.

The Microsoft VM for Java consists of the following files for the X86 platform:

NOTE: The versions of all files are the same in Internet Explorer (IE) 4.0 Platform Preview 2 and the SDK for Java Beta 2 except for Msjava.dll and Clspack.exe. In the SDK, the Msjava.dll version number is 2175, with a size of 670,208 and a date of 7/15/97. The SDK's Clspack.exe utility is version number 2174, with a size of 10,752 and a date of 7/14/97.

To find the version of a file using Explorer, find and highlight the file, right-click, and select Properties from the Shortcut menu.

The Internet Explorer 4.0 PP2 public beta release is available for download at http://www.microsoft.com/ie/. The SDK for Java 2.0 Beta 2 is available for download at http://www.microsoft.com/java.
