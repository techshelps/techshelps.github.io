The information in this article applies to:

  • Microsoft FORTRAN for MS-DOS, versions 5.0 and 5.1
  • Microsoft FORTRAN PowerStation for MS-DOS, versions 1.0 and 1.0a
  • Microsoft FORTRAN PowerStation 32 for Windows NT, version 1.0 and 4.0
  • Microsoft Visual C++ for Windows, versions 1.0 and 1.5
  • Microsoft Visual C++ for Windows NT, version 1.0

Microsoft IMSL Mathematical and Statistical Libraries, for both 16-bit and 32-bit applications, contain FORTRAN subroutines that are callable from programs developed with Microsoft FORTRAN, and C/C++ products. The support for these libraries is provided by Visual Numerics Product Support between the hours of 8:00 AM and 6:00 PM Central time, Monday through Friday, at (713) 954-6464. You can also contact Visual Numerics by FAX at (713) 781- 9260, or by electronic mail at pcsupport@houston.vni.com.

For sales information please contact Microsoft at (206) 426-9400.
