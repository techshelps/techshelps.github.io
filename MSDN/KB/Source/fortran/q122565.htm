FIX: FORTRAN PowerStation 32 Update for Windows NT 3.5

 Last reviewed: September 18, 1997
Article ID: Q122565
1.00 WINDOWS NT kbinterop kbtool kbfile kbbuglist kbfixlist

The information in this article applies to:

  • Microsoft FORTRAN PowerStation 32 for Windows NT, version 1.0

SYMPTOMS

The linker (version 1.0, dated 23-Jun-1994) included with Microsoft FORTRAN PowerStation 32 interacts incorrectly with Microsoft Windows NT version 3.5, but works fine with Windows NT version 3.1.

STATUS

Microsoft has confirmed this to be a problem with the version of LINK.EXE (version 1.0, dated 23-Jun-1994) that shipped with Microsoft FORTRAN PowerStation 32 version 1.0 for Windows NT. To correct this problem, please obtain the new version of LINK.EXE (version 1.0, dated 9-Aug-1994) that corrects the problem and replace the old LINK.EXE.

MORE INFORMATION

To obtain the update, download FPSNTU01.EXE, a self-extracting file, from the Microsoft Software Library (MSL) on the following services:

  • Microsoft Download Service (MSDL) 

          Dial (206) 936-6735 to connect to MSDL
      Download FPSNTU01.EXE (size: 209542 bytes)
  • Internet (anonymous FTP) 

          ftp ftp.microsoft.com
      Change to the SOFTLIB\MSLFILES directory
      Get FPSNTU01.EXE (size: 209542 bytes)
