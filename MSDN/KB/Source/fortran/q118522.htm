Numerical Recipes Disk Included with PowerStation

 Last reviewed: December 9, 1995
Article ID: Q118522
The information in this article applies to:
  • Microsoft FORTRAN PowerStation for MS-DOS, versions 1.0 and 1.0a
  • Microsoft FORTRAN PowerStation 32 for Windows NT, version 1.0 and 4.0

A disk containing numerical algorithms was included in some boxes of FORTRAN PowerStation. Support for this disk, including replacement disks, is provided by Numerical Recipes Software. You can also order copies of the "Numerical Recipes" book from Cambridge University Press, which contains the source code and explains the algorithms.
