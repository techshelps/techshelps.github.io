FORTRAN 5.1 Packing List

 Last reviewed: July 17, 1995
Article ID: Q73267
The information in this article applies to:
  • Microsoft FORTRAN for MS-DOS, version 5.1
  • Microsoft FORTRAN for OS/2, version 5.1

SUMMARY

This article contains the contents of the Microsoft FORTRAN Compiler version 5.10 PACKING.LST file for the high density 5.25-inch and 3.5-inch disk sets.
