OK Button Must Be Tapped Twice for Calendar Error Message

 Last reviewed: June 17, 1997
Article ID: Q170217
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

If you receive an error message while you are creating a recurring appointment in Calendar, you must tap the OK button twice instead of once to close the error message.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
