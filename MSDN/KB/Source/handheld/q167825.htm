Availability of Pocket Internet Explorer 1.1 for Windows CE

 Last reviewed: May 1, 1997
Article ID: Q167825
The information in this article applies to:
  • Microsoft Pocket Internet Explorer version 1.1

SUMMARY

This article describes the availability of Microsoft Pocket Internet Explorer version 1.1 for Windows CE.

MORE INFORMATION

Pocket Internet Explorer version 1.1 is available for download from Microsoft's Internet site at: 

   http://www.microsoft.com/windowsce/hpc/software/mspie.htm.
Pocket Internet Explorer version 1.1 includes the following new features:
  • Cookies
  • Caching
  • Save As
  • Security

NOTE: You must download the Security version of Pocket Internet Explorer to obtain security functionality.

For a list of Internet technologies that are not supported in Pocket Internet Explorer, see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q161319
   TITLE:      Unsupported Internet Technology with Pocket Internet  
               Explorer
For a list of HTML tags that are not supported in Pocket Internet Explorer, see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q158479
   TITLE:      Unsupported HTML Tags for Pocket Internet Explorer
