Keyboard Shortcuts for Microsoft Windows CE

 Last reviewed: February 18, 1998
Article ID: Q181331
The information in this article applies to:
  • Microsoft Windows CE, versions 1.0, 2.0

SUMMARY

This article lists keyboard shortcuts you can use in Windows CE.

MORE INFORMATION

This table lists the keyboard shortcuts you can use in Windows CE. 

   Key                               Result
   ----------------------------------------------------------------------
   ALT+ENTER                         Open the properties for the
                                     selected item.
   ALT+ESC                           Switch to another running program.
   ALT+TAB                           Start Task Manager.
   ALT+tap (tap with the stylus)     Open the shortcut menu for the item
                                     you tap.
   CTRL+ALT+DELETE                   Start Task Manager.
   CTRL+ESC                          Display the Start menu and the
                                     taskbar.
   SHIFT+BACKSPACE                   Delete the selected item.
   Windows logo                      Display the Start menu and the
                                     taskbar.
   Windows logo+MINUS SIGN           Start PC Link to connect to your
                                     desktop computer.
   Windows logo+C                    Open Control Panel.
   Windows logo+E                    Explore My Handheld PC.
   Windows logo+H                    Start Help in Windows CE.
   Windows logo+I                    Display the Stylus tool in Control
                                     Panel.
   Windows logo+K                    Display the Keyboard tool in Control
                                     Panel.
   Windows logo+M                    Minimize all open windows.
   Windows logo+R                    Open the Run dialog box.
   Windows logo+SPACEBAR             Display the Windows CE Keyboard
                                     Shortcuts topic in Help.
