Cannot View Tables in Web Sites Using Pocket Internet Explorer

 Last reviewed: December 11, 1997
Article ID: Q177984
The information in this article applies to:
  • Microsoft Windows CE, version 2.0

SYMPTOMS

When you use Microsoft Pocket Internet Explorer to view a Web site that contains a wide table, you may not be able to view the entire table.

CAUSE

This behavior can occur if you are using the Fit To Window feature.

RESOLUTION

To resolve this behavior, do not use the Fit To Window feature. To do so, use the following steps:

  1. Double-tap the The Internet icon.

  2. On the View menu, tap the Fit To Window command to remove the check mark.
