Default Fonts Included with Windows CE

 Last reviewed: April 14, 1997
Article ID: Q158510
The information in this article applies to:
  • Microsoft Windows CE version 1.0

The following default fonts are included with Windows CE:

  • Arial 8,9,10,12,16,20
  • Comic Sans MS 10
  • Courier New 8,10
  • MS Sans Serif 8,10
  • Symbol 10,12
  • Terminal 5
  • Times New Roman 8,10,12,14,18,22
  • Wingdings 10

These fonts are system fonts. They are hard-coded in the graphical user interface (GUI) and cannot be removed. No specific files are responsible for these fonts and they are referred to as resources. Additional fonts that are added to the H/PC are stored in the Windows\Fonts folder.
