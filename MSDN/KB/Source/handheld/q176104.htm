Undo Copy Command Deletes File Without a Warning Message

 Last reviewed: November 3, 1997
Article ID: Q176104
The information in this article applies to:
  • Microsoft Windows CE, version 2.0

SYMPTOMS

When you are using Windows Explorer on a Handheld PC (H/PC) and you tap Undo Copy on the Edit menu, the most recently created copy file may be deleted without a warning message.

STATUS

This behavior is by design.

MORE INFORMATION

Windows Explorer does not prompt you to confirm that you want to delete the copy file when you use the Undo Copy command. The original file is available to be copied again.
