Cannot Open Contacts Due to Lack of Available Storage Space

 Last reviewed: April 14, 1997
Article ID: Q162828
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SYMPTOMS

When you attempt to run Contacts, you may receive the following error message: 

   Out of storage memory. Unable to complete this operation.
When you close this error message, Contacts closes as well.

CAUSE

This problem can occur if there is not enough available storage memory to open Contacts.

RESOLUTION

To resolve this problem, use one or more of the following methods:

  • Increase available storage memory. To do so, follow these steps:

    1. Tap the Start button, tap Settings, and then double-tap System.

    2. Tap the Memory tab, move the slider to the right, and then tap OK.

    NOTE: Do not move the slider all the way to the right, as this may cause a lack of available program memory.

  • Delete any unneeded files to free additional storage memory.
