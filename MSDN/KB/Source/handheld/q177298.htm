Changes in E-mail Messages Are Not Synchronized

 Last reviewed: November 25, 1997
Article ID: Q177298
The information in this article applies to:
  • Microsoft Windows CE, versions 1.0, 2.0

SYMPTOMS

When you edit an e-mail message using Microsoft Pocket Outlook, and you then synchronize your Handheld PC (H/PC) with your desktop computer, the e-mail message is not updated on the desktop computer.

CAUSE

This issue occurs because Windows CE Services checks for and synchronizes changes made only to the following e-mail message attributes:

  • Read
  • Unread
  • Delete

NOTE: You can synchronize an e-mail message edited on your desktop computer with your H/PC.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
