Double-Tapping an Appointment Creates a New Appointment

 Last reviewed: November 25, 1997
Article ID: Q177296
The information in this article applies to:
  • Microsoft Windows CE, version 2.0

SYMPTOMS

When you try to edit an existing appointment in Calendar, you may create a new appointment.

CAUSE

This problem can occur if you double-tap the appointment when you can make changes to the appointment description.

RESOLUTION

To work around this problem, use the following steps:

  1. In Calendar, tap the appointment you want.

  2. On the Edit menu, tap Edit Item.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Pocket Outlook, version 2.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you view a daily or weekly calendar, you can tap an appointment to select it, and then tap the appointment again if you want to modify the appointment description. When you tap the appointment the second time, an insertion point appears and you can then make changes to the appointment description.
