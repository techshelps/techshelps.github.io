HTML Tags Supported in Pocket Internet Explorer

 Last reviewed: February 2, 1998
Article ID: Q158479
The information in this article applies to:
  • Microsoft Windows CE, versions 1.0, 1.01, 2.0

SUMMARY

This article lists the support in Microsoft Pocket Internet Explorer versions 1.0, 1.1, and 2.0 for the following items:

  • Hypertext Markup Language (HTML) tags
  • Multimedia Internet Mail Extension (MIME) formats
  • Security types
  • Universal Resource Locator (URL) formats

This table lists the security types supported in Pocket Internet Explorer versions 1.0, 1.1, and 2.0. 

   Security type                            1.0   1.1   2.0
   --------------------------------------------------------

   Secure Sockets Layer 2 (SSL2)            No    Yes   Yes
   Secure Sockets Layer 3 (SSL3)            No    Yes   Yes
   Private Communication Technology (PCT)   No    Yes   Yes
NOTE: PCT is not enabled in the French version of Windows CE.

This table lists the URL formats supported in Pocket Internet Explorer versions 1.0, 1.1, and 2.0. 

   URL formats                                1.0   1.1   2.0
   ----------------------------------------------------------

   HyperText Transfer Protocol (HTTP)         Yes   Yes   Yes
   Secure HTTP (HTTPS)                        No    Yes   Yes
   File Transfer Protocol (FTP)               Yes   Yes   Yes
   Universal Naming Convention (UNC)          Yes   Yes   Yes
NOTE: Pocket Internet Explorer 1.0 is included with Windows CE 1.0. Pocket Internet Explorer 1.1 is available on the Internet. Pocket Internet Explorer 2.0 is included with Windows CE 2.0.
