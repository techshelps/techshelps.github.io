H/PC Explorer Version 1.1 Availability and Features

 Last reviewed: February 18, 1998
Article ID: Q170483
The information in this article applies to:
  • Microsoft Windows CE version 1.0
  • Microsoft H/PC Explorer version 1.1

SUMMARY

Microsoft has released Microsoft H/PC Explorer version 1.1 for Windows 95 and Windows NT Workstation 4.0. New features in version 1.1 include:

  • Support for Windows NT Workstation 4.0-based desktop computers.
  • Synchronization with Microsoft Outlook 97.
  • Conversion of files from Microsoft Word 97 and Microsoft Excel 97 to Microsoft Pocket Word and Microsoft Pocket Excel formats.
  • An updated End User License Agreement that allows you to install H/PC Explorer version 1.1 on any desktop computer with which you want to dock your Handheld PC (H/PC).

MORE INFORMATION

H/PC Explorer 1.1 is available for download from Microsoft's Web site at the following location: 

   http://www.microsoft.com/windowsce/hpc/software/hpcexplorer.htm
For additional information about H/PC Explorer, visit the following Microsoft Web site: 

   http://www.microsoft.com/windowsce/
