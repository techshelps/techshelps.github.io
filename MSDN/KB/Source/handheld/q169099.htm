Cannot Add to List of Prefixes and Suffixes in Contacts

 Last reviewed: May 27, 1997
Article ID: Q169099
The information in this article applies to:
  • Microsoft Windows CE version 1.0

SUMMARY

You cannot add name prefixes and suffixes to the default lists of prefixes and suffixes provided in the Contacts program.

MORE INFORMATION

When you create or edit an entry in the Contacts program, the Parsing button is displayed next to the Name box. If you tap this button, the Confirm Name dialog box is displayed. The Confirm Name dialog box lists the following fields:

  • Mr/Ms/Dr
  • Suffix
  • First
  • Middle
  • Last

The Mr/Ms/Dr and Suffix fields include default entries for commonly used prefixes or suffixes. If a name prefix or suffix is not listed, you can manually enter the prefix or suffix for the contact entry. However, the prefix or suffix is not added to the list of default entries.
