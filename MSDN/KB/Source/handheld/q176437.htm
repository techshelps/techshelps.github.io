Err Msg: Error Getting Message: Invalid Message ID Specified...

 Last reviewed: November 7, 1997
Article ID: Q176437
The information in this article applies to:
  • Microsoft Windows CE, version 2.0

SYMPTOMS

When you download an e-mail message from a Post Office Protocol (POP3) e-mail server, you may receive the following error message: 

   Error Getting Message: Invalid Message ID Specified (Service Error: 25)

CAUSE

This error message can occur if an e-mail message is marked to be copied or moved in Microsoft Pocket Outlook.

RESOLUTION

To resolve this behavior, use the following steps:

  1. Double-tap Inbox.

  2. Double-tap each e-mail message that is marked to be copied or moved.

    Note that an icon that looks like a sheet of paper with an arrow pointing down indicates an e-mail message that has been marked to be copied or moved.

  3. Connect your Handheld PC (H/PC) to your desktop computer.

  4. Tap Start, tap Programs, tap Communications, and then tap ActiveSync.

  5. Tap Sync Now, and then tap Disconnect.
