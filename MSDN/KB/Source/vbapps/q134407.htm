XL7: Can't Run Solver Help, Receive Error About Macrofun.hlp

 Last reviewed: September 2, 1997
Article ID: Q134407
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel for Windows 95, version 7.0, if you try to access the Help topics for the Visual Basic for Applications commands for the Solver tool, the following message is displayed: 

   Macro Function Help file not installed

CAUSE

This error message is displayed because Microsoft Excel attempts to load the Help topics from the Macrofun.hlp Help file; Macrofun.hlp is not included with Microsoft Excel for Windows 95, version 7.0.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you try to access Help for one of these Visual Basic commands, the following help topic is displayed: 

   Macro Function Help file not installed

   To display help topics for Microsoft Excel 4.0 macro functions, you must
   first install the complete Macro Function help file. You can obtain the
   complete Macro Function help file (Macrofun.hlp), from CompuServe or
   from the Microsoft Network.

   To obtain the Macro Function Help file from Compuserve, type "go
   msexcel" (without the quotation marks) from any ! prompt and look for
   the Macrofun.zip file. To obtain the Macro Function Help file from the
   Microsoft Network, look on the Microsoft forum for information about
   obtaining Macrofun.hlp. If you are a member of the Microsoft Network,
   you can connect to the Microsoft forum by clicking here [>>].

   To install the Macro Function help file, copy the Macrofun.hlp file you
   have obtained to the Excel folder, replacing the copy of Macrofun.hlp
   that is already there.

   NOTE: The copy of Macrofun.hlp currently in your Excel folder contains
   only this topic plus topics documenting statistical functions for the
   Analysis ToolPak.
NOTE: The name of the file to download is Macrofun.exe, which is a self- extracting .exe file.

REFERENCES

For additional information, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q128185
   TITLE     : Macrofun.exe Available for Download on CompuServe
For more information about the Visual Basic Solver commands, choose the Index tab in Microsoft Excel Visual Basic Help and type: 

   Solver
The available Solver commands will be listed.
