PPT97: Run-Time Error Using Assistant.Application

 Last reviewed: February 5, 1998
Article ID: Q163610
The information in this article applies to:
  • Microsoft PowerPoint 97 for Windows

SYMPTOMS

You receive the following error message when you attempt to access the Application property from the Assistant property (for example, Assistant.Application). 

   Run-time error '-2147467259 (80004005)':
   Method 'Application' of object 'Assistant' failed

CAUSE

In the PowerPoint object model, the Application property is not accessible through the Assistant property.

RESOLUTION

This article assumes that you are familiar with the programming language being demonstrated and the tools used to create and debug procedures. For more information about the support options available from Microsoft, please see the following page on the World Wide Web: 

   http://www.microsoft.com/support/supportnet/refguide/default.asp
To prevent the error message, use the Application property directly: 

   Sub UseApplicationProperty
      Application.<Name of property or method>
   End Sub

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft PowerPoint 98 Macintosh Edition.

MORE INFORMATION

The Application property returns an Application object that represents the container application for the specified object. You can use the Application property with an Automation object to return that object's container application.
