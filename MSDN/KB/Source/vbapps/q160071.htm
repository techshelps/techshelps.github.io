WD97: VB Code to Check, Uncheck Word Dialog Box Check Boxes

 Last reviewed: February 11, 1998
Article ID: Q160071
The information in this article applies to:
  • Microsoft Word 97 for Windows

SUMMARY

In Word 97, when you click to select a check box in a dialog box, the check box has a value of true (-1). When you click to clear a check box, the check box has a value of False (0). This article provides a sample Microsoft Visual Basic for Applications macro that switches between the true and false state.

The following Visual Basic example turns the Background Repagination option on or off each time you run the macro. (To locate this option, on the Tools menu, click Options, and then click the General tab.) When the Background Repagination check box is selected its state is "true"; when the check box is cleared its state is "false."

When you use the NOT operator, you can switch between the true and false states each time you run the macro. When you use Options.Pagination=True, the option is turned on; when you use Options.Pagination=False, the option is turned off. 

   Sub ToggleCheckBoxes()
      'Switches between checked and unchecked states
      Options.Pagination = Not Options.Pagination
   End Sub
For more information about changing values in Word 97 dialog boxes, from the Visual Basic for Applications Editor, click the Office Assistant, type "built-in dialog boxes" (without the quotation marks), click Search, and then click to view "Displaying built-in Word dialog boxes."
