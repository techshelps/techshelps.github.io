XL97: Status Bar Text not Displayed when you Point to Menu Item

 Last reviewed: March 13, 1998
Article ID: Q161509
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, no text is displayed in the status bar if you place your mouse pointer over a toolbar button or menu item.

STATUS

This is by design in Microsoft Excel 97.

MORE INFORMATION

In earlier versions of Microsoft Excel, you can set status bar text for a Visual Basic for Applications procedure and the status bar text for the procedure appears in the status bar when you place your mouse pointer over the toolbar button or menu item that runs the procedure.

In Microsoft Excel 97, if you have a Visual Basic for Applications procedure that has the StatusBar parameter set, the parameter is ignored.
