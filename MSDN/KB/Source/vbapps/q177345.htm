VBE: Underscore Characters Not Printed from VB Editor

 Last reviewed: February 4, 1998
Article ID: Q177345
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you print Visual Basic for Applications module code, if the code contains an underscore character, the underscore character is not printed.

CAUSE

The Visual Basic Editor font may be set to Courier.

WORKAROUND

To work around this problem, change the Visual Basic Editor font from Courier to Arial or Times New Roman.

To change the Visual Basic Editor font, follow these steps while in the Visual Basic Editor:

  1. On the Tools menu, click Preferences.

  2. In the Preferences dialog box, click the Editor Format tab.

  3. Select Arial or Times New Roman in the Font list.

  4. Click OK.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
