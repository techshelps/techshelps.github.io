ACC95: Constants in the Microsoft Access 95 Object Library

 Last reviewed: August 28, 1997
Article ID: Q140214
The information in this article applies to:
  • Microsoft Access version 7.0

SUMMARY

This article lists the constant variables that are in the Microsoft Access for Windows 95 version 7.0 object library. The discrepancy in values for some of these constants is a result of differences in the menu organization between versions. You should not mix version 7.0 constants and constants from earlier versions of Microsoft Access in the same DoMenuItem statement.

MORE INFORMATION

A listing of all the available constants in the Microsoft Access 7.0 object library is available by using the Object Browser. To use the Object Browser, open a module in Design view, press F2, and select "Access - Microsoft Access for Windows 95" in the Libraries/Database box. Then, select "Constants" in the Modules/Classes box.

NOTE: In Microsoft Access 97, you can easily display a list of all the available constants by typing CTRL+J while in Design view of a module. You can also display the list of available constants by pressing F2 while in Design view of any module to bring up the Object Browser. Then, click Access or another library in the Project/Library box and select the class you want information on in the Classes box. If you would like to see a list of all members that can be assessed globally, click <Globals> in the Classes box.
