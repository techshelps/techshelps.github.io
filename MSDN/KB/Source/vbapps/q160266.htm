XL97: UserForm ListBox Is Blank When Run from a Binder Section

 Last reviewed: February 27, 1998
Article ID: Q160266
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you run a Visual Basic for Applications UserForm that contains a ListBox, and you are running the UserForm from a Microsoft Office Binder section, the ListBox appears to be blank.

CAUSE

This behavior occurs if the ListBox on the UserForm is linked to the worksheet. When you open the link in the Office Binder section, the link is not updated.

NOTE: If you open the file in Microsoft Excel, the UserForm ListBox is populated. A ListBox you create on worksheets using the Forms tools (instead of UserForms) is populated in both Microsoft Excel and in a Binder section.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about creating ListBoxes, click the Office Assistant, type "list boxes, creating", click Search, and then click to view "Create a standard list box".

NOTE: If the Assistant is hidden, click the Office Assistant button on the Standard toolbar. If Microsoft Excel Help is not installed on your computer, please see the following article in the Microsoft Knowledge Base:
