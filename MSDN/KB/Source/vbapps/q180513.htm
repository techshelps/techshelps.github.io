WD97: VBA Macro to List All Documents Found in a Folder

 Last reviewed: February 11, 1998
Article ID: Q180513
The information in this article applies to:
  • Microsoft Word 97 for Windows

SUMMARY

This article contains a sample Microsoft Visual Basic for Applications macro (Sub procedure) that can be used to create a list of file names found in a specified folder.

The following sample macro creates a new document with the file names from the specified folder. 

   Sub ListDocNamesInFolder()

      Dim MyDir As String
      Dim aDoc As String

      ' Enter the path to obtain files from here.
      MyDir = "C:\My Documents\"

      ' To search for documents with a particular extension, enter
      ' it here. For example, to return text files, "*.TXT", change
      ' the following from "*.DOC" to "*.TXT".
      aDoc = Dir(MyDir & "*.DOC")

      ' Add new document.
      Documents.Add

      ' Insert folder name in document.
      Selection.TypeText "Documents in folder " & MyDir & vbCr & vbCr

      While aDoc <> ""

         ' Insert filename in document.
         Selection.TypeText aDoc & vbCr

         ' Get next file name.
         aDoc = Dir()

      Wend

   End Sub
