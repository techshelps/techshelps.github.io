WD97: IPF Using Linked Image SourceName/SourcePath Property

 Last reviewed: March 3, 1998
Article ID: Q176995
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you use the SourcePath or SourceName properties of the LinkFormat object on a linked image in a document, Word stops responding, or you receive the following error message: 

   This program has performed an illegal operation and will be shut down.

   If the problem persists, contact the program vendor.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Word 98 Macintosh Edition.
