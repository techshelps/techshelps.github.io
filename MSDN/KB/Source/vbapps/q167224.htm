PPT97: Macro Recorder Does Not Record Show Command

 Last reviewed: December 2, 1997
Article ID: Q167224
The information in this article applies to:
  • Microsoft PowerPoint 97 for Windows

SYMPTOMS

When you record a macro that runs the Show command on the shortcut menu, which appears when you right-click a file in the File Open dialog box, the macro recorder does not record any code.

CAUSE

The PowerPoint object model does not record commands to show a presentation if the presentation is not already open (part of the current Presentations collection).
