XL97: WE1163: "Visual Basic Examples for Controlling UserForms"

 Last reviewed: February 27, 1998
Article ID: Q168067
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SUMMARY

The Application Note "Visual Basic for Applications Examples for Controlling UserForms" (WE1163) is now available from Microsoft Product Support Services. This Application Note is an introduction to manipulating UserForms. It includes examples and Microsoft Visual Basic for Applications macros that show you how to take advantage of the capabilities of UserForms and use each of the ActiveX controls that are available for UserForms.
