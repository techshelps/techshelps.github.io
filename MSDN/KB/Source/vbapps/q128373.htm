XL5: Macro Cannot Determine If Sheet Is xlVeryHidden

 Last reviewed: September 2, 1997
Article ID: Q128373
5.00 WINDOWS kbprg kbcode

The information in this article applies to:

- Microsoft Excel for Windows, version 5.0

SYMPTOMS

In Microsoft Excel 5.0 for Windows, you can set the Visible property of a sheet to True, False, or xlVeryHidden. However, you may not be able to differentiate whether or not an invisible sheet is "Very Hidden" or just "Hidden" (false). In either case, the return value of the Visible property for the sheet is False.

STATUS

Microsoft has confirmed this to be a problem with Microsoft Excel version 5.0 for Windows. This problem has been corrected in Microsoft Excel version 5.0c for Windows.
