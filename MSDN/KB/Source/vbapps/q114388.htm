XL: Error Message: FUNCRES.XLA Is Not a Valid Add-in

 Last reviewed: September 2, 1997
Article ID: Q114388

The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for Windows, version 5.0

SUMMARY

When you attempt to load Funcres.xla by using the add-in manager, Microsoft Excel will generate the error message: 

   'FUNCRES.XLA' is not a valid add-in.
NOTE: In Microsoft Excel 97, the path to the add-in will precede the name of the add-in in the above error message.

CAUSE

Funcres.xla is a support add-in. The Analysis ToolPak automatically loads and uses Funcres.xla; therefore, it is not necessary to load this add-in manually.

RESOLUTION

You do not need to load the Funcres.xla file manually. To avoid the above error message, you can open Funcres.xla by clicking Open on the File menu.

STATUS

This is expected behavior of the Funcres.xla add-in.
