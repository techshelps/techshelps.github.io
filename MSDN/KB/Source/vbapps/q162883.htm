Off97: Visual Basic Help File For Microsoft Graph Available

 Last reviewed: March 13, 1998
Article ID: Q162883
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Access 97
  • Microsoft Excel 97 for Windows

SUMMARY

The Visual Basic for Applications help file for Microsoft Graph is included in the ValuPack folder on the Microsoft Office 97 compact disc. The same help file (Vbagrp8.exe) is now available for download from the Microsoft Software Library: 

 ~ Vbagrp8.exe (size: 256837 bytes) 
                 251 KB self-extracting file
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
