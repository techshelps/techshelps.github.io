WD97: No Error Inserting Invalid Picture Via Macro

 Last reviewed: February 11, 1998
Article ID: Q165600
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you run a Visual Basic for Applications macro that inserts an invalid picture into a document, you do not receive a trappable error. When you attempt to insert the same picture using the Picture command on the Insert menu, you receive an error message.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
