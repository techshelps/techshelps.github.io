Visual SourceSafe Shortcut Keys

 Last reviewed: March 25, 1996
Article ID: Q138385
The information in this article applies to:
  • Microsoft Visual SourceSafe for Windows, versions 4.0
  • Microsoft Visual SourceSafe for Windows NT, versions 4.0
  • Microsoft Visual SourceSafe for Windows 95, versions 4.0

SUMMARY

The following list contains the shortcut keys that are available from within the SourceSafe Explorer. This functonality may not be available from other applications that integrate with Visual SourceSafe.

MORE INFORMATION

Key                     Function

F1                      Help
F5                      Refresh File List
F6                      Next Pane
TAB                     Next Pane
ENTER                   View
INSERT                  Add/Create (file/project list)
DELETE                  Delete
ALT+BACKSPACE           Undo
ALT+ENTER               Properties Dialog Box
ALT+X                   Exit
CTRL+A                  Add File
CTRL+D                  Set Working Directory
CTRL+G                  Get
CTRL+F                  Show Differences
CTRL+H                  Show History
CTRL+I                  Check in the selected file(s)
CTRL+INSERT             Copy the selected text to the Clipboard
CTRL+K                  Check Out
CTRL+L                  Select
CTRL+N                  File New
CTRL+O                  Check out a file
CTRL+Q                  Cancel Search
CTRL+R                  Last Results In Visual Sourcesafe 4.0a, this Views
                        a File.
CTRL+S                  Status Search
CTRL+U                  Check In
CTRL+W                  Wildcard Search
CTRL+X                  Cut text and place it on the Clipboard
CTRL+Z                  Undo
CTRL+ENTER              Views a file
SHIFT+CTRL+S            Quick Status Search
SHIFT+CTRL+L            Select All
SHIFT+DELETE            Cut selected text and copy to the Clipboard
SHIFT+INSERT            Paste text from the Clipboard to the edit box
SHIFT+F6                Previous Pane
+                       Expand Project
-                       Collapse Project
*                       Recursively Expand Project
CTRL+*                  Globally expand all projects
HOME                    Moves to and opens the Root project
END                     Moves to and opens the Bottom most project

REFERENCES

Visual SourceSafe Quick Reference Card

For more information about the Visual SourceSafe shortcut keys, search for Shortcut Keys in the Help menu.
