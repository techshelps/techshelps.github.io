INFO: New Files in Visual SourceSafe Version 4.0

 Last reviewed: January 12, 1998
Article ID: Q138342
The information in this article applies to:
  • Microsoft Visual SourceSafe for Windows, version 4.0
  • Microsoft Visual SourceSafe for Windows NT, version 4.0
  • Microsoft Visual SourceSafe for Windows 95, version 4.0

SUMMARY

Visual SourceSafe version 4.0 has a new set of files that were not available in version 3.1 or that have been moved to new locations. The files listed in this article are contained in the \Dos, \Win, and \Win32 directories unless otherwise noted.

MORE INFORMATION

File            Description

12520437.BIN    ANSI to OEM conversion table for MS-DOS.
ANALYZE.EXE     Detects and reports database errors without repairing them.
CTL3D32.DLL     3-D controls (32-bit).
CTL3DV2.DLL     3-D controls (16-bit).
DDCONV.EXE      Converts the database to VSS 4.0 format and repairs some
                   data errors.
DELTA_SS.EXE    Delta to SourceSafe utility for MS-DOS.
MFC30.DLL       32-bit MFC DLL used by the SSGUI.DLL.
MFC250.DLL      16-bit MFC DLL used by the SSGUI.DLL.
MKSS.EXE        Creates a new SourceSafe database.
PVCS_SS.EXE     PVCS to SourceSafe conversion utility for MS-DOS.
SS.EXE          SourceSafe command line.
SSADMIN.EXE     SourceSafe administration program.
SSAPI.DLL       32-bit API engine.
SSAPI16.DLL     16-bit API engine.
SSEXP.EXE       SourceSafe Explorer graphical interface.
SSINIT.COM      TSR login user name buffer for MS-DOS.
SSINIT.EXE      Updates the 16-bit or 32-bit Visual Basic initialization
                file (WINDOWS\VB.INI), as appropriate to which SSINT.EXE
                program is executed (win\ssint.exe or win32\ssint.exe).
SSLOGIN.EXE     SourceSafe login user name utility.
SSSGUI.DLL      32-bit MFC extension DLL for SSEXP.
SSGUI16.DLL     16- bit MFC extension DLL for SSEXP.
SSVB.DLL        32-bit VB add-in to call the SourceSafe API.
SSVB16.DLL      16-bit VB add-in to call the SourceSafe API.
SSxx.DLL        SourceSafe resource DLLs for the following countries:
ssxx16.DLL      US   United States
                DE   Germany
                FR   France
                IT   Italian
                JP   Japanese
                ES   Spanish
SSxxADM.HLP     Localized SourceSafe administration help file using the
                   previously listed country codes.
SSxxEXP.HLP     Localized SourceSafe Explorer help file.
TEMPLATE.INI    SS.INI template file.
TESTLOCK.EXE    Tests file locking for SourceSafe.
UPDINI.EXE      Converts INI files to the current format.
UNLOCK.EXE      Unlocks files.
