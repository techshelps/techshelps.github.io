PC DOS: How to Create Subfolders in the MS-DOS Client

 Last reviewed: October 20, 1995
Article ID: Q112203
The information in this article applies to:
  • Microsoft Mail for PC Networks, MS-DOS Workstation, versions 3.0 and 3.2

The following are the steps to create subfolders in the Microsoft Mail for PC Networks MS-DOS client.

  1. From the MS-DOS client (MAIL.EXE), choose Options, Folder, Modify. Select the folder that will contain the subfolder.

  2. Press the RIGHT ARROW key to display the Create form. At the top of the Create form is the folder path for the parent folder(s).

  3. Type a name for the subfolder you are creating and any other related changes to the folder.
