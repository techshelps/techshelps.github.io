FAX: Special Formats for Profile Parameters

 Last reviewed: October 20, 1997
Article ID: Q85356
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to FAX, version 3.0

In versions of the Microsoft Mail Gateway to Fax earlier than version 3.0, the following parameters need to be formatted correctly: 

   Parameter      Action
   ---------      ------

   Printer path   Add a backslash (\) at the end of the path.
   Logo file      Add a backslash (\) at the end of the path.
The semicolons and comments in the profile file should not be removed.

In version 3.0 of the Microsoft Mail Gateway to Fax, the PROFILE.EXE program is used to edit the PROFILE.GLB file; there are no special formats.

REFERENCES

"Microsoft Mail Gateway to Fax Administrator's Guide."
