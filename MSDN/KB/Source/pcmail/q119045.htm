PC Mac: Err Msg: This Message is Corrupt and Cannot Be...

 Last reviewed: October 26, 1994
Article ID: Q119045
The information in this article applies to:
  • Microsoft Mail for PC Networks, Macintosh workstation, version 3.0

SYMPTOMS

When using version 3.0 of Microsoft Mail for PC Networks, Macintosh workstation and attempting to read a mail message sent from Microsoft Mail for Windows NT (32-bit Mail client), the following error may occur: 

   This message is corrupt and cannot be accessed. Contact your System
   Administrator.
NOTE: The Microsoft Mail for PC Networks, MS-DOS workstation does not have a problem viewing the message.

STATUS

Microsoft has confirmed this to be a problem in version 3.0 and later of Microsoft Mail for PC Networks, Macintosh workstation. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
