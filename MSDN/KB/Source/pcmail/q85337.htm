FAX: Hardware Required of the Fax Gateway PC

 Last reviewed: October 20, 1997
Article ID: Q85337
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to FAX, version 3.0

The Microsoft Mail Gateway to Fax program requires this hardware:

  • Its own workstation on the network
  • A network connection to the MS Mail database
  • A fax board installed

For more information on compatible fax boards, query on these words in the Microsoft Knowledge Base: 

   fax and boards and Fax Gateway
If you are using one of the Intel fax boards, the gateway needs a dedicated IBM PC/XT/AT or 100-percent compatible with at least 512K of memory. If you are using the GammaLink GammaFax board, the gateway needs an IBM PC or compatible 386 with a minimum 2 MB of memory.

The gateway also needs an outside telephone line.

REFERENCES

The "System Requirements" chapter of the "Microsoft Mail Gateway to Fax Administrator's Guide"
