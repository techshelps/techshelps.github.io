PC OS2: PM Client Does Not Support UNC Names

 Last reviewed: October 12, 1994
Article ID: Q86314
The information in this article applies to:
  • Microsoft Mail for PC Networks, OS/2 workstation, version 3.0

Making run-time net connections is not supported under Windows Libraries for OS/2 (WLO) and is not implemented in Microsoft Mail for Presentation Manager. The network connection must be established before you start Mail.
