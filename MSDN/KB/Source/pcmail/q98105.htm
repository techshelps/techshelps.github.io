PC Gen: How to Search for Mail & Related KB Articles by Topic

 Last reviewed: March 20, 1997
Article ID: Q98105

kbref kbkeyword

The information in this article applies to:

  • Microsoft Electronic Forms Designer, version 1.0, running with Microsoft Mail for PC Networks, version 3.0 or 3.2
  • Microsoft Mail for AppleTalk Networks, versions 3.0, 3.1, 3.1a, 3.1c, and 3.1d
  • Microsoft Mail for PC Networks, versions 2.x, 3.0, and 3.2
  • Microsoft Mail for Windows, versions 2.x, 3.0, and 3.2
  • Microsoft Mail gateways (all)
  • Microsoft Mail Remote for MS-DOS, version 3.0
  • Microsoft Mail Remote for Windows, version 3.2
  • Microsoft Schedule+ for Windows, version 1.0

You can use keywords to easily query for specific topics in the Microsoft Knowledge Base. This article contains keywords for Microsoft Mail and related products. By using these keywords, you can quickly access common topics. These keywords are commonly found at the bottom of articles on the line that begins, "KBSubcategory."

For a general list of categories and keywords for all Knowledge Base articles, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles
In addition to these keywords, all article titles for Microsoft Mail and related products are in the following format 

   <prod> <category>: <title>
where <prod> is PC or Mac, and <category> is one of the categories listed below. For example: 

   PC Win: <title>
   Mac Srv: <title>
   PROFS: <title>
