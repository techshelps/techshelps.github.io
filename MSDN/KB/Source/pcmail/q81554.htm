PC Win: Only First File of Multiple Files Copies

 Last reviewed: May 20, 1996
Article ID: Q81554
The information in this article applies to:
  • Microsoft Mail for PC Networks, versions 3.0 and 3.5

SYMPTOMS

In the File Manager for Microsoft Windows version 3.1, you can select a range of files to copy. You can copy a message to the Clipboard to insert into a Microsoft Mail for Windows version 3.0 Mail message. When you highlight a range of files to copy to the Clipboard, it appears that all selected files are transferring, but only the first file is actually copied.

RESOLUTION

Copy the files one at a time.
