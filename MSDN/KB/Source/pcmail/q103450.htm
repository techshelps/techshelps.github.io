PC Win: Maximum Size of Inserted File is 32,655 Bytes

 Last reviewed: October 31, 1994
Article ID: Q103450
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.0, 3.0b, 3.1, and 3.2

In versions 3.0, 3.0b, 3.1, and 3.2 of Microsoft Mail for Windows, the maximum size of a sent message is 33,580 bytes. However, the maximum size of a file inserted in a new message or text that is added or pasted from another message is 32,655 bytes.

You cannot insert a saved message that contains the maximum amount of text into another message, unless you first edit the saved message so that it contains 32,655 bytes or less.
