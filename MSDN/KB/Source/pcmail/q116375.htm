FAX: WIN.INI File Entries Added During Installation

 Last reviewed: October 20, 1997
Article ID: Q116375
3.00 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Mail versions 3.0, 3.0b and 3.2
  • Microsoft Mail Gateway to Fax versions 3.0 and 3.0a

Page 7 of the "Microsoft Mail Fax User's Guide," says that "Microsoft Fax adds the printer driver to the WIN.INI file in the following format": 

   [Microsoft Fax, Port]
Under normal circumstances, the Microsoft Fax adds the printer driver to the [PrinterPorts] and [Devices] sections in the WIN.INI file.

Here is an example of the default entries added to the WIN.INI file:

[Microsoft Fax] TempDrive=

[PrinterPorts] Microsoft Fax=MSFAX,LPT1:,15,45

[Devices] Microsoft Fax=MSFAX,LPT1:
