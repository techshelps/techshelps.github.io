SMTP: SMTP Command-Line Parameters

 Last reviewed: October 20, 1997
Article ID: Q87042
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to SMTP, version 3.0

These command-line parameters can be used with the Microsoft Mail Gateway to SMTP: 

   -BD          -BT          -BR          -C[newi]
   -D           -G[acnsy]    -H           -I
   -L[acnsy]    -MR          -MS          -Q
   -P           -RE          -RR          -T
   -V20         -W           -X           -MD
For a full description of the usage of these parameters, see Chapter 4, "Running the Gateway," in the "Microsoft Mail Gateway to SMTP Administrator's Guide."
