PC DOS: Err Msg: Notice 38: Empty Personal Address List (PAL)

 Last reviewed: January 12, 1996
Article ID: Q142094
The information in this article applies to:
  • Microsoft Mail for PC Networks, MS-DOS workstation, version 3.0

SYMPTOMS

When you select Compose to send a new message, you may get the following error message: 

   Notice 38 - Empty Personal Address List
   Select Address option to define

CAUSE

You will get the above error when the Personal Address List (PAL) is your default address display, and you do not have any entries in the list.

WORKAROUND

To correct this error, do one of the following:

  • Press the ESC (escape) key, and go back to the main opening screen of the MS-DOS client. Select Address, and press the ENTER key. Add an address to the Address List.
  • Press the ESC (escape) key, and go back to the main opening screen of the MS-DOS client. Select Options, Configure, Display. Change the display so that an address list other than the Personal Address List is the default.
