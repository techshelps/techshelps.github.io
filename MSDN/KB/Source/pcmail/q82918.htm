PC Win: Modified Shadowed Message Is Not Updated in Mail

 Last reviewed: October 29, 1994
Article ID: Q82918
The information in this article applies to:
  • Microsoft Mail for Windows, version 3.0

SYMPTOMS

Version 3.0 of Microsoft Mail for Windows has the ability to shadow all incoming messages to the mailbag file at the postoffice; however, the shadowed message is not updated after the original message has been modified.

RESOLUTION

To have the modified message shadowed in the Mailbag, more the entire message out of the Inbox folder and then back in again.
