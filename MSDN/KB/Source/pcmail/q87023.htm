SMTP: TCP/IP Stacks Supported by the Mail Gateway to SMTP

 Last reviewed: October 20, 1997
Article ID: Q87023
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to SMTP, version 3.0

Version 3.0 of the Microsoft Mail Gateway to SMTP supports five TCP/IP stacks. These are:

  • Microsoft LAN Manager TCP/IP, versions 1.0 and later
  • FTP Software, Inc. PC/TCP Kernel, version 2.05
  • Novell LAN Workplace for DOS, version 4.0
  • Wollongong Group, Inc. WIN/TCP for DOS Run Time, version 4.1
  • Excelan LAN WorkPlace Network Software, version 3.4

Notes:
  • The gateway will not run on Microsoft Windows NT, Microsoft Windows 95, or Microsoft Windows 3.x; it can only be run under MS-DOS or LAN Workplace for MS-DOS version 5.0.
  • Microsoft LAN Manager can be broken down into MSCLIENT version 3.0 and LAN Manager version 2.2c.

For more information, see page 2 of Chapter 1, "A Brief Introduction," in the "Microsoft Mail Gateway to SMTP Administrator's Guide."
