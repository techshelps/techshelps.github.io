PC Setup: Err Msg: Can't Find MSSFS.DLL

 Last reviewed: October 23, 1995
Article ID: Q98975
The information in this article applies to:
  • Microsoft Mail for PC Networks, version 3.2

SYMPTOMS

During the Custom Installation process, if you specify to install only the Program files, you will receive the following error message when you try to run Mail. 

   Can't find MSSFS.DLL

CAUSE

This problem occurs because the program files need the associated driver files installed for proper execution. This occurs only if the user chooses not to install any drivers.

RESOLUTION

Install the driver files that are associated with the program files.
