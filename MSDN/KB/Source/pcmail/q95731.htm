PC Win: Name Finder Fails If Space Before Name

 Last reviewed: January 15, 1996
Article ID: Q95731
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.0 and 3.0b

SYMPTOMS

In version 3.0 or 3.0b of Microsoft Mail for Windows, if the friendly name is defined as [space]Name or [space]LastName,[space]FirstName, the name appears at the top of the address list, but using Name Finder to search for that name fails.

STATUS

Microsoft has confirmed this to be a problem in versions 3.0 and 3.0b Microsoft Mail for Windows. The problem was fixed in version 3.2.
