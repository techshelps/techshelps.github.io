X400: Err Msg: ERROR 1 (RTS): Failed to Decode PAccept

 Last reviewed: October 20, 1997
Article ID: Q123650
3.20 MS-DOS kbother kberrmsg

The information in this article applies to:

  • Microsoft Mail Gateway to X400, version 3.2

SYMPTOMS

Version 3.2 of Microsoft Mail Gateway to X.400 may display the following error when a message has been sent: 

   DEBUG (RT_decode_CData): cpynstr() failed
   ERROR 1 (RTS): Failed to decode PAccept

CAUSE

The other message transfer agent (MTA) does not have complete configuration information for the Microsoft Mail X400 Gateway. It is returning an ACCEPT session layer packet without an MTA name in it.

RESOLUTION

Ensure the MTA configuration is correct. Pay particular attention to the NSAP address defined on the other MTA for the Microsoft Mail X400 Gateway.
