MHS: Err Msg: User Not Found: Postmaster

 Last reviewed: October 20, 1997
Article ID: Q128444
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to MHS, version 3.0

SYMPTOMS

The following error may occur when you use the MHS gateway with a downstream postoffice. 

   User not found: Postmaster

CAUSE

The gateway tries to send mail to the POSTMASTER account if an address is invalid.

RESOLUTION

Create a POSTMASTER account on all downstream postoffices that will be using the MHS gateway.

MORE INFORMATION

The Microsoft Mail Gateway to MHS "Administrator's Guide" is incomplete; it suggests that you create a POSTMASTER mailbox on the gateway postoffice, but does not mention that you should also create a POSTMASTER account on all downstream postoffices.
