PC Ext: Err Msg: Error 21 Missing Routing File

 Last reviewed: October 24, 1995
Article ID: Q104485
The information in this article applies to:
  • Microsoft Mail for PC Networks, version 3.2

SUMMARY

In version 3.2 of the Microsoft Mail for PC Networks External Mail program (EXTERNAL.EXE), Mail's message transfer agent (MTA), you may receive the following error message: 

   Error 21 Missing Routing File

CAUSE

This error means that the corresponding .OVL file (ASYNC.OVL, X25EICON.OVL, or X25ATLAN.OVL) cannot be loaded.

RESOLUTION

To load this file, make sure the machine has MS-DOS extended memory available and the ASYNC.OVL, X25EICON.OVL, and X25ATLAN.OVL files are in the same directory as the EXTERNAL.EXE file.

NOTE: You should store these files on the computer's local disk drive and you should set the MS-DOS temporary (temp) workspace to the local drive.
