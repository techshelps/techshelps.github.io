PROFS: What to Do When Communication with LAN Ceases

 Last reviewed: October 20, 1997
Article ID: Q86155
3.20 VM kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to IBM PROFS and OfficeVision, version 3.2

If the VM side of the gateway appears to no longer be communicating with the LAN side, the administrator should use the Gateway PC to hot key to the host session to check the status.

If the VM side of the gateway is "alive," either PROFS:IDLE or PROFS:READY should be displayed on the screen. If not, you can restart the VM side by typing: 

   HX
   IPL CMS
Or, if you are already in CMS, type: 

   PROFILE EXEC
The gateway should eventually come up with the PROFS:IDLE or PROFS:READY prompt. Once this has happened, you should hot key back to MS-DOS and restart the HostDisp program.
