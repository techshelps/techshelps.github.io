FAX: Changes Summary

 Last reviewed: October 20, 1997
Article ID: Q85611
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to Fax, version 3.0

Changes made in version 3.0 of the Microsoft Mail Gateway to Fax:

  1. Support of GammaLink fax board.

  2. Improved profile editor, PROFILE.EXE

  3. Fax viewer for Microsoft Windows.

For more information, see the "Microsoft Mail Gateway to Fax Administrator's Guide."
