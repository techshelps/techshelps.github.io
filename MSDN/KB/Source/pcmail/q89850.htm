PC Ext: Err Msg: Already Attached to Server

 Last reviewed: October 29, 1994
Article ID: Q89850
The information in this article applies to:
  • Microsoft Mail for PC Networks, versions 2.1 and 3.0

SYMPTOMS

The following error message is received on the External Mail program (EXTERNAL.EXE) when you try to dynamically attach to more than one postoffice on a Novell NetWare server: 

   Already Attached to Server

CAUSE

If you have one Novell NetWare server, two or more postoffices, and one External machine, you cannot use dynamic connections (DYNADMIN.EXE). All connections must be done through direct drive connections.
