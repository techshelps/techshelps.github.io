 Last reviewed: September 30, 1997
Article ID: Q117511
The information in this article applies to:
  • Microsoft Electronic Forms Designer, version 1.0, running with Microsoft Mail for PC Networks, version 3.0, 3.2, or 3.5
  • Microsoft Mail for PC Networks, versions 2.x, 3.0, and 3.2
  • Microsoft Mail for Windows, versions 2.x, 3.0, and 3.2
  • Microsoft Mail Remote for MS-DOS, version 3.0
  • Microsoft Mail Remote for Windows, version 3.2
  • Microsoft Schedule+ for Windows, version 1.0

The Microsoft Product Support Services (PSS) Knowledge Base group regularly compiles the articles from the Microsoft Mail for PC Networks database and creates a Windows Help file that is available in the Microsoft Software Library on the Microsoft Network, the Microsoft Download Service (MSDL), Microsoft OnLine, and the Internet.

The Help file for Microsoft Mail for PC Networks and associated products is in a self-extracting file named MAILKB.EXE and contains several hundred technical articles written by PSS Support Engineers and Knowledge Base writers published prior to December 7, 1995. These articles contain:

  • References to Application Notes and other files available in the Microsoft Software Library
  • Bug lists and fix lists for Mail for PC Networks and associated products
  • Additional references
  • Questions and answers
  • Recent product information not addressed in the user documentation
  • Helpful tips

NOTE: You can run this Help file from Microsoft Windows 95 or Microsoft Windows NT version 3.51, using Winhlp32.exe.
  • If you have Windows NT 3.5 or earlier, you must upgrade to version 3.51 to view this Help file.
  • If you have Windows 3.x or Windows for Workgroups 3.x, you must obtain the Microsoft Win32s upgrade (Pw1118.exe) before you can run this Help file.
