PC Adm: Storage Status in ADMIN.EXE Reports 0 Bytes Used

 Last reviewed: October 26, 1994
Article ID: Q80158
The information in this article applies to:
  • Microsoft Mail for PC Networks, versions 2.1 and 3.0

The Mail Administrator program (ADMIN.EXE) may display improper storage status (0 mailbags, 0 files, and 0 bytes used) if the Admin mail logon does not have proper directory search rights. Under Novell, the Admin logon must have filescan rights in order to determine the file sizes of the .MAI or .MBG files and create a status on the files in those subdirectories.

The status display will show 0 bytes when you choose Local-Admin, Storage, Status, Post Office or choose Local-Admin, Storage, Specific Users.

The search rights for Novell can be viewed or changed by using the NetWare rights command.
