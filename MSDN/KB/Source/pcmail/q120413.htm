PC Gen: Microsoft Mail for PC Networks Version 3.2a Update

 Last reviewed: June 4, 1996
Article ID: Q120413
The information in this article applies to:
  • Microsoft Mail for PC Networks, version 3.2a

SUMMARY

The Microsoft Mail for PC Networks Update version 3.2a is available on OnLine, the Microsoft TechNet CD, and the Internet.

Microsoft Mail for PC Networks version 3.2a Update is available at no charge for customers who are currently running version 3.2 of Microsoft Mail for PC Networks. Customers currently using versions 2.1 or 3.0 of Mail must purchase the 3.2a Postoffice Update for a minimal update fee, to receive 3.2 documentation and software.

The files listed below are available to download from the Microsoft Software Library (MSL). For information on how to download these files, please see the "Instructions for Downloading" section below.

FILES AVAILABLE FROM ONLINE SERVICES

The Microsoft Mail for PC Networks 3.2a Update consists of three self- extracting files:

S14883      MAIL32AC.EXE      Client updates for 3.2a

MORE INFORMATION

IMPORTANT: MAIL32AS.EXE, MAIL32AC.EXE, and MAIL32AM.EXE contain READ_1ST.TXT, which includes the update policy, pertinent update information, and a list of all the files that will be modified. This file also contains instructions on how to create the setup disks. Please read this file first.

For Windows clients, you MUST run the Setup program to properly update each workstation to version 3.2a. Because of the number of files modified and version number consistency issues, it is not sufficient to copy individual .DLL and executable files in lieu of running SETUP.EXE. Running the Setup program is the only way to ensure each and every file is the latest version.

NOTE: This update has been placed on online services for your convenience. If you plan to download this update, please note that it is approximately 5 MB in size. Since the download time will be lengthy, the connect charges could be costly.

An extra utility, such as MMFClean, is not shipped with this Update. MMFClean is available as MMFUTIL.EXE, a self-extracting file, on the Microsoft Software Library (MSL).

For additional information regarding the MMFClean utility, please see the following article in the Microsoft Knowledge Base:
