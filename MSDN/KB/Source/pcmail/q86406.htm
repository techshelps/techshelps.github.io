PC MAPI: Excel Call Function Has Incorrect Arguments

 Last reviewed: October 21, 1994
Article ID: Q86406
The information in this article applies to:
  • Microsoft Mail for Windows, version 3.0

On page 78 of the version 3.0 Microsoft Mail "Technical Reference," the type_text parameter for the Call function to MAPISendDocuments is incorrect.

The type_text parameter should be "JJCCCJJ". The example is missing the last "J" and will return #VALUE! if this additional parameter is not included.
