PC Setup: Mail 3.0 Directory Entries in NetWare

 Last reviewed: October 7, 1994
Article ID: Q112268
The information in this article applies to:
  • Microsoft Mail for PC Networks, version 3.0

An initial installation of Mail for PC Networks on a Novell NetWare server requires 95 directory entries for the Mail executable files, which in turn requires 6,492,165 bytes.

The Mail database requires approximately 210 directory entries, which in turn requires 1,129,644 bytes.

You can use these numbers for planning during a NetWare installation.
