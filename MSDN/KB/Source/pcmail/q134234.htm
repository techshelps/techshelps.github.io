PC Adm: Err Msg: Notice 38 Cannot Add Local Postoffice Name...

 Last reviewed: August 3, 1995
Article ID: Q134234
The information in this article applies to:
  • Microsoft Mail for PC Networks, version 3.2

SYMPTOMS

When you create an external postoffice definition using the Administrator's program (ADMIN.EXE), the following error message may be displayed: 

   Notice 38 Cannot add local postoffice name to external list.

CAUSE

This error is displayed when you try to add the network and postoffice name of the current postoffice as an external postoffice. The current postoffice cannot be defined as an external postoffice to itself.

RESOLUTION

Add the correct name of the external postoffice.
