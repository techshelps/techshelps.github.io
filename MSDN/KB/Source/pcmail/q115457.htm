PROFS: Message Format for Version 3.4

 Last reviewed: October 20, 1997
Article ID: Q115457
3.40 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to IBM PROFS and OfficeVision, version 3.4

Messages received from version 3.4 of the Microsoft Mail Gateway on IBM PROFS and OfficeVision are in this format: 

   From: ADMIN   --PCNODE   Date and Time   08-24-93 19:15:20
   To: MARKEEF   -VMNODE

   From: Administrator
   To: FOSTER, MARKEE
   Subject: Testing Messages Format from 3.4 gateway to 3.4 Host Access
   Date: 1993-08-24 19:00
   Priority: R

   -------------------------------------------------------------------

   MESSAGE TEXT
