PC WFW: Notes Field Text Gets Cut Off

 Last reviewed: October 24, 1994
Article ID: Q91043
The information in this article applies to:
  • Microsoft Mail for PC Networks, version 3.0

SYMPTOMS

When you use Extensions for Windows for Workgroups to upgrade a Windows version 3.1 for Workgroups postoffice to a version 3.0 Microsoft Mail for PC Networks postoffice, the Notes field may be cut off in the MS-DOS Mail Administrator program and in the Mail clients. Although you can type more information in the Notes field with the Windows for Workgroups Postoffice Administrator program, only 53 characters are displayed after you run Extensions for Windows for Workgroups.

CAUSE

In a version 3.0 Microsoft Mail for PC Networks postoffice, any data that exceeds the maximum window width of 70 characters as a result of changing the field data starting position will be cut off and ignored.
