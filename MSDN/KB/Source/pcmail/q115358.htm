X400: TSAP and SSAP Fields Cannot Accept Hex Values V2.1, V3.0

 Last reviewed: October 20, 1997
Article ID: Q115358
2.10 3.00 3.20 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to X.400, version 2.1, 3.0 and 3.2

Versions 2.1 and 3.0 of the X.400 gateway allow you to enter ASCII values in the TSAP and SSAP fields, but not hexadecimal values, which were also required. Hex values have to be entered in the form $xx$xx, in order to be compatible with other hex fields, but this required more room than was available in the TSAP and SSAP fields.

This problem was fixed in version 3.2.
