PC OS/2:Err Msg:SYS1804: Setup Is Unable to Find File KEYBOARD

 Last reviewed: October 9, 1995
Article ID: Q124197
The information in this article applies to:

- Microsoft Mail for PC Networks, versions 3.2 and 3.2a

When you install Microsoft Mail for PC Networks on an OS/2 workstation, the following error message may appear: 

   SYS1804: Setup is unable to find file KEYBOARD.
To avoid this problem, run Setup within a WINOS/2 session.

Depending upon the version OS/2 being used, mail is installed on the OS/2 client machine with either PMSETUP.EXE or SETUP.EXE. PMSETUP.EXE was specifically designed to be used with Presentation Manager version 1.x. On any version of OS/2 later than 1.x, you must use the Windows SETUP.EXE program.

The problem generally arises when the customer is attempting to use SETUP.EXE within an OS/2 session. In order for SETUP.EXE to perform correctly, the user must run setup within a WINOS/2 session.
