PC OS2: Print Setup Dialog Selects Wrong Default Printer

 Last reviewed: October 21, 1994
Article ID: Q86401
The information in this article applies to:
  • Microsoft Mail for PC Networks, OS/2 workstation, versions 3.0, 3.0b, and 3.2

SYMPTOMS

When you choose Printer Setup from the File menu in Microsoft Mail for Presentation Manager, the dialog box selects the first printer in the list, regardless of which printer was designated as the default.

STATUS

Microsoft has confirmed this to be a problem in versions 3.0, 3.0b, and 3.2 of Microsoft Mail for Presentation Manager. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
