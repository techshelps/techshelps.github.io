SMTP: Definition of the IMPORT Utility

 Last reviewed: October 20, 1997
Article ID: Q88780
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to SMTP, version 3.0

The import utility, IMPORT.EXE, allows Microsoft Mail administrators to import external postoffice addresses and gateway addresses from a text file directly into their postoffice's address lists (for example, postoffice address lists and gateway address lists).

This text file is normally created either from a foreign mail system or by extracting address lists from another postoffice. You can also use the import utility to extract address lists from other postoffices.

For more information on this utility, see Appendix D of the "Microsoft Mail Administrator's Guide."
