PC Gen: PODIAG.EXE Creates INI Directory on 2.1 & 3.0 POs

 Last reviewed: September 16, 1997
Article ID: Q109466
3.20 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Mail for PC Networks Postoffice Diagnostic Utility, version 3.2

SUMMARY

The Microsoft Mail Postoffice Diagnostics Utility (PODIAG.EXE) version 3.2 creates an INI subdirectory in a Microsoft Mail version 2.1 or 3.0 postoffice database. PODIAG will create the INI subdirectory, if it does not exist, because it stores the PODIAG.INI file in the M:\MAILDATA\INI subdirectory. This is by design.

MORE INFORMATION

The INI subdirectory was an additional feature added in version 3.2 postoffices and did not exist in the version 2.1 and 3.0 databases.
