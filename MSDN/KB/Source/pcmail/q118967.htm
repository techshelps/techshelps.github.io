ATT: Removal of Gateway Fails to Update NETWORK.GLB

 Last reviewed: October 20, 1997
Article ID: Q118967
1.00 MS-DOS kbbug1.00 kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to AT&T, version 1.0

SYMPTOMS

The AT&T Gateway Access component does not complete a remove correctly from an AT&T Gateway postoffice.

CAUSE

The INSTALL -R command fails to update the NETWORK.GLB file.

RESOLUTION

Call Microsoft Product Support Services for assistance with correcting the NETWORK.GLB file.
