FAX: FAX Boards Supported with the FAX Gateway Version 3.0

 Last reviewed: October 20, 1997
Article ID: Q112228
3.00 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to FAX, version 3.0

Page 2 of the "Microsoft Mail Gateway to FAX Administrator's Guide" improperly states that the GammaLink GammaFAX XP board is supported. It is not supported.

These FAX boards are supported with the Microsoft Mail FAX Gateway version 3.0: 

   Intel SatisFAXtion
   Intel SatisFAXtion CoProcessor
   Intel SatisFAXtion Modem 200
   GammaLink GammaFAX CP
   GammaLink GammaFAX CP-DFX
The "Microsoft Mail Gateway to FAX Release Notes" included with version 3.0a corrects the Administrator's Guide.
