MICROSOFT MAIL SCRIPT INSTALLATION V3.0

Welcome to the Modem Script Setup program for Microsoft Mail. SETUP installs Microsoft Mail Modem Script files on your hard disk or on a network by copying the script source and compiled files.

To install the scripts:

  1. Insert the Modem Script Disk in the floppy drive.

  2. At the MS-DOS prompt, type 

         a:SETUP

    (if the disk is on drive A).

  3. Follow the instructions on the screen.
