Internet Information for MS Customers via ftp.microsoft.com

 Last reviewed: October 20, 1997
Article ID: Q102239

MS-DOS | WINDOWS kbref

The information in this article applies to:

  • Microsoft LAN Manager
  • Microsoft Mail for AppleTalk Networks
  • Microsoft Mail for PC Networks
  • Microsoft Mail gateways
  • Microsoft Schedule+
  • Microsoft SQL Server
  • Microsoft Windows NT

Microsoft has established a support server on the Internet. This server contains a variety of materials related to Microsoft products. Included are Knowledge Base articles, selected patches and fixes, documentation, drivers, papers, and product information.

The name of the server is "ftp.microsoft.com", (also aliased to "gowinnt.microsoft.com") and the IP address is 198.105.232.1. You can ftp to the server, logging on as "anonymous" with your full Internet email name as the password.

Information screens are displayed upon entering major directories, and further details are often available in README.TXT files. The Mail, Mail Gateways, and Schedule+ products are in the directory path bussys\mail. This server should contain no confidential or proprietary information.

If you experience difficulty obtaining information from this server, please email one of the following: 

   cslmad   - LAN Manager ftp server admin
   csntad   - Windows NT ftp server admin
   csmailad - Mail ftp server admin
   cssqlad  - SQL ftp server admin
   csodbcad - ODBC ftp server admin
   cssnaad  - SNA ftp server admin
   csftpad  - Overall ftp.microsoft.com admin.
NOTE: Sorry, we may not be able to send individual replies to mail. This is NOT a support alias.
