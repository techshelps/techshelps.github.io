SMTP: Err Msg: The Following Recipients Could Not Be Reached

 Last reviewed: October 20, 1997
Article ID: Q127727
3.00 MS-DOS kb3rdparty

The information in this article applies to:

  • Microsoft Mail Gateway to SMTP, version 3.0

SYMPTOMS

When a user sends mail to SMTP recipients, the mail is returned almost immediately with an error: 

   The following recipients could not be reached...

CAUSE

The postoffice was installed on a PC/NFS volume on a UNIX server. The permissions for the postoffice files may not be set to 7/7/7, which means read/write/execute for owner/group/world.

RESOLUTION

Ensure that all postoffice files--in particular, the SMTP gateway mailbag file--are set to 7/7/7.
