FAX: Err Msg: Could Not Find Setup Fil

 Last reviewed: October 20, 1997
Article ID: Q104112
3.00a WINDOWS kb3rdparty kbtlc

The information in this article applies to:

  • Microsoft Mail Gateway to Fax, version 3.00a, running under Windows NT version 3.10

SYMPTOMS

When you attempt to install the Microsoft Mail Fax printer driver within the Windows NT Control Panel, this error may occur: 

   Could not find Setup File <path> PRINTER.INF

CAUSE

Windows NT does not use the OEMSETUP.INF file like other versions of Windows. Windows NT looks for a PRINTER.INF file, which is not on the installation disk.

STATUS

Microsoft has confirmed this to be a problem in version 3.0a of the Microsoft Mail Gateway to Fax and Microsoft Windows NT. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
