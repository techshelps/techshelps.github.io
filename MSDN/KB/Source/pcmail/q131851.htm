PC Win: Err Msg: Some Information on the Screen Could Not...

 Last reviewed: June 23, 1995
Article ID: Q131851
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.0, 3.0b, and 3.2

SYMPTOMS

In version 3.0, 3.0b, and 3.2 of Microsoft Mail for Windows, you may minimize a mail message, and it will appear at the bottom of the screen. When it is maximize it, the following error may occur: 

   Some information on the screen could not be updated correctly.
   Close and reopen affected windows.

CAUSE

The usual cause of this error is that the computer has run low on system resources or memory during the mail session.

RESOLUTION

To correct the problem, close any unused windows and/or applications. Then, from the File menu of Microsoft Mail, choose Exit and Sign Out. Then launch Mail again.

If the system resources have been sufficiently restored, the error message will not appear again.
