PC Win: Network Error Using Mail Conversion Program

 Last reviewed: March 29, 1995
Article ID: Q97734
The information in this article applies to:
  • Microsoft Mail for Windows, versions 3.0, 3.0b, and 3.2

SYMPTOMS

When you use version 3.0 of the Microsoft Mail Conversion Program to convert Mail 2.x folders to Mail 3.0 or 3.2, CONVERT.EXE may report a network error and then exit.

CAUSE

The CONVERT.EXE program assumes the owner field in the .IDX file is in all uppercase letters and does not know how to handle hexadecimal numbers in lowercase letters.

STATUS

Microsoft has confirmed this to be a problem in versions 3.0, 3.0b, and 3.2 of Microsoft Mail for Windows. This problem was corrected in version 3.2a.
