INFO: Location of MAPI Debug Files for Windows NT 4.0

 Last reviewed: June 11, 1997
Article ID: Q169792
The information in this article applies to:
  • Extended Messaging Application Programming Interface (MAPI), version 1.0 included with: - Microsoft Windows Messaging

SUMMARY

The debug versions of the Messaging Application Programming Interface (MAPI) files that ship with Windows NT 4.0 Workstation or Server can be found on Disc 10 of the Microsoft Developer Network.

MORE INFORMATION

The disk is titled "Disc 10 - Win32 SDK (for Windows 95 and Windows NT Workstation 4.0 - U.S.) with Far East Updates."

The folder where the files are located for the i386 version of Windows NT 4.0 is \Mstools\Mapi\i386\Checked.

REFERENCES

Further documentation on installing the debug versions of these files can be found on the Microsoft Developer Network Library in an article titled, "Using a DEBUG Version of MAPI."
