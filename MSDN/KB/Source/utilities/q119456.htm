FIX: PROFILE /X Does Not Work with Line Coverage or Count

 Last reviewed: September 18, 1997
Article ID: Q119456
1.00 WINDOWS NT kbtool kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft Source Profiler for Windows NT, version 1.0

The information in this article is included in the documentation starting with Visual C++ 5.0. Look there for future revisions.

SYMPTOMS

The PROFILE /X option returns the exit code of the program being profiled. This option does not work with line-coverage or line-counting mode.

STATUS

This problem was corrected in the Microsoft Source Profiler for Windows NT, version 1.10
