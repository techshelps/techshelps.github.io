PWB Hangs When Run with Some Communication Programs

 Last reviewed: July 17, 1997
Article ID: Q64435
1.00 OS/2 kbtool

The information in this article applies to:

  • Microsoft Programmer's Workbench for OS/2, version 1.0

When running Programmer's WorkBench (PWB) under OS/2 while simultaneously running communication programs, PWB will appear to hang. To correct the problem, terminate any communication programs and PWB will resume as normal.
