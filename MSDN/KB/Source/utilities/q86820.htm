BUG: PWB Incorrectly Closes Editor Settings Dialog Box

 Last reviewed: July 17, 1997
Article ID: Q86820
2.00 MS-DOS kbtool kbbuglist

The information in this article applies to:

  • Microsoft Programmer's Workbench for MS-DOS, versions 2.0 and 2.1.49

SYMPTOMS

Programmer's WorkBench (PWB) version 2.0, which ships with Microsoft C/C++ 7.0, will close the Editor Setting dialog box in response to certain actions (other than choosing OK or Cancel).

Steps to Reproduce Problem

  1. From the Options menu, choose Editor Settings.

  2. Double-click on the Boolean, Text, or Numeric option button.

RESOLUTION

Do not double-click on the Boolean, Text or Numeric option button. You should only single-click these buttons.

STATUS

Microsoft has confirmed this to be a problem in PWB versions 2.0 and 2.1.49. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
