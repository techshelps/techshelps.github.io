Microsoft Language Utility Version History

 Last reviewed: August 7, 1996
Article ID: Q93400
The information in this article applies to:
  • Microsoft LIB utility for MS-DOS, versions 3.1, 3.17, 3.18, 3.2, 3.2.01, 3.31, and 3.4
  • Microsoft LIB utility for Windows NT, versions 1.0, 2.5, 3.0, 3.1, 4.2
  • Microsoft LINK utility for MS-DOS, versions 5.01.2, 5.1, 5.11, 5.13, 5.15, 5.3, 5.31.009, 5.5, 5.6, and 5.6.220
  • Microsoft LINK utility for Windows NT, versions 1.0, 2.5, 3.0, 3.1, 4.2
  • Microsoft NMAKE utility for MS-DOS, versions 1.11, 1.13, 1.2, 1.3, 1.4, and 1.5
  • Microsoft NMAKE utility for Windows NT, versions 1.4, 1.5, 1.6
  • Microsoft CodeView for MS-DOS, versions 3.14, 4.0, 4.01, and 4.1
  • Microsoft CodeView for Windows, versions 4.0, 4.01, and 4.1
  • Microsoft CodeView for Win32s, version 4.25

SUMMARY

The text below lists the versions of the Microsoft utilities distributed with various versions of Microsoft developer products.
