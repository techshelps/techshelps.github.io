OFF98: Tell Me More Help File Cannot Be Found

 Last reviewed: February 3, 1998
Article ID: Q177398
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

If you click Tell Me More while you are in the Visual Basic Editor for Microsoft Office 98 Macintosh Edition, you may receive one of the following error messages: 

   Microsoft Help - Cannot open Help file.

   -or-

   Microsoft Excel cannot open the required Help file. Please
   run the installer application to restore any missing files.

CAUSE

This problem occurs if the following conditions are true:

  • You are using the Visual Basic Editor and click Office Assistant.

    -and-

  • You click Options and click the Gallery tab.

    -and-

  • You click Next until you see the screen with information about different assistants, and then click Tell Me More.

WORKAROUND

The Tell Me More button works if you are not in the Visual Basic Editor. To view the Help file that is associated with the Tell Me More button, follow these steps:

  1. Switch out of or close the Visual Basic Editor.

  2. On the Standard toolbar, click Office Assistant.

  3. Click Options, and then click the Gallery tab.

  4. Click Next until the Tell Me More button is unavailable.

  5. Click Tell Me More.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
