The information in this article applies to:

  • Microsoft Theater Server, version 1.0 (Beta-1 and Beta-2)

SYMPTOMS

If you use network interface cards (NICs) that use an Auto-Sense or Auto- Negotiate line speed setting, NetShow Theater Server (NTS, previously known as the NetShow Pro Full Screen Video Server) may experience a performance degradation.

WORKAROUND

To work around this problem:

If your installation of NetShow Theater Server is experiencing poor video rendering performance, modify the setting on these NICs to a fixed line speed on both the server and client.

MORE INFORMATION

Testing of NetShow Theater Server with NICs that use an Auto-Sense or Auto- Negotiate option to set the NICs line speed have exhibited some performance degradation resulting in the disruption of the video signal between the servers and the rendering client.
