Err Msg: MPEG Layer-3 Beta Expired, Please Contact...

 Last reviewed: September 16, 1997
Article ID: Q165182
The information in this article applies to:
  • Microsoft NetShow version 2.0

SYMPTOMS

When I try to play a sound, I get the following error message: 

   MPEG Layer-3 Beta expired please contact
   http://www.microsoft.com/netshow/ for latest updates.

CAUSE

This is a problem with the FHG codec shipped in NetShow version 2.0 beta 1.

WORKAROUND

Delete the L3*.acm file from your System directory. If you get an access violation error when you try to delete this file, shut down to single DOS mode and delete this file from a DOS prompt.

Because this problem has been fixed in the final release of NetShow version 2.0, you can also resolve this problem by removing the codec file and reinstalling the final release of NetShow version 2.0.
