Can't Run NetShow Theater with Other Applications

The information in this article applies to:

  • Microsoft Theater Server, version 1.0 (Beta-1 and Beta-2)

SUMMARY

The NetShow Theater Server (NTS, previously known as the NetShow Pro Full Screen Video Server) title and content servers should only run the NetShow Theater Server application. Do not run other applications on these servers.

MORE INFORMATION

NetShow Theater Server title and content servers should only run the NetShow Theater Server application. Testing has shown that running other applications on the title or content servers can result in performance degradation, leading to the disruption of the video signal between the servers and the rendering clients.

This has occurred even with screen saver applications that are running simultaneously with the NetShow Theater Server application. If your installation of NetShow Theater Server is experiencing poor video rendering performance, verify that only the NetShow Theater Server application is running on the title or content servers.
