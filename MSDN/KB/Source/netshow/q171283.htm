Err Msg: This Page Contains Information of Type...

 Last reviewed: September 16, 1997
Article ID: Q171283
The information in this article applies to:
  • Microsoft NetShow version 1.0

SYMPTOMS

When I view NetShow content, I see it fine in Microsoft Internet Explorer. However, when I use Netscape browser, I get the following error message:

[ This page contains information of type "application/octet-stream" that can only be viewed with the application Plug-in?..]

WORKAROUND

Make sure the server is serving ASF and ASX files with the proper mime type.

If the server is not a Microsoft Internet Information Server (IIS) server or is an IIS server but NetShow is not being used on this computer, you need to add the mime types on the server. It should be serving ASX and ASF files as type video/x-ms-asf.
