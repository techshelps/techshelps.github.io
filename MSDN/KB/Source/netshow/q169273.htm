Bad or Missing Stream Format Error When View Multicast Link

 Last reviewed: September 16, 1997
Article ID: Q169273
The information in this article applies to: 

  - Microsoft NetShow version 2.0

SYMPTOMS

When you try to view a Multicast link, you get the following error message: 

   Bad or Missing Stream format.

WORKAROUND

When you create your Multicast channel, you need to make sure you add a stream format for each active stream.

For a live stream, you add the ASD file you save in from the encoder with the current settings.

For a stored .ASF file, you need to use the ASF file as the file you open to add the stream format to the channel.

After you make any changes to the encoder or other stream formats that are used in a particular program/channel, you will need to update the format information in the channel and export the channel file.
