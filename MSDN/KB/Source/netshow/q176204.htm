The information in this article applies to:

  • Microsoft NetShow Theater Server version 1.0 (Beta-2)

SUMMARY

The following is a list of specific hardware used in the Microsoft Testing Labs for the beta-2 release of NetShow Theater Server (NTS, previously known as the NetShow Pro Full Screen Video Server). See the NetShow Theater Server product documentation and release notes for specific updates and details on the use of this hardware.

Please visit the Web sites of each of the vendors listed to obtain the latest drivers. Our testing lab discovered dramatic performance increases when using the newest drivers available.

Other hardware that was not tested but meets the guidelines provide in the NetShow Theater Server documentation and the Microsoft Windows Hardware Compatibility listings may also work.
