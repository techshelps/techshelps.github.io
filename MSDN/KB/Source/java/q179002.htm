PRB: Cannot Install Visual J++ 1.0 on Windows NT 3.51

 Last reviewed: January 29, 1998
Article ID: Q179002
The information in this article applies to:
  • Microsoft Visual J++, version 1.0

SYMPTOMS

When trying to install Visual J++ 1.0 on Windows NT 3.51, Setup.exe gives the following error : 

   Visual J++ 1.0 requires Windows95 or Windows NT 4.0

CAUSE

The minimum requirement for Visual J++ 1.0 is either Windows 95 or Windows NT 4.0. Earlier documentation stated that Visual J++ 1.0 would run on Windows NT 3.51 even though the 32-bit version of Internet Explorer 3.0 would not work. That information is incorrect.

RESOLUTION

In order to use Visual J++ 1.0 on an existing Windows NT 3.51 system, the System Administrator will need to upgrade the machine to Windows NT 4.0.

STATUS

This behavior is by design.

REFERENCES
