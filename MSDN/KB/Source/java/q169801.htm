FILE: Microsoft VM for Java Download Locations

 Last reviewed: January 29, 1998
Article ID: Q169801
The information in this article applies to:
  • Microsoft Win32 Virtual Machine for Java on the following platform: - x86

SUMMARY

Java1518.exe contains IE30Java.exe, which installs the Microsoft Virtual Machine for Java. Start the self-extracting executable to begin the setup of the Virtual Machine. The Microsoft Virtual Machine for Java is used to execute Java class files using the command line interpreter jview.exe or the Microsoft Internet Explorer.

The following file is available for download from the Microsoft Software Library: 

 ~ Java1518.exe (size: 1630883 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

MORE INFORMATION

This installation of the Virtual Machine is only to be used to upgrade existing installations of the Microsoft Virtual Machine for Java, available in Internet Explorer version 3.x. Please read the license agreement that is presented at installation time for more information.

If you require a redistributable copy of the Virtual Machine, please obtain the most recently released SDK for Java from http://www.microsoft.com/java. The license agreement of the SDK for Java explains all redistribution requirements.
