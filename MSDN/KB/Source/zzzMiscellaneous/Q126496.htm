WD: How to Obtain the Updated 32-Bit Word 6.x/7.0 Converter

 Last reviewed: November 14, 1997
Article ID: Q126496
The information in this article applies to:
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, version 7.x

SUMMARY

Included below is information on how to obtain the updated 32-bit Word 6.0/7.0 for Windows converter from online services. The contents of the Readme.txt is also included, which provides installation instructions.

This converter update (Mswrd632.cnv) has been released to add support for import of Korean Word 7.0 and Simplified Chinese Word 7.0 for Windows 95 documents into the U.S. releases of Word 6.0 for Windows NT and Word 7.0 for Windows 95. For a complete list of Asian Word versions that Word supports, please see the following Microsoft Knowledge Base article: 

   ARTICLE-ID: Q132431
   TITLE     : How to Convert Asian Word Documents in Word
NOTE: Other 32-bit programs can also use this converter to open Word 6.0/7.0 files, such as PowerPoint 7.0 for Windows 95, Publisher 3.0 for Windows 95, and Works for Windows 4.0 for Windows 95. This capability remains unchanged from the version of the converter released with Word 6.0 for Windows NT and Word 7.0 for Windows 95.

CONTENTS OF THE README.TXT

Dear Microsoft Customer:

Word for Windows 95, version 7.0 currently includes a converter (MSWRD632.CNV) that is installed to allow other 32-bit programs including Microsoft PowerPoint 7.0, Microsoft Publisher 3.0 and Microsoft Works 4.0 to open Word 6.x for Windows, Macintosh and NT and Word 7.0 documents. This converter is also used by Word 7.0 itself to open English-language documents created in the following Asian versions of Word: 

   Word 6.0 for Windows Japanese version
   Word 6.0 for Windows Korean version
   Word 6.0 for Windows Simplified Chinese versions
   Word 6.0 for Windows Traditional Chinese versions
   Word 7.0 for Windows 95 Japanese version
   Word 7.0 for Windows 95 Traditional Chinese versions
Included in this self-extracting EXE is an update to this converter that allows Microsoft Word 6.0 for Windows NT and Word 7.0 for Windows 95 users to open English-Language documents created in the following additional versions of Word: 

   Word 7.0 for Windows 95 Korean version
   Word 7.0 for Windows 95 Simplified version
To use the converter follow the setup instructions included in this letter. If you have questions about installing or using the converter, one of our Product Support Services technicians will be glad to assist you. In the United States call (206) 462-9673 between 6:00 AM and 6:00 PM Pacific time. Outside the United States, contact your local Microsoft subsidiary. Setup Instructions:

NOTE: The Word 6.0 for Windows/Macintosh converter supplied with Word must be installed before installing this update.

Determining if the converter is already installed:

  1. Run Microsoft Office or Word setup.

  2. Select Add/Remove.

  3. Select Converters, Filters, and Data Access, then select Change Options.

  4. Select Converters and then select Change Options.

  5. If Word 6.0 for Windows/Macintosh is checked, the converter is installed. If the converter is not checked, do so and then choose OK and Continue.

Installing the Updated Converter:

  1. Launch File Manager or Explorer and navigate to the source path where you downloaded this converter.

  2. Copy MSWRD632.CNV to <dr:>\Program Files\Common Files\MicrosoftShared\Textconv if you are using Windows 95. If you are using Windows NT, copy MSWRD632.CNV to <dr:>\WINNT35\MSAPPS\TEXTCONV, where <dr:> is the drive where Windows is located.

Windows will prompt you to replace the existing file. Choose Yes.

Release Notes:

  • This converter is for use with Windows 95 and Windows NT only. It will not run under Windows 3.11 or prior releases. This converter will also only work with 32-bit clients programs that use 32-bit converters, for example, PowerPoint 7.0 for Windows 95, Publisher 3.0 for Windows 95, and Works for Windows 4.0.
