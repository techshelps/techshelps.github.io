Setting Up MS Internet News to Work with MSN

 Last reviewed: October 7, 1997
Article ID: Q160984
The information in this article applies to:
  • The Microsoft Network version 1.2, 1.3, 2.0, 2.5

SYMPTOMS

When you attempt to connect to a newsgroup on MSN, The Microsoft Network, you may not see any content or you may receive an error message.

CAUSE

Microsoft Internet News may not be properly set up to work with MSN.

RESOLUTION

Follow these steps to set up Internet News to work with MSN:

  1. Start Internet News.

  2. On the News menu, click Options.

  3. Click the Server tab.

  4. Type your name and e-mail address in the appropriate boxes.

  5. In the News Server box, click Add.

  6. In the News Server Name box, type "msnnews.msn.com" (without quotation marks).

  7. Click "My news server requires me to logon" and "Logon using secure password authentication" to select them.

  8. Click the Connection tab.

  9. Click "I use a modem to access my newsgroups," click The Microsoft Network, and then click OK.

  10. Repeat steps 5-9 to add the netnews.msn.com server or any other news 

        server to which you have access.

  11. Click OK until you return to Windows.
