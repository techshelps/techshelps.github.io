FILE: Storage.exe Contains Storage Device/Adapter Drivers Refs

 Last reviewed: February 17, 1998
Article ID: Q181301
The information in this article applies to:
  • Windows 95 Device Driver Kit (DDK), version 4.0

SUMMARY

Layered block device drivers are 32-bit flat-model device drivers that execute in protected mode. Windows 95 uses layered block device drivers to manage input and output to block devices, such as disks and CD-ROM devices. These drivers support partitioned and non-partitioned removable media as well as conventional, SCSI, and floptical floppy drives. Windows 95 uses a variety of device driver components to support different types of devices and software. Each component corresponds to a specific layer and has specific responsibilities and requirements. Storage.exe contains the Windows 95 DDK Storage Technology Reference Manual.
