NdisWriteConfiguration Not Supported in Windows 95

 Last reviewed: January 30, 1997
Article ID: Q162754
The information in this article applies to:
  • Microsoft Win32 Device Driver Kit (DDK) for the following platform: - Microsoft Windows 95

SUMMARY

NdisWriteConfiguration is used to write a parameter to the configuration database. It was not implimented in NDIS 3.x under Windows 95 nor was it implemented in NDIS 4.0 in the Windows 95 OEM Service Release 2.0. Any call to this function will always return NDIS_STATUS_FAILURE.

MORE INFORMATION

Microsoft plans to implement this function in the NDIS 4.1 implementation of the NDIS wrapper.
