XL98: Revision Number and Total Edit Time Boxes Are Blank

 Last reviewed: March 11, 1998
Article ID: Q182325
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

On the Statistics tab of the Properties dialog box (click Properties on the File menu), the Revision Number and Total Edit Time are blank.

CAUSE

This behavior occurs because Microsoft Excel 98 does not use either of these statistics.

STATUS

This behavior is by design of Microsoft Excel 98.

MORE INFORMATION

The Properties dialog box is shared by Microsoft Office 98 programs. The Revision Number and Total Edit Time properties are used by Microsoft Word 98 and Microsoft PowerPoint 98.

The Revision Number tracks the number of times that the file has been saved. The Total Editing Time tracks the number of minutes that the file has been opened for making changes since it was created.

REFERENCES

For more information about file properties, click Contents And Index on the Help menu (or on the Balloon Help menu if you are using a version of the Macintosh operating system earlier than 8.0), click the Index button in Microsoft Excel Help, type the following text 

   properties, file
and then click Show Topics. Select the "Use file properties to locate files" topic, and click Go To. If you are unable to find the information you need, ask the Office Assistant.
