Excel: Save As DBF 2 and DBF 3 Options Grayed

 Last reviewed: October 31, 1994
Article ID: Q43935

SUMMARY

The options to save a file as DBF 2 or DBF 3 in Microsoft Excel version 2.20 will save the defined database range on the active worksheet as a DBF 2 or a DBF 3 file. If no database is defined on the active worksheet, the options to save a file as DBF 2 or DBF 3 will be grayed and not selectable.

For more information on defining a database, see "Creating a Database" on Pages 152-153 of the "Microsoft Excel Reference" version 2.2 manual.
