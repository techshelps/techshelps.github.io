SBFM97: Update for Quickbooks Version 4.0 Filter

 Last reviewed: February 12, 1998
Article ID: Q165938
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Small Business Financial Manager 97 for Microsoft Excel 97

SUMMARY

An updated version of the Quickbooks version 4.0 filter for the Small Business Financial Manager 97 is available. This update corrects several problems in the Quickbooks version 4.0 filter. You can download this update free of charge. The self-extracting file, Sbfm97qb.exe, contains the following three files: Readme.txt, Aapqbw40.dll, and Msaap.dll.
