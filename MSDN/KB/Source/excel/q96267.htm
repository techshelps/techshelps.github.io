Excel: Cannot Insert Object from Microsoft PowerPoint 3.0

 Last reviewed: February 2, 1998
Article ID: Q96267
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 3.0 and 4.0
  • Microsoft PowerPoint for the Macintosh, version 3.0

SUMMARY

Microsoft PowerPoint version 3.0 for the Macintosh is an object linking and embedding (OLE) client application, but not an OLE server application; therefore, it is not possible to use the Insert Object command (on the Edit menu) in Microsoft Excel to embed a Microsoft PowerPoint object.

Microsoft PowerPoint version 3.0 and Microsoft Excel versions 3.0 and 4.0 for the Macintosh do support Publish and Subscribe under System 7.x. Therefore, the object (a slide for example) can be published as an edition from Microsoft PowerPoint. The edition can then be subscribed to a Microsoft Excel worksheet.

REFERENCES

"Microsoft Excel User's Guide 2," version 4.0, page 138 "Microsoft Excel User's Guide," version 3.0 for the Macintosh, page 319
