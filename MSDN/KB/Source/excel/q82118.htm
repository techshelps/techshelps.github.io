Excel: Options Button on Page Setup Dialog Box Doesn't Work

 Last reviewed: November 16, 1994
Article ID: Q82118
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0 4.0a

SYMPTOMS

When you choose Page Setup from the File menu in Microsoft Excel and select the Options button, nothing happens. There is a problem with some Hewlett-Packard (HP) LaserJet printer drivers that causes the Options button on the Page Setup dialog box to malfunction and do nothing when selected. This has been verified with the HP III driver versions 30.8.85 and 30.8.86.

NOTE: This also occurs when selecting the Options button from the Print Setup dialog that is displayed after choosing Print Setup from the File menu in the Microsoft Excel Help system.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.
