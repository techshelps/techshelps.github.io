MSQUERY: Some ODBC Drivers Do Not Support Delete

 Last reviewed: May 12, 1997
Article ID: Q134684
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0
  • Microsoft Query for Windows, version 1.0
  • ODBC Drivers
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Query for Windows 95, version 2.0
  • Microsoft Excel 97 for Windows
  • Microsoft Query 97 for Windows

SUMMARY

In Microsoft Query, you cannot delete database records if you are using the ODBC drivers designed to access the following file formats: 

   Microsoft Excel
   Text files
If you attempt to delete records from a file in either of these formats , you may receive the error message: 

   Feature not available

MORE INFORMATION

Microsoft Excel can use ODBC drivers to execute SQL queries on an external database. Using this capability, it is possible to perform actions on a database, such as Insert records, Update records, and Delete records.

The ODBC drivers shipped by Microsoft are all capable of performing the above operations, with the following exceptions: 

   Microsoft Excel ODBC driver - no Delete
   Microsoft Text ODBC driver  - no Delete
