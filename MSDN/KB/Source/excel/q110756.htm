XL5: Can't View Workbook Summary Info in Word for Windows

 Last reviewed: September 12, 1996
Article ID: Q110756
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0
  • Microsoft Word for Windows, versions 2.0, 6.0

SUMMARY

In Microsoft Word for Windows, you cannot view the summary information for a Microsoft Excel version 5.0 workbook.

MORE INFORMATION

In Microsoft Excel version 5.0, you have the option of saving a workbook with summary information. However, you cannot view this information in Microsoft Word for Windows when you open the workbook, or when you use the Find File feature.

If you open a Microsoft Excel version 5.0 workbook in Word for Windows version 6.0, when you choose Summary Info from the File menu, the Summary Info dialog box does not contain the summary information for the workbook.

If you use the Find File feature in Word for Windows to find a Microsoft Excel workbook, and you choose Summary from the View list, the summary information for the workbook is not displayed. The Document Size, Created Date, and Last Saved Date appear correctly.

Note that you CAN view the summary information for a Word for Windows document using the Find File feature in Microsoft Excel, though you cannot open the document in Microsoft Excel.
