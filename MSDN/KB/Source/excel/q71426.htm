Excel: Zoom Button Grayed in Page Preview

 Last reviewed: November 3, 1994
Article ID: Q71426
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.5, 2.2, 3.0, 4.0

SUMMARY

When Microsoft Excel is used on large screen monitors that are capable of displaying at least a full 8.5 x 11 page, the Zoom option in Page Preview may be dimmed (unavailable).

MORE INFORMATION

This behavior occurs because Microsoft Excel automatically displays worksheets zoomed to full size when in Page Preview on systems configured with monitors that allow the full size of the printed page to be viewed at once. The Zoom button is not needed and is therefore dimmed. There is also no way to display the document in the reduced (nonzoom) mode while in Page Preview when using a full-page display.
