XL: "Not Enough Memory" Error When Adding Page Breaks

 Last reviewed: March 27, 1997
Article ID: Q113250
2.x 3.00 4.00 5.00 5.00c 7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Excel for Windows versions 2.x, 3.0, 4.x, 5.x
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, if you attempt to add more than 1,000 manual page breaks, the following error message will be displayed: 

     Microsoft Excel
     Not enough memory.

CAUSE

Microsoft Excel is limited to 1,000 manual page breaks per worksheet. This limit is the combined total of horizontal and vertical page breaks. For example, if a worksheet contains 250 vertical page breaks, only 750 horizontal page breaks can be set.

STATUS

This behavior is by design of Microsoft Excel.

REFERENCES

"User's Guide," version 5.0, pages 249-250 "User's Guide 1," version 4.0, pages 532-533
