INSTALLATION INSTRUCTIONS FOR FINANCIAL MANAGER ADDITIONAL DRIVERS

Note: To use these drivers, Microsoft Excel for Windows 95 and Microsoft Small Business Financial Manager must be installed on your computer.

  1. Turn on your computer and start Microsoft Windows 95.

  2. Quit Microsoft Excel for Windows 95, if it is running.

  3. Insert the Microsoft Small Business Financial Manager Additional Drivers disk in the floppy disk drive (usually drive A).

  4. Use the appropriate procedure for the driver you want to install.

Businessworks for MS-DOS and Windows, version 9.0:

In Windows Explorer, copy the Aapbww.dll file from the floppy disk to the aap folder on your computer's hard disk (usually the C:\Msoffice\Excel\Aap folder).

M A S 90 Evolution/2 for MS-DOS, version 1.5:

In Windows Explorer, copy the Aapmas90.dll file from the floppy disk to the aap folder on your computer's hard disk (usually the C:\Msoffice\Excel\Aap folder).

Platinum Series for MS-DOS/Windows, version 4.1:

In Windows Explorer, copy the Aapplat.dll file from the floppy disk to the aap folder on your computer's hard disk (usually the C:\Msoffice\Excel\Aap folder).

Quickbooks for Windows version 3.1:

a. In Windows Explorer, copy the Aapqbw.dll and Qbdrivhk.dll files from 
   the floppy disk to the aap folder on your computer's hard disk
   (usually the C:\Msoffice\Excel\Aap folder).
b. In Windows Explorer, copy the Msaap.dll file to the \Windows\System 
   folder (usually the C:\Windows\System folder).

   Note: You will replace the existing Msaap.dll file with the file from
   the floppy disk.

ISSUES YOU MAY ENCOUNTER WHEN YOU IMPORT DATA

Businessworks for MS-DOS and Windows, version 9.0:

If you perform a prior period adjustment into a closed period for which you are no longer saving detailed transactions, you will need to re-import your accounting data instead of updating your accounting data.

Quickbooks for Windows, version 3.1:

A. You must have Quickbooks installed on your computer, and the company 
   .QBW file to import must be in the same folder as the Quickbooks program
   files.
B. While importing data, Financial Manager will need to open Quickbooks and 
   perform certain actions in Quickbooks. You should not touch you're
   keyboard or computer during this time when Quickbooks is open.
C. In a What-If Scenario profitability analysis, the detail information for 
   any sales items entered into QuickBooks without a product code are
   placed into the N/A item in Financial Manager. In QuickBooks, the sales
   reports by customer only include those items with a product code.
   Therefore, the total sales value in the QuickBooks sales report by
   customer will not match the total sales value in the Financial Manager
   profitability analysis worksheet.
D. In QuickBooks, the summary sales by item report includes sales tax 
   information. In Financial Manager, the profitability analysis worksheet
   does not include sales tax information. Therefore, the total sales
   revenue value in the QuickBooks summary sales by item report will not
   match the total sales value in the Financial Manager profitability
   analysis worksheet.
E. The QuickBooks summary sales by item report does not include inventory 
   adjustments, but all Financial Manager reports and analysis worksheets
   do. Therefore, if you have made inventory adjustments, the QuickBooks
   summary sales by item report will not match all of the Financial Manager
   reports and analysis worksheets. Since the QuickBooks profit and loss
   statement does include inventory adjustments, those values will match
   the Financial Manager values.

SUPPORT

Assistance with these drivers is provided to you through your Microsoft Small Business Financial Manager warranty support policy. Installing these drivers will not restart this warranty period. If your Microsoft Small Business Financial Manager warranty support period has expired, paid phone support as well as alternative options are available.

If your license to the Microsoft Small Business Financial Manger product was obtained from a hardware manufacturer, then technical support for these drivers is not available from Microsoft. Please contact your hardware manufacturer before installing these drivers to determine your support options.
