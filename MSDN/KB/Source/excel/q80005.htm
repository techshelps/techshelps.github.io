Excel: Placing a Carriage Return in a Dialog Text Edit Box

 Last reviewed: November 30, 1994
Article ID: Q80005
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0, 5.0
  • Microsoft Excel for OS/2, version 3.0

It is possible to force a carriage return to create multiple-line entries when you enter data into a text edit box in a custom dialog. To do this with Microsoft Excel for the Macintosh, place the cursor at the point where you want the carriage return to appear and press Option+Return. In Microsoft Excel for Windows and OS/2, use the key sequence Ctrl+Enter or Alt+Enter.
