Excel: GP Fault Using Resume() Function in MS Excel 4.0 Macro

 Last reviewed: September 13, 1996
Article ID: Q136143
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

You may receive a general protection (gp) fault when you use the Resume() statement in a macro written in the Microsoft Excel 4.0 Macro Language.

NOTE: In Microsoft Excel for Windows 95, version 7.0, you may receive an Illegal Instruction Error.

CAUSE

The GP fault may occur when you use a macro to open a file with a chart in it, pause the macro, and then run another macro to close the chart and resume the previous macro.
