Problems Saving Files Using FTP's Interdrive95

 Last reviewed: April 1, 1997
Article ID: Q147189

The information in this article applies to:

  • Microsoft Excel for Windows, version 5.0, 5.0c
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

If you are using FTP Software's Interdrive95 client to connect your Windows 95 computer to a NFS server, you may receive the following error messages when trying to save a file to the NFS server.

For Microsoft Excel: 

  Disk is Full

  -and-

  Document Not Saved

MORE INFORMATION

This problem occurs when you use the products listed at the beginning of this article and FTP Software's Interdrive 95 v1.0.

Interdrive95 is manufactured by FTP Software, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
