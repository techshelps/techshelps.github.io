Excel: Using Fill Down with Multiple Selections

 Last reviewed: November 30, 1994
Article ID: Q84871
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0
  • Microsoft Excel for OS/2, version 3.0

SYMPTOMS

When choosing Fill Down from the Edit menu, with multiple selections in the same column, the first value in the first selection is used to fill all the cells in each selection unless you hold down the CTRL key while making the first selection.

STATUS

This problem does not occur in later versions of Microsoft Excel.

MORE INFORMATION

When the multiple selections are in different columns, it is not necessary to hold down the CTRL key on the first selection. On the Macintosh, the COMMAND key is used to make a multiple selection.
