XL97: "Create Hyperlink Here" Does Not Work in Unsaved File

 Last reviewed: February 18, 1998
Article ID: Q156883
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, when you drag a cell with the right mouse button, and then click Create Hyperlink Here on the shortcut menu that appears, Microsoft Excel may not create a hyperlink.

CAUSE

This behavior occurs when you have not saved the workbook at least once.

RESOLUTION

Before you create a hyperlink in a workbook, save the workbook. After you save the workbook, you can create the hyperlink.

REFERENCES

For more information about hyperlinks in Microsoft Excel, click the Index tab in Microsoft Excel Help, type the following text 

   hyperlinks, creating
click Display, click "Create a hyperlink", and click Display again to see the "Create a hyperlink" help topic.
