XL98: Some Objects Not Saved in Custom Chart

 Last reviewed: February 26, 1998
Article ID: Q181770
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you save a text box, bitmap, or floating text in a user-defined chart, the objects fail to appear in the user-defined chart the next time you use it.

WORKAROUND

To work around this behavior, wait to save text boxes, bitmaps, or floating text in a user-defined chart. Add these objects to the chart after you finish creating the chart.

STATUS

This behavior is by design in Microsoft Excel 98 Macintosh Edition.

REFERENCES

For more information about creating charts, click Contents And Index on the Help menu (or on the Balloon Help menu if you are using a version of the Macintosh operating system earlier than 8.0), click the Index button in Microsoft Excel Help, type the following text 

   charts, creating
and then click Show Topics. Select the "Create a chart" topic, and click Go To. If you are unable to find the information you need, ask the Office Assistant.
