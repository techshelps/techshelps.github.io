Excel: README.TXT Documentation Errata Applies to Few Editions

 Last reviewed: September 12, 1996
Article ID: Q103098
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0

SUMMARY

The README.TXT file, dated 4/01/92, located in the directory or folder in which you installed Microsoft Excel, contains a section titled "Documentation Errata" that lists errors and changes in the documentation. This section applies only to early editions of the "User's Guide 1," "User's Guide 2," and "Function Reference." The changes listed in this document were incorporated into later releases of the manuals.

The documentation errata applies to the following document numbers: AB26296-0492, AB26297-0492, and AB26298-0392. The changes are incorporated in manuals whose document numbers end with -0692. The document number can be found on the back of the cover page of each manual.

NOTE: Microsoft Excel for Windows version 4.0a contains the updated manuals.
