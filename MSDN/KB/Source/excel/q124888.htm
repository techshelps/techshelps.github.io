XL: Intl Settings Changes Not Displayed with Open Dialog Box

 Last reviewed: March 27, 1997
Article ID: Q124888
5.00 7.00 WINDOWS kbusage kbenv

The information in this article applies to:

  • Microsoft Excel for Windows version 5.0
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

If a dialog box is displayed in Microsoft Excel, and you open the Control Panel dialog box and change the International settings (Regional Settings in Windows 95), the change is not reflected on the sheet.

RESOLUTION

To make the settings take effect, close the dialog box in Microsoft Excel before you change the International (Regional Settings) settings.

MORE INFORMATION

This behavior is by design. You cannot change the International (Regional Settings) settings while a Microsoft Excel dialog box is displayed on the screen.
