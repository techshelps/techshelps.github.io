Excel: Macro Slow If Run with Show Info Window Open

 Last reviewed: November 2, 1994
Article ID: Q47583

SUMMARY

If an Excel version 2.20 or 3.00 macro is run with the Show Info window open, the macro may slow down considerably. By closing the Show Info window, the macro returns to its normal speed. (The Show Info command is found in the Windows menu.)

The Show Info window shows information about the current active cell. As the macro runs, Show Info continues to display information about the active cell. This feature can cause a macro to slow down because each time the macro makes a new selection with the SELECT() function, the Show Info window must update the information.

The more options you select to display in the Show Info window (by choosing the Info menu and selecting Cell, Formula, Value, Format, Protection, Names, Precedents, Dependent, or Notes), the slower the macro may run.
