Excel: Pasting Defined Names from One Worksheet to Another

 Last reviewed: October 31, 1994
Article ID: Q41032

SUMMARY

In Microsoft Excel, when a cell containing a formula that uses a defined name is pasted into a new worksheet, the defined name is also pasted into the destination worksheet.

For example, in the source worksheet, cell A1 is named "myName" (myName is set equal to $A$1) and cell A2 contains the formula "=myName". When the formula is pasted into cell A10 on the destination worksheet, the destination worksheet lists "myName" as a defined name defined as $A$1.
