XL: External Link in WK4 File Displays #REF! Error Value

 Last reviewed: June 3, 1997
Article ID: Q123262
5.00 5.00c 7.00 97 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SYMPTOMS

In Microsoft Excel, when you open a Lotus 1-2-3 .wk4 file that contains a link to another file, you receive one of the following error messages: 

   Microsoft Excel 97
   ------------------

      The workbook you opened contains automatic links to information in
      another workbook. Do you want to update this workbook with changes
      made to the other workbook?

   When you click Yes, the cells are updated with the values. However, when
   you click No, the cells on the worksheet that contain a link to another
   file display the #REF! error value.

   Microsoft Excel 7.0
   -------------------

      This document contains links. Re-establish links?

   When you click Yes, the cells are updated with the values. However, when
   you click No, the cells on the worksheet that contain a link to another
   file display the #REF! error value.

   Microsoft Excel 5.0
   -------------------

      This document contains links. You can re-establish them with the Edit
      Links command.

   When you click OK, the cells on the worksheet that contain a link to
   another file display the #REF! error value.

CAUSE

This behavior occurs when you open a Lotus 1-2-3 .wk4 file that contains a link to another file, such as another Lotus 1-2-3 .wk4 file.

WORKAROUND

To update an external link in a Lotus 1-2-3 .wk4 file in Microsoft Excel, do the following:

  1. In Microsoft Excel, open the Lotus 1-2-3 WK4 file.

  2. Click OK in the dialog box that indicates that the document contains links.

  3. On the Edit menu, click Links.

  4. In the Links dialog box, select the link that you want to update.

  5. Click Update Now.

    NOTE: If you want to open the source document, click Open. This also updates the external link.

To avoid this behavior, save the file in the Microsoft Excel workbook format.

MORE INFORMATION

To open a Lotus 1-2-3 .wk4 file in Microsoft Excel 5.0, obtain the Lotus 1-2-3 .wk4 file converter. The Lotus 1-2-3 .wk4 file converter is available in the "Lotus 1-2-3 WK4 File Converter" (WE1130 ) Application Note.
