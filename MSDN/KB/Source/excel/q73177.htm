Location of Valid SendKeys, Send.Keys, and On.Keys Codes

 Last reviewed: July 14, 1995
Article ID: Q73177
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 5.0
  • Microsoft Excel for OS/2, versions 2.2, 2.21, 3.0

SUMMARY

The list of key codes that can be used with the Visual Basic SendKeys Method and the Send.Keys and On.Key functions in Microsoft Excel can be found in the following locations:

Microsoft Excel verison 5.0 - SendKeys Method help topic in the Visual 

                              Basic Reference

                            - On.Key help topic in the Microsoft Excel
                              Macro Functions Help
Microsoft Excel verison 4.0 - Page 302 of the "Microsoft Excel Function 
                              Reference," version 4.0 manual.
Microsoft Excel version 3.0 - Page 165 of the "Microsoft Excel Function 
                              Reference," version 3.0 manual.
Microsoft Excel version 2.x - Appendix on pages 373-375 of the "Microsoft 
                              Excel Functions and Macros," version 2.2.

REFERENCES

"Microsoft Excel Function Reference," version 4.0, page 302

"Microsoft Excel Function Reference," version 3.0, page 165

"Microsoft Excel Functions and Macros," version 2.2, pages 373-375
