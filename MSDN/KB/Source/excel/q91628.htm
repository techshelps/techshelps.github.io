Excel: Cannot Modify Patterns on Series Data Markers

 Last reviewed: November 30, 1994
Article ID: Q91628
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0

SUMMARY

When attempting to modify the pattern of a segment of a series on a 3-D line chart, the change is not displayed on the screen.

NOTE: This problem does not occur in later versions of Microsoft Excel.

MORE INFORMATION

Steps to Reproduce Problem

With a 3-D line chart containing more than 3 data points, do the following:

  1. Select a segment of the line (a line between only two data points) by holding down the CTRL key and clicking that segment with the mouse.

  2. From the Format menu, choose Patterns.

  3. Change the Border from Automatic to Custom and select a different Style from the drop-down style box.

  4. Change the Area setting from Automatic to Custom and change the Pattern, and then choose the OK button.

The changes made in the patterns dialog box are not seen on the chart. With other types of charts (3-D Column, 3-D Pie, and so on) the pattern on the screen will visibly change. This occurs in both embedded charts and chart files.
