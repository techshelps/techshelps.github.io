Excel: &c Code in Left or Right Header Section Overlaps Text

 Last reviewed: November 30, 1994
Article ID: Q82084
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 4.0, 5.0

SUMMARY

If you use the centering code (&c) in the Left or Right section of the Header Dialog under Page Setup, the text following it will be centered on the page as if it was actually entered into the Center section. As a result, the text located in these sections will overlap any text purposefully located in the Center section. (In Microsoft Excel version 5.0, the header text overwrites the information in the right section.)

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Excel listed above. We are researching this problem, and will post more information here in the Microsoft Knowledge Base as it becomes available.
