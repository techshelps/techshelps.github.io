MSQuery: SQL Server Connectivity Options

 Last reviewed: February 2, 1998
Article ID: Q121322
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 5.0
  • Microsoft Query for Macintosh, version 1.0

Microsoft Query version 1.0, which ships with Microsoft Excel version 5.0, provides an ODBC driver to access SQL Server for Windows NT. This driver allows access via two protocols: TCP/IP and ADSP.

TCP/IP

This protocol is included with all versions of SQL Server, and it requires MacTCP on the Macintosh client. MacTCP can be purchased from Apple Computer as a separate product for System 7.0 and System 7.1; MacTCP is included with System 7.5.

ADSP

This protocol is included with version 4.21a of SQL Server for Windows NT. Additional software for the Macintosh client is not required.
