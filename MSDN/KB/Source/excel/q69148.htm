font= Line in EXCEL.INI Doesn't Change Reference/Status Fonts

 Last reviewed: November 3, 1994
Article ID: Q69148
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0

SUMMARY

Changing the "font=" line in the EXCEL.INI file (versions of Excel prior to version 4.0) or in the EXCEL4.INI (version 4.0) does not change the fonts in the reference area or the status bar of an Excel 3.0 or 4.0 document. Both the reference area and the status bar are linked to the system font. This is a change from Excel 2.1x, where the font= line in the Excel section of the WIN.INI file would change the fonts in the reference area and status bar to the selected font.

For more information on the "font=" line, query on the following words in the Microsoft Knowledge Base: 

   WIN.INI and and EXCEL4.INI and font=
