XL5: Bad or Missing WPD File Prevents Proper Page Setup

 Last reviewed: September 12, 1996
Article ID: Q112290
The information in this article applies to:

- Microsoft Excel for Windows, version 5.0

SUMMARY

In Microsoft Excel version 5.0 for Windows, if you are using a PostScript printer and the Windows PostScript description (WPD) file for your printer is bad or missing, the following errors may occur:

  • You may be unable to print or view the file in print preview in landscape mode (even though you can change the orientation setting in the Page Setup dialog box).
  • You may be unable to print or view the file in print preview in a paper size other than the default paper size (even though you can change the paper size in the Page Setup dialog box).

To correct these problems, you may need to remove and reinstall your PostScript printer driver and associated WPD files.

MORE INFORMATION

When you install a PostScript printer driver in Microsoft Windows, two files are installed: the PostScript printer driver file (PSCRIPT.DRV) and a WPD file designed especially for the printer you are using. If this WPD file has been damaged or corrupted, or if the file is missing, Microsoft Excel may only be able to print in portrait mode, using the default paper size.

To remove and reinstall the PostScript driver and associated WPD files, do the following (before you proceed, make sure you have your Microsoft Windows disks available):

  1. In Windows Control Panel, choose the Printers icon.

  2. In the Printers dialog box, choose the Remove button to remove any PostScript printers that you have installed.

  3. Exit Microsoft Windows.

  4. Delete the following files from the WINDOWS and WINDOWS\SYSTEM directory: 

       PSCRIPT.DRV   (the PostScript printer driver file)
   *.WPD         (Windows PostScript description files)

  5. Restart Microsoft Windows.

  6. In Windows Control Panel, use the Add and Install buttons to re- install the drivers for your PostScript printer. Since the necessary files were deleted in step 3, you will be prompted to insert some of the Microsoft Windows disks so that the files may be re-installed.

Once you have completed these steps, return to Microsoft Excel and use Print Preview to see whether you can print in landscape mode or use different paper sizes.
