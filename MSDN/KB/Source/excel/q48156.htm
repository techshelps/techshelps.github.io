Excel 2.20 Err Msg: Out of Application Memory

 Last reviewed: October 31, 1994
Article ID: Q48156

SUMMARY

The error message "Out of Application Memory" can be caused by using an old version of MultiFinder, usually version 1.0. Early versions of MultiFinder do not distinguish between Excel version 2.20 and earlier versions of Excel, which were restricted to operation in the first megabyte of memory. In this case, Excel works under Finder, but not under MultiFinder.

To enable Excel to make full use of the system's memory under MultiFinder, be sure to use the version of MultiFinder included with your operating system. Excel is supported only under System 6.0.2 or later, with the accompanying MultiFinder.
