XL7: Typing Not Accepted While Picklist Is Displayed

 Last reviewed: March 27, 1997
Article ID: Q135571
7.00 WINDOWS kbprb

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you are typing in a cell and the Picklist drop-down list is displayed on the screen, keyboard input is not entered into the cell.

CAUSE

While the Picklist drop-down is displayed, most keyboard input is ignored.

WORKAROUND

To work around this problem, do any of the following to dismiss the Picklist drop-down list:

  • Press ESC
  • Reselect the current cell or select any other cell
  • Click the formula bar

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
