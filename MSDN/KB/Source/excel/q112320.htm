XL: Cannot Delete Default Headers or Footers

 Last reviewed: December 17, 1996
Article ID: Q112320
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SUMMARY

When you view the Header/Footer tab in the Page Setup dialog box, several default headers and footers are available in the drop-down Header and Footer list boxes. These default headers and footers cannot be deleted.

MORE INFORMATION

In the versions of Microsoft Excel listed at the beginning of this article, default headers and footers are created automatically based on information about the workbook and worksheet located in the File Summary Info dialog box.

To print a worksheet without a header or footer, select the (none) option in the appropriate header or footer list.

NOTE: These default headers and footers do not increase the file size of your document.
