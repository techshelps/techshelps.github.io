Q+E Err Msg: Cannot Open <Filename>.DBT

 Last reviewed: October 31, 1994
Article ID: Q45957

SUMMARY

Files with the extension .DBT are memo files created by dBASE that contain the variable-length text associated with the memo field data type. These files are used only when a memo is accessed by Q+E. If the .DBT does not have the same filename prefix as the .DBF, or if the .DBT is not in the same directory as the .DBF file, Q+E displays the following message when trying to view the memo from Q+E: 

   Cannot open <filename>.DBT.
