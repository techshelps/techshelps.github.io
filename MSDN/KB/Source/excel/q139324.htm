XL7 Err Msg: "Unexpected Error" with Multi-User and Access Form

 Last reviewed: March 27, 1997
Article ID: Q139324
7.00 WINDOWS kbinterop kberrmsg

The information in this article applies to:

  • Microsoft Excel for Windows 95, versions 7.0

SUMMARY

In Microsoft Excel 7.0 for Windows 95, when you attempt to use the Microsoft Access Report or Access Form command on the Data menu in a multi- user workbook file, you will receive the following error: 

   Unexpected error creating Microsoft Access form or report

MORE INFORMATION

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q135502
   TITLE     : XL7: Errors Using AccessLinks with Protected Worksheet

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel version 7.0 for Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
