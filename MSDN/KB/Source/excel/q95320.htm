Excel CDK: Setup Fails

 Last reviewed: September 12, 1996
Article ID: Q95320
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0

SUMMARY

The setup program for the Microsoft Excel Courseware Developer's Kit (CDK) may fail if your CONFIG.SYS file does not contain the line LASTDRIVE=Z. When the setup fails it may display the initial blue setup screen, then exit unexpectedly with no error and return to the Program Manager within Windows.

Workaround

To correct this problem, add the line LASTDRIVE=Z to your CONFIG.SYS file.

To add LASTDRIVE=Z to your CONFIG.SYS file:

  1. Open CONFIG.SYS in a text editor (such as Windows Notepad).

  2. Type LASTDRIVE=Z on any line.

  3. From the File menu, choose Save.

  4. Exit Windows Notepad, then Exit Windows and reboot (restart) your computer so that the new configuration information is used.

Now you are ready to run the Microsoft Excel CDK setup program again.
