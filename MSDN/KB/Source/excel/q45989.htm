What Q+E Loads into Memory When a File Is Opened

 Last reviewed: October 31, 1994
Article ID: Q45989

SUMMARY

When a file is opened in Q+E, the entire file is not loaded into memory. Only the records that are displayed on the screen are loaded into memory. This generates a lot of disk access but allows access to more data than can fit into memory.

If you have extended or expanded memory, a disk caching program such as SMARTDrive lowers the disk access and enhances the performance of Q+E.
