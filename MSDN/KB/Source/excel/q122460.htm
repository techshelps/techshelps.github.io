Excel: Default Font Used with Lotus WYSIWYG Spreadsheet

 Last reviewed: March 27, 1997
Article ID: Q122460
4.00 4.00a 5.00 5.00c 7.00 7.00a WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0, 5.0c
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a

SUMMARY

A Lotus 1-2-3 spreadsheet file that contains WYSIWYG (What You See Is What You Get) formatting can be opened in Microsoft Excel. The WYSIWYG fonts will be substituted for the a pre-defined Microsoft Windows font. The default font used by the Microsoft Excel worksheet will be the first font listed in the WYSIWYG Format Fonts menu.

MORE INFORMATION

Lotus 1-2-3 uses Courier 10 as its default font. However, additional fonts may be used through the WYSISYG fonts set installed through the WYSIWYG add-in.

The WYSIWYG fonts and their corresponding Microsoft Excel fonts are as follows: 

   Lotus 1-2-3 WYSIWYG             Microsoft Excel
   -----------------------------------------------

   SWISS                           HELV
   DUTCH                           TMS RMN
