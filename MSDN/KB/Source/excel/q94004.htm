Excel Err Msg: "Error Printing on <Printer>," in Group Edit

 Last reviewed: November 4, 1994
Article ID: Q94004
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0

SUMMARY

In version 4.0 of Microsoft Excel for Windows, if you are in Group Edit mode and you choose Print Preview from the File menu, you may receive the following error message 

   Error printing on <printer>
where <printer> is the name of the active printer. Printing from print preview in Group Edit mode may also cause a document to be printed multiple times.

Microsoft has confirmed this to be a problem in Microsoft Excel for Windows version 4.0. The problem has been corrected in Microsoft Excel for Windows version 4.0a.

MORE INFORMATION

Workaround

To work around this problem, use one of the following methods to print while in Group Edit mode:

  • Choose Print from the File menu.

    -or-

  • Choose the Print tool on the tool bar to print the active document.
