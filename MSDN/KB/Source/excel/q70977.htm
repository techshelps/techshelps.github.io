Excel: Files on Original Disks Cannot Be Used Without Setup

 Last reviewed: November 1, 1994
Article ID: Q70977

SUMMARY

The files on the original disks for Microsoft Excel version 3.00 cannot be used by themselves. They must be installed on a hard drive or another disk using the Microsoft Excel Setup program.

If the Microsoft Excel Setup program is not used, the error message "The file '<filename>' could not be opened/printed (the application is busy or missing)" may be displayed if you attempt to open these files.

MORE INFORMATION

The files on the original ship disks (except for the "Read Me" and "Teach Text" files) are in a compressed format. Microsoft Excel Setup decompresses the files which, in turn, makes the files usable.
