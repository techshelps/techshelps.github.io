Excel 1.0-4.0: Window Menu Displays Only 28 Document Names

 Last reviewed: November 2, 1994
Article ID: Q20293
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0

SUMMARY

The Window menu in Microsoft Excel versions 1.0 to 4.0 displays a maximum of 28 document names.

There are several options to activate a document that is not listed:

  1. Press COMMAND+M until that document becomes active. COMMAND+M selects the next active document. (COMMAND+SHIFT+M is another alternative; it selects the previous active document.)

  2. Drag the other document windows so that you can click on the desired document's window.

MORE INFORMATION

In Microsoft Excel version 2.20, 3.00 and 4.00, when more than 28 documents are open (and not hidden), the option "More Windows" becomes available under the Window menu. Selecting this option will bring up a dialog box with a list of the open documents so that you can select the document to be activated.
