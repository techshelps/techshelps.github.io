Excel Setup Fails (Error -49)

 Last reviewed: November 1, 1994
Article ID: Q70750

SUMMARY

If the Microsoft Excel Setup program is launched while Excel is running in the background, error -49, "File System Error, Unable to continue operation" may appear. After you click OK to close the alert box, another alert states "Install Interrupted. Some Options May Be Only Partially Installed," and the Setup program returns to the initial Setup screen.

This error occurs because the Setup program is trying to write information to the "Excel Settings" file, which is already in use by the other copy of Excel.

For best results with Setup, quit all copies of Excel before running the Setup program.
