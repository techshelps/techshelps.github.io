XL7: Data Map Distorted After Changing Row or Column

 Last reviewed: September 13, 1996
Article ID: Q134263
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, if a data map object is embedded in a worksheet, if you format the object with the Move And Size With Cells option, the map is distorted if you change the size (width or height) of the row or column over which the map is embedded.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
