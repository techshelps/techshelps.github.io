XL98: No Info Window in Microsoft Excel 98 Macintosh Edition

 Last reviewed: February 25, 1998
Article ID: Q181707
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel 98 Macintosh Edition, when you click Preferences on the Tools menu and click the View tab, no Info Window option appears on the View tab.

CAUSE

The Info window feature is not available in Microsoft Excel 98 Macintosh Edition. It is not possible to display the Info window by clicking an option or by running a macro.

MORE INFORMATION

In earlier versions of Microsoft Excel, the Info window feature allows you to display a window that contains detailed information about the active cell, such as cell formulas, values, format, and protection status. To display the Info window in Microsoft Excel 5.0, follow these steps:

  1. On the Tools menu, click Options.

  2. Click the View tab.

  3. Select the Info Window check box, and then click OK.

You can also display the Info window in earlier versions of Microsoft Excel with a macro command. The command to use is listed in the following table. 

   In this macro language              Use this command
   ------------------------------------------------------------------------

   Microsoft Excel version 4.0 (XLM)   =SHOW.INFO()

   Visual Basic for Applications       Application.DisplayInfoWindow = True
NOTE: If you attempt to display the Info window in Microsoft Excel 98 Macintosh Edition with a macro command, the command does not work, and no error messages appear.

REFERENCES

For more information about the Info window in Microsoft Excel 98 Macintosh Edition, click the Index button in Microsoft Excel Help, type the following text 

   info window
Click Show Topics, double-click the "Troubleshoot auditing a worksheet," and then click "What happened to the Info Window?."
