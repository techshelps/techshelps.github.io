UNCONF: WB1 Converter May Not Be Installed with Complete Setup

 Last reviewed: April 1, 1997
Article ID: Q147181
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Office Standard for Windows 95, version 7.0
  • Microsoft Office Professional for Windows 95, version 7.0

SYMPTOMS

When you try to open a Quattro Pro version 5.0 for Windows (*.WB1) file in Microsoft Excel version 7.0, you may receive the following error message: 

   Cannot open Quattro Pro for Windows file
The Setup program may not install the Quattro Pro for Windows converter even if you choose Complete installation.

RESOLUTION

To resolve this, follow these steps:

  1. Start the Setup program to enter Maintenance mode setup.

  2. Click the Add/Remove button. Click Microsoft Excel to select it, and click Change Options. Click Spreadsheet Converters to select it, and
then 
   click Change Option. Click to clear the check box next to Quattro Pro
for 
   Windows.

  • Click OK twice. The Components to remove should display 0.

  • Click Microsoft Excel to select it, and click Change Options. Click Spreadsheet Converters to select it, and then click Change Option. Click to select the check box next to Quattro Pro for Windows.

  • Click OK twice. The Components to Add should display 1.

  • Click Continue and insert the appropriate disks (or Compact Disc) as the Setup program asks for them.

    The Quattro Pro for Windows converter should now be installed properly.
