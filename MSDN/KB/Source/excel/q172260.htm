XL97: Page Fault Clicking Sheet Tab When Formula Bar Is Active

 Last reviewed: January 7, 1998
Article ID: Q172260
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, if you click a worksheet tab while an embedded chart is active, you may receive the following error message: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.
If you click Details, you receive an error message similar to the following: 

   EXCEL caused an invalid page fault in module KERNEL32.DLL at
   0137:bff9a5d0.

CAUSE

This problem occurs if the following conditions are true:

  • The embedded chart contains one or more drawing objects or AutoShapes. 

        -and-
  • You select one of the drawing objects or AutoShapes. 

        -and-
  • You click the formula bar.

WORKAROUND

To work around this problem, press ESC or press ENTER before you click another sheet. This deactivates the formula bar.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
