XL5: Arrow Keys Inactive in an Object Link Edit Box

 Last reviewed: September 12, 1996
Article ID: Q107834
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SUMMARY

When you select an object (such as a check box) that uses a linked edit box with a cell reference, you cannot use the arrow keys to change the reference in the cell link.

You will not be able to use the arrow keys to change the cell link because, when an object has been selected, there is no active cell.

MORE INFORMATION

Certain objects, such as check boxes or drop-down boxes, allow cell links to be assigned to them so that a reference on a worksheet can be linked to that object.

When you try to assign or change the cell link for these objects, you must either type the cell reference in the cell link box or select the cells with the mouse.

Steps to Reproduce Behavior

  1. Draw a check box using the Forms toolbar.

  2. Select the check box.

  3. From the Format menu, choose Object.

  4. Select the Control tab.

  5. On the Control tab, select the Cell Link box.

  6. Use the mouse to select a cell on the worksheet.

  7. Press arrow keys on the keyboard to change the cell reference.

Note that you are unable to change the linked cell address by pressing the arrow keys.
