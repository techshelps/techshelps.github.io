Cannot Setup Generic/Text Only Driver from Q+E

 Last reviewed: November 2, 1994
Article ID: Q67764

SUMMARY

When you try to use Printer Setup from Q+E and you select the Generic/Text Only printer, the Setup button will not generate the Generic/Text Only dialog box.

Microsoft is are researching this problem and will post new information here as it becomes available.

Workaround

Run the Control Panel from the Control menu (ALT+SPACEBAR) and proceed with the printer setup from there.
