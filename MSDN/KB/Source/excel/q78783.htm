Excel: IS Functions with External References Appear Incorrect

 Last reviewed: November 3, 1994
Article ID: Q78783
Summary:

Certain IS functions (ISNUMBER, ISBLANK, ISTEXT, ISLOGICAL, and ISNONTEXT) that contain references to external worksheets may appear to evaluate incorrectly if Microsoft Excel for the Macintosh version 3.0 opens a document created in Excel version 2.2. If you reenter the formulas by activating the Formula Bar and pressing RETURN, the functions calculate correctly.

More Information:

If a worksheet contains many IS functions, it may not be practical to enter the formulas again. To enter all formulas at once and ensure that functions containing external references are evaluated correctly, use the following procedure:

  1. From the Formula menu, choose Replace.

  2. Type an equal sign in the Find What field.

  3. Type an equal sign in the Replace With field.

  4. Choose Replace All.

This procedure only needs to be done once for each document. If you save the document in Normal format and open it again, the functions continue to evaluate correctly.

Microsoft is researching this problem and will post new information here as it becomes available.
