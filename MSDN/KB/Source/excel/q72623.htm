Multiplan Help Does Not Operate Correctly in Excel

 Last reviewed: November 3, 1994
Article ID: Q72623
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 5.0
  • Microsoft Excel for OS/2 version 3.0

SUMMARY

The Multiplan option under the Help menu in Microsoft Excel does not find the help topic correctly.

MORE INFORMATION

Workaround

Help for Multiplan commands can be accessed manually within Microsoft Excel Help:

  1. From the Microsoft Excel Help menu, choose Index.

  2. Select the Switching From Multiplan option.

The Help screen will show you a list of Multiplan topics from which to choose.

Microsoft is researching this problem and will post new information here as it becomes available.
