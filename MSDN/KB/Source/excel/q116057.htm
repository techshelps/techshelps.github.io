XL: Trendlines Unavailable for Area Chart Type

 Last reviewed: February 2, 1998
Article ID: Q116057
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel 97 for Windows
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel, you may be unable to add a trendline to an area chart.

CAUSE

When you use the Area chart type, the Trendlines command may be unavailable because trendlines can only be added if the area chart type is not stacked.

WORKAROUND

The default Area chart type is a stacked area, even if only one series is being charted. If this chart type is selected, then the Trendlines command (on the Insert menu) will be unavailable (dimmed). To unstack an area chart type, follow these steps:

  1. Select the area series.

  2. On the Format menu, click Chart Type. (Or, click Chart Type on the Chart menu.)

  3. In the Chart Type dialog box, make sure Area Group is selected. Then, click Options.

  4. In the Options dialog box, click the Subtype tab, click the first (far left) subtype, and then click OK.

The area chart becomes unstacked. To create trendlines, select the series and click Trendlines on the Insert menu (or Add Trendline on the Chart menu).

MORE INFORMATION

In Microsoft Excel, you can add trendlines to data series on a chart for the following chart types: area, bar, column, line, stacked column chart, and xy (scatter).

To create a trendline for a particular data series, follow these steps:

  1. Select the series.

  2. On the Insert menu, click Trendlines. (Or, click Add Trendline on the Chart menu.)
