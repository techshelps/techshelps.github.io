Excel: References Change in File Transferred to Lotus 1-2-3

 Last reviewed: October 7, 1997
Article ID: Q74821
2.20 3.00 MACINTOSH kbother

SUMMARY

When you transfer a file from Microsoft Excel for the Macintosh version 2.2 or 3.0 to Lotus 1-2-3, absolute column references are displaced by one column if cell A1 on the Excel spreadsheet is blank.

MORE INFORMATION

For example, the following formulas from Excel 

   =$C1
   =$C$1
convert to Lotus 1-2-3 as 

   +$B1
   +$B$1
if cell A1 on the Excel spreadsheet is blank.

To work around this problem, enter something in cell A1 before transferring the spreadsheet to Lotus. For a solution to this problem, contact Microsoft Product Support Services at (425) 635-7080.
