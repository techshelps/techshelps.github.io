Excel: allShare Can Save Only to the Hard Disk Desktop

 Last reviewed: November 2, 1994
Article ID: Q70103

SUMMARY

With certain settings, the allShare AFP File Server by EveryWare Development Corporation does not allow Microsoft Excel to save a file using the Save As command (on the File menu) to any folder (directory) other than to the hard disk desktop (root directory).

This is a confirmed problem with allShare. For more information on this problem, contact EveryWare Development Corporation at the following address and phone number: 

   EveryWare Development Corporation
   A Division of McKenzie Brown Canada, Inc.
   2176 Torquay Mews
   Mississauga, Ontario, Canada L5N 2M6
