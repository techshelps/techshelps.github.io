XL: Can't Open Quattro Pro 6.0 Files

 Last reviewed: October 21, 1997
Article ID: Q139665
7.00 97 WINDOWS kbusage kb3rdparty

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel, when you attempt to open a Quattro Pro 6.0 (.wb2) file, you receive the following error message: 

   Cannot read this binary file. If the file was created in a version of
   Microsoft Excel later than 7.0, use that version to save it as a
   Microsoft Excel 5.0 file. Open file as text?

CAUSE

Microsoft Excel does not have a converter for opening Quattro Pro for Windows, version 6.0 files (.wb2 file extensions).

WORKAROUND

In order for Microsoft Excel to import a .wb2 file saved from Quattro Pro for Windows version 6.0, use the following steps to first save the file in the Quattro Pro 5.0 format:

  1. Open the file in Quattro Pro for Windows version 6.0.

  2. On the File menu, click Save As.

  3. In the Save File As Type box, select QPW (*.WB1) and click OK.
