Blank Chart Is Generated in Excel When Pressing F11

 Last reviewed: November 2, 1994
Article ID: Q57906
The information in this article applies to:
  • Microsoft Excel for Windows, version 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2 versions 2.2, 2.21, 3.0

SUMMARY

With an Excel chart active, if you choose Select Chart from the Chart menu and then press F11 (or File New Chart), a new identical chart will be created. However, if the supporting worksheet has been closed or deleted, the new chart will be blank and the Paste and Paste Special command will be unavailable in the Edit menu.

MORE INFORMATION

If you choose Select Chart from the Edit menu, choose Copy from the Edit menu, and then press F11 for a new chart, the new chart is blank but Paste and Paste Special are available in the Edit menu. The new chart is also blank if, after choosing Copy on a chart, you go back to the worksheet, highlight a range of data, and press F11.
