Installation instructions for the Financial Manager - Peachtree 3.5 for Windows driver:

Note: To use this driver, Microsoft Excel for Windows 95 and Microsoft Small Business Financial Manager must be installed on your computer.

  1. Turn on your computer and start Microsoft Windows 95.

  2. Quit Microsoft Excel for Windows 95, if it is running.

  3. In Windows Explorer, copy the Aappw35.dll file from the folder in which the files were downloaded to the aap folder on your computer's hard disk (usually the C:\Msoffice\Excel\Aap folder).

  4. In Windows Explorer, copy the Msaap.dll file from the folder in which the files were downloaded to the \Windows\System folder on your computer's hard disk (usually the C:\Windows\System folder).

Note: You will replace the existing Msaap.dll file with the downloaded file.
