Excel: Problem with Database Tutorial When QE.XLA Loaded

 Last reviewed: November 22, 1994
Article ID: Q82007

The information in this article applies to:

  • Microsoft Excel for Windows, version 4.0

SUMMARY

You will be unable to complete the "Using a Database" lesson in the Microsoft Excel 4.0 for Windows Tutorial if you have previously opened the Q+E add-in, QE.XLA.

MORE INFORMATION

When you run the "Using a Database" lesson, you will be prompted to select the Database option from the Data menu. If you have previously loaded QE.XLA, you will be unable to make this selection because it changes the commands on the Data menu. Specifically the "Database" command becomes "Database...", thereby making it an invalid selection as far as the tutorial is concerned.

To continue with the lesson, choose the Demo button that the tutorial provides. This will step you through the lesson automatically. It appears as if the Demo executes exactly the same commands that you are instructed to execute. The selection, however, is actually made on a lower level that is not affected by the appearance of the commands on the Data menu.
