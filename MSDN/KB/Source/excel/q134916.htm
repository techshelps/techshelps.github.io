XL7: Template Wizard Functions Show Up in Function Wizard

 Last reviewed: March 27, 1997
Article ID: Q134916
7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, functions from the Template Wizard add-in are displayed in the user-defined function category when you use the Function Wizard.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Template Wizard can automatically copy the contents of cells to a database each time you save a workbook that is based on a template. The Template Wizard walks you through the steps of linking cells in the template to a field in a database or a Microsoft Excel List.

The Function Wizard makes it easier to enter formulas. To start the Function Wizard, click Function on the Insert menu. The functions are listed by category. When you select a function from the list box, the definition of the function and its arguments will automatically appear.
