XL: Documentation Incorrectly References ODS Driver

 Last reviewed: March 27, 1997
Article ID: Q131295
5.00 5.00c 7.00 WINDOWS kbnetwork kbdocerr

The information in this article applies to:

  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

The documentation for the Microsoft Excel versions listed above incorrectly imply that there is an ODBC ODS Gateway driver available from Microsoft. This information is incorrect. This driver was produced by a third-party vendor, MDI.

MORE INFORMATION

Microsoft Excel 5.0

Page 441 of the Microsoft Excel 5.0 "User's Guide," incorrectly implies that there is an ODS Gateway ODBC driver available from Microsoft.

Microsoft Excel 7.0

A help topic in the Microsoft Excel 7.0 help file incorrectly implies that there is an ODS Gateway ODBC driver available from Microsoft. To locate the help topic, click the Index tab in Microsoft Excel 7.0 Help, type the following text 

   ODBC Drivers
and click the Display button. Double-click the topic "Databases you can access with Microsoft Query".

For a list of some third party vendors for ODBC drivers, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q115293
   TITLE     : MSQuery: Third Party Vendors for ODBC Drivers
