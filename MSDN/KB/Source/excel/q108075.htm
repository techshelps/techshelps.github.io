Bitmap Image in Print Preview But Doesn't Print to Printer

 Last reviewed: March 27, 1997
Article ID: Q108075
3.00 4.00 4.00a 5.00 7.00 WINDOWS kbprint

The information in this article applies to:

  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

On a Microsoft Excel worksheet or chart, bitmap images such as drawing objects and pasted graphics appear in print preview but do not print to a pen-based plotter.

CAUSE

Pen-based plotter devices cannot print bitmap images.

WORKAROUND

Pen-based plotter devices are capable of printing Windows Metafile images. To insert a Windows Metafile picture in Microsoft Excel version 5.0, do the following:

  1. Select the worksheet or chart you want to insert the picture on.

  2. From the Insert menu, choose Picture, and from the File Name list, select a picture that is in the Windows Metafile format (.WMF extension) and choose OK.
