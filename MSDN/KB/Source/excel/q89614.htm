Excel: GET.DOCUMENT Doc Errors for TYPE_NUM 19 and 20

 Last reviewed: November 30, 1994
Article ID: Q89614
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

Page 197 of the "Microsoft Excel Function Reference Guide" contains a documentation error in the TYPE_NUM descriptions of the GET.DOCUMENT function. The word Options has been omitted in the descriptions for TYPE_NUM 19 and TYPE_NUM 20. The description for TYPE_NUM 19 should read "If the Precision As Displayed checkbox is selected in the Options Calculation dialog box, returns TRUE; otherwise FALSE." The description for TYPE_NUM 20 should read "If the 1904 Date system check box is selected in the Options Calculation dialog box, returns TRUE, otherwise FALSE."

REFERENCES

"Microsoft Excel Function Reference Guide," version 4.0, page 197
