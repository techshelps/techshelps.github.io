Excel: Links Dialog Box Doesn't Show Filenames in System 7.0

 Last reviewed: November 1, 1994
Article ID: Q75549

SUMMARY

When running under System 7.0, Microsoft Excel for the Macintosh versions 2.2 and 3.0 will not completely display linked filenames within the Links dialog box (on the File menu). Under Systems prior to System 7.0, Excel condenses the pathname in order to display the complete filename. Under System 7.0, the font in the Links file list box is condensed to show as many characters as possible, but the pathname is not shortened in order to display the complete filename.

MORE INFORMATION

This may cause confusion if a document has been linked to several files that were saved into the same folder, since the Links dialog box will show the same path for each of these files but may not include the filename. To see the filenames of the files that are being linked without their associated paths, you must open all the linked documents.
