XL: Auto Save Doesn't Save Sheets Bound in Workbooks

 Last reviewed: February 2, 1998
Article ID: Q96594

The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0 and 4.0a
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

The Auto Save add-in macro cannot detect when changes have been made to worksheets, charts, and macro sheets that are bound in workbooks. Therefore, documents that are bound in workbooks are not saved by the Auto Save add-in.

This feature is by design.

REFERENCES

"Microsoft Excel User's Guide 2," version 4.0, page 198
