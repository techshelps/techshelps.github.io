Excel: RANDBETWEEN() is in Add-In Functions

 Last reviewed: November 30, 1994
Article ID: Q89904
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

On page 344 of the Microsoft Excel version 4.0 Function Reference, the section on the RANDBETWEEN() function states: 

   If this function is not available, you must install the Analysis
   ToolPak add-in macro.
This should read: 

   If this function is not available, you must install the Add-in
   Functions macro.

MORE INFORMATION

RANDBETWEEN() is one of five worksheet functions added to Excel when you open the Add-in Functions macro from the Library directory.

To open the Add-in Functions macro:

Excel for Windows

  1. From the File menu, choose Open.

  2. Change to the Library directory by double-clicking Library in the directory list.

  3. Select ADDINFNS.XLA from the list of files.

  4. Choose OK.

    -or-

  5. From the Options menu, choose Add-ins.

  6. In the Add-in Manager dialog box, choose the Add button.

  7. Select ADDINFNS.XLA from the list of add-in files in the File Open dialog box.

  8. Choose OK, then choose the Close button in the Add-in Manager dialog box.

Excel for the Macintosh

  1. From the File menu, choose Open.

  2. Open the Macro Library folder by selecting it and choosing Open.

  3. Select Add-in Functions from the document list and choose Open.

    -or-

  4. From the Options menu, choose Add-ins.

  5. In the Add-in Manager dialog box, choose the Add button.

  6. Select Add-in Functions from the list of Add-ins.

  7. Choose Open, then choose Close in the Add-in Manager dialog box.

The five Add-in Functions will now be available for use on your worksheets.
