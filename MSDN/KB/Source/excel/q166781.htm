XL97: Incorrect Printout on HP Color LaserJet PostScript

 Last reviewed: February 12, 1998
Article ID: Q166781
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, when you print a worksheet or chart that contains drawing objects, AutoShapes, cells, or any other objects that contain colored patterns, fills, or textures, the objects are printed in black and white, with no colors.

CAUSE

This problem may occur if the following conditions are true:

  • You are printing to an HP Color LaserJet PostScript printer. 

        -and-
  • The printer driver is earlier than 4.1.

WORKAROUND

To eliminate this problem, download and install a newer printer driver from the Hewlett-Packard Web site.

To obtain a new driver for your printer, use your Web browser to go to the following address: 

   http://hpcc920.external.hp.com/cposupport/indexes1/cljser.html
Then, click the "Software & Drivers" link for your particular HP Color LaserJet printer, scroll down to the appropriate operating system, and click the link for the driver you need to download. Once the setup program for the printer driver is complete, print your worksheet or chart. The problem should no longer occur.

NOTE: Microsoft does not support the downloading and installation of printer drivers from the Hewlett-Packard Web site. If you require assistance downloading and installing drivers, contact Hewlett-Packard technical support. The above Web address is correct as of April 1997. You can also access the main Hewlett-Packard Web site at the following address: 

   http://www.hp.com/

STATUS

The problem described above has been corrected in version 4.1 of the HP Color LaserJet PostScript driver.
