Cannot Size a Legend Manually in Excel

 Last reviewed: November 3, 1994
Article ID: Q72109
The information in this article applies to:
  • Microsoft Excel for Windows versions 3.0, 4.0

SUMMARY

Page 453 of the "User's Guide" for Microsoft Excel for Windows version 3.0 states: 

   You can display the legend in an automatic position, or you can
   position and size the legend manually
This is a documentation error. It is not possible to size the legend manually.

The legend may resize depending on its location in the chart sheet and the size of the font used in the legend. Formatting a legend in a smaller font will decrease its size.

The statement that appears next to a chart legend (page 436 in version 3.0 "User's Guide" and page 435 in version 4.0 "User's Guide 1") is partially correct. The legend can be moved, but cannot be manually resized.

REFERENCES

"User's Guide 1," version 4.0, page 435 "User's Guide," version 3.0 for Windows, pages 436, 453
