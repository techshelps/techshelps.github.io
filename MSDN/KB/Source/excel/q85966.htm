Excel Err Msg: Unexpected Error in Scenario Manager

 Last reviewed: November 22, 1994
Article ID: Q85966
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SYMPTOMS

In Microsoft Excel, if you have worksheet that contains more than 253 scenarios, you will receive the following error message 

   An unexpected error occurred in Scenario Manager. Make sure you
   are not out of memory and your worksheet does not contain errors.
   Error at cell: <cell number>

CAUSE

In Microsoft Excel version 4.0, the maximum number of scenarios that can be summarized using the Summary option in Scenario Manager is 253.

The number of scenarios shown in the Summary is limited by the maximum number of columns a worksheet can have. Excel places each scenario summarization in a separate column beginning in column D. Columns A, B, and C are used by the Scenario Manager to format the Summary. With the first three columns taken, this leaves 253 columns for the summaries.

WORKAROUND

To work around this problem, do not create more that 253 summaries in Scenario Manager.

As long as the Summary feature is not used, the number of scenarios that can be added is limited by available memory. To see the summary for an individual scenario, select the desired scenario in the Scenario Manager dialog box and choose the Show button. The result for that particular scenario will be displayed.

MORE INFORMATION

The Scenario Manager creates and saves different sets of data as separate scenarios. You can use these scenarios to view multiple outcomes based on different assumptions. You can also create a separate summary report that shows the changing cell values and the resulting cell values for each scenario.
