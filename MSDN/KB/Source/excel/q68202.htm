How to Quickly Reenter the Contents of the Cell Above

 Last reviewed: November 3, 1994
Article ID: Q68202
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2 versions 2.2 and 3.0

SUMMARY

The following shortcut keystroke will reenter the data from the cell above into the cell that is selected below it 

   CTRL+APOSTROPHE
where ' is the APOSTROPHE key.

MORE INFORMATION

The following steps will demonstrate this feature.

  1. Enter the text string "Hello" into cell A1.

  2. Select cell A2 and press: 

          CTRL+APOSTROPHE

  3. The Formula Bar will display "Hello" with the edit cursor after the "o"

Note: Pressing CTRL+APOSTROPHE again will concatenate the contents of cell A1 onto the copy in cell A2. It is important to remember that this shortcut will only reproduce the contents of the cell directly above the currently selected cell.
