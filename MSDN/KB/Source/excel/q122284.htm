XL5: Excel FX Fields Not Updated in Notes Document

 Last reviewed: April 1, 1997
Article ID: Q122284

The information in this article applies to:

  • Microsoft Excel for Windows, version 5.0c

SYMPTOMS

When you insert a Microsoft Excel object in a Lotus Notes form for the first time, and you use one of the summary information fields in the form, the information in the field may not be updated in the form.

This problem has only been reported with Microsoft Excel objects. This problem does not occur with Microsoft Word or Microsoft PowerPoint.

WORKAROUND

To correct this problem, do the following: 

   1. Close the Lotus notes form that contains the embedded Microsoft Excel
      object.

   2. Create a new form.

   3. Insert a Word Document object, and add one of the summary information
      fields to the form.
Exchanging field information in the form with a Microsoft Excel object will now work correctly.
