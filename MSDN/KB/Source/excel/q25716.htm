Excel: Transferring a Worksheet or Chart into Aldus PageMaker

 Last reviewed: November 29, 1994
Article ID: Q25716

SUMMARY

Worksheet data or charts from Microsoft Excel can be transferred to Aldus PageMaker only when the information is copied as a graphic. Aldus PageMaker cannot directly import data or charts from Microsoft Excel binary files.

To transfer worksheet or macro sheet data from Excel to PageMaker, do the following:

  1. Select the data.

  2. Hold down the SHIFT key and choose Copy Picture from the Edit menu. A dialog box will appear and you will have the option of selecting either As Shown On Screen or As Shown When Printed.

    NOTE: If you have a version of Excel prior to 2.20, you will not see these options.

To transfer charts from Excel to PageMaker, do the following:

  1. Select the chart by choosing the Select Chart command from the Chart menu or by pressing COMMAND+A.

  2. Hold down the SHIFT key and choose Copy Picture from the Edit menu. At this point, a dialog box will appear and you will see options for Appearance and Size, which include As Shown On Screen or As Shown When Printed. NOTE: If you have a version of Excel prior to 2.20, choose the Copy Chart command from the Edit menu and a dialog box will appear with options for Copy Chart To Clipboard of As Shown On Screen or As Shown When Printed.
