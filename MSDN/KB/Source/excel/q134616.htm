XL7: No Properties Dialog Box When Saving from Custom Template

 Last reviewed: March 27, 1997
Article ID: Q134616
7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you save a new file created from a custom template, you will not be prompted with the Properties dialog box, even though the Prompt For File Properties option is selected on the General tab of the Options dialog box.

CAUSE

When you create a new custom template, you will be prompted with the Properties dialog box. However, any new file that you create using that custom template will use the template's properties and will not prompt you with the Properties dialog box.

WORKAROUND

To edit the properties of a workbook, click the Properties command on the File menu.
