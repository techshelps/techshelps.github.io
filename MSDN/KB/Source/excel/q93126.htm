Excel: Split Not Saved in Workbook

 Last reviewed: November 29, 1994
Article ID: Q93126
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

Summary:

When you are saving a sheet that contains a split in a Microsoft Excel workbook, you must use the Save Workbook command on the File menu. Using the File Save command will save changes to the sheet but not the split itself. If you have modified an existing split since the last time the workbook was saved, the changes will be saved only when the Save Workbook command is used.
