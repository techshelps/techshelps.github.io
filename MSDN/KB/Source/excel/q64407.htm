Excel: Dialog Box Text Overlaps Its Corresponding Check Box

 Last reviewed: November 2, 1994
Article ID: Q64407

SUMMARY

If Microsoft Excel is booted with a System containing the Sigma Design LaserView 19-inch Monitor Control Panel device (CDEV), a situation may occur where dialog boxes will contain text that overlaps its corresponding check boxes.

This situation occurs because of the Expansion Of Fonts option associated with this CDEV. To correct this situation, do the following:

  1. From the Apple menu, choose Control Panel.

  2. Select the Sigma Design LaserView 19-inch Monitor CDEV on the left side.

  3. Locate the Expansion Of Fonts box.

  4. Select Normal (this will display the fonts in the regular size instead of the expansion size).
