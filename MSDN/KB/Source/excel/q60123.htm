Excel: Collapse Columns to Fit Desired Fields on Data Form

 Last reviewed: November 2, 1994
Article ID: Q60123

SUMMARY

The maximum number of database fields that can be viewed in a data form is limited by the size of the Macintosh screen.

To view more fields of the database, collapse any unnecessary fields in the database by setting their column widths to zero before opening the data form.

Another way to view more fields in a data form is to set up a custom data form.

More information on custom data forms can be found on pages 163-166 of the "Microsoft Excel Reference" version 2.20 manual or pages 365-368 of the "Microsoft Excel User's Guide" version 3.00 manual.
