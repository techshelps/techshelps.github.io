XL97: Cannot Change Gridlines on Surface Chart

 Last reviewed: January 7, 1998
Article ID: Q174447
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

If you change the style, color, or weight of the gridlines on a chart, the changes do not appear in the chart, or you may receive the following error message: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.
If you click Details, you receive an error message similar to the following: 

   Excel caused an invalid page fault in module Excel.exe at
   0137:303767e6.

CAUSE

These problems occur when you attempt to change the style, color, or weight of gridlines on a surface chart. If you try to modify the style, color, or weight of the value axis gridlines (click "Value Axis Major Gridlines" on the Chart toolbar), changes do not appear in the chart.

If you try to change any of the following gridlines 

   Category Axis Major Gridlines
   Series Axis Major Gridlines
   Category Axis Minor Gridlines
   Series Axis Minor Gridlines
   Value Axis Minor Gridlines
Microsoft Excel unexpectedly quits, and you receive the error message described in the "Symptoms" section in this article.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about surface charts, click the Index tab in Microsoft Excel Help, type the following text: 

   surface
and then double-click the selected text to go to "Examples of chart types (surface)" topic.
