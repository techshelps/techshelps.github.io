Excel: Transferring Files Containing Dates to Microsoft File

 Last reviewed: November 29, 1994
Article ID: Q23672
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0

SYMPTOMS

In Microsoft Excel, when you import a file into a Microsoft File database, and a date format other than "m/d/yy" has been used, the following error message is received: 

   "Invalid Number or Date values have been left blank"

WORKAROUND

To format dates in Microsoft Excel for transfer into Microsoft File, do the following:

  1. Select the cells containing dates.

  2. From the Format menu, choose Number.

  3. Select "m/d/yy". Click OK.
