Can Use MultiSelect with DrawingObjects, ListBox, ListBoxes

 Last reviewed: September 12, 1996
Article ID: Q114538
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

In Microsoft Excel 5.0 Help, the Visual Basic Reference states the following: 

   Microsoft Excel does not support multi-select list boxes on user-defined
   dialogs or worksheets.
This information is incorrect. You can use the MultiSelect property with DrawingObjects, ListBox, and ListBoxes objects.

MORE INFORMATION

To find this documentation error, do the following:

  1. From the Help menu, choose Contents, and select Programming With Visual Basic.

  2. Choose the Search button, and, in the Search dialog box, type "listbox object" (without the quotation marks).

  3. Choose Show Topics, then choose Go To.

The second sentence under Description contains the error.
