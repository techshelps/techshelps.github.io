XL97: Forward and Back Buttons May Not Work with Hyperlinks

 Last reviewed: February 18, 1998
Article ID: Q155050
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows

SYMPTOMS

When you click a hyperlink in a Microsoft Excel workbook, the Forward and Back buttons on the Web toolbar may not take you forward or backward through your link(s).

CAUSE

You may experience this problem if your Microsoft Excel workbook is embedded in a Microsoft Word document.

RESOLUTION

If you want to be able to use the Forward and Back toolbar buttons with your hyperlinks in an embedded Microsoft Excel document, follow these steps:

  1. If the embedded Microsoft Excel object is activated, click into your Microsoft Word document to deactivate it.

  2. If the embedded Microsoft Excel object is not selected, click once on it to select it.

  3. On the Edit menu, point to "Worksheet Object," and then click Open.

    Your embedded Microsoft Excel document is opened in a Microsoft Excel window and if you have hyperlinks, the Forward and Back toolbar buttons work.

  4. To go back to your Word document, click "Close & Return to <your Word document name>" on the File menu in the Microsoft Excel window.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

If your embedded Microsoft Excel document is a linked embedded object, then when you activate this object, you will be able to use both the Forward and Back toolbar buttons to move through your links.

REFERENCES

For more information about hyperlinks, click the Index tab in Microsoft Excel Help, type the following text 

   hyperlinks
and then double-click the selected text to go to the "create a hyperlink (Microsoft Excel)" topic.
