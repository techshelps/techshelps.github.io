XL98: Now Contact and Now Up-to-Date Menus Don't Work

 Last reviewed: February 26, 1998
Article ID: Q177109
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel 98 Macintosh Edition, when you click the menus for Now Contact 3.6.5 or Now Up-to-Date 3.6.2, the menus do not work; nothing happens when you click them.

NOTE: This problem does not occur in Microsoft Excel for the Macintosh, version 5.0, or in Microsoft Word 98 Macintosh Edition.

CAUSE

This problem occurs because the method Microsoft Excel 98 Macintosh Edition uses to handle menus is different from the method used by Now Contact 3.6.5 and Now Up-to-Date 3.6.2. Because of this conflict, you cannot use the Now Contact and Now Up-to-Date menus when Microsoft Excel 98 Macintosh Edition is the active program.

WORKAROUND

To work around this behavior, do either of the following:

  • Hide Microsoft Excel 98 Macintosh Edition by clicking Hide Microsoft Excel on the Finder menu. After Microsoft Excel is hidden, you can use the Now Contact and Now Up-to-Date menu commands.

    When you are done, unhide Microsoft Excel 98 Macintosh Edition by clicking Microsoft Excel on the Finder menu.

    -or-

  • Quit Microsoft Excel 98 Macintosh Edition by clicking Quit on the File menu. After you quit Microsoft Excel, the menus work correctly.

You may also want to contact Now Software, Inc., to see if a newer version of Now Contact or Now Up-to-Date corrects this behavior. To view the Now Software Web site, use a Web browser to go to the following Web address:

STATUS

This problem occurs when you use Microsoft Excel 98 Macintosh Edition and Now Contact version 3.6.5 or Now Up-to-Date version 3.6.2.

MORE INFORMATION

Now Contact and Now Up-to-Date are manufactured by Now Software, Inc., a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.

When you install Now Contact and Now Up-to-Date on your computer, two new menus appear on the menu bar next to the Finder menu in the upper-right corner of the screen: a telephone menu (for Now Contact), and a clock menu (for Now Up-to-Date).

If you click the telephone menu, a list of contacts and options for finding or creating contacts appears. If you click the clock menu, a list of tasks to perform appears.

However, if you click either of these menus when Microsoft Excel 98 Macintosh Edition is the active program, nothing happens.
