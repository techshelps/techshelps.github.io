Cannot Select Drawing or Charting Tools from Toolbar in Excel

 Last reviewed: November 30, 1994
Article ID: Q73254
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2 version 3.0
  • Microsoft Excel for the Macintosh versions 3.0, 4.0

SUMMARY

You will be unable to select any of the drawing or charting tools from the tool bar if the Hide All option is selected in the Options Display dialog box. Attempting to select a tool causes a warning beep to sound.

MORE INFORMATION

The Hide All option is used when you do not want to see any graphic objects on the screen. This option can speed screen updating so that you can move around your spreadsheet more easily. When this option is selected you will not be able to create graphic objects on your sheet (such as circles, graphs, and text boxes).

To restore functionality to these buttons, choose Display from the Options menu. In the Object section of the dialog box that is displayed, choose Show All. Your drawing and charting tools will now work as expected.
