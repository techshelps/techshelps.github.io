XL5: DrawingObjects Shadow Property Listed Twice in Help File

 Last reviewed: September 12, 1996
Article ID: Q117598
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

Microsoft Excel version 5.0 online help lists the Shadow property twice for the Object DrawingObjects. Both Shadow topics refer to the same Shadow Property help topic.

To find the topics for the Shadow property, do the following:

  1. On a module sheet, type the word "DrawingObjects" (without the quotation marks), and select the word.

  2. Press F1 and select Excel: DrawingObjects(object) from the Help dialog box.

  3. From the DrawingObjects Object topic, select the Properties jump topic. The DrawingObjects Object Properties topic will be displayed.

In the list, you will see the Shadow property listed twice. Both Shadow topics refer to the same Shadow Property help topic.

STATUS

This documentation error does not occur in later versions of Microsoft Excel.
