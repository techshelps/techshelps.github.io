3-D Tool Bar Not Available on EGA Graphics Adapters

 Last reviewed: November 3, 1994
Article ID: Q69149
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0

Summary:

If the graphics adapter used with Microsoft Excel version 3.00 is less than VGA resolution, the tool bar icons will not be in 3-D when Excel is executed. For example, an EGA adapter will not display 3-D tool bar icons.

However, the line "NewLook=1" (without the quotation marks) can be added to the Microsoft Excel section of the EXCEL.INI file in order to show the status bar in 3-D.

More Information:

For more information on the "NewLook=1" line, query on NewLook \\xref
