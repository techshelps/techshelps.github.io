XL3/XL4: Cannot Open or View Password Protected Add-in Macro

 Last reviewed: February 2, 1998
Article ID: Q103161

The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0

SYMPTOMS

When you protect an add-in macro by choosing Save As from the File menu and choosing the Options button, it may seem that the add-in macro cannot be edited again.

WORKAROUND

When you open an add-in macro to edit it, press and hold down the SHIFT key while you open the file and hold it down again when you exit the password request dialog box.

STATUS

This behavior is by design of the versions of Microsoft Excel listed above.

NOTE: This information does not apply to later versions of Microsoft Excel.

REFERENCES

"User's Guide 1," pages 88-90
