Excel: Multiline List Boxes Display Text Incorrectly

 Last reviewed: November 3, 1994
Article ID: Q77152

SUMMARY

Microsoft Excel version 3.0 allows you to create multiline list boxes within a custom dialog box. Scrollbars are added to the right-hand side of these boxes (if the height of the box is greater than 18) to allow you to scroll through additional lines of text; however, these scrollbars may cover a portion of your text, making it unreadable.

MORE INFORMATION

When the DIALOG.BOX command is executed in your macro (to display your custom dialog box), a list box will display any text that is in the Init/Result column. The list box can be made to be more than one line high if you increase the height of the box to a number larger than 18. If this is done, any text that would extend beyond the right-hand side of the list box will now wrap. In addition, a scrollbar is added to the multiline list box, but may cover part of the text in the box.

Microsoft is researching this problem and will post new information here as it becomes available.

REFERENCES

"Microsoft Excel User's Guide," version 3.0, page 635
