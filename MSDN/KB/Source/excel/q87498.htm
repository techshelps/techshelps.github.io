Excel: USER= Line in EXCEL4.INI Doesn't Affect Help About Dial

 Last reviewed: November 16, 1994
Article ID: Q87498
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0, 5.0

SUMMARY

The USER= line in the EXCEL4.INI or EXCEL5.INI allows a name to be specified for use when sharing documents on a network. This line does not affect the name displayed in the About Microsoft Excel dialog box on the the Help menu. The information for the About Microsoft Excel dialog box is entered when Microsoft Excel is first installed.

MORE INFORMATION

When installing Microsoft Excel for Windows the first time, you are prompted for your name and your company name. When this information is entered, it is written to both the EXCEL.EXE and the EXCEL.INF on the setup diskettes. Any further installations performed with these disks will have the same name and company name. Changing the USER= line in the EXCEL4.INI and EXCEL5.INI does not affect this.

The USER= name is displayed if you are using a shared document and someone else on the network tries to open it. An alert message is displayed that the document is being modified by the user name specified in the file.

In version 5.0, to change the user name, chose Options from the Tools menu, then select the General tab and enter the desired name in the User Name input box.

REFERENCES

EXCELINI.TXT, Microsoft Excel 4.0 NETWORK.TXT, Microsoft Excel 4.0
