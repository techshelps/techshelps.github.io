XL7: Msjeterr.hlp File Is Not Shipped with Microsoft Excel 7.0

 Last reviewed: March 27, 1997
Article ID: Q135498
7.00 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

When you use Data Access Objects (DAO) that are included with the products listed at the beginning of this article, you may receive the following error message when you attempt to use Help: 

   Cannot find the Msjeterr.hlp file. Do you want to try to find this file
   yourself?

CAUSE

This behavior occurs because the Msjeterr.hlp file is not included with the products listed at the beginning of this article.

WORKAROUND

The Msjeterr.hlp file is available in application note WE1238 "Jet Error Help File (Msjeterr.hlp)." You can obtain WE1238 from the following sources:

You can obtain this Application Note from the following sources:

  • Microsoft's World Wide Web Site on the Internet
  • The Internet (Microsoft anonymous ftp server)
  • The Microsoft Network (MSN)
  • Microsoft Download Service (MSDL)
  • Microsoft Product Support Services

For more information about obtaining the Jet Error Help file, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119849
   TITLE     : AppNote: "Jet Error Help File (Msjeterr.hlp)" (WE1238)

MORE INFORMATION

The Msjeterr.hlp file includes information specific to error messages you receive when you use Data Access Objects. In order to conserve disk space, this Help file was not included with the products listed at the beginning of this article.
