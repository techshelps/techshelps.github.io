Excel 2.20: Minimum System Requirements (System 6.0.2)

 Last reviewed: October 31, 1994
Article ID: Q48182

SUMMARY

Microsoft Excel version 2.20 requires System version 6.0.2 or later.

Page 385 of the "Microsoft Excel Reference" version 2.2 manual lists the minimum System requirements for various Macintosh computers. Please note that these are the minimum System configurations recommended by Apple for each computer; Excel 2.20 will not run if the version of the System is earlier than 6.0.2.

The minimum System configuration is listed on the outside of the Excel 2.20 retail package.

To obtain the correct version of the System for your Macintosh, see your Apple dealer.
