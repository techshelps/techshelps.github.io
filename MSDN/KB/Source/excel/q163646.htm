XL97: Some On Events Are Not Displayed in Object Browser

 Last reviewed: February 12, 1998
Article ID: Q163646
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SUMMARY

The following properties and methods of the Workbook and Worksheet classes are not displayed in the Object Browser in Microsoft Excel 97 for Windows: 

   OnAction Property
   OnTime Method
   OnUndo Method
   OnRepeat Method

MORE INFORMATION

You can use the four properties and methods listed in this article in Microsoft Excel 97 even though they do not appear in the Object Browser. Note that these properties and methods have not changed from earlier versions of Microsoft Excel.

REFERENCES

For more information about these properties and methods, click the Index tab in Visual Basic for Applications Help, type one of the following words 

   OnAction
   OnTime
   OnUndo
   OnRepeat
and then double-click the selected text to go to the appropriate help topic.
