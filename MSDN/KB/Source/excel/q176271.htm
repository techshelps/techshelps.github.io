XL98: Page Setup Dialog Box Is Redrawn Slowly

 Last reviewed: February 2, 1998
Article ID: Q176271
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you switch from the Margins tab to another tab in the Page Setup dialog box, several seconds may pass before the dialog box is redrawn.

CAUSE

This problem may occur when you do the following:

  1. You first click Page Setup on the File menu, click the Page tab, and then change the Orientation setting.

  2. You click the Margins tab.

  3. You click the Page, Header/Footer, or Sheet tab.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about using Page Setup, click the Index tab in Microsoft Excel Help, type the following text 

   printing,page setup
and then double-click the selected text to go to the "Change the layout of the printed worksheet" topic.
