Cannot Change Alignment with Formulas Displayed

 Last reviewed: November 3, 1994
Article ID: Q69776
The information in this article applies to:
  • Microsoft Excel for Windows versions 3.0, 4.0, 5.0

SUMMARY

A selected cell or range of cells will not display alignment changes when an alignment button is clicked on the tool bar if formulas are displayed. The button appears depressed, but the alignment does not change. Both numbers and text will display as left aligned. Changes in number format produce similar results.

MORE INFORMATION

Version 5.0

To display alignment chnages, choose Options from the Tools menu, then select the View tab. Select Formulas in the Window Options box. To display values again, deselect Formulas in the Window Options box.

Version 4.0

From the Options menu, choose Display and cancel the selection of Formulas in the Cells box. The selected cell(s) should then align according to the alignment shown on the tool bar. To change back to values, you can also press CTRL+' (apostrophe) to toggle between displaying formulas and values.
