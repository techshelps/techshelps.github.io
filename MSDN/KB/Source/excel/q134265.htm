XL: Cannot Open Quattro Pro for Windows File

 Last reviewed: January 9, 1997
Article ID: Q134265
The information in this article applies to:
  • Microsoft Excel for Windows 95, versions 7.0
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel, when you attempt to open a Quattro Pro for Windows file with a protection level of high, you may receive the following error message: 

   Cannot Open Quattro Pro for Windows file

WORKAROUND

To work around this problem, use the following procedure:

  1. Open the file in the version of Quattro Pro for Windows in which it was created.

  2. Position the mouse pointer on top of the document title bar and click the right mouse button.

  3. Choose the Password Level option and set the level to None.

  4. Save the file and reopen it in Microsoft Excel.

You should now be able to open the file without error.

STATUS

This problem occurs when you use Microsoft Excel for Windows 95 version 7.0 or Microsoft Excel 97, and Quattro Pro for Windows.
