XL98: "Reference Is Not Valid" Error When You Click a Hyperlink

 Last reviewed: February 25, 1998
Article ID: Q181711
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you click a hyperlink in Microsoft Excel 98 Macintosh Edition, you may receive the following error message: 

   Reference is not valid.

CAUSE

This error message appears if you click a hyperlink that is linked to any of the following objects in a Microsoft Excel workbook:

  • A drawing object or AutoShape
  • A chart sheet
  • An embedded chart
  • An embedded object, such as an embedded Microsoft Word document
  • A Visual Basic for Applications module, class module, or UserForm
  • A defined name that refers to any of the previously listed objects

MORE INFORMATION

In Microsoft Excel 98 Macintosh Edition, when you insert a hyperlink to a workbook, you can link only to individual cells, ranges of cells, and defined names that refer to individual cells or ranges in worksheets and Microsoft Excel 4.0 macro sheets.

This behavior is by design of Microsoft Excel 98 Macintosh Edition.
