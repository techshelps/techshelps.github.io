Excel Err Msg: 'Not Enough Memory to Print'

 Last reviewed: November 4, 1994
Article ID: Q89393
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0 and 4.0

Summary:

When printing complicated Microsoft Excel spreadsheets or charts with System 7.0x, the Macintosh PrintMonitor may return the error message: 

   Not enough memory to print from Microsoft Excel. Quit one or
   more applications and try again.
In these cases, more memory should be allocated to the PrintMonitor if there are no other applications running.

More Information:

The PrintMonitor is a small print spooling application that is installed in the Extensions folder within the System folder.

With the recent introduction of Apple printers (such as the Personal LaserWriter LS, or the StyleWriter) which have less internal memory and utilize slower non-PostScript printing routines, the memory allocated to the print monitor may not be large enough to accommodate the information which must be held by the spooler while the printer processes data and clears its internal memory.

To allocate more memory to the PrintMonitor:

  1. Click the PrintMonitor icon a single time to select it.

  2. From the File menu, choose Get Info (or press COMMAND+I).

  3. In the lower-right corner of the Info window, type a larger figure in the Current Size field (the default value is 120).

Note: The Current Size field is not accessible with the PrintMonitor running in the background. Exiting the PrintMonitor occurs when there are no print jobs spooled and the application is sent to the background.

To exit the PrintMonitor:

  1. Make the PrintMonitor the active application.

  2. Insure that no documents are currently spooled.

  3. If Resume Printing is listed on the File menu, select that command to change the setting to Stop Printing.

  4. Pull down the PrintMonitor icon from the upper-right of the monitor and select the Finder.
