Creating a Template to Show in the File New Dialog in Excel

 Last reviewed: November 3, 1994
Article ID: Q68516
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 5.0

SUMMARY

Creating a Microsoft Excel worksheet or chart template will allow you to contain all the standard formatting, layouts, formulas, and text you use in a worksheet or chart document on a regular basis. Adding this template to the XLSTART subdirectory of EXCEL will enable you to choose it from the File New dialog box.

MORE INFORMATION

Use the following procedure to create a worksheet or a chart template to display in the File New dialog box:

  1. Create or open the worksheet or chart you want to use as a template.

  2. From the File menu, choose Save As.

  3. Change directories to the XLSTART subdirectory in the directory that contains the EXCEL.EXE file.

  4. Type a name for the chart or worksheet template.

  5. From the Save File As Type list, select Template.

    In Microsoft Excel version 3.0, choose the Options button. From the File Format list, select Template, and choose OK.

  6. Choose OK.

  7. In Microsoft Excel version 3.0, choose Exit from the File menu, and then start Microsoft Excel again. (not necessary in versions 4.0 and 5.0)

The worksheet or chart template that you created now appears in the File New dialog box.
