XL: Microsoft Excel KB Help Files Available

 Last reviewed: March 27, 1997
Article ID: Q141672

SUMMARY

This article lists the file names and Snumbers for Microsoft Excel KB help files that are available from online services. For information about how to obtain these files, see the "Instructions for Downloading" section of this article.

MORE INFORMATION

The Knowledge Base group extracts current articles from the Microsoft Excel Knowledge Base and creates a Windows Help file that is posted to the Microsoft Software Library. These files are updated as new information becomes available.
