SBFM: Updated Accounting Program Drivers Available

 Last reviewed: June 4, 1997
Article ID: Q156663
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Small Business Financial Manager for Microsoft Excel, version 1.0

SUMMARY

Several of the drivers for the Microsoft Small Business Financial Manager for Microsoft Excel (SBFM) have been updated. This article lists the updated drivers, what is corrected in each updated driver, and the contents of the Updated Drivers for the Microsoft Small Business Financial Manager for Windows 95 Readme.txt.

The updated drivers are available on Microsoft's World Wide Web site on the internet, Microsoft's ftp site, the Microsoft Network, and the Microsoft Download Service.

NOTE: The drivers are not new additional drivers for the SBFM. The files are updates to some of the drivers that shipped in the SBFM or in the Additional Drivers disk.

MORE INFORMATION

The Updated Drivers

The following table lists the updated SBFM drivers, the dates of the updated drivers, and whether the drivers shipped in the SBFM or the Additional Drivers disk: 

   Driver           Date        Source of original driver
   -----------------------------------------------------

   Aapacc.dll       8/28/96     SBFM
   Aapbww.dll       8/20/96     Additional Drivers
   Aapgreat.dll     8/20/96     SBFM
   Aapmas90.dll     8/20/96     Additional Drivers
   Aapqbw.dll       8/20/96     Additional Drivers
   Qbdrivhk.dll     8/20/96     Additional Drivers
The following sections describe what is fixed in each of the drivers listed in the above table.

Aapacc.dll - Computer Associates ACCPAC Plus for MS-DOS, version 6.1

Department information was not being read properly. This problem has been corrected.

Aapbww.dll - Businessworks for MS-DOS and Windows, version 9.0

Trailing white space on character strings was causing Profitability reports in Microsoft Excel to crash. The fix assures that any trailing white space is removed and all strings are NULL terminated.

Aapgreat.dll - Great Plains Accounting for MS-DOS, version 8.0

The company name was not being recognized. The search is now refined to get the record that contains the company information.

Aapmas90.dll - M A S 90 Evolution/2 for MS-DOS, version 1.5

The system file that contains MAS90 company information was not being processed correctly. The company file is now parsed to retrieve correct company names and to display them.

Aapqbw.dll, Qbdrivhk.dll - Quickbooks for Windows version 3.1

There was a problem working with the Accounts Receivable/Accounts Payable/Inventory analysis that is now fixed.

INSTALLATION INSTRUCTIONS FOR UPDATED FINANCIAL MANAGER DRIVERS

Note: To use these drivers, Microsoft Excel for Windows 95 and Microsoft Small Business Financial Manager must be installed on your computer.

  1. Turn on your computer and start Microsoft Windows 95.

  2. Quit Microsoft Excel for Windows 95, if it is running.

  3. If you previously installed a driver for which there is an update, delete the original driver from the Aap folder in the Microsoft Excel folder (usually the C:\Msoffice\Excel\Aap folder). The following table lists the names of the original Small Business Financial Manager drivers and their file dates. 

       Driver           Date
   ---------------------

   Aapacc.dll       5/1/1996
   Aapbww.dll       6/27/1996
   Aapgreat.dll     5/1/1996
   Aapmas90.dll     6/27/1996
   Aapqbw.dll       6/27/1996
   Qbdrivhk.dll     6/5/1996

  4. Use the appropriate procedure for the driver you want to install.

Businessworks for MS-DOS and Windows, version 9.0:

In Windows Explorer, copy the Aapbww.dll file to the Aap folder on your computer's hard disk (usually the C:\Msoffice\Excel\Aap folder).

Computer Associates ACCPAC Plus for MS-DOS, version 6.1:

In Windows Explorer, copy the Aapacc.dll file to the Aap folder on your computer's hard disk (usually the C:\Msoffice\Excel\Aap folder).

Great Plains Accounting for MS-DOS, version 8.0:

In Windows Explorer, copy the Aapgreat.dll file to the Aap folder on your computer's hard disk (usually the C:\Msoffice\Excel\Aap folder).

M A S 90 Evolution/2 for MS-DOS, version 1.5:

In Windows Explorer, copy the Aapmas90.dll file to the Aap folder on your computer's hard disk (usually the C:\Msoffice\Excel\Aap folder).

Quickbooks for Windows version 3.1:

In Windows Explorer, copy the Aapqbw.dll and Qbdrivhk.dll files from to the Aap folder on your computer's hard disk (usually the C:\Msoffice\Excel\Aap folder).

ISSUES YOU MAY ENCOUNTER WHEN YOU IMPORT DATA

Businessworks for MS-DOS and Windows, version 9.0:

If you perform a prior period adjustment into a closed period for which you are no longer saving detailed transactions, you will need to re-import your accounting data instead of updating your accounting data.

Quickbooks for Windows, version 3.1:

A. You must have Quickbooks installed on your computer, and the company 
   .QBW file to import must be in the same folder as the Quickbooks
   program files.
B. While importing data, Financial Manager will need to open Quickbooks 
   and perform certain actions in Quickbooks. You should not touch your
   keyboard or computer during this time when Quickbooks is open.
C. In a What-If Scenario profitability analysis, the detail information 
   for any sales items entered into QuickBooks without a product code are
   placed into the N/A item in Financial Manager. In QuickBooks, the
   sales reports by customer only include those items with a product code.
   Therefore, the total sales value in the QuickBooks sales report by
   customer will not match the total sales value in the Financial
   Manager profitability analysis worksheet.
D. In QuickBooks, the summary sales by item report includes sales tax 
   information. In Financial Manager, the profitability analysis
   worksheet does not include sales tax information. Therefore, the total
   sales revenue value in the QuickBooks summary sales by item report will
   not match the total sales value in the Financial Manager profitability
   analysis worksheet.
E. The QuickBooks summary sales by item report does not include 
   inventory adjustments, but all Financial Manager reports and analysis
   worksheets do. Therefore, if you have made inventory adjustments, the
   QuickBooks summary sales by item report will not match all of the
   Financial Manager reports and analysis worksheets. Since the QuickBooks
   profit and loss statement does include inventory adjustments, those
   values will match the Financial Manager values.

SUPPORT

Assistance with these drivers is provided to you through your Microsoft Small Business Financial Manager warranty support policy. Installing these drivers will not restart this warranty period. If your Microsoft Small Business Financial Manager warranty support period has expired, paid phone support as well as alternative options are available.

If your license to the Microsoft Small Business Financial Manger product was obtained from a hardware manufacturer, then technical support for these drivers is not available from Microsoft. Please contact your hardware manufacturer before installing these drivers to determine your support options.
