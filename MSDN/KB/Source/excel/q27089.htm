XL: Calculating the Number of Days Between Two Dates

 Last reviewed: February 2, 1998
Article ID: Q27089

The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0, 5.0c
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0, 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition

In Microsoft Excel, to find the number of days between two dates, you can use the following formula 

   =Date2-Date1
where "Date2" represents the later of the two dates and "Date1" represents the earlier of the two dates.

Format the cell that contains this formula as "General" so the cell displays the difference between the two dates in number of days.
