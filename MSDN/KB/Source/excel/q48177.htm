Excel 2.2: Documents Saved with MacSQZ! Won't Open

 Last reviewed: October 31, 1994
Article ID: Q48177
The information in this article applies to:
  • Excel for the Macintosh, version 2.2

SUMMARY

Documents that were saved in version 1.50 or earlier of Excel using MacSQZ! cannot be directly opened into Excel version 2.20.

To open a document that was saved in this way, open it into an earlier version of Excel and save it in the normal format. The documents then can be read by Excel 2.20.

Excel 2.20 documents cannot be saved using MacSQZ! version 1.50 because Excel 2.20 saves documents in a new format that is not compatible with the format MacSQZ! 1.50 uses.

This problem will need to be addressed by Symantec Utilities.
