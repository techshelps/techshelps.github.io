XL4: Low Option in AutoSave Add-in Saves Every 27 Minutes

 Last reviewed: February 2, 1998
Article ID: Q97567

The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a
  • Microsoft Excel for the Macintosh, version 4.0

SYMPTOMS

In Microsoft Excel version 4.0 (and version 4.0a for Windows), the Help topic for the AutoSave add-in macro incorrectly states that the frequency of the Low option is 30 minutes.

MORE INFORMATION

The following table lists the actual frequency of save intervals in the AutoSave add-in macro: 

   Option   Frequency
   -------------------

   High      3 minutes
   Medium    9 minutes
   Low      27 minutes
NOTE: In Microsoft Excel versions 5.0 and later, the AutoSave add-in macro allows you to specify the exact number of minutes between automatic saves.

The Auto Save add-in macro automatically saves worksheets at the intervals you select. It is installed as part of the Macro Library. For more information, see "Macro Library Overview" and "Auto Save" in Help.

REFERENCES

"Microsoft Excel Help," versions 4.0 and 4.0a
