XL7: Problems Printing Data Map to HP DeskJet 1600c

 Last reviewed: March 27, 1997
Article ID: Q151102
7.00 WINDOWS kbprint kbprb kberrmsg

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft DataMap for Windows 95, version 1.0

SYMPTOMS

In Microsoft Excel 7.0, when you try to print or print preview a sheet with an embedded DataMap using the Hewlett-Packard DeskJet 1600c ColorSmart Printer driver, Version 4.10, the following may occur: 

   Excel may stop responding to the system.

   -or-

   You may receive the error message:

   EXCEL caused an invalid page fault in module EXCEL.EXE at <location>

   You may also receive the error:

   DATAMAP caused an invalid page fault in module DATAMAP.EXE at <location>
The only option at this point is to exit Microsoft Excel.

WORKAROUND

Install and use the HP 1200c printer driver included with Windows 95.

STATUS

Microsoft has confirmed this to be a problem with the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The HP DeskJet 1600c ColorSmart Printer driver is manufactured by Hewlett- Packard, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

REFERENCES

For more information about installing printers in Windows 95, click the Start Button, click Help, click the Index tab and type: 

   Installing, Printers
