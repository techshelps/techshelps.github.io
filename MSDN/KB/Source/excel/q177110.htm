XL: Hyperlink Not Restored When You Click Undo Hyperlink

 Last reviewed: February 5, 1998
Article ID: Q177110
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In the versions of Microsoft Excel listed above, if you click Undo Hyperlink on the Edit menu after removing a hyperlink from a cell in a worksheet, the removed hyperlink is not correctly restored. Although the cell appears to contain a hyperlink, the hyperlink does not work when you click it.

CAUSE

This problem occurs if you perform the following steps:

  1. In Microsoft Excel, select a cell that contains a hyperlink. NOTE: You cannot select the cell by clicking it; you must use the keyboard or the Go To dialog box (on the Edit menu, click Go To) to select the cell.

  2. On the Insert menu, click Hyperlink.

  3. Click Remove Link.

    The hyperlink is removed from the cell.

  4. On the Edit menu, click Undo Hyperlink.

The cell now appears to contain a hyperlink. However, if you click the cell, the hyperlink does not work.

RESOLUTION

If this problem occurs, you must manually recreate the hyperlink by selecting the cell and clicking Hyperlink on the Insert menu. Enter the file name or URL in the edit box, or select the appropriate file name or URL in the list box. Then, click OK.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
