Q+E Reads One Field in Text File

 Last reviewed: November 3, 1994
Article ID: Q78660
Summary:

Q+E recognizes only one field in a text file if the file contains no headings and the first line is blank.

More Information:

Workaround 1

  1. Add field headings to the original text file.

  2. When opening the file in Q+E, choose Open from the File menu, choose the Options button, and enable First Line Contains Field Names.

Workaround 2

Delete the blank line at the beginning of the text file.

Steps to Reproduce Problem

  1. Create the following text file, leaving the first line blank: 

          red,white,blue
      orange,purple,green

    These should be entered in separate columns.

  2. Save and close the file.

  3. Open the file in Q+E as a text file. From the File menu, choose Open and choose the Options button. The First Line Contains Field Names check box should be disabled.

Only the first field in each record is recognized by Q+E.

Reference(s):

"Q+E for Microsoft Excel User's Guide," version 3.0, pages 135-140
