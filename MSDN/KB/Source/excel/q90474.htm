Excel: PRECISION() Function Documentation Error

 Last reviewed: November 29, 1994
Article ID: Q90474
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

Page 330 of the "Microsoft Excel Function Reference" for version 4.0 contains a documentation error in the description of the PRECISION() function. The first sentence describing the PRECISION() function should read: 

   Equivalent to selecting or clearing the Precision As Displayed
   check box in the Calculation Sheet Options Dialog box, which appears
   when you choose the Calculation command from the Options Menu.

REFERENCES

"Microsoft Excel Function Reference," version 4.0, page 330
