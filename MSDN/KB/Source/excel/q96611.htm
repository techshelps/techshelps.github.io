XL4 Err Msg: "No Match in Current Cell" When Match Does Exist

 Last reviewed: October 22, 1997
Article ID: Q96611
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a

SYMPTOMS

When the Alternate Navigation Keys check box is selected, and you attempt to replace data by choosing Replace from the Formula menu, you may receive the following error messages: 

   No match in current cell.

   -and-

   Could not find matching data to replace.
You receive these error messages even when the data DOES contain a match.

CAUSE

This problem occurs if you select the Whole option under Look At.

WORKAROUNDS

To work around this problem, use any of the following methods.

Method 1

From the Options menu, choose Workspace and clear the Alternate Navigation Keys check box.

Method 2

From the Formula menu, choose Replace, and select the Part option (under Look At).

Method 3

Insert the "?" wildcard character before the characters in the Find What dialog box. This will enable the Replace command while Alternate Navigation Key is on. Note, however, that the Replace All command will not work with this method.
