XL5: Dialog Box Frame and Group Box Ignore Locked Text

 Last reviewed: September 12, 1996
Article ID: Q111324
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.00c
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a

SYMPTOMS

In Microsoft Excel for Windows version 5.0, when you lock text on a dialog box frame or group box, the text in the dialog frame or group box can still be changed.

MORE INFORMATION

To protect the dialog frame or controls, do the following:

  1. Select dialog frame or control.

  2. From Format menu, choose Object, and select the Protection tab.

  3. On the Protection tab, select the Locked check box.

When you are protecting group box controls or protecting the frame, if the Locked Text option is the only one selected (the Locked check box is cleared), the text in the dialog frame or group box can still be changed.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel version 5.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
