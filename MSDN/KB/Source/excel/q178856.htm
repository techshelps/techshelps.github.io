XL98: Clicking Office Assistant Button Does Not Start Assistant

 Last reviewed: February 2, 1998
Article ID: Q178856
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you click the Office Assistant button while using a wizard in Microsoft Excel, you experience one of the following behaviors:

  • The Office Assistant does not appear, and no help is offered for the feature that you are using. 

        -or-
  • You receive the following error message:

    Cannot start the Office Assistant because QuickTime is not installed or is not currently enabled. Please install or enable QuickTime and then restart your Macintosh if you want to use the Office Assistant.

CAUSE

This problem may occur if either of the following extensions have been disabled or are not installed on your computer: 

   QuickTime
   QuickTime PowerPlug

STATUS

This is by design of Microsoft Excel 98 Macintosh Edition.

MORE INFORMATION

This problem may occur when using the following wizards: 

   Chart Wizard
   Conditional Sum Wizard
   File Conversion Wizard
   Internet Assistant Wizard
   Lookup Wizard
   PivotTable Wizard

REFERENCES

For more information about using the Chart Wizard, click the Index tab in Microsoft Excel Help, type the following text 

  charts, creating
and then double-click the selected text to go to the "Create a chart" topic.

For more information about using the Conditional Sum Wizard, click the Index tab in Microsoft Excel Help, type the following text 

  Conditional Sum Wizard
and then double-click the selected text to go to the "Summarize values in a list that meet specific conditions by using the Conditional Sum Wizard" topic.

For more information about using the Lookup Wizard, click the Index tab in Microsoft Excel Help, type the following text 

  Lookup Wizard
and then double-click the selected text to go to the "Find data in a list" topic.

For more information about using the PivotTable Wizard, click the Index tab in Microsoft Excel Help, type the following text 

  PivotTables, creating
and then double-click the selected text to go to the "Create a PivotTable" topic.
