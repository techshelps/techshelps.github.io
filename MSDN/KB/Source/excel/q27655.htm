MXL: File and Creator Types for All Versions

 Last reviewed: February 2, 1998
Article ID: Q27655

The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0, 5.0
  • Microsoft Excel 98 Macintosh Edition

The file types and creators for Microsoft Excel are listed in the following tables. 

   Application           File type         Creator
   -----------------------------------------------

   Microsoft Excel       APPL              XCEL
The file types and creators for Microsoft Excel documents are listed in the following table. 

   Saved as              File type         Creator
   -----------------------------------------------

   Add-In Macro          XLA               XCEL
   Add-In Macro (5.00)   XLA5              XCEL
   Add-In Macro (8.00)   XLA8              XCEL
   Chart (3.00)          XLC3              XCEL
   Chart (2.20)          XLC               XCEL
   Chart Template        sLC3              XCEL
   CSV                   TEXT              XCEL
   DBFx                  TEXT              XCEL
   DIF                   TEXT              XCEL
   International Macro   XLM3              XCEL
   Macro (4.00)          XLM4              XCEL
   Macro (3.00)          XLM3              XCEL
   Macro (2.20)          XLM               XCEL
   Macro Template        sLM3              XCEL
   Normal (8.00)         XLS8              XCEL
   Normal (5.00)         XLS5              XCEL
   Normal (4.00)         XLS4              XCEL
   Normal (3.00)         XLS3              XCEL
   Normal (2.20)         XLS               XCEL
   Normal (1.x)          XLBN              XCEL
   SYLK                  TEXT              XCEL
   Worksheet Template    sLS3              XCEL
   WKS                   TEXT              XCEL
   WKx                   TEXT              XCEL
   Text                  TEXT              XCEL
Note: File types are four characters long and case sensitive. Those listed as three digits actually have a trailing space at the end. (i.e. "XLA" would be "XLA ")
