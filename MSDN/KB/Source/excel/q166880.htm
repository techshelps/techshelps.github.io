XL97: Most Recently Used List Is Not Updated

 Last reviewed: February 12, 1998
Article ID: Q166880
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

After you open a file in Microsoft Excel, the most recently used list on the File menu may not be updated.

CAUSE

This problem may occur if any of the following conditions are true:

  • You open the file from Windows Explorer. 

        -or-
  • You open the file from File Manager.

    -or-

  • You open the file from within a mail message.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
