XL: Error Printing to HP1200C/PS in Windows 95

 Last reviewed: April 1, 1997
Article ID: Q139722

The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a

SYMPTOMS

When you try to print from Microsoft Excel running Windows 95 to a HP1200C/PS printer, you may receive the following error message: 

   Error printing to lpt1

CAUSE

This error occurs because Windows 95 uses the printer driver that shipped with the printer; that printer driver was created for Windows 3.1.

RESOLUTION

To resolve the issue, follow these steps:

  1. Click the Start button, point to Settings, and then click Printers.

  2. Double-click the Add Printers icon.

  3. In the Add Printers Wizard, click Next to choose whether the printer will be a local or network printer. Click Next.

  4. Click HP in the Manufacturers list.

  5. Click HP DeskJet 1200C/PS in the Printers list.

  6. The Add Printer Wizard then asks for the appropriate Windows 95 disk or Compact Disc. Follow the Add Printer Wizard instructions to complete the installation.
