Excel: File Saved in WK3 Displays Numbers Incorrectly in 1-2-3

 Last reviewed: September 13, 1996
Article ID: Q96379
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0, 5.0c
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you open a Microsoft Excel file that has been saved in a WK3 file format in Lotus 1-2-3 version 3.0 or later, the formula bar will display all numbers with two or more digits to the right of the decimal point out to sixteen decimal places.

WORKAROUND

To display the number correctly, save the file in WK1 file format and reopen it in Lotus 1-2-3.

STATUS

Microsoft is researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.
