Excel: Attaching Text to an Entire Chart Series

 Last reviewed: November 2, 1994
Article ID: Q50970

SUMMARY

Text cannot be attached to an entire series at once using the Attach Text option on the Chart menu in Microsoft Excel. Text can be added to only one point at a time using the Attach Text command.

To add text to an entire series, do the following:

  1. With the chart active, choose Column from the Gallery menu.

  2. Select the seventh column chart, which has text attached to each of the columns. Click OK.

  3. In Excel version 2.20 or later, choose Main Chart from the Format menu. Select the type of chart you want and the desired attributes. Click OK.

    - or -

    In Excel versions 1.50 and earlier, choose Main Chart Type from the Chart menu and select the type of chart desired. From the Format menu, choose Main Chart and select the desired attributes.
