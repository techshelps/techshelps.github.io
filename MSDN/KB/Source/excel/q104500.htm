XL5: Selected Range Not Reflected in Print Area Box

 Last reviewed: September 12, 1996
Article ID: Q104500
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

In Microsoft Excel 5.0, when you consistently print a specific range on a sheet, you can enter this range in the Print Area and/or Print Titles boxes on the Sheet tab in the Page Setup dialog box. When you do this, the names Print_Area and Print_Titles are defined and each time you print the sheet, these ranges are used.

If you select a range of cells and choose Page Setup from the File menu and then select the Sheet tab, the selected range is not reflected in the Print Area box because once you choose OK, the defined name Print_Area would be redefined to the selected range. (This is by design: if Microsoft Excel allowed you do do this, you could easily redefine your print area without intending to).

If you want to print the selected range on a sheet rather than the print area, choose Print from the File menu and select the Selection option under Print What.
