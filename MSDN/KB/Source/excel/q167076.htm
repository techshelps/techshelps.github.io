XL97: Text Problems in Controls When Worksheet is Zoomed

 Last reviewed: February 12, 1998
Article ID: Q167076
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, when you insert a control from the Control Toolbox toolbar in a worksheet, you may notice any of the following problems:

  • The font size of any text attached to a control, such as a check box, an option button, or a button, suddenly increases when you click the control.

    This prevents you from viewing all of the text attached to the control. 

        -or-
  • When you enter text into a text box or a combo box, the text becomes compressed when you click another control or a cell in the worksheet. 

        -or-
  • Text in list boxes is compressed vertically. This prevents you from viewing text in the list box. 

        -or-
  • Text in labels is compressed and unreadable, except when you click the label and leave the pointer positioned over the label.

CAUSE

These problems occur if the following conditions are true:

  • You zoom the worksheet to a setting of less than 100 percent. 

        -and-
  • The worksheet is not in design mode.

As the Zoom setting decreases towards the minimum (10 percent), the problems become more noticeable.

NOTE: These problems do not occur when the Zoom setting for the worksheet is between 100 and 400 percent.

WORKAROUND

To work around this problem, click Zoom on the Standard toolbar to increase the Zoom setting to a value equal to or greater than 100 percent.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

In Microsoft Excel 97, you can use the Control Toolbox toolbar to insert a control in a worksheet. By default, the following controls are available on the Control Toolbox toolbar: 

   Check Box          Option Button
   Combo Box          Scroll Bar
   Command Button     Spin Button
   Image              Text Box
   Label              Toggle Button
   List Box
Although you can use these controls in worksheets, text attached to these controls does not appear properly if you zoom the worksheet that contains them to a setting of less than 100 percent.
