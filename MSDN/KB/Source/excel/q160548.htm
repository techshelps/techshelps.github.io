XL97: Error Clicking Save As HTML

 Last reviewed: January 15, 1998
Article ID: Q160548
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, when you click Save As HTML on the File menu, you may receive the following message: 

   The macro <path to HTML.XLA>!DoHTMLCommand'
   cannot be found.

CAUSE

This problem occurs if the path to the Html.xla add-in contains an apostrophe. For example, if you install Microsoft Office in the following folder: 

   c:\Program Files\Microsoft Office '97
the message appears when you click Save As HTML on the File menu.

WORKAROUND

To work around this problem, run Microsoft Office Setup and click Remove All to remove all of the Office components. After you remove the components, rerun Microsoft Office Setup and install Office in a folder with a path that does not contain an apostrophe.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
