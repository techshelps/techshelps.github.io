MXL5: Auto_Add and Auto_Remove Macros Do Not Run

 Last reviewed: February 2, 1998
Article ID: Q153453
The information in this article applies to:
  • Microsoft Excel for Macintosh, versions 5.0, 5.0a

SYMPTOMS

In Microsoft Excel for the Macintosh, version 5.0, if you create a macro called Auto_Add in an add-in, then that macro should run whenever the add- in is added in the Add-in Manager. Likewise, if you create a macro called Auto_Remove in an add-in, then that macro should run whenever the add-in is removed from the Add-in Manager. These macros will not run as expected in Microsoft Excel for the Macintosh, version 5.0. When you check or uncheck the add-in in the Add-in Manager, the macros do not run.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

"Microsoft Excel Visual Basic User's Guide," version 5.0, page 278.
