Excel DAM: Unable to Establish Connection Returned Immediately

 Last reviewed: October 7, 1997
Article ID: Q63905
2.20 MACINTOSH CL1 DAL Language kbother

SUMMARY

The message "Unable to Establish Connection" is returned immediately when trying to log on to the mainframe using Microsoft Excel's Data Access Macro (DAM) if the Data Access (CL/1) resources have not been installed.

To be sure that the resources have been installed, run the Installer again. If the Installer gives the following message, the CL/1 resources have been previously installed (click Cancel in this dialog box): 

   The resources you are attempting to add already exist on the
   destination file. Would you like to replace them?"
If no error is received, the resources were not previously installed. Attempts to log on to the mainframe should now be successful.

If the error "Unable to Establish Connection" is received and the resources have been correctly installed, contact Microsoft Product Support Services at (425) 454-2030 for more information.
