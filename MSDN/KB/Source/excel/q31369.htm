Excel's Update Remote References Option Is Not Saved

 Last reviewed: September 22, 1995
Article ID: Q31369
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0
  • Microsoft Excel for OS/2, versions 2.2, 3.0
  • Microsoft Excel for the Macintosh, versions 3.x, 4.0

The Update Remote References option in the Options Calculation dialog box is an option that is specific to each sheet. This option is not saved with the sheet or, in the case of Microsoft Excel for Windows, in the Options= setting of the WIN.INI file; by default this option is turned on. If you want this setting turned off for a sheet, it must be manually set this way each time you open the sheet.

For information about this feature in later versions of Microsoft Excel, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q110006
   TITLE     : XL: Update Remote References Option Selected by Default
