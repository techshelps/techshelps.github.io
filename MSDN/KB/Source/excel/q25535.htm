Excel Err Msg: Can't Change Part of a Table

 Last reviewed: November 2, 1994
Article ID: Q25535

SUMMARY

When selecting a column and attempting to cut, clear, or delete it, the error message "Can't change part of a table" for version 2.20 or "Illegal table operation" for versions 1.50 and earlier will be displayed.

To modify only part of a table, do the following:

  1. Select the entire table by pressing COMMAND+SHIFT while double-clicking a cell in the table. (For versions 1.50 and earlier, double-click a cell in the table.)

  2. From the Edit menu, choose Copy.

  3. From the Edit menu, choose Paste Special and select Values.

This procedure removes the table definition from the array and allows the data to be edited like normal text. When you are finished editing, redefine the table by choosing Table from the Data menu and reentering the row and/or column input cells.
