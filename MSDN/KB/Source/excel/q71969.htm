OPEN.INDEX Not Working Correctly in Q+E Example

 Last reviewed: November 2, 1994
Article ID: Q71969

SUMMARY

Page 103 of the "Q+E for Microsoft Excel User's Guide" for version 3.00 shows an incorrect example for using the OPEN.INDEX command, which opens a dBASE index file. Executing this command will return a #REF! error.

MORE INFORMATION

The example on page 103 shows 

   OPEN.INDEX('EMPHIRE',TRUE)
which is incomplete. You need to add the filename extension .NDX, which will then be executed correctly. The following example shows the correct syntax: 

   OPEN.INDEX('EMPHIRE.NDX',TRUE)

REFERENCES

"Q+E for Microsoft Excel User's Guide." Version 3.00, page 103.
