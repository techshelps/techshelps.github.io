Excel: OPTION+RETURN Inserts a Carriage Return in Cell Notes

 Last reviewed: November 2, 1994
Article ID: Q50510

SUMMARY

There is a documentation error on page 8 of the "Microsoft Excel Quick Reference Guide" version 2.2 manual. The quick reference guide incorrectly indicates that CTRL+RETURN inserts a line break. To force a carriage return in a cell note, hold down the OPTION key while pressing RETURN.

Note that this is different from inserting multiple lines in a chart, for which you use COMMAND+RETURN.
