XL: Cannot Select Logo When Using Additional Templates

 Last reviewed: February 9, 1998
Article ID: Q180698
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In either of the versions of Microsoft Excel listed above, if you click the Customize button in any of the following templates 

   Business Planner
   Car Lease Advisor
   Change Request
   Loan Manager
   Personal Budgeter
   Sales Quote
   Timecard
and then click the Select Logo button, you receive the following error message: 

   This feature is not available in this version of Excel.

WORKAROUND

If you are running Microsoft Excel 98 Macintosh Edition, you can select a logo by manually running the InsertLogo macro. If you are using Microsoft Excel 5.0 for the Macintosh, it is not possible to select a logo.

Microsoft Excel 98 Macintosh Edition

If you want to select a logo in any of the templates that are listed in the "Symptoms" section, follow these steps:

  1. On the Tools menu, point to Macro, and then click Macros.

  2. In the list of macros, click InsertLogo. Then, click Run.

  3. In the dialog box, select the picture that you want to use as a logo. Then, click Insert.

The picture is inserted into the template as a logo.

Microsoft Excel for the Macintosh, Version 5.0

It is not possible to select a logo in Microsoft Excel 5.0. If you run the InsertLogo macro, you receive the following error message: 

   Show method of Dialog class failed

STATUS

This behavior is by design of the templates listed in this article.

MORE INFORMATION

The templates listed in this article are included in the Value Pack in Microsoft Office 98 Macintosh Edition. To install these templates, follow these steps:

  1. Start the Value Pack Installer program, which is located in the Value Pack folder on the Microsoft Office 98 Macintosh Edition compact disc (CD).

  2. In the list of components, click the arrow (>) next to Templates. Then, select the Additional Excel Templates check box.

  3. Click Install.

The templates are installed into the following folder: 

   Microsoft Office 98:Templates:Spreadsheet Solutions
These templates are designed for use in Microsoft Excel, versions 5.0 and later for Windows and the Macintosh.

Because the InsertLogo macro does not work in Microsoft Excel 5.0 for the Macintosh, the Select Logo button has been reassigned to run the NotAvail macro, which displays the error message in the "Workarounds" section, in the Macintosh versions of these templates. This is true even though the InsertLogo macro works correctly in Microsoft Excel 98 Macintosh Edition.
