    Note: To use these filters, Microsoft Excel 97 and Microsoft Small Business Financial Manager 97 must be installed on your computer.

    INSTALLATION INSTRUCTIONS

    1. Quit Microsoft Excel 97 if it is running.

    2. Copy the appropriate file (see the following list) to the SBFM folder on your computer's hard disk. By default, this is C:\Program Files\Microsoft Office\Office\Sbfm. Replace the existing file if it is present. 

         Accounting software           Filter
   -------------------------------------------

   Great Plains Accounting       Aapgreat.dll
   versions 8.0, 8.1, and 8.2
   for MS-DOS

   Platinum Series version       Aapplat44.dll
   4.4 for Windows

   Timeline MV Server            Aapmeta.dll
   versions 2.3, 2.4, and 2.5
    SUPPORT

    Assistance with these filters is provided to you through your Microsoft Small Business Financial Manager 97 for Microsoft Excel warranty support policy. Installing these filters will not restart this warranty period. If your Microsoft Small Business Financial Manager 97 warranty support period has expired, paid phone support and alternative options are available.

    If your license to the Microsoft Small Business Financial Manager 97 product was obtained from a hardware manufacturer, then technical support for these filters is not available from Microsoft. Please contact your hardware manufacturer before installing these filters to determine your support options.
