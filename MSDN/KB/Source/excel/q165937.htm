Installation Instructions

Note: To use this filter, Microsoft Excel 97 and Microsoft Small Business Financial Manager 97 must be installed on your computer.

The updated filter includes two .dll files, Aapqbw40.dll and Msaap.dll. To install the QuickBooks filter update:

  1. Quit Microsoft Excel if it is running.

  2. Copy Aapqbw40.dll to the SBFM folder on your computer's hard disk, replacing the existing Aapqbw40.dll file if present.

    By default this is C:\Program Files\Microsoft Office\Office\Sbfm.

  3. Copy Msaap.dll to the Windows System folder on your computer's hard disk, replacing the existing Msaap.dll file if present.

    By default this is C:\Windows\System.
