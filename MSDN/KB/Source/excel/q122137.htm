XL: GP Fault When Info Window Viewed in Print Preview

 Last reviewed: September 13, 1996
Article ID: Q122137
The information in this article applies to:

- Microsoft Excel for Windows, versions 5.0, 5.0c

  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, you may receive a general protection (GP) fault when you view the Info window in print preview.

CAUSE

This error will occur if the Info window is set to display the formats applied to the active cell on the worksheet.

WORKAROUND

To avoid this problem, avoid viewing the Info window in print preview when Format information is displayed in the window.

To remove the format information from the Info window:

  1. Activate the Info window.

  2. From the Info menu, choose Format (to remove the check mark that appears next to it).

The Info window should no longer contain format information.
