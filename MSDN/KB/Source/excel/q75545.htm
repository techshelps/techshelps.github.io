Data Form Changes Not Saved When Moving to Another Record

 Last reviewed: November 3, 1994
Article ID: Q75545
The information in this article applies to:
  • Microsoft Excel for Windows, version 2.x, 3.0, 4.0, 4.0a
  • Microsoft Excel for OS/2, version 3.0

SUMMARY

Microsoft Excel has a Data Form feature that allows you to browse through and edit records in a Database. The Note: This only refers to changes made to fields in a record, not to records deleted with the Delete button on the Data Form.
