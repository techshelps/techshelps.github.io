Excel: Autoexec Macros Can Run Resident Macros

 Last reviewed: November 2, 1994
Article ID: Q61495

SUMMARY

Autoexec macros can run macros that are located on the same macro sheet.

For example, if an autoexec macro named "autorun" is executed by a worksheet, a submacro called "sub" can be run within "autorun" by typing the following command in the autoexec macro: 

   =sub()
For more information on autoexec macros, see pages 618-620 in the "Microsoft Excel User's Guide" version 3.0 manual.
