Memo Fields Can Be Created in Q+E 2.50

 Last reviewed: November 2, 1994
Article ID: Q67510

SUMMARY

Q+E version 2.50 supports the creation of dBASE compatible files containing Memo fields. From the File menu, choose Define to create dBASE compatible files that contain Character, Date, Float, Logical, Numeric, and Memo fields.

MORE INFORMATION

When memos are added to a dBASE compatible file, Q+E creates a second file to store the memos. This file will have the same name as the dBASE compatible file, except the extension will be .DBT instead of .DBF. When you add or update Memo fields, Q+E automatically opens and maintains the .DBT file.

Once the file has been defined, memo fields may be edited by typing data directly into the field, by choosing Zoom Field from the Edit menu, or by double-clicking the field to bring up the Zoom Field window.

If you move the .DBF file to another location or directory, you must also move to .DBT file.
