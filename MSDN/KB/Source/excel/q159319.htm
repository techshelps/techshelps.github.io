XL97: CTRL + Autofill Does not Increment Values

 Last reviewed: January 15, 1998
Article ID: Q159319
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you press the Control key and drag the fill handle to autofill cells, the value in the destination cells does not increment.

CAUSE

When you Autofill from a cell with Merge Cell formatting, values do not increment when you press the Control key and drag the fill handle. This behavior is the opposite of behavior exhibited by non-merged cells.

WORKAROUND

In order for the filled cells to increment as expected, use the fill handle on the merged cell without using the Control key.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about filling cells, click the Index tab in Microsoft Excel Help, type the following text 

   filling cells
and then double-click the selected text to go to the "Automatically fill in data based on adjacent cells" topic.
