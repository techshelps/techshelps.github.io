XL: Can't Use Scroll Bar on Number Tab in Format Cells Dialog

 Last reviewed: December 30, 1996
Article ID: Q130045
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97

SYMPTOMS

In Microsoft Excel, when you display the Format Cells dialog box and click the Number tab, you cannot use the scroll bar in the Category list. In addition, when you click an item in the Category list, different lists appear in the Format Cells dialog box. These lists may contain scroll bars that are not available.

CAUSE

This behavior occurs when you can see all the items in the list and it is not necessary to use the scroll bar. Note that the scroll bar is visible for each of the lists.

MORE INFORMATION

The lists that appear when you click Date, Time, Fraction, or Special in the Category list may contain more items in localized versions of Microsoft Excel. Therefore, you must use the scroll bar to access these lists. This is true even if you cannot use the scroll bar in the U.S. version of Microsoft Excel.
