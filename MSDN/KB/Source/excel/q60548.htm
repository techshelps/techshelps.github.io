Using LOOKUP to Open Files in a Macro Loop in Excel

 Last reviewed: November 2, 1994
Article ID: Q60548
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 5.0
  • Microsoft Excel for OS/2, versions 2.2, 3.0

SUMMARY

The macro segment below opens existing files. Specific operations may be then performed on the files while within the macro loop.

It is assumed the valid filenames are located in column C starting in cell C1 of the macro sheet: 

   A1: Title                              B1: 1   C1: testa.xls
   A2: =FOR("count",1,3,1)                B2: 2   C2: testb.xls
   A3: =OPEN(LOOKUP(count,B1:B3, C1:C3))  B3: 3   C3: testc.xls
   A4: <Operation(s) Performed Here.>     B4:     C4:
   A5: <Operation(s) Performed Here.>     B5:     C5:
   A6: =NEXT()                            B6:     C6:
   A7: =RETURN()                          B7:     C7:
Note: the information above also applies to version 5.0 only when using version 4.0 macro language.
