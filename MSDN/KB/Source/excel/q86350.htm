Integration of Microsoft Excel 4.0 with HP New Wave

 Last reviewed: December 15, 1994
Article ID: Q86350
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0

NewWave 4.0 includes a generic OLE\DDE bridge that is used by Microsoft Excel 4.0. In order to use Microsoft Excel 4.0 with Hewlett-Packard (HP) NewWave, you must upgrade to NewWave version 4.0
