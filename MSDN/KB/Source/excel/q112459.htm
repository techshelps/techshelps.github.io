XL: Using the Update Add-In Links File UPDTLINK.XLA

 Last reviewed: December 18, 1996
Article ID: Q112459
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel 97 for Windows

SUMMARY

The Updtlink.xla file is the Update Add-in Links add-in that is included with Microsoft Excel. This file updates links to Microsoft Excel version 4.0 add-ins to directly access the built-in functionality in Microsoft Excel versions 5.0, 7.0 and 97.

MORE INFORMATION

Worksheet and macro sheet functions contained in the following Microsoft Excel 4.0 add-ins are either contained in Analysis.xll (Microsoft Excel version 5.0 Analysis ToolPak add-in), Analys32.xll (Microsoft Excel versions 7.0 and 97 Analysis ToolPak add-in), or the functions are built-in to Microsoft Excel:

Analysis.xla - Analysis ToolPak macro functions are contained in 

               Analysis.xll
Analysf.xla - Analysis ToolPak worksheet functions are contained in 
               Analysis.xll
Addinfns.xla - Add-in functions in Analysis.xll (DEGREES() and 
               RADIANS() functions are built-in functions in Microsoft
               Excel versions 5.0, 7.0 and 97.
Scenario.xla - Scenario Manager macro functions are built-in functions 
               in Microsoft Excel versions 5.0, 7.0 and 97.
Updated add-in files Analysis.xla, Analysf.xla, and Addinfns.xla are included with Microsoft Excel versions 5.0, 7.0 and 97 so that you can use Microsoft Excel version 4.0 documents with links to worksheet and macro sheet functions contained in these add-ins without modification. However, when you use the functions contained in Analysis.xll or built- in functions in Microsoft Excel, calculations are performed faster. If you open a document that references one of the add-ins listed above, your document takes less time to calculate if you change the reference to Analysis.xll or to the built-in function.

You can use the Update Add-in Links add-in to change these references for you. This add-in changes a link to a Microsoft Excel version 4.0 add-in to the Microsoft Excel built-in function or to the function contained in Analysis.xll.

When you open or load Updtlink.xla as an add-in, in Microsoft Excel versions 5.0 and 7.0, the "MS Excel 4.0 Add-in Links..." command appears on your Tools menu. In Microsoft Excel 97, the "Update Addin Links..." command appears. When you choose this command, a dialog box appears that allows you to update the links in the current document or all open documents.

NOTE: the BASE and FASTMATCH functions no longer exist in Microsoft Excel version 5.0, 7.0 and 97. Using these functions will return a #NAME! Error even after running the Updtlink.xla.

To use Updtlink.xla to change references to add-in functions, do the following:

  1. Open the documents that contain the references to functions in the Microsoft Excel 4.0 add-ins listed above.

  2. Open the file Updtlink.xla. This file is located in the LIBRARY subdirectory of the directory to which you installed Microsoft Excel version 5.0.

  3. Open the file Analysis.xll (open Analys32.xll if you are using Microsoft Excel versions 7.0 or 97). This file is located in the ANALYSIS subdirectory of the LIBRARY subdirectory of the directory to which you installed Microsoft Excel version. (This step is not necessary if your document references Scenario.xla or the DEGREES() or RADIANS() functions.)

  4. On the Tools menu, click MS Excel 4.0 Add-In Links (in versions 7.0 and 97 click Update Addin Links).

  5. To update the references in the active workbook only, select Active Document. To update the references in all open workbooks, select All Open Documents. Click OK.

For more information on Updtlink.xla, do the following:

  1. In Microsoft Excel 5.0 or 7.0, click Contents on the Help menu.

  2. From the Microsoft Excel Help Contents screen, select Reference Information.

  3. From the Reference Information screen, select Microsoft Excel Macro Functions Contents.

  4. Click the Search button and type "links, Microsoft Excel 4.0, changing" (without the quotation marks) and choose Show Topics.

  5. From the list of topics, select Changing links to Microsoft Excel version 4.0 add-ins and choose Go To.
