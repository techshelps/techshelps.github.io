XL97: Cannot Clear "Sort by" List in Query Wizard

 Last reviewed: January 15, 1998
Article ID: Q157062
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Query Wizard, you cannot clear the "Sort by" list in the Sort Order dialog box. There is no command that allows you to clear the "Sort by" list.

CAUSE

This problem occurs when you select a field by which to sort in the Sort Order dialog box, and then try to clear the "Sort by" list.

WORKAROUND

To work around this problem, do the following:

  1. Click Back twice to go back to the Choose Columns dialog box.

  2. In the Choose Columns dialog box, remove all the fields by clicking the remove all (<<) button.

  3. Select each table or column you removed in step 2, and click the add (>) button.

  4. Click Next twice.

The "Sort by" list in the Sort Order dialog is cleared.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
