XL98: No Shortcut Key Appears Next to Chart Item on Format Menu

 Last reviewed: February 2, 1998
Article ID: Q175940
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

If you select an item in a chart in Microsoft Excel 98 Macintosh Edition and click the Format menu, a shortcut key does not appear next to the "Selected <chart item>" menu item.

WORKAROUND

To work around this problem, press COMMAND+1 to format the item. Even though the Format menu does not list the shortcut key, you can still press COMMAND+1 to format the selected item in the chart.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

On the Macintosh, the COMMAND key is usually the key that is immediately to the left or right of the space bar. This key is also known as the clover- leaf key, the Apple key, or the "kyrka."

In Microsoft Excel for the Macintosh, you can press COMMAND+1 (hold down the COMMAND key and press 1) to format the selected cell or object in a worksheet or chart sheet. If you click the Format menu when a worksheet is active, the menu contains a menu item similar to the following 

   Cells...           #1
where the symbol "#" represents a clover-leaf symbol. This symbol indicates that you can press COMMAND+1 to use the menu item.

When a chart sheet is active, the shortcut key is not listed on the menu, for example: 

   Selected Axis...
Even though a shortcut key is not listed, you can still use the menu item by pressing COMMAND+1.
