Excel: International Versions Available

 Last reviewed: April 17, 1997
Article ID: Q85293
3.00 4.00 | 3.00 4.00 5.00 
WINDOWS   | MACINTOSH
kbref

The information in this article applies to:

  • Microsoft Excel for Windows, versions 3.0, 4.0
  • Microsoft Excel for the Macintosh, version 3.0, 4.0, 5.0

Microsoft Excel version 5.0 for Macintosh is available in the following international languages: 

Danish, Dutch, English, French, German, Italian, Japanese,             
Norwegian, Portuguese, Spanish, and Swedish
Microsoft Excel version 4.0 for Windows is available in the following sixteen international languages:

Chinese, Danish, Dutch, English, Finnish, French, German, Greek, Hebrew, Italian, Japanese, Korean, Norwegian, Portuguese, Spanish, Swedish

Microsoft Excel for the Macintosh, version 4.0 is available in the following eleven international languages:

Danish, Dutch, English, Finnish, French, German, Italian, Japanese, Norwegian, Spanish, Swedish

Microsoft Excel for Windows, version 3.0, is available in the following eleven international languages:

Dutch, Finnish, French, French Canadian, German, Hanguel, Italian, Kanji, Portuguese, Spanish, and Swedish.

Microsoft Excel for the Macintosh, version 3.0, is available in the following fifteen international languages:

Arabic, Danish, Dutch, Finnish, French, French Canadian, German, Greek, Hebrew, Italian, Norwegian, Portuguese, Spanish, Swedish, and Turkish.
