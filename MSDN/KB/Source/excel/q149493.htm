XL7: Data Map Refreshes Only Once per Session

 Last reviewed: September 13, 1996
Article ID: Q149493
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel version 7.0, when you create a map using Data Map and then change the map data and refresh the map more than once, the refresh icon fails to appear and the map fails to update.

CAUSE

This problem occurs when you do the following:

  • You use Data Map to create a map based on geographic data.

    -and-

  • You save and close the workbook.

    -and-

  • You reopen the workbook, and change data on which the map is based.

    -and-

  • You double-click the map object and click the exclamation point button to update the map.

    -and-

  • You change the map data and attempt to update the map again.

This problem does not occur in Microsoft Excel version 7.0a.

RESOLUTION

To resolve this problem, obtain Microsoft Excel 7.0a.

To work around the problem in Microsoft Excel 7.0, save and close the workbook, exit Microsoft Excel, and reopen the workbook. When you reopen the workbook, the map automatically updates.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem is corrected in Microsoft Excel 7.0a.

MORE INFORMATION

Microsoft Excel incorrectly increments the counter that determines when to update the map. When you restart Microsoft Excel, this counter is reset.
