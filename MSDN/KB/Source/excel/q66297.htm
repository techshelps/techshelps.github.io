Extracting Cell Notes into Another Application

 Last reviewed: November 2, 1994
Article ID: Q66297
The information in this article applies to:
  • Microsoft Excel for Windows versions 2.x, 3.0, 4.0

SUMMARY

There are two ways to extract cell notes and place them into another area or application, as follows:

First Method

  1. From the Formula menu, choose Note.

  2. Highlight the cell note.

  3. Highlight the text in the note box.

  4. Press CTRL+INS to copy the note to the Clipboard.

  5. Choose the application you want the note to go to and paste it into the application.

Note: You must use CTRL+INS to copy; you will not be able to select the Edit menu from the note box.

Second Method

  1. Connect to the Generic Text Only driver on file.

  2. Print the spreadsheet to a file choosing cell notes only in the print dialog box.

  3. The cell notes will then print to a file and you can load them into another application as a text file.
