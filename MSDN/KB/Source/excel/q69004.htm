HP PaintJet Prints Charts in Red in Excel 3.00

 Last reviewed: November 4, 1994
Article ID: Q69004
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0

SUMMARY

In Microsoft Excel, when you print a chart to the HP PaintJet, the chart may print in red even though it appears in another color on the screen.

MORE INFORMATION

If you select Patterns from the Format menu and choose any pattern other than the default, the chart displays correctly on the screen and under Print Preview, but prints in red. This is a problem with the HP PaintJet 1.00 printer driver. This problem does not occur with Microsoft Excel versions earlier than 3.0.

Workaround

Format the chart with solid colors instead of with patterns.

Upgrade

Upgrade to the Hewlett-Packard (HP) PaintJet driver (HPPJ1.DRV) version 1.01, which prints charts correctly. The driver is dated 2/17/91 and can be obtained from the Hewlett-Packard Fulfillment Center at (303) 353-7650. There is a charge for this driver.

For more information on the PaintJet, query on the following words: 

   paintjet and chart and print
