Excel: Drag and Drop Doesn't Move or Copy Sheets to Desktop

 Last reviewed: November 30, 1994
Article ID: Q81853
The information in this article applies to:
  • Microsoft Excel for Windows version 4.0
  • Microsoft Excel for the Macintosh version 4.0

SUMMARY

In Microsoft Excel, you can move and copy worksheets, charts and macro sheets between workbooks using the Drag and Drop feature. This, however, will not work for moving and copying sheets from the Microsoft Excel desktop into a workbook.

MORE INFORMATION

To move a file from one workbook to another, simply click on the file icon in the source workbook and drag and drop it onto the destination workbook. To copy this file in Microsoft Excel for Windows, hold down CTRL and drag the document to another Workbook Contents window or icon. In Microsoft Excel for the Macintosh, hold down OPTION and drag the document to another Workbook Contents window.

If you minimize a worksheet to an icon on the Microsoft Excel for Windows desktop then try to copy it to an open workbook using CTRL+Drag and Drop, the sheet icon will appear over the workbook but the sheet will not be added to the workbook. To add the sheet to the open workbook, select the Add button at the bottom of the active workbook, choose the sheet name from the list that is displayed and then select OK or press the Enter key. If the destination workbook is minimized to an icon, the document is added to the end of the list of documents in the workbook.
