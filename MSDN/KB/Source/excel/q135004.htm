XL7: Quattro Pro Array Constant Formulas Not Converted

 Last reviewed: March 27, 1997
Article ID: Q135004
7.00 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you open a Quattro Pro for Windows document in Microsoft Excel, and you use the Quattro Pro for Windows file filter, if you have formulas with array constants in them, these formulas will not be imported and the value of the formula will be displayed in the cell. You do not receive an alert message or cell note indicating that the formula was not imported.

CAUSE

Quattro Pro for Windows can use formulas with arrays built in, such as the following: 

   @SUM({1,2,3})
This formula sums the values 1, 2, and 3 with a result of 6. The Quattro Pro for Windows file filter is not designed to recognize these formulas.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
