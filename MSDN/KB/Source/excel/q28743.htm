Active Cell Placement Using FORMULA.GOTO Macro Command

 Last reviewed: October 31, 1994
Article ID: Q28743
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2 versions 2.2, 2.21, and 3.0

Question:

How is the placement of the active cell determined when using Excel's FORMULA.GOTO macro command?

Response:

If the destination cell is in the active window, the destination cell will become the active cell. If the destination cell is off the window in any direction, Excel will scroll the current window to display the active cell. Excel will scroll in horizontal and vertical directions only if it is not possible to display the active cell by scrolling in only one direction.

If the destination cell is off the window in the "up" direction, Excel will place the row containing the active cell at the top of the window. If the destination cell is off the window in the "down" direction, Excel will place the row containing the active cell at the bottom of the window.

If the destination cell is in a currently displayed row, but to the left or right of the current window, Excel will scroll horizontally to place the column containing the cell at the left or right edge of the window. Excel will not scroll vertically.

If the destination cell is in a currently displayed column, but above or below the current window, Excel will scroll vertically to place the column containing the cell at the top or bottom edge of the window. Excel will not scroll horizontally.
