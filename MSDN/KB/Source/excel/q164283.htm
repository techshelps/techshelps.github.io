SBFM97: Cannot Find the Aapc.hlp in Canadian SBFM

 Last reviewed: February 12, 1998
Article ID: Q164283
The information in this article applies to:
  • Microsoft Small Business Financial Manager 97 for Microsoft Excel 97
  • Microsoft Excel 97 for Windows

SYMPTOMS

In the Canadian version of the Microsoft Small Business Financial Manager (SBFM), when you use one of the Help topics under "Using What-If Scenarios" in Small Business Financial Manager Help, the following error message appears: 

   Cannot find the aapC.Hlp file.
   Do you want to try to find this file yourself?
If you click Yes, you find that the Help file is not installed.

CAUSE

This problem occurs when the following conditions are true:

  • You click Financial Manager Help Topics on the Help menu and click "How to Use Financial Manager." 

        -and-
  • You display the topic "Creating and Using What-If Scenarios." 

        -and-
  • You display one the following subtopics 

          Analysis selling an asset
      Analyze your cost of sales
      Analyze your sales revenue
      Analyze your AP, AR, and Inventory
      Analyze buy vs. lease options

This problem occurs because this section of the Help file fails to refer to the correct Help file name.

WORKAROUND

To work around this problem, rename the installed Help file. To do this, use the following steps:

  1. On the Start menu, point to Find, and then click Files Or Folders. Then, search for the SBFM Help file, Aap.hlp.

    This file is installed by default in the folder that contains the SBFM, which is in the folder that contains Microsoft Excel.

  2. Copy the file Aap.hlp

  3. Rename the copy of Aap.hlp to Aapc.hlp.

For more information about how to perform this task in Windows, see your Windows printed documentation or Windows Help.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem has been fixed in the updated versions of the Quickbooks driver and Msapp.dll.
