Excel: No Alert if User-defined Add-in Loaded Twice

 Last reviewed: November 30, 1994
Article ID: Q88456
The information in this article applies to:
  • Microsoft Excel for Windows, version 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

The Microsoft Excel Add-in Manager doesn't display an alert when loading a user defined add-in macro that is already loaded. This may cause problems, such as duplication of commands or menus that the user-defined macro adds to the Excel menu bar.

MORE INFORMATION

You use the Excel Add-in Manager to maintain a list of add-in macros that are loaded when you start Excel. When you add a macro to the list, Excel also loads it, even if your add-in macro is already loaded.

To avoid possible problems, check to see that your add-in macro is not already loaded before adding it to the Add-in Manager list.
