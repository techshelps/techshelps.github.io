No "New" Dialog Box When You Click New on the File Menu

 Last reviewed: March 27, 1997
Article ID: Q134335
7.00 WINDOWS kbui kbusage

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, when you click New on the File menu, the New dialog box will not be displayed unless templates exist in the Microsoft Office Templates folder.

STATUS

This behavior is by design.

MORE INFORMATION

In Microsoft Word version 7.0, when you click New on the File menu, the New dialog box is always displayed (even if the Microsoft Office Templates folder is empty). When this dialog box is displayed, Blank Document is the template choice on the General tab. This behavior is different from Microsoft Excel version 7.0.
