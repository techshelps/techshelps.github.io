XL97: Status Bar Is Bold When Display Setting Is in Large Font

 Last reviewed: May 7, 1997
Article ID: Q168142
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you are working in Microsoft Excel 97 for Windows, the font on the status bar appears to be bold.

CAUSE

This problem occurs if the font size of the Windows 95 display settings is set to Large Fonts.

WORKAROUND

To workaround this problem, use either of the following methods.

Method 1: Change the Font Size to Small Fonts

To change the font size, do the following:

  1. Click the Start button, point to Settings, and click Control Panel.

  2. In the Control Panel window, double-click the Display icon. In the Display Properties dialog box, click the Settings tab.

  3. In the Font Size box, click Small Fonts. Click OK.

  4. When you are prompted to restart the computer, click Yes.

Method 2: Use a Custom Font Size

To use a custom font size, do the following:

  1. Click the Start button, point to Settings, and click Control Panel.

  2. In the Control Panel window, double-click the Display icon. In the Display Properties dialog box, click the Settings tab.

  3. Click the Custom button.

    The Custom Font Size dialog box appears.

  4. Use the ruler to drag the font size to less than 117%. Click OK twice.

  5. When you are prompted to restart the computer, click Yes.

NOTE: You may use a custom font size only when you use a display resolution greater than 640 X 480. If you are using a resolution of 640 X 480, you must use Method 1 or increase the resolution to 800 X 600 or higher.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
