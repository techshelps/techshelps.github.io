XL: Can't Print Background Graphic Included on Worksheet

 Last reviewed: March 2, 1998
Article ID: Q134212
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SYMPTOMS

Although you can add a background graphic to a worksheet, this graphic is not visible when you view your document in Print Preview or when you print it.

STATUS

This behavior is by design in Microsoft Excel because printing a background is extremely slow on most standard printers.

MORE INFORMATION

To add a graphic to the background of a Microsoft Excel worksheet, on the Format menu, click Sheet and then click Background. Select the bitmap you want to appear as the sheet background, and click OK.
