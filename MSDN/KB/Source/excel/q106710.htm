XL5: Pivot Table Field Named "Data" Is Changed to Data2

 Last reviewed: September 12, 1996
Article ID: Q106710
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

When you create a pivot table from a list or database containing a field named "Data," that field name will be changed to the name "Data2."

This field name change is by design to avoid confusion with the Data area in the pivot table.

MORE INFORMATION

Steps to Reproduce the Problem

  1. In a new worksheet, type the following: 

          A1: Date        B1: Name    C1: Data
      A2: 10/28/93    B2: John    C2: 5287
      A3: 10/28/93    B3: Mary    C3: 7326
      A4: 10/28/93    B4: John    C4: 1259
      A5: 10/29/93    B5: Mary    C5: 2146

  2. From the Data menu, choose PivotTable.

  3. In Step 1, choose Microsoft Excel List or Database, and choose Next.

  4. In Step 2, select cells A1:C5 on the worksheet, and choose Next.

Note that the field buttons are Date, Name, and Data2.
