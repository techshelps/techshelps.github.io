Excel: FWRITELN Command Does Not Write a Linefeed to File

 Last reviewed: November 30, 1994
Article ID: Q75068
Summary:

The FWRITELN command in Microsoft Excel for the Macintosh versions 2.2 and 3.0 writes text and follows it with a carriage return. It does not add an additional linefeed after the carriage return as stated in the 2.x documentation. This documentation error has been corrected in Microsoft Excel 3.0.

More Information:

Page 99 of the "Microsoft Excel Function Reference" version 3.0 manual states that the FWRITELN command writes text followed by a carriage return and a linefeed to a file. However, a section of the documentation following this correctly states: 

   "Note: In Microsoft Excel for the Machintosh, FWRITELN writes text
   followed by a carriage return only."
Page 231 in the "Microsoft Excel Functions and Macros" version 2.2 manual does not state this.
