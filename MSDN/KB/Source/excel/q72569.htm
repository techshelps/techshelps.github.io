DB.LOGON Accepts Only the Source Arguement in Q+E

 Last reviewed: November 2, 1994
Article ID: Q72569

SUMMARY

Page 69 of the "Q+E for Microsoft Excel's User's Guide" for version 3.0 shows that the DB.LOGON command requires four arguments. Executing this command with the four arguments included will result in a #REF! error.

MORE INFORMATION

The example on page 69-70 shows: 

   DB.LOGON(source,database,username,password)
DB.LOGON will only accept the "source" argument. The three remaining arguments can be executed using the SEND.KEYS command.

REFERENCES

"Q+E for Microsoft Excel's User's Guide." Version 3.0, page 69-70.

README.TXT, Microsoft Excel for Windows. Version 3.0, EXCEL directory.
