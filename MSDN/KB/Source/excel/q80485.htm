Excel Err Msg: "Cannot Extend Database" with Data Form

 Last reviewed: May 17, 1995
Article ID: Q80485

The information in this article applies to:

  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, 4.0, 5.0

SUMMARY

You cannot add records to a database with the Microsoft Excel Form command (from the Data menu) if the database is on a protected document. This is true even when you want to place a new record in unlocked cells. The form appears when you choose the Form command, but when you enter new information, Microsoft Excel displays the error message: 

   Cannot Extend Database
All other database commands except Find are unavailable under these circumstances.

MORE INFORMATION

On protected documents, Microsoft Excel automatically removes commands that alter a worksheet's format or layout. The database cannot be extended because Microsoft Excel cannot insert rows or define names on a protected document. To redefine a database or to use certain database operations, choose Unprotect Document from the Options menu.
