XL: Split or Frozen Panes Not Saved in Workspace

 Last reviewed: August 14, 1997
Article ID: Q113803

The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SUMMARY

In Microsoft Excel for Windows, when files are saved in a workspace, splits and frozen pane settings are not saved.

This behavior is by design of Microsoft Excel.

MORE INFORMATION

When you open Microsoft Excel 5.0 files saved in a workspace, Microsoft Excel restores the selection, current active cell position, active page, and window. When a file saved within a workspace contains a split or frozen window pane, these settings will not be restored when the file is re-opened.
