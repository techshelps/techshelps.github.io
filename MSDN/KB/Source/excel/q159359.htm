OFF: How to Rename a Button on the Office Shortcut Bar

 Last reviewed: March 13, 1998
Article ID: Q159359
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0

SUMMARY

This article explains how to rename a button on the Microsoft Office Shortcut Bar that is included with Microsoft Office 7.0 or Microsoft Office 97 for Windows.

MORE INFORMATION

The following sets of steps work under Microsoft Windows 95 and Microsoft Windows NT 4.0.

NOTE: It is not possible to rename buttons on the Office Shortcut Bar under Microsoft Windows NT 3.51 because you cannot right-click shortcuts or buttons on the Office Shortcut Bar.

Renaming a Button on the Shortcut Bar in Microsoft Office 97

The Microsoft Office Shortcut Bar allows you to rename buttons without having to use Microsoft Windows Explorer. To rename a button, use the following steps:

  1. Right-click the button you want to rename. On the shortcut menu, click Rename.

  2. In the New Name box, type the new name for the button. Then, press ENTER or click OK.

The button is renamed.

Renaming a Button on the Shortcut Bar in Microsoft Office 7.0

Each button on the Office Shortcut Bar has an equivalent shortcut stored in a folder in the Msoffice\Office\Shortcut Bar on your hard disk. You can rename the button on the Office Shortcut Bar by renaming the shortcut. To do this, use the following steps:

  1. In Windows Explorer, go to the Msoffice\Office\Shortcut Bar folder on the hard disk.

  2. Double-click the folder that has the same name as the toolbar that with the button you want to rename. For example, if you want to rename a button on the Office toolbar, double-click the Office folder.

  3. In the list of shortcuts, right-click the shortcut for the button you want to rename, and click Rename on the shortcut menu.

  4. Type a new name for the button. Then, press ENTER.

After you rename the shortcut, the name for the corresponding button on the Office Shortcut Bar is automatically changed to match the new shortcut name.
