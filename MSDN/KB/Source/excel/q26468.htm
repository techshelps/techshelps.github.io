Excel: Suppressing a Leading Zero in a Decimal Number Format

 Last reviewed: October 31, 1994
Article ID: Q26468

SUMMARY

In Microsoft Excel, to suppress a leading zero in a decimal format, do the following:

  1. Select the cells to be formatted without a leading zero.

  2. From the Format menu, choose Number.

  3. In the Format box, type "#.##" (without quotation marks).

  4. Click OK.

For example, with the number format "#.##" applied, ".28" will appear as ".28" instead of "0.28".
