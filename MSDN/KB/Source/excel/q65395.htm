Excel: Change Button Unavailable in Links Dialog Box

 Last reviewed: November 30, 1994
Article ID: Q65395
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, 4.0
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a
  • Microsoft Excel for OS/2, versions 2.2, 2.21, 3.0

In Microsoft Excel, the Change button in the Links dialog box (on the File menu) will be unavailable (dimmed) for either of the following reasons:

  • No item is selected from the Links list on the left of the dialog box. To make the Change button available, select a document from the Links list.

    -or-

  • The document has been protected by choosing Protect Document from the Options menu. To enable the Change button, close the Links dialog box and choose Unprotect Document from the Options menu. When you then choose Links from the File menu and select a document, the Change button is enabled.

NOTE: Links can be changed on protected documents in Microsoft Excel version 1.5 for the Macintosh.
