LAN Manager 2.00 Enhanced Required to Access a SQL Server

 Last reviewed: November 2, 1994
Article ID: Q69956

SUMMARY

To use the SQL Server driver included with Q+E, you must be running Microsoft LAN Manager for DOS workstations. Q+E may work on networks that are compatible with Microsoft LAN Manager and provide named pipe support in protected mode.

MORE INFORMATION

If you attempt to access a SQL Server on networks that are not Microsoft LAN Manager-compatible, one of the following warning messages may be generated: 

   Unable to connect: SQL Server is unavailable or does not exist

   Network communication layer not loaded
For more information on whether your network provides protected mode named pipe support and is compatible with Microsoft LAN Manager, refer to your documentation or query on the name of the network software.

REFERENCES

"Microsoft SQL Server Installation Guide." Version 1.10, page 6.

"Release Notes" for Microsoft SQL Server version 1.1
