XL5: Setup Lists More Disk Space Than Needed for Spelling Tool

 Last reviewed: September 12, 1996
Article ID: Q111267
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

When you perform a custom installation of Microsoft Excel, the Setup program may incorrectly report that 529K of disk space is needed to install the Spelling tool. The actual amount of disk space needed to install the Spelling tool is 257K.

Although this error causes the Setup program to determine that more disk space is needed than is actually necessary, the installation of Microsoft Excel is not adversely effected.

MORE INFORMATION

In order to ensure that an adequate amount of free disk space exists to perform the installation, the Setup program shows the amount of disk space that will be used for each available option during a custom installation.
