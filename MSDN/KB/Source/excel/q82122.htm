Excel: Saving File Gets "Operation Failed" Alert

 Last reviewed: November 30, 1994
Article ID: Q82122
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0
  • Microsoft Excel for the Macintosh versions 3.0, 4.0

After you save a document with a write-reservation password, changes to the document will be saved only if it was opened with the password It is otherwise opened as read-only and while changes can be made to the document, they will only be saved if the document is saved to a different file name. Attempting to save it to the same file name will result in the alert: 

   "Operation Failed. 'Filename' is reserved by another user."
