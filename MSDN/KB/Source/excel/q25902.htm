Excel: Setting ImageWriter to Draft Mode Using Macro

 Last reviewed: November 29, 1994
Article ID: Q25902
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 2.x, 3.0, 4.0

SUMMARY

With Microsoft Excel versions 2.x, 3.0 or 4.0, it is not possible to set the ImageWriter to Draft mode from a macro.

With Excel versions 1.50 and earlier, you can use the PRINTER.SETUP() function to set the printer mode to TTY and set the baud rate to 9600.

MORE INFORMATION

The PRINTER.SETUP() function is not available in Excel 2.2, 3.0 or 4.0.
