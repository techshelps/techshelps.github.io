MSQuery: ODBC Information Not Displayed in Error Messages

 Last reviewed: March 27, 1997
Article ID: Q101449
1.00 2.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Query for Windows, version 1.0
  • Microsoft Query for Windows 95, version 2.0

By default, the error messages in Microsoft Query do not provide much detail or technical information.

If you want to display more technical detail about Microsoft Query errors, add the following line to the [Debug] section of the MSQUERY.INI file: 

   [Debug]
    Vendor Info = 1
This line instructs Microsoft Query to return the complete ODBC error messages.

The MSQUERY.INI file is located in your Windows directory and can viewed and edited in a text editor such as Windows Notepad.
