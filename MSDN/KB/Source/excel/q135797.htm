XL: ALT+DOWN ARROW Doesn't Display Pick From List

 Last reviewed: November 25, 1997
Article ID: Q135797
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, when you select a cell and press ALT+DOWN ARROW, the list of entries already entered in the column (the Pick From list) may not be displayed.

CAUSE

This behavior occurs if your worksheet contains a filtered list. If you apply the AutoFilter command to filter a list on your worksheet, the ALT+DOWN ARROW key combination does not work to display the Pick From list for a column of data on this worksheet.

RESOLUTION

To work around this behavior, you can display the Pick From list for a cell on a filtered worksheet by clicking the right mouse button on the cell that you want to fill and then clicking Pick From List from the menu that is displayed.

MORE INFORMATION

Microsoft Excel Help states that you can press ALT+DOWN ARROW in a selected cell to display a list of entries already in the column. However, if the selected cell is located on a worksheet that contains a filtered list, ALT+DOWN ARROW does not display the Pick From list. However, if you press ALT+DOWN ARROW while a column label in the filtered list is selected, the list of filter options is displayed instead of the Pick From list.

REFERENCES

For more information about using the Pick List command in Microsoft Excel, click the Index tab in Microsoft Excel Help, type the following text 

   pick list
double-click the selected text and then double-click "Quickly fill in repeated entries in a column" to go to the "Quickly Fill In Repeated Entries In A Column" topic.
