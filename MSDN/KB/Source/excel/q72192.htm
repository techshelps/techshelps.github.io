Excel: Text Overflows Text Box Outline on Some Printers

 Last reviewed: April 17, 1996
Article ID: Q72192
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0

SYMPTOMS

In Microsoft Excel, the last line of text in a text box may print over the bottom of a text box outline when you print a worksheet.

CAUSE

You will encounter this problem if you are not using one of the Windows vector fonts (that is, Modern, Roman, or Script) and the text extends below the text box.

WORKAROUNDS

To work around this problem, do either of the following:

  • Resize the text box on your screen until the last line is displayed completely. Use one of the vector printer fonts. These fonts will be clipped properly at the end of the text box, and will be printed the as they appear in print preview.

    -or-

  • If you are using Microsoft Windows version 3.1 and True Type fonts, select the Print True Type As Graphics check box in the options dialog box. (From the File menu, choose Print. Then, choose Page Setup, and choose Options.)

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
