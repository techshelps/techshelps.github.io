XL98: "Create Hyperlink Here" Does Not Work in Unsaved File

 Last reviewed: February 24, 1998
Article ID: Q181704
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel 98 Macintosh Edition, when you drag a cell with the mouse pointer while holding down the CONTROL key, and then click Create Hyperlink Here on the shortcut menu that appears, Microsoft Excel may not create a hyperlink. When this problem occurs, the following error message appears: 

   Microsoft Excel cannot paste the data.

CAUSE

This behavior occurs if you have not saved the workbook at least once.

RESOLUTION

Before you create a hyperlink in a workbook, save the workbook. After you save the workbook, create the hyperlink.

REFERENCES

For more information about hyperlinks in Microsoft Excel, click the Index button in MS Excel Help, type the following text 

   hyperlinks, creating
click Show Topics, and double-click "Create a hyperlink."
