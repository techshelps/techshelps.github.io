Excel: Incorrect Print Preview of Text Color With HPIII

 Last reviewed: November 4, 1994
Article ID: Q82261
Summary:

In Microsoft Excel for Windows version 3.0, the Hewlett-Packard (HP) LaserJet Series III version 3.85 and 3.86 printer drivers incorrectly show the print preview text in the same color as the screen text when you use a high-resolution video driver. This printer does not support color printing. This problem does not occur with the Windows VGA display driver.

More Information:

Print preview may look as if there is no text if the text color is white or yellow. Previous versions of the printer drivers, versions 3.42 and 3.77, correctly show the print preview text in black regardless of the screen text color.
