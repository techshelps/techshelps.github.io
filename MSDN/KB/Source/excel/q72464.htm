UAE or Internal Stack Overflow with Peak/DM BIOS

 Last reviewed: November 2, 1994
Article ID: Q72464

SUMMARY

Microsoft Excel 3.00 might return an Unrecoverable Application Error (UAE) or an Internal Stack Overflow error when running Windows 3.00 in enhanced mode and using version 1.10 of PEAK/DM AT BIOS.

MORE INFORMATION

This BIOS is made by Chips and Technologies, Inc. It is recommended that you upgrade the BIOS to version 1.3.0 dated March 21, 1991 or later to solve this problem.
