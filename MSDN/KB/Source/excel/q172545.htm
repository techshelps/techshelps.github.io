XL97: Error When Inserting Portfolio 7.0 Object into Worksheet

 Last reviewed: January 7, 1998
Article ID: Q172545
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Windows 95

SYMPTOMS

In Microsoft Excel 97, if you click Portfolio 7.0 Objekt on the Insert menu, or if you open an add-in file named Pfprof.xla, you may receive the following error message: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.
If you click Details, you receive one or more error messages similar to any of the following: 

   PF caused an invalid page fault in module KERNEL32.DLL at
   0137:bff782af.

   EXCEL caused an invalid page fault in module KERNEL32.DLL at
   0137:bff782af.
If you attempt to start any other programs, or if you attempt to start Microsoft Excel 97, you may receive an error message similar to either of the following: 

   <Program> caused an invalid page fault in module KERNEL32.DLL at
   0137:bff782af.

   <Program> caused an invalid page fault in module RPCRT4.DLL at
   0137:7fcc15de.
Also, your computer may stop responding (hang).

CAUSE

These problems may occur if all of the following conditions are true:

  • You install Graphitti Software GmbH's Portfolio 7.0 planning package on the computer. 

        -and-
  • In Microsoft Excel 97, you click Portfolio 7.0 Objekt on the File menu. 

        -and-
  • Your computer is running Microsoft Windows 95.

NOTE: This problem has been reported to occur under Microsoft Windows 95 but not under Microsoft Windows NT versions 3.51 or 4.0.

RESOLUTION

Portfolio 7.0 is designed to be used with Microsoft Excel for Windows, version 5.x, and Microsoft Excel for Windows 95, version 7.x. It is NOT designed to be used with Microsoft Excel 97 for Windows. Attempting to use this product in Microsoft Excel 97 is not supported by Microsoft.

If any of the problems described in this article occur, shut down and restart the computer. To do this, follow these steps:

  1. On the Start menu, click Shut Down.

  2. In the Shut Down Windows dialog box, click "Restart the computer?". Then, click Yes.

If you do not restart the computer, you may receive numerous Invalid Page Fault error messages.

After you restart the computer, use the following steps to prevent Microsoft Excel from automatically loading the Pfprof.xla add-in when you start Microsoft Excel 97:

  1. Start Windows Explorer and open the following folder: 

          Program Files\Microsoft Office\Office\Xlstart

  2. Right-click the Pfprof.xla add-in. On the Shortcut menu, click Cut.

  3. Use Windows Explorer to open the following folder: 

          Program Files\Microsoft Office\Office

  4. On the Edit menu, click Paste.

The Pfprof.xla add-in is moved from the Xlstart folder to the Office folder.

After you perform these steps, you can start Microsoft Excel 97.
