Excel: Trouble Printing Charts on HP PaintJet Printer

 Last reviewed: November 4, 1994
Article ID: Q93904

The information in this article applies to:

  • Microsoft Excel for Windows, version 4.0, 5.0

SUMMARY

In Microsoft Excel for Windows, when you print a chart to an HP PaintJet or an HP DeskJet 550C printer, you may receive one of the following error messages: 

   Out of Memory

   -or-

   Error Printing to Paint Jet on LptX

   -or-

   Scalable Font Initialization Error e:800
   Scalable Font Initialization Error b:

MORE INFORMATION

If you receive one of the above error messages when you print a chart to an HP PaintJet or an HP DeskJet 550C printer, use the following to print the chart:

  • Reduce the complexity of the chart (remove borders, set background to none, etc.).
  • Make sure that the font you are using is either a TrueType font or a printer font (and NOT a screen font).
  • If your chart is a bar chart, use a dithered fill rather than a solid fill.
  • Make sure that the PAINTJET.DRV and UNIDRV.DLL files have the same date, especially if you are working in a network environment.
