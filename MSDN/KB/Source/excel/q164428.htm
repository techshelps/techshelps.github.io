XL97: Invalid Page Fault When Selecting Shared Workbook Feature

 Last reviewed: December 17, 1997
Article ID: Q164428
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you click "Allow changes by more than one user at the same time" in the Editing tab of the Share Workbook dialog box to allow editing by more than one user, you may receive one of the following error messages when you save the file: 

   EXCEL caused an invalid page fault in module <unknown> at 0000:00560055

   -or-

   EXCEL caused an invalid page fault in module EXCEL.EXE at 0137:301dbedd

CAUSE

This problem occurs when you are trying to share a workbook that contains more than 98 worksheets. You may experience this problem with less than 98 worksheets if the workbook contains a large amount of data.

WORKAROUND

To work around this problem, reduce the number of worksheets in the workbook that you want to share for editing.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about the limitations of using shared workbooks, click the Index tab in Microsoft Excel Help, type the following text 

   shared workbooks, limitations
and then double-click the selected text to go to the "Limitations of shared workbooks" topic.
