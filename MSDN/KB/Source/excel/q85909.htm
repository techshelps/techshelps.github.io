Excel: Duplicate or Unexpected Logon Dialog Box

 Last reviewed: November 4, 1994
Article ID: Q85909
The information in this article applies to:
  • Excel for Windows, versions 3.0 and 4.0
  • Excel for OS/2, version 3.0

Summary:

It is possible to configure Q+E to bring up a logon dialog for an external file-server upon choosing Open from the File menu.

If configured in this way, a duplicate or unexpected logon dialog box may appear when using the application or accessing Q+E through a macro statement. The problem can be solved by reinstalling Q+E and specifying a different default connection, or by directly modifying the QE.INI file.

More Information:

To specify an alternate default driver through Setup:

  1. Launch the Excel Setup program and specify the appropriate destination directory.

  2. Click Custom Installation and clear all selections except Q+E.

  3. Click the Q+E button and select the appropriate drivers for installation.

  4. Click OK and enter a different default connection (dBASEFile, TextFile, and ExcelFile will not create a logon dialog box.)

  5. Click OK and then click Setup.

To edit change the default driver directly within the QE.INI file:

  1. Open the file QE.INI in a text editor such as Notepad.

  2. In the [Q+E] section, edit the 'connect=' statement (without quotation marks) to refer to non-logon type (for example, connect=dBASEFile).

  3. From the File menu, choose Save and exit Notepad.
