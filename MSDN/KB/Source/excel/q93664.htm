Excel: Message in Status Bar Applies to Paste Not Paste Link

 Last reviewed: November 4, 1994
Article ID: Q93664
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0

SUMMARY

In Microsoft Excel, when you choose Copy from the Edit menu, Microsoft Excel displays the following message in the status bar: 

   Select destination and press ENTER or choose Paste
This message refers to the Paste command only, and does not apply if you are linking cells with the Paste Link command. If you press the ENTER key after you choose Paste Link, Microsoft Excel pastes the contents of your source cells over the links in your destination cells.

MORE INFORMATION

To link cells:

  1. Select your source cells.

  2. From the Edit menu, choose Copy.

  3. Select your destination cells.

  4. From the Edit menu, choose Paste Link.

  5. To remove the moving border from your source cells, press ESC.
