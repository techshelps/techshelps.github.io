XL5: WIN.INI Entries Required for Lotus cc:Mail

 Last reviewed: September 12, 1996
Article ID: Q148619
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SUMMARY

This article lists the entries that are required in the WIN.INI file in order to use Lotus cc:Mail in conjunction with Microsoft Excel. These entries are installed by both Lotus cc:Mail and MAPI-compliant Microsoft programs.

MORE INFORMATION

The sections and entries in the WIN.INI file that are necessary for Lotus cc:Mail to interact with Microsoft Excel are as follows: 

   [MAIL]

   MAPI=1
   SMI=1
   MAPIDLL=C:\WINDOWS\SYSTEM\MAPIVIM.DLL
   MAPIVIMLASTAB=2

   [LOTUS MAIL]

   APPLICATION=WMAIL
   PROGRAM=U:\WINAPPS\CCM\WMAIL\WMAIL.EXE SENDMAIL
   MAPIVIMLASTAB=0
The following is an explanation of the sections and entries listed above. 

   [MAIL]

   MAPI=
    Mandatory for Microsoft programs to recognize that a mail
    system is installed. The options are 1 (MAPI support) or 0 (no
    MAPI support). This line is installed by a MAPI-compliant Microsoft
    Program.

   SMI=
      Simplified Mail Interface.

   MAPIDLL=
      Denotes the path of the vendor independent Mail dynamic link library
      (DLL) file (usually Mapivim.dll). This is the mail system that Lotus
      cc:Mail uses.

   [Lotus Mail]

   APPLICATION=<MailAppName>
      Denotes the name of the mail program.

   PROGRAM=<MailAppExe>
      Points to the mail executable file.

   MAPIVIMLASTAB=<Parameter>
      Denotes the last address book used by Lotus cc:Mail.
