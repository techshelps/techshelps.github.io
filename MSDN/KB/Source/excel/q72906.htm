Color Palette Option Is Dimmed on Object-Linked Charts

 Last reviewed: November 7, 1994
Article ID: Q72906
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0
  • Microsoft Excel for OS/2, version 3.0

SYMPTOMS

After creating an embedded chart and then activating the chart by double-clicking it, the Color Palette command is dimmed on the Chart menu.

CAUSE

This occurs because a chart created this way is an object attached to the worksheet. It is not a separate file with its own color palette.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Excel listed above. This problem was corrected in version 5.0 of Microsoft Excel for Windows.

MORE INFORMATION

To customize the color palette, activate the worksheet and change the color palette in the Options menu. Customize the color and the change will be reflected in the chart.

Another workaround is to create the chart as a separate file with customized colors, then copy and paste it into the worksheet:

  1. Create the chart as a separate file.

  2. Customize its colors in the Color Palette.

  3. Copy and paste the chart onto the worksheet.

  4. Activate the worksheet and choose Color Palette from the Options menu.

  5. Select Copy Colors From and select the Chart file. This copies the color palette from the chart to the worksheet.

REFERENCES

"Microsoft Excel User's Guide." Version 3.0, pages 394, 558-560.
