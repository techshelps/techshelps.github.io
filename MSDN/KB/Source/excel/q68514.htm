Wrap Text Is Not Dynamic in Microsoft Excel 3.0 and 4.0

 Last reviewed: October 24, 1997
Article ID: Q68514

The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0

SUMMARY

The Format Alignment Wrap Text feature is not dynamic. After it has been set for the first time on a given cell, it will not reduce the row height to adjust to reductions in that cell's contents. If you add to the contents of the cell and press Enter, Microsoft Excel wraps at the existing column width and adjusts the row height.

MORE INFORMATION

The Wrap Text feature wraps the contents of a cell at a given column width, and expands the row height to accomodate the text. Microsoft Excel will not reduce the row height to accomodate the text.
