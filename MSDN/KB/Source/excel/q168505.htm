XL97: Redraw Problems Applying Page Setup in Many Sheets

 Last reviewed: February 3, 1998
Article ID: Q168505
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

After you click OK in the Page Setup dialog box, the screen may fail to update completely in Microsoft Excel, leaving the screen white except for the program title bar.

CAUSE

This problem may occur if all of the following conditions are true:

  • You select a large number of worksheets in a workbook (for example, 50 or more). 

        -and-
  • You change page settings using the Page Setup dialog box. 

        -and-
  • You use any of the following printer drivers: 

          Canon BJ600
      Hewlett-Packard IIIsi
      Hewlett-Packard IIIsi PostScript
      Hewlett-Packard 4M
      Hewlett-Packard 4M PostScript
      Hewlett-Packard Deskjet 870cse & 870cxi
      Hewlett-Packard Deskjet 850cse & 850cxi

WORKAROUND

To work around this problem, minimize the screen, and then restore the screen. To do this, use the following steps:

  1. Press ALT+SPACEBAR. Press N.

    The screen is minimized.

  2. Switch to Microsoft Excel to restore the screen.

    If you are using Microsoft Windows NT 4.0 or Microsoft Windows 95, click Microsoft Excel on the Task Bar to restore it. If you are using Microsoft Windows NT 3.51, press CTRL+ESC. On the Task List, click Microsoft Excel and click Switch To. The screen is redrawn correctly.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
