XL98: "Set Language" Add-In Specifies Language for Proofing

 Last reviewed: March 3, 1998
Article ID: Q181974
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SUMMARY

Microsoft Office 98 Macintosh Edition includes several foreign language proofing tools that can be used by Office programs to proof documents in a language other than English. To change the language Microsoft Excel 98 Macintosh Edition uses to check the spelling in your workbooks, you must first install the Set Language add-in, which is available in the Value Pack:Proofing Tools folder on the Office compact disc (CD).

This article explains how to change the language Excel uses to check the spelling in your worksheets.

NOTE: Proofing tools other than dictionaries are available in the Value Pack and can be used by the other Office programs. However, Microsoft Excel can use only the foreign language dictionaries that are installed by the Value Pack Installer program to check the spelling in worksheets.

MORE INFORMATION

To change the language Microsoft Excel uses to check the spelling in Excel worksheets, follow these steps.

NOTE: If you have already run the Value Pack Installer to install one or more of the available foreign language proofing tools, skip to step 4.

  1. Double-click the Value Pack Installer in the Value Pack folder on the Office CD.

  2. In the Value Pack Installer dialog box, click the triangle next to the Proofing Tools component and click foreign language proofing tools that you want to install. Click Install.

  3. When the installer has finished copying files, click Quit.

  4. Copy the Set Language add-in from the Value Pack:Proofing Tools folder on the Office CD to the Microsoft Office 98:Office:Excel Add- Ins folder on your computer.

  5. Start Microsoft Excel.

  6. On the Tools menu, click Add-Ins.

  7. In the Add-Ins dialog box, click Set Language, and then click OK.

    This step loads the Set Language add-in.

  8. On the Tools menu, click Language.

  9. In the Language dialog box, click any one of the languages for which you have installed proofing tools. Click OK.

    NOTE: If you select a language for which no proofing tools are installed, you receive the following error message:

    The dictionary for <foreign language> is not installed in the Microsoft Office 98:Shared Applications:Proofing Tools folder. You can install more dictionaries from the Value Pack.

    To continue with this language selected, click OK. Make sure you install the dictionary and restart Excel before you check spelling

    To return to the Language dialog box and select a different language, click Cancel.

  10. Click OK to the dialog box that prompts you to quit and restart Excel.

  11. Quit Microsoft Excel and restart it.

You can check the spelling of your worksheets in the language you specified in step 9.
