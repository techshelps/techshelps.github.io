XL98: Title Not Displayed in Data Validation Alert

 Last reviewed: February 2, 1998
Article ID: Q178505
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you enter a value into a cell that conflicts with the validation criteria that has been applied to a cell, the title of the alert message may not be displayed.

CAUSE

This problem will occur if the following conditions are true:

  • You enter a title on the Error Alert tab of the Data Validation dialog box.

    -and-

  • You enter a value that conflicts with the validation criteria established on the Settings tab of the Data Validation dialog box.

    -and-

  • The Office Assistant is not enabled.

STATUS

This is by design of Microsoft Excel.

REFERENCES

For more information about using Data Validation to display an error message, click the Index tab in Microsoft Excel Help, type the following text 

   data validation,messages
and then double-click the selected text to go to the "Display messages that prompt for data or explain incorrect entries" topic.
