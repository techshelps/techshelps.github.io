Excel: Cut/Copy Description in Clipboard

 Last reviewed: November 29, 1994
Article ID: Q25536
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0

SUMMARY

When performing a Cut or Copy while editing a worksheet in Microsoft Excel, the Clipboard only shows a description of the cell selection, for example "Cut (or Copy) 2R x 3C". The Clipboard does not display the values of the cells.

When executing a Cut or Copy command, Excel displays a marquee around the selection and a description of the action and selection size in the Clipboard. Both of these actions are designed to show that Excel is acting in a special manner.
