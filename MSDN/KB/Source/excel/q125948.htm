MSQUERY: Reserved Keywords in MS Query

 Last reviewed: February 2, 1998
Article ID: Q125948
The information in this article applies to:
  • Microsoft Query for Windows, version 1.0
  • Microsoft Query for Windows 95, version 2.0
  • Microsoft Query for the Macintosh, version 1.0

SUMMARY

The following keywords are reserved for use in Open Database Connectivity (ODBC) function calls. When you use these keywords to name a table or database you may receive unexpected results or cause Microsoft Query to generate the following error message: 

   Syntax Error in From Clause
Using these Keywords as field names may also produce unexpected results. Their use as field names is not supported. In addition to the keywords listed below, the individual ODBC drivers might include additional keywords. Check your driver documentation for complete listings of reserved words.
