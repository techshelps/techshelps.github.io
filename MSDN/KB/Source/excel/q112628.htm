XL5: Can't Activate MSQuery from Embedded OLE 2.0 Object

 Last reviewed: September 12, 1996
Article ID: Q112628

The information in this article applies to:

  • Microsoft Excel for Windows, version 5.0
  • Microsoft Query for Windows, version 1.0

SYMPTOMS

If you activate an embedded Microsoft Excel version 5.0 object from Microsoft PowerPoint 4.0 or Microsoft Word 6.0, Microsoft Query is loaded but is not activated when you choose Get External Data from the Data menu. Instead, Microsoft Query is loaded and is running minimized (as an icon). After you activate Microsoft Query, the Return Data To Excel command is not available on the File menu.

Also, when you load Microsoft Query directly from Microsoft Word 6.0 or PowerPoint 4.0, the Select Data Source dialog box does not appear.

WORKAROUND

To work around this problem, activate Microsoft Query before you start the server application.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Excel version 5.0. We are researching this problem and will post new information here as it becomes available.
