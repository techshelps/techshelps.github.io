XL98: Cannot Use Add-In Manager to Manage Some Add-Ins

 Last reviewed: March 16, 1998
Article ID: Q182499
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

Some of the add-ins that are installed with Microsoft Excel 98 Macintosh Edition cannot be enabled or disabled by using the Add-In Manager.

STATUS

This behavior is by design of Microsoft Excel 98.

MORE INFORMATION

The following add-ins are not visible in Add-In Manager: 

   Condiitonal Sum Wizard
   File Conversion wizard
   Internet Assistant Wizard
   Lookup Wizard
   Report Manager
   Solver
   Template Wizard
If you want to prevent any of these add-ins from loading automatically when you start Excel, you must remove them from the following folder on your computer: 

   Microsoft Office 98:Office:Excel Add-Ins folder.
NOTE: You cannot design a custom add-in that is loaded in the same manner as these add-ins.
