Excel: Edit Commands Don't Work with Letter Writer Plus 1.0

 Last reviewed: November 2, 1994
Article ID: Q65091

SUMMARY

When Letter Writer Plus version 1.0 is running, it disables the Edit menu commands in Microsoft Excel version 2.20. The menu items are not dimmed, but the commands will not function if they are chosen.

This problem disappears when Letter Writer Plus is closed. This problem has been confirmed by PowerUp Software, the manufacturer of Letter Writer Plus. For more information on this problem, contact PowerUp Software at the following address and phone number: 

   PowerUp Software
   2929 Campus Drive
   San Mateo, CA  94403
   (415) 345-9381 (product support line)
