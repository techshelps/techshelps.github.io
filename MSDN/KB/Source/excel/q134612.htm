XL7: "R1C1 is not a valid reference" in AutoTemplate Wizard

 Last reviewed: March 27, 1997
Article ID: Q134612
7.00 WINDOWS kbtool kberrmsg

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Step 3 of the AutoTemplate Wizard, you may receive the following error message after you specify a cell in the Cell Field: 

   R1C1 is not a valid reference

CAUSE

This error message will occur when you reference a cell using the R1C1 notation in the Cell Field in Step 3 of the AutoTemplate Wizard. An R1C1 reference is created when you do either of the following:

  • Type an R1C1 style reference in the field. For example: 

          'Sheet1'!R1C5

    -or-

  • Select a cell on a worksheet while the Reference Style option is set to R1C1.

RESOLUTION

When you specify the worksheet cell in Step 3 of the AutoTemplate Wizard, use any of the following methods to specify the A1 Reference Style:

Method 1: When you type the reference into the field, use the A1 notation. 

          For example, use this notation:

             'Sheet1'!$A$1
Method 2: When you select the cell with the mouse, set the Reference Style 
          on the General tab in the Options dialog box to A1 before
          you start the AutoTemplate Wizard.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
