Excel: Definitions Reversed for step_num and end_num on Pg 145

 Last reviewed: November 1, 1994
Article ID: Q57383

SUMMARY

The definitions for the arguments end_num and step_num for the FOR function on page 145 of the "Microsoft Excel: Functions and Macros" manual for Excel version 2.2 are reversed. This is a documentation error.

The definitions should read as follows: 

   step_num  A value that is added to the counter. You set the counter
             to a start value, and each time the loop is executed, an
             increment is added to the counter.

   end_num  The number that stops the loop.
