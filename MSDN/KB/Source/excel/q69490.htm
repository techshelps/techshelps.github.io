Custom Data Form Dialog Title Does Not Appear

 Last reviewed: November 3, 1994
Article ID: Q69490
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2, versions 2.2 and 3.0

SUMMARY

By default, the data form and the custom data form are movable dialog boxes and, therefore, entries in the first row of the text column of the dialog box definition table will be ignored. The title for the custom data form will be the same as the name of the worksheet on which it is contained. There is no way to give it a different name.

MORE INFORMATION

A custom dialog box in Microsoft Excel for OS/2 or Microsoft Excel for Windows version 3.00 can be made into a movable dialog box if a description or title is included in the dialog box definition table. This description needs to be added to the first-row text column of the dialog table. This has the effect of turning an unmovable dialog box into a dialog box that can be moved by clicking and dragging the title bar.
