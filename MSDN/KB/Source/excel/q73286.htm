Creating Unstacked Area Charts in Excel

 Last reviewed: November 2, 1994
Article ID: Q73286

SUMMARY

To create an unstacked area chart in Microsoft Excel versions 2.x, select Main Chart from the Format menu and remove the check from the Stacked check box.

In Microsoft Excel version 3.0, the Stacked option is unavailable as a check box. To create an unstacked Area chart in Excel version 3.0, choose Format from the menu bar, select Main Chart, verify that the Main Chart Type selected is Area, and choose the first graph under Data View.

MORE INFORMATION

When using the Unstacked option for an area chart, Excel graphs each series of data on the Category (X) axis rather than graphing one series on top of another.

Note that area charts have no unstacked options under the Gallery menu. However, to create unstacked charts with any of the other chart types (bar, column, and so on), you can use the Gallery menu.
