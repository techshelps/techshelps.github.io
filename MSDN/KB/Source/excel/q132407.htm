MXL5: MS Excel Hangs When You Use ATM 3.8.1 Type 1 Fonts

 Last reviewed: March 13, 1998
Article ID: Q132407
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a

SYMPTOMS

In Microsoft Excel 5.0, when you apply fonts to data on your worksheet, if you are using Adobe Type Manager (ATM) version 3.8.1, your system may hang (stop responding).

CAUSE

This problem may occur in the following situations:

  • If you use the Format Cells dialog box, select the Font tab, and select an ATM font, Microsoft Excel may hang while the font is being displayed in the Preview box. This problem may occur if you enter a size in the Size box that is not available in the Size list for that font.
  • If you click Print Preview on the File menu after applying an ATM font to cells on your worksheet, Microsoft Excel may hang.
  • If you use the Style dialog box to modify a style, add the changes to the style, and then click OK, your system may crash when you drag your mouse pointer over the Font box on the Formatting toolbar.
  • When you position the mouse pointer over the Font box on the Formatting toolbar, Microsoft Excel may hang.

RESOLUTION

To correct these problems, you should obtain Microsoft Excel version 5.0a and Adobe Type Manager version 3.8.2.

STATUS

This problem is caused by an incompatibility between Microsoft Excel and Adobe Type Manager. This problem no longer occurs in Microsoft Excel version 5.0a for the Macintosh and Adobe Type Manager version 3.8.2.
