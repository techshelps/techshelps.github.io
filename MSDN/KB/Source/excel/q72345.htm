Edit Form Buttons Are Available When Select Distinct Is On

 Last reviewed: November 2, 1994
Article ID: Q72345

SUMMARY

When Select Distinct is turned on in Q+E, no editing is allowed in the database. However, the New, Dup, and Restore buttons in the Edit Form dialog box are available in Q+E for Microsoft Windows.

This behavior occurs only in Q+E for Windows. When Distinct is selected in Q+E for OS/2, all buttons in the Data Form are dimmed.

MORE INFORMATION

The following information refers only to the Windows version of Q+E.

The Dup button in the Edit Form dialog has no effect when Select Distinct is turned on.

The New and Restore buttons blank the fields within the dialog box. Even though the fields are empty, and the flashing bar cursor will appear in a field, attempting to type any text into the fields will just make the computer beep.

Microsoft has confirmed this to be a problem in Q+E for Windows. We are researching this problem and will post new information here as it becomes available.

References:

"Microsoft Excel Q+E User's Guide", version 3.0, pp. 39-41.
