File Conversion Wizard for Microsoft Excel Available

 Last reviewed: March 19, 1998
Article ID: Q148243
5.00 5.00c 7.00 WINDOWS kbtool kbusage

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for Windows, version 5.0, 5.0c
  • Microsoft Excel for Windows NT, version 5.0

SUMMARY

The File Conversion Wizard (Fileconv.xla) is one of several add-in wizards being made available in order to enhance your use of Microsoft Excel. The File Conversion Wizard helps you to convert Lotus 1-2-3 and Quattro Pro files (as well as many other formats) to Microsoft Excel files in a single step, rather than having to convert these files one at a time.

This add-in can be obtained on the Internet from the following location: 

   http://www.microsoft.com/excel
The add-in mentioned in this article is provided "as is" and Microsoft does not guarantee that it can be used in all situations. Although Microsoft Product Support (PSS) Engineers can help with the installation and existing functionality of this add-in, they will not modify the add-in to provide new functionality.

NOTE: This add-in was not designed for use with versions of Microsoft Excel for the Macintosh.

For information about using this add-in with Microsoft Excel for Windows NT, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q148140
   TITLE     : File Conversion Wizard Does Not Run Under MS Excel for NT
This add-in is offered in response to user input, as a demonstration of the Microsoft Excel development team's ongoing commitment to making Microsoft Excel easier to use.

MORE INFORMATION

The File Conversion Wizard (Fileconv.xla) will convert files of other types into the Microsoft Excel format. A batch conversion can be performed on files in the same location. Below is a detailed description of each step in the wizard, followed by an example.

Step 1 of the Wizard

Determine the location and type of the files to be converted.

In this step, select the folder where the files are located and the file type. If you need to, you can change the folder for the selected drive.

All the files in the current location will listed in the wizard.

Step 2 of the Wizard

To add the files be converted to the Files To Convert list, do either of the following:

  • To select a single file, click the file, and then click the ">" button.

    -or-

  • To select all the files, click the ">>" button.

Note that you can remove files from the list of Files To Convert by clicking the "<" button (to remove a single file) or the "<<" button (to remove all files).

Step 3 of the Wizard

Once the files are converted, they will be saved.

To specify the location for where the files will be saved, select the drive from the drop-down list, and change the current folder or create a new folder as desired.

Example of the File Conversion Wizard

  1. Obtain this add-in from one of the following sources: 

        - The Microsoft Internet site, http://www.microsoft.com/msexcel
    - Microsoft Network, Go MSEXCEL
    - Microsoft TechNet Compact Disc

    Once you've downloaded the add-in file, see the Readme.txt file for instructions about how to install it.

  2. Start Microsoft Excel.

  3. On the Tools menu, click File Conversion Wizard.

  4. In Step 1 of the wizard, switch to the folder that contains the *.dbf example files used with Microsoft Query. 

          For this
      version of

          MS Excel    Use this folder
      ---------------------------------------------------------------

      5.0         Windows\Msapps\Msquery
      7.0         Program Files\Common Files\Microsoft Shared\Msquery

   In the Files Of Type drop-down list, select "dBase (*.dbf)." Click
   the Next button.


  5. In Step 2 of the wizard, click the Select All (">>") button to move all
   the files to the Files To Convert list. Click the Next button.


  6. In Step 3 of the wizard, determine where you want the converted files to
   be saved and select that drive and folder. (Note that there is a button
   to create a new folder.) Click the Finish button.

    

When the conversion process is completed, there will be a File Conversion
Wizard report generated documenting the results and any problems that may
have occurred during the conversion process.
