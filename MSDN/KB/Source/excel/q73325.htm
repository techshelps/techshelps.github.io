Selecting Records Based on Multiple Conditions in Same Field

 Last reviewed: November 2, 1994
Article ID: Q73325

SUMMARY

It is possible to select records based on multiple conditions within the same field in Q+E.

MORE INFORMATION

To select records based on multiple conditions within the same field:

  1. In the Query window, select a value in the field you want to constrain.

  2. From the Select menu, choose Add Condition, select an operator, and choose OK.

  3. Select a value in the same field.

  4. From the Select menu, choose Add Condition. Note that the Add Condition dialog box now has a Connection option group, with buttons labeled And and Or.

  5. Choose Or in the Connection group, select an operator, and type the new value in the Value box.

  6. Repeat steps 3-5 to add as many condition values as desired.
