XL97: Borders Below Row 65536 Do Not Print

 Last reviewed: January 15, 1998
Article ID: Q157099
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

When you print a worksheet in Microsoft Excel 97, borders below row 65536 do not print.

CAUSE

This occurs because Microsoft Excel cannot print borders along the bottom limits of a worksheet. Row 65536 is the bottom-most row of a worksheet in Microsoft Excel 97.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Worksheets in Microsoft Excel 97 can contain up to 65536 rows. If you apply a border to the bottom of cells in row 65536, the borders will not print.

REFERENCES

For more information about applying borders in Microsoft Excel, click the Index tab in Microsoft Excel Help, type the following text 

   borders, in cells
click Display, click "Apply borders to cells", and then click Display.
