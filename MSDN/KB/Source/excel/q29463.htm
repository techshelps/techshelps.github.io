Excel: Chart Created with Preferred Format Has Incorrect Text

 Last reviewed: November 2, 1994
Article ID: Q29463

SUMMARY

If you change the text labels in a Microsoft Excel chart and then choose Set Preferred from the Gallery menu to create multiple charts with the same format, the text labels will not update when a new chart with different values is created.

If numeric values have been attached to individual data points or the text has been edited to display different decimal values or numerical formats, these values are converted to text and therefore do not give the "correct" result when new data is plotted.

For example, a chart formatted to be Gallery Pie 6 displays two digits to the right of the decimal. If these values are edited to display no decimals, these values become text and will not be correct for a new chart with different values.

There is no way to automatically create a pie chart without the two digits to the right of the decimal.

For more information regarding the Set Preferred option, see pages 52 and 384 in the "Microsoft Excel Reference" version 2.2 manual. For Excel version 1.50, see page 167 in the "Microsoft Excel User's Guide."
