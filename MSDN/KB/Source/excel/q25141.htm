Excel: Blank Worksheet Opened When Resume Excel Launched

 Last reviewed: November 2, 1994
Article ID: Q25141

SUMMARY

In versions 1.04, 1.06, and 1.50 of Excel, the Resume Excel file must be located in the System Folder to function properly. In versions 1.00 and 1.03, the Resume Excel file can be stored anywhere on the startup volume.

This change was implemented for the support of networks.

If the Resume Excel file is located anywhere other than the System Folder in versions 1.04, 1.06, and 1.50, a blank worksheet will appear when the file is launched.

Excel 2.20 replaced the Resume Excel file with Workspaces, which can be stored anywhere.
