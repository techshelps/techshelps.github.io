Using Multiple Fonts in Headers and/or Footers in Excel

 Last reviewed: November 2, 1994
Article ID: Q72542

SUMMARY

If you use alignment codes in a header or footer, you must also apply the desired font to each aligned segment. Each segment defaults to the normal font unless otherwise specified. This includes fonts, font sizes, or styles (bold, italic, etc.).

MORE INFORMATION

Example

The goal is to display the current time in the bottom left corner of the page, the name of the worksheet in the bottom center and the date in the bottom right. All three segments should be formatted using Helvetica 15.

  1. From the File menu, choose Page Setup.

  2. In the Footer box, type the following: 

          &L&"HELV"&15&T&C&"HELV"&15&F&R&"HELV"&15&D

    (Note that you could also specify a different font for each segment, if desired. Also, the font should match the font name located under the Fonts option of the Format menu and should be in quotation marks.)

  3. Choose OK.

  4. From the File menu, choose Print Preview to view the footer.
