XL5 Err Msg: "Call to Undefined Dynalink" During Setup

 Last reviewed: September 12, 1996
Article ID: Q112682
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SUMMARY

When you set up Microsoft Excel version 5.0 for Windows, you may receive the error message: 

   Application Error
   Call to Undefined Dynalink
If you receive this error message, the Setup program will terminate after you choose the Close button.

CAUSE

This error may be caused by if you have an early version of the CTL3D.DLL file on your computer.

WORKAROUND

To work around this problem, do either of the following:

  • Do the following to rename or remove all versions of CTL3D.DLL from your system:

    a. Use File Manager to search for all instances of CTL3D.DLL on your 

          hard disk.

    b. Delete or rename earlier versions of the file.

    c. Restart Microsoft Windows, and attempt the installation again.

    -or-

  • If you restart Microsoft Windows, you may be able to run Microsoft Excel Setup even if an earlier version of CTL3D.DLL exists on your computer (as long as you do not run any other programs before starting Microsoft Excel Setup). However, this workaround is not as certain to work as the first workaround.

MORE INFORMATION

The Setup program included with Microsoft Excel uses the CTL3D.DLL dynamic link library when displaying certain dialog boxes. If the CTL3D.DLL file is not located on your computer, Microsoft Excel Setup will install a version of the file on your hard drive.

If an earlier version of CTL3D.DLL is already present on your hard drive and is being used by another program that is running or that was run earlier in this Windows session, the old CTL3D.DLL file may not be compatible with Microsoft Excel Setup. If this is the case, the "Application Error -- Call to Undefined Dynalink" error message will appear and the Setup program will be terminated.
