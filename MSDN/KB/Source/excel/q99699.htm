EIS Builder 1.0a Maintenance Upgrade

 Last reviewed: September 12, 1996
Article ID: Q99699
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0

SUMMARY

If you are using the Microsoft Excel EIS Builder version 1.0 for Windows, you can order the 1.0a maintenance upgrade update directly from Microsoft, beginning May 17, 1993. This maintenance release is available to all registered users of EIS Pak version 1.0 for Windows at no charge. Note that if you are using the EIS Pak for the Macintosh, there is no need to order the upgrade since the Microsoft Excel EIS Builder version 1.0a is shipped with the Macintosh product.

MORE INFORMATION

The EIS Builder for Windows version 1.0a is compatible with the Microsoft Open EIS Pak for the Macintosh and fixes some minor problems with the earlier version. In addition to these changes, several important topics have been added to the README.TXT file, including information on network installation and cross-platform (Microsoft Windows and Apple Macintosh) development.

Applications with the 1.0 version are compatible with the 1.0a version, and will run without modifications.

Version information on your software is available by choosing the About command from the EIS menu. You can also look at the date of your AUTHOR.XLA file to determine your version: 

    Microsoft Excel EIS Builder 1.0
    -------------------------------

    AUTHOR.XLA     11/4/92     212746 bytes

    Microsoft Excel EIS Builder 1.0a
    --------------------------------

    AUTHOR.XLA     2/11/93     217116 bytes
To order your free version 1.0a update, call (800) 227-4679 or Microsoft Consumer Sales at (800) 426-9400.
