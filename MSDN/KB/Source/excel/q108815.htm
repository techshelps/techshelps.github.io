XL5: Opening Lotus 1-2-3 Version 4.0 for Windows Files

 Last reviewed: January 24, 1997
Article ID: Q108815
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SUMMARY

Microsoft Excel version 5.0 does not ship with a Lotus 1-2-3 WK4 file converter. To open a file in Microsoft Excel that you created in Lotus 1-2- 3 version 4.0 for Windows, you must do one of the following:

  • Obtain the Lotus 1-2-3 WK4 File Converter from Microsoft

    For additional information about obtaining this file converter, please see the following article(s) in the Microsoft Knowledge Base: 

          ARTICLE-ID: Q122583

          TITLE     : WE1130: Lotus 1-2-3 WK4 File Converter

   -or-


  • Save the file in the Lotus 1-2-3 version 3.0 file format.

    

MORE INFORMATION

 


In Microsoft Excel, you can open Lotus 1-2-3 files, and you can save
Microsoft Excel files as Lotus 1-2-3 files in the .WKS, .WK1, and .WK3 file
formats.


Lotus 1-2-3 Release 4 for Windows features a new file format and saves
files with a default .WK4 filename extension.


To save files in the Lotus 1-2-3 version 3.0 format, in Lotus 1-2-3 version
4.0 for Windows, do the following:



   1. From the File menu, choose Save As.

   2. In the File Type box, select 1-2-3 (.WK3).

   3. Type a filename and then choose OK.
