XL: Opening DBF4 Files That Have More Than 128 Fields

 Last reviewed: March 27, 1997
Article ID: Q109174
3.00 4.00 4.00a 5.00 5.00c 7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0, 5.0c
  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

Microsoft Excel versions 4.0a, 5.x, and 7.0 support DBF4 files with more than 128 fields for export (that is, a Microsoft Excel worksheet saved as DBF4 file), and versions 5.x and 7.0 will also import DBF4 files of more than 128 fields that have been exported from other applications.

DBF4 files created from other applications with more than 128 fields will only show the first 128 fields when opened in Microsoft Excel version 4.0a and earlier.

MORE INFORMATION

The table below summarizes the versions of Microsoft Excel with which you can use DBF4 files. Note that DBF4(import) represents DBF4 files created in other applications and opened in Microsoft Excel. DBF4(export) signifies Microsoft Excel worksheets saved in DBF4 file format. 

   Microsoft Excel      DBF4(import)         DBF4(export)
   Version              with >128 fields     with >128 fields
   ----------------------------------------------------------

   Version 3.0               No                    No
   Version 4.0               No                    No
   Version 4.0a              No                    Yes
   Version 5.x               Yes                   Yes
   Version 7.0               Yes                   Yes
