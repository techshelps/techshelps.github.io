Installing Excel 4.0 over Excel 3.0

 Last reviewed: November 29, 1994
Article ID: Q85492

The information in this article applies to:

  • Microsoft Excel for Windows, version 4.0

SUMMARY

This article contains general information on how you install Microsoft Excel for Windows, version 4.0, over Microsoft Excel for Windows, version 3.0.

MORE INFORMATION

When you install version 4.0 over an existing installation of version 3.0, a number of files are replaced or removed; no directories, however, are erased. User files are not erased unless they were given the same filename as one of the Microsoft Excel files that is erased (see below for a list of files that are removed during Setup).

Following are some special considerations when you set up Microsoft Excel over a previous version:

For more information on how you set up Microsoft Excel on a network see the NETWORK.TXT file located on the Excel Setup disk (disk 1).

If Q+E was installed in the previous version it may be necessary for you to remove the QEXXX.DLL files from the Windows\System directory. For complete details on this see the QEREADME.TXT file located in the Excel directory.

If you are using SQL Server version 4.2 or greater, the following two files should be removed from the Excel Directory after you install Microsoft Excel and Q+E: 

   W3DBLIB.DLL
   DBNMP3.DLL
Any add-ins that you have moved from the LIBRARY subdirectory to the XLSART subdirectory are not updated in the XLSTART subdirectory. After you install Microsoft Excel you need to copy the new versions of the add-ins to your XLSTART subdirectory.

For more information and changes to the documentation see the README.TXT file located in your Excel directory.
