XL98: Angled Text May Be Printed Incorrectly

 Last reviewed: February 2, 1998
Article ID: Q175679
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel 98 Macintosh Edition, when you print a worksheet with any cells that contain angled text, the angled text may extend into an adjacent merged cell or past a page break in the printed output.

Note that this problem does not occur when you view the worksheet in print preview, or when you print or view the same worksheet in print preview in Microsoft Excel 97 for Windows.

CAUSE

This problem occurs because of the way in which Microsoft Excel 98 Macintosh Edition processes information in cells when you print the worksheet to a PostScript printer. Angled text is not always properly processed; as a result, the text may appear incorrectly when you print it.

WORKAROUND

To work around this problem, widen any cell that contains angled text so that it is wide enough to display all of the text without allowing the text to extend into an adjacent merged cell or past a page break.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

In Microsoft Excel 97 and Microsoft Excel 98 Macintosh Edition, you can angle text in a cell by following these steps:

  1. Select the cell. On the Format menu, click Cells. Click the Alignment tab.

  2. In the Orientation section of the dialog box, use the controls to angle the text from 90 to -90 degrees. Then, click OK.

You can also merge two or more cells by following these steps:

  1. Select the cells. On the Format menu, click Cells. Click the Alignment tab.

  2. Click to select the Merge Cells check box. Then, click OK.

Or, you can select the cells and click the Merge And Center tool on the Formatting toolbar.

If a cell contains angled text, the text may extend out of the cell into an adjacent cell. If the adjacent cell is a merged cell, or if the text extends past a page break, the text may be cut off when you view the worksheet in print preview. However, when you print the worksheet, the angled text may extend into the adjacent cell or past a page break.
