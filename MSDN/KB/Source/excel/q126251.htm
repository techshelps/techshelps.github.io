MSQUERY: Can't Manually Edit Query Preferences File

 Last reviewed: February 2, 1998
Article ID: Q126251
The information in this article applies to:
  • Microsoft Query for the Macintosh, version 1.0

SUMMARY

In Microsoft Query for the Macintosh, you are unable to manually edit the Query Preferences file. However, you can use the Microsoft Query user interface to modify most of the configuration settings listed in this file.

MORE INFORMATION

Microsoft Query uses the Query Preferences file to store configuration options, such as whether to run the program maximized, and whether to enable or disable the display of the Cue Cards tutorial. The Query Preferences file is located in the Preferences folder of your System Folder.

For additional information about editing the Windows Msquery.ini file, please see the following article in the Microsoft Knowledge Base:
