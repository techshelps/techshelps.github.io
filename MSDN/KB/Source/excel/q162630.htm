XL97: Virus Search Add-In Was Not Included with Microsoft Excel

 Last reviewed: February 12, 1998
Article ID: Q162630
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, the "What's new with security?" topic in online Help includes information about a Virus Search add-in. The Help topic states that the add-in detects and removes all known macro viruses from your workbooks.

The Virus Search add-in was not available at the time Microsoft Excel 97 was released. However, you can download the Virus Search add-in from the following Microsoft Web site: 

   http://www.microsoft.com/excel/productinfo/vbavirus/add_in.htm
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
