Excel: Scenario Manager Fails

 Last reviewed: September 12, 1996
Article ID: Q105875
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a

SUMMARY

If the reference for changing cells in the Scenario Manager exceeds 255 characters, the reference will be truncated and may cause all of the Scenario Manager operations (Show, Add, Delete, Edit, and Summary) to fail. You will not receive an error message alerting you to this failure.

MORE INFORMATION

If any of the changing cells in a scenario has a defined name, Scenario Manager will replace the cell reference with its defined name. If lengthy names are used, the changing cells reference can exceed 255 characters. When this happens, the reference will be truncated and the only buttons in the Scenario Manager dialog box that will work are the Close and Help buttons. This occurs because the changing cells reference was longer than 255 characters and was truncated. When this happens, you will not receive an error message asking you to change the reference.

Workarounds

To work around this problem, either use a contiguous range of changing cells or use shorter defined names.

REFERENCES

"Microsoft Excel User's Guide 2," version 4.0, pages 68-73
