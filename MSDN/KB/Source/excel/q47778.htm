Data Form Displays General, Not Number, Formats

 Last reviewed: November 30, 1994
Article ID: Q47778
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, 4.0, 5.0

SUMMARY

When using the Data Form, values that have been formatted using custom number formats display in General format, instead of their custom number format.

MORE INFORMATION

Data Form displays the data from the database in the General format. There is currently no provision for displaying data in formatted form. User-defined custom data forms also have this limitation.
