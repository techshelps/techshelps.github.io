Excel: Use Arrow Keys to Select Different Items in a Chart

 Last reviewed: November 2, 1994
Article ID: Q60474

SUMMARY

To select different items in a Microsoft Excel chart, cycle through the items using the arrow keys.

On Excel charts, the arrow keys are used to switch between different chart items as displayed in the left side of the formula bar. The following table shows how to use the arrow keys in a chart: 

   Key              Selection
   ---              ---------

   UP ARROW         First item of next type
   DOWN ARROW       Last item of previous type
   RIGHT ARROW      Next item
   LEFT ARROW       Previous item
The UP and RIGHT ARROW keys go to the next chart item; the DOWN and LEFT ARROW keys go to the previous chart item.
