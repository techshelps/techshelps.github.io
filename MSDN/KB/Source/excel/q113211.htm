XL5: Underlining May Leave Gap with Accounting Number Format

 Last reviewed: September 12, 1996
Article ID: Q113211
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

In Microsoft Excel, if an accounting number format is applied to a cell, and either a single or double underline (not to be confused with single accounting or double accounting underline) is applied to that cell, a gap will be displayed to the left of the number, breaking the underline in that position. This behavior does not occur if you choose an Accounting underline format.

The underline should be a continuous line underlining the entire contents of the cell.

WORKAROUND

To work around this problem, do the following:

  1. Select the cells that need an underline.

  2. From the Format menu, choose Cells.

  3. Select the Font tab.

  4. Under Underline, select either Single Accounting or Double Accounting.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Excel for Windows version 5.0c.
