Cannot Load Files from Novell Network Drives

 Last reviewed: November 2, 1994
Article ID: Q67772

SUMMARY

Novell networks are not supported under Q+E 2.5. The File Open command may show all the network drives available, but you can select only the local drives. Q+E will allow you to save files to a Novell drive.

This problem does not occur with Q+E version 3.0.

Workaround

Copy the file from the network drive to a local drive. You can use File Manager, shell out to DOS, or use Excel.

From Excel

  1. Choose File Open.

  2. Connect to the network drive that contains the file.

  3. Open the file.

  4. From the File menu, choose Save As and select one of the local drives.

  5. Switch back to Q+E.

You can then access the file with Q+E.

Microsoft is researching this problem and will post new information here as it becomes available.
