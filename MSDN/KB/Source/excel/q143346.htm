XL: "Run-time Error 9" Using Conditional Sum Wizard

 Last reviewed: February 2, 1998
Article ID: Q143346
The information in this article applies to:
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a

SYMPTOMS

In the versions of Microsoft Excel listed above, when use the Conditional Sum Wizard, you may receive the following error message: 

   Run-time error '9':
   Subscript out of range
You can either click OK or Help. If you click OK, you immediately exit the Conditional Sum Wizard. If you click Help, you receive a Visual Basic Reference Help screen that tells you that you have referenced a nonexistent array element or collection number.

CAUSE

You may receive the error message if in Step 2 of the Conditional Sum Wizard you click the "no condition" item and then click "Remove Condition."

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem does not occur when you use the Conditional Sum Wizard included in Microsoft Excel 97 and Microsoft Excel 98 Macintosh Edition.

MORE INFORMATION

The Conditional Sum Wizard is a Microsoft Excel Add-in that is only available from the following locations:

  • Microsoft Network
  • The Microsoft Internet site, http://www.microsoft.com/msexcel
  • Microsoft TechNet

The Conditional Sum Wizard is not installed by the versions of Microsoft Excel listed above. It is included with both Microsoft Excel 97 and Microsoft Excel 98 Macintosh Edition; however, the problem described above does not occur when you use the Conditional Sum Wizard included with these programs.
