Pie Chart Prints as Vertical Line to PostScript Printer

 Last reviewed: November 4, 1994
Article ID: Q76611
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0

SUMMARY

In Microsoft Excel, when you print a two-dimensional (2-D) pie chart that has only one data point, to a postscript printer, the printout contains only a vertical line marking the radius. This problem does not occur with any of the three-dimensional (3-D) pie charts or two-dimensional pie charts with more than one data point.

WORKAROUND

To work around this behavior, copy the chart as a picture and paste it to the worksheet as a bitmap.

REFERENCES

"Microsoft Excel User's Guide," version 3.0, pages 408-411
