Excel: Zen Software's Excel Reporter

 Last reviewed: November 3, 1994
Article ID: Q79181
Summary:

Zen Software's Excel Reporter 2.1 is a Windows-based application that creates columnar listings, mailing labels, and forms from Microsoft Excel 3.0 database spreadsheets. Excel Reporter can select fields from multiple Excel tables, filter records with logical operators, and add calculated fields. DDE is supported as are formatting features such as colors, shading, and fonts.
