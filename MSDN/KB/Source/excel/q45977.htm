Removing the Last Condition with Edit Undo in Q+E

 Last reviewed: October 31, 1994
Article ID: Q45977

SUMMARY

When using Select Add Condition to add criteria to determine which records to show, you can use Edit Undo to remove the last condition applied to the current selection.

You can continue to select Edit Undo as long as there are selection conditions applied to the file.

Select Reset removes all of the conditions that have been applied to the file.
