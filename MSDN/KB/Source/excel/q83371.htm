Excel: 11"x14" Paper Does not Appear in List of Paper Sizes

 Last reviewed: November 4, 1994
Article ID: Q83371
Summary:

When using dot-matrix printer drivers based on the Windows 3.1 Universal Printer Driver, no entry specifically designating 11-by-14-inch paper will appear in the list of paper sizes in the File Page Setup dialog box in Microsoft Excel for Windows version 3.0. However, there is a placeholder for the paper size that appears to consist of a vertical bar character and an apostrophe or a period. If this placeholder is selected, Excel will use 11-by-14-inch paper.

More Information:

The text description for 11-by-14-inch paper in the Universal Printer Driver is 'US Standard Fanfold 11"x14"'. This text description is too long to fit in Excel 3.0's paper size listbox. As a result, the text does not appear correctly in the list.

This has been confirmed to be a problem in Microsoft Excel 3.0. This problem was corrected in Excel 4.0.
