Excel: Cell Formats Not Saving When Using WKS or WK1 Format

 Last reviewed: November 2, 1994
Article ID: Q64358
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 1.x, 2.2, 3.0, 4.0, 5.0

In Microsoft Excel, formatting (such as bold and italic text) and row height changes will not be retained when a document is saved in the WKS or WK1 file format.

If a document is imported into Excel from Lotus 1-2-3 in WKS or WK1 format, Excel continues to save that imported file in WKS or WK1 format until it is changed to Excel's Normal format.

To save an imported filed in Excel's Normal format, do the following:

  1. From the File menu, choose Save As.

  2. Choose the Options button.

  3. Select Normal and click OK.

  4. Name the document and click the Save button.
