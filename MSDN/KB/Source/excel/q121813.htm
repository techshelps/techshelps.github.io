Microsoft ODBC Drivers Fulfillment Kit

 Last reviewed: September 12, 1996
Article ID: Q121813
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SUMMARY

The Microsoft ODBC Drivers version 2.0 (ODBC Kit) is available from Microsoft as part #273-054-030. The kit includes drivers for the following databases: 

   Driver                     Database or file version
   -----------------------------------------------------

   Btrieve                    5.1 and 6.x
   dBASE                      III, IV
   Microsoft Access           1.0, 1.1, 2.0
   Microsoft Excel            3.0, 4.0, 5.0
   Microsoft FoxPro           2.0, 2.5, 2.6
   Paradox                    3.5, 4.0, 4.5
   Text                       n/a
   ORACLE                     6.0, 7.0
   SQL Server                 4.2x
For additional information, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q125235
   TITLE     : INF: ODBC Driver Pack 2.0 Fulfillment Kit Q&A

   ARTICLE-ID: Q124815
   TITLE     : INF: Explanation of the Two Available MS Access Drivers
The Microsoft ODBC Drivers version 1.0 (ODBC Kit) is available from Microsoft as part #273051-025. The kit includes drivers for the following databases: 

   Driver                     Database or file version
   -----------------------------------------------------

   Btrieve                    5.1
   dBASE                      III, IV
   Microsoft Access           1.0, 1.1
   Microsoft Excel            3.0, 4.0
   Microsoft FoxPro           2.0, 2.5
   ORACLE                     6.0
   Paradox                    3.5
   SQL Server                 Microsoft  1.1 through 4.2
                              Sybase     4.0 through 4.2
   Text files                 n/a

MORE INFORMATION

Microsoft Excel version 5.0 for Windows ships with ODBC drivers for Microsoft Access, dBASE, SQL Server, Paradox, and Microsoft FoxPro. The ODBC Kit also provides these drivers plus additional drivers, some of which were not available at the time Microsoft Excel version 5.0 was released. Newer versions of this kit will be released as other ODBC drivers become available.
