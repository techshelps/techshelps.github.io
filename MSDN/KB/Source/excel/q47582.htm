Excel: Custom Dialog Box Option Buttons Don't Dim

 Last reviewed: November 2, 1994
Article ID: Q47582

SUMMARY

Page 193 in the "Microsoft Excel Arrays, Functions, and Macros", version 1.5 manual, and page 176 in the "Microsoft Excel Functions and Macros" version 2.2 manual incorrectly state that if the Initial/Result column for an Option Button Group (item 11) contains #N/A, the buttons are grayed.

This information is incorrect. It is not possible to dim (or make gray) the Option Buttons in a Custom dialog box.

If the Initial/Result column for the Option Button Group (item 11) contains #N/A, no buttons are selected when the dialog box is displayed.

This feature is under review and will be considered for inclusion in a future release.
