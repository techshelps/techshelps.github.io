XL: Quattro Pro Defined Names in Microsoft Excel Formulas

 Last reviewed: December 18, 1996
Article ID: Q124930
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel 97 for Windows

SUMMARY

You can use range names created in Quattro Pro in Microsoft Excel. However, any formula referring to a defined name will not show the defined name in the formula bar--only the reference will be displayed. In order to update the formula to refer to a defined name, choose Name from the Insert menu, and then choose Apply.

MORE INFORMATION

The user-defined range names exist in the file but are not applied to formulas. Applying the name to the formula does not affect the calculation of the formula. While this behavior may be confusing, it does not affect the performance of the spreadsheet in Microsoft Excel.
