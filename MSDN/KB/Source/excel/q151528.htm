MXL5: Disk Contents: MS Excel for Mac 15-Digit Patches

 Last reviewed: February 2, 1998
Article ID: Q151528
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a

SUMMARY

This article contains the disk contents of the three different disk sets that are available for the Microsoft Excel for the Macintosh 15-digit patch.

MORE INFORMATION

There are three different disk sets for the Microsoft Excel for the Macintosh 15-digit patch, one for a Power Macintosh installation, one for a FAT installation, and one for a 68K installation. The following tables list the disk contents for each of these three disk sets.
