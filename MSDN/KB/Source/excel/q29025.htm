Excel Automatically Updating Links

 Last reviewed: November 2, 1994
Article ID: Q29025
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0, 5.0
  • Microsoft Excel for OS/2 versions 2.2, 2.21, 3.0

There are two ways to keep Microsoft Excel from automatically updating the dependent sheet when a supporting sheet is renamed. They are as follows:

  1. Close the dependent sheet before you rename or save the supporting sheet under a new name.

  2. After renaming the supporting sheet and changing all the references in the dependent sheet, use the Links command from the File menu on the dependent sheet. NOTE: the Links command is found under the Edit menu if you are using Microsoft Excel version 5.0. All occurrences of the new sheet name in the references can be found and automatically replaced with the old sheet name.
