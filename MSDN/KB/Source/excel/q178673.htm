XL98: Diagonal Cell Border May Disappear

 Last reviewed: February 2, 1998
Article ID: Q178673
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel 98 Macintosh Edition, when you try to format a cell with a diagonal border, the border may not appear, although the preview looks correct in the Format Cells dialog box.

CAUSE

You may experience this problem when the following conditions are true:

  • You format a cell with both diagonal borders. 

        -and-
  • You choose a different line style for the border. 

        -and-
  • You change one of the borders using one of the buttons under Border.

WORKAROUND

To change the line style, click the border in the preview window rather than the button under Border.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Both diagonal border formats use the same line style. That is, if you change the line style of one diagonal border, the line style for the other diagonal border will change also.

REFERENCES

For more information about cell borders, click the index tab in Microsoft Excel Help, type the following text 

   borders, cells
and then double-click the selected text to go to the "Apply borders to cells" topic.
