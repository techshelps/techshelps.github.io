Excel: Network Support

 Last reviewed: November 2, 1994
Article ID: Q59620

SUMMARY

The following list details which versions of Microsoft Excel are network aware, multi-launch, multi-user, and network compatible: 

   Product   Network Aware  Multi-Launch  Multi-User  Network Compatible
   -------   -------------  ------------  ----------  ------------------

   1.00           Yes            No          Yes*            No
   1.03           Yes            No          Yes*            No
   1.04           Yes            Yes         Yes             Yes
   1.06           Yes            No+         Yes             Yes
   1.50           Yes            Yes         Yes             Yes
   2.20           Yes            Yes         Yes             Yes
   3.00           Yes            Yes         Yes             Yes
* Once a document has been opened, it can be opened a second time by another user, but neither instance of the document is read-only; both users have full write privileges and can overwrite each others' changes.

+ Set the cached bit using a file editor, such as ResEdit. Doing so allows Excel version 1.06 to be launched multiple times.

Term Clarification

Network Aware:       This means that the program can access other
                     volumes on the network.

Multi-Launch:        This means that one or more people can run the
                     application at the same time.

Multi-User:          This means that a document can be opened by one
                     or more persons at the same time. When the
                     document is opened for the second time, it may or
                     may not be opened as read-only. Certain programs
                     allow the file to be modified even though other
                     users are currently using the document.
Network Compatible: This means that Microsoft has tested this version 
                     of the product and has determined that it is
                     compatible with an AppleShare network. If there
                     is a "No" in this column, Microsoft does not
                     guarantee that the product will operate properly
                     on a network.
