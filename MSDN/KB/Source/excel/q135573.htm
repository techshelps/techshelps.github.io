XL: Sample Number Not Displayed in Color

 Last reviewed: June 3, 1997
Article ID: Q135573
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SUMMARY

In Microsoft Excel 7.0 when you view a sample number format in the Number tab of the Format Cells dialog box, the sample number will not be displayed in color if you have chosen a number format that contains a color.

MORE INFORMATION

This behavior occurs because a Microsoft Excel standard dialog box is being used that does not support this color.
