NEC Pinwriter Driver Previews Incorrectly in Excel

 Last reviewed: November 2, 1994
Article ID: Q65335

SUMMARY

Problems have been encountered with the NEC24PIN driver when switching print resolutions with the Printer Setup command in Excel. For example, if you switch from high resolution (360 dpi) to low resolution (60 dpi), print that previously covered only one page will be expanded over several pages in Print Preview. It will also print this way.

To get the page to resize properly once the resolution is switched in Excel, open the File menu, choose Page Setup, and press OK in the Page Setup dialog box. This will resize the page properly so that it will preview and print correctly.
