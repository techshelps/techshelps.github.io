Excel: Regression Dialog Box Does Not Fit 9-inch Screen

 Last reviewed: November 16, 1994
Article ID: Q88686
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 4.0

SUMMARY

The Regression function of the Analysis ToolPak in Excel 4.0 displays a dialog box which exceeds the viewable dimensions of the 9-inch and 12-inch screens on various Macintosh models.

MORE INFORMATION

The Regression function dialog box is fixed in size. When this dialog box is displayed on a 9-inch or 12-inch monitor, portions of the dialog box may trail off the bottom of the screen or the title bar of the dialog box may be hidden. The dialog box can be repositioned to make all features of the dialog box accessible; however, the entire dialog box will still not fit properly. The size of the Regression dialog box does not affect its functionality.

Examples of Macintoshes that are standardly equipped with the 9-inch monitor are:

  • Mac Plus
  • Mac SE / SE/30
  • Mac Classic / Classic II

REFERENCES

"Microsoft Excel User's Guide 2," version 4.0, pages 41-45
