Introduction

The WE1238 disk included with this Application Note contains the Jet Error Help file, Msjeterr.hlp, for use with Microsoft Excel for Windows 95. Because this Help file is not shipped with Microsoft Excel for Windows 95, you may receive the following error message when you use Help while you are using Data Access Objects (DAO) in Microsoft Excel: 

   Cannot find the Msjeterr.hlp file. Do you want to try to find
   this file yourself?
The Msjeterr.hlp file contains information about the error messages you receive when you use Data Access Objects to manipulate databases in the native Jet database engine.

To copy the Jet Error Help file to your computer

  1. If you received this Application Note on a disk, insert the enclosed WE1238 disk in your floppy disk drive.

  2. In My Computer or Windows Explorer, double-click the drive that contains the enclosed WE1238 disk, or, if you received this file electronically, open the folder to which you downloaded the file.

  3. Click the Msjeterr.hlp file.

  4. On the Edit menu, click Copy.

  5. Open the folder to which you installed Microsoft Windows, such as C:\Windows. Open the System folder inside your Windows folder.

  6. On the Edit menu, click Paste.

The Jet Error Help file is now available on your computer, and the correct information will appear when you click Help in an error message that you receive while using DAO objects in Microsoft Excel.
