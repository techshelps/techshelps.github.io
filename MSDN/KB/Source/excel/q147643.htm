XL: Versions of Quattro Pro Supported in Microsoft Excel

 Last reviewed: March 13, 1998
Article ID: Q147643
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SUMMARY

In Microsoft Excel, the online Help file states that Microsoft Excel can open all file formats from Quattro Pro for Windows, and that for MS-DOS versions of Quattro Pro, Microsoft Excel can only open Quattro Pro for MS- DOS version 5.0 files.

This information is incorrect. In fact, Microsoft Excel 7.0 and 97 can open Quattro Pro for MS-DOS files (*.WQ1) as well as files from Quattro Pro for Windows, versions 5.0 and earlier (*.WB1). Microsoft Excel is unable to open Quattro Pro DOS, version 6.0, files (*.WQ2) or Quattro Pro for Windows, version 6.0, files (*.WB2).

MORE INFORMATION

The following table lists the Quattro Pro file formats and how Microsoft Excel 7.0 and 97 can work with these files: 

Quattro Pro version       Microsoft Excel Support
--------------------      -----------------------

DOS 1, 2, 3, 4 (wq1)      Read/Write
DOS 5 (wq2)               none
Windows 1, 5 (wb1)        Read only
Windows 6 (wb2)           none

REFERENCES

For more information about converting Quattro Pro files, click the Index tab in Microsoft Excel 97 Help, type the following text 

   Quattro Pro
and then double-click the selected text to go to the "Switch from Quattro Pro" topic.
