XL7: Minimum System Requirements for Running Excel 7.0

 Last reviewed: March 12, 1997
Article ID: Q131174
7.00 WINDOWS kbref kbwwt

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

The minimum hardware and software requirements for running Microsoft Excel version 7.0 for Windows 95 are the following:

  • Personal computer with a 386DX or higher processor (486 recommended)
  • Microsoft Windows 95 operating system or Microsoft Windows NT Workstation operating system version 3.51 or later (will not run on earlier versions of Windows)
  • 6 MB of memory for use on Windows 95; 12 MB of memory for use on Windows NT Workstation
  • Hard disk space required: 8 MB compact; 18 MB typical; 38 MB custom (maximum)
  • One 3.5-inch high-density disk drive
  • VGA or higher-resolution video adapter (SVGA 256-color recommended)
  • Microsoft Mouse or compatible pointing device

Options

  • Windows-compatible network
  • Windows-compatible printer
  • 2400 or higher baud modem (9600 baud modem recommended)
  • Audio board with headphones or speakers
