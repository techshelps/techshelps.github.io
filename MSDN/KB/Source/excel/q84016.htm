Cannot Embed PPTGraph 1.0 Object in Excel

 Last reviewed: November 29, 1994
Article ID: Q84016
The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0

SUMMARY

When you are trying to insert a Graph object using the Insert Object command from the Edit menu, Microsoft Excel will return the following error message: 

   Unable to activate the source application for this embedded object.

MORE INFORMATION

Object Linking and Embedding (OLE) allows you to embed objects from a variety of source applications into Excel worksheets. When you choose Graph from the Insert Object dialog box, Excel attempts to run PPTGraph, which is shipped with Microsoft PowerPoint for Windows, version 2.0. PPTGraph version 1.0 is not, however, compatible with Excel 4.0 or later.

NOTE: This problem does not occur if you are using Microsoft PowerPoint version 3.0 or later.
