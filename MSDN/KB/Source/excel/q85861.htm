Excel Err Msg: Cannot Find This File. Please Verify...

 Last reviewed: November 4, 1994
Article ID: Q85861
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0

SUMMARY

When opening Lotus 1-2-3 for Windows files, you may receive the error message "Cannot find this file. Please verify that the correct path and filename are given." This error message may be caused by a space in the filename.

MORE INFORMATION

Lotus 1-2-3 for Windows allows filenames to contain spaces. This is not supported by the MS-DOS file naming conventions. MS-DOS requires that filenames not contain spaces, commas, backslashes, or periods (except the period that separates the name from the extension).

Microsoft Excel will only see the portion of the filename before the space and will attempt to open the file based on this name. For example, "FILE 1.WK3" will be seen as "FILE" and will not be opened.

Microsoft Excel will be unable to open the file unless it is reopened in Lotus 1-2-3 for Windows and saved without a space in the filename.
