Second Wind Provides Software Link Between Excel and Oracle

 Last reviewed: November 30, 1994
Article ID: Q74273

SUMMARY

Second Wind is a software product from Anju Technologies that supports DDE (dynamic data exchange) links between Microsoft Excel and Oracle Database systems.

MORE INFORMATION

Second Wind combines the database and connectivity of Oracle with Microsoft Excel. With Second Wind, you can issue SQL commands interactively from Excel spreadsheets to query and update Oracle databases directly and return the results to an Excel spreadsheet. Second Wind also supports SQL statements delivered from Excel macros.

Second Wind is available on a variety of platforms, including Microsoft Windows, OS/2 Presentation Manager, and Apple Macintosh.

Second Wind is manufactured by vendors independent of Microsoft. Microsoft makes no warranty, implied or otherwise, regarding the performance or reliability of this product. Any questions regarding the performance of Second Wind should be directed to Anju Technologies, LCG Consulting.

NOTE: This information also applies to Microsoft Excel for the Macintosh, versions 2.2 and 3.0.
