XL: References Updated Incorrectly After Inserting Cells

 Last reviewed: January 9, 1998
Article ID: Q170936
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SYMPTOMS

When you insert a range of cells, references that the cells contain may not be updated relative to the addresses of the original cells you selected.

CAUSE

This problem occurs when the following conditions are true:

  • You select the range of cell and click Cut on the Edit menu. 

        -and-
  • You select a destination range of cells and click Cut Cells on the Insert menu.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about inserting cells, click the Index tab in Microsoft Excel 97 Help, type the following text 

  inserting, cells
and then double-click the selected text to go to the "Insert cells, rows, or columns" topic.
