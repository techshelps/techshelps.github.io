XL97: Files Installed with the Stand-Alone Version of MS Excel

 Last reviewed: January 7, 1998
Article ID: Q171741
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SUMMARY

This article contains a list of the folders and files that are installed when you perform a Complete installation of Microsoft Excel 97 for Windows and install all the options.

MORE INFORMATION

You can use this list to determine the files that are installed with Microsoft Excel so that you may remove them if necessary.
