Excel Err Msg: Can't Read Record

 Last reviewed: November 30, 1994
Article ID: Q88407
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0, 5.0

SUMMARY

In Microsoft Excel, when you open a Lotus 1-2-3 file containing formulas that include IF() functions nested more than seven levels deep, you receive the alert, 

   Can't read record (Cell = <Cell Address>)
Microsoft Excel reads only the values in cells containing such formulas, not the formulas themselves.

MORE INFORMATION

This occurs because Microsoft Excel limits nesting of IF() functions to seven levels, but Lotus allows deeper levels of nesting. The only workaround is to redesign your Lotus 1-2-3 formula so that it doesn't nest more than seven levels of IF() functions.
