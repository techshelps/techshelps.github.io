Excel: Normal Font Determines Font of Row and Column Headings

 Last reviewed: November 30, 1994
Article ID: Q78642
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0
  • Microsoft Excel for OS/2, version 3.0

SUMMARY

The type, size, and style of the font used for row and column headings in Microsoft Excel version 3.0 or 4.0 worksheets or macro sheets are determined by the the normal font. To change the definition of the normal font, use the following procedure:

  1. From the Format Menu, choose the Style command.

  2. Select Normal under Style Name.

  3. Choose Define.

  4. Choose Font.

  5. Select the type, size, and style for the normal font, and choose the OK button.

This changes the type, size, and style of the font used for row and column headings of the active worksheet or macro sheet. It also determines the font used for unformatted cells.

MORE INFORMATION

For more information on changing the color of the row and column headings, query on the following words: 

  color and row and column and heading
