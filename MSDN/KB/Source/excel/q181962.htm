XL98: Preview Available on Blank Sheet with Merged Cells

 Last reviewed: March 3, 1998
Article ID: Q181962
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you click Print Preview on the File menu or you click the Print Preview toolbar button, Microsoft Excel displays a blank page.

CAUSE

This behavior occurs when the following conditions are true:

  • The worksheet contains merged cells. 

        -and-
  • The worksheet does not contain any data.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Example

To see an example of this behavior, follow these steps:

  1. Start Microsoft Excel 98.

  2. On the File menu, click Print Preview.

    Note that the following error message appears: 

          Microsoft Excel did not find anything to print.

    Click OK.

  3. Select cells A1:B1. On the Format menu, click Cells. In the Format Cells dialog box, click the Alignment tab. Click Merge Cells and click OK.

  4. On the File menu, click Print Preview.

    Note that Excel displays a blank page.

REFERENCES

For more information about print preview, click Contents And Index on the Help menu (or on the Balloon Help menu if you are using a version of the Macintosh operating system earlier than 8.0), click the Index button in Microsoft Excel Help, type the following text 

   Print Preview
and then click Show Topics. Select the "About Print Preview" topic, and click Go To. If you are unable to find the information you need, ask the Office Assistant.
