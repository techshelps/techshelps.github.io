XL: Filtering to a Protected Worksheet Causes Error Message

 Last reviewed: February 2, 1998
Article ID: Q145763
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Macintosh, versions 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In Microsoft Excel, if you use the Advanced Filter feature to filter a list and copy it to another location, you may receive one of the following error messages: 

   Locked cells cannot be changed.

   -or-

   The cell or chart you are trying to change is protected and therefore
   read-only.

CAUSE

This problem occurs when you use the Copy To Another Location option in the Advanced Filter dialog box, if the location to which you filter the list is a range of locked cells and the worksheet is protected. Note that in Microsoft Excel 5.x and 7.x, the list may be filtered into that location even though the worksheet is protected.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Excel listed above. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

Microsoft Excel 7.0

For more information about filtering a list to another location, click the Answer Wizard in Help and type: 

   how do I filter a list to another location

Microsoft Excel 5.0

"User's Guide," version 5.0, Chapter 21, "Copying Filtered Data to Another Location"

For more information about the Advanced Filter command, choose the Search button in Help and type: 

   advanced filter
