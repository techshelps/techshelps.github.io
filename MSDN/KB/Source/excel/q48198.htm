XL: Setting Starting Page Number in Header/Footer

 Last reviewed: February 2, 1998
Article ID: Q48198

The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 1.5, 2.2, 3.0, 4.0, 5.0, 5.0a
  • Microsoft Excel for Windows, versions 2.1, 3.0, 4.0, 4.0a, 5.0, 5.0c
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel 98 Macintosh Edition

SUMMARY

The starting page number for a Microsoft Excel document can be changed by clicking the Page Setup command. Using the following commands in the header or footer prints a starting page number other than 1. 

   Command        Description
   --------------------------

   &P+##          Substitute a number for the number signs (#) to
                  increment the page numbers by that amount.

   &P-##          Substitute a number for the number signs (#) to
                  decrement the page numbers by that amount.
For example, to start page numbering at page 3, type the following in the header or footer section of the Page Setup dialog box: 

   &P+2

MORE INFORMATION

Microsoft Excel version 1.50 does not have the ability to indicate a starting page number. To start the page number at a number greater than 1, follow these steps:

  1. Insert rows at the beginning of the worksheet, each with a page break below the row to increase the page number. For example, insert two blank rows, each with a page break following them, to start the document at page 3.

  2. Select the worksheet cells, including the beginning page breaks, and click Set Print Area on the Options menu.

  3. On the File menu, click Print.

  4. Click From, and type in the page range to be printed.

Note that using this procedure in version 1.5 causes row numbers to start at a row other than 1. To simulate row numbers starting at 1, follow these steps:

  1. Turn off Print Row & Column Headings in the Page Setup dialog box.

  2. Insert a row and column at the top and left of the print area with simulated row and column headings.

In Excel 4.0, you can specify a starting page number by clicking Page Setup on the File menu. Select "Start Page No.'s At:" (default value is 1).

In Excel 5.0, 7.0, 97 and 98 Macintosh Edition, you specify a starting page number by clicking Page Setup on the File menu. Click the Page tab, and then click "First Page Number:" (default value is auto).
