XL5: Print Options Settings Are Not Retained

 Last reviewed: September 25, 1996
Article ID: Q156420
The information in this article applies to:
  • Microsoft Excel for Windows NT, version 5.0

SYMPTOMS

When you use Microsoft Excel 5.0 for Windows NT, and you select a printing option, such as the "Print Text as Graphics" check box, the setting may become deselected after you close the Print dialog box.

CAUSE

This problem occurs when the following conditions are met:

  • You click Print on the File menu and click Printer Setup.

    -and-

  • You click Setup and click Options.

    -and-

  • You select a printing option and click OK twice.

This problem does not occur in Microsoft Excel 7.0.

WORKAROUND

To work around this problem, set print options from the Page Setup dialog box. To do this, use the following steps:

  1. On the File menu, click Page Setup.

  2. Click Options.

  3. In the Document Properties dialog box, click Options.

  4. If is cleared, select the "Print Text as Graphics" check box.

  5. Click OK three times.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Excel for Windows, version 7.0.
