MXL: Error Creating Linking Formula That Points to an Alias

 Last reviewed: February 5, 1998
Article ID: Q176462
The information in this article applies to:
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you enter a formula in a cell that is linked to a cell in another Microsoft Excel workbook, the following message may appear: 

   Cannot find '<path>'. Copy from:
Your choices are to click Copy, Cancel, or Find File.

CAUSE

This problem occurs when you enter a formula that is linked to an alias for a Microsoft Excel workbook.

WORKAROUND

To work around this problem, select the alias for the workbook when the message appears, and then click Copy. A link to the Excel workbook to which the alias points is created.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
