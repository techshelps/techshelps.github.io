Notes in Excel Are Split Between Pages when Printing

 Last reviewed: November 30, 1994
Article ID: Q32379
The information in this article applies to:
  • Microsoft Excel for Windows, version 2.x, 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for OS/2, versions 2.2, 3.0
  • Microsoft Excel for the Macintosh, version 2.2, 3.0, 4.0

SUMMARY

Cell notes from Microsoft Excel will print continuously and break at the bottom of the page as determined by the margins defined in File Page Setup. As a result, notes may be split between pages.

MORE INFORMATION

Workaround

Although cell notes cannot be formatted directly, you can make other adjustments to attempt to fit them on one page. Cell notes are formatted in the normal font; this will be font #1 in Microsoft Excel version 2.1 and earlier, and the font defined in the Normal style in Microsoft Excel version 3.0 and later. More notes will fit on a page if a small normal font is selected.

You can also adjust the margins in File Page Setup to fit more or fewer on a page. In Microsoft Excel version 3.0 and later, you can adjust the margins directly in print preview.
