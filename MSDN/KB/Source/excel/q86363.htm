Excel: Cannot Record Sound Notes With System 6

 Last reviewed: November 3, 1994
Article ID: Q86363
The information in this article applies to:

- Microsoft Excel for the Macintosh, version 4.0

Summary:

Page 286 of "Microsoft Excel User's Guide 1," version 4.0, states that Macintosh system software version 6.07 or later is required to record sound notes. After choosing Notes from the Formula menu when running System 6, however, the Record button is unavailable.

To record a sound to be used as a note when using System 6, record the sound using the Sound Control Panel device. This sound will get added to the System file and can be attached to a note by doing the following:

  1. Select the cell to have the note attached.

  2. From the Formula menu, choose Note.

  3. Choose the Import button.

  4. Go to the System Folder and select System.

  5. A list of the sounds in the System will be presented, select the desired sound and choose the OK button.

Microsoft has confirmed this to be a problem in version 4.0 of Microsoft Excel for the Macintosh. We are researching this problem and will post new information here as it becomes available.

Reference(s):

"Microsoft Excel User's Guide 1," version 4.0, pages 286-287
