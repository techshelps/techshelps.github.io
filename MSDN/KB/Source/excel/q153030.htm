 Last reviewed: September 13, 1996

The information in this article applies to:
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Small Business Financial Manager for Microsoft Excel, version 1.0

SUMMARY

This article lists the .cab file names and sizes and the files that the .cab files contain for Microsoft Small Business Financial Manager for Microsoft Excel version 1.0. This article includes information about disks 1-3 (3 disks total) of the 3.5-inch DMF disks.
