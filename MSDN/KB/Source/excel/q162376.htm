XL: PivotTable Is Not Sorted Correctly After Updating Data

 Last reviewed: February 2, 1998
Article ID: Q162376
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a

SYMPTOMS

When you update a PivotTable in Microsoft Excel, the fields in the PivotTable may not be sorted even though the data that is used to create the PivotTable changes.

CAUSE

When you click Refresh Data, Microsoft Excel updates the data in the PivotTable, but does not perform a sort operation on the updated data.

WORKAROUND

To work around this problem, use the following steps:

  1. Select the cell that contains the field header for the field that you want to sort.

  2. On the Data menu, click Sort. In the Sort dialog box, click OK.

The updated data in the PivotTable is sorted correctly.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
