Excel: Cannot Select Wide Columns with Mouse

 Last reviewed: September 12, 1996
Article ID: Q121467
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a

SUMMARY

When you use a mouse to select wide columns in Microsoft Excel, the selection may be made incorrectly. When a single column is wider than the entire screen, you will be unable to use the mouse to select the column to the right. This problem also occurs when one column is too wide to be displayed on the screen and the Freeze Panes option is enabled.

Note that you will be able to make selections with the keyboard.

WORKAROUND

Use the SHIFT key and the arrow keys on the keyboard to select the wide column and the cells to the right.

STATUS

This behavior has been corrected in Microsoft Excel version 5.0.

REFERENCES

"User's Guide 1," version 4.0, pages 232-235
