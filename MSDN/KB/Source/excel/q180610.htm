XL98: Sheet Protection Does Not Disable Options Settings

 Last reviewed: February 6, 1998
Article ID: Q180610
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you protect a worksheet, Microsoft Excel 98 Macintosh Edition continues to allow calculation of formulas in the worksheet.

CAUSE

Microsoft Excel always updates the worksheet if you change any of the following in the Preferences dialog box: 

   Precision as displayed
   Accept labels in formulas
   1904 date system
NOTE: To view the Preferences dialog box, click Preferences on the Tools menu. Click the Calculation tab to view these options.

STATUS

This behavior is by design in Microsoft Excel 98 Macintosh Edition.

REFERENCES

For more information about the 1904 date system, click Contents and Index on the Help menu. Click the Index button, and type the following text: 

   1900/1904 date system
and then double-click the selected text to go to the "How Microsoft Excel performs date and time calculations" topic.

For more information about the Accept labels in Formulas preference, click Contents and Index on the Help menu. Click the Index button, and type the following text: 

   formulas, names
and then double-click the selected text to go to the "About labels and names in formulas" topic.

For more information about Precision as displayed, click Contents and Index on the Help menu. Click the Index button, and type the following text: 

   precision of calculation
and then double-click the selected text to go to the "Change the way Microsoft Excel calculates formulas " topic.
