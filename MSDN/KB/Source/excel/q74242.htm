DB.EXTRACT Requires the Destination Argument in Q+E

 Last reviewed: November 2, 1994
Article ID: Q74242

SUMMARY

Page 69 of the "Q+E for Microsoft Excel User's Guide" lists the arguments for the function DB.EXTRACT. The description for the Destination argument indicates that type 1 is the default. This is a documentation error. Because this argument is required, there is no default.

REFERENCES

"Q+E for Microsoft Excel User's Guide," page 69
