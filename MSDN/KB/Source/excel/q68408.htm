Unable to Page Through Print Preview in Excel 3.0

 Last reviewed: November 1, 1994
Article ID: Q68408

SUMMARY

Page 538 of the "Microsoft Excel User's Guide" states that you should be able to use the PAGE UP or PAGE DOWN key to move to the next or previous page while in Print Preview. This is not a documentation error, but rather a problem in Excel. The PAGE UP or PAGE DOWN key will move you to the first or last page.

Microsoft has confirmed this to be a problem in Microsoft Excel version 3.0. This problem was corrected in Microsoft Excel version 4.0.

MORE INFORMATION

Workaround

A workaround is to use the UP ARROW and DOWN ARROW keys instead. All other keys work as stated in the documentation.

Steps to Reproduce Problem

  1. Create an Excel spreadsheet that is greater than two pages long.

  2. From the File menu, choose Print Preview.

  3. Press PAGE DOWN.

If you start on page one, the next page that is shown is not page two, but rather the last page in the spreadsheet.
