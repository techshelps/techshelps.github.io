Excel: Inserted Disk Unrecognized While Using Help

 Last reviewed: November 29, 1994
Article ID: Q86568
The information in this article applies to:
  • Microsoft Excel for the Macintosh, version 4.0

SYMPTOMS

In Microsoft Excel 4.0 for the Macintosh, when you insert a floppy disk in the disk drive when Microsoft Excel Help is running, the Macintosh will not be able to read the disk.

When you quit Microsoft Excel Help and Microsoft Excel, the system does not recognize the disk and does not display a disk icon on the desktop.

WORKAROUND

To work around this problem, do the following:

  1. Press COMMAND+SHIFT+1 (or COMMAND+SHIFT+2 for a second disk drive) to force the Macintosh to eject the disk.

  2. Re-insert the disk into the floppy disk drive.

If the system still fails to recognize the disk, restart your Macintosh, and reinsert the disk.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
