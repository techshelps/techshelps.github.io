XL97: Microsoft Excel 97 Viewer Cannot Open Lotus Files

 Last reviewed: December 5, 1997
Article ID: Q174139
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SUMMARY

You cannot use Microsoft Excel 97 Viewer to open files that are created in Lotus 1-2-3. If you try to open a file created in Lotus 1-2-3 you receive the following error message: 

   Microsoft Excel Viewer cannot open files of this type.
Microsoft Excel Viewer for Windows 95, version 7.0, can open files created in Lotus 1-2-3 versions 5.0 and earlier.

MORE INFORMATION

To determine which version of Microsoft Excel Viewer you are using, click About Excel Viewer on the Help menu.

For more information about the Microsoft Excel Viewer, visit the Microsoft Excel Viewer Web page at the following address: 

   http://www.microsoft.com/excel/internet/viewer
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
