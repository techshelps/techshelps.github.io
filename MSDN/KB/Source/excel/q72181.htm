Cannot Paste Link a Picture of a Worksheet into WinWord

 Last reviewed: November 3, 1994
Article ID: Q72181
The information in this article applies to:
  • Microsoft Excel for Windows, version 3.0

SUMMARY

When a picture of a range on a worksheet is copied from Microsoft Excel using the Edit Copy Picture command, and a Paste Link command is executed from the Edit menu in Word for Windows, the data appears as a table, not as a picture. If the copied selection from Microsoft Excel contains any graphic objects, they do not appear in Word for Windows.

However, the Paste command in Word for Windows will produce the picture, including any graphic objects.

MORE INFORMATION

The Copy Picture command in Microsoft Excel is accessed by selecting the appropriate information, holding down the SHIFT key and selecting the Edit menu. The Copy Picture command will appear in place of Copy.

REFERENCES

"Microsoft Excel User's Guide," version 3.00, pages 521-522
