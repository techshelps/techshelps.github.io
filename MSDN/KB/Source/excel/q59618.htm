Excel: Guided Tour Does Not Support Color

 Last reviewed: November 2, 1994
Article ID: Q59618

SUMMARY

The Microsoft Excel version 2.20 Guided Tour is a HyperCard stack and requires HyperCard version 1.2 or later to run.

HyperCard does not support color or gray scaling, and for HyperCard's special effects, the color or gray scale options need to be turned off on color Macintoshes (Macintosh II, IIx, IIcx, IIci, SE/30).

MORE INFORMATION

To turn off these options, do the following:

  1. From the Apple menu, choose Control Panel.

  2. Select the Monitor icon.

  3. Do one of the following, depending on the System version you are using:

    a. If you are using System version 6.0.3 or earlier, set the number 

          of colors to 2 by selecting "2".


          -or-

    b. If you are using System version 6.0.4, set the number of colors 

          to 2 by selecting "Black and White."

Note that if you run the Guided Tour on a color Macintosh, a dialog box appears, warning you to set the number of colors to 2 if it has not already been set, with options to Continue or Cancel. Under System 6.0.4, "Black and White" replaces 2.
