XL98: How to Completely Remove Data Access

 Last reviewed: March 16, 1998
Article ID: Q182496
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SUMMARY

This article discusses how to remove the Data Access component of Microsoft Office 98 Macintosh Edition.

MORE INFORMATION

The Data Access component is a made up of the following items: 

   Microsoft Query
   Sample Databases
   ODBC Administrator
   dBASE ODBC Driver
   FoxPro ODBC Driver
   Oracle Driver
   SQL Server ODBC Driver
   Text ODBC Driver
For troubleshooting purposes, it may be necessary to remove the Data Access component of Office. The Value Pack Installer does not allow you to uninstall any if the Value Pack items; you must manually remove the Data Access files. To remove the Data Access component of Office 98, delete the files listed in the following table. 

   File Name                         Folder Location
   --------------------------------------------------------------------

   Microsoft Query                   Microsoft Office 98:Office
   MS Query Help                     Microsoft Office 98:Office
   Excel ODBC Support Library        Microsoft Office 98:Office:
                                     Excel Add-ins:MSQuery
   Xlodbc.xla                        Microsoft Office 98:Office:
                                     Excel Add-ins:MSQuery
   Xlquery.xla                       Microsoft Office 98:Office:
                                     Excel Add-ins:MSQuery
   MS Query AW                       Microsoft Office 98:Office:Help
   Customer.cdx                      Microsoft Office 98:Sample Files:
                                     Sample Databases
   Customer.dbf                      Microsoft Office 98:Sample Files:
                                     Sample Databases
   Detail.cdx                        Microsoft Office 98:Sample Files:
                                     Sample Databases
   Detail.dbf                        Microsoft Office 98:Sample Files:
                                     Sample Databases
   Employee.cdx                      Microsoft Office 98:Sample Files:
                                     Sample Databases
   Employee.dbf                      Microsoft Office 98:Sample Files:
                                     Sample Databases
   Orders.cdx                        Microsoft Office 98:Sample Files:
                                     Sample Databases
   Orders.dbf                        Microsoft Office 98:Sample Files:
                                     Sample Databases
   Product.cdx                       Microsoft Office 98:Sample Files:
                                     Sample Databases
   Product.dbf                       Microsoft Office 98:Sample Files:
                                     Sample Databases
   Supplier.cdx                      Microsoft Office 98:Sample Files:
                                     Sample Databases
   Supplier.dbf                      Microsoft Office 98:Sample Files:
                                     Sample Databases
   ODBC Setup PPC                    System Folder:Control Panels
   Microsoft 3.01 Base Lib           System Folder:Extensions
   Microsoft 3.01 Flat Lib           System Folder:Extensions
   Microsoft 3.01 Utilities Lib      System Folder:Extensions
   ODBC SQL Server ASDP Netlib PPC   System Folder:Extensions
   ODBC SQL Server Driver PPC        System Folder:Extensions
   ODBC SQL Server Setup PPC         System Folder:Extensions
   ODBC SQL Server TCP Netlib PPC    System Folder:Extensions
