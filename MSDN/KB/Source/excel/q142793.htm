XL7: Invalid Page Fault in Excel.exe When Using Paste Special

 Last reviewed: September 13, 1996
Article ID: Q142793
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you use the Paste Special command after you copy an embedded chart, the following message may appear: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.
If you click Details, a message similar to the following appears: 

   EXCEL caused an invalid page fault in module EXCEL.EXE at
   0137:501fa35a.

CAUSE

This problem occurs when the following conditions are true:

  • You copy the embedded chart.

    -and-

  • You delete the chart.

    -and-

  • You click Paste Special on the Edit menu, or use the right mouse button to click the workbook, and then click Paste Special.

RESOLUTION

To work around this problem, paste the chart before you delete it.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
