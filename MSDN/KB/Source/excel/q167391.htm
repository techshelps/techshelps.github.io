SBFM97: Additional Accounting Filters Available

 Last reviewed: February 12, 1998
Article ID: Q167391
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Small Business Financial Manager 97 for Microsoft Excel 97

SUMMARY

Additional filters for the Small Business Financial Manager 97 are available. You can download these updates free of charge. The self- extracting file, Sbfm97ad.exe, contains four files. These files are listed in the following table. 

   File Name       Description
   ----------------------------------------------

   Readme.txt      Readme file with instructions
   Aapgreat.dll    Great Plains Accounting, versions 8.0, 8.1, and
                   8.2 for MS-DOS
   Aapmeta.dll     Timeline MV Server, versions 2.3, 2.4, and 2.5
   Aapplat44.dll   Platinum Series, version 4.4 for Windows

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~ Sbfm97ad.exe (size: 283691 bytes)
