Excel: How to Create Mailing Labels with Excel Q+E

 Last reviewed: April 26, 1995
Article ID: Q92462
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0

SUMMARY

Microsoft Excel versions 3.0 and 4.0 includes a database application called Q+E that allows you to manipulate and update database files. You can use Q+E to create mailing label files that you can print to create mailing labels.

To use Q+E to create a mailing label file:

  1. Save your Excel database file as a dBASE (DBF) file.

  2. Open Q+E and load your DBF file.

  3. In Q+E, from the File menu, choose Save As.

  4. From the Destination list, select Mailing Labels.

  5. Choose the Options button.

  6. Fill in the information in the Label Destination box. Type the fields in the database that you want to appear on each line of the mailing label. To enter multiple fields on a line, separate each field with a plus sign (+). To separate the fields, use single quotation marks with a blank space or text string between them, as in the following example:

    City+','+State+' '+Zip

  7. Enter the number of blank lines you want between labels.

  8. If there is more than one row of labels on a page (as is the case with laser printer labels), fill in the Column Start Position boxes. The column start positions tell Q+E where to start each column of labels.

  9. Choose the OK button and name your mailing label file.

  10. To print your file, open it in a text editor such as Notepad.
