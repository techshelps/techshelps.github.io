XL97: Auditing Tools Clear Undo/Redo Stack

 Last reviewed: January 15, 1998
Article ID: Q157298
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

After you use the auditing tools, both the Undo and Redo commands on the Edit menu are unavailable (appear dimmed). Additionally, the Undo and Redo buttons on the Standard toolbar are unavailable. The ability to undo or redo actions previous to using the auditing tools has been cleared.

CAUSE

This behavior occurs when you use any of the following auditing tools: 

   Trace Precedents
   Remove Precedent Arrows
   Trace Dependents
   Remove Dependent Arrows
   Remove All Arrows
   Trace Error
   Circle Invalid Data
   Clear Validation Circles

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
