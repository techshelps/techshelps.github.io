XL98: Crash When Chart Gallery File Is Password Protected

 Last reviewed: February 2, 1998
Article ID: Q178863
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you click Chart Type on the Chart menu, and then click the Custom Types tab, you may receive the following error message: 

   The application "Microsoft Excel" has unexpectedly quit, because an
   error of type 2 occurred.

   You should save your work in other open applications and restart the
   computer.

    -or-

   Sorry, a system error occurred.
   "Microsoft Excel"
   error type 11

CAUSE

This problem occurs when either of the following conditions is true:

  • The Excel Chart Gallery(8) file is password protected. 

        -or-
  • The Excel Chart User Gallery file is password protected.

WORKAROUND

To resolve this problem, remove the password protection from the Excel Chart Gallery(8) or the Xlusrgal.xls file. To do this, follow these steps:

  1. Open the appropriate file (Excel Chart Gallery(8) or Excel Chart User Gallery). By default,the files are stored in the System Folder:Preferences folder.

    NOTE: The Excel Chart Gallery(8) and Excel Chart User Gallery files are chart "autoformat" files. These files store custom chart formats.

  2. On the File menu, click Save As, and then click Options.

  3. In the Password To Open box, click the password.

  4. Press DEL and Click OK.

  5. Click Save, and then click Yes when you are prompted to overwrite the existing copy of the file.

If you do not know the password, you must delete the chart autoformat file that is password protected.

If you delete the Excel Chart User Gallery file, you lose any custom chart formats that you created.

If you delete the Excel Chart Gallery(8) file, you can recreate the file only by using the following steps:

  1. Double-click the Microsoft Office First Run file located in the Microsoft Office 98:Office folder.

  2. In the first dialog box that appears, click Install.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Excel Chart Gallery(8) stores custom chart formats that come with Microsoft Excel 98. Excel Chart User Gallery stores any custom chart formats that you create. Microsoft Excel does not create Excel Chart User Gallery until you create a custom chart format.

For additional information about working with custom chart types in Microsoft Excel, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q161348
   TITLE     : XL97: Working With Default and User Defined Chart Formats

REFERENCES

For more information about Chart Autoformats, click the Index tab in Microsoft Excel Help, type the following text 

   chart creation, templates
and then double-click the selected text to go to the "Save a custom chart type to apply to other charts" topic.
