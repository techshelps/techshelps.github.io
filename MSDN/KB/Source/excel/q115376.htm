XL: Microsoft Excel Does Not Save Print Settings for Views

 Last reviewed: January 24, 1997
Article ID: Q115376
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

When you make changes to print settings in the Page Setup dialog box, views may lose their print settings.

CAUSE

This problem occurs with various printer drivers, including the following:

HP LaserJet 4/4M, driver version 31.V1.20 for Windows 3.1 HP LaserJet 4/4M Plus driver version 31.v1.30 HP LaserJet III driver version 31.3.89 HP LaserJet IID driver (HPPCL.DRV) version 2.34 Windows Printing System version 1.0 Epson LQ1000 V.2.11 Panasonic 1124 V.1.0 LaserJet IIIsi V.2.1 Brother HL-600 Series, version 3.00 (with Microsoft Excel version 7.0)

WORKAROUNDS

HP LaserJet 4/4M drivers

If you are using one of the LaserJet 4/4M drivers, do the following to work around this problem:

  1. Before you create views, change your default printer to any printer other than the HP LaserJet 4/4M, or the HP LaserJet III or IID printers using the driver versions listed above.

  2. Create all of your views.

  3. Change the default printer driver back to LaserJet 4/4M.

HP LaserJet III version 31.3.89

The problem has been corrected in the newest HP III driver for Windows 3.1 (HPPCL5MS.DRV - 11/16/92).

You can obtain this driver from either Hewlett Packard or the Microsoft Download Service (MSDL).

HP LaserJet IID version 2.34 (5/12/93)

The problem has been corrected in the HP LaserJet IID driver version 2.1 that ships with Microsoft Windows and Microsoft Windows for Work groups version 3.11.

Note that the problem does not occur in Microsoft Excel version 5.0c with this driver.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
