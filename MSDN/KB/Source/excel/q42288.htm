Date Formats Turning to Serial Numbers on Charts

 Last reviewed: November 30, 1994
Article ID: Q42288
The information in this article applies to:
  • Microsoft Excel for Windows, versions 2.x, 3.0, 4.0
  • Microsoft Excel for the Macintosh, versions 2.2, 3.0, 4.0
  • Microsoft Excel for OS/2, versions 2.2, 2.21, 3.0

SYMPTOMS

When creating and saving a chart using information formatted with a custom date format, the dates no longer appear when the chart is reopened. Their serial number representations are displayed.

NOTE: This problem does not occur in later versions of Microsoft Excel.

CAUSE

The supporting sheet (the sheet from which the chart was created) is not open.

WORKAROUND

Once you have both the worksheet and the chart open, the custom date formats display correctly.
