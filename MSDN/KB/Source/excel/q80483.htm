Excel: Blank Rows or Columns After Consolidating with Links

 Last reviewed: November 30, 1994
Article ID: Q80483
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0, 5.0

SUMMARY

Consolidating with links to source data to a destination area that contains an outline structure adds extra blank rows or columns to the worksheet. It also adds one more outline level to the worksheet. After multiple linked consolidations to the same destination area, the entire destination area may appear blank due to these inserted blank areas until the outline is expanded to show all levels.

Consolidating to an area containing an outline structure is not recommended. The existing destination area should first be cleared of outlining, then the sheet may be consolidated again with new or existing source references.

MORE INFORMATION

To clear an area of outlining before consolidating with links:

  1. Display all levels by choosing the highest numbered level button.

  2. Select all rows in the destination area.

  3. Use the step below appropriate to your version: 

        - For Excel versions 4.0 and earlier, choose the promote icon in the
      tool bar several times until there are no more outline symbols in the
      destination area.


        - For Excel 5.0, choose the Ungroup tool in the Query And Pivot tool
      bar several times until there are no more outline symbols in the
      destination area. You can also select the Data/Group and
      Outline/Ungroup option for the same results.
