XL: Pivot Table Refresh Overwrites Formatting

 Last reviewed: March 27, 1997
Article ID: Q110602
5.00 5.00c 7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Excel for Windows, versions 5.0, 5.0c
  • Microsoft Excel for Windows NT, version 5.0
  • Microsoft Excel for Windows 95, version 7.0

SUMMARY

When you apply formatting to individual cells in a pivot table, this individual formatting will be lost when the pivot table data is refreshed.

WORKAROUND

To change the automatic pivot table formatting, which is applied in the last step of the Pivot Table Wizard, do the following to apply an autoformat to the table:

  1. From the Format menu, choose AutoFormat.

  2. From the Table Format box, choose an autoformat.

  3. Choose OK.

Refreshing the data will not change the applied autoformat.

Note that if other cells not contained within the pivot table are included in the AutoFormat process, formatting will be lost from the pivot table when the data is refreshed. To prevent this from occurring, make sure that the autoformatting applied to the pivot table is not also applied to adjacent cells.

For additional information, please see the following article in the Microsoft Knowledge Base:

   TITLE     : XL: Data Number Formatting Lost Creating Pivot Table
