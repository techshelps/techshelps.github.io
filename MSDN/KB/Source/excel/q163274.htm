XL97: Error Using Canon MultiPASS C2500 or Bitware Fax Driver

 Last reviewed: February 20, 1998
Article ID: Q163274
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Excel 97, when you click Page Setup on the File menu, you may receive the following error message: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.
If you click Details, you receive the following error message: 

   EXCEL caused an invalid page fault in module MSO97.DLL at
   014f:306c34b9.
Note that the address of the invalid page fault that is displayed may vary from the address in this article.

CAUSE

This problem occurs when the following conditions are true:

  • The Canon MultiPASS C2500 multifunction printer driver, version 3.4, is installed on your computer.

    -and-

  • The Canon MultiPASS C2500 printer is your default printer.

    -or-

  • The Bitware Fax driver is installed on your computer and is set as the default printer.

RESOLUTION

Canon MultiPass C2500

To resolve this problem, obtain the Canon MultiPASS C2500 printer driver, versions 3.70 or later. You can obtain and install the newer printer driver from Canon World Wide Web site on the Internet.

To obtain the driver from the Internet, use the following steps:

  1. In a Web browser, go to the following address: 

          http://www.ccsi.canon.com/care/download/mfprinters.html

  2. The following four links for Canon MultiPASS C2500 files are available as of February 6, 1997: 

          Bj370d1.exe
      Bj370d2.exe
      Bj370d3.exe
      Bj370d4.exe

  3. Click each link once. When you are prompted, click "Save it to disk" and save the file in an empty folder on your hard disk.

  4. After you download all four files, click Run on the Start menu. In the Open box, type "command" (without the quotation marks) and click OK.

  5. At the command prompt, change to the folder that contains the four files you downloaded.

  6. Run each of the files by typing the file name and pressing ENTER. If you are prompted, do not replace any files.

  7. After you run all four files, type "exit" (without the quotation marks) and press ENTER.

Before you install the new printer driver, remove the old printer driver. To do this, use the following steps:

  1. On the Start menu, point to Settings, and then click Printers.

  2. In the list of printers, click Canon MultiPASS C2500 Printer. On the File menu, click Delete. Click Yes to confirm that you want to delete the printer.

  3. If you are prompted to remove unused files, click Yes.

To install the new printer driver, use the following steps:

  1. In the Printers window, double-click Add Printer.

  2. Click Next.

  3. Click Local Printer, and then click Next.

  4. In the third dialog box of the Add Printer Wizard, click Have Disk. In the Install From Disk dialog box, type the path to the new printer driver files that you downloaded and click OK.

  5. In the list of printers, click Canon MultiPASS C2500 Printer. Then, click Next. If you are prompted to replace the existing driver, click "Replace existing driver" and click Next.

  6. Select a port for the printer (LPT1:, for example) and click Next.

  7. Create a name for the printer and indicate whether to use the new printer as the default printer. Click Finish.

NOTE: Microsoft does not support the setup and installation of the new printer driver. If you require additional help, contact Canon Technical Support or visit the following Canon Web page: 

   http://www.ccsi.canon.com/care/download/downloadhelp.html
The third-party contact information included in this article is provided to help you find the technical support you need. This contact information is subject to change without notice. Microsoft in no way guarantees the accuracy of this third-party contact information.

Bitware Fax Driver

If you set another printer as the default printer, the problem described above should no longer occur. You may also want to contact the vendor to see if a newer Bitware fax driver is available.

STATUS

This problem no longer occurs in Microsoft Excel 97 and the Canon MultiPASS C2500 printer driver, version 3.70 or later.

MORE INFORMATION

The Canon MultiPASS C2500 printer includes a color printer, a copier, a fax machine, and a scanner in one unit. A set of disks that contain the necessary Microsoft Windows 95 printer drivers is included with the printer.

The drivers that are included with the printer may cause problems when you display the Page Setup dialog box in Microsoft Excel 97. If this problem occurs, download and install the newer Canon MultiPASS C2500 printer drivers. Or, contact Canon Technical Support and request a set of disks that contain the newer printer drivers.
