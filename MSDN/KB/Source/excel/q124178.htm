XL5: Replace Command in Group Edit Doesn't Change Active Sheet

 Last reviewed: September 12, 1996
Article ID: Q124178
The information in this article applies to:
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

If you are working in group edit mode, and you choose the Replace All button in the Replace dialog box, Microsoft Excel will behave as expected the first time you choose the button. However, when you attempt to use this button again, the active sheet will not be changed (note that other grouped sheets will be changed as expected).

STATUS

This problem has been corrected in Microsoft Excel 5.0c for Windows
