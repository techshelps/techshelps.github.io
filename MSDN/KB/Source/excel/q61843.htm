Excel: Running the CL/1 Installer with GateKeeper Active

 Last reviewed: November 2, 1994
Article ID: Q61843

SUMMARY

GateKeeper is a virus-prevention product that monitors hard drives and checks for virus-type actions, such as adding resources to the System file and creating INIT files.

To use Apple's CL/1 Database Access Language, certain CL/1 resources must be added to the System, and files must be copied into the System Folder. This action is performed by the Microsoft Excel CL/1 Installer.

If GateKeeper is active, it will flag several operations of the Installer as possible virus actions. If GateKeeper is in Notify and Deny mode, the Installer will not be able to install the necessary resources for CL/1 to operate. If GateKeeper is in Notify Only mode, it will display an alert that certain System modifications have occurred, but will allow the Installer to complete the installation.

MORE INFORMATION

GateKeeper is a Control Panel device (CDEV), which means it is controlled through the Control Panel. To change the GateKeeper settings, choose Control Panel from the Apple menu and click the GateKeeper icon. At this point, set Gatekeeper to allow the installation of CL/1 by performing one of the following steps:

  1. Click the Override button so that it reads "On." This will disable GateKeeper for up to a half hour, or until Override is turned off through the Control Panel.

  2. Click the settings portion of the slide switch in the upper-right corner of the GateKeeper dialog box. This will allow particular settings to be adjusted. Select the Notify Only option to allow the Installer to complete the installation but bring up an alert with each action.

  3. Click the settings portion of the slide switch in the upper-right corner of the GateKeeper dialog box. This will allow particular settings to be adjusted. Click the Add button to add the Microsoft Excel CL/1 Installer to the list of applications that can perform suspect operations. Clicking Add will present the standard Open dialog box and allow selection of the Microsoft Excel CL/1 Installer. Select the Installer and click OK. Under the Edit and Clear buttons, check the Sys box for Res, and check the Other box for File. This will allow the Installer to install files without presenting an alert.
