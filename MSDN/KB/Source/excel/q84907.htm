Excel: 3-D Pie Chart Legend Redraws Incorrectly

 Last reviewed: November 30, 1994
Article ID: Q84907
The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0
  • Microsoft Excel for the Macintosh, version 4.0

SYMPTOMS

In Microsoft Excel for Windows, if the legend for a 3-D pie chart is moved from its default location, the legend may move after printing or print previewing the chart. Also, if the legend is selected (so that the black handles appear around the legend) when performing a print preview, the handles will not move to the legend's new location when returning to normal mode.

In Microsoft Excel for the Macintosh, the legend may not move, but when the legend is moved with the mouse, a duplicate copy of the legend may remain in the old location.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Excel listed above. This problem was corrected in version 5.0 of Microsoft Excel for Windows.

MORE INFORMATION

Steps to Reproduce Problem

  1. Type the following information into a worksheet:

    A1: 1 A2: 2 A3: 3

  2. Select cells A1:A3.

  3. From the File menu, choose New.

  4. In the New box, select Chart.

  5. From the Gallery menu, choose 3-D Pie #3.

  6. From the Chart menu, choose Add Legend.

  7. Move the legend so that it is bordering on the edge of the pie chart. The exact positioning of the legend is important.

  8. From the File menu, choose Print Preview.

  9. From the File menu, choose Close to return to the chart sheet.

Depending on the placement of the legend, you may see that the legend has moved and that the handles appear in old legend location, displaced from the current legend location.

Although the problem is most notable in a 3-D pie chart, the problem can also be seen in other 3-D charts.

REFERENCES

"Microsoft Excel User's Guide 1," version 4.0, chapter 13
