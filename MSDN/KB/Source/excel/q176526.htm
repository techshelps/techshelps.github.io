XL98: Help Does Not Appear in the Internet Assistant Wizard

 Last reviewed: February 2, 1998
Article ID: Q176526
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

In the Internet Assistant Wizard, when you try to use the Office Assistant, Help does not appear, and the Internet Assistant Wizard quits.

CAUSE

This problem occurs when you do the following:

  • You click Add in step 1 of the Internet Assistant Wizard.

    -and-

  • You click "Yes, please provide help" on the Office Assistant, and then click OK in the Internet Assistant Wizard.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you use the Internet Assistant Wizard, you can leverage existing Microsoft Excel spreadsheet data to create and distribute documents that can be viewed online with popular Web browsers. By clicking a series of buttons in the wizard, you can convert spreadsheet data to Hypertext Markup Language (HTML) format and save it as a separate Web document or insert it into an existing Web document.

Example

To see an example of the behavior, follow these steps:

  1. In Microsoft Excel97, type some text in cell A1. On the File menu, click Save as HTML.

  2. In the "Internet Assistant Wizard - Step 1 of 4" dialog box, click Add. On the Office Assistant, click "Yes, please provide help."

    No Help appears.

  3. Click OK in the Internet Assistant Wizard reference dialog box.

    The wizard quits.
