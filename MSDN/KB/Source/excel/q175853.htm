XL98: Angled Text Is Unreadable or Not Displayed

 Last reviewed: February 2, 1998
Article ID: Q175853

The information in this article applies to:

  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you enter text in a cell in Microsoft Excel 98 Macintosh Edition, the following problems may occur:

  • The text does not appear in the cell, and you receive the following error message: 

          Not enough memory to display completely.

    If you scroll away from the cell, and then scroll back, you receive the error message again.

    -or-

  • The text appears in the cell, but it unreadable.

CAUSE

These problems may occur when the following conditions are true:

  • You use an intermediate cell orientation value other than zero degrees, 90 degrees, or -90 degrees (for example, 30 degrees or 45 degrees).

    -and-

  • You enter a large amount of text (120 or more characters, or 2 or more lines of text) in the cell.

NOTE: These problems are more likely to occur if you zoom the worksheet to a value that is greater than 100 percent (100%).

WORKAROUND

To work around this problem, do not enter a large amount of text in a cell that uses an intermediate cell orientation value.

Or, use a cell orientation of zero, 90, or -90 degrees. These problems do not occur when text is horizontally or vertically oriented.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

In Microsoft Excel 97 for Windows and Microsoft Excel 98 Macintosh Edition, you can specify the orientation of text in a cell by following these steps:

  1. Select the cell.

  2. On the Format menu, click Cells. Click the Alignment tab.

  3. Use the Orientation controls to set the orientation you want.

  4. Click OK.

Note that the problems described in this article do not occur in Microsoft Excel 97 for Windows. In Microsoft Excel 97, if you enter a large amount of angled text in a cell, the cell displays text up to a certain point; you do not receive any error messages, and the text does not overlap itself.
