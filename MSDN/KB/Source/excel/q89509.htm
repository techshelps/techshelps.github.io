Excel: Data Consolidate Returns Small Column Width

 Last reviewed: July 16, 1997
Article ID: Q89509

The information in this article applies to:
  • Microsoft Excel for Windows, versions 3.0, 4.0, 4.0a, 5.0
  • Microsoft Excel for the Macintosh, versions 3.0, 4.0
  • Microsoft Excel for OS/2, version 3.0

SUMMARY

When using the Data Consolidate feature with both the Use Labels In Left Column and Create Links To Source Data options selected, the first column of the destination area is formatted to a small column width.

MORE INFORMATION

This is done to provide a nicely formatted outline (when expanded). This occurs with or without the Use Labels In Top Row option enabled. This also occurs when using the macro function CONSOLIDATE().

The actual width depends upon your video resolution.

REFERENCES

"Microsoft Excel User's Guide 1," version 4.0, pages 376-383
