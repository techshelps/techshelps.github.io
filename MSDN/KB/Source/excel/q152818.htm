SBFM: How to Uninstall Small Business Financial Manager

 Last reviewed: September 13, 1996
Article ID: Q152818
The information in this article applies to:
  • Microsoft Excel for Windows 95, versions 7.0, 7.0a
  • Microsoft Small Business Financial Manager for Microsoft Excel, version 1.0

SUMMARY

This article describes the process to uninstall the Microsoft Small Business Financial Manager for Microsoft Excel (SBFM) so that the Windows registry is properly updated.

MORE INFORMATION

The directions to properly remove the SBFM assume that it has been properly installed using the Setup program on Disk 1 of the SBFM disks and that Microsoft Excel is currently running.

To uninstall the SBFM, follow these steps:

  1. On the Tools menu, click Add-Ins.

  2. In the Add-Ins dialog box, in the list of Add-Ins Available, if there is a check mark next to Small Business Financial Manager, click the check box to clear it.

  3. Click OK.

  4. Exit Microsoft Excel.

  5. Run the Setup program for SBFM.

    NOTE: There is a shortcut for the SBFM Setup program in the following folder: 

           \Msoffice\Excel\Aap\Setup

  6. In the first dialog box that appears after starting Setup, click Remove All, and then click Yes when asked to confirm your choice.

The Microsoft Small Business Financial Manager for Microsoft Excel should now be removed from your computer and the add-in should not be listed as an installed add-in.
