MSQuery: "Couldn't Read This File" Opening .qry File

 Last reviewed: September 12, 1996
Article ID: Q138282
The information in this article applies to:
  • Microsoft Query for Windows 95, version 2.0

SYMPTOMS

In Microsoft Query 2.0, when you attempt to open a Query (*.qry) file by double-clicking the file in Windows Explorer or on the desktop, if the file has a long file name, you may receive the following error message and the file will not be opened: 

   Couldn't read this file

WORKAROUND

To work around this problem, use either of the following methods.

Method 1: Rename the file so that the file name complies to the 8.3 file 

          naming convention. Then, you can successfully open the file by
          double-clicking either the file or a shortcut to the file in
          Windows Explorer or on the desktop.

          To rename a file, use the right mouse button to click the file
          and then click Rename on the Shortcut menu. Type the new file
          name and press ENTER.
Method 2: Start Microsoft Query and open the .qry file by clicking Open on 
          the File menu.
