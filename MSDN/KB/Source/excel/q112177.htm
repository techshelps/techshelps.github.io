XL5: Print Preview Hangs Using SpeedSTAR PRO & Other Drivers

 Last reviewed: March 31, 1997
Article ID: Q112177
The information in this article applies to:
  • Microsoft Excel for Windows, versions 5.0, 5.0c

SYMPTOMS

If you use the Diamond SpeedSTAR PRO 800x600 64K Small Fonts video driver, Microsoft Excel may stop responding (hang) when you do any of the following:

  • View a document in print preview
  • Choose print commands (such as Print or Page Setup)
  • Click Options on the Tools menu, and select the General tab
  • Click the Format menu, and then click Cells

This behavior may also occur with the following products:
  • SpeedSTAR PRO Large Fonts driver
  • Genoa 8500 VL video card when in 800x600 64K
  • VC-826W
  • Diamond SpeedSTAR PRO 800x600 64K Small Fonts driver

WORKAROUND

To avoid hanging in Microsoft Excel when you view a document in print preview, install a video driver that ships with Microsoft Windows, such as the VGA video driver. For more information on installing a video driver in Microsoft Windows, see your Microsoft Windows documentation.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
