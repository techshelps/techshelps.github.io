XL7: Quattro Pro 5.0 for Windows (.wb1) File Opens Slowly

 Last reviewed: March 27, 1997
Article ID: Q135393
7.00 WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel for Windows 95, version 7.0, when you open a Quattro Pro for Windows, version 5.0 file (a file with the extension .wb1) that contains more than 50 worksheets in the workbook, it may take Microsoft Excel a long time to convert and open the file.

Note that this problem has also been reported to occur with workbooks that contain fewer than 50 worksheets.

CAUSE

This slow file conversion process is due to the inefficiency of the record searching algorithms in Microsoft Excel.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
