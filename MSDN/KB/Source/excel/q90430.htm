Excel: FMT File Not Opened by Lotus 1-2-3 Macro Using /fr

 Last reviewed: July 16, 1997
Article ID: Q90430

The information in this article applies to:
  • Microsoft Excel for Windows, versions 4.0, 4.0a, 5.0

SUMMARY

In Microsoft Excel for Windows, when you run a Lotus 1-2-3 macro that uses /fr to open a .WK1 file, the corresponding FMT file (formatting file) is not opened.

MORE INFORMATION

When you open a Lotus 1-2-3 file by choosing Open from the File menu in Excel, the corresponding FMT or FM3 file is automatically opened.

When opening a file with a macro that uses /fr, the FMT (or FM3) file is not opened and your worksheet displays without the formatting.

You can get your Lotus 1-2-3 macro to work properly by first opening your .WK1 file in Excel and then saving it as an Excel file with a .WK1 extension. This way, it becomes one file with the formatting included in it.
