XL97: Products That Qualify for the MS Excel Upgrade

 Last reviewed: January 7, 1998
Article ID: Q174516
The information in this article applies to:
  • Microsoft Excel 97 for Windows

SUMMARY

Microsoft Excel 97 is available in two versions: a retail version that you can install on any compatible computer, and an upgrade version that requires that a qualifying product be installed on your computer or that Disk 1 of the qualifying product be available when you run the Setup program.

This article lists the products that qualify you to use the upgrade version of Microsoft Excel 97. The retail version does not require a qualifying product.

MORE INFORMATION

Qualifying product names are listed on the left and their version numbers are listed on the right in the following table. 

   Product                                Version
   ----------------------------------------------------------------------

   Microsoft Excel                        All versions for Microsoft
                                          Windows and Windows NT
                                          Workstation
   Microsoft Multiplan                    All versions
   Lotus 1-2-3 for Windows                1.0 through 4.01
   Lotus 1-2-3 for MS-DOS                 2.01 through 3.4
NOTE: When you install Microsoft Excel 97, the Setup program may not correctly recognize that one or more of the qualifying products is installed on your computer. If this problem occurs and you are unable to install Microsoft Excel 97, contact Microsoft Technical Support. Microsoft Technical Support for Microsoft Excel for Windows is available Monday through Friday, 6:00 A.M. to 6:00 P.M. Pacific time at (425) 635-7070. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

      http://www.microsoft.com/worldwide/default.htm
When you speak to a support engineer, indicate that you have a qualifying product that is listed in Microsoft Knowledge Base article Q174516. For additional information, please see the following article in the Microsoft Knowledge Base:
