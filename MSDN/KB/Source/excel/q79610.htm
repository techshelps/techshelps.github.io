Solver Err Msg: Solver an Unexpected Internal Error Occurred

 Last reviewed: November 30, 1994
Article ID: Q79610
The information in this article applies to:
  • Microsoft Excel for Windows, version 3.0
  • Microsoft Excel for the Macintosh, version 3.0
  • Microsoft Excel for OS/2, version 3.0

Summary:

When using Solver with Microsoft Excel version 3.0, you may receive the error message: 

   Solver: An unexpected internal error occurred ($M$115). Please
   verify that your model is correctly entered and all cells contain
   numeric values. If this does not correct the error, please report
   it to your regular Microsoft Excel support contact.
Note: In Microsoft Excel for Windows, the message specifies $M$100 not $M115.

This message appears when Solver is not able to change the value in one of the cells listed in the By Changing Cells box. If this error occurs, one of the By Changing Cells is probably part of an array formula. Since you cannot change part of an array, Solver is not able to attempt to solve the model.

To avoid receiving this message, the array formulas involved need to be converted to single formulas. For more information on array formulas, please refer to the "Microsoft Excel User's Guide" version 3.0 manual, pages 268-274.

Reference(s):

"Microsoft Excel Solver User's Guide," version 3.0, pages 11-16
