XL98: Page Numbers in Printing Dialog Box May Skip Numbers

 Last reviewed: February 2, 1998
Article ID: Q179042
The information in this article applies to:
  • Microsoft Excel 98 Macintosh Edition

SYMPTOMS

When you print a large number of pages in Microsoft Excel 98 Macintosh Edition, the page numbers in the Printing dialog box may skip some numbers. For example, the text in the Printing dialog box my change from "Now printing page 5 of 300" to "Now printing page 7 of 300."

CAUSE

To speed up printing, the Printing dialog box does not update for each printed page. This may cause some page numbers to appear to be skipped, but does not affect the actual number of pages printed.

STATUS

This behavior is by design of Microsoft Excel 98.

REFERENCES

For more information about printing workbooks, click Contents and Index on the Help menu. Click the Index button, and type the following text 

   printing, sheets and workbooks
and then double-click the selected text to go to the "Print the active sheets, a selected range, or an entire workbook" topic.
