XL5: Data Labels Displayed Incorrectly on Quattro Pro Graph

 Last reviewed: April 23, 1997
Article ID: Q114748
5.00 7.00 WINDOWS

The information in this article applies to:

  • Microsoft Excel for Windows, version 5.0
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In Microsoft Excel, when you open a Quattro Pro spreadsheet that contains a stacked bar graph, the following behavior occurs when data labels are displayed:

  • Data labels are displayed for each series that has attached data labels, even though in Quattro Pro, only data labels for the top series display on the graph.

    -and-

  • The data labels for the top series are displayed inside the series instead of above the series like they do in Quattro Pro.

WORKAROUND

You cannot control the way the chart is imported into Microsoft Excel. However, you can format the chart to be displayed as it appears in Quattro Pro by turning off the display of data labels for a selected series, and moving the data labels above the top series on a chart.

To turn off the display of data labels for a selected series

  1. Select the series that has the attached data labels.

  2. From the Insert menu, choose Data Labels.

  3. Under Data Labels, select None and choose OK.

To move data labels above the top series on a chart

Select a single data label on the top series and drag the label to the desired location above the top series. Each data label must be moved individually, although you can still select the labels as a group and apply formatting.
