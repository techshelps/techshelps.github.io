HOWTO: Determine Which Version of MDAC 1.5 Is on Your Computer

 Last reviewed: January 14, 1998
Article ID: Q178840
The information in this article applies to:
  • Microsoft Data Access Components, version 1.5
  • Microsoft Open Database Connectivity, version 3.5
  • Microsoft OLE DB, version 1.5
  • ActiveX Data Objects (ADO), version 1.5
  • Remote Data Service for ADO version 1.5

SUMMARY

The Microsoft Data Access Components version 1.5 has had five separate releases, each with different versions of the Open Database Connectivity (ODBC), OLE DB, ActiveX Data Objects (ADO), and Remote Data Service (RDS) components, as well as ODBC Drivers and OLE DB providers. This article presents details about when each version was released, and how you can determine if you have the latest version, Microsoft Data Access Components (MDAC) 1.5c, on your computer (or whether you need to upgrade).

MORE INFORMATION

The table below indicates what version of MDAC was distributed with a given product or distribution channel. 

   MDAC Release             Comments
   -----------------------------------------------------------------------

   MDAC 1.5 w/IE4.0         Shipped with IE 4.0 and IE 4.0 Client SDK
   MDAC 1.5 w/PDC           Distributed on the Professional Developers
                            Conference CD (Windows NT-only) and with the
                            PDC CD distributed with Nov. '97 MSDN. (You can
                            find the MDAC installation programs included
                            with the IslandHopper sample). This includes
                            some fixes to ODBC, OLEDB, ADO, and RDS.
   MDAC 1.5a                Available on the Web from 10/6/97 to 12/17/97,
                            includes numerous fixes to ODBC, OLEDB, ADO,
                            and RDS.
   MDAC 1.5b                Available with IIS 4.0 & Windows NT 4.0 Option
                            Pack, includes numerous fixes to ODBC, OLEDB,
                            ADO, and RDS.
   MDAC 1.5c                Available on the Web after 12/18/97, includes
                            fixes to ADO and ODBC (only).
