"Send Later" Message Sent Immediately When New Message Is Sent

 Last reviewed: November 26, 1997
Article ID: Q174453
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SYMPTOMS

When you compose a message and choose Send Later, the message is saved in your Outbox for sending later. However, sending another message causes the "Send Later" message in the Outbox to be sent immediately.

CAUSE

Outlook Express cannot deliver one of two or more messages in the Outbox at a time. When any message is sent, all messages in the Outbox are sent at the same time.

RESOLUTION

To work around this issue, move messages that must be sent later to the Drafts folder and then send them when you want.
