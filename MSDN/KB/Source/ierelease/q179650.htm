Error Message: Error Installing Dependency

 Last reviewed: January 21, 1998
Article ID: Q179650
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95

SYMPTOMS

When you install components for Internet Explorer, you may receive the following error message: 

   Error Installing Dependency

RESOLUTION

To prevent the error message from appearing, use one of the following methods:

Remove Setup Folder

  1. Click Start, point to Settings, and then click Control Panel.

  2. In Control Panel, double-click Add/Remove Programs.

  3. On the Install/Uninstall tab, click Microsoft Internet Explorer 4.0 Setup Files in the list of installed programs.

  4. Click Add/Remove, click Yes to remove the Internet Explorer Setup files, and then click OK.

Upgrade Browser

To upgrade only the Internet Explorer browser from the Internet Explorer 4.0 Component Download Web page, follow these steps:

  1. Click Start, point to Settings, and then click Control Panel.

  2. In Control Panel, double-click Add/Remove Programs.

  3. On the Install/Uninstall tab, click Microsoft Internet Explorer 4.0 in the list of installed programs, and then click Add/Remove.

  4. Click "Add a component to Internet Explorer," and then click OK.

  5. Click No when you receive the following prompt: 

          Active Setup is about to determine what Internet components are
      installed on your computer. Is this OK?

  6. Click the "Internet Explorer 4.0 Web Browser" check box to select it.

    NOTE: Do not select any other components to upgrade at this time.

  7. Click Next. Follow the instructions to install the Internet Explorer browser.
