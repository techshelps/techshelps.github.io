Internet Explorer Truncates Favorites Names

 Last reviewed: September 29, 1997
Article ID: Q172521

The information in this article applies to:

  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0 for Windows 95

SYMPTOMS

When you click Organize Favorites on the Favorites menu, Web addresses may appear truncated.

CAUSE

This behavior can occur for addresses whose name ends with a space.

RESOLUTION

To resolve this issue, rename the affected addresses without spaces. To rename an address, click the address to select it, and then click Rename.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Explorer versions 3.0, 3.01, 3.02, and 4.0 for Windows 95. We are researching this issue and will post new information here in the Microsoft Knowledge Base as it becomes available.
