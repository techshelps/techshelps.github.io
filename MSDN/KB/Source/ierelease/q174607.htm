Margin Settings Not Saved in Internet Explorer

 Last reviewed: November 26, 1997
Article ID: Q174607
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

If you change the top or bottom margins in the Page Setup dialog box, the next time you start Internet Explorer the margins are reset to the default setting of 0.75 inches.

RESOLUTION

To work around this problem, you must set the margins you want each time you start Internet Explorer. To set the margins, click Page Setup on the File menu.

STATUS

Microsoft has confirmed this to be a problem in Internet Explorer 4.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

If you change only the left and right margins, the margin settings are retained.

Note that by design, changes to the settings in the Paper, Headers And Footers, and Orientation areas in the Page Setup dialog box are not retained.
