Font Size Change Not Saved in Internet Explorer 4.0

 Last reviewed: March 20, 1998
Article ID: Q174378
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0
  • Microsoft Internet Explorer version 4.0 for Windows 3.1
  • Microsoft Internet Explorer version 4.01 for UNIX on Sun Solaris

SYMPTOMS

When you change the font size in Internet Explorer by using the Fonts command on the View menu or the Fonts button on the toolbar and then restart Internet Explorer, the font size is reset.

RESOLUTION

To change the default font size in Internet Explorer, follow these steps:

  1. On the View menu, click Internet Options.

  2. On the General tab, click Fonts.

  3. In the Font Size box, click the font size you want.

  4. Click OK.

  5. Click OK.

STATUS

This behavior is by design. Unlike Internet Explorer 3.0, you cannot change the default font size in Internet Explorer 4.0 by using the Fonts command on the View menu or Fonts button on the toolbar.
