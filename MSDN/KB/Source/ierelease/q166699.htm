Cannot Download Site List for Internet Explorer 4.0

 Last reviewed: September 29, 1997
Article ID: Q166699

The information in this article applies to:

  • Microsoft Internet Explorer version 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SYMPTOMS

When you are downloading the site list in the Active Setup Wizard for Internet Explorer 4.0, an error message stating that Internet Explorer is not set up may be displayed. When this occurs, Setup stops.

CAUSE

This behavior can occur if any of the following conditions occurs:

  • You are not connected to the Internet.
  • The Microsoft Web site is too busy.
  • No sites are available for the Setup language you are using.

RESOLUTION

To resolve this issue, use the appropriate method:

  • If you are not connected to the Internet, establish a connection and then run the Active Setup Wizard again. To verify that you are connected to the Internet, start Internet Explorer and then click Search. If the Search page appears, you are connected to the Internet. For information on how to run the Active Setup Wizard, view the following Microsoft Web site: 

          http://www.microsoft.com/ie
  • If the Microsoft Web site is too busy, try running the Active Setup Wizard again later, or connect to the Internet Explorer home page with your browser to see if the site is busy.
  • If no sites are available for the Setup language you are using, there is a possibility that the download site list does not have a matching language site for you.
