Clicking Footnote Link Displays Top of Web Page

 Last reviewed: September 29, 1997
Article ID: Q169537
The information in this article applies to:
  • Microsoft Internet Explorer versions 2.0, 3.0, 3.01 for Windows 95
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02 for Windows NT 4.0

SYMPTOMS

When you click a link that points to a footnote at the bottom of a Web page, the top of the page may be displayed instead.

CAUSE

This behavior can occur when the footnote link is not valid.
