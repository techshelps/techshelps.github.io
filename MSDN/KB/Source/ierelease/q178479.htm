Cannot Edit Java Custom Settings Using Internet Explorer 4.0

 Last reviewed: December 23, 1997
Article ID: Q178479
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0
  • Microsoft Internet Explorer version 4.0 for Windows 95

SYMPTOMS

When you try to modify your Java custom settings, the Edit button may be missing.

RESOLUTION

To resolve this problem, follow these steps:

  1. Click Start, point to Find, and then click Files Or Folders.

  2. In the Named box, type "zonedon.reg" (without quotation marks).

  3. Click Find Now.

  4. Double-click the Zonedon.reg file.

STATUS

This behavior is by design.

Note that if you upgrade to Internet Explorer 4.01, an Edit tab is added. To upgrade to Internet Explorer 4.01, visit the following Microsoft Web site: 

   http://www.microsoft.com/ie/download/
