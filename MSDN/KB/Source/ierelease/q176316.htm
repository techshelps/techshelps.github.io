Save Target As and Print Target Commands Are Unavailable

 Last reviewed: November 26, 1997
Article ID: Q176316
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95

SYMPTOMS

When you right-click a Hypertext Markup Language (HTML) link in Internet Explorer, the Save Target As and Print Target commands may be unavailable. In addition, the Print All Linked Documents check box may be unavailable when you click Print on the File menu.

CAUSE

This behavior can occur if Content Advisor is enabled.

RESOLUTION

To work around this behavior, use either of the following methods:

METHOD 1

Disable Content Advisor by following these steps in Internet Explorer:

  1. On the View menu, click Internet Options.

  2. Click the Content tab.

  3. Under Content Advisor, click Disable.

  4. Type the supervisor password in the Password box, and then click OK.

  5. Click OK when you receive the following message: 

          Content Advisor has been turned off. Make sure you turn it on again
      before you allow someone else to use your computer.

  6. Click OK.

METHOD 2

Click the HTML link to load the Web page, and then click Save As or Print on the File menu. This assumes that the Web page successfully passes through Content Advisor.

STATUS

This behavior is by design.

MORE INFORMATION

When Content Advisor is enabled, you can save or print a Web page only after it passes through Content Advisor. You cannot save or print a Web page by right-clicking an HTML link and then clicking Save Target As or Print Target, because this method bypasses Content Advisor.
