FrontPage 98 Starts When You Edit Stationery in Outlook Express

 Last reviewed: January 30, 1998
Article ID: Q180135
The information in this article applies to:
  • Microsoft FrontPage 98 for Windows
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows 95
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you try to edit stationery in Outlook Express, FrontPage 98 may start.

CAUSE

This behavior can occur if FrontPage 98 is your default Hypertext Markup Language (HTML) editor.

STATUS

This behavior is by design.

MORE INFORMATION

For information about how to change the default HTML editor, see the following article in the Microsoft Knowledge Base:

   TITLE     : Changing Source Code Editor in Internet Explorer
