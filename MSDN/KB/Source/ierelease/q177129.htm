How to Change Your Internet Explorer Start Page

 Last reviewed: November 21, 1997
Article ID: Q177129
The information in this article applies to:
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0 for Windows 95

SUMMARY

This article describes how to change your Internet Explorer start page.

MORE INFORMATION

The start page is the Web page that is loaded when you start Internet Explorer. To change your Internet Explorer start page, use the appropriate method:

Internet Explorer 3.x

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Internet.

  3. Click the Navigation tab.

  4. In the Page box, click Start Page.

  5. In the Address box, type the Web address of the Web page you want to be your start page.

  6. Click OK.

Internet Explorer 4.0

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Internet.

  3. In the Address box, type the Web address of the Web page you want to be your start page.

  4. Click OK.
