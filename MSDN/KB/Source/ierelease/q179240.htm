Microsoft Internet Explorer 4.0 for Macintosh Available

 Last reviewed: January 13, 1998
Article ID: Q179240
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Macintosh

SUMMARY

Microsoft has made Internet Explorer 4.0 for Macintosh available. This suite of Internet products for Macintosh includes the following programs:

  • Microsoft Internet Explorer 4.0 for Macintosh
  • Microsoft Outlook Express 4.0 for Macintosh
  • Microsoft Personal Web Server 4.0 for Macintosh
  • Microsoft NetShow 2.0 for Macintosh

MORE INFORMATION

You can obtain Internet Explorer 4.0 for Macintosh from the following Microsoft Web site: 

   http://www.microsoft.com/ie/mac/
New features in Internet Explorer 4.0 for Macintosh include:
  • Easy Setup and configuration: Setup automatically detects Internet and bookmark preferences from other browser programs, and includes the option to import them into Internet Explorer.
  • Active Setup: With Active Setup you can choose the components you want to download before you download the installer program.
  • AutoComplete: AutoComplete automatically remembers Web addresses you have typed before, and fills in the Address box for you.
  • Offline Browsing: You can use Offline Browsing to download your favorite Web pages for viewing at a later time. You can then view these Web pages when you are not connected to the Internet.
  • Subscriptions: With subscriptions you can subscribe to your favorite Web sites to receive notification when they are updated.
  • Explorer bars: You can use Explorer bars to enhance your searches, view your access history, and keep track of your subscriptions and channels.
  • Find in Favorites and History: With this feature you can search through pages you have already visited.
  • Security features: You can set up zones for Web sites to protect your computer while you are connected to the Internet. We include the Platform for Internet Content Selection (PICS) ratings system to screen out objectionable Web sites and content.
  • ToolTips: With ToolTips you can move your pointer over a toolbar button to find out what it does. You can use this feature to get information about search results without accessing Web pages.
  • Macintosh Operating System version 8.0 Contextual Menu support: To view a menu of options for a folder or Web page, press CTRL and click the folder or Web page.
  • Dynamic Hypertext Markup Language (DHTML): This feature can be used to help develop animated and interactive Web pages.
  • Java support: Internet Explorer includes Java support.
  • Active Channels: With Active Channels you can subscribe to channels that display active content. There is no fee or charge associated with this type of subscription.
