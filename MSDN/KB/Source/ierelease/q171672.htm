Error Displaying Channel Bar with Microsoft Office Manager

 Last reviewed: January 12, 1998
Article ID: Q171672

The information in this article applies to:

  • Microsoft Internet Explorer version 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SYMPTOMS

If the Microsoft Office Manager toolbar is enabled, you may receive the following error message as the channel bar is displayed: 

   Internet Explorer caused a General Protection Fault in module
   Shell32.dll

RESOLUTION

To work around this problem, disable the Microsoft Office Manager toolbar.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Explorer version 4.0 for Windows 95 and Windows NT 4.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Microsoft Office Manager toolbar is included with Microsoft Office 4.3 only. Microsoft Office 7.x (Office 95) and Microsoft Office 97 include the Office Shortcut Bar.
