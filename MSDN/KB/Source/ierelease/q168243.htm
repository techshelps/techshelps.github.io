How to Search the Knowledge Base for Internet Explorer Articles

 Last reviewed: January 7, 1998
Article ID: Q168243
The information in this article applies to:
  • Microsoft Internet Explorer versions 2.0, 3.0, 3.01 for Windows NT 3.51
  • Microsoft Internet Explorer versions 2.0, 2.1, 3.0, 3.0a, 3.01 for Macintosh
  • Microsoft Internet Explorer versions 2.0, 2.01, 2.1, 3.0, 3.01 for Windows 3.1
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0 for Windows NT 4.0
  • Microsoft Internet Explorer versions 2.0, 3.0, 3.01, 3.02, 4.0 for Windows 95

SUMMARY

This article describes how to search the Microsoft Support Online Web site for Internet Explorer information. The following topics are discussed:

  • Basic text searches of the Support Online Web site
  • Advanced text searches of the Support Online Web site

MORE INFORMATION

Basic Text Searches of the Microsoft Support Online Web Site

To perform a basic text search of the Microsoft Support Online Web site, follow these steps:

  1. Connect to the Microsoft Support Online Web site at: 

          http://support.microsoft.com/support/

  2. In the My Question Is About box, click Internet Explorer.

  3. In the My Question Is box, type one or more query words that pertain to your issue.

  4. Click Find. You are then presented with a list of articles that contain the query words you typed.

  5. Look at the article titles for topics that may apply to your issue, and then click a title to view the full text of the article.

Advanced Text Searches of the Microsoft Support Online Web Site

If a basic text search does not produce the results you want, you may need to try another search using advanced techniques. To use advanced techniques for searching, follow the steps for a basic search and add one or more of the methods listed below to step 3:

  • View search hints at the following Microsoft Web site: 

          http://support.microsoft.com/support/howto/default.asp

    This site lists advanced techniques for searching.
