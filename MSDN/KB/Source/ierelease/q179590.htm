How to Identify the Version of Outlook Express in Use

 Last reviewed: January 20, 1998
Article ID: Q179590
The information in this article applies to:
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows 95
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows NT 4.0

SUMMARY

This article describes how to identify the version of Outlook Express installed on your computer.

MORE INFORMATION

To determine the version of Outlook Express installed on your computer, look at the information in the About Microsoft Outlook Express dialog box, or the properties of the Msimn.exe file. To do so, follow these steps:

In Outlook Express

  1. In Outlook Express, click About Microsoft Outlook Express on the Help menu.

  2. In the About Microsoft Outlook Express dialog box, note the version number above the Copyright notice.

Properties of the Msimn.exe File

  1. Click Start, point to Find, and then click Files Or Folders.

  2. In the Named box, type "msimn.exe" (without the quotation marks), and then click Find Now.

  3. Right-click the Msimn.exe file, and then click Properties.

  4. Click the Version tab. The version of Outlook Express is displayed in the File Version box.
