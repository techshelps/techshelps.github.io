Red Gleams Do Not Appear on Channel Bar or Channel Pane

 Last reviewed: November 26, 1997
Article ID: Q175180
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you subscribe to a channel from the Channel bar and then update the subscription, a gleam may not appear on that channel's icon even though the contents of the subscribed Web site have changed.

STATUS

This behavior is by design.

MORE INFORMATION

When Microsoft Internet Explorer 4.0 checks a subscribed channel for updated information, it only checks the actual subscribed file. If the subscribed file changes, a red gleam appears on that channel's icon. If the subscribed file does not change, no gleam appears on the subscription's icon. This is true whether or not any other file in the subscribed channel has changed.
