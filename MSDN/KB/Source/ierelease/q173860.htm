Err Msg: Are You Sure You Want to Navigate Away from This Page

 Last reviewed: September 29, 1997
Article ID: Q173860
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SYMPTOMS

When you install additional components from the Internet Explorer 4.0 Components Download page, and you attempt to change Windows Explorer settings during the installation of the additional components, you may receive the following error message: 

   Are you sure you want to navigate away from this page?
   You would interrupt the setup process.
   Press OK to continue, or Cancel to stay on the current page.

RESOLUTION

Click Cancel and wait until Active Setup is finished before you change settings in Windows Explorer.

MORE INFORMATION

If you click OK, and then change Windows Explorer settings, the Active Setup ActiveX control (Asctrls.ocx) is closed, causing Setup to abort.
