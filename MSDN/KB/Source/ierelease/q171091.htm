Error Message: The Target File Exists and Is Newer Than...

 Last reviewed: October 1, 1997
Article ID: Q171091
The information in this article applies to:
  • Microsoft Internet Explorer versions 3.02, 4.0 for Windows 95
  • Microsoft Internet Explorer versions 3.02, 4.0 for Windows NT 4.0

SYMPTOMS

When you install Internet Explorer version 3.02 with Internet Mail after uninstalling Internet Explorer version 4.0, the following error message may occur: 

  The target file exists and is newer than the source.
  Overwrite the newer file?
The error message refers to the Wldap32.dll file.

CAUSE

This error message can occur if the Wldap32.dll file is installed by Internet Explorer version 4.0. This file can also be installed by other programs, and therefore is not removed when you uninstall Internet Explorer version 4.0.

RESOLUTION

Click Yes to allow Internet Explorer Setup to overwrite the existing Wldap32.dll file.
