Err Msg: Your Configuration Requires 12 M of RAM to Install...

 Last reviewed: January 13, 1998
Article ID: Q175595
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95

SYMPTOMS

If you run Internet Explorer Setup on a computer with 8 megabytes (MB) of random access memory (RAM), you may receive the following error message:

  • Your configuration requires 12 M of RAM to install Internet Explorer 4.0. Setup cannot continue.

Note that the minimum listed requirement for Internet Explorer 4.0 and 4.01 with Windows 95 is 12 MB of RAM.

CAUSE

This behavior can occur if you have a computer with 8 MB of RAM and a registry that is at least 1.6 MB in size. Internet Explorer cannot be installed under these conditions.

RESOLUTION

You may want to install more RAM in your computer. To work around this behavior, you can uninstall programs to remove entries in the registry.

STATUS

This behavior is by design.
