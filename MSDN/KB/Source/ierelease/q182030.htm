"Stop 1A" Error Message Printing from Internet Explorer

 Last reviewed: March 4, 1998
Article ID: Q182030
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you print from Internet Explorer, you may receive the following error message: 

   Stop 0x0000001A: Memory_managment

CAUSE

This error message can occur when you are using the NEC SuperScript 860 printer driver version 0.04 from NEC for Windows NT 4.0.

RESOLUTION

To work around this issue, use another printer driver, such as the Hewlett-Packard (HP) LaserJet 4 printer driver, until an updated driver is available from NEC. For information about printer driver compatability, consult your printer documentation or contact NEC Technical Support.

MORE INFORMATION

The STOP 0x1A error message may not occur with all pages that you print.
