"Highlight Links When Clicked" Option Does Not Function

 Last reviewed: December 9, 1997
Article ID: Q177866
The information in this article applies to:
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you click a link in Internet Explorer 4.0 or 4.01, an outline is drawn around the link. In Internet Explorer 3.x, there is no outline drawn around the link. Changing the Highlight Link When Clicked option has no effect.

CAUSE

This option is disabled in Internet Explorer 3.x and 4.0.

RESOLUTION

The option has been removed in Internet Explorer 4.01. In earlier versions of Internet Explorer, there is no workaround to change the default behavior of the browser.

STATUS

Internet Explorer 4.01 has been modified to remove the Highlight Links When Clicked option. The default behavior is retained from Internet Explorer 4.0.

Microsoft has confirmed this to be a problem in Internet Explorer versions 3.0, 3.01, 3.02, and 4.0 for Windows 95 and Windows NT 4.0.
