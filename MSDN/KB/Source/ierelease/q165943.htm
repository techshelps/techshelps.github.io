Newsgroup Viewing Options for Outlook Express

 Last reviewed: September 29, 1997
Article ID: Q165943

The information in this article applies to:

  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SUMMARY

This article lists the newsgroup viewing options available and describes how to change the current settings in Microsoft Outlook Express.

MORE INFORMATION

Outlook Express has four options for viewing newsgroup messages:

  • All Messages
  • Unread Messages
  • Downloaded Messages
  • Filtered Messages

To change the viewing option, click Options on the View menu, click Current View, and then click the option you want.
