Able to Modify Channel Screen Saver Properties When Locked

 Last reviewed: March 6, 1998
Article ID: Q182132
The information in this article applies to:
  • Microsoft Internet Explorer version 4.01 for Windows NT 4.0

SYMPTOMS

You may be able to modify the properties of the Channel screen saver on a locked Windows NT 4.0 server or workstation.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Explorer version 4.01 for Windows NT 4.0.

MORE INFORMATION

When a Windows NT 4.0 server or workstation is locked with the Internet Explorer 4.01 Channel screen saver, you can modify some of the screen saver's properties.

Note that you can modify only certain "benign" properties of the locked screen saver. You cannot perform operations on the locked computer or bypass the normal Windows NT security settings.
