Page Footers Do Not Appear When Printing Web Pages

 Last reviewed: January 23, 1998
Article ID: Q174560
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95

SYMPTOMS

Page footers may not appear when you print Web pages to some printers.

CAUSE

Some printers have a large non-printable area at the bottom of the page. When you print a Web page to one of these printers, the footer text may be cut off.

STATUS

Microsoft has confirmed this to be a problem in Internet Explorer 4.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
