The information in this article applies to:

  • Microsoft Internet Explorer version 4.0 for Windows 3.1
  • Microsoft Internet Explorer version 4.0 for Windows NT 3.51

SUMMARY

Microsoft has made Microsoft Internet Explorer 4.0 Platform Preview 2.0 for Windows 3.1 and Windows NT 3.51 available.

MORE INFORMATION

Platform Preview 2.0 for Windows 3.1 and Windows NT 3.51 is now available. You can obtain Internet Explorer 4.0 Platform Preview 2.0 for Windows 3.1 and Windows NT 3.51 from the following Microsoft Web site: 

   http://www.microsoft.com/ie/win31/ie40
