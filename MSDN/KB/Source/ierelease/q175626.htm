Download Time on Component Download Page Appears Incorrect

 Last reviewed: November 26, 1997
Article ID: Q175626
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you download add-on components from the Internet Explorer 4.0 Component Download Web page, the estimated download time displayed on this page may be different from the estimated download time displayed by Active Setup.

MORE INFORMATION

The download time calculations on the Component Download page are based on a static formula that assumes you have a 28.8 modem connection and a reasonable data transfer rate over the Internet. The formula does not take into account other possible factors that affect speed, such as network traffic, how busy the server is, and the actual speed of the connection to the Internet.

The estimated download time on the Component Download page is provided to give you the approximate time it takes to download a component before you decide to download it.
