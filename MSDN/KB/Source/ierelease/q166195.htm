NM: "A File Being Copied Is Older..." Installing Internet Explor

 Last reviewed: March 17, 1998
Article ID: Q166195
The information in this article applies to:
  • Microsoft NetMeeting version 2.0 for Windows 95
  • Microsoft Internet Explorer version 3.02 for Windows 95

SYMPTOMS

You may receive the following message when you install Internet Explorer version 3.02 for Windows 95 using the Full Install option: 

   A file being copied is older than the file currently on your computer.
   It is recommended that you keep your existing file.

      File name:    <filename>
      Description:  <description of file>
      Your version: <version number>

   Do you want to keep this file?
You have the option to click Yes (to keep the existing file) or No (to overwrite the file).

This message occurs once for each of the following files:

  • Wldap32.dll
  • Cb32.exe
  • Conf.exe
  • Gcc32.dll
  • Gccnc.dll
  • Lhacm.acm
  • Mcatipx.dll
  • Mcatpstn.dll
  • Mcs32.dll
  • Mcnc.dll
  • Mnmal_.dll
  • Mnmalp.exe
  • Mnmclp_p.dll
  • Mnmcmg_g.dll
  • Mnmcpi32.dll
  • Mnmft_.dll
  • Mnmmg_.dll
  • Mnmmsg.dll
  • Mnmnet.exe
  • Mnmnet_.dll
  • Mnmom.exe
  • Mnmom_.dll
  • Mnmshco.exe
  • Mnmshco_.dll
  • Mnmtapm_.dll
  • Mnmtddm_.dll
  • Mnmtddn_.dll
  • Mnmtdds_.dll
  • Mnmthk16.dll
  • Mnmut_.dll
  • Mnmvdd.386
  • Mnmwb_b.dll
  • Msconf.dll
  • Msgccmcs.dll
  • Wb32.exe

CAUSE

A Beta version of NetMeeting 2.0 is currently installed on your computer. When you install Internet Explorer 3.02 for Windows 95 using the Full Install option, NetMeeting version 1.0 is installed by default.

RESOLUTION

If you want to continue to use NetMeeting 2.0, use one of the following methods:

  • Install the Minimum Install option or the Standard Install option of Internet Explorer 3.02 instead of the Full Install option. The Minimum Install and Standard Install options of Internet Explorer 3.02 do not include NetMeeting 1.0.
  • Click Yes (to keep the existing NetMeeting 2.0 file) each time you are prompted.

    NOTE: If you click No (to overwrite the NetMeeting 2.0 files) for any of the files by mistake, NetMeeting 2.0 may not run properly and you may receive the following error message when you start NetMeeting 2.0: 

          Illegal operation. Conf executed an invalid instruction in
      module <unknown>

    You must reinstall NetMeeting 2.0 to correct this problem.

If you do not want to continue using NetMeeting 2.0, uninstall NetMeeting 2.0 before running Internet Explorer 3.02 Setup. To uninstall NetMeeting 2.0, follow these steps:

  1. Click Start, point to Settings, and then click Control Panel.

  2. Double-click Add/Remove Programs.

  3. Click Microsoft NetMeeting 2.0 Beta <x> (where <x> is the beta number) in the list of installed programs, and then click Add/Remove.

  4. Click Yes to uninstall NetMeeting 2.0.

To set up NetMeeting 1.0, reinstall Internet Explorer 3.02 using the Full Install option.
