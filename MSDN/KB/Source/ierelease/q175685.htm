Error Message: FPEXPRESS This Program Has Performed An...

 Last reviewed: October 28, 1997
Article ID: Q175685
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows 95
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SYMPTOMS

When you try to perform a dragging operation (for example, to copy and paste text) in FrontPage Express, you may receive the following error message: 

   FPEXPRESS

   This program has performed an illegal operation and wil be shut down.
   If the problem persists, contact the program vendor.

STATUS

Microsoft has confirmed this to be a problem in Microsoft FrontPage Express. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
