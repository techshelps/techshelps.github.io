How Internet Explorer Installs Netscape Plug-ins

 Last reviewed: September 29, 1997
Article ID: Q157217
3.00 WINDOWS kbenv kb3rdparty kbsetup

The information in this article applies to:

  • Microsoft Internet Explorer version 3.0 for Windows 95

SUMMARY

This article describes how Internet Explorer 3.0 detects Netscape plug-ins during startup.

MORE INFORMATION

Each time Internet Explorer starts, it searches for installed plug-ins. Internet Explorer searches for installed plug-ins in the following order:

  1. Searches for .ocx files in the Windows\System folder.

  2. Searches the following registry key: 

          HKey_Local_Machine\Software\Microsoft\Internet Explorer\Plugins

    NOTE: Each time Internet Explorer 3.0 is started, it searches and cleans this registry path, and then adds located plug-ins to the Extension and MIME key fields.

  3. Searches the Internet Explorer Plug-ins folder for installed DLLs. These are typically located in the Program Files\Microsoft Internet\Plugins folder.

  4. Searches the Netscape Navigator Plug-ins folder for installed DLLs. By default, this is the Program Files\Netscape\Navigator\Program\Plugins folder.
