Map Network Drive Button Missing from Toolbars

 Last reviewed: November 26, 1997
Article ID: Q172568
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

After you install Internet Explorer 4.0, the Map Network Drive button may be missing from your toolbars.

RESOLUTION

To add the Map Network Drive button to your toolbars, follow these steps:

  1. Double-click the My Computer icon on the desktop.

  2. On the View menu, click Folder Options.

  3. On the Advanced tab, click the "Show Map Network Drive button in toolbar" check box to select it.

  4. Click OK, and then close My Computer.

STATUS

This behavior is by design.
