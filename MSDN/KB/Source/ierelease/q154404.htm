Err Msg: You Are About to Use a Windows System Directory...

 Last reviewed: September 29, 1997
Article ID: Q154404
3.00 3.01 3.02 WINDOWS kbui kbenv

The information in this article applies to:

  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0 for Windows 95

SYMPTOMS

When you use Internet Explorer to try to move the Temporary Internet Files folder (also known as the cache folder) to a root folder or the Windows\System folder, you receive the following error message: 

   You are about to use a Windows system directory or a root directory.
   Please use another directory.
There is no option to override this error message.

CAUSE

You cannot move the Temporary Internet Files folder to a root folder or the Windows\System folder.

RESOLUTION

To resolve this issue, move the Temporary Internet Files folder to a new folder in a different location. For additional information on how to change the location of the Temporary Internet Files folder, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q172949
   TITLE     : How to Change the Location of Temporary Internet Files
