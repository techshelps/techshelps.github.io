Cannot View Web Pages in Internet Explorer When Working Offline

 Last reviewed: January 15, 1998
Article ID: Q179322
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you restart Internet Explorer and work offline, you may not be able to view Web pages you viewed previously.

CAUSE

This behavior can occur if you configure Internet Explorer to delete Web pages saved in the Temporary Internet Files folder when you quit Internet Explorer.

RESOLUTION

To resolve this behavior, you can change the security setting. To do so, follow these steps:

  1. On the View menu in Internet Explorer, click Internet Options.

  2. Click the Advanced tab, and then click the "Delete saved pages when browser closed" check box to clear it.

  3. Click OK.
