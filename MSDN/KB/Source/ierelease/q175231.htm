Toolbar Buttons Not Displayed Properly When Toolbar is Moved

 Last reviewed: October 16, 1997
Article ID: Q175231
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

In some programs, when you drag a toolbar to a new location so that it consists of more than one row of buttons, the bottom row of buttons may not be displayed properly.

CAUSE

This problem can occur after you install Internet Explorer. The Comctl32.dll file installed by Internet Explorer may cause display problems with programs that use Microsoft Foundation Classes (MFCs). Microsoft WordPad for Windows NT 4.0 is an example of an MFC program.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Explorer version 4.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
