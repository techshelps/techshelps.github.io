Only the Desktop.ini File Appears in the Subscriptions Window

 Last reviewed: March 11, 1998
Article ID: Q182317
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you click Manage Subscriptions on the Favorites menu in Internet Explorer, only the Desktop.ini file may appear in the Subscriptions window.

CAUSE

This behavior can occur if the read-only attribute is removed from the Subscriptions folder.

RESOLUTION

To resolve this behavior, set the read-only attribute on the Subscriptions folder. To do so, follow these steps:

  1. Click Start, point to Programs, and then click Windows Explorer or Windows NT Explorer.

  2. Locate the Subscriptions folder.

    The Subscriptions folder has the following default location for Windows 95: 

          C:\Windows\Subscriptions

    The Subscriptions folder has the following default location for Windows NT: 

          C:\Winnt\Subscriptions

  3. Right-click the Subscriptions folder, and then click Properties.

  4. On the General tab, click the Read-Only check box to select it, and then click OK.
