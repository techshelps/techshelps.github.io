Removing Internet Explorer for Windows NT 4.0 Using Ieremove.exe

 Last reviewed: October 22, 1997
Article ID: Q166314
The information in this article applies to:
  • Microsoft Internet Explorer version 4.0 for Windows NT 4.0

SUMMARY

This article describes how to remove Internet Explorer 4.0 for Microsoft Windows NT 4.0 by using the Ieremove.exe tool.

Follow the steps described in this article in the following situations:

  • You tried to uninstall Internet Explorer 4.0, but you were unable to do so.

Do not follow the steps described in this article in the following situations:
  • You have not tried to uninstall Internet Explorer 4.0 by using the steps described in the following article: 

          ARTICLE-ID: Q174265

          TITLE     : How to Uninstall Internet Explorer 4.0


  • You have removed the Internet Explorer 4.0 backup files. If you removed
   the backup files and you are unable to uninstall Internet Explorer 4.0
   by using the steps described in the article mentioned in the previous
   bulleted item, you cannot remove Internet Explorer 4.0 without
   reinstalling Microsoft Windows NT in a different folder.

    

MORE INFORMATION
 


Before proceeding, note the location of the "Integrated Browser NT.dat"
file and the Ie4bak.dat file. To do this, follow these steps:


  1. Click Start, point to Find, and then click Files Or Folders.


  2. In the Named box, type "Integrated Browser NT.dat" (with quotation
   marks), and then click Find Now.


  3. Note the path in the In Folder column for the "Integrated Browser
   NT.dat" file.


  4. Repeat steps 1-3 for the Ie4bak.dat file.


  5. On the File menu, click Close.

    

To remove Internet Explorer 4.0, use the Ieremove.exe tool. The following
steps guide you through uninstalling the Windows Desktop Update Component
and Internet Explorer.


Note that you should uninstall the Windows Desktop Update Component before
you uninstall Internet Explorer 4.0.


NOTE: Ieremove.exe is a command line executable designed to remove 
Internet Explorer and its components. Ieremove.exe may not remove 
Internet Explorer without using command line options. For information 
about the command line options avialable for Ieremove.exe, type "IEREMOVE 
/?" (without quotes).



Uninstall the Windows Desktop Update Component Using Ieremove.exe

 


  1. Restart your computer, and then log in to Windows NT.


  2. Press CTRL+ALT+DELETE to open the Windows NT Security dialog box,
   and then click Task Manager.


  3. On the File menu, click New Task (Run), type the following command in
   the Open box (with quotation marks), and then click OK

    

          "<path1>\ieremove" /i:"<path2>\integrated browser nt.dat"

    
   where <path1> is the full path to the Ieremove.exe file, and <path2> is
   the full path to the "Integrated Browser NT.dat" file.

    
   For example, if Windows NT is installed in the C:\Winnt folder,
   the Ieremove.exe file is located in the C:\Winnt\System32 folder, and
   the "Integrated Browser NT.dat" file is located in the C:\Program
   Files\Plus!\Microsoft Internet\Uninstall Information folder, type the
   following command (with quotation marks):

    

          "c:\winnt\system32\ieremove" /i:"c:\program files\plus!\microsoft
      internet\uninstall information\integrated browser nt.dat"

    

  4. Click OK.


  5. Minimize the Task Manager window.


  6. At the Internet Explorer 4.0 Uninstall prompt, click OK.


  7. Click Yes, and then click OK.


  8. Close Task Manager.


  9. Restart your computer.

    

Uninstall Internet Explorer Using Ieremove.exe

 


  1. Click Start, and then click Run.


  2. In the Open box, type the following command (with quotation marks), and
   then click OK

    

          "<path1>\ieremove" /n:"<path2>\ie4bak.dat"

    
   where <path1> is the full path to the Ieremove.exe file, and <path2> is
   the full path to the Ie4bak.dat file.

    
   For example, if Windows NT is installed in the C:\Winnt folder,
   the Ieremove.exe file is located in the C:\Winnt\System32 folder, and
   the Ie4bak.dat file is located in the C:\Winnt folder, type the
   following command (with quotation marks):

    

          "c:\winnt\system32\ieremove" /n:"c:\winnt\ie4bak.dat"

    

  3. Click OK, and then click OK again.


  4. After receiving confirmation that the Ieremove.exe tool has
   successfully uninstalled Internet Explorer 4.0, reinstall Internet
   Explorer 4.0, and then uninstall it normally.

    
   NOTE: This step is necessary to ensure that the additional components
   included with Internet Explorer 4.0 are removed.
