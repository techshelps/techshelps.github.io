Windows Address Book Does Not Dial Backup Server Automatically

 Last reviewed: October 1, 1997
Article ID: Q171004
The information in this article applies to:
  • Microsoft Outlook Express version 4.0 for Windows 95
  • Microsoft Outlook Express version 4.0 for Windows NT 4.0

SYMPTOMS

When you attempt to initiate a conference call by clicking Call Now on the NetMeeting tab in Windows Address Book, the backup server is not dialed automatically if you are unable to connect to the default server.

CAUSE

Windows Address Book does not have the ability to determine if a call successfully connects to the default server, and therefore does not dial the backup server if the call is unsuccessful.

RESOLUTION

Manually select the backup server and try the call again. To do this, follow these steps:

  1. Click Start, point to Programs, and then click Windows Address Book.

  2. Double-click the contact you want to call.

  3. Click the NetMeeting tab.

  4. Under Conferencing Directory Servers, click the backup server.

  5. In the Select Or Add New box, click the e-mail address of the contact you are attempting to call, and then click Call Now.
