Form-Based File Upload Support in Internet Explorer

 Last reviewed: February 1, 1998
Article ID: Q161395
The information in this article applies to:
  • Microsoft Internet Explorer versions 2.0, 3.0, 3.01, 3.02, 4.0 for Windows 95
  • Microsoft Internet Explorer versions 2.0, 3.0, 3.01, 3.02, 4.0 for Windows NT 4.0
  • Microsoft Internet Explorer versions 2.0, 2.01, 2.1, 3.0, 3.01, 3.02a for Windows 3.1
  • Microsoft Internet Explorer versions 2.0, 3.0, 3.01, 3.02a for Windows NT 3.51
  • Microsoft Internet Explorer versions 2.0, 2.1, 3.0, 3.0a, 3.01a for Macintosh

SUMMARY

This article describes the Request For Comments (RFC) 1867 specification (Form-Based File Upload in Hypertext Markup Language [HTML]) support in Internet Explorer.

MORE INFORMATION

HTML forms support the following types of input:

  • Check boxes
  • Passwords
  • Option buttons
  • Text

HTML forms accept these inputs and upload them to a location on the Internet.

The ability to upload data files using an HTML form is possible under the RFC 1867 specification. RFC 1867 proposes a method in which data files can be submitted through an HTML form. For information about RFC 1867 support in a specific version of Internet Explorer, please see the appropriate section below.

Note that Microsoft Desktop Systems (DTS) Technical Support cannot assist in the development of HTML code or documents.

Internet Explorer for Windows 95 and Windows NT 4.0

Of the versions of Internet Explorer for Windows 95 and Windows NT 4.0 listed above, only Internet Explorer 4.0 supports the RFC 1867 specification. However, Microsoft has released an add-on for Internet Explorer 3.02 that provides RFC 1867 support. For information about obtaining the add-on, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q165287
   TITLE     : Description of Internet Explorer 3.02 File Upload Add-On
If you are running an earlier version of Internet Explorer for Windows 95 or Windows NT 4.0 and you want the functionality provided by the add-on, you should upgrade to version 3.02 and then install the File Upload Add- on, or upgrade to Internet Explorer 4.0. For information about obtaining Internet Explorer 3.02 or 4.0, please see the following articles in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q164475
   TITLE     : Availability of Internet Explorer 3.02 for Windows 95 and
               NT 4.0

   ARTICLE-ID: Q170993
   TITLE     : How to Install Internet Explorer and Troubleshoot Setup
               Problems

Internet Explorer for Windows 3.1 and Windows NT 3.51

Internet Explorer 3.02a for Windows 3.1 and Windows NT 3.51 supports the RFC 1867 specification. Earlier versions of Internet Explorer for Windows 3.1 and Windows NT 3.51 do not. For information about obtaining Internet Explorer 3.02a, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q159584
   TITLE     : Microsoft Internet Explorer 3.02a for Windows 3.1 Available

Internet Explorer for Macintosh

The versions of Internet Explorer for Macintosh listed above do not support the RFC 1867 specification.
