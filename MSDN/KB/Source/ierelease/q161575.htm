Error Message: Internet Explorer Cannot Open the Internet...

 Last reviewed: September 29, 1997
Article ID: Q161575

The information in this article applies to:

  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0 for Windows 95
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02, 4.0 for Windows NT 4.0

SYMPTOMS

When you start Internet Explorer, you may receive the following error message: 

   Microsoft Internet Explorer

   Internet Explorer cannot open the Internet site <address>

   The protocol specified in this address is not valid. Make sure the
   address is correct, and try again.
When you click OK, Internet Explorer opens and displays Blank.htm instead of your start page.

CAUSE

This error message can occur if you have specified a start page address without "http://" at the beginning of the address.

RESOLUTION

Add "http://" to the beginning of the start page address. To do so, use the appropriate method:

Internet Explorer Versions 3.0, 3.01, 3.02

  1. In Internet Explorer, click Options on the View menu.

  2. Click the Navigation tab.

  3. In the Address box, type "http://" (without quotation marks) before the name of the start page. For example: 

          http://www.msn.com

  4. Click OK.

Internet Explorer Version 4.0

  1. In Internet Explorer, click Options on the View menu.

  2. Click the General tab.

  3. In the Address box, type "http://" (without quotation marks) before the name of the start page. For example: 

          http://www.msn.com

  4. Click OK.
