Deleted Messages Remain in Folder Message List

 Last reviewed: December 16, 1997
Article ID: Q178158
The information in this article applies to:
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows 95
  • Microsoft Outlook Express versions 4.0, 4.01 for Windows NT 4.0

SYMPTOMS

When you delete a message from an Outlook Express folder, a red X may appear on the message, but the message may remain in the folder's message list, even when you quit and then restart Outlook Express.

CAUSE

This behavior can occur if your incoming mail server is an Internet Mail Access Protocol (IMAP) server.

RESOLUTION

To remove deleted messages from an Outlook Express folder message list, follow these steps:

  1. Click the folder from which you want to remove deleted messages.

  2. On the Edit menu, click Purge Deleted Messages.

NOTE: You cannot undelete a message once you purge deleted messages.
