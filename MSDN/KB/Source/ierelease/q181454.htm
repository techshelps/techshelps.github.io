Error Message: This Package Does Not Contain File Upload

 Last reviewed: February 20, 1998
Article ID: Q181454
The information in this article applies to:
  • Microsoft Internet Explorer versions 3.0, 3.01, 3.02 for Windows 95

SYMPTOMS

When you try to install the JS Privacy Update for Internet Explorer 3.02 for Windows 95 & NT 4.0 (302jsupd.exe), you may receive the following error message: 

   This package does not contain File Upload.
   You must install the security package for File Upload.

RESOLUTION

To work around this issue, use the following steps:

  1. Connect to the following Microsoft Web site: 

          http://www.microsoft.com/msdownload/jsprivacy/privupd.htm

  2. Follow the instructions on the screen to download the file, and then choose Open It when you are prompted to save or open the file.
