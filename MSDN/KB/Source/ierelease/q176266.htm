Determining If Internet Explorer 4.0 Preview 2 Was Installed

 Last reviewed: November 26, 1997
Article ID: Q176266
The information in this article applies to:
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows 95
  • Microsoft Internet Explorer versions 4.0, 4.01 for Windows NT 4.0

SUMMARY

This article describes how to determine if Internet Explorer Platform Preview 2 was installed on your computer, even if you successfully uninstalled it.

MORE INFORMATION

If you install and then remove Internet Explorer Platform Preview 2, one or more files may remain on your computer. The following file is present regardless of the type of installation you select for Platform Preview 2: 

   Dataread.me
To determine if this file is present on your computer, follow these steps:

  1. Click Start, point to Find, and then click Files Or Folders.

  2. In the Named box, type the following line, and then click Find Now: 

          dataread.me

  3. If the file is found, Internet Explorer 4.0 Platform Preview 2 has been installed on your computer.
