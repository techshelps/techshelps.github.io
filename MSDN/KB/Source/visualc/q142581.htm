INFO: Availability of ODBC Drivers for VCMac 4.0

 Last reviewed: October 6, 1997
Article ID: Q142581
The information in this article applies to:
  • Microsoft Visual C++, Macintosh Cross-Development, version 4.0

SUMMARY

This article contains information on availability of ODBC drivers for Visual C++ Cross-Development Edition for the Macintosh, version 4.0.

MORE INFORMATION

Though Visual C++ Cross-Development Edition for the Macintosh version 4.0 includes support for ODBC, no ODBC drivers shipped with the product. This was because the ODBC drivers were not available at the time of manufacturing. This is documented in the README for Visual C++ 4.0.

From the README

ODBC Drivers Not Available for Macintosh

This release does not contain the ODBC drivers for the Visual C++ Cross Development edition for Macintosh as they are not yet available. We've provided a fulfillment coupon which entitles you to the shipping ODBC drivers at no additional charge as soon as they become available. Please refer to the coupon in your product box for further information regarding fulfillment.

When will drivers be available?

The drivers will be sent out to customers in the United States and Canada around the end of January 1996.

How can customers obtain the drivers?

Customers in the United States and Canada need to send in the fulfillment coupon for ODBC drivers that included with the product. This is a separate insert in the box and is titled: 

   Microsoft Visual C++ Cross-Development Edition for Macintosh ODBC
   Drivers 1.44 MB disk offer.
The drivers will be mailed once they are ready to be shipped to customers who have sent in the fulfillment coupon.

International customers need to contact their respective Microsoft Subsidiary to obtain the drivers.

What drivers will be available?

Only the SQL server ODBC driver is available for the Macintosh (68k and Power Macintosh).

How can additional drivers be obtained?

Visigenic Software, Inc., will be shipping other ODBC drivers for the Macintosh. For more information regarding these drivers, please contact Visigenic Software: 

   Visigenic Software, Inc
   Phone: 415-286-1900
   Fax: 415-286-246
   www.visigenic.com
