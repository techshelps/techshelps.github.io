DOCERR: Books Online: Class Library User's Guide Appendix A

 Last reviewed: July 22, 1997
Article ID: Q113113
1.50 WINDOWS kbtool kbdocerr

The information in this article applies to:

  • Microsoft Visual C++ for Windows, version 1.5

SYMPTOMS

The Table of Contents for the "Class Library User's Guide" in Books Online contains two entries for Appendix A: 

   Appendix A Getting Started
   Appendix A: Versions of the Microsoft Foundation Class Library
Selecting the first entry displays a list of sections for the Appendix, but attempting to view any of them results in the following error: 

   Help topic does not exist.
The message box may be displayed more than once. The Books Online screen will display the "Welcome to Books Online" title page.

Selecting the second Appendix A entry and viewing its subtopics works correctly.

RESOLUTION

The information listed for "Appendix A Getting Started" is contained in the printed "Getting Started" manual, which was included with Visual C++ 1.5. The actual printed "Class Library User's Guide" contains only one Appendix A, which contains the information about Microsoft Foundation Class (MFC) versions.
