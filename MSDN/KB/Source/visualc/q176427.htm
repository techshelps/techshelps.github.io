PATCH: Daorisc.exe Installs DAO for Alpha (RISC)

 Last reviewed: March 13, 1998
Article ID: Q176427
The information in this article applies to:
  • The DAO SDK included with:

    - Microsoft Visual C++, 32-bit Editions, version 5.0 on the following platform:

    - Alpha

SUMMARY

The DAO SDK that was provided with Visual C++ 5.0 (RISC Edition) does not install correctly. A new version is now available to correct the problem. Users on Alpha Visual C++ version 5.0 will not be able to access the DAO components until the updated DAO SDK is installed.

The file DAORISC.EXE (5.5 MB) is a self-extracting archive that extracts the new DAO SDK for RISC. Alpha users should run the SETUP.EXE located in the disk1 directory.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~ Daorisc.exe (size: 5746958 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
DAO does not support replication on the Alpha, even with this patch.
