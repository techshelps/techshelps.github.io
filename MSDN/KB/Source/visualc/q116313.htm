FIX: Some Common Dialog DLLs Don't Work with Windows 3.0

 Last reviewed: September 18, 1997
Article ID: Q116313
1.00 3.10 WINDOWS kbprg kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft Visual C++ for Windows, version 1.0
  • Microsoft Windows Software Development Kit (SDK) for Windows, version 3.1

SYMPTOMS

Some of the international versions of the common dialog boxes DLL (COMMDLG.DLL) that shipped with Visual C++ for Windows, version 1.0, do not work with Windows, version 3.0. These DLLs only work with Windows, version 3.1 and later. Specifically, these DLLs are: 

   COMMDLG.DEN  (Denmark)
   COMMDLG.DUT  (Netherlands)
   COMMDLG.FIN  (Finland)
   COMMDLG.ITN  (Italy)
   COMMDLG.NOR  (Norway)
   COMMDLG.SWE  (Sweden)
All other COMMDLG files shipped with Visual C++ for Windows, version 1.0, work with Windows, version 3.0 and later.

RESOLUTION

In Visual C++ for Windows, version 1.5, all of the DLLs were updated so that they all can work with Windows version 3.0 and later.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Visual C++ for Windows, version 1.0. This problem was corrected in Microsoft Visual C++ for Windows, version 1.5.
