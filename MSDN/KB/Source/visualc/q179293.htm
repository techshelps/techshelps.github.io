INFO: MFC Versions That Ship with Microsoft Visual C++

 Last reviewed: January 16, 1998
Article ID: Q179293
The information in this article applies to:
  • Microsoft Visual C++, 32-bit Editions, versions 2.0, 2.1, 2.2, 4.0, 4.1, 4.2, 5.0
  • Microsoft C/C++ version 7.0

SUMMARY

This article lists the versions of the Microsoft Foundation Classes (MFC) that shipped with the different versions of Microsoft C/C++ and Microsoft Visual C++.

MORE INFORMATION

   MFC Version   Shipped with:
   --------------------------------------------------------
   4.2           Microsoft Visual C++ versions 4.2 and 5.0
   4.0           Microsoft Visual C++ versions 4.0 and 4.1
   3.2           Microsoft Visual C++ version 2.2
   3.1           Microsoft Visual C++ version 2.1
   3.0           Microsoft Visual C++ version 2.0
   2.5           Microsoft Visual C++ version 1.5
   2.1           Microsoft Visual C++ version 1.0
   2.0           Microsoft Visual C++ version 1.0
   1.0           Microsoft C/C++ version 7.0
