ODBC DLLs Included in Visual C++ 1.5

 Last reviewed: July 22, 1997
Article ID: Q112755
1.50 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Visual C++ for Windows, version 1.5
  • Microsoft ODBC Software Development Kit (SDK) for Windows, version 1.0

SUMMARY

Below is a list of Open Database Connectivity (ODBC) specific dynamic-link libraries (DLLs) in Visual C++ version 1.5. These DLLs are located in the \MSVC\REDIST directory on the CD and are installed into the WINDOWS\SYSTEM directory by the Visual C++ Setup program. If you're experiencing problems with ODBC when using Visual C++, be sure to check the dates of each of these DLLs to be certain that the dates of your DLLs are the same or newer. Make sure that there aren't other versions of these DLLs in your PATH or on a network search drive. 

   Filename         Date        Size
   -------------------------------------------------------------------

   BTRV110.DLL      07/22/93    104560
   DBNMP3.DLL       11/11/93     10640
   MSJETDSP.DLL     05/11/93     85792
   ODBC.DLL         11/11/93     53200
   ODBCCURS.DLL     11/11/93     85488
   ODBCINST.DLL     11/11/93     83312
   ORA6WIN.DLL      05/20/93    285573
   PDX110.DLL       07/22/93    189168
   RED110.DLL       07/22/93    238416
   SIMADMIN.DLL     09/15/93    160080
   SIMBA.DLL        09/15/93    469904
   SQLSVR.DLL       09/01/93    155776
   SQORA.DLL        07/12/93    144096
   SQORASTP.DLL     07/06/93      9632
   TXTISAM.DLL      08/10/93    105184
   XBS110.DLL       07/22/93    263840
   XLSISAM.DLL      08/10/93     97680
