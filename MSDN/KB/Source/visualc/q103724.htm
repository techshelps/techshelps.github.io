INFO: Visual C++ 32-bit Edition Installs for Single User

 Last reviewed: September 30, 1997
Article ID: Q103724

The information in this article applies to:
  • Microsoft Visual C++ 32-bit Edition, versions 1.0, 2.0, 4.0, 5.0

SUMMARY

When Visual C++ 32-bit Edition is installed, the program group created by the Setup program is a personal program group. This program group appears only for a specific user. Other users of the same machine cannot see the Visual C++ for Windows NT program group; the registry key is not created for them, and the user environments aren't set up properly.

For information on how to install Visual C++ 32-bit Edition so that more than one person can use the same installation, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q101482
   TITLE     : Installing Visual C++ for Multiple Users on One Machine
For more information on Windows NT user privileges, see the README.WRI file, which can be accessed by selecting the README icon located in the program group.
