INFO: Summary List: Visual C++ Version 4.1 Known Problems

 Last reviewed: February 3, 1998
Article ID: Q178224
The information in this article applies to:
  • Microsoft Visual C++, 32-bit Editions, version 4.1

SUMMARY

This article lists the titles and Q numbers of those Microsoft Visual C++ Knowledge Base articles that describe bugs that exist in Visual C++ version 4.1. If the title of the article starts with "FIX:", then the bug was fixed in a later release. Please see the STATUS section of the FIX article to determine in which version the fix was incorporated.
