Mfcext.hlp Documents the Property Sheet and Socket Classes

 Last reviewed: July 18, 1997
Article ID: Q137696
1.52 1.52b WINDOWS kbref

The information in this article applies to:

  • The Microsoft Foundation Classes(MFC) included with: Microsoft Visual C++ for Windows, versions 1.52 and 1.52b

There is no installed online documentation for the following classes: 

   CPropertySheet
   CPropertyPage
   CAsyncSocket
   CSocket
These classes are extensions to the 16-bit version of MFC and were not included in the documentation. However, they are documented in a help file that is included on the Visual C++ distribution CD-ROM disc in this directory: 

  \Msvc15\Help\Mfcext.hlp
If you copy this file to your <MSVC>\Help directory on your hard drive then you will be able to search for help on these classes without needing the Visual C++ CD in your CD-ROM.

Please note that Mfcext.hlp refers to AppWizard and ClassWizard support for Property Sheets and Sockets that are only supported in Microsoft's 32-bit products, such as Visual C++ 2.1 and later.
