Patches Available for Visual C++ for Windows

 Last reviewed: July 18, 1997
Article ID: Q105439
1.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Visual C++ for Windows, version 1.0

Below are descriptions of patches available for Visual C++ for Windows, version 1.0:

WINDLL - This patch is for the Windows dynamic-link library (DLL) run-time libraries shipped with Visual C++. The patch corrects problems with several functions in the small and medium model DLL libraries. For more information, refer to Part 9 of the README.WRI file that is shipped with Visual C++. The patch is available on the Microsoft Download Service (MSDL).

OS2FIX - This patch provided by Phar Lap Software enables the Visual C++ DOS extended command-line tools to run in an MS-DOS compatibility box of OS/2 2.1. For more information, refer to the Microsoft Knowledge Base article Q104643, "Using Visual C++ 1.0 in an MS-DOS Box of OS/2 2.1."

B2FIX - This is a tool that can be used to modify the Visual C++ MS-DOS- extended command-line tools so they can run in Windows NT. For more information, refer to the MIGRATE.HLP file that is shipped with Visual C++ for Windows NT. In the section titled "Moving from Visual C++ for Windows," select "How do I build 16-bit applications from Windows NT?" Microsoft does not support the BETA2FIX tool. Please refer to the BETA2FIX.TXT file that is included in B2FIX.EXE.
