INFO: Installing Language Products on a Network

 Last reviewed: September 2, 1997
Article ID: Q34445

The information in this article applies to:
  • Microsoft C for MS-DOS, versions 6.0, 6.0a, 6.0ax
  • Microsoft C for OS/2, versions 6.0, 6.0a
  • Microsoft C/C++ for MS-DOS, version 7.0
  • Microsoft Visual C++ for Windows, versions 1.0, 1.5
  • Microsoft Visual C++ 32-bit Edition, versions 1.0, 2.0, 4.0, 5.0

SUMMARY

Installing all or any part of a Microsoft language compiler on a network is permitted only if each person that intends to use the product has a license to use the product. Microsoft language compilers include C, C/C++, FORTRAN, Macro Assembler (MASM), Pascal, QuickC, QuickC for Windows, and Visual C++. Various multiuser license packs are available for the various products. For more information in the United States regarding the availability of a license pack, call Microsoft Sales and Customer Service at (800) 426-9400. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, go to the Microsoft Web site http://www.microsoft.com/worldwide/default.htm
