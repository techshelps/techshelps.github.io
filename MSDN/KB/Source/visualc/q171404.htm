INFO: ODBC Files Shipped with Visual C++ 5.0

 Last reviewed: September 30, 1997
Article ID: Q171404
The information in this article applies to:
  • Microsoft Visual C++, 32-bit Editions, version 5.0

SUMMARY

This article lists the ODBC files that shipped with Visual C++ 5.0 and the components that the files are associated with.

NOTE: When you are checking to see if you have all of the proper files installed, you should check the ODBC files as well as the driver-specific files. Some files, such as MSVCRT40.DLL, are used by several drivers.
