XADM: Direct Reports Can Only Be Selected from User's Site

 Last reviewed: April 3, 1997
Article ID: Q147360
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SYMPTOM

When you use the Microsoft Exchange Administrator program to select Direct Reports for a user, the Global Address List (GAL) is not available. The only available lists are those which reside on the same Exchange site of which the user is a member.

However, from the mailbox, you can specify any user from the GAL for the manager option.

RESOLUTION

To allow managers and direct reports to be in different sites, connect to a server in the site of the user to be a direct report and adjust their Manager attribute.

STATUS

This Direct Report handling is intended by design for Exchange version 4.0. Since Direct Reports is a back-linked "calculated" attribute which is not directly writable, the Manager attribute must be modified on each recipient that you want to be a direct report.
