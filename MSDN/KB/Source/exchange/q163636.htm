XCLN: Cannot Modify Personal Address Book (PAB) Templates

 Last reviewed: January 29, 1998
Article ID: Q163636
The information in this article applies to:
  • Microsoft Exchange Windows 3.x client version 4.0
  • Microsoft Exchange Windows 95 client version 4.0
  • Microsoft Exchange Windows NT client version 4.0

SUMMARY

Unlike the Global Address List (GAL) templates that can be modified via the Exchange Administrator program, the Personal Address Book (PAB) templates cannot be changed through any Exchange UI.

MORE INFORMATION

For more information on changing the GAL template, see the Microsoft Knowledge Base article, Q152732, "How to Modify Fields in the Exchange Mailbox Template."
