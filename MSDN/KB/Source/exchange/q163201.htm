XCLN: Illegal Operation Err Msg When Form Minimized

 Last reviewed: November 19, 1997
Article ID: Q163201
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, version 4.0

SYMPTOMS

When you quit Microsoft Windows 95 while a minimized Exchange Forms Designer Form for Microsoft Exchange is running, the following exception may occur: 

   <Form executable name>
   This program has performed an illegal operation and will be shut down.

STATUS

Microsoft has confirmed this to be a problem with Microsoft Exchange version 4.0. We are researching the problem and will post new information here in the Knowledge Base as it becomes available.

Version           : Win95:4.0
Platform          : Win95
