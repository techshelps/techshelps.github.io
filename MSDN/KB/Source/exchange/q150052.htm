XCLN: Simple MAPI Not Supported Running as a Service

 Last reviewed: March 11, 1998
Article ID: Q150052
The Information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SUMMARY

Microsoft Exchange does not support running Simple Messaging Application Programming Interface (MAPI) applications that were developed to run as a service on Windows NT.

MORE INFORMATION

All MAPI applications require a profile to create a session on Microsoft Exchange. Extended MAPI is suggested as a solution because the Application Programming Interface (API) contains tools to create profiles and allow login from a service.

For more information concerning the MAPI 1.0 interface, please refer to the Win32 SDK.
