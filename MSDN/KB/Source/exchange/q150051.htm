XADM: Event Viewer Error: No License Was Available

 Last reviewed: February 3, 1998
Article ID: Q150051
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SYMPTOMS

You might see the following error message in the Windows NT Event Viewer on a computer running Microsoft Exchange Server: 

   LicenseService, Event ID 201,
   Error Description = No License was available for user <domain>\<account>
   using product MSExchangeIS 4.0.

CAUSE

This error indicates that there are no more available licenses for additional connections to the Microsoft Exchange Server.

RESOLUTION

To correct this, additional licenses should be purchased and the license number should be increased. To increase the license number follow these steps:

  1. Start the Windows NT License Manager, located in the Network Administration group.
2. Click on the Products View tab. 3. Highlight the MSExchangeIS 4.0 product. 4. Click License and New License, Or click on the appropriate button. 5. Enter the amount of licenses purchased. 6. Click OK, check the box, and OK.

After the next reboot, error will not appear.

For more information read Chapter 5 of "Update Information for Version 3.51" in the NT documentation.

If this does not resolve the problem, please refer to q150387 in the NT database to verify that NT was not installed from a test platform.

MORE INFORMATION

The Microsoft Exchange Server should still reflect the current number of purchased licenses.
