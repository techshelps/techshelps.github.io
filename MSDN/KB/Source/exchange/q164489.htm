XADM: MAPI Error While Generating Offline Address Book

 Last reviewed: November 18, 1997
Article ID: Q164489
The information in this article applies to:
  • Microsoft Exchange Server, versions 4.0 and 5.0

SYMPTOMS

When you try to generate the offline address book, you may encounter the following error messages: 

   Errors in Log

   When attempting to generate the OAB from Generate Address Book Now.

   Error 5004:

   Generation of the offline Address Book is complete. Result: The MAPI
   call failed.<<80004005>>.

CAUSE

This error occurs because the server from which the offline address book is being generated does not contain all of the necessary files for multiple language support for Microsoft Exchange.

RESOLUTION

Install the correct code pages, locales, and templates for all languages to be supported by Microsoft Exchange Server.

MORE INFORMATION

For additional information on adding code pages, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q151701
   TITLE     : How to Manually Add Code Pages for Mixed Languages
Refer to the International Issues (Section 14) of the Readme.wri file on the Microsoft Exchange Server CDROM for information on:
  • Installing CodePage Support - Section 14.2
  • Installing Foreign Language Display Templates - Section 14.3
