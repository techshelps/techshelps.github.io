XCLN: Location of Mapisvc.inf File Not Configurable

 Last reviewed: April 15, 1997
Article ID: Q158826

The information in this article applies to:

- Microsoft Exchange Windows 3.x client, version 4.0 - Microsoft Exchange Windows 95 client, version 4.0 - Microsoft Exchange Windows NT client, version 4.0

The Mapisvc.inf file is used by the Microsoft Exchange client to add services to the profile. It is located in the <Windir>\System directory on Windows 95 and Windows 3.x. In Windows NT, it is located in the <Windir>\System32 directory.

There is not a registry or .ini setting that controls the location of the Mapisvc.inf. You cannot store the Mapisvc.inf in any other directory, such as your home directory on the server.
