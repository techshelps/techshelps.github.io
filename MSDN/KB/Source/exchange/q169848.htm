XADM: Performance Optimizer Fails with Low Disk Space

 Last reviewed: October 28, 1997
Article ID: Q169848
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0
  • Microsoft Exchange Server, version 5.0

SYMPTOMS ========

When you run the Microsoft Exchange Server Performance Optimizer, it may fail and display the following error message: 

   The following drives could not be tested due to an error (drives affected).
RESOLUTION ==========

Performance Optimizer performs various tests on the drives during its optimization process. The error above can be generated when too little free space is available on the drive. Microsoft recommends a minimum of 10MB free on each drive before running Optimizer.

The error may also be generated by actual problems with the drive. Look for drive errors in the Windows NT Event Viewer System log from before the Performance Optimizer was run. Errors about the drive generated while Optimizer is being run may be caused by too little free space.

This behavior is by product design
