XGEN: How to Send Files to Microsoft Technical Support

 Last reviewed: February 20, 1998
Article ID: Q174696
The information in this article applies to:
  • Microsoft Exchange Server, versions 4.0 and 5.0

SUMMARY

This article explains how you can send files to Microsoft Technical Support when you are requested to do so.

  1. Connect to the Microsoft anonymous ftp server ftp.microsoft.com.

  2. Log on as "ANONYMOUS" using your e-mail address as the password. For example, if your e-mail address is JoeSmith@microsoft.com, use that as your password. You are now logged on to the Microsoft FTP server.

  3. Change directory using the following command: 

          cd /transfer/incoming/bussys/mail

    You are now in the correct directory to upload the file.

  4. The typical command that you use to upload the file will be similar to the following, but may vary based on what utility you are using: 

          PUT <filename>

    Before you perform the command, make sure that you are in binary mode.

NOTE: Unless you are advised to follow a different procedure, place the files in a zipped format and use the case number as the file name (for example, SRX970228699123.zip).
