XADM: Welcome Message to New Users Not Available

 Last reviewed: April 3, 1997
Article ID: Q152654

The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SUMMARY

Microsoft Exchange Server does not have an option for sending new users a welcome message.

MORE INFORMATION

In version 3.x of Microsoft Mail for PC Networks, it was possible to send a welcome message to new users via the WELCOME.GLB file. In Microsoft Exchange, it is by design that no user input source for welcome messages is available.
