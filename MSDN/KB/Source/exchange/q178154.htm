XWEB: Err Msg: Unable to Render This view424Object Required

 Last reviewed: March 6, 1998
Article ID: Q178154
The information in this article applies to:
  • Microsoft Outlook Web Access, version 5.5
  • Microsoft Exchange Server, version 5.5

SYMPTOMS

When you use the Outlook Web Access (OWA) client to access an Exchange Server mailbox, you may receive the following error message when selecting the Calendar icon on the Outlook Bar or the Calendar folder in the folder list: 

   Unable to render this view424Object required.

CAUSE

Some of the OWA files located on the Exchange Server computer may be corrupt or missing.

WORKAROUND

To correct this behavior, remove and reinstall the OWA components from the Exchange Server computer by performing the following steps:

  1. Back up the \Exchsrvr\Webdata directory on the Exchange Server computer.

  2. Run the Microsoft Exchange Server Setup program, and choose the Add/Remove option.

  3. Clear the Outlook Web Access components check box, and continue with Setup to remove the OWA components.

  4. Delete the \Webdata directory.

  5. Run Setup again, and select the OWA components.

  6. Continue with Setup to perform a clean installation of the OWA components.
