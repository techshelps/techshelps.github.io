XCLN: Folder Visible Option Not Available

 Last reviewed: February 6, 1998
Article ID: Q178466
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Outlook Windows 95 client, versions 8.00 and 8.01

SYMPTOMS

The Folder Visible box is missing from Exchange Clients.

WORKAROUND

The Folder Visible check box and functionality were not introduced until Exchange Client 5.0 and are not included in Outlook version 8.01.

Verify the correct version of the Exchange Client files by doing the following:

  1. From the Tools menu, select Services.

  2. Select Exchange Server and click the About button.

The versions that include the Folder Visible functionality are:
  • emsabp32.dll 50.1457.3
  • emsmbd32.dll 50.1457.3
  • emsui32.dll 50.1457.3

STATUS

Microsoft has confirmed this to be a problem with Microsoft Exchange Windows 95 client version 4.0 and Microsoft Outlook Windows 95 client version 8.01. This problem was corrected in Microsoft Exchange Windows 95 client version 5.0 and Microsoft Outlook Windows 95 client 8.02.
