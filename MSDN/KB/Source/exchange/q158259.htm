The information in this article applies to:

  • Microsoft Exchange Server, version 4.0

WARNING: Due to U.S. State Dept. export regulations, all SP3 client software on these online services contains EXPORTABLE (40-bit) security only. U.S. & Canadian customers should NOT install these, or else they will no longer be able to read existing 64-bit encrypted mail. A North American- only version of SP3 is available to US and Canadian customers (for $14.95 plus shipping and handling) by calling Microsoft at 1-800-360-7561.

VERY IMPORTANT: Please note that Microsoft Exchange Service Pack 3 (SP3) is not cumulative. You must install SP2 before installing SP3.

SUMMARY

This article lists the files available for SP3 and where they can be found. For information on how to install SP3, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q158263
   TITLE     : XGEN: README.TXT: Microsoft Exchange 4.0 Service Pack 3

AVAILABLE FORMATS FOR MICROSOFT EXCHANGE 4.0 SERVICE PACK 3

Service Pack 3 is available as single self-extracting archive files for Intel (x86), MIPS, PPC, ALPHA, and Macintosh-based computers.

ORDERING MICROSOFT EXCHANGE 4.0 SERVICE PACK 3

If you do not have access to the electronic services listed below, you can order Service Pack 3 (CD-ROM) from the Microsoft at 1-800-360-7561.

INSTRUCTIONS FOR DOWNLOADING

You can download any of these self-extracting files from the following services:

  • Internet (anonymous FTP) 

         1. Ftp to ftp://ftp.microsoft.com/softlib/mslfiles or

            ftp://198.105.232.1/softlib/mslfiles.
     2. Click the appropriate file's hyperlink.

  • Microsoft's World Wide Web Site on the Internet

         1. On the www.microsoft.com home page, click Support on the toolbar.
     2. Click Microsoft Technical Support.
     2. Select Exchange in the dropdown listbox and click Go.
     3. Click the Microsoft Exchange Service Packs hyperlink.
     4. Click the appropriate Library File.

  • Microsoft's Download Service

         1. Connect to MSDL.
     2. Enter [A] Download File.
     3. Enter the appropriate file name and hit return.

    

For additional information about downloading from Online Services,
please see the following article in the Microsoft Knowledge Base:
