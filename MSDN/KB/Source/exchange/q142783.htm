XCLN: Toolbar Buttons Require Mouse Click to Show Status

 Last reviewed: April 3, 1997
Article ID: Q142783

The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0

SYMPTOMS

You must click and hold the left mouse button on the Toolbar button to display the help text on the status bar. This is inconsistent with other Microsoft applications.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange version 4.0, Windows 3.x, Windows NT, and Windows 95 clients. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
