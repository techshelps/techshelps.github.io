XCON: Exchange RAS Does Not Support Windows NT Multilink

 Last reviewed: January 21, 1998
Article ID: Q171062
The information in this article applies to:
  • Microsoft Exchange Server versions 4.0 and 5.0

Microsoft Windows NT Server version 4.0 provides a multilink feature that allows multiple modem links to operate as one link. This maximizes bandwidth using multiple phone lines on a single connection.

The Microsoft Exchange MTA Dynamic RAS Connector does not support the multilink feature of Windows NT version 4.0.
