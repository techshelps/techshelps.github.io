XGEN: How to Search for MS Exchange KB Articles by Topic

 Last reviewed: January 6, 1998
Article ID: Q140950
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0
  • Microsoft Exchange MS-DOS client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0

You can use keywords to easily query for specific topics in the Microsoft Knowledge Base. This article contains keywords for Microsoft Exchange. By using these keywords, you can quickly access common topics. These keywords are found in the Title and at the bottom of articles on the line that begins, "KBSubcategory."

For a general list of categories and keywords for all Knowledge Base articles, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE ID: Q94671
   TITLE     : Categories and Keywords for All Knowledge Base Articles

   Keyword         Description
   -------------------------------------------------------
   XGEN            General issues, including documentation
   XADM            Administrative Issues
   XCLN            Client Issues
   XCON            Connector Issues
   XFOR            Foreign Connector Issues and Internet Chat Server
   XWEB            WEB Issues
