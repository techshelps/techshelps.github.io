XCLN: Cannot Drag Messages to a Window on Taskbar

 Last reviewed: April 2, 1997
Article ID: Q146458

The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0

SYMPTOMS

When you try to drag a message from the right pane to a minimized message on the Taskbar, you will get the following error message: 

   You cannot drag an item onto a button on the Taskbar. However, if you
   do this without releasing the mouse button, the window will open,
   and you can then drag the item into the window.
However, the window will not open.

CAUSE

If you drag a message to a message window that is minimized on the Taskbar, you will get the above error message.

WORKAROUND

If you drag the message to the Desktop first, you can drag and drop the message on the message window on the Taskbar.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange version 4.0 Windows 95 client. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
