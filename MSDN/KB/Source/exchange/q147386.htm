XADM: Err Msg: There Are No More Endpoints Available...

 Last reviewed: April 21, 1997
Article ID: Q147386
The information in this article applies to:

- Microsoft Exchange Server, version 4.0

SYMPTOMS

When creating a new mailbox on an Exchange server, the following error message appears: 

   There are no more endpoints available from the endpoint mapper.

   Microsoft Windows NT
   ID no:0xc00206d9

CAUSE

One reason for this error is because the Exchange System Attendant service has stopped running.

WORKAROUND

Restart the System Attendant service in the Services section of Control Panel and you should be able to create new mailboxes.
