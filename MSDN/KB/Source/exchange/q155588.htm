XADM: Error: Extension Could Not Be Loaded

 Last reviewed: April 3, 1997
Article ID: Q155588
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SYMPTOMS

When accessing the property page of the Internet Mail Connector (IMC), the Microsoft Mail connector, or the Free and Busy connector the following error message may appear: 

   Extension 'SMTP' could not be loaded. The specified module could not be
   found.
NOTE: The extension listed in the error message varies according to which connector you are trying to access the property page of. For the IMC, the extension is SMTP. For the Microsoft Mail connector, it is MS; for the Free and Busy Connector, it is MSFB.

WORKAROUND

Re-share the "ADD-INS" sub-directory located in the Exchsrvr main directory with the following permissions: 

Account                                           Rights

<account used to install Exchange originally>     Full Control
<Exchange Service Account>                        Full Control
Everyone                                          Read-Only
