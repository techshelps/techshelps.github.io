XADM: Rebuilding Exchange Server From SP3 Backup

 Last reviewed: January 29, 1998
Article ID: Q161436
The information in this article applies to:
  • Microsoft Exchange Server version 4.0

To rebuild Microsoft Exchange Server with Service Pack 3, perform the following steps:

  1. Install Exchange Server with Service Pack 2 by running setup. No flag, such as "/r" for recovery, is necessary. In fact, this flag may cause errors later when restoring from the backup.

  2. Select "Create a new site" and enter the same Organization and Site names as the backup.

  3. Apply Exchange Service Pack 3 updates. Again, no flag is necessary.

  4. Restore from the backup.

  5. Stop the Information Store and MTA services.

  6. Run ISINTEG -patch.

  7. Start the Information Store and MTA services.

Exchange Server with Service Pack 3 should be completely rebuilt and begin functioning.
