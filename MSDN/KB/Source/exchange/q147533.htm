XCLN: Invalid X.400 Address on To: Line Causes NDR

 Last reviewed: March 11, 1997
Article ID: Q147533
4.00 WINDOWS kbusage kberrmsg kbbug4.00

The information in this article applies to:

  • Microsoft Exchange Windows 3.x client, version 4.0
  • Microsoft Exchange Windows 95 client, version 4.0
  • Microsoft Exchange Windows NT client, version 4.0

SYMPTOMS

An invalid X400 address on the TO: line that also contains a folder will cause an NDR on not only the X400 address but also the folder address.

The error received will be similar to this:

Your message did not reach some or all of the intended recipients. 

      Subject: test to Invalid X400 and Folder1
      Sent: 2/22/96 7:11:04 PM
The following recipient(s) could not be reached: 

      Folder1 on 2/22/96 7:13:05 PM
            Unable to deliver the message due to a recipient problem
            [MSEXCH:MSExchangeIS:SMITH:JOHN_J]

      'invalid X400' on 2/22/96 7:13:05 PM
            Unable to deliver the message due to a recipient problem
            [MSEXCH:MSExchangeIS:SMITH:JOHN_J]

STATUS

Microsoft has confirmed this to be a problem in the Microsoft Exchange version 4.0 Windows 3.x, Windows 95, and Windows NT clients. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
