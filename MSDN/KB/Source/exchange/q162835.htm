XADM: SMTP Address is Stripped from Auto-forwarded Message

 Last reviewed: December 18, 1997
Article ID: Q162835
The information in this article applies to:
  • Microsoft Exchange Server version 4.0, Service Packs 1, 2, and 3

SYMPTOMS

When a message is received via the Internet Mail Service and is auto- forwarded by an Inbox Assistant rule, the SMTP address of the original sender is not included in the body of the forwarded message.

WORKAROUND

If you manually forwarded a message received via the Internet Mail Service, the SMTP address of the original sender is included in the body of the forwarded message.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server 4.0. This problem was corrected in Microsoft Exchange Server 5.0.
