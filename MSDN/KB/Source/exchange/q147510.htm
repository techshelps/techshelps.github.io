XCLN: Error Selecting Many Addresses from Address Book

 Last reviewed: March 31, 1997
Article ID: Q147510
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0
  • Microsoft Exchange MS-DOS client, versions 4.0 and 5.0

SYMPTOMS

When you select a large number of addresses from the address book, an hourglass appears. After three minutes the following message appears: 

   Too many items are selected. Try selecting fewer items.

CAUSE

The Microsoft Exchange client has a three minute timeout value for making large listbox selections. This limit applies to all listboxes. If after three minutes a selection cannot be completed, the above error message will be displayed.

A three minute delay can occur when selecting a complete address list on a system with a slow network or Remote Access (RAS) connection with as few as 5,500 entries chosen.

WORKAROUND

Make the large selection in smaller increments.
