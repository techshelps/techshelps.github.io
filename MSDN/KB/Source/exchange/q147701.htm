XADM: Exchange Service Account Permissions

 Last reviewed: September 23, 1997
Article ID: Q147701
The information in this article applies to:

- Microsoft Exchange Server, version 4.0

SUMMARY

To install and run Microsoft Exchange Server version 4.0, an Exchange Service Account needs to be created with Domain User rights. This is the account that will be used to run the Exchange services. The account should be created using the User Manager for Domains program provided with Microsoft Windows NT Server. During Exchange Server installation, Setup will ask for a service account and add "Log On As A Service" and "Restore Files And Directories" rights to this account.
