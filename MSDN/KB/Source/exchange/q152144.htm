XADM: Changing a Site's Display Name in the Address Book

 Last reviewed: April 3, 1997
Article ID: Q152144
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0
  • Microsoft Exchange MS-DOS client, versions 4.0 and 5.0

SUMMARY

Microsoft Exchange provides the ability to change the display name of a Microsoft Exchange Site. This change will also be reflected in the Address Book of Microsoft Exchange users.

MORE INFORMATION

While the Display name for the Site can be changed, the Directory name provided during installation cannot be changed.

To change the Site's Display name, a Microsoft Exchange administrator will need to start the Microsoft Exchange Administrator program and follow these steps: 

   1. Highlight the Site name (under Organization).
   2. Choose Properties from the File menu.
   3. Change the Display name.
   4. Choose OK.
The next time that a Microsoft Exchange client is started, the Display name of the Site will have been changed.
