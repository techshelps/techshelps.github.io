XCLN: Buttons/Menus Unavailable When Opening New Mail

 Last reviewed: April 2, 1997
Article ID: Q145903

The information in this article applies to:
  • Microsoft Exchange Windows client, versions 4.0 and 5.0

SYMPTOMS

In the Microsoft Exchange Windows 3.x, Windows 95, and Windows NT clients, when you open a message by clicking the Yes button in the New Mail Notification dialog box, the Next and Previous buttons and the menus are unavailable.

RESOLUTION

Close the message and reopen the message from the message viewer. All the functions will now be available.
