XCLN: Err Msg: The Attachment Could Not Be Saved

 Last reviewed: April 21, 1997
Article ID: Q151902
The information in this article applies to:
  • Microsoft Exchange Windows NT client, version 4.0
  • Microsoft Exchange Windows 95 client, version 4.0

SYMPTOMS

The Microsoft Exchange client for Windows generates the following error when you try to save an attachment: 

   The attachment(s) could not be saved. An error occurred while the file
   was being saved or transferred.

CAUSE

This will happen when the first 256 bytes of the attachment are zeros (x00).

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server version 4.0. This problem was corrected in the latest Microsoft Exchange Server U.S. Service Pack. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
