XCLN: Unable to Save Changes to Attachment in Private Msg.

 Last reviewed: March 28, 1997
Article ID: Q146275

The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0

SYMPTOMS

When you receive a Private (read-only) mail message with attachments, you can view and modify the attachments; however, you can not save the changes to the original message.

When you close the Private message, and you made changes to the attachment, you will get the following message:

If you make modifications to attachments in a read-only message until he closes the note and gets the message: 

   This message has been modified but this message is read-only.
   Do you want to discard changes?

RESOLUTION

If you wish to save the attachment, you must save it under a different name.
