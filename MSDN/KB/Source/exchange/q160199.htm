XCLN: URLs Do Not Activate Using WordMail As Default Editor

 Last reviewed: March 31, 1997
Article ID: Q160199
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0

SUMMARY

When you click a URL hyperlink in a Microsoft Exchange message, the default web browser will not be launched if WordMail is enabled as the mail editor.

CAUSE

This is by product design of Microsoft Word 95 (Word 7.0); this will work properly in Word 97.
