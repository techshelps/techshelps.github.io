XADM: November 1997 Select CD-ROM Update Kit Is Missing Setup

 Last reviewed: February 12, 1998
Article ID: Q178918
The information in this article applies to:
  • Microsoft Exchange Server, version 5.5

SYMPTOMS

The Setup.exe file for Microsoft Exchange Server version 5.5 is not included in the Microsoft Select CD-ROM Update Kit for November 1997.

CAUSE

This is by design.

WORKAROUND

To install Microsoft Exchange Standard Edition, run the Srvmin.exe file. To install the Microsoft Exchange Enterprise Edition, run the Srvmax.exe file.

MORE INFORMATION

The following text is an excerpt from the November 1997 Release Notes for Microsoft Select CD-ROM Update Kit: 

   Exchange 5.5 Server, Enterprise and Standard Editions are both located
   on the same CD-ROM. Installation for each version is initiated by using
   a differently named file. The setup file for Exchange 5.5 Enterprise is
   Srvmax.exe; the setup file for Exchange 5.5 Standard is Srvmin.exe.
