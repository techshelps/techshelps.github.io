XCLN: No Status Indicator When Attach/Save/Send Attachments

 Last reviewed: April 2, 1997
Article ID: Q146029
The Information in this article applies to:
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0

SUMMARY

There is no status indicator to indicate file progress or time with Exchange client attachments.

MORE INFORMATION

There is no status indicator to show progress when an attachment is attached, saved, or sent. There is no way to see how the attachment is progressing or how much time is needed to perform a file operation. This is by design.
