XCLN: Client Requirements for Exchange Forms

 Last reviewed: February 3, 1998
Article ID: Q147603
The information in this article applies to:

- Microsoft Exchange Windows 95 client, versions 4.0 and 5.0 - Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0 - Microsoft Exchange Windows NT client, versions 4.0 and 5.0

SUMMARY

The Electronic Forms Designer for Microsoft Exchange has differing memory requirements based on the platform and client used. Please refer to the table below to determine the appropriate memory and hard disk options for your particular client.

This information is taken directly from the Application Designer's guide included with various Exchange Server SKUs (Stock Keeping Units).

MORE INFORMATION

To design forms and folders, users need the following:

Hardware

CPU:       486 or better processor
Mouse:     Windows compatible mouse
Display:   VGA 16-color display minimum, 800 x 600 x 256 colors
           recommended.

Memory

Windows 3.x, Windows for Workgroups 3.x, and Windows 95: 
           12 MB of memory minimum, 16 MB recommended.
Windows NT 3.51 
           20 MB of memory minimum, 24 MB recommended.

Software

- Microsoft Windows version 3.1 or later - Microsoft Exchange Client version 4.0 - Microsoft Exchange Forms Designer version 4.0

To use forms and folders and run time, users need the following:

Hardware

CPU:       386 computer or higher, 486 or higher recommended
Mouse:     Windows compatible mouse
Display:   VGA or compatible display

Memory

Windows 3.x, Windows for Workgroups 3.x, and Windows 95: 
           8 MB of memory minimum, 12 MB recommended.
Windows NT 3.51 
           16 MB of memory minimum, 20 MB recommended.

Software

- Microsoft Windows version 3.1 or later - Microsoft Exchange Client version 4.0
