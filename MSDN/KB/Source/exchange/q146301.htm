XCLN: Exchange Form Label Minimum Value Is 300

 Last reviewed: March 25, 1997
Article ID: Q146301
4.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Exchange Windows 3.x client version 4.0
  • Microsoft Exchange Windows 95 client version 4.0
  • Microsoft Exchange Windows NT client version 4.0

SUMMARY

In the Microsoft Exchange Forms Designer, you cannot set the height property of a label field (using the Properties dialog box) to less than 300 twips. If you set this height property to a value below 300, the value will be changed to 300 when you close the dialog box.

MORE INFORMATION

This behavior is by product design.

WORKAROUND

To work around this problem:

  1. Click on the label field on the canvas.

  2. Use the sizing handles to adjust the height of the box to approximately
what you want it to be.

  1. Verify the height of the label field by double-clicking it to activate
the Field Properties dialog box. The number in the height field can be less than 300 and this value will remain after the box is closed.
