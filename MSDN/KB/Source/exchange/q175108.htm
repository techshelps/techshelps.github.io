XWEB: Cannot View Free/Busy For Distribution List Using OWA

 Last reviewed: February 10, 1998
Article ID: Q175108
The information in this article applies to:
  • Microsoft Outlook Web Access version 5.5

SYMPTOMS

If you use Microsoft Outlook Web Access (OWA), you cannot view the free/busy information for members of a distribution list. This is by product design.

WORKAROUND

If you use either the Exchange or Outlook client, you can see the free/busy information for every member of the distribution list.
