XADM: License Service Logs the Wrong Exchange Version Number

 Last reviewed: January 5, 1998
Article ID: Q176811
The information in this article applies to:
  • Microsoft Exchange Server, version 5.5

SYMPTOMS

When Exchange Server runs out of Licenses, the License Service logs the following event: 

   No License was available for user <DomainName>\<UserName> using
   product MSExchangeIS 5.0.
You would expect that this error would refer to MSExchangeIS 5.5.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server, version 5.5. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
