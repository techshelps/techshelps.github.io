XCON: A DN Object Allocation Command Failed

 Last reviewed: March 24, 1997
Article ID: Q161482
4.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Exchange Server, version 4.0

SUMMARY

The Exchange MTA may generate the following warning every time a pair of MTAs connects either within the Site or via a Site Connector: 

   Event ID: 2247
   A directory name (DN) object allocation command failed. [RD Server MAIN
   BASE 1 114] (12)
This occurs due to an event logging problem with the MTA.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange version 4.0. This has been fixed in version 5.0 of Microsoft Exchange.
