XCLN: MSN Sign-on Dialog Appears Behind Address List

 Last reviewed: January 5, 1998
Article ID: Q149188
The information in this article applies to:

- Microsoft Exchange Windows 95 client, version 4.0

SUMMARY

When you use multiple service providers such as Microsoft Exchange Server and Microsoft Network Service providers, the MSN sign in dialog might appear behind the drop down address list.

STATUS

Microsoft has confirmed this to be a problem in version 4.0 of the Microsoft Exchange Windows 95 Client. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
