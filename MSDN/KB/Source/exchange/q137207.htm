XCLN: No Gateway Headers with MS Mail Information Service

 Last reviewed: April 21, 1997
Article ID: Q137207
The information in this article applies to:
  • Microsoft Exchange Windows client, version 4.0
  • Microsoft Exchange Windows NT client, version 4.0
  • Microsoft Exchange Windows 95 client, version 4.0

You cannot view gateway headers when you use the Microsoft Mail Information Service to connect a Microsoft Exchange Windows client to a version 3.x Microsoft Mail for PC Networks postoffice. The Microsoft Exchange clients do not have functionality comparable to the MSMAIL.INI file StripGatewayHeaders entry in Microsoft Mail for Windows.
