XCLN: WFWG System Information Shows Negative Memory

 Last reviewed: April 2, 1997
Article ID: Q147351
The information in this article applies to:
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0

If you run the Microsoft Exchange Windows 3.x client on a computer with 64 megabytes of RAM installed and you check the total amount of free memory in Help About, the About Program Manager dialog box displays the negative amount of -1024 KB (kilobytes) Free.

Windows 3.x is limited to recognizing 16 mb of RAM.

For additional information on how Windows 3.x handles memory, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q84388
   TITLE     : Windows 3.1 Memory Limits
