XFOR: Internet Mail Service Provides Maximum Logging at Medium

 Last reviewed: February 12, 1998
Article ID: Q177543
The information in this article applies to:
  • Microsoft Exchange Server, version 5.0

SYMPTOMS

The Microsoft Exchange Server Internet Mail Service logs actual data of the messages when the logging level is set to Medium.

CAUSE

The Microsoft Exchange Server Service Pack 1 Internet Mail Service update logs the same information for both the Meduim and Maximum settings.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange Server, version 5.0 Service Pack 1. This problem has been corrected in the latest U.S. Service Pack for Microsoft Exchange Server version 5.0. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): S E R V P A C K
