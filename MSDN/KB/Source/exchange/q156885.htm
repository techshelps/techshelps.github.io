XGEN: Errors When Using the Header.exe Tool

 Last reviewed: January 6, 1998
Article ID: Q156885
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0

SUMMARY

When you use the Header.exe tool from the Microsoft Exchange Resource Kit, you might receive the following error: 

   Export Error...

   Following error occurred: Could not bind to the Microsoft Exchange
   Directory server XXXXX. The Microsoft Exchange Server computer does
   not respond.
This error will be generated if the Microsoft Exchange Directory Service is not running on the computer specified.

MORE INFORMATION

This tool must be run on a computer that the Microsoft Exchange Server has been installed on.
