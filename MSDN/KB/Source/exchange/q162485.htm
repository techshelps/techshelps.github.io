XWEB: ASC Doesn't Show Number of Unread Messages

 Last reviewed: November 10, 1997
Article ID: Q162485

The information in this article applies to:

  • Microsoft Exchange Server, version 5.0

SYMPTOMS

When you use the Microsoft Exchange Active Server Components to view messages in your inbox, the total number of messages in the inbox is displayed but the number of unread message is not.

CAUSE

This behavior is by design.
