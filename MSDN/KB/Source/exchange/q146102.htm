XCLN: Embedded Objects in Private Messages Can be Modified

 Last reviewed: April 2, 1997
Article ID: Q146102
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0

SUMMARY

You will be allowed to modify an object embedded within a Microsoft Exchange message that has been marked with a sensitivity level of Private. However, you will not be allowed to save that object in the original message.

This behavior is by design.
