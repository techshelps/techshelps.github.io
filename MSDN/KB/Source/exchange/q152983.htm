XCLN: Win95 Client Can't Send Fax When Offline Using .OST

 Last reviewed: May 5, 1997
Article ID: Q152983

The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0

SYMPTOMS

When you send a Microsoft Fax from Microsoft Word while Microsoft Exchange is offline (.OST file) and off the network, you will get the following error message: 

   Information Store could not be opened.

CAUSE

This happens when you use an .OST file only. It works fine when you use a .PST file.

MORE INFORMATION

Faxes can be sent if the Microsoft Exchange client is already started and working offline.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Exchange version 4.0 Windows 95 client. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
