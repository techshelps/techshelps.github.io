XWEB: Unable to Rename Folders with ASC

 Last reviewed: June 17, 1997
Article ID: Q162484

The information in this article applies to:

  • Microsoft Exchange Server, version 5.0

SUMMARY

When using the Microsoft Exchange Active Server Components to access a Microsoft Exchange Server mailbox, the ability to rename private folders is unavailable.

MORE INFORMATION

If you need to rename folders, you must sign into the Microsoft Exchange Client to do so.

For more information on limitations of the Web client, please see the following article in the Microsoft Knowledge Base:
