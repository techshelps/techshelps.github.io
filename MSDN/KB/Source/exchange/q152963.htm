XCLN: Autosignature Not Applied on MS Word Editor

 Last reviewed: April 3, 1997
Article ID: Q152963

The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0

SYMPTOMS

AutoSignatures are not included when you use the Microsoft Word version 7.0 e-mail Editor.

CAUSE

This is by design; only the Microsoft Exchange clients support AutoSignature.

MORE INFORMATION

Microsoft Word provides a method to add auto signature that is supported by Microsoft Word Editor. The following instructions can be found in the Microsoft Word Help files when you query on Auto Signature:

  1. Select the text you want to use as an auto signature in electronic mail messages.

  2. Click AutoText on the Edit menu.

  3. In the Name box, type the signature you want to appear.

  4. Click Add.

    The signature will be automatically added to the bottom of every message.
