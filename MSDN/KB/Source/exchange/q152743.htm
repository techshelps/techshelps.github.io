XGEN: The Microsoft Exchange FAQ is Available on the Web

 Last reviewed: January 6, 1998
Article ID: Q152743
The information in this article applies to:
  • Microsoft Exchange Server, version 4.0
  • Microsoft Exchange MS-DOS client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0

The information in the Web document contains Frequently Asked Question about Microsoft Exchange and is separated into four categories:

  • Microsoft Administration - Questions about Microsoft Exchange 
                                  Administration.
  • Clients - Questions about Microsoft Exchange Clients.
  • Connectors - Questions about Microsoft Exchange Connectors.
  • Foreign Connectivity - Questions about Microsoft Exchange's foreign 
                              connectivity.
You can obtain this Web page from Microsoft's World Wide Web Site on the Internet at the following location: 

   http://www.microsoft.com/kb/faq/backoffc/exchange/
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com
