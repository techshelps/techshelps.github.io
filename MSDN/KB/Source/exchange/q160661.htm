XCLN: Japanese Exchange Client Requires Japanese Windows 95

 Last reviewed: October 10, 1997
Article ID: Q160661
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0, 5.0

SUMMARY

If you run the Japanese Microsoft Exchange Windows 95 client on a computer that is not running Japanese Windows 95, Japanese characters are not displayed properly. Japanese Microsoft Exchange clients use a double- byte character set (DBCS), which is not completely supported by non- Japanese versions of Windows 95.
