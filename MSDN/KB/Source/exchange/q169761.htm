XCON: Mailbox.csv Not Included on Exchange 5.0 CD

 Last reviewed: February 17, 1998
Article ID: Q169761
The information in this article applies to:
  • Microsoft Exchange Server version 5.0

SUMMARY

There is no Maibox.csv file on the Exchange Server 5.0 CD. The Mailbox.csv file that was included on the Exchange Server 4.0 CD is not on the Exchange Server 5.0 CD. The mailbox.csv file is on the Exchange Server 4.0 CD in the following location: 

   Support\Samples\Csvs\Mailbox\Mailbox.csv

MORE INFORMATION

You can still use the Mailbox.csv file from the Exchange Server 4.0 CD for exporting and importing in Exchange Server 5.0. This file contains all 93 fields that can be associated with any Exchange Server mailbox. You can use this .csv file to create the "master headers" for a .csv file for use in importing user and mailbox information into Exchange Server. The BackOffice Resource Kit contains utility called Header.exe that can be used to create a .csv file with only the required headers.
