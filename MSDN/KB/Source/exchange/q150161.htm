XADM: Site Addressing Property Page Apply Button Inactive

 Last reviewed: April 3, 1997
Article ID: Q150161
The information in this article applies to:

- Microsoft Exchange Server, version 4.0

SUMMARY

In the Microsoft Exchange Administrator program, on the Site Addressing property page of the Configuration container, when you try to change the Routing calculation server, the Apply button is not active, therefore the change cannot take effect.

To make the Apply button active, check and uncheck the box labeled "Share address space with other X400 systems".

NOTE: This only applicable when multiple Microsoft Exchange Servers are in 

      the same site.
