XCLN: Save As Attachment Not Available in Messages

 Last reviewed: April 2, 1997
Article ID: Q152459

The information in this article applies to:
  • Microsoft Exchange Windows 95 client, versions 4.0 and 5.0
  • Microsoft Exchange Windows 3.x client, versions 4.0 and 5.0
  • Microsoft Exchange Windows NT client, versions 4.0 and 5.0

SUMMARY

The Save As Attachment feature appears to be unavailable when you try to save an attachment that was inserted in an RTF field inside a message created with a form designed using Microsoft Electronic Forms Designer version 1.0.

It is unavailable if you click Save As on the File menu. However, the option to Save As File is available by right-clicking the attachment and selecting Save As File.

MORE INFORMATION

Electronic Forms Designer 1.0 is an application used to design custom forms so that information can be entered and displayed in a desired format inside a message. Messages that are created using one of these forms can be sent to other users who also have that form installed.
