XCLN: Err Msg "Your Exchange Server Is Unavailable"

 Last reviewed: April 15, 1997
Article ID: Q157024
The information in this article applies to:
  • Microsoft Exchange Windows 95 client, version 4.0

SUMMARY

When you run the Microsoft Exchange Windows 95 client, you may receive the following error message: 

   Your Exchange Server is unavailable.

CAUSE

The Client for Microsoft Networks is not installed on the computer running Windows 95.

RESOLUTION

Install the Client for Microsoft Networks on the Windows 95 computer.

  1. In Control Panel Network, on the Configuration tab, click Add.

  2. In the "Select Network Component Type" dialog box, click Client, and click Add.

  3. In the "Select Network Client" dialog box click Microsoft, and click Client for Microsoft Networks. Click OK. When the installation is complete, restart the computer.
