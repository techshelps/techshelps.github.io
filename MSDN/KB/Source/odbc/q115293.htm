The information in this article applies to:

SUMMARY

The following is a list of third-party vendors who provide additional ODBC (open database connectivity) drivers for use with Microsoft Query.

Microsoft does not support the functionality or connectivity of these drivers with Microsoft Query or any Microsoft product.

The products mentioned here are manufactured by vendors independent of Microsoft; we make no warranty, implied or otherwise, regarding the performance or reliability of these products.

For more information about or for support for any of the third-party ODBC drivers listed below, please contact the appropriate vendor for that driver.
