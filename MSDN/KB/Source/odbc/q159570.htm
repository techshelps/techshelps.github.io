INF: Access ODBC Driver Is Double-Byte Enabled

 Last reviewed: December 17, 1996
Article ID: Q159570
The information in this article applies to:
  • Microsoft Open Database Connectivity, version 2.5
  • Microsoft Access ODBC Driver, version 3.40

The Microsoft Access ODBC Driver, starting with version 3.40, is enabled to support double-byte character strings (DBCS). However, the Access ODBC Driver version 3.40 does not yet support Unicode.
