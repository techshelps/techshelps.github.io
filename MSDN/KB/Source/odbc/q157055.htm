INF: ODBC Access Driver Does Not Support Create or Drop View

 Last reviewed: December 17, 1996
Article ID: Q157055
The information in this article applies to:
  • Microsoft ODBC Access Driver, version 3.40

SUMMARY

The ODBC Access driver does not support Create View or Drop View.

MORE INFORMATION

The Access driver uses the Jet engine. Jet does not support Create View or Drop View.
