FIX: Error on Disconnecting from ODBC Desktop Drivers

 Last reviewed: January 22, 1997
Article ID: Q156298
The information in this article applies to:
  • Microsoft Open Database Connectivity Desktop Driver version 3.4
BUG #: 3692 (Dartbugs)

SYMPTOMS

When you disconnect from an ODBC application that uses Access driver version 3.40 or other ODBC desktop drivers, you may receive the following error message: 

   Message Box Title: <appname.exe>
   The instruction at <addr1> referenced a memory at <addr2>.
   The memory could not be written.

WORKAROUND

This problem has been fixed in Msjt3032.dll version 3.00.3213. For information on how to obtain this file, see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q151186
   TITLE     : INF: Updated Jet DLLs Available on MSL
After you download this file, you can copy Msjt3032.dll to your <%winroot%>\system32 directory.

Additional reference words: 3.40 JET MFC exception
