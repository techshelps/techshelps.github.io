How To Obtain ODBC 3.0 Service Pack1

 Last reviewed: July 30, 1997
Article ID: Q168612
The information in this article applies to:
  • Microsoft Open Database Connectivity, version 3.0 Service Pack 1

This article describes how to obtain the Open Database Connectivity version 3.0 Service Pack 1. This service pack is available for Intel and Alpha computers, and contains the latest version of the ODBC 3.0 core components.

TO INSTALL ODBC 3.0 SERVICE PACK 1

  • Intel and Alpha versions are available on the Microsoft ODBC Web site and the Microsoft FTP site.

    Download the appropriate executable file from the following list: 

       Platform   Path and file name
   --------   ------------------

   Intel      http://www.microsoft.com/odbc/download/x86/odbcdmin.exe
   Intel      ftp://ftp.microsoft.com/developr/odbc/public/odbcdmin.exe

   Alpha      http://www.microsoft.com/odbc/download/x86/ odbcdmal.exe
   Alpha      ftp://ftp.microsoft.com/developr/odbc/public/odbcdmal.exe
  • You will need approximately 4 MB of free disk space to install the service pack.
  • You can either run the download and install the updated files from the Web, or download them to your disk to run them later.
  • When you run the executable file, it will decompress files in your temporary directory and run the Setup program.

If you need more information or assistance, please contact your primary support provider.
