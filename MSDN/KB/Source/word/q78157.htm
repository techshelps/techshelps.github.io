Vertical Alignments Visible Only in Print Preview in WinWord

 Last reviewed: February 6, 1998
Article ID: Q78157
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word for Windows, to correctly view justified or centered vertical alignment formatting in a section, you must be in print preview.

Steps to Reproduce Problem

Note: Make sure you are in page or page layout view before following these steps.

  1. Type two lines of text in a Word for Windows document.

  2. From the Insert menu, choose Break. Select the Next Page option, and choose the OK button.

  3. From the Format menu, choose Section Layout (in version 2.x) or Section (in version 1.x).

  4. From the Vertical Alignment box, select Center or Justified, and choose the OK button.

In page or page layout view, you can usually see all document formatting as it will print. The only way to view vertically centered or justified text in a section is to choose Print Preview from the File menu. Every other view displays the text as aligned at the top of the document.

Note: From any view, the document prints correctly with the centered or justified vertical alignment.
