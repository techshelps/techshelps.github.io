WD: How to Use Examples Found in the Style Gallery

 Last reviewed: February 3, 1998
Article ID: Q142750
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a and 6.0c
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SUMMARY

This article describes how to use the example and style samples in the Style Gallery (Format menu). When you select a template and then select Example in the Style Gallery, you see a sample document formatted with the styles from the selected template. When you select a template and then select Style Samples, you see a list of all of the styles in the selected template and samples of the style formats. The example and style samples are stored as AutoText entries in their respective templates.

MORE INFORMATION

To use the Style Gallery examples, use the appropriate method for your situation.

Method 1: If the Template Does Not Contain Boilerplate Text

  1. Create a new document based on one of Word's templates,

    for example Brochur1.dot (Word 6) or Brochure.dot (Word 7).

  2. On the Edit menu, click AutoText (Word 6 and 7) or on the Insert menu, point to AutoText, then click AutoText (Word 97 and 98), select Gallery Examples, and then click Insert.

This places the entire example in your document.

Method 2: If the Template Contains Boilerplate Text

  1. Create a new document based on the Normal template.

  2. On the File menu, click Templates (Word 6 and Word 7), or on the Tools menu, click Templates And Add-Ins (Word 97 and Word 98).

  3. Click Attach.

  4. Select the template that contains the Gallery Examples you want to use, and then click OK.

  5. Click OK.

  6. On the Edit menu, click AutoText (Word 6 and 7) or on the Insert menu, point to AutoText, then click AutoText (Word 97 and 98), select Gallery Examples, and then click Insert.

This places the entire example in your document.

You can modify the inserted text and store it as a new AutoText entry or save the document.
