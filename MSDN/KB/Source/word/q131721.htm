Internet Assistant Error: "Warning: Failed to Notify Library"

 Last reviewed: February 5, 1998
Article ID: Q131721
The information in this article applies to:
  • Microsoft Internet Assistant for Word, versions 1.0, 1.0z
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

You receive the following warning message intermittently while downloading an FTP URL, and you are able to download the URL: 

   Warning: Failed to notify library

CAUSE

This warning appears when you try to download an FTP URL from within two separate instances of Internet Assistant running at the same time. Since each instance of Internet Assistant is using Internet Assistant InternetWorks and is making the same calls to the same .DLL files, you will get the warning message.

RESOLUTION

To avoid getting this error message in the future, run only one instance at a time.
