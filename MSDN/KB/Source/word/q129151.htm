Insert ClipArt and File New Won't Default to Right Directories

 Last reviewed: February 6, 1998
Article ID: Q129151
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Office for Windows, version 4.3
  • Microsoft Bookshelf for Windows, 1994

SYMPTOMS

When you choose New from the File menu, the names of your templates do not appear in the Template list. Similarly, when you try to insert clipart and you choose Picture from the Insert menu, you have to change the directory to the location of your clipart pictures.

CAUSE

When you install Office 4.3 and Bookshelf '94, the Setup program may incorrectly set the ClipArt Pictures and User Templates file locations settings in Word to the Word program directory.

The ClipArt Pictures file location setting should specify the ClipArt directory, and the User Templates file location setting should specify the <WinWord>\TEMPLATES subdirectory.

Note: This problem occurs only with the integrated Office 4.3 and Bookshelf 

'94 package. It does not occur when Office 4.3 and Bookshelf '94 are
purchased and installed separately.

MORE INFORMATION

To correct the User Templates setting

  1. From the Tools menu, choose Options.

  2. Select the File Locations tab.

  3. Select User Templates and choose the Modify button.

  4. Double-click the <WinWord>\TEMPLATES subdirectory (or any other directory that contains your templates) and click the OK button.

  5. Choose the Close button.

Your templates should now appear when you choose New from the File menu.

To correct the ClipArt Pictures setting

  1. From the Tools menu, choose Options.

  2. Select the File Locations tab.

  3. Select ClipArt Pictures and choose the Modify button.

  4. Double-click the <WinWord>\CLIPART subdirectory (or any other directory that contains the ClipArt pictures) and choose the OK button.

  5. Choose the Close button.

Your ClipArt pictures will now appear when you choose Picture from the Insert menu.
