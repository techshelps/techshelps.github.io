WPHelp Navigation Key is Different from WordPerfect's Key

 Last reviewed: February 6, 1998
Article ID: Q79436
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In a Word document, some of the Navigation Keys for WordPerfect User's (WPHelp navigation keys, in Word 2.x) do not perform the same as actual WordPerfect navigation key 

                                                   In Word w/ WordPerfect

                        point to the end of        the end of the line.
                        the current screen.

HOME+LEFT ARROWS        Moves the insertion        Moves insertion point to
                        point to the beginning     the start of the line.
                        of the current screen.

HOME, HOME, UP ARROW    Moves the insertion        Moves insertion point
                        point to the top of the    top of the page.
                        document.

HOME, HOME DOWN ARROW   Moves the insertion        Moves insertions point
                        point to the end of the    to the bottom of the
                        document.                  page.
Microsoft has confirmed this to be a problem in Word for Windows versions 2.x and 6.x. We are researching this problem and will post new information here as it becomes available.
