WD6: Shortcut Displays as Box in Special Characters Dialog

 Last reviewed: February 5, 1998
Article ID: Q140183
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

The shortcut key for the Em Space or En Space is displayed as a box immediately followed by the keyboard character you selected when you created the shortcut. For example, if you specify Command+Z as the shortcut key combination, Word displays a box and the letter "Z" (without the quotation marks).

This shortcut combination is displayed incorrectly in the Special Characters dialog box. (To locate this dialog box, click Symbol on the Insert menu.)

CAUSE

Word does not correctly display the Command character in the dialog box.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
