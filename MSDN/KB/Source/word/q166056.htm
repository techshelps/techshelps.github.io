WD97: Word Viewer Upgrade Doesn't Prompt for Doc Open Option

 Last reviewed: February 4, 1998
Article ID: Q166056
The information in this article applies to:
  • Microsoft Word Viewer 97

SYMPTOMS

The Setup program for Microsoft Word Viewer 97 does not provide you with the option to decide whether to use Word 97 or Word Viewer 97 to open documents with a .doc file extension.

CAUSE

This problem occurs if you have a previous version of Microsoft Word Viewer installed on your computer when you upgrade to Microsoft Word Viewer 97.

RESOLUTION

To work around this problem, remove Microsoft Word Viewer 97 and reinstall it. Then, choose not to open documents with Word Viewer.

To remove Microsoft Word Viewer 97:

  1. On the Windows taskbar, click Start, point to Settings and click Control Panel.

  2. Double-click Add/Remove Programs.

  3. Click Microsoft Word Viewer 97 and click the Add/Remove button.

  4. Once Setup starts, click Uninstall.

Setup will uninstall Word Viewer and will prompt you to restart Windows. Now you can run Setup for Word Viewer 97 from where you originally ran it by running the setup.exe file. When prompted for the application you want to open .doc files with, click the option you prefer.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
