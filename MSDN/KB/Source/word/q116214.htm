Cannot Attach a Template to Another Template

 Last reviewed: February 5, 1998
Article ID: Q116214
2.00 2.00a 2.00a-CD 2.00b 2.00c 6.00 6.00a 6.00c WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a

SYMPTOMS

The option to attach a template is unavailable when the current document is a template. (To locate this option, from the File menu, choose Templates. In Word 6.0, the Attach button is unavailable; in Word 2.0x, the Attach Document To box will be dimmed.)

CAUSE

By design in Word for Windows, you cannot attach a template to another template.

WORKAROUND

To attach a template to your template, first convert your file into a document:

  1. From the Edit menu, choose Select All.

  2. From the Edit menu, choose Copy.

  3. Open a new document.

  4. From the Edit menu, choose Paste.

  5. From the File menu, choose Save. From the Save File As Type list, choose Word document.

REFERENCES

"Microsoft Word for Windows User's Guide," version 2.0, Chapter 37, "Document Templates"

"Microsoft Word User's Guide," version 6.0, Chapter 10, "Document Templates"
