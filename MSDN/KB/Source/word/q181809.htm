WD5x: Using Word 5.x Envelope Address Lists in Word 6.0

 Last reviewed: March 9, 1998
Article ID: Q181809
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 5.1a

SUMMARY

When you upgrade from Word 5.x to Word 6.0, you may want to use addresses generated with the Create Envelope command in Word version 5.1. To do this, you must convert the addresses to AutoText entries. You can do this in Word 6.0 by using the Word 5.1 Upgrade template, which is located in the Macros folder in the Word program folder.

To use the macros in the Word 5.1 Upgrade template, click Open on the File menu and open the 5.1 Upgrade template. Then, follow the directions in the template. Before you convert the addresses, make sure your Word 5.1 Word Toolbar Prefs file is in the "System Folder:Preferences" folder.

MORE INFORMATION

For additional information about converting Word 5.1 address lists, double- click the Help button on the Standard toolbar. Then, type "address lists" (without quotation marks).

For information about using these addresses when you print envelopes, see "Using Addresses from Word 5.1 for the Macintosh," in Chapter 23, "Printing," of the "Microsoft Word User's Guide."
