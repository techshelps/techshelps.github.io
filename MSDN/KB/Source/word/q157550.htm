WD97: Headings Inside Table Don't Display in Document Map

 Last reviewed: July 31, 1997
Article ID: Q157550
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you display the document map in a document that consists of a table with heading rows or styles, the heading rows or styles do not appear in the document map pane.

WORKAROUND

There is no workaround currently available.

MORE INFORMATION

The Document Map is a vertical pane that displays an outline of the document's headings. You can use the document map to quickly move through a long document or keep track of your location. To locate this option, click Document Map on the View menu.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
