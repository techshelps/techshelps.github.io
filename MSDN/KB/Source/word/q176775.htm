WD6X: Formatted Text in Tables Displayed Incorrectly in Browser

 Last reviewed: November 19, 1997
Article ID: Q176775
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Internet Assistant for Word, version 2.0 on the following platform: Macintosh

SYMPTOMS

When you create an HTML document containing a table with text formatted with increased font sizes and/or with bold, and you then view it in your browser, the font size in one or more of the table cells will change to the default size and/or the bold formatting will be missing. When you reopen the document in Microsoft Word, the format will have also changed.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Note, this problem will also happen when you reopen the document in Word and make any change to the document or if you save the document without making any changes.
