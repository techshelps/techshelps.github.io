WD97: Red Box Prints Behind WordArt Object on Canon BJC 4550

 Last reviewed: July 31, 1997
Article ID: Q171042
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you print a Microsoft Word 97 document that contains a particular style of WordArt to a Canon BubbleJet BJC-4550 printer, a red box is printed behind the WordArt object.

CAUSE

This problem occurs when you use the WordArt style in the third row and third column of the WordArt Gallery dialog box, and print with version 3.70 of the printer driver.

RESOLUTION

Upgrade the printer driver to version 3.90.
