WinWord: Documented and Undocumented Operating Limitations

 Last reviewed: July 30, 1997
Article ID: Q95408
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

The following is a list of Word for Windows operating limitations. Word for Windows may display an error message if you exceed these limits. Some of these limits are determined by the amount of available memory in your computer and whether or not your computer uses an extended memory manager (EMM). Features with fixed maximums are marked with an asterisk (*).
