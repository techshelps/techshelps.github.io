WD: Avery Label Wizard Available for Word

 Last reviewed: February 5, 1998
Article ID: Q162759
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SUMMARY

Microsoft Word 97 ships with the Avery Wizard located in the ValuPack folder.

The Avery Wizard is an add-in product available from Avery Dennison that you can use with Microsoft Word to easily design and print to your favorite Avery products. The Avery Wizard provides an easy interface for creating single labels, mail merges to labels, and reusable lists.

NOTE: The Avery Wizard is not provided on the 3.5-inch replacement disks that are available from Microsoft Customer Service.

An Avery Wizard Word 6.0 and 7.0 is available on the Avery Web Site, at the following address: 

   http://www.avery.com/
NOTE: Check the Avery Frequently Asked Questions (FAQ) page for more information about the Avery Wizard. Note also that Avery provides a Readme.doc file which is located in the ValuPack\Averywiz folder on the Word or Office CD-ROM.
