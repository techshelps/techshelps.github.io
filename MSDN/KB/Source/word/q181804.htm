WD5x: Using Word 5.x Envelope Address Lists in Word 98

 Last reviewed: March 9, 1998
Article ID: Q181804
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 5.1a

SUMMARY

After you install Word 98 Macintosh edition, the First Run Upgrade process automatically takes place when Word 98 is started. During this upgrade process, the Word 5.x Addresses from the Word Toolbar Prefs file are updated to address entries in the Address Book.

MORE INFORMATION

You can re-run the First Run Upgrade at any time by running the FirstRunUpgrade command. To run the command, you must add the command to the toolbar. To do this, follow these steps:

  1. On the Tools menu, click customize.

  2. Click the Commands tab.

  3. Under Categories, double-click the All Commands category.

  4. Under Commands, click FirstRunUpgrade, and drag the command to the toolbar.

  5. Click the toolbar button to run the command.

To remove the command from the toolbar, follow these steps:

  1. On the Tools menu, click Customize.

  2. Click the toolbar button and drag it off the toolbar.
