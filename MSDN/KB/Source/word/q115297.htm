Can't Find Text with Wildcard (*)

 Last reviewed: February 5, 1998
Article ID: Q115297
6.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0

SYMPTOMS

In Word 6.0 for Windows, when you search for a character using the wildcard character (*), Word may not find any characters. Instead, you receive the message: 

   Word has finished searching the document. The search
   item was not found.
Please note that to find text with the *, the Use Pattern Matching option must be selected. However, this error sometimes incorrectly occurs when Use Pattern Matching is selected.

CAUSE

This error occurs when you use key combinations to choose the Find Next button. It does not occur when you use the mouse.

STATUS

Microsoft has confirmed this to be a problem in Word version 6.0 for Windows. The problem was correct in Word 6.0a for Windows.

WORKAROUND

Upgrade to Word 6.0a, or use the mouse rather than key combinations to choose the Find Next button.
