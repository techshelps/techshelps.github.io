Doc Error: Heading Numbering Is Not a Section-Level Setting

 Last reviewed: February 5, 1998
Article ID: Q116363
6.00 6.00a WINDOWS kbusage kbdocerr

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SUMMARY

There is a documentation error on page 402 of the "Microsoft Word User's Guide" and in the "Removing Numbers from Headings in an Outline" online Help topic. The following incorrect statement appears in both locations: 

   In outline view, position the insertion point in the section from
    which you want to remove heading numbers.
This statement incorrectly implies that you can apply or remove heading numbering at the section level in your document. In Word for Windows, heading numbering is a document-level setting, so you can only change heading numbering formatting for the entire document, not for a single section.

REFERENCES

"Microsoft Word User's Guide," version 6.0, pages 402
