Err Msg: Word Cannot Find the Hyphenation File HYPH.DLL

 Last reviewed: February 5, 1998
Article ID: Q121606
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

When you open a document in which the Automatically Hyphenate Document option is selected, you receive the following error message 

   Word cannot find the hyphenation file HYPH.DLL for <Language>
where <Language> indicates the language used when creating the document.

CAUSE

This behavior can occur with any of the following conditions:

  • The HYPH.DLL file was renamed or moved from its original location.
  • The path in the WINWORD6.INI was changed to reflect an incorrect location for the HYPH.DLL file.
  • The HYPH.DLL file was deleted from the system.
  • You opened the document on another machine, and Word cannot find the original path to the HYPH.DLL file.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word versions 6.0, 6.0a, and 6.0c for Windows. We are researching this problem and will post new information here as it becomes available.

RESOLUTION

To correct this behavior, use the appropriate method(s) below:

  • Make sure that the HYPH.DLL file exists in the location specified in the WINWORD6.INI file.
  • Confirm that the HYPH.DLL file has not been renamed or deleted.
  • Install the HYPH.DLL file by running the Word Setup program.
  • Clear the Automatically Hyphenate Document check box if you are transferring documents to other machines.

MORE INFORMATION

In Word, you can activate the Automatically Hyphenate Document option by choosing Hyphenate from the Tools menu and selecting the corresponding check box. When you choose this option, Word embeds the path to the HYPH.DLL file in the document. The error is generated because Word cannot locate the HYPH.DLL file.
