Degree Symbol Lost Saving to MS-DOS Text with Line Breaks

 Last reviewed: February 5, 1998
Article ID: Q114745
6.00 6.00a WINDOWS kbusage kbprb

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

In Word 6.0 for Windows, you can create a degree symbol by holding down the ALT key and typing 248 on the numeric keypad (ALT+248). When you save a document containing the degree symbol in MS-DOS Text With Line Breaks file format, the degree symbol is converted to a space.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

To retain the degree symbol, save the document in one of the following file formats:

  • MS-DOS Text
  • MS-DOS Text with Layout
