DMM: New Instance of Word Started When Word is Already Running

 Last reviewed: March 4, 1998
Article ID: Q179965
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows

SYMPTOMS

Once you have imported your address list into Microsoft Direct Mail Manager, verified, and saved it, you have the option to use Microsoft Word to create a new form letter or open an existing one.

This process will cause Microsoft Direct Mail Manager to start a new instance of Microsoft Word even though Microsoft Word may already running.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.

REFERENCES

   TITLE: WD: How to Start a Mail Merge
