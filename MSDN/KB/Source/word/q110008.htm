Word: Unable to Print to Printer on PC-NFS Network

 Last reviewed: February 5, 1998
Article ID: Q110008
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0, 6.0a, 6.0c
  • Microsoft Windows for Workgroups, versions 3.1 and 3.11
  • Microsoft Office for Windows, versions 4.2, 4.3
  • Microsoft Project for Windows, version 4.0

SYMPTOMS

When you print from Word 6.0 for Windows to a printer shared on a PC-NFS network (version 4.0 or 5.0), you may receive an incomplete print job, or you may receive the following error message: 

   Error has been detected in the Printer Setup
-or- 

   Windows cannot print due to a problem with the current printer setup
Note: The behavior will also occur When printing consolidated projects within Microsoft Project for Windows version 4.0.

CAUSE

These problems occur when you print from Word 6.0 on a network that does not support Mandatory Record Locking. PC-NFS supports a form of file/record locking called Advisory Record Locking. Because Word supports OLE 2.0, which requires mandatory file/record locking, it cannot print directly to the network (bypassing Print Manager when printing to a network printer that has a server sharing it). Therefore, any attempt to print directly to the network causes the problems noted above.

RESOLUTION

To correct this problem, use one of the procedures below. Before performing the appropriate procedure, make sure Print Manager is enabled by doing the following:

  1. Run Windows Control Panel.

  2. Choose the Printers icon.

  3. Select the Use Print Manager check box.

  4. Choose Close.

You are using Windows 3.1

In Windows 3.1 use the following steps to correct the problem:

  1. From the Program Manager main group, run Print Manager.

  2. From the Options menu, choose Network Settings.

  3. Clear the Print Net Jobs Direct check box.

  4. Choose OK.

  5. From the Options menu, choose Printer Setup.

  6. Choose the Connect button.

  7. Clear the Fast Printing Direct To Port check box.

  8. Choose OK.

You are using Windows for Workgroups version 3.1 or 3.11

In Windows for Workgroups (3.1 and 3.11) use the following steps to correct the problem:

  1. From the Program Manager main group, run Print Manager.

  2. From the Options menu, choose Background Printing.

  3. Clear the "Send Documents Directly to Network" check box.

  4. Choose OK.

  5. From the Options menu, choose Printer Setup.

  6. Choose the Connect button.

  7. Clear the "Fast Printing Direct to Port" check box.

  8. Choose OK.
