WD98: Send Command Sends Attachment as Word Document

 Last reviewed: February 17, 1998
Article ID: Q179319
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you use the Send command on the File menu, the current document is attached as a Word document in the mail message even if you saved the file in a foreign format.

CAUSE

This behavior occurs when the Mail As Attachment option is selected on the General tab in the Preferences dialog box (on the Tools menu, click Preferences).

WORKAROUND

To mail a file in a foreign format, save the file in that format and then attach the file to your message from your mail program.

MORE INFORMATION

For information about this behavior in earlier versions of Word for the Macintosh, please see the following article in the Microsoft Knowledge Base:
