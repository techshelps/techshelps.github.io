WD97: Office Assistant Balloon Has No Controls After Replace

 Last reviewed: February 5, 1998
Article ID: Q160240
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you perform a find and replace while the Office Assistant is running, the balloon message that informs you of how many replacements were made does not include an OK or Close button. The Cancel button in the Find And Replace dialog box does not change to Close while the balloon message is displayed.

WORKAROUND

To work around this problem so that the Cancel button in the Find And Replace dialog box changes to Close, use one of the following methods:

Method 1: Click the Find and Replace dialog box.

Method 2: Press any key.

Method 3: Click the balloon message.

The balloon message is closed and the focus is returned to the Find and Replace dialog box, and the Cancel button changes to Close.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

A balloon message is an interactive message which requires you to confirm or cancel an action before you can close it. The balloon message that appears after you complete a find and replace operation does not include an OK or Close button. If you click the Office Assistant while the balloon message is visible, the Office Assistant will prompt you for a help topic. If you click in the document while the balloon message is visible, you will get a system beep.
