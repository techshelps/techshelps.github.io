Cannot Create New Directory on Novell Network

 Last reviewed: February 5, 1998
Article ID: Q112777
6.00 6.00a 6.00c WINDOWS kbnetwork kb3rdparty buglist6.00 buglist6.00a fixlist7.00

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

If you specify a full path, you may be unable to create a new directory when you choose the New Directory button in Word for Windows when you are running on a Novell network. You can, however, create a new subdirectory if you specify only the subdirectory name. (To locate the New Directory button, choose Options from the Tools menu, select the File Locations tab, and then choose the Modify button.)

STATUS

Microsoft has confirmed this to be a problem in Word versions 6.0, 6.0a and 6.0c for Windows. This problem was corrected in Word version 7.0 for Windows 95.

WORKAROUND

  1. In the Modify Location dialog box, select the location where you want to create the new directory in the Drive and Directory boxes, then choose the New Directory button.

  2. In the Create Directory dialog box, type only the new subdirectory name and then choose OK.
