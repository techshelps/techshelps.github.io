WinWord: No Page Break but Status Bar Shows 2-Page Document

 Last reviewed: February 6, 1998
Article ID: Q89489
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.0a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SYMPTOMS

In Microsoft Word for Windows, if you are looking at your document in normal view and the text of your document is long enough so that all but one paragraph mark fits on the first page, the status bar will indicate that the document is two pages long even though a page break may not be displayed. This second page is visible in print preview and will be printed when you send your document to a printer.

STATUS

Microsoft has confirmed this to be a problem in the versions of Microsoft Word for Windows listed above. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Steps to Reproduce Problem

  1. In normal view, make sure that paragraph marks are visible (if not, choose the Show/Hide button on the ribbon). From the Tools menu, choose Options. Select the General category and make sure the Background Repagination check box is selected. Press the ENTER key until a page break appears.

  2. Delete all paragraph marks beneath the page break to make the page break disappear.

  3. Press the ENTER key once and then press the UP ARROW key to move the insertion point above the last paragraph mark.

The status bar indicates that the document is two pages long even though no page break appears before the last paragraph mark. Using the arrow keys to move the insertion point to a paragraph will not cause the page break to appear. In addition, choosing the Repaginate Now command will not cause the page break to appear. However, if you position the insertion point in the last paragraph, press the ENTER key again, and choose Repaginate Now from the Tools menu (Utilities menu in version 1.x), the page break will appear.

Note: Widow/Orphan control does not seem to affect this behavior.
