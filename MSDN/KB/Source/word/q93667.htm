How to Change the Default "From" Value in MEMO2.DOT

 Last reviewed: July 30, 1997
Article ID: Q93667
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

There are two methods to change the default name that appears in the From dialog box when you create a document based on the MEMO2.DOT template supplied with Word for Windows.

Method 1: Add [MS Word User] Section to WIN.INI

This method changes the Name value in your Windows WIN.INI file. Use this method if you want to set a different default Name value for all templates that use the [MS Word User] section of your WIN.INI file.

  1. In a document based on MEMO2.DOT, choose Macro from the Tools menu.

  2. Select the Template Macros option.

  3. From the Macro Name list, select UserInfo and choose the Run button.

  4. In the Portrait Report Defaults dialog box, type a new default in the Name and Company boxes and choose the OK button.

The UserInfo macro writes a new section, [MS Word User], in your Windows WIN.INI file. The following is a sample WIN.INI entry: 

   [MS Word User]
   Name=Daffy Duck
   Company=Warner Bros.

  Note: This is not the same information you can edit in the User Info
  category of the Tools Options command.

Method 2: Modify MEMO2.DOT

This method changes the default value in the From dialog box in MEMO2.DOT. It makes no changes to the Name information in your WIN.INI file. Use Method 2 if you want to set a different default for MEMO2.DOT than you use in other Word documents.

  1. Open the MEMO2.DOT template file.

  2. Highlight the text in the cell to the right of the From cell in the memo heading (by default, this cell contains "Sender").

  3. Type a new default name.

  4. From the File menu, choose Close. Choose the Yes button when Word prompts you to save changes.
