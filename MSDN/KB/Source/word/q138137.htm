"Invalid Page Fault" When Dialog Open in Two Sessions of Word

 Last reviewed: February 5, 1998
Article ID: Q138137
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

When the Grammar Readability Statistics dialog box is open in both Word and a Word document session such a Binder or WordMail, you may get the following error message when you return to Word: 

   This Program has performed an illegal operation.
   Winword caused an invalid page fault in module <unknown>, at
   0000:00ec56e0

STATUS

Microsoft has confirmed this to be a problem in Word for Windows 95 version 7.0 and 7.0a. This problem was corrected in Microsoft Word 97 for Windows.

WORKAROUND

Close the Grammar Readability Statistics dialog box before activating a Word object such as Binders or WordMail.
