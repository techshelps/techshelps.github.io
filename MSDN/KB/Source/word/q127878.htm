WD: Statistics Button Is Unavailable in File Summary Info Dialog

 Last reviewed: February 19, 1998
Article ID: Q127878
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

When you use the built-in Summary Info dialog box in a macro, the Statistics button is unavailable (dimmed) when the dialog box is displayed.

WORKAROUND

To work around this problem, you can retrieve a document statistic and store it in a variable using the FileSummaryInfo command with the proper parameter. The following sample macro retreives the number of words in the active document without displaying the FileSummaryInfo dialog box: 

   Sub Main
      Dim dlg As FileSummaryInfo
      GetCurValues dlg
      x$ = dlg.NumWords
   End Sub
To view the list of arguments you can use to return document statistics information, while in the WordBasic macro editing pane, search for Help on FileSummaryInfo.

For additional information, please see the following article in the Microsoft Knowledge Base:

   TITLE     : WD: Accessing Summary Info Dialog Box from a Macro
