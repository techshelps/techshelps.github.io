Printing Envelopes on the HP LaserJet 4 from Word for Windows

 Last reviewed: July 30, 1997
Article ID: Q96206
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c
  • Microsoft Windows operating system version 3.1

SUMMARY

You can use the default settings to print an envelope using printer driver versions 31.1.03, 31.1.08, 31.1.12, 31.1.18, 31.1.20 (HPCL5E.DRV) for the Hewlett-Packard (HP) LaserJet 4 printer. This driver supports printing to the HP LaserJet 4 printer from Windows version 3.1.

This article describes how to print envelopes on the HP LaserJet 4 printer using Word for Windows. It also lists the specifications of envelopes supported by the HP envelope feeder.

MORE INFORMATION

Without Envelope Feeder Installed

If you don't have an envelope feeder, you can use the instructions below to change the paper size settings on the LaserJet 4 so the multipurpose tray serves as an envelope feeder.

Note: In these steps, the resulting LaserJet LCD display messages are enclosed in brackets ([]) after each step.

  1. Reset the printer by pressing the 'Shift' and 'Menu' buttons 
       simultaneously   on the front control panel of the printer.  The
   display will show '07 RESET'

  2. Press the Online button to switch the printer to Offline. [00 Ready]

  3. Press the Paper Size button. [MP TRAY = ... ]

    If the "MP TRAY =" message does not appear in the LCD display, the printer is not set correctly. Use the following steps to set this printer:

    a. Press the Online button to switch the printer Online.

    b. Press the Online button to switch the printer Offline.

    c. Press the Menu button 5 times. [CONFIG MENU]

    d. Press the Item button. [MP TRAY]

    e. Press the +/- button until [MP TRAY = Cass] appears.

    f. Press the Enter button. [MP TRAY = CASS*]

    g. Press the Online button. [00 Ready] (goto step 1)

  4. Press the +/- key until the correct paper size appears in the LCD display. For example, if you are printing COM 10 envelopes, choose COM 10 for your paper size.

  5. Press the Enter button to lock in your choice.

  6. Press the Online button.

  7. Start Word for Windows.

  8. From the Tools menu, choose Options and select the Print category. Under Envelope Options, Activate (place an X) the Printer's Envelope Feeder Has Been Installed check box and choose the OK button.

  9. The LaserJet 4 is ready to print envelopes using the Create Envelope command.

With the Envelope Feeder Installed

  1. Reset the printer by pressing the 'Shift' and 'Menu' buttons 
       simultaneously   on the front control panel of the printer.  The
   display will show '07 RESET'

  2. From the Tools menu, choose Options and select the Print category. Under Envelope Options, select the "Printer's Envelope Feeder Has Been Installed" check box and choose the OK button.

  3. From the Tools menu, choose Options and select the Print category. Under Printing Options, clear the Reverse Print Order check box and choose the OK button.

    Note: When the "Printer's Envelope Feeder Has Been Installed" check box is selected, the printer displays the following message when you turn the printer on: 

          EE Tray = COM10

  4. Print the envelope using the Create Envelope command.
