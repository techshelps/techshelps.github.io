6.00 6.00a WINDOWS kbref

The information in this article applies to:

  • Microsoft Word for Windows, version 6.0a
  • Microsoft Windows operating system version 3.1

SUMMARY

This article contains the directory of the 3.5-inch, high-density (1.44 MB) Disk #2 (disk 2 of 9) for version 6.0a of Word for Windows. The article includes the location where Setup installs the file as well as the decompressed filename for each file, which you must provide when you use the DECOMP.EXE utility to manually decompress a file.
