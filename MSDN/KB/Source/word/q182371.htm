DMM: Error Messages During Direct Mail Manager Setup

 Last reviewed: March 16, 1998
Article ID: Q182371
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows

SYMPTOMS

When you attempt to install Direct Mail Manager, you receive one or more of the following error messages: 

   DAO350.DLL was unable to register itself in the system registry.
   Microsoft Direct Mail Manager setup was not completed successfully.
Also, when you try to start Direct Mail Manager, you may receive the following error message: 

   The MSVBVM50.DLL file is linked to missing export OLEAUT32.DLL.
-or-

When the message "Setup is updating your system" is displayed, Direct Mail Manager Setup may stop responding (hang).

-or-

The following error message may appear: 

   This program has performed an illegal operation and will be shut down.

   If the problem persists, contact the program vendor.
When you click Details, the following error message will appear: 

   SETUP caused an invalid page fault in module SETUP.EXE at
   0137:<address>

   -or-

   SETUP caused an invalid page fault in module KERNEL32.dll at
   0137:bff9a28c
-or-

The following error message may also appear during Direct Mail Manager Setup: 

   Msexch35.dll was unable to register itself in the system registry.
   Microsoft Direct Mail Manager setup was not completed successfully.
Note that the resolution for this error message is different from that for the other error messages in this article. For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q182370
   TITLE     : DMM: Err Msg: "Msexch35.dll Was Unable to Register"
               During Setup

CAUSE

Your computer system was not restarted after you installed Microsoft Word 97 for Windows and before you attempted to install Direct Mail Manager.

RESOLUTION

Restart your computer and then install (reinstall) Direct Mail Manager.

NOTE: You may need to reinstall Microsoft Word 97 for Windows (or Microsoft Office 97 for Windows) and then restart your computer before you can install Direct Mail Manager successfully.
