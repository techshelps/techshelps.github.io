WD97: How to Import Word 97 Documents into Word 2.x, 6.x, 7.x

 Last reviewed: February 3, 1998
Article ID: Q157091
The information in this article applies to:
  • Microsoft Word 97 for Windows 

                                  SUMMARY
                                  =======
This article describes how to open Word 97 documents in Word versions 2.0, 6.0, and 7.0 (sometimes referred to as Word 95). 

                             MORE INFORMATION
                             ================

IMPORTING WORD 97 DOCUMENTS INTO WORD 6.0 and 7.0

Method 1: Install the Microsoft Word 97 Converter

The best way to open a Word 97 document in Word 6.0 or 7.0 for Windows is to use the Word 97 import converter. For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q162214
   TITLE     : How to Obtain the Word 97 Converter
NOTE: Word 97 does not have an import converter available for Word for the Macintosh. If you are using Word for the Macintosh, use Method 2 or Method 3.

Double-click the Wrd97cnv.exe file to extract and set up the converter. Depending on which version of Word you are currently using, the Setup program installs either the Mswrd832.cnv file (the 32-bit version for Word 7.0) or the Msword8.cnv file (the 16-bit version for Word 6.0). Setup also updates the necessary registry settings. For more information, see the Wrd97cnv.doc file that is installed with the Word 97 converter.

Method 2: In Word 97, Save the Document in Microsoft Word 6.0/95 Format

In Word 97, use the following steps to save the document using the Microsoft Word 6.0/95 converter:

  1. Open the document in Word 97.

  2. On the File menu, click Save As.

  3. In the Save As Type list, click "Word 6.0/95 (*.doc)."

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q162602
   TITLE     : Questions About Saving in the Word 6.0/95 File Format

Method 3: In Word 97, Save the Document as Rich Text Format (*.rtf)

  1. Open the document in Word 97.

  2. On the File menu, click Save As.

  3. In the Save As Type list, click "Rich Text Format (*.rtf)."

IMPORTING WORD 97 DOCUMENTS INTO WORD 2.0

To import Word 97 documents into Word 2.0, open the document in Word 97, and then save the document using the Word 2.x for Windows converter.
