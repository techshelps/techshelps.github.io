IA: Text Wraps to Right Window Border, Not Right Margin

 Last reviewed: February 5, 1998
Article ID: Q128498
The information in this article applies to:
  • Microsoft Internet Assistant for Word, versions 1.0z, 2.0z
  • Microsoft Word for Windows 95 version 7.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

When you are using Internet Assistant for Word, text lines wrap to the right window border instead of to the right margin of your document, and the At and Ln measurements in the status bar are missing.

CAUSE

The Wrap To Window option is turned on. (To locate the Wrap To Window option, choose Options from the Tools menu and select the View tab. The Wrap to Window option is not available in page layout view.)

If you deactivate the Wrap To Window option while using Internet Assistant and you switch between HTML Edit and Web Browse views, this behavior recurs.

By design, when you create a new document based on the HTML.DOT template or when you open an HTML document, Internet Assistant automatically opens the document in normal view and activates the Wrap To Window feature. This feature is designed to increase the speed that Word displays your fonts. When the Wrap To Window feature is turned on, the At and Ln measurements in the status bar do not display. This feature is not turned off when you close the HTML document.

RESOLUTION

To turn off the Wrap To Window feature, from the Tools menu, choose Options. On the View tab, clear the Wrap To Window check box, then choose OK.
