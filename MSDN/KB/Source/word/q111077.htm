How to Order Word for Windows Developer's Kit and Resource Kit

 Last reviewed: November 17, 1997
Article ID: Q111077
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0
  • Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

To obtain the Microsoft Press books "Word for Windows Developer's Kit" or "Microsoft Word 6.0 for Windows Resource Kit," contact Microsoft Press at (800) MSPRESS or (800) 677-7377. Do not contact Microsoft ISS Sales and Information to order these titles.

Note: You can also obtain these Microsoft Press titles from any local computer bookstore or by sending in the Word for Windows Supplemental Offers Coupon, which is included in your Word 6.x retail package.

MORE INFORMATION

Microsoft Word Developer's Kit

   Copyright 1994, Microsoft Press
   1024 pages, soft cover with one 3.5-inch disk
   Suggested retail price: (U.S.)$39.95, (Canada) $53.95
   ISBN 1-55615-630-8
Contains all the technical information and tools you need, including detailed information on using WordBasic, using Workgroup MAPI and ODBC extensions, and using the Word API. Comes with a disk that includes sample WordBasic macros and dynamic-link libraries (DLLs) for putting MAPI and ODBC to work. Also includes source files and development tools for programming with the Word API.

Microsoft Word 6.0 for Windows Resource Kit

   Copyright 1994, Microsoft Press
   637 pages, soft cover with two 3.5-inch disks
   Suggested retail price: (U.S.) $39.95, (Canada) $53.95
   ISBN 1-55615-720-7
Contains information and tools you need to roll out and support Word for Windows. Helps you and your users get up to speed quickly, customize your installation of Word, and find answers to your most common questions. Two 3.5-inch disks include customizable self-paced training exercises, 250 tips and tricks, and a sample installation file that makes custom installation easier.
