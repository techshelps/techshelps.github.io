WD: Page Setup Not Available on the File Menu

 Last reviewed: February 18, 1998
Article ID: Q176718
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

The Page Setup command on the File menu (Windows and Macintosh) and the Document command on the Format menu (Macintosh) is not available.

CAUSE/WORKAROUND

By design, the Page Setup command is not available when any of the following conditions exist.

Cause 1: You Do Not Have a Default Printer Driver Selected

To work around this problem, select a default printer driver.

Cause 2: The Installed Printer Driver is Damaged or Corrupt

To work around this problem, remove and reinstall the printer driver.

Cause 3: A Picture, Text Box, Object, AutoShape, or Frame Is Selected

To work around this problem, move the insertion point outside of the picture, text box, object, AutoShape, or frame.

Cause 4: The Document Is Locked for Comments

With the exception of comments, you cannot make changes to the document.

To work around this problem, unprotect the document.

Cause 5: The Document Is Protected for Forms

With the exception of completing an on-line form, you cannot make changes to the document.

To work around this problem, unprotect the document.

Cause 6: The Insertion Point Is Inside a Text Box or Callout

The text box and callout are objects which means that they do not have page properties.

To work around this problem, move the insertion point outside of the text box or callout.

Cause 7: You Are Using In-Place Editing in the Equation Editor or WordArt

Equation Editor and WordArt do not have page properties.

To work around this problem, close WordArt or Equation Editor.

MORE INFORMATION
