Word Assistant: List of Installed Files (Express Setup)

 Last reviewed: February 6, 1998
Article ID: Q110446
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Windows operating system version 3.1

SUMMARY

This article includes a list of all the files that are installed during a Word Assistant Express Setup.

MORE INFORMATION

Note: In the Location column, "Winword" is a placeholder for the location of your Word for Windows program directory (for example, "C:\WINWORD") and "Windows" is a placeholder for the location of your Windows program directory (for example, "C:\WINDOWS"). 

Filename      Location               Details
--------      --------               -------

*.DOT        winword\TEMPLATE       Forms and Templates for Word (~30).

*.WMF        winword\CLIPART        New clip-art images (~100).

*.WMF        winword\SPCLPPR        Special Paper designs used when
                                     choosing Insert Special Paper.

*.TTF         Windows\SYSTEM         New TrueType fonts (=25).

*.FIF         Windows\SYSTEM         Font Information Files for TrueType
                                     fonts (=25).

FONTDB.DLL    Windows                Used to create font database for Font
                                     Selector.

FONTUI.DLL    Windows                Used to create Font Selector user
                                     interface.

HELPPLUS.DLL  Windows                Used in conjunction with Word
                                     Assistant help files.

MSCPYDIS.DLL winword\SETUP          Deals with labeling the Setup Disk and
                                     personalizing the user's copy of Word
                                     Assistant (or whatever application is
                                     being installed).

MSSETUP.DLL  winword\SETUP          Handles various setup functions.

SPUI.DLL      Windows                Used to create Special Paper user
                                     interface.


ACMSETUP.HLP  Not installed          Used during setup.

FAST.HLP      TT_fonts               Font Assistant Help file (accessible
                                     when in the Font Assistant stand-alone
                                     application; located in the TrueType
                                     Tools group under Program Manager).

FONTSEL.HLP  winword                Font Selector Help file (only
                                     accessible through Format Font
                                     Selector).

SPCLPPR.HLP  winword                Special Paper Help file (only
                                     accessible through Insert Special
                                     Paper).

WORDASST.HLP winword                Located on the Help menu in Word 6.0
                                     for Windows. Listed as "Word Assistant
                                     Help."

BANNER.WLL   winword\STARTUP        One time only congratulatory message
                                     box.

FONTTOOL.WLL winword\STARTUP        Used for the FontSelector.

SPCLPPR.WLL  winword\STARTUP        Used for Special Paper.

WASSTHLP.WLL winword\STARTUP        Used for the Word Assistant Online
                                     Help.

FAST.BIN      Windows                Binary file that contains a list of
                                     all installed fonts. Allows the user
                                     with many fonts (over 1000) to use all
                                     of them on the system. Not limited to
                                     the 64K for .INI files.

FAST.EXE      TT_fonts               Font Assistant Executable (stand-alone
                                     application).

FAST.INI      Windows                Used to determine characteristics of
                                     the Font Assistant (stand-alone
                                     application).

READMEWA.TXT winword                README file for Word Assistant.
                                     Contains tips and known problems
                                     that are not in the associated Help
                                     files.

SPCLPPR.SPT  winword\SPCLPPR        Word Assistant Special Paper template.
                                     Attached to your document when you
                                     insert a special paper design into
                                     your document from Insert Special
                                     Paper.

SPCLPPR.INI   Windows                Initialization file for a list of
                                     contents of the Special Paper dialog
                                     box.

WA10_ART.IDX winword\CLIPART        Index file of images, used for the new
                                     Clip Art stand-alone application
                                     (already in Publisher), soon to be in
                                     Microsoft Office. Not used in Word for
                                     Windows

WA10_ART.THM winword\CLIPART        Thumbnail-sized images of the clip art
                                     from Word Assistant. Used to display
                                     Word Assistant clip art in the Clip
                                     Art Gallery. Not used in Word for
                                     Windows.

FONTSEL.INI   Windows                Used to determine characteristics of
                                     the Font Selector window.

WMIMP.FLT     Windows\MSAPPS\GRPHFLT Import filter for WMF files. Word
                                     Assistant updates or installs this
                                     file to ensure that the Special Paper
                                     and new clip-art images are readable.

Files on Word Assistant Disks That Are Not Installed

FILENAME      LOCATION               DETAILS
--------      --------               -------

WASETUP.IN_   Not installed          Used during setup.

WASETUP.DL_   Not installed          Used during setup.
NOTE: All of these files, with the exception of the .TTF files, are deleted when you choose Uninstall in Setup.
