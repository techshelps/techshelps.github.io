WD6: Using the 5.1 Upgrade Template to Convert Address Lists

 Last reviewed: February 5, 1998
Article ID: Q118986
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0. 6.0.1

SUMMARY

This article describes how you can use the 5.1 Upgrade template, supplied with Word 6.0 for the Macintosh, to convert your Word 5.1 address lists.

For more information, including a different method for converting your Word 5.1 address lists, double-click the Help button on the Standard toolbar and then type "address lists" (without the quotation marks).

MORE INFORMATION

Word 6.0 converts address book entries to AutoText entries and to a Word 6.0 table that you can use as a mail merge main document. Use the following procedure to convert your Word 5.1 address lists:

  1. Open the 5.1 Upgrade template, which is located in the Macros folder.

  2. On the View menu, click Toolbars.

  3. From the Toolbars list, select 5.1 Upgrade and then choose OK.

  4. Choose the Convert 5.1 Addresses button on the 5.1 Upgrade toolbar, and then follow the instructions on the screen.

During the conversion, you will be given the following options: 

   Create AutoText Entries
   -----------------------
   
   This option creates an AutoText entry for each address in your Microsoft
   Word 5.1 Address Book. To retrieve the AutoText entry, select the name
   from the Edit AutoText dialog or type the name of the address's AutoText
   entry and press COMMAND+OPTION+V.
   
   Save File As
   ------------
   
   This option automatically saves the resulting address table to a file
   that you can use as a mail merge data source. If you choose to save the
   address table, Word displays a Column Names dialog box, in which you can
   type a name for each column. (If you use this address table as a mail
   merge data source, the column names are the MERGEFIELD names.)
