Printers Not Supporting Mixed Page Size or Mixed Orientation

 Last reviewed: July 30, 1997
Article ID: Q80650
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

If you format a document with multiple orientations (for example, landscape and portrait within the same document) in Microsoft Word for Windows version 2.0, you may receive the following error message: 

   The page size of section X is different from the printer page size. To
   print correctly with the current printer, either change the printer
   paper size in File Print Setup, or change the document size in the
   Format Page Setup.
The above error message displays if you try to print to any of the following printers: 

   All dot-matrix, inkjet, and daisy-wheel printers
   All plotters
   Canon laser printers
   Hewlett-Packard (HP) DeskJet Original and HP DeskJet Plus
   IBM 4019 Laser Printer
   Microsoft Windows 3.0a laser printers not listed as certified (see
   list below)
This problem was corrected in the Microsoft Windows operating system version 3.1.

MORE INFORMATION

The above printers do not support mixed-orientation or mixed-page-size printing if you are running Word for Windows with Microsoft Windows version 3.0 or 3.0a. The printer drivers for these printers require that you set the paper size in the Print Setup dialog box to match the paper size in the Page Setup, Size and Orientation dialog box. This prevents documents that contain mixed orientations or mixed page sizes from printing when using these drivers.

Mixed-orientation and mixed-page-size printing do work correctly on the following Windows version 3.0a printer drivers: 

   HPPCL.DRV: PCL Level IV printers

   HPPCL5A.DRV: HP III family of printers
   - Version 30.3.85 ships with Word for Windows version 2.0.

   DJ500.DRV, DJ500C.DRV: HP DeskJet family
   - Provided by Hewlett-Packard to work with all DeskJet models,
     including the DeskJet and DeskJet Plus

   PSCRIPT.DRV: PostScript printers

   BROHL.DRV: Brother laser printers
   - Included in the Supplemental Driver Library (SDL)

   PANLP.DRV: Panasonic laser printers
   - Available from Panasonic
Note: Unless otherwise noted, the above drivers ship with Windows version 3.0a.
