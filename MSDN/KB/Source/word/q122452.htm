Wrong Font Applied from Drop-Down Box with Adobe Type Reunion

 Last reviewed: November 17, 1997
Article ID: Q122452
The information in this article applies to:
  • Microsoft Word for Macintosh, version 6.0, 6.0.1, 6.0.1a

SYMPTOMS

When you choose a font from the font drop-down box on the Formatting toolbar, an incorrect font is applied to your text.

CAUSE

This behavior is reported to occur when you have both Adobe Type Reunion version 1.1 or 1.2 and the MS LineDraw TrueType font installed in the Fonts folder.

WORKAROUND

To work around this problem, remove the MS LineDraw font from the Fonts folder and restart your computer.

STATUS

There is a conflict between the MS LineDraw font and Adobe Type Reunion. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The MS LineDraw TrueType font is a monospace font used for formatting characters that were captured as an MS-DOS screen shot. This font correctly translates the MS-DOS-based linedraw characters, providing you with an accurate image of what the MS-DOS screen looked like.
