GP Fault Entering Outline View from Print Preview

 Last reviewed: February 5, 1998
Article ID: Q111074
6.00 6.00a 6.00c WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

If you exit print preview by choosing Outline from the View menu, a general protection (GP) fault in WINWORD.EXE may occur.

CAUSE

The GP fault occurs if you open a Word 2.x header or footer pane before you switch to print preview, then you exit print preview by switching to outline view. A GP fault also occurs if you run the spelling checker in the Word 2.x header or footer pane, switch to print preview, apply page setup formatting, undo that formatting, and then exit print preview by choosing Outline from the View menu.

To reproduce the problem

  1. In your Word document, open the Word 2.x header or footer pane by running the following macro: 

       Sub MAIN
    NormalViewHeaderArea .Type = 1
   End Sub

  2. Switch to print preview by choosing Print Preview from the File menu or by choosing the Print Preview button on the Standard toolbar.

  3. In print preview, choose Page Setup from the File menu and select the Margins tab.

  4. Change one of the Margins settings and then choose OK.

  5. From the Edit menu, choose Undo Page Setup.

  6. From the View menu, choose Outline.

By contrast, the GP fault does not occur if, instead of step 6 above, you exit print preview by choosing the Outline button on the horizontal scroll bar.

STATUS

Microsoft has confirmed this to be a problem in Word versions 6.0 and 6.0a for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUNDS

Method 1: Close the Word 2.x header or footer pane before you switch to 

          print preview.
Method 2: Exit print preview by using a method other than choosing Outline 
          from the View menu. If you want to switch to outline view from
          print preview, choose the Outline button on the horizontal
          scroll bar.
