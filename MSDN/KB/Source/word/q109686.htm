WordBasic Error Running Third-Party Word 2.x Macro in Word 6.0

 Last reviewed: July 30, 1997
Article ID: Q109686
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Windows operating system version 3.1

SYMPTOMS

This article lists reports of problems running third-party Word 2.x macro packages in Word 6.0 for Windows.

Microsoft does not support the modification of third-party macros. You should contact the manufacturer of your macro for an updated version or for instructions on modifying the macro.

MORE INFORMATION

Ascend 4.0, by Franklin Quest

Ascend 4.0 macros generate a list of names, addresses, and phone numbers. When you run the Ascend package with Word 6.0, the following WordBasic error message occurs: 

   WordBasic Err=22
   Invalid array dimension
To contact Franklin Quest for Ascend 4.0 technical support, phone (801) 975-9999.

The Ascend products included here are manufactured by vendors independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.
