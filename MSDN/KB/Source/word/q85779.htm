OS/2 2.0: Printer Drivers Included on Device Driver Disk #1

 Last reviewed: August 4, 1997
Article ID: Q85779
1.10 1.10a 1.10b OS/2

SUMMARY

The following printer drivers are included on Device Driver Disk #1 of the OS/2 version 2.0 General Availability release.

Note: The filename listed in brackets ([]) is the filename of the printer driver used for the listed models.

MORE INFORMATION

PostScript Printers [PSCRIPT.DRV]

Note: This list includes printers that do not normally operate in PostScript mode. To use a PostScript driver with these types of printers, an optional add-on (either a board or cartridge) must be added to support the PostScript command language.
