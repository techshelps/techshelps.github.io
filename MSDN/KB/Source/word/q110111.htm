Customizing the Font Assistant in Word Assistant

 Last reviewed: February 6, 1998
Article ID: Q110111
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word Assistant version 1.0

SUMMARY

You can customize the Font Assistant (FA) through its initialization file, FAST.INI. This file is located in the Windows program directory.

MORE INFORMATION

The Word Assistant installs the Font Assistant in the TrueType Tools group under Program Manager. This utility is used to group fonts, display fonts as printed, and list font characteristics. It is true WYSIWYG. This utility can be modified to better suit your needs. Modifications are made through the FAST.INI file. This is the initialization file for the Font Assistant component of Word Assistant.

Below is a sample FAST.INI with comments and settings to modify the look of Font Assistant options.

Note: This information can be found in the Font Assistant Help file. From the Help menu, choose Contents. Choose the Reference option and select "Customizing Your FAST.INI File."
