Running Word for Windows Tutorial May Cause Low Memory Message

 Last reviewed: February 5, 1998
Article ID: Q110711
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

When you run Word 6.0 Windows tutorials (or attempt to perform other tasks) on a machine with 4 megabytes (MB) of memory, when Windows is in standard mode, you may receive "Low Memory" error messages.

CAUSE

This problem occurs because Word 6.0 for Windows uses most of the available RAM; the system does not have enough available memory to create a page frame. When this happens, the tutorial or other programs may not have enough memory available to run.

MORE INFORMATION

When Windows is running in standard mode you must have enough RAM to create a page frame in the reserved address area (640K to 1 MB address range). A page frame is a contiguous 64K segment. Windows in standard mode uses this method (paging) to swap code in and out of RAM.

WORKAROUND

To work around this problem:

  • Restart your computer and do not run any other programs while running Word.
  • Do not edit more than one document at a time.
  • Free up as much RAM as possible or add more RAM to the system.
