WD: Macro to Remove a Subdirectory in Word for Windows

 Last reviewed: February 16, 1998
Article ID: Q87376
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SUMMARY

The RmDir command can be used to remove a specified folder, directory or subdirectory. Before the directory can be deleted, all the files in the directory must be removed. The Kill statement can be used to delete files from a directory.

Syntax

RmDir DirName$

The following example deletes all files from a directory and then deletes the directory: 

   Kill "C:\WINWORD\DOCS\*.*"
   RmDir "C:\WINWORD\DOCS"
NOTE: If you are using Word 6.x for the Macintosh, you must include a valid Macintosh path, as in the following example: 

   RmDir "Mac HD:Microsoft Word 6:Test Document"
