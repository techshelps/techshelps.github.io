WD: Some WordBasic Commands Unavailable in Protected Forms

 Last reviewed: February 3, 1998
Article ID: Q126599
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

This article contains a list of WordBasic commands that are unavailable when you are working in a protected form. Some of the commands can be used with keystrokes but do not function when used in a protected form.
