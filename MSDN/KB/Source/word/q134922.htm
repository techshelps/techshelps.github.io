Using CTRL+ALT+HYPHEN with Right Mouse Click Causes Error

 Last reviewed: February 5, 1998
Article ID: Q134922
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

When you press CTRL+ALT+HYPHEN and then use the right mouse button to click a word to access the spelling shortcut menu, you may receive the following error message: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.

   Winword caused an invalid page fault in module Mso95.dll at
   0137:50602381.
NOTE: CTRL+ALT+HYPHEN is used to delete an item from a menu.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word for Windows 95, version 7.0. This problem has been corrected in Microsoft Word 97 for Windows.

WORKAROUND

Press the ESC key to cancel the CTRL+ALT+HYPHEN functionality before you use the right mouse button to click a word.
