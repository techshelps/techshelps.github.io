Table Gridlines Not Visible with Paradise Accelerator Card

 Last reviewed: July 30, 1997
Article ID: Q88895
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, and 2.0a-CD
  • Microsoft Windows operating system version 3.1

SUMMARY

If you have a Western Digital Paradise Accelerator Card and use the Western Digital 800x600 16-color video driver for Microsoft Windows operating system version 3.1, table gridlines may not be visible in your Microsoft Word for Windows documents.

MORE INFORMATION

Western Digital Corporation manufactures the Paradise Accelerator Card for use with Microsoft Windows. This card is an adapter that replaces the normal VGA card and increases graphics performance in Windows. The card also ships with its own video drivers for MS-DOS and versions 3.0 and 3.1 of Windows. The Windows version 3.1 video drivers support a range of resolutions from 640x480 to 1280x1024 in 16 or 32 colors.

If you install the 800x600 16-color video driver, table gridlines are not displayed on the screen. If you add border formatting to the table, gridlines are displayed correctly. If you want to display table gridlines without adding border formatting, use a different Paradise video driver, or use the 800x600x16 Super VGA video driver supplied with Windows version 3.1.

Western Digital Corporation has confirmed this to be a problem with the Western Digital 800x600 16-color video driver for Windows version 3.1. For information about updated video drivers, call Paradise technical support at (714) 932-4900 or access the Paradise bulletin board service (BBS) at (714) 753-1234.
