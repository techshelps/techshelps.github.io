Formatting for Avery Laser Business Cards Product Number 5371

 Last reviewed: July 30, 1997
Article ID: Q98451
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

This article contains information from the Avery documentation for "Laser Business Cards," product number 5371.

MORE INFORMATION

Use the following steps to format and print Avery Laser Business Cards with Word for Windows 2.0:

  1. Create a new document by choosing New from the File menu. Base the document on the NORMAL template.

  2. From the Format menu, choose Page Setup and set the Margins as follows (in inches): 

          Top:        .5
      Bottom:     .25
      Left:       .75
      Right:      .75

  3. From the Table menu, choose Insert Table and specify the following settings: 

          Columns:      2
      Rows:         5
      Column Width: 3.5

  4. From the Table menu, choose Select Table to select the entire table.

  5. From the Table menu, choose Row Height and specify the following settings: 

          Indent from left:   0
      Height of Rows 1-5: Exactly
      At:                 2 in
      Alignment:          Left

  6. While the table is still selected, choose Column Width from the Table Menu and specify the following settings: 

          Width of Columns 1-2:  3.5
      Space between Columns: 0

  7. You have completed setting up the page layout for your business cards. Design your business card in the upper-left cell of the table. When you are satisfied with your design, copy that cell and paste it into the other nine cells.

  8. Once you have formatted the cards correctly, choose Print from the File menu to print your business cards.

Avery suggests you print on plain paper first to check the formatting.
