WD: Some Menu Commands Unavailable (Document Protection)

 Last reviewed: February 2, 1998
Article ID: Q105697
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

Under certain conditions, you may be unable to access all of the commands in Word, or you may be unable to move the insertion point to certain areas of the document.

CAUSE

This problem may occur if document protection is enabled.

The "More Information" section of this document lists commands that are available when document protection is enabled.

NOTE: It is possible to customize all of the menus, keyboard commands, and so forth, with AutoNew macros. Also, add-in products (such as Microsoft Internet Explorer or Microsoft Bookshelf) modify menu choices. Customization may also produce the behavior described in the "Symptoms" section of this article.

WORKAROUND

To gain access to all of the menu commands and areas of a document, unprotect the document.

MORE INFORMATION

You can protect a document by clicking Protect Document on the Tools menu.

The following table lists the features that you can protect in a document (these features vary depending on the version of Word you are using). Each one of these features has settings that are enabled or disabled when you click Protect Document on the Tools menu. 

   Protected Item     Version of Word
   -----------------------------------------------------

   Tracked Changes    Microsoft Word 97 for Windows
                      Microsoft Word 98 Macintosh Edition

   Revisions          Microsoft Word 6.x for Windows
                      Microsoft Word 6.x for Macintosh
                      Microsoft Word for Windows 95,
                      version 7.0

   Comments           Microsoft Word 97 for Windows
                      Microsoft Word 98 Macintosh Edition

   Annotations        Microsoft Word 6.x for Windows
                      Microsoft Word 6.x for Macintosh
                      Microsoft Word for Windows 95,
                      version 7.0

   Forms              All versions of Word

Tracked Changes/Revisions

When you protect a document for Tracked Changes or Revisions, all modifications are entered with revision marks. The same menu choices are available as in a document that has no protection applied.
