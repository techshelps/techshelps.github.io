WD6: Envelopes Print Incorrectly to LaserWriter Select 300

 Last reviewed: February 5, 1998
Article ID: Q121711
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

Envelopes may print incorrectly to a LaserWriter Select 300. The line spacing of the return and destination addresses text may be incorrect; text may be truncated or overlap. The Manual Feed option does not work correctly, which causes a regular sheet of paper to be printed.

CAUSE

Word incorrectly addresses the printer driver when printing from the Envelope And Labels dialog box.

RESOLUTION

To correct this problem, upgrade to Word version 6.0.1 for the Macintosh.

To temporarily work around this problem, print a single envelope correctly from Word 6.0 to the LaserWriter Select 300. To do this, follow these steps:

  1. Create a new blank document.

  2. On the Tools menu, click Envelopes And Labels. Select the Envelope tab.

  3. Type the destination and return addresses.

    NOTE: If you want to omit a return address, do not select the Omit check box. Rather, delete the text in the Return Address box. If the Omit box is selected, you will not be able to complete later steps.

  4. Click the Add To Document button.

  5. Once the envelope is added to the top of the document, click Print on the File menu.

  6. Click the Range button.

  7. In the Pages box, type a "0" (zero, without the quotation marks) and choose OK.

  8. From the Paper Source list, select Manual.

  9. Click the Print button.

STATUS

Microsoft has confirmed this to be a problem in Word version 6.0 for the Macintosh. This problem was corrected in Word version 6.0.1.
