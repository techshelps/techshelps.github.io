WD98: Page Break Deleted if Table Drawn Immediately Below It

 Last reviewed: February 3, 1998
Article ID: Q180096
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you draw a table immediately below a manual page break, the page break will be deleted.

WORKAROUND

To draw a table below the manual page break without deleting the page break, follow these steps:

  1. Click on or below the page break.

  2. Press RETURN to insert a paragraph mark immediately below the page break.

  3. Draw the table below the paragraph mark you inserted. To draw the table, click the Insert Table button on the Standard toolbar.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
