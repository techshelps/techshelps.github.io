WD95: Quiet Mode Setup Does Typical Instead of Complete Setup

 Last reviewed: February 6, 1998
Article ID: Q139563
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

During Quiet Mode Setup, a Typical installation is performed instead of the expected Complete installation.

CAUSE

Typical is now the default installation type for all modes of Setup.

RESOLUTION

To force Quiet Mode Setup to perform a Complete installation, use the appropriate command line below:

To install from disk or disk images: 

   Setup /B4 /Q
To install after Administrative Setup with the Workstation option: 

   Setup /B5 /Q

MORE INFORMATION

The /B switch corresponds to the buttons shown when you are prompted to choose the installation type. Typical is /B1, Compact is /B2, Custom is /B3, Workstation is /B4 (if you are running Quiet Mode Setup from disk images, this switch represents Complete Setup), and /B5 is Complete (if you already did an Administrative Setup).
