Not All Installed Printers Show Up in Word for Windows

 Last reviewed: July 30, 1997
Article ID: Q89813
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

Sometimes not all the installed printers appear in the Word for Windows Print Setup dialog box.

To make name of the printer to which you want to print visible in the list of installed printers, follow the appropriate procedure below.

Word for Windows Versions 2.0, 2.0a, 2.0a-CD, 2.0b and 2.0c

  1. In the Windows Control Panel, choose the Printers icon.

  2. Select the desired printer in the Installed Printers box.

  3. Choose the Set As Default Printer button.

  4. Return to Word for Windows and print as usual.

Word for Windows Versions

  1. In the Windows Control Panel, choose the Printers icon.

  2. Choose a printer that you are not going to use and choose the Remove button.

  3. Choose the Yes button when Windows asks if you are sure you want to remove the printer.

  4. Count the total number of installed printers in the Control Panel and compare this to the number of printers visible in the Word for Windows Print Setup dialog box.

  5. Repeat steps 1 through 4 until the Word for Windows Print Setup dialog box and the Control Panel Installed Printers list are the same.

  6. Print as usual.

MORE INFORMATION

Word for Windows does not display all the installed printers when the number of printers is at or near the Windows version 3.1 maximum of 16 installed printers. The number of printers Word will display in the list of installed printers varies from one computer to another. Word for Windows removes printers from the list of installed printers if you have the maximum number of printers installed on one port.

In versions 1.0, 1.1, and 1.1a of Word for Windows, you cannot print to a printer that isn't displayed in the Printer Setup dialog box. If you try to print, the following error message occurs: 

   Cannot Print
