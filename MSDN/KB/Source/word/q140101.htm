Adobe Type Manager Fonts Don't Print Kerned in Windows 95

 Last reviewed: February 5, 1998
Article ID: Q140101
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

Your Adobe Type Manager fonts do not print "kerned" when you print from Word for Windows 95.

CAUSE

The print kerning feature has been disabled in Windows 95.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows 95 version 7.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Use a printer or TrueType font instead of the Adobe Type Manager font.

MORE INFORMATION

Kerning allows you to adjust the amount of space between characters by changing either spacing or kerning. The Spacing option changes the spacing between selected characters by the same amount for all characters. With kerning, the spacing between characters depends on the font and the selected characters.
