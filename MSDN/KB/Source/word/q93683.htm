System Error: Cannot Read from Drive "N"

 Last reviewed: October 8, 1997
Article ID: Q93683
The information in this article applies to:
  • Microsoft Word for Windows, version 2.0a

SUMMARY

The following error may appear when when you try to open a file in version 2.0a of Word for Windows: 

   System Error: Cannot read from drive "N"
where "N" is a drive letter from a-z. This error may occur even when Word should not be accessing the drive indicated in the error message. This article describes conditions under which Word incorrectly accesses a drive, thereby generating the above error message.

This problem does not occur in versions 2.0, 2.0b, and 2.0c of Word for Windows. In these versions, the "Cannot read from drive <n>" error message only occurs appropriately, when you attempt to read from the active drive when it is inaccessible. For example, Word displays this error message when you try to access a floppy disk drive when it contains no disk. To resolve the error, place a floppy disk in the drive and choose the Retry button.

MORE INFORMATION

In version 2.0a of Word for Windows, this error message occurs inappropriately in the following situations:

  • When a special driver is loaded to allow your PC to exchange data with a mainframe computer, the error message occurs when no connection to the mainframe exists. Word cannot access the mainframe drive, even though it appears in the Drives list in the Open dialog box. 

        -or-
  • If you remove a moveable hard disk or a floppy disk, the drive continues to appear in the Drives list. Word reports that it cannot read from the drive, even though it should not be accessing the moveable hard disk or floppy disk drive. 

        -or-
  • When you open and close a file from a floppy disk drive, then choose Open from the File menu. The "Cannot read from drive <n>" error appears, and you must choose the Cancel button eight or ten times before the error clears and the Open dialog box opens to allow you to open another file.

Steps to Reproduce Problem

  1. Insert a floppy disk containing a Word for Windows document into a floppy disk drive.

  2. Start version 2.0a of Word for Windows. From the File menu, choose Open.

  3. In the Drives list, choose the floppy disk drive and choose the OK button.

  4. In the File Name list, choose the Word document and choose the OK button.

  5. From the File menu, choose Close. Remove the floppy disk from the drive.

  6. From the File menu, choose Open. The error message occurs and you must choose the Cancel button eight or ten times before the Open dialog box opens.

    Note: This problem does not occur if you access the floppy drive but do not open a file from it.

Microsoft has confirmed this to be a problem in Word for Windows version 2.0a. This problem was corrected in Word for Windows version 2.0b.
