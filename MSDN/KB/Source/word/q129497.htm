WD6X: Doc Error: No Word51Keystrokes Macro (Resource Kit) on Mac

 Last reviewed: February 3, 1998
Article ID: Q129497
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

The "Microsoft Word 6 Resource Kit, 2nd Edition" (page 689), states that the 5.1 Upgrade template, provided with Microsoft Word 6.0 for the Macintosh, includes a macro called Word51Keystrokes. This macro did not ship with Word 6.0. In Word 6.0.1, the macro is called Word51Keys.

MORE INFORMATION

To obtain the Word51Keys macro, upgrade to 6.0.1. This macro is included in the 5.1 Upgrade template. Run the macro to add Word 5.1 key combinations to Word 6.0 for the Macintosh.

To work around the problem until you obtain the 6.0.1 update, you can convert the Word Settings (5) file in Word 6.0. For information on how to do this, start Word. Open the 5.1 Upgrade template (typically located in the Word:Macros folder) and follow the instructions in the section titled "Converting Settings files from Microsoft Word 5.1".

REFERENCES

"Microsoft Word 6 Resource Kit, 2nd Edition," page 689
