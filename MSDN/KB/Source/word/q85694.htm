Low Memory Error Message When Embedding Large Excel Object

 Last reviewed: February 6, 1998
Article ID: Q85694
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a
  • Microsoft Excel for Windows, versions 4.0, 4.0a

SYMPTOMS

If you attempt to embed a large Microsoft Excel version 4.0 or 4.0a worksheet object in Microsoft Word for Windows, you may receive a low memory error message.

Word Version 6.x

   There is insufficient memory. Save the document now.
Note: This is not a problem with Excel version 5.0.

CAUSE

Depending on the available memory, Excel cannot embed more than approximately 66 rows of data into Word for Windows. If you attempt to embed a larger selection, a low memory error will occur.

WORKAROUND

Embedding smaller objects works correctly.

Reference(s):

"Microsoft Word for Windows User's Guide," version 2.0, page 692
