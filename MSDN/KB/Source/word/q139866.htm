How to Install the CGM and CorelDraw Filters

 Last reviewed: February 6, 1998
Article ID: Q139866
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

When you try to insert a CGM or CorelDraw picture, its name is not available in the List Files Of Type list. If you select a file from the list and click OK, Word displays the Convert File From list. The file type you are looking for is not on this list. If you select one of the types anyway, you get the following error message: 

     This file was not recognized by the specified graphics filter.

CAUSE

When you installed Word, you chose to perform a Typical, Custom, or Compact installation. If you chose Custom, you may have cleared the Converters And Filters option. Some graphics filters are not installed with a Typical or Compact installation. 

Installed during Typical Setup        NOT installed during Typical Setup
------------------------------        ----------------------------------

TIF                                   EPS
BMP                                   Targa
WPG                                   DXF
DrawPerfect Export                    CGM
PICT                                  PCX
WMF                                   DRW
JPG                                   CorelDraw
                                      HPGL
                                      GIF
                                      Kodak

RESOLUTION

Windows 95

To install the missing graphics filters, use the following steps:

  1. Quit Word.

  2. Click the Windows Start button, point to Settings, and click Control Panel.

  3. Double-click Add/Remove Programs.

  4. Click the Install/Uninstall tab.

  5. In the list of installed software, select either Microsoft Word or Microsoft Office.

  6. Click Add/Remove.

  7. In the Setup box, click Add/Remove.

  8. If you are using Office Setup, follow these steps:

    a. Select Converters, Filters, and Data Access, and click Change Options.

    b. Select Graphics Filters, and click Change Option.

    c. Select the filters you want. If you want to install the CGM filter, select Computer Graphics Metafile Import. Click OK.

    d. Click OK again.

    e. Click Continue.

    f. When prompted by Setup, insert the appropriate disk or compact disc.

    The new component is installed.

  9. If you are using Word Setup, follow these steps:

    a. Select Converters and Filters. Click Change Option.

    b. Select Graphics Filters, and click Change Option.

    c. Select those graphics filters you want to install.

    d. Click OK.

    e. Click OK again.

    f. Click Continue.

    g. When prompted by Setup, insert the appropriate disk or compact disc.

    The new component is installed.

Windows NT

If you are running Word for Windows 95 on Windows NT, substitute the following steps for steps 1-5 above:

  1. Quit Word

  2. From Program Manager, choose Run from the File menu.

  3. Type a:\setup. Click OK. Insert Disk 1 of the Word Setup disks. Continue with step 6 above.
