WD: WordBasic ToggleFieldDisplay Statement Defined

 Last reviewed: February 2, 1998
Article ID: Q99957
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.10, 1.10a 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

The Microsoft WordBasic ToggleFieldDisplay statement toggles the display of the selection between field codes and field results. ToggleFieldDisplay is the equivalent of pressing SHIFT+F9 in a document window. The selection of Field Codes under the View menu is not affected by the ToggleFieldDisplay command.

The ViewFieldCodes statement affects the entire Word document, while ToggleFieldDisplay affects only the selected field in the Normal view. When the active view is page layout, the ToggleFieldCodes statement toggles all of the fields in the document.

The following macro example finds the next field in the document, if field codes are displayed in the current document, the field is toggled to display the field result instead of the field code. The field result text is then printed to the status bar with the Print statement. 

   Sub MAIN
      ViewNormal
      If NextField() Then
         If ViewFieldCodes() Then ToggleFieldDisplay
      End If
      Print Selection$()
   End Sub
