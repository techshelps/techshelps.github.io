WD6X: Cannot Suppress Print Dialog Box Using AppleScript

 Last reviewed: November 17, 1997
Article ID: Q125579
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0

SYMPTOMS

When you use the Print command in an AppleScript script to print a document, the Print dialog box appears. This dialog box requires you to click the Print button before the document will be printed.

CAUSE

There is no way to suppress the Print dialog box using AppleScript. This is a limitation of the current version of AppleScript.
