WD: "Selected Device Not Responding..." Error Message

 Last reviewed: November 17, 1997
Article ID: Q30797
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 3.0, 3.01, 3.02, 4.0, 5.0, 5.1, 6.0, 6.0.1, 6.0.1a

SYMPTOMS

When printing to the Apple ImageWriter LQ printer, the following error message will occasionally appear at the end of the print job: 

  The selected device is not responding as an LQ ImageWriter. Please
  check port selection, cabling, and device type.

CAUSE

This message is caused by a timing problem between the Macintosh and the printer and can be ignored because the error occurs after the document is printed.

MORE INFORMATION

This problem will occur more frequently with the printer connected to the printer port than when it is connected through the modem port.
