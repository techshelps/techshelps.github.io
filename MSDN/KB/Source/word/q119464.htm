WD6X: "Word Cannot Obtain the Data for the Unknown Link" Error

 Last reviewed: November 17, 1997
Article ID: Q119464
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

You may receive the following error message when you try to paste link from earlier versions of Word for the Macintosh into Word version 6.0: 

   Word cannot obtain the data for the Unknown link.
This error is generated regardless of which format you select in the Paste Special dialog box. This error is not generated when you choose the Paste command instead of Paste Link.

STATUS

Microsoft has confirmed this to be a problem in Word version 6.0 for the Macintosh. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Choose Paste instead of Paste Link.
