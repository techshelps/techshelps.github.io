Changing Envelope Feed Icon Has No Effect on Printout

 Last reviewed: February 5, 1998
Article ID: Q106045
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Windows version 3.1

SUMMARY

The Feed Method section of the Envelope Printing Options tab offers several different envelope feed methods. However, changing the feed method will not modify the way the printer prints the envelope. (To locate the Feed Method section, choose Envelopes And Labels from the Tools menu, choose the Options button, and select the Printing Options tab.)

MORE INFORMATION

Changing any of the values in the Feed Method section does not change the way the printer prints; it only lets Word know how the printer prints in the event that Word chooses the incorrect feed method for the printer.

For example, if you normally feed the envelope into the printer with the stamp on the opposite end from what is selected, selecting this option will let Word know that. This option will not make your printer print envelopes differently than it normally does.
