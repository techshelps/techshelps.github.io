MacWord Setup Creates Two Registration Databases

 Last reviewed: February 6, 1998
Article ID: Q120143
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

When you run Setup to install Word for the Macintosh, it creates the following two registration database documents in the System:Preferences folder: 

   Registration Database

   PPC Registration Database

CAUSE

By design, Setup creates two registration databases. The Registration Database file is for a 68K Macintosh, while the PPC Registration Database file is for a PowerPC. Setup installs both files--regardless of the computer you have--for compatibility, in case you upgrade your Macintosh to a PowerPC.

MORE INFORMATION

If you do not have a PowerPC, you do not need the PPC Registration Database, so you can delete it. If you later upgrade to a PowerPC, Word will automatically create a new PPC Registration Database file in the System:Preferences folder. Likewise, if you delete your Registration Database file, Word creates a new one.
