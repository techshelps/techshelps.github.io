WD: Unusual behavior with "WordPerfect Navigation Keys" On

 Last reviewed: February 6, 1998
Article ID: Q107584
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, verison 6.0

SUMMARY

Word incorporates an option that allows Word to emulate the behavior of WordPerfect. Selecting WordPerfect Help will cause unexpected results from what is the normal function of Word. Below is a list of expected outcomes for Word and a corresponding list of outcome if WordPerfect Help has been selected.

To select WordPerfect Help from the Tools menu choose Options, choose the General tab and select the Help for WordPerfect User's checkbox. When WordPerfect Help has been select the letters WPH will appear on the status bar in box furthest to the left.

To Disable Help for WordPerfect Users

  • Double-click the status bar button labeled WPH.

    -or-

  • Choose WordPerfect Help from the Help menu, choose Options, clear the "Help for WordPerfect Users" check box, and then choose OK.
