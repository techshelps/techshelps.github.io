WD6: Internet Assistant How to Set the Length of History List

 Last reviewed: February 5, 1998
Article ID: Q129558
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SUMMARY

This article describes how to set or change the length of the History list in Internet Assistant. During setup, the length of this list is set to 50.

MORE INFORMATION

To control the length of the History List, edit the HTMLHIST.INI file (located in the Windows program directory) using a text editor such as Notepad. Place the following command in the [Config] section 

   MRULENGTH=n
where n is the number of URLs you want the History list to display. You can use any number except 0.

Note: During any given browsing session, Internet Assistant displays every URL you jump to in the History list. When you exit Word, the list is refreshed and stores only the number you defined with MRULENGTH.
