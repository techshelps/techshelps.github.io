Word: Error Message "Style Too Complex"

 Last reviewed: July 30, 1997
Article ID: Q57831
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 3.0 , 3.01, 3.02, 4.0 5.0

SUMMARY

When applying a style in Microsoft Word for the Macintosh, versions 3.0, 3.01, 3.02, 4.0, and 5.0, the error message "Style Too Complex" can result. Clicking the OK button will often apply the style anyway.

The error message is usually generated because the style contains a great deal of formatting. The most common cause for receiving this message is a style based on a style, or several previous styles. The more a style is based on previous styles, the more complex it is. It may become too complex in this way.

Re-creating the style from scratch, instead of basing it on other styles (or basing it on the Normal style) usually solves the problem.

MORE INFORMATION

A corrupt Word Settings file can also cause this error message to appear. If this is the case, remove the file called Word Settings (4) from the System Folder on your startup disk and restart your Macintosh. Word will reconstruct a new Word Settings (4) file called Word Settings (4) in the System Folder on your startup disk. However, this will cause any styles and customization that were stored in the old Word Settings (4) file to be lost. (Note that this file will be called Word Settings (5) if using Word 5.0 and will be simply called Word Settings in Word 3.0x.)
