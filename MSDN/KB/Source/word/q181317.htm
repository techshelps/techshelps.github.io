WD5x: How to Use a Word 5.x Dictionary in Word 6.0 or Word 98

 Last reviewed: February 18, 1998
Article ID: Q181317

The information in this article applies to:

  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 5.1a

SUMMARY

When you upgrade from Word 5.x to Word 6.0 or Word 98, you may want to be able to use your Word 5.x custom dictionary in the later version of Word.

To install a custom dictionary from Word 4.x or 5.x for the Macintosh:

  1. Open a new document based on the Normal template by choosing the New button on the Standard toolbar.

  2. On the Tools menu, click Options (Preferences in Word 98).

  3. Click the Spelling tab (Spelling and Grammar in Word 98).

  4. Under Custom Dictionaries (in Word 98 click Dictionaries and then click Add), click Add.

  5. From the List Files Of Type box, select All Files.

  6. Select the custom dictionary you want to use in Word 6.0 or Word 98 and then click Open.

    Word adds the dictionary to the Custom Dictionary in the Options (Preferences in Word 98) dialog box and selects the check box next to it. When this happens, the custom dictionary is ready to use in Word.

  7. Choose OK.

To install a custom dictionary from Word 6.0 for Windows:

  1. Transfer the custom dictionary to the Macintosh environment using a file transfer utility such as Apple(R) File Exchange.

  2. Start Word 6.0 or Word 98 for the Macintosh.

  3. Follow steps 1 through 7 in the previous procedure, "To install a custom dictionary from Word 4.x or 5.x for the Macintosh."

MORE INFORMATION

Word 98:

For more information about selecting custom dictionaries, click Contents And Index on the Help menu (or on the Balloon Help menu if you are using a version of the Macintosh operating system earlier than 8.0), click the Index button in Microsoft Word 98 Help, type the following text 

   dictionaries, spelling
and then click Show Topics. Select the "Customize spelling and grammar checking" topic, and click Go To. If you are unable to find the information you need, ask the Office Assistant.

Word 6.0:

Double-click Help button on the Standard toolbar and then type "custom dictionaries" (without quotation marks).
