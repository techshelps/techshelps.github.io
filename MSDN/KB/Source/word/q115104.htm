Pleading Wizard (PLEADING.WIZ) Differs from Word 6.0 to 6.0a

 Last reviewed: February 5, 1998
Article ID: Q115104
6.00 6.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SUMMARY

The Pleading Wizard supplied with Word 6.0a for Windows prompts for and adds a caption box that includes the Plaintiff, Defendant, Case Number, and Pleading Title.

By contrast, the Pleading Wizard supplied with Word 6.0 for Windows does not prompt for or add this caption.

MORE INFORMATION

Below is a description of the changes in the Word 6.0a version of the Pleading Wizard.

Caption Screen (New Screen)

In Word 6.0a, this screen asks whether you want to add a caption to the document and gives you four options: Style 1, Style 2, Style 3, or No Caption. If you select one of the styles, the wizard displays the Caption Information screen (also a new screen) where you can type the caption information (Plaintiff, Defendant, Case Number, and Pleading Title). If you select No Caption, the wizard proceeds to the Finish screen.

The following is a sample caption, using the Style 1 caption option: 

PLAINTIFF'S NAME,               )  No. 12-3-456789-1
     Plaintiff,                 )
     vs.                        )  PLEADING TITLE
DEFENDANT'S NAME,               )
     Defendant                  )
                                )
                                )
                                )
                                )
                                )

Caption Information Screen (New Screen)

If you selected a caption style in the previous Caption Screen, the Pleading Wizard displays the Caption Information screen, where you can type the following information: Plaintiff, Defendant, Case Number, and Pleading Title. The wizard uses the information you provide to create a caption in your pleading document.

Courts Name Screen (Minor Change)

This is a wording change only. The name of the check box you can select if you want to center the name of the court is "Centered" in Word 6.0 and "Center" in Word 6.0a.
