Setup Repeatedly Prompts for Disk 2 with Netroom Installed

 Last reviewed: February 6, 1998
Article ID: Q122654
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

Microsoft Word for Windows Setup repeatedly prompts for Disk 2, even though the correct disk is in the floppy disk drive.

CAUSE

This problem can occur when you have Helix Netroom installed. The Multimedia Cloaking option of Netroom causes a memory conflict with Setup.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word versions 6.0, 6.0a, and 6.0c for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

RESOLUTION

To correct this problem, temporarily disable the Multimedia Cloaking option that Netroom installs in the AUTOEXEC.BAT file. The lines appear similar to the following: 

   C:\NETROOM\XLOAD.EXE -SD001 -M31488 C: \MMCLK\CACHECLK.EXE /K=3072
   /WK=3072 /H=2048 /WB=1

   -or-

   C:\NETROOM\XLOAD.EXE -SD001 -M47760 C: \MMCLK\MSCDEX.EXE /D:MSCD001 /L:D
   /M:15
Type "rem" (without the quotation marks) preceding each line and then reboot you computer. The lines should appear similar to the following: 

   REM C:\NETROOM\XLOAD.EXE -SD001 -M31488 C: \MMCLK\CACHECLK.EXE
   /K=3072 /WK=3072 /H=2048 /WB=1

   -or-

   REM C:\NETROOM\XLOAD.EXE -SD001 -M47760 C: \MMCLK\MSCDEX.EXE
   /D:MSCD001 /L:D /M:15
Now you should be able to install Word correctly. After you have installed Word has been installed, you can remove the REM statements from the AUTOEXEC.BAT file to re-enable Multimedia Cloaking. Multimedia Cloaking affects the Setup portion of Word only.
