WD: Alki Proofing Tools Upgrade Information

 Last reviewed: July 31, 1997
Article ID: Q171600
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SUMMARY

If you own Alki Software Corporation's Proofing Tools for Microsoft Word, you are eligible for a discount if you upgrade the Proofing Tools for a later version of Word.

To order an upgrade of the Proofing Tools, call Alki at (800) NOW-WORD (669- 9673) or (206) 286-2600. To qualify for the upgrade price, you must supply the registration number of the Alki software you already own, or provide other proof of purchase.

The Alki Proofing Tools for Microsoft Office 97 provide improvements over previous versions, including:

  • You can check spelling in Microsoft Access, Microsoft Excel, Microsoft Outlook, Microsoft PowerPoint, and Microsoft Word.
  • You can use thesauri including antonyms and synonyms in the following packages: Catalan, Finnish, Italian, and Swedish. Antonyms are also included in the Dutch, English, Russian, and Slovak thesauri.
  • You can use grammar checkers in the following packages: Dutch, German, Italian, and Spanish. Grammar is included in the French, English, Hungarian, and Russian packages.
  • The Proofing Tools include updated spelling and thesaurus files.
  • The Proofing Tools work with Microsoft Office 97, on the Windows 95, Windows NT 3.5.1, or Windows NT 4.0 platforms.

MORE INFORMATION

Owners of Alki's Proofing Tools packages used in Word 7.x may continue to use them with Word 97 if Office 97 is installed in the same folder as Word 7.x. However, these packages cannot be used with any other Office applications; nor do they include any updated tools for the Office 97 packages.

Contact the Alki Software Corporation for more information about how to upgrade your foreign language, legal, and medical dictionaries. Alki products use a different setup program for each version of Word.
