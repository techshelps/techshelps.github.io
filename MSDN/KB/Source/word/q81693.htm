NORMAL.DOT Template File Not Found in WinWord Directory

 Last reviewed: February 5, 1998
Article ID: Q81693
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

In Microsoft Word for Windows version 2.0, the Normal.dot template file does not appear in the Word for Windows directory until you have made a change to the Normal template and saved it.

Note that in later versions of Word for Windows, a Normal.dot template file is created the first time you exit the program.

MORE INFORMATION

To force Word for Windows version 2.0 to create a Normal.dot file, follow the steps below:

  1. From the File menu, choose New, then choose the OK button.

  2. Type "one" (without the quotation marks), and select it.

  3. From the Edit menu, select Glossary. In the Glossary Name box, type "one" (without the quotation marks).

  4. From the File menu, choose Save All. Choose No when prompted to save document changes. Choose Yes when prompted to save the global glossary and command changes.

At this point, a physical Normal.dot file is created in the Word for Windows version 2.0 directory.

REFERENCES
