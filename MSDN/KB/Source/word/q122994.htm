WD6X: Outline/Shadow Format Not Removed with Replace Command

 Last reviewed: February 5, 1998
Article ID: Q122994
The information in this article applies to:
  • Microsoft Word for Macintosh, version 6.0, 6.0.1, 6.0.1a

SYMPTOMS

When you try to replace Outline or Shadow character formatting, Microsoft Word reports that <x> replacements were made, but the formatting is not actually replaced.

RESOLUTION

When you replace Outline or Shadow formatting, use the Default Paragraph Font style instead of choosing Not Outline or Not Shadow as the Replace With formatting. To do this, follow these steps:

  1. On the Edit menu, click Replace.

  2. With your insertion point in the Find What box, click No Formatting to remove any existing formats.

  3. In the Format list, click Font.

  4. Click the Font tab in the Find Font dialog box and click to select the Shadow or Outline option under Effects. NOTE: An "x" (without the quotation marks) should appear in the selected option.

  5. Click OK.

  6. Move your insertion point to the Replace With box and click No Formatting to remove existing formats.

  7. In the Format list, click Style.

  8. In the Find Style dialog box, select Default Paragraph Font and click OK.

  9. Click Replace All.

  10. Click Close.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word versions listed above. This problem was corrected in Microsoft Word 98 Macintosh Edition.

REFERENCES

"Microsoft Word User's Guide," version 6.0, document number WB60460-0794, pages 61-63
