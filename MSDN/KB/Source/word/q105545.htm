Drawing Objects or Pleading Wizard Lines Do Not Display

 Last reviewed: July 30, 1997
Article ID: Q105545
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

A Microsoft Word 6.0 Picture drawing object, created or modified using the Word for Windows drawing tools, may not display on the screen, even if you are in page layout view and the Show Drawings option is selected. (To locate this option, choose Options from the Tools menu, and select the View tab.)

This symptom can also occur with the line numbers and attorney information in a document created with the Pleading Wizard.

CAUSE

Word does not display drawing objects if the Drawing Objects option is cleared. (To locate this option, choose Options from the Tools menu, and select the Print tab.) This happens because Word displays your document the way it will print, given the current printing options settings. If the Drawing Objects option is not selected, your drawing objects will not print, so Word does not display them. A blank space appears in your document in place of the drawing object.

This affects any objects you create using the tools on the Drawing toolbar, clip art that you modify with the drawing tools, and any other objects that you edit with the drawing tools.

This affects the Pleading Wizard because the line numbers and attorney information included by the wizard are drawing objects and not paragraph number formatting.

RESOLUTION

If you want to view drawing objects in your Word document, turn on the Drawing Objects option using the following instructions:

  1. From the View menu, choose Page Layout.

  2. From the Tools menu, choose Options.

  3. Select the Print tab.

  4. Under Include With Document, select Drawing Objects and then choose OK.

  5. Update your nondisplaying drawing objects. After you update them, they display correctly in Word.

    Note: To update your drawing object, double-click it to open it for editing. Add and then delete a line (or other drawing object), then choose the Close Picture button on the Pictures toolbar.
