DMM: Hang When Importing Outlook Folder and Network Unavailable

 Last reviewed: March 16, 1998
Article ID: Q182327
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows
  • Microsoft Outlook 97

SYMPTOMS

When you try to import an Outlook folder to use as a Microsoft Direct Mail Manager address list, the insertion point will change to an hourglass. You will not receive a Timeout error message.

This problem occurs when the network computer is unavailable.

CAUSE

Microsoft Direct Mail Manager hangs because it relinquishes control to Outlook and waits for Outlook to pass data back to it. Because Outlook hangs and cannot pass information, Microsoft Direct Mail Manager also hangs.

RESOLUTION

Work offline or wait until the network becomes available. Additionally, you should ensure that you are using a good network cable and that it is securely plugged in.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.

MORE INFORMATION

If the network does not become available and you use the hourglass to click the Cancel button, Microsoft Direct Mail Manager produces the following message: 

   An action cannot be completed because a component (Microsoft Outlook) is
   not responding. Choose 'Switch To' to activate the component and correct
   the problem.
When you click the Switch To button (and if you are using Microsoft Exchange Server), Microsoft Direct Mail Manager produces the following message: 

   Your Microsoft Exchange Server is unavailable.

   You can retry connecting to the server using the network, or you can
   work offline and connect using dial-up networking.

   Network problems are preventing connection to the Microsoft Exchange
   Server Computer. Contact your system administrator is this condition
   persists.
You are then offered a Retry button, a Work Offline button, and a Help button.

NOTE: If you click Retry, the message will be repeated.
