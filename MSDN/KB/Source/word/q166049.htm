WD97: Relative Links Don't Work in Word Viewer

 Last reviewed: February 4, 1998
Article ID: Q166049
The information in this article applies to:
  • Microsoft Word Viewer 97

SYMPTOMS

When you click to follow a relative hyperlink that was created with Internet Assistant, Microsoft Word Viewer 97 will be unable to parse the link information and open the document.

Instead, the URL Open dialog box from Internet Explorer will be displayed.

CAUSE

This problem occurs only when all of the following conditions are true:

  • The hyperlink was created in Internet Assistant, an Internet add-in that you can use to create Web pages in earlier versions of Microsoft Word (versions earlier than Word 97).
  • The link is a relative link. That is, rather than having a full path to the link, the location of the link is relative to the current directory.

WORKAROUND

Use the Internet Explorer URL Open dialog box that is displayed to navigate to the linked document.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
