WD98: How to Install Additional Fonts From the ValuPack Folder

 Last reviewed: February 17, 1998
Article ID: Q180509
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SUMMARY

This article describes the additional fonts that are available with the Microsoft Office 98 and Microsoft Word 98 compact discs in the ValuPack folder. These templates are located in the following folder on the Office 98 and Word 98 compact discs: 

   ValuPack:Fonts

MORE INFORMATION

When you install Microsoft Office 98, only certain fonts are installed. To install the fonts, you can either copy them from the compact disk to the System Folder:Fonts folder, or you can use the Value Pack Installer to add the Fonts Component.

To use the Value Pack Installer, double-click the icon labeled Value Pack Installer.
