Total Editing Time Reset to Zero When Document Saved

 Last reviewed: February 5, 1998
Article ID: Q155121
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0, 7.0a

SYMPTOMS

The Total Editing Time property for a document is reset to zero minutes when the document is saved for the first time. (To locate this option, click Properties on the File menu, and then select the Statistics tab.)

This problem occurs, for example, when you create a new document, edit it for an hour, and then save it. The Total Editing Time displays zero minutes instead of sixty minutes. In contrast, when you open an existing document and edit it for an hour, the total editing time increments to sixty minutes.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word versions 7.0. and 7.0a. This problem has been corected in Word 97 for Windows.

WORKAROUND

To work around this problem and track the editing time for a new document, save the file before you begin editing.
