Windows Resources Not Restored After Quitting Tutorial

 Last reviewed: February 5, 1998
Article ID: Q107673
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Windows operating system version 3.1

SYMPTOMS

The Examples And Demos command on the Help menu may consume Windows resources. After you exit the Examples and Demos tutorials, you may have fewer Windows resources than before you ran them.

If your overall Windows resource level becomes too low, you may experience low-memory problems in Word or other Windows-based applications.

Note: If you run the tutorials again in the same Windows session, no further Windows resources are lost. In other words, all the resource losses occur the first time you run the Examples And Demos tutorials.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

To restore your Windows resources, quit and restart Windows.
