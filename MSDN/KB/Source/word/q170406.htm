WD97: Allow Fast Save Option Not Selected by Default

 Last reviewed: September 8, 1997
Article ID: Q170406
The information in this article applies to:
  • Microsoft Word 97 for Windows, Service Release 1 (SR-1)

SYMPTOMS

In Microsoft Word 97 SR-1, the Allow FastSave option is turned off by default. This functionality is different from Microsoft Word 97.

MORE INFORMATION

The Allow Fast Saves option allows Word 97 to save documents quickly by appending changes to the end of a file on the hard disk instead of rewriting the entire file during a save operation. Since a Background Save option has been added to Word 97, the Allow Fast Saves option is no longer necessary because the Save operation occurs in the background.

To enable or disable the Allow Fast Saves option, follow these steps:

  1. On the Tools menu, click Options.

  2. Click the Save tab.

  3. Click to select or clear the Allow Fast Saves check box.

  4. Click OK.

The change you make to the Allow Fast Saves options will remain in effect until you change it again.
