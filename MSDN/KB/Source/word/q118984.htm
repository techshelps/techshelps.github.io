How to Add the Word 5.1 Font Menu to Word 6.0

 Last reviewed: February 5, 1998
Article ID: Q118984
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

Word 5.0 and 5.1 include a Font menu, but there is no Font menu in Word 6.0 for the Macintosh. The Word 5.x Font menu commands are located in the Font dialog box in Word 6.0 (to open this dialog box, click Font on the Format menu).

To add a Font menu to Word 6.0, follow these steps:

  1. Open the 5.1 Upgrade template, which is located in the Macros folder.

  2. From the View menu, choose Toolbars.

  3. From the Toolbars list, select the 5.1 Upgrade option, and then choose OK.

  4. Choose the Add Font Menu button on the 5.1 Upgrade toolbar.

    -or-

    Use the following steps to run the AddFontMenu macro:

    a. From the Tools menu, choose Macro.

    b. In the Macros Available In list, select 5.1 Upgrade (Template).

    c. In the Macro Name list, select AddFontMenu and then choose the 

          Run button.

  5. Choose Yes when Word displays the following message: 

          This will add a menu to the menu bar containing a list of all
      available fonts. Would you like to add this menu?

MORE INFORMATION

It may be necessary for you to add a Font menu to make Word 6.0 compatible with third-party menu utilities that cannot use the Font list on the Formatting toolbar.
