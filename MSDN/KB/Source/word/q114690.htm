Word Toolbars Are Not Confined to the Application Workspace

 Last reviewed: February 5, 1998
Article ID: Q114690
6.00 6.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

The toolbars in Word 6.0 for Windows are not confined to the Word application workspace. If you open Word in a restored window that does not fill the desktop, you can drag a toolbar outside Word's window and drop it on the desktop.

If you edit a Word object in place in a Microsoft Visual Basic application, Word's toolbars will appear in the Visual Basic application workspace.

Note: Word's menus will not appear in a Visual Basic 3.0 for Windows application workspace. This is a limitation of Visual Basic 3.0.

The toolbars in Microsoft Excel version 5.0 are implemented differently than Word and cannot be placed outside the Microsoft Excel application workspace.
