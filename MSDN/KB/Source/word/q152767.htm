IA: "Word Could Not Load This Add-In Program" or WordBasic Err

 Last reviewed: March 20, 1998
Article ID: Q152767
The information in this article applies to:
  • Microsoft Internet Assistant for Word, versions 2.0z
  • Microsoft Word for Windows 95, version 7.0, 7.0a

SYMPTOMS

When you attempt to access the Html.dot template by clicking New on the File menu, you receive the following error message: 

     WordBasic Err=124 Unknown Command, Subroutine or Function
In addition, when you click Templates on the File menu, the Wiahtm32.wll add-in is not loaded (there is no check mark in the check box). When you select the Wiahtm32.wll check box to load the add-in, you receive the following error message: 

   Microsoft Word: Word could not load this add-in program.

CAUSE

The WordBasic error occurs because the Wiahtm32.wll is not being loaded to Word for Windows. The .wll is necessary for the template to function. The .wll may not be loaded if the Inloader.dll or the Wininet.dll is missing or is damaged (these files are located in the Windows\System folder).

RESOLUTION

To resolve this problem, use the following steps:

  1. Rename the Wininet.dll and the Inloader.dll files in the Windows\System folder.

  2. Search for duplicate Wininet.dll and Inloader.dll on the system and rename these duplicate files.

  3. Reinstall Internet Assistant.

If the Inloader.dll file is not present on you computer after you complete steps 1-3, you will need to manually extract the file from the Ia1.cab. To do this, use the following steps to get access to the .cab file contained in the self-extracting WRDIA20Z.EXE:

  1. Run the IA Setup program.

    Immediately after the Initializing Setup screen you will be prompted for the installation type, minimize the Setup program and search for the Ia1.cab file. Make a copy of the IA1.CAB file.

  2. Switch to the running IA Setup and click Exit Setup.

  3. Use the Extract.exe command to extract the Inloader.dll file from the copy of the IA1.cab file.

    For example, to extract the Inloader.dll file from the Ia1.cab file on a disk in drive A to the Windows\System folder on drive C, use the following command: 

          extract a:\ia1.cab inloader.dll /l c:\windows\system

MORE INFORMATION

The Wininet.dll file is 104 kb (106,864 bytes) The Inloader.dll file is 78 kb (79,872 bytes)
