Open and Send Mail Commands Cause Error or Start QuickMail

 Last reviewed: August 4, 1997
Article ID: Q114757
5.00 5.10 MACINTOSH kb3rdparty

The information in this article applies to:

  • Microsoft Word for the Macintosh, versions 5.0, 5.1

SYMPTOMS

When you choose Open Mail or Send Mail from the File menu, Word generates an error message or opens CE QuickMail instead of Microsoft Mail.

CAUSE

These problems can occur if both QuickMail and Microsoft Mail extensions are installed in the System Folder.

WORKAROUND

To start Microsoft Mail using the Open Mail and Send Mail commands, deactivate QuickMail by removing its extensions from the System Folder.

MORE INFORMATION

The Open Mail and Send Mail commands in Microsoft Word allow you to open and send mail messages with Word document enclosures. These commands are designed specifically for use with Microsoft Mail for the Macintosh.
