Context-Sensitive Help (F1) Topic Does Not Exist for View Tab

 Last reviewed: February 5, 1998
Article ID: Q110627
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

If you try to obtain context-sensitive help for the View tab by pressing F1, you receive the following error message: 

   Help Topic Does Not Exist
This problem occurs in Page Layout view only; it does not occur in Normal view. (To access the View tab, choose Options from the Tools menu.)

STATUS

Microsoft has confirmed this to be a problem in Word for Windows version 6.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

This problem does not occur in Word version 6.0 for the Macintosh.

WORKAROUND

To work around this problem:

  1. From the View menu, choose Normal.

  2. From the Tools menu, choose Options.

  3. Select the View tab and press F1.
