WD: Some Document Properties Don't Print

 Last reviewed: July 31, 1997
Article ID: Q155947
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

 When you click Summary Info on the Print tab in the Options dialog box,
and then print a document, some summary document properties do not print. For example, the following properties do not print: 

   Manager
   Company
   Category
   Hyperlink Base (Microsoft Word 97 only)
NOTE: To locate the Print tab, click Options on the Tools menu.

WORKAROUND

To work around this problem, add an extra page to the document and then use the following steps:

  1. On the Insert menu, click Field.

  2. Click Document Information in the Categories list.

  3. Click DocProperty in the Field Names list.

  4. Click Options.

  5. Click the document property you want to print and click OK.

  6. Click OK.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
