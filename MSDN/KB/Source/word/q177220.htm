WD98: Embed TrueType Checked but Unavailable Saving As RTF

 Last reviewed: February 17, 1998
Article ID: Q177220
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you save a document as Rich Text Format (RTF) and you click Options in the File Save, or File Save As dialog box, the Embed TrueType fonts check box may be selected (checked) but is unavailable (appears dimmed).

WORKAROUND

Click to select or clear the Embed TrueType fonts check box before attempting to save your document.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
