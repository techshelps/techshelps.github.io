Word: Choosing Save As Command Causes Crash

 Last reviewed: July 30, 1997
Article ID: Q80154
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.0

SUMMARY

Microsoft Word for the Macintosh version 5.0 crashes when you choose the Save As command from the File menu when the Rebound 0.95 or the SFVol 1.5 freeware INITs are installed in the System Folder.

The Rebound INIT is compatible with Word version 4.0; however, it is not compatible with Word 5.0.

SFVol INIT 1.5, created in July 1988 by Raymond Lau, causes the same problem.

MORE INFORMATION

Rebound 0.95 is a freeware INIT available on many online services. It enhances the standard File (Open and Save) dialog boxes. When Rebound is used, the file list scrolls to and highlights the last opened file or folder in each directory. Rebound was created by Fred D. Reed.
