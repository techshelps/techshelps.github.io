No Revision Marks When Adding and Deleting Line Numbers

 Last reviewed: February 6, 1998
Article ID: Q80494
The information in this article applies to:
  • Microsoft Word for Windows versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word for Windows version 2.0, revision marks are not added when you delete or add line numbers even though the Mark Revisions check box is selected in the Tools Revision Marks dialog box.

MORE INFORMATION

The revision marks feature in Word for Windows marks only changes made to the text within the document, not changes made to the formatting of that text. Because the line numbers feature, accessed through the Format Section Layout Line Numbers dialog box, is a section formatting feature rather than a text formatting feature, changes in line number formatting are not marked as revisions.

Reference(s):

"Microsoft Word for Windows User's Guide," version 2.0, page 605-607
