WD: Character Styles Not Displayed in Style Area

 Last reviewed: February 5, 1998
Article ID: Q161615
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

Character style names do not appear in the Style Area (normal view).

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

There is no workaround to allow the Style Area to show character styles.

MORE INFORMATION

The Style Area is an area to the left of the text area that shows or hides applied styles. This option is available only when you are using normal view. To turn this option on or off, do the following:

  1. On the View menu, click Normal.

  2. On the Tools menu, click Options, and then choose the View tab.

  3. Set the Style Area Width measurement. For example, type 1" in the Style Area Width box to show the style area or type 0 (zero) in Style Area Width box to hide the style area. Click OK.
