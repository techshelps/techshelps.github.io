DDE Execute Instruction Returns Fail Code

 Last reviewed: October 10, 1997
Article ID: Q98348
The information in this article applies to:
  • Microsoft Word for OS/2, versions 1.1, 1.1a, 1.1b

SUMMARY

You will obtain different results when you send a DDEEXECUTE instruction to Word for OS/2 from another Presentation Manager application, depending upon which version of Word you are running.

Word for OS/2 Version 1.1

Word version 1.1 does not respond to any Execute instruction sent to it through a DDE channel. Word returns an error result code to the calling application.

Word for OS/2 Version 1.1a

Microsoft corrected Word version 1.1a so it responds to and executes any instructions sent to it through a DDE channel. However, whether the Execute instruction was successful or not, Word returns an error result code to the calling application.

Word for OS/2 Version 1.1b

Microsoft corrected Word version 1.1b so it not only responds correctly to Execute instructions sent to it through a DDE channel, but it also return a relevant result code depending upon whether the instruction was executed successfully or not.

To determine which version of Word for OS/2 you have, look up the date of the PMWORDU.DLL file on the following table: 

 PMWORDU.DLL            Word
 File Date              Version
 -----------            -------

 12/7/90                1.1

 3/29/91 or 4/1/91      1.1a

 4/1/92                 1.1b
