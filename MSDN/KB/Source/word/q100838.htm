Error Msg: General Protection Fault in Module S3645K.DRV

 Last reviewed: October 8, 1997
Article ID: Q100838
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c
  • Microsoft Excel for Windows, versions 4.0, 5.0

SYMPTOMS

The following error message may occur while you edit a Word for Windows document if you are using the Dell S3, 640 x 480 or 800 x 600, 256-color (512K) video driver version 1.2a or 1.3a (S3645K.DRV): 

   Word caused a General Protection Fault in Module S3645K.DRV at
   <Memory Address>

RESOLUTION

There is an updated version of the video driver that corrects this problem. Version 2.1 of the Dell video Driver is available from the Microsoft Download Service (MSDL), which you can reach by calling (425) 936-6735. This service is available 24 hours a day, 7 days a week. The highest download speed available is 14,400 bits per second (bps). For more information about using the MSDL, call (800) 936-4100 and follow the prompts. To obtain S3-801.EXE, download S3-801.EXE. S3-801.EXE is a compressed, self-extracting file. After you download S3-801.EXE, run it to extract the file(s) it contains.

WORKAROUND

Use the standard VGA video driver supplied with Windows instead.

MORE INFORMATION

Dell Computer Systems installs the S3645K.DRV video driver on some of its computer systems. For more information, contact Dell Computer Systems at the following phone numbers: 

   Dell Technical Support: (800) 624-9896
   Dell Bulletin Board Service (BBS):  (512) 728-8528
