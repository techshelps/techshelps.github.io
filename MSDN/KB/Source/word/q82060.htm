Documentation Updates

User's Guide (102) Additional LaserWriter Print Options:

None of these print options are available for the StyleWriter.

Getting Started (11) Hardware and Software Requirements:

To change the memory allocation for Word, do the following:

  1. If Word is running, choose Quit from the File menu.

  2. In the Finder, select the Microsoft Word icon.

  3. From the File menu in the Finder, choose Get Info.

  4. Change the current memory size in the lower right corner.

  5. Close the Get Info dialog.

User's Guide (117) ImageWriter Page Setup Options:

Fractional Widths is ON by default for PostScript printers. The Fractional Widths option does not appear in the Page Setup dialog when an ImageWriter is selected. To use the Fractional Width option with an ImageWriter, add this command to the File menu by using the following steps:

  1. From the Tools menu, choose Commands.

  2. In the commands list, select Fractional Widths.

  3. Under Menu, choose the Add button.

  4. Choose the Close button to close the dialog box.

For non-PostScript printers other than the ImageWriter, you can turn Fractional Widths ON by default by using the following steps:

  1. From the File menu, choose Page Setup.

  2. Select the Fractional Widths option.

  3. Select the Use As Default option.

  4. Choose OK to close the dialog box.

User's Guide (247) Finding and Replacing Special Characters:

To find or replace text with a nonbreaking hyphen, type ^~ in the Find What or Replace With text box.

User's Guide (269) To Remove Words From a Custom Dictionary:

To remove words from a custom dictionary, the dictionary must be unlocked. You cannot remove words from a locked dictionary. Note also that opening a large dictionary takes a long time.

User's Guide (595) To Define a New Field for Use in The Main Document:

If you define a new field for use in the main document and later add a field with the same name to the data document, Word displays the error message "Duplicate field name in data document" when you go back to the main document. This error message is only a reminder that this field was already defined, and you can continue working on your main document. If you used this field name in an ASK or SET instruction, Word displays an error message when you merge the document.

User's Guide (600) Using Merge Instructions:

Word inserts the path to the data document in the DATA instruction. If Word cannot find the data document in this location, Word asks you to locate the data document.

User's Guide (673) Publishing Part of a Word Document:

If there is no space on your system disk, you can still run Word, but you cannot create editions. Word displays the error message "Can't create edition."

User's Guide (684) Embedding Objects:

Word displays the error message "Unable to embed the object class" if there is not enough memory to run the embedding application. In this situation, you may be able to decrease the amount of memory given to Word in order to run both applications.
