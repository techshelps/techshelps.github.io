WD: Repeat Picture Opens Insert Picture Dialog Box

 Last reviewed: August 4, 1997
Article ID: Q118317
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word 97 for Windows

SYMPTOMS

The Repeat Picture command on the Edit menu (which appears after you insert a picture) opens the Insert Picture dialog box rather than inserting a second copy of the previously inserted picture.

STATUS

Microsoft has confirmed this to be a problem in the versions of Word listed above. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUNDS

Use one of the following methods to insert a copy of the original picture.

Method 1: Select the desired picture from the Insert Picture dialog box, 

          which appears when you choose Repeat Picture from the Edit menu.
Method 2: Highlight the picture, choose Copy from the Edit menu, position 
          the insertion point where you want a copy to appear, and choose
          Paste from the Edit menu.
