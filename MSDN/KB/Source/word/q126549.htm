Can't Use Go Back Commands in FTP Session

 Last reviewed: February 5, 1998
Article ID: Q126549
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

After loading an FTP Uniform Resource Locator (URL) and jumping to one of the folders or documents, you cannot go back to the place from which you jumped by using the Go Back button on the Formatting Toolbar or the Go Back command on the Window menu.

CAUSE

The Go Back commands take you to the previous document window instead of the previous hyperlink.

RESOLUTION

Click the right mouse button over the object to access the shortcut menu. Choose Back To:ftp://<hostname>.

MORE INFORMATION

While in an FTP object, you can use the right mouse button to access the following additional functions: 

   Copy URL to Clipboard
   Info
   Save As
   Reload
