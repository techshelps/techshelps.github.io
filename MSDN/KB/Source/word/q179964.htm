DMM: Spool32 Caused a GP Fault in Module Hpfmlc08.dll

 Last reviewed: March 4, 1998
Article ID: Q179964
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows

SYMPTOMS

When you test print an envelope from Microsoft Direct Mail Manager, you may receive the following message: 

   This program has performed an illegal operation and will be shut down.
   If the problem persists, contact the program vendor.
Clicking the Details button shows: 

   SPOOL32 caused a general protection fault in module HpfmlC08.dll.

RESOLUTION

Try printing a different size envelope, such as Business, #10, or try printing to another printer.

MORE INFORMATION

The file Hpfmlc08.dll is used by the following Hewlett Packard printers: 

   HP DeskJet 670C
   HP DeskJet 690C
