Error When Pasting SPSS Graph in Word

 Last reviewed: July 31, 1997
Article ID: Q145641
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When you try to use the Paste Special command to paste a graph created in SPSS statistical program, you may receive the following error message: 

   No Topic Specified.

CAUSE

The program that created the graph sends the information to the clipboard along with the image. Word depends on the creating program to place the information on the clipboard in a format that Word can interpret. If Word cannot interpret the information correctly, you may receive the error message described in the "Symptoms" section of this article.

WORKAROUND

When you click Paste Special on the Edit menu, select Picture instead of SPSS Chart Object.
