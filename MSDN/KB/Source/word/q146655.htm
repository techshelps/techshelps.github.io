WordBasic Error 24 with Modified Word 2.x Proposal Template

 Last reviewed: February 5, 1998
Article ID: Q146655
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0, 7.0a

SYMPTOMS

When you create a new document based on the Proposal.dot template, you may receive the following error message: 

     WordBasic Err=24 "Bad Parameter"

CAUSE

The Proposal.dot template that shipped with Microsoft Word version 2.0 was not designed to work with Microsoft Word version 7.0.

MORE INFORMATION

Microsoft modified the popular templates from Microsoft Word 2.x to function correctly in Microsoft Word for Windows 6.0. You can use these modified templates in Microsoft Word for Windows 95, version 7.0, also. All the modified Word 2.x templates function correctly in Word 7.0, except for the Proposal.dot template. When you try to use the Proposal.dot template, you may or may not receive the "Proposal Information" dialog box, but the error message, "WordBasic Err=24 Bad Parameter" is consistent with this type of error.
