WinWord: Field to Insert Previous Month Based on Current Date

 Last reviewed: February 5, 1998
Article ID: Q80507
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0

To insert the previous month's date in a Microsoft Word for Windows document, use the following field: 

   {QUOTE {IF {DATE \@ M} = 1 12 {={DATE \@ M} - 1}}/1/92 \@ MMMM}

"Microsoft Word for Windows User's Guide," version 2.0, pages 646, 752-753, 759-762
