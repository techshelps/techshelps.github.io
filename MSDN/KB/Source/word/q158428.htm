IA: GP Fault Opening HTML File w/ Graphics After Installing Word

 Last reviewed: January 30, 1998
Article ID: Q158428
The information in this article applies to:
  • Microsoft Internet Assistant, version 2.03z
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

If you are using Internet Assistant version 2.03z, you may receive a general protection (GP) fault in Winword.exe when you open an HTML document that contains a GIF or JPEG image.

CAUSE

This problem occurs after you install Microsoft Word 97, Microsoft Publisher 97, or Microsoft Office 97. These programs install shared graphics filters that conflict with the Inloader.dll file installed by Internet Assistant. The Inloader.dll file handles image import processes.

RESOLUTION

To work around this problem, use any of the following methods:

Method 1: Use Word 97 instead of Internet Assistant

Word 97 offers improved features and additional HTML capability that is not available in Internet Assistant.

Method 2: Upgrade to Internet Assistant version 2.04z

If you want to use Internet Assistant for Word for Windows 95 or Word for Windows NT, upgrade to Internet Assistant version 2.04z.

To upgrade to Internet Assistant 2.04z, do either of the following:

  • If you installed Word 97 or Office 97 from a compact disc, you can install the upgrade for Internet Assistant version 2.04z. Open the Valupack folder on the compact disc and double-click Wrdia204z.exe, a self-extracting, self-executing file.
  • Download the latest version from the following Web site: 

          http://www.microsoft.com/word/internet/ia/sysreq.htm

    NOTE: If you run Wrdia204z.exe on a computer with Word 97 installed but not Word for Windows 95, versions 7.x, or Word for Windows NT, Setup will fail and will display a dialog box titled "Unable to install Internet Assistant for Microsoft Word." Click Exit to finish Internet Assistant Setup.

MORE INFORMATION

For more information, see the Readme.doc file that is extracted when you double-click the Wrdia204z.exe file.
