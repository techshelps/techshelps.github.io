WD: How Word Borders Convert to Word 5.x for the Mac

 Last reviewed: February 5, 1998
Article ID: Q137608
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SUMMARY

This article discusses how borders are converted between Microsoft Word for the Macintosh and Microsoft Word for Windows.

MORE INFORMATION

Borders created with Word versions 6.0 or later are converted to the following single, thick, or double line borders when your document is saved in Word 5.x format. 

                  Word 6.0     Word 6.0     Word 7.0      Word 97
                    for          for          for           for
Border            Macintosh    Windows      Windows       Windows
------            ---------    --------     --------      -------
3/4 pt            Single       Single       Single        Single
1 1/2 pt          Thick        Thick        Thick         Thick
2 1/4 pt          Thick        Thick        Thick         Thick
3 pt              Thick        Thick        Thick         Thick
4 1/2 pt          Thick        Thick        Thick         Thick
6 pt              Thick        Thick        Thick         Thick
Double 3/4 pt     Double       Double       Double        Double
Double 1 1/2 pt   Double       Double       Double        Double
Double 2 1/4 pt   Double       Double       Double        Double
Hairline          Single
Dotted 3/4 pt     Single       Single       Single        Single
Dashed 3/4 pt     Single       Single       Single        Single
Borders created in Word 6.x for the Macintosh, Word 6.x for Windows, and Word 7.x for Windows are converted to the following line styles when you convert the document to Word 5.x using the Word 6.x for Windows & Macintosh import converter. 

                  Word 6.x     Word 6.x     Word 7.x
                    for          for          for
Border            Macintosh    Windows      Windows
------            ---------    --------     --------
3/4 pt            Single       Single       Single
1 1/2 pt          Single       Single       Single
2 1/4 pt          Single       Single       Single
3 pt              Thick        Thick        Thick
4 1/2 pt          Thick        Thick        Thick
6 pt              Thick        Thick        Thick
Double 3/4 pt     Double       Double       Double
Double 1 1/2 pt   Double       Double       Double
Double 2 1/4 pt   Double       Double       Double
Hairline          Single
Dotted 3/4 pt     Single       Single       Single
Dashed 3/4 pt     Single       Single       Single
Borders created in Word 5.x for the Macintosh are converted to the following line styles when you import them into the following versions of Word: 

Word 6.x     Word 6.x         Word 7.x          Word 97
  for          for              for               for
Macintosh    Windows          Windows           Windows
---------    --------         --------          -------
Dotted       Dotted 3/4 pt    Dotted 3/4 pt     Dotted 3/4 pt
Hairline     1/4 pt           1/4 pt            1/4 pt
Single       3/4 pt           3/4 pt            3/4 pt
Thick        1 1/2 pt         1 1/2 pt          1 1/2 pt
Double       Double 3/4 pt    Double 3/4 pt     Double 3/4 pt
