Can't Print Document Containing Inline Picture (NT 4.0)

 Last reviewed: February 5, 1998
Article ID: Q163548
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Office 97 for Windows
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

When you print a Word 97 document that contains a picture to a Hewlett Packard (HP) LaserJet 4 or 4MP printer, the following problems may occur:

  • If you print to a network printer, the network server may hang.

    -or-

  • If you printing to a local printer, the local computer may hang.

CAUSE

This problem occurs when the document contains an inline picture (the "Float over text" option has been disabled).

RESOLUTION

To resolve this problem, install Windows NT, version 4.0, Service Pack 2.

For more information on obtaining and installing the Microsoft Windows NT 4.0 Service Pack 2, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q152734
   TITLE     : How to Obtain Windows NT Version 4.0 U.S. Service Pack
To temporarily work around this problem, select the Float Over Text option before you insert the picture.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Windows NT, version 4.0. This problem was corrected in Microsoft Windows NT, version 4.0, Service Pack 2.
