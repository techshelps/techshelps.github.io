WD: Cannot Format TIME or EDITTIME to Display Seconds

 Last reviewed: February 2, 1998
Article ID: Q49459
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

There are two fields in Word for Windows that return a time-related result.

EDITTIME

This field returns the number of minutes the current document has been open for editing since it was created.

TIME

This field returns the current time of day.

You cannot format the result of a field to include seconds or partial seconds using a Word for Windows date-time picture switch. The smallest unit of time that Word for Windows can return or display is minutes.

For additional information, please see the following article in the Microsoft Knowledge Base:
