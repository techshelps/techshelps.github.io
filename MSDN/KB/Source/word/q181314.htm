WD97: Some Underline Styles Are Lost During Save to HTML

 Last reviewed: February 19, 1998
Article ID: Q181314
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you save a Word 98 Macintosh Edition file in HyperText Markup Language (HTML) format, the following underline styles will be lost: 

   Dotted
   Thick
   Dash
   Dot Dash
   Dot Dot Dash
   Wave
Word changes the Single, Words Only, and Double underline styles to a standard underline.

CAUSE

This behavior is by design in Word 98. When you are editing an HTML document, you can only select standard underline in the Font dialog box. No special effects are available.

NOTE: If your default save format is HTML, you have to close and reopen the document to view the HTML appearance of the document. Clicking Save will save the document as HTML, but the document on the screen will not be refreshed until the document is closed and re-opened.

MORE INFORMATION

For additional information about this problem in Microsoft Word 98 for Macintosh Edition, please see the following article in the Microsoft Knowledge Base:
