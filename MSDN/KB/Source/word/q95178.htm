Problems with the Hewlett-Packard PostScript Cartridge

 Last reviewed: July 30, 1997
Article ID: Q95178
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

The Hewlett-Packard (HP) PostScript Emulation cartridge for HP LaserJet printers is not supported by Microsoft Windows operating system versions 3.0, 3.0a, and 3.1, nor has it been tested with these versions of Windows.

Below are descriptions of problems you may experience when you print to the HP LaserJet IIID using the PostScript printer driver version 3.5-3.55 and the HP IIID description file (HP_3D522.WPD) version 2.1.

To work around these problems, install the HP III PostScript printer driver (HPIII522.WPD). Note: If you use HP III driver, you not be able to use the bin selection and envelope feeder options of the HP IIID.

MORE INFORMATION

Fonts

Fonts will print in mirrored image (meaning you would have to hold the paper up to a mirror to read the text) in portrait orientation. Fonts will also print in reverse landscape and in mirrored image in landscape orientation.

Envelopes

The HP PostScript cartridge does not recognize the envelope feeder code sent by the HP IIID description file and thus pulls the envelope from the default upper tray. The text of the envelope will also print in mirrored image.
