"Syntax Error" When You Drop Document on Printer under NT 4.0

 Last reviewed: February 5, 1998
Article ID: Q155504
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Windows NT, version 4.0

SYMPTOMS

The following error message appears when you drag a Word document to the printer: 

   Wordbasic err = 100: Syntax error

CAUSE

Microsoft Word 7.0 for Windows 95 implementation of the OLE Drag and Drop feature is not fully compatible with Microsoft Windows NT 4.0 WIN32 OLE components.

STATUS

Drag and Drop printing does not work with Microsoft Word 7.0 for Windows 95 when installed under Microsoft Windows NT 4.0.

WORKAROUND

Print the file from Word by clicking the Print button.
