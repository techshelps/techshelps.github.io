WD98: Some Word 5.1 Commands Missing from Word 5.1 Menus View

 Last reviewed: February 17, 1998
Article ID: Q177388
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you use the Word 5.1 menus (On the View menu, click Word 5.1 menus), some commands that are available in Microsoft Word 5.1 are missing.

CAUSE

This is by design.

MORE INFORMATION

The following table lists the Word 5.1 menu commands missing from various menus when you use the Word 5.1 view. 

   Word 5.1 Menus View     Where to Find the Command in Word 98
   -------------------     ------------------------------------

   File menu
      Find File command    1. On the File menu, click Open.

                           2. Click Find File.

   View Menu
      Print Merge Helper   1. Exit Word 5.1 view.

                           2. On the Tools menu, click Mail Merge.

      Play Movie           1. Press control and click the movie you
                              inserted into the document.

                           2. On the shortcut menu, click Show Movie
                              Toolbar.

                           3. On the Movie toolbar, click Play Movie.

   Insert Menu
      Address              The insert address feature is found in the
                           following locations:

                            - On the Envelopes And Labels dialog box.

                            - On several of the dialog boxes of the wizards
                              for creating forms and letters.

   Format menu
      Revert To Style      No equivalent.

   Insert
      Field                1. Exit Word 5.1 menu view.

                           2. On the Insert menu, click Field.
