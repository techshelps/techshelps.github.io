WD: WordBasic Examples: DrawNudge

SUMMARY

This article contains an example that demonstrates the use of the following WordBasic statements or functions: 

   DrawNudgeDown          DrawNudgeDownPixel
   DrawNudgeLeft          DrawNudgeLeftPixel
   DrawNudgeRight         DrawNudgeRightPixel
   DrawNudgeUp            DrawNudgeUpPixel
This article supplements the information in online Help. To open this Help topic in Word for Windows, choose Contents from the Help menu and then choose the "Programming with Microsoft Word" topic. In Word for the Macintosh, click the Help icon, select Microsoft Word Help and choose "Programming with Microsoft Word."

Syntax

DrawNudgeDown DrawNudgeDownPixel DrawNudgeLeft DrawNudgeLeftPixel DrawNudgeRight DrawNudgeRightPixel DrawNudgeUp DrawNudgeUpPixel

Example

The following macro will draw an ellipse, then use DrawNudge commands to move it around the page: 

   Sub MAIN
      FileNewDefault
      DrawEllipse
      FormatDrawingObject .FillColor = 12, .FillPattern = "1"
      For OutCount = 1 To 8
         For Incount = 1 To 50
            Select Case OutCount
               Case 1
                  DrawNudgeDown
               Case 2
                  DrawNudgeUp
               Case 3
                  DrawNudgeRight
               Case 4
                  DrawNudgeLeft
               Case 5
                  DrawNudgeDownPixel
               Case 6
                  DrawNudgeUpPixel
               Case 7
                  DrawNudgeRightPixel
               Case 8
                  DrawNudgeLeftPixel
            End Select
         Next Incount
      Next OutCount
   End Sub
