Envelopes Print in Wrong Orientation with Some DeskJets

 Last reviewed: February 5, 1998
Article ID: Q119294
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SYMPTOMS

If you print envelopes from Word for Windows to a Hewlett-Packard (HP) DeskJet 520, 540, 550c, 560 or 560c printer using the DESKJETC.DRV (version 3.1 or later) printer driver, the envelopes may print in the wrong orientation (that is, they may print in portrait instead of landscape orientation).

STATUS

Microsoft has confirmed this to be a problem in Word for Windows versions 2.x 6.0, 6.0a, and 6.0c. This problem was corrected in Word version 7.0 for Windows 95.

RESOLUTION

Word 6.0

To get the envelopes to print in the correct (landscape) orientation, adjust the default envelope printing options by following these steps:

  1. From the Tools menu, choose Envelopes And Labels.

  2. Select the Envelopes tab.

  3. Choose the Options button.

  4. Select the Printing tab.

  5. Choose the envelope feed icon showing the envelope feeding face-down, landscape, from the right. (It is the third icon from the right.)

  6. Choose the OK button to close the dialog box.

The envelope should now print correctly, in landscape orientation.

Word 2.x

To get the envelopes to print in the correct (landscape) orientation, set a printer heuristic of +512 by following these steps:

  1. From the Tools menu, choose Options.

  2. From the Category list, select Win.ini.

  3. Click once on any of the lines listed under Startup Options. It will appear in the Options and Settings boxes below.

  4. Select the line that appears in the Option box, type "deskjetc" (without the quotation marks).

  5. Highlight the line that appears in the Setting box, type "+512" (without the quotation marks).

  6. Choose the Set button. This will add the line "deskjetc=+512" (without the quotation marks) as the last line in the Startup Options list.

  7. Choose CLOSE.

The envelopes should now correctly print in landscape orientation.
