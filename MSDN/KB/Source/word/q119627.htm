"Word Cannot Edit the AutoSketch" Error in Word 6.0

 Last reviewed: August 5, 1997
Article ID: Q119627
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a
  • Microsoft Windows operating system, version 3.1

SYMPTOMS

In Word, when you try to edit an embedded AutoSketch object a second time, you may receive the following error message: 

   Word cannot edit the AutoSketch.
This error does not occur when using Word 2.x for Windows.

CAUSE

Although AutoSketch was designed as an OLE 2.0 application, this functionality is not working correctly.

STATUS

Autodesk Inc. has confirmed this to be a problem in OLE 2.0 functionality in AutoSketch version 2.0 for Windows. Customers should contact Autodesk for more information or support issues.

WORKAROUNDS

Method 1: Insert the object as a bitmap using the Copy and Paste commands.

Method 2: Edit the object before embedding it into Word.
