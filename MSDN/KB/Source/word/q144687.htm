Error Message: Not Enough Memory to Run Word Viewer

 Last reviewed: February 6, 1998
Article ID: Q144687
The information in this article applies to:
  • Microsoft Word Viewer, version 6.0

SYMPTOMS

When you start Word Viewer version 6.0 you receive the following error message: 

   Not enough memory to run Word Viewer.

CAUSE

Your system is using outdated OLE .dll files. Word Viewer requires version 2.01, 2.02, or 2.03 of the following files: 

   Compobj.dll
   Ole2.dll
   Ole2.reg
   Ole2conv.dll
   Ole2disp.dll
   Ole2nls.dll
   Ole2prox.dll
   Stdole.tlb
   Storage.dll
   Typelib.dll

RESOLUTION

Update your system to OLE version 2.03. You can obtain the OLE version 2.03 Application Note from a variety of sources. See the "More Information" section of this article for information about how to obtain this Application Note.

MORE INFORMATION

Microsoft has an Application Note available titled "WW1116: OLE Version 2.03" that contains an updated version of OLE for use with Microsoft Windows for Workgroups and Microsoft Windows versions 3.1 and 3.11. OLE 2.03 includes several improvements that affect other Microsoft programs, such as Microsoft Word for Windows and Microsoft Access.

You can obtain this Application Note from the following sources:

  • Microsoft's World Wide Web Site on the Internet
  • The Internet (Microsoft anonymous ftp server)
  • The Microsoft Network (MSN)
  • Microsoft Download Service (MSDL)
  • Microsoft Product Support Services

For more information about obtaining this Application Note, please see the following article in the Microsoft Knowledge Base:
