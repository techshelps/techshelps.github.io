How to Use "It's Legal for Windows" with Word

 Last reviewed: February 5, 1998
Article ID: Q118634
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a

SUMMARY

"It's Legal for Windows" helps you prepare up to 41 legal documents, customized for each state. "It's Legal" offers user-customizable document templates with blank spaces for completion by clients, tenants, and so on. The program quickly guides you through fill-in-the-blank document preparation and incorporates an extensive context-sensitive, plain English legal guide.

MORE INFORMATION

Features include: 

   Health Care Power of Attorney
   Living Trust
   Premarital Agreement
   Personal Fact Sheet
   Home Purchase Worksheet
   Business Planning Worksheet
   Memorial Service Worksheet
"It's Legal" presents data entry fields in context to show how user-added information fits into the document. Context-sensitive plain English help is constantly visible in the lower portion of the screen offering explanations in lay terms. Additional help includes on-line help and a legal guide.

System Requirements

  • An IBM or 100-percent-compatible personal computer w/ hard disk drive
  • Microsoft Windows version 3.1 or later
  • A Windows-supported printer

Installation

  1. From Windows Program Manager, choose Run from the File menu.

  2. Insert the "It's Legal" Setup Disk 1.

  3. In the command line, type "a:setup" (without the quotation marks) (or "b:setup" if you put the disk in drive B) and choose the OK button.

Note: The Setup program places all the system files in a directory on your hard disk called ILWIN unless you choose otherwise. Hard disk space requirement depends upon the number of documents you install. (You will need approximately 9 to 10 MB free if you install all of the documents.)

Running the Program

Enter your personal information (such as name, address, and social security number) so that the program can automatically insert this data in the appropriate fields as you create your documents.

Using "It's Legal" with Word for Windows

"It's Legal" does not integrate directly with Word; however, there are two ways that you can use "It's Legal" information in Word.

Method 1: Save your "It's Legal" document as an ASCII or Text-Only document 

          and open it in Word.
Method 2: Copy and paste information from "It's Legal" into Word.

These methods allow you to use existing Word documents or create new Word documents and transfer legal-related information from "It's Legal" to Word for Windows.

Uninstalling "It's Legal"

To uninstall the program, simply delete the ILWIN program directory from Windows File Manager.
