Characters (~)(`)(')(") and (^) Don't Display When Key Pressed

 Last reviewed: February 6, 1998
Article ID: Q95068
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0, 7.0a

SYMPTOMS

When you press the apostrophe (') key, the quote (") key, the tilde (~) key, the (`) key or the caret (^) key in Word for Windows, the character does not appear on the screen until you press another key.

CAUSE

The default keyboard layout is US-International.

WORKAROUND

Change the keyboard layout setting to US or English US. In Windows 3.x, use the International option in Control Panel. In Windows 95, use the Regional Settings in Control Panel. Change the keyboard layout or input locales setting to English US setting.
