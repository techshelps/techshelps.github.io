Multiple Instances of Access Open When Inserting Database

 Last reviewed: February 6, 1998
Article ID: Q105699
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Access for Windows, version 1.0

SYMPTOMS

When you insert a Microsoft Access version 1.0 database in Word for Windows, multiple instances of Access may be opened. The problem occurs when you insert the Access database using the Database command on the Insert menu or when you use the DDE MailMerge Get Data command.

This does not occur when you use Access version 1.1.

RESOLUTION

Upgrade Access to version 1.1.
