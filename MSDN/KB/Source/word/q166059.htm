WD97: Documents Open in Word Instead of Word Viewer on NT 3.51

 Last reviewed: February 4, 1998
Article ID: Q166059
The information in this article applies to:
  • Microsoft Word Viewer 97

SYMPTOMS

If you are running Microsoft Word Viewer 97 and Microsoft Word 6.0x on Windows NT 3.51, double-clicking a document will always start Word 6.0x instead of Microsoft Word Viewer 97, even if you have selected the option to open documents with Word Viewer.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The problem does not occur if you are running versions of Windows NT that are later than 3.51 or versions of Microsoft Word that are later than 6.0x.

NOTE: If you select the option to open documents with Word Viewer, .rtf (Rich Text Format) files will be opened in Word Viewer.

REFERENCES

To obtain help with Microsoft Word Viewer 97, please see the following resources:

  • Microsoft's World Wide Web Site on the Internet: 

         http://www.microsoft.com/word/internet/viewer/default.htm
  • Refer to the accompanying Readme.doc file for product information and support options.
  • Microsoft offers Word Viewer support on the standard Microsoft Word for Windows Technical Support phone line, at (425) 462-9673.
