Manual Page Breaks Ignored When Document Is Printed

 Last reviewed: July 30, 1997
Article ID: Q26264
The information in this article applies to:
  • Microsoft Word for the Macintosh versions 3.0, 3.01, 3.02, 4.0, 5.0, 5.1, 6.0

SYMPTOMS

In Microsoft Word for the Macintosh, a hidden manual page break, or section break which starts a new page, may not show in Print Preview and may be ignored when the document is printed.

In Word versions 5.x and earlier, a manual page break may be formatted inadvertently as hidden if it is inserted before an Index (.i.) or Table of Contents (.c.) marker in the text.

CAUSE

Word Versions 5.0, 5.1, 6.0

In these versions of Word, a manual page break formatted as hidden text will be ignored in Print Preview even when the Hidden Text option is activated in the view options. To set this option, choose Options from the Tools menu, and click the View tab (in Word 5.x, choose Tools/Preferences and click the View category).

However, if the Hidden Text check box is activated in the Print options (Tools/Options, Format/Document in Word 5.x) then the hidden page break will appear in print preview and will print. To set this option, choose Options from the Tools menu, and click the Print tab (in Word 5.x, choose Format/Document).

WORKAROUND

Workaround for Versions 5.0, 5.1, 6.0

  1. From the Edit menu, choose Replace.

  2. In the Find What box, type "^m", without the quotation marks (in Word 5.x, type "^d").

  3. From the Format menu, choose Character, select Hidden under Style, then choose OK.

  4. In the Replace With box type "^m", without the quotation marks (in Word 5.x, type "^d").

  5. From the Format menu, choose Character, and click twice on Hidden until the check box is white, not shaded.

  6. Choose the Replace All button.

Word Versions 4.0 and 3.x

In these versions of Word, manual page breaks are ignored when printed if they are formatted as hidden text and Print Hidden Text is cleared (turned off) in the Print dialog box.

Unlike Word 5.x, in these versions of Word, manual page breaks appear in page preview (print preview and page view in Word 4.0) and print if they are formatted as hidden text and the Show Hidden Text option is activated in the Preferences dialog box (Edit menu).
