WD:TAB/Arrow Keys Don't Select Options in Mac Help Search Dialog

 Last reviewed: February 5, 1998
Article ID: Q121444
The information in this article applies to:
  • Microsoft Word for Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

In the Word Help Search dialog box, using the TAB key does not change the focus. In other words, using the TAB key does not move you from the search text box to the list of topics, or to the buttons. The arrow keys do not select individual items in the list of topics.

WORKAROUND

Use the mouse to scroll through the list of topics, to highlight your choice and to activate a button. It is not possible at this time to navigate the Search dialog box with the keyboard.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
