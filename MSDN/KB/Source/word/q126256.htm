Windows Resources Are Not Restored After Leaving Word

 Last reviewed: August 5, 1997
Article ID: Q126256
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Excel for Windows, version 5.0

SYMPTOMS

After you quit Word or Microsoft Excel, Windows system resources do not return to the previous amount. Available system resources are 6 to 8 percent less than the original amount. (To determine the percentage of system resources available, choose About Program Manager from the Help menu in Program Manager. System resources will be shown on the last line of the dialog box.)

CAUSE

Word and Microsoft Excel will load shared system components, such as the OLE2 DLLs, if they are not already in memory and MAY not release them from memory when you exit. This behavior is by design and is intended to increase performance and to comply with established standards concerning an application's use of shared Windows components.

STATUS

The reduction in system resources occurs by product design.
