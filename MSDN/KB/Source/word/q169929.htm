WD97: Captions Update to Zero If Grouped with Objects

 Last reviewed: July 31, 1997
Article ID: Q169929
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you update a caption that has been grouped with an object, the caption may update to zero.

CAUSE

This problem occurs when objects that float over text are grouped with captions. The captions may not update correctly.

WORKAROUND

To work around this problem, follow these steps:

  1. Right-click the object that contains a caption.

  2. On the shortcut menu, point to Grouping, and then click Ungroup.

  3. Repeat steps 1-2 for each object that contains a caption.

  4. To update the caption fields, use any of the following methods:

    a. On the File menu, click Print Preview, and then click Close. 

          -or-

    b. Click to select a caption and then press F9 to update the caption. 

          Repeat this step for each of the caption fields.


          -or-

    c. Right-click a caption, and then click Update Field on the shortcut 

          menu. Repeat this step for each of the caption fields.

  5. Regroup each of your objects with their captions.

MORE INFORMATION

This problem also occurs when you insert the caption into a text box, group the object with the text box, and then update the fields.

For more information about how to group, ungroup, and regroup objects, click the Office Assistant, type "How do you group objects?" (without the quotation marks), click Search, and then click to view the "Group, ungroup, and regroup drawing objects" topic.
