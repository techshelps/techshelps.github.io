Drag-and-Drop Editing With CTRL Key Copies Selected Items

 Last reviewed: February 6, 1998
Article ID: Q78593
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word For Windows version 2.0, pressing the CTRL key while you perform a drag-and-drop edit allows you to copy the selected text or graphic to the drop point.

MORE INFORMATION

Performing a drag-and-drop edit without the CTRL key allows you to move the selected items to the drop point. If you select the desired information and hold down the CTRL key, you can copy the selected text or graphic to the drop point rather than moving it.

Reference(s):

"Microsoft Word for Windows User's Guide," version 2.0, pages 111-2 "Microsoft Word for Windows Jump Start," inside front cover, column 2
