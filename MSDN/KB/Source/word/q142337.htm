Word 95 Hangs When Canceling Schedule+ Login Without Exchange

 Last reviewed: February 5, 1998
Article ID: Q142337
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you cancel the Schedule+ logon from the Envelopes And Labels dialog box, Word hangs (stops responding).

When you cancel the Schedule+ logon from the Mail Merge Helper dialog box, you get the following error message: 

   Cannot start Schedule

CAUSE

You do not have Microsoft Exchange installed.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

RESOLUTION

Install Microsoft Exchange.
