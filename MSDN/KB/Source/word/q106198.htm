WordBasic Examples: InsertSectionBreak/PageBreak/ColumnBreak

 Last reviewed: July 30, 1997
Article ID: Q106198
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0

SUMMARY

This article contains an example that demonstrates the use of the following WordBasic statements or functions: 

   InsertSectionBreak

   InsertPageBreak

   InsertColumnBreak

InsertSectionBreak

Syntax

InsertSectionBreak

Example

Sub MAIN
FileNewDefault ViewPage For count = 1 To 50 
  Insert "This is a test."
  InsertPara
Next count StartOfDocument LineDown 15 InsertSectionBreak LineDown 15 InsertSectionBreak LineUp 5 WindowArrangeAll FilePageSetup .TopMargin = "1" + Chr$(34), .BottomMargin = "1" + Chr$(34), .LeftMargin = "2.75" + Chr$(34), .RightMargin = "2.75" + Chr$(34) 
End Sub

InsertPageBreak

Syntax

InsertPageBreak

Example

Sub MAIN
FileNewDefault For count = 1 To 5 
  Insert "This is a test of page breaks."
  InsertPageBreak
Next count Insert "This text should be on page 6" 
End Sub

InsertColumnBreak

Syntax

InsertColumnBreak

Example

Sub MAIN
FileNewDefault ViewPage FormatColumns .Columns = "2" For count = 1 To 50 
  Insert "This is a test."
  InsertPara
Next count StartOfDocument LineDown 25 InsertColumnBreak 
End Sub
