Introduction

The Word 5.0a Patcher included on the enclosed MW0554 disk is for use with version 5.0 of Microsoft Word for the Macintosh, and is not needed with version 5.1 as the problem has been fixed. This program corrects a problem with Word 5.0 that occurs when you exchange files between different Macintosh computers. This problem results in fonts being changed to the default font.

The Word 5.0a Patcher corrects the Word program and identifies your version as Word 5.0a rather than Word 5.0.

Running the Patcher

  1. If you are running Word, choose Quit from the File menu.

  2. Insert the MW0554 disk in the disk drive.

  3. Copy the file called Word 5.0a Patcher to the folder on your hard drive that contains the Microsoft Word version 5.0 program.

    NOTE: If you are running Gatekeeper, you must disable it before running the patcher. To do this, choose Control Panels from the Apple menu, double-click the Gatekeeper pane, select On/Off, and click the Off switch.

  4. Open the Word 5.0 folder and double-click the Word 5.0a Patcher icon.

  5. Read the information displayed and choose the OK button.

  6. From the list, select Microsoft Word and then choose the Open button. The patcher will run and correct the error in Word. The patcher won't affect any other portions of the Word program.
