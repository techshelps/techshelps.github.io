WD: Word Compatibility Options Described

 Last reviewed: March 4, 1998
Article ID: Q166016
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SUMMARY

This article describes the Compatibility Options that are available in Microsoft Word.

NOTE: To change the Compatibility Options, click Options on the Tools menu, and then click the Compatibility tab.

MORE INFORMATION

Font Substitution

Displays fonts used in the active document that are not available on your system by using fonts that are available on your system.

Recommended Options For

Selects the word-processing program you want to set options for. Selected options affect only the display of the document wile you work with it in Word. The document is not changed permanently.

Options

Selects the options you want to appear in the active document. Selected options affect only the display of the document while you work with it in Word. The document is not changed permanently.
