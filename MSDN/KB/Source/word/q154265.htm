General Information About SGML Author for Word

 Last reviewed: November 17, 1997
Article ID: Q154265
The information in this article applies to:
  • Microsoft SGML Author for Word, version 1.2
  • Microsoft Word for Windows 95, version 7.x
  • Microsoft Word 97 for Windows

SUMMARY

This article contains general information regarding the Microsoft SGML Author add-on designed for Word for Windows 95 and Word 97 for Windows.

MORE INFORMATION

SGML Author is an easy-to-use add-on tool and converter for Word for Windows. Users no longer need to spend hours manually tagging documents because SGML Author tags them automatically. And, administrators get flexible point-and-click tools that make configuration easy.

The SGML Author user model is divided between administrators and end users.

Administrators can use SGML Author to configure the SGML converter to any Document Type Definition (DTD). Configuring SGML Author using innovative point-and-click management tools enables administrators to quickly build associations between templates and predefined styles and structures in the DTD.

Users can create documents in Word using administrator-defined styles and templates for all formatting. To create SGML, just save the document as SGML from within Word. Thanks to the unique validation engine in SGML Author, users have a friendly feedback loop using the built-in Annotation feature in Word to identify and correct DTD compliance errors.

Additional features and capabilities include:

  • Pre-assembled association files and templates for CALS (MIL-M-28001b) and ATASB-100 DTDs
  • Convert to and from the CALS table model--the leading standard for representing tables as SGML.
  • Convert to and from the ISO/IEC TR9573-11:1992(E) standard for representing equations with SGML.
  • Import and export enhanced CGM and WMF graphics, as well as OLE objects.
  • Import GRP IV fax standard raster images.
  • Take advantage of native support for ISO character entity sets.
  • Incremental updates to Word Templates are supported in the administration tool.

SGML Author is available directly from Microsoft and through a number of Well-established channels, including third-party integrators, ISVs, and Solution Providers.
