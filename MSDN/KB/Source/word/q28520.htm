WD: "Text Replacing This Argument Too Long" in MacWord Merge

 Last reviewed: November 17, 1997
Article ID: Q28520

The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 3.0, 3.01, 3.02, 4.0, 5.0, 5.1

SUMMARY

In Microsoft Word, when you run a print merge and your main document contains an IF statement, one of the following error messages may appear: 

   Version of Word      Error message
   ----------------------------------

   5.1a                 The Text Replacing This Argument Is Too Long
   5.1                  Text Inside Merge Quotes Is Too Long
   5.0                  Text Inside Merge Quotes Is Too Long
   4.0                  The Text Replacing This Argument Is Too Long
   3.x                  The Text Replacing This Argument Is Too Long

CAUSE

If you get one of the error messages above, it means that the field entry that you are testing in your IF statement is too long.

NOTE: The maximum length for the text of any one field (in your data file) is 253 characters.
