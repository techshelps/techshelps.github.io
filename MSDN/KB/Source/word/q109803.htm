FormatBulletDefault, FormatBulletDefault()

Syntax:

FormatBulletDefault [Add] FormatBulletDefault()

Example:

This sample macro creates a document with some sample list items. It then bullets them using the FormatBulletDefault statement. 

   Sub MAIN
      FileNewDefault
      Insert "This is Item One."
      InsertPara
      Insert "This is Item Two."
      InsertPara
      Insert "This is Item Three."
      InsertPara
      MsgBox "Choose OK to bullet the items."
      EditSelectAll
      FormatBulletDefault
      StartOfDocument
      MsgBox "Choose OK to remove the bullets"
      EditSelectAll
      FormatBulletDefault
      StartOfDocument
   End Sub
