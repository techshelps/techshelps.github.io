GP Fault Using Word with Random House Webster's Dictionary

 Last reviewed: August 4, 1997
Article ID: Q114203
6.00 6.00a WINDOWS kb3rdparty

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

WordPerfect Technical Support has confirmed that you may receive a general protection (GP) fault in Word for Windows when you try to use Random House Webster's Dictionary version 1.0 for Windows.

CAUSE

The GP fault occurs because Word's status bar conflicts with Random House Webster's Dictionary.

STATUS

WordPerfect is working on an update to correct this conflict. For more information on this problem, please contact WordPerfect Technical Support at (800) 228-1029.

RESOLUTION

To avoid the GP fault, turn off Word's status bar, as follows:

  1. From the Tools menu, choose Options.

  2. Select the View tab.

  3. Clear the Status Bar check box and choose OK.

Random House Webster's Dictionary for Windows is manufactured by a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
