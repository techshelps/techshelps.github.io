Winword 2.x and 6.0: Printing Envelopes on the IBM 4019

 Last reviewed: February 5, 1998
Article ID: Q98899
The information in this article applies to:
  • Microsoft Word for Windows versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Windows operating system version 3.1, 3.11
  • Microsoft Windows for Workgroups, version 3.1, 3.11

SUMMARY

This article explains how to print envelopes on the IBM 4019 LaserPrinter from Word for Windows. This procedure requires that you use version 4.11 or later of the IBM 4019 printer driver (IBM4019.DRV), available from Lexmark.

To obtain version 4.11 or later of IBM4019.DRV, contact Lexmark technical support at (606) 232-3000 or download the driver from the Lexmark Bulletin Board (BBS) at (606) 232-5238. Support of this printer driver is the sole responsibility of Lexmark. IBM4019.DRV is manufactured by Lexmark, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

QUICK LIST OF ISSUES AND WORKAROUNDS

  1. In Word 6.0, the default position indicated by the envelope feed icon is incorrect.

    Workaround: Turn on the clockwise rotation check box so that the envelope feed method suggested by Word matches the feed method indicated by the printer.

  2. Envelopes of size 6 3/4 do not print.

    Cause: This is a physical limitation of the printer.

  3. The envelope pulls from the manual feed instead of from the envelope feeder.

    Workaround: In Word 6.0, make sure Envelope is selected as the paper source in the Printing Options dialog box. (To open the Printing Options dialog box, choose Envelopes And Labels from the Tools menu, choose the Options button, and then select the Printing Options tab.)

    In Word 2.x, select the "Printer's Envelope Feeder has been Installed" option in the Print Options dialog box. (To open the Print Options dialog box, choose Options from the Tools menu, and then select the Print icon under Category.)

    The envelope feeder on the printer must be configured to match the size of the envelopes being printed. For more information on how to configure the envelope feeder, see later sections in this article.

MORE INFORMATION

In the instructions below, when you are instructed to feed an envelope, insert the envelope in the manual feed slot face up with the top of the envelope on the left. Slide the envelope in the slot until the envelope cannot advance any farther. Envelopes print in landscape orientation on this printer.

USING THE MANUAL FEED ENVELOPE SLOT

If you do not have an optional envelope feeder for your IBM 4019 printer, you must feed envelopes through the manual feed slot on the back of the printer.

  1. Install version 4.11 of the IBM 4019 printer driver. Make sure the printer is in "native (4019)" mode. For information on switching to native mode, please see the following article in the Microsoft Knowledge Base:

    ARTICLE-ID: Q102441 

       TITLE     : Switching Emulation Modes on the IBM 4019 and 4029 Printers

  2. Start Word and create a new document based on the NORMAL template.

  3. In Word 6.0, switch to Windows Control Panel, choose the Printers icon, and choose the Setup button.

    In Word 2.x, choose Print Setup from the File menu. Select the IBM 4019 LaserPrinter in the Printer list and then choose the Setup button.

  4. Select the following settings in the Setup dialog box, choose OK, and then return to your Word document:

    a. Select the Use Printer Patterns option.

    b. Clear the Fast Printing Directly To Port option.

    c. Clear the TrueTypes As Graphics option.

    d. Clear all Cartridge or Optional Font settings.

    e. Select 300 x 300 in the Graphics Resolution list.

    f. Select Upper Tray in the Paper Source list.

    g. Select 8 1/2 x 11 in Portrait in the Paper Size list.

  5. Skip this step for Word 6.0. In Word 2.x, choose Options from the Tools menu and then select the Print icon under Category. Clear the "Printer's Envelope Feeder has been Installed" option and choose OK.

  6. Skip this step for Word 2.x. In Word 6.0, choose Envelopes And Labels from the Tools menu and select the Envelopes tab. Choose the Options button, and then select the Printing Options tab. In the Feed From list, select Manual Feed, and then choose OK.

  7. From the Tools menu, choose Envelopes And Labels (choose Create Envelope in Word 2.x or choose the Envelope button on the toolbar).

  8. Type the address information and then choose the Print Envelope button.

    -or-

    a. Type the address information and choose the Add To Document button.

    b. From the File menu, choose Print.

    c. Select the Pages option button and type "0-0" (zero, without the 

          quotation marks) in the Pages box. (In Word 2.x, select the Pages
      option button and type "0" in the From and To boxes.)

    d. Choose OK.

  9. The printer will go off-line and request manual feed. (Status 3 displays on the front panel of the printer). Feed an envelope into the manual feed slot on the back of the printer.

USING THE OPTIONAL ENVELOPE FEEDER

  1. Install and configure the IBM 4019 printer driver using steps 1 through 4 of the "Using the Manual Feed Envelope Slot" section above.

  2. Configure the envelope tray to match the envelope size you want to print using the control panel on the front of the printer. The following table lists the printer settings for each standard envelope size: 

          Printer       Control
      Envelope      Panel Setting
      --------      -------------

      Size 10       Paper Size 3
      Size 9        Paper Size 2
      Size 7 3/4    Paper Size 1
      DL            Paper Size 4
      C5            Paper Size 5
      B5            Paper Size 6

   For instructions on setting the envelope size on the printer, see the
   "How to Configure the IBM 4019 Envelope Feeder" section at the end of
   this article.

  3. Skip this step for Word 6.0. In Word 2.x, choose Options from the Tools menu and then select the Print icon under Category. Select the "Printer's Envelope Feeder has been Installed" option and choose OK.

  4. Skip this step for Word 2.x. In Word 6.0, choose Envelopes And Labels from the Tools menu and select the Envelopes tab. Choose the Options button, and then select the Printing Options tab. In the Feed From list, select Envelope, and then choose OK.

  5. From the Tools menu, choose Envelopes and Labels (choose Create Envelope in Word 2.x or choose the Envelope button on the toolbar).

  6. Type the address information and then choose the Print Envelope button.

    -or-

    a. Type the address information and choose the Add To Document button.

    b. From the File menu, choose Print.

    c. Select the Pages option button and type "0-0" (zero, without the 

          quotation marks) in the Pages box. (In Word 2.x, select the Pages
      option button and type "0" in the From and To boxes.)

    d. Choose OK.

    The envelope feeds from the envelope feeder.

PRINTING CUSTOM ENVELOPE SIZES USING THE ENVELOPE FEEDER

  1. Configure the envelope feeder for the next largest envelope size available using the front control panel. See the note at the end of this article for instructions on how to configure the envelope feeder.

  2. From the Tools menu, choose Envelopes And Labels (choose Create Envelope in Word 2.x or choose the Envelope button on the toolbar).

  3. In Word 6.0, select the Envelope tab and then choose the Options button. In the Envelope Size list, select the size that matches the envelope size setting on your printer. Choose OK.

    In Word 2.x, in the Envelope Size list, select the size that matches the envelope size setting on the printer.

  4. Type the address information and then choose the Add To Document button.

  5. In Word 6.0, choose Page Setup from the File menu, and then select the Margins tab.

    In Word 2.x, choose Page Setup from the Format menu, and then select the Margins option.

  6. Adjust (increase) the margin settings so the page size matches the dimensions of your envelope.

  7. From the File menu, choose Print. Select the Pages option button and type "0-0" (zero, without the quotation marks) in the Pages box. (In Word 2.x, select the Pages option button and type "0" in the From and To boxes.) Choose OK.

    The printer pulls the envelope from the envelope feeder.

MAIL MERGING ENVELOPES USING THE OPTIONAL ENVELOPE FEEDER

Note: In Word 2.x, "mail merge" is called "print merge."

  1. Install and configure the IBM 4019 printer driver using steps 1 through 4 of the "Using the Manual Feed Envelope Slot" section above.

  2. Open your mail merge main document.

  3. Skip this step for Word 6.0. In Word 2.x, choose Options from the Tools menu and then select the Print icon under Category. Select the "Printer's Envelope Feeder has been Installed" option and choose OK.

  4. Skip this step for Word 2.x. In Word 6.0, choose Envelopes And Labels from the Tools menu and select the Envelopes tab. Choose the Options button, and then select the Printing Options tab. In the Feed From list, select Envelope, and then choose OK.

  5. From the Tools menu, choose Envelopes And Labels (choose Create Envelope in Word 2.x or choose the Envelope button on the toolbar).

  6. Type all the address information except the MERGEFIELD fields and then choose the Add To Document button.

  7. Insert the MERGEFIELD fields in the envelope.

  8. Merge the document to the printer.

  9. In the Print dialog box, select the Pages option button and type "0-0" (zero, without the quotation marks) in the Pages box. (In Word 2.x, select the Pages option button and type "0" in the From and To boxes.)

  10. Choose OK.

POTENTIAL PROBLEMS WITH ENVELOPES

  • Paper Jams ----------

    Size 6 3/4 envelopes (3 5/8 x 6 1/2 inches) do not print correctly. The envelope address and return address text print approximately 3 inches too far to the right, which causes the envelope address to be cut off on the right side. After the envelope prints, the printer displays a paper jam error message (Status 2 on the front panel). This error is caused by a physical paper-handling limitation of the printer.

  • Envelope Pulls from Manual Feed Instead of Envelope Feeder ----------------------------------------------------------

    The "Printer's Envelope Feeder has been Installed" option must be selected in the Print Options dialog box, and the envelope feeder must be configured to match the size of the envelopes being print merged.

    If either of these conditions are not met, the envelope will pull from the manual feed slot.

HOW TO CONFIGURE THE IBM 4019 ENVELOPE FEEDER

  1. If the Ready light is on, press and release the Start/Stop button.

  2. Press and release the Paper button until Envelope is selected.

  3. While holding the Code button down, press and release the Paper button until a number is displayed that corresponds to the envelope size you want (see the above section for references to envelope sizes).

  4. Press the Start/Stop button to switch the printer on-line.
