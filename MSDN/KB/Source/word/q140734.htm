Properties Missing from Password-Protected Document

 Last reviewed: February 5, 1998
Article ID: Q140734
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

The summary and custom properties for your document are not retained.

CAUSE

The summary and custom properties were deleted when you saved the document with password protection.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows 95, version 7.0. This problem has been corrected in Microsoft Word 97 for Windows.

WORKAROUND

To retain the custom and summary properties, do not use Word's password protection.
