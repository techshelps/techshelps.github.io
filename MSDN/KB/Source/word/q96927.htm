How to Install WinFax Pro Version 3.0 Macros in WinWord

 Last reviewed: July 30, 1997
Article ID: Q96927
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c
  • Microsoft Windows operating system version 3.1

Summary

WinFax Pro version 3.0 supplies a WinFax macro that you can install in Word for Windows. The instructions supplied in the "WinFax Pro User's Guide" for installing this macro in Word are incomplete. This article provides the complete procedure for installing the WinFax macro.

WinFax Pro is a Windows-based application manufactured by Delrina Technology that can send and receive faxes. Winfax Pro can communicate with any Windows-based application that supports DDE (dynamic data exchange).

MORE INFORMATION

Use the following instructions to install the WinFax version 3.0 macro in Word for Windows:

  1. In Word for Windows, open NEWMACRO.DOC, located on the Word program directory (c:\WINWORD by default).

  2. From the Macro Name list, select InstallMacro and then choose the Install button.

  3. In the next dialog box, select NORMAL from the Template list. If you want to assign the InstallMacro macro to a Word menu, select a name from the Menu list.

  4. Choose the OK button.

  5. After Word installs the macro, choose the Cancel button to close the dialog box.

  6. Close NEWMACRO.DOC.

  7. Open WINFAX.DOT (see your WinFax Pro documentation for the location of this file).

  8. From the Tools menu, choose Macro.

  9. Select the Global Commands option. From the Macro Name list, select InstallMacro and choose the Run button.

  10. The InstallMacro macro displays a list of all the macros in the WINFAX.DOT template. From the Macro to Install list, select the WinFax macro. From the Template list, select NORMAL. If you want to assign the WinFax macro to a Word menu, select a name from the Menu list (Delrina recommends that you assign the WinFax macro to the File menu). Choose the Copy button.

  11. From the File menu, choose Save All. Save your documents as appropriate,, and choose the yes button when Word prompts you to save the global command and glossary changes.
