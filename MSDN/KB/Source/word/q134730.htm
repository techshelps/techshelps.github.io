WD: File Format for Word 7.x Is the Same as Word 6.x

 Last reviewed: February 5, 1998
Article ID: Q134730
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft Word for Windows NT, version 6.0

SUMMARY

The file format of Word for Windows 95, version 7.x, is the same file format as all versions of Microsoft Word 6.x. This includes Word 6.x for Windows NT and the Macintosh. There is no conversion necessary for opening or saving the file.

When you create files in Word 7.x using any of the following features, these features are ignored if you open the document in Word 6.x:

  • Highlight
  • Save Preview Picture in the Properties dialog box
  • Contents and Custom tabs in the Properties dialog box

(To locate the Properties dialog box, click Properties on the File menu. The Highlight button is on the Formatting toolbar.)
