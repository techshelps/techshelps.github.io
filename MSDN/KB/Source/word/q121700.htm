Word Hangs Inserting Caption with Revision Marking Turned On

 Last reviewed: February 5, 1998
Article ID: Q121700
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

An hourglass pointer may appear and persist, and Word may appear to hang (stop responding) when you insert a caption if revision marks are turned on. (To insert a caption, choose Caption from the Insert menu. To turn on revision marks, choose Revisions from the Tools menu and select the Mark Revisions While Editing check box.)

If you press CTRL+ALT+DEL after the hourglass pointer appears, Windows displays the following error message: 

   WINWORD.EXE
   This Windows application has stopped responding to the system.
NOTE: If you press ESC after the hourglass pointer appears (instead of pressing CTRL+ALT+DEL), Word sometimes endlessly updates the caption number.

CAUSE

This problem occurs if you select the Include Chapter Number option in the Caption Numbering dialog box to add a caption to an existing series of captions. For example, if your document already contains Figure 3-1 and Figure 3-2, the problem occurs if you insert Figure 3-3 with revision marking turned on. (To open the Caption Numbering dialog box, choose Caption from the Insert menu and choose the Numbering button.)

By contrast, the problem does not occur if you insert the first caption in a series with revision marking turned on; for example, Word does not hang when you insert Drawing 1-1.

STATUS

Microsoft has confirmed this to be a problem in the versions of Word listed above. This problem was corrected in Microsoft Word 97 for Windows.
