WD97: CC: Address Gone After Using Address Book in Letter Wizard

 Last reviewed: January 15, 1998
Article ID: Q179340
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word 97 for Windows, Service Release 1 (SR-1)

SYMPTOMS

When you use the Letter Wizard (on the Tools menu, click Letter Wizard), the first address you added to the address book will be missing after you close the Address Book dialog box, if you performed these steps:

  • On the Other Elements tab click Address Book, add an address for a CC: (Carbon Copy) recipient and then click OK to return to the Other Elements tab.

    -and-

  • Click Address Book again to add an address for another CC: recipient.

WORKAROUND

To work around this problem when you use the Address Book, add all addresses for the CC: recipients at one time.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
