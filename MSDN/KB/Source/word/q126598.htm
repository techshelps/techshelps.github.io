SGML: GP Fault in WINWORD.EXE at 0074:1C0A

 Last reviewed: August 5, 1997
Article ID: Q126598
The information in this article applies to:
  • Microsoft SGML Author for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

You may receive a general protection (GP) fault in WINWORD.EXE at 0074:1C0A when you do the following in the order shown:

  1. Start Windows and start SGML Author.

  2. Load any .map file.

  3. Minimize SGML Author.

  4. Start Word.

  5. Open a document.

  6. Press ALT+TAB to switch to Program Manager.

  7. Start File Manager and minimize it.

  8. Switch to Word and print the document.

  9. After the print is complete, import an .sgm file.

WORKAROUND

To work around this problem, perform the steps in an order that is different from the order listed in the "Symptoms" section of this article.

STATUS

Microsoft has confirmed this to be a problem in SGML Author version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
