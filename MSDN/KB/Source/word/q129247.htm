Can't Install Foreign Language Proofing Tools on Word for NT

 Last reviewed: August 5, 1997
Article ID: Q129247
The information in this article applies to:
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

You may receive the following error message when you attempt to install Alki Software's Foreign Language Proofing Tools on a computer that has Word for NT installed 

   Microsoft Word 6.0 for Windows must be installed before the proofing
   tools can be installed.
followed by another error message: 

   Setup was canceled before it could completely install <Language
   version> proofing tools. If there are problems, correct them and run
   Setup again.
These error messages only appear on systems running Windows NT and Word for Windows NT (32-bit Word). If Word for Windows (16-bit Word) is also installed, the error messages will not appear; however, files will be installed in directories appropriate only for Word for Windows, and Word for Windows NT will not be able to access them.

CAUSE

Alki Software's Foreign Language Proofing Tools Setup program looks for the header of the 16-bit WINWORD.EXE file. The Setup program cannot read the header of the 32-bit WINWORD.EXE file, and gives the error messages listed above.

STATUS

Alki software is aware of this problem.

WORKAROUND

Contact Alki for instructions. Currently, Alki is suggesting two workarounds: 1) expand the compressed files and copy them to the correct locations, or 2) run Setup on a computer on which Word for Windows (16-bit Word) is installed and then copy the decompressed files to the Word for Windows NT computer. Contact Alki for complete information: 

   Alki Software Technical Support:

   Phone (206) 286-2780
   Fax   (206) 286-2785

MORE INFORMATION

The following proofing tools will not be installable with Word for NT because they do not exist:

Spelling

Finnish <<other Eastern European Spell checkers TBA>>

Thesaurus

Portuguese (Brazilian) Portugueses Finnish Catalan

Grammar

All languages except US English
