WD6: SGML: Unable to Open Import Debug File: IMPDBG.RTF

 Last reviewed: March 6, 1998
Article ID: Q125501
The information in this article applies to:
  • Microsoft SGML Author for Word, version 1.0

SYMPTOMS

You receive the following error message when converting a Word document to SGML, but the converter will save the document correctly. 

   Unable to Open Import Debug File: IMPDBG.RTF
   Import Debug File Not Written.

CAUSE

This behavior occurs when the Association (*.MAP) file is located in a read- only directory and the GenDebugInfo variable in the MSTXTCNV.INI equals "yes."

RESOLUTION

Using a text editor, such as Notepad, edit the MSTXTCNV.INI file (located in the Windows directory). In the [SGML Author] section, remove the following line 

   GenDebugInfo=yes
and make sure the map file is located in a directory with full read/write privileges.
