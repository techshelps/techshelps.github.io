List of Items Installed with Custom Installation in 5.0

 Last reviewed: July 30, 1997
Article ID: Q86890
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.0

SUMMARY

In version 5.0 of Word, it is possible to perform a custom installation. To perform a custom installation, choose the Customize button in the Easy Install dialog box and select the desired options. The following article lists each option available in the Custom Install dialog box.

Package Title

  • Placement Files or Resources - Where placed (assuming a folder named "Word 5.0" on a disk named 

         "Hard Disk")
    - Other notes of interest

  • Embedding Preferences Resources - Hard Disk: System Folder: Preferences 

          Installed only under System 7.0

      Word User dictionaries is found and saved.
