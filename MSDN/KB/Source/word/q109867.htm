Envelope Prints Incorrectly on Panasonic KX-P4455 V.51.4

 Last reviewed: February 5, 1998
Article ID: Q109867
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When printing an envelope from Word 6.0 for Windows to a Panasonic KX- P4455 V.51.4 printer, the envelope may print incorrectly. The delivery or return address may print in the wrong position, or the return address may print directly above the delivery address.

RESOLUTION

Install the HP LaserJet IID PostScript printer driver and change the envelope feed method from center feed to right feed to match the envelope feed method of the Panasonic printer by following these steps:

  1. Install the HP LaserJet IID PostScript printer driver in the Windows Control Panel and make it the default printer.

  2. Start Word 6.0.

  3. From the Tools menu, choose Envelopes And Labels.

  4. Choose the Options button.

  5. Select the Printing Options tab.

  6. Under Feed Method, select the right feed icon.

  7. Under Feed From, select Envelope.

  8. Choose OK to return to the previous dialog box.

  9. Choose the Print button to print the envelope.

The envelope should now print correctly.

STATUS

Microsoft is researching this problem and will post new information here as it becomes available.

MORE INFORMATION

Setting printer heuristics by using the SetPrintFlags macro changes the way the envelope prints, but no combination of heuristics allows you to correctly print the envelope using the Panasonic printer driver.

The HP IID PostScript driver, because of the dual-bin setup of the HP IID, most closely resembles the dual-bin setup of the Panasonic printer.
