WD97: Error or Unexpected Quit When You Type Complex Sentence

 Last reviewed: February 5, 1998
Article ID: Q179310
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

Under extremely rare circumstances, when you type a long, complex sentence, Word may unexpectedly quit and you will not be prompted to save any open documents, or you may receive one of the following error messages: 

   Winword caused an Invalid Page Fault in module MSGREN32.dll at <address>

   -or-

   There was an unexpected error in the grammar checker. Word cannot
   complete the grammar check.

CAUSE

This problem occurs when the grammar checker attempts to evaluate a long, complex sentence that includes several conjunctions such as "or" and "and" along with at least one preposition such as "of," "to," "by," or "from."

NOTE: The chance of this error occurring is less than one in a million; so, it is extremely unlikely that you will encounter this error.

RESOLUTION

Microsoft has made available an updated version of the grammar checker. To resolve this problem, obtain this updated grammar checker from the Microsoft Software Library. The following file is available for download from the Microsoft Software Library: 

 ~ Gramupd.exe (size: 644320 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services

Installing the Updated Grammar Checker

To install the updated grammar checker, follow these steps:

  1. Quit Word. If you are using Word as your e-mail editor, quit your e-mail program as well.

  2. Locate and rename your current Msgren32.dll file. Write down the location of the file.

    To locate and rename the file, follow these steps:

    a. To locate the file, click the Windows Start menu, point to Find, and 

          click Files Or Folders.

    b. Right-click the Msgren32.dll file and rename it. For example, name 

          the file Msgren32.old.

    If you are running a standalone version of Word or Office, the default location for this file is: 

          C:\Program Files\Common Files\Microsoft Shared\Proof

    If you are running a network installation of Word, contact your network administrator for information about the location of the file.

  3. Download the Gramupd.exe file and save it to a temporary folder on your hard disk. Double-click the file to extract its contents.

  4. When prompted with a message asking for the location where you want to place the extracted files, type the default "C:\Program Files\Common Files\Microsoft Shared\Proof" or type the location you noted in step 2.

    The Msgren32.dll file will be copied to the location you specify.

When you restart Word, the new grammar checker will be installed.

The properties of the updated version of the file are listed in the following table. 

   File name        file size                  last modify date
   -------------------------------------------------------------

   Msgren32.dll     1,519 kilobytes (KB)       01/13/98  11:29a

Working Around This Problem If You Are Unable to Obtain the Updated DLL

To temporarily work around this problem, disable the grammar checker in Word. To do this, follow these steps:

  1. On the Tools menu, click Options, and select the Spelling & Grammar tab.

  2. Click to clear the Check Grammar As You Type and Check Grammar With Spelling check boxes.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
