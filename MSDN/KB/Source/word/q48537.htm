WD: Placing Footnotes on the Last Page of the Document

 Last reviewed: November 17, 1997
Article ID: Q48537
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 4.0, 5.0, 5.1

SUMMARY

To have footnotes print alone on the last page of a document, use one of the methods described in the "More Information" section of this article.

MORE INFORMATION

Method 1

  1. From the Format menu, choose Document. From the Footnote Position list, select End Of Document.

  2. Go to the end of the document and enter a section break (COMMAND+ENTER) just below the last section of text.

  3. With the cursor below the section break, choose Section from the Format menu. From the Start list, select New Page.

Method 2

  1. Choose Document from the Format menu and from the Footnote Position list, select End Of Section. If using Word 3.0x, choose Page Setup from the File menu and select Endnotes.

  2. Select the entire document.

  3. From the Format menu, choose Section. Deselect Include Endnotes.

  4. Enter a section break (COMMAND+ENTER) at the end of the document.

  5. With the cursor below the section break, choose Section from the Format menu and select Include Endnotes. Choose Section Start to be New Page.
