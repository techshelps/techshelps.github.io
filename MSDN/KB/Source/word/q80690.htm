Third-Party TrueType Font Converters

 Last reviewed: February 5, 1998
Article ID: Q80690
The information in this article applies to:
  • Microsoft Word for Windows versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

This article lists some of the independent software vendors (ISVs) who are currently developing TrueType font converters and editors. The list below includes brief descriptions of the products available and telephone numbers to call for more information.

FONT CONVERTERS

Alltype (shipping for MS-DOS)

An MS-DOS character-based application that converts virtually any font format to any other (including TrueType, Type-1, Type-3, Nimbus-Q, and Intellifont). 

   Atech Software
   Out of business. No support provided.

Font Monger for Windows (due by end of 1991)

Font Monger converts virtually any font format to any other (including TrueType, Type-1, Type-3, Intellifont). It also provides minor font editing by creating composite characters and rearranging the characters in a font. 

   Ares Software
   P.O. Box 4667
   Foster City, CA 94404
   (415) 578-9090

Font Monger for Macintosh (shipping)

Font Monger converts virtually any font format to any other (including TrueType, Type-1, Type-3, Intellifont). It also provides minor font editing by creating composite characters and rearranging the characters in a font. 

   Ares Software
   P.O. Box 4667
   Foster City, CA  94404
   (415) 578-9090

Metamorphosis Professional (Macintosh only)

A commercial product that converts virtually any font format to any other (including TrueType, Type-1, Type-3 and PICT format). One interesting feature of this product allows you to pull a Type-1 font from the ROM of an Apple LaserWriter and convert it to another format. 

   Altsys Corporation
   269 W. Renner Road
   Richardson, TX 75080
   (214) 680-2060

Evolution 2.0 (Macintosh only; announced, not shipping)

A commercial product that converts virtually any font format to any other (including TrueType, Type-1, Type-3 format). 

   Image Club Graphics, Incorporated
   1902 11th St. SE, Suite 5
   Calgary, Alberta, CANADA  T2G3G2
   (403) 262-8008

FONT EDITORS

FontStudio 2.0 (Macintosh only)

A complete font-editing tool that allows creation of fonts from scratch and modification of existing fonts. It imports a variety of formats, including TrueType and Type-1. It also includes an autohinter. 

   Letraset Graphic Design Software
   40 Eisenhower Drive
   Paramus, NJ  07653
   (800) 343-TYPE

Fontographer 3.5 (Macintosh only)

A complete font-editing tool that allows creation of fonts from scratch and modification of existing fonts. It imports a variety of formats, including TrueType and Type-1. It also includes an autohinter. 

   Altsys Corp.
   269 W. Renner Road
   Richardson, TX 75080
   (214) 680-2060
Version 4.1 is available. In addition, version 3.5 is available for Windows. Altsys Corporation is now a subsidiary of Macromedia.

TypeMan 1.0 (Macintosh only)

An high-end tool designed for font foundries. This product give you precise control over the hints in a font, and it includes an autohinter. It allows you to program the fonts in a high-level language, and it compiles that into a binary TrueType form. 

   Type Solutions, Inc.
   91 Plaistow Road
   Plaistow, NH 03865
   (603) 382-6400
