WD: Out of Memory Error Opening Word 97 Document with Hyperlinks

 Last reviewed: January 7, 1998
Article ID: Q178921
The information in this article applies to:
  • Microsoft Word for Windows NT, version 6.0 on the following platforms: NT, Win95
  • Microsoft Word 97 for Windows

SYMPTOMS

When you open a Microsoft Word 97 file in Microsoft Word 6.x, running under Microsoft Windows 95 or Microsoft NT, version 4.0, using the Word 97 converter (Wrd97cnv.exe) for Word 6.x, you will receive the following message and will be unable to open the file: 

   Out of Memory
When you click OK, you receive the following message: 

   Word Could Not Open The Document <path\document name>

CAUSE

The document contains more than 24 hyperlinks and you are using the original version of the Word 97 converter that shipped with Office or Word 97.

RESOLUTION

To correct this problem, install Microsoft Word 97 Service Release 1 (SR- 1). This release contains a new Word 97 converter, Wrd97cnv.exe, that when installed will produce a converter, Msword832.cnv, to be used by Word 6.x to open a Word 97 document.

NOTE: The version of the new Msword832.cnv file should be 97051600 or later. For information about how to determine the version of the file, see the "More Information" section of this article.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. This problem was corrected in Microsoft Word 97 SR-1.

For more information about SR-1, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q172475
   TITLE     : OFF97: How to Obtain and Install MS Office 97 SR-1

MORE INFORMATION

To determine the version of the Msword832.cnv converter, follow these steps:

  1. In Windows Explorer, locate the following folder: 

          C:\Program Files\Common Files\Microsoft Shared\Textconv

  2. Right-click the Mswrd832.cnv file, and then click Properties.

  3. Click the Version tab.

For information about how to obtain the updated Word 97 converter without upgrading to the SR-1 release, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q162214
   TITLE     : How to Obtain the Word 97 Converter
For information about converting Word 97 documents to Word 6.x format, please see the following articles in the Microsoft Knowledge Base:

   TITLE     : Limitation Converting Documents From Word 97 to 6.x/95
