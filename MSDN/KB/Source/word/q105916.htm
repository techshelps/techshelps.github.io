Formatting Lost Pasting Label Address Text

 Last reviewed: February 5, 1998
Article ID: Q105916
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

If you choose Envelopes And Labels from the Tools menu, select the Labels tab, cut or copy text that contains character formatting (such as underline, bold, or italic) in the Address box, and paste it in that same box, Word loses the character formatting.

For example, if you cut some underlined text in the Address box and paste it back in the Address box, Word pastes it without the underline character formatting.

STATUS

Microsoft has confirmed this to be a problem in Word. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

You can reapply formatting to text in the Tools Envelopes and Labels dialog box using the keyboard shortcut keys (for example, press CTRL+U or CTRL+B for underline or bold character formatting). CTRL+D or clicking the right mouse button to produce the shortcut menu can be used to access the Font menu. This is possible because the Tools Envelopes and Labels dialog box is a Rich Text Edit (RTE) dialog box.
