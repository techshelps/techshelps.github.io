"Display or Hide Scroll Bars" Help Topic Not Available

 Last reviewed: February 5, 1998
Article ID: Q139778
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you choose the Help topic "Display or Hide Scroll Bars," Word does not display the Help topic and does not provide a ghosting quick tip action. This behavior occurs whether you use the Answer Wizard or select the Help topic using the Index.

STATUS

Microsoft has confirmed this to be a problem in Word version 7.0 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

A quick tip for horizontal or verticle scroll bars is displayed by choosing the Tools menu, clicking Options and, using the secondary mouse button, clicking on Vertical Scroll Bar or Horizontal Scroll Bar.
