WD: Text from Drop-Down Form Field Disappears When Converted

 Last reviewed: February 3, 1998
Article ID: Q132564
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you save a Word document in another file format, and the document contains drop-down form fields, the text of the drop-down form fields is missing from the document when you reopen it.

Text typed in text form fields is preserved.

WORKAROUND

To maintain the text in the drop-down form fields, save the file as Text Only With Line Breaks. Note that this method will place a hard carriage return (paragraph mark) at the end of each line.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
