WD: "Error! No Text of Specified Style ..." with STYLEREF

 Last reviewed: February 3, 1998
Article ID: Q119820
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

If the header or footer of your long document (100 or more pages) or master document contains a STYLEREF field, the following error may occur in place of the STYLEREF field: 

   Error! No text of specified style in document.

RESOLUTION

To resolve this problem, upgrade to Word 97 for Windows or Word 98 Macintosh Edition.

To temporarily work around this problem, divide your long document into two or more separate files of 100 or fewer pages each.

For more information about the STYLEREF field, double-click the Help button on the Standard toolbar and then type "styleref field" (without the quotation marks).

STATUS

Microsoft has confirmed this to be a problem in the Word versions listed above. This problem was corrected in Microsoft Word 97 for Windows and Word 98 Macintosh Edition.
