DMM: Enhanced Version of Direct Mail Manager Available

 Last reviewed: March 16, 1998
Article ID: Q182372
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows

SUMMARY

An enhanced version of Direct Mail Manager is available from Envelope Manager Software.

MORE INFORMATION

If you would like to produce a mailing in one of the following categories, you will need to obtain an enhanced version of Direct Mail Manager from Envelope Manager Software:

  • Large Mailings (over 3,500 pieces)
  • Discount First-Class Mail-Automation Letters and Cards (over 500 pieces)
  • Enhanced Carrier Route sorted Standard Mail (A) Automation Letters
  • Nonprofit Mailings

You can obtain more information about the enhanced version by contacting: 

   Envelope Manager Software
   247 High Street
   Palo Alto, CA 94301-1041

   TELEPHONE: (800) 576-3279
   FAX      : (650) 321-0356
   World Wide Web : http://www.EnvelopeManager.com
The third-party contact information included in this article is provided to help you find the technical support you need. This contact information is subject to change without notice. Microsoft in no way guarantees the accuracy of this third-party contact information.

REFERENCES

For more information about the enhanced version of Direct Mail Manager, click the Help button in Direct Mail Manager, click the Index button in Direct Mail Manager Help, type the following text 

   enhanced
and then double-click the selected text to go to the "Enhanced Version" topic.
