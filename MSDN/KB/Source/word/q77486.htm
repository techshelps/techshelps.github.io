Saving a Hidden File Removes Hidden Attribute in WinWord

 Last reviewed: February 6, 1998
Article ID: Q77486
The information in this article applies to:
  • Microsoft Word for Windows versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

In Microsoft Word for Windows, if you open a hidden file, edit, save, and close it, the file is no longer hidden. You must change the file's attribute back to hidden after closing it.

MORE INFORMATION

In Microsoft Windows version 3.0, you can set the Hidden File Attribute in the Windows File Manager. To hide a selected file, choose Change Attributes from the File menu and select Hidden. Choose the OK button.
