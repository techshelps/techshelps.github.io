Importing Quark XPress Documents

 Last reviewed: July 30, 1997
Article ID: Q63033
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 3.0 , 3.01, 3.02, 4.0

SUMMARY

Quark XPress version 2.11 can save in a format readable by Microsoft Word or Microsoft Works. To do this, use the following steps:

  1. In Quark XPress, open the file to be transferred into Word or Works.

  2. From the File menu, choose Save Text.

  3. Within this dialog box, select the appropriate Word filter (see note below) from the Format menu, then save the file. Word or Works will now be able to read this file.

Note: Filters for Word version 3.0, Word version 4.0, and Works version 2.0 are included with Quark XPress version 2.11. These filters must be placed in the same folder as the XPress program or they will not appear in the format list in the Save Text dialog box.
