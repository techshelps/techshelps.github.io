WD97: Open For Editing from Word Viewer Creates Instance of Word

 Last reviewed: February 4, 1998
Article ID: Q166058
The information in this article applies to:
  • Microsoft Word Viewer 97

SYMPTOMS

If you choose the option to "Open a File for Editing" from Microsoft Word Viewer 97, a new instance of Microsoft Word 97 is started even if there is an instance of Word already running.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To open a file for editing, you must have Microsoft word 97 installed on your computer. To open a file for editing from inside Microsoft Word Viewer, follow these steps:

  1. Open the document in Word Viewer. (On the File menu, click Open and select the document you want to open.)

  2. Once the document is open in Word Viewer you can edit it in Microsoft Word 97. To do this, click Open For Editing on the File menu.

Word 97 will be started and the document will be opened in Word 97.
