WD98: Components Updated by the First Run Upgrade Process (FRU)

 Last reviewed: February 26, 1998
Article ID: Q181805
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SUMMARY

The following components are upgraded during the First Run Upgrade process (which takes place the first time you start Word 98):

  • Template folder locations are updated so that old templates will be available in Microsoft Word 98.
  • Custom dictionaries are upgraded and available in Word 98.
  • Word 5.x standard glossaries are upgraded and available as AutoText entries in Word 98.
  • The Normal template is upgraded.
  • Word 5.x Addresses from the Word Toolbar Prefs file are updated to Address entries in the Address Book.
  • Word Settings 5.x files: almost all settings are updated.
  • Word Settings 6.x files: only Template folder locations are updated.

Items not updated by the First Run Upgrade process include:
  • Filters and converters.
  • The Exclude dictionary.
  • Items in the Word 6.x Startup folder.

MORE INFORMATION

To disable the First Run Upgrade and any Auto macros, press and hold the OPTION+SHIFT keys after double-clicking the Word 98 program icon.

To disable the First Run Upgrade and any other preferences, templates, or add-ins, hold down the SHIFT key after double-clicking the Word 98 program icon.

You can re-run the First Run Upgrade at any time by running the FirstRunUpgrade command. To run the command, you must add the command to the toolbar. To do this, follow these steps:

  1. On the Tools menu, click Customize.

  2. Click the Commands tab.

  3. Under Categories, double-click All Commands.

  4. Click FirstRunUpgrade under Commands and drag the command to the toolbar.

  5. Click the toolbar button to run the command.

To remove the command from the toolbar, follow these steps:

  1. On the Tools menu, click Customize.

  2. Click the toolbar button and drag it from the toolbar.
