WD98: Globe Doesn't Show in Contemporary Letter Document

 Last reviewed: February 17, 1998
Article ID: Q178785
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you create a new document from the Contemporary Letter template, the globe does not show in page layout view or print preview.

NOTE: When you print the new Contemporary Letter document, the globe is printed behind the text of your document.

WORKAROUND

To display the globe when you create a new document from the Contemporary Letter template, follow these steps:

  1. Open the Contemporary Letter template.

    NOTE: The default location for the Contemporary Letter template is <Drive Name>:Microsoft Office 98:Templates:Letters & Faxes

  2. On the View menu, click Header and Footer.

    You should now be able to see the globe behind the text of the template.

  3. Double-click the globe to edit the picture.

  4. On the Picture toolbar, click the Close button.

  5. On the Header and Footer toolbar, click the Close button.

  6. On the File menu, click Save to save the changes you have made to the Contemporary Letter template.

  7. On the File menu, click Close.

When you create a new document from the Contemporary Letter template, the globe will now appear on screen behind the text of the document.

MORE INFORMATION

The globe is a floating (Float Over Text) picture with no wrapping applied and is located behind the text of the document. The globe should appear in page layout view and print preview, and will be printed behind the text of the document.

To Create a New Contemporary Letter Document

  1. In Word, click New on the File menu.

  2. Click to select the Letters & Faxes tab.

  3. Click to select the Contemporary Letter icon, and then click OK.
