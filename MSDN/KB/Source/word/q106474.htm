WD: WordBasic Errors with Table Wizard

 Last reviewed: February 2, 1998
Article ID: Q106474
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

When you run the Table Wizard, you may get one of the following error messages: 

   WordBasic Error 509 The Command is not Available

   WordBasic Error 159 The Value of One of the Fields is too High

   WordBasic error 6   Overflow

   WordBasic error 512 Value out of Range

CAUSE

  • You inserted a negative number for number of rows or columns. 

        -or-
  • You specified a number of rows greater than ~32,500.

RESOLUTION

Run the Table Wizard again and specify a valid number of rows or columns.
