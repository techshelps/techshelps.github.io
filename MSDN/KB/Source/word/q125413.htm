WD: SGML Author: System Requirements

 Last reviewed: March 6, 1998
Article ID: Q125413
The information in this article applies to:
  • Microsoft SGML Author for Word, version 1.0, 1.1, 1.2
  • Microsoft Word for Windows, versions 6.x
  • Microsoft Word for Windows 95, versions 7.x
  • Microsoft Word 97 for Windows

SUMMARY

This article describes what is required for an installation of Microsoft SGML Author for Word.

MORE INFORMATION

The following requirements are necessary for installing and running SGML Author for Word:

Version 1.20

  • Microsoft Word for Windows 95, version 7.x
  • Microsoft Word for Windows 97, Service Release 1
  • 8 MB of memory (Windows 95)
  • 14 MB of memory (Windows NT Workstation)
  • PC using an 80486 or greater microprocessor.
  • Microsoft Windows 95 or Microsoft Windows NT Workstation, version 3.51 or later.
  • A hard drive with 10 MB of free disk space. A complete installation of SGML Author requires 10 MB of hard disk space. A converter-only installation requires 1.5 MB of hard disk space for the converter and its supporting files.
  • CD-ROM drive
  • VGA or higher-resolution video adapter (SVGA 256-color recommended)
  • Microsoft Mouse or compatible pointing device

Version 1.10

  • Microsoft Word for Windows 95, version 7.0, 7.0a.
  • 8 MB of available RAM.
  • PC using an 80486 or greater microprocessor.
  • Microsoft Windows 95 (will not run on earlier versions of Windows) or Microsoft Windows NT Workstation, version 3.51.
  • A hard drive with 10 MB of free disk space. A complete installation of SGML Author requires 10 MB of hard disk space. A converter-only installation requires 1.5 MB of hard disk space for the converter and its supporting files.
  • CD-ROM drive
  • VGA or higher-resolution video adapter (SVGA 256-color recommended)
  • Microsoft Mouse or compatible pointing device

Version 1.00

  • Microsoft Word for Windows 6.0, 6.0a or 6.0c.
  • 6 MB of available RAM (8 MB recommended).
  • PC using an 80386 (25 MHz) or greater microprocessor.
  • Microsoft Windows version 3.1 or later, Microsoft Windows for Workgroups version 3.1 or later, or Microsoft Windows for Pen Computing.
  • MS-DOS operating system version 3.0 or later.
  • A hard drive with 16 MB of free disk space. A complete installation of SGML Author requires 16 MB of hard disk space. A converter-only installation requires 1.5 MB of hard disk space for the converter and its supporting files.
  • A 3.5-inch high-density floppy disk drive.
