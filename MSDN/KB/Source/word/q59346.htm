WinWord /n Starts Word for Windows Without Default DOCUMENT1

 Last reviewed: July 30, 1997
Article ID: Q59346
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

Using the /n parameter for Microsoft Word for Windows (that is, winword /n) starts Word without a default document.

By default, Word for Windows opens a document (DOCUMENT1, based on the NORMAL.DOT template) and allows you to immediately begin editing a new document. If you use the /n parameter, this option is disabled. Word only displays File and Help as choices on the menu bar.
