WD97: Help Topic for Supplemental Converters Listed Twice

 Last reviewed: February 5, 1998
Article ID: Q159236
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

The following topic appears twice in Word Help: 

   Obtain the Word 97 file format converter and other converters.
The first topic provides complete, updated information on how to obtain the supplemental converters. The second topic is incomplete.

RESOLUTION

To open the help topic that contains the correct information, use the following steps:

  1. On the Help menu, click Contents And Index, and click the Find tab.

  2. In box 1, type "supplemental" (without the quotation marks).

  3. In box 3, select the first occurrence of the "Obtain the Word 97 file format converter and other converters" topic, and then click Display.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To locate the topic that contains the incorrect information, use the following steps:

  1. On the Help menu, click Contents And Index, and click the Find tab.

  2. In box 1, type "supplemental" (without the quotation marks).

The topic is listed twice in box 3.
