Proofing Tools Available from Alki Software

 Last reviewed: July 30, 1997
Article ID: Q98274
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

Alki Software Corporation has foreign language, legal, and medical dictionaries for Microsoft Word for Windows.

MORE INFORMATION

Word proofing tools from Alki Software are available in many different languages. Proofing tools for some languages are available only for Word versions 2.0 and later. The tables below indicate which proofing tools are available for which language and version of Word.

Note: You cannot use the proofing tools for Word versions 1.x with Word versions 2.0 and later, and vice versa.

The Comprehensive Spelling Dictionary incorporates all the original spellings from the retail version of Word and adds 74,100 medical, legal, business, financial, and insurance terms. Word's standard spelling dictionary offers 104,000 spellings.

Approximately 516K of additional hard-disk space is needed if Word's original dictionary is not discarded. The original dictionary can be removed from the disk, and the net increase in needed disk spaced is only approximately 262K. No additional RAM is necessary.

The Comprehensive Thesaurus is a replacement for Word's thesaurus file. It does not change the way the Thesaurus feature is used. It offers a total of 600,000 synonyms for 50,000 key words. Word's standard thesaurus offers 200,000 synonyms for 24,000 key words. The Alki Thesaurus file size is 808,301 bytes. The original thesaurus can be removed from the disk. No additional RAM is necessary.

Hyphenation files are small (2K to 34K) in all languages, although the American English hyphenation file is 77K.

The proofing tools are published by Alki Software Corporation, under license from Microsoft. Alki, which also publishes MasterWord, specializes in products for all versions of Microsoft Word.
