WD: Wrong Label Settings for Avery WorkSaver Tabs

 Last reviewed: February 3, 1998
Article ID: Q125248
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

The label formats for the Avery Standard WorkSaver 5 Tabs and WorkSaver 8 tabs are not set up correctly as 4 columns by 30 rows.

Microsoft Word 98 Macintosh Edition

The label formats for the Avery Standard WorkSaver 5 Tabs are set up as 4 columns by 15 rows instead of 4 columns by 30 rows. WorkSaver 8 tabs are set up as 2 columns by 15 rows instead of 4 columns by 30 rows.

Word 97 for Windows

The label formats for the Avery Standard WorkSaver 5 Tabs and WorkSaver 8 tabs are set up as 2 columns by 15 rows instead of 4 columns by 30 rows.

Word 6.x and 7.x

The label formats for the Avery Standard WorkSaver 5 Tabs and WorkSaver 8 Tabs are set up as 2 columns by 30 rows instead of 4 columns by 30 rows.

WORKAROUND

Modify the label details before printing or adding the labels to a new document. To prepare your labels, use the appropriate method for your version of Word.

Word 97 and Word 98 Macintosh Edition

  1. On the Tools menu, click Envelopes and Labels.

  2. Click Options. In the Label Options dialog box, click Laser and Ink Jet (under Printer Options).

  3. In the Product Number list, click to select either "WordSaver 5 - Tabs" or "WorkSaver 8 - Tabs."

  4. Click New Label.

  5. In the Label Name box, type a name for your custom label.

  6. Modify the label settings: 

        - For WorkSaver 5 Tabs, change the following:


             Horizontal Pitch        2 inches (Word 97 only)
         Label Width             2 inches (Word 97 only)
         Number Across           4 (Word 97 only)
         Number Down             30

      Do not modify the other settings.

    - For WorkSaver 8 Tabs, change the following:

         Horizontal Pitch        2 inches
         Label Width             1.5 inches
         Number Across           4
         Number Down             30

      Do not modify the other settings.

  7. Click OK.

This creates a custom laser product that you can select from the Product Number list (see step 3).

Word 6.x, 7.x

  1. On the Tools menu, click Envelopes And Labels.

  2. Click the Laser option.

  3. In the Product Number list, select either "WordSaver 5 - Tabs" or "WorkSaver 8 - Tabs."

  4. Click Details.

  5. Modify the label settings: 

        - For WorkSaver 5 Tabs, change the following:


             Horizontal Pitch        2 inches
         Label Width             2 inches
         Number Across           4
         Number Down             30

      Do not modify the other settings.

    - For WorkSaver 8 Tabs, change the following:

         Horizontal Pitch        2 inches
         Label Width             1.5 inches
         Number Across           4
         Number Down             30

      Do not modify the other settings.

  6. Click OK.

  7. Click OK when prompted to save the changes.

This will create a custom laser product that you can select from the Product Number list (see step 3).

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
