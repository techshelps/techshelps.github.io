WD: Options Unavailable While in Print Preview

 Last reviewed: February 2, 1998
Article ID: Q106130
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0, 6.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SUMMARY

Although you can edit your document while in print preview, it is an "exclusive" mode, so you are unable to perform any action that switches focus out of print preview.

For example, you cannot click New on the File menu while in print preview. You must click Close on the File menu before clicking New.

MORE INFORMATION

The following options are unavailable (dimmed) while in print preview: 

   Menu      Commands unavailable
   --------------------------------------------------

   File      New
             Open
             Find File (Word 6.x)
             Versions  (Word97 and Later)
             Send To:  (Word97 and Later)
                      Mail Recipient      
                      Routing Recipient
                      Exchange Folder
                      Fax Recipient
             Templates (Word 6.x, 7.x)
             MRU list (Most recently used file list)

   Edit      Paste As Hyperlink (Word97 and Later)
             Object

   Insert    Database (Word 6.x, 7.x)

   Tools     Mail Merge
             Merge Documents (Word97 and Later)
             Templates and Add-Ins (Word97 and Later)

   Table     (Show) Gridlines

   Window    Everything
In addition, if you click Envelopes And Labels on the Tools menu, the New Document button is unavailable on the Labels tab.
