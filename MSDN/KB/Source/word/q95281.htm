WD: Changes Made in MacWord Table Are Not Reflected in Chart

 Last reviewed: February 2, 1998
Article ID: Q95281
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.1
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

If you use a table from a Word for the Macintosh document to create a chart in Microsoft Graph, modifications you make to the table in your document do not automatically update the chart. If you want to update the chart, you must open it in Microsoft Graph and update the datasheet.

CAUSE

When you use a Word table to create a datasheet in Microsoft Graph, no link is established between the table and the Graph datasheet.

STATUS

This is a design feature of Word, Microsoft Graph, and the OLE specification.

REFERENCES

"What's New in Microsoft Word Version 5.1," version 5.1, pages 40-41
