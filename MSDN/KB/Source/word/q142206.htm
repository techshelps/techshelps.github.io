File Find Delete Command Does Not Send File to Recycle Bin

 Last reviewed: February 5, 1998
Article ID: Q142206
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows 95 operating system

SYMPTOMS

When you a delete a file from the File Find search results window, the files are deleted from the system and are never placed in the Recycle Bin.

CAUSE

Programs that are not designed specifically for Windows 95 are not "Recycle Bin aware." Hence, they do not move files to the Recycle Bin; they delete the files permanently.

WORKAROUNDS

To place files into the Recycle Bin before you remove them permanently from the hard disk, do one of the following:

Method 1: Use the Windows Explorer to delete your files.

Method 2: Upgrade to Word for Windows 95.

MORE INFORMATION

The following table shows which programs delete files to the Recycle Bin in Windows 95. 

   Program                           Deleted to Recycle Bin
   -------                           ----------------------

   Explorer                          Yes
   32-bit Windows 95 programs        Yes
   MS-DOS Programs                   No
   MS-DOS Prompt                     No
   Windows File Manager              No
   16-bit programs                   No
