WD6x: Duplicate Spelling Files in the Microsoft Folder

 Last reviewed: February 6, 1998
Article ID: Q122836
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

The Microsoft Folder installed by Word 6.0 at the same level as the folder that contains the Microsoft Word file may contain the following two spelling files: 

   MS Spelling
   MS Spelling 2

MORE INFORMATION

The Microsoft Folder installed at the same level as your Word 6.0 folder contains shared proofing tools files. The Office applications can share all the files and programs that are in this folder.

The MS Spelling file is a duplicate of the Word spelling file, MS Spelling 2. The MS Spelling 2 file is a newer, more robust spelling program.

The MS Spelling file is installed by a stand-alone version of Microsoft Excel 5.0 and Microsoft PowerPoint 4.0. Neither the Microsoft Excel nor the Microsoft PowerPoint Setup programs checks to see whether a newer version of the dictionary exists. This problem does not occur when you install Microsoft Office.

NOTE: The old Microsoft Folder in the Extensions Folder contains files shared by earlier versions of the Microsoft Office applications (namely, Microsoft Excel 4.0 and Microsoft PowerPoint 3.0). If you remove the Microsoft Folder in the Extensions Folder, the Spelling Checker fails in Microsoft Excel 4.0 and PowerPoint 3.0.
