Unable to Edit Data Source After Attaching MS Excel Worksheet

 Last reviewed: February 3, 1998
Article ID: Q144390
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

You are unable to edit a Microsoft Excel Worksheet data source attached to a Word mail merge main document by clicking the Edit Data Source button on the Mail Merge toolbar or by using the Mail Merge Helper.

WORKAROUND

To work around this problem, open the worksheet in Microsoft Excel, make your changes, and save the worksheet. In Word, re-attach the Microsoft Excel worksheet.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem was corrected in Microsoft Word 98 Macintosh Edition.
