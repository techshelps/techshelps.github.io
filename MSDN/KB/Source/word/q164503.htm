MORE INFORMATION

Microsoft Office 97, Developer Edition Professional (ODE)

The newest version of Microsoft Office has everything for building and distributing solutions with Microsoft Office 97. Office 97, Developer Edition is designed for developers creating solutions with Microsoft Office. It includes:

  1. Microsoft Office 97, Professional Edition.

  2. Microsoft Office 97 Developer Edition Tools: 

        - Royalty-free Microsoft run-time license.


        - Microsoft Access/Microsoft Visual SourceSafe(TM) integration tools
      for team development.


        - Replication Manager for viewing and managing replicated databases on
      a network or the Internet.


        - Setup Wizard for creating familiar setup routines that work like
      Microsoft Office setup.


        - Additional ActiveX controls, including FTP, HTTP, and Gopher.

  3. Printed developer documentation: 

        - Office 97/Visual Basic Programmer's Guide, 496 pages.


        - Building Applications with Microsoft Access 97, 678 pages.


        - Microsoft Office 97 Object Model Guide, 33 pages.

  4. Technical information and code samples from the Microsoft Developer Network.

The Microsoft Office 97 Resource Kit

If you're an administrator or an IS professional, this book was written for you. Here you'll find the hands-on, in-depth information you need to roll out, support, and troubleshoot Microsoft Office 97. For systems running Microsoft Windows 95 and Microsoft Windows NT (3.51 and 4.0), this is the professional's guide to Microsoft Office 97. Here's hands-on information about updating from earlier versions of Microsoft Office, about switching from other applications such as Lotus 1-2-3 and WordPerfect, and about coexistence among different versions of Microsoft Office. Of course, you get full information on network installation, plus the timesaving Network Installation Wizard on the enclosed CD-ROM. And like all the tools and utilities on the CD-ROM, this wizard is a tested, supported application designed to make your job easier.

To visit the Microsoft Office 97 Resource Kit Web site, connect to: 

   http://www.microsoft.com/Office/ORK/
