Word Opens a Blank Document Instead of a URL

 Last reviewed: February 6, 1998
Article ID: Q125943
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

When attempting to open a Uniform Resource Locator (URL) or follow a link, Word appears to be searching for the URL or link, but nothing happens. Control is returned to Word, no error message is displayed in a message box or in the status bar, and the URL or link is not activated.

CAUSE

  • InternetWorks may be improperly registered in the registration database.

    -or-

  • The registration database may be damaged or corrupted.

RESOLUTION

Method 1: Manually register InternetWorks and Word. To do this, go to 

          Windows File Manager. Locate the IW.REG file in the
          <WinWord>\INTERNET subdirectory (where <WinWord> is your Word for
          Windows directory--usually, C:\WINWORD) and double-click it to
          register it. Locate the WINWORD6.REG file in the <WinWord>
          directory and double-click it to register it. Start InternetWorks
          and Word. If this procedure does not correct the problem, use
          Method 2.
Method 2: Rebuild the registration database. For additional information 
          on how to rebuild the registration database, please see the
          following article in the Microsoft Knowledge Base:

             ARTICLE-ID:Q122964
             TITLE     :How to Rebuild the Reg.Date File and Re-Register
                         Applications
