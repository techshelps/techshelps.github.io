WD: Can't Print Multiple Copies from Word

 Last reviewed: March 4, 1998
Article ID: Q145261
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a and 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

In Word, you cannot print multiple copies of your document when you print to any of the following printers:

  • Hewlett-Packard DeskJet 1200C, Hewlett-Packard 1600 printers, or Hewlett Packard DeskJet 550C, 560C, 660C, 693C, or 855C using the legacy printer drivers.
  • Hewlett-Packard LaserJet 5P, using the HPPCL5MS printer driver version G.1.20.
  • Hewlett-Packard LaserJet 5M.
  • Hewlett-Packard LaserJet 6MP.
  • Hewlett-Packard OfficeJet.
  • NEC SilentWriter II model 90 PostScript printer, using the Windows 95 printer driver.
  • Brother HL 960.
  • Brother HL630.
  • Epson Stylus 600 Inkjet Color

CAUSE

There are three possible causes of this problem:

  • You are using a Windows 3.1 printer driver.
  • The Registry key that contains Word settings is corrupt.
  • The printer you are using does not support printing multiple copies.

WORKAROUND

To work around this problem, choose the appropriate method below:

Method 1: Upgrade to a Windows 95 Printer Driver

Method 2: Clear the Background Printing Option

  1. On the Tools menu, click Options, and then click the Print tab.

  2. Click to clear the Background Printing check box.

  3. Click OK.

Method 3: Change the Graphics Mode From Vector to Raster

  1. Open the Control Panel Printers icon.

  2. Select the Printer icon.

  3. On the File menu, click Properties.

  4. Click the Graphics tab.

  5. Select Use Raster Graphics, click Apply, and click OK.

Method 4: Update Windows 95/Printer Driver

Do one of the following:

  • Download and install an updated printer driver from the following location: 

          ftp://ftp.microsoft.com/Softlib/MSLFILES/HP5XX.EXE

    -or-

  • Upgrade to Microsoft Windows 95 Service Pack #1. The latest driver can be downloaded from the Microsoft Internet site: 

          http://support.microsoft.com/support/a.asp?M=S


          Click "show long menu" then click Downloads

    NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

          http://www.microsoft.com/

    Windows 95 Service Pack #1 (SP1) contains updated printer drivers for the following printers: 

          HP DeskJet 310
      HP DeskJet 320
      HP DeskJet 500C
      HP DeskJet 520
      HP DeskJet 540
      HP DeskJet 550C
      HP DeskJet 560C
      HP DeskJet Portable

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
