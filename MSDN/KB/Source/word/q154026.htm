WD: Error Using FilePost in WordBasic

 Last reviewed: February 12, 1998
Article ID: Q154026
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

When you run the FilePost command in a macro and select the folder you want to post to in the Post Document dialog box, you receive the following error messages: 

   General Microsoft Exchange failure.
   Check your Exchange installation and try again.

   Wordbasic Err=102
   Command Failed.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows versions 7.0, 7.0a. This problem was corrected in Word97.

WORKAROUND

To work around the problem, from the document, click Post To Exchange Folder on the File menu.
