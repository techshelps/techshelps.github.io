Converters, Filters, Data Access Not Uninstalled but Checked

 Last reviewed: February 6, 1998
Article ID: Q114224
6.00 6.00a 6.00c WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

If you use Word Setup to remove Word or to remove the Converters, Filters, and Data Access from your system, and then you use Setup to reinstall, the check box for Converters, Filters, and Data Access appears selected, even though Text Conversion Filters and Graphic Import Filters are not installed.

CAUSE

By design in Word, when you uninstall Word Converters, Filters, and Data Access using Word Setup, Setup will not remove the ODBC drivers.

Because the ODBC drivers are not removed, when you reinstall, the initial screen shows the option for Converters, Filters, and Data Access as already installed. If you choose the Change Options button, you will see that the Text Converters, Graphics Filters, and Data Access [ODBC] options are not selected, and only ODBC is selected.

RESOLUTION

  1. Run Word Setup.

  2. Choose the Add/Remove button.

  3. Select the Converters, Filters, and Data Access option, and then choose the Change Option button.

  4. Select the Text Converters and Graphic Filters check boxes and choose the OK button.

  5. Choose the Continue button to complete the reinstall.

MORE INFORMATION

The following is the list of ODBC files installed by Word that are not removed when an uninstall is done: 

   DRVACCSS.HLP
   DRVDBASE.HLP
   DRVFOX.HLP
   DRVPARDX.HLP
   RED110.DLL
   PDX110.DLL
   XBS110.DLL
   MSJETDSP.DLL
   ODBC.DLL
   ODBCINST.DLL
   ODBCINST.HLP
   ODBCADM.EXE
   SIMBA.DLL
   SIMADMIN.DLL
