WD: Can't Change or Remove Electronic Mail Commands from File Me

 Last reviewed: August 5, 1997
Article ID: Q119290
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Office 97 for Windows
  • Microsoft Outlook 97 for Windows
  • Microsoft Windows 95

SYMPTOMS

You cannot change the name of the Send To (Send) command or change the names for Routing Recipient (Add Routing Slip), Mail Recipient, or similar commands on the File menu, nor can you remove these commands from the File menu in Word for Windows unless you disable the MAPI features of your electronic mail (E-mail) program.

CAUSE

The above commands on the File menu come from the .DLL file associated with your E-mail program (for example, MAPI.DLL or MAPIVIM.DLL). As a result, you cannot remove or change the names of these commands in Word.

NOTE: You can still modify the way these commands work by modifying the Word FileSendMail (FileSend) or FileRoutingSlip (FileAddRoutingSlip) commands in a macro editing window. The only thing you cannot change is their presence or names on the File menu.

WORKAROUND

In Windows 95, the MAPI entries are contained in the Registry and must be removed by un-installing Microsoft Exchange, Microsoft Outlook (if Office 97 is installed) and/or third-party mail-related components).

Use the appropriate steps below for your system:

For Office 97 and Windows 95, to Remove Outlook

  1. Click the Start button, point to Settings, then click Control Panel.

  2. Double-click Add/Remove Programs.

  3. Click the Install/Uninstall tab.

  4. On the list of installed applications, click Microsoft Office 97, then click the Add/Remove button. The Office Setup program starts.

  5. On the Office 97 Setup screen, Click Add/Remove. Then under Options, clear the check box for Microsoft Outlook. Click Continue to remove Microsoft Outlook.

For Windows 95, to Remove Exchange

  1. Click the Start button, point to Settings, then click Control Panel.

  2. Click Add/Remove Programs.

  3. Click the Windows Setup tab

  4. Clear the Microsoft Exchange check box.

This removes the Send To (Send) and Routing Recipient(Add Routing Slip) items from the File menu of Word.
