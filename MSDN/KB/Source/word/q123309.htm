WD: Grammar Command Missing from Tools Menu

 Last reviewed: November 17, 1997
Article ID: Q123309
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

The Grammar command does not appear on the Tools menu and the Grammar tab (accessed by clicking Options on the Tools menu) is unavailable.

CAUSE

The Word program file is not in the Word folder.

WORKAROUNDS

Method 1

Move the Word application program into the Word folder. Quit and restart Word. Grammar should now be on the menu.

Method 2

Specify the location for the Grammar checker. To do this, follow these steps:

  1. On the Tools menu, click Macros.

  2. Under Macros Available In list, select Word Commands.

  3. Select the ToolsGrammar command and click Run.

  4. Specify the location of the Grammar checker. Setup installs this in the Word Extensions Folder.

  5. Quit and restart Word.

Grammar should now be on the menu.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
