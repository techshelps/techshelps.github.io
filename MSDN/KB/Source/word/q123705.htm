HP Printer Loses Characters When Using A4 Paper and Landscape

 Last reviewed: February 5, 1998
Article ID: Q123705
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

If you print a document in landscape using A4 paper to a Hewlett-Packard (HP) II, III, or IIIsi printer, the resulting printed document may be missing characters along the left margin.

CAUSE

This problem occurs if you are using A4 paper and do not have an A4 tray. This will also occur if you do not have an A4 tray and you manually feed the paper without changing the default paper tray option.

Note: This problem does not occur when printing in PostScript mode.

STATUS

Microsoft has confirmed this to be a problem in Word 6.0, 6.0a, and 6.0c for Windows. This problem was corrected in Word version 6.0 for Windows NT This problem was corrected in Word version 7.0 for Windows 95.

WORKAROUND

  1. From the Tools menu, choose Options. Select the Print tab.

  2. From the Default Tray list, select Manual Feed. Choose OK.

RESOLUTION

Use an A4 paper tray in the printer.
