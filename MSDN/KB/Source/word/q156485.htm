WD97: Form Field Command No Longer on the Insert Menu

 Last reviewed: July 31, 1997
Article ID: Q156485
The information in this article applies to:
  • Microsoft Word 97 for Windows

SUMMARY

The Form Field command is no longer on the Insert menu.

MORE INFORMATION

The Form Field menu selection has been removed from the Insert menu in Microsoft Word 97 for Windows. Form fields are inserted into a Word document, by default, from the Forms Toolbar.

To activate the Forms Toolbar, point to Toolbars on the View menu, and then click Forms.

To add the Form Field command to the Insert menu, do the following:

  1. On the Tools menu, click Customize.

  2. Click the Commands tab.

  3. In the Categories list box, select All Commands.

  4. In the Commands list box, click InsertFormField.

  5. Drag InsertFormField to the Insert menu. Release the mouse button when you see a horizontal bar indicating the location on the menu where the command will be inserted.

  6. Click Close.
