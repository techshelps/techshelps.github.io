"Feed From" Setting Not Retained in Envelope Printing Options

 Last reviewed: February 5, 1998
Article ID: Q117404
6.00 6.00a WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a
  • Microsoft Windows operating system version 3.1

SYMPTOMS

When you set printing options for an envelope, Word for Windows does not retain the Feed From setting if you select Default Tray. By contrast, Word does retain all other Feed From settings. (To set envelope printing options, choose Envelopes And Labels from the Tools menu, choose the Options button, and then select the Printing Options tab.)

CAUSE

The Default Tray setting is printer-driver specific, which means it is stored with the printer driver, not in your WINWORD.OPT file. As a result, when you select the Default Tray setting, Word retrieves the name of the specific tray from the printer driver and changes the Default Tray setting accordingly. In other words, the Default Tray setting tells Word to retrieve the tray setting from the printer driver and replace Default Tray with the specific tray name in the Feed From box.

WORKAROUND

If you want Word to retain a specific Feed From setting for printing envelopes, do not select Default Tray from the Feed From list; instead, select a specific tray. Word stores the Feed From setting in your WINWORD.OPT file as long as it is a specific tray name. The only time Word does NOT store the Feed From setting is when you select Default Tray; in this case, Word refers to the printer driver for the setting information.
