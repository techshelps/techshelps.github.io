Files Installed in MSOFFICE\WINWORD\WINWORD Folder

 Last reviewed: February 6, 1998
Article ID: Q134810
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

During Setup, if you change the default directory location to the MSOFFICE\WINWORD folder (subdirectory), your files will be installed to the following location: 

   MSOFFICE\WINWORD\WINWORD

CAUSE

The Setup program defaults to the MSOFFICE directory because it installs folders that are shared by other Office programs. This behavior is by design.

WORKAROUND

To install the files to the MSOFFICE\WINWORD folder, run the Setup program to remove all components. Then re-run setup to install Word to the default location. The files will be installed to the correct folders.
