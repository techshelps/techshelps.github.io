Word 6.0a Patch Error: "Cannot Locate WINWORD.EXE"

 Last reviewed: February 6, 1998
Article ID: Q111718
6.00 WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, version 6.0
  • Microsoft Windows operating system version 3.1

SYMPTOMS

When you run UPGRADE.EXE to upgrade Word version 6.0 for Windows to Word version 6.0a for Windows, the following error message may occur: 

   Cannot locate the file <path>\WINWORD.EXE

CAUSE

If you renamed your Word 6.0 program file to something other than WINWORD.EXE, the Word 6.0a Patch cannot locate the file. You may have renamed WINWORD.EXE in order to run both Word 2.x and Word 6.0 at the same time.

If you choose either the OK or Cancel button in the "Cannot locate WINWORD.EXE" dialog box, the following message appears: 

   The patch to file <path>\WINWORD.EXE was not applied.
If you chose OK in this dialog box, UPGRADE continues to run. If you chose Cancel in the previous dialog box, UPGRADE stops.

RESOLUTION

To complete the installation of Word 6.0a, do the following to rename your Word 6.0 program file and rerun UPGRADEU:

  1. Exit UPGRADE and rename your Word 6.0 program file to WINWORD.EXE (note the name if you need to restore it after you upgrade to Word 6.0a).

  2. Run UPGRADE again to install Word 6.0a.

  3. If necessary, rename WINWORD.EXE to the previous name.
