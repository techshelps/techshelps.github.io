WinWord: Network Setup on DEC Pathworks Requires FILE SERVICE

 Last reviewed: February 6, 1998
Article ID: Q80621
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

SUMMARY

The Setup program in Microsoft Word for Windows does not recognize DEC Pathworks 4.x "DISK SERVICE" as a network drive. To do a network server installation in this situation, you must use a "FILE SERVICE" while using DEC Pathworks 4.x.

Windows considers DEC Pathworks 4.x "DISK SERVICE" to be a local disk drive. Windows considers DEC Pathworks 4.x "FILE SERVICE" to be a network drive.

This issue only pretains to Pathworks on a VAX. Pathworks may be used on a Vax or an OS/2 server. A VAX can be configured to include FILE SERVICES and DISK SERVICES. The OS/2 server can only use FILE SERVICES.
