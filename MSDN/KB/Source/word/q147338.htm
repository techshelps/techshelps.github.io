"No Help Topic" for Buttons in File Open Dialog Box

 Last reviewed: February 5, 1998
Article ID: Q147338
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When you try to view Help for any of the buttons across the top of the File Open dialog box by using the What's This button (?), a dialog box appears with the following message: 

   No help topic is associated with this item.
You receive the same message when you try to view Help by using the What's This button for the following items in the Advanced Find dialog box: 

   Property
   Define More Criteria
   Look In

WORKAROUND

For information about finding and opening documents, see "Getting Results with Microsoft Word for Windows 95," Part 1, "Find and Open Documents," pages 20-22.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows 95 version 7.0. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
