Equation Editor Objects Cropped When Printing to HP LaserJet

 Last reviewed: February 6, 1998
Article ID: Q122391
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

Characters at the left or right side of an Equation Editor 2.0 object may be cropped when you print to a Hewlett-Packard LaserJet printer.

CAUSE

This problem occurs because some calculation errors occur within the printer driver during computation of the printing size of the equation object.

STATUS

Microsoft has confirmed this to be a problem with the following printer drivers:

  • HP LaserJet Series II, IIP, and IID drivers version 2.1, universal printer driver 3.1.2
  • HP LaserJet III, IIIP, IIID, and IIISi drivers version 31.3.89
  • HP LaserJet III, IIIP, IIID, and IIISi version 2.1, universal printer driver 3.1.4

RESOLUTION

Use one of the following methods to correct the problem:

  • Select the Print TrueType As Graphics option in the printer driver. To do this, use the following steps:

    1. From the File menu, choose Print.

    2. In the Print dialog box, choose the Printer button.

    3. Choose the Options button.

    4. Select the Print True Type As Graphics option.

    5. Choose the OK button.

    6. Choose the Close button.

    7. Choose the OK button to print the document.

    -or-

  • Upgrade to Equation Editor version 2.0a. This version is included with Word version 6.0a for Windows.

    NOTE: After upgrading your version of Word, all equations exhibiting the problem must be updated. To do this, double-click each equation object to edit, then immediately click back in the document area. This updates the equation object to the new version of Equation Editor.

    -or-

  • Change to a different printer driver that is supported by your printer.
