Word: Choosing Save or Save As Hangs Machine

 Last reviewed: July 30, 1997
Article ID: Q80469
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0 5.1
  • Macintosh system software versions 6.0.7, 7.0 and later

SYMPTOMS

If you are running Word with Norton Directory Assistance version 2.0 or earlier and System 6.0.7, or 7.0, choosing the Save or Save As command from the File menu may cause your computer to hang (stop responding).

CAUSE

Word versions 5.0 and 5.1 take advantage of the System 7.0 advanced dialog box features that allow pull-down menus in the dialog box. Directory Assistance versions 2.0 and earlier incorrectly try to modify these dialog boxes. As a result, the system hangs.

STATUS

This problem does not occur in Word for the Macintosh version 6.0.

MORE INFORMATION
