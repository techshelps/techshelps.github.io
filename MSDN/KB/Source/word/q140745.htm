WD: Err Msg: "There Is No Address Field in the Address Book"

 Last reviewed: February 5, 1998
Article ID: Q140745
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

You get the following error message when you select an address from Word's Address Book: 

   There is no address field in the address book. Pick the desired
   address book and try again.

CAUSE

This error occurs when the record does not have any address information in the address fields. The names appear as part of the Personal Address Book from Exchange because they do contain email information. They do not necessarily contain contact information. This behavior is by design.

RESOLUTION

Add contact information for the address book record.

MORE INFORMATION

To confirm whether there is contact information, highlight the name of the contact, click Properties, and select the Business tab. If both the first and the last names appear in the First name box, there is no contact information for that person.

Word's Address Book uses the Exchange Personal Address Book. Word has no means of comparison to determine which names contain contact information, so all names from the Personal Address Book are listed.
