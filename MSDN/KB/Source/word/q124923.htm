Word Hangs with Document Created with QuickDraw GX

 Last reviewed: December 5, 1997
Article ID: Q124923
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

Word hangs (stops responding) when you choose one of the following commands: Print, Print Preview, Document Layout, or Page Setup.

CAUSE

This problem occurs when the document was created on an Apple Macintosh with QuickDraw GX installed, then is opened and saved under a non-GX system, and then finally printed under QuickDraw GX.

In other words, this happens when all of the following are true:

  • The document was saved as a Word document on a Macintosh with QuickDraw GX installed.

    -and-

  • You opened and saved the document on a Macintosh that is not using QuickDraw GX.

    -and-

  • You chose one of the following commands when running QuickDraw GX: Print, Print Preview, Document Layout, or Page Setup.

STATUS

Microsoft has confirmed this to be a problem in Word 6.0 for the Macintosh. This problem was corrected in Word version 6.0.1 for the Macintosh.

WORKAROUND

Save the file in Rich Text Format (RTF) before choosing Print, Print Preview, Document Layout, or Page Setup. This removes the GX print information from the document. Follow the steps below to save the file in RTF file format:

  1. From the File menu, choose Save As.

  2. From the Save File As Type list, select Rich Text Format, and click the Save button.

  3. Close the document and reopen it.
