Internet Assistant Not Included with Word for Windows 95

 Last reviewed: February 5, 1998
Article ID: Q132520
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Windows 95

SUMMARY

The Microsoft Office for Windows 95 Corporate Deployment Guide implies that Internet Assistant is included with Word for Windows 95. This is incorrect; Internet Assistant is not included with this product.

On page 10, the guide incorrectly states: 

   In addition, each Office application has been enhanced to incorporate
   new communications capabilities of its own. . .Microsoft Word for
   Windows 95 offers the Internet Assistant for Microsoft Word for Windows
   95.

MORE INFORMATION

The current version of Internet Assistant (1.0z) does not work with Word for Windows 95.
