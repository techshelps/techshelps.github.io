WD: Open/Send Mail with Enclosure Hangs Mac with Tokamac '04

 Last reviewed: January 30, 1998
Article ID: Q77912

The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 6.0

SYMPTOMS

If you use Microsoft Word for the Macintosh on a Macintosh with a Tokamac '040 board installed, Word may stop responding if you click the Send Mail command on the File menu and then click Enclosure.

STATUS

This problem was corrected with the Microsoft Mail Server version 3.0a.

To obtain an update, call the Microsoft Order Desk at (800) 360-7561 and request Microsoft Mail Server version 3.0a. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web page at: 

  http://www.microsoft.com/worldwide/default.htm
