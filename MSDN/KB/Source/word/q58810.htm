Word: Printing Envelopes with the HP DeskWriter

 Last reviewed: July 30, 1997
Article ID: Q58810
The information in this article applies to:
  • Microsoft Word for the Macintosh versions 3.0, 3.01, 3.02, 4.0, 5.0, 5.1

SUMMARY

Printing envelopes from Word to the Hewlett-Packard (HP) DeskWriter requires different steps than printing envelopes to many other printers. This article gives the steps for printing envelopes to the DeskWriter.

MORE INFORMATION

The following steps outline the correct procedure for an envelope with a return address:

  1. From the File menu, choose Page Setup and select Envelope for the page size. In Word version 3.x, hold the SHIFT key down before you choose Page Setup from the File menu.

  2. From the Format menu, choose Document and set the margins as shown below. (In Word 3.x, from the File menu, choose Page Setup - don't hold down the SHIFT key - in order to modify your margins.) 

          Left =.306 in.               Top = .306 in.
      Right = 1.0 in.             Bottom = 1.0 in.

  3. Type in the return address.

  4. Press the RETURN key five times.

  5. Move the left indent marker on the ruler over to the 3.5-inch mark.

  6. Type in the mailing address.

To print an envelope without a return address, do the following:

  1. From the File menu, choose Page Setup and select Envelope for the page size. In Word 3.02, hold down the SHIFT key and choose Page Setup from the File menu.

  2. From the Format menu, choose Document and set the margins as shown below. (In Word 3.x, from the File menu, choose Page Setup - don't hold down the SHIFT key - in order to modify your margins.) 

          Left = 3.50 in.           Top = 1.50 in.
      Right = 1.0 in.          Bottom = 1.0 in.

  3. Type in the mailing address.

Next, remove the plastic paper-path cover and insert the envelope face down, sliding it under the feed wings until it touches the printer wheels. Once this is done, choose Print from the File menu and then press the Select button on the printer when the printer prompts you to do so.

The DeskWriter can print only on U.S. Number 10 or DL size envelopes. For more information on printing envelopes, see pages 1-27 and 1-28 of the DeskWriter manual.
