WD: Text Prints Past Right Margin with PostScript Printer on Mac

 Last reviewed: February 5, 1998
Article ID: Q90783
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 4.0, 5.0, 5.1, 6.0
  • The Macintosh operating system versions 6.07, 6.08, 7.0, and later

SYMPTOMS

In Word, lines may extend past the right margin of a paragraph if the character formatting of the paragraph includes TrueType fonts formatted in italic, bold, or bold italic.

CAUSE

This problem occurs when the resident fonts of the printer are duplicated by TrueType outline fonts in the System Folder. Fonts with numbers after their names are screen fonts. Fonts without numbers are TrueType fonts. (If you have duplicate fonts, both types of font will be displayed.) To determine which fonts are duplicated, follow the appropriate procedure below. 

   System 7.x
   ----------

   Open the System and/or Fonts suitcases located in the System Folder
   and choose By Name from the View menu (System 7.0 and 7.0.1 users will
   not have a Fonts suitcase).

   System 6.x
   ----------

   Start the Font/DA Mover application and select the Font button.

WORKAROUND

To work around this problem, use any of the following methods.

Method 1

Follow the appropriate procedure below to remove the TrueType fonts that duplicate PostScript printer fonts: 

   System 7.1
   ----------

   1. Open the Fonts Folder (located in the System Folder).

   2. From the View menu, choose By Name.

   3. Remove all TrueType fonts that duplicated PostScript fonts by
      dragging them out of the Fonts folder. (Fonts with numbers
      after their names are screen fonts; fonts without numbers are
      TrueType fonts.)

   4. Also follow the steps for System 7.0 below. (Under some circum-
      stances, fonts will be installed in the System suitcase, even
      for Macintoshes running System 7.1.)

   System 7.0
   ----------

   1. Open the System suitcase (located in the System Folder).

   2. From the View menu, choose By Name.

   3. Remove all TrueType fonts that duplicate PostScript fonts by
      dragging them out of the System suitcase. (Fonts with numbers
      after their names are screen fonts. Fonts without numbers are
      TrueType fonts.)

   System 6.x
   ----------

   1. Start the Font/DA Mover application.

   2. Select the Font button.

   3. Remove all TrueType fonts that duplicate PostScript fonts by
      selecting them from the list and choosing the Remove Button.
      (Fonts with numbers after their names are screen fonts. Fonts
      without numbers are TrueType fonts.)

   4. Quit the Font/DA Mover.

Method 2

Format the text with a font not duplicated in the System.

Method 3

Remove bold and italic formatting from the paragraph.

Method 4

With System 7.x, obtain the LaserWriter driver version 7.1.2 or later, which addresses this problem. This driver is part of System version 7.1, but it is compatible with any version of System 7.
