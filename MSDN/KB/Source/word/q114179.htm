Cannot Find WORDCBT.CBT Error When Using Quick Preview

 Last reviewed: February 5, 1998
Article ID: Q114179
6.00 6.00a 6.00c WINDOWS kbusage kberrmsg

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows operating system versions 3.0 and 3.1

SYMPTOMS

Word generates the following error message when you choose Quick Preview from the Help menu: 

   Word cannot find the file WORDCBT.CBT

CAUSE

The problem may occur because you changed the location of the WORDCBT.CBT file from the installed directory but did not change the path to the file in the WINWORD6.INI file.

On a network, this error will occur if a universal naming convention (UNC), is used to specify the CBT location.

RESOLUTION

Method 1: The Word 6.0 for Windows tutorial files are placed in the 

          <WINWORD>\WORDCBT subdirectory during setup. You can move the
          files to another location, but the reference contained in the
          WINWORD6.INI file must be changed to correspond to the new
          location. To change this reference, do the following:

          a. From the Tools menu, choose Options.

          b. Select the File Locations tab.

          c. Under File Types, select Tutorial. Choose the Modify button.

          d. Select the new location or choose the New button to create
             the new location and choose OK.
Method 2: Use a text editor to add or update the CBT-PATH statement in the 
          [Microsoft Word] section of the WINWORD6.INI file to reflect the
          correct path, as in the following example:

             [Microsoft Word]
             CBT-PATH=C:\WORDCBT
Method 3 (On a Network): 

          1. From the Tools menu choose Options.

          2. Choose the File Locations tab.

          3. Select Tutorial and choose the Modify key.

          4. Choose the server location from the Drives pull- down list box
             or use the Network button to connect to the server. Choose OK.
