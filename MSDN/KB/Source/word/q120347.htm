Cross-Reference Loses Caption Number After Updating

 Last reviewed: February 5, 1998
Article ID: Q120347
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

If you update the cross-references in a document, Word 6.0 for Windows loses the caption numbering. The caption label remains, but the caption number is missing.

CAUSE

Bookmarks for captions change (shrink) to include only the caption label (instead of the full text of the caption) when you insert new captions later in the document. Specifically, Word redefines the bookmark so it includes only the caption label, not the caption number. When you update cross-references to these captions, the cross-reference includes only the caption label.

In addition, Word removes the caption number if either of the following is true:

  • You inserted another caption of the same type before the cross-referenced caption.

    -or-

  • There is no text following the cross-referenced caption number.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows. This problem was corrected in versions 6.0a and later of Word for Windows.

WORKAROUND

To work around this problem prior to obtaining a later version of Word, use the following instructions to add a nonbreaking space to the caption text when you insert it in your Word document:

  1. From the Insert menu, choose Caption.

  2. Type the caption text in the Caption dialog box, then press CTRL+SHIFT+SPACEBAR to insert a nonbreaking space.

NOTE: Any trailing character after the caption number other than a normal space prevents Word from removing the caption number when you update cross- references.

The most current version of Word 6.x for Windows is Word 6.0c. To obtain this update, contact the Microsoft Order Desk at (800) 360-7561 and request the Word 6.0c update.
