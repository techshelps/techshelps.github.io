Bin Selection in Word Does Not Work on QMS Printer

 Last reviewed: February 5, 1998
Article ID: Q117342
6.00 6.00a 6.00c WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

If you print a document from Word for Windows using a different bin selection than what is shown in Windows Control Panel to the QMS 1725 printer using the QMS version of the PostScript driver (PSCRIPT.DRV version 2.43 from QMS), the bin selection chosen in Word will be ignored and the bin selection in the Windows Control Panel will be used.

CAUSE

QMS printers use a different version of the PostScript driver than what is shipped with Windows. Both the QMS version of the PostScript driver and the Windows version of the PostScript driver have the same name, so when you install the QMS version of the driver, the Windows version of the driver gets overwritten.

WORKAROUNDS

Method 1: Set the desired bin selection in Windows Control Panel, and then 

          match those same settings from within Word.
Method 2: Use the Windows version of the PostScript driver by following 
          these steps:

          a. Exit Word.

          b. Using the File Manager, rename the PSCRIPT.DRV file to
             something else (it is located in the WINDOWS\SYSTEM
             subdirectory).

          c. Install another PostScript printer, such as the Apple
             LaserWriter.

          Note: Using the Windows PostScript driver will not allow you to
          access all the options on the QMS printer, such as multiple bin
          selections.
Method 3: Contact QMS for an updated version of the QMS PostScript driver. 
          The updated driver can be downloaded from the QMS bulletin board
          (BBS) by calling (205) 633-3632.

MORE INFORMATION

The QMS and PostScript products included here are manufactured by vendors independent of Microsoft; we make no warranty, implied or otherwise, regarding these products' performance or reliability.

Microsoft has not tested this problem in other QMS printers using the QMS version of PostScript driver version 2.43.
