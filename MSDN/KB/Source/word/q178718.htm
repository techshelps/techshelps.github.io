WD: Only First & Last Name Returned from Lotus Notes Data Source

 Last reviewed: January 5, 1998
Article ID: Q178718
The information in this article applies to:
  • Microsoft Word Viewer for Windows 95, version 7.0
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

In a Microsoft Word document, when you use the Address Book icon in the Envelopes and Labels dialog box (on the Tools menu) to insert address information from a Lotus Notes data source, only the first and last names are returned.

MORE INFORMATION

The Lotus Notes products included here are manufactured by Lotus Development Corporation, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

Lotus has reproduced this behavior for Lotus Notes version 4.0, a 32-bit program, using the products listed above. Lotus recommends that you contact them to upgrade to their latest version of Lotus Notes.

Additional information that should be returned from Lotus Notes is the address, city, state, and zip code. These fields are populated correctly when drawn from a Microsoft Exchange address book.

Microsoft Word can import information from a Notes database when you install a Lotus Notes client, the Windows Messaging components, and the Lotus NoteSQL ODBC driver. For information about how to do this, you will need to contact Lotus.
