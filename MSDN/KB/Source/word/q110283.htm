WD: AT Line in Status Bar May Not Display Correct Value

 Last reviewed: February 5, 1998
Article ID: Q110283
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0, 6.0a
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

When editing text within a table cell in normal view, the AT value in the status bar may not display the correct information.

CAUSE

This is caused by the cell flowing from one page to the next. The AT line displays a position value relative to the page the cell began on. This situation occurs only in normal view.

WORKAROUND

Edit the table in page layout view.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows versions 6.0 and 6.0a and Word for the Macintosh version 6.0. This problem was corrected in Word version 6.0c for Windows and Word version 6.0.1 for the Macintosh
