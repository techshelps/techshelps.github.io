WD6X: "Please Locate Custom Dictionary" Error During Spell Check

 Last reviewed: February 18, 1998
Article ID: Q123306
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SYMPTOMS

When running the spelling checker, you may receive the following error message: 

    Please locate custom dictionary.

CAUSE

You are using a custom dictionary, such as a foreign language dictionary, designed to be used with an earlier version of Word for the Macintosh. For example, the foreign language dictionaries, thesauruses, and hyphenation files published by Alki Software for Word 5.0 and 5.1 cannot be used with Word 6.0.

RESOLUTION

Upgrade to a newer version of the custom dictionary.
