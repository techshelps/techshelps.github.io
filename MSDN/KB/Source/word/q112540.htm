WD: Word Dialogs That Cannot Be Displayed Using WordBasic

 Last reviewed: February 2, 1998
Article ID: Q112540

The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

You cannot dimension or display the following Word dialogs using the Dialog statement or Dialog() function: 

   FormatBullet
   FormatBulletsAndNumbering
   FormatMultiLevel
   FormatNumber
   FormatHeadingNumbering
NOTE: Word version 6.x for the Macintosh will dimension the FormatBulletsAndNumbering dialog box without returning an error.

Word 7.x for Windows 95 or Word 6.0 for Windows NT

The following macro displays the BulletsAndNumbering dialog: 

   Sub Main
      ReDim dlg As FormatBulletsAndNumbering
      Dialog dlg
   End sub

Word 6.x

   Sub Main
      Dim dlg As FormatNumber
      Dialog dlg
   End Sub
When the above macro is run, the following error is posted and the Dialog dlg instruction is highlighted: 

   WordBasic Error 509 - This command is not available.
