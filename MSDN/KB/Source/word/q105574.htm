Cannot Convert Word Version 1.x for Windows Macros in Word 6.0

 Last reviewed: February 5, 1998
Article ID: Q105574
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

If you open a Word version 1.x for Windows template in Word version 6.0 for Windows, the following error message may occur: 

   Word is opening a Word version 1.x for Windows template.
   Macros from Word 1.x for Windows cannot be converted and
   will be lost if this file is saved in Word 6.0 for Windows format.
If you choose OK and then save the file in Word 6.0 file format, your Word 1.x macros are lost.

CAUSE

This happens because Word 6.0 does not support Word 1.x macros.

WORKAROUND

To convert the Word 1.x macros for use in Word 6.0, do the following:

  1. Open and save the Word version 1.x for Windows file in Word version 2.0 for Windows. The macros will be saved in version 2.0 format.

  2. Open the file in Word version 6.0 for Windows. The macros will be converted to Word version 6.0 for Windows format.

MORE INFORMATION

Although Word 6.0 does not convert Word 1.x macros, it does convert all styles and glossaries in your Word 1.x templates (glossaries change to autotext entries in Word 6.0).
