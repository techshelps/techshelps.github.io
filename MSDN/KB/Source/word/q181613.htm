WD98: Track Changes Accept Button Does Not Accept Revision

 Last reviewed: February 25, 1998
Article ID: Q181613
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

If you review tracked changes in a document that is protected for tracked changes, the Accept All and Reject All buttons will be available in the Accept Or Reject Changes dialog box. Choosing either button will not Accept or Reject changes in the document.

When the document is protected for Tracked Changes, the Accept All and Reject All buttons should be unavailable (dimmed) because accepting or rejecting tracked changes is not allowed in a document protected for tracked changes.

WORKAROUND

To accept or reject tracked changes, you must unprotect the document by clicking Unprotect Document on the Tools menu and typing in a password (if the document was protected using a password).

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
