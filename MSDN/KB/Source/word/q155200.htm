WD: Envelopes "Feed From" Option Reset in Word Under NT 3.51, 4.

 Last reviewed: July 31, 1997
Article ID: Q155200
The information in this article applies to:
  • Microsoft Word for Windows, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Windows NT operating system versions 3.51, 4.0

SYMPTOMS

The Feed From printing option for envelopes is reset to its default setting when you exit Word. (To locate this option, click "Envelopes and Labels" on the Tools menu, click the Envelopes tab, click Options, and then click Printing Options.)

WORKAROUND

To work around this problem, after you start Word, reset the Feed From printing option before you print envelopes.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
