WD: How to Obtain the Word 97 Converter

 Last reviewed: February 5, 1998
Article ID: Q162214
The information in this article applies to:
  • Microsoft Word 97 for Windows

SUMMARY

In order to open a Word 97 document in an earlier version of Word (6.x and 7.x), you must use a converter. For more information about what happens when you use the converter, see the "Contents of the Wrd97cnv.doc Readme File" section of this article.

The most recent version of the converter (Wrd97cnv.exe) is available in the ValuPack folder included with the Office 97 SR-1 Enterprise Update (the CD version of SR-1).

The converter is also available for download from the Microsoft Software Library: 

 ~ Wrd97cnv.exe (size: 984958 bytes) 

   This file includes a Setup program that installs either
   Msword8.cnv or Mswrd832.cnv on your computer (depending on the version
   of Windows you are running). The Setup program also installs the
   Wrd97cnv.doc Readme document.

   NOTE: Msword8.cnv is the converter for Word 6.0 for Windows.
   Mswrd832.cnv is the converter for Word 7.0 for Windows 95.
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
If you are unable to access the file from the ValuPack folder or from the Microsoft Software Library, see the "How to Obtain This Converter" section of this article.

MORE INFORMATION

Contents of the Wrd97cnv.doc Readme File

The Microsoft Word 97 converter converts documents from Microsoft Word 97 format to Word 95, and Word 6.x for Windows or Windows NT.

For Network Install of Word 97 converter:

  1. Type "source:\wrd97cnv.exe /A" (without quotation marks) and press ENTER, where "Source" is the directory containing Wrd97cnv.exe.

    NOTE: All the files are extracted into the source directory.

  2. Each Workstation will then need to run Setup.exe from the network server location.
