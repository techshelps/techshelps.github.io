WD: "Wrong WWINTL.DLL" Error After Installing Word 6.0a Patch

 Last reviewed: February 5, 1998
Article ID: Q98629
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0a

SYMPTOMS

The following error message may occur after you run the Word 6.0a for Windows Patch (UPGRADE.EXE) to upgrade your existing Word 6.0 installation: 

   The wrong WWINTL.DLL  has been loaded. Make sure that the Word
   directory is in the path.

CAUSE

U.S. (North American) Version of Word 6.0

If you ran the Word 6.0a Patch to upgrade a U.S. version of Word 6.0 for Windows, this error occurs because you received a bad patch disk from Microsoft. Microsoft sincerely apologizes for the inconvenience we have caused our valued customers.

NOTE: This problem does not occur when you run the patch that you downloaded from an online service. It only happens when you run the patch from a disk sent to you by Microsoft.

International Localized Version of Word 6.0

If you ran the Word 6.0a Patch to upgrade an international localized version of Word 6.0 for Windows, this error occurs because the patch is not designed for use with the international localized versions of Word (including the English/U.K. and English/Australian localized versions). The international localized versions of Word require a unique patch.

Please contact the Microsoft subsidiary for your area for information about obtaining a localized version of the Word 6.0a Patch disk. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

   http://www.microsoft.com/worldwide/default.htm

WORKAROUND

If you received a bad Word 6.0a Patch disk, Microsoft is sending you a replacement disk. You don't need to do anything further to obtain this corrected patch disk. If you want to download a correct version of the patch from an online service, see the "How to Download Word 6.0a" section at the end of this article.

To get Word 6.0 running again, whether you used the bad patch disk or upgraded an international localized version of Word, use the following instructions to uninstall and then reinstall Word:

NOTE: You will need your Word 6.0 for Windows program disks for this procedure.

  1. Choose the Word Setup icon in Windows Program Manager to start the Word Setup program.

  2. In the first Microsoft Word 6.0 Setup dialog box, choose the Add/Remove button.

  3. In the Maintenance Installation dialog box, clear the Microsoft Word Option check box. Choose OK when the following message appears: 

          This component is a vital part of this application. If it is not
      installed, this application will not run correctly.

  4. Choose the Continue button and follow the instructions on your screen.

  5. After Setup is complete, choose the Word Setup icon in Windows Program Manager to start Setup again.

  6. In the first Microsoft Word 6.0 Setup dialog box, choose the Add/Remove button.

  7. In the Maintenance Installation dialog box, select the Microsoft Word Option check box, and then choose the Continue button.

  8. Follow the instructions on your screen to complete the reinstallation of Word 6.0.
