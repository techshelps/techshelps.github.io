WD6X:Cannot Use Some GX Printing Commands in WordBasic Macro

 Last reviewed: February 5, 1998
Article ID: Q122499
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1, 6.0.1a

SYMPTOMS

You cannot include the following GX Printing commands in a WordBasic macro in Word for the Macintosh:

  • In the Page Setup dialog box: 

          Scale
      LaserWriter options (same for non-GX printer drivers also)
      Landscape 270 degrees orientation
  • Printing Extension information
  • Print Time information

CAUSE

These commands are handled by the Apple Macintosh operating system, not by WordBasic (the Word macro language).

WORKAROUND

Create an AppleScript command script to address these system-level printing functions. (AppleScript is the Apple common macro language for use with applications that support AppleScript.)

For more information about using Word 6.0 and AppleScript, see Appendix D, "AppleScript" in the "Microsoft Word Developer's Kit."

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
