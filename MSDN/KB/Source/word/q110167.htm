"Include Subdirectories" Option Does Not Remain Selected

 Last reviewed: February 5, 1998
Article ID: Q110167
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

When you run or dimension and call the WordBasic FindFile dialog, the Include Subdirectories check box in the Search and the Advanced Search dialog box options may not remain selected. By contrast, the dialog box works correctly when you choose Find File from the File menu.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

There is no WordBasic workaround available at this time. To manually work around this problem, select or clear the Include Subdirectories check box in the Search dialog box immediately prior to choosing OK to perform the search.
