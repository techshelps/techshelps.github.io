DMM: Protected Address List Causes Unusual Behavior

 Last reviewed: March 9, 1998
Article ID: Q181927
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows

SYMPTOMS

When you use a password protected file with Direct Mail Manager, you may not receive the results you expect.

Case 1

When you try to open a password protected address list in Direct Mail Manager, you may receive either an error message, the address list won't open, or you get unusual behavior.

NOTE: This article discusses password protected address lists from Word 97, Access 97 and Excel 97.

Case 2

When you try to open a Word address list that is protected for comments or forms, you receive an error message and cannot import your address list into Direct Mail Manager.

WORKAROUND

Remove the password or document protection from your address list file before you open it in Direct Mail Manager.

MORE INFORMATION

Word 97

Password Protected Address List:

When you open a password protected Word document in Direct Mail Manager, a Word session will start and prompt you for the password for the file. To continue with Direct Mail Manager after entering the password, you must switch back to Direct Mail Manager by clicking the Direct Mail Manager button on the Windows taskbar. When you click Next to import your address list records, the Word session will end and the Direct Mail Manager process will continue as usual.

NOTE: If you exit the Word session after typing the password for the address list, you will quit the Word session and Direct Mail Manager.

If you type an incorrect password to open the address list, the following error message will be displayed: 

   Error (5408) - The password is incorrect. Word cannot open the
   document.
   (<drive>:\<path>\<filename>)

   This error occurred when attempting to open the address list file
   '<drive>:\<path>\<filename>'. Please check that (1) the file exists,
   (2) you have selected a file format supported by Direct Mail Manager,
   and (3) it is not locked or your access is otherwise restricted.
To continue, click OK to close the error message and then click Next in Direct Mail Manager to open the address list. A Word session will start up and Word will again prompt you for the address list password. Type in the correct password and click OK.

Address List Protected for Comments 

   When you open an address list in Direct Mail Manager that has been
   protected for Comments, you will receive the following error message:

      Error (4605) - This method or property is not available because the
      document is locked for editing.

      This error occurred when attempting to review the fields in the file.
      Please check that you have selected a file that is formatted
      correctly and it is not locked or your access is otherwise
      restricted.

   When you click OK to close the error message, you can then select a
   different address list.
Address List Protected for Forms 

   When you open an address list in Direct Mail Manager that has been
   protected for Forms, you will receive the following error message:

      Error (4605) - This method or property is not available because the
      object refers to a protected area of the document.

      This error occurred when attempting to review the fields in the file.
      Please check that you have selected a file that is formatted
      correctly and it is not locked or your access is otherwise
      restricted.

   When you click OK to close the error message, you can then select a
   different address list.

Access 97

When you try to open a password protected Access database file in Direct Mail Manager, the following error message will appear: 

   Error (3031) - Not a valid password.

   This error occurred when attempting to open the address list file
   '<drive>:\<path>\<filename>'. Please check that (1) the file exists,
   (2) you have selected a file format supported by Direct Mail Manager,
   and (3) it is not locked or your access is otherwise restricted.
When you click OK to close the error message, you can then select a different address list.

Excel 97

When you try to open a password protected Excel worksheet in Direct Mail Manager, the following error message will appear: 

   Error (3274) - External table isn't in the expected format.

   This error occurred when attempting to open the address list file
   '<drive>:\<path>\<filename>'. Please check that (1) the file exists,
   (2) you have selected a file format supported by Direct Mail Manager,
   and (3) it is not locked or your access is otherwise restricted.
When you click OK to close the error message, you can then select a different address list.
