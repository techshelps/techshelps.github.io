Word Assistant Font Information (Including Reinstallation)

 Last reviewed: February 5, 1998
Article ID: Q110228
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Word Assistant version 1.0

MORE INFORMATION

The typefaces included with Word Assistant will not be uninstalled through the Maintenance Setup Remove All command.

If a font is damaged, delete the font files from Control Panel Fonts. This will delete the .TTF font file. Word Assistant also installs an .FIF file. This file is not deleted through Control Panel Fonts.

Use the following steps to reinstall the Word Assistant fonts:

  1. Choose the Add/Remove button in Word Assistant Setup (Maintenance Mode).

  2. Clear the check box for TrueType Fonts and choose Continue.

    Word Assistant Setup will display the message 

          This component is a vital part of this
      application. If it is not installed,
      this application will not run correctly.

    followed by: 

          You have chosen to remove 1 component.
      Are you sure you want to remove it?

    Choose Yes.

    This will delete the .FIF files in the <Windows>\SYSTEM subdirectory and alert Setup that the TrueType fonts component has been removed.

  3. Run Word Assistant Setup again and choose the Add/Remove option.

  4. Select the TrueType Fonts option and choose Continue.

    This will install all of the .TTF and .FIF files that are not found in the <Windows>\SYSTEM subdirectory.
