Office 4.2 Upgrade Does Not Contain Word 6.0a

 Last reviewed: February 6, 1998
Article ID: Q113484
6.00 6.00a WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a
  • Microsoft Office for Windows, version 4.2

The full retail package of Microsoft Office version 4.2 for Windows ships with Word 6.0a for Windows.

However, the Office 4.2 upgrade package does not contain either Word 6.0 or 6.0a. The Office 4.2 upgrade only contains PowerPoint 4.0 and Microsoft Office Manager (MOM) 4.2. The Office 4.2 upgrade does nothing to your Word 6.0 installation. That means you still need to obtain and run the Word 6.0a patch to upgrade your Word 6.0 installation.

IMPORTANT: There are problems running the Word 6.0a Patch after you upgrade to Office 4.2 using the upgrade package.

For additional information, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q113483
   TITLE     : Word 6.0a Patch Fails After Installing Office 4.2
