Merge Button Unavailable When All MERGEFIELDs in Text Boxes

 Last reviewed: February 5, 1998
Article ID: Q115789

The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

The Merge button in the Mail Merge Helper dialog box is unavailable (dimmed) when the MERGEFIELD fields are contained in text boxes, headers or footers in the main document.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUNDS

Method 1: Although the Merge button is unavailable when all MERGEFIELDs are 

          contained in text boxes, you can process the merge by choosing
          the Merge To New Document or Merge To Printer button on the
          toolbar.
Method 2: Placing at least one field outside of the text box will make the 
          Merge button in the Mail Merge Helper available. This field can
          be formatted as hidden if you do not want the result to display
          and print.
