WD: How to Unlink Form Fields

 Last reviewed: February 3, 1998
Article ID: Q137439
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

The Unlink Field keyboard shortcut, CTRL+SHIFT+F9(command+SHIFT+F9 in Word for the Macintosh) does not unlink form fields. That is, it does not convert form field results to text.

WORKAROUND

This article describes a macro that converts all form fields to text. Check boxes are converted to either a "1" (if the box was selected) or a "0" (if the box was not selected). If you will use the macro frequently, you can assign it to a toolbar, menu, or keyboard shortcut.

   Sub MAIN
      Dim DocState
      If CommandValid("ToolsUnprotectDocument") Then
          ToolsUnprotectDocument .DocumentPassword = ""
          DocState = 1
      EndIf
      EditSelectAll
      UnlinkFields
      StartOfDocument
      ViewFieldCodes 1
      EditFind .Find = "^d"
      While EditFindFound()
          ViewFieldCodes
          EditCut
          EditPasteSpecial .DataType = "Text"
          ViewFieldCodes 1
          EditFind
      Wend
      If docstate = 1 And CommandValid("ToolsProtectDocument") Then
          ToolsProtectDocument .DocumentPassword = "", .NoReset = 1, \
          .Type = 2
      EndIf
   End Sub
