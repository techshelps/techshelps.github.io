WD: How to Change the Page Size to A4 for Custom Labels

 Last reviewed: February 5, 1998
Article ID: Q115412

The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SUMMARY

When your labels are arranged on an A4-sized sheet, rather than a Letter- sized sheet, do the following to change the label page size:

Word 97

  1. On the Tools menu, click Envelopes And Labels.

  2. Select the Labels tab, then click the Options button.

  3. In the Label Products list, select Avery A4 And A5 Sizes.

  4. In the Product Number list, select a predefined label that has an A4
page size (L7160, for example).

  1. Click the Details button.

  2. In the Label Name box, type a name for your labels, change the settings
to match the custom labels, and then click OK.

  1. Click OK.

Word 6.x, 7.x

  1. On the Tools menu, click Envelopes And Labels.

  2. Select the Labels tab, then click the Options button.

  3. In the Label Products list, select Avery Pan European.

  4. In the Product Number list, select a predefined label that has an A4 page size (L7160, for example).

  5. Click the Details button, change the settings to match the custom labels, and then click OK.

  6. Word will display a dialog box asking you to save your changes; click OK.

The Label Information section of the Labels dialog box will now display A4 as the page size. To change it back to Letter, repeat the steps above except in step 4, choose a predefined label that has a Letter page size defined.
