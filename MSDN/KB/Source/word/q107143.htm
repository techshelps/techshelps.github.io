Word Hangs After Edit Repeat Typing Operation

 Last reviewed: February 5, 1998
Article ID: Q107143
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

Word for Windows may hang (stop responding) the second time you choose the Repeat Typing command from the Edit menu or press F4.

CAUSE

Word hangs if you type some text, insert a text box, and then run the Repeat Typing command twice. Word does not hang if you press ENTER to insert a paragraph mark before you insert the text box.

WORKAROUND

Press ENTER to insert a paragraph mark before you insert a text box. For example, if you use the following steps, Word does not hang:

  1. In a new Word document, type "hello" (without the quotation marks).

  2. Press ENTER.

    IMPORTANT: If you skip this step, Word hangs after step 7.

  3. Display the Drawing toolbar by choosing the Drawing button on the Standard toolbar.

  4. Choose the Text Box button on the Drawing toolbar. Drag to create a text box.

  5. In the text box, type "there" (without the quotation marks).

  6. Select "hello" in the document.

  7. From the Edit menu, choose Repeat Typing. Repeat this step.

    Note: If you skipped step 2 above, Word hangs now.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

"Microsoft Word User's Guide," version 6.0, Chapter 16, "Important and Creating Graphics," "Creating a Text Box" section
