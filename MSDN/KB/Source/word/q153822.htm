WD6.0.1: Internet Assistant Setup Not Completed Successfully

 Last reviewed: February 3, 1998
Article ID: Q153822
The information in this article applies to:
  • Microsoft Internet Assistant for Word for the Macintosh, version 2.0
  • Microsoft Word for the Macintosh, version 6.0.1

SYMPTOMS

When you run Internet Assistant Setup, you may receive one of the following error messages 

   Internet Assistant setup was not completed successfully.

    -or-

   Errors have been detected during AppSearch
followed by: 

   Internet Assistant Setup was not completed successfully.

CAUSE

This error message may occur in two situations.

Case 1: An Older Version of Internet Assistant Exists on Your Computer

Files from an older version of Internet Assistant are still installed on your computer.

Case 2: The Microsoft Word Setup Folder is Corrupted

The error message: "Errors have been detected during AppSearch" can also occur if an earlier installation of Microsoft Word for Macintosh, version 6.0 or 6.0.1 exists on your computer, and the Microsoft Word Setup data file is corrupted.

WORKAROUND

To work around this problem, use the appropriate method for your situation.

Case 1: Remove Previous Versions of Internet Assistant

Run the Setup program for the earlier version of Internet Assistant and click Remove All. Then, run the newer Internet Assistant Setup program.

If you are unable to run the Setup program for the earlier version of Internet Assistant, manually delete the files listed in the following table from the hard disk, and then run Setup for the updated version of Internet Assistant. 

   File/Folder to Remove                        Located in Folder
   --------------------------------------------------------------

   Internet Assistant Folder                    Microsoft Word
   Hypertext Markup Language                    Word Extensions
   HTML                                         Templates
   HTML view                                    Templates
   Webview                                      Templates
   Internet Assistant                           Word Startup Folder (6)
   Internet Assistant folder                    Preferences

Case 2: Remove the Setup Folder for Word and Reinstall

Drag the Setup folder to the trash can, and reinstall Microsoft Word.
