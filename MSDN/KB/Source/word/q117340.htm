GP Fault After Choosing Frame from the Format Menu

 Last reviewed: February 5, 1998
Article ID: Q117340
6.00 6.00a 6.00c WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

After you link an object in Word for Windows, a general protection (GP) fault in WINWORD.EXE occurs if you then choose Frame from the Format menu. This problem occurs only when a frame surrounds the linked object and the caption below it.

Note: Changing the video driver doesn't have an effect.

STATUS

Microsoft has confirmed this to be a problem in the versions of Word listed above. This problem was corrected in Word version 7.0 for Windows 95.
