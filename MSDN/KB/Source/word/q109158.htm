Using ToolsAdvancedSettings to View WINWORD6.INI and WIN.INI

 Last reviewed: July 30, 1997
Article ID: Q109158
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Windows operating system version 3.1

SUMMARY

This article describes how you can use the ToolsAdvancedSettings macro command to view Word for Windows entries in the WINWORD6.INI and WIN.INI files.

The ToolsAdvancedSettings macro corresponds to the Win.ini category in the Tools Options dialog box for Word versions 2.x for Windows.

MORE INFORMATION

The advantage of using the ToolsAdvancedSettings command instead of opening the Tools Options dialog box and selecting the File Locations tab is that the Tools Advanced Settings dialog box lists the actual setting instead of simply describing it. For example, Word displays "DOC-PATH" in the Tools Advanced Settings dialog box, whereas Word describes the DOC-PATH as "Documents" under File Locations in the Tools Options dialog box. Furthermore, you can only modify the listed settings in the Tools Options dialog box, whereas you can use the ToolsAdvancedSettings command to add any valid setting to the Word sections of your WINWORD6.INI or WIN.INI.

Use the following procedure to run the ToolsAdvancedSettings command:

  1. From the Tools menu, choose Macro.

  2. From the Macros Available In list, select Word Commands. From the Macro Name list, select ToolsAdvancedSettings and then choose the Run button.

  3. In the Advanced Settings dialog box, select the WIN.INI or WINWORD6.INI section you want to modify or view from the Categories list.

Note: You can add the ToolsAdvancedSettings command to your Tools menu for easy access. For online instructions, double-click the Help button on the Standard toolbar. Then type "customize" (without the quotation marks).
