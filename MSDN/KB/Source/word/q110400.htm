Delete Button Does Not Change in Grammar Checker

 Last reviewed: February 5, 1998
Article ID: Q110400
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

If you are grammar checking a sentence that contains a word the grammar checker suggests you delete, you can choose the Delete button to delete the highlighted word. If instead, you manually edit the sentence in the Sentence box, the Delete button in the dialog does not change back to a Change button as expected. Instead, the Delete button functions as the Change option.

For example, the grammar checker will suggest in the following sentence that the sentence is too wordy and that you should consider deleting the word "as" to correct the problem: 

   I consider you as my most interesting friend.
If instead of choosing the Delete button, you manually delete the word "as" in the Sentence box, the Delete button remains available.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
