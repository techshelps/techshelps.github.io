Preview Page Contains No Lines in File Page Setup Dialog Box

 Last reviewed: February 5, 1998
Article ID: Q114995
6.00 6.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a
  • Microsoft Windows operating system version 3.1

SYMPTOMS

In Word for Windows, the page in the Preview area of the File Page Setup dialog box may not display any lines.

CAUSE

No lines display when the Blue Background, White Text option is selected. (To locate this option, choose Options from the Tools menu and select the General tab.)

Word uses the Window Text color to display the lines on the page in the Preview area. When you turn on the Blue Background, White Text option, Word adds a setting to your WINWORD.OPT file that changes the Window Text color to white while you are working in Word. In the File Page Setup dialog box, Word draws white lines on a white background, so the lines are not visible.

Note: Word uses this special WINWORD.OPT Window Text setting instead of changing the Window Text color in Windows Control panel, where it would affect all your Windows-based applications.

WORKAROUNDS

Method 1: Use the following instructions to turn off the Blue Background, 

          White Text option:

          a. From the Tools menu, choose Options.

          b. Select the General tab.

          c. Clear the Blue Background, White Text option and choose OK.
Method 2: Use the following procedure to change your Windows Background 
          color to blue in Windows Control Panel:

          a. In Windows Control Panel, choose the Colors icon.

          b. Choose the Color Palette button.

          c. From the Screen Element list, select Windows Background.

          d. Under Basic Colors, select blue, and then choose OK.
