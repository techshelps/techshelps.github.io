WD6X: How to Create a Work Menu in Word 6.0 for the Macintosh

 Last reviewed: February 5, 1998
Article ID: Q120935
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

You can add frequently used documents to a custom menu in Word 6.0. However, the command in Word versions 4.0 and 5.x for the Macintosh that automatically create a custom Work menu (namely, COMMAND+OPTION+=), do not work in Word 6.0 for the Macintosh.

MORE INFORMATION

To create a custom or Work menu in Word 6.0, follow these steps:

  1. On the Tools menu, click Customize.

  2. Click the Menus tab.

  3. Click the menu bar.

  4. Type the name of the menu you want to add (for example, Work).

  5. Select the position on the menu bar where you would like the menu to appear (the default is the last position).

  6. Click Add.

  7. Click Close.

To add a document to a custom menu in Word 6.0, follow these steps:

  1. Open the document you want to add to the custom menu.

  2. On the Tools menu, click Customize.

  3. Select the Menus tab.

  4. On the Categories list, select File.

  5. On the Commands list, select FileOpenFile.

  6. On the Change What list, select your custom menu name (for example, Work).

  7. On the FileOpenFile list (under the Menu Bar), select the name of the open document that you want to add to your custom menu.

  8. On the Save Changes In list, select the template you would like this change saved in. (Select your Normal template if you would like the custom menu available in all documents.)

  9. Click Add and then click Close.

To add an AutoText entry to a custom menu in Word 6.0, follow these steps:

  1. On the Tools menu, click Customize.

  2. Click the Menus tab.

  3. From the Categories list, select AutoText.

  4. From the Commands list, select the AutoText entry that you want displayed on the menu.

  5. From the Change What list, select your custom menu name (for example, Work).

  6. On the Save Changes In list, select the template you would like this change saved in. (Select your Normal template if you would like the custom menu available in all documents.)

  7. Click Add.

  8. Click Close.
