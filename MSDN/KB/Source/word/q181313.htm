WD98: How to Open HTML Files Directly into Source View

 Last reviewed: February 19, 1998
Article ID: Q181313
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SUMMARY

When you open a HyperText Markup Language document in Microsoft Word 98 Macintosh Edition, it open in Online Layout view by default. To view and edit the HTML code, you can open the HTML file as a text only document.

MORE INFORMATION

To open an HTML file in Microsoft Word 98 as a text only document, follow these steps:

  1. On the Tools menu, click Preferences.

  2. Click the General tab.

  3. Click to select the "Confirm conversion at Open" check box, and then click OK.

  4. On the File menu, click Open.

  5. Select the HTML file that you want to open, and then click Open.

  6. In the Convert File From list, click Text Only, and then click OK.
