DrawReshape

Syntax:

DrawReshape

Example:

The following sample macro draws a freeform drawing object in the shape of the letter "N". Next, the macro changes the handles on the freeform drawing object between the vertices of the freeform shape (where you use the handles to reshape the object) and the bounding rectangle (where you use the handles to scale or resize the object).

Before running the macro place the insertion point in a new blank document and select Page Layout view. 

   Sub MAIN
      DrawFreeformPolygon
      Dim pointsArray$(4, 2)
      pointsArray$(1, 1) - "150 pt"
      pointsArray$(1, 2) - "200 pt"
      pointsArray$(2, 1) - "150 pt"
      pointsArray$(2, 2) - "100 pt"
      pointsArray$(3, 1) - "200 pt"
      pointsArray$(3, 2) - "200 pt"
      pointsArray$(4, 1) - "200 pt"
      pointsArray$(4, 2) - "100 pt"
      DrawSetPolyPoints 4, pointsArray$()
      DrawReshape
      MsgBox "The handles are on the vertices."
      DrawReshape
      MsgBox "The handles are on the bounding rectangle."
   End Sub
