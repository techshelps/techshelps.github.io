System Requirements for Internet Assistant for Word

 Last reviewed: September 24, 1997
Article ID: Q136383
The information in this article applies to:
  • Microsoft Internet Assistant for Word, versions 1.0, 1.0z, 2.0, 2.0z, 2.03z, 2.04z
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for Windows, versions 6.0a, 6.0c
  • Microsoft Word for the Macintosh, versions 6.0.1, 6.0.1a

SUMMARY

This article describes the system requirements for Microsoft Internet Assistant for Word.

MORE INFORMATION

Internet Assistant, Version 2.03z

The following are required to run Internet Assistant, version 2.03z, for Word for Windows:

  • A personal computer with an 80386DX or higher processor (486 Recommended).
  • Microsoft Windows 95 or Microsoft Windows NT Workstation version 3.51 or later (does not run on earlier versions of Windows).
  • Microsoft Word for Windows 95, version 7.0 or 7.0a.
  • 6 megabytes (MB) of RAM memory (8 MB recommended).
  • 2 MB of free hard disk space.
  • Microsoft Mouse or compatible pointing device.
  • VGA or higher-resolution video adapter (SVGA 256-color recommended).

Web viewing support requires an existing Internet connection, including TCP/IP and Windows Socket support. Internet connectivity can be obtained separately from an Internet service provider.

Internet Assistant, Versions 2.0z and Later

The following are required to run Internet Assistant versions 2.04z and laster for Word for Windows:

  • A personal computer with an 80386 or higher processor.
  • Microsoft Word for Windows 95, version 7.0.
  • 8 MB of RAM.
  • 2MB of free hard disk space.
  • VGA or other high-resolution display.
  • Microsoft Windows 95 or Microsoft Windows NT version 3.51.
  • Mouse or other Windows pointing device.

Internet Assistant's browsing, link validation, and Copy Hyperlink features require an Internet connection. To use these features, you must have the following installed or connected to your computer:
  • A 32-bit Winsock (Wsock32.dll) compliant TCP/IP product (consult your TCP/IP vendor).
  • A physical or dial-up connection to the Internet.

IMPORTANT: If you are using a 16-bit Winsock.dll, you cannot connect to the Internet with Internet Assistant. Contact your Internet service provider for instructions on upgrading to Wsock32.dll.

Internet Assistant, Version 2.0

The following are required to run Internet Assistant version 2.0 for Word for the Macintosh:

  • Apple Macintosh with a 68030 or 68040 processor (68040 recommended).
  • System 7 operating system or later (System 7.5.3 recommended).
  • Microsoft Word for the Macintosh, version 6.0.1 or 6.0.1a.
  • 6 MB of RAM (8 MB recommended).
  • Hard-disk space required: 3 MB during installation (Internet Assistant uses 1 MB after it is installed).

Web viewing support requires an existing Internet connection, including TCP/IP support. Internet connectivity can be obtained separately from an Internet service provider. Typically this means MacTCP (version 2.06 recommended) or Open Transport (version 1.1 recommended).

Internet Assistant, Version 1.0z

The following are required to run Internet Assistant version 1.0z for Word for Windows:

  • A personal computer with an 80386 or higher microprocessor.
  • Microsoft Word for Windows, version 6.0a or later. You can obtain the 6.0a upgrade patch on the Internet in the following location: 

          ftp://ftp.microsoft.com/softlib/mslfiles/word60a.exe
  • 6 MB of RAM (8 MB recommended).
  • 2 MB of free hard disk space.
  • MS-DOS version 3.0 or later.
  • Microsoft mouse or compatible pointing device.
  • Microsoft Windows version 3.1 or later; Microsoft Windows for Workgroups version 3.1 or later; Microsoft Windows NT; or Microsoft Windows for Pen Computing.

Web viewing support requires an existing Internet connection, including TCP/IP and Windows Socket support. Internet connectivity can be obtained from an Internet service provider.
