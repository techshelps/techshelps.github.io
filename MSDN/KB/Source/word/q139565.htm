System Hangs During Spell Check or Thesaurus

 Last reviewed: August 5, 1997
Article ID: Q139565
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

Word hangs when you check spelling or use the thesaurus on a shared network installation.

CAUSE

The administrator changed the server path to the spelling and thesaurus files, and your Winword6.ini has not been updated.

RESOLUTION

Edit the Winword6.ini file to reflect the new location of the spelling and thesaurus files. In the Winword6.ini file, look for [Microsoft Word] section, and find the following lines: 

   Spelling1033,0=C:\WINDOWS\MSAPPS\PROOF\MSSPEL2.DLL,C:\WINDOWS\MSAPPS
                  \PROOF\MSSP2_EN.LEX
   Spelling2057,0=C:\WINDOWS\MSAPPS\PROOF\MSSPEL2.DLL,C:\WINDOWS\MSAPPS
                  \PROOF\MSSP2_EN.LEX
   Thesaurus1033,0=C:\WINDOWS\MSAPPS\PROOF\MSTHES.DLL,C:\WINDOWS\MSAPPS
                  \PROOF\MSTH_AM.LEX
Correct each of the path statements, ensuring that it points to the new server location.
