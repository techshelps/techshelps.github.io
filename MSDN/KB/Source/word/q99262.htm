Foreign Language Translation Software Available

 Last reviewed: February 6, 1998
Article ID: Q99262
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SUMMARY

There are several translation packages available that allow you to translate documents between English and other languages. Some packages are unidirectional (for example, from English to Spanish only) and some are bidirectional (for example, from English to Spanish and also from Spanish to English).

This article lists several manufacturers and briefly describes their translation packages.

MORE INFORMATION

MicroTac Software, Inc.

MicroTac Software produces language translation software that bidirectionally translates the following languages:

  • English
  • Spanish
  • French
  • German

MicroTac also has a one-way translator from Italian to English.

MicroTac's translation software may only be used with Word 2.0. Contact MicroTac for information about available upgrades.

Toltran Limited

Toltran Limited produces Foreign Correspondent and Toltran Professional version 2.0 unidirectional and bidirectional language translators. 

   Foreign Correspondent
   ---------------------

   This product is a low-end, MS-DOS-based, unidirectional translator
   that is sold in three separate modules:

    - English to Spanish
    - Spanish to English
    - French to English

   Toltran Professional 2.0
   ------------------------

   This product is a high-end, MS-DOS-based, bidirectional translator
   that features the ability to translate from any source language to
   any destination language that is available with the product. For
   instance, you can translate from German to French or from Spanish
   to German. The languages available are:

    - English
    - German
    - French
    - Spanish

   Other features include a lexicon for adding words, and an on-line
   thesaurus.

Globalink

Globalink produces two foreign language translating products that operate in the MS-DOS environment: Power Translator 2.0 and GTS Power Translator Professional 3.0. 

   Power Translator 2.0
   --------------------

   This product bidirectionally translates the following languages:

    - English
    - French
    - German
    - Spanish

    GTS Power Translator Professional 3.0
    -------------------------------------

    This product bidirectionally translates the following languages:

     - English
     - French
     - German
     - Russian
     - Spanish
The main difference between the two products is that the GTS Power Translator Professional 3.0 comes with the capability of adding subject dictionaries.

Globalink also makes Macintosh translation software packages.

Linguistic Products

Linguistic Products produces PC Translator 3.4, which bidirectionally translates the following languages:

  • English
  • Danish
  • French
  • Italian
  • Spanish
  • Swedish

PC Translator unidirectionally translates the following language combinations:
  • English to Portuguese
  • German to English
  • Norwegian to English

PC Translator preserves document formats for Microsoft Word in rich-text format (RTF) and runs in the MS-DOS environment. For more information on PC Translator, contact:
