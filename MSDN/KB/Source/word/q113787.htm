Certain Wizards Print and Preview Blank Pages

 Last reviewed: February 5, 1998
Article ID: Q113787
6.00 6.00a 6.00c WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

A document created using the Newsletter, Award, Calendar, or Pleading Wizards may print blank pages and display in print preview incorrectly if the Drawing Objects option is not selected.

CAUSE

The Newsletter, Award, Calendar, and Pleading Wizards all use the drawing layer feature. If the Drawing Objects option is disabled, the print preview and normal views will not display portions of the document, and sometimes the entire document will be blank. Printing may also be affected.

By design, normal view will not display objects on the drawing layer. Page layout view is not affected.

RESOLUTION

Use the following procedure to correct this problem:

  1. From the Tools menu, choose Options.

  2. Select the Print tab.

  3. Under "Include with Document," select the Drawing Objects check box.

  4. Choose OK.

MORE INFORMATION

If Drawing Objects is not selected, the following symptoms will occur with the listed Wizard:

Documents Created with the Award Wizard

  • Do not display at all in print preview
  • Print blank pages

Documents Created with the Calendar Wizard

  • Do not display at all in print preview
  • In normal view, only page breaks appear
  • Print blank pages

Documents Created with the Pleading Wizard

  • Do not display the line numbers and vertical lines in print preview
  • Do not display the Attorney Name, Company Name, Street Address, City/State/Province/Zip/Postal Code, Telephone #, Fax #, and Attorney For: information in the top-left corner of the document, when in print preview
  • Will not print the above listed objects
