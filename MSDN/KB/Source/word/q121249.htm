Word for Windows NT Features

 Last reviewed: February 5, 1998
Article ID: Q121249
The information in this article applies to:
  • Microsoft Word for Windows NT, version 6.0

SUMMARY

Word 6.0 for Windows NT is an adaptation of Word 6.0 for Windows. All of the new features available in Word 6.0 for Windows NT were added specifically to take advantage of Windows NT itself, namely:

  • 32-bit architecture
  • Long filename support
  • Multithreaded printing
  • Registry support

MORE INFORMATION

Word for Windows NT offers the following features over its 16-bit Word for Windows counterpart:

32-Bit Architecture

Word for Windows NT is written to the Win32 API specification and is, therefore, a full 32-bit application. This has allowed enhancements to Word for Windows NT's capacity. Support is now available for larger master documents and macros, for instance.

Long Filename Support

Word for Windows NT supports up to 255 characters for document filenames. All Word dialog boxes that involve filenames, such as File Open, are wider to accommodate longer filenames as well.

Multithreaded Printing

The background printing feature in Word for Windows NT runs as a separate thread. This makes working on documents and background printing smoother.

Registry Support

Word for Windows NT stores all its setting in the Windows NT Registry. Because of this, multiple configurations of Word can exist on the same computer for different users. Word will also support user profiles allowing users to move from one machine to another and continue to use their same Word configuration.
