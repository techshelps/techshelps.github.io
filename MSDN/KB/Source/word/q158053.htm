All Forms of Words Marked With Index Entry Field

 Last reviewed: February 5, 1998
Article ID: Q158053
The information in this article applies to:
  • Microsoft Word Windows 95, versions 7.0, 7.0a

SYMPTOMS

When you use Mark All to mark a word for an index entry, other words containing the text string of the main entry are also marked. For example, if you are trying to mark the word "field" (without the quotation marks), and your document contains words that include the text string, "field"— such as "infield", "outfield", "fields", or "field-work"--these words are also be marked.

This problem does not occur in previous versions of Word.

WORKAROUND

To work around this problem, mark each index entry one at a time instead of using the Mark All feature.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word for Windows 95, version 7.0 and 7.0a.
