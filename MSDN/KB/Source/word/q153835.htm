WD6X: Errors Starting MacWord After Installing Empowerment Pack

 Last reviewed: February 5, 1998
Article ID: Q153835
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft Empowerment Pack for the Mac, version 1.0

SYMPTOMS

When you update to Word 6.0.1a from the Empowerment Pack, you may receive either of the following error messages when you attempt to start Word: 

   The application "Microsoft Word" could not be opened because
   "Microsoft_OLE2" could not be found.

    -or-

   The application "Microsoft Word" could not be opened because
   "MicrosoftDialogLib" could not be found.

CAUSE

This problem occurs when the following conditions are true:

  • You are using a Power Macintosh 

        -and-
  • You had only the non-Power Macintosh version of Word(that is, the 68K version of Word rather than the Power Macintosh or FAT version of Word) installed on your computer prior to updating to version 6.0.1a.

When you run the Power Macintosh or FAT versions of Word 6.0.1a on a Power Macintosh, certain OLE files are necessary to start the program. The Empowerment pack version of Word (the FAT version) does not install these required OLE files on your computer.

NOTE: These errors can be caused by other factors also. For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q142097
   TITLE     : Word Fails to Start or Produces OLE Error Message

WORKAROUND

To correct this problem, use the appropriate method below.

Method 1

If You Have the Word 6.0 PowerPC Upgrade Disks or Word 6.0.1, follow these steps:

  1. Remove (delete) the 6.0.1a version of Word from you computer.

  2. Install Word for the Power Macintosh on your computer.

  3. From the Empowerment Pack, run the Word 6.0.1a Update again.

Method 2

If you do not have Word 6.0 PowerPC upgrade disks or the 6.0.1 version of Word, follow these steps:

  1. Remove the 6.0.1a version of Word from your computer.

  2. Temporarily reinstall the 68K Macintosh version of Word.

  3. Call the Microsoft Order Desk at (800) 360-7561 to obtain the full Word 6.0.1a upgrade.

If you are outside the United States, contact your local subsidiary. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

   http://www.microsoft.com/worldwide/default.htm
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
