Cannot Find Thesaurus File MSTH232.DLL

 Last reviewed: July 31, 1997
Article ID: Q147956
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a

SYMPTOMS

The following error message may appear when you run the Thesaurus in Microsoft Word for Windows 95: 

   Word cannot find the Thesaurus file MSTH232.DLL for English (UK)

CAUSE

This error message will occur when all of the following are true:

  • You are running Microsoft Word for Windows 95, version 7.0 or 7.0a.

    -and-

  • You are running Microsoft Windows NT 3.51 or Windows 95.

    -and-

  • The language in the document is set to English (UK).

    -and-

  • You run the Thesaurus on the document.

WORKAROUND

To work around this problem, change the language of the document to English (US) as a temporary setting and run the Thesaurus.

To change the language of the document

  1. With the document open in Word, click Select All on the Edit menu.

  2. On the Tools menu, click Language.

  3. Under Mark Selected Text As, select English (US) and click OK.

  4. On the Tools menu, click Thesaurus.

    The Thesaurus will check your document.

  5. On the Edit menu, click Select All.

  6. On the Tools menu, click Language.

  7. Under Mark Selected Text As, select English (UK), and click OK.
