WD98: Close All Doesn't Close Multiple HTML Documents

 Last reviewed: March 16, 1998
Article ID: Q182467
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you have multiple HTML documents open, the Close All command (accessed by holding down SHIFT while choosing the File menu) closes only the current document instead of all open documents.

WORKAROUND

Close each of the HTML documents individually.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
