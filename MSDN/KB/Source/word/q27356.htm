WD: Character Formatting Keyboard Commands

 Last reviewed: November 17, 1997
Article ID: Q27356
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 4.0, 5.0, 5.1, 6.0, 6.0.1, 6.0.1, 6.0.1a

SUMMARY

The following is a quick reference guide that lists the keyboard commands for formatting characters in Microsoft Word for the Macintosh. 

For This Format     Word 6.x            Word 5.x          Word 4.0
---------------     --------            --------          -------

Bold                COMMAND+B           COMMAND+B         COMMAND+SHIFT+B
Italic              COMMAND+I           COMMAND+I         COMMAND+SHIFT+I
Underline           COMMAND+U           COMMAND+U         COMMAND+SHIFT+U
Word Underline      COMMAND+SHIFT+W     COMMAND+SHIFT+]   COMMAND+SHIFT+]
Double Underline    COMMAND+SHIFT+D     COMMAND+SHIFT+[   COMMAND+SHIFT+[
Dotted Underline    NO KEYSTROKE        COMMAND+SHIFT+\   COMMAND+SHIFT+\
Strikethrough       NO KEYSTROKE        COMMAND+SHIFT+/   COMMAND+SHIFT+/
Outline             NO KEYSTROKE        COMMAND+SHIFT+D   COMMAND+SHIFT+D
Shadow              NO KEYSTROKE        COMMAND+SHIFT+W   COMMAND+SHIFT+W
Small Caps*         COMMAND+SHIFT+K     COMMAND+SHIFT+H   COMMAND+SHIFT+H
All Caps*           COMMAND+SHIFT+A     COMMAND+SHIFT+K   COMMAND+SHIFT+K
Hidden              COMMAND+SHIFT+H     COMMAND+SHIFT+X   COMMAND+SHIFT+X
Plain text          COMMAND+SPACEBAR    COMMAND+          COMMAND+
                    or COMMAND+SHIFT+Z  SHIFT+SPACEBAR    SHIFT+SPACEBAR
Font change         COMMAND+SHIFT+F     COMMAND+SHIFT+E   COMMAND+SHIFT+E
Increase font size  COMMAND+SHIFT+>     COMMAND+SHIFT+>   COMMAND+SHIFT+>
Decrease font size  COMMAND+SHIFT+<     COMMAND+SHIFT+<   COMMAND+SHIFT+<
Symbol font         NO KEYSTROKE        COMMAND+SHIFT+Q   COMMAND+SHIFT+Q
Subscript           COMMAND+=           COMMAND+SHIFT+    COMMAND+SHIFT+
                                        HYPHEN            HYPHEN
Superscript         COMMAND+SHIFT+=     COMMAND+SHIFT+    COMMAND+SHIFT+
                                        PLUS              PLUS
*These formats affect only lowercase characters.
