WD95: Large Envelopes Print in Portrait to HP 5si MX, 5n on NT 4

 Last reviewed: August 9, 1997
Article ID: Q169258
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Windows NT Server, version 4.0
  • Microsoft Windows NT Workstation, version 4.0

SYMPTOMS

When you print a custom envelope to a Hewlett Packard (HP) 5si MX, or HP 5n printer from Word 7.0 running on Windows NT 4.0, your text may be printed in portrait rather than landscape orientation.

CAUSE

This problem occurs when you print a custom envelope with a height greater than 4.25 inches.

WORKAROUND

If you have an HP 5n printer, downloading and installing the updated HP 5n NT printer driver (version 3.71) corrects the problem.

NOTE: If you have an HP 5si MX printer, this does not correct the problem.

For more information, please see the Hewlett Packard World Wide Web site: 

   http://www.hp.com/

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

This problem does not occur when using Word 7.0 on Microsoft Windows 95.
