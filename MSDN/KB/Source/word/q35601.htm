Maximum Number of Characters in WinWord Search Function

 Last reviewed: July 30, 1997
Article ID: Q35601
The information in this article applies to:
  • Microsoft Word for Windows versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

Word for Windows Version 2.0 and later

From the Edit menu, after choosing the Find or Replace commands, the Find What and Replace With text boxes will contain a maximum of 256 characters.

If the text that is searched for is to be replaced with text that is greater then 256 characters, or contains a graphic, the replacement text or graphic can be copied into the Clipboard. Typing ^C in the Replace With dialog box will replace the searched for text with the contents of the Clipboard. Text that is being searched for cannot be accessed from the Clipboard.

Word for Windows Version 1.0 and 1.1

From the Edit menu, after choosing the Search or Replace commands, the Search For Replace With text boxes will contain a maximum of 256 characters.

If the text that is searched for is to be replaced with text that is greater than 256 characters, or contains a graphic, the replacement text or graphic can be copied into the Clipboard. Typing 

   ^C
in the Replace With box will replace the searched for text with the contents of the Clipboard. Text that is being searched for cannot be accessed from the Clipboard.
