WD: Deleted Text Not Displayed When Hidden Text Option Set

 Last reviewed: February 18, 1998
Article ID: Q176753
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you use Track Changes While Editing, text marked as Deleted Text disappears, even when the option to view all non-printing or hidden characters is turned on.

CAUSE

The Deleted Text Mark option has been set as Hidden.

WORKAROUND

Change the Deleted Text Mark option to Strikethrough.

To change the Deleted Text Mark option to Strikethrough, follow these steps:

  1. On the Tools menu point to Track Changes, and click Highlight Changes.

  2. Click Options.

  3. In the Delete text list, select Strikethrough.

  4. Click OK to accept and return to the Highlight Changes dialog box.

  5. Click OK to return to the document.
