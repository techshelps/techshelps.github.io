WD: Table Smaller Than Expected When Placed on Page with Columns

 Last reviewed: March 18, 1998
Article ID: Q144702
The information in this article applies to:
  • Microsoft word 97 for Windows

SYMPTOMS

When you insert a table on a page that is divided into unequal columns (on the Format menu, click Columns) the table may not be the size you expect.

CAUSE

The width of the table is based on the dimensions of the smallest column. This behavior is by design. Word bases the table width on the smallest column size to ensure that the table will fit into any column even if a column break is deleted.

WORKAROUND

To work around this problem, manually adjust the size of the table.
