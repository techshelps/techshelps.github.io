Where to Find Information About the Windows and WinWord APIs

 Last reviewed: February 6, 1998
Article ID: Q117789
6.00 6.00a 6.00c WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

This article describes the information you can obtain from the following resources about the Microsoft Windows application programming interface (API) and the Word for Windows API (CAPI):

  • "Microsoft Word Developer's Kit" (WDK)
  • Microsoft Developer Resource Group
  • "Microsoft Systems Journal" (MSJ)

MORE INFORMATION

"Microsoft Word Developer's Kit" (WDK)

This Microsoft Press publication is a book and a disk that includes files and libraries you can use to write WLLs (Word add-in libraries).

ISBN 1-55615-630-8, $39.95 (plus freight and applicable sales tax). Includes a complete discussion on using WordBasic to customize Word 6.0 to work the way you do, streamline routine tasks, or build cross-application solutions. The kit also includes the WordBasic function reference, sample macros on a disk, as well as advanced programming topics on using the Word API, Microsoft Messaging API (MAPI), and Open Database Connectivity (ODBC) with Word 6.0.

To order the "Microsoft Word Developer's Kit," phone Microsoft Press at (800) 677-7377.

Microsoft Developer Resource Group

If you are a consultant or software developer working with Word for Windows and would like more information, please call the Microsoft Developer Resource Group at (800) 227-4679, ext. 11771.

"Microsoft Systems Journal"

There is a good article about using the Word API on pages 41-54 of the "Microsoft Systems Journal," Volume 9, Number 5., May 1994. This article goes beyond what you will find in the WDK, and it contains complete code examples.

To place an order or obtain information about the "Microsoft Systems Journal," call one of the following phone numbers: 

   (800) 669-1002 (Subscriptions)
   (614) 382-3322 (Information)

Microsoft TechNet

Comprehensive technical and strategic information, including the Knowledge Base, is available on the Microsoft TechNet and Microsoft Developer Network (MSDN) CD-ROMs. For TechNet subscriptions, call (800) 344-2121 extension 035. For MSDN subscriptions, call (800) 759-5474.
