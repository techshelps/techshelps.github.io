WD6X: Comparison of Shortcut Keys for MacWord 6 and WinWord 6

 Last reviewed: February 5, 1998
Article ID: Q119621
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

This article compares the shortcut keys in Word for the Macintosh and Word for Windows.

How to Obtain a List of Keyboard Assignments

To obtain a list of shortcut key assignments in Microsoft Word, use the appropriate procedure for your version of Word.

Word for Windows NT and Word for the Macintosh:

  1. From the Tools menu, choose Macro.

  2. From the Macros Available In list, select Word Commands.

  3. In the Macro Name list, select ListCommands.

  4. Click Run.

  5. In the List Commands dialog box, choose either Current Menu and Keyboard Settings or All Word Commands.

  6. Click OK.

Word creates a new document that lists the commands and their associated shortcut key assignments.

Word for Windows:

ListCommands is not available in Microsoft Word for Windows. You can use the HelpKeyboard command to open the Help topic "Using Keyboard Shortcuts in Word." You can also search for "Keys, Shortcut" in Help and then choose the "Using Keyboard Shortcuts in Word" topic.
