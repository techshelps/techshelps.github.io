Customizing the Font Selector in Word Assistant

 Last reviewed: February 5, 1998
Article ID: Q106657
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SUMMARY

You can customize the Font Selector (FS) through its initialization file, FONTSEL.INI. This file is located in the Windows program directory.

Word Assistant (WA) is an add-on program to Word version 6.0 for Windows. WA contains several components, including: 

   Templates
   Forms
   TrueType Fonts
   Special Paper
   Font Selector
   Extensive Help
   Clip Art
   Font Assistant

MORE INFORMATION

WA modifies the Format menu in Word 6.0 for Windows to include FS. This utility is used to format text and to display fonts as printed. It is true WYSIWYG ("what you see is what you get"). You can modify this utility to better suit your needs. The modification is done through the FONTSEL.INI file. This is the initialization file for the FS component of WA.

The following is a sample FONTSEL.INI file with comments and settings to modify the look of Font Selector options: 

   [Window Position]
   Left=340
   Top=83              > Determines the position and size of
   Width=501             the FS on startup.
   Height=619

   [Settings]
   Tab=Normal          > Used to set the default tab on startup.
                         Normal or As Font.

   AsFontSize=24       > Sets the size of fonts displayed in
                         the As Font view.

   NormalSize=24       > Sets the size of fonts displayed in
                         the Normal view.

   [Misc]
   button=1            > Puts the Font Selector Button on the
                         Standard Toolbar.
NOTE: Changes will take effect during the next instance of Word 6.0 for Windows. If you make a change to the FONTSEL.INI file, the FS database file, FONTTOOL.DBF, must be regenerated. This is done when FS is started from the Format menu.
