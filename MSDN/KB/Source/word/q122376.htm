WD6x: Default Dialog Box Font (Geneva) Appears Jagged

 Last reviewed: November 17, 1997
Article ID: Q122376
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a

SYMPTOMS

In Word for the Macintosh dialog boxes, the default font (Geneva) is jagged instead of smooth.

CAUSE

This problem occurs when the Geneva font is not installed in your System folder. (In System 7.1 and later, it should be installed in the System Suitcase, in the System:Fonts folder.)

This problem can also occur if the Geneva font is damaged.

RESOLUTION

  • Install the Geneva font.

    -or-

  • If the font is damaged, that is, if it has a file size of 0 kilobytes (KB), delete it, and then reinstall it using the Fonts disk supplied with your Macintosh system software.

MORE INFORMATION

The following table lists the fonts and font sizes that are used in Word 6.0 for the Macintosh. 

   Font   Size      Used for
   ---------------------------------------------------------------------
   
   Geneva  10       Word-owned dialog boxes (any 3-D dialog box) such
                    as the Options dialog box (Tools menu) or the Font
                    dialog box (Format menu).
   
                    Drop-menus such as the Font list and Style list on the
                    Formatting toolbar and the Zoom list on the Standard
                    toolbar.
   
   Geneva  10       (Bold) Button text (such as OK or Cancel) in Word-
                    owned dialog boxs.
   
   Geneva   9       Status Bar
