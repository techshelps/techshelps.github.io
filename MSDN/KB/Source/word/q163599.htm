WD: Slow Printing Table Borders to LaserJet 5/5M PostScript

 Last reviewed: February 5, 1998
Article ID: Q163599
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

If you are running Windows NT 4.0, and if you print a document containing table borders from Word 7.0 or Word 97 to an HP Color LaserJet 5/5M PostSscript printer using version 9.x of the printer driver, your document may take a very long time to print.

CAUSE

The NT 4.0 printer driver for the HP Color LaserJet prints table borders, especially dotted lines, very slowly.

WORKAROUND

If possible, use a different printer driver.

STATUS

Microsoft has confirmed this to be a problem in the products listed above. Microsoft is researching this problem and will post new information here as it becomes available.

MORE INFORMATION

This problem does not occur with Windows 95
