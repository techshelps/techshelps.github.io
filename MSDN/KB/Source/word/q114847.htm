Newsletter Wizard (NEWSLTTR.WIZ) Produces Blank Page

 Last reviewed: February 5, 1998
Article ID: Q114847
6.00 6.00a 6.00c 7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

Sometimes when you create a document based on the Newsletter Wizard (NEWSLTTR.WIZ), Word for Windows creates a blank document instead of a newsletter.

CAUSE

Sometimes this happens when you type "1" when the wizard prompts you for the number of pages in your newsletter.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

If this problem occurs when you create a document based on the Newsletter Wizard, type "2" (without the quotation marks) instead of "1" when the wizard asks, "How many pages will you need for your newsletter?"

REFERENCES

"Microsoft Word User's Guide," version 6.0, page 206

"Microsoft Word Quick Results," version 6.0, pages 89-91, 98
