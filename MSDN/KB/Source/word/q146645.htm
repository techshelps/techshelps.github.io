WD: Page Setup Not Available on the File Menu

 Last reviewed: February 5, 1998
Article ID: Q146645
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 5.0, 5.1, 5.1a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

The Page Setup command (Document Layout command in Word for the Macintosh) on the File menu is not available.

CAUSE

By design, the Page Setup command is not available when any of the following conditions are true:

  1. You do not have a printer driver selected in the Chooser (Macintosh) or the Control Panel (Windows).

  2. The installed printer driver is damaged or corrupted.

  3. The insertion point is inside a frame. Frames cannot span multiple pages.

  4. The document is locked for annotations. With the exception of annotations, you cannot make changes to the document.

  5. The document is protected for forms. With the exception of completing an online form, you cannot make changes to the document. (Forms and forms protection is not available in Word for the Macintosh version 5.0 and 5.1.)

  6. The insertion point is inside a text box or callout. The text box and callout are objects which do not have page properties. (These drawing objects are not available in Word for the Macintosh version 5.0 and 5.1, however, the Page Setup command is not available when you create a Microsoft Draw object.)

  7. You are in-place editing in the Equation Editor or WordArt. Equation Editor and WordArt do not have page properties.

RESOLUTION

To resolve this problem, use the appropriate method for your situation.

  1. Select a printer driver.

  2. Remove and reinstall the printer driver.

  3. Move the insertion point outside of the frame.

4 and 5. Unprotect the document.

  1. Move the insertion point outside of the textbox or callout.

  2. Close WordArt or Equation Editor.
