Word 6.0 for the Macintosh: Default Settings

 Last reviewed: February 5, 1998
Article ID: Q118835
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 6.0

SUMMARY

The following table lists all common default settings for Word 6.0 for the Macintosh. 

   Feature                                 Default Setting
   ------------------------------------------------------------------

   Fractional Widths                       Off
   (only with a LaserWriter selected)
   Page Margins in print preview           Off
   Section Start                           New Page
   Save option                             Word Document
   View                                    Normal
   Show Paragraph Marks                    Off
   Smart Quotes                            On
   Background Repagination                 On
   Include Formatted Text                  On
   Drag And Drop                           On
   Show Nonprinting Characters             Off
   Show Hidden Text                        Off
   Show Field Codes                        Off
   Show Bookmarks                          Off
   Show Table Gridlines                    On
   Text Bounds in Page Layout View         Off
   Picture Placeholders                    Off
   Open in page layout                     Off
   Formatting Toolbar                      On
   Ruler                                   On
   Show Function Keys on Menu              On
   List Most Recently Used Files           On
      (Off for small screen Macintoshes)
   Short Menu Names                        Off
   Always Interpret RTF                    On
   Always Make Backups                     Off
   Allow Fast Saves                        On
   Prompt for Summary Info                 Off
   Prompt to Save Normal                   Off
   Default Font                            Times
      (user selects during install)
   Main Dictionary                         English Dictionary
   Language                                None
   Custom Dictionary                       Selected
   Always Suggest                          On
   From Main Dictionary Only               On
   Ignore Uppercase                        Off
   Ignore Words with Numbers               Off
   Grammar Rule Groups                     For Business Only
   Show Document Statistics                On
   Thesaurus                               US English Thesaurus
   Hyphenation                             English Hyphenation
