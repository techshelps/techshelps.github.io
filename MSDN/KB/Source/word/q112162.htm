Removing Setup Billboards to Speed Network Installation

 Last reviewed: February 6, 1998
Article ID: Q112162
6.00 6.00a WINDOWS 
 kbnetwork
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0, 6.0a

When performing a Workstation installation of Word for Windows, the number of billboards that are loaded may slow down the installation process. Billboards are the images displayed on the screen while Word is being installed. To make Word load only the first billboard and keep it on the screen for the entire installation, thereby speeding up Setup, do the following:

NOTE: The following modifications to SETUP.STF should be performed on the server installation.

  1. Make a backup of the SETUP.STF file, by copying SETUP.STF to SETUPSTF.BAK.

  2. Make a working copy of SETUP.STF, by copying SETUP.STF to SETUPSTF.TMP.

  3. Load SETUPSTF.TMP from your Server installation of Word into Microsoft Excel (or any other application you want to edit the file in).

  4. Cursor down to the ======BILLBOARDS====== section.

  5. On the Group line, which is two lines under the BILLBOARDS line, delete all numbers to the right of the first number, leaving all leading and trailing tabs and spaces.

  6. Save SETUPSTF.TMP.

  7. To ensure that only the correct lines have been edited, exit to an MS-DOS prompt and change directories to where your SETUP.STF and SETUPSTF.TMP files are located. Then type the following command: 

          fc.exe /t setupstf.tmp setupstf.bak

  8. If the comparison is successful, the result should be six lines displayed, which corresponds to the line changed and the lines before and after the change.

  9. If the above comparison is successful, rename SETUPSTF.TMP to SETUP.STF, overwriting the previous version. If the comparison fails, make another working copy of SETUP.STF and repeat the above steps, using a different editor.

When the next Setup is run from this installation, the first billboard will appear and remain until the conclusion of the installation.
