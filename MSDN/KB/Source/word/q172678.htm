WD97: Changes in Word SR-1 Spelling and Thesaurus

 Last reviewed: August 16, 1997
Article ID: Q172678
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

The U.S. proofing tools that ship with Microsoft Word 97, Service Release 1 contain a few changes from the U.S. proofing tools that shipped with Microsoft Word 97.

MORE INFORMATION

The U.S. proofing tools that ship with Microsoft Word 97, Service Release 1, contain the following changes:

  • The word millenum is now flagged as misspelled. "Millennium" is suggested.
  • Many common, socially unacceptable thesaurus suggestions have been removed
  • The Msth_am.lex and Mssp2_en.lex files were updated to remove spelling and thesaurus suggestions of obscene or politically incorrect words. Obscene words will not be suggested for misspellings that closely resemble them.
