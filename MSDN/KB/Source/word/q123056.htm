"Not a Valid Link" Linking MS Excel Object in Word Table

 Last reviewed: February 5, 1998
Article ID: Q123056
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

When you paste link a Microsoft Excel spreadsheet as formatted text into a Word document with the Paste Special command, you may receive the following error message: 

   Error! Not a valid link
CAUSE

When you insert a Microsoft Excel spreadsheet as formatted text it is contained in a table. By design, however, you cannot insert a table into an existing table in Word.

RESOLUTION

Use one of the following methods to link the Microsoft Excel spreadsheet:

  • Remove the table before inserting the spreadsheet as formatted text.

    -or-

  • Insert the spreadsheet as a Microsoft Excel spreadsheet object, unformatted text, or a picture instead of as formatted text.
