SUMMARY

Parsons Technology has a software package called Quick Verse for Windows, a comprehensive Bible concordance with features to perform complex searches and compare Bible verses. The program is for general and in-depth study in a Windows environment.

This article discusses how to integrate Quick Verse with Word for Windows. The installed Quick Verse program includes a file (QVWIN.DOT) that is a Word for Windows template.

MORE INFORMATION

Quick Verse Setup installs QVWIN.DOT in the QVWIN directory. To add the Quick Verse functionality to Word, copy or move QVWIN.DOT to the Word for Windows TEMPLATE subdirectory (if you are using Word 6.0 or 6.0a) or to the Word for Windows directory (if you are using Word 2.x).

To use Quick Verse from Word:

  1. From the File menu, choose New. Under New, select Document.
2. Select QVWIN from the Template list (the Use Template list in 2.x) and 
   choose OK.
3. When you want to include a verse in your document, choose Paste Verses 
   from the Edit menu.
4. Search the Bible to paste different text and verses. 5. When you save the file in Word 6.0, you will be prompted to save as Word 
   6.0 or 2.0. Make the appropriate selection and choose OK.
