Word 5.0 for the Macintosh IVR Fax: Mailing Labels

SUMMARY

              Microsoft Word for the Macintosh, Versions 5.0
               Automated Product Support Service Fax-Script
                   Questions and Answers: Mailing Labels
               --------------------------------------------

1. Q. How do I create mailing labels and print them on my LaserWriter?

   A. The following 11-step process will explain how to create 30
      mailing labels per page on a LaserWriter:

      a. From the File menu, choose Page Setup.
      b. Choose the Options button.
      c. Select Larger Print Area and click OK.  This will take you
         back to Page Setup.
      d. Click the Document button. Please enter your margins here.

            The top margin should be set at 0.5 inches.
            The bottom margin should be set at 0.0 inches.
            The left margin should be set at 0.5 inches.
            The right margin should be set at 0.25 inches.

      e. Click OK to close out of the Document dialog box and return to
         Page Setup.
      f. Click OK to close out of Page Setup.
      g. From the Format menu, choose Paragraph.
      h. Change the Line Spacing from Auto to Exactly (-12) minus
         twelve. Click OK.
      i. From the Insert menu, choose Table.

            The number of columns should be set at (3) three.
            The number of rows should be set at (10) ten.
            The cell width should be set at 2.75 inches.

      j. Choose the Format button.
      k. Change the Minimum Row Height from Auto to Exactly (-72) minus
         seventy-two. Click OK.

      You will see your table appear. Type the appropriate information
      in each cell and print the labels.
For more information on mailing labels and performing print merges in Mac Word 5.0, please refer to Chapter 34, titled "Mailing Labels," on page 624 of the "User's Guide to Microsoft Word."
