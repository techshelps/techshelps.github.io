IA: Files Installed by Internet Assistant for Word for Macintosh

 Last reviewed: November 17, 1997
Article ID: Q153979
The information in this article applies to:
  • Microsoft Internet Assistant for Word for the Macintosh, version 2.0
  • Microsoft Word for the Macintosh, version 6.0.1

SUMMARY

This article identifies the files that are installed by Internet Assistant for the Macintosh.

MORE INFORMATION

Internet Assistant for the Macintosh installs files to the following locations:

<Microsoft Word 6 folder>:Internet Assistant 

   Creating a Web Doc
   Default
   Favorite Places
   Internet Assistant Help
   Internet Assistant Installer
   Microsoft Help Run Time
   Readme
   WordIA Internet Access
<Microsoft Word 6 folder>:Internet Assistant:Temporary Files 

   <Empty>
<Microsoft Word 6 folder>:Internet Assistant:Filters 

   Ebif9.flt
   Empct9.flt
   Ibgif9.flt
   Ibjpg9.flt
   Impct9.flt
   Isqdr9.dll
<Microsoft Word 6 folder>:Templates 

   HTML (Template)
   HTMLView (Template)
   WebView (Template)
<Microsoft Word 6 folder>:Word Extensions 
   HyperText Markup Language
System Folder:Preferences:Internet Assistant 

   Internet Assistant Prefs
System Folder:Preferences:Internet Assistant:Word Startup Folder (6) 

   Internet Assistant
