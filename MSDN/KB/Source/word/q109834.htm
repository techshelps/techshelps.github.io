Footnote Reference Contains Trailing Space

 Last reviewed: February 5, 1998
Article ID: Q109834
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

If you insert a footnote in a Word for Windows document using AutoNumber numbering, the footnote reference number contains a trailing space which you cannot delete.

STATUS

Microsoft has confirmed this to be a problem in Word for Windows. This problem was corrected in version 6.0a and later of Word for Windows. In later versions of Word, the white space is inserted as a regular SPACE character to the right of the footnote reference mark, which means you can edit or remove it if you want.
