WD98: Deleted Text from Tracked Changes Appears in HTML

 Last reviewed: February 25, 1998
Article ID: Q181616
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

If you enable the Track Changes While Editing command, and mark text for deletion, when you save the document as HTML and reopen the document, the deleted text will appear in the document and it will not be marked as deleted text (that is, it will look like normal text).

CAUSE

HTML does not support marking for tracked changes.

RESOLUTION

Accept all tracked changes before you save the document as HTML.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
