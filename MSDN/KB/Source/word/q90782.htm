WD: Text Prints Past Right Margin with QuickDraw Printer on Mac

 Last reviewed: February 5, 1998
Article ID: Q90782

The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 4.0, 5.0, 5.1, 6.0, 6.0.1, 6.0.1a
  • The Macintosh operating system versions 6.07, 6.08, 7.0, and later

SUMMARY

In Word, lines may extend past the right margin of a paragraph if the character formatting of the paragraph includes TrueType fonts formatted in italic, bold, or bold italic.

CAUSE

This problem occurs when a particular font has TrueType outline fonts AND screen fonts in the System Folder.

Fonts with numbers after their names are screen fonts. Fonts without numbers are TrueType fonts. If you have duplicate fonts, both types of font will be displayed in the System suitcase. To determine which fonts are duplicated, use the appropriate method for your system.

System 7.1 or later

  1. Open the Fonts suitcase, located in the System Folder.

  2. On the View menu, click By Name.

  3. Repeat this process for the System suitcase, in case you also have fonts stored in the System.

System 7.0

  1. Open the System suitcase located in the System Folder.

  2. On the View menu, click By Name.

System 6.x

Start the Font/DA Mover application and choose the Font button.

WORKAROUND

Method 1

To work around this problem, remove the screen fonts that duplicate the TrueType fonts, using the appropriate procedure below.

System 7.x:

  1. Open the appropriate suitcase (System and/or Fonts), located in the System Folder.

  2. On the View menu, click By Name. Fonts with numbers after their names are screen fonts. Fonts without numbers are TrueType fonts. If you have duplicate fonts, both types are displayed in the suitcase(s).

  3. Remove all screen fonts that duplicate the TrueType fonts by dragging them from the System Folder.

System 6.x:

  1. Start the Font/DA Mover application.

  2. Click Font. Fonts with numbers after their names are screen fonts. Fonts without numbers are TrueType fonts. If you have duplicate fonts, both types of font are displayed in the System Folder.

  3. Remove all the screen fonts that duplicate the TrueType fonts by selecting them from the list and clicking Remove.

  4. Quit the Font/DA Mover.

Method 2

Format the text with a font not duplicated in the System Folder.

Method 3

Remove bold and/or italic formatting from the paragraph.
