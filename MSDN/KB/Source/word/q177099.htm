WD98: Picture Disappears Returning from Picture Editor

 Last reviewed: February 17, 1998
Article ID: Q177099
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you choose to insert a picture while in the Microsoft Word Picture Editor and then click Close Picture to return to your document, the picture may disappear. Also, the picture does not print.

CAUSE

Word does not have enough allocated memory to display or print the picture.

RESOLUTION

To resolve this problem, follow these steps:

  1. Quit Word.

  2. Click the Word icon once to select it.

  3. On the Edit menu, click Get Info.

  4. Adjust the current maximum memory size so that it is at least equal to 12000.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
