WD: Changing the Memo Wizard Document Formatting

 Last reviewed: February 3, 1998
Article ID: Q119628
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SUMMARY

The Memo Wizard does not prompt you to modify character formatting such as font, point size, and so forth. This article describes how you can use the Memo Wizard to create your own custom styles when using the wizard.

NOTE: Microsoft does not support the modification of its wizards or templates. Modifications done by the user are at the user's risk and constitute no obligations or warranties, either written or expressed, by Microsoft.

MORE INFORMATION

The Memo Wizard uses three style templates to format the document it creates. The templates are listed in the following tables. 

   Word 7.x and Later (Windows and Macintosh):

      Memo Wizard style       Style template
      --------------------------------------

      Contemporary            Contemporary Memo.dot
      Elegant                 Elegant Memo.dot
      Professional            Professional Memo.dot

   Word 6.x:

      Memo Wizard style       Style template
      --------------------------------------

      Classic                 Memo1.dot (Memo1 for Macintosh)
      Contemporary            Memo2.dot (Memo2 for Macintosh)
      Typewriter              Memo3.dot (Memo3 for Macintosh)
The wizard primarily uses a limited number of the styles found in each template. By modifying these styles, you can change any character attributes when you create a memo. To accomplish this, follow these steps:

  1. On the File menu, click Open. Select the template you want to modify. The templates are stored in one of the following locations: 

       Version of Word         Location of template files
   --------------------------------------------------------------------

   98 (Macintosh)          Macintosh HD:Microsoft Office 98:Templates:Memos
   97                      Program Files\Microsoft Office\Templates
   7.x                     MSOffice\Templates\Other Documents
   6.x (Windows)           Winword\Template
   6.x (Macintosh)         Macintosh HD:Microsoft Word:Templates

   From the Files Of Type list (List Files Of Type list in Word 6.x and
   Word 98 Macintosh Edition), select Document Templates (*.dot). From the
   File Name list, select one of the templates, depending on which memo
   style you want to use.

   Click the OPEN or OK (Word 6.x for Windows) button.

  2. The following styles are used in the template: 

          This style name               Is used for this
      ----------------------------------------------

      Document Label                Memorandum text
      Message Header First          Date line
      Message Header                To, From, and Re lines
      Message Header Label          To, From, and Re lines
      Message Header Last           CC line
      Body Text                     Body of the memo
      Company Name (Word 97)        Company Logo

   To modify these styles, click Style on the Format menu. From the List
   box, choose All Styles. From the Styles list, select the style you want
   to modify, and click Modify.

  3. From the Modify Style dialog box, click to select the Add To Template check box. Click the Format button and, from the list that appears, select any of the formatting options you want to modify. Click OK to return to the Modify Style dialog box. Click OK twice to return to the Style dialog box.

  4. Repeat steps 3-4 for each additional style you want to modify.

  5. Click Close.

  6. Save the template.

The next time you use the Memo Wizard you can select the Memo Style corresponding to the modified template.
