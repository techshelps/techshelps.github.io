WD98: Custom "Page Design" Changes Not Used by Letter Wizard

 Last reviewed: February 17, 1998
Article ID: Q179760
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

When you use the Letter Wizard to create a new letter based on a custom page design, changes in your custom template may not be inserted into the new document.

This problem occurs even when you have correctly applied the proper styles in your custom template.

NOTE: To select a custom page design, on the Letter Format tab, change the "Choose a page design" box to your custom letter template.

WORKAROUND

Do not use the Letter Wizard. To create a new document using your custom page design, follow these steps:

  1. On the File menu, click New.

  2. On the Letters & Faxes tab, select your custom page design template.

  3. Under Create New, click to select Document, and then click OK.

You will now have a new document based on your custom page design template.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article.

MORE INFORMATION

The Letter Wizard Word can help you write a letter quickly and easily. The wizard provides frequently used letter elements, such as salutations and closings, that you can select from lists. The wizard also helps you structure your letters based on frequently used letter styles.

Use the following procedure, which requires the Office Assistant, to create a letter with the Letter Wizard. If you do not have the Office Assistant installed, you can start the Letter Wizard by clicking New on the File menu, clicking the Letters & Faxes tab, and then double-clicking Letter Wizard.

To create a letter with the Letter Wizard, follow these steps:

  1. Type a salutation that starts with the word "Dear" or "To," followed by a name and a colon or a comma, such as "Dear Thomas," or "To All Board Members:" and then press ENTER.

    The Office Assistant will appear, prompting you with the question "It looks like you're writing a letter. Would you like Help?"

  2. In the Office Assistant, click "Get help with writing the letter."

  3. Follow the instructions in the Letter Wizard. If you want to skip a step or go to a specific panel, click one of the tabs.

NOTE: To use the Letter Wizard to modify or complete an existing letter, open the letter in Word, and then click Letter Wizard on the Tools menu.
