GP Fault When Inserting a Microsoft Access 1.0 Database

 Last reviewed: February 5, 1998
Article ID: Q106646
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0
  • Microsoft Access for Windows, version 1.0

SYMPTOMS

If you insert a Microsoft Access 1.0 database by choosing Database from the Insert menu, a general protection (GP) fault error may occur in module MSAccess.

STATUS

This problem was corrected in Access version 1.1.

RESOLUTION

This problem does not occur in Microsoft Access version 1.1.

MORE INFORMATION

Steps to Reproduce Problem

  1. From the Insert menu, choose Database.
2. Choose Get Data, select NWIND.MDB, and choose OK. 3. Select the DDE option and choose OK. 4. Select Categories and choose OK. 5. Choose Insert Data, and then OK to accept all data.

This procedure results in a general protection (GP) fault in MSAccess.
