WD6: GP Fault in IWIA.EXE Saving IWIA Object

 Last reviewed: February 5, 1998
Article ID: Q128689
The information in this article applies to:
  • Microsoft Internet Assistant for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0a, 6.0c

SYMPTOMS

You get the following general protection fault error when you attempt to save an InternetWorks Internet Assistant Document object. 

   INTERNET caused a General Protection Fault in module IWIA.EXE
   at 0001:0FF2.

CAUSE

You did not insert the object as an icon, and you chose Save As from the shortcut menu after right-clicking on the object and the object displayed the default InternetWorks Internet Assistant document.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Internet Assistant for Word, version 1.0 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

RESOLUTION

To insert an InternetWorks Internet Assistant object, do the following:

  1. Open a Word document.

  2. From the Insert menu, choose Object.

  3. Do one of the following:

    a. Select the Create From File tab.

    b. Select the document you want to insert.

    c. Choose either Link to File or Display as Icon.

    d. Choose OK.

    Note: If the file has an HTM extension, you can associate the file with InternetWorks Internet Assistant or Word in the Windows File Manager.

    - or -

    a. Select the Create New tab, highlight InternetWorks Internet 

          Assistant Document object.

    b. Check Display as Icon.

    c. Choose OK.

    d. Load a URL or local file by typing the URL in the URL text box.

    e. Press Enter on the keyboard or click the Launch the selected URL 

          button.  (Note: At this point if you want to save the downloaded
      information to a file, you can choose Save As from the File menu.)

    f. From the File menu, choose Exit & Return to <Document name>.

    Note: This method saves a pointer to InternetWorks Internet Assistant; it does not save the downloaded file in the object.
