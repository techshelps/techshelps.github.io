WD97: Cannot Open Word 97 Documents in Word 6.0 or 7.0

 Last reviewed: February 3, 1998
Article ID: Q157666
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you attempt to open a Microsoft Word 97 document into Microsoft Word versions 6.x or 7.x, the document contains garbage characters.

CAUSE

The document file format used to save files for Word 97 has changed. Word versions 6.x or 7.x cannot read this format.

WORKAROUND

The best way to open a Word 97 document in Word 6.x or 7.x for Windows is to use the Word 97 import converter. For information about how to obtain the Word 97 converter, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q162214
   TITLE     : How to Obtain the Word 97 Converter
Note: Currently, there is not a Word 97 import converter for Word for the Macintosh.

Method 1: Install the Microsoft Word 97 converter

In Microsoft Word for Windows, versions 6.x and Word versions 7.x, you will need to install the Microsoft Word 97 for Windows converter.

Method 2: In Word 97, Save the Document as a Word 6.0/95 File

  1. Open the document in Word 97.

  2. On the File menu, click Save As.

  3. In the Save As dialog, change the "Save as type:" to "Word 6.0/95 (*.doc)".

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q162602
   TITLE     : Questions About Saving in the Word 6.0/95 File Format

Method 3: In Word 97, save the document as Rich Text Format (*.rtf)

  1. Open the document in Word 97.

  2. On the File menu, click Save As.

  3. In the Save As dialog, change the "Save as type:" to "Rich Text Format (*.rtf)".
