Cannot Shrink Selection When Using Some Key Combinations

 Last reviewed: February 6, 1998
Article ID: Q86870
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c, 6.0, 6.0a, 6.0c

In Microsoft Word for Windows, if you select a field code or its results by pressing one of the keyboard shortcuts listed below, you cannot subsequently cancel the selection when using these key combinations to shrink the selection. If you are shrinking a selection using these keyboard shortcuts and encounter a field code or its results, Word for Windows stops shrinking the selection. 

   SHIFT+LEFT ARROW
   SHIFT+RIGHT ARROW
   F8, LEFT ARROW
   F8, RIGHT ARROW
   CTRL+SHIFT+LEFT ARROW
   CTRL+SHIFT+RIGHT ARROW
Other key combinations used to extend or shrink a selection do not stop when you encounter a field code.

This occurs whether you are viewing field codes or their results.

REFERENCES

"Microsoft Word for Windows User's Guide," version 2.0, page 105, back cover
