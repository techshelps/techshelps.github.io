SGML Err Msg: "SGML Converter Error"

 Last reviewed: August 5, 1997
Article ID: Q126980
The information in this article applies to:
  • Microsoft SGML Author for Word, version 1.0
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

When you try to save a document as SGML, you receive the error message: 

   SGML Converter ERROR  UNKNOWN MESSAGE
After choosing OK, you receive the error message: 

   Word caused a General Protection Fault in module MSDB.DLL

CAUSE

There is insufficient disk space for Word to convert the file.

STATUS

Microsoft has confirmed this to be a problem in SGML Author for Word version 1.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

RESOLUTION

To resolve this problem, free up a large amount of disk space, or change the swap file setting.
