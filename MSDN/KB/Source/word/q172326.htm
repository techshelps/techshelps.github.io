WD97: Infinite Loop When Using Spelling Checker

 Last reviewed: September 23, 1997
Article ID: Q172326
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When checking the spelling in a document, the spelling checker enters an infinite loop.

CAUSE

This problem occurs when all of the following conditions are true:

  • You click Change All in the spelling checker dialog box.

    -and-

  • The spelling checker finds more than one instance of a misspelled word.

    -and-

  • The misspelled word is not in the main or custom dictionaries.

WORKAROUND

To break out of this infinite loop, hold down the ESC key.

To prevent this problem from occurring, use any of the following methods:

  • Add the misspelled word to the custom dictionary. To do this, click Add in the spelling checker dialog box, or right-click the misspelled word and click Add on the shortcut menu.

    -or-

  • On the Edit menu, click Replace to find the misspelled word and replace it with the correctly spelled word.

    -or-

  • Instead of using Change All, click Change for each individual occurrence and repeat until all misspelled words have been changed.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
