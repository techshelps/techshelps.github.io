WD97: Products That Qualify for the Word Upgrade

 Last reviewed: February 3, 1998
Article ID: Q162596
The information in this article applies to:
  • Microsoft Word 97 for Windows

SUMMARY

Microsoft Word 97 is available in two versions: a retail version which can be installed on any compatible computer, and an upgrade version which requires that you have a qualifying product installed on your computer or that you have Disk 1 from the qualifying product available when you run the Setup program.

NOTE: The retail version does not require a qualifying product.

This article lists the products that qualify you to use the upgrade version of Microsoft Word 97.

MORE INFORMATION

In the following table, qualifying product names are listed on the left and version numbers of the products are listed on the right. 

   Product                                     Versions
   ---------------------------------------------------------

   Ami Pro for Windows                         1.0 or later
   Lotus Word Pro for Windows                  96
   Microsoft Word for MS-DOS                   2.0 or later
   Microsoft Word for Windows                  2.0 or later*
   Microsoft Word for Windows NT               6.0
   WordPerfect for MS-DOS and Windows          4.0 or later
   WordStar 2000 for MS-DOS                    3.0 or later
   WordStar for MS-DOS and Windows             1.0 or later
   WordStar Pro for MS-DOS                     1.0 or later
   Word Perfect SCO/UNIX                       all versions
* Note: The Microsoft Word for Windows, Upgrade box states "any version of Word for Windows" will qualify.

NOTE: This list applies to the U.S. English version of Microsoft Word 97 only.

When you install Microsoft Word 97, the Setup program may not correctly recognize that you have one or more of the qualifying products installed on your computer. If this occurs and you are unable to install Microsoft Word 97, contact Microsoft Technical Support. When you speak to a support engineer, inform them that you have a qualifying product as listed in Microsoft Knowledge Base article Q162596.
