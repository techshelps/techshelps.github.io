WD97: Borders Print Poorly with Hewlett-Packard 820Cse

 Last reviewed: July 31, 1997
Article ID: Q171607
The information in this article applies to:
  • Microsoft Word 97 for Windows on the following platform: Win95

SYMPTOMS

Some paragraph borders print with very poor quality, extra lines, or red dots across the border when printing from Microsoft Word 97 to a Hewlett- Packard (HP) 820Cse printer.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word 97 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Hewlett-Packard has released only two versions of drivers for the HP 820Cse printer; the problem occurs with both versions of the printer driver. (The latest version is 8.2.) Unlike other HP DeskJet printers, the HP 820Cse cannot use any other printer driver. Hewlett Packard has confirmed this information.
