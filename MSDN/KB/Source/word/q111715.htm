How to Manually Install Word 6.0 If SETUP.EXE Fails

 Last reviewed: February 5, 1998
Article ID: Q111715
6.00 6.00a WINDOWS
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Windows operating system version 3.1

SYMPTOMS

When you run the Setup program (SETUP.EXE) to install Word for Windows, an error may occur at Disk 1, and Setup will stop. Word will not be successfully installed, and you will receive the following error message: 

   Invalid Object ID Error

WORKAROUND

If usual troubleshooting steps fail to resolve this problem, you may need to install Word without running SETUP.EXE. Before you use the instructions below, you should exhaust other troubleshooting options. For information about troubleshooting Word Setup problems, query on the following words in the Microsoft Knowledge Base: 

    and and general and 6.00
The procedure below installs Word without using the Setup program. Using this procedure, you will manually decompress several installation files on your hard disk, then you will run the ACMSETUP.EXE program to install Word.

  1. At the MS-DOS command prompt, type the following to create a directory for the setup files: 

          MD C:\~SETUP

    Note: During a normal Setup these files are placed in a hidden directory named ~MSSTFQF.T, which is automatically created on the last available local drive that Setup has sufficient access to.

  2. Place Word Disk 1 in your floppy disk drive.

  3. At the command prompt, type "A:" or "B:" (without the quotation marks) to change to the floppy disk drive where you inserted Disk 1.

  4. At the command prompt, type the following, pressing ENTER after each line:

    DECOMP _MSSETUP.EX_ C:\~SETUP\_MSSETUP.EXE DECOMP ACMSETUP.EX_ C:\~SETUP\ACMSETUP.EXE DECOMP ACMSETUP.HL_ C:\~SETUP\ACMSETUP.HLP DECOMP MSCPYDIS.DL_ C:\~SETUP\MSCPYDIS.DLL DECOMP MSSETUP.DL_ C:\~SETUP\MSSETUP.DLL DECOMP WINWORD6.IN_ C:\~SETUP\WINWORD6.INF DECOMP SETUP.ST_ C:\~SETUP\SETUP.STF DECOMP WWSETUP.TT_ C:\~SETUP\WWSETUP.TTF DECOMP WORD_BB.DL_ C:\~SETUP\WORD_BB.DLL DECOMP ODBCINST.DL_ C:\~SETUP\ODBCINST.DLL

    NOTE: These files are listed in the [Files] section of SETUP.LST, which is a text file located on Disk 1.

  5. In Windows File Manager, select the ACMSETUP.EXE file in the C:\~SETUP directory (do not double-click the file). From the File menu, choose Run. Type the following, and then choose OK: 

          ACMSETUP /T SETUP.STF /S A: <or B: if Disk 1 is in the B: drive>

    When Setup starts, follow the instructions on your screen.
