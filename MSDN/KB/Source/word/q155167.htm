"Online assistance is not available" Running Help in Word

 Last reviewed: February 5, 1998
Article ID: Q155167
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0, 7.0a

SYMPTOMS

When starting help from within Word for Windows 95 version 7.0, you may receive the following error message: 

   Online assistance is not available. Rerun Setup to make sure all
   necessary Help files are correctly installed.

CAUSE

There are two possible causes for this error message:

  • The Winword.exe file has been copied to another location and you are starting Word from the new location

    -or-

  • The Winword.exe file has been moved from the original location to another location, and the shortcut you are using is pointing to the new location.

RESOLUTION

Check to see if Winword.exe exists in the C:\Msoffice\Winword folder. If so, delete the copy of Winword.exe in the other location. Create a new shortcut to the original location, usually C:\Msoffice\Winword\Winword.exe. 

   -or-
Move Winword.exe back to the original location, usually C:\MSOFFICE\WINWORD, and change the shortcut to point to the original location.
