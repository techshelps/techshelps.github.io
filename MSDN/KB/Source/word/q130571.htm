WD6: Letter Wizard Produces Error Messages

 Last reviewed: February 5, 1998
Article ID: Q130571
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

When you try to create a new document with the Letter Wizard, you may receive the following error message: 

   The Letter Wizard cannot find the Letters folder.

CAUSE

This can be caused by any of the following situations:

  • The User Templates setting is incorrect. To examine this setting, use these steps:

    1. From the Tools menu, choose Options.

    2. Select the File Locations tab.

    3. Check the User Templates setting to be sure it indicates the correct 

          folder. If the wrong folder is listed, use the Modify button to
      correct it.
  • The Letters folder is missing from the Templates folder. If this folder is missing, search the hard drive for the Letters folder and move it into the Templates folder.
  • The Word Wizards Preferences file may be corrupt. Temporarily move it from the Preferences Folder (located in the System Folder) and restart the Letter Wizard to determine if this is the source of the problem.
