Word Basic Error 504 Using PackRat 5.0 Print Merge Command

 Last reviewed: February 5, 1998
Article ID: Q115268
6.00 6.00a 6.00c 7.00 WINDOWS

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When doing a mail merge using Word 6.0 or 6.0a and PackRat 5.0, you may receive the following error: 

   WordBasic Error 504
   Window does not exist
This error does not occur when doing a print merge in Word 2.x for Windows.

CAUSE

When doing a merge using PackRat 5.0, PackRat first examines the mail merge main document and removes any attachments using a macro called PRMERGE. When it doesn't find any attachments, it generates the above error and does not consider the current active document a mail merge main document.

STATUS

Polaris is aware of the problem with the PRMERGE macro and and has a workaround for the problem. For more information, please contact Polaris by calling(619)735-2300.

WORKAROUNDS

Method 1: Create an empty datafile and attach it to the document so it is a 

          mail merge main document.
Method 2: Use ODBC to merge with PackRat 5.0. The PackRat files are based 
          on dBASE. Make sure Confirm Conversions is checked when selecting
          the data source. In the Confirm Data Source dialog box, select
          "dBase files via ODBC" then choose OK.
