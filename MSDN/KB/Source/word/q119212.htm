Error Starting Word's "Examples and Demos" in Online Help

 Last reviewed: February 5, 1998
Article ID: Q119212
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

When you select Examples And Demos in Word Help, Word displays one of the following error messages: 

   Microsoft Help - Unable to run the specified file.

   -or-

   Word Cannot Start Examples and Demos.

CAUSE

These errors can be caused by any one of the following:

  • HyperCard version 2.1 or later is not installed on the Macintosh.
  • You performed a Typical Install, which does not install Examples And Demos.
  • The files necessary to run Examples And Demos have been deleted. (These files are installed in the Tutorial folder located in the Word folder.)

WORKAROUND

Method 1: HyperCard version 2.1 or Later Is Not Installed on the Macintosh

To work around this problem, copy HyperCard to the hard disk from the HyperCard disk that ships with the Apple System disks or CD-ROM.

NOTE: HyperCard needs 2000 kilobytes (KB) of memory available when used by Word.

Method 2: A Typical Installation Was Used to Install Word

Use the steps to reinstall (under "Examples and Demos Files Have Been Deleted" below) to install Examples And Demos.

Method 3: Examples And Demos Files Have Been Deleted

Use the Setup program to remove Examples And Demos (even though these files are already deleted from the hard drive), and then reinstall Examples And Demos to the hard drive.

To uninstall Examples And Demos, follow these steps:

  1. Run Microsoft Word Setup from the Setup folder in the Word folder.

  2. Click the Add/Remove button.

  3. Under Installed Components, click the triangle to the left of online Help. Select Examples And Demos. Click Continue to remove this component.

After Maintenance Setup has removed Examples And Demos, you can reinstall them. To do this, follow these steps:

  1. Run Microsoft Word Setup from the Setup folder in the Word folder.

  2. Click Add/Remove.

  3. Under Components Not Yet Installed, click the triangle to the left of online Help. Check Examples And Demos. Click Continue. Setup will prompt you for the necessary disks to reinstall these files.
