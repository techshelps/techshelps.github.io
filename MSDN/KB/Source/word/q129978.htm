WD6: Envelope Prints Wrong with PacificPage PostScript Cartridge

 Last reviewed: November 17, 1997
Article ID: Q129978
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

When printing envelopes from Microsoft Word to a Hewlett-Packard (HP) LaserJet 4 with Pacific Data's PacificPage PostScript emulation cartridge, the text on the envelopes prints too far to the left.

CAUSE

The PacificPage cartridge doesn't recognize the envelope page size; it positions the text as if it were on an 8.5-by-11-inch page.

RESOLUTION

To fully resolve the problem, contact Pacific Data for the correct .PPD file to use with this printer.

In the meantime, you can work around the problem by adjusting the From Left settings on the envelope:

  1. From the Tools menu, choose Envelopes And Labels.

  2. Select the Envelope tab.

  3. Type the Delivery Address.

  4. Select the Options button.

  5. In the Delivery Address area, change the From Left measurement to 6".

  6. In the Return Address area, change the From Left measurement to 2".
