WD95: How to Install the Missing Grammar Checker

 Last reviewed: August 5, 1997
Article ID: Q139797
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

You are experiencing one of the following symptoms:

  • The Grammar command is missing from the Tools menu.
  • The Grammar tab is unavailable (dimmed) when you click Options on the Tools menu.
  • When you click Grammar on the Tools menu, or from a toolbar button, you receive the following error message: 

          Word cannot find C:\Program Files\Common Files\Microsoft Shared\
      Proof\Gram32.dll for Engligh(US).

NOTE: The grammar checker is installed with a Typical installation of Word version 7.0a for Windows 95.

CAUSE

When you installed Word, you chose to perform a Typical, Custom, or Compact installation. If you chose Custom, you may have cleared the Grammar option. The grammar checker is among the components not installed during a Typical or Compact installation.

RESOLUTION

Windows 95

To install the grammar checker, use the following steps:

  1. Quit Word.

  2. Click the Windows Start button, point to Settings, and click Control Panel.

  3. Double-click Add/Remove Programs.

  4. Click the Install/Uninstall tab.

  5. In the list of installed software, select either Microsoft Word or Microsoft Office.

  6. Click Add/Remove.

  7. In the Setup box, click Add/Remove.

  8. If you are using Microsoft Office, select Microsoft Word and click the Change Option button. If you are using Microsoft Word Setup, skip to step 9.

  9. Select Proofing Tools and click Change Option.

  10. Select the Grammar check box.

  11. Click OK.

  12. Click Continue.

  13. When prompted by Setup, insert the appropriate disk or compact disc.

The new component is installed.

Windows NT

To install the grammar checker, use the following steps:

  1. Quit Word.

  2. From Program Manager, choose Run from the File menu.

  3. Type a:\setup. Click OK. Insert Disk 1 of the Word Setup disks.

  4. Click Add/Remove.

  5. In the Setup box, click Add/Remove.

  6. If you are using Microsoft Office, select Microsoft Word and click the Change Option button. If you are using Microsoft Word Setup, skip to step 9.

  7. Select Proofing Tools and click Change Option.

  8. Select the Grammar check box.

  9. Click OK.

  10. Click Continue.

  11. When prompted by Setup, insert the appropriate disk or compact disc.

    The new component is installed.
