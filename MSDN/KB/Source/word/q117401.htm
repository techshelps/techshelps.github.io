CTRL+Left Mouse Drag-and-Drop Results in Move

 Last reviewed: August 4, 1997
Article ID: Q117401
6.00 6.00a WINDOWS kb3rdparty kbprb

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

The functionality of holding down CTRL and clicking the left mouse button to drag text changes from Copy to Move when you have POPTHES.WLL installed in the Word for Windows STARTUP subdirectory and if it is currently activated as a global add-in. POPTHES.WLL is installed with the Comprehensive Thesaurus, a Word add-in manufactured by ALKI Software, Corporation.

STATUS

Microsoft has confirmed this to be a problem when using the Comprehensive Thesaurus add-in with Microsoft Word version 6.0 and 6.0a for Windows.

WORKAROUND

To use the drag-and-drop functionality in Word, disable the POPTHES.WLL add- in. To do this, from the File menu, choose Templates. Clear the POPTHES.WLL check box in the list of Global Templates and Add-ins.
