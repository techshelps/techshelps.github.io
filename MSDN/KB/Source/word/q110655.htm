Cannot Fax from Word 6.0 Using HP LaserJet Fax

 Last reviewed: February 5, 1998
Article ID: Q110655
The information in this article applies to:
  • Microsoft Word for Windows, version 6.0

SYMPTOMS

If you send a fax from Word 6.0 for Windows using the Hewlett-Packard (HP) LaserJet Fax, you may receive one of the following error messages:

  • Printer is busy--cannot print. Try again when the printer is free.
  • Error saving raster files. You may need to increase the available disk space.

CAUSE

If you use this fax driver to send a fax with background printing turned on, you receive the "Printer is busy..." error message. Background printing requires that the printer driver be able to switch environment settings "on the fly." Some older printer and fax drivers cannot handle this type of a printing environment.

If you use this fax driver to send a fax with background printing turned off, you receive the "Error saving raster files..." message. This error occurs when HP LaserJet Fax attempts to create a cover sheet while Word is running.

If you get one of these error messages, you must exit Windows and restart Windows.

STATUS

HP has discontinued the LaserJet Fax and will not be updating the driver. The latest driver version available is A.2.05a, and it is available by calling HP support at (303) 353-7650.

WORKAROUND

To avoid the "Printer is busy..." error message, disable background printing in Word:

  1. From the Tools menu, choose Options.

  2. Choose the Print tab.

  3. Clear the Background Printing check box.

To avoid the "Error saving raster files..." error message, disable the Include A Cover Page and the HP Header on Each Page option in the fax driver:

  1. Compose the fax in Word.

  2. From the File menu, choose Print, and then choose OK (or just click the Print icon).

    A Fax Options dialog box will appear.

  3. Clear the Include A Cover Page check box.

  4. Clear the HP Header on Each Page check box.

NOTE: You may want to create a fax cover page in Word to be faxed with your document.
