Grammar Checker, Word 2.0 Converter Missing from Typical Setup

 Last reviewed: February 6, 1998
Article ID: Q135238
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

When running Word for Windows 95, you notice several components are missing. For example:

  • The Grammar command is missing from the Tools menu, or when you look under Tools Options, the Grammar tab is unavailable (dimmed).
  • In the File Save As box, the option to save a file as Word 2.x is missing.

CAUSE

The grammar checker and the Word 2.0 converter are among the components not installed with a Typical installation. Shared applications such as WordArt and Equation Editor, as well as most graphics filters, are also not installed when you choose Typical during Setup.

This behavior was addressed in Office versions 7.0a and 7.0b. When you select Typical Setup in Office 7.0a or 7.0b, the following components are installed: Grammar Checker, WordMail, Resume Wizard, Contemporary Resume, Elegant Resume, and Professional Resume.

RESOLUTION

If you want to add these components, run Setup and add them using the Add/Remove button.
