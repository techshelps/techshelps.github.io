SGML Author Support Policy

 Last reviewed: October 10, 1997
Article ID: Q154264
The information in this article applies to:
  • Microsoft SGML Author for Word, versions 1.0, 1.1
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0, 7.0a

SUMMARY

SGML Author is an add-on tool and converter for Microsoft Word. SGML Author 1.0 is designed for Word for Windows 6.0, 6.0a, and 6.0c. SGML Author 1.1 is designed for Word for Windows 95, versions 7.0, and 7.0a.

HOW TO GET SUPPORT FOR SGML AUTHOR

If you have a question about Microsoft Word only, contact Microsoft Technical Support at (425) 462-9673.

If you have a question about SGML Author, contact the Microsoft Technical Support Priority Comprehensive Line.

Calls into the Microsoft Technical Support Priority Comprehensive Line will be handled on a call-back basis. You will be charged a flat rate of $195 per call. 

   (900) 555-2100; $195 (US) per incident. Charges appear on your telephone
   bill.

   (800) 936-5900; $195 (US) per incident, billed to your Visa
   card, MasterCard, or American  Express card.
For the most up-to-date information about Priority Comprehensive support, please see the Microsoft Web site at:
