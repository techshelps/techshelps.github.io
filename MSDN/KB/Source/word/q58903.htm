WD: How to Install Graphics Filters

 Last reviewed: July 30, 1997
Article ID: Q58903
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a, 2.0

SUMMARY

In Microsoft Word for Windows version 1.0, you can update the graphics filter information in the win.ini file by openning the EXAMPLES.DOC file in the LIBRARY directory and following the steps outlined below. To install the graphics filters in Word for Windows 1.1, 1.1a, and 2.0 versions, you must run the Setup program from the original diskettes.

To install the graphics filters in Word for Windows 1.0, do the following:

  1. From the File menu, choose Open (Press ALT+F, O).

  2. Open the LIBRARY directory and select the EXAMPLES.DOC file. Choose the Open button.

  3. In the section titled "InstallGraphicsFLTs", choose the Install button by either clicking it or placing the cursor on it and pressing ENTER.

The macro will check the [WWFilters] section of the WIN.INI file and install all the default graphics filters that were not installed.
