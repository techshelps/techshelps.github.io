WD97: Error Installing Word Foreign Language Proofing Tools

 Last reviewed: February 6, 1998
Article ID: Q163096
The information in this article applies to:
  • Microsoft Word 97 for Windows

SYMPTOMS

When you attempt to install the ALKI Foreign Language Proofing Tools designed for Word 7.0 on a computer running Microsoft Word 97 for Windows, you will receive the following error message: 

   Setup is unable to locate Word 7.0. Make sure Word 7.0 is installed,
   then run Proofing Tools Setup again.
When you click OK, the following error message is displayed: 

   Microsoft Word Proofing Tools Setup was interrupted before your new
   software could be fully installed. You can run the Microsoft Word
   Proofing Tools Setup again at a later time to complete the
   installation.

CAUSE

The Alki Foreign Language Proofing Tools software was designed to work with Microsoft Word for Windows 95, versions 7.0 and 7.0a.

RESOLUTION

Contact Alki Software Corporation for an updated version of the Alki Foreign Language Proofing Tools program.

For additional information, please see the following articles in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q82061
   TITLE     : Proofing Tools Language Numbers

   ARTICLE-ID: Q107223
   TITLE     : Foreign Language Dictionaries for Word
