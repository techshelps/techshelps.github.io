Ribbon Buttons Contain Dots Using Tseng Labs ET4000 Chip Set

 Last reviewed: July 30, 1997
Article ID: Q97554
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c

SUMMARY

In Word for Windows, formatting buttons on the ribbon (such as the Bold, Italic or Underline buttons) contain four dots each when selected. The paragraph alignment buttons contain two dots each when selected. These extraneous dots may appear in a color such as red or blue. This happens when your video card contains the Tseng Labs ET4000 chip set and 1 megabyte (MB) of RAM and you run the Super VGA (SVGA) video driver supplied with Windows.

Workaround

Use the standard VGA video driver supplied with Windows instead of the Super VGA driver.
