WD: Underlining Different Font Sizes Doesn't Line Up in MacWord

 Last reviewed: February 2, 1998
Article ID: Q51557
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 4.0, 5.0, 5.1, 5.1a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

If you format a line of text or a word as underlined and you have applied two or more different font sizes, the text looks fine on the screen, but the increased font size causes a thicker underline to be printed.

STATUS

This behavior is by design. As the font size increases, the size of the character formatting increases. Since the underline format is a character format, the underline lines will increase proportionally.
