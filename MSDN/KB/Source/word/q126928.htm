WD: Avery Label 5389 Does Not Print Correctly

 Last reviewed: February 3, 1998
Article ID: Q126928
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, version 7.0, 7.0a
  • Microsoft Word for the Macintosh, version 6.0, 6.0.1, 6.0.1a

SYMPTOMS

When you print on Avery 5389 labels, the text does not print correctly on the label.

CAUSE

The automatic Side Margin setting for label size 5389 is incorrectly set to 1.75 inches. This causes the label text to be skewed off the label. The margin should be set to 1.25 inches.

WORKAROUND

To print on Avery 5389 labels correctly, use the following steps:

  1. On the Tools menu, click Envelopes And Labels.

  2. Select the Labels tab, and then click Options.

  3. In the Product Number list, select 5389-Postcard.

  4. Click Details, and in the Custom Laser Information dialog box, change the Side Margin to 1.25 inches.

  5. Click OK, and then click OK again when asked if you want to save the changes.

NOTE: This is a temporary workaround. Because the Avery label sizes shown in the Custom Laser Information dialog box are part of Word's internal settings (which you cannot modify permanently), you will have to change the Side Margin setting from 1.75 inches to 1.25 inches each time you want to print these labels. (To work around this problem, add the labels to a new document, format the labels as desired, and save the document as a document template.)

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. This problem was corrected in Microsoft Word 97 for Windows and Word 98 Macintosh Edition.
