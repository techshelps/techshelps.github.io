WD97: Some Graphic Filters Not Installed During Setup

 Last reviewed: February 4, 1998
Article ID: Q158009
The information in this article applies to:
  • Microsoft Word 97 for Windows
  • Microsoft Office 97 for Windows

SYMPTOMS

You are unable to insert a clipart or graphic file in Word and you may receive the following error message: 

   Microsoft Word: An error occurred while importing this file. <filename>

CAUSE

When you try to insert a graphic or clipart picture into Word 97, a graphic filter must be available for Word to convert the graphic or clipart picture. If Word does not recognize the graphic type, Word presents the Convert File dialog.

If the filter for the graphic you are inserting is not present on this list, then the filter is not installed. Follow the steps below to install the graphic filter.

If the graphic filter is listed here, and you select it and click OK, and the graphic still does not insert, the filter or the graphic may be bad.

RESOLUTION

  1. Quit Word and/or Office.

  2. On the Start menu, point to Settings, then click Control Panel.

  3. Double-click Add/Remove Programs.

  4. Click the Install/Uninstall tab.

  5. In the list of installed software, select either Microsoft Word 97 or Microsoft Office 97. Then click Add/Remove. The Setup program starts.

  6. On the Setup screen, click Add/Remove.

  7. On the Maintenance screen, under Options, select Converters and Filters. (Do not clear the check mark if it already exists for Converters and Filters.) Then click Change Option.

  8. On the Converters and Filters screen, select (do not remove the check mark if it exists) Graphic Filters. Then click Change Option.

  9. On the Graphic filters screen, select each of the desired graphic filters to install. Then click OK.

  10. Click OK to move back to the Setup Maintenance screen. Then click Continue and complete the Setup process.

There are five graphic filters that are built-into Word 97. These filters are embedded into the Winword.exe and are independent of the filters installed through the Setup program:

  • Enhanced Metafile Import (EMS)
  • Windows Bitmap Import (BMP)
  • Windows Metafile Import (WMF)
  • JPEG File Interchange Format Import (JPG)
  • Portable Network Graphics (PNG)

These four graphic filters are installable through the Setup program for compatibility reasons with the rest of the Microsoft Office Suite. In other Words, these filters are installed so that graphics can be imported into the other Office applications, such as Excel.

For more information on exactly which components are installed during each type of Setup, please check Word help:

  1. On the Help menu, click Contents and Index.

  2. On the "Help Topics: Microsoft Word" screen, select the Contents tab.

  3. Double-click the "Installing and Removing Word" book.

  4. Double-click the "Install or remove individual components" topic.

  5. On the help window, under step 5, click the arrow button for the line that states: "For the location of components in Setup..."
