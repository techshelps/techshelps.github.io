DMM: Not All Addresses Checked For Duplicates

 Last reviewed: March 17, 1998
Article ID: Q182607
The information in this article applies to:
  • Microsoft Direct Mail Manager for Windows

SYMPTOMS

When you use Direct Mail Manager to verify your addresses, not all addresses are checked for duplicates.

CAUSE

This functionality is by design. Direct Mail Manager will not check non- verified addresses for duplicates.

NOTE: Only addresses that are verified can be included in a discount mailing.

WORKAROUND

To work around this behavior, either correct or remove the non-verified addresses in your address list.

Correcting Non-Verified Addresses

For more information about correcting (editing) your address list, click Help in Direct Mail Manager, click the Index button in Direct Mail Manager Help, type the following text 

   Edit addresses
and then double-click the selected text to go to the "Edit List step" topic.

Removing Non-Verified Addresses

To remove non-verified addresses, follow these steps:

  1. In Direct Mail Manager, step through the process of importing and verifying your address list.

  2. When Direct Mail Manager shows you the "Address Verification Results" dialog box, click OK.

  3. When Direct Mail Manager prompts you with "To edit, click anywhere and type the correct information," change Limit List To "Non-verified Addresses."

  4. Select each non-verified address (and duplicate) by holding down CTRL and clicking on the little gray box directly to the left of each address.

  5. Once the non-verified addresses are all selected, press DELETE to delete the non-verified addresses.

  6. Click Yes when the following message is displayed: 

          Are you sure you want to delete these addresses?

Once the non-verified addresses are deleted from your updated address list, click Next to continue Direct Mail Manager.

MORE INFORMATION

After your address list has been imported, Direct Mail Manager runs the Address Verification step.

NOTE: If Direct Mail Manager does not verify your address list, follow these steps:

  1. Click the Back button to return to the prompt "Your address list has been imported."

  2. Click Options.

  3. On the Optional Steps tab, click to select the Run Address Verification check box.

  4. If you want Direct Mail Manager to check your verified addresses for duplicates, click to select the Check For Duplicates check box.

    NOTE: Under Check For Duplicates, you can choose how you want Direct Mail Manager to check for duplicate addresses. Select one of the following: 

        - Identical ZIP Code (all 11 digits)
    - Identical ZIP Code and delivery address
    - Identical ZIP Code, delivery address, and name

  5. Click OK.

When you click Next, Direct Mail Manager verifies your address list and shows you address verification results similar to the following: 

   Total number of addresses:                     <number>
   Addresses with verified 9-digit ZIP Codes:     <number>  <percentage>
   Addresses with verified 5-digit ZIP Codes:     <number>  <percentage>
   Not verified or non-USA addresses:             <number>  <percentage>
When you click OK, Direct Mail Manager will allow you to edit the address list results.
