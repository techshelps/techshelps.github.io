List of Items Installed with Custom Installation in 5.1

 Last reviewed: July 30, 1997
Article ID: Q92663
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.1

SUMMARY

When you are installing Word, you have the option of performing a custom installation. To perform a custom installation, choose the Customize button in the Easy Install dialog box and select the desired options. The following text lists each package available in the Custom Install dialog box. The first entry below shows the format of each package entry listed.
