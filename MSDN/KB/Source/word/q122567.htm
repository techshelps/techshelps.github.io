WD: Default Font Not Shown in Envelopes And Labels Dialog Box

 Last reviewed: February 5, 1998
Article ID: Q122567
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows NT, version 6.0

SYMPTOMS

When the default font is something other than Times New Roman, the text in the Envelopes And Labels dialog box under the Labels tab is still displayed in Times New Roman.

CAUSE

When you choose the New Document button to create a document of labels, Word uses the actual default font, despite how the text appears on the Labels tab.

Similarly, if you choose the Print button, Word uses the correct default font to print the sheet of labels.

RESOLUTION

To resolve this problem, upgrade to the latest version of Microsoft Word.

STATUS

Microsoft has confirmed this to be a problem in the versions of Word listed above. This problem was corrected in Microsoft Word 97 for Windows.
