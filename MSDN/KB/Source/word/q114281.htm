WD: Custom Caption Labels Not Stored with Document

 Last reviewed: February 5, 1998
Article ID: Q114281
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

If you define a custom caption label for a document and then delete or rename your NORMAL.DOT template or open the document on another computer, Word for Windows does not display the custom caption names in the Insert Caption dialog box. As a result, you must redefine the custom caption label if you want to insert additional captions in your Word document.

This problem does not affect the three default caption labels (Figure, Equation, and Table), which are part of the Word program, not part of the Normal template.

CAUSE

Even though custom captions are document-specific information, Word stores the custom caption label in your Normal template (NORMAL.DOT). Therefore, if that template is not available, Word has no record of the custom caption labels.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
