Word 2.0 Cannot Open Word 6.x/7.x File With Bookmark on Win95

 Last reviewed: February 5, 1998
Article ID: Q155346
The information in this article applies to:
  • Microsoft Word for Windows, versions 2.x

SYMPTOMS

When you open a Word 6.0/7.0 document in Microsoft Word for Windows 2.0x, you may receive the following error messages: 

   Microsoft Word 6.0 Conversion: Not enough memory to complete operation
followed by: 

   Microsoft Word: Word cannot open the document (<path> <filename>)

CAUSE

This problem occurs if the document contains a bookmark.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Save the document from Word 6.x/7.0 as a "Word for Windows 2" document. Use one of the following procedures to save the document.

Method 1: If you are using Word 6.0: 

          a. In Word 6.0, open the document.

          b. On the File menu, click Save As.

          c. In the Save As dialog box, select the "Word for Windows 2.0"
             file type,  and click OK.
Method 2: If you are using Word 7.0: 

          a. In Word 7.0, open the document.

          b. On the File menu, click Save As.

          c. In the Save As dialog box, select the "Word 2.x for Windows
             (*.doc)" file type, and click the Save button.

MORE INFORMATION

Microsoft Word for Windows 2.0 does not contain a converter for Word 6.x/7.0 documents. There is a supplemental converter file required for Word 2.0 to convert these documents. This converter contains the problem as listed above when running it under Windows 95.

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q105318
   TITLE     : WD: How to Obtain a Word 6.x/7.0 Converter for Use with Word 
               2.x
