WD6X: Random Hangs, Unable to Fax with Geoport, System 7.5

 Last reviewed: November 17, 1997
Article ID: Q145917
The information in this article applies to:
  • Microsoft Word for the Power Macintosh, versions 6.0, 6.0.1

SYMPTOMS

Your system stops responding (hangs) or you can only fax intermittently when you use Geoport for the Power Macintosh with System 7.5.

CAUSE

System 7.5 on the Power Macintosh requires version 1.0.1 of Geoport for the Power Macintosh. Other Geoport versions are not compatible with System 7.5 on the Power Macintosh.

WORKAROUND

Install Geoport for the Macintosh, version 1.0.1. This version is available on the System 7.5 CD-ROM.

For more information about Geoport, contact Apple Computer, Inc., at (800) SOS-APPL.
