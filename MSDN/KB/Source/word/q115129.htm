GP Fault with Microsoft Excel Object When Exiting Word

 Last reviewed: February 5, 1998
Article ID: Q115129
6.00 6.00a 6.00c WINDOWS kbole kbinterop buglist5.00 fixlist5.00c

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Excel for Windows, version 5.0c

SYMPTOMS

In Word for Windows, a general protection (GP) fault in EXCEL.EXE is generated if you insert a Microsoft Excel object, begin entering data into the object worksheet, and then close Word without finishing the entry in the object worksheet by pressing ENTER.

CAUSE

The OLE2 link is still active at the time Word is closed.

STATUS

Microsoft has confirmed this to be a problem in Excel for Windows version 5.0. This problem was corrected in Excel for Windows version 5.0c
