Underlined Bullet Doesn't Show Underline At 115% Zoom

 Last reviewed: February 5, 1998
Article ID: Q143265
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

A bullet with underline formatting applied to it may not display the underline in Normal view.

CAUSE

This behavior occurs when the Zoom Control is set to 115%. If the Zoom To setting is Page Width, the Zoom Control setting will automatically be 115%.

STATUS

Microsoft has confirmed this to be a problem in Word for the Macintosh versions 6.0 and 6.0.1. Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

WORKAROUND

Switch to Page Layout view, or use a different Zoom Control setting. To change the Zoom Control setting, click Zoom on the View menu. Select one of the predefined Zoom To settings (except for Page Width), or change the value in the Percent list. Click OK.

MORE INFORMATION

You can apply underline formatting to a bullet by underlining the paragraph mark associated with the bulleted paragraph.

The Zoom Control setting determines the magnification of the text shown in a particular view; it does not affect the font size.
