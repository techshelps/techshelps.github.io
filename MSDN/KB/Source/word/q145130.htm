Callout or Text Box Printed Without Color Fill

 Last reviewed: February 5, 1998
Article ID: Q145130
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

A callout or text box that is positioned on top of another image and that is formatted with a color fill, is printed as though it does not contain a fill.

CAUSE

This problem occurs when you print to the Hewlett-Packard (HP) 4si/MX printer using the HP LaserJet 4 printer driver, version 3.78, and the Universal Printer driver version 4.0 that shipped with Windows 95.

This problem does not occur when the text box is placed on top of a Windows Metafile (WMF) formatted graphic.

WORKAROUND

To work around this problem, use either of the following methods:

Method 1: Use the PostScript version of the HP LaserJet 4si/MX printer 

          driver.
Method 2. Change the graphics mode from Vector to Raster. To do this, in 
          Word, use the following steps:

          a. On the File menu, click Print.

          b. Click the Properties button.

          c. Click the Graphics tab, select the Use Raster Graphics option,
             click Apply, and then click OK.

          d. Click OK to print the document or click Close to continue
             editing the document.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Word version 7.0 for Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
