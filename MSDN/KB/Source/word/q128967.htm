Keystroke Changes Between Word 5.1 and 6.0

 Last reviewed: February 5, 1998
Article ID: Q128967
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SUMMARY

This article lists the shortcut key changes between Word 5.1 and Word 6.0.

Note that COMMAND+SHIFT+DOWN ARROW and COMMAND+SHIFT+UP ARROW select an entire paragraph in Word version 5.0. Word 6.0 does not have a similar feature. The following macro simulates this behavior when assigned to either of those key combinations: 

     Sub Main
        EditGoTo "\para"
     End Sub
