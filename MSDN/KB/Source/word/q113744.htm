Word Does Not Find Text Plus Field Results

 Last reviewed: February 5, 1998
Article ID: Q113744
6.00 6.00a WINDOWS kbusage

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a

SYMPTOMS

When you use the Find or Replace command (on the Edit menu), Word for Windows does not find the search string if the string contains both field code results and plain text.

Word finds field results, and Word finds plain text, but Word cannot find a combination of the two.

STATUS

Microsoft has confirmed this to be a problem in Word versions 6.0 and 6.0a for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
