Spelling Checker Replaces Flagged Word with Page Break

 Last reviewed: February 5, 1998
Article ID: Q120089
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SYMPTOMS

If you select a word and run the spelling checker, Word for Windows may delete the word and insert a page break instead of checking the spelling of the word.

CAUSE

This happens if WordPerfect Help is turned on.

Note: When WordPerfect Help is turned on, the following entry appears in the WINWORD6.INI file: 

   WPHELP=1

WORKAROUND

Turn off WordPerfect Help using the following procedure:

  1. From the Help menu, choose WordPerfect Help.

  2. Choose the Options button.

  3. Clear the "Help for WordPerfect Users" and "Navigation Keys for WordPerfect Users" options and then choose OK.

  4. In the Help For WordPerfect Users dialog box, choose the Close button.

REFERENCES

"Microsoft Word User's Guide," version 6.0, page 17
