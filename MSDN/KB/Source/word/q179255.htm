WD98: Supported Graphics in Microsoft Word 98

 Last reviewed: February 17, 1998
Article ID: Q179255
The information in this article applies to:
  • Microsoft Word 98 Macintosh Edition

SUMMARY

This article lists the graphics filters that are included with Microsoft Word 98.

MORE INFORMATION

Graphics filters are installed in the following default location: 

   Microsoft Office 98:Shared Applications:Graphic Filters

Graphics Formats Native to Microsoft Word

The following graphics formats are native to Microsoft Word:

  • PICT
  • Windows Bitmap (BMP)
  • WMF (Word for Macintosh converts WMF to PICT format)

Support for GIF, JPEG and PNG graphics is new to Microsoft Word 98. You may find filters for some of these formats installed on your computer. They are used by other Microsoft Office applications and by some of the Microsoft Word Text Converters.

Graphics Filters Not Included with Microsoft Word

Microsoft Word 98 does not include graphics filters for the following file types:

  • AutoCAD (DXF)
  • Computer Graphics Metafile (CGM)
  • CorelDraw 3.0 to 6.0 (CDR)
  • Micrografx Designer/Draw (DRW)
  • PC Paintbrush
  • TrueVision Targa (TGA)

Note, however, that some third-party software vendors may include Macintosh filters for these file formats.
