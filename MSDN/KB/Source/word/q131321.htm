MacWord 6.0 Fails to Start or Is Slow with Suitcase

 Last reviewed: November 17, 1997
Article ID: Q131321
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1

SYMPTOMS

When using Word 6.0 with the Symantec Suitcase Extension, Word may not start, or it may be slower to load, sometimes taking several minutes.

CAUSE

Word 6.0 will not start if you are running System 7.1 and if Suitcase 2.1 or earlier is installed.

RESOLUTION

Upgrade to Suitcase 2.13 or later.

NOTE: If Word is still unable to run with Suitcase 2.13 or later, it is possible that a corrupted font is being installed by Suitcase that is preventing Word from starting. Remove the fonts from the Suitcase, and Word will be able to start.

MORE INFORMATION

Word scans for the active fonts in the System Fonts:Folder when it starts. If Word encounters a corrupted font, Word may stop responding or run slowly.

Suitcase allows for fonts to be activated in other folders besides the Fonts folder, so a corrupted font can be active in memory other than in the System Folder.
