Key Combination for Update Command in Equation Editor

 Last reviewed: July 30, 1997
Article ID: Q80245
The information in this article applies to:
  • Microsoft Word for the Macintosh, version 5.0

SUMMARY

Page 85 of the "Microsoft Word Equation Editor User's Guide" for Microsoft Word version 5.0 for the Macintosh states that COMMAND+U can be used as a keyboard shortcut for the Update command, which is available on the File menu. This information is incorrect.

While the Update command works correctly when it is chosen from the File menu, pressing COMMAND+U produces a beep. In addition, the COMMAND+U key combination doesn't appear on the File menu.

MORE INFORMATION

The Update command is active only when using Apple System 7.0 or later. This command updates the equation in a Word document without closing the Equation Editor window.
