WD: Ampersand Displays Incorrectly in Drop-Down Form Field

 Last reviewed: February 3, 1998
Article ID: Q125892
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1, 6.0.1a
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word 97 for Windows
  • Microsoft Word 98 Macintosh Edition

SYMPTOMS

If you are creating a drop-down form field, and one of your list items includes an ampersand (&), the resulting item in the drop-down list is missing the ampersand, and the character adjacent to the ampersand may be underlined. For example, "AT&T" (without the quotation marks) may appear in the drop-down list as: 

   ATT
     -
If you try to work around the problem by adding a second ampersand, the drop-down list will correctly display "AT&T" (without the quotation marks), but when you select "AT&T," it displays in the form field box as 

   "AT&&T"
   (without the quotation marks).
In Word 97 for Windows or Word 98 Macintosh Edition, use of the ampersand has the following effects:

Word 97 for Windows

  • Using a single ampersand, as in AT&T, does not display the ampersand but the character next to the ampersand is underlined, as in

    ATT 

         -

    In this case, pressing T selects ATT from the drop-down list.

  • Using a double ampersand, as in AT&&T, displays a single ampersand, as in

    AT&T

    but the character next to the ampersand is not underlined. In this case, pressing T does not select AT&T from the drop down list.

Word 98 Macintosh Edition

  • Using a single ampersand, as in AT&T, does not display the ampersand nor is the character next to the ampersand underlined, as in

    ATT

  • Using a double ampersand, as in AT&&T, displays a single ampersand, as in

    AT&T

    but the character next to the ampersand is not underlined. In both cases, pressing T does not select AT&T from the drop down list.

CAUSE

By design, the ampersand is used to designate the shortcut key in drop-down lists for quick keyboard access. The shortcut key is underlined.
