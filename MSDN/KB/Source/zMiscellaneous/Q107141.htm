Word 6.0 for Windows Supplemental File Conversions Disks

 Last reviewed: October 14, 1997
Article ID: Q107141
6.00 6.00a 6.00c WINDOWS kbinterop kbfile

The information in this article applies to:

  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

Included below is information on how to obtain the Microsoft Word 6.0 for Windows Supplemental File Conversions Disks, as well as how to obtain the these same converters from online services. The contents of the cover letter that accompanies the Supplemental File Conversions Disks are also included, and the cover letter provides installation instructions.

Supplied Text Converters

The supplied text converters support files in the listed formats. Unless otherwise specified, conversion is supported both to and from Word for Windows. "Import only" means conversion is only supported to Word for Windows.

  • DisplayWrite version 4v1, 4v2, and 5.0 formats (import only)
  • Lotus 1-2-3 version 2.x and 3.x formats (import only)

    NOTE: An updated Lotus 1-2-3 converter that adds support for Lotus 1-2-3 version 4.x is now available separately. For additional information, please see the following article(s) in the Microsoft Knowledge Base:

    ARTICLE-ID: Q124871 

       TITLE     : GC1133: Lotus 1-2-3 Converter Update Available for WinWord
               6.0
  • Microsoft Publisher for Windows version 2.0 format
  • Microsoft Word for MS-DOS glossary files version 5.0, 5.5, and 6.0 formats (import only)
  • Microsoft Word for Windows version 1.x format (only required for export)
  • Microsoft Works for MS-DOS format
  • Microsoft Works for Windows version 2.0 and 3.0 formats
  • MultiMate version 3.3, 4.0, MultiMate Advantage, and MultiMate Advantage II formats
  • Multiplan version 3.x and 4.2 format (import only)
  • WordPerfect version 4.1 and 4.2 format
  • WordStar version 3.3, 3.45, 4.0, 5.0, 5.5, 6.0 and 7.0 format

Supplied Graphics Filters

The supplied graphics filters support files in the listed formats. All supplied filters only support conversion to Word for Windows.

  • AutoCAD (DXF) format (import only)
  • AutoCAD plotter files (ADI) format (import only)
  • Corel DRAW 3.0 (CDR) format (import only)
  • HP Graphics Language (HPGL) / HP Plotter Print File (PLT) format (import only)
  • Kodak Photo CD (PCD) format (import only)
  • Lotus 1-2-3 graphics file (PIC) format (import only)
  • Truevision Targa (TGA) format (import only)

Setup Instructions

  1. Start Microsoft Windows.

  2. Insert the disk labeled Supplemental File Conversions--Setup in drive A.

  3. From Program Manager, choose Run from the File menu.

  4. Type "a:\setup" (without the quotation marks) and press ENTER.

For more information about using Word with documents created in other word- processing programs, see the "File Conversion" topic of the Word Readme Help (WDREADME.HLP) file installed with Word 6.0 for Windows. More information about the supplied converters and filters is included in the online document SCNVINFO.DOC, which is installed with the converters and filters.
