Files Installed by DirectX 2.0a

 Last reviewed: January 14, 1997
Article ID: Q158960
The information in this article applies to:
  • Microsoft Close Combat for Windows, version 1.0a
  • Microsoft Deadly Tide for Windows, version 1.0
  • Microsoft Flight Simulator for Windows 95, version 6.0
  • Microsoft GEX, version 1.0
  • Microsoft Hellbender for Windows, version 1.0
  • Microsoft Monster Truck Madness, version 1.0
  • Microsoft NBA Full Court Press for Windows, version 1.0
  • Microsoft Soccer, version 1.0
  • Microsoft Golf, version 3.0

SUMMARY

The following is a list of the DirectX system files installed to your hard drive by DirectX 2.0a.

NOTE: DirectX 2.0 includes newer versions of DirectX 1.0 files plus the introduction of Direct3D files.

MORE INFORMATION

All files are marked as version 4.03.00.1096.

Direct3D

  • D3dhalf.dll
  • D3dim.dll
  • D3drampf.dll
  • D3drg16f.dll
  • D3drg24f.dll
  • D3drg32f.dll
  • D3drg8f.dll
  • D3drgbf.dll
  • D3drm.dll
  • D3drm16f.dll
  • D3drm24f.dll
  • D3drm32f.dll
  • D3drm8f.dll
  • D3dxof.dll

DirectDraw

  • Ddhelp.exe
  • Ddraw.dll
  • Ddraw16.dll

DirectPlay

  • Dplay.dll
  • Dpserial.dll
  • Dpwsock.dll

DirectSound

  • Dsound.dll
  • Dsound.vxd
