WD: History of WinWord File Converters Categorized by Release

 Last reviewed: March 6, 1998
Article ID: Q96119
The information in this article applies to:
  • Microsoft Word for Windows, versions 1.0, 1.1, 1.1a
  • Microsoft Word for Windows, versions 2.0, 2.0a, 2.0a-CD, 2.0b, 2.0c,
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SUMMARY

The following is a history of the file converters that have shipped with the U.S. releases of Word, have been made available as part of the U.S. releases of Word Supplemental File Conversions Disks, or have been made available separately.

For additional historical information, see the More Information section at the end of this article.

MORE INFORMATION

  • In some early versions of Word, menu options for some formats erroneously do not list all supported versions. For example, in Word versions 1.0, 1.1, and 1.1a, the WordStar converter only lists support for WordStar 3.3, 3.45, and 4.0 when in fact the converter also officially supports versions 5.0 and 5.5, and is unofficially capable of reading WordStar 6.0 and 7.0 files.
  • Some converters are capable of reading information from versions or sources of a non-Word file format other than those listed in the Open dialog box (on the File menu, click Open).
  • All versions of FoxBASE+ for MS-DOS and for the Macintosh, as well as FoxPro 2.0 for MS-DOS and FoxPro 2.5 for MS-DOS and Windows, use the dBASE III+/IV file format as their internal file format. Therefore, although we do not specifically list an option to open FoxBASE+ or FoxPro files in Word, these files can be opened as dBASE files in Word versions prior to Word 6.0.

    The only change to the dBASE converter in Word version 2.0b was a correction to a problem with the converter not recognizing FoxPro 2.0 database files that contain memo fields.

    In Word 6.0 and later, the dBASE converter has been discontinued, and it's functionality has been replaced by use of ODBC to import data from FoxBase, FoxPro, dBASE and other database formats.

  • The Works 2.0 for MS-DOS converter can read Works version 3.0 for MS-DOS word processing files; it will ignore formatting features newly introduced in Works version 3.0 for MS-DOS.
  • A Works 3.0 for Windows converter was first introduced as part of the Word 6.0a Supplemental File Conversions Disks.
  • The WordPerfect 5.x converter can read data from WordPerfect versions 5.0, 5.1, and 5.1 Plus for MS-DOS, as well as from WordPerfect 5.1 and 5.2 for Windows documents.
  • Some files in the Word 2.0a-CD for Windows release have different file dates because the supplemental converters were included in the WORD\SUPPCONV directory on the CD-ROM and were not file-date synched when the CD-ROM was built.
