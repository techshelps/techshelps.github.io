Invalid Page Fault When Internet Explorer Displays a Web Site

 Last reviewed: December 1, 1997
Article ID: Q170447
The information in this article applies to:
  • Microsoft Internet Explorer version 3.02 for Windows 95
  • Microsoft Internet Explorer version 3.02 for Windows NT 4.0

SYMPTOMS

When you visit a Web site that attempts to load a Java program, you may receive the following error message: 

   IEXPLORE caused an invalid page fault in module MSJAVA.DLL at
   137:7c025b57.

CAUSE

This error message can occur when the Java virtual machine (VM) byte code verifier cannot properly load data for a Java program.

RESOLUTION

Install the latest Microsoft Java Virtual Machine for Internet Explorer update. To obtain the latest Java VM update, visit the following Microsoft Web site: 

   http://www.microsoft.com/java/
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
