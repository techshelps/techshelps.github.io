PUB97: Publisher Setup Quits After MSVCRT40.DLL Error Message

 Last reviewed: March 2, 1998
Article ID: Q165642
The information in this article applies to:
  • Microsoft Publisher 97

SYMPTOMS

When Microsoft Publisher 97 Setup is almost finished, you receive the following error message: 

   This version of MSVCRT40.DLL is intended for Win32s only.
When you click OK, you receive the following additional message: 

   Artgalry.dll was not registered correctly.
When you click OK, Setup quits. Publisher does not run successfully.

NOTE: This problem may also occur in other applications the use the same setup program.

CAUSE

The copy of Msvcrt40.dll in your C:\Windows\System folder is out of date.

NOTE: This error does not occur if you are running Windows NT Workstation, version 4.0. If you have Windows NT 4.0, you should not replace Msvcrt40.dll.

RESOLUTION

Make sure you have only one copy of Msvcrt40.dll on your hard disk. You can use the Windows 95 Find File feature to search for all copies of this file. Rename any copies of Msvcrt40.dll that are not in the C:\Windows\System folder.

If you still cannot run Publisher setup correctly, replace the copy of Msvcrt40.dll on your hard disk with a copy from the installation media. If you installed Publisher from compact disc, copy Msvcrt40.dll from the root directory on the Publisher 97 compact disc to the C:\Windows\System folder.

If you installed Publisher from disks, use the Extract utility to replace this file. The steps below assume you installed Windows 95 into the default folder and installed Publisher from the A: drive.

  1. Click the Start button, point to Programs, and then click MS-DOS Prompt.

  2. Type the following command, and then press ENTER: 

          cd\windows\system

    Your command prompt now looks similar to this: 

          C:\Windows\System>

  3. Type the following command, and then press ENTER: 

          ren msvcrt40.dll msvcrt40.old

  4. Insert Publisher Disk 2 into the A: drive.

  5. Type the following command, and then press ENTER: 

          extract a:\pub4_2.cab msvcrt40.dll

  6. When the Extract utility is finished, type the following command, and then press ENTER: 

          exit

NOTE: After you replace Msvcrt40.dll, rerun Publisher setup to install it correctly.
