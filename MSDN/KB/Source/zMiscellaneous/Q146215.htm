This document provides information on how to install and uninstall the updated Point32 drivers for IntelliPoint 1.1, IntelliType 1.1, EasyBall 1.0. The Point32 drivers have been modified to address known problems with the SnapTo and PointerWrap features that were causing General Protection Faults in some 16-bit applications. Changes have also been added to fix problems interacting with some third party software programs.

To view the Readme file on-screen in Windows Notepad, maximize the Notepad window and click Word Wrap on the Edit menu. To print the Readme file, open it in Notepad or another word processor, and then use the Print command on the File menu.

1. INSTALLING THE UPDATED POINT32 DRIVERS

If Windows 95 is installed on your computer and you wish to install the updated Point32 drivers, one of the following must also be installed on your computer:

  • Microsoft IntelliPoint version 1.1
  • Microsoft IntelliType version 1.1
  • Microsoft EasyBall version 1.0

To install the Point32 drivers on a computer running Windows 95, double- click Setup.exe, and then follow the instructions on the screen.

2. UNINSTALLING THE UPDATED POINT32 DRIVERS

Perform one of the following steps to uninstall the updated Point32 drivers:

  • Run Setup from any of the following products: 

        - Microsoft IntelliPoint version 1.1
    - Microsoft IntelliType version 1.1
    - Microsoft EasyBall version 1.0

    Reinstallation of the earlier software prompts you for confirmation before overwriting the updated drivers. Choose OK to accomplish this.

    -or-

  • Uninstall the files manually:

    1. Restart your computer. When you see the "Starting Windows 95" 

          message, press the F8 key, and then choose Safe Mode Command Prompt
      Only from the Startup menu.

    2. Change to the folder where you installed IntelliPoint, IntelliType, 

          or EasyBall (typically this is the C:\MSINPUT folder).

    3. Copy Point32.e11 to Point32.exe.

    4. Copy Point32.d11 to Point32.dll.

    5. Restart Windows 95 for the original drivers to take effect.
