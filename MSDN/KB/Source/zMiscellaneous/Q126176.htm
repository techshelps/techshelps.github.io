WD6x: WordPerfect Conversion Kit Available

 Last reviewed: October 14, 1997
Article ID: Q126176

The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

A Supplemental WordPerfect Conversion Kit is available for Word 6.0, 6.0a, and 6.0c for Windows users. Converters supplied on this disk include an updated WordPerfect 5.x converter and an import-only WordPerfect 6.0 converter.

NOTE: For Word 6.0 for Windows NT users, the WordPerfect 5.x/6.x converters are now available through an Application Note (GC1183). For information on how to obtain this application note, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q133134
   TITLE     : WordPerfect 5.x/6.x Converters for WordNT Available

COMPONENTS SUPPLIED IN THE KIT

The kit includes two converters, five fonts, and two templates.

Two converters for use with Word 6.0 for Windows are:

  • WordPerfect 6.x import text converter that allows Word 6.0 for Windows to read files created by WordPerfect 6.0, 6.0a, and 6.0b for MS-DOS and WordPerfect 6.0, 6.0a, and 6.1 for Windows.
  • Updated WordPerfect 5.x import/export text converter that allows Word 6.0 for Windows to convert to and from WordPerfect 5.0, 5.1, and 5.1+ for MS-DOS format and WordPerfect 5.1 and 5.2 for Windows format.

The five TrueType fonts can be used by Word 6.0 for Windows for improving import conversion of characters from the following WordPerfect character sets: Greek, Iconic Symbols, Math/Scientific, Multinational, and Typographic.

Two WordPerfect templates ease the transition to Word for Windows (one template for WordPerfect for Windows users, the other template for WordPerfect for MS-DOS users). These templates emulate some of the keyboard shortcuts, macros, toolbars, and styles as in WordPerfect versions 5.x and 6.x.

SETUP INSTRUCTIONS

Use these steps to install the Conversion Kit:

  1. Quit Word if it is running.

  2. If you are installing the Conversion Kit from a floppy disk, insert the disk in the appropriate floppy disk drive.

    If you downloaded Wpcnv.exe from an online service, you'll need to expand the files so they can be installed:

    a. In File Manager, locate Wpcnv.exe and double-click the filename.

    b. Click OK.

    This will expand the files so that you can proceed to the next step.

  3. In File Manager, select the drive letter and directory where Wpcnv.exe is located. Locate the Setup.exe file. (If you ordered the kit on a disk, Setup.exe will be on the disk in drive A or B. If you downloaded the file, Setup.exe will be in the same directory as Wpcnv.exe.) Double-click Setup.exe to set up the conversion files.

IMPORTANT NOTES

Before you open WordPerfect documents in Word, make sure the Confirm Conversions check box is selected in the Open dialog box (click Open on the File menu).

After you select the name of the WordPerfect 6.0 or 6.1 document you want to open, a Convert File dialog box will appear. Select the WordPerfect 6.0 or 6.1 file type in the Convert File dialog box. You must do this each time you open a WordPerfect file.
