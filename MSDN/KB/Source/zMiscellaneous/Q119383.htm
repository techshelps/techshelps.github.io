AVI Files Available for Testing Video for Windows Runtime

 Last reviewed: May 1, 1997
Article ID: Q119383
1.00 1.10 1.10a WINDOWS kbmm kbdisplay kbtshoot kbfile

The information in this article applies to:

  • Microsoft Video for Windows versions 1.0, 1.1, 1.1a

SUMMARY

If you are experiencing problems with the run-time version of Microsoft Video for Windows, you may want to obtain the two .AVI files below to help troubleshoot the problem. The following AVI files ship with Video for Windows and are known to run properly: 

   SAMPLE.AVI    1,022,776 bytes
   WNDSURF1.AVI  1,042,796 bytes
These files are also available in AVISAMPL.EXE. 

You can find AVISAMPL.EXE (size: 1827729 bytes)

MORE INFORMATION

 


Because the run-time version of Video for Windows is shipped with a variety
of products, finding reliable .AVI files for troubleshooting can be
difficult. The only .AVI files available for testing are those that shipped
with the application, and they may be embedded within the application. As a
result, there is no definitive way to troubleshoot problems.


WNDSURF1.AVI and SAMPLE.AVI ship with the Video for Windows retail product
and are known to work correctly. If you are having problems with the run-
time version of Video for Windows, download these two files and try to
reproduce the problem. If you have no problems running these files, the
issue is unique to the third-party video files or the application itself.
For more information, contact the third-party product manufacturer. If you
can reproduce the problem with SAMPLE.AVI or WNDSURF1.AVI, contact
Microsoft Product Support Services at (425) 635-7172 for further
assistance.
