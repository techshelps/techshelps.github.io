WD: How to Obtain a Word 6.x/7.0 Converter for Use with Word 2.x

 Last reviewed: February 2, 1998
Article ID: Q105318
The information in this article applies to:
  • Microsoft Word for Windows 95, version 7.x
  • Microsoft Word for Windows, versions 2.x, 6.x
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for the Macintosh, version 6.x

SUMMARY

A converter is available that allows Microsoft Word 2.0 for Windows to read files created by Word 6.0 and 7.0 for Windows, Word 6.0 for Windows NT, and Word 6.0 for the Macintosh. Word 2.0 for Windows users who don't yet have immediate access to Word 6.0 or who want to convert Word 7.0 files can obtain this converter by ordering or downloading the Word 6.x-7.0 Converter Kit (for Word 2.0 for Windows users).

Alternatively, if a complete installation of Word 6.0 for Windows exists on the same computer as Word 2.0, an earlier version of the converter is immediately available for installation that allows Word 2.0 to open a Word 6.0 for Windows document. This earlier version of the converter cannot open a Word 6.0 for the Macintosh or Word 6.0 for Windows NT document.

A complete installation of Word 7.0 does not install a 16-bit converter for Word 2.0, and you must order or download the converter.

For information on obtaining and installing either version of the Word 6.0 converter, see the appropriate section below.

To Obtain the Word 6.x/7.0 for Windows, Windows NT, and Macintosh Converter

The following file is available for download from the Microsoft Software Library: 

 ~ Mswrd6.exe (size: 763107 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base:

   TITLE     : How to Obtain Microsoft Support Files from Online Services

Pacific time at (425) 462-9673. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at:

To Set Up the Word 6.x/7.0 for Windows, Windows NT, and Macintosh Converter

  1. Start Microsoft Windows.

  2. If you ordered the Word 6.x/7.0 Converter Kit, insert the disk you received in the appropriate floppy disk drive.

    If you downloaded Mswrd6.exe from an online service, run Mswrd6.exe to extract the files it contains and skip to step 3.

  3. Click the Start button, and then click Run.

  4. Type "source:\setup" (without quotation marks) and press ENTER. "Source" is either the letter of your floppy disk drive or the directory containing extracted contents of Mswrd6.exe.

Installing the Word 6.0 for Windows Converter as Supplied with Word 6.x

  1. Quit Microsoft Word 2.0 if it is running.

  2. Copy the file Msword6.cnv from the \Msapps\Textconv subdirectory in your Windows directory to the Word 2.0 for Windows program directory. If this file is not available, run Word 6.0 for Windows Setup again to install the converter and repeat step 2.

  3. Restart Word 2.0 and open a Word 6.0 for Windows document that has a .DOC file name extension. The converter should now be properly installed.

NOTE: Although the file type referenced during conversion of Word 7.0 files is Word 6.0, the converter, Msword6.cnv, is the correct converter to use.

If you have questions about installing or using either version of the converter, in the United States call Product Support Services at (206) 462- WORD (462-9673) between 6:00 A.M. and 6:00 P.M. Pacific time. Outside the United States, contact your local Microsoft subsidiary.

MORE INFORMATION

OLE 2.0 support is required for this converter to successfully read a Word 6.0 or 7.0 for Windows file. The converter therefore does not work correctly if copied to a computer on which only Word 2.0 is installed.

This converter does not run under Windows 3.0. This limitation exists because Word 6.0 for Windows use object linking and embedding (OLE) 2.0 for file storage, so several OLE 2.0 files are required for the converter to read Word 6.0 for Windows files. These OLE files are supplied with the converter but are not guaranteed to work under Windows 3.0.

Note also that those formatting features added in Word 6.0 or 7.0 for Windows that did not exist in Word 2.0 are not retained in conversion.

The version of Msword6.cnv that is installed with a complete installation of Word 6.0 is used automatically by Microsoft programs that share converters, including Microsoft Works for Windows and Publisher for Windows.

The 32-bit version of the converter, Mswrd632.cnv that is installed with a complete installation of Word 7.0 is used automatically by 32-bit Microsoft programs that share converters including Microsoft Works and Publisher for Windows 95.

This converter is not designed for use from within Word for Windows itself.

Versions of Msword6.cnv that can read only Word 6.0 for Windows files include: 

   Supplied with           File date   File size   Version stamp
   -------------------------------------------------------------
   Word 6.0 for Windows    10-27-93    136704      93102000
   Word 6.0a for Windows   01-26-94    137216      94012410
   Word 6.0c for Windows   08-10-94    138240      94080100
All newer releases can read Word 6.0 and 7.0 for Windows, Word 6.0 for Windows NT, and Word 6.0 for the Macintosh files.
