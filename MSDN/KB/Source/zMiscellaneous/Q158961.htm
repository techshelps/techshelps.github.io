Files Installed by DirectX 3.0

 Last reviewed: February 26, 1998
Article ID: Q158961
The information in this article applies to:
  • DreamWorks Interactive, The Neverhood for Windows, version 1.0
  • DreamWorks Interactive, Scholastic's Goosebumps Series: 

          Escape from HorrorLand for Windows, version 1.0

SUMMARY

The following article contains a list of the DirectX system files installed to your hard drive by DirectX 3.0. DirectX 3.0 marks the introduction of special DirectInput files, and a special Joystick Control Panel applet. It also includes updated Virtual Math Coprocessor Device files.

MORE INFORMATION

All files are marked as version 4.04.00.0068 or 4.04.00.0069.

Direct3D

  • d3dhalf.dll
  • d3dim.dll
  • d3drampf.dll
  • d3drg16f.dll
  • d3drg24f.dll
  • d3drg24x.dll new
  • d3drg32f.dll
  • d3drg32x.dll new
  • d3drg55x.dll new
  • d3drg56x.dll new
  • d3drg8f.dll
  • d3drg8x.dll new
  • d3drgbf.dll
  • d3drgbxf.dll new
  • d3drm.dll
  • d3drm16f.dll
  • d3drm24f.dll
  • d3drm32f.dll
  • d3drm8f.dll
  • d3dxof.dll

DirectDraw

- Ddhelp.exe - Ddraw.dll - Ddraw16.dll

DirectPlay

  • dplayx.dll new
  • dpmodemx.dll new
  • dpwsockx.dll new

DirectSound

  • Dsound.dll
  • Dsound.vxd

DirectInput

  • dinput.dll
  • joy.cpl Joystick applet for Control Panel
  • vjoyd.vxd Joystick vxd

Virtual Math Coprocessor Device

Updated version 4.00.1111; also included in Windows 95 OSR2 (improved for Intel Pentium MMX access)

  • vmcpd.vxd (source file is vmcpd.x86)
