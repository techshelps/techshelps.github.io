Problems Using HP LaserJet 3.1 or DeskJet 4 Driver in Windows

 Last reviewed: September 9, 1996
Article ID: Q114377
The information in this article applies to:
  • Microsoft Windows operating system versions 3.1, 3.11
  • Microsoft Windows for Workgroups versions 3.1, 3.11

SYMPTOMS

When you use the Hewlett-Packard (HP) printer driver for the LaserJet 4 Plus or 4M Plus Printing System for Windows (version 3.1) or for the DeskJet 500 Series (version 4.0), one or more of the following problems occurs:

  • 0 or zero-byte files appear in Print Manager.
  • SPL files remain in the TEMP directory after the print job is finished.
  • A document sent to the printer multiple times does not print.
  • Several print jobs are in the queue, but only the first job prints.

CAUSE

HP ships a version of PRINTMAN.EXE on its disk set that is incompatible with Windows and Windows for Workgroups. Therefore the resolution below requires a non-HP disk set.

RESOLUTION

To correct this problem, expand PRINTMAN.EXE from Microsoft Windows or Windows for Workgroups disks. For example, to manually expand PRINTMAN.EXE from drive A to a Windows directory called "Windows," type the following at an MS-DOS command prompt: 

   expand a:\printman.ex_ c:\windows\printman.exe
PRINTMAN.EX_ is on
