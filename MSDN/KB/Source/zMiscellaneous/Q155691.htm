"Fatal Exception Error 06" with Sharp Mebius

 Last reviewed: July 17, 1997
Article ID: Q155691

The information in this article applies to:

  • The Microsoft Network versions 1.2, 1.3, 2.0

SYMPTOMS

After using MSN, The Microsoft Network, on your Sharp Mebius notebook computer, you receive the following error message when you close Japanese Windows 95: 

   - Fatal Exception Error 06

CAUSE

The Cirrus Logic display driver version 1.20 included with Mebius models PC-A435, A445, A455, and A435J needs to be upgraded to version 1.31.

RESOLUTION

You can download the display driver update from the following location: 

   http://www.sbc.co.jp/mebius/download
