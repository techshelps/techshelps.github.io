WD: Ami Pro 3.x Converter Limitations

 Last reviewed: November 13, 1997
Article ID: Q125299

The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word 97 for Windows

SUMMARY

This article contains the text of AMILIMIT.DOC, which is included with the Application Note "Ami Pro 3.0 Converter" (GC1064).

MORE INFORMATION

For additional information about GC1064, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q120976
   TITLE     : WD: GC1064: Ami Pro 3.0 Converters for Word 6.x and Word 7.0
