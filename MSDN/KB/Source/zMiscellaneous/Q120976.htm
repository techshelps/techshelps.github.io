WD: GC1064: Ami Pro 3.0 Converters for Word 6.x and Word 7.0

 Last reviewed: July 25, 1997
Article ID: Q120976
The information in this article applies to:
  • Microsoft Word for Windows 95, versions 7.0, 7.0a
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c
  • Microsoft Word for Windows NT, version 6.0

SUMMARY

Microsoft has an "Ami Pro 3.0 Converter" (GC1064) Application Note available that contains two Ami Pro text converters (AMI_PRO.CNV and AMI332.CNV). You can use these converters with the Word versions listed at the start of this article. These converters allow you to convert documents from Ami Pro versions 3.0, 3.01, and 3.1 to Word and from Word to Ami Pro 3.0, 3.01, and 3.1.

OVERVIEW

This Application Note includes two Ami Pro version 3.0 text converters that convert Ami Pro 3.0 files to and from Microsoft Word version 6.0 for Windows, Microsoft Word version 6.0 for Windows NT, or Microsoft Word version 7.0 for Windows 95. Ami_pro.cnv is for use with Word for Windows, and Ami332.cnv is for use with Word for Windows NT and Word for Windows 95.

Also included is Amilimit.doc, a Word 6.0 document that contains a table of conversion-limitation information about the Ami Pro converters. The table lists only the features that are not converted, that are not supported in one product or the other, or that require comment.

NOTE: After you install a new converter, you must convert at least one file from the "foreign" file format (that is, the format for which the converter was designed--in this case, you must open at least one Ami Pro file). Opening a foreign file causes Word to rebuild the list from which it selects the converter. The converter will not be available until you do this.

INSTALLATION AND CONVERSION

WORD VERSION 6.0 FOR WINDOWS

To use the converter with Word for Windows, first copy the converter file to your Word program folder (usually, C:\Winword), quit and restart Word, and then open an Ami Pro file.

To copy Ami_pro.cnv to your Word for Windows program folder

  1. If you received a disk with this Application Note, insert the disk in the appropriate floppy disk drive.

    If you downloaded this Application Note from an online service, skip to step 2. This procedure assumes that you have already downloaded and extracted the compressed file.

  2. From the Main group in Windows Program Manager, choose the MS-DOS Prompt icon. When the MS-DOS prompt appears, type the following and press ENTER 

          copy <source>:\ami_pro.cnv <WinWord>

    where <source> is either the letter of your floppy disk drive or the path to the folder containing the file you downloaded and extracted from online services, and <WinWord> is the path to your Word for Windows program folder (usually, C:\Winword). For example, if you are installing from a floppy disk that you inserted in drive A, and C:\Winword is the path to your Word for Windows program folder, type the following: 

          copy a:\ami_pro.cnv c:\winword

    Or, for example, if you downloaded Gc1064.exe from online services to C:\Download and C:\Ww is the path to your Word for Windows program folder, type the following: 

          copy c:\download\ami_pro.cnv c:\ww

To open an Ami Pro 3.0 file in Word for Windows

If you have Word for Windows running, first quit and restart Word before continuing with the steps below.

  1. On Word's File menu, click Open.

  2. In the List Files Of Type list, click All files.

  3. In the File Name box, type or select the filename of the document you want to open.

    If the document you want to open does not appear in the File Name box, select a different drive or folder.

  4. Click OK to convert a copy of the document to Word format.

  5. If you have the Confirm Conversions check box selected, Word will present a dialog box that asks you to confirm that you want to convert the file from Ami Pro 3.0. Click OK.

Word will automatically install the converter, and you should now be able to open your Ami Pro documents in Word and save your Word documents in Ami Pro file format from Word.

To save a Word for Windows document in

Ami Pro 3.0 format from Word for Windows

  1. On the File menu, click Save As.

  2. In the Save File As Type list, click Ami Pro 3.0. (Word will automatically change the filename extension to .SAM.)

  3. Click OK.

To save an Ami Pro 3.0 document in Word format from Word for Windows

  1. On the File menu, click Save As.

  2. In the Save File As Type list, click Word Document. (Word will automatically change the file name extension to .doc.)

  3. Click OK.

WORD VERSION 6.0 FOR WINDOWS NT AND WORD VERSION 7.0 FOR WINDOWS 95

To use the converter with Word for Windows NT or Word version 7.0 for Windows 95, first copy the Ami332.cnv file from the enclosed disk to your Word for Windows NT program folder (usually C:\Winword) or your Word for Windows 95 program folder (usually C:\MSOffice\Winword or C:\Winword), and then open an Ami Pro file in Word.

To copy Ami332.cnv to your Word for Windows NT or Word for Windows 95 program folder, repeat the instructions given for copying the Ami_pro.cnv file to the Word for Windows program folder, except wherever "Ami_pro.cnv" occurs, replace it with "Ami332.cnv."

To open an Ami Pro 3.0 file in Word for Windows NT or Word for Windows 95, repeat the instructions given for Word for Windows.

To save a Word for Windows NT or Word for Windows 95 document in Ami Pro 3.0 format (or to save an Ami Pro 3.0 document in Word format from Word for Windows NT or Word for Windows 95), repeat the instructions given for saving documents in Word for Windows.
