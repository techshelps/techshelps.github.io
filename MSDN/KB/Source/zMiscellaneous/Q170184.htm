Err Msg: DirectSound or DirectDraw Drivers Disk Is Now Required

 Last reviewed: February 18, 1998
Article ID: Q170184
The information in this article applies to:
  • DreamWorks Interactive, The Neverhood for Windows, version 1.0
  • DreamWorks Interactive, Scholastic's Goosebumps Series: Escape from Horrorland for Windows, version 1.0
  • Microsoft Close Combat for Windows, version 1.0a
  • Microsoft Deadly Tide for Windows, version 1.0
  • Microsoft Flight Simulator for Windows 95, version 6.0
  • Microsoft GEX, version 1.0
  • Microsoft Hellbender for Windows, version 1.0
  • Microsoft Monster Truck Madness, version 1.0
  • Microsoft NBA Full Court Press for Windows, version 1.0
  • Microsoft Soccer, version 1.0

SYMPTOMS

When you attempt to reinstall a sound card driver, you receive the following message: 

   Insert Disk
   The disk labeled 'DirectSound Drivers' is now required
   This disk is provided by your computer manufacturer.
   Click OK to continue.
If you are reinstalling a display card driver, the message will instead refer to DirectDraw Drivers not DirectDraw Drivers.

RESOLUTION

Insert your DirectX game CD-ROM. If necessary, use the Browse button to search for one of the following folders on your game CD-ROM: 

   DIRECTX\DRIVERS\AUDIO\BIN
   DIRECTX\DRIVERS\DISPLAY\BIN
For video card drivers, use the Display\Bin folder. For sound card drivers, use the Audio\Bin folder.

If this doesn't work, insert your original manufacturer-provided driver disc. Many newer drivers from hardware manufacturers are also DirectSound or DirectDraw drivers. You can use the Browse button to look for the driver folder on the manufacturer-provided disk.

MORE INFORMATION

Many games designed for Windows 95 use DirectX. DirectX includes a set of updated drivers for popular audio and display cards. These games include the full DirectX package on the CD-ROM, including the updated drivers.

Hardware manufacturers also provide new drivers that support DirectX. The "Insert Disk" message can refer to either the game compact disc or a manufacturer-provided disc.

Replacing Drivers

If you are attempting to replace a driver, and have another disc that is not being accepted, you can remove the list of DirectSound or DirectDraw drivers from Windows 95. The list is installed in the Windows\Inf folder.

To remove the list of DirectSound drivers, delete the following folders: 

   C:\Windows\Inf\Msdsound.inf
   C:\Windows\Inf\Msdsound (the entire folder)
Where C:\ is the letter of the hard drive where you installed Windows.

To remove the list of DirectDraw drivers, delete the following: 

   C:\Windows\Inf\Msddraw.inf
   C:\Windows\Inf\Msddraw (the entire folder)
You can restore these files by running Dxsetup.exe, located on your game CD- ROM.
