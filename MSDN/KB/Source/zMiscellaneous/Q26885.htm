PPT7: Clipart Gallery 2.0a Update for Windows NT Users

 Last reviewed: March 12, 1998
Article ID: Q26885

The information in this article applies to:
  • Microsoft PowerPoint for Windows 95, version 7.0

SUMMARY

Microsoft has released a file that corrects problems that may prevent you from using ClipArt Gallery when you are running PowerPoint for Windows 95 on the Windows NT operating system.

The following file is available for download from the Microsoft Software Library: 

 ~ Cag20a.exe (size: 234438 bytes)
The Cag20a.exe file is a self-extracting file that contains the following files: 

   Artgalry.exe
   Readme.doc
The Readme.doc contains a complete description of the problem as well as instructions for installing the updated version of ClipArt Gallery.

For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
