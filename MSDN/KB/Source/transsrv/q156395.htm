FIX: Icons Look Strange When Using 256 Colors or Less

 Last reviewed: October 29, 1997
Article ID: Q156395
The information in this article applies to:
  • Microsoft Transaction Server, version 1.0

SYMPTOMS

If you are using a video color resolution of 256 colors or less, the Transaction List, Transaction Statistics, and the Trace Messages icons in the right pane may appear chunky, as if the box for the icon is larger than the actual icon, and the icon is being stretched to fit the box.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Transaction Server version 1.0. This problem was corrected in Microsoft Transaction Server version 1.1. For information on obtaining version 1.1, please see the following article the Microsoft Knowledge Base:
