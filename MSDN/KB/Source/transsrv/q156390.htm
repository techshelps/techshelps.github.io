FIX: "The Topic Does Not Exist" Error Using F1 Key for Help

 Last reviewed: October 29, 1997
Article ID: Q156390
The information in this article applies to:
  • Microsoft Transaction Server, version 1.0

SYMPTOMS

If you press the F1 key to open the online Help, you receive the following error message: 

   The topic does not exist. Contact your application vendor for an
   updated Help file. (129)
This error appears if you press the F1 key after doing one of the following:
  • Click Trace Messages in the hierarchy tree, then click in the right pane.
  • Click Transaction List in the hierarchy tree, then click in the right pane.
  • Hold the mouse pointer over one of the following toolbar buttons and wait for the pop-up Help to appear: 

        - Up one level


        - Open a new window


        - Refresh

STATUS

Microsoft has confirmed this to be a problem in Microsoft Transaction Server version 1.0. This problem was corrected in Microsoft Transaction Server version 1.1. For information on obtaining version 1.1, please see the following article the Microsoft Knowledge Base: 

   ARTICLE-ID: Q168031
   TITLE     : How to Obtain Microsoft Transaction Server Version 1.1
