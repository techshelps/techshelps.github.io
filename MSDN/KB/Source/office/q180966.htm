OFF98: Zoom Box for Help Window Does Not Size the Window

 Last reviewed: February 13, 1998
Article ID: Q180966
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SYMPTOMS

After you click the Zoom box in the title bar of Help in any of the Microsoft Office 98 Macintosh Edition programs, the size of the Help window may not change.

WORKAROUND

To workaround this behavior, use one of the following methods:

Method 1: Under OS 8.0 or 8.1

If you are using the OS 8.0 or 8.1, click the Zoom box a second time. The window is then maximized and restored. Otherwise, use the Size box to resize the window.

Method 2: Under System 7.x

If you are using System 7.x, resize the window by using the Size box.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

You can use the Zoom box to quickly change the size of the selected window. Clicking the Zoom box the first time maximizes the window. Clicking it a second time restores the window to its original state. When you click the Zoom box to maximize the window, it fits all of the contents of the window onto the screen if possible.

REFERENCES

For more information about arranging windows, click the Index button in Microsoft Excel Help, type the following text 

   windows, arranging
and then double-click the selected text to go to the "View multiple sheets or workbooks at the same time" topic.
