Office: Gateway Office CD Missing Online Documentation

 Last reviewed: September 16, 1996
Article ID: Q118342
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3-CD

SUMMARY

If you have purchased the Microsoft Office 4.3-CD from Gateway 2000, you may not be able to run the online documentation for Microsoft Office. This Microsoft Office version 4.3-CD is a Gateway-distributed CD, and does not contain any online documentation. Gateway 2000 is responsible for the support of this product.

According to the documentation offer in your Microsoft Office package, you can call the Gateway Fulfillment number, (800) 326-0401 to obtain documentation. You can also use this number to obtain product information.

For technical support with an issues you encounter using this CD, contact Gateway 2000 technical support at (800) 846-2301. If you would like to receive answers to commonly asked questions via fax, you can call (800) 846- 4526.

MORE INFORMATION

As of 6/27/94, the Gateway Microsoft Office CD contains only disk images of Microsoft Office version 4.3 disks; this version of Microsoft Office does not contain the online Help files. When you look at Gateway's jewel case booklet, it contains pictures and instructions for using the online documentation (however, the documentation is not included on the CD). The Office Setup concludes without installing the viewer icon on the MOM toolbar or in Program Manager. There is, however, a coupon offer in your Microsoft Office package that allows you to purchase the hardcopy documentation.
