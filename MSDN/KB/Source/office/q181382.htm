OFF98: "Microsoft Office Is Incompatible with SAM" Error

 Last reviewed: February 27, 1998
Article ID: Q181382
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SYMPTOMS

When you install Microsoft Office 98 Macintosh Edition by using the Office Custom Installer, or when you start a Microsoft Office 98 program for the first time, you may receive one of the following error messages: 

   Microsoft Office is incompatible with the version of SAM that you are
   currently running. Please disable the SAM Intercept extension, restart
   your computer, and try again.

   -or-

   Microsoft Office is incompatible with the version of SAM that you have
   installed on your Macintosh. Although the SAM Intercept extension is
   currently disabled, you need to ensure this extension remains disabled
   until you can upgrade to a compatible release.
When you receive the first error message, your only option is to click Quit, and the installation of Microsoft Office is not completed. When you receive the second error message, you can click Continue to complete the installation of Office.

CAUSE

This problem occurs if SAM Intercept version 4.5.2 or earlier is installed on your computer.

WORKAROUND

To correct the problem, update to SAM Intercept version 4.5.3. To do this, obtain the Samint453patch.sea.hqx file from Symantec.

The file is available at the following Symantec Web sites: 

   http://service.symantec.com/sam/

   ftp://ftp.symantec.com/public/english_us_canada/products/
   symantec_antivirus_macintosh/ver4.5/updates/
NOTE: The previous address should be one line of text.
