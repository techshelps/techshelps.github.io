Buttons Missing on MOM after Office 7.0 Installation

 Last reviewed: March 27, 1997
Article ID: Q135808
7.00 WINDOWS kbtool kbprb

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Windows 95

SYMPTOMS

After you successfully complete an installation of Microsoft Office version 7.0, the number of buttons on your Old Office toolbar, located on the Office Shortcut Bar, may be fewer than you had on the Microsoft Office Manager (MOM) prior to installing version 7.0.

In addition, the Office application buttons on MOM may run the version 7.0 Office applications rather than the earlier versions of your applications.

RESOLUTION

If you want to have the same number of buttons on your Office Shortcut Bar Old Office toolbar as you had on MOM, you need to manually customize the Old Office toolbar. In order to customize the Old Office toolbar, follow these steps:

  1. While the Old Office toolbar is displayed, use the right mouse button to click the Office Shortcut Bar and click Customize on the shortcut menu.

  2. Click the Buttons tab.

  3. For each missing button, click the Add File button.

  4. In the Add File dialog box, locate the application you want added to the Old Office toolbar, and then click the Add button.

If you want to have your Old Office toolbar buttons for the Office applications start the Office version 4.0, 4.2, or 4.3 programs, do the following:

  1. While the Old Office toolbar is displayed, use the right mouse button to click the Office Shortcut Bar and click Customize on the shortcut menu.

  2. Click the Buttons tab.

  3. Select the entry for the application you want to change, and then click the Delete button.

  4. Click the Add File button.

  5. In the Add File dialog box, locate the application you want to add to the Old Office toolbar, and then click the Add button.

  6. When you are finished in the Customize dialog box, click the OK button. If you deleted any buttons in step 3, you will be prompted to see if you really want to delete the item(s). If you click the Yes button, you may also be prompted to see if you really want to move the items to the recycle bin.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.
