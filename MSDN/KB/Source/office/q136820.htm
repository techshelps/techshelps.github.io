OFF95: Office Professional Contains Office Standard Disk(s)

 Last reviewed: March 10, 1997
Article ID: Q136820
7.00 WINDOWS kbother

The information in this article applies to:

  • Microsoft Office Professional for Windows 95, version 7.0

SYMPTOMS

When you open your Office Professional box to install Microsoft Office, each of the floppy disks, or the CD, is labeled "Microsoft Office for Windows 95 Standard" instead of "Professional" as expected.

CAUSE

When you purchase Microsoft Office for Windows 95 Professional Edition, you are entitled to Microsoft Access for Windows 95 when it is available, even though your Office disks are labeled "Standard."

If you purchased Microsoft Office Professional on floppy disks, you have the option to receive Office for Windows 95 Professional Edition on CD or the Microsoft Access standalone program on floppy disks (when these are available). If you purchased Microsoft Office Professional on CD, you will receive Office for Windows 95 Professional Edition on CD (when it is available).

MORE INFORMATION

A red emblem on the Office for Windows 95 Professional box displays the following information: 

   Voucher enclosed entitles you to Microsoft Access for Windows 95
   software when available.
The voucher contains the following information: 

   The Professional Edition of Microsoft Office for Windows 95 includes
   Microsoft Access for Windows 95, which is not yet available. You must
   return this voucher to Microsoft to receive your Microsoft Access for
   Windows 95 software. You will receive this software at no additional
   cost as soon as it becomes available.
