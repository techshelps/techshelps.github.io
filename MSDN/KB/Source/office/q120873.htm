Office: Setup May Fail Under OS/2 2.0

 Last reviewed: September 16, 1996
Article ID: Q120873
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.2, 4.3

SUMMARY

The Setup program for Microsoft Office version 4.2 may fail if you are trying to run Setup under a Win-OS/2 session for IBM OS/2 version 2.0. If you run Setup under a WIN-OS/2 session, the following error message will be displayed: 

   The WIN-OS/2* Session does not support this application.
NOTE: This error message will not be received when you run Setup under a WIN-OS/2 session for IBM OS/2 version 2.1.

MORE INFORMATION

The Microsoft Office version 4.2 Setup program requires Microsoft Windows version 3.1 or higher to run. The Win-OS/2 session in IBM OS/2 version 2.0 only emulates Microsoft Windows version 3.0, and Setup fails. IBM OS/2 version 2.1 emulates Microsoft Windows version 3.1 and should run correctly.
