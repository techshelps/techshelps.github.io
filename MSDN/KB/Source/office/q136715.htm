OFF95: Files Installed with Run From Network Server Installation

 Last reviewed: March 10, 1997
Article ID: Q136715
7.00 WINDOWS kbsetup kbnetwork

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

The following is a list of the folders and files installed when you perform a Run From Network Server installation of Microsoft Office (with the shared components installed on the server).

MORE INFORMATION

You can use this list to determine which files are installed by Microsoft Office and where they are installed, so that you can remove them if necessary.
