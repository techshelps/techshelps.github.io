OFF97: Hyperlinks to Same Location Don't Appear Followed

 Last reviewed: December 6, 1996
Article ID: Q160542
The information in this article applies to:
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft PowerPoint 97 for Windows

SYMPTOMS

In a document that contains more than one hyperlink that points to the same location, when you click one of these hyperlinks, the program does not format all the hyperlinks in a style that indicates you followed the hyperlinks.

CAUSE

This problem occurs because the program formats the hyperlink as followed only when you click the hyperlink. This behavior is not consistent with the behavior in Microsoft Internet Explorer.

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post more information here in the Microsoft Knowledge Base as it becomes available.
