OFF95: Disk Space Error After Changing to Drive

 Last reviewed: March 27, 1997
Article ID: Q135649
7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

When you run Microsoft Office Setup, you may receive an "Out Of Disk Space" error message, indicating that you do not have enough disk space to install Office with the components selected. If you use the Change Options button and select the Custom installation option, if you then use the Change Folder button to install Office to a folder on a different drive on your computer, you may continue to receive the "Out Of Disk Space" error, even if there is plenty of disk space available on the selected drive.

CAUSE

This problem occurs if you use the Change Folder button during Setup after you receive the "Out Of Disk Space" error message because the Change Folder button only changes the destination for the selected Office component, such as Office Binder And Shortcut Bar. The Change Folder button does not change the folder for your entire Office installation in this case, and there still may not be enough disk space available on the destination drive.

RESOLUTION

To resolve this problem, when there is not enough disk space available to install Office on the destination drive, and there is enough disk space available on another drive, click the Change Folder button for each of the following Office components, and select a folder on the drive that does have enough disk space available: 

   Office Binder and Shortcut Bar
   Microsoft Excel
   Microsoft Word
   Microsoft PowerPoint
   Microsoft Schedule+
   Office Tools
   Converters, Filters, and Data Access
