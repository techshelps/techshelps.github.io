OFF: Toolbar Buttons Unavailable in Office Binder

 Last reviewed: December 3, 1996
Article ID: Q134743
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

In Microsoft Office Binder, when you click a button on the toolbar while a section in the Binder is active, the button may be unavailable (causes the computer to beep) or may produce unexpected behavior.

CAUSE

This unexpected behavior occurs because the toolbars that appear in Office Binder belong to the application that you used to create the section. For example, when you insert a Microsoft Excel worksheet in your Binder, the Microsoft Excel toolbars appear in Office Binder while the worksheet section is active. However, some of these buttons provide functionality that is not available in the Office Binder. Additionally, some of these buttons perform a different action when you use them in Office Binder than when you use them in the application.
