Office: How to Start the Setup Program Manually

 Last reviewed: March 27, 1997
Article ID: Q121466
4.20 4.30 7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows, versions 4.2, 4.3
  • Microsoft Office for Windows 95, version 7.0

In some situations, Microsoft Office Setup may need to be started manually in order for the set up to be completed successfully. This may be required if you want to install Microsoft Office on a removable drive.

To manually start the Microsoft Office Setup routine, do the following:

  1. For Microsoft Office for Windows 4.x, exit Windows. At the MS-DOS prompt, type "MD ~SETUP" (without the quotation marks) at the root level of the C: drive to create a directory.

    For Microsoft Office for Windows 95, go to the MS-DOS prompt and type "MD ~MSSTFQF.T" (without the quotation marks) at the root level of the C: drive to create a directory.

  2. Switch to the drive that contains Disk 1 of Microsoft Office by typing, "A:" (without the quotation marks) where A is the appropriate floppy disk drive.

  3. If you have Microsoft Office version 4.2, decompress the following files by typing the command lines at the prompt: 

          decomp _mssetup.ex_ c:\~setup\_mssetup.exe
      decomp acmsetup.ex_ c:\~setup\acmsetup.exe
      decomp acmsetup.hl_ c:\~setup\acmsetup.hlp
      decomp mscpydis.dl_ c:\~setup\mscpydis.dll
      decomp mssetup.dl_ c:\~setup\mssetup.dll
      decomp xlsetup.tt_ c:\~setup\xlsetup.ttf
      decomp wwsetup.tt_ c:\~setup\wwsetup.ttf
      decomp excel_bb.dl_ c:\~setup\excel_bb.dll
      decomp off42_bb.dl_ c:\~setup\off42_bb.dll
      decomp word_bb.dl_ c:\~setup\word_bb.dll
      decomp setup.st_ c:\~setup\setup.stf
      decomp setup.in_ c:\~setup\setup.inf
      decomp odbcinst.dl_ c:\~setup\odbcinst.dll

    If you have Microsoft Office version 4.3, decompress the following files by typing the command lines at the prompt: 

          decomp _mssetup.ex_ c:\~setup\_mssetup.exe
      decomp acmsetup.ex_ c:\~setup\acmsetup.exe
      decomp acmsetup.hl_ c:\~setup\acmsetup.hlp
      decomp mscpydis.dl_ c:\~setup\mscpydis.dll
      decomp mssetup.dl_ c:\~setup\mssetup.dll
      decomp xlsetup.tt_ c:\~setup\xlsetup.ttf
      decomp wwsetup.tt_ c:\~setup\wwsetup.ttf
      decomp acsetup.tt_ c:\~setup\acsetup.ttf
      decomp excel_bb.dl_ c:\~setup\excel_bb.dll
      decomp off43_bb.dl_ c:\~setup\off43_bb.dll
      decomp pp4_bb.dl_ c:\~setup\pp4_bb.dll
      decomp word_bb.dl_ c:\~setup\word_bb.dll
      decomp msacahbb.dl_ c:\~setup\msacahbb.dll
      decomp setup.st_ c:\~setup\setup.stf
      decomp setup.in_ c:\~setup\setup.inf
      decomp odbcinst.dl_ c:\~setup\odbcinst.dll

If you have Microsoft Office for Windows 95, extract the following files by typing the command lines at the prompt: 

      extract msoff1.cab /L c:\~msstfqf.t acmsetup.exe
      extract msoff1.cab /L c:\~msstfqf.t acmsetup.hlp
      extract msoff1.cab /L c:\~msstfqf.t mssetup.dll
      extract msoff1.cab /L c:\~msstfqf.t setup.stf
      extract msoff1.cab /L c:\~msstfqf.t off95std.inf
      extract msoff1.cab /L c:\~msstfqf.t odbccp32.dll
      extract msoff1.cab /L c:\~msstfqf.t odbcint.dll
      extract msoff1.cab /L c:\~msstfqf.t ffast_bb.dll
      extract msoff1.cab /L c:\~msstfqf.t odbcstf.dll
      extract msoff1.cab /L c:\~msstfqf.t odbckey.inf
      extract msoff1.cab /L c:\~msstfqf.t msvcrt20.dll
      extract msoff1.cab /L c:\~msstfqf.t off95_bb.dll
      extract msoff1.cab /L c:\~msstfqf.t offsetup.ttf
      extract msoff1.cab /L c:\~msstfqf.t off95inv.dll
      copy setup.ini c:\~msstfqf.t
      copy setup.tdf c:\~msstfqf.t

  • After decompressing/extracting the appropriate files, restart Microsoft Windows.

  • For Microsoft Office for Windows 4.x, in Program Manager, click Run on the File menu. On the command line, type the following, and click OK: 

          C:\~setup\acmsetup /T setup.stf

    For Microsoft Office for Windows 95, click Run on the Start menu and type the following command: 

          C:\~msstfqf.t\acmsetup /T setup.stf /S a:\

    Microsoft Office Setup should now start and proceed normally through the setup routine.

    TIP: You can create a batch file containing the above commands, and then run this batch file on individual workstations.
