OFF97: "This Setup Program Is Not Intended" Error During Setup

 Last reviewed: March 13, 1998
Article ID: Q166343
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you run the Setup program for Microsoft Office 97, you may receive the following error message: 

   This Setup program is not intended to be used with your version of
   Windows

CAUSE

This problem may occur if there is an invalid entry in the Compatibility section of your Win.ini file.

RESOLUTION

To resolve this problem, remove the "bootstrp" entry from the Compatibility section of your Win.ini file. To do this, use the following steps:

  1. Click the Start menu, and then click Run.

  2. In the Run box, type "win.ini" (without the quotation marks), and then click OK.

  3. Click Find on the Search menu. In the Find What box, type "[Compatibility]" (without the quotation marks), and then click Find Next. Click Cancel to dismiss the Find dialog box.

  4. Look for a line in the [Compatibility] section that contains the following text 

          BOOTSTRP=

    and type a semicolon (;) in front of this line.

  5. Click Save on the File menu. Click Exit on the File menu.

  6. Restart Microsoft Windows and run the Microsoft Office 97 Setup program.
