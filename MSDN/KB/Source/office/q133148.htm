OFF95: 0K Space Required/Available for Getting Results Book

 Last reviewed: March 27, 1997
Article ID: Q133148
7.00-CD WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0-CD

SYMPTOMS

When you run the Microsoft Office Setup program from a CD, and you choose the Custom installation option (the first time you run Setup) or the Add/Remove install option (when you run Setup in maintenance mode), if you click the Getting Results Book item in the Options list, the value displayed for the Space Required and Space Available fields is 0K.

CAUSE

This behavior occurs because the Getting Results Book file is not actually installed to your computer. The Getting Results Book option in Setup is the online version of the manual that ships with Microsoft Office. If you select this option during Setup, a shortcut to the online file is installed to your local hard disk in the following locations 

   MSOffice
   MSOffice\Office\Shortcut Bar\Office
where MSOffice is the folder to which you installed Microsoft Office.

Although the disk space required on your computer for this item is correctly displayed in Setup as 64K (for the shortcuts and the Results.gid file), Setup displays the drive that contains the CD as the destination for the Getting Results Book. Therefore, the disk space required and available (on the CD) is reported as 0K.

Note that this behavior does not affect the functionality of Setup or installing the Getting Started Book. This is only a visual symptom that may be confusing if you are viewing the disk space information for this individual item.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
