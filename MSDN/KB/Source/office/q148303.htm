OFF95: NIW Error: "Isn't an Index in This Table"

 Last reviewed: October 6, 1997
Article ID: Q148303
7.00a WINDOWS kbsetup kbnetwork kbtool

The information in this article applies to:

  • Microsoft Network Installation Wizard, version 1.0a
  • Microsoft Office Professional for Windows 95, version 7.0a
  • Microsoft Office Standard for Windows 95, version 7.0a, 7.0b

SYMPTOMS

When you run the Microsoft Network Installation Wizard (NIW) version 1.0a, if you select a Microsoft Office for Windows 95, versions 7.0a or 7.0b,(Standard or Professional edition) setup table file (Setup.stf) in the "Please choose the proper SETUP.STF file" dialog box, and you then click OK, the file appears to load completely; However, you then receive the following error message: 

   " isn't an index in this table

CAUSE

Microsoft Office 95, version 7.0a and 7.0b Setup.stf files are incompatible with the Microsoft Network Installation Wizard, version 1.0a.

WORKAROUND

To work around this problem, obtain the Microsoft Network Installation Wizard, version 1.1 or later. Netinst.exe, a self-extracting file, contains version 1.1 of the wizard. You can download the file from the following services:

- The Microsoft World Wide Web Site:

http://www.microsoft.com/TechNet/desk/office/reskit/orkf/apadf.htm#contents

- Internet (anonymous FTP)

ftp://ftp.microsoft.com/services/technet/office95/netinst.exe

STATUS

Microsoft has confirmed this to be a problem in the Microsoft Network Installation Wizard 1.0a. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
