OFF97: Camcorder Stop Button Setting not Retained

 Last reviewed: December 6, 1996
Article ID: Q159614
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

If you click to select the "Hide stop button while recording" check box in the Camcorder Preferences dialog box, the stop button may reappear when you quit and restart Microsoft Camcorder.

CAUSE

The setting for "Hide stop button while recording" is not retained by Camcorder; each time you quit Camcorder this setting is cleared.

RESOLUTION

You will not be able to have the "Hide stop button while recording" setting permanently retained by Camcorder. Each time you start Camcorder you will have to reset this setting in the Preferences dialog box.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The "Hide stop button while recording" setting is used to hide the Stop toolbar while you are recording a movie. If you have turned on the "Hide stop button while recording" setting, and the Stop toolbar is not displayed, you will have to use your specified key combination to stop a movie recording. By default, the ESC key stops a recording.

For additional information, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q159561
   TITLE     : OFF97: Description of the Microsoft Camcorder

REFERENCES

For more information about stopping a movie recording, click the Contents tab in Microsoft Camcorder Help, double-click Microsoft Camcorder and then double-click the "Show or hide the Stop button while recording a movie" topic.
