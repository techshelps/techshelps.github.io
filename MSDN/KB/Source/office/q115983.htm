Microsoft Office Assistant Button Missing From MOM Toolbar

 Last reviewed: September 16, 1996
Article ID: Q115983
The information in this article applies to:
  • Microsoft Office Assistant for Windows, version 1.0

SYMPTOMS

After you install the Microsoft Office Assistant, the Microsoft Office Assistant button does not appear on the Microsoft Office Manager (MOM) toolbar.

CAUSE

This behavior occurs if, for some reason, the MSOFFICE.INI file is not updated during Microsoft Office Assistant Setup.

WORKAROUNDS

To add the Microsoft Office Assistant icon to the Microsoft Office toolbar, do either of the following:

  • Manually add the Microsoft Office Assistant toolbar button to the MOM toolbar with the following steps:

    1. From the MOM toolbar, choose the Microsoft Office button, and choose 

          Customize.

    2. Select the Toolbar tab, and choose the Add button.

    3. In the Description box, type "Microsoft Office Assistant" (without 

          the quotation marks). In the Command Line box, type "WINHELP
      C:\MSOFFICE\OFFASST\OFFASST.HLP" (without the quotation marks) where
      C:\MSOFFICE\OFFASST is the location of the Microsoft Office Assistant
      files. From the Button Image list, select OABUTTON.BMP and choose
      OK.

    4. In the Customize dialog box, choose OK.

    -or-

  • Reinstall the Microsoft Office Assistant with following steps:

    1. Close the MOM toolbar.

    2. Insert the Microsoft Office Assistant Setup Disk 1 in the appropriate 

          floppy disk drive.

    3. In Program Manager, or File Manager, choose Run from the File menu. 

          In the Command Line box, type "A:SETUP" (without the quotation marks)
      where A is the letter of the floppy drive that contains the Setup
      disk.

    4. In the Microsoft Office Assistant Setup dialog box, choose the 

          Reinstall button.

Note that the last workaround will also create a new program icon in Program Manager if this icon is missing as well.
