Office 4.2x Update for Power Mac Extension Not Installed

 Last reviewed: November 17, 1997
Article ID: Q145867
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2.1b (compact disc)

SYMPTOMS

When you run the Setup program for Microsoft Office for the Macintosh version 4.2.1b, the Office 4.2x Update for Power Mac version 1.0.1 extension is not installed on your hard disk. This behavior differs from the Setup program for Microsoft Office for the Macintosh version 4.2.1a on compact disc; the Office 4.2.1a Setup program installs the Power Macintosh extension.

NOTE: Microsoft Office for the Macintosh, version 4.2.1a installs version 1.0 of the Office 4.2x Update for Power Mac extension. This version of the extension (that is, version 1.0) should be replaced with the later version of the extension (that is, version 1.0.1).

CAUSE

This behavior is by design because the extension is not needed on all computers and can be installed manually if necessary.

MORE INFORMATION

How to Install the Extension

To manually install the Office 4.2x Update for Power Mac version 1.0.1 extension, do the following:

  1. If you have a version 1.0 of the Office 4.2x Update for Power Mac extension in your Extensions folder, move this file to the Trash.

  2. Drag the Office 4.2x Update for Power Mac extension version 1.0.1 onto your Desktop. The extension is located in the Office Update for PowerMac folder that is in the Install Disk 1 folder of the Office 4.2.1b compact disc.

  3. Hold down the SHIFT key to disable extensions while you restart your computer.

  4. After your computer is restarted, drag the Office 4.2x Update for Power Mac extension into your System Folder.

    You will receive a warning message about placing the extension into the Extensions folder. Click OK to move the extension into the Extensions folder.

  5. Restart your computer (without holding down the SHIFT key).

The extension is now properly installed.

Why Should You Install the Extension

The Office 4.2x Update For Power Mac extension prevents your computer from hanging when you use Office for the Power Macintosh programs after you run Word or if you are using QuickDraw GX 1.1.1 or 1.1.2. This extension has a file size of 2884 bytes. This extension is also available with Application Note MC1164.
