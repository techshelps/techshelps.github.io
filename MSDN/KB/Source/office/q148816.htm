Office Programs May Quit When Using the Office Manager

 Last reviewed: February 2, 1998
Article ID: Q148816
The information in this article applies to:
  • Microsoft Office for the Power Macintosh, versions 4.2.1, 4.2.1a, 4.2.1b

SYMPTOMS

If you try to use the Office Manager menu while a Microsoft Office program is running, your Microsoft Office program may unexpectedly quit or your computer may stop responding (hang) without warning.

CAUSE

You may experience this behavior if both of the following are true:

  • You are running System 7.5.3 with the Apple Menu Options control panel installed. 

        -and-
  • You have the Office4.2x Update for Power Mac extension version 1.0 in your Extensions folder.

RESOLUTION

To resolve this problem, remove the Office4.2x Update for the Power Macintosh version 1.0 extension from your Extensions folder. Or, if you need to have this extension on your computer, obtain Application Note MC1164, which contains version 1.0.1 of the Office4.2x Update for Power Macintosh extension. This version of the extension (1.0.1) does not exhibit the same behavior as the version 1.0 extension.

MORE INFORMATION

The Office 4.2x Update for Power Macintosh extension is for use with Office 4.2 and 4.2.1 for the Power Macintosh, with Word 6.0 and 6.0.1 for the Power Macintosh, and with Microsoft Excel 5.0 and 5.0a for the Power Macintosh. Installing and enabling this extension will correct the following two problems:

  • Starting an application after Word causes a System Error 11 message and your computer stops responding (hangs). 

        -and-
  • Printing with QuickDraw GX 1.1.1 or 1.1.2 causes a System Error 11 message and your computer hangs.
