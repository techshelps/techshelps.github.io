OFF98: Not All Languages Listed in Language Dialog are Available

 Last reviewed: March 3, 1998
Article ID: Q181965
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Word 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition

SYMPTOMS

When you display the Language dialog box in any of the programs listed at the beginning of this article, languages that are not available in the Value Pack are listed.

CAUSE

This behavior occurs because the Value Pack Installer installs only a subset of all the languages that are listed in the Language dialog box. The other languages are listed in case you have installed a third-party product that includes proofing tools for the other languages.

STATUS

This behavior is by design of Microsoft Office 98 Macintosh Edition.

MORE INFORMATION

The following languages are currently available for installation by running the Value Pack Installer and are the only languages that are currently available for use with Microsoft Office: 

   Danish
   Dutch
   English (English US, English UK, and English AUS)
   French (French and French Canadian)
   German (German and Swiss German)
   Italian
   Norwegian (Norwegian Bokmal only)
   Portuguese
   Spanish
   Swedish
The following languages are listed in the Language dialog box but are not available in the Value Pack. If you have not installed any third-party proofing tools that include the following languages and you click one of these languages in the Language dialog box, you receive an error message: 

   Brazilian Portuguese
   Catalan
   Czech
   Finnish
   Greek
   Hungarian
   Japanese
   Korean
   Norwegian Nynorsk
   Polish
   Russian
   Turkish
