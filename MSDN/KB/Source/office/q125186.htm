"Please Insert <hard drive> Error Message

 Last reviewed: November 21, 1997
Article ID: Q125186
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2

SYMPTOMS

When you start an application by choosing the menu item for the application from the Microsoft Office Manager (MOM) for the Macintosh, you may receive the following error message 

   Please insert <hard drive>
where <hard drive> is the name of your hard drive.

Note, if you start the application by double-clicking the application file in the Finder, you do not receive an error message.

WORKAROUNDS

To work around this problem, use one of the following methods:

Method 1: To reset the applications, do the following: 

          a. From the Microsoft Office Manager, choose Customize.

          b. From the list of applications, select the application that you
             cannot run from MOM, and choose Edit.

          c. In the dialog box that appears, choose the Change button.

          d. Select the application file in the appropriate folder, and
             choose Add.

          e. Choose OK, and close the Microsoft Office Manager dialog box.

          Repeat these steps for each application that is having this
          problem.
Method 2: To create a new preferences file that will reset MOM: 

          a. Move the Office Manager Preferences file from the System
             Folder:Preferences folder to the desktop.

          b. Restart your computer.

          A new preferences file will be created and you should be able to
          start the application by choosing the appropriate MOM icon. If
          the new preferences file works, delete the old file from the
          desktop.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
