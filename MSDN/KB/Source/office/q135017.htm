OFF95: Keystrokes to Navigate Between Sections in Binder

 Last reviewed: March 27, 1997
Article ID: Q135017
7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

You can use keystrokes to switch between the sections in a Microsoft Office Binder.

MORE INFORMATION

To switch between different sections in the Microsoft Office Binder, use the following keystrokes. Press the following keys simultaneously: 

   To do this          Use this keystroke
   ----------------------------------------------

   To switch upwards   CTRL+ALT+SHIFT+PAGE UP
   To switch downwards CTRL+ALT+SHIFT+PAGE DOWN
