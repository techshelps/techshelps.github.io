Office 4.3c-CD Installs Earlier Version of PICTIMP.FLT

 Last reviewed: September 16, 1996
Article ID: Q125038
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD

SUMMARY

The version of the PICTIMP.FLT file that is shipped with Microsoft Office version 4.3c-CD is not the latest version of this file. The version included on the 4.3c-CD is version 2.2.0.1, the version that was shipped with Microsoft Word version 6.0c, instead of version 2.3.0.4 that was shipped with PowerPoint version 4.0c.

The version of the PICTIMP.FLT file that is located in the SETUP.ADM directory on the Disk1 compact disk is the latest version of this file. However, this version of the file is only installed if you run the Setup program from an administrative installation of Microsoft Office.

Note that Microsoft Office version 4.2c and 4.3c contain the latest version of this graphics filter file. You can use the following table to determine the version of this filter that is installed on your computer: 

                   Version    Installed
                              File Size
   ------------------------------------
   PICTIMP.FLT     2.2.0.1    57600
   PICTIMP.FLT     2.3.0.4    58256

WORKAROUNDS

To install the latest version of this file without running Setup from an administrative installation of Microsoft Office, you can extract the file from the MSOFF24.CAB file located in the DISK24 subdirectory of the SETUP.ADM directory on the Disk 1 CD. To extract this file, use the following steps:

  1. Copy EXTRACT.EXE (located in the Disk1 subdirectory of the SETUP.ADM directory) to the root directory of your local hard drive, or to a directory that is located in the PATH statement in your AUTOEXEC.BAT file.

  2. At the MS-DOS prompt, type the following as one line 

          extract /y <drive>:\setup.adm\disk24\msoff24.cab /L
       <directory>\msapps\grphflt pictimp.flt

    where <drive> is the drive that contains the Microsoft Office Disk 1 CD and <directory> is the path to you your Windows directory. For example if Disk 1 is in drive D and c:\windows is the path to you your Windows directory, type the following (all on one line): 

          extract /y d:\setup.adm\disk24\msoff24.cab /L
       c:\windows\msapps\grphflt pictimp.flt

    Note that if you use the /y switch, the existing compressed file will be overwritten, and you will not be prompted whether you want this change to occur.

MORE INFORMATION

PICTIMP.FLT is the QuickDraw Graphic Import Filter. This file allows you to insert Macintosh PICT and EPS graphic files into a document in Microsoft Word or Microsoft PowerPoint, or it allows you to open a Macintosh file that contains graphics in Word or PowerPoint.
