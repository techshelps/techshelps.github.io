Office Assistant Actor Files Not Removed During Remove All

 Last reviewed: January 30, 1998
Article ID: Q162426
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Access 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Outlook 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Word 97 for Windows

SYMPTOMS

Office Assistant Actor (.act) files may remain on your computer after you perform a Remove All through the Setup program.

WORKAROUND

To work around this problem, remove or delete any remaining .act files from the following location after performing a Remove All: 

   <drive>:\Program Files\Microsoft Office\Office\Actors

STATUS

Microsoft has confirmed this to be a problem in the products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Office Assistant Actor files listed in the following table may remain in the Actors folder after you perform a Remove All. 

   Office Assistant
   Actor File              File Size
   ---------------------------------

   Clippit.act               458KB
   Dot.act                   432KB
   Genius.act              1,855KB
   Hoverbot.act              445KB
   Logo.act                  349KB
   Mnature.act             1,873KB
   Powerpup.act              469KB
   Scribble.act              369KB
   Will.act                  391KB
When you change Office Assistant characters, the .act file for that particular character replaces the existing .acp (preview) file for that character in the Actors folder.
