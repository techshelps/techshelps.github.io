OFF95: How to Determine CRC Checksums for Office 7.0

 Last reviewed: April 2, 1997
Article ID: Q141182
The information in this article applies to:
  • Microsoft Office for Windows 95, Professional edition
  • Microsoft Office for Windows 95, Standard edition

SUMMARY

A utility that allows you to verify the integrity of each Microsoft Office Standard for Windows 95 disk is available. This utility tests the internal cyclic redundancy check (CRC) checksum for Distribution Media Format (DMF) disks included in the U.S. version of Microsoft Office.

To obtain this utility see the section, To Download This File, at the end of this article

To use the utility

This procedure assumes that you have already downloaded and extracted the compressed file.

  1. At the MS-DOS prompt, type the following and press ENTER 

          copy <source>:\crcdsk.exe <location>

    where <source> is the folder (directory) containing the file you downloaded from online services, and <location> is the path to your C:\Dos or your C:\Windows\Command folder. For example, if you downloaded the Crc.exe file from online services to C:\Dnload, and C:\Dos is the path to your Microsoft DOS folder, type: 

          copy c:\dnload\crcdsk.exe c:\dos

  2. Type any of the following commands (without quotation marks) depending on which edition (Standard or Professional) of Office 95 you are using and which disk you are verifying.

    If you are using Microsoft Office for Windows 95, Standard edition ------------------------------------------------------------------ 

          Type this command               To verify this disk
      ---------------------------------------------------

      "crcdsk a:"                     for disks 2 through 23
      "crcdsk a: -x0,19,2085,2090"    for disk 1
      "crcdsk a: -x0,19,20"           for disk 24

   If you are using Microsoft Office for Windows 95, Professional edition
   ----------------------------------------------------------------------

      Type this command               To verify this disk
      ---------------------------------------------------

      "crcdsk a:"                     for disks 2 through 29
      "crcdsk a: -x0,19,2185,2191"    for disk 1
      "crcdsk a: -x0,19,20"           for disk 30 (PPT Viewer)

  3. Put the Microsoft Office disk in question in the A drive (assuming it is a 3.5-inch drive), and press ENTER.

  4. Compare the resulting checksum number with the checksum in the CRC checksums table in this article.

The CRC checksum utility allows you to detect a problem with any of the Microsoft Office 7.0 Distribution Media Format (DMF) disks. If the CRC checksum reported by the utility does not match the checksum listed in the table for the disk in question, the disk format is no longer valid.

If the disk format of your Microsoft Office for Windows 95 DMF disk is no longer valid, call the Microsoft Order Desk at (800) 360-7561 and request a replacement set of disks. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

   http://www.microsoft.com/worldwide/default.htm
