Office: Starting Screen Saver from MOM Causes Sound Error

 Last reviewed: April 2, 1997
Article ID: Q118299
4.20 WINDOWS kb3rdparty kbsetup

The information in this article applies to:

  • Microsoft Office for Windows, version 4.2

SYMPTOMS

When you start a screen saver from Microsoft Office Manager (MOM), sounds that are assigned to the App Start event in Windows are cut short when they are activated.

CAUSE

This problem occurs with the standalone version of MOM 4.2 that was shipped with the Microsoft Office 4.0 fulfillment applications (Microsoft Excel 5.0 and Microsoft PowerPoint 4.0).

STATUS

Microsoft has confirmed this to be a problem in MOM version 4.2. This problem has been corrected in MOM version 4.2b.

To obtain MOM 4.2b, call the Microsoft Order Desk at (800) 360-7561 and request Microsoft Office Manager version 4.2b. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

   http://www.microsoft.com/worldwide/default.htm
