OFFICE: Microsoft Word Clipart Not Added to ClipArt Gallery

 Last reviewed: September 16, 1996
Article ID: Q123852
The information in this article applies to:
  • Microsoft Office for Windows NT, version 4.2

SUMMARY

When you insert a ClipArt Gallery object in a document, after installing Microsoft Office, the Microsoft Word clipart is not added to the ClipArt Gallery.

CAUSE

Microsoft Office version 4.2 for Windows NT does not ship with an integrated Setup program. You install each of the Microsoft Office applications by running the individual Setup program for that application. Because there is no integrated Setup program, Microsoft Office for Windows NT does not ship with an index (.IDX) file for the Microsoft Word clipart, and the clipart is not automatically added to the ClipArt Gallery.

WORKAROUNDS

To add the Microsoft Word clipart to the ClipArt Gallery, do the following:

  1. In a new document in one of the Microsoft Office applications, such as Microsoft Word, choose Object from the Insert menu.

  2. Select the Create New tab, and from the Object Type list, select Microsoft ClipArt Gallery. Choose OK.

  3. In the ClipArt Gallery, choose the Options button. In the Options dialog box, choose Add.

  4. From the Directories list, select the CLIPART subdirectory of the Microsoft Word directory and choose OK.

  5. From the Picture Name list, select the clipart file that you want to add, or choose the Select All button to add all of the clipart files in the CLIPART directory. Choose OK.

You can then select an existing category for the clipart, or enter a new category.

MORE INFORMATION

When you insert a Microsoft ClipArt Gallery object in a document after installing Microsoft Office for Windows NT, you receive a dialog box that contains the following message: 

   Would you like to add clipart from PowerPoint now? This may take a few
   minutes.
If you choose Yes, the clipart from PowerPoint is added to the ClipArt Gallery. However, you are not prompted to add clipart from Microsoft Word, and the Microsoft Word clipart is not added to the ClipArt Gallery.
