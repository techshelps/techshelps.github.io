OFF97: No Graphics on "Getting Results" Web Page

 Last reviewed: December 31, 1996
Article ID: Q161472
The information in this article applies to:
  • Microsoft Office 97 Windows
  • Microsoft Internet Explorer, versions 3.0, 3.01

SYMPTOMS

When you click Getting Results Book on the Microsoft Office 97 Shortcut Bar, the "Getting Results with Microsoft Office 97" Web page contains no graphics. Instead, each graphic is replaced by a "no graphic available" symbol.

CAUSE

This problem may occur when you install Microsoft Office 97 from an administrative installation (setup /a) point on a network server.

Note that this problem may occur when you use any Internet browser.

WORKAROUND

You can prevent this problem from occurring by copying the necessary graphic files from the Microsoft Office 97 compact disc to a folder in the Microsoft Office 97 folder on the server. Note that your network administrator or a person who has full access to your network server must copy the files. To copy the files, use the following steps:

  1. Start Microsoft Windows Explorer, browse your Microsoft Office 97 compact disc, and then open the folder called Cdonline.

  2. Right-click the Art folder. On the shortcut menu, click Copy.

  3. Using Windows Explorer, connect to the network server that contains the administrative installation of Microsoft Office 97.

    In Windows Explorer, you should see a series of folders called Aamsstp, Bitmaps, Cdonline, etc.

  4. Open the Cdonline folder.

    Note that the Cdonline folder contains a single file, Result97.htm. This is the "Getting Results with Microsoft Office 97" Web page.

  5. On the Edit menu, click Paste.

    This step pastes the Art folder and the four graphics files it contains to the administrative installation point for Microsoft Office 97 on the server.

The graphics appear correctly on the "Getting Results with Microsoft Office 97" Web page. It may be necessary to refresh the Web page by clicking Refresh on the Microsoft Internet Explorer toolbar, or by pressing F5.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you perform an administrative installation (setup /a) of Microsoft Office 97, all of the files that are required to install Microsoft Office 97 are copied to a share on a network server.

However, during an administrative installation, the Setup program does not copy the Art folder in the Cdonline folder to the network server. Although this situation does not cause any problems when you use the link on the "Getting Results with Microsoft Office 97" Web page, it causes the Web page to appear incorrectly when you click Getting Results Book on the Microsoft Office 97 Shortcut Bar.

The following table lists the files in the Art folder. 

   File name       Graphic
   ----------------------------------------------------
   Home_ofc.gif    "Microsoft Office" logo with puzzle pieces
   Homemsft.gif    "Microsoft" logo
   Name.gif        "Microsoft Office 97 Getting Results"
   Navigate.gif    "Go" button
