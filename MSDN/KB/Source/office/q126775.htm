"Cannot open Help File" Accessing Browse Help from MOM

 Last reviewed: September 16, 1996
Article ID: Q126775
The information in this article applies to:
  • Microsoft Office for Windows, version 4.2

SYMPTOMS

In Microsoft Office Manager (MOM), when you change the properties of an item on the MOM menu or toolbar (from the MOM menu, choose Customize, select the appropriate tab, select a program item, and choose the Edit button), you receive the "Cannot open Help file" error message if you choose Browse and then choose the Help button in the Browse dialog box.

RESOLUTION

To work around this problem, do the following to access Browse Help from Program Manager:

  1. From the File menu, choose New, and choose OK.

  2. Choose Browse, and then choose Help.

To close Help, choose Cancel.

STATUS

Microsoft has confirmed this to be a problem with Microsoft Office Manager version 4.2 for Windows. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
