Components of Office version 4.2 Reported as Not 32-Bit Clean

 Last reviewed: November 17, 1997
Article ID: Q135240
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a

SYMPTOMS

When you run MacCheck version 1.0.5 on a Macintosh that has any of the Microsoft Office for the Macintosh versions mentioned above installed on it, MacCheck reports that some of the Microsoft Office components "may not be 32-bit clean."

CAUSE

MacCheck is a utility that examines applications on the hard disk and reports their RAM sizes and checks to see if they are 32-bit clean. All Microsoft Office components are 32-bit clean. However, there is a bit in the resource fork of some components that is not set correctly, and therefore MacCheck lists these components as not being 32-bit clean. In Microsoft Office for the Macintosh 4.2.1, Microsoft Help and Microsoft Query are the only components that MacCheck lists as not being 32-bit clean.
