OFF97: Picture Disappears When Edited or Ungrouped

 Last reviewed: June 23, 1997
Article ID: Q170450
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Word 97 for Windows

SYMPTOMS

When you insert a picture into a Microsoft Office 97 program, and then attempt to edit or ungroup the picture, the picture disappears.

CAUSE

This problem occurs when you do the following:

  • You insert a Windows Metafile (.wmf) picture into a Microsoft Office 97 document.

    -and-

  • You attempt to either edit or ungroup the .wmf picture.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
