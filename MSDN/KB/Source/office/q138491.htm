How to Reinstall Clip Art in Microsoft Office

 Last reviewed: March 12, 1998
Article ID: Q138491

The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Publisher 97

SYMPTOMS

When you insert clip art into a Microsoft Office program, you may receive one of the following error messages: 

   The ClipArt Gallery cannot find this picture. If you've deleted
   this file and you want to remove its preview from the ClipArt
   Gallery, choose remove.

   -or-

   This Clip collection is not designed for use with the Microsoft Clip
   Gallery.

CAUSE

This error message will occur if either of the following is true:

  • The file Popular.pcs (or any of the .wmf files) has been deleted, has been moved, or has been corrupted. (This does not apply to Microsoft Office 97.)

    -or-

  • The ArtGalry.cag file is corrupted.

RESOLUTION

To resolve this problem, use the following steps to reinstall the Microsoft Office clip art:

  1. On the Start menu, point to Settings, and click Control Panel. Then, double-click Add/Remove Programs.

  2. In the list of installed programs, click Microsoft Office. Then, click Add/Remove.

  3. In the Microsoft Office Setup dialog box, click Add/Remove.

  4. Select Office Tools, and click Change Option.

  5. Clear the Clip Art (or Popular Clipart) check box.

  6. Click OK.

  7. Click Continue.

  8. After Setup is done, delete ArtGalry.cag. This file is usually in your Windows folder.

  9. Repeat steps 1-4.

  10. Select the Clip Art (or Popular Clipart) check box.

  11. Click OK.

  12. Click Continue.

After Setup is complete, you should be able to insert clip art into any of your Microsoft Office programs.
