OFF97: Disk Space Requirements for Microsoft Office 97

 Last reviewed: March 13, 1998
Article ID: Q158704

The information in this article applies to:

  • Microsoft Office 97 for Windows, all editions

SUMMARY

When you install Microsoft Office 97 on a computer, the amount of hard disk space required may vary, depending on the type of installation you perform. This article contains a listing of the amounts of space required for the various types of installation.

MORE INFORMATION

There are several different types of installations for Microsoft Office 97: Typical, Complete, Run from CD, Administrative, and Run from Server. The following table lists the maximum hard disk space in megabytes (MB) that is required by each type of installation. 

                              Microsoft Office 97,    Microsoft Office 97,
Type of Installation          Professional Edition        Standard Edition


Typical                                      112.9                    91.5

Complete                                     159.9                   133.9

Run from CD                                   60.2                    51.1

Administrative ("Setup /a")                  323.0                   281.0

Run from Server with Shared                   24.1                    22.8
Applications on the Server 

Run from Server with Shared                   53.2                    51.4
Applications copied locally 

Run from Server when using                     5.2                     5.2
Shared Windows

NOTE: The amount shown for an administrative installation is the amount of space required on the server, not on your workstation.
