OFF98: How to Remove Earlier Versions of Office Programs

 Last reviewed: March 4, 1998
Article ID: Q179862
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SUMMARY

When you perform a drag-and-drop installation of Microsoft Office 98 Macintosh Edition, earlier versions of Microsoft Word, Microsoft Excel, and Microsoft PowerPoint are not removed from the computer. This article explains how to use the Remove Old Office Versions program to remove earlier versions of the Office programs from the computer.

MORE INFORMATION

The Remove Old Office Versions program searches the hard disk for earlier versions of Office programs. The Remove Old Office Versions program can detect and remove the following versions of the Office programs. 

   Program                Versions
   ---------------------------------------------------

   Microsoft Excel        4.0, 5.0, 5.0a
   Microsoft PowerPoint   3.0, 4.0
   Microsoft Word         5.0, 5.1, 6.0, 6.0.1, 6.0.1a
To run the Remove Old Office Versions program, follow these steps:

  1. Insert the Microsoft Office compact disc (CD) into the CD-ROM drive.

  2. Open the Value Pack:Administration Tools folder.

  3. Double-click Remove Old Office Versions.

  4. Click Continue on the welcome screen.

    After the program searches for earlier versions of the Office programs, you are prompted to select the programs you want to remove.

  5. Select the programs that you want to remove, and then click Continue.

    NOTE: If Remove Old Office Versions finds two earlier versions of an Office program, you cannot remove one of the programs without removing the other one. For example, if Remove Old Office Versions finds both Excel version 4.0 and Excel version 5.0 on the computer, a check box labeled "Microsoft Excel (2 copies found)" appears in the dialog box. You can leave the check box selected to remove both versions of Excel, or clear the check box to remove neither version.

  6. Click Delete.

  7. After the software is successfully deleted, click Quit.

The Office programs that you selected in step 5 are removed from the computer. However, some files are not removed by the Remove Old Office Versions program.

NOTE: The Remove Old Office Version program will not search mounted network drives for earlier versions of the Office programs.

NOTE: Many of the files listed, especially those files that are stored in the System Folder, may be shared with other programs or with the System Software. Manually removing these files may affect non-Office programs. If you delete any of the files that are stored in the System Folder, move them to another folder, and then restart the computer. If none of the other programs are affected, it is probably safe to delete the files.
