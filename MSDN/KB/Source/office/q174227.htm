OFF97: Data Access Pack Drivers Not Visible in Control Panel

 Last reviewed: September 24, 1997
Article ID: Q174227
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you create a data source after you install the Microsoft Data Access Pack, the Open Database Connectivity (ODBC) driver for Lotus 1-2-3 or Windows Messaging is not available.

CAUSE

The Data Access Pack installs Installable Indexed Sequential Access Method (IISAM) drivers for Lotus 1-2-3 and Windows Messaging but does not install ODBC drivers for these products. The Microsoft Jet engine can use the IISAM drivers to import or export data from Microsoft Access or Microsoft Excel. However, Microsoft Query cannot be used with these IISAM drivers.

STATUS

This behavior is by design of the Data Access Pack.
