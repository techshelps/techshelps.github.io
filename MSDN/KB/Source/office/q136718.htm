OFF95: Files Installed with Custom (Select All) (Part 1 of 3)

 Last reviewed: March 10, 1997
Article ID: Q136718
7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

The following is a partial list of the folders and files installed when you perform a Custom installation of Microsoft Office and you click the Select All button. This list contains the files installed to the Microsoft Office folder (Msoffice by default), Clipart, Office, Schedule, Sounds, and Templates folders.

To view the additional files installed when you perform a Custom (Select All) installation of Microsoft Office, please see the following article(s) in the Microsoft Knowledge Base:
