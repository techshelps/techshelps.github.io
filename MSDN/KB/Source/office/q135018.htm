OFF: Temporary Files May Be Displayed in "Open" Dialog Box

 Last reviewed: December 3, 1996
Article ID: Q135018
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0, 7.0a
  • Microsoft Office 97 for Windows
  • Microsoft Word for Windows 95, version 7.0, 7.0a
  • Microsoft Word 97 for Windows

SYMPTOMS

In Microsoft Office, you may see Temporary files when you use the Open A Document button on the Office Shortcut Bar.

CAUSE

This problem occurs when you open an existing file in an application, and Show All Files is selected in the View tab of the Options dialog box in Explorer.

WORKAROUND

To work around this problem, follow these steps:

  1. Start Windows Explorer.

  2. On the View menu, click Options.

  3. Select Hide Files Of These Types instead of Show All Files.

  4. Click Apply.

  5. Click OK.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

Temporary files usually have a ~$ at the beginning of the filename. For example, ~$Mydoc.doc is created when you open the document named Mydoc.doc in Word 7.0. This temporary file is located in the same folder that the original file was saved in and opened from.

The behavior is not seen in Microsoft Excel for Windows 95, version 7.0, or in Microsoft PowerPoint for Windows 95, version 7.0.
