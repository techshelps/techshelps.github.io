OFF97: Sounds Not Supported on Windows NT 3.51

 Last reviewed: March 12, 1998
Article ID: Q157046
The information in this article applies to:
  • Microsoft Windows NT, version 3.51
  • Microsoft Office 97 for Windows

SYMPTOMS

When you run Sounds.exe to install Microsoft Office 97 sounds from the ValuPack\Sounds folder, the following message appears: 

   This install program requires Windows 95 or Windows NT 4.0 to run.
   Proceed with the installation of Microsoft Office 97 Sounds?
If you click Yes, the installation proceeds. After you click Yes to accept the End-User License agreement, the following message appears: 

   Initializing...Please Wait.
Then, the following message appears: 

   Windows 95 or Windows NT 4.0 Beta 2 or greater is required for this
   installation.
If you click OK, the program quits.

CAUSE

Microsoft Office 97 sounds are not supported under Microsoft Windows NT 3.51. Microsoft Office 97 sounds are supported only under Microsoft Windows 95 and Microsoft Windows NT 4.0.

STATUS

This behavior is by design of Microsoft Office 97.

MORE INFORMATION

Additional Microsoft Office 97 sounds are provided in the Microsoft Office 97 ValuPack to add animated audio cues to screen elements as you work with them in any Microsoft Office 97 program. For example, sounds play when you work with toolbars, scroll bars, dialog boxes, send mail, or sort information.

After installing ValuPack sounds, you can turn on animated sounds in your Microsoft Office 97 programs by doing the following:

  1. Start an Office 97 program.

  2. On the Tools menu, click Options and click the General tab.

  3. Select the "Provide feedback with sound" check box.

  4. Click OK.
