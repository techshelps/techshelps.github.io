Office: Office 4.3 Includes Access 2.0 Not Version 1.1

 Last reviewed: September 16, 1996
Article ID: Q119405
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.2, 4.3

SUMMARY

Pages 3 and 15 of the "Microsoft Office Getting Started Guide," for versions 4.2 and 4.3, state the following: 

   Note: The Microsoft Office Professional edition includes Microsoft
   Access, a relational database management system. If you have Microsoft
   Access version 1.1, you must install it separately from Office by using
   the Microsoft Access Setup program. For more information, refer to your
   Microsoft Access documentation.
This information is incorrect for Microsoft Office versions 4.2 and 4.3. Microsoft Access version 2.0 (not version 1.1) comes integrated with Microsoft Office version 4.3 Professional. Microsoft Office version 4.2 Standard does not include any version of Microsoft Access.

MORE INFORMATION

This statement is left over from the Microsoft Office 4.0 manual. Microsoft Office version 4.0 Professional did include Microsoft Access version 1.1 which had to be installed separately from the Microsoft Office version 4.0 setup.

REFERENCES

"Office Getting Started Guide," versions 4.2 and 4.3, page 3
