OFF95: Description of Office Setup Installation Options

 Last reviewed: March 10, 1997
Article ID: Q132352
7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

When you run the Microsoft Office Setup program, a dialog box appears with at least the following three installation options: 

   Typical
   Compact
   Custom
If you run Setup from a network server where an administrative installation (Setup /a) has already been performed, the following installation option is available in addition to the above options: 

   Run From Network Server
When you run Setup from a Microsoft Office CD, the following installation option is available in addition to the above options (not including the Run from Network Server option): 

   Run From CD-ROM
For a description of the Microsoft Office installation options, you can do the following in the dialog box that displays the installation option buttons:

  1. Click the Help button in the upper right corner of the Setup dialog box.

  2. Click the installation options button for which you want to view a description.

Note that when you press F1 in this Setup dialog box, the description for the Typical Installation option is displayed. This description contains the most detail about which components are and are not installed when you use this installation option.

MORE INFORMATION

The following is the description that appears for each of the installation option buttons in the Setup dialog box.

Typical

To install all the files most commonly used to run the application, click Typical. The Microsoft Office 7.0 Typical installation option includes Microsoft Office application program files for Microsoft Excel, Word, PowerPoint, and Schedule+, as well as Office Tools for use with Office applications, such as Spelling Checker, Graph 5.0, ClipArt Gallery, and Find Fast. The Typical installation option does not include WordArt, Data Map, the Microsoft Query add-in program, online Help for Visual Basic for Applications, Microsoft Organizational Chart, or the grammar and thesaurus proofing tools.

To view or install items that are not included in the Typical installation option, click Custom. The items selected by default when you choose the Custom installation option are the same items that will be installed automatically if you choose the Typical installation option.

Compact

To install only the minimum number of files needed to run the application, click Compact. The Compact installation option includes the Microsoft Office application program files and minimal Microsoft Office tools. The Compact installation does not include online Help for Microsoft Office applications.

To install items that are not included in the Compact installation option, click Custom. The items selected by default when you choose the Custom installation option are the same items that will be installed automatically if you choose the Typical installation option.

Custom

To manually select the files you want to install, select the check box next to the name of the item in the Options box, and then click Change Options.

To Install all the files for all Office applications, click Select All.

The items selected by default when you choose the Custom installation option are the same items that will be installed automatically if you choose the Typical installation option.

Run From CD-ROM or Run From Network Server

To install on your computer only those files needed to run the applications from the CD-ROM or from the Network, click Run From CD-ROM or Run from Network.
