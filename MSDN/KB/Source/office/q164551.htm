MOM Updater Folder Is Not Installed with Disc 2 Installer

 Last reviewed: November 17, 1997
Article ID: Q164551
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2.1b

SYMPTOMS

When you run the "Install Bookshelf and Tools" program on disc 2 of Microsoft Office for the Macintosh, version 4.2.1b on Compact Disc, the MOM Updater folder may not be copied to your hard disk.

Also, if you perform a Custom installation with the Bookshelf installer program and you select only the Microsoft Office Manager Updater, you may receive the following error message: 

   No installation was necessary. The files to be installed are already
   on your disk, or are not needed for your hardware setup.

   If you are finished, click Quit to leave the Installer. If you wish
   To perform additional installations, click Continue.

CAUSE

The MOM Updater folder is not copied to your hard disk if either of the following conditions are true:

  • You click Easy Install but the Microsoft Office Manager Control Panel icon is not installed.

    -or-

  • You click Custom Install but the Microsoft Office Manager Updater component is not selected.

The error message that is described in this article appears if both of the following conditions are true:
  • You click Custom Install and select only the Microsoft Office Manager Updater.

    -and-

  • The Microsoft Office Manager Control Panel icon is not installed.

RESOLUTION

If you want to copy the MOM Updater folder to your hard disk but the Microsoft Office Manager Control Panel icon is not installed, manually copy the MOM Updater folder from disc 2 onto your hard disk.
