Microsoft PowerPoint and Schedule+ Web Add-ins Available

 Last reviewed: February 25, 1998
Article ID: Q149894
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Schedule+ for Windows 95, version 7.0

SUMMARY

Two add-in products for Microsoft Office 7.0 that make it easier to create and publish information to the Internet or an intranet are available at no charge.

The add-in products are tools for Schedule+ and the PowerPoint presentation graphics program. The tools consist of the Internet Assistant for Schedule+ 7.0, and the Microsoft ActiveX Animation Player for PowerPoint 7.0.

The new Internet Assistant allows you to use Schedule+ to publish information in hypertext markup language (HTML). The HTML format allows Web browsers to view the information across an intranet or via the Internet.

The Microsoft ActiveX Animation Player for PowerPoint 7.0 is a browser extension for publishing and viewing live PowerPoint 7.0 presentations on the World Wide Web.

MORE INFORMATION

Internet Assistant for Schedule+

Internet Assistant for Schedule+ for Windows 95 allows you to publish calendar and schedule information to a web site. Anyone with a web browser is able to view the posted schedule. No special viewer is required.

To download the Internet Assistant for Schedule+, go to the following URL: 

   http://www.microsoft.com/scheduleplus/internet/ia/default.htm

Microsoft ActiveX Animation Player

With the Microsoft ActiveX Animation Player for PowerPoint, you can take advantage of the enhanced animation, hyperlinks, special effects, and built- in sound functionality in PowerPoint 7.0 to build dynamic, animated Web pages without having to learn and use a complex programming tool. In addition, ActiveX Animation Player makes downloading and viewing PowerPoint effects and presentations over the Internet faster by compressing files before you post the files to a web site.

To download the Microsoft ActiveX Animation Player, go to the following URL: 

   http://www.microsoft.com/powerpoint/internet/player/default.htm
NOTE: You have the option to download either the ActiveX player or the viewer. The player tool allows you to both view and publish PowerPoint presentations on the web. The viewer allows you to only view PowerPoint presentations on the web.
