Office: Changes in MOM 4.2b, 4.2c, and 4.3

 Last reviewed: September 16, 1996
Article ID: Q120375
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.2b, 4.2c, 4.3

SUMMARY

The following information discusses the changes that occurred in Microsoft Office Manager (MOM) versions 4.2b, 4.2c, and 4.3.

MORE INFORMATION

  • Because the compression scheme used to shrink a regular button bitmap onto the small buttons causes the bitmaps to display poorly with some video drivers, the following new entry is added by MOM automatically to the [Options] section of MSOFFICE.INI: 

          ShrinkSmallButtons=0

    This entry is set to 0 by default. If you notice the change in appearance of the icons on the small buttons, close MOM, change this value to 1 and restart MOM.

  • Another new entry is automatically added to the [Options] section of MSOFFICE.INI: 

          DontHideMOM=0

    By default, MOM will be hidden when a Microsoft PowerPoint slide-show is running. If you want MOM to be displayed in the foreground during a Microsoft PowerPoint slide show, set this value to 1.

  • If MOM is installed from a CD, an Online Documentation button and menu item will appear by default. This button/menu item starts the online documentation (MSOFF42.MVB) on the CD. To achieve this functionality, Setup writes the following to the [Options] section of MSOFFICE.INI: 

          DocHelp=1
      DocHelpExe=<path>\mviewer2.exe
      DocHelpFile=<path>\msoff42.mvb
  • Beginning with MOM 4.2b, MOM can now successfully find and launch both Access 1.x and 2.0, and FoxPro 2.6 and earlier versions, by searching for REG.DAT registrations (latest versions) and [Extensions] section of WIN.INI (earlier versions that did not register in REG.DAT).
