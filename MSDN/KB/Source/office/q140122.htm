WC1270: File to Allow SBP Setup Without MS Access 7.0

 Last reviewed: September 30, 1997
Article ID: Q140122

The information in this article applies to:
  • Small Business Pack for Windows 95, version 1.0
  • Microsoft Office for Windows 95, version 7.0

SUMMARY

The Application Note "Installing Small Business Pack Without Microsoft Access 7.0" (WC1270) is now available from Microsoft Product Support Services (PSS).

The Office Small Business Pack (OSBP) Setup program requires that you have Microsoft Access for Windows 95, version 7.0, installed on your system; however, the Wc1270 disk included with this Application Note contains a modified Sbp.stf setup file that allows you to install the Small Business Pack without having Microsoft Access version 7.0 on your system.

You can obtain this Application Note from the following sources:

  • Microsoft's World Wide Web Site on the Internet
  • The Internet (Microsoft anonymous ftp server)
  • The Microsoft Network (MSN)
  • Microsoft Download Service (MSDL)
  • Microsoft Product Support Services

For complete information, see the "To Obtain This Application Note" section at the end of this article.

INTRODUCTION

When you run the Setup program for Microsoft Office Small Business Pack for Windows 95, if Microsoft Access for Windows 95, version 7.0, is not installed on your computer, you may receive the following error message: 

   Setup could not find Microsoft Access installed on your system.
   Please install Microsoft Access before installing this application.
The Small Business Pack Setup program requires that Microsoft Access 7.0 be installed on your computer because the database files included with Small Business Pack can only be used with Microsoft Access 7.0.

The modified Sbp.stf setup file included with this Application Note allows you to install Small Business Pack without having Microsoft Access 7.0 installed on your computer.

USING THE MODIFIED SBP.STF FILE

To use the Sbp.stf file, copy it to your hard disk, run the Setup program for Small Business Pack, and use the /T switch to specify that Setup should use the modified .stf file.

To copy the modified .stf file to your hard disk

  1. If you received a disk with this Application Note, insert the Wc1270 disk in the appropriate floppy disk drive.

    If you downloaded this Application Note from an online service, skip to the "To use the modified .stf to install Small Business Pack" section of this document. This procedure assumes that you have already downloaded and extracted the compressed file.

  2. Copy the Sbp.stf file to an empty folder on your hard disk.

To use the modified .stf to install Small Business Pack

  1. Insert Disk 1 of the Microsoft Office Small Business Pack disks in the appropriate floppy disk drive.

  2. On the Start menu, click Run, and type the following in the Open box 

          <source>:\Setup /t <location>\sbp.stf

    where <source> is the letter of the drive that contains Disk 1 of Microsoft Office Small Business Pack and <location> is the path to the modified Sbp.stf file. For example, if Disk 1 is in drive A, and C:\Temp is the path to the folder that contains the Sbp.stf file, type: 

          a:\setup /t c:\temp\sbp.stf

    NOTE: If the path for the source or the location contains long file names, you must enclose the entire path in quotation marks.

The Setup program will install Small Business Pack without error.

WHAT TO DO IF YOU INSTALL MICROSOFT ACCESS 7.0 AFTER YOU USE THIS FILE

If you install Microsoft Access 7.0 and would like to install the Microsoft Access databases that come with the Microsoft Small Business Pack, use the following steps to remove and then reinstall the Microsoft Small Business Pack:

  1. Run Small Business Pack Setup.

  2. Click the Remove All button to remove Microsoft Small Business Pack from your hard disk.

  3. Run Setup again from the Small Business Pack disks (when you run Setup this way, you will access the original Setup.stf file).

All of the Small Business Pack components, including the Microsoft Access 7.0 components, will be installed.
