OFF95: DAO Entry Does Not Appear in Help

 Last reviewed: March 27, 1997
Article ID: Q135501
7.00 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

If you install the Data Access component of Microsoft Office, version 7.0, you may not see a listing for Data Access Objects (DAO) on the Help Contents tab for Microsoft Excel. Additionally, the Help file for DAO may not be listed in the File Options dialog box (to access this dialog box, click the Options button on the Find tab and then click Files).

CAUSE

This problem occurs, for example, when you do a Custom installation of any of the Microsoft products listed at the beginning of this article and you choose not to install the Data Access component, which includes the Help file for Data Access Objects. However, even when you add the Help file through a maintenance setup, the Help file may not be listed in either the Contents tab or the File Options dialog box because the index reference to the DAO Help file is not updated correctly.

Also, when you run a Help file, you can use the Find Setup Wizard (the first time you click the Find tab) to create a list that contains every word from the Help files for your applications. When the word list is created, you can click the Options button on the Find tab to select the options such as which files to search. However, the list of files to search may not contain all of the Help files that are on your computer.

RESOLUTION

If you have the DAO Help file installed, but you cannot see it listed anywhere in Microsoft Excel Help, do the following:

  1. Quit Microsoft Excel if it is running.

  2. In a text editor such as Microsoft WordPad, open the Mainxl.cnt file (this file should be located in your Microsoft Excel folder).

  3. Insert the following text, just below the other entries that begin with ":Index Microsoft Excel ...": 

          :Index DAO Help=dao.hlp

  4. Close the Mainxl.cnt file and save the changes.

  5. Restart Microsoft Excel, and click Microsoft Excel Help Topics on the Help menu.

Microsoft Data Access Objects (DAO) should be listed on the Contents tab. If you click the Find tab, click the Options button, and then click the Files button, you should see the DAO Help file listed.

MORE INFORMATION

For more information about missing files when you build a Help index, please see the following article in the Microsoft Knowledge Base:
