Office 4.2x Update 1.0 May Conflict with Apple Menu Options

 Last reviewed: November 17, 1997
Article ID: Q147389
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2.1, 4.2.1a, 4.2.1b

SYMPTOMS

In Microsoft Office for the Macintosh, your computer may stop responding (hang), or it may crash with various system errors, if both of the following conditions are true:

  • You are using the Office 4.2x Update for Power Mac Extension version 1.0

    -and-

  • You are using the Apple Menu Options control panel version 1.1

RESOLUTION

If you are using the Office 4.2x Update for Power Mac Extension version 1.0, you should obtain and install version 1.0.1 of this file. You can obtain the Office 4.2x Update for Power Mac extension version 1.0.1 from Microsoft Online Services as Application Note MC1164.

For additional information about how to obtain Application Note MC1164, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q130776
   TITLE     : MC1164: Extension to Prevent Hanging Power Macintosh
               Programs
Instructions about how to install the the Office 4.2x Update for Power Mac Extension are contained in Application Note MC1164.
