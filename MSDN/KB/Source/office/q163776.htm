OFF97: IE Shortcut Installs Correct Version 3.01

 Last reviewed: August 13, 1997
Article ID: Q163776
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Internet Explorer for Windows, version 3.01

SYMPTOMS

After you install Office 97 on your computer, a shortcut labelled "Setup for Microsoft Internet Explorer 3.01" appears on your Windows desktop. If you double-click this shortcut, the first dialog box that appears contains the following message: 

   Microsoft Internet Explorer 3.0
   This will install Microsoft Internet Explorer 3.0 (Build 1215). Do you
   wish to continue?
This message may lead you to believe that you are about to install Internet Explorer, version 3.0, instead of version 3.01.

STATUS

This is by design of the Setup program for Internet Explorer, version 3.01. If you continue Setup, the correct version of Internet Explorer will be installed.

MORE INFORMATION

When you run the Setup program for Internet Explorer, version 3.x, the first dialog box that appears contains a Build number. This Build number tells you exactly what version of Internet Explorer 3.x you are about to install. 

   Build Number   Version
   -------------------------------------------------------

   1155           Microsoft Internet Explorer, version 3.0
   1215           Microsoft Internet Explorer, version 3.01
Many of the dialog boxes and help topics in Internet Explorer 3.01 contain the text "Microsoft Internet Explorer 3.0". This is normal behavior, as version 3.01 is a maintenance release and not all references to the version number have been changed.
