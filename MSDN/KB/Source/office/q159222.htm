OFF97: Output Quality Problems Printing Drawing Objects

 Last reviewed: February 5, 1998
Article ID: Q159222
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft PowerPoint 97 for Windows

SYMPTOMS

The following problems can occur when you print drawing objects from Microsoft Word, Microsoft Excel, and Microsoft PowerPoint:

  • Memory Overflow
  • Print Overrun
  • Fill effects missing from objects or do not print correctly
  • Blending of bitmap and vector-based drawing objects that overlap
  • Large oval shapes missing
  • Printing to file causes output to printer rather than to file
  • Textured fills do not print correctly
  • Semitransparent fills, lines, or shadows do not print correctly
  • Pattern fills may not print correctly or do not print in color

The printers covered in this article include: 

   Hewlett Packard LaserJet Series 4
   Hewlett Packard LaserJet Series 5
   Hewlett Packard LaserJet Series 6
   Hewlett Packard LaserJet 5Si
   Hewlett Packard DeskJet 1600C
   Hewlett Packard Color LaserJet/Color LaserJet Postscript
   Microsoft Postscript Printer Driver
   Lexmark Optra R Series
   Lexmark Optra L Series

WORKAROUND

Use the following workarounds for your particular printer.

Determine What Printer Driver is Installed

  1. On the File menu, click Print.

  2. Click Properties.

  3. Click the Paper tab.

  4. Click the About button.

The bottom half of the dialog shows the name of the printer driver installed.

Hewlett Packard LaserJet Series 4, Series 5 and Series 6 Printers

Using the Microsoft Universal Printer Driver in Microsoft Windows 95 can cause many output quality problems with graphics but the problems can be fixed by changing the graphics mode of the printer driver.

If you are using the Microsoft Universal Printer driver, use the following steps to change the graphics mode:

  1. On the File menu, click Print.

  2. Click Properties and then click the graphics tab.

  3. Select the Use Raster Graphics option.

  4. Click OK.

Postscript Printing from Windows NT Workstation version 3.51 and 4.0

Upgrade to the latest version of the Microsoft Postscript Driver for Windows NT Workstation 3.51 or 4.0. The latest driver can be downloaded from the Microsoft Internet site: 

   http://www.microsoft.com/

Postscript Printing from Windows 95

When you use certain Postscript printers and you select the Print to File option, the file may be sent to the printer rather than to the file. The problem can be fixed by upgrading to the latest Postscript driver based on Adobe Postscript 4.1.

Hewlett Packard LaserJet 5Si and Windows 95

To fix the problems, do the following:

  1. On the File menu, click Print.

  2. Click Properties and then click the Print Quality tab.

  3. Click the Advanced button, and then click Settings.

  4. Select the Raster Graphics option and click OK.

Hewlett Packard DeskJet 1600C and Windows 95

To resolve the print quality problems with drawing objects, use the DeskJet printer driver version 4.2 or later. You can download the latest HP DeskJet printer driver from the Hewlett Packard Internet site: 

   http://www.hp.com/cposupport/indexes/dj1600.html#doswin.

Hewlett Packard Color LaserJet and Windows 95

To fix the problem printing drawing objects, use version 1.300 or later of the HP Color LaserJet printer driver. To check the version of the driver currently installed, use the steps listed earlier in this article.

Hewlett Packard Color Postscript and Windows 95
