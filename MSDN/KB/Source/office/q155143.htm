OFF97: One Office Assistant Installed With Complete Setup

 Last reviewed: March 12, 1998
Article ID: Q155143
The information in this article applies to:
  • Microsoft Office 97 for Windows

SUMMARY

When you do a Complete installation of Microsoft Office 97, only one Office Assistant is fully installed. The Clippit character is installed by default.

MORE INFORMATION

When you do a Complete installation of Microsoft Office 97, the following Office Assistant files are installed in the \Office\Actors folder: 

   Clippit.act
   Dot.acp
   Genius.acp
   Hoverbot.acp
   Logo.acp
   Mnature.acp
   Powerpup.acp
   Scribble.acp
   Will.acp
Note that the Clippit character file has the .act (actor file) extension. This is because it is the default character used for the Office Assistant. The other eight files that are installed (*.acp), are preview files and are installed so that you can preview these characters in the Gallery tab of the Office Assistant dialog box.

When you change the Office Assistant, the .act file for that particular character will replace the existing .acp (preview) file for that character in the \Office\Actors folder.

To change Office Assistants, do the following:

  1. Display the Office Assistant if it is not currently displayed.

  2. Use the right mouse button to click the Office Assistant and click "Choose Assistant..." on the shortcut menu.

  3. On the Gallery tab of the Office Assistant dialog box, click either the Back or Next buttons to preview the available Office Assistant. When you locate the one you want to replace the existing character, click OK.

Note, if your Microsoft Office installation source is not available, you will be prompted to insert it (if you installed from a compact disc or floppy disks), or you will be prompted to connect to the network drive from which you installed Microsoft Office.
