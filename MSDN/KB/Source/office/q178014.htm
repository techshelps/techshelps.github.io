OFF98: Button Editor Dialog Box Cannot Be Moved

 Last reviewed: February 2, 1998
Article ID: Q178014
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Word 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition

SYMPTOMS

In the programs listed at the beginning of this article, it is not possible to move the Button Editor window when you edit a button image. As a result, the Button Editor dialog box may cover or obscure tools on some of your toolbars.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

In the programs included in Microsoft Office 97 for Windows and Microsoft Office 98 Macintosh Edition, you can display the Button Editor dialog box by following these steps:

  1. On the Tools menu, click Customize.

  2. Right-click (Windows) or press CONTROL and click (Macintosh) a tool on a toolbar. On the shortcut menu, click Edit Button Image.

The Button Editor dialog box appears. In the programs included in Microsoft Office 97 for Windows, you can move the Button Editor dialog box; in the programs included in Microsoft Office 98 Macintosh Edition, you cannot.

Note that the Button Editor dialog box displays a preview of the button you are currently editing.
