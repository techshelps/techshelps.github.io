"Updating Multimedia System Files..." During Setup

 Last reviewed: September 16, 1996
Article ID: Q121806
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD

SYMPTOMS

When you run the Microsoft Office Setup program in maintenance mode, and you choose to add or remove a component, you receive the following message before the selected item is added or removed: 

   Updating multimedia system files, which will require some additional
   disk space. See the README file for more information.

CAUSE

This behavior is by design of Microsoft Office. This message is initially displayed when you install Microsoft Bookshelf. Because the Microsoft Bookshelf Setup program does not have a maintenance mode, this message appears when you run the Microsoft Office Setup program, even if you are not installing or removing a Bookshelf item.
