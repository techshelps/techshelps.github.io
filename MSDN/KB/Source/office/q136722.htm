OFF95: Files Installed with Custom (Select All) (Part 3 of 3)

 Last reviewed: March 10, 1997
Article ID: Q136722
7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

The following is a partial list of the folders and files installed when you perform a Custom installation of Microsoft Office and you click the Select All button. This list contains the files installed to the Excel, PowerPnt and Winword folders inside the Microsoft Office (Msoffice by default) folder, and the files installed to the Windows folder.

To view the additional files installed when you perform a Custom (Select All) installation of Microsoft Office, please see the following article(s) in the Microsoft Knowledge Base:

   TITLE     : OFF95: Files Installed with Custom (Select All)
               (Part 1 of 3)

   ARTICLE-ID: Q136723
   TITLE     : OFF95: Files Installed with Custom (Select All)
               (Part 2 of 3)
