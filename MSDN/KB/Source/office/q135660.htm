Changing File Properties in Open Dialog Box Not Updated

 Last reviewed: September 16, 1996
Article ID: Q135660
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

When you change the properties of a file in the Open dialog box in any Microsoft Office 7.0 application, the properties are not updated immediately. The old properties continue to be displayed.

WORKAROUND

After you change the properties of the file in the Open dialog box, click the Cancel button in the dialog box. Redisplay the dialog box and file properties. The changes will be reflected in the dialog box.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Office for Windows 95 version 7.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To display the properties of a file in the Open dialog box, click the file to select it and click the Properties button (upper-right corner of the dialog box).

To change the properties from the Open dialog box, select the file that you would like to change the properties for by clicking once on the file. Click the right mouse button once. A shortcut menu will appear with the Properties menu item.
