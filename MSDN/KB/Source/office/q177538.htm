OFF98: Application Error Message Starting Unbind Program

 Last reviewed: February 2, 1998
Article ID: Q177538
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SYMPTOMS

When you run the Unbind program that is included in the Microsoft Office 98 Macintosh Edition Value Pack, you may receive the following error message: 

   The application "Unbind" could not be opened because
   "Microsoft_Office_98" could not be found.

CAUSE

This problem occurs when you run the Unbind program, and the program is not located in either of the following folders on your hard disk 

   <hard disk>:Microsoft Office 98
   <hard disk>:Microsoft Office 98:Office
where <hard disk> is the name of your hard disk. (The default name is Macintosh HD.)

RESOLUTION

To resolve this problem, move the Unbind program to either the Microsoft Office 98 folder or to the Microsoft Office 98:Office folder, and then run the program.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Unbind program that is included in the Microsoft Office 98 Macintosh Edition Value Pack allows you to extract the sections from a binder file and save them as separate workbooks, documents, and presentations.

Note that Unbind is designed only for use on PowerPC-based Macintosh computers. If you attempt to run Unbind on a 680x0-based Macintosh computer, you receive an error message similar to the following: 

   The application program "Binder" could not be opened because an error
   of type -192 occurred.
