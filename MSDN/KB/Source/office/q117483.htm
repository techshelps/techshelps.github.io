Office: How to Install Microsoft Office 4.2 Fulfillment

 Last reviewed: September 16, 1996
Article ID: Q117483
The information in this article applies to:
  • Microsoft Office for Windows, version 4.2

The fulfillment package for Microsoft Office 4.2 that includes Microsoft Office Manager 4.2 (MOM) and PowerPoint 4.0 does not include installation instructions. You must install Microsoft PowerPoint first, because the MOM Setup program specifically searches for certain Microsoft PowerPoint files and cannot complete the installation process if it fails to find those files.

To install the fulfillment pack:

  1. Install Microsoft PowerPoint 4.0.

  2. Quit Microsoft Office Manager if it's running.

  3. Insert the Microsoft Office Disk into the appropriate floppy disk drive.

  4. From Program Manager or File Manager, choose Run from the File menu.

  5. In the Command Line box, type the following 

          <drive>:\setup.exe

    where <drive> is the floppy disk drive where you inserted the Microsoft Office Disk. For example, if you inserted the Microsoft Office Disk in drive A, type: 

          A:\setup.exe

  6. Choose OK.
