Office Resource Kit Glossary Conversion Documentation Error

 Last reviewed: April 2, 1997
Article ID: Q149253
7.00 WINDOWS kbtool kbdocerr

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0

SUMMARY

The Readme.txt file that is included in the Convert\Windows\Word\Glossary folder in the Microsoft Office for Windows 95 Resource Kit is incorrect. The file discusses the conversion of Word 5.0, 5.5, and 6.0 for MS-DOS glossary entries to Word 7.0 AutoText entries.

The following is the incorrect text: 

   The files in this folder (50CONVRT.GLY, 55CONVRT.GLY, 60CONVRT.GLY, and
   MSWORD.DOT) provide the means for converting the contents of Word 5.0,
   5.5, and 6.0 for MS-DOS glossary entries into Word 7.0 for Windows 95
   and Windows NT AutoText entries.

MORE INFORMATION

The files in the Glossary folder (50convrt.gly, 55convrt.gly, 60convrt.gly) are designed to enable you to convert the contents of Word 5.0, 5.5, and 6.0 for MS-DOS glossary entries into AutoText entries in Word 7.0 for Windows 95 and Windows NT. However, a template file required for conversion, Msword.dot, was not included in the folder.

To obtain Msword.dot, call the Microsoft Order Desk at (800) 360-7561 and request the Microsoft Word 6.0 for Windows Supplemental File Conversions Disks. This kit contains an Msword.dot file that you can use with Word 7.0. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

   http://www.microsoft.com/worldwide/default.htm
