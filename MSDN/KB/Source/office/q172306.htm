OFF97: View in New Dialog Box Is Not Saved Between Sessions

 Last reviewed: August 4, 1997
Article ID: Q172306
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Word 97 for Windows

SYMPTOMS

In any of the programs listed at the beginning of this article, when you click List or Details in the New dialog box (click New on the File menu), and then quit the program, the setting is not saved.

CAUSE

This problem occurs because the programs listed at the beginning of this article use the List or Details view only in the current session of the program. When you quit and restart the program, the program uses the large icons view.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To change the view in the New dialog box, follow these steps:

  1. On the File menu, click New.

  2. Above Preview, click Large Icons, List, or Details to change the view.

  3. Click the template that you want to use and click OK.
