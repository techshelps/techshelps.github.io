OFF: "Illegal Operation" Closing Binder That Contains Map

 Last reviewed: March 12, 1998
Article ID: Q134270

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Office 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows

SYMPTOMS

In Microsoft Office Binder, when you create a Microsoft Excel worksheet section and insert a Data Map object, you may receive an illegal operation error message in Microsoft Excel when you close the Binder. When you click Details, information similar to the following is displayed: 

   Excel caused an invalid page fault in module EXCEL.EXE at
   0137:50048605

CAUSE

This problem occurs when you insert a Microsoft Map (Data Map in version 7.0) object in the Microsoft Excel worksheet section of your Binder if you size the map object, and then close the Binder while the map object is still being sized. This behavior occurs if you size the map and choose not to save changes to the Binder (the first time you save the Binder) or if you close a Binder that has already been saved after sizing a Microsoft Map (Data Map in version 7.0) object in the Binder. This problem occurs if the map has not been completely resized when you close the Binder that contains the object.

Note that when you insert a Map object in a worksheet section in a Binder, the Microsoft Map (Data Map in version 7.0) application opens in a new window (that is, you are unable to edit the object in-place).

WORKAROUND

To avoid receiving an illegal operation error message when you close a Binder that contains a map object that you have just sized in a worksheet section, wait until the object has been completely resized (and has been displayed) before you close the Binder.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about how to add a new section to a Binder, click the Index tab in Binder Help, type the following text 

   Binder, a
double-click the selected text, and then double-click "Add A New Document To A Binder" to go to the "Add A New Document To A Binder" topic.
