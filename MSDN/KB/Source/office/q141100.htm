OFF95: Upgrade Voucher Missing from Office Professional Box

 Last reviewed: March 10, 1997
Article ID: Q141100
7.00 WINDOWS kbother

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Access for Windows 95, version 7.0

SYMPTOMS

Microsoft Office Professional for Windows 95 (without Microsoft Access) should include a voucher for Microsoft Office Professional upgrade. It is possible that you received a package without the voucher. If you purchased Microsoft Office Professional on floppy disks, you have the option to receive Office for Windows 95 Professional Edition on compact disc or the Microsoft Access standalone program on floppy disks (when these are available). If you purchased Microsoft Office Professional on compact disc, you will receive a new copy of Office for Windows 95 Professional Edition on compact disc that contains Microsoft Access version 7.0 (when it is available).

RESOLUTION

To receive a voucher for Microsoft Office Professional, do the following:

  1. Prepare the following information and enclose it in an envelope: 

        - A signed letter stating that the voucher was missing.
    - A copy of your receipt.
    - Your return address, daytime phone number, and fax (if available).

  2. Mail the above information to the following address:

    In the U.S. -----------

    Microsoft Access and Bookshelf offer for Windows 95 P.O. Box 660 Buffalo, NY 14207-0660

    -or-

    Fax the information to the following number:

    (716) 873-0906

    If you have additional questions concerning the voucher, call (800) 360-7561 to speak to a Microsoft representative.

    In Canada ---------

    Microsoft Order Centre PO Box 643 Fort Erie Ontario L2A 6M1

    If you have additional questions concerning the voucher, call (800) 375-0816 to speak to a Microsoft representative.

STATUS

Microsoft has confirmed this to be a problem with the products listed at the beginning of this article. We will post new information here as it becomes available.
