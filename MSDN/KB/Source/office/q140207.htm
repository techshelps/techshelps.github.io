OFF95: Unable to Save/Open Office Files with Novell Server 2.x

 Last reviewed: March 27, 1997
Article ID: Q140207
7.00 WINDOWS kbusage kbnetwork

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

When you save files to or open files from a Novell 2.15 or 2.2 server, you may receive the following error: 

   You do not have access to the folder <network drive letter>

WORKAROUND

To open and save files to Novell Server 2.x from a workstation, use Windows Explorer to copy, move, and open files to and from the network locations.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Office for Windows 95 version 7.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
