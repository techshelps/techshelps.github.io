Office 97 Requires Internet Explorer 3.01

 Last reviewed: September 10, 1997
Article ID: Q159707
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Access 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Project 97 for Windows
  • Microsoft Word 97 for Windows

SUMMARY

Microsoft Office 97 for Windows requires that Microsoft Internet Explorer 3.01 be installed for proper Internet and Hypertext Markup Language (HTML) functionality.

MORE INFORMATION

Determining What Version of Microsoft Internet Explorer Is Installed

  1. Start the Internet Explorer.

  2. On the Help menu, click About Internet Explorer.

The version of the Microsoft Internet Explorer appears near the top of the About Internet Explorer dialog box. If the version is "Version 3.0 (4.70.1215)," the correct version of Internet Explorer is installed.

Installing Internet Explorer 3.01

To install Internet Explorer 3.01, use one of the following methods. (Note that you should quit all open Microsoft programs before you install Internet Explorer 3.01.)

NOTE: Methods 1 and 2 are available only if you installed Office 97 from the Office 97 compact disc. If the Office 97 compact disc is not available, use Method 3.

Method 1

Click the shortcut "Setup for Microsoft Internet Explorer 3.01" on the Windows 95 desktop. This shortcut, which is created when you install Office 97, allows you to install Microsoft Internet Explorer 3.01.

If you deleted or moved the shortcut to the Windows 95 Recycle Bin, use method 2 or 3.

Method 2

Use the Office 97 compact disc to install Internet Explorer 3.01. To do so, follow these steps:

  1. On the Windows Start menu, click Run.

  2. Click Browse.

  3. In the "Look in:" list, click the drive that contains the Office 97 compact disc (for example, click drive D).

  4. Double-click the ValuPack folder.

  5. Double-click the IExplore folder.

  6. Click msie301.exe, and then click Open.

  7. In the Run dialog box, click OK.

Method 3: Download Internet Explorer 3.01 from an Online Location

The following file is available for download from the Microsoft Software Library:
