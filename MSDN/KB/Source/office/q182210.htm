OFF98: Office Programs Fail to Start with Outdated Library File

 Last reviewed: March 9, 1998
Article ID: Q182210
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SYMPTOMS

When you start one of the Microsoft Office 98 Macintosh Edition programs, you may experience one of the following problems:

  • You receive the following error message: 

          The application <Program Name> has unexpectedly quit, because an
      error of type 3 occurred.


          You should save your work in other open applications and restart the
      computer.

    where <Program Name> is the name of the Office Program that you are starting.

    -or-

  • The startup screen for the Office program appears briefly, but the program does not start.

CAUSE

These problems will occur if an older version of the file Microsoft Component Library exists in the System Folder:Extensions folder. Office 98 installs version 2.22 of the file Microsoft Component library.

WORKAROUND

To resolve the problem, follow these steps:

  1. Open the System Folder:Extensions folder.

  2. Move the file Microsoft Component Library to the Trash.

  3. Click Empty Trash on the Special menu.

  4. Start any of the Office 98 programs.

The first run installation process will run; then, the Office program will be started.

MORE INFORMATION

To find the version number of the file Microsoft Component Library, follow these steps:

  1. Open the System Folder:Extensions folder.

  2. Click Microsoft Component Library.

  3. Click Get Info on the File menu.

The version number of the file is listed in the Info window. If the version number of the file is less that 2.22, you need to follow the steps listed in the "Workaround" section of this article.

Other programs may install a version of Microsoft Component Library that is not compatible with Office 98. For example, when you install the full version of Macintosh operating system 8.1, version 2.20 of Microsoft Component Library is placed in the System Folder:Extensions:MS Library Folder folder. Normally, this would not cause a conflict with the Office 98 programs. However, if this version of the file is moved into the System Folder:Extensions folder, you will experience the problems described in the "Symptoms" section of this article.

If you have any version of Microsoft Component Library in the System Folder:Extensions:MS Library Folder folder, it will be removed if the first run installation process runs.
