OFF98: Office Installation Replaces 128-bit Encryption Library

 Last reviewed: February 26, 1998
Article ID: Q181507
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SYMPTOMS

When you install Microsoft Office 98 Macintosh Edition, the shared library file that provides 128-bit encryption for Microsoft Internet Explorer may be replaced with a version that provides only 40-bit encryption.

CAUSE

When you perform a drag-and-drop installation of Office 98, version 4.0.4 of the MS Internet Library (PPC) file is installed in the System Folder:Extensions:MS Library Folder. This version of MS Internet Library (PPC) is designed to work with the Web features that are included in each of the Office 98 programs. However, version 4.0.4 replaces version 4.0.3 of this file, which is the version that is installed with Microsoft Internet Explorer and that provides 128-bit encryption.

NOTE: Version 4.0.3 of MS Internet Library (PPC) is replaced by the first run installation process. Under certain circumstances, the first run installation process may run after the initial installation of Office 98.

For additional information on the first run installation process, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q179217
   TITLE     : OFF98: Using First Run Installation to Troubleshoot Mac
               Office

WORKAROUND

To work around this behavior and use 128-bit encryption with Microsfot Internet Explorer, reinstall version 4.0.3 of the MS Internet Library (PPC) file.

NOTE: The Web features in the Office 98 programs have not been tested with version 4.0.3 of MS Internet Library (PPC). Using this version of MS Internet Library (PPC) with Office may lead to unexpected results when you use the Web features in the Office 98 programs.

To check for an update to the 128-bit version of Microsoft Internet Explorer that is compatible with Office 98, visit the following Microsoft Internet Explorer download site: 

   http://www.microsoft.com/ie/download/128bit.htm
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/

STATUS

This behavior is by design of Microsoft Office 98 Macintosh Edition.
