MC1164: Extension to Prevent Hanging Power Macintosh Programs

 Last reviewed: March 13, 1998
Article ID: Q130776
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b

The Application Note "Extension to Prevent Hanging When You Start Power Mac Programs or Use QuickDraw GX" (MC1164) is now available from Microsoft Technical Support(MTS). This Application Note includes the Office4.2x Update for Power Mac extension version 1.0.1. This extension is for use with Office 4.2 and 4.2.1 for the Power Macintosh, Word 6.0 and 6.0.1 for the Power Macintosh, and Microsoft Excel 5.0 and 5.0a for the Power Macintosh. Installing and enabling this extension will correct the following two problems:

  • Starting an application after Microsoft Word causes a System Error 11 message and your computer stops responding (hangs).

    -and-

  • Printing with QuickDraw GX 1.1.1 or 1.1.2 causes a System Error 11 message and your computer hangs.

Note, because there are many situations for which you may receive a System Error 11, the Office4.2x Update for Power Mac extension may not prevent it from happening. If you still receive Type 11 errors after installing the extension, you may have to troubleshoot this problem further.

INTRODUCTION

This Application Note includes the Office4.2x Update for Power Mac extension version 1.0.1. This extension is for use with Office 4.2 and 4.2.1 for the Power Macintosh(TM), Word 6.0 and 6.0.1 for the Power Macintosh, and Microsoft Excel 5.0 and 5.0a for the Power Macintosh. Installing and enabling this extension will correct the following two problems:

  • Starting an application after Word causes a System Error 11 message and your computer stops responding (hangs).

    If you are using a Power Macintosh, you may receive a System Error 11 message, and the computer may hang when you perform the following steps (note that this occurs most frequently when the Virtual Memory option is set to On):

    1. Start Word.

    2. Quit Word.

    3. Start another Power Mac application or restart Word.

  • Printing with QuickDraw GX 1.1.1 or 1.1.2 causes a System Error 11 message and your computer hangs.

    If you are using a Power Macintosh running QuickDraw GX 1.1.1 or QuickDraw GX 1.1.2, you may receive a System Error 11 message and your computer may hang when you try to print from either Word or Microsoft Excel.

    NOTE: As with all extensions, the Office4.2x Update for Power Mac extension is not required to run Microsoft Office, and it can be turned off (disabled) or removed. If the extension is disabled or removed, the problems described here will return.

INSTALLATION

For the Office4.2x Update for Power Mac to work, it must be installed in the Extensions folder in the System Folder. To install the extension, use these steps:

  1. If you received a disk with this Application Note, insert the disk in the disk drive.

    If you downloaded this Application Note from an online service, skip to step 2. This procedure assumes that you have already downloaded and extracted the compressed file.

  2. Drag the Office4.2x Update for Power Mac icon to the System Folder.

    You will receive a message stating "Extensions need to be stored in the Extensions folder in order to be available to the Macintosh. Put 'Office4.2x Update for Power Mac' into the Extensions folder?"

    When you choose Yes, the System automatically places the file in the Extensions folder.

  3. Restart the Macintosh.

QUESTIONS AND ANSWERS

1. Q. Are 68K Macintosh computers affected by these problems?

   A. No, the problems only affect Power Macintosh computers.
      Customers running Office applications on 68K Macintoshes do not
      need the extension.

2. Q. What is the relationship between these problems and
      virtual memory?

   A. The first problem described in the Introduction section of this
      Application Note is more likely to occur when virtual memory is
      on. There is no relationship between the QuickDraw printing
      problem and virtual memory. Nonetheless, we recommend that all
      Power Mac users running Office 4.2 or 4.2.1 for the Power
      Macintosh, Word 6.0 or 6.0.1 for the Power Macintosh, or
      Microsoft Excel 5.0 or 5.0a for the Power Macintosh install the
      extension.

3. Q. If I have QuickDraw GX 1.1.2, should I use the extension?

   A. Yes. We recommend that all Power Macintosh users running Office
      4.2 or 4.2.1 for the Power Macintosh, Word 6.0 or 6.0.1 for the
      Power Macintosh, or Microsoft Excel 5.0 or 5.0a for the Power
      Macintosh install the extension.
