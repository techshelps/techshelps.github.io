How to Determine CRC Checksums for Office 4.3c

 Last reviewed: April 2, 1997
Article ID: Q149492
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c

SUMMARY

A utility that allows you to verify the integrity of each Microsoft Office for Windows disk is available. This utility tests the internal cyclic redundancy check (CRC) checksum for Distribution Media Format (DMF) disks included in the U.S. version of Microsoft Office.

To obtain this utility, see the "To Download This File" section at the end of this article.

To use the utility

This procedure assumes that you have already downloaded and extracted the compressed file.

  1. At the MS-DOS prompt, type the following and press ENTER 

          copy <source>:\crc.exe <location>

    where <source> is the directory containing the file you downloaded from online services, and <location> is the path to your C:\Dos directory. For example, if you downloaded the Crc.exe file from online services to C:\Dnload, and C:\Dos is the path to your Microsoft DOS folder, type: 

          copy c:\dnload\crc.exe c:\dos

  2. At the MS-DOS prompt, type any of the following commands (without quotation marks) depending on which disk you are verifying. 

          Type this command               To verify this disk
      ---------------------------------------------------

      "crcdsk a:"                     for disks 2 through 23
      "crcdsk a: -x0,19,2363"         for disk 1
      "crcdsk a: -x0,19"              for disk 24

  3. Put the Microsoft Office disk in question in the A drive (assuming it is a 3.5-inch drive), and press ENTER.

  4. Compare the resulting checksum number with the checksum in the CRC checksums table in this article.

The CRC checksum utility allows you to detect a problem with any of the Microsoft Office 4.3c DMF disks. If the CRC checksum reported by the utility does not match the checksum listed in the table for the disk in question, the disk format is no longer valid.

If the disk format of your Microsoft Office for Windows DMF disk is no longer valid, call the Microsoft Order Desk at (800) 360-7561 and request a replacement set of disks. If you are outside the United States, contact the Microsoft subsidiary for your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

   http://www.microsoft.com/worldwide/default.htm
