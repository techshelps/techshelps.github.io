Word, Excel Crash When Printing with QuickDraw GX 1.1.1 or 1.1.2

 Last reviewed: November 17, 1997
Article ID: Q130830
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a

SYMPTOMS

When you print from Microsoft Word or Microsoft Excel using QuickDraw GX 1.1.1 or 1.1.2, Word or Microsoft Excel may hang or crash. This problem occurs only on a Power Macintosh with QuickDraw GX (1.1.1 or 1.1.2) installed. This problem does not occur with 68K computers.

NOTE: If you are experiencing this problem and you are NOT using a Power Macintosh, it is from a different cause and you should follow standard troubleshooting steps to locate the source of the difficulty.

CAUSE

This is caused by a problem with the compiler used to compile Word and Microsoft Excel. Microsoft PowerPoint is not affected by this problem because it uses a different compiler.

RESOLUTION

This problem was corrected with an extension available in an Application Note from Microsoft titled "Extension to Prevent Hanging When You Start Power Mac Programs or Use QuickDraw GX." The Application Note number is MC1164.

For information on how to obtain this application note, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q130776
   TITLE     : AppNote: "Office 4.2x Update For Power Mac" (MC1164)
