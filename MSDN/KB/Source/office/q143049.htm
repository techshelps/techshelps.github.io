Windows NT CPS Utility Doesn't Duplicate Office Correctly

 Last reviewed: March 12, 1998
Article ID: Q143049

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Office 97 for Windows
  • Microsoft Windows NT Resource Kit for Windows NT version 3.51

SYMPTOMS

The Computer Profile Setup (CPS) utility that ships with the Microsoft Windows NT Resource Kit, version 3.51, does not correctly duplicate Microsoft Office for Windows programs.

CAUSE

This behavior is by design of the CPS utility.

RESOLUTION

To install Microsoft Office for Windows, perform an administrative installation to the server location, and then for each workstation, perform a workstation installation from the administrative installation point.

If you are installing Microsoft Office for Windows 95, you can use the Network Installation Wizard that is included in the "Microsoft Office for Windows 95 Resource Kit." The Network Installation Wizard allows you to modify the Microsoft Office for Windows 95 Setup information files to create a customized network installation of Microsoft Office. The Wizard is fully documented in Chapter 9, "Customizing Client Installations," in the "Microsoft Office for Windows 95 Resource Kit." Please review the information in this chapter before you attempt to use the Network Installation Wizard.

For additional information regarding the "Microsoft Office for Windows 95 Resource Kit," please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q140619
   TITLE     : OFF7: Contents of the MS Office for Windows 95 Resource
Kit
NOTE: If you are installing Microsoft Office for Windows, versions 4.0, 4.2, or 4.3, you can modify the setup table files (.stf) to automate custom installations of programs; however, you cannot use the Network Installation Wizard to modify the .stf file.

For additional information, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119357

   ARTICLE-ID: Q123848
   TITLE     : Creating Administrative Installation for Multiple
               Platforms

MORE INFORMATION

The CPS utility is included in the Microsoft Windows NT 3.51 Resource Kit. The utility allows network administrators to easily install either Windows NT Workstation 3.51 or Windows NT Server 3.51 on multiple identical x86- based computers.

If you use the utility to duplicate Microsoft Office programs, the registry is not updated correctly. The utility is designed to duplicate only Windows NT Workstation or Windows NT Server.

Note: To run Microsoft Office 97 on Windows NT, Service Pack 5 (SP5) is required.
