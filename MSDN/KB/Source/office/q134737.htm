Office: Imager Description Is Truncated in Setup Dialog Box

 Last reviewed: September 16, 1996
Article ID: Q134737
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD

SUMMARY

When you run the Microsoft Office 4.3c-CD Setup from CD-ROM, the description for Microsoft Imager is truncated, as follows: 

   An application that allows you to

MORE INFORMATION

Microsoft Imager is an application that helps Microsoft Word and Office users import, edit, and manipulate pictures easily within Microsoft Word or other Microsoft Office products.

To find the Microsoft Imager option, follow the appropriate procedure below.

If You Are Running Setup for the First Time

  1. Run SETUP.EXE from the Microsoft Office CD-ROM.

  2. When you are prompted for the type of installation to run, choose Complete/Custom.

  3. Select the Shared Applications option and click the Change Option button.

  4. In the Shared Applications dialog box, click the Imager option.

If Microsoft Office Is Already Installed

  1. Run Setup via the Microsoft Office Setup icon in the Microsoft Office program group.

  2. When you are prompted for the type of installation to run, choose Add/Remove.

  3. Select the Shared Applications option and click the Change Option button.

  4. In the Shared Applications dialog box, click the Imager option.

STATUS

Microsoft has confirmed this to be a problem in the version of Microsoft Office listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
