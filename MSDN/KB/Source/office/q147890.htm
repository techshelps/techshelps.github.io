OFF95: Summary List of Fixed Bugs in MS Office Version 7.0a

 Last reviewed: April 2, 1997
Article ID: Q147890
7.00a WINDOWS kbref kbfixlist

The information in this article applies to:

  • Microsoft Office for Windows 95, versions 7.0a
  • Microsoft Excel for Windows 95, versions 7.0a
  • Microsoft PowerPoint for Windows 95, versions 7.0a
  • Microsoft Word for Windows 95, versions 7.0a

The following table lists fixed problems in Office version 7.0a. This is not a comprehensive list and is current as of February, 1996. For more information about each issue, query in the Microsoft Knowledge Base on the article identifier (ID).
