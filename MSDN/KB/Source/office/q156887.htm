OFF97: Button Displays Image of First Icon on OSB

 Last reviewed: March 13, 1998
Article ID: Q156887

The information in this article applies to:

  • Microsoft Office 97 for Windows

SYMPTOMS

When you add a button to the Office Shortcut Bar (OSB) by using the Customize Dialog, the button displays the same icon image as the first button on the toolbar.

RESOLUTION

To correct this behavior, right-click on an empty area of the Office Shortcut Bar and click Refresh Icons. This causes the icon images to redraw and display the correct icons for the buttons you added to the Office Shortcut Bar.

STATUS

We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

REFERENCES

For more information about Adding Shortcuts to the Office Shortcut Bar, click the Index tab in Microsoft Excel Help, type the following text 

   Adding Buttons
and then double-click the selected text to go to "Adding Buttons" topic.
