ORK: No Autorun on Office Resource Kit Compact Disc

 Last reviewed: September 16, 1996
Article ID: Q149031
The information in this article applies to:
  • Microsoft Office Resource Kit for Windows 95, version 7.0

SYMPTOMS

When you insert the Microsoft Office Resource Kit for Windows 95, version 7.0, compact disc in the CD-ROM drive, the Autorun program does not execute.

CAUSE

The Microsoft Office Resource Kit compact disc has a label that reads as follows: 

   For installation, insert CD-ROM and follow onscreen instructions.
This label suggests that there is an Autorun program similar to that in Microsoft Office for Windows 95, version 7.0. This is not the case. In actuality, an Autorun program does not exist on the Microsoft Office Resource Kit for Windows 95 compact disc.

RESOLUTION

To resolve this problem, perform the following steps:

  1. Start Microsoft Word for Windows 95, version 7.0.

  2. On the Office Resource Kit compact disc, open the Readme.doc file located in the root folder.

    This Readme file lists the contents of the compact disc, and the procedures you can use to install each component.

MORE INFORMATION

Page 881 (Appendix D) in the Microsoft Office Resource Kit manual also implies that the Microsoft Office Resource Kit compact disc has an Autorun feature by stating the following: 

   The CD runs automatically. Insert it in your CD-ROM drive, and follow
   the instructions on your screen to browse the contents of the CD, copy
   files, or install software.
