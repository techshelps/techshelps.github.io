OFF97: Setup Switches for Microsoft Office 97 SR-1 Patch

 Last reviewed: August 13, 1997
Article ID: Q172399
The information in this article applies to:
  • Microsoft Office 97 for Windows, Service Release 1 (SR-1)

SUMMARY

The Microsoft Office 97 SR-1 Patch accepts a number of optional switches that you can use to control how the Setup program operates. This article lists the switches and describes the functionality of each switch.

MORE INFORMATION

Setup Switches

The following table lists the Setup switches you can use with SR-1. 

Switch              Function


/q                  Specifies that Setup be run in quiet installation mode.
                    No user prompts are displayed. This switch can be used
                    to automate installation so that no user intervention
                    is required.

                    NOTE: Windows is not automatically restarted after you
                    run the patch with the /q switch. You must restart the
                    computer immediately after you install the patch with
                    this switch for the update to function properly.

/t:path             Changes the temporary folder to path. This switch
                    specifies that a location other than the system Temp
                    folder be used to extract the files during
                    installation.

/c                  Extracts files to the folder you specified with the /t
                    switch but does not run the patch program.

                    NOTE: After you extract the files to a path, you must
                    run the Off97sr1.exe file to begin the installing the
                    patch. Do not run the Setup.exe file that is in the
                    same folder.
NOTE: Switches are not case sensitive. For example, typing "/Q" (without the quotation marks) is the same as typing "/q" (without the quotation marks).

Using the Setup Switches

To use command line switches with the patch, follow these steps:

  1. If you downloaded the patch from the Microsoft Web site, download the Sr1off97.exe file to your hard disk without running it.

    If you are using the patch from a CD or a floppy disk, run the program on the original media or copy it to your hard disk first.

  2. Under Microsoft Windows 95 or Windows NT Workstation 4.0, click Run on the Start menu, and then type the path and file name for Sr1off97.exe followed by the command line switches you want to use. For example, use the following syntax: 

          <path>\sr1off97.exe /<switch>

    Under Windows NT Workstation 3.51, click Run in the Program Manager or on the File menu in File Manager, and then type the path and file name for Sr1off97.exe followed by the command line switches you want to use. For example, use the following syntax: 

          <path>\sr1off97.exe /<switch>

    For example, if you downloaded the patch to the Temp folder on drive C, type the following 

          c:\temp\sr1off97.exe /q

    to run the installer with no user prompts.
