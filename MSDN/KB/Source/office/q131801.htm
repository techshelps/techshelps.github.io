OFF: Can Only Unhide One Binder Section at a Time

 Last reviewed: December 2, 1996
Article ID: Q131801

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Office 97 for Windows

SYMPTOMS

In Microsoft Office Binder, when you use the Unhide command (Section menu) to unhide a section in your Binder, you can only select a single section at a time in the Unhide Sections dialog box.

CAUSE

This behavior is by design of Microsoft Office Binder. Although you can hide multiple sections in a Binder at a time (by holding down either the SHIFT key or the CTRL key and clicking the sections), you can unhide only one section at a time. In the Unhide Sections dialog box, you cannot use the CTRL key or the SHIFT key to select multiple sections. You can select only one section at a time, and then choose OK to display the selected section.

Note that this behavior is the same in Binders as it is in Microsoft Excel; in Microsoft Excel, you can hide multiple worksheets at a time, but you can unhide only one worksheet at a time.
