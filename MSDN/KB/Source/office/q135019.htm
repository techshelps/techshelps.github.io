Access 2.0 ODBC Driver Is Not Listed in Office Setup

 Last reviewed: September 16, 1996
Article ID: Q135019
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD with Bookshelf

SUMMARY

The Microsoft Access version 2.0 Open Database Connectivity (ODBC) driver is not listed as an option when you run Microsoft Office version 4.3 for Windows with Bookshelf.

MORE INFORMATION

The Microsoft Access version 2.0 ODBC driver is installed only if you choose to install Microsoft Access during the installation of Microsoft Office. This behavior occurs only in the version Microsoft Office 4.3c-CD that is shipped on CD-ROM with the Microsoft Bookshelf application.

To install the Access 2.0 ODBC driver, perform the following steps:

  1. Run Setup.exe from the Microsoft Office version 4.3 CD-ROM.

  2. Choose the Custom installation option when prompted for the type of installation to run.

  3. Make sure that Microsoft Access has been selected.

  4. Choose any other components of Microsoft Office that you want to install and continue with the Setup program.
