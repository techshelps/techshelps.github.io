Office: Cannot Run Demo Using 8514 or 8514/a Display Driver

 Last reviewed: September 16, 1996
Article ID: Q121803
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD

SYMPTOMS

When you run one of the Office Compatible Demos included with Microsoft Office 4.3c-CD, you may receive the following error message 

   "<demo>" cannot be run on the current display driver:
   "<display driver>".
   Please change the display driver if you wish to run this demo.
where <demo> is the name of the demo that you selected to run, and <display driver> is the type of video display driver that you are using, such as 8514 or 8514/a.

CAUSE

This behavior occurs if you are using the 8514 or 8514/a video display driver when you run one of the following Office Compatible Demos: 

   Capsule
   METZ Phones - Address and Telephone Number Management
   Microsoft Project
   ProdeaSynergy

WORKAROUND

To run one of the demos listed above, use the Windows Setup icon to change your display driver to the Microsoft Windows VGA display driver. For more information about Changing System Settings, choose the Search button in Setup Help and type: 

   changing hardware settings

MORE INFORMATION

When you run the A-Talk for Windows demo, the introduction displays the following message: 

   Note: the pictures in this presentation look best if Windows is set up
   with a driver resolution of 800 by 600 pixels.
However, you can still run the demo even if you are using a video driver that does not have a resolution of 800 by 600 pixels.
