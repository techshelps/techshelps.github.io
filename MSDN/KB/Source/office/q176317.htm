OFF97: Error Submitting Web Search with Grouped Fields

 Last reviewed: November 7, 1997
Article ID: Q176317
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you use the Microsoft Office 97 Web Find Fast search page to return Web search results to your Hypertext Markup Language (HTML) page, the following error message appears: 

   Error in query or search: One of the property names
   appears twice. See your administrator to correct the
   error."

CAUSE

This problem occurs when you choose the same item in more than one of the Group By boxes in the Web search page.

WORKAROUND

To work around this problem, do not select the same item in the Group By boxes when you submit a web search. For example, if your search criteria groups the Author field twice, change the second box to a different field or select (none).

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Microsoft Office Web Find Fast indexer is available on the Microsoft Office compact disc in the Srvpack folder. This utility allows you to index multiple document locations on your Web server and create a hierarchical list of documents sorted or grouped by one or more document fields. You can group or sort by Office document fields, such as author, category, keywords, subject, or title. For more information about creating a Web search page that incorporates this functionality, see the Webpage.htm file that is located in the Websrch folder in the Srvpack folder on your Microsoft Office 97 compact disc.
