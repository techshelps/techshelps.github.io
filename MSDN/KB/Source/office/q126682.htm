OFFICE: Word Clipart Not Added To ClipArt Gallery

 Last reviewed: November 17, 1997
Article ID: Q126682
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2

SYMPTOMS

After you install Microsoft Office, when you insert a Microsoft ClipArt Gallery object in a document for the first time, you are prompted to add clipart from Microsoft Office 4.2; however, the Microsoft Word clipart files are not added to ClipArt Gallery.

CAUSE

This behavior occurs because the Microsoft Word clipart files are not automatically added to the Microsoft ClipArt Gallery. When you start Microsoft ClipArt Gallery for the first time after installing Microsoft Office, you can choose to add clipart from Microsoft Office 4.2. If you choose yes, the Microsoft PowerPoint clipart is added to the ClipArt Gallery, but the Microsoft Word clipart is not added.

WORKAROUNDS

To work around this problem, you can add the Word clipart files by doing the following:

  1. In a new document in a Microsoft Office application, choose Object from the Insert menu.

  2. From the Object Type list, select Microsoft Clipart Gallery, and choose OK.

    Note that the Clipart Gallery can only be run from within a host application.

  3. In the Microsoft Clipart Gallery dialog box, choose the Options button.

  4. Choose the Add button to add thumbnail pictures for files you select to the Clipart Gallery.

  5. From the list of folders, select the Microsoft Office:Microsoft Word:Clip Art folder.

  6. When the Clip Art folder is selected, choose the Select All In Folder button, and choose Add.

Note that you can choose to either categorize your clipart, or not. Categorizing your clipart will make it easier for you to locate your clipart.

The ClipArt Gallery will then add all of your Microsoft Word clipart.
