OFFICE: Location of Files on Setup Disks (PowerMac)

 Last reviewed: November 17, 1997
Article ID: Q125011
The information in this article applies to:
  • Microsoft Office for the Power Macintosh, version 4.2

SUMMARY

The information in this article provides the location of each file on the Microsoft Office for the Power Macintosh Setup disks. Note that these files cannot be decompressed or extracted individually from the disks, they can only be installed using the Setup program.

NOTE: This information may be useful when troubleshooting a Setup issue. For example, if Setup cannot read a particular disk, or if the disk appears to be damaged, you can use the following information to determine which item in Setup to clear, so that Setup does not request the disk that is not being read.
