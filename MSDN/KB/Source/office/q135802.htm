OFF95: Assign Macro Dialog Box May Not Appear

 Last reviewed: March 10, 1997
Article ID: Q135802
7.00 WINDOWS kbusage

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0

SYMPTOMS

In the Microsoft programs listed above, you can click Assign Macro on the Tools menu or on the shortcut menu. However, the Assign Macro dialog box may not always appear when you click this command.

CAUSE

The Assign Macro dialog box may not appear if all of the following conditions are true:

  • You have a Microsoft Excel version 4.0 macro that has been defined as a command macro.

    -and-

  • You insert a Microsoft Excel worksheet or chart sheet object into a document in one of the programs listed at the beginning of this article.

    -and-

  • You assign the version 4.0 macro to an object, such as a button, in the embedded Microsoft Excel worksheet or chart sheet.

    -and-

  • You then try to either assign another macro to the button or reassign the same macro to the button.

If all of the above conditions are met, the Assign Macro dialog box may not appear and it may appear that your computer has stopped responding. Note, however, that the Assign Macro dialog box is actually displayed behind other windows.

RESOLUTION

If you choose Assign Macro from the Tools menu, or if you use the right mouse button to click an object and choose Assign Macro from the shortcut menu, if the Assign Macro dialog box does not appear, you can do any one of the following to make the dialog box visible:

  • Click the container program's title bar.

    -or-

  • Click any of the menus.

    -or-

  • Click any of the buttons that size either the program window or the document window.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
