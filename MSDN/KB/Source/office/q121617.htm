OFF4: XLCONVMP.DLL Is Installed in the MSAPPS Directory

 Last reviewed: March 12, 1997
Article ID: Q121617
This information in this article applies to:
  • Microsoft Office for Windows, version 4.20

SUMMARY

When you install Microsoft Office version 4.2, the XLCONVMP.DLL file will be installed in the SHEETCNV directory under the MSAPPS directory. The FILELIST.TXT file, located on Disk 1 of the Microsoft Office program, incorrectly states that the XLCONVMP.DLL file will be installed to the EXCEL directory.

MORE INFORMATION

The XLCONVMP.DLL file is a shared file used by Microsoft Graph version 5.0. By default, all shared application files are installed to the MSAPPS directory. The information in FILELIST.TXT is incorrect.

Note: The XLCONVMP.DLL file is installed to the Excel directory if Microsoft Excel is stand-alone (separate from the Microsoft Office package).
