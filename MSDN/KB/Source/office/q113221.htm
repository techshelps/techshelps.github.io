Microsoft Word 6.0 Should Be Installed Before MOM

 Last reviewed: February 25, 1998
Article ID: Q113221
The information in this article applies to:
  • Microsoft Office for Windows, version 4.0

SYMPTOMS

If Microsoft Word 6.0 is not already installed on your computer when you attempt to install Microsoft Office Manager (MOM) from the Microsoft Office 4.0 Upgrade package, you will receive the following error message: 

   Setup could not find Microsoft Word 6.0 installed on your system. Please
   install Microsoft Word 6.0 before installing this application.
NOTE: MOM is not a stand-alone application; it is only available to licensed Microsoft Office owners.

This problem occurs only with Microsoft Office version 4.0 when you are installing from the 1 Microsoft Office Disk and the 9 Microsoft Word disks.

CAUSE

Microsoft Office Manager (MOM) is designed to be installed on a computer on which Microsoft Word 6.0 is already installed. This setup order is enforced to ensure that the clip art index and thumbnail files are installed properly for Microsoft Word's clip art to work with the Microsoft Office ClipArt Gallery.

The above error message may appear if you are using a word processor other than Microsoft Word 6.0.

NOTE: If Microsoft Word 6.0 is not on your system, MOM will not install the ClipArt Gallery. Re-installing MOM after you install word will install the ClipArt Gallery successfully.

WORKAROUND

To work around this problem, follow the appropriate procedure below.

If you plan to use Word 6.0

To correct the setup issue with MOM, install Word 6.0 before you attempt to install MOM.

If you are using a word processor other than Word 6.0, and you do not want Word installed on your computer

To use MOM when you are using a word processor other than Word 6.0 (such as WordPerfect), do the following to add a "dummy" WINWORD6.INI file to your Windows system directory:

  1. In a text editor, such as Windows Notepad, type the following 

          [Microsoft Word]
      Picture-Path=<drive>:<path>

    where <drive> and <path> refer to the drive and directory where clip art has been installed (this path will most likely be your Windows directory). For example, if your Windows directory is called windows and is on your C drive, type: 

          PICTURE-PATH=C:\WINDOWS

  2. Save this file in your Windows directory and name it "WINWORD6.INI" (without the quotation marks).

NOTE: If you use this workaround and then later you decide to install Word 6.0, you will need to re-install MOM after you install Word 6.0.

MORE INFORMATION

MOM specifically looks for GOLF.WMF to determine whether the clip art has been installed correctly. If the GOLF.WMF file is not in the directory specified in the Picture-Path statement in your WINWORD6.INI file, the ClipArt Gallery will not be installed and you will receive the error message stating that you need to install Word 6.0.
