OFF95: Office Icons on Start Menu Are Larger Than System Icons

 Last reviewed: March 27, 1997
Article ID: Q140545
7.00 WINDOWS kbui

The information in this article applies to:

  • Microsoft Office Standard for Windows 95, version 7.0
  • Microsoft Office Professional for Windows 95, version 7.0

SYMPTOMS

The icons added to the Start menu by Microsoft Office are larger than the existing icons.

CAUSE

When you install Microsoft Office, the New Office Document and Open Office Document icons are added to the top of the Start menu. These icons appear larger than the existing Start menu icons. This behavior is by design and will occur when you display large icons on the Start menu.

RESOLUTION

If you want all of the icons on the Start menu to be the same size, show small icons on the Start menu. When you show small icons on the Start menu, all icons on the Start menu are small (smaller than their default size). To change your Start menu to show small icons, do the following:

  1. Use the right mouse button to click the Windows 95 taskbar, and click Properties on the shortcut menu.

  2. Select the Taskbar Options tab, and select the Show Small Icons In Start Menu check box.

  3. Click OK.

All icons on the Start menu will now be the same size.
