OFF97: OSB Getting Results Book Not Available on NT 3.51

 Last reviewed: March 12, 1998
Article ID: Q157205

The information in this article applies to:

  • Microsoft Office 97 for Windows
  • Microsoft Windows NT, version 3.51

SYMPTOMS

The Getting Results Book shortcut on the Office Shortcut Bar is not available.

CAUSE

This problem occurs when you install Microsoft Office 97 on a computer that is running Microsoft Windows NT 3.51 and Service Pack 5.

WORKAROUND

You can access the Getting Results Book on the Microsoft Web site. To view to this book on the Internet, link to: 

   http://www.microsoft.com/OfficeReference/GettingResults/
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

  http://www.microsoft.com/

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you click the Getting Results Book shortcut on the Office Shortcut Bar, Office 97 documentation in Hypertext Markup Language (HTML) format appears. This documentation assists you with upgrade and transition issues, as well as usage and interoperability issues between the Microsoft Office 97 programs.
