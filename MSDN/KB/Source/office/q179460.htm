OFF98: Search Page Error When You Click Search the Web Tool

 Last reviewed: February 2, 1998
Article ID: Q179460
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Word 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition

SYMPTOMS

When you click the Search The Web tool on the Web toolbar in any of the programs listed at the beginning of this article, you may receive the following error message: 

   You do not have a World Wide Web search page registered on your
   computer.

CAUSE

This problem occurs if either of the following conditions is true:

  • A Web browser is not installed on your computer.

    -or-

  • In the Configuration Manager, you have cleared the Search Page address box, either manually or by clicking the Use None button.

RESOLUTION

To resolve this problem, install a Web browser on your computer before you use the Search The Web tool in the programs included in Microsoft Office 98 Macintosh Edition. To install a Web browser, do either of the following:

  • The Microsoft Office 98 Macintosh Edition compact disc includes Microsoft Internet Explorer, version 4.0, for the Macintosh. To install it, double-click the Internet Explorer 4.0 Installer icon in the Microsoft Internet folder.
  • The Mac OS 8.0 compact disc includes two different Web browsers that you can install on your computer: Microsoft Internet Explorer and Netscape Navigator.

After you have installed a Web browser, reset the search page for your computer. To do this, follow these steps:

  1. On the Apple menu, point to Control Panels, and click Configuration Manager.

  2. Under Web Browser (on the left side of the dialog box), click Home And Search.

  3. Under Search Page, click Use Default. Or, enter a valid Uniform Resource Locator (URL) in the Address box.

  4. Click OK.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
