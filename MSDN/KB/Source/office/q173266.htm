OFF97: Setup Wizard "Creating a List of Duplicate Files ..."

 Last reviewed: February 23, 1998
Article ID: Q173266
The information in this article applies to:
  • Microsoft Office 97 for Windows

IMPORTANT: This article contains information about editing the registry. Before you edit the registry, you should first make a backup copy of the registry files (System.dat and User.dat). Both are hidden files in the Windows folder.

SUMMARY

When you run the Microsoft Office Setup program you may see a dialog box at the beginning of setup that contains the following message: 

   Creating the list of previous Office files.
You may also see a dialog box at the end of the Office Setup program with the following message: 

   Creating a list of duplicate files that can be removed.
These two messages are misleading; there are not any files created that allow you to view these lists.

MORE INFORMATION

If you want the Office Upgrade Wizard to create viewable files that contain the aforementioned lists, you can make an entry in the Windows registry before running the Office Setup program.

WARNING: Using Registry Editor incorrectly can cause serious problems that may require you to reinstall Windows 95. Microsoft cannot guarantee that problems resulting from the incorrect use of Registry Editor can be solved. Use Registry Editor at your own risk.

For information about how to edit the registry, view the Changing Keys And Values online Help topic in Registry Editor (Regedit.exe). Note that you should make a backup copy of the registry files (System.dat and User.dat) before you edit the registry.

To have the Office Setup program create the list of files flagged and removed by the Upgrade Wizard, do the following:

  1. On the Start menu click Run. Type Regedit in the Open box and then click OK.

  2. Double-click the key for HKEY_LOCAL_MACHINE.

  3. Double-click the key for SOFTWARE.

  4. Double-click the key for Microsoft.

  5. With the Microsoft key selected, on the Edit menu, point to New and click Key.

  6. Rename the key to "Office Upgrade Wizard" (without quotation marks.)

  7. With the Office Upgrade Wizard key selected, on the Edit menu point to New and click Key.

  8. Rename the key to "Generate Log Files", (without quotation marks.)

    NOTE: Do not set any value for this key, just the existance of the key causes the setup program to generate the log files.

  9. On the Registry menu, click Exit.

  10. Run the Office Setup program.

The log files, Pcpass1.txt and Pcpass2.txt, will be created in your \Windows folder.
