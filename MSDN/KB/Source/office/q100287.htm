Office: Installing PowerPoint Using CD-ROM Version 3.0

 Last reviewed: November 17, 1997
Article ID: Q100287
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 3.0
  • Microsoft PowerPoint for the Macintosh, version 3.0

SUMMARY

Microsoft PowerPoint version 3.0, which is included in the Office CD-ROM version 3.0, does not have a PowerPoint Installer Program.

To install PowerPoint version 3.0 from the Office CD:

  1. Insert the Office CD-ROM into the CD-ROM drive.

  2. Double-click the Microsoft Office CD-ROM icon.

  3. Open the following folders:

    Products:Microsoft PowerPoint 3.0:Install

  4. Drag the "PowerPoint 3.0" folder from the CD-ROM to your hard drive.

  5. Open the "Fonts-Drag Contents onto System" folder.

  6. Drag all of the items from the "Fonts-Drag Contents onto System" folder to the "System Folder" on your hard drive.

  7. Open the "System Folder" on your hard drive.

  8. Open the "Drag Contents into Extensions" folder on the CD-ROM.

  9. Drag all the items into the Extensions folder (this folder is located in the "System Folder" on your hard drive).

  10. From the Special menu, choose Restart.

MORE INFORMATION

To create disks that you can use to install PowerPoint, do the following:

  1. Open the "Compressed Disk Images" folder.

  2. If you are using 1.4 megabyte (MB) disks, open the "1.4MB" folder. If you do not have 1.4 MB disks, open the "800K" folder.

  3. Copy the contents of each folder to one disk. Name each disk with the same name as the folder you copy to it.

  4. Place the Install disk in your disk drive.

  5. Follow the installation directions found in the "Microsoft PowerPoint Handbook."
