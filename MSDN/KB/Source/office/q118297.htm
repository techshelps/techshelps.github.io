OFF: Black/White OutPut Printing to Color DeskWriter Printers

 Last reviewed: February 5, 1998
Article ID: Q118297
The information in this article applies to:
  • Microsoft PowerPoint 98 Macintosh Edition
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b
  • Microsoft PowerPoint for the Macintosh, version 3.0, 4.0
  • Microsoft Creative Writer for the Macintosh, version 1.0
  • Microsoft Fine Artist for the Macintosh, version 1.0
  • Microsoft Works for the Macintosh, version 3.0, 4.0

SYMPTOMS

When printing from any Microsoft application on the Macintosh to a Hewlett-Packard (HP) DeskWriter color printer, output may be black and white instead of color. No error message is displayed.

RESOLUTION

Allocate more memory to the application by following these steps:

  1. Quit the application.

  2. From the Apple menu, choose About This Macintosh.

  3. Make a note of how much Total Memory exists and the Largest Unused Block. If there are other applications shown in this dialog box, exit them to get an accurate reflection of the available memory.

  4. Click the application icon to highlight it. (Do not double-click the icon to run the application; if the application is running, the following steps will not work.)

  5. From the File menu, choose Get Info.

  6. In the Current (or Preferred) Size dialog box, type a larger memory amount (up to the size of the Largest Unused Block, from Step 3 above).

  7. Click the upper-left corner of the Get Info dialog box to close it.

  8. Run the application and print the document.

The document should now print in color.

MORE INFORMATION

If the Macintosh does not have enough memory available to allocate more to the application, you may need to use Virtual Memory. See the "Macintosh User's Guide" for information about Virtual Memory.
