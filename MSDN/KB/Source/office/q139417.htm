OFF95: Long Delay When You Select Custom Installation Option

 Last reviewed: March 27, 1997
Article ID: Q139417
7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

When you install Microsoft Office for Windows 95 and you choose the Custom installation option, your computer may take a very long time to display the next Setup screen.

CAUSE

Setup is building a list of installation options.

MORE INFORMATION

When you choose Custom installation, the Microsoft Office Setup program reads information from the setup information file (Setup.stf) to build a list of options to install. This process may take a long time depending on several factors:

  • The media from which Setup was run (for example, a local or shared CD- ROM, floppy disk, or network server).
  • The processor speed and data transfer rate of the computer or workstation to which Setup is being installed.
  • The processor speed and data transfer rate of the network server from which setup is being run.

The delay you experience depends on your system configuration. Factors include what hardware and software you are running, how much free memory is available, and the processor speed of your system. For example, you might see a delay of one minute if you are using a 90 megahertz (MHz) Pentium computer with 32 megabytes (MB) of memory. Slower processor speed and less memory will lengthen the delay.
