OFF95: Gradient Fill Doesn't Cover Entire Length of Toolbar

 Last reviewed: March 27, 1997
Article ID: Q131563
7.00 WINDOWS kbtool

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SYMPTOMS

When you customize the color of a toolbar on the Microsoft Office Shortcut Bar to use the Gradient Fill option, the gradient fill color extends only a portion of the length of the toolbar instead of extending the entire length of the toolbar.

CAUSE

This behavior is by design of Microsoft Office. The gradient fill color option has a fixed length on a toolbar, approximately the length of the toolbar name, instead of extending the entire length of a toolbar. The rest of the toolbar that does not display the gradient fill color is displayed with the color you have set for the Windows "3D Objects" item. Conversely, if your toolbar has a very long name, the toolbar may be displayed as a single color, without any gradient.

Note that when you clear the Use Gradient Fill check box for a toolbar in the Office Customize dialog box, the color extends the entire length of the toolbar.

When you select the Use Gradient Fill check box in the Customize dialog box, the Smooth check box becomes available. When you select this check box, the gradient coloring on the toolbar appears smoother. However, the selected color still extends approximately the length of the toolbar name.

To modify the Office Shortcut toolbar color

To change the color or color option (Gradient Fill or Smooth) of a toolbar on the Office Shortcut Bar, use the following steps:

  1. On the Office Shortcut bar, click the right mouse button on the Shortcut Bar.

  2. Click Customize.

  3. Click the View tab.

  4. In the Toolbar list, click the toolbar that you want to customize.

  5. Under Color, click the Change Color button. Click the color you want to be displayed on the toolbar, and click OK.

  6. Select or clear the Use Gradient Fill check box. If you select the Use Gradient Fill check box, you can also select the Smooth check box if you want a smoother look to the color. Click OK.
