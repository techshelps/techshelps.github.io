OFF97: Clicking "Remove All" In Setup Leaves Some Files

 Last reviewed: March 12, 1998
Article ID: Q160044
The information in this article applies to:
  • Microsoft Office 97 for Windows

SUMMARY

When you click Remove All in the Microsoft Office 97 for Windows Setup program, Setup does not remove all files.

MORE INFORMATION

Setup leaves certain files because they are shared by other programs. To completely remove Microsoft Office, you must remove all of the files and folders listed below. However, because some of the files or folders may be shared you may need to reinstall other programs. This article lists those folders and files that remain.

The following folders are not removed: 

 Program Files\Common Files\Microsoft Shared\VBA
 Program Files\Microsoft Office\
 Program Files\Microsoft Office\Office
 Program Files\Microsoft Office\Office\Setup
 Program Files\Microsoft Office\Office\Shortcut Bar
 Program Files\Microsoft Office\Templates
