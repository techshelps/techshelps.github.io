OFF97: Getting Results Book Opens in Internet Browser

 Last reviewed: March 20, 1997
Article ID: Q165379
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you click Getting Results Book on the Microsoft Office Shortcut Bar, the file Results97.htm opens in your Internet browser. This file is located in the following folder 

   <drive>:\Cdonline\Result97.htm
where <drive> is the drive letter of your CD-ROM drive.

CAUSE

The online Getting Results Book is not stored on the Microsoft Office 97 compact disc as it is in earlier versions of Microsoft Office. The online Getting Results Book is located on the Microsoft World Wide Web. To use the online Getting Results Book, you must have access to the Internet. To use this book, click Go in Result97.htm in your Internet browser. This links you to the following site on the World Wide Web: 

   http://WWW.Microsoft.com/OfficeReference/GettingResults/
NOTE: Because the Microsoft Web site is constantly updated, the site address may change without notice. If this occurs, link to the Microsoft home page at the following address: 

   http://www.microsoft.com/
For additional information, please see the following article in the Microsoft Knowledge Base: 

   Article-ID: Q161472
   TITLE     : OFF97: No Graphics on "Getting Results" Web Page
