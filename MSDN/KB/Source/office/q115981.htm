Wish FAX Template WISHFAX.DOT Does Not Exist

 Last reviewed: March 10, 1997
Article ID: Q115981
4.20 4.30 WINDOWS kbusage kbprb kbdoccerr docerr

The information in this article applies to:

  • Microsoft Office for Windows, versions 4.2, 4.3

SUMMARY

In the Microsoft Office Help file, available when you choose Help from the Microsoft Office button on the Microsoft Office Manager (MOM) toolbar, the reference to the Wish FAX template is incorrect. A Wish FAX template does not ship with Microsoft Office.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article.

MORE INFORMATION

You can view this incorrect information in the Help file by doing the following:

  1. From the MOM toolbar, choose the Microsoft Office button, and choose Help.

  2. In Microsoft Office Help, choose the Search button.

  3. Type "wish" (without the quotation marks), and from the list of keywords, select either Wish template or wish, communicating to Microsoft. Choose the Show Topics button.

  4. From the topic list, select Faxing Ideas To Microsoft, and choose Go To.

The following information appears, along with steps to use the Wish template: 

   Faxing Ideas to Microsoft
You can use the Microsoft Wish FAX template in Word to let Microsoft know about an idea you have for a new application feature or improvement for the next version of the application. The Microsoft Wish FAX template is a fax cover sheet where you can enter information about your product wish.

This information is incorrect, the Wish FAX template does not exist.
