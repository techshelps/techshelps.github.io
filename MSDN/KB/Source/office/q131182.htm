OFF: Buttons Supplied with the Office 7.0 and 97 Toolbars

 Last reviewed: January 8, 1998
Article ID: Q131182
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, version 7.0

SUMMARY

The Office toolbar is the only visible toolbar on the Office Shortcut Bar when you start Office the first time. Office supplies several other toolbars, which are hidden by default.

For more information about the other toolbars supplied with the Office Shortcut Bar, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q130849
   TITLE     : OFF7: Toolbars Supplied with the Office Shortcut Bar
The following table contains a list of the buttons on the Office toolbar and a description of each. 

 Office 97 Button Name    Description
 ---------------------    -----------

 New Office Document*     Opens the New Office Document dialog box

 Open Office Document*    Opens the Open Office Document dialog box

 New Message*             Opens a new message in Microsoft Outlook

 New Task*                Opens a new Task in Microsoft Outlook

 New Appointment*         Opens a new Appointment in Microsoft Outlook

 New Contact*             Opens a new Contact in Microsoft Outlook

 Microsoft Word           Runs Word

 Microsoft Excel          Runs Microsoft Excel

 Microsoft PowerPoint     Runs PowerPoint

 Microsoft Access         Runs Microsoft Access

 Microsoft Outlook        Runs Microsoft Outlook

 Microsoft Photo Editor   Runs Microsoft Photo Editor

 Microsoft Binder         Runs Microsoft Binder

 Office 7.0 Button Name   Description
 ----------------------   -----------

 Start A New Document*    Opens the Start A New Document dialog box

 Open A Document*         Opens the Open A Document dialog box

 Send A Message*          Opens a new message in Microsoft Exchange

 Add A Task*              Opens the New Task dialog box in Schedule+

 Make An Appointment*     Opens the Appointment dialog box in Schedule+

 Add A Contact*           Opens the Contact List in Schedule+

 Microsoft Word           Runs Word

 Microsoft Excel          Runs Microsoft Excel

 Microsoft PowerPoint     Runs PowerPoint

 Microsoft Access         Runs Microsoft Access

 Microsoft Schedule+      Runs Schedule+

 Screen Saver             Starts your selected screen saver

 MS-DOS Prompt            Opens an MS-DOS window

 Online Documentation     Opens the Office online document files
                          (available only with CD-ROM versions of Office)

 Microsoft Binder         Runs Microsoft Binder

 Answer Wizard*           Runs Answer Wizard

 * Office displays this button by default. Other buttons are hidden
   by default.
