OFF: Hang Using Send/Add Routing Slip Command

 Last reviewed: December 8, 1997
Article ID: Q135468
The information in this article applies to:
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel 97 for Windows
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Mail for Windows, versions 3.0, 3.0b, 3.2

SYMPTOMS

In Microsoft Excel or Microsoft PowerPoint, when you use the Send command or the Add Routing Slip command (File menu), your application may stop responding (hang).

CAUSE

This problem occurs when you are using Microsoft Mail under Windows 95. The Microsoft Office version 7.0 programs are designed to work with a 32- bit MAPI client (such as Microsoft Exchange) or a 16-bit VIM application (such as Lotus cc:Mail and Lotus Notes Mail) on a computer running Windows 95. The Office version 7.0 programs do not work with Microsoft Mail running under Windows 95.

If you are using Microsoft Mail under Windows 95, the Send and Add Routing Slip commands are available in Microsoft Excel and Microsoft PowerPoint. However, if you use either of these commands to send your document as a mail message, the application hangs and your document is not sent.

This problem does not occur in Word for Windows 95 version 7.0 or later; the Send and Add Routing Slip commands are not available in Word if you are using Microsoft Mail under Windows 95.

WORKAROUNDS

To close Microsoft PowerPoint or Microsoft Excel if they have stopped responding, press CTRL+ALT+DELETE, click the program name, and click End Task.

To avoid hanging your program, install one of the mail applications that is supported by the Office version 7.0 or 97 programs. The following mail applications are supported by the Office version 7.0 and 97 programs: 

   Microsoft Exchange
   Lotus cc:Mail
   Lotus Notes Mail (please see the NOTE that follows)
   Microsoft Mail for Windows NT (under Windows NT only)
NOTE: If you are using Lotus Notes Mail, you may receive an error message when trying to send mail from the Office programs. In this case, remove Microsoft Office version 7.0, and then install Microsoft Office version 7.0b or Microsoft Office 97.

To send your Microsoft Office document to another person in a Microsoft Mail message, you can run your Microsoft Mail application and send the saved document as an attachment.
