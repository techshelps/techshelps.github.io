OFF97: Setup Errors Appear Inserting Disks 22 or 27

 Last reviewed: April 30, 1997
Article ID: Q167717
The information in this article applies to:
  • Microsoft Windows 95
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Access 97
  • Microsoft PowerPoint 97 for Windows
  • Microsoft Word 97 for Windows
  • Microsoft Outlook 97

SYMPTOMS

When you install Microsoft Office 97 from disks, one the following error messages appears: 

   Microsoft Office 97 Standard Edition
   ------------------------------------

      Setup was unable to locate the file 'Office\Forms\SCDCNCLL.ICO'
      within the compressed source file 'MSO97_22.CAB'

   Microsoft Office 97 Professional Edition
   ----------------------------------------

      Setup was unable to locate the file 'Office\Forms\SCDCNCLL.ICO'
      within the compressed source file 'MSO97_27.CAB'

CAUSE

This problem occurs when either of the following conditions are true:

  • In the English (United States) version of Microsoft Windows 95, regional settings for the computer are set to Spanish (Mexican) or Spanish (Traditional Sort).

    -or-

  • In the Spanish version of Windows 95, regional settings for the computer are set to Espanol (Mexico).

WORKAROUND

Depending on the language version of Windows 95 you are using, use one of the following methods to work around this problem.

Method 1: English Version of Windows 95

In the Windows 95 Control Panel, change the regional settings to English (United States), and then run the Microsoft Office 97 Setup program.

Method 2: Spanish Version of Windows 95

In the Windows 95 Control Panel, change the regional settings to Ingles (Estados Unidos), and then run the Microsoft Office 97 Setup program.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The regional settings in the Windows 95 Control Panel affect the way programs display and sort dates, times, currency, and numbers.
