Off97: Visual Basic Help File for Microsoft Binder Available

 Last reviewed: July 15, 1997
Article ID: Q162672
The information in this article applies to:
  • Microsoft Office Binder 97
  • Microsoft Office 97 for Windows

The Visual Basic for Applications help file for Microsoft Binder is available for downloading. This help file is the same file included in the ValuPack folder on the Microsoft Office 97 compact disc.

The following file is available for download from the Microsoft Software Library: 

 ~ Vbabdr8.exe (size: 63941 bytes) 
                          63K self-extracting file
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
