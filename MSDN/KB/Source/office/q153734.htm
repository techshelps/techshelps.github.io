Read Me Doc Error in The Microsoft Empowerment Pack

 Last reviewed: November 17, 1997
Article ID: Q153734
The information in this article applies to:
  • The Microsoft Empowerment Pack for the Mac, version 1.0
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b

SUMMARY

In the file "Read Me -- Important!", contained on The Microsoft Empowerment Pack for the Mac compact disc, the paragraph titled "Installing Updates for Microsoft Word" incorrectly states you should make sure you have at least 2700 bytes of RAM free before running either of the Word updates. Instead of 2700 bytes, it should read 2700 kilobytes (KB), or roughly 2.7 megabytes (MB).

MORE INFORMATION

The following is the actual paragraph taken from the "Read Me - Important!" file on The Microsoft Empowerment Pack for the Mac compact disc: 

   Installing Updates for Microsoft Word
   -------------------------------------

   In order to install the Internet Assistant for Microsoft Word or the
   Word 6.0.1a Update from within the guided tour, you will need at least
   2700 bytes of additional RAM free. Because the System typically takes
   at least two megabytes of RAM, users with 8 Mb systems will need to
   enable Virtual Memory from the Memory Control Panel in order to install
   these programs from within the guided tour (see the Virtual Memory Tips
   in the "Tips & Tricks" folder of this CD for more information about
   using virtual memory on your Macintosh). You can also launch the setup
   applications for these programs directly from the CD without going
   through the guided tour.
Below are brief descriptions of the two Microsoft Word updates included with the Empowerment Pack compact disc:
  • Internet Assistant 2.0 for Microsoft Word for the Macintosh.

    Working from within Word, Internet Assistant 2.0 for Word is a full-featured authoring tool that automatically converts Word files to HTML and provides users with an interface for inserting hyperlinks, images, and forms into their documents.

  • Microsoft Word 6.0.1a Update for the Macintosh.

    The update enhances performance and stability with faster startup time and word count, improved font handling, and resolved extension conflicts.
