Location of Office 4.3 Files Installed on Server

 Last reviewed: September 16, 1996
Article ID: Q117479
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3

SUMMARY

The following is a list of the files and directories that are installed on the network server when you run an administrative installation (Setup /a) of Microsoft Office version 4.3.

MORE INFORMATION

You can use this list to determine which files are installed by the administrative installation of Microsoft Office and where they are installed, so that you can remove them if necessary.

The names inside the brackets are subdirectory names. The files included in each subdirectory are not listed here, because you can remove the entire subdirectory if you want to remove the files installed during the administrative installation.
