Information on Troubleshooting Notes/FX Issues

 Last reviewed: December 13, 1996
Article ID: Q121399
The information in this article applies to:
  • Microsoft Office for Windows, version 4.2c
  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Office 97 for Windows
  • Microsoft Excel 97 for Windows
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft Excel for Windows, version 5.0c
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows, version 4.0c
  • Microsoft Word 97 for Windows
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Word for Windows, version 6.0c

SUMMARY

The Microsoft applications listed above support Lotus Notes Field Exchange (FX). The following information discusses how to troubleshoot problems using Lotus Notes/FX.

MORE INFORMATION

The Lotus Notes/FX support in the applications listed above allows you to use one of these applications to write to fields in a Lotus Notes databases.

The Lotus Notes/FX capability allows for some bidirectional exchange, but is limited to certain fields in certain applications, such as moving author, title, and subject matter of a Microsoft Word memo into a Notes document.

Check Help File

Use the specific steps given in the individual program's Help file to troubleshoot the problem first.

For more information about Using Microsoft Excel, Microsoft Word, or Microsoft PowerPoint With Lotus Notes Field Exchange, click the Search button in Help in the specific application and type: 

   Lotus Notes

Use Different Application, File, Database

  1. Try the same basic steps you are attempting with another Microsoft Office application. For example, if you are attempting to embed a Microsoft Excel object in Lotus Notes, try embedding a Microsoft Excel object in Microsoft Word.

  2. Try the same basic steps using a new document. For example, if you are attempting to use a defined name in a Microsoft Excel worksheet as a Lotus Notes field, try the same steps using another worksheet.

  3. Try the same basic steps using a new Lotus Notes form.

System Specific Settings to Check

  1. Make sure that the proper settings are present in the WIN.INI file. Specifically, make sure that the path information in the [embedding] section is correct.

  2. Make sure that the information in the REG.DAT file is correct and not damaged.

For additional information about the REG.DAT file, please see the following article(s) in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q80393
   TITLE     : How to Rebuild the Default Windows REG.DAT File

  • Make sure that the Microsoft Office applications that you are using are the correct versions, that is, Microsoft PowerPoint version 4.0c, Microsoft Excel version 5.0c, and Microsoft Word version 6.0c.
