Office 4.2x Update for Power Mac Extension Not on Floppy Disks

 Last reviewed: November 17, 1997
Article ID: Q147219
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1b

SUMMARY

The Office 4.2x Update for Power Mac extension does not ship with Microsoft Office versions 4.2, 4.2.1, or 4.2.1b on floppy disk.

MORE INFORMATION

The Office 4.2x Update for Power Mac extension is for use with Office 4.2 and 4.2.1 for the Power Macintosh, with Word 6.0 and 6.0.1 for the Power Macintosh, and with Microsoft Excel 5.0 and 5.0a for the Power Macintosh. Installing and enabling this extension will correct the following two problems:

  • Starting an application after Word causes a System Error 11 message and your computer stops responding (hangs).

    -and-

  • Printing with QuickDraw GX 1.1.1 or 1.1.2 causes a System Error 11 message and your computer hangs.

If you purchased Microsoft Office on floppy disk, you can obtain the Office 4.2x Update for Power Mac extension, as AppNote MC1164, from the following sources:
  • Microsoft's World Wide Web Site on the Internet
  • The Internet (Microsoft anonymous ftp server)
  • The Microsoft Network (MSN)
  • Microsoft Download Service (MSDL)
  • Microsoft Product Support Services

Note that this extension also ships with Microsoft Office for the Macintosh, versions 4.2.1a and 4.2.1b on compact disc, but you want to make sure you have version 1.0.1 of the extension. The following table lists the version numbers for the extension according to the source of the extension. 

   Source                Extension Version
   ------------------------------------------

   Office 4.2.1a CD      1.0
   Office 4.2.1b CD      1.0.1
   AppNote MC1164        1.0.1
