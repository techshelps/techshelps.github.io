Office Find File Dialog Box Renders Font in Bold Type

 Last reviewed: September 16, 1996
Article ID: Q121082
The information in this article applies to:
  • Microsoft Office Manager, version 4.2

SYMPTOMS

In Microsoft Office Manager (MOM) version 4.2, when you use the Find File feature (from the MOM menu, choose Find File), the font used in the dialog box is not displayed as expected: the text in the dialog box may appear bold or distorted.

This behavior independent of video resolution.

STATUS

Microsoft has confirmed this to be a problem with the products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you use Find File in Microsoft Word 6.0 or Microsoft Excel 5.0, the font in the dialog box appears as expected: the font appears similar to the system font displayed in the menus. However, when you run Find File from MOM, the font in the dialog box appears to be bold or is blurred and rough looking.
