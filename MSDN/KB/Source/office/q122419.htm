OFFICE: Description and Contents of Version 4.30c-CD

 Last reviewed: September 16, 1996
Article ID: Q122419
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD

SUMMARY

The following information is about Microsoft Office 4.3c-CD; it contains a description of the application and what it contains.

MORE INFORMATION

Microsoft Office version 4.3c-CD is contained on two compact discs. The discs contain the following:

Microsoft Word (version 6.0c)

A full-featured word processor that includes automatic formatting, easy-to- create tables, and the ability to create and position graphics.

Microsoft Excel (version 5.0c)

A comprehensive spreadsheet application that lets you manage, format, chart, and analyze data.

Microsoft PowerPoint (version 4.0c)

A complete presentation graphics package that creates fully formatted slides and presentations.

Microsoft Access (version 2.0)

A relational database management system that lets you store, organize, and easily retrieve information.

Microsoft Office Manager (version 4.3)

A control center for the Office applications. The Office Manager toolbar lets you quickly start and switch between Office applications.

Microsoft Office Assistant (version 1.0)

Fifty-four business tools that work seamlessly with Office Professional. Direct marketing analysis forms, financial statements, five-year projection and business planning templates, purchase orders, and more. The Microsoft Office Assistant is located in the BONUS\OA directory on the CD.

Bookshelf (94)

With the click of a button, you can open seven reference books. Whenever you start Office Professional, the QuickShelf feature of Bookshelf starts automatically. QuickShelf is a toolbar that appears in the application windows and includes buttons for the books in Bookshelf.

Online Documentation

For access to information about the applications, use the index to the online Microsoft Office Professional User's Guide. You can access online documentation from Microsoft Office Manager after installation.

Office Compatible Demos

Included on the second CD, in the DEMOS.OC directory, are demos of several Windows applications by Microsoft and other software companies that you can use easily from within Office Professional.

For additional information, please see the following article(s) in the Microsoft Knowledge Base:

Bonus Software

In addition, several other products are included as a bonus: CapScreen, software that can record visually what is on your computer screen; Vid Edit, a simple tool for editing digital video clips; the Microsoft Word Supplemental File Conversion Kit and ODBC drivers, all of which can be installed from the BONUS directory along with Microsoft Office Assistant.
