OFF97: Presentations Don't Fill Entire Screen in Binder

 Last reviewed: March 12, 1998
Article ID: Q160756
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft PowerPoint 97 for Windows

SYMPTOMS

When you display a PowerPoint presentation using the Slide Show in Microsoft Binder, the slide occupies the space in the Binder window, but does not occupy the full screen. The menu bar and the program title screen are visible.

CAUSE

This behavior occurs because the controlling program (Microsoft Binder in this case) has control over programs that run within it. PowerPoint uses all of the screen space that Binder allocates to it.

WORKAROUND

To work around this behavior, use either of the following workarounds.

Method 1: Switch to full screen mode after you start the presentation. To 

          do this, use the following steps:

          a. In Microsoft Binder, click View Show on the Slide Show menu.

          b. After Binder starts the presentation, right-click the first
             slide, and then click Full Screen (or click Full Screen on the
             Browse menu). This step displays the presentation using the
             full screen. To end the slide show, press the right mouse
             button and click End Show.
Method 2: View the presentation outside of Microsoft Binder. To do this, 
          use the following steps:

          a. Click View Outside on the Section menu. This step opens the
             presentation in PowerPoint.

          b. In PowerPoint, click View Show on the Slide Show menu. This
             step displays the presentation using the full screen.

STATUS

This behavior is by design of Microsoft Office Binder 97.

MORE INFORMATION

Programs in Microsoft Binder that use the full screen (for example, when you click Full Screen) affect only the active window.

REFERENCES

For more information about displaying presentations in Microsoft Binder, click the Index tab in Microsoft PowerPoint Help, type the following text 

   Powerpoint, sharing data with other programs
and then double-click the selected text to go to the "Ways to share information between PowerPoint and another Office program" topic.
