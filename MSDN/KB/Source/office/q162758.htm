OFF97: Files Installed with Run From Network Server

 Last reviewed: January 31, 1997
Article ID: Q162758
The information in this article applies to:
  • Microsoft Office 97 for Windows

SUMMARY

The following is a list of the folders and files installed when you perform a Run From Network Server installation of Microsoft Office 97 for Windows (with the shared components installed on the network server).

MORE INFORMATION

You can use this list to determine which files are installed by Microsoft Office and where they are installed, so that you can remove them if necessary.
