OFF95: Some Microsoft Excel Add-ins Do Not Work in Binder

 Last reviewed: March 27, 1997
Article ID: Q138762
7.00 WINDOWS kbusage kbtool

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0

SYMPTOMS

In a Microsoft Excel worksheet section of a Microsoft Office Binder, some of the add-ins may not function properly or may not work at all. Even though the add-ins appear to be installed, some of their functionality may be missing while you are in an Office Binder.

WORKAROUND

If the add-in you are trying to use does not appear to be functioning as it does outside of Office Binder, view the Binder section outside of the Binder. In order to view your Binder section outside of the Binder, click View Outside on the Section menu.

To return to the Binder, click Close on the File menu.

MORE INFORMATION

The following is a list of the add-ins that may not have full functionality when you try to use them in a Microsoft Excel section of an Office Binder: 

   View Manager
   Report Manager
   Analysis ToolPak
   AutoSave
The following descriptions assume you are in a Microsoft Excel section of an Office Binder and that each add-in is already installed.

View Manager

If you click View Manager on the View menu, you may receive the error message: 

   Cannot create or edit views while workbook is an OLE object in another
   application or in an Office Binder.

Report Manager

If you click Report Manager on the View menu, you may receive the error message: 

   Cannot print or edit reports while workbook is an OLE object in another
   application or in an Office Binder.

Analysis ToolPak

Any of the functions that are only available when the Analysis ToolPak is installed will work properly and will show up in the Function Wizard dialog box. However, if you click Data Analysis on the Tools menu, the Data Analysis dialog box that lists all of the analysis tools is not displayed.

AutoSave

The AutoSave dialog box will appear when you click AutoSave on the Tools menu; however, if you have the Prompt Before Saving option selected, the dialog box that confirms whether or not you want to save your file may not appear. If the confirmation dialog box does appear, it may show up behind the Binder and will not be visible until you minimize the Binder.

REFERENCES

For more information about working with Sections in an Office Binder, click the Index tab in Microsoft Office Help, and type the following text: 

   Sections
