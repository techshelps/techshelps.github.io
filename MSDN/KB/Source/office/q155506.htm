ODBC Error Message: "Failure to Load Expression Service"

 Last reviewed: September 30, 1997
Article ID: Q155506
7.00 WINDOWS kbreadme kberrmsg

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0
  • Microsoft Excel for Windows 95, version 7.0
  • Microsoft PowerPoint for Windows 95, version 7.0
  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Windows NT, version 4.0

SYMPTOMS

If you install Microsoft Office Standard for Windows 95, version 7.0, onto a computer running Microsoft Windows NT 4.0, you may receive the following error message 

   Failure to load expression service
when using the Access ODBC driver from Office applications, such as Microsoft Excel or Word.

MORE INFORMATION

To correct this problem:

Method 1

You will need to obtain the Microsoft Office for Windows 95 version 7.0b release from Microsoft Product Support.

You can obtain a copy of this release by calling the Microsoft Order Desk at (800) 360-7561.

If you are outside of the United States, contact the Microsoft Subsidiary in your area. To locate your subsidiary, see the Microsoft World Wide Offices Web site at: 

   http://www.microsoft.com/worldwide/default.htm

Method 2

You will need to obtain the Microsoft ODBC Drivers for 32-bit programs. To obtain the ODBC drivers, download GE1263.EXE, a self-extracting file, from the Microsoft Software Library (MSL) on the following services:

- Microsoft's World Wide Web Site on the Internet 

      On the www.microsoft.com home page, click the Support icon.
      Click on Download a Driver, Patch or Sample File.
      Go to Find a Specific File, Enter GE1263.EXE, and click GO!
      Click on the file to start downloading.
- Internet (anonymous FTP) 
      ftp ftp.microsoft.com
      Change to the Softlib\Mslfiles directory
      Get GE1263.EXE (size: 2804116 bytes)
  • Microsoft Download Service (MSDL) 

          Dial (425) 936-6735 to connect to MSDL
      Download GE1263.EXE (size: 2804116 bytes)
