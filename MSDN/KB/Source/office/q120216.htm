Fonts Automatically Installed with Office Assistant

 Last reviewed: September 16, 1996
Article ID: Q120216
The information in this article applies to:
  • Microsoft Office Assistant for Windows, version 1.0

Office Assistant documents require certain fonts. The Office Assistant Setup program automatically installs the following three required fonts:

  • Abadi MT Condensed
  • Abadi MT Condensed Extra Bold
  • Abadi MT Condensed Light

According to the Office Assistant Help file, Arial and Time New Roman, which are included with Microsoft Windows version 3.1, are also required for Office Assistant.

REFERENCES

OAREADME.TXT readme file (dated 4/19/94), that shipped with Microsoft Office Assistant version 1.0.

For more information about troubleshooting font issues in Office Assistant, choose the Search button in the Microsoft Office Assistant and type: 

   troubleshooting
