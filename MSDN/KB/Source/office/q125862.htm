Office: Files Left After Remove All Version 4.2 (PPC)

 Last reviewed: November 17, 1997
Article ID: Q125862
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2

SUMMARY

When you run Microsoft Office Setup in maintenance mode, and you choose the Remove All button, some the files that Microsoft Office Setup installed will remain on your hard disk drive.

CAUSE

The Microsoft Office Setup leaves a file on your hard disk drive by design when you choose Remove All during a maintenance mode installation for one of the following reasons:

  • Setup did not install the file

    -or-

  • Setup did install the file; however, the same file was also installed by another application, and has not been removed by that application's Setup program.

Additionally, the Microsoft Office version 4.2 for the Power Macintosh Setup program may leave some files on your hard drive that do not fall into either of the above cases.
