Office 4.x: Accessing the Office Manager Menu With a Keyboard

 Last reviewed: September 16, 1996
Article ID: Q128871
The information in this article applies to:
  • Microsoft Office for Windows, versions 4.0, 4.2, 4.2c, 4.3, 4.3c

SUMMARY

You can access the Microsoft Office menu from the Microsoft Office Manager (MOM) toolbar by using the key combination ALT+O. This procedure only works when the Office Manager is activated. To activate Office Manager, use the following steps:

  1. To activate the Office Manager Application, do one of the following: 

        - Press and hold down the ALT key and then press TAB repeatedly
      until you see Office Manager.


        -or-


        - Press CTRL+ESC to display the Task List. Use the arrow keys to select
      Office Manager, then press ENTER.

  2. To display the Office Manager menu, hold down the ALT key, and press the letter "O" on your keyboard.

After you activate the Office Manager menu, you can use the arrow keys to select an application. Once the application is selected, press ENTER to start or switch to the application.

NOTE: You can only activate the Office Manager Menu with the keyboard as described above when the Office Manager toolbar is displaying small buttons.

MORE INFORMATION

If you must access this menu using the keyboard (for example, if you do not have a mouse or other pointing device) and the MOM toolbar is NOT displaying small buttons, you cannot change to small buttons because you cannot access the Microsoft Office menu. In this case, you can modify the MSOFFICE.INI file to display small buttons by doing the following:

  1. In Notepad or in another text editor, open the MSOFFICE.INI file (located in your Microsoft Windows directory).

  2. Locate the [Options] section, and modify the UseLargeBtn line to appear as follows: 

          UseLargeBtn=2

  3. Save the changes to MSOFFICE.INI, and close the file.

Note that you must restart the MOM toolbar after you make this change.

For more information about how to Customize Microsoft Office Manager, choose the Search button in Microsoft Office Help and type: 

   customize
