CD Setup Key Provided for 4.2.1b CD Not Used During Setup

 Last reviewed: November 17, 1997
Article ID: Q145649
The information in this article applies to:
  • Microsoft Office for the Macintosh, version 4.2.1b (compact disc)

SUMMARY

On the back of the jewelcase for the Microsoft Office 4.2.1b compact disc is a sticker that contains a CD Key 10-digit number. Also, just below the CD Key number on the printed material inside the jewelcase are instructions stating that you should not to lose the number because you will need it every time you run Setup. These instructions are incorrect: you will never be prompted for this key during the Setup of Microsoft Office for the Macintosh from the version 4.2.1b compact disc.

MORE INFORMATION

The 10-digit CD key number that is on the back of the Microsoft Office version 4.2.1b compact disc jewelcase is similar to the one that is on the jewelcase for Microsoft Office for Windows 95; however, on the Macintosh version you do not need to use this number during any part of Setup.

The CD key is unrelated to the product serial number for Microsoft Office for the Macintosh, version 4.2.1b.
