OFF97: Server Pack Setup Does not Complete

 Last reviewed: May 19, 1997
Article ID: Q168693
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Windows NT Server versions 3.51, 4.0

SYMPTOMS

While installing the Server Pack for Microsoft Office 97, after you enter your logon information in the "Office Server Pack Setup" dialog you encounter one of the following problems.

Microsoft Windows NT 4.0

The following error message appears: 

   An application error has occurred

   ACMSETUP.exe
   Exception: access violation (0xc0000005), Address: 0x12181

Microsoft Windows NT 3.51

The Setup program stops and never prompts you that it has completed or to reboot. If you try to run the Find Fast program in the Windows NT Control Panel, the 

   Please locate FindFast.EXE
dialog appears asking you to locate the Findfast.exe program. When you locate the Findfast.exe program it does not start.

When you run the Server Pack Setup program again it does not recognize you have already installed the program and begin the Maintenance Mode Setup as you would expect. Instead, a new installation of the Server Pack is started.

CAUSE

This problem will occur when you install the Microsoft Office 97 Server Pack to a folder whose path exceeds 113 characters.

WORKAROUND

Install the Server Pack to a folder whose path does not exceed 113 characters in length.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The Microsoft Officer Server Pack is located in the Srvpack folder on the Microsoft Office 97 CD-ROM. When you run the Setup program, a Find Fast service is installed in the Microsoft Windows NT Control Panel. This service creates indexes on Web servers, on network servers, or on your local hard disk. By default, Find Fast creates a single index on each hard disk on your computer.

For additional information about Web Find Fast, please see the Readme.txt located in the Srvpack folder on the Microsoft Office 97 CD-ROM.
