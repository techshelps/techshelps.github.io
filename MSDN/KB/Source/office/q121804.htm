Transition Question Dialog Appears Removing Word or PowerPoint

 Last reviewed: September 16, 1996
Article ID: Q121804
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD

SYMPTOMS

When you run the Microsoft Office Setup program in maintenance mode, and you choose to add or remove an item, you may receive the following message: 

   Microsoft Excel includes support for people familiar with Lotus 1-2-3.
   Would you like to read more about this support?

CAUSE

This behavior occurs if you installed Microsoft Office using the Typical installation option, and you then add or remove an item, such as the Microsoft PowerPoint application. This message appears by design when you install Microsoft Excel, and allows you to choose whether you want to use the slash (/) key to display Help For Lotus 1-2-3 users. However, this message appears incorrectly under the above conditions.

WORKAROUND

If you receive the above message when you add or remove an item in the Setup program, choose No in the dialog box that displays this message to allow the Setup program to continue.

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

When you receive the Transition Question dialog box that contains the message above, if you choose Yes, you receive the following message: 

   Microsoft Excel can open and save existing Lotus 1-2-3 worksheets and
   run the macros they contain. You can use the slash (/) key to display
   help on Microsoft Excel equivalents for Lotus 1-2-3 commands. Do you
   want to use the slash key to display help?
If you choose Yes, the Lotus 1-2-3 Help option is selected under Settings on the Transition tab of the Options dialog box in Microsoft Excel. To change this setting, choose Options from the Tools menu in Microsoft Excel, and under Settings, select either the Microsoft Excel Menus option, or the Lotus 1-2-3 Help option.
