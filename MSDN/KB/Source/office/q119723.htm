DXF Graphic Filter Does Not Ship with Office 4.0

 Last reviewed: September 30, 1997
Article ID: Q119723
The information in this article applies to:
  • Microsoft Office for Windows, version 4.0
  • Microsoft PowerPoint for Windows, version 3.0

SUMMARY

The AutoCAD DXF graphic import filter that ships with the standalone PowerPoint 3.0 package is not included with Microsoft Office 4.0. If you purchased PowerPoint as part of Microsoft Office 4.0, you will not have this filter.

The AutoCAD DXF filter is available on the Microsoft Word for Windows version 6.0 supplemental conversion disk. This filter is also available on the Microsoft Download Service (MSDL).

MORE INFORMATION

To obtain the Word 6.0 Supplemental File Conversions disk, do either of the following:

  • Send in your Supplemental Offers Coupon, included with your Word package.

    -or-

The following file(s) are available for download from the Microsoft Software Library: 

 ~ DXF.EXE (size: 159500 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
After you download DXF.EXE, run it to extract the files it contains and view the README.TXT file for installation instructions.

MSDL

Each of the supplemental text converters and graphic filters on the Word 6.0 supplemental conversion disk is also available by modem from the Microsoft Download Service (MSDL), which you can reach by calling (425) 936-6735. This service is available 24 hours a day, 7 days a week. The highest download speed available is 14,400 bits per second (bps). For more information about using the MSDL, call (800) 936-4100 and follow the prompts.

To obtain the supplemental converters and filters, download the compressed, self-extracting file for the text converter or graphic filter that you want. After you download the file(s) you want, run the executable file to extract the files it contains and view the README.TXT file for installation instructions.
