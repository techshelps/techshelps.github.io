OFF98: System and Disk Space Requirements for MS Office 98

 Last reviewed: February 3, 1998
Article ID: Q178010
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Excel 98 Macintosh Edition
  • Microsoft Word 98 Macintosh Edition
  • Microsoft PowerPoint 98 Macintosh Edition

SUMMARY

This article contains a list of the system and disk space requirements for Microsoft Office 98 Macintosh Edition and each of the programs that are included in Microsoft Office 98 Macintosh Edition.

MORE INFORMATION

System Requirements

To use Microsoft Office 98 Macintosh Edition, you need:

  • Any PowerPC processor-based, Mac OS-compatible system (clock speed of at least 120 MHz recommended).
  • Apple Macintosh operating system version 7.5 or later (System 7.5.5 or later recommended)
  • 16 megabytes (MB) of random access memory (RAM) to run programs individually; more memory is required to run programs simultaneously (32 MB recommended)
  • 49-120 MB of available hard-disk space; approximately 90 MB for an Easy Install, depending on your configuration
  • CD-ROM drive (or connection to a Local Area Network if you are installing Office over a network)
  • Monitor or display supporting 16 grays or 256 colors, with 640 x 400 or higher resolution

Additional items or services that are required to use certain features:
  • 9600 baud or higher modem (14,400 recommended)
  • Some Internet functionality may require Internet access and payment of a separate fee to an Internet service provider

Disk Space Requirements

Each program requires a certain amount of free disk space; this amount may vary, depending on the components you choose to install with the program.

  • To install Microsoft Office 98 Macintosh Edition, you need from 49-100 MB of free hard disk space: 

          Easy Install: 85 MB
      Complete Install: 100 MB
      Run From CD or Run From Server Install: 10 MB
  • To install the stand-alone version of Microsoft Excel 98 Macintosh Edition, you need 32-51 MB of free hard disk space (43 MB for an Easy Install)
  • To install the stand-alone version of Microsoft Word 98 Macintosh Edition, you need 33-64 MB of free hard disk space (56 MB for an Easy Install)
  • To install the stand-alone version of Microsoft PowerPoint 98 Macintosh Edition, you need 39-63 MB of free hard disk space (57 MB for an Easy Install)

Additional Requirements for Value Pack Components

To install the Value Pack components, you need:

  • Up to 160 MB to install optional clip art, templates, Help files, or other components from the Value Pack
  • Up to 10 MB to install Internet Explorer
