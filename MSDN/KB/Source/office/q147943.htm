Network.txt Documentation Error--Long Filenames on Novell

 Last reviewed: September 16, 1996
Article ID: Q147943
The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0

SUMMARY

The "Support for Long Filenames on Novell 3.11" section of the Network.txt file that comes with Office for Windows 95, version 7.0, states the following: 

   There is a patch available from Novell to correct this. It
   is in a file called os2ptch.nam which replaces the standard
   Novell 3.11 file os2.nam.
This information is incorrect. Instead, this section of the Network.txt file should state the following: 

   There is a patch available from Novell to correct this. It
   is in a file called Os2opnfx.nlm which replaces the standard
   Novell 3.11 file Os2.nam.
