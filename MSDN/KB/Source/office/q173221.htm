OFF97: Clip Gallery: Import Clips Button Missing from Dialog Box

 Last reviewed: February 25, 1998
Article ID: Q173221
The information in this article applies to:
  • Microsoft Office 97 for Windows

SYMPTOMS

When you attempt to insert a Microsoft Clip Gallery object in an application, the Import Clips button may be missing in the Microsoft Clip Gallery 3.0 dialog box.

CAUSE

With some video cards, the Import Clips button will be missing if the system Font Size has been set to Large Fonts or has been scaled beyond 100% of normal size.

NOTE: Actual percentages may vary.

RESOLUTION

To enable the Microsoft Clip Gallery dialog box to display the Import Clips button, change the system Font Size to Small Fonts. To do so, follow these steps:

  1. Click the Start button, point to Settings, and then click Control Panel.

  2. In Control Panel, double-click the Display icon.

  3. In the Display Properties dialog box, click the Settings tab.

  4. Near the bottom of the Settings tab, there is an option for Font Size. Click Small Fonts on the Font Size list. Click OK.

  5. When prompted to restart, click Yes.

NOTE: You may be able to use a larger font than Small Fonts. To see if you can do so on your system, follow these steps:

  1. Click the Start button, point to Settings, and then click Control Panel.

  2. In Control Panel, double-click the Display icon.

  3. In the Display Properties dialog box, click the Settings tab.

  4. Under Font Size, click Other. Or, if Other is not in the list, click the Customize button.

  5. In the Custom Font Size dialog box, for the Scale Fonts To Be option, type in a percentage between 100 and 125, which is the percentage for Large Fonts. You can drag the ruler to accomplish this.

    NOTE: The maximum percentage that behaves correctly may vary from system to system. You will need to use trial and error to find what percentage works for you.

  6. Click OK in the Custom Font Size dialog box, and then click OK in the Display Properties dialog box. You may be prompted that "This font size setting may cause some programs to appear incorrectly." Click OK. When prompted to restart Windows 95, click Yes.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Office 97 for Windows 95 and Windows NT. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

To start the Microsoft Clip Gallery in a Microsoft Office 97 product, point to Picture, and then click Clip Art on the Insert menu. When the Microsoft Clip Gallery 3.0 dialog box is displayed, you should see three buttons at the bottom: Edit Categories, Clip Properties, and Import Clips.
