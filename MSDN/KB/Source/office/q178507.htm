OFF98: No User Info or Serial Number in About Office Manager

 Last reviewed: February 2, 1998
Article ID: Q178507
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition

SYMPTOMS

If you click About Office Manager on the Microsoft Office Manager menu, the dialog box may contain the following information: 

   This product is licensed to:


   Serial Number:
There is no user or organization information, and no serial number.

CAUSE

This problem will occur if, for any reason, you delete or rename the Microsoft Office Settings (8) file in your System Folder:Preferences folder. The Microsoft Office Settings (8) file contains the user and organization information used by the Microsoft Office Manager; if you delete or rename the file, the new settings file does not contain the correct information.

This problem may also occur if you install Microsoft Office Manager on your computer before you run any of the programs included in Microsoft Office 98 Macintosh Edition.

RESOLUTION

It is not necessary to correct this problem if it occurs. However, if you want the About Office Manager dialog box to display the correct user and organization information, follow these steps:

  1. Using the Finder, delete or rename the following files, located in the specified folders on your hard disk: 

          Mac HD:System Folder:Preferences:Microsoft Office Settings (8)
      Mac HD:System Folder:Extensions:Microsoft OLE Library
      Mac HD:Microsoft Office 98:Office:Microsoft Office 98

  2. On the Special menu, click Empty Trash, and then click OK.

  3. Using the Finder, copy the following file: 

          Microsoft Office 98

    from the Microsoft Office 98:Office folder on your Microsoft Office 98 Macintosh Edition CD-ROM to the Microsoft Office 98:Office folder on your computer's hard disk.

  4. Start Microsoft Excel 98 Macintosh Edition.

    The Microsoft Office First Run dialog box appears.

  5. In the Microsoft Office First Run dialog box, type your Name and Organization. Click OK. Click OK again to confirm.

  6. Type your CD key, and click OK. Record your Product ID, and click OK.

  7. In Microsoft Excel 98 Macintosh Edition, click Quit on the File menu, and then restart the computer.

When you click About Office Manager on the Microsoft Office Manager menu, the correct information should appear.

STATUS

This behavior is by design of the Microsoft Office Manager included in Microsoft Office 98 Macintosh Edition.

MORE INFORMATION

The Microsoft Office Manager is included in the Microsoft Office 98 Macintosh Edition Value Pack. When you install the Microsoft Office Manager control panel and restart your computer, the Microsoft Office Manager menu appears to the left of the application menu, in the upper-right corner of the screen.
