OFF: Problems Using Version 1.0 of Power Mac Office Extension

 Last reviewed: March 13, 1998
Article ID: Q134909
The information in this article applies to:
  • Microsoft Word for the Macintosh, versions 6.0, 6.0.1
  • Microsoft Excel for the Macintosh, versions 5.0, 5.0a
  • Microsoft Office for the Macintosh, versions 4.2.1

SYMPTOMS

In Microsoft Word 6.0 or Microsoft Excel 5.0, when you click the File menu, Word may produce a "Type 3" error or it may stop responding (hang).

This problem occurs if the following items are installed on your computer:

  • Global Village TelePort or PowerPort software
  • Office4.2x Update for Power Mac (version 1.0)

Note also that a similar problem may occur with a product called PopupFolder from Inline Software if you are using the Open or Save dialog box.

CAUSE

This problem is caused by an extension conflict between the Global Village Tool Box (or PopupFolder) and the Office 4.2x Update for Power Mac extension. Removing either of these extensions will keep the problem from occurring.

RESOLUTION

This problem has been corrected with version 1.0.1 of the "Office4.2x Update for Power Macintosh" extension, available in an updated version of the Application Note. The Application Note number is MC1164.

For information on how to obtain this Application Note, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q130776
   TITLE     : AppNote: "Office 4.2x Update For Power Mac" (MC1164)
