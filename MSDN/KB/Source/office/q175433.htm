OFF97: Error Starting "Microsoft Office 97 Starts Here"

 Last reviewed: November 26, 1997
Article ID: Q175433
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office 97 Starts Here ISBN 1-57231-303-X

SYMPTOMS

When you start the program that comes with the Microsoft Press book "Microsoft Office 97 Starts Here" on a computer on which Microsoft Office 97 is installed, you may receive an error message similar to the following: 

   These Microsoft Office 97 applications are not installed on your
   computer:

   Microsoft Access

   You can run all the lesson step movies, but you will not be able to run
   interactive lessons that use the missing applications.
When you click OK, the program proceeds normally. However, you cannot run interactive lessons that use the programs listed in the error message.

CAUSE

This problem may occur if any of the programs included in Microsoft Office 97 are not installed on your computer. This may be true for any of the following reasons:

  • You installed Microsoft Office 97 Standard Edition on the computer. Microsoft Access 97 is not included in this edition of Microsoft Office 97.
  • You installed Microsoft Office 97 Small Business Edition (SBE), on the computer. Microsoft Access 97 and Microsoft PowerPoint 97 are not included in this edition of Microsoft Office 97.
  • When you installed Microsoft Office 97 on the computer, you chose not to install one or more of the programs (Microsoft Excel 97, Microsoft Word 97, Microsoft Access 97, or Microsoft PowerPoint 97) that are included in Microsoft Office 97.

RESOLUTION

To resolve this problem, install all the available Microsoft Office 97 programs. To do this, run the Microsoft Office 97 Setup program again, click Add/Remove, and then install the missing programs.

If you do not have all of the programs listed in the error message, or if you do not want to install these programs, click OK in the error message dialog box. The interactive lessons work for all of the programs that are installed.

STATUS

This behavior is by design of "Microsoft Office 97 Starts Here."
