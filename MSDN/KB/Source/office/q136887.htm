No "Run From CD" Option for Macintosh Office CD

 Last reviewed: February 2, 1998
Article ID: Q136887
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2.1, 4.2.1a, 4.2.1b
  • Microsoft Office 98 Macintosh Edition

SUMMARY

When you run the Setup program from the CD there is no option for Run From CD.

MORE INFORMATION

When you run Setup from either a disk or a CD-ROM, the Setup options are:

Microsoft Office 98 Macintosh Edition

Easy Install Custom Install

Microsoft Oiffice versions 4.2.1, 4.2.1a 4.2.1b

Typical Custom/Complete Minimum.

If you run Setup from a network installation, you will also have a Workstation option. However, no matter how you start the Setup program for Microsoft Office for the Macintosh, you will not be presented with a Run From CD option.

If you run Setup from the CD, you must have the CD in the CD-ROM drive. Similarly, if you want to use the online documentation, you must have the CD in the CD-ROM drive. Online documentation is not installed to your hard disk drive.
