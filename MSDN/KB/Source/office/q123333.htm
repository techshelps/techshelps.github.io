Office: "Not Enough Disk Space" When Installing MSAPPS

 Last reviewed: March 27, 1997
Article ID: Q123333
4.20-CD 7.00-CD WINDOWS kbsetup kbenv

The information in this article applies to:

  • Microsoft Office for Windows, version 4.2-CD
  • Microsoft Office for Windows 95, version 7.0-CD

SUMMARY

When you run Office Setup, if there is not enough room to install the shared applications folder (directory), Setup may request more disk space than is actually required to complete the installation.

MORE INFORMATION

If you choose to install the shared applications (the applications that are stored in the Msapps folder in version 4.2-CD or the "Common Files" folder in version 7.0), Microsoft Office Setup attempts to write these files to the "Windows\Msapps" folder or the "Programs\Common Files" folder. If there is not enough room to install the shared applications in that folder, Setup requests that you free up enough space on that drive for the entire installation, even if the remaining applications are being installed on another drive.

This stated requirement is incorrect: you only need to free up enough space on the Windows drive to install the shared applications. As an alternative, you can perform a Custom Setup and choose not to install some or all of the shared applications.
