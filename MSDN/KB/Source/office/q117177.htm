Office: "Insert Disk 1 of CD-ROM Setup..." with 4.2-CD Setup

 Last reviewed: September 16, 1996
Article ID: Q117177
The information in this article applies to:
  • Microsoft Office for Windows, version 4.2-CD

SUMMARY

When you run the Microsoft Office version 4.2-CD Setup program, and you have previously installed Microsoft Office version 4.2 from floppy disks, the 4.2-CD Setup program runs in maintenance mode; the Setup dialog box contains the Add/Remove, Reinstall, and Remove All buttons instead of the standard "Welcome to MS Office Setup" message.

In addition, if you choose the Add/Remove button in the maintenance mode of the version 4.2-CD Setup program, and you select items to add or remove, you are prompted to "Insert Disk 1 of the CD-ROM setup into drive <x>:" where <x> is the floppy disk drive from which you previously installed Microsoft Office.

NOTE: This is not a problem when you install Microsoft Office version 4.3-CD, even when you have previously installed Microsoft Office version 4.3 from floppy disks.

Two advantages of installing Microsoft Office version 4.2-CD when you already have Microsoft Office version 4.2 installed on your computer are as follows:

  • This method of installation can save disk space (if you remove the Microsoft Office files from the hard drive and then run Microsoft Office from the CD-ROM drive).

    -and-

  • You can use the online documentation provided on the Microsoft Office CD.

CAUSE

This error occurs because the Microsoft Office version 4.2-CD Setup program does not recognize the Microsoft Office version 4.2 installed on your computer as a previous version of Microsoft Office. Therefore, the Setup program runs in maintenance mode, using the existing Microsoft Office setup table file (SETUP.STF), instead of running in the standard Setup mode.

WORKAROUNDS

To work around the this problem, follow either of the procedures below.

To Run Microsoft Office from the CD

  1. Run the Microsoft Office Setup program (either from the CD-ROM drive or from the SETUP subdirectory of your Microsoft Office directory). In the Setup dialog box, choose the Remove All button.

  2. Re-run the Microsoft Office version 4.2-CD Setup program.

  3. Choose the Run From CD installation option.

To Use the On-line Documentation

  1. Using File Manager, delete or rename the SETUP.STF file in the SETUP subdirectory of your Microsoft Office directory.

  2. Run the Microsoft Office version 4.2-CD Setup program.

  3. Choose the Complete installation option.

The Setup program should run quickly when you use this procedure, because the files already exist on your hard drive, so the Setup program does not have to copy them again. The Setup program adds the online documentation icons to your Microsoft Office group in Program Manager.
