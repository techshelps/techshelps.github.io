System 7.5.3 Update File Contains Incorrect Office Information

 Last reviewed: February 2, 1998
Article ID: Q148720
The information in this article applies to:
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1, 4.2.1a, 4.2.1b

SUMMARY

After you install Macintosh System 7.5.3 on your computer, the file "New in This Update - 3" may be installed into the System Update Information folder.

MORE INFORMATION

The "New in This Update - 3" text file contains the following paragraph: 

   If you use Microsoft Office 4.2.1, you need the extension "Office 4.2x
   Update For Power Mac," version 1.0.1, and the Microsoft Office Manager
   control panel, version 4.2.1b. You can download these files from the
   Internet. Download the MC1164.HQX and MOMPATCH.HQX "patch" (update)
   files from Microsoft. The patches are available on the Internet at

      "ftp://ftp.microsoft.com/softlib/mslfiles"
To run Microsoft Office under System 7.5.3, you do not necessarily need either the Office 4.2x Update for Power Mac extension version 1.0.1 or the Microsoft Office Manager version 4.2.1b.

Microsoft recommends that you install the Office 4.2x Update for Power Mac extension version 1.0.1 on a Power Macintosh computer that is running System 7.5.3 if one of the following conditions is true:

  • You are running a version of Office that is earlier than version 4.2.1b and the Office 4.2x Update for Power Mac extension is not installed. 

        -or-
  • You are using QuickDraw GX when you print. 

        -or-
  • You are using version 1.0 of the Office 4.2x Update for Power Mac extension and Global Village Teleport or PowerPort fax software.

    NOTE: Version 1.0 of the extension is known to conflict with Global Village Teleport or PowerPort fax software. For additional information, please see the following article in the Microsoft Knowledge Base: 

          ARTICLE-ID: Q134909

          TITLE     : OFF: Problems Using Version 1.0 of Power Mac Office
                  Extension


NOTE: If you are using Microsoft Office 4.2.1a or 4.2.1b, you do not need
this extension when you print from Word with QuickDraw drivers.


Also, you do not need to run the patch for the Microsoft Office Manager
version 4.2.1b if you are running System 7.5.3. The patch for the Office
Manager is only for computers that are running Microsoft Office version
4.2.1 and later and System 7.5.2.


NOTE: A new Microsoft Office Manager Updater is available. The Office
Manager Updater is a patch to the Microsoft Office Manager to update any
version of Microsoft Office Manager to version 4.2.1c.
