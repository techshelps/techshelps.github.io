OFF97: How to Configure Setup to Install Extra Clip Art

 Last reviewed: March 13, 1998
Article ID: Q162914
The information in this article applies to:
  • Microsoft Office 97 for Windows
  • Microsoft Office 97 Network Installation Wizard

SUMMARY

In Microsoft Office 97, it is possible to customize an administrative installation (setup /a), so that when your users run Microsoft Office 97 Setup, an option to install extra clip art is automatically added to the Add/Remove Programs dialog box on their computer.

This article explains how you can perform this task using the Microsoft Office 97 Network Installation Wizard.

MORE INFORMATION

When you perform an administrative installation of Microsoft Office 97 to a network server, a large amount of additional clip art is copied to the server. However, when your users run the Microsoft Office 97 Setup program, this clip art is not copied to their computers. To install this additional clip art, they must run a separate setup program for Microsoft Clipart Extra.

If you perform the following steps, an option to run the Microsoft Clipart Extra Setup program will automatically appear in each user's Add/Remove Programs dialog box in the Windows Control Panel.

NOTE: The following steps assume that you have successfully performed an administrative installation of Microsoft Office 97 to a network server, and that you have installed the Network Installation Wizard on a workstation that has full rights to the network server.

How to Configure the Setup Table File Using Network Installation Wizard

  1. On your network server, make a copy of the Off97pro.stf or Off97std.stf file. Rename the copy to an appropriate name, such as Custom.stf.

  2. Start the Network Installation Wizard.

  3. Select the setup table file you want to customize (for example, Custom.stf), and follow the steps in the Network Installation Wizard until you reach the "Add Registry Entries" dialog box.

  4. Click Add.

  5. In the Add dialog box, set the following values 

          Root         HKEY_LOCAL_MACHINE
      Data type    SZ
      Key          Software\Microsoft\Windows\CurrentVersion\Uninstall\
                      ClipArt
      Value name   DisplayName
      Value data   Install Microsoft Clipart Extra

   and click OK.

  6. Click Add.

  7. In the Add dialog box, set the following values 

          Root         HKEY_LOCAL_MACHINE
      Data type    SZ
      Key          Software\Microsoft\Windows\CurrentVersion\Uninstall\
                      ClipArt
      Value name   UninstallString
      Value data   X:\Msoffice\Clipart\Setup.exe

   and click OK.

   NOTE: The path shown for "Value data" assumes that the network server
   that contains Microsoft Office 97 is mapped to each user's computer as
   drive X:, and that Microsoft Office 97 is stored in the Msoffice
   folder on that drive. However, you can also specify a UNC (universal
   naming convention) path to point to Setup.exe. For example, this is a
   valid "Value data" string:

      \\Myserver\Share\Msoffice\Clipart\Setup.exe

  8. Click Next, and proceed through the remainder of the Network Installation Wizard process.

After you complete the above steps, the Install Microsoft Clipart Extra item will appear in the Add/Remove Programs control panel of any user who performs an installation of Microsoft Office 97 using the modified setup table file (for example, Custom.stf).

How to Run the Microsoft Office 97 Setup Program

  1. On the Start menu, click Run. In the Open box, type the path to the Setup program, along with any setup switches that you want to use. For example: 

          X:\Msoffice\Setup.exe /l custom.lst
      \\Myserver\Share\Msoffice\Setup.exe /l custom.lst /b2 /q

    Click OK.

  2. Proceed normally through the Microsoft Office 97 Setup program. If you are prompted to restart Windows, do so.

For more information on using custom setup table files with Microsoft Office 97, see the following articles in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q158503
   TITLE     : OFF97: Setup Switches for Microsoft Office 97

   ARTICLE-ID: Q158281
   TITLE     : OFF97: How to Use Custom STF Files with MS Office 97 Setup

How to Install the Additional Clip Art

After Microsoft Office 97 Setup is complete, you can install the additional clip art by performing the following steps:

  1. On the Start menu, point to Settings, and click Control Panel.

  2. Double-click Add/Remove Programs. Click the Install/Uninstall tab.

  3. In the list of installed programs, click Install Microsoft ClipArt Extra, and then click Add/Remove.

The Microsoft Clipart Extra Setup program will run and install all of the clip art onto your computer's hard drive.
