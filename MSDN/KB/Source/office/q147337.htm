Network Installation Wizard Supported on English/US Version

 Last reviewed: April 23, 1997
Article ID: Q147337
7.00 WINDOWS kbsetup

The information in this article applies to:

  • Microsoft Office for Windows 95, version 7.0

SUMMARY

The Microsoft Office Network Installation Wizard is supported only on the English/U.S. version of Microsoft Office for Windows 95.

MORE INFORMATION

The Network Installation Wizard in "Microsoft Office for Windows 95 Resource Kit" by MS Press was not intended or designed for localized versions of Office.

REFERENCES

"Microsoft Office for Windows 95 Resource Kit," Chapter 9, "Customizing Client Installations," pages 192-204, "Modifying Client Setup with the Network Installation Wizard"
