OFF95: Disk Space Required for Each Installation Type

 Last reviewed: June 23, 1997
Article ID: Q134740

The information in this article applies to:
  • Microsoft Office for Windows 95, version 7.0, Standard Edition

SUMMARY

When you run Microsoft Office Setup, you will have several different installation types, based on whether you initiate Setup from a compact disc (CD), an administrative installation, or from a floppy disk. Each of the installation types requires a different amount of hard disk space, because each will install a different number of files on your computer.

MORE INFORMATION

There are several different types of installations for Microsoft Office: Compact, Typical, Custom, Run from CD, Administrative, and Run from Network. Depending on whether you did your installation from a CD, floppy disk, or from an administrative installation, you will be able to choose from a different combination of installation types.

The following table is a cross-reference of the installment types based the way you run Setup. 

   Type              CD         Floppy Disk         Admin Install
   --------------------------------------------------------------

   Compact           x              x                    x
   Typical           x              x                    x
   Custom            x              x                    x
   Run from CD       x
   Run from Network                                      x
