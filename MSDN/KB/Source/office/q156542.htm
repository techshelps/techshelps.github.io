OFF97: Microsoft Office 97 Service Release 1 Enterprise Update

 Last reviewed: March 13, 1998
Article ID: Q156542
The information in this article applies to:
  • Microsoft Office 97 for Windows, Service Release 1

SUMMARY

There are two versions of Microsoft Office 97 SR-1: an SR-1 Patch and an SR- 1 Enterprise Update. The SR-1 Enterprise Update requires that you completely reinstall Office and is available only on compact disc and disk. This article contains a list of updates that are included in the SR-1 Enterprise Update version and NOT in the SR-1 Patch.

MORE INFORMATION

Microsoft Office 97 SR-1

The Microsoft Office 97 SR-1 Enterprise Update includes the following updates for Office 97:

  • When you install Microsoft Office 97, you may receive error messages about the Bdrintl.dll file. This problem is corrected in Microsoft Office 97 SR-1.

    For more information, please see the following articles in the Microsoft Knowledge Base: 

          ARTICLE-ID: Q162295

          TITLE     : OFF97: Err Msg: "Bdrintl.dll Was Unable to Register
                  Itself"

      ARTICLE-ID: Q162936
      TITLE     : OFF97 Setup: Banyan Vines Patch Available


  • In Microsoft Office 97, the Microsoft Office Shortcut Bar is NOT
   installed when you perform a Typical installation. In Microsoft Office
   97 SR-1, the Office Shortcut Bar is installed when you perform a Typical
   installation.


  • The Microsoft Office 97 SR-1 Enterprise Update includes updated Help
   (.hlp) and text (.txt) files. These files are not included in the
   patch version.

    

Microsoft Word 97 SR-1

 


The Microsoft Office 97 SR-1 Enterprise Update includes the following
updates for Microsoft Word 97:


  • The Avery Wizard is updated for improved integration and stability with
   Microsoft Word 97 SR-1.


  • The Support Template, Macros8.dot, does not return an error message when
   you run the SuperDocStatistics macro.

    

Microsoft Excel 97 SR-1

 


The Microsoft Office 97 SR-1 Enterprise Update includes the following
updates for Microsoft Excel 97:


  • A new Microsoft Excel 97 Viewer designed for use in Microsoft Windows 95
   and Microsoft Windows NT is included.

    

Microsoft PowerPoint 97 SR-1

 


The Microsoft Office 97 SR-1 Enterprise Update includes the following
updates for Microsoft PowerPoint 97:


  • A new Microsoft PowerPoint 97 Viewer that is designed for use in
   Microsoft Windows 95 and Microsoft Windows NT is included.


  • A filter that allows Microsoft PowerPoint 4.0 to directly open Microsoft
   PowerPoint 97 presentation files is included.

    

Microsoft Access 97 SR-1

 


The Microsoft Office 97 SR-1 Enterprise Update includes the following
updates for Microsoft Access 97:


  • A new Microsoft Access 97 Snapshot Viewer for Windows 95 and Windows NT
   allows any user to view and print data contained in a Microsoft Access
   97 Snapshot.


  • The Microsoft Office 97 SR-1 Enterprise Update includes a Service Pack
   for the Microsoft Office 97 Developer Edition. This Service Pack
   should be installed if you use the following:

    

        - Computers running Microsoft Windows NT 4.0 with Service Pack 3
      and the Microsoft Office 97 Developer Edition

    

          -and-

    

        - Computers on which the ODBC Driver Manager 3.0 is installed

    

Microsoft Outlook 97 SR-1

 


The Microsoft Office 97 SR-1 Enterprise Update includes the following
updates for Outlook 97:


  • Support for new features in Microsoft Exchange Server 5.0 is included.


  • When you switch between the Microsoft Exchange client and Microsoft
   Outlook when you use forms, performance is improved.


  • The Daylight Savings Time option works correctly when you update
   appointments in Microsoft Outlook 97 SR-1.


  • Lotus cc:Mail messages that use Greek characters do not display
   extraneous characters when you import them into Microsoft Outlook 97
   SR-1.


  • The Delegate With Permissions option is displayed correctly in
   Microsoft Outlook 97 SR-1. Instead of "Custom", the option is displayed
   as "Delegate".


  • The security features in Microsoft Outlook 97 SR-1 include a Send
   Security Keys option.


  • Microsoft Outlook 97 SR-1 can inherit Microsoft Schedule+ permissions.


  • The forms size limitation has expanded. The limit is no longer 32
   kilobytes (K).


  • Thanksgiving and the Lunar New Year are correctly identified as
   holidays.
