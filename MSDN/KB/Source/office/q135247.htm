Office: Online Documentation for Microsoft Access Unreadable

 Last reviewed: September 16, 1996
Article ID: Q135247
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3-CD

SYMPTOMS

The online documentation for Microsoft Access included on the Microsoft Office 4.3-CD contains some unreadable text. The file that contains this error is Msoff43.mvb located in the Mviewer folder on the compact disc (CD).

STATUS

Microsoft has confirmed this to be a problem in the Microsoft products listed at the beginning of this article. This problem has been corrected in the Microsoft Office 4.3c-CD.

MORE INFORMATION

To locate the error in the documentation:

  1. From the Main screen in the online documentation, click Building Applications under Microsoft Access.

  2. Under Building Applications, click Introduction.

  3. Move to page 2 of the Introduction (by clicking the right arrow to the right of the number 2).

If you scroll down to the bottom of page 2, you will notice that the description of the conventions contains scrambled or overwritten text that is unreadable.
