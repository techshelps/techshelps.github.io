OFF95: How to Upgrade Office 7.0x to 7.0b

 Last reviewed: March 10, 1997
Article ID: Q153328
7.00 7.00a 7.00b WINDOWS kbsetup kbref kbhowto

The information in this article applies to:

  • Microsoft Office for Windows 95, versions 7.0, 7.0a, 7.0b

SUMMARY

Before installing Microsoft Office for Windows 95, version 7.0b on a computer that already has Microsoft Office for Windows 95, version 7.0 or 7.0a installed, you do not have to remove the currently installed version. However, it is important to install the maintenance release of Office version 7.0b into the same folder where Office was previously installed, to choose the same type of installation as before, and to install the same, or more, components if a Custom installation was originally performed.

MORE INFORMATION

Microsoft Office for Windows 95 version 7.0b is a maintenance release of Microsoft Office for Windows 95 versions 7.0 and 7.0a. Although the file, Offreadme.txt, exists on the first Microsoft Office for Windows 95 disk and on the root of the Microsoft Office for Windows 95 compact disc to assist you in upgrading Microsoft Office for Windows 95, there are no instructions sent with the release or in the Offreadme.txt file to guide you in "upgrading" the existing Microsoft Office for Windows 95 version 7.0 or 7.0a installation. It is not necessary (and it is not suggested) that you remove the previous version of Microsoft Office for Windows 95 because customizations for the products may be lost. Below are some instructions for upgrading in different, existing installation scenarios.

NOTE: If you want to change the installation type or installation destination folder, you must run setup from the previous Microsoft Office for Windows 95 media and choose Remove All from maintenance mode setup before installing Microsoft Office for Windows 95, version 7.0b.

For An Existing Administrative Installation

Delete the current administrative installation and follow the same directions to perform another administrative installation of Office to the same network share. See the Network.txt file on the first Microsoft Office for Windows 95 disk or on the root of the Microsoft Office for Windows 95 compact disc for additional instructions. Please note that each "Run from Network Server" installation must also be performed again (see below).

For an Existing Run from Network Server Installation

Each "Run from Network Server" installation must be performed again after the administrative installation has been updated. The destination folder must be the same. See the Network.txt file for additional instructions.

For an Existing Custom Installation

Perform another Custom installation. The destination folder must be the same and the same items (or more) in the options list must be selected in order to update older files.

NOTE: If you do not recall which components you previously installed, you must either install every component of Microsoft Office for Windows 95, version 7.0b, or you must run setup from the previous Microsoft Office for Windows 95 media and choose Remove All from maintenance mode setup, and then install Microsoft Office for Windows 95 version 7.0b into the same folder.

For an Existing Compact, Typical, Run from CD Installation

Perform another installation of the same type with the maintenance release. The destination folder must also be the same.

REFERENCES

The Network.txt file on the first Microsoft Office for Windows 95 disk or on the root of the Microsoft Office for Windows 95 version 7.0 compact disc.

The Offreadme.txt file on the first Microsoft Office for Windows 95 disk or on the root of the Microsoft Office for Windows 95 version 7.0 compact disc.
