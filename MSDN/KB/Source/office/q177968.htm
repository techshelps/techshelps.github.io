OFF: "Could Not Find <Program>" When You Click Office Menu

 Last reviewed: February 2, 1998
Article ID: Q177968
The information in this article applies to:
  • Microsoft Office 98 Macintosh Edition
  • Microsoft Office for the Macintosh, versions 4.2, 4.2.1

SYMPTOMS

If you click a menu command for a program on the Microsoft Office Manager menu, you may receive an error message similar to the following: 

   Could not find <Program>.
If you click Customize on the Microsoft Office Manager menu, click the menu command for the program that could not be found, and then click Edit, the dialog box that appears indicates that the program still exists in the specified location.

CAUSE

This problem may occur if you remove a program from your computer. The Microsoft Office Manager does not automatically remove menu commands when you remove a program from your computer.

RESOLUTION

If this problem occurs, remove the menu command from the Microsoft Office Manager by following these steps:

  1. On the Microsoft Office Manager menu, click Customize.

  2. In the list of menu items, click the menu command for the removed program.

  3. Click Remove. Click OK to remove the menu command from the Office menu.

Or, if you reinstall the program to the same location in which it existed originally, the menu command will work correctly.

If you reinstall the program, or if the program is installed in an alternate location, click New, select the program in its new location, and then click Add. A new menu command is created for the program.

STATUS

This behavior is by design of the Microsoft Office Manager included in Microsoft Office for the Macintosh, versions 4.2 and 4.2.1, and Microsoft Office 98 Macintosh Edition.

MORE INFORMATION

The Microsoft Office Manager is automatically installed when you install Microsoft Office for the Macintosh, versions 4.2 and 4.2.1, on your computer. The Microsoft Office Manager menu appears to the left of the Application menu, in the upper-right corner of the screen. The Microsoft Office Manager is also included in the Microsoft Office 98 Macintosh Edition Value Pack.
