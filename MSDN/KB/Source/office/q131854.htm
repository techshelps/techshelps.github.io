OFF95: Can't Create or Record Macro in WordMail or Binder

 Last reviewed: March 27, 1997
Article ID: Q131854
7.00 WINDOWS kbprg

The information in this article applies to:

  • Microsoft Word for Windows 95, version 7.0
  • Microsoft Office
  • Microsoft Windows 95

SYMPTOMS

When you try to create a macro in Word Mail or in a Word document in a Binder, the Record, Create, and Edit buttons are unavailable. The only option available is to Run or Delete an existing macro.

CAUSE

These buttons are unavailable because not all of the commands that are available in WordMail or Binder can be recorded in Word's macro language.

WORKAROUND

To work around this problem, switch to Word and create the macro in Word. The macro will be available the next time you use WordMail or the Binder. In a Binder, you can switch to Word by clicking View Outside on the Section menu.
