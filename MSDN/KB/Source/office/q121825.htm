Office 4.3c-CD Demos: Descriptions and Exiting Instructions

 Last reviewed: September 16, 1996
Article ID: Q121825
The information in this article applies to:
  • Microsoft Office for Windows, version 4.3c-CD

SUMMARY

Microsoft Office ships with a collection of Office Compatible Demos, along with a Microsoft Money demo. A separate icon is created for the Compatible Demos item and for the Money Demo item, and these icons are added to the Microsoft Office group in Program Manager.

The following information describes how to exit each of the demos while they are running, and the brief description that appears with each of the Office Compatible Demos.

Exiting Demo While Running

The following table describes how to exit an individual demo while it is running.

Description of Each Demo

The following is a list of each of the Office Compatible Demos, and the brief description that appears with each.

A-Talk for Windows - A-Talk for Windows

Telecommunications for the Microsoft Office. Supports Mail, Network DDE, Clipboard, Drag & Drop, VT220 and Tektronix 4014. To order call (800) 300-1803 (end users) or 213-669-1497 (resellers).

Action! - Macromedia's Action 3.0 for Windows

Action! is a revolutionary package that you can use to create dazzling presentations that incorporate digital video, sound, animation, graphics, and interactivity. For more info, call (800) 288-4797.

Capsule - Capsule

Put your Office on Autopilot! Automate tasks that involve moving data between Office products. Without programming, scripts or macros. With Capsule from Metaphor. (800) 399-5111.

Fileware for Microsoft Word - Fileware: Essential Microsoft Word Tools

Do everything you need without ever leaving Word for Windows! Fileware is a powerful Word add-in suite. File management, Address Book, and more. "48 Best Add-In Products"--PC Magazine. Call (718) 375-1261.

Mapland - Mapland

Mapland is Spreadsheet Mapping software that is an add-in to Microsoft Excel 5.0. With MapLand, you can chart your data on a map inside of Microsoft Excel. Contact Software Illustrated at (510) 463-9898.

METZ Phones--Address & Telephone Number Management

METZ Phones is Microsoft Office Compatible with an easy-to-use and common interface, yet powerful enough for all your Address and Telephone contact management needs. METZ Phones is designed for network and individual users. Call (206) 641-4525.

Microsoft Project - Microsoft Project

Microsoft Project version 4.0 is the most popular project planning tool on both the Windows and Macintosh platforms. It helps you plan, manage and communicate with ease. To order Microsoft Project today call (800) 451- 4239.

Microsoft Publisher - Microsoft Publisher

Microsoft Publisher version 2.0 for Windows gives you the tools you need to design professional looking newsletters, brochures, flyers, and more--all in a few easy steps. To order Microsoft Publisher today, call (800) 451- 4239.

Microsoft Works - Microsoft Works

Microsoft Works for Windows is the integrated application that allows Windows users with little or no computing experience to get their daily computing tasks done quickly and easily. To order Microsoft Works today call (800) 451-4239.

ProdeaSynergy - ProdeaSynergy

ProdeaSynergy streamlines tasks that use multiple Office applications into a single step that you can save and run at the click of a button. Limited offer $49. Call (612) 942-1000.

Recruiting Manager - Recruiting Manager

Recruiting Manager boosts the competitiveness of growing companies by automating the employee recruiting process. Save time and reduce costs while landing the most qualified candidates. Call (610) 697-6600.

RoboHELP - The Windows Help Style Guide

Build your own professional looking Windows Help system the easy way! (Built with RoboHELP , Help Authoring Tool for Windows & Windows NT). Call (619) 459-6366 (fax) or (800) 677-4WIN.

SmarTerm for Windows - SmarTerm for Windows
