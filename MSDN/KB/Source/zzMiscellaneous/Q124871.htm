The information in this article applies to:

- Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

The Application Note "Lotus 1-2-3 Versions 2.0-4.0 Converter" (GC1133) is now avaiable from Microsoft Product Support Services. This Application Note contains an updated Lotus 1-2-3 converter to use with Microsoft Word version 6.x for Windows. The converter allows you to import information from Lotus 1-2-3 version 2.x, 3.x, and 4.x worksheets into Word 6.x.

This converter replaces the Lotus 1-2-3 converter supplied with the Word 6.0 Supplemental File Conversions Disks. This earlier version of the converter did not support Lotus 1-2-3 version 4.x.

You can obtain this Application Note from the following sources:

  • Microsoft's World Wide Web Site on the Internet
  • The Internet (Microsoft anonymous ftp server)
  • Microsoft Download Service (MSDL)
  • Microsoft Product Support Services

For complete information, see the "To Obtain This Application Note" section at the end of this article.

OVERVIEW

This Application Note contains an updated Lotus 1-2-3 spreadsheet converter (LOTUS123.CNV) to use with Microsoft Word version 6.0 for Windows. The converter now allows you to convert worksheets from Lotus 1-2-3 versions 2.0, 2.2, 3.0, and 4.0 to Word 6.0. (Earlier releases of the converter did not support Lotus 1-2-3 version 4.0.)

This Application Note also contains the 123LIMIT.DOC file, a Word for Windows document that contains a table of conversion-limitation information about the Lotus 1-2-3 converter. The table lists only the features that are not converted, that are not supported in one product or the other, or that require comment.

INSTALLATION AND CONVERSION

To copy the new LOTUS123.CNV converter

to your Word for Windows program directory

  1. If you received a disk with this Application Note, insert the disk in the appropriate floppy disk drive.

    If you downloaded this Application Note from an online service, skip to step 2.

  2. At the MS-DOS prompt, type the following and press ENTER 

          copy <source>\lotus123.cnv <location>

    where the <source> is either the letter of your floppy disk drive or the directory containing the converter you downloaded from online services, and <location> is the path to your Word for Windows program directory. For example: 

          copy a:\lotus123.cnv c:\winword

To install the updated Lotus 1-2-3 versions 2.0-4.0 converter

  1. Exit and restart Word.

  2. From the File menu, choose Open. In the Open dialog box, be sure the Confirm Conversions check box is selected.

  3. In the File Name box, specify any Lotus 1-2-3 worksheet file.

  4. If Word proposes to open the worksheet file as Text Only, scroll down through the list of available file types and select the Lotus 1-2-3 option.

  5. Choose the OK button to convert a copy of the worksheet to Word format.

Word will automatically install the converter, and you should now be able to open your Lotus 1-2-3 worksheets in Word.
