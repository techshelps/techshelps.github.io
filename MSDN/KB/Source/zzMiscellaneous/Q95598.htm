PPT: Apple File Exchange (AFE) Translator Available

 Last reviewed: March 20, 1998
Article ID: Q95598
The information in this article applies to:
  • Microsoft PowerPoint for the Macintosh, versions 2.0 and 2.01
  • Microsoft PowerPoint for Windows, version 2.0

SUMMARY

The Microsoft PowerPoint Apple File Exchange (AFE) Translator is required to convert files from versions 2.0 and 2.01 of Microsoft PowerPoint for the Macintosh to version 2.0 of Microsoft PowerPoint for Windows and vice versa.

PowerPoint 3.0 for the Macintosh does not require the translator to exchange files with PowerPoint 3.0 for Windows. These files can be moved from platform to platform without any translation.

TO OBTAIN THIS FILE

  • Application Notes are available by modem from the Microsoft Download Service (MSDL), which you can reach by calling (425) 936-6735. This service is available 24 hours a day, 7 days a week. The highest download speed available is 14,400 bits per second (bps). For more information about using the MSDL, call (800) 936-4100 and follow the prompts. To obtain PPTEXG, download PPTEXG.SEA. PPTEXG.SEA is a compressed, self-extracting file. After you download PPTEXG, run it to extract the file(s) it contains.
  • On the Internet, Application Notes are located on the Microsoft anonymous ftp server, which you can reach by typing "ftp ftp.microsoft.com" (without the quotation marks) at the ">" command prompt. You can find PPTEXG.HQX in the SOFTLIB\MSLFILES directory.
