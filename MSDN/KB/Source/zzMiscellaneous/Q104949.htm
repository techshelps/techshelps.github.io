Supported Foreign File Formats in WinWord 6.0 / 6.0a / 6.0c

 Last reviewed: December 10, 1997
Article ID: Q104949
The information in this article applies to:
  • Microsoft Word for Windows, versions 6.0, 6.0a, 6.0c

SUMMARY

Word versions 6.0, 6.0a, and 6.0c support conversion to and/or from a number of foreign file formats, including word processor, spreadsheet, and database files. Word interprets some foreign formats internally and others through use of external converter files. Word now interprets database formats using Open Database Connectivity (ODBC).

A list of supported formats is included below. Support for file formats listed beneath the heading "CONVERTERS SUPPLIED WITH WORD" is included in your Word package. All other converters are available either independently or via the Word 6.0 Supplemental File Conversions Disks.

  • For information on how to obtain the Supplemental File Conversions Disks, see the "More Information" section below.
  • For information on how to obtain converters supplied independently, see the Knowledge Base article reference included with the converter name below.

MORE INFORMATION

For information about specific converters supplied Word, and for information on how to obtain additional converters supplied on the Word 6.0 Supplemental File Conversions Disks, double-click the Help button on the Standard toolbar in Word and then type "readme" (without the quotation marks). Press ENTER twice, choose the File Conversion topic, and then choose the Overview topic. Alternatively, see the Supplemental Offers coupon included with your Word 6.0 retail package or see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q107141
   TITLE     : Word 6.0 for Windows Supplemental File Conversion Disks
For more information on enhancements made to specific Word 6.0 converters, query on the following words in the Microsoft Knowledge Base: 

   textconv and 6.0 and enhancements and <converter name>
For example, to find information on enhancements made to the WordPerfect 5.0, 5.1, and 5.2 converter, query on the following words in the Microsoft Knowledge Base: 

   textconv and 6.0 and enhancements and wpft5
For information on third-party vendors that support conversion to and/or from foreign file formats not directly supported by Word, query on the following words in the Microsoft Knowledge Base: 

   textconv and third and party and <file format>

   NOTE: <file format> is the name of the foreign file format you want to
   convert.
Word 6.0 also installs a converter (MSWORD6.CNV) that other Microsoft applications (namely, Microsoft Works for Windows, Microsoft PowerPoint for Windows, Microsoft Publisher for Windows) can use to read Word 6.0 documents. These applications share the converters that are installed, by default, in the MSAPPS\TEXTCONV subdirectory of your Windows program directory. Once installed, the converter is automatically available to any of the above applications that you have on your computer. Word 6.0 itself does not use this converter.
