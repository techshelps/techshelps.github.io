File Corruption Occurs with Texel 3024 CD-ROM Drive

 Last reviewed: May 13, 1997
Article ID: Q110204
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you access or copy files from a Texel 3024 CD-ROM drive, file corruption may occur. The drive functions properly under MS-DOS.

CAUSE

The Texel 3024 CD-ROM drive is not on the Windows NT Hardware Compatibility List.

WORKAROUND

This CD-ROM drive requires a firmware revision to be compatible with Windows NT.
