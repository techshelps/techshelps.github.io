Doc Err: NBPING and WNBSTAT

 Last reviewed: May 14, 1997
Article ID: Q113842
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Resource Kit for the Windows NT operating system and Windows NT Advanced Server version 3.1

Page 766 of the Windows Resource Kit, Volume 1 makes reference to two utilities used for trouble-shooting network problems. The two programs are NBPING and WNBSTAT. Neither one of these programs are included with Windows NT or the Windows NT Resource Kit. NBPING is not currently available, and WNBSTAT is a utility that is included only in the Microsoft LAN Manager Resource Kit.
