Mouse Leaves Horizontal Lines on Display with Matrox Video

 Last reviewed: February 11, 1998
Article ID: Q160358

The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

When you use the Matrox Millennium video card with Windows NT Server or Windows NT Workstation version 4.0, you will see a trail of single pixel lines left at irregular intervals behind the cursor.

CAUSE

This problem is caused by the way Windows NT Server and Windows NT Workstation version 4.0 read the Millennium frame buffer read cache.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 4.0. This problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack. Query on the following work in the Microsoft Knowledge Base (without spaces) for information on obtaining the service pack: 

   S E R V P A C K
