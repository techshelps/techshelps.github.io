Command Prompt Scrolls Slowly

 Last reviewed: May 8, 1997
Article ID: Q102136

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

If you have selected a video resolution that provides 256 colors, you may notice that scrolling in the Command Prompt is very slow.

CAUSE

Certain super VGA chip sets do not implement planar mode, which is used by the display driver to manipulate console memory quickly. Therefore, you may experience this problem if you use any of the following drivers: 

   Qvision
   Western Digital/Paradise
   Trident
   Compaq AVGA

RESOLUTION

There is no work around for this problem because the hardware does not support the necessary functionality.
