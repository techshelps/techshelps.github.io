Error Message When Starting Add Printer Wizard

 Last reviewed: April 24, 1997
Article ID: Q167376
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you start the Add Printer Wizard, you may receive the following error message: 

   Out of resources.

   Printer operation cannot continue due to lack of resources. The
   specified module could not be found.

CAUSE

The Ntprint.dll file in the %SystemRoot%\System32 folder may be missing or damaged.

RESOLUTION

Expand a new copy of the Ntprint.dll file from the Windows NT 4.0 CD-ROM to the %SystemRoot%\System32 folder using the following steps:

  1. If an Ntprint.dll file already exists in the %SystemRoot%\System32 folder, rename it to Ntprint.old.

  2. Place the Windows NT 4.0 CD-ROM in the CD-ROM drive.

  3. Type the following line at a command prompt: 

          expand d:\i386\ntprint.dl_ c:\winnt\system32\ntprint.dll

    NOTE: This article assumes that Windows NT is installed in the Winnt folder on drive C and that the CD-ROM drive is drive D. If this is not the case, adjust the command as necessary.
