FreeCell Help Opens Extra Windows

 Last reviewed: March 27, 1997
Article ID: Q155396
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows 95

SYMPTOMS

If you open and minimize an online Help topic for FreeCell and then click Contents on the FreeCell Help menu, the following windows are opened:

  • A new FreeCell Help Topics window with the Contents dialog box displayed.
  • The FreeCell Help topic window that you had minimized.
  • A FreeCell Help window with the following message: 

          Click Help Topics to return to the list of topics

The expected behavior is for the Help topic window you were reading to close and a new Help Topics window to open with the Contents dialog box displayed.

NOTE: This issue also occurs in the versions of Minesweeper and Solitaire included with Windows NT 4.0.

STATUS

Microsoft has confirmed this to be a problem in Windows NT 4.0 and Windows 95. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
