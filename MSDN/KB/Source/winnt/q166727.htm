McAfee NetShield or VirusScan May Cause Hangs with SP3

 Last reviewed: January 12, 1998
Article ID: Q166727
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0 Service Pack 3
  • Microsoft Windows NT Server version 4.0 Service Pack 3

SYMPTOMS

You may experience STOP 0x0000001E, STOP 0x0000000A bugchecks, lockups, or other random buchecks when you run McAfee NetShield or VirusScan on Windows NT 4.0 Service Pack 3.

CAUSE

These problems occur when the filter driver (Mcfilter.sys) from these McAfee applications corrupts memory within Windows NT kernel's address space. itself. actually

RESOLUTION

This problem is fixed in McAfee version 3.02 and later. Please contact McAfee Technical support for the updated version.
