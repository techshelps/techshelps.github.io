Multimedia SDK Setup Does Not Work on Windows NT

 Last reviewed: May 8, 1997
Article ID: Q102511
The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

The Microsoft Windows Multimedia Software Development Kit (SDK) does not install properly under Windows NT, yielding the following: 

   Please exit Windows & run setup from the DOS prompt to install...

CAUSE

The Windows Multimedia SDK was designed for Windows version 3.0. You cannot load it under Windows NT.

RESOLUTION

Do not use the Windows Multimedia SDK on Windows NT. There is no practical reason to use it, because the application programming interfaces (APIs) have been documented and included as part of Windows version 3.1 and Windows for Workgroups.
