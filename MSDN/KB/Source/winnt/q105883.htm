Ambiguous Error Net Message ID:3716

 Last reviewed: May 12, 1997
Article ID: Q105883
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

The content of Net Message ID:3716 is misleading and ambiguous. When you type "NET HELPMSG 3716" (without the quotation marks), you receive the following message: 

   EXPLANATION

   You typed an invalid device name.

   ACTION

   Check the spelling of the device name. Valid device names are LPT1:
   to LPT9: and COM1: to COM9: for printers and communication devices,
   and A: to Z: for disk devices.

MORE INFORMATION

The above error message could lead you to believe that you can actually make a connection by typing the following 

   net use COMx: \\servername\sharename
or connect to a modem shared on another computer running Windows NT. These are not supported features under Windows NT.
