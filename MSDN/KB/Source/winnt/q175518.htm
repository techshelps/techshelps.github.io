Cannot use Computer Setup Wizard on Small Business

 Last reviewed: March 5, 1998
Article ID: Q175518
The information in this article applies to:
  • Microsoft BackOffice Small Business Server Version 4.0

SYMPTOMS

When you use the Computer Setup Wizard in BackOffice Small Business, the following error is displayed: 

   Windows NT Client Setup

   This setup program must be run on a Windows NT Version 4.x client
   system.

CAUSE

This error is generated when the computer that you are trying to set up is the Small Business Server itself. The Small Business Server should not be set up as a client computer, even for the administrator account.

RESOLUTION

Do not attempt to set up the Small Business Server as a client. Select a workstation computer to configure for the Administrator account.
