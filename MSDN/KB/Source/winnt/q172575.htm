Running RegClean 4.1 Causes Access Violation in Windows NT 4.0

 Last reviewed: March 18, 1998
Article ID: Q172575
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you run RegClean version 4.1 in Windows NT 4.0, the following error message may be displayed: 

   Application Error
   RegClean.exe
   Exception: access violation(0xc0000005), address:0x00414053

CAUSE

RegClean version 4.1 is not compatible with Windows NT 4.0.

RESOLUTION

To work around this behavior, download the latest version of RegClean from the following Microsoft site: 

   ftp://ftp.microsoft.com/Softlib/MSLFILES

MORE INFORMATION

For additional information, please see the following article in the Microsoft Knowledge Base:
