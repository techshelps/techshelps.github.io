How to Upgrade an Evaluation Version of Cluster Server

 Last reviewed: March 20, 1998
Article ID: Q178924
The information in this article applies to:
  • Microsoft Windows NT Server, Enterprise Edition version 4.0 - Microsoft Cluster Server

SYMPTOMS

An installation of an evaluation copy of Microsoft Windows NT 4.0 Enterprise Server may produce the following error: 

   STOP:0x00000098(0x647CDCB0,0x01BC9900,0x0002A300,0x00000000)
   END_OF_NT_EVALUATION_PERIOD

   The evaluation period for this installation of Windows NT has expired.
   This system will shutdown in one hour. To restore access to this
   installation of Windows NT, please upgrade this installation using a
   licensed distribution of this product.

RESOLUTION

You must upgrade to a licensed version of Microsoft Windows NT 4.0 Enterprise Server.

There is, however, a specific procedure to be followed to maintain an existing Microsoft Cluster Server Installation.

If the loss of printer shares and folder shares is not an issue, follow the steps outlined in the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q173507
   TITLE     : Evaluation Period Expires on Non-Evaluation Copy of Windows
               NT
If the Cluster Server has share information that cannot be lost, do the following:

  1. Evict one of the nodes from the cluster.

  2. Uninstall Cluster Server on the evicted node.

  3. Follow the directions listed in Knowledge Base article Q173507 on this node.

  4. Complete the "In place upgrade" of a licensed copy of Microsoft Windows NT 4.0 Enterprise Server.

  5. Reinstall Cluster Server, and rejoin the cluster.

  6. Move all the groups and resources from the evaluation node, to the licensed node.

  7. Evict the remaining evaluation node.

  8. Uninstall Cluster Server on the evicted node.

  9. Follow the directions listed in Knowledge Base article Q173507 on this node.

  10. Complete the "In place upgrade" of a licensed copy of Microsoft Windows NT 4.0 Enterprise Server.

  11. Reinstall Cluster Server, and rejoin the cluster.
