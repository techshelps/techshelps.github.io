Wangtek Tape Drives Require Firmware Version 73j or Later

 Last reviewed: December 20, 1997
Article ID: Q102042

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

Wangtek Tape 5525es tape drives require firmware version 73j or later for use with Windows NT. Earlier versions of the firmware failed when tested with Windows NT. For example, firmware version 17G used with NTBackup results in the following error message 

   BACKUP ERROR
   Unable to read the tape in drive 1.
followed by: 

   TAPE DRIVE ERROR DETECTED
   A tape device has been detected, and tape driver started. However,
   the tape device is not responding.
To upgrade your firmware, call Tecmar Technologies Inc. technical support at (800) 992-9916. Select option 3 to speak with a support engineer. You will be asked for your assy number.
