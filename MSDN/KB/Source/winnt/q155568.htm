Telcom Fax 3.0 Fax/Print Service Prevents Adding New Printer

 Last reviewed: March 27, 1997
Article ID: Q155568
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you attempt to add a new printer, you may receive the following error message: 

   Unable to complete operation.
In addition, all installed printers do not print.

CAUSE

The Windows NT printing subsystem has completely changed for version 4.0. Print products designed for Windows NT 3.5x do not work or cause unpredictable failures.

RESOLUTION

Remove Telcom Fax 3.0 from the computer.

Contact the manufacturer of the product for an updated version for Windows NT 4.0.
