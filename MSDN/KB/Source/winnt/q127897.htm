WINSADMN Database Active Column Symbol Descriptions

 Last reviewed: March 25, 1997
Article ID: Q127897
The information in this article applies to:
  • Microsoft Windows NT Server version 3.5
  • Microsoft Windows NT Server version 4.0

The Microsoft Windows NT Server version 3.5 Windows Internet Name Service (WINS) database displays several symbols within the Mappings active column. Each symbol identifies specific conditions about each database. 

   Symbol         Description
   ------         -----------

   Check Mark     Represents an active entry that is registered
                  with a WINS server.

   Minus          Represents an interval between when an entry is
                  marked as ‘released’, no longer active, and when
                  it is marked extinct.

   Cross          Represents an interval between when an entry is
                  marked as extinct, and when it is scavenged
                  (removed) from the WINS database.
