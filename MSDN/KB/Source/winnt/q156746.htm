Print Jobs Are Deleted When Printer Is Resumed After Restart

 Last reviewed: October 31, 1997
Article ID: Q156746
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

Print jobs that are spooled to a paused Microsoft Windows NT printer may be deleted when the printer is resumed after the server has been restarted.

WORKAROUND

This problem has been fixed in Windows NT 4.0 Service Pack 2 and later.

STATUS

Microsoft has confirmed this to be a problem in Microsoft Windows NT version 4.0. This problem was corrected in the latest Microsoft Windows NT 4.0 U.S. Service Pack. For information on obtaining the service pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
