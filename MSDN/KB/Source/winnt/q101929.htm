16-Bit Applications Included with Windows NT

 Last reviewed: May 5, 1997
Article ID: Q101929

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Windows NT ships with several 16-bit applications, which are included for either backwards-compatibility or as utilities.

The following are 16-bit Windows-based applications included with Windows NT: 

   DRWATSON.EXE
   REGEDIT.EXE
   SYSEDIT.EXE
   WINHELP.EXE
   WRITE.EXE
NOTE: Since these are not 32-bit applications, they cannot use long filenames.

The following are MS-DOS-based applications included with Windows NT: 

   EDIT.COM
   EDLIN.EXE
   INTRO.EXE
   QBASIC.EXE
