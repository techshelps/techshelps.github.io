STOP Message Appears With Norton AntiVirus 4.0 for Windows NT

 Last reviewed: March 11, 1998
Article ID: Q173236
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

After you apply Symantec Norton AntiVirus 4.0 beta for Windows NT, the following STOP error may appear: 

   Stop 0x0000001E ( 0xC0000005, 0xFDE38AF9, 0x00000001, 0x7E8B0EB4 )
   KMODE_EXCEPTION_NOT_HANDLED
   Address has Base at - will reference  SYMEVENT.SYS

RESOLUTION

If possible, start the computer into Windows NT, and then uninstall Norton AntiVirus 4.0. Stopping the Norton AntiVirus services may not be sufficient to prevent the problem from recurring, however.

If you are unable to restart the computer in Windows NT and the file system is FAT, start the computer with MS-DOS and rename or delete the following files located in the %SystemRoot%\System32\Drivers folder: 

   Symevent.sys
   Navex.sys
   Navap.sys
If the file system is NTFS, install a parallel copy of Windows NT into a different folder to gain access to the files listed above. After this is done, rename or delete the three files.
