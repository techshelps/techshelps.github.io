Locating Microsoft SNA Server Companion Products Catalog

 Last reviewed: April 11, 1997
Article ID: Q139492

The information in this article applies to:
  • Microsoft SNA Server for Windows NT, versions 2.1, 2.11, and 3.0

SUMMARY

This article describes how to locate the Microsoft SNA Server Companion Products Catalog that contains up-to-date information on hardware and software products from third-party vendors that are compatible with Microsoft SNA Server.

This catalog includes:

  • Third party 3270 and 5250 emulation products.
  • "Middleware" connectivity products, including database connectivity, file transfer, AS/400 connectivity, and administrative functions.
  • Third-party adapter support.

Each SNA Server compact disc includes a hardcopy and softcopy version of the product companion catalog. The softcopy version can be found in the \COLLATRL\GENERAL\ISVCATAL.DOC that is saved in Microsoft Word for Windows version 6.0 format.

Updates to this catalog are not tied to a product revision; changes are made periodically as new products become available.

The latest version of the companion catalog can be retrieved electronically from several Microsoft electronic distribution services listed below, also distributed in Word for Windows version 6.0 format.

For additional information on how to view the Word for Windows files without owning Word for Windows 6.0, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q125420
   TITLE     : What is Word Viewer and How Can I Get It?

MORE INFORMATION

CompuServe

On CompuServe you have two options to view the catalog:

  • Forum: MSSNA Library: 1 Filename: isvcatal.doc

    -or-

    From within Compuserve Information Manager (WinCim), choose the following folders: 

          Computers/Technology
       Software Forums

            Microsoft Connection
         Support Forums and Services
          US Product Support
           Microsoft BackOffice Products
            Microsoft WinNT SNA Forum
             Select Library, Search
              Select MS Info and Index (Library 1)
               Select Search
                Select SNA Server Companion Product Catalog (isvcatal.doc)
                 Retrieve


NOTE: The version of the file posted on this forum is not maintained by
Microsoft, and may not be the latest version of the file.



Internet World Wide Web

 


Location: http://www.microsoft.com


On the Microsoft World Wide Web choose:



   Products/BackOffice/Partner Forum/The BackOffice and Windows NT
   Workstation Directory of Products and Services


If you are on the SNA Server page on the Web (http://www.microsoft.com/sna)
there is also a partner forum box that you can click that also brings you
to the InfoSource Directory.



Internet FTP

 


Address: ftp://ftp.microsoft.com


Note: You must have access to the internet to use FTP.


To find the files using FTP:


  1. Start FTP.


  2. Open a connection by typing:

    

         open ftp.microsoft.com

    

  3. When you are prompted for a user name, type:

    

         anonymous

    

  4. When prompted for the password, type your email name as the password.


  5. To locate the file, type each of the following lines and press Enter:

    

          cd bussys
      cd winnt
      cd sna-docs
      isvcat.exe
