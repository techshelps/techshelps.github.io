List of Documentation Errors in Service Pack 5 Readme.txt File

 Last reviewed: April 11, 1997
Article ID: Q152177
The information in this article applies to:
  • Microsoft Windows NT Server version 3.51
  • Microsoft Windows NT Workstation version 3.51

SUMMARY

This article lists known documentation errors in the Readme.txt file of Windows NT 3.51 Service Pack 5.

MORE INFORMATION

Section 2.2 of the Readme.txt file states that Windows NT 3.51 Service Pack 5 addresses the problem described in the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q142617
   TITLE     : Server Rejects TreeConnectAndX and DirectorySearch SMB
This problem was not corrected in Service Pack 5 for Windows NT 3.51. However, it was corrected in Microsoft Windows NT version 4.0. For more information, contact Microsoft Technical Support.
