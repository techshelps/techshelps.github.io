APC PnP UPS Cable for Windows 95 Does Not Work with Windows NT

 Last reviewed: February 28, 1996
Article ID: Q141384
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

SUMMARY

American Power Conversion (APC) makes a cable specific to their Plug-N-Play UPS units that does not function with their Windows NT software or the UPS software included in Windows NT.

MORE INFORMATION

APC's Plug-N-Play UPS only uses custom serial interface cable 940-0095, from APC, for the Windows 95 drivers. This cable is gray in color. Windows NT is able to use APC custom serial interface cables 940-0024B or 940-0024C. Both of these cables are black in color.

Reference Chapter 1: Installing PowerChute Pro, page 3, or the PowerChute Pro User's Guide for additional information.
