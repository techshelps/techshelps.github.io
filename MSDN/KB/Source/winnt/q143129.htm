Location of Windows NT Server Tools for Windows 95

 Last reviewed: March 7, 1998
Article ID: Q143129
The information in this article applies to:
  • Microsoft Windows NT Server versions 3.5 and 3.51
  • Microsoft Windows NT Resource Kit version 3.51
  • Microsoft Windows 95

SUMMARY

The Windows NT Server Tools for Windows 95 is available in the following locations:

  • Windows NT Resource Kit version 3.51. Go to the Win95Admin directory and follow the instructions in SRVTOOLS.TXT.
  • Microsoft World Wide Web site. Go to:

http://www.microsoft.com/ithome/resource/technet/windows/win95/nexus/nexus. htm
