Administrators Granted Connection with No License Available

 Last reviewed: June 6, 1995
Article ID: Q130933
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

Administrators are granted a licensed connection to a server running in the Per Server mode even though the new connection will exceed the number of available licenses on that server. This is to ensure that administrators can access servers at all times.

NOTE: If a normal user is connected to the server and they close their connection, another connection will not be issued until a legal license is freed.
