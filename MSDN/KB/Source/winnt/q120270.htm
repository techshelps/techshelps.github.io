Win32 DDK/SDK Don't Contain What is Necessary to Create an IFS

 Last reviewed: November 12, 1997
Article ID: Q120270
3.10 WINDOWS kbprg

The information in this article applies to:

  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Workstation, version 3.X, and 4.0
  • Microsoft Windows NT Advanced Server, version 3.X, and 4.0

Even if you have the Platform SDK and DDK, you do not have everything you need to write an Installable File System (IFS) for Windows NT. For more information on the Windows NT IFS kit see: 

 http://www.microsoft.com/hwdev/ntifskit/
