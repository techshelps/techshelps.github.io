Nikon CoolScan Scanner May Not Work Under Windows NT

 Last reviewed: May 9, 1997
Article ID: Q104664

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

According to Nikon Electronic Imaging, the software that currently comes with the Nikon CoolScan only supports Windows 3.1 and may not enable the scanner to function properly under Windows NT. The scanner also comes with a version of Adobe PhotoShop which also may not work under Windows NT.

When you install Windows NT on a computer that has a Nikon CoolScan connected to a SCSI adapter, the SCSI device may be identified during setup. However, when you later try to run the scanning software, you may have difficulty identifying the device.

For additional information, contact Nikon Electronic Imaging at (516) 547-4311.
