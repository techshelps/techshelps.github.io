Err Msg: Dynamic Link Library NWAPI32.DLL Could Not Be Found

 Last reviewed: May 13, 1997
Article ID: Q109086
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

If the Novell NetWare Requestor has been incorrectly installed or removed from a Windows NT installation, when you open Control Panel, the following error message may occur: 

   CONTROL.EXE - unable to locate DLL

   The dynamic link library NWAPI32.DLL could not be found in the specified
   path.

CAUSE

This problem is caused by the Novell Control Panel file (NOVELL.CPL) still being in the <Windows NT directory>\system32 directory.

RESOLUTION

To work around this problem, close Control Panel and use either the File Manager or a command prompt (CMD.EXE) to delete the NOVELL.CPL file.
