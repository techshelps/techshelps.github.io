Duplex Printing Causes Problems After Applying SP4

 Last reviewed: September 7, 1996
Article ID: Q153202
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.51 Service Pack 4
  • Microsoft Windows NT Server versions 3.51 Service Pack 4

SYMPTOMS

After applying Service Pack 4 (SP4) for Windows NT 3.51, you may get incorrect printouts when printing in duplex mode.

CAUSE

SP4 comes with a new version of Rasdd.dll. This DLL causes the problem.

RESOLUTION

Apply the fix mentioned below.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.51 Service Pack 4. This problem was corrected in the latest Windows NT 3.51 U.S. Service Pack. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
