Blank Screen on Orchid Fahrenheit VA/VLB

 Last reviewed: May 8, 1997
Article ID: Q101956

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

After starting a Command Prompt, switching to full screen, and then switching back to a window, the screen goes blank.

CAUSE

This is a timing problem. This does not happen on some VESA local bus computers. One PS/2 Valuepoint computer exhibits this problem.
