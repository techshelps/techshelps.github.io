Unable to Connect Using Cabletron EISA F70XX FDDI NIC

 Last reviewed: October 12, 1995
Article ID: Q135777
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

SYMPTOMS

If you install both NetBEUI and TCP/IP under Windows NT on a AST Manhattan P Series 5090 (dual processor) server computer using a Cabletron EISA F70XX FDDI network interface card (NIC), client computers using the NetBEUI protocol are unable to connect to the server.

RESOLUTION

To correct this problem:

  • Download F70NT35.ZIP from the Cabletron Bulletin Board Service (BBS) at (603) 335-3358 and install the updated Cabletron EISA F70XX FDDI NIC driver.

    -or-

  • Upgrade to Windows NT 3.51 (if you have not already done so) and install the latest U.S. Service Pack for Windows NT version 3.51.

STATUS

Microsoft has confirmed this to be a problem in Windows NT versions 3.5 and 3.51. This problem has been corrected in the latest U.S. Service Pack for Windows NT version 3.51. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces):
