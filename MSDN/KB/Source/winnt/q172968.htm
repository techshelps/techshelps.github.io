Disk Subsystem Recovery Documentation Error

 Last reviewed: March 20, 1998
Article ID: Q172968
The information in this article applies to:
  • Microsoft Windows NT Server, Enterprise Edition version 4.0 - Microsoft Cluster Server

SUMMARY

The Microsoft Cluster Server (MSCS) Administrator's Guide contains a section that discusses recovery from disk subsystem failures. The information contained in the section may not pertain to version 1.0 of the product.

MORE INFORMATION

Chapter 4, "Managing MSCS," in the MSCS Administrator's Guide contains a section called "Recovering Disk-Subsystem Failures." Because MSCS version 1.0 does not support software RAID, this section may not apply.

Recovery procedures for hardware RAID may be specific to the RAID implementation. For these procedures, consult the documentation provided with your hardware, or consult with the hardware vendor.
