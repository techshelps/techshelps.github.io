Err Msg: DLC Protocol ver 3.1 Already Exists...

 Last reviewed: August 29, 1996
Article ID: Q119684
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you attempt to remove the DLC protocol from the Network option of Control Panel, the following message may appear: 

   DLC protocol ver 3.1 already exists on the system.

WORKAROUND

To work around this problem:

  1. Rename the existing DLC.SYS to another filename located in the %systemroot%\SYSTEM32\DRIVERS subdirectory.

  2. Expand the DLC.SY_ file to DLC.SYS from the Windows NT disk set or CD to the %systemroot%\SYSTEM32\DRIVERS subdirectory.

    NOTE: If the system has been upgraded to Service Pack 2 (Build 528), 

             expand the DLC.SY_ from the Service Pack 2 disk set or CD.

  3. Restart your computer.
