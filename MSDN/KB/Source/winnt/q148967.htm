Err Msg: "This Request Not Accepted by Network" on WFWG Client

 Last reviewed: March 26, 1997
Article ID: Q148967
The information in this article applies to:
  • Microsoft Windows NT Server version 3.51
  • Microsoft Windows for Workgroups version 3.11
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

If the per-server licensing has connections set to zero on a Windows NT 3.51 server, a Windows for Workgroups client trying to connect to a share or print to a printer on the server may receive the following error: 

   ERROR 71: This request is not accepted by the network.
An administrator can connect or print while other users cannot.

RESOLUTION

Set the connection limit in Control Panel Licensing to a number other than zero.
