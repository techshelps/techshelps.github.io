Turbo Modem Plus Modem Will Not Auto-answer

 Last reviewed: May 14, 1997
Article ID: Q110561
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

The Turbo Modem Plus is not on the Windows NT Hardware Compatibility List and is not supported. However, the Turbo Modem Plus may be compatible when it is set up as a Hayes 14400 Ultra; except when it used as a Remote Access Service (RAS) Server, the modem may not auto-answer incoming calls.

WORKAROUND

A possible workaround to this problem is to disable error-checking in RAS, which is on by default.

To disable error-checking in RAS, do the following:

  1. From the Control Panel windows, choose the Networks icon.

  2. Select Remote Access Service, and then choose the Configure button.

  3. Select the modem, and then choose the Settings button.

  4. Clear the Enable Error Checking check box, and then choose the OK button.

If this does not correct the problem, contact the manufacturer of the Turbo Modem Plus.
