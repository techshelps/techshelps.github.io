Eicon S51 Adapter Supported in Windows NT 3.51

 Last reviewed: March 12, 1997
Article ID: Q141584
3.51 WINDOWS kb3rparty kbhw

The information in this article applies to:

  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

The Eicon S51 wide-area communications coprocessor card for ISA-bus based workstations is supported in Windows NT 3.51. Although this adapter is not included on the Windows NT 3.51 Hardware Compatibility List (HCL), this adapter was tested by Microsoft and is considered compatible with Windows NT 3.51. The Eicon S51 adapter will be included on the next Windows NT HCL.

The S51 is manufactured by Eicon, a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
