Formats that Windows NT Backup Reads from and Writes to

 Last reviewed: May 9, 1997
Article ID: Q104167
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Backup reads the following tape formats:

  1. Microsoft Tape Format v1.0 (MTF) or Maynard Tape Format Specification v1.03.
2. "OS/2 tape format" (Sytos Plus). 3. Archive tape format. 4. Conner tape format.

Windows NT Backup should be able to restore information from tapes that were backed up using any of the above formats.

Windows NT Backup only writes to tapes created using Microsoft Tape Format version 1.0. It can only perform backups (including appends) to tapes using this format.
