Third-Party Undelete Utility for Windows NT 3.51

 Last reviewed: May 3, 1996
Article ID: Q149782
The information in this article applies to:
  • Microsoft Windows NT Workstation 3.51
  • Microsoft Windows NT Server 3.51

SUMMARY

Windows NT does not have an "undelete" command or feature. This article discusses the undelete utility for Windows NT from Executive Software, Inc.

MORE INFORMATION

Executive Software has created a utility called "UNDELETE UTILITY FOR WINDOWS NT 3.51". This utility will undelete files from a Windows NT 3.51 system when the Executive Software DISKEEPER defragmentation utility is installed. DISKEEPER must be installed since the undelete utility uses DISKEEPER's file-handling mechanisms to retrieve deleted files.
