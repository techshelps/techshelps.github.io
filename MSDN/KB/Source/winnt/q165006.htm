DHCP Event 1020

 Last reviewed: December 9, 1997
Article ID: Q165006
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.51 and 4.0
  • Microsoft Windows NT Server versions 3.51 and 4.0

SUMMARY

DHCP logs the following Event in Event Viewer: 

   1020   Type         Warning              Source       DhcpServer
In this case, the scope aaa.bbb.ccc.0 is 80 percent used. This Event is provided to alert the DHCP Server administrator that their scope is close to becoming fully used, so that they can take action before running out of addresses.

MORE INFORMATION

There is no means to customize this threshold for this event through the registry because a value of 80% is hardcoded. There may be higher percentages displayed because the check against this value happens only when the DHCP database is scavenged.
