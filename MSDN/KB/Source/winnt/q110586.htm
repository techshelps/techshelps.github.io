Err Msg: I/O Operation Timed Out...with Pioneer DRM-600/604x

 Last reviewed: May 14, 1997
Article ID: Q110586
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you are using a Pioneer DRM-600 or DRM-604x reading from two or more CD-ROMs, a time-out I/O error may occur, generating a message similar to one of the following: 

   I/O operation timed out.
-or- 

   Requested source is in use.
This error results with your keyboard, and possibly your entire system, being locked. This is particularly a problem when the CD-ROM drive is a shared device on a network.

CAUSE

The AHA-1640 and UltraStor 34f support only a single disk with the Pioneer DRM-600 or DRM-604x, and the Future Domain MCS-600 is not supported at all with the Pioneer DRM-600 or DRM-604x. This information is included in the Windows NT Hardware Compatibility List (HCL).

STATUS

Microsoft has confirmed this to be a problem in Windows NT and Windows NT Advanced Server version 3.1. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
