Windows NT Takes Long Time to Start with Aztech CD-ROM Driver

 Last reviewed: July 27, 1995
Article ID: Q128608
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

After you download and install the Aztech CDA 268-01A CD-ROM driver (CDA_NT.101) from the Aztech Labs bulletin board service (BBS), Windows NT takes a long time to start.

According to Aztech Labs Technical Support, this is by design. For more information, contact Aztech Labs Technical Support at (800) 886-8879. The Aztech Labs BBS number is (510) 623-8933.

The Aztech product discussed here is manufactured by Aztech Labs, Inc., a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.
