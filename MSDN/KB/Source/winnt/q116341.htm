Characters Change When Pasted into 16-bit Applications

 Last reviewed: May 15, 1997
Article ID: Q116341

The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

When you paste certain characters into 16-bit applications, they become different characters. For example, using the English locale, if you copy the trademark symbol, and attempt to paste it into Word for Windows, it is pasted as an exclamation mark (!).

STATUS

Microsoft has confirmed this to be a problem in Windows NT Workstation and Windows NT Server version 3.5. This problem was corrected in the latest U.S. Service Pack for Windows NT version 3.5. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
