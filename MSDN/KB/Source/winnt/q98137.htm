Novell Drivers for Windows NT

 Last reviewed: May 6, 1997
Article ID: Q98137
The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1

Novell drivers for Windows NT are available from Novell.

To obtain the beta version of Novell's NetWare Requester for Windows NT, do the following:

  1. Connect to CompuServe.

  2. Type GO NOVFILES.

  3. Select Client Kits.

  4. Select Windows NT Versions.

Support for the NetWare redirector is handled by Novell through the NDSG forum on CompuServe. Or you can call Novell at (800) 638-9273.
