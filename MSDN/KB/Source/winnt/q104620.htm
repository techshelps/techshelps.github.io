Returning to Boot Loader After Choosing MS-DOS

 Last reviewed: May 9, 1997
Article ID: Q104620
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you choose MS-DOS from the Boot Loader menu, you are returned to the menu. If you choose MS-DOS again, MS-DOS starts normally.

CAUSE

This behavior is caused by QEMM 7.x DOS-UP.

RESOLUTION

To correct this problem change the DOS-DATA.SYS line in the CONFIG.SYS file to: 

    DEVICE=DOS-DATA.SYS /BOOTFILE=BOOTSECT.DOS
NOTE: This parameter will be stripped out of the CONFIG.SYS file if you run Optimize again.

-OR-

Disable the DOS-UP feature by removing the DEVICE=DOS-DATA.SYS line from the CONFIG.SYS file.
