Cannot Install Sidekick 95 in Windows NT 4.0

 Last reviewed: March 27, 1997
Article ID: Q155957
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

If you attempt to install Sidekick 95 in Windows NT version 4.0, you may receive an error message that indicates you must be running Windows NT version 3.51 or later.

CAUSE

Sidekick 95 performs a version check and may incorrectly detect Windows NT version 4.0 as a version earlier than Windows NT 3.51.

RESOLUTION

Contact Starfish Software Technical Support for an updated version of Sidekick 95 that corrects this problem.
