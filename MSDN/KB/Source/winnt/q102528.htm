TCPTSR's 22 TCP Connection Limit Requires Small Windows

 Last reviewed: May 8, 1997
Article ID: Q102528
The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

TCPTSR can support up to 22 TCP connections, but that maximum is attainable only with small window sizes.
