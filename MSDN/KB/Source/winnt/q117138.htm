Printing Direct with TCP to HP JetDirect Requires LPD Support

 Last reviewed: March 25, 1997
Article ID: Q117138
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5 and 3.51
  • Microsoft Windows NT Server version 3.5 and 3.51
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SUMMARY

In order for Windows NT version 3.5 Workstation or Server to print over the network to a Hewlett-Packard (HP) JetDirect card using TCP/IP without a print server acting as an intermediary, the JetDirect card must support Line Printer Daemon (LPD).

MORE INFORMATION

Only the latest versions of the HP JetDirect card support LPD. In order for an HP JetDirect card to support LPD, version A.03.03 or later of Hewlett- Packard's firmware is required.

If the JetDirect card does not have this version or later of the firmware, it cannot support LPD and you cannot print directly to it from Windows NT 3.5. The firmware in these cards cannot be upgraded.

For additional information, contact Hewlett-Packard technical support at (208) 323-2551.
