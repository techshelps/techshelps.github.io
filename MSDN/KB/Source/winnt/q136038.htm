Diskeeper 1.0 Compatibility In Windows NT 3.51

 Last reviewed: April 30, 1996
Article ID: Q136038
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

Diskeeper 1.0, a third-party on-line disk Defragmentor for Windows NT, does not work under Windows NT 3.51. It does not have the capability to support the newer features of the Windows NT file system (NTFS) in 3.51 nor does it support the new file and directory compression feature.

A Windows NT 3.51 compatible version of Diskeeper, version 1.01 is available on Executive Software's on-line services. The incremental release (DK101I.EXE) is available free to all registered users. Executive Software's on-line services:
