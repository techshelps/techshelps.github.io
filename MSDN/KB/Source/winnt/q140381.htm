Evaluation Copies of SWEEP On July 1995 TechNet Disk

 Last reviewed: January 12, 1996
Article ID: Q140381
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51 -------------------------------------------------------------------------

SUMMARY

The November 1995 issue of "Exploring Windows NT" Vol. 3 No.11 page 15 states that "evaluation copies of SWEEP for Windows NT are available on the Microsoft TechNet Disk 2 under the directory SAMPLES\BOES\BO\WINNTAS\YPAGES\SOPHOS.

Some customers assume that this evaluation copy of SWEEP is on the November 1995 TechNet when it is actually on the July 1995 TechNet.

MORE INFORMATION
