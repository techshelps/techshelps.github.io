NWCONV Migrates NetWare Server to Windows NT Domain Controller

 Last reviewed: September 30, 1997
Article ID: Q137397
3.51 4.00 WINDOWS kbnetwork

The information in this article applies to:

  • Microsoft Windows NT Server version 3.51 and 4.0
  • Microsoft File and Print Services for NetWare, version 3.51 and 4.0

The Migration Tool for NetWare (NWCONV.EXE) version 1.1 included with Microsoft File and Print Services for NetWare (FPNW) 3.51 does not allow migration of a NetWare server to a non-domain controller Windows NT Server.

If you do attempt to migrate a NetWare server to a non-domain controller Windows NT server, you may receive the following message during migration: 

   “<ntservername> is not a server”

WORKAROUND

Perform the migration of a NetWare server to a Windows NT Server running as a primary domain controller (PDC) or a backup domain controller (BDC).
