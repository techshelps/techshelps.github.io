Err Msg: The Version of Windows for Workgroups You Are...

 Last reviewed: May 12, 1997
Article ID: Q106052

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you install Microsoft Access database 1.1 upgrade on a system running Windows NT version 3.1 that had previously been running Windows for Workgroups, you may receive the following error message: 

   The version of Windows for Workgroups you are using must be upgraded
   before Access can be installed.

CAUSE

Microsoft Access requires an updated WFWNET.DRV file when it runs with Windows for Workgroups. An updated driver is shipped with the product and is on the last disk.

RESOLUTION

To resolve this problem, rename or remove the file WFWNET.DRV in the SYSTEM subdirectory. It is not required when Microsoft Access runs under Window NT. However, if you are dual-booting to Windows for Workgroups, you need to run the update from the last Microsoft Access disk.
