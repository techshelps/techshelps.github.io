Windows NT Control Panel: DEC DE422 Network Adapter Not Listed

 Last reviewed: May 10, 1995
Article ID: Q130011
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

The Windows NT Hardware Compatibility List (HCL) lists the following adapter as supported, but it is missing from the Windows NT Control Panel network card driver list: 

   DEC DE422 EtherWORKS EISA TP/BNC
To install the correct driver for this card select the following network card driver in Windows NT Control Panel: 

   DEC EtherWORKS Turbo EISA Adapter
NOTE: This choice appears in Control Panel only when you run Windows NT on an EISA bus computer.
