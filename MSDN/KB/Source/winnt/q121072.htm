Stop 0x0000006B Installing Windows NT on Toshiba T6600c

 Last reviewed: July 25, 1995
Article ID: Q121072
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

While installing Windows NT on a Toshiba T6600c Multimedia Computer, you may encounter this error: 

   STOP 0x0000006B
   (PROCESS1_INITIALIZATION_FAILED)
The Toshiba T6600c Multimedia Computer is on the Hardware Compatibility list but for hard drives larger than 500M it requires that you have BIOS version 1.23 or later, and that you set the hard drive selection to Standard in the Toshiba setup program.

To change the hard drive selection, run the Toshiba setup program and go to Hard Drive. Select the option: 504 standard. You may have to restart the Windows NT Setup program afterwards.

If the BIOS version is earlier than 1.23, the hard drive options are not available and you must obtain a BIOS upgrade from Toshiba.
