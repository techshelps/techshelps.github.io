COPS Network Adapter Card Uses DayStar Driver

 Last reviewed: May 9, 1997
Article ID: Q103658
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

When installing the Co-Operative Printing Solutions (COPS) LTI ISA network adapter, you must use the DayStar Digital LocalTalk Adapter driver.

MORE INFORMATION

The COPS LTI ISA network adapter, which is on the Windows NT Hardware Compatibility List, does not have a corresponding COPS driver entry in the network adapter section appearing during the network configuration portion of setup, or in the Network option of Control Panel. This is because DayStar Digital sold its LocalTalk adapter to COPS, which has changed the name. According to COPS, the adapter is unchanged except for its name.

COPS can be reached at (404) 840-0810.
