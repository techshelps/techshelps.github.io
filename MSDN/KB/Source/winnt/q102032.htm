Windows NT: Bookshelf 1992 Edition Doesn't Install Works Demo

 Last reviewed: May 8, 1997
Article ID: Q102032

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

Microsoft Bookshelf 1992 Edition installs a demo of Microsoft Works. If you are running Windows NT on a RISC-based computer, the demo is not installed.

CAUSE

Microsoft Bookshelf 1992 Edition Setup checks for the type of processor using the GetWinFlags application program interface (API) function. If the processor is not a 386 (according to GetWinFlags), Bookshelf 1992 Edition does not install the demo. GetWinFlags under Windows NT on a RISC-based computer returns 286.

NOTE: Bookshelf 1992 Edition does not determine whether Windows is running in enhanced mode. On a 386 running standard mode, Bookshelf 1992 Edition still installs the demo.
