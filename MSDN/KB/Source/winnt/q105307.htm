NCR Model 3450 or Model 3550 Processor Support in Windows NT

 Last reviewed: May 12, 1997
Article ID: Q105307

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

In order for Windows NT to use more than one microprocessor with the NCR System 3000 Model 3450 or Model 3550, an updated Hardware Abstraction Layer (HAL) from NCR Corporation must be installed. Windows NT will operate with these computers with more than one processor installed, but it will only use one processor.

NCR Models 3450 and 3550 are listed as tested multiprocessor systems and appear on page 18 of the Windows NT Hardware Compatibility List under "x86 Architecture Multiprocessor Computers."

To obtain the updated Hardware Abstraction Layer, call NCR at (800) GO-WINNT (469-4668).

NCR provides support for models 3450 and 3550 and the updated Hardware Abstraction Layer.
