Print Wizard Does Not Display Printers

 Last reviewed: October 22, 1997
Article ID: Q172228
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

When you add a printer using the Add Printer Wizard, there are no printers on the printer list to choose from.

CAUSE

The file %systemroot%\inf\NTPRINT.INF is missing or corrupt.

RESOLUTION

  1. Delete the files %SystemRoot%\Inf\Ntprint.inf and %SystemRoot%\Inf\Ntprint.pnf.

  2. Expand the file Ntprint.inf from the original distribution CD or service pack CD to the %SystemRoot%\Inf directory.

(expand %Sharepoint%\i386\Ntprint.in_ to %SystemRoot%\Inf\Ntprint.inf)

NOTE: Ntprint.pnf will be created automatically the next time you launch the print wizard.
