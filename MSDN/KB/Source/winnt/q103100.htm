Bindings List Blank when a Network Adapter is Selected

 Last reviewed: May 8, 1997
Article ID: Q103100
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

From Control Panel, in the Network Bindings option, when you select a Network Interface Card (NIC) from the Show Bindings For drop down list box, the Bindings list is empty.

This behavior is by design. The Bindings dialog box is designed to show the bindings below the component selected in the Show Bindings For drop down list box. Because an adapter has no bindings downward, there is nothing to show. If you select the Driver for the adapter, you will see a binding from the driver to the adapter.
