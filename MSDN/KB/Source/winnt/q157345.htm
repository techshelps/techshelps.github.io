No Color Output with HP PaintJet XL300

 Last reviewed: March 20, 1997
Article ID: Q157345
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you print a color image from a program in Windows NT 4.0 to a Hewlett-Packard PaintJet XL300 printer, the output is produced in black and white only.

  • or -

Colors fail to print on an HP PaintJet XL300. Only black prints through all applications, on both graphics and text. Anything that is not black will be blank.

STATUS

A new version of this printer driver for Windows NT 4.0 is now available from the Windows NT Download Library.

For additional information on this driver, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q142643
   TITLE     : Windows NT 4.0 Driver Library
