DPT 2011 with Cache Modules Does Not Support Removable Media

 Last reviewed: May 8, 1997
Article ID: Q102541

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

Removable media attached to a DPT 2011 SCSI controller do not properly handle media changes if you are using the DPT cache module. For example, using the DIR command on a removable media drive after changing the media may show the previous volume's information.

CAUSE

The host bus adapter (HBA) is not returning Test Unit Ready commands so media changes are ignored.

RESOLUTION

To work around this problem, do not attach removable media devices to the DPT 2011 SCSI controller card, do not change removable media, or remove the cache modules.
