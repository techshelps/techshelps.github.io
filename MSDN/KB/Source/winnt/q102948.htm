Incorrect RAS Statement in Concepts and Planning Guide

 Last reviewed: May 8, 1997
Article ID: Q102948
The following information applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Page 6 of the "Concepts and Planning Guide" for Windows NT Advanced Server states in the second paragraph: 

   Two other components, Windows NT Services for Macintosh and
   Windows NT Remote Access Server, are included with Windows NT
   Advanced Server. These components are not available on the
   Windows NT workstations.
This statement is incorrect. The Windows NT workstation does ship with RAS connectivity, but not Services for Macintosh.
