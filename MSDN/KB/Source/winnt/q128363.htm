Aztech Nova Sound Board Support in Windows NT

 Last reviewed: March 31, 1995
Article ID: Q128363
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

The Aztech Nova (MMCD16W) sound board provides audio functionality and a CD- ROM interface for Windows NT. The sound board supports the following CD-ROM devices:

  • Aztech CDA 268-01A (Firmware 1.8)
  • Sony CDU 33a
  • Mitsumi FX001
  • Panasonic CD-5xx

The sound board audio features are compatible with the Sound Blaster 1.x driver that ships with Windows NT. However, for full audio and CD-ROM support, obtain drivers from the Aztech Labs Bulletin Board Service (BBS) or FTP server. The BBS number is (510) 623-8933. The FTP server address is FTP.NETCOM.COM (\pub\az\aztech). Download NOVA_NT.101 (Audio support driver) and CDA_NT.101 (Support for Aztech CD-ROM). For more information, contact Aztech Labs Technical Support at (800) 886-8879.
