Case Studies Available Describing Windows NT 4.0 Deployments

 Last reviewed: February 11, 1998
Article ID: Q155643
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0

SUMMARY

Several detailed case studies describing corporate deployments of Windows NT Workstation version 4.0 are available for study on the World Wide Web.

MORE INFORMATION

The "Case Studies Featuring Microsoft Windows NT Workstation" Web page provides access to several in-depth discussions of the successful deployment of Windows NT 4.0. Deployments at large and medium-sized companies are discussed.

This page is available at the following location: 

   http://www.microsoft.com/corporate_solutions/cases/cases-ntwork.htm
