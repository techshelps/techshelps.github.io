Cannot Cancel Location Information

 Last reviewed: August 13, 1997
Article ID: Q155021
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SUMMARY

When you start the Telephony tool in Control Panel, you cannot cancel the Location Information dialog box that appears.

MORE INFORMATION

You must enter an area code to close this dialog box. If you are unsure what number to enter, enter any number and then correct the number later when you know what number to use.

Once you have entered an area code and closed the dialog box, you can start the Telephony tool and close it with a blank area code field.
