Ntdetect Message Appears Twice During System Startup

 Last reviewed: March 28, 1997
Article ID: Q158185
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation versions 3.5, 3.51, and 4.0
  • Microsoft Windows NT Server versions 3.5, 3.51, and 4.0

SYMPTOMS

After you select a Windows NT operating system to start in the OS menu, the following message appears: 

   NTDETECT V1.0 Checking Hardware ...

   NTDETECT V1.0 Checking Hardware ...
The system does not continue the startup process. Normally, only one instance of the message is displayed, but in this cause, it is displayed twice.

CAUSE

This message is caused by a missing or corrupt Ntdetect.com.

RESOLUTION

Copy Ntdetect.com from the most recently installed service pack, or from the Windows NT Setup boot disk if no service packs have been installed.
