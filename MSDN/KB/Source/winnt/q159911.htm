Firmware Requirements for LexMark MarkVision Monitor

 Last reviewed: March 28, 1997
Article ID: Q159911
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SUMMARY

Lexmark MarkVision is a Windows program that provides printer configuration and status information, and gives you control over your Lexmark laser printers by using the bidirectional communication capability of the printer. MarkVision works with printers connected to your workstation through TCP/IP and DLC, and with printers on the Microsoft Network.

MORE INFORMATION

To use MarkVision in a Windows NT environment, it is important to have the correct firmware revision to ensure proper results. Please refer to the table below for more information:

The firmware mentioned is the minimum requirement. 

Model                   Firmware Required       Protocols Supported
-----                   -----------------       -------------------

MarkNet XL              15.62.01                LexLink, TCP/IP, NetWare
                                                Appletalk.

MarkNet External XLe    03.51.01                Netware, Appletalk, and
                                                TCP/IP

MarkNet (Small Card)    07.62.01                Lexlink , NetWare, and
                                                Appletalk

MarkNet (Small Card)     09.62.01**             Lexlink and TCP/IP
Updated firmware may be obtained at the following web site: 

   http://www.lexmark.com/service/service.html
Select the Drivers and Utilities option, and follow the instructions.
