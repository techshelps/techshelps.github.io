Archive 2150s Gives "Fatal Error on Drive #1" with Verify

 Last reviewed: May 9, 1997
Article ID: Q103764
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

The Archive 2150S tape drive will not perform a backup when the Verify option is selected. Instead, you receive the error message "Fatal Error on Drive #1." You can successfully backup and restore files if this option is not selected.

CAUSE

The tape backup system doesn't have the latest firmware.

RESOLUTION

Contact Archive for a new version of the firmware. The current version of the firmware that has problems is 21247-005.

Temporary Workaround

Log in under an account that can perform backups. Make sure that the Windows NT Backup program has been run once under this account. Under this account start REGEDT32. Change the value for HKEY_CURRENT_USER\ Software\Microsoft\Ntbackup\Backup Engine\Use Fast File Restore: REG_SZ from 1 to 0.

NOTE: There is a drawback to this workaround. If you try to restore a file, Backup will search the whole tape for the file instead of going right to the file.
