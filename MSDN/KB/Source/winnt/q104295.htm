Differences Between Windows NT Backup and Conner's Backup

 Last reviewed: May 9, 1997
Article ID: Q104295

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Windows NT Backup, written by Conner Software, provides the necessary tools for simple data backups and restores using a local tape drive.

The following list are features of the Conner Backup program not included in Windows NT Backup:

  • Backup all Windows NT servers on the network (or Internet)
  • Backup of remote Registry databases
  • Unattended and scheduled backup
  • Jobs and scripts
  • Tape to disk compare
  • Tape password security
  • Tape catalog on disk
  • Tape catalog search with file histories
  • Disk grooming and file migration
  • Backup or automatic retry of open files
  • Advanced file selection
  • Run multiple instances of the application
  • E-Mail of unattended job results
  • Support for connecting and disconnecting network drives

For additional information, contact Conner technical support at (800) 426-6637.
