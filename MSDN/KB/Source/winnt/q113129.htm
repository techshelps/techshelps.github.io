Cannot Use Parallel Port on Trantor T-348 MiniSCSI

 Last reviewed: May 14, 1997
Article ID: Q113129
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The Trantor T-348 device connects to the parallel adapter, providing an SCSI port and an output parallel port. The Windows NT Trantor T-348 adapter driver, however, does not support simultaneous use of the parallel and T-348 ports, so when the driver is loaded you cannot use the parallel port.
