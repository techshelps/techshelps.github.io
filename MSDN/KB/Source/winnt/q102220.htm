Pull-down Menus Disable the Screen Saver

 Last reviewed: May 8, 1997
Article ID: Q102220

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

The Screen Saver utility in Windows NT will not be activated if a pull-down menu is open. This behavior is by design.

Steps to Reproduce Problem

  1. In Windows Control Panel, choose the Desktop icon, set the screen saver to 1 minute, and choose OK.

  2. In Program Manager, press ALT+F or ALT+D to activate the File menu.

Note that even after one minute, the screen has not been activated.

  1. Close the File menu.

  2. After one minute, the Screen Saver will be activated.
