Err Msg: Error 720: No PPP Control Protocols Configured

 Last reviewed: November 19, 1997
Article ID: Q122748

The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5, 3.51, and 4.0
  • Microsoft Windows NT Server versions 3.5, 3.51, and 4.0

SYMPTOMS

When you attempt to connect to a Remote Access Service (RAS) Server using the TCP/IP protocol, you may receive the following error message: 

   Error 720: No PPP control protocols configured.

CAUSE

The RAS Server to which you are dialing does not have the TCP/IP protocol installed or configured for RAS.

RESOLUTION

To resolve this issue you will need to edit your phone book entry to use whatever protocol the RAS Server is using.
