Starting from WinNT 4.0 CD-ROM Using an Adaptec AHA-2940UW

 Last reviewed: December 18, 1997
Article ID: Q165960
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

If you have the Adaptec 2940UW BIOS setup to boot from CD-ROM and try to Boot from the Windows NT 4.0 CD-ROM you may receive one of the following error messages.

NOTE: Different Adaptec BIOS versions results in different error 

      messages:
Trying to boot from a 2940UW BIOS Version 1.22s results in: 

   CD-ROM became Drive C: ,Hard Drive became Drive D:

   Error: "MISSING OPERATING SYSTEM"
Trying to boot from a 2940UW BIOS version 1.23 results in: 

   CD-ROM does not become DRIVE C:

   Error: "BOOT: Memory overflow error Please insert another disk."

RESOLUTION

To resolve this problem, update the BIOS to revision 1.25 or later. Please contact Adaptec for more information.
