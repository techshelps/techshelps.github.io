Jetpack Error Codes for Windows NT 4.0

 Last reviewed: December 16, 1997
Article ID: Q172570
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0

SUMMARY

This article lists the error codes and messages used by Jetpack.exe (the Microsoft Windows NT Server utility that can be used to compact a WINS or DHCP database) for Windows NT Server version 4.0.

The following error code ranges are reserved for external use. As is true for Jet error codes, these ranges cover the negative as well as positive form of the numbers in the range. 

  30000 through 30999 for use by Vt Object as defined in JETEB.H
  32000 through 32767 for use by Import
