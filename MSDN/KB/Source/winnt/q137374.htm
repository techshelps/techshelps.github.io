Devices Attached to Adaptec 2740 Not Recognized

 Last reviewed: September 27, 1995
Article ID: Q137374
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SYMPTOMS

When you add 256 megabytes (MB) or more of RAM memory to your Windows NT 3.5 computer, devices attached to your secondary Adaptec 2740 (Rev. C) SCSI controller are no longer recognized, and the Event Viewer shows that Driver Arrow (Adaptec 2740 SCSI driver) and Physical Memory have claimed two overlapping memory regions.

RESOLUTION

This has been fixed in Windows NT version 3.51.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.5. This problem has been corrected in Windows NT Workstation and Windows NT Server version 3.51.
