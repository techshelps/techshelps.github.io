Error: Could Not Add Port: The Printer Already Exists.

 Last reviewed: March 27, 1997
Article ID: Q155235
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5, 3.51, and 4.0
  • Microsoft Windows NT Server versions 3.5, 3.51, and 4.0

SUMMARY

While you attempt to create an LPR printer on Windows NT, one of the following error messages may appear:

Windows NT 3.5x

   Could not add port: The printer already exists.

Windows NT 4.0

   Specified Port cannot be added. The printer already exists.

MORE INFORMATION

The TCP/IP printer port address you are attempting to create was already created, possibly from a previously created and then deleted printer. The IP address remains in the registry to simplify adding future printers.

RESOLUTION

Check to see if the port exists before trying to add an LPR port.

In Windows NT 3.5x, existing LPR ports will be listed at the top of the "Print to:" list box in Print Manager.

In Windows NT 4.0, existing LPR ports will be listed at the top of the "Available Ports:" list in the Ports tab of the Printer Properties.
