Err Msg Accessing Netware or Compatible Network: Network Request

 Last reviewed: March 21, 1997
Article ID: Q136665 
 The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.51
  • Microsoft Windows NT Server version 3.51

    SYMPTOMS

    When you click on Netware or Compatible Network in File Manager, the following message appears: 

       Network Request is not Supported.

    CAUSE

    An internal calculation causes the redirector to behave as if burst had failed after LIP (large internet packet) failed, when it has not.

    STATUS

    Microsoft has confirmed this to be a problem in Windows NT version 3.51. This problem was corrected in Windows NT Workstation or Server version 4.0.
