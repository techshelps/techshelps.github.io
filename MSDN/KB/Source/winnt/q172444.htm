FPNW Generates Many Event ID 2024s

 Last reviewed: December 9, 1997
Article ID: Q172444
The information in this article applies to:

- Microsoft Windows NT Server versions 3.51 - Microsoft Windows NT Server versions 4.0 - File and Print Service for NetWare versions 3.51 and 4.0

SYMPTOMS

File and Print Service for NetWare (FPNW) logs many Event ID 2024s. 

   Event ID 2024, Source FPNWSrv:
   The client at address below was not logged in and attempted to open
   <volumename>.
where <volumename> is a NetWare-compatible volume defined on the FPNW server.

CAUSE

This can occur when any application or process is trying to access an FPNW volume on the network but is only attached to the FPNW server, not logged on to the server.

RESOLUTION

Ensure that the user is logged on to the FPNW server before trying to access any FPNW volumes.
