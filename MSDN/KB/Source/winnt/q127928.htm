Support for Windows NT Hardware Compatibility Testing

 Last reviewed: November 14, 1997
Article ID: Q127928
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.X and 4.0
  • Microsoft Windows NT Server version 3.X, and 4.0

SUMMARY

This article discusses reporting problems, Windows NT hardware compatibility tests (HCT) results, the hardware compatibility list (HCL) and the Windows NT device certification.

MORE INFORMATION

Windows NT Hardware Compatibility Tests

If you have any questions or problem running the HCT, report them to the Windows NT HCT group by completing PROBREP.TXT (located in the I386 directory on the Windows). To send the completed PROBREP.TXT to Microsoft:

  • FAX the PROBREP.TXT contents to Microsoft Windows NT HCT, Building 2 at (425) 93-MSFAX
  • Mail PROBREP.TXT (as an attachment) to NTHCTEXT@Microsoft.com

NOTE: Microsoft does not provide product support for HCT.

Windows NT Hardware Compatibility List

In order for computers to be listed on the Windows NT Hardware Compatibility List, the device must pass all of the system compatibility tests in the HCT kit (refer to Chapter 2 of the HCT Guide for additional information). Every floppy disk that you send should have one set of test results. The floppy disks must be labeled with the following information: 

   <Your company name>
   Windows NT version X.X
   <Computer ID>
   <Marketed Model Name the results represent>
NOTE: If a single test configuration is marketed as multiple products please list the other model names on the floppy disk as well. If you send a README file or other notes with your results, please enclose a printed copy.

Microsoft reserves the right to require a computer be sent to the Microsoft HCT Labs for verification of compatibility if problems are encountered on the computer that has previously passed compatibility testing.

After the computer passes all the required tests, please send the floppy disks with the test results to the following address by Federal Express or by DHL (to prevent Customs delays if outside of the United States): 

   Windows NT Hardware Compatibility Test
   Building 2
   Microsoft Corporation
   One Microsoft Way
   Redmond, WA 98052-6399
   USA

Windows NT Device Certification

In addition to the HCT program, Microsoft also offers a certification program for hardware devices and adapters. This program is fee-based and is administered by the Microsoft Compatibility Labs (MCL). For more information on this program, contact MCL at (425) 635-4949.
