NEC Pinwriter 3200 Doesn't Print Spaces from Notepad

 Last reviewed: May 13, 1997
Article ID: Q110197
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you print to the NEC Pinwriter 3200 from Notepad, spaces may not print correctly or may not be print at all.

CAUSE

Printing with the Courier font in any application with NEC Pinwriter 3200 causes spaces to print incorrectly. For example, the text "A B C D" prints as "ABCD." Notepad defaults to Courier; therefore, Notepad always experiences this problem. This problem may also occur with other printers using the same NEC mini driver. This problem also occurs on the NEC Pinwriter 5300.

STATUS

Microsoft has confirmed this to be a problem in Windows NT and Windows NT Advanced Server. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
