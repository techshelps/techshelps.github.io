No Modem Responses Section in MODEM.INF

 Last reviewed: May 15, 1997
Article ID: Q114726
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The Windows NT MODEM.INF file contains the following commented statement: 

   "Modem responses are normally located in the global [Modem Responses]
   section."
This is incorrect. The global section that contains modem response sequences is titled [Responses], not [Modem Responses]. Therefore, if you need to add a new modem response sequence to the MODEM.INF file, it should be added to the [Responses] section.
