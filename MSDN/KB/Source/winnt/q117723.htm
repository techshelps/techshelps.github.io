Err Msg: KERNEL_STACK_INPAGE_ERROR (0x77) w/Future Domain 1680

 Last reviewed: May 16, 1997
Article ID: Q117723
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

If you are using a Future Domain 1680 SCSI Controller with a BIOS earlier than 3.2, the following error message may appear: 

   KERNEL_STACK_INPAGE_ERROR        (0x77)

CAUSE

Even though this board is listed under the Windows NT hardware compatibility list, using an earlier BIOS may cause Windows NT to fail.

RESOLUTION

To avoid this problem, upgrade to a later BIOS. Call Future Domain technical support at (800) 879-7599 for information on obtaining a BIOS upgrade.
