Cannot Install Driver for ESS1888 Sound Card

 Last reviewed: October 5, 1997
Article ID: Q166729
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SUMMARY

The latest Windows NT 4.0 Hardware Compatibility List (HCL) lists the ESS1888 sound card as a supported device with included drivers in Windows NT 4.0. Windows NT 4.0 does not actually include a driver for this device.

For information about the Windows NT 4.0 HCL, see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q131303
   TITLE     : Latest Windows NT Hardware Compatibility List (HCL)

MORE INFORMATION

This is a documentation error. The ESS1888 sound card should not be listed on the Windows NT 4.0 HCL. A driver for this device is not included on the Windows NT 4.0 CD-ROM. Drivers are included for other ESS sound cards, but those drivers do not work with the ESS1888 sound card. Windows NT 4.0 does not support the ESS1888 sound card and does not contain drivers for it.

For additional information about ESS sound cards, visit the ESS Technology, Inc. web page at: 

   http://www.esstech.com
