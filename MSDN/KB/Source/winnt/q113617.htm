Err Msg: File Manager Cannot Lock Volume

 Last reviewed: May 14, 1997
Article ID: Q113617
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you attempt to format a floppy disk from File Manager, the following error message appears: 

   File Manager cannot lock volume.

CAUSE

This error message appears when another process on your computer has access to the floppy disk drive and File Manager cannot access it.

Access to the floppy disk drive is limited to one process at a time. When a process accesses the floppy disk drive, it "locks" the drive for its exclusive use, even though the process may not be physically accessing the drive at all times.

The most common cause of this error message appearing is when a command prompt is left at the drive A or drive B prompt.

WORKAROUND

To work around this problem if a command prompt has been left at the drive A or drive B prompt, switch to the command prompt application and change the prompt to drive C, or quit the command prompt application.

To work around this problem if another application has locked the drive, change the focus of that application to another drive, or quit the application.
