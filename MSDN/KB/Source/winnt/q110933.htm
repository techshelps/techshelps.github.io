STOP Message: 0xC0000143 Error...on Dell 486 Models

 Last reviewed: May 14, 1997
Article ID: Q110933
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When start Windows NT on a Dell 486 computer with the built-in S3 graphics chip set, the following error message may appear: 

   STOP: 0xC0000143 [Missing System File] The required system file
   KERNEL_VIDEO_DRIVER.SYS is bad or missing.
This error message appears at the blue initialization screen before the Windows NT logon screen appears. The computer locks, and you will not be able to continue starting Windows NT.

CAUSE

Some models of the Dell 486 line of computers may need a BIOS upgrade for the Phoenix 80486 ROM BIOS Plus, which is installed on these computers. This problem may occur on computers with BIOS versions 1.10A14 and earlier, which includes Dell 486 models with the following letter identifications: L, M, T, LV, MX, S, and MXV. For example, the problem could occur on the Dell Performance 466/T, but probably not on the Dell PowerLine.

RESOLUTION

You can correct this problem by upgrading to Phoenix ROM BIOS version 1.10A15. You can obtain the upgrade by downloading it from the Dell BBS service at (512) 728-8528. The file to download is WA15BA06.EXE. This is not a self-extracting executable file; run the file and it presents you with a user interface with options for installing the Phoenix ROM BIOS Plus upgrade to version 1.10A15.

You can call the 24-hour Dell technical support service at (800) 247-9362 or (800) 642-9896.
