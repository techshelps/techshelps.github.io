LAN Manager Toolbar Hangs When Accessing Windows NT Computer

 Last reviewed: February 13, 1996
Article ID: Q142161
The information in this article applies to:
  • Microsoft Windows NT Server versions 3.5 and 3.51
  • Microsoft LAN Manager version 2.2x

SYMPTOMS

When you use the Toolbar utility in the LAN Manager Resource Kit and attempt to access any application on a Windows NT computer, the LAN Manager client computer stops responding (hangs).

WORKAROUND

To work around this problem, remove the Toolbar utility from the LAN Manager client computer.

STATUS

Microsoft has confirmed this to be a problem in Toolbar for LAN Manager. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
