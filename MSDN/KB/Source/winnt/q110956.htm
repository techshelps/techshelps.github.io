Help Button Fails for Olicom Token Ring 16/4 Adapter

 Last reviewed: May 14, 1997
Article ID: Q110956
The information in this article applies to:
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you install the Olicom Token Ring 16/4 adapter and select the option to configure the adapter under the Network applet of Control Panel, you are given the option of selecting a Help button. However, when you choose the Help button, the help file doesn't run and an error message does not appear.

RESOLUTION

To avoid this problem, use the documentation and materials shipped with the Olicom Token Ring 16/4 adapter.

STATUS

Microsoft has confirmed this to be a problem in Windows NT Advanced Server version 3.1. We are researching this problem and will post new information here as it becomes available.
