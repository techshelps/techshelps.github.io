Help Button Not Always Present When Disconnecting User

 Last reviewed: May 9, 1997
Article ID: Q103739

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

When you use Server Manager and disconnect a user from a share, sometimes no Help button is displayed.

CAUSE

The following two messages may be displayed when users are about to be disconnected:

  • Are you sure you want to....
  • The selected user has open resources....

There is no Help context associated with the latter of these messages.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.1. We are researching this problem and will post new information in the Microsoft Knowledge Base as it becomes available.
