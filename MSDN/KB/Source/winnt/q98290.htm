Upgrading RAM Can Slow System Performance

 Last reviewed: May 6, 1997
Article ID: Q98290
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows operating system versions 3.0, 3.0a, and 3.1
  • Microsoft Windows for Workgroups version 3.1

If you experience a loss in system performance after installing additional RAM (for example, after upgrading from 8 MB to 16 MB), check your hardware documentation for information on any hardware caches you might have on your computer. Some hardware caches are designed for a specific amount of memory and can actually degrade overall system performance if there is too much RAM available. You may need to disable your hardware cache for best results. If you are unsure, contact your hardware vendor for more information.
