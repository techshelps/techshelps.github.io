STOP 0x0000001E When Installing Small Business Server

 Last reviewed: March 5, 1998
Article ID: Q177318
The information in this article applies to:
  • Microsoft BackOffice Small Business Server version 4.0

SYMPTOMS

When you install Small Business Server, the installation process stops with the following STOP error message in Setupdd.sys: 

   STOP: 0x0000001E (0xC0000005, 0x8021222D,0x000000000,0x00650072)
   KMODE_EXCEPTION_NOT_HANDLED
NOTE: The second parameter in the STOP code may be different depending on your system configuration.

CAUSE

The error message occurs because there were no formatted partitions available. A formatted partition must be created before installing Small Business Server. This is noted in the release notes for Small Business Server.

RESOLUTION

Create and format a partition before installing Small Business Server.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 4.0. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
