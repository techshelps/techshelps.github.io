Word for Windows Print Preview Leaves Printing Dialog Box

 Last reviewed: May 8, 1997
Article ID: Q102033

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

SYMPTOMS

When you run Microsoft Word for Windows version 2.0 under Windows NT, you may have remnants of the Printing dialog box on your screen. This situation occurs when you print two or more pages from the Print Preview screen in two-page layout.

CAUSE

Word for Windows incorrectly tells the system it has restored the region occupied by the Printing dialog box when it has not. This is a video display problem--no data corruption occurs.

RESOLUTION

To correct the corrupted video display, minimize and then restore the Word for Workgroups window.
