IBM 4029 Printer Does Not Save Paper Bin Changes

 Last reviewed: April 4, 1997
Article ID: Q101868

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

The Printer Properties dialog box for the IBM 4029 allows you to specify the forms that are in the different trays. However, it does not specify a default tray or a default form.

The Printer Driver dialog box displays two boxes, one for the paper tray and one for the form name in that tray. If you specify a different tray, the form box changes to the specified form for that tray. You can then specify a different form.

These settings are not saved as defaults. If you choose the OK button, and then re-enter the dialog box, the first tray and it's associated form are displayed.
