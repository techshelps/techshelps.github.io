How to Submit a Print Driver Request for Windows NT

 Last reviewed: January 16, 1998
Article ID: Q140986
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.51 and 4.0
  • Microsoft Windows NT Server versions 3.51 and 4.0

To submit a request for a print driver or print driver enhancement for Windows NT, please send your request to the following E-Mail Alias: 

   ntwish@microsoft.com
The following website also allows you to submit production suggestions to Microsoft: 

   http://www.microsoft.com/regwiz/regwiz.asp
The request should have the following Information: 

   Requester's Name
   Business Name
   Requester's Contact Information (Phone # or Email Address)
   Printer Manufacturer
   Printer Model
   Nature of Request (Feature Request or Driver Request)
For additional information on shipping drivers that have been released, please see the following articles in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q100654
   TITLE     : Windows NT 3.51 Driver Library

   ARTICLE-ID: Q142643
   TITLE     : Windows NT 4.0 Driver Library
