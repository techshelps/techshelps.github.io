Trantor T338, T348, and T358 SCSI Available

 Last reviewed: May 9, 1997
Article ID: Q103232

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Drivers for T338, T348, and T358 for Windows NT are available from either the Trantor BBS at (510) 656-5159, or from the Trantor library on CompuServe [GO TRANTOR].
