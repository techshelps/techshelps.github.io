Cardfile Font Setting Is Not Saved

 Last reviewed: August 22, 1995
Article ID: Q121405
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5
  • Microsoft Windows NT Server version 3.5

The default Cardfile font is Lucida Console. You can change the font by choosing Set Font from the Edit menu. However, after you save, close, and open the file, the font reverts to Lucida Console.

Microsoft has confirmed this to be a problem in Windows NT version 3.5. This problem has been corrected in Windows NT Workstation and Windows NT Server version 3.51.
