Support for COMPAQ Tape Drives Under Windows NT

 Last reviewed: May 12, 1997
Article ID: Q106213
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

COMPAQ provides several different tape drives as options on their computer systems. None of these drives are listed on the Windows NT Hardware Compatibility List (HCL). According to COMPAQ, the following Tape Drives will work with drivers provided in the retail version of Windows NT: 

   COMPAQ 525-MB Tape Drive
   COMPAQ 320-/525-MB Tape Drive
   COMPAQ 1.3-/2.0-GB DAT Drive
   COMPAQ 5.0-GB DAT drive

MORE INFORMATION

Use the following table to identify which Windows NT Tape driver should be used for each COMPAQ Tape drive: 

   Tape Drive                        Windows NT Tape Driver
   --------------------------------------------------------
   COMPAQ 525-MB Tape Drive          Wangtek 150/250 and 525
   COMPAQ 320-/525-MB Tape Drive     Wangtek 150/250 and 525
   COMPAQ 1.3-/2.0-GB DAT Drive      4 millimeter DAT drives
   COMPAQ 5.0-GB DAT drive           4 millimeter DAT drives
COMPAQ only supports these drivers when one of the COMPAQ tape drives mentioned above is connected to one of the following SCSI Controllers: 

   COMPAQ Integrated Fast SCSI-2 Options Port (ProSignia imbedded SCSI)
   COMPAQ Fast SCSI-2 Controller option (EISA SCSI Card)
   COMPAQ Integrated SCSI-2 options port (Systempro/XL imbedded SCSI)
   COMPAQ 6260 SCSI Controller
   COMPAQ Portable 486c Integrated 6260 SCSI Controller
   COMPAQ Portable 486m Integrated 6260 SCSI Controller
   COMPAQ Portable 486c Integrated 53C90 SCSI Controller
   COMPAQ Portable 486m Integrated 53C90 SCSI Controller
   COMPAQ SCSI Compression Adapter (shipped with the 1.3-/2.0-GB DAT)
   COMPAQ SCSI Option Adapter (shipped with the 320-/525-MB Tape Drive)
The following COMPAQ tape drives are NOT currently supported by COMPAQ under Windows NT: 

   COMPAQ 40-/60-MB tape drive
   COMPAQ 80-/120-MB tape drive
   COMPAQ 135-MB tape drive
   COMPAQ 150-/250-MB tape drive

REFERENCES

The information in this article was taken from the file TAPE.RDM which can be found on the COMPAQ Support Software Diskette (SSD) for Microsoft Windows NT.
