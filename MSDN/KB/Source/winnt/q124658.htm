DCA IRMA Token Ring Adapter is not Detected

 Last reviewed: February 9, 1996
Article ID: Q124658
The information in this article applies to:
  • Microsoft Windows NT Workstation version 3.5 and 3.51
  • Microsoft Windows NT Server version 3.5 and 3.51

SYMPTOMS

When you install Windows NT on a computer using a DCA IRMA Token Ring adapter, the network adapter is not automatically detected.

WORKAROUND

To work around this problem, continue installing Windows NT and manually select the DCA IRMA Token Ring Adapter.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.5. We are researching this problem and will post new information in the Microsoft Knowledge Base as it becomes available.
