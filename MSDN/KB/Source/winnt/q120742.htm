Qualcomm Eudora v1.4 Displays Only 1 Mail Message in Window

 Last reviewed: October 3, 1994
Article ID: Q120742
The information in this article applies to:
  • Microsoft Windows NT workstation version 3.5
  • Microsoft Windows NT Server version 3.5

SUMMARY

The shareware version of Qualcomm's Eudora v1.4 has a bug that causes it to display only one mail message in a window.

MORE INFORMATION

Eudora v1.4.2 contains the fix for this problem. It can be found at ftp.qualcom.com in the quest\windows\eudora\1.4\beta directory.
