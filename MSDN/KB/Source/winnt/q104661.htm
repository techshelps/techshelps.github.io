Help for Backup Is Not Available from Command Prompt

 Last reviewed: May 9, 1997
Article ID: Q104661

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Although you can run Backup from Command Prompt, there is no on-line help available. If you try to use the NTBACKUP /? command, Backup is invoked.

Once you start Backup, you can view on-line help by pressing F1 or by choosing the Help menu.
