Cannot Log On to Domain, Lowercase Extended Characters

 Last reviewed: May 12, 1997
Article ID: Q105328

The information in this article applies to:

- Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

After changing the password of a Macintosh client computer from a Windows NT computer, you can no longer log on to the domain.

CAUSE

This problem occurs if you use lowercase, extended characters in your password. Windows NT Services for Macintosh does not correctly handle lowercase, extended characters in passwords.

STATUS

Microsoft has confirmed this to be a problem in Windows NT and Windows NT Advanced Server version 3.1. This problem was corrected in the latest U.S. Service Pack for Windows NT and Windows NT Advanced Server version 3.1. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
