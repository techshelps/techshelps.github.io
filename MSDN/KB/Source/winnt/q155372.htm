Analysis of Helpdesk Impacts During Windows NT 4.0 Deployment

 Last reviewed: March 18, 1997
Article ID: Q155372
The information in this article applies to:
  • Microsoft Windows NT Workstation version 4.0

SUMMARY

This article describes a study that analyzes the impact on Helpdesks of the deployment of Windows NT Workstation version 4.0.

MORE INFORMATION

"Rescue for the Helpdesk: The Impact of Microsoft Windows NT Workstation" is a study by Workgroup Technologies, Inc. This study demonstrates that switching to Windows NT Workstation provides substantial Helpdesk savings.

This study is available for viewing on the World Wide Web at 

   http://www.microsoft.com/ntworkstation/plan.htm
