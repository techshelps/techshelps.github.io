Ask3Com Forum News Flash on CompuServe

 Last reviewed: May 14, 1997
Article ID: Q112660
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

The News Flash on the Ask3Com forum on CompuServe states that an updated Windows NT driver for the 3Com 3C509 is available in Library 2 of the WINNT forum. There is no such driver available on the WINNT forum of CompuServe.

MORE INFORMATION

The exact text referring to the updated 3C509 driver in the Ask3Com News Flash is as follows: 

   Microsoft's Windows NT driver for the EtherLink III (3c509) can be
   downloaded from the WINNT forum on CompuServe. Type GO WINNT, enter
   library 2, and download file ELNK3.ZIP. *NOTE*: This driver is supported
   only by Microsoft!
As of Service Pack 2 for Windows NT, there have been no updates to the ELNK3.SYS driver. At this time, Microsoft does not have an updated Windows NT driver for the 3Com 3C509 network adapter.
