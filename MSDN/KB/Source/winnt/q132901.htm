3COM 3C619 TokenLink III Network Card Support in Windows NT 3.5x

 Last reviewed: May 5, 1997
Article ID: Q132901
The information in this article applies to:
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

The Windows NT 3.5 Hardware Compatibility List (HCL) incorrectly states that the 3Com 3c619 TokenLink III network interface card (NIC) driver is located on the Windows NT Driver Library in the Microsoft Download Service (MSDL).

The Windows NT 3.51 Hardware Compatibility List (HCL) incorrectly states that the 3Com 3c619 TokenLink III network interface card (NIC) driver is located in the \DRVLIB directory on the Windows NT 3.51 CD-ROM or the Windows NT Driver Library in the Microsoft Download Service (MSDL). The driver is located in the \DRVLIB\TLNK3 directory, but not on the MSDL.

The 3Com 3c619 NIC driver for Windows NT is also available in the Threecom forum on CompuServe. Download NT351EIS.EXE, a self-extracting file that includes the NIC driver. The driver is also available through the 3Com Bulletin Board Service (BBS) at (408) 980-8204.
