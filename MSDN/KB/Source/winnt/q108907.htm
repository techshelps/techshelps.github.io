Windows NT Directories on the MSDN Fall 93 CDs

 Last reviewed: May 13, 1997
Article ID: Q108907
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The Microsoft Developer Network (MSDN)Fall ‘93 CDs contain several Windows NT versions on two CDs:

Systems CD 1 contains: 

<root>\winnt\x86\FRN   - French Windows NT for Intel machines.
<root>\winnt\x86\GER   - German Windows NT for Intel machines.
<root>\winnt\x86\SPA   - Spanish Windows NT for Intel machines.
<root>\winnt\Mips\FRN - French Windows NT for MIPS machines. <root>\winnt\Mips\GER - German Windows NT for MIPS machines. <root>\winnt\Mips\SPA - Spanish Windows NT for MIPS machines.

Systems CD 2 contains:

<root>\i386 - U.S. (English) Windows NT for Intel machines. <root>\Mips - U.S. (English) Windows NT for MIPS machines. <root>\Alpha - U.S. (English) Windows NT for Alpha machines.

The general telephone support number for the MSDN is (800) 936-5800.
