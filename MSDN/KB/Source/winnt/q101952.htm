Maximum Rows and Columns for MS-DOS-Based Applications

 Last reviewed: May 8, 1997
Article ID: Q101952

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The maximum size of a window that an MS-DOS-based application can run in is 80 columns by 50 rows.
