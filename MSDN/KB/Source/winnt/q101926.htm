Adaptec 174x SCSI Adapter May be Detected as a 154x

 Last reviewed: May 7, 1997
Article ID: Q101926

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

In Windows NT, the Adaptec 174x SCSI adapter may be detected as a 154x model.

If the Adaptec 174x SCSI adapter is configured for standard mode, it emulates the 154x adapter. Windows NT detects it as a 154x adapter and installs the Windows NT 154x SCSI driver. This is expected behavior and this mode is supported in Windows NT.

If the Adaptec 174x is configured for enhanced mode, the adapter is detected as a 174x.

The mode, standard or enhanced, can be changed through the Adaptec setup utility. Please refer to the Adaptec manual for details on changing this mode.
