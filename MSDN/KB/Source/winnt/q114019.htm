HCL Incorrectly Lists Archive 51250Q (SuperHornet)

 Last reviewed: May 15, 1997
Article ID: Q114019
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Page 20 of the Windows NT version 3.1 Hardware Compatibility List incorrectly documents the first entry of the QIC-40/QIC-80 Floppy Tape Drive as the "Archive 5150Q (SuperHornet)." This error also appears in the March 1994 HCL update.

There is no model 5150Q. The correct model designation is the Archive 51250Q SuperHornet.
