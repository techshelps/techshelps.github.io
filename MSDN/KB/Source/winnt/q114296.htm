Compaq 5/60M with QVision Video Card and Driver in Slot 5 Hang

 Last reviewed: May 15, 1997
Article ID: Q114296

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

When you install the Compaq QVision driver on a Compaq 5/60M computer with Compaq's QVision video card installed in the fifth expansion slot, a STOP message appears when you restart your computer. The standard VGA (640x400) mode does work correctly and the computer passes all Compaq diagnostic tests. The QVision driver also works correctly with the QVision card installed in a Compaq Deskpro/M 486/66.

WORKAROUND

Although Compaq requires that the card be in slot 5 for full functionality, moving the video card to a different slot avoids this problem in some cases.

STATUS

Microsoft has confirmed this to be a problem in Windows NT and Windows NT Advanced Server version 3.1. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
