ZEOS Brand NCR 77C22E Video Card and Windows NT

 Last reviewed: May 13, 1997
Article ID: Q107639
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

When using a ZEOS video card that includes the NCR 77C22E chip with Windows NT, 16-bit applications may fail.

MORE INFORMATION

Some ZEOS machines include a video card using NCR’s 77C22E chip. Windows NT includes a driver for NCR 77C22E video adapters. If this driver is installed on one of these machines, when you attempt to start a 16-bit application, you will receive the following error message: 

   An application error has occurred:
      NTVDM.EXE
   Exception: Access violation (0xC000005)
The drivers supplied with Windows NT are not fully compatible with the ZEOS version of this video adapter, and should not be expected to operate properly.

The product included here is manufactured by ZEOS International, Ltd., a vendor independent of Microsoft; we make no warranty, implied or otherwise, regarding this product's performance or reliability.

The only Windows NT supplied driver that functions properly with this video card is the Standard VGA driver.

For more information, call ZEOS International, Ltd. at (800) 228-5390.
