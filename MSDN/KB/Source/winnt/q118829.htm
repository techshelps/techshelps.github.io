Supported NetWare Utilities and Applications Under Windows NT

 Last reviewed: April 11, 1997
Article ID: Q118829
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

The following lists NetWare utilities and NetWare-aware applications that are supported under Windows NT version 3.1 using the Microsoft Client and Gateway Services for NetWare.

Supported NetWare Utilities

The following table lists NetWare utilities and any known issues corresponding to each of them.
