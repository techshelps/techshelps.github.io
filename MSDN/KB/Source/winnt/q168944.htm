Epson Stylus Color 800 Does Not Print with Service Pack 3

 Last reviewed: September 15, 1997
Article ID: Q168944
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

After you install printer drivers version 1.01a dated 4/22/97 from Epson's Web site for the Epson Stylus Color 800 printer on a computer with Windows NT 4.0 Service Pack 3 installed, you may experience the following symptoms:

  • When you attempt to print a test page, the following error message is displayed: 

          Test page failed to print. Would you like to view the print
      troubleshooter for assistance? The system does not support the
      command specified.
  • When you attempt to print, the following error message is displayed: 

          Cannot access your printer. Be sure that your printer is connected
      properly and use Control Panel to verify that the printer is
      configured properly.

RESOLUTION

Either uninstall Service Pack 3, or install updated printer drivers. For information about updated printer drivers, please contact Epson.
