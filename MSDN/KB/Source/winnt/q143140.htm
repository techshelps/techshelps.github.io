Using IPXROUTE to Find Out IPX Address and Computer Name

 Last reviewed: September 6, 1996
Article ID: Q143140
The information in this article applies to:
  • Microsoft Windows NT Server version 3.51
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SUMMARY

The utility IPXROUTE, included in the multiple provider router (MPR) software, provides three new options:

  1. IPXROUTE Table, which displays the IPX routing table.
2. IPXROUTE Servers, which displays the SAP table. 

   NOTE: The default setting is all server types. In the setting
   [/type=xxxx], xxxx refers to server type.

  • IPXROUTE Stats, which displays or clears IPX internal routing statistics.

    NOTE: The setting [/Show] displays the internal routing table; [/Clear] clears the table.

    These options provide information on RIP, SAP, and statistics. Use this utility to display and modify information about the source routing tables used by IPX.

    MORE INFORMATION

    If the client computer is running NWLINK, RIP for NWLINK, IPX, and SAP, you can use the IPXROUTE Servers option to find out that client's IPX Address, Server type, and name.
