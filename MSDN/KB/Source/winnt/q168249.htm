Availability of Printer Driver for HP OfficeJet Printer

 Last reviewed: May 8, 1997
Article ID: Q168249
The information in this article applies to:
  • Microsoft Windows NT Server version 3.51
  • Microsoft Windows NT Workstation version 3.51

NOTE: This article provides information about a configuration that is not supported by Microsoft Technical Support. This article is provided for informational purposes only; Microsoft makes no guarantee that this configuration will function properly.

SYMPTOMS

There is no printer driver available for the Hewlett-Packard (HP) OfficeJet printer when you attempt to install the printer in Print Manager.

RESOLUTION

Normal printer functionality should be available using any of the following drivers:

  • HP DeskJet 520
  • HP DeskJet 510
  • HP DeskJet 550C
