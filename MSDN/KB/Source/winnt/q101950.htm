Martian Memorandum Runs Slowly with Sound Enabled

 Last reviewed: May 7, 1997
Article ID: Q101950

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SYMPTOMS

With sound enabled, Martian Memorandum runs very slowly.

CAUSE

This problem is due to the fact that Martian Memorandum speeds up the timer to 16.5 kilohertz (KHz). Windows NT is trapping all interrupts, and therefore this causes very slow playing.

WORKAROUND

To avoid this slowdown, disable the sound.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.1. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
