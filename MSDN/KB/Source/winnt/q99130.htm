DocErr: IsDomainMasterBrowser Not Valid Key in Windows NT 3.5

 Last reviewed: May 9, 1997
Article ID: Q99130

The information in this article applies to:
  • Microsoft Windows NT Resource Kit versions 3.5, and 3.51
  • Microsoft Windows NT Server Resource Kit version 4.0

On page 491, the Windows NT version 3.5x Resource Guide incorrectly documents one of the Browser Service Entries as IsDomainMasterBrowser. This same error appears on page 133 and 134 of the Windows NT 4.0 Networking Guide. This is the correct key for Windows NT 3.1. The correct key for Windows NT 3.5x and 4.0 is IsDomainMaster.
