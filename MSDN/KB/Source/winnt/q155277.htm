Cannot Use NET VIEW Command with Loopback Address

 Last reviewed: March 27, 1997
Article ID: Q155277
The information in this article applies to:
  • Microsoft Windows NT Server version 4.0
  • Microsoft Windows NT Workstation version 4.0

SYMPTOMS

When you use the NET VIEW command at a command prompt to the IP loopback address (127.0.0.1), you receive the following error message: 

   System Error 53 has occurred.
   The network path was not found.
Any other valid IP address returns the shares on that address correctly.

CAUSE

NetBT (NetBIOS over TCP/IP) does not bind to the loopback address. Computers cannot be viewed using the loopback address.

RESOLUTION

Use the computer name or type "net share" (without quotation marks) at the command prompt to view your own computer.
