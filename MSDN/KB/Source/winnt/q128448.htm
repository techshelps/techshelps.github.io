RAS Event ID 20013: The User Connected to Port...

 Last reviewed: August 22, 1996
Article ID: Q128448
The information in this article applies to:

- Microsoft Windows NT Workstation version 3.5 - Microsoft Windows NT Server version 3.5

SYMPTOMS

After you connect from a Remote Access Service (RAS) client to a RAS Server in Windows NT version 3.5 and perform file transfer using the TCP/IP protocol in heavy network traffic, the following event may appear in the Event Viewer of the RAS Server: 

   Event ID: 20013
   Source: Remote Access
   Type: Error
   Description: The user connected to port <port> has been disconnected
   due to a fatal network error on the local network.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.5. This problem has been corrected in the latest U.S. Service Pack for Windows NT version 3.5. For information on obtaining the Service Pack, query on the following word in the Microsoft Knowledge Base (without the spaces): 

   S E R V P A C K
