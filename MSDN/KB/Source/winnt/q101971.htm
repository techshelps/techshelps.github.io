Print Manager: Size of Spooling Document Is Zero

 Last reviewed: May 8, 1997
Article ID: Q101971

The information in this article applies to:
  • Microsoft Windows NT operating system, version 3.1
  • Microsoft Windows NT Advanced Server, version 3.1

When Print Manager is spooling a print job, the size of the spool file is not available. When the document has been completely spooled, the display in Print Manager gets updated correctly.

Print Manager does not update the print job size while spooling because:

  • The information is not available when printing to an OS/2 or Windows for Workgroups print server.
  • Continually updating the display would cause a serious performance decrease. Because spooling is a performance-sensitive measure of print speed, the print-job size is not update while the job is being spooled.
