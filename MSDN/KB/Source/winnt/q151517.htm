Stop: Common Dialog Error 0x00A When Printing From Notepad

 Last reviewed: July 23, 1996
Article ID: Q151517
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

SUMMARY

When you attempt to print from Notepad in Windows NT, the following message may be displayed: 

     Stop: Common dialog error 0x00A when printing from Notepad

MORE INFORMATION

This error message may indicate a problem with either a corrupted printer driver or an invalid registry entry for the printer.

RESOLUTION

To resolve this problem, remove the printer according to the steps in Microsoft Knowledge Base article referenced below.

For additional information, please see the following article in the Microsoft Knowledge Base: 

  ARTICLE-ID: Q102116
  TITLE     : How to Manually Delete Print Shares or Printers
