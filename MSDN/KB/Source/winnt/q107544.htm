Excessive Static with ProAudio Spectrum 16

 Last reviewed: May 13, 1997
Article ID: Q107544
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

You may hear excessive static when you play wave audio from Cinemania or Encarta under Windows NT using MediaVision's ProAudio Spectrum 16.

MORE INFORMATION

In the setup of the ProAudio Spectrum 16 card, you have the choice of four 8-bit and three 16-bit DMA channels (0-3 are 8-bit and 5-7 are 16-bit). If you select an 8-bit DMA channel to use with the 16-bit sound card, you will hear excessive static when you play wave audio from one of the above applications.

To correct this problem, set up the card to use a 16-bit DMA channel instead of an 8-bit DMA channel.
