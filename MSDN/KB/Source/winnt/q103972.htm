Changing Setup Media, Description of Setup Information Files

 Last reviewed: April 10, 1997
Article ID: Q103972

The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

Windows NT Setup copies setup information files designed for the media Setup is running from. If you install Windows NT from floppy disks, setup information files containing directions and disk numbers for floppy disks are copied to your hard disk drive. If you later need to use files from a CD-ROM disc, Windows NT Setup prompts you for floppy disks.

You can change the source media for a given device driver or Windows NT component by copying the appropriate setup information file to your Windows NT \SYSTEM32 subdirectory. In this manner, you can install new device drivers and components from any media you choose.

The names and locations of the setup information files you need are listed later in this article in the section titled "Setup Information Filenames, Descriptions, and Locations." The following example describes how to install a new pointing device or mouse driver.

INSTALLING A NEW MOUSE DRIVER FROM FLOPPY DISKS

To install a new pointing device or mouse driver from floppy disks after installing Windows NT Advanced Server from a compact disc, use the following procedure:

NOTE: To install a different type of device or component, see the following section, "Setup Information Filenames, Descriptions, and Locations." Substitute the appropriate filename and disk for POINTER.IN_ and "Disk 2" respectively.

  1. Insert the disk labeled "Disk 2" from the Microsoft Windows NT Advanced Server 3.1 disk set.

  2. Start Command Prompt from the Main group in Program Manager.

  3. Go to the \SYSTEM32 subdirectory.

  4. Substitute the appropriate floppy disk drive letter for <floppy disk drive> in the following command line: 

          expand <floppy disk drive>:\pointer.in_ pointer.inf

    NOTE: Some information files are not compressed and do not need to be expanded. For example, the PRINTER.INF file is not compressed. If this is the case, substitute the COPY command for EXPAND.

  5. Start Setup to install the new device driver as you would usually do.

You are now prompted for the appropriate floppy disk instead of a CD- ROM disc.
