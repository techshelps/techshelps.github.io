Network Card Options Unavailable

 Last reviewed: May 14, 1997
Article ID: Q112082
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

SUMMARY

When you attempt to install a network adapter driver that doesn't support the bus architecture of your computer, some of the card options are unavailable.

MORE INFORMATION

When you choose the Add Adapter option button, it looks at the .INF files of the network card to determine what bus architecture the network card driver supports. For example, if you have an IBM Micro Channel computer, the listing includes only network adapter drivers that support the MCA architecture.
