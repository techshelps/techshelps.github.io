Using the Magitronics EZ2000 Card With Windows NT

 Last reviewed: August 4, 1995
Article ID: Q133474
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1
  • Microsoft Windows NT Workstation versions 3.5 and 3.51
  • Microsoft Windows NT Server versions 3.5 and 3.51

Magitronics EZ2000, an NE2000 compatible network interface card (NIC), is not on the Windows NT HCL and does not work properly with any driver included with Windows NT.

Magitronics driver written from Windows NT 3.1; it is reported to operate under Windows NT 3.5 and 3.51. For more information, contact Magitronics.
