Nwconv.exe Does Not Convert Volumes Larger than 4 GB

 Last reviewed: May 16, 1997
Article ID: Q149916
3.51 4.00 WINDOWS kbtool kbbug3.51.sp4

The information in this article applies to:

  • Microsoft Windows NT Server version 3.51 with Service Pack 4
  • Microsoft Windows NT Server version 4.0

SYMPTOMS

During file conversion, Nwconvert (Nwconv.exe) will count the files it is converting. When it reaches 4 gigabytes (GB), Nwconvert starts over again. It does not finish the conversion process.

WORKAROUND

To work around this problem, select files in smaller chunks or smaller amount of data at a time. The amount can be equal to or less than 4 GB.

STATUS

Microsoft has confirmed this to be a problem in Windows NT version 3.51 with Service Pack 4. We are researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.
