Some UART Chips Do Not Support 14400 Baud

 Last reviewed: May 14, 1997
Article ID: Q112215
The information in this article applies to:
  • Microsoft Windows NT operating system version 3.1
  • Microsoft Windows NT Advanced Server version 3.1

UART chips 8250, 16450, and 16550 do not support 14400 baud rates.

If you attempt to use the Windows Terminal applet with a modem that uses a 8250, 16450, or 16550 UART chip, the following error message appears if you attempt to set the baud rate to 14400: 

   The selected COM port is either not supported or is being used by
   another device.

   Select another port.
