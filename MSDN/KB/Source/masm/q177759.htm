FILE: PATCH: MASM 6.13 Patch Available

 Last reviewed: February 3, 1998
Article ID: Q177759
The information in this article applies to:
  • Microsoft Macro Assembler for Windows NT, versions 6.11, 6.11a, 6.11d, 6.12

SUMMARY

Ml613.exe contains the files you need to update the Microsoft Macro Assembler (MASM) from versions 6.11, 6.11a, 6.11d and 6.12 to version 6.13. To use this update, you must have either MASM versions 6.11, 6.11a, 6.11d, or 6.12 currently installed. One or more of the following files are affected by this update depending on the version installed: Ml.exe, Ml.err, H2inc.exe, H2inc.err, Readme.txt, Errmsg.txt, or Support.txt. See the Readme.txt contained in this update for other information and installation instructions.

MORE INFORMATION

The following file is available for download from the Microsoft Software Library: 

 ~ Ml613.exe (size: 834436 bytes)
For more information about downloading files from the Microsoft Software Library, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID: Q119591
   TITLE     : How to Obtain Microsoft Support Files from Online Services
Due to changes in the MS-DOS extender used in earlier versions of MASM, MASM 6.12 and later do not run on Windows 3.1x or MS-DOS. Either Windows NT or Windows 95 is required to run MASM 6.12 and later. If you are working in MS-DOS or Windows 3.1x, you need to continue using version 6.11d. For more information about upgrading to 6.11d, please see the following article in the Microsoft Knowledge Base: 

   ARTICLE-ID:  Q138983
   TITLE     :  PATCH: MASM 6.11d Patch Available
Note that you can use MASM 6.13 to develop applications that run under MS- DOS or Windows 3.1x.

The files provided in Ml613.exe do not patch MASM version 6.11c, which was made available only in the Windows NT version 3.5 DDK and Windows 95 DDK, in the form of the files Ml.exe and Ml.err. You must replace these two files with their 6.11 or 6.11d counterparts prior to installing the 6.13 patch. The versions of Ml.exe and Ml.err from MASM version 6.11d can also be found in the Windows NT version 3.51 DDK. You can reinstall the Ml.exe and Ml.err files for MASM version 6.11 from the original disks. To do this, select "Copy a file from the distribution disks" from Setup's Main menu and specify Ml.exe and Ml.err.

If you need to order MASM 6.11, the part number is 016-051V611.
