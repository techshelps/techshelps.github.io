FIX: Code Segment Defined within a Procedure

 Last reviewed: September 10, 1997
Article ID: Q26830
5.00 5.00a 5.10 5.10a MS-DOS kbtool kbfixlist kbbuglist

The information in this article applies to:

  • Microsoft Macro Assembler for MS-DOS, versions 5.0, 5.0a, 5.1, and 5.1a

SYMPTOMS

Microsoft Macro Assembler (MASM) versions 5.0, 5.0a, 5.1, and 5.1a assembles the following invalid sample code without genering any error messages: 

   myproc proc far
   code segment
   ret
   code ends
   myproc endp
   end

STATUS

Microsoft has confirmed this to be a problem in MASM version 5.0, 5.0a, 5.1, 5.1a. This problem was corrected in MASM version 6.0.
