The MASM ASSUME Directive Does Not Generate Code

 Last reviewed: January 6, 1995
Article ID: Q10462
The information in this article applies to:
  • Microsoft Macro Assembler for MS-DOS, versions 5.0, 5.1, 6.0, 6.0a, and 6.0b

The ASSUME directive does not generate code, therefore you will not see it expanded in the listing file, only echoed.
