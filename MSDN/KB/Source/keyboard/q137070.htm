CTRL+ESC Opens HP Dashboard's Task Manager

 Last reviewed: November 18, 1996
Article ID: Q137070
The information in this article applies to:
  • Microsoft IntelliType software versions 1.0, 1.1

SYMPTOMS

If you install the IntelliType software on a Windows 3.x-based computer running the Hewlett-Packard Dashboard desktop, and Dashboard is configured to use its own task manager, pressing CTRL+ESC opens Dashboard's task manager instead of IntelliType's task manager.

Pressing the Windows key does open IntelliType's task manager.

CAUSE

By design, the Setup program for the IntelliType software does not replace or override a third-party task manager.

RESOLUTION

Reconfigure Dashboard not to use its own task manager.
