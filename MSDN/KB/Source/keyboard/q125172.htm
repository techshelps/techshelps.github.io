MNK: Right Windows Key Does Not Work with Jetkey BIOS

 Last reviewed: March 12, 1997
Article ID: Q125172
1.00 WINDOWS kb3rdparty kbbuglist kbhw

The information in this article applies to:

  • Microsoft Natural Keyboard with IntelliType software version 1.0

SYMPTOMS

The right Windows key on your Microsoft Natural Keyboard does not work.

CAUSE

This problem can occur if your computer contains a Jetkey keyboard BIOS version 3.0.

STATUS

Microsoft is researching this problem and will post new information here in the Microsoft Knowledge Base as it becomes available.

MORE INFORMATION

The keyboard BIOS that displays this problem is identified as: 

   Jetkey
   Version 3.0
   Fastest Keyboard BIOS
