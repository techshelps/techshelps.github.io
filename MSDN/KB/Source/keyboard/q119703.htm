Remote Control Programs Do Not Work with Natural Keyboard

 Last reviewed: October 25, 1996
Article ID: Q119703
The information in this article applies to:
  • Microsoft Natural Keyboard with IntelliType software version 1.0

SYMPTOMS

When the Microsoft IntelliType software is installed with the Microsoft Natural Keyboard, Remote Control programs no longer work.

CAUSE

These programs use keyboard and mouse drivers that are not compatible with the Natural Keyboard equivalent drivers.

MORE INFORMATION

Remote Control programs that do not work with the Natural Keyboard software include PC Anywhere, ReachOut, and Carbon Copy. These programs use their own replacement keyboard and mouse drivers, which do not implement the additional subroutines required for the Microsoft Natural Keyboard software.
