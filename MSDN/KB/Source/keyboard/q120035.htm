MSINPUT.386/Media Vision Conflict Causes Repetitious Sound

 Last reviewed: October 25, 1996
Article ID: Q120035
The information in this article applies to:
  • Microsoft Natural Keyboard with IntelliType software version 1.0
  • Microsoft IntelliPoint software version 1.0

SYMPTOMS

When IntelliType software is installed, a repeating sound occurs as Microsoft Windows is starting.

CAUSE

This problem occurs when a Media Vision sound card (such as the Pro Audio Studio) is set to interrupt 12. As a result, a conflict occurs between MSINPUT.386 and the Media Vision sound card driver.

RESOLUTION

To correct this problem, change the interrupt for the sound card to something other than 12.
