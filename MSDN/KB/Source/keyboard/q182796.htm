The information in this article applies to:

  • Microsoft Natural Keyboard Elite

SUMMARY

To use the Universal Serial Bus (USB) adapter included with the Natural Keyboard Elite, you must have a USB port and either Microsoft Windows 98 or Microsoft Windows NT 5.0. To use the USB adapter in MS-DOS, your computer's BIOS must support USB keyboards.

MORE INFORMATION

The USB adapter included with the Natural Keyboard Elite is not compatible with the Microsoft Natural Keyboard version 1.0.
