What Databases Can Be Used with Commerce Server

 Last reviewed: July 10, 1997
Article ID: Q156370
The information in this article applies to:
  • Microsoft Commerce Server, version 2.0

You can use any ODBC version 2.5 compliant relational database management system (RDBMS) with Microsoft Commerce Server version 2.0. However, Commerce Server uses ActiveX Data Objects (ADO) for data retrieval and storage from the RDBMS. You must make sure that the ODBC driver you intend to use works with ADO. Microsoft has tested Commerce Server with the Microsoft SQL Server ODBC driver and with the Oracle ODBC drivers from Microsoft and Visigenic. For more information on ADO, visit the following Web site: 

   http://www.microsoft.com/ado/
Also, in the order processing pipeline (OPP), you can write your own component(s) to get data from a legacy system.
