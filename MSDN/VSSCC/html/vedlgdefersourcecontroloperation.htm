Defer Source Control Operation Dialog Box

   

Use this dialog box to delay a source control pull action (Check Out, Undo Check Out, or Sync). While trying to perform this action, the development environment found the items listed to be in danger of possible overwriting with the server copies because the development environment cannot reload the items at this time. You should delay the source control action until you are done working on this solution and close it.

Options

Items

Displays a list of the items that are in conflict with your current action. Copies from the source control server may overwrite these items unless you delay this source control action.

Defer

Delays the pull action (Check Out, Undo Check Out, or Sync) until you close the current solution. The development environment completes the source control pull action automatically when the solution is closed. When you open the solution again, the items will be synchronized and your changes will be merged with those changes made by other users.

Skip

Skips over the items listed in the Items box for this source control action. Only the items listed in this Items box are skipped, not all of the possible items in the source control "pull" action (this action can be Check Out, Undo Check Out, or Sync).

Pull Action Now

Performs the source control pull action (Check Out, Undo Check Out, or Sync) now, possibly overwriting those items listed in the Items box with copies from the source control server.

Cancel

Aborts the entire source control "pull" action (Check Out, Undo Check Out, or Sync).
