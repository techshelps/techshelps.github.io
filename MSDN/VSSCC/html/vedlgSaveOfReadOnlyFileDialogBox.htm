Save of Read-Only File Dialog Box

   

Use this dialog box to save a file marked as read-only. Sometimes a file is marked read-only if you copy it from a CD to a disk.

Options

Overwrite

Overwrites the read-only version of the file with the current version.

Save As

Saves the file with a different name than the read-only copy.

Don’t Save

Cancels the save operation on just this file. Any other files involved are not affected.

Cancel

Cancels the entire save operation for all files.
