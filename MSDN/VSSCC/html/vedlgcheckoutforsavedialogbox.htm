Check Out for Save Dialog Box

   

Use this dialog box to preserve the changes you made to the item and perform any simultaneous source control action. Otherwise, you might be prompted about the item you are editing being read-only and not checked out from source control.

You can save these changes in the source control copy of the item by checking that item out from source control with this dialog box. Items marked with a check mark in the Items box are checked out from the source control server and the changes saved to that item.

If you do not want to save your changes to this item, do not check the item out from source control.

Options

Items

Displays a list of the items that are not checked out of source control. Select the check box next to the items you wish to check out from source control and a check mark will appear next to the item.

Comment

Provides a place to enter comments. If your source control program is Microsoft® Visual SourceSafe™, you can type up to 4,095 characters.

Check Out

Checks out the items selected in the Items box. Your changes to these items are preserved if you save this item and check the item back into source control.

Save As

Saves the item you are currently modifying with a different name.

Advanced

Displays more options on adding items to source control. Your source control program supplies these options.
