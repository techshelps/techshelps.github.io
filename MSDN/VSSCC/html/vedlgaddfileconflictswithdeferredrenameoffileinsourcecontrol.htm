Incomplete Rename of Added Item Dialog Box (I)

   

Use this dialog box to resolve a situation that is interfering with your adding items to the source control server. The items you renamed locally were not renamed on the source control server (the renaming action was deferred). These deferred items (using the outdated names) match the names of the items you are trying to add now.

Options

Items

Displays a list of the items that are in conflict with your current action. These items were renamed locally, but not renamed on the source control server. The outdated names of these items match the names of the items you are trying to add to the source control server.

Push Rename

Renames the list in the Items box on the source control server to match the local names. Afterward, you will be prompted to continue with your original action to add items.

Skip Add

Cancels these selected items to Add to Source Control and ignores the outstanding delete items. The development environment will no longer keep a record of the items that were only renamed locally. The source control server will not reflect the new names.
