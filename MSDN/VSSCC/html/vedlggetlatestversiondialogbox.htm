Get Latest Version Dialog Box

   

Use this dialog box to retrieve a read-only copy of solutions, projects, or individual items from source control. Items marked with a check mark in the Items box are copied from the source control server.

Get Latest Version places a copy of the requested item or items in the local copy of the solution.

Options

Items

Displays a list of the items based on your current selection in the Project Explorer (in the development environment). Select the box next to the items you wish to get from source control and a check mark will appear next to the item. You can click Check All to select all of the items at once.

This box displays the first level of folders and contents to which you can apply these operations. To see additional items contained within subfolders, click Recurse. Items contained within a subfolder are not retrieved from source control unless they are displayed individually in the Items box and selected with a check mark.

Recurse

Forces items to be displayed in subfolders in the Items box. You can then select these items for source control actions.

By default, the Items box displays only the first level of items contained in the Project Explorer for the development environment. Clicking Recurse displays all of the subfolders and all of their items for any remaining levels of depth in the directory tree, as shown in the Project Explorer. You can then select additional items for actions in source control.

Directories

Shows all of the directories of all of the items in your current selection. You can use the Recurse box to show additional directories.

Don't show any more Get dialogs

Suppresses any more instances of the Get Latest Version dialog box. Get Latest Version becomes a direct command on the environment menus. You will not be able to add comments or use advanced options until you enable this dialog box again.

Get items from subdirectories

Shows all of the items in the directories that you selected (checked) when Directories is selected. You can use this with Directories to Get items that are directly nested in your project but not visible in the Project Explorer. This is useful for exposing new subfolders in a directory-based project.

Check All

Selects with a check mark all of the items visible in the Items box. To see additional subfolders and levels of items, click Recurse. Only items that are visible in the Items box and are shown with a check mark are retrieved from source control.

Advanced

Displays more options on adding items to source control. Your source control program supplies these options.
