Conflicting Modifications Detected Dialog Box

   

Use this dialog box to finish a File Save action. While trying to save the item, the development environment found two copies of the item with the same name but differing contents. You can specify what resolution to take or click Cancel to temporarily postpone the resolution. If you cancel, you can investigate the situation. You will be prompted to resolve this situation again if you cancel now.

Options

Save As

Saves with a different name the item you are currently modifying. This item is then labeled differently from the other item the development environment found (same name, different content).

Overwrite

Proceeds with the save action, which may overwrite or change the items listed in the Items box on your local machine. This finishes the File Save action.

Cancel

Cancels the entire File Save action and this dialog box so that you can examine the items in question.
