Rename Item in Source Control Dialog Box

   

Use this dialog box to rename an item in the source control database. Do not use this dialog box to rename your item in the local project — use the Rename command on the File menu. This dialog box renames items only within the source control server's database, and these changes are not reflected in the local machine.

Options

Current item name

Displays the current name of the item.

New item name

Sends the rename request to the source control system to be evaluated. Your source control server will supply you with any possible results. This new name changes only in the source control server's database and is not reflected on your local machine. Name restrictions and limitations are defined by your source code control program.
