Initial Solution Add Dialog Box

   

Use this dialog box to specify the scope of the files to add to source control. You can add the entire solution, consisting of the solution file, all project files, and all files contained in the projects. You can also add just the projects that are selected in the Project Explorer.

Options

Solution

Adds the entire solution, projects, and folder structure to source control.

Selection

Adds only the projects you originally highlighted in the Project Explorer to source control.

Cancel

Cancels the action to add files to source control.
