Pull Removed Items From Server Dialog Box

   

Use this dialog box to control when an item is removed on your local machine to match a removal on the source control server. The Items box lists the items that were removed from the source control server. You can remove these items from your local machine or delay the action.

If you delay the removal, the development environment will remind you that there is a deferred removal of items not reflected on your local machine. You can also choose to do nothing, and keep the items on your local machine that were removed on the source control server.

Options

Items

Displays the items removed from the source control server. Click the box next to the items you wish to remove from your local machine, based on the button you choose (Remove Now, Remind Later, or Do Nothing). A check mark will appear next to selected items.

Remove Now

Removes the items selected with check marks in the Items box. This action removes the local copy of the items so that the contents of the solution will match in both places.

You will be reminded later about adding any items you did not select with a check mark in the Items box.

Remind Later

Delays removing the items from the local machine until you are reminded. To select specific items to remove from the local machine, click the items to remove, and then click Remove Now.

Pull reminders occur when a project is pulled. Pull actions include Get Latest, Check Out, and Undo Check Out. This is the default setting for reminder dialog boxes. You can change the timing of the dialog boxes by changing the options listed in the Source Control page (on the Tools menu, click Options, and then expand the Environment node and click the Source Control node).

Do Nothing

Leaves all of the items on the local machine. This means that a local copy of the item, removed on the source control server, is not removed from the local machine. You will not be prompted with any more reminders.

Don't show pull reminder dialog boxes

Suppresses any more instances of the Pull Removed Items from Server dialog box. You will not be able to add comments or use advanced options until you enable this dialog box again.

You can enable this dialog box again by changing the options listed in the Source Control page (on the Tools menu, click Options, and then expand the Environment node and click the Source Control node).

Advanced

Displays more options on adding items to source control. Your source control program supplies these options.
