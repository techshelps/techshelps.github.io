README.TXT

DESCRIPTION: 
 
    The deljob sample program illustrates how to use the SMS SDK to delete 
    an SMS job. 
 
 
USAGE: 
 
    When the deljob program is run, it displays help text that 
    describes how to use the program. 
 
 
FILES PROVIDED: 
 
    readme.txt 
    makefile 
    deljob.cpp 
    status.cpp 
 
 
BUILDING THE SAMPLE: 
 
    To build this sample program type 'nmake'. 
 
 
SMS API CALLS USED: 
 
    SmsDataSourceConnect 
    SmsDataSourceDisconnect 
    SmsOpenContainer 
    SmsPopulate 
    SmsCloseContainer 
    SmsGetNextFolder 
    SmsGetFolderID 
    SmsUnlinkFolder 
    SmsCommitFolder 
    SmsCloseFolder
