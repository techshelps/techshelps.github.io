Close Method

Description

Closes the ImageFile object. Closing an ImageFile object deletes it and all Page and PageRange objects associated with it are deleted. The Application object no longer has an ImageFile object associated with it.

Syntax

ImageFileObject.Close [savechangesflag]

Data Type

Integer (Boolean)

Remarks

The Close method savechangesflag argument has the following settings:

Setting

Description

True

Changes are saved when the image file closes.

False

(Default) Changes are not saved when the image file closes.
