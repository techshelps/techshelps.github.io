CompressionType Property

Description

Returns this page's compression type. Read-only property.

Usage

ImageFileObject.Pages(pagenumber).CompressionType

Data Type

Short

Remarks

The CompressionType property settings are:

Setting

Description

0

Unknown

1

No Compression

2

Group 3 1D FAX

4

Packed Bits

5

Group 4 2D FAX

6

JPEG


Example

This example returns this page's compression type.

x = Img.Pages(1).CompressionType
