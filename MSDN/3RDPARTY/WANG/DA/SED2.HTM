ImageFile Object Methods

Table 3.2 lists the ImageFile object methods.

Table 3.2 ImageFile Object Methods

Method

Description

AppendExistingPages

Appends existing pages to the end of the ImageFile object.

Close

Closes the ImageFile object.

Help

Displays online Help.

InsertExistingPages

Inserts existing pages in the ImageFile object.

New

Creates a new blank ImageFile object.

Open

Opens the ImageFile object.

Pages

Returns a Page or PageRange object for the ImageFile object.

Print

Prints the ImageFile object.

RotateAll

Rotates all ImageFile object pages (AWD files only).

Save

Saves changes to the ImageFile object.

SaveAs

Saves the ImageFile object under another name.
