Left Property

Description

Sets or returns the distance between the left edge of the physical screen and the Application object's window in pixels. Read/write property.

Usage

ApplicationObject.Left = [setting]

Data Type

Long

Remarks

The Left property must be set prior to opening the ImageFile object. This property only takes effect if the Height, Width, and Top properties are also set.

The Left property only returns the value that you set programmatically prior to opening the window. It does not return changes made to the window after it has been opened.
