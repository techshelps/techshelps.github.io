DisplayScaleAlgorithm Property

Description

Sets or returns the scaling algorithm used for displaying images. Read/write property.

Usage

ApplicationObject.DisplayScaleAlgorithm = [setting]

Data Type

Short

Remarks

The DisplayScaleAlgorithm property settings are:

Setting

Description

0

(Default) Normal decimation.

1

Gray4. 4-bit gray scale. 16 shades of gray.

2

Gray8. 8-bit gray scale. 256 shades of gray.

3

Stamp. Represents the image as a thumbnail.

4

Optimize. Changes the display scale algorithm based on the image type of the displayed image.


Note The DisplayScaleAlgorithm property must be set prior to opening the ImageFile object. For this property to take effect after an image is open, you must reopen the image.
