Open Method

Description

Opens an image file in the parent application window. This associates an image file with the ImageFile object. If the current image file is unsaved, the end-user is prompted to save it before the new image file is opened.

The Imaging application has the focus after an Open. You can reset focus programmatically after an Open, if desired.

Syntax

ImageFileObject.Open [imagefile][, annotationflag][, pagenumber]
[, dialogflag]

Remarks

The Open method uses these parts:

Part

Description

imagefile

Name string of ImageFile object to open.

annotationflag

If True, if the image has annotations, the annotations are displayed. If False, if the image has annotations, the image is displayed without the annotations. The default is True.

pagenumber

Page number in the image file to display.

dialogflag

If True, displays a dialog box that allows the end-user to select a file to open. The default is False.
