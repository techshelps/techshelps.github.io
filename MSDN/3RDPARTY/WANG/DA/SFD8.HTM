ImageResolutionY Property

Description

Sets or returns this page's vertical resolution. Read/write property.

Usage

ImageFileObject.Pages(pagenumber).ImageResolutionY = [setting]

Data Type

Long

Example

This example sets this page's vertical resolution.

Img.Pages(1).ImageResolutionY = 200

This example returns this page's vertical resolution.

YRes = Img.Pages(1).ImageResolutionY
