ImagePalette Property

Description

Sets or returns the image palette used to display an image. Read/write property. The ImagePalette property must be set prior to opening the ImageFile object. For this property to take effect after an image is open, you must reopen the image.

Usage

ApplicationObject.ImagePalette = [setting]

Data Type

Short

Remarks

The ImagePalette property settings are:

Setting

Description

0

(Default) Custom

1

Common

2

Gray8 (256 shades of gray)

3

RGB24 (millions of colors)

4

Black and white
