Edit Property

Description

Sets or returns the Application object's ability to edit the displayed object. It is recommended to set the Edit property prior to opening each ImageFile object. Read/write property.

Usage

ApplicationObject.Edit = [{True|False}]

Data Type

Integer (Boolean)

Remarks

The Edit property settings are:

Setting

Description

True

(Default) Image editing is available.

False

The displayed object cannot be changed.


Note You must set the Edit property prior to opening the ImageFile object. You can only set the Edit property once.
