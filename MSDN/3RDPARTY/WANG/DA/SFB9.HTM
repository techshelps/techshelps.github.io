TopWindow Property

Description

Sets or returns this Application object's top window flag. Read/write property.

Usage

ApplicationObject.TopWindow = [{True|False}]

Data Type

Integer (Boolean)

Remarks

The TopWindow property settings are:

Setting

Description

True

The application is a stay-on-top window

False

(Default) The application is not a stay-on-top window


Example

This example makes the application window a stay-on-top window.

App.TopWindow = True
