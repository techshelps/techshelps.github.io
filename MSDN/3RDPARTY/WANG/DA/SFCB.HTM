InsertExistingPages Method

Description

Inserts page(s) in the ImageFile object.

Page(s) to be inserted must come from an existing file. If the pages being inserted come from an image file of a type different than the active image file, the pages are converted before being inserted. After inserting page(s), all PageRange objects are invalid. You can optionally cause a dialog box to open for the end-user to select a file from which to insert page(s).

Syntax

ImageFileObject.InsertExistingPages [imagefile][, page][, count]
[, pagenumber][, dialogflag]

Remarks

The InsertExistingPages method uses these parts:

Part

Description

imagefile

The image file from which page(s) will be inserted (the source image file).

page

The page to insert before. Page(s) are inserted before the active page if page is not specified.

count

The number of pages to insert. The default is 1.

pagenumber

The page in the source image file from which to start inserting pages.

dialogflag

If True, displays a dialog box that allows the end-user to select a source image file. The default is False.


Example

This example inserts pages 4 and 5 from the file BW.TIF before page 1.

Img.InsertExistingPages "c:\bw.tif", 1, 2, 4

This example inserts page 2 from a file selected by the end-user from a dialog box before page 1.

Img.InsertExistingPages "", 1, 1, 2, True
