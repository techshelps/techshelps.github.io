Print Method

Description

Prints the image file associated with the ImageFile object. You can optionally display a dialog box to allow the end-user to select the print options.

Syntax

ImageFileObject.Print [dialogflag]

Remarks

The Print method dialogflag argument has the following settings:

Setting

Description

True

Displays a dialog box that allows the end-user to select print file options.

False

(Default) No dialog box is displayed.


Example

This example prints the specified image file.

x = Img.Print
