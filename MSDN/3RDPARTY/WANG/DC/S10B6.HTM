ShowAttribsDialog Method

Shows an Annotation Attributes dialog box, which the end user can use to change the attributes of the selected annotation mark.

Applies To

Image Edit control

Syntax

object.ShowAttribsDialog

Arguments

None

Return Value

None

Remarks

An annotation must be selected before invoking the ShowAttribsDialog method.
