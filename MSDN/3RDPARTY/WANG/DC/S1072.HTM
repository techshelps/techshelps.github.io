MouseIcon Property

Returns or sets the mouse icon to be displayed when the mouse pointer is moved over an Image Edit control.

Applies To

Image Edit control

Syntax

object.MouseIcon=LoadPicture(pathname)

Data Type

String

The pathname is a string expression specifying the path and filename of the file containing the custom icon (.CUR or .ICO files).

Remarks

The MouseIcon property provides a custom icon that is used when the MousePointer property is set to 99.
