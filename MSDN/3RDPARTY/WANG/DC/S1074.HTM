Page Property

Returns the page number of an image file where an action was performed, or sets the page number of an image file where an action is to be performed.

Applies To

Image Edit control

Syntax

object.Page[=integer]

Data Type

Long

Remarks

The default value for this property is 1. This property is typically used to specify the page to be displayed by the Display method.
