Start Visual Basic and Add the Imaging Controls

  1. Create a new Project.
  2. Add the Imaging controls to the project by choosing Custom Controls from the Tools menu. Within the Available Controls list box, select the check box next to each of the following items:
    • Wang Image Admin Control
    • Wang Image Edit Control
    • Wang Image Scan Control
    • Wang Image Thumbnail Control
  3. Choose OK to update your Toolbox.
