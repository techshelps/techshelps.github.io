FileType Property

Returns the file format of the image file.

Applies To

Image Admin control

Syntax

object.FileType[=value]

Data Type

Integer (enumerated)

Property settings are expressed using the following values:

Setting

Description

0

Unknown

1

TIFF

2

AWD

3

Bitmap (BMP)

4

PCX

5

DCX

6

JPEG

7

XIF


Remarks

The file format is determined by the Image property.

This property is read-only.
