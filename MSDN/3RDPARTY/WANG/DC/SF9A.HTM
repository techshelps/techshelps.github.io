Overview

The Getting Started application demonstrates how to use the Wang Imaging OLE controls to add image and image annotation functions to a Visual Basic application. Once compiled, it enables users to display, annotate, and manipulate image files. The application uses the following Imaging controls:

· Image Admin

· Image Edit

· Image Thumbnail

Wang Laboratories, Inc., assumes that you are familiar with Visual Basic, Version 4.0. If necessary, refer to your Visual Basic documentation for assistance.
