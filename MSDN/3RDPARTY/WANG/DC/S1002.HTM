Annotation Groups

Annotation groups make it possible to organize and manage annotations, giving the end user the capability to:

· Logically organize annotations into functional groups; for example, Accounts Payable Group, Accounts Receivable Group, Purchasing Group, Receiving Group, and so on.

· Toggle the visibility of the groups; for example, to Turn On Accounts Payable Comments, Turn On All Comments, and so on.

A group can contain one or more annotation marks and multiple groups can be associated with a single image.
