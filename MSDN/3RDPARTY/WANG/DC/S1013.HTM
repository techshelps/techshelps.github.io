Image Files

Images are stored in image files, which are standard Windows files. Image files can be organized in folders, copied, renamed, deleted, and so on, using standard Windows commands.

Depending on the file format, image files can contain one or more images. In multipage image files, each image is stored in a single image page. Image files can be stored on the hard drive of a local PC, or on redirected network drives. Applications can access any available image file by specifying its proper path (drive, folder, and filename).
