Image Property

Returns or sets the filename of the current image.

Applies To

Image Admin control

Syntax

object.Image[=filename]

Data Type

String

Remarks

When a new image is specified, the PageNumber property is set to 1.

This property is used by the following Image Admin control methods:

Append

DeletePages

Insert

VerifyImage

Replace

ShowFileDialog

ShowPrintDialog


When the Image property changes, the following Admin control properties are updated:

CompressionInfo

ImageResolutionX

PageNumber

CompressionType

ImageResolutionY

PageType

FileType

ImageWidth

ImageHeight

PageCount
