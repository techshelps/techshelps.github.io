ThumbCaptionFont Property

Returns or sets the font used to display captions.

Applies To

Image Thumbnail control

Syntax

object.ThumbCaptionFont[=font]

Data Type

OLEFONT

Remarks

The default font is MS Sans Serif, regular weight, 12 point; no italic, underline, or strikethrough.
