PageType Property

Returns the image type of the image specified in the Image property and the page specified in the Page property.

Applies To

Image Edit control

Syntax

object.PageType

Data Type

Integer (enumerated)

Property settings are expressed using the following values:

Setting

Description

1

Black-and-white

2

Gray4

3

Gray8

4

Palettized4

5

Palettized8

6

RGB24

7

BGR24


Remarks

The PageType property value is independent of an image being displayed. If the image page specified in the Page property is being displayed, the value is the image type of the displayed image. If the image page specified in the Image and Page properties is not being displayed, the value is the image type of the page specified in the Image and Page properties.

Available at run-time as read-only.
