GetSelectedAnnotationImage Method

Returns the fully-qualified filename of the image that is being used in a selected image annotation.

Applies To

Image Edit control

Syntax

object.GetSelectedAnnotationImage

Arguments

None

Return Value

String

Remarks

The GetSelectedAnnotationImage method is valid only at run-time. It is used with annotations of the following types:

Image Embedded

Image Reference
