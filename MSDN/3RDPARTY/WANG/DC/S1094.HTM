GetCurrentAnnotationGroup Method

Returns the name of the annotation group to which subsequent annotations will belong.

Applies To

Image Edit control

Syntax

object.GetCurrentAnnotationGroup

Arguments

None

Return Value

String

Remarks

The GetCurrentAnnotationGroup method is valid only at run-time.
