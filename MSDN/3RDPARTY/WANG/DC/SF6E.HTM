Events

The Image Edit control uses the following standard Events:

Click

KeyUp

DblClick

MouseDown

Error

MouseMove

KeyDown

MouseUp

KeyPress


The Error event occurs only when an error condition triggers it. It does not occur when a property is set or a method is invoked.

If you are using Visual Basic, it adds the DragDrop, DragOver, GotFocus, and LostFocus events.

The Image Edit control also uses the following custom Events:

Close

Scroll

Load

SelectionRectDrawn

MarkEnd

ToolPaletteHidden

MarkSelect

ToolTip

PasteCompleted

ToolSelected
