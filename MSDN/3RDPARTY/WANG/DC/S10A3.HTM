RotateLeft Method

Rotates the displayed image 90 degrees to the left.

Applies To

Image Edit control

Syntax

object.RotateLeft

Arguments

None

Return Value

None

Remarks

The Display method must be invoked prior to calling this method.

Invoking the RotateLeft method modifies the displayed image. The Save or SaveAs method must be invoked to save the modified image.
