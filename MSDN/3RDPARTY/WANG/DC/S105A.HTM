AnnotationLineStyle Property

Returns or sets the line style for line and Hollow Rectangle annotation objects.

Applies To

Image Annotation Tool Button control, Image Edit control

Syntax

object.AnnotationLineStyle[=value]

Data Type

Integer (enumerated)

Setting

Description


0 (default)

Transparent -- The underlying image data is visible

1

Opaque -- The underlying image data is obscured


Remarks

The AnnotationLineStyle property is used with the following annotation types:

Freehand Line

Hollow Rectangle

Straight Line
