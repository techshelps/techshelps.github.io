MousePointer Property

Returns or sets the type of mouse cursor displayed when the cursor is positioned over the control.

Applies To

Image Thumbnail control

Syntax

object.MousePointer[=integer]

Data Type

Integer (enumerated)

Setting

    Description

0 (default)
