Thumbnail Control

Private Sub ImgThumbnail1_Click(ByVal ThumbNumber As Long)

ImgEdit1.Display

End Sub

Open Command

Enter the following code to

· Filter the file types for the Open dialog box,

· Invoke the Open dialog box, and

· Display the selected image in the Image Edit and Image Thumbnail controls

when the Click event occurs:

Private Sub mnuOpen_Click()

ImgAdmin1.Filter = "All Image Files(*.tif, *.bmp, *.awd, *.jpg) |

*.tif; *.bmp; *.awd; *.jpg |All files (*.*)|*.*|"

ImgAdmin1.ShowFileDialog OpenDlg

ImgEdit1.Image = ImgAdmin1.Image

ImgEdit1.Display

ImgThumbnail1.Image = ImgEdit1.Image

ImgThumbnail1.GenerateThumb 0

End Sub
