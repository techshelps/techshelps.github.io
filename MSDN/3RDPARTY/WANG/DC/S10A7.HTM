ScrollImage Method

Scrolls a displayed image.

Applies To

Image Edit control

Syntax

object.ScrollImage Direction,ScrollAmount

Arguments

Parameter

        Data Type

                Setting

Direction

Integer (enumerated)

The direction in which to scroll the image:

0 -- Down
1 -- Up
2 -- Right
3 -- Left

ScrollAmount

Long

The number of pixels to scroll the image


Return Value

None

Remarks

The Display method must be invoked prior to calling this method.
