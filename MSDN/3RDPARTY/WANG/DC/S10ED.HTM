Draw the Controls and Set Their Properties

Image Admin Control

  1. Click on the ImgAdmin tool, and then draw the control anywhere on Form1.
  2. Set its properties to suit your requirements, or accept the default values.

Image Edit Control

  1. Click on the ImgEdit tool, and then draw the control on Form1. Draw the control so that it takes up approximately 80 percent of the form. Reserve the remaining space for the Image Thumbnail control.
  2. Set the following properties for the Image Edit control:
    • Height to 4455
    • Width to 6135
  3. Set the other properties to suit your requirements, or accept the default values.

Image Thumbnail Control

  1. Click on the ImgThumbnail tool, and then draw the control on Form1. Draw the control so that its width fills the remaining space on the form, and its height is the same as that of the Image Edit control.
  2. Set the following properties for the Image Thumbnail control:
    • Height to 4455
    • Width to 1335
    • ThumbHeight to 100
    • ThumbWidth to 55
  3. Set the other properties to suit your requirements, or accept the default values.
