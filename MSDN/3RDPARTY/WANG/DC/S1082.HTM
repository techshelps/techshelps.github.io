ClearDisplay Method

Clears a displayed image in the control.

Applies To

Image Edit control

Syntax

object.ClearDisplay

Arguments

None

Return Value

None

Remarks

An image must be displayed before the ClearDisplay method can be invoked.

The ClearDisplay method does not save any changes that may have been made to an existing image, nor does it change the value of the Image property. It does, however, trigger the Close event.
