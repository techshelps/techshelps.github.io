Close Event

This event occurs whenever one of the following methods is invoked:

· Display

· ClearDisplay

· DisplayBlankImage

The Close event can be used to indicate to the end user that the displayed image is about to be closed.

Applies To

Image Edit control

Syntax

Sub object_Close( )

Arguments

None

Remarks

Once a Close event occurs, any changes made to displayed images are lost unless the Save or SaveAs method is invoked.

If desired, your program can check the ImageModified property to see if any changes have been made to the displayed image. If changes have been made, the program can be written so it prompts the end user to save the changes.
