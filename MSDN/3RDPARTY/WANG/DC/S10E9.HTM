Events

The Events used by the Thumbnail control are all standard.

*Click

*MouseDown

*DblClick

*MouseMove

KeyDown

*MouseUp

KeyUp


Events with an asterisk (*) use the additional argument ThumbNumber.

Argument

Parameter

        Data Type

Description

ThumbNumber

Long

The number of the thumbnail on which the event occurred. This parameter is 1 relative (that is, a value of 1 means the event occurred on the thumbnail for page 1).

A 0 (zero) value means an event occurred on the control, but not on a thumbnail.
