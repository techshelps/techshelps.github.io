The Internet Client SDK makes it easy to find what you need. The tools, information, and samples are divided into the following downloadable components. This content is best viewed with Internet Explorer 4.0 or later. Navigate to the Internet Explorer download page to get the latest version.

Dynamic HTML

 Illustrates the innovative features that allow content providers to create compelling content for the Web.

Delivering Content to the Web

  Contains information on new ways Web publishers can deliver their content with Internet Explorer 4.0 and explains several new features using "push" technologies, including Active Channels, Active Desktop items, and Software Update Channels.

Component Development

  Provides the hard-core details you need to create ActiveX Controls and Java applets for your own applications.

Internet Tools & Technologies

  Describes the services and technologies provided with Internet Explorer 4.0 and explains how to exploit them in your own components and applications.

Component Library

  Contains documentation on reusable components that can be used to enhance your Web pages.
