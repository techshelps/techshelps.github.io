System Requirements for the Address Book Application

To set up the Address Book sample application, you need to meet the following software and database requirements.

Software Requirements

The server computer software requirements for running this Web application include:

  • Microsoft® Windows NT® Server x86, version 4.0 Service Pack 3 or later.

  • Microsoft Internet Information Server (IIS) 3.0 or later, with Active Server Pages.

  • Server-side DLLs for Remote Data Service. (Remote Data Service setup does this for you automatically.)

The client computer software requirements for running this Web application include:

  • Microsoft Internet Explorer 4.0 or later.

  • Microsoft Windows NT Workstation or Microsoft Windows® 95.

  • Client-side DLLs for Remote Data Service installed on the client computer. They are already included as part of the Internet Explorer 4.0 installation.

See Also For more information about installing Remote Data Service, see "Setting Up Remote Data Service."

Database Requirements

To use this sample, you must have:

  • An operational Microsoft SQL Server version 6.5 database.

  • Access permissions to the database.

  • Write privileges to populate the database with the sample data.

  • Verification of the populated data through Enterprise Manager or ISQL utilities.

If you don't have privileges, your database administrator may need to set up the system and give you access permission to the database, or set up the database for you.

Once you meet these database requirements, follow the instructions in "Running the Address Book SQL Script" and "Setting Up an ODBC Connection for Address Book.”
