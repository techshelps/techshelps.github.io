Refresh Method Example

The following example shows how to set the necessary parameters of RDS.DataControl at run time. The manner in which a Recordset is retrieved using the Refresh method is determined by the settings of the ExecuteOptions and FetchOptions properties. To test this example, cut and paste this code between the <Body></Body> tags in a normal HTML document and name it ADCapi2.asp. ASP script will identify your server.
