Getting Assistance While You Work (Remote Data Service)

You can get assistance while you work by:

  • Consulting the documentation.

  • Contacting Microsoft® Technical Support.

  • Visiting the Remote Data Service home page on the World Wide Web.

  • Joining the Remote Data Service public newsgroup.

  • Exploring the Microsoft Knowledge Base.

Consulting the Documentation

The Remote Data Service documentation includes the following pieces.
PieceDescription
Getting StartedExplains how to install Remote Data Service and gives a brief tour of the product and documentation.
TutorialsShow how to build simple, data-aware Web applications.
Developer's GuideDiscusses Remote Data Service technology in greater detail. It describes the architectural underpinnings, components, and data flow mechanisms, and how to customize the RDS server-side components.
API ReferenceProvides reference topics for all of the Remote Data Service objects, methods, and properties, including syntax, quick reference information, and code examples.
Sample applicationsDemonstrates how to use Remote Data Service to build intranet and Internet applications.
Readme.txtExplains how to install the Remote Data Service client and server components and specifies the hardware and software requirements. It also provides late-breaking product information not included in the core documentation, and lists known problems.

Contacting Microsoft Technical Support

For technical support on Remote Data Service, the following options are available.

In the United States and Canada, Microsoft Technical Support provides Pay-Per-Incident, call back only support for the Remote Data Service.

  • Call (800) 936-5800; $95 (U.S.) per incident, billed to your VISA card, MasterCard, or American Express card.

  • Call (900) 555-2300; $95 (U.S.) per incident. Charges appear on your telephone bill.

Visiting the Remote Data Service Web Site

You can visit the Remote Data Service home page on the World Wide Web for documentation updates and information about other related technologies. The address is http://www.microsoft.com/adc/.

Joining the Remote Data Service Public Newsgroup

To facilitate discussion, information sharing, and tips trading, Microsoft has set up a public newsgroup: microsoft.public.adc. You are welcome to post articles and messages to this unmoderated, unsupported newsgroup. Microsoft doesn't guarantee responses or direct support. For more information, visit the Remote Data Service Web site, http://www.microsoft.com/adc/.

Exploring the Microsoft Knowledge Base

You can consult the Microsoft Knowledge Base to access a comprehensive collection of more than 70,000 detailed articles with technical information about Microsoft products, bug and fix lists, and answers to commonly asked technical questions.

To get information about Remote Data Service and Internet Information Server

  1. Go to the http://www.microsoft.com/ home page, and click the Support icon.

  2. Click Microsoft Technical Support, and then click Knowledge Base.

  3. Choose the Internet Information Server product, and enter a search phrase, such as "Remote Data Service."
