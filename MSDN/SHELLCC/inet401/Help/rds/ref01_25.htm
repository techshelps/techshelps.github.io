CreateObject Method

Creates the proxy for the target business object and returns a pointer to it. The proxy packages and marshals data to the server-side stub for communications with the business object to send requests and data over the Internet. For in-process component objects, no proxies are used, so just a pointer to the object is provided.

Syntax

Remote Data Service supports the following protocols: HTTP, HTTPS (HTTP over Secure Socket Layer ), DCOM, and in-process.
ProtocolSyntax
HTTPSet object = DataSpace.CreateObject("ProgId", "http://awebsrvr")
HTTPSSet object = DataSpace.CreateObject("ProgId", "https://awebsrvr")
DCOMSet object = DataSpace.CreateObject("ProgId", "machinename")
In-processSet object = DataSpace.CreateObject("ProgId", "")

PartDescription
objectAn object variable that evaluates to an object that is the type specified in ProgID.
DataSpaceAn object variable that represents an RDS.DataSpace object used to instantiate the new object.
ProgIDA String that is the programmatic ID that identifies a server-side business object that implements your application's business rules.
awebsrvr or machinenameA String that represents a URL identifying the Internet Information Server (IIS) Web server where an instance of the server business object is created.

Applies To

RDS.DataSpace
