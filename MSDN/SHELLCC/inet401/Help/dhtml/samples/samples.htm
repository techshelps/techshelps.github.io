Samples and Tools



Samples

The following samples demonstrate various features of Dynamic HTML.

Dynamic Styles   Demonstrates an example of Dynamic Styles. Change text colors, font styles, visibility, and even cursors without refreshing the page!

Dynamic Content   Demonstrates an example of Dynamic Content. Stay up-to-the-minute with a DHTML clock. Change the text on the page with the click of a mouse.

Positioning

  • Position  Demonstrates 2-D layering of moving images with a mouse.
  • Asteroids  You want interactivity? You've got it with a fully functional Dynamic HTML version of Asteroids! This demonstration takes advantage of positioning and JScript to deliver shoot-em-up action.

Centering  Demonstrates centering an object within a division on the page.

Resizing

  • Sample1  Demonstrates various samples of resizing capabilities.
  • Sample2  Demonstrates the resizing of div, image, and text.

Image  Switches images using preloaded images.

Domain  Illustrates accessing a second-tier domain in an IFRAME.

Replacing Custom Controls with Dynamic HTML

  • Path    Demonstrates how to replace the path control through the use of scripting.
  • Pop-up Menu    Demonstrates how to create a pop-up menu through the use of Dynamic HTML.

DirectAnimation Controls

  • MMAudio  Illustrates mixing several .wav files.
  • Structured Graphics Control
    • MMSGX  Demonstrates how to use a structured graphic control.
    • Surface  Shows how the DirectAnimation DrawingSurface API is used to create a single vector shape with independent internal and external rotations.
  • Path Control
    • Tricycle   Shows a sprite control on a path, and allows you to control its behavior.
  • Sprite Control
    • Sprite  Demonstrates the ability to add still and animated images on your Web page, with control over frames per second, forward or reverse playback, or dynamic reading of source files.
    • Banner  Demonstrates the ability to use the sprite control as a clickless control for a banner-type advertisement.
    • Concentrate  Demonstrates the ability to use the sprite control to contain images for this match game.
    • Swapgame  Demonstrates the ability to use the sprite control to contain images for puzzle-type games.
  • Sequencer Control
    • Sequencer-JScript  Demonstrates how to use the sequence control through JScripting.
    • Sequencer-VBScript  Demonstrates how to use the sequence control through VBScripting.

Filters and Transitions

  • Filters    
    • Filters    Demonstrates the different filters.
  • Transitions    
    • Transitions    Demonstrates the different transitions.
    • Blend Transition    Demonstrates the blend transition.

Tools

Tools for Debugging Scripts

  • Microsoft Script Debugger for Internet Explorer

Wizards for Generating Simple Filters and Transitions Pages

  • Filter Wizards    
    • Filters Wizard  HTML-based wizard that generates working code for the different filters, based on the parameters entered by the user.
    • Wave Filter Wizard  HTML-based wizard that generates working code for the wave filter, based on the parameters entered by the user.
    • Light Filter Wizard  HTML-based wizard that generates working code for the light filter, based on the parameters entered by the user.
  • Transitions Wizard    
    • Transitions Wizard  HTML-based wizard that generates basic code for the different transitions, based on the parameters entered by the user.

CSS Syntax Validation Tool

  • StyleT 2.0 Guide    Provides information on the StyleT 2.0 Test Tool. This test tool performs a series of tests to ensure that the syntax of the files matches the CSS Specification. StyleT.htm is available in the \bin directory of the Internet Client SDK installation.

Wizards for Generating Simple DirectAnimation Control Pages

  • Path Wizard  HTML-based wizard that generates working code for moving an object around, based on the parameters entered by the user.
  • Graphics  HTML-based wizard that generates working code (for the Structured Graphics and Sequencer controls) that produces a rotating text logo, based on the parameters entered by the user.
