Renumber Resource Id's Dialog Box

Use the Renumber Resource Id’s dialog box to change a selected block of resource IDs in a string table. Resource IDs can be renumbered sequentially, or by an offset from a base number.

You can open the Renumber Resource Id’s dialog box by clicking the Renumber Rows toolbar button when two or more resource ID’s are selected in the String Table editor.

Toolbar Shortcut:

Dialog Box Elements

Renumber Id’s

Renumbers the selected resource IDs sequentially.

Rebase Id’s

Renumbers the selected resource IDs using an offset.

Base Value

Specifies the new starting number.
