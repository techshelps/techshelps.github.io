Cursor Resources

A cursor resource is a cursor file (.cur) stored within a resource file. Each cursor resource is identified by a unique index or resource ID. Cursors are extracted from a resource file at run-time using the LoadResPicture function, which takes the index and the format (2 or vbResCursor) as arguments.

Cursor resources provide a convenient way to store custom cursors for an application while keeping the source files hidden from the user. You can use the Image Edit application (available from the Tools directory of the Visual Basic CD) to edit or create cursors.
