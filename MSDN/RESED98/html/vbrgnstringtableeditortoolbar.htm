String Table Editor Toolbar

The toolbar has buttons that act as shortcuts for various operations of the String Table Editor. Pausing the mouse pointer over a button will display a description of that button. Clicking a button will cause the associated operation to occur.

The String Table Editor toolbar buttons are:

Button Description
Inserts a new row at the bottom of the string table.
Inserts a new string table (column)
Displays the language selection list box for the selected string table
Opens the Renumber Resource Id’s dialog box
Cuts the selected text
Copies the selected text
Pastes text into the string table
Undoes the last operation
Deletes the selected row
Deletes the selected string table
Displays String Table Editor Help
