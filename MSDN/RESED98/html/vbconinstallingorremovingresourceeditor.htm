Loading or Unloading the Resource Editor

The Resource Editor add-in is loaded from the Add-In Manager dialog box, available from the Add-Ins menu. You can load the Resource Editor by selecting Resource Editor in the Available Add-Ins list and selecting the Loaded/Unloaded check box; you can unload it by clearing the check box. When you select the Resource Editor add-in, it is loaded every time you restart Visual Basic until you clear the check box.

When the Resource Editor is loaded, it adds a Resource Editor button to the Standard toolbar. It also adds two new menu commands: Add New Resource File on the Project menu, and Resource Editor on the Tools menu.
