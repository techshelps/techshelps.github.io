Opening a Resource File

An existing resource file (.res) can be opened for editing in the Resource Editor by choosing Add New Resource File from the Project menu, or by clicking the Open Resource File toolbar button on the Resource Editor toolbar. The selected resource file will be added to the current project in the Project Explorer.

A new resource file can be created by clicking the New Resource File toolbar button on the Resource Editor toolbar. It will not appear in the Project Explorer until it has been saved.
