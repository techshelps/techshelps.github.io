String Table Editor Window (Resource Editor Add-in)

Use the String Table Editor window to add, delete, and edit string table resources in a resource file (.res) associated with your project.

You can open the String Table Editor window from:

  • The Edit String Tables button on the Resource Editor toolbar.

  • The list window, by double-clicking a String Table resource.

Window Elements

Toolbar

Contains buttons representing shortcuts for various operations of the String Table Editor.

String Table Grid

Displays resource IDs and string resources for all string tables in a resource file.
