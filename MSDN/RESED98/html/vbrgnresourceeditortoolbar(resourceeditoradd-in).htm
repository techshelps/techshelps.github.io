Resource Editor Toolbar (Resource Editor Add-in)

The toolbar has buttons that act as shortcuts for various operations of the Resource Editor add-in. Pausing the mouse pointer over a button will display a description of that button. Clicking a button will cause the associated operation to occur.

The Resource Editor toolbar buttons are:

Button Description
Adds a new resource file, replacing the current file
Displays the Open A Resource File dialog box
Saves the current resource file and updates the copy in the current project
Opens the Properties dialog box for the currently selected resource
Duplicates the selected resource and assigns the next available resource ID
Undoes the last operation
Deletes the selected resource
Opens the String Table Editor
Opens the Add Cursor dialog box
Opens the Add Icon dialog box
Opens the Add Bitmap dialog box
Opens the Add Custom Resource dialog box
Displays Resource Editor Help
