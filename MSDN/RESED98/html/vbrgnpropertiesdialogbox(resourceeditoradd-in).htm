Properties Dialog Box (Resource Editor Add-in)

Use the Properties dialog box to edit resource properties in a resource file (.res) associated with your project. The properties that may be edited are dependent on the resource type. The language ID may be edited for all resource types, the resource ID may be edited for all types except string table resources, and the type property may be edited for custom resources.

You can open the Properties dialog box by clicking the Edit Properties toolbar button when a resource is selected in the list window.

Toolbar shortcut:

Dialog Box Elements

ID

Contains the resource ID for the selected resource.

Language

Contains a drop-down list of language IDs for the selected resource.

Preview

Displays an image of the selected resource (bitmap, icon, and cursor resources only).

Type

Contains the type name for the selected custom resource.
