Editing a Resource File

Resources within a resource file (.res) are added, deleted, and edited in the Resource Editor window. The techniques are basically the same for all resource types; some aspects of string table resources are handled differently.

To add a cursor, bitmap, icon, or custom resource, click the appropriate toolbar button on the Resource Editor toolbar. A File Open dialog box will open, allowing you to select an existing resource. The new resource will be assigned the next available resource ID for that resource type and a language ID based on the system LCID of your computer. Custom resources will be added with their type listed as "Custom".

To add a new string table resource, click the Edit String Table toolbar button. A resource files can only contain one string table resource which can contain multiple string tables. You can add additional string tables to an existing string table resource, or you can delete the existing string table resource and start a new one.

To edit a resource entry, select it in the Resource Editor tree view and click the Edit Properties toolbar button. This will display the Edit Properties dialog box, where you can edit the language ID for all resource types, the Resource ID for any type except string table resources, and the type name for custom resources. String table resources are edited in the String Table Editor. The Edit Properties dialog box also shows a thumbnail preview of bitmaps, cursors, and icons.

To add a duplicate cursor, bitmap, icon, or custom resource, select the resource to be duplicated and click the Duplicate toolbar button. A new copy of the resource will be added with the next available resource ID for that resource type.

To delete a resource of any type, click the Delete toolbar button.

Note   The Resource Editor will not allow entry of resource IDs greater than 32767. This is due to a limitation of the LoadResource functions in Visual Basic. Resources created in another editor may have resource IDs as high as 65535. You can open these files in the Resource Editor, however you will not be able to edit any resource with an ID greater than 32767.
