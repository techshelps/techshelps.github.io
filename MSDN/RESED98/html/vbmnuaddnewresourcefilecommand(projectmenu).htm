Add New Resource File Command (Project Menu)

Displays the Open A Resource File dialog box so you can insert an existing resource file (.res) into your active project. Each project can contain only one resource file. If a project already contains a resource file it will be replaced with the selection.

The Add New Resource File command is only visible when the Resource Editor add-in is loaded.

Note   A resource file can also be added to a Project using the Add File command on the Project Menu.
