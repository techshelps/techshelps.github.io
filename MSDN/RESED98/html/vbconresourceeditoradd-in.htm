Resource Editor Add-in

The Resource Editor add-in adds the capability to work with resource files (.res) within the Visual Basic integrated development environment. Resource files are compiled automatically by the Resource Editor, so you don’t need a separate resource compiler or .rc files.

You can use the Resource Editor to:

  • Add a resource file (.res) to your project

  • Add new string table, cursor, bitmap, icon, or custom resources to a resource file.

  • Edit the properties of resources within a resource file.

  • Delete resources from a resource file.

For More Information   To learn more about resource files and how they can be used in a project, see "Working with Resource Files" in the Visual Basic Programmer’s Guide.
