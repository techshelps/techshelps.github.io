Starting the Resource Editor

The Resource Editor add-in can be started in three different ways:

  • From the Project window, by double-clicking on a resource file, or by selecting a resource file and pressing the ENTER key.

  • From the Tools menu, by clicking the Resource Editor command.

  • From the Resource Editor toolbar button on the Visual Basic Standard toolbar.

Note   The Resource Editor check box must be loaded in the Add-In Manager dialog box in order for the Resource Editor to be visible.
