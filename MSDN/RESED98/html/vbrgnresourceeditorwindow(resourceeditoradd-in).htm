Resource Editor Window (Resource Editor Add-in)

Use the Resource Editor window to add, delete, and edit resources in a resource file (.res) associated with your project. Only a single resource file may be edited at a time; a project can contain only a single resource file.

The Resource Editor window is dockable.

You can open the Resource Editor window from:

  • The Project window, by double-clicking on a resource file, or by selecting a resource file and pressing the ENTER key.

  • The Tools menu, by selecting the Resource Editor command.

  • The Resource Editor toolbar button on the Visual Basic Standard toolbar.

Note   The Resource Editor add-in must be selected in the Add-In Manager dialog box in order for the Resource Editor to be visible.

Window Elements

List Window

Displays a hierarchical list of resource types and individual resources contained in the selected resource file.

Toolbar

Contains buttons representing shortcuts for various operations of the Resource Editor.

Status Bar

Displays an icon and a message to indicate that the file has been changed but not saved.
