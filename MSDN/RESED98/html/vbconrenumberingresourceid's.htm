Renumbering Resource Id's

Sometimes you may find it necessary to renumber some of the resource IDs for strings in a string table resource. For example, you may have grouped all strings for a particular form by numbering them 1–25. If you now find that you need 26 or more strings, you may want to move them to a larger block of numbers. The Renumber Resource dialog box, available from the Renumber Rows toolbar button, allows you to do this.

There are two options for renumbering resource IDs:

  • The Renumber Id’s option will renumber sequentially from the base value. For example, 101, 102, 106 would become 1001, 1002, 1003 (with a base value of 1000).

  • The Rebase Id’s option will renumber from the base value and maintain the existing offset. For example, 101, 102, 106 would become 1001, 1002, 1006 (with a base value of 1000).
