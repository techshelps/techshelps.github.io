Controlling the CEC

There are two ways to control the CEC: calling methods and setting properties.

There are methods that can perform the following tasks:

  • Create a PKCS #10 certificate request, either as a file or as an object in memory.
  • Receive certificates from a certificate server and store them in a certificate store or save them as a file.
  • Enumerate containers and providers.

For method details, see Certificate Enrollment Control Methods.

The methods use some properties to determine how to process the creation of a certificate request, and for processing a received certificate. For a detailed list of the properties, see Certificate Enrollment Control Properties.
