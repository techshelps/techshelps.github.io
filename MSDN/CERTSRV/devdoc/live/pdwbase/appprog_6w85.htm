Using the Certificate Server Programmer's Reference

Developers using the Certificate Server Programmer's Reference should first become familiar with the Certificate Server architecture and theory of operation as discussed in Certificate Server Concepts. Portions of this material assume that the reader has a basic familiarity with public-key cryptography, digital certificates, and authentication. Readers who are new to these topics are advised to read Overview of Certificates and Authentication in Certificate Server Concepts before proceeding.

For explanations of acronyms and other technical terms, please refer to the Glossary of Terms.

Note  The Certificate Server Release Notes contain important additional information about Certificate Server's programmable interfaces and may contain corrections to errors in this documentation release. The Certificate Server Release Notes are located in the Microsoft Windows NT 4.0 Option Pack documentation under Getting Started.
