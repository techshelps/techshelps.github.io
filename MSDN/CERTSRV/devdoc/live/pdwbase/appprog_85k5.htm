ICertEncodeStringArray::SetValue

The SetValue method sets a string value at the specified index of a string array.

[VB] void SetValue(
  long Index,  
  BSTR str 
);
 
[JAVA] void SetValue(
  int Index,                
  java.lang.String str  
);
 
[C++] HRESULT SetValue(
  LONG Index,         // in
  BSTR const str      // in
);

Parameters

[VB][JAVA][C++] Index
Specifies the index into the string array at which to set a string value.
[VB][JAVA][C++] str
Specifies the string value to set.

Return Values

This method does not return a value.
