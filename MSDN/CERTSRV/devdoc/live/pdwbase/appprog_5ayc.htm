ICertEncodeAltName::Reset

The Reset method resets an alternate name array to a specified number of elements. The values of all the elements are set to 0.

[VB] void Reset(
  long NameCount  
);
 
[JAVA] void Reset(
  int NameCount                
);
 
[C++] HRESULT Reset(
  LONG NameCount         // in
);

Parameters

[VB][JAVA][C++] NameCount
Specifies the count of elements to be initialized.

Return Values

This method does not return a value.
