Microsoft Internet Security Framework

Microsoft® Certificate Server is part of the Microsoft Internet Security Framework (ISF), which is an integral part of the Microsoft Windows® security model. The integration of ISF on the Microsoft Windows NT® platform allows certificates to be mapped to Windows NT users and groups. Network administrators can then use standard Windows NT administrative tools, such as access control lists, to handle Internet and intranet security matters. Windows NT users benefit from the convenience of a single logon to access organization resources on the corporate network, corporate intranet, or the Internet.
