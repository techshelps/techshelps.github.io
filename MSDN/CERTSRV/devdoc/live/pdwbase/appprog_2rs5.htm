ICertEncodeDateArray::GetValue

The GetValue method returns the date value at a specified index of a date array.

[VB] DATE GetValue(
  long Index  
);
 
[JAVA] double GetValue(
  int Index    
);
 
[C++] HRESULT GetValue(
  LONG Index,            // in
  DATE * pValue          // out
);

Parameters

[VB][JAVA][C++] Index
Specifies the index into the date array from which to get the date value.
[C++] pValue
Points to the return value. See Return Values.

Return Values

Returns the date value in the date array at the specified index.
