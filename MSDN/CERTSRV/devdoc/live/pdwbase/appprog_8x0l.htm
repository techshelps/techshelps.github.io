ICertServerPolicy::EnumerateAttributesClose

The EnumerateAttributesClose method frees any resources connected with attribute enumeration. All applications that use EnumerateAttributes should call this when finished enumerating.

[VB] void EnumerateAttributesClose(void);
 
[JAVA] void EnumerateAttributesClose(void);
 
[C++] HRESULT EnumerateAttributesClose(void);

Parameters

None.

Return Values

This method does not return a value.
