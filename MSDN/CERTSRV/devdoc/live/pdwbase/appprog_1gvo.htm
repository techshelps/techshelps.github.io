ICertEncodeLongArray::Reset

The Reset method resets a long array to a specified number of elements. The values of all the elements in the array are set to 0.

[VB] void Reset(
  long Count  
);
 
[JAVA] void Reset(
  int Count                
);
 
[C++] HRESULT Reset(
  LONG Count         // in
);

Parameters

[VB][JAVA][C++] Count
Specifies the number of elements in the long array.

Return Values

This method does not return a value.
