ICertEncodeStringArray::GetValue

The GetValue method returns the string value at a specified index of a string array.

[VB] BSTR GetValue(
  long Index  
);
 
[JAVA] java.lang.String GetValue(
  int Index    
);
 
[C++] HRESULT GetValue(
  LONG Index,            // in
  BSTR *pstr             // out
);

Parameters

[VB][JAVA][C++] Index
Specifies the index into the string array from which to get the string value.
[C++] pstr
Points to the return value. See Return Values.

Return Values

Returns the string value in the string array at the specified index.
