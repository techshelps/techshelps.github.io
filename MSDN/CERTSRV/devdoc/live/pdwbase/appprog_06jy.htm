Interface Documentation

For each interface a general description is provided and then the interface's methods are described. For each method, the syntax for all three languages: Microsoft® Visual Basic®, Java, and C++, is shown. The format of the method descriptions resembles C/C++ function prototypes, and is used uniformly for all languages to present information about the calling syntax. The symbols [VB], [JAVA], and [C++] are shown beside each item to indicate which language uses that particular syntax. Any items that have no symbols beside them pertain to all three languages.
