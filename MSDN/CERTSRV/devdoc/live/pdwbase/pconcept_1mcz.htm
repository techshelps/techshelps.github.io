Web Browsers

Microsoft® Certificate Server can be used to issue certificates to browsers which support client authentication, such as Microsoft Internet Explorer version 3.0 or later, or Netscape Navigator 3.0. Internet Explorer can install server certificates for Microsoft Internet Information Server (IIS) and other Web servers used for server authentication such as Netscape Enterprise Server. Internet Explorer also uses certificates for client authentication on servers that the user wants to access. You can view and examine the details of certificates that are installed in Internet Explorer.

    Complete these steps to view certificates installed in Internet Explorer:
  1. Start Internet Explorer.
  2. If running Internet Explorer 3.x, on the View menu, click Options; then click the Security tab. If running Internet Explorer 4.x, on the View menu, click Internet Options; then click the Content tab.
  3. If running Internet Explorer 3.x, click Personal to view client certificates or Sites to view server certificates. If running Internet Explorer 4.x, click Personal to view client certificates or Authorities to view server certificates.
  4. If there are any certificate names shown in the list box, click on a name; then click the View Certificate button.
  5. Click on each item in the Fields list box to view the details of the certificate's fields in the Details text box.
