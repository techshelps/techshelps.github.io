ICertEncodeStringArray::Encode

The Encode method performs ASN encoding on a string array stored in the COM object and returns the ASN encoded string array. The Reset method must first be used to size the array, and the SetValue method must be called to set each element in each distribution point structure.

[VB] BSTR Encode(void);
 
[JAVA] java.lang.String Encode(void);
 
[C++] HRESULT Encode(
  BSTR *pstrBinary  // out
);

Parameters

[C++] pstrBinary
Points to the return value. See Return Values.

Return Values

Returns the ASN encoded string array.
