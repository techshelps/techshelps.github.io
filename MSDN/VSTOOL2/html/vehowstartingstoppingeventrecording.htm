Starting and Stopping Event Recording

You start, pause, and stop event recording on a specific event log. The Record Events button is a two-state button. That means it is turned on until you explicitly turn it off by clicking it. Pausing is not the same thing as stopping and does not affect the Record Events button.

To start recording into an event log

  1. Select the event log into which you want to record events.

  2. Click Analyzer, and then click Record Events.

    You can also click the Record Events button on the Visual Studio Analyzer toolbar, or right-click the event log to display the shortcut menu and then click Record Events.

To pause recording into an event log

  1. Select the event log into which you are recording events.

  2. Click Analyzer, and then click Pause Event Recording. You can also click the Pause Event Recording button on the Visual Studio Analyzer toolbar, or right-click the event log to display the shortcut menu and then click Pause Event Recording.

To resume recording into an event log

  1. Select the event log into which you want to record events.

  2. Click Analyzer, and then click Pause Event Recording.

    You can also click the Pause Event Recording button on the Visual Studio Analyzer toolbar, or right-click the event log to display the shortcut menu and then click Pause Event Recording.

To stop recording into an event log

  1. Select the event log into which you want to stop recording events.

  2. Click Analyzer, and then click Record Events.

    You can also click the Record Events button on the Visual Studio Analyzer toolbar, or right-click the event log to display the shortcut menu and then click Record Events.
