Microsoft Components That Generate Visual Studio Analyzer Events

   

Several Microsoft-supplied technologies support Microsoft® Visual Studio™ Analyzer. The latest versions of these technologies are able to generate Visual Studio Analyzer events.

If your distributed applications rely on any of these technologies, you can use Visual Studio Analyzer to see the interactions between these technologies and your application, in most cases without changing any of your application code.

The following sections provide more information about each of these Microsoft-supplied technologies and the specific Visual Studio Analyzer events they generate.
