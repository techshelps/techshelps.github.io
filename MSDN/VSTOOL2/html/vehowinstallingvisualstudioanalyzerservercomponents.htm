Installing the Visual Studio Analyzer Server Components

You must install the Visual Studio Analyzer Server components on each Windows NT and Windows 95 machine in your network that you plan to monitor using Visual Studio Analyzer. If you install the Visual Studio Enterprise Edition on those machines the server components are installed automatically.

If you do not want to install all of Visual Studio on a machine, you can choose to install only the Visual Studio Analyzer Server components.

To install the Visual Studio Analyzer Server components

  1. Run the Visual Studio setup program.

  2. Step through the Installation Wizard screens until you reach the screen with Client - Server Setup Options.

  3. Select Server Setup.

  4. Click Next.

  5. Select Server Component Setup, and then click Install.

  6. Select Custom, and then click Next.

  7. Click Visual Studio Analyzer, and then click Next.

  8. Step through the rest of the Installation Wizard by clicking Next until you get to the point where the setup program installs the components you selected.
