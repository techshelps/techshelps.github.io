RegisterStockEvent

         

Registers each system-defined (stock) event a component might generate. See System-defined Events for a list of the system-defined events Visual Studio Analyzer supports. Microsoft defines stock events; you define custom events.

Syntax

Visual Basic object.RegisterStockEvent(guidSourceID As String, guidEventID As String)
Visual C++ HRESULT RegisterStockEvent([in] REFGUID guidSourceID,
[in] REFGUID guidEventID);
Visual J++ object.RegisterStockEvent(String guidSourceID, String guidEventID);

Parameters

object

A unique instance of the installer object.

guidSourceID

A unique identifier for this component.

guidEventID

A unique identifier for this event.
