Playing Back an Event Log

You can play back an event log to see the sequence in which events occurred and the interactions between components. Playback works on all Visual Studio Analyzer views. Using playback in the Block Diagram highlights the interactions between components.

To play back an event log

  1. Display the event log in the Visual Studio Analyzer view you choose, such as the Block Diagram.

    Note   You can also combine views. A combination of views you might find useful is the Block Diagram and the Event List, which shows you both the sequence of events as they occur and the interactions between the components that generate the events.

  2. Click the Analyzer menu, and then click Replay Events.

Through other commands on the Analyzer menu, you can control the playback. You can pause or stop playback, move to the first or last event in the event log, single-step from event to event, and increase or decrease the playback speed. You can also find these commands on the Analyzer toolbar.
