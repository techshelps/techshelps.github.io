Summary View

   

Use the Summary view to get additional information on a selection in any Visual Studio Analyzer view. For example, you can use Summary view with Chart view to see how Performance Monitor data relates to what is happening in your application. You could select the time period in which a spike in processor usage occurs and then open the Summary view for detailed information on what is happening with your application during that time.

Options

AutoSelect

Controls whether the Summary view is linked to other open views, so that when you select an event in any open view, the same event is also selected in the Summary view.
