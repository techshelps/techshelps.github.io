The name you have entered is either not a valid filename, or does not contain a file extension

   

Either the specified file name is invalid or it includes a dot but not a file extension. If you do not place the dot, Visual Studio Analyzer fills in the extension for you. Visual Studio Analyzer file names conform to standard file name requirements.

Correct the file name, or specify one of the following extensions:

  • .vaf for a Visual Studio Analyzer filter

  • .val for a Visual Studio Analyzer event log
