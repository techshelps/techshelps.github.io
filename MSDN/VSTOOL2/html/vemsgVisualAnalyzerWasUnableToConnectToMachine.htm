Visual Studio Analyzer was unable to connect to the machine. Ensure that it is powered

   

Visual Studio Analyzer cannot connect to a machine for monitoring for reasons other than the security settings on the machine.

Try the following:

  • Confirm that the machine is powered and connected to the network, and that the Visual Studio Analyzer server-side components are installed correctly.

  • Reinstall the Visual Studio Analyzer server-side components if necessary by running the Visual Studio Installation Wizard and selecting the Server setup option.

  • Confirm that Visual Studio Analyzer server has been started.
