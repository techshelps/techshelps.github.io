Import failed

   

Visual Studio Analyzer was unable to import a selected file. In addition to errors such as insufficient space in the target event log, this message can occur when the file you are trying to import is not in the correct format. Files you want to import into Visual Studio Analyzer must be standard comma-separated values (.csv) format.

Try the following:

  • Import the file into a different event log

  • Open the file using a text editor and try to determine what is wrong with it.
