Event View

   

Use the Event view to see detailed information on a selected event. All information in the event log associated with a particular event is displayed as either text or binary data. You can use the Event view in conjunction with the Block Diagram, Event List, Timeline, or Chart views when you are trying to isolate a fault. Open the Event view on top of the other views and step through your application. The Event view will display details for each event you select.

Options

AutoPlay Track

Controls whether the Event view changes during playback to contain detailed event information for each event as playback progresses.

AutoSelect

Controls whether the Event view is linked to other open views, so that when you select an event in any open view, the Event view shows the details of that event.
