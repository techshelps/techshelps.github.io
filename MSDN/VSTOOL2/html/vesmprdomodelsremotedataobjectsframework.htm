RDO: Models the Remote Data Objects Framework

   

Click to open or copy the RDO model and project files.

The RDO sample shows you a model of the Remote Data Objects (RDO) framework. This framework provides code to create and manipulate components of a remote database system. By reverse engineering the RDO framework into the model of the client/server application you are developing, you can show, in the model, how the application uses the RDO objects to access the database.

For ease of reading, the RDO model has been separated into two logical packages, one containing only constants, and the other containing the RDO objects.

See http://www.microsoft.com/vbasic/techinfo for more information on RDO.

The RDO model was built with Microsoft Visual Modeler, version 2.0.
