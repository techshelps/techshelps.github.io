Connect to Machine Dialog Box

   

Use the Connect to Machine dialog box to identify machines that you want to monitor for Visual Studio Analyzer events. The Visual Studio Analyzer Server components must be installed on any Windows machine you want to monitor. If you are connecting to a remote machine running Windows 95, make sure that the Visual Studio Analyzer Server has been started.

Options

Browse

Displays a dialog box through which you can browse the network for machines.

Connect

Connects to the machine you specified and adds the machine to the Machines node in the Project Explorer.
