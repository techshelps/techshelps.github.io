Visual Studio Analyzer Import From and Export To File Dialog Box

   

Use the Export To File dialog box to identify an event log that you want to export to a standard comma-separated variable (.csv) file. You can then open this .csv file in Microsoft® Access or Microsoft® Excel for further analysis.

Options

File name

Provides a name for the exported file.

Save as type

Specifies that the file is saved as a standard comma-separated variable (.csv) file. You cannot change this field.

Save in

Identifies the location where you want to save the exported file.
