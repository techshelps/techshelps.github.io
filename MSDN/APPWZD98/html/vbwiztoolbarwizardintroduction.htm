Toolbar Wizard—Introduction

Helps you to create a new toolbar when you add the Toolbar control to a form.

Wizard Options

Profile List

Allows you to choose a wizard profile from the list of previously saved profiles. A wizard profile (.rwp) is a file that contains the settings you chose and saved when you previously ran the Toolbar Wizard. To run the Toolbar Wizard without previously saved settings, choose (None).

Profiles are useful if you create multiple applications with the same toolbar. You do not have to remember and choose the same options every time you create toolbar.

Displays the Open Profile dialog box where you can locate and open a profile.

Help

Displays the help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Toolbar Wizard.

Back

Disabled in this step.

Next

Moves you to the next step.

Finish

Disabled in this step.
