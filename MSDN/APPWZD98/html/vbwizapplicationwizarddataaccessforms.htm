Application Wizard—Data Access Forms

Creates child forms for your application and adds either a Data menu with commands to your application's menus or additional commands if you already added a Data menu.

Wizard Options

Data Forms List

Lists the data forms you have selected for your application. Each active data form has a check box preceding the table name. You can clear the check box to keep the table but not use it in your application.

Create New Form

Opens the Data Form Wizard where you can design Visual Basic forms that contain individual bound controls and procedures to manage information derived from database tables and queries.

Help

Displays the help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Application Wizard.

Back

Moves you to the previous step.

Next

Moves you to the next step. You cannot move to the next step if you choose to create forms from your database and do not select a database format and enter a database name.

Finish

Accepts any selections you have entered and the defaults for the remaining steps, then creates the application.
