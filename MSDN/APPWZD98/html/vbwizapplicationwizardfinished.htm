Application Wizard—Finished!

Indicates that all of the necessary information needed to build your application has been collected and asks if you want to see a report with additional actions you can take. Also, allows you to save your settings as a profile that you can use for other applications.

Note   Your application will be saved in memory but not on your disk unless you use the Save Project command on the File menu after the application is created.

Wizard Options

Save Profile

Allows you to select an existing profile or type a name for your profile and to save it. The default is (None). If you chose (None), your profile will not be saved.

Displays the Save Profile dialog box where you can name and select a location in which to save your settings.

View Report

Displays a summary report in your default text editor.

Help

Displays the help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Application Wizard.

Back

Moves you to the previous step.

Next

Disabled in this step.

Finish

Creates an application based on the settings you selected and displays a confirmation dialog. In the confirmation dialog box, you can choose to not have the dialog box appear when you run the Application Wizard again.
