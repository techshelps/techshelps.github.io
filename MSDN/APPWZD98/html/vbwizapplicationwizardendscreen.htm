Application Wizard Summary Report

Creates a report for the application you have created that:

  • Suggests you search the ToDo comments in the code for additional steps you can take to add more functionality to your application.

  • Lists other wizards you can use to refine the functionality of your application.

  • Tells you how to save the report and how to run your application.

Wizard Options

Cancel

Closes the Summary Report dialog box and discards the information.

Save

Displays the Save Report dialog box where you can give a name to your report and indicate where you want it saved.
