Sample Property Sheet Extension

This sample adds to the messaging client's standard message property sheets, for messages belonging to the IPM.Document class. The new property sheet displays the document's summary properties which include information such as author, title and date. .

Quick Info

Executable: PRSHEXT.DLL
Directory for source code: PROPSH.EXT
Written in: C++
Supports: Win32 platforms only

Remarks

This sample implements the IExchExt : IUnknown and IExchExtPropertySheets : IUnknown interfaces.
