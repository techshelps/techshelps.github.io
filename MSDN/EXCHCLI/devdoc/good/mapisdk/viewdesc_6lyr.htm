Personal Views

Personal views are defined by the user, and like common views, are global in usage. That is, they are available no matter what folder the user is looking in. Personal views are located in the bottom bin of the View.Personal Views cascade menu. Personal views are stored in a private message store and are only available to the user who created them. Personal view descriptors are stored in the default message store.

    To locate personal view messages using MAPI
  1. Determine the default store.
  2. Read the PR_VALID_FOLDER_MASK property on the store object. This property indicates the validity of the entry identifiers of special folders in the store. If the FOLDER_VIEWS_ENTRYID flag is set, the personal views are located in this store.
  3. The user's personal views will be stored in the Views folder directly under the root of the message store.
  4. Read the messages from the folder specified above. Views all have the message class IPM.Microsoft.FolderDesign.NamedView.
