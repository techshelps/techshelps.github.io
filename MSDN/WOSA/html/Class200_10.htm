2.7. Text Terminal Unit

Class Name TTU

Class Identifier WFS_SERVICE_CLASS_TTU = 7

The WOSA/XFS Text Terminal service defines the functions provided by a generic Text Terminal Unit (TTU).

A Text Terminal Unit is a text i/o device, which applies both to ATM operator panels and to displays incorporated in devices such as PIN pads and printers. This service allows for the following categories of functions:

  • Forms oriented input and output

  • Direct display output

  • Keyboard input

  • LED settings and control
