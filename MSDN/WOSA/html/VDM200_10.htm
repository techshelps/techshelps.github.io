4.2 WFS_CMD_VDM_ENTER_MODE_ACK

Description This command is issued by a registered application as an acknowledgement to the WFS_SRVE_VDM_ENTER_MODE_REQ Event message and it indicates that the application is ready for the system to enter Vendor Dependent Mode. Completion of this command is immediate.

Input Param None.

Output Param None.

Error Codes There are no additional error codes returned by this command.

Events There are no additional events generated by this command.

Comments None.
