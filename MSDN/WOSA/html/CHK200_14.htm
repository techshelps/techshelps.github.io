6.1 WFS_EXEE_CHK_NOMEDIA

Description

This event specifies that the physical check must be inserted into the device in order for the execute command to proceed.

Event Param
LPSTR     lpszUserPrompt;

lpszUserPrompt

Points to a null-terminated string which identifies the prompt string which is configured for the form (the USERPROMPT attribute of the XFSFORM section).

Comments

The application may use the lpszUserPrompt in any manner it sees fit. For example, it might display the string to the operator, along with a message that the check should be inserted.
