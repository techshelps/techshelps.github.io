2. Service Classes

The following classes of services have been defined for the second version of the WOSA Extensions for Financial Services to include the data and methods needed for the support of self-service, unattended, operations:

  • Printers

  • Identification Card Units

  • Cash Dispensers

  • Personal Identification Number Keypads (PIN pads)

  • Depository Units

  • Text Terminal Units

  • Sensors and Indicators Units

  • Vendor Dependent Mode

  • Cameras

The following sections detail for each of the service classes defined for this version of the WOSA Extensions for Financial Services:

  • the standard values to be used as class attribute in the configuration information;

  • the unique number assigned to each service class;

  • the types of devices defined and supported by the service class specifications.
