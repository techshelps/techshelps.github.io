4.5 WFS_CMD_IDC_RESET_COUNT

Description This function resets the present value for number of cards retained to zero. The function is possible for motor-driven card readers only.

The number of cards retained is controlled by the service and can be requested before resetting via the WFS_INF_IDC_STATUS.

Input Param None.

Output Param None.

Error Codes There are no additional error codes generated by this command.

Events There are no additional events generated by this command.

Comments This is a fundamental capability of an ID card unit; thus if this request is received by a device with no retain capability, the WFS_ERR_UNSUPP_COMMAND error is returned.
