Property Page Wizard — Introduction

Helps you create property pages for an existing user control.

Options

Skip this screen in the future.

When selected, the Property Page Wizard will not show the Introduction step the next time you start it.

If you want to see this step in the future, click the Back button on the first step of the wizard.

Help

Opens the Help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Property Page Wizard.

Back

The Back button is unavailable on the Introduction step.

Next

Moves you to the next step.

Finish

Accepts the defaults and builds the appropriate property page. The Finish button is unavailable when you haven't made enough choices for the Property Page Wizard to create the appropriate property page.
