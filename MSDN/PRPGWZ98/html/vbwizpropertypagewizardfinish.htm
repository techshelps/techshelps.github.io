Property Page Wizard — Finished!

Creates your property pages.

Options

View Summary Report

Allows you to view a summary report which tells you what to do after the Property Page Wizard is finished. You can save this report to review at a later time.

Yes — Creates and opens a summary report.

No — Does not create a summary report.

Help

Opens the Help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Property Page Wizard.

Back

Moves you to the previous step where you can check or change your options.

Next

Unavailable because there are no additional steps for the Property Page Wizard.

Finish

Creates a property page based on the settings you selected.
