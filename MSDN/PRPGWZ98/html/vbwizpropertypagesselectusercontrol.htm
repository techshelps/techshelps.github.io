Property Page Wizard — Select a User Control

Allows you to select the user control for which you want to create the property pages.

You can create property pages for only one control at a time. If you want to create property pages for more than one user control:

  1. Open the Property Page Wizard and create the property pages.

  2. Then re-open the Property Page Wizard for each user control and add the appropriate property pages.

Options

User Controls List

Lists the user controls in your project group with the project name followed by a colon and the name of the control.

Help

Opens the Help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Property Page Wizard.

Back

Moves you to the previous step where you can check or change your options.

Next

Moves you to the next step.

Finish

Accepts the defaults and builds the appropriate property page. The Finish button is unavailable when you haven’t made enough choices for the Property Page Wizard to create the appropriate property page.
