Property Page Wizard — Select the Property Pages

Helps you add and choose the property pages you want to use with your user control.

Important   If you use the ActiveX Control Interface Wizard to remove a property from a control, the control's property page is not updated to reflect the change. You must remove all references to the deleted property from the control's property page or the next time you run the Property Page Wizard, you will get an error.

Options

Available Property Pages List

Displays a list of the custom property pages in your project and the default property pages.

Default property pages are automatically selected if your user control exposes properties of type OLE_COLOR, Font, or Picture.

Changes the order of the appearance of the property pages. Each time you click the button, the selected property page in the Available Property Pages list moves up one position.

Changes the order of the appearance of the property pages. Each time you click the button, the selected property page in the Available Property Pages list moves down one position.

Add

Allows you to add to your project and name a property page.

Rename

Renames the currently selected property page.

Help

Opens the Help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Property Page Wizard.

Back

Moves you to the previous step where you can check or change your options.

Next

Moves you to the next step.

Finish

Accepts the defaults and builds the appropriate property page. The Finish button is unavailable when you haven’t made enough choices for the Property Page Wizard to create the appropriate property page.
