Property Page Wizard — Add Properties

Helps you to choose the properties you want on the available property pages.

Options

Available Properties List

Displays a list of the properties available in your user control.

Available Property Pages

Contains Property Pages as tabs where you place the properties you want on each Property Page. There is one tab per property page listed on the previous screen.

  • To make a Property Page active, click the appropriate tab.

  • To place properties on the active Property Page, select the property or properties you want and either click one of the buttons described below or drag the property or properties to the appropriate Property Page.

    Note   You can drag a property to any of the Property Pages tab to place it on that property page.

Moves the selected field in the Available Properties list to the selected tab of the Available Property Pages.

Moves all of the fields in the Available Properties list to the selected tab of the Available Property Pages.

Moves the selected field in the Available Properties list to the selected tab of the Available Property Pages.

Moves all of the fields in the Available Properties list to the selected tab of the Available Property Pages.

Note   OLE_COLOR, Font, or Picture properties cannot be moved. They are automatically inserted into built-in property pages.

Help

Opens the Help topic for this step. You can also press F1 for help.

Cancel

Cancels your previous actions and closes the Property Page Wizard.

Back

Moves you to the previous step where you can check or change your options.

Next

Moves you to the next step.

Finish

Accepts the defaults and builds the appropriate property page.
