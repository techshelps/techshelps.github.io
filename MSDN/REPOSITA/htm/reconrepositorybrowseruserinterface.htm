Repository Browser User Interface

The Repository Browser is designed to provide a look-and-feel that is similar to the Windows Explorer. This familiar interface allows a user to easily browse the contents of a Repository.

The following tables describe the purpose of some of the Browser menus:

File Menu Description
Open Repository Opens a Repository database file.

Edit Menu Description
Copy Copies property values from the Browser into the Windows clipboard.
Paste Pastes property values from the clipboard into the Browser.
Delete Deletes the selected object.
Relationship Displays the Relationship dialog box where you can view and edit the relationship between the selected target object and its source object.
Delete Relationship Deletes the relationship between the selected target object and its source object.

View Menu Description
Toolbar Toggles the display of the toolbar.
Status Bar Toggles the display of the status bar.
Refresh Updates the Browser view.
Properties Displays a dialog box to view object properties.
Filters Displays the Repository Filters dialog box to enable you to change the Repository view.
