Deleting Objects

See Also

The Repository Browser enables you to delete an object instance from the Repository. Deleting an object instance removes it from both the Browser interface and the Repository.

To delete an object instance

  1. Click the object to select it.

  2. Click Delete, on the Edit menu
    –or-
    The Delete toolbar button
    -or-
    Delete on the context menu.

    A confirmation dialog box appears as shown in the following figure.

  3. To confirm the deletion click OK.
    -or-
    To cancel the process click Cancel.
