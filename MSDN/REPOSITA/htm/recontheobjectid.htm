The Object ID

See Also

An object ID is a twenty-byte number maintained in the Repository as binary data. This number is converted to ASCII text and displayed in the right pane of the Repository Browser. For example, an object ID would be displayed as:

{{90591B0C-24F7-11CF-920A-00AA00A1EB95},0000002C}

The object ID is also displayed in the Properties dialog box as a read-only value as shown in the following figure. You can copy the object ID into the Windows clipboard, either from the Browser, or from the Properties dialog box.
