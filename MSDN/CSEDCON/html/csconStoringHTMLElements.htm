Storing HTML Elements

You can use the Toolbox to store any selection from an HTML document — text, images, objects, and so on — to use again. Selections that you store on the Toolbox remain there until you remove them, so you can use them in the same document or in a different document.

To store elements on the Toolbox

  1. In the Toolbox, select the tab (or create a new one) where you want to store the elements.

  2. In the HTML editor, select the element or elements you want to store.

  3. Drag the selection to the Toolbox.

    When you drop the element, the Toolbox creates a new entry called HTML Fragment. To rename the entry, right-click it in the Toolbox and choose Rename.

When you store an element on the Toolbox, the editor copies the exact HTML text from your document. When you later drag the element from the Toolbox to another location or document, you might have to adjust the text that is created.
