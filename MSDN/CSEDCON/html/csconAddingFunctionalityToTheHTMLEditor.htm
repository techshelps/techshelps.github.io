Adding Functionality to the HTML Editor

The HTML editor can be extended through the use of add-on Wizards. These wizards, which are available from Microsoft and from third parties, provide an easy-to-use interface for tasks not already built into the HTML editor. For example, one add-on Wizard might guide you through the steps for adding multimedia effects to your page. Another might allow you to check the spelling of your document.

The exact add-on wizards you can use depends on what you have installed. For information about installing an HTML Wizard, refer to the documentation that accompanies the wizard.

To run an add-on HTML editor wizard

  1. Select the text you want to work with.

  2. From the HTML menu, choose HTML Wizards.

  3. In the Choose Builders dialog box, select the wizard you want to run, and then choose OK.

For information about how to create HTML Wizards, see the Visual Studio Web site at Microsoft Visual Studio Web site.
