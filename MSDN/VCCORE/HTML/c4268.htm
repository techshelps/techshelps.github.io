Compiler Warning (level 4) C4268

'identifier' : 'const' static/global data initialized with compiler generated default constructor fills the object with zeros

A const global or static instance of a non-trivial class is initialized with a compiler-generated default constructor. For example:

class X {
public:
   int m_data;
};

const X x1;   // Generates warning C4268

As this instance of the class is const, the value of m_data can never be changed.
