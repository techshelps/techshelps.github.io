Other Important Tips

The ODBC Data Source Must Be a System DSN

The ODBC data source that ISAPI uses must be a system DSN, although Visual C++ Enterprise Edition will work with either a user or a system DSN.

Additional User Rights are Required to Successfully Host a Debugging Session

To successfully host a debugging session that requires both Visual C++ and IIS, the user id that you use to log onto Windows NT must have the following two user rights:

  • Act as part of the operating system.

  • Generate security audits.

To set the user rights

  1. Run the User Manager from the Administrative Tools program group.

  2. From the Policies menu, select User Rights.

  3. Select the Show Advanced User Rights checkbox.

  4. Add the user id to the Rights: "Act as part of the operating system" and "Generate security audits."
