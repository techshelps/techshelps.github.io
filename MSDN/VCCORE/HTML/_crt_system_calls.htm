System Calls

The following functions are Windows NT and Windows 95 operating-system calls:

System Call Functions

Function Use
_findclose Release resources from previous find operations
_findfirst, _findfirsti64, _wfindfirst, _wfindfirsti64 Find file with specified attributes
_findnext, _findnexti64, _wfindnext, _wfindnexti64 Find next file with specified attributes
