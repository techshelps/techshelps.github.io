Dialog Data Exchange and Validation

Home |  Overview |  How Do I |  Sample |  Tutorial

Dialog data exchange (DDX) is an easy way to initialize the controls in your dialog box and to gather data input by the user. Dialog data validation (DDV) is an easy way to validate data entry in a dialog box. To take advantage of DDX and DDV in your dialog boxes, use ClassWizard to create the data members and set their data types and specify validation rules. For additional information about DDX/DDV and for examples, see Working with Dialog Box Data and Adding a Dialog Box.
