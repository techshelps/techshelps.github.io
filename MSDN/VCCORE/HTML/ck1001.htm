CVPACK Fatal Error CK1001

out of memory

CVPACK ran out of memory.

One of the following may be a solution:

  • Free some space on disk.

  • Increase the size of the Windows NT paging file or the Windows swap file.

  • Recompile one or more of the object files without debugging information. If the file was compiled using /Zi or /Z7, either use /Zd or do not use a debugging option.
