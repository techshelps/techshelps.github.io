Build Settings Generated When You Create a Visual C++ Program

Home |  Overview |  How Do I |  Details |  Sample

Build settings are automatically generated for your Visual C++ program, whenever you create one either by using a wizard (such as the MFC AppWizard) or one of the icons (such as the Application icon) available on the Projects tab of the New dialog.

To view, change, or reset the build settings for your program: from the Project menu, click Settings and then click the C/C++ tab.

Which type of program do you want to know the build settings for?

  • An MFC EXE program

  • An MFC DLL

  • A Win32 program

  • A Win32 DLL

  • A Static library

  • A Console program
