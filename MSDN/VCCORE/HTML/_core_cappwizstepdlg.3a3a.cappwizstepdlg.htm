Feature Only in Professional and Enterprise Editions   Creating a Custom AppWizard is supported only in Visual C++ Professional and Enterprise Editions. For more information, see Visual C++ Editions.

CAppWizStepDlg( UINT nIDTemplate );

Parameters

nIDTemplate

Contains the ID number of a dialog box template resource.

Remarks

Accepts a template ID number, usually with an IDD_ prefix (for example, IDD_CUSTOM1) and constructs a resource-based dialog box: a step, in the case of a custom AppWizard.
