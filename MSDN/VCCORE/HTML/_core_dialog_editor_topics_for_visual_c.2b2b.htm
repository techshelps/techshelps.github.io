Dialog Editor Topics for Visual C++

Use the dialog editor to edit the dialog template resource that Windows uses to display a dialog box — or a form view, record view, or dialog bar.

Visual C++ allows you to import Visual Basic forms, converting them to form views or dialog boxes. There are also a few tips for using the dialog editor specific to Visual C++.
