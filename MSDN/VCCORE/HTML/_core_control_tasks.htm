Windows common controls simplify creating complex dialog boxes. The list of available controls expanded with Windows 95.

What do you want to do?

  • Use Windows common controls

  • Add controls to a dialog box with the dialog editor

  • Add controls with code

  • Derive a control from a standard Windows control

  • Access dialog-box controls with type safety

  • Use a common control as a child window

  • Receive notification messages from common controls

For information about Windows common controls in the Win32 Software Development Kit (SDK), see Common Controls in the Win32 Programmer's Reference.
