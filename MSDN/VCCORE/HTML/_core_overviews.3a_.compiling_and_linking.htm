Overviews: Compiling and Linking

Home |  How Do I |  Details

The first step in the process of building a program is creating source code files with the code statements in header and/or source files. When you invoke the compiler, the preprocessor runs first to create the compiler input. The compiler creates an object file that contains machine code, linker directives, sections, external references, and function and data names generated from the source files. Finally, the linker combines all of the object code from statically-linked libraries and other object files, resolves the named resources, and creates an executable file. Typically, a makefile coordinates the combination of these elements and tools in creating the executable file.
