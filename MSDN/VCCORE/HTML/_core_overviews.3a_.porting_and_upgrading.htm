Home |  How Do I |  FAQ |  Details

There are many ways to adapt your existing applications.

  • Make them more Internet friendly
    • By upgrading to Active document servers

    • By upgrading to ActiveX controls with asynchronous properties
  • Make their user-interfaces up-to-date
    • By using new ActiveX control features such as transparent and windowless controls

    • By adding Windows 95 functionality
  • Make them more adaptable to many kinds of platforms
    • By using MFC libraries
  • Port from different platforms
    • Differences Between 16-Bit Windows and 32-Bit Windows: Overview

    • Porting Tools, including PORTTOOL
  • Port to other C++ compilers
    • By using the Standard C++ library
  • Port from previous 32-bit versions and 16-bit versions of Visual C++
