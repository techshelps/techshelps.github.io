Symbol Codes in the Browse Window

Home |  Overview/Tasks

A browse window uses symbol codes to describe the displayed query types. For example, f V CAboutDlg::GetMessageMap(void) indicates a virtual (V) function (f).

The following table shows the codes and their meanings.

Browse Window Symbol Codes

Code Meaning
c Class
f Function
d Data
m Macro
t Type (other than class)
V Virtual function
S Static function or data member
