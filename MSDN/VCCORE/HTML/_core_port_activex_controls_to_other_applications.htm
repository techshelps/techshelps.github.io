Port ActiveX Controls to Other Applications

Home |  Overview |  How Do I |  FAQ |  Details

You can use ActiveX controls developed in Visual C++ in other applications, as long as they support ActiveX control containment. A number of applications, including the following from Microsoft, support ActiveX control containment:

  • Internet Explorer 3.x

  • Internet Explorer 4.x

  • Visual Basic 4.x

  • Visual Basic 5.x

  • Visual Basic 6.0

  • Visual C++ 4.x

  • Visual C++ 5.0

  • Visual C++ 6.0

  • Access 95

  • Excel '97

  • Word '97

  • Access '97

  • Front Page '97

  • Power Point '97

  • Visual InterDev '97

The following are non-Microsoft applications that support ActiveX control containment:

  • PowerBuilder

  • Delphi

  • C++ Builder

  • NCompass Plug-in for Netscape Navigator

Note   Visual FoxPro did not fully meet the containment guidelines in version 3.0 but will in the next version.
