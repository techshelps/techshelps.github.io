/NOLOGO   (Suppress Startup Banner)

Home |  Overview |  How Do I |  Linker Options

This option prevents display of the copyright message and version number.

The Suppress Startup Banner option also suppresses echoing of command files. For details, see LINK Command Files.

By default, this information is sent by the linker to the Output window. On the command line, it is sent to standard output and can be redirected to a file.

To find this option in the development environment, click Settings on the Project menu. Then click the Link tab, and click Customize in the Category box.
