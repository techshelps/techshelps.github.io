For a tour of the MFC development process with real applications, see the tutorials:

  • The Scribble Tutorial shows you how to use the basic MFC framework.

  • The COM component, Container, and Automation tutorials teach OLE and Automation programming techniques.

  • The ActiveX control tutorial explains ActiveX control development.

  • The ODBC database tutorial explains ODBC database programming techniques.

  • The DAO database tutorial explains DAO database programming.

While it is possible to do these tasks by hand or using other tools, your savings in time, energy, and errors suggest that using the tools and framework is greatly to your benefit.

In addition to the tutorials listed above, see Windows 95 Functionality for a guide to Windows 95 logo compliance.
