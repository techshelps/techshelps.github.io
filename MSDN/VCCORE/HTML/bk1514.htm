BSCMAKE Error BK1514

all .SBR files truncated, none found in filename

None of the .SBR files specified for an update was a part of the original browse information (.BSC) file. This message is always preceded by warning BK4502, which gives the name of the .SBR file that caused the error.

One of the following may be a cause:

  • The wrong filename was specified.

  • The file was corrupted, and a full build was required.
