Visual C++  provides ODBC drivers for the following databases:

  • SQL Server

  • Microsoft Access

  • Microsoft FoxPro

  • Microsoft Excel

  • dBASE

  • Paradox

  • Microsoft Oracle ODBC

  • Text files
