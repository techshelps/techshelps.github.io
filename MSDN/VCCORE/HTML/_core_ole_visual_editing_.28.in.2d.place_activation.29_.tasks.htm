OLE Visual Editing (In-Place Activation) Tasks

Visual editing, also called in-place activation, lets users edit linked or embedded items, such as Excel spreadsheets, right in a document, such as a Word for Windows document.

What do you want to do?

  • Create an OLE visual editing application (see Activation)

  • Manage menus in an OLE visual editing container application

  • Manage menus in an OLE visual editing server application

  • Merge menus for OLE visual editing and in-place activation
