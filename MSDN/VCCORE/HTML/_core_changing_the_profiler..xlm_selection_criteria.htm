Changing the PROFILER.XLM Selection Criteria

Home |  Overview |  Details

Feature Only in Professional and Enterprise Editions   Profiling is supported only in Visual C++ Professional and Enterprise Editions. For more information, see Visual C++ Editions.

The standard PROFILER.XLM macro displays hit counts greater than zero (for CTRL+C) and times greater than .01 millisecond (for CTRL+T). If you need to narrow these selections without analyzing the macro, edit the formulas in cells C10 and D10.
