Data-Source Controls

A data-source control lets you define a query with which to access data. You then connect data-bound controls to a data source.

Control Support Files
Microsoft ADO Data Control Msadodc.ocx; help file is Adodc98.chm.
Microsoft RemoteData Control Msrdc20.ocx; help file is RDO98.chm.
