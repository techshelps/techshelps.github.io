Changes from MFC Versions 2.0 and 2.5

You may be upgrading to version 3.0 of the Microsoft Foundation Class Library from one of the previous 16-bit versions: either MFC version 2.0 or 2.5, which are part of Visual C++ versions 1.0 and 1.5, respectively.
