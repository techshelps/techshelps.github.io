Displaying a Graph of Calling Functions

Home |  Overview/Tasks

A callers graph displays all the functions that call a selected function.

To display the graph of all functions that call a selected function

  1. Select the function name in a source file or the standard toolbar Find box.

  2. On the Tools menu, click Source Browser.

  3. In the Select Query list box, click Callers Graph.

  4. Click OK.

  5. If the function is an overloaded function or a member function of more than one class, the Resolve Ambiguity dialog box appears. Select the function that you want from the list.

    The Callers Graph window appears, with the selected function displayed at the top of the left pane. The following figure shows the Callers Graph window for a sample function.
