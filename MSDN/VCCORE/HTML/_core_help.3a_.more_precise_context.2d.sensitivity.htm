Help: More Precise Context-Sensitivity

Home |  Overview |  How Do I |  Sample |  Tutorial

F1 help and SHIFT+F1 help are activated based on help context IDs. The standard help implementation in the framework can obtain a help context from a window, dialog box, message box, menu item, or toolbar button. If you need more precise control over this mechanism, you can override parts of the mechanism.

For additional information, see Technical Note 28.
