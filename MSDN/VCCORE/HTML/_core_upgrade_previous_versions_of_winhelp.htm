Upgrade Previous Versions of WinHelp

To upgrade previous versions of WinHelp

  1. Start the Help Workshop by running HCW.EXE.

    This file is installed in \Program Files\Microsoft Visual Studio\VC98\bin in a default product installation.

  2. In the Help Workshop, open your previous .HPJ file.

    Help Workshop automatically adds entries to the file.

  3. Use Help Workshop to make any modifications to your previous Help file and then save the file.

    Help Workshop saves your file in the new format.

For more information, see online help in the file HCW.HLP, installed by default in the \Program Files\Microsoft Visual Studio\VC98\bin directory. You can open the file by double-clicking it, or view it by choosing Help from within Help Workshop.
