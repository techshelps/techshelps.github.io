MFC helps you with two kinds of menu manipulation:

  • Adding, deleting, and modifying menus on the fly.

  • Updating the enabled/disabled state of menu items (and toolbar buttons with the same command ID) as program conditions change.

What do you want to know more about?

  • Updating menu items as program conditions change
