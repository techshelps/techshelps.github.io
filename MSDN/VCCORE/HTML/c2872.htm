Compiler Error C2872

'symbol' : ambiguous symbol

The compiler could not determine which symbol you are referring to.

The following code shows an example of this error.

namespace A {
   int i;
}

using namespace A;

int i;

void z() {
   ::i++;   // ok
   A::i++;   // ok
   i++;      // error C2872, ::i or A::i?
}
