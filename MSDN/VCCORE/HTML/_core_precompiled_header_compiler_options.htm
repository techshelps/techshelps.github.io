Precompiled Header Compiler Options

Home

The compiler options described in the following sections control the creation and use of precompiled headers. Using the hdrstop pragma gives you extra control over the behavior of these options.

  • /YX (Automate Precompiled Header) option

  • /Yc (Create Precompiled Header) option

  • /Yu (Use Precompiled Header) option

  • /Yd (Duplicate Debugging Information in All Object Files) option

  • /Fp (Specify Precompiled Header Filename) option
