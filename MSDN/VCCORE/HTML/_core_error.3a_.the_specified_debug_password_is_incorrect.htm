Error: The Specified Debug Password Is Incorrect

Full Error Text: The specified debug password is incorrect.

Both the debugger host and remote target machines must use the same Debug monitor password. The password for the remote target machine is entered on the local machine using the Debugger Remote Connection of the Build menu. Click Settings and check to make sure that the Debug monitor password is valid. At each machine that allows remote debugging, you must specify the same Debug monitor password.

Tip   You can also specify the Debug monitor password by running MSVCMON.EXE.
