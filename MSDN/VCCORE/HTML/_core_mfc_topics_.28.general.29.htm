This family of articles includes technical details about the Microsoft Foundation Class Library (MFC) and provides an overview of the MFC framework and its key components and subsystems. MFC and the Active Template Library (ATL) are part of Microsoft Foundation Classes and Templates.

Technical details include:

  • How to use MFC.BSC, the MFC browse information file

  • MFC library versions

  • Managing the state data of MFC modules

  • Idle loop processing in MFC
