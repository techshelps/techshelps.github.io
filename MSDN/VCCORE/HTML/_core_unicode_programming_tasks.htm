Unicode and multibyte character sets (MBCS) are two implementations of "wide characters," that is, characters two bytes wide rather than one byte wide.

For related information on both, see International Programming Topics. The tasks below are for Unicode, which is available on Windows NT but not on Windows 95.

What do you want to do?

  • Install Unicode support via MFC

  • Enable Unicode in my program

  • Enable both Unicode and MBCS in my program

  • Use Unicode to create an internationalized program

  • Learn the benefits of Unicode, including how using Unicode makes my program more efficient on Windows NT

  • Use wmain so I can pass wide-character arguments to my program

  • See a summary of Unicode programming

  • Learn about generic-text mappings for byte-width portability
