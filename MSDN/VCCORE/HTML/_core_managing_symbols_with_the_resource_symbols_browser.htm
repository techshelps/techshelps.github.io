Managing Symbols with the Resource Symbols Browser

Home |  Overview |  How Do I

As your application grows in size and sophistication, so does the number of resources and symbols Tracking large numbers of symbols scattered throughout several files can be difficult. The Resource Symbols browser simplifies symbol management by offering a central tool through which you can:

  • Quickly browse existing symbol definitions to see the value of each symbol, a list of symbols being used, and the resources assigned to each symbol.

  • Create new symbols.

  • Change the name and value of a symbol that is not in use.

  • Delete a symbol if it is not being used.

  • Move quickly to the appropriate resource editor where the symbol is being used.

What do you want to do?

  • Open the Resource Symbols browser

  • Create new symbols

  • Change unassigned symbols

  • Open the Resource editor for a given symbol

To open the Resource Symbols browser

  • From the View menu, choose Resource Symbols.
