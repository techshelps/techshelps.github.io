Compiler Warning (level 1) C4620

no postfix form of 'operator ++' found for type 'type', using prefix form

There was no postfix increment operator defined for the given type. The compiler used the overloaded prefix operator.

This warning can be avoided by defining a postfix ++ operator. This is done by creating a two-argument version of the ++ operator as shown below:

class prepost
{
public:
   prepost operator++ ()
   {
 // inline prefix routine
   };
   prepost operator++ ( int )
   {
 // inline postfix routine
   };
}
