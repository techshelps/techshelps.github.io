CObject, The Root Base Class in MFC: Overview

Home |  How Do I |  FAQ |  Details

Class CObject is the root base class in the MFC class hierarchy. This class provides a number of services, with minimal overhead, to any class derived from it. You can:

  • Dump the object's data for diagnostic purposes

  • Validate the object's internal state

  • Obtain information about the class at run time

  • Have the class serialize itself to persistent storage

  • Make the class dynamically creatable
