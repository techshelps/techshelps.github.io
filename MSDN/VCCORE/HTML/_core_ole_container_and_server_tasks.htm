OLE Container and Server Tasks

Home |  Overview |  Details |  Sample |  Tutorial

OLE containers contain "items" linked or embedded from other applications. OLE servers supply items that can be linked or embedded in other applications.

What do you want to do?

  • Implement a container application

  • Implement a COM component (server) application

  • Create an COM component (container/server) application

  • Link to embedded objects in a COM component (container/server)

  • Implement COM component (server) documents

  • Implement in-place frame windows

  • Implement COM component (server) items

  • Work with COM container (client) items

  • Handle COM client-item notifications

  • Use the compound files format for your data

  • Cause your COM component (server) to register itself
