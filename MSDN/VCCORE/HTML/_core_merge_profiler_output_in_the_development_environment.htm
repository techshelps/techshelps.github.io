Merge Profiler Output in the Development Environment

Home |  Overview |  Details

Feature Only in Professional and Enterprise Editions   Profiling is supported only in Visual C++ Professional and Enterprise Editions. For more information, see Visual C++ Editions.

After you perform the first profiler run in an application, you can merge output. After you have done one profiler run, a *.pbi file will exist in your project subdirectory.

Click the Merge button in the Profile dialog box.
