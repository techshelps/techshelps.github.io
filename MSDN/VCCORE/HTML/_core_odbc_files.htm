Use the ODBC Installer DLL to install the ODBC files on your user's system from your application's setup program. The redistributable files include the ODBC Driver Manager and the various ODBC drivers included in \OS\System. For information, see "Installing ODBC Software" and "Configuring Data Sources," in the ODBC Programmer's Reference.
