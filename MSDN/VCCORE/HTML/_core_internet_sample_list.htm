Visual C++ includes these Internet client sample programs:

  • FTPTREE   A dialog-based application that displays the contents of an FTP site in a tree control.

  • TEAR   Shows how to write an MFC console application that uses WININET.DLL to communicate with the Internet. This sample shows how to form an HTTP request using CHttpFile against CHttpConnection and CInternetSession objects.

Visual C++ includes these Internet server sample programs. The first three samples use the MFC ISAPI classes:

  • MFCUCASE   Demonstrates an ISAPI filter that uses the filter functions OnSendRawData and  OnUrlMap to convert strings to uppercase.

  • WWWQUOTE   Demonstrates an ISAPI server extension DLL that uses an SQL Server database. The sample shows proper use of the parse map macros, the CHttpServer class, and thread-safe programming techniques.

  • COUNTER   Demonstrates using an ISAPI server extension DLL to send image data (rather than HTML data) back to a Web browser.

  • HTTPSVR   Demonstrates the use of MFC and its Windows Sockets (WinSock) classes in implementing a simple World Wide Web HTTP server.
