SQL Debugging Components

Feature Only in Enterprise Edition   This feature is supported only in Visual C++ Enterprise Edition. For more information, see Visual C++ Editions.

The Visual C++ 6.0 Enterprise Edition setup program provides options for installing client-side and server-side SQL debugging components on x86 platforms. The SQL Server machine must have Microsoft SQL Server 6.5 and Service Pack 3 installed. The following table shows the components needed for SQL server debugging.
