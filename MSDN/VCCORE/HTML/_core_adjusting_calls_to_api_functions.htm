Adjusting Calls to API Functions

Most of your source code is not affected by differences between the APIs of Windows 3.x and Win32. The underlying definitions in WINDOWS.H automatically adjust data to the correct size. But you may need to revise code if you call API functions in any of the following categories:

  • Graphics functions

  • Functions accessing “extra” window data

  • MS-DOS system calls

  • File Operations

  • Far-pointer functions

  • Functions getting list and combo box contents
