Profiling from the Development Environment

Home |  How Do I |  Details

Feature Only in Professional and Enterprise Editions   Profiling is supported only in Visual C++ Professional and Enterprise Editions. For more information, see Visual C++ Editions.

The development environment provides the Profile dialog box (available from the Build menu) for access to profiler functionality. There are options to do line and function profiling, and to do multiple profiling runs.

If you need to use nondefault PROFILE, PREP Phase II, or PLIST commands, you can make a batch file and run that batch file either directly from the command line or via the Custom option in the Profile dialog box.

See Run the Profiler from the Development Environment.
