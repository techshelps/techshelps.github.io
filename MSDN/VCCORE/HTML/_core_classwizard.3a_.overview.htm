ClassWizard: Overview

Home |  How Do I |  Details

ClassWizard is a Visual C++ tool that helps you create and manage the classes in your program. ClassWizard works both with the classes you create initially with AppWizard and the classes you create later with ClassWizard. ClassWizard also lets you browse and edit the classes in your program. You can create classes, map messages, override virtual functions, define Automation methods and properties, navigate through your files, and more.
