Compiler Error C2776

only one 'get' method can be specified per property

You can only specify one get function in the property extended attribute. This error occurs when multiple get functions are specified. For example:

struct A {
   __declspec(property(get=GetProp,get=GetPropToo))
      int prop;      // error C2776
   int GetProp(void);
   int GetPropToo(void);
};
