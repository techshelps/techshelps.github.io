Compiler Error C2638

'identifier' : __based modifier illegal on pointer to member

The __based modifier was specifed for a pointer to a member.

The __based modifier can not be used for pointers to members.

The following is an example of this error:

void *a;

class C
{
public:
   int i;
   int j;
   int func();
};
int __based (a) C::* cpi = &C::i;  // error
int (__based (a) C::* cpf)() = &C::func; // error
