File Handling: Overview

Home |  How Do I |  Details

Visual C++ supplies a number of ways to work with files, including:

  • Run-time library functions for file handling

  • Win32 API functions for file handling

  • MFC classes: CFile and its derived classes

MFC's class CFile encapsulates access to files in a C++ class. The MFC library also supplies derived classes like CStdioFile and CMemFile for working with text files, in-memory files, OLE streams, and other file-like objects, such as Windows sockets.
