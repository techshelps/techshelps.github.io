Menu Editor Topics for Visual C++

Home |  Overview |  How Do I

For Visual C++ projects, use the Menu editor to edit your program’s menus, including pop-up menus.

For MFC programs, you can use ClassWizard to hook menu items to code. Double-click the menu item in the menu editor to open ClassWizard.
