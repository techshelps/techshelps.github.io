Filtering Browse Information for Files

Home |  Overview/Tasks

When you do a file outline query, five filter buttons appear at the top of the browse window. Their settings determine which symbols are displayed in the left pane of the browse window. When a button is selected (pushed in), the corresponding symbols are displayed.

To filter the symbol display

  • Select the buttons corresponding to the symbols you want to see, as shown in the following table.
    Button Keyboard Displays
    Classes ALT+C Classes
    Functions ALT+U Functions
    Data ALT+A Data symbols
    Macros ALT+M Macros
    Types ALT+Y Types (other than classes)
