To test your MFC DLL, you must have a program that uses the exported classes or calls the exported functions. Click here for suggestions on managing the project workspace for the calling application and the DLL.

To debug a DLL, you must select the program that calls your DLL by clicking the Settings command on the Project menu. In the Project Settings dialog box, click the Debug tab. In the Executable for debug session box, specify which application to run while debugging.
