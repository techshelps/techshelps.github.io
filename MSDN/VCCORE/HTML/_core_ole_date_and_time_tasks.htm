OLE Date and Time Tasks

Home |  Overview |  Details |  Sample |  Tutorial

MFC supplies several classes derived from the VARIANT data type for manipulating date and time data in Automation.

What do you want to do?

  • Get the current time with COleDateTime

  • Calculate elapsed time with COleDateTimeSpan

  • Format a string representation of a time or elapsed time (OLE)
