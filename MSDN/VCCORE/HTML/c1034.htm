Fatal Error C1034

'file': no include path set

The INCLUDE environment variable was not set.

Set this variable using the Options command on the Tools menu. In the Options dialog box, select the Directiories tab. In the Show Directories For box, select Include Files. Then, in the Directories box, specify the path for the include files.

If you are using the command-line compiler, run the VCVARS32.BAT batch file to set the INCLUDE environment variable.
