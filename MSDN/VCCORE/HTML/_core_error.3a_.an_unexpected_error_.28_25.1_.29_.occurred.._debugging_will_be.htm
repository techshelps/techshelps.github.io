Error: An unexpected error ( %1 ) occurred. Debugging will be turned off for connection %2, %3

Feature Only in Enterprise Edition   This feature is supported only in Visual C++ Enterprise Edition. For more information, see Visual C++ Editions.

Look at the error number and associated message that is supplied by the operating system.

  • %1   A hexadecimal error number that is supplied by the operating system.

  • %2   The SPID.

  • %3   An error message that is supplied by the operating system.
