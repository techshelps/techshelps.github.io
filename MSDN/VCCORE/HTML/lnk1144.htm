Linker Tools Error LNK1144

error with LINK_REPRO var; cannot open "filename"

The linker could not create filename.

One of the following may be a cause:

  • Check that you've set your LINK_REPRO environment variable to an existing directory and not the current directory.

  • Make sure there are no read-only files in the directory.

  • Make sure none of the files in the LINK_REPRO directory are open by another process, because the linker needs to write to them.
