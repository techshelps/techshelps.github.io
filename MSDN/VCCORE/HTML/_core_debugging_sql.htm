Debugging SQL

Feature Only in Enterprise Edition   This feature is supported only in Visual C++ Enterprise Edition. For more information, see Visual C++ Editions.

Visual C++ Enterprise Edition provides an integrated debugging process that can help find and fix errors in SQL Server stored procedures. Debugging SQL source is the process of finding and correcting SQL syntax errors so that your application can run smoothly and work as expected.

Note   You can easily create and modify databases and SQL queries using the Visual Database Tools. For more information, see Visual Database Tools.
