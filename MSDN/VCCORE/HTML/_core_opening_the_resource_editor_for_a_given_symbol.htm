Opening the Resource Editor for a Given Symbol

Home |  Overview |  How Do I

When you are browsing symbols in the Resource Symbols browser, you may want more information on how a particular symbol is used. The View Use command provides a quick way to get this information.

To move to the resource editor where a symbol is being used

  1. In the Name box of the Resource Symbols browser, select the symbol you want.

  2. In the Used By box, select the resource type that interests you.

  3. Click the View Use button.

    The resource appears in the appropriate editor window.
