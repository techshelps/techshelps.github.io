An ActiveX control (formerly known as an OLE control) is a reusable software component that supports a wide variety of OLE functionality and can be customized to fit many software needs. ActiveX controls are designed for use both in ordinary ActiveX control containers and on the Internet, in World Wide Web pages. You can create ActiveX controls either with MFC, described in this article family, or with Active Template Library (ATL).

For some recent information on optimizing your ActiveX controls, see ActiveX Controls: Optimization.
