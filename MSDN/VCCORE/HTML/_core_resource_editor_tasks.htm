Resource Editor Tasks

Home |  Overview |  Details

The integrated development environment provides resource editors for editing the resources in your program’s resource (.RC) file. The following tasks cover using those editors. Visual C++ also supplies sample resources, such as icons, cursors, and toolbar buttons, that you can use in your programs.
