Port to and from Non-Intel Platforms

Home |  Overview |  How Do I |  FAQ |  Details

Visual C++ and its cross-development add-ons allow you to port your applications from Intel-based Visual C/C++ to RISC platforms, or from RISC-based Visual C++ applications to Intel platforms.

  • Port non-Visual C++ applications from different platforms to Visual C++

  • Port Visual C++ applications between Intel-based and RISC platforms (Alpha or PowerPC)

  • Port from UNIX to Win32
