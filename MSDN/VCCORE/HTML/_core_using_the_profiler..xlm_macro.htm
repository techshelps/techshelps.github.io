Using the PROFILER.XLM Macro

Feature Only in Professional and Enterprise Editions   Profiling is supported only in Visual C++ Professional and Enterprise Editions. For more information, see Visual C++ Editions.

To run the PROFILER.XLM macro from within Microsoft Excel

  1. On the File menu, click Open to open PROFILER.XLM.

  2. On the File menu, click Open to open the tab-delimited file that was created by PLIST.

  3. If you have several open worksheets, activate the one containing the profiler data by clicking it with the mouse or by choosing its title from the Window menu.

  4. Run the macro:
    • Press CTRL+C for a chart based on hit counts.

    • Press CTRL+T for a chart based on timing.  You cannot get a timing chart if the file contains only counting or coverage information.

The macro typically takes only a few seconds to run. When it is complete, Microsoft Excel displays a three-dimensional bar chart based on the results in the file (see the following figure). You can change the chart type by using the Gallery menu.
