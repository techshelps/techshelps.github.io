.OBJ Files as Linker Input

Home |  Overview |  FAQ |  Details |  Linker Options

LINK accepts .OBJ files that are either Common Object File Format (COFF) or 32-bit Object Module Format (OMF). Microsoft’s compiler creates COFF .OBJ files. LINK automatically converts 32-bit OMF objects to COFF. See Set Linker Options for related information.
