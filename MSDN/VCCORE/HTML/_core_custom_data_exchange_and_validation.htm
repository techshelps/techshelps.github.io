Custom Data Exchange and Validation

Home |  Overview |  How Do I |  Tutorial

Although you can write a dialog box class that gathers and validates its own dialog box data using custom message handlers, you may find that you have routines for data exchange and validation (containing your own variable types and data formats) that you want to use repeatedly. You can extend the ClassWizard user interface to reuse your own DDX and DDV routines. For more information on this subject, see MFC Technical Note 26.
