Adding Third-Party Components or ActiveX Controls to the Gallery

Home |  Overview |  How Do I

To add a third-party component or ActiveX control to the Gallery, run the setup program supplied with the component or ActiveX control. This may include, for example, downloading an ActiveX control from the Internet. This registers the ActiveX control with your system, adding a shortcut to the control in the default Registered ActiveX Controls folder.

Note   Not all components provide setup programs. Without a valid setup routine, you cannot add a third-party component or control to the Gallery. To register third-party ActiveX controls, run Regsvr32.exe, which by default is located in:
\Program Files\Microsoft Visual Studio\VC98\bin

To unregister a third-party component or control, see Deleting a Gallery File or Folder.
