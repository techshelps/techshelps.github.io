Views Available from AppWizard

Home |  Overview |  How Do I |  Sample |  Tutorial

All view classes listed in the table in Derived View Classes, except CCtrlView and CDaoRecordView, are available in the MFC AppWizard. In step 6 of the AppWizard, select the application's view class, then in the Base class drop-down menu, select a base class for that view.

Three common styles of applications, based on various view classes, are available from AppWizard:

  • Forms-based applications

  • Web browser-style applications

  • Explorer-style application

In addition to these types of views, the MFC AppWizard also provides the option to create an application without document/view architecture. In step 1 of the MFC AppWizard, clear the Document/View architecture support check box. For more information, see Alternatives to the Document/View Architecture.
