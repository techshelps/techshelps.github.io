Working with Classes: Overview

Home |  How Do I |  Details

Once you have created a project workspace, you may want to change some of the functionality, or add to it. ClassView, ClassWizard, and WizardBar are tools designed to make certain routine tasks easy. Such tasks include creating new classes, adding new member functions and variables, defining Windows message handlers, and overriding MFC virtual functions. These tools also provide a convenient way to navigate or browse through the class structure.
