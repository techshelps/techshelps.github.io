The Gallery stores code for later sharing or reuse. Such code can be prepackaged components or ActiveX controls, or user-defined components that you have added to the Gallery from your own projects.

You can enhance your projects by using items from the Gallery. The Gallery comes with default folders containing prepackaged Visual C++ Components and Registered ActiveX controls.

You can also add items to the Gallery, and organize the contents of the Gallery by performing ordinary Windows Explorer tasks such as creating new folders and moving items between folders. You can also use other language packages to build your own reusable Gallery components. A component need not be part of a project to reside in the Gallery.

What do you want to do?

  • Add files to a project

  • Adding a Gallery item to a project

  • Create a Gallery folder
