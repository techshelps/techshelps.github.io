Edit and Continue Error ENC1005

Symbol was renamed, removed, or modified: symbol

This error can occur if you apply code changes or use Automatic Edit and Continue after one of the following changes:

  • Removing a function

  • Renaming a function

  • Changing a function prototype (arguments, calling convention, or return value)

  • Renaming a global or static variable

Edit and Continue does not work with these changes. For more information, see Limitations of Edit and Continue.
