Run the Profiler from the Command Line

Home |  Overview |  Details

Feature Only in Professional and Enterprise Editions   Profiling is supported only in Visual C++ Professional and Enterprise Editions. For more information, see Visual C++ Editions.

You can issue profiler commands from your command line prompt. You can also place profiler commands in a batch file and execute them that way. See Profiling from the Command Line for more information on why you would use this method instead of profiling from within the development environment.

See Select Code to Profile for information on how to specify which code shall be profiled.

There are three topics that discuss running the profiler from the command line.

  • Using PROFILE, PREP, and PLIST

  • Profiler Batch Response Files

  • Standard Batch Files
