Error: Unable to Resolve an IP Address

Full Error Text: Unable to resolve an IP address for the remote machine “xxxxxxx”.

This means that the machine name “xxxxxxx” could not translate into an IP address. The machine name for the remote connection is entered using the Debugger Remote Connection of the Build menu. Click Settings and check to make sure that the Target machine name or address is valid. At each machine that allows remote debugging, you must specify the Target machine name or address.

Tip   You can also specify the Target machine name or address by running MSVCMON.EXE.
