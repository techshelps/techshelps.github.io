Adding Files to a Project

Home |  Overview |  How Do I

You can add files to a project no matter where the files reside. Note that the Gallery accommodates only recognized components and controls: the default ActiveX controls and components, and .OGX files (components you add to the Gallery from your own projects). Other types of files, such as header files, are not stored in the Gallery. To insert files into the active project workspace, click the Project menu, point to Add To Project, then click Files.

For more information on adding files to a project, see Adding and Removing Files from Projects in the Visual C++ User's Guide.

To add a file to a project

  1. Open the project workspace to which you want to add a file.

    If you have more than one project loaded in the workspace, select the appropriate project: On the Project menu, point to Set Active Project, and click the name.

  2. On the Project menu, point to Add To Project, then click Files.

  3. In the Insert Files into Project dialog box, select the file to add to the project workspace.
