Reusing Code: Overview

Home |  How Do I |  Details

The Gallery stores code for later sharing or reuse. Such code can be wizards; ActiveX controls; HTML, text, or graphics files (including .BMP, .GIF, or .JPG); or shared classes that you have added to the Gallery from your own projects (using applicable languages).

You can enhance your projects by using items from the Gallery. The Gallery comes with default folders containing prepackaged Visual C++ Components and Registered ActiveX Controls.

You can also add items to the Gallery, and organize the contents of the Gallery by performing ordinary Explorer tasks such as creating new folders, and moving items between folders. You can also use specific language packages to build your own reusable Gallery components. A component need not be part of a project to reside in the Gallery.
