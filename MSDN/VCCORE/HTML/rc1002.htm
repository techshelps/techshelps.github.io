Resource Compiler Fatal Error RC1002

out of heap space

The Resource Compiler ran out of storage for items that it stores in its heap. One of the following may be a solution:

  • Increase the Windows swap file space. For more information about increasing the swap-file space refer to the Microsoft Windows User's Guide.

  • Split the current file into two or more files and compile them separately.

  • Remove other programs or drivers running in the system, which could be consuming significant amounts of memory.
