The following tasks involve reusing code with the aid of the Gallery.

You can enhance your projects by using items from the Gallery. The items can be prepackaged items such as the Visual C++ Components and Registered ActiveX Controls, or code you have created and added to the Gallery. You can also add third-party components and controls.

What do you want to do?

  • Access and view Gallery contents

  • Add a Gallery item to a project

  • Add items to the Gallery

  • Organize Gallery items

  • Learn about Registered ActiveX Controls

  • Learn about Visual C++ Components
