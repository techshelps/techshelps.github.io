Editing the Properties of Multiple Controls Together

Home |  Overview |  How Do I

In the dialog editor, you can select multiple controls and open a property page for all of them by clicking Properties on the View menu. The changes you make on this page are reflected in all the controls you selected.

If you make a mistake, choose Undo from the Edit menu. A single undo will correct the mistake for all the controls.
