BSCMAKE Error BK1509

out of heap space

BSCMAKE ran out of memory, including virtual memory.

One of the following may be a solution:

  • Increase the size of the swap file.

  • Free some space on disk.

  • Increase the size of the Windows swap file.

  • Reduce the memory that BSCMAKE will require by using one or more options. Use /Ei or /Es to eliminate some input files. Use /Em to eliminate macro bodies.
