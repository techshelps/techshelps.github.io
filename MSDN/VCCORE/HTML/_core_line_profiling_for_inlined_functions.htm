Line Profiling for Inlined Functions

Home |  Overview |  Details

Feature Only in Professional and Enterprise Editions   Profiling is supported only in Visual C++ Professional and Enterprise Editions. For more information, see Visual C++ Editions.

The profiler can provide line-level hit count and coverage information for inlined functions. Be sure to include the source lines in the Phase I PREP command-line options, just as you would include source lines in the calling program.
