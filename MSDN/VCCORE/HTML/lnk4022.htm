Linker Tools Warning LNK4022

cannot find unique match for symbol "symbol"

LINK or LIB found multiple matches for the given undecorated symbol and it could not resolve the ambiguity. No output file (.EXE, .DLL, .EXP, or .LIB) is produced. This warning is followed by one warning LNK4006 for each duplicate symbol and is finally followed by fatal error LNK1152.

To prevent this warning, specify the symbol in its decorated form. Run DUMPBIN on the object to see decorated names.
