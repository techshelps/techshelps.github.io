MFC provides class CMenu for manipulating your program’s menus. MFC message maps also simplify managing the enabled/disabled state of menu items and toolbar buttons.

What do you want to do?

  • Update menus and toolbar buttons to reflect program conditions

  • Dynamically modify menus using class CMenu
