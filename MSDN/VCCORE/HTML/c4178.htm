Compiler Warning (level 1) C4178

case constant 'constant' too big for the type of the switch expression

The specified constant in a switch expression was too large to fit in the type to which it was assigned. The following example causes this warning:

void main()
{ 
    int i;  // maximum size of unsigned long int is 4294967295
    switch( i ) 
    {
        case 4294967295:   // OK
            break;
        case 4294967296:   // warning
            break;
    }
 }
