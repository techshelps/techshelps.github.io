Accessing WizardBar

Home |  Overview |  How Do I |  Tutorial

To view the WizardBar, right-click on an empty (unused) portion of the toolbar area, and select WizardBar from the menu. By default the docked WizardBar appears in the toolbar area.

To undock the WizardBar, double-click the double bar on the far left edge of the toolbar. To redock, double-click on the title bar of the undocked toolbar. The docked and undocked versions of the toolbar are shown below:

WizardBar, Docked

WizardBar, Undocked

As shown in the preceding figures, the WizardBar has the following controls:

  • a Class drop-down list, which displays the classes in the project

  • a Filter drop-down list, which specifies the class members to list in the Members drop-down list

  • a Members drop-down list, which displays the members of the selected class

  • an Action button, which when clicked executes the default action (the default action depends on the WizardBar's current context)

  • an Action menu, which is a drop-down menu listing all the tasks the WizardBar can currently perform

For a complete description of WizardBar controls, see Overview: The Universal WizardBar.
