To create a Win32 (exe) program

  1. On the File menu, click New and then click the Projects tab.

  2. Specify the Project Name, Location, Workspace, Dependency, and Platforms options and then double-click the Application icon.

  3. In Step 1 of the wizard, specify the level of file support you want for the project.

  4. Click here for more information on how to prepare your non-MFC program for use.
