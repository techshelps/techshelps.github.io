Linker Tools Warning LNK4074

unable to load filename; cannot checksum

The linker tried to load the dynamic-link library filename but could not. A function in filename is used to checksum the executable file.

One of the following may be a cause:

  • The linker could not find filename because it is not in the current directory or in a directory in the PATH environment variable.

  • There was a problem with filename. Recopy filename from the installation disks.
