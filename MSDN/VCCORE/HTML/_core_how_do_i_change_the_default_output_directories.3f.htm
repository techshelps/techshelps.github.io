How do I change the default output directories?

When you build a project, you can select the directories in which to put the intermediate and final output files. By putting these files in different directories, you can maintain copies of the same files built in different ways — for instance, the Debug and the Release versions of your project.

To select output directories

  1. On the Project menu, click Settings.

  2. In the Settings For pane, select the item for which you want to set directories.

    If you select the project configuration, you can set both intermediate and output directories; if you select a file, you can set only the intermediate directory.

  3. Select the General tab.

  4. In the Intermediate Files text box, type the directory name for the intermediate files (.OBJ files, for instance).

  5. If you are setting directories for the project level, type the directory name for the final output files (.EXE files or .CLASS, for instance) in the Output Files text box.
