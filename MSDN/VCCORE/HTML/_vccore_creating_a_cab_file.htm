Creating a CAB File

This section describes creating CAB files for distributing ATL and MFC controls over the Internet. If you need more information about CAB files, see the Cabinet File Reference in the Platform SDK section Setup and System Management Services\Setup API\Overview\Cabinet Files. The Platform SDK is included with the MSDN online documentation.

To create a CAB file:

  1. Create an INF file.

  2. Run the CABARC utility.
