Status bars are control bars that let you display information to your user in a set of panes or “indicators.”

Create a status bar in your program by selecting the Initial Status Bar option in AppWizard.

What do you want to do?

  • Work with status bars

  • Display command information in the status bar

  • Create a new status-bar pane

  • Make a status-bar pane display text

  • Update the text of a status-bar pane
