To create a console program

  1. On the File menu, click New and then click the Projects tab.

  2. Specify the Project Name, Location, Workspace, Dependency, and Platforms options and then double-click the Console Application icon.

  3. In Step 1 of the wizard, select the file support you want in the project.

  4. Prepare the console program for use
