Running a Custom Batch File

Home |  Overview |  Details

Feature Only in Professional and Enterprise Editions   Profiling is supported only in Visual C++ Professional and Enterprise Editions. For more information, see Visual C++ Editions.

When a batch file is executed using the Custom option in the Profile dialog box, the development environment substitutes the project’s program name for the %1 parameter.

To specify your program’s command-line arguments

  1. On the Build menu, click Settings to display the Project Settings dialog box.

  2. Click the Debug tab and type the arguments in the Program arguments text box.

For more information on creating profiler batch files, see Run the Profiler from the Command Line.
