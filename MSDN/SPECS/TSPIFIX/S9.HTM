1.7 Related Documentation

Other documentation that will help you understand the Telephony services, as they apply to Windows, include:

  • Microsoft Windows Software Development Kit (SDK). Documents of specific interest in the SDK are the Multimedia Programmer's Guide, and the Multimedia Programmer's Reference.
  • Microsoft Windows Device Driver Kit (DDK). Documents of specific interest in the DDK are the Multimedia Device Adaptation Guide.
  • As mentioned earlier, the Windows Telephony API document gives telephony background information and describes the telephony model as presented to application writers.
