MCI_DGV_INFO_PARMS

The MCI_DGV_INFO_PARMS structure contains parameters for the MCI_INFO message for digital-video devices. When assigning data to its, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the fields.

typedef struct {

DWORD dwCallback;

LPSTR lpstrReturn;

DWORD dwRetSize;

DWORD dwItem;

} MCI_DGV_INFO_PARMS;

Fields

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

lpstrReturn

Specifies a long pointer to a user-supplied buffer for the return string.

dwRetSize

Specifies the size in bytes of the buffer for the return string.

dwItem

Specifies a constant indicating the information requested.
