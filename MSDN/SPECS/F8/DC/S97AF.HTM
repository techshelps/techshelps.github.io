MCI_DGV_CAPTURE_PARMS

The MCI_DGV_CAPTURE_PARMS structure contains parameters for the MCI_CAPTURE message. When assigning data to its fields, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the fields.

typedef struct {

DWORD dwCallback;

LPSTR lpstrFileName;

RECT rc;

} MCI_DGV_CAPTURE_PARMS;

Fields

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

lpstrFileName

Specifies a long pointer to a user-supplied buffer for the filename.

rc

Specifies a rectangle.
