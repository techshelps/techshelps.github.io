MCI_DGV_STOP_PARMS

The MCI_DGV_STOP_PARMS structure contains the information for MCI_STOP command message for digital-video devices. When assigning data to its fields, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the fields.

typedef struct {

DWORD dwCallback;

} MCI_DGV_STOP_PARMS;

dwCallback


The low-order word specifies a window handle used for the MCI_NOTIFY flag.
