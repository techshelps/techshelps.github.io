MCI_DGV_STEP_PARMS

The MCI_DGV_STEP_PARMS structure contains parameters for the MCI_STEP message for digital-video devices. When assigning data its fields, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the fields.

typedef struct {

DWORD dwCallback;

DWORD dwFrames;

} MCI_DGV_STEP_PARMS;

Fields

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

dwFrames

Specifies the number of frames to step.
