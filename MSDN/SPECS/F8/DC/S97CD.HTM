MCI_DGV_WINDOW_PARMS

The MCI_DGV_WINDOW_PARMS structure contains parameters for the MCI_WINDOW message for digital-video devices. When assigning data to its fields, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the fields.

typedef struct {

DWORD dwCallback;

WORD hWnd;

WORD wReserved1;

WORD nCmdShow;

WORD wReserved2;

LPSTR lpstrText;

} MCI_DGV_WINDOW_PARMS;

Fields

dwCallback


The low-order word specifies a window handle used for the MCI_NOTIFY flag.

hWnd

Specifies a handle to the display window.

wReserved1

Reserved.

nCmdShow

Specifies how the window is displayed.

wReserved2

Reserved.

lpstrText

Specifies a long pointer to a null-terminated string containing the window caption.
