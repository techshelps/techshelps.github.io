MCI_DGV_SAVE_PARMS

The MCI_DGV_SAVE_PARMS structure contains the information for the MCI_SAVE message for digital-video devices. When assigning data to its fields, set the corresponding MCI flags in the lParam1 parameter of mciSendCommand to validate the fields.

typedef struct {

DWORD dwCallback;

DWORD lpstrFileName;

RECT rc;

} MCI_DGV_SAVE_PARMS;

Fields

dwCallback

The low-order word specifies a window handle used for the MCI_NOTIFY flag.

lpstrFileName

Specifies a far pointer to the buffer containing a null-terminated string.

rc

Specifies a rectangle.
