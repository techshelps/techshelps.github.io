Document Overview

This document presents information concerning Telephony API functions, messages, and data types. Following this introduction:

Chapter 2, Simple Telephony, provides a complete listing of the simple telephony functions, messages and data types. Simple Telephony is targeted towards applications what want to add simple elements of telephony but don't require the flexibility or control provided by the Full Telephony API. The Full Telephony API is described in chapters 2 through 6;

Chapter 3, Overview, provides an overview of the Full Telephony API. It describes basic concepts, the usage model underlying the Telephony API, its relation to other Windows 3.1 APIs, and an overview of the functions, messages, and data structures provided by the Telephony API. They are described in logical functional groupings;

Chapter 4, Functions, provides a detailed listing of all of the functions provided by the Telephony API;

Chapter 5, Messages, provides a complete listing of all of the messages associated with the Telephony API;

Chapter 6, Data Types, describes the data types and structures defined by the Telephony API.
