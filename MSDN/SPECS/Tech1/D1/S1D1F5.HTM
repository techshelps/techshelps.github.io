IConnectionPoint::Unadvise

HRESULT IConnectionPoint::Unadvise([in] DWORD dwCookie);

Terminates an advisory connection previously established through IConnectionPoint::Advise. The dwCookie argument identifies the connection to terminate.

Argument

Type

Description

dwCookie

DWORD

[in] The connection cookie previously returned from IConnectionPoint::Advise.


Return Value

Meaning

S_OK

The connection was successfully terminated.

E_UNEXPECTED

An unknown error occurred.

CONNECT_E_NOCONNECTION

dwCookie does not represent a value connection to this connection point.
