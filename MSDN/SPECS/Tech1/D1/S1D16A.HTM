IMalloc::GetSize

ULONG IMalloc::GetSize(pv)

Return the size, in bytes, of the memory block allocated by a previous call to IMalloc::Alloc or IMalloc::Realloc on this memory manager.

Argument

Type

Description

pv

void *

The pointer to be tested. May be NULL, in which case -1 is returned.

return value

ULONG

The size of the allocated memory block
