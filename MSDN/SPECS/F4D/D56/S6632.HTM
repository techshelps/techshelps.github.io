hmtx: Horizontal Metrics

The 'hmtx' table provides Advance Widths and Left Side Bearings for each glyph in the font, to describe the spacing of the glyphs when they are written horizontally.

Requirements

  • Data in 'hmtx' is important for vertical writing, even for glyphs used only in vertical writing. GDI uses a glyph's left side bearing and advance width to determine its position relative to the vertical baseline. In most cases, vertical characters in Far East fonts should be visually centered, via 'hmtx' data, on the full unit width (Han width).
Recommendations

  • None additional to TTF spec.
