GSUB: Glyph Substitution

The 'GSUB' table provides information on glyph-substitution features that are required or appropriate when the font is used to render certain languages and scripts. These can be classified according to TrueType Open Script, Language, and Feature tags.

Requirements

  • Fonts should use only registered script, language, and feature tags. For complete information on the tags available, see the "Tag Registry" in the TrueType Open specification. Registered script tags include:

'hani'

Han Ideographic

'kana'

Kana (Hiragana & Katakana)

'hang'

Korean Hangul

'latn'

Latin


Registered language tags include:

'ZHS '

Chinese (PRC)

'ZHT '

Chinese (Taiwan)

'JAN '

Japanese

'KOR '

Korean


Note that the language tags end with a space character (0x20).

  • The 'vert' feature lets a font identify a list of horizontal glyph forms, and a corresponding list of vertical glyph forms to be substituted for vertical writing. Far East GDI will use this feature to substitute glyphs when an application requests a vertical writing font.
'vert'

Vertical glyph variants


  • Although there are many possible ways of implementing the 'vert' feature, Far East editions of Windows 95 and Windows NT 3.51 are guaranteed to read only one specific implementation.
