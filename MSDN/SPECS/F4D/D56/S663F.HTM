Additional References

Additional information related to Far East TrueType is available in the following documents:

  • "TrueType in Windows 95", Microsoft Typography (available on MSDN) An overview of the new features available for TrueType fonts on Windows 95.

  • Developing International Software for Windows 95 and Windows NT, Nadine Kano, Microsoft Press, 1995. A thorough resource covering many issues in international software design, including chapters on character sets and processing Far East writing systems.

  • "Character Sets and Codepages", Microsoft Typography (available on MSDN) An overview of the relationships between character sets, glyphs, character encodings, Unicode, and codepages.

  • NLSAPI Functional Specification (available on MSDN) The specification for the Win32 APIs to support linguistic and cultural conventions.

  • The Unicode Standard, multiple versions 1991-1995, Unicode Inc. (Version 1.1 change list available on MSDN) The official reference for the Unicode character encoding. A two-volume trade edition of Version 1.0 was published by Addison-Wesley in 1991-1992. Information is also available at http://www.unicode.org and ftp://unicode.org/pub/
