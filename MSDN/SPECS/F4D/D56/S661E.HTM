Summary of Requirements and Recommendations

Far East font vendors should ensure that their TrueType fonts provide the following features wherever possible:

  • Unicode encoding in 'cmap' and 'name'
  • Character set identification flags in 'OS/2'
  • Vertical metrics in 'vhea' and 'vmtx'
  • Vertical glyph substitution in 'GSUB'
  • Close adherence to TrueType specifications
Vendors may also choose to add many optional features to their TrueType fonts. This document contains guidelines for supporting the following features:

  • Embedded bitmaps
  • TrueType Collections
  • Gray scale rendering (font smoothing)
  • Baseline positioning information
