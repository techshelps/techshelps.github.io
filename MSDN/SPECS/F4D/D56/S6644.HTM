4.1 Definition of Physical Font File Format

The physical font file contains the following types of data:

  1. General header The general header consists of an identification ID, a parameter structure, and segment tables.

  2. Code area table This is a segment containing a list of areas of character codes.

  3. Patch tables These are segments containing patch information for virtual bitmap index tables. In general, a patch table is used for vertical-writing characters.

  4. Bitmaps These are segments that store bitmaps. For horizontal-writing character bitmaps, patch bitmaps for horizontal-writing characters and patch bitmaps for vertical-writing characters
