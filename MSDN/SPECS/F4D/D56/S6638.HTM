vhea: Vertical Metrics Header

The vertical header table contains global vertical writing metrics which apply to all glyphs in the font.

Requirements

  • The advanceHeightMax, minTopSideBearing, and minBottomSideBearing fields must reflect the actual data in 'glyf' and 'vmtx'. Glyphs with no contours should be ignored for the purposes of these calculations.
  • A font with vertical metrics must have both a 'vhea' and a 'vmtx' table.
Recommendations

  • None additional to TTF spec.
