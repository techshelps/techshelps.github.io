Implementing TrueType Fonts (TTFs)

This section provides detailed information on specific requirements and recommendations for TrueType fonts for Far East Windows 95 and Windows NT 3.51, arranged according to the tables in a TrueType font file.

The discussion assumes an understanding of the TrueType and TrueType Open file format specifications:

  • TrueType 1.0 Font Specification, version 1.66, Microsoft Typography (available on MSDN 10/95) Detailed information about the file formats for TrueType fonts and TrueType Collections, as well as general Microsoft recommendations for Windows fonts.

  • TrueType Open Font Specification, version 1.0, Microsoft Typography (available on MSDN 10/95) Detailed information about the five new tables (GSUB, GPOS, JSTF, BASE, GDEF) recently added to the TrueType format to support high-quality international typography.

Note: The requirements and recommendations listed here are in addition to those contained in the general specification documents.
