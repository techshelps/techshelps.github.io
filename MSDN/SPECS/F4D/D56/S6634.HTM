LTSH: Linear Threshold

The 'LTSH' table defines, on a glyph-by-glyph basis, the ppem size at which it is reasonable to assume linearly scaled advance widths.

Requirements

  • This table should be included in a font if bit 4 of the flags field in the 'head' table is set, i.e., the font's advance widths scale non-linearly.
Recommendations

  • None additional to TTF spec.
