8.8 Computer Object

The Computer container object represents computer and machine objects in a namespace. In some namespaces, a computer is a full-fledged DS container. In others, it is simply the representation of a device. For Active Directory version 1.1, the definition of a computer is simple and does not include objects on the desktop. The Computer component supports the IADsContainer interface. One COM interfaces, IADsComputer and IADsComputerOperations are also supported.
