10.3 Resource Object

The Resource object represents an open resource on a file service, such as an open file. Like the Session and Print Job objects, the Resource object is unlike other Active Directory objects in that it does not have a parent container.

The Resource object supports the IADs interface and one COM interface, IADsResource.

The Resource object is dynamically created by the Active Directory provider. Although Resource objects have an Active Directory Path property, the contents of the property cannot be used to instantiate the object.
