11.2 Print Job Object

The Print Job object represents a print job that is waiting to be serviced in a queue. This object is different from other Active Directory Active Directory objects in that it is not stored in a directory and thus does not have a parent container.

The Print Job object supports the mandatory IADs interface and the COM IADsPrintJob and IADsPrintJobOPeration interfaces.

The Print Job object is dynamically created by the Active Directory provider. Although Print Job objects have an Active Directory Path property, the contents of the property cannot be used to instantiate the object.
