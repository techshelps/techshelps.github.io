11.1.2 IADsPrintQueueOperations

The interface definition is as follows:

interface IADsPrintQueueOperations: IADs


{


// Read-only properties


[propget]


HRESULT Status ([out, retval]long *piStatus)


// Methods.


HRESULT PrintJobs ([out, retval]IADsCollection **ppPrintJobs);


HRESULT Pause (void);


HRESULT Resume (void);


HRESULT Purge (void);


};

Method

Syntax

Description

Status

Integer

integer value indicating current status of service. The value returned is an Active Directory constant

PrintJobs

Method

Returns a collection of references to print job objects.

Pause

Method

Pauses processing of print jobs.

Resume

Method

Resumes paused processing.

Purge

Method

Purges all jobs from the print queue.
