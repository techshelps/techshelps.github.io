10.2 Session Object

The Session object represents a connected session on a file service. Like the Print Job object, the Session object is unlike other Active Directory Active Directory objects in that it does not have a parent container.

The Session object supports the mandatory IADs interface and one COM interface, IADsSession.

The Session object is dynamically created by the Active Directory provider. Although Session objects have an Active Directory Path property, the contents of the property cannot be used to instantiate the object.
