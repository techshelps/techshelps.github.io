11.2.2 IADsPrintJobOperations

The interface definition is as follows:

interface IADsPrintJobOperations: IADs


{


// Read-only properties


[propget]


HRESULT Status ([out, retval]long *piStatus)


[propget]


HRESULT TimeElapsed ([out, retval]long *plTimeElapsed);


[propget]


HRESULT PagesPrinted ([out, retval]long *plPagesPrinted);


// Read-write properties


[propget]


HRESULT Position ([out, retval]long *plPosition);


[propput]


HRESULT Position ([out, retval]long lPosition);


// Methods.


HRESULT Pause (void);


HRESULT Resume (void);


};

Method

Syntax

Description

Status

Integer

integer value indicating current status of service. The value returned is an Active Directory constant. See 0 for values.

TimeElapsed

Interval

Gets the time elapsed, in seconds, since the job started printing.

PagesPrinted

Integer

Gets the number of pages complete.

Position

Integer

Gets and sets the numeric position of the job in the queue.

Pause

Method

Pauses processing of this print job.

Resume

Method

Resumes paused processing of this print job.
