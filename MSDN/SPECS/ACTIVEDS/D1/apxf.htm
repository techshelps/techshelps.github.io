F. Active Directory Service Constants

The following list of manifest constants appear in the Active Directory header files. These values are presented (and returned) in the ServiceType, StartType, and ErrorControl fields in the IADsServiceOperations Functional Set.

ServiceType

-----------------

//Service Type Valid Values

ADS_SERVICE_OWN_PROCESS

ADS_SERVICE_SHARE_PROCESS

ADS_SERVICE_KERNEL_DRIVER

ADS_SERVICE_FILE_SYSTEM_DRIVER

Start Type

---------------

//Start Type Valid Values

ADS_SERVICE_BOOT_START

ADS_SERVICE_SYSTEM_START

ADS_SERVICE_AUTO_START

ADS_SERVICE_DEMAND_START

ADS_SERVICE_DISABLED

Error Control Values

-----------------------------

//Error Control Values

ADS_SERVICE_ERROR_IGNORE

ADS_SERVICE_ERROR_NORMAL

ADS_SERVICE_ERROR_SEVERE

ADS_SERVICE_ERROR_CRITICAL
