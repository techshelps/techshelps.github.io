Size

The size of text and graphics affects usability as well as accessibility. These guidelines should followed to ensure that type size is not a problem for users:

  • Allow the user to select font sizes for displayed information.
  • If feasible, provide a zoom feature.
  • Draw objects using the sizes (screen metrics) selected in Control Panel.
  • If you draw lines, determine the proper width rather than using a fixed value.
  • Make sure you test display scaling ratios using the Custom Font Size feature.
