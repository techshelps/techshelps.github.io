Designed for Microsoft BackOffice Logo Requirements:

BackOffice Server System Requirements

  • The machine must be on the Windows NT™ Hardware Compatibility List.
    • A machine going through BackOffice testing must first pass the current Windows NT Hardware Compatibility tests.
  • The machine must have a minimum of an Intel® Pentium®-90 (or compatible) or equivalent RISC processor.
  • The machine must have a minimum of 32mb of RAM per processor expandable to at least a system total of 128mb.
  • The machine must support a 32-bit bus architecture.
  • The video must support at least 640x480x256 colors.
  • PCI based machines must properly report interrupts used on the PCI bus.
    • PCI based machines must properly conform to the PCI 2.0 specification.
  • The machine must support memory parity enabled or double-bit ECC memory.
  • The machine must have a minimum of 256k of Level-2 cache.
  • MP machines must have separate Level-2 caches per processor.
  • The machine MUST NOT have an embedded single-FIFO dual-channel IDE controller.
  • Components included with the system, or embedded onto the systemboard (net, video, SCSI etc.) must pass Hardware Compatibility Testing (not BackOffice Certification) in the machine being tested.
    • Components included with the system, or embedded onto the systemboard must have drivers available for them (or use drivers included with Windows NT) for the machine being tested. Drivers tested in this way will (optional) become part of the Windows NT Driver Library.
  • Machines available in complete server configurations from the manufacturer must also be certified in those configurations.
    • If a machine is available in a "bare bones" configuration, it will be tested with industry standard add-in options.
    • If a machine is available completely configured as a server, it will be tested in that configuration(s).
  • Machine must include any/all configuration utilities required.

Prepare The System For Testing

Verify System Information Form is filled out completely.

Verify the system is free of viruses.

Run the current Windows NT™ Hardware Compatibility Tests.

To make sure you have the current test kit, email NTHCTEXT@microsoft.com

Verify IDE/Floppy connectors are keyed as to not allow the user to plug in these devices incorrectly.

Verify all expansion slots are accessible. (Not blocked by some other device)

Verify the external ports are labeled identifying what they are.

FOLLOW THESE TIPS - this will prevent most common system failures

Make sure you contact your BIOS manufacturer and get the most recent BIOS on your system.

Make sure connector icons are on the system.

Send the system which you pre-tested, not a different one.

Make sure all motherboard devices have Windows NT drivers. All third party Windows NT drivers must come on floppy disk.

If you have questions regarding the tests, email BOHTL@microsoft.com
