DCI Manager Clipping Information Functions

DCIMAN provides a set of functions to allow an application to query a visible region of a window and to be notified when this region changes.

These functions work only on the "visible region" of the window i.e., the part of the window that is visible on the screen. It does not work with the current clip region of an HDC. This is only important if an application has selected a custom clip region.

There are two ways to get clipping information via DCIMAN: directly or indirectly
