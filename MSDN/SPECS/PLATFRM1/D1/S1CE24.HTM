DCICreatePrimary

Creates a DCI primary surface.

int FAR PASCAL _loadds DCICreatePrimary(hdc, *lplpSurface)

HDC

hdc;


LPDCISURFACEINFO FAR *lplpSurface;

INPUT PARAMETERS

hdc

Device context returned from the DCIOpenProvider call.

lplpSurface

Far pointer to a far pointer to a DCISURFACEINFO structure.


RETURNS

Returns DCI_OK on success, or an error code on failure.
