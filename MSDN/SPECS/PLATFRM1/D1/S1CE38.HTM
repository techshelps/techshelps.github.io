Benefits To Graphics IHVs

  • IF09 will be supported in future versions of Indeo, so IHV's investment in color conversions will be protected.
  • There will be no performance degradation between this method and the original DCI-enabled Indeo, in which color conversion to YUV formats was done in the codec.
  • Sample code for some of YUV 4:2:2, is available in the Beta Kit. Other YUV flavors are easily derivable from this sample code.
