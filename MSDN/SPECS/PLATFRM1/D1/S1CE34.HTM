Definition of Common Indeo Intermediate Data Format - IF09

The Indeo Intermediate Common Format (Proposed FOURCC code 'IF09'), is a data format which is guaranteed to be supported as an output format via the ICM_DECOMPRESSEX message for all Indeo codecs. The purpose of this definition is to enable the "externalization" of color conversion for specific graphics hardware, allowing IHVs to support custom YUV formats.
