3.5.3 LanguageId

The language ID is included in each resource to specify the language that the strings are written with when they need to be translated back to single-byte strings. In addition, there may be multiple resources of exactly the same type and name that differ in only the language of the strings within the resources.

The language IDs are documented in Appendix A of the NLS specification, or in WINNLS.H. The language of a resource or set of resources is specified by the LANGUAGE statement.
