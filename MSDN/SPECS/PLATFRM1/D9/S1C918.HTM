1. Overview

This document details the structure of the resource binary file (.RES) format for Microsoft® Windows NT™ and MS-DOS® Win32™. The structure is similar to the existing Windows 16 (Windows 3.0/3.1) structure, but supports essential new features such as Unicode™ strings, version headers, and DWORD alignment. To support these changes, the file written by the resource compiler must be changed from Windows 16.
