9.2. INQUIRY command

All devices are required to return INQUIRY data in a timely manner without excessive delays. This enables a SCSI host to scan the SCSI bus and boot without significant delays.

Note Historically, some SCSI devices when first powered up performed operations that delayed their response to selection for lengthy mechanical actions and device initialization (e.g., rewinding an inserted tape, spinning up the medium, etc.). Excessive delays are not acceptable in a PnP SCSI environment.
