5.0. Conjoining Korean Jamos

Unicode 1.1 contains both a large set of precomposed modern Hangul syllable blocks and a set of conjoining Korean jamos, which can be used to encode all modern and ancient syllable blocks.

Jamos can be classified into three sets of characters: Choseong (leading consonants, or syllable-initial characters), Jungseong (vowels, or syllable-peak characters), and Jongseong (trailing consonants, or syllable-final characters). In the following discussion, these can be abbreviated by L (leading consonant), V (vowel), and T (trailing consonant).
