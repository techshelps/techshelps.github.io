2.0. Changes in Unicode 1.0

As discussed in Volumes 1 and 2, small changes have been made to Unicode 1.0 in order to align it with the international character encoding standard, ISO/IEC 10646-1. In order to expedite use of Unicode in the interim, the Unicode Consortium issued an intermediate version, Unicode 1.0.1, which consisted of Unicode 1.0 modified by the changes necessary to make the character codes a proper subset of ISO/IEC 10646-1. The text of the 1.0.1 document is subsumed by this document.
