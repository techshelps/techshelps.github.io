3.3. Byte Order Mark

U+FEFF

ZERO WIDTH NO-BREAK SPACE


In addition to the meaning of BYTE ORDER MARK as defined in Volume 1 of the Unicode standard, the code value U+FEFF may now also be used as ZERO WIDTH NO-BREAK SPACE (ZWNBSP). For convenience in discussion, it can also be referred to by this name (which is the ISO/IEC 10646-1/Unicode 1.1 name for U+FEFF).

ZWNBSP behaves like a U+00A0 NO-BREAK SPACE in that it indicates the absence of word boundaries; however, ZWNBSP has no width. For example, this character can be inserted after the fourth character in the text "base + delta" to indicate that there should be no line break between the "e" and the "+" (for more information, see Volume 2, pp. 6–7).
