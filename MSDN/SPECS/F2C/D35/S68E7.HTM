2.7. Control Characters

ISO/IEC 10646-1 forbids the use of the C1 control characters from U+0080 - U+009F. Instead, it reserves those code locations, and requires use of an ESC FE sequence instead (see ISO/IEC 6429). This does not affect Unicode implementations because Unicode does not specify the use of control codes (beyond how to express them in a 16-bit form), and considers such specification to be subject to other protocols.
