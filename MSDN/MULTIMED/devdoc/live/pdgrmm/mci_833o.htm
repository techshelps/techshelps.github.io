Video-Overlay Command Set

Video-overlay devices support the following set of commands:

String form Message form
break MCI_BREAK
capability MCI_GETDEVCAPS
close MCI_CLOSE
freeze MCI_FREEZE
info MCI_INFO
load MCI_LOAD
open MCI_OPEN
put MCI_PUT
save MCI_SAVE
set MCI_SET
status MCI_STATUS
sysinfo MCI_SYSINFO
unfreeze MCI_UNFREEZE
where MCI_WHERE
window MCI_WINDOW
