Multimedia Joystick Functions

An application uses the joystick functions to query a joystick driver and to prepare an application to receive notification messages from a joystick driver.

The following functions are used with multimedia joysticks.

joyGetDevCaps
joyGetNumDevs
joyGetPos
joyGetPosEx
joyGetThreshold
joyReleaseCapture
joySetCapture
joySetThreshold
