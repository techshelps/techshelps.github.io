Videodisc Command Set

Videodisc devices support the following set of commands:

String form Message form
break MCI_BREAK
capability MCI_GETDEVCAPS
close MCI_CLOSE
escape MCI_ESCAPE
info MCI_INFO
open MCI_OPEN
pause MCI_PAUSE
play MCI_PLAY
resume MCI_RESUME
seek MCI_SEEK
set MCI_SET
spin MCI_SPIN
status MCI_STATUS
step MCI_STEP
stop MCI_STOP
sysinfo MCI_SYSINFO
