ICGetDefaultKeyFrameRate

The ICGetDefaultKeyFrameRate macro queries a video compression driver for its default (or preferred) key-frame spacing. You can use this macro or explicitly call the ICM_GETDEFAULTKEYFRAMERATE message.

DWORD ICGetDefaultKeyFrameRate(
  hic  
);

Parameters

hic
Handle of a compressor.
wParam
Address to contain the preferred key-frame spacing.

Return Values

Returns ICERR_OK if the driver supports this message or ICERR_UNSUPPORTED otherwise.

Remarks

The ICGetDefaultKeyFrameRate macro returns the default key frame rate.

QuickInfo

  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
