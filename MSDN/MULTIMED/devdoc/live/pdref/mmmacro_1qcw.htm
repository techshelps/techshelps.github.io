capGrabFrameNoStop

The capGrabFrameNoStop macro fills the frame buffer with a single uncompressed frame from the capture device and displays it. Unlike with the capGrabFrame macro, the state of overlay or preview is not altered by this message. You can use this macro or explicitly call the WM_CAP_GRAB_FRAME_NOSTOP message.

BOOL capGrabFrameNoStop(
  hwnd  
);

Parameters

hwnd
Handle of a capture window.

Return Values

Returns TRUE if successful or FALSE otherwise.

Remarks

For information about installing callback functions, see the capSetCallbackOnError and capSetCallbackOnFrame macros.
