ICDrawEnd

The ICDrawEnd macro notifies a rendering driver to decompress the current image to the screen and to release resources allocated for decompression and drawing. You can use this macro or explicitly call the ICM_DRAW_END message.

DWORD ICDrawEnd(
  hic  
);

Parameters

hic
Handle of a driver.

Return Values

Returns ICERR_OK if successful or an error otherwise.

Remarks

The ICM_DRAW_BEGIN and ICM_DRAW_END messages do not nest. If your driver receives ICM_DRAW_BEGIN before decompression is stopped with ICM_DRAW_END, it should restart decompression with new parameters.
