ICGetDefaultQuality

The ICGetDefaultQuality macro queries a video compression driver to provide its default quality setting. You can use this macro or explicitly call the ICM_GETDEFAULTQUALITY message.

DWORD ICGetDefaultQuality(
  hic  
);

Parameters

hic
Handle of a compressor.
wParam
Address to contain the default quality value. Quality values range from 0 to 10,000.

Return Values

Returns ICERR_OK if the driver supports this message or ICERR_UNSUPPORTED otherwise.

Remarks

The ICGetDefaultQuality macro returns the default quality value.

QuickInfo

  Windows: Requires Windows 95 or later.
