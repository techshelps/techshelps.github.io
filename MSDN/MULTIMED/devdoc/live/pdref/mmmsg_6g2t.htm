MM_MOM_DONE

The MM_MOM_DONE message is sent to a window when the specified MIDI system-exclusive or stream buffer has been played and is being returned to the application.

MM_MOM_DONE 
wParam = (WPARAM) hOutput 
lParam = (LPARAM) lpMidiHdr

Parameters

hOutput
Handle of the MIDI output device that played the buffer.
lpMidiHdr
Address of a MIDIHDR structure identifying the buffer.

Return Values

This message does not return a value.

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in mmsystem.h.
