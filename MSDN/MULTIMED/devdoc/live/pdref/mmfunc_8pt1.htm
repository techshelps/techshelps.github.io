acmStreamMessage

The acmStreamMessage function sends a driver-specific message to an ACM driver. 

MMRESULT ACMAPI acmStreamMessage(
  HACMSTREAM has, 
  UINT uMsg,      
  LPARAM lParam1, 
  LPARAM lParam2  
);

Parameters

has
Handle of an open conversion stream.
uMsg
Message to send.
lParam1 and lParam2
Message parameters.

Return Values

Returns the value returned by the ACM device driver.

QuickInfo

  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.

  Import Library: Use msacm32.lib.
