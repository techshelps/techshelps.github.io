MCI_DGV_INFO_PARMS

The MCI_DGV_INFO_PARMS structure contains parameters for the MCI_INFO command for digital-video devices. 

typedef struct {
    DWORD  dwCallback; 
    LPSTR  lpstrReturn; 
    DWORD  dwRetSize; 
    DWORD  dwItem; 
} MCI_DGV_INFO_PARMS;

Members

dwCallback
The low-order word specifies a window handle used for the MCI_NOTIFY flag.
lpstrReturn
Address of buffer for return string.
dwRetSize
Size, in bytes, of return buffer.
dwItem
Constant describing information to return.

Remarks

When assigning data to the members of this structure, set the corresponding flags in the fdwCommand parameter of the mciSendCommand function to validate the members.
