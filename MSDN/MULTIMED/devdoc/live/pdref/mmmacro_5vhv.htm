MCIWndGetStyles

The MCIWndGetStyles macro retrieves the flags specifying the current MCIWnd window styles used by a window. You can use this macro or explicitly send the MCIWNDM_GETSTYLES message. 

UINT MCIWndGetStyles(
  hwnd  
);

Parameters

hwnd
Handle of the MCIWnd window.

Return Values

Returns a value representing the current styles of the MCIWnd window. The return value is the bitwise OR operator of the MCIWnd window styles (MCIWNDF flags).
