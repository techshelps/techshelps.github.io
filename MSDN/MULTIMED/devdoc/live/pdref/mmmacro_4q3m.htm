capFileSaveDIB

The capFileSaveDIB macro copies the current frame to a DIB file. You can use this macro or explicitly call the WM_CAP_FILE_SAVEDIB message.

BOOL capFileSaveDIB(
  hwnd,  
  szName 
);

Parameters

hwnd
Handle of a capture window.
szName
Address of the null-terminated string that contains the name of the destination DIB file.

Return Values

Returns TRUE if successful or FALSE otherwise.

If an error occurs and an error callback function is set using the capSetCallbackOnError macro, the error callback function is called.

Remarks

If the capture driver supplies frames in a compressed format, this call attempts to decompress the frame before writing the file.
