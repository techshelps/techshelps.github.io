undo

The undo command reverses the action taken by the most recent successful copy, cut, delete, undo, or paste command. Digital-video devices recognize this command. 

wsprintf(lpstrCommand, "undo %s %s", lpszDeviceID, lpszFlags);

Parameters

lpszDeviceID
Identifier of an MCI device. This identifier or alias is assigned when the device is opened.
lpszFlags
Can be "wait", "notify", "test", or a combination of these. For more information about these flags, see The Wait, Notify, and Test Flags.

Return Values

Returns zero if successful or an error otherwise.

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
