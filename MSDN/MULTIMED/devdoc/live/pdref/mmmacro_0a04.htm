MCIWndSetSpeed

The MCIWndSetSpeed macro sets the playback speed of an MCI device. You can use this macro or explicitly send the MCIWNDM_SETSPEED message. 

LONG MCIWndSetSpeed(
  hwnd,  
  iSpeed 
);

Parameters

hwnd
Handle of the MCIWnd window.
iSpeed
Playback speed. Specify 1000 for normal speed, larger values for faster speeds, and smaller values for slower speeds.

Return Values

Returns zero if successful or an error otherwise.

QuickInfo
