TIMECAPS

The TIMERCAPS structure contains information about the resolution of the timer.

typedef struct { 
    UINT wPeriodMin; 
    UINT wPeriodMax; 
} TIMECAPS;

Members

wPeriodMin
Minimum supported resolution.
wPeriodMax
Maximum supported resolution.
