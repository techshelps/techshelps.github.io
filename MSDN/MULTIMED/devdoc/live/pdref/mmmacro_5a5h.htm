MCIWndGetFileName

The MCIWndGetFileName macro retrieves the filename used by an MCI device. You can use this macro or explicitly send the MCIWNDM_GETFILENAME message. 

LONG MCIWndGetFileName(
  hwnd,  
  lp,    
  len    
);

Parameters

hwnd
Handle of the MCIWnd window.
lp
Address of an application-defined buffer to return the filename.
len
Size, in bytes, of the buffer.

Return Values

Returns zero if successful or 1 otherwise.

Remarks

If the null-terminated string containing the filename is longer than the buffer, MCIWnd truncates the filename.

QuickInfo

  Windows: Requires Windows 95 or later.
