WOM_DONE

The WOM_DONE message is sent to a waveform-audio output callback function when the given output buffer is being returned to the application. Buffers are returned to the application when they have been played, or as the result of a call to the waveOutReset function.

WOM_DONE 
dwParam1 = (DWORD) lpwvhdr 
dwParam2 = reserved

Parameters

dwParam1
Address of a WAVEHDR structure identifying the buffer.
dwParam2
Reserved; must be zero.

Return Values

This message does not return a value.

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in mmsystem.h.
