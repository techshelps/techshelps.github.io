GetSaveFileNamePreview

The GetSaveFileNamePreviewfunction selects a file by using the SaveAs dialog box. The dialog box also allows the user to preview the currently specified file. This function augments the capability found in the GetSaveFileName function.

BOOL GetSaveFileNamePreview(
  LPOPENFILENAME lpofn  
);

Parameters

lpofn
Address of an OPENFILENAME structure used to initialize the dialog box. On return, the structure contains information about the user's file selection.

Return Values

Returns a handle to the selected file.
