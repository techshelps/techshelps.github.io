MCIWndCanSave

The MCIWndCanSave macro determines if an MCI device can save data. You can use this macro or explicitly send the MCIWNDM_CAN_SAVE message. 

BOOL MCIWndCanSave(
  hwnd  
);

Parameters

hwnd
Handle of the MCIWnd window.

Return Values

Returns TRUE if the device supports saving data or FALSE otherwise.

QuickInfo

  Windows NT: Requires version 3.1 or later.
  Windows: Requires Windows 95 or later.
  Windows CE: Unsupported.
  Header: Declared in vfw.h.
