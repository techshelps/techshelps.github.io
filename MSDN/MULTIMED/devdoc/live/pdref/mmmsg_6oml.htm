MCIWNDM_GETZOOM

The MCIWNDM_GETZOOM message retrieves the current zoom value used by an MCI device. You can send this message explicitly or by using the MCIWndGetZoom macro.

MCIWNDM_GETZOOM 
wParam = 0; 
lParam = 0;

Return Values

Returns the most recent values used with MCIWNDM_SETZOOM.

Remarks

A return value of 100 indicates the image is not zoomed. A value of 200 indicates the image is enlarged to twice its original size. A value of 50 indicates the image is reduced to half its original size.
