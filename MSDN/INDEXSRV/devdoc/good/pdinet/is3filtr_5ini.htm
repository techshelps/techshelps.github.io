The description property is exposed by the HTML filter. It corresponds to the <META NAME="DESCRIPTION" ...> tag. If this property exists, it is used exclusively to generate the document abstract. Note that meta descriptions are not emitted by the sample HTML filter.

Note that other properties of type PRSPEC_LPWSTR in the same property set as the description property correspond to other HTML meta tags with the equivalent name.
