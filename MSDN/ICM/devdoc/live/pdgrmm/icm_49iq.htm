SetColorProfileHeader

[This is preliminary documentation and subject to change.]

The SetColorProfileHeader function sets the header data in a specified color profile.

BOOL WINAPI SetColorProfileHeader(
  HPROFILE hProfile,  
  PPROFILEHEADER pHeader  
);

Parameters

hProfile
Specifies a handle to the color profile in question.
pHeader
Pointer to the profile header data to write to the specified profile.

Return Values

If this function succeeds, the return value is TRUE.

If this function fails, the return value is FALSE. For extended error information, call GetLastError.

Remarks

If the color profile was not opened with read/write permission, SetColorProfileHeader fails.

SetColorProfileHeader overwrites the current header in the profile.
