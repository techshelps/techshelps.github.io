GetColorProfileHeader

[This is preliminary documentation and subject to change.]

The GetColorProfileHeader function retrieves the header of a color profile.

BOOL WINAPI GetColorProfileHeader(
  HPROFILE hProfile,
  PPROFILEHEADER pHeader
);

Parameters

hProfile
Specifies a handle to the color profile in question.
pHeader
Points to a variable in which the header is to be placed.

Return Values

If this function succeeds, the return value is TRUE.

If this function fails, the return value is FALSE. For extended error information, call GetLastError.
