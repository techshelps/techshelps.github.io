ICM 2.0 Functions

[This is preliminary documentation and subject to change.]

In the ICM 2.0 function reference, ICM functions are organized by category. This allows you to browse among functions that have related uses. In later releases of this information, these functions are more likely to appear in alphabetical order in the Platform SDK documentation.

The functions are grouped as follows:

  • Basic ICM 2.0 Functions for Use Within a Device Context
  • Advanced ICM 2.0 Functions for Use Outside of a Device Context
  • Device Calibration and Characterization Functions in ICM 2.0
  • Profile Management Functions in ICM 2.0
  • ICM Functions for Color Management Modules (CMMs) to Implement
  • Alphabetical List of All ICM 2.0 API Functions
