UninstallColorProfile

[This is preliminary documentation and subject to change.]

UninstallColorProfile removes a specified color profile from a specified computer. Associated files are optionally deleted from the system.

BOOL WINAPI UninstallColorProfile(
  PCTSTR pMachineName,
  PCTSTR  pProfileName,
  BOOL bDelete
);

Parameters

pMachineName
Reserved for future use; must be NULL. This parameter is intended to point to the name of the machine from which to uninstall the specified profile. A NULL pointer indicates the local machine.
pProfileName
Points to the fully qualified path name of the profile to uninstall.
bDelete
If set to TRUE, the function deletes the profile from the COLOR directory.

Return Values

If this function succeeds, the return value is TRUE.

If this function fails, the return value is FALSE. For extended error information, call GetLastError.
