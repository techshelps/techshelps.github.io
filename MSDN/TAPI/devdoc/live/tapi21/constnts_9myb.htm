LINEAGENTFEATURE_ Constants

The LINEAGENTFEATURE_ constants list features that are available for an agent on an address.

LINEAGENTFEATURE_SETAGENTGROUP
The lineSetAgentGroup function can be invoked on this address.
LINEAGENTFEATURE_SETAGENTSTATE
The lineSetAgentState function can be invoked on this address.
LINEAGENTFEATURE_SETAGENTACTIVITY
The lineSetAgentActivity function can be invoked on this address.
LINEAGENTFEATURE_AGENTSPECIFIC
The lineAgentSpecific function can be invoked on this address.
LINEAGENTFEATURE_GETAGENTACTIVITYLIST
The lineGetAgentActivityList function can be invoked on this address.
LINEAGENTFEATURE_GETAGENTGROUPLIST
The lineGetAgentGroupList function can be invoked on this address.
