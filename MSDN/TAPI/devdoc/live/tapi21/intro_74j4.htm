TAPI Functions, Messages, Structures, and Constants for Version 2.0

TAPI version 2.0 includes a number of functions, messages, structures, and constants that were either new (not available in previous versions of TAPI) or that changed in TAPI version 2.0. Although TAPI has now updated to version 2.1, the itemization of 2.0 changes is retained for the benefit of anyone previously using TAPI 1.4 or earlier.
