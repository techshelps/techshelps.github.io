Assisted Telephony Services Reference

The Assisted Telephony functions are listed in the following table. Assisted Telephony functions allow any Win32-based application to establish voice calls and media calls. For background information, see the Assisted Telephony overview.

Assisted Telephony Services Functions

tapiRequestMakeCall Submits a request to place a voice call.
tapiRequestMediaCall Obsolete. Do not use.
tapiRequestDrop Obsolete. Do not use.
tapiGetLocationInfo Returns country code and city/area code information.
