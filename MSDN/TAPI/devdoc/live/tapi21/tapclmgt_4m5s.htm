Overview of Setup

The setup program, called tcmsetup, has a command line interface. The tcmsetup program can also be used to modify the existing server or client setup. You can make modifications by running the tcmsetup program and specifying new settings. Upon completion of the program, the modifications take effect.

Note  The tcmsetup program must be run on the machine you are trying to set up. It cannot be used to configure machines remotely.
