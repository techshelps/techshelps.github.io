TAPICLIENT_Load

The TAPICLIENT_Load function is called after TAPI has loaded the Client Management DLL (through a call to LoadLibrary).

Every Client Management DLL must export TAPICLIENT_Load.

LONG TAPICLIENT_Load(
  LPDWORD lpdwAPIVersion,  
  DWORD dwReserved1,       
  DWORD dwReserved2,       
  DWORD dwReserved3        
);

Parameters

lpdwAPIVersion
dwReserved1
Reserved parameter. This value is 0.

Reserved parameter. This value is 0.

dwReserved3
Reserved parameter. This value is 0.
