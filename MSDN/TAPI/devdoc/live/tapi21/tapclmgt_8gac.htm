TAPI Client Management

Microsoft TAPI Client Management is a programming interface that extends the Microsoft Telephony API (TAPI) to a full client/server model. By extending TAPI, you can develop call center control applications.

In addition to extending TAPI, TAPI Client Management can itself be extended to allow third parties to implement additional administrative features. These features can include, but are not restricted to:

  • Client device privileges
  • Phone number verification
  • Least cost routing number modification
