Document Overview

This documentation presents information on how to use TSPI functions, messages, structures, constants, and datatypes. The sections include:

  • Overviews, which provides an overview of TSPI. It describes basic concepts, the usage model underlying TSPI, its relation to other Win32 service provider interfaces, and an overview of the functions, messages, data structures, and device classes provided by TSPI.
  • Reference, which contains functional descriptions of the following:

    TSPI Functions, which describes service provider, line device, phone device, and callback functions

    Telephony Service Provider UI DLL Interface, TSPI UI and TUISPI service provider, line device, phone device, and callback functions

    Structures, which defines TSPI data structures.

    Messages, which contains a list of TSPI messages that notify TAPI32.DLL of the occurrence of asynchronous events.

    Constants, which provides a detailed listing of TSPI phone device constants.

    Device Classes, which describes related physical devices or device drivers through which applications send and receive the data that makes up a call.

    Service Provider Datatypes, which describes specific values passed as parameters through functions or opaque handles through structures.
