Document Conventions

This document uses the following conventions.

Convention Description
Bold text Bold letters indicate terms, such as function, structure, and structure member names, that you must enter exactly as shown.
ALL CAPS All capital letters typically indicate terms, such as message and constant names, that you must enter exactly as shown.
Italic text In introductory and explanatory text, italicized words indicate that a key term or concept is being introduced. In describing functions and messages, words in italics indicate a placeholder; you are expected to provide the actual value.
Monospaced text Monospaced type indicates code samples, syntax, and data-structure definitions.
