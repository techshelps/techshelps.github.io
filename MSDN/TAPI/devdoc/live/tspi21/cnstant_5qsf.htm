TUISPIDLL_OBJECT_

TUISPIDLL_OBJECT_LINEID 0x00000001
TUISPIDLL_OBJECT_PHONEID 0x00000002
TUISPIDLL_OBJECT_PROVIDERID 0x00000003
TUISPIDLL_OBJECT_DIALOGINSTANCE 0x00000004

Parameters

TUISPIDLL_OBJECT_LINEID
dwObjectID is a line device identifier (dwDeviceID).
TUISPIDLL_OBJECT_PHONEID
dwObjectID is a phone device identifier (dwDeviceID).
TUISPIDLL_OBJECT_PROVIDERID
dwObjectID is a permament provider identifier.
TUISPIDLL_OBJECT_DIALOGINSTANCE

dwObjectID is a dialog instance handle (either HDRVDIALOGINSTANCE or HTAPIDIALOGINSTANCE, depending on the context).
