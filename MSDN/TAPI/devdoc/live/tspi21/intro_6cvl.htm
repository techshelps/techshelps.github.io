What's New for TSPI Version 2.1

  • Part of the Telephony Application Programming Interface (TAPI) information set. TSPI documentation now appears in the same information set as TAPI.
  • Updated document organization. The documentation supporting TSPI is organized to parallel TAPI documentation.
  • Functions updated. Functions that are implemented in service providers for TAPI version 1.4 and earlier but were replaced by Telephony user interface SPIs (TUISPI) in version 2.0 have been annotated as obsolete. However, owing to their value for earlier TAPI versions, these functions remain in the information set as well as cross-reference to their replacements.
