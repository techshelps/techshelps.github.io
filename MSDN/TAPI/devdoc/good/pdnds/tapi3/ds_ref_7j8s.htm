IH26XRTPControl

[This is preliminary documentation and subject to change.]

The element bRTPHeader must be set to TRUE prior to StartStreaming. If this element is TRUE then dwPacketSize and dwPacketLoss can be set at any time, and will take effect on the next frame. The state of bRTPHeader cannot be changed between StartStreaming and StopStreaming.
