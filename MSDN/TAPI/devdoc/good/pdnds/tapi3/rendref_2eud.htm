Time

[This is preliminary documentation and subject to change.]

This component is a representation of the time attributes in an SDP component. There may be several lines with the "t" type. This component does not expose the corresponding "r" and "z" types.

It exports interfaces to get and set the time properties. It also provides interfaces to any generic attributes for the media.

Time Interfaces Description
ITTime Provides access to a set of activation times for the session.
