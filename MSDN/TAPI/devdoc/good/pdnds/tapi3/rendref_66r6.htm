Conference Announcement Enumerator

[This is preliminary documentation and subject to change.]

A conference directory allows enumeration of any subset of its conference announcements through a conference directory enumerator. While an enumerator instance is dependent upon a conference directory, it is a separate component.

Conference Announcement Enumerator Description
IEnumVARIANT Enables enumeration of a filtered subset of conference announcements in the conference directory with Next, Skip, Reset and Clone methods.
