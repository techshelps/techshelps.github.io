ISilenceSuppressor::GetAverageIncrementor

[This is preliminary documentation and subject to change.]

If sample buffer energy is greater than EchoThreshold plus running average energy for RunningAverageTime buffers, the running average is increased by AverageIncrementor. This methods retrieves a pointer to AverageIncrementor.

HRESULT GetAverageIncrementor(
  LPDWORD lpdwAverageIncrementor
);

Parameters

lpdwAverageIncrementor
[out] Incrementor for running average.

Return Values

E_POINTER
The pointer was not valid.
NOERROR
Success.
