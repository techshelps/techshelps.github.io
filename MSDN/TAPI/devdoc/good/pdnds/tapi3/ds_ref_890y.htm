RTP Receive Payload Handler

[This is preliminary documentation and subject to change.]

This filter is responsible for reordering and reassembling RTP packets into multimedia buffers. It also places a valid DirectShow timestamp on the buffers. See RTP Receive Payload Handler (RPH) Filter for more detailed information.
