RTP Send Payload Handler Filter

[This is preliminary documentation and subject to change.]

This filter is responsible for segmenting multimedia buffers into RTP packets as well as prepending appropriate RTP headers with RTP timestamps.

For more detailed information, please see RTP Send Payload Handler (SPH) Filter.
