ITBasicCallControl::SelectMediaTerminals

[This is preliminary documentation and subject to change.]

Select a set of ITMediaTerminals to use on the call. To unselect a terminal, use the UnselectMediaTerminal method. Multiple calls to SelectMediaTerminal will continue to select more media terminals — it will not reset the selection.

HRESULT SelectMediaTerminals(
  VARIANT MediaTerminals
);

Parameters

MediaTerminals
[in] Array of MediaTerminals to use on this call.
