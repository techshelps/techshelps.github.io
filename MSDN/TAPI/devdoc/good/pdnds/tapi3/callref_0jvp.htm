Call and Media Control Reference

[This is preliminary documentation and subject to change.]

The Call and Media Control reference material contains information on interfaces, methods, enumerator interfaces, structures, and types.

You may also wish to search the Platform SDK for information on DirectShow™ and the Component Object Model (COM). TAPI 3.0 uses DirectShow filters and filter managers to control the media stream following connection setup. COM is the basic coding model for TAPI 3.0, and familiarity with it is assumed throughout this document.
