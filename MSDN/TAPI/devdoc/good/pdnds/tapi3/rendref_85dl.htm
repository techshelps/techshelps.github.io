ITConnection::SetEncryptionKey

[This is preliminary documentation and subject to change.]

Sets the encryption key needed to decrypt the session or an indication to a mechanism to obtain a usable key by external means (BSTR). (Not implemented for Windows NT 5 beta 1.)

HRESULT SetEncryptionKey(
 BSTR pKeyType
 BSTR *ppKeydata
);

Parameters

pKeyType
[in] Encryption key type.
ppKeydata
[in] Encryption key information.
