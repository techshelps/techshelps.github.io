Conference Announcement

[This is preliminary documentation and subject to change.]

A conference announcement may be created by a user in the local network, or it may be added to the conference directory by an external SAP gateway for the WAN. (The SAP gateway is not planned for NT 5.0). The conference blob attribute allows for provider-specific extensions, as the provider is free to store its attributes in this uninterpreted attribute. An attribute in the conference announcement schema identifies the provider. The section on conference blob describes a mechanism for synchronizing any conference announcement attributes that are duplicated in the schema.

The conference blob interfaces are aggregated by this component.

Conference Announcement Interface Description
ITConference Retrieves and sets conference announcement data attributes.
ITNotification Notifies application of changes in conference characteristics.
