1,544,000 bits per second (bps). See also bits per second (bps).

One million cycles per second; a unit of frequency used to measure band and bandwidth. See also band, bandwidth.

Microsoft Messaging API (MAPI)
Set of API functions and an OLE interface that lets messaging clients, such as Microsoft Exchange, interact with various message service providers and various computer telephony servers running Windows NT Server.
multi-link PPP
A communications protocol that allows the combination of two or more B channels into a single, faster PPP connection. With multi-link PPP, you can have a 128-Kbps PPP connection over a Basic Rate ISDN line. See also Basic Rate Interface (BRI-ISDN), point-to-point protocol (PPP).
