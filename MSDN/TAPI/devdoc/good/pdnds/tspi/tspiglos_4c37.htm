kilobits per second (Kbps)
One thousand bits per second. Standard measure of data rate and transmission capacity.
kilohertz (kHz)
In data communications, kHz means a thousand bits per second. In computer memory terms, kHz means 1,024, which is the figure for two raised to the 10th power.
