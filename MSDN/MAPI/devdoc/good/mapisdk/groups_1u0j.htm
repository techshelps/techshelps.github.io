The personal information properties are used to provide supplementary identification and access information for a messaging user. They are all of property type PT_TSTRING and are all optionally exposed by messaging user objects. Their content is defined by the messaging user and the user's organization. For more information on the personal information properties, see IMailUser::IMAPIProp.
