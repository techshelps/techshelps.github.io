PR_MHS_COMMON_NAME

The PR_MHS_COMMON_NAME property contains the common name of a messaging user for use in a message header.

Quick Info

Header file: MAPITAGS.H
May be exposed by: Message objects
Identifier: 0x3A0F
Property type: PT_TSTRING
Property tag: 0x3A0F001E (0x3A0F001F for Unicode)
Corresponding X.400 attribute: MH_T_COMMON_NAME
