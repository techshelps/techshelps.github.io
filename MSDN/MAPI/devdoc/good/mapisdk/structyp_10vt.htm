CbNewSPropProblemArray

The CbNewSPropProblemArray macro computes the number of bytes to be allocated for a new SPropProblemArray structure containing a specified number of SPropProblem structures.

Quick Info

Header file: MAPIDEFS.H
Related structure: SPropProblemArray

CbNewSPropProblemArray (_cprob)

Parameters

_cprob
Count of SPropProblem structures to be included in the new SPropProblemArray structure.
