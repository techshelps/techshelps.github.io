Sample Form

This sample implements the MAPI form interfaces and a very simple message, containing only text and recipients. The form can be used by form-aware clients for sending and receiving messages of IPM.Note.Sample.Simple message class instead of standard send/read forms. It is a good basis for reuse in a more complicated form.

Quick Info

Executable: SMPFRM.EXE
Directory for source code: SIMPLE.FRM
Written in: C++
Supports: Win32 platforms only

Supported Features

  • Sending messages
  • Replying to and forwarding messages
  • Next, Previous
  • Saving and deleting messages

Unsupported Features

  • Attachments
  • Reply All, Post
  • Moving and copying messages
  • Printing
