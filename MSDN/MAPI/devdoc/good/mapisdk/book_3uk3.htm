Finding API Elements

There are several ways to find the reference topic for an API element:

  • In the Contents pane, open the appropriate Reference section and browse the alphabetic list.
  • In the Contents pane, open the Appendixes section. Open the Functionality Groups section, choose the type of element that interests you, and browse the alphabetic list.
  • On the toolbar, click the button. In the list box that appears, type the name of the programming element you want to see, or scroll through the list and select a keyword. As you type, the list automatically scrolls to the word that most closely matches the text you are typing.
