PR_AB_DEFAULT_PAB

The PR_AB_DEFAULT_PAB property contains the entry identifier of the address book container that is to be used as the personal address book.

Quick Info

Header file: MAPITAGS.H
Identifier: 0x3D07
Property type: PT_BINARY
Property tag: 0x3D070102

Remarks

Do not use this property. It is reserved for use by MAPI.
