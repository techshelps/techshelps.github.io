MAPIDBG.INI [TNEF] Section

The entry in this section controls debugging features in the MAPI TNEF facility.

ForceUncompressedRTF = 0|1
If this entry is set to 1, TNEF assumes that all RTF text is uncompressed and checks the stream explicitly. If it is 0, TNEF assumes the text is compressed. The default value is 0.
