PR_ORIGINAL_DISPLAY_CC

The PR_ORIGINAL_DISPLAY_CC property contains the display names of any carbon copy (CC) recipients of the original message.

Quick Info

Header file: MAPITAGS.H
Must be exposed by: Report message objects
Identifier: 0x0073
Property type: PT_TSTRING
Property tag: 0x0073001E (0x0073001F for Unicode)

Remarks

The PR_ORIGINAL_DISPLAY_CC property contains an ASCII list separated by semicolons. It is furnished by MAPI and is copied directly from PR_DISPLAY_CC when a delivery or nondelivery report or a read or nonread report is generated. PR_ORIGINAL_DISPLAY_CC may be present on other messages as defined by their message classes.
