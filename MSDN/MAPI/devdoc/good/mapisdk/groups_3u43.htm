Represented Sender Properties

The following are the address properties for a messaging user being represented by the sender:

PR_SENT_REPRESENTING_ADDRTYPE
PR_SENT_REPRESENTING_EMAIL_ADDRESS
PR_SENT_REPRESENTING_ENTRYID
PR_SENT_REPRESENTING_NAME
PR_SENT_REPRESENTING_SEARCH_KEY

For more information about how these properties are used, see Delegate Access.

For more information on the address properties, see Base Address Properties.
