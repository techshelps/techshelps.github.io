SLongArray

The SLongArray structure contains an array of values used to describe a property of type PT_MV_LONG.

Quick Info

Header file: MAPIDEFS.H 

typedef struct _SLongArray 
{ 
     ULONG     cValues; 
     LONG      FAR *lpl; 
} SLongArray;

Members

cValues
Count of values in the array pointed to by the lpl member.
lpl
Pointer to an array of LONG values.

Remarks

For more information on PT_MV_LONG, see List of Property Types.
