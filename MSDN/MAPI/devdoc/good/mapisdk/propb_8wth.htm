PR_WIZARD_NO_PST_PAGE

The PR_WIZARD_NO_PST_PAGE property contains TRUE if the Profile Wizard is to suppress the PST page.

Quick Info

Header file: MAPIWZ.H
May be exposed by: Provider objects
Identifier: 0x6700
Property type: PT_BOOLEAN
Property tag: 0x6700000B

Remarks

Service providers can set this property when calling a function based on the LAUNCHWIZARDENTRY function prototype. PR_WIZARD_NO_PST_PAGE tells the Profile Wizard that the provider does not want the personal message store (PST) page to be displayed during the user dialog.
