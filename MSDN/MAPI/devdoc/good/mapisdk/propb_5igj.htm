PR_SERVICES

The PR_SERVICES property contains a list of identifiers of message services in the current profile.

Quick Info

Header file: MAPITAGS.H
Identifier: 0x3D0E
Property type: PT_BINARY
Property tag: 0x3D0E0102

Remarks

Do not use this property. It is reserved for use by MAPI.
