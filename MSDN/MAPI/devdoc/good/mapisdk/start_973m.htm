Message Store Viewer

This tool displays message, folder, and message store properties in raw form, and exercises nearly all methods on message store provider interfaces. It is useful both for exercising message store providers and for understanding what properties are present on objects in a store.

Quick Info

Executable: MDBVU.EXE
Directory for source code: MDBVIEW.CLI*
Written in: C++ using Microsoft Foundation Classes (MFC)
Supports: All platforms

*This directory is located at ftp://ftp.microsoft.com/developr/mapi/samples/mfcapps/mfcapps.zip.
