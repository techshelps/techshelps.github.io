SDateTimeArray

The SDateTimeArray structure contains an array of time values used to describe a property of type PT_MV_SYSTIME.

Quick Info

Header file: MAPIDEFS.H 

typedef struct _SDateTimeArray 
{ 
     ULONG         cValues; 
     FILETIME FAR *lpft; 
} SDateTimeArray;

Members

cValues
Count of values in the array pointed to by the lpft member.
lpft
Pointer to an array of FILETIME structures containing the time values.

Remarks

For more information on PT_MV_SYSTIME, see List of Property Types.
