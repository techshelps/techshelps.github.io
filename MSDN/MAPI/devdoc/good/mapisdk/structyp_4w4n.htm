PROP_TAG

The PROP_TAG macro returns a property tag created by combining a specified property type and identifier.

Quick Info

Header file: MAPIDEFS.H
Related structure: SPropValue

PROP_TAG (ulPropType, ulPropID)

Parameters

ulPropType
Property type for the new property tag.
ulPropID
Property identifier for the new property tag.

Remarks

The PROP_TAG macro creates a property tag for a property of type ulPropType and the identifier that is specified in ulPropID. For example, a property tag for an entry identifier can be created with the PROP_TAG macro as follows:

PROP_TAG( PT_BINARY, 0x0FFF)

The low-order 16 bits of the returned property tag contain the property type, PT_BINARY, and the high-order 16 bits contain the property identifier, 0xFFFF.

For more information about property tags, see About Property Tags.
