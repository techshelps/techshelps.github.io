PR_CONTENT_IDENTIFIER

The PR_CONTENT_IDENTIFIER property contains a key value that enables the message recipient to identify its content.

Quick Info

Header file: MAPITAGS.H
May be exposed by: Message objects
Identifier: 0x0008
Property type: PT_TSTRING
Property tag: 0x0008001E (0x0008001F for Unicode)
Corresponding X.400 attribute: MH_T_CONTENT_IDENTIFIER
