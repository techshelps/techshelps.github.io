ScLocalPathFromUNC

The ScLocalPathFromUNC function locates a local path counterpart to the given universal naming convention (UNC) path.

Note This function may not be supported in future versions of MAPI.

Quick Info

Header file: MAPIUTIL.H
Implemented by: MAPI
Called by: Client applications and service providers 

SCODE ScLocalPathFromUNC(
  LPSTR szUNC,     
  LPSTR szLocal,   
  UINT cchLocal    
);

Parameters

szUNC
[in] A path in the format \\[server]\[share]\[path] of a file or directory.
szLocal
[out] A path in the format [drive:]\[path] of the same file or directory as for the szUNC parameter.
cchLocal
[in] Size of the buffer for the output string.
