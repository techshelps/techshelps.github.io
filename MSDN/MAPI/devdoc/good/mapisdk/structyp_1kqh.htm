SLPSTRArray

The SLPSTRArray structure contains an array of string values used to described a property of type PT_MV_STRING8.

Quick Info

Header file: MAPIDEFS.H 

typedef struct _SLPSTRArray 
{ 
     ULONG      cValues; 
     LPSTR FAR *lppszA;
} SLPSTRArray;

Members

cValues
Count of values in the array pointed to by the lppszA member.
lppszA
Pointer to an array of null-terminated 8-bit character strings.

Remarks

For more information on PT_MV_STRING8, see List of Property Types.
