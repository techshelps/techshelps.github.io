CMC_boolean

CMC_boolean is a CMC unsigned integer data type that is a Boolean value.

Quick Info

Header file: XCMC.H 

typedef      CMC_uint16      CMC_boolean;

Remarks

The CMC_boolean data type can contain the symbolic constants CMC_TRUE and CMC_FALSE. The C interface denotes FALSE by using 0 and TRUE by using any other integer. CMC denotes CMC_FALSE by using 0, in line with C interface usage. However, CMC_TRUE is denoted specifically by the integer 1.
