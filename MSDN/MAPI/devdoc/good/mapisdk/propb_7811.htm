PR_POSTAL_CODE

The PR_POSTAL_CODE property contains the postal code for the recipient's postal address.

Quick Info

Header file: MAPITAGS.H
May be exposed by: Messaging user objects
Identifier: 0x3A2A
Property type: PT_TSTRING
Property tag: 0x3A2A001E (0x3A2A001F for Unicode)

Remarks

The PR_POSTAL_CODE property is one of the properties that provide identification and access information for a recipient. These properties are defined by the recipient and the recipient's organization. For more information on this group of properties, see IMailUser::IMAPIProp.

The postal code is specific to the recipient's country. In the United States of America, this property contains the ZIP code.
