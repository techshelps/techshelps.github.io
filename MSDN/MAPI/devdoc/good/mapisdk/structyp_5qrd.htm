SMAPIFormPropArray

The SMAPIFormPropArray structure contains an array of SMAPIFormProp structures.

Quick Info

Header file: MAPIFORM.H
Related macro: CbMAPIFormPropArray 

typedef struct 
{ 
     ULONG         cProps; 
     ULONG         ulPad; 
     SMAPIFormProp aFormProp[MAPI_DIM]; 
} SMAPIFormPropArray, FAR * LPMAPIFORMPROPARRAY;

Members

cProps
Count of named properties in the array in the aFormProp member.
ulPad
Eight bytes of padding used to ensure proper alignment.
aFormProp
Array of form properties.

Remarks

The SMAPIFormPropArray structure is passed as a parameter to the following methods:

IMAPIFormInfo::CalcFormPropSet

IMAPIFormMgr::CalcFormPropSet

IMAPIFormContainer::CalcFormPropSet
