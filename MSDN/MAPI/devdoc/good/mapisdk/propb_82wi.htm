PR_ORGANIZATIONAL_ID_NUMBER

The PR_ORGANIZATIONAL_ID_NUMBER property contains an identifier for the recipient used within the recipient's organization.

Quick Info

Header file: MAPITAGS.H
May be exposed by: Messaging user objects
Identifier: 0x3A10
Property type: PT_TSTRING
Property tag: 0x3A10001E (0x3A10001F for Unicode)

Remarks

The PR_ORGANIZATIONAL_ID_NUMBER property is one of the properties that provide identification and access information for a recipient. These properties are defined by the recipient and the recipient's organization. For more information on this group of properties, see IMailUser::IMAPIProp.

PR_ORGANIZATIONAL_ID_NUMBER is commonly set to an employee number.
