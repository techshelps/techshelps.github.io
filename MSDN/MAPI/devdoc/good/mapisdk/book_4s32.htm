Finding Other Documentation

Articles, information about Internet resources for MAPI developers, and instructions for installing and using MAPI samples and tools can be found on the Microsoft For Developers Only Web site. Go to the Win32 Development topic from the home page. At the bottom of the Win32 home page, click Messaging (MAPI).

For more information about OLE programming, see Inside OLE, Second Edition, by Kraig Brockschmidt (Redmond, WA: Microsoft Press, 1995) or the COM and ActiveX Object Services section in the Platform SDK.

For more information about programming with Microsoft® Visual Basic® programming system, see your Visual Basic documentation.
