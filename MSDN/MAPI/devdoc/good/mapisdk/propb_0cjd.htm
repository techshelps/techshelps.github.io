PR_PHYSICAL_DELIVERY_BUREAU_FAX_DELIVERY

The PR_PHYSICAL_DELIVERY_BUREAU_FAX_DELIVERY property contains TRUE if the messaging system should use a fax bureau for physical delivery of this message.
