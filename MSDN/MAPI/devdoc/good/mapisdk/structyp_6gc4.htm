CbNewADRLIST

The CbNewADRLIST macro computes the number of bytes that should be allocated for a new ADRLIST structure containing a specified number of recipients represented by ADRENTRY structures.

Quick Info

Header file: MAPIDEFS.H
Related structure: ADRLIST

CbNewADRLIST (_centries)

Parameters

_centries
Count of ADRENTRY structures to be included in the new ADRLIST structure.
