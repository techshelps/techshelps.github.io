PR_SERVICE_UID

The PR_SERVICE_UID property contains the MAPIUID structure for a message service.

Quick Info

Header file: MAPITAGS.H
Must be exposed by: Profile section objects
Must be a column in: Message service tables
Identifier: 0x3D0C
Property type: PT_BINARY
Property tag: 0x3D0C0102

Remarks

The PR_SERVICE_UID property is computed by MAPI on profile section objects. MAPI uses this property to group all the providers that belong to the same message service. PR_SERVICE_UID is supplied as a parameter to most of the IMsgServiceAdmin methods. It must not appear in MAPISVC.INF.
