MAPI File Formats

MAPI publishes the formats of various configuration files that clients and service providers use to describe message services and their service providers, forms, and options for the MAPI debugging utility.

The following table describes these files by name or type:

Configuration file Description
MAPISVC.INF Contains configuration information for the MAPI subsystem, message services, and service providers.
Files with the .CFG extension Contains configuration information for a form.
MAPIDBG.INI Contains configuration information for the MAPIDBG utility for testing and debugging.

For detailed information about the file format of MAPISVC.INF, see File Format of MAPISVC.INF.

For detailed information about the file format of a form configuration file, see File Format of Form Configuration Files.

For detailed information about the file format of MAPIDBG.INI, see File Format of MAPIDBG.INI.
