PR_STORE_PROVIDERS

The PR_STORE_PROVIDERS property contains a list of identifiers of message store providers in the current profile.

Quick Info

Header file: MAPITAGS.H
Identifier: 0x3D00
Property type: PT_BINARY
Property tag: 0x3D000102

Remarks

Do not use this property. It is reserved for use by MAPI.
