PR_TEMPLATEID

The PR_TEMPLATEID property contains an entry identifier that can find the code associated with a service provider.

Quick Info

Header file: MAPITAGS.H
May be exposed by: Address book provider objects
Identifier: 0x3902
Property type: PT_BINARY
Property tag: 0x39020102

Remarks

The entry identifier can bind the code to the data.

See Also

IABLogon::OpenTemplateID, IMAPISupport::OpenTemplateID
