CbNewSPropTagArray

The CbNewSPropTagArray macro computes the number of bytes to be allocated for a new SPropTagArray structure containing a specified number of property tags.

Quick Info

Header file: MAPIDEFS.H
Related structure: SPropTagArray

CbNewSPropTagArray (_ctag)

Parameters

_ctag
Count of property tags to be included in the new SPropTagArray structure.
