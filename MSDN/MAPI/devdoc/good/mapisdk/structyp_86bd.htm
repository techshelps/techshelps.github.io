SBinary

The SBinary structure describes a property of type PT_BINARY.

Quick Info

Header file: MAPIDEFS.H 

typedef struct _SBinary 
{ 
     ULONG      cb; 
     LPBYTE     lpb; 
} SBinary, FAR *LPSBinary;

Members

cb
Count of bytes in the lpb member.
lpb
Pointer to the PT_BINARY property value.
