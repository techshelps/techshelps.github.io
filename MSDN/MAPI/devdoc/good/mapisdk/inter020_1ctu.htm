IMAPIMessageSite::GetFolder

The IMAPIMessageSite::GetFolder method returns the folder in which the current message was created or opened, if such a folder exists. This method returns NULL in the ppFolder parameter for embedded messages, which are not stored directly in a folder.

Quick Info

See IMAPIMessageSite : IUnknown.

HRESULT GetFolder(
  LPMAPIFOLDER FAR * ppFolder  
);

Parameters

ppFolder
[out] Pointer to a pointer to the returned folder.

Return Values

S_OK
The call succeeded and has returned the expected value or values.
S_FALSE
No folder exists for the message.
