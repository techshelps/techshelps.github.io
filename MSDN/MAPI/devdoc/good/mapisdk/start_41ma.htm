Status Table Viewer

This tool displays the MAPI status table in raw form and exercises provider IMAPIStatus interfaces.

Quick Info

Executable: STATVU.DLL
Directory for source code: STATUSVU.CLI*
Written in: C++ using Microsoft Foundation Classes (MFC)
Supports: All platforms

*This directory is located at ftp://ftp.microsoft.com/developr/mapi/samples/mfcapps/mfcapps.zip.
