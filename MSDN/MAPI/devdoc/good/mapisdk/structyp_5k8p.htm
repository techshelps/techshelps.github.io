SBinaryArray

The SBinaryArray structure contains an array of binary values.

Quick Info

Header file: MAPIDEFS.H 

typedef struct _SBinaryArray 
{ 
     ULONG        cValues; 
     SBinary      FAR *lpbin; 
} SBinaryArray;

Members

cValues
Count of values in the array pointed to by the lpbin member.
lpbin
Pointer to an array of SBinary structures holding the binary values.

Remarks

The SBinaryArray structure is used to describe properties of type PT_MV_BINARY.

For more information on PT_MV_BINARY, see List of Property Types.
