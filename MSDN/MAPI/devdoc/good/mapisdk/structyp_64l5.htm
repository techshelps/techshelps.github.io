CbMAPIFormPropArray

The CbMAPIFormPropArray macro computes the number of bytes in an existing SMAPIFormPropArray structure containing a specified number of SMAPIFormProp structures.

Quick Info

Header file: MAPIFORM.H
Related structure: SMAPIFormPropArray

CbMAPIFormPropArray (_c)

Parameters

_c
Count of SMAPIFormProp structures in the SMAPIFormPropArray structure in question.
