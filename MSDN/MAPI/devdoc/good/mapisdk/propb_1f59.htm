PR_SUBJECT_IPM

The PR_SUBJECT_IPM property contains a binary value that is copied from the message for which a report is being generated.

Quick Info

Header file: MAPITAGS.H
May be exposed by: Report message objects
Identifier: 0x0038
Property type: PT_BINARY
Property tag: 0x00380102
Corresponding X.400 attribute: IM_SUBJECT

Remarks

The PR_SUBJECT_IPM property, like the PR_REPORT_TAG property, can be used to correlate a report with the original message. The PR_PARENT_KEY property can also be used for correlation.
