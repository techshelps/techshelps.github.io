SWStringArray

The SWStringArray structure contains an array of character strings used to describe a property of type PT_MV_UNICODE.

Quick Info

Header file: MAPIDEFS.H 

typedef struct _SWStringArray 
{ 
     ULONG           cValues; 
     LPWSTR     FAR *lppszW; 
} SWStringArray;

Members

cValues
Count of strings in the array pointed to by the lppszW member.
lppszW
Pointer to an array of null-terminated Unicode character strings.

Remarks

For more information on PT_MV_UNICODE, see List of Property Types.
