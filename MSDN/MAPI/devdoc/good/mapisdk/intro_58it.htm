Using UUIDGEN.EXE

UUIDGEN.EXE is a Microsoft utility that generates unique, byte-order independent identifiers. This utility has several uses in the MAPI environment. The following table indicates some of the ways that MAPI components can use this utility.

MAPI component Use of UUIDGEN.EXE
Service provider Creating new hardcoded MAPIUIDs.
Form Creating new class identifiers for forms.
Client application or service provider Identifying hardcoded global profile section.
Client application or service provider Creating new property sets for named properties.
