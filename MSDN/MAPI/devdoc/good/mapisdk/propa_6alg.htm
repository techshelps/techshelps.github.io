PR_CONTENT_CONFIDENTIALITY_ALGORITHM_ID

The PR_CONTENT_CONFIDENTIALITY_ALGORITHM_ID property contains an identifier for the algorithm used to confirm message content confidentiality.

Quick Info

Header file: MAPITAGS.H
May be exposed by: Message objects
Identifier: 0x0006
Property type: PT_BINARY
Property tag: 0x00060102
Corresponding X.400 attribute: MH_T_ALGORITHM_ID or MH_T_CONFIDENTIALITY_ALGORITHM
