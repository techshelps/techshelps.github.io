PR_USER_X509_CERTIFICATE

The PR_USER_X509_CERTIFICATE property contains X.509 version 3 security certificates for a messaging user.

Quick Info

Header file: MAPITAGS.H
May be exposed by: Messaging user objects
Identifier: 0x3A70
Property type: PT_MV_BINARY
Property tag: 0x3A701102

Remarks

The PR_USER_X509_CERTIFICATE property is used by applications that utilize public-key security. It holds a binary representation of one or more X.509 version 3 security certificates.

Various applications and clients can use this property for their own security certificates. The binary format of the X.509 data can vary among vendors.
