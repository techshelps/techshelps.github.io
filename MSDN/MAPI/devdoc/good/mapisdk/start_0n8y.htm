Sample Messaging Hook Provider

This sample processes inbound and outbound messages. It can archive sent and deleted messages into folders by month and year, place incoming messages into folders based on variable criteria, and automatically respond to incoming messages in Rich Text Format.

Quick Info

Executable: SMH.DLL
Directory for source code: MANAGER.SH
Written in: C
Supports: Win32 platforms only

Supported Features

  • Filtering of inbound messages
  • Archiving sent mail
  • Archiving deleted mail
