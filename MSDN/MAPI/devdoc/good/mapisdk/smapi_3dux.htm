Addressing Messages with Simple MAPI

Message headers form the "envelope" of a message. The message header contains all the information about the recipients of the message, as well as a subject for the message.

Addressing messages involves three major tasks:

  • Generating recipient lists for different recipient classes. There should be separate lists of addresses for primary recipients, carbon copy (CC) recipients, and blind carbon copy (BCC) recipients. See Generating Recipient Lists with Simple MAPI.
  • Checking names that the user enters against the address book to resolve them into actual addresses. See Checking Names with Simple MAPI.
  • Getting details about address book entries to help the user choose between ambiguous entries. See Getting Details About a User with Simple MAPI.

Creating a subject line is a simple matter of generating a text string containing the subject.
