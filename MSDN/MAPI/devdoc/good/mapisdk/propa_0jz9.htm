PR_ATTACH_MIME_SEQUENCE

The PR_MIME_SEQUENCE property contains the MIME sequence number of a MIME message attachment.

Quick Info

Header file: MAPITAGS.H
May be exposed by: Attachment subobjects
Identifier: 0x3710
Property type: PT_LONG
Property tag: 0x37100003

Remarks

The PR_MIME_SEQUENCE property is used for MHTML support. It represents the sequence number of the attachment within the parent MIME multipart body part of the MIME message.
