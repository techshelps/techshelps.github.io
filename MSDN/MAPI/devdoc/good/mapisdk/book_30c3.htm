Finding Information About Objects

Information about objects is documented in various sections. Because nearly all MAPI objects expose a single MAPI interface, the approach is to treat objects and the interfaces that they expose as synonymous.

You can find information about objects and interfaces in the following sections:

  • MAPI Interfaces in the Reference. For example, locate information about implementing and using a folder under its standard interface, IMAPIFolder.
  • Objects and Interfaces in the Guide. This section introduces the Component Object Model as it relates to MAPI, describes MAPI's object hierarchy and containment model, maps objects to interfaces, and describes implementation responsibilities.
  • Properties in the Guide. This section describes how objects manage properties and how clients work with them.
  • Tables in the Guide. This section describes in detail how to implement and use a table object and discusses the composition and purpose of the different types of tables.
  • Forms in the Guide. This section describes the various objects used in the implementation and use of forms.
  • MAPI Programming Basics in the Guide. This section provides information about the following types of objects:

    Folders

    Messages

    Attachments

    Recipients

    Status objects

    Profile and message service objects

  • Developing a Client Application in the Guide. This section describes how to use the objects to implement a MAPI client application.
  • Service Provider Basics in the Guide. This section describes how to use a support object and a progress object and implement a status object and display table.
  • Developing an Address Book Provider in the Guide. This section describes how to implement the address book logon and provider objects, address book containers, messaging users, distribution lists, and related tables.
  • Developing a Transport Provider in the Guide. This section describes how to implement the transport logon and provider objects, status object, TNEF object, and folders.
  • Developing a Message Store Provider in the Guide. This section describes how to implement the message store logon and provider objects, message stores, folders, messages, attachments, recipients, and related tables.
  • Developing MAPI Form Servers in the Guide. This section describes how to implement form objects.
