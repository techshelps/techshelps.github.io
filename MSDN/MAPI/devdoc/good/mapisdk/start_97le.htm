Installing the Address Book Viewer

To install the Address Book Viewer, place all of the following executable files either in the same directory or in one or more directories in your path. Several of these files are shared with the Message Store Viewer, another MAPI tool. If you are using a Win16 platform, you will need:

  • ABVIEW.EXE
  • PROPVU.DLL
  • TBLVU.DLL
  • XVPORT.DLL

If you are using a Win32 platform, you will need:

  • ABVIEW32.EXE
  • PROPVU32.DLL
  • TBLVU32.DLL
  • XVPORT.DLL

The DLL files support the display of specialized viewers. PROPVU and PROPVU32 are Property Viewers; they display the properties of address book objects and allow you to make calls to IMAPIProp methods. The Property Viewer appears as follows when used to show the properties of a messaging user:

XVPORT.DLL is used by the Address Book Viewer to maintain a session history by displaying log data and saving that data to one or more files.
