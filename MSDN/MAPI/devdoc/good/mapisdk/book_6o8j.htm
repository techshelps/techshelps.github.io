Finding Lists of Required Properties

Lists of required and optional properties are documented in several places:

  • See the following interfaces for lists of properties associated with objects that support these interfaces:

    IABContainer : IMAPIContainer

    IAttach : IMAPIProp

    IDistList : IMAPIContainer

    IMailUser : IMAPIProp

    IMAPIContainer : IMAPIProp

    IMAPIStatus : IMAPIProp

    IMessage : IMAPIProp

    IMsgStore : IMAPIProp

    IProfSect : IMAPIProp

  • For lists of properties required on tables, see the documentation for each type of table in Tables.
  • For lists of required and optional message properties for message classes defined by MAPI, see:

    Required Properties for All Messages

    Required Report Message Properties

    Optional Report Message Properties

    Recipient Properties for All Messages

    Recipient Properties for Delivery Status Reports

  • For lists of properties related by functionality, browse the subtopics in Property Groups.
