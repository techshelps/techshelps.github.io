Sample Remote Transport Provider

This sample uses named pipes to communicate with a server process running on Windows NT. It uses TNEF exclusively to encode message content. It supports all configuration options and the remote transport interfaces.

Quick Info

Executable: XPWDSR.DLL
Directory for source code: REMOTE.XP
REMOTE.SRV (sample server)
Written in: C++
Supports: Win32 platforms only
Windows NT only (sample server)

Supported Features

  • Slow-link connection to a remote host
  • Deferred delivery
  • Forced immediate connection and submission
  • Server-based messaging
  • TNEF to encapsulate all the message properties
  • IMAPIStatus interface for client access to transport status
  • Thread safety
  • Configuration dialog boxes
  • Generation of delivery and nondelivery reports
  • Profile Wizard for interactive profile configuration
  • WINDS address type
  • MAPI spooler marshalled remote connections
