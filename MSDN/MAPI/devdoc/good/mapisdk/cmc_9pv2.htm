Getting Information About the CMC Implementation

A client can get information about the CMC implementation by calling the cmc_query_configuration function. You must pass in an item code that specifies the type of configuration information for the cmc_query_configuration function to return. You must also pass in a pointer to a buffer where the cmc_query_configuration function can pass back the requested information. It is your responsibility to ensure that the buffer is large enough to hold data of the requested type.
