CbMAPIVerbArray

The CbMAPIVerbArray macro computes the number of bytes in an existing SMAPIVerbArray structure containing a specified number of SMAPIVerb structures.

Quick Info

Header file: MAPIFORM.H
Related structure: SMAPIVerbArray

int CbMAPIVerbArray (_c)

Parameters

_c
Count of SMAPIVerb structures in the existing SMAPIVerbArray structure.
