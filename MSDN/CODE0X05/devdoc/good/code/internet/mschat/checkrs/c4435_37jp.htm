OVERVIEW OF THE CHECKRS SAMPLE 
============================== 
This is a Visual Basic Application which enables you to play Checkers  
game with the people you are chatting with. 
 
After building the sample, you can enter a chat session by clicking on the Enter  
Room button. You will need to enter a chat server, room name and alias name in the  
Enter Room dialog box.  
 
BUILDING THE CHECKRS SAMPLE 
=========================== 
 
The checkrs sample can be built with Visual Basic 4.0.  
 
1. Select the Open project command from the File menu. 
 
2. Open the checker.vbp file and build the application.
