Edit User Attributes

See Also

To edit user attributes:

  1. From the VSS Administrator user list, select a user.

  2. On the Users menu, click Edit User to display the Edit User dialog box.

  3. Select the Read only check box to give the user read-only rights. (By default, VSS gives all new users read/write rights.)

    – Or –

    Rename the user by typing a new name.

  4. Click OK. VSS changes the selected user's rights and/or logon name.
