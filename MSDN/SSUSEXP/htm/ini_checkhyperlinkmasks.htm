Check_Hyperlink_Masks Initialization Variable

Controls the file types in which hyperlinks are checked.

Syntax

Check_Hyperlink_Masks = *.HTM, *.HTML, *.ASP

Example

Specifies that when Web hyperlinks are checked, the links are checked in files with the *.htm, *.html, and *.asp file extension:

Check_Hyperlink_Masks = *.HTM, *.HTML, *.ASP
