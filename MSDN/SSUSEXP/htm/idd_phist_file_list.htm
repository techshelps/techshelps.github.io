Contents of <Project Name> Dialog Box

Displays the contents of the selected VSS project

To display the Contents of <Project Name> dialog box:

  1. Select a project in VSS Explorer.

  2. On the Tools menu, click Show History.

  3. Click OK in the History Options dialog box.

  4. Select a file from the list in the History of Project dialog box.

  5. Click View.

Dialog Box Options

Date/Time stamp

Lists the timestamp of the latest version of the file in VSS.

Files

Lists the files in the project along with their current version numbers.
