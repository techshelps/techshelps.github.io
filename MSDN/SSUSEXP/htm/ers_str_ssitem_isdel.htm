<Item> has been deleted

See Also

This error has the following cause and solution:

  • You are trying to act on a file or project that has been deleted in the current project.

    Before you can do anything to the file, you must recover it using the Recover command.

    If you want to delete the file from the VSS database permanently, on the File menu, click Properties, and then on the Deleted Items tab, click Purge.
