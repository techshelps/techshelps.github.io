This label <label> is already used. Remove the old label?

A VSS label is used to identify a particular version of a file or project. This error has the following cause and solution:

  • You are attempting to assign a label to a file or project, and another version of that file or project has the same label.

    Click Yes to apply the new label and delete the old label. Click No to leave the old label unchanged. Click Cancel to cancel the labeling operation.
