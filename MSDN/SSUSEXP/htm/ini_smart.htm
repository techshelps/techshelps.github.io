Smart_Mode Initialization Variable

See Also

Enables and disables the VSS SmartMode feature.

Syntax

Smart_Mode = {Yes|No}

Example

Disables the VSS SmartMode feature:

Smart_Mode = No

Remarks

Smart_Mode tells VSS whether or not to use its SmartMode features. The SmartMode features are:

  • When you create or recover a project, VSS sets the new project as the current project.

  • When you delete or destroy a file, VSS deletes the corresponding local copy.

  • When you rename a file, VSS renames the corresponding local copy.

You can also set this variable in your SS.INI file by using the Remove Local Files After Delete option on the Local Files tab in the SourceSafe Options dialog box.
