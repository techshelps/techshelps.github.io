The labels.ini file could not be properly updated.  Have your VSS Administrator run the Analyze utility to recreate it.

This error has the following cause and solution:

  • While trying to keep the labels.ini file up to date, VSS was unable to write to the file. Any attempts to perform a get by label will not see the label until your VSS Admin runs Analyze.

    Ask your VSS Administrator to run the Analyze utility.
