A VB project by this name already exists in the source code control project <project>. Would you like to reconnect this VB project to the existing source code control project?

This error has the following cause and solution:

  • You tried to add a project, but a project with that name was already under source control.

Choose Yes to reconnect to the existing project, or choose No if the project in source control is a different project by the same name. Then, choose a new name for the new project.
