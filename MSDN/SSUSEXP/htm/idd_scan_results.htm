Find In Files Results Dialog Box

This dialog box shows the results of the Find in Files command.

Dialog Box Options

File

Shows the name of the file(s) containing matching strings.

Matches

Shows the number of matches found in the file.

Matches

Lists the matches found, in context.

Close

Closes the Find in Files results dialog box.

Report

Allows you to print a report of the match information to the Clipboard, a printer, or to a file.

Links

Shows any links the file has to other projects.
