Illegal version number

This error has the following cause and solution:

  • An illegal version number was found in the VSS file.

    Run the Analyze utility to make sure there are no errors in your VSS database. If running Analyze with the –F switch does not correct the problem, contact Product Support.

Note   When running the Analyze utility, first run it with no switches to find any errors in the database.  Then run the Analyze utility with just the –F switch in autofix mode.
