Create Web Site Projects

See Also

VSS includes several features that make managing World Wide Web sites easier. For example, you can designate a specific VSS project as a Web site project, and then quickly create a site map of the HTML files in the project (and subprojects), check hyperlinks in the project files, and deploy the files to an Internet server location when they are ready. To be able to use any of these features, you need to first designate the project as a Web project.

To create a Web site project:

  1. On the VSS Administrator Tools menu, click Options.

  2. Click the Web Projects tab.

  3. Click Browse to search for the project you want to designate as a Web project, then click OK to quit the Set Web Site At dialog box.

  4. Enter a Web address in the URL box.

  5. Click OK to quit the SourceSafe Options dialog box.

  6. Quit the Administrator and have all your clients quit and restart VSS Explorer.

You should now see a special icon () on the project in VSS Explorer that you've designated a Web project.

Note   You must set a proxy if you plan to deploy the files using FTP. See the Web tab in the Options dialog box on the Tools menu.
