Search for Checked Out Files

See Also

Note   You stay in search mode until you explicitly cancel the search.

To search by check out status:

  1. On the View menu, click Search Status Search to display the Search for Status dialog box.

  2. To display files checked out to a particular user, click Display Files Checked Out To, then select a user from the dropdown list.

  3. For a global search, click Search in All Projects.

  4. Click OK to start the search.
