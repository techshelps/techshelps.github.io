Journal_File Initialization Variable

Sets a file name where all VSS commands that change stored data are logged. Used in the SRCSAFE.INI file only.

Syntax

Journal_File = filename

Example

Sets SSLog.txt as a journal of all VSS commands:

Journal_File = O:\Docs\SSLog.txt

Remarks

This variable is generally set in SRCSAFE.INI, rather than in SS.INI, so that it applies to all users. If this variable is set, the file becomes a journal of VSS actions. The journal is a text file that is updated automatically by VSS to show the various actions that users take. Only actions that change data stored in the database are logged.

To set this variable in the SRCSAFE.INI file, on the Tools menu, click Options, then select Log All Actions In Journal File from the General tab.
