Invalid time or date string.

VSS cannot use the date or time syntax entered. This error has the following causes and solutions:

  • The date you entered is invalid.

    Proper date syntax is Month/Day/Year, with each number separated by a slash: for example, 2/29/64 or 11/5/1992.

  • The time you entered is invalid.

    Proper time syntax is Hour:Minute, followed by a one-letter a.m. or p.m. indicator: for example, 9:30A or 5:00p. (The indicator is not case-sensitive.) Or, you can use a 24-hour syntax such as 13:45.
