Rename Command (File Menu)

See Also

Renames a file or project.

Access Rights

You must have the Add access right to use this command.

Remarks

To use the Rename command, select the file or project you want to rename, then choose the command. You will be able to edit the item name directly in the VSS Explorer window.

When you use the Rename command on a file or project, the item's file history is retained. If multiple projects share a file and the file is renamed, it is renamed in all projects that share it. However, if any share links have been broken by the Branch command, the branched files are not renamed. The Paths tab in the Properties dialog box shows how the name has changed in different branches of the file.
