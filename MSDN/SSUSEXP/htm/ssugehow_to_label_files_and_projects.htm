How to Label Files and Projects

See Also

Note   When you label a project with a descriptive text string, all the files in that project and subproject inherit the label.

To label the current version of a file or project:

  1. Select the file or project you want in VSS Explorer.

  2. On the File menu, click Label to display the Label dialog box.

  3. In the Label box, type a string of up to 31 characters; optionally, in the Comment box, type a comment.

  4. Click OK.

To label an older version of a file or project:

  1. Select the file or project you want in VSS Explorer.

  2. On the Tools menu, click Show History to display the History dialog box.

  3. Click Details to display the History Details dialog box.

  4. In the Label box, type a user-defined label; optionally, in the Comment box, type a comment.

  5. Click Close to exit the History Details dialog box.

  6. Click Close to exit the History dialog box.
