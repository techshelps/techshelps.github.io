Move (Command Line)

See Also

Relocates a subproject from one parent project to another.

Syntax

ss Move <project to move> <new project path> [-H] [-I-] [-Y] [-?]

Examples

Moves the project $/WORD from $/JOE to $/APPS, creating $/APPS/WORD:

ss Move $/JOE/WORD $/APPS

The following table describes the command-line options available with this command.

Option Description
-I- Ignore: Do not ask for input under any circumstances.
-Y Specify a username and/or password.
-?, -H Request online Help on a command.

Access Rights

You must have the Add access right in the parent project being moved to, and the Destroy access right in the parent project being moved from, to use this command.
