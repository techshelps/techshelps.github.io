Add a User

See Also

To add a user to a VSS database:

  1. Make sure the correct database is selected in VSS Administrator. For more information, see Open a VSS Database.

  2. On the VSS Administrator Users menu, Click Add User to display the Add User dialog box.

  3. In the User name box, type a name. You can also type a password in the Password box.

    Note   User names can be up to 31 characters long, cannot begin or end with a space, and cannot include any punctuation. Passwords can be up to 15 characters long, and can contain any characters.

  4. To give the new user read-only rights, select the Read only check box.

    Note   If you do not select this check box, the new user has read/write rights. See Security Access Rights for more information.

  5. Click OK.
