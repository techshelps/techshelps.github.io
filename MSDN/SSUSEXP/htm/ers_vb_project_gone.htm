The project <project> no longer exists under <provider>.

This error has the following cause and solution:

  • The specified project has been deleted (or moved). You must recover (or select) the specified project before any source code control commands can be applied to it.

    If the project was moved, you receive a dialog box to help you search for the new project. If the project was deleted, you must run the source code control provider to reinstate the project.
