Using Visual SourceSafe 6.0

When you work with VSS, you work with projects and the files they contain. You can store any type of file in VSS. Operations that VSS enables you to perform include:

  • Add Files, Folders, and Projects

  • Branch and Share Files and Projects

  • Check In and Check Out Files

  • Cloak Projects

  • Create New Projects

  • Customize the SS.INI and SRCSAFE.INI Files

  • Customize VSS

  • Delete and Recover Files and Projects

  • Edit Files

  • Get and View Files and Projects

  • Label Files and Projects

  • Merge Files

  • Move Files and Projects

  • Open/Close a Database

  • Pin Files and Projects

  • Rollback to Previous Versions

  • Search for Files

  • Set Working Folders

  • Set Default File Types

  • Set Passwords

  • Use the Command Line

  • Use the Options Dialog Box

  • Version Control and History

  • View or Modify File or Project Properties

  • Work with Shadow Folders

  • Work with Web Projects

  • Writing Batch Files

  • Use VSS Off-line
