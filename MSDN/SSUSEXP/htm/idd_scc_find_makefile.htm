Designate Makefile Dialog Box

Allows the user to specify which Visual Basic project file in VSS to make a new Visual Basic project from.

Note   This dialog box will only appear when a user clicks the Create Project From SourceSafe command and they have two or more Visual Basic project files stored in the same VSS project. Since Visual Basic doesn't know which of the VSS items to create a new Visual Basic project from, it will prompt the user to decide.

Dialog Box Options

Select the Visual Basic file

Shows the user a list of the Visual Basic project files stored in VSS from which they can use to create a new Visual Basic project file.

OK

VSS will create a new Visual Basic file from the item selected in the Select the Visual Basic file list.

Cancel

Cancels the Create Project from VSS command.
