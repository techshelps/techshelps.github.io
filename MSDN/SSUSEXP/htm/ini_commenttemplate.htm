Comment_Template Initialization Variable

Sets a template file to be used for comments in VSS Explorer or the comment editor.

Syntax

Comment_Template = template file

Example

Sets Template.txt as the template to be used when entering comments:

Comment_Template = R:\SS\Template.txt

Remarks

This variable points to a template a text file that you create containing a comment template. When the user is prompted for a comment, either in VSS Explorer or in a Comment_Editor used by the command line, the template is placed into the comment field.
