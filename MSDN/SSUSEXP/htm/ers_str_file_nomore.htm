No more files

This warning has the following cause and solution:

  • VSS was unable to find any more files to process.

    Run the Analyze utility to make sure there are no errors in your VSS database. If running Analyze with the –F switch does not correct the problem, contact Product Support.

Note   When running the Analyze utility, first run it with no switches to find any errors in the database, then with just the –F switch to run in autofix mode.
