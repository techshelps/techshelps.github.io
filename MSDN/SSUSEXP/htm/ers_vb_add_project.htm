Unable to add the new project to source code control.

This error has the following cause and solution:

  • There are insufficient source code control provider access rights.

    Ask your source code control administrator to increase your access rights, and try again, or try to add the project to a different place in the project tree.


    Log on again to VSS with the proper password.


    (If you have had other errors just proceeding this one, review their causes for more possible causes.)
