Clean Temporary Folder

See Also

As the VSS administrator, you are responsible for cleaning the Temp folder pointed to by the Temp_Path initialization variable in the SRCSAFE.INI file.

VSS usually places files in its Temp folder while running, and deletes them before quitting. Some circumstances, such as having to restart, can cause files to remain in the Temp folder a waste of disk space.

Once every few weeks, when no users are running VSS or VSS Administrator, clean the Temp folder.

To clean the temporary folder:

  1. On the Tools menu, click Clean up Temp Directory to display the Clean Up Local Temporary Files Directory dialog box.

  2. Click OK.
