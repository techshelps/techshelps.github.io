Share and Branch a File in One Operation

See Also

Sharing a file means it exists equally in all the projects to which it is shared. Branching a file breaks the shared link, making the file in that project independent of all other projects.

Note   To view projects sharing a file, click the Links tab in the Properties command on the File menu. To view the branch history of a file, click the Paths tab in the Properties command on the File menu.

To branch and share a file on one operation:

  1. In VSS Explorer, select the project into which you want to branch/share a file.

  2. On the SourceSafe menu, click Share to display the Share dialog box.

  3. In the File to share list, select the file you want to share. If the file you want to share is not displayed, select the project containing the file from the Projects list.

  4. Select the Branch after share check box.

  5. Click Share to display the Share File dialog box.

  6. In the Comment box, type a comment describing the branch/share, if desired.

  7. Click OK to close the Share File dialog box.

  8. Click Close to close the Share dialog box.
