Pin Files

See Also

You can pin any file.

To pin a particular version of a file:

  1. In VSS Explorer, select the file you want to pin.

  2. On the Tools menu, click Show History to display the History Options dialog box.

  3. Click OK to display the History of File dialog box.

  4. Select the version of the file you want, then click Pin. A pushpin icon appears next to the pinned file.

  5. Click Close.

When it is time to use the most recent version of the file, you unpin it.
