How to Install VSS

During installation, VSS asks you to choose the type of installation you want to perform. Select one of the following:

  • Client Installation from a Disk. Allows VSS users to install from a compact disc to the hard disk of their client computer only the files necessary to run VSS. The client installation does not install the VSS database.

  • Client Installation from a Network. Allows VSS users to install from a network to the hard disk of their client computer only the files necessary to run VSS. The client installation does not install the VSS database.

  • Server Installation. Allows the VSS administrator to install VSS to a computer that is accessible to all users.

  • Custom Installation. Allows the VSS administrator to select which parts of the server installation to install.

Depending on which installation option you've selected, VSS installs folders at specific project locations; see Folders Installed by Setup for details.
