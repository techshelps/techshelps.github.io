URL Initialization Variable

See Also

Marks a project as a Web site project. When a Web site project is selected, VSS Explorer's Web menu is activated.

Syntax

URL = url

Example

Specifies that the project under which this variable is listed corresponds to a Web site Uniform Resource Locator (URL):

URL = http://www.microsoft.com/ssafe

Remarks

This variable can also be set on the Web Sites tab that is accessed from the Tools menu, by clicking Options.
