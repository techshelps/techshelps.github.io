View Options Tab

In this tab you can set VSS Explorer view options.

To display the View options tab:

  1. On the Tools menu, click Options.

  2. Select the View tab.

Tab Options

SourceSafe Window

  • Show Results Pane — Select this option to show the Results pane in VSS Explorer. The Results pane, at the bottom of VSS Explorer, shows you the results of your VSS commands.

  • Show Toolbar — Select this option to show the toolbar in VSS Explorer. The toolbar contains a set of buttons you can use to carry out commonly used commands.

  • Show Status Bar — Select this option to show the status bar in VSS Explorer. The status bar shows you status information about VSS, such as the current file pane sort order, the current username, and the number of items in the file list.

File List

  • Always Use MS-DOS Format Filenames — Select this option to truncate the long file names supported in 32-bit platforms. This option keeps the file name column of the display narrow, and is also useful if you have computers with several different operating systems using your VSS database.

  • Make Selected Project the Current Project — This option is normally selected. However, if your network suffers from poor performance, you may want to be able to scroll the project list without VSS updating the current project as you do so. In this case, turn off this option. If it is turned off, you must press the ENTER key on a project (or double-click with the mouse) to make it the current project.

  • Automatically Refresh File List — Select this option to have VSS automatically refresh its file list. Refreshing keeps VSS Explorer up-to-date, but it also causes a momentary delay (and a network load). If you select this option, enter the refresh interval in minutes in the box below. If you do not select this option, you can manually refresh the file list using the Refresh File List command on the View menu.
