Rename Files or Projects

See Also

VSS allows you to edit the name of a file and still retain its history. If a file is shared by multiple projects and it is renamed, it will be renamed in all the projects that share the file. However, if any of the share links have been broken by the Branch command, branched files are not renamed.

To rename a file:

  1. In VSS Explorer, select the item whose name you want to change.

  2. On the File menu, click Rename.

  3. Type the item's new name.

  4. Press ENTER when finished.
