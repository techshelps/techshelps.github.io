A deleted link to <File> already exists

See Also

A deleted file remains part of a project, but it doesn't appear in the file list. This error has the following cause and solution:

  • You are attempting to share a file from another project with the current project. However, the file already exists in the current project. It doesn't appear in the file list because it has been deleted.

    To make the file appear in the file list, don't share it from another project; rather, recover it in this project using the Recover button on the Deleted Items tab in the Properties dialog box on the File menu.
