Invalid VSS path <path>

See Also

This error has the following cause and solution:

  • You have entered an invalid VSS project or file name.

    Check to make sure that you typed the path correctly, or use VSS Explorer to select the item from the project list or file list.

    If you chose the item from VSS Explorer, it is possible that another user has deleted or renamed the item since you last used VSS. Collapse the root project and expand it again to refresh the project pane.
