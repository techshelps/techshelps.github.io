Help (Command Line)

See Also

Displays Help on using VSS in general, or on a specific command.

Syntax

ss Help [Command] [-H] [-I-] [-Y] [-?]

Examples

Shows general help:

ss Help

Shows help on the Share command:

ss Help Share

The following table describes the command-line options available with this command.

Option Description
-I- Ignore: Do not ask for input under any circumstances.
-Y Specify a username and/or password.
-?, -H Request online Help on a command.

Access Rights

Any VSS user can use this command.

Remarks

Help for VSS command-line commands is displayed in a Microsoft Windows Help file. You must be running Microsoft Windows for Help to be displayed there is no separate command-line help.
