Create Project Command (File Menu)

See Also

Creates a new subproject under the currently selected project.

Access Rights

You must have the Add access right in the parent project to use this command.

Remarks

Use the Create Project command to create a subproject under the current VSS project. The subproject is empty until you place files or other subprojects in it. All access rights that you have in the parent project propagate to the newly created project.

Dialog Box Options

Project

Type a name for the new project.

Comment

Type a brief description of the project.
