Virtual_Root Initialization Variable

Sets the virtual root for a particular Web project.

Syntax

Virtual_Root = [/Project name]

Example

Specifies that when Web hyperlinks are checked, links including a virtual root designation (supported by Microsoft Internet Information Server) of /VBASIC are checked:

Virtual_Root = /VBASIC

Remarks

If your Web project doesn't include hyperlinks to URLs with a virtual root, you do not need to set this variable.
