Deploy_Path Initialization Variable

See Also

Specifies the location to which files are copied when a user chooses the Deploy command from the VSS Web menu.

Syntax

Deploy_Path = folder

Example

Instructs VSS to deploy files to the \\internet\server1\ssafe location when the Deploy command is selected:

Deploy_Path = \\internet\server1\ssafe

Remarks

Used only for projects marked as Web site projects by the URL initialization variable. (You can also mark a project as a Web site project using the VSS Administrator Web Sites tab from Options on the Tools menu.) The location for deploying files can also be set from the Web Sites tab. You can set multiple deployment paths, separated by a comma.
