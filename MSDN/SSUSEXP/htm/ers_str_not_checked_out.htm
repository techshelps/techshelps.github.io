You do not have file <File name> currently checked out

This warning has the following cause and solution:

  • You are attempting to check in or undo checkout on a file that you don't have checked out.  The file may be checked out by another user.

    Identify who, if anyone, has the file checked out and have them perform the Check In or Undo Check Out command.  If you can't find the individual, the Admin user can check in files another user has checked out.

    To identify who has a file checked out, view the Check Out Status tab on the file properties dialog or use the ss status command.
