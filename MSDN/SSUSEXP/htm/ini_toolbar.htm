Toolbar Initialization Variable

See Also

Shows or hides the toolbar in VSS.

Syntax

Toolbar = {Yes|No}

Example

Displays the toolbar:

Toolbar = Yes

Remarks

At the top of VSS Explorer is the toolbar, or icon bar, which gives quick mouse access to some of the most common VSS features.

You can also set this variable in your SS.INI file by using the Show Toolbar option on the View tab in the SourceSafe Options dialog box.
