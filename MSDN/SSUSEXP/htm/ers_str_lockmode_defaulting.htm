Lock_Mode INI variable has illegal value.  Assuming Native.

This warning has the following cause and solution:

  • VSS can be set up to use its own form of file locking.  This is set by the VSS Administrator adding a Lock_Mode = INI file setting to the SRCSAFE.INI file.  At the present time, the setting is incorrect.

    Edit the SRCSAFE.INI to correct the Lock_Mode INI file setting to be either Native or Lockfile.
