Add Multiple Files

See Also

You can add multiple files to VSS by following the steps below or by using drag and drop. See Add Files and Folders Using Drag and Drop.

To add multiple files to VSS:

  1. Select the project you want to add files to.

  2. On the File menu, click Add Files to display the Add File dialog box.

  3. In the Drives list, select the drive where the file resides.

  4. In the Folders list, select the folder where the file resides.

  5. In the List files of type list, select a file type. This is an optional step used to filter the list down to a more manageable size.

  6. In the File name list, hold down the CTRL key and click all the files you want to add.

    Note   To select all displayed files, press the HOME key to move to the beginning of the list, then SHIFT+END to move to the end. This dialog box will only accommodate 256 characters.

    Files already in the current project are not displayed in the File name list.

  7. Click Add.

  8. VSS displays a dialog box in which you can enter a comment. Select the Apply Same Comment For All check box to have this comment applied to all the files being added; otherwise the dialog box is repeated for each file you have selected.

  9. Click OK to add the files with a comment or click Cancel to cancel the Add File operation. VSS returns to the Add File dialog box.

  10. Click Close.
