Client Installation from a Network

See Also

This option installs on the hard disk of a client computer only those files necessary to run VSS. It does not install the VSS database or support files. Typically, client installation is done by a VSS user, not by an administrator.

The Netsetup program performs a client setup over a network. Netsetup does not require you to have an installation disc at your computer.

Note   Netsetup must follow a server installation.

Choose this option in the following situations:

  • You have heavy traffic on your network. The client option can help decrease the network load and improve performance on the client computer compared with running VSS from a server.

  • You are integrating VSS with Microsoft Access, Visual FoxPro, Visual Test, Visual Basic, or Visual C++. The client installation (either from compact disc or using Netsetup) is necessary for VSS to work properly with Visual Basic or Visual C++.

    Important   To integrate VSS with Visual Basic, first perform a server setup of VSS, install Visual Basic, then either run Netsetup or perform a client setup from disc. To integrate VSS with Visual C++, run Netsetup or perform a client setup from disc, either before or after installing Visual C++. For more information on VSS integration with Visual Basic, Visual C++, Visual FoxPro, Visual Test, and Microsoft Access, see documentation for those products.

To perform a client installation from a network:

  1. In the folder in which VSS is installed on the server, double-click Netsetup.

  2. Follow the setup instructions in the setup wizard.
