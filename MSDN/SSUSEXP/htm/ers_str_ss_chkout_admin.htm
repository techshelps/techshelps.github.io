File <File name> is currently checked out by <user name>. Proceed anyway?

This warning has the following cause and solution:

  • You are attempting an operation on a file checked out to another user that would remove the check out information.  For example, the Admin user can undo check out of a file checked out by anyone.

    Click Yes to continue the operation, losing the check out information.

    Check Yes All to have your choices in this dialog box apply to all other files checked out to other users.

    Click No to skip the operation for this file.

    Click No All to skip the operation for all files checked out to other users.

    Click Cancel to cancel the operation.
