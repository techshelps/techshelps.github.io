Sort Command (View Menu)

Sorts the file list in VSS Explorer. Sort order options are by file name, type, user, date, or check out folder.

Access Rights

Any VSS user can use this command.

Remarks

The file list in VSS Explorer is usually sorted by file name. You can change the sort order:

  • Name (file name)

  • Type (file type extension)

  • User (usernames of users who have files checked out)

  • DateTime

  • Check Out Folder

Type, Name, User, and Check Out Folder sort in ascending order; DateTime sorts in descending order, so that the most recent files are on top. The current sort order is always displayed in the status bar.
