Right Mouse Button

See Also

VSS Explorer provides another way of working with the file list and project list. If you click a file with the right mouse button, VSS displays a shortcut menu of common file commands. If you click a project with the right mouse button, VSS displays a menu containing commands that operate on projects.

When you click the right mouse button, then drag items from Windows Explorer into VSS, a list of options including Add File(s), Check In, Undo Check Out, and Cancel becomes available.

Note   You cannot drag items from one copy of VSS to another unless you have the same VSS database open in both copies.
