window turn

Turning a window refers to the process of filling an MCB with the appropriate retrieval pointers in order to map a VBN for a file to an LBN for a disk.

working set

The set of pages that are resident for a given process while it has one or more running threads. See also balance set and quota.

The size of each process’s working set is bounded by a minimum number of pages that the NT Memory Manager guarantees to be resident while the process has one running thread and by a maximum. However, NT does not necessarily constrain a process’s working set to its maximum if many free pages are available.
