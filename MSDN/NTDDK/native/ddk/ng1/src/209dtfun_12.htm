9.1.12 NIC Information String Indexes

This summarizes the definition and format of the strings that can be output by the NcDetectIdentify function of a NIC_detection DLL:

Index*

Description

Format

nn00

Option string for netcard in INF

Unicode string; no embedded delimiters

nn01

Displayable name of netcard

Unicode string

nn02

Displayable name of manufacturer

Unicode string

nn03

Search order

Unicode decimal numeric string

* nn = 10 or more; nn00 = base card index (lNetcardId)
