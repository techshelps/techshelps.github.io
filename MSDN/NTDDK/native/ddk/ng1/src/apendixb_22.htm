B.5.2 Streams IOCTL Commands

STREAMS uses the STREAMS IOCTL commands listed below. The command functions are documented in the streamio section of Further Readings on Streams, Reference 1.

Plumbing Commands:

  • I_LINK

  • I_PUSH

  • I_UNLINK

Specialized Commands:

  • I_FDINSERT

  • I_STR

A special IOCTL command, DATAL_IBIND, is described in Section 31.7 of this specification.

STREAMS does not implement the stream head IOCTL commands listed below.
