B.4 Streams Message Types

STREAMS supports the message types listed below. These message types are documented in Further Readings on Streams, Reference 1, Appendix B.
Regular Priority:

M_BREAK (1,2)
M_DELAY (1,2)
M_SETOPTS
M_CTL (1,2)
M_IOCTL
M_DATA
M_PROTO

High Priority:
M_ERROR
M_IOCACK
M_PCPROTO
M_FLUSH
M_IOCNAK

 (1) Never generated at the write queue of the stream head

 (2) Discarded by the read queue of the stream head

STREAMS does not support the message types listed below. Neither the stream head nor the dndis driver generates or reacts to these message types.
M_COPYIN
M_PCRSE
M_START
M_COPYOUT
M_PCSIG
M_STARTI
M_HANGUP
M_READ
M_STOP
M_IOCDATA
M_RSE
M_STOPI
M_PASSFP
M_SIG
