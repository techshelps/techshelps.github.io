5.4 WAN Miniport-Specific NDIS Library Requests

The functions described here are NDIS functions that only a WAN NIC driver can call and OIDs that only a WAN miniport will receive. A WAN miniport calls NdisMWanXxx functions instead of the comparable function used by a LAN-miniport NIC driver. Otherwise, a WAN miniport can call any NdisMXxx that has no WAN counterpart.

Only the WAN-specific NDIS calls are described next. For a complete summary of all functions that a WAN miniport driver can call, see the preceding information about miniport NIC drivers.
