octet

A set of eight bits transmitted over a network.

OID (object identifier)

An OID is a 32-bit identifier a netcard driver uses to denote an object in its MIB.

OOB (Out of Band)

Out of band data, if supported by and supplied by a network driver, is associated with a packet descriptor. OOB data can contain one or more of the following: media-specific information, priority and timestamps. NDIS drivers access OOB data using NDIS-supplied macros.
