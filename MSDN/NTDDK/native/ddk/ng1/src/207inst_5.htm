7.2.1 Optional Parameters in the PCMCIA Database

The optional parameters in the database inform pcmcia.sys how the slot is to be configured. This configuration information is needed since most network cards do not specify the correct tuple information. The following are the valid parameters in this section:

    AttributeMemorySize: REG_DWORD: Size

    CardMemorySize: REG_DWORD: Size

This information is stored in the driver section of the registry. Since only a single set of optional parameters can be associated with a PCMCIA card, this implies that only one card of a particular type can be in the machine.
