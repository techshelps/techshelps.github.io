9.2.2 File ncparam.inf

The file ncparam.inf is similar to utility.inf in its provision of a standard set of INF sections that can be called and used through the Win32 Setup Shell command. These INF sections interface to the detection-wrapper functions described previously.

The details of the interaction between a NIC .inf file and ncparam.inf are very complex. Provided in sample files, the DDK contains comments for versions of utility.inf, ncparam.inf, and one or more ISA NIC .inf files. These should be read carefully to understand the use of ncparam.inf.
