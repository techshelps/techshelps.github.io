8.1 Overview of Network Interface Card Detection

The process of detecting a network interface card includes the following:

  • Initial Setup Detection (primary network interface card and NIC driver installation)

  • Configuration Querying (secondary NIC installation)

  • Configuration Verification (secondary NIC installation)

  • Resource Validation (secondary NIC installation)

The initial setup detection, configuration querying, and configuration verification are supported by DLLs associated with network interface cards. For more information about these DLLs, see Chapter 9. The resource validation process is implemented by functions contained in code for Setup or the NCPA.
