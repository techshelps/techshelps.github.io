A.1 Introduction

The client for a Microsoft® Windows NT® NIC driver is a generic transport driver. A TDI driver is a transport driver. For information about TDI, see Part 3.

Both the driver and its client use calls to or from the NDIS interface library, which is a set of functions and macros that forms a wrapper for interfacing between bound transport drivers and network interface cards for a particular network. Windows NT supports the following types of network interface card:

  • Ethernet (802.3)

  • Token Ring (802.5)

  • FDDI

  • WAN (point-to-point and WAN network interface cards)

  • LocalTalk

  • DIX

  • ARCNET
