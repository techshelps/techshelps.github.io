Version 4.0 unless otherwise specified.

NDIS library

A DLL containing a set of abstract functions that interface between transport driver functions, intermediate driver functions and netcard driver functions, submit a request to the operating system, or cause a local action that does not require communication with other software functions. The main purpose of the NDIS library is to form a wrapper that allows network drivers to send and receive packets on a LAN or WAN in an operating system-independent manner.

netcard

A network adapter card residing in a computer and controlled in its transmission and reception of network information by a particular netcard driver.

netcard driver

A driver that provides datalink layer functions to higher-level network drivers through the NDIS interface library. A single netcard driver can support numerous netcards of a particular type.

NIC (network interface card)

A netcard.
