Chapter 5 WAN Miniport NIC Drivers

This chapter describes NDIS functions for supporting WAN NIC miniport drivers and the features of WAN miniport drivers. WAN miniports use the set of NDIS functions used by nonWAN miniports and functions that do the following:

  • Replace a set of NDIS functions used by nonWAN miniports

  • Provide additional WAN-specific capabilities to WAN miniports

WAN NIC miniport drivers are developed using the requirements and functions described in Chapter 2, Chapter 3, and Chapter 4 of this part with the exceptions and additional considerations described in this chapter. The differences between WAN miniports and LAN miniports are described in Section 5.2.1.

Windows NT® RAS enables remote workstations to dial in to the LAN and transparently access resources on the LAN.
