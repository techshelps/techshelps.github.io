2.4 Setup Functions

The Setup functions work with INF files to provide functionality such as:

  • General Setup Functions

    • Queuing files

    • Prompting for media and handling errors

    • Installing files

    • Updating registry entries

    • Logging installed files

    • Storing the most recently used source paths

  • Device Installation Functions

    • Searching INF files for compatible drivers

    • Display driver choices in user selection dialogs

    • Install drivers

Unicode and ANSI versions are available for most Setup functions. The ANSI functions can be used to create setup applications for the Windows 95 operating system.
