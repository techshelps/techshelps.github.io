2.1.2.2 The HKEY_LOCAL_MACHINE\SOFTWARE Tree

This tree of the registry contains per-machine information about the software on the local computer. Per-machine information is independent of per-user configurations. To install information about your product in this tree, create a CompanyName key. Under this key, create a key for each software product. Under the product key, create a key for each version of the product. You can create value entries and additional subkeys to store whatever information you prefer.

HKEY_LOCAL_MACHINE\SOFTWARE\CompanyName\ProductName\Version
