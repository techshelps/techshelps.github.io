A.3 GUI INF Script Language

This section describes the GUI INF script language used in oemsetup.inf files. A driver writer can use this language to create custom installation and configuration scripts for Windows NT device drivers. The Windows NT Setup program acts as an interpreter and executes the instructions in these text file scripts.
