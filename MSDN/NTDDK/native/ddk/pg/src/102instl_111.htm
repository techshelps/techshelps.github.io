SP_CLASSIMAGELIST_DATA

typedef struct _SP_CLASSIMAGE_DATA {
    DWORD cbSize;
    HIMAGELIST ImageList;
    DWORD Reserved;
} SP_CLASSIMAGE_DATA, *PSP_CLASSIMAGE_DATA;
 

SP_CLASSIMAGELIST_DATA contains the class image list.

Members
cbSize
Contains the size, in bytes, of the SP_CLASSIMAGE_DATA structure.
ImageList
A handle to the class image list.
