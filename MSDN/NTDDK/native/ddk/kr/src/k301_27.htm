ScsiPortReadRegisterUchar

UCHAR
    ScsiPortReadRegisterUchar(
        IN PUCHAR  Register
        );

ScsiPortReadRegisterUchar reads an unsigned byte value from the HBA.

Parameters

Register
Points to the register. The given Register must be in a mapped memory-space range returned by ScsiPortGetDeviceBase.

Return Value

ScsiPortReadRegisterUchar returns an unsigned byte from the HBA’s register.

Comments

ScsiPortReadRegisterUchar ensures that the data is transferred correctly.
