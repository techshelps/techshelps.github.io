ExDeletePagedLookasideList

VOID
    ExDeletePagedLookasideList(
        IN PPAGED_LOOKASIDE_LIST  Lookaside
        );

ExDeletePagedLookasideList deletes a paged lookaside list.

Parameters

Lookaside
Points to the header of the lookaside list to be deleted.

Return Value

None.

Comments

ExDeletePagedLookasideList removes the given lookaside list from the system list of paged lookaside lists and frees any remaining entries in the list. This routine does not deallocate the list header.

Callers of ExDeletePagedLookasideList must be running at IRQL < DISPATCH_LEVEL.
