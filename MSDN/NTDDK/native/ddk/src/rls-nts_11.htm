Multimedia Drivers

A new message has been added to the standard driver messages that user-mode multimedia drivers can receive:

  • DRV_PNPINSTALL

    The new DRV_PNPINSTALL message, like the existing DRV_INSTALL message, is sent to user-mode multimedia drivers at driver installation time. Parameters passed to the driver with the DRV_PNPINSTALL message provide Plug and Play configuration information in a format that can be passed directly to device installation functions provided by setupapi.dll.

    The DRV_PNPINSTALL message is described in the Introduction to Multimedia Drivers. Device installation functions are described in the Programmer’s Guide.
