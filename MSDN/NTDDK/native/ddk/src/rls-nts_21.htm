Documentation Changes for V4.0

The following table outlines how the Windows NT DDK documentation has changed for this release.

Online Documentation

Changes and Status

Getting Started

Revised for V4.0

Provided in a separate help file. Added documentation roadmap, documentation conventions, and instructions on how to navigate through the documentation.

Legal Information

Revised for V4.0

Minor revisions to update version numbers, and so forth.

Release Notes

NEW for V4.0

Highlights changes for this release.

Programmer’s Guide

Revised for V4.0

Includes documentation for the new INF file format and the device installation functions provided by the Setupapi DLL. Driver debugging information has been updated and expanded.

Kernel-Mode Drivers

Revised for V4.0

Includes documentation for a variety of new features, routines, structures, IOCTLs, and so forth. The graphics material has been moved to the graphics driver manuals.

Graphics Drivers

NEW for V4.0

The Graphics Driver Design Guide provides design information about kernel-mode graphics drivers and user-mode spooler components that can be developed by IHVs.

The Graphics Driver Reference contains reference pages for GDI, DDI, the DirectDraw driver model, the videoport driver, video miniport drivers, and for all printer-related components that can be developed by an IHV.

Multimedia Drivers

Revised for V4.0

Contains extensively revised introductory, MCI, audio, audio compression, and video capture chapters.

Network Drivers

Rewritten for V4.0

The Network Driver Design Guide has been completely rewritten.

The Network Driver Reference also has been completely rewritten.

Virtual DOS Drivers

Same as V3.51

This material has not changed for V4.0.

Code Samples

NEW for V4.0.

Code samples are now included in the online Help.

Knowledge Base

Updated for V4.0

Contains new articles.
