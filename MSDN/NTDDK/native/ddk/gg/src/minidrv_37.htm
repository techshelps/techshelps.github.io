Compression Dialog Box

A number of sets of graphics compression commands can be set up for the minidriver. To specify graphics compression commands, select Graphics Info from the Printer Data menu on the minidriver’s Printer Data window. Then select Compression to display the initial Compression dialog box.

To select an existing set of graphics compression commands, highlight the desired set of Compression commands to edit on the minidriver’s Printer Data window. Then, double click on the selection to display the Compression Modes dialog box.

Some printers can receive raster graphic data in a compressed format that typically results in increased performance. The increased performance is due to the reduced amount of data sent to the printer to print bitmaps. If your printer supports compression of graphics data (bitmaps) using run-length encoding (RLE), or Tag Information File Format (TIFF) version 4.0, use the Compression Modes dialog box.

The Diconix 150 Plus printer does not support data compression. The HP LaserJet IIP printer supports both run-length encoding and TIFF. The HP LaserJet examples are based on the dialog box that contains commands for TIFF encoding.

When you finish entering data in the Compression dialog box:

  • Click the OK button to accept the additions or changes.

  • Click the Add button to add an additional set of commands.

  • Use the Previous or Next buttons to page to the preceding or next set of commands.

  • Click the Delete button to delete a set of commands.

Compression Type

Click the Compression Type box and select the type of compressionas follows:

  • Run Length Encoding

  • Tiff Encoding (4.0)

(Delta Row compressions is not currently supported)

Diconix 150 Plus: Does not support compression. The Compression Type box is empty.

HP LaserJet IIP: Supports RLE as specified in the Compression Type box.

Begin Compression

The command in the Begin textbox enables the compression of graphics data.

Diconix 150 Plus: The Begin box is empty.

HP LaserJet IIP: Use the Set Compression Mode command (ESC*bnM) to enable and disable the various types of graphics compression. If n is 0, graphics compression is disabled; if n is 1, run-length encoding is enabled; if n is 2, TIFF encoding is enabled; and if n is 3, delta-row compression is enabled.

To specify the command to enable run-length encoding, type the following in the Begin box: 

\x1B*b1M
 

End Compression

The command in the End listbox disables the compression of graphics data.

Diconix 150 Plus: The End box is empty.

HP LaserJet IIP: Use the Set Compression Mode command (ESC*bnM) to enable and disable various types of graphics compression. If n is 0, graphics compression is disabled; if n is 1, run-length encoding is enabled; if n is 2, TIFF encoding is enabled; and if n is 3, delta-row compression is enabled.

To specify the command to disable TIFF encoding, type the following in the Begin box: 

\x1B*b0M
