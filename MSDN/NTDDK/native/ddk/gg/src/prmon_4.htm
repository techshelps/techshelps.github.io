Print Monitor Installation

A language monitor is associated with a printer driver when the Win32 AddPrinterDriver function is called with the language monitor’s name specified in the pMonitorName field of the DRIVER_INFO_3 structure. This parameter information is specified in a printer driver INF file using the LanguageMonitor keyword.

Default Windows NT printer driver installation information can be found in the system-supplied ntprint.inf file. OEMs can supply an INF file that associates a printer driver with a custom language monitor or a Microsoft-supplied one. See the Programmer’s Guide for driver installation details.
