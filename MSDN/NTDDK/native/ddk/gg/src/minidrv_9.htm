Commonly Used Minidriver Values

Unitool separates the minidriver data into a set of related application functions. These functions can be selected using the cascading menu near the top or from the individual lists shown in the minidriver's Printer Data window. When a function is selected, a corresponding dialog box is displayed to permit minidriver data to be entered or updated. Each dialog box can contain two type of values: selectable values and user-defined values.

Selectable values are listed as a set of values commonly used by the minidriver, such as the predefined values for standard paper sizes, paper sources, and paper destinations. Unitool dialog boxes also request user-defined values such as a name, a printer command string, or a control value. In addition, defaults are provided for the user-defined values commonly entered for the source driver.
