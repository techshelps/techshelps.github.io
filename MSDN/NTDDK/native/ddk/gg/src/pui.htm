Chapter 9 Printer User Interface Functions and Structures

This chapter describes the functions and structures that a printer driver’s user interface component must support. Printer driver function and structure declarations are found in winddiui.h, which is provided with this DDK.

If you implement a printer user interface module for your printing device, you must ensure that this module can communicate with the Common Property Sheet User Interface. Reference material for the CPSUI can be found in Common Property Sheet User Interface Functions and Structures.
