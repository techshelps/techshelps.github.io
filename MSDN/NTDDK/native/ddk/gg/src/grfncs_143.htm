FLOATOBJ_Div

VOID FLOATOBJ_Div(

    PFLOATOBJ pf,

    PFLOATOBJ pf1

   );

FLOATOBJ_Div divides the two FLOATOBJs, and returns the result in the first FLOATOBJ parameter.

Parameters

pf
Pointer to the first FLOATOBJ operand. The division of pf by pf1 is returned in this parameter.
pf1
Pointer to the second FLOATOBJ operand.

Comments

The FLOATOBJ_Xxx services allow graphics drivers to emulate floating point arithmetic. Windows NT does not support kernel-mode floating point operations on some systems.
