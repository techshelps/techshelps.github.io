Chapter 14 Print Monitors

Windows NT supports two types of print monitors: port monitors and language monitors. This chapter describes both monitor types, and provides design and implementation guidelines.

The Windows NT DDK supplies sample code for each type of monitor:

Directory Name

Print Monitor Type and Comments

localmon

Port monitor. The resultant executable, localmon.exe, is the default port monitor shipped with Windows NT.

pjlmon

Language monitor for bidirectional printers that understand PJL.
