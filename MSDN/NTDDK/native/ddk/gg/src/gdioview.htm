Chapter 2 GDI Support for Graphics Drivers

This chapter describes the Microsoft® Windows NT® graphics device interface (GDI). It then details the support that GDI provides to graphics drivers.

References to “GDI” in this book are implicit references to kernel-mode GDI; Win32 GDI will be explicitly identified. Kernel-mode GDI is also known as the Graphics Engine.

GDI function and structure references are documented in the Windows NT DDK Graphics Driver Reference.
