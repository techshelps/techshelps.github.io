ModelData Dialog Box Messages

Default Font ID must refer to a valid font
An invalid value is entered in the Default Font ID box.
Default CTT ID must refer to a valid CTT
The value entered in the Default CTT ID box does not identify a valid character-translation table.
Every Model must have a name
The string entered in the Name of Model box is invalid.
Look Ahead Region must be non-negative
The value entered in the LookAhead Region box is negative.
Max Page Width must be > 0 or equal -1
The value entered in the Max-Width page size limits box is equal to 0 or less than -1.
Max Page Height must be > 0 or equal -1
The value entered in the Max-Width page size limits box is equal to 0 or less than -1.
minidriver name supports Font Cartridges, but has no Cartridge slots
Zero is entered in the sCartSlots box, and one or more font cartridges is selected in the supported attributes Font Cartridges dialog box. The Font Cartridges dialog box is associated with the ModelData dialog box.
minidriver name has no PageControl
A set of supported page-control commands is not identified in the ModelData dialog box.
minidriver name has no CursorMove
A set of supported cursor-movement commands is not identified in the ModelData dialog box.
minidriver name has no Resolutions
A set of supported resolution commands is not identified in the ModelData dialog box.
minidriver name has no Paper Sizes
A set of supported paper-size commands is not identified in the ModelData dialog box.
minidriver name: Memory Values not used for non-HPPCL devices
One of the listed memory configurations in the ModelData dialog box is for a printer that does not use HPPCL.
Min Page Width must be > 0
The value entered in the Min-Width page size limits box is less than or equal to 0.
MODELDATA Min Page Height must always be > 0
The value entered in the Min-Height page size limits box is less than or equal to 0.
sCartSlots must be non-negative
The value entered in the sCartSlots box is negative.
sLeftMargin must be non-negative
The value entered in the sLeftMargin box is negative.
sMaxPhysWidth must be > 0
The value entered in the sMaxPhysWidth box is not greater than or equal to zero.
