VideoPortWriteRegisterBufferUchar

VOID VideoPortWriteRegisterBufferUchar(

    PUCHAR  Register,

    PUCHAR  Buffer,

    ULONG  Count

   );

VideoPortWriteRegisterBufferUchar writes a number of unsigned bytes to a mapped register.

Parameters

Register
Points to the register. The given Register must be in a mapped memory-space range returned by VideoPortGetDeviceBase.
Buffer
Points to a buffer containing the bytes to be written.
Count
Specifies the number of bytes to be transferred to the adapter.

Comments

A miniport’s HwVidInterrupt or HwVidSynchronizeExecutionCallback function can call VideoPortWriteRegisterBufferUchar.
