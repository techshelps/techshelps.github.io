Chapter 12 Printer Monitor Functions and Structures

This chapter provides function and structure reference pages for both port and language monitors. Function prototypes and structure definitions can be found in the Windows NT DDK winsplp.h include file.

Except for InitializePrintMonitor, InitializeMonitorEx, and InitializeMonitor, this chapter’s functions have pseudonames that can be renamed by monitor developers.

All functions are noted as required or optional for each monitor type.
