String-Table Resources

The GPC data refers to string-table resources by indices. String resources are specified in a number of the Unitool dialog boxes. For example, strings indicate the printer models the driver supports. In the user interface, strings define how to format the resolution for printing and how to display the information for the user. Other strings define information, such as the names of the paper trays. The string indices track the actual IDs in the .rc file. The strings are stored in that file and can be translated into other languages if necessary.
