DrvDeleteDeviceBitmap

VOID DrvDeleteDeviceBitmap(

    IN DHSURF dhsurf

   );

DrvDeleteDeviceBitmap deletes a device bitmap created by DrvCreateDeviceBitmap.

Parameters

dhsurf
Identifies the bitmap to be deleted. This handle identifies the bitmap created by DrvCreateDeviceBitmap.

Comments

A display driver must supply DrvDeleteDeviceBitmap if it supplies DrvCreateDeviceBitmap.
