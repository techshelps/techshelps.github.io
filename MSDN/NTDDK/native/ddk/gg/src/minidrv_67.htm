General Error Messages

Bad format Font file: minidriver name
The format of the specified font file is invalid.
Bad Format GPC file
A GPC file with an invalid format is requested for loading.
Bad Format RC File
The specified RC file uses an invalid format.
Can’t add atom for minidriver name
The PC cannot process a string. If this error occurs, exit Unitool and Windows.
Can’t change drives: minidriver name
The PC cannot switch to the specified driver.
Can’t create directory:minidriver name
The PC cannot create the specified directory.
Can’t Find File minidriver name
The requested file cannot be found.
Can’t find minidriver name in UNITOOL.INF
The PC cannot find an entry for the specified minidriver in the UNITOOL.INF file.
Can’t find string with requested ID
The GPC file contains an entry in its string table that is not present in the corresponding RC file.
Can’t find UNITOOL.INF in Windows directory
The PC cannot locate the file named UNITOOL.INF.
Can’t read font file: minidriver name
The specified font file cannot be read.
Can’t read PFMEXTENSION, unable to access DRIVERINFO
The PFMEXTENSION structure in the PFM file cannot be parsed. (If Unitool cannot parse the PFMEXTENSION structure, Unitool cannot read the DRIVERINFO structure.)
Can’t read RC File: minidriver name
The specified font file cannot be read.
Can’t save CTT reference change in: minidriver name
After a reference to a CTT is changed, the font file cannot be changed.
Can’t Save File minidriver name
The file cannot be saved using the specified filename.
Can’t save RC File: minidriver name
The specified resource file cannot be saved.
Can’t update Read Only file: minidriver name
A font file saved with read-only permission cannot be saved.
Default Font ID in: minidriver name Is no longer valid
The font file containing the minidriver default font was deleted.
Global Memory Allocation failed
The required memory cannot be allocated. If this error occurs, exit Unitool and Windows without attempting to save data.
GPC and/or RC file is read only,Save will be disabled:minidriver name
Either the specified resource file or the associated GPC file was saved with read-only permission.
Illegal CTT file entry:minidriver name
The entry for a CTT file in the specified resource file cannot be parsed.
Illegal Font file entry:minidriver name
The entry for a font file in the specified resource file cannot be parsed.
Illegal GPC file entry:minidriver name
The entry for the GPC file in the resource file cannot be parsed.
Illegal DRV name, must be between 1-8 chars
The entered minidriver name exceeds 8 characters.
Illegal string
The entered formatted string is invalid.
Illegal STRINGTABLE entry in:minidriver name
The specified resource file contains an invalid string table entry.
Must Provide a Destination Directory
The directory name (specified as part of the minidriver name) is invalid.
Newer version of Unitool needed to read this file
A newer version of Unitool must be loaded in order to edit the specified GPC file.
No errors were found
No errors were found after the Validate Now command is selected.
No structures of this type have been defined
A modification is attempted on data that in not defined.
Not a device font PFM file: minidriver name
The specified font files do not correspond to a device font.
Numeric Value required
A non-numeric value is entered instead of a numeric value.
OverWrite the file: minidriver name
An existing minidriver file is about to be overwritten.
RC file doesn’t reference GPC file:minidriver name
The specified resource file does not contain a reference to the listed GPC file.
RC file lists duplicate font files
The specified resource file contains duplicate font file entries.
RC file lists duplicate CTT files
The specified resource file contains duplicate CTT entries.
RC file has non-contiguously numbered font files
The specified resource file contains noncontiguous identifiers for minidriver font files.
Requested CTT ID value already used by: minidriver name
The specified CTT identifier is already in use.
Replacement CTT ID doesn’t exist
The specified replacement CTT does not exist.
Save Failed, restoring original files
The GPC file cannot be saved and the original files must be restored.
STRINGTABLE missing END statement
The specified resource file contains a string table that is missing an END statement.
Target Directory doesn’t exist, Create?
The specified directory does not exist.
Unable to retrieve CTT file names, Can’t save file
The specified CTT files cannot be retrieved and the corresponding resource file cannot be saved.
Unable to retrieve font file names, Can’t save file
The specified font files cannot be retrieved and the corresponding resource file cannot be saved.
Unable to retrieve RC strings, Can’t save file
The minidriver’s resource strings cannot be retrieved and the file cannot be saved.
Unresolved STRINGTABLE reference for ID:minidriver name
The specified string-table reference cannot be resolved.
