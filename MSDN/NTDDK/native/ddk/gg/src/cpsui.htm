Chapter 10 Common Property Sheet User Interface Functions and Structures

This chapter describes the functions and structures that comprise the Common Property Sheet User Interface (CPSUI). The CPSUI executable is a system-supplied module called compstui.dll. If you implement a printer user interface module for your printing device, you must ensure that this module can communicate with the CPSUI. CPSUI function and structure declarations are found in compstui.h, which is provided by this DDK.

Reference material for the printer user interface are found in Printer User Interface Functions and Structures.
