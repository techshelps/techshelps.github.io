PATHOBJ_bPolyLineTo

BOOL PATHOBJ_bPolyLineTo(

    IN PATHOBJ *ppo,

    IN POINTFIX *pptfx,

    IN ULONG cptfx

   );

PATHOBJ_bPolyLineTo draws lines from the current position in a path through the specified points.

Parameters

ppo
Points to the PATHOBJ created by the driver.
pptfx
Points to an array of POINTFIX structures that define control points. The first line is drawn from the current position to the first point in this array; lines are then drawn to each subsequent point in the array.
cptfx
Specifies the count of points in pptfx. This is also the number of lines that will be added to the path.

Return Value

The return value is TRUE if the function is successful. Otherwise, it is FALSE, and an error code is logged.

Comments

PATHOBJ_bPolyLineTo should only be called with PATHOBJ objects created by EngCreatePath.
