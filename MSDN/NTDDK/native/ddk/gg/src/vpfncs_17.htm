VideoPortMoveMemory

VOID VideoPortMoveMemory(

    PVOID  Destination,

    PVOID  Source,

    ULONG  Length

   );

VideoPortMoveMemory copies data from the source location to the destination location in system memory.

Parameters

Destination
Points to the destination location.
Source
Points to the location of the data to copy.
Length
Specifies the number of bytes to copy.

Comments

VideoPortMoveMemory moves memory either forward or backward, aligned or unaligned, in 4-byte blocks, followed by any remaining bytes.

The given Destination and Source must be in a mapped logical range returned by VideoPortGetDeviceBase and/or a system-space RAM address, such as an address on the stack. Attempting to transfer data to or from a device-memory block, such as an adapter frame buffer or ROM block, can cause access violations in certain Windows NT® RISC-based platforms.
