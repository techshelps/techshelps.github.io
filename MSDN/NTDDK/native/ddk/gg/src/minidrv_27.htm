Control Information

RasDD sends page-control commands to the printer to begin a document, to begin a page, to end a document, to end a page, to cancel a print job, and so on. RasDD sends cursor-movement commands to the printer to move the cursor or print head horizontally across the page, vertically down (and sometimes up) the page, and so on.

A number of sets of page-control and cursor-control commands can be set up for the minidriver. To specify the page-control commands, select Control Info from the Printer Data menu. then select PageControl to display the initial Page Control dialog box. To specify cursor-movement commands, select Control Info from the Printer Data menu. Then select CursorMove to display the initial Cursor Movement dialog box.
