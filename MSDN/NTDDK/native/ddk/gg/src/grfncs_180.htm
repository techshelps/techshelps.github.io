PATHOBJ_vGetBounds

VOID PATHOBJ_vGetBounds(

    IN PATHOBJ *ppo,

    OUT PRECTFX *prectfx

   );

PATHOBJ_vGetBounds retrieves the bounding rectangle for the specified path.

Parameters

ppo
Points to a PATHOBJ structure that describes the path for which a bounding rectangle is to be calculated.
prectfx
Points to the address where the RECTFX structure is to be written. The returned rectangle is exclusive of the bottom and right edges. An empty rectangle is specified by setting all four RECTFX members to zero.
