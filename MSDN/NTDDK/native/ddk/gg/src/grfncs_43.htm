DrvQueryTrueTypeTable

LONG DrvQueryTrueTypeTable(

    IN ULONG iFile,

    IN ULONG ulFont,

    IN ULONG ulTag,

    IN PTRDIFF dpStart,

    IN ULONG cjBuf,

    OUT BYTE *pjBuf

   );

DrvQueryTrueTypeTable accesses specific tables in a TrueType font-description file.

Parameters

iFile
Identifies the driver-provided TrueType font file.
ulFont
Specifies the one-based index of the driver font.
ulTag
Specifies the table to access. If ulTag is zero, access is to the entire file.
dpStart
Specifies the offset from the beginning of the table to begin access. If ulTag is zero, dpStart is the offset from the beginning of the file.
cjBuf
Specifies the size of the buffer pointed to by pjBuf.
pjBuf
Points to the buffer in which to copy the table or file data.

Return Value

If pfBuf is null, the return value is the size, in bytes, required for this buffer. If pfBuf is not null, the return value is the number of bytes copied. If an error occurs, the return value is FD_ERROR.

Comments

DrvQueryTrueTypeTable is required for TrueType font drivers.
