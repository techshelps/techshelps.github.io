EngDeleteSemaphore

VOID EngDeleteSemaphore(

    HSEMAPHORE hsem

   );

EngDeleteSemaphore deletes a semaphore object from the system’s resource list.

Parameters

hsem
Handle of the semaphore to be deleted. The semaphore was created in EngCreateSemaphore.
