Unitool Error Messages

Unitool provides two ways of checking the data described in the minidriver for possible problems and reporting that information back to you. From the Options menu, you can choose two commands: Validate Dialog Boxes and Validate Now!.

Validate Dialog Boxes

The Validate Dialog Boxes command may be chosen or canceled at your discretion. When it is chosen, whenever you display a dialog box and choose OK, Next, or Previous (that is, any choice that will cause the current dialog box contents to be saved rather than ignored), Unitool will perform some limited error checking on the contents of that dialog box. If any potential problems are found, a new dialog box listing all the potential problems will be displayed, and you may choose to ignore these warnings or return to the dialog box to correct them. All of the error messages, along with a corresponding explanation, are listed at the end of this appendix. If the Validate Dialog Boxes command is not chosen, error checking will not occur. This command is enabled by default.

Validate Now

The Validate Now command on the Options menu performs some limited error checking on the contents of the entire minidriver file currently loaded. If any potential problems are found, a new dialog box listing all the potential problems is displayed. All of the error messages, along with a corresponding explanation, are listed at the end of this appendix.
