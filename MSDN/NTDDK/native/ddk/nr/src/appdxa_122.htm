TrDeleteFilter

VOID
    TrDeleteFilter(
        IN PTR_FILTER Filter
        );

TrDeleteFilter is called by a NIC driver, usually when it is unloading, to delete the memory allocated for a Token Ring filter library database.

Parameters

Filter
Points to the filter library database memory to be deleted.

Comments

The NIC driver must clear any active filters from the database before it calls TrDeleteFilter.

Callers of TrDeleteFilter run at IRQL PASSIVE_LEVEL.
