The following describes the OID_XXX listed in the preceding chart more fully.

OID_ARCNET_RECONFIGURATIONS

This OID returns the number of network-reconfiguration interrupts received by the NIC driver for an ARCNET card since the system was started.

A reconfiguration takes place every time an ARCNET card is connected to or disconnected from the network. A reconfiguration also takes place every time an ARCNET card is reset. So every time an ARCNET card is reset on, added to, or removed from the network, every other NIC on the ARCNET network will augment its reconfiguration count.
