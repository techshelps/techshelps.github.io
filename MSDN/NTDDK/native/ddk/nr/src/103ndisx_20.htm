NdisCloseConfiguration

VOID
    NdisCloseConfiguration(
        IN NDIS_HANDLE  ConfigurationHandle
        );

NdisCloseConfiguration releases the handle to the registry Parameters key returned by NdisOpenConfiguration or NdisOpenProtocolConfiguration.

Parameters

ConfigurationHandle
Specifies the handle returned by NdisOpenConfiguration or NdisOpenProtocolConfiguration.

Comments

This function frees any temporary storage allocated in calls to other NdisXxx functions that required the returned ConfigurationHandle as a parameter.

Callers of NdisCloseConfiguration run at IRQL PASSIVE_LEVEL.
