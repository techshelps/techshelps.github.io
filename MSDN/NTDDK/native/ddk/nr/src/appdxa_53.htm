FddiFilterIndicateReceiveComplete

VOID
    FddiFilterIndicateReceiveComplete(
        IN PFDDI_FILTER  Filter
        );

FddiFilterIndicateReceiveComplete is called by a NIC driver to indicate to all bound protocol drivers that FddiFilterIndicateReceive has returned.

Parameters

Filter
Points to a previously created and initialized FDDI filter library database.

Comments

To deliver the necessary indications, FddiFilterIndicateReceiveComplete calls the ProtocolReceiveComplete of each bound protocol once for each indication.

Callers of FddiFilterIndicateReceiveComplete run at IRQL <= DISPATCH_LEVEL.
