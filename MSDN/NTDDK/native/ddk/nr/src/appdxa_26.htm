A.4 NDIS Library Functions Called Only by Full-NIC Drivers

This provides references for NDIS library functions that only legacy full-NIC drivers call. These references are in alphabetical order.

In addition to the functions described here, existing full-NIC drivers call some of the NDIS functions described in Part I of the Network Driver Reference. For a complete list, which includes only functions and macros that NDIS V3.0 also exported, see the preceding section, A.3 Summary of NDIS Library Functions for Full-NIC Drivers.
