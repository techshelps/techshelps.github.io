Platform SDK Release Notes

The April 1998 release of the Microsoft® Platform Software Development Kit (SDK) documentation describes the application programming interfaces (API) you need to write Win32®-based applications. The Platform SDK contains a variety of components. These components are integrated into functional groups within the SDK, making them easier to find.

The Platform SDK documentation contains APIs that depend on the version of the operating system or other components that are installed. After you have determined that a system has installed support for a particular API, you can use its functionality. When you build your application, use the new header files and import libraries from the Platform SDK.

The following topics describe the new features introduced in each release of the Platform SDK.

  • Release Notes for Windows NT 4.0 Service Pack 3
