Subject Property

Returns or sets a String representing the subject of the specified Discussion object.

expression.Subject

expression    Required. An expression that returns a Discussions object.

Example

This example sets the specified discussion's subject to "More Supply Questions".

Dim dscDiscussions As Discussions
Set dscDiscussions = OpenDiscussions("http://samples.microsoft.com")
dscDiscussions(1).Discussions(3).Subject = _
    "More Supply Questions"
