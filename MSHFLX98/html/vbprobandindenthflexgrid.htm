BandIndent Property (MSHFlexGrid)

      

Specifies the number of columns by which to indent a band.

Syntax

object.BandIndent [=number]

The BandIndent property syntax has these parts:

Part Description
object An object expression that evaluates to an object in the Applies To list.
number A Long value that specifies the band to be indented in the MSHFlexGrid. The default setting is 0.
