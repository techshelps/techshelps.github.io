RowColChange Event (MSHFlexGrid)

      

Occurs when the currently active cell changes to a different cell.

Syntax

Private Sub object_RowColChange()

The RowColChange event syntax has one part:

Part Description
object An object expression that evaluates to an object in the Applies To list.

Remarks

The following order of events occurs when the active cell changes: first LeaveCell, next EnterCell, finally RowColChange. The RowColChange event occurs when a user clicks a new cell. It does not occur while a user drags a selection across the MSHFlexGrid.
