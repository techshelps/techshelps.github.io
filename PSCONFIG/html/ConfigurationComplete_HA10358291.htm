Configuration complete

When the SharePoint Products Configuration Wizard completes all of the required configuration tasks, SharePoint Central Administration is available for you to use to configure your Web site.

Configuration Successful

When the necessary configuration tasks are successfully completed, if you are performing a Standalone installation, you can click Finish to close the configuration wizard. The home page for the default Web site opens.

If you are performing an Server Farm installation or upgrade, the configuration settings you entered are displayed, and then you can click Finish to close the configuration wizard. The SharePoint Central Administration site is displayed in your Web browser.

If you are performing a repair, click Close to exit the configuration wizard.

 Note   If you are using Microsoft Internet Explorer and have not modified the default settings, Enhanced Security will be enabled. When the configuration wizard starts Internet Explorer, a pop-up appears that prompts you to add the site to the list of trusted sites.

Configuration Failure

If there was an error in one or more configuration tasks, events are written to the event log and to the installation log. To enable you to diagnose the problem, detailed information about the configuration task that was running when the configuration wizard failed, any specific error messages, and a link to the log file, are displayed.
