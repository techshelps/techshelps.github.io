How to: Repair SharePoint products

Use this procedure to start the SharePoint Products Configuration Wizard in Repair mode.

  1. Click Start, click Control Panel, and then click Programs and Features.
  2. Select the installation you want to repair, and then click Change.
  3. In the Setup Wizard, select Repair, and then click Continue. A binary file validation is performed.
  4. On the final Setup Wizard screen, select Run the SharePoint Products Configuration Wizard now, and then click Close.
  5. The SharePoint Products Configuration Wizard is automatically called in Repair mode, detects missing components, and then determines the installation mode that was originally selected.
  6. If the installation was a multi-server deployment, you will be prompted for application pool credentials.
  7. Provide the credentials for the Run-as account for the application pool.
  8. The configuration wizard verifies prerequisites and executes tasks required to repair the configuration.
