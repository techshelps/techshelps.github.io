Read the Microsoft Software License Terms

The Microsoft Software License Terms defines the legal conditions under which you can use your software. In Setup, select the I accept the terms of this agreement check box to enable the Continue button.

View the Microsoft Software License Terms for your product after it is installed

  1. Click the File tab.
  2. Click Help, and then under About Microsoft Office, click Microsoft Software License Terms.

If you receive an error message or encounter a problem, visit The Microsoft Office 2010 Setup Resource Center for troubleshooting information and how to contact Microsoft Product Support.
