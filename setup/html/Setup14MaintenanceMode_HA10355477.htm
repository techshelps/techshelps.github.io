Change your installation of Microsoft Office

Add or remove features   This option allows you to add or remove programs and components, and also to customize how your installed programs run.

Repair   This option allows you to reinstall programs. This is useful in cases in which installed program files become corrupted.

Remove   This option allows you to uninstall programs from your computer.

Convert   This option allows you to convert from a Trial version of the program to the corresponding Full version. Converting from a Trial version to the Full version removes usage limits from the program. No program settings or data are affected by the conversion.

If you receive an error message or encounter a problem, visit The Microsoft Office 2010 Setup Resource Center for troubleshooting information and how to contact Microsoft Product Support.
