Choose a file location

The installation path indicates where Microsoft Office 2010 will be installed.

If you want to install Office to an alternate location, click Browse, and then select the location that you want. The alternate location is usually on another drive on your hard disk and is chosen to avoid space constraints.

If you receive an error message or encounter a problem, visit The Microsoft Office 2010 Setup Resource Center for troubleshooting information and how to contact Microsoft Product Support.
