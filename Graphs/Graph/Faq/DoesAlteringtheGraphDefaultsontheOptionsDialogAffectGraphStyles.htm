Does altering the graph defaults on the Options dialog affect graph styles?

Only the default graph styles will be affected by changes made in the Documents/Graphs: Display and Documents/Graphs: Settings options panes in the Options dialog (accessible via the Tools menu). All other user-defined styles will not be affected these changes. Default system graph styles change to mirror the current system graph settings (as specified Documents/Graphs: Display and Documents/Graphs: Settings options panes) without any need to manually update them. Conversely, user-defined graph styles will retain all of their internal settings, despite any changes made Documents/Graphs: Display and Documents/Graphs: Settings options panes.
