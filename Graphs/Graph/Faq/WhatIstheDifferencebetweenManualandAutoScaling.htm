What is the difference between manual and auto scaling?

When the axis scaling is set to Manual in the Mode box, then the minimum, maximum, and step size for the axis are determined by the current values of the Minimum, Maximum, and Step size (as entered in the respective boxes in the Range group on the Axis: Scaling options pane of the Graph Options dialog. If the Mode is set to Auto (i.e., automatic), then STATISTICA will automatically determine the scaling based on the range of values to be plotted.
