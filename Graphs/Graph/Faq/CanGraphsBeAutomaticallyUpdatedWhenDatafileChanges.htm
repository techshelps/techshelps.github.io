Can graphs be automatically updated when the data file changes?

Yes. All Graphs menu graphs can maintain automatic links to the data from which they were created. Options for auto-updating graphs are available on the Options 1 tab of all graph specification dialogs.

From the Help - Animated Overviews menu, select How To: Auto-update Graphs with New Data to view a brief video about STATISTICA's graph updating and brushing facilities and select How To: Use the Dynamic "Data-Graphs" Link" to view a multimedia presentation of the use of dynamic features to automatically update graphs.
