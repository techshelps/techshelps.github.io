How do I define categories in categorized graphs?

If categorized graphs are requested from output dialogs of specific procedures that involve analyses of subsets of data, then they will automatically display the subsets that are currently analyzed (i.e., the subsets are already defined as part of the current analysis). On the other hand, the categorized graphs requested from the Graphs menu offer a variety of methods to specify subsets using one or two grouping variables. Also, custom-defined subset definitions can be used that can involve all variables in the current datafile.

Specifically, categories can be defined by:

· Integer values of grouping variables (Integer Mode),

· Dividing grouping variables into a requested number of equal-length intervals (Categories),

· Custom intervals (ranges) of grouping variables, defined by specific interval boundaries (Boundaries),

· Specific values (i.e., codes) of grouping variables (Codes), and

· User-defined "multiple-subset" definitions (Multiple Subsets) that can be entered as logical case selection conditions of virtually unlimited complexity (this categorization method can involve values of all variables in the current datafile (see Methods of Categorization).
