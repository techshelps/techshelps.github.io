How do I place text in a STATISTICA graph?

Even large portions of text (e.g., a report several pages long) can be pasted into STATISTICA Graphs using the Clipboard operations. Additionally, you can paste a portion of a document into the graph window using the Paste Special command. To edit and customize the text (within STATISTICA Graphs), double-click the text to display the Graph Titles/Text dialog (for custom text) or the respective OLE server application (for pasting in text via the Paste Special command).

Both the Clipboard-based as well as inserting operations listed in the previous topic apply to all Windows compatible graphs and artwork (linking and embedding operations support any OLE-compliant objects).
