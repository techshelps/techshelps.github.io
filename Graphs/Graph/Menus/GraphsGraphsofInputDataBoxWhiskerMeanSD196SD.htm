Graphs - Graphs of Input Data - Box-Whisker - Mean/SD/1.96*SD

Select Mean/SD/1.96*SD from the Graphs - Graphs of Input Data - Box-Whisker menu to create a box plot that shows the mean (small box in the plot) of the variable surrounded by a larger box (± 1 times the standard deviation). If the distribution is normal, then the whiskers in this plot represent a 95% confidence interval defined as the variable mean ± 1.96 times the variable standard deviation.
