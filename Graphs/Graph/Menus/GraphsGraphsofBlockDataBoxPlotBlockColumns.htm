Graphs - Graphs of Block Data - Box Plot: Block Columns

Select Box Plot: Block Columns from the Graphs - Graphs of Block Data menu (or the spreadsheet shortcut menu) to create a box plot of medians (and min/max values and 25th and 75th percentiles) for each of the columns in the block. To create a box plot of medians for each of the rows in the block, use the Custom Graph from Block by Row option.
