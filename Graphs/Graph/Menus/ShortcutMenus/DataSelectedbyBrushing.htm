Data Selected by Brushing

Select Data Selected by Brushing from the Graph Data Editor shortcut menu to display a submenu containing the Copy To Clipboard, Copy To New Plot, Move To New Plot, Delete, and De-select commands. Use these commands to define actions to be taken on graph data that have been defined by Brushing using any of the Point, Box, Lasso, Cube, Slice X, or Slice Y options appropriate for the active graph. The selected option will be applied to the entire set of brushing-selected cases.
