Pattern

Select Pattern to display a dialog relative to that particular Graph shortcut menu option. For example,

Select Pattern from the Histogram Bars shortcut menu or the box and whiskers shortcut menu to display the Area Properties dialog, which is used to modify area properties (e.g., patterns, colors, etc.).

Select Pattern from the Point Markers shortcut menu to display the Marker Properties dialog, which is used to change the size, color, pattern, and style of markers in the graph.

Select Pattern from the Line Plot shortcut menu or the axis shortcut menu to display the Line Properties dialog, which is used to change the line width, pattern, and color in the graph.
