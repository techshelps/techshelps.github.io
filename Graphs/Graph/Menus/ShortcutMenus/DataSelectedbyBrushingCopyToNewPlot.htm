Data Selected by Brushing - Copy To New Plot

Select Data Selected by Brushing - Copy To New Plot from the Graph Data Editor shortcut menu or the Graph Data Editor Layouts menu to add a new plot of the same type to the Graph Data Editor (and to the graph), and transfer a copy of the points selected to the new plot. One line will be copied for each case in the Graph Data Editor from the original plot (i.e., one for each point on the graph), but only those points selected by brushing will be represented by non-missing values in the new plot. The selected cases in the original plot in the Graph Data Editor will not be affected and will be effectively duplicated in the new plot.
