Properties

Select Properties to display a dialog relative to that particular graph shortcut menu option. For example,

Select Properties from the Graph Axis shortcut menu to display the Axis: Scaling options pane of the Graph Options dialog, which is used to customize the graph axis.

Select Properties from the Histogram Bar shortcut menu to display the Histogram dialog, which is used to customize the classes used in graphs.

Select Properties from the Point Markers or Line Plot shortcut menu to display the General dialog, which is used to customize general plot features including marker, point, and line displays and the assignment of axes.

Select Properties from the Box and Whiskers shortcut menu to display the Spreads dialog, which is used to customize various display characteristics (line colors and patterns, fill colors and patterns, etc.) of the components that make up box and whiskers type plots.

Select Properties from the Icon Plot shortcut menu to display the Plot Layout dialog, which is used to customize graph plots.

Select Properties from the Surface Plot shortcut menu to display the Fitting dialog. The options on this dialog are for fitting plots to the data including line patterns and other options.
