Disconnect Text from Graph

Select Disconnect text from Graph from the Graph shortcut menu to break the connection between text containing equations or statistics posted on the graph and the graph data. After selecting this command, subsequent changes made to data in the Graph Data Editor will have no effect on values displayed in the selected equations or statistics. The connection between equations/statistics and graphs must also be broken before they can be edited via the Graph Titles dialog, either by selecting Disconnect text from Graph before opening the dialog or by selecting Disconnect Object(s) from Graph on the dialog.
