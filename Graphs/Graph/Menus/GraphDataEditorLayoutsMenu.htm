Graph Data Editor - Layouts Menu

When the Graph Data Editor is displayed, the Layouts menu is available, which contains the Graph Options, Plot Layout, Calculated Graph Data, Fit Values, Voronoi Values, Custom Function Values, Point Label/ID Info, Brushing Mark, Brushing Label, Brushing Exclude, Brushing Hide, Brushing Off, Brushing On, Reset (Brushing) Block, and Data Selected by Brushing commands.
