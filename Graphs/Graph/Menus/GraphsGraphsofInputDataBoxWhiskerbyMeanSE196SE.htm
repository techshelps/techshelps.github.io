Graphs - Graphs of Input Data - Box-Whisker by - Mean/SE/1.96*SE

Select Mean/SE/1.96*SE from the Graphs - Graphs of Input Data - Box-Whisker by menu to create a categorized box plot that shows the mean (small box in the plot) of the variable surrounded by a larger box. However, in this plot, the larger box represents ± 1 times the standard error. If the distribution is normal, then the whiskers in this plot represent a 95% confidence interval defined as the variable mean ± 1.96 times the variable standard error.
