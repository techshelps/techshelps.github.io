Graphs - Multiple Graph Layouts - Wizard

Select Wizard from the Graphs - Multiple Graph Layouts menu to display the AutoLayout Wizard - Step 1 dialog. Use the options in this dialog to gather the graphs for the multi-graph layout you are designing. Once you have gathered graphs for the layout, click OK in this dialog to proceed to the AutoLayout Wizard - Step 2 dialog. Use the options in his dialog to specify an appropriate (for the number of selected graphs) layout for your multiple graph.
