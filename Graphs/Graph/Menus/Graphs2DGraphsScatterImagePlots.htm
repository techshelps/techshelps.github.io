Graphs - 2D Graphs - Scatter Image Plots

Select Scatter Image Plots from the Graphs - 2D Graphs menu to display the Scatter Image Plots dialog. Like scatter icon plots, a scatter image plot is a scatterplot whose point markers are images. However, these images are not icons defined by specific variables, but user-selected graphics files (bitmap, metafile, PNG, and JPG formats are supported). Different image files can be attached to different data points (defined by the scatterplot X and Y coordinates), and a weight variable can be used to scale specific image markers accordingly.
