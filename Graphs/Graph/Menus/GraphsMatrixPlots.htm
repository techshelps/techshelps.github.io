Graphs - Matrix Plots

Select Matrix Plots from the Graphs menu to display the Matrix Plots dialog. Matrix plots summarize the relationships between several variables in a matrix of X-Y plots. The most common type of matrix plot is the scatterplot matrix, which can be considered the graphical equivalent of the correlation matrix.

If one list of variables is selected when defining the matrix plot, a square matrix plot is produced and the histograms of frequency distributions of the respective variables are plotted on the diagonal. If variables are selected from two lists, a rectangular matrix plot is produced. In this style of matrix plot, the histograms for the respective variables are displayed in the first row and column of the matrix.
