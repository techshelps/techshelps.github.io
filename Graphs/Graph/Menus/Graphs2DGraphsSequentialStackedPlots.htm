Graphs - 2D Graphs - Sequential/Stacked Plots

Select Sequential/Stacked from the Graphs - 2D Graphs menu to display the Sequential/Stacked Plots dialog. Sequential/stacked plots are used to present sequences of values of selected variables. The stacked feature of this type of graph is specifically designed to represent the broad category of datasets in which consecutive variables represent parts ("portions") of one whole. Display options include a variety of combinations of lines, stepped lines, areas, and columnar displays.
