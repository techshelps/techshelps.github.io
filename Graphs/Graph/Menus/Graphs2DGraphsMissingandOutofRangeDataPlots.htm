Graphs - 2D Graphs - Missing and 'Out of Range' Data Plots

Select Missing/Range Data Plots from the Graphs - 2D Graphs menu to display the Missing and 'Out of Range' Data Plots dialog. Use this graph to examine the pattern or distribution of missing and/or user-specified "out of range" data points in the current dataset (or in a subset of variables and cases).

This graph is useful in exploratory data analysis to determine the extent of missing (and/or "out of range") data and whether the patterns of those data occur more or less randomly.

See also, Conceptual Overviews - Missing/Range Data Plots.
