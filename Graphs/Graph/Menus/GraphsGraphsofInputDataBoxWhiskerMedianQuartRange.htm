Graphs - Graphs of Input Data - Box-Whisker - Median/Quart./Range

Select Median/Quart./Range from the Graphs - Graphs of Input Data - Box-Whisker menu to create a box plot that describes the central tendency of the selected variable in terms of the median of the values (represented by the smallest box in the plot). The spread (variability) in the variable values are represented in this plot by the quartiles (the 25th and 75th percentiles, larger box in the plot) and the minimum and maximum values of the variable (the "whiskers" in the plot).
