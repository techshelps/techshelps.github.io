Insert - Surface Legend

Select Surface Legend from the Insert menu to display the Add Legend dialog. Use the options in this dialog to add a surface legend to the current graph.  Note that this option is for creation of the legend only. To add text or objects, customize fonts, etc., use the options in the Graph Titles/Text options pane of the Graph Options dialog [accessible by clicking Graph Options on the Format tab (ribbon bar) or by selecting Graph Options from the Format menu (classic menus)].
