Suppressing Scale Values Near Scale Breaks

In order to suppress displaying a scale value on the left side (or below for vertical axes) of the break, (after you have made the scale break) you can:

1. Double click on the axis to be edited to display the Graph Options dialog, and select Custom Units. Enter the values that you want to be displayed on the axis in the Position column and select the corresponding check boxes in the Value column. For example, if you only want values 25 - 35 (with a step size of 5), then enter 25, 30, and 35 into the Position column.  

2. Next, click select Scale Values. Select the Custom labels check box and clear the Automatic - at major tickmarks check box.

3. Finally, click the OK button in the Graph Options                             dialog to return to your graph.
