Function/Fit Values

Select Fit Values from the Graph Data Editor shortcut menu or the Graph Data Editor Layouts menu to display the Function/Fit Values display spreadsheet. This spreadsheet shows the fitted values that are plotted in the respective graph, for the respective plot. Note that the values displayed in the Function/Fit Values spreadsheet cannot be edited (changed).

For example, suppose you create a histogram with a normal fit. Next, display Graph Data Editor, for example by right-clicking on the graph and choosing Graph Data Editor from the shortcut menu. Then, with the Graph Data Editor active, select Fit Values - Plot 1 from the Layouts menu. You now have the following three windows on the screen.

Fit. The Function/Fit Values spreadsheet shows the respective function that is plotted (the normal curve in this case) in the Fit box; this box cannot be edited. The values in the two columns show the respective X and Y coordinates for the fitted line.

Copy. Click the Copy button to copy the currently selected (highlighted) block of data to the clipboard.

Close. Click the Close button to close the spreadsheet.
