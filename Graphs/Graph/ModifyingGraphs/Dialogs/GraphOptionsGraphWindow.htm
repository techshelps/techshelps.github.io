Graph: Window

In the Graph Options dialog tree view, select Window (under Graph) to access the options described here. Use these options to customize background colors and borders in the graph window. You can also access the appropriate background color dialog by double-clicking in either the outside background or inside background of the graph.

Outside Background Color/Inside Background Color. Click either the Outside Background Color or the Inside Background Color drop-down box to display a variety of background color options for the graph. In addition to a selection of predefined colors, you can choose a Transparent background or an Automatic background. You can also choose from a list of background color styles. Select Style Name to view the list of styles. Select More Colors to display the Color dialog, from which you can select a custom color.

Borders around the graph. Select the Borders check box to include a border around the graph window. Click the Borders button to display the Line Properties dialog, in which you can choose the color, pattern, and thickness of that border. The currently specified border line properties are shown in the view box on the Borders button. Note that if the Borders check box is cleared, STATISTICA will not include a border around the graph.

Size. The boxes in the Size group display the Width and Height of the graph. Enter a new Width or Height and click OK to change the size of the graph. The units of measurement are controlled by the Inches and Centimeters option buttons directly below the Size group. Note that you can change the default graph size in the Documents/Graphs: Display options pane in the Options dialog.

Inches. Select this option button to use inches as the unit of measure in the Size and Graph margins groups.

Centimeters. Select this option button to use centimeters as the unit of measure in the Size and Graph margins groups.

Graph margins. Use the boxes in the Graph Margins group box to specify the margins of the graph. The units of measurement are controlled by the Inches and Centimeters option buttons directly below the Size group. Note that the sum of horizontal or vertical margins cannot be larger than the width and height specified above.

Scaling of fonts/markers. Use the options in the Scaling of fonts/markers group box to specify how fonts and point markers will be scaled as the document size and margins are modified. By default, 100 is the specification. To specify a different percentage, enter the preferred percentage in the % box.

Style. Instead of using the options on this tab (e.g., Width, Height), you can select the style you want to use for this plot from the Style drop-down box. To view a menu of options related to styles (Save, Save As, Revert to unmodified, etc.), click the button to the right of this box. For more information on styles, see Graphics Styles.

Lock graph. Click the Lock graph button to display the Lock graph dialog. Using the options on this dialog, you can prevent changes to graph titles/legends, objects, appearance, axes scales, data, and brushing. You can also specify and confirm a password. When the password feature is in use, only authorized users (i.e., users who know the password) can modify the locking conditions and edit the graph.
