Insert Control

From the Help - Animated Overviews menu, select How To: Program Custom Graphics Applications to view a brief video about available custom graphics applications.

When a graph is displayed, select OLE Control from the Insert menu to display the Insert Control dialog.

Select the desired control from the list of controls.

Insertable objects only.  Select the Insertable objects only check box to view the list of object controls that can be inserted.

OK. After selecting a control from the list, click the OK button to insert the control into the graph.

Cancel. Click the Cancel button to close the dialog without inserting any controls.
