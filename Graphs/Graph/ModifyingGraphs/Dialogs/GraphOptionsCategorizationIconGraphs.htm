Categorization (Icon Plots)

The options described here are used to create categories of icons in the Icon plot using the selected grouping variable. In order to depict the categories of icons, STATISTICA encloses each icon in a box; all boxes belonging to the same category are drawn using the same line properties. The options on this tab also are used to change the line properties of the boxes drawn for different categories. These options are available on the Categorization options pane of the Plot Layout and Graph Options dialogs when you have created an Icon plot.

Intervals. Use the options in the Intervals group to select a method (Integer mode, Unique values, Categories, Boundaries, Codes, or Multiple subsets) to define the manner in which the values in the selected grouping variable will be used to determine the categories of icons to be displayed in the graph. Each icon in the plot will be enclosed in a separate box; all boxes belonging to the same category are drawn with a line of the same width, color, and pattern. The categorization methods are discussed in detail in Methods of Categorization. Note that these options are not available unless you have selected the On check box.

On. Select the On check box to make the Intervals group options and the Intervals value format button available.

Change Variable/Specify Boundaries/Specify Codes/Specify Subsets. Note that the name of this button changes depending on the Interval option button that is selected. Click the Change Variable button to display a standard variable selection dialog, which is used to change the selection of the grouping variable to which the Intervals options apply. Click the Specify Boundaries/Codes/Subsets button to display various dialogs in which you enter the boundaries, codes, or subsets that are used to create user-defined categories.

Intervals value format. Click the Intervals value format button to display the Value Format dialog from which you can choose a display format to use when labeling the values in the legend for the subsets defined by categorization. Note that this button is not available unless you have selected the On check box.

Lines. The Lines group is used to make changes to the line properties of the category-boxes.

Pattern. The current line properties of the boxes belonging to each category are displayed in the numbered square in the Pattern group; seven numbered squares appear at a time. If there are more than seven categories, you can view the line properties of other category-boxes by using the scrollbar below the Pattern group. You can change the Width, Pattern, and Color of a category-box border by clicking on its square to display the Line Properties dialog where you can make the desired selection.
