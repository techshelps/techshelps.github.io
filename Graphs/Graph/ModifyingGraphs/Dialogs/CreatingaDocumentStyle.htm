Creating a Document Style

STATISTICA's document styles are collections of various aspects of the graph pertaining to overall size, margins, etc, of the graph document. See the Document Style Overview for details.

Here is an example of how to create a document style and then use it to quickly customize a graph. First, to create the document style, select Options from the Tools menu. In the Documents/Graphs: Settings options pane of the Options dialog, review the current Document size and Graph margins. For this example, create a Document style specifying a particular graph document size (e.g., that makes the graph suitable for immediate inclusion into some report).

Set the Width field in the Document size group box in the Documents/Graphs: Settings options pane to 3 and the Height field to 2. Then set the margins to 0.5 in each Graph margins field.

To save these proportions as a document style click the button to the right of the Graph document style box, and select Save As, then type in the name of the new style in the Save As dialog.  

Click the Save button, and the respective Document style (new document size and margins) will be saved. Next, close the Options dialog; note that from now on all graphs will be created using these new sizes and margins, unless you select the Normal Document Size (or other) style again in the Documents/Graphs: Settings options pane of the Options dialog.

Applying the new style to an existing graph. Suppose that as part of some project you had previously created the following graph.

You want to quickly redraw this graph using the new specifications for sizes and margins as previously defined in the new document style. To apply this style, double-click on the background of the graph to display the Graph Options dialog. In the Graph: Window options pane of this dialog, select My New Document Style in the Style box.

Then click the OK button in the Graph Options dialog to redraw the graph, and the new sizes and margins will be applied.

Selecting the predefined (European) A4 style. There are several default document styles "built in". For example, if you wanted to further customize the graph and then print it on a European A4-sized page, select the A4 Document Size style in the Style box in the Graph Options - Graph Window options pane. After closing the Graph Options dialog, the graph will be redrawn according to those specifications.

Note that document styles can also be applied to a specific graph before it is created by selecting the document style in the Document style group box on the graph specification Appearance tab.
