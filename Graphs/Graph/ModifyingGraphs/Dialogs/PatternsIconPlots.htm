Patterns (Icon Plots)

Selecting Patterns in the Graph Options dialog is one way to access the options described here. Note that you can change the pattern in the Layout options pane. Use these options to make changes to the icons, such as colors, pattern etc. of the line, area, or point composing the icons in your Icon Plot. The options available on this tab depend on the type of icons plotted in your original graph.

Chernoff Faces. If you used Chernoff Faces as the icons in your Icon plot, the following options are available:

Chernoff Faces Patterns. The Chernoff Face Patterns group box is used to change the line properties: width/color/pattern of the line used in drawing different parts of the Chernoff faces. Click the Face, Nose, Mouth, Eyes, Eye Brows, or Pupils button to display the Line Properties dialog in which you can select the desired combination of width, color, and pattern for the selected feature of the face. The current specification of each of these icon features will be shown in the view window of the respective button.

Chernoff definitions. Click the Chernoff definitions button to display the Chernoff Face Definition dialog; here you can customize the assignment of variables to different aspects (features) of the face, as well as the scaling ("mapping") of the variables into those features.

Pies. If you used Pies as the icons in your Icon plot, the following options are available: (Note that any change made by these options will change all the pies on the Icon plot in the same fashion.)

Type. You can change the appearance of the pie by choosing either the 2-dimensional (2D) or 3-dimensional (3D) display in the Type group box.

Shape. Use the Shape group box to select the pie shapes. They can be either circular (select Circle) or elliptical (select Ellipse). However, choosing the Ellipse option button may cause the slices to be disproportionate in size (i.e., the slices that follow radii close to the vertical axis of the screen or paper will look relatively larger than their real size as compared to the "horizontal" slices which will be "squeezed" in perspective and will appear smaller).

Slices display. The options in the Slices display group control a variety of display features.

Number. Each pie slice has an assigned number to aid you in customizing the various slices.

Area. Each pie slice is represented by a numbered color square (the first slice is in the "12 o'clock" position in the pie) on this tab. You can change the pattern and color of a slice Area by clicking on its color square to display the Area Properties dialog where you can make the desired selection .

Exploded. Select the Exploded check box under the slice number to "explode" some or all of the slices of the pie.

Slice borders. Click the Slice borders button to display the Line Properties dialog from which you can de-activate the border or select a pattern, color, and thickness for the border around the slice. The current slice border specifications will be shown in the view window on the Slice borders button.

Placement. The options in the Placement group affect the position and appearance of the plot.

Thickness. Specify the thickness (in percentages of the Y radius) of the 3D pie chart in the Thickness field.

Angle. The Angle field is used to rotate the pie in order to change the starting position for the first slice of the pie (i.e., change the Angle in degrees; by default 0Â° = "12 o'clock").

Stars/Sun Rays/Polygons/Columns/Lines/Profiles. If you used Stars, Sun Rays, Polygons, Columns, Lines, or Profiles as icons in your Icon plot, the following options are available:

Points. Click the Points button to display the Markers Properties dialog in which you can select the Marker size, Marker pattern, and Foreground/Background color of the markers, or select a predefined Marker Style that you want to use for the icons in the modified Icon plot. The current marker specifications will be shown in the view window of the Points button.

Lines. Click the Lines button to display the Line Properties  dialog in which you can select the Line width, Line pattern, and Foreground/Background color of the lines, or select a predefined Line Style that you want to use for the icons in the modified Icon plot. The current line specifications will be shown in the view window of the Lines button.

Areas. Click the Areas button to display the Area Properties dialog in which you can select the Area pattern, and Foreground/Background color of the areas, or select a predefined Area Style that you want to use for the icons in the modified Icon plot. The current area specifications will be shown in the view window of the Areas button.
