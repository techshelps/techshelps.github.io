Style Properties

Right-click on the Graphic Styles Box and select Style Properties from the shortcut menu to display the Style Properties dialog, which displays the properties of the current graph-feature style.

Name. The Name text box displays the name of the current style.

Style. The Style text box displays the scope of the style (i.e. the area of the graph to which the style is associated with).

Type. The Type text box indicates whether the style is a system or user-defined style.

See also Graphics Styles Overview for further information regarding graphic styles.
