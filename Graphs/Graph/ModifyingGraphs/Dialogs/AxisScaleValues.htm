Axis: Scale Values

The options described here are used to select scale value options for each of the graph axes. These options are available in the Axis: Scale Values options pane of the Graph Options dialog. Separate options are available for each graph axis and the changes that you make for each axis are independent of one another (unless any of the Copy to options are selected to transfer the settings to other axes, see below). Therefore, the following descriptions of the scale options will apply to each axis in the graph. Note options for modifying the scales themselves are available in the Axis: Scaling options pane of the Graph Options dialog.

Axis. Although all of the options are the same for each axis, you can make changes to each axis independently of the other. Select here the axis (X, Y-left, Y-right, or Top axis for 2D graphs, X, Y, or Z axes for 3D graphs, or X, Y, Z, or V axes for triangular axis surface graphs) to which you want to make changes.

Copy axis specs to. Instead of manually selecting the same specifications for each of the axes, click this button to display the Copy axis specs to dialog. Use the options in this dialog to automatically copy to the remaining axes.

Display scale values. A variety of options are available for controlling the display of scale values. If you don't want to show any scale values, clear all available check boxes. Note that Variability plots have separate control over the direction of labels for each category, so these options are disabled (dimmed) for this type of plot.

Automatic - at major tickmarks. Select this check box to specify that STATISTICA will automatically format and display numeric scale values for you. Use the Value format options (see below) to select a specific display format for numeric scale values.

Use text labels from data set. If you want Text labels as scale value labels, select this check box (if the option is dimmed, no text labels exist).

Data values. Select the Data values check box to use the data values as scale value labels.

Related to plot. Click the Related to plot button to display the Related to plot dialog.

Custom labels. Select the Custom labels check box to use the custom labels as scale value labels. Note that this option is not available unless custom labels have been created (see Edit custom labels below).

Edit custom labels. Click the Edit custom labels button to display the Axis: Custom units options pane of the Graph Options dialog, where you can create or edit existing custom labels.

Skip values. Sometimes the scale values (numbers, dates, or labels) are so close together on the axis that they are hard to distinguish. This option enables you to skip values, displaying only the specified values (e.g., every fourth label). Select the Off option button to not skip any values. Select Auto to have STATISTICA decide what values to skip. Select Show every (and enter the appropriate number in the corresponding field) to explicitly determine the number of values to skip.

Value format. Select from a variety of number formats to use for the numeric scale values including General, Number, Date, Time, Scientific, Currency, Percentage, Fraction or Custom. For more information on these number formats, see Number Formats.

Options. This group box contains additional options that affect the display of scale values.

Layout. This option affects the layout of the scale values with respect to their axis. When you select Auto, STATISTICA will use either the Parallel, Perpendicular, or Alternate layout, whichever is appropriate for the current graph. The Perpendicular layout option will position the scale values perpendicular (vertically) to their respective axis, while the Parallel option will position the scale values horizontally (parallel) to their respective axis. The Alternate layout option will position the scale values in an alternating pattern (one above, one below) along the axis.

Font. Click this button to display the Font dialog, from which you can choose a different font for the scale values of the currently selected axis.
