Axis: Major Units

The options described here are used to customize the major units in the current graph; they are available in the Axis: Major Units option pane of the Graph Options dialog. Separate options are available for each graph axis and the changes that you make for each axis are independent of one

Axis. Although all of the options are the same for each axis, you can make changes to each axis independently of the other. Select here the axis (X, Y-left, Y-right, or Top axis for 2D graphs, X, Y, or Z axes for 3D graphs, or X, Y, Z, or V axes for triangular axis surface graphs) to which you want to make changes.

Copy axis specs to. Instead of manually selecting the same specifications for each of the axes, click this button to display the Copy axis specs to dialog. Use the options in this dialog to automatically copy to the remaining axes.

Step. The options in the Step group box apply to the step size used on the axis.

Mode. Select Auto in the Mode drop-down box to have STATISTICA determine the step size for you based on the current range, or select Manual to specify the step size yourself.

Edit range. Click the Edit range button to display the Axis: Scaling options pane, where you can edit the range of values for the selected axis.

Step size. Use the Step size field to specify the step size if you have selected Manual in the Mode drop-down box.

Related to plot. Click the Related to plot button to display the Related to plot dialog.

Integer step. In some cases it is desirable to apply integer steps starting from zero to the respective axis. For example, if the axis represents a (integer) count such as the number of failures in a study of product reliability, then you may want to select the Integer step check box. Note that this option is not available when Mode is set to Manual, in which case you can specify an integer step size in the Step size field if desired.

Step count. Use the Step count drop-down box to count off the step-sizes starting From 0, or from the current minimum value for this scale (select From minimum). For example, suppose the scale minimum is 0.5, and you select a step size of 1. If you select to start the step count From minimum, you will see scale values 0.5, 1.5, 2.5, etc. However, if you select From 0, you will see scale values 1, 2, 3, etc. (while the actual minimum value that can be plotted in the respective dimension is still 0.5).

Display units. Use the options in this group box to specify how the major units will be displayed.

Show tickmarks. Select the Show tickmarks check box to display a tickmark at each major unit. The style and size of the tickmark will be determined by the specifications in the Tickmark format box.

Gridlines. Select the Gridlines check box to display gridlines, and click the Gridlines button to display the Line Properties dialog, where you can modify the gridlines.

Tickmark format. Customize the style and size of major tickmarks for each of the axes with these options. The color and thickness of the tickmarks are determined by the current specifications of the respective Axis Line (see the Axis: General options pane).

Style. When you select one of the Inside, Outside, or Crossed style settings, the tickmarks will be displayed on the inside, outside, or both sides of the axis (respectively). To remove tickmarks from the axis, clear the Show tickmarks check box in the Display units group box (above).

Size. You can choose between Small, Medium, Large, or Custom sized tickmarks. If you select Custom, specify the relative length (from 0 to 9) of the tickmarks in the Points field to the right.
