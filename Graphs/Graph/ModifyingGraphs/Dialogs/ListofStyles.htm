List of Styles

This dialog displays all of the graphics styles for the currently selected feature.

Sort by. The Sort by box is used to sort the list of styles in the following methods:

Type (System first). The Type (System first) selection will sort the system styles first, then the custom styles.

Type (Custom first). The Type (Custom first) selection will sort the custom system styles first, then the system styles.

Name. The Name selection will sort the styles alphabetically.

See also, Graphics Styles - Overview for further information regarding graphic styles.
