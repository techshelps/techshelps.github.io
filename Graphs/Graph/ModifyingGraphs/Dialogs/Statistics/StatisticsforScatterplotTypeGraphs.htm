Statistics (for Scatterplot-Type Plots)

Click the Statistics button in the Scatterplot dialog (as well as on the Scatterplot tab of the Plot Layout dialog and in the Plot: Scatterplot options pane of the Graph Options dialog) to display the Statistics dialog. Note that this dialog can also be displayed by clicking the OK button in the Add Legend dialog.

Use this dialog to add to the graph (e.g., scatterplot) the results for linear least squares correlation. Note that regardless of the specific fit that is currently applied (e.g., polynomial fit), the reported statistics always pertain to the simple linear model that includes the intercept. Simple and multiple linear regression are discussed in detail in the context of the Multiple Regression facilities.

Plot. Although all of the options are the same for each plot, you can make changes to each plot independently of the other. Select here the plot to which you want to make changes. Note that this option will only be available if the Statistics dialog has been displayed via the Insert - Statistics menu.

R square. Select the R square check box to include the R-square value in the plot.

Correlation and p. Select the Correlation and p check box to include the value of the correlation coefficient and the p-value in the plot.

Regression equation. Select the Regression equation check box to include the regression equation in the plot.

Note: If you do select statistic(s) to include in your plot, the Add Legend dialog will also be displayed, which is used to select where and how to attach the statistic(s) to the plot.

Value Format. Use the Value Format group box to select from a variety of number formats to use for the numeric values including General, Number, Date, Time, Scientific, Currency, Percentage, Fraction or Custom. Most of the options in this list have more than one representation, except the General and Percentage options. Some options also allow you to specify decimal information. For more information on number formats, see Number Formats.
