Statistics (Box-Whisker Type Plots)

Click the Statistics button in the Box/Whisker dialog (as well as in the Box/Whisker options pane of the Graph Options dialog) to display the Statistics dialog. Note that this dialog can also be displayed by clicking the OK button in the Add Legend dialog or the Statistics button in the Plot: Variability plot dialog.

Use this dialog to add to the graph (e.g., box-whisker plot) the results of a Kruskal-Wallis nonparametric test, or the ANOVA test. Both compare the central tendencies (e.g., means) across the groups for each (dependent) variable. The Kruskal-Wallis test is further described in the context of the Nonparametric Statistics; the ANOVA test is described in the Breakdown Tables section of Basic Statistics, as well as in the General ANOVA/MANOVA section.

Plot. Although all of the options are the same for each plot, you can make changes to each plot independently of the other. Select here the plot to which you want to make changes. Note that this option will only be available if the Statistics dialog has been displayed via the Insert - Statistics menu.

Kruskal-Wallis-Test. Select the Kruskal-Wallis-Test check box to include the Kruskal-Wallis test statistic in the plot.

F Test, p. Select the F Test, p check box to include the F test and p statistics (analysis of variance results) in the plot.

Note: If you do select statistic(s) to include in your plot, the Add Legend dialog will also be displayed, which is used to select where and how to attach the statistic(s) to the plot.

Value Format. Use the Value Format group box to select from a variety of number formats to use for the numeric values including General, Number, Date, Time, Scientific, Currency, Percentage, Fraction or Custom. Most of the options in this list have more than one representation, except the General and Percentage options. Some options also allow you to specify decimal information. For more information on number formats, see Number Formats.
