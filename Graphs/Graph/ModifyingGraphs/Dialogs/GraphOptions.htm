Graph Options

Select Graph Options from the graph Format menu or from the graph shortcut menu to display the Graph Options dialog. Select items in the left pane (the tree view) to display customization options for graphs in the right pane (options pane). The options described below are available at the bottom of the dialog regardless of the item selected in the tree view.

Styles. Click the Styles button to display the Graphics Styles dialog. In this dialog you can choose the style to attach to the selected/specified object. Note that the styles available in this dialog depend on which options in the All Options dialog you have currently selected. For more information about graphic styles, see Graphics Styles Overview.

Record macro. Select this check box to start recording a macro after the OK button is clicked.

OK. Click the OK button to update the graph with the customizations made in this dialog.

Cancel. Click the Cancel button to return to the graph without making any customizations or changes.
