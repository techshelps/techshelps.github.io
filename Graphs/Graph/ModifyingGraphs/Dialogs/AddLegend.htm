Add Legend

Select Statistics from the Insert menu to display the Add Legend dialog. Note that this dialog is also displayed after you have selected statistics on the various Statistics dialogs when you have clicked the Statistics button in the Graph Options dialog.

Type. Use the Type box to specify whether the statistics are added as movable (Floating) or fixed (Fixed (title)) text.  

Position. Use the Position box to specify the placement of the statistics. If you have selected a Floating type, you can choose from Left-up, Left-down, Right-up, or Right-down. If you have selected to add the statistics to a Fixed (title) position, you can select from one of the five default title positions: Title, Subtitle, Footnote, Left, or Right.

OK. Click the OK button after you have made your selections to display the appropriate Statistics dialog or to return to the Graph Options dialog.

Cancel. Click the Cancel button to return to the graph or the Graph Options dialog, ignoring any changes you have made to the Add Legend dialog.
