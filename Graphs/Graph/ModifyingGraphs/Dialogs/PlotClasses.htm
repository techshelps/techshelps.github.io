Plot: Classes

The options described here are for customizing the classes that are used in graphs. These options are available in the Classes dialog as well as on the Classes tab of the Plot Layout dialog and via the Graph Options dialog.

Plot. Although all of the options are the same for each plot, you can make changes to each plot independently of the other. Select here the plot to which you want to make changes. If you have several plots in one graph, you can assign names to each plot using the Name option on the Plot: General tab.

Copy to. Instead of manually selecting the same specifications for each of the plots, click this button to display the Copy to dialog. Here you can select options to automatically copy to the remaining plots.

Type. You can choose between a Histogram or a Pie chart. If you select a Histogram, specify whether to show Standard, Cumulative, Hanging Bars or Stacked.

Intervals. These options are used to choose the method of categorization for each of the selected grouping variables. (Note that grouping variables do not need to be selected if Multiple subset categories are selected.)

Categorization is used in two classes of graphs in STATISTICA: categorized graphs (e.g., Categorized Scatterplots) and graphs that include grouping or categorized variables (e.g., 2D Histograms, or 2D Box Plots). You can choose from several methods of categorization, which are described below.

Properties. Specify the bar width in the Bar Width field. Typically, bars for continuous variables are connected (i.e., a width of 100%) is used, while discrete variables are not connected (i.e., a width of less than 100% is used). Click on the Intervals number format to edit the number format for the current variable in the Number Format dialog.

Styles. Click this button to display the Graphics Styles dialog. In this dialog, you can choose the plot style to use. For more about graphic styles, see Graphics Styles Overview.

Advanced. Click this button to display a dialog used to access a greater variety of customizations tools for this graph. Note this button is not available on the Graph Options dialog.
