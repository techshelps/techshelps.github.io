Plot: MD

Selecting the MD tab of the Graph Options dialog is one way to access the options described here. Use these options to modify the existing Missing or range data plot by defining new ranges (Min/Max values) for the variables that were used in the plot.  

Plot. The plots (ranges) that you used in your Missing or range data plot will be displayed in the Plot drop-down box from which you can select the plot (range) to which you want to make the changes. If you have several plots in one graph, you may want to assign names to each plot using the Name option on the Plot: General tab; the ranges in the Plot drop-down box will be replaced by the names.

Specify range for outliers. The Specify range for outliers group box is used to alter the minimum and maximum values defining the 'Out of range' values of the selected variable. Select the variable range from the Plot box and enter the desired values in the Min and Max fields. Values that fall above or below the newly defined range will be plotted in the graph. A different point pattern will be used for each defined range.
