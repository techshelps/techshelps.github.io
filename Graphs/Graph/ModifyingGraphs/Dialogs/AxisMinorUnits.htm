Axis: Minor Units

The options described here are used to customize the minor units in the current graph. These options are available in the Minor Units dialog (accessed by right-clicking on the axis and selecting Minor Units from the shortcut menu) and in the Axis: Minor Units options pane of the Graph Options dialog. Separate options are available for each graph axis and the changes that you make for each axis are independent of one another (unless any of the Copy to options are selected to transfer the settings to other axes, see below). Therefore, the following descriptions of the scale options will apply to each axis in the graph.

Axis. Although all of the options are the same for each axis, you can make changes to each axis independently of the other. Select here the axis (X, Y-left, Y-right, or Top axis for 2D graphs, X, Y, or Z axes for 3D graphs, or X, Y, Z, or V axes for triangular axis surface graphs) to which you want to make changes.

Copy axis specs to. Instead of manually selecting the same specifications for each of the axes, click this button to display the Copy axis specs to dialog. Use the options in this dialog to automatically copy to the remaining axes.

Number of units. Use the Mode drop-down box to allow STATISTICA to determine how many minor units to include (select Auto), or you can specify the number to use yourself (select Manual). If you select Manual, specify a number between 1 and 15 in the Number list box.

Display units. Use the options in this group box to specify how the minor units will be displayed.

Show tickmarks. Select the Show tickmarks check box to display tickmarks. Clear this check box to not display them.

Gridlines. Select the Gridlines check box if you want a gridline for each unit. You can customize the appearance of the gridlines in the Line Properties dialog, which is available by clicking the Gridlines button. The current specifications for gridlines will be shown in the view box on the Gridlines button.

Tickmark format. Customize the style and size of minor tickmarks for each of the axes with these options. The color and thickness of the tickmarks are determined by the current specifications of the respective Axis Line (see the Axis: General options pane).

When you select one of the Inside, Outside, or Crossed style settings, the tickmarks will be displayed on the inside, outside, or both sides of the axis (respectively). To remove tickmarks from the axis, clear the Show tickmarks check box in the Display units group box (above).

Size. You can choose between Small, Medium, Large, or Custom sized tickmarks.

Points. If you select Custom, in the Size box (see above), specify the relative length (from 0 to 9) of the tickmarks in the Points field.
