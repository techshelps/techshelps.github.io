Data Update Details

Click the Details button in the Graph Options dialog - Graph Layout options pane to access the Data Update Details dialog.

File. In this box, specify the path of the data source that is connected to the current graph. If the graph consists of multiple plots with separate data sources, choose the desired data source from the drop-down list.

Selection conditions. Click the Selection conditions button to display the Graph Case Selection Conditions dialog.

Break link with this file. When you click the Break link with this file button, the link will be broken, and the path shown in the File box (described above) will be deleted.

Change file. Click the Change file button to display a standard locate file dialog, where you can choose a different data source for the graph.

Variable. In this box, specify the variables used for the graph. You can also select a different variable from the drop-down list.

Change variable. Click the Change variable button to display a standard variable selection dialog, where you can choose a different variable for the graph.
