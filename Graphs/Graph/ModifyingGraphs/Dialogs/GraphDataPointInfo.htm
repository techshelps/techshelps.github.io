Graph Data Point Info

Select Point Label/ID Info from the Graph Data Editor shortcut menu or the Graph Data Editor Layouts menu to display the Graph Data Point Info dialog, which is used to change the brushing characteristics (i.e., marking, labeling, highlighting, or turning off) of a selected case or block of cases in the Graph Data Editor.

Marked. Select the Marked check box to mark the cases selected in the Graph Data Editor. By default the data for the selected cases will be displayed in bold characters in the Graph Data Editor and the corresponding point markers on the graph will become solid.

Labeled. Select the Labeled check box to label the cases selected in the Graph Data Editor. By default the data for the selected cases will be displayed in italic characters in the Graph Data Editor and the corresponding point markers on the graph will be labeled with case names (or numbers if case names are not available).

Excluded . Select the Excluded check box to exclude the cases selected in the Graph Data Editor. A case that is marked as excluded will be omitted from the graph (just as if it were excluded via selection conditions).

Hidden. Select the Hidden check box to hide the cases selected in the Graph Data Editor. A case that is marked as hidden will not be visible in the graph, but will be used in plot calculation.

OK. Click the OK button to accept the options selected and close the Graph Data Point Info dialog.

Cancel. Click the Cancel button to close the Graph Data Point Info dialog. Any changes made to the dialog will be ignored.

Use the corresponding check boxes on the dialog to assign or delete specific characteristics.
