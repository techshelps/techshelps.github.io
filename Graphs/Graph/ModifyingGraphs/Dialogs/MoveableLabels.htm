Moveable Labels

Click the Moveable point labels button in the Graph Options dialog - Layout options pane to display the Moveable Labels dialog.

Use moveable labels. Select this check box to enable labels.

Connect labels. Choose one of the following options: Auto, On, or Off.

Connect labels distance: _% of frame size. Enter the percentage for the distance. A label will not have a connection line until it is this % of the frame away from its respective object.

Draw frame when connecting point and label. Select this check box to draw the frame.
