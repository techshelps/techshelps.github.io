Graph Template

This dialog is displayed when you choose the Multiple Graph Layouts - Templates via the Graph menu. Select this option in order to create a Multiple AutoLayout Graph containing the specified number of blank graphs ("place holders"). This graph is useful when you want to use the Paste Special option to replace each blank graph with OLE objects.

Number of blank graphs to arrange. Specify the number (1 to 25) of blank graphs to be included in the Multiple AutoLayout Graph. The blank component graphs in the template can later be replaced with OLE objects (e.g., STATISTICA graphs or other artwork) using the Paste Special command from the Edit menu, or you can add extra text in place of the inserted blanks.

OK. Clicking OK will display the AutoLayout Wizard - Step 2 dialog in which you can select the desired layout and other options. Note that you can click on the Change Graph List in this dialog in order to bring up to AutoLayout Wizard - Step 1 dialog. Here, you can add current graphs or graphs from file to the list of blank graphs, then proceed as described in the AutoLayout Wizard - Step 1 dialog. Alternatively, you can create a Multiple AutoLayout Graph with current graphs or graphs from file and blank graphs using the Graphs - Multiple Graph Layouts - Wizard command.

Cancel. Click this button to close the dialog without creating a Multiple AutoLayout Graph.
