Lock Graph Message

This message is displayed when you select options in the Lock Graph dialog, but don't enter a password. Although a password is not required, it is strongly recommended. If a password is not entered and confirmed, then any user can unlock graph features by simply clearing the check boxes in the Prevent changes to box in the Lock Graph dialog.

Do not show this message again. Select this check box if you don't want this message to be displayed again.

OK. Click the OK button to close this dialog.
