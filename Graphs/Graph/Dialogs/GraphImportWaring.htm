Graph Import Warning

When you open a graph that was originally produced in STATISTICA Version 5, a graph import warning message will be displayed that says, "The imported graph may not fully reproduce the original Version 5 graph display."

Do not display this message again. Select this check box if you do not want to see this message every time you open a graph from an earlier version of STATISTICA.

Close. Click the Close button to close the graph import warning dialog.
