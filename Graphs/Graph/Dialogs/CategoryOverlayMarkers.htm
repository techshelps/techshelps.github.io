Category Overlay Markers

To display the Category Overlay Markers dialog, select the Multiple markers check box and click the Multiple markers button in the General dialog (accessible by double-clicking on any point marker in a graph) or in the Plot: General options pane of the Graph Options dialog [accessible by clicking Graph Options on the Format tab (ribbon bar) or by selecting Graph Options from the Format menu (classic menus)] .

Pattern. Click any of the Pattern buttons to display the  Marker Properties dialog, in which you can change the size, color, pattern, and style of markers on the graph.

Close. Click Close to accept your changes and return to the previous dialog.
