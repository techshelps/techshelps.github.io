Copy Axis Specs To

Click the Copy axis specs to button in any of the Axis options panes (i.e., General, Title, Scaling, Major Units, Minor Units, Scale Values, or Custom Units) of the Graph Options dialog to display the Copy axis specs to dialog. Use the options in this dialog to transfer properties (e.g., formatting, labeling, scaling) from one graph axis to another. Note that before clicking the Copy axis specs to button, the source axis whose properties you want to copy should be selected in the Axis drop-down box next to the Copy axis specs to button.

The Copy axis specs to dialog contains relevant options in the form of check boxes from each of the seven axis-related tabs in seven boxes bearing the same names. Any combination of options from all of the boxes can be selected for copying to another axis. The destination axis or axes to which the properties are to be copied should be selected under Copy to. After clicking the OK button and closing the dialog, the designated properties will be copied and displayed on the selected axis/axes.

General. The options under General are related to the copying of relevant axis-line-related features as specified in the Axis: General options pane of the Graph Options dialog. Click the Select All button or the Deselect All button to select or clear all of the check boxes simultaneously.

Scaling. The options under Scaling control axis-scaling characteristics to be copied from the source axis to the destination axis. Specifications of the properties are summarized in the Axis: Scaling options pane of the Graph Options dialog. Click the Select All button or the Deselect All button to select or clear all of the check boxes simultaneously.

Note that scale breaks, when added to a specific graph axis, are automatically posted on all axes parallel to that axis on the graph, but for the purposes of change they remain associated only with the axis on which they were originally defined. Scale breaks cannot be transferred to other axes using the Copy axis specs to dialog, and existing scale breaks displayed on any axis other than the axis on which they were originally defined will not be affected by scale changes made by Copy axis specs to or other operations.

Scale values. The options under Scale values are for copying relevant scale labeling characteristics (as specified in the Axis: Scale Values options pane of the Graph Options dialog) from the source axis to the destination axis. Click the Select All button or the Deselect All button to select or clear all of the check boxes simultaneously.

Title. The options under Title control the copying of axis titles and their characteristics (see also the Axis: Title options pane of the Graph Options dialog). Click the Select All button or the Deselect All button to select or clear all of the check boxes simultaneously.

Major units, Minor Units, Custom. The options under Major units, Minor units, and Custom control copying of tickmarks and gridlines from source axis to destination axis. Specifications of the properties that can be copied are summarized in the Axis: Major Units, Axis: Minor Units, and Axis: Custom Units topics (these options are located in the Graph Options dialog). Click the Select All button or the Deselect All button to select or clear all of the check boxes simultaneously.

Copy to. Under Copy to is a list of the axes available on the current graph to which the selected properties of the source axis can be copied. Select the axis (or multiple axes) to which you want to copy the selected properties and close the dialog by clicking the OK button. Click the Select All button or the Deselect All button to select or clear the entire list simultaneously. Note that if you want to copy axis properties to an axis that does not currently exist on the graph, the new axis must first be added to the graph using the Add new axis button located in the Axis: General options pane of the Graph Options dialog.  

OK. Click the OK button to return to the previous dialog after you have made your selections.

Cancel. Click the Cancel button to close this dialog, ignoring any changes you have made.
