Icon Scatterplots

Click the Picture markers button in the Plot: General options pane of the Graph Options dialog when an icon scatterplot has been created to display the Icon scatterplots dialog. This dialog contains options to modify the icons on the icon scatterplot.

Display icons. Select this check box to display the selected icons in the icon scatterplot.

Type. Change the type of icon on the scatterplot by selecting a different option from the Type list.

Properties. Click the Properties button to display dialogs to modify the icons.

Size.

Horizontal size, Vertical size. In these fields, specify the horizontal and vertical point sizes for the icons.

Use weight. Select this check box if you want to use a weight variable when creating the icon scatterplot.

Variable order. You can change the order of variables in this box by highlighting a variable and dragging it to a different position.

Original. Click the Original button to return the variables to the original position.

Picture markers. Click the Picture markers button to display the Picture Markers - Graphics File Images dialog.
