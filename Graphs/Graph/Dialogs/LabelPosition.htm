Label Position

When point labels are displayed in a scatterplot, right-click in the Graph Data Editor (accessible by right-clicking on the scatterplot and selecting Graph Data Editor from the shortcut menu) and select Label Position to display the Label Position dialog.

Plot. This field displays the name of the selected scatterplot.

Case. This field shows the case number of the cell in the scatterplot that was selected. To specify a different case, type in the case number or use the microscrolls. The adjacent field displays the label.

Move label. You can move the label to the right and/or up, in inches.

OK. After making the specification, click the OK button to close the dialog and return to the Graph Data Editor.

Cancel. Click the Cancel button to cancel any specifications made, close the dialog, and return to the Graph Data Editor.
