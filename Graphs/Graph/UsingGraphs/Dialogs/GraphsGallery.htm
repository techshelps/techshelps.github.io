Graphs Gallery

Click the Graphs Gallery button on any graph specification dialog to display the Graphs Gallery dialog. This dialog displays a "gallery" style menu of STATISTICA graphs and can be used to "switch" quickly and easily between the different groups of Graphs menu graphs. The following graphs are available in this dialog:

2D Graphs

Icon Graphs

3D Sequential Graphs

Categorized Graphs

3D XYZ Graphs

Multiple Graph Layouts

Matrix Graphs
