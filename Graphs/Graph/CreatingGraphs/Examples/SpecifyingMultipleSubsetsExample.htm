Specifying Multiple Subsets Example

Subset 1: Include if: v1<=10

Subset 2: Include if: v1>10 AND v2 = 'YES'

If this dialog is accessed by clicking the Mark Selected Subsets button (see the Advanced tabs of 2D Scatterplots and 3D Scatterplots), then all cases (observations) that meet the condition specified in Subset 1 (i.e., cases for which the value of variable 1 is less than or equal to 10) will be assigned a specific point marker in the graph (e.g., a star). All cases that meet the condition outlined in Subset 2 (i.e., cases for which the value of variable 1 is greater than 10 and the value of variable 2 is equal to 'YES') will be assigned a different point marker in the graph (e.g., a circle). In this way, the points in a single graph are subsetted or categorized according to some user-defined conditions.

If this dialog is accessed by clicking the Specify Subsets button (see Multiple Subset Method of Categorization), then the first category (i.e., small graph) will contain all cases (observations) that meet the condition specified in Subset 1 (i.e., cases for which the value of variable 1 is less than or equal to 10). The second category (i.e., small graph) will contain all cases that meet the condition outlined in Subset 2 (i.e., cases for which the value of variable 1 is greater than 10 and the value of variable 2 is equal to 'YES').
