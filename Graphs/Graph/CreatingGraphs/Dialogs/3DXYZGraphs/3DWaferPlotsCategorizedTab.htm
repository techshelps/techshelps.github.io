3D Wafer Plots - Categorized Tab

Graphical Analytic Techniques

Select the Categorized tab of the 3D Wafer Plots dialog to access the options described here.

X- and Y- Categories. Use the options in these group boxes to select the method of categorization for each of the selected grouping variables. (Note that grouping variables do not need to be selected if Multiple subset categories are selected.)

On. Select this check box to create a categorized 3D wafer plot. The On check box in the Y-Categories group box is disabled until you select the On check box in the X-Categories group box.

Categorization is used in two classes of graphs in STATISTICA: categorized graphs (e.g., Categorized Scatterplots) and graphs that include grouping or categorized variables (e.g., 2D Histograms, or 2D Box Plots). You can choose from the following methods of categorization; click the links for descriptions:

Integer mode

Unique values

Categories

Boundaries

Codes

Multiple subsets

Change Variable. Click the Change Variable button in either the X- or Y-Category group to display the Select Categorization Variable dialog.
