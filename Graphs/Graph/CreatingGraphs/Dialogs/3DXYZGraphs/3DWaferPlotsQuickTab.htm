Graphical Analytic Techniques

Use the options on the Quick tab of the 3D Wafer Plots dialog to specify the variables for X, Y, and Z

Variables. Click the Variables button to display the Select Variables for 3D Graph dialog, in which you select the variables to be plotted.

Show raw data points. Select this check box if you want the raw data points to be overlaid on the graph surface. (If the points are not located directly on the surface, then they will be "connected" to the surface by "spikes.")
