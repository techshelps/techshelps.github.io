Ternary Graphs - Quick Tab

Graphical Analytic Techniques

Select the Quick tab of the Ternary Graphs dialog to access options to specify the variables for either a 2D or 3D ternary scatterplot. More advanced options (including a greater selection of graph types) are available on the Advanced tab.

Graph Type. Select either a 2D or 3D ternary scatterplot to be plotted from the Graph Type list. Click the desired plot link below for a brief description of that type of graph. More graph types are available on the Advanced tab and the Appearance tab.

Variables. Click the Variables button to display the standard variable selection dialog, in which you can select the X-, Y-, Z-, and one or more V-axis variables for the graph. The selection that you make will then be displayed in the area of the dialog below the Variables button.
