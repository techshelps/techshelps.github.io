3D Scatterplots - Quick Tab

Graphical Analytic Techniques

Select the Quick tab of the 3D Scatterplots dialog to specify the variables to use in creating either a 3D scatterplot or a 3D space plot. For more options and graph types, see the Advanced tab.

Variables. Click the Variables button to display the standard variable selection dialog, in which you can select the x-, y-, and one or more z-axis variables for the graph. The selection that you make will then be displayed in the area of the dialog below the Variables button.

Graph Type. Select the type of 3D scatterplot to be plotted from the Graph Type list. Click the desired plot link below for a brief description of that type of graph. Note that additional graph types are available on the Advanced tab.
