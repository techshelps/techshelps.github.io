3D Contour Plots - Categorized Tab

Graphical Analytic Techniques

Select the Categorized tab of the 3D Contour Plots dialog to create a categorized 3D contour plot.

X and Y Categories. Use the options in these group boxes to select the method of categorization for each of the selected grouping variables. (Note that grouping variables do not need to be selected if Multiple subset categories are selected.)

On.  Select this check box to make a categorized 3D contour plot. The On check box in the Y Categories group box is disabled until you select the On check box in the X Categories group box.  

Categorization is used in two classes of graphs in STATISTICA: categorized graphs (e.g., Categorized Scatterplots) and graphs that include grouping or categorized variables (e.g., 2D Histograms, or 2D Box Plots). You can choose from the following methods of categorization; click the links below for descriptions:

Change Variable. Click the Change Variable button in either the X or Y category group to display the Select Categorization Variable dialog.
