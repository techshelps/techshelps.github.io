Plots

When creating a graph with multiple plots, such as, Histograms, Scatter Plots, or Means w/Error Plots, you can enable or disable the display of individual plots.

Once the plots are created and displayed, right-click anywhere in the plot area and select Plots from the shortcut menu to display the Plots dialog. In the window listing the plot variables you selected, clear the check box next to variable whose plot you do not want displayed, and select the check box for those plots you do want displayed. By default, all the variable check boxes are selected when you display the dialog.

OK. Click the OK button to re-display your plot(s) once you've selected or cleared the variable check boxes.

Cancel. Click the Cancel button to return to your plot without applying any selections you have made.
