Categorized Means with Error Plots - Appearance Tab

Graphical Analytic Techniques

Select the Appearance tab of the Categorized Means with Error Plots dialog to access options to specify the appearance of the graph. After the graph is created, you can further customize the arrangement and appearance of various components via the Graph Options dialog (accessible by selecting Graph Options from the Format menu when a graph is active, or by double-clicking in the background of a graph).

Graph style. STATISTICA has several built-in graph styles. The options in the Graph style group box are used to choose one of these styles for your graph and, if you so desire, save the selected style as default style. Refer to Graphics Styles for additional details.

Use graph style. Select the graph style (Normal, Matrix, or Ternary) you want to use from the Use graph style drop-down box. For more details on using and creating styles, see Graphics Styles Overview.

Set as default style. Select the Set as default style check box to set the currently specified graph style as the new default style. It will then be used for all graphs unless otherwise specified.

Get system default. If you no longer want the graph style that you set as the default style (see above), click the Get system default button to retrieve the system default graph style.

Document style. The document style is used to control the dimensions of the graph you want to create and the options in the Document style group box are useful when you want to print graphs (on paper of various sizes, etc.). These options are available before or after the graph has been created. STATISTICA has several built-in document styles from which you can choose for your graph (see Document Styles Overview).

Use document style. Use the Use document style drop-down box to select from a list of built-in document styles to use for the current graph. For more details on the document styles available, see Document Styles Overview.

Set as default document style. Select the Set as default document style check box to set the currently selected document style as the new default style. The document style will then be used for all graphs unless otherwise specified.

Get system default. If you no longer want the document style that you set as the default style (see above), click the Get System default button to retrieve the system default document style.
