2D Bar/Column Plots - Appearance Tab

Graphical Analytic Techniques

Select the Appearance tab of the 2D Bar/Column Plots dialog to access options to specify the appearance of the plot. After the graph is created, you can further customize the arrangement and appearance of various components via the Graph Options dialog (accessible by selecting Graph Options from the Format menu when a graph is active, or by double-clicking in the background of a graph).

Graph type. Select the type of bar/column plot to be plotted from the Graph type list.

Regular

Multiple

Graph style. Use the options under Graph style to specify which graph style to use when creating the specified bar/column plot.

Use graph style. Select the graph style you want to use from this box. For more details on using and creating styles, see Graphics Styles - Overview.

Set as default style. Select this check box to set the currently specified graph style as the new default style. It will then be used for all graphs unless otherwise specified.

Get System default. Click this button to retrieve the system default graph style.

Document style. Use the options under Document style to specify which document style to use when creating the specified plot.

Use document style. Select the document style you want to use from this box. For more details on the document styles available, see  Document Styles - Overview.

Set as default document style. Select this check box to set the currently selected document style as the new default style. The document style will then be used for all graphs unless otherwise specified.

Get System default. Click this button to retrieve the system default document style.
