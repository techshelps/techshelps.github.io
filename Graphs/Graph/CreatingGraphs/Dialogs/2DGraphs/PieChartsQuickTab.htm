Pie Charts - Quick Tab

Graphical Analytic Techniques

Select the Quick tab of the Pie Charts dialog to access the options described here. For more options, select the Advanced tab. Other specific pie layout, location, proportions, and exploding options are available in the Plot: Pies options pane of the Graph Options dialog.

Graph type. Select the type of pie chart to be plotted from the Graph type list. Click the desired plot link below for a brief description of that type of graph.

Counts

Values

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the variable(s) for the graph. If you select more than one variable for the pie charts, then a cascade of graphs (one for each selected variable) will be produced. The selection that you make will then be displayed in the area of the dialog below the Variables button.

Frequency Intervals. This option applies when the Counts type of pie chart is selected (see above), which displays frequency data. Use the options in this group box to select a method of categorization for the selected variable(s). Each of the methods is discussed in the Method of categorization topic. Note you can only select from four categories on this tab: Integer mode, Unique Values mode, Categories, and Codes. For additional pie categories, select the Advanced tab.

Explode slice. Select this option to "explode" a specified number of slices of the pie; see also the  Plot: Pies options pane of the Graph Options dialog for more details.
