Pie Charts - Categorized Tab

Graphical Analytic Techniques

Select the Categorized tab of the Pie Charts dialog to access options for creating categorized graphs. Categorization is used in two classes of graphs in STATISTICA: categorized graphs (e.g., Categorized Scatterplots) and graphs that include grouping or categorized variables (e.g., 2D Histograms, or 2D Box Plots).

X-Categories / Y-Categories.

On.  Select this check box to create a categorized scatterplot. The On check box in the Y-Categories group box is disabled until you select the On check box in the X-Categories group box.  

Select Integer mode, Unique values, or Categories to specify that method of categorization for each of the variables selected via the Change Variable button, or use the Boundaries, Codes, or Multiple subsets options.
