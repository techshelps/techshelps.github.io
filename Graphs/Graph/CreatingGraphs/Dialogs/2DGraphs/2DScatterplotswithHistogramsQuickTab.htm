2D Scatterplots with Histograms - Quick Tab

Graphical Analytic Techniques

Use the options on the Quick tab of the 2D Scatterplots with Histograms dialog to create a 2D scatterplot with histograms. For more options, select the Advanced tab.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the X and Y variables to be plotted in the component scatterplot graph. A histogram will also be produced for each variable (the X variable histogram will be placed above the scatterplot while the Y variable histogram will be placed to the right of the scatterplot in the compound graph display). The selection that you make will then be displayed below the Variables button.

Intervals for X-Histogram/Intervals for Y-Histogram. Use these options to specify a method of categorization of the values for each of the component histograms. Note that only three methods are available on the Quick tab: Integer mode, Unique Values mode, Categories, and Codes. More options are available on the Advanced tab.

Fit type. There is one fit type available for scatterplots and one fit type available for histograms on the Quick tab. Other fit types for histograms are available on the Advanced tab.

Scatterplot: Linear. Select this check box to superimpose a linear fit over the scatterplot.

Histograms: Normal. Select this check box to superimpose a normal fit over the histogram.
