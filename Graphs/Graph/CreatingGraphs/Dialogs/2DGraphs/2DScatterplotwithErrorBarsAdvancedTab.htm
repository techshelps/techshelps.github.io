2D Scatterplot with Error Bars - Advanced Tab

The Advanced tab of the 2D Scatterplots with Error Bars dialog contains a greater variety of options for the scatterplot.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the X and Y variables to be plotted in the scatterplot graph. The selection that you make will then be displayed in the area of the dialog below the Variables button.

Graph type. Select the type of 2D scatterplot to be plotted from the Graph type list. Click the type of the plot link listed below for a brief description of that type of graph.

Regular

Multiple

Fit. You can fit an equation to the points in the line plots by selecting one of the predefined functions in this box.
