Sequential/Stacked Plots - Advanced Tab

Graphical Analytic Techniques

The Advanced tab of the Sequential/Stacked Plots dialog contains a selection of graph types as well as graph styles.

Variables. Click the Variables button to display the standard variable selection dialog, in which you can select the variable(s) for the graph either one or two lists of variables can be selected, depending on the graph type. The selections that you make will then be displayed in the area of the dialog below the Variables button.

Graph type. Select the type of sequential/stacked plot to be plotted from the Graph type list. Click the desired plot link below for a brief description of that type of graph.
