Pie Charts - Advanced Tab

Graphical Analytic Techniques

Select the Advanced tab of the Pie Charts dialog to access the options described here. Other specific pie layout, location, proportions, and exploding options are available in the Plot: Pies options pane of the Graph Options dialog [accessible by clicking Graph Options on the Format tab (ribbon bar) or by selecting Graph Options from the Format menu (classic menus)].

Graph type. Select the type of pie chart to be plotted from the Graph type list. Click the desired plot link below for a brief description of that type of graph.

Counts

Values

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the variable(s) for the graph. If you select more than one variable for the pie charts, then a cascade of graphs (one for each selected variable) will be produced. The selection that you make will then be displayed in the area of the dialog below the Variables button.

Frequency Intervals. This option applies when the Counts type of pie chart is selected (see above), which displays frequency data. Use the options in this group box to select a method of categorization for the selected variable(s). Each of the methods is discussed in Method of Categorization.

Pie Legend. Select one of six formats for pie legends (Off, Percent, Values, Text labels, Text & percent, Text & values) to be placed in the graph. If you select Counts as the graph type, then selecting Text labels will use the category descriptions in the pie legend. Otherwise (for Values pie charts), the Text labels used will depend on the Pie labels (values) selected (e.g., if you select Case names as pie labels, then the Text labels will display the actual case names instead of Case1, Case2, etc.; if you select a variable in the Pie labels option and then choose Text labels, then STATISTICA will use the text labels of the selected variable in the pie legend).

Explode slice. Select this option to "explode" a specified number of slices of the pie; see also the  Plot: Pies topic (these options are located in the Graph Options dialog) for more details.

Type. You can change the appearance of the pie by selecting either the 2-dimensional (2D) or 3-dimensional (3D) display option button; see also the Plot: Pies topic (these options are located in the Graph Options dialog).

Shape. The pie shape can be either circular (select Circle) or elliptical (select Ellipse). However, selecting the Ellipse option button may cause the slices to be disproportionate in size (i.e., the slices that follow radii close to the vertical axis of the screen or paper will look relatively larger than their real size as compared to the "horizontal" slices which will be "squeezed" in perspective and will appear smaller); see also the Plot: Pies topic (these options are located in the Graph Options dialog).

Pie Labels (Values). You can label the pie with Case Names (select any of the Text options in the Pie Legend list) or the values (text or numeric depending on the setting in the Pie Legend list) of a specified variable. If you do not want any pie labels, select Off.
