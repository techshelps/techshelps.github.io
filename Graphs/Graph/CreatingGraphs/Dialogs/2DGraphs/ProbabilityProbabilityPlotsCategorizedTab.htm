Probability-Probability Plots - Categorized Tab

Graphical Analytic Techniques

Select the Categorized tab of the Probability-Probability Plots dialog to access options to create categorized graphs. Categorization is used in two classes of graphs in STATISTICA: categorized graphs (e.g., Categorized Scatterplots) and graphs that include grouping or categorized variables (e.g., 2D Histograms, or 2D Box Plots).

X-Categories / Y-Categories.

On. Select the On check box to enable the options in these boxes.

Select Integer mode, Unique values, or Categories to specify that method of categorization for each of the variables selected via the Change Variable button, or use the Boundaries, Codes, or Multiple subsets options. Note that the categorization method and as well as the (default) formatting of the labels for the categories in the graph can be changed after the graph has been created via the Categorization tab of the respective Graph Options dialog.

Layout. Select the type of layout for the graph(s).

Separate. Select this option button to produce a Separate plot layout (where each subset of cases is displayed in a separate graph) for the categorized plots.

Overlaid. Select this option button to produce an Overlaid plot layout (where all subsets are overlaid in one graph and identified by patterns and colors) for the categorized plots.
