Scatter Icon Plots - Quick Tab

Graphical Analytic Techniques

Use the options on the Quick tab of the Scatter Icon Plots dialog to specify the necessary parameters in creating a scatterplot icon.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the variable(s) for the graph. You need to specify the X and Y variables as well as the icon variables. The weight variable is optional. The selections you make will then be displayed in the area of the dialog below the Variables button.

Graph type. Select the type of icon plot from the Graph type list. Click the desired icon plot link below for a brief description of that type of graph.
