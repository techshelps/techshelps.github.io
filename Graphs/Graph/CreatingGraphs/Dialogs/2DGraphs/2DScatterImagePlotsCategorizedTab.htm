2D/3D Scatter Image Plots - Categorized Tab

Graphical Analytic Techniques

The Categorized tab of the Scatter Image Plots dialog contains options to create categorized graphs. Categorization is used in two classes of graphs in STATISTICA: categorized graphs (e.g., Categorized Scatterplots) and graphs that include grouping or categorized variables (e.g., 2D Histograms, or 2D Box Plots).

X-Categories / Y-Categories.

On. Select the On check box to create a categorized Means with Error Plots. Note that the other options in the X- and Y-Categories boxes are disabled unless this box is checked.

Select Integer mode, Unique values, or Categories to specify that method of categorization for each of the variables selected via the Change Variable button, or use the Boundaries, Codes, or Multiple subsets options.
