2D Scatterplot with Error Bars - Quick Tab

Use the options on the Quick tab of the 2D Scatterplot with Error Bars dialog to quickly specify the variable and type of scatterplot with error bars to create. This tab has a limited selection of graph types, categories, and fit types. For more options, use the Advanced tab.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the variable(s) to be plotted. When you select more than one variable for the regular scatterplot, then a sequence of graphs is produced (one for each of the variables). The selection that you make is then displayed in the area of the dialog below the Variables button.

Graph type. Select the type of 2D scatterplot to be plotted from the Graph type list. Click the type of the plot link listed below for a brief description of that type of graph.

Regular

Multiple

Fit type.

Linear. Select this check box to fit a linear equation to the data in the scatterplot. Other fit types are available on the Advanced tab.
