2D Bar/Column Plots - Quick Tab

Graphical Analytic Techniques

Select the Quick tab of the 2D Bar/Column Plots dialog to access options to quickly create a bar/column plot. See the Advanced tab for more options.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the variables to be included in the analysis.

Graph type. Select the type of bar/column plot to be plotted from the Graph type list.

Regular

Multiple

Orientation. Select the orientation of the variables in the bar/column graph.

Vertical. If you select the Vertical option button, the values of the selected variable(s) will be plotted as columns (against the y-axis).

Horizontal. If you select the Horizontal option button, the values will be plotted as horizontal bars (against the x-axis).
