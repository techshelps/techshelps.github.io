Missing and 'Out of Range' Data Plots - Quick Tab

Graphical Analytic Techniques

Use the options on the Quick tab of the Missing and 'Out of Range' Data Plots dialog to specify a missing and 'out of range' data plot quickly using the system defaults. More options are available on the Advanced tab.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the variable(s) to be plotted. The variable(s) that you select here will be plotted on the x-axis in the graph (against Cases on the y-axis).
