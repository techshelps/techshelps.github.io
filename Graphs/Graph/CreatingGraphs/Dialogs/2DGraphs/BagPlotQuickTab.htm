Bag Plots - Quick Tab

Graphical Analytic Techniques

Use the options on the Quick tab of the Bag plot dialog to specify the variable and type of bag plot to create. This tab has a limited selection of options. For more options, select the Advanced tab.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the variable(s) to be plotted. When you select more than one variable for the regular bag plot, a sequence of graphs is produced (one for each of the variables). The selection that you make is then displayed in the area of the dialog below the Variables button. 

Graph type. Select the type of bag plot to be plotted from the Graph type list. Click the type of the plot link listed below for a brief description of that type of graph.
