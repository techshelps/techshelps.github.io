Customize Graph Menu

Graphical Analytic Techniques

Select Customize List from the Graphs - Graphs of Block Data menu to display the Customize Graph Menu dialog, in which you can create a new custom graph as well as edit, delete, or rename an existing graph. You can also hide the custom graph options on the Graphs of Block Data submenu. Select the graph you want to modify from the graphs listed. Once you have specified the changes/additions you want to make, click Close to return to the spreadsheet.

Edit. Click this button to edit the selected graph. The Select Graph dialog is displayed, in which you can change the graph category, type and subtype for the currently selected graph.

New. Click this button to create a new custom graph. The Specify Name for Custom Graph dialog is displayed, in which you can name the graph. After you click OK on this dialog, the Select Graph dialog will be displayed, in which you specify the graph category, type, and subtype for the new graph.

Delete. Click this button to delete the selected graph from the Graphs of Block Data menu. You will be prompted to verify that you want to delete the graph.

Rename. Click this button to rename the selected graph. The Specify Name for Graph dialog is displayed, in which you specify a new name for the graph.

Hide the custom graph options for block graphs. Select this check box to hide (remove) the Custom Graph... options from the Graphs of Block Data submenu. If this check box is cleared (the default), you will see four Custom Graph options (Custom Graph from Block by Column, Custom Graph from Block by Row, Custom Graph for Entire Column, and Custom Graph for Entire Row) on the Graphs of Block Data submenu.
