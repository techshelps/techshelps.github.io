3D Raw Data Plots - Quick Tab

Graphical Analytic Techniques

Select the Quick tab of the 3D Raw Data Plots dialog to specify the variables to use in your graph, as well as to select from two graph types. To select from a greater variety of graph types, use the Advanced tab.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the variable(s) for the graph. The selection that you make will then be displayed in the area of dialog below the Variables button.

Graph type. Select one of two types of raw data charts to be plotted from the Graph type list on this tab. Click the desired plot link below for a brief description of that type of graph. More graph types are available on the Advanced tab.
