3D Raw Data Plots - Advanced Tab

Graphical Analytic Techniques

Select the Advanced tab of the 3D Raw Data Plots dialog to specify the variables to use in your graph, as well as select from a variety of graph types.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the variable(s) for the graph. The selection that you make will then be displayed in the area of the tab below the Variables button.

Graph type. Select the type of raw data chart to be plotted from the Graph type list in this dialog. Click the desired plot link below for a brief description of that type of graph.
