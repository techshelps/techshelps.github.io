3D Bivariate Histograms - Quick Tab

Graphical Analytic Techniques

Select the Quick tab of the 3D Bivariate Histograms dialog to specify the variables to use in your graph, as well as to select interval methods for the X and Y variables. To select from a greater variety of interval methods, click the Advanced tab.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the X and Y variables for the graph. The selection that you make will then be displayed in the area of the dialog below the Variables button. Variables do not need to be selected if the Multiple Subsets categories are analyzed (instead of variables); you can select Multiple Subsets on the Advanced tab.

Intervals (for X and Y Variables). Use the options in these group boxes to choose the method of categorization for each of the selected variables. Each of the methods is discussed in the Method of categorization topic. Note that only four categorization methods are available on this tab: Integer mode, Unique Values mode, Categories, and Codes.  For more options, select the Advanced tab.
