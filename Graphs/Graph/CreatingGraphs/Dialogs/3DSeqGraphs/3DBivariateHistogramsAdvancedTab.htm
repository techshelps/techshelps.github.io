3D Bivariate Histograms - Advanced Tab

Graphical Analytic Techniques

Select the Advanced tab of the 3D Bivariate Histograms dialog to specify the variables to use in your graph as well as select interval methods for the X and Y variables and statistics to include in the plot.

Variables. Click the Variables button to display the standard variable selection dialog, in which you select the X and Y variables for the graph. The selection that you make will then be displayed in the area of the dialog below the Variables button. Variables do not need to be selected if the Multiple Subsets categories are analyzed (instead of variables).

Intervals (for X and Y Variables). Use the options in these group boxes to choose the method of categorization for each of the selected variables. Each of the methods is discussed in the Method of categorization topic.

Statistics. You can include a variety of statistics as footnotes in the graph by selecting one or more of the statistics listed in this group box.

Total Count. Select this check box to include the total number of cases used in the plot.

Chi-square. Select this check box to include the Pearson Chi-square statistic for goodness-of-fit.
