Graphs of Block Data - Line Plot: Block Rows

Select a block of data within a spreadsheet, and either select Graphs of Block Data - Line Plot: Block Rows from the Graphs menu, or right-click on the block of data and select Graphs of Block Data - Line Plot: Block Rows from the resulting shortcut menu. A 2D line plot is created for each row of the block, and is then plotted in a separate graph window.

To create a line plot for each column of the block, select Graphs of Block Data - Custom Graph from Block by Column from the spreadsheet shortcut menu.

If you want to create a multiple line plot to allow for comparisons between the values in block rows,

  1. select Graphs of Block Data - Custom Graph from Block by Row from the spreadsheet shortcut menu,

  2. click the More button on the Select Graph dialog,

  3. and select the Single Window (Overlaid) option button.

Note that you can also create line plots for entire variables (columns) or cases (rows) by selecting Graphs of Block Data - Custom Graph for Entire Column or Graphs of Block Data - Custom Graph for Entire Row.

For more information on 2D line plots, see Stats Graphs - 2D Line Plots or Stats Graphs - 2D Line Plots Case Profiles.
