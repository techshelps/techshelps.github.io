Graphs of Block Data - Histogram: Block Columns

When you select a block of data within a spreadsheet and select Graphs of Block Data - Histogram: Block Columns from the Graphs menu (or right-click on the selected block of data and select Graphs of Block Data - Histogram: Block Columns from the resulting shortcut menu), a 2D histogram with a fitted normal curve is created for each column of the block, and is then plotted in a separate graph window.

To create a histogram for each row of the block, select Graphs of Block Data - Custom Graph from Block by Row from the Graphs menu.

If you want to create a multiple histogram to allow for comparisons between the frequency distributions of values in block columns,

  1. select Graphs of Block Data - Custom Graph from Block by Column from the Graphs menu,

  2. click the More button on the Select Graph dialog,

  3. and select the Single Window (Overlaid) option button.

Note that you can also create histograms for entire variables (columns) or cases (rows) by selecting Graphs of Block Data - Custom Graph for Entire Column or Graphs of Block Data - Custom Graph for Entire Row from the Graphs menu.

For more information on 2D histograms, see Graphs of Input Data - Histogram by ... or 2D Histograms.
