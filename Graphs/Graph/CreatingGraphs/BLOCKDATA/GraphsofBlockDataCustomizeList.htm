Graphs of Block Data - Customize List

As with most features of STATISTICA, the list of Graphs of Block Data is fully customizable. Select Customize List from the Graphs - Graphs of Block Data menu to display the Customize Graph Menu dialog, from which you can remove, rename, or edit the currently listed graphs, as well as add new (user-defined) graphs.

For example, if you want to include a normal fit on the histograms created when you select Histogram: Block Columns from the Graphs - Graphs of Block Data menu, select Histogram: Block Columns in the Customize Graph Menu dialog, click the Edit button, and select Normal Fit in the Select Graph SubType box.
