Graphs of Block Data - Line Plot: Entire Columns

When you select a block of data within a spreadsheet, right-click, and select Graphs of Block Data - Line Plot: Entire Columns  from the resulting shortcut menu, a 2D line plot is created for each variable (column) represented in the block (e.g., not just the values highlighted in the block but all the values in the entire column), and is then plotted in a separate graph window.

To create a line plot for each (case) row represented in the block, select Graphs of Block Data - Custom Graph for Entire Row from the spreadsheet shortcut menu..

If you want to create a multiple line plot to allow for comparisons between the values in the columns,

  1. select Graphs of Block Data - Custom Graph for Entire Column from the spreadsheet shortcut menu,

  2. click the More button on the Select Graph dialog,

  3. and select the Single Window (Overlaid) option button.

For more information on 2D line plots, see Stats Graphs - 2D Line Plots or Stats Graphs - 2D Line Plots Case Profiles.
