Graphs of Block Data - Custom Graph for Entire Row

In addition to the default graph types available from the Graphs of Block Data submenu (accessible from the Graphs menu or by selecting a block within a spreadsheet and right-clicking to display a shortcut menu), you can create customized graphs, which are created using each variable (column) represented in the block (e.g., not just the values highlighted in the block but all the values in the entire column).

Select Graphs of Block Data - Custom Graph for Entire Row from the Graphs menu to display the Select Graph dialog, which contains a variety of options to create customized graphs.
