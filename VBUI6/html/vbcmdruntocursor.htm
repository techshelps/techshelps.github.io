Run to Cursor Command (Debug Menu)

   

When your application is in design mode, use Run To Cursor to select a statement further down in your code where you want execution to stop. Your application will run from the current statement to the selected statement and the current line of execution margin indicator, appears in the Margin Indicator bar.

You can use this command, for example, to avoid stepping through large loops.

Keyboard shortcut:  CTRL+F8.
