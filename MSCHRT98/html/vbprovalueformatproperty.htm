ValueFormat Property

      

Returns or sets the format used to display the label as a value.

Syntax

object.ValueFormat [ = format]

The ValueFormat property syntax has these parts:

Part Description
object An object expression that evaluates to an object in the Applies To list.
format String. Describes the format used to display a label as a value.

Remarks

Use the DataPointLabel object's Component property to change the label type.
