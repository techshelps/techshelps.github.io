BrushHatch Constants

VtBrushHatch provides valid brush types if VtBrushStyle is set to VtBrushStyleHatch.

Constant Description
VtBrushHatchHorizontal Horizontal hatch lines
VtBrushHatchVertical Vertical hatch lines
VtBrushHatchDownDiagonal Down diagonal hatch lines
VtBrushHatchUpDiagonal Up diagonal hatch lines
VtBrushHatchCross Cross hatch lines
VtBrushHatchDiagonalCross Diagonal cross hatch lines
