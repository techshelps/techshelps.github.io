Limit Property

      

Returns or sets the joint limit, in points, of the line.

Syntax

object.Limit [ = joint ]

The Limit property syntax has these parts:

Part Description
object An object expression that evaluates to an object in the Applies To list.
joint Single. A joint limit as a multiple of the line width. If two lines meet at a sharp angle, a mitered join results in a corner point that extends beyond the actual corner. If the distance from the inner join point to the outer join point exceeds the value in this variable, the join automatically changes to a bevel.
