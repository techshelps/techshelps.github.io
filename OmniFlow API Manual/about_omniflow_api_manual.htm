About OmniFlow API Manual

The OmniFlow API Manual provides you the information on following sections:

  • About Newgen OmniFlow- This section provides you an overview about the workflow. It also provides the details about the product- "OmniFlow Server", its features, terminology used and the Server architecture.

  • Getting Started: This section is broadly classified into two parts. Firstly, the customization with external application, which explains the basic steps to start. Server terminology, system requirements and installations and configuration required for getting started are also explained.

The second part includes, the procedure and a sample code for using the API calls.

  • Classification of Functions: This section is divided into following parts- Connection Functions, Process Control Functions, Activity Functions, Process Status Functions, Activity Status Functions, Worklist Functions, Administration Functions and Miscellaneous Functions. It includes the detailed description of XML input / output parameters and error messages.

  • Sample Codes: This section includes the step-wise description of the particular calls which are bare essential for working with the OmniFlow Server.

  • External Interface: This section includes the details for getting the external data along with the sample code.

  • Trouble Shooting and Debugging: This section includes the details for trouble shooting and debugging.

  • Glossary: This section includes the description of all the terminologies used in the API Manual alphabetically.

  • Appendix: This section includes the list of all the API functions used with their description. It also includes the list of Error Messages and Action Ids and their corresponding action listing.
