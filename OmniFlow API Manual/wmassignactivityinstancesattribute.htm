WMAssignActivityInstancesAttribute

This call assigns the proper attribute to set of activity instances within a process definition that match the specific filter criterion.

You can view the XML input and output parameters, which have to be passed for fetching activity instance attribute. There is a list of errors and warning that can occur in the execution.
