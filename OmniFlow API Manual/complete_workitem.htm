Complete Workitem

This command allows a workflow participant to tell the WFM Engine that a work item has been completed. The work can be route to the next workstep in the process.

For completing the workitem:

 

Step 1: Call WMCompleteWorkItem.

 

View the call details.

Result

The workitem is completed and can be routed to the next workstep.
