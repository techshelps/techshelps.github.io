Fetching Worklist

After starting the process instance the users can fetch their respective worklist using the WMFetchworkitem call.

 

Step 1 Call WMFetchworkitem call

View the call details for fetching the worklist.

Result

Returns the list work item from the worklist on the login user that met the selection criterion. This fetch call will return subsequent work items after every call in batches.
