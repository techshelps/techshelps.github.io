WMAssignProcessInstancesAttribute

This call assigns the proper attribute to a set of process instances within a process definition. This command tells the WFM Engine to assign an attribute, or to change an attribute or to change the values of an attribute of a set of process instances within a named process definition.

You can view the XML input and output parameters, which have to be passed for assigning process instance attribute. There is a list of errors and warning that can occur in the execution.
