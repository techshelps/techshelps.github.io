Activity Control Functions

Activity Control Functions can be defined as those, which change the operational state of one or more activity instances. These API calls are intended for use by the WFM end user. However, some of the API calls, or parameters within some of the API calls, may affect multiple users and would normally be restricted to the use of a process administrator.

This section includes the description of the following calls:

WMGetActivityInstance

Returns the record of a specific activity instance.

WMFetchActivityInstances

Returns all the activity instance depending upon the filter criterion and batch size.

WMFetchActivityInstanceAttributes 

 

Returns all the activity instance attributes depending upon the filter criterion.

WMGetActivityInstanceAttributeValue

Returns the value, type and length of an activity instance attribute specified.

WMAssignActivityInstanceAttribute

Changes the value of the attributes of a activity instance
