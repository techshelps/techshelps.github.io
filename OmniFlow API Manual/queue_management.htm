Queue Management

Queue management calls includes the queue addition, deletion, fetching queue list, fetching queue properties, changing properties and setting the preferences for the queue list.

 

This section includes the details for the following calls:

WMAddQueue

Adds the queue

WFDeleteQueue

Deletes the queue

WMGetQueueList

Fetches the list of queue

WFGetQueueProperty

Fetches the specific queue properties.

WMChangeQueueProperty

Changes the specific queue properties.

WFSetPreferredQueueList

Sets the queue preferences for the user

WFGetPreferredQueueList

Returns the queue preferences for the user.
