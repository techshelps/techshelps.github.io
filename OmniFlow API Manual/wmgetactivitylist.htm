DataFlag

BatchInfo

It is flag which specifies whether to fetch the data or not.

'Y' or "N'

SortOrder

BatchInfo

It is the order of sorting for the records.

"A" for ascending.

"D" for Descending

OrderBy

BatchInfo

It is the field on which the sorting has to be done.

1 = ActivityID

2 = Activity Name

3 = Number of Workitems

4 = Number Of Delayed Workitems

5 = Process Name

LastValue

BatchInfo

 It is the activity name of the last value of the fetched activity list.

 

LastIndex

BatchInfo

It is the ID of the last item of the fetched activity list

 

LastProcessDefinitionId

BatchInfo

 It is the Id of process whose activity is the last activity of the fetched activity list.

 

NoOfRecordsToFetch

BatchInfo

 It is the batch size for the activity to be fetched.

 

ProcessName

Root

It is the name of the Process to be given only if ProcessDefinitionID is 0 or none.

 

ActivityType

Filter

It is the type of the  activity.

1=Introduction

2=Exit

3=Discard

4=Hold

5 =Distribute

6=Collect

7=Rule

8=Query

10=Custom

11=Expiry

18=SubProcess
