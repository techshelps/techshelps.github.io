WFReferWorkitem

The Workitem currently picked by a User can be referred to another user . The Workitem is fix assigned to the Target User and locked by in suspended  state by the Original User.

You can view the XML input and output parameters, which have to be passed for referring workitems. There is a list of errors and warning that can occur in the execution.
