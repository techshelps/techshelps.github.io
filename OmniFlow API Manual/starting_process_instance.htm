Starting Process Instance

In order to initiate a process instance into workflow, you need to start the process instance.

Step1: Call WMStartProcessInstance for starting the process instance.

 

View the Function details for starting the process instance

Result

If the call is successful, then a process instance id is returned to be used in the subsequent calls. The first activity (s) of the process will be started.
