External Interfaces

System uses ImageWorks as default program for processing any workstep but system integrators can write customize programs and can hook them for specialized processing. ImageWorks property pages are developed using this framework.

What is an External Interface?

External Interfaces are user defined applications which perform some specific tasks) for instance, executing an external program, launching user defined applications), printing / faxing/ emailing a document and so on. You can define these applications and include them in the process.

Properties of these applications can be defined for any workstep, or can be defined globally and can be associated with the workstep depending upon the application.

The different types of external applications are:

  • External programs

  • Archive

  • PrintFaxEmail

  • WorkDesk
