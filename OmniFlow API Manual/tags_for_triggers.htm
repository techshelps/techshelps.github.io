Tags for Triggers

If you are importing an external interface, there can be any trigger associated with it. In that case the while fetching the external data the associated trigger definition is also provided. The trigger definition would contain the-"TriggerIndex","TriggerName", "TriggerType" and the data according to the trigger type provided by the user.

 

Note:

The trigger tag can vary according to the user's external data.
