rdoErrors Property

      

Returns a specific member of an rdoErrors collection either by position or by key.

Syntax

object.rdoErrors(index)

The rdoErrors property syntax has these parts:

Part Description
object An object expression that evaluates to an object in the Applies To list.
index Required. An expression that specifies the position of a member of the collection. If a numeric expression, index must be a number from 1 to the value of the collection's Count property. If a String expression, index must correspond to the key argument specified when the member referred to was added to the collection.

Remarks

This property is used to access collections of rdoErrors objects.  For example, the rdoEngine object has an rdoErrors property which is the collection of rdoErrors objects that apply to the rdoEngine object.
