Deleting a Workflow Step
 

UII Administrator can use the UII Workflow management features to delete a workflow step.

To delete a workflow step

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings > UIIWorkflow Steps.
  3. The UII Workflow Steps list page appears, as shown in the following illustration. Select the workflow step that you want to delete, and then click Delete. The selected workflow step is removed from the workflow steps list.
