Deploying Required Server Artifacts
 

To deploy the server artifacts

  1. Copy the following run-time artifacts to a designated folder on the Citrix Server computer:
    • Microsoft.Uii.Csr.CitrixIntegration.exe
    • Microsoft.Uii.Csr.CitrixIntegration.exe.config
    • Microsoft.Uii.Common.dll
    • Microsoft.Uii.Common.Configuration.dll
    • Microsoft.Uii.Common.TypeProvider.dll
    • Microsoft.Uii.Common.Logging.dll
    • Microsoft.Uii.Common.Providers.dll
    • Microsoft.Uii.Common.Logging.Providers.dll
    • Microsoft.Uii.Common.Listeners.dll
    • Microsoft.Uii.Csr.Core.dll
    • Microsoft.Uii.AifServices.dll
    • Microsoft.Uii.Csr.Win32Api.dll
    • Microsoft.Uii.Csr.Browser.Web.dll
  2. Copy the required adapter assemblies, if there are any, and their respective dependencies. The following are some examples:
    • Microsoft.Uii.HostedApplicationToolkit.DataDrivenAdapter.dll (class: DdaAutomationAdapter)
    • Microsoft.Uii.Samples.Citrix.ApplicationAdapter.dll (class:SampleCitrixAdapter)
