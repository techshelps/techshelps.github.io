Searching for a Workflow Step
 

UII Administrator can use the UII Workflow management features to search for a workflow step.

To search for a workflow step

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings > UII Workflow Steps.
  3. The UII Workflow Steps list page appears, as shown in the following illustration. In the Search for records box, type the name of the workflow step that you want to search for, and then click the Search icon.


    If the workflow step exists in the database, the search results show the workflow step details. If the search cannot locate the workflow step, it returns the following error message: "No workflow step records are available in this view."
