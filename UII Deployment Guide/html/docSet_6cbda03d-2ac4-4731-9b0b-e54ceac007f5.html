Document Content Overview
 

The UII Deployment Guide includes the following topics:

  • UII Architecture and Components – Provides an overview of the UII components and describes how they interact. The topic is intended to be used as a quick reference.
  • Preparing for UII Installation – Provides information about prerequisite software and the required pre-installation setup and configurations.
  • Installing UII Components – Provides step-by-step guidelines for installing the UII components and verifying their installation.
  • Post Installation Procedures – Provides information about post-installation configuration.
  • UII Administration – Provides essential administration tasks for hosting and managing applications and workflows.
  • Appendix A – Provides the information and steps to migrate from a previous version to the current version (UII and CCA R3 for Microsoft Dynamics CRM 2013 and Microsoft Dynamics CRM Online). The appendix also describes client requirements, server requirements, and the optional components.
