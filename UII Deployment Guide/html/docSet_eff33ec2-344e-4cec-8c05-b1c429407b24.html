Modifying a Workflow
 

UII Administrator can use the UII Workflow management features to modify a workflow.

To modify a workflow

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings > Workflow.
  3. The Workflow list page appears. Double-click the name of the workflow that you want to modify.
  4. The Workflow Information page appears. Change the information as necessary, and then click Save or Save and Close.
    Note:
    The More Actions menu includes options to share the workflow with other teams and deactivate or activate the workflow. For more information and detailed instructions, see Managing a Workflow.
