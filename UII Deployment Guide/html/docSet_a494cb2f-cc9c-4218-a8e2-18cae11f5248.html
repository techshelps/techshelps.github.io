Mapping a Workflow to a User/Team
 

UII Administrator can use the UII Workflow management features to map a workflow to a user or team.

To map a workflow to a user or a team

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings > Workflow.
  3. The Workflow list page appears. Double-click the name of the workflow.
  4. The Workflow Information page appears. In the action pane, click Sharing.
  5. The Who would you like to share the selected uii workflow with? dialog appears, as shown in the following illustration. Click AddUser/Team, and then click OK.


  6. The Look Up Records dialog appears. Type the information that you are looking for in the Search for records box, and then click the Search icon.
  7. Select the record, and then click OK. The selected workflow is mapped to the user or team.
