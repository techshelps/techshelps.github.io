Using the Installer to Install the UII Solution Package
 

This section explains how to install the UII Solution package.

Note:
The UII Solution package can be installed on your computer and has to be imported onto the server (for both online and on-premises). For more information, see Importing the UII Solution Package.


To install UII Solution package

  1. Complete steps 1–3 of the procedure, Using the Installer to Install UII Client Components.
  2. The Change, repair, or remove installation page appears. Click Change, as shown in the following illustration.


  3. The Custom Setup page appears. Click the down arrow next to UII Solution, and then click Will be installed on local hard drive. Click Next.


  4. On the Pre-Requisites page, click Next.
  5. The Ready to change User Interface Integration for Microsoft Dynamics CRM page appears, as shown in the following illustration. Click Change.


  6. Click Finish to install the files to the default location, at [Install Directory]\Microsoft Uii\UII Solution.
  • For more information about importing the UII solution to the server, see Importing the UII Solution Package.
