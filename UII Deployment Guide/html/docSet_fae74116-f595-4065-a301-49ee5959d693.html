Client Requirements
 

The following software must be installed on each client computer:

  • .NET Framework 4.0 or later – You can download the .NET Framework 4 from the .NET Framework Developer Center on MSDN.
  • Microsoft Visual Studio® 2010 Professional Edition – Visual Studio is required to run the UII Visual Studio components. You can download a trial version of Visual Studio from the Microsoft Visual Studio page.
  • Windows Identity Foundation (WIF) framework – WIF is required to externalize user access from applications via claims; it also reduces development effort by providing pre-built security logic and integrated .NET Framework tools. Users can benefit from single sign-on and seamless collaboration across organizational boundaries. You can download WIF from the MSDN Evaluation Center.
