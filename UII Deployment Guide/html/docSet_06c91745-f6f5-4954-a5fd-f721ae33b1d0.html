Modifying a Non-Hosted Application
 

UII Administrator can use the UII Non-Hosted Application management features to update any information related to a non-hosted application.

To modify non-hosted application information

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings > UII Non-Hosted Application.
  3. The UII Non-Hosted Application page appears. To select a non-hosted application to modify, double-click the name of the application in the right pane.
  4. The application details page appears, as shown in the following illustration. Change the information to match your requirements, and then click Save or Save and Close
