Creating a Non-Hosted Application
 

UII Administrator can use the UII Non-Hosted Application management features to create a new non-hosted application.

To create a non-hosted application

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings > Non-Hosted Application.
  3. The UII Non-Hosted Application page appears. In the action pane, click New. The New page appears, as shown in the following illustration.


  4. Under General on the New page, type the name of the application in the Name box, select the owner in the Owner box, and then type the command that starts the application in the Command Line box.
  5. In the left navigation pane, click Notes, and then type any additional information about the application.
  6. Click Save or Save and Close.
