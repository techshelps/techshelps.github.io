Data Migration from a Previous Version of UII
 

This section guides the users to migrate from Release 1 to Release 2. This process involves the following steps:

  1. Create a Backup Copy of the Data
  2. Install the UII Server Components
  3. Delete the UII Entities of a Previous Version
  4. Delete UII Settings from the Site Map
  5. Import the Solution
  6. Import the Data
