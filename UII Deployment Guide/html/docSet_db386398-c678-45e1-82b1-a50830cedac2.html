Involved Technologies
 

Users of this guide should be familiar with the following technologies:

  • Microsoft® Dynamics CRM 2013 or Microsoft Dynamics CRM Online Fall '13
  • .NET Framework 4.0
  • Operating systems:
    • Windows Vista® with SP2
    • Windows 7
    • Windows 7 with SP1
    • Windows Server® 2008 R2
  • Microsoft® Visual Studio® 2010 Professional development system

If users are working in a multiple-developer environment, they should also be familiar with their source code management system and build process.
