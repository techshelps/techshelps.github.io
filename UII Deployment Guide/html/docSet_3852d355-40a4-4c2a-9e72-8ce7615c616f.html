Mapping a Workflow Step to a User or Team
 

Mapping a workflow step to a user or a team is similar to mapping an application or a workflow to a user or team.

To map a workflow step to a user or team

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings > UIIWorkflow Steps.
  3. The UII Workflow Steps list page appears. Select the workflow step that you want to map, and then click Share.
  4. Repeat steps from 5–7 of Mapping a Hosted Application to a User or Team.
