Managing Actions for a Hosted Application
 

A UII administrator can use the UII Hosted Application management features to manage the hosted application mapping for an action.

To change the hosted application mapping for an action

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings.
  3. Click UII Hosted Application. The list of records page appears.
  4. Double-click the name of a hosted application.
  5. In the left navigation pane, click UIIActions, and then double-click an action in the right pane. The UII Action page appears, as shown in the following illustration. Click the Search icon next to the Hosted Application box.


  6. The Look Up Record dialog appears, as shown in the following illustration. Type the search criteria in the Look for box, and then click the Search icon.


  7. Select the record, and then click OK. The action is mapped to the selected hosted application, as shown in the following illustration.


  8. In the UII Action action pane, click Save or Save and Close.
