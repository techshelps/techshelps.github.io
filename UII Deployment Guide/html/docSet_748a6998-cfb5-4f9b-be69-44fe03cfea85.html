Import the Data
 

The data as a part of the backup data performed through the Create a Backup Copy of the Data process is imported to the server in the form of .csv files.

To import the data

  1. Open the .xls file that you exported during the Create a Backup Copy of the Data process.
  2. Save the .xls file to .csv file format.
  3. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  4. In the left navigation left pane, click Settings > UII Settings, and then open a feature to import its data. For example, open UII Hosted Applications to import Hosted Application.csv file.
  5. The Hosted Applications page appears, as shown in the following illustration. In the action pane, click Import Data.


  6. The Upload Data File dialog appears, as shown in the following illustration. Click Browse to locate the.csv file (recommended format). Select the file, and then click Next.


  7. The Review settings and Import Data dialog appears, as shown in the following illustration. Click Submit.


  8. The Data Submitted for Import dialog appears, as shown in the following illustration. Click Finish.


The process to import data of UII hosted applications, non-hosted applications, workflows, workflow steps, and so on, is similar to the preceding procedure.
