Searching for a Hosted Application
 

The UII administrator can use the UII search function to find a hosted application.

To search for a hosted application

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings > UII Hosted Application. The UII Hosted Applications page appears, showing the details of the existing hosted application.
  3. To search for another hosted application, type the name of the hosted application in the Search for records text box, and then click Search for records.

    The Search Results page appears, as shown in the following illustration.


    Note:
    If the hosted application exists in the database, the search results show the hosted application details. If the search cannot locate the application, it returns the following error message: "No hosted application records are available in this view."
