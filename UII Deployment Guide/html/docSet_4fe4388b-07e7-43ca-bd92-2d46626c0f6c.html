Deleting a Non-Hosted Application
 

UII Administrator can use the UII Non-Hosted Application management features to delete a non-hosted application.

To delete a non-hosted application

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the navigation pane, click Settings > UII Settings > UII Non-Hosted Application.
  3. The UII Non-Hosted Application page appears. Select the non-hosted application that you want to delete, and then click Delete.
  4. Click Save or Save and Close.
