Modifying a Hosted Application
 

UII Administrator can use the UII Hosted Application management features to update a hosted application.

To modify a hosted application

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings > Hosted Application.
  3. To select an application to modify, select the Hosted Control check box in the right pane, and then click Edit in the UII Hosted Applications action pane, as shown in the following illustration.


  4. Change the values to match your requirements, and then click Save or Saveand Close.
