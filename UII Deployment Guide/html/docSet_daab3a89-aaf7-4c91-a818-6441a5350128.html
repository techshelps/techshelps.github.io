Installing UII Components
 

This section explains how to install the UII components. You must complete the installation process on development computers, and you must have local administrator credentials to run the installer. To import the UII managed solution, the user must be assigned to the organization administrator role.


Note:
Note: If you are using CRM online, you only have to complete the installation process as a CRM organization administrator.
