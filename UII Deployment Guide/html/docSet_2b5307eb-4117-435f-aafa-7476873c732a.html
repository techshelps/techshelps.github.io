Delete the UII Entities of a Previous Version
 

UII R2 is a managed solution, which eliminates conflicts between the managed and unmanaged solutions. If you try to import the solution without deleting the UII entities, an error message appears, as shown in the following illustration.


To avoid this error, make sure you delete the UII entities in the following order.

Note:
If you try to delete the entity without deleting relevant SDK messages, an error message appears.



Order to Delete UII Entities

  1. Audit
    (Navigation path: Components > Entities > Custom)
  2. SDK message related to workflow steps and workflow step mapping
    (Navigation path: Components > SDK Message Processing Steps)
  3. UII workflow-step mapping
    (Navigation path: Components > Entities > Custom)
  4. UII workflow step
    (Navigation path: Components > Entities > Custom)
  5. SDK message related to workflow
    (Navigation path: Components > SDK Message Processing Steps)
  6. UII workflow
    (Navigation path: Components > Entities > Custom)
  7. SDK message related to UII option
    (Navigation path: Components > SDK Message Processing Steps)
  8. UII option
    (Navigation path Components > Entities > Custom)
  9. Context
    (Navigation path: Components > Entities > Custom)
  10. Session transfer
    (Navigation path: Components > Entities > Custom)
  11. Saved session
    (Navigation path: Components > Entities > Custom)
  12. SDK messages related to a non-hosted application
  13. UII non-hosted application
    (Navigation path: Components > Entities > Custom)
  14. SDK messages related to a UII action
    (Navigation path: Components > SDK Message Processing Steps)
  15. UII action
    (Navigation path: Components > Entities > Custom)
  16. SDK messages related to a UII hosted application
    (Navigation path: Components > SDK Message Processing Steps)
  17. UII hosted application
    (Navigation path: Components > Entities > Custom)
  18. UII customizations
    (Navigation path: Components > Entities > Custom)
  19. Microsoft UII plug-ins

Deleting a Sample UII Entity

To delete an Audit entity, use the suggested order listed in the preceding procedure.

To delete UII entities

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click select Settings > Customizations.
  3. Under Which feature would you like to work with?, click Customize the System.


  4. The Entities page appears. In the left navigation pane, click Components > Entities.
  5. In the View drop-down box, click Custom, as shown in the following illustration.


  6. Under Display Name, select the Audit check box, and then click Delete.
  7. A Confirm Deletion dialog appears, as shown in the following illustration. Click OK.

Deleting a Sample SDK Message

To delete SDK messages

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings >Customizations.
  3. Under Which feature would you like to work with?, click Customize the System.
  4. The Entities page appears. In the left navigation pane, click Components > SDK Message Processing Steps.
  5. Under SDK Messages, select the check boxes for the messages that you want to delete, and then click Delete.

    The following illustration shows an example of deleting an SDK message related to workflow steps and workflow step mapping.


  6. A Confirm Deletion dialog appears, as shown in the following illustration. Click OK.

Deleting UII Plug-ins

To delete Microsoft UII plug-ins

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > Customizations.
  3. Under Which feature would you like to work with?, click Customize the System.
  4. The Entities page appears. Click Plug-in Assemblies.
  5. The Plug-in Assemblies page appears, as shown in the following illustration. Under Component Type, select the Microsoft.UII.Customizations.Plugin check box, and then click Delete.


  6. A Confirm Deletion dialog appears, as shown in the following illustration. Click OK.
