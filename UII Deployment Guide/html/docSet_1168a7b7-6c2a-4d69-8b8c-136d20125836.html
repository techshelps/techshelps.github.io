Importing Sample Audit Reports
 

Reports are not installed by the installer. You have to manually install them.

To install sample audit reports

  1. Using CRM organization administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Workplace > Reports.
  3. The Reports page appears, as shown in the following illustration. In the Reports action pane, click New.


  4. The Report New page appears, as shown in the following illustration. In the Report Type drop-down list, click Existing File.


  5. In the File Location field, click Browse, and then select the .rdl file. By default, the files are available at [Install Directory]\Microsoft Uii\Quickstarts\Sample Audit Reports.
  6. In the Details section, type User Summary in the Name box, and then click Save and Close. The list of reports appears on the Reports page, as shown in the following illustration.


  7. To run a report, double-click the report name.
