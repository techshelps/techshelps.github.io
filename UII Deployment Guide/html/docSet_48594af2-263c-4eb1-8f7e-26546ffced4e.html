Managing UII Options
 

This section describes how to configure the general settings through UII Options. These are settings that drive the behavior of the Integrated Agent Desktop. These settings can be overridden by the settings in the Integrated Agent Desktop configuration file.

For example, when the Integrated Agent Desktop executes, if LoadSession is set to True, the Integrated Agent Desktop loads the previous session. When LoadSession is set to False, the Integrated Agent Desktop loads a new session each time it is executed.
