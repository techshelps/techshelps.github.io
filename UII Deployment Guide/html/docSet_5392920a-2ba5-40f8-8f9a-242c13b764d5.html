Deleting a Workflow
 

UII Administrator can use the UII Workflow management features to delete a workflow.

To delete a workflow

  1. Using CRM administrator credentials, log on to Microsoft Dynamics CRM.
  2. In the left navigation pane, click Settings > UII Settings > Workflow. The Workflow list page appears.
  3. Find the workflow that you want to delete, select it, and then click Delete.
