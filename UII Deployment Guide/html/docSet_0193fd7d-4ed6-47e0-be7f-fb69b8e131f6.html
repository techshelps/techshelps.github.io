Optional Components
 

The following components are required for specific configurations:

  • Java Access Bridge – The Java Access Bridge is required by UII Studio and the client environment if the Java DDAs are going to be used to host Java applications. The Java Access Bridge is available on the Java SE Desktop Accessibility page on the Oracle website.
  • Java Runtime Environment – The Java Runtime Environment is required by UII Studio and the UII client environment to run Java applications. The current version supported by UII is Java Runtime Environment 1.6. The Java Runtime Environment is available from the Java SE Runtime Environment page of the Oracle Download Center.
  • Citrix server and client components – The Citrix online plug-in version 12.0 is required by the client components if you are hosting and automating Citrix applications. The plug-in is available from the Citrix Download page. Citrix XenApp 6.0 server is the required server product associated with the Citrix online plug-in version 12.0.
