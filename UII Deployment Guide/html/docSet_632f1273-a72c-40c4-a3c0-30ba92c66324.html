Using the Installer to Install the Citrix Server Component
 

This section explains how to install the Citrix Server component by running the installer.

To install the Citrix Server component

  1. Complete steps 1–4 of the procedure, Using the Installer to Install UII Client Components.
  2. Change the Citrix Server Component option to Will be installed on local hard drive, clear the selection of all other components, and then click Next. See the following illustration.


  3. Complete steps 8–10 of the procedure, Using the Installer to Install UII Client Components.
