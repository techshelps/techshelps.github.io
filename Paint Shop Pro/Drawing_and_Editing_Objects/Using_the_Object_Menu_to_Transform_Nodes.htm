Using the Object Menu to Transform Nodes

The Objects menu will transform selected nodes using the current settings for each transformation command.

To transform nodes with the current settings:

  1. Select the nodes you want to transform.

  2. Choose Objects > Transform Selected Nodes and choose a com­mand:

Flip (Ctrl+Shift+F) Moves selected nodes along the vertical axis—those on top go the bottom and vice versa. Applied to a single node, the control arms (if any) will flip.

Mirror (Ctrl+Shift+M) Moves selected nodes along the horizontal axis—those on the left go to the right and vice versa. Applied to a single node, the control arms (if any) will mirror.

Rotate Rotates the selected nodes by the current rotation setting. Applied to a single node, the control arms (if any) will rotate.

Skew X Moves selected nodes to the left or right (along the hori­zontal axis) using the current skew setting.

Skew Y Moves selected nodes to the left or right (along the verti­cal axis) using the current skew setting.

Contract (F8) Moves the selected nodes closer to each other by the current contraction setting. Cannot be applied to a single node.

Expand (F9) Moves the selected nodes away from each other by the current expansion setting. Cannot be applied to a single node.

Note: To adjust the transform settings, see Using the Tool Options Palette to Transform Nodes.
