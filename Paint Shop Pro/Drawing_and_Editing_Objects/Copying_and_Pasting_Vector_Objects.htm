Copying, and pasting vector objects

Cut, copy, and paste vector objects just like any other data. Select the object and from the Edit menu, select Cut or Copy.

  • To paste vector objects within the same layer, choose Edit > Paste > As New Vector Selection.

  • To paste vector objects as a new layer, choose Edit > Paste > As New Layer.

  • To paste vector objects as a new image, choose Edit > Paste > As New Image.
