Using the Updated Print Layout Feature

Placing Text Captions on the Layout

To place a box containing custom text:

  1. Open the image(s) you wish to print.

  2. Choose File > Print Layout. This puts Paint Shop Pro in Print Layout mode. Your open images will appear along the left side of the window.

  3. Drag the images into the layout area on the right, or load an existing template and drag your images into the template's cells.

  4. In the Print Layout toolbar, click the Text tool , or choose Image > Text Field.

  5. Drag the cursor to create the text box. When you release the mouse button, the Text Field dialog will appear.

  6. Enter the text you want to appear on the printed page.

  7. Set your options for font, size, style, color, and alignment. Note that if you want to continue using these settings, mark the Retain text settings check box at the bottom of the Text Field dialog. Click OK to place your text on the layout.

  8. If necessary, drag the text box to a smaller or larger size to fit the text it contains. Drag the text box to move it.

  9. To rotate the text, select the text box (if it isn't already selected), then choose Image > Rotate +90 / Rotate -90. You could also right-click and choose either rotation command from the context menu.

Using the Undo/Redo Commands

Print Layout now has Undo/Redo functionality. Click the buttons in the toolbar, or choose Edit > Undo / Redo.

Resizing Images to Specific Dimensions

You can now resize an image to specific print dimensions. To do so:

  1. Click the image you wish to resize.

  2. Choose Image > Resize.

  3. You can resize (by percentage or by entering specific dimensions without changing the image's aspect ratio. Choose one of two options:

  4. Click the Percent of original option, and then enter Width and Height percentages; or

  5. Click the Actual / Print size option, choose a units value (Inches or Centimeters), and then set the Width and Height values to the desired size.

Creating an Elliptical Image Cell

To change a cell from rectangular to elliptical:

  1. In Print Layout, load a template containing at least one cell.

  2. Click the cell you want to make elliptical, and then click the Make cell elliptical button on the Print Layout toolbar. The cell's image will then appear through the ellipse.

Note: To make the cell rectangular again, click the Make cell rectangular button .

Editing Multiple Cells Simultaneously

Print Layout now allows you to apply modifications to multiple cells at the same time.

Enter Custom Template Details

To enter your own detail text that will appear when subsequently browsing for templates:

  1. Load a template, or create your own.

  2. In the Print Layout toolbar, click the Save button . The Save dialog appears.

  3. In the Name field, enter a custom name, or leave this field unchanged to keep the original name.

  4. In the Details field, enter your custom detail information.

  5. If desired, mark the Save with images check box at the top of the dialog to save the template with the images currently loaded in the cells.

  6. Click OK. (Note that if you didn't change the original template name, you'll be prompted to overwrite the template or cancel the operation.)

When you now browse for templates in the Templates dialog, you'll see the detail info in the lower left corner of the dialog.
