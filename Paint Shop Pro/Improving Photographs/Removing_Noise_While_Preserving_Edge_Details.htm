Removing Noise While Preserving Edge Details

Use the Edge Preserving Smooth command to remove noise in an image without losing edge details. This command finds details such as object edges and preserves them, while smoothing the areas between the edges. For example, you can preserve the edges of facial features while removing a blotchy complexion. You can also use the Edge Preserving Smooth command to minimize film grain.

You can apply the Edge Preserving Smooth command to a selection or an image. It is faster and more effective when applied to a selection.

To remove noise while preserving edge details:

  1. Select the area that contains the specks. See Using the Selection Tools.

  2. Choose Adjust > Add/Remove Noise > Edge Preserving Smooth.

  3. For Amount of smoothing, change the value until you are satisfied with the adjustment. Choose the smallest amount of smoothing that removes the specks while retaining image detail.

  4. Click OK.
