Excluding File Types from Browsing

You can choose the exclude specific file types from appearing in the Image Browser. To do so:

  1. Choose File > Preferences > General Program Preferences.

  2. Click the Browser tab.

  3. In the lower right corner of this tab, click the File Format Exclusions button.

  4. In the File Format Exclusions dialog, select the file types you do not want browsed.

  5. Click OK to close this dialog and return to the Browser preferences tab.
