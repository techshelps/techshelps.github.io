Setting General Program Preferences  

General Program Preferences define the behavior of various fea­tures and commands.

To set General Program Preferences:

  1. Display the General Program Preferences dialog by doing one of the following:

  2. Choose File > Preferences > General Program Preferences; or

  3. Right-click on an empty portion of the workspace or a toolbar to display the main context menu, and then choose General Program Preferences.

  1. Click the tab containing the preference information you want to change.
