Display and Caching Preferences

The Display and Caching tab controls items related to presentation and caching.

About color pickers: In previous versions of Paint Shop Pro, you could use the Jasc Color Picker or the Windows Color Picker. In Paint Shop Pro, the Windows Color Picker is not available.

To set display and caching preferences:

  1. In the Presentation group box choose the following options:

  • Mark the Re-use last type used in save-as dialog check box to use the last file format used to save an image. The file format automati­cally appears in the Save as type field.

  • Clear the Display all color dropper ToolTips check box to hide the color information that appears next to the dropper tool when it is positioned over a color.

  • Mark the Use precise cursors check box to use a cross hair for tool cursors instead of the tool’s button shape.

  • Clear the Show brush outlines check box to hide the outline (shape, size) of the current brush as the cursor moves over an image.

  • Adjust the Zoom / Rescaling Quality slider to set whether the Zoom tool quality is faster or better.

  2. When holding the cursor over a layer name in the Layers palette; and

  3. In the Effects Browser dialog.

  1. In the Caching Options group box choose the following options:

Note: These options allow for faster image redraws after you make changes. By default, they are all marked. Turning these options off will likely slow image redraws.

  •  Merged Image: Clear this check box to stop saving an image cache of your entire merged image.

  • Merged Below Active Layer: Clear this check box to stop saving an image cache of all the layers below the active one.

  • Merged Group: Clear this check box to stop saving an image cache of the merged image for each group.

  • Merged Below Active Layer: Clear this check box to stop saving an image cache of all the group layers below the active one.
