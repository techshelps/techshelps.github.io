Adding or Deleting File Locations

You can add or delete folders for Paint Shop Pro to store/retrieve the various resources. Deleting a folder means that Paint Shop Pro will no longer look in the folder for any items (the folder is not deleted from your computer).

Note: Some of the resource file types (such as Cache and CMYK Profiles) can have only one location. You can only change the file location from one folder to another. For more information on changing file locations, see Changing File Locations.

To add file locations:

  1. Choose File > Preferences > File Locations to open the File Loca­tions dialog.

  2. From the File types list, select the resource file type for which you want to add locations.

  3. Click the Add button, or click the Browse for Folder button at the end of the selected folder. The Browse for Folder dialog appears.

  4. Click the Create New Folder button and then enter the name for the new folder.

  5. Click OK to add the new folder to the list.

  6. Click OK to close the File Locations dialog.

To delete file locations:

  1. Choose File > Preferences > File Locations to open the File Loca­tions dialog.

  2. From the File types list, select the resource file type that you want to change file locations for.

  3. In the list of folders, select the folder you want to delete.

  4. Click the Delete button.

  5. Click OK.

Another way to view file locations: You can quickly open the File Locations dialog by clicking the File Locations button which appears in resource-related Paint Shop Pro dialogs and toolbars.
