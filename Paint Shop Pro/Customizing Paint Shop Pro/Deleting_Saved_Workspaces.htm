Deleting Saved Workspaces

If you no longer want to use a saved workspace, you can easily delete it.

To delete a saved workspace:

  1. Choose File > Workspace > Delete, or press Shift + Alt + D to open the Workspace dialog.

  2. Select the workspace file you want to delete.

  3. Click Delete.
