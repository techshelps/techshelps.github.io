Setting Kodak PhotoCD Preferences

The Kodak PhotoCD format (PCD) is used by many photo develop­ers. This format can provide multiple image resolutions in one file.

To set PCD file format preferences:

  1. Choose File > Preferences > File Format Preferences to open the File Format Preferences dialog.

  2. Click the PCD tab.

  3. Choose:

  4. A default resolution setting. For example, 1536 x 1204 pixels.

  5. Ask when loading each file PSP will ask you to pick a default resolution each time you open a PCD file.

  1. Click OK.
