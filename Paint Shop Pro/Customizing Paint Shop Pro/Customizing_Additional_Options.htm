Customizing Additional Options

Customize how paint  Shop Pro displays icons, screen tips and captions using the Options tab.

Set the following toolbar and palette options:

  1. Choose View > Customize to open the Customize dialog, and then click the Options tab.

  2. Show Screen Tips on toolbars Clear this check box to hide the screen tips that appear over tools and buttons.

  3. Show shortcut keys in Screen Tips Clear this check box to hide the shortcut key (if available) that appear in screen tips.

  4. Large icons Mark this check box to display larger toolbar icons.

  5. In the Personalized Menus and Toolbars group box, set the following options:

  6. Menus show recently used commands first Mark this check box to have Paint Shop Pro simplify its menus to show only recently used commands. You can expand the menus by clicking the expansion button.

  7. Show full menus after a short delay If menus are only showing the most recent items, all items will appear in a moment.

  8. Reset my usage data Click this button to erase the memory of your recently used menu commands.

  1. In the Palettes group box, mark the Show captions on docked palettes check box to give layout captions a different appearance.
