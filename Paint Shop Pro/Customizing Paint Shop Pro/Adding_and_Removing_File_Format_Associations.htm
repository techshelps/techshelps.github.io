Adding and Removing File Format Associations

You can have certain file formats automatically open in Paint Shop Pro. For example, when you associate the JPEG file format with Paint Shop Pro, all JPEG files will open in Paint Shop Pro when you double-click them from the Windows desktop. The associated JPEG files will also display Paint Shop Pro icons. For a list of supported file formats, see Paint Shop Pro 8 Supported File Formats.

To associate a file with Paint Shop Pro:

  1. Choose File > Preferences > File Format Associations to open the File Format Associations dialog. Do any of the following:

  2. To associate a specific file format, mark the format’s check box.

  3. To associate all listed formats, click the Select All button.

  4. To associate all listed formats that are not already associated with other applications, click the Select Unused button.

  5. To associate no file formats with Paint Shop Pro, click the Remove All button.

  1. Click OK.

Note: To add or remove file extensions you want associated with the selected format, see Changing File Format Extensions.
