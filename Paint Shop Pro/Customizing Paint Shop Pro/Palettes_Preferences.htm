Palettes Preferences

The Palettes tab controls behaviors of the materials palette as well as how palettes dock.

To set palette preferences:

  1. In the Materials Palette Options group box, choose from the following sets of options:

  • Display colors in RGB format or Display colors in HSL format
    This option controls the color information that appears in the materials palette and with the dropper tool.

  • Decimal display or Hexadecimal display
    This option controls how color values will display throughout the program.

  • Show rainbow picker or Show document palette
    For images with limited palettes (8-bit, 256 color), the document palette shows only colors that are actually in the image.

  1. In the Docking Options group box, clear the check box for each palette that you don't want to dock. This will allow you to drag the palette anywhere in the workspace.
