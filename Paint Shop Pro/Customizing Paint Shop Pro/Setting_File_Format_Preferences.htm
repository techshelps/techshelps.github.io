Setting File Format Preferences

You can specify how Paint Shop Pro handles images in the following formats: Windows Metafile, Kodak PhotoCD, PNG, PostScript, RAW Graphics, and Raw Camera Data.  For a list of supported file formats, see Paint Shop Pro 9 Supported File Formats.
