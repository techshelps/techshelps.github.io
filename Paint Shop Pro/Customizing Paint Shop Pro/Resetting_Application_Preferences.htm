Resetting Application Preferences

Various program preferences can be reverted back to their defaults.

To reset application preferences:

  1. Choose File > Preferences > Reset Preferences to open the Reset the Application dialog.

  2. Select from these choices:

  3. Reset general preferences Mark this check box to reset all general program preference settings to the program defaults.

  4. Reset file locations Mark this check box to reset all file loca­tion settings to the program defaults.

  5. Reset workspace to default configuration Mark this check box to reset the current workspace to the program defaults.

  6. Reset dialog positions and last used settings When you mark this check box, any dialogs that open will appear in their default locations. Also, Paint Shop Pro will forget the last set­tings you used in all dialogs.

  7. Delete all cache files Mark this check box to delete (reset) all existing cache files. Paint Shop Pro saves cache files on your computer to improve program performance. Cache files can become corrupted by computer errors.

  8. Make all toolbars/palettes visible and on screen Mark this check box to view all toolbars and palettes.

  1. Click OK.
