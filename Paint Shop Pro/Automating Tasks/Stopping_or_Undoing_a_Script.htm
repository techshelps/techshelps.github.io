Stopping or Undoing a Script

Stopping a Running Script

To stop a script while it is running , click Stop Script . The Stop Script button is only active while the script is running. Depending on the script, this may only be a few seconds.

Undoing an Applied Script

Like any other action applied to an image, you can always undo an applied script by clicking Undo on the standard Toolbar.
