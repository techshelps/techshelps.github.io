Using the Script Menu Items

Most of the Script toolbar buttons have equivalent menu commands accessed by choosing File > Script.

The Clear Output Window command, by default, can only be chosen from this menu path (unless you add the command to a toolbar). Use this command to clear out the text dis­played in the Script Output window. You can also clear this window by right-clicking it and choosing Clear from the context menu.

The Single Step command is also, by default, only accessed via the File > Script menu but can be added to a toolbar if desired. For more information on this command, see Trouble Shooting a Script.

About the Script Output Window

To display this window, choose View > Palettes > Script Output. Note, however, that this window may automatically appear to dis­play information such as script error information such as Python syntax errors or runtime errors.

Text in the Script Output window can be selected and copied to the clipboard. This can be helpful if you need to report error information.
