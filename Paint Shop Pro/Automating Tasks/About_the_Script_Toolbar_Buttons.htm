About the Script Toolbar Buttons

The following describes the buttons in the Script toolbar:

  • Select Script drop-down . By default, this drop-down lists the scripts saved in Paint Shop Pro’s Scripts-Trusted and Scripts-Restricted folders. Using the File Locations dialog, you can add any folder to the list of folders that this drop-list displays scripts from..

  • Run Selected Script . Click this button to run the script dis­played in the Select Script drop-down.

  • Edit Selected Script . Click this button to edit the script dis­played in the Select Script drop-down.

  • Interactive Script Playback Toggle . Click this button to switch between Silent and Interactive mode when running scripts. When the button shows a border, it is set to Interactive mode.

  • Run Script . Click this button to display the Run Script dialog. Navigate (if necessary) to and select the script you wish to load, and then click Open.

  • Stop Script . Click this button to stop the running script. Note that this button will only be active while the script is running.

  • Start Script Recording . Click this button to begin recording actions that will comprise a script.

  • Pause Script Recording . Click this button to pause the record­ing of a script. Click it again to resume recording.

  • Cancel Script Recording . Click this button to cancel script recording.

  • Save Script Recording . Click this button to display the Save As dialog. If necessary, navigate to the desired folder, enter a name for the script, and then click Save. Note that this button is only active after you have completed a “scriptable” action.
