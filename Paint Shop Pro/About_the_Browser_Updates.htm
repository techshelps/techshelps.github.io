About the Browser Updates

The Image Browser contains two tabs, Find and Info, in the left side of the Browser window. The Find tab contains the familiar Windows Explorer-like navigation view, and the Info tab displays image information, creator data, and EXIF data. Additionally, with the Browser active you can choose File > Sort and use the Sorting dialog to sort images by EXIF data.
