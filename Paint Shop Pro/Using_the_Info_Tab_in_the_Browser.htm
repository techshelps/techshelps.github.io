Using the Info Tab in the Browser

The Image Browser now includes two tabbed panes on the left side, the Find tab (for navigation) and the Info tab (for displaying image information).

Use the Info tab to display pertinent image information, such as creator details as well as EXIF details for digital photos. To use the tab:

  1. Choose File > Browse, or click the Browse button on the Standard toolbar. The Image Browser will appear.

  2. Click the Info tab located in the upper left portion of the Browser window.

  3. Click on an image thumbnail to view its details in the Info tab. If you're browsing digital camera images, the EXIF data associated with the image will appear in the Details section of this tab.

 

Note: At this time, although you can click on fields in the Creator Data fields, you cannot type text into the fields.
