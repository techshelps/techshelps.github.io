About the Updated Vector Drawing Tools

Updates and new features for vector drawing include:

  • Changes to the Pen tool's features and functionality, including new and relabeled Tool Options palette items, as well as changes to the tool's general operation compared to the previous version of the application.

  • A new Rectangle tool for quickly and easily creating squares, rectangles, and rounded rectangles. This feature was part of the Preset Shapes tool in the previous version. This tool is grouped with the Ellipse, Symmetric Shape, and Preset Shape tools.

  • A new Ellipse tool, allowing you to quickly and easily create circles and ellipses. This feature was part of the Preset Shapes tool in the previous version. This tool is grouped with the Ellipse, Symmetric Shape, and Preset Shape tools.

  • A new Symmetric Shapes tool, allowing you to quickly and easily create polygons and star shapes having from 3 to 1000 sides.

  • The Preset Shapes tool no longer includes the ellipses and rectangles found in the previous version as those shapes were built into the Ellipse and Rectangle tools.

  • Placing text on a vector path now has some new functionality associated with it. Refer to the Text tool topic linked below for more information.
