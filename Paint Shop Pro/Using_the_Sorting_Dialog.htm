New Sorting Options

While in Browser mode, the Sorting dialog now contains new options for sorting images via their EXIF data. (Also note that the dialog no longer contains two tabs.)

To use these new options:

  1. Start up the Image Browser by choosing File > Browse, or clicking the Browse button on the Standard toolbar.

  2. In the Find tab, navigate to the desired folder.

  3. Choose File > Sort. This displays the Sorting dialog.

  4. In the Sort condition drop list of either the Primary sort order or the Secondary sort order sections, choose the desired EXIF data field to sort by, and then click OK.
