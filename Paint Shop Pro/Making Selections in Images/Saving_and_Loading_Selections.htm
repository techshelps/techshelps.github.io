Saving and Loading Selections

It’s helpful to save complicated selections that you may use again or to save selections to use in other images to create effects. Save a selection in one of two ways:

  • Save the selection as a file with a .PspSelection extension on a hard disk or removable disk; see Saving Selections to Disk. To load this selection file into Paint Shop Pro, see Loading Selections from Disk.

  • Save the selection as an alpha channel within the image. The selection is stored as part of the image, not as a separate file. See Saving Selections to Alpha Channels. To load the selection into the same or a different image, see Loading Selections from Alpha Channels.
