Printing a Single Image

To print the active image:

  1. Choose File > Print, or click the Standard toolbar's Print button , or press Ctrl + P to open the Print dialog.

  2. Do any or all of the following:

  3. In the Printer group box, select the printer and whether to print to file.

  4. On the Placement tab, choose the number of copies, the page orientation, and the size and position of the image.

  5. On the Options tab, choose the print output, color options, and print marks.

  6. On the Template tab, choose to print a template with images or fill a template with the current image.

Selected options display on the image preview. For more information on the options, see About Printing Options.

  1. Click the Print button to print the image. Or, click the Close button to save the print options without printing the image.
