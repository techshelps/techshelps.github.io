Deleting a Print Template

If you have saved a print template that you no longer need, you can easily delete it.

To delete a template:

  1. Do one of the following:

  2. From the Paint Shop Pro workspace, choose File > Print Layout.

  3. From the Paint Shop Pro Browser, choose File > Print Layout.

  1. In the Print Layout window, choose File > Open Template or click Open Template to open the Templates dialog.

  2. Choose a category from the Select a category list.

  3. Select the thumbnail of the template you want to delete.

  4. Click Delete.
