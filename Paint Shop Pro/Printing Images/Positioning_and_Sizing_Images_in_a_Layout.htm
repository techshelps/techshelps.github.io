Positioning and Sizing Images in a Layout

You can position and size individual or multiple images in a layout.

To rotate an image in a layout:

  1. Select the image or images.

  2. On the toolbar, click Rotate 90+ or Rotate 90- . Or, select the same commands from the Image menu.

To resize an image in a layout:

  1. Select the image.

  2. Drag a corner handle until the image is the size you want it. The image maintains its aspect ratio.

To position an image in a layout:

  1. Click and drag the image to a new position on the page. You can also temporarily place images in the workspace around the page. You can also position an image by clicking one of the command buttons. These buttons align the selected image to the layout page.

  2. Position image at X,Y Click this button to open the Place Image dialog. Enter values to set the position of the image’s top left corner.
