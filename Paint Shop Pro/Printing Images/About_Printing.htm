About Printing

There are several ways to ways to print in Paint Shop Pro:

  • Print a single image. For more information, see Printing a Single Image.

  • Print multiple images on one page. Arrange and size images in any way you like. Choose from over 100 templates, or create your own custom template. For more information, see Printing Multiple Images on a Page.

  • Print images from the Browser as thumbnails or in any arrangement. For more information, see Printing Images from the Browser.
