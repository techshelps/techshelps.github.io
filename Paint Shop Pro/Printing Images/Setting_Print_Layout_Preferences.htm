Setting Print Layout Preferences

Use the options in the Preferences menu, or choose an option from the toolbar, to show and hide grids, snap images to grids, arrange, position, and size images in the Print Layout window. For more information, see: Using Grids,  Automatically Arranging Images in a Layout, Positioning and Sizing Images in a Layout, and Setting Print Layout Options.
