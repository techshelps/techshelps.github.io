Editing Template Cells

You cannot move or resize cells in a template. A cell’s position and size is locked when a layout is saved as a template.

However, if you saved images with a template, you can revert the template back to a layout.

To revert a template (with images) to a layout:

  1. Open the template you want to edit. For more information, see Working with Templates.

  2. Choose File > New Template or click the New Template button . A dialog will open asking, “Do you want to keep the current images in the new template?”

  3. Click Yes.
    The Print Layout window will display a new layout that contains all the images that were in the template.
