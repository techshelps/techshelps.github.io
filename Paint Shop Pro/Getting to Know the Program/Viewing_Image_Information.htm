Viewing Image Information

As you work with an image, you may want to view information about it. There are several ways to do this, as described in the sections below.

To view general image information:

  • With the cursor inside the image window, look at the Status bar (along the bottom of the Paint Shop Pro main window) to see the cursor position, image height and width in pixels, and the color depth.

  • Click the Info tab of the Overview palette to view information about the image. (If this palette is not displayed, choose View > Palettes > Overview.)

Note: Use the Image Information command (detailed below) to enter creator information and other data about the image.

To view image color information:

Choose Image > Count Image Colors to display the number of unique colors in the image.

To view detailed image information:

  1. Choose Image > Image Information, or press Shift + I. The Image Information dialog appears.

  2. Do one of the following:

  3. Click the Image Information tab to view the file name and format, image dimensions, resolution, and color depth, modification status, number of layers and alpha channels, and the amount of RAM and disk space used for the image as well as its selections, masks, alpha channels, and undo storage.

  4. Click the Creator Information tab to view or edit the image title, artist name, copyright, description, date created, and date modified.

  5. Click the Watermark Information tab to view and enter information on Digimarc watermarking.

  6. Click the EXIF Information tab to view information for images taken with digital cameras. In the Section group box, mark the check boxes for the information you want to display: Input Device, Artist, Date, Image, Shot Conditions, GPS, and Makernote. Note that some Items are editable (as denoted by the * character). To enter customized text for an editable Item, click in the Value column to get an active text entry cursor, and then type your custom information.

To view layer color information:

Choose Layer > Count Layer Colors to display the number of unique colors in the current layer.
