The Status Bar

The Status bar displays brief information about the selected tool or menu command, as well as information about image dimensions, color depth, and cursor position. Unlike other toolbars, the Status bar cannot be customized or moved. It only appears at the bottom of the main Paint Shop Pro window. For more information, see Viewing Image Information.
