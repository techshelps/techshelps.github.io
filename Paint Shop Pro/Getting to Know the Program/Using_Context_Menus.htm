Using Context Menus

Context menus provide quick access to commands specific to the active tool, selection, or palette.

Note: For a complete list of Context menus, choose View > Customize to open the Customize dialog. Click the Menu tab, and select a context menu from the Select context menu drop-down list.

To use a context menu:

  1. Position the cursor over an image, an empty area of a palette, a layer name on the Layers palette, a thumbnail in the Browser, or a toolbar.

  2. Right-click to display the context menu.

  3. Choose a command from the menu.
