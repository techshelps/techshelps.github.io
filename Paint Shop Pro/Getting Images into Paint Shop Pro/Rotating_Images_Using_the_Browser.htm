Rotating Images Using the Browser

Use the Browser to rotate JPEG images with lossless rotation and retain all of the original image data.

To rotate images:

  1. Select the JPEG images that you want to rotate.

  2. Choose ImageFile > JPEG Lossless Rotation, and then select an option:

  3. Clockwise 90°

  4. Clockwise 180°

  5. Counter Clockwise 90°

Important: Be aware that these three options will not only rotate the JPEG image as it's viewed in the Browser, but will also rotate the actual JPEG image itself.
