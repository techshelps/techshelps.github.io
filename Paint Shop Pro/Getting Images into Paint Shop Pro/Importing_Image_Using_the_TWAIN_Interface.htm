Importing Images Using the TWAIN Interface

Most cameras and scanners are TWAIN-compliant. Use this setting if your camera, scanner, or card reader is TWAIN-compliant.

To import images using the TWAIN interface:

  1. Choose File > Import > TWAIN > Select Source. The Select Source dialog lists the TWAIN-compliant devices connected to your computer.

Note: You only need to select the source the first time you connect to the camera, card reader, or scanner.

  1. Click the name of the device, then click Select.

  2. Choose File > Import > TWAIN > Acquire to run the device’s software.

  3. Do one of the following:

  4. For scanners, perform the scan using the device’s software. (Refer to the scanner manufacturer’s documentation for details.)

  5. For digital cameras, an Import dialog displays. Select the images that you want to download, and then click Download.

When the device and software finish processing the image, the image is sent to Paint Shop Pro and placed in an image window. The software remains open so you can continue to acquire images or close the software.

  1. Click the Paint Shop Pro icon in the Windows task bar to display Paint Shop Pro and your image.

  2. Choose File > Save As to save the image.
