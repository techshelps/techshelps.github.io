Renaming Files Using the Browser

You can easily rename files using the Browser.

To rename files:

  1. Press Ctrl + R or right-click the thumbnail and select Rename from the context menu.

  2. Type the new filename. The Browser does not automatically add an extension.

  3. Click OK.
