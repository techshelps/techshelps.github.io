Saving a New Image File

The first time you save an image, Paint Shop Pro automatically opens the Save As dialog, where you select a location, name, and file format for the image. When you subsequently save the image by choosing File > Save (or clicking the Save button on the Standard toolbar, or pressing Ctrl + S), Paint Shop Pro saves changes without opening the dialog.

To save a new image:

  1. Choose File > Save, or press Ctrl + S, or click the Save button on the Standard toolbar. The Save As dialog opens.

  2. Use the Save In drop-list to navigate to the folder where you want the file to reside.

  3. In the File Name field, type a name for the file.

  4. In the Save As Type drop-list, choose a file format.

  5. If the file format has save options, click the Options button to open the Save Options dialog. Select new settings, if appropriate. If you are unsure of the settings, use the default values. Click OK to return to the Save As dialog.

  6. Click Save to close the dialog and save the file.

You can have Paint Shop Pro display the last-used file type when opening the Save As dialog. Select the Re-use last type in file save-as dialog option in the Display and Caching tab of the General Program Preferences dialog. To access this dialog, choose File > Preferences > General Program Preferences, and then click the Display and Caching tab.
