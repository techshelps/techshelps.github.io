Viewing EXIF Data

EXIF stands for Exchangeable Image File Format. If your camera supports this standard, it will record the time & date the image was taken, exposure information (ISO, shutter, aperture) and other camera details in the header of each image file. To view EXIF data, choose Image > Image Information to open the Current Image Information dialog, and then click the EXIF Information tab.
