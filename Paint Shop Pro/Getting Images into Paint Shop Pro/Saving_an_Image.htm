Saving an Image

You can easily save an image with the same file name and location. Note that this will overwrite the original image with the modified image. To save the image using a different name or file format, choose File > Save As.

Save an image using one of these methods:

  • Choose File > Save to automatically save the image with the same file name and location (this will overwrite the original image with the modified image; or

  • Click the Save button on the Standard toolbar; or

  • Press Ctrl + S.
