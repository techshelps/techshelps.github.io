Opening Existing Images

You can open images in a variety of file formats. The list of formats you can open in Paint Shop Pro appear in the Open dialog.

You can select which formats Paint Shop Pro displays as available to open. See Setting File Format Associations.

You can open a file using the Open dialog or the Browser or by choosing a recently used file from the File > Recent Files menu. With the Open dialog, you can view a preview a highlighted image when the Show Preview option is checked. With the Browser, you can preview all image files within a folder. When you open a file via the Browser, the Browser window stays open until you close it.

To open an image file using the Open dialog:

  1. Display the Open dialog using one of the following methods:

  2. Choose File > Open; or

  3. Click the Open button on the Standard toolbar; or

  4. Press Ctrl + O.

  1. In the Look in drop-list, select the folder where the file is stored.

  2. Click the name of the file you want to open. To select multiple files:

  3. Hold down Ctrl and click each name to select non-adjacent files; or

  4. Hold down Shift and click each name to select adjacent files.

  1. To view information or a preview of the image, choose an option:

Details Click this button to open a pop-up that displays information about the selected image. Click OK to close this pop-up.

Show preview Mark this check box to display the selected image in the preview area. If you have selected multiple files, no preview is displayed; use the Browse window instead.

  1. Click Open.

To open an image file using the Browser:

  1. Do one of the following:

  2. Choose File > Browse; or

  3. Click the Browse button ; or

  4. Press Ctrl + B.

The Browse window opens. Note that the Paint Shop Pro menu bar displays commands specific to when the Browse window is active. The Browse menu is active until you open a new file or click an open image.

  1. In the left side of the Browse window you’ll see the Find and Info tabs. Use the Find tab like you would Windows Explorer to navigate and then select the folder that contains the file you want to open. Use the Info tab to display image information on a selected image. The right side of the Browse window displays thumbnails of image files located in the selected folder.

  2. Select files using one of these methods:

  3. Click a thumbnail image; or

  4. To select non-adjacent files, hold down Ctrl and click each file; or

  5. To select adjacent files, click the first file, hold down Shift, and click the last file.

Note: To deselect a file, press Ctrl and click its thumbnail.

  1. Open the selected file(s) using one of these methods:

  2. Double-click a single file’s thumbnail or Ctrl + double-click one thumbnail when multiple files are selected.

  3. Select a file or multiple files and then drag them onto a blank part of the work space.

  4. Press Enter.

  5. Right-click and select Open from the context menu.

  6. Choose Image File > Open.

After the file or files open, you can close the Browse window by clicking its Close button. You may also want to leave the Browse window open in the workspace.

To open a recently used file:

Choose File > Recent Files and then choose the file name.

By default, the Recent Files menu lists four recently used files. You can include up to 10 files. For more information, see Miscellaneous Preferences.
