Mapping Images  

Use the Image Mapper dialog to create image maps for your Web pages. An image map is a graphic that contains a mapped area (called a region), that links to a URL. The cell can be a circle, rectangle, or irregular shape. When user moves the mouse over the cell, the cursor changes to a hand indicating that you can click that area to jump to another Web page.

Follow these main steps to map an image:

Create the map area (region)

Assign Map Area Properties

Set Map Preferences

Save and Load the Map Settings

Save the Image Maps
