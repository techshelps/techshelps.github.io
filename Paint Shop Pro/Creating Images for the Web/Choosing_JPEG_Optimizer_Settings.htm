Choosing JPEG Optimizer Settings

The JPEG Optimizer dialog contains two tabs where you configure the settings and a third that displays estimated download times of the image at various Internet speeds once you have optimized the image, you'll need to save it. Each time you open and save the JPEG image data is discarded, so it is a good idea to keep a copy of the original image.
