Saving and Loading the Map Settings

When you save the map settings, you save the map area arrangement and the information you entered in the Image Mapper dialog. You can then re-load them into the same image, or use the settings with a different image.

To save the map settings:

  1. Click the Save Settings As button to open the Save Map Settings dialog.

  2. Navigate to the folder in which you want to save the settings file. The file is saved in the .JMD format.

  3. Type a name for the file and click Save.

To load the map settings:

  1. Open the image in which you want to load map settings.

  2. Click the Load Settings button. The Load Map Settings dialog opens.

  3. Navigate to and select the .JMD file you want to load.

  4. Click Open.
