Saving and Loading Slice Settings

When you save the slice settings, you save the cell arrangement and the information you entered in the Image Slicer dialog. You can then re-load them into the same image or use them with a different image.

To save the slice settings:

  1. Click the Save Settings button. The Save Slice Settings dialog opens.

  2. Navigate to the folder in which you want to save the settings file. The file is saved in the .JSD format.

  3. Type a name for the file and click Save.

To load the slice settings:

  1. Click the Load Settings button. The Load Slice Settings dialog opens.

  2. Select the JSD file you want to load and click Open.

If you are loading a grid to a different image, Paint Shop Pro automatically adjusts the grid size if the dimensions of the image are not the same.
