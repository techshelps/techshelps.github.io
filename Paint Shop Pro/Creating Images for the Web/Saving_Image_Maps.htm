Saving an Image Map to the Clipboard

Save map settings to the clipboard so you can paste them into the HTML code.

To save an image map to the clipboard:

  1. In the Image Mapper dialog, click the Save to Clipboard button to open the HTML Destination Folder dialog.

  2. Navigate to the folder where the HTML file is saved, and click OK to open the Image Map Save as dialog.

  3. Navigate to the folder where the image file is saved, and click OK.

  4. Open the HTML file, place the cursor where you want to insert the HTML map code, and press Ctrl + P.

Saving Image Maps

Save image maps to a directory on your hard drive so you can use or edit them.

To save image maps:

  1. In the Image Mapper dialog, click the Save or Save as button to open the HTML Save As dialog.

  2. Navigate to the folder where you want to save the HTML code and click OK.
