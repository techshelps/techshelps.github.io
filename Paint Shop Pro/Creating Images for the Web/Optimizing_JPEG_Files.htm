Optimizing JPEG Files

Paint Shop Pro provides a JPEG Optimizer feature to assist you in saving JPEG and files that maximize quality while minimizing file size.

Lossy vs. Lossless compression: Lossy Compression is a compression method that eliminates data to reduce the file size, while lossless compression retains all of the original image data and reduces the file size by storing patterns of pixels in the image.

To optimize and save a JPEG file:

  1. Choose File > Export > JPEG Optimizer. The JPEG Optimizer dialog opens. The preview windows show the current image on the left and the resulting JPEG file on the right.

  2. Do one of the following:

  3. Configure the settings on the Quality and Format tabs, and view estimated download times on the Download Times tab. For more information, see Choosing JPEG Optimizer Settings.

  4. Click the Use Wizard button to let Paint Shop Pro guide you through the process. For more information, see Using the JPEG Wizard.
