GIF Optimizer Download Times Tab

This tab displays the size of the compressed file and an estimate of the download time at four Internet speeds. If the file size is too large, decrease it by clicking the Colors tab and reducing the number of colors.

Saving the GIF File

  1. After you have selected the options, click OK to open the Save As dialog.

  2. Navigate to the folder in which you want to save the new image and type a name for the file.

  3. Click Save, Paint Shop Pro saves a copy of the image as a GIF file.
