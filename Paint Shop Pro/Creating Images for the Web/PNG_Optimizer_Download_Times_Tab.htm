PNG Optimizer Download Times Tab

This tab displays the size of the compressed file and an estimate of the download time at four modem speeds. If the file size is too large, decrease it by clicking the Colors tab and reducing the number of colors.

To save a PNG file:

After you have selected the options, click OK to open the Save As dialog. Navigate to the folder in which you want to save the new image, and type a name for the file. When you click Save, Paint Shop Pro saves a copy of the image and leaves the original image unchanged.
