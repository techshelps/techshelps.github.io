Saving Slices

Save slices to a directory on your hard drive so you can use or edit them.

To save slices:

  1. In the Image Slicer dialog, click the Save or Save as button to open the HTML Save As dialog.

  2. Navigate to the folder where you want to save the HTML code and click OK.
