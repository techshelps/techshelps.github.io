Updates in Preferences and Customize Dialogs

General Program Preferences Updates

Updates to the General Program Preferences dialog include the following:

  • Access the dialog faster by right-clicking in the main workspace or in a toolbar and then choosing General Program Preferences from the context menu.

  • What was formerly named the Transparency tab is now the Transparency and Shading tab. Use the Shading section of this tab to enable shading outside the Crop rectangle, and set the color and opacity of the shaded area.

  • In the Palettes tab's Allow Docking Of section you'll find check boxes for the new History palette and Mixer palette.

  • At the bottom of the Undo tab you'll find a numeric edit control for the History steps per open image setting. This setting determines how many steps the History palette will list for open images.

Customize Dialog Updates

Updates to the Customize dialog include the following:

  • The new Script tab provides an easier way to associate a script to a BoundScript button. These buttons can then be dragged onto a new or existing toolbar to provide instant access for your favorite scripts.

  • In the Toolbar section of the Options tab are two new items. The Animate Auto Hide check box and its Speed slider let you set the speed at which palettes will roll up. And the Use Quadrant Rollups check box determines whether palettes will roll up and slide into the nearest workspace quadrant, or roll up in their current location.

  • A new section in the Options tab, called View Options, provides a check box to display open images in tabs across the top of the workspace.
