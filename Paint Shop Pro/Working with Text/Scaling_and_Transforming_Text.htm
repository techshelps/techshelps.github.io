Scaling and Transforming Text

To scale (resize) and transform text:

  • For vector text, use the Object Selection tool to scale, rotate, shear, distort, or apply perspective to text. For more information, see Resizing and Reshaping Vector Objects.

  • For raster text, use the Deform tool to scale, rotate, shear, distort, or apply perspective to a raster selection or a layer. For more information, see Scaling and Transforming Images.
