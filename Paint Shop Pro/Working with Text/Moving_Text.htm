Moving Text

To move or position vector text on a layer, use the Object Selection tool. To move raster text, use a selection tool like the Magic Wand tool to select and move it.

Note: To move text to another layer, select the text, then cut and paste the text to another layer.

To move vector text:

  1. On the Tools toolbar, click the Object Selection tool .

  2. Select the text you want to move. A bounding box surrounds the text, click inside the bounding box, the cursor changes to .

  3. Click and drag the text anywhere within the image.

To move raster text:

  1. On the Tools toolbar, select one of the Selection tools. For details, see Using the Selection Tools.

  2. Select the text you want to move. A bounding box surrounds the text, click inside the bounding box, the cursor changes to .

Note: If you move the cursor outside of the bounding box, the cursor changes to the selection tool cursor.

  1. Click and drag the text anywhere within the image.
