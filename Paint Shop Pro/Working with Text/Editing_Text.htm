Editing Text

Use the Text tool to edit vector text. Edit the characters, and/or any properties used to define the appearance of the text.

Note: You cannot edit the characters or formatting of raster text. To modify Raster text, use the Deform tool.

To edit text:

  1. First, you'll need to open the Text Entry dialog by doing one of the following:

  2. Choose the Text tool , position the cursor over the text in the image until the cursor turns to , and then click; or

  3. Choose the Object Selection tool, select the text, and then right-click and from the Context menu choose Edit Text; or

  4. On the Layer palette, double-click the layer containing the text, or right-click on the layer and select Edit Text.

  1. The Text Entry dialog opens with the text highlighted. (Only the highlighted text will be edited.)

  2. Edit the text in any of these ways:

  3. To format all text, keep it all highlighted and choose options on the Tool Options palette or choose new colors or materials on the Materials palette.

  4. To format one or more characters, click and drag to highlight the text you want to change, and choose options on the Tool Options or Materials palette.

  5. To add new text, click where you want to insert it and begin typing.

  6. To delete text, highlight the text and press Delete.
