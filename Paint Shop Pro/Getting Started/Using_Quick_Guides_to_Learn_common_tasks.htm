Using the Quick Guides to Learn Common Tasks  

Quick Guides are short, fast tutorials that guide you through new tasks and projects in just a few minutes using your own image. Perform each step yourself, or in some cases have the program complete the step for you as you work toward completing the task.

To use Quick Guides:

  1. Choose View > Palettes > Learning Center, or press F10 to display the Learning Center palette.

  2. Scroll through the list of Quick Guides and click the title to view the selected topic.
