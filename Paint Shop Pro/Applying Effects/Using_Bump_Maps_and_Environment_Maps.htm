Using Bump Maps and Environment Maps

Effects such as Balls and Bubbles and Magnifying Lens use Bump Maps and Environment Maps to apply an optional 3-D appearance to the effect.

  • The dialog for the Balls and Bubbles effect includes a Map tab where you can choose to apply a Bump Map and/or an Environment Map.

  • The dialog for the Magnifying Lens effect has an Environment Map option in the Properties tab.

Be aware that Bump Maps and Environment Maps are Paint Shop Pro resources that you can manage via the File Locations dialog or the Resource Manager dialog.
