Applying 3D Effects

The 3D effects create images or selections that appear to have three dimensions. These effects are particularly useful for creating images for Web pages.

  • To see the appropriate effect for the Buttonize, Chisel, Cutout, Drop Shadow, Inner Bevel, and Outer Bevel effects, first make a selection in the image. For the Drop Shadow effect in particular, if you want to apply a drop shadow to the entire image, expand the canvas or add a border first, select the image area, and then apply the drop shadow.

  • To apply an effect, choose it from the Effects Browser or choose Effects > 3D Effects and select one of the following effects.

  • To apply these effects to an entire layer, increase the canvas size to add pixels around the edges.

Important: These effects work on greyscale and 16 million color (24-bit) images only. To increase the color depth of an image, see Increasing the Color Depth of an Image.

To apply an effect:

Choose the effect from the Effects Browser or choose Effects >  3D Effects and select one of the following effects:

Buttonize

Chisel

Cutout

Drop Shadow

Inner Bevel

Outer Bevel
