Using Effect Dialogs

Most effects open a dialog in which you adjust settings to customize the effect. These dialogs contain several common features:

  •  Use the preview windows to view the original image and the image with the effect applied to it. For more information on using these preview windows, see Using Preview Windows.

  • To reset settings to default values, in the Presets drop-down list select Default.

  •  When you first open a dialog, Paint Shop Pro displays the last-used settings. (The Presets drop-down list is set to Last Used.) Use this preset to apply the same effect to multiple images without having to adjust the settings each time.

  •  To save and load your own preset effects, see Using and Creating Presets.

  • For general information on dialogs, see Working with Dialogs.
