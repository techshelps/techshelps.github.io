Saving Deformation Maps

You can save the deformations you have created to warp other images. Paint Shop Pro creates a Deformation Map file that contains the positions of the nodes and the number of mesh lines. Save the mesh during the warping process or after you have applied the deformation.

To save a deformation map:

  1. On the Mesh Warp Tool Options palette, in the Deformation map group box, click Save to open the Save Deformation Map dialog.

  2. In the New deformation map group box, type a name for the map file.

  3. In the Checkerboard size drop-down list, choose a checkerboard size that helps you preview the deformation effect. This informa­tion is not saved with the map file.

  4. Click Save to add the file to the default Deformation Maps folder. Click the Edit Paths button to choose a different folder for saving the deformation maps.
