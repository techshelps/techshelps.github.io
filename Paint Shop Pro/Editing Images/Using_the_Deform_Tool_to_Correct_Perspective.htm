Using the Deform Tool to Correct Perspective

When you take pictures of tall objects, the resulting photographs may have perspective distortion—the objects seem to be leaning or angled. This distortion happens when the camera is at an angle to the subjects.

To use the Deform tool to correct perspective:

  1. Select the layer to correct in the Layer palette.

  2. Use grid lines to help you correct lines in your photograph that should be vertical or horizontal. To turn on the grid, choose View > Grid.

  3. On the Tools toolbar, select the Deform tool .

  4. Press Ctrl and click and drag a corner handle. The image updates when you release the mouse button. Continue adjusting the perspective until the image looks correct.

Note: If the correction causes some image data to fall outside the image canvas, you can restore that data by increasing the size of the canvas. See Changing Canvas Size.
