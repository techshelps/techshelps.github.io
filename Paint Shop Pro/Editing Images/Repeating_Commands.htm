Repeating Commands

Choose Edit > Repeat to apply the same command to other images or to apply a command multiple times to the same image. This command allows you to apply the same effect or correction without opening a dialog each time.

The last command that can be repeated always displays in the Edit menu. If the previous command cannot be repeated, the Repeat command is inactive.

To repeat a command:

You can repeat a command (or apply a command to another image) in multiple ways:

  • Choose Edit > Repeat, or press Ctrl + Y.

  • Hold down Shift while selecting any toolbar button or menu item. This will bypass the dialog and run the command with the last used settings.

  • With the History palette open, drag the desired command action listed in the History palette and drop it onto another image. This will make the second image active and apply the command to it.

To repeat multiple actions: Record a script and run it on each image. For more information on using scripts, see Scripting Basics.
