Using Mesh Warp

Use the Mesh Warp tool to deform images, layers, and selections. Clicking the tool places a grid, or mesh, on the image. The grid intersections have mesh points, or nodes, that you drag to create deformations.

The Mesh Warp tool is limited to deforming the interior of a selection. Use the Warp brush when you want to blend the selection into the rest of the image.

To use the Mesh Warp tool:

  1. On the Tools toolbar, select the Mesh Warp tool . (It is grouped with the Deform and Straighten tools.)

  2. Drag the nodes to deform the image:

  3. Click and drag a node to move it.

  4. Press Shift while dragging a node to move the entire row or column.

  5. Press Ctrl while dragging a node to deform the row or column into a smooth curve.

  1. Choose settings from the Tool Options palette:

  2. Draft quality Choose the quality of the preview that displays while you edit the mesh nodes. Increasing the quality can slow the warping process in large images.

  3. Final apply Clear the Best Quality check box to create a finished mesh warp at the lower quality of the mesh preview.

  4. Mesh horizontal and Mesh vertical The values indicate the number of mesh lines within the frame. The border lines are not counted. You can change the grid size as you work. Entering new Mesh values reconfigures the grid and returns it to the original position. The image retains any deformations you have made.

  5. Symmetric Mark this check box to create a grid that evenly covers the image. If you image is not square, there will be different values for Mesh horizontal and Mesh vertical. Clear this check box to adjust the Mesh horizontal and Mesh vertical independently.

  6. Show grid Clear the check box to hide the mesh warp grid.

  1. You can save the position of the mesh nodes (the deformation map) to apply the deformation to other images. For instructions see Saving and Loading a Deformation Map.

  2. Use Edit > Undo to undo changes to the mesh as you edit.

  3. To apply the mesh warp, double-click the image or click Apply . To cancel the mesh warp, click Cancel .
