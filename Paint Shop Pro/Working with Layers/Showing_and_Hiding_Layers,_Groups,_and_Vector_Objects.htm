Showing and Hiding Layers, Groups, and Vector Objects

Use the Visibility buttons on the Layers palette to make layers, layer groups, or vector objects visible or invisible in the image. These items remain in the image but are hidden. When a layer is visible, the Visibility button displays . When a layer is hidden, the Visibility button displays .

To show or hide individual layers:

Do one of the following on the Layers palette:

  • Click the Visibility button for each layer or Layer Group you want to show or hide.

Note: Hiding a layer group hides all of the layers within that group.

  • To show one layer only, right-click a layer name and choose View > Current Only.

To show all layers:

Choose Layers > View > All, or right-click a layer name and choose View > All from the context menu.

To invert which layers are shown or hidden:

  • Right-click a layer name and choose View > Invert from the context menu. All previously visible items become invisible and all invisible items become visible.

  • Right click the layer name of any layer that is a member of the link set, and choose View > Invert.

To show or hide layer groups or vector objects:

On the Layers palette, click the Visibility button for each vector layer, vector layer group, or vector object you want to show or hide.
