Editing Layer Properties

Layer properties include most of what you see on the Layers palette: the name, layer grouping and linking, layer opacity, blending, transparency protection, and layer icon highlight colors. You can change the layer properties for raster, vector, or Art Media layers.

To change properties:

Do one of the following to display the Layer Properties dialog:

  • Choose Layers > Properties to open the Layers Properties dialog.

  • Right-click on a layer and choose Properties from the context menu.

  • Double-click a layer name to display the Layer Properties dialog.

  • Make changes in the right panel of the Layers palette.

Note: For information about editing an adjustment layer, see Editing an Adjustment Layer.
