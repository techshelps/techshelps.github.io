Converting Objects to Paths

The Convert to Path command allows you to convert any selected vector object to a fully-editable path (as though it were created by the Pen tool). This allows you to use the Pen tool's full complement of node editing commands to modify the converted object.

To use this command:

  1. Use the Object Selection tool to select one or more vector objects.

  2. Choose Objects > Convert to Path, or right-click and choose this command from the context menu. The object(s) will be converted to a path which is now editable with the Pen tool's Edit mode.

Note: Be aware of the following points about this command:

  • Each of the vector tools (Rectangle, Ellipse, Symmetric Shape, Preset Shape, and vector Text) has this command available in the tool's context menu. The Pen tool, when its Create on Vector tool option is checked, automatically creates path objects.

  • When using this command on text fit to a path, this command will first convert the text to a path, and then convert the vector object if necessary.

  • For vector text, applying this command is the equivalent of applying the Objects > Convert Text to Curves > As Single Shape command.
