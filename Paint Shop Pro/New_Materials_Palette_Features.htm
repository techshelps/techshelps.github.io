New Materials Palette Features

Changes to the Materials palette include:

  • A new Frame tab, providing another method of setting color that may be more intuitive to some users.

  • What was formerly called the Color tab is now called the Rainbow tab. The functionality remains the same.

  • When using the Mixer palette and the Art Media tools, some behaviors of the Materials palette are modified. Refer to the linked topics below for more information.
