Specifying the HTML Code for Colors

With the Jasc Color Picker, you can view the HTML (Hypertext Markup Language) code for each color you select. You can also specify a color by entering the HTML code.

Note

By default, Paint Shop Pro displays decimal (base 10) numbers in the Jasc Color Picker. You can also display hexadecimal (base 16) numbers the format of HTML color codes. For more information, see Display and Caching Preferences.

To specify or view HTML code for colors:

  1. Display the Jasc Color Picker.

  2. Do one of the following:

  3.  To specify a color using its HTML code, type the code in the HTML edit box.

  4.  To view the HTML code of a particular color, choose a color by other methods (such as entering RGB values or selecting from the color wheel), and then view the number in HTML edit box.
