WindowFromDC

The WindowFromDC function returns the handle of the window associated with the given display device context (DC). Output functions that use the given device context draw into this window.

HWND WindowFromDC(

HDC hDC // handle of window
);  

Parameters

hDC

Identifies the device context from which a handle for the associated window is to be retrieved.

Return Values

If the function succeeds, the return value is the handle of the window associated with the given display device context.

If the function fails, the return value is NULL.
