ImmGetCompositionWindow

[Now Supported on Windows NT]

The ImmGetCompositionWindow function gets information about the composition window.

BOOL ImmGetCompositionWindow(

HIMC hIMC,  
LPCOMPOSITIONFORM lpCompForm  
);  

Parameters

hIMC

Handle to the input context.

lpCompForm

Pointer to the COMPOSITIONFORM structure that receives information about the composition window.

Return Values

If the function succeeds, the return value is a nonzero value.

If the function fails, the return value is zero.
