SOPutCharFontByName

Sets the font for characters in text.

VOID SOPutCharFontByName(

WORD wFontType,  
WORD pFontName,  
HPROC reserved  
);  

Parameters

wFontType

Font family. This parameter can be one of these values:

SO_FAMILYDECORATIVE Fancy display font
SO_FAMILYMODERN Fixed width
SO_FAMILYROMAN Variable width with serifs
SO_FAMILYSCRIPT Handwriting
SO_FAMILYSWISS Variable width without serifs
SO_FAMILYSYMBOL Symbol font
SO_FAMILYUNKNOWN Not known

pFontName

Address of a null-terminated string specifying the name of the font.

reserved

Reserved; do not use.

Return Values

None.
