NEWHEADER

The NEWHEADER structure contains the number of icon or cursor components in a resource group.

struct NEWHEADER {

WORD Reserved;

WORD ResType;

WORD ResCount;

} NEWHEADER, *PNEWHEADER;

NEWHEADER NewHeader;

Members

Reserved

Reserved; must be zero.

ResType

Specifies the resource type. This member must be 14 for icons and 12 for cursors.

ResCount

Specifies the number of icon or cursor components in the resource group.

Remarks

One or more RESDIR structures immediately follow the NEWHEADER structure in the .RES file. The ResCount member specifies the number of RESDIR structures.
