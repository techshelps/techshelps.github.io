Tooltip Control Reference

Adding and Deleting Tools

TOOLINFO

TTM_DELTOOL

TTM_NEWTOOLRECT
TTM_RELAYEVENT
TTM_SETDELAYTIME
TTN_POP
TTN_SHOW

Tool Text

TOOLTIPTEXT
TTM_GETTEXT
TTM_UPDATETIPTEXT
TTN_NEEDTEXT

Tooltip Hit Testing

TTHITTESTINFO
TTM_HITTEST
TTM_WINDOWFROMPOINT

TTM_ENUMTOOLS

TTM_GETTOOLINFO
