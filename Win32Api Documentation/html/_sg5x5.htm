ListView_SetColumn

The ListView_SetColumn macro sets the attributes of a list view column. You can use this macro or explicitly send the LVM_SETCOLUMN message.

BOOL ListView_SetColumn(

HWND hwnd,  
int iCol,  
LV_COLUMN FAR *pcol  
);  

Parameters

hwnd

Handle to the list view control.

iCol

Index of the column.

pcol

Pointer to an LV_COLUMN structure that contains the new column attributes. The mask member specifies which column attributes to set.

If the mask member specifies the LVCF_TEXT value, the pszText member is the pointer to a null-terminated string and the cchTextMax member is ignored.

Return Values

Returns TRUE if successful or FALSE otherwise.

See Also

LV_COLUMN, LVM_SETCOLUMN
