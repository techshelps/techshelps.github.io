LVM_GETORIGIN

The LVM_GETORIGIN message retrieves the current view origin for a list view control. You can send this message explicitly or by using the ListView_GetOrigin macro.

LVM_GETORIGIN

wParam = 0;

lParam = (LPARAM) (LPPOINT) lpptOrg;

Parameters

lpptOrg

Pointer to a POINT structure that receives the view origin.

Return Values

Returns TRUE if successful or FALSE if the current view is list or report view.
