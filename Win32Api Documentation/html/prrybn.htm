SERVER_INFO_1512

The SERVER_INFO_1512 structure contains information about the specified server, including name, platform, type of server, and associated software.

typedef struct _SERVER_INFO_1512 {
DWORD sv1512_maxnonpagedmemoryusage;
} SERVER_INFO_1512, *PSERVER_INFO_1512, *LPSERVER_INFO_1512;

Members

sv1512_maxnonpagedmemoryusage

Specifies the maximum size of nonpaged memory that the server can have allocated at any time. Adjust this member if you want to administer memory quota control.
