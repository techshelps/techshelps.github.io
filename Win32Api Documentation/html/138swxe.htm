TB_INSERTBUTTON

The TB_INSERTBUTTON message inserts a button in a toolbar.

TB_INSERTBUTTON

wParam = (WPARAM) iButton;

lParam = (LPARAM) (LPTBBUTTON) lpButton;

Parameters

iButton

Zero-based index of a button. The message inserts the new button to the left of this button.

lpButton

Pointer to a TBBUTTON structure containing information about the button to insert.

Return Values

Returns TRUE if successful or FALSE otherwise.
