Delay-Rendered Formats

You can specify the list of clipboard formats to be delay-rendered under the HKEY_CLASSESROOT\CLSID\{clsid}\DataFormats\PriorityCacheFormats key. The IDataObject of a scrap object with this CLSID will offer these formats in addition to the native data and cached data. When the drop target requests one of these formats, the shell runs the application and renders the format from the active object. However, you should avoid using this mechanism because it does not work if the server is not available or if the application is a non-OLE application.
