TTM_NEWTOOLRECT

The TTM_NEWTOOLRECT message sets a new bounding rectangle for a tool.

TTM_NEWTOOLRECT

wParam = 0;

lParam = (LPARAM) (LPTOOLINFO) lpti;

Parameters

lpti

Pointer to a TOOLINFO structure. The hwnd and uId members identify a tool, and the rect member specifies the new bounding rectangle. Before sending this message, you must set the cbSize member of this structure to sizeof(TOOLINFO).

Return Values

No return value.
