TVN_BEGINRDRAG

The TVN_BEGINRDRAG notification message notifies a tree-view control's parent window about the initiation of a drag-and-drop operation involving the right mouse button. This notification message is sent in the form of a WM_NOTIFY message.

TVN_BEGINRDRAG

pnmtv = (NM_TREEVIEW FAR *) lParam

Parameters

pnmtv

Pointer to an NM_TREEVIEW structure. The itemNew member is a TV_ITEM structure that contains valid information in the hItem, state, and lParam members about the item to be dragged. The ptDrag member specifies the current screen coordinates of the mouse.

Return Values

No return value.

See Also
