ListView_SetItemPosition

The ListView_SetItemPosition macro moves an item to a specified position in a list view control, which must be in icon or small icon view. You can use this macro or explicitly send the LVM_SETITEMPOSITION message.

BOOL ListView_SetItemPosition(

HWND hwnd,  
int i,  
int x,  
int y  
);  

Parameters

hwnd

Handle to the list view control.

i

Index of the list view item.

x and y

New position of the item's upper-left corner, in view coordinates.

Return Values

Returns TRUE if successful or FALSE otherwise.

Remarks

If the list view control has the LVS_AUTOARRANGE style, the list view control is arranged after the position of the item is set.
