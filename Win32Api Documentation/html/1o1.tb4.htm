IShellLink::GetWorkingDirectory

[Now Supported on Windows NT]

Retrieves the name of the working directory for a shell link object.

HRESULT STDMETHODCALLTYPE GetWorkingDirectory(

ISHELLLINK FAR *pShlLnk,  
LPSTR pszDir,  
int cchMaxPath  
);  

Parameters

pShlLnk

Pointer to the IShellLink interface. In C++, this parameter is implicit.

pszDir

Pointer to a buffer that receives the name of the working directory.

cchMaxPath

Maximum number of characters to copy to the buffer pointed to by pszDir. The name of the working directory is truncated if it is longer than the maximum specified by this parameter.

Return Values

Returns NOERROR if successful or an OLE-defined error value otherwise.
