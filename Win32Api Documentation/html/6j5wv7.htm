BindImage

The BindImage function computes the virtual address of each function that is imported from all DLLs.

BOOL BindImage(

IN LPSTR ImageName,  
IN LPSTR DllPath,  
IN LPSTR SymbolPath  
);  

Parameters

ImageName

The filename of the file to be bound. This can contain only a filename, a partial path, or a full path.

DllPath

A root path to search for ImageName if the filename contained in ImageName cannot be opened.

SymbolPath

A root path to search for the corresponding symbol file.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, then the return value is FALSE. To retrieve extended error information, call GetLastError.

Remarks

Calling BindImage is equivalent to the following call to BindImageEx:

BindImageEx( 0, ImageName, DllPath, SymbolPath, NULL );

For more information, see BindImageEx.
