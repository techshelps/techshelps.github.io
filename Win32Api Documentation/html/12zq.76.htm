USER_INFO_1013

The USER_INFO_1013 structure contains user information for network accounts.

typedef struct _USER_INFO_1013 {
LPWSTR usri1013_parms;
} USER_INFO_1013, *PUSER_INFO_1013, *LPUSER_INFO_1013;

Members

usri1013_parms

Points to a Unicode string that is set aside for use by applications. The string can be a null string, or it can have any number of characters before the terminating null character. Microsoft products use this member to store user configuration information. Do not modify this information. The Windows NT components that use this member are services for Macintosh, file and print services for Netware, and the Remote Access Server (RAS).
