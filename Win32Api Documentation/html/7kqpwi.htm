LVN_BEGINRDRAG

The LVN_BEGINRDRAG notification message notifies a list view control's parent window that a drag-and-drop operation involving the right mouse button is being initiated. This notification message is sent in the form of a WM_NOTIFY message.

LVN_BEGINRDRAG

pnmv = (NM_LISTVIEW FAR *) lParam;

Parameters

pnmv

Pointer to an NM_LISTVIEW structure. The iItem member identifies the item being dragged, and the other members are zero.

Return Values

No return value.

See Also

NM_LISTVIEW, WM_NOTIFY
