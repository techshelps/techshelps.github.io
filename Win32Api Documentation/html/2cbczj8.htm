HDN_ENDTRACK

The HDN_ENDTRACK notification message notifies a header control's parent window that the user has finished dragging a divider. This notification message sent in the form of a WM_NOTIFY message.

HDN_ENDTRACK

phdn = (HD_NOTIFY FAR *) lParam;

Parameters

phdn

Pointer to an HD_NOTIFY structure that contains information about the header control and the item whose divider was dragged.

Return Values

No return value.
