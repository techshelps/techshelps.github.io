Object Working Directory

The working directory is the directory where the corresponding object of a shortcut loads or stores files when the user does not identify a specific directory. A link file contains the name of the working directory for the corresponding object. An application can set the name of the working directory for the corresponding object by using the IShellLink::SetWorkingDirectory member function and can retrieve the name of the current working directory for the corresponding object by using the IShellLink::GetWorkingDirectory member function.
