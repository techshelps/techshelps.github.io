UDM_GETACCEL

The UDM_GETACCEL message retrieves acceleration information for an up-down control.

UDM_GETACCEL

wParam = (WPARAM) cAccels;

lParam = (LPARAM) (LPUDACCEL) paAccels;

Parameters

cAccels

Number of elements in the array specified by paAccels.

paAccels

Address of an array of UDACCEL structures that receives acceleration information.

Return Values

The return value is the number of accelerator structures retrieved.
