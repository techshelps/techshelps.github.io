TOKEN_USER

The TOKEN_USER structure identifies the user associated with an access token.

typedef struct _TOKEN_USER { // tu

SID_AND_ATTRIBUTES User;

} TOKEN_USER;

Members

User

Specifies an SID_AND_ATTRIBUTES structure representing the user associated with the access token. There are currently no attributes defined for user security identifiers (SIDs).
