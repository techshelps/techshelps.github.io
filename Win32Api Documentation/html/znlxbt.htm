USEROBJECTFLAGS

The USEROBJECTFLAGS structure contains information about a window station or desktop handle.

typedef struct tagUSEROBJECTFLAGS {

BOOL fInherit;

BOOL fReserved;

DWORD dwFlags;

} USEROBJECTFLAGS;

Members

fInherit

Specifies whether the handle is inherited when a new process is created. If this member is TRUE, new processes will inherit the handle.

fReserved

Reserved for future use. This member must be FALSE.

dwFlags

For window stations, this member can contain the following window station attribute flag:

Value Description
WSF_VISIBLE Window station has visible display surfaces.

For desktops, the dwFlags member can contain the following desktop control flag:

Value Description
DF_ALLOWOTHERACCOUNTHOOK Allows processes running in other accounts on the desktop to set hooks in this process.
