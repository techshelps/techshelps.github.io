LVM_UPDATE

The LVM_UPDATE message updates a list view item. If the list view control has the LVS_AUTOARRANGE style, this macro causes the list view control to be arranged. You can send this message explicitly or by using the ListView_Update macro.

LVM_UPDATE

wParam = (WPARAM) iItem;

lParam = 0;

Parameters

iItem

Index of the item to update.

Return Values

Returns TRUE if successful or FALSE otherwise.
