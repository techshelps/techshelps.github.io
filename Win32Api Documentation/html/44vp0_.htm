POINT

The POINT structure defines the x- and y- coordinates of a point.

typedef struct tagPOINT { // pt

LONG x;

LONG y;

} POINT;

Members

x

Specifies the x-coordinate of the point.

y

Specifies the y-coordinate of the point.
