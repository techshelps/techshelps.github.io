ImageList_Replace

The ImageList_Replace function replaces an image in an image list with a new image.

BOOL ImageList_Replace(

HIMAGELIST himl,  
int i,  
HBITMAP hbmImage,  
HBITMAP hbmMask  
);  

Parameters

himl

Handle to the image list.

i

Index of the image to replace.

hbmImage

Handle to the bitmap that contains the image.

hbmMask

Handle to the bitmap that contains the mask. If no mask is used with the image list, this parameter is ignored.

Return Values

If the function succeeds, the return value is nonzero.

If the function fails, the return value is zero.

Remarks

The ImageList_Replace function copies the bitmap to an internal data structure. Be sure to use the DeleteObject function to delete hbmImage and hbmMask after the function returns.
