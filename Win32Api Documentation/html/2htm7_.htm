SnmpMgrClose

The SnmpMgrClose function closes communications sockets and data structures associated with the specified session.

BOOL SnmpMgrClose(

LPSNMP_MGR_SESSION session // SNMP session pointer
);  

Parameters

session

[in] Points to an internal structure that specifies the session to close.

Return Values

If the function succeeds, the return value is TRUE.

If the function fails, the return value is FALSE. This function may return Windows Sockets Error codes.

See Also

SnmpMgrOpen, SnmpMgrRequest
