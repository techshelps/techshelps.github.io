IRichEditOle::GetClipboardData

Retrieves a clipboard object for a range in an edit control.

HRESULT GetClipboardData(

CHARRANGE FAR * lpchrg,  
DWORD reco,  
LPDATAOBJECT FAR * lplpdataobj  
);  

Parameters

lpchrg

Pointer to the CHARRANGE structure specifying the range to create the clipboard object for.

reco

Clipboard operation flag. Can be one of these values.

RECO_COPY Copy to the clipboard.
RECO_CUT Cut to the clipboard.
RECO_DRAG Drag operation (drag and drop)
RECO_DROP Drop operation (drag and drop).
RECO_PASTE Paste from the clipboard.

lplpdataobj

Pointer to the DATAOBJECT structure that receives the clipboard object representing the range specified in the lpchrg parameter.

Return Values

Returns success status.
