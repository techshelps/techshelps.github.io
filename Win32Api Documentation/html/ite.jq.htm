USER_INFO_1006

The USER_INFO_1006 structure contains user information for network accounts.

typedef struct _USER_INFO_1006 {

} USER_INFO_1006, *PUSER_INFO_1006, *LPUSER_INFO_1006;

Members

usri1006_home_dir

Points to a Unicode string containing the path of the home directory for the user specified in the user_name member. The string can be null.
