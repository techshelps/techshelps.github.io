IMN_SETOPENSTATUS

The IMN_SETOPENSTATUS message is sent when the open status of the input context is updated. The application receives this notification message through the WM_IME_NOTIFY message.

Return Values

No return value.

Remarks

The application can get information about the open status by using the ImmGetOpenStatus function.
