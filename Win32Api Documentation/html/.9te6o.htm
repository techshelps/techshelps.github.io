TabCtrl_AdjustRect

The TabCtrl_AdjustRect macro calculates a tab control's display area given a window rectangle or calculates the window rectangle that would correspond to a specified display area. You can use this macro or explicitly send the TCM_ADJUSTRECT message.

VOID TabCtrl_AdjustRect(

HWND hwnd,  
BOOL fLarger,  
RECT FAR *prc  
);  

Parameters

hwnd

Handle to the tab control.

fLarger

Operation to perform. If this parameter is TRUE, prc specifies a display rectangle and receives the corresponding window rectangle. If this parameter is FALSE, prc specifies a window rectangle and receives the corresponding display area.

prc

Pointer to a RECT structure that specifies the given rectangle and receives the calculated rectangle.

Return Values

No return value.

See Also

RECT, TCM_ADJUSTRECT
