SOPutParaAlign

Sets the alignment for a paragraph.

VOID SOPutParAlign(

WORD wType,  
HPROC reserved  
);  

Parameters

wType

Type of alignment. This parameter can be the SO_ALIGNLEFT, SO_ALIGNRIGHT, SO_ALIGNCENTER, or SO_ALIGNJUSTIFY value.

reserved

Reserved; do not use.

Return Values

None.
