ListView_GetCallbackMask

The ListView_GetCallbackMask macro retrieves the callback mask for a list view control. You can use this macro or explicitly send the LVM_GETCALLBACKMASK message.

UINT ListView_GetCallbackMask(

HWND hwnd  
);  

Parameters

hwnd

Handle to the list view control.

Return Values

Returns the callback mask.

See Also

LVM_GETCALLBACKMASK, LVM_SETCALLBACKMASK
