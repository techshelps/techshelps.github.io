TVM_SETIMAGELIST

The TVM_SETIMAGELIST message sets the normal or state image list for a tree-view control and redraws the control using the new images. You can send this message explicitly or by using the TreeView_SetImageList macro.

TVM_SETIMAGELIST

wParam = (WPARAM) iImage;

lParam = (LPARAM) (HIMAGELIST) himl;

Parameters

iImage

Type of image list to set. For a list of possible values, see the description of the TVM_GETIMAGELIST message.

himl

Handle to the image list. If himl is NULL, all images are removed from the tree-view control.

Return Values

Returns the handle to the previous image list, if any, or NULL otherwise.

See Also

TreeView_SetImageList, TVM_GETIMAGELIST
