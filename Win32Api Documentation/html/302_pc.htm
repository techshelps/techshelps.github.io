TreeView_GetChild

The TreeView_GetChild macro retrieves the first child item of the specified tree-view item. You can use this macro, or you can explicitly send the TVM_GETNEXTITEM message with the TVGN_CHILD flag.

HTREEITEM TreeView_GetChild(

hwnd,  
hitem,  
);  

Parameters

hwnd

Handle to the tree-view control.

hitem

Handle to a tree-view item.

Return Values

Returns the handle to the item if successful or NULL otherwise.

TreeView_GetNextItem, TreeView_GetNextSibling, TreeView_GetParent, TreeView_GetPrevSibling, TVM_GETNEXTITEM
