PSM_QUERYSIBLINGS

The PSM_QUERYSIBLINGS message forwards the message to each page in the property sheet. If a page returns a nonzero value, the property sheet does not send the message to subsequent pages. You can send the PSM_QUERYSIBLINGS message explicitly or by using the PropSheet_QuerySiblings macro.

PSM_QUERYSIBLINGS

wParam = (WPARAM) param1;

lParam = (LPARAM) param2;

Parameters

param1

First application-defined parameter.

param2

Second application-defined parameter.

Return Values

Returns the nonzero value from a page in the property sheet, or zero if no page returns a nonzero value.
