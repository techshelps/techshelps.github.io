TVN_GETDISPINFO

The TVN_GETDISPINFO notification message requests that a tree-view control's parent window provide information needed to display or sort an item. This notification message is sent in the form of a WM_NOTIFY message.

TVN_GETDISPINFO

ptvdi = (TV_DISPINFO FAR *) lParam

Parameters

ptvdi

Pointer to a TV_DISPINFO structure. The item member is a TV_ITEM structure whose mask, hItem, state, and lParam members specify the type of information required. You must fill the members of the structure with the appropriate information.

Return Values

No return value.
