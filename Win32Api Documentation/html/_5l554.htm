TreeView_GetVisibleCount

The TreeView_GetVisibleCount macro obtains the number of items that are fully visible in the client window of a tree-view control. You can use this macro or explicitly send the TVM_GETVISIBLECOUNT message.

UINT TreeView_GetVisibleCount(

hwnd  
);  

Parameters

hwnd

Handle to the tree-view control.

Return Values

Returns the number of items that are fully visible in the client window of the tree-view control.

Remarks

Note that the return value is the number of fully-visible items. If you can see all of 20 items, and part of one more item, the return value is 20, not 21.
