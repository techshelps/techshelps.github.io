PropSheet_UnChanged

The PropSheet_UnChanged macro informs a property sheet that information in a page has reverted to the previously saved state. The property sheet disables the Apply Now button if no other pages have registered changes with the property sheet. You can use this macro or explicitly send the PSM_UNCHANGED message.

VOID PropSheet_UnChanged(hPropSheetDlg, hwndPage)

hPropSheetDlg,  
hwndPage  
);  

Parameters

hPropSheetDlg
