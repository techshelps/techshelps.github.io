LVM_DELETEITEM

The LVM_DELETEITEM message removes an item from a list view control. You can send this message explicitly or by using the ListView_DeleteItem macro.

LVM_DELETEITEM

wParam = (WPARAM) (int) iItem;

lParam = 0;

Parameters

iItem

Index of the list view item to delete.

Return Values

Returns TRUE if successful or FALSE otherwise.
