Windows NT Files for SNMP

The following table describes the files that relate to the SNMP Service.

Filename Description
DHCPMIB.DLL DHCP SNMP agent (not in Windows NT 3.1). Only installed on DHCP servers.
INETMIB2.DLL MIB II extension agent DLL.
LMMIB2.DLL LAN Manager MIB 2 extension agent DLL.
MGMTAPI.DLL SNMP component; Management API library.
WINSMIB.DLL WINS SNMP agent (not in Windows NT 3.1). Only installed on WINS servers.
SNMP.EXE SNMP Agent Service; proxy agent that listens for requests and hands them off to the appropriate network provider.
SNMPTRAP.EXE Receives SNMP traps and forwards them to MGMTAPI.DLL.
SNMPAPI.DLL SNMP Utilities DLL.
MIB.BIN SNMP component; SNMP Service.

For complete installation instructions, refer to the Microsoft Windows Resource Kit.
