TBM_GETTIC

The TBM_GETTIC message retrieves the position of a tick mark in a trackbar.

TBM_GETTIC

wParam = (WPARAM) (WORD) iTic;

lParam = 0;

Parameters

iTic

Zero-based index identifying a tick mark.

Return Values

Returns the position of the specified tick mark or - 1 if iTic does not specify a valid index.
