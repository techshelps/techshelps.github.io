TabCtrl_RemoveImage

The TabCtrl_RemoveImage macro removes an image from a tab control's image list. You can use this macro or explicitly send the TCM_REMOVEIMAGE message.

void TabCtrl_RemoveImage(

HWND hwnd,  
int iImage  
);  

Parameters

hwnd

Handle to the tab control.

iImage

Index of the image to remove.

Return Values

No return value.

Remarks

The tab control updates each tab's image index, so each tab remains associated with the same image it had been.
