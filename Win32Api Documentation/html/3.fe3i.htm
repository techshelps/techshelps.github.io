Adding or Removing File Parsers

For performance reasons, the file viewer builds a cache of the file parsers in the system the first time the Quick View feature is used. This cache is stored in the registry. If a file parser is added or removed, this cache must be rebuilt. To make the system rebuild the cache, set verify data not equal to zero under the following key.

\\HKEY_LOCAL_MACHINE\SOFTWARE\SCC\Viewer Technology\MS1
