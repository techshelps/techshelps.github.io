TCM_GETIMAGELIST

The TCM_GETIMAGELIST message retrieves the image list associated with a tab control. You can send this message explicitly or by using the TabCtrl_GetImageList macro.

TCM_GETIMAGELIST

wParam = 0; // not used; must be zero

lParam = 0; // not used; must be zero

Return Values

Returns the handle to the image list if successful or NULL otherwise.
