TCM_GETITEMRECT

The TCM_GETITEMRECT message retrieves the bounding rectangle for a tab in a tab control. You can send this message explicitly or by using the TabCtrl_GetItemRect macro.

TCM_GETITEMRECT

wParam = (WPARAM) (int) iItem; // index of the tab

lParam = (LPARAM) (RECT FAR *) prc; // pointer to a structure to receive rectangle of tab

Parameters

iItem

Index of the tab.

prc

Pointer to a RECT structure that receives the bounding rectangle of the tab, in viewport coordinates.

Return Values

Returns TRUE if successful or FALSE otherwise.

See Also

RECT, TabCtrl_GetItemRect
