GetParent

The GetParent function retrieves the handle of the specified child window's parent window.

HWND GetParent(

HWND hWnd // handle of child window
);  

Parameters

hWnd

Identifies the window whose parent window handle is to be retrieved.

Return Values

If the function succeeds, the return value is the handle of the parent window. If the window has no parent window, the return value is NULL. To get extended error information, call GetLastError.
