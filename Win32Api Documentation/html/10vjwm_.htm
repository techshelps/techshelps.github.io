IMC_SETCONVERSIONMODE

An application sends the IMC_SETCONVERSIONMODE message to an IME window to set current conversion mode.

msg = (UINT) WM_IME_CONTROL;
wParam = (WPARAM) IMC_SETCONVERSIONMODE;
lParam = (LPARAM) cMode;

Parameters

lParam

Conversion mode. Can be one of the values given in the "IME Conversion Modes" table in Input Method Editor Constants.

Return Values

Returns zero if successful, nonzero otherwise.
