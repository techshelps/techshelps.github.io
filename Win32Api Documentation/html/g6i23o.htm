SERVER_INFO_1005

The SERVER_INFO_1005 structure contains information about the specified server, including name, platform, type of server, and associated software.

typedef struct _SERVER_INFO_1005 {
LPTSTR sv1005_comment;
} SERVER_INFO_1005, *PSERVER_INFO_1005, *LPSERVER_INFO_1005;

Members

sv1005_comment

Points to a Unicode string that contains a comment describing the server. The comment can be null.
