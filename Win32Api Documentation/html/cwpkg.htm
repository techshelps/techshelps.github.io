ListView_GetTopIndex

The ListView_GetTopIndex macro retrieves the index of the topmost visible item when in list or report view. You can use this macro or explicitly send the LVM_GETTOPINDEX message.

int ListView_GetTopIndex(

HWND hwnd  
);  

Parameters

hwnd

Handle to the list view control.

Return Values

Returns the index of the item if successful or zero if the list view control is in icon or small icon view.
