ListView_GetOrigin

The ListView_GetOrigin macro retrieves the current view origin for a list view control. You can use this macro or explicitly send the LVM_GETORIGIN message.

BOOL ListView_GetOrigin(

HWND hwnd,  
LPPOINT lpptOrg  
);  

Parameters

hwnd

Handle to the list view control.

lpptOrg

Pointer to a POINT structure that receives the view origin.

Return Values

Returns TRUE if successful or FALSE if the current view is list or report view.
