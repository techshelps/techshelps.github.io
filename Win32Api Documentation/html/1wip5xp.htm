TabCtrl_InsertItem

The TabCtrl_InsertItem macro inserts a new tab in a tab control. You can use this macro or explicitly send the TCM_INSERTITEM message.

int TabCtrl_InsertItem(

HWND hwnd,  
int iItem,  
const TC_ITEM FAR *pitem  
);  

Parameters

hwnd

Handle to the tab control.

iItem

Index of the new tab.

pitem

Pointer to a TC_ITEM structure that specifies the attributes of the tab.

Return Values

Returns the index of the new tab if successful or - 1 otherwise.

See Also

TC_ITEM, TCM_INSERTITEM
