CopyCursor

The CopyCursor function copies a cursor.

HCURSOR CopyCursor(

HCURSOR pcur // handle to cursor to copy
);  

Parameters

pcur

Identifies the cursor to be copied.

Return Values

If the function succeeds, the return value is the handle to the duplicate cursor.

If the function fails, the return value is NULL. To get extended error information, call GetLastError.

Remarks

CopyCursor enables an application or dynamic-link library (DLL) to obtain the handle to a cursor shape owned by another module. Then if the other module is freed, the application is still able to use the cursor shape.
