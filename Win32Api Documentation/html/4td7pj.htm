GetOEMCP

The GetOEMCP function retrieves the current OEM code-page identifier for the system. (OEM stands for original equipment manufacturer.)

UINT GetOEMCP(VOID)

Parameters

This function has no parameters.

Return Values

If the function succeeds, the return value is the current OEM code-page identifier for the system or a default identifier if no code page is current.
