SB_GETRECT

The SB_GETRECT message retrieves the bounding rectangle of a part in a status window.

SB_GETRECT

wParam = (WPARAM) iPart;

lParam = (LPARAM) (LPRECT) lprc;

Parameters

iPart

Zero-based index of the part whose bounding rectangle is to be retrieved.

lprc

Pointer to a RECT structure that receives the bounding rectangle.

Return Values

If the operation succeeds, the return value is TRUE.

If the operation fails, the return value is FALSE.
