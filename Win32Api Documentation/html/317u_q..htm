EM_GETLIMITTEXT

An application sends the EM_GETLIMITTEXT message to retrieve the current text limit, in characters, for an edit control.

EM_GETLIMITTEXT

wParam = 0; // not used, must be zero

lParam = 0; // not used, must be zero

Parameters

This message has no parameters.

Return Values

The return value is the text limit.
