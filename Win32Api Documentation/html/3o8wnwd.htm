IFileViewerSite::SetPinnedWindow

[Now Supported on Windows NT]

Sets a new pinned window.

When the user selects a new file to view, the shell directs the file viewer to display the new file in the pinned window instead of creating a new window.

HRESULT SetPinnedWindow(

IFileViewerSite FAR *pFileVS,  
HWND *hwnd  
);  

Parameters

pFileVs

Address of the IFileViewerSite interface. In C++, this parameter is implicit.

hwnd

Handle of the new pinned window or NULL if there is to be no pinned window.

Return Values

Returns the NOERROR value if successful or an OLE-defined error value otherwise.
