ListView_GetBkColor

The ListView_GetBkColor macro retrieves the background color of a list view control. You can use this macro or explicitly send the LVM_GETBKCOLOR message.

COLORREF ListView_GetBkColor(

HWND hwnd  
);  

Parameters

hwnd

Handle to the list view control.

Return Values

Returns the background color of the list view control.
