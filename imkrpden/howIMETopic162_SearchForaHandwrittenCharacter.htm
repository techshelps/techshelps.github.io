Search for a Handwritten Character

Use Handwriting Search to enter Hanja when you don't know the phonetic reading.

  1. On the Language Bar, click IME Pad (), then click Handwriting ().
    The Handwriting applet is displayed.

  2. Click Clear to erase any hand-drawn image in the input frame.

  3. Click Recog to enable automatic recognition after each stroke.

  4. Draw the unknown Hanja in the left input frame.
    If Recog was enabled the contents of the right frame change with each stroke. Otherwise click Recog now. A list of Hanja and other symbols similar to the character you have drawn appear in the right frame.

  5. When the Hanja you want to input appears in the list, click it.

  6. When you are finished, close the IME Pad window.

Tips

  • If the Handwriting applet is not available, try adding the applet to the Applet Bar.

  • If you make a mistake drawing a character in the input frame, click Revert to erase the last stroke drawn. If you click Clear, the entire character will be erased.

  • Note the following points when handwriting characters:
    You do not have to write neatly.
    Do not draw the same stroke twice.

  • To enlarge or shrink the frame for handwriting, drag the right edge of the frame.

  • If you draw characters with Recog on, each stroke drawn in the left frame will automatically be recognized. If Recog is off, automatic recognition will not be performed.
    When the Always Recognize (Character Auto Complete) check box in the Handwriting Properties dialog box is on, Recog will be continuously in the On state and will not be released even if you click Recognize. When the Always Recognize (Character Auto Complete) check box is cleared, you can toggle the Recog button.
    You can turn the Always Recognize (Character Auto Complete) check box in the Handwriting Properties dialog box on and off by double clicking the Recognize button. indicates On, and indicates Off.
    The Handwriting Properties dialog box will appear if you right-click Handwriting, then select Handwriting Properties.

  • To display Help while using IME Pad, click the IME Pad menu (), then select IME Pad Help.

  • Sometimes this help window may steal the input focus so input via the IME Pad is no longer possible. To enable input again, close the IME Pad for a moment. Then click the application window, and after making sure that the cursor is in the position you want characters to be input, click IME Pad again.

  • If the application you are using supports handwriting tips, Handwriting Tips will be displayed in the above-mentioned menu.
    For more information about Handwriting Tips, see Help for Handwriting Tips.
