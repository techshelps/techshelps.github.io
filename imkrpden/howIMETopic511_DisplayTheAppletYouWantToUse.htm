Add the Applet You Want to Use

If the applet you want to use is not displayed on the IME Pad applet bar or Applet menu, follow the instructions below to enable it.

  1. On the Language Bar, click IME Pad ().
    The IME Pad is displayed.

  2. Click the IME Pad menu (), select the Applet menu, then click User Settings.
    The Customize Applet Menu dialog box is displayed.

  3. Click the applet you wish to enable under Available Applets, then click Add ->.
    The applet you selected is moved to Current Applets.

  4. Click Close.
