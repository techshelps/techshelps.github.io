Search for Hanja by Radical Stroke Count

Use the Radical Applet to search for an unknown Hanja by its radical.

  1. On the Language Bar, click IME Pad (), then click Radical ().
    The Radical applet is displayed.

  2. Select the number of strokes in the radical of the unknown Hanja from the pull-down list. Radicals with the stroke count you have selected are displayed.

  3. Click on the radical of the Hanja you are looking for.
    Hanja with the selected radical are displayed.

  4. Click the Hanja you want to enter.
    The Hanja is input at the cursor position.

  5. When you are finished, close the IME Pad.

Tips

  • If the Radical applet is not displayed, try adding the applet to the Applet Bar.

  • Hanja with the same radical are sorted by stroke count.

  • Click Enlarged/Advanced view () to change the list display. Each time you click it, the list will alternate between Enlarged and Advanced view. You can also select between Enlarged view and Advanced view via the pull-down menu next to this button.
    Enlarged view Large characters are displayed
    Advanced view The candidate and its stroke count, sense, reading, and description are displayed.
  • Sometimes this help window may steal the input focus so input via the IME Pad is no longer possible. To enable input again, close the IME Pad for a moment. Then click the application window, and after making sure that the cursor is in the position you want characters to be input, click IME Pad again.
