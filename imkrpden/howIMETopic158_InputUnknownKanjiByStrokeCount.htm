Enter Hanja by Stroke Count

Using the Stroke Count applet, you can search for Hanja by stroke count. Useful if the Hanja's phonetic reading is not known.

  1. On the Language Bar, click IME Pad (), then click Stroke Count ().
    The Stroke Count applet is displayed.

  2. Select the number of strokes from the pull-down menu.
    Any Hanja with the selected stroke count are listed.

  3. Click the Hanja you wish to enter.
    The Hanja is input at the cursor position.

  4. When you are finished, close the IME Pad.

Tips

  • If the Stroke Count applet is not displayed, try adding the applet to the Applet Bar.

  • Hanja with the same number of strokes are sorted by radical.

  • Click Enlarged/Advanced view () to change the list display. Each time you click it, the list will alternate between Enlarged and Advanced view. You can also switch between Enlarged view and Advanced view by clicking the pull-down menu next to this button.
    Enlarged View Large characters are displayed
    Advanced View The candidate and its Radical, sense, phonetic reading, and description are displayed.
  • To display Help while using IME Pad select IME Pad Help from the IME Pad menu ().

  • You can learn about how to use each applet by selecting the Applet menu from the IME Pad menu (), then clicking Help for AppletName.

  • Sometimes this help window may steal the input focus so input via the IME Pad is no longer possible. To enable input again, close the IME Pad for a moment. Then click the application window, and after making sure that the cursor is in the position you want characters to be input, click IME Pad again.
