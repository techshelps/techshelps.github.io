Plotting problems

 Top Up Down  A A

The Plots menu calls Microsoft Excel to perform many plotting functions. This enables you to use all the functionality of Excel to customize the plots to your needs.

 

Many different versions of Excel have been published by Microsoft, and they have some incompatibilities. You may find that plotting fails with an error code, e.g., "1004".

 

Excel has a limit of 65,536 rows, so attempting to plot more than about 65,000 points will result in Excel failure.

 

The Winsteps-to-Excel interface is in a module in your Winsteps folder named winexcel.exe. Please replace this module with a module that matches your version of Excel. These are available from www.winsteps.com/problems.htm
