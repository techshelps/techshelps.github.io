 Top Up Down  A A

(controlled by LINELENGTH=, MRANGE=, MAXPAGE=, NAMLMP=, ISORT=, PSORT=, T1I#=, T1P#=)

 

You can use NAMLMP= to limit the number of characters of each name reported. LINELENGTH= changes the available space on the line. MAXPAGE= controls the length of the Table. MRANGE= controls the displayed range of measures. ISORT= and PSORT= control the sort order within rows.

 

Table 1.2 is printed if the item map can be squeezed into one page. Table 12 shows the complete item labels.

 

Table 1.3 is printed if the person map can be squeezed into one page. Table 15 shows the complete person labels.

 

"TSMST" summarize the left-hard and right-hand distributions. An "M" marker represents the location of the mean measure. "S" markers are placed one sample standard deviation away from the mean. "T" markers are placed two sample standard deviations away.

 

Note: These Tables can be copied-and-pasted into Excel spreadsheets, and then Excel "Text to columns" used to beautify the output.

Or get the map to look how you want on your screen by adjusting font sizes, type faces, MRANGE=, MAXPAGE=, etc., then use your PrintScreen key to make a copy of the screen. Paste into Word, PowerPoint, etc.
