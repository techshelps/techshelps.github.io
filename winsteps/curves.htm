CURVES= specifies which curves in Table 21 are to display. "1" to display the curves. "0" to omit the curve. Formerly also controlled Table 2, now done with T2SELECT=.

 

CURVES=000 indicates no curves are to be drawn - Table 21 will be skipped, unless STEPT3=N, in which case only the structure summaries are output.

 

CURVES=101 displays subtables 21.1, 21.3 (and 21.4, 21.6, ...) when Table 21 is selected.
