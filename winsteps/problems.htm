For Windows-related and other problems, see www.winsteps.com/problems.htm

 

Winsteps does not function under Windows-95 nor 3.1.

 

Error codes are reported by Winsteps for which no automatic action is possible.

 

Winsteps expects its input files to be in MS-DOS text file format, i.e., readable by a text editor such as NotePad.

If in doubt,

1. Open the Winsteps file in Microsoft Word (or equivalent)

2. "Save As": "Text with line breaks"

File may not have CR+LF line breaks

The file is not in standard "text" file format. This can be reported for files created on a Mac.

This can occur when a Facets control file is submitted to Winsteps.

Warning: IGNORED (too long):

The control-variable value is too long. The usable characters are actioned. The excess characters are ignored.

Oops!  Item labels = ..., but NI=...4

The count of item labels between &END and END LABELS does not match the number of items. The item labels on the Output Tables may be incorrect.
