W300= Output files in Winsteps 3.00 format = No

 Top Up Down  A A

New or revised statistics are added as extra or changed columns in IFILE= and PFILE= when they are introduced into Winsteps.

 

To revert to an earlier format of IFILE= and PFILE=, specify W300=Yes. This produces the PFILE= and IFILE= in the format of Winsteps 3.00 1/1/2000.

 

Example: IFILE= in current format:

 

;ACT          FILE FOR

;ENTRY MEASURE STTS COUNT    SCORE  ERROR IN.MSQ IN.ZSTD OUT.MS OUT.ZSTD DISPL PTME WEIGHT  DISCR G M NAME

     1    -.89  1    75.0    109.0    .23    .74  -1.97    .67  -1.89    .00    .64   1.00   1.06 0 R WATCH BIRDS

     2    -.61  1    75.0    116.0    .20    .76  -1.54    .56  -1.55    .00    .58   1.00   1.07 0 R READ BOOKS ON ANIMALS

 

IFILE= in W300=Yes format:

 

;ACT    FILE FOR

;ENTRY MEASURE ST COUNT  SCORE  ERROR IN.MSQ IN.ZSTD OUT.MS OUT.ZSTD DISPL CORR G M NAME

     1    -.89  1    75    109    .23    .74  -1.97    .67  -1.89    .00    .64 0 R WATCH BIRDS

     2    -.61  1    75    116    .20    .76  -1.54    .56  -1.55    .00    .58 0 R READ BOOKS ON ANIMALS

 

PFILE= in current format:

 

;PUPIL        FILE FOR

;ENTRY MEASURE STTS COUNT    SCORE  ERROR IN.MSQ IN.ZSTD OUT.MS OUT.ZSTD DISPL PTME WEIGHT  DISCR NAME

     1     .49  1    25.0     30.0    .35    .96   -.15    .84   -.43    .00    .69   1.00    .00 ROSSNER, MARC DANIEL

;    2    5.99  0    25.0     50.0   1.84   1.00    .00   1.00    .00    .00    .00   1.00    .00 ROSSNER, LAWRENCE F.

 

PFILE= in W300=Yes format:

 

;PUPIL  FILE FOR

;ENTRY MEASURE ST COUNT  SCORE  ERROR IN.MSQ IN.ZSTD OUT.MS OUT.ZSTD DISPL CORR NAME

     1     .49  1    25     30    .35    .96   -.15    .84   -.43    .00    .69 ROSSNER, MARC DANIEL

     2    5.99  0    25     50   1.84   1.00    .00   1.00    .00    .00    .00 ROSSNER, LAWRENCE F.

 

Notes:

TOTAL=YES is active for both current and old formats.

";" shown for extreme scores, such as person 2, in current format, but not in old format.

COUNT and SCORE are shown rounded to nearest integer in old format.
