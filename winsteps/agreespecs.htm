This is the dialog box for Table 35 from the Output Tables menu, and AGREEFILE= from Output Files menu.

 

 

% Paired Response Agreement

Display Table 35 plot: check this box to output Table 35, "Paired-person response agreement"

Output AGREEFILE=: check this box to ouput the AGREEFILE= agreement file.

 

Select persons to compare: enter the PSELECT= value, or * to select all persons

To omit persons from selection using PDELETE=, click on "Other Specifications" push-button.

 

Compare within classification groups: enter PSUBTOTAL= value to specify the groups. Leave blank if all persons are to be part of one classification-group.
Compare only within classification-groups: check the box if persons are not to be compared across classification-groups for the overall plots and AGREEFILE=

 

Place codes to be grouped together on the same line. This allows multiple classification groups to be combined into one group.

 

Update from EQFILE= reinitializes the  equivalent codes from the EQFILE= information in the control file.

 

Other Specifications: other control variables can be changed.

 

OK - launch the paried-agreement computation
