Data can be simulated with SIFILE= . See simulated data file.

 

SIRESAMPLE= 0

The simulated corresponds exactly with the original data

SIRESAMPLE= number

The simulated data file contains number person records. These are produced by:

1. SIMEASURE=Yes - resampling the person measures.

2. SIMEASURE=No - resampling the person response strings.
