SPSS functional check

 Top Up Down  A A

Winsteps can read and write SPSS ".sav" files. To do this it requires a functioning copy of the SPSS interface library "spssio32.dll".

 

When Winsteps is installed, it checks that a functioning SPSS interface exists, or else it installs a basic one.

 

If SPSS file conversion fails, or is the wrong version:

 

If you have SPSS on your computer:

1. Locate file spssio32.dll (it may be on your SPSS installation CD-ROM)

2. Copy it into your Winsteps folder

 

If you don't have SPSS on your computer:

1. Check that spssio32.dll or spssio32.hld are in your Winsteps program folder. Reinstall Winsteps if neither is present.

2. If spssio32.dll is not present, rename spssio32.hld to spssio32.dll
