For ease in comparing the outputs from multiple runs, and to force consistent x-axis scaling by using MRANGE=, T1I#= and T1P#=. Choose T1P#= to be the largest number of persons summarized by one "#" from any of the separate runs.

 

T1P#= can increase the number of persons for each # above the value computed by Winsteps.

 

Example: In one run, the bottom of Table 1 (or Table 12) states that

   EACH "#" IN THE PERSON COLUMN IS 250 PERSONS

In another run: EACH "#" IN THE PERSON COLUMN IS 300 PERSON

To make the runs visually comparable, specify the bigger value:

  T1P#=300
