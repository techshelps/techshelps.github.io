Status Dialog Box: Active Jobs Tab

This tab shows information about active print jobs. It appears when you click the More Status button in the status area at the bottom of a print driver tab, and then select the Active Jobs tab.

Deleting active jobs

  1. Click Active Jobs.
  2. Select the print job.
  3. Click Delete Job.

Using menus

  • File menu
    • Refresh: Refreshes the table shown on the tab.
    • Exit: Closes the Status dialog box.
  • View menu
    • Show only my jobs or Show all jobs: Select whether to show only your jobs in the table on this tab or show everyone's jobs.
    • Delete selected jobs: Deletes the jobs selected in the active jobs table. You can only delete your jobs.
  • Notification menu
    • <notification type>: Select whether you want to be notified when your job has completed and/or failed.
    • <alert sound>: Select whether to play an alert sound when you are notified.
  • Help menu

    • Note: Some features are only supported by specific printers, operating systems, or driver types. You may see more features or options listed than what your printer supports.

    • Help: Displays this help file.
    • About: Displays a dialog box with your current print driver version number and copyright information.
    • CentreWare Internet Services: Displays your printer's settings. It also manages, configures, and monitors network printers.
    • PhaserSMART: Diagnoses problems with your network printer and proposes solutions. It also helps you troubleshoot issues.
    • Xerox® Support Centre: Provides a central location for accessing user manuals and videos, printer and supplies status, and more.
