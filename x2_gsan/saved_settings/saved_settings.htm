Saved Settings Area

On the bottom of the print driver tabs.

You can associate the current driver settings with a name so that you can easily call up the same settings for a similar print job. For example, you may want to create a setting called "Presentation Review" that uses automatic 2-sided printing and prints 4 pages per side. Whenever you want to print a review copy before a presentation, you can select "Presentation Review" from the Saved Settings menu to automatically use these settings for that print job.

  • Driver Defaults: Select this item to return all settings on the print driver tabs to the default values.
  • <application> Defaults: Select this item to return all settings on the print driver tabs to the defaults for the current application.
  • <saved setting names>: Select a saved setting name to return all settings on the print driver tabs to the settings saved under this name.
  • Save As: Select this item to open the Save As dialog box so that you can save the new saved setting. Select Share This Saved Setting to share it.
  • Rename: Select this item to open the Rename dialog box so that you can specify a new name for the current saved setting.
  • Save As <application> Defaults: Select this item to save the current settings as the default for jobs printed from the current application.
  • Delete: Select this item to delete the current saved setting.
  • Share: Select this item to share the current setting. When the Share Saved Settings dialog box appears, select the setting, and click Share Saved Setting.

Note: When you select a saved setting, a check mark appears next to the name. If you make more changes to driver settings, an asterisk appears after the name.

To create a new saved setting:

  1. Select the settings that you want to save on the print driver tabs.
  2. Select Save As from the Saved Settings menu.
  3. Type the name for the new setting in the Save As dialog box. Select Share this Saved Setting to share it.
  4. Click the OK button. This adds the new saved setting name to the menu and selects it.

To use a saved setting, select the saved setting name from the menu. If you want to restore the driver settings to the default, select Driver Defaults from the menu.

To edit an existing saved setting by using Save As:

  1. Select the saved setting that you want to change.
  2. Make the desired changes to the settings on the print driver tabs.
  3. Select Save As from the Saved Settings menu.
  4. Type the existing saved setting name in the Save As dialog box.
  5. Click the OK button.
  6. When prompted, click the Yes button to replace the settings saved for this name.

To delete a saved setting:

  1. Select the saved setting name.
  2. Select Delete.
  3. When prompted, click the Yes button to delete the saved setting name.
