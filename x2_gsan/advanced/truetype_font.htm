TrueType Font

On the Advanced tab.

TrueType is a trademark for a scalable font technology developed by Apple. Scalable fonts are fonts whose size can be modified without causing distortion on the screen or in print. TrueType fonts also print at the full resolution offered by the printer.

Select one of the following settings to specify which fonts are used:

  • Substitute with Device Font: Uses the fonts residing in the printer rather than download TrueType fonts with the print job.

    Note: This setting permits faster printing. However, you might lose special characters not supported by the printer font.
  • Download as Softfont: Downloads all TrueType fonts to the printer with each print job, even if those fonts reside on the printer already.

    Note: When you use this setting, jobs usually print slower. However, the fonts on the prints look the same as those printed on another printer with this setting.
