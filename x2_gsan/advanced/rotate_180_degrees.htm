Rotate Image 180 Degrees

On the Advanced tab (in the Image Options folder).

The options on this page let you automatically or manually set page image rotation.

The Rotate Image 180 Degrees feature allows you to select the behavior when printing mixed-size jobs where both SEF and LEF media are used in the same job. This avoids pages being rotated 180 degrees out of the desired state and overcomes problems where folding and punching occurs on the wrong edge.

Note: Some features are only supported by specific printers, operating systems, or driver types (PostScript or PCL). You may see more features or options listed than what your specific system supports.

  1. Click Setup.
  2. To automatically specify rotation, select Intelligent 180 Degree Rotation (Recommended). If selected, the driver rotates the page image 180 degrees, automatically fixing incorrect image rotation.

    Note: If selecting this option does not give you the desired results, you can configure rotations manually.

  3. To manually specify rotation:
    1. Select Subsequent 180 Degree Rotation. If selected, the driver adds another 180 degree rotation to whatever the Intelligent 180 Degree Rotation option may have added. Note that two 180 degree rotations cancel each other out.
    2. To use the menus to set rotation, clear Intelligent 180 Degree Rotation, if selected.

      The two menus below Subsequent 180 Degree Rotation become active.

    3. From the first menu, select a media size. This selection controls which media size is rotated 180 degrees.
    4. From the second menu, select an orientation. This selection controls which pages of the selected orientation are rotated 180 degrees.
  4. Click OK to keep your changes and close the window.
