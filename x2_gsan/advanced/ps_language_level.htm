PostScript Language Level

On the Advanced tab.

Select the PostScript language level to use for printing. It is usually best to select the highest level available in order to enable more features. Your Xerox® printer is PostScript Level 3.

You may want to select a lower level when troubleshoot printing problems or archiving documents so they can be printed on a printer that has that lower language level. Printing errors may occur if you select PostScript Language Level 2 when the document is longer than one page.
