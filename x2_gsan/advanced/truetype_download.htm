TrueType Font Download Option

On the Advanced tab.

Select one of the following settings to specify how TrueType fonts are downloaded to the printer:

  • Download as TrueType: Downloads the TrueType font as a scalable, vector-based font. The fonts download to the printer exactly how they display on the screen. If your printer supports TrueType rendering, this is the best way to print TrueType fonts. If you are unable to print using the Bitmap setting due to a printer memory shortage, use this setting instead.
  • Download as Bitmap: Downloads the TrueType font as a bitmap font. A bitmap font is a non-scalable, rasterized font made up of numerous pixels combined to form each character. This setting is best when the font used in the job is one style and size, the document uses smaller point sizes (14 points or less), or if the font is used in combination with graphics, such as a logo. Do not use this setting when you are printing to a file that will be printed on different printers; bitmaps are not portable to printers with different resolutions.
  • Print as Graphics (None): Downloads the TrueType font as a non-scalable graphic.
  • Automatic: The driver automatically selects the best format for the fonts in the job.
  • Outline: Downloads the TrueType font as an outline font. An outline font is a scalable, vector-based font that defines each character as a set of distinct objects (such as lines, circles, ellipses, and boxes) that specify the shape and size of the character. This setting uses less memory and reduces printing time for documents that contain large TrueType fonts (10 points or more). Use this setting to create a more portable PostScript file for printers that cannot process TrueType fonts.
  • Bitmap: Downloads the TrueType font as a bitmap font. A bitmap font is a non-scalable, rasterized font made up of numerous pixels combined to form each character. This setting is best when the font used in the job is one style and size, the document uses smaller point sizes (14 points or less), or if the font is used in combination with graphics, such as a logo. Do not use this setting when you are printing to a file that will be printed on different printers; bitmaps are not portable to printers with different resolutions.
  • Native TrueType: Downloads the TrueType font as a scalable, vector-based font. The fonts download to the printer exactly how they display on the screen. If your printer supports TrueType rendering, this is the best way to print TrueType fonts. If you are unable to print using the Outline or Bitmap settings due to a printer memory shortage, use this setting instead.

Note: Not all options are available in all print drivers.
