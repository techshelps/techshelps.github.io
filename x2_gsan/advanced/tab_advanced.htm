Advanced Tab Overview

The Advanced tab enables you to select advanced printing options:

  • Advanced Printing Features
  • Application Reduce/Enlarge (Expert)
  • Binding Edge
  • Booklet Image Order
  • Collate
  • Copies
  • Draft Mode
  • Earth Smart Settings
  • Job Identification
  • Mirrored Output
  • Negative Output
  • PostScript Language Level
  • PostScript Output Option
  • PostScript Pass-through
  • Print Optimization
  • Rotate Image 180 Degrees
  • Request Offset
  • Restore Original Driver Defaults
  • Send PostScript Error Handler
  • TrueType Font
  • TrueType Font Download Option
  • Web Links: Click to reach helpful Xerox Web sites, for using your printer.

The following items are located on the bottom of the tab:

  • About button
  • Status area and More Status button
  • Earth Smart Settings button
  • Saved Settings
