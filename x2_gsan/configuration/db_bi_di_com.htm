Bi-Directional Communication Dialog Box

Appears when you click the Bi-Directional Setup button on the Configuration tab.

Use this dialog box to enable bi-directional communication. When enabled, the driver receives status information from the printer and displays information about jobs, trays, supplies, installed options, and problems.

  • Communication: Select Automatic (Recommended) to enable bi-directional communication, Manual to enable bi-directional communication and change the settings, or Off to disable communication from the printer.
  • Device Name or TCP/IP Address: Displays the name of the printer or its TCP/IP address when your computer connects to a network printer. Enables after you select Manual.
  • SNMP Community Name button: Displays the SNMP Community Name dialog box, where you type new SNMP community names.
  • Refresh Rate: Select the frequency that the status information is polled and updated (in minutes).
  • Job Notification: Select On to receive a notification when your print job finishes, or select Off to disable the notification.

Note: These selections are only available for administrators.
