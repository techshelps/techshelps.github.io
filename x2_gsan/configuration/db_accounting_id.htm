Accounting Dialog Box (When Submit Job)

Note: Some features are only supported by specific printers, operating systems, or driver types (PostScript or PCL). You may see more features or options listed than what your specific system supports.

Displays when you click the Print button in an application and Always Prompt or Only Prompt was selected for Accounting on the Configuration tab.

Use this dialog box to enter your user and account identification information for your print job.

  • User ID: Enter the account or departmental billing code and Passcode, if required, assigned to you to authorize printing.
  • Passcode: Enter the passcode assigned to you.
  • Account ID: Enter the account or departmental billing code assigned to you to authorize printing.
