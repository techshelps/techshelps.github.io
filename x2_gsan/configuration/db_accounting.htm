Accounting Dialog Box

Note: Some features are only supported by specific printers, operating systems, or driver types (PostScript or PCL). You may see more features or options listed than what your specific system supports.

Appears when you click the Accounting button on the Configuration tab.

Use this dialog box to associate user and account identification information with each document you print.

Note: Accounting support and features may vary with the configuration of your printer. See the user documentation for more information about its accounting features.

Accounting Methods: Select the accounting system used by your organization:

  • None
  • Local Accounting
  • Network Accounting
  • Xerox® Standard Accounting
  • Charge Print

Then select one of the following options:

Accounting Prompt: Choose when the Accounting dialog box appears so that you can provide the User ID/passcode, Billing ID/passcode, or Account ID for that job. Your choices are:

  • Always Prompt: Select this option if you want the Accounting dialog box to appear every time you submit a print job so that you can provide the appropriate ID or passcode for that job.

    • When you select Always Prompt, you can select one or more of the following features:

    • Mask User ID (****): Prevents the User ID or passcode from being viewed by displaying asterisks (*) instead of the actual characters.
    • Mask Account ID (****): Prevents the Account ID or passcode from being viewed by displaying asterisks (*) instead of the actual characters.
    • Remember Last Entered Codes: The ID or passcode is saved when you finish printing. The same ID or passcode is used each time you print until you enter different numbers.
  • Never Prompt: Selecting this option requires you to set default accounting codes.
  • Only Prompt For Fax: Only show the accounting prompt if a job is being sent as a fax.

Default Accounting Codes: Select this option if you want to enter default values that will automatically be used each time you submit a print job. Enter the following default values:

  • Default User ID: Enter the User ID/passcode if required, to be used for your print jobs.
  • Default Account ID: Enter the Account ID or departmental billing code to be used for your print jobs.
  • Default Billing ID: Enter the Billing ID, and Passcode if required, to be used for your print jobs.
