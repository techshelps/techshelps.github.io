Installable Options Dialog Box

Appears when you click the Installable Options button on the Configuration tab.

Use this dialog box to select the options that are installed on your printer:

  • Paper Trays: Select the total number of trays.
  • High Capacity Feeder: Select whether the high capacity feeder is Installed or Not Installed.
  • Finisher/Output Tray: Select the option that is installed on your printer.
  • Hole Punch Unit: Select which hole punch unit is installed on your printer.
  • LAN Fax: Select whether the LAN fax is Installed or Not Installed.

Note: If your printer is on a network and bi-directional communication is enabled, the correct options should already be selected.
