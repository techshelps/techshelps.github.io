SNMP Community Name Dialog Box

Appears when you click the SNMP Community Name button on the Bi-Directional Communication dialog box.

SNMP is the protocol that regulates the exchange of configuration and status information between the driver and the printer. It provides the following functions:

  • Read: Read or obtain configuration data.
  • Write: Send or write configuration data.

To provide a measure of security, each function has an associated community name. To use the bi-directional capabilities of the printer, the community name entries in the driver must match their equivalents on the printer. For example, if the printer's designated community name for the Read function is public, all clients using bi-directional drivers must assign the value "public" to the SNMP Read function to obtain regular printer status updates.

Contact your administrator for the correct Read and Write community names to type in this dialog box. SNMP community names are case-sensitive. If you leave the entries blank, the driver uses the default names.

Note: If you type community names incorrectly, it disables bi-directional communication for the driver.
