Edit Inserts Dialog Box

Appears when you select an insert in the table on the Special Pages tab, and then click the Edit button below the table.

Use this dialog box to edit the characteristics of pages inserted into the document.

  • Insert Options: Select the type of inserts:
    • After Page(s): Inserts the paper after the specified pages.
    • Before Page 1: Inserts the paper before page 1.
  • Insert Quantity: Specify the number of pages to be inserted at each location.
    Note: The insert quantity is the same for each page/page range that you specify for Page(s). To vary the insert page quantities, add each insert separately.
  • Page(s): Specify the location (page or range of pages) to place the inserts. Separate individual pages or pages ranges with commas. Specify ranges with a dash. For example, to add inserts after pages 1, 6, 9, 10, and 11, type: 1, 6, 9-11.
  • Paper: The paper summary displays the size, color, and type of paper to use for the inserts. To change the paper to use for the inserts, click the button to the right of the summary to open the paper selection drop-down menu, which contains the following menu items:
    • Other Size: Select this menu item, and then select Use Job Setting or the paper size desired for the inserts.
    • Other Color: Select this menu item, and then select Use Job Setting or a color for the paper to use for the inserts.
    • Other Type: Select this menu item, and then select Use Job Setting or the type of paper or media to use for the inserts.
  • Job Settings: Displays the attributes of the paper for the majority of the document. If you select Use Job Setting for any paper attribute (size, color, or type), the setting displayed here is used for that attribute of the insert.
