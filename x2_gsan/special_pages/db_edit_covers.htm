Edit Covers Dialog Box

Appears when you select a cover in the table on the Special Pages tab, then click the Edit button below the table.

Use this dialog box to edit the size, color, and type of paper to use for the cover.

  • Cover Options: Select the type of covers:
    • No Covers: Does not attach covers to your job. Select No Covers to cancel any previous settings.
    • Front Only: Attaches a cover to the front of each copy of the document.
    • Back Only: Attaches a cover to the back of each copy of the document.
    • Front and Back Same: Attaches covers to the front and back of each copy of the document. The same paper settings are used for both the front and back covers.
    • Front and Back Different: Attaches covers to the front and back of each copy of the document. You specify the paper to use for the front covers separately from the paper to use for the back covers.

      Note: If you select a cover for booklet jobs, a single sheet of paper is used; it becomes the front and back covers when the job is folded.
  • Front Cover Paper and Back Cover Paper: Is available when a Front or Back Cover Paper option is selected. The paper summary displays the size, color, and type of paper to use for the front and back covers. To change the paper to use for the covers, click the button to the right of the summary to open the paper selection drop-down menu, which contains the following menu items:
    • Other Size: Select this menu item, and then select Use Job Setting or the paper size desired for the covers.
    • Other Color: Select this menu item, and then select Use Job Setting or a color for the paper to use for the covers.
    • Other Type: Select this menu item, and then select Use Job Setting or the type of paper or media to use for the covers.
  • <printing options>: Is available when a Front or Back Cover Paper option is selected. Select one of the following options for the front and/or back covers:
    • Blank or Preprinted: Uses blank or pre-printed document covers. Collates and processes covers with the print job but no images are printed on them.
    • Print on Both Sides: Prints the first two pages of your document on the front and back of the front cover.

      • Note: Insert a blank second page into the document to keep an image from printing on the back of the front cover.

      Prints the last two pages of your document on the front and back of the back cover.

  • Job Settings: Displays the attributes of the paper for the main body of the document. If you select Use Job Setting for any paper attribute (size, color, or type), the setting displayed here is used for that attribute of the cover.
