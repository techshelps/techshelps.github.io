Special Pages Tab Overview

The Special Pages tab enables you to add, modify, or delete cover pages, inserts, or exception pages.

Note: Some features are only supported by specific printers, operating systems, or driver types. You may see more features or options listed than what your printer supports.

The table on the tab provides a summary of the special pages for your print job. Up to 250 inserts and up to 250 exception pages can be included in the table. You can change the width of the columns in the table.

  • To add a cover, insert, or exception, click the corresponding Add button above the table.
  • To edit a cover, insert, or exception, select it in the table and click the Edit button below the table.

    Note: You can select multiple rows of the same type and click the Edit button to change the properties for that special page type (except for the page number).
  • To delete a cover, insert, or exception, select it in the table and click the Delete button below the table. To delete all special pages shown in the table, click the Delete All button.
  • To change the order of an insert within a group of inserts, select it in the table and click the Move Up or Move Down button below the table.

    Note: To move an insert, it must have the same number of pages as the other inserts.

The following items are located on the bottom of the tab:

  • Saved Settings
  • Earth Smart Settings button
  • Status area and More Status button
