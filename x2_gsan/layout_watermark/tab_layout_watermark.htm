Layout/Watermark Tab Overview

The Layout/Watermark tab contains the following items for selecting layout options and watermarks:

  • Page Layout selections, including the Booklet Options button.
  • Page Layout Options, including orientation, 2-sided printing, and page borders.
  • Watermark selections, including the New and Edit selections that access the Watermark Editor dialog box.

The following items are located on the bottom of the tab:

  • Status area and More Status button
  • Earth Smart Settings button
  • Saved Settings
