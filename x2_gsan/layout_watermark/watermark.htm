Watermark

On the Layout/Watermark tab.

A watermark is a picture or text (independent of that provided by the application) that prints on every page or the first page of the job.

From the Watermark menu, select:

  • None if you do not want a watermark.
  • An existing watermark name to use that watermark.
  • Options to print the watermark in the background, blend with the job, or print in the foreground. Also select whether to print the watermark on all pages or only on the first page.
  • New or Edit to display the Watermark Editor dialog box, where you create or edit watermarks.
  • Delete to delete the selected watermark.

Notes:

  • Not all applications support watermark printing.
  • The Watermark menu deactivates after you select Booklet Layout.
