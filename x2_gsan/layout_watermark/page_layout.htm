Page Layout

On the Layout/Watermark tab.

Select one of the following options:

  • Pages Per Sheet (N-Up): Select the button to print one, two, four, six, nine, or 16 pages on each side of a sheet of paper. Selecting multiple pages per sheet saves paper and is useful when you want to review the layout of the document.

    • Note: Pages Per Sheet (N-Up) is also part of the Earth Smart settings. It automatically chooses the best way to print a job without wasting paper.

  • Booklet Layout: Select to print two pages on each side of each sheet of paper. The print driver changes the order of the pages so that they can be folded and stapled to create a booklet. To select additional booklet settings, click the Booklet Options button to display the Booklet Options dialog box.



    Note: If your document is already formatted as a booklet, do not select Booklet Layout.

The sample image on the tab changes to reflect your choices.
