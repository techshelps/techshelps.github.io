Watermark Editor Dialog Box

Appears when you select New or Edit in the Watermark menu on the Layout/Watermark tab.

Use this dialog box to create a new watermark or edit an existing watermark.

  • Name: When creating or editing a watermark, type the name (no more than 255 characters).
  • Options: Select one of the following options:
    • Text: Uses the characters that you type in the Text field as the watermark.
    • Time Stamp: Uses a time stamp as the watermark. Select whether to include the weekday, date, time, and/or time zone.
    • Picture: Uses a picture as the watermark. The picture can be in .BMP, .GIF, or .JPG format. Specify the file name in the File Name field or click the Choose File button to browse to the picture that you want to use for the watermark.
  • Font button: If you select Text or Time Stamp, click this button to access the Font dialog box, where you can change the font family, style, and size to use for the text.
  • Color button: If you select Text or Time Stamp, click this button to access the Color dialog box, where you can change the color to use for the text.
  • Angle: If you select Text or Time Stamp, either specify an angle or move the slider to select the angle of the text on the page; the default angle is 45 degrees.
  • Scale: If you select Picture, either specify the percentage or move the slider to select the percentage to scale the picture.
  • Density: Either specify a density for the watermark or move the slider to select the density; the default density is 25 percent.
  • Position (from center): Select the number of inches or millimeters to shift the watermark from the center of the page or click the Center button to center the watermark.

    • You can also click the preview image and drag the image to its new location.

  • <preview image>: Represents a scaled version of the paper displaying the watermark with the settings you selected.
  • Preview Options: Select the paper size and orientation to use for the preview image of the watermark. Select Color or Black and White to display the watermark in the selected font color or gray respectively.

Note: Not all applications support watermark printing.
