Using Help

This help system guides you through information about using this print driver.

In the help system's Contents tab:

  • Double-click a book to open it.
  • Click the topic you want to view.

Click the Back button (on the toolbar) to move back through topics you have already viewed.

In the help system's Index tab, do one of the following:

  1. Type part of an index entry in the field at the top; the list scrolls to that part of the index.
  2. Select the desired index entry in the list, and then click the Display button.
  3. If there is more than one topic for an index entry, select the desired topic title, and then click the Display button.

In the help system's Search tab:

  1. Type the search word(s) in the field at the top of the screen. For example:
    • Type paper or source to include any topic containing the word "paper" or the word "source".
    • Type either paper source or paper and source to include any topic containing both words, in any order.
    • Type "paper source" to include any topic containing the phrase "paper source" in that order.
  2. Click the List Topics button. The help system lists topics that contain those word(s).
  3. Select the desired topic in the list.
  4. Click the Display button.
