Phonebook Preferences Dialog Box

Appears when you click the Preferences button on the Recipients tab, Cover Sheet tab, and Options tab.

Use this dialog box to select the phonebook and set preferences.

  • Default Phonebook: Select the type of phonebook for the fax to automatically access:
    • Personal Phonebook
    • Shared Phonebook

    If a phonebook does not exist, the printer creates one.

  • Personal Phonebook: Shows the location of your personal phonebook (*.xpb, *.pb), which you maintain on your own computer and can edit. It is not shared with others. You can create a personal phonebook using this driver or you can import a comma-separate value (.CSV) file (Windows only). You can store up to 1,000 entries in a personal phonebook.
    • : Click to open the personal phonebook file.
    • : Click to browse to a personal phonebook file on your computer.
  • Shared Phonebook: Shows the location of a shared phonebook, usually on a network server. You can copy information from the shared phonebook to a personal phonebook, but shared phonebooks are generally set to read-only mode.
    • : Click to open the shared phonebook file.
    • : Click to browse to a shared phonebook file on the network.
  • Shared LDAP Phonebook: If enabled, shows the location of a shared LDAP (Lightweight Directory Access Protocol) phonebook. LDAP is a protocol that is often used for corporate address books (such as Microsoft Outlook, Lotus Notes, GroupWise, and/or Netscape), which is located on a network. You can copy information from the LDAP phonebook to a personal phonebook. To access an LDAP phonebook, bi-directional communication must be configured in the print driver.
    • : Click to view details about your LDAP settings and mappings.
  • Prompt When Adding Duplicate Recipients: When you select this check box, whenever you attempt to add a recipient to the Fax Recipients table who is already in the table, you are warned that the recipient already exists in the table.
  • Prompt When Removing a Recipient: When you select this check box, whenever you attempt to remove a recipient from the Fax Recipients table, you are asked whether you really want to remove the recipient.
  • Always Use Current Recipient List: Select this check box if you always send faxes to the same recipients; this option eliminates the need to create a recipient list each time you send a fax.
  • Always Use Current Cover Sheet Notes: Select this check box if you always send faxes with the same message on the fax cover sheet; this option eliminates the need to create a cover sheet message each time you send a fax.
