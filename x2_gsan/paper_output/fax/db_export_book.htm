Export Phonebook Dialog Box

Appears when you select Export from the File menu in the Fax Phonebook dialog box. Use this dialog box to save data from the current phonebook to another file.

You can export to comma-separated value (.CSV) files (Windows only) or text files (.TXT).

  • File Name: Type the file name (and path) of the new file or click the Browse button to browse to the directory where you want to save the file.
  • Record Delimiter: Select the delimiter (a character or format) used to separate records from one another in the phonebook file. If you select User Defined, Max of 4 characters, type the four characters in the field to the right.
  • Field Delimiter: Select the delimiter (a character or format) used to separate fields from one another in the phonebook file. If you select User Defined, type the character in the field.
  • Fields: Select the check boxes next to the fields you want to export. Use the up and down arrows to change the order of the fields in the list.
  • Use Unicode Encoding (Recommended): Select the check box to use Unicode (UTF-8) encoding. Clearing the box may cause information loss on files that require this encoding.

    Note: The mailbox feature is not compatible with DTMF (Dual Tone Multi-Frequency).
