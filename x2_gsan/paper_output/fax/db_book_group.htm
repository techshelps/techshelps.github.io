Edit Group Dialog Box

Appears when you select an existing group name in the CentreWare Fax Phonebook dialog box, then click or select Edit from the Edit menu.

Use this dialog box to view or edit the group name or members of the group.

  • Group Name: Displays the name of the group.
  • Members: Displays the individuals assigned to this group.
  • : Accesses the Edit Recipient dialog box, which displays information about the selected member.
  • : Removes the selected member from the group.
  • : Accesses the Add to Group dialog box, where you can add members to the group.
