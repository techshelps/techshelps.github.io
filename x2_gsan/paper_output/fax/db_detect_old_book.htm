Older Phonebook File Message

Appears when you first open the Fax portion of the print driver and the driver finds an older fax phonebook file on your computer.

Choose one of the following options, and then click the OK button:

  • Use the older phonebook and copy it to the default location.
  • Use the older phonebook and keep it in the same location.
  • Create a new phonebook in the default location.
  • Convert phonebook to new file format (Recommended): Select this option to convert older phonebook to the phonebook format. The new phonebook file format supports multi-byte characters (UTF-8) used in some languages. It is recommended that the old phonebook be converted to this new format.
