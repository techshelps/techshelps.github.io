Cover Sheet Dialog Box

Appears when you choose to have a cover sheet and select Print a Cover Sheet under Cover Sheet Options on the Cover Sheet tab.

Use this dialog box to design the cover sheet for this fax.

  • Recipient Information: Select one of the following:
    • Show Recipient Names: Displays the names of the recipients after "To" near the top of the fax cover sheet.
    • Hide Recipient Names: Conceals the names of the recipients on the cover sheet.
    • Show Custom Text: Displays the custom text after "To" near the top of the fax cover sheet. You then type the custom text in the field.
  • Sender's Information: Select one of the following:
    • Show Sender's Information: Displays information about you after "From" on the fax cover sheet. You then type the information that you want to display for Name, Fax Number, Organization, Telephone Number, and Email Address.
    • Hide Sender's Information: Conceals "From" or any information about you on the fax cover sheet.
  • Cover Sheet Image: You can include an image, such as a company logo or background text, on your fax cover sheet. Select one of the following:
    • None if you do not want to use an image on the cover sheet.
    • An existing image name to use that image on the cover sheet.
    • Options to print an image in the background, in the foreground, or blend with the job.
    • New to display the Cover Sheet Image Editor dialog box, where you can create a new cover sheet image.
    • Edit to display the Cover Sheet Image Editor dialog box, where you can edit the selected cover sheet image.
    • Delete to delete the selected cover sheet image.
  • Cover Sheet Paper Size: Select the paper size to use for the fax cover sheet.

The sample cover page image changes to reflect your cover page choices.

Note: To include notes on the cover sheet, type the text in the Cover Sheet Notes field on the Cover Sheet tab.
