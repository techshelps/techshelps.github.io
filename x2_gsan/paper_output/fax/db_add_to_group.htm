Add to Group Dialog Box

Appears when you click in the New Group dialog box or the Edit Group dialog box.

Use this dialog box to add individuals to the group.

  • <table of phonebook entries>: Use the scroll bar to locate the desired name in the current phonebook. To quickly locate a name in the phonebook, type the first few characters in the Type Name or Select from List field. As you type each character, only the matching names are shown in the table. For example, if you type "S", only those names that begin with the letter "S" display in the table. If you then type "T", only those names that begin with "ST" display. If you separate the letters with a space, such as "S T", only people whose first names that begin with an "S" and whose last names begin with a "T" display.
  • : Select an entry in the phonebook, then click this button to add the entry to the Members table.
  • (below the phonebook): Click this button to access the Edit Recipient dialog box, where you can view information about the selected phonebook entry.

    Note: You can select multiple entries by holding down the Ctrl key while you select each entry. You can select a range of entries by holding down the Shift key while clicking the first and last entries.
  • Members: Displays the individuals in the group.
  • : Click this button to remove the selected individual from the Members table.
  • (below the Members table): Click this button to access the Edit Recipient dialog box, where you can view information about the selected member.
