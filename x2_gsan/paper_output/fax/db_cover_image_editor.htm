Cover Sheet Image Editor Dialog Box

Appears when you select New or Edit in the Cover Sheet Image menu in the Cover Sheet dialog box.

Use this dialog box to create a new image or edit an existing image for your fax cover sheet.

  • Name: When creating a new cover sheet image, type the name (no more than 100 characters).
  • Options: Select one of the following options:
    • Text: Uses the characters that you type in the Cover Sheet Image Text field as the cover sheet image.
    • Picture: Uses a picture as the cover sheet image. The picture can be in .BMP, .GIF, or .JPG format. Specify the file name in the File Name field or click the Choose File button to browse to the picture that you want to use for the image.
  • Font button: If you selected Text, click this button to access the Font dialog box, where you can change the font family, style, and size to use for the text.
  • Color button: If you selected Text, click this button to access the Color dialog box, where you can change the color to use for the text.
  • Angle: If you selected Text, either specify an angle or move the slider to select the angle of the text on the page; the default angle is 45 degrees.
  • Scale: If you selected Picture, either specify the percentage or move the slider to select the percentage to scale the picture.
  • Density: Either specify a density for the image or move the slider to select the density; the default density is 25 percent.
  • Position (from center): Select the number of inches or millimeters to shift the image from the center of the page or click the Center button to center the watermark.
  • <preview image>: Represents a scaled version of the paper showing the cover sheet image with the settings you selected.
  • Preview Paper Size: Select the paper size to use for the preview image of the cover sheet image.
