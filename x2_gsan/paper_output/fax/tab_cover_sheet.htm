Fax: Cover Sheet Tab

Use the Cover Sheet tab to select and design a cover sheet to be sent with your fax.

  • Cover Sheet Options: Select whether or not you want a cover sheet sent with your fax. If you select Print a Cover Sheet, the Cover Sheet dialog box appears, which is where you design the cover sheet. You can include the list of recipients, include information about yourself, add an image, and change the cover page size. The sample cover page image changes to reflect your cover page choices.
  • Cover Sheet Notes: Type the text that you want on the cover page, such as special notes, instructions, or general information.
