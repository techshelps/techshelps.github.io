Import Phonebook Dialog Box

Appears when you select Import from the File menu in the Fax Phonebook dialog box. Use this dialog box to import phonebook entries from another file into the fax phonebook.

You can import comma-separated value (.CSV) files (Windows only) or text files (.TXT).

  • File Name: Type the file name (and path) of the source file you want to import or click the Browse button to navigate to the source file.
  • Record Delimiter: Select the delimiter (a character or format) used to separate records from one another in the phonebook file. If you select User Defined, Max of 4 characters, type the four characters in the field to the right.
  • Field Delimiter: Select the delimiter (a character or format) used to separate fields from one another in the phonebook file. If you select User Defined, type the character in the field to the right.

    Note: When you import data into a phonebook, the record and field delimiters you select must match the delimiters in the source file or the data may not import properly.
  • Fields: Select the check boxes next to the fields you want to import. Use the up and down arrows to change the order of the fields in the list.

    Note: When you import data, the fields you select and their order must match the fields in the source file or the data may not import properly.
  • Use Unicode Encoding (Recommended): Select the check box to use Unicode (UTF-8) encoding. Clearing the check box may cause information loss on files that require this encoding.
