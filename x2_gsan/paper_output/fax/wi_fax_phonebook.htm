CentreWare Fax Phonebook Dialog Box

Appears when you click the Open button for a fax phonebook in the Phonebook Preferences dialog box.

The table displays the entries in the phonebook. You can use the scroll bar to locate the desired name. To quickly locate a name in the phonebook, type the first few characters in the top field. As you type each character, only the matching names are shown in the table. For example, if you type "S", only those names that begin with the letter "S" display in the table. If you then type "T", only those names that begin with "ST" display. If you separate the letters with a space, such as "S T", only people whose first names that begin with an "S" and whose last names begin with a "T" display.

Use the following to make changes or view information about the fax phonebook.

File menu:

  • New Phonebook: If available, enables you to create a new fax phonebook with a name that you specify.
  • Open: If available, enables you to select another fax phonebook file to view.
  • Save As: If available, enables you to save the fax phonebook entries with a name that you specify.
  • Import: Opens the Import Phonebook dialog box, so you can import phonebook entries from another file into your personal fax phonebook.
  • Export: Opens the Export Phonebook dialog box, so you can save the data from the current phonebook to another file.
  • Exit: Closes this dialog box and returns you to the Phonebook Preferences dialog box.

Note: The New Phonebook, Open, and Save As menu items are not available when you access the phonebook from the print driver.

Edit menu:

  • Edit: If an individual is selected in the table, opens the Edit Recipient dialog box, so you can view and edit the information. If a group is selected in the table, opens the Edit Group dialog box, so you can view and change the group information.

    Note: You can instead click the Edit button above the table.
  • Delete: Deletes the selected entry from your personal phonebook.

    Note: You can instead click the Delete button above the table.
  • New Recipient: Opens the New Recipient dialog box, so you can add a new entry to your personal phonebook.

    Note: You can instead click the New Recipient button above the table.
  • New Group: Opens the New Group dialog box, so you can add a new group to your personal phonebook.

    Note: You can instead click the New Group button above the table.

Help menu:

  • Help: Opens this help file.
  • About: Opens the About information dialog box, which displays the current driver version number and copyright information.
