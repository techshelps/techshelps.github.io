Convert Phonebook File Dialog Box

Appears when you select Convert phonebook to new file format (Recommended) from the Older Phonebook File dialog box.

Read the message and then click the OK button to convert the phonebook file.

The message is a confirmation to proceed with the conversion of an old phonebook file to the new (UTF-8) format.

If you click No the file will not be converted, maintaining phonebook compatibility with older printer drivers.
