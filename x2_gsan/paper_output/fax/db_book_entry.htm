Edit Recipient Dialog Box

Appears when you:

  • Select an entry in the phonebook in the Fax Phonebook dialog box, then click or select Edit from the Edit menu.
  • Select an entry in the phonebook on the Add from Phonebook dialog box, then click .

Note: Some features are only supported by specific printers, operating systems, or driver types. You may see more features or options listed than what your printer supports.

Use this dialog box to view and, if allowed, change information about the phonebook entry.

  • Name: Displays the name of the individual.
  • Fax Number: Displays the fax number of the individual. It can include numbers, spaces, and some special characters.
  • Organization: Displays the name of the company or organization associated with the individual.
  • Phone Number: Displays the phone number of the individual.
  • Email Address: Displays the email address of the individual.
  • Mailbox: Displays the number of the individual's fax mailbox so the fax is sent directly to that mailbox. Use it to store and retrieve documents.

    Note: The mailbox feature is not compatible with DTMF (Dual Tone Multi-Frequency).
  • Previous and Next arrow buttons: Displays the previous or next entry in the phonebook.
