Add Fax Recipient Dialog Box

Appears when you click on the Recipients tab. Use this dialog box to type the name and fax number of an individual recipient for your fax.

Note: Some features are only supported by specific printers, operating systems, or driver types. You may see more features or options listed than what your printer supports.

The recipient entry must include the name and fax number. You can also include additional information about the individual and save the information in your personal phonebook to serve as a reference for future communications.

  • Name: Type the name of the individual to whom you are sending the fax.
  • Fax Number: Type the fax number of the individual. You may type numbers, spaces, and some special characters.

    Note: If you need to dial a prefix to obtain a dial tone, type the Dialing Prefix on the Options tab. You cannot customize dialing prefixes for individual fax recipients.
  • Organization: If desired, type the name of the company or organization associated with the individual.
  • Telephone Number: If desired, type the phone number of the individual.
  • Email Address: If desired, type the email address of the individual.
  • Mailbox: Type the number of the individual's fax mailbox if you want to send the fax directly to that mailbox. Use it to store and retrieve documents.

    Note: The mailbox feature is not compatible with DTMF (Dual Tone Multi-Frequency).
  • Save to Personal Phonebook: Select this check box if you want to add this individual to your personal phonebook.
