Add from Phonebook Dialog Box

Appears when you click on the Recipients tab.

Use this dialog box to view names and fax numbers in your phonebook and select someone to receive your fax.

  • <current phonebook>: Select the name of the fax phonebook to use.
  • <table of phonebook entries>: Use the scroll bar to locate the desired name in the current phonebook. To quickly locate a name in the phonebook, type the first few characters in the Type Name or Select from List field. As you type each character, only the matching names are shown in the table. For example, if you type "S", only those names that begin with the letter "S" display in the table. If you then type "T", only those names that begin with "ST" display. If you separate the letters with a space, such as "S T", only people whose first names begin with an "S" and whose last names begin with a "T" display.
  • : Select an entry in the phonebook, then click this button to add the entry to the Fax Recipients table.

    Note: You can select multiple entries by holding down the Ctrl key while you select each entry. You can select a range of entries by holding down the Shift key while clicking the first and last entries.
  • : Click this button to access the Phonebook Recipient dialog box, where you can view the information about the selected phonebook entry.
  • Fax Recipients: This table displays the names of the recipients for your fax.
  • : Click this button to remove the selected recipient from the Fax Recipients table. Click Yes to confirm the deletion.
  • (below the Fax Recipients table): Click this button to access the Edit Fax Recipient dialog box, where you can view or edit information about the selected recipient.
