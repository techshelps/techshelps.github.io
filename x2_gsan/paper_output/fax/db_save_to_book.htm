Save to Personal Phonebook Dialog Box

Appears when you select a recipient in the Fax Recipients table on the Recipients tab, then click .

Note: Some features are only supported by specific printers, operating systems, or driver types. You may see more features or options listed than what your printer supports.

Use this dialog box to view or change the information about a recipient, then add that recipient to your phonebook.

  • Name: Displays the name of the individual.
  • Fax Number: Displays the fax number of the individual. It can include numbers, spaces, and some special characters.
  • Organization: Displays the name of the company or organization associated with the individual.
  • Telephone Number: Displays the phone number of the individual.
  • Email Address: Displays the email address of the individual.
  • Mailbox: Displays the number of the individual's fax mailbox so the fax is sent directly to that mailbox. A fax mailbox is an electronic memory storage location where you can store and retrieve documents.
