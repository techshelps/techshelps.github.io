Using Special Characters

You can use the following special characters in the fax numbers.

Character

 

Use

0–9

  

Phone numbers, prefix numbers, and other numeric entries.

( )

  

Aid readability; the driver ignores them.

spaces

  

Aid readability; the driver ignores them.

+

  

Dial a long-distance number.

=

  

Wait for dial tone.

, (comma)

  

Pause; the default pause period is 3 seconds.

!

  

Designates a mailbox signal.

/

  

Delineates private data.

: (colon)

  

Switch to DTMF mode.

*

  

Begin DTMF.

#

  

End DTMF.

A–D

  

DTMF digit to dial.

P

  

Insert a one-second pause.

W

  

Insert a tone wait.

$

  

Wait for a confirmation dial tone.

S

  

Check a password.

<1>

  

(Before a fax number) Uses fax line 1.

<2>

  

(Before a fax number) Uses fax line 2.
