New Group Dialog Box

Appears when you click or select New Group from the Edit menu in the CentreWare Fax Phonebook dialog box.

Use this dialog box to organize your phonebook entries by creating a new group in your phonebook. You may want to create groups of individuals that share a common attribute or association with you. When you want to fax them a document, you can select the group name from your phonebook instead of having to select each individual recipient.

  • Group Name: Type the name for the group.
  • Members: Displays the individuals assigned to this group. Click the Add button to select members.
  • : Accesses the Edit Recipient dialog box, which displays information about the selected member.
  • : Removes the selected member from the group.
  • : Accesses the Add to Group dialog box, where you can add members to the group.
