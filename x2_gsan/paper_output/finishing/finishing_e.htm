Finishing

On the Paper/Output tab.

The following options appear if your printer has the appropriate finisher installed.

<finishing summary>: Displays the finishing options for the print job, such as stapling or hole punching.

To change the finishing options, click the button to the right of the summary to display the finishing selection menu, which contains:

  • <stapling options>: Select No Staples or select the number or location of staples to insert in each copy. The stapling options vary depending on the type of finisher.
  • <hole punching options>: Select No Hole Punching or select the number of holes to punch in the paper.

    Note: The number of holes punched and their location depend on the hole punch unit in the finisher.
  • Booklet Creation: Select to display the Booklet Creation dialog box, where you can select booklet layout and select booklet finishing options.

Note: Some media types, sizes, and weights cannot be stapled, punched, or folded. See the User Guide for more information.
