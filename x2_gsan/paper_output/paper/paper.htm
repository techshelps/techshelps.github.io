Paper

On the Paper/Output tab.

<paper summary>: Displays information about the paper to use for the print job, such as size and type.

To change the paper to use for printing, click the button to the right of the summary to display the paper selection menu, which contains:

  • <paper in the trays>: Displays the paper size, color, or type in the printer trays. This list appears when you enable and connect bi-directional communication.
  • Other Size: Select to display the Paper Size dialog box, where you can select the paper size and scaling amount.
  • Other Color: Select Automatically Select or select a paper color to use for the print job.
  • Other Type: Select Automatically Select or select a particular type of paper or media to use for the print job.
  • Select by Tray: Select Automatically Select (Recommended) to have the printer automatically use a tray loaded with the size and type of paper that you select for the job. Or select a specific tray (to have the printer only use the paper in the selected tray, even when more than one tray contains the desired paper size and type).
  • Transparency Separators: Select to display the Transparency Separators dialog box, where you can add separators for transparencies and/or the tray to use for printing.
