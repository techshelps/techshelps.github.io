New Custom Size Dialog Box

Appears when you select New in a size menu on the Paper Size dialog box or Advanced Paper Selection dialog box (accessed from the Paper selection menu on the Paper/Output tab).

Use this dialog box to create a new custom size when you plan to print on paper that is not a standard size.

  • Name: Type a descriptive name for this custom size.
  • Height and Width: Specify the dimensions for this custom size.
  • <sample image>: Represents a scaled version of the custom size based on the dimensions.
  • Units: Select to display the size information in inches or millimeters.
  • <range the printer supports>: Displays the range of custom paper sizes that can be loaded in the printer's paper trays.

Note: The dimensions that you specify for height and width must fall within the range the printer supports. See the User Guide for more information.
