Paper Size Dialog Box

Appears when you select Other Size in the Paper selection menu on the Paper/Output tab.

Use this dialog box to select the paper size to use for printing and specify the scaling amount, if any.

  • Output Paper Size: Select the paper size. You can select a standard size or a custom size. To create a custom size, select New. You can also edit an existing custom size (select it and then select Edit) or delete an existing custom size (select it and then select Delete).
  • Scale Options: If the original size differs from the output size, select one of the following options:
    • Automatically Scale: Scales the document to fit the Output Paper Size. Displays the amount in the percentage field.
    • No Scaling: Does not scale the document. Displays 100% in the percentage field. If the original size is larger than the output size, the page image is cropped. If the original size is smaller than the output size, extra space appears around the page image.
    • Manually Scale: Scales the document by the amount that you type in the percentage field.
  • <anchor menu>: Select either Center, Top Left, Top, Top Right, Right, Bottom Right, Bottom, Bottom Left, or Left for the location of the anchor point. The anchor point controls placement of the image on the paper when the Original Document Size differs from the Output Paper Size.
  • Original Document Size: If you set the paper size in your application, this size is used for the document size. Otherwise, select a standard size or a custom size.
  • Original Document Orientation: Select either Portrait, Landscape, or Rotated Landscape for the page orientation.
  • Original Document Orientation: Select either Portrait or Landscape for the page orientation.
  • <sample image>: Represents a scaled version of the paper displaying the scaling amount that you selected.
