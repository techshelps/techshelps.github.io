Job Type

On the Paper/Output tab.

Select one of the following job types:

  • Normal Print: Prints the job immediately without requiring a passcode.
  • Secure Print: Prints the job only after you enter your passcode on the printer's control panel. When you select this job type, the Secure Print dialog box appears so you can save your passcode.
  • Sample Set: Prints one copy of the job as a sample and holds the remaining copies of the job at the printer until you release them from the printer's control panel.
  • Delay Print: Prints the job at the time you specify. When you select this job type, the Delay Print dialog box appears so you can specify the time to print.
  • Store File in Folder: Stores the job on the printer's hard drive so that it can then be printed on demand from the printer's control panel or printed from CentreWare Internet Services. When you select this job type, the Store File in Folder dialog box appears so you can assign a job name and, if desired, print it and save it.
  • Fax: Sends the current job as a fax. When you select this job type, the Fax dialog box appears so you can specify the fax recipients and specify other fax settings.

    Note: To fax your job, fax capacity must be installed in your printer and fax must be enabled in the Installable Options dialog box.
