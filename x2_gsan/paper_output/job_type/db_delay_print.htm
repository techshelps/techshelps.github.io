Delay Print Dialog Box

Appears when you select Delay Print for Job Type on the Paper/Output tab.

Use this dialog box to delay the printing of a job for up to 24 hours from the original job submission time. Type the time at which you would like the job to print. Use either a 12-hour format or a 24-hour format (dictated by the settings on your computer):

  • If you are using a 12-hour format, type the hour (1–12), minutes (00–59), and select AM or PM.
  • If you are using a 24-hour format, type the hour (1–23) and the minutes (00–59).

The Delay Print feature operates within a 24 hour time frame which means that your document will print in the next 24 hour cycle if you send the document to the printer AFTER the scheduled time. For example, if it is currently 9:00 a.m. and you set a Delay Print job for 9:01 a.m. but do not click OK in the driver until 9:02 a.m., the document will not print until 9:01 a.m. the next day. In situations where the document arrives at the printer after the scheduled time due to network or print queue delays (or different clock settings on the printer or computer), the document prints immediately. To avoid problems with Delay Print jobs, check and synchronize the time settings on your computer and the printer.

Note: Delay Print jobs can be manually released at the printer if you no longer want to wait until the specified printing time.
