Chapter Summary

There are six elements in the Office Assistant's balloon in all Office applications: a heading, icons, text, labels, check boxes, and button sets. Using these elements, you can create custom balloons that look and function like balloons displayed by Office applications. In addition, if the Office Assistant is visible, you can use the Assistant's balloon to display messages rather than using the Visual Basic for Applications MsgBox function. This allows you to mimic Office behavior. Because the Assistant and Balloon objects are defined in the Microsoft Office 9.0 Object Library, you can write code once and reuse the code for multiple Office applications.
