Preview of the Next Chapter

The next chapter describes how to create custom menus and toolbars. In addition, you'll learn how to create custom controls on menus and toolbars similar to built-in controls like the Zoom controls on the Standard toolbar in Word, Excel, and PowerPoint. Because the objects, methods, and properties required to create custom menus, toolbars, and controls are in the shared Microsoft Office 9.0 Object Library, the code in the next chapter can be used without modification in any Office application.
