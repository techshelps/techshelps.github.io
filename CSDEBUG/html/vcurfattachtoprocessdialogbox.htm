Attach To Process Dialog Box

The Attach to Process dialog box opens when you select a program from the Available Processes box in the Programs dialog box and choose Attach.

Process

The path name for the executable from which the process was created.

Choose Program Types

In this box, select or clear the check boxes next to the program types you want to be able to attach to for debugging.

All

Choose this button to enable attaching to all program types.

None

Choose this button to disable attaching to any program type.

Existing programs that will be debugged

This box shows all program types that are currently running and available for debugging based on the types chosen in Choose Program Types.
