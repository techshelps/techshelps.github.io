Breakpoint Operations in the Disassembly Window

In the Disassembly Window, you can view, set, enable, disable, delete, or edit a file, function, or address breakpoint. You can insert a data breakpoint, but data breakpoints do not appear in the Disassembly window. Once you have created a data breakpoint, you cannot view, set, enable, disable, delete, or edit it from the Disassembly window. If you need to perform one of those operations on a data breakpoint, use the Breakpoints window.

The following list shows the breakpoint operations you can perform from this window:

  • Insert a file breakpoint
  • Insert a function breakpoint
  • Insert an address breakpoint
  • Insert a data breakpoint
  • Delete a breakpoint
  • Enable or disable a breakpoint
  • Edit a breakpoint
