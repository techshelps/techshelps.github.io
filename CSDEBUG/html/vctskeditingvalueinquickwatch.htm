Editing a Value in QuickWatch

To change the value of a variable

  1. Type the variable name in the Expression text box.
  2. Click Recalculate.
  3. If you typed the name of an array or object variable in the Expression box, a tree control appears next to the name in the Current value box. Click the plus (+) or minus (–) in the Name column to expand or collapse the variable to find the element whose value you want to edit.
  4. Double-click the Value column of the Current value box to select the value.
  5. Use the mouse and keyboard to edit the value.
  6. Press ENTER.

    Caution   Editing floating-point values can result in minor inaccuracies due to decimal-to-binary conversion of fractional components. Even a seemingly innocuous edit can result in changes to some of the least significant bits in a floating-point variable.
