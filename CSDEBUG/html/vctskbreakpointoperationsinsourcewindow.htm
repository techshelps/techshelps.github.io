Breakpoint Operations in a Source Window

In a source window, you can view, set, enable, disable, delete, or edit a file, function, or address breakpoint..

The following list shows the breakpoint operations you can perform from a source window:

  • Insert a file breakpoint
  • Insert a function breakpoint
  • Insert an address breakpoint
  • Delete a breakpoint
  • Enable or disable a breakpoint
  • Edit a breakpoint
