Find Symbol, Debugging, Options Dialog Box

The Find Symbol pane of the Options dialog box specifies where the debugger will look for symbols when debugging the solution.

Look for symbol files in the following directories

This box contains a list of directories in which the debugger searches for symbol files when debugging the solution.

Do not look for these symbol files

In this box, you can enter the names of any files that you do not want the debugger to read symbols from. If the debugger finds one of these files in one of the directories specified above, it will ignore it.
