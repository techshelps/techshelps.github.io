Opening the Hit Count Dialog Box

To specify or change a hit count when creating a new breakpoint

  • In the New Breakpoint dialog box, click the Hit Count button.

To specify or change a hit count for an existing breakpoint

  1. In the Breakpoints window, select the breakpoint you want.
  2. In the toolbar at the top of the window, select the Properties tool.
  3. In the Breakpoint Properties dialog box, click the Hit Count button.

To specify or change a hit count for an existing breakpoint in a source or Call Stack window

  1. Right-click the breakpoint and choose Breakpoint Properties from the shortcut menu.
  2. In the Breakpoint Properties dialog box, click the Hit Count button.
