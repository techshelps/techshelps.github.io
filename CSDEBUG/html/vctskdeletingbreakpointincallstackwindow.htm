Deleting a Breakpoint in the Call Stack Window

In the Call Stack window, breakpoint symbols appear in the left margin next to the name of the function call where they are set. You can delete a breakpoint using the procedure given below.

To delete a breakpoint in the Call Stack window

  1. Right-click the line containing the breakpoint symbol (a solid dot or empty circle, depending on the breakpoint state).
  2. From the shortcut menu, choose Remove Breakpoint.
