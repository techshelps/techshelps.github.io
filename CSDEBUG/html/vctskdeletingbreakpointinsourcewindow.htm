Deleting a Breakpoint in a Source Window

In a source window, you can view file, function, or address breakpoints as symbols in the left margin. You can delete these breakpoints using the procedure given below.

To delete a file, function, or address breakpoint

  1. Right-click the line containing the breakpoint symbol (a solid dot or empty circle, depending on the breakpoint state).

    Note   If the breakpoint was set on a specific character (with Step By set to statement), you must right-click the exact character where the breakpoint is set.

  2. From the shortcut menu, choose Remove Breakpoint.
