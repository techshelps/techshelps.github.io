Selecting a Remote Machine

To select a remote machine

  1. In the Processes dialog box, select the machine you want to connect to from the Machine list. If you do not find the machine you want on the list:
    1. Check the Transport box and make sure you have the proper transport selected. Default is the correct setting for most programs. Select Native-only TCP/IP if you are debugging a native program and you cannot use DCOM for some reason.
    2. Click the Ellipsis () button next to the Machine list.
    3. In the Browse for Computer dialog box, use the tree control to locate the computer you want on the selected transport.
    4. Click OK.
    5. The computer you selected in the Browse for Computer dialog box should now appear in the Machine list. Makes sure it is selected.
    6. Click OK.
