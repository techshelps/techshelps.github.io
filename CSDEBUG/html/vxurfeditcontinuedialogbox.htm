Edit and Continue Dialog Box

This dialog box appears when you have edited code while debugging and Edit and Continue recognizes that it can apply code changes.

Don't show me this dialog again

If you select this checkbox, Edit and Continue will apply code changes without asking permission in the future. You can turn this warning on again by going to the Options dialog box, opening the Debugging Folder, Edit and Continue category, and selecting Invoked by debug commands and Ask me first.
