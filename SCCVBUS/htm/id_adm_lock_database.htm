Lock SourceSafe Database Command (Tools Menu)

See Also

Allows you to lock all users out of the VSS database while you run a utility, upgrade the executables, or back up the database.

Access Rights

You must have Administrator permission to use this command.

Remarks

Use this command if you need to archive, back up, or run a VSS utility on the database. Note that the act of selecting the check box in the Lock SourceSafe Database dialog box is what actually locks users out. After this box is checked, no one will be allowed to log into the database. The users listed, who are already logged in, will be unaffected you will need to notify them manually to log out.
