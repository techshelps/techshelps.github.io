Destroy cannot be undone; information will be lost permanently!

See Also

This warning has the following cause and solution:

  • When you use the Destroy command on a file or project, the action is not reversible.

    Click Yes to continue with the Destroy operation. Click No to cancel the operation.

Note   If you do not want to receive this message every time you destroy a file or project, you can disable it on the Warnings tab in the SourceSafe Options dialog box on the Tools menu.
