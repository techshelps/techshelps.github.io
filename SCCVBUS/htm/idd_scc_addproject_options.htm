Add SourceSafe Project Dialog Box (Visual SourceSafe)

Project

Type the name of the VSS project you want, or click in the project list to choose a project. You can enter a full project path specification (as in $/CODE/RESOURCES) or just type a project name to have a project created under the current project.

OK

Adds the current development environment project to VSS.

Cancel

Cancels the Add Project to SourceSafe command.

Create

Creates a new VSS project. Type a name under Project and click Create.
