Source Code Control Results Dialog Box

Places command results in a window.

To display the command result dialog box:

  1. Set the Results_Window variable to yes in the SRCSAFE.INI file.

  2. Perform a get or check out of a file.

When you carry out a command that generates a status or error message, VSS displays the results of the command in the Results window.  This window remains displayed until you press the ENTER key, click Cancel (or Close), or click the VSS Explorer window.

If you'd like, you may instruct VSS not to display the Results window.  Clear the Show Results Window box in the View tab of the SourceSafe Options dialog box (Tools menu).  This automatically sets the Results_Windows initialization variable to No in your SS.INI file.  Error messages, however, are still displayed.
