The deploy path <path> is not a proper file system or FTP deploy path

This error has the following cause and solution:

  • One of the paths entered for deploying the project either is not a proper operating system file system path or is not a proper ftp URL.

    Ftp paths can look like one of the following.

    ftp://user:user@www.microsoft.com/ssafe
ftp://user@www.microsoft.com/ssafe
ftp://www.microsoft.com/ssafe

    They always start with "ftp://". They always end with the ftp address (server name and, optionally, directory path). In between, you may have nothing (as in the bottom example above). Or, you may have a username (as in the middle), or a username and password (as on the top).
