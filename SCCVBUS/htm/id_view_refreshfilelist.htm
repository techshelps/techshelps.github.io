Refresh File List Command (View Menu)

See Also

Refreshes the file list in VSS Explorer.

Access Rights

Any VSS user can use this command.

Remarks

Refreshes the file list only, not the project list. To refresh both lists, collapse the project list to a parent project, then double-click that project to expand the project list.

You can refresh the file list automatically. Set the timing with the Automatically Refresh File List option on the View tab in the SourceSafe Options dialog box on the Tools menu.
