Create Local Project From SourceSafe Dialog Box

Allows user to create a new development project from an existing VSS project.

To use the Download SourceSafe Project command:

  1. From the File menu, click Open Workspace.

  2. Select the Source Control button in the lower right corner of the dialog box.

Dialog Box Options

Create a new project in this folder

The working folder for the selected project.  Click Browse to create or change the working folder.

SourceSafe Project to download

VSS project list.  Click the project you want to open.

OK

VSS gets the files, places them in your working folder, and opens the project in your development environment so that you can edit it.

Cancel

Cancels the Download SourceSafe Project command.
