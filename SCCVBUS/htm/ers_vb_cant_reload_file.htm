Can't reload file <file> in the Visual Basic project.

This error has the following cause and solution:

  • The file cannot be re-loaded in the Visual Basic project.

    Be sure the Visual Basic project file is checked out. Check the filename, and if it is correct, you may have to reload the file in the Visual Basic project. This may require getting a new copy of the file from your source code control provider and placing it into your working folder, or may require restarting Visual Basic.
