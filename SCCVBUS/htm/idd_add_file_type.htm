Add File Group Dialog Box

See Also

Adds a new file group to the File Types tab in the SourceSafe Options dialog box (Tools menu).

To display the Add File Group dialog box:

  • On the Tools menu, click Options, and then click File Types.  Click the Add button.

Dialog Box Options

Name

Type the name of the file group as you would like to identify it in the file group list on the File Types tab.

OK

Adds the file group.

Cancel

Cancels the Add File Group command.
