Browse Folder Dialog Box

For more information on the Set Working Folder dialog, see the Set Working Folder Command (File Menu)topic.

Dialog Box Options

Name

Name and path of folder.

Folder

Your current folder.

Folders

Specifies a folder.

Drives

Selects the drive that contains the folder.

OK

Confirms the folder selection.

Cancel

Cancels the selection, closes the dialog box, and returns to VSS Administrator.

Create Folder

Creates a new folder as specified. Type a new folder name in the Name box, then click Create Folder to create the folder.
