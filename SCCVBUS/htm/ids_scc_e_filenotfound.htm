File <File name> (file <File name> in VSS) was not found

This error has the following cause and solution:

  • VSS was unable to locate the specified file.

    Run the Analyze utility to make sure there are no errors in your VSS database. If running Analyze with the –F switch does not correct the problem please contact Product Support.

Note   When running the Analyze utility first run it with no switches to find any errors in the database and then with run it with the –F switch to run in autofix mode.
