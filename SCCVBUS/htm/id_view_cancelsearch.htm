Cancel Search Command (View Menu)

See Also

Restores the full file list in VSS Explorer for the current project. This command is used after a search operation, which usually filters the file list to a smaller subset.

Access Rights

Any VSS user can use this command.

Remarks

When you execute the Search command, VSS displays the word Search in the status bar. Click the Cancel Search command to return to the full VSS Explorer file display. The word Search disappears from the status bar.
