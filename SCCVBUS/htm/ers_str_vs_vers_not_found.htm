Version not found

See Also

The version specified cannot be found. This error has the following causes and solutions:

  • You specified a version that does not exist.

    In VSS, click the Show History command on the Tools menu, and then select a version from within the History of File dialog box.

  • If you are using VSS from the command line, you may not be using the -V switch properly.

    Search online Help for more information on using the parameter correctly (click the Contents button).
