Undo Check Out Command (Development Environment)

Overwrites your changes to a file, and copies the latest checked in version to your development environment working folder.

To use the Undo Check Out command

  • On the Visual Basic Tools SourceSafe menu, click Undo Check Out.

Dialog Box Options

Files to Un-Check Out

Lists the currently checked out files. Click the files whose check out you want to undo, or click Select All to select all the listed files.

OK

Confirms the selection.

Cancel

Cancels the selection.

Select All

Selects all files listed.
