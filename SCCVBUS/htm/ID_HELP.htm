Help Menu Commands

The Help menu commands provide Help on using VSS and Help on specific VSS commands. In addition, you can press F1 at any time to view Help on the currently selected menu item or dialog box.

Help command Action
Contents Runs Help and displays the VSS Table of Contents.
Search For Help On Displays the Help Search dialog box so you can quickly find the information you want.
Technical Support Displays information about Microsoft Product Support Services.
About Microsoft Visual SourceSafe Displays a dialog box with information about your copy of Microsoft VSS, including the version number and the copyright, legal, and licensing notices.

Access Rights

Any VSS user can use these commands.

Remarks

A quick way to find information is to use Search, an automated index with hundreds of index entries and extensive cross-referencing.

You can also look for Help topics in the VSS Table of Contents.

As you read a topic in VSS Help, you can scroll through the topic with the scroll bar, the arrow keys, or the PAGE UP and PAGE DOWN keys.
