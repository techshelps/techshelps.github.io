Checkouts Dialog Box

Shows the various locations to which you have a file checked out.

To display the Checkouts dialog box:

  • On the SourceSafe menu, click File Properties and select the Check Out Status tab.

Dialog Box Options

Check Outs

Lists the locations that the file is checked out to.

Details

Shows check out status for the selected file.

Check In

Checks in the selected file.

Undo Check Out

Undoes the check out of the selected file.

Close

Closes the dialog box, and cancels the command.
