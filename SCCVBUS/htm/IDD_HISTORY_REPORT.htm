History Report Dialog Box

See Also

Sends a report of historical information on a file or project to a printer, a file, or the Clipboard.

To display the History Report dialog box:

  1. Select a file or project in VSS Explorer, and on the Tools menu, click Show History. 

  2. Make selections in the File History Options or Project History Options dialog box (if enabled), and click OK.

    Note   Enable the display of the File History Options dialog box on the Command Dialogs tab in the SourceSafe Options dialog box (Tools menu).

  3. In the History of File or History of Project dialog box, click Report.

Dialog Box Options

Include Details

Includes all historical details of the selected file(s) in the report.  Details include creation date, user who modified the file for each check in, comments applied to the file, and time/date information for each event.

Include Differences

This option is useful when you are comparing two different versions of a file or project.  The report includes differences between the selected files or projects in the report. 

Report To

Sends the history report to a printer, a file, or the Clipboard.

OK

Prepares the specified report.

Cancel

Cancels the report.

Preview

Shows a preview of the specified report, so that you can determine if it is what you wanted before committing it to the printer queue or a file.
