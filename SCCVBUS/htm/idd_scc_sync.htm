Get From <Project Name> Dialog Box

Gets the selected files from your source code control provider and places a read-only file in the development environment project.

To use the Get command:

  • On the Visual Basic Tools menu, click Get.

Dialog Box Options

Files to Get/Check In

Selects the file or files you want to get or check in.

Advanced

Displays the Advanced Options dialog box.

OK

Confirms your selections.

Cancel

Cancels your selections.

Select All

Selects all files listed.
