Difference Options Tab

See Also

In this tab you can specify options for viewing differences between files or projects. These options are used in other VSS dialog boxes such as the Project Difference dialog box, the Visual Merge dialog box, the default VSS editor, and the VSS results pane.

To display the Difference options tab:

  1. On the Tools menu, click Options.

  2. Select the Difference tab.

Tab Options

You can specify/change all of the following options.

Colors

Select colors for lines VSS will use. The selected line color is shown in the Sample box.

Foreground

Select the foreground color.

Background

Select the background color.

Font

Select the font to be used in VSS Explorer and all its dialog boxes. To modify the font, click the Modify button.

Sample

The option(s) you've selected are displayed in this box.

Reset

Click here to reset all values on this tab to their defaults.

Tabstop Width

Specify the equivalent number of spaces each tab stop equals.

Shows Line Numbers

Select this box to show line numbers in the visual difference viewers.

Ignore White Space Differences when Merging Files

Select this box to ignore differences in white space (tabs, space characters, and so on) when merging two files.
