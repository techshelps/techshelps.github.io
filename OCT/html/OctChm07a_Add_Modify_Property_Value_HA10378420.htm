Add/Modify Property Value dialog box

In the Modify setup properties dialog box, if you click Add, the Add Property Value dialog box opens. If you select an entry in the Modify setup properties dialog box, and then click Modify, the Modify Property Value dialog box opens. You can enter the following information in the Add Property Value or Modify Property Value dialog boxes:

  1. In the Add Property Value or Modify Property Value dialog boxes, in the Name box, type the name of the property. Property names must be uppercase.
  2. In the Value box, type the value of the property.
  3. Click OK.
