Office Customization Tool

Add installations and run programs

Runs additional executable programs before or after the Office installation is complete.

Setup runs all command lines in the order listed and does not complete the installation until all command lines are executed.

 Note   Do not add a program that requires that the computer restart, unless it is the last program in the list. If a custom program restarts the computer, the Office installation will be completed. Any custom programs that appear later in this list will not be run.

To add a program to the list, click Add. To modify a program, select it, and then click Modify. To remove a program, select it, and then click Remove. To change the order of the list, select a program, and then click the Move arrows to move the program up or down in the list.
