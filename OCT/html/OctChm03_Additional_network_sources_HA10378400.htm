Office Customization Tool

Additional network sources

Specifies additional servers that have a copy of the network installation point.

If the original network installation point is unavailable and Setup is installing a feature on demand or repairing Office, then Setup checks the servers in this list in the order specified.

To add a server to the list, click Add. To edit an entry in the list, select a server, and then click Modify. To remove a server from this list, select it, and then click Remove. To change the order of the list, select a server, and then click the Move arrows to move the server up or down in the list.

To remove all servers from the list, click Clear existing server list.

 Note   Office Customization Tool (OCT) does not verify the server path or attempt to connect to the server when you add it to the list.
