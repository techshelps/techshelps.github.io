Office Customization Tool

Delete/Modify Registry Entry dialog box

In the Delete Registry Entry or Modify Registry Entry dialog box, type the following information for each registry entry, and then click OK:

Expand the Root menu, and then select the branch that contains the entries that you want to remove or modify.
In the Key box, type the name of the subkey where the entry is stored; for example, Software\Microsoft\Office\14.0\Common.
For Value name, type the name of the registry entry that you want to remove. To remove the entire subkey, including all of its value entries, leave this field blank.
