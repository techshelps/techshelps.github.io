Office Customization Tool

Add/Modify Network Server Entry dialog box

The Add Network Entry dialog box opens when you click Add in the Additional network sources dialog box. If you select an entry in the Additional network sources dialog box, and then click Modify, the Modify Network Entry dialog box opens.

  1. In the Add Network Entry or Modify Network Entry dialog boxes, in the Network server box, type the path and name of the server. You can use a UNC path or, if all users have access to the server with the same drive letter, you can use a drive letter in the path. The path can contain environment variables delimited by percent signs (%) if they are defined for all users.

    For example:

    z:\office
    \\server1\share\office_root
    %USERNAME%\office_root

    You can also click Browse to search for a server on the network.
  2. Click OK.
