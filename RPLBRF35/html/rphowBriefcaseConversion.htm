Convert a database while in Briefcase

The database file you selected isn't a replica .mdb file. To make a replica of your database, it must first be converted to a Design Master. Converting your database increases the size of your file because new, hidden system fields and tables are added to your database.

Before you convert your database, remove the database password. Setting user permissions to access the database doesn't interfere with synchronization.
