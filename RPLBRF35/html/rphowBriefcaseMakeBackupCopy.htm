Make a backup copy of a replicated Microsoft Access database

When you drag a Microsoft Access database file to the My Briefcase icon or you point to Replication on the Microsoft Access Tools menu and click Create Replica, you are given the option to create a backup copy of the original database. It's a good idea to make a backup; however, you should use it only in emergency situations. Any copy made from the backup copy of the database, or any replica made from a converted backup copy, will not be able to synchronize with the existing members in the replica set.
