GetFile Method		  Scripting Run-Time Reference 
Version 3 

See Also                  Applies To

Description
Returns a File object corresponding to the file in a specified path.
Syntax
object.GetFile(filespec)

The GetFile method syntax has these parts:

Part Description
object Required. Always the name of a FileSystemObject.
filespec Required. The filespec is the path (absolute or relative) to a specific file.

Remarks
An error occurs if the specified file does not exist.

The following example illustrates use of the GetFile method: 

Function ShowFileAccessInfo(filespec)
  Dim fso, f, s
  Set fso = CreateObject("Scripting.FileSystemObject")
  Set f = fso.GetFile(filespec)
  s = f.Path & "<br>"
  s = s & "Created: " & f.DateCreated & "<br>"
  s = s & "Last Accessed: " & f.DateLastAccessed & "<br>"
  s = s & "Last Modified: " & f.DateLastModified  
  ShowFileAccessInfo = s
End Function
