Materials and Mapping Tutorials
 
 
 

In these tutorials, you'll learn how to use 3ds Max for creating materials and maps and applying them to your models and scenes. You'll also learn about some of the tools in 3ds Max for using materials and maps.

Features Covered in This Section

  • Using the 3ds Max Material Editor to create, edit and assign materials.
  • An introduction to Shaders.
  • How mapping coordinates work, and how to manipulate them using 3ds Max modifiers.
  • Using Displacement Mapping along with surface properties.
  • Manipulating transparency and raytrace properties.
  • Common techniques for creating materials for architectural realism.
