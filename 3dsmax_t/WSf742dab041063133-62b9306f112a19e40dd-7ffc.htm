Modeling a Chess Set
 
 
 

In this tutorial, you will create four pieces of a chess set—a pawn, a bishop, a rook and a knight—using various modeling tools and techniques.

In this tutorial, you will learn how to:

  • Create and edit spline objects.
  • Use Lathe modifier to create a 3D object.
  • Use Face extrusion to create geometry.
  • Use Boolean compound objects.
  • Use viewport background images.
  • Use the Surface modifier.

Skill level: Beginner

Time to complete: 1 hour

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the \tutorials\intro_to_modeling directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
