Project Management
 
 
 

The tutorials in this section relate to different asset management and scene configuration features that help speed up collaborative workflow and ultimately save time in your projects. You will also learn how to work with files from the AutoCAD package and create 3D scenes that can be viewed with Autodesk's standalone DWF viewer.

Features Covered in This Tutorial

  • Revising asset paths.
  • Creating a new vault.
  • Logging into the vault using 3ds Max.
  • Setting a working folder.
  • Adding files to the vault.
  • Checking files in and out of the vault.
  • Checking version history.
  • Checking files out of the vault.
  • Preparing an AutoCAD drawing for import.
  • Creating AEC objects using a 2D CAD drawing as a template.
  • Importing a 3D CAD model and making final adjustments.
  • Using the File Link Manager to create a live link between AutoCAD and 3ds Max.
  • Diagnosing problems you might encounter with the File Link Manager.
  • Fixing problems you might encounter with CAD drawings.
  • Solving common problem with CAD drawings and models.
  • Exporting 3D scenes to the standalone DWF viewer program and resolving limitations of the DWF format.
