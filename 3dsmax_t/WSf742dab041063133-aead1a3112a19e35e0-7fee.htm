Creating Pleats
 
 
 

In this final Cloth tutorial, you will learn more about creating specific pleats within a garment to make it look as if it's just been ironed and pressed.

In this tutorial, you will learn how to:

  • Use a pattern with pleats in Garment Maker.
  • Create seams for pleats.

Skill level: Intermediate

Time to complete: 20 minutes

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the \tutorials\cloth directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
