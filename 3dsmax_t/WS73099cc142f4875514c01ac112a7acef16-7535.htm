Using Asset Tracking to Revise Asset Paths
 
Autodesk 3ds Max Tutorials > Project Management > 
Asset Tracking
 
 
 

You can perform asset management directly from 3ds Max using two features:

  • Open from Vault
  • Asset Tracking

These features provide direct access to Asset-Tracking Systems (ATS) such as:

  • Autodesk Vault
  • Perforce
  • SourceSafe

ATS applications provide a variety of features such as version control and tools for backing up project assets. Typically, ATS applications are employed in a networked team environment where assets are shared among members. The recommended solution for asset tracking with 3ds Max is Autodesk Vault.

You can use the asset tracking feature in 3ds Max independent of external ATS applications. For example, you can use it to modify paths for bitmaps and XRefs. If you do have an external ATS setup on the network, then you can also use asset tracking directly to access these applications. 3ds Max includes Autodesk Vault. You can use the asset tracking feature of 3ds Max to access items from Autodesk Vault and other asset-tracking systems directly.

Before starting this tutorial, install Autodesk Vault or a comparable product on your computer network. Also, make sure you've installed at least one of the Autodesk Vault plug-ins, available from the Install page of the 3ds Max installation program.

In this tutorial, you will learn how to:

  • Revise asset paths.
  • Create a new vault.
  • Log into the vault using 3ds Max.
  • Set a working folder.
  • Add files to the vault.
  • Check files in and out of the vault.
  • Check version history.
  • Check files out of the vault.

Skill level: Advanced

Time to complete: 2 hours

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the \tutorials\asset_management directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
