Grooming a Dog
 
 
 

In this tutorial you'll use Hair to add fur to a dog model, and then groom the fur. The tutorial also covers preparing the model for automatically growing fur of different lengths by adjusting polygon sizes.

In this tutorial, you will learn how to:

  • Use the Hairbrush, Translate and Recomb tools to groom fur.
  • Scale fur to the desired length.
  • Add color and other material settings to the fur.

Skill level: Intermediate

Time to complete: 30 minutes

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the \tutorials\hair directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
