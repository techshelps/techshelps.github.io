Creating Explosions
 
 
 

In this tutorial, you’ll hit an asteroid with a glowing laser blast, blowing the asteroid to bits and creating a fiery explosion.

In this tutorial you will learn how to:

  • Use material effects channel IDs.
  • Use animated opacity mapped plane objects to simulate an explosion.
  • Use particle array fragmentation.
  • Set visibility keys.
  • Add motion blur.
  • Use render effects glows and blurs.

Skill level: Intermediate

Time to complete: 2+ hours

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the \tutorials\space directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
