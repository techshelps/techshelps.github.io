Checking the File Back into the Vault
 
 
 

Check the file into the Vault:

  1. Open the Asset Tracking dialog.
  2. Highlight the file you have open: Full House.max.
  3. From the File menu, choose Checkin.

    A warning appears indicating that the scene has been modified and needs to be saved.

  4. Click Yes.

  5. In the comment window for this version, enter “Text Added”.
    NoteThe Keep Checked Out check box should be off.

  6. Click OK to close the dialog.

    The check mark that appeared next to the Full House entry no longer appears, indicating that the file isn't checked out.

  7. Close the Asset Tracking dialog.
