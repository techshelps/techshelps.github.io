Logging into the Vault Server from 3ds Max
 
 
 
  1. From the File menu, choose Asset Tracking.
  2. From the Server menu, choose Log In.
  3. Click the ellipsis button adjacent to the Database drop-down list.
  4. Highlight the FullHouse database and click OK.

  5. Click OK to close the Vault Log In dialog.
    TipIf the vault is on your local machine and you have trouble logging in using the machine name as the Server name, try using localhost as the Server name instead.
  6. Close the Asset Tracking dialog.
