Character Animation Tutorials
 
 
 

The following tutorials introduce you to the fundamentals of character animation using 3ds Max 's biped system. You will also learn how to create and control a virtual skeleton inside your character, which will then drive its motions.

Features Covered in This Section

  • Adjusting your biped with Figure Mode.
  • Applying Physique.
  • Creating a walk cycle using Footsteps.
  • Creating and editing a generated walk cycle using Footsteps.
  • Setting keys in freeform mode.
  • Combining animation clips to create an animated sequence.
  • IK linking different biped limbs to objects.
  • Animating multi-legged creatures.
  • Working with motion-capture data.
  • Setting up a simple IK structure.
  • Skin a character to a skeleton.
