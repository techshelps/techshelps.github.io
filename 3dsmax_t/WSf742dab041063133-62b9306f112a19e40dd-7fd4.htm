Constructing the Thumb
 
 
 

You will now create a thumb using the same extrusion method you used in the previous lesson.

Construct the thumb:

  1. Continue with the previous scene, or load the scene file called stool_fingers.max.
  2. In the Selection rollout, set the sub-object level to Polygon.

  3. Select the polygon on the side of the hand.

  4. On the Modify tab > Edit Poly modifier > Edit Polygons rollout, click the Extrude button to open the Extrude Polygons dialog.

  5. On the Extrude Polygons dialog, set Extrusion Type to Group. Set Extrusion Height to 0.76m. Click OK to accept the edits.

  6. Using the same extrusion settings, extrude the top face of the thumb part.

  7. Scale and move the top face as shown below.

  8. Extrude the thumb top polygon using an extrusion height of 0.5m.

  9. Scale and move the top face as shown below.

  10. Extrude the thumb top polygon once again using an extrusion height of 0.5m.

  11. Continue to extrude and rotate the last face until the thumb resembles the one shown in the example below.
