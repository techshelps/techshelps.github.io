Special Effects
 
 
 

This section covers a range of effects you may encounter in your everyday life, such as simple cigarette smoke, water from an opened fire hydrant, fog effects — even tumbling down a set of stairs! You will learn how to use particle systems with space warps, add environment and lens effects, as well as create many simulations using the reactor toolset.

Features Covered in This Section

  • Creating particle systems.
  • Using forces and space warps.
  • Emitting particles from geometry.
  • Using materials with particles.
  • Creating and manipulating a camera viewport.
  • Using different lens effects.
  • Setting up a volume light.
  • Creating fog.
  • Creating rigid body collections.
  • Creating simulations with reactor.
  • Setting physical properties for objects in simulation.
  • Using Hinge and Rag Doll constraints.
