Using the Limit Controller
 
 
 

In this tutorial, you will learn how to limit the range of a rotation by using the Limit Controller.

In this tutorial you’ll learn how to:

  • Assign a controller to an animation track.
  • Set minimum and maximum limits to a controller range.

Skill level: Beginner

Time to complete: 15 min.

Files for This Tutorial

All the files necessary for this tutorial can be found on the program disc in the \tutorials\controllers directory. Before starting the tutorials, copy the \tutorials folder to your local program installation.
