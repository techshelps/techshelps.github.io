Refining the Airplane
 
 
 

3ds Max has modeling features that can help you build models more efficiently. These lessons describe each feature and show you practical ways you can use them when constructing models.

In this tutorial, you will learn how to:

  • Weld vertices with the Vertex Weld modifier.
  • Add detail to your model using Editable Poly tools.
  • Select scene elements using soft selection.
  • Use the HSDS modifier.
  • Use the Edit Normal modifier.

Skill level: Beginner to Intermediate

Time to complete: 2 hours (15 to 20 minutes per lesson)

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the \tutorials\p38_lightning directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
