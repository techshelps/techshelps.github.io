Modeling an Apple
 
 
 

In this tutorial, you will create an apple. You’ll add Taper and Displace modifiers to reshape a sphere to look like an apple, then use additional tools to improve its shape and add a stem.

In this tutorial, you will learn how to:

  • Use the Taper modifier.
  • Create sub-object selection sets.
  • Use soft selections to blend changes to a mesh.
  • Use a Displacement modifier to shape your mesh.
  • Use the Bend modifier to shape a stem.

Skill level: Beginner

Time to complete: 25 minutes

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the \tutorials\intro_to_modeling directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
