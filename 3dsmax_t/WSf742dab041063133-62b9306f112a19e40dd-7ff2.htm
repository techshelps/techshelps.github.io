Creating a Realistic Hourglass
 
 
 

Here, you will learn to model a realistic hourglass by taking cylinder and hose objects and combining them with Free-Form Deformation (FFD), Taper, and Slice modifiers. You'll even create the flow of sand using a particle system.

In this tutorial, you will learn how to:

  • Use the Taper, and FFD modifiers to shape the hourglass.
  • Use an animated Slice modifier to create the shifting sand level, in the upper globe of the hourglass.
  • Use a Particle system to create the falling sand in the lower globe of the hourglass.
  • Use an animated hemisphere to create the sand piling up.

Skill level: Beginner

Time to complete: 40 minutes

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the \tutorials\hourglass directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
