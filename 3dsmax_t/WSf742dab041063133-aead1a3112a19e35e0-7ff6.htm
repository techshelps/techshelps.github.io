Copying Hair: Transplants, Wigs, and Toupees
 
 
 

Once you’ve created mesh-grown hair, you can use a number of ways to move it from one character or creature to another. This tutorial demonstrates three of them.

In this tutorial, you will learn how to:

  • Copy mesh-grown hair from objects by selecting essential geometry only.
  • Convert hairs on existing objects to editable splines, which can subsequently be used to grow new hair.
  • Copy hair splines (from which the hairdo is styled) to the surface of a new object and re-apply the original hair settings.

Skill level: Intermediate

Time to complete: 1 hour

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the \tutorials\hair directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
