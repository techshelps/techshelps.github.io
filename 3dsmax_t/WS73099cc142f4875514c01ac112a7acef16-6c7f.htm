Biped Beyond the Basics
 
 
 

This section expands on the fundamentals of biped animation by introducing new tools and techniques which let you to do more with your character, such as adding wings to his back, letting him swing a sword around, and assigning him a set of motion-captured movements. Note that you should be accustomed with the basics of biped animation before starting the following tutorials.

Features Covered in This Section

  • IK linking different biped limbs to objects.
  • Animating multi-legged creatures.
  • Working with motion-capture data.
  • Setting up a simple IK structure.
  • Skin a character to a skeleton.
