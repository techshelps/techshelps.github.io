Creating the Fingers
 
 
 

In this lesson you will continue to use the Edit Poly modifier to add fingers to the stool base.

Add the fingers:

  1. In the Perspective viewport, select the upper back-facing polygon.

  2. Press F2 to get an outlined selection of the face.

  3. In the Modify tab > Edit Poly modifier > Edit Geometry rollout, click the Tessellate button to open the Tessellate Selection dialog.

  4. On the Tessellate Selection dialog, set the options as shown below and click OK.

  5. Redo the last tessellate operation to divide the face into 16 sections.

  6. On the Selection rollout, set the sub-object level to Edge.

  7. Select the following edges using the Ctrl key.

  8. On the Edit Edges rollout, click the Remove button to delete the selected edges. Do not use the delete key to remove the edges.

  9. On the Selection rollout, set the sub-object level to Vertex.

  10. Select the following vertices using the Ctrl key.

  11. On the Edit Vertices rollout, click the Remove button to delete the selected vertices.
  12. Press F2 to exit the outlined selection display mode.
  13. On the Selection rollout, set the sub-object level to Polygon and select the four newly-created faces.

  14. On the Modify tab > Edit Poly modifier > Edit Polygons rollout, click the Extrude button to open the Extrude Polygons dialog.

  15. On the Extrude Polygons dialog, set the Extrusion Type to By Polygon and Extrusion Height to 0.76m. Click OK to accept the edits and close the dialog.

  16. Scale the selected polygons as shown below.

  17. On the Modify tab > Edit Poly modifier > Edit Polygons rollout, click the Bevel button to open the Bevel Polygons dialog.

  18. On the Bevel Polygons dialog, set Bevel Type to Group. Set Bevel Height to 0.5m and Outline Amount to 0.13m. Do not click OK yet.

  19. Click on the top faces of the finger base using the Ctrl key to start creating the fingers.

  20. On the Bevel Polygons dialog, click Apply.

  21. On the Bevel Polygons dialog, set Outline Amount to -0.13m. Click Apply.

  22. On the Bevel Polygons dialog, set Outline Amount to -0.13m. Click Apply.

  23. On the Bevel Polygons dialog, set Outline Amount to -0.13m. Click Apply.

  24. On the Bevel Polygons dialog, set Outline Amount to -0.13m. Click Apply.

  25. On the Bevel Polygons dialog, set Outline Amount to -0.13m. Click Apply, then OK to close the dialog.

  26. Save the scene as my_stool_fingers.max.
