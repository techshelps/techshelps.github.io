Animating a Multi-legged Character
 
 
 

In this tutorial, you’ll use a biped to animate a quadruped (four-legged creature). You’ll use IK Blend on the hands and feet to create a freeform animation with changing pivot points.

In this tutorial, you will learn how to:

  • Animate a leaping quadruped with freeform animation.
  • Use IK Blends and pivot points on the hands and feet.
  • Use Bones to create extra limbs.

Skill level: Intermediate to Advanced

Time to complete: 1.5 to 2 hours

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the tutorials\character_animation\quadruped directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
