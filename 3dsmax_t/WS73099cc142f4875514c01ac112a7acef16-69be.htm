Skinning
 
 
 

This tutorial shows you how to use skin and flex modifiers. The skin modifier associates a skeleton with a mesh, allowing you to animate the mesh by animating the skeleton. This modifier works on simple animated structures as well as character meshes. You can use it to control creasing and muscle bulging. The flex modifier lets you enhance character realism by adding secondary motion.

In this tutorial, you will learn how to:

  • Apply the Skin modifier.
  • Use envelopes to adjust the effects of the Skin modifier.
  • Use a Joint Angle deformer to improve skin deformation.
  • Use a Bulge Angle deformer to simulate a flexing muscle.
  • Use the Flex modifier to add secondary motion to animation.

Skill level: Intermediate

Time to complete: 2 hours

Files for This Tutorial

All the files necessary for this tutorial are provided on the program disc in the \tutorials\character_animation\skin_flex directory. Before starting the tutorials, copy the \tutorials folder from the disc to your local program installation.
