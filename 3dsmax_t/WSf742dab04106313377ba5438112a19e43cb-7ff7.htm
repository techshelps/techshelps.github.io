Checking Version History
 
 
 

Check version history:

  1. From the File menu, select the Asset tracking dialog.
  2. Highlight Full House.max.
  3. From the File menu choose History.
  4. A dialog appears indicating that Full House.max is on its second version.

    The Comments field helps indicate the changes that were made.

  5. Close the History dialog.
  6. Close the Asset Tracking dialog.
  7. From the File menu, choose Reset.
  8. At the prompt, click Yes.

    A warning appears indicating that you are trying to save the file when it’s not checked out.

  9. Click No.

    An IO Error dialog indicates that you cannot overwrite your local version of the file because it is read-only. Version control helps ensure the Vault files are kept synchronized with any local versions.

  10. Click OK to close the IO Error dialog.
  11. From the File menu, choose Reset.
  12. This time, click No at the prompt to save changes.
  13. Close 3ds Max.
