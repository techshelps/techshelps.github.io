About the Desktop Menu

The Desktop menu is the third item on the NVIDIA Control Panel menu bar.   

 

Desktop Menu Command

Description

Add Desktop Context Menu

This is selected by default, and adds the NVIDIA Control Panel menu item to the desktop context menu.

Add "Run with graphics processor" Option

Select this option to add "Run with graphics processor" to the program context menu.

When this option is selected, you can choose which graphics processor to use when starting a program as follows:

  1. Right-click the program icon.

  2. Click Run with graphics processor and then click the graphics processor to use.

The selection applies only at the time the program is launched.  

NOTES:

  • This menu option appears only with systems using NVIDIA's power-saving GPU technology.

  • Some programs require elevated user privileges in order to use the Run on graphics processor option from the program's right-click menu. A pop-up dialog will let you know if that is the case. In this case, use the controls in the NVIDIA Control Panel->Manage 3D Settings page to select the graphics processor to use.

Display GPU Activity Icon in Notification Area

Select this option so that the GPU Activity icon appears in the Windows notification area of the taskbar. You can then click the icon to see which programs and displays are using the NVIDIA GPU.

This menu option appears only with systems using NVIDIA's power-saving GPU technology.

Show Notification Tray Icon

Clicking the NVIDIA Control Panel notification icon on the Windows taskbar notification area offers a quick way to configure key NVIDIA Control Panel settings.

Select Desktop > Show Notification Tray Icon to toggle showing/hiding the NVIDIA Control Panel notification tray icon appears in the Windows taskbar notification area.
