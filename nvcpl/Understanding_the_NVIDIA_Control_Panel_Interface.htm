Understanding the NVIDIA Control Panel Interface

The NVIDIA Control Panel provides control of NVIDIA graphics driver settings as well as other NVIDIA utilities that you might have installed on your system.

The NVIDIA Control Panel user interface is divided into these main areas: the menu bar and toolbar, the task area, and the navigation pane, which are explained in these sections:
