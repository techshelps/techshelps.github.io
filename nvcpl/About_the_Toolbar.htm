About the Toolbar

You can use the NVIDIA Control Panel toolbar to do the following:

  • Use the arrow icons to scroll back and forth between the NVIDIA Control Panel task pages that you have visited.

  • Depending on the PC manufacturer, there is a home button that opens the NVIDIA Welcome page.
