Format data in Microsoft Query

Some of the content in this topic may not be applicable to some languages.

Microsoft Query saves the changes when you save your query. If you format data in Query, the formatting is not displayed when you return the result set to Microsoft Excel. If you change the order of the fields or the fields that are included in the result set, those changes are retained when you return data to Excel.

What do you want to do?

Change a field column heading so that it's more descriptive

Change the width of a column or the height of rows

Change the font, style, or size of text in a result set

Hide or display a field in a result set

Change the layout of a result set
