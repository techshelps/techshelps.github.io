Microsoft Query won't let me print the result set.

Query has no print capability Microsoft Query is a tool that retrieves data from a database. It cannot create printed reports directly.

Print the records from Excel To print the result set, return it to Microsoft Excel by clicking Return data to Microsoft Excel on the File menu. Once you have the result set in Microsoft Excel, you can use all of the formatting and printing commands in Microsoft Excel to arrange and print the data. For more information about printing in Microsoft Excel, see Microsoft Excel Help.
