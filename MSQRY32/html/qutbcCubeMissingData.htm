I can't add fields to my OLAP cube.

After you define an OLAP cube in the OLAP Cube Wizard, you cannot insert more fields from the original relational database. To add more fields, you'll need to modify the database query (.dqy) file and then re-create the cube.

Open or create the query If you saved the database query that you created when you created the cube, you can open the .dqy file. If you did not save the query, you can create a new query.

Add fields to the query Add the fields you need to the query. If you think you might need a cube with different fields in the future, save a copy of the query so that you don't have to re-create the query each time.

Create a new cube After you have a query with all of the fields you want for the cube, you can create a new cube.
