ViewPane Command (DDE)

Specifies which panes in the query window are displayed. This command is available on both the system channel and the query channel.

Syntax

ViewPane(pane)

pane Specifies which panes are displayed. Can be one of the following values:

Value Description
0 (zero) Data pane only
1 Data pane and Table pane
2 Data pane and Criteria pane
3 Data pane, Table pane, and Criteria pane
