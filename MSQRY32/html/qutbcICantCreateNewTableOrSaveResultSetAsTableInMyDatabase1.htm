I can't save the result set as a table in my database.

Microsoft Query 2000 and later do not allow you to save the result set as a table in the database.

Import the data into the database from Microsoft Excel If your database can import Microsoft Excel data, you can return the result set to Microsoft Excel and then use the commands in your database to import the Microsoft Excel data as a table. For example, Microsoft Access has a spreadsheet import wizard that you can use to create a table from Microsoft Excel data. For more information, see the documentation or Help for your database.

Import a text file into the database If your database can import text files, you can return the result set to Microsoft Excel, save it as a text file, and then use the commands in your database to import the text file as a table. For information about text conversion, see Microsoft Excel Help.
