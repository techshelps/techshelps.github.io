Can't edit multiple-table query.

You can't edit data in a query that contains more than one table.

  • To edit data in a table, add only that table to the query, and then click Allow Editing on the Records menu.
  • To edit data in the current query, delete any extra tables from the query; keep only the table that contains the data you want to edit. Then click Allow Editing on the Records menu.

Note  When you delete a table in a query, the data in the underlying table is not deleted. The deletion only removes the table and its data from the current query.
