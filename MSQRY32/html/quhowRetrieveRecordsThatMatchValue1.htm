Retrieve records that match a specific value

  1. Make sure the Auto Query button is pressed in.
  2. Add the field for which you want to specify criteria to the Data pane.

    How?

    For example, if you want to retrieve records that have France in the Country field, add the Country field to the Data pane.

  3. In the result set, click a cell that contains the value you want to match.
  4. Click Criteria Equals .
