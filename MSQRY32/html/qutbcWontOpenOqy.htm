A message tells me the OLAP Cube Wizard can't open my .oqy file.

Make sure the file was originally created by the wizard You can open and modify .oqy files only if they were created and saved by using the OLAP Cube Wizard. You cannot use the wizard to open and modify other .oqy files that do not contain a cube definition, such as those that connect to an OLAP server database or an offline cube file.

Re-create the .oqy file If you can't open an .oqy file that was created by the OLAP Cube Wizard, the file might have become damaged. Try opening a backup copy of the file, or if you don't have a backup copy, create the cube again.
