References

References are analogous to shortcuts. Reference to a document or folder lying deep in Folder hierarchy can be created at some more easily accessible level using thisthe feature of OmniDocs Engine. No copy of the object Object will be made. Only a pointer will be maintained to the actual folder or document location. Any changes in properties Properties made from here will reflect in actual document and folder.

 

NGOCreateReferences

Create Reference references for an Objectobject.

NGOMoveReferences

Move Reference references for an Objectobject.

NGODeleteReferences

Delete Reference references for an Objectobject.
