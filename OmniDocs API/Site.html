Site

 

Image Server is designed for handling the sStorage and rRetrieval of iImages in WAN environment where the iImage storage is distributed across multiple sSites. The iIndex information for the archived iImages is maintained in centralized dDatabase sServer that provides a single and centralized access to the iImages stored at multiple sSites.

 

An iImage sServer cCabinet can support multiple lLabels on a single SMS sServer as well as multiple SMS sServers. A sSite is identified by three aAttributes viz.namely, SMS ipIP, SMS pPort and lLabel . It serves as a repository for vVolumes each consisting of multiple volblocks. Site operations include API for registration of a new sSite, retrieval of lList of existing sSites and retrieval of list of lLabels of a given sSite.
