Configuration OmniDocs Engine

 

 

OmniDocs Engine can be configured in two ways. The First way is to use the Web based Configuration Assistant provided with the installation. OmniDocs Engine also provides Programmable API for configuration of individual / all components. Client application can configure OmniDocs Engine by calling a single API that abstracts the sequence and complexity of individual component configuration from user.
