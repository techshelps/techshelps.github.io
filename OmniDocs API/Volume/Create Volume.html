Create Volume

This functionality is used to create a new VolumeVolume in the Image Server at the specified Site. It returns the indexIndex for the new VolumeVolume. If volumeVolume is to be replicated at some other siteSite(s), then lList of siteSites should be provided as input. In that case it will return the sStatus of each replication.
