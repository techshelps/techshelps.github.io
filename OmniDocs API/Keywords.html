KeyWords

Keywords are the words that the userUser would like to associate with a document, so that userUser can perform sSearch on them. These keywords Keywords act as a quick reference to the documents. Further, a same keyword Keyword can be associated with the same type of documents, makinge it feasible to search all of them in one go. Two types of Keywords are supported by OmniDocs viz.namely, Authorized and UnAuthorized. Authorized Keywords cannot be deleted from the system System while UnAuthorized keywords Keywords can be.
