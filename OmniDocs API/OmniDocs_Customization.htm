OmniDocs Customization

 

OmniDocs system is built, in an open interoperable manner and various Server-side and Client-side Integration tools are available for easy integration of 3rd party business applications with OmniDocs.

 

System Integration Toolkit

 

System, Integration Toolkit is available on all major platforms, application servers and industry-standard databases.

 

         XML based Document Management and Workflow API toolkit for System Integrators. This XML is a complete functionality-level interface using which clients can easily develop their applications, while unleashing the full power of the OmniDocs Engine.

         Java Beans over the XML API available for use by any web-based application.

         COM Interface over the XML API available for integration with Windows-based applications.

         ERP Connector for integration with ERP systems.

         Image processing libraries that support image enhancements, image recognition, image compression, image manipulations, data extraction and form processing.

         ODMA support is available for integration with 3rd party DMS and business applications

         WebDAV support is available for integration with 3rd party content management tools

         Integration Points available in various client-side components such as OmniDocs Desktop through support for events and Add-Ins

         Localization kit for localizing to various languages

         Client components for Scanning, Document Viewing, etc.

         Image Enable tool for image enabling 3rd party business applications through both HTTP as well as COM interface.
