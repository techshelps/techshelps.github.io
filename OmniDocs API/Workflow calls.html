Workflow calls

This section provides the calls related to Workflow.

 

NGOCreateActionItem

To create an ActionItem.

NGOOperateActionItem

To perform various operations on an ActionItem like initiate Initiate ActionItem, fForward ActionItem, rRefer ActionItem, and rReturn ActionItem etc.

NGOSetDivert

To divert all the documents from its Inbox with a particular datadefinition DataDefinition attached to them for a specified period of time Time to some other users’ Users’ Iinbox.

NGOGetDivert

To fetch the list List of diverted documents.

NGODeleteDivert

To delete the diverted actionsActions.

NGOProcessRoutes

To process Local Routes. A User can add, delete or modify routes.

NGOOperateActionItemExt

To perform various operations on an ActionItem like Initiating, Forward, Refer and Return etc. The ActionItem document is added to a Temporary folder whose FolderIndex is specified in the input. Upon initiation the ActionItem document is moved to the Inbox of the specified Users.

NGOGetTrackInfo

To get the Track information for a given Track Number. A Track Number is a unique number used to track the ActionItem.

NGOGetRoutesList

To fetch the List of all local Routes defined by the logged-in User.

NGOGetRoutesInfo

To obtain the Route information for the login user for specified Route names.
