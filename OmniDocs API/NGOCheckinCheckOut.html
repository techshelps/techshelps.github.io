NGOCheckInCheckOut

This call provides the CheckIn and CheckOut functionalities for the document.  User who has checked out the document can alone modify the Document Property.  Other Users can modify the Document Property after the document has been checked in. To create a new Version for the checked in document, User has to add the document to the Image Server and make this call by providing the Image Index and the Volume Index returned by the Image Server.
