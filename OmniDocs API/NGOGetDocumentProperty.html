NGOGetDocumentProperty

This transaction retrieves the Property of a document. User can specify whether to fetch the data related to the associated Data Class. This call also provides information about Global Indexes and Thumbnail.

Errors returned from call

Code

Message

Comments

-50025

Document not found.

Document with the specified Document Index not found in its Parent Folder.

-50023

Document not found.

Document with the specified Document Index does not exist.

-50022

Insufficient rights for the current operation.

The logged in User does not possess sufficient Rights to get the Document Property.

-50132

Document has been deleted.

The specified document has been deleted and is in Users Trash Folder.

-50076

Invalid object access type.

 

Access type of document whose property is to be fetched is improper. This error arises in the process of rights checking on the document.
