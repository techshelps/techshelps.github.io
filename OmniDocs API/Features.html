Features of OmniDocs

Powerful Document Scanning Support

·          Scanning and uploading paper document images

·          Support for TWAIN and ISIS scanning

·          Support for scanning though browser plug-in

·          Production-level scanning through OmniCapture tool

·          Scanning multiple pages into batches for processing later

·          Scanning additional pages to existing documents

 

Simplified Document Acquisition

·          Uploading documents through HTTP and Desktop client

·          Importing image and electronic documents

·          Supporting fax to be sent to Inbox / any folder.

·          Uploading from ODMA compliant applications

·          Uploading from MS-Office applications through Add-Ins

·          Sending e-mail and Print Redirection from thin client

·          Scheduled capturing from mapped windows folders through OmniAcquire offline document acquisition tool

 

Exhaustive Document and Folder Searches

·          Powerful search facilities to locate documents and Folders

·          Combination of Profile Search, Indexed Search, and Full Text Search

·          ODMA support for search and retrieval

·          Search and retrieval of documents and folders through MS-Office AddIns

·          Search for documents or folders using document or folder profile information such as name, created, modified or accessed times, keywords, owner etc.

·          Search facilities for documents using user-defined DataClass index field values

·          Full Text Search on image and electronic documents to search through the contents of the documents

·          Search facilities across multiple versions of a document

·          Saving search queries and search results

·          Boolean and complex logical searches

·          Search facilities on annotations and notes

 

Easy Document Viewing

·          Support for viewing and annotating of image documents

·          Platform independent image document view through Image View Applet

·          Support for Document View through browser-based Plug-In

·          Support for viewing more than 200 common electronic file formats

·          Support for all standard Image Operations such as deskew, despeckle, rotate, invert, and brightness/contrast

·          Facility of zoom-in and zoom-out, zoom percentage

·          Zoom lens to zoom-in on a part of image

·          Facility to traverse to next page/previous page or specific page for viewing multi page documents

·          Display of documents properties like owner, date modified, type etc. along with the document view

·          Display of Exchange notes on documents and Secure notes for users

·          Facility to browse documents by notes,(ideal for review comments)

·          Support for Thumbnails on image and electronic documents including current and older versions

 

Flexible Annotation support

·          Facility to add text, graphics, and image annotations on document pages

·          Annotate text and image stamps on documents to indicate signatures, approval seals etc.

·          Annotations for highlighting, marking text, underlining, pointing arrows, putting sticky notes on documents, free-hand, object-embedding

·          Print documents with or without annotations

·          Multiple annotation colors

·          Facility to modify or delete annotations

·          Annotation groups for selectively hiding, deleting, or copying

·          Set secure annotations selectively for users

 

Powerful Image Editing

·          Image Editing operations such as page insertion, deletion, merge / split pages etc.

·          Document Maker tool with facility for merging/splitting documents based on bar-code /page count etc. to assemble documents from scanned batches

·          Clipping Maker tool for preparation of clippings from multiple source documents

 

Complete Information Management

·          Document Repository for managing information

·          Organizing documents in Cabinets, Folders and Subfolders for classification of information and hierarchical storage management

·          Creating user-defined DataClasses with multiple user-defined indexes for various data types

·          Associating DataClasses with documents and folders for easy indexing and retrieval of documents and information

·          Filing & Indexing of documents for quick retrieval

·          Standard Move/Copy/Delete operations for Folders and documents

·          Document and Folder shortcuts

·          Inbox and Outbox for individual users for posting documents

·          Linking of documents for cross-referencing related documents

·          Automatic Full Text Indexing

 

Seamless Content Management capability

·          WebDAV support

·          Enables users of enterprise applications to seamlessly interface with OmniDocs

·          Integration with Content Management

·          Supports for users to create, modify, review, and publish content

 

Automatic Data Capture

·          Form Builder tool to generate forms for accepting Data Class values

·          Automatic Form generation from Data Class fields

·          Zone-based OCR for automatic indexing

·          Automatic data extraction using OCR, MICR and Barcode

·          Template-based Filing

 

Strong Version Control

·          Version Management for tracking document revisions

·          Supports for image and electronic document versions, including version comments and search across versions

·          Support for auto-versioning at Cabinet and Folder level

·          Check-In and Checkout support for collaborative working on documents, allowing users to exclusively checkout documents for editing

·          Support for notifications on check-in / checkout

 

Wide-ranging Document Delivery

·          Download through HTTP

·          Support for Print/Mail/Fax of documents

·          Auto CD viewer for remote distribution

 

Adaptive Workflow for efficient document processing

·          Adaptive Workflow for routing and tracking of documents, messages, and forms

·          Collaborative working on documents in a secure environment through Workitems

·          Strong searching, reporting, and monitoring of work process and status to help in automating work processes and improve efficiency

·          Creating Ad-hoc or predefined routes for automatic document routing

·          Supports both sequential and parallel routes

·          Routing Workitems to inboxes of users for their action

·          Users can act upon, forward, return or complete their Workitems

·          Referring Workitems to other users for reference

·          Diverting Workitems to other users for delegating or substituting

·          Sending and Receiving Time-based and event-based reminders

·          Users can collaborate on a Workitem through shared and secure notes

·          Assigning tasks for each user

·          Form-based workflow through PDF forms on the Web

·          Automatic data extraction and indexing from PDF forms

 

Extensive Reports and Audit Trails

·          Extensive Audit-trails at user, Folder and Cabinet levels.

·          Audit trails on separate actions, and between specific date/times

·          System Reports and Statistics

·          Extensive tracking of each Workitem through track-sheet, which records all user actions

·          Elaborate search facilities on workflow progress through status searches, searches on user-defined indexes, full-text search and searches on Workitems date and time

·           Reporting facilities: daily, weekly or monthly reports on completed, incomplete Workitems to identify slack and improve efficiency

 

Effortless Administration

·          Easy deployment and administration

·          Definition of Users and Groups for the system

·          Option of assigning rights to access, modify, or delete documents and Folders.

·          Complete Document Lifecycle Management for managing online, offline, near-online documents, and for document archival

·          Progress tracking of Worktems

·          Facility of Notifications & Reminders

·          Support for remote administration and configuration

 

Industry-strength Security

·          Definition of Users and Groups in the system

·          Assigning users and groups access permissions on Folders, documents and Data Classes

·          Assigning multiple levels of access rights (Delete/Edit/View/None)

·          Assigning users with privileges of creating Users and Groups

·          Facility of Secured Notes sharing( Only specified users can look at the secured notes)

·          Provision of secure login and passwords for each user

·          LDAP authentication and single sign-on support

·          Support for data and password encryption

·          Support for Digital Signatures and Biometric integration

·          Support for SSL/HTTPS for secure transmission

 

Disaster Recovery

·          Tools for creating back-ups and restoring of information in the event of disaster recovery

·          Support for incremental and full Backups

·          Support for replicating data to remote sites

 

Enterprise-class Scalability and Availability

·          Multi-tier architecture with each tier made fully scalable.

·          Multiple Web servers that can hook to single or multiple transaction servers for scalability, availability, and fault tolerance.

·          J2EE and XML-based transaction server can work in a clustered environment for providing scalability, availability, and fault tolerance

·          Scalable for multiple transaction servers that can abstract the complexity of the underlying Database Server

·          Support for multiple databases that can connect to single or multiple transaction servers

 

Support for Multiple National Languages

·          Support for multiple national languages including English, Hindi, Arabic, Chinese and German

·          Localization kits for each language
