Transaction Server API

JTS API is a complete description of XML API calls used in OmniDocs and exposed by JTS. It aims at providing detail description of calls to Programmers who want to build a DMS Application using JTS. Individual description of call contains its general description, input parameters with explanation and default values, output parameter description, list of error codes and warnings thrown by that call and comments for it, which help the User to find a solution for that error or warning.  API has been logically divided into various sections for easy reference according to the functionalities supported by OmniDocs eEngine. For example,  for e.g. Cabinet, Folder, and Document etc. Object lLevel calls are those calls, which operate on any oObject like Folder, Document etc.
