Object Level calls

OmniDocs exposes some calls, which can be operated on multiple objects Objects like folderFolder, Document, Document etc for assigning rightRights, for linking objectObjects etc. Following are the calls that are exposed for objectObject level functionality.

 

 

Rights

 

NGOSetAcl

To set rightRights on an objectObject for multiple Users or groupsgroups.

NGOGetRights

To get rightRights on an objectObject.

NGOSetRights

To set rightRights on an objectObject.

DataDefinition

 

NGOAddDataDefinition

Add Data Definition to the systemSystem.

NGOGetDataDefListExt

Get list of data definitions Data Definition in the systemSystem.

NGOGetDataDefIdForName

To get Data Definition Data Definition Iindex on the basis of nameName.

NGOGetDataDefProperty

Get Data Definition propertyProperty.

NGOChangeDataDefinition

Change Data Definition propertyProperty.

NGOPickList

Get associated values for a field.

NGODeleteDataDefinition

Delete existing Data Definition. data definition

NGOAddFieldToDataDefinition

Adds fields to an existing Data Definition.

Audit Log

 

NGOFetchLog

For fetching log Log from the systemSystem.

NGOPurgeLog

To purge Llog from the systemSystem.

NGOGenerateAuditTrail

Generates the Audit trail information for a specified Action.

NGOFetchAuditTrail

Retrieves the Audit trail information for a given Action.

NGOPurgeAuditTrail

Purges the Audit trail information.

Alarms and Reminders

 

NGOGetAlarmReminderCount

Get count of generated alarms Alarms and Rreminders.

NGOSetAlarms

Set Alarms alarm on objectObject.

NGODeleteAlarms

Delete Alarmsalarms.

NGOAcknowledgeAlarms

Acknowledge Alarmsalarms.

NGOGetAlarmsForMail

Get all Alarms for mail.

NGOGetRemindersForMail

Get all Reminders Reminders for mail.

NGOGetAllAlarms

Get all Alarmsalarms.

NGOGetPendingReminders

Get all pending Remindersreminders.

NGODeleteReminder

Delete specified Remindersreminders.

NGOModifyReminderExt

Modifies the specified Reminders.

NGODeleteReminderExt

Deletes the specified list of Reminders.

NGOGetAlarms

Gets all the Alarms set on an Object.

References

 

NGOCreateReferences

To create references References for the objectObject.

NGOMoveReferences

To Move References references for the objectObject.

NGODeleteReferences

To Delete References references for the objectObject.

Link Objects

 

NGOLinkObjectObjects

To link objects Objects in the systemSystem.

NGODelLinkObjects

For deleting lLinked objectObjects.

NGOGetLinkObjectsList

To get lLinked objectObjects for a specified objectObject.
