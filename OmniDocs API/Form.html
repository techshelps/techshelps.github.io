Form

A form is basically a visual representation of a Data Definition. All the changes that the userUser wants to do in the data classData Class can be done in a more convenient way.

 

NGORegisterForm

To Register a form (which is basically a visual representation of a dData dDefinition) for a dDataDdefinition.

NGODeleteForm

Delete forms associated with the dDataDdefinition.

NGOChangeFormProperty

Change the pProperty of an existing form.

NGOGetForm

Get the FormBuffer data for a particular form.

NGOGetFormList

Get the lList of all forms associated with a dDataDdefinition.
