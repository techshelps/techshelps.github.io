Address List

An address listAddress List is a lList of userUsers or gGroups existing in the sSystem. The number of userUsers or groups Groups in an address listAddress List can be anything from 0 to the maximum number of userUsers or gGroups present in the sSystem.

NGODeleteAddressLists

NGOAddMembersToLists

Add members to address listAddress Lists for of the logged in uUser.

NGODeleteMembersFromLists

Deletes members from address listAddress Lists for of the logged in uUser.

NGOGetAddressLists

Gets all the address listAddress Lists forof the logged in uUser.

NGOGetMembersOfLists

NGOModifyAddressLists
