Object Level calls

This section contains sample Input and Output XML for Object related operations.

 

Rights

 

NGOSetAcl

To set Rights on an Object for multiple Users or groups.

NGOGetRights

To get Rights on an Object.

NGOSetRights

To set Rights on an Object.

DataDefinition

 

NGOAddDataDefinition

Add Data Definition to the System.

NGOGetDataDefListExt

Get list of Data Definition in the System.

NGOGetDataDefIdForName

To get Data Definition Index on the basis of Name.

NGOGetDataDefProperty

Get Data Definition Property.

NGOChangeDataDefinition

Change Data Definition Property.

NGOPickList

Get associated values for a field.

NGODeleteDataDefinition

Delete existing Data Definition.

NGOAddFieldToDataDefinition

Adds fields to an existing Data Definition.

Audit Log

 

NGOFetchLog

For fetching Log from the System.

NGOPurgeLog

To purge Log from the System.

NGOGenerateAuditTrail

Generates the Audit trail information for a specified Action.

NGOFetchAuditTrail

Retrieves the Audit trail information for a given Action.

NGOPurgeAuditTrail

Purges the Audit trail information.

Alarms and Reminders

 

NGOGetAlarmReminderCount

Get count of generated Alarms and Reminders.

NGOSetAlarms

Set Alarms on Object.

NGODeleteAlarms

Delete Alarms.

NGOAcknowledgeAlarms

Acknowledge Alarms.

NGOGetAlarmsForMail

Get all Alarms for mail.

NGOGetRemindersForMail

Get all Reminders for mail.

NGOGetAllAlarms

Get all Alarms.

NGOGetPendingReminders

Get all pending Reminders.

NGODeleteReminder

Delete specified Reminders.

NGOSetReminder

Get count of generated Alarms and Reminders.

NGOModifyReminderExt

Modifies the specified Reminders.

NGODeleteReminderExt

Deletes the specified list of Reminders.

NGOGetAlarms

Gets all the Alarms set on an Object.

References

 

NGOCreateReferences

To create References for the Object.

NGOMoveReferences

To Move References for the Object.

NGODeleteReferences

To Delete References for the Object.

Link Objects

 

NGOLinkObjects

To link Objects in the System.

NGODelLinkObjects

For deleting Linked Objects.

NGOGetLinkObjectsList

To get Linked Objects for a specified Object.
