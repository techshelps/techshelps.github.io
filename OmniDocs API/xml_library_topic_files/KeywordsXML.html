KeyWords

This section contains sample Input and Output XML for Keywords related operations.

 

NGOAddKeywords

To add Keywords in the System.

NGODeleteKeywords

To delete Keywords from the System.

NGOAuthorizedKeywords

To authorize existing Keywords in the System.

NGOAddKeywordsToDoc

To associate Keywords with the document.

NGODeleteKeywordsFromDoc

To delete Keywords from document.

NGODeleteAliasesFromKeywords

To delete Aliases from Keywords.

NGOGetAliasList

To get List of Aliases for Keywords.

NGOGetAllKeywordsFromDictExt

To get all Keywords from dictionary.

NGOChangeKeywords

To change existing Keywords in the System.

NGODefineAliasForKeywords

To define Aliases for Keyword.

NGOGetKeywordsListExt

To retrieve the list of Keywords for the given criteria.
