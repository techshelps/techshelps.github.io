Alarms and Reminders

This section contains sample Input and Output XML for Alarms and Reminders related operations.

 

NGOGetAlarmReminderCount

Gets the count of generated Alarms and Reminders.

NGOSetAlarms

Sets Alarm on an object.

NGODeleteAlarms

Deletes Alarms.

NGOAcknowledgeAlarms

Acknowledges Alarms.

NGOGetAlarmsForMail

Gets all Alarms for mail.

NGOGetRemindersForMail

Gets all Reminders for mail.

NGOGetAllAlarms

Gets all Alarms.

NGOGetPendingReminders

Gets all pending Reminders.

NGODeleteReminder

Deletes the specified Reminders.

NGOSetReminder

Sets Reminder for a set of Users on an object.

NGOModifyReminderExt

Modifies the specified Reminders.

NGODeleteReminderExt

Deletes the specified list of Reminders.

NGOGetAlarms

Gets all the Alarms set on an Object.
