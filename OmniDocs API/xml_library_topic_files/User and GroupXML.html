User and Group

 

This section contains sample Input and Output XML for User and Group related operations.

 

User

 

NGOAddUser

To add a User to the System.

NGOGetUserListExt

To get the User List.

NGODeleteUser

To delete the User from the System.

NGOIsAdmin

To check whether the User is Admin.

NGOChangeUserProperty

To change Properties of User.

NGOGetUserProperty

To get Property of User.

NGOSearchUser

To search for Users in the System.

NGOGetLoggedInUserLIst

To get List of currently logged in User in the Cabinet.

NGOGetShortUserProperty

To get short Properties for multiple User.

NGOCreateUserTable

To create a User-defined Table in the database.

NGODropUserTable

To drop the User Table created by using NGOCreateTable call.

NGOUserSQL

To perform basic operations like Insert, Delete, Modify, Select on User-defined Tables.

NGOValidateUserForSession

Validates a User for a specified Session.

 

 

Group

 

NGOAddGroup

To add User Group to the System.

NGODeleteGroup

To delete Group from the System.

NGOAddMemberToGroup

To add members to Group.

NGODeleteMemberFromGroup

To delete member from Group.

NGOGetGroupProperty

To get Properties of a Group.

NGOChangeGroupProperty

To change Properties of a Group.

NGOGetGroupListExt

To get List of Groups  in the System.

NGOAddaMemberToGroups

To add a User to various Groups.
