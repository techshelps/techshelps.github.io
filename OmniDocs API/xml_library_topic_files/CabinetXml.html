Cabinet

This section contains sample Input and Output XML for Cabinet operations

 

           

NGOConnectCabinet

Connects to a Cabinet.

NGODisConnectCabinet

Disconnects from the Cabinet.

NGOGetCabinetReport

Gets a report on the Cabinet.

NGOGetCabinetProperty

Retrieves the Cabinet Properties.

NGOChangeCabinetProperty

Modifies the Cabinet Properties.

NGORegisterServices

Registers a Service on the Cabinet.

NGOGetServices

Gets the List of Services registered on the Cabinet.

NGOModifyServices

Modifies the registered Services on the Cabinet.

NGODeleteServices

Deletes the registered Services from the Cabinet.
