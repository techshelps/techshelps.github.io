Address List

This section contains sample Input and Output XML for Address List related operations.

 

NGOAddAddressLists

Adds an Address List for a User.To add an address list for a user.

NGODeleteAddressLists

Deletes the specified Address Lists of the logged in User.To delete the specified address lists of the login user

NGOAddMembersToLists

Add members to Address Lists of the logged in User.To add members to address lists for the login user

NGODeleteMembersFromLists

Deletes members from Address Lists of the logged in User.To delete members from address lists for the login user

NGOGetAddressLists

Gets all the Address Lists of the logged in User.To get all the address lists for the login user

NGOGetMembersOfLists

Gets members of a specific Address List of the logged in User.To get members of a specific address list of the login user

NGOModifyAddressLists

Modifies Address Lists of the logged in User by removing the earlier members and adding new members.To modify address lists of the login user by removing the earlier members and adding the new members
