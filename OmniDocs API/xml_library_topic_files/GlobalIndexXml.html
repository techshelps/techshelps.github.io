GlobalIndex

This section contains sample Input and Output XML for Global Index related Document operations.

 

NGOGetGlobalIndexList

To get the List of Global Indexes.

NGOAddGlobalIndexToDocument

TO associate a Global Indexes for the document.

NGOChangeGlobalIndex

To change the Global Index.

NGOGetGlobalIndexFromDocument

To get Global Indexes associated with a document.

NGODeleteGlobalIndex

To delete Global Indexes from the System.

NGODeleteGlobalIndexFromDocument

To dissociate Global Indexes from the document.

NGOAddGlobalIndex

To add a new Global Index or data field to the System.

NGOModifyGlobalIndexToDocument

To modify the value of associated Global Indexes with the document.

NGOGetGlobalIndexProperty

To fetch the properties of a specified Global index.
