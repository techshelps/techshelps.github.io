Annotations

This section contains sample Input and Output XML for Annotations related Document operations.

 

NGOGetAnnotationGroupList

To get list of annotation Annotation groupsGroups.

NGOSetAnnotation

To set Annotations annotations on a document.

NGOAddAnnotation

To add Annotations annotations on a document.

NGOModifyAnnotation

To modify Annotation annotation on a document.

NGOGetAnnotationGroupProperty

To get properties of Annotation annotation Groupgroup.

NGODeleteAnnotation

To delete Annotation annotation Groupgroup.
