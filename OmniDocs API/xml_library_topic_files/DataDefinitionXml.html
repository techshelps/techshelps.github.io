DataDefinition

This section contains sample Input and Output XML for DataDefinition on Object related operations.

           

NGOAddDataDefinition

Adds Data Definition to the System.

NGOGetDataDefListExt

Gets a List of Data Definitions in the System.

NGOGetDataDefIdForName

Gets Data Definition Index on the basis of name.

NGOGetDataDefProperty

Gets Data Definition Property.

NGOChangeDataDefinition

Changes Data Definition Property.

NGOPickList

Gets the associated values for a field.

NGODeleteDataDefinition

Deletes the existing Data Definition.

NGOAddFieldToDataDefinition

Adds fields to an existing Data Definition.
