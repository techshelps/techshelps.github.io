Document

This section contains sample Input and Output XML for Document operations

 

NGOAddDocument

To add a document to the specified Folder.

NGOGetDocStatusCount

To get the count of document on the basis of Status.

NGOGetDocumentProperty

To get the Properties of the document.

NGOGetDocumentListExt

To get the List of documents in a Folder.

NGODeleteDocumentExt

To delete the single or multiple documents.

NGOCopyDocumentExt

To copy single or multiple documents.

NGOMoveDocumentExt

To move single or multiple documents

NGOCheckInCheckOutExt

To perform CheckIn-CheckOut operation on single or multiple documents.

NGOCheckInCheckOut

To perform checkout-checkout operation on single document.

NGOCreateDocumentVersion

For creating a Version of the document.

NGOSetAsLatestVersion

To set the specified Version as the latest Version.

NGOGetVersionList

To get the List of Versions for a document.

NGOModifyDocumentVersion

To modify the existing Version of a document.

NGOSearchDocumentExt

To search for documents within the specified Folders.

NGOPostDocument

For posting document to Inbox Folder of a User.

NGOChangeDocumentProperty

Change the Properties of an existing document.

NGOIsDocExistForISIndex

Fetches the number of documents having the specified image and volume indexes.

Annotations

 

NGOGetAnnotationGroupList

To get the List of Annotation Groups.

NGOSetAnnotation

To set Annotations on a document.

NGOAddAnnotation

To add Annotations on a document.

NGOModifyAnnotation

To modify Annotation on a document.

NGOGetAnnotationGroupProperty

To get the Properties of Annotation Group.

NGODeleteAnnotation

To delete the Annotation Group.

Keywords

 

NGOAddKeywords

To add Keywords in the System.

NGODeleteKeywords

For deleting Keywords from the System.

NGOAuthorizedKeywords

To authorize existing Keywords in the System.

NGOAddKeywordsToDoc

For Associating Keywords with document.

NGODeleteKeywordsFromDoc

For deleting Keywords from the document.

NGODeleteAliasesFromKeywords

To delete Aliases from Keywords.

NGOGetAliasList

To get List of Aliases for a Keyword.

NGOGetAllKeywordsFromDictExt

To get all Keywords from the dictionary.

NGOChangeKeywords

To change existing Keywords in the System

NGODefineAliasForKeywords

To define Aliases for the Keywords.

NGOGetKeywordsListExt

To retrieve the list of Keywords for the given criteria.

Thumbnail

 

NGOGetNextDocForThumbNail

To get the next document for generating Thumbnail.

NGOSetThumbNail

To set Thumbnail data for the document.

NGOGetThumbNail

To get Thumbnail data for the document.

FTS

 

NGOInitializeFTS

To enable the Cabinet for Full Text Search (FTS).

NGOGetFTSText

To get the extracted text for a specified document.

NGOFTSGetDocForFTS

To get the List of documents of a folder that is not yet extracted for FTS.

NGOFTSSetFTSDataEx

To set the extracted data for a document, which is useful in performing FTS.

NGOFTSSetFTSData

To set the FTS text data for a document and populates the full-text catalog that is useful in performing FTS.

NGOGetDocListForFts

To fetch documents with which the specified FTSIndex is associated.

GlobalIndex

 

NGOGetGlobalIndexList

To Get List of Global Indexes.

NGOAddGlobalIndexToDocument

To associate a Global Index for the document.

NGOChangeGlobalIndex

To change Global Index.

NGOGetGlobalIndexFromDocument

To get Global Index associated with a document.

NGODeleteGlobalIndex

To delete Global Index from the System.

NGODeleteGlobalIndexFromDocument

To dissociate Global Index from document.

NGOAddGlobalIndex

To add a new Global Index or data field to the System.

NGOModifyGlobalIndexToDocument

To modify the value of associated Global Index with the document.

NGOGetGlobalIndexProperty

To fetch the properties of a specified Global index.
