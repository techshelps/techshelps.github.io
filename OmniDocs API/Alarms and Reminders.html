Alarms and Reminders

Alarms are alerts that are sent to userUsers to inform that a particular action has been performed and it furnishes the details about that operation. Alarms can be set on Folder or Document for specified userUsers and gGroups. There are basically two types of alarmAlarms.  They are i.e. System Alarm, and UserUser Alarm. Alarms can either be acknowledged or deleted by the userUser. Alarms that are acknowledged by  a userUser will be generated again when the operation is performed again.

Reminders are alerts based on Date and Time. Reminders can only be set on Documents. When userUser sets a reminderReminder on document for some specified dDate and tTime, reminderReminder will be generated for that document when the time elapses. It can be set for different userUsers or Groups in the System.

 

NGOGetAlarmReminderCount

Gets the count of generated alarmAlarms and reminderReminders.

NGOSetAlarms

Sets alarmAlarm on an object.

NGODeleteAlarms

Deletes alarmAlarms.

NGOAcknowledgeAlarms

Acknowledges alarmAlarms.

NGOGetAlarmsForMail

Gets all Alarms for mail.

NGOGetRemindersForMail

Gets all Reminders for mail.

NGOGetAllAlarms

Gets all alarmAlarms.

NGOGetPendingReminders

Get Gets all pending reminderReminders.

NGODeleteReminder

Deletes the specified reminderReminder.

NGOSetReminder

Sets reminderReminder for a set of Users on an object.

NGOModifyReminderExt

Modifies the specified Reminders.

NGODeleteReminderExt

Deletes the specified list of Reminders.

NGOGetAlarms

Gets all the Alarms set on an Object.
