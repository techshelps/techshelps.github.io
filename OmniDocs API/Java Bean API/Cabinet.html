Cabinet

Cabinet is a starting point for accessing the Document Management System. After connecting to the Cabinet, Users can perform operations on the Cabinet. A cabinet contains various Folders in which Documents, which can be Image Documents or Non-Image Documents, can be uploaded. The User can create categories and User- defined iIndexes and dData cClass for easier and faster retrieval / search for one or more documents. The information pertaining to Cabinet can be stored in Database. Following functions are exposed for cCabinet operations.
