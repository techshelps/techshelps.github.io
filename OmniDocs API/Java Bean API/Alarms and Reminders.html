Alarms and Reminders

Alarms are alerts that are sent to userUsers to inform that a particular action has been performed and it furnishes the details about that operation. Alarms can be set on Folder or Document for specified userUsers and gGroups. There are basically two types of alarmAlarms.  They are i.e. System Alarm, and UserUser Alarm. Alarms can either be acknowledged or deleted by the userUser. Alarms that are acknowledged by  a userUser will be generated again when the operation is performed again.

Reminders are alerts based on Date and Time. Reminders can only be set on Documents. When userUser sets a reminderReminder on document for some specified dDate and tTime, reminderReminder will be generated for that document when the time elapses. It can be set for different userUsers or Groups in the System. Following functions are exposed for the various operations related to Alarms and Reminders.
