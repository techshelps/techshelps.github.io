Annotations

Similar to the manner in which userUser can mark the physical paper for some important information or make a note on a sheet, a facility is supported in OmniDocs Engine, for adding Annotations to the document. There are two types of annotationAnnotations in the systemSystem. These They are Image Annotations and Text AnnotationAnnotations. All annotationAnnotation Data is stored in the form of AnnotationBuffer. UserUser can also assign rRights for the annotationAnnotations in order to secure them with other userUsers and groupGroups in the systemSystem.
