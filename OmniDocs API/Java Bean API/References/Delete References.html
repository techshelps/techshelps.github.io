Delete References

 

Function Name: getDeleteReferencesXml

 

Class: com.newgen.dmsapi.DMSInputXml

 

Description: This function is used to delete the Shortcuts or References of a document or a Folder. DeleteAll flag enables the deletion of all the References for that Folder or document anywhere in the System.
