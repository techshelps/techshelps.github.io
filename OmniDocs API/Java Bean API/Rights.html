Rights

In order to secure objectsObjects, userUser can set Rights rights on different Objects objects like (dDocument, fFolder, aAnnotation or cCabinet) in DMS. One of the major features of OmniDocs Engine is that you can share your documents, folder, Data classData Class and Annotations with other userUsers of your groupGroup. You can also customize the sharing of Objects by assigning aAccess rights Rights on the objectObject. Depending upon the Rights, rights object Object can be available to the authentic userUsers and can be hidden from others. This provides sSecurity to the Objects objects created by userUser.
