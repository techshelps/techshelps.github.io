Document

It lists out the following calls that are exposed for document related operations.

 

NGOAddDocument

To add a document to the specified fFolder.

NGOGetDocStatusCount

To get the count of document on the basis of statusStatus.

NGOGetDocumentProperty

To get the  Pproperties of the document.

NGOGetDocumentListExt

To get the lList of documents in for a fFolder.

NGODeleteDocumentExt

To delete the single or multiple documents.

NGOCopyDocumentExt

To copy single or multiple documents.

NGOMoveDocumentExt

To move single or multiple documents

NGOCheckInCheckOutExt

To perform CheckIn-cCheckOout operation on single or multiple documents.

NGOCheckInCheckOut

To perform checkout-checkout operation on single document.

NGOCreateDocumentVersion

For creating a Version of the document.

NGOSetAsLastestVersion

To set the specified  some Version as the latest Version.

NGOGetVersionList

To get the lList of Versions for a document.

NGOModifyDocumentVersion

To modify the existing Version of a document.

NGOSearchDocumentExt

To search for documents within the specified fFolders.

NGOPostDocument

For posting document to iInbox fFolder of a User.

NGOChangeDocumentProperty

Change the pProperties of an existing document.

NGOChangeDocumentOrder

Change the order of documents in a particular Folder.

NGODeleteDocumentVersion

Deletes the previous Versions of the document.

NGOIsDocExistForISIndex

Fetches the number of documents having the specified image and volume indexes.
