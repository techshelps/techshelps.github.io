Link Objects

OmniDocs engine Engine provides the facility to lLink objects Objects among themselves.  For example,  e.g. a document can be linked with other documents such so that, when userUser sees the document the next time, lList of lLinked documents is also displayed.

 

NGOLinkObjects

Link objectObjects in the systemSystem.

NGODelLinkObjects

Delete lLinked objectObjects.

NGOGetLinkObjectsList

Get lLinked objectObject for an objectObject.
